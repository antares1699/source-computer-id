Untuk Mac, pengguna GitHub Tuan-Kamina telah menulis Deluge-Magnet-Handler , solusi AppleScript. Dipasang di folder Aplikasi dan dijalankan sekali untuk mengasosiasikan dirinya dengan tautan magnet:, URL tersebut akan terbuka secara otomatis di Deluge sesudahnya. 

on quit\n    try\n        tell application \"System Events\" to tell process \"Deluge\"\n            activate\n            set frontmost to true\n            windows where title contains \"Add Torrents\"\n            if result is not {} then perform action \"AXRaise\" of item 1 of result\n        end tell\n    end try\nend quit\n\non open location this_URL\n    try\n        tell application \"Finder\" to set delugePath to POSIX path of (application file id \"org.deluge\" as string)\n        set appExists to true\n    on error\n        set appExists to false\n        display alert \"Deluge.app must be installed in order to use this plug-in.\" as critical\n        return\n    end try\n    set delugeArgument to quoted form of this_URL\n    try\n        tell application \"Deluge\" to activate\n    end try\n    do shell script (quoted form of (delugePath & \"/Contents/MacOS/Deluge\") & \" add \" & delugeArgument & \"; return;\")\n    quit\nend open location\ntell application \"Finder\" to set thisPath to (POSIX path of (application file id \"org.deluge.MagnetURIHandler\" as string))\nlog thisPath\nset lsToolRead to do shell script (quoted form of (thisPath & \"/Contents/Resources/lstool\") & \" read url magnet\")\nif (lsToolRead contains \"Magnet Handler.app\") then\n    display dialog \"Magnet Handler is already the default application to handle magnet URIs\" buttons {\"OK\"} with title \"Notification\" with icon POSIX file (thisPath & \"/Contents/Resources/deluge_magnet.icns\")\nelse\n    try\n        set dialogResult to display dialog \"Magnet Handler has not been configured to handle magnet URIs.\n\nWould you like to do this now?\" buttons {\"Yes\", \"No\"} default button \"Yes\" cancel button \"No\" with title \"Notification\" with icon POSIX file (thisPath & \"/Contents/Resources/deluge_magnet.icns\")\n    on error number -128\n    end try\n    try\n        if button returned of dialogResult is \"Yes\" then\n            do shell script (quoted form of (thisPath & \"/Contents/Resources/lstool\") & \" write url magnet org.deluge.MagnetURIHandler\")\n            display dialog \"Magnet Handler has now been configured as the default application to handle magnet URIs\" buttons {\"OK\"} with title \"Notification\" with icon POSIX file (thisPath & \"/Contents/Resources/deluge_magnet.icns\")\n\n        end if\n    end try\nend if\nquit\n

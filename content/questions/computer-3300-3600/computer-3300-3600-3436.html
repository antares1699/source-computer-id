Saya tidak yakin apa yang dimaksud sebagai file sementara, tetapi ada beberapa tempat yang harus Anda periksa.

\n\n

%temp% biasanya menunjuk ke %systemdrive%\\Windows\\Temp - yang merupakan folder sementara seluruh sistem default. Namun, praktik keamanan yang baik untuk setiap pengguna memiliki folder temp mereka sendiri yang terletak %systemdrive%\\users\\{USERNAME}\\AppData\\Local\\Temp (menggantikan "{NAMA PENGGUNA}" untuk setiap folder di folder pengguna). Juga, periksa tempat sampah - apa saja yang bersembunyi di sana?

\n\n

Secara umum seperti yang disebutkan @Ramhound sangat berharga untuk memeriksa seluruh disk Anda secara visual untuk benar-benar melihat apa yang terjadi, jangan bergantung pada wizard.

\n\n

Di bawah ini adalah tangkapan layar dari TreeSize gratis , yang menyediakan tampilan struktur pohon grafis dari drive Anda dan grafik latar belakang menunjukkan di mana ruang Anda digunakan di setiap folder.

\n\n

Berhati-hatilah dengan pendekatan ini, Anda dapat merusak aplikasi atau komponen windows dengan menghapus file-nya. Tetapi dengan sedikit akal sehat dan beberapa penelitian Anda dapat menghemat ruang dengan baik dan mengungkap pemborosan yang mungkin tidak akan Anda perhatikan dan dapatkan kembali beberapa penyimpanan yang berharga.

Periksa Desktop, Dokumen Saya dan Musik, Gambar dll - hapus semua sampah, file default. WinSxS

\n\n

PERINGATAN!! JANGAN sentuh / ubah / hapus di dalam C:\\Windows\\WinSxS - gunakan di bawah perintah DISM saja - ini adalah folder sistem penting.

\n\n

WinSxS adalah folder sistem terkadang membengkak (sisa dari tambalan atau pencopotan pemasangan), Anda dapat menjalankan:

\n\n

Dism.exe /Online /Cleanup-Image /AnalyzeComponentStore

\n\n

Jika merekomendasikan pembersihan, jalankan:

\n\n

Dism.exe /online /Cleanup-Image /StartComponentCleanup /ResetBase

\n\n

CATATAN - menjalankan perintah di atas akan menghemat ruang, tetapi mencegah penghapusan pembaruan yang saat ini diinstal.

\n\n

\n\n

Folder ini diperlukan, tetapi tidak sering digunakan. Anda dapat dengan aman mengaktifkan kompresi folder ini untuk menghemat sedikit ruang. (Mengaktifkan kompresi menghemat ruang, tetapi membutuhkan lebih banyak waktu CPU - saya hanya akan mengaktifkan ini pada folder yang jarang digunakan).

Aplikasi Windows

\n\n

Ada beban komponen opsional default terpasang yang tidak muncul di panel kontrol tambah hapus. Anda perlu meluncurkan jendela Powershell sebagai Administrator. Jalankan perintah ini untuk membuat daftar paket yang diinstal: 

Get-AppxPackage -AllUsers | select -Property Name \n

Kemudian gunakan perintah berikut untuk menghapus paket (Bing News dalam hal ini diambil dari keluaran perintah sebelumnya. 

Get-AppxPackage -AllUsers  | where {$_.Name -like \"Microsoft.BingNews\" } | Remove-AppxPackage -AllUsers\n

Perintah di atas menghapus paket, set perintah berikutnya menghapus paket instalasi seluruhnya (dan c: \\ Windows \\ InfusedApps akan menjadi lebih kecil - jangan edit folder ini secara langsung). Proses yang sama, daftarkan paket yang tersedia, lalu hapus sesuai kebutuhan. Anda harus mencopot pemasangan paket sebelum Anda menghapusnya, dan beberapa paket tidak dapat dilepas (setidaknya tanpa peretasan yang jelek / tidak didukung) 

Get-AppxProvisionedPackage -Online | select DisplayName\n

Untuk menghapus paket terkait xbox gunakan Microsoft.Xbox* 

Get-AppxProvisionedPackage -Online | where {$_.DisplayName -like \"Microsoft.Xbox*\" } | Remove-AppxProvisionedPackage -Online\n

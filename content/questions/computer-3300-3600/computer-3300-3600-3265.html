Seperti yang disebutkan oleh Paul R, ada baiknya memeriksa Monitor Aktivitas untuk melihat berapa banyak RAM yang sebenarnya digunakan dan apakah lebih banyak RAM akan membantu atau tidak. Premiere CS4 adalah tidak 64-bit di Mac OS X, Premiere sendiri tidak dapat menggunakan lebih dari 4GB RAM miliknya sendiri.

\n\n

Keuntungan terbesar memiliki lebih banyak RAM adalah memungkinkan Anda untuk beralih antar program tanpa perlu halaman masuk atau keluar memori dari hard drive. Hard drive sering kali menjadi hambatan dalam hal kecepatan. (mis. Komentar Anda untuk jawaban SamB tentang: Mac OS X lambat saat start up, login kembali, dll. Adalah semua kecepatan hard drive dan bukan jumlah RAM pada saat itu).

\n\n

Mac OS X menyukai RAM dan akan menggunakan sebanyak mungkin sebelum menyentuh hard drive untuk memori virtual.

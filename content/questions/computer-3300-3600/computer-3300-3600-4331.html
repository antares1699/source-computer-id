Saya akan menghindari menggunakan drive USB Blu-Ray. Masalahnya adalah bahwa meskipun maks teoritis USB 2.0 adalah 480 Mbits / detik, itu hampir tidak mendekati maks itu saat mempertahankan streaming. Anda akan mengalami potongan sewaktu-waktu saat menonton film melalui drive USB Blu-Ray. Jika Anda bisa mendapatkan penutup USB 3.0 dan dapat memasukkan drive Blu-Ray ke dalamnya, Anda mungkin baik-baik saja.

Film Blu-ray dibaca dengan bandwidth antara 36 dan 72 Mbit / s, USB 2.0 memiliki kecepatan data 480 Mbit / s. Jadi saya pikir itu harus mungkin jika pembaca / host USB keduanya mendukung setidaknya 36/73 Mbit / s throughput.

Itu pertanyaan yang bagus, dan pertanyaan yang bahkan tidak akan dipertimbangkan oleh banyak orang sebelum mengambil risiko. Firewire akan lebih baik untuk tujuan ini, hanya karena paket informasi ditangani secara berbeda (pengontrol di kedua ujung kabel bukan hanya satu). USB 2.0 umumnya tidak mempertahankan throughput data yang konstan dan Anda mungkin mengalami kelambatan. Firewire dirancang sangat tepat untuk streaming data. USB 2.0 bagus untuk menghubungkan periferal tetapi tidak bagus untuk diandalkan untuk aplikasi bandwidth tinggi. Namun, jika itu USB 3.0 Anda akan baik-baik saja!

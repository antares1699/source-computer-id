Saya menambahkan partisi Ubuntu 11.10 (Oneiric Ocelot) ke sistem yang sudah memiliki Windows 7. Saya memilih opsi "sesuatu yang lain" untuk instalasi. (Saya ingin mempertahankan Windows, dan saya tidak ingin menginstal Ubuntu di partisi yang sama dengan Windows).

\n\n

Dari daftar partisi penginstal, saya dapat melihat partisi kosong tempat saya ingin menginstal. Saya dapat memilih sistem file sebagai ext4 , tapi saya tidak tahu apa yang harus dipilih untuk "Mount Point".

\n\n

Tepat ada satu partisi untuk Ubuntu. Partisi pertama hard drive memiliki Windows 7, partisi kedua (24 Gbytes) kosong dan tidak diformat, dan partisi ketiga adalah drive NTFS lain yang digunakan untuk Windows.

\n\n

Saya menggunakan Terabyte's BootIt untuk memilih OS pada saat boot. Apakah ada hal lain yang perlu saya lakukan untuk mencegah Ubuntu menginstal manajer bootnya sendiri yang mungkin bertentangan dengan BootIt?

Karena Anda hanya memiliki satu partisi, maka partisi tersebut harus root (/).

\n\n

Ketika penginstalan mendekati akhir, ia akan meminta Anda untuk menginstal GRUB . Anda tidak boleh menginstalnya ke seluruh disk (seperti /dev/sda), tetapi ke partisi itu sendiri tempat root diinstal.

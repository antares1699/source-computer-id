Saya membeli SSD baru dan menginstal Windows di dalamnya, tetapi saya tetap memasang HDD lama saya untuk berjaga-jaga jika saya memerlukan sesuatu. Namun, meskipun saya tidak menggunakan apa pun di HDD, saya masih bisa mendengarnya mengeluarkan suara.

\n\n

Ini terjadi terutama ketika saya mengubah program yang saya gunakan, bahkan ketika semua program dan file yang saya gunakan ada di SSD. Kebisingan bukanlah masalah; masalahnya adalah setiap kali HDD mengeluarkan suara, semuanya menjadi lambat.

\n\n

Setelah saya menggunakan program yang sama selama satu atau dua menit, HDD berhenti mengeluarkan suara dan semuanya kembali lancar.

\n\n

Apa yang menyebabkan ini?

Secara default Windows 7 menggunakan semua drive yang tersedia untuk memori virtual.

\n\n

Buka Panel Kontrol, Sistem dan Keamanan, Sistem. Klik pada "Pengaturan sistem lanjutan" di panel kiri. Buka tab Advanced. Klik "Pengaturan" di bawah Kinerja. Buka tab Advanced. Klik "Ubah" di bawah Memori Virtual. Matikan "Secara otomatis mengatur ukuran file paging untuk semua drive".

\n\n

Pilih drive lama dan pilih opsi "No paging file". Klik "Set", lalu OK. Anda kemudian harus mem-boot ulang mesin. Mudah-mudahan masalahnya akan hilang.

Karena Anda menjalankan Windows 7, Anda dapat menggunakan Resource Monitor untuk memantau semua aktivitas disk.

\n\n

Anda dapat mengklik Start dan kemudian mengetik Resource Monitor atau meluncurkannya dari Task Manager di tab Performance. Buka tab Disk untuk memantau semua aktivitas disk. Anda dapat melihat program apa yang menggunakan disk dan file apa.

\n\n

Kemungkinan besar yang akan Anda lihat adalah pengindeks sistem yang menjalankan file pengindeksan, atau layanan sistem dasar lainnya yang normal.

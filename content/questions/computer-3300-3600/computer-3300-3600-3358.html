Saya sudah menemukan solusinya. Menurut saya, cara termudah adalah:\nKatakanlah Anda memiliki 100 bookmark di Mozilla dan 4 file lagi dengan bookmark dalam format json / html.\nIdenya adalah memiliki setiap file dalam html.

\n\n
    \n
  • Jadi, ekspor yang ada di Mozilla ke html.
    • \n
  • Hapus mereka dari Mozilla sesudahnya.
    • \n
  • Tampilkan semua bookmark -> Impor dan cadangan -> Pulihkan -> Pilih file -> pilih salah satu json yang Anda miliki (jika Anda tidak punya, lewati langkah ini).
    • \n
  • Setelah itu, ekspor bookmark yang baru diperoleh ke html dan hapus dari Mozilla.
    • \n
  • Ulangi sebanyak yang diperlukan.
    • \n
\n\n

Sekarang, Anda hanya memiliki .html bookmark, sedikit file. Pastikan semua penanda dari dalam Mozilla dihapus dan mulailah mengimpor html (Impor penanda dari HTML).\nLakukan ini untuk setiap file html, jangan hapus apa pun.

\n\n

Setelah itu, cukup atur mereka, Anda akan memiliki banyak folder di dalam bookmark tetapi tidak boleh lebih dari 5 menit dan secara manual menghapus duplikat setelah mengurutkannya berdasarkan nama.

\n\n

Dan voila, tidak ada perangkat lunak, tidak ada bookmark yang hilang, semuanya membutuhkan waktu paling lama 15 menit (dan saya memiliki ratusan bookmark dalam 5 file html yang berbeda).

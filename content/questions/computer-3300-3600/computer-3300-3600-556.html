Saya memiliki modem ADSL2 + dan VDSL, dan flat saya memiliki Fiber Optik untuk saluran telepon. Saya tidak ingin mengeluarkan uang untuk modem FTTH karena ISP membatasi kecepatan dan lalu lintas paket Fiber. Mereka sama dengan paket VDSL. Mungkin ada keuntungan ping dalam mendapatkan FTTH tetapi saya tidak memerlukannya untuk kebutuhan saya.

\n\n

Apakah ISP dapat menyediakan koneksi internet XDSL melalui jalur Fiber Optik?

\n\n

\n\n
\n

Asymmetric digital subscriber line (ADSL) adalah jenis teknologi digital subscriber line (DSL), teknologi komunikasi data yang memungkinkan transmisi data lebih cepat melalui tembaga saluran telepon daripada yang dapat disediakan oleh modem pita suara konvensional.

Pertama-tama, Anda dapat melihat bahwa modem Anda memiliki sambungan listrik untuk xDSL - tetapi tidak memiliki input optik, jadi bagaimana Anda akan menghubungkan serat ISP? Hanya dengan menggunakan konverter. Entah Anda menggunakan perangkat untuk mengubah sinyal optik menjadi Ethernet tembaga, atau perangkat untuk mengubah sinyal optik menjadi VDSL tembaga, tetapi Anda memerlukan konverter dengan satu atau lain cara.

\n\n

Dan peralatan yang Anda perlukan untuk menyediakan modem dengan sinyal xDSL akan dikenakan biaya beberapa kali lebih banyak daripada konverter Ethernet biasa, belum lagi Anda akan membayar ekstra untuk listrik. (Berdasarkan pencarian ebay cepat, tampaknya bahkan DSLAM bekas mungkin bernilai ratusan dolar, sedangkan konverter GPON-ke-Ethernet mungkin berharga $ 50.)

\n\n

Jadi mengonversi ke xDSL hanya praktis di tingkat ISP. (Memang ISP sangat sering berjalan begitu saja serat ke tepi jalan dan menggunakannya untuk memberi makan peralatan yang menyediakan VDSL lokal cepat untuk semua orang di jalan.) Tapi itu bukan sesuatu yang akan bisa Anda dapatkan jika Anda sudah memiliki serat langsung di rumah, dan sejujurnya itu bukan sesuatu yang akan membuat segala jenis praktis masuk akal untuk sampai di rumah.

\n\n
\n\n

Konon, "modem serat" tidak perlu menggantikan router Anda saat ini, ia dapat bertindak murni sebagai jembatan (konverter media atau ONT). Beberapa ISP bahkan menyediakan konverter secara terpisah dari router sebenarnya sehingga Anda dapat mencampur dan mencocokkan. Yang lain memaketkan ONT bersama dengan router rumah lengkap dalam satu perangkat, tetapi biasanya mudah untuk mengubahnya ke "mode jembatan".

\n\n

Jadi jika Anda benar-benar ingin terus menggunakan modem xDSL Anda saat ini sebagai router, Anda bisa biasanya melakukannya, selama dapat dikonfigurasi untuk menggunakan Ethernet sebagai input WAN.

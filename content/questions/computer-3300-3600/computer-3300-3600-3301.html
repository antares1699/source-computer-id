Saya perlu melakukan hal berikut:

\n\n
    \n
  • Buka koneksi VPN dari rumah ke domain Windows perusahaan saya, menggunakan kredensial domain Windows perusahaan saya.
    • \n
  • Saat VPN tersambung, sambungkan ke berbagi file di jaringan kerja yang aksesnya saya miliki.
    • \n
  • Saat VPN tersambung, buka klien Desktop Jarak Jauh ke PC kerja yang aksesnya saya miliki.
    • \n
\n\n

Saya tidak pernah membayangkan perlu terhubung ke arah yang berlawanan.

\n\n

Saya adalah admin IT di komputer saya. Pengguna akhir di sini dengan edisi premium rumah Win 7 tidak dapat menggunakan fitur VPN di komputer mereka untuk masuk ke domain kami. Namun mereka dapat melakukan RDP ke jaringan kami. Jadi, jika Anda memerlukan akses lengkap di jaringan dan akses ke semua komputer dan file bersama, Anda tidak menginginkan Windows edisi rumahan. RDP hanya akan memungkinkan Anda mengakses ke komputer yang Anda masuki, bukan seluruh jaringan atau sumber dayanya seperti printer, file, faks, pemindai, dan komputer lain. Tentu saja itu yang terjadi di jaringan saya, hubungi departemen TI Anda untuk memastikannya. Semoga membantu.

Saya sedang melihat sebuah Motherboard Asus dan dikatakan: 

Integrated Graphics Processor \nMulti-VGA output support : HDMI/DVI-D/RGB ports \n - Supports HDMI with max. resolution 4096 x 2160 @ 24 Hz / 2560 x 1600 @ 60 Hz\n - Supports DVI-D with max. resolution 1920 x 1200 @ 60 Hz\n - Supports RGB with max. resolution 1920 x 1200 @ 60 Hz\nMaximum shared memory of 1024 MB\nSupports Intel® InTru™ 3D, Quick Sync Video, Clear Video HD Technology, Insider™\nSupports up to 3 displays simultaneously\n

Setelah membaca manual, tidak jelas apakah tiga monitor yang didukungnya adalah {1 HDMI + 1 DVI-D dan 1 RGB} atau {3 HDMI mencakup area 4096 * 2160}.

\n\n

Idealnya saya ingin mendukung tiga monitor HDMI (tanpa membeli kartu grafis), yaitu, untuk membagi 4096 * 2160 menjadi dua atau tiga monitor dengan HDMI.

\n\n

Saya memperhatikan untuk melakukan permainan apa pun, saya hanya ingin pengaturan rumah yang sangat bagus untuk pengembang web dan komputasi umum. Karena saya tidak akan bermain game, saya lebih suka tidak mendapatkan kartu grafis terpisah.

\n\n

Apakah saya tidak mengetahui trik yang jelas untuk mendukung tiga monitor dari satu port? Atau apakah saya harus mendapatkan kartu grafis untuk mendukung banyak monitor melalui antarmuka yang sama? Atau (mengingat saat ini saya hanya memiliki satu monitor) bertujuan untuk mendapatkan monitor berikutnya melalui antarmuka yang berbeda? Meskipun saya sudah membacanya menggunakan port VGA dengan monitor modern adalah pemborosan waktu .

Multi-VGA output support : HDMI/DVI-D/RGB ports \n - Supports HDMI with max. resolution 4096 x 2160 @ 24 Hz / 2560 x 1600 @ 60 Hz\n - Supports DVI-D with max. resolution 1920 x 1200 @ 60 Hz\n - Supports RGB with max. resolution 1920 x 1200 @ 60 Hz\nMaximum shared memory of 1024 MB\nSupports Intel® InTru™ 3D, Quick Sync Video, Clear Video HD Technology, Insider™\nSupports up to 3 displays simultaneously

\n\n

Berdasarkan informasi ini, ya dapat mendukung 3 monitor, 1 koneksi dari HDMI, 1 dari DVI-D dan 1 dari Port VGA.

\n\n

\n\n

Tidak semua dari 1 koneksi HDMI. Dan jika menggunakan VGA menjadi masalah, dapatkan konverter VGA ke DVI.

\n\n

\n\n

sunting: jika Anda benar-benar menginginkan DVI / HDMI penuh dengan hingga 4 monitor, saya sarankan melihat ke kartu video berdiri sendiri yang lebih canggih yang biasanya mendukung 2 dvi, 1 HDMI, dan 1 port tampilan. Atau kombinasi lain yang tidak menyertakan VGA.

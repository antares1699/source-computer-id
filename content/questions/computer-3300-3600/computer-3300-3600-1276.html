Grup sudo tidak istimewa dengan sendirinya. Ini memiliki akses sudo karena (jika) ada baris seperti ini di file sudoers: 

%sudo   ALL=(ALL:ALL) ALL\n

Jadi, bagaimanapun, file itulah yang paling penting.

Konfigurasi default Ubuntu /etc/sudoers memiliki dua grup untuk memungkinkan akses sudo. Anggota grup ini diizinkan mengakses sudo tanpa memerlukan pengeditan ke /etc/sudoers atau penambahan file konfigurasi ke /etc/sudoers.d. 

# Members of the admin group may gain root privileges\n%admin ALL=(ALL) ALL\n\n# Allow members of group sudo to execute any command\n%sudo   ALL=(ALL:ALL) ALL\n

Cara yang disukai untuk melakukan penambahan konfigurasi lokal adalah dengan menambahkan file ke /etc/sudoers.d dengan konfigurasi yang diperlukan. Ini memungkinkan konfigurasi default diperbarui secara otomatis. Gunakan visudo untuk setidaknya memverifikasi perubahan ini

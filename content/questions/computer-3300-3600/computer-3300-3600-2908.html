Tampaknya ini adalah pesan kesalahan standar yang hanya berarti bahwa URL tidak valid telah diminta.\nItu dapat memiliki banyak kemungkinan sumber, seperti plugin yang buruk atau situs web yang diprogram dengan buruk.\nSayangnya sumber pesan tidak ditentukan.

\n\n

Jika Anda seorang pengembang, Anda dapat mencoba men-debug apa yang terjadi menggunakan konsol Alat Pengembang Chrome dan Firebug Lite untuk Google Chrome .\nJika tidak, lanjutkan dengan jawaban ini.

\n\n

Saya sarankan untuk membersihkan cache DNS Anda: 

sudo dscacheutil -flushcache\n

Saya ini tidak membantu, saya sarankan Anda mencoba melihat apakah ini terjadi menggunakan browser lain.\nJika masih terjadi, maka masalahnya ada pada: ISP Anda, router Anda, server DNS Anda.

\n\n

Jika ini tidak terjadi dengan browser lain, maka uji Chrome tanpa plugin.\nJika masalah hilang, maka ini disebabkan oleh salah satu plugin Anda, jadi Anda harus mencari yang mana dengan coba-coba.

\n\n

Jika masalah masih terjadi di Chrome tanpa plugin, maka hapus instalan Chrome, bersihkan semua folder,\ninstal ulang Chrome, lalu tambahkan plugin apa pun dengan hati-hati satu per satu sambil memeriksa apakah masalahnya kembali.

\n\n

Jika tidak ada yang berhasil, verifikasi sistem Anda dengan beberapa produk antivirus terkenal.

\n\n

Jika masalahnya hanya di Chrome tetapi mencopot pemasangannya tidak membantu,\nmaka satu-satunya saran yang tersisa adalah menggunakan browser lain.

\n\n

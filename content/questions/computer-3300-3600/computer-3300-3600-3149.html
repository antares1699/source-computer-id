AxCrypt hanya mengenkripsi file tunggal. Anda dapat mengenkripsi banyak file sekaligus, tetapi masing-masing file dienkripsi secara terpisah (dengan kata sandi yang sama jika Anda ingin melakukannya).

\n\n

... dan file kunci khusus (seperti di TrueCrypt).

\n
\n\n

AxCrypt juga dapat menghasilkan file kunci.

\n\n
\n

Akan lebih bagus (tetapi tidak wajib) jika file tidak memerlukannya\n perangkat lunak pihak ketiga untuk mengenkripsi dan mendekripsi, dalam arti saya\n berikan kata sandi dan kunci ke file secara langsung.

\n
\n\n

AxCrypt memberi Anda opsi untuk membuat self-extracting .exe file alih-alih file terenkripsi yang khas. Penggalian diri .exe File tidak perlu menginstal AxCrypt untuk mendekripsi, tetapi jelas masih membutuhkan kata sandi atau file kunci untuk mendekripsi.

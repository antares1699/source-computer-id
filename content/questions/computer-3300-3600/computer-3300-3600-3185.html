Sudahkah Anda menginstal libdvdcss2?\nHalaman ini akan membantu Anda: https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs

\n\n

Hanya untuk Memperjelas, Anda hanya perlu menggunakan dua perintah ini: 

sudo apt-get install libdvdread4\n

lalu: 

sudo /usr/share/doc/libdvdread4/install-css.sh\n

Setelah ini, saya biasanya menggunakan VLC untuk memutar DVD, karena IMHO, itu adalah pemutar Media terbaik di luar sana.

Apakah ada cara untuk terhubung dari jarak jauh ke pengguna lain saat masih ada pengguna di PC yang sama.\nJadi satu PC dengan pengguna dan keduanya masuk pada saat yang sama, satu secara lokal dan satu lagi dari jarak jauh?

\n\n

Maaf saya tidak menjelaskan pertanyaan saya dengan sangat baik, saya tidak bermaksud 2 pengguna melihat layar yang sama. Maksud saya 2 akun pengguna. Katakanlah UserA sedang duduk di meja di akunnya. dan kemudian UserB ada di akunnya tetapi terhubung dari jarak jauh. Jadi mereka berdua memiliki desktop yang berbeda di depan pada layarnya masing-masing.

","source":"https://superuser.com/questions/95354/remote-desktop-to-a-user-while-still-logged-on","author":"Jonathan.","replies":[],"answers":[{"answer":"

Sejauh yang saya tahu, OS desktop Microsoft Windows (XP, Vista, 7) tidak memungkinkan untuk beberapa sesi simultan. Hanya Windows Server (dengan Terminal Server) yang memungkinkan untuk itu. Sesi desktop jarak jauh dalam jumlah terbatas diizinkan secara default dengan setiap lisensi Windows Server.

\n\n

Pada Windows XP, mereka melaporkan berhasil menyetelnya menggunakan DLL dari Windows Server. Ini adalah peretasan yang jelek. Anda dapat menemukan petunjuknya di a pasangan dari tempat .

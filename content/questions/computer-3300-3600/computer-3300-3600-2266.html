Catatan: Saya TIDAK mengacu pada Internet Explorer.

\n\n

Saya menggunakan Windows XP dan Windows Server 2008 dan perlu menghapus nilai riwayat dari file browser (alias Windows Explorer). Seseorang memasukkan kata sandi ke dalam bilah alamat sebagai ftp: // user: pass , dan sekarang saya tidak dapat menghapus nilainya.

\n\n

Beberapa forum mengatakan untuk menghapus kunci registri ini:

\n\n

HKEY_CURRENT_USER \\ Software \\ Microsoft \\ Windows \\ Curre ntVersion \\ Explorer \\ TypedPaths

\n\n

Di Windows XP, kuncinya tidak ada, dan di Windows Server 2008, kuncinya ada di sana, tetapi menghapusnya sepertinya tidak membantu.

\n\n

Jawabannya

\n\n

Untuk XP, Anda perlu menggunakan CCleaner, pilih hapus Riwayat dan URL yang Baru Diketik dari Internet Explorer dan Dokumen Terbaru dan MRU Penjelajah Lain dari Windows Explorer

\n\n

Untuk Windows Server 2008, login ke situs FTP, tekan ALT, lalu lakukan File> Login, di mana Anda dapat mengubah atau menghapus kata sandi.

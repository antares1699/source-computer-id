Saya mengalami kesalahan yang sama dengan versi 2.2.5, saya akhirnya menghapus wireshark sepenuhnya 

sudo apt-get remove wireshark\n

Saya kemudian menginstal ulang versi qt yang sepertinya memperbaikinya, meskipun saya tidak yakin mengapa 

sudo apt-get install wireshark-qt\n

Saya pikir itu adalah sesuatu dalam versi saat ini yang mengacaukan segalanya, tetapi saya tidak dapat mengatakan dengan pasti

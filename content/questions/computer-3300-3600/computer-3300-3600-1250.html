Setelah pengukuran saya, saya berakhir dengan deretan tes yang gagal dan lulus (x = gagal, o = lulus).\nBaris ini dimulai dengan x atau o dan berubah satu kali. 

Example A) xxxoo\nExample B) ooox\nExample C) ooxxxxx\n

Saya sekarang mencari fungsi yang memberi saya bagian kedua dari string di mana semua huruf sama hingga maksimal 4 huruf.\nJadi hasilnya adalah: 

A) oo\nB) x\nC) xxxx\n

Saya tahu bahwa saya bisa mendapatkan digit terakhir dengan RIGHT() tetapi saya tidak tahu bagaimana memeriksa panjang bagian "hanya x" atau "hanya o" di akhir.

Mengingat kendala pada data Anda, di mana ada hanya satu transisi , Anda dapat mengulangi karakter paling kanan berapa kali karakter itu ada dalam string Anda: 

=REPT(RIGHT(A1),LEN(A1)-LEN(SUBSTITUTE(A1,RIGHT(A1),"")))\n

\n

Tidak yakin apakah "empat huruf" merujuk ke batasan data atau fungsi rumus, tetapi jika mungkin ada lebih dari empat huruf, dan Anda hanya ingin mengembalikan empat huruf terakhir, lalu sematkan rumus di atas dalam fungsi KANAN: =right(the_formula,4) *

Jika string hanya dapat berubah satu kali, maka yang perlu Anda lakukan hanyalah "membuang" karakter kiri, lalu membatasi sisanya menjadi 4. "Substitusi" akan menukar string dengan yang lain, sehingga Anda dapat menukar karakter paling kiri dengan tidak ada , betapapun itu muncul. 

=LEFT(SUBSTITUTE(A1,LEFT(A1,1),""),4)\n

Jika string bisa berubah nol kali misalnya ooooo atau xxxxxx maka ini akan memberikan hasil kosong. Ini mungkin dianggap benar jika Anda hanya menginginkan kedua surat karena tidak ada satu pun.

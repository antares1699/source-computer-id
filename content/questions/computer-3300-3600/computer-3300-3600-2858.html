Mengikuti saran shaneselman ( http://www.hanselman.com/blog/TheComputerBackupRuleOfThree.aspx ), Saya menyiapkan drive eksternal 2TB sebagai cadangan, dan mengenkripsinya menggunakan TrueCrypt. Ini di bawah Windows 7. (Saya telah menggunakan TrueCrypt sebelumnya, tetapi tidak sejak hari XP.) Saya menginstal TrueCrypt dengan opsi default.

\n\n

Setelah mengambil 23 jam untuk memformat penampung 2TB, ini diakhiri dengan kesalahan yang mengatakan "TrueCrypt tidak dapat memperoleh hak administrator." Ketika saya mengklik OK, ia mengatakan itu tidak dapat memformat wadah sebagai NTFS, apakah saya ingin memformatnya sebagai FAT, dan saya tidak melakukannya, jadi saya klik Tidak, dan itu keluar tanpa menunjukkan apa pun selama 23 jam komputasi waktu.

\n\n

Jadi saya menetapkan "Jalankan sebagai bendera administrator" pada pintasan TrueCrypt, dan coba lagi. Kali ini, ketika saya pergi ke format isi (siap untuk menunggu 23 jam LAIN), itu muncul dialog yang mengatakan Anda memformat ini sebagai administrator. "Volume dapat dibuat dengan izin yang tidak memungkinkan Anda untuk menulis ke volume saat dipasang. Jika Anda ingin menghindarinya, tutup contoh Volume Creation Wizard ini dan luncurkan yang baru tanpa hak administrator."

\n\n

Bagus - Saya dapat menjalankan sebagai non-administrator, dan volume sama sekali akan gagal, atau saya dapat menjalankan sebagai administrator dan membuat volume yang tidak dapat saya tulis.

\n\n

Saya pikir saya melakukan sesuatu yang salah, tetapi tidak jelas bagi saya apa. Ada bantuan?

Jawaban untuk ini adalah bahwa TrueCrypt, pada penyelesaian operasi pemformatan, memunculkan kotak dialog yang meminta akses Administrator (mungkin jika Anda mengaktifkan UAC, yang saya lakukan). Tetapi jika tidak ada respons untuk dialog itu dalam waktu yang wajar (saya kira beberapa menit), maka dialog akan habis waktu dan gagal, menyebabkan pesan kesalahan yang diamati.

\n\n

Karena operasi format memakan waktu hampir 24 jam, saya tidak duduk di sana menonton hingga selesai. Ketika saya menemukan ini dan menjalankannya kembali, saya memperhatikan waktu yang tersisa, dan memastikan bahwa saya ada di sana untuk menanggapi Dialog ketika operasi format selesai.

Anda bisa menggunakan asciiview perintah, dari pria aview :

\n\n
\n

aview hanya dapat menampilkan file dalam format file PNM. Tapi asciiview\n secara otomatis mengubah format file ke PNM (dengan program eksternal\n seperti NetPBM dan ImageMagic) dan kemudian menjalankan tampilan.

\n
\n\n

Jadi gunakan saja: 

$ asciiview file.jpg\n

Jika tidak berhasil, Anda mungkin perlu menggunakan: 

$ sudo apt-get install imagemagick\n

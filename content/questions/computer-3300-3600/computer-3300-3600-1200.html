Pertanyaan serupa pernah diajukan sebelumnya, tetapi saya tidak dapat menemukannya. Intinya adalah bahwa Windows bekerja secara berbeda jika dibandingkan dengan sistem operasi ini. Mereka tidak memiliki registri, dan tidak menjalankan program dengan cara yang sama. Saya pikir lebih mudah untuk menambahkan kekacauan pada sistem Anda di Windows jika Anda tidak tahu apa yang Anda lakukan dan benar-benar menurunkan kecepatan. Di Linux dan Mac, kebanyakan hal hanyalah file datar. Folder sementara biasanya dihapus saat boot dan jumlahnya tidak banyak jika Anda ingin menghapusnya sendiri. Ada program yang tersedia yang akan menghapus folder ini untuk Anda. Onyx untuk Mac muncul dalam pikiran sebagai satu kesatuan.

OS X sudah memiliki set skrip pemeliharaan yang dijalankan (harian, mingguan, dan bulanan) untuk membersihkan cache dan memutar log.

\n\n

Anda dapat mengunduh utilitas seperti Onyx untuk menjalankannya kapan pun Anda mau, atau Mac Anda hanya akan menjalankannya pada jadwal cron-nya. Ini membuat sistem Anda cukup bersih.

\n\n

Untuk menjalankannya secara manual: 

sudo periodic daily weekly monthly\n

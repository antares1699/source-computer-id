Solusi Kerja

\n\n

Di baris perintah: 

vssadmin List Shadows\n

Ini akan menampilkan daftar salinan bayangan Anda (bayangan folder CSC tidak akan dikunci). Cari dan salin alamat asli bernomor tertinggi.

\n\n

Kedua, kami membuat tautan ke salinan bayangan: 

mklink /D C:\\LatestShadow \\\\?\\GLOBALROOT\\Device\\HarddiskVolumeShadowCopy417\\\n

Bagian paling kanan adalah alamat asli yang Anda salin sebelumnya

\n\n

Selanjutnya gunakan robocopy untuk mengeluarkan semua file dari cache 

mkdir C:\\CSC_copy\nrobocopy C:\\LatestShadow\\Windows\\CSC C:\\CSC_copy /E /COPY:DT\n

Anda akan mendapatkan salinan dari semua file cache di C: \\ CSC_Copy.

Gunakan perintah ini untuk memberikan akses penuh 

cd %systemroot%\ntakeown /f csc /r /a /d y\nicacls csc /grant Everyone:(F) /t /l /q\n

lalu reboot. 

vssadmin List Shadows\n

Ini akan menampilkan daftar salinan bayangan Anda (bayangan folder CSC tidak akan dikunci). Cari dan salin alamat asli bernomor tertinggi.\nKedua, kami membuat tautan ke salinan bayangan: 

mklink /D C:\\LatestShadow \\\\?\\GLOBALROOT\\Device\\HarddiskVolumeShadowCopy417\\\n

mkdir C:\\CSC_copy\nrobocopy C:\\LatestShadow\\Windows\\CSC C:\\CSC_copy /E /zb /copyall \n

kemudian mengambil alih kepemilikan file di C:\\CSC_Copy tersebut

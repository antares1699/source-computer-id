\n\n
    \n
  • Tarik baris saat ini ke papan klip X11 : " + y y
    • \n
  • Tempel dari papan klip X11 : " + p
    • \n
  • Tarik jalur saat ini ke pilihan X11 (utama) : " * y y
    • \n
  • Tempel dari pilihan X11 (utama) : " * p
    • \n
\n\n

Modifikasi bagian gerak sesuai selera. vim harus mengetahui bahwa server X ada di sana, tentu saja. vi yang termasuk dalam banyak distribusi GNU / Linux adalah versi kecil dari Vim yang telah dikompilasi tanpa banyak fitur Vim, termasuk antarmuka ke papan klip Sistem X Window. Bahkan program mode terminal vim mereka sering dibuat tanpa dukungan untuk X. Tanpa dukungan untuk X, vi dan vim Anda tidak akan dapat mengakses papan klip dan Anda tidak akan dapat menyalin dan menempel di antara berbagai contoh program.

\n\n

Salah satu solusi untuk itu adalah dengan menjalankan program gvim dalam mode terminal dengan menggunakan opsi -v, sebagai 

$ gvim -v your_file\n

Untuk menghindari mengetik itu, atau lupa mengetiknya, setiap saat, Anda bisa menambahkan alias ini ke file ~ / .bashrc Anda: 

alias vim='gvim -v'\nalias vi='gvim -v'\n

Setelah Anda menggunakan versi Vim yang dikompilasi dengan dukungan untuk X, Anda dapat menggunakan perintah yang dijelaskan dalam jawaban JdeBP. Anda juga dapat membaca tentang perintah tersebut di manual pengguna internal Vim dan manual referensi dengan menjalankan 

:help 04.7\n:help 09.3\n

XPS 1340 memiliki konektor SATA standar yang digunakan pada hard drive 3,5 inci dan 2,5 inci. Jadi, SSD SATA 2,5 inci apa pun bisa digunakan. SATA 6Gbps kompatibel dengan SATA 3Gbps, jadi jangan khawatir.

\n\n

Drive yang lebih tebal dari 9,5 mm mungkin tidak muat di ruang hard drive, tapi menurut saya tidak ada SSD mainstream yang lebih tebal dari 9,5 mm.

\n\n

Ada ruang untuk HDD / SSD 2,5 inci kedua. SSD kedua saya disimpan di teluk ODD. Saya tidak menggunakan disk lagi jadi saya mengeluarkan ODD. Anda memerlukan adaptor untuk dapat menggunakan hard drive 2,5 inci di tempat ODD karena ODD SATA adalah konektor yang berbeda (lebih kecil). Tapi itu mungkin.

\n\n

Saya menggunakan satu drive untuk sistem dan aplikasi dan yang lainnya untuk data. Namun perlu diperhatikan bahwa BIOS tidak memungkinkan Anda melakukan boot dari port SATA ODD secara default - jadi pastikan untuk meletakkan drive boot Anda di tempat hard drive utama.

Bagi saya, ini terlihat seperti port IDE, meskipun saya tidak yakin 100%.

\n\n

Jika ya, mungkin sulit menemukan SSD. Berikut daftar IDE SSD yang ada Amazon .

\n\n

Idealnya, Anda menginginkan tempat yang cukup untuk Windows dan program yang paling sering Anda gunakan. SSD 32GB akan cukup untuk Windows dan mungkin beberapa program, tetapi tentu saja tidak untuk game besar.

\n\n

64GB memungkinkan beberapa game ditempatkan di SSD. Jika Anda menginginkan cukup ruang, pilih 120GB / 128GB. Apa pun di atas itu akan menjadi mahal per GB.

\n\n

Hal lain yang harus dicari adalah tersedia TRIM.

\n\n

Sejauh kecepatan berjalan, IDE tidak akan memberi Anda banyak manfaat. Selain itu, saat Anda membeli PC baru, kemungkinan besar Anda tidak dapat menggunakan SSD. Semuanya SATA saat ini.

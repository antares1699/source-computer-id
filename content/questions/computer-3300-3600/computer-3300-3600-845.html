Ketika saya mengatur XBox 360 saya sebagai Windows Media Extender meskipun saya dapat melihat semua media yang saya miliki di PC Windows Vista, sangat lambat untuk berpindah antar album, dll.

\n\n

Adakah yang dapat saya lakukan untuk memperbaikinya atau hanya karena jumlah file media yang saya miliki - sekitar 5GB?

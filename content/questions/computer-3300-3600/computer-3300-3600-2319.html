Saya memiliki Fallout Collection dan saya mencoba menjalankannya dalam jendela karena resolusinya yang rendah. Ketika saya mengubah mode menjadi berjendela melalui file .ini, bagaimanapun, itu memulai permainan dengan jendela didorong ke sudut kiri atas layar saya.

\n\n

Mengklik bilah judul jendela hanya mengklik kembali ke program gim. Jadi bagaimana cara memindahkannya ke tengah layar saya? Saya menggunakan Windows 7 professional 64-bit.

","source":"https://superuser.com/questions/136886/moving-a-window-that-cant-be-dragged-by-the-title-bar","author":"ccat","replies":[],"answers":[{"answer":"

Saya selalu menggunakan Alt-Space dalam skenario jenis ini. Itu akan memunculkan menu di mana Anda dapat memilih "Pindah", lalu gunakan tombol panah untuk memindahkan jendela. Tekan Enter untuk menguncinya pada posisinya.

Anda dapat menggunakan perangkat lunak "AutoHotKey", yang memiliki fungsi WinMove. 

WinMove, 100,100\n

Akan memindahkan jendela yang sedang aktif ke (100,100), dan 

WinMove, Window Title,,100,100\n

Akan memindahkan [Judul Jendela] ke 100.100.

\n\n

Misalnya: 

#q::WinMove, 100,100\n

Ketika ditekan saat FO Trilogy aktif akan memindahkan jendela ke 100,100, bukan 0,0.

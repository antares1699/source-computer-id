Kebanyakan laptop tidak mengizinkan akses ke voltase dan frekuensi melalui bios, dalam hal ini Anda mungkin tidak dapat menyelesaikan tugas Anda.

\n\n

Namun, jika Anda cukup beruntung memiliki laptop yang memungkinkan akses, maka tugasnya relatif sederhana. Berhati-hatilah bahwa melakukan kesalahan berikut dapat menyebabkan masalah, membatalkan banyak barang Anda, atau bahkan merusak komputer Anda tanpa batas. Sangat disarankan agar Anda membaca tentang overclocking / underclocking sebelum mencobanya. Masuk ke BIOS laptop Anda (Biasanya DEL saat booting, terkadang berbeda. Perhatikan layar boot) dan cari kontrol Frekuensi / Tegangan. Lihat apakah Anda dapat menurunkan BCLK Anda, atau pengganda CPU Anda. Jatuhkan itu ke Kecepatan CPU apa pun yang Anda cari. Setelah itu, turunkan voltase Anda sedikit , dan boot. Jika komputer Anda tetap stabil, Anda bisa turunkan voltase Anda lagi. Pastikan, bagaimanapun, bahwa Anda tetap pada tegangan yang stabil, jika tidak laptop Anda mungkin layar biru secara acak, restart, atau lebih buruk.

\n\n

Namun, jika Anda tidak dapat mengakses perintah ini di bios, cari pengaturan kipas di bios, dan maksimalkan hingga 100% setiap saat. Jika tidak ada, unduh Speedfan , dan lihat apakah itu dapat mengubah kecepatan kipas Anda.

\n\n

Jika semuanya gagal, dapatkan alas pendingin laptop yang bagus. Targus, dan banyak perusahaan lain yang membuatnya, dan mereka dapat ditemukan di banyak lokasi, termasuk Walmart.

\n\n

Saya harap Anda menyelesaikan masalah panas ini;)

Mungkin cara yang paling alami adalah dengan menggunakan program cpufreqd. Setelah menginstal, uji dengan cpufreq-info jika CPU Anda mati (profil default harus "sesuai permintaan", yang berarti, jika Anda membutuhkannya, ini akan memaksimalkan CPU kecepatan, jika tidak maka paling rendah).

\n\n

cpufreqd mungkin hanya berfungsi untuk CPU Intel. Jika Anda memiliki AMD, gunakan powernowd sebagai gantinya.

\n

Di Linux Anda memiliki perintah cpufreq-set (paket 'cpufrequtils') dan cpufreq-set (paket linux-tools-common).

Untuk mengatur frekuensi maksimal 2 GHz: 

sudo cpufreq-set -u 2.0        # Using cpufrequtils\n\n# Or\ncpupower frequency-set -u 2000 # Using linux-tools-common\n

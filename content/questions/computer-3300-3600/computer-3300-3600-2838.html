Bagaimana saya bisa memberi userA akses ke semua file yang dibuat oleh userB / userC, termasuk izin rwx?

\n\n

Apakah ada cara untuk melakukan ini tanpa memberikan akses root?

\n\n

Menambahkan ke grup yang sama tidak akan berfungsi karena tidak semua file memiliki akses grup.\nDan setfacl dapat dijalankan pada file saat ini tetapi tidak berlaku untuk file yang dibuat di masa mendatang.

Gunakan ACL. 

setfacl -R -m user:User:rwX /some/path\n

(perhatikan huruf besar X)

\n\n

setfacl tidak berlaku untuk file mendatang yang dibuat, jika Anda juga menyetel ACL "default": 

setfacl -R -m default:user:UserA:rwx /some/path\n

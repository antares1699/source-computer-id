Jika Anda ingin tidak terlihat oleh google analytics dari beberapa situs - cara terbaik untuk melakukannya adalah dengan menimpa URL Google Analytics di file host windows:

\n\n

Buka file ini di notepad: 

C:\\Windows\\System32\\drivers\\etc\\hosts\n

Anda akan melihat string seperti itu: 

# localhost name resolution is handled within DNS itself.\n#   127.0.0.1       localhost\n#   ::1             localhost\n

tambahkan string ini ke akhir file: 

   209.85.225.113  localhost\n   209.85.225.138  localhost    \n   209.85.225.139  localhost \n   209.85.225.100  localhost \n   209.85.225.101  localhost\n   209.85.225.102  localhost\n

yang akan membuat semua panggilan Google Analytics menjadi lokal, yaitu bukan ke server GA yang sebenarnya

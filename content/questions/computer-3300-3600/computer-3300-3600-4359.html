Saya sedang berpikir untuk menggunakan headset bluetooth untuk digunakan terutama sebagai mikrofon komputer. Pertanyaannya adalah: apakah biasanya memiliki kualitas yang baik dibandingkan dengan mikrofon nirkabel tradisional (atau mikrofon berkabel klasik)? Apakah mereka cukup baik untuk perangkat lunak pengenalan suara? Saya tidak butuh kualitas terbaik, hanya sesuatu yang layak.

\n\n

Saya memiliki headset Bluetooth Plantronics Calisto B70 yang disertakan dengan Dragon Naturally Speaking ( Amazon.com: Dragon NaturallySpeaking Premium 11 Bluetooth ).

\n\n

Jika saya mencoba menggunakan perekam suara, saya juga dapat mendengar suara berderak serta suara saya, jadi menurut saya kualitasnya tidak bagus.

\n\n

Suatu kali, saya bisa membuatnya bekerja 99% secara akurat dengan DNS, tetapi sejak itu saya belum dapat mengulangi keakuratan itu.

\n\n

Headset lebih buruk di Pengenalan Ucapan Windows, mungkin karena tidak disetel untuk Bluetooth seperti DNS.

\n\n

Untuk apa nilainya, saya mencoba melakukan rekaman suara menggunakan beberapa pasang headphone kabel analog, dan ada noise juga, jadi saya pikir headphone USB kabel akan menjadi yang terbaik, tetapi tidak dapat memastikannya.

\n\n

MEMPERBARUI

\n\n

Saya baru saja membeli Headset USB Plantronics 625 .

\n\n

Training Dragon Naturally Speaking tampaknya bekerja lebih baik daripada dengan B70 Bluetooth.

\n\n

Saya juga mencoba merekam suara saya sebagai cara untuk membandingkannya. B70 terdengar oke, tetapi memiliki sedikit desisan di latar belakang dan memiliki rentang frekuensi yang lebih rendah. 625 terdengar jauh lebih jelas dan lebih alami.

\n\n

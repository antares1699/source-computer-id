Cara menentukan nomor Indeks ikon

\n\n

Menggunakan Ikon Ekstrak dari NirSoft :

\n\n
\n

Utilitas IconsExtract memindai file dan folder di komputer Anda, dan mengekstrak ikon dan kursor yang disimpan dalam EXE, DLL, OCX, CPL, dan jenis file lainnya. Anda dapat menyimpan ikon yang diekstrak ke file ICO (atau file CUR untuk kursor), atau menyalin gambar ikon tunggal ke clipboard.

Preferensi Kebijakan Grup Pintasan Baru properti sebenarnya tidak membiarkan Anda membuka file ikon (dengan asumsi itu dapat diakses oleh komputer tempat Anda mengkonfigurasi preferensi!):

\n\n
    \n
  1. Klik ikon di sebelah kanan Jalur file ikon kolom tulisan
    2. \n
  2. Buka file ikon target Anda
    3. \n
  3. Pilih ikon yang diinginkan
    4. \n
  4. Itu Indeks ikon nilai akan diisi dengan nomor indeks yang benar
    5. \n

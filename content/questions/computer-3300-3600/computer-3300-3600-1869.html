Saat saya masuk ke SSH saya, saya memiliki 2 sesi layar: 

screen -list\nThere are screens on:\n        4278.chat       (07/16/15 14:26:47)     (Attached)\n        15127.notifier  (07/16/15 13:27:57)     (Detached)\n2 Sockets\n

Saya mencoba melanjutkan layar obrolan: 

screen -r chat\nThere is a screen on:\n        4278.chat       (07/16/15 14:26:47)     (Attached)\nThere is no screen to be resumed matching chat.\n

Bagaimana cara melanjutkan layar 'obrolan'?

","source":"https://superuser.com/questions/942866/how-to-resume-a-screen-session","author":"yarek","replies":[],"answers":[{"answer":"

itu masih terpasang. Artinya Anda tidak dapat melampirkan kembali dengan -r. Coba -x sebagai gantinya. Ini seharusnya berhasil. Dengan -x, Anda memiliki banyak pihak yang dilampirkan ke layar yang sama

\n\n

Edit 1\nAnda juga dapat melepaskan layar dari jarak jauh dengan -D. Atau gabungkan dengan -D -r. Paksakan dengan -D -R. Atau bahkan lakukan segala kemungkinan dengan mengatakan -D -RR.

\n\n

-x hanya memungkinkan Anda untuk menggunakan layar ini bersama-sama. Saya menggunakan ini untuk jaringan rumah saya, ketika saya terlalu malas untuk beralih ke keyboard lain untuk mengetik sesuatu.

\n\n

Di sini, lihat manualnya. Ini menggambarkannya dengan cukup baik: layar pria

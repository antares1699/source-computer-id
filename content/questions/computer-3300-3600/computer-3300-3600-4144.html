Apple menggunakan font Lucida Grande sebagai font sistem standar dalam berbagai ukuran dan berat. Beberapa area sistem operasi seperti area teks yang dapat diedit menggunakan font lain, Helvetica secara default. Mac OS X menggunakan anti-aliasing font di seluruh sistem untuk membuat tepian tampak lebih halus.

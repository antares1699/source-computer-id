Dari Cara Mengekstrak File dan Folder dari Titik Pemulihan Sistem di Windows 7 dan Vista :

\n

Di Penjelajah Pemulihan Sistem, pilih titik pemulihan tertentu yang ingin Anda pasang, dan klik tombol Pasang.

\n

\n

Ini sekarang akan membuka titik pemulihan yang dipasang di jendela Windows Explorer di mana Anda sekarang dapat menelusuri dan menyalin file dan folder apa pun yang Anda suka.

\n


\n (sumber: sevenforums.com )

Masalahnya dapat disebabkan oleh beberapa tugas terjadwal yang membantu Anda\n hapus pintasan. Utilitas Pembersihan Desktop tidak ada di Windows 7.\n Ini dapat disebabkan oleh beberapa program pihak ketiga. Harap perhatikan secara khusus program pengoptimalan sistem apa pun.

\n \n

Metode lainnya adalah mengaktifkan audit objek untuk folder Desktop.

\n \n
    \n
  1. Klik Mulai, masukkan GPedit.msc di kotak Mulai Pencarian.
    2. \n
  2. Buka cabang berikut.
    3. \n
\n \n

Konfigurasi Komputer \\ Pengaturan Windows \\ Pengaturan Keamanan \\ Lokal\n Kebijakan \\ Keamanan Audit

\n \n
    \n
  1. Aktifkan kebijakan berikut:
    2. \n
\n \n

Peristiwa masuk akun audit Mengaudit akses objek

\n \n
    \n
  1. Buka Windows Explorer, buka folder C: \\ Users [username].
    2. \n
  2. Klik kanan folder "Desktop", pilih Properties-> Security. Klik tombol Advanced.
    3. \n
  3. Klik tab Auditing. Klik Lanjutkan.
    4. \n
  4. Klik Add. Kemudian klik Advanced.
    5. \n
  5. Klik tombol Temukan Sekarang.
    6. \n
  6. Tunggu prosesnya selesai. Kemudian dari daftar pengguna tambahkan pengguna berikut.
    7. \n
\n \n

ANONIM LOGON BATCH CREATOR PEMILIK Semua Orang Tamu LAYANAN LOKAL\n SISTEM LAYANAN JARINGAN JARINGAN

\n \n
    \n
  1. Setelah memilih setiap pengguna, pilih "Hapus" dan "Hapus subfolder dan file" sebagai entri audit.
    2. \n
\n \n

Jika Anda ingin memeriksa akun mana yang mencoba menghapus item di\n folder, buka Event Viewer, periksa Windows Logs \\ Security\n Log untuk informasi detail.

\n \n

Karena masalah ini terjadi setiap hari Senin, saya sarankan Anda mengaktifkannya\n kebijakan pada Minggu malam. Jika tidak, akan ada banyak acara.\n Jika Anda menemukan bahwa pintasan tersebut hilang, harap nonaktifkan pengauditan\n sekaligus dan periksa Log Peristiwa.

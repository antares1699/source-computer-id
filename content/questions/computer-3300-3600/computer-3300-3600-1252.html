Di Command Prompt Windows, saya dapat menggunakan file dir perintah untuk mengeluarkan nama semua file PNG dalam direktori: 

dir *.png /od /b > files.txt\n
\n
file 1.png\nfile 2.png\n...\n

Namun, file gambar yang saya kerjakan mengandung spasi dalam nama file mereka, dan karenanya tidak akan berfungsi dengan ImageMagick kecuali saya mengapit nama mereka dengan tanda kutip , seperti: 

"file 1.png"\n"file 2.png"\n...\n

Apa cara terbaik untuk masalah ini?

\n

Apakah ada perintah yang mencantumkan nama file yang dikelilingi oleh tanda kutip?

\n

Anda dapat menggunakan FOR /F untuk menjalankan perintah DIR dan mengapit output dengan tanda kutip:

\n

Ini dari prompt 

for /f "delims=" %A in ('dir /b /od *.png') do @echo "%A"\n

Dalam skrip batch, Anda akan menggandakan tanda persen, jadi %A menjadi %%A di kedua tempat.

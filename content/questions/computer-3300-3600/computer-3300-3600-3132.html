Saya mencoba untuk menghapus semua jung yang diketahui dari dir jendela mis. $ NTUninstall ..., file temp, dll dan pengaturan no pagefile. Tetapi saya masih mendapatkan sekitar 4GB penggunaan di c: setelah instalasi windows baru (jalur instalasi saya adalah winxp sp2 dari CD, kemudian WindowsUpdate ke SP3 / IE8). Apakah mungkin untuk turun lebih jauh? Saya yakin ada banyak file yang tidak dibutuhkan di jendela dir tetapi tidak yakin mana yang akan dihapus. Apakah ada yang pernah mencapai ukuran yang lebih kecil?

\n\n

Jika Anda ingin tahu alasannya, saya menjalankan WinXp SP3 di Mesin Virtual (saya menggunakan VirtualBox) dan perlu membuat gambar OS sekecil mungkin untuk disimpan di flash drive USB.

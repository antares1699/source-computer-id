Saya menggunakan Chromium pada flash drive melalui LiberKey dan melakukan semua yang saya ingin lakukan sejauh menyinkronkan bookmark Chrome saya dari rumah, dll. Tapi sangat lambat. Jika saya membuka lebih dari 1 tab, atau jika tab berat dengan kode seperti Gmail atau GMaps, maka seluruh program hang selama sekitar 5-10 detik.

\n\n

Saya tidak tahu persis bagaimana aplikasi portabel bekerja saat dijalankan dari flash drive, jadi apakah ada cara untuk mempercepat waktu muat?

\n\n

Komputer yang saya gunakan di tempat kerja memiliki CPU P4 3GHz dan RAM 2GB. Flash drive adalah SanDisk 2GB (mungkin berumur 2-3 tahun).

\n\n

Catatan Kinerja: Google Chrome Portable mungkin berjalan lambat dari beberapa flash drive. Anda dapat mempercepatnya dengan menyalin GoogleChromePortable.ini dari direktori GoogleChromePortable \\ Other \\ Source ke direktori GoogleChromePortable dan mengeditnya untuk menyetel RunLocally = true. Jika Anda melakukannya, pastikan untuk memberikan waktu Google Chrome Portable untuk menyalin kembali profil Anda setelah mematikannya. Tentu saja, terdapat implikasi privasi untuk menyalin data pribadi Anda secara lokal ke PC yang tidak Anda kendalikan.

Windows 8 hadir dalam beberapa versi ... dengan dua di antaranya

\n\n
    \n
  • Windows 8
    • \n
  • Windows 8 Pro
    • \n
\n\n

Memeriksa grafik perbandingan di wiki , fitur pro memiliki ini (di mana non-pro tidak) ..

\n\n

Bisakah bergabung dengan domain Windows? Windows 8: tidak . Windows 8 Pro: ya.

\n\n

Kebijakan Grup? Windows 8: tidak . Windows 8 Pro: ya.

\n\n

Saat ini, ketika saya Vpn dari mesin Windows 7 saya berfungsi, saya perlu menyediakan file domain / nama pengguna dan kata sandi .

\n\n

Apakah ini berarti saya tidak dapat melakukan ini dengan Windows 8 dan saya perlu membeli edisi Windows 8 Pro?

Tidak. Ini akan bekerja dari Windows 8 juga tanpa versi Pro karena Anda memberikan kredensial domain ke koneksi VPN untuk tidak masuk ke mesin Anda.

\n\n

Kedua fitur tersebut berarti bahwa mesin Anda dapat menjadi bagian dari domain Windows di jaringan Anda sendiri dan dikelola melalui kebijakan grup.

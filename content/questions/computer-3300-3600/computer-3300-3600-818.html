Di mana Anda membutuhkan jalannya? Biasanya Anda cukup menyeret & menjatuhkan file atau folder ke antarmuka di mana Anda memerlukan jalurnya. Ini berfungsi di aplikasi Terminal misalnya.

Ini pertanyaan lama dan sekarang ada cara lain untuk menyalin nama jalur yang mungkin lebih mudah daripada yang disarankan (kecuali Terminal adalah tujuan yang dimaksudkan untuk nama jalur dalam hal ini menyeret file ke Terminal lebih mudah). Itu telah ada setidaknya sejak Yosemite dan mungkin sebelumnya.

\n\n

Untuk menyalin nama jalur file, pilih file di Finder, tekan dan tahan Pilihan kunci dan buka Sunting menu di mana Anda harus melihat bahwa biasa Salin <filename> pemilihan menu sekarang harus ditampilkan sebagai Salin <filename> sebagai Pathname .

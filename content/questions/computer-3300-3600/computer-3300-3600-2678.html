Anda dapat menggunakan AppleScript berikut: 

\ntell application \"Firefox\"\n    activate\n    set newTabURLs to takeSafariTabURLs() of me\n    repeat with tabURL in newTabURLs\n        open location tabURL\n        delay 1\n    end repeat\n\nend tell\n\non takeSafariTabURLs()\n    set tabURLs to {}\n    tell application \"Safari\"\n        repeat with w in windows\n            if name of w is not \"\" then --in case of zombie windows\n                repeat with t in tabs of w\n                    set tabURL to URL of t\n                    set the end of tabURLs to tabURL\n\n                end repeat\n            end if\n        end repeat\n        return tabURLs\n    end tell\nend takeSafariTabURLs\n

Perubahan kecil untuk mempertahankan jendela. Saya memasukkan inti dari ini juga ... jika orang lain ingin memperbaikinya. Lihat https://gist.github.com/amanuel/81e70673b057687e904a248218c50ce2 

    tell application \"Firefox\"\n    activate\n    set safariWindows to getSafariWindows() of me\n    repeat with w in safariWindows\n        set newTabURLs to takeSafariTabURLs(w) of me\n        repeat with tabURL in newTabURLs\n            open location tabURL\n            delay 0.5\n        end repeat\n        tell application \"System Events\" to keystroke \"n\" using command down\n        delay 1\n    end repeat\nend tell\n\non getSafariWindows()\n    set safariWindows to {}\n    tell application \"Safari\"\n        repeat with w in windows\n            if name of w is not \"\" then --in case of zombie windows\n                set the end of safariWindows to w\n            end if\n        end repeat\n        return safariWindows\n    end tell\nend getSafariWindows\n\non takeSafariTabURLs(w)\n    set tabURLs to {}\n    tell application \"Safari\"\n        repeat with t in tabs of w\n            set tabURL to URL of t\n            set the end of tabURLs to tabURL\n        end repeat\n        return tabURLs\n    end tell\nend takeSafariTabURLs\n

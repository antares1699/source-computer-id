Saya menjalankan Kubuntu 12.10. Saya memiliki 3 monitor yang dicolokkan ke AMD Radeon 7970. Saya telah mencoba mengganti urutan output, dan mengkonfigurasinya berulang kali dengan amdcccle tetapi kubuntu sepertinya selalu membuat monitor terjauh di sebelah kiri sebagai monitor utama. Solusinya sejauh ini adalah melakukan hal berikut. 

xrandr --query\n

Yang mengembalikan daftar monitor saya. Kemudian secara eksplisit mengatur primer dengan xrandr dengan 

xrandr --output <monitor identifier> --primary\n

Saya menambahkan perintah ini ke skrip start up, tetapi masih tampak hacky. Sistem masih melakukan booting dengan monitor paling kiri sebagai yang utama, dan kemudian beralih ke monitor tengah saya setelah penyalaan. Saya menemukan bahwa mengedit xorg.conf bekerja paling baik untuk saya (saya menggunakan KDE).

\n\n

Saya menambahkan Opsi "Utama" "benar" di bawah monitor yang saya inginkan sebagai yang utama dan sekarang hampir semuanya bekerja seperti yang diharapkan.

Jika saya tidak salah xrandr hanya mengubah pengaturan dengan cepat, jadi jika Anda ingin membuatnya secara permanen, Anda harus menyimpannya menggunakan metode lain.

\n\n

Cara termudah di Gnome adalah mengedit ~/.config/monitors.xml. Di sana Anda dapat mengatur monitor mana yang utama. Contoh: 

<monitors version=\"1\">\n  <configuration>\n      <clone>no</clone>\n      <output name=\"LVDS1\">\n          <vendor>AUO</vendor>\n          /* (...) */\n          <primary>yes</primary>\n      </output>\n      <output name=\"VGA1\">\n      </output>\n      <output name=\"HDMI1\">\n          <vendor>SAM</vendor>\n          /* (...) */\n          <primary>no</primary>\n      </output>\n      <output name=\"DP1\">\n      </output>\n  </configuration>\n</monitors>\n

Ini berfungsi di Ubuntu (Gnome / Unity), tetapi saya belum pernah menggunakan Kubuntu sehingga mungkin menggunakan lokasi yang berbeda, atau bahkan menggunakan file lain. Selain itu, ini hanya akan diterapkan setelah Anda masuk.

\n\n

Jika itu tidak berhasil, Anda harus mengedit xorg.conf. Saya tidak dapat banyak membantu Anda dalam hal itu, tetapi dalam pencarian cepat saya menemukan ini posting yang akan membantu. Lihat juga ini Halaman Wiki Ubuntu.

\n\n

Semoga ini membantu.

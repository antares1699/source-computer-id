Saya hanya mencoba membuat daftar file di direktori saat ini dengan pola tertentu, tetapi tidak berfungsi dengan file tersembunyi. Bagaimana saya bisa mencocokkan semua file?

\n\n

Saya mencoba 

ls *foo*\n

dan 

ls -a *foo*\n

Tidak menemukan apa pun di Google, kata kunci yang terpikir untuk mencari ini mengarah ke banyak informasi yang tidak terkait ...

","source":"https://superuser.com/questions/865962/how-to-include-hidden-files-in-ls-pattern-match","author":"Ixxzz","replies":[],"answers":[{"answer":"

Aktifkan globbing untuk file tersembunyi: 

shopt -s dotglob\nls *foo*\n

Nonaktifkan: 

shopt -u dotglob\n

Tampilkan status globbing: 

shopt dotglob\n

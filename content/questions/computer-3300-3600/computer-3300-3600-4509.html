Saya masih bermain-main dengan ini karena sepertinya hal yang cukup masuk akal untuk diharapkan dapat dilakukan ... tetapi sejauh ini satu-satunya cara yang saya temukan adalah dengan curang yang agak jahat memasukkan bidang nama file dengan jalur dan kemudian menerapkan warna font putih ke teks (yaitu nama file itu sendiri) dan Anda berakhir dengan hanya jalur dan blok teks tak terlihat setelahnya.

\n\n

Masalahnya adalah hal ini dapat mengacaukan pemformatan dan sangat rumit.

\n\n

- = EDIT = -

\n\n

Solusi yang sedikit kurang kotor ...

\n\n

Jika Anda tidak keberatan memiliki makro di normal.dotm Anda dan memiliki variabel tersembunyi di dokumen Anda, maka Anda mungkin dapat menggunakan makro ini dan mengatur tombol pada akses cepat Anda untuk menjalankannya ... Saya sudah mencobanya dan berfungsi pada dokumen bebas makro sehingga dokumen itu sendiri tidak memerlukan makro untuk menggunakan metode ini, saya akan meletakkan ini di template normal.dotm saya ... 

Sub updatePath()\n'\n' updatePath Macro\n'\n'\nDim myPath As String\nmyPath = ActiveDocument.Path\nIf myPath = \"\" Then\n    'do nothing as the document has no path... needs to be saved first\nElse\n    If ActiveDocument.Variables.Count = 0 Then\n        ActiveDocument.Variables.Add Name:=\"myPath\", Value:=myPath\n    Else\n        i = 1\n        Do While i < (ActiveDocument.Variables.Count + 1)\n            If ActiveDocument.Variables.Item(i).Name = \"myPath\" Then\n                ActiveDocument.Variables.Item(i).Value = myPath\n            End If\n            i = i + 1\n        Loop\n    End If\nEnd If\n\nEnd Sub\n

Dan kemudian tambahkan kode bidang

\n\n

DOCVARIABLE myPath

\n\n

yang ketika diperbarui setelah menjalankan makro di atas akan melakukan apa yang Anda inginkan. Memang itu adalah dua atau tiga klik dan bukan hanya pembaruan sederhana, tetapi itu berarti bahwa dokumen yang diekspor mendapatkan jenis bidang yang tepat dan hanya orang dengan makro ini yang dapat mengatakan di mana dokumen tersebut. Sebaiknya disimpan :)

Sayangnya, tidak ada cara untuk melakukan ini dengan bidang FILENAME tanpa makro, bidang FILENAME tidak menawarkan opsi untuk menghilangkan nama file.

\n\n

Bagaimanapun, berikut adalah makro untuk menyisipkan jalur file, tanpa nama file, bagi mereka yang mungkin membutuhkannya. 

\nSub InsertPath()\n  Dim sPath As String\n\n  sPath = ActiveDocument.Path\n  If sPath = \"\" Then\n    MsgBox \"You need to save the document before running this macro.\", _\n           vbOKOnly, \"Document not saved\"\n  Else\n    sPath = sPath & Application.PathSeparator\n    Selection.TypeText(sPath)\n  EndIf\nEndSub\n

Hmm. Ini tidak benar (atau saya salah paham apa yang Anda maksud). Biasanya program diinstal menggunakan Sistem manajemen paket Ubuntu (paket deb), dan tersedia di seluruh sistem untuk semua pengguna. Sejauh ini, ini adalah metode penginstalan perangkat lunak yang disukai.

\n\n

Saat (pada kesempatan yang jarang) memasang sesuatu di luar sistem manajemen paket , Saya cenderung menggunakan /usr/local/programs, dan membuat subfolder khusus di sana untuk program yang berbeda. Seringkali, saya juga akan meletakkan symlink (atau skrip startup sederhana) di bawah /usr/local/bin untuk mempermudah peluncuran aplikasi ini. Tentu saja, /opt atau lokasi lain juga dapat digunakan, selama Anda memastikan semua pengguna dapat mengakses file dan direktori ( seperti yang ditunjukkan Bobby ).

\n\n

Jika Anda ingin program semacam itu muncul di menu mulai lingkungan desktop Anda, Anda harus mengurusnya sendiri. Misalnya. ini menunjukkan cara menambahkan entri (untuk satu pengguna?), dengan asumsi Anda menggunakan Gnome. Saya yakin ada cara untuk menambahkannya semua pengguna , tapi di luar kepalaku, aku tidak yakin bagaimana; mungkin orang lain dapat membantu Anda di sana.

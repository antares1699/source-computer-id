 VARIABLE=\"$(cat /sys/class/net/$IFACE/address)\"\n

Sulit untuk melakukannya dengan lebih sederhana.

\n\n

Sebenarnya: 

 read -r MACADDRESS < /sys/class/net/$iFACE/address\n

bahkan lebih sederhana, dan tidak memulai proses tambahan.

Ini berhasil untuk saya. Mungkin bisa sedikit dioptimalkan: 

/sbin/ifconfig eth0 | grep -o 'HWaddr  *[0-9A-F:]*' | sed -s 's/HWaddr *//'\n

Saya tidak berpikir Anda dapat menyederhanakannya banyak dari ini. Saya melakukan strace di ifconfig, dan tampaknya Anda menarik MAC addr dengan ioctl () pada soket, bukan dari file / proc.

Di Windows Task Manager, di bawah tab Performance, Page File Usage History ditampilkan dalam panel kontinu, sedangkan Riwayat Penggunaan CPU dibagi menjadi 8 panel sempit, yang membuatnya terlihat seperti grafik hanya "dilapisi" dengan pemisah panel .

\n\n

Kenapa ini? Mengapa tidak menampilkan grafik dalam panel kontinu?

","source":"https://superuser.com/questions/165704/why-is-the-cpu-usage-history-pane-split-in-windows-task-manager-performance","author":"Tomas Aschan","replies":[{"reply":"Anda juga dapat mengubahnya kembali, dengan View => CPU History => One Graph, All CPUs.","author":"Hello71"}],"answers":[{"answer":"

Masing-masing untuk satu inti prosesor. Kecuali jika itu adalah chip Intel dengan Hyperthreading (HT), dalam hal ini ada dua panel per inti, satu untuk setiap inti dan satu untuk HT-nya. Misalnya, chip quad-core dengan HT akan menampilkan delapan panel, bukan empat.

Anda mungkin menggunakan prosesor quad-core dengan hyperthreading, yang membuatnya terlihat seperti prosesor 8-core di Task Manager. Pengelola Tugas secara default menampilkan setiap inti sebagai grafik terpisah. Jika Anda lebih suka grafik tunggal, pilih Lihat -> Riwayat CPU -> Satu Grafik, Semua CPU opsi menu.

Sebagian besar CPU desktop dari era P4 dan seterusnya akan menampilkan setidaknya 2 grafik, apakah intinya fisik (CPU multi-core seperti Core2 Duo), logis (hyper-threaded seperti di P4) atau keduanya (Core i7).

\n\n

Seperti yang dikatakan coneslayer, kemungkinan besar Anda memiliki CPU quad-core dengan hyper-threading, seperti Core i7.

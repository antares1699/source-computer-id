# pear search http\nThe value of config option cache_dir (/tmp/pear/cache) is not a directory and attempts to \ncreate the directory failed.

Ketika saya membuat direktori dengan tangan: 

mkdir -p /tmp/pear/cache
kesalahan "hilang" dan saya dapat menginstal: "pecl install mongo".

Saya baru saja menginstal vim7.3 menggunakan "--with-features=big", karena saya menyukai hampir semua fitur tersebut. Namun, satu hal yang saya tidak suka adalah mengaktifkan penyelesaian-tab: dalam mode normal, jika Anda menekan tab, vim "secara cerdas" memulai penyelesaian-omni jika Anda berada di dekat "kata".

\n\n

Sekarang, saya menggunakan Plugin Tab Cerdas , dan saya perlu tombol tab saya agar tidak dilengkapi secara otomatis. Namun, saya menyukai semua fitur "--with-features=big" lainnya, dan saya tidak suka membuka "--with-features=normal", yang tidak memengaruhi tombol tab saya.

\n\n

Adakah yang tahu cara menonaktifkan penyelesaian-tab saat mengkompilasi vim --with-features = huge?

Dalam mode normal, <tab> melompat ke entri berikutnya yang lebih baru dalam daftar lompat. Perilaku ini tidak dipengaruhi oleh fitur Vim apa pun. Dalam mode penyisipan, <tab> menyisipkan karakter <tab>. Jika Anda melihat perilaku yang berbeda, ini disebabkan oleh ~ / .vimrc atau plugin Anda. Untuk mengetahui di mana <tab> sedang dipetakan, jalankan 

:verbose nmap <tab>\n

dan Anda bisa mendapatkan petunjuk untuk menangani masalah tersebut.

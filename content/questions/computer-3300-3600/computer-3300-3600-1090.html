Saya memiliki MacBook Pro dengan OS X 10.6.7 dan saya memiliki mesin Windows XP. Saya ingin memulai ulang mesin Windows melalui baris perintah OS X. Misalnya di Windows kita bisa restart dengan perintah ini:

\n\n

shutdown -m \\\\IP -r -t 00

\n\n

Ini akan segera dimulai ulang.

\n\n

Bagaimana saya bisa melakukan ini dengan Mac?

Anda bisa menggunakan Alat Windows RDP untuk Mac dan RDP ke mesin Windows XP Anda dan jalankan perintah, 

shutdown -r -t 0\n

Untuk me-restart sistem tanpa RDP maka Anda dapat menginstal Server SSH (Ini utas forum mungkin menarik, tetapi tautan di bawah ini menyertakan laman ke binari OpenSSH untuk Windows.) ke Mesin Windows Anda dan kemudian cukup mengakses mesin Windows seperti: 

ssh User@192.168.0.5\n

Akan membawa Anda ke prompt perintah Windows, dan saya yakin Anda bisa menjalankan Perintah Shutdown melalui SSH.

\n\n

Inilah situs web yang saya temukan, mungkin Anda bisa memahaminya: http://www.tonyspencer.com/2005/12/03/restart-windows-remotely-through-vpn/

Anda dapat menggunakan debug untuk menulis file COM pendek. Ketika Anda menjalankan debug Anda akan mendapatkan satu prompt tanda hubung, masukkan yang berikut ini (termasuk baris kosong) dan Anda akan mendapatkan file color.com di direktori kerja Anda saat ini: 

a 100\nmov ah, 06\nxor al, al\nxor cx, cx\nmov dx, 184f\nmov bh, 07\nint 10\nmov ah, 4c\nxor al, al\nint 21\n\nrcx\ne\nncolor.com\nw\nq\n

Ia memanggil int 10 ah=06 untuk membersihkan layar, mengatur kursor di bagian bawah dan mengisi dengan atribut di bh. Gigitan tinggi adalah latar belakang, rendah latar depan, warna:

\n\n
    \n
  • 0 = hitam
    • \n
  • 1 = biru
    • \n
  • 2 = hijau
    • \n
  • 3 = cyan
    • \n
  • 4 = merah
    • \n
  • 5 = ungu
    • \n
  • 6 = kuning
    • \n
  • 7 = putih
    • \n
  • 8 = hitam muda
    • \n
  • 9 = biru muda
    • \n
  • A = hijau muda
    • \n
  • B = sian muda
    • \n
  • C = merah muda
    • \n
  • D = ungu muda
    • \n
  • E = kuning muda
    • \n
  • F = putih muda
    • \n
\n\n

(Jadi bit 4 adalah intensitas tinggi.) Untuk mendapatkan latar belakang merah dengan latar depan hitam, ubah 07 menjadi 40.

\n\n

Mengurai argumen baris perintah untuk mengatur warna bukanlah hal yang sepele, jadi cukup gunakan kode favorit Anda. Atau lakukan ini di debug untuk memperbaruinya: 

debug color.com\ne 10a\n40\nw\nq\n

Atribut disimpan di lokasi 010A, memperbaruinya secara langsung lebih mudah daripada mengetik semuanya lagi.

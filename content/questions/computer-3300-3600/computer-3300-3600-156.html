Oke, cara melakukannya dengan drive yang dapat dilepas (tidak yakin apakah ini berfungsi, misalnya hard drive Anda), adalah membuat file autorun.inf di root drive dengan konten ini (misalnya di E :) 

[autorun]\nicon=drive.ico\n

di mana drive.ico adalah nama file dari ikon yang Anda inginkan untuk drive Anda. Perhatikan bahwa itu mungkin harus berada di drive yang sama dengan file autorun. Anda mungkin ingin menyembunyikan file ikon - ini tidak akan memengaruhi tampilan ikon dengan cara apa pun.

Ada sebuah artikel sini tentang cara melakukan ini. Langkah-langkah di bawah ini:

\n\n

Untuk mengubah ikon drive:

\n\n
    \n
  1. Klik Mulai | Jalankan | Ketik regedit dan klik OK
    2. \n
  2. Arahkan ke: HKEY_LOCAL_MACHINE | PERANGKAT LUNAK | Microsoft | Windows | CurrentVersion | Penjelajah
    3. \n
  3. Buat kunci baru bernama DriveIcons
    4. \n
  4. Di bawah DriveIcons buat kunci baru dengan huruf drive (misalnya C)
    5. \n
  5. Di dalam kunci huruf drive ini buat kunci dan nama baru DefaultIcon.
    6. \n
  6. Berikan nilai default, ikon yang ingin Anda gunakan mis. c: \\ windows \\ system32 / shell32.dll, 74
    7. \n

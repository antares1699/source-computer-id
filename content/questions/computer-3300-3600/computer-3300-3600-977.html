Saya memiliki laptop dengan Aptio BIOS dan OEM Windows 8. Hard disk drive saya rusak, tetapi garansi telah kedaluwarsa. Jadi saya membeli hard disk drive baru, yang sudah terpasang di dalamnya dan sekarang saya mencoba menginstal Windows 8 dari disk setup Windows 8 asli. Setiap kali saya menyalakan laptop, pengaturan BIOS muncul. Sekarang saya mencapai menu boot, tetapi satu-satunya opsi boot adalah: 

UEFI: IP4 Realtek Ethernet Controller\n

Bagaimana saya bisa boot dari disk setup?

\n\n

Setelah mengaktifkan Modul Dukungan Kompatibilitas (CSM), penyiapan masih muncul, dan saya masih tidak dapat mem-boot dari CD.

Setelah melakukan beberapa penelitian dan mencoba mencari tahu, saya menemukan sebuah artikel online yang menyebutkan tentang holding ESC saat menggunakan tombol daya untuk menyalakan komputer. Setelah pengujian lebih lanjut saya menemukan bahwa saya perlu menonaktifkan opsi Boot Aman di BIOS atau saya mendapat kesalahan keamanan ketika saya mencoba untuk boot ke perangkat USB. Saya mengubahnya dengan membuka menu Keamanan dan menonaktifkan opsi "Kontrol Boot Aman".

\n\n

Saya juga memiliki pengaturan bernama "Launch CSM" di bawah menu boot yang harus saya aktifkan (tidak mengaktifkan "Launch PXE OpROM"). Dengan opsi boot aman dinonaktifkan dan meluncurkan CSM diaktifkan, saya bisa menahannya ESC sambil menyalakan laptop dan boot ke perangkat USB saya. Melakukan ini memberi saya menu saat boot di mana saya dapat memilih apa yang akan saya boot. Saya memilih opsi: "UEFI: General USB Flash Disk 1.0" (Saya telah membuat media penyelamatan Macrium Reflect dan selama penyiapan mencentang kotak "Aktifkan dukungan multiboot MBR / UEFI USB", jadi saya tidak yakin apakah Anda masih ingin pilih ini jika tidak).

\n\n

Saya sedang mengerjakan laptop ASUS (menurut saya SonicMaster) menggunakan Aptio Setup Utility - Copyright (C) 2012 American Megatrends, Inc. Jika Anda menggunakan BIOS yang berbeda, langkah-langkah untuk ini mungkin tidak sama, tapi semoga ada cukup informasi di sini agar Anda dapat membuat ini berfungsi di komputer Anda.

\n\n

Saya juga memperhatikan bahwa setelah membuat ini bekerja menggunakan metode di atas, saya mendapat beberapa opsi tambahan di BIOS yang sebelumnya tidak ada. Ini termasuk opsi boot tambahan (menu boot) dan menu opsi override boot tambahan (simpan & keluar). Opsi boot override baru "UEFI: General USB Flash Disk 1.0" bekerja dengan baik setelah langkah-langkah di atas.

\n\n

Sebelum saya menemukan opsi ini, saya juga telah melihat secara online di mana orang-orang menyebutkan holding Bergeser sambil mengklik tombol restart di bawah Windows 8 untuk mendapatkan opsi menu untuk boot ke perangkat. [ https://askleo.com/how_do_i_boot_from_cddvdusb_in_windows_8/] . Ini tidak berhasil untuk saya, tetapi Anda mungkin ingin mencoba ini terlebih dahulu untuk melihat apakah berhasil untuk Anda.

Saya mencoba mendapatkan pandangan untuk meneruskan email secara otomatis ke akun pribadi saya, membiarkannya di kotak masuk, tetapi pada saat yang sama tidak mengisi kotak terkirim saya. Saya tidak dapat menemukan cara untuk meneruskannya tetapi tidak menempelkan pesan terkirim di pesan terkirim saya. Saya lelah membuat dua aturan, satu untuk meneruskan dan satu lagi menghapus, tetapi aturan untuk diterapkan pada pesan terkirim tidak menyertakan opsi hapus.

Di Outlook 07 Anda dapat menentukan baik menghapus dan meneruskan aturan seperti yang ditunjukkan pada gambar di bawah ini. Tetapkan aturan sebagai "Periksa pesan saat mereka tiba" untuk tujuan pengujian, saya memilih "hanya untuk saya" lalu mencentang terus ke Gmail saya dan opsi hapus. Saya kemudian mengirim dan mengirim email langsung ke saya dari gmail saya dan meneruskan pesan kembali ke Gmail saya dan menghapusnya, saya juga tidak melihat apa pun di folder terkirim. Jadi ini akan bekerja jika Anda memiliki 07, sekali lagi akan membantu jika Anda menentukan versi.

inilah kode warna pinout standar untuk konektor 1.x / 2.0 (dipinjam dari Wikipedia) 

Pin    Name\n\n
1  VBUS    Red +5 V
\n\n
2  D−  White(Gold *)   Data −
\n\n
3  D+  Green   Data +
\n\n4  GND   Black(blue*)    Ground

.- D + D- +

\n\n

.-.-.-.-.-.-.-.

\n\n

| ======== |

\n\n

4 3 2 1

\n\n

NB ..tapi Anda akan kehilangan jalur pin USB ke-5 dari kabel pengisi daya asus untuk memberitahukannya untuk beralih ke pengisian daya 15V, bukan 5V. Meskipun perbaikan tampaknya dapat dilakukan dengan membumikan kabel ke-5 (pin7) dari konektor Asus USB 3 menggunakan pengganti USB 2.0 (dijembatani / disolder ke pin 4 seperti di atas), Anda sebaiknya menggunakan konektor USB 3 / kabel jika Anda memilikinya.

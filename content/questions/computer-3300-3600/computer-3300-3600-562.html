TortoiseSVN's Unshelve tidak menghapus (menjatuhkan) rak untuk saya, terlepas dari apakah konflik hasil unshelving atau tidak. (Saya menggunakan TortoiseSVN 1.11.1, Build 28492)

\n\n

Solusi: hapus / jatuhkan rak menggunakan SVN secara langsung, bukan melalui TortoiseSVN. Di konsol: 

$ svn help x-shelf-drop\nx-shelf-drop: Delete a shelf.\nusage: x-shelf-drop SHELF [PATH ...]\n\n  Delete the shelves named SHELF from the working copies containing PATH\n  (default PATH is '.')\n\nâ€¦\n\n$ svn x-shelf-drop \"Name of shelf\"\ndeleted 'Name of shelf'\n

Dokumen: https://cwiki.apache.org/confluence/display/SVN/Shelving+in+Svn-trunk

Sepertinya Anda menginginkan: 

set -xg PATH /opt/local/bin /opt/local/sbin $PATH\n

Edit: Saya mengerti maksud Anda, sintaks ini membutuhkan waktu untuk membiasakan diri. Ia menginginkan spasi, bukan titik dua, dan tidak boleh dalam tanda kutip, dan semua jalur di PATH Anda saat ini harus benar-benar ada (dalam kasus saya, saya memiliki /usr/local/sbin di PATH saya meskipun sebenarnya tidak ada, dan itu masalah untuk fish).

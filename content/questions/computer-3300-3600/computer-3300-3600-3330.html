saya harus menyimpan sekitar 1000 file csv yang berbeda dan setiap kali saya mengklik file save as, file tersebut masuk ke beberapa direktori default.

\n\n

apakah ada cara default untuk mengingat direktori tempat saya menyimpan file terakhir?

","source":"https://superuser.com/questions/64263/excel-why-doesnt-file-save-as-remember-the-last-directory-that-you-saved-a","author":"leora","replies":[],"answers":[{"answer":"

Anda dapat mencoba menambahkan pengaturan registri untuk Excel.

\n\n
    \n
  • mulai regedit
    • \n
  • buka HKEY_CURRENT_USER\\Software\\Microsoft\\Office\\12.0\\Excel\\Options (12.0 untuk Excel 2007, jika Anda memiliki versi Excel yang lebih lama akan berbeda, 11.0 untuk 2003, 10.0 untuk 2002 ...)
    • \n
  • tambahkan nilai dword bernama AlwaysSaveToCurrentPath dan setel ke 0
    • \n
\n\n

Ini akan membuat "Save As" menggunakan folder yang sama di mana file dibuka. Jika Anda menyetel nilainya ke 1, ini akan menggunakan folder Default saat menggunakan "Save As"

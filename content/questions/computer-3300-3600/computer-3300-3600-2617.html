Saya mendapatkan versi komunitas Visual Studio 2013 dan saya pikir c ++ 11 seharusnya diaktifkan secara default.

\n\n

Namun, makro __cplusplus memiliki nilai 199711L (c ++ 98) daripada 201103L (c ++ 11). Selanjutnya, jika saya mencoba menjalankan kode spesifik c ++ 11 (likeconstexpr int a = 5;) itu memberikan kesalahan pada kompiler. Bagaimana cara mengaktifkan c ++ 11?

","source":"https://superuser.com/questions/940717/enabling-c11-on-visual-studio-2013","author":"user2950265","replies":[],"answers":[{"answer":"

Bahwa nilai __cplusplus masih 199711L adalah bug, yang telah dilaporkan ke Microsoft .

\n\n

Itu seharusnya tidak menghentikan Anda dari menulis dan mengompilasi kode C ++ 11. Seharusnya hanya berfungsi (tidak ada yang khusus untuk dikonfigurasi).

\n\n

Awas bahwa implementasi Visual Studio 2013 C ++ 11 adalah belum lengkap . Di antara fitur yang tidak didukung adalah constexpr, yang menjelaskan mengapa Anda mendapatkan kesalahan kompilasi ;-)

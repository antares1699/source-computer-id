Saya sedang dalam proses mendesain komputer khusus untuk bekerja. Saya telah condong ke arah membangun sistem dengan 4 prosesor fisik di dalamnya, Namun tidak ada versi windows kecuali rilis server yang mendukung lebih dari 2 prosesor fisik.

\n\n

Katakanlah saya membangun rig dengan 4x Hex-core Intel Xeons, 512 GB RAM, dan menginstal Windows Server 2012 ke sistem. Bisakah saya menggunakannya seperti saya menggunakan Windows 8 biasa? Saya tidak akan menggunakan sistem ini sebagai server, melainkan workstation Pengeditan Grafik dan Video- Saya akan menjalankan perangkat lunak seperti Cinema4D, Photoshop, Davinci Resolve, Premiere, VRay, dll. Akankah perangkat lunak tersebut dapat beroperasi dengan baik pada Server OS dengan banyak prosesor fisik ini?

","source":"https://superuser.com/questions/792608/using-windows-server-2012-like-a-regular-operating-system","author":"Ben Franchuk","replies":[],"answers":[{"answer":"

Tanpa mengetahui secara spesifik, jawaban saya adalah Iya , Server 2012 pada dasarnya adalah Windows 8 dengan bel dan peluit masih terpasang dan tidak berpakaian seret, tidak ada yang menghentikan Anda menggunakannya sebagai workstation biasa.

\n\n

Perlu diingat bahwa beberapa program mungkin tidak tahu apa yang harus dilakukan dengan semua inti tersebut (mungkin hanya menggunakan 4/8 dll.), Beberapa program juga berperilaku aneh di komputer dengan lebih dari 20 inti logis (IE. Jangan instal / jangan lari), jadi teliti dulu untuk apa Anda akan menggunakannya.

\n\n

Program apa pun yang kompatibel dengan Windows 8 Sebaiknya instal di Server 2012 tanpa masalah.

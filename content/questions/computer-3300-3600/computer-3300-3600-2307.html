Mencoba https://chocolatey.org . Ini dirancang sebagai "Manajer Paket Mesin, agak seperti apt-get, tetapi dibangun dengan mempertimbangkan Windows." Dan mari Anda berdua mengunduh dan menginstal paket dalam satu perintah.

\n\n

Misalnya, untuk menginstal Notepad ++, cukup ketik 

choco install notepadplusplus\n

Dan pengunduhan dan penginstalan dilakukan oleh PowerShell untuk Anda. Ini membuat peningkatan menjadi mulus juga karena database paket yang dilacaknya.

\n\n

Alat ini sejauh ini paling mudah untuk mengotomatiskan penginstalan sebagian besar alat sumber terbuka atau gratis seperti firefox. Namun, dalam lingkungan perusahaan, saat menyiapkan multi-mesin mungkin berguna untuk digunakan Layanan Penerapan Windows , peran gratis untuk Windows Server. WDS memungkinkan Anda menyelesaikan penginstalan zero-touch dari boot jaringan tftp, melalui partisi drive, enkripsi BitLocker, penginstalan OS dan driver, melalui penginstalan perangkat lunak Microsoft tertentu seperti Office dan menggabungkan komputer ke domain.

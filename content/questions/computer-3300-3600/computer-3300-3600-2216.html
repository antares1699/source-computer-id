Saya tidak memilikinya dari sumber bersertifikat - tapi saya pikir itu Alt + Numpad hanya berfungsi untuk nilai dalam kisaran 0..255. Jadi, semua yang Anda coba dengan nilai yang lebih tinggi 'dilemparkan' ke kisaran ini.
Jadi misal: 336-256 = 80.\nKarena itu -> Alt + 336 sama dengan Alt + 80 . Yaitu 'P'.

\n\n

Edit: Itu Alt + Numpad metode ini hanya berfungsi pada halaman kode aktif Anda, jadi metode ini sebenarnya mengetik karakter dalam pengkodean lokal Anda (mis. ISO 8859-16 untuk Hongaria).\nJadi pada dasarnya keying Alt + 193 akan mengetik karakter yang panjangnya hanya 1 byte dan memiliki nilai 0xC3 (193 dalam HEX). Secara kebetulan, 193 dalam ISO 8859-16 memiliki nilai tampilan yang sama dengan \\u0193 adalah peta Unicode. Di halaman kode lokal saya Alt + 193 menghasilkan karakter ┴ yaitu \\u2534 di peta unicode.

\n\n

Rekomendasi saya, jika Anda perlu menampilkan nilai-nilai ini di beberapa aplikasi, gunakan konvensi unicode alih-alih memasukkannya sebagai Alt + Numpad kombinasi. Kamu bisa mengunduh BableMap untuk ini - ini adalah alat yang hebat untuk tujuan ini dan gratis.

Untuk menyalin instalasi OS X, file dan semuanya, Anda cukup melakukan backup dengan Time Machine, reboot dari DVD instalasi menggunakan opsi "Restore System from Backup", dan kemudian tunggu. Proses ini sepenuhnya otomatis.

\n\n

Jika Anda ingin menginstal ulang hanya Sistem, tanpa menyentuh data Pengguna, gunakan Time Machine lagi: pulihkan semuanya dari tertua salinan disk Home yang Anda miliki, yang memiliki semua utilitas yang Anda inginkan. Setelah selesai, gunakan Time Machine untuk memulihkan file terbaru Pencadangan folder pengguna.

\n\n

(Saya tidak begitu yakin mengapa Anda hanya ingin menginstal ulang Sistem. Seperti kebanyakan sistem UNIX, OS X tidak benar-benar menurun dari waktu ke waktu. Saya rasa ini mungkin berguna jika Anda benar-benar tidak beruntung dan terkena serangan MalWare.)

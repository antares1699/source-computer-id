Ketika saya mengetikkan kata sandi saya pada prompt baris perintah sudo, ini menunjukkan * untuk setiap penekanan tombol baru-baru ini. Apakah ini perilaku default baru? Bagaimana cara mematikannya lagi?

\n\n

Tidak ada opsi pwfeedback yang disetel dalam file /etc/sudoers saya. Saya menggunakan Linux Mint 19 yang pada dasarnya adalah Ubuntu Bionic, versi paket sudo adalah 1.8.21p2-3ubuntu1 dan sudo biner saya menunjukkan hash sha256 yang sama dengan yang ada di paket.

\n\n

Terima kasih atas sarannya!

Saya menemukan ini di utas lain dan memastikan bahwa, jika pengembang Mint memperkenalkan kembali fitur buruk ini dalam pembaruan baru, modifikasi Anda sendiri akan menimpanya:

\n\n

Jawaban yang diterima hanya dengan menghapus /etc/sudoers.d/0pwfeedback akan bekerja dengan tepat sampai pembaruan berikutnya untuk mintsystem mengembalikan file itu ke tempatnya.

\n\n

Cara untuk kembali ke perilaku lama, yang direkomendasikan oleh orang yang bertanggung jawab untuk yang baru, adalah dengan mengganti pengaturan di file lain yang akan dibaca nanti: 

echo 'Defaults !pwfeedback'|sudo tee /etc/sudoers.d/9_no_pwfeedback\n

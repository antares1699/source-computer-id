Saya memiliki Dell Precision 490 yang menjalankan Windows 7 dan kartu video NVidia Quadro NVS 285. Saya memiliki dua monitor, dan kiri saya dicolokkan ke kabel bertanda "1" dan kanan dicolokkan ke "2". Di dalam Windows, Panel Kontrol NVidia dan Resolusi Layar Windows standar keduanya mengidentifikasi setiap monitor dengan benar.

\n\n

Tidak peduli berapa kali saya memperbaikinya, ketika sistem mulai, tampilan memiliki monitor 1 di sebelah kanan monitor 2, dan monitor 2 ditetapkan sebagai layar default. Saya dapat menggunakan Panel Kontrol NVidia dan memberitahukannya ke 'Pulihkan Default', dan kemudian masuk ke konfigurasi yang benar.

\n\n

Tidak ada virus atau malware, jadi saya bingung mengapa pengaturannya salah lagi saat saya boot. Bagaimana saya bisa mencegah hal ini terjadi?

Saya telah melihat masalah ini berkali-kali dan langkah-langkahnya tampak sedikit berbeda untuk mesin, monitor, dan versi Windows yang berbeda. Saya bahkan pernah melihat masalah serupa di Linux.

\n\n

Apa yang saya sarankan adalah:

\n\n
    \n
  1. Berhenti memperluas desktop ke monitor 2.
    2. \n
  2. Cabut kabel video monitor 2 dan biarkan Windows menyadari itu hilang.
    3. \n
  3. Matikan mesin.
    4. \n
  4. Nyalakan mesin hanya dengan satu layar.
    5. \n
  5. Biarkan booting, masuk lalu matikan lagi.
    6. \n
  6. Nyalakan kembali mesin, biarkan boot dan masuk.
    7. \n
  7. Colokkan kembali kabel video monitor 2.
    8. \n
  8. Perluas desktop untuk memantau 2.
    9. \n
\n\n

Jika Anda beruntung, lain kali Anda memulai, layar akan tetap berada di tempat Anda meletakkannya.

\n\n

Di bawah XP / Vista jika gagal, saya akan mencoba menghapus perangkat monitor di pengelola perangkat setelah langkah 2 (setelah menampilkan perangkat tersembunyi). Tidak yakin apakah ini akan membuat perbedaan di Windows 7.

\n\n

Akhirnya, jika itu tidak berhasil, saya akan mencoba menghubungkan Monitor 1 ke port video lain antara langkah 3 dan 4.

\n\n

Saya memiliki klien tipis berbasis Linux yang terhubung ke server windows melalui RDP. Saya memiliki masalah perpanjangan di excel saat menggunakan numpad.

Num pad minus dan minus pada keyboard utama adalah kode tombol yang berbeda.

\n\n

Remap kode kunci di registri dan masalahnya hilang !!

\n\n

Berikut adalah regfile yang akan memetakan numpad - (minus) ke keyboard - (minus) 

Windows Registry Editor Version 5.00\n\n[HKEY_LOCAL_MACHINE\\SYSTEM\\CurrentControlSet\\Control\\Keyboard Layout]\n\"Scancode Map\"=hex:00,00,00,00,00,00,00,00,02,00,00,00,0C,00,4A,00,00,00,00,0\n

0

\n\n

Anda perlu reboot tetapi langsung berfungsi !!

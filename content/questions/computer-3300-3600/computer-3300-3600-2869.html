Saya memiliki file .mp4 yang saat ini berukuran 830x1100 (ini screencast) yang perlu saya ubah ukurannya menjadi 114x150. Saya telah bereksperimen selama berjam-jam dengan pengaturan ffmpeg dan tidak ada yang terlihat bagus dan tajam. File aslinya terlihat bagus, dan saya tidak tahu mengapa downscaling membuatnya terlihat sangat buram.

\n\n

Sayangnya saya tidak dapat membagikan video karena berisi panduan untuk perangkat lunak yang belum dirilis.

","source":"https://superuser.com/questions/496738/what-is-the-best-way-to-downscale-mp4-video-while-keeping-it-as-sharp-as-possibl","author":"Brandon Durham","replies":[],"answers":[{"answer":"

114x150 sangat kecil mengingat ukuran sumbernya dan saya tidak yakin apakah metode penskalaan apa pun akan memberi Anda hasil yang memuaskan. Karena itu, Anda dapat bereksperimen dengan opsi -sws_flags di ffmpeg. 

ffmpeg -i input -filter:v scale=114:-1 -sws_flags lanczos output\n

Ini hanyalah sebuah contoh: Saya tidak menyarankan bahwa lanczos akan menjadi pilihan terbaik Anda. Default-nya adalah bicubic. Lihat daftar lengkapnya di ffmpeg -h full.

\n\n

Juga ingat untuk memberikan kualitas yang cukup tinggi sehingga artefak kompresi tidak mengganggu. Lihat Panduan Pengkodean FFmpeg dan x264 sebagai contoh.

\n\n

Pilihan lain, meskipun mungkin bukan saran yang bagus, adalah mencoba filter unsharp.

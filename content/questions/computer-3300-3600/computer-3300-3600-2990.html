apa yang ingin saya lakukan

\n\n
    \n
  1. mengedit satu file
    2. \n
  2. run '! svn diff%' (atau perintah lain yang menggunakan nama file saat ini )
    3. \n
  3. letakkan output di buffer (atau tab) baru
    4. \n
\n\n

Ini harus bekerja: 

:tabnew | r !svn diff #\n

# mereferensikan buffer sebelumnya, yaitu buffer yang telah Anda buka sebelum :tabnew dijalankan.

\n\n

Bahkan lebih baik 

:tabnew | set buftype=nowrite | r !svn diff #\n

karena vim tidak akan mengeluh ketika Anda menutup tab, tetapi ini sudah terlalu banyak untuk diketik setiap saat.

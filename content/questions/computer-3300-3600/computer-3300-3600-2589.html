Adobe Photoshop (yang saya kira adalah standar emas untuk perangkat lunak pengedit grafis hari ini) adalah secara eksplisit dirancang untuk menggunakan GPU untuk merender dan melakukan berbagai transformasi dan proses lainnya. Namun, paint.net tidak - dan saya akan membahasnya sebentar lagi.

\n\n

Konon, salah satu kekuatan paint.net adalah kemampuannya plugin sistem, dan meskipun Anda tidak dapat memiliki jenis akselerasi GPU yang sama, plugin dapat ditulis yang memanfaatkan akselerasi GPU. Berikut adalah contoh salah satunya - a plugin yang menerapkan pengaburan akselerasi GPU

Ini adalah masalah umum pada driver NVidia. Mereka payah memungkinkan Anda untuk menentukan kartu grafis mana yang akan digunakan dalam kasus beberapa kartu grafis di sistem Anda.

\n\n

Anda dapat memaksa penggunaan kartu grafis kelas atas saat program Paint.net diluncurkan dengan membuka NVIDIA Control Panel -> 3D Settings -> Manage 3D Settings -> Program Settings tab, pilih Paint.net dari menu tarik Select a program to customize (gunakan tombol Add jika tidak ada dalam daftar untuk menambahkan jalur yang dapat dieksekusi. Pilih GPU yang diinginkan dari daftar tarik Select the preferred graphics processor for this program. Tutup NVidia Control Panel. Itu saja.

\n\n

NVIDIA diketahui juga menggunakan profil bayangan yang memblokir akselerasi GPU pada program tertentu tanpa memberi tahu pengguna (VLC adalah masalah yang diketahui, misalnya). Saya tidak yakin Paint.net adalah salah satunya. Jika cara di atas tidak berhasil, coba ganti nama file Paint.net .exe, lalu tambahkan file yang diubah namanya ke daftar tarik di Panel Kontrol NVIDIA seperti dijelaskan di atas. Itu akan melewati profil bayangan apa pun.

Saya telah membuat layanan pemula dari inti ini . Saya memasukkannya ke dalam: 

[andrey@vm01 ~]$ ll /etc/init/play.conf\n-rw-r--r--. 1 root root 1255 Dec  1 20:50 /etc/init/play.conf\n

Saya juga menginstal pemula, karena tidak ada di sistem saya: 

sudo yum install upstart\n

Saya memulai ulang beberapa kali, tetapi saya masih mendapatkan kesalahan ini: 

[andrey@vm01 ~]$ sudo service play status\n[sudo] password for andrey:\nplay: unrecognized service\n

Apakah Anda punya ide mengapa?

sudo initctl status play\n

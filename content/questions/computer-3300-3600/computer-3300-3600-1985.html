Aplikasi default ditentukan dalam file ~/.local/share/applications/mimeapps.list. Buka file di editor teks favorit Anda dan ubah 

x-scheme-handler/http=firefox.desktop\nx-scheme-handler/https=firefox.desktop\n

untuk 

x-scheme-handler/http=chrome.desktop\nx-scheme-handler/https=chrome.desktop\n

Itu akan bekerja selama Anda memiliki chrome.desktop di \n/usr/share/applications/.

\n\n
\n\n

MEMPERBARUI:

\n\n

Jika Anda tidak memiliki file chrome.desktop di /usr/share/applications/ buat saja dengan baris berikut: 

Name=Google Chrome\nTerminal=false\nIcon=google-chrome\nType=Application\nCategories=Network;WebBrowser;\nMimeType=text/html;text/xml;application/xhtml_xml;x-scheme-handler/http;x-scheme-handler/https;x-scheme-handler/ftp;\nX-Ayatana-Desktop-Shortcuts=NewWindow;NewIncognito\nExec=/opt/google/chrome/google-chrome %U\n

Ubah /opt/google/chrome/google-chrome menjadi apa pun yang dikembalikan oleh which google-chrome.

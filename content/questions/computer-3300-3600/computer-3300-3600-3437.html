Bagaimana cara mengaktifkan kompresi memori Windows 10?

\n
\n\n

Kompresi memori diaktifkan secara default. Namun, itu muncul karena alasan apa pun, saat ini Anda telah menonaktifkannya. Dalam prompt perintah Administrator PowerShell ketik perintah berikut untuk menonaktifkan kompresi memori

\n\n

Disable-MMAgent -MemoryCompression

\n\n

\n\n

Dalam prompt perintah Administrator PowerShell ketik perintah berikut untuk mengaktifkan kompresi memori

\n\n

Enable-MMAgent -MemoryCompression

\n\n

\n\n

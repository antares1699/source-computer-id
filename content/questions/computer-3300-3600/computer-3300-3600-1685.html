Ada beberapa cara.

\n\n
    \n
  1. Menggunakan FEBE dan CLEO addons untuk membuat file XPI terpaket yang dapat diinstal dalam satu kesempatan
    Itu FEBE addon itu sendiri cukup keren
    tetapi CLEO addon membantu mengemas file-nya menjadi satu 'addon' yang dapat diinstal
    2. \n
  2. Membuat anda Koleksi Firefox halaman untuk mengunduh dan menginstal kapan pun diperlukan
    Itu Addon Collector juga akan berguna
    3. \n
\n\n

Bergantung pada kebutuhan Anda, salah satu dari dua skema di atas akan berfungsi.

\n\n

Perhatikan bahwa file Urutan FEBE sebenarnya dapat mencadangkan lebih dari sekadar daftar ekstensi Anda.
Ini juga akan menyimpan preferensi Anda dan jika Anda memilih, kata sandi, cookie, bookmark, dll.
Oleh karena itu, metode FEBE adalah cara terbaik untuk mereplikasi instalasi firefox Anda di komputer lain
(atau bahkan login lain di mesin yang sama).

\n\n

Itu Metode koleksi sementara itu sangat baik untuk merujuk daftar ekstensi Anda ke orang lain, atau
dapatkan add-on favorit Anda di mesin lain di mana Anda tidak dapat (atau tidak ingin) mengakses cadangan FEBE Anda.

Untuk menyinkronkan ekstensi Anda, Anda cukup menyinkronkan profil firefox Anda.\nGunakan dropbox untuk menyinkronkan folder profil Firefox Anda.

\n\n

Biasanya saya tidak menyinkronkan ekstensi saya, begitulah cara saya menjaga konsistensi data saya\ndi semua komputer

\n\n

Pada Sistem Saat Ini\n 1. Tandai semua ekstensi saya di Delicious\n 2. Saya menggunakan Foxmarks (sekarang XMarks) untuk menyinkronkan kata sandi dan penanda

\n\n

Goto sistem kedua

\n\n
    \n
  1. Pasang Firefox
    2. \n
  2. Instal semua ekstensi yang dibookmarkr
    3. \n
  3. Instal XMarks dan sinkronkan kata sandi dan bookmaks Anda
    4. \n
\n\n

Dengan cara ini saya menjaga data saya tetap konsisten di seluruh sistem, dan saya tetap menyimpannya\nsetiap instalasi bersih (dengan tidak menyalin ekstensi dan profil)

\n\n

Alternatif untuk Xmarks Anda juga dapat menggunakan Mozilla Weave.

\n\n

Anda kehilangan argumen nama file - yang menginstruksikan Vim untuk mengisi buffer dari stdin; cp. :help -- 

echo $PATH | vim -\n

Sebagai alternatif, Anda dapat menggunakan cangkang Anda proses substitusi untuk membuat deskriptor file sementara dan meminta Vim mengedit file "virtual" itu. 

vim <(echo $PATH)\n

Bagaimana cara mengalihkan keluaran dari stout ke vim?

\n\n

Pertanyaan Anda telah terjawab di AskUbuntu.

\n\n

Solusi paling sederhana adalah menambahkan - ke perintah Anda: 

echo $PATH | vim -\n
\n

Kamu bisa gunakan proses substitusi (ini juga berfungsi dengan\n aplikasi yang tidak dapat membaca dari STDIN): 

vim <(ls -la)\n

Atau gunakan fungsi vim untuk membaca dari STDIN: 

ls -la | vim -\n
\n\n

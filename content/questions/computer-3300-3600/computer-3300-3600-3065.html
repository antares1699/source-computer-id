Bagaimana Anda membuat mesin Windows 7 disinkronkan dengan gratis NTP server di Internet?

\n\n

Ini berisi detail untuk Windows Server 2008 , tetapi setelah mengubah registri dan mencoba memulai ulang layanan, tampaknya tidak berfungsi pada Windows 7. 

C:\\Windows\\system32>net stop w32time\nThe Windows Time service is not started.\n\nMore help is available by typing NET HELPMSG 3521.\n\n\nC:\\Windows\\system32>net start w32time\nSystem error 1058 has occurred.\n\nThe service cannot be started, either because it is disabled or because it has no enabled devices associated with it.\n
","source":"https://superuser.com/questions/308188/ntp-sync-on-windows-7","author":"Jonathan","replies":[],"answers":[{"answer":"

Buka services.msc dan pastikan layanan Windows Time disetel ke otomatis, tidak dinonaktifkan. Jika ya, kirim kembali, dan kami dapat mencoba memperbaikinya.

\n\n

Jika komputer ada di domain, Anda dapat mengikuti saran yang sama untuk tahun 2008, meskipun jika kebijakan grup disetel, itu akan menimpa pengaturan Anda saat diperbarui lagi.

\n\n

Jika tidak dalam domain, Anda dapat mengklik jam dan tab Waktu Internet membuatnya mudah. Tab ini tersembunyi jika merupakan bagian dari domain.

\n\n

\n\n

Berikut langkah-langkah untuk memperbaiki layanan Windows Time. Dari baris perintah, lakukan hal berikut:

\n\n
    \n
  1. w32time berhenti bersih
    2. \n
  2. w32tm / unregister (abaikan pesan error)
    3. \n
  3. w32tm / batalkan pendaftaran
    4. \n
  4. w32tm / register
    5. \n
  5. w32time mulai bersih
    6. \n
\n\n

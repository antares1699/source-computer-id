Semoga pertanyaan sederhana dengan jawaban yang sama sederhananya ...

\n\n

Saya mencoba membuat skrip kunci otomatis untuk mengalihkan transparansi jendela.

\n\n

Saat ini, saya mengatur transparansi jendela aktif dengan yang berikut: 

^!RButton::WinSet, Transparent, 150, A\n

Namun, saya ingin memperpanjang ini untuk beralih antara 150 dan 255, tetapi saya mengalami kesulitan memahami cara mengaktifkannya.

\n\n
\n\n

Edit: Dengan toggle, maksud saya pukul Ctrl + Alt + Tombol kanan mouse sekali untuk menyetel transparansi ke 150, lalu menekannya lagi untuk menyetel jendela kembali ke transparansi penuh.

Sesuatu seperti ini mungkin akan melakukannya (belum teruji): 

^!RButton::\nWinGet, currentTransparency, Transparent, A\nif (currentTransparency = OFF)\n{\n    WinSet, Transparent, 150, A\n}\nelse\n{\n    WinSet, Transparent, OFF, A\n}\nreturn\n

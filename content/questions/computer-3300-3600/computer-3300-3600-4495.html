saya menemukan komentar di blog Cameron Cooke yang menyarankan mengubah port IP yang didengarkan Apache. Defaultnya adalah port 80.

\n
    \n

  1. Ubah ke port 8666 (atau apa pun di atas 1024)

    \n
    2. \n

  2. Temukan file httpd.conf di direktori berikut: 

     [install directory]\\xampp\\apache\\conf\n
    3. \n

  3. Temukan baris yang bertuliskan, Listen 80 dan ubah ke Listen 8666. (Atau nomor berapa pun yang Anda lakukan)

    \n
    4. \n

  4. Simpan dan Tutup file

    \n
    5. \n

  5. Mulai ulang layanan Apache di panel kontrol Xampp.

    \n
    6. \n
\n

Ada peringatan yang mereka sebutkan:

\n
\n

Sekarang satu-satunya tangkapan untuk metode ini adalah Anda tidak bisa hanya pergi ke http://localhost/xampp lagi Anda harus memberi tahu browser Anda port mana yang khusus digunakan (secara default akan menggunakan 80), jadi Anda harus menggunakan http://localhost:8666/xampp/ (port ditetapkan dengan titik dua dan kemudian angkanya).

\n

Yang keren adalah saya dapat menjalankan http://localhost:8666 untuk menjalankan Apache dan http://localhost:8616 untuk menjalankan IIS lokal saya untuk proyek .NET.

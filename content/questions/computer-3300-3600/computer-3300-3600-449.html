Kami mendapatkan broadband kantor kami melalui router WiFi Broadband Bisnis BT. Kami memiliki campuran Mac dan Windows PC / laptop yang terhubung ke sana pada satu waktu.

\n\n

Semua perangkat dapat terhubung ke sinyal nirkabel dengan sinyal yang cukup lengkap.

\n\n

Namun, hanya dua komputer (satu windows, satu os x) yang dapat terhubung ke internet secara konsisten. Tiga lainnya (satu jendela, dua os x), sementara mereka selalu dapat terhubung ke wifi, menunjukkan salah satu dari tiga karakteristik.

\n\n
    \n

  1. Tidak ada internet sama sekali.

    2. \n

  2. Program seperti Skype berfungsi, tetapi tidak ada internet melalui browser.

    3. \n

  3. Internet berfungsi, tetapi dengan kelambatan terputus-putus saat beralih di antara situs yang berbeda. Saya berasumsi ketika mencoba menyelesaikan alamat yang berbeda.

    4. \n
\n\n

Ini menunjukkan konfigurasi DNS yang tidak tepat.

\n\n

Alasan mengapa Skype masih berfungsi tetapi bukan browser web adalah karena Skype tidak bergantung pada DNS. Pastikan Anda memiliki alamat server DNS yang benar untuk antarmuka jaringan Anda atau gunakan alamat IP router sebagai gantinya.

\n\n

Solusi saya adalah

\n\n
    \n
  1. Buka Koneksi Jaringan dan Dial-up
    2. \n
  2. Klik kanan pada koneksi dan pilih 'Properties'
    3. \n
  3. Pilih Protokol Internet (TCP / IP)
    4. \n
  4. Klik 'Properties'
    5. \n
  5. Klik 'Advanced'
    6. \n
  6. pilih Tab DNS
    7. \n
  7. tambahkan rute DNS IP (192.168.X.X)\nSimpan itu.
    8. \n
\n\n

Dalam kasus saya, setelah .168. Saya memiliki 1 dan 2 digit untuk alamat IP dan 1 dan 3 digit untuk gateway.

Saya mengalami situasi yang sangat tidak biasa di mana saya melakukan remote desktop ke PC Windows 7 dari laptop saya. Proses Explorer.exe telah dimatikan sehingga menu Start dll tidak tersedia. Oleh karena itu, sangat sedikit yang dapat saya lakukan dengan PC saya. Jadi saya perlu me-restart PC saya. Bagaimana cara me-restart PC saya setelah proses Explorer.exe dimatikan?

","source":"https://superuser.com/questions/230619/restart-pc-via-remote-desktop-with-explorer-exe-dead","author":"amateur","replies":[],"answers":[{"answer":"

Anda bisa masuk ke Task Manager dalam sesi desktop jarak jauh dengan menekan CTRL + BERGESER + ESC .

\n\n
    \n
  1. Klik Mengajukan
    2. \n
  2. Klik Tugas Baru (Jalankan ...)
    3. \n
  3. Tipe explorer.exe dan klik baik
    4. \n
\n\n

Itu akan langsung me-restart shell Explorer tanpa perlu me-restart PC Anda.

shutdown /r /m \\\\remotepcname\n

Ini berfungsi untuk saya karena saya menggunakan akun domain yang sama di kedua mesin. Bergantung pada bagaimana domain dan kebijakan grup Anda disiapkan, ini mungkin berhasil atau tidak untuk Anda.

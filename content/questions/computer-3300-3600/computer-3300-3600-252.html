Saya tahu folder induk yang akan berisi file mp3 yang saya cari, tetapi saya memiliki banyak sub-folder di dalamnya.

\n\n

Bagaimana cara mencari semua file dan sub-folder untuk file yang diakhiri dengan .mp3?

","source":"https://superuser.com/questions/290243/search-all-sub-folders-for-mp3-files","author":"user27449","replies":[],"answers":[{"answer":"

Anda dapat menggunakan perintah terminal find. Buka Terminal.app (Aplikasi> Utilitas> Terminal.app) dan jalankan perintah ini: 

find /path/to/search -name '*.mp3'\n

Anda perlu meletakkan pola dalam tanda kutip agar tidak diperluas oleh shell.

Karena Anda tidak menentukan bahwa Anda membutuhkan Terminal, Anda sebenarnya dapat menggunakan Finder. Masuk ke folder induk, tekan Cmd - F , lalu tekan + tombol di sebelah kanan.

\n\n

Sesuaikan pengaturan Anda seperti yang terlihat di bawah ini. Anda akan mendapatkan daftar semua file MP3.

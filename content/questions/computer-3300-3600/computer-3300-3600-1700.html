Apakah Anda menyadari di Panel Preferensi Keamanan Anda dapat meminta kata sandi untuk membangunkan dari screen saver, tetapi secara default tidak memerlukan otentikasi?

\n\n

Kemudian Anda dapat mengaktifkan screen saver dengan metode Anda, atau AppleScript ini: 

do shell script \"open /System/Library/Frameworks/ScreenSaver.framework/Versions/A/Resources/ScreenSaverEngine.app\"\n

Anda juga dapat menambahkan kunci Keychain dan menggunakan AppleScript yang saya temukan ini sini . 

activate application \"SystemUIServer\"\ntell application \"System Events\"\n    tell process \"SystemUIServer\"\n        repeat with i from 1 to number of menu bar items of menu bar 1\n            tell menu bar item i of menu bar 1\n                click\n                try\n                    if name of menu item 1 of front menu is \"Lock Screen\" then\n                        click menu item \"Lock Screen\" of front menu\n                        exit repeat\n                    end if\n                end try\n            end tell\n        end repeat\n    end tell\nend tell\n

Skrip ini memerlukan Akses Universal -> Aktifkan akses untuk perangkat bantu yang diperiksa.

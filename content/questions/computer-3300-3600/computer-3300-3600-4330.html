Ditambahkan ke /etc/sudoers

\n\n

yourusername ALL=NOPASSWD: ALL

\n\n

Telah menjadi veteran Unix selama 20 tahun, jauh sebelum dimulainya Ubuntu, saya juga tidak memerlukan pelajaran lain tentang akses root, dapat ditemukan di mana-mana mencari informasi sederhana.

\n\n

Tidak peduli apa yang dikatakan dokumen, bahkan sekarang dengan 11.10 popup sialan itu muncul setiap beberapa menit ketika saya memiliki lusinan penginstalan pusat perangkat lunak yang mengantri. Begitu banyak untuk mencoba menyelesaikannya dalam semalam.

\n\n

Hal lain yang sering saya gunakan adalah sudo -i ke shell root dari terminal untuk menjalankan software-center atau aplikasi gui umum lainnya (seperti gparted) yang sebaliknya hanya suka popup dialog kata sandi modal.

Pertanyaan aneh tapi inilah dua ide:

\n\n

a) mengaktifkan pengguna root yang menetapkan kata sandi: sudo passwd root

\n\n

b) masuk sebagai root

\n\n

Yang lain adalah men-tweak file sudoers:

\n\n

Buka terminal dan digit:

\n\n

sudo visudo

\n\n

Dan pindahkan paragraf besar ke akhir, hapus # di depan sudo dan tambahkan diri Anda ke grup sudo

\n\n

ps\nJika Anda ingin membuatnya tetap sederhana tambahkan di bawah:

\n\n

root SEMUA = (SEMUA) SEMUA

\n\n

yourusername ALL = (ALL) ALL

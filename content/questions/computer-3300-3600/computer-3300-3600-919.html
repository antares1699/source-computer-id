Bagaimana Anda dapat mengubah konsol startup default ConEmu?

\n\n

Saat ini, ketika saya memulai ConEmu, itu membuka satu tab dengan prompt perintah Windows default, tetapi saya ingin ConEmu untuk memulai PowerShell sebagai gantinya, dan mungkin konsol PowerShell administratif.

","source":"https://superuser.com/questions/786778/conemu-select-startup-console","author":"Zoltán","replies":[],"answers":[{"answer":"

Ada Startup halaman pengaturan di mana pengguna dapat memilih salah satu dari Tasks bernama (banyak shell sudah ditentukan sebelumnya) atau memilih baris perintah startup tertentu.

\n\n

Dan tentu saja, pengguna dapat menentukan sendiri Tasks atau menyusun ulang yang sudah ada.

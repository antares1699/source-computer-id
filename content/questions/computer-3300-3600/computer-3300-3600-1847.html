Ini seharusnya cukup mudah dilakukan.

\n\n

Folder startup pribadi Anda (per pengguna) terletak di: 

C:\\Users\\Your_user_name\\AppData\\Roaming\\Microsoft\\Windows\\Start Menu\\Programs\\Startup\n

Anda dapat menambahkan pintasan ke aplikasi yang ingin Anda luncurkan di folder ini untuk pengguna yang Anda inginkan. Sebagai jalan pintas, Anda dapat menuju start> run> ketik "shell:Startup" (tanpa tanda kutip) dan kemudian ok dan itu akan membawa Anda ke folder di atas secara otomatis. Salin pintasan apa pun ke folder lain sehingga Anda dapat menghapusnya dengan aman dan kemudian menambahkan entri baru Anda (pintasan ke perangkat lunak yang ingin Anda luncurkan "per pengguna")

\n\n

Anda juga dapat meluncurkan program melalui registri dengan membuka RegEdit (Start> Run> Regedit.exe) dan menavigasi ke: 

HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Run\n

Anda dapat menemukan entri program di sini (lihat gambar di bawah) Jika Anda mencatat nilai yang ditampilkan di sini, Anda dapat menghapus entri tersebut. Pastikan untuk mencatat nilai atau mencadangkan kunci terlebih dahulu jika Anda perlu memulihkannya di lain waktu.

\n\n
\n\n

Anda kemudian perlu menghapus item startup untuk "Semua pengguna" (alasan peluncurannya di akun admin Anda juga). Jalan untuk ini adalah: 

C:\\ProgramData\\Microsoft\\Windows\\Start Menu\\Programs\\Startup\n

Jalan pintas untuk ini adalah: shell:Common Startup

\n\n

Lintasan registri untuk startup pengguna umum adalah: 

HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Run\n

Anda dapat mencadangkan, menghapus, dan menambahkan entri di lokasi ini seperti yang dijelaskan di bagian "per pengguna" di jawaban saya.

\n\n

Anda mungkin juga merasa lebih mudah (jika berurusan dengan pintasan di folder startup) untuk hanya membuka 2 jendela dan menyeret-jatuhkan pintasan dari "semua pengguna" ke "pengguna Anda" dan sebaliknya.

\n\n

Semoga ini membantu.

Saya tidak memiliki pengalaman dengan launchd tetapi ingin mengatur Tor dan Polipo sebagai layanan yang secara otomatis dimulai ulang setiap 24 jam atau lebih. Bagaimana saya bisa melakukan ini?

Untuk itu, Anda dapat menggunakan daftar peluncuran ini untuk memulainya saat peluncuran sistem. (Tidak perlu pengguna untuk masuk yang ideal untuk server dan orang yang menjalankan jembatan atau node yang akan aktif 24/7.)

\n\n

Simpan file ini /Library/LaunchDaemons/org.torproject.tor.plist (Anda akan membutuhkan hak akses root untuk itu jadi gunakan sudo.) 

<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n<!DOCTYPE plist PUBLIC \"-//Apple//DTD PLIST 1.0//EN\" \"http://www.apple.com/DTDs/PropertyList-1.0.dtd\">\n<plist version=\"1.0\">\n<dict>\n    <key>Disabled</key>\n    <false/>\n        <key>Label</key>\n        <string>org.torproject.tor</string>\n        <key>ProgramArguments</key>\n        <array>\n                <string>/path/to/your/binary/of/tor</string>\n                <string>-f</string>\n                <string>/path/to/your/torrc</string>\n        </array>\n        <key>RunAtLoad</key>\n        <true/>\n    <key>OnDemand</key>\n    <false/>\n    <key>UserName</key>\n    <string>userNameToRunTorUnder-UsuallyTorOrYourOwnUsershortname</string>\n    <key>GroupName</key>\n    <string>daemon</string>\n</dict>\n</plist>\n

Ganti informasi berikut dengan yang cocok dengan sistem Anda:

\n\n

/path/to/your/binary/of/tor Ini biasanya /usr/local/bin/tor jika Anda menginstal tor dari sumber atau /Applications/Vidalia.app/Contents/MacOS/tor jika Anda ingin menggunakan biner tor dari Vidalia. Atau cari dengan which tor pada sistem Anda.

\n\n

/path/to/your/torrc Jika Anda menjalankan tor sebagai pengguna terpisah, ini harus berada di .torrc di direktori home pengguna itu. Anda perlu menentukan jalur absolut ke file ini. Misalnya jika pengguna Anda bernama tor dan berada di /Users/tor maka jalur itu akan menjadi /Users/tor/.torrc

\n\n

userNameToRunTorUnder-UsuallyTorOrYourOwnUsershortname Ini adalah nama pengguna untuk menjalankan tor di bawah. Jika Anda memiliki pengguna terpisah (yang kemungkinan akan diberi nama tor ) lalu gunakan yang satu itu, jika tidak Anda dapat menggunakan hampir semua nama pengguna lain. Jika ini hanya untuk Anda sendiri, buatlah usernamehortname Anda sendiri. (Aka apa nama direktori home Anda.)

\n\n

Pastikan hak istimewa dari item launchd aman seperti yang diharuskan oleh launchd. 

sudo chown root:wheel /Library/LaunchDaemons/org.torproject.tor.plist\nsudo chmod 755 /Library/LaunchDaemons/org.torproject.tor.plist\n

Kemudian luncurkan dengan\n sudo launchctl load -w /Library/LaunchDaemons/org.torproject.tor.plist

\n\n

Anda dapat menggunakan item yang sangat mirip untuk meluncurkan polipo.\nSimpan item Launchd ini sebagai /Library/LaunchDaemons/fr.jussieu.pps.polipo.plist 

<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n<!DOCTYPE plist PUBLIC \"-//Apple//DTD PLIST 1.0//EN\" \"http://www.apple.com/DTDs/PropertyList-1.0.dtd\">\n<plist version=\"1.0\">\n<dict>\n    <key>Disabled</key>\n    <false/>\n        <key>Label</key>\n        <string>fr.jussieu.pps.polipo</string>\n        <key>ProgramArguments</key>\n        <array>\n                <string>/path/to/your/binary/of/polipo</string>\n                <string>-c</string>\n                <string>/path/to/your/polipo.conf</string>\n        </array>\n        <key>RunAtLoad</key>\n        <true/>\n    <key>OnDemand</key>\n    <false/>\n    <key>UserName</key>\n    <string>userNameToRunPolipoUnder-UsuallyPolipoOrYourOwnUsershortname</string>\n    <key>GroupName</key>\n    <string>daemon</string>\n</dict>\n</plist>\n

Sekali lagi ganti informasi berikut dengan yang cocok dengan sistem Anda: / path / ke / Anda / biner / dari / polipo Jika Anda ingin menggunakan yang dari Vidalia gunakan /Applications/Vidalia.app/Contents/MacOS/polipo yang lain cari tahu di mana biner polipo Anda dengan which polipo. (Misalnya MacPorts tidak menginstal ini pada /opt/local/bin/polipo)

\n\n

/path/to/your/polipo.conf Sekali lagi, jika Anda ingin menggunakan file yang disediakan oleh Vidalia /Applications/Vidalia.app/Contents/Resources/polipo.conf atau menggunakan jalur absolut ke konfigurasi kustom Anda.

\n\n

userNameToRunPolipoUnder-UsuallyPolipoOrYourOwnUsershortname Ini bisa menjadi nama pengguna yang sama dengan yang Anda jalankan tor, tetapi juga bisa menjadi pengguna terpisah. Saya ingin dasmon saya dipisahkan.

\n\n

Sekali lagi, pastikan hak istimewa dari item launchd aman seperti yang diharuskan oleh launchd. 

sudo chown root:wheel /Library/LaunchDaemons/fr.jussieu.pps.polipo.plist\nsudo chmod 755 /Library/LaunchDaemons/fr.jussieu.pps.polipo.plist\n

Kemudian luncurkan dengan\n sudo launchctl load -w /Library/LaunchDaemons/fr.jussieu.pps.polipo.plist

\n\n

Peringatan:\nSetup ini bisa tidak restart daemon setiap 24 jam seperti yang diminta. Saya rasa itu tidak perlu karena saya tidak pernah memiliki masalah dengan stabilitas dengan tor. (Saya telah menjalankan node selama beberapa tahun di Mac OS X.)

\n\n

tor tidak tidak menangani sistem tidur sama sekali. Jadi jika Anda menidurkan Mac, Anda harus memulai ulang tor agar dapat membangun sirkuit kembali dengan andal.

\n\n

Deskripsi ini ditujukan untuk Mac yang masih aktif dan berjalan. Ini dimungkinkan untuk menjalankan ini sebagai pengguna Anda dan hanya jika Anda login secara grafis tanpa memerlukan hak istimewa root.

\n\n

Seharusnya begitu.

\n\n

NB: Harap diingat bahwa userNameToRunPolipoUnder-UsuallyPolipoOrYourOwnUsershortname dimaksudkan untuk memungkinkan akses di mana pemblokiran dan penyensoran terjadi serta anonimitas dan privasi online dan mengapa menjalankan BitTorrent di atas tor bukanlah ide yang baik .

Anda tidak dapat menemukannya versi mana yang diinstal oleh kunci tersebut.

\n\n

Sejak Vista, Anda memiliki satu kunci dan dua disk pengaturan: untuk versi 32 bit dan 64 bit. (Versi kotak dari Peningkatan Windows 8 juga memiliki dua disk.)

\n\n

Jadi untuk mengetahui versi mana yang diinstal, Anda harus mem-boot ke sistem itu atau setidaknya melihat sistem filenya. Ada folder Program Files (x86) di root instalasi 64 bit, dan folder SysWOW64 di folder Windows.

\n\n
\n\n

Kemudian kamu tidak perlu tahu karena itu tidak terlalu penting.

\n\n
    \n
  • Jika ada lebih dari 4 GB RAM yang dipasang, Anda pasti membutuhkan versi 64 bit.
    • \n
  • Jika ada lebih dari 2 GB, Anda dapat menggunakan versi mana pun.
    \n (32 bit membutuhkan lebih sedikit ruang disk, lebih sedikit memori dalam beberapa kasus.)
    • \n
  • Jika kurang dari 2 GB, Anda memerlukan 32 bit karena 64 bit memerlukan RAM 2 GB untuk berjalan.
    • \n
\n\n

Anda juga dapat memeriksa apakah prosesor yang dipasang berkemampuan 64 bit atau tidak. Kemungkinan besar prosesor dapat berjalan dalam mode 64 bit kecuali jika sudah sangat tua.

\n\n

Jika Anda tidak mengetahui jumlah RAM atau prosesor, Anda dapat melihat info ini di Internet menggunakan nomor model laptop Anda.

Namun, saya hanya dapat menebak berapa banyak data yang sebenarnya perlu diunduh oleh Asisten Peningkatan - Saya tidak dapat menemukan ukurannya di mana pun di situs web Microsoft atau situs Windows 8. Windows 8 / Windows 8 Pro x64 = 3,33 GiB (3,58,18,53,696 byte) ETA: 6.6 hrs at 1.2 Mbps

\n\n

Windows 8 / Windows 8 Pro x86 = 2,45 GiB (2,63,24,60,288 byte) ETA: 4.8 hrs at 1.2 Mbps

\n\n

Baik edisi Core (normal) dan Pro termasuk dalam satu ISO. Pengaturan Windows 8 secara otomatis menginstal edisi yang tersedia untuk kunci produk. Anda juga bisa mendapatkan DVD dengan tambahan $ 20.

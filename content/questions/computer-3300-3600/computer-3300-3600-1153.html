Saya mencoba TrueCrypt tetapi sepertinya tidak meminta kata sandi setiap kali Anda mencoba membuka file (yah, kecuali saya melewatkan sesuatu).

\n\n

Sebagai solusi sederhana, saya memutuskan untuk menggunakan WinRAR untuk tugas ini (dengan menambahkan kata sandi ke arsip). Ini bekerja dengan baik dan semuanya tetapi saya memiliki sedikit perhatian. Setiap kali saya membuka file dengan WinRAR itu mendekompresi file tersebut ke folder sementara. Sekarang saya khawatir datanya bisa diakses di folder sementara. Saya tidak tahu apakah WinRAR menghapus konten folder sementara nanti, tetapi meskipun demikian, konten masih dapat diakses oleh beberapa utilitas (salah satu utilitas yang memungkinkan Anda untuk melihat / menyimpan file yang dihapus) jadi tidak ' Bagi saya, sepertinya bukan pilihan yang aman.

\n\n

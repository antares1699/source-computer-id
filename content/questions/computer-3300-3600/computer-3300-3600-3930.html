Jika Anda hanya mengetik SUBST sendiri, tidak ada parameter, itu akan mencantumkan semua huruf drive yang dibuat oleh subst, mis. 

C:\\>subst Z: C:\\Windows\nC:\\>subst\n Z:\\: => C:\\Windows\nC:\\>subst z: /D\nC:\\>subst\n (blank)\n

map Z: ke folder windows, tampilkan daftar pemetaan, hapus Z: pemetaan, tampilkan pemetaan lagi (tidak ada)\nJadi jika Anda tidak melihatnya di sana, itu tidak dibuat oleh subst.

$ subst\nI:\\: => C:\\Users\\ME\\Dropbox\\Images\n\n$ subst /D I:\nInvalid parameter - D:/\n\n$ subst I:\\ /D\nInvalid parameter - I: /D\n\n$ subst I: /D\nPath not found - D:/\n

$ which subst\n/c/WINDOWS/system32/subst\n

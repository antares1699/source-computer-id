Pada beberapa komputer saya memiliki tampilan keamanan yang meminta saya untuk "menekan Ctrl + Alt + Del" sehingga saya dapat membuka jendela Login dengan kata sandi nama pengguna dan seterusnya, bahkan saat boot (saya kira begitu) ...

\n\n

Tetapi di beberapa komputer saya tidak memiliki pesan itu saat boot.

\n\n

Apakah Anda mengetahui parameter yang dapat saya ubah untuk beralih di antara dua mode? (Menampilkan atau tidak jendela keamanan ini).

Anda dapat mengubahnya melalui Akun Pengguna di Panel Kontrol

\n\n
\n
    \n
  • Masuk ke komputer sebagai administrator.
    • \n
  • Klik Mulai, klik Panel Kontrol, lalu klik dua kali Akun Pengguna.
    • \n
  • Klik Ubah cara pengguna masuk atau keluar.
    • \n
  • Klik untuk memilih kotak centang Gunakan layar Selamat Datang jika Anda ingin\n untuk menonaktifkan CTRL + ALT + DELETE\n urutan
    • \n
  • Klik Mulai, klik Jalankan, ketik Control Userpasswords2, dan kemudian\n tekan enter. Catatan Anda mungkin menerima\n kotak dialog berikut: Anda harus menjadi\n anggota grup Administrator di\n komputer ini untuk membuka Akun Pengguna.\n Anda masuk sebagai nama pengguna, dan\n akun ini bukan anggota dari\n Grup administrator. Di dalam\n syaratnya, Anda diminta untuk mengetik\n nama pengguna dan sandi saat\n sistem meminta kotak dialog untuk mengetik\n nama pengguna dan kata sandi\n administrator komputer.
    • \n
  • Klik tab Advanced.
    • \n
  • Di bagian logon aman, klik untuk menghapus Wajibkan pengguna untuk menekan\n Ctrl + Alt + Hapus kotak centang jika Anda mau\n untuk menonaktifkan CTRL + ALT + DELETE\n urutan.
    • \n

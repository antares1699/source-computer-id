Di Linux dengan GNU grep : 

grep -F --text -o --byte-offset mystring binaryfile\n

Contoh: 

$ grep -F --text -o --byte-offset option /bin/tar\n226542:option\n237529:option\n237612:option\n...\n

Penjelasan parameter: 

-F, --fixed-strings\n      Interpret PATTERN as a list of fixed strings, separated by newlines,\n      any of which is to be matched.  (-F is specified by POSIX.)\n\n-a, --text\n      Process  a binary file as if it were text; this is equivalent to the\n      --binary-files=text option.\n\n-o, --only-matching\n      Print  only the matched (non-empty) parts of a matching line, with\n      each such part on a separate output line.\n\n-b, --byte-offset\n      Print the 0-based byte offset within the input file before each line\n      of output. If -o (--only-matching) is specified, print the offset of\n      the matching part itself\n

Untuk menghitung kejadian, tambahkan | wc -l ke baris perintah.

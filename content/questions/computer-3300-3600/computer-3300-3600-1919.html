Lupakan semua alat lainnya ... gunakan baris perintah!

\n\n
\n

sudo du -hd 1 /

\n
\n\n

akan menunjukkan ukuran folder dari root drive Anda ke kedalaman satu tingkat.

\n\n
    \n
  • du adalah alat statistik penggunaan disk
      \n
    • -h memberi Anda output "yang dapat dibaca manusia"
      • \n
    • -d 1 melintasi kedalaman satu tingkat. Ini adalah titik awal yang baik untuk melihat di mana pelakunya berada
      • \n
    • / adalah folder root
      • \n
    • \n
\n\n

Anda harus menjalankan ini dengan sudo untuk mendapatkan hitungan yang akurat dan menghindari kesalahan izin. Ketika Anda menemukan "what the heck ...", ubah jalur ke folder itu dan selidiki. Sebagai contoh

\n\n
    \n
  • sudo du -hd 1 /what_the_heck_is_this_folder
    • \n
\n\n

dan Anda akan mulai mendapatkan gambaran yang lebih jelas. Semoga berhasil.

Ini tidak menjawab pertanyaan Anda tentang "Bagaimana cara mengetahui bagaimana memori saya digunakan? Folder mana yang mengambil sebagian besar memori?" (orang lain telah menyediakan itu). Tetapi saya biasanya melakukan hal berikut karena ruang kosong menyusut.

\n\n
    \n
  1. Hapus riwayat browser
    2. \n
  2. sudo rm -rf /Library/Caches/*
    3. \n
  3. rm -rf ~/Library/Caches/*
    4. \n
\n\n

~/Library/Caches/ tampaknya mengakumulasi banyak cruft dari waktu ke waktu. Jika Anda belum pernah membersihkannya, maka Anda harus memulihkan gigabyte.

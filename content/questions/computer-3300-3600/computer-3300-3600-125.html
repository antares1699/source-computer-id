Saya baru-baru ini mulai menggunakan Linux dan dibuat kagum oleh kemudahan di mana jendela dapat dipindahkan dan diubah ukurannya dengan kombinasi keyboard sederhana menggunakan Compiz. Jika saya ingin memindahkan terminal ke kanan atas layar, CTRL + ALT + Numpad9 sederhana dapat membantu saya, dan mengubah ukuran jendela pada saat yang bersamaan. Saya yakin kalian tahu apa yang saya bicarakan.

Jika Anda menggunakan Windows versi lama, standby lama adalah Alt + Ruang untuk mendapatkan menu sistem, M dan kemudian tombol panah untuk memindahkan jendela. Jika Anda dimaksimalkan, pertama-tama Anda perlu memukul R pada menu sistem, yang akan membuat Anda tidak maksimal. Tidak seanggun, tetapi bekerja dalam keadaan darurat.

Saya memiliki laptop Windows 8, saya akan segera meningkatkan ke Windows 8.1. Versi standar 8.1 memungkinkan enkripsi disk jika PC Anda memiliki chip TPM.

\n\n

Bagaimana cara mendeteksi chip ini di PC Windows 8 saya?

","source":"https://superuser.com/questions/658133/how-to-verify-the-presence-of-a-tpm-trusted-platform-module-in-my-laptop","author":"Scandalist","replies":[],"answers":[{"answer":"

TPM mungkin tidak muncul di Device Manager jika dinonaktifkan oleh firmware.

\n\n

Masuk ke firmware (bahasa sehari-hari BIOS) sebelum mem-boot Windows (biasanya F2, F10 di layar logo pabrikan sebelum Windows melakukan boot) dan cari opsi TPM apa pun.

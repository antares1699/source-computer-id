Saya baru saja memutakhirkan dari fedora 23 ke 24, dan itu merusak beberapa binari yang saya miliki di sistem saya yang tidak diinstal dari repositori fedora, karena peningkatan ncurses dari versi 5 ke versi 6: 

error while loading shared libraries: libtinfo.so.5: cannot open shared object file: No such file or directory\n

Saya bertanya-tanya apakah mungkin untuk menginstal secara paralel ncurses5 & 6 di fedora 24 jadi saya dapat tetap menggunakan binari ini? Setelah semua itu akan menjadi inti dari memiliki versi postfixes ini, instalasi paralel kan? Tapi saya tidak melihat paket ncurses5 fedora 24 ...? Jadi .. mungkinkah?

","source":"https://superuser.com/questions/1093042/parallel-install-ncurses-6-ncurses-5-on-fedora-24","author":"Emmanuel Touzery","replies":[],"answers":[{"answer":"
\n\n

Harap selalu menggunakan pencarian file: # dnf provides */libtinfo.so.5

\n\n

.... dan jawabannya adalah ncurses-compat-libs-6.0

\n\n

Instal: # dnf install ncurses-compat-libs

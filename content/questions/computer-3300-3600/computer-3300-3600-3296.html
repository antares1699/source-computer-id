Word tidak memiliki fungsi bawaan untuk membalik teks, memutarnya 180 derajat, atau menghasilkan teks gambar cermin. Solusi sederhana adalah dengan memilih teks dan kotaknya dan menyalin ke clipboard kemudian menempelkannya ke aplikasi grafik, seperti Windows Paint. Di sana, terdapat alat untuk membalik atau memutar teks sesuai kebutuhan. Sisi negatifnya adalah teks menjadi tidak dapat diedit lagi. Pendekatan ini harus bekerja di semua versi Word.

Ada cara resmi yang direkomendasikan oleh Microsoft untuk memutar kotak teks dalam versi sebelum Microsoft Office 2010.

\n\n
    \n
  1. Pilih kotak teks.
    2. \n
  2. Tekan Ctrl + X (potong)
    3. \n
  3. Pilih "Tempel Spesial", lalu tempel sebagai PNG.
    4. \n
  4. Sekarang Anda dapat memutarnya seperti gambar (karena sekarang diubah menjadi gambar.)
    5. \n
\n\n

\n\n
\n

Sekarang Anda dapat memutarnya dengan menyeret titik hijau.

\n
\n\n

Mengubahnya menjadi PNG adalah satu-satunya cara untuk memutar kotak teks. (Dengan mengubah arah teks, Anda tidak dapat secara tepat menentukan derajat rotasi.)

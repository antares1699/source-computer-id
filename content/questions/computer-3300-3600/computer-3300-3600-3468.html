Solusi di Youtube

\n\n

Jika pembuat vid akan menghapusnya, unggah kembali vid ini ke Bitchute.com (: D) atau baca instruksi versi teks:

\n\n
    \n
  1. Ketik di bilah alamat about:profiles
    2. \n
  2. Bukalah Direktori ROOT dengan mengklik "tombol Buka direktori". Tidak Direktori Lokal.
    3. \n
  3. Buat direktori "chrome" di sini dengan perintah mkdir chrome di terminal atau melalui tombol klik kanan mouse Anda.
    4. \n
  4. Buka folder yang Anda buat dan buat file css di sini dengan menamainya "userContent.css"
    5. \n

  5. Tempel kode ini ke file "userContent.css" Anda:

    \n\n

    @ -moz-document url ("about: newtab") {\n tubuh {\n background-color: # 000000! penting;\n }\n}

    6. \n

  6. Restart browser Anda secara manual atau dengan mengklik tombol "Restart normal ..." pada alamat about:profiles.

    7. \n
\n\n

Anda akan mendapatkan hasil yang Anda butuhkan! :)

Apakah ada perintah unix untuk menyalin konten direktori ke direktori lain, tetapi selalu menyimpan file terbaru?

\n\n

Maksud saya, memiliki direktori A dan direktori B, saya perlu menyalin konten B ke A. Jika nama file di direktori B sudah ada di A, simpan yang terbaru, sesuai dengan tanggal pembuatannya.

Anda dapat menggunakan rsync untuk menyinkronkan konten dari dua direktori. Opsi -u/ --update memiliki efek

\n\n
\n

lewati file yang lebih baru di receiver

\n
\n\n

Jadi Anda bisa menggunakan rsync -auvh dir1/ dir2/

\n\n

-a = mode arsip. Mengaktifkan seluruh pemuatan opsi seperti rekursi.

\n\n

-v dan -h adalah opsi verbose dan dapat dibaca manusia. Tidak penting, tapi berguna untuk melihat apa yang sedang terjadi.

\n\n

Saya sarankan Anda menjalankan perintah rsync terlebih dahulu dengan opsi â€”dry-run sehingga Anda dapat memeriksa hasil yang tidak terduga.

Anda bisa menggunakan cp biasa, dengan opsi ini: 

-u, --update\n       copy only when the SOURCE file is newer than the destination file\n       or when the destination file is missing\n

Jika ada file baru yang tersebar di setiap direktori, Anda dapat menjalankan cp sekali dari setiap direktori, hanya menyalin file yang lebih baru dari yang lebih lama.

Buka pengelola tugas (di Windows 7, CTRL-ALT-DEL) dan buka tab proses. Temukan proses bernama "plugin-container.exe" dengan deskripsi "Plugin Container for Firefox" dan akhiri proses tersebut dan lihat apakah Firefox mulai berfungsi dengan baik.

\n\n

Saya telah menemukan ini adalah cara yang relatif dapat diandalkan untuk mendapatkan FF lama dan berjalan kembali tanpa harus memulai semuanya dari awal. Anda mungkin perlu memuat ulang tab yang terbuka ke situs video dan halaman intensif plugin lainnya, tetapi jika tidak, semuanya akan baik-baik saja.

\n\n

Ini juga merupakan petunjuk bahwa masalahnya ada pada satu (atau lebih) plugin Anda. Anda setidaknya harus memeriksa untuk memastikan Anda memiliki versi Java dan Flash terkini yang diinstal untuk FF. Gunakan situs ini untuk memeriksa status terbaru plugin Anda: https://www.mozilla.org/en-US/plugincheck/

\n\n

Mengenai Linux. Saya tidak yakin apakah mereka menggunakan nama yang dapat dieksekusi yang sama untuk wadah plugin, dan saya belum memeriksa perbaikan ini di sana.

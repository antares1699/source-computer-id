Bagaimana cara memeriksa apakah tiga karakter terakhir dari variabel cocok dengan variabel lain?

\n\n

Gunakan sintaks substring variabel untuk mengekstrak 3 karakter terakhir dari setiap variabel.

\n\n

Contoh file batch: 

@echo off\nsetlocal\nset /p var1=\"UserInput: \"\nset var1last3=%var1:~-3%\nset var2=file.log\nset var2last3=%var2:~-3%\nif %var1last3% == %var2last3% (\n  echo Match\n  ) else (\n  echo No Match\n  )\nendlocal\n
\n\n

Mengapa ini bukan ide yang bagus?

","source":"https://superuser.com/questions/305806/irssi-list-considered-harmful","author":"kzh","replies":[],"answers":[{"answer":"

Pada jaringan besar, perintah LIST dapat menyebabkan sejumlah besar data dibuang ke pengguna (saat menulis posting ini: ~ 820 kB di DALnet, ~ 1,7 MB di Freenode). Di masa lalu, ketika dial-up 33 kbps dianggap cepat, ini dapat menyebabkan pengguna terputus dari IRC, mungkin karena buffer overflow keluaran ("SendQ terlampaui") atau tidak membalas PING tepat waktu ("Ping timeout ").

\n\n

FWIW, saya tidak pernah mengalami masalah seperti itu bahkan saat menggunakan DSL 256 kbps, jadi peringatan itu bisa dianggap usang.

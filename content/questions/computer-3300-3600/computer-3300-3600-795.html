Jika Anda hanya menginginkan konten, bukan jendela browser (dan bilah alatnya) itu sendiri, maka Paparazi mungkin cukup berguna. Ini tidak hanya memiliki GUI sendiri, tetapi juga memiliki Sintaks URL yang mendukung pemanggilan dari browser apa pun, termasuk menyetel ukuran: 

paparazzi:(maxwidth=1000,maxheight=700)http://www.google.com/

Ubah ini menjadi bookmarklet seperti ini: 

javascript:void(window.location.href='paparazzi:(maxwidth=1000,maxheight=700)'+window.location.href)

Di Safari Anda dapat menetapkannya ke salah satu dari 9 tempat pertama di folder bookmark, dan karenanya secara otomatis memberikannya sesuatu seperti pintasan keyboard Command-1. Perhatikan bahwa browser apa pun yang Anda gunakan, tangkapan layar yang sebenarnya akan selalu dibuat menggunakan WebKit (seperti yang dilakukan Safari dan Chrome, tetapi berbeda dari Firefox Tokek ).

\n\n

Paparazzi juga mendukung beberapa AppleScript. Atau jika Anda lebih suka baris perintah, gunakan webkit2png : 

python webkit2png -W 1000 -H 700 http://google.com/


\n\n

Jika Anda juga ingin memasukkan jendela peramban itu sendiri, saya ragu ada alat khusus peramban untuk itu, jadi Anda harus menggunakan program tangkapan layar . Untuk mengubah ukuran jendela browser untuk itu, saya menggunakan bookmarklet berikut (yang, untuk beberapa browser mungkin berlaku untuk format dalam, tetapi untuk Safari dan Firefox mengacu pada ukuran luar): 

javascript:void(window.resizeTo(1000,700))

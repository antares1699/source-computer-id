Saya tahu saya dapat menemukan direktori yang terakhir diubah dengan menggunakan ls -ltr, tetapi item terakhir yang dimodifikasi mungkin berupa file, bukan direktori. Kedua saya masih harus mengetik nama direktori. Nama direktori saya seperti kode hash, yang tidak benar-benar dapat dibaca dan sangat sulit untuk diketik setiap kali.

\n\n

Adakah cara yang mudah untuk melakukannya?

","source":"https://superuser.com/questions/406176/bash-how-to-enter-a-subdirectory-thats-most-recently-modified","author":"Vendetta","replies":[],"answers":[{"answer":"
cd \"$(ls -dt */ | head -1)\"\n

Simpan ini sebagai alias di ~/.bashrc, mis. 

alias lcd='cd -- \"$(ls -dt -- */ | head -1)\"'\n

dan sekarang lcd di direktori mana pun akan berubah ke direktori yang paling baru diubah di direktori saat ini.

Menambahkan fungsi ini ke ~/.bashrc bekerja untuk saya 

function lcd(){\ncd $(ls -v1td --color=never */ | head -1)\n}\n

Jika saya mencoba untuk mendefinisikannya sebagai alias, ekspresi akan dievaluasi sekali ketika shell dimulai dan selalu mencoba untuk melakukan cd ke subdirektori yang sama di direktori home saya. Tanpa pernyataan --color=never, saya mendapatkan kesalahan -bash: cd: mydir/: No such file or directory yang disebutkan di komentar lain.

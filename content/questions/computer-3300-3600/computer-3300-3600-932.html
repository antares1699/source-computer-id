Kamu akan membutuhkan

\n\n
    \n
  • Perangkat lunak untuk membuat gambar iso (saya menggunakan k3b)
    • \n
  • Perangkat lunak untuk membuat flash drive yang dapat di-boot dari USB (saya menggunakan unetbootin)
    • \n
\n\n

Menciptakan gambar

\n\n

Anda perlu membuat ISO yang dapat di-boot. Anda mungkin harus memilih file boot (untuk Windows 7 dan kemungkinan 8, ini adalah etfsboot.com). Saya menggunakan k3b untuk melakukan ini, tetapi program lain mendukungnya dan, pada kenyataannya, saya menemukan cara melakukan ini sebagian besar berkat panduan untuk perangkat lunak Windows. Inilah yang berhasil untuk saya:

\n\n
    \n
  1. Buka k3b
    2. \n
  2. Buka Proyek Baru> Proyek Data Baru
    3. \n
  3. Seret file Anda dari disk yang belum dibuka ke bagian data di bagian bawah
    4. \n
  4. Klik "Edit Gambar Boot"
    5. \n
  5. Klik Baru ...
    6. \n
  6. Pilih gambar boot - jika gambarnya sama di Windows 8 (dan mungkin memang demikian), gambar itu akan ada di folder boot. saya berpikir file yang Anda butuhkan adalah etfsbootcom. Jangan tergoda oleh bootmgr di root! Bukan yang itu.
    7. \n
  7. Ini akan menanyakan tentang emulasi hard disk. Saya menggunakan "tidak ada emulasi"
    8. \n
  8. Opsi default seharusnya baik-baik saja
    9. \n
  9. Pergi ke Project> Properties dan di tab tulisan centang "Buat Gambar" dan "Hanya Buat Gambar".
    10. \n
  10. Anda perlu mengaktifkan ekstensi UDF untuk default sistem file, tetapi bagi saya ini terjadi secara otomatis ketika saya membiarkannya
    11. \n
  11. Tutup itu dan "bakar" (ingat, Anda benar-benar hanya membuat gambar tetapi Anda masih menggunakan tombol bakar) gambar. Ingatlah untuk mengatur nama file.
    12. \n
\n\n

Memformat flash drive

\n\n

Anda memerlukan flash drive dengan partisi pertama yang diformat ke NTFS dan cukup besar untuk menampung disk Windows (4GiB lebih dari cukup bahkan untuk penginstal Windows 7 32/64 bit universal saya dan seharusnya bagus untuk Windows 8 juga). Saya menggunakan GParted. Inilah yang berhasil untuk saya:

\n\n
    \n
  1. Lepaskan flash drive Anda jika sudah terpasang - Anda mungkin dapat melakukannya dengan mengklik kanan di file manager Anda
    2. \n
  2. Buka GParted
    3. \n
  3. Pilih flash drive Anda dari menu drop-down di kanan atas - kemungkinan besar akan bernama / dev / sd [SURAT]. Periksa apakah ukuran dan partisi cocok! Anda lakukan tidak ingin tidak sengaja menggunakan harddisk Anda.
    4. \n
  4. Kosongkan cukup ruang (Anda cukup mengklik kanan dan "Hapus" atau "Ubah ukuran" partisi, tetapi hati-hati untuk mencadangkan data)
    5. \n
  5. Buat partisi baru (Partition> New). Sistem file harus NTFS, ukurannya cukup besar untuk ISO Anda dan harus menjadi partisi pertama pada drive (setidaknya untuk Windows).
    6. \n
  6. Pasang kembali flash drive Anda (keluarkan / colokkan kembali atau klik di manajer file Anda)
    7. \n
\n\n

Membuat flash drive yang dapat di-boot

\n\n

Anda sebenarnya dapat melakukan ini dengan beberapa cara, tetapi saya menggunakan unetbootin (mungkin yang termudah dan paling ramah pengguna di Linux). Perhatikan bahwa Anda perlu menggunakan versi yang lebih lama karena yang terbaru telah menjatuhkan dukungan untuk flash drive berformat NTFS ( membangun 494 harus bekerja). Inilah yang berhasil untuk saya:

\n\n
    \n
  1. Buka unetbootin (Anda harus menjalankan sebagai root tetapi itu akan meminta Anda memasukkan kata sandi)
    2. \n
  2. Pilih radio button "Diskimage" dan masukkan path ke image ISO Anda
    3. \n
  3. Alihkan opsi untuk jenis drive dari "USB" ke "Harddisk" dan pilih flash drive berformat NTFS Anda - berhati-hatilah untuk tidak memilih hard drive Anda yang sebenarnya!
    4. \n
  4. Klik oke dan tunggu. Setelah selesai, Anda akan memiliki flash drive Windows yang dapat di-boot
    5. \n

Saat ini OS saya adalah Ubuntu 13.10. Saya ingin membuat USB yang dapat di-boot dengan Windows 8 di dalamnya.

\n
\n\n

Anda dapat membuat Drive USB Anda dapat di-boot dengan Windows 8 di dalamnya tanpa menggunakan perangkat lunak tambahan.

\n\n

Yang harus Anda lakukan adalah:

\n\n
    \n
  • Boot ke Ubuntu Anda atau sistem Linux lainnya;
    • \n
  • Format drive USB Anda (lebih suka berukuran 8GB atau lebih) dengan sistem file NTFS;
    • \n
  • Salin semuanya dari bentuk ISO Windows yang diekstrak ke dalam drive USB.
    • \n
\n\n

Setelah penyalinan selesai, berikan perintah berikut di terminal pilihan Anda:

\n\n
    \n
  1. os-prober, itu harus mendeteksi Drive USB Anda sebagai Lingkungan Pemulihan Windows dengan lokasi drive yang sesuai seperti / dev / sdb1 ;
    2. \n
  2. Jika os-prober mendeteksi drive USB Anda, ketik grub-mkconfig -o grub.cfg_location di mana grub.cfg_location adalah lokasi file konfigurasi grub Anda, biasanya adalah /boot/grub/grub.cfg
    3. \n
\n\n

Perintah terakhir akan membuat entri untuk Windows 8 dengan nama Lingkungan Pemulihan Windows di menu Grub Anda. Sekarang Anda dapat me-restart sistem Anda, boot ke menu Grub dan memilih opsi yang sesuai untuk boot ke instalasi Windows 8.

\n\n

Catatan : Anda juga dapat dengan mudah menghapus entri Grub baru ini dengan menjalankan perintah yang sama dengan USB Drive mencabut.

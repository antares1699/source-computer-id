Periksa apakah Anda telah menentukan variabel lingkungan% HOME%.

\n\n

Jika variabel itu ada dan mengacu pada H:\\ (bukan %USERPROFILE%), maka variabel itu akan mencari global git config file pada H:\\.gitconfig (artinya %HOME%\\.gitconfig).

\n\n

Mengubah variabel %HOME% ke jalur lokal seperti %USERPROFILE% akan menyelesaikan masalah itu.

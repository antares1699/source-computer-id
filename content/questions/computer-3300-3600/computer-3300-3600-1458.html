Setelah menginstal VirtualBox, saya perhatikan bahwa komputer saya (Windows 7) membutuhkan waktu yang sangat lama untuk membuat koneksi wifi saat startup.\nSaya menemukan "trik" ini dan saya mencobanya: http://scottiestech.info/2010/12/31/fix-slow-windows-networking-problems-when-virtualbox-is-installed/

\n\n

Menonaktifkan adaptor Jaringan Hanya Host tidak melakukan apa pun. Saat reboot lebih lanjut, masih membutuhkan waktu hampir satu menit penuh untuk masuk ke jaringan wifi saya.

\n\n

Untuk mengonfirmasi bahwa ini terkait VirtualBox, saya benar-benar menghapus VirtualBox dari mesin saya. Setelah uninstall dan reboot, koneksi wifi saya langsung terhubung.

\n\n

Apakah ada hal lain yang dapat saya coba, selain Menonaktifkan Adaptor Jaringan Hanya Host, untuk memperbaiki masalah ini?

Ya itu berhasil, tapi itu bukan perbaikan yang tepat (setidaknya di Windows 7). Host hanya diperlukan untuk folder bersama, dan ini bukan alasan utama jaringan lambat. Alasan jaringan lambat adalah deteksi otomatis yang buruk dari pengaturan jaringan. Anda TIDAK ingin Windows secara otomatis mendeteksi pengaturan ketika Virtualbox diinstal. Pergi ke Panel Kontrol => Properti Internet => tab Koneksi => Pengaturan LAN, dan pastikan semua kotak centang tidak dicentang. Setelah saya melakukan ini dan memulai ulang, perbedaannya adalah siang dan malam!

Ketika saya menekan Start-> Run dan mengetik "Chrome", tidak ada yang terjadi (tidak ada proses "chrome.exe" baru yang diluncurkan, tidak ada kesalahan, tidak ada).

\n\n

Ketika saya menekan Start (tanpa "Run"), ketik Chrome dan jalankan, itu berfungsi.

\n\n

Bersama dengan jalur yang terdaftar oleh Madsen, jalan pintas yang direferensikan terdaftar di Registri.

\n\n

Jalankan regedit, lalu navigasikan ke:

\n\n

Jalur Komputer \\ HKEY_LOCAL_MACHINE \\ SOFTWARE \\ Microsoft Windows \\ CurrentVersion \\ App

\n\n

(ini mungkin sedikit berbeda pada 7, tapi saya ragu).

\n\n

Anda perlu memastikan ada entri (Default) dan Jalan untuk chrome.exe di sini, dengan bidang data (untuk keduanya) menjadi jalur file tempat program berada.

Sejauh yang saya tahu, uTorrent mendukung organisasi di direktori jika Anda menerapkan label sebelum penyelesaian torrent. Namun, jika Anda menambahkan / mengubah label torrent Anda setelah penyelesaiannya, uTorrent tidak akan memindahkan file ke direktori baru. Ini membuat pengelolaan ruang direktori uTorrent Anda cukup sulit.

\n\n

Skrip python ini dapat membantu Anda mengatur ulang torrent setelah mengubah label: https://github.com/m000/cliutils/blob/master/bin/utorrent_tidy.py . Satu-satunya ketergantungan skrip adalah versi terbaru Python dan modul Python bencode.

\n\n

Ini berfungsi seperti ini (dengan asumsi Anda menggunakan Mac - Windows harus langsung ke kanan): 

# uTorrent must be stopped!\nkillall uTorrent\n\n# Rewrite resume.dat and move files using the script.\ncp ~/Library/Application\\ Support/uTorrent/resume.dat .\n./utorrent_tidy.py resume.dat resume2.dat\ncp resume2.dat ~/Library/Application\\ Support/uTorrent/resume.dat\n\n# restart uTorrent\nopen /Applications/uTorrent.app\n

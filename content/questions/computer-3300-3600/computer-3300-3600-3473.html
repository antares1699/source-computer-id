Dalam jawaban saya yang lain, saya tidak memberikan kelonggaran untuk poin terakhir yang bergerak (yaitu, menambahkan poin data). Bukan masalah besar untuk menyesuaikan data Anda untuk memungkinkan satu label bergerak.

\n\n

Saya hanya menampilkan satu seri data, tetapi cukup mudah untuk melakukan sebanyak yang Anda butuhkan, setelah Anda mengetahui triknya. Dalam kumpulan data di bawah ini, saya memiliki kolom Tanggal, dan dua kolom dengan tajuk yang sama, Nilai. Kolom nilai pertama memiliki nilai di tengah-tengah Rentang tanggal. Kolom kedua memiliki rumus yang hanya menampilkan nilai terakhir. Ini rumus di sel C2: 

=IF(AND(ISNUMBER(B2),LEN(B3)=0),B2,NA())\n

Rumus ini disalin ke bawah kolom.

\n\n

Sekarang buat bagan dari semua data. Seri nilai pertama adalah lingkaran dan garis biru, yang kedua hanya lingkaran oranye pada titik biru terakhir. Saya telah menambahkan label ke seri oranye, bukan yang biru.

\n\n

\n\n

\n\n

Jika Anda memilih keseluruhan rangkaian, Excel akan memberi label pada setiap titik dalam rangkaian.

\n\n

Pilih hanya satu titik yang ingin Anda beri label: klik sekali untuk memilih rangkaian, lalu klik lagi untuk memilih satu titik. Sekarang ketika Anda menggunakan menu klik kanan atau ikon tanda plus untuk menambahkan label data, itu akan menambahkan label hanya pada satu titik yang Anda pilih.

\n\n

Untuk pendekatan VBA yang hanya menambahkan label ke titik terakhir dan tidak meninggalkan banyak label "" di bagan, berikut adalah prosedur sederhana dari tutorial saya Beri Label Titik Terakhir untuk Excel 2007 . Ini secara khusus memberi label titik terakhir dari setiap seri di bagan aktif dengan nama seri. 

Sub LastPointLabel()\n  Dim mySrs As Series\n  Dim iPts As Long\n  Dim vYVals As Variant\n  Dim vXVals As Variant\n\n  If ActiveChart Is Nothing Then\n    MsgBox \"Select a chart and try again.\", vbExclamation\n  Else\n    Application.ScreenUpdating = False\n    For Each mySrs In ActiveChart.SeriesCollection\n      With mySrs\n        vYVals = .Values\n        vXVals = .XValues\n        ' clear existing labels\n        .HasDataLabels = False\n        For iPts = .Points.Count To 1 Step -1\n          If Not IsEmpty(vYVals(iPts)) And Not IsError(vYVals(iPts)) _\n              And Not IsEmpty(vXVals(iPts)) And Not IsError(vXVals(iPts)) Then\n            ' add label\n            mySrs.Points(iPts).ApplyDataLabels _\n                ShowSeriesName:=True, _\n                ShowCategoryName:=False, ShowValue:=False, _\n                AutoText:=True, LegendKey:=False\n            Exit For\n          End If\n        Next\n      End With\n    Next\n    ' legend is now unnecessary\n    ActiveChart.HasLegend = False\n    Application.ScreenUpdating = True\n  End If\nEnd Sub\n

Saya ingin menyarankan Anda kedua opsi. Pertama adalah Non-VBA dan lainnya adalah Kode VBA.

\n\n

Solusi Non-VBA:

\n\n

Karena Excel tidak memiliki pengaturan apa pun untuk menetapkan hanya Satu Label Data, jika rangkaian Sel terlibat untuk membuat Bagan.

\n\n

Tetapi setelah Bagan dibuat, Anda dapat Mengatur Ulang Label.

\n\n

Dalam kasus Anda, setelah Label diterapkan, Klik Kanan Garis, Anda menemukan Label siap untuk Diedit. Pilih Label satu per satu, lalu Klik Kanan & Hapus atau hapus centang pada Kotak Centang Nilai di sebelah Area Bagan.

\n\n

Solusi VBA:

\n\n

Buat satu tombol Command dan masukkan kode ini.

\n\n

Ingat, Anda cukup membuat Bagan tetapi tidak menerapkan Label Data. Setelah semuanya siap maka Klik tombol Command. Kode pertama-tama akan membuat semua Label Data akhirnya menyimpan SATU TERAKHIR.

\n\n

Perhatikan bahwa Anda dapat menggunakan kode ini sebagai JALANKAN MAKRO meskipun tanpa Tombol Perintah. 

Sub LastDataLabel()\n\nDim ws As Worksheet\nDim chrt as Chart\nDim srs as Series\nDim pnt as Point\nDim p as Integer\n\nSet ws = ActiveSheet\nSet chrt = ws.ChartObjects(\"Line Chart\")\nSet srs = chrt.SeriesCollection(1)\n\n    srs.ApplyDataLabels\n\n    For p = 1 to srs.Points.Count - 1 \n        Set pnt = srs.Points(p)\n\n        p.Datalabel.Text = \"\"\n    Next\n\n    srs.Points(srs.Points.Count).DataLabel.Format.TextFrame2.TextRange.Font.Size = 10\n\n\nEnd Sub\n

Semoga ini membantu Anda .

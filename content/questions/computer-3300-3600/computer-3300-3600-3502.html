Saya menggunakan Windows 7. Sejak beberapa minggu, perilaku "Simpan sebagai" berubah di semua program MS office yang saya gunakan (Excel, Word, Project). Perilaku sekarang adalah:

\n\n
    \n
  • Saya membuka dokumen mengatakan D: \\ dir1 \\ subdir1 \\ subdir2 ... \\ subdir5 \\ Document_old.doc atau D: \\ dir2 \\ subdir6 \\ subdir7 ... \\ subdir10
    • \n
  • Saya membuat perubahan dan ingin menyimpannya sebagai document_new.doc di folder yang sama
    • \n
  • Saya klik simpan sebagai dan dapatkan, jadilah default, pustaka Dokumen saya sebagai jalur penyimpanan
    • \n
\n\n

Perilaku yang saya inginkan sebagai default adalah menyimpan dokumen yang diubah di folder yang sama seperti aslinya.

\n\n

Ada beberapa solusi yang mungkin, seperti memiliki favorit di penjelajah sehingga saya menemukan jalur target saya lebih mudah. Karena penanganan dokumen yang kami gunakan di tempat kerja, ini tidak praktis lebih jauh daripada subdir1 dalam contoh saya, jadi saya masih memiliki banyak klik yang harus dilakukan. Saya melihat di bawah 'opsi' di dalam excel dan tidak menemukan cara untuk mengubah perilaku simpan-sebagai.

\n\n

Untuk lebih jelasnya, memiliki satu lokasi penyimpanan default, di mana pun, tidak menyelesaikan masalah saya karena saya bekerja di banyak direktori setiap saat.

\n\n

Sebaliknya, xchange viewer memungkinkan saya memilih "Dir yang berisi dokumen" sebagai lokasi simpan-sebagai ... 'default.

\n\n

Bagaimana saya bisa mengubah perilaku kembali untuk menyimpan di dir yang berisi dokumen secara default?

Seperti yang disarankan @DavidPostill, beri titik "." di bidang "Lokasi file lokal default" tampaknya berfungsi. Excel kemudian menggunakan direktori kerja saat ini, yang akan menjadi tempat terakhir Anda membuka atau Disimpan Sebagai .. buku kerja.

Bagaimana cara mengubah lokasi penyimpanan default?

\n\n
\n

Excel, PowerPoint, Project, dan Word 2016 :

\n \n
    \n

  1. Klik tab "File", lalu klik "Options".

    2. \n

  2. Klik "Simpan".

    3. \n

  3. Di bawah "Simpan buku kerja", "Simpan presentasi", "Simpan proyek", atau "Simpan dokumen", ketikkan jalur di "Lokasi file lokal default"\n kotak.

    4. \n
\n
\n\n

Sumber Tetapkan folder kerja default

\n\n
\n

Word 2013 :

\n \n
    \n

  1. Klik tab "File", lalu klik "Options".

    2. \n

  2. Klik "Simpan".

    3. \n

  3. Temukan dan centang kotak "Simpan ke Komputer secara Default".

    4. \n

  4. Di bawah "Simpan dokumen", ketik jalur di kotak "Lokasi file lokal default".

    5. \n
\n \n

\n
\n\n

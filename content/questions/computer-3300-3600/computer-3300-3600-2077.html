Putty dan WinSCP menggunakan protokol SSH2 dasar yang sama, tingkat keamanan sebagian besar ditentukan oleh protokol ini.

\n\n

Kedua program tidak benar-benar sebanding, WinSCP adalah klien Secure Copy (SCP), Putty hadir dengan padanan yang disebut pscp, tetapi pscp adalah klien baris perintah bukan klien GUI. Putty mungkin terutama digunakan untuk akses shell jarak jauh - memasukkan perintah (seperti yang Anda sebutkan).

\n\n

Mungkin tidak ada alasan mengapa Anda tidak dapat menggunakan Putty dan WinSCP. Anda bahkan dapat membagikan kunci pribadi antara kedua aplikasi.

\n\n

Saya tidak tahu ada masalah keamanan besar saat ini dengan Putty, atau dengan WinSCP.

Saya tahu SQL cukup baik untuk melakukan kueri dasar dan memanipulasi data yang dibagi menjadi beberapa tabel dalam database. Saya mengetahui Excel 2007 sedikit lebih baik dan sering menggunakannya untuk mengelola proyek teknik mesin saya.

\n\n

Terkadang di Excel saya ingin memanipulasi data dari beberapa spreadsheet dan menggabungkannya menjadi satu tabel - tugas yang cocok untuk kueri SQL. Biasanya ketika saya menghadapi masalah seperti ini, reaksi langsung saya adalah mulai membuat pertanyaan di kepala saya. Sayangnya, sering kali kueri SQL tidak mudah diterjemahkan ke fungsi Excel dan ini menghentikan langkah saya.

\n\n

Apakah mungkin menggunakan SQL untuk fungsi-fungsi ini di dalam Excel?

Anda dapat menggunakan ADO dengan Excel VBA. Sebagai contoh: 

Dim cn As ADODB.Connection\nDim rs As ADODB.Recordset \n\n''This is just a convenient name to test, it would probably be\n''better to use the full file name eg C:\\Docs\\XL.xls\nstrFile = Workbooks(1).FullName\n\n''For ACE see: http://www.connectionstrings.com/excel-2007\nstrCon = \"Provider=Microsoft.Jet.OLEDB.4.0;Data Source=\" & strFile _\n    & \";Extended Properties=\"\"Excel 8.0;HDR=Yes;IMEX=1\"\";\"\n\nSet cn = CreateObject(\"ADODB.Connection\")\nSet rs = CreateObject(\"ADODB.Recordset\")\n\ncn.Open strCon\n\n''Pick one:\nstrSQL = \"SELECT * FROM DataTable\" ''Named range\nstrSQL = \"SELECT * FROM [Sheet1$A1:E346]\" ''Range\nstrSQL = \"SELECT * FROM [Sheet1$]\" ''All the data in a sheet\nstrSQL = \"SELECT * FROM [Excel 8.0;HDR=YES;IMEX=1;\" _\n       & \"database=C:\\Docs\\LTD.xls].[Sheet1$]\" ''Refer to second workbook\n\nrs.Open strSQL, cn\n\n''Write a recordset to a sheet\nWorksheets(\"Sheet3\").Cells(2, 1).CopyFromRecordset rs\n

Kueri dapat menggunakan apa pun yang dapat diterima di Jet SQL:

\n\n

\n Intermediate Microsoft Jet SQL for Access 2000
\n Advanced Microsoft Jet SQL for Access 2000

\n\n

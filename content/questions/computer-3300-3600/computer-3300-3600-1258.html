Apa artinya yang ada untuk mengakses dan mengubah opsi daya dan prosesor yang tersembunyi di Windows 10?

\n\n
    \n
  • Cara untuk menemukan informasi tentang daya dan opsi prosesor pengenal unik global (GUID) yang tersembunyi.
    • \n
  • Sebuah cara untuk mengkonfigurasinya untuk tujuan pengujian.
    • \n
\n\n
\n\n

Perintah CMD yang ditinggikan berikut hanya menampilkan GUID yang terlihat dari jendela opsi daya: 

powercfg /q\n

\n\n

Sepertinya ada perbedaan dengan ekspor powercfg normal di internet dan konfigurasi tweak posting dan posting powercfg penuh .\nSecara umum kurang lengkap dokumentasi tentang hal-hal ini, fakta bahwa Microsoft PowerCfg Dokumentasi tidak menjelaskan hal ini, itulah alasan dari pertanyaan ini. Ini juga merupakan perpanjangan dari jawaban parkir inti sini .

Meninjau Opsi GUID Daya Tersembunyi di Windows 10

\n\n
    \n

  1. Jalankan cmd yang ditinggikan, bandingkan hasil dengan mengekspor: 

    powercfg /Q >%UserProfile%\\Desktop\\PowerPlanSettings.txt\n\npowercfg /Qh >%UserProfile%\\Desktop\\PowerPlanSettingsHidden.txt\n
    2. \n

  2. Temukan informasi lebih lanjut tentang GUID tersembunyi yang dimaksud dengan mencari di registri.

    3. \n
\n\n

\n\n
    \n
  1. Untuk memperlihatkan GUID menggunakan format berikut di cmd yang ditinggikan, pastikan untuk menggunakan judul sub grup juga, ini akan membuatnya terlihat di jendela opsi daya, ganti sub grup dan GUID dengan yang dari log / Qh:
    2. \n
\n\n

Tata Letak Perintah: 

    powercfg -attributes SUB-GROUP GUID -ATTRIB_HIDE\n

Contoh Perintah: 

    powercfg -attributes SUB_PROCESSOR 8baa4a8a-14c6-4451-8e8b-14bdbd197537 -ATTRIB_HIDE\n
    \n
  1. Untuk menyembunyikan GUID, gunakan perintah format berikut di cmd yang ditinggikan. Ganti sub grup dan GUID lagi.
    2. \n
\n\n

Tata Letak Perintah: 

    powercfg -attributes SUB-GROUP GUID +ATTRIB_HIDE      \n

Contoh Perintah: 

    powercfg -attributes SUB_PROCESSOR 8baa4a8a-14c6-4451-8e8b-14bdbd197537 +ATTRIB_HIDE\n
    \n
  1. Buat daftar semua GUID konfigurasi daya yang mungkin di Windows menggunakan Powershell yang ditinggikan:
    2. \n
\n\n
# List all possible power config GUIDs in Windows\n# Run: this-script.ps1 | Out-File powercfg.ps1\n# Then edit and run powercfg.ps1\n# (c) Pekka \"raspi\" JÃ¤rvinen 2017\n
$powerSettingTable = Get-WmiObject -Namespace root\\cimv2\\power -Class Win32_PowerSetting\n$powerSettingInSubgroubTable = Get-WmiObject -Namespace root\\cimv2\\power -Class Win32_PowerSettingInSubgroup\n\nGet-WmiObject -Namespace root\\cimv2\\power -Class Win32_PowerSettingCapabilities | ForEach-Object {\n  $tmp = $_.ManagedElement\n  $tmp = $tmp.Remove(0, $tmp.LastIndexOf('{') + 1)\n  $tmp = $tmp.Remove($tmp.LastIndexOf('}'))\n\n  $guid = $tmp\n\n  $s = ($powerSettingInSubgroubTable | Where-Object PartComponent -Match \"$guid\")\n\n  if (!$s) {\n    return\n  }\n\n  $tmp = $s.GroupComponent\n  $tmp = $tmp.Remove(0, $tmp.LastIndexOf('{') + 1)\n  $tmp = $tmp.Remove($tmp.LastIndexOf('}'))\n\n  $groupguid = $tmp\n\n  $s = ($powerSettingTable | Where-Object InstanceID -Match \"$guid\")\n\n  $descr = [string]::Format(\"# {0}\", $s.ElementName)\n  $runcfg = [string]::Format(\"powercfg -attributes {0} {1} -ATTRIB_HIDE\", $groupguid, $guid)\n\n  Write-Output $descr\n  Write-Output $runcfg\n  Write-Output \"\"\n\n}\n

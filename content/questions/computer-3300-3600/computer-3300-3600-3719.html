Saya baru saja membeli keyboard baru (Inca IWS-549).\nKeyboard memiliki gaya yang bagus dan 18 tombol multimedia.\nTetapi di antara mereka di sebelah kiri, saya memiliki tombol shutdown (tepat di atas tombol esc). Jika saya menyentuhnya, sistem akan mati dan sangat sensitif. Saya ingin menonaktifkan kunci ini. Bagaimana saya bisa melakukan ini?

\n\n

Saya mencoba menggunakan keytweak, tetapi pertama-tama saya harus menekan tombol itu untuk mengubahnya yang tidak mungkin karena sistem mati setiap kali saya mencobanya.

\n\n

Ada ide?

\n\n

Saya menggunakan windows 7.

","source":"https://superuser.com/questions/816088/disable-shutdown-key-on-my-keyboard","author":"hullabaloon","replies":[],"answers":[{"answer":"

Buka Panel Kontrol> Opsi Daya> Lanjutan> Saat Saya Menekan Tombol Daya di Komputer Saya

\n\n

\n\n

Pilih "Tanya saya apa yang harus dilakukan" dan sekarang ketika tombol itu ditekan, Anda mendapatkan opsi matikan komputer.

Di bawah Setup> Config, lihat Aturan Pemangkasan hampir sampai ke bawah.

\n\n

Dari bantuan dalam alpine:

\n\n
\n

OPSI: Aturan Pemangkasan

\n \n

Secara default, Alpine akan menanyakan pada awal setiap bulan apakah atau tidak\n Anda ingin mengubah nama folder email terkirim Anda menjadi nama seperti\n dikirim-surat-bulan-tahun. (Lihat fitur Pangkas Menggunakan YYYY-MM untuk mengubah\n format folder ke sent-mail-yyyy-mm.) Ini juga akan menanyakan apakah Anda\n ingin menghapus folder surat terkirim lama. Jika Anda sudah mendefinisikan Baca Pesan\n Folder atau Pruned Folder Alpine juga akan menanyakan tentang pemangkasan folder tersebut.

\n \n

Dengan opsi ini Anda dapat memberikan jawaban otomatis untuk pertanyaan-pertanyaan ini. Itu\n nilai defaultnya adalah menanyakan apa yang ingin Anda lakukan.

\n \n

Enam kemungkinan nilai untuk opsi ini adalah:

\n \n

bertanya tentang mengganti nama, bertanya tentang menghapus
Ini adalah defaultnya. Alpine akan menanyakan apakah Anda ingin mengganti nama file\n folder dan apakah Anda ingin menghapus setiap folder lama.

\n \n

tanyakan tentang ganti nama, jangan hapus
Alpine akan menanyakan apakah Anda ingin mengganti nama folder, tetapi tidak akan bertanya\n tentang atau menghapus folder lama.

\n \n

selalu ganti nama, tanyakan tentang menghapus
Ini berarti Anda ingin selalu menjawab ya dan memiliki Alpine secara otomatis\n ganti nama folder jika memungkinkan. Anda juga akan ditanya tentang penghapusan\n folder lama.

\n \n

selalu ganti nama, jangan hapus
Ini berarti Anda ingin selalu menjawab ya dan memiliki Alpine secara otomatis\n ganti nama folder jika memungkinkan. Tidak akan ada penghapusan yang lama\n folder.

\n \n

jangan ganti nama, tanyakan tentang menghapus
Artinya Anda ingin selalu menjawab tidak. Alpine tidak akan mengganti nama file\n map. Anda akan ditanya tentang menghapus folder lama.

\n \n

jangan ganti nama, jangan hapus
Artinya Anda ingin selalu menjawab tidak. Alpine tidak akan mengganti nama file\n map. Folder lama juga tidak akan dihapus.

Instal Python 3.2.2 (CentOS 6.5 AMD64):

\n\n

Untuk mengkompilasi Python, Anda harus terlebih dahulu menginstal alat pengembangan dan beberapa\nlib ekstra. Lib ekstra tidak benar-benar diperlukan untuk mengkompilasi Python tetapi tanpa\nmereka penerjemah Python baru Anda akan sangat tidak berguna 

sudo yum groupinstall \"Development tools\"\nsudo yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel\n

Berikut adalah perintah untuk mengunduh, mengkompilasi dan menginstal Python 

cd /usr/local/src\nsudo wget http://python.org/ftp/python/3.2.2/Python-3.2.2.tar.xz --no-check-certificate\nsudo tar xf Python-3.2.2.tar.xz\ncd Python-3.2.2\nsudo ./configure --prefix=/usr/local --enable-shared LDFLAGS=\"-Wl,-rpath /usr/local/lib\"\nsudo make && sudo make altinstall\n

Setelah menjalankan perintah di atas, interpreter Python yang baru Anda instal akan\ntersedia sebagai /usr/local/bin/python3.2 

/usr/local/bin/python3.2\n

Unduh dan instal Setuptools + pip 

cd /usr/local/src\nsudo wget https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py\nsudo /usr/local/bin/python3.2 ez_setup.py\nsudo /usr/local/bin/easy_install-3.2 pip\n

Buat lingkungan Python 3.2 Anda yang terisolasi 

sudo /usr/local/bin/pip3.2 install virtualenv\nsudo /usr/local/bin/virtualenv /usr/local/virtenv3.2\ncd /usr/local/virtenv3.2\nsource bin/activate\npython --version # (To check version)\n

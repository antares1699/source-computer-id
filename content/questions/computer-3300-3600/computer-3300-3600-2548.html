Itu adalah masalah koneksi.

\n

Saya ingat bahwa saya bermain dengan konfigurasi proxy secara tidak sengaja seperti ini: 

   npm config set proxy http://localhost:8080/\n   npm config set https-proxy http://localhost:8080/\n   npm config set strict-ssl false\n

yang membuat klien npm mencoba untuk menekan localhost: 8080 untuk menarik modul daripada titik akhir internet yang benar.

\n

jadi setelah beberapa hari frustrasi saya mengunjungi tautan ini

\n

https://docs.npmjs.com/cli/config

\n

lalu lari 

npm config edit\n

yang membuka file di dalam file itu saya menghapus tiga baris yang saya tambahkan di atas, lalu semuanya berfungsi dengan baik.

\n

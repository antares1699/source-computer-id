Saya memiliki iMac dan dua buku Mac. Dapatkah saya membeli hanya satu lisensi OS X Lion baru dan menginstalnya di ketiga komputer? ... Jika ya, apa pedoman menginstal OS di banyak komputer?

","source":"https://superuser.com/questions/313396/number-of-mac-osx-lion-licenses-i-need-purchase-to-install-on-my-three-computers","author":"Tawani","replies":[],"answers":[{"answer":"

Anda hanya perlu membeli Lion sekali untuk menjalankannya di semua komputer Anda. Saat Anda mengunduhnya dari App Store, seharusnya ada aplikasi "Instal Mac OS X Lion" di folder Aplikasi yang dapat Anda salin ke komputer lain. Anda juga dapat mengaksesnya dari App Store di komputer Anda yang lain menggunakan file Membeli tab.

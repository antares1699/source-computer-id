Saya menggunakan find untuk mencari file di mesin saya seperti ini: 

find / -name version-1.0.*\n

dan hasilnya hanya satu file di direktori home saya, version-1.0.23. Namun, jika saya menjalankan yang berikut ini: 

find / -name version-1.0.1*\n

hasilnya adalah /data/somelongpath/version-1.0.19.

\n\n

Mengapa hasil kedua ini dihilangkan dari hasil dalam kasus pertama?

Anda memiliki file yang cocok dengan glob di direktori saat ini, dan shell Anda sedang melakukan globbing. Keluar dari glob. 

find / -name 'version-1.0.*'\n

","source":"https://superuser.com/questions/443982/how-can-i-show-the-current-year-in-ubuntus-clock-indicator-applet","author":"LonnieBest","replies":[],"answers":[{"answer":"

Sehubungan dengan itu menjawab bantu saya mencari tahu, mari kita sesuaikan indikator-datetime. 

gsettings set com.canonical.indicator.datetime time-format \"'custom'\"\ngsettings set com.canonical.indicator.datetime custom-time-format \"'%a, %d. %h  %H:%M %Y'\"\n

Itu akan memberikan output "Sen, 02, Jul 21:00 2012".

\n\n

Anda dapat menyesuaikan format asalkan dipahami oleh strftime. Untuk format lain, Anda dapat melihat halaman manual untuk perintah strftime dan sesuaikan.

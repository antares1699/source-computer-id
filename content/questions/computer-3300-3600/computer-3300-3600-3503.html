Apakah ada cara di WinSCP untuk mengkonfigurasi file known_hosts dengan sidik jari kunci agar tidak ditanyai oleh WinSCP saat Anda ingin menyambungkan?

","source":"https://superuser.com/questions/1215002/known-hosts-file-for-winscp","author":"Mr. Roland","replies":[],"answers":[{"answer":"

WinSCP tidak menggunakan file known_hosts. Seperti PuTTY, ini menyimpan kunci host yang dikenal ke dalam registri Windows (atau secara opsional file INI) 

[HKEY_CURRENT_USER\\SOFTWARE\\Martin Prikryl\\WinSCP 2\\SshHostKeys]\n

Format kunci berbeda dengan known_hosts.

\n

Jika Anda memiliki kunci host yang di-cache di komputer lain, Anda dapat menyalinnya ke komputer baru.

\n
\n

Tapi kamu bisa impor kunci host dari file known_hosts .

\n
\n

Cara lain adalah meminta WinSCP menyimpan situs baru dengan sidik jari yang diketahui dari kunci host, seperti: 

winscp.exe sftp://username;fingerprint=ssh-rsa-xxxxxxxxxxx...=@example.com/;save\n

Lihat https://winscp.net/eng/docs/integration_url#save

Anda belum menentukan arsitekturnya (amd64 atau ia64 atau lainnya?)

\n\n

Jika Anda menggunakan amd64, lib32gcc1 ada dan berisi libgcc_s.so.1. Saya dapat menginstalnya ketika saya mencoba.

\n\n

Jika Anda menggunakan ia64, Anda harus menggunakan ia32-libs-core.

\n\n

Link ini dapat memberi tahu paket yang diperlukan untuk arsitektur Anda.

\n\n

Terkadang mirror Debian kehilangan beberapa paket. Anda dapat mencoba menginstal dari repositori AS utama untuk melihat apakah itu masalahnya.

Saya mengalami masalah yang sama dan mengalami beberapa masalah saat menginstal lib 32-bit di Wheezy 7.6. Saya akhirnya berhasil menyelesaikannya dengan menargetkan libgcc1 32-bit secara eksplisit: 

sudo dpkg --add-architecture i386\nsudo apt-get update\nsudo apt-get install libgcc1:i386\n

Ini akan menambahkan libgcc_s.so.1 ke / usr / lib32 /

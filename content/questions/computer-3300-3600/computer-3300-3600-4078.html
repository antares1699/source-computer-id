Saya telah menggunakan w3m akhir-akhir ini untuk menjelajahi web, tetapi terkadang saya ingin melihat halaman di firefox sebagai gantinya. Apakah ada cara untuk membuka halaman saat ini di w3m di firefox dengan pintasan keyboard?

","source":"https://superuser.com/questions/422692/open-current-page-in-w3m-in-firefox","author":"Idr","replies":[],"answers":[{"answer":"

Sementara di w3m, ketik o untuk membuka Panel Pengaturan Opsi. Gulir ke bawah ke bagian Pengaturan Program Eksternal dan baris Browser Eksternal. Di kolom teks di sebelah kanan, masukkan firefox. Pindah ke kotak [OK] beberapa baris ke bawah dan tekan Enter.

\n\n

Sekarang, jika Anda berada di halaman yang ingin Anda lihat menggunakan Firefox, ketik saja M. Untuk mengikuti link menggunakan Firefox, ketik <Esc>M.

\n\n

Anda juga dapat menambahkan browser eksternal kedua dan ketiga jika Anda ingin menggunakan sesuatu selain Firefox.

Saya ingin mengaktifkan mikrofon di komputer saya melalui baris perintah.\nSaya menggunakan amixer untuk ini. 

amixer set \"Front Mic\" 66%\namixer set \"Digital\" 100%\namixer set \"Front Mic Boost\" 100%\namixer set \"Capture\" 100%\n

Ini mengatur pengaturan volume. Masalahnya adalah penangkapan itu dinonaktifkan. Amixer menampilkan keluaran ini:

\n\n

Kontrol mixer sederhana 'Capture', 0\n Kemampuan: cvolume cswitch\n Tangkap saluran: Kiri Depan - Kanan Depan\n Batas: Tangkap 0 - 54\n Kiri Depan: Tangkap 27 [50%] [-18.00dB] [mati]\n Kanan Depan: Tangkap 27 [50%] [-18.00dB] [mati]

\n\n

Saya dapat menangkap aktif melalui kmix setelah itu amixer menunjukkan saluran seperti pada. Tetapi halaman manual tidak menunjukkan cara mengaktifkan saluran melalui amixer.

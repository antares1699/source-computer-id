Bisakah saya mencadangkan menggunakan FTP?

\n

Tidak. Saat ini, CrashPlan harus diinstal dan dijalankan di tujuan. Kami mungkin mendukung FTP di beberapa titik, tetapi itu akan jauh lebih lambat dan kurang dari penyimpanan terjamin daripada menjalankan CrashPlan.\nDengan CrashPlan berjalan di kedua ujungnya, perangkat lunak tujuan akan mensimulasikan pemulihan dan memvalidasi file saat tujuan menganggur dan tidak digunakan. Ini masalah besar - sesuatu yang tidak dapat Anda lakukan dengan FTP secara efisien - dan memastikan bahwa file Anda dalam kondisi yang baik saat Anda membutuhkannya

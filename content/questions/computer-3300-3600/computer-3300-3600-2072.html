Apakah ada cara untuk memasukkan rentang angka di Sublime Text 2? Saya tidak dapat menemukan perintah atau plugin, dan saya harus memasukkan urutan seperti ini semi-sering. Saya mencari sesuatu yang mirip dengan :put =range(11,15) di Vim, yang akan dicetak 

11\n12\n13\n14\n15\n
","source":"https://superuser.com/questions/473366/insert-a-range-of-numbers-in-sublime-text-2","author":"Ben Jacobs","replies":[],"answers":[{"answer":"

Itu Paket Text Pastry mendukung sintaks dari 'Sisipkan Angka', tetapi menambahkan fitur tambahan, seperti menyisipkan urutan kata dan prasetel untuk pembuat angka 1-indeks dan 0-indeks.

","votes":"5","source":"https://superuser.com/a/598519","author":"AllanLRH","replies":[{"reply":"Juga berfungsi dalam teks luhur 3","author":"Michael"}]},{"answer":"

Saya menemukan sebuah paket (di manajer paket) bernama "Sisipkan Angka." Pilih semua wilayah tempat Anda ingin menempatkan angka, jalankan perintah (CTRL-OPT-n di Mac saya CTRL-Alt-n untuk Win / * NIX) dan Anda akan dapat memasukkan angka. Paket ini memungkinkan Anda untuk menentukan awal, langkah, dan bantalan angka.

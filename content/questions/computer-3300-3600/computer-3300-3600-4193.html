Asalkan Anda memiliki firmware yang tepat, IMO pengaturan terbaik adalah menggunakan DNS Forwarder yang pada dasarnya adalah server nama miniatur untuk host LAN. Firmware router open source seperti Pfsense atau DD-WRT akan memiliki fitur ini. Sebagai contoh, lihat tautan ini untuk info umum tentang Pfsense DNS Forwarder: http://doc.pfsense.org/index.php/DNS_Forwarder .

\n\n

","source":"https://superuser.com/questions/166420/change-file-from-being-flagged-as-a-system-file-back-to-normal-state-on-wind","author":"Duane","replies":[],"answers":[{"answer":"

"File sistem" di Windows 7 adalah file yang diberi tag dengan dua atribut berikut:

\n\n
    \n
  • Arsip
    • \n
  • Sistem
    • \n
\n\n

"File sistem tersembunyi" di Windows 7 adalah file yang diberi tag dengan tiga atribut berikut:

\n\n
    \n
  • Arsip
    • \n
  • Tersembunyi
    • \n
  • Sistem
    • \n
\n\n

Untuk mengatur ulang "File sistem", cukup jalankan perintah berikut: 

attrib -A -S file.ext\n

Untuk mengatur ulang "File Sistem Tersembunyi", jalankan perintah berikut: 

attrib -A -H -S file.ext\n

Jika itu ditandai sebagai file sistem, itu mungkin juga ditandai sebagai file tersembunyi, jadi buka command prompt, kemudian cd C:\\path\\to\\folder dan kemudian attrib -h -s song.mp3. Tanda minus berarti menghapus atribut, 'h' berarti tersembunyi, 's' untuk sistem . Untuk menambahkan atribut, ubah -h menjadi + h.

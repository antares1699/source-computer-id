Anda mungkin harus melihat beberapa hal.

\n\n

Pertama, terkait penggunaan folder bersama, Anda dapat menggunakan Windows Live Sync untuk membuat folder apa pun di laptop Anda (atau di tempat lain) dicerminkan ke folder Pengguna WHS pilihan Anda. Ini akan menyebabkan pesan palsu "Konflik File" karena WHS tidak dapat mereplikasi file terbuka .fslock yang dibuat oleh Live Sync (Saya telah mengajukan laporan bug tentang ini di situs WHS Connect). Namun demikian, idenya adalah bahwa Live Sync akan membuat folder pilihan Anda dicerminkan di mana pun Anda berada. Ini TIDAK akan memberi Anda cadangan historis, hanya versi saat ini.

\n\n

Kedua, tentang backup WHS itu sendiri. Saya merekomendasikan menggunakan add-in BDBB untuk memaksa WHS menduplikasi file database cadangan ke beberapa drive. Ini akan membuat Anda tahan terhadap crash drive utama di server WHS (partisi C: SYSTEM dan D: DATA). Saya telah selamat dari kecelakaan itu dua kali sambil mempertahankan riwayat cadangan lengkap saya.

\n\n

Ketiga, jika Anda menginginkan cadangan reguler server WHS di luar kantor, Anda dapat menggunakan drive USB eksternal dengan ukuran yang memadai. Saat Anda menyambungkannya ke server WHS, Anda dapat memilih untuk menggunakannya untuk pencadangan sistem. Ini akan menyalin semuanya ke drive eksternal dan kemudian memberitahu Anda untuk turun. Sekarang bawa drive itu ke luar kantor. Ini bukan pencadangan berkelanjutan, tetapi lebih baik daripada tidak sama sekali.

\n\n

Keempat, Anda dapat menggunakan salah satu alat luar biasa yang tercantum di posting lain ... Mozy , JungleDisk (kesukaanku), KeepVault dan biarkan semuanya mencadangkan ke internet. Ini bisa jadi agak mahal, dan sangat lambat untuk dipulihkan, tetapi data Anda mahal sangat aman.

\n\n

Terakhir, Anda bisa melakukan hal favorit saya. Menggunakan CrashPlan Anda dapat mencadangkan ke komputer orang lain melalui internet. CrashPlan sangat bagus karena pada dasarnya Anda membuat perjanjian dengan satu atau lebih orang (seperti ayah, saudara, dan saya) yang semuanya menyediakan ruang cadangan untuk program CrashPlan. Ia memampatkan, mengenkripsi, dan menyimpan cadangan saya di semua mesin bersama di "jaringan" orang kita, dan mesin lain di jaringan akan menyimpan barang-barang di mesin saya. Cara saya menggunakannya adalah dengan memiliki hard disk besar yang saya beli untuk ayah dan saudara laki-laki saya, menginstalnya di mesin mereka dan menyiapkan CrashPlan. Sekarang kita semua bisa selamat dari serangan nuklir.

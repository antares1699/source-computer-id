Ubah format sel sebelum pintasan adalah solusi paling sederhana?

\n\n

Format sel setelah pintasan tidak berfungsi.

\n\n

Cara makro ... 

Sub Main\n\n    Dim Doc As Object\n    Dim Sheet As Object\n    Dim Cell As Object\n    Dim NumberFormats As Object\n    Dim NumberFormatString As String\n    Dim NumberFormatId As Long\n    Dim LocalSettings As New com.sun.star.lang.Locale\n\n    Doc = ThisComponent\n    Sheet = Doc.Sheets(0)\n    Cell = Doc.getCurrentSelection\n    Column = Cell.CellAddress.Column\n    Row = Cell.CellAddress.Row\n\n    Cell.Value = Now()\n\n    LocalSettings.Language = \"en\"\n    LocalSettings.Country = \"us\"\n\n    NumberFormats = Doc.NumberFormats\n    NumberFormatString = \"YYYY-MM-DD HH:MM:SS\"\n\n    NumberFormatId = NumberFormats.queryKey(NumberFormatString, LocalSettings, True)\n    If NumberFormatId = -1 Then\n    NumberFormatId = NumberFormats.addNew(NumberFormatString, LocalSettings)\n    End If\n\n    MsgBox NumberFormatId\n    Cell.NumberFormat = NumberFormatId\n\nEnd Sub\n

Untuk Excel (diuji untuk MS Excel 2010 untuk Windows): memasukkannya ke dalam satu sel tanpa makro atau perangkat lunak tambahan, satu-satunya solusi adalah menggunakan 3 pintasan setelah yang lain:

\n\n
    \n
  • CTRL + ;
    • \n
  • ruang
    • \n
  • CTRL + BERGESER + ;
    • \n
\n\n

Setidaknya ini akan memberi Anda DD-MM-YYYY HH:MM

\n\n

Dari sana Anda hanya perlu mengubah format yang ditentukan pengguna ke pilihan Anda: YYYY-MM-DD HH:MM:SS

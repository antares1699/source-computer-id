Bagaimana cara mengirimkan beberapa input ke baris perintah demi baris (mis. Menjalankan perintah untuk setiap baris)?

\n\n

xargs tidak berfungsi karena meneruskan garis sebagai argumen, bukan sebagai input standar.

\n\n

Kasus spesifik di mana ini muncul adalah mendekode file yang barisnya adalah string yang dikodekan base64.

","source":"https://superuser.com/questions/882721/run-command-for-every-line-of-a-file-in-bash-passing-lines-as-standard-input","author":"Tgr","replies":[],"answers":[{"answer":"

Bisa dilakukan dengan while loop: 

produce | while IFS= read -r line; do echo \"$line\" | process; done | consume\n

(untuk beberapa perintah produce, process, consume sewenang-wenang yang semuanya menggunakan input / output standar) tetapi meh. Harus ada cara yang lebih elegan untuk melakukannya.

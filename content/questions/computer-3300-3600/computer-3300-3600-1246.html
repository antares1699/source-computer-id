Saya memiliki Tablet Media Aiptek yang telah saya gunakan di komputer lama saya untuk membuat seni digital.\nSelama sekitar satu bulan sekarang, saya juga memiliki sistem Quad-core Windows Vista 64-bit baru dengan banyak ruang disk dan RAM. Dan tablet bekerja dengan sempurna pada sistem baru saya. Kemudian, tiga minggu yang lalu saya juga membeli monitor kedua dan mengubah pengaturan saya menjadi sistem monitor ganda. Saya tidak menggunakan tablet saya selama tiga minggu ke depan sehingga saya tidak menyadari bahwa monitor ganda akan meningkatkan ukuran desktop. Dalam kasus saya, desktop sekarang 3520x1200.

Saya juga menggunakan dan menyukai Altap Salamander di mesin / VM Windows saya. Dan saya juga menghabiskan banyak waktu untuk beralih ke OS X untuk menemukan pengelola file panel ganda.

\n\n

Saya menemukan Path Finder terlalu "berat", dalam hal waktu start-up dan kelambanan secara keseluruhan ketika saya mendemonstrasikannya pada awalnya. Saya juga mencobanya kembali beberapa bulan yang lalu karena inkarnasi terbaru memungkinkan panel ganda dan ternyata lebih baik, tetapi tetap tidak cepat " Salamander untuk OS X " Saya ingin.

\n\n

Saya menggunakan mucommander untuk sementara waktu (perangkat lunak hebat gratis dan lintas platform), tapi kemudian saya menemukan yang sempurna " Salamander untuk OS X "- Forklift

\n\n

\n\n

Faktanya, saya menemukan bahwa Forklift (terutama V2beta) lebih baik daripada Salamander dalam beberapa hal. Yang paling menonjol, modul FTP / SFTP berukuran besar lebih cepat daripada Salamander di mesin yang sama (11mb / s vs. 1,5mb / s di jaringan lokal kami), dan saya menyukai fitur tumpukan, kontrol riwayat yang lebih baik, dan integrasi OS (Salamander adalah kikuk pada Windows 7, dukungan unicode kurang, dll). Kontrol keyboard tidak sempurna tetapi pengembangnya sangat responsif dan cepat dalam mengimplementasikan fitur navigasi keyboard. Saya sekarang menggunakan Forklift untuk sebagian besar tugas manajemen file yang dulu saya gunakan untuk Salamander, dan ketika saya menggunakan Salamander mulai ketinggalan forklift :)

Selain perintah normal yang diketahui Midnight Commander, paling-paling saya melewatkan bookmark direktori atau riwayat direktori.

\n
\n\n

Meskipun Finder pasti memiliki kekurangan, ini adalah dua hal yang dapat dilakukannya.

\n\n

Dua tombol panah pada bilah alat bergerak mundur dan maju melalui riwayat penelusuran direktori Anda di Jendela Finder itu.

\n\n

Bilah sisi dapat digunakan untuk menyeret folder atau folder pintar dengan mudah untuk disimpan sebagai "penanda" untuk akses cepat.

\n\n

Jika Anda mencari sesuatu yang sedikit lebih "gemuk" daripada Finder, lihat Pencari Jalur . Ini memiliki bookmark bergaya browser, "tumpukan drop" yang sangat berguna yang membantu mengumpulkan file untuk dipindahkan / disalin ke lokasi lain, penelusuran file panel ganda, baris perintah bawaan, dan banyak lagi.

\n\n

Tambahan lain yang berguna di Path Finder yang mungkin Anda cari sejauh "riwayat direktori" berjalan, adalah ketika Anda mengklik opsi panah riwayat mundur / maju, Anda mendapatkan menu tarik-turun yang menunjukkan jalur yang Anda miliki telah menjelajah.

apps try to make changes to my computer(default)\" \n    } \n    ElseIf($ConsentPromptBehaviorAdmin_Value -Eq 2 -And $PromptOnSecureDesktop_Value -Eq 1){ \n        \"Always notIfy\" \n    } \n ConsentPromptBehaviorAdmin_Value \n\n    If($Level -In 0, 1, 2, 3) { \n        $ConsentPromptBehaviorAdmin_Value = 5 \n        $PromptOnSecureDesktop_Value = 1 \n        Switch ($Level)  \n        {  \n          0 { \n              $ConsentPromptBehaviorAdmin_Value = 0  \n              $PromptOnSecureDesktop_Value = 0 \n          }  \n          1 { \n              $ConsentPromptBehaviorAdmin_Value = 5  \n              $PromptOnSecureDesktop_Value = 0 \n          }  \n          2 { \n              $ConsentPromptBehaviorAdmin_Value = 5  \n              $PromptOnSecureDesktop_Value = 1 \n          }  \n          3 { \n              $ConsentPromptBehaviorAdmin_Value = 2  \n              $PromptOnSecureDesktop_Value = 1 \n          }  \n        } \n        Set-RegistryValue -Key $Key -Name $ConsentPromptBehaviorAdmin_Name -Value $ConsentPromptBehaviorAdmin_Value \n        Set-RegistryValue -Key $Key -Name $PromptOnSecureDesktop_Name -Value $PromptOnSecureDesktop_Value \n\n

Windows RDP tidak mendukung ini ...

\n\n

Ini akan mengubah ukuran / mengatur ulang layar menjadi 1 monitor seperti yang Anda gambarkan.

\n\n

Namun, Anda perlu mencatat bahwa mesin jarak jauh (yang Anda hubungkan juga) harus edisi Windows 7 Ultimate atau Windows 7 Enterprise karena mereka adalah satu-satunya yang mendukung penggunaan beberapa monitor untuk RDP

Windows RDP tidak mendukung ini. Saat Anda terhubung ke RDP, itu hanya memulai sesi baru dengan pengaturan monitor baru. Itu tidak melihat perangkat keras dari server jarak jauh.

\n\n

Juga dengan suara dan semacamnya. Ini mentransfer suara ke komputer lokal dan bukan komputer jarak jauh.

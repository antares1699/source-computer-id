Bagaimana cara mengubah file host di Vista?

","source":"https://superuser.com/questions/57429/how-to-change-hosts-file-in-vista","author":"MicTech","replies":[],"answers":[{"answer":"

Tipe 

notepad %windir%\\system32\\drivers\\etc\\hosts\n

ke dalam kotak telusur di menu mulai dan tekan Ctrl + Shift + Enter.

\n\n

Hak istimewa admin, file yang benar dari awal dan bahkan dibuat jika tidak ada (setelah diminta oleh Notepad).

\n\n

Tombol disimpan karena tidak membutuhkan Menang + R tersesat Ctrl + Bergeser + Memasukkan , meskipun begitu. Jadi ya, ini jauh lebih rumit untuk dilakukan di Vista daripada di XP.

\n\n

Tidak mudah seperti di XP.

\n\n
    \n
  • Mulai -> Semua Program -> Aksesoris
    • \n
  • Jalankan "Notepad" sebagai administrator
    • \n
  • Klik File -> Buka
    • \n
  • Buka C: \\ Windows \\ System32 \\ Drivers \\ etc
    • \n
  • Ubah filter file menjadi "Semua File ( . ) "
    • \n
  • Buka file "host"
    • \n
  • Lakukan perubahan dan simpan.
    • \n
  • Selesai
    • \n

    2. \n

    3. \n
  3. DHCP dinonaktifkan.
    4. \n
  4. Setel alamat IP ke 192.168.7.2 (karena router utama yang akan dihubungkan melalui ethernet ini adalah 192.168.7.1).
    5. \n

Saya mengalami masalah yang sangat mirip hari ini. Apa yang memperbaikinya untuk saya adalah mengubah konfigurasi NetworkManager:

\n\n
    \n
  1. Edit file /etc/NetworkManager/NetworkManager.conf (lokasi mungkin berbeda-beda, tergantung distro Anda)
    2. \n

  2. Tambahkan baris berikut ke file itu: 

    [device]\nwifi.scan-rand-mac-address=no\n
    3. \n
  3. Mulai ulang layanan NetworkManager: sudo service network-manager restart
    4. \n

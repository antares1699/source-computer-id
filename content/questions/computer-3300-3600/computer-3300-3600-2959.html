Saya ingin dapat mengatur prompt khusus (untuk membantu mengotomatiskan pekerjaan, prompt itu haruslah sesuatu yang dapat saya deteksi dengan andal). Saya menemukan saya dapat mengatur prompt dari jarak jauh, seperti ini: 

wim@wim-acer:~$ ssh guest@192.168.1.124 -i ~/.ssh/guest_nopassphrase \n~ # echo $PS1\n\\w \\$\n~ # PS1='Setting a custom prompt \\w \\$ '\nSetting a custom prompt ~ # exit\nConnection to 192.168.1.124 closed.\n

Apakah ada cara untuk melakukannya dalam satu kali pengambilan, dengan menggunakan perintah ssh yang berbeda? Saran yang saya coba dalam menjawab pertanyaan terkait tidak berhasil untuk saya: 

wim@wim-acer:~$ ssh guest@192.168.1.124 -i ~/.ssh/guest_nopassphrase -t \"PS1='Sending a custom prompt \\w \\$ '; exec sh\"\n~ # set\nHOME='/var/tmp'\nIFS='   \n'\nLOGNAME='guest'\nPATH='/sbin:/usr/sbin:/bin:/usr/bin'\nPPID='1039'\nPS1='\\w \\$ '\nPS2='> '\nPS4='+ '\nPWD=''\nSHELL='/bin/sh'\nTERM='xterm'\nUSER='guest'\n

Apakah Anda mempertimbangkan untuk mengedit file .bashrc di host jarak jauh?

\n\n

Sederhana: 

export PS1='Setting a custom prompt \\w \\$ '\n

pada akhir ~/.bashrc pada host jarak jauh akan memastikan prompt berubah setiap kali Anda masuk. Ini jelas akan mempengaruhi pengguna akun lainnya juga.

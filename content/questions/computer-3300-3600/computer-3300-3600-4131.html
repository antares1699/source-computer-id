Saya sudah mencoba OwnCloud sinkronisasi perangkat lunak baru-baru ini dan berfungsi dengan cukup baik.

\n\n

Ini menyinkronkan semua file dengan cukup cepat, tetapi saya agak khawatir tentang keamanan.

\n\n

Jadi, ide saya adalah memasukkan file pribadi saya ke dalam file kontainer TrueCrypt dan kemudian memasukkan file kontainer ini ke dalam folder OwnCloud WebDAV agar secara otomatis disinkronkan dan dicadangkan.

\n\n

\n\n

\n\n

Ini disinkronkan hanya ketika saya turun dari wadah file TrueCrypt saya.

\n\n

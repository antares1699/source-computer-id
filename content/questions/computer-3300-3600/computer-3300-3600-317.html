penginstalan SP3 memiliki tombol baris perintah untuk / nobackup dan tidak akan menyimpan info pencopotan pemasangan jika Anda menggunakannya. Cukup lakukan run -> cmd dan kemudian pindah ke direktori tempat distro xp sp3 Anda berada. Kemudian jalankan perintah di bawah ini (tentu saja menyesuaikan dengan patch) dan Anda harus melakukannya dengan baik!

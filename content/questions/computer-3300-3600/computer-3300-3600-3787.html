Buka Manajemen Disk menggunakan perintah: diskmgmt.msc dan Anda akan melihat disk terdaftar dengan banyak partisi. Pastikan ukuran total disk cocok dengan drive USB Anda. Anda kemudian harus dapat menetapkan huruf drive ke disk dan memformatnya. Anda mungkin perlu menginisialisasi disk sebelum dapat mengelolanya. Anda juga mungkin perlu menghapus partisi secara manual dan kemudian membuat partisi baru untuk mengelolanya sebagai satu kesatuan.

\n\n

Dalam kebanyakan kasus, disk Linux Live tidak mendapatkan huruf drive yang secara otomatis dicolokkan ke mesin Windows kecuali Anda menggunakan utilitas seperti Linux Live (LiLi) yang memberi Anda opsi untuk menyembunyikan file pada drive dari Windows atau tidak.

\n\n

Berikut adalah artikel tentang Manajemen Disk di Windows jika Anda tidak terbiasa dengannya

Saya menggunakan lftp untuk menyinkronkan semua foto saya dari NAS ke server web murah saya secara otomatis. Jika ada foto berubah maka secara otomatis akan memperbaruinya. 

#!/bin/ash\nSOURCE=/volume2/PHOTO/2011\nBACKUPDIR=/foto.whatever.com/albums/2011\n\nlftp -u username,password ftp.whatever.com << EOF\nmirror -R -n -I *.jpg -I *.JPG -X @eaDir/ -X Collage/ -X ‘whatever/’ -X .piccache/ -X .recent/ -X Originals/ -X *.Db $SOURCE $BACKUPDIR\nquit\n

Anda dapat menggunakan opsi -a dalam perintah get pada sftp untuk melanjutkan unduhan yang sebelumnya dibatalkan.

\n\n

Contoh: 

echo -e \"lcd ~/localFolder\\nget -ar *\" | sftp username@host.com:/some/path\n

Atau ditulis sebagai 

sftp username@host.com:/some/path <<EOF\nlcd ~/localFolder\nget -ar *\nEOF\n

Namun ini tidak secara otomatis terhubung kembali ke server jika koneksi terputus. lftp adalah alternatif yang lebih baik untuk masalah ini.

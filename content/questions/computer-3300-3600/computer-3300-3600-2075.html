Lintasan apa yang dijamin ada di contoh Windows Server 2008 R2? Klien mengharuskan beberapa instruksi menentukan jalur yang tepat dalam semua kasus. (Orang yang melaksanakan instruksi tersebut tidak seharusnya harus memutuskan jalan mana pun sendiri, bahkan ketika jalannya sama sekali tidak ada bedanya.) Jadi saya perlu tahu jalan mana yang dapat saya andalkan untuk berada di sana. Tidak masalah bagi saya jika mereka melibatkan variabel lingkungan, tetapi mereka perlu variabel dijamin untuk memegang jalur yang ada. (Artinya, tidak ada modifikasi pada jalur yang tidak mungkin ada.)

\n\n

Atau apakah tidak ada jalur yang dijamin?

Anda dapat menggunakan variabel lingkungan. Ini adalah variabel yang digunakan sistem dan karenanya harus menjadi jalur yang valid. Mereka juga yang akan bekerja di berbagai platform Windows, jadi meskipun jalur kode keras standar berubah, jalur yang dimuat ke variabel oleh windows akan tetap valid.

\n\n

% HOMEPATH% - menunjuk ke direktori home dari pengguna yang sedang login. Jalur ini akan selalu dapat ditulis oleh pengguna sehingga Anda tidak akan memiliki masalah perizinan jika pengguna akan menginstal perangkat lunak itu sendiri.

\n\n

% HOMEDRIVE% - menunjuk ke drive tempat sistem diinstal (biasanya C :, tetapi dapat berubah). Ini bukan pilihan terbaik untuk penginstalan, kebijakan grup sering kali mencegah pengguna untuk menulis di sini.

\n\n

% PROGRAMFILES% - folder file program default, tempat umum untuk instalasi.

    2. \n

    3. \n
  3. folder unduhan - folder tempat ia meletakkan file saat sedang diunduh.
    4. \n

Untuk penggunaan daemon murni di server tanpa kepala, rtorrent akan menjadi klien yang akan saya tuju. Ini baris perintah dan fitur yang sangat lengkap, meskipun itu bukan hal termudah untuk mengaturnya menjadi baris perintah murni. Ada banyak antarmuka web untuk rtorrent juga, meskipun, sekali lagi, mereka tampaknya tidak mudah diatur.

\n\n

Untuk klien GUI, saya lebih suka qBittorrent karena ia bekerja dengan sangat baik dan memiliki antarmuka yang bagus. Ini memiliki antarmuka web juga, meskipun saat ini saya tidak menggunakannya.

\n\n

Deluge baik-baik saja, tetapi ketika saya menggunakannya 2-3 tahun yang lalu saya menemukan itu akan berhenti terus-menerus dan kinerjanya jauh lebih rendah daripada klien lain. Namun, saya tidak dapat mengatakan apakah ini benar hari ini.

\n\n

Transmisi bekerja dengan cukup baik, tetapi selalu tampak sangat sederhana bagi saya.

\n\n

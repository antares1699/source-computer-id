Dalam .vimrc saya, saya punya 

filetype indent on\n

Sekarang, caranya mencoba mengindentasi file yaml secara otomatis sangat membingungkan bagi saya, jadi saya ingin mematikan filetype indent, tetapi hanya untuk file yaml, bukan untuk file lain.

\n\n

Bagaimana cara saya membedakannya di .vimrc saya?

Jawaban @ maiki memberikan solusi yang tepat; jika Anda perlu menyimpan ini di ~/.vimrc, yang berikut ini setara: 

autocmd FileType yaml let b:did_indent = 1\n

Jika ini tidak berhasil, Anda juga dapat mencoba menghapus metode indentasi setelahnya: 

autocmd FileType yaml setlocal indentexpr=\n

Saya tetap menyarankan Anda untuk mengubah pengaturan Anda sehingga lebih dari .vimrc Anda dapat dengan mudah disinkronkan. Pertimbangkan bahwa Anda mungkin ingin menggunakan plugin atau konfigurasi lain (seperti ~/.gitconfig) di masa mendatang. Ada banyak pendekatan untuk menyinkronkan konfigurasi pengguna di seluruh sistem; banyak pengguna memilikinya di repositori pusat (Git) dan menggunakan symlink untuk menginstalnya.

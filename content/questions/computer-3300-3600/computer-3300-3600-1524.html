Untuk memformalkan ini: Ya, itu mungkin , tetapi tidak, Anda tidak ingin melakukan itu.

\n\n

Alasan utamanya adalah saat Anda meningkatkan, Anda mungkin mengubah file konfigurasi dengan skrip pemutakhiran. Tidak ada skrip downgrade, jadi versi paket yang lebih lama mungkin tidak berfungsi dengan konfigurasi Anda lagi. Lebih aman untuk menginstal ulang dan memigrasi data Anda kembali.

\n\n

Jika Anda ingin melakukannya, Ask Ubuntu membuat Anda tercakup di sana: Turunkan dari satu versi ke versi sebelumnya? .

\n\n

Jika Anda hanya terganggu dengan desainnya, yang disebut "Unity", Anda bisa nonaktifkan di 11.04 . Di layar login Anda, pilih Desktop Klasik Ubuntu dari pada Ubuntu Desktop Edition .

\n\n

\n\n

Perhatikan bahwa di versi Ubuntu mendatang, tidak akan ada lagi desktop klasik, jadi Anda harus menggunakan Unity. Jika Anda ingin mengkonfigurasinya, ada banyak pilihan meskipun.

\n\n

Ide lain adalah menginstal manajer desktop lain, seperti berubah menjadi GNOME 3 .

Saya menggunakan apa yang disebut "apt-pinning" dalam kasus saya untuk menurunkan versi dari Ubuntu versi campuran ke lucid: 

$ cat /etc/apt/preferences.d/downgrade-to-lucid\nPackage: *\nPin: release a=lucid\nPin-Priority: 1010\n\nPackage: *\nPin: release a=lucid-updates\nPin-Priority: 1011\n\nPackage: *\nPin: release a=lucid-security\nPin-Priority: 1012\n

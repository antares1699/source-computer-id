tmux menggunakan logika ini untuk menangani file ctrl pengubah di key-string.c (backtick adalah kode 96): 

    /* Convert the standard control keys. */                               \n    if (key < KEYC_BASE && (modifiers & KEYC_CTRL) && !strchr(other, key)) {\n            if (key >= 97 && key <= 122)\n                    key -= 96;\n            else if (key >= 64 && key <= 95)\n                    key -= 64;\n            else if (key == 32)\n                    key = 0;\n            else if (key == 63)\n                    key = KEYC_BSPACE;\n            else\n                    return (KEYC_NONE);\n            modifiers &= ~KEYC_CTRL;\n    }\n

Meskipun beberapa aplikasi mungkin memperlakukan seluruh rentang dari 64 hingga 126 sebagai valid untuk ctrl , tmux tidak termasuk backtic, curly braces, vertical bar dan tilde. Kemungkinan alasannya adalah bahwa mereka kurang berguna untuk pengikatan kunci karena kunci yang tidak dimodifikasi sering digunakan dalam skrip shell.

\n\n

Di sisi lain, ctrl-@ akan memiliki interpretasi yang sama, dan Anda dapat menggunakannya sebagai gantinya.

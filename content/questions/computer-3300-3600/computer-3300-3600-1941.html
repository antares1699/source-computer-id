Untuk memperbaiki, gunakan perintah berikut pada prompt perintah ( dengan hak istimewa ) 

DISM /Online /Cleanup-Image /RestoreHealth\n

RestoreHealth: Opsi sakelar ini memeriksa kerusakan penyimpanan komponen, mencatat kerusakan ke file log, dan MEMPERBAIKI kerusakan gambar menggunakan Pembaruan Windows. Ini harus mengambil sekitar 10-15 menit hingga sekitar satu jam untuk menyelesaikan tergantung pada tingkat korupsi.

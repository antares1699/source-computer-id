Ada ~/.bashrc, yang mungkin mengganggu perintah jarak jauh (seperti scp). 

module load LANG/PYTHON/2.7.6\nmodule load TOOLS/TMUX/2.1\n

Jika Anda ingin melakukan sesuatu seperti ini, Anda harus melakukannya hanya pada shell interaktif (bukan untuk merusak skrip). Anda dapat menambahkan ke awal ~/.bashrc Anda: 

[ -z \"$PS1\" ] && return\n

Biasanya ketika saya menonton sesuatu dengan Windows Media Player, kontrol secara otomatis menyembunyikan diri setelah beberapa detik tidak menyentuhnya. Namun, beberapa kali terakhir, mereka hanya tetap di layar sepanjang waktu, yang cukup mengganggu. Bagaimana cara menyingkirkannya?

","source":"https://superuser.com/questions/618101/why-wont-the-controls-disappear-in-windows-media-player","author":"Jeremy Lin","replies":[],"answers":[{"answer":"
    \n

  1. Buka Windows Media Player

    2. \n

  2. Tekan F10

    3. \n

  3. Buka Tools > Options

    4. \n

  4. Ada kotak centang bernama Allow autohide of playback controls: periksa

    5. \n

  5. Tekan Apply

    6. \n
\n\n

Pastikan mouse Anda dibiarkan melayang di atas video itu sendiri agar kontrolnya disembunyikan, itu persyaratan yang mengganggu tetapi jika mouse Anda berada di layar lain atau di luar video, mereka tidak akan bersembunyi.

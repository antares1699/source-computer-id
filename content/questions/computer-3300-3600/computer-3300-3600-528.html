Saya akan mencoba Ubuntu PPC build sebagai titik awal membangun apa yang Anda butuhkan dari itu.

\n\n

Menggunakan CD Ubuntu alternatif, penginstalan baris perintah sangat mudah dan memberi Anda titik awal dasar. Anda dapat menambahkan apa pun yang Anda suka, bergantung pada sumber daya apa yang akhirnya Anda miliki. Bermain-main dengan konfigurasi yang berbeda dan periksa menggunakan alat baris perintah free, jika Anda mau.

\n\n

Ubuntu dasar (hanya baris perintah) hanya menggunakan memori 39MB (puncak), buffer 19MB disesuaikan. Setelah menginstal paket untuk membuat semuanya berjendela (X, openbox, firefox, dll), itu berakhir pada puncak 446MB, buffer 120MB disesuaikan.

\n\n

Saya tidak yakin bagaimana perbandingan build PPC dasar, tapi saya rasa mereka akan mendekati. Anda bisa melihatnya Crunchbang untuk beberapa inspirasi dalam membuat instalasi ringan berbasis Ubuntu.

OS X tampaknya lamban terutama karena sifat haus memori dari aplikasi OS X modern. Jika Anda menggunakan aplikasi yang di-porting ke Cocoa dari Linux, di mana membuang perangkat keras yang luas pada masalah kinerja adalah praktik yang kurang standar, maka Anda dapat menemukan bahwa kerangka OS X tidak selambat yang Anda kira.

\n\n

Kuncinya adalah bersikap kejam dalam menolak babi sumber daya: selain aplikasi yang selalu menghasilkan pizza, coba hapus aplikasi yang merusak kinerja secara keseluruhan, seperti Mail (sering kali aktif dan menukar jumlah besar), dan program dengan ukuran penduduk yang besar ( lihat output dari ps). \n\n
    \n

  1. Menggunakan Emacs mungkin adalah langkah pertama terbaik yang dapat Anda ambil: Anda dapat menggunakannya sebagai pengganti Terminal, Mail (Anda harus menggunakan postfix untuk mengirim email Anda ke folder lokal), serta sebagian besar editor teks lainnya.

    2. \n

  2. Pilihan browser adalah yang sulit. Ada browser X11 yang sangat ringan, seperti Dillo , tetapi mereka tidak berfitur lengkap, dan peralihan antara X11 dan Aqua tidaklah mulus.     Coba gunakan browser teks di samping browser grafis (Emacs memiliki mode w3 untuk browsing teks) dan matikan browser grafis dari waktu ke waktu.
    Kebenaran brutal adalah bahwa pengalaman penjelajahan web modern mengasumsikan Anda memiliki lebih dari 256 juta: Firefox di Ubuntu lebih cepat daripada Firefox di OS X, tetapi bahkan di sana, itu bukan yang terbaik.

Dari: Paksa Uninstall - Cara Menghapus Sepenuhnya Perangkat Lunak yang Tidak Diperlukan dari PC Anda dalam Hitungan Menit

\n\n
\n

Klik Start dan pilih Run di menu (Jika Anda menggunakan Windows Vista\n lalu tekan Win + R pada keyboard Anda).

\n \n

Ketik regedit dan tekan Enter.

\n \n

Di sisi kiri adalah pohon pengaturan registri, gunakan untuk pergi ke\n HKEY_LOCAL_MACHINE \\ SOFTWARE \\ Microsoft \\ Windows \\ CurrentVersion \\ Uninstall

\n \n

Di dalam kunci itu Anda akan menemukan banyak kunci milik yang berbeda\n program. Beberapa diberi nama menurut nama program, yang lainnya sebagai campuran\n angka dan huruf yang tidak masuk akal. Perhatikan masing-masing\n sampai Anda menemukan salah satu yang memiliki kunci DisplayName (di sebelah kanan) dengan\n nama program Anda di dalamnya.

\n \n

Perhatikan kunci UninstallString - kunci ini menunjuk ke uninstall\n program, dan file log biasanya berada di folder yang sama dengan itu\n program.

\n \n

Jika Anda menghapus kunci tempat Anda menemukan kunci DisplayName dengan\n nilainya sama dengan nama program Anda, maka program Anda tidak akan muncul\n pada daftar Tambah / Hapus program.

Jawaban singkatnya: Anda mungkin tidak bisa.

\n\n

Jawaban panjang: Sebagian besar penginstal akan memiliki daftar persyaratan yang diinginkan program saat diinstal. Namun ini bukanlah kebenaran seluruhnya. Sebuah program dapat membuat perubahan pada komputer setelah fakta, misalnya ketika sedang dijalankan dan Anda mengubah pengaturan itu dapat menulis ini ke registri. Anda dapat menambahkan asosiasi file Anda sendiri setelah fakta dan ini akan ditulis ke registri.

\n\n

Jadi ketika uninstaller dijalankan, itu mungkin hanya akan melihat daftar yang digunakan installer dan menghapus perubahan tersebut. Perubahan lain apa pun yang dibuat oleh program setelah diinstal tidak akan dihapus oleh uninsnstaller.

\n\n

Anda mungkin bisa memperbaiki ini dengan menjalankan Monitor Proses misalnya dan lacak setiap perubahan yang dilakukan program ke komputer Anda saat dijalankan. Namun itu mungkin akan membuat program sangat tidak mungkin digunakan karena Anda harus menonton Monitor Proses pada saat yang bersamaan.

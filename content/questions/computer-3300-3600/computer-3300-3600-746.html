Apakah melihat kode sumber plugin mungkin?

\n
\n\n

Iya.

\n\n

Baik Minimalis untuk Gmail dan Minimalis untuk Pustaka Google adalah proyek sumber terbuka dengan Lisensi GPLv3 . Anda dapat menjelajah pohon kode sumber mereka, mengunduh repositori kode sumber lengkap dan lebih banyak lagi di masing-masing Google Code halaman muka:

Apakah seseorang tahu persis apa yang mereka akses? Atau apakah itu lebih pro forma?

\n
\n\n

Sedikit bercanda: ya, pengembang tahu apa yang mereka akses. Artinya, Anda harus bertanya kepada mereka di forum mereka. Atau baca kode sumbernya. Saya yakin mereka akan dengan senang hati mendiskusikan rutinitas akses data mereka dengan Anda. Perangkat lunak open source juga memiliki banyak forum diskusi terbuka di sekitarnya dan itu adalah A Good Thing (tm).

Pilihan lainnya adalah mengunduh file CRX (klik kanan tombol Install dan pilih Save-As), lalu buka di pengarsip yang kompatibel dengan ZIP seperti 7-Zip.

\n\n

Ekstensi harus memiliki file manifes yang pada dasarnya hanya file teks JavaScript. Kode ekstensi itu sendiri juga merupakan file JavaScript yang dapat Anda lihat.

\n\n

Beberapa ekstensi tidak melakukan pekerjaan apa pun secara lokal dan hanya meneruskan pekerjaan ke server eksternal untuk dilakukan yang tentu saja berarti Anda tidak dapat melihat kode atau apa yang terjadi pada data Anda di sana. Namun untuk ekstensi yang lebih sederhana, Anda dapat melihat kode dan memeriksanya sendiri sebelum menginstal (dengan asumsi pengetahuan yang layak tentang JavaScript tentunya).

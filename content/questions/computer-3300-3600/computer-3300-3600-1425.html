Saya tahu ini sudah terlambat beberapa bulan, tetapi saya mengalami ini dan menemukan solusinya.

\n\n
    \n
  1. Hapus %HOME%/AppData/Roaming/MetroTwit.
    2. \n

  2. Luncurkan MetroTwit dan konfigurasi ulang.

    \n\n

    %HOME% sama dengan C:\\Users\\kpax, jika Anda atau orang lain sama sekali bingung. *

    \n\n

    Selain itu, Anda akan melihat aplikasi pada dasarnya dikembalikan ke status yang baru dipasang setelah melakukan ini.

    3. \n
\n\n

Saya yakin mungkin ada cara yang lebih diskriminan untuk memperbaikinya dengan menghapus satu atau lebih file konfigurasi di dalam folder MetroTwit jika Anda gugup kehilangan beberapa pengaturan. Cukup cadangkan folder dan coba hapus berbagai file sampai Anda (semoga) dapat meluncurkannya lagi dengan pengaturan Anda utuh. Saya tidak mengotak-atik dan menghapus seluruh folder jadi saya minta maaf karena tidak tahu persis file atau file mana yang pelakunya.

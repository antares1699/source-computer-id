Di kolom A, saya punya tanggal. Di kolom B, berat badan saya dicatat untuk hari itu. Saya menambahkan satu baris ke setiap hari ketika saya menimbang diri saya sendiri, jadi ini berarti data diurutkan berdasarkan tanggal naik. Bobotnya, tentu saja, berfluktuasi (meskipun alangkah baiknya jika mereka turun setiap hari untuk keuntungan pribadi saya).

\n\n

Untuk beberapa kalkulasi, saya ingin mendapatkan terbaru (atau terakhir) bobot yang dimasukkan di kolom B. Bukan max atau min, tetapi yang terakhir dimasukkan ke dalam kolom. Saya ingin ini berfungsi tidak berapa banyak baris yang saya masukkan.

\n\n

Saya menggunakan Excel 2007, jika itu penting.

Anda dapat meletakkan di sel lain (misalkan tidak ada sel kosong di kolom B): 

=INDEX(B:B,COUNT(B:B))\n

Perhatikan bahwa Anda juga dapat mencoba (jika bobot Anda tidak dikenali sebagai angka dan anggap Anda memiliki judul di baris pertama): 

=INDEX(B:B,COUNTA(B:B) + 1)\n

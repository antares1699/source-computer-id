Saya mengalami masalah pada lingkungan dev lokal saya dengan Chrome karena momen yang tidak diketahui di masa lalu, beberapa permintaan (menurut saya sekitar 90% di antaranya) sangat lambat, perlu sekitar 10 detik untuk mulai mentransfer data.

\n\n

\n\n

Saya menjalankan Mac OS X (Lion, tapi sudah pernah terjadi sebelumnya) dan MAMP sebagai server web lokal. Ini juga terjadi dengan Apache2 + PHP5 yang diinstal melalui MacPorts.

\n\n

Ini hanya terjadi di Chrome, server web berfungsi seperti yang diharapkan di Safari dan Firefox. Tentang saya: versi mengatakan ini:

\n\n

Google Chrome 12.0.742.122 (Bentukan Resmi 91910)

\n\n

Saya mencoba mengaktifkan DNS Pre-fetching tetapi opsinya hilang, saya tidak yakin sejak kapan itu diperbarui dengan sendirinya.

\n\n

Sepertinya saya harus bangun sendiri Firefox 64-bit untuk Linux, karena Mozilla tidak akan mendukungnya hingga Firefox 4.

\n\n

Kenapa ini? Bagi saya sepertinya itu berfungsi dengan baik, meskipun tanpa beberapa peningkatan kecepatan pada mesin Javascript yang didapat versi 32-bit.

\n\n

(Sunting: Ya, saya dapat menjalankan versi 32-bit tetapi saya mencoba untuk menjaga sistem saya bersih dari cruft 32-bit dan perpustakaan dll, dan semua plug-in berfungsi dengan baik di versi tidak resmi 3.0.11 64-bit.)

\n\n

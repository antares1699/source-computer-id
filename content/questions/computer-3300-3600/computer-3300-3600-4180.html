Saya sedang membangun mesin Windows 7 untuk dioperasikan tanpa kepala dengan VNC. Saya ingin mengatur resolusi agar sesuai dengan monitor yang akan digunakan (pada 1680x1050) daripada resolusi monitor yang dipasang sementara untuk gedung (pada 1366x768). Itu tidak akan membiarkan saya melakukannya dengan monitor yang terpasang saat ini. Adakah cara untuk melakukan ini, mungkin dengan menginstal driver monitor lain? Saya menggunakan grafik Sandy Bridge asli pada mobo Asus P8Z68-V Pro.

Anda dapat melakukan ini hanya dengan mencabut monitor dari PC jarak jauh, kemudian Anda dapat mengatur resolusi ke apa pun yang Anda inginkan (saya anggap ini dapat diterima karena Anda menunjukkan "pengaturan tanpa kepala"). Namun jika Anda saat ini berada di lokasi yang jauh, Anda dapat menonaktifkan monitor di pengelola perangkat dan kemudian reboot. Ini hanya berfungsi jika aplikasi VNC Anda berjalan sebagai layanan dan memungkinkan akses prapelogin seperti Teamviewer atau jika komputer Anda melakukan boot langsung ke desktop windows dan host VNC ada di folder start up.

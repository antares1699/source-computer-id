Agar berfungsi, Anda perlu menginstal dukungan tamu vbox Linux jika Anda ingin menggunakannya dalam mode desktop. Dan jika Anda tidak menggunakan centos sebagai desktop dan dalam mode CLI, saya akan mengatakan terhubung ke tamu menggunakan aplikasi terminal menggunakan ssh. Aplikasi terminal memungkinkan Anda menyalin dan menempelkan cara Mac biasa dengan cmd + c dan cmd + v.

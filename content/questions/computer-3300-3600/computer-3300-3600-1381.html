Saya memiliki beberapa ratus file keluaran, yang beberapa di antaranya berisi pesan kesalahan. Katakanlah saya ingin secara otomatis meletakkan semua file yang berisi string "ERROR_1" dalam "subfolder_1", dan sisanya dalam "subfolder_2" kedua. Ini seharusnya bisa dilakukan dengan mem-pipe grep. Apa sintaks yang berguna?

","source":"https://superuser.com/questions/641784/sort-files-with-grep","author":"aldorado","replies":[{"reply":" Ini mungkin membantu","author":"50-3"}],"answers":[{"answer":"
for a in *;do grep -q ERROR_1 \"$a\" && mv \"$a\" subfolder_1 || mv \"$a\" subfolder_2;done\n

Ini seharusnya berhasil.

Skrip ini akan memindahkan file ke subfolder. Ubah wildcard * sesuai kebutuhan (mis. *.log) - kode ini akan mencari semua file di direktori saat ini 

for f in *\ndo\n    grep -f ERROR_1 \"$f\"\n    if [ $? -eq 0 ]; then\n       mv \"$f\" subfolder_1\n    else\n       mv \"$f\" subfolder_2\n    fi\ndone\n

\n\n

setiap situs web menyarankan untuk membuat kunci reg ini:

\n\n

HKEY_LOCAL_MACHINE \\ SYSTEM \\ CurrentControlSet \\ Services \\ Tcpip6 \\ Parameters \\ DisabledComponents

\n\n

sebagai DWORD dan menyetel nilainya ke 0xFFFFFFFF

\n\n

Saya telah melakukan ini, tetapi ketika saya reboot, adaptor NIC saya masih menunjukkan IPV6 diaktifkan.

\n\n

Hanya karena dimatikan di registri tidak berarti kotak centang Anda di NIC akan hilang. Itu hanya berarti itu dinonaktifkan. Hanya untuk memastikan saya biasanya melakukannya di kedua tempat (di properti registri dan NIC). Untuk Windows 7 namun melakukannya pada properti NIC seharusnya sudah cukup, tetapi untuk Exchange 2010 (sebagai catatan tambahan) itu membuat Windows Server bertekuk lutut.

\n\n

Beberapa cara untuk menonaktifkan IPv6 sepenuhnya (hanya untuk kelengkapan jawaban):

\n\n
    \n
  • Jalankan ini dari command prompt reg add hklm\\system\\currentcontrolset\\services\\tcpip6\\parameters /v DisabledComponents /t REG_DWORD /d 255 dan kemudian reboot.
    • \n
  • Jalankan Microsoft ini Memperbaikinya utilitas http://support.microsoft.com/kb/929852
    • \n

Saya baru saja mengalami masalah ini sendiri di mana Windows hanya akan terhubung ke earbud bluetooth saya dalam bentuk mono, memanggil perangkat "[Nama perangkat] Hands-free". Saya tidak dapat menemukan cara untuk mengaktifkan stereo jenis apa pun, tetapi tampaknya itu mendeteksi bahwa stereo didukung, tetapi hanya memilikinya sebagai default. Oleh karena itu, versi stereo disembunyikan karena saya terputus darinya.

\n\n

Jadi saya membuka Sound di Control Panel, mengklik kanan pada daftar di tab Playback, dan mencentang "Show Disconnected Devices". "[Nama perangkat] Stereo" kemudian muncul di daftar, yang saya klik kanan dan tekan "Hubungkan". Ini kemudian mengaturnya sebagai perangkat default dan berfungsi dengan baik sekarang.

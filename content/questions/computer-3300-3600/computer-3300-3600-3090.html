Saya telah menyiapkan instans EC2 dan memiliki 2 volume EBS yang menyertainya. Saya ingin melihat isi setiap volume. Bagaimana cara melihatnya melalui terminal (cd / somePath)? Saya sudah memiliki koneksi SSH yang valid ke instans EC2.

\n\n

Edit:

\n\n

Dari konsol AWS, salah satu volumenya adalah i-9167d3ff:/dev/sda1. Saya mencoba melakukan cd /dev/sda1 tetapi saya mendapatkan pesan -bash: cd: /dev/sda1: Not a directory

","source":"https://superuser.com/questions/292134/how-to-view-contents-of-an-ebs-volume-attached-to-an-ec2-instance-via-terminal","author":"David","replies":[],"answers":[{"answer":"

Item di direktori dev adalah perangkat, jadi Anda tidak dapat melihatnya secara langsung. Ini sebenarnya adalah disk untuk volume EBS yang dimaksud. Biasanya ini dipasang pada direktori tertentu, seperti / mnt / volumename.

\n\n

Jika volumenya sudah dipasang, Anda dapat menjalankan perintah 'df' untuk mendaftarnya. Ini akan menunjukkan kepada Anda nama perangkat dan titik pemasangan. Anda kemudian dapat melihat konten titik pemasangan.

\n\n

Jika mereka belum dipasang, maka Anda perlu mengkonfigurasi / etc / fstab dengan tempat yang benar untuk memasang volume Anda. Mereka kemudian akan dipasang pada saat boot, atau dengan menjalankan perintah mount.

Setelah Anda memasang volume EBS ke mesin EC2 Anda.\nAnda harus memasang perangkat ini.\nAnda dapat melihat perangkat yang terpasang menggunakan salah satu dari perintah berikut: 

ls /dev\nsudo parted -l\n

Anda akan melihat banyak perangkat, termasuk beberapa yang bernama sda, xvda.\nSekarang buat subdirektori di direktori /media. 

mkdir /media/new\nmount /dev/sda /media/new   //mounts /dev/sda at mount point /media/new\n

Anda dapat membuat daftar partisi yang dipasang menggunakan df -T.\nJelajahi partisi Anda melalui titik pemasangan; mis., cd /media/new.

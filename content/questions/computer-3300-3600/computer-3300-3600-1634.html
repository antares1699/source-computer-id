Ubuntu memiliki paket yang disebut "pastebinit", yaitu skrip yang meletakkan file di pastebin dan kemudian mencetak URL-nya.

\n\n

Instal dengan: 

$ sudo apt-get install pastebinit\n

Menggunakannya sangat mudah: 

$ pastebinit your_file.txt\nhttp://pastebin.com/whatever\n

Saya menggunakan PAMIE ( http://pamie.sourceforge.net/ ) untuk mengotomatiskan beberapa rutinitas pengujian di situs web klien melalui IE8, dan ingin dapat menjalankan beberapa pengujian dengan kredensial pengguna yang berbeda.

\n\n

Situs yang saya uji menggunakan cookie untuk mengingat pengguna (tanpa opsi "ingat saya", saya dapat membatalkan pilihan). Oleh karena itu, ketika saya menjalankan contoh kedua IE8, cookie akan dibagikan dan saya tidak dapat masuk sebagai pengguna yang berbeda.

\n\n

Adakah cara agar IE8 menggunakan kumpulan cookie yang terisolasi di setiap jendela?

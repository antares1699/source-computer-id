Saya menggunakan windows server 2012R2 dan tidak dapat memperbaiki kesalahan ini.

\n\n

Program tidak dapat dimulai karena MSVCR120.dll hilang dari komputer Anda. Coba instal ulang program untuk memperbaiki masalah ini.

\n\n

Menghapus instalasi itu menghapusnya dari c: \\ windows \\ system32 dengan baik.

\n\n

Setelah reboot tidak ada. Saya menginstal ulang menggunakan re distributable http://www.microsoft.com/en-us/download/details.aspx?id=40784 dan kesalahan yang sama muncul.

\n\n

Pengembang program yang saya gunakan dengannya telah mengkompilasinya dengan flag / MT jadi dia hanya mengatakan itu masalah saya.

\n\n

Saya telah mencoba SFC seperti di bawah ini dengan opsi scannow

\n\n

http://technet.microsoft.com/en-us/library/bb491008.aspx

\n\n

Saya menduga ini adalah masalah registri tetapi saya TIDAK ADA PETUNJUK bagaimana cara memperbaikinya.

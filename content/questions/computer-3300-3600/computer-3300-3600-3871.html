Fungsi COUNTIF dan COUNTIFS mengambil rentang sebagai argumen pertamanya, dan tidak akan mengizinkan Anda untuk beroperasi pada rentang ini, misalnya COUNTIF(MONTH(A:A),6) akan membuat kesalahan.

\n\n

Singkatnya mengubah tata letak data Anda di lembar Anda, satu-satunya pilihan Anda adalah menggunakan SUMPRODUCT seperti ini: 

=SUMPRODUCT(--(MONTH(Sheet1!A1:A25)=6),--(Sheet1!B1:B25=\"Brian\"))\n

Ini akan menghitung berapa kali Brian muncul di bulan Juni. Perhatikan bagaimana -- digunakan untuk mengonversi nilai Boolean TRUE dan FALSE (yang tidak dapat dikalikan dengan SUMPRODUCT) menjadi numerik 1 atau 0.

\n\n

Opsi Anda yang lain adalah memasukkan kolom lain di data sumber: 

A = Date    B = Names    C = \"=Month(A)\"\n

Jadi di C1 Anda akan memasukkan =Month(A1). Dengan ini Anda dapat menggunakan: 

=COUNTIF(Sheet1!A1:A25,6,Sheet1!B1:B25,\"Brian\")\n

COUNTIF akan bekerja lebih cepat daripada SUMPRODUCT tetapi membutuhkan kolom tambahan.

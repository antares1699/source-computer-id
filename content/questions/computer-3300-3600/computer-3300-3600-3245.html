Saya memiliki pengaturan direktori sebagai berikut: 

/hosted/partner1/logo.png\n/hosted/partner2/logo.png\n/hosted/partner3/logo.png\n/hosted/partner4/logo.png\n/hosted/partner5/logo.png\n..etc.\n

Saya ingin menulis skrip yang dapat MENYALIN file-file itu ke lokasi yang berbeda, dengan nama file yang berbeda, seperti ini: 

/partners/partner1.png\n/partners/partner2.png\n/partners/partner3.png\n..etc.\n

Ada ide? Saya tidak begitu hebat dengan skrip shell dan ada banyak file yang perlu saya pindahkan ke satu direktori ...

Atau 

find /hosted -maxdepth 1 -name \"partner*\" -type d | while read -r dir\ndo\n    cp \"${dir}/logo.png\" \"/partners/${dir##*/}.png\"\ndone\n

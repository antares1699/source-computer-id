Apakah ada cara untuk mendapatkan daftar ringkas dari semua repositori git yang dikelola oleh gitosis.

\n\n

Saya mencari sesuatu yang mirip dengan 

svn list url_to_repo\n

Saya tahu bahwa saya dapat melihat gitosis.conf, tetapi itu terlalu bertele-tele.

","source":"https://superuser.com/questions/267056/gitosis-list-all-repositories","author":"BetaRide","replies":[],"answers":[{"answer":"

Sejauh yang saya tahu Anda tidak dapat memeriksa repositori apa yang Anda akses dengan gitosis, tanpa memeriksa file gitosis.conf atau tanpa akses ke dir repo gitosis (baik jarak jauh atau lokal)

\n\n

Kamu harus menggunakan gitolite.dll untuk memenuhi kebutuhan itu.

\n\n

Gitolite mudah dipasang dan penggunaannya hampir sama dengan gitosis (akses pengguna tunggal, menggunakan repositori dengan file konfigurasi dan keydir untuk mengkonfigurasi layanan, dll), tetapi memungkinkan pengaturan izin per cabang (jika perlu), sinkronkan dengan gitweb, dan banyak opsi lainnya (sebenarnya banyak!)

\n\n

Bagian terbaiknya adalah Anda dapat mengirim perintah 'info' ke server menggunakan ssh dan ini akan mencantumkan repositori yang dapat Anda akses.

\n\n

mis .: Perintah 

anUser@computer> ssh git@my-git-server info\n

Akan muncul di layar seperti 

hello anUser, the gitolite version here is v1.5.7-4-g270e0b9\nthe gitolite config gives you the following access:\n     R+  W  Repo1\n     R   W  Repo2\n     R      Repo3\n    @R_ @W_ testing\n

Saya memasukkan kode berikut ke .bashrc saya: 

function repos\n{\n   gitosisDir=\"/cygdrive/c/Users/Ernst/Source/gitosis-admin\"\n   awk 'BEGIN { OFS = \"[] \"; ORS = \"\\n\" } \n        /repo/ {print $2}' \"$gitosisDir/gitosis.conf\" | sed 's/[][]//g'\n}\n

Sekarang saya cukup mengetik repos dan mendapatkan daftar praktis dari semua repositori yang ditentukan pada gitosis.conf.

","source":"https://superuser.com/questions/316578/how-can-i-view-live-logs-on-a-linux-server-ubuntu","author":"wowpatrick","replies":[],"answers":[{"answer":"

Di atas Terminal

\n\n

Dalam kasus yang paling sederhana: 

tail -f /var/log/name-of-logfile-you-want-to-watch\n

Opsi -f "mengikuti" file dan secara otomatis akan memperbarui tampilan saat file sedang diperbarui. Untuk daftar file log yang mungkin, Anda dapat melihat artikel ini atau cukup jelajahi sendiri.

\n\n
\n\n

Menggunakan penampil log GUI

\n\n

Jika Anda menginstal GNOME sebagai lingkungan grafis, ia dilengkapi dengan log viewer bernama gnome-system-log. Buka dengan: 

gnome-system-log &\n

Kamu akan temukan panduannya di sini .

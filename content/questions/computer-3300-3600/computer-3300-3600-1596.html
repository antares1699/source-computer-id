Ini mungkin berguna jika Anda belum mencobanya: iPad Peek .

\n\n

Untuk informasi lebih lanjut: apakah sebuah situs akan menampilkan versi iPad dari situs tersebut kemungkinan besar bergantung pada permintaan HTTP yang dikirim oleh browser; itu berisi browser apa yang telah mengirim permintaan dan situs akan memutuskan untuk mengirim kembali versi iPad (cari "permintaan HTTP" dan "Agen-Pengguna" jika Anda tertarik untuk membaca lebih lanjut).

\n\n

Anda bisa membuat permintaan HTTP yang menggunakan Agen Pengguna iPad :

\n\n

Mozilla/5.0(iPad; U; CPU iPhone OS 3_2 like Mac OS X; en-us) AppleWebKit/531.21.10 (KHTML, like Gecko) Version/4.0.4 Mobile/7B314 Safari/531.21.10

\n\n

menggunakan: Pemain biola dan lihat tanggapannya.

\n\n
\n\n

EDIT: Berikut informasi tentang mengubah agen pengguna Anda di Chrome . Itu sedikit lebih sederhana daripada membuat permintaan HTTP Anda.

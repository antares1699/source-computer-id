Di sini adalah beberapa langkah dari Mydigitallife yang mungkin menyelesaikan masalah Anda :

\n\n

Sambil menunggu Microsoft memperbaiki dan menambal kesalahan definisi virus dan spyware yang tidak dapat diperbarui secara otomatis, berikut beberapa langkah yang dapat diambil pengguna untuk memastikan bahwa Microsoft Security Essentials memperbarui secara otomatis jika diperlukan.

\n\n

- Pastikan bahwa layanan Pembaruan Otomatis (wuauserv) dan Background Intelligent Transfer Service (BITS) sedang berjalan

\n\n

Microsoft Security Essentials, meskipun bergantung pada program klien untuk melakukan pembaruan klien dan definisi, tetapi masih tergantung pada pipa layanan Pembaruan Otomatis (wuauserv) dan secara langsung, Background Intelligent Transfer Service (BITS) untuk melaksanakan udpate. Jadi, pastikan bahwa wuauserv dan BITS sedang berjalan (melalui Control Panel -> Administrator Tools -> Services) atau MSE tidak akan update.

\n\n

- Perbarui Microsoft Security Essentials secara Manual

\n\n

Microsoft Security Essentials dapat dipaksa atau diperbarui secara manual. Untuk melakukannya, cukup buka antarmuka pengguna Microsoft Security Essential, buka tab Perbarui, dan klik tombol Perbarui untuk memeriksa pembaruan.

\n\n

\n\n

Pembaruan manual akan memungkinkan pengguna untuk mengetahui apakah ada masalah mendasar dengan mekanisme pembaruan MSE, atau hanya pembaruan otomatis yang gagal menjalankan tugas.

\n\n

- Paksa MSE untuk Memeriksa Pembaruan Definisi Baru Sebelum Menjalankan Pemindaian Terjadwal

\n\n

Ini adalah solusi untuk sistem di mana Microsoft Security Essentials tidak atau gagal memeriksa dan memperbarui definisi virus dan spyware secara otomatis. Triknya adalah dengan menjadwalkan pemindaian otomatis pada sistem, dan memaksa Microsoft Security Essentials untuk memeriksa pembaruan tanda tangan baru sebelum dimulainya proses pemindaian. Untuk melakukannya, buka antarmuka pengguna Microsoft Security Essentials, buka tab Pengaturan, pilih opsi Pemindaian Terjadwal.

\n\n

Pastikan pemindaian terjadwal diaktifkan (Jalankan pemindaian terjadwal pada komputer saya (disarankan) kotak centang dicentang) dengan interval harian, dan centang kotak Periksa definisi virus dan spyware terbaru sebelum menjalankan opsi pemindaian terjadwal. Hapus centang dan hapus centang pada kotak Mulai pemindaian terjadwal hanya ketika komputer saya hidup tetapi tidak digunakan sehingga pemindaian terjadwal akan selalu berjalan, begitu juga pembaruan tanda tangan virus otomatis.

\n\n

\n\n

- Unduh dan Instal Pembaruan Definisi Microsoft Security Essentials Terbaru secara Manual

\n\n

Microsoft menyediakan versi terbaru dari mesin lengkap dan paket definisi dari Pusat Perlindungan Malware Microsoft . Pengguna dapat mengunduh paket secara gratis, dan menerapkannya ke PC yang menjalankan MSE, terutama di komputer tanpa koneksi Internet.

\n\n

- Tingkatkan Interval Pembaruan Tanda Tangan Definisi Virus dan Spyware Microsoft Security Essentials

\n\n

Interval pembaruan standar 24 jam tanda tangan MSE mungkin terbukti terlalu lama untuk beberapa komputer, karena sebagian besar komputer tidak dibiarkan berjalan 24 jam setiap hari, menyebabkan Microsoft Security Essentials tidak memperbarui basis data definisi virus dan spyware menjadi kedaluwarsa dan membahayakan komputer.

\n\n

Dengan peretasan registri sederhana, Microsoft Security Essentials dapat diinstruksikan untuk memeriksa pembaruan lebih dari sekali sehari (standarnya adalah setiap 24 jam - sekali sehari). Ikuti panduan di bawah ini untuk mengubah kunci registri agar MSE dapat memeriksa dan memperbarui basis data tanda tangan definisi virus dan spyware setiap beberapa jam. Kemampuan untuk melakukan dan memperbarui proses lebih sering juga mampu menyelesaikan masalah ketika MSE tidak memperbarui basis data definisinya secara otomatis.

\n\n

Ikuti panduan untuk mengubah registri untuk mengubah interval pembaruan tanda tangan Microsoft Security Essentials:

\n\n
    \n
  • Jalankan Registry Editor (RegEdit).
    • \n

  • Arahkan ke kunci registri berikut: 

    HKEY_LOCAL_MACHINE/Software/Microsoft/Microsoft Antimalware/Signature Updates\n
    • \n

  • Di panel kanan, ambil kepemilikan dan berikan izin kontrol penuh pada kunci registri, nilai REG_DWORD bernama SignatureUpdateInterval.

    • \n

  • Klik dua kali pada SignatureUpdateInterval. Secara default, nilainya adalah 24 (dalam Desimal) atau 18 (dalam Heksadesimal). Ubah nilainya menjadi angka antara 1 dan 24 (keduanya inklusif) di Desimal.

    \n\n

    Nilai numerik menentukan seberapa sering dan seberapa cepat (dalam jam) Microsoft Security Essentials akan memeriksa pembaruan dan memperbarui definisi virus dan spyware.

    • \n

  • Nyalakan ulang komputernya.

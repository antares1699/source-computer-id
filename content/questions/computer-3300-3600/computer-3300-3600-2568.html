Cara untuk mencapai ini adalah dengan menggunakan file cdparanoia.dll alat baris perintah. Di Ubuntu Linux Anda dapat menginstalnya melalui

\n\n

sudo apt install cdparanoia

\n\n

lalu lari

\n\n

cdparanoia -Q

\n\n

Dimana "-Q" adalah singkatan dari "query".\nJika tidak ada disk audio di drive, Anda akan mendapatkannya 

cdparanoia III release 10.2 (September 11, 2008)\n004: Unable to read table of contents header\nUnable to open disc.  Is there an audio CD in the drive?\n

Kalau tidak, akan terlihat seperti ... 

Table of contents (audio tracks only): track        length             begin        copy pre ch\n===========================================================\n  1.     8315 [01:50.65]        0 [00:00.00]    no   no  2\n  2.    12830 [02:51.05]     8315 [01:50.65]    no   no  2\n  3.    15642 [03:28.42]    21145 [04:41.70]    no   no  2\n  4.    16238 [03:36.38]    36787 [08:10.37]    no   no  2\n

Bagaimana cara mendeteksi jika disk optik yang dimasukkan adalah disk audio? 

#!/bin/bash\ncheckforaudio=`gvfs-info -f cdda://sr0/ | grep \"filesystem::type: cdda\"`\nif [ \"$checkforaudio\" == \"  filesystem::type: cdda\" ]; then\n    vlc cdda:///dev/sr0\n    else\n        checkfordvd=`find /media/*/ -name \"VIDEO_TS\" | grep \"VIDEO_TS\"`\n        if [ ${checkfordvd:(-8)} == \"VIDEO_TS\" ]; then\n            vlc dvd:///dev/sr0\n            else\n            pcmanfm\n        fi\nfi\nexit 0\n

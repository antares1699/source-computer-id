Jika Anda tidak memiliki direktori ~/bin, Anda dapat membuatnya dengan mdkir ~/bin. Ini memungkinkan Anda untuk mengikuti panduan.

\n\n

Jika Anda melakukan ini dan Sublime masih tidak berfungsi untuk Anda setelah Anda mengatur semuanya, maka folder baru ini mungkin tidak termasuk dalam $PATH Anda (daftar folder yang dicari oleh shell baris perintah untuk file yang dapat dieksekusi). Anda dapat memeriksa apakah ini masalahnya dengan menjalankan echo $PATH dan melihat apakah itu berisi /Users/<your username>/bin. Jika Anda tidak melihat folder baru yang kami buat di $PATH Anda, maka Anda harus menambahkannya: Tambahkan folder ke PATH

\n\n

Lokasi yang lebih tepat untuk menghubungkan exceutable subl mungkin /usr/local/bin jika ada.

    \n

  1. Buat direktori bin di folder rumah Anda menggunakan Finder atau dengan mengetik mkdir ~/bin di terminal.

    2. \n

  2. Buat symlink ke aplikasi subl di folder bin itu dengan mengetik sudo ln -s \"/Applications/Sublime Text 2.app/Contents/SharedSupport/bin/subl\" ~/bin/subl di terminal.

    3. \n

  3. Buka atau buat file bernama .bash_profile (catat titiknya!) Di direktori home Anda menggunakan Sublime Text. (Direktori home Anda akan menjadi direktori yang ada di Finder dengan ikon home. Anda bisa mendapatkan Finder untuk menampilkan direktori home secara cepat dengan mengetik open ~ di terminal.)

    4. \n

  4. Tambahkan direktori ~/bin ke $PATH Anda dengan mengedit file .bash_profile yang baru saja Anda buat dan menambahkan baris export PATH=$PATH:~/bin. Kemudian simpan file tersebut.

    5. \n

  5. Muat ulang .bash_profile Anda sehingga $ PATH yang diperbarui diambil dengan mengetik source ~/.bash_profile di terminal.

    6. \n
\n\n

Anda sekarang dapat menggunakan perintah subl pada prompt terminal. Perlu juga dicatat bahwa Anda dapat meletakkan folder bin di mana pun Anda suka jika Anda tidak ingin meletakkannya di folder rumah Anda.

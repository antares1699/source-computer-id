\n\n

Berapa banyak TTY yang dapat dimiliki linux dalam satu waktu? Apakah ada batasannya?

Anda hanya dapat menjalankan konsol virtual (VC) di perangkat tty yang ada. Secara default, Debian Linux dikirimkan dengan 64 file perangkat semacam itu ( / dev / tty0 melalui / dev / tty63 ). Bacaan lain menunjukkan bahwa batas di kernel secara default adalah 256 dari ini, jadi dimungkinkan untuk membuat hingga / dev / tty255 . Jika Anda membutuhkan lebih dari itu, Anda mungkin harus mengkompilasi ulang kernel.

\n\n

Batasan yang lebih praktis, bagaimanapun, adalah bagaimana Anda sebenarnya menggunakan sebanyak itu VC? LeftAlt F1 melalui LeftAlt F12 Anda hanya akan mendapatkan hingga VC 12 pada sebagian besar keyboard (10 untuk keyboard lainnya). Untuk menambah jumlah ini, sebagian besar sistem Linux dikirimkan dengan RightAlt F1 melalui RightAlt F12 dipetakan ke VC 13 hingga 24. Selain itu, Anda dapat melakukan pemetaan keyboard Anda sendiri, yang dapat Anda baca sini jika ini adalah sesuatu yang ingin Anda lakukan.

\n\n

Anda juga bisa menggunakan Alt kiri atau Alt Baik untuk menggilir semua VT. Dan perintah chvt dapat digunakan untuk langsung mengubah ke VT yang aktif.

Saya baru-baru ini menemukan G24, pdf yang diperbarui secara berkala oleh The Guardian dengan ringkasan berita, yang kemudian dapat saya cetak.

\n\n

Apakah ada metode untuk mengunduh pdf tersebut setiap 24 jam? Ini memiliki tautan statis, karena mereka hanya memperbarui pdf, dan mudah diakses (tidak perlu login).

\n\n

tautan: http://download.guardian.co.uk/g24/World.pdf

","source":"https://superuser.com/questions/519939/download-a-pdf-from-a-website-every-24-hours","author":"brunston","replies":[{"reply":"OS Anda? Windows? win7? ?","author":"Logman"},{"reply":"Saya menggunakan Windows 7.","author":"brunston"}],"answers":[{"answer":"
$source = \"http://download.guardian.co.uk/g24/World.pdf\"\n$dest = \"C:\\Users\\Public\\Downloads\\World.pdf\"\n\n$wc = New-Object System.Net.WebClient\n$wc.DownloadFile($source, $dest)\n

Powershell di atas, lalu gunakan Penjadwal Tugas untuk Menjalankan Skrip Windows PowerShell.

\n\n

Untuk menambahkan stempel tanggal ke file, ubah variabel $ dest: 

$dest = \"C:\\Users\\Public\\Downloads\\World $(get-date -f yyyy-MM-dd).pdf\"\n

Nama file = "Dunia 2012-12-15.pdf".

\n\n

Tanggal ini akan menjadi tanggal eksekusi tanggal yang dijadwalkan. Jika file tidak diperbarui, Anda akan mendapatkan pdf yang sama dengan cap tanggal yang berbeda.

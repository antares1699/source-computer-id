Ini tidak mungkin. Recycle Bin ada di setiap drive, dan ketika Anda mengirim file ke sana, tidak ada operasi tulis yang dilakukan. Windows hanya memperbarui Tabel File Master untuk menunjukkan operasi penghapusan yang tertunda, yang kemudian muncul di Recycle Bin untuk drive itu. Jika menghapus file pada drive C: menempatkannya di Recycle Bin pada drive D :, file tersebut sebenarnya akan menjadi tertulis ke drive D :. Juga, Microsoft artikel tentang pengaturan Recycle Bin tidak menyebutkan fitur seperti ini, jadi saya tidak yakin itu didukung.

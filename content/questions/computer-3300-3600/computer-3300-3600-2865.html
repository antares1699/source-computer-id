Saya ingin menonaktifkan driver untuk touchpad saya. Ini disebut "Mouse yang Kompatibel dengan PS / 2". Jika saya membuka properti, tombol nonaktifkan berwarna abu-abu. Juga tidak ada opsi untuk menonaktifkan di menu konteks jika saya mengklik kanan di pengelola perangkat. Lampirkan tangkapan layar dari masalah di bawah. Saya menggunakan Windows 8 tetapi saya menduga masalahnya akan sama di Windows 7.

Klik tombol kanan mouse untuk "Mouse Kompatibel PS / 2"> pilih update driver> driver dari komputer Anda (tidak secara otomatis)> pilih driver dari daftar> hapus klik "Tampilkan perangkat keras yang kompatibel" ... dan pilih jenis lain (saya memilih mouse dari Acer - API USB). Touchpad Anda akan dinonaktifkan setelah komputer Anda dihidupkan ulang. Kemudian (pasti) klik nonaktifkan driver di pengelola perangkat Anda.

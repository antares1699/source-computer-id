Saya mengklik tombol 'Map network drive' di 'Komputer' server saya. Ada baris yang menanyakan 'folder' atau pada dasarnya lokasi drive jaringan. Saya memiliki server lain yang berjalan pada IP 10.132.2.98. Saya dapat memetakan drive jaringan ke alamat '\\ 10.132.2.98 \\ myfolder', tetapi saya mendapatkan kesalahan saat mencoba memetakan drive ke '\\ 10.132.2.98'. Apakah ada cara agar saya dapat memetakan seluruh hard drive saya di server kedua?

","source":"https://superuser.com/questions/506360/mapping-an-entire-network-drive-not-a-folder","author":"Lloyd Banks","replies":[{"reply":"Tidak Anda tidak bisa..","author":"PeterMmm"}],"answers":[{"answer":"

Gunakan \\\\10.132.2.98\\C$. Anda harus menjadi anggota grup Administrators lokal komputer jauh agar ini bekerja.

","votes":"6","source":"https://superuser.com/a/506361","author":"user207421","replies":[]},{"answer":"

Anda juga dapat melakukannya di baris perintah: 

net use x: \\\\10.132.2.98\\C$\n

Untuk masuk sebagai pengguna lain: 

net use x: \\\\10.132.2.98\\C$ /user:username password\n

Ini akan memasang c: drive dari 10.132.2.98 sebagai x:.\nYakinkan Saham Administratif diaktifkan di komputer induk atau pengguna Anda memiliki hak administratif di komputer induk.

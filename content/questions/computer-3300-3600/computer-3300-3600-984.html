Menggunakan PowerShell, saya menemukannya Invoke-Item adalah perintah kunci: 

Invoke-Item \"C:\\Program Files\\Internet Explorer\\iexplore.exe\"\n

Faktor lain:

\n\n
    \n

  1. Perintah hanya berfungsi untuk saya ketika saya memberikan path lengkap ke iexplore.exe.

    2. \n

  2. Anda mungkin mengalami masalah eksekusi skrip jika Anda menyimpan instruksi dan menjalankan dari file .ps1. Dalam kasus tersebut, jalankan yang berikut ini dari prompt PowerShell yang ditinggikan: 

    Set-ExecutionPolicy RemoteSigned\n
    3. \n

Metode yang dicoba dan benar di setiap versi Windows yang relevan: 

iexplore.exe google.com\n

akan membuka IE dan menavigasi ke google. Saya menggunakan invoke-item untuk beberapa hal, tetapi ini dapat dilakukan dari kotak run (yang dinyatakan OP) yang menghemat penekanan tombol. Ia juga bekerja di cmd dan PowerShell, jalur b / c iexplore ada di variabel env PATH.

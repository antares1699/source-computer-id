Untuk gambar:

\n\n

File -> "kurangi ukuran file" dari sana Anda dapat memilih berbagai opsi termasuk memampatkan semua atau hanya gambar yang dipilih. Anda juga dapat meminta file menghilangkan data daerah yang dipotong yang mungkin masih mendukung gambar.

\n\n

Ini tidak akan membantu Anda mengidentifikasi gambar yang menyinggung tetapi merupakan cara cepat untuk memampatkan semua gambar untuk menghilangkannya sebagai kemungkinan pelanggar.

Makro ini akan memberi tahu Anda PPI dari setiap gambar dan menyarankan pengurangan atau peningkatan ukuran. Saya menemukannya di komunitas microsoft. Seperti yang dicatat, makro dibuat oleh Richard Michaels. FYI, itu menyisipkan komentar untuk setiap gambar, jadi jika Anda memiliki banyak komentar, itu bisa membuat berantakan. 

Sub PixelsMatter()\n    'Created by Richard V. Michaels\n    'http://www.greatcirclelearning.com\n    'Creating custom and off-the-shelf productivity apps for Office\n\n    On Error GoTo ErrHandler\n\n    Dim doc As Word.Document, rng As Word.Range, iRng As Word.Range\n    Dim shp As Word.Shape, iShp As Word.InlineShape\n    Dim PixelCount As Integer, FullWidth As Integer, PPI As Integer\n    Dim Mac As Boolean\n\n    Set doc = Word.ActiveDocument\n    Set rng = Word.Selection.Range\n    'Check only the range selected, else check entire body of the document\n    If rng.Start = rng.End Then Set rng = doc.Content\n\n    #If Win32 Or Win64 Then\n        'this is a PC\n        PixelCount = 96\n    #Else\n        'this is a Mac\n        PixelCount = 72\n        Mac = True\n    #End If\n\n    For Each iShp In rng.InlineShapes\n        'only looking for embedded or linked pictures\n        If iShp.Type = wdInlineShapeLinkedPicture Or iShp.Type = wdInlineShapePicture Then\n            'determining original width before scaling\n            FullWidth = iShp.Width / (iShp.ScaleWidth / 100)\n            'calculate PPI density based on the current scaled size of inserted image\n            PPI = FullWidth / (iShp.Width / PixelCount)\n            Select Case PPI\n                Case Is < 150\n                    iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \" and will result in poor print quality. \" & _\n                            \"We suggest either reducing the size of the picture in the document or replacing with a higher quality source image.\"\n                Case Is < 200\n                    iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \", which is marginal for a good print quality. \" & _\n                            \"We suggest you print a sample and check if the print quality is satisfactory for your needs. \" & _\n                            \"Higher print quality can be achieved by either reducing picture size or replacing with a higher quality source image.\"\n                Case Is < 240\n                    'PPI density is optimal, no comment made\n                Case Else\n                    iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \" and does not contribute to better print quality... \" & _\n                            \"it is only creating a larger than necessary file size for this document. We suggest replacing the image with a more appropriately sized source image.\"\n            End Select\n        End If\n    Next\n\n    If Mac Then GoTo ErrHandler\n    'With a Mac running Office 2011 there is no need to go further\n    'The excessively buggy Office 2011 VBA errantly places all \"floating\" shapes into the inline shapes collection\n    'No other PC version of Office VBA does this and if the following code is executed on a Mac, double comments\n    'would be placed on all floating shapes that met the PPI criteria specified in the following Select Case command.\n\n    If doc.Shapes.count = 0 Then GoTo ErrHandler\n    Dim wrapType As Integer, i As Integer\n\n    For i = doc.Shapes.count To 1 Step -1\n        Set shp = doc.Shapes(i)\n        If shp.Type = Office.MsoShapeType.msoPicture Or _\n            shp.Type = Office.MsoShapeType.msoLinkedPicture Then\n            If shp.WrapFormat.Type <> Word.WdWrapType.wdWrapNone Then\n                wrapType = shp.WrapFormat.Type\n                Set iShp = shp.ConvertToInlineShape\n                If iShp.Range.InRange(rng) Then\n                    'determining original width before scaling\n                    FullWidth = iShp.Width / (iShp.ScaleWidth / 100)\n                    'calculate PPI density based on the current scaled size of inserted image\n                    PPI = FullWidth / (iShp.Width / PixelCount)\n                    Select Case PPI\n                        Case Is < 150\n                            iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \" and will result in poor print quality. \" & _\n                                    \"We suggest either reducing the size of the picture in the document or replacing with a higher quality source image.\"\n                        Case Is < 200\n                            iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \", which is marginal for a good print quality. \" & _\n                                    \"We suggest you print a sample and check if the print quality is satisfactory for your needs. \" & _\n                                    \"Higher print quality can be achieved by either reducing picture size or replacing with a higher quality source image.\"\n                        Case Is < 240\n                            'PPI density is optimal, no comment made\n                        Case Else\n                            iShp.Range.Comments.Add iShp.Range, \"PPI is \" & PPI & \" and does not contribute to better print quality... \" & _\n                                    \"it is only creating a larger than necessary file size for this document. We suggest replacing the image with a more appropriately sized source image.\"\n                    End Select\n                   iShp.ConvertToShape\n                   shp.WrapFormat.Type = wrapType\n                Else\n                    iShp.ConvertToShape\n                    shp.WrapFormat.Type = wrapType\n                End If\n            End If\n        End If\n    Next\n\nErrHandler:\n    Select Case Err\n        Case 0\n            MsgBox \"Action Complete\", vbInformation, \"Pixels Matter\"\n        Case Else\n            MsgBox Err.Number & vbCr & Err.Description, vbExclamation, \"Pixels Matter\"\n            Err.Clear\n    End Select\n\nEnd Sub\n

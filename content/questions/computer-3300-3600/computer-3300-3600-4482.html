Bergantung pada apa yang Anda gunakan untuk menjalankan MySQL, Anda dapat melihatnya instruksi resmi .

\n\n

Jika Anda menggunakan MAMP atau sesuatu yang serupa Anda perlu mencari di mana mysqladmin berada. 

/Applications/MAMP/Library/bin/mysqladmin -u root -p password ’somepassword’\n

Jika Anda menggunakan instalasi mysql lain, Anda mungkin dapat mengetahui di mana lokasinya dengan: 

$ which mysqladmin \n

Berikut adalah prosedur untuk mereset kata sandi pengguna root.

\n\n

1) Hentikan mysql (Bunuh proses mysql atau jalankan perintah berikut) 

sudo /usr/local/mysql/support-files/mysql.server stop\n

2) Mulailah dalam mode aman 

sudo mysqld_safe --skip-grant-tables\n

3) Buka terminal lain dan jalankan perintah berikut (Biarkan terminal terakhir tetap terbuka) 

mysql -u root\n

4) Jalankan perintah berikut dengan kata sandi baru yang sesuai di konsol mysql 

mysql > UPDATE mysql.user SET Password=PASSWORD('password') WHERE User='root';\n

5) mysql > FLUSH PRIVILEGES;

\n\n

6) Keluar dari kedua terminal dan buka terminal baru dan hubungkan ke mysql dengan pengguna root dan kata sandi baru 

mysql -uroot -p\n

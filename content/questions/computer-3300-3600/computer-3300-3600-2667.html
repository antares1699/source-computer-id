Ketika saya menjalankan kueri SELECT di meja kerja MySQL, kisi hasil muncul. Kemudian, saat saya menjalankan UPDATE, kisi menghilang. Kemudian ketika saya menjalankan SELECT lagi untuk melihat perubahan, itu muncul kembali.

\n\n

Ini sangat mengganggu dan menyebabkan layar terus berfluktuasi ke atas dan ke bawah. Apakah ada cara agar saya dapat memberikan grid hasil sebuah rumah permanen di suatu tempat?

\n\n

Bahkan jika itu hanya diam di suatu tempat dan dihapus dari data ketika UPDATE berjalan, itu akan sempurna.

Inilah salah satu cara Anda dapat melakukannya. Sematkan tab hasil untuk memaksanya tetap dan tidak terpengaruh oleh pernyataan UPDATE. Lakukan itu dengan mengklik kanan tab hasil dan pilih "Pin Tab" dari menu konteks, atau klik kiri ikon pin kecil untuk mengaktifkannya. Sekarang, jalankan kueri Anda yang lain dan kemudian segarkan tab yang disematkan (ada ikon "segarkan" di menu kisi hasil).

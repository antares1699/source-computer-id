Kemungkinan Duplikat:
\n Garis vertikal untuk margin kanan di Notepad ++

\n
\n\n\n\n

Hai, saya menggunakan Notepad ++. Di beberapa IDE seperti Netbeans, kita dapat melihat garis vertikal di editor kode untuk menekankan jumlah karakter yang disarankan per baris. Apakah ada plugin untuk notepad ++ yang membantu hal yang sama?

\n\n

\n\n

Pada gambar di atas, Anda dapat melihat garis vertikal tipis di editor.

","source":"https://superuser.com/questions/262192/code-length-limiter-in-notepad","author":"bdhar","replies":[],"answers":[{"answer":"

Buka Pengaturan -> Preferensi ... -> Pengeditan -> Tampilkan tepi vertikal.\nSetel nilai "Jumlah Kolom". Ini akan menunjukkan garis vertikal di layar.\nPilih teks yang lebarnya ingin Anda atur.\nLalu pergi ke TextFX Edit >> ReWrap teks dengan lebar (Clipboard atau 72)\nIni akan mengatur lebar menjadi 72 secara default.\nJika Anda ingin mengatur lebar tertentu daripada menulis lebar atau angka itu (katakanlah 80) di mana saja dan salin dan hapus.\nKemudian pergi ke TextFX Edit >> ReWrap teks dengan lebar (Clipboard atau 72)\nKali ini akan mengatur lebar teks yang dipilih menjadi 80.

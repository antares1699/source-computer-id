\n\n

Berikut ini solusi siap pakai: 

Public Sub Demo()\n  Dim cel As Range\n  Dim myRange As Range\n\n  Set myRange = Range(\"A1:E10\")\n\n  For Each cel In myRange\n    cel.Interior.Color = RGB(cel.Value, cel.Value, cel.Value)\n    If cel.Value > 127 Then\n      cel.Font.Color = RGB(0, 0, 0)\n    Else\n      cel.Font.Color = RGB(255, 255, 255)\n    End If\n  Next cel\nEnd Sub\n

Edit:

\n\n

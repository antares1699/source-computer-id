Saya memiliki beberapa aturan dalam pandangan saya.
Baru-baru ini saya telah menambahkan aturan yang menunda pesan keluar dengan menit "x".
Saya ingin membagikan aturan ini dengan beberapa kolega saya yang merasa ini berguna.
Ketika saya mencoba mengekspor aturan dari dialog Rules and Alerts - tidak memungkinkan saya mengekspor satu aturan (atau set tertentu).

\n\n

Bagaimana saya bisa mengekspor hanya seperangkat aturan yang dipilih dari pandangan saya?

\n\n
    \n
  • ekspor semua aturan Anda ke file bernama ALL.rwz (sebagai contoh)
    • \n
  • hapus semua aturan tetapi pertahankan yang ingin Anda bagikan dengan kolega Anda
    • \n
  • ekspor aturan Anda (sekarang hanya satu) ke file lain bernama Myrule.rwz (sebagai contoh)
    • \n

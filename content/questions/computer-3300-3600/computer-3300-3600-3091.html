Saya menjalankan OS tamu Windows 7 VMware Player dari Windows 7. Namun, ketika saya memasukkan flash drive USB, OS tamu tidak dapat mengaksesnya. VMware Player memberi tahu saya bahwa tongkat terhubung, tetapi tidak muncul di OS tamu 'komputer saya'. Hal yang sama berlaku dengan kartu jaringan saya. VMware Player mendeteksinya, tetapi saya tidak dapat bergabung dengan jaringan apa pun. Bagaimana cara memperbaiki masalah ini?

Untuk USB:

\n\n

Pastikan Anda memiliki pengaturan pengontrol USB untuk VM:

\n\n

Menu Mesin Virtual-> Pengaturan Mesin Virtual

\n\n

Jika Anda tidak melihat "Pengontrol USB" dalam daftar, matikan sistem operasi tamu mesin virtual. Klik "Tambah" di bagian bawah jendela Pengaturan Mesin Virtual, klik "Pengontrol USB" dalam daftar, klik "Berikutnya" dan klik "Selesai".

\n\n

Kemudian pastikan itu terhubung ke tamu dan bukan tuan rumah:

\n\n

Menu Mesin Virtual-> Perangkat yang Dapat Dilepas -> -> Hubungkan (Putuskan Sambungan dari Host) "

\n\n

Jika pengontrol jaringan Anda juga USB, ini dapat membantu Anda membuatnya berfungsi juga. Jika tidak, mungkin hanya masalah memastikan adaptor kompatibel dengan VMWare dan driver dimuat dengan benar ke OS tamu.

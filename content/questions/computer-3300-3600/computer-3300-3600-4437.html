Saya lupa kata sandi pada netbook Windows 7 saya dan saya tidak memiliki disk pengaturan ulang kata sandi. Bagaimana saya bisa masuk?

\n\n

Ini adalah laptop baru jadi saya tidak keberatan melakukan pemulihan sistem tetapi saya bahkan tidak bisa melakukannya karena saya tidak memiliki drive CD dan saya tidak tahu kata sandinya.

","source":"https://superuser.com/questions/154722/how-to-reset-the-password-on-a-windows-7-netbook","author":"Ben","replies":[],"answers":[{"answer":"

Di XP biasanya ada akun administrator tersembunyi yang dibuat saat Anda menginstal sistem. Ini adalah akun yang hanya dapat diakses saat Anda boot ke safe mode, dan secara default tidak memiliki kata sandi.

\n\n

Anda dapat mencoba mem-boot ke mode aman Windows 7 dan melihat apakah ini masih benar. Untuk masuk ke mode aman, cukup tekan F8 saat komputer Anda boot. Lihat halaman ini untuk info lebih lanjut.

Di XP, Anda dapat mengakses akun administrator tersembunyi dalam boot normal dengan menekan ctrl-alt-del dua kali. Dialog seperti Windows 2000 akan muncul sehingga Anda dapat memasukkan administrator nama pengguna. Jika tidak ada kata sandi yang terkait dengan akun itu, itu tidak perlu dipikirkan lagi. Saya akan berkomentar tetapi belum memiliki hak istimewa itu.

\n\n

Windows 7 tidak mengizinkan Anda menggunakan metode ini. Anda dapat mencoba mem-boot ke mode aman untuk melihat apakah itu berfungsi.

Jika Anda memiliki editor dengan dukungan UTF-8, Anda dapat memasukkan Karakter gambar kotak Unicode dan bentuk geometris langsung. Sesuatu seperti: 

PS1='┌─[blah][blah]\n└─▪ '\n

Tulis ini ke file menggunakan editor Anda, lalu sumber file tersebut ke sesi bash Anda dengan source ./myfile. Saat Anda siap untuk menyimpannya, tambahkan ke file .bashrc Anda.

\n\n

Cara lain yang lebih lama untuk melakukannya adalah dengan menggunakan kode escape VT100 untuk beralih ke kumpulan karakter gambar garis terminal. Dalam kasus ini, baris PS1 Anda terlihat seperti ini: 

PS1='\\[\\e(0\\]lq\\[\\e(B\\][blah][blah]\n\\[\\e[0\\]mq~\\[\\e(B\\] '\n

\\[ dan \\] memberitahu bash bahwa bagian dari prompt ini tidak dicetak, yang membantunya mengetahui di mana kursor berada saat melakukan pengeditan baris perintah. \\e(0 beralih ke kumpulan karakter penggambaran garis, dan \\e(B beralih kembali. Saat dalam mode menggambar garis:

\n\n
    \n
  • l adalah ┌
    • \n
  • q adalah ─
    • \n
  • m adalah └
    • \n
  • ~ adalah ▪
    • \n
\n\n

Anda dapat membuat daftar lengkap dengan menjalankan hal-hal seperti printf '\\e(0 qwertyuiop \\e(B \\n' pada prompt bash.

Anda dapat mengatur laptop Anda untuk menggunakan EnableDynamicScaling dengan membuka regedit dan mengubah EnableDynamicScaling dari 0 ke 1.

\n\n

Berita buruk: Microsoft menjadi licik dan sekarang telah menonaktifkan fungsi itu di Windows 8.1. Setelah Anda mengatur skala ke 1 dan mengubah resolusi Anda menjadi 1024x768 dan menginstal Windows 8.1, Anda tidak akan lagi memiliki 1024x768 atau DynamicScaling untuk netbook dengan resolusi maksimal 1024x600. Setelah Anda meningkatkan, Anda terjebak.

\n\n

Di regedit, buka Edit - Temukan, dan masukkan Display1_DownScalingSupported. Pada semua kejadian, ubah nilainya menjadi 1. F3 menemukan kejadian berikutnya. Setelah reboot resolusi lain harus tersedia.

\n\n

Jika perangkat keras Anda mengizinkannya, sambungkan sementara monitor eksternal dengan resolusi minimal 1024x768. Anda kemudian dapat menggunakan Store untuk mengunduh pemutakhiran Windows.

Saya memiliki komputer windows 7 64 bit yang gagal menginstal pembaruan kb2918614. Ini telah menginstal semua pembaruan lainnya.

\n\n

Setiap kali mencoba untuk menginstal, gagal dengan kode kesalahan 8024200d - mencoba lagi setiap kali saya me-restart komputer.

\n\n

Saya telah mencoba mengganti nama C:\\Windows\\SoftwareDistribution\\ menjadi C:\\Windows\\SoftwareDistribution.old dan menginstal lagi, tetapi hal yang sama terjadi.

\n\n

Saya juga mencoba menjalankan "alat kesiapan Sistem Pembaruan Windows", tetapi itu juga tidak berhasil.

\n\n

Adakah yang bisa membantu saya?

Kesalahan 8024200d berarti WU_E_UH_NEEDANOTHERDOWNLOAD (The update handler did not install the update because it needs to be downloaded again.)

\n\n

Unduh dan instal Pembaruan dengan bantuan File MSU dari pusat unduhan , alih-alih menggunakan Pembaruan Windows.

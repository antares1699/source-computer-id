Jika Anda mengunduh '.NET Framework SDK Version 1.1', Anda dapat mengekstrak isinya.\nDi dalam Anda akan menemukan netfxsd1.cab.\nDi dalamnya Anda akan menemukan 'FL_setreg_exe _____ X86.3643236F_FC70_11D3_A536_0090278A1BB8' yang merupakan setreg.exe yang Anda cari. (Cukup ganti nama file. Ini persis sama dengan versi yang disertakan dengan W2K.)

\n\n

'setup.exe' adalah file SFX .CAB.

\n\n

Jadi jika Anda memiliki metode untuk membuka file .CAB (sekali lagi ganti nama dari .exe menjadi .exe.cab) cukup mudah untuk membukanya atau Anda bisa menggunakan alat, uniextract.

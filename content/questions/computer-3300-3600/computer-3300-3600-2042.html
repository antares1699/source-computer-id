Begitu pula saya mendapatkan hak itu.

\n\n

Pengguna "bla" ingin menjalankan perintah sebagai Pengguna "blub"?

\n\n

Kemudian sesuatu seperti ini bisa berhasil: 

sudo -u blub /path/to/command\n

Jika Anda ingin ini dilakukan tanpa diminta kata sandi, tambahkan baris berikut ke / etc / sudoers (melalui perintah visudo): 

bla    ALL=(blub) NOPASSWD: /path/to/command\n

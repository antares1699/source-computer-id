Saya mencari cara bagi pengguna untuk meluncurkan aplikasi Windows yang hanya akan berjalan selama jangka waktu tertentu, katakanlah satu jam. Setelah satu jam berlalu, jendela peringatan akan muncul, mengatakan "Program akan dihentikan dalam lima menit, kecuali Anda mengklik di sini". Jika pengguna mengklik tombol, program akan berjalan selama satu jam lagi, setelah itu peringatan akan muncul lagi.

\n\n

Jika tidak ada cara mudah untuk melakukannya, maka saya ingin program tersebut dihentikan begitu saja setelah waktu tertentu, tanpa peringatan pop-up.

\n\n

Idealnya, metode ini harus bekerja pada Windows XP dan Windows 7.

Saya menjawab pertanyaan saya sendiri. Sebenarnya, pertanyaan tersebut telah dijawab secara akurat dan lengkap oleh pengguna BigBadWolf_000 di Pertukaran Ahli . Saya menyalin kode VBScript di bawah ini: 

Dim PromptTime, DelayTime, StrAppPath, AppExec, MsgTxt, intAnswer, intRet\n\nSet objShell = CreateObject(\"WScript.Shell\")\n\nPromptTime = 60\nDelayTime = 5\nStrAppPath = \"C:\\windows\\\"\nAppExec = \"notepad.exe\"\nMsgTxt = \"Do you want Notepad to close in 5 minutes?\"\n\nobjShell.Run chr(34) & StrAppPath & AppExec & chr(34), 1, \"False\"\nDo\n    WScript.Sleep (1000 * 60 * PromptTime)\n    intAnswer = Msgbox(MsgTxt, vbYesNo, \"Please select Yes or No\")\n    If intAnswer = vbYes Then Exit Do\nLoop\n\nWScript.Sleep (1000 * 60 * DelayTime)\nSet objWmi = GetObject(\"winmgmts:\") \nSet objQResult = objWmi.Execquery(\"Select * from Win32_Process where name like '\" & AppExec & \"'\")\nFor Each objProcess In objQResult \nintRet = objProcess.Terminate(1) \nNext \n\nSet objShell = Nothing\nSet objWmi = Nothing \nSet objQResult = Nothing\n

Berikut adalah komentar penjelasan yang disertakan dengan kode BigBadWolf_000: 

The script below will do the following...\n- lauch your application (app must be launched with this script)\n- Promt you for action after an hour \"Do you want YourApp to close in 5 minutes?\" Yes/No\n- Yes: Terminates all instances of the app after 5 minutes\n- No: Restarts the clock and will prompt you again in an hour\n\nThe default PromptTime is set to 60 minutes and the terminate app DelayTime is set to 5 minutes.\nYou can change number for PromptTime = 60 and DelayTime = 5 to any number in minutes (minimum 1). The default app is set to Notepad so you can test.\n\nChange the path of the app executable...\nStrAppPath = \"your path goes here, end with  \\\" \n\nChange the name of the app executable...\nAppExec = \"whatever.exe\"\n\nChange the popup message txt...withing the quotes\nMsgTxt = \"Do you want Notepad to close in 5 minutes?\"\n\nCopy and paste script to notepad and save as yourfilename.vbs. Double-click on the file to run or create a shortcut to it. Will work with Windows 7 and Windows XP SP3.\n

Jika Anda memiliki XP atau yang lebih baru, Anda dapat menggunakan TASKKILL. Ini bukan NT sekalipun.

\n\n

Saya pikir Anda harus mengunduh sesuatu untuk melakukan ini. Saya akan merekomendasikan pskill dari Sysinternals .

\n\n

Anda dapat menggunakan ini baik dengan ID proses atau hanya dengan nama proses. Sebagai contoh: 

pskill notepad.exe\n

Pilihan lainnya adalah KILL dari NT Resource Kit .

Kami dapat menggunakan VBScript dan WMI untuk menghentikan proses: 

Dim strComputer,objWMIService,objProcessList,objProcess\nstrComputer = "."\nSet objWMIService = GetObject("winmgmts:" & "{impersonationLevel=impersonate}!\\\\" & strComputer & "\\root\\cimv2")\nSet objProcessList = objWMIService.ExecQuery("SELECT * FROM Win32_Process WHERE Name = 'THE PROCESS NAME'")\nFor Each objProcess in objProcessList\n  objProcess.Terminate()\nNext\n

Atau di edisi windows modern, kita bisa menggunakan Powershell: 

Stop-Process -Name "PROCESS NAME WITHOUT EXTENSION"\n

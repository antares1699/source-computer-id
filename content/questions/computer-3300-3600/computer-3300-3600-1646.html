Saya dapat mengatur pengaturan jenis file khusus untuk jenis file tahu menggunakan file *.vim di direktori ftplugin, mis. c.vim untuk (c, cpp, h, hxx), python.vim dll.

\n\n

Tetapi saya ingin mengatur jenis file untuk ekstensi file kustom seperti *.text, *.letter, *.journal dll. Saya mencoba menggunakan text.vim dan seterusnya tetapi tidak berhasil.

","source":"https://superuser.com/questions/157252/vim-how-to-set-preference-for-custom-file-types","author":"Xolve","replies":[],"answers":[{"answer":"

Anda sedang mencari perintah otomatis . Tambahkan baris untuk setiap ekstensi file kustom di .vimrc Anda: 

au! BufRead,BufNewFile *.ext setfiletype ext\n

Kemudian Anda dapat menentukan pengaturan untuk file .ext di ftplugin/ext.vim.

\n\n

PS: Perhatikan bahwa perintah otomatis harus ditentukan di dalam augroup.

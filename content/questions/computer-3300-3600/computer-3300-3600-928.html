Tampaknya tidak ada pintasan keyboard resmi yang terpasang di Chrome untuk menu Ekstensi. Namun, tanpa menggunakan software tambahan, masih ada cara untuk dapat mengakses menu dengan relatif cepat.

\n\n

Anda dapat menambahkan menu Extensions sebagai bookmark di bilah bookmark. URL-nya adalah chrome: // extensions /.

\n\n

\n\n

Ini membuatnya satu klik. Namun, jika Anda ingin mengaksesnya dari keyboard saja, ada cara lain. Klik kanan Omnibar dan pilih "Edit mesin telusur". Di bawah "Mesin telusur lainnya", isi informasi seperti yang ditunjukkan, lalu pilih Selesai.

\n\n

\n\n

Ini menciptakan "mesin pencari" yang dapat diakses melalui kata kunci 'ex'. Sekarang, untuk mengakses Ekstensi, Anda dapat menekan Ctrl + L untuk memilih Omnibar, ketik ex, dan tekan Memasukkan . Tentu saja, Anda dapat menyesuaikan kata kunci "ex".

Karena Ekstensi adalah item menu di Chrome, Anda dapat menetapkannya sebagai pintasan keyboard melalui Preferensi Sistem OS X.

\n\n
    \n
  • Buka Panel 'Keyboard'
    • \n
  • Pilih tab 'Pintasan'
    • \n
  • Pilih 'Pintasan Aplikasi' dari kolom kiri
    • \n
  • Klik + di bawah kolom kanan untuk menambahkan Google Chrome sebagai Aplikasi dalam daftar.
    • \n
  • Di munculan, ketik "Ekstensi" (tanpa tanda kutip) di ruang di samping 'Judul Menu'
    • \n
  • Tetapkan pintasan keyboard (saya menggunakan ⌥ + ⌘ untuk semua pintasan pribadi saya, jadi saya gunakan ⌥ + ⌘ + E untuk yang ini)
    • \n
\n\n

Selesai.

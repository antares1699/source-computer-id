Saya mencoba menggabungkan beberapa file audio di ffmpeg. Saya menggunakan perintah berikut: 

 ffmpeg -f concat -i myfile.txt -c copy output.mp3\n

Di mana myfile.txt berisi jalur file audio. Saya dapat bergabung dengan audio mp3, tetapi apakah mungkin untuk menggabungkan audio format lain (seperti avi, wav, m4a, dll.)?

Gunakan ini: 

ffmpeg -i input1.wav -i input2.mp3 -i input3.aac -filter_complex 'concat=n=3:v=0:a=1[a]' -map '[a]' <encoding options> output.mp3\n

Di sini kita mengeluarkan ke mp3 dari 3 input codec audio yang berbeda. Anda harus memasukkan parameter pengkodean mp3. Saya berasumsi bahwa file masukan Anda hanya audio.

\n\n

Parameter yang baik untuk mp3 mungkin 256 kbps. Kemudian gunakan ini: 

-codec:a libmp3lame -b:a 256k\n

Tetapi ini dianggap boros untuk mp3 dan Anda mungkin ingin menggunakan: -q:a 3

\n\n

Lihat ini dan juga ini .

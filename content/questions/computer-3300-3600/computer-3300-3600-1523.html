Saya menggunakan curl ( http://curl.haxx.se/ ). Ini memiliki banyak fitur bagus untuk unduhan massal. Ini memungkinkan Anda menentukan banyak file dalam satu pemanggilan, seperti ini: 

CURL.EXE http://example.com/file[1-10].mp3\n

Itu akan mengunduh "file1.mp3", "file2.mp3", dan seterusnya.

\n\n

Saya telah menggunakannya untuk mengunduh banyak file, tetapi tidak 5.000. Namun, pada prinsipnya itu harus berhasil.

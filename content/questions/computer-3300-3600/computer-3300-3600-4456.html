Saya menemukan bahwa ketika saya mengetik karakter Cina (dikodekan dengan UTF-8) ke dalam VIM, saya tidak dapat melihatnya sama sekali saat mereka ada di sana. Saya dapat menyalin dan menempelkannya ke editor teks lain dan tampaknya semuanya baik-baik saja. Bagaimana cara memperbaiki masalah ini?

","source":"https://superuser.com/questions/141956/cant-see-the-chinese-characters-in-vim","author":"SpawnST","replies":[],"answers":[{"answer":"

Anda tidak menggunakan fonta yang berisi mesin terbang yang diperlukan untuk menampilkan karakter yang Anda ketikkan.

\n\n

jadi, pilih font yang tepat via 

:set guifont=*\n

atau 

:set guifontwide=*\n

yang akan memunculkan dialog pemilihan font. dan setelah Anda memiliki font yang berfungsi membuatnya permanen dengan meletakkan 

set guifont=YOURFONT\n

ke .gvimrc / .vimrc Anda

Buka file konfigurasi VIM 

$ sudo -H gedit /etc/vim/vimrc\n

Menambahkan baris berikut: 

set fileencodings=utf-8,ucs-bom,gb18030,gbk,gb2312,cp936\nset termencoding=utf-8\nset encoding=utf-8\n

Simpan dan keluar, dan perintah terminal: 

$ source /etc/vim/vimrc\n

Saat ini VIM akan menampilkan bahasa Mandarin dengan benar.

ok, jika Anda membuat modifikasi berikut ke registri Anda, opsi akan tersedia lagi di netplwiz. Pengaturan default adalah 2. 

Windows Registry Editor Version 5.00\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\PasswordLess\\Device]\n"DevicePasswordLessBuildVersion"=dword:00000000\n

0 = aktif
2 = mati

Kamu bisa gunakan Autologon.exe (Alat Sysinternals) atau menggunakan kunci registri: 

Windows Registry Editor Version 5.00\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Winlogon]\n"AutoAdminLogon"="1"\n"DefaultDomainName"="DOMAINNAME"\n"DefaultUserName"="USERNAME"\n"DefaultPassword"="PASSWORD"\n

Diekstrak dari: http://www.sysadmit.com/2016/01/windows-configurar-autologin.html

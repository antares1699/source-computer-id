","source":"https://superuser.com/questions/19639/how-can-i-activate-the-window-my-mouse-is-currently-hovering-over","author":"Jeff Leonard","replies":[],"answers":[{"answer":"

Alat tikus X tepat seperti yang diperintahkan dokter.

\n\n

Unduh X Mouse Di Sini

\n\n

Jika Anda memikirkan lebih banyak tentang peretasan registri ...

\n\n

Windows memang mendukung fokus-mengikuti-mouse, meskipun tidak ada konfigurasi GUI yang mengeksposnya. Sebaliknya Anda harus mengedit kunci registri dan kemudian keluar dan masuk kembali agar perubahan menjadi efektif. Anda dapat menggunakan regedit untuk mengedit kunci.

\n\n

Di Windows 2000, XP, atau 2003, Anda perlu mengubah kunci registri bernilai biner: 

HKEY_CURRENT_USER\\Control Panel\\Desktop\\UserPreferencesMask \n

Ini adalah bitmask little-endian. Untuk fokus-mengikuti-mouse, tambahkan bendera 0x1. Misalnya, laptop XP SP2 saya awalnya memiliki nilai 9E 3E 05 80, yaitu 0x80053E9E. Untuk mengaktifkan fokus-mengikuti-mouse saya mengubah ke 0x80053E9F, atau 9F 3E 05 80 di regedit.

\n\n

Berdasarkan http://www.winguides.com/registry/display.php/18/ Anda juga bisa mencapai peningkatan pada fokus dengan menambahkan bendera 0x40.

Saya menggunakan Mac OS X dan Ubuntu pada dua komputer berbeda yang disinkronkan. Di Mac saya, saya sering menggunakan alias (bukan dalam arti bash, dalam pengertian Command-L) untuk menautkan ke file di komputer saya. Apakah ada cara untuk melakukan ini agar kompatibel dengan Ubuntu?

\n\n

Secara khusus, saya ingin suatu cara (di luar terminal) untuk membuat tautan ke file lain (yang dapat saya klik) yang berfungsi di Mac OS X dan Ubuntu, yang dapat saya buat di kedua sistem.

","source":"https://superuser.com/questions/37331/alias-links-in-os-x-vs-ubuntu","author":"David Zureick-Brown","replies":[],"answers":[{"answer":"

Gunakan rasa bash (ln -s). Anda membuat tautan simbolik unix, ditangani seperti itu di Ubuntu dan ditampilkan di Finder sebagai alias Mac.

\n\n

Anda harus menggunakan Terminal.

Finder membuat alias, "ln -s" membuat tautan simbolis; ini adalah konsep yang berbeda (misalnya jika yang asli dipindahkan alias akan mengikuti tetapi tautan simbolis tidak.

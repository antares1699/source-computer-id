Saya membeli PC baru dan menginstal Windows 10. Saya memiliki hampir 1TB file di OneDrive, dan saya ingin semuanya tersedia secara offline. Namun, ketika saya mencoba menonaktifkan opsi "Hemat ruang dan unduh file saat Anda menggunakannya" di bawah "Pengaturan", opsi itu tidak mati.

\n

Saya mengklik kotak centang, dan kemudian ada kotak dialog:

\n

"Nonaktifkan File Sesuai Permintaan

\n

Semua file dan folder OneDrive akan diunduh ke PC ini

\n

OK / Batal "

\n

Saya klik OK, lalu ... Kotak centang masih dicentang! Jadi sepertinya tidak ada cara untuk menghapus centangnya. Ini adalah perilaku yang sangat aneh, mungkin bug.

\n

Adakah ide untuk mematikan File Sesuai Permintaan mengingat kotak centang tidak ingin dicentang? Mungkin pengeditan registri atau semacamnya?

\n

\n

Jika opsi File Sesuai Permintaan masih dicentang, Anda mencoba memutuskan tautan PC dan keluar dari OneDrive. Kemudian coba masuk kembali OneDrive. Periksa apakah Anda dapat menghapus centang opsi.

\n

Jika masalah terus berlanjut, Anda dapat mencoba mengatur ulang OneDrive sesuai dengan Setel ulang OneDrive . Harap buat cadangan file jika perlu.

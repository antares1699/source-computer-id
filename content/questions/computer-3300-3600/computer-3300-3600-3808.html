Saya mengambil risiko dan melakukan mengutak-atik file .conf yang saya coba hindari, dan menemukan apa yang perlu saya lakukan.

\n\n

Di /usr/share/X11/xorg.conf.d/ adalah 50-synaptics.conf, yang merupakan file conf yang mengontrol touchpad pada T440p (dan lainnya). Dan di sana ada bagian ini: 

Section \"InputClass\"\n    Identifier \"Default clickpad buttons\"\n    MatchDriver \"synaptics\"\n    Option \"SoftButtonAreas\" \"50% 0 82% 0 0 0 0 0\"\n

Itu adalah pengaturan default untuk area di bidang sentuh yang sesuai dengan tombol klik kanan di bidang sentuh. Setelah banyak, banyak mengutak-atik untuk mendapatkan area yang didefinisikan ulang dengan benar, inilah yang harus disalin-tempel untuk mengubah area klik kanan menjadi strip tipis di area kanan atas touchpad: 

Section \"InputClass\"\n    Identifier \"Default clickpad buttons\"\n    MatchDriver \"synaptics\"\n    Option \"SoftButtonAreas\" \"50% 0 0 45% 0 0 0 0\"\n

Sejauh yang saya mampu lakukan saat ini, tetapi itu menyelesaikan sebagian besar dari apa yang saya inginkan. Saya hanya memposting ulang di sini kalau-kalau ada orang lain yang perlu melihat cara memetakan ulang zona di touchpad Synaptics.

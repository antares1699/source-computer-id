Saya perlu mengetahui jalur Xcode.app. Spotlight tahu di mana menemukan .apps, jadi ada sistem di MacOS untuk menemukan hal-hal seperti itu. Apakah ada alat baris perintah yang akan memberikan informasi ini, misalnya: 

$ tool Xcode\n/Applications/Xcode.app\n

Akses database Spotlight dari baris perintah, mencari aplikasi bernama "Xcode": 

mdfind \"kMDItemDisplayName==Xcode&&kMDItemKind==Application\"\n

Untuk melihat metadata Spotlight untuk file atau folder tertentu, gunakan mdls, mis. mdls /Developer/Applications/Xcode.app.

\n\n
\n\n

Atau, Anda dapat menggunakan AppleScript untuk mendapatkan jalur jika Xcode sedang berjalan. 

tell application \"System Events\"\n    tell application process \"Xcode\"\n        set a to application file\n        POSIX path of a\n    end tell\nend tell\n

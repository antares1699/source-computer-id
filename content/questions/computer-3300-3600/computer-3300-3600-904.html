Jika Windows tidak mencoba untuk boot sama sekali, pertama-tama saya akan mencoba memperbaiki partisi sistem secara manual. Ini dapat dilakukan dengan prosedur yang sama seperti yang Anda gunakan setelah memulihkan WIM. Lihat menggunakan imagex

\n\n
    \n
  1. Boot ke Windows PE baik dari media PE yang telah Anda siapkan sebelumnya, atau dari disk penginstal windows. Jika menggunakan metode media instalasi, Anda ingin membuka prompt perintah. Saya pikir ada cara untuk mengakses ini melalui wizard mengikuti langkah-langkah 'perbaikan', tetapi Anda harus cukup menekan shift + F10 pada halaman 'pilih disk' dari wizard instalasi baru windows untuk membuka prompt perintah.
    2. \n
  2. Jalankan DISKPART dengan mengetik DISKPART.
    3. \n
  3. Dari disk yang terdaftar, identifikasi disk yang merupakan SSD baru.
    4. \n
  4. Ketik 'Select Disk 0' di mana '0' adalah id disk yang diinginkan.
    5. \n
  5. Sekarang ketik 'List Volume' untuk mendaftar partisi.
    6. \n
  6. Identifikasi partisi windows dan partisi sistem.
    7. \n
  7. Ketik 'Pilih Volume 0' di mana '0' adalah partisi windows.
    8. \n
  8. Ketik 'Assign Letter =' W '' untuk memberikan huruf drive W (jangan khawatir, ini tidak akan mengubahnya dari drive C di Windows saat boot berhasil).
    9. \n
  9. Ketik 'Pilih Volume 1' di mana '1' adalah partisi sistem.
    10. \n
  10. Ketik 'Assign Letter =' S '' untuk memberikan huruf drive S
    11. \n
  11. Ketik 'aktif'
    12. \n
  12. Keluar dari Diskpart dengan mengetik 'exit'
    13. \n
  13. Sekarang jalankan 'W: \\ Windows \\ System32 \\ bcdboot W: \\ Windows / l en-US / s S:', perhatikan bahwa Anda mungkin perlu mengubah perintah ini sesuai dengan Opsi BCDboot .
    14. \n
  14. Mulai ulang.
    15. \n
\n\n

Ini bisa menjadi proses temperamental karena ada banyak variabel, tetapi jika Anda mencari cara memulihkan file WIM ke hard drive, dan mempelajari proses pasca-pemulihan untuk membuat partisi sistem, Anda mungkin akan berhasil.

\n\n

Jika prosedur ini gagal, pastikan gambar yang dicadangkan benar-benar menyertakan partisi sistem. Jika Anda tidak memiliki 2 partisi, atau partisi sistem yang diduga tidak diformat, Anda mungkin perlu membuatnya dengan ruang yang tersisa. Anda harus dapat memisahkan langkah-langkah yang diperlukan dari prosedur yang dirinci sini . Ini akan menjadi dua perintah tambahan di bagian disk sebelum Anda menetapkan huruf drive 's'. Sesuatu seperti 'buat partisi ukuran primer = 100' dan format FS = NTFS LABEL = "Volume Sistem" CEPAT '. Jangan lupa untuk mengaktifkan partisi sistem.

\n\n

Semoga berhasil :).

Cara Memperbaiki MBR

\n\n
    \n
  1. Boot ke Disk Perbaikan Windows 7
    2. \n
  2. Pilih bahasa Anda, lalu pilih "Perbaiki Komputer"
    3. \n
  3. Disk akan mencari Instalasi Windows, dalam hal ini tidak akan menemukan apa-apa, lalu pilih tombol radio pertama (Gunakan opsi pemulihan ...) dan tekan Berikutnya
    4. \n
  4. Setelah Anda masuk ke "Startup Recovery Options," pilih Command Prompt
    5. \n
  5. Saat diminta masuk: C:
    6. \n
  6. Kemudian: bootrec.exe /FixMbr
    Ini akan memperbaiki catatan boot master
    7. \n
  7. Mulai ulang
    8. \n
  8. Jika Anda masih tidak bisa boot ke windows, masukkan juga perintah ini bootrec.exe /RebuildBcd
    9. \n

Langkah 1: Pulihkan Partisi - Dari PC dengan instalasi windows

\n\n
    \n
  1. Di Acronis True Home, pilih untuk memulihkan partisi cadangan.
    2. \n
  2. Pilih drive yang ingin Anda pulihkan di bawah "Pilih Tujuan Pemulihan"
    3. \n
  3. Pergilah ke Partition Properties dan ubah tipe menjadi Aktif Utama
    4. \n
  4. Klik OK dan pulihkan partisi
    5. \n
\n\n

Langkah 2: Penyebaran BCD - Dari PC dengan penginstalan windows

\n\n
    \n
  1. Install EasyBCD versi gratis
    2. \n
  2. Setelah partisi dipulihkan di Acronis, Buka Easy BCD dan buka tombol Penyebaran BCD
    3. \n
  3. Pilih partisi yang Anda inginkan dari menu dropdown
    Catatan: Anda mungkin perlu masuk ke manajemen komputer dan memberikannya huruf drive
    4. \n
  4. Klik Instal BCD dan ikuti petunjuknya
    5. \n
\n\n

Langkah 3: Perbaikan Startup - Dari PC asli

\n\n
    \n
  1. Masukkan drive ke PC yang dimaksudkan
    2. \n
  2. Masukkan disk Windows RE atau instal disk
    3. \n
  3. Saat disk dimuat, Anda akan menerima prompt untuk memperbaiki kesalahan
    4. \n
  4. Mulai ulang dan boot ke windows
    5. \n

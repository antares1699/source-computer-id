Saya menggunakan skrip konfigurasi otomatis yang dihosting di server saya. Ini dikonfigurasi untuk menggunakan proxy pada pola URL, tetapi tampaknya gagal. Lebih lanjut tentang masalah ini, ini sebelumnya berfungsi di IE11 ketika saya masih menggunakan Windows 8.1, setelah memutakhirkan ke Windows 10, file pac tampaknya diabaikan.

\n\n

Apakah ada cara untuk memastikan bahwa file PAC digunakan atau apakah ada cara untuk membuat IE atau Edge memaksa memuat ulang file PAC?

","source":"https://superuser.com/questions/952870/how-do-i-force-ie-and-or-microsoft-edge-to-reload-pac-file","author":"skeith","replies":[],"answers":[{"answer":"

Saya tidak bisa mendapatkan Microsoft Edge untuk menggunakan file PAC yang saya atur di Pengaturan Internet. Ini akan menggunakan server proxy. IE 11 di Windows 10 seharusnya tidak memiliki masalah dalam menggunakan PAC seperti yang terjadi di Windows 8. Jika tidak, periksa apakah Automatically detect settings diaktifkan dan coba nonaktifkan.

\n\n

Untuk memastikan IE menggunakan file PAC saat ini, Anda dapat menonaktifkan cache: 

Key: HKEY_CURRENT_USER\\Software\\Policies\\Microsoft\\Windows\\CurrentVersion\\Internet Settings\n\nValue: EnableAutoproxyResultCache (REG_DWORD)\n

1 (default) = aktifkan caching dan 0 = nonaktifkan caching

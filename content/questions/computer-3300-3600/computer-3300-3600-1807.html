\n\n

Apakah ada pintasan keyboard untuk menyimpan file seperti biasa di aplikasi lain - untuk segera menimpanya tanpa pertanyaan tambahan?

","source":"https://superuser.com/questions/1145534/how-to-save-file-in-nano-fast","author":"warvariuc","replies":[],"answers":[{"answer":"

Edit file ~/.nanorc dan tambahkan ini: 

bind ^S savefile main\n

Sekarang ctrl + s akan menyimpan file Anda ke nama file yang sama tanpa diminta.

\n\n

Namun, gunakan fitur ini dengan hati-hati. Ini tidak akan memberi tahu Anda jika ada hal lain yang telah mengubah file sejak penyimpanan terakhir Anda. Jika itu bukan masalah yang mungkin Anda alami, ini akan bekerja dengan baik!

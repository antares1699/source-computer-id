Apakah mungkin untuk memindahkan folder pengguna (bukan folder home itu sendiri) mis. dokumen, foto, musik dll ke lokasi lain mis. di dalam saya Dropbox di Mac? Jika demikian, bagaimana cara melakukannya? Pada kasus ini? Tidak. Namun, Anda dapat mengatasinya dengan melakukan salah satu hal berikut:

\n\n
    \n

  1. Ubah lokasi folder Dropbox Anda
    Untuk melakukannya, buka preferensi Dropbox dan tekan Pindah ... dan pilih folder utama Anda. Ini bukan pilihan yang saya rekomendasikan. (Lebih mudah dan lebih mungkin dengan Dropbox 0.8.x dengan sinkronisasi selektif dan dapat memindahkan folder Dropbox Anda)

    2. \n

  2. Gunakan Symlinks
    Anda dapat menggunakan symlink untuk membuatnya tampak seolah-olah folder Dokumen, Film, dll. Anda ada di folder Dropbox tanpa benar-benar merusak sistem file (dan membuat folder dibuat ulang) yang diharapkan Mac OS X.

    \n\n

    Anda harus melakukan ini di Terminal tetapi Anda dapat menjalankan ln -s /Users/YOUR_SHORT_USERNAME/Documents /Users/YOUR_SHORT_USERNAME/Dropbox/Documents menggantikan kata Dokumen untuk folder mana pun yang Anda perlukan.

    \n\n

    Anda dapat melihat lebih banyak tentang ini di Wiki Dropbox

    3. \n

Pertama-tama salin folder Dokumen Anda ke Dropbox, lalu buka terminal OS X dan lakukan hal berikut: 

sudo mv Documents OldDocuments\nln -s /Users/$(whoami)/Dropbox/Documents Documents\n

Setelah itu, Anda dipersilakan untuk menghapus OldDocuments, meskipun Anda mungkin perlu melakukan: 

sudo rm OldDocuments\n

Saya menunggu Dokumen saya selesai disinkronkan ke DropBox sebelum saya mencoba menghapus.

Menggunakan klien Windows 7 32-bit Saya mencoba membuat ikon desktop yang memungkinkan saya masuk ke sesi konsol Windows Server 2003.

\n\n

Ketika saya membuka RDP melalui "mstsc / admin" dan kemudian masuk ke server, itu akan terhubung ke konsol. Namun, jika saya membuka "mstsc / admin" lalu mencoba menyimpan profil itu ke desktop saya, profil itu tidak terhubung sebagai sesi konsol.

\n\n

Menggunakan metode yang sama persis dengan profil Server 2008 RDP, saya tidak punya masalah.

Sederhana.

\n\n

Buat profil seperti yang Anda miliki sebelumnya.

\n\n

Alih-alih memuat profil untuk masuk ke server, buat pintasan yang memiliki 

mstsc /admin c:\\path\\to\\the\\profile.rdp\n

di dalamnya.

\n\n

Pintasan itu seharusnya bekerja untuk Anda.

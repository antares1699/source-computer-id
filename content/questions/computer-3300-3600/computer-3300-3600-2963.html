Saya beralih kembali ke vimperator dari pentadactyl (terlalu lambat) dan saya membutuhkan fitur ini. Ini disediakan oleh perintah :ignorekeys, seperti: 

:ignorekeys add http://example.com/\n

Ini akan mengabaikan semua kunci, termasuk : dan / jadi Anda mungkin menginginkan setidaknya sesuatu seperti: 

:ignorekeys add -except=:,/ http://example.com/\n

Lihat :help :ignorekeys untuk lebih lanjut!

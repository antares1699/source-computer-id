Untuk menjawab pertanyaan terakhir Anda:

\n\n

Telur Python dijelaskan dalam: " Telur Python - Pengantar Sederhana ".

\n\n

Itu juga menyebutkan Zope.

\n\n
\n\n

Instal Mudah yang digunakan untuk memasang telur adalah bagian dari setuptools.

\n\n

Petunjuk penginstalan Windows untuk alat pengaturan (tautan ke penginstal Windows ada di akhir halaman itu).

\n\n

Kutipan:

\n\n
\n

Setelah instalasi selesai, Anda akan menemukan program easy_install.exe di subdirektori Python Scripts Anda. Pastikan untuk menambahkan direktori ini ke variabel lingkungan PATH Anda, jika Anda belum melakukannya.

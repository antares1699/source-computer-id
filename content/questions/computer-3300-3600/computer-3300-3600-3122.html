Inkscape sebenarnya memiliki opsi resample (itu mungkin menjadi bagian dari ImageMagick ). Opsi ini tersedia dari menu Extensions -> Raster.

\n\n

Untuk mencontoh ulang item di gambar Anda:

\n\n
    \n
  1. Sematkan gambar melalui Extensions -> Images -> Embed Images...
    2. \n
  2. Pilih gambar yang disematkan
    3. \n
  3. Resample melalui Extensions -> Raster -> Resample
    4. \n
\n\n

\n\n
    \n
  1. Turunkan nilai Width dan Height ke nilai terendah yang dapat diterima
    2. \n
  2. Apply dan Close
    3. \n
\n\n

Metode ini akan bekerja dengan baik selama proses pembuatan. Saya masih mengerjakan cara untuk mengotomatiskan ini dan akan memperbarui posting ini jika saya menemukan metode yang baik.

Jika ImageMagick tidak berfungsi seperti yang Anda butuhkan, coba GraphicsMagick (beranda sini ). Ya, ini adalah cabang dari ImageMagick tetapi saya pernah membenturkan kepala saya terhadap ImageMagick karena fitur yang tidak lengkap dan / atau tidak diimplementasikan dengan benar, hanya untuk menemukan bahwa fitur yang sama bekerja seperti pesona (dan dengan cara yang saya anggap logis dan konsisten) di GraphicsMagick.

\n\n

Ya, ini adalah bukti anekdotal dalam arah untuk mencoba GraphicsMagick, tetapi menurut saya patut dicoba. (Secara umum, saya menemukan ImageMagick sebagai perangkat lunak yang solid. Ini hanya beberapa kasus di mana GraphicsMagick lebih andal dan karenanya saya tetap menggunakannya.)

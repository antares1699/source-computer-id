Instal pustaka uuid melalui manajer paket node, yaitu npm 

npm install uuid      \n

Kemudian impor uuidv4 dari pustaka uuid 

   const { v4: uuidv4 } = require('uuid');\n

Fungsi di dalam: 

  setTodo(prevTodo => {\n\n    return [...prevTodo,{id:uuidv4(), name: name, complete:false}]\n  }) \n

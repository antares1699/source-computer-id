Itu adalah AppArmor yang mencegah saya mengakses bin. Aku berubah: 

/etc/apparmor.d/local/usr.sbin.clamd\n

dengan menambahkan: 

/bin/** r,\n

Untuk itu dan mampu memindai file di / bin. Jika saya ingin menambahkan direktori lain, saya hanya perlu menambahkan baris ke file itu untuk memasukkannya. Baris AppArmor di atas berarti / bin dan semua folder anak, secara rekursif, dengan akses hanya-baca, yang merupakan semua yang dibutuhkan ClamAV. File yang dimaksud adalah petunjuk AppArmor untuk izin khusus apa yang akan diterima / usr / sbin / clamd yang merupakan daemon layanan ClamAV. Jadi hanya ini yang perlu saya lakukan untuk membuatnya bekerja. Terima kasih @Ramhound untuk petunjuknya!

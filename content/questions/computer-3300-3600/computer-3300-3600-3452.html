Ini adalah fitur yang disebut Edit Cepat konsol, bukan PowerShell. Di versi cmd sebelumnya itu dimatikan secara default . Itu diklaim Windows 10 menyalakannya secara default tetapi saya tidak dapat memeriksa karena saya selalu mengaktifkannya sejak Windows 7

\n\n

Untuk menonaktifkan fitur itu cukup buka menu konsol, baik dengan

\n\n
    \n
  • mengklik kanan pada bilah judul konsol
    • \n
  • mendesak Alt + Ruang
    • \n
  • atau klik kiri pada logo bilah judul
    • \n
\n\n

lalu pilih Properti atau Default tergantung pada apakah Anda ingin menyetel opsi untuk sesi itu atau untuk semua eksekusi lebih lanjut. Dalam Pilihan tab, Edit Pilihan grup hapus centang Mode Edit Cepat

\n\n

Sekarang Anda tidak dapat lagi menempelkan dengan klik kanan tetapi Anda juga kehilangan kemampuan untuk memilih teks dengan mouse atau Bergeser + Tombol panah dan harus beralih ke mode tanda dengan membuka menu konteks> Edit> Tandai untuk memilih

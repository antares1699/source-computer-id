Saya membiarkan Tmux memberi tahu saya ketika konten jendela yang tidak saat ini berubah. 

setw -g monitor-activity on\nset -g visual-activity on  \n

Satu-satunya masalah adalah, warnanya jelek. Saya ingin mengubah warna fg / bg dari abu-abu menjadi warna yang menyatu dengan lebih baik. Bagaimana saya bisa melakukannya (apa saja nama opsinya)?

Fungsi pemantauan aktivitas dan keheningan berbagi serangkaian opsi konfigurasi tampilan:

\n\n
    \n
  • window-status-activity-attr
    • \n
  • window-status-activity-fg
    • \n
  • window-status-activity-bg
    • \n
\n\n

Nilai default dari "attr" adalah reverse, jadi Anda mungkin ingin menyetelnya ke sesuatu yang lain jika Anda juga mengubah warna (sehingga â€¦-fg akan menyetel warna latar depan yang efektif alih-alih dibalik menjadi warna latar belakang yang efektif); lihat daftar nama atribut dalam deskripsi halaman manual message-attr.

\n\n

Warna default ke default, yang hanya meninggalkan apa pun yang terakhir mereka atur saat membangun baris status; lihat daftar warna dalam deskripsi halaman manual message-bg. 

set-option -gw window-status-activity-attr bold\nset-option -gw window-status-activity-bg black\nset-option -gw window-status-activity-fg red\n

"Peringatan" lainnya (pemantauan konten dan bel) memiliki opsi tampilan sendiri (ganti activity dengan content atau bell pada nama opsi).

Sebenarnya, di versi tmux yang lebih baru (saya melihat ini di 1.9a), sepertinya ada perubahan dalam cara mengatur gaya.

\n\n

Sementara jawaban @ ChrisJohnsen masih berlaku (dan tmuxconfig saya pasti masih menggunakan pernyataan ini) cara baru untuk melakukan ini seperti yang dijelaskan oleh kelola adalah dengan menggunakan *-style daripada triplet *-attr *-bg *-fg, jadi Anda dapat menentukan gaya untuk jendela seperti itu opsi dengan satu pernyataan, bukan tiga. 

message-command-style style\n    Set status line message command style, where style is a\n    comma-separated list of characteristics to be specified.\n\n    These may be `bg=colour' to set the background colour,\n    `fg=colour' to set the foreground colour, and a list of\n    attributes as specified below.\n\n    The colour is one of: black, red, green, yellow, blue,\n    magenta, cyan, white, aixterm bright variants (if sup-\n    ported: brightred, brightgreen, and so on), colour0 to\n    colour255 from the 256-colour set, default, or a hexadec-\n    imal RGB string such as `#ffffff', which chooses the\n    closest match from the default 256-colour set.\n\n    The attributes is either none or a comma-delimited list\n    of one or more of: bright (or bold), dim, underscore,\n    blink, reverse, hidden, or italics, to turn an attribute\n    on, or an attribute prefixed with `no' to turn one off.\n\n    Examples are:\n\n          fg=yellow,bold,underscore,blink\n          bg=black,fg=default,noreverse\n\n    With the -a flag to the set-option command the new style\n    is added otherwise the existing style is replaced.\n

Jadi deklarasi ini kemudian menjadi: 

set-option -gw window-status-activity-style fg=red,bg=black,bold\n

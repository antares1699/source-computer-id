Ini adalah kesalahan besar yang saya buat, saya hanya berpikir untuk menghapus python dan menginstalnya kembali, tetapi akhirnya seluruh sistem saya tidak lagi berfungsi, sepertinya semuanya sudah dihapus.

\n\n

Saya mencoba sudo apt-get install ubuntu-desktop di layar masuk (Alt + ctrl + F1), tetapi saya tidak dapat terhubung ke jaringan sama sekali, oleh karena itu semuanya gagal mengambil. Setelah restart, itu hanya macet saat memuat halaman.

\n\n

Apakah ada hal lain yang bisa saya lakukan? Tolong bantu. Terima kasih.

\n\n

P.S. Bertanya-tanya apakah perbaikan boot berfungsi, saya mengunduhnya.

Pertama, karena saya tidak dapat terhubung ke Internet, saya menjalankan perintah berikut: 

sudo dhclient -r;sudo dhclient \n

Dan sekarang sudah terhubung. Setelah itu, ikuti saja langkah-langkah berikut ini: 

sudo apt-get install -f\nsudo apt-get install ubuntu-minimal\nsudo apt-get install ubuntu-desktop\nsudo apt-get install ubuntu-standard meta-package\n

Dan Ubuntu saya sudah pulih.

Apt-get berbasis python, dan menghapusnya mungkin merusak beberapa hal. Saya sangat menyarankan untuk mencadangkan / home, / etc dan apa pun yang Anda hargai dan instal ulang. Kemudian Anda dapat menyalin kembali / home dan / etc

\n\n

Anda bisa, secara bergantian mem-boot ke livecd, dan mengatur chroot, lalu menjalankan apt dari livecd, tetapi ini sepertinya akan terlalu merepotkan. Askubuntu memiliki instruksi tentang menyiapkan chroot , dan setelah ini selesai, coba jalankan apt-get install python (dan desktop ubuntu dan semua yang Anda butuhkan. Kemudian reboot, dan silangkan jari Anda bahwa itu berfungsi.

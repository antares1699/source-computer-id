Saya memiliki Mac dengan Safari dan Firefox sebagai browser utama saya. Saya suka menggunakan keduanya. Apakah ada cara untuk membagikan bookmark Firefox saya dengan Safari saya? Saya kira salah satu caranya adalah mengubah pengaturan konfigurasi browser saya agar keduanya mengarah ke file bookmark yang sama. Namun, saya tidak yakin apakah penunjuk untuk Safari akan kompatibel dengan salah satu Firefox?

\n\n

Cara lain mungkin dengan mengekspor atau mengimpor bookmark dari satu ke yang lain. Tidak yakin apakah ini dapat dilakukan dengan skrip?

Gunakan Xmarks. Ini menyinkronkan antara Safari, Firefox, Internet Explorer dan di seluruh platform (Mac, PC). Lewat situ semua bookmark Anda sinkron jika Anda ingin melakukannya.
Anda dapat menentukan profil (yaitu untuk menjaga pekerjaan dan privasi terpisah) dan jika xmarks tidak tersedia (pikirkan kafe internet) Anda dapat menggunakan bookmark Anda langsung dari halaman web Xmarks setelah masuk.

\n\n

Saya payah dalam hal layanan baru yang berjalan di latar belakang (semakin sedikit, semakin baik) dan saya senang dengan Xmarks dalam hal ini juga. Hanya dapat merekomendasikannya.

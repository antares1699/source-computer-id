Jadi saya memiliki motherboard ASUS P7P55D-E Pro. Ini memiliki pengontrol LAN Realtek 8112L on-board w / AI NET2. Saya menduga bahwa pengontrol LAN saya mati tetapi saya tidak begitu tahu pasti. Inilah yang saya ketahui sejauh ini:

\n\n
    \n
  • Semuanya berfungsi, saya memiliki sistem tiga boot dan ethernet berfungsi di Linux, Win7, dan OSX.
    • \n\n
  • Ethernet saya tidak lagi berfungsi di bawah ketiga sistem operasi.
    • \n\n
  • Saya mengalami pemadaman internet sesaat secara acak sebelum semuanya akhirnya menjadi gelap.
    • \n\n
  • Saya tidak tahu banyak tentang alat AI NET BIOS tetapi saya yakin alat ini hanya memeriksa masalah ethernet sebelum Anda boot ke sistem operasi. Bagaimanapun, itu tidak menemukan koneksi saat boot.
    • \n\n
  • Saya telah memeriksa koneksi pada beberapa mesin lain dan semuanya bekerja dengan baik.
    • \n\n
\n\n

Saya rasa saya sudah tahu jawaban atas pertanyaan saya, tetapi apakah ada penjelasan lain yang mungkin, atau apakah sudah mati?

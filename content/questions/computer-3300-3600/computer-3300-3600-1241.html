Kemungkinan Duplikat:
\n Aplikasi mana yang terbaik untuk menyinkronkan dua folder?

\n
\n\n

Saya kesulitan menyinkronkan folder saya. Saat ini saya melakukan RAID-1 manual.

\n\n

Ini menyebalkan karena kebetulan saya memperbarui folder di drive C: yang perlu "dicerminkan" ke drive F :.

\n\n

Ketika saya melakukan mirror, windows ingin menimpa SEMUA atau menimpa NONE (yang berarti waktu salin lebih lama), daripada membiarkan saya hanya "menimpa jika file sumber lebih baru".

\n\n

Bagaimana saya bisa melakukan itu?

Biasanya, untuk penggunaan Anda, Anda akan membuat pasangan folder dengan C: \\ Sesuatu di kiri, F: \\ Sesuatu di kanan, dan tindakan "Echo". Itu akan membuat folder kanan menjadi cadangan dari yang kiri, hanya menyalin file yang dimodifikasi.

\n\n

Jika saya mengerti dengan baik, ini yang Anda butuhkan?

\n\n

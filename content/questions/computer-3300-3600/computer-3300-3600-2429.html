Benar!

\n\n

Hampir semua program saat ini menyimpan pengaturannya di dalam registri di suatu tempat. Jadi jika itu ada di registri dan Anda ingin mengotomatiskannya, Anda beruntung.

\n\n

Langkah pertama adalah menemukan kunci registri yang berisi konfigurasi khusus yang akan Anda otomatisasi. Setelah Anda mengidentifikasi kunci registri, ekspor kunci tersebut ke jenis file REG. Kemudian tulis sendiri skrip yang akan memanggil file .REG dari baris perintah.

\n\n

Contoh konten file REG di bawah ini berkat Ivo 

Regedit4\n\n[HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Internet Settings]\n\"MigrateProxy\"=dword:00000001\n\"ProxyEnable\"=dword:00000001\n\"ProxyHttp1.1\"=dword:00000000\n\"ProxyServer\"=\"http://ProxyServername:80\"\n\"ProxyOverride\"=\"<local>\"\n

Contoh untuk benar-benar meletakkan konten file REG di dalam registri adalah ... 

C:> REGSRV32 myregsettings.REG\n

Jika itu meminta Anda untuk menjawab seperti pertanyaan Y / T gunakan contoh berikut sebagai gantinya 

C:> REGSRV32 myregistrysettings.REG < yes.txt\n

di mana yes.txt adalah Anda harus membuat file teks yang berisi karakter Y dan carriage return untuk menyelesaikan respons.

Dorong Pengaturan Browser di Skrip Login (untuk Internet Explorer)

\n\n

Internet Explorer menyimpan pengaturan proxy di registri. Ini membuatnya sangat mudah untuk diperbarui, menggunakan berbagai metode. Meskipun pengguna Anda biasanya tidak menggunakan Internet Explorer seperti browser biasa, Anda akan ingin mengkonfigurasi pengaturan proxy karena banyak aplikasi lain menonaktifkan pengaturan proxy di Internet Explorer.

\n\n

Dalam metode ini, Anda menentukan kunci registri yang tepat untuk versi IE Anda, mengekspor pengaturan ke file .REG, dan kemudian menggunakan REGEDIT di skrip login untuk mendorong pengaturan ke PC.

\n\n

Untuk membuat file SETPXY.REG, buka notepad & ketik sebagai berikut: 

REGEDIT4 (or whatever version of REGEDIT is on your PC, such as REGEDT32)\n[HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Internet Settings]\n\"ProxyServer\"=\"<your proxy IP address>:8080\"\n\"ProxyEnable\"=dword:00000001\n\"ProxyOverride\"=\"<local>\"\n

Ubah ke alamat IP pribadi Proxy HTTP Anda, dan 127.0.0.1 atau kombinasi URL apa pun yang ingin Anda lewati proxy tersebut. (Saya menyarankan Anda untuk bereksperimen dengan ini sebentar agar sintaksnya dilakukan dengan benar).

\n\n

Buat file EnableProxy.bat dengan konten berikut: 

@reg add \"HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Internet Settings\" /f /v ProxyEnable /t REG_DWORD /d 1\n

dan DisableProxy.bat 

@reg add \"HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Internet Settings\" /f /v ProxyEnable /t REG_DWORD /d 0\n

Ini hanya akan mengaktifkan / menonaktifkan proxy, jika Anda juga ingin mengubah alamat server, tambahkan baris lain ke EnableProxy.bat 

@reg add \"HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Internet Settings\" /f /v ProxyServer /t REG_SZ /d your.proxy.server\n

Menggunakan REG memungkinkan Anda untuk mengubah pengaturan individu secara diam-diam (/ f switch) dan tidak memerlukan UAC prompt (mungkin karena contoh ini hanya memodifikasi HKCU).

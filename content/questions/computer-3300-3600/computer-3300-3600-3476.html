Ketika saya membuka program apa pun di laptop saya, ikon bilah tugas sangat kecil

\n\n

Gunakan prosedur berikut untuk mengubah ukuran ikon bilah tugas:

\n\n
\n
    \n

  1. Klik kanan pada ruang kosong di desktop.

    2. \n

  2. Pilih Pengaturan tampilan dari menu kontekstual.

    \n \n

    3. \n

  3. Pindahkan penggeser di bawah "Ubah ukuran teks, aplikasi, dan item lainnya" ke 100%, 125%, 150%, atau 175%.

    \n \n

    4. \n

  4. Tekan Apply di bagian bawah jendela pengaturan. Windows mungkin meminta Anda untuk keluar dan masuk kembali untuk mendapatkan pengalaman yang lebih konsisten.

    5. \n
\n \n

Ini tidak hanya akan membuat ikon di bilah tugas sedikit lebih besar, tetapi juga\n juga akan membuat teks lebih besar di aplikasi seperti Microsoft Edge dan Kalender,\n serta di jendela notifikasi Windows 10.

\n
\n\n

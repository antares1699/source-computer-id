Saya memiliki Asus T102HA yang memiliki adaptor Qualcomm Atheros QCA9377 WiFi + Bluetooth terintegrasi. Bluetooth tiba-tiba berhenti bekerja pagi ini dan akibatnya mouse nirkabel tidak lagi berfungsi (begitulah cara saya mengetahui Bluetooth berhenti saat saya menggunakan mesin pada saat itu).

\n\n

Saya memeriksa di Pengelola Perangkat dan perangkat Bluetooth berwarna abu-abu meskipun tidak dinonaktifkan (harus mengaktifkan 'Tampilkan perangkat tersembunyi' di Pengelola Perangkat untuk melihat bagian Bluetooth di pohon perangkat).

\n\n

Saya melanjutkan untuk menghapus dan menginstal ulang driver Atheros QCA9377 yang tersedia dari situs Asus ( https://www.asus.com/uk/support/Download/3/919/0/1/EKnJ4vWdmuN3xLti/45/ ). Setelah menginstal ulang driver, Bluetooth tidak lagi terdaftar di mana pun di Device Manager.

\n\n

Saya telah menjalankan "Pindai perubahan perangkat keras" di Pengelola Perangkat tetapi ini tidak mendeteksi sesuatu yang baru.

\n\n

Seolah-olah perangkat Bluetooth tidak ada lagi di sistem. Namun, WiFi berfungsi dengan baik yang menunjukkan bahwa perangkat itu sendiri mungkin baik-baik saja dan masih ada.

\n\n

Satu-satunya hal yang mungkin mencurigakan untuk diperhatikan di Device Manager adalah perangkat di bawah: 

Universal Serial Bus controllers > Unknown USB Device (Device Descriptor Request Failed).\n

Saya tidak memiliki perangkat USB yang terhubung ke sistem sehingga tidak jelas apa perangkat USB yang tidak dikenal ini di atas - mungkin itu bisa jadi adaptor Bluetooth tetapi saya sepertinya tidak bisa mendapatkan perangkat yang tidak dikenal itu untuk menemukan pembaruan driver secara online.

\n\n

'Layanan Dukungan Bluetooth' Windows sedang berjalan dan dapat memulai / berhenti dengan sukses - ini juga diatur ke Startup otomatis.

\n\n

Mode pesawat tidak diaktifkan dan mengaktifkan / menonaktifkannya tidak berpengaruh pada situasi Bluetooth, meskipun itu mengaktifkan / menonaktifkan WiFi sebagaimana mestinya.

\n\n

Saya kehabisan opsi yang saya tahu untuk memecahkan masalah ini karena Bluetooth tidak muncul apa pun yang saya lakukan.

\n\n

Bagaimana saya bisa memecahkan masalah lebih lanjut dan memasang kembali adaptor Bluetooth dengan benar?

Setelah membaca ini , inilah kira-kira yang saya lakukan:

\n\n
    \n

  • Menghapus instalasi perangkat USB Tidak Dikenal dari Windows Device Manager (itu menyatakan " Permintaan deskriptor perangkat gagal ");

    • \n

  • Pergi ke situs web driver resmi Asus, unduh driver bluetooth terbaru untuk motherboard saya dan instal;

    • \n

  • Matikan komputer, dan "cabut" daya selama lebih dari 30 detik;

    • \n
\n\n

Setelah menyalakannya kembali, windows memiliki Bluetooth yang berfungsi kembali.

\n\n

Ingatlah bahwa saya telah menginstal ulang driver Bluetooth banyak kali, selalu restart komputer di antaranya, dan tidak ada yang berubah. Hanya setelah mencabut daya, apakah saya benar-benar berhasil menyelesaikan masalah ini.

\n\n

Ini memberi tahu saya bahwa masalah yang mendasarinya adalah perangkat keras terkait, dan mencabut motherboard dari daya mengatur ulang perangkat keras Bluetooth ke kondisi yang baik.

Jika Bluetooth menghilang dari pengelola perangkat dan menginstal ulang driver tidak berfungsi, gunakan dongle USB Bluetooth, Anda bisa mendapatkannya dengan harga 3,99-4,99 di Amazon. Kemudian instal ulang driver Qualcomm Bluetooth atau perbarui driver Qualcomm Bluetooth jika sekarang muncul di pengelola perangkat. Kemudian lepaskan dongle USB, dan boot ulang PC. Qualcomm Bluetooth Anda sekarang akan berfungsi. Saya tahu itu menyebalkan untuk membayar 3,99 untuk memperbaikinya, tetapi percayalah, saya mengalami masalah ini selama 2 bulan sampai saya menemukan solusi ini.

Akan lebih baik untuk mengisolasi input pengguna dari penggunaan input dalam nama UNC. Jadi saya bisa membantu dengan masukan pengguna:

\n\n

Cukup gunakan perintah set untuk mendapatkan respons pengguna: 

set /P response=INPUT VALUE PLEASE\ndir /A/D /B \"\\\\%response%\\users\\\"\n

Anda dapat menulis perintah yang lebih canggih seperti 

set /P response=\"INPUT VALUE HERE > \"\n

tetapi tanda centang ganda tidak diperlukan untuk perintah set. Mereka diminta untuk mengabaikan karakter khusus >. Selanjutnya visibilitas karakter spasi pada prompt akan ditingkatkan.

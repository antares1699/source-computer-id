Saya mengalami masalah dengan komputer saya; Saya baru-baru ini menginstal Deepin Linux, berdasarkan Debian, dan saya ingin menggunakan HDD internal berformat ntfs saya di atasnya.

\n\n

Saya membuat entri fstab untuk memasangnya secara otomatis saat startup, berikut adalah barisnya: 

UUID=BE9823F89823AE39 /media/Data ntfs defaults,gid=1000,uid=1000 0 0\n

Masalahnya adalah bahwa dalam opsi default, disk seharusnya dipasang sebagai rw, tetapi masih dalam mode hanya baca.

\n\n

Apakah ada yang punya ide untuk membantu saya?

\n\n

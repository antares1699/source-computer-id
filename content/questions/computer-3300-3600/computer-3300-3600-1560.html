Saya telah mengedit keyword.url ke mesin pencari favorit saya dan menonaktifkan pelengkapan otomatis domain, tetapi Firefox masih mencoba menyambung ke domain yang tidak valid saat saya mengetik satu kata di bilah alamat (seperti http: // foo / untuk foo) daripada menggunakan mesin pencari saya secara langsung. Bagaimana cara mengatasinya?

","source":"https://superuser.com/questions/292604/searching-a-single-term-from-firefox-address-bar","author":"user33758","replies":[],"answers":[{"answer":"

Perilaku ini normal - foo mungkin nama host yang valid di jaringan lokal atau di bawah domain yang sama (misalnya, setelah menambahkan sufiks utama - home. atau example.com. - atau dengan menggunakan NetBIOS untuk mengatasinya). Bahkan di Internet, domain berlabel tunggal masih valid - http://tk./ dan (sekarang mati) http://to./ sebagai contoh. Oleh karena itu Firefox (dan mungkin semua program lain) akan selalu menganggap alamat satu kata sebagai nama domain.

\n\n
\n\n

Sebagai solusinya, Anda dapat menggunakan penelusuran kata kunci: klik menu tarik-turun mesin telusur, pilih "Edit mesin telusur", lalu edit kata kunci Anda.

\n\n

\n\n

Cara lain adalah menambahkan bookmark yang mengarah ke mesin pencari Anda. Misalnya, saya memiliki kata kunci g dengan URL http://www.google.com/search?q=%s yang saya gunakan setiap hari.

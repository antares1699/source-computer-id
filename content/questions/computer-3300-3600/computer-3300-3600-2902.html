Saya menggunakan Windows 8 Pro RTM.\nSekarang memiliki dukungan untuk memasang file ISO sebagai drive sistem, tetapi ketika komputer di-restart, ISO tidak dipasang lagi.

\n\n

Bagaimana saya bisa membuat pemasangan ini permanen (baik melalui opsi Windows atau skrip batch)?

Pada dasarnya Anda ingin file ISO dipasang setiap kali Anda Logon ke PC Anda. Anda memiliki dua metode untuk melakukan ini:

\n\n
    \n

  1. Buka run dan ketik shell:startup, ini akan membuka Folder Startup. Tempatkan pintasan ke ISO di sini, dan Ini akan berfungsi.

    2. \n

  2. ( Opsi yang Lebih Baik ) Jalankan ini di Admin CMD

    3. \n
\n\n
\n

reg menambahkan\n HKEY_CURRENT_USER \\ Software \\ Microsoft \\ Windows \\ CurrentVersion \\ Run / v\n "Pasang ISO" / d "{lokasi ke ISO}"

\n
\n\n

Ini seharusnya berhasil.

Sejauh yang saya tahu, Windows secara otomatis mengingat hal-hal per folder. Jika Anda mengubah pengaturan tampilan folder satu kali, kecuali Anda membuat perubahan pada konten folder, itu akan tetap sama. Saya dapat mengatur Windows Explorer untuk menggunakan Tampilan Daftar pada folder tertentu, atau Ubin, atau tampilan lainnya, dan ketika saya keluar dan kembali ke Windows Explorer, dan bahkan jika saya mem-boot ulang, itu akan tetap sama.

\n\n

Jadi, Anda benar-benar hanya perlu menahan rasa sakit saat memuat semua thumbnail ubin sekali, lalu Anda dapat mengubahnya ke tampilan lain dan kemungkinan besar tidak perlu mengkhawatirkannya lagi.

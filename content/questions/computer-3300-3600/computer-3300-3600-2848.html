Saya menggunakan Sublime Text 2 di Mac dan PC. Alangkah baiknya jika saya hanya perlu melakukan satu set pintasan ke memori otot. Apakah ada cara untuk mengekspor binding Windows saya dan menggunakannya dengan OS X?

","source":"https://superuser.com/questions/517589/use-sublime-text-2-windows-keybindings-on-a-mac","author":"Thalecress","replies":[],"answers":[{"answer":"

Di Mac Anda:

\n\n
    \n
  1. Keluar dari Sublime Text.
    2. \n
  2. Di Finder (atau Terminal, pilihan Anda), navigasikan ke ~ / Library / Application Support / Sublime Text 2 / Packages / Default /.
    3. \n
  3. Ubah nama "Default (OSX) .sublime-keymap" menjadi "Default (OSX old) .sublime-keymap", atau nama file sejenis lainnya. (Jangan hapus jika Anda ingin mengubahnya kembali suatu hari nanti.)
    4. \n
  4. Buat duplikat dari "Default (Windows) .sublime-keymap".
    5. \n
  5. Ubah nama duplikat menjadi "Default (OSX) .sublime-keymap". Ingatlah bahwa ini peka huruf besar / kecil.
    6. \n
  6. Luncurkan Ulang Sublime Text. Sekarang, jika Anda membuka "Sublime Text 2"> "Preferences"> item menu "Key Bindings - Default", pintasan tombol default yang terdaftar harus sama seperti untuk Windows.
    7. \n

Saya telah menginstal cygwin di windows 7. Sebelumnya, tidak ada spasi antar baris. Saya dapat memuat ~ 90 baris di layar saya. Untuk beberapa alasan, sekarang ada ruang kecil (terlihat seperti 2 pt) di antara setiap baris, dan sekarang saya hanya dapat memuat ~ 75 baris di layar.

\n\n

Saya pertama kali memperhatikan ini ketika saya bermain-main dengan simbol autobot ascii. Dulunya halus, tetapi sekarang terlihat seperti ini:

\n\n

\n\n

Cukup menjengkelkan karena saya kehilangan 15 baris per layar. Itu membuat pelacakan file log lebih sulit. Apakah mungkin untuk mengedit ini? Saya belum dapat menemukan opsi spasi baris di mana pun.

","source":"https://superuser.com/questions/1056554/mintty-on-cygwin-how-to-change-line-spacing","author":"ewok","replies":[{"reply":"Saya baru saja memperbarui Cygwin dan memperhatikan ini sendiri. Karena saya menggunakan misc-fix 7x14, RowSpacing = -1 memberi saya tampilannya sebelumnya (meskipun saya mungkin mencoba -2 untuk mendapatkan lebih banyak baris di terminal saya).","author":"samiam"}],"answers":[{"answer":"

Menambahkan 

RowSpacing=-2\n

untuk ~/.minttyrc memecahkannya untuk saya. Anda mungkin ingin mencoba nilai (negatif) yang berbeda tergantung pada font yang Anda gunakan. Saya lebih suka Lucida Console 9, mengatur RowSpacing ke -2 mengembalikan tampilan seperti sebelumnya Versi mintty 2.2.4 .

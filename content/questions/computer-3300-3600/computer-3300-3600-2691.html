Saya memiliki profil terminator default yang dikonfigurasi untuk membagi jendela menjadi empat kuadran saat diluncurkan, ini berfungsi dengan baik.

\n\n

Ada juga opsi "Perintah Kustom" untuk setiap terminal, yang saya anggap berarti sesuatu untuk dijalankan di terminal. Namun, jika saya meletakkan sesuatu di sana, perintah dijalankan dan terminal ditutup (saya yakin itu dijalankan karena saya sudah mencoba echo hi > /tmp/hi, dan file itu memang dibuat dengan "hi" di dalamnya).

\n\n

Bagaimana saya bisa membuat setiap terminal menjalankan perintah yang saya tentukan tetapi tetap terbuka setelahnya?

\n\n

EDIT: Kubuntu 14.04.1, dengan distribusi terminator standar di repo terpercaya.

Dalam setiap bidang "perintah khusus", masukkan 

INITIAL_TERMINATOR_WINDOW=1 bash\n

Kemudian, dari dalam .bashrc Anda, Anda dapat memeriksa untuk melihat apakah variabel telah disetel, dan merespons sesuai: 

if [ -n \"$INITIAL_TERMINATOR_WINDOW\" ]; then\n  # Do something from an initial window\nelse\n  # Do something different when opening subsequent windows\nfi\n

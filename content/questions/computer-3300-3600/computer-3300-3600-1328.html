Toolbar "Links" di OS adalah folder ajaib yang menunjuk ke toolbar favorit IE.

\n\n

Jika Anda menjelajah dengan windows explorer ke "c: \\ users {username} \\ favourites" Anda akan melihat folder bernama "Bar Favorit". Jika Anda masuk ke folder itu dan kemudian mengklik ruang kosong di area lokasi, jalur yang Anda lihat sebenarnya adalah "C: \\ Users {username} \\ Favorites \\ Links" yang merupakan jalur yang sama dengan yang kami lihat di munculan keamanan Anda.

\n\n

Saya dapat membuat tautan ke calc.exe di bilah alat bilah tugas (dengan menjatuhkan pintasan ke folder di windows explorer) dan kemudian membukanya tanpa diminta, tetapi ketika saya meluncurkan IE dan mengaktifkan bilah favorit di dalam program itu, mengklik pada tautan di IE menghasilkan prompt keamanan dan kemudian tautan bilah alat bilah tugas mulai meminta saya bahkan setelah saya menutup IE, dan berlanjut sekarang setelah reboot.

\n\n

Saya percaya bahwa bilah Favorit sebenarnya adalah "zona internet" dan pertanyaannya harus dibingkai sebagai "mengapa saya tidak kadang-kadang diminta ".

\n\n

Untuk mengatasinya, Anda dapat membuat folder bernama mis. "quicklaunch" atau apa pun dan kemudian tambahkan sebagai bilah alat bilah tugas ("bilah alat baru", lalu arahkan ke folder baru), lalu sembunyikan nama bilah alat. Anda kemudian dapat mencadangkan / memulihkan folder itu ketika Anda menginstal ulang atau pindah ke komputer baru (asalkan tautannya valid)

Berdasarkan pesan "Open File - Security Warning", sepertinya ada Zone.Identifier NTFS Alternate Data Streams (ADS) di sana. Biasanya IE akan menambahkan ADS ini saat mengunduh file dari Internet, tidak yakin mengapa itu terjadi di pintasan Anda juga.

\n\n

Untuk menghapusnya, dapatkan utilitas Windows Sysinternals Aliran , masuk ke folder yang jalankan di bawah perintah untuk menghapus semua ADS untuk semua file. Tambahkan -s jika Anda ingin menyertakan subfolder. 

streams -d *\n

Sini adalah penjelasan lebih detail tentang NTFS ADS.

Windows 7 dan seterusnya memiliki fitur paranoia yang mengubah izin Desktop saat mengalami masalah saat bergabung dengan domain. Entah bagaimana itu pasti telah dipicu di komputer Anda (kegagalan daya, router mati, apa pun). ICACLS \"C:\\Users\\<username>\\Desktop\" /Setintegritylevel (OI)(CI)M\n

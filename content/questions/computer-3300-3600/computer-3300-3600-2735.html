Saya menggunakan Easeus Partion Manager di Windows 7, dan secara tidak sengaja memilih partisi yang telah diinstal Ubuntu untuk diformat / dihapus. Ketika komputer dihidupkan ulang, saya untungnya, meluncurkan Ubuntu dari Grub.

\n\n

Saya tahu jika saya memulai ulang dan memilih opsi Windows 7 dari Menu Grub, Easeus akan memulai operasi yang saya minta (pemformatan / penghapusan partisi Ubuntu). Apakah ada cara untuk membatalkannya dari mencopot pemasangan Ubuntu?

\n\n

(Drive DVD saya tidak dapat dioperasikan, dan saya tidak akan dapat menggunakan flash drive untuk menginstalnya kembali.) Jadi saya berharap saya dapat membatalkan Easeus dari memulai operasi.

\n\n

Apakah ini mungkin?

","source":"https://superuser.com/questions/695735/i-accidentally-told-easeus-partition-manager-to-format-ubuntu-can-i-cancel-this","author":"Undisclosed","replies":[],"answers":[{"answer":"

Ini adalah masalah Windows, bukan masalah Ubuntu.

\n\n

Untuk mencegah EaseUS Partition Master melakukan tugasnya saat Anda boot ke Windows, boot pertama ke Safe Mode di Windows. Anda dapat melakukan ini dengan menekan F8 saat Windows melakukan booting.

\n\n

Saat Anda berada dalam Mode Aman, buka C:\\Windows dan hapus file bernama EPMBatch.ept. File tersebut tersembunyi, jadi Anda harus membuat file tersembunyi terlihat dengan membuka Appearance and Personalization > Folder Options > View > Advanced Settings > Show hidden files, folders, and drives. Pilih opsi itu dan hapus file.

\n\n

Setelah itu, reboot komputer Anda dan EaseUS seharusnya tidak melakukan operasi yang tidak sengaja Anda lakukan (menghapus Partisi Ubuntu).

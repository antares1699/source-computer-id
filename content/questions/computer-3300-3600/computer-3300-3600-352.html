Kabel yang saya gunakan untuk menghubungkan headphone ke laptop saya putus. Saya merasa tidak masuk akal untuk membayar $ 55 untuk yang baru, jadi saya memutuskan bahwa mencoba mencari cara untuk memasangkannya dengan laptop saya akan lebih mudah dan lebih hemat biaya. Selama 6 hari terakhir, saya telah googling cara melakukannya, namun belum berhasil. Inilah yang saya kumpulkan sejauh ini:

\n\n

Laptop saya tidak mengalami masalah saat memasangkan dengan perangkat lain. Saya bingung dengan apa ini atau cara kerjanya, tetapi beberapa forum mengatakan bahwa saya memerlukan profil A2DP untuk melakukannya.

\n\n

Saya dapat menyambungkan headphone ke ponsel saya, menggunakan aplikasi yang direkomendasikan oleh mereka (NFC Easy Connect.) Namun, tablet saya tidak dapat menemukan perangkat tanpa aplikasi tersebut.

\n\n

Saat mencoba mengunduh adaptor bluetooth Lenovo, dikatakan bahwa tidak ada perangkat bluetooth yang terdeteksi. ( https://gyazo.com/8fae27fd4791c2ef0254ed3d8964389d ) Di bawah pengelola perangkat, saya memiliki perangkat bluetooth yang terdaftar di bawah kolom "Bluetooth" dan kolom "Adaptor Jaringan". ( https://gyazo.com/1a6f68efeec1fce13f7496975e87cacc ) Saya rasa ini bukan masalahnya, karena laptop saya dapat menemukan perangkat bluetooth lain dengan mudah.

\n\n

Saya benar-benar bingung apa yang harus saya lakukan. Headphone dimaksudkan untuk terhubung ke laptop dengan adaptor bluetooth, namun sepertinya tidak mungkin. Situs web Sony mengatakan bahwa jika Anda mencari, itu harus dapat dengan mudah mendeteksi dan memasangkan dengan perangkat. Ada saran tentang apa yang harus dilakukan? Saya akan memperbarui informasi di komputer atau headphone saya jika perlu. Terima kasih atas balasannya.

\n\n

(Saya saat ini menjalankan Lenovo Edge 2-in-1.)

Dari Manual headphone sony mdr-xb950bt Anda:

\n\n
    \n
  1. letakkan unit 1 meter dari perangkat bluetooth
    2. \n
  2. masuk ke mode berpasangan dengan menekan tombol POWER selama 7 detik
    3. \n
  3. ketika perangkat terdeteksi, pilih MDR-XB950BT, jika kunci sandi diminta masukkan "0000"
    4. \n
  4. buat koneksi bluetooth dari perangkat bluetooth
    5. \n
\n\n

Dengan ini, Anda mungkin harus mengatur pengaturan izinkan untuk menemukan komputer ini ke "ON" agar berada di sisi yang aman.

\n\n

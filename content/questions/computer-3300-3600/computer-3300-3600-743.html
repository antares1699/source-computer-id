Saya membangun PC baru dan menginstal Windows 7 Ultimate Trial dan semua aplikasi yang diperlukan (Firefox, MSE, Skype dll ..)

\n\n

Saya kemudian membeli serial dan mencoba mengaktifkannya. Saya mendapat pesan kesalahan yang mengatakan serial ini untuk versi Windows 7 yang berbeda dan mendapat klarifikasi bahwa nomor seri untuk Windows 7 Ultimate N.

\n\n

Adakah yang tahu cara mengubah edisi Windows 7 Ultimate saya yang terinstal ke Windows 7 Ultimate N, perbedaan antara kedua edisi sangat kecil (menurut Wikipedia ) bahwa saya tidak suka harus memulai dari awal lagi.

Kemungkinan Duplikat:
\n Notebook akan bangun setelah melepas mouse

\n
\n\n\n\n

Di laptop baru saya, hal berikut terjadi:

\n\n
    \n
  1. Saya menekan tombol Daya dan komputer masuk ke mode tidur. BAIK.
    2. \n
  2. Saya mencabut kabel, dan ketika saya menarik mouse USB, komputer bangun lagi.
    3. \n
\n\n

Tahu bagaimana saya bisa mencegahnya?

\n\n

Ini adalah Toshiba Qosmio G50, menjalankan Windows 7 Ultimate.

\n\n

Untuk mengetahui penyebabnya, saya ingin mendengar apakah ada yang mengalami masalah yang sama dengan Win7 di laptop atau Qosmio yang menjalankan Vista atau XP?

","source":"https://superuser.com/questions/80993/laptop-wakes-up-from-sleep-mode-when-i-remove-the-mouse","author":"Henk","replies":[{"reply":"Saya memiliki laptop Windows 7 Professional yang melakukan hal yang sama (merek ASUS, meskipun).","author":"Cameron"}],"answers":[{"answer":"

Masuk ke Manajemen Perangkat (Klik Start / Orb, Klik Kanan pada Computer dan klik Manage. Klik Device Manager).

\n\n

Berikutnya Luaskan "Tikus dan perangkat penunjuk lainnya".

\n\n

Klik tab Manajemen Daya dan hapus centang "Izinkan perangkat ini untuk membangunkan komputer"

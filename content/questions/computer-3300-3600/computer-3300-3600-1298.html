Saya tidak dapat menemukan tombol hapus pada keyboard android (bukan backspace). Bagaimana cara melanjutkan tindakan hapus? Haruskah saya menyiapkan sesuatu?

","source":"https://superuser.com/questions/1065296/delete-not-backspace-on-android-keyboard","author":"vico","replies":[{"reply":"Tidak ada satu pun yang asli di android.","author":"Randy"}],"answers":[{"answer":"

Keyboard AOSP, Google, dan OEM tidak menerapkan menghapus kunci sehingga Anda membutuhkan perangkat lunak lain. Keyboard Peretas adalah papan ketik virtual pihak ketiga yang mencoba menawarkan pengalaman papan ketik pc lengkap sehingga memiliki a menghapus kunci.

\n\n

P.S: keyboard pihak ketiga lainnya mungkin memiliki opsi yang sama, saya hanya memberikan satu contoh dan saya tidak berafiliasi dengan perangkat lunak itu dengan cara apa pun.

Ada sebuah menghapus masukkan semua keyboard secara default, tapi menurut saya itu bukan aturan untuk menghapus .

\n\n

Banyak keyboard khusus seperti Swiftkey , Flesky , dll. memiliki opsi untuk langsung menghapus satu kata atau seluruhnya .... tetapi tidak ada menghapus demikian juga.

\n\n

Anda dapat mencoba menggunakan ini "Keyboard peretas" , tetapi masih tidak terlihat di layar, dan Anda perlu beralih ke tampilan tambahan untuk mendapatkan tombolnya. Bagi saya itu hampir, dalam hal tindakan yang diperlukan, seperti mengetuk teks, di mana Anda perlu menghapus, dan kemudian menekan tombol menghapus tombol.

\n\n

Ini tidak dapat dilakukan di keyboard android default apa pun, kecuali pabrikan meletakkannya di sana sejak awal.

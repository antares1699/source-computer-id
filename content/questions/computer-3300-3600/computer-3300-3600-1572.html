Mungkin ffmpeg Anda tidak diinstal dengan dukungan LAME. Saya hanya mengatakan Anda mengunduhnya dari sumber dan mengkompilasinya dengan --enable-libmp3lame, atau tepatnya: 

$ ./configure --enable-gpl --enable-liba52 --enable-libgsm --enable-libxvid \\\n--enable-libamr_nb --enable-libamr_wb --enable-libmp3lame --enable-libogg \\\n--enable-libvorbis --enable-libfaac --enable-libfaad --enable-shared\n

Ini kemudian harus menggunakan instalasi lame Anda. Jika tidak bisa: Dapatkan LAME dari sini .

Tidak perlu menginstal dari sumber ... Paket ffmpeg yang berasal dari deb-multimedia.org (5:0.7.13-dmo2 ATM) memiliki dukungan libmp3lame.

\n\n

Saya melakukan hal berikut di Squeeze (sebagai root): 

apt-get install deb-multimedia-keyring\n

Tambahkan ke /etc/apt/sources.list: 

deb http://www.deb-multimedia.org squeeze main non-free\n

kemudian 

apt-get update\n

maka jika Anda memeriksanya dengan sudo apt-cache policy ffmpeg, itu akan menunjukkan bahwa ffmpeg akan datang dari deb-multimedia.org, bukan dari repo default. Begitu: 

apt-get install ffmpeg\n

dan itu harus mencakup dukungan LAME. (Saya juga menginstal lame ... Saya tidak yakin apakah itu diperlukan.)

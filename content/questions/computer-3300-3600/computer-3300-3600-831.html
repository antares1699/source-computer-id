Saya ingin melakukan beberapa tugas (memulai program) di komputer yang menjalankan Windows setiap kali (ulang) dimulai. Ini komputer jarak jauh dan hanya menempatkan pintasan di mulai otomatis tidak berfungsi karena dijalankan setiap kali saya masuk menggunakan desktop jarak jauh.

\n\n

Bagaimana saya bisa melakukannya?

Anda dapat menggunakan Penjadwal Tugas Windows. Sudah lama sejak saya menggunakannya di XP, tetapi di Vista dan Windows 7 Anda dapat menjadwalkan tugas agar terjadi saat startup. Ini bisa berupa file batch, atau dapat dieksekusi.

Ini Halaman wiki Emacs menyarankan menggunakan cuplikan berikut untuk menghitung ukuran file dan direktori yang dipilih: 

 (defun dired-get-size ()\n  (interactive)\n  (let ((files (dired-get-marked-files)))\n    (with-temp-buffer\n      (apply 'call-process \"/usr/bin/du\" nil t nil \"-sch\" files)\n      (message \"Size of all marked files: %s\"\n               (progn \n                 (re-search-backward \"\\\\(^[0-9.,]+[A-Za-z]+\\\\).*total$\")\n                  (match-string 1))))))\n\n (define-key dired-mode-map (kbd \"?\") 'dired-get-size)\n

Jika saya memahami Anda dengan benar, Anda ingin memperbarui nilai buffer Dired secara langsung. Saya tidak cukup akrab dengan internal Dired, tapi saya rasa seharusnya tidak terlalu sulit.

Jika versi Emacs Anda> = 24.4, maka Anda bisa\ncoba dired-du Perpustakaan. Ini tersedia dari repositori ELPA.

\n\n

Setelah Anda menginstal lib ini:\nJika buffer Anda saat ini dalam mode Dired, maka Anda dapat melakukan: 

C-x M-r\n

yang mematikan `dired-du-mode 'dan menampilkan rekursif\nukuran direktori 'di tempat' di buffer Dired. Jika kamu\nkunjungi buffer Dired baru, maka buffer rekursif akan ditampilkan\njuga, hingga Anda mematikan mode.

\n\n

Tip lain: 

C-x C-h\n

mengubah format ukuran. Ada 3 format:

\n\n
    \n

  1. Default satu dari perintah `ls '.

    2. \n

  2. Format yang dapat dibaca manusia.

    3. \n

  3. Format numerik dengan pemisah ribuan koma.

    4. \n
\n\n

Anda dapat menyesuaikan opsi `dired-du-size-format 'menjadi\nbuatlah ukuran Anda menjadi pilihan yang persisten.

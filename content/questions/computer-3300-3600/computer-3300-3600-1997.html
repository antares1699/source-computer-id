Posting lama, tetapi untuk siapa pun yang masih menginginkan jawaban:

\n\n

Pada dasarnya, dengan XMonad.StackSet.index, Anda bisa mendapatkan daftar jendela di ruang kerja saat ini seperti ini: 

do\n    windows <- gets (W.index . windowset)\n    -- something...\n

Jadi findWindow dapat diimplementasikan seperti ini: 

import qualified XMonad.StackSet as W\nimport Data.List (find)\n\nfindWindow :: (Window -> Bool) -> (Window -> X()) -> X() -> X()\nfindWindow condition actionIfFound actionIfNotFound = do\n    windows <- gets (W.index . windowset)\n    let found = find condition windows  -- found has type Maybe Window\n    case found of\n        Nothing -> actionIfNotFound\n        Just w -> actionIfFound w\n

PS: Tiga baris terakhir bisa disingkat menjadi maybe actionIfNotFound actionIfFound found

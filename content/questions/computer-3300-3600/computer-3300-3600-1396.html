Potongan dari referensi di bawah ini (Sunting: INI TAMPAKNYA HANYA BEKERJA UNTUK Vista ):\nCara Mengaktifkan atau Menonaktifkan Pratinjau Thumbnail Taskbar di Windows 7

\n\n
    \n
  1. Klik tombol Start dan ketik GPEdit.msc di kotak teks Pencarian Instan, lalu tekan Enter.
    2. \n
  2. Arahkan ke Konfigurasi Pengguna -> Template Administratif -> Start Menu dan Taskbar di panel kiri Editor Kebijakan Grup Lokal.
    3. \n
  3. Cari dan klik dua kali pada Matikan thumbnail taskbar di panel kanan editor Kebijakan Grup Lokal.
    4. \n
  4. Pilih tombol radion Diaktifkan.
    5. \n
  5. Klik OK untuk membuat perubahan efektif.
    6. \n
  6. Logoff dan logon lagi atau restart komputer jika diperlukan.
    7. \n
\n\n

Referensi sini

\n\n

Edit: Untuk Windows 7:

\n\n
    \n
  1. Registri cadangan
    2. \n
  2. Menang + R -> regedit
    3. \n
  3. Arahkan ke: HKEY_CURRENT_USER \\ Software \\ Microsoft \\ Windows \\ CurrentVersion \\ Explorer \\ Taskband
    4. \n
  4. Klik kanan folder Taskband -> New -> DWORD 32-bit value
    5. \n
  5. Beri nama DWORD MaxThumbSizePx baru
    6. \n
  6. Klik kanan DWORD baru -> Modify -> masukkan "1"
    7. \n
  7. Mulai ulang
    8. \n
\n\n

Referensi Win7

Solusi terbaik yang saya temukan adalah masuk ke kunci registri ini: 

HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\Advanced\n

Kemudian buat nilai DWORD baru bernama ExtendedUIHoverTime dengan nilai 20000. Itu seharusnya menyebabkan penundaan 20 detik sebelum pratinjau muncul, tetapi bagi saya itu menghentikannya terjadi sepenuhnya.

\n\n

Sumber: jawaban.microsoft.com

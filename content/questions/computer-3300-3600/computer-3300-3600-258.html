Apakah perintah Defrag %SystemDrive% /b benar-benar berfungsi untuk mengoptimalkan waktu boot Windows? (Opsi defrag /b tidak didokumentasikan oleh Microsoft).

\n\n

Saya mencoba menggunakannya sebagai upaya untuk mengoptimalkan file pada hard disk yang digunakan Windows untuk mem-boot sistem. Saya menduga bahwa saya perlu menjadwalkan defrag sebelum boot Windows, namun saya tidak berhasil dengan defrag. Apakah ada perangkat lunak lain yang berfungsi untuk itu?

Defrag adalah program Windows dan berjalan saat Windows sedang berjalan. Anda dapat menjadwalkan disk pemindaian sebelum Windows dimuat sepenuhnya, tetapi tidak untuk defrag. Ada alat yang dapat Anda gunakan defragment di luar Windows, tetapi defrag bawaan Windows lebih dari cukup untuk sebagian besar kebutuhan.

\n\n

Saya menjadwalkan defrag untuk dijalankan selama beberapa jam beberapa malam setiap minggu. Dengan cara itu, ia menjaga disk pada tingkat fragmentasi yang sangat rendah.

\n\n

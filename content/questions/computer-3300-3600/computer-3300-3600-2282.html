Saya baru saja meningkatkan dari xp 32bit ke win7 ultimate 64bit. saya memformat drive OS dan menginstalnya, tetapi saya meninggalkan drive saya yang lain. Sekarang ketika saya masuk ke windows 7, setiap kali saya mencoba menambah, memindahkan, atau menghapus file di salah satu drive / partisi sekunder ini, saya diminta dengan dialog yang mengatakan sesuatu seperti ini:

\n\n
\n

Anda harus memberikan izin administrator untuk memindahkan file ini.

\n
\n\n

Ini sedikit mengganggu, tetapi masalah sebenarnya muncul ketika program lain mencoba memanipulasi atau menghapus file pada partisi ini. Ketika saya mencoba melakukannya dengan program lain, saya tidak mengizinkannya karena persyaratan administrator; misalnya saya tidak dapat menyimpan file teks sederhana melalui notepad ke salah satu drive ini.

\n\n

Pertanyaan saya adalah bagaimana cara mengatur izin pada drive saya yang lain sehingga saya tidak perlu memberikan akses setiap kali untuk operasi file, dan agar program lain juga dapat mengakses drive ini secara normal? Saya telah mengambil kepemilikan untuk drive dan mengatur pemiliknya menjadi akun saya dan grup administrator, dan saya telah mengatur izin untuk meniru izin drive C sebanyak yang saya bisa (kecuali saya belum menemukan cara mengaturnya izin khusus yang dimiliki grup "Pengguna yang Diotentikasi" pada drive itu). Ada tips?

\n\n

Edit: Saya juga mencoba menjalankan program ini sebagai administrator yang tidak mengubah hasilnya.

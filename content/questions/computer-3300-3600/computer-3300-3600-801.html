\n

Menurut pemahaman saya, semua cadangan Time Machine setelah yang pertama adalah perbedaan dari cadangan sebelumnya, bukan gambar disk. Jadi menghapus folder ini akan membuat backup berikutnya tidak berguna (menurut saya).

\n
\n\n

Untungnya bukan itu masalahnya. Cadangan Time Machine adalah cadangan lengkap, tetapi berbagi ruang disk dengan cadangan sebelumnya untuk file yang tidak berubah (menggunakan tautan keras).\nSelama salah satu tautan keras tersedia, Anda dapat mengakses file tersebut.

\n\n

Jadi, jika Anda memiliki file besar di cadangan hari Senin, dan file tidak berubah pada hari Selasa, file yang sama juga akan ada di cadangan hari Selasa, tetapi tidak akan menggunakan ruang disk tambahan. Anda dapat dengan aman menghapus cadangan hari Senin, dan "salinan" untuk hari Selasa akan terus berfungsi.

\n\n
\n

Dialog "Mempersiapkan Mengosongkan Sampah ... Item yang akan dihapus: [angka]" mendapatkan hingga 1,5 juta file (!) Sebelum saya menghentikannya. Dan saya tidak memiliki izin, bahkan sebagai root, untuk meng-rm -rf mereka.

\n
\n\n

Ya, karena tautan keras ini, ada banyak sekali file dalam cadangan Time Machine (semuanya dihitung berkali-kali). Anda tidak dapat menggunakan alat berbasis file sederhana pada cadangan Time Machine, Anda memerlukan sesuatu yang memahami (dan mempertahankan) tautan keras. Misalnya, jika Anda mencoba menyalin semua file pada disk Time Machine, Anda akan mendapatkan duplikasi yang sangat besar. Satu-satunya cara untuk menyalin / mencadangkan Time Machine adalah mengkloning struktur disk (menggunakan sesuatu seperti Utilitas Disk).

Saya beruntung memecahkan masalah ini dengan masuk sebagai root: 

su\n

dan kemudian ketikkan ini di terminal: 

cd /Volumes/<name of your drive>/.Trashes\n

maka Anda bisa melakukan 

ls -la\n

yang akan menampilkan nama direktori sampah (dalam kasus saya 501) yang kemudian saya hapus menggunakan perintah ini 

rm -rf 501\n

yang, seperti perintah finder 'Empty Trash', membutuhkan waktu beberapa saat, tetapi setidaknya perintah itu tidak akan gagal dan hang selamanya. Plus, ini memiliki manfaat tambahan karena TIDAK harus memformat ulang drive Anda.

\n\n

Sekarang, sejauh situasi Anda saat ini, jika upaya Anda untuk mengosongkan sampah sudah menghapus beberapa file, kerusakan apa pun mungkin sudah dilakukan. Jika Anda menggunakan Leopard, Anda harus melakukannya secara manual, tetapi jika Anda menggunakan Snow Leopard Anda dapat pergi ke tempat sampah, pilih semua file, kontrol klik dan pilih "Taruh Kembali".

\n\n

Setelah Anda membalikkan kerusakan sebanyak mungkin, saya akan mengklik Time Machine di bilah menu dan mencoba memilih masukkan mesin waktu. Lihat apa yang ditunjukkannya kepada Anda. Lihat-lihat dan lihat apakah barang lama Anda masih ada.

\n\n

\n\n

Anda mungkin lebih suka tidak melakukan itu.

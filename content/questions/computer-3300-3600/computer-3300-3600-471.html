\n\n

\n\n

Mengapa, kemudian, jika saya mengetik python.exe di prompt Win + R, itu berhasil?

Perintah Jalankan mencari jalur registri untuk program yang akan diluncurkan serta% jalur%. Lihatlah HKLM\\Software\\Microsoft\\Windows\\CurrentVersion\\App Paths - Anda mungkin akan menemukan node bernama "Python.exe" dengan entri default yang mengarah ke Python yang dapat dieksekusi. Pemasang Sebaiknya tambahkan entri registri untuk semua executable utama di sini, tetapi perintah Run adalah fitur yang kurang dihargai, jadi banyak pengembang lupa untuk menambahkan langkah sederhana ini ke penginstal mereka.

Menyadari balasan ini sudah tidak berlaku lagi, saya memiliki masalah yang sama; Win 7 64 bit dengan tiba-tiba boot up yang sangat lambat sampai boot gagal sama sekali tetapi kadang-kadang. Diperlukan waktu 20 menit untuk mencapai boot "stabil" dan bahkan masukan apa pun tertinggal secara dramatis. Saya membaca beberapa ide seperti mengedit regedit untuk mengubah HKEY_LOCAL_MACHINE \\ SYSTEM ...

\n\n

Singkatnya, c-drive saya adalah WD 1002 ... 1TB. Saya memiliki sekitar 360GB dan telah mencoba defrag radikal tidak berhasil, meskipun tampaknya berhasil sekali. Setelah saya mendapatkan komputer untuk akhirnya reboot setelah lima kali gagal, saya mulai memindahkan banyak barang dari itu ke drive lain hanya untuk menyimpannya, menutup hanya dengan program senilai 100 GB dan file non-data.

\n\n

Saya kemudian melakukan defrag dan mengoptimalkan file sisa menggunakan Ausilogics, dan saya membuat cadangan gambar menggunakan Macrium Reflect. Upaya sebelumnya untuk menyimpan drive menggunakan Macrium Reflect, dirayapi pada 1-4 MB / s. Kali ini, kecepatannya> 650 Mb / s dan hanya membutuhkan waktu sekitar 30 menit untuk menyelesaikannya. Upaya saya sebelumnya berlangsung selama berjam-jam dan kemudian gagal.

\n\n

Saya tidak menggunakan opsi regedit. Baru saja menghapus banyak file dan kemudian mendefraged dan mengoptimalkan sisanya dan sekarang booting dan berjalan seperti baru. Sementara itu, saya telah membeli WD 1002 lagi, karena menurut saya itulah masalahnya, tapi saya rasa saya akan menyimpannya sebagai cadangan karena semuanya berfungsi dengan baik.

\n\n

Bagaimanapun, itulah ceritaku. Semoga bisa membantu seseorang. Kedengarannya sangat mirip masalah yang sama.

\n\n

PS - chkdsk tidak menemukan masalah. Namun drive mulai bertindak tidak stabil dan kemudian menjadi lebih buruk selama dua hari dengan pencadangan yang berjalan pada 1-3 Mb / s dan kemudian menghentikan dan berulang kali gagal untuk mem-boot windows sama sekali. Sekarang, dengan cara di atas, semuanya 100% kembali normal. Saya masih curiga mungkin ada masalah dengan drive, bagaimanapun, dan saya senang sekarang saya memiliki cadangan baru untuk berjaga-jaga - dan gambar terbaru yang berjalan dengan baik. Saya benci ketika saya harus menginstal ulang Windows dan memuat ulang setiap program sialan dari awal.

Karena tidak disertakan dengan CD instalasi apa pun, saya khawatir Anda tidak akan dapat melakukan instalasi baru. Dan sekarang, jika Anda harus menginstal ulang windows karena masalah serius (hard drive gagal, kesalahan besar, dll.), Anda akan kurang beruntung.

\n\n

Sunting: Sepertinya saya salah, karena CD instalasi tersedia untuk diunduh secara legal: https://superuser.com/a/305434/24256 . Anda harus memiliki kunci lisensi, dan pertanyaannya adalah apakah disk ini akan menerima lisensi OEM.

\n\n

Anda dapat menggunakan Windows Backup untuk membuat cadangan gambar lengkap Anda saat ini (termasuk perangkat lunak yang diinstal) jika terjadi kesalahan di masa mendatang. Idealnya simpan ini di beberapa hard drive lain, seperti hard drive eksternal. Cadangan Windows juga akan menyarankan Anda untuk membuat disk pemulihan - ini memungkinkan Anda untuk memulai sistem dan memulihkan cadangan tersebut.

\n\n

Untuk keamanan ekstra (karena Anda tidak memiliki CD instalasi windows), bukan ide yang buruk untuk menyimpan DVD instalasi Linux yang dapat di-boot (seperti Ubuntu) di suatu tempat. Jadi jika terjadi kesalahan (mis. Disk pemulihan tidak dapat memulihkan cadangan), Anda setidaknya dapat mem-boot Linux dan mengakses file dan internet Anda.

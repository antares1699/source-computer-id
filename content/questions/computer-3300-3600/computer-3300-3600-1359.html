Untuk mengubah ukuran default menjadi mm:

\n\n
    \n
  • Mengajukan Tab -> Pilihan -> Maju Tab -> Tampilan Bagian -> Unit Penggaris
    • \n
  • Pilih Millimeters
    • \n
\n\n

Untuk mengubah lebar / tinggi kolom / baris (dalam mm):

\n\n
    \n
  • Melihat tab -> Tata letak halaman
    • \n
  • Pilih semua sel
    • \n
  • Klik kanan pada Kolom -> Lebar kolom...
    • \n
  • Klik kanan pada Rows -> Tinggi Baris ...
    • \n

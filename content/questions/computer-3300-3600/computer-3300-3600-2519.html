Saya memiliki beberapa laptop / PC dengan Windows 10 yang ingin saya tingkatkan ke versi Windows 10 terbaru (Peningkatan Pembuat Konten). Untuk mempercepat dan menghindari mengunduh ulang hal yang sama berulang kali, saya telah menggunakan Alat Pembuatan Media untuk membangun stik USB dengan media instalasi. Sekarang saya menjalankan setup.exe dari tongkat, dan semuanya berjalan dengan baik.

\n\n

Kapan saya dapat melepaskan stik USB untuk memulai peningkatan pada PC berikutnya? Apakah saya perlu menunggu hingga seluruh peningkatan selesai, atau apakah Windows memiliki semua file yang diperlukan pada PC yang memutakhirkan pada satu titik, dan jika demikian, bagaimana saya dapat mengetahui kapan harus mencabut stik?

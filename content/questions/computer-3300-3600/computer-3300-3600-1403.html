stunnel aku s apa yang kamu cari: 

sudo stunnel -c -r google.com:443 -d 127.0.0.1:8888\n

Ini menyiapkan sesi SSL ke pihak jarak jauh (Google dalam hal ini), dan membuat pendengar di localhost port 8888. Anda dapat menggunakan 80 jika Anda belum memiliki pendengar.

\n\n

Kemudian Anda mengakses localhost: 8888 dan Anda akan mendapatkan situs remote.

\n\n

Jika Anda menggunakan Windows, opsi baris perintah tidak didukung, jadi buat file stunnel.conf dengan parameter di dalam: 

[remote]\naccept = 8888\nconnect = google.com:443\n

Kemudian sebut saja dengan 

stunnel -c stunnel.conf\n

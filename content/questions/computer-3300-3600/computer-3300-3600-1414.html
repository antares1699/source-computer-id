Berikut ini skrip (alias "file batch") yang akan melakukannya untuk Anda jika Anda tidak ingin mendownload program dari suatu tempat: 

\n@echo off\nsetlocal enabledelayedexpansion\nfor /r %%F in (*.m4a) do (\n    set mp3name=%%~dpnF.mp3\n    if exist !mp3name! (\n        fc/b %%F !mp3name! > nul\n        if errorlevel 1 (\n            echo Warning: %%F and !mp3name! both exist but are not identical.\n        ) else (\n            echo DEL %%F\n        )\n    )\n)\n

Penafsiran:

\n\n
    \n
  • for %%letter in (wildcard) do some_command memindai direktori saat ini untuk file yang cocok dengan wildcard dan menjalankan perintah dengan variabel yang diidentifikasi oleh set letter ke nama file.
    • \n
  • for /r %%letter in ... sama tetapi rekursif.
    • \n
  • (… ) memungkinkan Anda menentukan blok perintah, bukan hanya satu.
    • \n
  • Seperti yang ditunjukkan di atas, %%F adalah nama file M4A yang ditemukan.
    • \n
  • %%~dpnF adalah huruf kandar, jalur, dan bagian nama (basis) dari\n%%F –– yaitu, semua kecuali ekstensi. Untuk ini kami menambahkan ekstensi .mp3; yaitu, nama file yang sama dengan %%F kecuali memiliki ekstensi .mp3. Tetapkan nama ini ke variabel mp3name.
    • \n
  • Jika file ini ada, lakukan enam baris berikutnya. (Perhatikan bahwa kami kembali menggunakan (… ) untuk menentukan blok perintah.)
    • \n
  • fc/b %%F !mp3name! > nul –– fc adalah f ile c membandingkan.\nfc suka membandingkan file teks, jadi kami katakan fc/b untuk melakukan a b perbandingan singkat. Jadi fc/b %%F !mp3name! membandingkan kedua file tersebut.\n> nul artinya jangan menampilkan hasil perbandingan di layar;\nkami hanya ingin skrip mengetahui hasil perbandingan.
    • \n
  • if errorlevel 1 berarti melakukan perintah berikut hanya jika perintah sebelumnya "gagal". Dalam kasus fc, "kegagalan" berarti file tidak identik, jadi kami memiliki skrip yang memberi tahu Anda bahwa Hey_Jude.mp3 dan Hey_Jude.m4a keduanya ada tetapi tidak identik.
    • \n
  • Akhirnya, DEL %%F berarti hapus file tersebut.\n(echo DEL %%F berarti menampilkan perintah hapus, tetapi jangan jalankan.)
    • \n
\n\n

Mempersiapkan:

\n\n
    \n
  • Di Windows Explorer, pergi ke root (atas) direktori musik Anda, lakukan file Teks Baru, dan beri nama.
    • \n
  • Klik dua kali di atasnya untuk membuka (mengedit) di Notepad. Salin dan tempel skrip di atas.\nSimpan dan keluar.
    • \n
  • Kembali ke Windows Explorer, ganti nama file Text (.txt) untuk memiliki ekstensi .bat. Abaikan peringatan itu; klik "Ya".
    • \n
  • Mulai Prompt Perintah. cd ke direktori di atas. Ketik nama file skrip (batch) (Anda dapat mengetikkan ekstensi .bat, tetapi Anda mungkin tidak perlu melakukannya). Pastikan daftar perintah DEL yang dilaporkan terlihat masuk akal.\nJika ya,
    • \n
  • Kembali ke Windows Explorer, klik kanan pada file .bat, dan pilih Edit.
    • \n
  • Di Notepad, ubah echo DEL menjadi DEL. Simpan dan keluar.
    • \n
  • Kembali ke Command Prompt, jalankan kembali perintah tersebut.
    • \n

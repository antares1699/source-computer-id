Program enkripsi drive tidak menghapus data pada volume yang telah mereka nyalakan. Mereka akan mengenkripsi semua data, yang dapat memakan waktu tergantung pada jenis enkripsi, jumlah data yang akan dienkripsi, kecepatan drive dan sistem, dan banyak faktor lainnya. Tetapi kecuali jika terjadi kegagalan besar selama proses enkripsi, data Anda tidak akan dihapus selama proses ini.

\n\n

Yang mengatakan, jika proses enkripsi gagal secara serempak (saya sendiri belum mengamati kejadian seperti itu, kemungkinannya SANGAT jauh), Anda cenderung tidak dapat memulihkan data daripada jika drive yang tidak dienkripsi mengalami masalah serupa. Ini hanyalah sifat enkripsi. Tetapi kemungkinan ini sangat kecil sehingga umumnya tidak dianggap sebagai alasan untuk tidak mengenkripsi.

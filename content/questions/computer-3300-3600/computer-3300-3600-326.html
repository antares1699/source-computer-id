Secara jujur? Saya memiliki G15 sekarang dan saya tidak pernah menggunakan layar di TF2. Saya tidak dapat menemukan sesuatu yang berharga untuk itu selain modul jam built-in. Tombol samping berguna tetapi hanya menggunakan 1 bank, bukan 3 yang tersedia. Tidak ada yang tidak bisa ditangani oleh tombol yang tidak digunakan di keyboard.

\n\n

Jika Anda bermain WoW, atau semacam RTS / RPG yang mengharuskan Anda memiliki gambaran umum mendalam tentang game, dengan semua trik keyboard biasa maka ini adalah keyboard yang bagus (ayah tiri saya memainkan WoW dengannya dan menggunakan kuncinya. bank banyak). Untuk permainan kasual Anda mungkin tidak akan menggunakan banyak fitur (tetapi tombol media adalah berguna untuk dimiliki - meskipun tidak jarang).

\n\n

Jika Anda seorang night owl, 2 level lampu latar keyboard berguna - itu bisa sangat terang tetapi ada mode redup.

\n\n

Sejauh mengetik, itu adalah keyboard yang cukup kokoh. Tata letak standar - dan mungkin Anda dapat menemukan penggunaan tombol khusus G1-G6 di editor Anda jika Anda seorang programmer. Jika keyboard dilengkapi dengan PC, ambillah, jika Anda membayar ekstra, Anda mungkin membuang-buang uang untuk hal-hal yang tidak Anda butuhkan.

Mungkin ada solusi yang lebih efisien, tetapi Anda dapat menyelesaikan pekerjaan dengan sedikit bantuan dari AutoHotkey .

\n

Pertama, menyusun skrip berikut: 

ClipboardBackup := Clipboard\nClipboard =\n\nSend, !{Space} ;Menu\nSleep, 100\nSend, e ;Edit\nSend, s ;Select All\n\nSend, {Enter} ;Copy\nClipWait\n\nFileAppend, %Clipboard%, *\n\nClipboard := ClipboardBackup\n\nExitApp\n
\n

Tempatkan file .exe di file % PATH% direktori. Saya menamai milik saya savecmd.exe.

\n

Sekarang, Anda dapat secara efektif menyimpan konten jendela command prompt dengan menggunakan sintaks berikut: 

savecmd > file.txt\n
\n

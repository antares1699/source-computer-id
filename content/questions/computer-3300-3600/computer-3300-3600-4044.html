Ini sama di Windows dan mungkin semua sistem operasi. Kamera web bukanlah perangkat input sederhana: untuk membaca bingkai darinya Anda perlu secara aktif mengirim perintah sesuai dengan beberapa protokol. Ini membuatnya menjadi stateful dan oleh karena itu aplikasi yang mengelola statusnya harus memiliki akses eksklusif ke sana.

\n\n

Di windows Anda dapat menggunakan splitcam untuk menduplikasi feed. Di Linux, Anda dapat menggunakan WebcamStudio.

Hai Excel tidak dapat melakukan penghitungan RATA-RATA sederhana dengan nilai yang berasal dari pemotongan teks.

\n\n
    \n
  • C19 dihitung sebagai = KIRI (C18,1)
    • \n
  • C24 dihitung sebagai = KIRI (C23,1)
    • \n
  • C29 dihitung sebagai = KIRI (C28,1)
    • \n
  • Quality Avg (C30) dihitung sebagai
    • \n
\n\n

= IF (NOT (AND (ISBLANK (C19), ISBLANK (C24), ISBLANK (C29))), AVERAGE (C19, C24, C29), "")

\n\n

Pemeriksaan kesalahan menunjukkan bahwa setiap nilai dalam fungsi DAN adalah angka yang benar. Tapi Excel tidak dapat menghitung rata-rata dan mengembalikan # DIV / 0 !. Bagaimana pembagian dengan 3 mengembalikan kesalahan itu? Ini ada di Excel 2013.

\n\n

Juga, mengapa kolom kosong mengembalikan # DIV / 0 !? Pernyataan IF memeriksa kekosongan.

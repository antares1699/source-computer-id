Saya sarankan mengunduh distribusi PHP yang Anda inginkan sebagai sumber, dan kemudian mengatur opsi parameter ./configure (lihat ./configure --help) sehingga perangkat lunak diinstal baik di lokasi yang unik atau dengan nomor versi yang ditambahkan ke namanya. Dengan cara ini ketika Anda menjalankan make install penginstalan PHP 5.3 tidak ditimpa. Terakhir, siapkan pemetaan sufiks terpisah di Apache yang merutekan ke versi yang Anda inginkan, atau ganti pemetaan PHP default untuk stanza VirtualHost atau Directory tempat skrip Anda berada.

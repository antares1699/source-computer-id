Apakah ada alat untuk Mac OS X yang mendukung pemotongan file .AVI, tetapi tidak mengompres ulang? Alat baris perintah juga diterima.

","source":"https://superuser.com/questions/422698/cutting-avi-videos-in-mac-os-x","author":"ohho","replies":[],"answers":[{"answer":"

Install FFmpeg.dll melalui Homebrew : 

brew install ffmpeg\n

Kemudian potong video mulai dari posisi yang ditentukan dengan -ss untuk durasi yang ditentukan dengan -t. copy memastikan bahwa video dan audio tidak dikodekan ulang. 

ffmpeg -i video.avi -c:v copy -c:a copy -ss 00:01:30 -t 0:0:20 output.avi\n

Di Windows Vista dan Windows 7, pengelola tugas berjalan non-elevasi, dan Anda tidak menghadapi prompt UAC kecuali Anda memilih "Lihat Proses Semua Pengguna".

\n\n

Di Windows 8 Preview, di luar kotak Task Manager mulai dinaikkan setiap saat. Bagaimana cara mengkonfigurasinya untuk memulai non-elevasi jadi saya tidak terpukul dengan prompt UAC setiap kali saya memeriksa penggunaan CPU atau melihat daftar proses yang berjalan untuk melihat apakah aplikasi ditutup sepenuhnya?

\n\n

Kompatibilitas "shims" dapat digunakan untuk menimpa manifes; Anda dapat memulai Task Manager dengan variabel lingkungan __compat_layer diatur ke runasinvoker, mis. dari prompt perintah:
set __compat_layer=runasinvoker
\ntaskmgr

\n\n

Atau, gunakan Administrator Kompatibilitas untuk menerapkan perbaikan kompatibilitas "RunAsInvoker" ke taskmgr.exe sehingga secara otomatis meluncurkan non-elevasi.

\n\n

Referensi: http://csi-windows.com/toolkit/uac-prompt-guide

\n\n
\n\n

Gim kompatibilitas RunAsInvoker dapat diaktifkan di registri, tanpa memerlukan Toolkit Kompatibilitas Aplikasi atau bahkan penginstalan SDB. Berikut ini skrip registri untuk melakukannya (sesuaikan jalur yang sesuai untuk sistem Anda): 

Windows Registry Editor Version 5.00\n\n[HKEY_CURRENT_USER\\Software\\Microsoft\\Windows NT\\CurrentVersion\\AppCompatFlags\\Layers]\n\"C:\\\\Windows\\\\System32\\\\taskmgr.exe\"=\"RunAsInvoker\"\n

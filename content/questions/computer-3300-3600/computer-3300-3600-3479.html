Saya ingin mencadangkan log obrolan WeChat saya, jadi saya mencoba mencadangkan log obrolan ke komputer saya menggunakan aplikasi OS X WeChat. Itu berhasil dicadangkan, tetapi saya tidak dapat menemukan log obrolan di mana pun. Saya memeriksa isi paket di folder aplikasi dan tidak menemukan apa pun. Idealnya saya akan menemukan file teks atau sesuatu yang serupa dengan semua konten obrolan yang dapat saya cadangkan sendiri ke Google Drive, karena saya tidak ingin 100% mempercayai aplikasi ini untuk mencadangkan obrolan ke komputer saya (yang mana hanya bersifat lokal untuk memulai, saya lebih suka sesuatu yang berbasis cloud), dan saya ingin mengirim file obrolan ke teman saya yang kehilangan obrolan dengan saya.

\n\n

Apakah ada solusi untuk ini?

\n\n

Ponsel adalah Android, komputer menggunakan OS X High Sierra.

\n\n

Jika Anda tidak dapat menemukan folder Library, tekan "Go" pada menu bar dan sambil menekan "Alt", Library akan terlihat.\nSemoga membantu.

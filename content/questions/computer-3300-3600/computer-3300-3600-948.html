Anda juga dapat menyimpan Access-, Modification- and Creation-Date saat menggunakan format 7-zip ".7z". Untuk melakukannya, cukup tentukan parameter berikut saat mengemas data:

\n\n
    \n
  • GUI 7-zip: tulis " tc ta "menjadi" Parameter: "textfield
    • \n
  • Baris perintah 7-zip: tentukan " -mtc -mta -mtm "parameter
    • \n
\n\n

tc singkatan dari waktu penciptaan,
\n ta singkatan dari waktu akses dan
\n tm singkatan dari waktu modifikasi

\n\n

Waktu modifikasi disimpan secara default.

\n\n

Terlepas dari itu, atribut ini adalah bagian dari penjurnalan NTFS.

\n\n

Anda dapat menambahkan atribut ini ke Windows explorer dengan mudah untuk semua jenis folder / file:

\n\n

\n\n

Namun, mereka tidak selalu akurat atau aktif.

\n\n

\n\n

\n\n

Saya juga dapat menjalankan perintah PowerShell untuk mendapatkan informasi

\n\n

gci | LastAccessTime memberi saya:

\n\n

\n\n

Ini sama akuratnya dengan tampilan penjelajah.

\n\n

Sebaliknya, apa yang perlu dilakukan adalah memastikan Windows memilikinya jorunaling diaktifkan.

\n\n

fsutil behavior set disablelastaccess 0 \n

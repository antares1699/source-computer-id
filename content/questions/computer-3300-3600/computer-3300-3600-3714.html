Saya tahu cara mengikat kunci ke perintah di tmux, tetapi saya ingin membuat perintah khusus (yang dapat Anda ketik dengan kunci awalan + titik dua). Dalam perintah khusus ini, saya ingin menjalankan beberapa perintah lainnya.

\n\n

Ide saya adalah memiliki sesuatu seperti ini: 

no-side-status() {\n    set status-left-length 0\n    set status-right-length 0\n}\n\nside-status() {\n    set status-left-length 50\n    set status-right-length 150\n}\n

Jadi saya dapat mengetik :no-side-status untuk menyembunyikan bilah status kiri dan kanan, dan mengetik :side-status untuk memulihkan bilah status kiri dan kanan.

\n\n

Apakah mungkin untuk membuat perintah khusus seperti itu? Jika ya, bagaimana caranya? Jika tidak, ada cara lain untuk mencapai apa yang saya inginkan?

Awalnya, tmux tidak memiliki dukungan untuk perintah khusus kecuali untuk menjalankan skrip shell eksternal.

\n\n

Ada mod yang menambahkan dukungan skrip lengkap ke tmux: http://ershov.github.io/tmux/

\n\n

Ini juga memungkinkan untuk membuat perintah pengguna. Misalnya, milik Anda akan terlihat seperti: 

proc no-side-status {} {\n    set status-left-length 0\n    set status-right-length 0\n}\n\nproc side-status {} {\n    set status-left-length 50\n    set status-right-length 150\n}\n

Untuk menggunakan dari baris perintah tmux cukup ketik C-b : dan side-status atau no-side-status.

\n\n

Untuk mengikatnya ke sebuah kunci, gunakan bind C-p tcl side-status.

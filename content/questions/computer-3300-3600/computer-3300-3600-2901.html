Anda dapat meletakkan ini di file .htaccess Anda. 

RewriteEngine On\nRedirectMatch permanent ^~username/$ http://example.com/~username/php/index.php\n

Itu harus mengarahkan setiap permintaan root ke file indeks Anda (tidak diuji). Sesuaikan agar cocok dengan url dan domain public_html Anda.

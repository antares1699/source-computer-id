File definisi penyorotan sintaks terletak di: 

~/.kde/share/apps/katepart/syntax/\n/usr/share/kde4/apps/katepart/syntax/\n

Mengedit file di direktori pertama memerlukan izin root, dan file penyorotan sintaks akan tersedia untuk setiap pengguna kate di lingkungan Anda. Jika direktori tidak ada, buatlah untuk menempatkan file.
Temukan tag <language>, dan ubah atribut extensions, sebagai berikut: 

<language name=\"CSS\" ... extensions=\"*.css\" ... >\n<language name=\"CSS\" ... extensions=\"*.css;*.less\" ... >\n

Alih-alih memodifikasi css.xml, saya sarankan untuk mendapatkan file definisi untuk less css, sehingga sintaks yang lebih sedikit juga disorot dengan benar . Lihat mtorromeo / kate-sintaks-file / less.xml di GitHub.

\n\n

Salin-tempel perintah berikut, lalu mulai ulang Kate. 

mkdir -p ~/.kde/share/apps/katepart/syntax\nwget https://raw.github.com/mtorromeo/kate-syntax-files/master/less.xml\n

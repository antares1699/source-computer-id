Anda dapat melihat semua alamat IP yang dikonfigurasi menggunakan perintah ini:

\n\n

netsh interface ip show address | findstr \"IP Address\"

\n\n

Anda juga dapat menambahkan nama adaptor untuk mendapatkan alamat IP dari antarmuka jaringan tertentu.

\n\n

netsh interface ip show address \"Ethernet\" | findstr \"IP Address\"

\n\n

Ini harus bekerja setidaknya di versi terbaru Windows.

Di Microsoft Word 2011 di Macintosh, Anda bisa menyisipkan referensi silang dalam dokumen Anda ke hyperlink dari satu lokasi ke lokasi lain dalam dokumen. Setelah referensi silang dibuat, Anda dapat mengklik referensi silang tersebut untuk membuka bagian yang dirujuk pada dokumen.

\n\n

Setelah referensi silang diklik, nama perintah, penekanan tombol, tindakan mouse, atau item menu apa yang akan mengembalikan saya ke posisi sebelumnya dalam dokumen sebelum melintasi referensi silang?

\n\n

Adam menyebutkan perintah WebGoBack, dan ini tampaknya untuk melakukan hal yang benar. Di Word 2011, WebGoBack tidak terdaftar di "Semua Perintah" di Alat-> Sesuaikan keyboard ..., jadi Anda tidak dapat menetapkan penekanan tombol di sana, tetapi ada.

\n\n

Apakah itu hilang dari daftar "Semua Perintah" dengan sengaja (misalnya karena tidak dapat diandalkan atau tidak didukung) atau tidak sengaja, saya tidak dapat memberi tahu Anda. Tetapi Anda dapat membuat makro di Normal.dotm yang memanggilnya, dan menetapkan penekanan tombol untuk itu: 

Sub myWebGoBack\nWordBasic.WebGoBack\nEnd Sub\n

atau Anda dapat menggunakan VBA satu kali untuk menetapkan perintah ke penekanan tombol. misalnya, kode ini memberikan perintah ke Option + tombol Panah Kiri: 

Sub AssignOptionLeftArrowToWebGoBack()\n  CustomizationContext = NormalTemplate\n  KeyBindings.Add KeyCode:=BuildKeyCode(37, wdKeyOption), _\n    KeyCategory:=wdKeyCategoryCommand, _\n    Command:=\"WebGoBack\"\nEnd Sub\n

Jika Anda lebih suka menggunakan kunci yang berbeda, gunakan Kode Kunci yang berbeda - mis. untuk menetapkan Option + koma (di keyboard saya "<", yang dapat saya anggap sebagai panah kiri, berada di atas ",") Anda dapat menggunakan: 

Sub AssignOptionCommaToWebGoBack()\n  CustomizationContext = NormalTemplate\n  KeyBindings.Add KeyCode:=BuildKeyCode(wdKeyComma, wdKeyOption), _\n    KeyCategory:=wdKeyCategoryCommand, _\n    Command:=\"WebGoBack\"\nEnd Sub\n

Untuk menghapus semua tugas ke WebGoBack (sehingga Option-Panah Kiri kembali ke fungsi "kata kiri", Anda dapat menggunakan 

Sub ClearWebGoBackKeyBindings()\nDim kb As KeyBinding\nCustomizationContext = NormalTemplate\nFor Each kb In Application.KeyBindings\n  If kb.Command = \"WebGoBack\" Then\n    Debug.Print kb.KeyString\n    kb.Clear\n  End If\nNext\nEnd Sub\n

Hebatnya, perintah Go Back di Word untuk Mac bernama "GoBack". Pintasan keyboard default di Mac adalah SHIFT F5. (SHIFT FN F5 jika Anda tidak memiliki keyboard tambahan).

\n\n

Untuk mengubah pintasan keyboard (saya telah mengubahnya menjadi COMMAND LEFT-ARROW):

\n\n
    \n
  • Lihat> Bilah Alat> Sesuaikan Bilah Alat dan Menu ... "
    • \n
  • Tekan tombol "Keyboard" di pojok kanan bawah kotak dialog
    • \n
  • Dari daftar "Kategori" di sebelah kiri, pilih "Semua Perintah"
    • \n
  • Dari daftar "Perintah" di sebelah kanan, pilih "GoBack" (atau klik dalam daftar di sebelah kanan dan mulailah mengetik "GoBack"
    • \n
  • Masukkan pintasan keyboard baru Anda di bawah.
    • \n
  • Klik "TUGASKAN" Ini tidak tombol "OKE".
    • \n
  • LALU klik "OKE."
    • \n

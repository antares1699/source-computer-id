Prosedur instalasi:

\n \n

Langkah 1: Copot pemasangan versi sebelumnya dari\n Sambungan Jaringan Juniper. (Jika Anda pernah mencoba menghubungkan kantor Anda\n komputer maka Anda mungkin telah menginstal perangkat lunak ini pada mesin Anda.\n Periksa di Tambah atau hapus program)

\n \n

Langkah 2: Buka Komputer dan klik Atur -> Folder dan cari\n options -> tab View -> Di bawah Hidden files and folder, klik Show\n file, folder, dan drive tersembunyi dan klik OK.

\n \n

Langkah 2: Sekarang Anda harus menghapus beberapa file di komputer Anda. Pergi ke\n Komputer -> Disk Lokal (C :) -> Pengguna -> "Klik nama pengguna Anda" (Jika\n Anda tidak yakin dengan nama pengguna Anda, lalu klik mulai. Anda akan menemukan\n nama pengguna Anda di atas Dokumen)

\n \n

Langkah 3: Klik pada folder AppData (folder tersembunyi) -> Roaming -> Hapus\n folder Juniper Networks.

\n \n

Langkah 4: Instal perangkat lunak Juniper Networks yang Anda miliki\n diunduh dari tautan di atas. (Klik Ya untuk semua otentikasi\n pesan)

\n \n

Langkah 5: Sekarang Jalankan perangkat lunak. Pergi ke Start -> All Programs -> Juniper\n Jaringan -> Sambungan Jaringan 6.30. -> Sambungan Jaringan

\n \n

Langkah 6: Sekarang masukkan alamat VPN dan masuk.

Saya suka opsi "Tampilan Halaman" di Adobe Reader disetel ke "Single Page Continuous", jadi tidak tiba-tiba melompat ke halaman berikutnya.

\n\n

Tetapi untuk beberapa alasan yang tidak diketahui, Adobe Reader di tempat kerja saya (versi 9.1.1) tidak selalu mengingat pengaturan ini. Adakah yang tahu di mana Adobe Reader menyimpan pengaturan ini? Saya menduga itu di suatu tempat di registri.

","source":"https://superuser.com/questions/115341/where-does-adobe-reader-store-the-page-display-setting","author":"compie","replies":[],"answers":[{"answer":"

Tahukah Anda bahwa file PDF dapat menentukan tata letak awal dan pengaturan zoom yang berlaku saat Anda membukanya? (Misalnya, saat Anda memilih perintah Ekspor ke PDF di OpenOffice, kotak dialog memungkinkan Anda menentukan file tampilan awal dan opsi antarmuka pengguna lainnya yang disimpan dalam file PDF.)

\n\n

Masalah Anda mungkin disebabkan oleh file PDF tertentu yang mengubah pengaturan tampilan saat Anda membukanya. Jika demikian, Adobe Reader memiliki opsi aksesibilitas untuk mengabaikan pengaturan yang ditentukan dalam file PDF.

\n\n

Saya tidak memiliki Adobe Reader 9.1.1. Tapi di Adobe Reader 8.2.0, saya bisa pergi ke Edit, Preferensi, Aksesibilitas, dan ada kotak centang "Selalu gunakan Gaya Tata Letak Halaman" dengan kotak drop-down di mana Anda dapat memilih "Halaman Tunggal Berkelanjutan".

Anda dapat menggunakan skrip untuk mengubah pengaturan (atau melakukannya secara manual di registri).

\n\n

Contoh di bawah ini menyetel opsi tampilan ke "mulus" untuk Acrobat X: 

REM Set Acrobat X settings to Smooth text for LCD/Laptop\nREM Set to LCD\nreg add \"HKCU\\Software\\Adobe\\Acrobat Reader\\10.0\\Originals\" /V bAntialiasText /T REG_DWORD /F /D 1\nREM Set to \"For Laptop/LCD screens\"\nreg add \"HKCU\\Software\\Adobe\\Acrobat Reader\\10.0\\Originals\" /V benableDDR /T REG_DWORD /F /D 1\nREM Set threshold \nreg add \"HKCU\\Software\\Adobe\\Acrobat Reader\\10.0\\Originals\" /V iAntialiasThreshold /T REG_DWORD /F /D 18\n

Bavi_H benar - setiap dokumen memiliki pengaturannya sendiri yang tertanam di dalamnya. Ada sedikit trik yang dapat Anda lakukan untuk membantu. Di Pustaka 10, buka Edit -> Preferences -> Documents dan centang "Ingat pengaturan tampilan sebelumnya untuk setiap dokumen".

\n\n

Dengan begitu, saat Anda membuka sesuatu untuk pertama kalinya, Anda dapat mengubah pengaturan dan mereka akan diingat untuk dokumen itu. Mengubahnya untuk menimpa pengaturan khusus dokumen kemungkinan akan melibatkan beberapa pekerjaan registri seperti yang disebutkan tentang. Tidak yakin apakah Anda siap untuk itu, tapi semoga sukses untuk Anda.

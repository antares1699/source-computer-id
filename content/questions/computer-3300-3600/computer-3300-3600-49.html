Coba ini - ini berhasil untuk saya:

\n\n

Menghapus profil "Default" tampaknya telah menyelesaikan semua masalah (sesuai halaman bantuan Chrome):

\n\n
    \n
  1. Tutup Google Chrome sepenuhnya.
    2. \n
  2. Buka menu Buka> Buka Folder.
    3. \n
  3. Masukkan direktori berikut di bidang teks, lalu tekan Buka. ~ / Perpustakaan / Dukungan Aplikasi / Google / Chrome /
    4. \n
  4. Temukan folder bernama "Default" di jendela direktori yang terbuka dan ubah namanya menjadi "Backup default".
    5. \n
  5. Coba buka Google Chrome lagi. Folder "Default" baru secara otomatis dibuat saat Anda mulai menggunakan browser.
    6. \n

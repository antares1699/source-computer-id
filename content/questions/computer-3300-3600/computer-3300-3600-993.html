Jika Anda mencari versi "Metro", versi itu mungkin tidak tersedia karena Anda memiliki browser lain yang disetel sebagai browser default. Jika Internet Explorer bukan browser default Anda, maka aplikasi IE10 Metro tidak tersedia. Ini harus tetap tersedia di desktop.

\n\n

Anda dapat mencari "Program default" dan menjadikan IE sebagai browser default dan aplikasi Metro IE akan muncul secara ajaib.

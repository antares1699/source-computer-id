Saya menggunakan PCIE Nand Drive sebagai readyboost untuk meningkatkan kinerja Windows 10 saya (dan itu meningkatkan kinerja secara drastis). Masalahnya adalah Windows mengenalinya sebagai drive internal, bukan drive portabel, dan karena itu akan memunculkan pemberitahuan penyimpanan rendah.

\n\n

Saya telah mengikuti instruksi yang ditemukan di sini: http://www.neowin.net/forum/topic/1264602-windows-10-disable-the-low-disk-space-warning/

\n\n

Pada dasarnya yang saya lakukan adalah: 

LowDiskMinimumMBytes DWORD 0 --> HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows Search\\Gather\n\nBackOffLowDiskThresholdMB DWORD 0 --> HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows Search\\Gathering Manager\n\nNoLowDiskSpaceChecks DWORD 1 --> HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Policies\\Explorer\n

Dan dimulai kembali.

\n\n

Sayangnya itu tidak menghentikan notifikasi.

Ini adalah sesuatu yang orang akan perjuangkan. Microsoft mungkin harus memperbarui panduan untuk menonaktifkan pemberitahuan ini untuk Windows 10. 

LowDiskMinimumMBytes DWORD 0 --> HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows Search\\Gather\n\nBackOffLowDiskThresholdMB DWORD 0 --> HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows Search\\Gathering Manager\n

Kedua modifikasi ini tidak akan menghasilkan apa-apa. Ini untuk Pencarian Windows, bukan sistem. Jangan main-main dengan mereka. Anda tidak dapat mengubah kepemilikan buruk menjadi TrustedInstaller. Jika sudah, jangan panik, Anda tidak akan mengalami banyak masalah jika menggunakan pengguna yang memiliki hak istimewa administrator dan tidak perlu mengubah setelan ini dengan pengguna dari grup lain.

\n\n

Tidak, inilah triknya. Alih-alih di pengguna saat ini, Microsoft memindahkan pengaturan sepenuhnya ke mesin lokal. Jadi, Anda sebaiknya hanya: 

NoLowDiskSpaceChecks DWORD 1 --> HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows\\CurrentVersion\\Policies\\Explorer\n

Seseorang dari Microsoft harus memeriksa pengeditan halaman ini sehingga pengguna tidak bingung: https://support.microsoft.com/en-us/kb/555622

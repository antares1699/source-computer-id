Misalkan Anda memiliki 2 file pdf yang "dipindai".

\n\n
    \n
  1. Besar, tapi tanpa lapisan teks.
    2. \n
  2. Lebih kecil (dengan gambar berkualitas lebih rendah), tetapi dengan lapisan teks yang benar.
    3. \n
\n\n

Kedua file tersebut berisi sama gambar, hanya berbeda dengan kompresinya.

\n\n

Tujuannya adalah untuk menyematkan sama lapisan teks ke pdf pertama.

\n\n

\n\n

Jadi, saya melihat dua kemungkinan cara:

\n\n
    \n
  1. Ekspor-impor lapisan teks entah bagaimana
    2. \n
  2. Ganti gambar di lapisan gambar entah bagaimana.
    3. \n
\n\n

Berikut ini skrip shell sederhana untuk melakukan ini pada baris perintah:

\n

Simpan ini sebagai ~/pdf-merge-text.sh (dan chmod +x itu): 

#!/usr/bin/env bash\n\nset -eu\n\npdf_merge_text() {\n    local txtpdf; txtpdf="$1"\n    local imgpdf; imgpdf="$2"\n    local outpdf; outpdf="${3--}"\n    if [ "-" != "${txtpdf}" ] && [ ! -f "${txtpdf}" ]; then echo "error: text PDF does not exist: ${txtpdf}" 1>&2; return 1; fi\n    if [ "-" != "${imgpdf}" ] && [ ! -f "${imgpdf}" ]; then echo "error: image PDF does not exist: ${imgpdf}" 1>&2; return 1; fi\n    if [ "-" != "${outpdf}" ] && [ -e "${outpdf}" ]; then echo "error: not overwriting existing output file: ${outpdf}" 1>&2; return 1; fi\n    (\n        local txtonlypdf; txtonlypdf="$(TMPDIR=. mktemp --suffix=.pdf)"\n        trap "rm -f -- '${txtonlypdf//'/'\\\\''}'" EXIT\n        gs -o "${txtonlypdf}" -sDEVICE=pdfwrite -dFILTERIMAGE "${txtpdf}"\n        pdftk "${txtonlypdf}" multistamp "${imgpdf}" output "${outpdf}"\n    )\n}\n\npdf_merge_text "$@"\n

Sekarang sebut saja: 

~/pdf-merge-text.sh txt.pdf img.pdf out.pdf\n

Idenya adalah untuk gambar strip dari PDF OCR, lalu gabungkan melalui teknik dalam jawaban di atas .

Jawaban ini di stackoverflow punya solusinya. Anda dapat mengekstrak teks dengan koordinat dari pdf-2 Anda menggunakan pdftotext -bbox atau paket Python PDFMiner , lalu tulis teks tersembunyi ini ke dalam PDF baru dengan paket Python ReportLab , lalu gabungkan PDF teks tersembunyi ini dengan pdf-1 Anda menggunakan PDFtk (Ada GUI untuk Windows di halaman web; baris perintah untuk Unix disebut PDFtk Server sekarang.)

\n\n

Atau, Anda bisa mencoba langsung menggabungkan pdf-1 dan pdf-2 menggunakan PDFtk. Jalankan pdftk pdf-2 multistamp pdf-1 output out.pdf. Ini akan menempatkan setiap halaman pdf-1 di depan halaman terkait dari pdf-2, jadi Anda hanya akan melihat gambar dari pdf-1 (dengan asumsi itu adalah pindaian, dan tidak memiliki latar belakang transparan), tetapi teks tersembunyi dari pdf-2 akan disertakan. Kelemahannya adalah ini mungkin sangat besar, karena ini akan menyertakan dua salinan dari setiap gambar halaman. Saya telah memverifikasi bahwa ini berfungsi, dan ukuran pdf keluaran adalah jumlah dari ukuran masukan.

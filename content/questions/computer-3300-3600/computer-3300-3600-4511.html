Ya, Anda harus dapat menghapus direktori Windows XP lama yang semuanya harus berada di bawah Windows.old.

\n

Itu Panduan migrasi Microsoft memiliki bagian "Pembersihan pasca-pemasangan (opsional)" di "langkah 5":

\n
\n

Selama penginstalan Windows 7, jika Anda tidak memformat partisi, file yang digunakan di Windows XP disimpan di folder Windows.old. Jenis file di folder ini bergantung pada komputer Anda.

\n

Setelah Anda menggunakan Windows 7 untuk sementara waktu, misalnya satu atau dua minggu, dan Anda yakin bahwa file dan pengaturan Anda kembali ke tempat yang Anda inginkan, Anda dapat dengan aman mendapatkan kembali ruang disk dan menggunakan Pembersihan Disk untuk menghapus folder Windows.old.

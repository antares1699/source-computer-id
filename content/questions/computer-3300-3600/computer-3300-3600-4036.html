Saya pikir taruhan terbaik Anda adalah menginstal ulang seluruh sistem dengan mengingat hal-hal berikut:

\n\n
    \n
  1. cadangkan direktori /etc/ Anda - penginstalan ulang akan menimpa file konfigurasi yang tersisa.
    2. \n
  2. Kumpulkan daftar paket yang tidak sengaja Anda hapus seperti yang diinstruksikan di bawah ini sebelum melakukan instalasi ulang (karena file log akan ditimpa)
    3. \n
  3. Melakukan TIDAK format / inisialisasi / mkfs sistem file apa pun saat melakukan instal ulang
    4. \n
  4. Setelah melakukan a minimal instal ulang sistem, instal ulang paket yang dihapus yang Anda kumpulkan dari yum log.
    5. \n
  5. terakhir, ganti konfigurasi default dari paket yang Anda instal dari / etc / backup. Anda mungkin perlu melakukan beberapa langkah dengan tangan, seperti membuat ulang akun dan grup pengguna.
    6. \n
\n\n

Untuk melihat apa yang Anda unistall, lihat file /var/log/yum.log mis. sudo cat /var/log/yum.log | grep Erased untuk mendaftar semua paket yang telah dicopot (sejak pembuatan file log itu!)

\n\n

Untuk mencoba menginstal ulang semuanya, Anda dapat mencoba sesuatu seperti sudo cat /var/log/yum.log | grep Erased | awk '{print $5}'|xargs sudo yum install

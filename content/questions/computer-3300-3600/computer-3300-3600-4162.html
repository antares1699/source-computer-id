Ya, Anda bisa menginstal hMailServer atau Mercury Mail untuk memiliki server SMTP.

\n\n

Masalah besarnya, bagaimanapun, adalah spam.

\n\n
    \n

  1. Banyak (hampir semua) server email akan memblokir pesan yang datang dari koneksi perumahan, karena masif masalah spam.

    \n\n

    ISP Anda mungkin juga memblokir semua koneksi SMTP pada port 25, karena alasan yang sama.

    2. \n

  2. Sebagian besar penyedia email saat ini menggunakan SPF, yang membatasi server yang diizinkan untuk mengirim pesan dari domain mereka.

    \n\n

    Dengan kata lain, pesan dengan From: ...@gmail.com hanya dapat dikirim dari server Google sendiri. Jika Anda menggunakan server Anda sendiri, pesan kemungkinan besar akan ditandai sebagai spam atau ditolak seluruhnya, karena pemeriksaan SPF gagal.

    3. \n
\n\n

Dengan kata lain, Anda tidak akan bisa hanya mengirim email secara langsung. Kamu akan masih harus mengkonfigurasi server SMTP Anda ke menyampaikan semuanya melalui server penyedia email Anda.

\n\n

Selain itu, komputer hanya bekerja saat dihidupkan. Jika Anda mengirim pesan dan server penerima bermasalah, server Anda akan mencoba kembali pengiriman - tetapi tidak dapat melakukannya jika Anda mematikannya untuk malam itu. Di sisi lain, server penyedia email Anda selalu aktif.

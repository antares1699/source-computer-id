Setelah memutakhirkan buku catatan kerja saya dari Mountain Lion (10.8.5) ke Mavericks (10.9.4) akhir pekan ini, setiap kali saya membuka terminal, saya sekarang mendapatkan pesan kesalahan berikut: 

Unable to find any JVMs matching version \"1.6\"\n

Saya belum menginstal Java 6, beberapa alat yang dijalankan pada awal sesi shell harus bergantung padanya.

\n\n

Bagaimana saya bisa mengetahui apa akar dari masalah ini? Tolong beri tahu saya jika saya dapat memberikan informasi lebih lanjut untuk pemecahan masalah.

","source":"https://superuser.com/questions/792093/after-upgrade-to-osx-mavericks-unable-to-find-any-jvms-matching-version-1-6","author":"reinhard.codes","replies":[{"reply":"Apa versi OSX Anda sebelumnya?","author":"ssssteffff"}],"answers":[{"answer":"

Penginstal Mac OSX Mavericks menghapus versi Java yang diinstal sebelumnya. Anda mungkin memiliki referensi ke Java 6 di file .bash_profile Anda (atau lebih). Ini akan menjelaskan pesan tentang start-up Terminal.

\n\n

Versi terbaru Java (dari Java 7) didukung langsung oleh Oracle (Anda dapat menemukan petunjuk penginstalan di Java.com: Bagaimana cara menginstal Java untuk Mac saya ).

\n\n

Versi sebelumnya (termasuk Java 6) didukung oleh Apple. Jika Anda ingin menginstal Java 6, unduh Java untuk OS X 2014-001 patch langsung dari situs web Apple.

\n\n

Catatan : Anda dapat menemukan informasi berguna tentang kompatibilitas Java dan Mac OS X tentang ini FAQ Java .

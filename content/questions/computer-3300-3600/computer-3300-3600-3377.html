Saya memiliki LCD Samsung T220HD dengan speaker terintegrasi. Ketika saya menghubungkannya ke kartu video saya melalui kabel VGA, speaker berfungsi dengan baik. Tapi, saya baru saja membeli kabel DVI ke HDMI, (kartu video saya juga memiliki keluaran dvi). Saat saya menyambungkan kabel HDMI ke DVI, speaker tidak berfungsi. Saya biasa menghubungkan speaker monitor saya ke kartu audio melalui satu kabel jack. Tapi saya ulangi, ini hanya berfungsi pada VGA. Saya juga mencoba jack yang diakhiri dengan dua kabel analog. Saya menghubungkan jack ke kartu suara saya, dan input analog (Kiri dan Kanan) ke audio-in monitor saya. Masih tidak ada suara.

\n\n

Jadi, apakah ada solusi untuk membuat speaker monitor saya berfungsi pada koneksi DVI ke HDMI? Terima kasih.

Pastikan Anda menyambungkan kabel audio ke port 'HDMI / PC / DVI-D AUDIO IN' di bagian belakang TV menggunakan konektor aneh Merah / Putih ke Biru Tunggal yang mungkin disertakan dengan TV / Monitor Anda.

\n\n

Mengapa Anda tidak menggunakan kabel DVI sebagai pengganti kabel konverter DVI ke HDMI? TV Anda mendukung DVI, dan kabel HDMI memiliki kemampuan untuk memuat audio, jadi TV mungkin mencoba mengeluarkan audio yang tidak ada dari kabel HDMI.

Ini mungkin masalah perangkat lunak.

\n\n

Karena DVI dan HDMI kompatibel secara elektrik, dan HDMI dapat memiliki audio, komputer Anda mungkin mengarahkannya secara tidak benar.

\n\n

Saya pribadi belum melihat ini dengan kartu yang hanya DVI, tetapi mungkin ada baiknya memeriksa program / manajer driver suara Anda dan manajer audio Windows (jika menggunakan windows) dan hanya memastikan bahwa audio diarahkan ke port yang benar.

\n\n

Untuk laptop saya, saat saya mencolokkan kabel HDMI, audio lokal saya hanya diam kecuali saya mengatur speaker internal ke default.

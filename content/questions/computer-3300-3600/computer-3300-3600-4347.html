\n\n

Jika ada yang tertarik dengan solusi VB.Net, saya telah memprogram solusi Winform dengan Textbox multiline untuk menyalin / menempelkan anggota (bagaimanapun, terima kasih atas bantuan Anda): 

  Imports System.DirectoryServices 'first add a refernce to it from .Net Tab'\n

.... 

  Public Function MembersOfGroup(ByVal GroupName As String) As List(Of DirectoryServices.DirectoryEntry)\n        Dim members As New List(Of DirectoryServices.DirectoryEntry)\n        Try\n            Using search As New DirectoryServices.DirectoryEntry(\"WinNT://./\" & GroupName & \",group\")\n                For Each member As Object In DirectCast(search.Invoke(\"Members\"), IEnumerable)\n                    Dim memberEntry As New DirectoryEntry(member)\n                    members.Add(memberEntry)\n                Next\n            End Using\n        Catch ex As Exception\n            MessageBox.Show(ex.ToString)\n        End Try\n        Return members\n    End Function\n\n    Private Sub TxtGroup_KeyDown(ByVal sender As Object, ByVal e As System.Windows.Forms.KeyEventArgs) Handles TxtGroup.KeyDown\n        If e.KeyCode = Keys.Enter Then\n            Me.TxtGroupMembers.Text = String.Empty\n            If Me.TxtGroup.Text.Length <> 0 Then\n                Dim members As List(Of DirectoryServices.DirectoryEntry) = MembersOfGroup(Me.TxtGroup.Text)\n                For Each member As DirectoryServices.DirectoryEntry In members\n                    Me.TxtGroupMembers.Text &= member.Name & vbCrLf\n                Next\n            End If\n        End If\n    End Sub\n

Jika Anda dapat menggunakan PowerShell maka ini harus melakukan trik (berfungsi di komputer lokal, ubah variabel $GroupName dan jalur file keluaran sesuai dengan kebutuhan Anda): 

$GroupName = \"Administrators\"\n$GroupMembers = @()\n$Server = $env:computername\n$Group= [ADSI]\"WinNT://$Server/$GroupName,group\"\n$Members = @($Group.psbase.Invoke(\"Members\"))\n$Members | ForEach-Object { $GroupMembers += $_.GetType().InvokeMember(\"Name\", 'GetProperty', $null, $_, $null) }\nWrite-Output $GroupMembers | out-File \"C:\\somefolder\\somefile.txt\"\n

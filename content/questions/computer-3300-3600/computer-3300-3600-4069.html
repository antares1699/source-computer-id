Anda mungkin memiliki file sertifikat Agen Pemulihan Data EFS kedaluwarsa untuk domain Anda.

\n\n

Uji apakah ini masalahnya dengan menggunakan prosedur ini:

\n\n
    \n
  • simpan file teks di workstation Anda (c: \\ temp \\ myfile.txt misalnya)
    • \n
  • edit propertinya dan mengenkripsi file
    • \n
\n\n

Jika agen pemulihan tidak valid, Anda tidak akan dapat mengenkripsi file dan Anda akan menerima pesan kesalahan yang menunjukkan adanya masalah dengan agen pemulihan.

\n\n
\n\n

Untuk mengatasi masalah ini, ganti sertifikat kedaluwarsa dan perbarui Kebijakan Grup di workstation yang terkena dampak.

\n\n

Dari Blog Tim Layanan Direktori Microsoft :

\n\n

Temukan sertifikatnya

\n\n
    \n

  • Di server, cari dan buka Default Domain Policy arahkan ke Computer Configuration -> Windows Settings -> Public Key Policies -> Encrypting File System

    • \n

  • Di panel kanan, klik kanan pada sertifikat yang kedaluwarsa dan pilih All Tasks | Export

    • \n
\n\n

Buat sertifikat baru

\n\n
    \n
  • Dari workstation, jalankan perintah cipher /r:<filename-without-extension> dan gunakan kata sandi yang Anda inginkan saat diminta
    • \n
  • Salin file .CER dan .PFX yang dihasilkan ke server Anda
    • \n
\n\n

Impor sertifikat dan segarkan Kebijakan Grup

\n\n
    \n
  • Kembali ke server, temukan kembali Default Domain Policy, sorot Encrypting File System, dan pilih Add Data Recovery Agent
    • \n
  • Jelajahi file .CER dan .PFX yang baru saja Anda salin dan jawab YES jika Anda diperingatkan tentang pencabutan atau diminta untuk menginstal sertifikat.
    • \n
  • Tambahkan sertifikat ke penyimpanan Akar Tepercaya Domain (Konfigurasi Komputer -> Pengaturan Windows -> Kebijakan Kunci Publik -> Otoritas Sertifikat Akar Terpercaya)
    • \n
  • Paksa pembaruan kebijakan grup menggunakan gpupdate /force pada prompt perintah
    • \n
\n\n

Konfirmasikan bahwa file Anda yang ada telah diperbarui dengan sertifikat DRA baru

\n\n
    \n
  • periksa Atribut Lanjutan dari file yang dienkripsi dan bandingkan sidik jari DRA dengan sidik jari sertifikat yang baru saja Anda buat.
    • \n

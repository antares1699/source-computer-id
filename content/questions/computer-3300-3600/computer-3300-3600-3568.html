Di bash Anda dapat memperluas alias yang Anda ketik. Misalnya jika saya memiliki alias cmd=\"cd ~/user\", ketik cmd lalu tekan Ctrl + Alt + e akan mengubahnya menjadi cd ~/user. Apakah ada cara untuk melakukan ini dengan ikan?

","source":"https://superuser.com/questions/1091943/how-to-expand-aliases-in-fish-shell","author":"user3927950","replies":[{"reply":"Sudahkah Anda melihat bug ini dan perbaikan selanjutnya: github.com/fish-shell/fish-shell/issues/393 ? Kedengarannya seperti apa yang Anda cari, tapi saya tidak terlalu paham tentang ikan; sepertinya cmd + enter akan menggantikan alias dengan definisi.","author":"Argonauts"}],"answers":[{"answer":"

Di ikan, perintah alias hanya membuat fungsi sepele. Sebagai contoh, 

alias cmd 'cd ~/user'\n

Apakah hanya singkatan dari 

function cmd\n  cd ~/user\nend\n

Karena ini adalah fungsi pada ikan, Anda tidak dapat meluaskan badan di baris perintah saat Anda mengetik alias. Untuk situasi Anda, saya merekomendasikan singkatan; mis., abbr cmd 'cd ~/user'.

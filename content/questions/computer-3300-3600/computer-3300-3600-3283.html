Jawaban saya ( Baik jawaban), untuk anak cucu: 

HKEY_LOCAL_MACHINE\\SYSTEM\\CurrentControlSet\\Control\\Windows\\CSDVersion (a REG_DWORD)\n

Nilai level Paket Layanan (jika ada) diperoleh dengan melakukan CSDVersion >> 8 yaitu menggeser ke kanan CSDVersion REG_DWORD sebanyak 8 bit; misalnya REG_DWORD nilai akan menjadi 0x100 untuk SP1, 0x200 untuk SP2, dan seterusnya.

\n\n

Ini berfungsi untuk versi Windows kembali ke NT4, dan adalah cara yang disukai Microsoft untuk memperoleh tingkat SP sebelum pengenalan file OSVERSIONINFOEX struktur , seperti yang diuraikan dalam Persyaratan Dasar Windows [1], khususnya persyaratan " 7. Lakukan Pemeriksaan Versi Windows dengan Benar "(lihat tautan referensi di bawah untuk lebih jelasnya)

\n\n

[1] Referensi: http://msdn.microsoft.com/en-us/library/ms954375.aspx

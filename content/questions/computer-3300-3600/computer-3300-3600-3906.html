Saya memiliki akun penyimpanan di windows azure dan saya perlu menggunakan salah satu disk cloud VHD saya di komputer lokal saya. Saya tahu bahwa dimungkinkan untuk mengunduh seluruh gambar tetapi saya tidak membutuhkan ini. Saya berencana untuk menyalin beberapa informasi dari VHD.

\n\n

Bisakah disk VHD biru dipasang sebagai VHD lokal tanpa mengunduh?

","source":"https://superuser.com/questions/566307/is-it-possible-to-mount-windows-azure-vhd-disk-to-my-local-machine","author":"Anton Semenov","replies":[],"answers":[{"answer":"

Tidak, Anda tidak dapat memasang vhd yang didukung blob dari Windows Azure ke mesin lokal. Anda hanya dapat memasang ke Mesin Virtual (baik Linux atau Windows) dan instance peran web / pekerja (sebagai Drive Azure).

\n\n

Salah satu pilihan, selain mengunduh vhd, adalah membangun server file sendiri di Mesin Virtual (ftp, misalnya) yang memungkinkan Anda menelusuri / mengunggah / mengunduh file dari vhd Anda. Ini memungkinkan Anda menyimpan System of Record di satu tempat (vhd yang didukung blob).

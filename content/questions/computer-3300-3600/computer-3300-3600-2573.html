Sejak saya meningkatkan ke DSM 6.0, saya tidak lagi memiliki kemampuan untuk 'membangunkan perangkat lan di jaringan saya.\nSaya biasa melakukan SSH dari komputer jarak jauh dan menjalankan perintah berikut: 

ether-wake -i eth0 AA:BB:CC:DD:EE:FF\n

Ini akan mengirim paket WOL ke desktop saya. Namun ketika saya mencoba dan menjalankan perintah ini setelah pembaruan, saya mendapatkan kesalahan berikut: 

~ash: ether-wake: command not found\n

Apakah ada yang punya solusi untuk ini?

","source":"https://superuser.com/questions/1060254/synology-dsm-6-0-ssh-command-ether-wake-no-longer-working","author":"KawaGreen","replies":[],"answers":[{"answer":"

Mereka mengganti perintah dengan: 

synonet --wake [MAC address] eth0 \n

atau jika Anda memiliki ikatan: 

synonet --wake [MAC address] bond0 \n

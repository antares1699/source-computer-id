Ternyata bukan tanpa kode VBA. Inilah sesuatu yang saya temukan yang dimaksudkan untuk melakukan hal itu. ( Sumber )

\n\n

Deskripsi: Sampel Outlook VBA ini membuat dan menampilkan balasan untuk pesan yang saat ini terbuka atau dipilih termasuk lampiran dalam aslinya. 

Sub ReplyWithAttachments()\n    Dim rpl As Outlook.MailItem\n    Dim itm As Object\n\n    Set itm = GetCurrentItem()\n    If Not itm Is Nothing Then\n        Set rpl = itm.Reply\n        CopyAttachments itm, rpl\n        rpl.Display\n    End If\n\n    Set rpl = Nothing\n    Set itm = Nothing\nEnd Sub\n\nFunction GetCurrentItem() As Object\n    Dim objApp As Outlook.Application\n\n    Set objApp = Application\n    On Error Resume Next\n    Select Case TypeName(objApp.ActiveWindow)\n        Case \"Explorer\"\n            Set GetCurrentItem = objApp.ActiveExplorer.Selection.Item(1)\n        Case \"Inspector\"\n            Set GetCurrentItem = objApp.ActiveInspector.CurrentItem\n    End Select\n\n    Set objApp = Nothing\nEnd Function\n\nSub CopyAttachments(objSourceItem, objTargetItem)\n   Set fso = CreateObject(\"Scripting.FileSystemObject\")\n   Set fldTemp = fso.GetSpecialFolder(2) ' TemporaryFolder\n   strPath = fldTemp.Path & \"\\\"\n   For Each objAtt In objSourceItem.Attachments\n      strFile = strPath & objAtt.FileName\n      objAtt.SaveAsFile strFile\n      objTargetItem.Attachments.Add strFile, , , objAtt.DisplayName\n      fso.DeleteFile strFile\n   Next\n\n   Set fldTemp = Nothing\n   Set fso = Nothing\nEnd Sub\n

Catatan tentang kode:

\n\n
    \n

  1. Menggunakan GetCurrentItem() untuk mengembalikan item yang saat ini dipilih atau ditampilkan.

    2. \n

  2. Menggunakan CopyAttachments() prosedur untuk menyalin lampiran ke balasan.

    3. \n

  3. Ganti itm.Reply dengan itm.ReplyAll jika Anda lebih suka membalas ke semua.

    4. \n
\n\n

(Oh, dan saya menemukan ini dalam waktu sekitar 30 detik melakukan pencarian web untuk "balas ke semua dengan lampiran dalam pandangan").

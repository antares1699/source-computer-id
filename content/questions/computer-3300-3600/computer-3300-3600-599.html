Saya telah menggunakan git bash 1.9.5 dari https://git-scm.com/download/win (URL itu memulai unduhan) di ConsoleZ sebagai shell dan tidak ada masalah untuk membukanya di dalam tab.

\n\n

Namun, dengan Git baru untuk windows 2.4.5.1 RC4, itu selalu terbuka di jendela baru. Apa saja opsi baris perintah untuk digunakan sebagai shell dan terbuka di dalam tab?

\n\n

Saya belum menemukan doc atau info tentang opsi baris perintah git-bash.exe.

","source":"https://superuser.com/questions/937539/how-to-open-git-for-windows-shell-inside-a-tab-on-consolez","author":"Juan Antonio Tubío","replies":[],"answers":[{"answer":"

Perintah berikut digunakan di ConEmu default tugas untuk GitBash. 

git-cmd.exe --no-cd --command=usr/bin/bash.exe -l -i\n

Sebenarnya, git-bash.exe hanyalah pembungkus yang memulai bash.exe. Dan git-cmd.exe melakukan hal yang sama. Namun git-bash.exe adalah aplikasi yang ditautkan ke subsistem GUI, dan git-cmd.exe adalah aplikasi konsol yang sebenarnya. ConsoleZ (tidak seperti ConEmu) tidak dapat menjalankan aplikasi GUI di dalam tabnya.

Pertanyaan ini untuk tujuan menonaktifkan layanan yang tidak perlu berjalan di latar belakang program yang disebut TeamViewer.

\n\n

Biasanya, layanan latar belakang berjalan sepanjang waktu.

\n\n

Saya ingin itu dinonaktifkan dari prompt Administrator PowerShell dan diaktifkan kembali atas permintaan / dan juga dari Terminal Linux.

\n\n

Lebih disukai dengan satu perintah sederhana atau fungsi yang ditentukan.

\n\n

Note1: Saya membutuhkan ini di Windows (10) dan Linux (Mint 18).

\n\n

Note2: Saya butuh solusi CLI.

Linux (Terminal)

\n\n
    \n

  • Mengaktifkan: 

    sudo teamviewer daemon enable\n
    • \n

  • Menonaktifkan: 

    sudo teamviewer daemon disable\n
    • \n
\n\n

Alias ​​pribadi Anda akan terlihat persis seperti itu.

\n\n

Cukup tambahkan ke file alias Bash pribadi Anda:

\n\n
~/.bash_aliases\n
\n\n
    \n

  • Mengaktifkan: 

    alias tv-enable='sudo teamviewer daemon enable'\n
    • \n

  • Menonaktifkan: 

    alias tv-disable='sudo teamviewer daemon disable'\n
    • \n
\n\n
\n\n

Windows (PowerShell)

\n\n
    \n

  • Mengaktifkan: 

    sc.exe config \"TeamViewer\" start= auto\nnet start TeamViewer\n
    • \n

  • Menonaktifkan: 

    sc.exe config \"TeamViewer\" start= disabled\nnet stop TeamViewer\n
    • \n
\n\n

Alias ​​pribadi Anda akan terlihat persis seperti itu.

\n\n

Cukup tambahkan ke file profil PoweShell pribadi Anda:

\n\n
$PROFILE\n
\n\n
    \n

  • Mengaktifkan: 

    function tv-enable {\n    sc.exe config \"TeamViewer\" start= auto\n    net start TeamViewer\n}\n
    • \n

  • Menonaktifkan: 

    function tv-disable {\n    sc.exe config \"TeamViewer\" start= disabled\n    net stop TeamViewer\n}\n
    • \n

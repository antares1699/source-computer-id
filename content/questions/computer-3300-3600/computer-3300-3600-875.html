Anda harus melakukan sedikit pekerjaan pemformatan manual, karena Excel tidak memiliki format angka yang Anda inginkan.

\n\n

Anda bisa membuat grafik terlihat seperti ini dengan sedikit usaha:

\n\n

1 - Format angka khusus

\n\n

Klik kanan pada nomor sumbu, pilih "format sumbu", pergi ke bagian "angka", dan masukkan format Kustom berikut: 

\"10^\"#\n

Buat sumbu Anda menggunakan format kustom ini.

\n\n

Ini akan menambahkan teks "10 ^" ke depan nomor yang ditampilkan.

\n\n

2 - Loggifikasi data Anda

\n\n

Format kustom dari 1) mengasumsikan bahwa data Anda sudah dicatat. Jadi kami perlu mencatat data Anda, dan membuat grafik sebagai gantinya. Temukan log dari setiap titik data: 

Data            \nX   Y   Log10(X)    Log10(Y)\n30  300 1.477121255 2.477121255\n28  300 1.447158031 2.477121255\n26  300 1.414973348 2.477121255\n300 200 2.477121255 2.301029996\n280 200 2.447158031 2.301029996\n260 200 2.414973348 2.301029996\n

Buat grafik plot Anda menjadi dua kolom kanan.

\n\n

3 - Perbaiki garis log

\n\n

Plot log Excel mengasumsikan bahwa baris log utama harus di 1, 10, 100, 1000 dll. Tetapi Anda ingin baris utama di 1, 2, 3 karena Anda sudah mencatat data Anda.

\n\n

Jadi kita perlu membuat garis kita sendiri, dan memformatnya menjadi garis abu-abu tipis.

\n\n

Pertama, hapus garis kisi dari plot saat kita membuatnya sendiri.

\n\n

Kemudian tambahkan pasangan titik data yang menentukan awal dan akhir setiap baris setiap 10, 100, 1000 dll, lalu catat datanya. Letakkan spasi di antara setiap pasangan titik untuk memutus garis dan juga memudahkan untuk melihat apa yang terjadi. Anda harus mendapatkan tabel seperti:

Jadi pendekatannya adalah menambahkan deret dummy di sepanjang setiap sumbu, di tempat yang Anda inginkan label sumbu. Sembunyikan titik-titik ini, dan tambahkan label data, masukkan 101, 102, dll (untuk 10 ^ 1, 10 ^ 2, dll), dan format eksponen menjadi superskrip. Dan ini sulit dilakukan dengan tangan, karena sulit untuk memilih eksponen dan menerapkan pemformatan, di antara hal-hal sulit lainnya.

\n\n

Jadi saya rutin menulis sedikit. Pilih plot log-log dengan sumbu di tepi kiri dan bawahnya, dan jalankan kode di bawah ini. 

Sub NiceExponentialAxisLabels()\n  Dim cht As Chart\n  Dim iPt As Long, iLog As Long, iMin As Long, iMax As Long\n  Dim vXVals As Variant, vYVals As Variant\n  Dim dFont As Double\n\n  Set cht = ActiveChart\n\n  ' HORIZONTAL AXIS ------------------------------------\n  cht.Axes(xlCategory).TickLabels.NumberFormat = \";;;\" ' hide tick labels\n\n  ' build arrays of X and Y values\n  iMin = WorksheetFunction.Log10(cht.Axes(xlCategory).MinimumScale)\n  iMax = WorksheetFunction.Log10(cht.Axes(xlCategory).MaximumScale)\n  ReDim vXVals(1 To 1)\n  ReDim vYVals(1 To 1)\n  iPt = 0\n  For iLog = iMin To iMax\n    iPt = iPt + 1\n    ReDim Preserve vXVals(1 To iPt)\n    ReDim Preserve vYVals(1 To iPt)\n    vXVals(iPt) = 10 ^ iLog\n    vYVals(iPt) = cht.Axes(xlValue).MinimumScale\n  Next\n\n  ' add series, hide points, add and format labels\n  With cht.SeriesCollection.NewSeries\n    .Name = \"horizontal\"\n    .XValues = vXVals\n    .Values = vYVals\n    .Format.Line.Visible = False\n    .MarkerStyle = xlMarkerStyleNone\n    .HasDataLabels = True\n    .DataLabels.Position = xlLabelPositionBelow\n    For iPt = 1 To .Points.Count\n      With .DataLabels(iPt)\n        dFont = .Font.Size\n        .Text = 10 & WorksheetFunction.Log10(vXVals(iPt))\n        With .Characters(3, Len(.Text) - 2)\n          .Font.Superscript = True\n          .Font.Size = dFont + 2\n        End With\n        With .Characters(1, 2)\n          .Font.Size = dFont\n        End With\n      End With\n    Next\n  End With\n\n  ' VERTICAL AXIS ------------------------------------\n  cht.Axes(xlValue).TickLabels.NumberFormat = \"_0_0_0_0_0_0_0\" ' hide but maintain margin\n\n  ' build arrays of X and Y values\n  iMin = WorksheetFunction.Log10(cht.Axes(xlValue).MinimumScale)\n  iMax = WorksheetFunction.Log10(cht.Axes(xlValue).MaximumScale)\n  ReDim vXVals(1 To 1)\n  ReDim vYVals(1 To 1)\n  iPt = 0\n  For iLog = iMin To iMax\n    iPt = iPt + 1\n    ReDim Preserve vXVals(1 To iPt)\n    ReDim Preserve vYVals(1 To iPt)\n    vXVals(iPt) = cht.Axes(xlCategory).MinimumScale\n    vYVals(iPt) = 10 ^ iLog\n  Next\n\n  ' add series, hide points, add and format labels\n  With cht.SeriesCollection.NewSeries\n    .Name = \"vertical\"\n    .XValues = vXVals\n    .Values = vYVals\n    .Format.Line.Visible = False\n    .MarkerStyle = xlMarkerStyleNone\n    .HasDataLabels = True\n    .DataLabels.Position = xlLabelPositionLeft\n    For iPt = 1 To .Points.Count\n      With .DataLabels(iPt)\n        dFont = .Font.Size\n        .Text = 10 & WorksheetFunction.Log10(vYVals(iPt))\n        With .Characters(3, Len(.Text) - 2)\n          .Font.Superscript = True\n          .Font.Size = dFont + 2\n        End With\n        With .Characters(1, 2)\n          .Font.Size = dFont\n        End With\n      End With\n    Next\n  End With\nEnd Sub\n

Catatan: kode dapat disalin dari sini dan ditempelkan ke modul kode biasa. Lihat Cara: Menggunakan Makro Orang Lain di blog saya jika Anda belum pernah melakukan ini sebelumnya.

Di bawah ini adalah dua bagan, yang asli, dan yang satu dengan label eksponensial yang bagus.

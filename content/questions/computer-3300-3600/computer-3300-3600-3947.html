    \n\n

    Periksa Windows 7 dan Windows 8 menggunakan subnet dan gateway yang sama. Gunakan perintah berikut 

    ipconfig\n

    Jika Windows 7 dapat melakukan ping ke perangkat, Windows 8 juga dapat melakukannya.

    2. \n

  2. Matikan firewall

    \n\n

    Di file explorer, buka mengikuti dan matikan firewall 

    Control Panel\\All Control Panel Items\\Windows Firewall\n

    Jika Anda memiliki perangkat lunak firewall lain seperti norton360, Anda harus menonaktifkannya di dalam perangkat lunak tersebut.

    3. \n

    \n\n

    Di file explorer, lanjutkan ke mengikuti 

    Control Panel\\All Control Panel Items\\Network and Sharing Center\\Advanced sharing settings\n

    Di bawah bagian All Networks, centang File sharing connections, pilih Enable file sharing for devices that use 40- or 56-bit encryption

    4. \n

  4. Periksa kredensial yang diingat

    \n\n

    Di file explorer, buka 

    Control Panel\\All Control Panel Items\\Credential Manager\n

    Klik Windows Credentials. Periksa apakah ada kredensial yang terkait dengan NAS itu, hapus.

    5. \n

  5. WiFi dan Kabel

    \n\n

    Jika Anda memiliki koneksi w-fi dan kabel, coba matikan wi-fi.

Saya memiliki masalah yang sama setelah baru-baru ini menginstal windows 8, dan setelah mencoba beberapa solusi yang kurang lebih rumit, yang memperbaikinya sama dengan solusi davefromcamp (Saya belum diizinkan untuk memberi suara positif, tetapi saya ingin berbagi bahwa ini juga berhasil untuk saya, dan itu adalah solusi yang sangat sederhana)!

\n\n

Triknya adalah:

\n\n
    \n
  1. Buka "Sistem" dari Panel Kontrol. Salah satu cara untuk mengakses panel kontrol adalah dengan mengarahkan mouse ke sudut kanan atas atau bawah, geser kursor ke Pengaturan, klik, lalu pilih Panel Kontrol dari daftar. Klik "Sistem dan Keamanan" lalu "Sistem".
    2. \n
  2. Di bawah "Pengaturan nama komputer, domain, dan grup kerja", klik tautan "Ubah pengaturan", yang terletak di sebelah kanan di sebelah ikon perisai administrator.
    3. \n
  3. Pada tab pertama "Nama komputer", klik tombol yang bertuliskan "ID Jaringan ..." Ini membuka jendela dengan hanya dua pilihan.
    4. \n
  4. Pilih "Ini adalah komputer rumah; ini bukan bagian dari jaringan bisnis". Klik selesai dan biarkan windows restart sendiri.
    5. \n
\n\n

Semoga ini bisa membantu lebih banyak orang dengan masalah yang sama!

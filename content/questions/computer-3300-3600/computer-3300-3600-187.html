Menghapus folder .git dapat menyebabkan masalah pada repositori git Anda. Jika Anda ingin menghapus semua riwayat komit Anda tetapi menyimpan kode dalam keadaannya saat ini, sangat aman untuk melakukannya seperti berikut ini:

\n\n

Periksa 

git checkout --orphan latest_branch\n

Tambahkan semua file 

git add -A\n

Lakukan perubahan 

git commit -am \"commit message\"\n

Hapus cabangnya 

git branch -D master\n

Ubah nama cabang saat ini menjadi master 

git branch -m master\n

Terakhir, paksa perbarui repositori Anda 

git push -f origin master\n

PS: ini tidak akan menyimpan riwayat commit lama Anda.

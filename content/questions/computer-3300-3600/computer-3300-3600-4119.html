Saya baru-baru ini memutakhirkan komputer Windows XP (!) Saya ke satu SSD M4 Krusial. Sistem berjalan sangat cepat dan tanpa masalah, tetapi menjalankan HD Tune Pro (dan CrystalDisk) mengungkapkan bahwa kecepatan baca berurutan hanya 80MB / s (dengan ukuran blok 64K) dan tulis adalah 60MB / s. Selain itu, IOPS hanya 6.000 (untuk 4KB). Ini juga tampaknya sangat rendah.

\n\n

Ada dua pengontrol SATA (mengapa?) Pada motherboard: Promise dan VIA. Spesifikasi ( http://www.asus.com/Motherboards/AMD_Socket_939/A8V_Deluxe/#specifications ) menyarankan bahwa pengontrol Promise (yang saat ini saya gunakan) dapat dibatasi ke SATA 1.5. Namun, jika itu masalahnya maka kecepatan transfer seharusnya jauh lebih tinggi, saya pikir. 1,5 Gb / dtk = 187,5 MB / dtk. Juga, mengapa IOPS sangat rendah? (6.000 * 4KB = 24MB).

\n\n

Penelitian di internet menunjukkan bahwa kinerjanya harus lebih dari 400MB / dtk, jadi hard disk seharusnya jauh lebih cepat.

\n\n

Saat ini menggunakan port Promise SATA. Saya mencoba mencolokkan SSD ke port VIA SATA pada motherboard, tetapi saya mendapatkan kesalahan bios "Tidak ada BIOS yang diinstal." dan itu tidak bisa boot. Aneh karena ketika tidak ada hard drive yang dipasang, saya mendapatkan pesan "Tidak ada hard drive yang ditemukan."

\n\n

Saya telah menginstal driver Promise 378 selama penginstalan Windows.

\n\n

Apakah menurut Anda ada masalah dengan Windows XP, atau mungkin drivernya? Bisakah motherboard diatur untuk RAID? Saya mencoba mengubah pengaturan, tetapi tidak membantu.

Bagaimana Anda melakukan peningkatan dan proses penyalinan dari HDD lama Anda ke SSD?

\n\n

Jika Anda tidak menggunakan alat yang mengetahui properti khusus SSD seperti persyaratan untuk penyelarasan partisi yang benar, Anda akan kehilangan banyak kecepatan.

\n\n

Selain itu, Anda harus selalu menggunakan port SATA southbridge untuk SSD. Selama Anda tidak memiliki kartu pengontrol PCI-E 4x khusus, southbridge akan lebih cepat. Dengan menggunakan port ini, Anda bisa mendapatkan hingga 200-300MB / detik; kecepatan maksimum tidak memungkinkan karena hanya port SATA-3Gbit.

\n\n

Dan pastikan port SATA southbridge diatur ke AHCI bukan IDE.

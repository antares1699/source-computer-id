224.0.0.22 adalah a Multicast alamat ip untuk Protokol Manajemen Grup Internet (IGMP). Kemungkinan besar ini dihasilkan dari TeamSpeak atau Google Talk.

\n\n

224.0.0.252 adalah alamat multicast lain milik Resolusi Nama Multicast lokal-tautan , bawaan Windows:

\n\n
\n

Tautan Nama Multicast Lokal\n Resolusi (atau LLMNR) adalah protokol\n berdasarkan Sistem Nama Domain (DNS)\n format paket yang memungkinkan kedua IPv4\n dan host IPv6 untuk menjalankan nama\n resolusi untuk host di lokal yang sama\n tautan. Ini termasuk dalam Windows Vista,\n Windows Server 2008 dan Windows 7.

\n
\n\n

Ini bukan alamat Internet, hanya digunakan di jaringan lokal.

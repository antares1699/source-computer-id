Saya membuka sekitar 200 tab halaman web di Firefox 59.0.2 (64-bit) di Ubuntu 16.04.

\n\n

Saya menutup Firefox, lalu mem-boot ulang Ubuntu.

\n\n

Ketika saya memulai Firefox lagi, saya memiliki kesempatan untuk memulihkan 200 tab halaman web sebelumnya, dengan mengklik Riwayat-> Pulihkan Sesi Sebelumnya. Tetapi alih-alih melakukan itu, saya mengklik tab baru dan membuka halaman baru, dan menutup tab, yang menutup Firefox dari harapan saya (di browser Chrome, menutup satu-satunya tab tidak menutup browser Chrome).

\n\n

Sekarang mengklik History-> Restore Previous Session hanya mengembalikan satu tab halaman web. Adakah cara untuk memulihkan sesi sebelumnya yang terdiri dari 200 tab laman web?

\n\n

Perhatikan bahwa menurut saya sulit untuk menemukan URL 200 halaman web sebelumnya dari riwayat terbaru di Firefox. Sesi sebelumnya dari 200 tab halaman web juga dibuat dengan mengklik Riwayat-> Pulihkan Sesi Sebelumnya, dan Firefox sebenarnya tidak membuka semua 200 halaman web saat saya memulihkannya, tetapi membuka setiap halaman web ketika saya benar-benar mengklik tabnya, dan saya tidak mengklik semua dari 200 tab.

Maaf atas tanggapan yang terlambat, saya menghadapi masalah yang sama dengan Firefox 78: Saya menutup jendela firefox secara tidak sengaja saat masih memulai, dan semua 200+ tab saya yang terbuka hilang :(

Saya menempelkan prosedur yang benar di sini untuk referensi di masa mendatang, tetapi semua kredit diberikan kepada Martin Brinkmann dari Ghacks: Cara memulihkan sesi Firefox jika Pemulihan Sesi tidak berfungsi dengan benar :

\n

Saya menyelesaikan langkah-langkah berikut:

\n
    \n
  1. Tutup browser Firefox. Pastikan tidak ada browser yang berjalan.
    2. \n
  2. Buka folder profil dan ganti nama file sesi saat ini sessionstore.jsonlz4 untuk sessionstore-backup.jsonlz4 .
    3. \n
  3. Buka sessionstore-backups folder profil.
    4. \n
  4. Ganti nama recovery.jsonlz4 untuk recovery-backup.jsonlz4 .
    5. \n
  5. Ganti nama recovery.backlz4 untuk recovery-backup.backlz4 .
    6. \n
  6. Salin sesi pemulihan cadangan yang ingin Anda pulihkan, mis. Previous.jsonlz4 .
    7. \n
  7. Kembali ke direktori root profil Firefox.
    8. \n
  8. Tempel file yang disalin ke folder root.
    9. \n
  9. Ubah namanya menjadi sessionstore.jsonlz4 .
    10. \n
  10. Mulai ulang browser Firefox.
    11. \n
  11. Firefox mungkin mengambil sesi secara otomatis. Jika tidak, tekan Alt-key pada keyboard Anda dan pilih History> Restore Previous Session.
    12. \n

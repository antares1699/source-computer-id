Pemeriksaan VBA cepat.
Ini menampilkan Kotak Pesan setiap kali menemukan rumus pada lembar apa pun dan mewarnai sel dengan warna kuning. 

Sub ColorAllFormulas()\n    Application.ScreenUpdating = False\n    For Each Sheet In ActiveWorkbook.Sheets\n        For Each cell In Sheet.UsedRange\n            If cell.HasFormula Then \n              cell.Interior.Color = 65535\n              Msgbox \"WHOA, I FOUND SOMETHING AT \" & cell.address\n            Endif\n        Next\n    Next\n    Application.ScreenUpdating = True\nEnd Sub\n

Untuk menjalankan ini, pilih Tools -> Macro -> Visual Basic Editor dari menu bar. Kemudian dalam tampilan Proyek, klik kanan pada tingkat atas "VBAProject (test.xlsx)" -> Sisipkan -> Modul. Tempel kode di atas ke dalam jendela "test.xlsx - Module1 (Code)" dan klik tombol Run.

Ada cara yang jauh lebih mudah daripada menggunakan makro. Buka dokumen excel Anda. Anda akan melihat bentuk yang mirip dengan saya.

\n\n

\n\n

Klik pada opsi menu Rumus (pada bilah hijau.) Selanjutnya, klik tombol Tampilkan (ini adalah tombol kelima dari kiri,) dan pilih Tampilkan Rumus. Ini akan mengubah spreadsheet Anda untuk menampilkan rumus, bukan nilai yang dievaluasi. Lihat tangkapan layar saya di bawah.

\n\n

\n\n

Dalam hal ini, kolom A berisi informasi numerik statis. Kolom B berisi rumus sederhana. FYI, opsi ini telah ada di Excel selama saya menggunakannya (kembali ke Excel 97.) Cara menampilkan / menyembunyikan rumus telah berbeda, tetapi fitur ini mungkin telah ada sejak awal Excel.

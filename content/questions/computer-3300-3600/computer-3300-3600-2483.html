Saya mencoba menggunakan sed untuk menambahkan string setelah pola tertentu tetapi saya ingin mengecualikan kasus di mana string tambahan sudah ada.

\n\n

Sebagai contoh: 

\"text.txt\"\n\nHello\nBar\nHello world!\n

Jika saya menggunakan sed -i -e 's/Hello/Hello world!' text.txt baris ketiga menjadi

\n\n

Hello world! world!

\n\n

tapi saya ingin tetap utuh.

\n\n

Saya telah menemukan ini pertanyaan terkait tetapi tidak semua baris di file saya perlu dimodifikasi.

\n\n

Adakah yang punya resep untuk ini?

Jawaban atas pertanyaan Anda agak tersembunyi di salah satu jawaban di pertanyaan terkait: 

/Hello world!/ ! s/Hello/Hello world!/\n

Bagian pertama adalah "alamat", yaitu perintah berikut hanya berlaku untuk baris yang cocok. ! meniadakan kondisi, yaitu perintah hanya akan diterapkan ke baris tidak cocok dengan alamatnya. Jadi, dengan kata lain, Ganti Hello dengan Hello world! pada baris yang tidak berisi Hello world! .

Anda dapat menggunakan awal dan akhir jangkar baris eksplisit (^ dan $): 

$ sed -e 's/^Hello$/Hello world!/' <<EOF\n> Hello\n> Bar\n> Hello world!\n> EOF\nHello world!\nBar\nHello world!\n

Ini juga bisa sedikit ditingkatkan untuk keterbacaan / singkatnya dengan menggunakan \\0 yang menggantikan seluruh pertandingan ... Dengan cara ini Anda tidak perlu mengulangi Hello: 

$ sed -e 's/^Hello$/\\0 world!/' <<EOF\n> Hello\n> Bar\n> Hello world!\n> EOF\nHello world!\nBar\nHello world!\n

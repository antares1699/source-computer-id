Windows 10 terus memberi tahu saya bahwa akun yang terhubung ke akun login lokal PC saya perlu "diperbaiki", tetapi akun yang bermasalah sudah tidak ada lagi. Saya tidak dapat "memutuskan" karena Windows 10 memberi tahu saya bahwa itu bukan milik domain (yang benar) dan menolak untuk melewati dialog itu. Dan saya tidak dapat "mengelola" akun tersebut karena tidak ada sehingga saya tidak dapat masuk ke dalamnya.

\n\n

\n\n

\n\n

Bagaimana cara menghapus akun ini dari akun login lokal PC? Akun login PC awalnya dibuat tanpa akun online, lalu saya login ke akun online saya dan akun itu terlampir. Saya tidak tahu bagaimana akun kedua yang rusak dapat dilampirkan, tetapi saya mendapat peringatan berkelanjutan tentang hal itu yang perlu diperbaiki.

\n

Versi Windows saya: Windows 10, 1909.

Di Windows '"Info Anda", pilih "Masuk dengan akun lokal" dan ubah akun Anda ke akun lokal. Anda harus memasukkan kata sandi untuk akun lokal.

\n\n

Ini akan mengeluarkan Anda dan kemudian masuk kembali menggunakan kata sandi baru.

\n\n

Kembali ke "Info Anda" dan putuskan semua akun yang ingin Anda hapus baik di halaman "Akses kantor atau sekolah" atau halaman "Info Anda".

\n\n

Setelah selesai, kembali ke "Info Anda" dan masuk dengan akun Microsoft Anda lagi.

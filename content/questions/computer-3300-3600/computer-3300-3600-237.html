Coba yang berikut ini di PowerShell (setelah masuk ke direktori yang benar):

\n\n

$files = get-childitem
foreach ($file in $files) {gzip $file}

\n\n

Itu akan melewati semua file di direktori, dan memampatkan semuanya.

\n\n

Edit: Jika Anda ingin melakukan semua file dalam pohon direktori (yaitu dalam folder dan semua subfoldernya) ubah saja $files = get-childitem menjadi $files = get-childitem -recurse

GZIP dapat digunakan untuk memampatkan file individual pasti ... tetapi sebagian besar waktu Anda akan melihat jumlah penghematan ruang disk yang minimal. File yang menempati 1 sektor akan menggunakan ruang disk 1 sektor penuh, meskipun file lebih kecil dari ukuran sektor. (yaitu ukuran-sektor = 16kb, file 8kb ... 'terkompresi' adalah 5kb ... masih membutuhkan 16kb dari "ruang-disk") Kekuatan gzip jauh lebih jelas ketika Anda menggabungkan semua file ke dalam file tunggal (biasanya menggunakan tar) dan kemudian kompres arsip tunggal. (digabungkan, "arsip" dapat mengambil semua file dan menggabungkannya menjadi 1 file tunggal di disk ... yang akan memiliki lebih sedikit "ruang mati" antar sektor)

\n\n

Sebagai contoh: 

tar -czf compressed_archive.tgz folder/*\n

tar -c membuat arsip, z menggunakan kompresi gzip, dan f menulis hasilnya ke file bernama compressed_archive.tgz. Anda juga dapat beralih menggunakan kompresi bz2 (yang lebih baik dalam banyak kasus) hanya dengan mengalihkan tombol z ke j.

\n\n

Jika Anda ingin mengekstrak file dari arsip, Anda dapat melakukan sebaliknya 

tar -zxf compressed_archive.tgz\n

yang akan x mengekstrak file yang z gzip'd, f dari file arsip bernama compressed_archive.tgz.

find menggunakan sintaks globbing, dan Anda dapat menggunakan -type d untuk menemukan direktori saja: 

find . -type d -name '.?*' | vim -\n

Anda harus menjadi sangat Berhati-hatilah saat melakukan globbing atau menggunakan regex untuk menemukan file .hidden, karena Anda dapat dengan mudah mengambil . dan menghapus seluruh folder Anda, atau lebih buruk lagi, cocokkan .. dan Anda menghapus seluruh folder induk. Pertimbangkan konsekuensi dari perintah berikut: 

/home/someuser bash$ rm -rf .*\n

Saya menggunakan DLink DNS-323 untuk ini. Ini adalah penutup NAS 2-drive sehingga Anda dapat membeli hard drive Anda sendiri. Daya rendah: berputar saat menganggur. Dan memiliki port USB yang dapat Anda gunakan sebagai server cetak. Itu juga dapat bertindak sebagai server DHCP jika Anda membutuhkannya karena alasan tertentu. Saat penulisan, saya melihatnya di Amazon seharga $ 128 AR.

\n\n

Ia juga memiliki RAID tapi samar-samar saya ingat pernah membaca beberapa hal aneh dengannya. Sesuatu seperti Anda harus memformat ulang kedua drive hanya untuk mengaktifkan RAID-1.

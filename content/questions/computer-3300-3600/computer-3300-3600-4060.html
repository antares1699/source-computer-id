Bergantung pada alat yang Anda gunakan, ini mungkin sesederhana memberi tahu alat untuk mengawasi folder desktop Anda. Selesai. Namun, tidak semua dari mereka melakukan itu.

\n\n

Dropbox, misalnya, tidak. Yang saya rekomendasikan adalah symlinking folder Desktop Anda ke dalam folder Dropbox Anda dalam situasi ini. Symlinking adalah cara untuk memberi tahu sistem file bahwa file atau folder tertentu sebenarnya adalah file atau folder lain di drive Anda.

\n\n

mklink /D \"C:\\Users\\Your User\\DropBox\\Desktop\\\" \"C:\\Users\\Your User\\Desktop\\\"

\n\n

Ini sama dengan mengatakan "Folder saya di DropBox \\ Desktop Betulkah hanya folder desktop saya. "Windows menangani semua ini, jadi jika aplikasi (seperti DropBox) mencoba menelusuri folder, ia bahkan tidak akan tahu bahwa windows sedang menerjemahkan jalurnya.

\n\n

Menggunakan metode ini memungkinkan Anda untuk lebih atau kurang meninggalkan folder desktop Anda sendiri. Beberapa program tidak suka jika Anda memindahkan folder desktop Anda, dan jika Anda memindahkan desktop Anda ke DropBox dan kemudian memutuskan untuk menghapus dropbox atau menggunakan alat lain untuk mencadangkan / menyinkronkan konten, Anda harus memindahkan semuanya lagi. Dengan metode ini, Anda cukup menghapus symlink, atau bahkan memiliki beberapa layanan sinkronisasi yang mencadangkan desktop Anda pada saat yang bersamaan (Anda dapat menautkan beberapa folder virtual ke folder nyata yang sama pada saat yang bersamaan).

Windows memungkinkan pengubahan lokasi folder pengguna, termasuk folder Desktop dan Dokumen.

\n\n
    \n
  1. Di File Explorer, klik kanan folder Desktop / Dokumen di bawah Akses Cepat dan klik Properti .
    2. \n
  2. Beralih ke Lokasi tab, dan klik Pindah tombol.
    3. \n
  3. Buka folder yang ingin Anda gunakan, (dalam kasus Anda, folder Dropbox), klik Pilih folder , dan Menerapkan perubahan Anda.
    4. \n
  4. Jika Anda melihat dialog yang menanyakan "Apakah Anda ingin memindahkan semua file dari lokasi lama ke lokasi baru?", Klik Iya untuk memindahkan semua file ke lokasi baru.
    5. \n
\n\n

Jika semuanya dilakukan dengan benar, desktop Anda akan menampilkan konten folder Dropbox Anda.

Seperti yang disebutkan dalam jawaban lain yang ingin Anda gunakan dired .

\n\n

Pertama tandai file melalui kunci m. Alternatifnya, Anda dapat menggunakan % m hingga dired-mark-files-regexp (dalam hal menandai semua file, masukkan .* sebagai regexp Anda).

\n\n

Anda akan ingin menggunakan % R hingga dired-do-rename-regexp. Ini akan membatasi penggantian nama ekspresi reguler ke file yang ditandai. Anda harus menggunakan sintaks regexp Emacs yang berbeda dari sintaks unix biasa.

\n\n

Jika saya memiliki sejumlah file di direktori bernama berikut ini: 

 01.png\n 02.png\n 03.png\n 04.png\n

Dan saya ingin mereka diganti namanya sehingga semuanya diawali dengan bab-01, saya akan memasukkan penekanan tombol berikut. 

% m - mark regexp\n.*\\.png - mark all png files\n% R - rename regexp\n\\([0-9]+\\)\\(\\.png\\) - matching regexp, save the parens into variables\nchapter-01-\\1\\2 - utilizing the previous variables`\n! - during the query to denote rename all the rest\n

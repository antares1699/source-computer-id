Saya memiliki banyak koleksi file gambar, dari berbagai kamera, ponsel, dll. Semuanya diurutkan dengan rapi ke dalam folder berdasarkan tanggal / subjek (misalnya: 2011-12-25-Family Christmas Dinner) dan saya mengelolanya menggunakan Picasa, yaitu melakukan pekerjaan yang cukup bagus sebenarnya. Picasa memungkinkan saya menelusuri menurut nama, tanggal, atau ukuran (file) di dalam map, dan menurut warna untuk semua gambar.

\n\n

Sekarang saya ingin mencari dengan kriteria berikut misalnya:

\n\n
    \n
  • 10 gambar dengan resolusi terbesar (secara rekursif di semua direktori)
    • \n
  • 10 gambar terbesar (ukuran file, di semua direktori)
    • \n
  • gambar "serupa" (semua direktori)
    • \n
  • dll.
    • \n
\n\n

Saya telah menemukan ImgSeek, dan berfungsi untuk gambar yang serupa, tetapi saya tidak dapat menemukan apa pun untuk mencari kriteria lain ... Ada saran?

Mengingat Windows 7, Anda harus dapat melakukan sebagian besar dari apa yang Anda inginkan di Windows Explorer.

\n\n
    \n
  1. Jelajahi folder level atas, dan di kotak pencarian di sudut kanan atas ketik name:* dan tekan Enter (atau tunggu saja). Itu akan menemukan semua gambar secara rekursif.
    2. \n
  2. Dari tampilan (menu atau dropdown) pilih Details
    3. \n
  3. Klik kanan tajuk daftar file (bilah dengan 'Nama', 'Tanggal', dll di dalamnya), dan pilih bidang yang ingin Anda sortir; misalnya Anda dapat memilih:
      \n
    1. Horizontal resolution dan Vertical resolution untuk mengurutkan berdasarkan resolusi gambar (dpi)
      2. \n
    2. Height dan Width untuk mengurutkan berdasarkan ukuran piksel
      3. \n
    3. Ukuran untuk mengurutkan berdasarkan ukuran file
      4. \n
    4. Waktu pencahayaan untuk mengurutkan berdasarkan, yah, waktu pencahayaan (ada juga properti EXIF ​​serupa lainnya)
      5. \n
    4. \n
\n\n

... dll.\nPerhatikan bahwa Windows 7 juga memungkinkan Anda untuk memfilter menurut salah satu bidang tersebut dengan mengklik panah di sebelah kanan bidang header.

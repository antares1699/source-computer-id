Saya baru saja menemukan metode serupa, menggunakan pipa baris perintah, tetapi menurut saya itu jauh lebih kuat: 

:tabnew | args views/* | vertical all\n

Ini membuka tab baru, lalu memuat semua file di direktori views. Alih-alih meneruskan glob ke args kita dapat menentukan file terpisah. vertical membagi panel menjadi jendela vertikal terpisah, yang saya sukai. all memberitahu itu untuk menggunakan semua file yang diteruskan oleh args.

Ketika Anda mengatakan "dalam satu baris", saya kira Anda tidak hanya berbicara tentang penggunaan karakter batang vertikal |, bukan baris baru. Sebaliknya, saya curiga Anda sedang mencari cara yang lebih singkat untuk melakukan tugas Anda.

\n\n

Anda bisa mendapatkan apa yang Anda inginkan dengan menggabungkan saran yang ada ke dalam perintah yang ditentukan pengguna (lihat help :command). Perintah yang ditentukan pengguna harus dimulai dengan huruf kapital.

\n\n

Misalnya, jika Anda seorang programmer C ++, mungkin Anda ingin membuka tab baru yang memiliki implementasi dan file header secara berdampingan. Pada titik itu Anda dapat menentukan perintah seperti ini di .vimrc Anda: 

:command -nargs=1 -complete=file Tabv tabe <args>H | vs <args>C\n

Yang dapat digunakan sebagai berikut: 

:Tabv Foo\n

untuk membuka Foo.C di kiri dan Foo.H di kanan di tab baru.

\n\n

Jika kasus penggunaan tipikal Anda tidak sespesifik ini, mungkin lebih sulit untuk menyimpan penekanan tombol. Karena Anda menanyakan ini di SU dan bukan SO, mungkin Anda bukan seorang programmer, tetapi mudah-mudahan Anda dapat menyesuaikan saran ini agar sesuai dengan kebutuhan Anda.

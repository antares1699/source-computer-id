Saya perhatikan bahwa titik koma ; adalah karakter yang valid untuk file Windows (NTFS) dan nama direktori.

\n\n

Namun, titik koma digunakan sebagai pemisah elemen untuk variabel lingkungan PATH.

\n\n

Katakanlah saya memiliki folder bernama C:\\ "main() {printf('%d', 42);}" (heh, bagus, juga berisi tanda persen) - bagaimana saya bisa menambahkan folder ini ke sistem PATH?

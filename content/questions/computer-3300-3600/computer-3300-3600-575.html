Itu Vagrantfile skrip upaya Homestead (dengan memanggil scripts / homestead.rb script) untuk membuat mesin dengan nama default 'homestead-7'. Itu sebabnya gagal, jika kotak dengan nama itu sudah ada.

\n\n

Anda memiliki 2 cara untuk menyelesaikan ini:

\n\n
    \n
  1. Buka Oracle VirtualBox dan ganti nama mesin yang ada (melalui pengaturannya) dari 'homestead-7' ke nama lain, lalu jalankan kembali perintah vagrant up.
    2. \n
  2. Letakkan baris name: SomeOtherName (ganti 'SomeOtherName' dengan yang Anda inginkan) di Homestead.yaml file mesin baru, dan skrip akan menggunakan nama itu alih-alih 'homestead-7' (tidak didokumentasikan, tetapi lihat ke dalam scripts / homestead.rb menunjukkan bahwa ia pertama kali memeriksa nama yang ditentukan pengguna dan menggunakannya, jika tidak, ia menggunakan nama 'homestead-7' secara default: config.vm.define settings[\"name\"] ||= \"homestead-7\")
    3. \n

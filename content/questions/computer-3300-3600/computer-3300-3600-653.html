XFCE Thunar File manager memiliki fitur pengubah nama massal yang bagus.

\n\n

Cukup pilih lebih dari satu file dan tekan F2 untuk menampilkan dialog Ubah Nama Beberapa File.\nDari menu drop down pilih opsi Search & Replace, centang kotak [x] Regular Expression dan ketik kriteria Search dan Replace.

\n\n

Misalnya saya punya beberapa file dari grabber DVB-T bernama 

20190813 Okno do New Yorku.avi\n20190814 Okno do Rima.avi\n20190815 Okno do Bankoku.avi\n20190820 Okno do Hamburgu.avi\n

Menggunakan pencarian ([0-9]+) (.*) dan mengganti DOC \\2 \\1 transformasi menciptakan 

DOC Okno do New Yorku 20190813.avi\nDOC Okno do Rima 20190814.avi\nDOC Okno do Bankoku 20190815.avi\nDOC Okno do Hamburgu 20190820.avi\n

yang sangat keren.

\n\n

Pengganti nama massal juga memiliki pratinjau langsung sehingga Anda dapat melihat garis yang cocok dan transformasinya di sebelah nama file sumber yang bagus untuk men-debug regexp. Nama file yang tidak cocok tidak berubah.

\n
    \n
  1. Instal ganti nama utils:
    2. \n
sudo apt install renameutils\n

atau 

sudo pacman -Syu renameutils\n
    \n

  1. Jalankan qmv dengan editor favorit Anda:

    \n

    2.1 Luhur: 

    qmv -f do -e "subl -w" ./PATH/TO/FILES\n

    2.2 Atom: 

    qmv -f do -e "atom -w" ./PATH/TO/FILES\n

    2.3 Gedit: 

    qmv -f do -e "gedit" ./PATH/TO/FILES\n
    2. \n

  2. Gunakan kemampuan regex lengkap editor (termasuk multiselection / dan Ctrl+D) untuk membuat daftar nama baru.

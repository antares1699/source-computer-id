Melakukan sedikit lebih banyak penelitian tentang ini, dan ini tampaknya memang disengaja. Ini sebenarnya tidak repro jika Anda menjalankan Notepad ++ dengan cara biasa, menggunakan notepad++ Dockerfile alih-alih menimpa Image File Execution di registri. Notepad ++ memiliki beberapa kode di convertParamsToNotepadStyle yang pada dasarnya meniru apa yang dilakukan Windows Notepad secara default, yaitu menambahkan ekstensi .txt secara default.

\n\n

Lantas, apa yang menyebabkan Notepad ++ meniru Notepad? Ternyata itu adalah parameter -notepadStyleCmdline, yaitu disertakan dalam instruksi tentang cara mengganti Notepad. Jika Anda menghapus parameter dan mengganti Notepad seperti ini: 

reg add \"HKLM\\Software\\Microsoft\\Windows NT\\CurrentVersion\\Image File Execution Options\\notepad.exe\" /v \"Debugger\" /t REG_SZ /d \"\\\"%ProgramFiles%\\Notepad++\\notepad++.exe\\\" -z\" /f\n

Kemudian Notepad ++ akan berperilaku seperti Notepad ++.

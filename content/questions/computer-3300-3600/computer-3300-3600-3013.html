Saya mencoba menggunakan AutoHotkey untuk menentukan pintasan keyboard sehingga jika saya menekan ini tiga tombol secara bersamaan : Ctrl + Alt + c , AutoHotkey akan menyalin teks yang dipilih tanpa pemformatan ke papan klip.

\n\n

Apakah AutoHotkey mendukung ini? Jika ya, bagaimana caranya? Sebagian besar dari apa yang saya lihat dalam dokumentasi di situs resmi bergantung pada pintasan keyboard yang terbuat dari dua penekanan tombol, misalnya: 

LControl + LAlt:: \n; Do something\n

Apakah ada cara untuk melakukan ini dengan tiga kunci? Juga, bagaimana saya bisa menyalin dan menghapus pemformatan dari teks yang dipilih?

Lihat Bagaimana cara menggabungkan tiga kunci sebagai hotkey dengan Autohotkey? untuk paruh pertama pertanyaan Anda.

\n\n

Sejauh pemformatan pengupasan, tampaknya mungkin dengan sesuatu seperti ini (dari pertanyaan Stack Overflow ini ): 

Send ^c\nclipboard = \"%clipboard%\"\n; Remove space introduced by WORD\nStringReplace, clipboard, clipboard,%A_SPACE%\",\", All\n

Ada juga posting forum AutoHotkey ini tentang itu.

\n\n

EDIT: Berikut ini ada dalam dokumentasi AutoHotkey di Clipboard, ClipboardAll, dan OnClipboardChange :

\n\n
\n

Apakah itu harus menjadi rute AutoHotKey, secara khusus?

\n\n

Untuk hal persis yang Anda gambarkan, saya telah menemukannya PureText bekerja dengan sangat baik di berbagai aplikasi.

\n\n

Ini adalah program kecil, yang baru saja Anda tambahkan ke folder Startup Anda, dan kemudian beri tahu pintasan apa yang akan digunakan. (Dan matikan efek suara yang agak mengganggu, jika Anda mau)

\n\n
\n

Cara termudah untuk menggunakan PureText adalah dengan menggunakan tombol pintasnya untuk menempelkan teks daripada menggunakan tombol pintas CTRL + V standar yang ada di sebagian besar aplikasi Windows. Untuk mengkonfigurasi PureText, klik kanan pada ikon tray-nya dan pilih "Options" dari menu pop-up. Tombol pintas default adalah WINDOWS + V, tetapi ini dapat diubah. Di jendela Opsi ini, Anda juga dapat mengonfigurasi PureText untuk dijalankan setiap kali Anda masuk ke Windows.

\n
\n\n

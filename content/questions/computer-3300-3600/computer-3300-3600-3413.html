Saya telah menggunakan Brave selama beberapa bulan sekarang dan ingin mengubah beberapa pengaturan ekstensi saya.

\n\n

Masalahnya adalah saya tidak dapat menemukan di mana pintasan keyboard ekstensi disetel karena tampaknya tidak memiliki halaman khusus seperti di Google Chrome.

\n\n

Menekan pintasan keyboard yang sama dengan yang saya miliki di Google Chrome membantu saya menggunakan ekstensi saya.

\n\n

Apakah ini berarti saya harus mengubah pintasan keyboard ekstensi Google Chrome saya di Google Chrome untuk melihat perubahan di Brave?

\n\n

Saya tidak mencari cara untuk memodifikasi pintasan keyboard bawaan Brave.

\n\n

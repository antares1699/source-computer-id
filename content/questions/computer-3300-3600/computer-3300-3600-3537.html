\n\n

\n

Perilaku ini umumnya disebabkan oleh SDL, pustaka yang umum digunakan untuk membuat konteks OpenGL untuk game yang akan dirender. Komitmen ini ke libSDL2 pada tahun 2012 menambahkan variabel lingkungan, SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS, yang, seperti namanya, mengontrol apakah game akan meminimalkan dirinya sendiri atau tidak jika kehilangan fokus jendela.

\n

Untuk menghentikan perilaku ini, Anda perlu menyetel variabel lingkungan ini ke 0. Ada beberapa cara untuk melakukannya:

\n
    \n
  • Ubah file lingkungan lokal Anda, yang terletak di suatu tempat di direktori home Anda, mungkin .profile atau .xprofile; .bashrc akan mengaturnya untuk shell Bash Anda tetapi itu mungkin tidak membantu jika Anda masuk melalui xdm atau salah satu dari banyak alternatifnya. Ini akan memengaruhi semua game yang menggunakan SDL dan dijalankan sebagai pengguna Anda.
    • \n
  • Ubah file lingkungan global Anda, biasanya /etc/profile atau /etc/environment. Dibandingkan dengan file lingkungan di direktori home Anda, ini bukan pilihan yang disarankan, tetapi jika Anda satu-satunya pengguna di sistem dan perubahan ke .profile tampaknya tidak berlaku bahkan setelah keluar dan masuk lagi, ini adalah hal kedua untuk dicoba.
    • \n
\n

Untuk kedua opsi di atas, Anda akan menambahkan baris ini: 

    export SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS=0\n
    \n

  • Jika ini adalah gim Steam, Anda dapat menambahkannya ke opsi peluncuran Anda: 

      SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS=0 %command%\n
    • \n

  • Jika ini bukan game Steam, tetapi Anda masih ingin mengubahnya hanya untuk game yang satu ini, Anda dapat membuat file launch_game.sh di direktori yang sama dengan file utama game yang dapat dieksekusi: 

      #!/bin/sh\n  export SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS=0\n  ./SomeGame "$@"\n
    • \n

Jika Anda kesulitan menemukan file variabel lingkungan lokal Anda, ada cara mudah untuk menambahkan variabel baru melalui panel kontrol.

\n

Ikuti langkah-langkah ini dari https://www.java.com/en/download/help/path.html

\n
    \n
  1. Pada Search, cari dan kemudian pilih: System (Control Panel)
    2. \n
  2. Klik tautan pengaturan sistem Advanced.
    3. \n
  3. Klik Environment Variables.
    4. \n
  4. Di bagian System Variables temukan variabel lingkungan PATH\ndan pilih itu.
    5. \n
\n

Sesampai di sana pilih "baru" dan tambahkan variabel SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS dengan nilai 0 dan simpan.

\n

Ini seharusnya menyelesaikan masalah Anda tanpa harus mengedit file secara langsung atau menggunakan baris perintah jika Anda tidak dapat menemukan file tersebut.

\n\n

Sebagai contoh, dan sebagai contoh, inilah Arlequin 3.5, perangkat lunak genetika populasi yang digunakan lab saya dengan biner Linux:

Aplikasi Linux sering kali perlu dikompilasi ulang di bawah OS host-nya, bahkan di Linux. Pengguna program sumber tertutup bergantung pada vendor mereka. Tidak ada sumber? Sedikit harapan.

\n\n

Untuk menjalankan perangkat lunak Linux, taruhan terbaik Anda adalah menginstal Linux ke mesin virtual seperti VirtualBox .

\n\n

Mungkin Anda bisa menggunakan file tiruan open source dari MATLAB atau itu Bahasa pemrograman Statistik GNU R yang telah dikompilasi atau tersedia untuk Mac OS X.

\n\n

Semoga berhasil!

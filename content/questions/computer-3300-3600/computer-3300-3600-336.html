Saat mengakses drive eksternal dengan BitLocker / BitLocker "buka kunci otomatis" diaktifkan, terkadang Windows 10 melaporkan, di kiri atas layar, bahwa "Drive yang dilindungi oleh Bitlocker sudah dibuka kuncinya" tetapi File Explorer tidak mengizinkan akses ke file di drive.

\n\n

Di File Explorer, drive juga muncul dengan ikon "Locked", meskipun faktanya "buka kunci otomatis" diaktifkan dan Windows 10 melaporkan drive tidak terkunci saat Anda mengkliknya.

Saya memiliki pengalaman serupa, di mana drive ditampilkan sebagai terkunci, tetapi memilih drive melaporkan kesalahan "Drive yang dilindungi oleh Bitlocker sudah tidak terkunci". Setelah sedikit percobaan, tampaknya shell Explorer agak bingung dan menunjukkan drive terkunci padahal sebenarnya tidak.

\n\n

Hasil dari kebingungan ini adalah bahwa Explorer menetapkan "Buka Kunci Drive" sebagai tindakan default untuk drive, bukan "Buka" yang biasa; memilih drive dalam daftar folder di sebelah kiri, atau mengklik dua kali, akan mencoba tindakan "Buka Kunci Drive" yang tidak perlu.

\n\n

Saat Explorer dalam status, Anda dapat mengklik kanan drive, dan memilih "Open" dan kemudian dengan senang hati akan menampilkan konten drive (meskipun masih menganggap drive terkunci). Sebagai alternatif, Anda dapat menggunakan Task Manager untuk mematikan explorer.exe dan kemudian memulai instance baru - Explorer yang baru dibuat akan menampilkan drive dengan benar sebagai Unlocked dan memungkinkan Anda untuk berinteraksi dengannya seperti biasa.

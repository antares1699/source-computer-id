Saya pikir yang Anda cari adalah chntpw (sama dengan yang disebutkan dalam pertanyaan Ben, btw). Itu datang sebagai paket (Ubuntu sudo apt-get install chntpw) dengan distro modern dan memungkinkan daftar atau pengeditan registri, mengingat file sarang dari instalasi Windows. File sarang adalah ntuser.dat di direktori profil pengguna serta file di bawah %SystemRoot%\\system32\\config seperti software.

\n\n

Dari halaman man chntpw: 

-e     Registry editor with limited capabilities.\n

Perhatian: perlu diingat bahwa formatnya tidak didokumentasikan secara resmi oleh Microsoft dan oleh karena itu alat pengeditan pihak ketiga apa pun berpotensi merusak sarang yang mereka muat dan manipulasi. Ambil cadangannya!

\n\n
\n\n

Adapun untuk menerapkan file .reg saya rasa Anda tidak akan bisa menguraikannya sendiri. Ini pada dasarnya hanyalah file INI / INF dengan baris pertama khusus dan bagian yang dinamai sesuai kunci dan nilai yang telah di-escape sesuai dengan jenis nilainya. @ adalah nama nilai khusus untuk kunci default. Sisanya terbukti dengan sendirinya dari melihat file tersebut.

\n\n

Python hadir dengan kemampuan bawaan untuk mengurai file INI, sehingga menyediakan titik awal yang baik.

Untuk magang saya ingin menggunakan Linux. Satu-satunya pilihan nyata yang diberikan kepada saya adalah menggunakan Hyper-V untuk menjalankan mesin Linux virtual. Saya berhasil menjalankan Manajer Hyper-V dengan mengaktifkan semua fitur Hyper-V di bawah "Aktifkan atau nonaktifkan fitur Windows".

\n\n

Sayangnya, ketika saya membuka Hyper-V Manager, buka "Connect to Server", pilih "Local computer" dan tekan "OK". Saya mendapatkan pesan kesalahan berikut:

\n\n
\n

Terjadi kesalahan saat mencoba menyambung ke server "nama-pc".\n Periksa apakah layanan Manajemen Mesin Virtual sedang berjalan dan itu\n Anda berwenang untuk terhubung ke server.

\n \n

Hyper-V mengalami kesalahan saat mencoba mengakses objek di komputer\n "pc-name" karena objek tidak ditemukan. Objeknya mungkin saja\n dihapus. Verifikasi bahwa layanan Manajemen Mesin Virtual di\n komputer sedang berjalan.

\n
\n\n

Pada PC Windows 10 ini (Dell Latitude E7270) saya telah diberikan hak superuser dan dapat menjalankan program sebagai admin, dan itu sudah selesai. Selanjutnya, layanan Manajemen Mesin Virtual sedang berjalan - melihat "Layanan". Bisakah kita menyimpulkan bahwa saya entah bagaimana tidak berwenang untuk terhubung ke server? Jika ya, kenapa bisa?

\n\n

Apakah ada yang punya ide bagaimana memperbaiki masalah ini?

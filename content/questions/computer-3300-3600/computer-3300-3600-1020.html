Saya memiliki AMD Phenom II 960T dengan 2 inti tidak terkunci dengan total 6 inti. Saya tidak melakukan overclock sama sekali. Saya punya Pendinginan Arktik ACALP64 Heatsink / Fan terpasang. Saat ini saya menjalankan ESXi 5.0 jadi saya harus masuk ke BIOS untuk membaca suhu CPU, yang saat idle tampaknya berada dalam kisaran 71-74C. Bagi saya, ini cukup tinggi, tetapi saya tidak dapat menemukan kisaran suhu resmi yang menurut AMD dapat digunakan oleh CPU dengan baik.

\n\n

Tampaknya ada banyak pertanyaan tentang superuser dan banyak forum seputar suhu CPU, tetapi tampaknya tidak ada yang memiliki konsensus yang jelas tentang pabrikan kisaran suhu untuk CPU tertentu. Saya sudah mencoba mencari melalui situs AMD tidak berhasil.

\n\n

Pada titik ini, saya bersedia untuk mematikan 2 inti ekstra jika itu membuat panas tetap rendah tetapi sampai saya mendapatkan semacam toleransi atau kisaran suhu, saya tidak tahu apakah CPU rusak atau tidak. Adakah yang bisa menunjuk ke sumber langsung, artikel, FAQ dari AMD yang secara khusus menyatakan kisaran suhu CPU mereka? Atau apakah rentang suhu CPU sangat bervariasi sehingga tidak ada garis dasar yang mungkin? Apakah saya terlalu paranoid tentang ini? Bagi saya, apa pun di atas 65C agak berlebihan dan jika saya berada dalam kisaran rendah-pertengahan 70-an dengan TIDAK ADA VM yang berjalan, apa yang dapat saya harapkan jika saya menjalankan beberapa VM?

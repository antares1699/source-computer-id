Katakanlah saya memiliki banyak file konfigurasi yang ingin saya simpan di tempat lain, tetapi jika saya memindahkannya dan kemudian menautkan direktori, aplikasi melihatnya sebagai symlink dan tidak akan menggunakan file yang ada di dalamnya.

\n\n

Sebaliknya yang ingin saya lakukan adalah mencari tahu perintah yang memungkinkan saya membuat symlink secara batch untuk semua file dalam direktori tertentu. Bisakah ini dilakukan?

","source":"https://superuser.com/questions/254060/in-os-x-how-could-i-create-symlinks-to-everything-within-a-given-directory-rath","author":"Wilco","replies":[],"answers":[{"answer":"

Cukup gunakan wildcard. Seperti ini: 

ln -s /path/to/config/files/* /path/to/symlinks/\n

Meskipun Anda dapat menggunakan jalur relatif untuk direktori tujuan, direktori tempat target tautan berada harus ditentukan dengan jalur absolut.

\n\n

Perhatikan bahwa jika ada direktori di direktori dengan target, itu akan membuat symlink ke sana daripada membuat direktori dengan symlink di dalamnya. Jika ada terlalu banyak (jika ada) untuk dikerjakan secara manual, Anda harus menulis skrip shell untuk melakukannya.

Ini telah terjadi pada saya dua kali: Saya menekan tombol daya di Mac Book Pro saya, memilih tidur, menutupnya, mencabut semuanya, mengonfirmasi bahwa tidak aktif (dengan menekan telinga saya ke sana) dan memasukkannya ke dalam tas saya. Beberapa menit kemudian, laptop bangun dengan sendirinya.

\n\n

Kedua kali saya menangkapnya tepat waktu. Kedua kalinya begitu panas sehingga saya tidak bisa menyentuh beberapa bagian - ia menolak untuk benar-benar bangun, dan layarnya kosong. Memulai kembali itu berhasil.

\n\n

Adakah ide apa yang mungkin terjadi dan / atau bagaimana mencegahnya?

\n\n

Ini dulu terjadi pada saya sepanjang waktu. Ada beberapa penyebab, tetapi biasanya masalahnya adalah Anda membenturkan mac di ransel Anda dan mengira tutupnya terbuka. Itu kemudian bangun penuh, dan tidak akan ditangguhkan lagi sampai tutupnya dibuka dan ditutup.

\n\n

Apple sama sekali tidak berdaya saat Anda memanggil mereka. Jangan ganggu.

\n\n

Menukar papan CPU tidak akan berfungsi. Ini adalah bug perangkat keras dan OS.

\n\n

Inilah cara saya memecahkan masalah: 

 sudo pmset -a lidwake 0\n sudo pmset -a acwake 0\n sudo pmset -a sms 1\n

Yang pertama membuatnya sehingga Mac Anda tidak akan bangun saat penutupnya diangkat; Anda harus menahan tombol shift (atau tombol apa saja).

\n\n

Yang kedua membuat mesin Anda tidak bangun ketika sumber daya berubah dari baterai ke AC atau sebaliknya.

\n\n

Yang ketiga memastikan bahwa sensor gerak tiba-tiba diaktifkan.

\n\n

Ini memecahkan masalah saya. Ini mungkin akan menyelesaikan masalah Anda.

Secara pribadi saya menyerah dan telah belajar untuk menerima Start Menu baru yang telah coba diterapkan oleh Microsoft sejak Vista. Tetapi jika Anda tidak ingin menginstal aplikasi untuk ini, berikut cara melakukannya (semacam):

\n\n

Hasil akhirnya akan seperti ini:

\n\n

\n\n

Ayo bersenang-senang:

\n\n

Pertama, pastikan folder My Documents Anda ditampilkan sebagai Menu di Start Menu

\n\n
    \n
  • Klik kanan Menu Mulai dan Pilih Properti
    • \n
  • Dari Tab Start Menu, klik Sesuaikan ...
    • \n
  • Telusuri Dokumen item dan\nmemeriksa Ditampilkan sebagai Menu . Menerapkan\ndan kembali ke Desktop.
    • \n
\n\n

Sekarang kita akan memasukkan folder Program dalam daftar itu dan menghapus (jika Anda mau) folder Dokumen.

\n\n
    \n
  • Klik Start Menu dan Klik Kanan\nitu Dokumen item di\ndaftar sebelah kanan. Pilih Properti .
    • \n
  • Klik Sertakan Folder ... dan\nnavigasi ke\nC:\\ProgramData\\Microsoft\\Windows\\Start\nMenu\\Programs. Klik Sertakan\nMap .
    • \n
  • Jika mau, sekarang Anda dapat menghapus\ndua folder dokumen dengan memilih\nmereka satu per satu dan mengklik Menghapus . Jangan khawatir. Anda dapat melihat file Pulihkan Default tombol bawah itu\nkotak dialog yang akan meletakkan segalanya\nkembali ke tempatnya jika Anda tidak suka\nsaya t.
    • \n
  • Terima perubahan Anda.
    • \n
\n\n

Satu sentuhan terakhir

\n\n
    \n
  • Klik Kanan Dokumen item di\nStart Menu lagi dan pilih Ganti nama .
    • \n
  • Ubah namanya menjadi Programs.
    • \n
\n\n

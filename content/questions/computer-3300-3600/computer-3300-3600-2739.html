Default Mac OS X di PyCharm adalah -

\n\n

Cmd + Fn + <UP>: Bagian atas dokumen (setara dengan gg in vi / m)

\n\n

Cmd + Fn + <DOWN>: Bagian bawah dokumen (setara dengan G in vi / m)

Anda dapat menyalin skema warna dari defaults.json dengan mengklik Alt pada pengaturan (pengaturan ditemukan di menu dropdown di tab bar).

\n

Salin semua skema warna. Kemudian di file settings.json Anda (klik kiri sederhana pada pengaturan), ganti array "schemas" dengan array yang Anda salin di defaults.json. Anda akan mendapatkan hasil seperti ini: 

"schemes":\n[ \n    {\n        "name": "Campbell",\n        // ...\n    },\n    {\n        "name": "Campbell Powershell",\n        // ...\n    },\n    // ...\n]\n

Kemudian, lebih jauh, Anda harus menemukan objek default profil yang kosong. Buat properti "colorScheme" di sana dan setel nilainya ke nama skema yang Anda inginkan. Anda akan mendapatkan hasil seperti ini: 

"profiles":\n{\n    "defaults":\n    {\n        // Put settings here that you want to apply to all profiles.\n        "colorScheme": "One Half Light"\n    },\n    "list":\n    [ \n        /*...you will see things like cmd, powershell, or ubuntu/WSL profiles here*/\n    ]\n

Dari sana, Anda dapat menambahkan skema baru ke larik skema lalu memilihnya menurut namanya.

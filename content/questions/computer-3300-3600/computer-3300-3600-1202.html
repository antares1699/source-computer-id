Jika Anda menjalankan daemon SSH di rumah, Anda dapat membuat proxy SOCKS, dan mengirimkan lalu lintas Anda melaluinya. Ini juga akan berfungsi untuk program lain, seperti olahpesan cepat (menggunakan pidgin) skype, dan mungkin lainnya.

\n\n

Jika Anda menggunakan SSH (formulir kerja) dari baris perintah, sambungkan menggunakan perintah berikut: 

ssh -D 1080 yourhome.no-ip.com\n

Jika Anda menggunakan Putty atau beberapa klien SSH lain dari tempat kerja, maka Anda harus menggali opsi untuk menyiapkan proxy SOCKS (ini juga dapat disebut proxy Dinamis).

\n\n

Kemudian, atur aplikasi Anda (Firefox, skype, dll) untuk menggunakan proxy SOCKS yang berjalan pada port 1080 di localhost.

\n\n

Perlu diingat, bahwa melakukan ini pasti melanggar AUP perusahaan Anda, dan Anda bisa mendapat masalah (termasuk pemutusan hubungan kerja) karenanya.

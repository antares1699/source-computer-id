Tampaknya Anda dapat menambahkan ampersand ke baris Anda:

\n\n

Windows: 

start \"Title\" /MIN <application>\n

Linux: 

<application> &\n

Hasilnya, Anda akan melihat ID proses aplikasi dan di jendela terminal tempat Anda berada, Anda akan melihat keluaran aplikasi.

Apakah chmod go-rx /usr/bin/passwd Pengguna normal kemudian tidak dapat menjalankan passwd. Jika Anda ingin beberapa pengguna dapat melakukannya, Anda dapat menempatkan mereka dalam grup khusus mungkin.

passwd -n 9999 user akan mencegah user mengubah kata sandinya selama hampir 274 tahun.

\n\n

Jika Anda ingin memiliki user tanpa kata sandi, yang tidak dapat mengubah kata sandinya, buka /etc/shadow sebagai root, temukan baris yang dimulai dengan nama pengguna, dan ubah konten antara titik dua pertama dan kedua menjadi U6aMy0wojraho.

Apakah ada cara untuk mengekstrak pola Layar Mulai Windows 8, untuk menggunakannya sebagai wallpaper mandiri di sistem lain?

\n
\n

Ya ada.

\n

Latar belakang berada di %systemroot%/system32/imageres.dll. Untuk menggunakannya, saya sarankan metode berikut:

\n

Mulailah dengan membuka folder tempat file sumber daya kita berada, dengan menekan Menang R dan memasuki %systemroot%/system32.

\n

Ekstrak sumber daya dengan menggunakan 7-Zip , (klik kanan) pada imageres.dll dan pilih 7-Zip > Open archive.\nDuabelas ke dalam arsip dengan membuka .rsrc/png. Di sinilah Anda akan menemukan latar belakang. Untuk mengekstrak, pilih file yang Anda inginkan dan (klik kiri) tombol Ekstrak dan pilih di mana file yang dipilih harus berakhir, mis. C:\\Windows 8 Start Screen Backgrounds Catatan: Garis miring ke belakang itu penting!

\n

Sumber daya yang diekstrak tidak memiliki ekstensi file, kita perlu menambahkannya dengan mengganti nama file kita. Arahkan ke folder tempat Anda mengekstrak sumber daya, tahan Bergeser dan (klik kanan) di dalam folder, (klik kiri) di Buka jendela perintah di sini .

\n

Di dalam Command Prompt, ketik dan jalankan perintah berikut: REN * *.png

\n

Selamat! Sekarang Anda memiliki semua sumber daya yang Anda inginkan.

\n

catatan: Windows 8.1 menghapus latar belakang layar mulai lama 10000-11904, jika Anda menginginkannya, Anda harus mendapatkan file imageres.dll dari penginstalan Windows 8.0. Latar belakang Windows 8.1 yang baru mengikuti prinsip yang sama untuk ekstraksi, tetapi memiliki nomor 20001-22101 sebagai gantinya.

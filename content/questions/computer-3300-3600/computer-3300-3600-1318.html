Saya relatif baru mengenal git. Adakah yang bisa, tolong, bantu menjalankan git rebase --interactive dengan benar? Ini membuat kesalahan saat saya mencoba menjalankannya.

\n\n

Catatan : Saya tidak ingin menghapus Xcode kecuali tidak bisa dihindari (karena beberapa paket memerlukan alatnya untuk dikompilasi).

\n\n

Inilah kesalahannya:

\n\n

$ git rebase -i\n/Applications/Xcode.app/Contents/Developer/usr/libexec/git-core/git-rebase--interactive: line 234: -w: command not found\nCould not execute editor

\n\n

File ini, tentu saja, ada, adalah skrip shell yang dapat dieksekusi dan tidak memiliki opsi --help.

\n\n

Masalahnya adalah git rebase -i saya tidak diberi tahu editor apa yang harus digunakan. Jadi, itu diperbaiki dengan menetapkan satu.

\n\n

Mari kita klaim bahwa editor keluar adalah vim (atau nano, atau w / e). Lalu baiklah

\n\n

export GIT_EDITOR=vim (global)

\n\n

atau

\n\n

git config --global core.editor \"vim\" (lokal)

\n\n

Akan menyelesaikan masalah.

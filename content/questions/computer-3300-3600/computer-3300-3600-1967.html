Di Windows 7, jika Anda tidak ingin mengedit registri, itu Mulai> Panel Kontrol> Program Default> Mengaitkan jenis file atau protokol dengan program tertentu. (Seringkali membutuhkan waktu lama untuk memuat!) ".Cmd" harus dikaitkan dengan "Windows Command Script". Karena jelas tidak, klik Ubah Program, Jelajahi \\ windows \\ system32 \\ cmd.exe dan pilih, centang kotak untuk "selalu gunakan", dan itu akan memperbaiki pengaitan.

Saya ingin Manajer 7-Zip untuk membuat zip folder wwwroot saya setiap minggu. Apakah mungkin untuk mengotomatiskan ini dengan Penjadwal Tugas Windows 7, dan jika demikian, apa langkah yang tepat untuk melakukan ini?

","source":"https://superuser.com/questions/34820/how-do-i-setup-task-scheduler-in-windows-7-to-compress-a-folder-using-7-zip-file","author":"ymasood","replies":[],"answers":[{"answer":"

Anda perlu membuat file Windows Batch yang benar-benar ZIP, dan kemudian menjadwalkannya di Penjadwal Tugas untuk berjalan pada jadwal mingguan.

\n\n

Sampel baris perintah 7-Zip tersedia sini dan panduan langkah demi langkah untuk Penjadwal Tugas Windows 7 tersedia sini . Terakhir untuk membuat file batch gunakan notepad dan buat file baru dengan ekstensi .BAT sebagai berikut 

\n@echo off\n[7zip command line here]\n

Simpan file di suatu tempat di drive Anda dan arahkan tugas yang dijadwalkan ke sana. Rincian lengkap tentang file batch tersedia sini .

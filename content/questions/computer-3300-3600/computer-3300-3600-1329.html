Saya mencoba menggunakan perintah Git Add untuk menjalankan beberapa file untuk dilakukan nanti. Namun, saya tidak dapat menambahkannya karena Git bash tidak menyukai direktori dengan spasi di dalamnya. Bagaimana cara mengatasi masalah ini?

\n\n

Saya menggunakan Win 7 OS. Perintah saya terlihat seperti di bawah ini: 

git add folder1/folder2/new folder/folder3/myFile.txt\n

Kesalahan yang saya dapatkan adalah sebagai berikut:\nbaru: Tidak ada file atau direktori seperti itu

\n\n

Terima kasih!

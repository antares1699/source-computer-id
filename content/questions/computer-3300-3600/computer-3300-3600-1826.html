Ini sepenuhnya didasarkan pada batesan pendekatan yang sangat baik (yang diotomatiskan oleh makro ini) .

\n\n

Katakanlah kita mulai dengan satu kolom dan, mengikuti saran batesan, menambahkan kolom =RAND():

\n\n

\n\n

Dalam contoh sederhana saya, tabel hanya memiliki 18 entri dan saya ingin memilih 3 sembarangan.

\n\n

Menjalankan makro pendek: 

Sub PickThree()\n    Dim sh As Worksheet, rng As Range\n\n    Set sh = ActiveSheet\n    Set rng = sh.Range(\"A1:B19\")\n\n    If sh.AutoFilterMode = False Then rng.AutoFilter Field:=2\n    If ActiveSheet.FilterMode Then ActiveSheet.ShowAllData\n\n    Calculate\n    rng.AutoFilter Field:=2, Criteria1:=\"3\", Operator:=xlTop10Items\nEnd Sub\n

menghasilkan:

\n\n

\n\n

Setiap kali makro dijalankan kembali, kumpulan baru tiga baris acak dipilih!

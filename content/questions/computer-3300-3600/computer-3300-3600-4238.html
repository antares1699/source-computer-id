Jika memang hanya terjadi pada font tertentu, Anda dapat membuat file .fonts.conf di direktori home Anda dengan cara berikut: 

<fontconfig>\n  <match target=\"font\">\n    <edit name=\"embeddedbitmap\" mode=\"assign\">\n      <bool>false</bool>\n    </edit>\n  </match>\n</fontconfig>\n

Anda mungkin perlu membuat ulang cache font Anda dan / atau memulai ulang X agar perubahan ini diterapkan.

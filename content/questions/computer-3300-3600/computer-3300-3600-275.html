Saya baru saja mengalami masalah ini sendiri! Saya memiliki monitor ganda desktop Windows 7 (satu TV 37 "dan satu monitor 20") dan laptop Windows 7. Saya ingin berbagi keyboard dan mouse desktop dengan laptop. Saya mencoba menggunakan Synergy, tetapi tidak berhasil sama sekali. Kemudian, saya mencoba Input Director (www.inputdirector.com) dan hasilnya bagus.

\n\n

Satu hal yang saya rekomendasikan adalah untuk menghapus Synergy sebelum mencoba menggunakan Input Director. Saya tidak melakukan ini, dan akibatnya keyboard dan mouse saya mengalami masalah besar, bahkan pada komputer utama.

\n\n

Di samping catatan, jika Anda menggunakan Microsoft Touch Mouse, fungsi pengguliran normal akan berfungsi di komputer mana pun tempat kursor berada, tetapi gerakan sentuh dua jari dan tiga jari akan selalu diikuti oleh komputer utama sekalipun kursor ada di desktop budak.

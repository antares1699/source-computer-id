Saya mencoba menginstal GNOME paket ke Ubuntu 14.04 (kesatuan).\nSaya menggunakan kode-kode ini.

\n
\n

sudo apt-get install gdm

\n

sudo apt-get install gnome-shell

\n

sudo apt-get install ubuntu-gnome-desktop

\n

sudo add-apt-repository ppa: gnome3-team / gnome3-staging

\n

sudo apt-get update

\n

sudo apt-get dist-upgrade

\n

sudo apt-get install gnome-weather gnome-music gnome-maps cheese gnome-documents

\n
\n

Itu tidak berhasil, Jadi saya menghapusnya sepenuhnya menggunakan

\n
\n

sudo apt-get remove gdm

\n

sudo apt-get hapus gnome-shell

\n

sudo apt-get hapus ubuntu-gnome-desktop

\n
\n

Dan sekarang saya tidak dapat mengubah wallpaper desktop saya. Klik kanan di desktop tidak berfungsi . Saya pikir itu juga menghapus paket yang diperlukan dari Unity. Bahkan tampilan layar login tidak berubah dari gaya GNOME ke gaya Unity.\nApakah ada cara untuk mendapatkan kembali gaya kesatuan Ubuntu asli?

Coba ini, ini berhasil untuk saya: 

gsettings set org.gnome.settings-daemon.plugins.background active true\n

Namun kasus ini lebih cocok untuk AskUbuntu, dan jawaban yang hampir tidak jelas telah dijawab: Desktop menampilkan latar belakang putih atau hitam, bukan wallpaper (solusi di atas berasal dari sana).

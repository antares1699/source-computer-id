Coba edit file .fonts.conf Anda di bawah direktori home Anda (~/.fonts.conf).

\n\n

Ini bekerja dengan baik untuk saya: 

 <?xml version='1.0'?>\n <!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>\n <fontconfig>\n  <match target=\"font\" >\n   <edit mode=\"assign\" name=\"hinting\">\n    <bool>false</bool>\n   </edit>\n  </match>\n </fontconfig>\n

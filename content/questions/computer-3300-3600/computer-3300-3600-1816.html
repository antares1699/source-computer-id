SMB menggunakan port 445/tcp dan / atau 139/tcp (biasanya satu saja sudah cukup, tetapi terkadang lebih baik meneruskan keduanya). Selain itu, klien Windows SMB memerlukan Ping (ICMP Echo) agar terbuka.

\n\n

Perhatikan, bagaimanapun, bahwa banyak ISP perumahan secara global memblokir koneksi WAN dari / ke port ini, karena banyaknya worm yang menyerang server SMB yang belum ditambal.

\n\n

Perhatikan juga bahwa klien Windows SMB selalu menggunakan port yang sama. Anda tidak dapat meneruskan 4451 / tcp ke komputer A dan 4452 / tcp ke komputer B, karena tidak ada cara untuk menentukan port saat menghubungkan.

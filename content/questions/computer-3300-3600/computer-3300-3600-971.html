OS X terkadang mengabaikan perubahan pada layanan setelah disimpan di Automator, kemungkinan karena cache. Jadi jika layanan dimulai sebagai file teks layanan, kemungkinan perubahan Anda tidak diambil. Untuk menghindari masalah ini, pilih File »Duplikat di Automator dan simpan layanan dengan nama baru, dan coba lagi. Kemudian hapus layanan asli dari ~/Library/Services.

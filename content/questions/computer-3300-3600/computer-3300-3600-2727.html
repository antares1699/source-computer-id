Saya punya Asus N56VZ, dan Sharp Aquos LED HDTV. Saya ingin memproyeksikan game saya ke TV melalui HDMI (agar orang lain dapat menonton), tetapi bermain game di layar yang lebih kecil terasa lebih alami dengan mouse dan keyboard - terutama di game RTS. Jadi, saya ingin menduplikasi tampilan saya.

\n\n

Ketika laptop saya menjalankan Windows 8 (dan driver default ketika masih baru), saya dapat menekan Win + P, pilih duplikat, dan ini akan berhasil.

\n\n

Masalah:

\n\n

Tetapi sejak saya memutakhirkan ke Windows 8.1, itu tidak lagi. Saya juga memperhatikan bahwa sejak peningkatan, Driver Tampilan Intel diinstal, dan tidak lagi dapat melakukan tampilan Duplikat.

\n\n

Saya telah memperbarui Driver Tampilan nVidia saya, tetapi tidak berhasil. Saya dapat memperluas tampilan saya, saya hanya dapat menggunakan tampilan utama (laptop), saya hanya dapat menggunakan tampilan kedua (TV), tetapi tampilan duplikat akan:

\n\n
    \n
  • Gunakan layar utama saya (laptop), tetapi bukan layar kedua saya (TV tidak menerima sinyal).
    • \n
  • Gunakan layar utama saya (laptop), diperlakukan sebagai dua layar (1 | 2), dan perpanjang ke layar kedua (TV), diperlakukan sebagai layar ketiga (3).
    • \n
\n\n

Apa yang saya coba:

\n\n

Sebenarnya, pertama kali saya mencolokkan TV melalui HDMI ke laptop saya, tampilan diduplikasi tanpa saya disuruh. Karena saya tidak sedang bermain apa pun, saya pikir yang terbaik adalah berhenti memproyeksikan ke TV dan mematikannya sampai saya mengatur segalanya.

\n\n

Kemudian itu tidak pernah berhasil lagi.

\n\n

Saya mencoba mengutak-atik Intel Display Driver melalui menu konteks dari desktop, dan saya melihat opsi "klon ke", tetapi sepertinya tidak pernah berhasil, sebaliknya, opsi dua dari atas.

\n\n

Saya juga mencoba mengutak-atik Driver Tampilan nVidia, melalui Panel Kontrol nVidia, tetapi tidak melihat opsi untuk mengkloning tampilan saya.

\n\n

Link ini kata OP menyelesaikannya dengan 'menggabungkan' tampilan menggunakan nVidia, tetapi tidak memberikan instruksi bagaimana melakukannya.

\n\n

Penelitian saya di internet mengatakan sesuatu tentang nVidia Optimus mengacaukan sesuatu, tetapi saya enggan menyentuhnya - bukankah itu bertanggung jawab untuk beralih antara grafis Intel dan nVidia dengan cepat untuk menghemat baterai?

\n\n

Saya akan mengupload gambar jika diperlukan.

\n\n

MEMPERBARUI:

\n\n

Saya mencoba mencopot pemasangan driver Display Intel saya sebagai persiapan untuk penginstalan baru. Wah, apakah itu kesalahan - Saya tidak dapat menginstal apa pun yang telah saya unduh dari situs web mereka (64 bit):

\n\n

Versi 15.33.5.0
\n Versi 15.33.8

\n\n

Keduanya mengembalikan ini:

\n\n

\n\n

Mencoba memperbarui Driver secara manual melalui pengelola perangkat Windows 8 sendiri memberi tahu saya bahwa "Windows telah menentukan perangkat lunak driver adalah yang terbaru." ketika jelas tidak berfungsi sebagaimana mestinya.

\n\n

Aneh, meskipun - sepertinya tidak ada yang berubah meskipun menghapus driver Display Intel. Saya masih dapat melihat menu konteks Intel jika saya mengklik kanan pada desktop.

\n\n

Sekarang, saya benar-benar bingung.

\n\n

Perangkat keras saya:

\n\n

Asus N56VZ, dengan kartu Intel Graphics, dan GeForce GT 650M, menjalankan OS Windows 8.1, diperbarui dari Windows 8 (OS laptop default).

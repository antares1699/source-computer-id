Bagaimana cara mendorong pelengkapan otomatis zsh untuk mengambilnya

\n\n

Gunakan rehash:

\n\n
    \n
  • rehash menghitung ulang daftar hash internal dari isi direktori yang terdaftar di variabel lingkungan jalan untuk memperhitungkan perintah baru yang ditambahkan.
    • \n
\n\n
\n

Ulangi terus-menerus

\n \n

Biasanya, compinit tidak akan secara otomatis menemukan file executable baru di file\n $PATH. Misalnya, setelah Anda menginstal paket baru, file dalam format\n /usr/bin tidak akan langsung atau otomatis disertakan dalam\n penyelesaian. Jadi, untuk memasukkan exectuables baru ini, seseorang akan melakukannya\n Lari: 

$ rehash\n

'Rehash' ini dapat diatur agar terjadi secara otomatis. Cukup sertakan\n mengikuti di zshrc Anda: 

~/.zshrc\nzstyle ':completion:*' rehash true\n
\n\n

Anda tidak akan pernah mendapatkan rendering font yang sama di Linux seperti di Windows karena rendering font Windows (ClearType) sangat aneh, rumit, dan dilindungi oleh paten.

\n\n

Bagi siapa pun yang terbiasa dengan sistem Mac, rendering font windows terlihat salah. Rendering font Mac membuat font menjadi lebih mirip dengan tampilannya saat dicetak. Sistem Windows mencoba meningkatkan keterbacaan dengan mengurangi keburaman dan menyelaraskan semuanya secara sempurna dengan piksel, tetapi hal ini dilakukan dengan mengorbankan keakuratan.

\n\n

Rendering font Linux cenderung sedikit tertinggal, tetapi cenderung lebih mirip dengan Mac OS daripada rendering windows seiring berjalannya waktu. Font itu sendiri sering disediakan oleh Microsoft untuk digunakan pada sistem yang berbeda, hanya renderingnya saja yang berbeda.

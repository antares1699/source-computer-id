Karena kami adalah nama kembar, saya merasa seperti saya benar-benar harus mencoba dan menjawab yang ini.

\n\n

Ini bukan solusi sempurna, tapi saya sarankan menginstal AutoHotKey dan membuat tombol pintas yang mengaktifkan dan menonaktifkan panel navigasi. Pada Windows 8 Hal ini dapat dilakukan dengan tombol Alt,V,N,Space. Pada Win7 seharusnya sangat mirip.

\n\n

Setelah menginstal AutoHotKey, Anda dapat membuat skrip berikut 

F12::\nID := WinExist(\"A\")\nWinGetClass,Class, ahk_id %ID%\nWClasses := \"CabinetWClass ExploreWClass\"\nIfInString, WClasses, %Class%\n{\n    Send !vn{Space}\n}\nreturn\n

Ini memetakan F12 untuk mengaktifkan Bilah Navigasi, hanya saat di Explorer. Di aplikasi lain, F12 akan terus melakukan apa pun yang dilakukannya di aplikasi itu.

\n\n

Jika ternyata tombol pintas di Win7 berbeda maka cukup mudah untuk menemukan dan mengubahnya. Cukup tekan Alt di penjelajah dan lihat apa yang muncul!

\n\n

Ini tentu saja bukan solusi yang sempurna. Saat Anda beralih antar jendela, Anda mungkin perlu terus mengaktifkan dan menonaktifkannya, tetapi ini jauh lebih cepat daripada menjalankan file .bat dari desktop.

\n\n

Alternatif lain, seperti yang disebutkan beberapa orang, adalah menggunakan penjelajah alternatif. Saya tidak bisa bekerja tanpa xplorer2 , Saya sangat merekomendasikannya.

Menurut penelitian saya, tata letak Explorer ditentukan oleh entri registri\nPageSpaceControlSizer ditemukan di:
HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\Modules\\GlobalSettings\\Sizer.

\n\n

Kunci registri ini dikonsultasikan setiap kali jendela Explorer baru dibuka atau kapan pun\nfolder yang ditampilkan berubah.

\n\n

Oleh karena itu, idenya adalah membuat dua file .reg dengan konten registri ini\nmasuk saat panel navigasi aktif atau nonaktif.\nIsinya bisa diekspor dari dalam regedit.\nBerikut ini berasal dari tata letak saya.

\n\n

nav_on.reg 

Windows Registry Editor Version 5.00\n\n[HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\Modules\\GlobalSettings\\Sizer]\n\"PageSpaceControlSizer\"=hex:ce,00,00,00,01,00,00,00,00,00,00,00,10,04,00,00\n

nav_off.reg 

Windows Registry Editor Version 5.00\n\n[HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\Modules\\GlobalSettings\\Sizer]\n\"PageSpaceControlSizer\"=hex:ce,00,00,00,00,00,00,00,00,00,00,00,10,04,00,00\n

Anda kemudian dapat membuat dua file batch (.bat) yang akan menggunakan file perintah reg atau cukup lakukan impor file .reg secara diam-diam: 

regedit /s nav_on.reg\n

Membuat dua pintasan di desktop untuk dua file batch akan memberi Anda\ncara untuk mengaktifkan atau menonaktifkan panel navigasi untuk Explorer berikutnya\njendela atau folder.

\n\n

Jika Anda benar-benar ingin menjadi mewah, Anda juga dapat membuat dua ekstensi shell Explorer\nyang akan menambah ke menu klik kanan opsi "Buka dengan panel navigasi" dan "tanpa".

\n\n

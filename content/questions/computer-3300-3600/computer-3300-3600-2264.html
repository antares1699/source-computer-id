","source":"https://superuser.com/questions/220465/how-can-i-access-dos-from-windows-xp","author":"user48353","replies":[],"answers":[{"answer":"

Windows XP tidak termasuk MS-DOS . Anda bisa menjalankan meniru DOS dalam DOSBox , tetapi program yang berjalan di dalam kotak itu tidak akan memiliki akses ke BIOS.

\n\n

Anda dapat membuat disket boot DOS dari Windows XP, tetapi mungkin tidak dapat mengakses hard disk Anda, jadi tidak ada gunanya jika citra BIOS Anda tidak muat pada disket.

\n\n

Alat standar untuk peningkatan BIOS (atau penurunan, bagaimanapun juga itu adalah hal yang sama) FreeDOS . Ini adalah tiruan DOS gratis, dengan dukungan dasar yang cukup untuk menjalankan perangkat keras modern. Lihat contohnya Membuat stik boot USB FreeDOS untuk flashing BIOS .

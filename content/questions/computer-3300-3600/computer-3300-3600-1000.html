Saya mencoba menginstal program statistik yang membutuhkan GNU Scientific Library (GSL). Saya telah berhasil menginstal GSL melalui perintah yum, tetapi program statistik saya memberikan kesalahan ketika saya mencoba menjalankan make install. Saya rasa ada masalah penautan. Bagi saya ini terlihat seperti Anda telah menginstal binari perpustakaan, tetapi bukan header dan file tambahan lain yang diperlukan untuk "pengembangan" (yang mencakup hanya mengompilasi aplikasi siap pakai yang menggunakan perpustakaan).

\n\n

Cari paket dengan nama seperti gsl-dev, gsl-headers dan sejenisnya. Mereka (-dev dll.) Biasanya berisi file yang diperlukan untuk mengkompilasi perangkat lunak terhadap pustaka yang dimaksud. Instal paket yang sesuai dan coba lagi.

\n\n

Selain itu, cari file INSTALL, BUILD (atau serupa) di pohon kode sumber. Biasanya ada langkah configure yang Anda ikuti dalam kumpulan perintah yang Anda daftar di posting Anda.

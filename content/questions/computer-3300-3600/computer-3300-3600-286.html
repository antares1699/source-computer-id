Ini adalah kotak baru dengan tidak ada yang terpasang selain yang disediakan langsung oleh Dell. Saya memasang Saphire ATI PCI Express 5570 HD. Setelah mencoba menginstal driver 10.4 Catalyst, sistem akan layar biru. Ini layar biru selama fase deteksi perangkat keras dari proses instalasi.     • \n

    • \n
  • Memastikan Windows 7 telah diperbarui sepenuhnya
    • \n
  • Mem-flash BIOS motherboard ke versi saat ini
    • \n
  • Kartu video yang dihapus dan dipasang kembali
    • \n

    • \n
  • Keluaran suplai terverifikasi dengan benar

\n\n

Masalahnya adalah proses penginstalan driver tidak pernah cukup jauh untuk benar-benar menginstal driver. Pusat resolusi di Windows menyediakan solusi untuk menginstal driver 10.4 Catalyst untuk menyelesaikan masalah (yang gagal).

\n\n

Mencari beberapa tampilan alternatif untuk diselesaikan.

Solusi yang berhasil untuk saya berikut:

\n\n

Saya memiliki Sapphire ATI PCI Express 5570 juga dan mengalami BSOD 0x000000CA yang sama persis di Windows 7 ketika mencoba menginstal driver untuknya. Saya mencoba driver Catalyst di situs web Sapphire dan juga di situs web ATI. Saya mencoba versi Catalyst 10.4, 10.5, dan 10.6 dengan masalah yang sama. Saya juga mencoba memperbarui Optiplex (755) BIOS ke A17 dan mencoba berbagai pengaturan BIOS tidak berhasil. BSOD selalu terjadi ketika perangkat lunak Catalyst mencoba meminta kartu video.

\n\n

Pekerjaan saya adalah tidak membiarkan Catalyst menanyakan kartu itu. Saya berharap ini semudah opsi penginstalan untuk menghindari kueri di penginstalan Catalyst, tetapi tentu saja tidak bisa semudah itu. Sebagai gantinya, saya membiarkan perangkat lunak Catalyst mengekstrak dirinya sendiri ke suatu lokasi, menempatkan file driver .inf di lokasi itu, dan secara manual memperbarui driver kartu video (Device Manager-> Perbarui Driver -> "Biarkan saya menentukan driver terbaik ..." -> tentukan lokasi-> dll). Dua driver ATI 5570 yang identik muncul di jendela pemilihan driver, jadi saya memilih yang pertama.

\n\n

Kemudian saya menginstal Pusat Kontrol Catalyst secara manual dengan mencari hanya file instalasi CCC yang dibuka oleh paket ATI (saya menjalankan dua pengaturan, CCC-core saya pikir dan nama lain yang menyerupai CCC-sesuatu-atau-lainnya. Saya mungkin kehilangan beberapa minor komponen tetapi resolusi layar, kedalaman warna, dan fitur CCC semuanya tampak berfungsi dengan baik.

\n\n

Pada dasarnya semuanya bekerja dengan pengecualian beberapa peringatan DLL kartu video jika saya menjalankan tes Skor Sistem di Windows 7, meskipun itu kembali dengan apa yang terasa seperti skor yang sesuai 5,9 untuk kartu ini.

\n\n
\n\n

Sebagai tindak lanjut: setelah melakukan semua langkah tersebut, Penginstal Catalyst berjalan dengan sukses. Namun saya akhirnya harus menghapus file-file berikut untuk membuat beberapa kesalahan sesekali hilang:

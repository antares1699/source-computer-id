Windows 7 menyimpan daftar aplikasi yang baru-baru ini digunakan. Segera setelah Anda mulai menggunakan program lain lebih sering, program itu tidak akan muncul lagi dalam daftar.

\n\n

Untuk menghapusnya selamanya, cukup cari entri Start Menu dan hapus dari daftar. (Anda dapat dengan mudah menjangkau daftar khusus pengguna dengan mengklik kanan "Semua program" dan memilih "Buka", atau "Buka Semua Pengguna" untuk Menu Mulai di seluruh sistem.

\n\n

Sejauh yang saya tahu, Anda tidak dapat mengecualikan satu aplikasi secara spesifik. Namun, Anda dapat menonaktifkan tampilan program yang baru-baru ini digunakan di Start Menu. Untuk melakukan ini:

\n\n
    \n
  1. Klik Mulailah
    2. \n
  2. Klik kanan Semua program dan pilih Properti
    3. \n
  3. Di bawah tab Menu Mulai , hapus centang pada opsi pertama di Pribadi (dan pilihan kedua juga)
    4. \n

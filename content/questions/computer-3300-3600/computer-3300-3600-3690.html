Saya telah menginstal paris-traceroute dan saya tidak ingat apa yang saya lakukan untuk mengganti ping yang dipaketkan dengan Ubuntu Trusty dengan paris-ping, yang membutuhkan root untuk dijalankan. Sekarang saya ingin menyingkirkannya. 

$ ping --version\nversion 1.0\n\n$ whereis ping\nping: /bin/ping /usr/share/man/man8/ping.8.gz\n\n$ ping\n-bash: /usr/local/bin/ping: No such file or directory\n

paris-ping berada di /usr/local/bin/ping. Saya menghapus file itu tetapi sekarang perintah ping masih tidak berjalan dari /bin/ping. Apa yang saya lewatkan di sini?

\n\n

Gunakan perintah itu untuk memuat ulang lokasi biner: 

hash -r\n

Atau cukup mulai ulang cangkangnya.

(Dengan asumsi Windows 7)

\n\n
    \n
  1. Masuk ke Control Panel -> Network and Internet -> Network Connections\n(Namun, pasti ada banyak cara lain untuk sampai ke sana ...)
    2. \n
  2. Di sana, Anda akan melihat semua koneksi yang tersedia di komputer Anda: Ethernet, nirkabel, semua jenis VPN, dll.
    3. \n
  3. Pilih koneksi nirkabel Anda. Klik kanan -> Properties
    4. \n
  4. Buka tab Berbagi.
    5. \n
  5. Centang "Izinkan pengguna jaringan lain untuk terhubung melalui koneksi Internet komputer ini"
    6. \n
  6. EDIT: Lupa satu langkah: pilih koneksi nirkabel Anda dari menu tarik-turun.
    7. \n
  7. Klik OK, keluar dari sana. Harus bekerja!
    8. \n
\n\n

Melakukannya sekali, tetapi sebaliknya, berbagi melalui Ethernet menggunakan koneksi nirkabel ke Internet. Bekerja dengan baik.

\n\n

Ini disebut ICS (Internet Connection Sharing) di bawah Windows, Anda juga dapat memeriksa file bantuan untuk itu.

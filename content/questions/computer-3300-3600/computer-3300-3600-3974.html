Saya sedang menyiapkan proses arsip untuk menyimpan file lama di hard drive eksternal. Komputer yang dimaksud menjalankan Windows 7 Pro 32bit. Kami memiliki folder server dengan 150.000+ file di dalamnya, yang sebagian besar berukuran cukup kecil (di bawah 200k). Saya mencoba menggunakan robocopy dalam file batch untuk melakukan ini. Itu berfungsi dengan baik beberapa hari yang lalu, sekarang yang dilakukannya saat diluncurkan hanyalah duduk di sana. Ini menunjukkan kepada saya semua opsi dan yang lainnya, dan juga mencantumkan jumlah file dalam direktori dan direktori itu sendiri, tetapi tidak pernah melewati baris itu. Jika saya mengalihkan tujuan ke drive C lokal, itu akhirnya mulai menyalin file. Apakah ada sesuatu di file batch saya yang perlu diubah? Atau mungkinkah ada masalah dengan drive Western Digital eksternal yang saya gunakan? Drive WD saat ini menyimpan sekitar 175.000 file. Berikut adalah file batch satu baris yang saya miliki: 

    robocopy \"\\\\cgifp01\\Prepress\\Public\\ImportedPDF\" \"E:\\OldFiles\" *.* /R:2 /W:10 /MINAGE:15 /MOV /B /XJ /XF \"blank_test.pdf\"\n

Terima kasih atas tip atau ide apa pun.

\n\n

Nate

jika tujuan E: \\ OldFiles Anda sudah berisi file, robocopy akan memindai file tersebut terlebih dahulu untuk melihat apa yang harus dilakukan (menimpa dengan yang lebih baru, biarkan, ...) dan kemudian mulai dengan perintah salin.\nJadi, jika Anda memiliki 150000 file di tujuan, itu bisa memakan waktu cukup lama. jika Anda menyalinnya ke E: \\ OldFiles1, ini akan segera dimulai karena belum ada file di sana.

Saya telah menggunakan robocopy selama bertahun-tahun untuk menyalin dari server saya ke drive USB - 64GB, 1TB, 2TB, 3TB. Tetapi sekitar sebulan yang lalu saya melihat bahwa salinan file akan terhenti di beberapa file biasa, misalnya JPG. Jika saya pergi ke direktori target atau direktori lain pada drive target dan membuat file teks baru, salinan akan dilanjutkan, tetapi terhenti beberapa saat kemudian. Mengulangi trik memungkinkan pencadangan dilanjutkan. Mencoba banyak opsi tetapi tidak ada solusi. Sangat frustasi. Win7Pro32bit, Kaspersky Internet Security 13.0.1.4190.

\n\n

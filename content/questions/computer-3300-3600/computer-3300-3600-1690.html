Ketika saya mencoba mencetak PDF, saya mendapatkan kotak pesan yang bertuliskan "dokumen tidak dapat dicetak". Saya kemudian mendapatkan kotak pesan lain yang mengatakan "tidak ada halaman yang dipilih untuk dicetak". Pengirim dapat mencetaknya. Pengguna lain di komputer yang sama tidak dapat mencetak dokumen mereka tetapi saya dapat mencetak dari Word.

Dengan file PDF yang ditargetkan dibuka di Adobe Reader, klik "cetak". Di jendela dialog printer, saya mencantumkan "Adobe PDF" sebagai pilihan yang memungkinkan untuk printer yang tersedia. Memilih itu dan mengklik cetak akan memunculkan jendela lain untuk memungkinkan Anda mengetikkan nama file dan lokasi untuk file PDF baru yang akan disimpan. Saya kemudian dapat mencetak file PDF yang baru dibuat tanpa masalah.

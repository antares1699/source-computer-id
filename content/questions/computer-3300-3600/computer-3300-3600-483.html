Saya tahu pertanyaan ini agak lama tetapi Anda sebenarnya dapat mengubah folder instalasi default.
Itu tidak akan memindahkan program yang saat ini Anda instal ke tempat itu tetapi menginstal wizard yang Anda gunakan di masa mendatang akan menggunakan nilai-nilai ini sebagai jalur default mereka.

\n\n

Kunci untuk folder default disimpan dalam registri di

\n\n

HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion

\n\n

Anda akan dapat melihat dua subkunci
\n ProgramFilesDir
\n ProgramFilesDir (x86)

\n\n

Yang pertama adalah untuk aplikasi 64 bit Anda. (C: \\ Program Files)
Yang kedua untuk aplikasi 32 bit Anda. (C: \\ Program Files (x86))
Saya tidak berpikir bahwa mereka harus berbeda.

\n\n

Perlu diketahui bahwa Anda masih memerlukan C: \\ Program Files \\ Folder dan Anda tidak bisa begitu saja menyalin dan men-baste dari C: \\ ke D: \\
Untuk memindahkan perangkat lunak antara Folder lama dan yang baru, Anda perlu menghapus penginstalan perangkat lunak dan menginstalnya kembali di jalur yang baru.
Beberapa program tidak akan menanyakan di mana harus menginstal sendiri dan mereka mungkin akan tetap menggunakan C: \\ Program Files \\

\n\n

Saya mengubah pengaturan ini beberapa saat dan berfungsi dengan baik.
Ada beberapa pemasang yang tidak peduli dengan entri registri tetapi pemasang yang paling umum menggunakan nilai-nilai ini sebagai defaultnya.

\n\n

Bagaimana cara mengubah nilai-nilai ini?

\n\n

Metode 1
Anda dapat mengubah nilai-nilai ini dengan membuka regedit.exe melalui menu start atau cmd dan menelusuri entri. Bagaimana saya melakukan ini?

\n\n

Metode 2
Dan inilah beberapa promosi diri:
Saya menulis program c # open source kecil untuk mengubah dua nilai registri ini
\n Lihat di GitHub (unduhan biner tersedia)

\n\n

Metode 3
Anda juga dapat mengunduh file .reg yang akan mengubah entri untuk Anda. Unduh file .reg dan ubah nilainya di Editor Teks (seperti Notepad ++). Kemudian klik saja dan Pekerjaan selesai.
\n Unduh file .reg

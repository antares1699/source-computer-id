sudo apt-get install cpufrequtils sysfsutils\n

kemudian 

echo performance | sudo tee /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor\n

dan ulangi untuk setiap core / CPU yang Anda miliki.

\n\n

Anda dapat memverifikasi hasilnya dengan cpufreq-info.

\n\n

Untuk membuat perubahan permanen: tambahkan baris berikut (atau baris - untuk beberapa core / CPU) 

devices/system/cpu/cpu0/cpufreq/scaling_governor = performance\n

ke /etc/sysfs.conf (atau edit baris yang sudah ada).

\n\n

Anda mungkin juga melihatnya posting ini .

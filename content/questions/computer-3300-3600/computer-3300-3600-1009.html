Saya ingin trailing whitespace ditampilkan sebagai ., dan tab ditampilkan sebagai spasi kosong, bukan ^I. Saya pikir solusinya akan mudah: 

set listchars=tab:  ,trail:.\nset list\n

tapi itu tidak cukup berhasil: 

Error detected while processing .vimrc:\nline 11:\nE474: Invalid argument: listchars=tab:\n

Saya kira saya harus melarikan diri atau menyandikan spasi putih entah bagaimana, tapi bagaimana caranya?

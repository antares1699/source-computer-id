\n\n

Untuk mengubah jenis file dan asosiasi program: Sumber dengan gambar Sini

\n\n
    \n

  1. Pergi ke [Start], lalu tekan tombol [Default Programs]:

    2. \n

  2. Di jendela yang terbuka, pilih tautan [Mengaitkan tipe file atau protokol dengan program]:

    3. \n

  3. Mungkin diperlukan beberapa saat saat sistem memindai PC untuk semua jenis file dan program yang tersedia:

    4. \n

  4. Setelah pemindaian dilakukan, daftar semua jenis file dan program yang didukung dan tersedia akan ditampilkan:

    5. \n

  5. Pilih jenis file yang Anda perlukan untuk dikaitkan dengan program lain (misalnya, .GIF) dan tekan tombol [Ubah Program] untuk mengubah program default:

    6. \n

  6. Jendela baru dengan program yang tersedia muncul, pilih yang paling sesuai:

    7. \n

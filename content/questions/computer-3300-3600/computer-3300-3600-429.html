Jika panjang string Anda tidak sama, Anda dapat menggunakan gabungan perintah mid() dan len().

\n\n

Dalam contoh Anda, perintahnya adalah:\n=MID(A1,2,LEN(A1))

\n\n

Seret ke bawah seperti pada jawaban di atas.

\n\n

Bagaimana cara kerjanya?

\n\n

MID mengembalikan sub-string dari string yang ditentukan sebagai: MID([string], [start index], [end index]). Jika [end index] bervariasi, menggunakan LEN(A1) menentukan panjang string tertentu.

Saya perlu menjalankan batch Windows (atau perangkat lunak baris perintah lainnya akan baik-baik saja) yang akan menjalankan program saat komputer menganggur selama satu menit dan menghentikannya saat digunakan. Saya jelas harus memulainya kembali saat tidak digunakan lagi.

\n\n

Ada ide? Tidak dapat menemukan apa pun yang tidak menggunakan GUI.

Ini adalah pekerjaan yang sempurna untuk AutoIt: http://autoitscript.com

\n\n

Ini skrip yang aku buat untukmu. Taruh di file .au3, ganti notepad dengan exe Anda, dan untuk Run, sertakan path lengkapnya: 

#include <Timers.au3>\nWhile 1\n   Sleep(10)\n   $idleTimer = _Timer_GetIdleTime()\n   If $idleTimer > 60000 And Not ProcessExists(\"notepad.exe\") Then\n      Run(\"notepad.exe\")\n   ElseIf $idleTimer < 10 Then\n      ProcessClose(\"notepad.exe\")\n   EndIf\nWEnd\n

Jika Anda tidak ingin menggunakan AutoIt, lihat file quser perintah yang menampilkan, antara lain, waktu idle hingga menit terdekat: 

C:\\Users\\Richard>quser\n USERNAME              SESSIONNAME        ID  STATE   IDLE TIME  LOGON TIME\n>richard               console             1  Active      none   26/06/2014 15:29\nC:\\Users\\Richard>\n

Salah satu cara sederhana adalah menguji keluaran dari:

\n\n

quser | findstr /I %USERNAME% | findstr \"none\"

\n\n

dan jika kembali kosong, maka mereka menganggur lebih dari 1 menit.

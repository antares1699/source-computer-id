Saya menggunakan Windows 7 "Problem Steps Recorder" melalui remote desktop untuk mencatat bagaimana beberapa orang melakukan pekerjaan mereka. Dan ketika itu selesai saya ngeri melihat bahwa tangkapan layar belum diambil ... beberapa ada tetapi setidaknya setengahnya hilang dan diganti dengan kotak yang bertuliskan:

\n\n

No screenshots were saved for this problem step

\n\n

Saya telah menggunakan alat ini sebelumnya dan tidak pernah mengalami masalah dengannya (meskipun memang ini mungkin pertama kalinya saya menggunakannya saat melakukan remote melalui RDP).

\n\n

Saya ingin menggunakan alat ini pada mesin orang yang menunjukkan pekerjaan mereka, tetapi semua mesin itu menjalankan Windows XP.

\n\n

Apa yang menyebabkan ini? Apakah karena saya menjalankan desktop jarak jauh? Mungkin langkahnya terlalu banyak? Ada 323 anak tangga.

\n\n

Ada kabar baik dan kabar buruk.

\n\n

Kabar baiknya adalah Anda dapat meningkatkan jumlah tangkapan layar yang diambil dari 25 tangkapan layar.

\n\n

Kabar buruknya adalah ada batasan jumlah tangkapan layar dan semuanya tergantung pada versi Windows yang Anda gunakan.

\n\n

· Jika menggunakan Windows 7 / Windows 2008, screenshot dapat ditingkatkan dari 25 menjadi 99.

\n\n

· Jika menggunakan tangkapan layar Windows 8 / Windows 2012 dapat ditingkatkan dari 25 menjadi 999!

\n\n

Kelemahan lain dari menggunakan PSR adalah ia tidak memberi tahu Anda kapan Anda mencapai jumlah tangkapan layar maksimum, jadi ingatlah ini.

C:\\Program Files (x86)\\Foxit Software>takeown /f \"Foxit Reader\"\nERROR: Access is denied.\n\nC:\\Program Files (x86)\\Foxit Software>takeown /a /f \"Foxit Reader\"\nERROR: Access is denied.\n\nC:\\Program Files (x86)\\Foxit Software>icacls \"Foxit Reader\" /setowner Administrator\nFoxit Reader: Access is denied.\nSuccessfully processed 0 files; Failed processing 1 files\n

Saya tidak tahu mengapa Pusat Kontrol Brother berjalan - Saya tidak menjalankannya, tidak memulai saat boot - tetapi memiliki kunci. Saya menghentikan proses dan direktori menghilang.

Saya memiliki tumpukan XAMPP lokal yang berjalan di Macbook Pro saya selama berminggu-minggu (OS X Yosemite 10.10.3, XAMPP 5.5.24). Kemudian, saya menghabiskan beberapa waktu mencoba menginstal perpustakaan, di mana saya entah bagaimana mengacaukan instalasi XAMPP. Sekarang Apache tidak bisa dijalankan sama sekali. Inilah yang saya coba.

\n\n

Jalankan Apache 

$ sudo /Applications/XAMPP/xamppfiles/xampp startapache\nXAMPP: Starting Apache...fail.\nXAMPP:  Another web server is already running.\n\n$ sudo /Applications/XAMPP/xamppfiles/xampp stopapache\nXAMPP: Stopping Apache...not running.\n

Jadi saya mencoba mencari web server mana yang sudah berjalan.

\n\n

Pendekatan zombie

\n\n

Saya memeriksa Apache zombie . 

$ sudo lsof | grep AMP | grep apache\n$ ps -x | grep apache\n 7980 ttys000    0:00.00 grep apache\n$ ps -x | grep AMP\n 7990 ttys000    0:00.00 grep AMP\n

Jadi saya tidak melihat server web berjalan.

\n\n

Saya sedang mencari solusi untuk ini (saya menggunakan Python 3.6.1). Menemukan halaman ini:

\n\n

http://www.lfd.uci.edu/~gohlke/pythonlibs/#twisted

\n\n

Ini berisi binari windows untuk sejumlah besar paket python. Untuk python 3.5 gunakan versi dengan 'cp35' di nama filenya, dan pilih win32 / amd64 berdasarkan instalasi Anda. Anda dapat menginstal file .whl dengan mudah menggunakan pip: 

pip install Twisted‑17.1.0‑cp35‑cp35m‑win_amd64.whl\n

Ini memecahkan masalah saya.

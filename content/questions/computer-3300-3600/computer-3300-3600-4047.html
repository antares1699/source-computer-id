Semua paket Python tidak dihosting di pypi.python.org, tetapi easy_install akan melihat halaman PyPi untuk link download. Banyak paket umum seperti PIL dan lxml menggunakan server distribusinya sendiri (yang sebenarnya sering menimbulkan masalah bagi konsumen paket). Contoh: http://pypi.python.org/pypi/PIL/

\n\n

pypi.python.org sendiri tampaknya tidak menawarkan dukungan HTTPS dalam bentuk apa pun.

\n\n

Jika Anda ingin menyediakan lingkungan easy_install / pip yang aman, saya sarankan Anda mencerminkan paket yang diperlukan ke server tempat Anda mengelola HTTPS sendiri dan kemudian membatasi unduhan ke server ini menggunakan opsi --allow-hosts:

\n\n

http://packages.python.org/distribute/easy_install.html#restricting-downloads-with-allow-hosts

Mode tidur dapat menggunakan berbagai nama, termasuk Stand By (untuk Microsoft Windows 98-Server 2003), Sleep (untuk Mac OS 8-Mac OS X, Windows Vista, Windows 7, Windows Server 2008), dan Suspend (Windows 95, Linux). Saat ditempatkan dalam mode tidur ini, selain dari RAM, yang diperlukan untuk memulihkan status mesin, komputer mencoba memutus daya ke semua bagian mesin yang tidak diperlukan. Karena penghematan daya yang besar, sebagian besar laptop secara otomatis memasuki mode ini saat komputer berjalan dengan baterai dan penutupnya ditutup.

\n\n

Komputer harus mengonsumsi daya saat tidur untuk menjalankan sirkuit yang diperlukan untuk mengenali dan bertindak pada acara pengaktifan; komputer tidur adalah kasus perangkat yang memakan daya siaga.

\n\n
\n

Tidur adalah kondisi hemat daya yang memungkinkan komputer dengan cepat melanjutkan operasi daya penuh (biasanya dalam beberapa detik) saat Anda ingin mulai bekerja kembali. Menyetel komputer Anda ke mode tidur seperti menjeda pemutar DVD — komputer segera menghentikan pekerjaannya dan siap untuk memulai kembali saat Anda ingin melanjutkan pekerjaan.

Ketika komputer tidur, semua proses ditangguhkan, dan status OS serta semua proses disimpan di Random Access Memory (RAM) PC Anda. Karena RAM sangat cepat diakses, komputer dapat melanjutkan dari mode tidur dengan sangat cepat. Namun, RAM membutuhkan daya yang konstan untuk memelihara datanya, sehingga komputer dalam mode tidur masih menggunakan daya, tetapi tidak sebanyak saat dijalankan. Jelas, karena OS ditangguhkan, yang lainnya juga ditangguhkan, seperti layar, periferal, dan sebagian besar perangkat input (Windows mengontrol perangkat mana yang masih dapat beroperasi saat dalam mode tidur).
Sebagai perbandingan, hibernasi melakukan hal yang serupa, tetapi menyimpan semua data ini di Hard Drive, yang membutuhkan waktu lebih lama untuk membaca semua data dalam kasus resume, tetapi tidak memerlukan daya untuk memelihara, sehingga komputer mati secara efektif saat hibernasi. .

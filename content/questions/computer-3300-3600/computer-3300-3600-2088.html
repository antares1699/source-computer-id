Nah, ternyata Anda tidak bisa. Aneh, tapi begitulah. Skrip kecil ini akan melakukan apa yang Anda inginkan (dengan asumsi arsip Anda memiliki ekstensi .7z, jika tidak, ubah ke ekstensi apa pun yang mereka miliki): 

for n in *7z; do 7zr x $n; done\n

Gunakan 'x' daripada 'e' karena 'e' akan mengekstrak semua file ke direktori saat ini sementara 'x' akan mempertahankan jalur aslinya.

\n\n

Untuk membuat hidup Anda lebih mudah, Anda dapat menambahkan fungsi kecil ini ke file ~ / .bashrc Anda: 

function multi7z () {\n    for n in $@; do 7zr x $n; done\n}\n

Anda kemudian dapat membuka kompresi beberapa arsip seperti yang diharapkan: 

$ multi7z *7z\n

atau 

$ multi7z a.7z b.7z n.7z\n

EDIT: Ini akan berfungsi jika Anda membuka beberapa kompresi, independen arsip. Jika Anda memiliki satu arsip dibagi menjadi banyak file .7z, gunakan jawaban @ don_crissti sebagai gantinya.

Jelas saya tidak tahu program apa kamu digunakan (apakah itu File Scavenger ?), tetapi selain dari program pemulihan data yang tak terhitung jumlahnya (misalnya, Recuva ), Saya dapat menyarankan untuk menggunakan hex-editor seperti HxD atau editor disk seperti DiskExpl untuk mencari drive. Anda dapat membuka drive (coba buka volume logis terlebih dahulu dan gunakan fisik hanya jika diperlukan; juga pilih hanya baca) lalu gunakan Ctrl + F untuk mencari string Anda. Setelah Anda menemukan file Anda, ekspor atau salin sektor ke file dan potong / gabungkan sesuai kebutuhan.

\n\n

Photorec memiliki opsi untuk mencari hanya cluster yang tidak digunakan, namun tidak memiliki opsi untuk mencari string tertentu (dan tidak memiliki cara yang layak untuk mencari teks biasa karena tidak ada tanda tangan unik untuk mereka).

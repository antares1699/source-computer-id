Saya memiliki pengaturan PC media / game di ruang tamu saya, dengan dua akun; milik saya, administrator yang dilindungi kata sandi, dan pengguna biasa, yang disebut "Steam".

\n\n

Karena saya menginstal Windows pada SSD 120GB, saya memutuskan untuk meletakkan folder akun saya di drive lain. Ini terdiri dari apa saja mulai dari media, gambar, hingga file terkait pekerjaan dan dokumen pribadi.

\n\n

Saya memiliki akun "Steam" yang disiapkan dengan cara yang sama.

\n\n

Masalahnya adalah akun Steam dapat mengakses file dan folder saya, dan tidak meminta kata sandi akun saya, sejauh yang saya tahu.

\n\n

Idealnya, pengguna Steam akan memiliki akses hanya-baca ke video dan gambar saya, sementara tidak memiliki akses nol ke file dan folder saya. Folder tersebut tidak mencentang opsi "Bagikan".

\n\n

Bagaimana saya melakukan ini?

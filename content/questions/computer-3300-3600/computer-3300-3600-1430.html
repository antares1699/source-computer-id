Sedikit tambahan, jika Anda memiliki mobo yang mendukung RAID (memiliki pengontrol RAID), Anda dapat mengimplementasikan RAID dan tetap menjalankan sistem Anda dalam mode AHCI. Begini caranya:

\n\n

Atur BIOS ke mode SATA RAID, simpan dan mulai ulang. Saat reboot, Control + I untuk masuk ke pengaturan Intel RAID. Buat Array RAID Anda. Keluar dan reboot.

\n\n

Setelah reboot, masuk ke bios dan ubah kembali ke mode SATA AHCI. Simpan dan mulai ulang.

\n\n

Pada titik ini, jika tidak ada OS yang diinstal, saat drive Anda muncul di layar Penginstalan Windows, Anda dapat menelusuri dan menginstal driver pra-instal RAID / AHCI dan menginstal OS Anda pada larik RAID jika diinginkan.

\n\n

Jika Anda hanya menggunakannya untuk penyimpanan, saat Windows melakukan boot, instal IRST jika Anda belum melakukannya. Anda kemudian harus dapat melihat array RAID Anda dan membuat volume di atasnya.

\n\n

Bersulang.

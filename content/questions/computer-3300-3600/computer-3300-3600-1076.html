telnet microsoft.com 80 \n

Menghubungkan ke microsoft.com muncul dan kemudian jendela telnet terbuka, tetapi tidak ada apa-apa di dalamnya.

\n\n

Jika saya mengetik perintah apa pun seperti GET / HTTP/1.1, itu juga tidak muncul.

\n\n

Apakah langkah awal saya sudah benar?

Telnet tidak memiliki gaung secara default. Memasukkan telnet dalam mode interaktif (tanpa argumen, cukup jalankan telnet ). Lalu ketik atur localecho . Kemudian buka port host .

\n\n

Anda mencoba menyambung ke server HTTP, itulah yang paling sering ada di port 80. Protokol HTTP mengharuskan tamu mulai berbicara terlebih dahulu. Anda mungkin ingin membaca tentang HTTP jika Anda ingin menjelajahi: http://en.wikipedia.org/wiki/Hypertext_Transfer_Protocol

\n\n

Tetapi menghubungkan ke port 80 tidak akan memberi Anda shell, server telnet biasanya beroperasi pada port 23. Tetapi telnet tidak benar-benar digunakan melalui Internet lagi karena tidak aman.

Saya menggunakan Windows Vista di tempat kerja (edisi Bisnis). Biasanya jika saya membuka Explorer dalam tampilan detail saya melihat kolom Nama, Tanggal diubah, Jenis dan Ukuran. Terkadang saya akan membuka folder yang berisi gambar dan kolomnya adalah Nama, Tanggal pengambilan, Tag, Ukuran dan Peringkat.

\n\n

Saya merasa ini sangat menjengkelkan. Bisakah saya menghentikan Windows melakukan ini?

Setelah menyesuaikan satu folder sesuai keinginan Anda, cukup lakukan ini: Penjelajah Windows => Mengatur => Folder dan opsi pencarian (kalau tidak Alat => Pilihan folder ) => Melihat Tab => Di bawah Tampilan folder klik " Terapkan ke Folder ".

\n\n

Dengan asumsi Anda melakukan itu untuk gambar, itu akan menggunakan penyesuaian ini untuk semua folder lain yang menggunakan jenis folder gambar. Anda juga dapat melakukan hal yang sama untuk folder dokumen, video, musik.

Anda dapat menerapkan ini ke semua folder dengan memilih semua folder di direktori C: \\ dan melakukan Properties-> Customize -> "Apa jenis folder yang Anda inginkan". Jika Anda memilih "terapkan juga template ini ke semua subfolder", semua folder Anda akan menggunakan gaya itu! Saya baru saja menyetel semua milik saya ke gaya "dokumen". Jika Anda membuat folder baru di direktori C: \\, Anda harus melakukannya lagi untuk folder baru tersebut.

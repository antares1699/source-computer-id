Sesuatu yang membutuhkan sejumlah argumen, di mana setiap argumen akan menjadi jalur ke file atau direktori. Jika pengguna memiliki jalur tertentu, maka pengguna harus memeriksanya untuk melihat apakah jalur tersebut mewakili file normal DAN apakah file tersebut dapat dieksekusi. Jika ya, maka skrip Anda harus mengeksekusi / menjalankan file tersebut.

Tes -O memeriksa apakah pengguna saat ini adalah pemilik file atau folder. 

if [ -O \"$FILENAME\" ]; then echo 'Owned!'; else echo 'Nope!'; fi\n

Lihat daftar operator uji file (dan bagaimana melakukan tes secara umum .)

id -u\n

untuk mendapatkan pemilik file, Anda melakukan: 

stat -c \"%u\" file.name\n

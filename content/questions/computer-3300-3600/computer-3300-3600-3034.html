Saya memiliki konverter USB <-> RS232 tanpa penunjukan produsen di atasnya. Setelah menghubungkannya ke port USB pada komputer Windows Vista 64bit saya memilih opsi untuk secara otomatis mencari dan menginstal driver dari situs Pembaruan Windows. Driver (untuk Konverter Serial USB dan Port Serial USB ) berhasil dipasang. Di Device Manager, di bagian Pengontrol Universal Serial Bus dalam menu konteks untuk USB Serial Converter di bidang Produsen ada FTDI.

\n\n

Bagaimana saya bisa mengetahui jenis chip yang (saya kira itu salah satu yang terdaftar di http://www.ftdichip.com/Products/ICs.htm ) digunakan di konverter saya?

Ada pustaka sumber terbuka - libftdi [1] - yang dapat membaca chip eeprom dan mendekode jenis chip dari nilainya, tetapi ini mengasumsikan tidak ada yang mengacaukan data eeprom.

\n\n

Eeprom memiliki beberapa ID tambahan di dalamnya, dan perpustakaan tahu bagaimana memetakan ID tersebut ke chip yang tepat.
Tentu saja ini mengasumsikan eeprom tidak dimodifikasi setelah rilis pabrik, tetapi untuk kasus umum ini tidak terjadi, jadi mungkin itu cukup untuk Anda.

\n\n

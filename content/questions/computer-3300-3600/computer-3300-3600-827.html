Saya mencoba untuk mem-boot Ubuntu dari drive USB saya. Bagaimana saya tahu nama drive di Ubuntu karena saya belum menginstal Ubuntu?

\n\n

Orang ini menggunakan / dev / sda1 untuk drive USB-nya. Bagaimana dia mengetahuinya?

Saat Anda mem-boot Ubuntu dari Flash drive (tempat diinstal), ini menjadi perangkat USB pertama dan oleh karena itu dinamai sda1.

\n\n

Jika Anda menginstal Ubuntu menggunakan opsi pembukaan LiveCD Ubuntu (gunakan opsi instal daripada yang mengatakan coba LiveCD) ke flash drive yang Anda pasang sebelum mem-boot CD, itu akan terinstal dengan benar. Saat Anda mem-boot dari USB nanti, perangkat akan muncul sebagai '/dev/sda'

\n\n
\n\n

Alasan disk USB menjadi / dev / sda1 adalah karena perangkat penyimpanan berbasis USB, serta perangkat eSATA dikelola oleh subsistem SCSI, bukan subsistem IDE. Ini menjelaskan S dalam SDA. D adalah singkatan dari Disk, dan huruf A untuk disk pertama. Disk kedua adalah B, C, D, dll. Akhirnya, "1" di bagian akhir adalah untuk nomor partisi 1. Ada juga perangkat / dev / sda, yang merupakan keseluruhan disk, bukan satu partisi. Anda akan menggunakannya misalnya untuk mengelola partisi disk. Anda akan memasukkan perintah seperti:

\n\n

sudo fdisk / dev / sda

\n\n

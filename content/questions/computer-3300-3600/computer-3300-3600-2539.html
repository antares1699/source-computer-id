Saya memiliki 3 tata letak keyboard berbeda yang disiapkan di laptop dan saya gunakan Ctrl + Bergeser + [1-3] kombinasi tombol untuk beralih di antara keduanya.\nSeperti itu:

\n\n

\n\n

Namun, setelah mengunci dan membuka kunci Windows (dengan kata sandi), tombol pintas ini tidak berfungsi lagi dan urutan kunci digulung kembali ke (Tidak Ada):

\n\n

\n\n

Keluar dari Windows atau me-reboot laptop membuat semuanya kembali normal, hingga mengunci / membuka kunci lagi.

\n\n

Saya pikir perilaku ini terkait dengan penggunaan hotkey yang sama di File Explorer - untuk beralih tampilan (Ikon ekstra besar / Ikon besar / ikon Medium yang sesuai). Setelah tampilan sakelar kunci / buka kunci mulai berfungsi kembali.

\n\n

Sepertinya File Explorer menimpa tombol pintas bilah bahasa setelah membuka kunci.

\n\n

Bagaimana cara menonaktifkan fungsi ini? Menonaktifkan sepenuhnya Ctrl + Bergeser + jumlah fungsionalitas di File Explorer akan berfungsi, namun saya tidak dapat menemukan di mana pun di internet cara melakukannya.

\n\n

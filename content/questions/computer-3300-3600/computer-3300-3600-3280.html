Mungkin saya sudah ketinggalan zaman, tapi saya ingat bahwa semua lalu lintas Opera Mini melewati server Opera untuk dikompres dan ditransmisikan ke layar kecil. Opera Mobile, di sisi lain, adalah browser yang berdiri sendiri (karena itu "lebih berat").

\n\n

Pihak privasi tampaknya menjadi gelisah tentang Mini (lalu lintas melalui server lain, keamanan kata sandi, dll.).

Jika Mini menarik minat Anda, Anda mungkin juga ingin mencoba Skyfire.

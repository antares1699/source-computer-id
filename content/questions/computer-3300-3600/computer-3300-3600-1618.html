Saya ingin memutakhirkan PC saya ke Windows 7, tetapi sebelum saya membuat komitmen, saya ingin tahu bahwa semua driver saya berfungsi dengan baik terlebih dahulu.

\n\n

Modelnya adalah Acer Veriton 3900Pro. Saya telah menjalankan Upgrade Advisor dan saya diberitahu bahwa semuanya menerima Intel(R) Active Management Technology - SOL kompatibel dalam kategori Perangkat.

\n\n

Saya enggan menginstal Windows 7 karena saya tidak dapat menemukan sumber driver untuk itu di situs web resmi Acer. Saat melihat-lihat driver untuk model saya ( AJAX bodoh ) tidak ada untuk Windows 7.

\n\n

Bisakah komputer saya digunakan dengan Windows 7?

\n\n

Situs Dukungan Acer menyatakan Acer Veriton 3900Pro kompatibel dengan Vista, tetapi tidak dengan Windows 7.

\n\n

Ini tidak berarti bahwa Windows 7 tidak akan berfungsi pada model ini, tetapi hanya opsi tampilan yang paling canggih (seperti Aero) mungkin tidak sepenuhnya tersedia.

\n\n

Jika terjadi masalah, di mana Windows 7 gagal menginstal driver terbaik untuk suatu perangkat, Anda selalu dapat menginstal driver Vista, karena mereka sepenuhnya kompatibel dengan Windows 7.

\n\n

Kesimpulan: Ini seharusnya berfungsi, tetapi selalu ada area abu-abu yang disebabkan oleh fakta bahwa Acer tidak mendukung Windows 7 untuk model ini. Saya akan menyarankan untuk menghubungi Dukungan mereka untuk menanyakan alasannya. Windows 7 akan berfungsi, tetapi sebaiknya bersiaplah untuk beberapa gangguan.

\n\n

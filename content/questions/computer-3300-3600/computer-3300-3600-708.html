Sertifikat sisi server digunakan untuk mengautentikasi dan mengidentifikasi server ke klien, serta untuk mengenkripsi koneksi. Hal ini memungkinkan klien untuk mendapatkan jaminan tertentu saat menghubungkan dan berkomunikasi dengan server. Situs yang membutuhkan keamanan seperti bank, dll. Menggunakannya.

\n\n

Sertifikat sisi klien digunakan untuk mengautentikasi klien ke server. Dengan cara ini server dapat memastikan siapa yang terhubung ke server dengan cara yang hampir sama seperti menggunakan pasangan nama pengguna / kata sandi, tetapi biasanya tanpa memerlukan interaksi dengan pengguna. Mereka digunakan dengan layanan di mana klien harus diidentifikasi tetapi mungkin belum tentu ada seseorang yang memasukkan nama pengguna dan kata sandi, atau semacam itu tidak diinginkan.

Saya telah membuat tautan simbolis seperti ini: 

PS C:\\dev\\bin> cmd /c mklink \"wd\" c:\\dev\\code\nsymbolic link created for wd <<===>> c:\\dev\\code\n

Oke, sederhana. Jadi saya mencoba untuk cd itu: 

PS C:\\dev\\bin> cd wd\nSet-Location : Cannot find path 'wd' because it does not exist.\n\nPS C:\\dev\\bin> cd .\\wd\nSet-Location : Cannot find path '.\\wd' because it does not exist.\n\nPS C:\\dev\\bin> cd \\wd\nSet-Location : Cannot find path 'C:\\wd' because it does not exist.\n

Jauh lebih mudah untuk melakukan ini di Linux. Bagaimana cara melakukan cd ke symlink di PowerShell?

Lihat 'mklink /?' untuk opsi lain.

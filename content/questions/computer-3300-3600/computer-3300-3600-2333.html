Inilah jawabannya (orang-orang, tolong beri suara positif):

\n\n
    \n
  1. Tutup meja jika Anda sudah membukanya
    2. \n
  2. Klik nama tabel di panel Koneksi
    3. \n
  3. Klik tab Data (hanya) dan tidak melakukan klik lain
    4. \n
  4. Tekan Control-A pada keyboard
    5. \n
  5. Tekan Control-C pada keyboard
    6. \n
  6. Tempel data ke aplikasi lain
    7. \n
\n\n

Jika ada klik antara langkah 3 dan 4, Control-A biasanya akan dinonaktifkan. Setelah dinonaktifkan, saya tidak yakin cara mengaktifkan tanpa menutup tabel dan membukanya kembali.

solusi ini akan mempertahankan pengaturan Konfigurasi Kunci Anda yang sebenarnya:

\n\n

1) ikuti menu: Tools -> Preferences -> Key Configuration

\n\n

2) isi item "Edit / Tempel" dengan kombinasi Ctrl + V.

\n\n

Sebenarnya Anda mungkin perlu mengisi juga "Edit / Salin", "Edit / Potong", "Edit / Pilih Semua", dll. Agar lebih nyaman.

\n\n

Jika Anda tidak tertarik untuk menyimpan pengaturan Key Configuration sebelumnya, cukup ikuti Tools -> Preferences -> Key Configuration dan tekan tombol "Reset to Default ..."

Mesin lokal saya memiliki banyak CPU, masing-masing memiliki banyak inti, dan masing-masing dapat mendukung banyak utas. Saya memiliki proses multithreading yang ingin saya paksa berjalan hanya pada satu utas. Saya mengerti numactrl dapat melakukan itu.

\n\n

Bagaimana saya bisa menggunakan numactl untuk memaksa proses berjalan hanya pada satu utas? Atau, apakah ada utilitas bawaan yang lebih sederhana yang dapat melakukan itu?

","source":"https://superuser.com/questions/692138/how-to-force-a-process-to-run-on-a-single-thread-only-with-numactl","author":"Oak","replies":[],"answers":[{"answer":"
numactl --physcpubind=+1 /path/to/your/executable\n

ini harus menjalankan proses Anda pada inti / cpu kedua yang ditetapkan ke chipset Anda (indeks 1).\nlihat bagian Contoh halaman manual untuk lebih jelasnya: http://linux.die.net/man/8/numactl

\n\n

Edit:\nSaya harus menunjukkan, bahwa ini berarti program akan berjalan paling banyak satu utas secara bersamaan, tetapi itu tidak berarti bahwa seluruh proses hanya akan terdiri dari satu utas. jika program ditulis untuk menelurkan utas baru, program akan melakukannya, tetapi akan dijalankan pada inti / cpu yang sama dengan proses lainnya. perbedaan tipis, tapi tetap berpotensi penting.

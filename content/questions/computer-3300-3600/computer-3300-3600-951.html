Ketika saya menjalankan tree /a /f > output.txt di cmd Windows, saya mendapatkan struktur direktori yang diformat dengan baik. Tetapi daftar direktori tidak dalam urutan tertentu. Bagaimana saya bisa mendapatkan pohon untuk mengurutkan folder dan file dalam urutan abjad?

\n\n

Catatan: Saya memerlukan keluaran yang diformat sebagai hierarki, jadi menggunakan dir mungkin tidak akan cukup. Saya mungkin perlu menggunakan tree.

\n\n

Perhatikan bahwa nama folder tidak dalam urutan abjad. Mereka juga tidak dalam urutan tanggal atau ukuran yang dimodifikasi, atau urutan lain yang dapat saya pikirkan.

\n\n

Juga, ini adalah urutan default yang saya dapatkan saat menjalankan dir tanpa menentukan urutan sortir.

Tree adalah program yang sangat sederhana yang melakukan loop melalui semua direktori (dan jika ditentukan file). Itu tidak melakukan buffering. Anda dapat melihat ini karena ia membangun pohon direktori langsung di layar (jika keluaran tidak diarahkan). Dengan banyak direktori, ini lambat dan keluaran segera dimulai.

\n\n

Karena tree tidak membaca semua direktori sekaligus, ia tidak dapat melakukan penyortiran. Jadi itu tergantung pada sistem file untuk menyajikan direktori secara berurutan. Jika filesystem tidak melakukan ini, tree tidak akan dipesan.

\n\n

Jika Anda menginginkan daftar yang dipesan, Anda harus mencari utilitas yang dapat membaca semua direktori dan file sekaligus dan kemudian menyajikan tampilan seperti pohon yang terurut.

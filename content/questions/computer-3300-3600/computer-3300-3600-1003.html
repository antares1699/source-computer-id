Di OS X Lion, ketika saya menekan tombol halaman atas / bawah di browser (Chrome, Safari), halaman tidak langsung melompat seperti yang saya inginkan; alih-alih, laman bergulir dengan mulus ke posisinya. Saya merasa itu mengganggu. Bagaimana saya bisa menonaktifkannya sehingga melompat ketika saya menggunakan tombol halaman atas / bawah, saya tidak melihatnya menggulir?

","source":"https://superuser.com/questions/466039/disabling-smooth-scrolling-in-os-x-lion-when-using-page-up-down","author":"Vultan","replies":[],"answers":[{"answer":"

Lion dan versi sebelumnya dari OS X memiliki opsi untuk menonaktifkan pengguliran mulus di panel preferensi Umum atau Penampilan.

\n\n

Di Mountain Lion, Anda dapat menjalankan defaults write -g NSScrollAnimationEnabled -bool false. Menerapkan perubahan membutuhkan aplikasi yang dibuka kembali.

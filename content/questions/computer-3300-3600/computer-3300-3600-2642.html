Sepertinya solusi terbaik adalah menginstal MPlayer di Mac OS X dan kemudian jalankan skrip Bash baris perintah seperti ini yang ditampilkan di sini : 

find . -type f -name \"*.avi\" -print0 | xargs -0  mplayer -vo dummy -ao dummy -identify 2>/dev/null | perl -nle '/ID_LENGTH=([0-9\\.]+)/ && ($t +=$1) && printf \"%02d:%02d:%02d\\n\",$t/3600,$t/60%60,$t%60' | tail -n 1\n

Seperti yang dinyatakan "syssyphus" pengguna:

\n\n
\n

Ubah * .avi menjadi apa pun yang ingin Anda cocokkan, Anda dapat menghapusnya\n sekaligus jika Anda ingin memeriksa semua file.

# -*- coding: utf-8 -*-\n\n\"\"\"\n\npip install --upgrade setuptools\npip install moviepy\n\nusage：\npython compute_duration.py --path ~/Movies/ --type .mp4\n\"\"\"\n\nimport os\nimport datetime\nimport argparse\nfrom moviepy.editor import VideoFileClip\n\n\ndef main():\n    parser = argparse.ArgumentParser(\n        description='Compute Total Time of a Series of Videos')\n    parser.add_argument(\"--path\", metavar=\"PATH\", default=\".\",\n                        help=\"the root path of the videos(default: .)\")\n    parser.add_argument(\"--type\", metavar=\"TYPE\", default=\".mkv\",\n                        help=\"the type of the videos(default: .mkv)\")\n    args = parser.parse_args()\n    filelist = []\n    for a, b, c in os.walk(args.path):\n        for name in c:\n            fname = os.path.join(a, name)\n            if fname.endswith(args.type):\n                filelist.append(fname)\n    ftime = 0.0\n    for file in sorted(filelist):\n        clip = VideoFileClip(file)\n        print(\"{}: {}\".format(file, clip.duration))\n        ftime += clip.duration\n    print(\"%d seconds: \" % ftime, str(datetime.timedelta(seconds=ftime)))\n\n\nif __name__ == \"__main__\":\n    main()\n\n

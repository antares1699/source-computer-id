Saya memiliki file teks yang sangat besar. File ini memiliki banyak baris dan kebanyakan memiliki lebih dari N karakter sebelum baris akhir.

\n\n

Bagaimana saya dapat menggunakan sed untuk memotong garis dari N th karakter sampai akhir baris?

\n\n

Catatan: Jelas setelah operasi ini panjang baris maksimum adalah N.

","source":"https://superuser.com/questions/529345/remove-characters-from-column-n-until-end-line","author":"masoud","replies":[],"answers":[{"answer":"

Dengan sed Anda bisa melakukannya seperti ini: 

sed 's/\\(.\\{1,80\\}\\).*/\\1/'\n

Untuk memotong menjadi 80 kolom. Ini lebih mudah dibaca dengan ERE (Extended Regular Expressions), diaktifkan dengan -r di GNU sed: 

sed -r 's/(.{1,80}).*/\\1/'\n

Tapi, Anda bisa melakukannya dengan lebih efisien dengan memotong: 

cut -c 1-80\n

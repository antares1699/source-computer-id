@echo off \nsetlocal enabledelayedexpansion\nfor %%a in (*.mp3) do ( \n  set oldName=%%a \n  Set newName=!oldName:~3! \n  Ren \"!oldName!\" \"!newName!\" ) \nexit\n

_Strip 3.cmd adalah nama yang bagus untuk menyimpan ini. Bekerja untuk saya.\nAku memilikinya ( .mp3) agar hanya berfungsi dengan file mp3 ... Anda dapat mengubahnya menjadi ( . *)

\n\n

Ubah ~ 3! ke nomor lain jika Anda ingin ...... ~ 4! akan memotong 4 karakter pertama dari nama file.

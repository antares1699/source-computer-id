Jika Anda hanya ingin melanjutkan dan melihat-lihat perangkat lunak, lihat Bantuan jarak jauh yang ada di dalam Windows - Buka MSN dan minta dia untuk mengklik "minta bantuan jarak jauh" adalah cara tercepat.

\n

Setelah itu, lihat VNC (saya suka TightVNC) dan perangkat lunak akses jarak jauh lainnya.

\n

Selanjutnya, jika Anda masih tidak yakin setelah Anda aktif, tanyakan tentang masalahnya di sini. Mudah-mudahan ini hanya masalah driver, dalam hal ini Anda dapat melakukannya dari perangkat lunak - jika masalah perangkat keras seperti hard drive atau PSU dll, Anda tidak akan dapat membantu tanpa melihat komputer.

\n

Jika ini adalah masalah BIOS, atau Anda memerlukan sedikit kontrol ekstra - Anda mungkin ingin membeli KVM over IP, saya tidak dapat merekomendasikan yang lain tetapi saya dapat mengatakan ini sangat berguna dan menyenangkan untuk memilikinya yang dapat Anda poskan kepada orang-orang dalam situasi seperti ini!

Berikut adalah lima alat desktop jarak jauh terbaik (lihat sumbernya di Lifehacker untuk daftar yang diformat lebih baik):

\n\n

LogMeIn (Windows / Mac)\nLogMeIn adalah salah satu solusi desktop jarak jauh populer pertama yang ditujukan langsung kepada konsumen, menawarkan pengaturan cepat dan mudah untuk mengontrol komputer Anda dari jarak jauh dari kenyamanan browser web mana pun. LogMeIn hadir dalam berbagai rasa, tetapi dua yang dirancang untuk memenuhi kebutuhan desktop jarak jauh Anda adalah LogMeIn Pro dan LogMeIn Free. Akun Pro menambahkan lebih banyak fitur ke layanan, termasuk transfer file seret dan lepas, sinkronisasi file, dan alat rapat. LogMeIn Pro tidak terlalu murah, dengan harga $ 13 / bulan atau $ 70 / tahun, dan sementara akun Pro menawarkan lebih banyak fitur daripada akun gratis, banyak pengguna masih sangat senang dengan LogMeIn Gratis.

\n\n

TightVNC (Windows / Linux)\nTightVNC adalah aplikasi desktop jarak jauh sumber terbuka dan lintas platform. Dengan TightVNC, Anda perlu menyiapkan server VNC di komputer yang ingin Anda akses dari jarak jauh; Anda kemudian dapat mengakses komputer itu dari jarak jauh dari mana saja dengan penampil VNC mana pun. Kami telah merinci cara menyiapkan TightVNC di komputer rumah Anda, dan jika Anda lebih suka mengontrol TightVNC dari browser web daripada membawa klien VNC, Anda juga dapat mengontrol TightVNC dari web.

\n\n

TeamViewer (Windows / Mac)\nTeamViewer — seperti LogMeIn — menawarkan akun gratis dan berbayar untuk mengontrol PC mana pun dari jarak jauh. Tidak seperti LogMeIn, TeamViewer gratis untuk semua pengguna non-komersial. Itu tidak menawarkan remote control berbasis browser, sebagai gantinya menggunakan utilitas kecil untuk menghubungkan antar komputer. TeamViewer bahkan tersedia sebagai aplikasi portabel yang dapat Anda bawa ke mana saja di thumb drive Anda. Apakah Anda ingin mengatur komputasi jarak jauh pribadi atau Anda sering melakukan tugas dukungan teknis, TeamViewer memiliki banyak hal untuk ditawarkan.

\n\n

Koneksi Desktop Jarak Jauh Windows (Windows)\nWindows Remote Desktop — aplikasi desktop jarak jauh default yang disertakan dengan Windows — masih lebih dari cukup untuk sebagian besar pengguna Windows yang mencari kontrol desktop jarak jauh berfitur lengkap. Jika Anda belum pernah menemukan aplikasi Remote Desktop Connection yang terkubur di folder Accessories dari menu Start Anda, sekarang mungkin saat yang tepat untuk mencobanya. Pastikan Anda telah mengaktifkan akses desktop jarak jauh.

\n\n

UltraVNC (Windows)\nUltraVNC adalah aplikasi desktop jarak jauh bersumber terbuka khusus Windows. UltraVNC mendukung serangkaian fitur yang kuat, termasuk obrolan teks, dukungan transfer file, dan dukungan untuk plug-in opsional. Meskipun UltraVNC hanya berjalan di Windows, Anda masih dapat mengakses komputer Anda dari sistem operasi apa pun menggunakan browser web Anda.

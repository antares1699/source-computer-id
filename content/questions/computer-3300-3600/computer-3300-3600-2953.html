Saya memiliki data dalam berbagai sel (katakanlah enam kolom dan seratus baris). Empat kolom pertama berisi data dan kolom keenam memiliki nilai pembatas. Untuk data di setiap baris nilai pembatasnya berbeda. Saya memiliki seratus baris seperti itu. Saya berhasil menggunakan pemformatan bersyarat (misalnya sel yang berisi data kurang dari nilai pembatas di lima kolom pertama dibuat merah) untuk baris pertama. Tetapi bagaimana menyalin pemformatan bersyarat ini sehingga dapat diterapkan untuk seluruh seratus baris dengan nilai pembatas masing-masing.

\n\n

Untuk menambah jawaban mischab1:\nMenggunakan Excel 2007, mendapatkan rumusnya sangat sederhana, tetapi juga mudah terlewatkan:

\n\n

1) Klik "Kelola Aturan"

\n\n

2) Klik "Edit Aturan"

\n\n

3) Dalam aturan tersebut, Anda mungkin memiliki: 

  Format only cells with Cell Value =$A$1  \n

Hapus $ di depan kolom # - sehingga hanya nilai kolom (A) yang absolut: 

  Format only cells with Cell Value =$A1 \n

4) Pilih OK

\n\n

5) Sekarang gunakan pelukis format untuk menerapkan aturan itu ke seluruh lembar

Jika menempelkan format tidak berfungsi, Anda perlu melihat rumus kondisi Anda. Ini mungkin memiliki referensi baris absolut, bukan referensi baris relatif. Anda dapat mengetahui jenis referensi yang Anda miliki dengan mencari $.

\n\n

Jadi, misalnya, jika rumus kondisi Anda adalah =$B$2 > $E$2 maka perlu diubah menjadi =$B2 > $E2. Perhatikan bahwa Anda perlu menyimpan referensi kolom absolut.

\n\n

Untuk info lebih lanjut tentang referensi sel absolut dan relatif, baca ini ozgrid.dll halaman.

Jika Anda menggunakan rumus yang benar, rumus tersebut akan menyesuaikan untuk setiap baris, mis. jika baris pertama adalah 2 dengan data pada A2:E2 dan nilai pembatas pada F2 Anda dapat memilih kisaran A2:E2 dan menggunakan rumus ini dalam pemformatan bersyarat

\n\n

=A2<$F2

\n\n

Perhatikan satu $ dalam rumus

\n\n

Jika Anda menggunakan pelukis format untuk menyalin rumus akan menyesuaikan sesuai kebutuhan (meskipun di Excel 2007 dan versi yang lebih baru Anda tidak akan melihatnya secara eksplisit)

\n\n

Alternatif untuk menggunakan pelukis format adalah dengan memilih seluruh rentang terlebih dahulu, yaitu A2: E100 lalu terapkan rumus yang sama seperti di atas ...... atau di Excel 2007 atau yang lebih baru Anda cukup menyesuaikan "berlaku untuk" rentang dalam pemformatan bersyarat

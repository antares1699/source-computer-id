Firefox selalu menanyakan saya di mana menyimpan file. Saya mengaturnya seperti itu dan saya menginginkannya seperti itu.

\n\n

Namun, itu muncul kotak dialog tambahan yang tidak perlu meminta saya untuk melakukannya dengan file: 'Anda telah memilih untuk membuka file xyx.exe' yang merupakan file biner. Apakah Anda ingin menyimpan file ini ?. Pilihannya adalah 'Simpan' dan 'Batal'.

\n\n

Hanya setelah menampilkan kotak dialog ini, browser mengizinkan saya untuk masuk ke WHERE untuk menyimpan file.

\n\n

Bagaimana cara menghilangkan dialog pertama dan langsung beralih ke dialog kedua (di mana menyimpan)?

Unduh file zip, pilih "simpan sebagai" dan pilih "lakukan ini untuk jenis file seperti ini di masa mendatang". Tutup firefox saat unduhan selesai. Buka C: \\ Users \\ Username \\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles \\ Profilename.default \\ mimeTypes.rdf menggunakan notepad. Ganti semua kemunculan zip dengan exe. Tutup file dan pilih ya saat diminta untuk menyimpan. Buka firefox. Unduh file zip, pilih "simpan sebagai" dan pilih "lakukan ini untuk jenis file seperti ini di masa mendatang". Firefox sekarang akan mengunduh file zip dan file exe tanpa disuruh. Jika Anda ragu-ragu tentang susunan kata yang digunakan oleh firefox di bagian jenis mime untuk file exe, lakukan pencarian lain di file mimeTypes.rdf dan ganti kalimat "Folder Terkompresi (zip)" dengan "File yang dapat dieksekusi".

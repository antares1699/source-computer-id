Karena FCKeditor dapat dibaca sebagai F * ckeditor, mereka memutuskan untuk mengganti nama menjadi CKeditor (FCK adalah singkatan dari nama pembuatnya, Frederico Knabben)

\n\n

Pengembangan berjalan lebih jauh dengan nama "CKEditor", "FCKeditor" disimpan untuk tujuan warisan. Aplikasi baru harus menggunakan CKEditor, itu jauh lebih cepat dan antarmuka terlihat cukup baik.

\n\n

Catatan terakhir: FCKeditor berisi pengunggah / browser file bawaan. Fitur ini telah dihapus dari CKEditor, dan dipisahkan ke CKFinder. Sayangnya, CKFinder tidak gratis dan tanpa membayarnya, Anda hanya dapat menggunakannya dalam "mode demo". Fitur-fiturnya akan tersedia sepenuhnya, tetapi pemberitahuan buruk akan ditampilkan dan aplikasi tidak dapat digunakan untuk aplikasi dunia nyata:

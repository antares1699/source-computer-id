Tidak ada Win 3.1 gratis. Tapi, Anda bisa mendapatkan floppy drive USB cukup murah di ebay.

\n\n

Sekarang yang harus Anda lakukan adalah menemukan mesin MS-DOS untuk menginstalnya - apakah Anda ingat bahwa Win 3.1 membutuhkan MS-DOS? Salin file instalasi Win 3.1 ke CD atau disk lokal terlebih dahulu, karena saya rasa Anda tidak akan beruntung mendapatkan MS-DOS untuk membaca disket USB.

\n\n

Jangan lupa bahwa dukungan OS Windows 32-bit menjalankan aplikasi Win 3.1 - lebih atau kurang. Versi x64 tidak (setidaknya saya pikir dukungan Win16 masih ada dalam versi 32-bit - Anda mungkin perlu memeriksa). Sebaiknya Anda melihat apakah penginstalan Windows biasa Anda akan menjalankan program.

\n\n

Saya benar-benar tidak dapat membayangkan aplikasi Win 3x apa yang ingin Anda jalankan kecuali karena penasaran (tentu saja, mungkin ada lebih banyak alasan untuk ingin menjalankan MS-DOS: http://blogs.msdn.com/virtual_pc_guy/archive/tags/Gaming+under+Virtual+PC/default.aspx ).

\n\n

Akhirnya, jika Anda benar-benar melalui penginstalan yang sebenarnya, saya harap Anda berencana untuk melakukannya di Mesin Virtual - menginstal hal itu di VM akan cukup menyakitkan - saya kira mencoba menginstalnya di sistem modern sepertinya akan sia-sia.

\n\n

\n\n

Akses administrator hanya diperlukan di Windows saat aplikasi ingin mengakses file dan folder sistem.

\n\n

Aplikasi Windows Store dikotak pasir di mana mereka tidak bisa mendapatkan akses ke file sistem atau akses sistem file mentah. Aplikasi Windows Store hanya dapat mengakses perpustakaan dan media yang dapat dilepas dengan sendirinya dan jika ingin mengakses file lain, pengguna harus memilihnya melalui pemilih-buka-file untuk memberikan izin kepada aplikasi untuk melakukannya.

Anda dapat menggunakan kotak pc lama / baru. Menginstal sistem operasi Jika Anda memilih Linux, Anda dapat menginstal zfx-on-linux. Aktifkan SSH dan pertahankan agar tetap dikonfigurasi melalui terminal. Putty bagus untuk ini jika Anda menggunakan jendela. Setelah Anda menginstal sistem operasi Anda, hubungkan semua drive Anda di sana, hard drive lama, dan hard drive ekstra membuat penguji yang baik. Saya suka ZFS, Jika Anda memilih Linux Anda dapat menginstal ZFS terbuka, https://github.com/openzfs/zfs . Jika tidak, jika Anda menginstal Solaris, atau mungkin FreeBSd, Anda dapat menggunakan ZFS secara langsung. Aktifkan NFS atau Samba Share di sistem. tambahkan drive Anda ke dalam array, bahkan satu drive berfungsi, Anda dapat menambahkan lebih banyak nanti. Atur Bios untuk boot saat listrik mati. Colokkan kabel jaringan. Anda sekarang memiliki Server NAS.

\n\n

Catatan, Anda tidak memerlukan ZFS untuk membuat ini berfungsi, itu hanya sebuah contoh. Anda dapat menggunakan server windows dalam hal ini. Tapi biasanya ada banyak bagian pc tambahan yang diletakkan di sekitar, atau tersedia di toko barang bekas yang dapat dengan mudah dibuat menjadi server nas di mana Anda dapat mencolokkan semua drive Anda ke dalamnya.

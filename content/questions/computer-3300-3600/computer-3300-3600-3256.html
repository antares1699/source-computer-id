Tidak! Protokol IRC tidak menggunakan / s. Misalnya, mengubah nama panggilan Anda dilakukan dengan perintah "NICK name", dan ucapan biasa dilakukan dengan perintah "PRIVMSG #channel: text". Tidak.\nItu hanya cara klien IRC berbeda dari teks 'normal'.

\n\n

Anda dapat mengakses semua perintah server yang didukung dengan menggunakan / quote cmd atau / raw cmd

\n\n

Ex: 

/quote whois bgy\n

Itu akan mengirim perintah mentah ke server.

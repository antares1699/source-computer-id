Saya baru saja beralih dari tcsh ke bash, dan saya terbiasa melakukan hal-hal seperti sudo `alias netstat` tetapi karena alias memberikan name=value dalam bash, saya tidak dapat melakukan ini lagi. Apakah ada yang setara dalam bash, jadi saya tidak perlu melakukan sudo `alias netstat | sed -r \"s/.*='(.*)'/\\1/\"`?

","source":"https://superuser.com/questions/373504/get-value-of-an-alias-in-bash","author":"Jayen","replies":[{"reply":"Bungkus dalam sebuah fungsi dan tambahkan ke .bashrc. Masalah terpecahkan.","author":"Daniel Beck♦"}],"answers":[{"answer":"

Anda mencoba untuk memiliki pesta perluas alias setelah sudo. Anda tidak perlu melakukannya dengan cara yang persis sama; sebenarnya, ada cara yang jauh lebih nyaman dalam bash - tambahkan alias untuk sudo yang diakhiri dengan spasi ... 

alias sudo=\"sudo \"\n

... dan sudo netstat akan diperluas secara otomatis.

","votes":"2","source":"https://superuser.com/a/373538","author":"user1686","replies":[{"reply":"luar biasa, dan sudo \\netstat tidak akan memperluas alias netstat!","author":"Jayen"}]},{"answer":"

Bash menyimpan daftar aliasnya dalam array asosiatif BASH_ALIASES. Setara dengan sudo `alias netstat` kemudian sudo ${BASH_ALIASES[netstat]}. Namun, saya akan menyarankan hal berikut, yang bekerja dengan perintah shell bawaan dan menangani kutipan dengan benar: 

sudo bash -c \"${BASH_ALIASES[netstat]}\"\n

Masih banyak yang tidak akan berhasil dengan ini, seperti mis. alias bertingkat.

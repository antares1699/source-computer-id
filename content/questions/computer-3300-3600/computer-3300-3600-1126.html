updatedb/locate tentunya merupakan pilihan yang baik, jika Anda ingin mengindeks nama file. Lihat / upvote jawaban cYrus.

\n\n

Namun, opsi lain adalah dengan menggunakan find untuk membuat daftar nama semua file di hard drive Anda, dan memasukkannya ke dalam file: 

sudo find / > files.list\n

Nanti, Anda dapat mencari file dengan grep: 

grep 'myfavoritesong' files.list\n

Meskipun ini terlihat cukup timpang, sebenarnya cenderung cepat dalam praktiknya, meskipun Anda memiliki banyak file.

\n\n

Tentu saja, itu hampir tidak ada yang lebih baik dari locate. Namun, ini adalah cara yang cepat dan mudah untuk mengindeks hard drive eksternal.

\n\n

Melangkah lebih jauh, Anda mungkin ingin melakukan hash semua file di hard drive Anda: 

sudo find / -type f -exec sha256sum {} \\; > files.sha256\n

Ini mungkin akan memakan waktu lama.

\n\n

Apa yang dilakukannya untuk Anda adalah memberi Anda cara untuk menemukan file yang identik. Katakanlah Anda menemukan file .iso yang sangat besar di komputer Anda: 

$ sha256sum huge.iso\n2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824  huge.iso\n

Anda merasa bahwa file yang sama ada di tempat lain di hard drive Anda, dan Anda ingin menghapus salah satunya untuk menghemat ruang. Anda dapat menelusuri daftar hash untuk menemukannya: 

$ grep 2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824 files.sha256\n2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824  huge.iso\n2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824  archive/path/to/other/huge.iso\n

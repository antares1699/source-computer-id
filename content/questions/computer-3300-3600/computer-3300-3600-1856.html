Ambil kepemilikan dari kunci HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows Defender. Sekarang setel nilai DisableAntiSpyware ke 1.

\n

Ini menghentikan Windows Defender.

\n

\n

Di Windows 10 Build yang lebih baru (1703, 1709), ini tidak berfungsi lagi. Windows mencegah pembuatan kunci di bawah HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows Defender.

\n

Di sini, buka HKEY_LOCAL_MACHINE\\SOFTWARE\\Policies\\Microsoft\\Windows Defender, buat DWORD DisableAntiSpyware dan setel ke 1:

\n

\n

Sekarang reboot dan Defender tidak boleh mulai lagi.

\n

Seseorang dapat menonaktifkan perlindungan Real-time Windows Defender melalui aplikasi Pengaturan,\ntetapi ini hanya sementara, karena Windows akan mengaktifkannya kembali secara otomatis setelahnya\nreboot berikutnya.

\n

Untuk mematikan perlindungan Real-time secara permanen, saya mencantumkan dua metode di bawah ini.\nReboot diperlukan setelah masing-masing.

\n

Metode 1: Menggunakan Editor Kebijakan Grup Lokal

\n

Metode ini membutuhkan Windows 10 versi Pro atau lebih tinggi.

\n
    \n
  • Buka Editor Kebijakan Grup Lokal (gpedit.msc)
    • \n
  • Navigasi ke:
    \n Konfigurasi Komputer / Template Administratif / Komponen Windows / Antivirus Pembela Windows / Perlindungan Real-time
    • \n
  • Klik dua kali Matikan perlindungan real-time
    • \n
  • Pilih Diaktifkan
    • \n
  • Klik baik .
    • \n
\n

Metode 2: Pengeditan registri

\n

Metode ini akan berfungsi untuk semua versi Windows.

\n

Buat dan jalankan file .reg berikut: 

Windows Registry Editor Version 5.00\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Policies\\Microsoft\\Windows Defender\\Real-Time Protection]\n"DisableRealtimeMonitoring"=dword:00000001\n
\n

Mengubah pengaturan ini mengharuskan perlindungan Tamper dimatikan.

\n

Untuk informasi lebih lanjut tentang fitur ini lihat artikel Microsoft
\n Lindungi pengaturan keamanan dengan perlindungan kerusakan .

\n

Untuk metode menonaktifkan fitur ini, lihat artikel
\n Cara Mengaktifkan atau Menonaktifkan Perlindungan Tamper untuk Windows Defender Antivirus di Windows 10 .

\n

Namun, mematikan perlindungan Tamper tetapi tidak melalui aplikasi Pengaturan\nmemerlukan kepemilikan kunci registri\nHKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows Defender\\Features,\nyang mungkin menyebabkan masalah di masa mendatang dengan Pembaruan Windows.

Taruhan terbaik Anda benar-benar membangun kotak OpenSolaris, ZFS benar-benar bersinar untuk aplikasi semacam ini. Anda dapat membangun satu larik Raid-Z dari semua disk Anda, ZFS akan melakukan pemeriksaan CRC pada semua blok disk untuk Anda, Anda dapat mengekspor disk sebagai bagian SMB atau pemasangan NFS atau blok iSCSI.

\n\n

Manajer volume bawaan akan membiarkan Anda mengambil snapshot sistem file untuk tujuan cadangan atau untuk memindahkannya ke host ZFS lain, dan jika Anda memiliki SSD, Anda dapat menggunakannya dengan baik untuk log maksud ZFS - artinya tiba-tiba semua data Anda mengakses akan secepat SSD.

\n\n

Ini memenuhi empat dari lima persyaratan Anda, dan saya tidak akan terkejut jika Anda dapat mengatur entah bagaimana untuk membuatnya tidur dan bangun melalui WOL.

Buat daftar file dan urutkan secara manual Notepad ++

\n \n

Saya perlu MEMESAN (mengurutkan) secara manual beberapa file ini… Saya macet! Siapa saja\n tahu plugin yang bagus untuk itu atau bagaimana cara mengatasinya? Saya perlu memiliki file\n daftar mudah dengan semua file saya dan dapat mengurutkannya (mengubah posisi) seperti saya\n perlu. Misalnya, pindahkan file x ke dekat file y agar tidak terus “melompat”\n dari satu bagian daftar panel ke bagian lainnya.

\n
\n\n

Saya telah menyertakan beberapa metode berbeda untuk membantu Anda secara potensial:

\n\n
    \n

  1. Menyortir semua tab yang terbuka dengan mudah Notepad ++ oleh Name, Path, atau\nType.

    \n\n
      \n
    • Bergantung pada kebutuhan Anda, mengurutkan menurut salah satu bidang ini dapat membantu Anda lebih cepat mengurutkan atau memindahkannya dengan memindahkan tab.
      • \n
    2. \n

  2. Dapatkan daftar lengkap semua tab dengan mudah Notepad ++ tanpa perlu menelusuri tab yang sebenarnya di bilah tab atau panah di samping bilah tab.

    \n\n
      \n
    • Bergantung pada kebutuhan Anda, membuka tab yang diperlukan lebih cepat dengan cara ini dapat membantu Anda lebih cepat menyortir atau memindahkannya dengan memindahkan tabnya.
      • \n
    3. \n
\n\n
\n\n

METODE 1

\n\n

Sepertinya Notepad ++ memiliki tab urutan bawaan dengan nama file dalam urutan naik atau turun.

\n\n

Di bawah ini adalah beberapa tangkapan layar dengan contoh, tetapi berikut adalah langkah-langkah navigasinya:

\n\n

Pilih atas Window | Windows , lalu di Windows opsi jendela, klik pada Name column untuk mengurutkan dalam urutan naik atau turun, lalu klik Sortir tab opsi dan baik setelah selesai.

METODE 2

\n\n

Saat masuk Notepad ++ dan Anda mengklik pada lembar aktif, tekan dan tahan Ctrl lalu tekan Tab untuk menampilkan daftar tab. Pada titik ini pertahankan Ctrl tahan dan tekan Tab namun berkali-kali untuk beralih ke tab berikutnya dalam daftar. Anda juga bisa menggunakan panah bawah daripada Tab setelah Anda menekan Ctrl + Tab sekali dan kemudian gunakan panah bawah daripada Tab untuk beralih melalui daftar tab (lihat di bawah untuk visual).

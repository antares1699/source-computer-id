Windows 10 1607 memperkenalkan kunci registri baru untuk menentukan yang Folder yang Dikenal muncul di bawah Panel Navigasi dan layar "PC Ini".

\n\n

Untuk sebagian besar pengguna, Anda harus membuat kunci registri baru di: 

Computer\\HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\AllowedEnumeration\n

Kemudian di dalam kunci registri (folder) itu Anda dapat menambahkan nilai DWORD baru dengan nama:
{0AC0837C-BBF8-452A-850D-79D08E667CA7} dan nilai 1.

\n\n

Secara default, semuanya akan disembunyikan kecuali untuk Folder yang Diketahui yang Anda tambahkan ke folder registri itu.
Jika Anda ingin membuat folder lain terlihat lagi, lihat ID Folder yang Dikenal artikel dari Microsoft. GUID Folder akan menjadi nama kunci registri dan nilai 1 menunjukkannya, sedangkan 0 menyembunyikannya.

\n\n

Anda harus memulai ulang File Explorer Anda untuk melihat perubahan di panel navigasi.

Perhatikan perbedaan dalam opsi menu konteks saat Anda mengklik kanan PC ini di Panel Navigasi (tangkapan layar Anda) vs. klik kanan di latar belakang panel konten (tangkapan layar Mr. Ethernets). Ini adalah latar belakang Panel Konten yang menawarkan ekstensi Kelompokkan Menurut dan Sortir dengan pilihan.\nSolusi Tn. Eternet menghadirkan Perangkat dan Drive kelompok ke atas. Jika itu cukup untuk kebutuhan Anda, lakukan saja --- ini lebih mudah.\nSolusi Romen, pembuatan AllowedEnumeration key, akan menyembunyikan folder Pengguna di bawah PC ini , tapi juga bersembunyi Perpustakaan , OneDrive , dll. kecuali Anda menambahkan GUID-nya ke daftar folder yang diizinkan: Anda bisa lebih selektif dengan mengatur T hisPCPolicy nilai registri ke Menyembunyikan untuk setiap folder di bawah: 

HKLM\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Explorer\\FolderDescriptions\\<Folder GUID>\\PropertyBag\n

Sedikit ini PowerShell melakukannya untukmu. Ini harus dijalankan dari Admin Jendela konsol PowerShell. IIRC, itu PropertyBag subkunci & nilai untuk Objek 3D tidak ada secara default, jadi kode ini akan membuatnya jika tidak ada: 

$FDPath = 'HKLM:\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Explorer\\FolderDescriptions'\n$RegEx = '3D|My |Local [^A]|Personal|^Downloads|^Desktop'\nGet-ChildItem $FDPath |\nWhere-Object{ (gp $_.PSPath).Name -match $RegEx } |\n   %{\n       $PBPath = join-path $_.PSPath 'PropertyBag'\n       If (!(Test-Path $PBPath)) {\n          New-Item $PBPath | Out-Null\n       }\n       $splat = @{\n           'Path'  = $PBPath\n           'Name'  = 'ThisPCPolicy'\n           'Value' = 'Hide'\n       }\n       Set-ItemProperty @Splat\n   }\n

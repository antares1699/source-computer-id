Bagaimana cara menonaktifkan pembungkusan kata untuk file yang diberikan?

\n\n
\n

Saya sudah mencoba menyetel: \"word_wrap\": false di setelan Spesifik Sintaksis Pengguna, tetapi opsi ini tidak berpengaruh.

\n
\n\n

Menurut beberapa sumber online, seharusnya ada tanda koma , setelah salah.

\n\n
\"word_wrap\": \"false\",\n
\n\n

Anda juga dapat mengatur lebar bungkus menjadi angka besar:

\n\n
\"wrap_width\": 9999,\n
\n\n

Sumber SUBLIME TEXT 3: bagaimana cara menonaktifkan bungkus kata dalam output build? - Dukungan Teknis - Forum Sublime

\n\n

Dan ada juga Word Wrap: Toggle dari palet perintah:

\n\n
\n

Item Word Wrap: Toggle dari Command Palette menggunakan\n toggle_setting perintah. Ini menetapkan preferensi langsung di\n tampilan, yang memang menimpa default, pengguna, atau sintaks tertentu\n preferensi.

\n \n

Anda dapat kembali ke perilaku pengaturan khusus pengguna / sintaks dengan\n menutup dan membuka kembali semua file di mana Word Wrap: Toggle item\n dari Command Palette digunakan, sebagai tampilan pengaturan spesifik akan\n hilang saat file ditutup. Atau, Anda dapat mencapai\n hal yang sama tanpa menutup file apa pun dengan membuka konsol ST Python\n (Lihat menu -> Tampilkan Konsol) dan ketik\n view.settings().erase('word_wrap') Memasukkan . Ini akan jelas\n pengaturan tampilan khusus untuk tab aktif.

\n \n

Anda juga bisa membuatnya untuk menghapus pengaturan dari semua tampilan terbuka di\n jendela saat ini sekaligus, menggunakan [view.settings().erase('word_wrap') for\n view in window.views()] Memasukkan .

\n

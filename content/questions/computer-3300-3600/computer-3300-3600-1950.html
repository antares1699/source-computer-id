Saya perlu kawat gigi untuk menentukan prioritas evaluasi ekspresi, tapi saya tidak ingin kurung kurawal membuat sub-skrip. Lihat apa yang terjadi ketika kode contoh itu dijalankan:

\n\n

Contoh skrip 

#!/bin/bash\n\nfalse || (echo \"First\" && exit 1)\nfalse || (echo \"Second\" && exit 2)\nexit 3\n

Keluaran 

First\nSecond\n

Tampaknya braces membuat sub-skrip, di mana keluar tidak menyebabkan skrip utama keluar.

\n\n

Apa cara paling elegan untuk melakukan apa yang saya inginkan? Lebih disukai tanpa blok dan sarang. Terima kasih!

() ini adalah tanda kurung, dan mereka meluncurkan subkulit

\n\n

{} ini adalah kawat gigi, dan mereka untuk dikelompokkan dalam shell saat ini. exit akan keluar dari skrip Anda.

\n\n

Perhatikan bahwa persyaratan spasi lebih ketat ({ harus diikuti dengan spasi; } harus diawali dengan titik koma plus spasi, atau baris baru) 

false || { echo foo; echo bar; }     # this is OK\nfalse || {echo foo; echo bar}        # not OK\n

Lihat Perintah Pengelompokan di manual untuk detailnya.

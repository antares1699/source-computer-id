Apakah ada add-on untuk Excel untuk memungkinkan saya mengurutkan secara sewenang-wenang\n kolom dengan satu klik?

\n \n

Excel di sisi lain memaksa Anda untuk melalui Sort-> Custom Sort\n jendela untuk menyelesaikan pekerjaan yang sama jauh lebih efisien.

\n
\n\n

Sebenarnya, Anda dapat menyisipkan tabel seperti itu dengan data Anda di Excel 2007. Seharusnya tidak perlu add-on khusus untuk mendapatkan fungsionalitas ini dengan Excel 2007 (atau 2010 dalam hal ini juga).

\n\n
    \n
  1. Pilih semua kolom dan baris Anda di lembar kerja Excel Anda, buka tab Insert, dan pilih opsi Table.
    2. \n
  2. Di jendela dialog Create Table yang muncul, pilih apakah kolom data Anda memiliki header atau tidak, dan tekan OK.
    3. \n
  3. Sekarang lembar kerja data Anda memiliki panah filter yang dapat dipilih di bagian atas setiap tajuk kolom yang akan digunakan untuk mengurutkan nilai setiap kolom.
    4. \n
\n\n

\n\n\n

\n\n

TANPA KEPALA KOLOM (masih memiliki fungsi filter satu klik per kolom)

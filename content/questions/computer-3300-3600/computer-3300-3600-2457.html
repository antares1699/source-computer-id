Dalam Visual Studio, dimungkinkan untuk membuat tab untuk jendela editor yang berbeda melewati beberapa baris sehingga Anda dapat melihat semuanya sekaligus bahkan ketika Anda membuka 20 tab.

\n\n

Apakah mungkin untuk mengaktifkan ini dalam Visual Studio Code juga?

","source":"https://superuser.com/questions/1466771/is-there-a-way-to-make-editor-tabs-stack-in-vs-code","author":"ORcoder","replies":[],"answers":[{"answer":"

Di rilis terbaru VS Code v1.53 fitur ini telah ditambahkan

\n

Anda dapat menambahkan "workbench.editor.wrapTabs": true ke file konfigurasi json Anda

\n

Atau cari Workbech> Editor: Wrap Tabs in Settings UI.

\n

Selain itu Anda dapat mengaktifkan penyusutan pada tab sehingga mereka menggunakan lebih sedikit ruang (hanya lebar karakter) dengan "workbench.editor.tabSizing": "shrink"

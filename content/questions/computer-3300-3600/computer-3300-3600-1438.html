Kemungkinan Duplikat:
\n Apa perbedaan antara Windows 8 Pro dan Windows RT?

\n
\n\n\n\n

Saya tahu bahwa dulu ada Home Edition, Pro Edition, dan Starter Edition, tapi bagaimana dengan kali ini? Yang selalu saya dengar adalah Versi Pro, dan versi tabletnya. Itu saja?

\n\n

Atau masih ada lagi?

","source":"https://superuser.com/questions/493399/what-are-the-different-versions-of-windows-8","author":"TheXed","replies":[{"reply":"Lihat juga pertanyaan ini: superuser.com/questions/439470/â€¦ ","author":"Brad Patton"}],"answers":[{"answer":"

Ada empat edisi Windows kali ini: Windows RT, Windows 8, Windows 8 Pro dan Windows 8 Enterprise.

\n\n
    \n

  • Windows RT dirancang untuk CPU ARM, untuk saat ini hanya untuk tablet. Ini akan menjalankan aplikasi UI Modern baru, tetapi tidak akan menjalankan program Desktop dari versi Windows sebelumnya.

    • \n

  • Windows 8 adalah versi 'standar' untuk CPU x86 dan x86-64 (32-bit dan 64-bit), yang saat ini digunakan pada tablet, laptop, dan desktop, untuk menggantikan Home dan Home Premium. Ini akan menjalankan aplikasi UI Modern dan program Desktop tradisional. Sebagian besar program yang berjalan di Windows 7 akan berfungsi di sini.

    • \n

  • Windows 8 Pro adalah edisi 'premium' untuk CPU x86 dan x86-64 (32-bit dan 64-bit), yang saat ini digunakan pada tablet, laptop, dan desktop, untuk menggantikan Professional dan Ultimate. Ini memiliki beberapa fitur tambahan di atas Windows 8 (standar).

    • \n

  • Windows 8 Enterprise adalah edisi lisensi bisnis / volume, untuk menggantikan Perusahaan. Sebagai pengguna rumahan, jangan khawatir tentang ini. Ini mirip dengan Pro, tetapi media center hilang. Salah satu fitur penting adalah Windows-To-Go, yang memungkinkan Anda mem-boot dari flash drive.

    • \n
\n\n

Microsoft memiliki perbandingan fitur dari versi yang berbeda. Wikipedia menjelaskan lebih detail.

Ketika saya ingin mengakses superuser.com saya membuka Google Chrome, ketik Alt+D untuk mengaktifkan bilah alamat dan ketik su. Sekarang Chrome menyarankan alamat "superuser.com" dan saya hanya perlu menekan Enter.

\n\n

Namun, ketika saya ingin membuka file yang disimpan secara lokal di mesin Windows 7 saya (katakanlah, C:\\music\\interestingRecords.txt), prosesnya jauh lebih rumit. Apakah ada cara untuk menavigasi sistem file dengan kemudahan yang sama seperti alur kerja di atas?

Anda mungkin ingin memeriksanya Launchy . Saya sudah lama tidak menggunakannya tetapi saya cukup yakin Anda benar-benar menggunakan ALT + D (atau semacamnya) untuk menariknya. Kemudian Anda mulai mengetik dan menebak apa yang Anda inginkan. Jika sudah benar, Anda tekan enter dan voila!

Menu Start Windows 7 cukup bagus dengan sendirinya. Tekan tombol logo Windows (atau klik tombol Start), lalu mulailah mengetik nama file. Saat Anda mengetik, nama file yang cocok akan ditampilkan, serta yang memiliki konten yang cocok. Windows terkadang menunjukkan tebakan yang disorot - tekan enter untuk meluncurkan file yang disorot. Atau klik pertandingan lain kapan saja.

Cara lain adalah dengan menggunakan FireFox. Anda dapat menavigasi drive Anda dengannya. Ketik "C:" di bilah alamat dan tekan enter. Anda akan melihat apa yang saya maksud. Biasakan untuk menggunakan FF untuk menavigasi, dan membuat bookmark menggunakan tag saja (jangan mengatur dengan membuat hierarki folder; itu akan menggagalkan tujuan). Anda hanya perlu menemukan semuanya dan mem-bookmark-nya sekali. Sejak saat itu Anda dapat melakukan apa yang Anda gambarkan di atas. Tidak tahu apakah Chrome akan melakukannya, tetapi saya menggunakan FF untuk melakukannya di tempat kerja. Drive lokal, folder jaringan, NAS, dll ... Saya sangat merekomendasikannya.

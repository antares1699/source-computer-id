Anda dapat mengatur disk RAM dan meletakkan file VM VHD Anda di dalamnya.

\n\n

IMDisk adalah software RAMdisk gratis yang bagus di Windows jika Anda ingin mencobanya. Membayar alternatif seperti Disk Primo RAM Memberi anda kinerja yang lebih baik , lebih banyak opsi manajemen cadangan dan alokasi memori dinamis (sehingga Anda dapat membuat disk ram 16GB, tetapi itu hanya akan menggunakan 10GB jika itu adalah ukuran VM).

\n\n

Namun, Anda juga harus mempertimbangkan untuk meletakkan VM Anda di SSD. Kecepatan mereka cukup cepat untuk jenis tugas ini, jauh lebih sederhana untuk dikelola, dan menyisakan lebih banyak RAM untuk tamu VM atau tugas lainnya. Saya sangat menyarankan Anda untuk membandingkan beberapa tugas Anda untuk melihat apakah peningkatan pada SSD sepadan dengan masalahnya sebelum melakukan penggunaan RAMDisk biasa.

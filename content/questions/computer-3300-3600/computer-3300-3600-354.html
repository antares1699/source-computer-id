Saya baru-baru ini mencoba atom dan ada satu hal yang menentangnya: ia mencoba melakukan pintar hal dan menghapus semua spasi tambahan saat saya menyimpan file - tanpa memberi tahu saya. Beberapa editor melakukan ini tetapi ini buruk ketika harus mengubah kendali. Tidak yakin mengapa programmer menjadikan ini perilaku default dan saya tidak dapat menemukan opsi untuk menonaktifkannya.

\n\n

Apakah ada cara untuk menonaktifkan ini di Atom?

Ya!. Ini tidak ada di panel 'Pengaturan' inti, melainkan bagian dari salah satu paket inti. Buka tampilan pengaturan (Mac: Atom > Preferensi), pilih 'Paket' dari daftar di sebelah kiri, dan temukan paket whitespace (ketik "spasi" di kotak pencarian di bawah Paket Terpasang :

\n\n

Apakah ada cara untuk membuat snapshot dari sistem Windows 7?

\n\n

Tujuan saya adalah dapat menyimpan status OS Windows 7 saya saat ini, pada penyimpanan eksternal, dan dapat memulihkannya pada perangkat keras yang sama di masa mendatang. Mirip seperti snapshot di mesin virtual.

\n\n

Salah satu solusinya adalah menginstal dua hard drive di RAID 1, lalu mengambil salah satunya untuk menyimpannya. Apakah ada solusi yang lebih mudah?

Penyimpanan yang paling andal adalah dengan mengkloning seluruh partisi (bagian dari disk tempat OS Windows 7 berada). Ini membutuhkan lebih banyak waktu dan kapasitas disk daripada memilih detail konfigurasi sistem secara selektif, tetapi juga jauh lebih kecil kemungkinannya untuk menghasilkan pemulihan sistem yang "hampir" rusak atau "hampir".

\n\n

Ada beberapa alat. Saya telah menggunakan open source Clonezilla berhasil pada beberapa mesin dengan Windows XP, 7, 8. Dalam satu kasus, disk dienkripsi.

\n\n

Jika disk tidak dienkripsi, Clonezilla akan menyimpan gambar yang dikompresi dan dioptimalkan. Untuk disk terenkripsi, itu hanya membuat salinan lengkap dari partisi terenkripsi, tanpa pengoptimalan.

\n\n

Anda perlu membuat USB, CD, atau DVD yang dapat di-boot (lihat Clonezilla Live CD untuk detail tentang membuat ini). yang akan mem-boot Linux secara otomatis dan menuntun Anda selangkah demi selangkah melalui prosesnya.

\n\n

Anda memerlukan koneksi LAN kabel ke server file dengan berbagi SAMBA / CIFS (Windows) terbuka, berbagi NFS, atau server SSH. Ada opsi untuk menggunakan nirkabel, tetapi saya tidak pernah berhasil mengkonfigurasi ini. Koneksi nirkabel membutuhkan drop ke baris perintah dan secara manual mengkonfigurasi adaptor wifi. Koneksi kabel muncul secara otomatis, dan mendukung DHCP.

\n\n

Kecepatan transfer sekitar 3 Gb / menit, jadi anggarkan sekitar satu jam untuk setiap 120 Gb. Jika disk tidak dienkripsi, sektor kosong akan diabaikan (misalnya, disk 800 Gb dengan 240 Gb yang digunakan akan memakan waktu sekitar dua jam dan membutuhkan 240 Gb atau kurang untuk file image).

\n\n

Disk target yang digunakan untuk menyimpan gambar harus memiliki kapasitas kosong setidaknya sebanyak ukuran konten yang dikompresi pada disk (atau partisi) yang sedang dikloning.

Anda tidak dapat benar-benar melakukan snapshot tradisional, tetapi Anda dapat membuat image sistem komputer Anda yang nantinya dapat digunakan untuk memulihkannya dengan fungsionalitas bawaan dari Windows 7 Backup.

\n\n

Saya akan memasukkan informasi dari artikel yang saya temukan sini

\n\n
\n

Untuk mencadangkan program, pengaturan sistem, dan file Anda

\n \n
    \n

  1. Buka Backup dan Restore dengan mengklik tombol Start Gambar\n Tombol Start, klik Control Panel, klik System dan\n Maintenance, lalu klik Backup and Restore.

    2. \n

  2. Di panel kiri, klik Buat gambar sistem, lalu ikuti\n langkah-langkah di wizard. Izin administrator diperlukan jika Anda\n diminta kata sandi atau konfirmasi administrator, ketikkan\n kata sandi atau berikan konfirmasi.

    3. \n
\n \n

Menyimpan versi gambar sistem yang berbeda

\n \n

Jika Anda menyimpan gambar sistem Anda di drive internal atau eksternal,\n atau pada CD atau DVD, Anda dapat menyimpan beberapa versi image sistem. Di\n hard drive internal dan eksternal, image sistem yang lebih lama akan dihapus\n saat drive kehabisan ruang. Untuk membantu menghemat ruang disk, Anda bisa\n menghapus gambar sistem yang lebih lama secara manual. Untuk informasi lebih lanjut, lihat Apa\n pengaturan cadangan yang harus saya gunakan untuk memaksimalkan ruang disk saya?

\n \n

Jika Anda menyimpan gambar sistem Anda di lokasi jaringan, Anda bisa\n hanya menyimpan image sistem terbaru untuk setiap komputer. Sistem\n gambar disimpan dalam format drive \\ WindowsImageBackup \\ komputer\n nama. Jika Anda memiliki image sistem yang ada untuk komputer dan\n membuat yang baru untuk komputer yang sama, image sistem baru akan\n menimpa yang sudah ada. Jika Anda ingin mempertahankan sistem yang ada\n gambar, Anda dapat menyalinnya ke lokasi lain sebelum membuat yang baru\n image sistem dengan mengikuti langkah-langkah berikut:

\n \n
    \n

  1. Arahkan ke lokasi gambar sistem.

    2. \n

  2. Salin folder WindowsImageBackup ke lokasi baru.

    3. \n
\n \n

Catatan

\n \n

Untuk membuat citra sistem drive, itu harus diformat untuk menggunakan\n Sistem file NTFS. Jika Anda menyimpan image sistem pada hard drive atau USB\n flash drive, itu harus diformat untuk menggunakan sistem file NTFS.

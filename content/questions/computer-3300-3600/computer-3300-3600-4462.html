Keduanya sama. Setidaknya, dalam kasus Ubuntu. Saat Anda mem-boot CD, Anda dapat memulai Ubuntu Linux dalam mode LIVE. Mode ini akan memungkinkan Anda menguji perangkat keras Anda untuk kompatibilitas, misalnya. Hanya satu hal yang perlu diingat: karena semuanya dimuat dari CDROM, semuanya jauh lebih lambat daripada yang diharapkan saat boot dari hard disk. Namun, Anda akan dapat menguji semua perangkat keras Anda tanpa menyentuh disk Anda. Ini memungkinkan Anda melihat Linux dengan lebih baik sebelum mengambil langkah besar.

\n\n

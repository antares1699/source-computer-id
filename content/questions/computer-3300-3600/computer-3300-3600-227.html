Ya, batasan kode 16-bit adalah ketika prosesor berjalan dalam "mode panjang". OS 32-bit tidak akan mengalihkan CPU ke mode lama.

\n\n

Anda juga dapat menggunakan emulasi (seperti DosBox) untuk menjalankan program 16-bit.

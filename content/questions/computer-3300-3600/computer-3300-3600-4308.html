Menggunakan baris perintah Linux (Bash shell) Saya ingin memuat situs web tertentu di browser saya (biasanya Firefox, tetapi yang lain akan baik-baik saja selama berjalan di Linux) dan mengisi formulir yang telah ditentukan dengan beberapa data. Sebenarnya, ini harus dijalankan dari Makefile. Sebagian besar data formulir bersifat statis dan akan disimpan sebagai variabel di Makefile, hanya beberapa bidang yang harus diisi secara manual sebelum mengirimkan formulir secara manual.

\n\n

Saya tahu cara memuat file situs web yang dimaksud dari baris perintah menggunakan: 

firefox <URL>\n

Tetapi sepertinya tidak ada kemungkinan untuk mengisi formulir secara otomatis dengan variabel yang diberikan pada baris perintah. Apakah ada plugin, dapat dijalankan, atau JavaScript yang memungkinkan saya melakukan ini? Setiap saran dan petunjuk dipersilakan. Saya tidak keberatan mengkodekan beberapa JavaScript.

Saya menyelesaikan ini sekarang menggunakan skrip Greasemonkey yang menyalin parameter URL ke formulir.\nMakefile memanggil firefox dengan URL termasuk parameternya.

\n\n

Ini bekerja untuk kebutuhan saya. Beberapa perbaikan akan diperlukan untuk masukan yang lebih kompleks, yaitu beberapa pengkodean URL.

\n\n

Skrip Pengguna: 

// ==UserScript==\n// @name           Fill CTAN upload form\n// @namespace      http://www.scharrer-online.de/namespace/\n// @description    Fills the CTAN upload form with the data provided as URL variables\n// @include        http://*.ctan.org/upload.html*\n// @require        http://ajax.googleapis.com/ajax/libs/jquery/1.3.2/jquery.min.js\n// ==/UserScript==\n\n//var $ = unsafeWindow.jQuery;\n\n// From http://papermashup.com/read-url-get-variables-withjavascript/\n// Added: decodeURI\nfunction getUrlVars() {\n    var vars = {};\n    var parts = window.location.href.replace(/[?&]+([^=&]+)=([^&]*)/gi, function(m,key,value) {\n        vars[key] = decodeURI(value);\n    });\n    return vars;\n}\n\nvar vars = getUrlVars();\n\nfor (name in vars) {\n    $(\"input[name=\" + name + \"]\").val(vars[name]);\n    $(\"select[name=\" + name + \"]\").find(\"option:selected\").removeAttr('selected');\n    $(\"select[name=\" + name + \"]\").find(\"option[value=\" + vars[name] + \"]\")\n            .attr('selected','selected');\n}\n\nif ('DoNotAnnounce' in vars) {\n    var val = vars['DoNotAnnounce'].toLowerCase();\n    if (val == '1' || val == 'yes' || val == 'true') {\n        $(\"input[name=DoNotAnnounce]\").attr('checked','checked');\n    }\n}\n

Bagaimana cara membuat Windows 7 masuk mode tidur setelah x detik?

\n\n

Gunakan file batch berikut, dan jalankan sebagai Administrator 

@echo off\nrem disable hibernate\npowercfg -hibernate off\nrem wait x seconds, eg 1 hour\ntimeout 3600 /nobreak\nrem sleep\n%windir%\\System32\\rundll32.exe powrprof.dll,SetSuspendState 0,1,0\n
\n\n

Bacaan lebih lanjut

  • waktu habis - Penundaan eksekusi selama beberapa detik atau menit, untuk digunakan dalam file batch.

Satu baris (berdasarkan jawaban yang diterima DavidPostill): 

timeout /T 3600 & rundll32.exe powrprof.dll,SetSuspendState 0,1,0\n

Untuk memastikannya berhasil, Anda bisa mencobanya seperti ini: 

timeout /t 10 & notepad\n

Notepad akan terbuka setelah 10 detik.

\n

Catatan penting: Jika Anda menginstal git di komputer Anda, folder bin-nya mungkin ada di PATH, dan melakukan timeout di baris perintah akan berjalan\nC:\\Program Files\\Git\\usr\\bin\\timeout.exe bukan default C:\\Windows\\System32\\timeout.exe! Dalam hal ini Anda harus menyertakan jalur lengkap timeout, untuk menggunakan alat batas waktu Windows.

Saya pribadi meletakkan tombol di Outlook Quick Access Toolbar (QAT). Untuk melakukannya, cukup klik menu drop-down di pojok kiri atas jendela Outlook dan pilih "More Commands ...". Dalam dialog "Sesuaikan Bilah Alat Akses Cepat", pilih "Semua Perintah" dalam daftar "dari" dan Tambahkan >> tombol "Opsi Pesan ..." di daftar kanan. Klik OK dan Anda berada di sana.

\n\n

Sekarang Anda dapat memilih pesan yang diperlukan dalam daftar dan klik tombol baru ini pada QAT untuk menampilkan "Internet Headers".

Tentukan tombol untuk memanggil makro VBA berikut untuk menampilkan header email yang saat ini dipilih: 

Public Sub mailHeaderView()\n    Dim exp As Explorer\n    Dim sel As Selection\n    Dim please As String\n    Dim s As String\n    Dim mi As Outlook.mailItem\n    Dim TransportMessageHeadersSchema As String\n\n    ' https://social.msdn.microsoft.com/Forums/vstudio/en-US/e9342b09-02ba-441f-a824-5186bd9835a9/mailitemread-event?forum=outlookdev\n    TransportMessageHeadersSchema = \"http://schemas.microsoft.com/mapi/proptag/0x007D001E\"\n\n    Set exp = Application.ActiveExplorer\n\n    If exp Is Nothing Then\n      MsgBox \"No Explorer!\"\n    Else\n      Set sel = exp.Selection\n\n      please = \" Please select one Mail!\"\n      If sel Is Nothing Then\n        MsgBox \"No Selection!\" & please\n      ElseIf sel.count > 1 Then\n        MsgBox \"Multiple selection!\" & please\n      ElseIf sel.count < 1 Then\n        MsgBox \"No Selection!\" & please\n      ElseIf sel.item(1).Class <> olMail Then\n        MsgBox \"No Mail selected!\" & please\n      Else\n        Set mi = sel.item(1)\n\n        If mi Is Nothing Then\n          MsgBox \"No mail selected!\"\n        Else\n            s = mi.PropertyAccessor.GetProperty(TransportMessageHeadersSchema)\n\n            MsgBox s, vbInformation, \"Mail Header\"\n        End If\n      End If\n    End If\nEnd Sub\n

Makro serupa dijelaskan sini . Daripada membuka MsgBox, header email dapat ditampilkan sebagai email atau sebagai file teks melalui editor teks atau browser web.

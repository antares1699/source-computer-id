Jika saya memiliki file CSV seperti ini: 

Id,Name,Reference\n1,Alice,123EFG\n2,Bob,456E89\n3,Charles,7E9K2M\n4,Donald,12345678E28496023749\n

dan saya membukanya di LibreOffice, beberapa nilai diubah menjadi notasi ilmiah, seperti ini:

\n\n

\n\n

Perhatikan bahwa ini bukan hanya bagaimana nilai diformat untuk tampilan. Anda dapat melihat dari kotak rumus bahwa nilai sebenarnya telah diubah menjadi notasi ilmiah, jadi tidak ada cara dalam LibreOffice pada saat ini untuk memulihkan data yang telah hilang dalam konversi ini.

\n\n

Bagaimana saya bisa membuka file di LibreOffice tanpa nilai-nilai ini diubah menjadi notasi ilmiah?

","source":"https://superuser.com/questions/1151150/open-a-csv-in-libreoffice-calc-without-converting-values-to-scientific-notation","author":"Liam","replies":[],"answers":[{"answer":"

Coba opsi ini dalam dialog Impor Teks:

\n\n
    \n
  1. Hapus centang pada kotak 'Deteksi Nomor Khusus'
    2. \n
  2. Di area pratinjau, klik tajuk kolom yang berlabel 'Standar' dan ubah menjadi 'Teks'. Menurut dokumentasi :
      \n
    • Standar: LibreOffice menentukan jenisnya.
      • \n
    • Teks: Data yang diimpor diperlakukan sebagai teks.
      • \n
    3. \n
\n\n

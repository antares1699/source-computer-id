Lihat petunjuk di bawah ini untuk memasukkan template HTML Anda:

\n\n
\n

Saat Anda membuat pesan HTML, Anda\n dapat menyisipkan tag HTML atau mengedit yang ada\n Tag HTML di dalam pesan:

\n \n
    \n
  1. Secara opsional, untuk mengedit tag HTML yang ada, pilih bagian dari pesan,\n atau pilih seluruh pesan (untuk\n Misalnya dengan menekan Ctrl + A atau dengan\n memilih Edit -> Pilih Semua).
    2. \n
  2. Pilih: Sisipkan -> HTML ...
    3. \n
  3. Edit kode HTML sesuai keinginan.
    4. \n
  4. Tekan Sisipkan.
    5. \n
\n \n

Lihat di bawah untuk ekstensi yang menyediakan\n cara lain untuk mengedit tag HTML.

\n \n

Catatan: Beberapa orang yang menerima Anda\n pesan mungkin tidak melihat efek dari\n perubahan apa pun yang Anda lakukan pada HTML\n tag. Misalnya, pengguna Thunderbird\n mungkin memilih untuk menampilkan pesan sebagai\n teks biasa atau HTML sederhana, dan\n klien webmail mungkin mengubah HTML Anda\n dengan berbagai cara.

\n
\n\n

TOLONG DICATAT:

\n\n

Saat Anda melakukan ini, pastikan bahwa Anda memiliki semua gambar yang dihosting di suatu tempat. (server web, server hosting gambar, dll.) Tentu gambar mungkin terlihat saat berada di komputer Anda, tetapi saat gambar tersebut ada di orang lain, itu tidak akan terjadi. Mereka akan melihat placeholder gambar yang rusak.

\n\n

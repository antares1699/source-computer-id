Salah satu kemungkinannya adalah folder "WindowsApps" di bawah Program Files, yang menyimpan aplikasi Metro UI. Karena mereka berjalan di kotak pasir yang aman, ada banyak duplikasi file dan semuanya bisa menjadi relatif besar dengan cepat jika Anda menginstal beberapa aplikasi toko.

\n\n

Folder ini tidak hanya disembunyikan, tetapi Anda sebagai pengguna (atau bahkan Anda sebagai administrator) tidak memiliki izin untuk membacanya dan Anda juga bukan pemiliknya. Hal ini dapat mencegah pencarian atau klik kanan + properti pada folder untuk menghitungnya. Mungkin ada file lain dalam keadaan diblokir seperti itu di drive C, tetapi ini akan menjadi potongan terbesar yang dapat saya pikirkan.

\n\n

Lihat jawaban ini untuk detail lebih lanjut tentang cara mengakses folder WinApps.

\n\n

Saya tidak akan menyarankan untuk menghapus semua yang ada di sana, tetapi saya tidak mengalami masalah (sejauh ini!) Menghapus folder yang saya tahu tidak saya gunakan.

\n\n

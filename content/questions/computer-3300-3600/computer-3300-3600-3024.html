Mendapatkan pesan vmware telah menghentikan mesin virtual ini karena disk yang disimpan mesin virtual hampir penuh. untuk melanjutkan, kosongkan ruang disk sebesar 962 MB.\nMac saya memiliki lebih dari 160GB gratis. Bootcamp memiliki 40GB tetapi hanya menampilkan 1GB gratis. Mesin virtual VMware COPY memiliki 5 GB dan mesin virtual VMware memiliki 25 GB. Namun ketika saya masuk ke mesin vitual VMware, drive C mengatakan ukuran 40GB dan 11GB digunakan dengan 29GB gratis. Di mana saya bisa mengosongkan ruang ??

Coba opsi di VMware untuk "mengecilkan" disk Anda. Saya yakin itu akan merebut kembali ruang yang tidak terpakai.

Untuk menghindari persentase ruang kosong sistem file, file forum berikut mengacu pada parameter lanjutan untuk ditambahkan ke file konfigurasi teks VMX: 

mainMem.freeSpaceCheck = \"FALSE\"\n

Tetapi kemudian Anda harus berhati-hati untuk memiliki cukup ruang kosong, mungkin dengan alat lain untuk memeriksanya untuk Anda.

\n\n

Pilihan lainnya adalah menggunakan tindakan "menyusut", yang akan mengambil kembali ruang yang tidak digunakan dari gambar disk. Namun tidak semua jenis citra disk virtual mendukung "penyusutan" dan operasi tersebut mungkin memerlukan setidaknya 2 GiB ruang kosong atau paling buruk ukuran citra disk itu sendiri.

\n\n

Saya merekomendasikan jenis disk virtual "1: pemisahan disk virtual yang dapat dibesarkan dalam file 2GB" untuk memudahkan manajemen disk. VMWare Virtual Disk Manager alat berguna untuk membuat, mengembangkan, dan mengubah disk gambar. Ini tersedia di produk Workstation atau sebagai eksekusi mandiri di VMWare Server.

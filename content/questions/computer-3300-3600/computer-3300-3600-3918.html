Mendapatkan tag saat ini: 

echo 'local awful = require(\"awful\") ; return awful.tag.selected(2).name' | awesome-client\n

Beralih ke tag 1: 

echo 'local awful = require(\"awful\") ; return awful.tag.viewonly(tags[2][1])' | awesome-client\n

Dalam kedua kasus ini menangani layar nomor 2

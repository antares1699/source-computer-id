Saya perhatikan bahwa dengan unduhan dial-up, kecepatan unduh bisa beberapa kali lebih cepat. Tapi saya pikir throughput dial-up tercepat yang pernah ada adalah 53 K. Tapi saya pasti melihat kecepatan di atas itu di awal pengunduhan.

Setelah Anda masuk ke dialog simpan file di sebagian besar browser, unduhan Anda sudah dimulai saat Anda mengetikkan nama file. Setelah Anda mengeklik simpan, sejumlah besar data telah diunduh dan "meteran" jika Anda mau, harus menyusul sehingga pengukuran akan lebih tinggi selama beberapa detik lalu melambat ke normal.

\n\n

Sedangkan untuk unggahan, saya kira metode serupa sedang terjadi. Data mungkin sedang dikumpulkan sebelum koneksi dinegosiasikan sepenuhnya, itulah mengapa pada awalnya tampaknya "meledak". Meskipun saya tidak pernah mengalami hal ini dengan unggahan, mungkin karena saya tidak mengunggah cukup untuk menyadarinya. Kedua keanehan ini tidak spesifik untuk dial-up.

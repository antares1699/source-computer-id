Saat saya bekerja dengan Aplikasi Grafik, saya banyak menekan CTRL + S untuk Quicksave. Terkadang, saya bertindak terlalu jauh dan membuat keputusan yang buruk, terkadang Undo juga tidak membantu.

\n\n

Saya ingin menyimpan file versi lama. Biasanya, Kontrol Sumber akan digunakan di sini, tetapi itu adalah proses manual (sama seperti hanya membuat beberapa salinan).

\n\n

Saya ingin tahu apakah ada cara otomatis untuk melakukan itu? Setiap kali file berubah, simpan cadangannya. Saya percaya bahwa di Windows Server, Shadow Copies dapat melakukannya. Ketika saya memeriksa di Windows 7 (Ultimate) saya, saya melihat "Versi Sebelumnya" sebagai tab, tetapi itu tampaknya menjadi bagian dari fungsi cadangan yang sekali lagi manual.

\n\n

Apakah ada cara untuk mendapatkan jenis pembuatan versi otomatis itu?

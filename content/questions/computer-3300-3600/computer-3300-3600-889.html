Setidaknya ada dua metode untuk memilih saluran audio kiri dan keluaran sebagai mono.

\n\n

Menggunakan -map_channel

\n\n

Dengan asumsi input hanya berisi audio: 

ffmpeg -i input -map_channel 0.0.0 output\n
    \n

  • Angka pertama adalah masukan id file : Anda hanya memiliki satu masukan sehingga akan menjadi 0. Angka kedua adalah penentu aliran : jika audio adalah aliran kedua, seperti yang sering terjadi pada file video pada umumnya, Anda akan menggunakan 1 sebagai gantinya. Angka ketiga adalah nomor saluran : dalam input stereo, saluran pertama biasanya Kiri Depan, dan saluran kedua biasanya Kanan Depan.

    • \n

  • Jika Anda ingin saluran yang benar, gunakan -map_channel 0.0.1.

    • \n

  • Jika masukan berisi video, maka audio kemungkinan akan (tetapi tidak selalu) terdaftar sebagai aliran kedua, jadi Anda harus menggunakan 0.1.0 sebagai gantinya.

    • \n
\n\n

Menggunakan filter audio pan 

ffmpeg -i input -af \"pan=mono|c0=FL\" output\n
    \n
  • Alternatifnya, Anda bisa menggunakan pan=mono|c0=c0.
    • \n
  • Jika Anda menginginkan saluran yang benar, gunakan pan=mono|c0=FR (atau pan=mono|c0=c1).
    • \n
\n\n

Apakah ada yang punya alat yang bekerja dari clipboard ke file?

\n\n

Ini dapat dilakukan dengan menggunakan file batch yang berisi beberapa VBS yang disematkan.

\n\n

GetClip.cmd: 

@echo off\n(\necho set objHTML = CreateObject(\"htmlfile\"^)\necho ClipboardText = objHTML.ParentWindow.ClipboardData.GetData(\"text\"^)\necho set objFSO = CreateObject(\"Scripting.FileSystemObject\"^)   \necho set objFile = objFSO.OpenTextFile(\"clip.txt\", 2, true^)\necho objFile.WriteLine ClipboardText\necho objFile.Close\n) > \"%temp%\\clip.vbs\"\n\"%temp%\\clip.vbs\"\nendlocal\n

\n\n
    \n
  • Isi clipboard ditulis ke clip.txt di direktori saat ini
    • \n

Cara terbaik yang saya tahu, adalah dengan menggunakan alat mandiri bernama WINCLIP .

\n

Anda bisa mendapatkannya di sini: Mengecoh

\n

Pemakaian:

\n
    \n

  • Simpan clipboard ke file: winclip -p file.txt

    \n
    • \n

  • Salin stdout ke clipboard: winclip -c Contoh: Sed 's/find/replace/' file | winclip -c

    \n
    • \n

  • Clipboard pipa ke sed: winclip -p | Sed 's/find/replace/'

    \n
    • \n

  • Gunakan output winclip (clipboard) sebagai argumen dari perintah lain:

    \n

    FOR /F "tokens=* usebackq" %%G in ('winclip -p') Do (YOUR_Command %%G )

    \n
    • \n
\n

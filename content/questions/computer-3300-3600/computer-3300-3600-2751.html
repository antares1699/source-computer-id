untuk melihat apa yang Anda dapatkan: 

file casperjs.tar\n

dan kemudian untuk melihat isinya, jika itu adalah file tar: 

tar tvf casperjs.tar\ntar tvzf casperjs.tar  #if that was a gzip-ed tar file\ntar tvbf casperjs.tar  #if that was a bzip-ed tar file\ntar tvZf casperjs.tar  #if that was a compress-ed tar file  #now very unlikely...\n #note that after the 'f', you need to have a SEPARATOR (space, or tab) followed by the FILENAME. \n #ie, you can't place an option after the f\n #ex: \"tar tvfz something.tar.gz\", would try to open file \"z\" instead and find \"something.tar.gz\" inside it...\n

kemudian ubah 't' menjadi 'x' untuk mengekstrak (saya sarankan untuk menggunakan 't' terlebih dahulu, meskipun ...)

Atau ... di Windows, Anda cukup membuka %appdata%\\mozilla\\firefox\\profiles\\[your default profile]\\ dan menyalin direktori ekstensi ke drive yang dapat dilepas atau lokasi cadangan lainnya. Namun, ini mungkin tidak akan menyimpan data ekstensi Anda (hal-hal seperti riwayat sesi, dll.).

\n\n

Saya hanya merekomendasikan untuk mencadangkan isi direktori %appdata%\\mozilla\\firefox and %userprofile%\\local settings\\application data\\mozilla\\firefox Anda (di XP; ini sedikit berbeda untuk Vista / 7); langkah ini menyediakan cadangan lengkap dari keseluruhan pengaturan, riwayat, bookmark, dan ekstensi firefox Anda; jika Anda ingin atau membutuhkan, Anda bahkan dapat mentransfer cadangan ini ke sistem yang baru diinstal atau komputer Windows yang berbeda sama sekali, meskipun ini mungkin menyebabkan kesalahan ...

\n\n

Lihat Memindahkan folder profil Anda untuk lebih; ini adalah artikel yang bagus untuk siapa saja yang ingin mencadangkan atau memindahkan profil Firefox mereka.

\n\n

Sedangkan untuk sistem non-windows seperti Linux, saya menduga bahwa struktur tata letak direktori mungkin serupa, meskipun lokasi default direktori induk akan (jelas) berbeda (misalnya, ~/.mozilla/firefox). Saya belum mencoba, tetapi mungkin saja untuk mentransfer pengaturan dan data Firefox Anda dari komputer Windows ke komputer non-Windows (atau sebaliknya); Lihat Pindah dari Windows ke Linux untuk lebih.

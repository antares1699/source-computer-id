Anda dapat melakukannya dengan Linux. Baik mem-boot stik USB; atau jika Linux sudah diinstal, Anda dapat menulis ulang MBR sebagai "tindakan terakhir", yang akan membuatnya tidak dapat diakses (sampai Anda menginstal ulang Linux).

\n\n

Pertama, Anda harus mengidentifikasi disk. Di Terminal: 

sudo fdisk -l\n

untuk membuat daftar partisi. Disk mungkin akan menjadi /dev/sda atau /dev/hda. Kemudian: 

sudo apt-get install lilo\nsudo lilo -M /dev/sda mbr\n

lilo tidak lagi diinstal secara default, jadi baris pertama akan melakukannya (pada distro berbasis Debian, termasuk Ubuntu) jika perlu.

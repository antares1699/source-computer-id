Saya ingin mengetahui arsitektur prosesor saya.

\n\n

pertama yang saya lakukan adalah mengetik env di cmd dan saya menemukan ini. 

PROCESSOR_ARCHITECTURE=x86\nPROCESSOR_IDENTIFIER=x86 Family 6 Model 23 Stepping 10, GenuineIntel\nPROCESSOR_LEVEL=6\nPROCESSOR_REVISION=170a\n

Jadi, apakah arsitektur prosesor saya x86? dan bukan x86_64?

Jawabannya adalah: keduanya. Anda memiliki prosesor yang mendukung rangkaian instruksi x86_64 (a.k.a. x64, amd64, em64t, IA-32e, IA-64t,…) dan keluarga instruksi x86_32 (a.k.a. IA-32, i686, Pentium,…). Situs web intel melaporkan set instruksi 64-bit, menunjukkan x86_64, tetapi semua prosesor tersebut juga mendukung set instruksi 32-bit IA-32 yang lebih lama (serta set instruksi 16-bit dan 8-bit yang lebih tua).

\n\n

Sistem operasi Anda melaporkan arsitektur 32-bit karena Anda menjalankan versi sistem operasi 32-bit. Karena Anda tidak dapat menginstal aplikasi 64-bit di OS ini, tidak ada gunanya melaporkan arsitektur 64-bit. Tetapi Anda dapat menjalankan Windows versi 64-bit atau OS lain, dan itu akan melaporkan arsitektur x86_64.

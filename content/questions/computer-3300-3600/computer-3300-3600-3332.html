Saya menggunakan AutoHotKey untuk ini: 

EasyWheel(d)\n; if _WHEELAUTOFOCUS if set, check which window is under the mouse and gives it focus if it hasn't already\n; then send scroll event to the control under the mouse\n; original code from Shimanov: http://www.autohotkey.com/forum/viewtopic.php?t=6772#54821\n{\nGlobal _WHEELACC\n   , _WHEELMAXN\n   , _WHEELAUTOFOCUS\nStatic t, s\n\n\n   if ( A_TickCount > 500+t) {\n      t := A_TickCount\n      s :=0x780000\n   }\n   else if (s < _WHEELMAXN)\n      s += _WHEELACC\n\n   MouseGetPos x, y, hwnd\n   h := DllCall(\"WindowFromPoint\", \"int\", x, \"int\", y)\n   if _WHEELAUTOFOCUS && (hwnd<>WinExist(\"A\"))\n      WinActivate, ahk_id %hwnd%\n     testan:=WinActive(\"A\")\n    if (testan<>hwnd){\n        SendMessage, 0x20A, d*s,(y<<16)|x,, ahk_id %h%\n        }\n        else if (d=-1){\n        send {WheelDown}\n        }else{\n        send {WheelUp}\n        }\n}\n\n\nWheelUp:: EasyWheel(1)\nWheelDown:: EasyWheel(-1) \n

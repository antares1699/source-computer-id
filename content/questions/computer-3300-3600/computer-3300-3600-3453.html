Ada dua cara untuk memasukkan Linux ke WSL: Dapatkan dari Microsoft Store, atau buat sendiri.

\n\n

Ketika distro Linux tertentu tidak tersedia di Windows Store, Anda dapat membuatnya sendiri. Untuk mengetahui caranya, dan mengunduh alat yang dibutuhkan, kunjungi:

\n\n
    \n
  • "Membuat Distro Linux Kustom untuk WSL" . Docs . Microsoft (27 Maret 2018).
    • \n
\n\n

Tentu saja, Microsoft telah mengumumkan bahwa Fedora akan segera tersedia di Windows Store juga.

Dengan pemutar media yang bagus (mis. KMplayer atau VLC ) Anda dapat membuka file yang tidak lengkap.

\n\n

Kecuali urutan pengunduhan file, ini tidak benar-benar bergantung pada pengelola unduhan .... Jadi Anda dapat melakukannya dengan versi µTorrent apa pun, hanya saja tidak terintegrasi dalam versi 2.1 alpha sebelumnya.

\n\n

Kondisi sebenarnya adalah mendownload bagian pertama dan terakhir dari file tersebut. Ada opsi di Option > Preferences > Advanced > bt.prio_first_last_piece yang memungkinkan Anda memprioritaskan bagian ini, tetapi tidak disarankan untuk kecepatan jaringan global ... (Saya harap ini tidak menjadi default di 2.1)

\n\n

Jadi, jika Anda telah mengunduh bagian-bagian ini, Anda hanya perlu masuk ke direktori unduhan Anda dan membukanya dengan pemutar media yang memadai. (Untuk kenyamanan, jika Anda memeriksa Option > Preferences > General > \"Append .!ut to incomplete files\" Anda dapat mendaftarkan ekstensi ini dengan pemutar media khusus ...)

bash: syntax error near unexpected token '('

\n\n

Kamu butuh melarikan diri tanda kurung: 

mv hiscore\\(pre_mame0133u1\\).dat /mnt/three/usr/local/share/xmame/hiscore.dat\n

catatan:

\n\n

Untuk referensi di masa mendatang, Anda dapat menggunakan ShellCheck untuk menemukan bug di kode bash Anda. Memasukkan skrip yang tidak dikoreksi memberikan yang berikut: 

$ shellcheck myscript\n\nLine 1:\nmv hiscore(pre_mame0133u1).dat /mnt/three/usr/local/share/xmame/hiscore.dat\n^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.\n          ^-- SC1036: '(' is invalid here. Did you forget to escape it?\n          ^-- SC1088: Parsing stopped here. Invalid use of parentheses?\n

Memperbaiki kesalahan pertama: 

$ shellcheck myscript\n\nLine 1:\nmv hiscore\\(pre_mame0133u1).dat /mnt/three/usr/local/share/xmame/hiscore.dat\n^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.\n                          ^-- SC1089: Parsing stopped here. Is this keyword correctly matched up?\n

Dan mengoreksi kesalahan kedua: 

$ shellcheck myscript\n\nLine 1:\nmv hiscore\\(pre_mame0133u1\\).dat /mnt/three/usr/local/share/xmame/hiscore.dat\n^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.\n
\n\n

% = memungkinkan Anda untuk membagi pembenaran kiri dan kanan.

\n\n

Saya menggunakan ini di baris status saya, dengan cabang git saya saat ini rata kiri, dan info file rata kanan. Meskipun tidak ada informasi git yang tersedia, semua yang lain tetap sepenuhnya benar.

\n\n

edit : Saya harus mengatakan bahwa saya menggunakan% = untuk memperbaiki item dengan cara ini karena saya kesulitan membuatnya berfungsi dengan cara lain. Anda dapat mengatur lebar minimum menggunakan % -10.f (dengan 10 lebar minimum untuk beberapa item f).

\n\n

edit 2 : Untuk menjawab komentar Idigas di bawah ini.\nBerikut adalah baris status Anda saat ini (diformat ulang untuk ruang):

\n\n

set statusline=%mb:%n\\ %f%R%Y\\ [row\\ %l/%L,\\ column\\ %v\\ (%c)]

\n\n

Saya akan mengubahnya menjadi:

\n\n

set statusline=b:%n\\ %f%R%Y%=%m[row\\ %l/%L,\\ column\\ %v\\ (%c)]

\n\n

Dengan mengatur split kiri / kanan dengan % = dan memindahkan file yang telah dimodifikasi % m setelah itu, posisi baris status tampaknya diperbaiki.

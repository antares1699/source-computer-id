Saya biasa menjalankan VMware Player tanpa GUI dengan mengedit file .vmx dan menonaktifkan preferensi untuk mengonfirmasi sebelum menutup mesin virtual. Prosesnya adalah menonaktifkan opsi itu di preferensi VMware Player dengan menambahkan dua baris di akhir file .vmx.

\n\n

Ini akan membuat penutupan VMware Player mengirim VM ke latar belakang dan menutup GUI Player.

\n\n

Saya mencari di Internet untuk mencari tahu apakah kedua baris itu tetapi tidak berhasil; semua posting hanya berbicara tentang menggunakan perintah vmrun atau menggunakan VMware Server.

\n\n
\n\n

\n\n

Istilah yang Anda cari adalah menjalankan VM (bukan VMware Player, yang sebenarnya merupakan UI untuk berinteraksi dengan VM) tanpa kepala .

\n\n

Tutup semua instance VMware Workstation dan Player dan coba edit file preferensi Anda (%USERPROFILE%\\Application Data\\VMware\\preferences.ini di Windows, ~/.vmware/preferences di Linux) untuk menambahkan baris: 

pref.vmplayer.exit.vmAction = \"disconnect\"\npref.vmplayer.confirmOnExit = \"FALSE\"\n

Ini mungkin dilakukan dengan menjalankan notepad.exe sebagai administrator. Cara saya melakukannya adalah dengan menyalin file-hosts ke dalam "Copy of hosts", dan membuat perubahan pada file itu. Kemudian saya menghapus file-host (yang memungkinkan untuk beberapa alasan), dan mengganti nama file yang disalin menjadi "host".

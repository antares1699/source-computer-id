CATATAN: Seperti yang ditunjukkan oleh komentar @ richard, ini jawabannya sudah kadaluwarsa . Alternatif 'All-in-one' di tempat lain menjawab dan beri komentar di bawah.

\n\n

Tidak ada "solusi All-in-one" gratis untuk perekaman layar dalam format gif. Anda dapat menggunakan perekam layar apa pun untuk merekam dalam format video dan kemudian menggunakan beberapa alat untuk mengonversi file video ke gif.

\n\n

Satu solusi yang mungkin:

\n\n
    \n

  1. Menggunakan CamStudio Portable untuk merekam aktivitas layar Anda ke AVI.

    2. \n

  2. Menggunakan VirtualDubMod untuk mengubah AVI menjadi urutan gambar (PNG).

    3. \n

  3. Menggunakan Konverter Grafis Mudah untuk mengonversi PNG menjadi GIF.

    4. \n

  4. Menggunakan UnFREEz untuk membuat GIF animasi dari gambar.

    5. \n
\n\n

ScreenToGif adalah aplikasi gratis untuk Windows untuk merekam langsung ke GIF animasi.

\n\n

Anda hanya perlu memposisikan jendela transparan di atas konten yang ingin Anda rekam dan kemudian menyediakan alat yang diperlukan untuk menghapus bingkai, menyesuaikan kecepatan animasi, dan menambahkan overlay.

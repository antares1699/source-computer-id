Saya menemukan kendali untuk mengubah kecepatan semut berbaris di bawah

\n\n

edit > preferensi > Gambar Windows

\n\n

Saya menemukan ini pilihan yang lebih baik daripada mengubah file gimprc secara langsung karena

\n\n
    \n
  • Jika Anda belum terbiasa dengan sintaks gimprc, Anda mungkin menyebabkan skrip gagal saat penguraian.
    • \n
  • di GUI mengontrol pembaruan kecepatan semut berbaris secara real time saat Anda menyesuaikan panggilan cepat.
    • \n
  • Jika Anda menyesuaikan di GUI, Anda tetap memiliki opsi untuk "menyetel ulang semua konfigurasi alat" - jika Anda merusak sesuatu dan tidak dapat mengingat perubahan apa yang Anda buat!
    Karena file gimprc di-hardcode, saya curiga mereka tidak akan reposnd ke "reset tool config"

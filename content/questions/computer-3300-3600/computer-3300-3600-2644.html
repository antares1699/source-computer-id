Folder Instalasi Sublime Text 2 saya terus berkembang seiring dengan bertambahnya ukuran folder cadangan. Saya pikir itu karena plugin kontrol paket. Setiap kali, itu memperbarui, itu membuat folder bertanggal di Cadangan .

\n\n

Adakah cara untuk menghapus cadangan otomatis ini? Apakah ada pengaturan dalam kontrol paket yang dapat saya ubah? tolong bantu.

\n\n

Ini folder cadangannya.

\n\n

","source":"https://superuser.com/questions/870148/sublime-text-2-keeps-growing-due-to-backup-folder","author":"Cricrazy","replies":[],"answers":[{"answer":"

Folder ini dibuat oleh Kontrol Paket, setiap kali paket ditingkatkan atau dihapus. Itu ada sebagai cadangan (yah, ya :)) jika ada yang salah selama proses, atau Anda ingin mengembalikannya. Sayangnya, tampaknya tidak pernah dipangkas, oleh karena itu akumulasi dari waktu ke waktu. Jika semuanya baik-baik saja dengan Kontrol Paket, dan semua plugin berfungsi dengan baik, Anda dapat dengan aman menghapus folder di Backup.

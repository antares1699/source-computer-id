OK saya baru saja menemukan jawabannya. Ini adalah fitur Windows 2000 khusus untuk menyimpan halaman HTM (L), menurut posting 2001 ini:

\n\n

JSI Tip 3841. Apakah yang dimaksud dengan fitur File Terhubung Windows 2000?

\n\n

Ia juga mengatakan bahwa itu dapat dimatikan dengan menambahkan kunci registri REG DWORD ini dengan nilai 1:

\n\n

HKEY_CURRENT_USER \\ Software \\ Microsoft \\ Windows \\ CurrentVersion \\ Explorer \\ NoFileFolderConnection

\n\n

Dari sini dapat disimpulkan bahwa Anda juga dapat memutuskan koneksi untuk halaman HTML tertentu dengan mengganti nama EKSTENSI halaman html, mis. ke foo.htmlXXX. Saya menguji ini, dan Anda harus mengklik OK dua kali: untuk mengonfirmasi bahwa Anda mengubah ekstensi, dan untuk mengonfirmasi bahwa Anda memutuskan koneksi dengan direktori HTML terkait, yang persis seperti yang saya inginkan. Kemudian cukup hapus direktori itu, dan ubah ekstensinya kembali ke foo.html.

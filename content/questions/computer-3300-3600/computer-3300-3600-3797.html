Mengonfigurasi Mode Kios di Komputer Windows XP Anda

\n\n

Untuk mengkonfigurasi komputer agar secara otomatis masuk ke akun pengguna tertentu, Anda perlu mengubah Registri Windows. Lakukan ini dengan hati-hati dengan mengikuti langkah-langkah berikut:\n1. Klik Mulai »Jalankan.\n2. Ketik perintah regedit. Klik OK.\n3. Jelajahi kunci registri berikut: HKEY_LOCAL_MACHINE »Perangkat Lunak» Microsoft »Windows NT» CurrentVersion »Winlogon.\n4. Klik dua kali entri DefaultUserName dan masukkan nama akun pengguna yang ingin Anda masuki secara otomatis. Klik OK.

\n\n

5. Klik dua kali entri DefaultPassword dan masukkan kata sandi untuk akun pengguna yang ingin Anda masuki. Klik OK

\n\n

Jika tidak ada entri DefaultPassword, buat nilai menggunakan langkah-langkah berikut:\n1. Buka Edit »Baru» Nilai String.\n2. Beri nama nilai baru DefaultPassword dan tekan Enter.\n3. Klik dua kali entri yang baru dibuat dan masukkan kata sandi untuk akun pengguna yang ingin Anda masuki. Klik OK.

\n\n

CATATAN: Jika tidak ada string DefaultPassword yang ditentukan, Windows XP secara otomatis mengubah nilai kunci registri AutoAdminLogon dari 1 (true) ke 0 (false) untuk menonaktifkan fitur AutoAdminLogon.

\n\n

6. Klik dua kali entri AutoAdminLogon dan masukkan 1 untuk Value Data-nya. Klik OK.

\n\n

Jika tidak ada entri AutoAdminLogon, buat nilai menggunakan langkah-langkah berikut:\n1. Buka Edit »Baru» Nilai String.\n2. Beri nama nilai baru AutoAdminLogon dan tekan Enter.\n3. Klik dua kali entri yang baru dibuat dan masukkan 1 untuk Value Data-nya. Klik OK.

\n\n

7. Tutup Registry Editor dengan membuka File »Exit atau dengan mengklik X merah di sudut kanan atas jendela.\n8. Restart komputer dengan membuka Start »Shutdown, lalu pilih Restart.

\n\n

Untuk mengonfigurasi akun tertentu untuk boot ke Mode Kios, Anda perlu mengubah Registri Windows. Lakukan ini dengan hati-hati dengan mengikuti langkah-langkah berikut:\n1. Masuk ke akun pengguna yang ingin Anda boot ke Mode Kios.\n2. Klik Mulai »Jalankan.\n3. Ketik perintah regedit. Klik OK.\n4.Browse ke HKEY_CURRENT_USER »Software» Microsoft »Windows NT» CurrentVersion »Winlogon.\n5. Tambahkan nilai string baru ke registri ini dengan membuka Edit »Baru» Nilai String dan beri nama Shell.\n6. Klik dua kali pada nilai string baru ini dan masukkan jalur eksekusi yang ingin Anda jalankan, bukan explorer.exe (mis. C: \\ LV Startup \\ MyLabVIEWVI.exe).\n7. Keluar dari akun pengguna dan masuk kembali.\nDengan menggabungkan dua konfigurasi ini, Anda dapat mengatur mesin Windows XP Anda untuk secara otomatis melewati layar masuk pengguna dan boot ke akun pengguna tertentu yang akan menjalankan LabVIEW apa pun yang dapat dieksekusi yang Anda buat alih-alih Windows Explorer (explorer.exe). Anda mungkin ingin menyesuaikan tampilan yang dapat dieksekusi dengan memodifikasi VI Properties-nya. Misalnya, Anda dapat membuat maksimalkan yang dapat dieksekusi untuk mengisi seluruh layar dengan mengubah opsi Posisi diatur ke Dimaksimalkan dalam kategori Posisi Waktu Jalankan Jendela. Anda juga dapat menyesuaikan kategori Tampilan Windows sehingga pengguna tidak dapat menutup atau mengubah ukuran file yang dapat dieksekusi.

\n\n

TIPS: Jika Anda ingin mengubah file yang dapat dieksekusi yang sedang dijalankan untuk akun tertentu, tetapi mengalami masalah untuk kembali ke Editor Registri karena tidak ada lagi Start Menu, Anda dapat menekan Ctrl-Alt-Delete dan memilih untuk menjalankan Task Manager (jika belum berjalan). Kemudian pilih File »New Task (Run ...) dan ketik regedit di kotak yang muncul dan klik OK. Ini akan menjalankan Editor Registri.

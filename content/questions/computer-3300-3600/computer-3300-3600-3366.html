Di Windows Task Manager saya dapat mengatur (atau ketika saya mengklik kanan saya dapat melihat) prioritas proses. Is menunjukkan "real time", "high", "above normal", dll. Hal yang sama berlaku untuk "ProcessExplorer" (dari sysinternals, sekarang microsoft). Pertanyaannya adalah: Apakah ada cara untuk melihat utas apa yang berjalan dengan prioritas apa dalam proses tertentu? (bersama dengan angka, dari 0 hingga 31, menurut saya). Jadi itu akan menjadi 32 tingkat prioritas, bukan hanya 5 tingkat. (6,7,8,9,10)

Oke, saya menemukannya. Di ProcessExplorer Anda dapat mengklik kanan pada proses, pilih properti. Dari sana, pilih Tab Threads, ini menunjukkan semua thread yang berjalan di dalam proses dengan "prioritas dinamis".\nItulah yang saya cari.

","votes":"6","source":"https://superuser.com/a/30085","author":"ra170","replies":[]},{"answer":"

Aktifkan kolom 'Prioritas' di jendela utama penjelajah proses. Di sistem saya (yang tidak menjalankan apa pun sebagai waktu nyata), saya melihat prioritas dari 0-13, jadi menurut saya ini membaca prioritas sebenarnya.

\n\n

Lakukan ini dengan mengklik kanan pada tajuk kolom, lalu pilih 'Pilih Kolom ...' dari menu pop-up, lalu di bawah 'kinerja proses' pilih prioritas.

Saya sedang menulis skrip instalasi termasuk membuat file Diffie Hellman dengan perintah 

openssl dhparam -out /tmp/dhparam.pem 2048\n

Karena ini bisa memakan waktu dan tidak diperlukan untuk langkah-langkah berikut, saya berpikir untuk membuatnya berjalan di latar belakang tetapi saya tidak dapat menemukan cara untuk membuatnya berjalan dengan tenang, itu terus masuk ke terminal tempat skrip sedang berlari. Berikut beberapa upaya yang gagal: 

openssl dhparam -out /tmp/dhparam.pem 2048 > /dev/null &\nopenssl dhparam -out /tmp/dhparam.pem -quiet 2048 &\n

Sepertinya tidak menulis ke stdout, (melainkan langsung di / dev / tty?) Jadi saya tidak tahu bagaimana membuatnya diam: ada petunjuk?

Saya memiliki Macbook Pro dengan SSD dan saya ingin sepenuhnya menghapus setiap bit data di dalamnya. Bagaimana saya melakukannya? Di PC, saya hanya akan menjalankan Darik's Boot And Nuke, tetapi dapatkah saya menjalankannya di Mac juga, atau adakah alternatif yang lebih baik?

","source":"https://superuser.com/questions/290403/how-to-do-a-secure-erase-on-a-macbook-pro","author":"superuser","replies":[],"answers":[{"answer":"

Jika Anda ingin menghapus file boot disk, Anda harus boot dari CD penginstalan Mac OS X (yang seharusnya disertakan dengan Mac Anda) dan menjalankan program Disk Utility di dalamnya. Jika Anda ingin menghapus hard drive yang bukan boot disk Anda, Anda dapat menjalankan program Disk Utility yang dapat ditemukan di Applications / Utilities / Disk Utility.

\n

Untuk melakukan penghapusan aman seperti yang Anda inginkan:

\n
    \n

  1. Buka Disk Utility

    \n
    2. \n

  2. Pilih hard drive yang ingin Anda hapus dari daftar

    \n

    \n
    3. \n

  3. Pilih "Hapus" dari bilah di bagian atas

    \n

    \n
    4. \n

  4. Klik "Opsi Keamanan" dan pilih metode penghapusan mana yang ingin Anda gunakan. Pastikan Anda memilih salah satu opsi "# pass erase" untuk melakukan apa yang Anda inginkan.

    \n

    \n
    5. \n

  5. Klik hapus, dan disk akan dihapus setelah dihapus dengan nol.

    \n
    6. \n

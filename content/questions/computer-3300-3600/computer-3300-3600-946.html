Biasanya Anda menginstal perangkat lunak atau paket deb pada Ubuntu / Mint linux melalui Synaptic, Ubuntu Software Center / ppa, atau perintah apt-get dari terminal. Di sisi lain, Fedora, centos, mandriva, dan redhat mendukung paket rpm.

\n\n

Suatu saat Anda harus menginstal paket rpm ke kotak ubuntu / mint linux / debian Anda, ada utilitas bernama Alien, Alien akan mengubah paket dari satu format ke format lain dengan mudah.

\n\n

Berikut adalah cara menginstal paket rpm di ubuntu, linux mint, atau debian

\n\n

Instal alien dan semua dependensi yang dibutuhkannya 

# apt-get install alien dpkg-dev debhelper build-essential\n

Untuk mengubah paket dari format rpm ke debian 

# alien packagen.rpm\n

Terakhir, instal paket Anda! 

# dpkg -i package.deb\n

Untuk info lebih lanjut, gunakan 

$ alien -h\n

Bagaimana dengan mengabaikan mouse dan hanya menggunakan keyboard untuk melakukan ini?

\n\n
    \n
  • Untuk membuat pilihan yang tidak bersebelahan: Gunakan tombol panah untuk pindah ke file pertama yang ingin Anda pilih. Sekarang, tahan Ctrl dan terus gunakan tombol panah untuk pindah ke file yang Anda inginkan dan tekan Ruang untuk memilih masing-masing.
    • \n
  • Untuk membuat pilihan (rentang) yang berdekatan: Seperti di atas, gunakan tombol panah untuk pindah ke file pertama yang ingin Anda pilih, lalu tahan Ctrl dan gunakan tombol panah untuk pindah ke file terakhir dalam rentang tersebut. Sekarang lepaskan Ctrl dan tekan Bergeser + Ruang .
    • \n
\n\n

Anda juga dapat menggabungkan mouse dan keyboard, mis. klik file pertama yang ingin Anda pilih, sekarang tahan Bergeser dan klik file terakhir dan Anda akan mendapatkan pilihan yang berdekatan.

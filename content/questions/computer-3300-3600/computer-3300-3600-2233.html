saya menemukan sedikit informasi ini :

\n\n
\n

Saat Anda mengklik kanan Desktop dan memilih Atur Ikon Menurut sub-menu,\n tiga opsi ini mungkin hilang:

\n \n

Tampilkan ikon Desktop / Kunci item web aktif\n Desktop / Jalankan Pembersihan Desktop\n Penyihir

\n \n

Ini terjadi jika kebijakan\n NoActiveDesktopChanges atau\n NoActiveDesktop diaktifkan melalui\n Editor Kebijakan Grup atau melalui\n Registry. Ikuti langkah-langkah yang diberikan di bawah ini\n untuk mendapatkan kembali opsi yang hilang:

\n \n

Klik Start, Run dan ketik REGEDIT. Arahkan ke kunci berikut:

\n \n

HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Policies\\Explorer

\n \n

Klik dua kali NoActiveDesktopChanges dan setel ke 0\n Klik dua kali NoActiveDesktop dan setel ke 0 (jika ada)

\n \n

Tutup Peninjau Suntingan Registri dan mulai ulang Windows.

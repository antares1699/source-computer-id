Saya memiliki dua file MPEG-4 yang dikodekan H.264 yang perlu saya gabungkan. Saya telah menggunakan MP4Box untuk tugas ini dengan terlebih dahulu memastikan file dikodekan secara identik (bahkan melangkah lebih jauh dengan membandingkan output dari h264_parse di trek video mereka) dan kemudian menggabungkan dengan perintah ini: 

MP4Box -cat file1.mp4 -cat file2.mp4 output_file.mp4\n

Ini berfungsi dan file keluaran dapat diputar, tetapi saat pemutaran di Quicktime atau VLC, audio video kedua dimulai terlalu cepat, membuat seluruh bagian kedua dari file yang digabungkan tidak sinkron.

\n\n

Saya telah mencoba mengenkode ulang output melalui ffmpeg dengan -vcodec copy dan -acodec copy tetapi masalah sinkronisasi tetap ada.

\n\n

Saya juga mencoba mengonversi ke format MPEG-2 terlebih dahulu, menggabungkan dengan cat file1.mpg file2.mpg > output.mpg sederhana dan menyandikan ulang hasilnya dengan ffmpeg. Ini bahkan lebih buruk.

\n\n

Saya tahu bahwa saya dapat memberikan perintah ke MP4Box untuk menyesuaikan waktu mulai trek audio, tetapi saya mencoba melakukan ini secara terprogram untuk video input apa pun (dalam pengkodean yang sama tentu saja). Saya mencari solusi yang mungkin tidak memerlukan intervensi manusia / penyesuaian manual. Atau, setidaknya, pemahaman tentang apa yang terjadi sehingga bagian kedua dari video yang digabungkan tidak sinkron.

Saya menghadapi masalah yang sama. Untuk sinkronisasi yang benar, Anda harus menulis perintah MP4Box sebagai berikut: 

MP4Box -add file1.mp4 -cat file2.mp4 -new output_file.mp4\n

alih-alih menggunakan -cat, gunakan -add sebelum video pertama. Pilihan lainnya sudah benar dan Anda dapat meletakkan -new sebelum parameter video keluaran.

Coba gunakan wadah video Matroskra (MKV).

\n\n

Pertama, Anda membutuhkan mkvtoolnix.dll . Unduh dan instal, lalu Anda perlu menjalankan mkvmerge GUI (kecuali jika Anda menyukai alat baris perintah, yang tidak masalah bagi saya - meskipun Anda mungkin ingin melihatnya bagian tautan file dari dokumentasi mkvmerge ).

\n\n

Tambahkan file pertama Anda dengan mengklik "tambahkan". Kemudian, klik "tambahkan", dan buka file kedua Anda. Atur file keluaran, tekan "Mulai muxing" di bagian bawah, dan Anda pergi!

\n\n

Jika wadah tidak sesuai untuk kebutuhan Anda, Anda dapat mengenkode ulang, atau mencoba mengonversinya ke format lain ... Meskipun MKV adalah wadah yang sangat bagus!

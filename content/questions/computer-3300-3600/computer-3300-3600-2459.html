Lokasi folder itu sendiri sebenarnya menjawab pertanyaan tersebut. Tapi untuk kejelasan, biar saya jelaskan.

\n\n

Folder AppData terletak di folder pengguna, misal: C: \\ Users \\ Username \\ AppData, sedangkan folder program files terletak di C: \\

\n\n

Perbedaan terbesar di sini adalah izin. Jika Anda hanya memiliki satu pengguna di komputer Anda, maka pengguna tersebut memiliki semua hak di mana-mana dan itu mungkin tidak masuk akal. Tetapi jika Anda memiliki banyak pengguna, maka itu mulai masuk akal. Setiap pengguna memiliki foldernya sendiri di C: \\ Users, dengan folder AppData mereka sendiri. Ini tidak hanya berarti bahwa pengguna lain tidak memiliki akses ke folder AppData Anda, tetapi juga memungkinkan satu program untuk diinstal hanya sekali, namun memiliki pengaturan untuk setiap pengguna.

\n\n

Misalnya, jika Anda memiliki Mozilla Thunderbird atau Microsoft Outlook (mereka adalah klien email bagi mereka yang tidak menyadarinya), Anda tidak ingin menginstal aplikasi berulang kali untuk setiap pengguna. Anda lebih suka menginstal aplikasi sekali saja, tetapi memiliki pengaturan yang berbeda untuk tiap pengguna. Karena tidak setiap pengguna akan memiliki alamat email yang sama.

\n\n

Oleh karena itu, AppData ada. Untuk pengaturan pengguna.

\n\n

Alasan kedua adalah keamanan. Meskipun Program Files dapat menjadi tempat untuk program yang hanya menjalankan tugas dan tidak memiliki pengaturan, jika pengguna bukan administrator pada pc itu, mereka tidak dapat menulis ke folder Program Files, sehingga program itu sendiri tidak dapat membuat perubahan sama sekali. Namun di AppData, pengguna selalu memiliki hak untuk menulis di sana, meskipun mereka bukan administrator. Jadi program yang perlu menulis data, misalnya browser web dengan cache-nya harus menggunakan AppData untuk menulis cache tersebut atau pengguna akan mendapatkan prompt UAC yang meminta akses administratif setiap kali mereka memulai program.

mengapa program menyimpan datanya di folder AppData dan bukan di folder instalasi Program Files mereka?

\n
\n\n

Program Files adalah untuk menyimpan program itu sendiri, AppData adalah untuk menyimpan informasi khusus pengguna yang berkaitan dengan program. (Dan ProgramData adalah untuk menyimpan bukan informasi khusus pengguna yang terkait dengan program).

\n\n
\n

Apakah mereka benar-benar menyimpan semua data lokalnya di folder AppData, atau dapatkah mereka menyimpannya di mana pun mereka suka?

\n
\n\n

Seperti yang sudah disebutkan di atas, mereka juga dapat menyimpannya di ProgramData, dan keduanya hanyalah perilaku umum, ada juga program yang hanya membuat foldernya sendiri di C: \\, atau di folder Dokumen.

\n\n

","votes":"1","source":"https://superuser.com/a/164543","author":"Greg Bray","replies":[]},{"answer":"

NotePad ++ dilengkapi dengan plugin NppExec.
Salah satu dari banyak fiturnya adalah Jendela Konsol ... Memiliki keunikan tersendiri, tetapi mungkin sesuai dengan kebutuhan Anda.
Menurut saya, ini paling berguna jika dikaitkan dengan pintasan seperti CTRL + ~
Terkait dengan Dialog-Toggle internal-command, CTRL + ~ membuat konsol sangat mudah diakses. (tentu saja pilihan tombol pintas bersifat pribadi)
Anda dapat mengatur properti jendela (ukuran, warna, dan lainnya) melalui
menu -> Plugin -> NppExec ...
Pintasan-hotkey dapat diatur melalui
menu -> Pengaturan -> Pemeta pintas --- Perintah Plugin - Alihkan Dialog Konsol
Item dalam daftar ditampilkan (tetapi tidak diberi label) dalam grup per-Plugin, jadi mereka tampak agak acak ...

\n\n

Saya belum cukup memahami "apa dan mengapa", tetapi saya telah menemukan bahwa saya perlu keluar (di jendela yang sama) untuk memulai beberapa perintah .. mis. prompt> cmd / c tr.exe --help
Ini tidak identik dengan jendela edit NotePad ++; itu memiliki salin / tempel dan bilah Cari-nya sendiri (gaya Firefox).
Saya harap ini membantu.

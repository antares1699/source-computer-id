Keyboard OS X / Apple menawarkan pintasan keyboard praktis ini untuk karakter Unicode yang funky, seperti ⌥ + . memberikan karakter elipsis (…). Bagaimana saya bisa menggunakan ini di dalam iTerm2.app? Setiap kali tombol ⌥ ditekan dalam kombinasi dengan tombol lain, tampaknya memicu kesalahan (dengan suara dan semuanya)

\n\n

Saya menemukan ini di FAQ iTerm 2 :

\n\n
\n

T: Bagaimana cara membuat opsi / tombol alt berfungsi seperti Meta atau mengirim kode pelolosan?
J: Buka Preferensi → tab Bookmark. Pilih bookmark Anda di sebelah kiri, lalu buka tab Keyboard. Di bagian bawah adalah sekumpulan tombol yang memungkinkan Anda memilih perilaku tombol Option. Bagi banyak pengguna, esc + akan menjadi pilihan terbaik.

\n
\n\n

Jadi saya pergi ke Preferensi → Profil → [Profil Saya] → Keyboard dan di sana saya mengubah dua opsi terbawah menjadi “+ esc” bukan “normal” (default).

\n\n

\n\n

Tetapi meskipun demikian, saya tidak dapat mengetik karakter khusus menggunakan kombinasi tombol dengan tombol ⌥.

\n\n

\n\n

\n\n

Dengan demikian, Anda dapat memasukkan kombinasi tombol untuk umlaut, aksen, dll.

\n\n

Saya sudah mulai menggunakan tmux baru-baru ini, dan mengubah binding sehingga itu akan merespons C-a, bukan C-b.\nIni adalah baris di file .tmux.conf untuk melakukan ini: 

unbind C-b\nset -g prefix C-a\n

Tapi, saya tidak bisa membuat C-a bekerja untuk pergi ke awal baris. C-a, C-a tidak berfungsi, begitu pula C-a, a.\nSelain itu, pintasan lain seperti C-a, <beberapa tombol panah> untuk berpindah antar panel tidak berfungsi.\nApa yang saya lewatkan?

Anda adalah bagian dari perjalanan ke sana. unbind C-b menonaktifkan awalan default, dan set -g prefix C-a menyiapkan C-a sebagai awalan perintah baru seperti layar. Untuk mendapatkan perilaku layar lainnya, tambahkan beberapa baris lagi: 

set -g prefix C-a\nunbind C-b\n\n# C-a a should send ctrl-a to the underlying shell (move to start of line)\nbind-key a send-prefix    \n\n# C-a C-a\nbind-key C-a last-window\n

Perintah berikut mencantumkan semua file (dengan baris) yang memiliki baris yang disebutkan: 

$ grep  \"\\[oow\\] running\" *.log\n

Namun, yang ingin saya ketahui adalah semua file * .log yang berfungsi tidak memiliki garis itu. Menggunakan opsi -v untuk grep mencetak banyak baris lainnya. Tetapi saya hanya ingin file yang tidak memiliki baris yang disebutkan. Bagaimana orang biasanya melakukan ini di Unix / Mac (dalam baris perintah tentu saja)?

-L, --files-without-match\n       Suppress  normal  output;  instead  print the name of each input\n       file from which no output would normally have been printed.  The\n       scanning will stop on the first match.\n

ImDisk akan me-mount file image partisi kecuali Anda membuatnya menggunakan format "khusus" (opsi '-s') dari ntfsclone.

\n\n

Jika partisi tersebut diklon menggunakan format khusus, Anda perlu mengkloningnya lagi menggunakan ntfsclone.

\n\n

contoh: ntfsclone -ro newfile.img oldfile.img

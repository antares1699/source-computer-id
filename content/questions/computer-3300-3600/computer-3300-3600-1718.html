Masalah yang Anda hadapi sangat terkenal dan terjadi dengan beberapa perangkat keras nirkabel.

\n\n

Masalahnya adalah, Windows memindai jaringan nirkabel secara teratur, bahkan jika Anda sudah terhubung. Saat memindai, beberapa kartu nirkabel tidak dapat melakukan hal lain (seperti benar-benar bertukar paket), itulah mengapa "cegukan" terjadi.

\n\n

Untungnya, ada solusinya:

\n\n

Di Windows XP: saat Anda tersambung ke jaringan Nirkabel, nonaktifkan layanan "konfigurasi otomatis LAN Nirkabel". Perhatikan bahwa jika layanan ini dinonaktifkan, pemindaian jaringan tidak akan berfungsi, jadi Anda harus terhubung SEBELUM menonaktifkannya.

\n\n

Di Windows Vista: gunakan perangkat lunak seperti Pengoptimal WLAN atau Vista Anti-Lag .

\n\n

Di Windows 7: sayangnya, saya tidak tahu solusi apa pun untuk Windows 7. Kedua solusi yang disebutkan di atas tampaknya tidak berfungsi.

Mungkin Anda mengalami gangguan pada jaringan nirkabel Anda oleh beberapa agen yang tidak dikenal. Cobalah untuk mengubah saluran dan lihat apakah itu "memperbaiki" masalah. Buku koeksistensi jaringan nirkabel oleh Robert Morrow mengatakan ini:

\n\n
\n

Jaringan nirkabel beroperasi secara tetap\n saluran rentan terhadap\n gangguan dari orang lain di dekatnya\n jaringan saluran tetap. Satu cara yang jelas\n untuk menghindari gangguan tersebut adalah untuk\n ubah saluran seperti frekuensi itu\n ortogonalitas tercapai.

\n \n

Di Amerika Utara, tidak tumpang tindih\n set terdiri dari saluran 1,6, dan 11\n atau satu set yang tumpang tindih terdiri dari\n saluran 1, 3 5. 7, 9, dan 11, bisa\n terpilih. Set yang sesuai di\n Eropa adalah saluran 1, 3, 5, 7, 9, dan\n 11, bisa dipilih.

\n
\n\n

Ini memberi Anda nomor saluran terbaik untuk dicoba.

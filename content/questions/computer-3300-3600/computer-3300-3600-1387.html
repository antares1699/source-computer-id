Dari: http://jeetworks.org/node/151 :\nKesalahan ini: 

m_syscall.c:525:1: error: unknown type name '__private_extern__'\nm_syscall.c:526:1: error: expected '=', ',', ';', 'asm' or '__attribute__' before 'do_syscall_unix_WRK'\nm_syscall.c:549:1: error: unknown type name '__private_extern__'\nm_syscall.c:550:1: error: expected '=', ',', ';', 'asm' or '__attribute__' before 'do_syscall_mach_WRK'\nm_syscall.c: In function 'vgPlain_do_syscall':\nm_syscall.c:703:10: warning: implicit declaration of function 'do_syscall_unix_WRK' [-Wimplicit-function-declaration]\nm_syscall.c:708:10: warning: implicit declaration of function 'do_syscall_mach_WRK' [-Wimplicit-function-declaration]\nmake[3]: *** [libcoregrind_amd64_darwin_a-m_syscall.o] Error 1\nmake[2]: *** [all] Error 2\nmake[1]: *** [all-recursive] Error 1\nmake: *** [all] Error 2\n

Ini bisa diperbaiki dengan menambahkan baris: 

#define __private_extern__ extern\n

ke file berikut:

\n\n
    \n
  • coregrind/m_syscall.c
    • \n
  • coregrind/m_syswrap/syswrap-darwin.c
    • \n
  • coregrind/vg_preloaded.c
    • \n
\n\n

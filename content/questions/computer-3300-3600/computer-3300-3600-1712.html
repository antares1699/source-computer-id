Itu Pencadangan Server Windows perangkat lunak yang disertakan dengan Windows Server 2008 tidak lagi mendukung backup ke tape .

\n\n

Namun, ada banyak alat cadangan pihak ketiga untuk Windows yang masih mendukung tape. Untuk volume data yang besar, saya belum memiliki pesaing yang jelas untuk rekaman. Sesuatu seperti LTO yang dapat menyimpan 800GB rekaman yang digabungkan dengan file perpustakaan pita otomatis dapat dengan mudah menyediakan penyimpanan cadangan dalam jumlah besar.

\n\n

Kaset juga lebih nyaman daripada HDD penyimpanan di luar situs .

Itu Large Hadron Collider (LHC) CERN adalah ilustrasi yang bagus dari apa data dalam jumlah besar Artinya: menurut artikel itu mereka diharapkan mengumpulkan 15 PiB data per tahun bahwa itu operasional. \n\n

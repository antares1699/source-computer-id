Saya menggunakan fitur Koneksi Masuk Windows 8 untuk menjalankan server VPN PPTP dalam jaringan saya.

\n

VPN berfungsi dengan baik, kecuali bahwa komputer jarak jauh tidak dapat terhubung ke host VPN itu sendiri.

\n

Saya dapat melakukan ping dan terhubung ke semua komputer lain di jaringan lokal saya, tetapi saya bahkan tidak dapat melakukan ping ke alamat IP internal dari host VPN (yang juga merupakan file share)

\n

Pada Windows 7, saya memecahkan masalah ini dengan mengatur lokasi jaringan Antarmuka RAS (Dial In) VPN (pada mesin host) ke Pribadi.

\n

Namun, pada Windows 8, defaultnya adalah Publik, dan saya tidak dapat menemukan tempat di UI untuk mengubahnya. Lokasi jaringan sekarang diatur dalam Daftar Jaringan UI Modern, yang tidak menunjukkan koneksi masuk.

\n

Jalankan perintah PowerShell berikut: 

$networkListManager = [Activator]::CreateInstance([Type]::GetTypeFromCLSID([Guid]\"{DCB00C01-570F-4A9B-8D69-199FDBA5723B}\"))\n$networkListManager.GetNetworks(3) | Where-Object { $_.\"GetName\".Invoke(@()) -eq \"RAS (Dial In) Interface\" } | % { $_.SetCategory(1) }\n

Perintah pertama membuat objek NetworkListManager.
Perintah kedua terdiri dari tiga bagian (dipisahkan oleh |s):

\n\n
    \n
  • Bagian pertama mencantumkan semua jaringan terdaftar
    • \n
  • Bagian kedua menyaring daftar jaringan dengan nama tertentu.
    (hilangkan bagian ketiga untuk melihat hasil pemfilteran)
    • \n
  • Bagian ketiga mengatur lokasi jaringan untuk semua jaringan yang cocok. 0 adalah publik; 1 bersifat pribadi.
    • \n
\n\n

Untuk informasi (teknis) lebih lanjut, lihat Dokumentasi API Manajer Daftar Jaringan .

Anda dapat mengonfigurasi ini menggunakan Editor Kebijakan Grup Lokal seperti yang dijelaskan di pembahasan Technet ini . Jawaban ini juga berlaku untuk Windows 10.

\n\n

Tekan Win + R dan ketik GPEdit.msc.

\n\n

Lihat perincian sebagai berikut: 

Computer Configuration ->\nWindows Settings ->\nSecurity Settings ->\nNetwork List Manager Policies ->\n\"RAS (Dial In) Interface\" ->\nNetwork Location\n

Set Jenis Lokasi hingga Private.

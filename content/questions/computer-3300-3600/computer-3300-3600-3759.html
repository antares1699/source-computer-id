Larutan

\n\n

Ada opsi khusus yang dapat Anda aktifkan dengan mengikuti langkah-langkah ini:

\n\n
    \n
  1. tekan Menang + u untuk membuka Pusat Kemudahan Akses .
    2. \n
  2. Klik Jadikan keyboard lebih mudah digunakan tautan.
    3. \n

  3. Aktifkan Garis bawahi pintasan keyboard dan tombol akses opsi, dan tekan baik .

    \n\n
    \n

    Opsi ini menyoroti tombol akses untuk kontrol di kotak dialog sehingga Anda dapat berpindah-pindah dengan cepat. Untuk info lebih lanjut, lihat Pintasan keyboard .

    \n \n

    Sumber: Ubah pengaturan keyboard

    \n
    4. \n
\n\n

Sebelum Windows Vista, opsi serupa tersedia di pengaturan tampilan. Setelah desain ulang antarmuka pengguna, opsi dipindahkan ke dialog aksesibilitas.

Klik Tools pada menu, lalu Folder Options. Di bagian tengah halaman itu Anda akan melihat ⃰ ⃰Folder Options ***,\nklik itu. Di bagian tengah, Anda akan melihat bahwa biasanya Anda dapat memilih gaya garis bawah jika Anda memilih:

\n\n

⚫ Sekali klik untuk membuka item (tunjuk untuk memilih)

\n\n

⚫ Garis bawahi judul ikon yang konsisten dengan browser saya

\n\n

⚪ Garis bawahi judul ikon hanya jika saya menunjuk ke sana.

\n\n

⚪ D̲klik dua kali untuk membuka item (klik sekali untuk memilih)

\n\n

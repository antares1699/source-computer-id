Seorang klien ingin tautan ke Gmail-nya disematkan ke bilah tugas Windows 10 miliknya. Dia tidak ingin itu menjadi item Pinned di Daftar Langsung untuk Firefox. Sebaliknya, dia menginginkan item tersemat yang terpisah untuk tautannya sendiri. Saya tahu bahwa saya dapat menggunakan sakelar / argumen yang ditambahkan ke pintasan desktop untuk Firefox, tetapi saya rasa saya tidak dapat menyematkan pintasan ke bilah tugas, setidaknya tidak tanpa menginstal sesuatu seperti DisplayFusion (yang merupakan pilihan terakhir saya). Bisakah saya menyematkan tugas yang dijadwalkan entah bagaimana?

\n\n

Klien khusus ini sangat khusus tentang berbagai hal, tetapi dia sangat kaya dan memberi saya banyak pekerjaan, jadi saya mencoba dan melakukan semua yang dia inginkan sesepat mungkin dengan spesifikasinya.

Meskipun keseluruhan prosesnya tidak sesederhana Chrome atau Edge, ini masih relatif mudah:

\n\n
\n
    \n
  1. Buat pintasan Desktop baru yang mengarah ke Firefox. Metode termudah adalah dengan membuka direktori instalasi Firefox, klik kanan\n firefox.exe, dan pilih Buat pintasan dari menu konteks. Itu\n Direktori instalasi Firefox 64-bit default adalah C: \\ Program\n File \\ Mozilla Firefox , sedangkan standar instalasi Firefox 32-bit\n direktori adalah C: \\ Program Files (x86) \\ Mozilla Firefox .
    2. \n

  2. Pilih Iya ketika Windows memberi tahu Anda bahwa pintasan tidak dapat dibuat di direktori Firefox. Pintasan dibuat di\n desktop, sebagai gantinya.

    \n \n

    3. \n
  3. Klik kanan pada pintasan Firefox baru dan pilih Properti dari menu.
    4. \n

  4. Tambahkan URL situs yang ingin Anda buka dengan pintasan di dalam Target bidang, misalnya, "C: \\ Program Files \\ Mozilla Firefox \\ firefox.exe"\n -url https://superuser.com/

    \n \n

    5. \n
  5. Klik baik untuk menyimpan perubahan.
    6. \n
  6. Klik kanan pada pintasan desktop dan pilih Ganti nama untuk mengubahnya menjadi sesuatu yang lebih sesuai. Dalam contoh kami, kami memutuskan "Pengguna Super"
    7. \n

  7. Klik kanan pada pintasan desktop lagi dan pilih Sematkan ke bilah tugas . Secara opsional, Anda dapat menghapus pintasan desktop setelah Anda selesai.

    \n \n


    \n

    8. \n

Di Linux, periksa direktori ini. 

/sys/class/leds/\n

Jika ada subdirektori, lihat ke dalam file bernama trigger\nMembaca file trigger akan mencantumkan parameter kontrol LED yang tersedia, dengan yang saat ini dikelilingi oleh tanda kurung.

\n\n

Untuk mengubah parameter trigger, cukup tulis nama parameter ke file trigger.

\n\n

File lain, trigger dapat digunakan untuk menonaktifkan LED juga. Menulis 0 hingga brightness akan mematikan LED yang dikontrol oleh subdirektori tertentu.

\n\n

Untuk menonaktifkan semua LED yang dapat dikontrol sebagai tes, Anda dapat mencoba perintah ini: 

sudo echo -n 0 > /sys/class/leds/*/brightness\n

Jika Anda tidak dapat mengontrol LED dengan cara ini, mencabut papan mungkin satu-satunya cara.

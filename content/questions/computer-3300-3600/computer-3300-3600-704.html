Saya memiliki layar sentuh Windows 7 PC yang saya gunakan untuk aplikasi Kios. Pengguna tidak pernah perlu memasukkan teks.

\n\n

Saat saya mengetuk layar, baki keyboard kecil keluar dari margin kiri layar. Saat saya menyeretnya ke kanan, itu meluas ke Panel Input PC Tablet, baik keyboard virtual atau tampilan pengenalan tulisan tangan.

\n\n

Bagaimana cara menonaktifkan fitur ini, tanpa mempengaruhi fitur dukungan tablet lainnya?

","source":"https://superuser.com/questions/360489/how-do-i-disable-the-tablet-pc-input-panel-in-windows-7","author":"Jeremy Wadhams","replies":[],"answers":[{"answer":"

Klik tombol Start.

\n\n

Ketik "Panel Input PC Tablet" dan klik tautan pertama yang disarankan otomatis.

\n\n

Jika keyboard muncul dan bukan jendela Options pada saat ini, klik Tools, lalu Options.

\n\n

Hapus centang ketiga kotak di bawah "Pilih tempat untuk menampilkan ikon dan tab Panel Input"

\n\n

\n\n

Klik OK.

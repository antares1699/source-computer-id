Saya tidak dapat mengatakan dengan pasti, tetapi jika itu seperti program Anti Virus lainnya, kemungkinan besar itu memisahkan aplikasi ujung depan dari mesin pemindai ujung belakang.

\n\n

Pastikan Anda mencentang kotak (atau memberikan otentikasi UAC) di sudut kiri bawah yang menyatakan Show processes from all users.

\n\n

Jika Anda masih tidak melihatnya, pada tab Services, cari mesin pindai dan klik kanan dan pilih Go To Process. Ini harus Anda program apa pun yang terkait dengan layanan itu.

\n\n

Terakhir, jika ini tidak berhasil, saya akan menyarankan untuk melihat Microsoft / Sysinternals Process Explorer .

\n\n

Ekstra tidak terlalu membantu. Terakhir, jika Anda mengalami masalah, saya sarankan beralih ke Microsoft Security Essentials. Saya menggunakannya secara eksklusif sekarang dan saya sangat senang dengannya.

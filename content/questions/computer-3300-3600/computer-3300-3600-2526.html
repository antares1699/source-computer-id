Visual Studio 2015 saya menggunakan " sgen.exe ".

\n\n

Saya tidak menemukannya di mana pun di "C: \\ Program Files (x86) \\ Microsoft Visual Studio 14.0"

\n\n

Darimana ini diambil?

\n\n

Saya perlu tahu ini karena Studio visual tidak mengungkapkan detail kesalahan "SGEN exited with code 1". Saya ingin melihat detail ini untuk mengetahui alasan sebenarnya dari SGEN kesalahan.

\n\n

Bos saya menyarankan saya untuk lari SGEN manual, karena dalam kasus ini SGEN memiliki beberapa keluaran, misalnya, "... Cannot deserialize type 'TypeName' because...". Dan keluaran ini persis seperti yang saya butuhkan.

\n\n

Terima kasih!

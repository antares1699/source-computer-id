Menggunakan for loop dengan string input akan terbagi pada spasi kosong. 

LIST=\"value1  value2 value3\"\nfor x in $LIST ; do\n    echo $x\ndone\n

Hasil 

value1\nvalue2\nvalue3\n

atau 

LIST=\"value1  value2 value3\"\nset -A STRING \"$LIST\"\nfor x in $STRING ; do\n    echo $x\ndone\n

Hasil 

value1\nvalue2\nvalue3\n

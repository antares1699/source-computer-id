Jika Anda membuka properti untuk setiap folder, Anda sebenarnya dapat menemukan namanya, tetapi tidak ditampilkan di File Explorer. Terima kasih sebelumnya atas balasan apa pun.

Windows File Explorer memiliki sejarah panjang dalam berbohong kepada Anda.
Itu tidak menunjukkan kepada Anda apa yang sedang terjadi.\nIni hanya menunjukkan kepada Anda apa yang menurutnya harus Anda ketahui.\nYang bagus untuk pengguna rata-rata yang hanya memiliki gagasan samar apa file atau folder sebenarnya, tapi ini membodohi cukup menjengkelkan untuk powerusers dan sys-admin yang mencoba memecahkan masalah.

\n\n

Dan tampaknya versi Windows 10 lagi-lagi sedikit lebih buruk dari versi sebelumnya.
Folder masih ada: Explorer tidak menampilkannya, karena Explorer berpendapat bahwa folder tidak boleh memiliki ekstensi.
(Ini adalah gagasan yang kembali ke MS-DOS: Dalam folder sistem file FAT lama secara teknis TIDAK dapat memiliki ekstensi sama sekali.\nSistem file modern jauh lebih fleksibel, tetapi Windows masih memiliki pemikiran yang tersisa sejak masa itu. Jika Anda tidak percaya saya: Buat folder yang diakhiri dengan satu titik. Lalu coba hapus.)

\n\n

Satu-satunya hal yang dapat Anda lakukan adalah menggunakan utilitas pengelola file alternatif (tidak dibuat oleh Microsoft) yang menunjukkan kenyataan kepada Anda.
Dan alternatif tersebut biasanya jauh lebih mampu daripada Explorer.

\n\n

Secara pribadi: Saya belum pernah menggunakan Explorer untuk sesuatu yang serius sejak Windows 3.1. Saya selalu menggunakan pengelola file alternatif.

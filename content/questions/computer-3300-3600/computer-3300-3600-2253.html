Adakah yang tahu jika perangkat lunak tertentu yang memungkinkan saya mengontrol waktu di mana fitur pencarian Windows 7 mengindeks hard drive saya? Akan luar biasa jika hanya mengindeks drive (atau mengaktifkan layanan indeks) antara beberapa jam setiap hari. Tidak ada cara untuk menjadwalkan indeks tanpa menulis skrip untuk menghentikan dan memulai indeks. Untungnya, skripnya cukup mendasar.

\n\n

Buat 2 file batch:

\n\n

StopIndex.bat 

@ECHO OFF\nNET STOP \"Windows Search\"\nEXIT\n

StartIndex.bat 

@ECHO OFF\nNET START \"Windows Search\"\nEXIT\n

Lalu masuk ke penjadwal tugas Anda dan buat 2 tugas terpisah:

\n\n
    \n
  1. Start Index: arahkan tindakan ke file StartIndex.bat
    2. \n
  2. Stop Index: arahkan tindakan ke file StopIndex.bat
    3. \n
\n\n

Catatan: pastikan untuk memilih kotak centang "Jalankan dengan hak istimewa tertinggi" di bawah tab Umum, jika tidak menghentikan dan memulai akan gagal.

Ada masalah ketika Anda menghentikan layanan pencarian windows, itu dimulai setelah beberapa saat secara otomatis

\n\n

Jadi saya memodifikasi skrip sebagai berikut

\n\n

Mulai indeks 

@ECHO OFF\nsc config Wsearch start= delayed-auto\nNET START \"Windows Search\"\nEXIT\n

Hentikan Indeks 

@ECHO OFF\nsc config Wsearch start= disabled\nNET STOP \"Windows Search\"\nEXIT\n

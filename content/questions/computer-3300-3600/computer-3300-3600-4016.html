Anda dapat mengubah pengaturan ini di opsi. Jika Anda kebetulan telah memeriksanya:

\n\n

Excel Options > Advanced > General > Ignore other applications that use Dynamic Data Exchange

\n\n

itu akan berperilaku seperti yang Anda gambarkan. (jalur ini untuk Office 2010/2007, saya tidak yakin jalur yang tepat untuk tahun 2003 tetapi seharusnya ada di sana di suatu tempat yang cukup dekat dengan ini)

\n\n

Sunting : Sini jalur untuk perbaikan ini pada tahun 2010, 2007, dan 2003.

\n\n

Jika ini dicentang, setiap kali Anda membuka file Excel baru dari Windows Explorer, Excel akan memaksa pembuatan instance baru. Sangat menyenangkan jika Anda memiliki banyak monitor dan Anda ingin melihat dua file sekaligus tanpa melakukan manipulasi jendela yang mengganggu. Ini masih memungkinkan Anda untuk membuka file dalam instance saat ini ketika Anda membukanya dari dalam instance itu sendiri. Masalah yang muncul adalah menyalin dan menempel tidak seelegan. Anda hanya dapat C / P sebagai teks atau HTML atau format lain yang mengganggu. Dan Anda biasanya kehilangan rumus Anda.

\n\n

Jika opsi ini sudah tidak dicentang, saya tidak yakin kapan masalahnya bisa terjadi. Tapi, menurut saya tidak perlu memiliki Office Starter di komputer Anda setelah Anda menginstal Office asli. Saya akan menghapus seluruh paket Starter itu jika saya jadi Anda.

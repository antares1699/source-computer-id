\n\n

Cari nama file yang cocok dengan ?, ?? dan ???. Jika file memiliki ekstensi, coba ?.*, ??.* dan ???.*.

\n\n

\n\n

Mungkin lebih mudah menemukan file kosong yang dibuat dalam 30 hari terakhir.

\n\n

Buka menu Start - Search

\n\n

Klik When was it modified pilih Past month atau pilih tanggal secara manual.

\n\n

Klik What size is it pilih Specify size at most 1KB

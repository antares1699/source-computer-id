AutoHotkey adalah off-spin yang dibuat khusus untuk tombol pintas ...

\n\n

Anda mungkin ingin mencoba yang asli AutoIt sebaliknya, ia memiliki banyak fitur otomatisasi, termasuk acara jendela!

\n\n

Lebih spesifik, Anda bisa temukan skripnya di sini yang melacak jendela dan bekerja di jendela baru. 

#include <Array.au3>\n\n; Initialize tracking arrays\nGlobal $avWinListPrevious[1][2] = [[0, \"\"]], $avWinListCurrent\n\n; Monitor unique window handles\nWhile 1\n    $avWinListCurrent = WinList(\"[REGEXPTITLE:.+[ \\- ]GIMP]\", \"GNU Image Manipulation Program\")\n    For $n = $avWinListCurrent[0][0] To 1 Step -1\n   ; Check has title and visible\n        If ($avWinListCurrent[$n][0] <> \"\") And BitAND(WinGetState($avWinListCurrent[$n][1]), 2) Then\n       ; Check for already seen\n            $fFound = False\n            For $i = 1 To $avWinListPrevious[0][0]\n                If $avWinListCurrent[$n][1] = $avWinListPrevious[$i][1] Then\n                    $fFound = True\n                    ExitLoop\n                EndIf\n            Next\n\n       ; New window found\n            If Not $fFound Then\n                WinMove(\"[REGEXPTITLE:.+[ \\- ]GIMP]\", \"GNU Image Manipulation Program\", 169, 0, 893, 771 )\n            EndIf\n        Else\n            _ArrayDelete($avWinListCurrent, $n)\n        EndIf\n    Next\n    $avWinListCurrent[0][0] = UBound($avWinListCurrent) - 1\n    $avWinListPrevious = $avWinListCurrent\n    Sleep(500)\nWEnd\n

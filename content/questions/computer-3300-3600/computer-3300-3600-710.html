Saya tahu grep -irn \"#page-title\" akan mencari semua file untuk teks yang cocok tetapi untuk alasan kinerja saya ingin membatasi pencarian HANYA untuk file yang merupakan file css. Di Windows PowerShell ini: 

ls *.css -r | select-string \"#page-title\"\n

Poin bonus, bagaimana cara membatasi ini ke file yang TIDAK diawali dengan css_

","source":"https://superuser.com/questions/351375/how-to-find-all-css-files-containing-page-title-in-linux","author":"George Mauer","replies":[],"answers":[{"answer":"
find -name \"*.css\" ! -name \"css_*\" | xargs grep -H \"#page-title\"\n

find adalah baris perintah unix find, dengan 2 set parameter. Parameter pertama, -name \"*.css\", memberi tahu find untuk mencari file dengan ekstensi CSS, parameter ! -name \"css_*\" memberi tahu find untuk mengecualikan atau mengabaikan file yang dimulai dengan css_. Daftar file ini kemudian disalurkan ke perintah xargs, yang meneruskan daftar sebagai parameter ke dalam perintah grep, yang diperintahkan untuk menemukan "# page-title" dari daftar, dan mencetak nama file, menggunakan parameter -H.

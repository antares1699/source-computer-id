Saya akhirnya menemukan alasan mengapa Firefox tidak ingin mengingat kata sandi lagi.

\n\n

Saya sedang membaca halaman bantuan Mozilla di Pengelola Kata Sandi , dan baca di mana dikatakan "Mengklik di luar prompt Remember Password akan menyembunyikannya. Untuk mengembalikannya, cukup klik ikon kunci di sisi kiri bilah lokasi." Alih-alih kunci, saya melihat kunci di versi firefox saya (15.0.1) jadi saya mengkliknya. Itu memunculkan beberapa info tentang game tersebut. Saya mengklik tombol "Informasi Lebih Lanjut", lalu "Lihat sandi yang disimpan". Saya menyadari saya bisa mendapatkan ini melalui menu, tetapi saya melihat sesuatu yang mengakses melalui jalur ini yang tidak saya perhatikan melalui menu (mungkin ada di sana dan saya hanya tidak memperhatikannya sebelumnya, saya tidak tahu) - ada entri untuk situs web yang memiliki kata sandi tetapi tidak ada nama pengguna. Setelah saya menghapus entri ini, saya kemudian mencoba masuk ke situs tersebut, dan kemudian Firefox segera meminta saya untuk mengingat info login atau tidak.

\n\n

Singkatnya, jika Anda mengalami masalah dengan Firefox karena tidak ingin mengingat lebih banyak kata sandi, periksa apakah ada kata sandi yang tidak memiliki nama pengguna. Dalam kasus saya, ini membuat Firefox tidak ingin lagi mengingat kata sandi.

Saya merekomendasikan menggunakan ekstensi berikut:

\n

\n
\n

Multifox adalah ekstensi yang memungkinkan Firefox terhubung ke situs web menggunakan nama pengguna yang berbeda. Serentak!

\n

Misalnya, jika Anda memiliki beberapa akun Gmail, Anda dapat membuka semuanya pada saat yang bersamaan. Setiap jendela Firefox, dikelola oleh Multifox, mengakses akun tanpa mengganggu satu sama lain

\n

Setiap jendela Multifox ditandai dengan nomor yang menunjukkan profil identitas. Login yang dibuat di windows dengan nomor berbeda diisolasi.

\n

Proses masuk dipertahankan dengan cara yang sama seperti di jendela "biasa". Meskipun Anda menutup jendela atau keluar dari Firefox.

\n

Profil identitas setiap jendela dipertahankan saat Firefox memulihkan sesi tersebut.

\n
\n

Itu juga dapat dengan mudah beralih di antara profil:
\n

\n

Cukup gunakan profil yang berbeda untuk setiap akun Gmail yang Anda miliki, dan Multifox harus mengingat semuanya. Tidak perlu login ulang saat menggunakan banyak akun dengan Multifox.

\n
\n

Tentang Firefox yang tidak meminta untuk mengingat kata sandi untuk banyak akun, saya tidak mengalami masalah itu dengan Multifox.     • \n
  • Pilihan mudah dan nyaman untuk banyak login (banyak pengguna)
    • \n
  • Terintegrasi dengan pengelola kata sandi Mozilla Firefox.
    • \n
  • Memberikan akses mudah ke kata sandi yang disimpan.
    • \n
  • Menampilkan URL login dan jumlah login yang tersedia di tooltip tombol toolbar dan ikon statusbar.

Kamu bisa gunakan Manajer Profil Inbuilt untuk mengelola beberapa profil pengguna di firefox. Anda dapat memiliki beberapa profil Firefox, masing-masing berisi kumpulan informasi pengguna yang terpisah, termasuk kata sandi yang disimpan untuk setiap pengguna. Saat Anda membuat profil baru, meluncurkan dengan profil baru akan bertindak sebagai Pemasangan Baru Firefox dan akan meminta Simpan Kata Sandi.

\n\n

Untuk membuat profil baru, Tekan Menang + R , ketik firefox.exe -p dan tekan enter.
\n (Langkah ini hanya diperlukan pertama kali, nanti Anda dapat meluncurkan seperti yang Anda lakukan sekarang.)

\n\n

Jika Anda ingin Firefox menanyakan setiap kali Anda memulainya, profil mana yang harus digunakan, lalu Ingat untuk Hapus centang itu Jangan tanya saat memulai kotak centang,

\n\n

Penting: Sebelum Anda dapat memulai Manajer Profil, Firefox harus ditutup sepenuhnya.

\n\n

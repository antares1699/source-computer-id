Ada cara untuk membuat jenis sel Excel dari kotak pilihan dropdown. Anda hanya akan pergi ke Data-> Validasi dengan sel atau rentang sel yang dipilih, pilih daftar sebagai tipe data dan tentukan daftar nilai.

\n\n

Googling memberi saya contoh makro seperti itu tetapi itu jauh dari pengalaman pencarian multi nilai nyata ketika Anda hanya menandai item yang dipilih dengan kotak centang di sebelah kiri atau memilih nilai dalam semacam jendela sembulan. \n\n

Apa cara tampilan paling asli untuk membuat sel banyak pilihan di Excel?

Lihat bagian Formulir di Bantuan Excel untuk topik "Menambahkan kotak daftar atau kotak kombo ke lembar kerja".

\n\n

Dari halaman itu (mengacu pada kotak daftar kontrol Formulir):

\n\n
\n

Catatan: Jika Anda menyetel tipe pemilihan ke\n Multi atau Perluas, sel itu\n ditentukan dalam kotak tautan sel kembali\n nilai 0 dan diabaikan. Multi\n dan Perluas tipe seleksi membutuhkan\n penggunaan Microsoft Visual Basic untuk\n Kode Aplikasi (VBA). Dalam ini\n kasus, pertimbangkan untuk menggunakan daftar ActiveX\n kontrol kotak.

\n
\n\n

Dan mengacu pada kotak daftar kontrol ActiveX:

\n\n
\n

Untuk membuat kotak daftar dengan beberapa\n seleksi atau seleksi diperpanjang\n diaktifkan, gunakan properti MultiSelect.\n Dalam hal ini, properti LinkedCell\n mengembalikan nilai # N / A. Anda harus menggunakan VBA\n kode untuk memproses kelipatan\n pilihan.

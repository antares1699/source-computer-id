Di Powershell, apakah ada cara, setelah mengetik perintah yang panjang atau rumit, untuk memasukkan perintah itu ke dalam buffer riwayat, tetapi tidak menjalankannya?

\n\n

Misalnya, Anda mulai mengetik git commit, di dalam posh-git: direktori: 

git commit -m \"Added the library 'ASDF' and did some initial integration into the project\" [pause]\n

Dan tepat ketika Anda mendapatkan ketikan itu, saya berpikir ... Eh, saya perlu melakukan sesuatu terlebih dahulu, seperti menambahkan file tambahan untuk melakukan sesuatu terlebih dahulu, atau membuat direktori atau sesuatu (situasi yang dibuat-buat, saya menyadari Powershell akan buat otomatis dan saya menyadari git dapat "mengubah komit".) Jadi saya ingin menyimpan pesan komit itu ke dalam buffer riwayat, tetapi tanpa menjalankannya, untuk diingat nanti setelah saya katakan, buat file tambahan atau lakukan sesuatu sebelum.

\n\n

Menurut artikel ini, Tombol Pintas PowerShell Windows: http://technet.microsoft.com/en-us/library/ee176868.aspx , tampaknya tidak ada penekanan tombol untuk melakukan ini.

\n\n

Tombol ESC (dan Ctrl + C) akan membatalkan perintah, dan menghapusnya ... Yang saya butuhkan adalah menyimpan ke riwayat, untuk dijalankan nanti, sehingga pengetikan saya tetap terjaga. Apakah ada penekanan tombol untuk ini?

Ini tampaknya berfungsi juga, setelah beberapa bermain dengan pemanggilan: 

$VariableName = \"Windows Command\"\n

Kemudian, untuk mengeksekusi: 

Invoke-Expression \"$VariableName\"\n

Contoh dengan Ping:

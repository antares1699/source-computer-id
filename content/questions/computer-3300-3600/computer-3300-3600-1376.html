Di PowerPoint 2013, ini sangat sederhana:

\n\n
    \n

  1. Di Presenter View, klik Show Taskbar (pojok kiri atas)

    2. \n

  2. Ketika taskbar terbuka di bagian bawah layar, klik pada tab PowerPoint,

    3. \n

  3. Dalam tiga tampilan yang diperlihatkan (Editor, Presenter dan SlideShow), klik pada tampilan Editor.

    4. \n

  4. Jika Anda ingin audiens melihat perubahan Anda saat Anda membuatnya, tampilkan slide yang sama di semua 3 tampilan.

    5. \n

  5. Jika Anda tidak ingin audiens melihat perubahan hingga Anda selesai, klik ikon layar Hitam di Tampilan Penyaji (Monitor dengan garis diagonal melintasi), sebelum Anda memulai perubahan, ATAU ubah ke slide lain dalam tampilan Penyaji.

    6. \n

  6. Tayangan slide akan menampilkan slide Anda yang diubah di layar besar, saat Anda membukanya.

    7. \n
\n\n

Selamat PowerPoint!

Ini sangat sederhana, terutama karena Anda mengatakan Anda ingin mengubah slide yang saat ini tidak terlihat (yang bisa menjadi rumit karena bug di beberapa versi PPT).

\n\n

Tambahkan ini ke modul VBA di presentasi Anda.
Anda harus menyimpan presentasi sebagai PPTM atau PPSM daripada PPTX / PPTX.

\n\n

Ikuti instrux yang disertakan sebagai komentar: 

Option Explicit\n\n' We'll modify slide #4 ... change as needed\n' Make sure that the slide has no empty content or picture placeholders on it\nConst lSlideNum As Long = 4\n\nSub AddAnImage()\n    ' add a shape to any slide you like\n    ' assign the shape an Action Setting of Run Macro: AddAnImage\n\n    Dim oSl As Slide\n    Dim oSh As Shape\n\n    Set oSl = ActivePresentation.Slides(lSlideNum)\n\n    ' bring in the image; setting width/height to -1 ensures that you\n    ' don't distort it\n    Set oSh = oSl.Shapes.AddPicture(\"c:\\temp\\photo.jpg\", msoFalse, msoTrue, 0, 0, -1, -1)\n\n    With oSh\n        .LockAspectRatio = msoTrue ' to make sure it stays undistorted\n        ' change its position/size as you wish\n        ' for example, let's make it the full width of the slide:\n        .Width = ActivePresentation.PageSetup.SlideWidth\n    End With\n\nEnd Sub\n

Option Explicit\n\n' We'll modify slide #4 ... change as needed ' Make sure that the slide has no empty content or picture placeholders on it Const lSlideNum As Long = 4\n\nSub AddAnImage()\n    ' add a shape to any slide you like\n    ' assign the shape an Action Setting of Run Macro: AddAnImage\n\n    Dim oSl As Slide\n    Dim oSh As Shape\n\n    Set oSl = ActivePresentation.Slides(lSlideNum)\n\n    ' NEW:  Delete any existing image we may have added previously\n    On Error Resume Next\n    Set oSh = oSl.Shapes(\"MagicImage\")\n    If Err.Number = 0 Then\n       oSh.Delete\n    End If\n\n    ' bring in the image; setting width/height to -1 ensures that you\n    ' don't distort it\n    Set oSh = oSl.Shapes.AddPicture(\"c:\\temp\\photo.jpg\", msoFalse, msoTrue, 0, 0, -1, -1)\n\n    With oSh\n        ' NEW:  Name it so we can locate it easily later:\n        .Name = \"MagicImage\"\n        .LockAspectRatio = msoTrue ' to make sure it stays undistorted\n        ' change its position/size as you wish\n        ' for example, let's make it the full width of the slide:\n        .Width = ActivePresentation.PageSetup.SlideWidth\n    End With\n\nEnd Sub\n

Saya biasa membuka jendela Terminal baru dengan versi serupa dengan tombol pintas yang dipetakan oleh FastScripts. 

tell application \"Chromium\" to activate\n\ntell application \"System Events\"\n tell process \"Chromium\"\n  click menu item \"새 창\" of menu \"파일\" of menu bar 1\n end tell\nend tell\n

Anda harus mengganti "새 창" dan "파일" di atas dengan label menu bahasa Anda,\ndan "Chromimum" dengan "Chrome" jika Anda tidak menggunakan foto seperti saya.

Saat ini dukungan AppleScript di Chrome sangat terbatas (di luar skrip GUI standar yang asli untuk Mac OS X) tetapi diharapkan meningkat di Milestone 5. (Tanggal rilis tidak yakin) Lihat http://code.google.com/p/chromium/issues/detail?id=27468 untuk lebih jelasnya.

\n\n

Jadi saat ini tidak ada cara yang andal untuk membuat jendela baru jika Chrome sudah berjalan. Jika tidak berjalan, mudah digunakan: 

tell application \"Chrome\"\n    activate\nend tell\n

atau 

open /Applications/Chrome.app\n

Catatan berjalan: 

open /Applications/Google\\ Chrome.app/Contents/MacOS/Google\\ Chrome \"URL HERE\"&\n

adalah cara yang bagus untuk menjalankan banyak contoh aplikasi yang membuat pemukulan preferensi dan cache Anda menjadi sangat mudah. Saya tidak menyarankan menggunakan ini.

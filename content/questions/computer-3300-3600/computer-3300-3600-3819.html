Saya sedang menyiapkan PC Windows 8 baru milik ayah saya. Di beberapa titik setelah menggunakan utilitas Transfer Mudah untuk mentransfer file-nya dari XP dan menginstal Norton Internet Security - Saya tidak tahu apakah itu terkait, mereka hanya variabel yang saya ketahui, IE10 tidak lagi mencari dari bilah alamat - itu memperlakukan semuanya seperti alamat yang sebenarnya, jadi mengetikkan "anak kucing lucu" akan membuatnya benar-benar mencoba masuk ke http titik tebas garis miring lucu% 20kittens (SU tidak suka saya benar-benar memasukkan tautan ke situs yang tidak ada). \n\n

Saya tidak melihat "pencarian dari bilah alamat" sederhana atau pengaturan serupa di pengaturan IE10, tapi mungkin saya hanya melewatkan sesuatu yang sederhana seperti itu.

\n\n

Saya juga menghadapi masalah serupa. Bilah dan ikon penyedia penelusuran tidak akan muncul di bawah menu tarik-turun di bilah alamat. Juga menggunakan? atau Ctrl + E tidak akan menelusuri.

\n\n

Saya bisa mendapatkan kembali pencarian di bilah alamat dengan mengaktifkan pengaturan Search in the address bar di bawah Tools > Manage add-ons > Search Providers.

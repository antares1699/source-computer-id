Dari Konfigurasi Alpine :

\n\n
\n

Mulailah dengan membuka "Setup\n Konfigurasi "layar (M S C) dan temukan\n bagian Preferensi Folder. Dibawah\n bagian ini, set\n aktifkan-folder-masuk dan mulai ulang\n alpine agar perubahan diterapkan.\n Sekarang buka Pesan Masuk Anda\n Folder (F Folder Masuk), tapi\n jangan buka folder INBOX Anda.

\n \n

Sekarang tiba bagian menyenangkan.

\n \n

Karena Anda ingin menambahkan folder baru,\n tipe A. Anda akan disajikan dengan file\n baris perintah yang menanyakan "Nama\n server berisi folder tambahan "dan\n empat opsi di bawah itu. Kami\n tertarik dengan opsi. Sejak\n Anda akan menarik email dari\n server pop3 jarak jauh, Anda mau\n mengkonfigurasi mail drop, yaitu file\n opsi terakhir, jadi tekan Ctrl W. Anda akan\n sekarang ditanyakan nama suratnya\n drop server (server yang Anda tarik\n surat dari). Masuk ke server\n (dan nomor port jika diinginkan), file\n nama pengguna (yang mungkin termasuk atau tidak termasuk\n bagian dari alamat server), dan pop3.\n Setelah selesai, Anda akan mendapatkan sesuatu\n yang terlihat seperti berikut ini:

\n \n

pop.server.tld:port/user=username@server.tld/pop3

\n \n

Jika Anda menggunakan koneksi aman,\n Anda perlu menambahkan dua hal ke file\n masuk. Pertama, tambahkan ssl ke bagian akhir.\n Ketika itu selesai (dan dengan asumsi Anda\n tidak mau, atau mengalami kesulitan\n pengaturan, validasi sertifikat),\n tambahkan novalidate-cert tepat setelah\n mail drop server. Setelah Anda selesai,\n Anda harus memiliki sesuatu itu\n menyerupai berikut ini:

\n \n

pop.server.tld:port/novalidate-cert/user=username@server.tld/pop3/ssl

\n \n

Sekarang tekan Enter. Anda sekarang akan diminta\n folder yang ingin Anda salin suratnya\n dari; jawaban: Inbox.

\n \n

Berikutnya: "Nama server yang akan dimuat\n folder tujuan ". Biarkan kosong,\n dan lanjutkan.

\n \n

Sekarang untuk "Folder untuk menyalin email ke".\n Anda dapat menamai ini apa pun yang Anda inginkan,\n tapi saya biasanya menggunakan nama pengguna\n akun email apa pun yang saya atur\n mail drop up untuk.

\n \n

Dan saya melakukan hal yang sama seperti di atas untuk\n "Nama panggilan untuk folder" nama pengguna "".

\n \n

Dan sekarang Anda memiliki mail drop baru\n map. Untuk menyiapkan lebih banyak, ulangi sebagai\n dibutuhkan.

\n
\n\n

Pengaturan email Alpine juga memiliki petunjuk penyiapan Alpine, termasuk cara menyiapkan Gmail menggunakan POP3 (termasuk gambar):

\n\n
\n

Ini adalah contoh penggunaan POP3 dari\n Gmail. Anda harus mengaktifkan POP3\n dari akun Gmail Anda di bawah\n Pengaturan → Penerusan dan POP / IMAP.

\n \n

Untuk membuat folder masuk baru untuk\n POP, gunakan Menu Utama → Folder\n Daftar → Folder-Masuk → Tambah.

\n \n

Ini akan meminta Anda untuk Nama\n server berisi folder tambahan:

\n \n

Name of server to contain added folder: pop.gmail.com:995/novalidate-cert/user=username@gmail.com/pop3/ssl

\n \n

Ini kemudian akan meminta untuk meminta file\n 'Folder di' pop.gmail.com… "untuk menambahkan:

\n \n

Folder on \"pop.gmail.com..\" to add:\n INBOX

\n \n

Isi nama panggilan:

\n \n

Nickname for folder \"INBOX\" : INBOX at\n gmail.com

\n \n

Ini kemudian akan terhubung ke\n server gmail dan akan meminta file\n namapengguna@gmail.com sandi. Masukkan itu,\n dan INBOX akan terbuka.

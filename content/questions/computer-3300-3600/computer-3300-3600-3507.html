Bagaimana jendela tertentu ditutup dari baris perintah, tanpa mematikan aplikasi pengendali mereka?

\n\n

Pertanyaan berdasarkan contoh agar lebih jelas:

\n\n

Saya baru-baru ini memiliki aplikasi yang menjalankan malapetaka dan membuka banyak jendela firefox. Saya perlu menutupnya, tanpa menghentikan firefox (karena ini hanya akan memulihkan jendela melalui pemulihan sesi firefox). Bagaimana windows bisa "baru saja ditutup" dari shell, seperti menekan tombol tutup?

Cobalah dengan sederhana 

wmctrl -c firefox\n

Dari wmctrl halaman manual : Berinteraksi dengan X Window Manager yang kompatibel dengan EWMH / NetWM.

\n
    \n
  • Anda bisa lebih ketat dengan opsi -F:
    \n

    \n
    \n
    • \n
  • Untuk mematikan jendela tertentu Anda dapat mencantumkannya dengan wmctrl -l dan kemudian menutupnya dengan wmctrl -ic ID.

Kombinasi dari xlsw dan xdotool melakukan triknya.

\n\n

xlsw mencantumkan jendela menurut judul dan jenisnya, xdotool melakukan tindakan padanya. Pada prinsipnya, xdotool juga dapat mencari nama jendela, tetapi tidak menemukan jendela sembul.

\n\n

Pada bash: 

xlsw | grep 'Firefox/Popup' | awk '{print $1}' | \\\n  while read _windowID; do xdotool windowclose \"${_windowID}\"; done\n

Penjelasan:

\n\n

xlsw mengeluarkan daftar ID jendela bersama dengan informasi tambahan. Outputnya terlihat seperti: 

[...]\n0x04800001  u--  Pale moon/pale moon  Pale Moon\n0x04800092  uio  NA           Pale Moon\n0x04800093  u-o  Pale moon/palemoon  Pale Moon\n0x04800099  u-o  Pale moon/palemoon  Pale Moon\n0x04828D21  u-o  Pale moon/Popup  Pale Moon\n0x0489C5D6  u-o  Pale moon/Popup  Pale Moon\n0x03600001  ui-  NA           NA\n0x04604E77  --o  Claws-mail/claws-mail  claws-mail\n0x04604EAC  --o  Claws-mail/claws-mail  claws-mail\n0x04604F22  --o  Claws-mail/claws-mail  claws-mail\n0x05C00001  uio  NA           NA\n0x05E00001  u--  NA           NA\n0x05200007  ui-  NA           NA\n0x05E00002  u--  Wine/explorer.exe  \n[...]\n

Kemudian, dengan perintah grep, jendela popup firefox akan dipilih (dalam contoh di atas semuanya sudah hilang). awk mencetak entri pertama, yang merupakan ID jendela, dan kemudian xdotool windowclose akan menutup jendela (sesuatu Suka menekan tombol tutup) tanpa mematikan aplikasi pengendali itu. (xdotool windowkill akan mematikan aplikasi.)

\n\n

xdotool juga memiliki build in search: xdotool search 'firefox' windowclose juga akan menutup semua jendela dengan 'firefox' di judulnya, tetapi saya tidak membuatnya berfungsi untuk membedakan 'Popup'.

Saya perhatikan di profil Anda bahwa Anda adalah pengguna di Ask Ubuntu, jadi saya akan membagikan metode yang saya gunakan untuk memilih dan menutup jendela yang terbuka dari shell di Ubuntu. Metode yang sama juga berfungsi pada banyak distrubsi Linux lainnya.

\n\n
    \n

  • wmctrl -mlpx mencantumkan jendela yang terbuka dengan ID jendela dan deskripsi dari setiap jendela. Untuk Firefox, deskripsi adalah judul tab yang memiliki fokus dan jendela pop-up terdaftar sebagai jendela terpisah.

    • \n

  • wmctrl -ic <window-id> menutup jendela yang memiliki id = window-id dari terminal.

    • \n
\n\n

Jika Anda belum menginstal wmctrl, ini dapat diinstal dengan sudo apt install wmctrl di semua versi Ubuntu yang saat ini didukung tanpa memerlukan penginstalan perangkat lunak lain dari GitHub. Ada juga .rpm dari wmctrl dalam database paket Fedora.

Anda perlu mengunduh gaya referensi dari BibWord

\n\n

https://github.com/mback2k/bibword-styles Klik pada tombol hijau (pojok kanan - unduh)

\n\n

Unduh styles.zip

\n\n

Buka zip file dari styles.zip, Anda akan melihat beberapa file di folder (ini adalah gaya referensi).

\n\n

Pasang gaya referensi. Harus menutup Microsoft Word sepenuhnya sebelum melanjutkan.

\n\n

Buka folder Microsoft Word

\n\n

PC ini → Disk C: → Program Files → Microsoft Office → Office 12 → Bibliografi → StyleSalin gaya referensi dari file zip yang telah dibuka ke folder Style

\n\n

Tip: Anda dapat menemukan banyak ide berguna dari http://bibword.codeplex.com/ Di bagian Diskusi.

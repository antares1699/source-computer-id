Saya sering menulis teks dengan format seperti ini di Vim— 

- talking point 1\n\n- talking point 2 ....\ncontinue on point 2\n

\n\n

Idealnya, saya berharap Vim dapat menyelaraskan secara otomatis untuk saya seperti: 

- talking point 1\n\n- talking point 2 \n  continue on point 2\n

Apakah ini mungkin?

Saya punya ini di .vimrc saya: 

set comments=s1:/*,mb:*,ex:*/,://,b:#,:%,:XCOMM,n:>,fb:-,fb:[+],fb:[x],fb:[-]

Jika saya ingat benar, tambahkan baris ini di .vimrc Anda dan pekerjaan akan selesai: 

set comments +=fb:-

Untuk penjelasan rinci coba: 

:help comments

Jika Anda memiliki flag n yang disetel pada formatoptions (mis. Dengan set fo+=n), Vim sudah tahu cara memformat daftar dengan poin numerik. formatlistpat (nama pendek flp) adalah ekspresi reguler yang digunakan Vim untuk mencocokkan ini, jadi yang Anda butuhkan adalah menyempurnakan ekspresi reguler tersebut. Ini akan membantu Anda (tetapi hanya menambahkan dukungan untuk peluru -): 

set formatlistpat=^\\\\s*\\\\(\\\\d\\\\+[\\\\]:.)}\\\\t\\ ]\\\\|-\\\\)\\\\s*\n

Maaf untuk sakit kepala backslash di sana. Melakukan set flp? menunjukkan lebih jelas seperti apa regex itu: 

formatlistpat=^\\s*\\(\\d\\+[\\]:.)}\\t ]-\\)\\s*\n

Untuk informasi lebih lanjut, lihat posting ini .

Saya punya pertanyaan yang sangat sederhana. Bagaimana cara mengkonfigurasi firewall untuk memblokir pembaruan Adobe Illustrator CC 2014 di Windows 7?

","source":"https://superuser.com/questions/837616/how-can-i-configure-my-firewall-to-block-abobe-illustrator-cc-2014-updates-on-wi","author":"John Sonderson","replies":[],"answers":[{"answer":"

Anda perlu membuat aturan Windows Firewall. Pergi ke Panel Kontrol -> Semua Item Panel Kontrol -> Windows Firewall . Klik Pengaturan lanjutan di sisi kanan, klik Aturan keluar di sisi kiri, lalu klik Aturan baru di kanan.

\n\n

\n\n

Di jendela berikutnya, pastikan Anda punya Program dipilih dan klik Lanjut . Pilih tombol untuk Jalur program ini: dan klik Jelajahi . Dari sini, Anda harus menavigasi ke file .EXE untuk Illustrator. Ini akan di bawah C: \\ Program Files (x86) (tanpa x86 jika Anda tidak menjalankan Windows 64-bit) dan mungkin dalam sub-folder Adobe.

\n\n

Setelah Anda menemukan file .EXE, klik Lanjut , Blokir koneksi , terapkan ke Domain / Pribadi / Publik jaringan, beri nama, dan klik Selesai .

Ini untuk FFmpeg ( Lihat disini untuk versi Windows).

\n\n

Pertama, siapkan gambar Anda agar diberi nama image-001.jpg, image-002.jpg, dan lain-lain. Taruh di satu folder.

\n\n

Sekarang, gunakan perintah berikut: 

ffmpeg -y -loop 1 -f image2 -r 0.5 -i image-%03d.jpg -s:v 1280x720 -b:v 1M \\\n       -i soundtrack.mp3 -t 01:05:00 -map 0:0 -map 1:0 out.avi\n
\n\n

Anda tentu saja dapat mengubah parameternya. Berikut penjelasan tentang apa yang mereka lakukan:

\n\n
    \n

  • -loop_input - memutar gambar. Nonaktifkan ini jika Anda ingin menghentikan pengkodean ketika semua gambar digunakan atau soundtrack selesai.

    • \n

  • -r 0.5 - setel framerate ke 0,5, yang berarti setiap gambar akan ditampilkan selama 2 detik. Ambil saja kebalikannya, misalnya jika Anda ingin setiap gambar bertahan selama 3 detik, setel ke 0,33.

    • \n

  • -i image-%03d.jpg - gunakan file input ini. %03d berarti akan ada tiga digit angka untuk gambar tersebut.

    • \n

  • -s 1280x720 - mengatur ukuran bingkai keluaran.

    • \n

  • -b 1M - mengatur bitrate. Anda menginginkan 500MB untuk satu jam, yang sama dengan 4000MBit dalam 3600 detik, dengan demikian bitrate kira-kira 1MBit / s sudah cukup.

    • \n

  • -i soundtrack.mp3 - gunakan file soundtrack ini. Bisa dalam format apapun.

    • \n

  • image-001.jpg - atur panjang keluaran dalam format hh:mm:ss.

    • \n

  • out.avi - buat file keluaran ini. Ubah sesuka Anda, misalnya menggunakan wadah lain seperti MP4.

    • \n

Saya mencoba membuat video menggunakan banyak gambar dan trek suara, Ikuti langkah-langkah ini, entah bagaimana berfungsi untuk saya:

\n\n
    \n

  1. Buat file yang mencantumkan jalur gambar dan durasi untuk setiap gambar\nimage-list.txt 

    file 'imgs/114_1.png'\nduration 9\nfile 'imgs/114_2.png'\nduration 7\nfile 'imgs/114_2.png'\n

    Catatan: Ulangi gambar terakhir dua kali dan jangan masukkan durasi untuk entri terakhir.

    2. \n

  2. Buat file lain yang berisi jalur file audio\naudio-list.txt 

    file 1.mp3\nduration 9\nfile 2.mp3\nduration 6\n
    3. \n

  3. ffmpeg ajaib! 

    ffmpeg -f concat -safe 0 -i img-list.txt -f concat -safe 0 -i audio-list.txt -c:a aac -pix_fmt yuv420p -crf 23 -r 24 -shortest -y -vf \"scale=trunc(iw/2)*2:trunc(ih/2)*2\" video.mp4\n
    4. \n

TERPECAHKAN

\n\n

Buat file .ftpaccess di direktori yang harus bersifat read-only untuk akun namapengguna@domainanda.com.

\n\n

Isi dari file .ftpaccess: 

<Limit WRITE>\nDenyUser username@yourdomain.com\n</Limit>\n

Itu saja.

\n\n

Tolong dicatat:

\n\n
    \n

  • Perintah dalam file .ftpaccess bersifat rekursif (yaitu mempengaruhi semua file\ndi direktori saat ini dan setiap subdirektori).

    • \n

  • Syaratnya, server mendukung ProFTPD.

    • \n
\n\n

Lebih lanjut tentang ProFPTD di sini:

\n\n

http://www.proftpd.org/docs/howto/Limit.html

\n\n

Peringatan: ProFTPD cukup kuat. Jika Anda melakukan kesalahan dan karenanya Anda tidak dapat mengedit file Anda melalui FTP lagi, Anda masih dapat mengedit file .ftpaccess melalui baris perintah.

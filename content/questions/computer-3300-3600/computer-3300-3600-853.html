Anda dapat mencoba mengakses ini melalui Sistem Halaman

\n\n

Windows + Pause/Break akan menampilkan Sistem Halaman

\n\n

Lalu klik Pengaturan sistem lanjutan
\n

\n\n

Ini mungkin membuat Anda melakukannya auth pada titik ini - tetapi bahkan jika tidak, itu harus memastikan Anda melanjutkan sebagai administrator

\n\n

Klik Variabel Lingkungan tombol di bagian bawah
\n

\n\n

Anda sekarang harus berada di halaman variabel lingkungan sebagai Admin.

\n\n
\n\n

Jika mengakses melalui menu start:
(Baru saja perhatikan jawaban lain dari Niall895 kurang lebih sama seperti di bawah ini)

\n\n

Saat Anda mengetik ' lingkungan Hidup 'ke menu mulai, pastikan Anda memilih Edit Sistem variabel lingkungan , daripada Edit variabel lingkungan untuk akun Anda </strong> .

\n\n

Jika tidak, mereka akan berwarna abu-abu seperti pada tangkapan layar Anda.

\n\n

(Ini mungkin atau mungkin tidak membuat Anda mengetik ulang kredensial Anda terlebih dahulu)

\n\n

\n\n
\n\n

Mungkin Anda memiliki pengaturan UAC yang berbeda pada contoh ini

\n\n

(Ketik UAC ke dalam menu mulai pada masing-masing dan lihat pengaturannya)

\n\n

\n\n

Meskipun mengubah ini biasanya menghilangkan pop-up prompt

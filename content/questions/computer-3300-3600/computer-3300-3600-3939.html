Beberapa bulan yang lalu, saya memperoleh kunci produk Windows 7 Ultimate dan memperbarui dari Home Premium ke Ultimate. Nah, sekarang saya mendapatkan pesan "Salinan Windows ini tidak asli". Saya memiliki kunci produk Home Premium yang saya gunakan sebelum saya meningkatkan ke Ultimate. Saya ingin tahu bagaimana saya dapat menurunkan versi dari Ultimate ke Home Premium tanpa memiliki CD dan hanya kunci produk. Jika membantu, itu adalah OS 64-bit. Terima kasih sebelumnya!

Anda memerlukan disk untuk melakukan ini. Google untuk digital river windows 7 iso dan unduh ISO, gunakan alat yang disebut Utilitas Penghapusan ei.cfg untuk mengubah ISO ke Home Premium.

\n\n

Sekarang jalankan Tool Penurun Windows 7 (baca komentar di sana untuk langkah-langkah khusus pada sistem 64bit),

\n\n

\n\n

klik "Downgrade to Home Premium", masukkan DVD Windows 7 dan lakukan Peningkatan / Perbaikan Instalasi di Tempat :

\n\n
    \n
  • Di Windows 7 yang sedang berjalan, pasang ISO atau masukkan DVD
    • \n
  • Jalankan setup dari DVD dan pilih "Upgrade"
    • \n
\n\n

Ini menurunkan Win7 Ultimate ke Home Premium.

\n\n

CATATAN: Ini tidak secara resmi didukung oleh MS, tetapi berfungsi.

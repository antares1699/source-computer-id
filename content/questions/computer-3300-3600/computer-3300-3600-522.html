Postingan ini bisa jadi solusinya @ http://ubuntuforums.org/archive/index.php/t-493848.html

\n\n
\n

Jika Anda perlu mematikan firefox, Anda dapat mencoba masuk ke System-> Adminstration-> System Monitor dan pilih tab proses. Cari firefox dan klik kanan di atasnya. Ini akan memberi Anda opsi untuk menghentikan proses. Jika tidak berhasil, Anda dapat mencoba masuk ke Terminal dan mengetik:
ps -ef | grep firefox

\n \n

Jika firefox sedang berjalan, itu akan muncul sebagai proses dan memiliki nomor yang menyertainya. Ini akan menjadi angka pertama setelah nama pengguna. Sebagai contoh:

\n \n

jayhawk @ jayhawk: / $ ps -ef | grep firefox
jayhawk 8137 26283 0 14:21 pts / 0 00:00:00 grep firefox
jayhawk 24213 1 0 07:22? 00:00:00 / bin / sh / usr / bin / firefox-granparadiso

\n \n

Dalam skenario ini, nomor prosesnya adalah 24213 untuk firefox (Saya menjalankan versi alfa firefox sehingga nama firefox Anda akan berbeda). Ketik saja:
bunuh 24213

\n \n

Kemudian periksa kembali dengan menggunakan perintah ps -ef grep firefox. Jika masih ada, Anda dapat menggunakan:
sudo kill -9 24213

\n \n

Dan itu seharusnya membunuhnya.

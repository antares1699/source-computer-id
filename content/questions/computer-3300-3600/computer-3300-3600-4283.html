Sepertinya Anda menggunakan Windows. Anda perlu mengasosiasikan IDLE sebagai editor untuk file .py. Saya tidak tahu tentang Windows 7 tetapi dengan XP:

\n\n
    \n
  • Buka Windows Explorer, klik menu 'Tools', pilih 'Folder Options'
    • \n
  • Pilih tab 'Jenis file'.
    • \n
  • Gulir ke bawah dan pilih "py Python File" dalam daftar jenis file Terdaftar.
    • \n
  • Klik tombol Advanced untuk membuka kotak dialog Edit File Type.
    • \n
  • Jika Anda melihat "Edit dengan IDLE", klik tombol Edit dan pastikan kotak "Aplikasi yang digunakan" bertuliskan "C: \\ PythonXX \\ pythonw.exe" "C: \\ PythonXX \\ Lib \\ idlelib \\ idle.pyw" - e "% 1" (menggantikan XX dengan versi Python Anda.
    • \n
  • Jika Anda tidak melihat "Edit dengan IDLE", klik tombol New, masukkan "Edit dengan IDLE" di kotak Tindakan dan string "Aplikasi yang digunakan" yang sama seperti di atas, lalu centang kotak DDE.

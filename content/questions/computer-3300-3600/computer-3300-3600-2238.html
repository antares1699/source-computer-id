Saya bekerja di lingkungan komputasi bersama dan pengaturan default adalah r-x untuk grup dan lainnya; terserah pengguna untuk mengubahnya. Saya dapat mengubah dan mengubah izin untuk semua file. Namun setiap file baru yang dibuat semuanya memiliki izin default. Apakah ada cara untuk mengubahnya sehingga saya tidak perlu melakukan chmod setiap saat atau menjalankan chmod sebagai cronjob?

","source":"https://superuser.com/questions/251306/can-i-as-a-user-change-the-default-chmod-settings-for-my-account","author":"user70365","replies":[],"answers":[{"answer":"

Pengaturan yang Anda cari disebut umask , dan itu juga nama perintah yang mengubahnya. Untuk membuat perubahan terus-menerus, tambahkan perintah umask ke file startup shell Anda - mungkin bernama .profile atau .bash_profile, di direktori home Anda; jika Anda tidak memiliki file seperti itu, posting output dari perintah ini: 

$ grep $LOGNAME /etc/passwd\n$ (cd; ls -ld .??*)\n

Anda dapat menormalkan volume file suara dengan SoX cukup mudah.\nIni dijuluki pisau Swiss Army untuk program pengolah suara dan memiliki banyak pilihan.

\n\n

Ada juga normalize dan digunakan seperti ini: 

normalize -b file1.wav file2.wav file3.wav\n

Saya belum pernah menggunakan normalisasi diri saya sendiri.

\n\n

Namun SoX dan normalize bekerja dengan memodifikasi data audio.\nJika Anda tidak mau, Anda bisa mencobanya mendapatkan kembali . Itu masih memodifikasi file audio tetapi hanya meta data.

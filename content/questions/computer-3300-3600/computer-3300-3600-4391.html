Karena Ctrl + A dan Ctrl + F9 tidak akan membahas header / footer, dan mungkin cerita lain, VBA atau kode lain akan diperlukan, yang mungkin merupakan pertanyaan yang paling cocok untuk Stack Overflow.

\n\n

Kode VBA mungkin terlihat seperti: 

Sub updateFieldsIncludeHeadersFooters()\n    Dim sec As Section\n    Dim hdrftr As HeaderFooter\n\n    ActiveDocument.Fields.Update 'address the fields in the main text story\n\n    'now go through headers/footers for each section, update fields per range\n    For Each sec In ActiveDocument.Sections\n        For Each hdrftr In sec.Headers\n            hdrftr.Range.Fields.Update\n        Next\n        For Each hdrftr In sec.Footers\n            hdrftr.Range.Fields.Update\n        Next\n    Next\nEnd Sub\n

Saya tidak akan merekomendasikan mengambil alih acara Word's Save untuk menjalankan ini secara otomatis, melainkan mengaitkannya ke tombol atau memiliki cara lain bagi pengguna untuk secara eksplisit menyebutnya.

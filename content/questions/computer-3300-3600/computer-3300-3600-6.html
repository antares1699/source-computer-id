Langkah pertama: Pertama-tama kita perlu mengambil profil Firefox dari komputer lama atau hard drive. Di komputer lama, telusuri (Windows XP) C: \\ Documents and Settings \\\\ Application Data \\ Mozilla \\ Firefox \\ Profiles; (Windows 7,8, 8.1) C: \\ Users \\\\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles. Anda akan menemukan di bawah folder Profiles profil xxxxxx.default.

\n\n

Salin folder profil Firefox ke flash drive atau lokasi jaringan.

\n\n

Langkah Kedua: Buka komputer baru Anda. Pasang Mozilla Firefox. Jika Anda menjalankan Windows 7 atau 8 / 8.1, buka C: \\ Users \\\\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles. Di bawah folder Profil, Anda akan menemukan nama profil (biasanya serangkaian angka dan huruf dengan .default di bagian akhir).

\n\n

Pastikan Firefox ditutup dan potong folder default Firefox xxxxxxx.default ke desktop Anda.

\n\n

Langkah ketiga: Pastikan Firefox ditutup di komputer baru. Dapatkan flash drive Anda dan salin profil Firefox dari komputer lama ke folder C: \\ Users \\\\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles (Windows 7, 8 atau 8.1) di komputer baru.

\n\n

Langkah Keempat: Buka desktop Anda dan salin nama profil Firefox komputer baru. Selanjutnya, kembali ke folder profil (C: \\ Users \\\\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles) di komputer baru, dan ganti nama profil Firefox komputer lama ke nama yang sama persis dengan profil Firefox komputer baru. Dalam kasus saya, saya akan mengganti nama profil Firefox baru (35jar636.default) menjadi H9gzaar6.default.

\n\n

Buka Mozilla Firefox dan semuanya akan ada di sana seperti profil lama Anda. Saya harap ini membantu Anda. Terima kasih.

\n\n

Cara termudah adalah menyalin semua file di dalam profil lama Anda dari mesin XP dan meletakkannya di folder profil baru dari Windows 7.

\n\n

Ikuti langkah-langkah berikut untuk ini:\n - arahkan ke C: \\ Documents and Settings \\ user_name \\ Application Data \\ Mozilla \\ Firefox \\ Profiles \\ profile_old.default \\ di komputer XP\n - pilih semua file dari folder itu dan salin ke stik usb\n - buka Firefox di PC baru untuk membuat folder, lalu tutup\n - arahkan ke C: \\ Users \\ user_name \\ AppData \\ Roaming \\ Mozilla \\ Firefox \\ Profiles \\ profile_new.default \\ dan letakkan semua file pada tongkat di sana dan pilih "Ganti semua" untuk menimpa file.

\n\n

Maka Anda harus memiliki semua kata sandi / bookmark / riwayat dari PC lama ke yang baru.

\n\n

Catatan: Folder AppData disembunyikan sehingga Anda harus pergi ke pengaturan explorer dan memilih "Tampilkan File dan Folder Tersembunyi".

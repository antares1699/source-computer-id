Beberapa Bash-love; beri nama apa pun yang Anda suka. Saya menggunakan thescript.sh 

#!/bin/bash\n\necho '[General]'\necho 'StartWithLastProfile=1'\necho ''\n\nn=0\n\nfor file in * ; do\n  if [[ -d "$file" ]] ; then\n    if [[ "$file" =~ .+\\.(.+) ]] ; then\n      echo "[Profile${n}]"\n      echo "Name=${BASH_REMATCH[1]}"\n      echo "IsRelative=1"\n      echo "Path=${file}"\n      if [[ "${BASH_REMATCH[1]}" == default ]] ; then\n        echo "Default=1"\n      fi\n      echo ""\n      let n++\n    fi\n  fi\ndone\n

Pemakaian

\n

Pertama, letakkan di direktori profil mozilla Anda (mis. ~ / .Mozilla / firefox untuk saya). 

chmod +x thescript.sh\n./thescript.sh\n./thescript.sh > profiles.ini\n

Keluaran

Saya tidak dapat menemukan apa pun di Google, jadi saya menulis skrip perl: 

#!/usr/bin/perl\n\nuse strict;\nuse warnings;\n\nmy $dir = glob($ARGV[0] || '~/.mozilla/firefox/');\n\nchdir $dir or die \"Unable to change to dir $dir: $!\";\nopendir my $dh, $dir or die \"Unable to open dir $dir: $!\";\nmy @dirs = grep { /^[^.]/ && -d $_ } readdir $dh;\n\n# print some boilerplate\nprint <<'START';\n[General]\nStartWithLastProfile=0\n\nSTART\n\n# try to sort by oldest first (uses a schwartzian transform)\n# the 'chrome' folder in each profile folder seems to be the oldest file per profile generally\n@dirs = reverse\n        map { $_->[0] }\n        sort { $a->[1] <=> $b->[1] || $a->[0] cmp $b->[0] }\n        map { [ $_, -C \"$_/chrome\" ] }\n        grep { -e \"$_/chrome\" } @dirs;\n\nmy $i = 0;\nforeach my $profile_dir (@dirs) {\n    # folder names are usually of the form zyxwabc.My Profile Name\n    my ($name) = $profile_dir =~ /^[^.]+\\.(.*)/;\n    next if ! $name;\n\n    print <<\"PROFILE\";\n[Profile$i]\nName=$name\nIsRelative=1\nPath=$profile_dir\n\nPROFILE\n\n    $i++;\n}\n

Saya memiliki file batch sederhana yang menetapkan beberapa informasi jalur per sesi yang diperlukan dan kemudian meluncurkan aplikasi: 

SET PATH=%PATH%;C:\\Qt\\473_x64\\lib;C:\\Qt\\jom\nstart C:\\Qt\\qt_creator_220_x64\\bin\\qtcreator.exe\n

Ini berfungsi dengan baik kecuali itu tetap membuka jendela perintah hantu. Jendela perintah ini mati saat aplikasi dimatikan. Agar jelas, jendela perintah yang muncul ketika saya awalnya mengklik dua kali file batch ditutup. Ini adalah jendela baru yang terbuka. Bagaimana cara mencegah jendela kedua ini ditampilkan? Saya hanya ingin mengklik dua kali pada jalan pintas ke file batch dan itu membuka aplikasi seolah-olah saya telah mengklik dua kali pada aplikasi itu sendiri. (Plus, tentu saja, mengatur informasi jalur yang diperlukan.)

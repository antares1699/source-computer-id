Saat saya membuka localhost di browser, saya mendapatkan halaman yang bertuliskan, "Berhasil!" yang bukan halaman yang saya buat jadi saya tidak yakin di mana lokasinya. Apakah ini dari instalasi PHP atau hanya Apache?

\n\n

Ketika saya membuka IDE saya (PHPStorm) dan menjalankan situs web, ini berfungsi jika saya menggunakan port seperti 8080. Jika saya menyetel port ke 80, gagal dan IDE mengatakan: 

/usr/local/php5/bin/php -S 0.0.0.0:80 -t /mywebsitepath/ \n\nFailed to listen on 0.0.0.0:80 (reason: Permission denied)\n

Ketika saya mencari kesalahan khusus itu, yang saya temukan hanyalah bantuan NGINX , tapi saya menggunakan Apache. Apa yang perlu saya ubah agar dapat menjalankan kode saya secara lokal di port 80?

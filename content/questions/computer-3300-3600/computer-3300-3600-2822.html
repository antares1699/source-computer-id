Saya telah mengamati bahwa "tanggal akses" terakhir dari file atau folder di windows 7 tidak diperbarui secara real time. Itu tidak memperbarui bahkan setelah menyegarkan folder, navigasi kembali ke folder dan mengakses kontennya dan memulai ulang OS. Bagaimana cara memperbarui "tanggal terakhir diakses"?

Menurut dokumentasi Microsoft dari Win32 GetFileTime api:

\n\n
\n

NTFS menunda pembaruan ke waktu akses terakhir untuk file hingga satu\n jam setelah akses terakhir. NTFS juga mengizinkan pembaruan waktu akses terakhir\n untuk dinonaktifkan. Waktu akses terakhir tidak diperbarui pada volume NTFS oleh\n default.

\n
\n\n

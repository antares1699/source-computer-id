Tidak, Anda tidak dapat memutakhirkan seri Dual Core ke seri I3 / I5 / I7 yang baru, karena arsitekturnya lebih baru dan minimal memerlukan soket yang berbeda.

\n\n
    \n
  1. Anda dapat mengganti CPU di sebagian besar laptop, karena tidak disolder pada motherboard. Jelas Anda harus tetap berpegang pada soket yang tepat. Dalam kasusnya, Anda dapat mengupgrade dual core 1.6 ke T9900 3.06 GHz misalnya.
    2. \n
\n\n

\n\n

Silakan beli model CPU yang mereka laporkan BEKERJA untuk laptop Anda.

\n\n
    \n

  1. Anda tidak memerlukan sistem pendingin tambahan, produsen laptop biasanya memasang sistem pendingin yang tepat di semua model untuk memangkas biaya dengan jelas. Biayanya lebih mahal untuk mencari atau membuat sistem pendingin yang berbeda daripada hanya menggunakan 1 pada semua model dan hanya mengubah spesifikasi CPU / RAM / HDD.

    2. \n

  2. Jika ingin segera mengupgrade hasil ram terlebih dahulu, seperti yang disarankan oleh member lain. 1GB ram sangat kecil hari ini, dan tongkat ram tidak memerlukan biaya. Meskipun saya harus mengatakan bahwa saya menjalankan server ubuntu dengan java / mysql / apache pada prosesor 2gb dan intel ATOM dan tidak memiliki masalah dengan kecepatan.

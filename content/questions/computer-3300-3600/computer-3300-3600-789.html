Saya telah mengunduh font Consolas dari Microsoft dan menginstalnya di kotak Windows Vista saya. Consolas juga hadir di kotak Windows 7 saya. Ketika saya menggunakan PuTTY, pastikan untuk menggunakan pengaturan yang sama pada kedua mesin, kotak Windows 7 dapat membuat karakter gambar garis / kotak Unicode di Consolas, tetapi kotak Windows Vista tidak bisa.

\n\n

Apa perbedaan yang relevan di antara keduanya? Jika Consolas memiliki karakter, mengapa mereka hanya muncul di satu sistem, dan bukan di sistem lain? Saya masuk ke host jarak jauh yang sama setiap kali, dan saya telah dengan sangat hati-hati memeriksa pengaturan PuTTY untuk memastikan bahwa keduanya sama di kedua mesin.

\n\n

Bagaimana cara membuat Consolas merender karakter gambar garis Unicode di Vista?

Microsoft memodifikasi Consolas dengan Windows 7 untuk memasukkan karakter gambar garis. Alasan Anda tidak melihatnya di sistem Vista adalah karena versi Consolas yang telah Anda instal tidak mengandung karakter tersebut.

\n\n

Untuk menampilkan karakter baris / kotak di sistem Vista Anda, salin font Consolas dari sistem Windows 7 Anda.

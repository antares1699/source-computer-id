Apakah mungkin menggunakan perintah dd untuk membuat image ISO yang dapat di-boot? Saya menggunakan perintah berikut untuk melakukannya: 

dd if=/dev/sda of=/mnt/sdb/my.iso\n

Saya bisa membuat ISO, namun saya tidak bisa boot dengannya. Ketika saya mencoba untuk boot, saya mendapatkan kesalahan ini: 

Operating System not found.\n

/dev/sda mendaftar 4 partisi dan di partisi 1, grub diinstal: filesystem ext3.

Anda tidak dapat mem-boot HDD sebagai DVD atau CD, karena sistem file yang berbeda. Anda mungkin dapat menyalin semua file dengan rsync dan membakar DVD / CD dengan semua file, tetapi Anda tidak bisa hanya dd gambar HDD ke DVD. Kamu bisa gunakan dd untuk menyalin satu hard disk ke hard disk lainnya, dan hard disk kedua akan dapat di-boot.

\n\n

Di sisi lain, Anda dapat mencoba genisoimage ( cdrkit , dari proyek debian) atau mkisofs ( cdrtools , yang merupakan percabangan debian). Saya telah menggunakan genisoimage untuk membuat disk yang dapat di-boot sebelumnya, seperti ketika saya menyalin semua file dari disk debian, mengubah beberapa, dan membuat ulang .ISO. Ini mungkin tidak berfungsi untuk windows dan beberapa OS lainnya, tetapi itu layak dicoba.

\n\n

Saya membuat CD bootable saya mengikuti halaman ini:
\n https://wiki.debian.org/DebianInstaller/Modify/CD

Saat Anda masuk ke shell di sistem Unix biasanya ada beberapa file yang dijalankan di direktori home Anda. Anda akan menggunakannya untuk, misalnya, mengatur alias dan tambahan ke $PATH Anda.

\n\n

Tapi, bagaimana saya tahu persis file mana itu? Bagaimana cara mengetahui dengan tepat file mana yang perlu saya edit (atau tambahkan)?

\n\n

Profil biasanya dijalankan pada setiap login. File login seluruh sistem adalah /etc/profile dan file khusus pengguna biasanya ~/.profile. Beberapa shell memiliki profil dan file rc sendiri, misalnya bash memiliki .bashrc yang dijalankan oleh shell bash non-login dan .bash_profile yang dijalankan oleh login bash kerang. Bashrc seluruh sistem adalah /etc/bash.bashrc

\n\n

Jadi, Anda akan menggunakan .profile untuk hal-hal yang harus dijalankan oleh shell login dan .bashrc (atau yang setara) untuk hal-hal yang harus dijalankan oleh shell non-login (alias, menyiapkan env dan sejenisnya)

\n\n
\n\n

klarifikasi bash:

\n\n

Ada dua jenis shell: shell login dan non-login. Shell login adalah shell yang dijalankan saat pengguna login. Shell non-login adalah semua shell lainnya. Misalnya, ketika Anda masuk melalui .profile atau di konsol, shell yang Anda dapatkan adalah shell untuk masuk.

\n\n

shell login bash dijalankan saat startup:

\n\n
    \n
  • /etc/profile
    • \n
  • File pertama yang ada dari berikut ini: ~/.bash_profile, ~/.bash_login, ~/.profile
    • \n
\n\n

bash shell interaktif non-login dijalankan saat startup:

\n\n
    \n
  • /etc/bash.bashrc
    • \n
  • ~/.bashrc
    • \n
\n\n

Dalam beberapa (kebanyakan?) Gabungan yang datang dengan bash, sumber profil ~/.bashrc, jadi ~/.bashrc dijalankan untuk shell interaktif login dan non-login.

\n\n

Jika sebuah file tidak ada maka akan dilewati.

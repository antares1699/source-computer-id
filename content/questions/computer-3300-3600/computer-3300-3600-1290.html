Apa yang ingin saya lakukan (bagian pendek dari postingan)

\n\n

Saya ingin mengganti tombol kiri dan kanan mouse pada level driver.

\n\n

Artinya, saya ingin memodifikasi atau menimpa driver mouse untuk mengganti tombol mouse kiri dan kanan pada sistem operasi Windows. (Solusinya sebaiknya bekerja dengan Windows 7 dan yang lebih baru.)

\n\n

Mengapa di Bumi saya ingin melakukannya (bagian panjang dari posting)

\n\n

Terminologi

\n\n

Agar singkat dan jelas, saya akan menggunakan yang berikut ini tebal istilah:

\n\n
    \n
  • kiri - tombol kiri mouse fisik
    • \n
  • Baik - tombol mouse fisik kanan
    • \n
  • klik - sinyal yang biasanya dikirim saat Anda menekan kiri , dulu misalnya pilih file, klik tombol, seret dan lepas jendela
    • \n
  • Tidak bisa - sinyal yang biasanya dikirim saat Anda menekan Baik , dulu misalnya buka menu konteks
    • \n
  • Menang-menukar - Panel Kontrol Windows> Pengaturan Mouse yang membuat file kiri api Tidak bisa dan Baik api klik .
    • \n
\n\n

Masalah

\n\n

Saya kidal. Saya menggunakan mouse dengan tangan kiri saya. Sebagai manusia normal, saya picu klik dengan jari telunjuk saya dan Tidak bisa dengan jari tengah saya - yang artinya saya gunakan Baik untuk klik dan kiri untuk Tidak bisa . Biasanya, Menang-menukar cukup untuk itu - dan berfungsi untuk browser, perangkat lunak perkantoran, editor media, IDE pemrograman, dan sejenisnya.

\n\n

Sayangnya, saya juga seorang gamer. Disebabkan oleh keunikan dalam cara DirectX menangani tombol mouse , cukup banyak game - terlepas dari apakah itu rilis anggaran tinggi atau freeware developer tunggal - abaikan Menang-menukar dan (AFAIK) membaca sinyal tombol dari level pengemudi.

\n\n

Ini biasanya berarti bahwa (jika Anda tidak dapat memasang kembali tombol mouse di dalam gim) seluruh gim pada dasarnya tidak dapat dimainkan atau itu (jika Anda bisa rebind) menavigasi menu dalam game membutuhkan banyak konsentrasi dan memiliki risiko konstan gagal melakukan tugas-tugas sederhana secara tidak sengaja.

\n\n

Bagaimanapun, itu agak tidak menyenangkan, menjengkelkan dan membuat menikmati hobi saya sulit, karena itu berarti saya harus meninggalkan game yang saya minati.

\n\n

Solusi yang saya coba

\n\n

Saya membeli Razer Abyssus. Kedua versi drivernya sangat disadap sehingga - untuk mengaktifkan tata letak tombol kidal - saya perlu mengatur rangkaian Kontrol Tombol driver-X-Mouse Abyssus-tua. Karena Razer seharusnya menjadi salah satu pemimpin periferal terkait game, fakta bahwa mereka tidak dapat membuat driver dengan tingkat kegunaan dasar, membuat saya waspada untuk menaruh kepercayaan dan uang saya pada mouse gaming apa pun.

\n\n

Selama 5 tahun terakhir, saya telah mencoba sekitar 8 aplikasi berbeda (termasuk demo perangkat lunak berbayar) yang seharusnya menukar tombol. Sayangnya, tidak ada yang berfungsi - keduanya merupakan hamparan UI untuk Menang-menukar atau swap berada pada level yang lebih dangkal dari Menang-menukar (Tikus-D).

\n\n

Kontrol Tombol X-Mouse paling mendekati, tetapi sayangnya itu tidak benar-benar mencegat sinyal asli, jadi Anda akhirnya mengirim keduanya klik dan Tidak bisa saat mengklik salah satu tombol mouse.

\n\n

Saya juga mencoba AutoHotKey, tetapi makro sederhana di sepanjang baris 

left::right\nright::left\n

tidak berhasil, dan komunitas kurang membantu. (Forum AHK dulu diisi dengan tanggapan RTFM.)

\n\n

Jebakan yang mungkin terjadi

\n\n

Kabarnya, laptop merek tertentu memiliki menu mouse kustom di Control Panel yang merambat Menang-menukar ke tingkat pengemudi.

\n\n
\n\n

Solusi yang sangat tidak dapat diandalkan untuk masalah ini adalah menggunakan skrip AutoHotKey berikut (berfungsi untuk AHK v1.1.24.02 dan mungkin lebih awal): 

#NoEnv  ; Recommended for performance and compatibility with future AutoHotkey releases.\nSendMode Input  ; Recommended for new scripts due to its superior speed and reliability.\nSetWorkingDir %A_ScriptDir%  ; Ensures a consistent starting directory.\n\nLButton::RButton\nRbutton::LButton\n

Ini memperbaiki masalah untuk beberapa permainan.

\n\n

Sayangnya, ini bukan solusi peluru perak dan menjengkelkan untuk digunakan, karena ini mengharuskan Anda untuk menonaktifkan WinSwap setiap kali Anda memulai permainan yang bermasalah (dan mengaktifkannya kembali saat Anda keluar), karena praktis merusak bilah tugas Windows saat aktif ( yang juga membuatnya agak sulit untuk dimatikan).

\n\n

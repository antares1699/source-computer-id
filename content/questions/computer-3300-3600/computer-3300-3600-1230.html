Saya menggunakan Desktop saya jauh lebih sering daripada menggunakan Dashboard saya. Namun, MacBook Pro saya dilengkapi dengan kunci khusus untuk Dasbor tetapi tidak disertakan dengan satu untuk Desktop.

\n\n

Menggunakan artikel ini , Saya dapat memetakan ulang kunci Dasbor saya untuk menampilkan desktop dengan mengubah nilai untuk kunci 62 dan 63 ("Dasbor") ke nilai yang sama dengan yang digunakan oleh tombol 36 dan 37 ("Tampilkan Desktop"). Secara khusus, saya mengubah nilai untuk kedua indeks array # 1s menjadi 111.

\n\n

Saya tahu ini adalah pertanyaan yang sangat lama, tetapi jika seseorang menemukannya, saya telah menemukan solusi yang cukup bagus.

\n\n

Langkah pertama adalah mengunduh Function Flip, yang dapat ditemukan di sini: http://kevingessner.com/software/functionflip/ . Instal panel preferensi dan aktifkan flip pada tombol F3. Juga pastikan bahwa Function Flip dimulai saat login.

\n\n

Selanjutnya, masuk ke panel preferensi Mission Control dan tetapkan Show Desktop ke tombol F3.

\n\n

Sekarang tombol khusus Kontrol Misi akan menampilkan desktop, dan menahan fn sambil menekan tombol akan menampilkan Kontrol Misi. Untuk sepenuhnya tukar fungsinya, Anda dapat masuk ke Preferensi Keyboard, dan mengubah pintasan keyboard Kontrol Misi ke Command + F3.

\n\n

Tentu saja, Anda juga dapat melakukan ini ke tombol f4 dan juga untuk membalik tombol dasbor / launchpad.

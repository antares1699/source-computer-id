\n\n

Ini terdengar seperti bug kecil di Windows tapi inilah tebakan saya.

\n\n

Saya pikir Anda harus benar-benar me-reboot mesin Anda melalui Windows -> Power -> Reboot setelah mengubah pengaturan dan tidak mematikannya terlebih dahulu.

\n\n

Saat Anda berkata

\n\n
\n

Membuka editor variabel envoronment dan mengklik OK, tanpa membuat perubahan apa pun, menyelesaikan masalah ... hingga boot ulang berikutnya.

\n
\n\n

Saya curiga maksud Anda sampai saat berikutnya Anda mematikan dan kemudian mem-boot mesin. Perbedaan ini penting di Windows 8 dan 10 karena shutdown tidak sama seperti sebelumnya dan reboot lebih dekat dengan shutdown yang sebenarnya sekarang.

\n\n

Shutdown sekarang menutup aplikasi pengguna Anda, mengeluarkan Anda dan kemudian menempatkan inti OS ke dalam mode hibernasi. Ketika Anda me-restart mesin Anda, itu membatalkan hibernate inti dan kemudian memungkinkan Anda masuk kembali. Karena pengaturan telah (pada tahap ini) hanya diterapkan ke profil pengguna saat ini, maka Anda mendapatkan jalur default (lama).

\n\n

Melakukan booting ulang yang benar akan melewatkan langkah hibernasi dan memungkinkan sistem melakukan boot dari awal, memuat pengaturan dan program baru. Inilah sebabnya mengapa pembaruan Windows memperbaiki masalah Anda juga, karena mereka melakukan reboot daripada mematikan dan memulai ulang.

\n\n

Ada beberapa informasi di sini cara kerja hybrid shutdown .

\n\n

Saya mengatakan ini sepertinya sedikit bug karena Anda benar mengharapkan perubahan Anda didorong ke sistem hibernasi, tetapi karena ini adalah salah satu yang mudah diperbaiki (dengan me-reboot) Saya berharap masalah semacam ini adalah masalah yang relatif prioritas rendah memperbaiki.

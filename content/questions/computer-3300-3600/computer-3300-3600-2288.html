Anda dapat menggunakan grep -v \"pattern\" seperti yang Anda katakan. Tapi bukannya pattern digunakan 

`cat file_with_entires_to_skip`\n

Ingatlah untuk memasukkan backtics. Jadi perintah Anda akan terlihat seperti: 

tail -v FILE | grep -v \"`cat FILE_WITH_ENTRIES_TO_SKIP`\"\n

Grep saya menyertakan opsi --exclude-from=FILE yang memungkinkan saya menambahkan pola pengecualian dari file. Aku mendapat: 

my-macbook-pro:Sun America ianchesal$ grep --version\ngrep (GNU grep) 2.5.1\n

Jadi Anda bisa melakukan: 

tail -f <file> | grep --exclude-from=excludes.txt\n

Saya memiliki file SVG yang dihasilkan yang karena alasan tertentu tidak akan terbuka di Adobe Illustrator ketika saya menyajikan konten file dari aplikasi ASP.net saya, tetapi jika saya menulis file yang dihasilkan ini langsung ke sistem file dari aplikasi ASP.net saya , file terbuka dengan baik!

\n\n

Saya telah meletakkan zip dari dua file di sini: http://dl.dropbox.com/u/1761973/Files.zip

\n\n

Zip memiliki dua file:

\n\n
    \n
  • Map.svg
    • \n
  • Test.svg
    • \n
\n\n

Map.svg tidak terbuka di Illustrator, itu muncul dialog Opsi Impor Teks, dan setelah mengklik OK, akan menampilkan konten XML dari file tersebut.

\n\n

Test.svg terbuka dengan baik di Illustrator.

\n\n

Melakukan perbandingan dua file menghasilkan TIDAK ADA PERBEDAAN apa pun!

\n\n

\n\n

Inilah kisah di baliknya:

\n\n

\n\n

\n\n

\n\n

Adakah ide mengapa saya tidak dimintai izin saat menyalin ke folder sistem?

Seperti yang dicatat oleh jawaban poster asli, masalah disebabkan oleh TortoiseCVS. Jika Anda ingin memperbaiki masalah tetapi tetap mempertahankan program, Anda dapat menghapus ekstensi shell.

\n\n

Temukan Tortoiseshell64.dll dalam file program TortoiseCVS. Mengganti nama itu akan menyebabkan Explorer tidak dapat memuatnya, sehingga memperbaiki masalah. Atau, Anda dapat menggunakan Sysinternals Autoruns untuk menonaktifkan ekstensi shell dan menghentikan Explorer untuk mencoba memuatnya. (Ini akan ada di tab Explorer di bawah salah satu header ShellEx.)

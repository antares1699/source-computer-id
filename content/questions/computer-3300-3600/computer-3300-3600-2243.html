Jika Anda terdaftar sebagai pengembang iOS, dan Anda memiliki mac, unduh xcode.

\n\n

Buat salinan file .ipa, klik kanan dan buka dengan Archive Utility.app (ikon hijau)

\n\n

Buka folder yang tidak diarsipkan, dan di bawah "Payloads", Anda akan menemukan file .app.

\n\n

Di Xcode, buat aplikasi kosong di bawah iOS. Ini akan dibuat tanpa Target, tanpa pengaturan build atau apapun.

\n\n

Tambahkan file .app yang disebutkan di atas ke proyek baru Anda, setel setelan build Anda dan "Build and Run" atau "Run" di Xcode. Selama Anda telah menetapkan target sebagai "Simulator iPhone", itu akan terbuka di Simulator iPhone / iPad, dan Anda akan dapat menggunakan aplikasi.

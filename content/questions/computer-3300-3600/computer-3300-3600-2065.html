Saya mencoba mencari tahu mengapa buku kerja .XLS EXCEL sederhana membutuhkan waktu, secara acak, 30 detik untuk dibuka.

\n\n

Sebelum menjawab:

\n\n

Harap perhatikan hal-hal berikut:

\n\n

Gejala utama

\n\n

Menggunakan Penjelajah Windows , berikut urutan yang lebih akurat:

\n\n
    \n
  1. Saya mengklik dua kali pada file EmptyWorkbook.xls ;
    2. \n
  2. Sepertinya tidak ada yang terjadi;
    3. \n
  3. Navigasi pada folder menjadi segera tidak responsif setelah klik dua kali ;
    4. \n
  4. Setelah jeda antara 30 detik dan 1 menit, jendela menjadi responsif dan mulai memuat dokumen biasa, yang setelah 1 detik, menampilkan layar splash EXCEL dan seterusnya.
    5. \n
  5. Jika, selama hang, saya berhenti layanan spooler printer , dokumen dimuat segera.
    6. \n
\n\n

Gejala lainnya

\n\n
    \n
  • Masalahnya secara konsisten tidak muncul kapanpun Printer Spooler dihentikan.
    • \n
  • Menonaktifkan kabel jaringan menghentikan masalah;
    • \n
  • Selama nongkrong di sini tidak ada aktivitas cpu atau disk ;
    • \n
  • Ini hanya terjadi selama pemuatan dokumen. Setiap berjalan mulus setelah itu;
    • \n
  • Windows Explorer.exe hang di folder, tetapi semua folder, sistem, dan aplikasi lainnya masih responsif;
    • \n
  • Tidak ada gantungan yang berurutan. Saya harus menunggu beberapa saat untuk mereproduksi perilaku ini;
    • \n
  • Semua dokumen sampel berada di drive lokal (C: \\ BPI);
    • \n
  • Dokumen memiliki makro dan memiliki penggunaan addins;
    • \n
  • Masalahnya tidak terjadi pada ekstensi file lain seperti .PDF, misalnya;
    • \n
  • Office 2003 digunakan selama beberapa tahun;
    • \n
  • Komputer menjalankan Windows XP;
    • \n
  • Komputer memiliki beberapa drive yang dipetakan jaringan, semuanya ditujukan ke server file utama;
    • \n
  • Baru-baru ini, server file utama digantikan oleh Windows 2011 SBS Standard Edition

    • \n
  • Saya telah mencoba berbagai dokumen (.xls dan .doc), semuanya lebih kecil dari 30 KB;
    • \n
  • Saya telah, untuk sementara, menghapus semua pintasan di folder Dokumen Pengguna yang mengarah ke drive atau share jaringan;

    • \n

    • \n
  • Saya telah meneliti secara ekstensif selama berjam-jam di web;
    • \n

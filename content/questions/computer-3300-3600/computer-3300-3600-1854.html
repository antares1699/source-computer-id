Saya kira orang mungkin sadar tetapi pinfo sekarang tersedia dari brew: 

brew install pinfo\n

Jika Anda ingin tahu tentang mendapatkan autopoint di MacOS / X - ini tersedia dari formula minuman gettext - tangkapannya adalah hanya-tong yang berarti tidak terhubung ke / usr / local secara default karena dapat bertentangan dengan MacOS versi internal, namun MacOS tidak menyediakan autopoint jadi dapatkanlah sebagai berikut - instal gettext: 

brew install gettext\n

lalu sym-link autopoint ke / usr / local / bin: 

ln -s `brew ls gettext | grep bin/autopoint` /usr/local/bin  \n

Apakah file .chm di jaringan bersama? Atau apakah itu diunduh dari internet atau dari jaringan berbagi? Windows melindungi file yang dianggap "tidak tepercaya" dan mencegahnya ditampilkan. Untuk memperbaiki masalah ini, salin chm ke komputer lokal Anda, kemudian klik kanan pada file tersebut dan pilih "Properties" dan klik tombol "Unblock".

Biar saya tebak file CHM diunduh dari internet? Ini adalah mekanisme keamanan. Jika Anda mempercayai file itu (dan ingat - file CHM berisi HTML dan skrip yang berjalan di bawah executable eksternal), cukup klik kanan, pilih properti, lalu pilih "buka blokir".

\n\n

Karena ini terlihat seperti file Mercury (haruskah saya katakan HP?), Anda mungkin bagus - selama Anda mendownloadnya dari situs resmi.

Solusi lain ketika benar-benar bekerja melalui jaringan (tepercaya) berbagi adalah dengan mentransfer file CHM dalam sebuah arsip (zip, rar, dll.). Setelah mengekstraknya dari arsip, CHM akan ditampilkan tanpa batasan yang dapat diamati. Alasannya adalah (setelah KB896358 ) untuk file yang diunduh pembatasan keamanan khusus berlaku . Anda dapat menonaktifkan pembatasan ini dengan memasukkan dua kunci ke dalam registri seperti yang dijelaskan di artikel (Jerman) ini:
\n Soal karena Anzeigen von .chm-Dateien - Tipps & Tricks

\n\n

Patch yang diekstrak ke registri (mengaktifkan konten intranet): [1] 

Windows Registry Editor Version 5.00\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\HTMLHelp\\1.x]\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\HTMLHelp\\1.x\\HHRestrictions]\n\"MaxAllowedZone\"=dword:00000001\n\n[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\HTMLHelp\\1.x\\ItssRestrictions]\n\"MaxAllowedZone\"=dword:00000001\n\n
\n\n

[1] Saya mencoba modifikasi selangkah demi selangkah dan menemukan kunci HHRestrictions tidak relevan.

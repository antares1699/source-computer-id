Pertama, Salin Data Paling Penting dengan Cepat

\n\n

Jika Anda memiliki hard drive lain di sistem Anda, segera salin file terpenting Anda ke drive itu. Setidaknya Anda akan memiliki duplikat data penting Anda. Jika Anda mendapatkan data yang rusak, itu masih lebih baik daripada menunggu sementara situasinya bisa semakin memburuk.

\n\n

Kedua, Lakukan Pencadangan Gambar Penuh

\n\n

Kemudian, saya akan mendapatkan sebanyak mungkin informasi dari drive secepat mungkin.\nSaya akan mencoba program pencitraan disk (seperti Acronis True Image ). Itu adalah cara tercepat yang saya tahu untuk mendapatkan sebanyak mungkin informasi dari hard drive secepat mungkin. Jika ada korupsi, Anda akan mendapatkannya juga ... tetapi setidaknya Anda memiliki sesuatu untuk dikerjakan.

\n\n

Ketiga, Temukan dan Pulihkan Data yang Rusak

\n\n

Jika cadangannya bagus, Anda dapat mengembalikan gambar ke hard drive baru. Jika Anda menemukan korupsi , maka saya akan mencoba program seperti SpinRite . Program itu dirancang untuk bekerja tanpa lelah pada hard drive Anda pada level terendah (bahkan di bawah level sistem operasi) dan benar-benar menggali data hard drive Anda dan mengambil setiap bit terakhir data yang akan diambil.

Lihat nilai swappiness Anda di /proc/sys: 

$ cat /proc/sys/vm/swappiness\n60\n

Anda dapat memaksa Ubuntu untuk tidak menggunakan swapfile hingga benar-benar diperlukan dengan mengubah nilai ini menjadi nol: 

$ sudo sysctl vm.swappiness=0\nvm.swappiness = 0\n

\n\n
\n

100Mb RAM digunakan dari 312Mb dan\n 31Mb dari Virtual Mem juga digunakan.

\n
\n\n

Anda belum menyebutkan penggunaan SWAP dan hanya menyebutkan bahwa 1/3 dari RAM Anda digunakan.

Satu hal penting yang harus diingat pada sistem Linux, adalah bahwa RAM Anda akan sering kali terlihat "lebih banyak digunakan" daripada yang sebenarnya karena Kernel akan menggunakan ram untuk apa pun yang dapat membuat hidup Anda lebih cepat (mis .: informasi sistem file / inode cache / etc ...). Ini tidak terlalu bermasalah karena informasi yang disangga ini dianggap sebagai warga negara kelas dua dan akan dihancurkan jika perlu ..

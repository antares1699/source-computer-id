Saat melihat dokumentasi bantuan di Xcode 4.2 (iOS SDK 5) kami mendapatkan pesan kesalahan ini:

\n\n
"index.html" dikunci untuk pengeditan dan Anda mungkin tidak dapat menyimpan perubahan Anda. Apakah Anda ingin membukanya?
\n\n
File "index.html" tidak dapat dibuka kuncinya. File tersebut adalah sumber daya jarak jauh. Coba buat salinan lokal.
\n\n

Bagaimana kami dapat melihat bantuan online?

","source":"https://superuser.com/questions/348117/unable-to-view-xcode-4-2-help-index-html-is-locked-for-editing","author":"rjobidon","replies":[],"answers":[{"answer":"

Solusi yang sangat sederhana namun efektif:

\n\n
    \n
  1. Buka Xcode menu> Preferensi ...
    2. \n
  2. Klik Download kategori
    3. \n
  3. Klik Dokumentasi tab
    4. \n
  4. Pilih "Perpustakaan iOS 5.0"
    5. \n
  5. Klik Periksa dan Instal Sekarang
    6. \n
\n\n

Ini akan mengunduh dokumentasi (sekitar 360 MB) dan menyelesaikan masalah.

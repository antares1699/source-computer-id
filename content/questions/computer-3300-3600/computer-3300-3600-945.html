Saya telah mengunduh gambar debian vm untuk beberapa pengujian: http://exploit-exercises.com/download (280mb)

\n\n

Ini adalah mesin Pemeras Debian. Saya ingin menggunakan tata letak Belgia (yaitu AZERTY) sebagai ganti QWERTY default. Tidak ada server X yang terpasang.

\n\n

Apa yang saya coba (semuanya sebagai root)

\n\n
    \n

  • setxkbmap be -> 'perintah tidak ditemukan'

    • \n

  • dpkg-reconfigure keyboard-configuration -> berfungsi dengan baik, tetapi keyboard tetap dalam QWERTY setelah memilih keyboard Belgia

    • \n

  • dpkg-reconfigure console-data -> "'console-data' tidak diinstal dan tidak ada info yang tersedia"

    • \n

  • apt-get install console-data -> paket 'console-data' tidak memiliki kandidat penginstalan. Saya juga menjalankan apt-get update dan daftar saya berisi url debian.

    • \n

  • edit /etc/default/keyboard secara manual. Saya kemudian membaca saya harus memulai ulang /etc/init.d/hal, tetapi layanan itu tidak ada

    • \n

  • beberapa hal lain yang tidak dapat saya ingat

    • \n

  • locale-gen fr_FR -> Ia mengatakan "Generating locales", tetapi hanya menampilkan en_US.UTF-8. Tidak ada fr_FR yang bisa ditemukan.

    • \n

  • dpkg-reconfigure locales Daftar pertama memiliki semua lokal di mana saya memilih fr_Fr. Layar kedua (lokal default) hanya memberikan en_US sebagai opsi.

    • \n
\n\n

Apa lagi yang bisa saya coba?

SOLUSI

\n\n

Saya menemukan bahwa ada semacam kesalahan Chrome (seolah-olah itu adalah satu-satunya ...) yang dapat Anda baca di sini: https://code.google.com/p/chromium/issues/detail?id=60876

\n\n

Sepertinya solusinya (ini berhasil untuk saya) adalah menjelajahi folder Google Chrome (bagi saya itu adalah nama pengguna / AppData / Lokal / Google / Chrome / Aplikasi saya) dan menghapus folder versi lama, bersama dengan file old_chrome.exe. Kemudian ganti nama file new_chrome.exe kembali menjadi hanya chrome.exe.

\n\n

Itu sepertinya memperbaiki semuanya, kecuali ikon pintasan desktop, yang dapat dengan mudah diperbaiki dengan membuat pintasan baru atau mengubah ikon untuk pintasan yang ada secara manual. Bahkan WebMatrix melihat Chrome saya dipasang lagi setelah ini.

\n\n

Juga disarankan bahwa menggunakan pembersih registri (CCleaner bagus), dapat membantu beberapa masalah ikon, tetapi ini tidak berdampak pada situasi ini bagi saya.

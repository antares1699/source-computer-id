Saya menggunakan ssh untuk masuk ke mesin Windows lainnya. Saya mencoba membuat pssession pada localhost. Tetapi saya tidak dapat melakukannya dalam prompt mewah yang normal: 

$ New-PSSession\nNew-PSSession : [localhost] Connecting to remote server localhost failed with the\nfollowing error message : Access is denied. Sepertinya saya harus berada di atas cangkang mewah yang ditinggikan. Apakah ada cara untuk melakukan ini tanpa hak administrator?

","source":"https://superuser.com/questions/644510/powershell-remoting-as-non-administrator","author":"joon","replies":[],"answers":[{"answer":"

Powershell Remoting memiliki default PSSessionConfiguration disebut Microsoft.PowerShell dengan a SDDL yang hanya memungkinkan orang dengan hak administratif untuk menjalankan perintah jarak jauh di mesin itu.

\n\n

Anda perlu menghubungi administrator dan meminta mereka membukanya untuk Anda: 

PS C:\\> Set-PSSessionConfiguration -Name Microsoft.PowerShell -showSecurityDescriptorUI\n

Info lebih lanjut di sini: http://blogs.msdn.com/b/powershell/archive/2009/11/23/you-don-t-have-to-be-an-administrator-to-run-remote-powershell-commands.aspx

\n\n

Jangan lupa Aktifkan-PSRemoting bersama dengan membuka port yang sesuai (5985 HTTP / 5986 HTTPS).

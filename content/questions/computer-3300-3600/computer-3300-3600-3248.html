Bagaimana cara membuat tautan keras untuk file di partisi NTFS menggunakan Windows?

\n\n

NTFS jelas dapat menangani hardlink, karena membuatnya dengan NTFS-3g berfungsi - tautan tersebut bahkan berfungsi di Windows. Apakah ini satu-satunya cara untuk membuat hardlink di NTFS?

","source":"https://superuser.com/questions/141784/hardlinks-on-ntfs-with-windows","author":"knittl","replies":[],"answers":[{"answer":"

cmd termasuk mklink perintah yang dapat membuat tautan keras:

\n\n
\n

Membuat tautan simbolis.

\n \n

MKLINK [[/D] | [/H] | [/J]] Link Target

\n \n
\n

/D Membuat link simbolis direktori. Default adalah tautan simbolis file.

\n \n

/H Membuat tautan keras alih-alih tautan simbolis.

\n \n

/J Membuat Directory Junction.

\n \n

Link menentukan nama tautan simbolik baru.

\n \n

Target menentukan jalur (relatif atau absolut) yang dirujuk tautan baru.

\n
\n
\n\n

Selain itu, luar biasa Manajer Jauh memiliki dukungan untuk membuatnya ( Alt + F6 ).

\n\n

Ada juga sejumlah ekstensi shell yang memungkinkan untuk melakukannya.

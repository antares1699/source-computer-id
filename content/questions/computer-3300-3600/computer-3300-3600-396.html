Asosiasi ekstensi didasarkan pada skema penyorotan sintaks.

\n\n

Untuk menghapus asosiasi yang tidak diinginkan, buka ST2 dan pilih Preferensi â†’ Jelajahi Paket ... , folder dengan paket akan terbuka. Temukan paket yang salah terdeteksi (dalam kasus ini Pelat ) dan buka a .tmLanguage file ( Lisp.tmLanguage ) di ST2. Baris pertamanya terlihat seperti ini: 

<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n<!DOCTYPE plist PUBLIC \"-//Apple Computer//DTD PLIST 1.0//EN\" \"http://www.apple.com/DTDs/PropertyList-1.0.dtd\">\n<plist version=\"1.0\">\n<dict>\n    <key>comment</key>\n    <string></string>\n    <key>fileTypes</key>\n    <array>\n        <string>lisp</string>\n        <string>cl</string>\n        <string>l</string>\n        <string>mud</string>\n        <string>el</string>\n    </array>\n(...)\n

String tersebut pada baris 9-13 adalah ekstensi file yang akan diasosiasikan oleh ST2 dengan Lisp. Anda dapat menghapus beberapa baris tersebut (letakkan kursor pada baris yang tidak diinginkan dan tekan Ctrl + Bergeser + K ), perubahan akan segera diterapkan saat Anda menyimpan file.

\n\n

Untuk menambahkan asosiasi baru, buka a .tmLanguage file dari paket dan menambahkan string baru di antara yang sudah ditentukan.

Dalam versi Sublime Text 2 (Build: 2139) saat ini, Anda dapat mengatur sintaks untuk semua file dari ekstensi file tertentu menggunakan opsi di bilah menu. Buka file dengan ekstensi yang ingin Anda setel defaultnya dan navigasikan melalui menu berikut: View -> Syntax -> Open all with current extension as... ->[your syntax choice].

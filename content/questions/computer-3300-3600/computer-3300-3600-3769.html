","source":"https://superuser.com/questions/732952/how-to-make-a-3tb-virtual-drive-for-virtualbox","author":"Daniel","replies":[],"answers":[{"answer":"

Anda tidak dapat menggunakan GUI untuk melakukannya, tetapi Anda dapat menggunakan baris perintah seperti: 

 vboxmanage createhd --filename 3TB.vdi --size 3000000 --format VDI --variant Standard\n

Menghabiskan berjam-jam mencoba membuat ini bekerja tadi malam dan akhirnya bisa. Saya memiliki masalah yang sama di mana iexplore.exe hilang di Program Files dan Program Files x86, dan juga tidak memiliki opsi untuk itu di bagian "menghidupkan atau mematikan fitur windows" atau di panel Pengaturan baru untuk yang tersedia fitur. Berikut langkah-langkah yang membuatnya berhasil untuk saya:

\n\n
    \n
  1. Pertama, dapatkan ISO Windows 10 terbaru di sini: https://www.microsoft.com/en-us/software-download/windows10
    2. \n
  2. Gunakan alat pembuatan media dan biarkan saja itu mengeluarkan gambar ISO di suatu tempat di hard drive Anda.
    3. \n
  3. Selanjutnya, pastikan untuk memperbarui Windows 10 ke versi terbaru dengan masuk ke Pengaturan> Perbarui & Keamanan> Periksa pembaruan.
    4. \n
  4. Setelah Windows diperbarui sepenuhnya, buka PowerShell sebagai administrator.
    5. \n
  5. Selanjutnya, Anda akan ingin mencari file ISO yang Anda letakkan di hard drive Anda dan memasangnya sehingga Anda dapat menelusuri file
    6. \n
  6. Jelajahi ISO yang terpasang. Anda akan ingin membuka sources> sxs dan seharusnya ada sekitar 4 atau lebih file di sana, salah satunya bernama "microsoft-windows-interenetexplorer-optional-package.cab"
    7. \n
  7. Di PowerShell, masukkan yang berikut (ganti E: \\ sources .... saya dengan lokasi sebenarnya dari file cab di sistem Anda): dism / online / add-package / packagepath: E: \\ sources \\ sxs \\ microsoft- windows-internetexplorer-optional-package.cab
    8. \n
  8. Jika ISO Anda mutakhir dan OS Windows 10 Anda mutakhir, itu harus menginstal dan meminta Anda untuk memulai ulang
    9. \n
  9. Setelah memulai ulang, Anda harus dapat membuka Pengaturan> Aplikasi> Kelola Fitur Opsional> Tambahkan Fitur> Internet Explorer 11\nATAU\nBuka Panel Kontrol dengan mencarinya setelah menekan tombol mulai, lalu buka Program dan Fitur> Hidupkan atau matikan fitur Windows lalu centang kotak Internet Explorer 11 dan mulai ulang

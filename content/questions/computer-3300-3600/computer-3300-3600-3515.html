Menggunakan Firefox

\n\n
    \n
  1. Di Firefox:
      \n
    • buka halaman web Anda
      • \n
    • jangan gunakan mode layar penuh
      • \n
    • memastikan Anda dapat melihat bilah gulir Anda, Anda akan membutuhkannya
      • \n
    • buka Alat> Pengembang Web> Inspektur
      • \n
    2. \n
  2. Pastikan bahwa alat pengembang «Tampilkan di jendela terpisah»
    3. \n
  3. Pilih «Mode Desain Responsif» ( RDM )
    \n\n\n
      \n
    • letakkan jendela Inspektur itu ke samping, Anda tidak akan membutuhkannya lagi
      • \n
    4. \n
  4. Kembali ke halaman web Anda, menggunakan RDM toolbar, pilih ukuran jendela virtual yang sesuai dengan kebutuhan Anda; semakin besar jendelanya, semakin banyak memori dan sumber daya
    5. \n
  5. Menggunakan ikon kamera di RDM toolbar atau Add-on Firefox, tangkap gambar ( † )
    \n
    6. \n

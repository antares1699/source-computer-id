\n\n

Kedengarannya seperti Apache diatur untuk hanya mendengarkan alamat loopback / localhost, jadi hanya mesin itu sendiri yang dapat terhubung dengannya. Temukan file konfigurasi, ubah antarmuka yang didengarkan, dan kemudian mesin lain harus dapat membuka server web.

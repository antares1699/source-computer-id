Edit: Gulir ke bawah untuk mencari solusi.

\n\n

Saya dapat mengonfirmasi bahwa fitur Microsoft Outlook "Deteksi bahasa secara otomatis "hanya berfungsi dalam keadaan khusus. Sekali melalui email.\nAda 3 prasyarat:

\n\n
    \n
  1. Mengatur default bahasa selain bahasa Inggris ( Ulasan > Bahasa -> Setel Bahasa Pemeriksa ... > [Pilih bahasa]> Ditetapkan sebagai default )
    2. \n
  2. Mulailah menulis dalam bahasa itu bukan bahasa Inggris dengan tata letak keyboard yang dipilih sesuai
    3. \n
  3. Ganti tata letak keyboard ( Alt + Bergeser atau Kunci kemenangan + Ruang di Windows) sebelum menulis dalam bahasa Inggris.
    4. \n
\n\n

Gambar berikut menunjukkan dan menjelaskan langkah-langkah yang dilakukan. Tes bahasa pemeriksa Outlook

\n\n

Jadi pada dasarnya, Anda dapat menulis dalam satu bahasa dan kemudian beralih ke bahasa Inggris sekali. Jika Anda ingin menulis lagi dalam bahasa sebelumnya Anda dapat menyalin-tempel teks atau hanya satu karakter dari sebelumnya atau menggunakan menu untuk mengatur bahasa pemeriksa.

\n\n

Ngomong-ngomong, tanda tangan memiliki properti "Jangan periksa ejaan atau tata bahasa" secara otomatis disetel jadi jika Anda memilih dan menghapusnya, pemeriksaan untuk teks berikut tidak akan diaktifkan kecuali Anda menyetelnya secara manual.

\n\n

Solusi

\n\n

Selain kondisi yang disebutkan di atas, opsi tercepat dan paling andal untuk pemeriksaan berbagai bahasa dalam satu email (sejauh ini) adalah dengan menyetel bahasa secara manual. Agar tombol mudah diakses:

\n\n
    \n
  1. Klik kanan pada Ribbon bar> Kustomisasi Pita ... (Atau masuk ke opsi Outlook: Mengajukan > Pilihan > Kustomisasi Pita )
    2. \n
  2. Di panel kanan tekan Grup baru (Secara opsional, ganti namanya menjadi mis. " Bahasa ")
    3. \n
  3. Di panel kiri pilih di Pilih perintah dari: " Semua perintah "
    4. \n
  4. Gulir ke dan pilih " Setel Bahasa Pemeriksa "perintah dan tekan" Menambahkan "atau seret dan lepas di grup yang baru dibuat.
    5. \n
\n\n

Opsi Outlook

\n\n

Tekan " baik "dan Anda akan memiliki" Set Proofing Language "sehingga mudah diakses dari Ribbon bar setiap saat sebelum Anda perlu menulis dalam bahasa yang berbeda (selain default) Anda dapat memilih bahasa pemeriksa dengan 3 klik.

\n\n

Larutan

\n\n

Rupanya itu bermuara pada pengaturan Bahasa Windows: Jika Anda hanya menginstal satu bahasa (mis. Bahasa Inggris) dengan tata letak keyboard beberapa bahasa di Pengaturan Windows, alat pemeriksa tidak akan mengenali teks dengan benar secara otomatis.

\n\n

Agar alat pemeriksa ejaan dapat mengenali teks dalam setiap bahasa, Anda perlu menginstal bahasanya, bukan hanya tata letak keyboard. Untuk melakukan itu (pada Windows 10 ) pergi ke Pengaturan Windows ( Kunci kemenangan + saya )> Waktu & Bahasa > Bahasa > Tambahkan bahasa > [pilih bahasa Anda]> Lanjut

\n\n

Dari jendela Instal fitur bahasa, cukup " Pengetikan dasar "Opsi cukup. Tekan Install .

\n\n

\n\n

Untuk menguji buka a email baru jendela di Outlook ( Ctrl + N ) dan mulai mengetik dalam bahasa pilihan Anda. Sekarang beralih ke bahasa lain dengan menekan Alt + Bergeser atau menggunakan menu pilihan bahasa di dekat jam di bilah tugas Windows:

\n\n

\n\n

Alat pemeriksaan harus berfungsi secara otomatis :

\n\n

\n\n

Perhatikan bahwa masing-masing pasangan Kata-kata ditulis dalam bahasa pilihan yang berbeda dan secara otomatis (tanpa menyentuh mouse) dikenali oleh alat pemeriksa ejaan.\n(Baris pertama: Inggris, Latvia, Jerman\nBaris ke-2: Jerman, Inggris, Latvia)

\n\n

Catatan - dua hal yang perlu diingat:

\n\n
    \n
  1. Jika Anda telah mengaktifkan default Tanda tangan untuk email baru di opsi Outlook ( Mengajukan > Pilihan > Surat > Tanda tangan... ) atau tambahkan tanda tangan secara manual (melalui Memasukkan > Tanda tangan ) sebelum menulis teks apa pun dan mulailah menulis email baru dalam suatu bahasa lain daripada default di pengaturan Outlook Anda, itu tidak akan dikenali dengan benar terlepas dari bahasa yang dipilih saat ini di menu bahasa Windows Anda. Untuk mengatasinya, cukup lakukan siklus ( Alt + Bergeser ) bahasa Anda satu kali.
    2. \n
  2. Ada sedikit penundaan setelah memilih bahasa yang berbeda untuk pertama kali. Jika Anda mulai mengetik segera setelah beralih bahasa, alat pemeriksa ejaan mungkin memperlakukan teks seperti dalam bahasa yang dipilih sebelumnya.\nUntuk menghindarinya, tunggu sebentar sebelum mulai menulis dalam bahasa baru untuk pertama kalinya.
    3. \n

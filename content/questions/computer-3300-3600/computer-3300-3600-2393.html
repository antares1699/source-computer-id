Jika Anda memiliki inotify di kernel Anda, periksa inotify-tools .

\n\n

Ada contoh di halaman itu: 

#!/bin/sh\n# A slightly complex but actually useful example\ninotifywait -mrq --timefmt '%d/%m/%y %H:%M' --format '%T %f' \\\n -e close_write /home/billy | while read date time file; do\n    rsync /home/billy/${file} rsync://billy@example.com/backup/${file} && \\\n    echo \"At ${time} on ${date}, file ${file} was backed up via rsync\"\ndone\n

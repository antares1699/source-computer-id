Tidak ada solusi sempurna, tetapi kandidat teratas Anda adalah NTFS dan exFAT :

\n\n
    \n
  • Windows (versi terbaru) akan membaca dan menulis keduanya, secara native
    • \n
  • OS X akan membaca NTFS dan membaca / menulis exFAT
    • \n
  • Linux akan membaca NTFS (dukungan kernel)
    • \n
\n\n

Driver pihak ketiga dapat mengisi celah yang hilang (non-Windows); beberapa distro Linux menyertakannya. Jika penggunaan Anda sebenarnya hanya membaca dari Windows, maka NTFS akan berfungsi di luar kotak. Jika Anda terutama OS X dan bukan Linux, maka exFAT akan lebih baik.

\n\n

Salah satunya dapat diformat langsung di Windows (File) Explorer GUI di desktop .

Pada dasarnya jika Anda hanya ingin menjalankan chkdsk / f atau mungkin chkdsk / r setiap kali boot, Anda dapat mengaturnya agar berjalan saat Anda mematikan daripada mencoba membuatnya bekerja pada HKLM\\SYSTEM\\CurrentControlSet\\Control\\Session Manager.

\n\n

Begini caranya:

\n\n

Buat file C: \\ WINDOWS \\ system32 \\ confirm.txt dengan HANYA y di dalamnya.

\n\n

Kemudian buat C: \\ WINDOWS \\ system32 \\ Chkdsk_Reboot.bat dengan kode ini di dalamnya: 

echo y > confirm.txt\nchkdsk c: /f /x < confirm.txt\n

Sekarang jika Anda memiliki XP Pro, atau OS level Pro lainnya, buka gpedit.msc> Konfigurasi Komputer> Pengaturan Windows> Skrip> Shutdown, dan tambahkan Chkdsk_Reboot.bat itu sebagai skrip shutdown.

\n\n

Reboot berikutnya Anda akan menjalankan perintah chkdsk Anda.

\n\n

Saya tidak yakin apakah ini akan berfungsi untuk XP home karena tidak memiliki gpedit.msc. Anda mungkin dapat membuat struktur direktori ini dan menambahkan file batch di sana tetapi saya tidak yakin apakah itu akan berfungsi karena Beranda tidak menggunakan Kebijakan Grup: C: \\ WINDOWS \\ System32 \\ GroupPolicy \\ Machine \\ Scripts \\ Shutdown

\n\n

chkdsk telah diatur untuk dijalankan pada setiap boot.

\n\n

Lihatlah HKLM\\SYSTEM\\CurrentControlSet\\Control\\Session Manager dan temukan entri multi-string (REG_MULTI_SZ) bernama BootExecute. Nilai "autocheck autochk *" mengatakan bahwa chkdsk akan berjalan saat boot, namun Anda tidak akan melihatnya karena berjalan di latar belakang, mencari kesalahan disk.

\n\n

Jika chkdsk mendeteksi beberapa masalah disk, maka Anda akan melihat layar biru dengan pemindaian disk penuh.

\n\n

Edit:

\n\n

Sampel opsi pemeriksaan otomatis, menurut sini (bagian: "The Dirty bit vs the Registy" Autochk "entry"): 

Sample command              Resulting registry entry \n==============              ========================\nChkdsk C: /F                Autocheck autochk /p \\??\\C: \nChkdsk C:\\mountpoint /F     Autocheck autochk /p \\??\\VOLUME{GUID} \nChkntfs D: E: /X            Autocheck autochk /k:D /k:E * \nChkntfs G: /C               Autocheck autochk /m \\??\\G: \n

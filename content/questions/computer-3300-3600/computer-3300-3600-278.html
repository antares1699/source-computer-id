Melakukan pekerjaan TI belum sama sejak saya mulai menggunakan banyak monitor. Saya merasa itu meningkatkan produktivitas saya sedikit dan saya tidak bisa membayangkan menyulap semua jendela RDP dan VNC dengan 1 layar. Beberapa monitor berguna untuk banyak hal ...

\n\n
    \n
  • Pemrograman
    • \n
  • Menonton file log
    • \n
  • Menonton monitor status pada aplikasi (mis. Cisco SDM / ASDM)
    • \n
  • Bekerja pada beberapa desktop jarak jauh sekaligus
    • \n
  • Melacak jendela email / obrolan saat sibuk dengan hal lain
    • \n
\n\n

Bukan hal IT:

\n\n
    \n
  • Menyelamatkan Anda dari kebosanan saat mengerjakan sesuatu yang lain (mis. Menonton video di layar lain)
    • \n
  • Bekerja dengan grafik saat video selesai rendering (memungkinkan Anda melacak apa yang Anda lakukan alih-alih memaksimalkan jendela untuk memeriksa berbagai hal setiap saat)
    • \n
  • Ganti lagu dengan cepat di pemutar musik Anda
    • \n
\n\n

... dan seterusnya.

\n\n

Intinya bagi saya adalah monitor kedua memudahkan untuk mengelola jendela, dan mengawasi hal-hal saat Anda sibuk dengan hal lain. Setelah beberapa saat, menggunakan monitor kedua menjadi kebiasaan. Saya menemukan diri saya bekerja pada sistem orang lain dan kadang-kadang mencari tempat untuk mendorong jendela aplikasi!

\n\n

Jika semua ini tidak berlaku, mungkin Anda kurang sibuk! :)

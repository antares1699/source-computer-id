Cara menggunakan wmic untuk mematikan contoh cmd.exe yang mencari dengan perintah itu\n garis?

\n
\n\n

Tampaknya Anda dapat melakukan ini secara native dari batch Windows dengan format di bawah ini. Cukup masukkan nama file batch Anda (yang merupakan string yang ditemukan dengan opsi WMIC COMMANDLINE) di bagian %start.bat% seperti yang saya lakukan di bawah — saya menggunakan start.bat dalam contoh saya di bawah ini seperti yang Anda gunakan dalam contoh Anda.

\n\n

CONTOH PROSES FILE BATCH BATCH KHUSUS WMIC 

WMIC PROCESS WHERE \"COMMANDLINE LIKE '%start.bat%'\" CALL TERMINATE\n
\n\n

\n\n

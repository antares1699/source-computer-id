Saya memiliki penginstalan Windows 7 yang baru dan aktif yang ingin saya gunakan sebagai penginstalan Uji. Jadi saya ingin membuat Snapshot dan dapat sepenuhnya kembali ke snapshot itu.

\n\n

Karena ini bukan Enterprise / Ultimate, VHD Booting bukanlah pilihan, dan karena ini adalah Netbook, VMWare / VPC juga bukan pilihan.

\n\n

Jadi saya bertanya-tanya apakah Pemulihan Sistem cukup baik, atau apakah saya memerlukan solusi pencitraan? (Jika membantu: Saya memiliki Windows Home Server)

\n\n

Saya harus menambahkan bahwa HD memiliki 2 partisi, dan keduanya adalah instalasi Windows 7. Partisi kedua harus dibiarkan tidak tersentuh karena ini adalah Setup utama.

Pemulihan sistem sangat bagus sekarang, tetapi masih belum mengalahkan cadangan penuh - jika Anda terkena virus atau sesuatu yang sangat serius, Anda akan mendapat masalah.

\n\n

Windows dilengkapi dengan utilitas cadangan yang dapat membuat gambar penuh, saya sangat menyarankan Anda melakukan ini ke komputer lain atau drive eksternal.

\n\n

\n\n

Ketika Anda ingin memulihkan, Anda cukup memiliki disk Windows USB yang dapat di-boot, pergi ke perbaikan dan selama Anda memiliki jaringan atau akses langsung ke gambar, Anda akan dapat memulihkan dengan mudah.

\n\n

Pemulihan Sistem tidak cukup baik. dan begitu pula fitur Image Sistem jika Anda ingin menguji sistem operasi lain selain Windows 7 pada drive / partisi tersebut (Microsoft tidak terlalu peduli atau mereka dalam keadaan lalai dalam hal OS Linux :).

\n\n

Favorit saya adalah Symantec Ghost Enterprise dan Drive Snapshot (keduanya produk komersial), jika Anda mencari beberapa alat pencitraan drive gratis yang layak, lihat Clonezilla, DriveImageXML atau EASEUS Disk Copy.

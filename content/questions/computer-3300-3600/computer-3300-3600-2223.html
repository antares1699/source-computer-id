Anda dapat menggunakan varian pada pengiris larik untuk melakukan ini: 

args1to3=\"${*:1:3}\"  # Three arguments starting from $1\nargs4on=\"${*:4}\"     # The arguments starting from $4\n

BTW, ini mungkin bukan yang Anda inginkan, karena itu hanya menempelkan argumen bersama dengan spasi di antara mereka; jika ada argumen juga berisi spasi, itu akan kehilangan jejak mana yang ada di dalam argumen dan mana yang ada di antara mereka (lihat BashFAQ # 50 ). Jika Anda ingin dapat menjaganya tetap lurus, gunakan array sebagai gantinya: 

args1to3=(\"${@:1:3}\")   # Three arguments starting from $1, as an array\nargs4on=(\"${@:4}\")      # The arguments starting from $4, as an array\n\nothercmd \"${args1to3[@]}\"             # Pass the first 3 arguments intact\nfor somearg in \"${args4on[@]}\"; do    # Process args 4 on, one at a time\n  othercmd2 \"$somearg\"\ndone\n

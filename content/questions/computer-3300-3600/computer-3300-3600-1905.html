Saya memiliki video input yang direkam pada 640x360 (itu dari webcam jika ada bedanya).

\n\n

Saya ingin hasilnya menjadi 1280x720 (dua kali lipat lebar aslinya) dan saya ingin video asli digeser ke kanan dan di tengah secara vertikal.

\n\n

Saya memiliki dua atau lebih gambar (png) yang berukuran 640x720 dan itu harus ditampilkan pada waktu-waktu tertentu dalam video di sisi kiri.

\n\n

Inilah yang harus saya ukur ulang dan padukan dengan video asli: 

ffmpeg -i \"E:\\Videos\\testvid.wmv\" -q:a 0 -q:v 0 -vf \"pad=1280:720:(1280-iw):(720-ih)/2\" -s 1280:720 \"E:\\Videos\\new.wmv\"\n

Inilah yang saya harus memasukkan gambar ke dalam video: 

ffmpeg -i \"E:\\Videos\\new.wmv\" -i \"E:\\Pictures\\red_640x720.png\" -filter_complex \"[0:v][1:v] overlay=0:0:enable='between(t,1,10)'\" \"E:\\Videos\\new2.wmv\"\n

Kedua pernyataan di atas berfungsi dengan baik. Namun, saya ingin melakukan semuanya sekaligus sehingga FFMPEG hanya berjalan sekali (jika memungkinkan). Adakah yang bisa membantu saya dengan sintaks?

\n\n

Terima kasih.

Hubungkan filter dengan koma. Urutan linier dari satu atau lebih filter membuat rantai filter. Hubungkan rantai filter dengan titik koma untuk membuat grafik filter. Lihat Pengenalan Pemfilteran dan Deskripsi Filtergraph untuk info lebih detail.

\n\n

Contoh: 

-filter_complex \\\n\"[0:v]filter_0,filter_1,filter_2[chain1]; \\\n [1:v]filter_3,filter_4,filter_5[chain2]; \\\n [chain1][chain2]filter_6\"\n

Jadi perintah Anda akan terlihat seperti: 

ffmpeg -i video.wmv -i image0.png -i image1.png -filter_complex \\\n\"[0:v]pad=iw*2:ih*2:ow-iw:(oh-ih)/2[padded]; \\\n [padded][1:v]overlay=0:0:enable='between(t,1,10)'[bg]; \\\n [bg][2:v]overlay=0:0:enable='between(t,10,20)'[out]\" \\\n-map \"[out]\" -map 0:a output.wmv\n

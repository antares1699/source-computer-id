Anda dapat menggunakan KVM berbasis IP jarak jauh untuk menghasilkan penekanan tombol di komputer lain secara real time.

\n\n

Cukup sambungkan keyboard KVM (dan juga video dan mouse) ke komputer yang ingin Anda kirimi penekanan tombol jarak jauh.

\n\n

Anda kemudian terhubung ke IP di KVM melalui browser web Anda atau aplikasi sistem asli yang disediakan, dan mengirim penekanan tombol apa pun yang Anda inginkan.

\n\n

Salah satu produk tersebut adalah Remote Control Server Tripp Lite, KVM Eksternal melalui IP dan lainnya adalah Lantronix Spider KVM

\n\n

Atau, Anda dapat membangun kembali versi DIY salah satunya dengan hanya dukungan keyboard (bukan dukungan keyboard, video, dan mouse) menggunakan Papan Pengembang USB Remaja terhubung ke a Modul ethernet WIZ712MJ dengan Kit Adaptor Ethernet WIZ812

\n\n

Pada saat itu Anda bisa menggunakan Teensy's ethernet dan papan ketik perpustakaan untuk menulis program kecil yang menelurkan server menggunakan Server.begin(), menunggu koneksi, dan kemudian memiliki loop sementara tak terbatas dengan sesuatu seperti konten berikut untuk membuat server gema jaringan satu arah: 

if (myclient.available()) {\n    Keyboard.print(myclient.read());\n}\n

Anda mungkin ingin menyempurnakannya dengan beberapa pemeriksaan kesalahan dan sejenisnya, tetapi itu harus berfungsi sebagai kerangka.

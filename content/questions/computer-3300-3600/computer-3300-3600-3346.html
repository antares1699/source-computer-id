Saya tidak tahu program apa pun yang melakukan ini (saya tidak membeli trek iTunes). Tetapi inilah langkah-langkah dasar, dan metode pilihan terakhir mutlak untuk mencapai apa yang ingin Anda lakukan.

\n\n

Metode umum:

\n\n
    \n
  1. Pecahkan enkripsi.
    2. \n
  2. Dekode data AAC ke PCM / WAV.
    3. \n
  3. Encode data PCM / WAV ke MP3.
    4. \n
\n\n

Metode kasus terburuk mutlak saya-tidak-bercanda-temukan-cara lain:

\n\n
    \n
  1. Rekam trek ke CDR menggunakan iTunes (Anda mungkin ingin menggunakan CDRW, terutama jika Anda memiliki banyak trek).
    2. \n
  2. Ripping trek dari disk yang dibakar ke MP3, menggunakan perangkat lunak ripping apa pun yang Anda inginkan. Jika Anda menggunakan iTunes untuk ripping, atur opsi encoding ke MP3.
    3. \n
  3. (opsional) seka disk CDRW dan bakar kumpulan trek berikutnya.
    4. \n
\n\n

(Perhatikan ini mungkin tidak akan berfungsi jika Anda tidak dapat memutar trek AAC di iTunes Anda.)

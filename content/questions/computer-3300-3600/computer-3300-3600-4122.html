Saya menggunakan beberapa jaringan IRC dan tidak aman untuk menyimpan kata sandi dalam teks biasa. Di sisi lain, mengetik setiap kata sandi setelah koneksi tidak nyaman.

\n\n

Bagaimana cara menyimpan kata sandi dalam file terenkripsi dan hanya mengetik kata sandi database pada saat pengaktifan Irssi? Apakah Anda tahu skrip yang menyediakan hal seperti itu?

","source":"https://superuser.com/questions/383022/how-to-safely-store-passwords-in-irssi","author":"Adrian Sad≈Çocha","replies":[{"reply":"Atau, WeeChat memungkinkan menyimpan data pribadi (seperti kata sandi) dalam file terenkripsi.","author":"Adrian Sadłocha"}],"answers":[{"answer":"

Siapkan eCryptFS, encfs, atau sistem file enkripsi serupa. Pasang di ~/Private atau serupa. Pindahkan ~/.irssi Anda ke direktori privat dan tautkan kembali ke tempat lama. 

ecryptfs-setup-private\necryptfs-mount-private\nmkdir ~/Private/config\nmv ~/.irssi ~/Private/config/irssi\nln -s \"Private/config/irssi\" ~/.irssi\n

eCryptFS direkomendasikan; ini ada di dalam kernel dan memberikan kinerja yang cukup baik. Tetapi ada opsi lain juga, seperti EncFS.

Pada Windows saya menggunakan PuTTY untuk masuk ke server jauh melalui SSH. Saya ingin menggunakan skrip batch ke SSH ke server jarak jauh menggunakan PuTTY. Server menjalankan OS berbasis Linux.

\n\n

Saya menggunakan perintah di bawah ini untuk melakukan ini: 

start C:\\Windows\\System32\\putty.exe -ssh server_name -l pankmish -pw wxyz\n

Namun saya mendapat kesalahan berikut:

\n\n
\n

tidak dapat terhubung ke host jarak jauh

\n
\n\n

Jika saya menggunakan perintah ini sebagai gantinya: 

start C:\\Windows\\System32\\putty.exe -ssh server_name -l user_name\n

Semuanya bekerja dengan baik dan saya mendapatkan jendela PuTTY dengan nama pengguna "user_name" di dalamnya. Jika saya memberikan kata sandi yang benar, saya dapat terhubung ke server. Namun melalui skrip batch saya tidak dapat memberikan kata sandi saat diminta.

\n\n

Bagaimana saya bisa memecahkan masalah ini?

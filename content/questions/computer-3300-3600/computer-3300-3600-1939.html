Oke, baru saja bereksperimen dengan Debian Wheezy guest di host Windows XP (VirtualBox 4.3.8).

\n\n

Prasyarat:

\n\n
    \n

  • Tuan rumah:

    \n\n
      \n
    • Buat folder bersama (menggunakan UI atau perintah vboxmanage).
      • \n

    • Perhatikan bahwa ia memiliki kedua nama jalan dan "nama pendek".

      \n\n

      Misalnya, saya mungkin membagikan folder C:\\Documents and settings\\User sehingga tersedia dengan nama user.

      • \n
    • \n

  • Tamu Debian:

    \n\n
      \n
    • Memiliki virtualbox-guest-dkms diinstal, dikompilasi untuk kernel saat ini (dilakukan secara otomatis; jika tidak, instal paket linux-headers yang cocok dan jalankan dpkg-reconfigure virtualbox-guest-dkms).
      • \n
    • Memiliki virtualbox-guest-utils untuk menyediakan mount.vboxsf.
      • \n
    • \n
\n\n

Sekarang tamu Anda bisa Anda lakukan 

# mkdir /tmp/foo\n# mount.vboxsf user /tmp/foo\n

atau, sebagai alternatif, langkah terakhir dapat dieja sebagai 

# mount -t vboxsf user /tmp/foo\n

Perhatikan penggunaan nama pendek untuk argumen "perangkat" dalam panggilan ke mount.

\n\n

Jawabannya berdasarkan komentar ini di utas berurusan dengan masalah serupa di tempat lain di SO.

\n\n
    \n
  1. Login sebagai root (atau gunakan sudo)
    2. \n
  2. Perbarui database APT Anda dengan apt-get update
    3. \n
  3. Instal pembaruan keamanan terbaru dengan apt-get upgrade
    4. \n
  4. Instal paket yang diperlukan dengan apt-get install build-essential module-assistant
    5. \n
  5. Konfigurasi sistem Anda untuk membangun modul kernel dengan menjalankan m-a prepare
    6. \n
  6. Klik Install Guest Additionsâ€¦ dari Menu perangkat, lalu jalankan mount /media/cdrom
    7. \n
  7. Buka VirtualMachine's Settings->General->Advanced->Shared Clipboard (Bidirectional) dan Settings->General->Advanced->Drag'n'Drop (Bidirectional)
    8. \n
  8. Jalankan sh /media/cdrom/VBoxLinuxAdditions.run,
    9. \n
  9. Instal pacapt sebagai pacman:
    wget -O /usr/local/bin/pacapt https://github.com/icy/pacapt/raw/ng/pacapt
    chmod 755 /usr/local/bin/pacapt
    ln -sv /usr/local/bin/pacapt /usr/local/bin/pacman || true
    10. \n
  10. Instal paket lain yang diperlukan (jawab Y ketika ditanya Y or I : install the package maintainer's version) :
    for i in virtualbox-guest-utils virtualbox-guest-additions virtualbox-modules virtualbox-host-modules virtualbox-source virtualbox-host-source; do echo $i ; pacman -S $i ; echo \"------------\" ; done
    11. \n
  11. Tambahkan vboxsf ke / etc / modules
    12. \n
  12. Jalankan modprobe vboxsf
    13. \n
  13. Periksa keberhasilan dengan: mount -t vboxsf tmp ~/shared/tmp/ (dengan asumsi folder bersama Anda ditentukan seperti di bawah ini)
    catatan: jika Anda mendapatkan /sbin/mount.vboxsf: mounting failed with the error: No such file or directory atau serupa, itu berarti Anda tidak membuat sub-direktori mount point.
    14. \n

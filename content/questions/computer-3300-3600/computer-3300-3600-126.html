Inilah yang Anda dapatkan dari setiap paket:

\n\n

sun-java6-bin: Basefiles untuk menjalankan program Java

\n\n

sun-java6-jre: File pelokalan dan semua yang Anda butuhkan untuk memenuhi persyaratan Runtime "lengkap". Tergantung pada bin.

\n\n

sun-java6-jdk: Semua yang Anda butuhkan untuk mengkompilasi sesuatu. Ini tidak perlu Runtime untuk menjalankan program java. Tergantung pada bin.

\n\n

Jadi Anda dapat menulis perintah Anda lebih pendek seperti: 

sudo apt-get install sun-java6-jre sun-java6-jdk\n

Oke, ini adalah jawaban kedua saya untuk pertanyaan ini, dan ini adalah cara Anda menginstal JDK saat ini di linux! (Ini untuk Debian, tetapi pengguna Ubuntu seharusnya tidak melihat banyak perbedaan).

\n\n

Alat yang Anda butuhkan: 

apt-get install fakeroot zip unzip mkisofs\n

Sekarang pergi ke java.sun.com dan unduh JDK6 terbaru. Seharusnya namanya kira-kira seperti jdk-6u20-linux-i586.bin!

\n\n

Kami tidak ingin menginstalnya secara langsung untuk membuatnya berperilaku seperti paket yang baik. Jadi kami menggunakan fakeroot untuk menginstalnya sebagai pengguna. 

mv jdk-6u20-linux-i586.bin ~someuser\nsu - someuser\nfakeroot\nchmod a+x jdk-6u20-linux-i586.bin\nsh ./jdk-6u20-linux-i586.bin\ncd jdk1.6.0_20/man\nfor i in `find -type f`; do gzip $i; done;\ncd ../..\n# Tar the results to have them handy for the real installation\ntar cfz jdk1.6.0_20.tgz jdk1.6.0_20/\n

Sekarang kita tinggalkan fakeroot (Ctrl-D) dan someuser-shell (Ctrl-D lagi), dan instal tar kita ke / usr / local: 

cd /usr/local\ntar xfvz ~someuser/jdk1.6.0_20.tgz\nln -s /usr/local/jdk1.6.0_20/ /usr/local/jdk\nln -s /usr/local/jdk1.6.0_20/jre/ /usr/local/jre\n\nfor program in appletviewer apt extcheck idlj jar jarsigner java javac \\\n           javadoc javah javap javaws jconsole jdb jhat jinfo jmap \\\n           jps jrunscript jsadebugd jstack jstat jstatd jvisualvm \\\n           keytool native2ascii orbd pack200 policytool rmic rmid \\\n           rmiregistry schemagen serialver servertool tnameserv \\\n           unpack200 wsgen wsimport xjc ; do \\\n    update-alternatives --install \"/usr/bin/$program\" \"$program\" \\\n    \"/usr/local/jdk1.6.0_20/bin/$program\" 1130 \\\n    --slave \"/usr/share/man/man1/$program.1.gz\" \"$program.1.gz\" \\\n    \"/usr/local/jdk1.6.0_20/man/man1/$program.1.gz\"; done;\n

Biarawati testen ob alles ok ist: 

java -version\nman java\n

Selesai.

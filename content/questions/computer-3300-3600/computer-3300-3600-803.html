Saya menggunakan Windows 7 Ultimate 64-bit. Sejak beberapa minggu, saya tidak lagi memiliki screensaver di layar login Windows. Jika pengguna mencatat dan kemudian mengunci sesinya, screensaver akan dimulai pada akhirnya tetapi jika tidak ada pengguna yang masuk, layar login tidak akan pernah memulai screensaver.

\n\n

Ada yang tahu cara memasang screensaver di sana?

Saya tidak 100% yakin apakah langkah-langkah ini akan berhasil jika seseorang tidak masuk!

\n\n
\n

Untuk mengaktifkan screensaver, pertama\n pilih screensaver yang Anda inginkan. Lihat\n di C: \\ WINDOWS \\ SYSTEM32 untuk file dengan\n ekstensi .SCR - biasanya ada\n adalah sebagai berikut (setidaknya di saya\n Instalasi Perusahaan) ...

\n \n
    \n
  • SSTEXT3D - Teks 3D
    • \n
  • BUBBLES.SCR - Gelembung
    • \n
  • MYSTIFY.SCR - Membingungkan
    • \n
  • PHOTOSCREENSAVER.SCR - Foto
    • \n
  • RIBBONS.SCR - Pita
    • \n
\n \n

Salah satu hal yang harus diperhatikan adalah\n bahwa Anda tidak memiliki Direct3D\n dukungan di layar masuk sehingga\n screensaver bawaan di atas tidak akan berfungsi.\n Kecuali Anda memiliki file lain\n ingin mengambil file LOGON.SCR dari\n sistem operasi sebelumnya.

\n \n

Setelah Anda memiliki screensaver (di file\n case LOGON.SCR), buka registri Anda\n editor dan arahkan ke berikut ini\n kunci - 

HKEY_USERS \\ .Default \\ Control Panel \\ Desktop\n

\n \n

Di bawah kunci ini buat yang berikut ini\n Nilai REG_SZ ... 

SCRNSAVE.EXE = C:\\WINDOWS\\SYSTEM32\\LOGON.SCR\nScreenSaveActive = 1\nScreenSaveIsSecure = 0\nScreenSaveTimeOut = 60 (or whatever timeout you want in seconds)\n

Itu saja - Anda akan mendapatkan file\n screensaver di layar masuk lalu.

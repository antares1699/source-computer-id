Saya menjalankan Max OSX 10.9.3 dan saya mencoba menyiapkan berbagi file SSHFS antara MacBook Pro saya dan sistem file jarak jauh. Namun, ketika saya mencoba melakukannya, tidak berhasil.

\n\n

Anehnya, itu membuat direktori target menghilang. Adakah orang lain yang melihat ini terjadi? Apakah ini bug?

\n\n

Pertama, lihat bahwa saya dapat melakukan ssh secara normal ke mesin target: 

% ssh remoteuser@XXX.XXX.XXX.XXX            # <--- SSH to remote system works! See below.\nremoteuser@XXX.XXX.XXX.XXX % ls -altr remoteDir\ntotal 8\ndrwxr-xr-x 26 remoteuser remoteuser 4096 Jun 22 01:00 ..\ndrwxrwxrwx  2 remoteuser remoteuser 4096 Jun 22 01:08 .\nremoteuser@XXX.XXX.XXX.XXX % exit\n%                                           # <--- Logged out of remote system\n

Selanjutnya, saya membuat direktori secara lokal dan memverifikasi itu dibuat: 

% pwd\n/mnt\n\n% ls\ntotal 0\ndrwxr-xr-x  31 root  admin  1122 Jun 18 18:34 ../\ndrwxr-xr-x   2 root  admin    68 Jun 23 08:11 ./\n\n% sudo mkdir share1 \n% ls\ndrwxr-xr-x  31 root  admin  1122 Jun 18 18:34 ../\ndrwxr-xr-x   4 root  admin   136 Jun 23 08:50 ./\ndrwxr-xr-x   2 root  admin    68 Jun 23 08:50 share/\n

Sekarang saya mencoba mengatur share SSHFS: 

% sudo sshfs remoteuser@XXX.XXX.XXX.XXX:remoteDir /mnt/share1\nremoteuser@XXX.XXX.XXX.XXX's password:\n%\n

Baik. Ini terlihat telah berfungsi. Tidak ada kesalahan. Jadi mari kita lihat bagian yang kita buat, oke? 

% ls\nls: share1: No such file or directory\ntotal 0\ndrwxr-xr-x  31 root  admin  1122 Jun 18 18:34 ../\ndrwxr-xr-x   3 root  admin   102 Jun 23 08:12 ./\n

Apa? Bukan hanya File Sharing tidak berfungsi, tetapi direktori share1 tampaknya telah lenyap! (Meskipun sistem file sepertinya tahu itu hilang, yang aneh).

\n\n

Kemana perginya / mnt / share1 dan bagaimana cara menyiapkan SSHFS ini?

Saya menambahkan 'allow_other' dan itu berhasil untuk saya. Ini string penghubung saya: 

sshfs -p 22 user@example.net:/home/user/ ~/RemoteHome -oauto_cache,reconnect,defer_permissions,negative_vncache,allow_other,volname=RemoteHome\n

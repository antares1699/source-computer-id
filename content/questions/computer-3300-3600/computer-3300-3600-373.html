Memulai program yang meluncurkan konsol memang akan menutup jendela, karena CMD tidak lagi diperlukan.

\n\n

Jika Anda menekan start dan mengetik cmd dan menekan enter, Anda akan meluncurkan prompt perintah.

\n\n

Tekan driveletter program aktif jika ini tidak di C :, diikuti oleh a: dan tekan enter.

\n\n

Dalam contoh di bawah ini, _ di akhir mewakili kursor Anda.

\n\n

Contoh: 

C:\\Users\\User>D:_\n\nD:\\>_\n

sekarang, gunakan perintah cd untuk menavigasi ke direktori program. Anda bisa menekan tab untuk melengkapi nama secara otomatis. Terlalu tab akan membuat Alat jika folder itu ada.

\n\n

Contoh: 

D:\\>cd Tools_\n\nD:\\Tools>cd \"My Program\"_   <- type 'My' and press tab to Autocomplete to \"My Program\"\n\nD:\\Tools\\My Program>_\n

Sekarang masukkan nama program yang ingin Anda jalankan.

\n\n

Contoh: 

D:\\Tools\\My Program>program_\nHello, I am the error message you want to see.\n\nD:\\Tools\\My Program>_\n

Anda dapat menggunakan file .bat untuk menjalankan exe itu.

\n\n

Juga, Anda dapat menggunakan perintah berikut di cmd untuk mencatat kesalahan / output dari file exe itu ke dalam file teks:

\n\n

myFile.exe >> "C: \\ logFile.txt"

\n\n

Dua kali lebih besar dari simbol ditambahkan ke file yang ada ">>"\nnamun, simbol tunggal menimpa file yang ada ">".

\n\n

Semoga ini bisa membantu Anda untuk mencatat kesalahan.

Menurut Ini:

\n\n
\n

|| [...] perintah1 || command2 Gunakan untuk menjalankan perintah berikut ||\n hanya jika perintah sebelumnya || gagal. Cmd.exe menjalankan file\n perintah, dan kemudian menjalankan perintah kedua hanya jika perintah pertama\n tidak berhasil diselesaikan (menerima kode kesalahan lebih dari\n nol).

\n
\n\n

dan jika Anda membuat .exe, Anda hanya perlu meletakkan perintah pause setelah kalimat Anda.

\n\n

Dan contoh di c ++ yang memanggil bash: 

#include <iostream>\n#include <windows.h>\n\n\nusing namespace std;\n\nint main()\n{\n    system(\"cd  akfhaskfn|| pause\");\n    system(\"cd  || pause\");\n\n    return 0;\n}\n

Anda akan melihat bahwa hanya yang pertama yang akan memanggil perintah jeda.

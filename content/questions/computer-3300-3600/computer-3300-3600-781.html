Saya telah menyetel hlsearch di .vimrc saya.

\n\n

Masalahnya adalah: ketika saya menemukan apa yang saya butuhkan dalam dokumen saya, saya ingin itu secara otomatis menonaktifkan hlsearch ketika saya memasuki mode penyisipan untuk string yang dicari itu.

\n\n

Jadi, jika saya mencoba mencari string lain, itu dimulai lagi. Perilaku ini terkenal di antara editor lainnya.

\n\n

Bagaimana saya melakukannya?

","source":"https://superuser.com/questions/156248/disable-set-hlsearch-when-i-enter-insert-mode","author":"Somebody still uses you MS-DOS","replies":[],"answers":[{"answer":"

nonaktifkan penyorotan saat memasuki insertmode melalui saya : 

noremap i :set nohlsearch<CR>i\n

untuk mengaktifkannya kembali dengan keluar dari mode penyisipan (melalui ESC ): 

inoremap <ESC> <ESC>:set hlsearch<CR>\n

Anda bisa menerapkan ide ini ke semua jenis kunci seperti Sebuah , Hai dll.

\n\n

ATAU 

autocmd InsertEnter * :set nohlsearch\nautocmd InsertLeave * :set hlsearch\n

(pujian untuk ide acara 'InsertEnter')

Saya menemukan pendekatan yang sama sekali berbeda, tetapi itulah PERSIS yang saya cari. 

:set hlsearch\nautocmd InsertEnter * :let @/=\"\"\nautocmd InsertLeave * :let @/=\"\"\n
\n

Ini mengatur register "pencarian terakhir"\n ke string kosong, jadi tidak ada\n akan disorot. Pencarian sebelumnya\n masih bisa diingat dengan menggunakan\n kunci setelah menekan / atau? dan\n opsional "awal" dari string\n untuk dicari.

Saya mencoba melakukan hal yang sama baru-baru ini dan dalam menelusuri web menemui pertanyaan Anda. Saya juga gagal menemukan cara apa pun untuk menggunakan folder pencarian untuk memeriksa header. Asumsi saya adalah karena header internet sebenarnya bukan "kolom Exchange", artinya saya tidak percaya email "intranet" yang dikirim dalam server Exchange itu sendiri (seperti email yang Anda kirim dengan gambar kucing lucu [LOLZ!] Ke rekan kerja Anda ...) memiliki tajuk seperti itu. Tapi inilah solusi yang saya dapatkan yang saya gunakan.

\n\n

Saya menyiapkan aturan baru di wizard aturan yang menerima pesan baru untuk memeriksa header. Jika string tertentu ditemukan, tetapkan pesan ke kategori tertentu. Dalam kasus saya, saya mencari pesan yang berasal dari server Pelacak Permintaan Praktis Terbaik kami, di masing-masing header tersebut ada string "RT-tiket:". Jika aturan saya menemukan string ini di header, mereka kemudian menetapkan pesan ke kategori yang saya buat bernama "Tiket RT". (Saya membuat kategori ini tanpa warna sehingga tidak terlalu mencolok.) Saya kemudian dapat membuat folder pencarian yang mencari pesan dari kategori itu saja. Langkah ekstra tetapi masalah terpecahkan. Saya telah melakukan ini di Outlook 2007 tetapi seharusnya berfungsi di semua versi Outlook yang mendukung tajuk pemeriksaan panduan aturan dan kemudian menetapkan ke kategori.

\n\n

Kekurangannya adalah 1) aturan lain dalam daftar saya yang terus bertambah, dan 2) jika saya ingin menerapkan ini ke email yang sudah diterima, saya harus melalui dan secara manual menjalankan aturan terhadap folder tersebut terlebih dahulu. Pada titik ini saya hanya peduli tentang pesan baru yang saat ini ada di kotak masuk saya atau yang akan datang, jadi saya cukup menetapkan aturan untuk dijalankan pada saat pembuatan dan itu sudah diurus. Manfaat yang ditemukan dari penggunaan kategori adalah Anda dapat menampilkannya sebagai kolom dalam daftar pesan Anda. Saya tidak yakin saya bahkan akan menggunakan folder pencarian seperti yang saya inginkan, saya dapat pergi dan mengurutkan kotak masuk saya berdasarkan kategori untuk menemukan pesan yang diinginkan.

\n\n

Jika ada yang membutuhkan langkah-langkah membuat aturan diperluas, saya dapat melakukannya, tinggalkan komentar. Saya berharap bahwa jika seseorang cukup berpengetahuan untuk menggali di header email internet bahwa membuat aturan Outlook akan menjadi topi lama. Penyihir yang mereka miliki cukup mudah.

Skrip PowerShell ini mencari kecocokan pada semua header di Inbox. Mungkin perlu beberapa saat untuk menjalankannya, tergantung pada ukuran kotak masuk Anda. Beberapa peringatan berlaku, tidak sedikit di antaranya adalah kemungkinan proses outlook.exe sisa. Ini dapat dimatikan baik secara manual di Task Manager, atau secara terprogram melalui get-process \"outlook\" | kill. Diasumsikan Anda memiliki akses ke PoweShell karena keberadaannya di mana-mana, namun OS tertentu yang Anda gunakan mungkin memiliki dukungan terbatas untuk ini. 

$MatchString = \"X-Mailer: YahooMailWebService/0.8.201.700\"\nAdd-type -assembly \"Microsoft.Office.Interop.Outlook\" | out-null\n$namespace = new-object -comobject outlook.application\n$MAPI = $namespace.GetNamespace(\"MAPI\")\n$Inbox = $MAPI.GetDefaultFolder([Microsoft.Office.Interop.Outlook.OlDefaultFolders]::olFolderInbox).Items\n$Headers = `\n    foreach ( $MailItem in $Inbox ) { \n        $MailItem.PropertyAccessor.GetProperty(\"http://schemas.microsoft.com/mapi/proptag/0x007D001E\") \n    }\n$namespace.Quit()\n$MatchingHeaders = $Headers | where { $_.contains( $MatchString ) }\n#sample output\n$MatchingHeaders | Select-Object -First 1\n

Jika ada kumpulan kecocokan bukan nol, sesuatu seperti berikut ini dikembalikan.

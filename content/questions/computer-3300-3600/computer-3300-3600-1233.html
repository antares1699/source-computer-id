Seorang teman memiliki MacBook (13 inci Akhir 2007), entah bagaimana dia berhasil mengacaukan semua tombol fungsi.

\n\n

Saya telah berhasil mengatur ulang untuk dasbor, mengekspos, dll tetapi putar / jeda / maju cepat / mundur / naikkan volume / kurangi volume tidak berfungsi.

\n\n

Ada ide tentang apa yang terjadi di sini dan bagaimana cara memperbaikinya?

","source":"https://superuser.com/questions/31022/reset-mac-function-keys","author":"cust0s","replies":[],"answers":[{"answer":"

Ada beberapa hal:

\n\n
    \n
  1. Menu Apple -> Preferensi Sistem -> Keyboard
    2. \n
  2. Hapus centang "Gunakan Semua tombol F1, F2, dll sebagai tombol standar"
    3. \n
  3. Klik "Pintasan Keyboard"
    4. \n
  4. Klik "Pulihkan Default"
    5. \n
\n\n

Itu seharusnya memperbaiki masalah.

\n\n

Sunting :

\n\n

Beberapa orang menyarankan untuk mematikan MBP, atau mengatur ulang SMC sepenuhnya. Ini a utas forum yang menggambarkan situasi serupa.

\n\n

Ini dia cara mengatur ulang SMC di Portabel Apple terbaru.

\n\n

SMC bertanggung jawab atas beberapa fungsi IO sistem, seperti trackpad, serta fungsi manajemen daya. Sangat mungkin keyboard Anda tidak berfungsi karena SMC ini.

\n\n

Selain itu, apakah perilaku ini terjadi di bawah akun pengguna baru? Jika tidak, saya akan cenderung membuang file apa pun dari pustaka pengguna pertama yang terkait dengan keyboard - atau cukup bermigrasi ke akun pengguna baru, mana saja yang lebih mudah.

Saya memiliki server Linux dengan sekitar 20 pengguna. Mereka menggunakan perintah wall yang mengganggu. Saya memiliki izin root dan masih ingin menggunakannya sendiri. Tapi, bagaimana saya bisa membatasi pengguna biasa untuk menggunakan perintah Linux wall?

","source":"https://superuser.com/questions/373897/how-can-i-disable-any-users-except-root-from-executing-wall","author":"isara","replies":[{"reply":"mesg n, dapat menolak pesan yang berfungsi untuk pengguna individu, tetapi saya mencari cara untuk menonaktifkan / menolak semua pengguna kecuali root pada host ini.","author":"isara"}],"answers":[{"answer":"

Jika Anda ingin menolak akses ke untuk semua pengguna, Anda mungkin ingin menghapus bit setgid pada /usr/bin/wall: 

$ ls -l /usr/bin/wall\n-rwxr-sr-x 1 root tty 13884 2011-08-09 12:15 /usr/bin/wall\n$ sudo chmod g-s /usr/bin/wall\n\n$ echo foo | wall\n

Anda harus selalu dapat mem-walli diri Anda sendiri dan root harus dapat mem-wall semua orang, tetapi pengguna A tidak dapat menyiarkan ke pengguna lainnya.

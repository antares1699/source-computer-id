Ini adalah pertanyaan lanjutan untuk pertanyaan lain ini , yaitu tentang keputusan Google untuk mulai tidak mempercayai sertifikat CA Root Symantec tertentu. Microsoft di sisi lain belum membuat (belum) keputusan apa pun tentang sertifikat Root CA itu dan masih ada di mesin Windows 7 saya.

\n\n

Memperbarui: Ini gambar sertifikat Root CA dari mesin saya. Sidik Jari di sini sama dengan Sidik Jari Versi MD2 (SHA-1) pada sertifikat CA Root yang diterbitkan oleh Google di posting blog mereka :

\n\n

74:2C:31:92:E6:07:E4:24:EB:45:49:54:2B:E1:BB:C5:3E:61:74:E2

\n\n

\n\n

Katakanlah saya menghapus sertifikat dari PC saya tetapi kemudian saya menelusuri (menggunakan, katakanlah, IE v.11) ke beberapa situs web yang mengidentifikasi dirinya sendiri menggunakan sertifikat itu.

Tipe about: config , jawablah bahwa Anda akan berhati-hati, lalu cari kata kunci.URL

\n\n

Mengetik string berikut di sana akan membuat perilaku bilah alamat Firefox terlihat lebih mirip dengan Chrome:

\n\n
\n

http://www.google.com/search?btnG=Google+Search&q=

\n
\n\n

dan set kata kunci.enabled untuk benar .

\n\n

Lihat http://support.mozilla.org/en-US/questions/759303

\n\n

Untuk membuat Firefox terlihat lebih mirip dengan Chrome, Anda mungkin juga mempertimbangkan untuk mengunduh tema https://addons.mozilla.org/en-uS/firefox/addon/fxchrome/ serta meletakkan unduhan di bilah bawah, seperti yang dilakukan Chrome: https://addons.mozilla.org/en-US/firefox/addon/download-statusbar/

\n\n

Diuji pada 16.0.1 di Mac.

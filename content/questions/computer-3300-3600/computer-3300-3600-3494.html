Untuk menyambungkan pembaca USB 2.0 (cepat), Anda memerlukan kartu PCI yang disebut "USB 2.0 kartu pengontrol host ", bukan" ekspansi "(karena tidak ada USB 2.0 untuk" memperluas "). Contoh: Startech .

\n\n

Tidak ada pengontrol host USB 2.0 dalam format PCIexpress, mereka hanya ada dalam standar PCI lama, jadi Compaq 2002 lama Anda akan baik-baik saja.

\n\n

Anda perlu mencari driver OS yang tepat untuk ini, dan driver yang lebih atau kurang fungsional hanya ada di bawah Windows XP dan di atasnya. Jika Anda memiliki Vista W2k, proyek ini mungkin menghadapi beberapa tantangan.

Saya ingin menghapus daftar panjang direktori saat ini di prompt shell.

\n\n

Terkadang mengganggu ketika Anda menggunakan jendela shell kecil dan itu menempati seluruh ruang di jendela

\n\n

Saya lebih suka hanya direktori saat ini pada prompt.

\n\n
\n

root @ Kiran: ~ / Downloads / Director1 / Director2 / Director3 / Director4 #\n Saya lebih suka sesuatu seperti:\n root @ Kiran: .. Directory4 #

\n
\n\n

atau bahkan lebih baik,

\n\n
\n

root: Directory4 #

\n
\n\n

Saya memiliki disk USB yang telah mengalami bollixed dan seumur hidup saya, saya tidak dapat memformatnya ulang dan kembali ke titik awal.

\n\n

Saya mencoba membuat file Kit Kelangsungan Hidup Acronis menggunakan disk USB WD Elements 2TB. Acronis sedang memformat ulang disk agar dapat di-boot, tetapi melontarkan pesan kesalahan dan gagal. Itu membuat drive USB tidak dapat digunakan; itu tidak muncul di Windows Explorer saat dicabut dan kemudian dicolokkan kembali.

\n\n

Saya mencoba menggunakan Manajemen Disk untuk memformat ulang tetapi tidak berhasil. Saya membuat volume sederhana tetapi formatnya gagal:

\n\n

"Volume berhasil dibuat tetapi tidak diformat. Coba lagi format menggunakan tugas format."

\n\n

Melakukan ini juga gagal:

\n\n

"Windows tidak dapat memformat volume karena volume sedang offline. Coba online volume dengan menetapkan huruf drive atau jalur ke volume terlebih dahulu."

\n\n

Saya bisa menetapkan huruf drive, tapi ini tidak membawa saya ke mana-mana; format masih gagal karena volumenya offline. Ini memang mendapatkan disk untuk ditampilkan di Windows Explorer, tetapi tidak dapat diakses.

\n\n

Upaya ini membawa saya ke batas keahlian saya; Saya mulai menggunakan DOS di pertengahan 80-an tetapi saya belum pernah berurusan dengan masalah semacam ini sebelumnya. Saya juga mencoba menggunakan DISKPART dari baris perintah untuk melihat apakah saya dapat membuat beberapa kemajuan (termasuk mencoba online disk), tetapi tanpa banyak pengalaman dengan alat itu, saya tidak berhasil.

\n\n

Saya berharap disk USB ini dapat diselamatkan dan dapat diformat ulang serta dibawa kembali dengan beberapa langkah sederhana yang dapat dibantu oleh orang yang lebih berpengetahuan.

Anda dapat mencoba alat pihak ketiga seperti gratis DiskGenius atau MiniTool Partition Wizard untuk sepenuhnya menghapus semua partisi. Kemudian buat satu partisi NTFS.

\n\n

Di DiskGenius,

\n\n
    \n
  • Klik kanan pada drive untuk setiap hal berikut.
    • \n
  • Ubah Status Perangkat untuk memastikannya online, bukan hanya baca.
    • \n
  • Hapus Semua Partisi .
    • \n
  • Partisi Cepat , mungkin memilih 1 partisi, NTFS, dari disk penuh.
    • \n
\n\n

Secara opsional, Anda dapat mencoba,

\n\n
    \n
  • Verifikasi Atau Perbaiki Sektor Buruk , jika ada masalah di atas.
    • \n
  • Ubah Mode Boot , agar dapat di-boot.
    • \n
\n\n

\n\n

Beri tahu saya jika ini menyelesaikan masalah.

Bagaimana cara membuat AppData muncul di File Explorer?

\n\n
    \n

  1. Menu "Alat"> "Opsi folder ..."

    2. \n

  2. Klik tab "Tampilan"

    3. \n

  3. Gulir ke bawah hingga Anda melihat "File dan folder tersembunyi"

    4. \n

  4. Aktifkan "Tampilkan file, folder, dan drive tersembunyi"

    5. \n

  5. Klik "OK"

    \n\n

    6. \n
\n\n
\n\n

Apa izin file di bawah properti yang membuatnya tersembunyi atau tidak?

\n\n
    \n

  1. Klik kanan folder AppData

    2. \n

  2. Pilih "Properties"

    3. \n

  3. Klik tab "Umum"

    4. \n

  4. Aktifkan / Nonaktifkan "Tersembunyi" sebagaimana mestinya

    \n\n

    5. \n

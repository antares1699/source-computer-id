Sekarang Anda dapat berbagi folder dari OS tamu ke windows => karena saya tidak bisa membuatnya bekerja sebaliknya. Ini berguna bagi saya karena saya jarang mengangkut barang dari vista ke ubuntu. Dengan mindterm diinstal pada Windows dan opensh-server diinstal pada OS tamu, saya kemudian dapat mengaktifkan transfer file dua arah antara win7 dan ubuntu, yang sangat ideal, menggunakan plugin SFTP.

Apa yang berhasil bagi saya adalah mengalihkan "Network Adapter" saya di VirtualBox dari "NAT" ke "Bridged". Ini dapat dilakukan di menu VirtualBox, "Perangkat" -> "Adaptor Jaringan". Semua ini perlu dilakukan setelah menginstal "Edisi Tamu". Untuk beberapa alasan, penginstalan saya default ke "NAT", yang tidak pernah bisa saya lihat di komputer jaringan saya.

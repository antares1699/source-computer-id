Dengan CD WinPE saya selalu mendapatkan prompt "Tekan sembarang tombol untuk boot ke CD / DVD". Bagaimana cara membuat CD yang secara otomatis melakukan boot ke CD tanpa perintah?

","source":"https://superuser.com/questions/228727/automatically-boot-cd","author":"kyle","replies":[],"answers":[{"answer":"

Perilaku yang Anda lihat khusus untuk disk PE yang menjalankan segala bentuk Windows PE. Lebih khusus lagi, ini terjadi jika ada sektor boot yang terdeteksi pada hard drive.

\n\n

Cara mudah untuk mendemonstrasikan hal ini adalah dengan menghapus semua partisi dari hard drive dan memasukkan disk PE Anda - ini akan segera boot tanpa disuruh.

\n\n

Mengenai menjawab pertanyaan Anda, saya tidak percaya itu mungkin.

WinPE sendiri berisi satu "bootfix.bin". Saat mengonversi WinPE ke ISO, ISO tersebut akan memiliki satu lagi "bootfix.bin".

\n\n

Saya pikir saat boot melalui WIMBOOT itu akan menggunakan bootfix.bin di dalam file * .wim.\nDan ketika boot melalui ISO itu akan menggunakan bootfix.bin di bawah "media / boot / bootfix.bin" (ini untuk 32bit).

\n\n

Saya bisa boot tanpa "tekan sembarang tombol prompt" setelah menghapus "bootfix.bin" dari file WIM dan juga dari file ISO.

Apakah ada alat baris perintah untuk mengukur seberapa cepat file teks berkembang, dalam satuan baris / detik?

\n\n

Untuk file small-ish, saya dapat melakukan sesuatu dengan watch dan wc -l, tetapi itu mengukur file lagi setiap saat. Biasanya saya ingin menonton file log yang besar, atau saluran pipa.

\n\n

Jika ada alat yang baru saja membaca stdin, dan setiap detik alat itu mengeluarkan berapa banyak baris baru yang telah dibaca selama detik terakhir, itu akan sempurna.

\n\n

Mungkin cukup mudah untuk menyatukan sesuatu di salah satu HLL pilihan saya, tetapi jika sudah ada alat yang melakukan ini, itu lebih baik.

Saya mendapatkan alamat ip saya sendiri (mungkin 123.45.6.7). 

iptables -I INPUT -s 123.45.6.7 -j DROP\n

Sekarang saya diblokir sendiri, ketika saya login server dengan pc lain, bagaimana cara menulis perintah iptables untuk membuka blokir diri saya sendiri?

\n\n

Saya tidak ingin menjalankan iptables -F.

","source":"https://superuser.com/questions/832314/how-to-unblock-ip-address-with-iptables-command","author":"showkey","replies":[{"reply":"Mungkin duplikat dari http://askubuntu.com/questions/193858/how-to-delete-or-unblock-an-ip-address-listed-in-iptables-firewall","author":"vembutech"}],"answers":[{"answer":"

Jika hanya satu aturan yang cocok dengan tindakan dan alamat sumber, Anda dapat menghapus aturan hanya dengan mengganti -I (sisipkan) dengan -D (hapus): 

iptables -D INPUT -s 123.45.6.7 -j DROP\n

Jika ada beberapa aturan yang cocok, Anda dapat mencantumkan semua aturan: 

iptables -L INPUT -n --line-numbers\n

dan hapus aturan dengan nomornya n : 

iptables -D INPUT n\n

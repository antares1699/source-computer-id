ReadyBoost memang menghasilkan peningkatan kinerja yang sangat besar terlepas dari berapa banyak RAM yang telah Anda instal di sistem Anda. ReadyBoost bekerja dengan menyimpan data yang sering digunakan ke memori flash perangkat yang dialokasikan untuk menggunakan ReadyBoost. Cache ReadyBoost bekerja bersama-sama dengan file halaman sistem.

\n\n

File halaman di Windows sebenarnya adalah area hard disk yang dialokasikan seolah-olah itu adalah RAM. Oleh karena itu, Windows menyimpan data dan menjalankan program dari area hard disk yang dialokasikan ini dengan cara yang sama seperti RAM. Ketika ReadyBoost berfungsi dan diaktifkan (menggunakan perangkat yang cukup cepat seperti kartu SD SDHC ultra Sandisk 15MB / dtk, atau flash drive berkinerja tinggi), ini bekerja dengan file halaman dengan memindahkan data yang jika tidak akan dimasukkan ke harddisk. disk ke perangkat memori flash yang dialokasikan oleh ReadyBoost.

\n\n

File halaman yang lebih besar sebenarnya menurunkan kinerja dengan memaksa drive mekanis untuk membaca dan menulis data lebih sering. Kami memiliki RAM di komputer kami karena ketika kami meluncurkan program atau file, windows mengambil data dari hard drive (program, file, dll) dan memindahkannya ke RAM, tempat data diproses secara tradisional. Gunakan kartu Sandisk SD 15MB / s atau flashdrive, atur file halaman Anda ke 2GB atau lebih jika Anda mau.

\n\n

Apa pun itu, Windows akan menyimpan data ini ke cache ReadyBoost alih-alih hard disk Anda. Anda akan melihat peningkatan kinerja yang sangat besar. Selain itu, gunakan hanya hard drive 7200 RPM dan RAM tercepat yang didukung sistem Anda. Selain itu, sebaiknya gunakan minimal RAM 4 GB.

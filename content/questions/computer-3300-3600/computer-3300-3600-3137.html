Gmail dan hotmail sendiri tidak merender HTML. Bagian HTML email adalah dirender oleh browser tempat mereka sedang dilihat.

\n\n

Namun, jika Gmail, misalnya, menampilkan stylesheet, atau menyematkan gaya untuk antarmuka Gmail, itu aku s mungkin gaya tersebut memengaruhi cara perenderan browser anda HTML; namun, mereka mungkin menggunakan atribut class dan id yang tidak Anda gunakan di html, jadi seharusnya tidak menjadi masalah.

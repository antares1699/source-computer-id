Apakah D: \\ drive NTFS? Jika demikian, maka satu atau lebih file dapat memiliki ekstensi Aliran Data Alternatif yang merupakan cara melampirkan metadata ke file. Mereka biasanya tidak terlihat oleh filesystem, jadi Anda membutuhkan alat seperti ADS viewer. Biasanya metadatanya kecil, tetapi tidak ada yang menghentikan file agar tidak memiliki file besar yang dilampirkan ke ADS-nya. (Perangkat lunak perusak juga suka bersembunyi di arus.)

\n\n

Sysinternals menawarkan utilitas (gratis) Aliran untuk dapat melihat / menghapus ADS. Buka prompt perintah dan jalankan streams -s d:\\*. Ini akan menampilkan semua aliran data alternatif dari semua file di drive. Kemudian Anda dapat menggunakan sakelar -d untuk menghapus ADS yang besar dan tidak diinginkan.

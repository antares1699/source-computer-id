Sebagai catatan tambahan: saat menggunakan Time Machine Anda mungkin tertarik untuk mengecualikan file log, yang secara default disembunyikan di direktori yang bahkan akun admin tidak dapat mengakses: 

ls -la /usr/local/mysql-5.0.51a-osx10.5-x86\n[..]\ndrwxr-x---   4 _mysql  wheel    136 Jul 10 23:06 data\n[..]

Atau, ubah pengaturan MySQL untuk memindahkan log ke beberapa folder temp.

\n\n

(Untuk menghapus file log MySQL yang ada dari disk cadangan, Anda perlu jalankan Time Machine sebagai root .)

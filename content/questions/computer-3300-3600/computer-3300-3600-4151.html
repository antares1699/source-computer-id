Penting untuk diingat bahwa URL adalah cara untuk menentukan sumber daya. Sumber daya ini bisa dari jenis apa pun. Di situs yang lebih sederhana, sumber daya meminta file yang terletak di folder tertentu di server. Tapi itu menjadi jauh lebih umum untuk URL menjadi kurang langsung 'berikan saya file ini', dan sebagai gantinya diurai oleh mesin web.

\n\n

Di situs yang lebih canggih ini (atau setidaknya lebih didorong oleh perangkat lunak khusus), URL lebih merupakan petunjuk ke perangkat lunak daripada penentu untuk file tertentu, dan karenanya tidak memiliki alasan untuk menyertakan ekstensi (yang hanya merupakan cara untuk sistem operasi untuk dengan mudah mengetahui jenis file apa itu).

\n\n

Anda tidak meminta file di situs seperti superuser; Anda membuat kueri terhadap API (bisa dikatakan begitu). Jadi tidak membutuhkan ekstensi file.

StackExchange adalah aplikasi ASP.NET MVC. MVC berarti Model, View dan Controller. URL https://superuser.com/questions/ask adalah mengatakan pergi ke pengontrol 'pertanyaan' dan temukan saya metode 'tanya' yang kemudian akan berbalik dan membuat tampilan kembali ke pengguna. Jadi dengan satu cara Anda tidak benar-benar menavigasi ke halaman, Anda sedang menavigasi ke metode programatik yang terletak di server web.

\n\n

Kedua, ada situs lain yang mungkin memiliki URL seperti itu https://superuser.com/questions/ask yang sebenarnya menulis ulang url di latar belakang jadi bukan http://superuser.com?q=123&a=789 , Anda akan mendapatkan URL cantik https://superuser.com/questions/ask . Jenis pekerjaan latar belakang ini dapat ditemukan di aplikasi seperti WordPress.

\n\n

Semoga ini bisa membantu menjelaskan hal ini untuk Anda.

"File" ini seringkali sebenarnya adalah direktori. Sebagian besar server mengizinkan pengembang untuk menentukan file indeks default (biasanya index.html), sehingga URL untuk direktori dapat dimasukkan tanpa nama file individual. Misalnya, Apache akan mengizinkan Anda untuk mengonfigurasi ini menggunakan direktif DirectoryIndex di file .htaccess: 

DirectoryIndex index.html\n

Dan kemudian permintaan untuk direktori yang berisi index.html akan disajikan dengan file indeks.

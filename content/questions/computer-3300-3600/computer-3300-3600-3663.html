Jika tersedia, organisasi akan dapat membuat bagian pribadi sendiri dari Windows Store yang ada. Otentikasi untuk mengakses toko pribadi tersebut akan diaktifkan melalui Azure Active Directory dan akan menampung aplikasi khusus organisasi. Aplikasi dapat diberikan kepada pengguna melalui admin TI dan / atau pengguna diberi akses ke bagian pribadi organisasi di Store yang memungkinkan untuk mengunduh aplikasi yang telah disetujui sebelumnya.

\n\n

Rincian lebih lanjut tentang ini dapat ditemukan di sini: https://technet.microsoft.com/itpro/windows/whats-new/windows-store-for-business-overview

\n\n

Karena tidak ada garis waktu yang tersedia untuk umum tentang kapan Windows 10 Enterprise menyimpan, organisasi saat ini dapat menggunakan Intune / System Center untuk mengaktifkan Portal Perusahaan yang juga dapat membuat repositori aplikasi pribadi di luar instans Windows Store. Detail lebih lanjut mengenai solusi ini dapat ditemukan di sini: http://blogs.technet.com/b/canitpro/archive/2014/02/11/step-by-step-enabling-mobile-application-distribution-via-windows-intune.aspx

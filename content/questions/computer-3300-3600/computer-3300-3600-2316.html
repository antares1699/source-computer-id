File Anda di komputer lama dan baru adalah file .tex. Mungkin komputer Anda dikonfigurasi untuk tidak menampilkan ekstensi file (.tex), tetapi terlepas dari itu semua masih file .tex (Saya menggunakan TeXnicCenter hingga 2 minggu yang lalu untuk LaTeX, dan semua file saya adalah .tex). Komputer (atau haruskah saya sebut PC?) Akan memiliki perangkat lunak yang "disukai" untuk membuka file. Saat Anda menginstal TeXnicCenter di komputer lama dan membuka / membuat file .tex pertama Anda, Anda akan ditanya apakah Anda ingin selalu membuka file jenis itu dengan TeXnicCenter atau melakukannya secara otomatis. Sejak saat itu, setiap kali Anda mengklik dua kali pada file .tex, file itu akan terbuka dengan TeXnicCenter, ikon pada file .tex adalah ikon TeXnicCenter, dan semua file jenis itu dikaitkan dengan perangkat lunak itu.

\n\n

Ketika Anda mendapatkan komputer baru dan menginstal MixTex, TexWorks telah diinstal dengannya (Anda dapat memeriksa di situs web MixTex, distribusinya disertakan dengan TexWorks secara default). Jadi file .tex Anda di komputer itu dikaitkan dengan TexWorks dan akan membawa ikon perangkat lunak itu. Itu tidak berarti file Anda berubah, atau jenis file berubah.

\n\n

Anda dapat dengan mudah mengubah program yang terkait dengan file .tex Anda dengan membuka file .tex, klik kanan, pilih "buka dengan" lalu penting bahwa Anda mengklik "pilih program", bahkan jika Anda melihat TeXnicCenter di daftar, pilih "lainnya". Sebuah jendela baru akan terbuka, dan di sana Anda perlu memilih TeXnicCenter dan pastikan Anda mencentang kotak "Selalu gunakan program ini untuk membuka file jenis ini".

\n\n

File .tex Anda sekarang akan memiliki ikon TeXnicCenter :)

\n\n

Ini posting mungkin berguna, meskipun saya akan memastikan bahwa pada langkah 5 Anda jangan pilih TeXnicCenter, dan klik "lainnya" sehingga jendela baru dengan opsi "Selalu gunakan program ini untuk membuka file jenis ini" muncul.

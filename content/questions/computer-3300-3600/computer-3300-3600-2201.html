Sudahkah Anda mencoba menggunakan Hard Drive Virtual? Ini akan memungkinkan Anda melakukan ini, tanpa perangkat lunak atau perangkat keras tambahan.

\n\n

Di Windows 7, menggunakan Manajemen Disk, Anda dapat membuat Hard Disk Virtual (VHD), dan memasangnya sebagai drive di komputer Anda. Anda kemudian dapat menetapkannya sebagai huruf drive, dan mempartisinya, seperti drive biasa.

\n\n

VHD adalah file, yang berada di hard drive Anda. Anda memiliki dua pilihan:

\n\n
    \n
  • Berukuran dinamis - File akan mulai berukuran sangat kecil, dan akan meluas agar sesuai dengan isinya. Anda menentukan batas atas seberapa besar file bisa didapat.
    • \n
  • Ukuran tetap - Anda menentukan ukuran file. Ukurannya akan selalu sebesar itu, tidak peduli seberapa banyak data sebenarnya disimpan di dalamnya.
    • \n
\n\n

Panduan tentang cara melakukan ini ada di sini: http://www.windowsnetworking.com/articles_tutorials/creating-attaching-mounting-vhd-files-windows-7-windows-2008-r2.html

Saya tidak berpikir ini mungkin dengan citra ISO - Anda tidak dapat "menulis" ke citra disk. Bukan untuk itu format itu. Tidak ada yang namanya drive CD yang dapat langsung menulis file ke CD.

\n\n

Kamu bisa , bagaimanapun, gunakan alat seperti MagicISO atau PowerISO untuk menghapus atau menambahkan file ke image disk ISO yang ada. Karena itu, ketika Anda benar-benar menyimpan citra ISO, itu akan menulis ulang seluruh gambar disk karena cara formatnya.

\n\n

Terakhir, jika filesystem yang mendasari file ISO adalah Berbasis UDF , dimungkinkan untuk mengimplementasikan penulisan langsung ke drive virtual dengan membiarkannya bertindak sebagai CD-RW virtual. Karena itu, saya yakin kinerja yang dicapai akan jauh lebih buruk daripada hanya menggunakan alat pengeditan ISO.

\n\n

Jika Anda ingin mensimulasikan flash drive yang sebenarnya, yang Anda cari adalah sistem file virtual. Salah satu cara termudah adalah dengan menggunakan TrueCrypt untuk membuat sistem file virtual (terenkripsi) dalam satu file. Anda kemudian dapat memasang file itu dan menggunakannya seperti hard drive / flash drive lainnya. Lepaskan, dan data Anda disimpan dalam satu file (yang dapat Anda pasang ulang kapan pun Anda mau, di sistem komputer mana pun).

\n\n

Ada program alternatif yang dapat melakukan hal yang sama (dan tanpa enkripsi), tetapi TrueCrypt gratis, open source, lintas platform, dan cukup mudah digunakan.

\n\n
\n\n

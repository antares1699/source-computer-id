Mulai Chrome 59, kotak centang untuk tidak mengaktifkan mode penjelajahan tamu hilang. Banyak sistem kontrol orang tua bergantung pada ini. Akun Chrome yang diawasi tidak dapat lagi dibuat. Jawaban Google adalah Family Link, tetapi ini hanya berfungsi di perangkat seluler, bukan desktop. Jadi, bagaimana Anda menonaktifkan penjelajahan tamu di Chrome hari ini?

Untuk menonaktifkan sepenuhnya di Windows mode tamu di Chrome:

\n
    \n

  • Mulai aplikasi Command Prompt menggunakan Run as Administrator

    \n
    • \n

  • Salin dan tempel perintah berikut ke Command Prompt dan tekan Memasukkan : 

      REG ADD HKLM\\SOFTWARE\\Policies\\Google\\Chrome /v BrowserGuestModeEnabled /t REG_DWORD /d 0\n  REG ADD HKLM\\SOFTWARE\\Policies\\Google\\Chrome /v BrowserAddPersonEnabled /t REG_DWORD /d 0\n
    • \n

  • Mulai ulang Chrome.

    \n
    • \n
\n

Informasi tentang bendera BrowserGuestModeEnabled dan BrowserAddPersonEnabled datang dari Proyek Chromium - Daftar Kebijakan :

\n
\n

BrowserGuestModeEnabled

\n

Aktifkan mode tamu di browser

\n

BrowserAddPersonEnabled

\n

Aktifkan tambahkan orang di pengelola pengguna

\n
\n

Dibawah Mac / Linux ini adalah nama-nama preferensi.
Nilai mereka adalah:\n0 (Windows), false (Linux), <false /> (Mac).

Singkatnya, Anda membuat satu kunci registri, keluar, dan masuk kembali. Berikut adalah konten file .reg yang dapat Anda buat untuk melakukannya:

\n\n
\n

Windows Registry Editor Versi 5.00

\n \n

[HKEY_LOCAL_MACHINE \\ SOFTWARE \\ Policies \\ Google \\ Chrome]\n "BrowserGuestModeEnabled" = dword: 00000000

Tangkapan layar tersebut membuat pertanyaannya jauh lebih jelas.

\n\n

Saya melihat Anda menggunakan %~nx1 untuk menampilkan nama file dalam judul prompt perintah yang muncul.

\n\n

Untuk menggunakan nama file tanpa ekstensi , alih-alih menggunakan %~nx1, gunakan %~n0.

\n\n

yaitu: 

@echo off\ncolor 0d\ncall :sub *.bat\n:sub\nTitle Hacking %~n0 Called Domain\necho Test 1\ntimeout /t 2\necho Test 2\ntimeout /t 2\necho Completed\ntimeout /t 2\nexit\n

Saya mendapat %~n0 dari sini .

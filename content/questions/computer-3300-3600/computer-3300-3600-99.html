Baik...

\n

Player (gratis) - sangat pemula, tetapi berfungsi dengan baik - Jika saya memiliki mesin yang disiapkan dan saya perlu menunjukkannya kepada seseorang yang bukan teknis, saya akan memberikan file-file tersebut dan mereka menginstal dan menjalankannya ... sangat sederhana.

\n

Workstation (berbayar) - Virtualisasi yang sangat stabil dan tangguh, memberikan banyak fitur canggih yang menciptakan lingkungan pengujian dan pengembangan yang brilian.

\n

Server (gratis) - Sangat mirip dengan Workstation tetapi dengan beberapa batasan, ia dirancang agar lebih kuat, lebih mudah dikelola dan meskipun tidak harus demikian, ia dirancang untuk berjalan di server dan dibiarkan begitu saja.

\n

Fusion - seperti Workstation, tetapi untuk Mac.

\n

Vsphere (Sangat mahal) - Pada dasarnya, ini dipasang pada logam kosong dan memberikan lingkungan virtualisasi kinerja SANGAT tinggi - ini SANGAT bagus!

\n

(tidak terdaftar tetapi ...) ESXi (Gratis) - Vsphere tetapi dengan banyak alat manajemen yang hilang.

\n

Adapun apa yang terjadi jika Anda menggunakan satu dan kemudian pindah ke yang lain - Banyak produk yang benar-benar kompatibel satu sama lain - Anda dapat membuat di satu produk dan menggunakannya di produk lain selama Anda memilih level yang benar saat membuat VM baru:

\n

\n

Untuk Readynas, sejujurnya, kecuali ini adalah lingkungan perusahaan besar, saya akan merekomendasikan hanya menggunakan penyimpanan lokal.

\n

Sebagai alternatif, Anda dapat menggunakan Hyper-V, tetapi ... Tidak satu pun dari produk ini mendukung Mac sebagai tamu (selain VMware Fusion) - TI merusak EULA Apple.

\n

Secara pribadi, Jika ini untuk satu mesin yang Anda miliki, gunakan VMWare Workstation. Jika ini untuk server yang sedang Anda gunakan dan tidak ingin memformat / mengubah, gunakan VMWare Server. JIKA ini untuk server baru, gunakan Esxi.

Periksa beranda VMWare ( http://www.vmware.com ).
Pada dasarnya:
Workstation - Jika Anda membutuhkan virtualisasi desktop normal, di mana Anda dapat memiliki tamu dengan akselerasi 2 dan 3D. Itulah yang dibutuhkan kebanyakan orang. Seperti yang Anda inginkan untuk iPhone dev (namun, Anda harus mencobanya AirPlaySDK .)
Player - Solusi paling dasar untuk desktop, gratis, dan mengetahui hal-hal dasar.
Server - Untuk Server. Pada dasarnya jika Anda memiliki server khusus di suatu tempat, dan ingin menjalankan VM, Anda menggunakan ini. Tidak ada GUI yang disertakan. (Dan itu juga gratis.)
Fusion - Workstation / player untuk MAC OS X.
vSphere - "Komputasi awan pribadi." (Baca halaman info. Jika Anda tidak tahu apa itu komputasi awan dan tidak ingin membangun awan ... Anda tidak memerlukan ini.)

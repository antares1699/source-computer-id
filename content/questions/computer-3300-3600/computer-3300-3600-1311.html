Gunakan utilitas Microsoft robocopy. 

robocopy SOURCE_FOLDER DEST_FOLDER /E /LEV:1 /XF * \n

Pilihan:

\n\n
    \n
  • /E Salin Subfolder, termasuk Subfolder Kosong
    • \n
  • /LEV:1 Hanya salin n LEVel teratas dari pohon sumber.
    • \n
  • /XF * eXclude File yang cocok dengan nama / jalur / wildcard yang diberikan.
    • \n
\n\n

Menggunakan baris perintah, jika Anda hanya ingin menyalin folder kosong, itu membuat folder di lokasi baru katakanlah di suatu tempat di c: ...: 

set newlocation=c:\\...\n@for /f \"tokens=3,4\" %i in ('dir') do @if \"%i\" equ \"<DIR>\" if \"%j\" neq \".\" if \"%j\" neq \"..\" mkdir %newlocation%\\%j\n

Jika Anda ingin melangkah lebih dalam, ulangi perulangan for sekali lagi: 

@for /f \"tokens=3,4\" %i in ('dir') do @if \"%i\" equ \"<DIR>\" if \"%j\" neq \".\" if \"%j\" neq \"..\" mkdir %newlocation%\\%j && for /f \"tokens=3,4\" %u in ('dir %j') do @if \"%u\" equ \"<DIR>\" if \"%v\" neq \".\" if \"%v\" neq \"..\" mkdir %newlocation%\\%j\\%v\n

Perhatikan bahwa jika Anda ingin memasukkan ini ke dalam file batch, ganti semua % dengan %%.

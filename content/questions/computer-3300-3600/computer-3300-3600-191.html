Akan sangat menyenangkan memiliki matplotlib di lingkungan cygwin saya. Saya memiliki 64-bit Windows 8 dengan 64-bit cygwin.

\n\n

Saat saya mengetik: 

pip install matplotlib\n

Saya mendapatkan pesan kesalahan berikut: 

 * The following required packages can not be built:\n * freetype\n

Sesuai apt-cyg saya sudah menginstal paket berikut: 

Searching for installed packages matching freetype: \ncygwin32-freetype2\nfreetype2-debuginfo\nlibfreetype-devel\nlibfreetype-doc\nlibfreetype6\n

Apa lagi yang dibutuhkannya? Bagaimana cara menginstal freetype & matplotlib dengan benar?

Saya memiliki masalah yang sama di Windows 7 64-bit; Saya menyelesaikannya dengan menginstal paket Cygwin pkg-config dan libX11-devel sebelum mengetik easy_install matplotlib==1.1.1 (versi 1.3.1 mengalami kesalahan pada mesin saya, karenanya 1.1.1).

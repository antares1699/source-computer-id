Dalam kebanyakan kasus, ini berarti Anda ingin menyimpan file di lokasi di mana Anda memerlukan izin administrator untuk melakukannya. Ada dua solusi yang mungkin untuk ini:

\n\n
    \n

  1. Simpan file di tempat lain di mana Anda tidak memerlukan izin admin (seperti dokumen sendiri atau semacamnya).

    2. \n

  2. Jalankan program yang Anda gunakan untuk menyimpan file sebagai Administrator (jika Anda memiliki izin untuk melakukannya) dan simpan file di mana pun Anda inginkan. !Peringatan! Jika Anda menyimpan file dengan cara ini, Anda dapat menimpa file lain dengan nama yang sama meskipun sangat penting dan Anda dapat menyimpan file di lokasi di mana pengguna biasa tidak dapat membacanya.

Saya mengalami masalah serupa dengan file hosts, yang saya coba edit menggunakan notepad.
Anda hanya dapat mengedit dan menyimpan file ini sebagai Administrator, jadi coba ini:

\n\n

Pergi ke Start > All Programs > Accessories, klik kanan Notepad dan kemudian klik Run as Administrator (mungkin meminta kata sandi).
Kemudian lakukan File > Open dan arahkan ke file tnsnames.ora.

\n\n

Lakukan perubahan maka itu akan memungkinkan Anda untuk menyimpan ( CTRL + S ).

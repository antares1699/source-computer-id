Ini adalah masalah izin. Ada beberapa solusi untuk itu.

\n\n
    \n
  1. sudo chmod 777 /var/www: Bekerja dengan pasti jika ini adalah masalah perizinan, tetapi untuk digunakan dengan hati-hati. Tidak disarankan, karena akan membahayakan keamanan.
    2. \n
  2. ls -l /var |grep www, lihat grup tersebut, dan tambahkan diri Anda ke grup itu, lalu sudo chmod g+rwx /var/www. Anda juga dapat membuat grup baru dan menambahkan diri Anda sendiri ke grup tersebut, lalu sudo chgrp <newGrp> /var/www dan kemudian sudo chmod g+rwx /var/www
    3. \n
  3. sudo chown <yourUserName> /var/www: Lebih baik dari 777, tapi tidak terlalu elegan.
    4. \n
\n\n

Jika Anda punya pilihan, saya merekomendasikan yang kedua. Ini cara yang paling elegan.

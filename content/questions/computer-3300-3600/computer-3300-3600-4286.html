AVG dan Avira keduanya menawarkan disk penyelamat yang dapat Anda unduh dan bakar ke cd.

\n\n

Versi AVG memerlukan koneksi internet untuk pembaruan virus, sedangkan Avira memasukkannya ke dalam .iso. Namun, keduanya harus berfungsi jika malware lebih tua dari .iso.

\n\n

Kedua CD tersebut berbasis Linux, dan boot ke lingkungan Linux untuk menjalankan pemindaian virus. Tidak perlu menginstal apapun ke PC Anda.

\n\n

Edit: Hanya ingin menambahkan itu Kasperky juga menawarkan disk penyelamat.

Panggilan yang bagus untuk membakar CD Anda. Jika mesin Anda bahkan MEMILIKI port USB, Anda tidak dapat benar-benar menggunakan stik USB karena dukungan tanpa batas telah ditambahkan setelah Windows 98.

\n\n

Minimal, perkakas yang menyadari kebutuhan warisan membutuhkan setidaknya Windows 98. Tidak peduli apa versi Windows Anda: freeware antivirus yang serius bahkan tidak akan membutuhkan Anda untuk menjalankan windows dan menginstalnya. Sebagai gantinya, banyak CD boot Anda boot dari CD dan menjalankannya secara independen dari versi Windows manapun. Sebagai keuntungan yang bagus, alat tersebut tidak dibutakan oleh rootkit potensial yang disengaja di dalam sistem Anda yang terinfeksi.

\n\n

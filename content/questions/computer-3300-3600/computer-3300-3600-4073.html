Saya mem-boot komputer dari stik USB menggunakan UNetBootin dan Linux Kecil Sial . Sekarang saya ingin menyertakan beberapa file pada stick sehingga akan tersedia untuk saya setelah komputer di-boot di Linux. Saya mencoba menyalinnya ke direktori di tingkat atas struktur direktori tongkat, tetapi tidak tahu cara mengaksesnya dari dalam Damn Small Linux.

\n\n

Bagaimana cara menyalin file ke stik yang dapat di-boot sehingga saya dapat mengaksesnya?

\n\n

Catatan: Saya terbuka untuk menggunakan alat lain dari membuat USB yang dapat di-boot, selama mereka membantu saya mengakses file saya yang juga ada di USB.

Untuk menyimpan file pada stik USB yang dapat di-boot dengan OS, Anda perlu mengatur penyimpanan persisten saat Anda pertama kali membuatnya, maka setiap kali Anda mem-boot ke file OS USB dan pengaturan dari sesi sebelumnya harus disimpan. Lihat instruksi ini tentang cara mencapai penyimpanan persisten di banyak distro Linux yang berbeda.

\n\n

Anda juga dapat memasang hard drive (di sebagian besar distro Linux) dari komputer yang Anda gunakan untuk menjalankan OS USB dan menyimpannya ke folder di hard drive. Jadi pada saat Anda boot ke OS komputer normal, file harus berada di tempat Anda menyimpannya dan dapat mengambilnya setiap kali Anda boot ke OS USB. Namun opsi ini tidak portabel seperti opsi penyimpanan persisten.

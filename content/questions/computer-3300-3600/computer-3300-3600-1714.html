1) Pada menu Alat, klik Template dan Add-In.
2) Klik Organiser, lalu klik tab Styles.
3) Untuk menyalin item ke atau dari template yang berbeda (template: File atau file yang berisi struktur dan alat untuk membentuk elemen seperti gaya dan tata letak halaman dari file yang sudah jadi. Misalnya, template Word dapat membentuk satu dokumen, dan Templat FrontPage dapat membentuk seluruh situs Web.) Atau file, klik Tutup File untuk menutup dokumen aktif (dokumen aktif: Dokumen tempat Anda bekerja. Teks yang Anda ketik atau gambar yang Anda sisipkan di Microsoft Word muncul di dokumen aktif. Bilah judul dokumen aktif disorot.) Dan templat yang dilampirkan atau untuk menutup templat Normal (Templat normal: Templat global yang dapat Anda gunakan untuk semua jenis dokumen. Anda dapat memodifikasi templat ini untuk mengubah format dokumen default atau kandungan.). Kemudian klik Open File, dan buka template atau file yang Anda inginkan.
4) Klik item yang ingin Anda salin di salah satu daftar, lalu klik Salin.

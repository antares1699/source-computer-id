Saya bekerja untuk perusahaan Jepang, dan saya sering berkolaborasi dengan rekan kerja saya dalam dokumen teknis menggunakan Microsoft Word. Dokumen-dokumen ini terkadang berisi jalur file, yang memiliki garis miring terbalik di dalamnya. Dalam bahasa Jepang, garis miring terbalik ditampilkan sebagai tanda yen.

\n\n

Masalahnya adalah terkadang saya membuat dokumen untuk audiens Jepang, yang berarti bahwa garis miring terbalik harus ditampilkan sebagai tanda yen. Di lain waktu saya membuat atau mengedit dokumen untuk audiens berbahasa Inggris, yang berarti bahwa garis miring terbalik harus ditampilkan sebagai garis miring terbalik.

\n\n

Ketika rekan kerja Jepang saya membuat dokumen, itu menggunakan tanda yen, bahkan saat mengeditnya di PC saya, dan ketika saya membuat dokumen, itu menunjukkan garis miring terbalik.

\n\n

Apakah ada pengaturan yang memungkinkan saya untuk mengubah tanda yen rekan kerja Jepang saya menjadi garis miring terbalik, saat menargetkan audiens Inggris?

Kotak centang "Ubah karakter garis miring terbalik menjadi tanda Yen". Tampaknya berlaku hanya untuk dokumen individu. Ini ada di Word 2013, mungkin berbeda untuk versi yang lebih lama.

Dalam kasus saya, jendela dokumen telah dibuat jauh lebih lebar dari layar monitor saya sehingga saya tidak dapat melihat sisi kanannya (termasuk di mana bilah gulirnya). Gejala lain yang saya alami adalah bahwa menggunakan panah kanan untuk memindahkan kursor melintasi kolom tidak mengakibatkan layar bergulir ke kiri. Saya hanya melihat satu daripada dua set ikon perkecil / layar penuh / tutup di bagian atas layar saya (tanda hubung, kotak, X) seperti yang seharusnya (satu untuk program Excel dan satu untuk file itu sendiri). Terpikir oleh saya bahwa mereka harus keluar dari layar di sisi kanan halaman dan bahwa saya perlu mengubah ukuran jendela file untuk mengembalikan bilah gulir ke layar di bagian bawah. Jika ini terdengar seperti masalah Anda ...

\n\n
    \n
  1. Seret file Anda dengan bilah nama file di bagian atas file Excel ke kiri sampai Anda melihat tepi kanan file.
    2. \n
  2. Ambil tepi kanan file dan seret ke kiri (dan pindahkan file kembali ke kanan) hingga cukup kecil sehingga batas kiri dan kanan terlihat di monitor Anda.
      \n
    • Scroll bar Anda sekarang akan terlihat di kanan bawah.
      • \n
    3. \n
  3. Klik layar penuh (ikon kotak) untuk kembali ke tampilan itu jika itu yang Anda inginkan.
    4. \n

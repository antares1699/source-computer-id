Saya mengembangkan aplikasi web di Mac dan mengunggah ke server yang dihosting Windows.
Saya dulu menggunakan fitur perbandingan direktori Filezilla untuk melihat file mana dalam direktori yang perlu diunggah ke server. Tetapi sekarang, file tertentu selalu ditampilkan dengan warna merah (berbeda).
Contoh terburuk: jika saya mengupload file baru, file tersebut langsung ditandai sebagai berbeda dari file itu sendiri.

\n\n

Saya berasumsi ini ada hubungannya dengan berbagai cara Windos dan Mac OS menghitung ukuran file - tetapi saya hanya mengalami ini dengan jenis file tertentu: html dan teks.\nFile PHP (meskipun pada dasarnya adalah file teks), biner, gambar, dan PDF semuanya melaporkan ukuran yang benar.

\n\n

Apa yang ada dalam file html yang ukurannya salah dilaporkan? Haruskah saya beralih menggunakan waktu modifikasi sebagai kriteria perbandingan saya?

Bisa saya gunakan repo untuk menyinkronkan menurut tanggal? (mungkin dalam git )

\n\n

Misalnya, saya ingin mengunduh sumber mulai 1 Maret jam 3 sore. Saya perlu cara untuk mengambil basis tertentu dari CyanogenMod9 sumber. Sebaiknya gunakan tag atau commit ID, tetapi sumbernya diatur dalam beberapa sub-repositori yang tidak memiliki tag umum. Saya pikir saya bisa checkout berdasarkan tanggal. Saran?

","source":"https://superuser.com/questions/396703/how-to-repo-sync-by-date","author":"tony19","replies":[],"answers":[{"answer":"

Sepertinya saya harus menggunakan repo forall. Ini memerlukan sinkronisasi semuanya terlebih dahulu (tidak dapat melakukan pembayaran kode lama hingga repositori diunduh sepenuhnya). 

repo forall -c 'git checkout `git rev-list --all -n1 --before=\"2012-03-01 15:00\"`'\n

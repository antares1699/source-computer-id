Buat efek kaleidoskop dengan ffmpeg

\n\n

\n\n

Contoh perintah 

ffmpeg -i video.mkv -loop 1 -i mask.png -filter_complex \\\n\"[1:v]alphaextract,split[a1][a2]; \\\n [0:v][a1]alphamerge,transpose=1[e]; \\\n [0:v][a2]alphamerge,transpose=2[w]; \\\n [0:v]vflip,hflip[s]; \\\n [0:v]pad=ih*2:ih*2:x=(ow-iw)/2[n]; \\\n [n][s]overlay=W/2-w/2:W/2[bg]; \\\n [bg][e]overlay=W/2:H/2-h/2[bg2]; \\\n [bg2][w]overlay=0:H/2-h/2\" \\\n-codec:a copy output.mkv\n

Filter FFmpeg digunakan

\n\n
    \n
  • alphaextract dan alphamerge untuk komponen alfa
    • \n
  • split untuk membuat banyak salinan dari komponen alfa
    • \n
  • transpose / vflip / hflip atau kombinasi filter serupa apa pun yang Anda sukai
    • \n
  • pad untuk membuat latar belakang "kanvas"
    • \n
  • overlay untuk menempatkan setiap video di kanvas
    • \n
\n\n

Catatan

\n\n
    \n

  • Gunakan ffmpeg terbaru unduh versi Linux ffmpeg atau ikuti langkah demi langkah panduan untuk mengkompilasi ffmpeg .

    • \n

  • Anda perlu membuat gambar yang berisi topeng alfa. Ukuran frame harus sama dengan input video Anda, jadi jika video.mkv adalah 1920x1080, maka mask.png juga harus 1920x1080. Kamu bisa unduh topeng alfa dari contoh ini.

    • \n

  • split digunakan karena saya tidak berhasil menggunakan kembali label tautan yang sama untuk keluaran alphaextract, dan saya belum pernah menggunakan filter alpha* sebelumnya, jadi contoh mungkin dapat dipersingkat.

    • \n

  • Warna latar belakang hitam filter pad terlihat di sudut. Filter crop dapat digunakan untuk menghapusnya jika Anda mau.

    • \n

  • Audio-nya aliran disalin bukannya dienkode ulang.

    • \n

  • Lihat Panduan Pengkodean FFmpeg dan x264 untuk informasi tambahan tentang pengendalian kualitas keluaran.

    • \n

Kami tidak dapat menonaktifkannya menggunakan BIOS, karena Turbo dimatikan serta efek sampingnya (BIOS yang jelek!). Saya mencari alat baris perintah sederhana untuk Ubuntu 12.04 LTS Server yang dapat melakukan ini.

\n\n

Saya tahu di Windows, jika Anda menyetel Opsi Daya ke Kinerja Tinggi, SpeedStep dinonaktifkan, tetapi bagaimana saya dapat melakukan ini di Ubuntu Linux menggunakan aplikasi baris perintah?

Kernel Ubuntu dikirimkan dengan pengatur pengontrol CPU, biasanya diatur ke ondemand mereka akan mengatur kinerja CPU Anda di sistem Ubuntu Anda.

\n\n

Anda dapat mengubah pengaturan kinerja CPU di Ubuntu per CPU logis.

\n\n

Mulailah dengan membaca pengaturan yang tersedia saat ini untuk sistem Anda: 

sudo cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_available_governors\n

ini akan mengembalikan daftar pengaturan yang tersedia yang dapat Anda gunakan untuk mengatur setiap inti CPU Anda, jika bisa, Anda harus mengaturnya ke kinerja maksimal dengan memilih opsi performance. Ini akan membuat inti CPU Anda selalu berjalan pada frekuensi maksimal.

\n\n

Mengetahui opsi apa yang Anda miliki dan jika opsi performance tersedia, Anda kemudian dapat mengatur setiap inti ke mode performance dengan perintah: 

sudo echo performance > /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor\n

ini akan membuat cpu0 (inti pertama) berjalan sepanjang waktu dengan performa maksimal. Lakukan untuk semua inti logis di CPU Anda.

\n\n

Anda kemudian dapat memeriksa apakah opsi berhasil diubah dengan perintah: 

sudo cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor\n

ini akan memeriksa opsi set saat ini untuk cpu0 (inti pertama). Periksa apakah perubahan berhasil untuk semua inti dan jika semuanya diatur dengan benar, Anda siap melakukannya: Intel SpeedStep akan di tetapi semua inti Anda akan berjalan pada kecepatan frekuensi maksimal sepanjang waktu.

\n\n

Anda kemudian dapat memeriksa dan mendapatkan informasi tentang frekuensi aktual menggunakan cpufreq-info dari paket cpufrequtils di gudang semesta, ini akan melaporkan kecepatan aktual per inti dan banyak informasi lainnya termasuk gubernur yang ditetapkan, tetapi Anda perlu menginstalnya terlebih dahulu menggunakan perintah sudo apt-get install cpufrequtils.

\n\n

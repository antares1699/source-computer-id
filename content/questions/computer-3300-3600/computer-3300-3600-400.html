Tahan Shift dan klik dua kali teks tersebut. Ini mengirimkan input mouse langsung ke terminal (xterm, terminal, dll) dan melewati mode mouse tmux.

\n\n

Catatan: Saya menguji ini di xterm dan gnome-terminal. Untuk terminal lain, Anda mungkin perlu menahan tombol pengubah yang berbeda.

Seperti xtmux bisa melakukan itu, tetapi itu tidak akan berlaku di ssh. Setara iTerm dari Shift adalah Alt. Saya belum menemukan solusi tmux murni, tetapi Anda dapat mempertimbangkan salin / tempel mode vim:\n\nset-window-option -g mode-keys vi\nbind-key -t vi-copy 'v' begin-selection\nbind-key -t vi-copy 'y' copy-selection\nbind-key -t vi-copy 'Escape' copy-selection\n\nDengan itu dan set -g mode-mouse on, Anda dapat mengklik dan menahan di awal pilihan Anda, tekan w dan perintah vim lainnya sampai Anda memilih yang Anda inginkan, dan kemudian lepaskan.

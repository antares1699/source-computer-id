Saat Anda mengetik rumus ke dalam Excel 2007, ini akan memunculkan petunjuk untuk fungsi. Misalnya, ketik =L ke dalam file f x lapangan dan banyak fungsi akan muncul di menu drop-down (Besar, LCM, Kiri, dll).

\n\n

Dengan menggunakan tombol atas dan bawah, Anda dapat memilih fungsi. Tetapi saya tidak tahu bagaimana memilih fungsi itu tanpa menggunakan mouse! Tombol enter, spasi, tombol sisipkan semua tidak berfungsi. Namun, mengklik dua kali fungsi tersebut akan menempatkannya di bilah rumus. Apakah ada kunci untuk melakukan itu?

","source":"https://superuser.com/questions/185092/how-to-use-fx-autocomplete-in-excel-w-o-mouse","author":"Matt","replies":[],"answers":[{"answer":"

Pilih fungsi yang Anda inginkan menggunakan tombol panah seperti yang Anda lakukan sekarang, lalu tekan tombol TAB untuk melengkapi otomatis ke fungsi itu.

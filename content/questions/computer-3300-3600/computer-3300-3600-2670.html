Saya baru-baru ini membeli keyboard biasa untuk MacBook saya untuk menyelesaikan set DJ stasioner saya. Tetapi ketika saya melakukannya, saya tidak menemukan cara untuk memetakan ulang kunci F [1-12] ke kunci media.

\n\n

Pada dasarnya yang saya cari adalah kemampuan untuk membuatnya F1 , misalnya, jeda iTunes. Tentu saja, saya juga ingin berpaling F -keys ke tombol media untuk mundur dan melewati iTunes saya, karena saya merasa ini sangat nyaman.

\n\n

Adakah alat atau cara yang memungkinkan saya melakukan itu? Saya di 10.9.

Unduh skrip berikut: http://gskinner.com/blog/assets/MediaControlScripts.zip

\n\n

Kemudian ikuti saja petunjuk ini:

\n\n
    \n

  1. Klik dua kali pada setiap file skrip untuk perintah yang ingin Anda instal, dan klik tombol "Instal" di dialog

    2. \n

  2. Buka Preferensi Sistem> Keyboard> Pintasan Keyboard

    3. \n

  3. Pilih Layanan di panel kiri, lalu gulir ke bawah ke skrip yang ingin Anda tetapkan pintasannya

    4. \n

  4. Klik dua kali di sisi kanan baris untuk mengedit pintasan

    5. \n

  5. Tekan kombinasi tombol yang ingin Anda gunakan, dan pastikan kotak centang di sebelah kiri dicentang (ini suka tidak dicentang saat Anda mengedit kombo tombol)

    6. \n
\n\n

\n\n

Install Karabiner dan simpan file seperti ini sebagai ~/Library/Application Support/Karabiner/private.xml: 

<?xml version=\"1.0\"?>\n<root>\n  <item>\n    <name>ctrl+f-keys to media keys</name>\n    <identifier>mediakeys</identifier>\n    <autogen>__KeyToKey__ KeyCode::F1, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_BRIGHTNESS_DOWN</autogen>\n    <autogen>__KeyToKey__ KeyCode::F2, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_BRIGHTNESS_UP</autogen>\n    <autogen>__KeyToKey__ KeyCode::F8, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_PREV</autogen>\n    <autogen>__KeyToKey__ KeyCode::F9, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_PLAY</autogen>\n    <autogen>__KeyToKey__ KeyCode::F10, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_NEXT</autogen>\n    <autogen>__KeyToKey__ KeyCode::F11, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_VOLUME_DOWN, ModifierFlag::OPTION_L | ModifierFlag::SHIFT_L</autogen>\n    <autogen>__KeyToKey__ KeyCode::F12, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_VOLUME_UP, ModifierFlag::OPTION_L | ModifierFlag::SHIFT_L</autogen>\n  </item>\n</root>\n

Kemudian buka aplikasi Karabiner, tekan tombol ReloadXML, dan aktifkan pengaturan.

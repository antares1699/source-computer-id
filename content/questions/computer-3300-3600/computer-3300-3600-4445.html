Apakah ada cara untuk memformatnya dengan benar\n untuk diambil Server Ubuntu 9.10\n keuntungan dari format lanjutan? Seperti itu\n seperti ukuran sektor?

\n
\n\n

Masalah potensial muncul bukan dari memformat drive tetapi dari tempat partisi itu berada di drive. Drive sektor 4K perlu disejajarkan pada alamat blok logis yang merupakan kelipatan 8.

\n\n

Seharusnya alat Linux telah diperbarui untuk melakukan hal yang benar dalam hal drive sektor 4K.

\n\n

Jika Anda menggunakan versi terbaru dari editor partisi Linux yang populer untuk membuat satu atau lebih partisi baru pada drive, Anda akan baik-baik saja.

\n\n

Satu peringatan: Drive WD dapat diubah sebagai kludge untuk bekerja "dengan benar" dengan partisi yang dibuat oleh Windows XP. Karena Anda menggunakan Linux, pastikan Anda melakukannya tidak memiliki jumper yang disetel untuk Windows XP. (Jika Anda memiliki set jumper XP maka Anda akan mendapatkan kinerja yang buruk dari drive).

\n\n
\n\n

Memperbarui:

\n\n

Oke, setelah googling sedikit lebih jauh, saya bingung. Di satu sisi, saya telah membaca artikel ikhtisar yang mengklaim Linux tidak memiliki masalah sama sekali dengan drive sektor 4K. Namun di sisi lain, ada artikel seperti ini yang justru mengklaim sebaliknya.
\n Linux Tidak Sepenuhnya Disiapkan untuk Hard Drive Sektor 4096-Byte
\n Format Advanced Hard Drive Linux WD EARS

\n\n

Tautan kedua di atas mungkin lebih berguna karena tampaknya menawarkan saran tentang cara mempartisi drive sektor 4K dengan benar.

\n\n

Yang saya sarankan adalah Anda terlebih dahulu memeriksa penyelarasan partisi Anda dengan menggunakan fdisk -lu untuk menampilkan LBA awal mereka. Jika partisi Anda sudah sejajar pada LBA yang merupakan kelipatan 8 maka Anda akan baik-baik saja. (Anda juga dapat mengatur waktu transfer file besar hanya sebagai pemeriksaan kewarasan ekstra.)

\n\n

Jika Anda memiliki partisi yang tidak selaras maka Anda mungkin ingin menghapus dan membuat ulang menggunakan metode di artikel kedua untuk memastikan partisi tersebut disejajarkan dengan benar.

\n\n

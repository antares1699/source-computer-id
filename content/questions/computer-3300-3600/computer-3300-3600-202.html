Konten pada OS X 10.8.3 dari Situs web Open Source Apple : 

# System-wide .profile for sh(1)\n\nif [ -x /usr/libexec/path_helper ]; then\n        eval `/usr/libexec/path_helper -s`\nfi\n\nif [ \"${BASH-no}\" != \"no\" ]; then\n        [ -r /etc/bashrc ] && . /etc/bashrc\nfi\n
\n\n

Untuk solusi yang lebih umum (atau jika Anda tidak menemukannya di opensource.apple.com), Anda dapat menggunakan Pasifis : Ini memungkinkan Anda untuk membuka paket penginstal OS pada DVD OS X 10.8 atau citra disk (InstallESD.dmg, bagian dari unduhan App Store) dan menginstal ulang satu file dari paket ini.

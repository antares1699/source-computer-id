Alamat IP dari berbagai antarmuka jaringan disimpan di bawah:

\n\n

HKEY_LOCAL_MACHINE\\SYSTEM\\CurrentControlSet\\Services\\Tcpip\\Parameters\\Interfaces

\n\n

tetapi hanya mengubah nilai di sana tidak berarti alamat IP aktif Anda berubah.

\n\n

Gunakan perintah Windows yang ada untuk mengubahnya, seperti:

\n\n

Set-NetIPAddress

\n\n

di PowerShell

netsh interface ipv4 add dnsserver \"Ethernet\" 8.8.8.8 index=1\n

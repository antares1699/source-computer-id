Saya mencoba mengonversi file HTML ke penurunan harga menggunakan Pandoc, tetapi saya kesulitan menemukan opsi yang benar.

\n\n

Saya ingin mengubah seluruh file HTML menjadi penurunan harga sesederhana mungkin. Artinya, tidak ada tag <div> dan tidak ada gaya CSS dalam tanda kurung kurawal. Namun, ketika saya memasukkan HTML berikut ke pandoc dengan menelepon pandoc -f html -t markdown reduced.html -o res.md: 

<div class=\"section-content\">\n<div class=\"section-inner sectionLayout--insetColumn\">\n    <h3 name=\"2ee9\" id=\"2ee9\" class=\"graf graf--h3 graf--leading graf--title\">Post Title</h3>\n    <p name=\"021f\" id=\"021f\" class=\"graf graf--p graf-after--h3\">Preamble <a href=\"https://cogsci.stackexchange.com/users/4397/seanny123?tab=questions\" data-href=\"https://cogsci.stackexchange.com/users/4397/seanny123?tab=questions\" class=\"markup--anchor markup--p-anchor\" rel=\"noopener\" target=\"_blank\">link</a> and conclusion.</p>\n</div>\n

\n\n

Saya mendapatkan output Markdown berikut dengan semua hal yang ingin saya abaikan: 

<div class=\"section-content\">\n\n<div class=\"section-inner sectionLayout--insetColumn\">\n\n### Post Title {#2ee9 .graf .graf--h3 .graf--leading .graf--title name=\"2ee9\"}\n\nPreamble\n[link](https://cogsci.stackexchange.com/users/4397/seanny123?tab=questions){.markup--anchor\n.markup--p-anchor} and conclusion.\n\n</div>\n\n</div>\n

Saya mencoba membaca dokumen Pandoc Markdown, tetapi saya tidak dapat menemukan indikasi parameter apa yang harus saya gunakan.

","source":"https://superuser.com/questions/1261472/make-pandoc-ignore-css-style-class","author":"Seanny123","replies":[],"answers":[{"answer":"

Jika Anda menggunakan pandoc -f html -t markdown_github-raw_html reduced.html -o res.md, ini menyelesaikan masalah.

\n\n
    \n

  • markdown_github mengurangi set output untuk mengabaikan CSS

    • \n

  • -raw_html menghapus ekstensi raw_html, yang memperbaiki tautan

    • \n

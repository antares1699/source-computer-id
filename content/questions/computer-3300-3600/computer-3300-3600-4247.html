Saya pengguna Linux Ubuntu (Lucid Lynx) yang menjalankan Apache. Saya memiliki koleksi file zip di folder di direktori home saya (~ / zip_files) yang ingin saya tautkan melalui apache, seperti ketika seseorang yang mengunjungi situs web saya yang saya gunakan Apache untuk meng-host mengklik a tautan ke salah satu file zip, dia dapat mengunduhnya melalui web. Bagaimana saya bisa memberi Apache akses ke file dan mengatur izin? Terima kasih, saya baru mengenal linux!

Ada dua pendekatan yang bisa Anda lakukan:

\n\n
    \n

  1. Ubah DocumentRoot di file conf Apache Anda ke direktori home Anda. Ini akan segera menyajikan dokumen Anda melalui HTTP. Anda perlu memuat ulang Apache untuk melihat perubahannya.

    2. \n

  2. Pendekatan kedua adalah dengan menggunakan a symlink . Symlinks pada dasarnya adalah referensi ke file atau folder lain. Di direktori / var / www ketik: 

    ln -s /var/www /home/username\n

    Ini adalah deskripsi symlink yang bagus . Anda masih perlu mengatur opsi FollowSymLinks di conf Apache dan memuat ulang Apache.

    3. \n
\n\n

Untuk kedua pendekatan, pastikan bahwa izin folder setidaknya 755 (gunakan chmod 755 / home / nama pengguna) untuk memastikan izin sudah benar.

Apache memiliki fitur yang memungkinkan per direktori web pengguna yang memungkinkan orang mengakses file Anda menggunakan http://www.example.com/~mark/.

\n\n

Anda bisa menggabungkan ini dengan ide symlink 

cd \nln -s zip public_html\n

atau 

cd\nmkdir public_html\ncd public html \nln -s ../zip zip\n

Saya rasa Anda perlu memberi tahu Apache untuk mengikuti symlink.

\n\n

Anda juga perlu mengatur izin pada direktori dan file agar dapat dibaca oleh pengguna yang menjalankan Apache.

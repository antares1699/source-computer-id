Tidak. Windows 7 tidak mengizinkan Anda melakukan ini. Tapi... Windows MultiPoint Server akan. Ini memungkinkan menghubungkan beberapa monitor, beberapa keyboard, beberapa mouse melalui USB HUB untuk bekerja pada satu mesin oleh banyak orang. Saya telah melihat ini beraksi dan mengaturnya beberapa kali jadi itu sangat bagus. Sebagian besar untuk siswa, ruang kelas dengan satu server besar dan beberapa mouse / keyboard / monitor.

\n\n
\n

Dengan Windows MultiPoint Server, satu komputer mendukung banyak pengguna secara bersamaan, masing-masing bekerja secara independen menggunakan monitor, keyboard, dan mouse mereka sendiri serta dengan pengalaman komputasi Windows yang sudah tidak asing lagi. Sekolah dapat memberi lebih banyak siswa akses ke teknologi terbaru, bahkan dengan anggaran terbatas.

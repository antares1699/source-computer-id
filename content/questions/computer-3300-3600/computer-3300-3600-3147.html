Saya membuka file Microsoft Project yang ada hari ini dan tidak dapat membuka / menutup subtugas menggunakan kursor dan tanda '+/-' di samping tugas induk.

\n\n

Saya bisa meluaskan dan menciutkan menggunakan ikon pita 'View-> Outline'. Namun ketika saya mencoba melakukan ini dengan menggunakan mouse pada tugas tertentu, kursor saya tidak akan melakukan tugas tersebut. Kursor saya adalah 'salib putih' saat saya membutuhkannya sebagai 'panah'.

\n\n

Saya berasumsi bahwa ini adalah sesuatu yang kecil yang saya lewatkan. Setiap bantuan dihargai.

Seperti posting sebelumnya, menutup dan membuka kembali memperbaiki masalah. Satu catatan untuk proyek yang dimaksud, editor sebelumnya telah menyimpannya dengan filter pada nama sumber daya, sehingga ketika saya membukanya, filter itu masih ada. Setelah menghapus filter, +/- untuk memperluas / menciutkan tugas tidak berfungsi. Saya menyimpan proyek tanpa filter, lalu menutup / membuka kembali dan semua bekerja dengan normal.

Saya mencoba untuk meningkatkan kinerja CPU dalam sebuah permainan, untuk melakukan ini saya ingin mencoba memberikannya 2-4 inti CPU yang HANYA menjalankan aplikasi itu, dan tidak ada layanan yang berhubungan dengan jendela. Apakah ada saya di mana saya dapat mengatur afinitas default untuk SEMUA aplikasi, layanan, proses, dll. di komputer jadi intinya secara efektif tidak digunakan sama sekali?

","source":"https://superuser.com/questions/602837/set-default-processor-affinity-for-all-applications","author":"Novicode","replies":[],"answers":[{"answer":"

di PowerShell: 

$instances = Get-Process\nforeach ($i in $instances) { $i.ProcessorAffinity=1 }\n

akan membuat sebagian besar proses berjalan pada inti pertama (inti 0 di pengelola tugas). Ya, ini akan memunculkan cukup banyak kesalahan "Akses ditolak" untuk proses yang afinitasnya tidak dapat diubah. abaikan saja ... untuk sisanya itu berhasil.

\n\n

dan kemudian secara manual atau dengan perintah setel game Anda ke inti yang diinginkan, seperti: 

(Get-Process -name doom).ProcessorAffinity = 14\n

untuk mengatur game 'malapetaka' Anda agar berjalan dengan sisa 3 inti pada 4 inti CPU.

Beberapa metode lain untuk memaksa Windows menjalankan aplikasi hanya pada satu inti dijawab sini . Itu bisa membebaskan tiga inti Anda yang lain untuk game Anda. Dan cara mengatur afinitas untuk yang satu ini secara manual melalui taskmgr atau cmdline menjawab Keltari dengan petunjuk berguna lainnya di mana kemacetan kinerja bisa terjadi.

Ini bisa jadi sebagian dicapai dengan banyak pekerjaan manual. Anda dapat mengatur afinitas prosesor melalui pengelola tugas atau dari baris perintah. Lihat ini artikel tentang cara melakukannya. Sayangnya, metode ini tidak berfungsi pada layanan.

\n\n

Namun, saya tidak percaya ini akan banyak membantu Anda. Saat berada dalam game, OS tidak melakukan banyak CPU dengan bijaksana. Tentu saja, ini mengasumsikan Anda tidak melakukan tugas lain pada saat yang sama, tetapi Anda tidak akan melakukannya, karena Anda mengeluh tentang kinerja. Apakah semua core Anda maksimal 100% saat bermain game? Saya akan meragukannya. Kehilangan kinerja Anda kemungkinan besar tidak terikat pada CPU, tetapi sesuatu yang lain, seperti grafik dan / atau disk. Anda perlu menjalankan pemantau kinerja (seperti perfmon Windows) untuk benar-benar melihat apa yang memperlambat Anda.

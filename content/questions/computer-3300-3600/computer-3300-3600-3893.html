When you extract the library from its zip file, you must preserve its internal directory structure (for example by using the -d option when extracting). If you didn't do that when extracting, then you'd better stop reading this, delete the files you just extracted, and try again!

Klik kanan pada file .zip, pilih 7-zip> Extract files..., dan pada dialog berikut pilih tujuan ekstraksi file, pastikan untuk memilih full path pada kotak dropdown folders structure.

\n\n

Dengan cara ini, Anda yakin bahwa struktur folder dipertahankan, bagaimanapun, saya pikir itu adalah perilaku normal bahkan dengan perintah ekstrak sederhana.

\n\n

P.S. Harap diperhatikan bahwa saya tidak menggunakan 7zip versi bahasa Inggris, jadi perintahnya mungkin sedikit berbeda dari yang saya ketik.

Kemarin, saya membeli game (The Outer Worlds) melalui Microsoft Store. Saya mulai mengunduh dan menginstalnya melalui UI. Pada awal proses ini, saya diberi tahu bahwa saya memerlukan ruang kosong yang relatif besar (sekitar 24 GB). Tidak apa-apa, saya memiliki 32 GB gratis di drive C: saya, jadi saya mengklik OK dan mulai mendownload.

\n\n

Unduhannya sangat besar dan saya tidak dapat menyelesaikannya malam itu. Saya mengklik "jeda" dan mematikan komputer saya, mengharapkan unduhan dilanjutkan ketika saya membuka kembali UI Microsoft Store (dengan cara yang sama, katakanlah, Steam akan melanjutkan unduhan yang terputus). Ini tidak terjadi. Ups!

\n\n

Sebagai gantinya, saya sekarang diminta untuk menginstal The Outer Worlds fresh, tanpa pengakuan bahwa saya sudah mendownloadnya sebagian. Lebih mengkhawatirkan, jumlah ruang kosong yang terdeteksi oleh Windows di drive C: saya sekarang jauh lebih rendah .

\n\n

Ini juga tidak tersedia sebagai opsi untuk mencopot pemasangan di layar "Tambah atau Hapus Program" Windows.

\n

Iya; Aplikasi yang diinstal dari Windows Store ditempatkan di WindowsApps direktori yang ada di dalamnya File program .

\n

Mengubah izin pada direktori WindowsApps akan merusak aplikasi UWP apa pun yang telah Anda instal. Mengubah izin akan menyebabkan lebih banyak masalah daripada menyelesaikannya.

\n
\n

Bagaimana saya bisa mengosongkan ruang ini yang dicadangkan oleh unduhan yang gagal dari Microsoft Store?

\n
\n

Tindakan berikut seharusnya cukup untuk menyelesaikan masalah Anda.

\n
\n
    \n

  1. Tekan tombol logo Windows + X dan klik Jalankan.

    \n
    2. \n

  2. Ketik WSReset.exe di ruang kosong dan tekan ENTER

    \n
    3. \n

Menemukan solusinya. Langkah-langkah yang saya gunakan untuk memperbaikinya:

\n\n
    \n
  1. Buka C: \\ Program Files \\ WindowsApps folder, tempat Windows Store meletakkan unduhannya. Folder WindowsApps tidak diizinkan untuk pengguna biasa, jadi ini harus dilakukan menggunakan beberapa utilitas yang dijalankan sebagai administrator (baik jendela prompt perintah, atau beberapa pengelola file lain seperti 7zip; h / t ManSamVampire untuk saran 7zip). JANGAN tetapkan kepemilikan folder WindowsApps ke akun pengguna Anda sendiri untuk mendapatkan akses ke sana, karena hal itu akan menyebabkan masalah besar dan tidak dapat dengan mudah dibatalkan (h / t Jawaban Ramhound untuk peringatan ini).
    2. \n
  2. Urutkan subfolder di dalam WindowsApps berdasarkan tanggal terakhir kali diubah; ini akan memudahkan Anda menemukan map yang baru saja memasukkan barang baru ke dalamnya.
    3. \n
  3. Dalam kasus saya, subfolder yang melanggar dipanggil MSIXVC , yang mana tampaknya jenis penginstal yang digunakan oleh Windows Store. Di dalam, saya memiliki file 37GB anomali yang dibuat tadi malam tanpa ekstensi file, yang disebut "8BBF2763-A1C6-4F1B-96CA-B79DD5FDA4EE". Mengingat bahwa file ini sangat besar, dibuat pada waktu yang tepat, dan dalam subfolder yang menyarankan itu adalah penginstal, saya menyimpulkan bahwa ini adalah droid yang saya cari.
    4. \n
  4. Dari dalam utilitas run-as-administrator Anda, hapus file yang melanggar. Saya menghapus file yang sangat aneh dan namanya membingungkan dari folder MSIXVC, dan lihatlah saya memiliki semua ruang disk saya kembali!
    5. \n
\n\n

Sekarang mengunduh ulang dan memasang Outer Worlds. Kali ini saya akan tahu lebih baik daripada mematikan komputer saya di tengah jalan.

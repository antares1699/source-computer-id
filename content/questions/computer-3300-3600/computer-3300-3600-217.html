Protokol http memungkinkan klien untuk mengidentifikasi diri mereka sendiri menggunakan file\n Bidang tajuk Agen-Pengguna. Ini memungkinkan untuk membedakan perangkat lunak www,\n biasanya untuk keperluan statistik atau untuk penelusuran protokol\n pelanggaran. Wget biasanya diidentifikasikan sebagai ' Wget / versi ', Versi sedang\n nomor versi Wget saat ini.

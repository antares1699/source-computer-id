Saya pasti akan menggunakan kompresi bawaan NTFS (yang Anda lihat di Windows 7) melalui utilitas kompresor tingkat file seperti 7zip. Melakukannya akan jauh lebih mudah untuk dikerjakan karena kompresi / dekompresi terjadi dengan mulus saat baca / tulis, dan Anda tidak perlu khawatir tentang arsip yang rusak atau menggunakan utilitas pihak ketiga untuk menelusuri konten.

\n\n

Sejauh bagaimana kompresi NTFS menumpuk terhadap metode kompresi sistem file lain, saya tidak yakin. Jika Anda ingin tetap menggunakan vanilla Windows 7 tanpa menginstal filesystem lain, saya akan mengatakan NTFS akan menjadi taruhan Anda yang paling mudah.

Jika file yang Anda cadangkan berukuran kecil (<50MB) dan banyak, maka memadatkannya menjadi arsip dengan kompresi 'tercepat' akan meningkatkan kecepatan disk. Saya telah melakukan ini sebelumnya untuk membaca dan menulis proyek media besar, pembandingan menunjukkan peningkatan kecepatan 20-100%, tetapi hanya untuk kompresi yang sangat ringan atau nol.

\n\n

Apa yang akan saya lakukan:

\n\n
    \n
  1. Kompres drive kecuali Anda perlu menggunakannya lintas platform (jangan pernah mengandalkan kompatibilitas atau standar untuk backup).
    2. \n
  2. Kompres sebagai arsip padat dengan opsi 'tercepat' dipilih (saya menggunakan winrar, jadi opsi Anda mungkin berbeda. Untuk peningkatan kecepatan, Anda memerlukan kompresi paling ringan yang tersedia)
    3. \n
  3. Masukkan versi perangkat lunak pengarsipan yang tidak diarsipkan ke dalam disk. Pastikan itu versi yang sama yang Anda gunakan. Anda akan senang melakukan ini jika Anda membutuhkan cadangannya.
    4. \n

Tingkat kompresi akan bergantung pada jenis file yang ingin Anda cadangkan, tetapi hasilnya akan selalu lebih rendah daripada utilitas kompresi eksternal (dari yang lebih rendah ke jauh, jauh lebih rendah).

\n\n

Buat perbandingan pada beberapa file tipikal Anda. Klik kanan pada folder atau file apapun dan centang "Compressed". Anda kemudian dapat mengklik kanan pada file, pergi ke properti dan itu akan memberi tahu Anda kompresi yang dicapai dalam%. Bandingkan hasilnya dengan 7zip dan buat keputusan Anda.

\n\n

Jika Anda menggunakan rute 7zip, saya sarankan Anda mencadangkan folder individual sebagai file arsip, dan menghindari zip seluruh drive dalam satu file besar.

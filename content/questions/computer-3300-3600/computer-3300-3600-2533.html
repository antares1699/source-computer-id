Masalah Anda tidak ada hubungannya dengan mengonversi video dan segala sesuatu yang berkaitan dengan lokasi biner ffmpeg.

\n\n

Saya berasumsi Anda mengunduh ffmpeg dari evermeet.cx . Tidak apa-apa, dan saya sering memberikan tautan itu ke pengguna macOS lain untuk mendapatkan versi terbaru.

\n\n

Mengunduh tautan .dmg mungkin lebih mudah daripada .7z, karena macOS dapat mengekstrak .dmg tanpa alat pihak ketiga.

\n\n

Setelah diekstrak, Anda dapat memindahkan file ffmpeg ke /usr/local/bin (ini akan meminta kata sandi untuk melakukannya). Anda kemudian dapat menjalankan perintah ffmpeg dari direktori mana pun.

\n\n

Sebagai alternatif, Anda dapat memindahkannya ke mana pun Anda mau, tetapi Anda harus memberikan jalur lengkap ke lokasi tersebut, seperti ~/Downloads/ffmpeg -i ... atau pertama-tama navigasikan ke direktori tempatnya lalu jalankan ./ffmpeg -i ... (perhatikan awalan ./).

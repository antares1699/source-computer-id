Saya mencari cara untuk mengubah string menjadi bersih url.

\n\n

Sebagai contoh : 

 wget http://myurl.com/toto/foo bar.jpg\n

Ini akan mengunduh http://myurl.com/toto/foo dan http://bar.jpg.

\n\n

Saya ingin mengunduh http://myurl.com/toto%20bar.jpg.

\n\n

Saya mencoba beberapa bendera seperti --restrict-file-names=ascii tetapi tidak berhasil.

\n\n

Saya tidak ingin cara untuk menyandikan bukan semacam ganti satu per satu.

\n\n

Ada ide ?

Berisi URL dalam tanda kutip: 

wget \"http://myurl.com/toto/foo bar.jpg\"\n

Ini adalah cara umum untuk memuat string dengan spasi sebagai argumen tunggal.

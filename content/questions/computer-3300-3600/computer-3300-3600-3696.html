Saya memiliki dua salinan vim yang diinstal.

\n\n

Satu diinstal dengan macvim dan perlu dikonfigurasi dengan -X11 (untuk bekerja dengan Cocoa), sementara yang lain dikonfigurasi ke + X11 (digunakan di terminal, untuk bekerja dengan plugin yang saya gunakan)

\n\n

Saya ingin semua yanks pergi ke papan klip sistem. Jika + X11 diaktifkan, saya bisa melakukannya di .vimrc dengan set clipboard=unnamedplus. Saat -X11, saya bisa melakukannya dengan set clipboard=unnamed.

\n\n

Bagaimana saya bisa melakukan ini dalam .vimrc secara kondisional?

\n\n

Sesuatu seperti 

if &x11\n set clipboard=unnamedplus\nelse\n set clipboard=unnamed\nendif\n

Tentu saja ini tidak berhasil karena & x11 bukan variabel nyata.

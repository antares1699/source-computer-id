Saya memiliki dua file masukan, gambar diam yang akan berulang sebagai bingkai dan file audio.

\n\n

Bagaimana saya bisa menggunakan file audio tanpa harus menyandikan ulang untuk menghasilkan aliran audio dengan format yang sama seperti file audio?

","source":"https://superuser.com/questions/650517/how-can-i-use-an-audio-file-without-having-to-re-encode-it-to-output-an-audio-st","author":"Karolinger","replies":[],"answers":[{"answer":"

Yang termudah adalah: 

ffmpeg -loop 1 -i image.jpg -i audio.m4a -c:v libx264 -c:a copy -shortest out.mp4\n

Ini akan menyandikan gambar dengan x264 ke aliran video H.264 dan membiarkan aliran audio tidak tersentuh (dengan menggunakan encoder copy). Opsi -shortest membuat ffmpeg berhenti ketika input terpendek berakhir, yaitu file audio.

\n\n

Perhatikan bahwa aliran audio menentukan format keluaran yang tersedia untuk Anda. Jika Anda memiliki file audio PCM (mis. Sejenis .wav), Anda tidak dapat menggunakan wadah MP4. Jika Anda memiliki audio MPEG-1 Layer III (MP3) atau audio AAC (misalnya .m4a), wadah MP4 akan berfungsi. Periksalah Perbandingan format wadah untuk mendapatkan gambaran tentang apa yang mungkin.

\n\n

Lihat juga: Buat tayangan slide video dari gambar dan x264 Panduan Pengkodean untuk menyesuaikan kualitas keluaran.

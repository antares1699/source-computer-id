Saya mencoba menambahkan beberapa pengguna LDAP kami ke grup yang ditentukan secara lokal di server RHEL kami, namun saya mendapatkan pesan kesalahan yang menyatakan bahwa pengguna LDAP tidak ditemukan di / etc / passwd.

\n\n

Apa cara terbaik untuk mengizinkan pengguna LDAP ditambahkan ke grup lokal?

\n\n

Menurut saya ini harus dilakukan secara manual. Saya dapat mengedit: /etc/group dan menambahkan grup LDAP ke daftar. Apakah itu ideal? 

[server]# id apache\nuid=409(apache) gid=409(apache) groups=409(apache) context=user_u:system_r:unconfined_t:s0\n[server]# id john.doe\nuid=11389(john.doe) gid=6097(ABC_Corporate_US) groups=6097(ABC_Corporate_US) context=user_u:system_r:unconfined_t:s0\n[server]# /usr/sbin/usermod -a -G apache john.doe\nusermod: john.doe not found in /etc/passwd\n

OS: RHEL (Red Hat Enterprise Linux Server rilis 5.3 (Tikanga))\nCatatan: Memperbarui OS di mesin ini bukanlah suatu pilihan.

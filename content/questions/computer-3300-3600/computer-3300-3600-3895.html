Saya perlu membuka browser dengan url tertentu menggunakan ssh melalui mesin jarak jauh, saya telah mengaktifkan koneksi tanpa kata sandi, saya bisa mendapatkan perintah o / p yang berjalan di mesin jarak jauh menggunakan ssh, tetapi bagaimana cara membuka browser menggunakan ssh? Mencoba dengan opsi "-X" yang akan membuka browser mesin jarak jauh di mesin saya (mengingat saya menjalankan ssh dari mesin saya).

\n\n

Mencoba
1). ssh client@IP_ADDRESS -X \"firefox http://google.com - ini membuka browser di komputer saya.
2). ssh client@IP_ADDRESS \"firefox http://google.com perintah ini menunjukkan kesalahan mengatakan Error: no display specified.

Pada dasarnya, kebutuhan saya adalah, melalui mesin saya, saya harus meluncurkan browser dengan url tertentu di mesin jarak jauh (yaitu suse linux).

Di sisi Server (atau di mesin saya) saya memiliki skrip yang terlihat seperti ini
\n
#!/bin/bash
ssh client@IP_ADDRESS \"nohup sh openBrowser.sh $1\" &
PID=$$;
echo \"PID IS |$PID|\\n\";
sleep 3 && kill -9 $PID;

Di sisi klien (di mana saya perlu membuka browser dari jarak jauh, saya memiliki skrip (openBrowser.sh) yang terlihat seperti ini

#!/usr/bin/sh
#read "sebagai kutipan latar
browserPID=\"ps aux | grep firefox | grep -vc grep\";
if [[ \"$browserPID\" > 0 ]];
then
killall -9 firefox
fi
nohup firefox --display=:0 $1 &

Juga tunjuk untuk memastikan bahwa, saya telah membuat koneksi tanpa kata sandi menggunakan ssh-keygen.

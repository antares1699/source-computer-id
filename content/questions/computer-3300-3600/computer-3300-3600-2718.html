Di sini saya mengasumsikan bahwa file asli Anda ada dalam container mp4, tetapi metode ini juga harus berfungsi untuk container lain.

\n\n

Metode satu langkah - (sedikit lebih terlibat)\nLihat juga jawaban yang diberikan oleh ptQa di direferensikan benang. Tapi di sini sama dengan audio yang ditambahkan: 

ffmpeg -i inputfile.mp4 \\\n       -filter_complex \"[0:v]trim=start=10:end=16,setpts=PTS-STARTPTS[a];\n                        [0:v]trim=start=20:end=26,setpts=PTS-STARTPTS[b];\n                        [0:a]atrim=start=10:end=16,asetpts=PTS-STARTPTS[c];\n                        [0:a]atrim=start=20:end=26,asetpts=PTS-STARTPTS[d];\n                        [a][c][b][d]concat=n=2:v=1:a=1[e][f]\" \\\n       -map '[e]' -map '[f]' -strict -2 outputfile.mp4    \n

Ini menambahkan aliran audio dan bantalan yang diperlukan untuk mendapatkan audio.

\n\n

Disediakan untuk kelengkapan:

\n\n

Cara Mudah: Anda dapat menggunakan proses 2 langkah yang sudah Anda ketahui:

\n\n

Pertama Potong dan ekstrak 2 bagian yang Anda inginkan dari video berdurasi penuh.
\n Kedua Gabungkan 2 bagian ini.

\n\n

Ini harus menjadi jalan keluar yang mudah.

\n\n

Untuk memotong sepotong, katakanlah detik ketiga hingga detik kesembilan: 

ffmpeg -i input.mp4 -ss 00:00:03 -to 00:00:09 -c:v copy -c:a copy part1.mp4    \n

Anda dapat mengulangi proses untuk bagian kedua. Jika dari detik ke-10 hingga ke-16: 

ffmpeg -i input.mp4 -ss 00:00:10 -to 00:00:16 -c:v copy -c:a copy part2.mp4\n

Lihat juga ini utas untuk lebih jelasnya.

\n\n

Sekarang Anda memiliki 2 file yang dapat Anda gabung. Anda dapat "menggabungkan" setidaknya dalam 3 cara berbeda. Cara melakukannya tanpa re-encoding adalah dengan melakukannya melalui opsi demuxer. Buat file teks bernama (katakanlah) joinlist.txt dengan konten ini: 

file 'part1.mp4'\nfile 'part2.mp4'\n

Untuk menggabungkan dua file, tempatkan 2 film dan teks (joinlist.txt) dalam folder yang sama dan gunakan perintah ini: 

ffmpeg -f concat -i joinlist.txt -c copy joinedfile.mp4\n

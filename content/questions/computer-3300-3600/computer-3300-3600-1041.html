Solusi berikut telah memecahkan masalah ini untuk saya:

\n\n
    \n

  1. Unduh dan pasang Tangkapan Layar Halaman Penuh dari https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaechififmbbbbbknoalclacl

    2. \n

  2. Setelah penginstalan selesai, buka "Tools \\ Extensions"

    3. \n

  3. Cari "Tangkapan Layar Halaman Penuh" dan centang kotaknya "Izinkan akses ke file URL"

    4. \n
\n\n

Sekarang Anda dapat mengambil gambar dari URL lokal / file :)

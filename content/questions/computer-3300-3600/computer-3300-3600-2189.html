Saya ingin PC saya (dan Firefox) tetap masuk selama berminggu-minggu. Saat saya memulai Firefox, Firefox meminta kata sandi utama saya saat pertama kali saya memerlukan kata sandi yang telah saya simpan. Setelah itu, selalu otomatis mengisi setiap kata sandi.

\n\n

Dapatkah saya mengatur ulang sehingga akan kembali seperti yang diminta?

\n\n

Tidak setiap kali, hanya pertama kali setelah reset.

\n\n

Dengan cara ini saya dapat mengunci kembali sesi saya dalam semalam. Saya tahu saya bisa mematikan firefox dan menyalakannya kembali tetapi saya tidak ingin melakukannya karena saya sering membuka banyak jendela untuk mengingatkan saya keesokan harinya tentang apa yang saya kerjakan pada malam sebelumnya.

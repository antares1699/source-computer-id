Kata kunci pencarian Chrome disimpan dalam file SQLite Web Data di dalam folder Data Pengguna. Jalur yang tepat tergantung pada OS yang Anda gunakan:

\n\n

macOS 

~/Library/Application Support/Google/Chrome/Default/Web Data\n

Windows XP 

C:\\Documents and Settings\\<username>\\Local Settings\\Application Data\\Google\\Chrome\\User Data\\Default\\Web Data\n

Windows Vista / 7/10 

C:\\Users\\<username>\\AppData\\Local\\Google\\Chrome\\User Data\\Default\\Web Data\n

Linux 

~/.config/google-chrome/Default/Web Data\n

Di dalam file database SQLite Web Data itu, tabel keywords memiliki daftar kata kunci pencarian.

\n\n

Anda dapat mengeditnya menggunakan apa pun yang dapat menggunakan database SQLite (mis Microsoft Access dengan plugin SQLite ODBC , membuat skrip solusi dengan Modul sqlite3 Python , dll.)

\n\n

Sayangnya, Chrome membuat file tetap terkunci saat sedang berjalan, jadi menurut saya solusi sinkronisasi otomatis di luar Chrome tidak akan mungkin dilakukan.

\n\n

Jika Anda benar-benar menginginkan keseluruhan kurus pada database Chrome Web Data, Anda dapat menyisirnya bagian itu dari kode sumber Chromium untuk petunjuk .

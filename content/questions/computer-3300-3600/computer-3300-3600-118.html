Di terminal:

\n\n

Untuk membuat daftar semua file di direktori saat ini dan semua sub-direktori: 

$ find .\n

Untuk mencari file berdasarkan nama di direktori saat ini dan semua sub-direktori: 

$ find . -name <file_name>\n

Atau untuk mencari di semua file di direktori saat ini dan semua sub-direktori untuk string tertentu: 

$ grep -R \"<string>\" *\n

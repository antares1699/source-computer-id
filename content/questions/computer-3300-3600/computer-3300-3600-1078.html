Terkadang "mode permainan" diaktifkan di Lycosa, yang terkunci Tombol Windows ( kunci perintah di Mac).

\n\n

Petunjuk tentang cara menonaktifkan mode permainan tersedia di Situs Razer

\n\n
\n

Anda dapat mengaktifkan mode permainan untuk Razer Lycosa dengan menekan dan\n memegang Logo Razer di Panel Sentuh dan kemudian menekan Windows\n kunci. Ini akan menonaktifkan kunci Windows. Untuk mengaktifkan kembali Windows\n Anda hanya perlu menekan dan menahan Logo Razer di Panel Sentuh\n lalu tekan kembali tombol Windows.

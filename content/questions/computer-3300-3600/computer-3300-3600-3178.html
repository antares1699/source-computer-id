Ada program tertentu yang selalu saya jalankan yang mengganggu proses shutdown. Ketika saya mencoba mematikan laptop saya, sebuah pesan muncul yang mengatakan bahwa sebuah program mengganggu proses mematikan, tetapi ketika saya memeriksa program itu sudah dimatikan. Program-program ini mungkin hanya membutuhkan waktu sedikit lebih lama untuk dimatikan. Apakah ada cara yang baik untuk mematikan OSX tanpa harus berurusan dengan gangguan ini?

","source":"https://superuser.com/questions/207228/how-to-shutdown-osx-and-avoid-programs-inturrupting-the-shutdown","author":"tony_sid","replies":[],"answers":[{"answer":"

Anda dapat menggunakan sudo shutdown -h now di terminal Anda. Program shutdown mengirimkan SIGTERM dan setelah beberapa waktu SIGKILL menyela ke semua program yang sedang berjalan. Jadi shutdown tidak boleh diganggu.

Nah, saya mengalami hal yang sama di salah satu antarmuka saya. Output dari ethtool dan mii-tool sedikit berbeda, yang ditunjukkan di bawah ini.

\n\n

mii-tool -v memberi tahu kita: 

eth0: negotiated 100baseTx-FD flow-control, link ok\n  product info: vendor 00:07:32, model 17 rev 2\n  basic mode:   autonegotiation enabled\n  basic status: autonegotiation complete, link ok\n  capabilities: 1000baseT-HD 1000baseT-FD 100baseTx-FD 100baseTx-HD 10baseT-FD 10baseT-HD\n  advertising:  100baseTx-FD 100baseTx-HD 10baseT-FD 10baseT-HD flow-control\n  link partner: 1000baseT-HD 1000baseT-FD 100baseTx-FD 100baseTx-HD 10baseT-FD 10baseT-HD flow-control\n

Yang tampaknya kami tidak mengiklankan 1000BaseT-HD / FD.

\n\n

Namun jika Anda melihat keluaran dari ethtool Anda akan mendapatkan yang berikut: 

Settings for eth0:\n        Supported ports: [ TP MII ]\n        Supported link modes:   10baseT/Half 10baseT/Full\n                                100baseT/Half 100baseT/Full\n                                1000baseT/Half 1000baseT/Full\n        Supports auto-negotiation: Yes\n        Advertised link modes:  10baseT/Half 10baseT/Full\n                                100baseT/Half 100baseT/Full\n                                1000baseT/Half 1000baseT/Full\n        Advertised pause frame use: No\n        Advertised auto-negotiation: Yes\n        Link partner advertised link modes:  10baseT/Half 10baseT/Full\n                                             100baseT/Half 100baseT/Full\n        Link partner advertised pause frame use: No\n        Link partner advertised auto-negotiation: Yes\n        Speed: 100Mb/s\n        Duplex: Full\n        Port: MII\n        PHYAD: 0\n        Transceiver: internal\n        Auto-negotiation: on\n        Supports Wake-on: pumbg\n        Wake-on: g\n        Current message level: 0x00000033 (51)\n        Link detected: yes\n

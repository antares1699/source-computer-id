Buka Start - Control Panel - Autoplay dan pilih tindakan default untuk setiap jenis media. Secara default, Windows 7 telah autorun.ini dinonaktifkan untuk perangkat USB untuk melindungi dari malware. Jika ini tidak menyelesaikan masalah Anda, Anda mungkin perlu mengaktifkan autorun.ini dan membuatnya di media untuk saudara perempuan Anda.

\n\n

Terenkripsi 20,04 Penuh Instal USB yang Mem-boot Mode BIOS dan UEFI

\n

Ubuntu 20.04 membuat enkripsi disk penuh menjadi mudah.

\n
    \n

  • Cabut HDD

    \n
    • \n

  • Boot USB Langsung dalam mode BIOS / Legacy, masukkan drive Target.

    \n
    • \n

  • Mulai Instal Ubuntu 20.04 LTS.

    \n
    • \n

  • Pilih Bahasa, Keyboard, Nirkabel, Instalasi Normal, Instal Pihak ketiga ...\n.

    \n
    • \n

  • Pada Instalasi ketik Tag "Hapus disk dan instal Ubuntu" dan kemudian pilih "Fitur lanjutan". Klik "Gunakan LVM dengan instalasi Ubuntu baru" dan kemudian "Enkripsi instalasi Ubuntu baru untuk keamanan".

    \n
    • \n
\n

\n
    \n

  • Pilih kunci keamanan. Timpa ruang disk kosong jika mau.

    \n
    • \n

  • Pilih Negara, lalu Nama pengguna dan Kata Sandi.

    \n
    • \n

  • Ketika Instalasi selesai, drive akan boot dengan enkripsi dalam mode BIOS.

    \n
    • \n

  • Buka file ISO 20.04 dan salin folder boot dan ESP ke partisi 1.

    \n
    • \n
\n

\n
    \n

  • Salin grub.cfg dari Partisi 5 / boot / grub / ke Partisi 1 / boot / grub / timpa grub.cfg yang ada.

    \n
    • \n

  • Instal Ulang GRUB:

    \n

    sudo mount /dev/sdx1 /mnt
    sudo grub-install --boot-directory=/mnt/boot /dev/sdx

    \n
    • \n

  • USB penginstalan Penuh yang dienkripsi sekarang seharusnya berfungsi dalam mode BIOS dan UEFI.

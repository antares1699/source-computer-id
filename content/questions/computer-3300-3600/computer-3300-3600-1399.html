Berdasarkan tanggapan dari Larry dan user193782, saya menemukan bahwa ini berfungsi untuk LXDE di Debian stable (wheezy): 

$ su\n$ cd /etc/X11/Xsession.d\n$ vi 99-synclient\n

File 99-synclient harus berisi: 

#! /bin/sh\nsynclient TapButton1=1\n

Kemudian buat 99-synclient dapat dieksekusi: 

$ chmod 555 99-synclient\n

Silakan coba langkah-langkah ini:

\n\n
    \n

  1. Buka Terminal Root (Menu> Aksesoris> Terminal Root)

    2. \n

  2. Masukkan kata sandi Administratif

    3. \n

  3. Ketik perintah berikut: 

    mkdir /etc/X11/xorg.conf.d\nleafpad /etc/X11/xorg.conf.d/synaptics.conf\n
    4. \n

  4. Salin skrip berikut 

    Section \"InputClass\"\nIdentifier      \"Touchpad\"\nMatchIsTouchpad \"yes\"  \nDriver          \"synaptics\"                     \nOption          \"MinSpeed\"              \"0.5\"\nOption          \"MaxSpeed\"              \"1.0\"\nOption          \"AccelFactor\"           \"0.075\"\nOption          \"TapButton1\"            \"1\"\nOption          \"TapButton2\"            \"2\"    \nOption          \"TapButton3\"            \"3\"     \nOption          \"VertTwoFingerScroll\"   \"1\"     \nOption          \"HorizTwoFingerScroll\"  \"1\"     \nOption          \"VertEdgeScroll\"        \"1\"\nOption          \"CoastingSpeed\"         \"8\"\nOption          \"CornerCoasting\"        \"1\"\nOption          \"CircularScrolling\"     \"1\"\nOption          \"CircScrollTrigger\"     \"7\"\nOption          \"EdgeMotionUseAlways\"   \"1\"\nOption          \"LBCornerButton\"        \"8\"   \nOption          \"RBCornerButton\"        \"9\"     \nEndSection\n
    5. \n

  5. Kemudian tempelkan ke file synaptics.conf

    6. \n

  6. Klik File dan Simpan

    7. \n

  7. Logoff dan login lagi .....

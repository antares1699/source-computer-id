Menurut saya, Anda dapat melakukan ini dengan mudah dengan skrip PowerShell (dijalankan sebagai administrator).

\n\n

Berikut adalah contoh skrip PowerShell yang ditemukan di Internet : 

#Powershell Script To Install SNMP Services\nImport-Module ServerManager\n#Check If SNMP Services Are Already Installed\n$check = Get-WindowsFeature | Where-Object {$_.Name -eq \"SNMP-Services\"}\nIf ($check.Installed -ne \"True\") {\n        #Install/Enable SNMP Services\n        Add-WindowsFeature SNMP-Services | Out-Null\n}\n

Anda dapat menemukan informasi lebih lanjut tentang menemukan nama fitur di:
\n Windows Server 2008 R2: Menambahkan Fitur melalui PowerShell

\n\n

Jika WMIC adalah sebuah opsi, lihat artikel ini:
\n Tips Server Core Baru
\n Menggunakan alat Windows Server 2008 Core OCList dan OCSetup CLI baru untuk Menambah & Menghapus Peran Server

\n\n

Jika pemrograman adalah sebuah opsi, lihat artikel stackoverflow ini:

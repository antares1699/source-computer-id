Komputer Windows Versi 10.0.15063 saya (Dell Precision 5520) akan tidur saat saya menutup penutupnya, meskipun dalam pengaturan yang saya tentukan tidak melakukan apa-apa. Lihat gambar:

\n

\n

Saya sudah mencoba memulai ulang komputer saya dan mengaktifkannya, dan membuat profil daya saya sendiri. Pengaturan tetap ada tetapi jendela tidak menghormatinya.

\n

Bagaimana cara memperbaikinya?

Masalahnya adalah bahwa pengaturan di panel kontrol tidak mengubah atribut di registri. Ini mungkin karena kebijakan grup, alat manajemen daya pihak ketiga, atau bug windows.

\n\n
    \n
  1. jalankan perintah berikut di prompt cmd untuk melihat pengaturan daya apa yang dijalankan oleh sistem Anda: PowerCfg /q
    2. \n
  2. Temukan GUID pengaturan untuk menutup tutup Anda. Bagi saya itu
    \n

    GUID subkelompok: 4f971e89-eebd-4455-a8de-9e59040e7347
    GUID Pengaturan Daya: 5ca83367-6e45-459f-a27b-476b1d01c936 Perhatikan setting saat ini adalah 1. Artinya akan sleep saat tutupnya ditutup.

    \n
    3. \n
  3. Buka regedit. (Anda dapat mencarinya di menu mulai)
    4. \n
  4. Pergi ke entri berikut:\nComputer\\HKEY_LOCAL_MACHINE\\SYSTEM\\CurrentControlSet\\Control\\Power\\PowerSettings\\4f971e89-eebd-4455-a8de-9e59040e7347\\5ca83367-6e45-459f-a27b-476b1d01c936
    5. \n
  5. Ubah Kunci Attributes agar memiliki nilai DWORD 0
    6. \n
  6. Jika Anda ingin mengubah profil daya lainnya, Anda dapat masuk ke subfolder DefaultPowerSchemeValues dan mengubah nilai Attributes menjadi 0 juga.
    7. \n

Jawaban di atas secara teknis benar, namun, saya menemukan hal tambahan yang harus saya lakukan untuk benar-benar menghentikan laptop saya memasuki mode tidur.

\n

Saya sebelumnya telah memilih power plan Performa, tetapi baru-baru ini saya menemukan ada beberapa opsi yang tidak dihargai. Jadi hanya dengan iseng, saya pikir saya akan mencoba menyetel laptop saya kembali ke paket daya Seimbang dan menyesuaikan pengaturannya agar sesuai dengan kebutuhan saya. Lihatlah pengaturannya tiba-tiba semua mulai dihormati. Saya juga mencatat bahwa rencana daya Kinerja yang dipilih sebelumnya menghilang dari daftar opsi rencana daya saya. Jadi tampaknya salah satu Pembaruan Windows baru-baru ini berpotensi menghapus rencana daya Kinerja dari daftar opsi default, dan saya berasumsi karena itu hanya menawarkan kompatibilitas mundur sebagian.

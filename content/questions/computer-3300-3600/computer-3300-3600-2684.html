Seringkali Anda dapat melakukan ekspor secara manual dengan menyalin file xpi dari profil firefox Anda. Ini adalah contoh Linux, tetapi metode serupa dapat digunakan untuk Windows: 

mkdir somextension\ncd somextension/\ncp ~/.mozilla/firefox/5x0w1g3w.weave24a/extensions/jid0-k75TfRGfOXPHfEZmJ9cKu5eCgLc\\@jetpack.xpi .\nmkdir orig\ncd orig\nunzip ../jid0-k75TfRGfOXPHfEZmJ9cKu5eCgLc\\@jetpack.xpi\n

Contoh di atas menyalin xpi dari profil firefox saya ke direktori kerja dan kemudian menggunakan unzip untuk mengekstrak file ke subdirektori orig /

\n\n

Anda tidak memerlukan langkah ekstrak jika Anda hanya tertarik dengan file xpi.

Kutipan dari :: Momzillazine :: - Memasang Ekstensi

\n\n

File ekstensi terletak di subfolder "ekstensi" dari :: folder profil :: (single-user) dan di subfolder "extensions" dari :: direktori instalasi :: (global); ada satu folder per ekstensi, dinamai menggunakan kode ID mereka. Untuk mengetahui yang mana, lihat di dalam setiap folder "chrome" subfolder di mana Anda akan menemukan beberapa informasi yang dapat dibaca manusia mengenai ekstensi.

\n\n

Catatan : Mulai di Gecko 2.0 (Firefox 4 / Thunderbird 3.3 / SeaMonkey 2.1), File XPI tidak lagi dibuka ketika ekstensi dipasang. Sebaliknya, XPI itu sendiri ditempatkan di direktori ekstensi, dan file dimuat langsung dari paket.

\n\n

Di Windows 7, buka folder: "% APPDATA% \\ Mozilla \\"\nBuka Mozilla> Firefox> Profil> * .default> ekstensi

\n\n

Di sini Anda akan menemukan beberapa file xpi, dan ekstensi firefox .xpi Anda akan termasuk di antara daftar itu.

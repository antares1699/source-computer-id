Apakah mungkin untuk membuka aplikasi x pada tampilan pengguna lain?

\n\n

Katakanlah saya memiliki pengguna A yang masuk ke desktop ubuntu, dan pengguna B yang mengakses mesin ubuntu melalui SSH.

\n\n

Saya tahu bahwa dimungkinkan bagi pengguna B untuk membuka aplikasi x secara lokal, tetapi apakah mungkin bagi pengguna B untuk membuka program (yaitu firefox) dan menampilkannya pada sesi desktop pengguna A?

\n\n

Dan jika ya, bagaimana caranya?

","source":"https://superuser.com/questions/603604/ubuntu-x11-is-it-possible-to-open-an-x-application-in-another-users-session","author":"Martin Nielsen","replies":[{"reply":"Terkait: Cara membuka aplikasi dalam menjalankan sesi X. ","author":"slhck"}],"answers":[{"answer":"

Ya, tapi itu akan tergantung pada Anda xhost pengaturan. Jadi, jika Anda telah terhubung sebagai penggunaB ke mesin yang menjalankan sesi X milik penggunaA, Anda perlu melakukan dua hal:

\n\n
    \n

  1. userA harus mengizinkan Anda mengakses: 

    userA@foo $ xhost +\n
    2. \n

  2. Anda perlu menyetel variabel DISPLAY Anda ke remote display: 

    userB@foo $ export DISPLAY=:0;\n
    3. \n
\n\n

Sekarang userB akan dapat menjalankan program grafis.

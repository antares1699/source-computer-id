Saya tidak bisa menentukan spasi ikon desktop Windows per tema.

\n\n

Dulu mungkin di Windows 7 , untuk menyematkannya di tema Anda (file mystyle). Saya tidak dapat menemukan efeknya di Winodws 8, dan saya bertanya-tanya apakah itu masih dalam file gaya dan akses kami telah dibatasi?

Ya, itu mungkin, tetapi hanya ada satu cara yang saya tahu sejak mereka menghapus warna dialog pengaturan lanjutan (mungkin utilitas pihak ketiga untuk melakukannya juga).

\n\n
    \n

  1. tekan + R dan ketik regedit (atau cari di Apps).

    2. \n

  2. Arahkan ke HKEY_CURRENT_USER\\Control Panel\\Desktop\\WindowMetrics:

    \n\n

    3. \n

  3. Di sana Anda perlu mengedit IconSpacing dan IconVerticalSpacing, yang pertama untuk jarak horizontal dan yang terakhir untuk yang vertikal (berkisar -480 hingga -2730).

    4. \n

  4. Untuk ruang horizontal, semakin dekat ke -2730, semakin banyak ruang antara ikon dan sebaliknya (tetapi di sini Anda mengubah ukuran kotak yang mengelilingi ikon:

    \n\n

    -2730:
    \n

    \n\n

    -480:
    \n

    5. \n

  5. Hal yang sama untuk ruang vertikal, tetapi Anda mengubah pemisahan antara kotak yang mengelilingi ikon; misalnya pada -480 mereka tumpang tindih:

    \n\n

    -2730:
    \n

    \n\n

    -480:
    \n

    6. \n
\n\n

Untuk menerapkan perubahan, satu-satunya cara yang saya tahu adalah log off dan kembali lagi dan kemudian memindahkan satu ikon, memulai ulang Explorer tidak memotongnya kali ini.

\n\n

Jika Anda memiliki akses ke mesin Windows 7, saya akan mengatakan lebih mudah untuk mengubah jarak di sana menggunakan opsi biasa dalam dialog pengaturan lanjutan untuk warna, dan kemudian melihat registri untuk nilai yang ingin Anda terjemahkan ke Windows Anda 8 mesin.

\n\n

Bagaimanapun ini akan menjadi beberapa nilai untuk membantu Anda mengedit nilai tersebut: 

...\n45: -1155\n44: -1140\n43: -1125\n42: -1110\n41: -1095\n...\n

Ngomong-ngomong, nilai lain yang berguna untuk diubah adalah BorderWidth yang mengontrol lebar batas jendela.

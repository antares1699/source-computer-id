Saya ingin membuat sejumlah lingkungan mesin virtual untuk menguji rilis beta baru - Windows 7, VS2010, Windows Server 2008, dan seterusnya. OS host saya adalah Windows XP. Apa cara terbaik untuk melakukan ini?

Jika Anda menginginkan solusi gratis, unduh Virtual PC 2007 .

\n\n

Instal dan kemudian harus lurus ke depan - Anda memperlakukan setiap mesin virtual sebagai komputer fisiknya sendiri, jadi masukkan ke dalam CD / DVD atau pasang ISO (atau pasang VHD) dan lanjutkan.

\n\n

Jika Anda tidak keberatan membayar, saya sangat merekomendasikan VMware Workstation , pada akhirnya, Anda akan dapat mencapai hal yang sama, tetapi Workstation memiliki begitu banyak fitur (seperti snapshot) yang tidak dapat saya lakukan tanpanya.

VirtualBox itu bagus lintas platform , Gratis aplikasi yang memungkinkan Anda melakukan ini. Anda dapat meniru hampir setiap sistem operasi di bawah satu sama lain. Daftar lengkap sistem operasi tamu yang kompatibel dapat ditemukan sini .

\n\n

Setelah Anda menginstal program, Anda dapat membuat "mesin virtual" individual untuk setiap komputer. Anda dapat membuat masing-masing memiliki hard-disk sendiri (atau beberapa), masing-masing dengan jaringan yang dijembatani dan dukungan perangkat USB.

\n\n

Hal bagus lainnya adalah Anda dapat memuat file ISO ke driver CD-Rom virtual, yang jauh lebih cepat daripada "memuat dari disk" konvensional (kebanyakan hard-drive memiliki throughput yang lebih tinggi daripada semua drive CD / DVD).

Saat berada di VIM, saya memiliki pintasan pemimpin untuk berikut ini: 

:!git commit % -m 'updated '; git push;                                                                                                                                                                   \n

Kursor saya mendarat di karakter setelah diperbarui setelah saya menjalankan pintasan pemimpin ini. Sekarang pertanyaannya adalah:

\n\n

Bagaimana cara menghapus sampai akhir baris itu menggunakan tombol pintas? Dalam bash, itu akan menjadi "CTRL-k", tapi itu tidak berfungsi di sini. Saya hanya ingin menghapus semuanya setelah kata "diperbarui".

\n\n

Tidak ada pemetaan pintasan default untuk melakukan apa yang Anda minta. Dokumentasi lengkap tentang pengeditan mode perintah dapat ditemukan melalui :help cmdline-editing. Anda dapat menjelajahinya untuk menemukan sesuatu yang berguna bagi Anda. Tapi saya rasa fungsi yang Anda minta tidak ada di dalamnya.

\n\n

Saya kira cara default pengeditan baris perintah tidak memerlukan operasi seperti itu. Biasanya kami menghapus kata sebelum kursor pada <Ctrl-W> tanpa memindahkan kursor. Tetapi tentu saja, jika Anda memiliki cara sendiri untuk mengeditnya, Anda selalu dapat menyesuaikan perilaku vim sesuai keinginan Anda.

\n\n

Anda dapat memiliki pintasan yang Anda inginkan dengan memetakan <C-K> ke fungsi yang ditentukan agar sesuai dengan perilaku yang Anda inginkan. Di sini saya menulis contoh. Anda cukup menempelkan kode berikut ke ~/.vimrc Anda. Sumber ulang file. Dan Anda bisa mendapatkan jalan pintas dengan cara Anda sendiri. 

cnoremap <C-k> <C-\\>estrpart(getcmdline(),0,getcmdpos()-1)<CR>\n

Jika Anda tertarik, ini memanggil pintasan mode perintah default (<C-K>) untuk mengevaluasi {expr} dan mengganti seluruh baris perintah dengan hasilnya. strpart() adalah fungsi pemotongan string yang diberikan oleh vim.

\n\n

Setelah penekanan tombol <C-k> Anda, vim akan mengekstrak konten dari mode perintah sebagai string. Dan letakkan kembali hanya konten dari awal string ke satu posisi di depan posisi kursor. Konten lainnya akan terpotong dan dengan demikian dihapus.

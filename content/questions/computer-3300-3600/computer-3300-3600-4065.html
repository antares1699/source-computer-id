Format tekstual .reg tidak mendukung fitur keamanan WinNT; itu hanya menyimpan data yang diekspor, seperti yang Anda lihat dengan membuka file .reg dengan editor teks.

\n\n

Format "sarang" identik dengan yang digunakan oleh Windows untuk menyimpan registri itu sendiri. Artinya itu mendukung semua fitur digunakan oleh registri, termasuk izin keamanan dan kepemilikan. Anda dapat memeriksa file "sarang" yang diekspor dengan menggunakan "Muat sarang" di regedit atau reg load di baris perintah, dan impor ke registri utama dengan reg restore.

\n\n

Namun, ingat bahwa pengguna di Windows diidentifikasi oleh SID mereka; akun lokal akan memiliki SID yang berbeda pada mesin yang berbeda, jadi Anda mungkin masih harus menyesuaikan izin yang diimpor secara manual. (Akun bawaan seperti "NT AUTHORITY \\ SYSTEM" memiliki SID yang sama di mana-mana. Akun domain selalu memiliki SID yang sama di setiap anggota domain.)

Gatal-gatal

\n\n

Registri terdiri dari sejumlah bagian logis, atau " gatal-gatal ". Sarang umumnya dinamai menurut definisi Windows API, yang semuanya dimulai dengan" HKEY ". Seringkali disingkat menjadi nama pendek tiga atau empat huruf yang diawali dengan" HK "(misalnya HKCU dan HKLM). Secara teknis, sudah ditentukan sebelumnya menangani (dengan nilai konstan yang diketahui) ke kunci tertentu yang disimpan dalam memori, atau disimpan dalam file sarang yang disimpan dalam sistem file lokal dan dimuat oleh kernel sistem pada saat boot dan kemudian dibagikan (dengan berbagai hak akses) antara semua proses yang berjalan pada sistem lokal, atau dimuat dan dipetakan dalam semua proses yang dimulai dalam sesi pengguna saat pengguna masuk ke sistem.

\n\n

Bagaimana cara menambahkan kunci reg

\n\n

Cukup klik dua kali padanya dan itu akan meminta Anda untuk mengkonfirmasi operasi dan klik Ok itu akan ditambahkan ke file registri pada OS. Atau buka editor registri melalui perintah run dan klik File> Import dan kemudian Anda dapat menambahkan file .reg atau hive.

\n\n

\n\n

\n\n

File sarang tidak dapat diakses secara langsung saat Anda dapat menjalankan file registri. Sarang hanya dapat ditambahkan melalui opsi import.

\n\n

\n\n

Ketika Anda mencoba mengimpor sarang, itu akan sepenuhnya mengembalikan kunci di bawahnya yang telah Anda buat cadangannya. Sementara hanya satu kunci .reg akan mempengaruhi satu-satunya di bawah sarang ke tempat tertentu (kunci) yang dimilikinya.

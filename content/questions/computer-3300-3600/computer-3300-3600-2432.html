Saya rasa Anda tahu nama partisi Anda dari gparted. (mari gunakan / dev / sda3 untuk contoh ini)

\n\n

Yang perlu Anda lakukan hanyalah mengedit fstab Anda dan menambahkan dan memasukkan partisi ini agar dapat dipasang sebagai / home: (buat cadangan fstab terlebih dahulu) 

$ sudo cp /etc/fstab /etc/fstab.bak\n$ sudo gedit /etc fstab\n

Tambahkan garis: 

/dev/sda3 /home ext3 defaults 0 2\n

Sebuah reboot harus membawa drive ini ke mount / home seperti yang Anda cari.\nJika UID pengguna Anda pada penginstalan asli dan penginstalan baru berbeda, Anda mungkin mengalami masalah ... tapi itu untuk pertanyaan lain.

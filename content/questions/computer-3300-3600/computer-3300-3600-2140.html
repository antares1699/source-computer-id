Saya memiliki masalah yang sama (VirtualBox 4.1.10, Host Ubuntu 11.10, Guest Ubuntu 11.10). Begini cara saya memperbaikinya:

\n\n

Setel dua antarmuka jaringan untuk VM. Yang pertama harus NAT dan yang kedua harus Host-Only. Jika Anda tidak dapat menambahkan koneksi khusus-host, pastikan Anda telah mengaturnya di Preferensi -> Jaringan.

\n\n

Di OS Tamu (Ubuntu untuk saya), edit /etc/network/interfaces, tambahkan yang berikut di bagian akhir: 

auto eth1 # this refers to the Host-only network interface\niface eth1 inet static\naddress 192.168.56.10 # Arbitrary IP address\nnetmask 255.255.255.0\n

dan reboot. Dan itu dia. Jangan memodifikasi apa pun untuk eth0, karena kita perlu mendapatkan pengaturannya secara otomatis melalui DHCP untuk mengakses internet. eth1 memungkinkan tuan rumah dan tamu berkomunikasi satu sama lain.

Anda dapat mengubah IP Tamu VB Anda menjadi statis dengan memodifikasi file-file ini. 

vim /etc/network/interfaces\n

Tambahkan baris ini di akhir: 

auto eth0\niface eth0 inet static\naddress xxx.xxx.xxx.xxx\nnetmask xxx.xxx.xxx.xxx\ngateway xxx.xxx.xxx.xxx\n

Untuk Pengaturan DNS 

vim /etc/resolv.conf\n

dan tambahkan: 

nameserver xxx.xxx.xxx.xxx\n

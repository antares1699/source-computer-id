Saya ingin memblokir akses ke alamat ip tertentu melalui host windows.

\n\n

Ini adalah alamat ip 41.190.8.17. Dan ini adalah isi dari file host saya. 

# localhost name resolution is handle within DNS itself.\n\n127.0.0.1 41.190.8.17\n

Jika saya membuka browser dan memasukkan 41.190.8.17, saya langsung diarahkan ke situs.

\n\n

Tolong bisakah seseorang memberi tahu saya mengapa mengabaikan ini?

Ini tidak akan berfungsi seperti itu, karena file-host hanya digunakan untuk resolusi nama melalui DNS. Karena ini sudah merupakan Alamat IP, Windown (atau aplikasi Anda) tidak perlu melakukan Pencarian DNS.

\n\n

Anda tidak dapat memblokir alamat ip melalui file host, hanya resolusi domain!\nJika Anda ingin meneruskan ips ke localhost Anda harus membuat aturan proxy, tetapi jika Anda hanya ingin memblokir lalu lintas alamat ip, saya akan menggunakan ini.

\n

Simpan ini sebagai block.bat 

set ip=%1\nnetsh advfirewall firewall add rule name="BLOCK IP ADDRESS - %ip%" dir=in action=block remoteip=%ip%\nnetsh advfirewall firewall add rule name="BLOCK IP ADDRESS - %ip%" dir=out action=block remoteip=%ip%\n

kemudian dari baris Perintah administratif, jalankan\nblock.bat 192.168.0.1 atau alamat IP apa pun yang ingin Anda blokir\nIni tidak mengalihkan ke 127.0.0.1, hanya memblokirnya.

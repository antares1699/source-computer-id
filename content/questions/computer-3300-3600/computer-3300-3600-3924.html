Oke, jadi akhirnya saya menemukan "retasan". Peretasan ini membutuhkan pembuatan ekstensi untuk menangani permintaan tersebut. Saya sudah mencoba beberapa upaya untuk "membajak" Chrome dengan cara yang lebih mudah tetapi tampaknya Chrome cukup aman untuk menghentikan saya melakukan itu dan ini yang paling dekat yang bisa saya dapatkan.

\n\n

Pertama, buat direktori kosong di suatu tempat yang dapat diakses di hard drive.

\n\n

Buat file manifest.json dengan konten berikut: 

{\n    \"name\": \"Open Chrome URLs\",\n    \"version\": \"1.0\",\n    \"manifest_version\": 2\n}\n

Buat file open.html dengan konten berikut: 

<html>\n<head>\n<title>Open Chrome URLs</title>\n<script type=\"text/javascript\" src=\"open.js\"></script>\n</head>\n<body>\n</body>\n</html>\n

Buat file open.js dengan konten berikut: 

window.addEventListener(\"load\", function(){\n    var key = \"secretKey\"; // replace \"secretKey\" with your own secret key\n    if(window.location.search == \"?key=\" + key && window.location.hash.length > 1){\n        chrome.tabs.update({\n            'url': \"chrome://\" + window.location.hash.substr(1) + \"/\"\n        });\n    }else{\n        document.body.appendChild(document.createTextNode(\"Invalid\"));\n    }\n});\n

Ganti kunci rahasia dengan milik Anda sendiri jika diinginkan.

\n\n

Kemudian, buka halaman Extensions (chrome://extensions/).

\n\n

Centang kotak "Mode pengembang" dan klik "Muat ekstensi yang belum dibuka paketnya" dan pilih direktori yang baru saja Anda buat.

\n\n

Anda sekarang akan melihat ekstensi baru muncul.

\n\n

\n\n

Salin id ekstensi.

\n\n

Terakhir, mulai Chrome dengan URL berikut sebagai parameter. 

chrome-extension://nihlceAnywayPutTheExtensionIdHere/open.html?key=secretKey#settings\n

Ganti bagian pertama dengan id ekstensi dengan yang baru saja Anda salin.

\n\n

Juga ganti secretKey dengan yang Anda setel di atas.

\n\n

Anda juga dapat menggunakan sebagian besar URL Chrome selain pengaturan.

\n\n

Catatan: Anda memerlukan pintasan ke Chrome, bukan tautan Internet.

\n\n

Semoga berhasil!

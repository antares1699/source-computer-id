Saya membuat arsip tar yang besar dan saya ingin membuat checksum arsip juga. Saya bisa mencapainya seperti ini: 

$ tar cfz archive.tar.gz files\n$ sha256sum archive.tar.gz > archive.tar.gz.sha256sum\n

Tetapi file arsipnya sangat besar dan pada media yang lambat, jadi saya memilih untuk tidak membaca semuanya lagi setelah menulisnya.

\n\n

Dapatkah saya membuat pipeline yang akan mencirikan file saat ditulis? Saya pikir mungkin saya bisa melakukan ini dengan utilitas tee, tetapi itu hanya menulis ke file, bukan ke input standar dari perintah lain.

Saya memiliki memori 6 GB di MacBook Pro saya tetapi setelah beberapa waktu saya memulai Chrome, itu memakan sekitar 3-4 GB memori saya. Adakah cara untuk membatasi penggunaan memori Chrome?

\n\n

P.S. untuk Mac, tampaknya tidak ada cara untuk menyetel argumen start-up untuk Chrome, juga tidak memiliki tombol "bersihkan memori" di pengelola tugas. Jadi cara biasa untuk Windows tidak berlaku untuk Mac.

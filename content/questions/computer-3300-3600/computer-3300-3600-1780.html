Saya memiliki MacBook Pro (Retina, 15 inci, Akhir 2013), 2,3 GHz, i7, Ram DDR3 16 GB dengan macOS High Sierra versi 10.13.3 tersambung ke monitor ganda LG. Laptop saya terhubung ke pengisi daya, jadi saya dapat menutup tutupnya dan hanya menggunakan 2 monitor eksternal.

\n\n

Masalah saya datang secara bertahap setiap saat. Penggunaan 20-30 menit pertama tampak normal, ini berfungsi dengan baik. Setelah itu CPU menjadi gila karena suatu alasan. Monitor Aktivitas menunjukkan kernal_task di lebih dari 500% penggunaan CPU sepanjang waktu. Itu tidak normal sama sekali.

\n\n

Saya tidak tahu apa yang salah atau bagaimana mengatasi masalah ini. Saya tidak pernah mengalami masalah ini ketika saya menggunakan laptop saya ketika tidak terhubung ke monitor eksternal. Ide ide?

\n\n

\n\n

Ini bisa menjadi tanda masalah perangkat keras, tetapi dalam kasus menjalankan 2 monitor eksternal mungkin diharapkan.

\n\n

Anda dapat mencoba mengatur ulang SMC, untuk berjaga-jaga, tetapi gejala itu sendiri bukanlah pertanda ada yang salah.

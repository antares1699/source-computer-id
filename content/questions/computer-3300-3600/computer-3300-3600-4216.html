Apakah ada penginstal tertentu yang Anda cari?","author":"music2myear"},{"reply":"Saya mencari semua program yang diinstal selama masa hidup komputer pada dasarnya dan berpikir saya mungkin dapat membangun sejarah dengan memasangkan operasi pemasangan dan pencopotan pemasangan","author":"Draineh"}],"answers":[{"answer":"

Anda dapat memfilter menurut EventID untuk melihat jenis acara tertentu. Ada banyak ID Peristiwa Penginstal Windows yang sesuai dengan berbagai jenis tindakan. 1033 menunjukkan produk telah diinstal, 1034 menunjukkan produk telah dihapus.

\n\n

Untuk Penginstal Windows, Anda juga dapat memfilter menurut Sumber. MsiInstaller adalah sumber untuk semua acara Penginstal Windows.

\n\n

InstallShield cenderung menjadi pembungkus untuk skrip MSI, sehingga umumnya memiliki ID peristiwa yang sama. Namun, paket penginstal lain umumnya tidak akan menggunakan EventID yang sama. Untuk penginstal apa pun, Anda harus menemukan id acara yang sesuai yang digunakannya (jika ada).

\n\n

PERBARUI detail:\nOh, dan itu semua ditemukan di penampil acara di bawah Aplikasi.

\n\n

UPDATE tanggapan:\nID Peristiwa Windows tidak berubah dari OS ke OS karena ini ditentukan oleh aplikasi. Dalam hal ini, Penginstal Windows.

\n\n

Saya menguji kotak XP saya dan server 2003 dan mereka berdua menggunakan 1033 untuk menginstal dan 1034 untuk uninstall.

\n\n

Anda dapat mencoba menginstal patch pembaruan windows atau serupa. Sesuatu yang kecil, seperti pembaruan sertifikat root atau sejenisnya. Kemudian periksa log peristiwa untuk entri yang sesuai. Ini akan memungkinkan Anda untuk melihat apakah log telah dihapus sejak pemasangan terakhir.

\n\n

PERBARUI detail lebih lanjut, ID alternatif:\nAda banyak sekali informasi online mengenai ID peristiwa, termasuk daftar semua EventID yang mungkin untuk Penginstal MSI. Melihat sekilas saya tidak begitu yakin apa perbedaan antara 1033/1034 dan 11707/11708 ID. Dokumentasi MS tampaknya menunjukkan bahwa dua jenis yang berbeda menunjukkan hal yang sama, yaitu berhasil menginstal atau menghapus, tetapi berisi jenis informasi yang berbeda.

Saya memiliki mesin Dell Inspiron n5110 dengan Intel Celerino WiFi N1030.

\n\n

Sudah sekitar dua minggu saya mulai menggunakan Windows 8. Sebelumnya, saya menggunakan Windows 7 Home premium di mana saya tidak mengalami masalah apa pun. Kemarin pagi, ketika saya mencoba menghubungkan WiFi saya, saya menemukan tanda silang merah di bar WiFi. Saya pikir itu mungkin telah dinonaktifkan tetapi ternyata tidak. Driver diinstal dan windows tidak menemukan masalah apa pun dengan driver dan adaptor itu sendiri. Saya mencoba menginstal ulang driver tetapi tidak berhasil. Pikir adaptor mungkin malfuntioning; mem-boot Linux dari USB dan adaptor mendeteksi jaringan nirkabel. Dan terakhir, saya mencoba System Restore untuk terakhir kali saat berfungsi.

\n\n

Tapi Ethernet bekerja dengan baik.

\n\n

EDIT: Saya mengunduh driver terbaru dari Intel untuk Windows 8 dan menginstal utilitas koneksi WiFi. Dan utilitas koneksi sebenarnya mendeteksi jaringan nirkabel tetapi jaringan tidak muncul di Jaringan.

Anda tahu adaptor itu baik-baik saja karena berfungsi di sistem operasi lain. Anda juga tahu bahwa jaringan dikonfigurasi dengan benar pada ujung modem / router karena Anda dapat terhubung dengan baik melalui Ethernet. Itu berarti kami sedang melihat masalah fisik atau masalah konfigurasi.

\n\n

Masalah fisik dengan kartu nirkabel umumnya disebabkan oleh penerimaan yang buruk. Untuk memeriksanya, coba boot kembali ke Linux dan periksa kekuatan jaringan nirkabel Anda dengan menggunakan alat seperti Stumbler ( http://tools.meraki.com/stumbler ). Anda akan terkejut betapa sering nuansa dalam kode driver berarti kekuatan sinyal cukup di satu sistem operasi tetapi tidak di yang lain.

\n\n

Adapun konfigurasi ada beberapa hal yang dapat Anda lihat.

\n\n

Mulailah dengan masuk ke Control Panel \\ Network and Internet \\ Network and Sharing Center \\ Advanced sharing settings dan periksa apakah adaptor nirkabel diaktifkan. Ini adalah yang mendasar dan sepertinya bukan penyebab masalah Anda karena utilitas koneksi mendeteksi pengaturan jaringan.

\n\n

Jika gagal, masuk ke Pengelola Perangkat. Temukan kartu jaringan nirkabel Anda di bawah "Adaptor jaringan". Klik kanan dan buka properti. Ada beberapa hal yang perlu diperiksa.

\n\n
    \n

  1. Di tab umum, periksa Status Perangkat yang menyatakan "Perangkat ini bekerja dengan benar".

    2. \n

  2. Di tab lanjutan, periksa properti disetel dengan benar. Sebagian besar adalah jargon teknologi, jadi pastikan untuk menggunakan jargon. Secara khusus, lihat properti manajemen daya dan matikan.

    3. \n

  3. Terkait dengan poin terakhir tersebut, hapus centang semua manajemen daya di tab manajemen daya.

    4. \n

  4. Akhirnya kita sampai pada tab driver. Menurut pengalaman saya, di sinilah letak masalah Anda. Intel Celerino WiFi N1030 untuk menginstal dengan driver yang salah. Saya tahu Anda mengatakan Anda mendapatkan driver terbaru tetapi coba dapatkan dari Dell, bukan Intel.

    5. \n
\n\n

Jika tidak ada yang berhasil, kembali ke daftar adapter jaringan di pengelola perangkat dan periksa perangkat duplikat. Pastikan untuk mengaktifkan menampilkan perangkat tersembunyi. Secara khusus Anda mencari duplikat: Microsoft ISATAP Adapters, atau Microsoft Virtual WiFI Miniport Adapters, atau Wan Miniport Drivers.

\n\n

Satu penunjuk terakhir, saya pernah mengalaminya di masa lalu di mana ini disebabkan oleh terlalu panas tidak ada hubungannya dengan driver. Jika ini menjadi masalah sesekali coba pantau suhu komputer Anda.

\n\n

Ini komprehensif jadi jika Anda mengikuti langkah saya, Anda akan memperbaiki masalah Anda. Semoga berhasil sobat!

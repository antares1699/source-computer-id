Sekali lagi, jawaban untuk masalah Anda adalah pertanyaan! Dan berikut ini yang Anda butuhkan: 

SELECT Name FROM FirstTable\nUNION\nSELECT Name FROM SecondTable\nUNION\nSELECT Name FROM ThirdTable\n

dll.

\n\n

Ini akan memberi Anda sekumpulan rekaman dari kolom name yang dikumpulkan oleh kueri dari FirstTable, SecondTable dan ThirdTable.

\n\n

Anda kemudian dapat menggunakan panduan pencarian dalam desain tabel untuk memilih kueri setelah Anda menyimpannya untuk mengisi menu drop down Anda.

\n\n
\n\n

Anda bisa menjalankan kueri di Access dengan melakukan hal berikut:

\n\n
    \n

  1. Buka tab "Buat" dan klik "Desain Kueri"
    \n

    2. \n

  2. Tutup saja jendela yang muncul yang meminta Anda untuk memilih tabel, kami tidak membutuhkannya.

    3. \n

  3. Pergi ke tab "Design" dan klik tombol dengan panah sampai Anda mendapatkan layar input tekstual. (Secara default, dikatakan SELECT;).
    \n

    4. \n

  4. Hapus teks default dan tempel kueri di atas.

    5. \n

  5. Klik "Jalankan".
    \n

    6. \n

  6. Simpan dengan menekan Ctrl + S sedangkan tab Query adalah tab aktif.

    7. \n

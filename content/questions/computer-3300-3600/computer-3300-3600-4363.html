Ketika saya menambahkan pengguna baru ke Windows (7 atau Server 2k8) dan saya menjadikannya anggota grup Administrator, apakah penting jika saya menyimpan atau menghapus keanggotaan Pengguna? Sebagian besar waktu saya menghapus keanggotaan Pengguna karena tampaknya berlebihan, tetapi apakah itu penting?

Jika Anda benar-benar merasa perlu untuk menghapus pengguna dari grup Pengguna, lakukanlah.","author":"User"}],"answers":[{"answer":"

Ini penting karena hak istimewa Tolak lebih diutamakan daripada Izinkan. Jika Pengguna Anda memiliki izin Tolak pada apa pun, keanggotaan Administrator mereka tidak akan mengizinkan mereka mengakses sumber daya. Anda dapat menguji ini pada Pengguna dengan membuat izin penolakan untuk mencantumkan konten folder, Anda akan mendapatkan dialog peringatan berikut: 

---------------------------\nSecurity\n---------------------------\nYou are setting a deny permissions entry. Deny entries take precedence over allow entries. This means that if a user is a member of two groups, one that is allowed a permission and another that is denied the same permission, the user is denied that permission.\n\nDo you want to continue?\n\n---------------------------\nYes   No   \n---------------------------\n

Jadi, kecuali Anda berencana untuk TIDAK PERNAH menetapkan Tolak ke Pengguna, hapus Administrator dari grup Pengguna.

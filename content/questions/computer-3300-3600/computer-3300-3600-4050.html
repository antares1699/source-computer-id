Jika shutdown Anda menerima sakelar -a (periksa dengan shutdown --help), Anda dapat melakukan hal berikut:

\n\n
    \n

  • Tambahkan pengguna yang seharusnya dapat mematikan sistem ke /etc/shutdown.allow.

    \n\n

    Contoh: 

    userA\nuserB\n
    • \n

  • Matikan sistem menggunakan sakelar -a.

    \n\n

    Contoh: 

    shutdown -a -h now\n
    • \n
\n\n

Sumber: Halaman manual UNIX: shutdown (8)

Saya menggunakan SL 6.4. Ini memiliki versi pengguna poweroff, halt, reboot yang disediakan oleh paket kode pengguna. Saya bisa mematikan, reboot sebagai pengguna biasa (dari baris perintah juga) 

    $ which {poweroff,reboot,halt}\n    /usr/bin/poweroff\n    /usr/bin/reboot\n    /usr/bin/halt\n\n    rpm -qf $(which poweroff reboot halt)\n    usermode-1.102-3.el6.x86_64\n    usermode-1.102-3.el6.x86_64\n    usermode-1.102-3.el6.x86_64\n

Sebagai root 

    # which {poweroff,reboot,halt}\n    /sbin/poweroff\n    /sbin/reboot\n    /sbin/halt\n\n    # rpm -qf $(which poweroff reboot halt)\n    upstart-0.6.5-12.el6.x86_64\n    upstart-0.6.5-12.el6.x86_64\n    upstart-0.6.5-12.el6.x86_64\n

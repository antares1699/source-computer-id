Tujuan saya adalah untuk mengoreksi jawaban linuxdev2013 tetapi saya tidak memiliki reputasi yang cukup untuk mengomentarinya.

\n\n

Saya perlu melakukan (perhatikan bahwa tidak ada tanda hubung sebelum mengunduh): 

sudo dnf system-upgrade download --refresh --releasever=26\n

Saya juga perlu menggunakan --nogpgcheck untuk mengatasi masalah gpg yang tampak jelas dengan paket alpha kernel, tetapi saya berharap itu adalah kesalahan sementara.

Cara yang jauh lebih mudah dan waras untuk ini.

\n\n

Jalankan perintah berikut di terminal satu per satu: 

sudo dnf install dnf-plugin-system-upgrade\nsudo dnf system-upgrade download --refresh --releasever=26\nsudo dnf system-upgrade reboot\n

Setelah bagian penginstalan selesai dan menjalankan boot ulang: 

sudo dnf system-upgrade clean\n

... dan nikmati penginstalan f26 alpha (ditingkatkan di tempat)!

\n\n

Untuk informasi lebih lanjut tentang plugin peningkatan sistem dnf lihat, https://fedoraproject.org/wiki/DNF_system_upgrade .

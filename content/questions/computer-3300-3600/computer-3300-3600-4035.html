Saya telah menginstal Google Chrome di Windows 8 beberapa waktu lalu. Hari ini saya perhatikan, bahwa ketika saya memulai Chrome dari menu awal, Chrome memulai versi metro. Saya hanya dapat memulai versi desktop dari ikon yang disematkan di bilah tugas. Bagaimana cara menjalankannya dalam mode desktop jika saya melepas pin dari bilah tugas?

Sebuah jalan pintas sederhana ke C:\\Users\\<username>\\AppData\\Local\\Google\\Chrome\\Application\\chrome.exe di desktop harus melakukan triknya. Jika ada, itu berhasil untuk saya.

\n\n

EDIT: Maaf, tidak membaca pertanyaan cukup dekat, melakukan ini untuk menu start juga berfungsi untuk saya.

\n\n

EDIT2: Solusi telah dikonfirmasi oleh penanya: Menelusuri ke C:\\Users\\<username> \\AppData\\Local\\Google\\Chrome\\Application, memilih chrome.exe dan memilih Pin to start dari menu konteks melakukan trik.

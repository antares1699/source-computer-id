Beberapa hari yang lalu saya menggunakan komputer saya seperti biasa dan saya mematikannya. Hari berikutnya tidak bisa boot. Itu hanya tinggal setelah jendela diagnostik perangkat keras pada layar garis bawah yang terputus-putus. Jadi jelas itu tidak bisa boot. Saya mencoba menyalakan dan mematikannya beberapa kali tetapi tidak berhasil. Akhirnya saya menggunakan disk windows 7 dan sepertinya tidak ada HDD. bahkan instalasi tidak akan melihat HDD. Jadi saya pikir itu sudah mati, saya membeli yang baru dipasang dengan windows dan menggunakan casing eksternal dengan HDD lama. colokkan dan masih tidak bisa melihatnya. Saya akhirnya mengunduh program partisi EASUS atau sesuatu dan HDD saya ada di sana terdaftar TANPA surat sistem. Namun saya bisa menjelajahinya dan saya menetapkannya sebagai Tidak Tersembunyi dan itu hidup kembali. Saya bisa melihatnya secara normal.

\n\n

Ada beberapa kemungkinan di sini. Bisa dengan mudah menjadi virus karena virus boot sector telah diketahui melakukan hal-hal seperti itu. Idenya adalah untuk tidak mempercayai hard drive lama tanpa format level rendah, ini akan memberi tahu Anda jika ada bagian dari hard drive yang rusak, atau jika ada virus, hilangkan seluruhnya.

\n\n

Hard drive bisa saja tidak dipetakan, tetapi itu tidak terjadi dengan sendirinya. Saya akan prihatin.

\n\n

Ini adalah pertanyaan yang sangat sulit untuk dijawab tanpa komputer fisik di depan Anda karena ada banyak hal yang dapat menyebabkan hal seperti itu. Tarik semua data yang Anda butuhkan dari hard drive itu, bersihkan (level rendah) dan instal ulang. Saya juga akan melakukan pemindaian virus pada data apa pun yang keluar dari hard drive itu. Anda tidak pernah bisa berhati-hati.

\n\n

Dan saya akan menambahkan ini hanya karena dari pengalaman saya orang tidak melakukannya. Backup backup backup!

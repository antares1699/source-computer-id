Saya menghapus file portdetect.log dengan menghapus dua folder di folder / Library / StartUpItems. Folder HWNetMgr dan HWPortDetect dikirim ke sampah, lalu saya memulai ulang Mac. Menghapus file log itu sendiri tidak akan berfungsi karena HWPortDetect akan menggantikannya secara otomatis.

Pertama-tama, Anda dapat mengabaikan kesalahan itu dengan aman. Perintah akan berjalan dengan sukses dan itu akan, dengan benar, mengabaikan all.java itu sendiri. Ini hanya memberi tahu Anda bahwa itu terjadi.

\n\n

Bagaimanapun, untuk menghindari kesalahan, Anda dapat menggunakan opsi tee dan find's exec: 

$ find . -name '*.java' -exec cat {} + | tee all.java\n

Dari man find: 

   -exec command ;\n          Execute  command;  true  if 0 status is returned.  All following\n          arguments to find are taken to be arguments to the command until\n          an  argument  consisting of `;' is encountered.  The string `{}'\n          is replaced by the current file name being processed  everywhere\n          it occurs in the arguments to the command, not just in arguments\n          where it is alone, as in some versions of find. \n\n   -exec command {} +\n          This  variant  of the -exec action runs the specified command on\n          the selected files, but the command line is built  by  appending\n          each  selected file name at the end; the total number of invoca‐\n          tions of the command will  be  much  less  than  the  number  of\n          matched  files. \n

Jadi, Anda dapat menggunakan -exec untuk memberi tahu find untuk menjalankan perintah pada setiap hasilnya. {} diganti dengan nama file / direktori sebenarnya yang ditemukan. + hanyalah penanda yang memberi tahu find bahwa perintah berakhir di sini. Saya menggunakannya daripada \\; karena ini akan menjalankan lebih sedikit perintah karena akan mencoba menggabungkannya menjadi sesedikit mungkin proses.

\n\n

Atau gunakan find ! untuk mengecualikan all.java: 

$ find . -name '*.java' ! -name all.java -exec cat {} +  >> all.java\n

Atau Globbing : 

$ shopt -s globstar\n$ cat **/*.java > all.java\n

Masalahnya adalah itu pertama file all.java dibuat, kemudian find menemukannya dan memasukkannya ke cat, yang tidak menyukainya karena dikeluarkan di sana.

\n\n

Solusi yang mungkin: 

$ find . -name '*.java' | xargs cat >> all.j\n

Atau: 

$ find . -name '*.java' | xargs cat >> ../all.java\n

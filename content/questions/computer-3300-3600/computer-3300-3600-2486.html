Ada banyak folder di direktori saya saat ini. Setiap folder memiliki file tar.gz. Untuk mengekstrak file tar.gz saya harus berada di dalam setiap folder dan menjalankan perintah berikut setiap saat. 

tar xvzf tar.gz -C /path/to/targetdirectory\n

Di dalam direktori saya saat ini, tampilannya seperti di bawah ini: 

   current directory\n            ├──Folder1\n                ├── A.tar.gz\n            ├──Folder2\n                ├── B.tar.gz\n            ├──Folder3\n                ├── C.tar.gz\n            ├──Folder4\n                ├── D.tar.gz\n            ├──Folder5\n                ├── E.tar.gz\n

Untuk mengekstrak semua sekaligus saya mencoba seperti ini 

tar xvzf */*.tar.gz -C /path/to/targetdirectory\n

Ini memberi saya kesalahan: 

tar: Folder1/A.tar.gz: Not found in archive\ntar: Folder2/B.tar.gz: Not found in archive\ntar: Folder3/C.tar.gz: Not found in archive\ntar: Folder4/D.tar.gz: Not found in archive\ntar: Folder5/E.tar.gz: Not found in archive\n
","source":"https://superuser.com/questions/1308374/how-to-extract-all-tar-gz-files-present-in-multiple-folders-at-a-time-to-another","author":"beginner","replies":[],"answers":[{"answer":"

Gunakan find dan jalankan perintah pada setiap file yang ditemukan, di direktori file itu: 

find . -name '*.tar.gz' -execdir tar -xzvf '{}' \\;\n

Opsi -execdir mengeksekusi tar dari dalam folder file yang ditemukan, dan {} akan diganti dengan nama tarfile.

\n\n

Lihat find dokumentasi untuk info lebih lanjut.

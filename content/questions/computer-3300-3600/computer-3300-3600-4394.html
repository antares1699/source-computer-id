Anda hanya perlu menghapus file pada gambar dan itu akan membuka kunci semua versi dengan pilihan 32 dan 64 bit.

\n\n
\n

Prasyarat

\n \n
    \n
  1. Windows 7 DVD atau File Gambar (iso)
    2. \n
  2. Alat Editor Gambar
    3. \n
  3. Alat Pembakar Gambar
    4. \n
  4. Pengetahuan Dasar Komputer
    5. \n
\n \n

catatan:

\n \n
    \n
  • Tidak masalah versi Windows 7 mana yang Anda miliki
    • \n
  • Tidak masalah platform mana yang Anda miliki, yaitu x64 atau x86
    • \n
  • Anda dapat menggunakan perangkat lunak editor gambar seperti MagicISO , PowerISO , UltraISO dll.
    • \n
  • Untuk Alat Penulisan Gambar, biasanya saya lebih suka Nero
    • \n
  • Anda harus tahu cara memasang, menyimpan, mengedit file gambar menggunakan alat di atas
    • \n
  • Buat salinan file Gambar atau DVD Anda jika Anda mengacaukan banyak hal
    • \n
\n \n

Instruksi untuk DVD

\n \n
    \n
  1. Buka perangkat lunak Image Editing dan pilih Drive DVD Anda
    2. \n
  2. Pilih opsi Save As dari menu File dan simpan ke dalam format .iso
    3. \n
  3. Jelajahi folder "sumber"
    4. \n
  4. Temukan file bernama "ei.cfg" dan hapus
    5. \n
  5. Simpan file Gambar
    6. \n
  6. Bakar Gambar menggunakan Nero atau perangkat lunak Penulisan Gambar lainnya
    7. \n
\n \n

Instruksi untuk File Gambar

\n \n
    \n
  1. Buka File Gambar dengan perangkat lunak Pengeditan Gambar
    2. \n
  2. Jelajahi folder "sumber"
    3. \n
  3. Temukan file bernama "ei.cfg" dan hapus
    4. \n
  4. Simpan file Gambar
    5. \n
  5. Bakar Gambar menggunakan Nero atau perangkat lunak Penulisan Gambar lainnya
    6. \n
\n \n

Lain kali Anda boot dengan DVD, Anda akan mendapatkan layar di mana Anda bisa\n pilih di antara berbagai versi jendela seperti Ultimate,\n Profesional, Home Basic, dll.

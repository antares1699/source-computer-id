Saya akan mencoba dan membedah setiap poin untuk mempermudah.

\n\n

Sehubungan dengan Ninite:

\n\n
\n

memasang versi terbaru dari\n perangkat lunak

\n
\n\n

Iya. Ninite akan menginstal rilis perangkat lunak terbaru (stabil).

\n\n
\n

mendukung penginstalan 64 bit di mana\n bisa jadi

\n
\n\n

Pastinya. Dari halaman Ninite - Ninite memeriksa bahasa PC Anda dan dukungan 64-bit\nuntuk menginstal versi terbaru dan terbaik dari setiap program.

\n\n
\n

menghapus iklan / bilah alat / hal serupa

\n
\n\n

Iya. Ninite menginstal aplikasi dengan pengaturan default dan berkata Tidak ke bilah alat dan sampah lainnya jika memungkinkan. Saya telah mengujinya dengan Java dan tidak ada toolbar yang diinstal. Mengutip situs Ninite - Ninite menginstal perangkat lunak dengan cepat dengan pengaturan default dan\nmengatakan "tidak" ke toolbar browser dan sampah lainnya.

\n\n
\n

menyediakan cara untuk menyimpan program\n diperbarui setelah instalasi

\n
\n\n

Sayangnya tidak ada. Perangkat lunak diinstal "sebagaimana adanya". Jika perangkat lunak (seperti Firefox) dilengkapi dengan pembaru, Anda dapat menggunakannya, tetapi Ninite sebenarnya bukan pengelola paket dalam arti yang sebenarnya, lebih merupakan alat penyebaran perangkat lunak yang lebih mudah. Anda mungkin tertarik dengan Perbarui Pemberitahu untuk tujuan ini.

\n\n
\n

jika saya dapat menambahkan penginstal khusus ke file\n perangkat lunak, itu nilai tambah yang besar.

\n
\n\n

Tak satu pun dari solusi di utas lain yang dapat melakukan ini sejauh yang saya tahu. Jika Anda ingin menginstal perangkat lunak lain yang tidak terdaftar di Ninite secara otomatis pada penginstalan bersih Windows, sebagian besar paket penginstal menyertakan sakelar baris perintah dapat Anda gunakan untuk menginstal perangkat lunak secara diam-diam.

\n\n

Untuk informasi hebat tentang menyebarkan perangkat lunak pada penginstalan bersih, AppDeploy.com adalah Bagus sumber.

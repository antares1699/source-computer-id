Kesalahan keras terjadi ketika informasi yang diminta (seperti mencoba membuka program untuk pertama kali) tidak disimpan dalam memori. Karena tidak ada dalam memori, komputer harus memeriksa hard drive Anda untuk menemukan informasinya. Di tangkapan layar Anda, Anda memiliki sekitar 8 gigabyte yang tersedia, dan 5 di antaranya benar-benar gratis.

\n\n

Sebelum Anda memutakhirkan memori Anda, Anda mungkin mengalami kesalahan keras karena tidak ada lagi memori bebas, jadi "Windows terpaksa mendorong ke dalam file halaman". Anda sekarang mengalami kesalahan besar karena Anda membuka program untuk pertama kalinya. Ini sebenarnya normal.

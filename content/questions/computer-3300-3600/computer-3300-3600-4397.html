Saya memiliki jaringan rumah kecil dengan 3 mesin - mesin Windows 2000, mesin XP, dan mesin Vista. Saya menggunakan Remote Desktop untuk menghubungkan keduanya, yang berfungsi dengan baik saat menghubungkan ke mesin XP dan Vista (bahkan dari mesin Win2K). Namun saya belum bisa menemukan cara menghubungkan untuk mesin Win2K. Apakah ini mungkin dengan Windows 2000 Pro?

\n\n

Saya tahu itu mungkin dengan Win2K Server (menggunakan Terminal Services), dan saya tahu ada utilitas non-Microsoft di luar sana (misalnya VNC) untuk mencapai ini (yang saya tidak menentang untuk digunakan jika saya harus). Saya hanya ingin mencari tahu dulu apakah ini hanya masalah konfigurasi, atau hanya beberapa komponen yang perlu saya instal ... sesuatu seperti itu.

Tulis JNK Itulah yang ada di tautan MS di jawaban saya.

\n\n

Sayangnya, instruksi dalam tautan MS tersebut membutuhkan penggunaan W2K server CD, tapi pertanyaannya adalah tentang RDP ke sistem W2K-Pro.

\n\n

Sana adalah tidak ada komponen Windows yang tersedia untuk ditambahkan ke W2K Pro sistem yang akan memungkinkannya menyediakan layanan RDP yang dapat disambungkan oleh klien jarak jauh.

Saya memiliki file photoshop yang merupakan sekumpulan teks tetapi disimpan sebagai gambar (jadi saya tidak dapat menggunakan alat teks apa pun untuk mengubahnya).

\n\n

Teksnya pada dasarnya hitam dan saya ingin mengubahnya menjadi ungu.

\n\n

Saya mencoba menggunakan alat ganti warna tetapi Anda hanya dapat memindahkan bilah geser di sekitar warna yang ada jadi karena warnanya hitam hanya berubah menjadi abu-abu jika saya menggesernya.

\n\n

Bagaimana cara mengubah gambar hitam menjadi ungu di adobe photoshop 7?

Saya rasa tidak ada opsi untuk itu, karena Photoshop-7 tidak akan tahu teksnya. Apa yang dapat Anda coba adalah menggunakan "Magic Wand Tool" dan memilih huruf (saya harap ini hanya satu atau dua baris ...).

\n\n

Setelah memilih huruf, coba ubah warnanya.

\n\n

Jika Anda membutuhkan bantuan untuk ini, tanyakan saja.

\n\n

JUGA: Anda mungkin dapat menggunakan perangkat lunak OCR untuk mengenali teks, lalu menyalinnya ke MS Word, lalu mengubah warnanya lalu mengembalikannya ke Photoshop, tetapi jika latar belakangnya selain putih sederhana, Anda mungkin memilikinya masalah saat memasangnya kembali.

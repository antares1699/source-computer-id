Saat mengakses situs web tertentu yang memiliki video YouTube yang disematkan, saya perhatikan bahwa kadang-kadang ketika halaman web memuat, video YouTube akan diputar secara otomatis dan melihat kode sumber, tampaknya hal ini ditentukan oleh pengubah ke URL video yang disematkan.

\n\n

Pertanyaan saya adalah apakah ada semacam pengaturan (di browser atau YouTube itu sendiri) atau mungkin ekstensi Chrome yang memungkinkan Anda menonaktifkan putar otomatis dari situs web apa pun terlepas dari apakah URL telah disiapkan untuk putar otomatis?

\n\n

Tidak hanya untuk video Youtube, tetapi untuk semua video:

\n\n

Di firefox, untuk menghentikan putar otomatis, buka halaman about:config dan setel media.autoplay.enabled ke false

\n\n

Di Chrome, untuk menghentikan putar otomatis, buka halaman chrome://flags/#autoplay-policy dan pilih Document user activation is required

\n\n

Setelah berubah, buka kembali browser Anda.

Ini dia pengguna naskah Saya menulis bahwa dapat melakukan ini: 

// ==UserScript==\n// @name         Stop Youtube Embedded Video Autoplay\n// @namespace    none\n// @version      0.1\n// @description  Stop autoplaying YouTube Embedded Videos.\n// @author       none\n// @match        *://*/*\n// @grant        none\n// ==/UserScript==\n\n(function() {\n'use strict';\n\nvar vid = document.getElementsByTagName(\"iframe\");\nfor (var i = 0, l = vid.length; i < l; i++) {\n    var vidno = vid[i];\n    var reg = new RegExp(/.*youtube.*autoplay=1.*/i);\n    if (reg.test(vidno.src)){\n        vidno.src=vidno.src.replace(\"autoplay=1\", \"autoplay=0\");\n    }\n}\n\n\n})();\n

Anda hanya perlu membuat UserScript baru di Greasemonkey, Tampermonkey, dll. Ini akan berjalan di setiap halaman, mencari video YouTube yang disematkan dan menghentikan pemutaran otomatis.

\n\n

Saat ini saya memiliki dua folder sangat penting yang diisi dengan banyak file yang membuatnya sangat sulit untuk melacak file.

\n\n

Saya memiliki folder dokumen yang berisi lebih dari 5.000 dokumen. Saya menggunakan ini untuk menyimpan pekerjaan sekolah, dokumen terkait pekerjaan, dokumen pribadi, dan bahkan beberapa lusin eBook yang saya dapatkan dari dropbox. \n\n

Jadi saya tidak tahu harus mulai dari mana mengatur semua ini. Saya pikir mengaturnya secara kronologis atau abjad mungkin membantu, tetapi biasanya beberapa hal disalin sekaligus sehingga tanggal modifikasi berubah dan saya tersesat atau namanya mengandung karakter non-alfabet seperti "_" atau "@" dan angka juga.

\n\n

Jika Anda mengatur file dengan cara tertentu, saya ingin tahu.

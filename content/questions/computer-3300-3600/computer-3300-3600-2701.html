curl https://sdk.cloud.google.com | bash\n

\n\n

Mulai ulang shell atau Terminal Anda.

\n\n

Saya baru saja mengalami masalah yang sama (dan bahkan berakhir di sini dari pencarian google juga). Tetapi bagi saya masalah itu diselesaikan dengan memisahkan perintah yang diberikan google sini di Langkah 4: 

sudo apt-get update && sudo apt-get install google-cloud-sdk\n

menjadi perintah terpisah: 

sudo apt-get update\nsudo apt-get install google-cloud-sdk\n

Karena ada beberapa paket yang tidak terkait dengan Google SDK yang gagal diunduh selama perintah apt-get update, satu-satunya tebakan yang saya miliki adalah menghentikan perintah di sisi lain dari logika && agar tidak dijalankan.

\n\n

Ketika saya menjalankan perintah kedua dengan sendirinya, paket google-cloud-sdk diinstal sebagai paket baru, jadi saya menganggap perintah gabungan dari sebelumnya gagal.

\n\n

Setelah ini instruksi pada Langkah 5 (untuk mengeksekusi gcloud auth login) berhasil.

\n\n

Semoga ini membantu orang lain di Ubuntu / Debian yang solusi lainnya tidak berfungsi.

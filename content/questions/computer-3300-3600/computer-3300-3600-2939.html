Saya baru-baru ini menjalankan skrip yang mengkonfigurasi VIM dan menginstal plugin yang diperlukan untuk pengembangan. Saya menginstalnya seperti ini: 

$ git clone https://github.com/sontek/dotfiles.git\n$ cd dotfiles\n$ ./install.sh vim\n

Saya tidak memahami plugin apa pun dan sekarang saya ingin menghapus semua yang saya lakukan.

\n\n

Saya pada dasarnya ingin menginstal VIM baru sehingga saya dapat mengkonfigurasinya sendiri dan memahami semuanya.

\n\n

Bagaimana saya harus melakukan ini? Saya menggunakan Linux Mint LXDE.

Coba jalankan perintah berikut di shell:

\n\n

sudo apt-get purge vim && sudo apt-get install vim

\n\n

Ini sepenuhnya menghapus vim, termasuk semua file konfigurasi, dan menginstalnya lagi.

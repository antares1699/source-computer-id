Saya baru saja selesai membangun komputer baru dengan salah satu CPU Haswell model baru; khususnya, file Intel Core i5-4570 .

\n\n

Saya mencolokkan daya, menyalakannya, dan melihat semuanya mulai berputar segera ... kecuali kipas pendingin CPU stok. Sebaliknya, dalam 5 detik pertama atau lebih kipas pendingin CPU hanya bergerak beberapa kali, setelah itu mulai dengan keras. desir selama 2 atau 3 detik, sebelum berputar ke kecepatan dan ketenangan normal.

\n\n

Menjadi pembuat PC pertama kali, tentu saja saya khawatir ada yang tidak beres dengan pendingin CPU saya, terutama melihat bagaimana bilah kipasnya. berkedut dalam beberapa detik pertama. Tapi ternyata, ini normal, dan sesuatu yang banyak motherboard (termasuk saya ASRock H87M Pro4 ) lakukan secara default sebagai bagian dari prosedur kontrol kipas mereka, jadi saya akan berasumsi bahwa itu aman. Aku tidak akan melakukan overclocking.

\n\n

Tapi apa itu tujuan perilaku ini? Mengapa pendingin CPU tidak berputar secara normal seperti semua kipas lainnya (casing, GPU dan PSU)? Saya sebenarnya menyadap perwakilan Intel kami. tentang perilaku serupa (kecepatan penuh, lalu melambat) di masa lalu. :)

\n\n

Pada dasarnya sampai SMBus bekerja, MB tidak memiliki cara untuk mengetahui suhu dan karenanya memutarnya dengan kecepatan penuh untuk memastikannya mendingin sebanyak mungkin. Dengan cara itu jika BIOS / POST / SMBus (dll.) Gagal karena suatu alasan maka kipas akan dibiarkan penuh sehingga, jika tidak ada yang lain, sistem akan melakukan yang terbaik untuk memperbaiki kemungkinan masalah overheat yang mungkin menyebabkan POST. / Masalah SMBus di tempat pertama.

\n\n

Pemahaman saya tentang mengapa mereka menunggu / gagap sedikit pada awalnya (yang normal dari pengalaman saya dengan beberapa / beberapa generasi mesin terakhir - tetapi bukan 5 detik "nyata" seperti yang Anda sarankan, lebih seperti 1 atau 2 mungkin) adalah karena itu mendapatkan daya awal, tetapi itu sengaja dihentikan / dibatasi untuk bit pertama untuk memastikan bahwa sisa sistem / MB mendapat arus yang tepat untuk memberi energi penuh (kipas menyedot banyak jus saat berputar dan dapat mencegah bagian lain dari mendapatkan mereka daya yang dibutuhkan selama penyalaan awal). Setelah itu terjadi (atau beberapa waktu berlalu - saya tidak yakin bagaimana sebenarnya diperiksa / diputuskan) maka arus listrik penuh yang dibutuhkan oleh kipas disuplai dan kipas berputar.

Saya memiliki Dell Latitude D620 yang baru saja saya instal ulang Windows 7. Wisaya BitLocker melaporkan kesalahan berikut saat saya membukanya.

\n\n
\n

Perangkat Keamanan Trusted Platform Module (TPM) yang kompatibel harus ada di komputer ini, tetapi TPM tidak ditemukan ...

\n
\n\n

Saat saya membuka snap-in Administrasi TPM, ia melaporkan:

\n\n
\n

Modul Platform Tepercaya yang Kompatibel (TPM) tidak dapat ditemukan di komputer ini. Pastikan komputer ini memiliki 1.2 TPM dan sudah dihidupkan di BIOS.

\n
\n\n

Sejauh ini, saya telah melakukan yang berikut:

\n\n
    \n

  • TPM terverifikasi diubah DI di BIOS

    • \n

  • TPM terverifikasi sebelumnya AKTIF di BIOS

    • \n

  • Driver TPM Terverifikasi telah diinstal dan muncul di Windows Device Manager dan dilaporkan berfungsi

    • \n

  • Diverifikasi tidak ada perangkat tambahan di Windows Device Manager yang tidak memiliki driver, atau tampak tidak berfungsi

    • \n

  • Menghapus instalan driver TPM (driver default Windows 7) dan dipilih Hapus perangkat lunak driver untuk perangkat ini dan menginstal ulang driver menggunakan yang dari Dell.com

    • \n

  • Menggunakan BERSIH Opsi pengaturan TPM di BIOS

    • \n

  • Berbalik MATI TPM di BIOS, melakukan boot ulang, dan memutarnya DI lagi

    • \n
\n\n

Sesuatu yang aneh yang saya perhatikan adalah perangkat TPM tidak muncul di bawah Perangkat keamanan di Windows Device Manager, ini muncul di bawah Perangkat sistem . Saya ingat itu di bawah Perangkat keamanan di masa lalu, dan beberapa hasil Google mengonfirmasi bahwa itu akan muncul di sana. Apakah ada hal lain yang bisa saya coba agar ini berfungsi? BitLocker adalah persyaratan perusahaan, jadi saya benar-benar perlu membuatnya berfungsi.

\n\n

Catatan: Sebelum menginstal ulang Windows 7, saya mengaktifkan BitLocker pada PC yang sama ini tanpa masalah. Jadi, saya tahu TPM kompatibel.

Cara menonaktifkan Trusted Platform Module (TPM) di BitLocker:\nBuka Editor Kebijakan Grup:

\n\n

Jika Editor Kebijakan Grup tampaknya tidak tersedia, ikuti instruksi untuk mengaktifkan BitLocker terlebih dahulu.

\n\n

Buka Kebijakan Komputer Lokal> Konfigurasi Komputer> Template Administratif> Komponen Windows> Enkripsi Drive BitLocker> Drive Sistem Operasi.

\n\n

Edit Memerlukan otentikasi tambahan pada kebijakan permulaan.

\n\n

Setel kebijakan ke Diaktifkan dan pastikan Izinkan BitLocker tanpa TPM yang kompatibel dicentang.\nBitLocker di Windows 7 / Vista tidak mendukung sandi untuk drive sistem jika TPM tidak tersedia. Jika Anda tidak suka menggunakan kunci Startup USB atau mengingat Kunci Pemulihan BitLocker (yang dapat digunakan seperti kata sandi), tingkatkan ke Windows 8 atau yang lebih baru yang mendukung kata sandi untuk drive sistem meskipun TPM tidak tersedia.

\n\n

Saya baru saja membeli laptop baru dan ingin memainkan beberapa game di dalamnya. Saya perhatikan bahwa saya tidak dapat menekan lebih dari 2 tombol sekaligus tanpa 2 tombol pertama menghalangi input tombol ke-3. Ini membuat bermain game FPS sulit karena mengharuskan saya melepaskan satu kunci sebelum pemain merespons kunci lainnya.

\n\n

Saya tidak tahu apakah ini masalah perangkat keras atau masalah driver karena saya dapat menggunakan kombinasi tombol yang berisi 3 tombol. Misalnya Ctrl-Shift-F akan bekerja dengan benar pada aplikasi yang menggunakan 3 kombinasi tombol.

\n\n

Adakah yang punya saran tentang cara memperbaikinya atau apa yang mungkin menyebabkan ini?

Ini terjadi karena cara penekanan tombol dikirim ke PC. Semua kunci terhubung melalui "matriks" sirkuit. Beberapa kombinasi berfungsi (biasanya, kontrol dan shift serta alt dan super berfungsi dengan apa saja), tetapi ini adalah masalah perangkat keras. Anda mungkin dapat mengatasinya dengan keyboard eksternal, meskipun saya belum pernah mencoba menggunakannya di laptop.

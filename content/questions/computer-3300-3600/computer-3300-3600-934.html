Ini akan menghemat banyak waktu saya, mengunduh biner, meletakkannya di berbagi file atau cloud dan kemudian:

\n

Linux: 

aws s3 cp 's3://yourAwsBucket/postgresql-9.6.21-1-linux-x64-binaries.tar.gz' .\ntar xvzf postgresql-9.6.21-1-linux-x64-binaries.tar.gz\ncd pgsql/bin\npsql -c 'Select version();' "postgresql://<user>:<password>@<dbendpoint>/<dbName>"\n

atau jika Anda mendapatkan file tidak ada file atau direktori seperti itu 'psql' menggunakan subproses, coba letakkan pesta sebagai argumen pertama: 

print(subprocess.run(['bash','psql','-c','Select version();','postgresql://<user>:<password>@<dbendpoint>/<dbName>'],shell=isWindows, stdout=subprocess.PIPE, stderr=subprocess.PIPE).stdout.decode('ascii'))\n

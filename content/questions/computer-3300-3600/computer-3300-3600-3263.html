Saya mencari cara untuk mendapatkan alamat IP saya menggunakan command prompt di Linux. Saya tahu ketika Anda mengetik "ifconfig" Anda bisa mendapatkan alamat IP lokal Anda (yaitu 192.168.0.103), tetapi saya mencari alamat IP saya yang saya dapatkan dari ISP saya. Bagaimana saya bisa mendapatkan ini dari Linux tanpa harus mengunjungi beberapa situs web?

","source":"https://superuser.com/questions/127224/finding-ip-address-using-linux-command-prompt","author":"John Kube","replies":[],"answers":[{"answer":"

Jika mesin Anda diberi alamat IP lokal, mesin mungkin tidak mengetahui alamat publik Anda. Anda dapat meminta server yang tidak ada di jaringan lokal Anda untuk menemukan alamat publik Anda. 

$ curl whatismyip.org\n

Jika komputer Anda menyala, mis. LAN perusahaan dengan firewall yang menerjemahkan alamat, sayangnya tidak ada informasi di komputer Anda tentang alamat yang terlihat secara eksternal. Untuk semua komputer Anda tahu itu memiliki IP lokal (misalnya 192.168.0.103) yang dapat digunakan untuk terhubung ke semua host di Internet. Firewall penerjemah alamat sepenuhnya transparan dari sudut pandang komputer Anda.

\n\n

Satu-satunya cara adalah menggunakan "pengamat" eksternal seperti whatismyip.org untuk membantu. Jika Anda tidak suka harus mengaksesnya melalui browser web lengkap, curl adalah perubahan seperti yang ditunjukkan oleh Samuel K.

Tambahkan ini ke file ~ / .bashrc Anda di Linux atau ~ / .bash_profile di Mac. 

alias myip='wget http://automation.whatismyip.com/n09230945.asp -O - -q ; echo'\n

Lalu lari 

source ~/.bashrc\n

atau: 

source ~/.bash_profile\n

Sumber akan mengaktifkan perubahan yang Anda buat pada file bash Anda. Ini hanya diperlukan sekali.

\n\n

Sekarang ketik 'myip' dari baris perintah, dan Anda mendapatkan alamat IP publik. 

:# myip\n23.23.133.111\n

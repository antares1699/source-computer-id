Anda dapat menerapkan gaya ke semua situs web dengan menggunakan namespace XHTML: 

@namespace url(http://www.w3.org/1999/xhtml);\n\nYour CSS here.\n

Alternatifnya, untuk hanya memodifikasi UI Firefox, Anda akan menggunakan namespace XUL: 

@namespace url(http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul);\n

Jadi jika kamu tidak menyertakan namespace, Stylish akan mengubah semua elemen di Firefox.

Saya menemukan aturan namespace XHTML @namespace url(http://www.w3.org/1999/xhtml); untuk diterapkan ke semua hal, situs web dan Firefox UI (chrome).

\n\n

Untuk menata semua situs web, saya menggunakan aturan @document berikut sebagai gantinya (diuji dengan FF28.0 dan Stylish 1.4.3): 

@-moz-document url-prefix(http://), url-prefix(https://) {\n    /* Make all Links UPPERCASE */\n    a {\n        text-transform: uppercase !important;\n    }\n}\n

Memaksakan ke dalam folder bersarang tidak dimungkinkan dengan UI default.

\n\n

Satu-satunya pilihan adalah memindahkan folder setelah Anda membuat cadangan. Semua informasi ada di folder itu sendiri.

\n\n

Faktanya, ini adalah rekomendasi Microsoft untuk membuat cadangan untuk berbagi file jaringan. Cadangan hanya akan memungkinkan ada satu gambar Cadangan per berbagi file. Mengganti nama folder setelah fakta memungkinkan Anda menyimpan beberapa versi gambar per berbagi file. Ada entri blog tentang membuat skrip ini menggunakan wbadmin, yang merupakan baris perintah Cadangan di blog Penyimpanan Windows beberapa waktu lalu. Ini juga yang saya lakukan dengan mesin saya.

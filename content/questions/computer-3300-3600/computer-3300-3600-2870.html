Apakah ada daftar master perintah Pengenalan Ucapan untuk Windows 8?

\n\n

Khususnya untuk menggunakan komponen yang baru diperkenalkan seperti "Charms Bar" dll.

Ada daftar perintah umum untuk pengenalan suara di Windows 8 (sedikit perubahan dibandingkan dengan versi Windows yang lebih lama). Dari halaman Dukungan Microsoft Windows: 

To do this                              Say this\n------------------------------------------------------------------------------\nSelect any item by its name             Click File; Start; View\nSelect any item or icon                 Click Recycle Bin; Click Computer;\n                                        Click file name\nDouble tap or double-click any item     Double-click Recycle Bin;\n                                        Double-click Computer;\n                                        Double-click file name\nSwitch to an open app                   Switch to Paint; Switch to WordPad;\n                                        Switch to program name;\n                                        Switch application\nScroll in one direction                 Scroll up; Scroll down;\n                                        Scroll left; Scroll right\nInsert new paragraph/line in a document New paragraph; New line\nSelect a word in a document             Select word\nSelect a word and start to correct it   Correct word\nSelect and delete specific words        Delete word\nShow a list of applicable commands      What can I say?\nUpdate speech commands available        Refresh speech commands\nTurn on listening mode                  Start listening\nTurn off listening mode                 Stop listening\nMove Speech Recognition microphone bar  Move speech recognition\nMinimize the microphone bar             Minimize speech recognition\nView Help/Support about specific tasks  How do I do something?\n                                        For example,\n                                        say “How do I install a printer?”\n

Secara khusus, perintah yang tersedia tergantung pada konteksnya (aplikasi sedang aktif), jadi ingatlah bahwa Anda dapat bertanya "Apa yang bisa kukatakan?" kapan saja untuk mendapatkan perintah yang tersedia.

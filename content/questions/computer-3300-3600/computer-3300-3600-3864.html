Saya memiliki tugas terjadwal di server Windows 2008 R2 saya yang seharusnya menjalankan skrip PowerShell untuk mengunduh cadangan file dari server lain. Di server yang sama ini, saya telah menjadwalkan tugas yang sama persis untuk dijalankan dan diunduh dari 2 server lain - tugas ini berfungsi dengan baik.

\n\n

Tugas yang satu ini secara khusus sepertinya tidak pernah melakukan apapun. Riwayat tugas mengatakan itu berhasil dijalankan tetapi file tidak pernah diunduh. Jika saya menjalankan skrip PowerShell secara manual, file akan diunduh. Jika saya menjalankan tugas otomatis secara manual, file akan diunduh lagi. Jika saya memutuskan sambungan dan datang untuk memeriksa keesokan harinya, file tersebut belum diunduh.

\n\n

Saya telah mengaturnya untuk menjalankan apakah pengguna masuk atau tidak. Mengapa tidak berhasil ?? Bagaimana saya dapat mengaktifkan pencatatan yang lebih rinci?

Berikut ini adalah perintah pengeditan izin tujuan umum untuk file yang saat ini sedang diedit. Untuk penjelasan lebih detail tentang plugin dan mengedit menu Sublime Text 2, lihat posting ini .

\n\n

Ini akan menambahkan file Ubah Mode perintah di Sunting Tidak bisa. Saat dipilih, pengguna akan diminta untuk memasukkan string argumen yang valid ke chmod (misalnya u+rwx atau 755; defaultnya adalah string izin oktal 4 digit yang saat ini disetel seperti 0644), yang kemudian diterapkan ke file yang sedang diedit.

\n\n

\n\n

Pilih Alat Â»Plugin Baru , masukkan konten berikut dan simpan sebagai chmod.py di ~/Application Support/Sublime Text 2/Packages/User/: 

import sublime, sublime_plugin, subprocess\n\ndef chmod(v, e, permissions):\n    subprocess.call( [ \"chmod\", permissions, v.file_name() ] )\n\ndef stat(filename):\n    proc = subprocess.Popen( [ \"stat\", \"-f\", '%Mp%Lp', filename ], stdout=subprocess.PIPE )\n    return str(proc.communicate()[0]).strip()\n\nclass ChangeModeCommand(sublime_plugin.TextCommand):\n    def run(self, edit):\n        if sublime.platform() != 'osx':\n            return\n\n        fname = self.view.file_name()\n\n        if fname == None:\n            sublime.message_dialog(\"You need to save this buffer first!\")\n            return\n\n        perms = stat(fname)\n\n        def done(permissions):\n            chmod(self.view, edit, permissions)\n\n        sublime.active_window().show_input_panel(\n            \"permissions to apply to the file \" + fname + \": \", perms, done, None, None)\n

Untuk menyisipkan item menu untuk perintah ini, tambahkan berikut ini ke ~/Application Support/Sublime Text 2/Packages/User/Main.sublime-menu, gabungkan dengan konten file yang ada jika file sudah ada: 

[\n    {\n        \"id\": \"edit\",\n        \"children\":\n        [\n            {\"id\": \"wrap\"},\n            { \"command\": \"change_mode\" }\n        ]\n    }\n]\n

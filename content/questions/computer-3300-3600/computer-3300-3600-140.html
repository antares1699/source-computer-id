Beberapa server email - seperti MS Exchange - memiliki fitur "penarikan kembali pesan" - jika serangkaian kondisi terpenuhi, pesan yang sudah terkirim dapat dipanggil kembali seolah-olah tidak pernah terkirim.

Tidak ada cara universal untuk mengingat pesan email yang terkirim. Seperti yang Anda nyatakan, fitur seperti ini khusus untuk menggunakan server dan klien email tertentu, misalnya Exchange / Outlook. Saya rasa Lotus Domino dan Notes juga memiliki fitur serupa.

Seperti yang Anda sebutkan, server Exchange memiliki beberapa fungsi dalam hal ini. Anda dapat mengaktifkan fitur tersebut untuk digunakan dengan Outlook. Ini bukan jaminan 100% pesan Anda akan ditarik kembali. Situs Microsoft ini menjelaskan lebih rinci tentang ini:

\n

Berikut\n ada empat skenario yang menjelaskan apa\n terjadi dalam berbagai situasi, dan\n skenario tambahan yang mencakup\n mengingat pesan yang dikirim ke a\n Microsoft Exchange Server publik\n map.

\n\n

Taruhan terbaik Anda adalah menggunakan tindakan pencegahan untuk menjaga pesan dari kotak keluar Anda dengan menggunakan pengiriman tertunda. Itu secara pribadi telah menyelamatkan saya berkali-kali.

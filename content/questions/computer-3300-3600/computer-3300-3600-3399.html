PowerShell dapat mencetak file apa pun, menggunakan tindakan yang sama seperti kata kerja Print\nditentukan untuk tipe file.

\n

Perintah PowerShell untuk digunakan adalah Mulai-Proses ,\ndan idenya hanyalah memanggil file itu sendiri dengan kata kerja Print.

\n

Sintaksnya sangat sederhana: 

Start-Process "path-to-file" -Verb print\n

Saya mengujinya dengan file PDF, dan file tersebut dicetak dengan benar.

\n

Contoh yang lebih rumit dapat ditemukan di dokumentasi:

\n
\n

Contoh 2: Cetak file teks

\n

Contoh ini memulai proses yang mencetak file C: \\ PS-Test \\ MyFile.txt. 

Start-Process -FilePath "myfile.txt" -WorkingDirectory "C:\\PS-Test" -Verb Print\n

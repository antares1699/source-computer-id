\n\n

Tetapi ketika saya akan menginstal VirtualBox saya melihatnya hanya mendukung x86 / AMD64 .

\n\n

\n\n

Baik laptop dan komputer baru saya memiliki prosesor Intel 64 bit.

\n\n

Agak membingungkan karena halaman Utama untuk VirtualBox mengatakan itu mendukung AMD64 / Intel64. Tetapi unduhan menghilangkan Intel64.

\n\n

Apakah saya kurang beruntung? Atau adakah cara untuk membuatnya berhasil? (Atau apakah itu akan berfungsi meskipun dikatakan hanya untuk amd64?)

AMD64 adalah x86_64 standar.

\n\n

AMD melisensikan file 64-bit x86 implementasi ke Intel setelah Intel tidak kompatibel ke belakang IA64 implementasi adalah a kegagalan pasar .

\n\n

Semua Core 2 Duo dan yang lebih baru (semua i3 , i5 , i7 ) (pengecualian: awal Atom ) CPU Intel mendukung implementasi AMD64 x86_64 AMD.

\n\n

Jadi pilih saja x86 / amd64 (asalkan Anda menjalankan OS 64-bit)

\n\n

Catatan: Seringkali perangkat lunak merujuk sebagai kompatibel dengan x86 mengacu pada x86 32-bit. x86_64 CPU dapat menjalankan OS 32-bit atau 64-bit.

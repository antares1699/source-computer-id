Saya sering melakukan ssh ke server yang memiliki shell Bash. Sebelumnya semua pintasan normal berfungsi yaitu up akan menjadi perintah terakhir, ctrl+r akan memungkinkan saya untuk mencari perintah sebelumnya.

\n\n

Namun saya masuk hari ini dan ini tidak lagi berfungsi mis. up sekarang menampilkan ^[A dan ctrl+r menampilkan ^R. Bagaimana cara mengembalikannya ke keadaan semula?

Saya akan memposting komentar saya sebagai jawaban. Sepertinya shell jarak jauh yang Anda gunakan bukanlah bash. Anda dapat memeriksa dengan menjalankan ps -p $$. Jika itu tidak menghasilkan bash Anda menjalankan shell lain.

\n\n

Untuk mengatur shell default Anda menjadi bash lagi, Anda dapat menjalankan perintah ini: 

chsh USER\n

Di mana USER adalah nama pengguna Anda di server jarak jauh.

Bagaimana cara menggunakan perangkat lunak BIND untuk mengalihkan Nama Domain dari DNS atau file zona?

","source":"https://superuser.com/questions/235779/using-bind-to-perform-dns-redirection","author":"Jean","replies":[],"answers":[{"answer":"

Jenis "pengalihan" di DNS hanya CNAME, penerusan, dan pengalihan NS. Melakukan pengalihan URL bukanlah pertanyaan DNS, dalam hal ini Anda mengarahkan DNS ke server web yang dapat menyediakan pengalihan URL.

\n\n

CNAME pada dasarnya adalah alias dari satu Nama Domain Berkualifikasi Penuh (FQDN) ke yang lain. Resolusi CNAME dilakukan di klien.

\n\n

Contoh CNAME: 

 ; zone file\n example.com IN CNAME some-other-host.example.com.\n

Penerusan DNS akan melakukannya secara transparan permintaan proxy ke server DNS lain. Resolusi dilakukan di server.

\n\n

Contoh seorang forwarder: 

// named.conf\nzone \"example.com\" {\n    type forward;\n    forwarders { 127.0.0.127; };\n};\n

Pengalihan server nama adalah persilangan antara melakukan CNAME dan penerusan DNS. Anda memberikan data server nama (NS) yang selanjutnya akan ditanyakan oleh klien yang meminta data tersebut. 

; zone file\nexample.com IN NS 127.0.0.127;\n

Saya menggunakan PuTTy ke SSH ke server, tetapi saya membutuhkan jendela terminal yang terbuka menjadi kecil, hanya ukuran x dengan x tertentu, apakah ada cara untuk melakukan ini dari baris perintah?

PuTTY Parameter baris perintah standar tidak menyebutkan opsi apa pun untuk menentukan ukuran jendela atau ukuran font.

\n\n

Namun, Anda dapat menentukan profil untuk dimuat, dan itu dapat digunakan untuk mencapai apa yang Anda cari.\nProfil pada dasarnya adalah kumpulan pengaturan, termasuk properti jendela, nama server jarak jauh, kunci otentikasi, dan banyak lagi. Beberapa pengaturan tersebut, seperti nama server, dapat dikosongkan jika diinginkan.

\n\n

Buat profil menggunakan GUI. Untuk mengubah ukuran jendela, buka layar opsi Jendela. (Itu adalah atas layar "Penampilan". Kata "Jendela" adalah kategori yang berada di atas layar lain dalam hierarki. Tapi, ini juga merupakan layar opsi.) Kemudian, setelah menentukan ukuran yang Anda inginkan, buka layar "Sesi" dan ketik nama sesi, dan klik "Simpan". Itu menciptakan profil.

\n\n

Kemudian, gunakan:

PuTTY -load "sessionName"
\n\n

Sesuaikan bagian setelah -load dan di antara tanda kutip. Tentukan jalur lengkap ke PuTTY sesuai kebutuhan.

\n\n

Beberapa dokumentasi resmi lainnya untuk Anda: Putty: Menentukan sesi untuk memulai .

Saya tidak berpikir PuTTY mendukung ini di baris perintah. Namun, Anda dapat mengatur profil dengan properti "Kolom" & "Baris" yang disetel pada panel pengaturan "Jendela". Anda kemudian dapat memuat profil menggunakan: 

putty -load \"Profile Name\"\n

Seperti menu lainnya, Anda perlu memuat profil untuk melakukannya.\nJika Anda masih ingin masuk secara otomatis, masukkan nama pengguna + kata sandi setelah parameter -load seperti ini (ini adalah tampilan pintasan desktop windows saya):

\n\n
\n

C: \\ apps \\ putty.exe -ssh -P 2222 -muat "NAMA PROFIL" USERNAME@192.168.0.10 -pw PASSWORD

\n
\n\n

Jika -load adalah parameter terakhir, memasukkan nama pengguna dan kata sandi tidak akan berhasil.\n-P hanya untuk server yang tidak menjalankan port default 22.

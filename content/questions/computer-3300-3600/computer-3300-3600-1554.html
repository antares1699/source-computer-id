Buka gpedit.msc dari salah satu Lari prompt atau dari Cari kotak.

\n\n

Dibawah Kebijakan Komputer Lokal , pergi ke Konfigurasi Komputer -> Template Administratif -> Sistem -> Logon

\n\n

\n\n

Pilih entri yang berbunyi: Sembunyikan titik masuk untuk Fast User Switching

\n\n

\n\n

Anda perlu menonaktifkan sesuatu yang dikenal sebagai Fast User Switching. Saya berani bersumpah bahwa ini mungkin dilakukan di GUI, tetapi saya tidak dapat menemukan opsi ini di mana pun.

\n\n

Itu masih dapat dilakukan dari editor Kebijakan Grup di edisi Professional/Enterprise/Ultimate (tidak yakin tentang yang lain).

\n\n

Anda dapat melakukannya dengan:

\n\n
    \n

  1. tekan Bendera Windows + R .

    2. \n

  2. Ketik gpedit.msc.

    3. \n

  3. Navigasikan ke Local Computer Policy> Computer Configuration> Administrative Templates> System> Logon.

    4. \n

  4. Kemudian buka Set Hide entry points for Fast User Switching.

    5. \n

  5. Setel ini ke Diaktifkan.

    6. \n

  6. Nyalakan ulang mesin Anda.

    7. \n
\n\n

Dan Anda selesai!

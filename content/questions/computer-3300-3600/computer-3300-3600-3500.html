Variabel shell tidak terlihat oleh proses anak kecuali mereka diekspor jadi Anda harus mengatakannya 

export GLOBAL_VAR\n

suatu tempat sebelum proses anak diluncurkan; dalam cangkang modern, ekspor dapat digabungkan dengan penugasan 

export GLOBAL_VAR=\"some global value\"\n

" Variabel shell tidak terlihat oleh proses anak kecuali mereka diekspor "- dan meskipun demikian mereka masih tidak terlihat jika Anda menjalankan perintah dari jarak jauh. Kecuali jika Anda menggunakan env_parallel yang merupakan antarmuka untuk GNU Parallel yang menyalin lingkungan: 

   . `which env_parallel.bash`\n\n   aliases\n             alias myecho='echo aliases'\n             env_parallel myecho ::: work\n             env_parallel -S server myecho ::: work\n             env_parallel --env myecho myecho ::: work\n             env_parallel --env myecho -S server myecho ::: work\n\n   functions\n             myfunc() { echo functions $*; }\n             env_parallel myfunc ::: work\n             env_parallel -S server myfunc ::: work\n             env_parallel --env myfunc myfunc ::: work\n             env_parallel --env myfunc -S server myfunc ::: work\n\n   variables\n             myvar=variables\n             env_parallel echo '$myvar' ::: work\n             env_parallel -S server echo '$myvar' ::: work\n             env_parallel --env myvar echo '$myvar' ::: work\n             env_parallel --env myvar -S server echo '$myvar' ::: work\n\n   arrays\n             myarray=(arrays work, too)\n             env_parallel -k echo '${myarray[{}]}' ::: 0 1 2\n             env_parallel -k -S server echo '${myarray[{}]}' ::: 0 1 2\n             env_parallel -k --env myarray echo '${myarray[{}]}' ::: 0 1 2\n             env_parallel -k --env myarray -S server echo '${myarray[{}]}' ::: 0 1 2\n

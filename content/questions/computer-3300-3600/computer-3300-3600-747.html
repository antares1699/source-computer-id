Di India, harga versi yang berbeda:

\n\n

Ultimate - 11.200 INR
Profesional - 10.700 INR
Home Premium - 6.600 INR

\n\n

Biaya absolut dari dua yang pertama begitu tinggi bagi saya sehingga perbedaannya (500 INR) tidak menjadi masalah. Jadi bagi saya tidak ada pilihan antara dua yang pertama - Jika saya memutuskan untuk membeli versi Profesional, saya lebih suka memilih Ultimate itu sendiri.

\n\n

Yang ingin saya ketahui adalah, apakah Home Premium sudah mencukupi kebutuhan saya.

\n\n

Saya mencoba mencari perbandingan tetapi banyak yang terlihat seperti sampah pemasaran dari MS. Mereka pendek dan tidak jelas. Berdasarkan halaman ini , perbedaan utama antara Pro dan HomePremium adalah

\n\n
    \n
  1. Jalankan banyak program produktivitas Windows XP dalam Mode Windows XP.
    2. \n
  2. Terhubung ke jaringan perusahaan dengan mudah dan lebih aman dengan Domain Join.
    3. \n
\n\n

Anda dapat melakukan keduanya di Pro tetapi tidak di Home Premium.

\n\n

Saya bermaksud menggunakan Windows 7 saya untuk bisnis kecil - baru memulai. Jadi saya akan berurusan dengan yang berikut ini:

\n\n
    \n
  • Semua jenis alat pengembangan, server
    • \n
  • Sangat penting - Saya akan menjalankan Perangkat Lunak Mesin Virtual (MS VPC atau VMWare atau Sun VirtualBox dll ..)
    • \n
  • Sistem saya akan bertindak sebagai server untuk sebagian besar tujuan sampai saya mampu membeli server khusus.
    • \n
  • Hubungkan sistem ke berbagai perangkat jaringan (PC, Printer, dll ..)
    • \n
  • Jalankan aplikasi produktivitas, bisnis, dan keuangan
    • \n
  • Persyaratan bisnis startup perangkat lunak kecil lainnya yang belum saya pikirkan.
    • \n
\n\n

Profesional (dan Ultimate) dua kali lebih mahal dari Home Premium. Jadi alangkah baiknya jika seseorang dapat menunjukkan hal-hal yang tidak dapat Anda lakukan dengan Home Premium, ketika Anda menggunakannya seperti yang saya jelaskan di atas, sehingga saya dapat memutuskan mana yang akan dibeli. Saya membutuhkan beberapa pengalaman kehidupan nyata sehingga saya dapat membuat keputusan yang tepat - bukan keputusan berdasarkan sampah pemasaran.

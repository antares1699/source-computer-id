Saya secara khusus menggunakan bash di Mac OS X, tetapi menurut saya jawabannya adalah Unix-generik.

\n\n

Saya sudah menggunakan warna ANSI, tetapi saya ingin menetapkan ulang warna yang berbeda ke folder, file tersembunyi, dll.

\n\n

\n\n

Memperbarui:\nIni tidak berguna seperti yang saya harapkan, karena Anda tidak dapat menetapkan warna tertentu ke file tersembunyi, dan juga direktori tidak dapat selalu diwarnai sama, karena warna bit izin lengket lebih diutamakan daripada direktori (hijau zaitun di tangkapan layar)

Ubah profil ~ /. Anda dengan berikut ini. Anda juga bisa meletakkan ini di ~ / .bashrc atau ~ / .bash_profile Anda tergantung pada apa yang sudah Anda miliki dan gunakan. CATATAN: ini menunjukkan pengaturan warna default. Ubah e pertama menjadi B jika Anda ingin melihat direktori merah. 

    LSCOLORS=exfxcxdxbxegedabagacad\n\n    export LSCOLORS\n\n    export CLICOLOR=1    \n

Simpan ~ / .profile Anda dan kemudian lakukan: 

    . .profile\n

Dari baris perintah.

\n\n

Berikut adalah pengaturan warna dan urutan atribut yang berbeda dari halaman manual ls:

The actual display may differ depending on the color capabilities of the terminal in\n                 use.\n\n                 The order of the attributes are as follows:\n\n                       1.   directory\n                       2.   symbolic link\n                       3.   socket\n                       4.   pipe\n                       5.   executable\n                       6.   block special\n                       7.   character special\n                       8.   executable with setuid bit set\n                       9.   executable with setgid bit set\n                       10.  directory writable to others, with sticky bit\n                       11.  directory writable to others, without sticky bit\n\n                 The default is \"exfxcxdxbxegedabagacad\", i.e. blue foreground and default background for regular directories, black foreground and red\n                 background for setuid executables, etc.\n

Saya mencoba mencari cara untuk menambahkan teks di akhir sel di kolom yang memiliki angka. Beberapa sel tidak memiliki apa pun jadi saya tidak ingin menyalin kata tersebut ke dalam sel tersebut.

\n\n

Jika saya memiliki kolom "B" yang hanya memiliki beberapa sel dengan nomor di dalamnya dan saya ingin menambahkan kata "watt" di belakang nomor tersebut, bagaimana cara melakukannya?

\n\n

\n\n

Saya ingin menyimpan nomornya, tetapi menambahkan kata "watt" di akhir.

","source":"https://superuser.com/questions/487942/how-to-append-text-to-existing-number-at-the-end-of-a-cell-in-a-column","author":"Dan Kelly","replies":[],"answers":[{"answer":"

Dengan asumsi kolom data Anda (dengan beberapa sel memiliki angka) adalah Kolom B seperti yang dinyatakan dalam pertanyaan, gunakan rumus berikut untuk sel Anda di, misalnya Kolom C:
=IF(ISNUMBER(B1),B1&\"Watts\",\"\")

\n\n

Ini memeriksa:
1. jika nilai sel adalah angka menggunakan fungsi ISNUMBER ()
2. jika ya, tambahkan "Watt" menggunakan operator penggabungan string ("&")
3. lain, itu membuat sel kosong (jika Anda menghapus "", Anda akan mendapatkan nilai kembali dari fungsi IF () dalam sel mengatakan "FALSE")

Cara lain adalah menggunakan pemformatan khusus untuk mengubah tampilan in situ:

\n\n

Pilih seluruh rentang data Anda di kolom B, klik kanan> Format Sel> Angka> Kustom

\n\n

dan di tipe kotak "Type:"

\n\n

0 "watt"

\n\n

persis seperti itu

\n\n

Itu hanya mengubah tampilan sehingga Anda masih dapat melakukan penghitungan dengan angka (seperti jumlah kolom atau serupa) ..... dan itu tidak akan diterapkan ke sel yang kosong atau teks.

\n\n

Catatan: format itu mengasumsikan bilangan bulat seperti contoh yang ditampilkan, jika Anda mungkin ingin desimal atau bilangan bulat berubah menjadi umum "watt"

Langkah 1: Pilih rentang di mana Anda akan menambahkan teks tertentu;

\n\n

Langkah 2: Tahan tombol Alt + F11 di Excel, dan itu akan membuka jendela Microsoft Visual Basic for Applications.

Langkah 3: Klik Sisipkan> Modul, dan tempel kode VBA berikut di Jendela Modul. 

Sub AppendToExistingOnRight()\nDim c as range\nFor each c in Selection\nIf c.value <> \"\" Then c.value = c.value & \" watts\" \nNext\nEnd Sub\n

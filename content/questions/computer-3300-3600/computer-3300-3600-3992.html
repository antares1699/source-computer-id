Secara default, penginstalan baru Google Chrome akan menempatkan entri di Layar Mulai yang ketika diklik akan membuka Chrome versi UI / Metro Modern. Ini membuat frustrasi, karena saya ingin membukanya di desktop saya. Apakah ada cara untuk memperbaikinya?

","source":"https://superuser.com/questions/493843/how-do-i-make-the-default-google-chrome-link-on-the-windows-8-start-screen-open","author":"Kirk Ouimet","replies":[],"answers":[{"answer":"

Ah, saya melacaknya. Buka regedit dan atur semua kunci ini menjadi kosong: 

HKEY_CLASSES_ROOT\\ChromeHTML\\shell\\open\\command\\DelegateExecute\n\nHKEY_CLASSES_ROOT\\Chrome\\.exe\\shell\\open\\command\\DelegateExecute\n\nHKEY_CLASSES_ROOT\\Chrome\\.exe\\shell\\opennewwindow\\command\\DelegateExecute\n\nHKEY_CLASSES_ROOT\\Chrome\\.exe\\shell\\run\\command\\DelegateExecute\n

Setelah Anda menutup regedit, saat Anda membuka Chrome dari layar awal, Chrome akan terbuka di desktop Anda.

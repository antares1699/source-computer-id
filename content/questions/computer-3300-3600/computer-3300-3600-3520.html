Dengan asumsi gambar terlampir cocok untuk format Anda, lakukan hal berikut:

\n\n
    \n
  1. Ubah data Anda menjadi 2 kolom (dengan nama kolom di atas).
    2. \n
  2. Buat tabel pivot yang berisi 2 kolom data Anda.
    3. \n
  3. Seret nama bidang data ke bagian label baris.
    4. \n
  4. Klik kanan pada tabel pivot dan pilih Opsi Tabel Pivot , pilih Tampilan tab, pilih kotak centang Tabel Pivot Klasik tata letak, klik OK
    5. \n
  5. Klik kanan pada kolom pertama dan hapus pilihan Nomor Model Subtotal (atau apa pun nama kolom pertama Anda
    6. \n
\n\n

Saya tidak memilih nilai kosong di daftar kolom pertama untuk membersihkan tabel

Saya memiliki ribuan email yang akan segera saya hapus dari Outlook. Saya berharap untuk membebaskan beberapa ruang disk setelah melakukannya. Akankah Outlook secara otomatis melepaskan memori yang digunakan oleh email setelah saya menghapusnya? Bagi saya sepertinya tidak mungkin karena Outlook menyimpan emailnya dalam satu file. Bagaimana cara agar Outlook melepaskan ruang disk setelah menghapus banyak email?

","source":"https://superuser.com/questions/719436/reclaim-disk-space-after-deleting-outlook-emails","author":"Agnel Kurian","replies":[],"answers":[{"answer":"

Mudah. Buka saja Pengaturan akun , ikuti File Data tab, klik dua kali file PST yang ingin Anda ringkas dan klik Ringkas Sekarang pada dialog File Data Outlook.

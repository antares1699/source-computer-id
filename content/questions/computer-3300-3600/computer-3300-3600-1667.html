Ketika saya menempelkan dua gambar ke Microsoft Word dan saya mengatur Bungkus Teks ke Persegi:

\n\n

\n\n

dan kemudian kelompokkan mereka sehingga mereka menempel dengan baik bersama-sama, saya tidak bisa lagi Bungkus Teks di sekitar gambar yang baru dikelompokkan ini.

\n\n

Ada penjelasan mengapa pembungkusan teks dinonaktifkan untuk gambar yang dikelompokkan?

\n\n

Catatan: Jika saya tidak mengubah opsi Bungkus Teks, saya tidak dapat mengelompokkannya. Ini untuk Word 2010 di Windows 7, tetapi saya mengalami masalah ini dengan setiap versi Word.

Saya mengalami masalah ini ketika gambar yang saya kelompokkan adalah kombinasi gambar (tangkapan layar) dan objek gambar (garis panah dipilih dari "Bentuk Otomatis"). Di Word 2010, saat Anda mengklik grup untuk pertama kalinya, opsi Alat Menggambar terbuka di Pita. Anda dapat mengubah tata letak dari sini.

\n\n

Jika Anda mengklik grup dua kali, Anda akan mendapatkan opsi Alat Gambar di Pita. Anda tidak dapat mengubah tata letak grup dari Alat Gambar.

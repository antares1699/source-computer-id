menggunakan

\n\n

commandtoexecute &> /dev/null &

\n\n

itu akan menjalankan proses Anda di latar belakang, dan mencetak semua output ke / dev / null.

\n\n

Ganti /dev/null dengan file lain untuk melihat hasilnya.

\n\n

misalnya\ncommandtoexecute &> /tmp/file1 &

\n\n

menggunakan\ntail -f /tmp/file1 untuk dipasang ke keluaran lagi

\n\n

Anda juga dapat mengarahkan stdin, lihat ini http://www.tuxfiles.org/linuxhelp/iodirection.html

\n\n

Jika Anda ingin melepaskan dari proses yang semuanya sudah berjalan. Menggunakan\ndisown <pid>\ndi mana pid adalah id proses Anda.

\n\n

Anda juga dapat mengubah terminal ke terminal lain:

\n\n
    \n
  1. memulai layar
    2. \n
  2. dapatkan pid dari proses Anda
    3. \n
  3. jalankan reptyr <pid>
    4. \n
  4. lepaskan menggunakan CTRL + A + D
    5. \n
\n\n

Menggunakan Karabiner

\n\n

Aplikasi ini dapat memberikan fungsionalitas persis seperti yang diminta OP. Jawaban pengguna495470 tentang KeyRemap4MacBook sangat membantu dalam mengarahkan saya ke arah yang benar. Namun, ini sudah usang dan tidak lagi berfungsi seperti yang ditulis karena KeyRemap4MacBook telah diperbarui. Sekarang disebut Karabiner . Tampaknya terpelihara dengan baik dan merupakan perangkat lunak gratis dan sumber terbuka (tidak demikian halnya dengan BetterTouchTool yang disebutkan dalam jawaban lain yang memerlukan pembayaran setelah 45 hari).

\n\n

Menyiapkannya

\n\n

Download terlebih dahulu dan instal aplikasi Karabiner. Setelah terinstal, untuk memetakan ulang tombol fungsi sebagai tombol fungsi standar, Anda perlu menambahkan "Modifikasi-Kompleks" dalam preferensi. Ada beberapa modifikasi rumit yang telah dibuat sebelumnya dan dapat diunduh dari situs web Karabiner, tetapi Anda juga dapat membuatnya sendiri di json.

\n\n

Gunakan kode berikut dan letakkan dalam sebuah file (bernama seperti "functionkeys.json") di folder ~/.config/karabiner/assets/complex_modifications. Setelah itu, pergi ke preferensi Karabiner dan klik Complex-Modifications, lalu Add rule. Anda akan melihat teks Function Keys in Terminal Apps dan di sana klik Enable All.

\n\n

Bagaimana mengubah aplikasi latar depan mana yang menyebabkan efek tersebut

\n\n

Kode ini secara khusus menyediakan fungsionalitas standar tombol fungsi saat menggunakan salah satu dari beberapa aplikasi terminal (dan tidak ada apa pun saat menggunakan aplikasi lain). Kode ini juga memungkinkan penggunaan kunci media di aplikasi terminal dengan menggunakan fn + function key.

\n\n

Sementara OP bertanya tentang aplikasi spesifik generik, VirtualBox disebutkan secara khusus. Kode di bawah ini dapat diubah untuk membuat fungsionalitas tombol fungsi dalam aplikasi apa pun yang diperlukan dengan mengganti konten setiap instance \"Bundle Identifiers: [ ... ]\" dengan aplikasi apa pun. (VirtualBox mungkin \"^org\\\\.virtualbox\\\\.VirtualBoxVM\" meskipun saya belum mencobanya.) Gunakan Karabiner EventViewer yang disertakan untuk menemukan pengenal aplikasi untuk aplikasi apa pun yang diinginkan, lalu masukkan nilainya.

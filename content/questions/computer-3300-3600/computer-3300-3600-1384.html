Inisial dari nmbd adalah singkatan dari NetBIOS Message Block Daemon. Daemon server nmbd memahami dan menjawab permintaan layanan nama NetBIOS seperti yang dihasilkan oleh SMB / CIFS dalam sistem berbasis Windows.

\n\n

Daemon server smbd menyediakan berbagi file dan layanan pencetakan untuk klien Windows. Inisial dari smbd adalah singkatan dari Server Message Block Daemon.

\n\n

Daemon nmbd dan smbd dikendalikan oleh layanan smb (Server Message Block).

\n\n

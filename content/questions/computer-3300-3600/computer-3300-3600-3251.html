Saya berasumsi bahwa ini adalah virus, meskipun pemeriksa virus saya tidak dapat menemukannya.

\n\n

Saya mengunduh penginstal VMware Workstation 7 terbaru. Saya menjalankan Windows 7 64bit.

\n\n

Saya pergi untuk menginstalnya, dan jendela penginstal adalah situs web Disney.

\n\n

Setelah penyelidikan lebih lanjut, yang terjadi adalah penginstal VMware mengekstrak / menulis banyak file ke direktori temp. Salah satu file tersebut adalah file index.htm. Ketika saya membukanya, cukup yakin itu file Disney.

\n\n

Saya menggunakan Sysinternals Process Monitor untuk mencari sesuatu yang mencurigakan, tetapi satu-satunya hal yang saya lihat menyentuh file index.htm itu adalah penginstal VMware dan explorer.exe

Lupa menjawab ini. Faktanya, itu adalah bug di penginstal vmware. File index.html disney ada di direktori "download" saya karena anak saya menyeret / menjatuhkan / menyimpan.

\n\n

Ketika penginstal vwware - yang berada di direktori yang sama - dijalankan, ia membongkar file index.html-nya sendiri ke dalam direktori ... kecuali jika file tersebut sudah ada. Jadi karena ada file index.html yang tidak berhubungan di sana, itu tidak menimpanya dengan miliknya sendiri; namun, file index.html dimuat, yang menyebabkan perilaku ini.

\n\n

Saya melaporkan bug ini ke vmware, mereka mengonfirmasi, dan itu telah diperbaiki.

\n\n

Intinya: ini bukan virus.

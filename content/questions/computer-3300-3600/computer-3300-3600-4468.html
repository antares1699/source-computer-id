Saya telah menjadi pengguna Revo Uninstaller (gratis) untuk beberapa waktu dan merasa itu melakukan pekerjaan pembersihan yang sangat baik dengan aplikasi biasa. Hari ini saya ingin membersihkan mesin saya lebih banyak, jadi saya melanjutkan untuk menghapus Visual Studio 2005 dengan Revo Uninstaller. Penginstal VS menghapus aplikasi tanpa masalah, lalu Revo melaporkan sekitar 20.000 kunci registri yang tersisa. Saya pada dasarnya hanya melihat Arpcache dan Muicache ... karena saya bukan ahli registri, saya tidak tahu tentang sebagian besar dari 20.000 yang terdaftar.

\n\n

Jadi saya membuat cadangan registri lalu membiarkan Revo menghapus 20.000. Berikutnya melaporkan sekitar 1.500 file sisa yang menyertakan aplikasi Microsoft Office saya (!) Yang saya tahu seharusnya tidak menyentuh. Jadi saya tidak menghapus file apa pun dengan Revo. Mencurigai bahwa beberapa kunci yang dihapus juga terkait dengan Office, saya mencoba membuka Word dan Excel, yang keduanya tahu ada sesuatu yang terjadi, ketika penginstal menendang (meskipun hanya sebentar) untuk masing-masing. Pada titik ini, karena saya tahu ada masalah, saya baru saja memulihkan registri dan sekarang (tampaknya) berjalan dengan baik.

\n\n

Pertanyaan saya, kemudian: Kapan aman untuk mempercayai Revo Uninstaller?

\n\n

Sebagai seorang profesional perangkat lunak berpengalaman, jawaban saya sendiri untuk ini akan menjadi jelas "Ketika kunci yang dilaporkan adalah sesuatu yang Anda pahami dan ketahui aman untuk dihapus" tetapi kemudian itu membuat Revo sedikit berguna kecuali untuk ahli registri, bukan .. .?

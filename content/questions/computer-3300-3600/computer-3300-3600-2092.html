Saya menjalankan Windows tetapi memiliki Cygwin sehingga Windows atau versi Linux sederhana keduanya OK.

\n\n

Ponsel kamera saya menyimpan gambar dalam pohon direktori berdasarkan tanggal, tetapi tampaknya masih menamai setiap file secara unik. Saya ingin menyalin semua file ke satu direktori di PC saya tanpa semua sub-direktori. Bagaimana saya bisa melakukan ini?

\n\n

Menyalin dengan struktur sub-dir itu mudah: xcopy f:\\images\\*.jpg C:\\Users\\Me\\Pictures\\Phone

","source":"https://superuser.com/questions/449925/copy-all-files-from-a-directory-tree-into-one-directory","author":"Mr. Boy","replies":[],"answers":[{"answer":"

Coba ini di PowerShell: 

dir f:\\images -recurse -filt *.jpg | copy -dest C:\\Users\\Me\\Pictures\\Phone\n

catatan:\n- Ini akan menimpa file dengan nama yang sama yang sudah ada di tujuan meskipun dihasilkan dari salinan ini\n- Tambahkan -verbose ke copy untuk melihat apa yang sedang disalin.

find /cygdrive/f/images -name '*.jpg' -exec cp '{}' /cygdrive/c/Users/Me/Pictures/Phone \\;\n

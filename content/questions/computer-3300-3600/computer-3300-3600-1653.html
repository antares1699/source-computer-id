Saya memiliki lembar excel yang terlihat seperti ini: 

date        text        value        other-value\ndate        text        value        other-value\ndate        text        value        other-value\ndate        text        value        other-value\ntotals                    sum          other-sum\n

Jumlahnya tentu saja merupakan penjumlahan dari semua nilai di atasnya. Apa yang saya ingin tahu adalah apakah ada cara saya dapat menambahkan baris dengan sub-total di tempat-tempat acak melalui daftar itu. Misalnya jika saya ingin memiliki sub-total untuk setiap bulan baru. Masalah saya tentu saja bahwa jumlah total akan menjadi besar karena juga akan menjumlahkan baris dengan sub-total. Apakah Excel memiliki fitur yang dapat membantu saya sehingga saya dapat memiliki sub-total dan total-total yang benar?

\n\n

Saya menggunakan Excel 2010.

Jika Anda menggunakan 

=subtotal(9,YourRange)\n

dari pada 

=sum(YourRange)\n

dalam semua kasus, maka itu tidak akan menghitung subtotal Anda dua kali lipat. Dengan kata lain, fungsi subtotal mengabaikan fungsi subtotal lainnya.

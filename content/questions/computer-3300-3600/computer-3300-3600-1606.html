Saya sedang mengembangkan aplikasi web asp.net dan saya menggunakan banyak permintaan ajax. Jadi saya perlu tahu bagaimana saya bisa menyembunyikan permintaan ini dari firebug. Apakah ada cara untuk mencapai hal ini?

","source":"https://superuser.com/questions/214626/hiding-my-request-from-firebug","author":"kbvishnu","replies":[{"reply":"Stack Overflow?","author":"ephilip"}],"answers":[{"answer":"

Jika Anda ingin menggunakan permintaan AJAX; cukup banyak tidak. Anda dapat memeriksa untuk melihat apakah Firebug sedang berjalan dengan menggunakan: 

if (window.console && window.console.firebug) {\n  //Something - Firebug is active\n}\n

Namun pilihan Anda dalam hal menonaktifkan atau mengurangi apa yang orang dapat lihat sangat terbatas dan saya ragu sepadan dengan usahanya. Jika Anda khawatir Firefox mogok, Anda selalu dapat menyarankan agar orang-orang menonaktifkannya untuk situs web Anda. (Seperti yang dilakukan Gmail).

\n\n

Bahkan jika Anda berhasil memblokir Firebug, apa yang harus menghentikan seseorang untuk menggunakan WebKit Inspector atau Alat Pengembang IE agar tidak mendapatkan informasi yang sama? Keamanan melalui ketidakjelasan tidak terlalu efektif.

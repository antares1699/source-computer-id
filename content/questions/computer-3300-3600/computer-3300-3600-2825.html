\n\n

Saya sudah menghubungi dukungan teknis mereka karena saya mengalami masalah dalam menggunakan aplikasi yang memerlukan port tertentu agar mudah diakses (Teman saya menggunakan program sumber terbuka untuk mengendus port saat saya mencoba menggunakan aplikasi). Menurut dukungan teknis, tampaknya ada satu atau lebih modem industri yang melakukan perutean untuk keseluruhan kompleks. Modem-modem tersebut kemudian dihubungkan ke sakelar Jaringan yang menyediakan koneksi ke setiap ruangan (colokan Ehernet di dinding).

\n\n

-Firewall mati masih menunjukkan port tertutup\n-Tampaknya tidak ada NAT yang menyamar ketika saya mengamati detail jaringan\n-Ada lebih dari satu port dan lebih dari satu aplikasi yang bermasalah\n-Bandwidth cukup bagus (10 down, 10 up).

Anda mungkin memiliki IP publik tetapi jika kompleks apartemen Anda memiliki firewall perangkat keras antara Anda dan gateway (bahkan bisa jadi gateway itu sendiri), Anda hanya memiliki dua opsi.

\n\n
    \n
  1. Berharap firewall mereka merespons UPnP permintaan untuk membuka port firewall (ini tidak mungkin karena kemungkinan besar program Anda sudah mencoba melakukan itu dan akan berhasil.)
    2. \n
  2. Tanyakan kepada manajemen apakah Anda bisa membuka port untuk koneksi jaringan Anda. Siapa tahu, mungkin mereka hanya memblokir untuk berhati-hati (saya tidak akan pernah meletakkan perangkat apa pun di internet tanpa firewall perangkat keras di tempatnya (NAT di router rumah melayani tujuan itu)) tetapi mereka dapat membuka port firewall untuk Anda jika Anda bertanya dengan baik .
    3. \n

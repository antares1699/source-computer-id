\n\n

Saya tidak memiliki pengalaman dengan windows 10 IoT, tetapi metode biasa untuk melewatkan\nlayar login dan untuk login langsung saat boot ke akun pengguna adalah\nuntuk masuk ke Panel Kontrol -> Akun Pengguna, klik akun pengguna Anda untuk memilihnya\ndan hapus centang pada kotak berlabel "Pengguna harus memasukkan nama pengguna dan sandi untuk menggunakan komputer ini".

\n\n

Anda akan diminta memasukkan sandi akun pengguna - masukkan sandi\ndan klik OK.

\n\n
\n\n

Jika opsi ini tidak ada di Windows Iot, berikut ini cara melakukan hal yang sama\nmelalui registri (regedit):

\n\n
    \n
  • Navigasi ke\nHKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Winlogon
    • \n
  • Masukkan 1 sebagai nilai AutoAdminLogon
    • \n
  • Masukkan nama komputer (atau domain) sebagai nilai DefaultDomainName
    • \n
  • Masukkan nama akun dengan nilai DefaultUserName
    • \n
  • Masukkan kata sandi akun dengan nilai DefaultPassword
    • \n
\n\n

Info lebih lanjut dapat ditemukan sini .

\n\n
\n\n

Untuk menonaktifkan Layar Selamat Datang Windows 10 di Editor Kebijakan Grup:

\n\n
    \n
  • Jalankan gpedit.ms
    • \n
  • Arahkan ke: Konfigurasi Komputer> Template Administratif> Sistem> Logon
    • \n
  • Klik dua kali "Jangan tampilkan layar pembuka Memulai saat logon"
    • \n
  • Pilih "Nonaktifkan"
    • \n
  • Klik OK
    • \n
\n\n

Lain kali Anda me-restart komputer, komputer Anda akan secara otomatis melewati layar login Windows 10.

Ini adalah satu-satunya pengaturan yang berhasil bagi saya untuk menyembunyikan layar selamat datang secara visual di Windows 10: 

reg add "HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows Embedded\\EmbeddedLogon" /v "HideAutoLogonUI" /t REG_DWORD /d 1 /f\n

Saya menggunakan pengaturan ini bersama dengan AutoAdminLogon dari sini .

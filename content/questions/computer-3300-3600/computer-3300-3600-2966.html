Ini sudah lama mengganggu saya. Ketika saya ssh atau sftp ke server saya melalui PuTTY atau WinSCP, itu keluar setelah 2 menit, kemudian harus tersambung kembali. Putty hanya akan mengunci dan tidak membiarkan saya mengetik ... itu tidak akan mengatakan tidak aktif hanya penguncian murni. WinSCP akan berkata "tuan rumah tidak berkomunikasi" lalu saya tekan sambungkan kembali dan tersambung kembali ...

\n\n

Ini terjadi dalam 3-5 menit! Ini mendorong saya gila !

\n\n

Di PC saya yang lain, menghubungkan ke server yang sama dengan versi yang sama dari kedua program itu akan tetap terhubung selama 24 jam ! Saya bahkan memformat ulang PC saya dan masih masalah yang sama, apakah Anda punya ide untuk mengatasinya?

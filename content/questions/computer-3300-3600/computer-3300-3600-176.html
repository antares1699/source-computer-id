Situasi saya: Saya ingin memperbarui dari VirtualBox 5.1.22 ke 5.1.28. Penginstal versi yang lebih baru macet di dekat bagian akhir, jadi saya perlu menghapus instalan 5.1.22 dengan benar terlebih dahulu agar berfungsi:

\n\n
    \n

  1. Unduh psexec.exe dan CCleaner . Kemudian cabut kabel jaringan, matikan wifi. Bunuh google drive, dropbox, dan alat invasif sistem lainnya (dan lakukan ini lagi jika Anda memulai ulang selama prosedur).

    2. \n

  2. Copot pemasangan menggunakan pencopot VirtualBox dari Control Panel (karena alasan tertentu saya tidak memilikinya di mesin saya).

    3. \n

  3. Hapus folder C:/Program Files/Oracle/VirtualBox dan %userprofile%\\.VirtualBox

    4. \n

  4. Hapus driver kotak virtual (file dimulai dengan VBox ..) dari C:\\Windows\\system32\\drivers\\

    5. \n

  5. Bersihkan registri windows: mulai regedit.exe sebagai administrator, cari VirtualBox di mana-mana dan dengan sungguh-sungguh tetapi hati-hati (jangan merusak hal-hal lain) hapus folder, kunci, dan nilai yang sesuai.

    6. \n
\n\n

Pada titik ini, saya diblokir karena saya tidak dapat menghapus folder anak-anak (yang merupakan driver jaringan VirtualBox) dari folder Computer/HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Enum/Root/NET/. Bagi saya, solusinya terdiri dari

\n\n
    \n

  • Unduh PsExec.exe ke file Desktop

    • \n

  • Pergi Mulai dan dalam "Mulai pencarian" ketik: cmd. Tahan tombol CTRL dan SHIFT, tekan Enter. Jendela prompt perintah akan terbuka.

    • \n

  • Salin dan tempel perintah berikut:\n\"%userprofile%\\desktop\\psexec\" -i -d -s c:\\windows\\regedit.exe

    • \n
\n\n

Menggunakan versi regedit yang diberdayakan, Anda perlu

\n\n
    \n

  • Ambil kepemilikan folder ../NET (klik kanan, "Izin ...", Lanjutan, tab Pemilik, centang "Ganti pemilik pada subkontainer dan objek")

    • \n

  • Tambahkan Kontrol Penuh ke HAK PEMILIK

    • \n

  • Hapus semua folder anak ../NET\nMungkin Anda perlu bermain-main dengan izin lebih lanjut, restart mesin mungkin membantu.

    • \n
\n\n

Sasaran: ../NET harus bersih dari folder driver VirtualBox, dan memiliki izin Kontrol Penuh untuk administrator dan HAK PEMILIK juga.

\n\n

Kemudian beberapa lintasan CCleaner diikuti dengan restart, dan akhirnya Anda bersih. Jika semuanya dilakukan dengan benar, Anda sekarang akan berhasil menginstal versi terbaru VirtualBox, jika perlu.

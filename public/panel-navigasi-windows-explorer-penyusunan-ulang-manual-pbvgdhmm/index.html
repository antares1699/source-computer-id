Panel Navigasi Windows Explorer - Penyusunan ulang manual

Saya benar-benar ingin This PC menjadi elemen pertama di panel navigasi saya, setelah Quick access, tetapi hal-hal seperti OneDrive dan Dropbox memberikan prioritas yang lebih tinggi:

Bagaimana cara menyusun ulang atau menghapus entri tertentu?

Saya hanya menemukan Panel Navigasi dapat dikonfigurasi langsung di Windows Registry.

Entri pihak ketiga berada di bawah jalur \HKEY_USERS\[*SID*]_Classes\CLSID\[*ApplicationHash*]. Jika Anda satu-satunya pengguna yang masuk, Anda SID (atau Pengenal Keamanan) akan terlihat jelas karena hanya akan ada satu. Itu yang panjang. Entri hanya ada untuk pengguna yang saat ini masuk, jika tidak Anda bisa tentukan SID Anda dengan menggunakan pernyataan Command Prompt wmic useraccount get name,sid.

Entri dapat ditemukan dengan mencari nama yang diperlihatkan di Panel Navigasi. Yang tercantum dalam pertanyaan adalah sebagai berikut:

  1. {018D5C66-4533-4307-9B53-xxxxxxxxxxxx} OneDrive
  2. {0E270DAA-1BE6-48F2-AC49-xxxxxxxxxxxx} File Awan Kreatif
  3. {E31EA727-12ED-4702-820C-xxxxxxxxxxxx} Dropbox

Perhatikan bahwa orang yang berbeda mungkin memiliki nilai yang berbeda untuk segmen 12 digit terakhir di jalur.

Entri dikonfigurasi menggunakan dua nilai System.IsPinnedToNameSpaceTree (boolean) dan SortOrderIndex (integer) yang mengontrol visibilitas dan urutan peringkat , masing-masing:

Susun ulang entri dengan memperbarui semua SortOrderIndex nilai (mengabaikan yang Anda nonaktifkan, tentu saja), dengan nilai baru sehingga mengurutkan yang terkecil ke terbesar akan berada dalam urutan yang sama Anda ingin entri Panel Navigasi 3 .

Misalnya untuk menata ulang sedemikian rupa PC ini datang sebelum DropBox lalu OneDrive , Anda dapat menyetel SortOrder dari DropBox hingga 998 dan OneDrive hingga 999.

Setelah Anda menentukan konfigurasi Anda, paksa restart Windows Explorer untuk menjalankan pembaruan dengan mematikannya dengan Task Manager (buka langsung dengan Ctrl + Bergeser + ESC ) dan meluncurkan kembali dari sana (File > Run new task > explorer.exe).

Akses Cepat dapat dihapus dengan menambahkan REG_DWORD HubMode baru dengan nilai 1 ke jalur HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer. Info lebih lanjut .

Catatan

  1. Anda harus mengulangi langkah-langkah ini saat aplikasi diperbarui. Izin hanya dapat diatur pada kunci (yaitu folder), bukan nilai individu untuk mencegahnya dimodifikasi, dan menolak akses ke kunci memiliki efek yang sama seperti pengaturan System.IsPinnedToNameSpaceTree ke 0 (yaitu menghapusnya sama sekali).
  2. Anda mungkin perlu menjalankan Peninjau Suntingan Registri sebagai administrator (gunakan salah satu metode sini ) dan mengambil kepemilikan nilai sebelum Anda dapat memodifikasinya. Ini berfungsi untuk saya hanya menjalankan sebagai Admin tanpa harus mengambil kepemilikan, tetapi saya akan meninggalkan ini di sini untuk referensi.
  3. Quick Access dan This PC tampaknya dikonfigurasi secara berbeda dan saya belum tahu di mana mereka didefinisikan, jadi saya menggunakan peringkat urutan besar untuk memastikan entri pihak ke-3 muncul setelahnya.

Ya, ada di Windows API: SetProcessAffinityMask () . Anda juga dapat mengatur afinitas proses menggunakan pengelola tugas sebagai ditampilkan di sini .

Tidak banyak alasan yang valid mengapa Anda ingin melakukan itu. OS umumnya melakukan tugas penjadwalan program dengan baik pada CPU yang benar.

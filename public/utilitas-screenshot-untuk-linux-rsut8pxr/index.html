Utilitas screenshot untuk linux

Untuk sementara saya mencari alat screenshot yang bagus dan cepat untuk Linux. Saya turun untuk menggunakan Shutter , bagaimanapun, saya hanya menginginkan sesuatu yang melakukan hal berikut:

  • Klik tombol pintas
  • Pilih apa yang ingin Anda jepret
  • Klik enter
  • Unggah ke FTP di latar belakang
  • Beri saya pemberitahuan bahwa URL langsung tangkapan layar sekarang ada di papan klip saya

Adakah yang tahu tentang beberapa proyek sumber terbuka, atau serupa yang mencapai ini? Saya tahu Scrot dapat melakukan beberapa di antaranya, namun, saya belum dapat membuat skrip yang dapat melakukan ini dengan benar untuk saya.

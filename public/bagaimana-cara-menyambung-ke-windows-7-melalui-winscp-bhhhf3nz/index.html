Anda tidak dapat menggunakan SSH di Windows di luar kotak. Windows tidak menginstal server SSH.

freeSSHd, disarankan oleh jawaban lain, belum diperbarui selama bertahun-tahun. Saya tidak akan merekomendasikan Anda untuk menggunakannya.

Ada banyak alternatif lain. Cukup cari "server ssh gratis untuk windows".

Microsoft baru-baru ini merilis Windows build dari server OpenSSH.
Lihat panduan saya untuk menginstal SFTP / SSH Server di Windows menggunakan OpenSSH .

Contoh lainnya adalah Bitvise SSH Server , yang gratis untuk penggunaan pribadi.

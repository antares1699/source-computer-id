Seberapa besar unduhan untuk pemutakhiran Windows 8 Pro?

Saya memiliki koneksi Internet yang relatif lambat (1,2 Mbps) dan saya berencana untuk membeli upgrade Windows 8 Pro secara online. Namun, saya hanya dapat menebak berapa banyak data yang sebenarnya perlu diunduh oleh Asisten Peningkatan - Saya tidak dapat menemukan ukurannya di mana pun di situs web Microsoft atau situs Windows 8. Dengan menentukan ukuran unduhan yang sebenarnya, saya dapat merencanakan waktu yang sesuai, karena kemungkinan besar kita berbicara tentang beberapa jam di sini.

Secara khusus, saya bertanya tentang edisi 64-bit - tetapi saya memutuskan untuk membuat pertanyaan ini umum kalau-kalau seseorang bertanya-tanya seberapa besar edisi 32-bit itu (ukuran file ISO yang saya unduh untuk Pratinjau Konsumen adalah agak berbeda).

Windows 8 / Windows 8 Pro x64 = 3,33 GiB (3,58,18,53,696 byte) ETA: 6.6 hrs at 1.2 Mbps

Windows 8 / Windows 8 Pro x86 = 2,45 GiB (2,63,24,60,288 byte) ETA: 4.8 hrs at 1.2 Mbps

Baik edisi Core (normal) dan Pro termasuk dalam satu ISO. Pengaturan Windows 8 secara otomatis menginstal edisi yang tersedia untuk kunci produk. Anda juga bisa mendapatkan DVD dengan tambahan $ 20.

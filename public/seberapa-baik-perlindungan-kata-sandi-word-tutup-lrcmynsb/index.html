Seberapa baik perlindungan kata sandi Word? [Tutup]

Saya memiliki file MS-Word 2007 yang dilindungi kata sandi yang harus dirahasiakan. Seberapa baik perlindungan Word? Jika tidak terlalu bagus, dapatkah Anda menyarankan metode yang lebih baik untuk menjaga agar file tetap terlindungi?

EDIT: tujuan saya adalah mengirim file yang dilindungi ke penerima (yang tahu kata sandinya). Saya berasumsi penerima ini tidak tahu apa-apa tentang enkripsi / dekripsi, tetapi jika saya benar-benar harus, saya akan mengenkripsi file dan dengan susah payah mengajari penerima cara mendekripsinya.

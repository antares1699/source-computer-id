Bagaimana cara menjalankan PHP dengan Nginx di Windows 10 bash WSL?

Saya telah menginstal PHP7 dan Nginx di Windows 10 bash.

PHP diinstal di /etc/php/7.1 Nginx diinstal di / etc / nginx

Saya mendapatkan 502 Bad Gateway ketika saya mencoba pergi ke localhost dan memuat index.php

Bagaimana cara mengkonfigurasi / etc / nginx / sites-available / default agar dapat menjalankan skrip PHP dengan benar? Sejauh ini, tidak ada yang berhasil saya coba.

Oke, akhirnya saya mendapatkan PHP + Nginx yang berfungsi di Windows 10 bash.

/ etc / nginx / sites-available / default

location ~ \.php$ {
    fastcgi_split_path_info ^(.+\.php)(/.+)$;
    fastcgi_pass 127.0.0.1:9000;
    fastcgi_index index.php;
    include fastcgi_params;
  }

/etc/php5/fpm/pool.d/www.conf

listen = 127.0.0.1:9000

Menginstal php-fpm menggunakan apt-get install php5-cgi php5-fpm

Edit php.ini di /etc/php5/fpm/php.ini

Mulai ulang menggunakan service nginx restart && service php5-fpm restart

Satu hal yang saya perhatikan adalah tautan simbolis tidak berfungsi. Jadi saya harus menyalin file konfigurasi untuk host virtual dari direktori sites-available ke sites-enabled.

Tidak dapat mengubah wallpaper desktop di Ubuntu 14.04

Saya mencoba menginstal GNOME paket ke Ubuntu 14.04 (kesatuan). Saya menggunakan kode-kode ini.

sudo apt-get install gdm

sudo apt-get install gnome-shell

sudo apt-get install ubuntu-gnome-desktop

sudo add-apt-repository ppa: gnome3-team / gnome3-staging

sudo apt-get update

sudo apt-get dist-upgrade

sudo apt-get install gnome-weather gnome-music gnome-maps cheese gnome-documents

Itu tidak berhasil, Jadi saya menghapusnya sepenuhnya menggunakan

sudo apt-get remove gdm

sudo apt-get hapus gnome-shell

sudo apt-get hapus ubuntu-gnome-desktop

Dan sekarang saya tidak dapat mengubah wallpaper desktop saya. Klik kanan di desktop tidak berfungsi . Saya pikir itu juga menghapus paket yang diperlukan dari Unity. Bahkan tampilan layar login tidak berubah dari gaya GNOME ke gaya Unity. Apakah ada cara untuk mendapatkan kembali gaya kesatuan Ubuntu asli?

Coba ini, ini berhasil untuk saya:

gsettings set org.gnome.settings-daemon.plugins.background active true

Namun kasus ini lebih cocok untuk AskUbuntu, dan jawaban yang hampir tidak jelas telah dijawab: Desktop menampilkan latar belakang putih atau hitam, bukan wallpaper (solusi di atas berasal dari sana).

Saya telah mencoba mengomentari pertanyaan Anda tetapi saya tidak dapat melakukannya karena kurangnya reputasi 50.

Untuk menginstal ulang Unity (default untuk Ubuntu) Anda dapat mengetik di terminal Anda:

sudo apt-get update
sudo apt-get install ubuntu-desktop

Setelah reboot, Anda dapat memilih wallpaper lagi.

Apa itu Google Chrome Native Client?

Apakah yang Klien Asli Google Chrome ?

Native Client adalah teknologi sumber terbuka yang memungkinkan Anda melakukannya membangun aplikasi web yang dengan mulus mengeksekusi kode terkompilasi asli di dalamnya browser. Proyek Google Code ini untuk memelihara Native Implementasi klien, termasuk kompiler dan dukungan browser. Jika Anda sedang menulis kompiler, tertarik untuk mem-porting Native Client ke browser baru, atau meneliti mekanisme keamanan yang mendasarinya Klien Asli, Anda dapat menemukan sumber daya yang berguna dalam proyek ini.

Apakah itu berarti bahwa jika saya tidak membangun aplikasi web, saya akan menonaktifkannya? Atau akankah itu benar-benar membuat pengalaman menjelajah saya lebih cepat atau hal lain yang memberi saya beberapa manfaat?

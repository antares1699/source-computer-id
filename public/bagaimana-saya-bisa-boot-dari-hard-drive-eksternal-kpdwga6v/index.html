Anda harus masuk ke BIOS komputer Anda dan memilih USB sebagai perangkat boot pertama.

Sangat sering, BIOS dimasukkan dengan menekan sebuah tombol di awal urutan boot, tepat setelah komputer dihidupkan:

  • F1
  • F2
  • ESC
  • DEL
  • SUPPR (pada keyboard Prancis)
  • (lihat dokumentasi sistem Anda ...)

Setelah Anda berada di BIOS, Anda dapat mengkonfigurasi urutan boot: misalnya coba boot dari floppy terlebih dahulu, kemudian cdrom / dvdrom, lalu hard-drive ... Urutan boot yang disimpan di BIOS tetap sama setiap kali komputer dimulai.

Seperti yang disarankan dalam jawaban dan komentar lain, beberapa BIOS memungkinkan Anda untuk menekan tombol (misalnya F12) yang akan memberi Anda menu boot sekali pakai: tujuan menu ini adalah untuk memungkinkan Anda melakukan booting secara luar biasa dari perangkat lain.

Jika Anda sangat ingin mem-boot dari hard-drive USB, Anda mungkin ingin membuka menu boot. Jika Anda ingin selalu boot dari hard-drive USB yang tetap terhubung ke komputer Anda, kemungkinan besar Anda akan mengubah urutan boot di BIOS Anda.

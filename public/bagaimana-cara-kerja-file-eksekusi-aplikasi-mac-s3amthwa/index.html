Bagaimana cara kerja file eksekusi aplikasi Mac?

Bagaimana cara kerja file eksekusi .app? Bisakah itu diuraikan? Di dunia Windows, file exe dimodifikasi melalui editor hex dan semacamnya dan sebagian besar executable mengandalkan file perpustakaan seperti dll dan file ini.

Tapi bagaimana cara kerja file .app?

File .app adalah arsip, yang berisi file biner yang dapat dieksekusi itu sendiri, serta sumber daya apa pun yang menyertainya. Intinya, ini berisi semua file yang akan Anda lihat di dalam folder program di Windows, tetapi untuk UI itu muncul sebagai item monolitik tunggal. Hal ini memungkinkan program untuk diinstal dan dihapus instalasinya hanya dengan menempatkan file .app di folder / Applications (atau di mana pun pengguna menginginkannya), dan menghapusnya saat tidak lagi diinginkan.

Saat Anda mengklik dua kali file .app, pengendali terbuka OS tahu untuk mengeksekusi biner internal.

Anda dapat mengklik kanan pada file .app dan tekan "tampilkan konten paket" untuk melihat struktur folder internal dari paket tersebut.

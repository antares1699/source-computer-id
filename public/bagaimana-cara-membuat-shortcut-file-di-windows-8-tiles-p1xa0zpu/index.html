Bagaimana cara membuat shortcut file di Windows 8 Tiles?

Di Windows 8, apakah mungkin membuat pintasan file di layar Mulai baru dengan Live Tiles? Jika demikian, bagaimana caranya?

Untuk membuat pintasan ke file di Ubin baru Windows 8, cukup buka Windows Explorer, klik kanan file Anda, dan klik Pin to Start.

  1. Klik kanan pada Desktop, dan pilih New-> Shortcut.
  2. Dalam Buat Jendela Pintasan yang muncul, Jelajahi dan pilih file yang pintasannya ingin Anda buat.
  3. Terakhir, klik kanan pada pintasan yang dibuat dan pilih Sematkan untuk Memulai .

Jendela telah diatur ke koordinat layar yang tidak valid, tetapi mudah untuk diperbaiki:

  1. Klik tombol prompt perintah di bilah tugas
  2. Tekan tombol berikut:
    1. Alt + Ruang
    2. S
    3. Salah satu tombol panah (mis., → )
  3. Gerakkan mouse (jendela prompt perintah sekarang seharusnya ada di layar)
  4. Simpan posisi baru dengan menekan tombol berikut:
    1. Alt + Ruang
    2. P.
    3. Memasukkan atau Kembali

Sekarang tutup command-prompt dan buka yang lain. Jendela harus di layar.

(Perhatikan, metode di atas untuk menyimpan eksploitasi posisi sebuah bug di Windows 7. Anda dapat memilih Default di menu konteks, bukan Properti jika Microsoft memutuskan untuk memperbaiki bug tersebut. Selain itu, untuk Windows XP, Anda dapat melakukan tindakan yang sama, tetapi harus memilih Ubah pintasan yang memulai jendela ini setelah menekan Memasukkan .)

RAM tidak membersihkan, selalu dalam keadaan siaga

Saya mengalami masalah dengan ram saya di windows 7. Ram sedang membersihkan atau program menggunakan dan membuangnya lebih cepat daripada yang bisa dibersihkan. Saya memiliki 8gb ram, saya hanya menggunakan 4gb dan sisanya dalam mode Standby. Saat terbaik saya bisa mendapatkan 100mb gratis tetapi kebanyakan 15mb atau kurang. Saya menduga ada kebocoran memori tetapi saya tidak tahu program apa itu. Saya belum menginstal perangkat keras baru apa pun baru-baru ini dan program yang baru diinstal telah bin ditutup. Apakah ada yang tahu apa yang sedang terjadi dan bagaimana cara memperbaikinya?

Maaf, informasi ini sangat sedikit, tetapi hanya itu yang saya miliki.

Edit: Saya mencoba RamMap dan menemukan "File yang Dipetakan" saya memiliki 3.291.784k ram dalam keadaan standby. Bagaimana cara mengetahui program apa dan bagaimana cara membatasi kemampuannya untuk mengatur ram dalam standby?

Edit: Saya terus mendapatkan masalah ketika saya memiliki 0 ram gratis dan komputer saya akan berhenti bekerja dengan baik. Ini akan menjadi sangat lambat di mana perlu 3-4 menit untuk memulai pengelola tugas sehingga saya dapat menutup beberapa program. Ketika saya membebaskan beberapa ram dengan menutup beberapa program, semuanya berjalan normal. Tidak masalah program mana yang saya tutup hanya yang saya bebaskan beberapa ram. Saya tahu banyak dari Anda mengatakan seharusnya seperti itu dan begitulah cara kerja jendela. Tapi jelas ada masalah dan komputer saya tidak pernah bekerja seperti itu, begitu juga komputer saya yang lain. Saya ingin memperbaikinya, tolong bantu saya.

Nilai standby yang besar sangatlah fantastis. Ini berarti data di-cache oleh layanan superfetch dan tidak menjadi aktif bekas. Ini masih tersisa di ram fisik tapi akan digunakan kembali terlebih dahulu oleh pengelola memori (baik dikembalikan ke daftar aktif atau dikosongkan dan digunakan kembali) jika sesuatu membutuhkan ram fisik untuk halaman aktif. Halaman siaga pada dasarnya adalah cache - lebih baik menyimpan data yang jarang digunakan di RAM "untuk berjaga-jaga" daripada memasukkannya ke disk saat memori tidak diperlukan untuk hal lain.

Saya mereka, mereka akan tahu bahwa banyak hal di Windows tidak berfungsi seperti yang dikatakan Microsoft.

Cara grep baris dimulai dengan "*" menggunakan grep

Bagaimana cara menggunakan 'grep' untuk mendapatkan baris yang dimulai dengan '*' di file saya?

Saya mencoba grep "*" myfile dan grep "* " myfile tetapi mengembalikan semua baris file saya.

4

11

"*" Adalah karakter khusus yang berarti "0 atau lebih dari karakter sebelumnya." Anda harus melarikan diri dari karakter ini.

grep '^\*' myfile

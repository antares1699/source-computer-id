Di mana Kredensial disimpan untuk Drive Jaringan di WinXP?

Saya memiliki drive yang dipetakan ke folder di mesin jarak jauh yang saya sambungkan dengan menggunakan klien Cisco VPN.

Kata sandi ke akun Windows yang saya gunakan di mesin jarak jauh itu telah berubah.

Saya telah menyimpan nama pengguna / kata sandi secara lokal, menggunakan fitur ingat kata sandi saya di Window, jadi saya tidak perlu memasukkannya setiap saat (dialog masuk pengguna / kata sandi biasanya muncul setiap kali saya mencoba membuka folder jarak jauh, dan saya harus mencari dan memasukkan kredensial saya).

Kata sandi untuk akun Windows jarak jauh itu telah berubah.

Sekarang, saya tidak lagi diminta untuk memasukkan nama pengguna / kata sandi, tetapi sebagai gantinya, saat mencoba membuka folder jarak jauh, menerima pesan: nama pengguna yang tidak dikenal atau kata sandi yang buruk.

Bagaimana cara melihat dan mengubah kredensial yang disimpan ini?

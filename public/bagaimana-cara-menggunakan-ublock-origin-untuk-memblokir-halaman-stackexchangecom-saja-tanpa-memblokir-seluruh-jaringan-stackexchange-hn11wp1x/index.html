Bagaimana cara menggunakan uBlock Origin untuk memblokir halaman stackexchange.com saja tanpa memblokir seluruh jaringan stackexchange?

Saya menemukan HNQ benar-benar mengganggu saya, jadi saya ingin memblokir mereka. Pertanyaan ini memiliki beberapa cara untuk memblokir mereka di sidebar, tetapi daftar yang sama ada di https://stackexchange.com/ . Saya ingin memblokir halaman itu, tanpa memblokir semua stackexchange lainnya. Saya sedang menggunakan

|https://stackexchange.com^

tetapi itu menampilkan halaman di mana saya dapat mengklik untuk mengaksesnya: Saya lebih suka memuatnya tetapi menampilkan halaman kosong. Saya melakukan ini untuk penggunaan reddit

reddit.com##body

tetapi menggunakan filter serupa untuk blok stackexchange, misalnya, semua tex.stackexchange.com. Saya mencoba

~stackexchange.com##body

Jawaban ini mencoba untuk blokir Pertanyaan Jaringan Panas (HNQ) hanya di beranda Stack Exchange (SE) (sesuai Komentar OP ).

HNQ aktif Beranda SE berada di dalam div dengan question-list ID, jadi filter yang dibutuhkan adalah stackexchange.com###question-list.

Filter ini hanya akan memblokir HNQ di homepage SE. Itu tidak akan menghalangi Pertanyaan SE Realtime (penampung induk adalah main-area, yang dibagikan dengan daftar HNQ) maupun daftar pertanyaan apa pun di situs lain (mereka menggunakan div dengan question-mini-list ID pada beranda mereka dan questions di Daftar pertanyaan )

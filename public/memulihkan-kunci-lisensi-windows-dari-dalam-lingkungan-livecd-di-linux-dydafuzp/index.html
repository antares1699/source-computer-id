Memulihkan kunci lisensi Windows dari dalam lingkungan LiveCD di Linux

Saya ingin tahu apakah mungkin untuk mengambil kunci lisensi mesin Windows XP dari dalam lingkungan LiveCD di Ubuntu.

Biasanya, jika saya menggunakan Windows, saya hanya akan menjalankan Magical Jellybean Keyfinder, tetapi adakah cara untuk melakukan ini dari dalam lingkungan LiveCD?

File Windows Registry berada di

%SystemRoot%\System32\config\
  1. Dari distro Linux live Anda, salin folder %SystemRoot%\System32\config\ ke drive USB.
  2. Di komputer Windows lain, buat file Windows direktori di suatu tempat dan kemudian buat file Sistem 32 folder di dalamnya untuk meniru struktur direktori dari instalasi Windows yang sebenarnya.
  3. Salin config folder di drive USB ke dummy baru Anda Windows \ System32 .
  4. Unduh dan pasang Magical Jelly Bean Keyfinder v2.0.1 (MJBKF).
  5. Titik MJBKF ke yang baru dibuat Windows \ System32 Anda buat sebelumnya.
  6. Membiarkan MJBKF lakukan keajaibannya.

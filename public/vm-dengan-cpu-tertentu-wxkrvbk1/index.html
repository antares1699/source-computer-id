VM dengan CPU tertentu

Apakah mungkin untuk membuat VM dengan CPU tertentu yang hanya mendukung set instruksi tertentu.

Misalnya, saya ingin mencoba sesuatu pada Mesin yang memiliki CPU yang tidak mendukung instruksi AVX atau SSE, apakah mungkin membuat VM dengan CPU yang tidak mendukung AVX atau SSE?

Semacam. VMware dan VirtualBox tidak mendukung emulasi, hanya virtualisasi. Jadi itu artinya Anda tidak bisa Menambahkan fitur. Namun, menghapus fitur seharusnya tidak menimbulkan masalah.

Artikel ini memiliki beberapa detail tentang cara memalsukan CPUID seperti yang dilaporkan kepada tamu. Ingatlah bahwa ini tidak benar-benar menonaktifkan fitur apa pun.

Instruksi untuk produk VMware (kutipan):

Gunakan editor (misalnya notepad) buka file konfigurasi mesin virtual (.vmx)
Menambahkan baris berikut:

cpuid.1.eax = "00000000000000110100011010101001"
cpuid.1.edx = "10111111111010111111101111111011"

Jika CPUID masih tidak berubah atau tidak valid, coba tambahkan:

cpuid.1.ebx = "00000000000000110100011010101001"
cpuid.1.ecx = "10111111111010111111101111111011"

Simpan file konfigurasi dan mulai Mesin Virtual Anda.

Instruksi untuk VirtualBox (kutipan):

Buka Command Prompt dan alihkan ke direktori kerja VirtualBox.
Jalankan perintah berikut untuk mengubah CPUID.

vboxmanage modifyvm  --cpuidset 1 000306a9 00100800 3d9ae3bf bfebfbff

Mulai Mesin Virtual Anda.

Bagaimana Anda mendapatkan nilai CPUID itu terserah Anda, Anda mungkin perlu membaca manual Arsitektur Set Instruksi vendor CPU Anda.

Jika Anda ingin meniru fitur, Anda perlu menggunakan emulator. QEMU dapat melakukan virtualisasi dan emulasi penuh. Ini juga mendukung banyak arsitektur CPU selain x86 / x64.

Dokumentasi VMware menyebutkan "CPUID Mask", tetapi saya tidak tahu seberapa bisa dikonfigurasi.

Jika Anda ingin bereksperimen dengan VMM lain, QEMU dapat mengaktifkan dan menonaktifkan tanda CPUID di baris perintah. Misalnya, qemu-system-i386 -cpu n270,fpu=off mengarah ke CPU virtual dengan fitur n270 tetapi dengan flag fpu dinonaktifkan. Ada daftar lengkap model CPU yang didukung dan tanda CPUID di qemu-system-i386 -cpu help.

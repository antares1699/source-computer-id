Apa format arsip lain yang menyimpan Tanggal Dibuat dan Tanggal Diakses?

7-Zip di Windows akan menyimpan Tanggal Dibuat dan Tanggal Diakses saat mengompresi file .zip, namun tidak melakukannya untuk format .7z aslinya (mengacu pada 7-Zip). Semua format berbasis Unix tidak memenuhi syarat. Apakah ada format lain di luar sana yang mendukung kedua bidang ini?

Anda juga dapat menyimpan Access-, Modification- and Creation-Date saat menggunakan format 7-zip ".7z". Untuk melakukannya, cukup tentukan parameter berikut saat mengemas data:

  • GUI 7-zip: tulis " tc ta "menjadi" Parameter: "textfield
  • Baris perintah 7-zip: tentukan " -mtc -mta -mtm "parameter

tc singkatan dari waktu penciptaan,
ta singkatan dari waktu akses dan
tm singkatan dari waktu modifikasi

Waktu modifikasi disimpan secara default.

Tanggal Dibuat, Dimodifikasi dan Diakses bukanlah fungsi dari format file. Selain itu, 7-Zip bukanlah aplikasi asli Windows.

Terlepas dari itu, atribut ini adalah bagian dari penjurnalan NTFS.

Anda dapat menambahkan atribut ini ke Windows explorer dengan mudah untuk semua jenis folder / file:

Namun, mereka tidak selalu akurat atau aktif.

Di komputer saya, misalnya jurnal yang diakses tampaknya tidak aktif. Mengklik kanan dan melihat properti file, saya melihat sedikit berbeda (dan atribut yang lebih akurat)

Saya juga dapat menjalankan perintah PowerShell untuk mendapatkan informasi

gci | LastAccessTime memberi saya:

Ini sama akuratnya dengan tampilan penjelajah.

Sebaliknya, apa yang perlu dilakukan adalah memastikan Windows memilikinya jorunaling diaktifkan.

Ini dapat diaktifkan dari prompt perintah administrator dengan perintah:

fsutil behavior set disablelastaccess 0

Keyboard tertinggal; kehilangan penekanan tombol

Saya menggunakan Mint 17.3 di Toshiba Satellite L500, dan tiba-tiba (mungkin setelah pembaruan), keyboard saya mulai bertindak sangat unik.

Beberapa penekanan tombol diterima dengan jeda, beberapa tidak, dan beberapa digandakan. Ini tidak terikat pada kunci tertentu.

Adakah ide tentang apa masalahnya dan bagaimana mendiagnosisnya?

Ini terlihat seperti masalah perangkat keras / driver - Saya juga mendapatkan perilaku di konsol virtual, dan keyboard USB eksternal berfungsi dengan baik. Sayangnya, menggunakan keyboard eksternal dengan laptop tidak terlalu praktis.

Saya tidak mengharapkannya, tetapi saran dari http://forums.toshiba.com/t5/Keyboards-Touchpads/Toshiba-Satellite-L500-Keyboard-not-working/td-p/160306 yaitu:

Berikut ini perbaikan untuk kerusakan keyboard:

  1. Matikan komputer, lepaskan kabel daya, lepas baterai.
  2. Tekan tombol Power On selama 10 detik
  3. Ganti baterai dan / atau sambungkan kabel listrik.
  4. Nyalakan komputer + segera tekan tombol F2 berulang kali
  5. Tekan F9 lalu tekan Enter
  6. Tekan F10 lalu tekan Enter

oleh pengguna petir tampaknya telah memperbaiki masalah saya.

Semoga seseorang merasa berguna.

Kemungkinan besar ini adalah masalah sementara yang terkait dengan baterai.

Matikan, keluarkan baterai, boot .

Jika semuanya baik-baik saja, coba pasang kembali baterai. Sekali lagi ... matikan, pasang kembali baterai, boot.

Jika semua masih bekerja maka Anda baik-baik saja. Jika masalah dengan keyboard / mouse kembali, ganti baterai dengan yang baru.

Masalah ini tidak terbatas pada laptop Toshiba, tetapi masalah umum yang dapat terjadi pada semua merek.

Tosh Satellite L755D kami terus tertinggal saat memasukkan teks, berjalan lambat, kursor melompat-lompat. Saya mencoba trik baterai keluar yang dijelaskan di sini, tetapi tidak ada peningkatan. Saya menginstal BIOS yang diunduh, mem-boot ulang dan sekarang laptop berfungsi normal kembali. Pantas untuk dicoba jika Anda masih mengalami masalah setelah melepas baterai.

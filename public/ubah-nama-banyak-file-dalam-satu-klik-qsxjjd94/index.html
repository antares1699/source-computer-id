Di Windows pada baris perintah: Gunakan PowerShell, dan foreach-nya dan perintah ganti nama ke. Di sana Anda dapat menggunakan semua fungsi PowerShell (mengonversi huruf, mengganti, sha-1 dll) untuk memanipulasi nama.

Windows atau Ubuntu: Gunakan editor direktori Emacs - Anda pada dasarnya membuka daftar file di editor, mengeditnya (cari / ganti, apa pun yang Anda suka) dan kemudian Anda "menerapkan" semua perubahan sekaligus. (Perhatikan bahwa jika Anda tidak menyukai Emacs - dan ini adalah semacam hubungan cinta-atau-benci - Anda mungkin juga tidak menyukai solusi ini).

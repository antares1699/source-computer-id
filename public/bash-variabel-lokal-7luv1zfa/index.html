Bash variabel lokal

Bagaimana cara mendeklarasikan variabel yang bersifat lokal ke fungsi? Variabel yang dideklarasikan secara teratur diekspor ke lingkup global. Contoh:

$ cat t.sh
function locals_test() 
{
  x="bar"
}

$ echo $x

$ locals_test 
$ echo $x
bar

Gunakan local:

local x=bar

Kemudian, x akan menjadi lokal untuk fungsi tersebut. Untuk mendeklarasikan mis. array atau angka, local mengambil opsi yang sama seperti declare. Output akan menjadi dua baris kosong:

#!/bin/bash
function locals_test() 
{
  local x=bar
}

echo $x
locals_test
echo $x

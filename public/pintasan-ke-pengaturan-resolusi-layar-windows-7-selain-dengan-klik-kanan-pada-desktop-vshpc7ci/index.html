Pintasan ke pengaturan resolusi layar Windows 7 selain dengan klik kanan pada desktop

Saya secara teratur harus mengakses pengaturan "Resolusi layar" Windows 7 pada netbook dengan banyak jendela layar penuh terbuka, seringkali dengan CPU yang macet.

Ini adalah langkah-langkah biasa yang harus saya lakukan:

  • alt-tab ke desktop
  • klik kanan pada desktop
  • pilih "Resolusi layar"
  • dari menu tarik-turun "Beberapa tampilan", pilih "Tampilkan hanya desktop di 2"
  • klik OK

Apakah ada cara untuk mengakses pengaturan ini selain dari menu mulai, bilah tugas, atau tombol pintas? Lebih baik lagi apakah ada cara untuk membuat tindakan "Tampilkan desktop hanya di 2" langsung dari salah satu tempat itu?

Jika Anda hanya ingin mengubah tampilan mana yang Anda gunakan, daripada resolusi setiap tampilan, Anda dapat beralih di antara opsi tampilan saat menggunakan beberapa monitor dengan Menang + P. hotkey.

Atau (jika Anda ingin mengubah resolusi layar):

Anda dapat langsung mengakses halaman Display> Screen Resolution dengan mengetik:

resolution

di bilah pencarian Windows, dan memilih hasil teratas (harus Sesuaikan resolusi layar). Anda masih harus menonaktifkan atau mengaktifkan layar pilihan Anda.

Saya yakin ada trik untuk mendapatkan kunci otomatis untuk melakukannya juga, tetapi saya terhindar.

Beberapa versi yang lebih cepat:

Menggunakan Menang + D untuk meminimalkan semua jendela, lalu klik kanan, dll.

Atau:

Klik kanan di desktop, buat pintasan baru, lokasi desk.cpl. Sebut saja apa pun yang Anda inginkan. Ketika Anda mengklik ikon, dialog resolusi layar muncul, dari mana Anda dapat memilih monitor.

Klik kanan pada pintasan baru, di tab pintasan buat pintasan yang Anda inginkan. Jika tidak berhasil, mungkin itu sudah digunakan. Coba yang lain.

Saya yakin akan ada cara pintas ke satu monitor, mungkin dengan kunci otomatis.

Right click on desktop > New > Shortcut. Sebagai tipe lokasi di:

rundll32.exe shell32.dll,Control_RunDLL desk.cpl

Beri nama seperti Screen Resolution.

Ikon dapatkan dari %windir%\System32\Display.dll.

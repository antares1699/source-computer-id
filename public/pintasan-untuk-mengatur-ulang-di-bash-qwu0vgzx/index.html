Pintasan untuk mengatur ulang di bash

Sepertinya saya terlalu sering memasukkan perintah reset pada bash.

Saya ingin mengkonfigurasinya ke pintasan keyboard.

Ucapkan Ctrl + r atau sesuatu seperti itu.

Bagaimana cara melakukannya.

Memperbarui:

Untuk menjawab pertanyaan, mengapa saya ingin melakukan ini, dan tidak hanya jelas,

  1. Seringkali, saya ingin melihat git (atau svn atau hg) diffs dan memulai dari atas itu mudah, tanpa repot-repot di mana perintahnya.

  2. Seringkali, karakter pada bash dimakan (dan tidak ditampilkan), ketika digunakan sebagai server dan beberapa hal python dilakukan (Dan saya tahu ini adalah masalah bash karena, zsh dan ikan tidak memiliki masalah). Satu-satunya cara untuk membuatnya berfungsi kembali adalah dengan mengatur ulang.

Anda dapat memodifikasi file / etc / inputrc atau ~ / .inputrc sesuai dengan sintaks ini .

Bash juga memiliki file perintah mengikat yang memungkinkan Anda mengubah keybindings.

Saya pikir mereka berbagi sintaks yang serupa untuk menjalankan perintah shell. Contoh:

$ bind -x '"\C-r"':reset

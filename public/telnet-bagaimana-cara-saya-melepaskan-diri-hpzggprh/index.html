Telnet: Bagaimana cara saya melepaskan diri?

Saya tahu bahwa ketika terhubung melalui Telnet, saya dapat menekan Ctrl + ] untuk melarikan diri. Ini membawa saya ke antarmuka baris perintah telnet. Bagaimana cara kembali ke sesi telnet saya?

Saya sesekali memukul Ctrl + ] secara tidak sengaja dan ingin melanjutkan pekerjaan saya. Saya menggunakan telnet di ketiga mesin utama (MacOS, Windows, & Linux).

Perhatikan bahwa saya tidak mencoba untuk keluar dari telnet; Saya hanya ingin melanjutkan sesi saya.

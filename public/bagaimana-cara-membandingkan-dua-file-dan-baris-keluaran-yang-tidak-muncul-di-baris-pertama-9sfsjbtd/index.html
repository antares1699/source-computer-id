Bagaimana cara membandingkan dua file dan baris keluaran yang tidak muncul di baris pertama?

Jika saya memiliki file A yang berisi daftar bidang:

2017-04-23
2017-04-30
2017-05-07
2017-05-14
2017-05-21
2017-05-28
2017-06-04
2017-06-11
2017-06-18
2017-06-25

Dan file lain B yang berisi daftar bidang:

2017-04-23
2017-04-30
2017-05-07
2017-05-14
2017-05-21
2017-05-28
2017-06-04
2017-06-11
2017-06-18
2017-06-25
2017-07-02
2017-07-09
2017-07-16
2017-07-23

Bagaimana saya dapat dengan cepat membedakan kedua file ini di mana saya ingin mengetahui semua bidang dalam file B yang tidak hadir dalam file A?

Ini bukan perbedaan biasa di mana saya ingin melihat perbedaan relatif antara file tetapi lebih seperti perbandingan hash di mana setiap baris adalah entri di peta. Saya ingin mendapatkan daftar semua baris dalam file B yang tidak ada dalam file A sehingga saya dapat menghapusnya di mana setiap baris dalam file A mewakili direktori yang akan dipertahankan.

Saya mencari solusi Bash / CoreUtils.

grep adalah alat yang tepat untuk pekerjaan itu, meskipun bukan dari Bash maupun dari CoreUtils:

grep -Fxvf A B

Semua opsi ini sesuai dengan POSIX. Dari man 1 grep :

-f pattern_file

Baca satu atau lebih pola dari file yang dinamai dengan pathname pattern_file. Pola pada pattern_file akan diakhiri oleh a. Pola null dapat ditentukan dengan baris kosong di pattern_file. Kecuali jika opsi -E atau -F juga ditentukan, setiap pola akan diperlakukan sebagai BRE, seperti yang dijelaskan dalam volume Definisi Dasar POSIX.1-2008, Bagian 9.3, Ekspresi Reguler Dasar.

-F

Cocokkan menggunakan string tetap. Perlakukan setiap pola yang ditentukan sebagai string, bukan sebagai ekspresi reguler. Jika baris masukan berisi salah satu pola sebagai urutan byte yang berdekatan, baris tersebut harus dicocokkan. String nol harus cocok dengan setiap baris.

-v

Pilih garis yang tidak cocok dengan pola yang ditentukan. Jika opsi -v tidak ditentukan, garis yang dipilih harus yang cocok dengan salah satu pola yang ditentukan.

-x

Pertimbangkan hanya baris masukan yang menggunakan semua karakter dalam baris kecuali terminating untuk mencocokkan seluruh string tetap atau ekspresi reguler menjadi baris yang cocok.

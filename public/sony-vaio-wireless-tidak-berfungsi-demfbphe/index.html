Sony Vaio Wireless tidak berfungsi?

Saya memiliki laptop Sony Vaio E-series (VPCEG28FN) dan saya menggunakan Windows-7. Kemarin, Saat saya menjalankan Netscout sniffer, laptop masuk ke layar Blues untuk pertama kalinya mengatakan-

untuk mencegah kerusakan perangkat keras lebih lanjut ... dan dimulai ulang.

Kemudian saya melihat bahwa wifi nirkabel saya tidak berfungsi, segera saya mencopot pemasangan perangkat lunak sniffer profesional tetapi masalahnya tetap seperti itu sampai sekarang. Tapi, laptop bekerja dengan baik dengan kabel Ethernet.

Saya telah memeriksa Vaio Smart Network & ini menampilkan- LAN Nirkabel dihidupkan. Saya juga telah memeriksa pengelola perangkat [Adaptor Jaringan] dan tertulis-

Atheros AR9285 Wireless Network Adapter bekerja dengan baik.

Apa yang dapat saya lakukan untuk mengaktifkan kembali LAN nirkabel saya?

Terima kasih sebelumnya!

Di Sony Vaio w / Atheros saya, setelah menggunakan CD Bootable Linux, kartu nirkabel saya juga menolak untuk bekerja. Saya memperbaikinya dengan menghapus driver sepenuhnya dan menginstalnya kembali (yang resmi Sony). Saya dapat mengulangi masalah dan solusinya - ternyata komputer saya memiliki tombol on / off WiFi fisik dan "berbasis perangkat lunak".

Saya tidak yakin apakah "Viao Smart Network" adalah utilitas yang memungkinkan Anda menghidupkan / mematikan sakelar perangkat lunak. Utilitas Wifi yang disertakan dengan driver, itulah yang berfungsi untuk saya.

Jika ini tidak berhasil, Anda juga dapat mencoba menginstal ulang sniffer, mungkin menginstal driver khusus dan menghapusnya dengan tidak bersih. Sangat tidak mungkin itu dapat merusak fisik apa pun.

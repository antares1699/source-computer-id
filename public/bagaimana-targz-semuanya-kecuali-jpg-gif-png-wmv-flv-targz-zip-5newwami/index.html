Bagaimana tar.gz semuanya kecuali .jpg / gif / png / wmv / flv / tar.gz / zip?

bagaimana saya tar.gz semuanya selain dari jpg/gif/png/wmv/flv/zips, secara rekursif, melalui shell Unix?

Gunakan opsi --exclude:

tar czvf myarchive.tar.gz --exclude=\*.{jpg,gif,png,wmv,flv,tar.gz,zip} rootdir

find -type f -a ! \( \
                     -name '*.jpg' -o \
                     -name '*.gif' -o \
                     -name '*.png' -o \
                     -name '*.wmv' -o \
                     -name '*.flv' -o \
                     -name '*.tar.gz' -o \
                     -name '*.zip' \) \
 | xargs tar rvfz files.tgz

Saya menggunakan banyak baris untuk membuatnya lebih mudah diedit nanti jika Anda membuatnya menjadi skrip shell atau fungsi shell.

Saya menggunakan -r flag untuk tar jika jumlah file melebihi 1 panggilan ke xargs. Dengan cara ini Anda tidak bisa melebihi buffer argumen untuk tar.

Jika Anda memilih 'Perbaiki kesalahan sistem file secara otomatis' untuk disk yang sedang digunakan (misalnya, partisi yang berisi Windows), Anda akan diminta untuk menjadwal ulang pemeriksaan disk untuk waktu berikutnya Anda memulai ulang komputer. Itu tidak bisa diperbaiki dengan cepat.

Jika Anda ingin memeriksa hard disk Anda tanpa memulai ulang:

  • Klik untuk membuka Komputer.
  • Klik kanan hard disk yang ingin Anda periksa, lalu klik Properties.

  • Klik tab Alat, lalu, di bawah Pemeriksaan kesalahan, klik Periksa sekarang. Jika Anda diminta kata sandi atau konfirmasi administrator, ketik kata sandi atau berikan konfirmasi.

  • Untuk secara otomatis memperbaiki masalah dengan file dan folder yang terdeteksi oleh pemindaian, pilih "Perbaiki kesalahan sistem file secara otomatis". Jika tidak, pemeriksaan disk akan melaporkan masalah tetapi tidak memperbaikinya.
  • Untuk melakukan pemeriksaan disk secara menyeluruh, pilih Pindai dan coba pemulihan bad sector. Pemindaian ini mencoba menemukan dan memperbaiki kesalahan fisik pada hard disk itu sendiri, dan dapat memakan waktu lebih lama untuk menyelesaikannya.
  • Untuk memeriksa kesalahan file dan kesalahan fisik, pilih "Perbaiki kesalahan sistem file secara otomatis" dan "Pindai dan coba pemulihan bad sector."

  • Klik "Mulai".

    (Ini bisa memakan waktu cukup lama).

    Anda dapat memeriksa bad sector tanpa prompt, tetapi tidak dapat memperbaiki saat Windows sedang digunakan.

Chkdsk tidak dapat memperbaiki sistem file atau kesalahan Sektor saat di Windows, baik jadwalkan chkdsk saat restart atau gunakan Windows RE dan prompt perintah untuk menjalankan chkdsk.

Cara menggunakan WinRE, restart PC, gunakan F8 berulang kali untuk mendapatkan layar Windows Advanced, lalu pilih "Repair Your computer", ini akan memuat WinRE, jika Anda mendapatkan pesan tentang RE menemukan beberapa kesalahan dan ingin memperbaiki, katakan TIDAK, biarkan RE terus memuat, ketika dimuat pilih "perbaiki komputer Anda" lagi, lalu pilih Command prompt dari daftar Alat dan jalankan perintah ini

chkdsk / r C:

.

Ekstrak * .7z di Ubuntu, dengan menjaga struktur direktori tetap utuh

Bagaimana cara mengekstrak file * .7z di Ubuntu sambil menjaga struktur direktori tetap utuh?

Berikut panduan ini , Saya mencoba

7za e My.7z file.

Tetapi itu tidak menyimpan file dalam struktur direktori SAMA dengan semua file di direktori induk.

Terima kasih.

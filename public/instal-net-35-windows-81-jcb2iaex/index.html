Instal .NET 3.5 Windows 8.1

Saya telah mencoba menginstal .NET Framework 3.5 di komputer Windows 8.1 tanpa hasil. Saya mencoba pendekatan dasar mengunduh dan menginstalnya dari MSDN yang gagal mengatakan tidak dapat terhubung ke internet.

Kemudian saya mencoba berhenti seperti yang dijelaskan di beberapa posting forum:

Dism.exe /Online /Enable-Feature /FeatureName:NetFX3 /All /Source:E:\sources\sxs /LimitAccess

tapi saya terus mendapatkan kesalahan:

Error: 0x800f081f
The source files could not be found.

dan di file log saya menemukan sesuatu yang aneh:

Encountered an unknown option "source" with value "E:\sources\sxs"

Saya juga mencoba menyalin folder sxs ke c:\temp\sxs dan meletakkannya di sumbernya.

Berdasarkan posting blog ini tidak mungkin untuk menginstal .NET 3.5 setelah paket bahasa diinstal. Apakah ada solusi yang diketahui untuk ini sekarang? Denmark adalah satu-satunya bahasa yang tersedia. Saya tidak dapat beralih kembali ke bahasa Inggris atau bahkan menginstal paket bahasa Inggris.

Ada cara yang diketahui untuk melakukannya. Anda memiliki 2 cara. Jika Anda memiliki DVD dengan bahasa kedua yang telah Anda instal, salin folder SxS dari DVD ke-2 ini ke folder c:\temp\sxs atau download paket MUI dari bahasa yang Anda instal dan ekstrak LP.cab juga di folder c:\temp\sxs dengan luaskan .exe yang merupakan bagian dari Windows.

Sekarang Anda dapat menjalankan perintah DISM dan Windows akan menemukan file untuk kedua bahasa tersebut.

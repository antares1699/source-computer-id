Batch-file, menjalankan aplikasi secara berurutan vs. sekaligus

Jika saya membuat file batch (batch.bat) dan mengetik ini di:

"C:\Program Files\WinRAR\WinRAR.exe"
"C:\Windows\System32\notepad.exe"

Aplikasi yang dimaksud akan berjalan secara berurutan, pertama WinRAR akan dimuat dan setelah ditutup, Notepad akan memuat berikutnya secara berurutan.

Q: Bagaimana cara sintaks file batch untuk menjalankan semua aplikasi pada saat yang sama, saya ingin WinRAR dan Notepad terbuka pada saat yang bersamaan.

Gunakan start sebagai gantinya:

start "" "command here"

Edit: Jangan lewatkan sepasang tanda kutip kosong pertama, ini adalah judul proses / jendela.

start <title> <command> <parameters>

Lihat start /? untuk rincian lebih lanjut.

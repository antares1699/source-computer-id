Motherboard tidak mengenali Model CPU yang baru dipasang?

WINDOWS 7 ULTIMATE

Saya memiliki motherboard nview abit nf-m2 dengan AMD Athlon 64 X2 5200+ sebelumnya yang berfungsi dengan baik dan keren. Baru-baru ini saya menukar beberapa hal termasuk CPU ke Athlon 64 X2 5050e Brisbane, sistem masih berjalan dengan baik, tetapi cpu dilaporkan sebagai model Athlon 64 X2 Tidak Dikenal. Saya sedang memikirkan flash bios, tetapi ingin mendapatkan beberapa pendapat terlebih dahulu tentang alasan apa lagi yang mungkin terjadi?

Hal-hal yang saya coba:

1: Menjalankan pembaruan driver dari pengelola perangkat, tetapi tidak menemukan apa pun. Mengatakan K8 adalah hal terbaik yang bisa ditemukan windows dan sudah diinstal.

2: Menghapus penginstalan driver chipset CPU dari pengelola perangkat dan memulai ulang.

Jika ada yang bertanya-tanya, ini adalah item yang ditukar:

500W PSU -> 300W PSU

65W CPU -> 45W CPU

HDD 7200RPM - HDD 5400RPM

4x1 GB DDR2 -> 2x2 GB DDR2

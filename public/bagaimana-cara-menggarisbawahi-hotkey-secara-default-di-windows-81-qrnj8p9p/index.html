Bagaimana cara menggarisbawahi hotkey secara default di Windows 8.1?

Dengan Windows 8.1 Pro, bagaimana cara mengonfigurasi sistem sedemikian rupa sehingga (menu, tombol perintah, dll.) Hotkey ditampilkan secara default dan bukan hanya setelah menekan secara eksplisit Alt ?

Larutan

Ada opsi khusus yang dapat Anda aktifkan dengan mengikuti langkah-langkah ini:

  1. tekan Menang + u untuk membuka Pusat Kemudahan Akses .
  2. Klik Jadikan keyboard lebih mudah digunakan tautan.

  3. Aktifkan Garis bawahi pintasan keyboard dan tombol akses opsi, dan tekan baik .

    Opsi ini menyoroti tombol akses untuk kontrol di kotak dialog sehingga Anda dapat berpindah-pindah dengan cepat. Untuk info lebih lanjut, lihat Pintasan keyboard .

    Sumber: Ubah pengaturan keyboard

Sebelum Windows Vista, opsi serupa tersedia di pengaturan tampilan. Setelah desain ulang antarmuka pengguna, opsi dipindahkan ke dialog aksesibilitas.

Anda dapat menggunakan solusi di atas, atau menggunakan solusi yang sudah ada sejak sebelum dikumpulkan ⃰ ⃰ di 1 tempat dan diberi nama "Kemudahan Akses" (yaitu tempat yang mereka buat untuk meletakkan apa pun yang terkait dengan "kemudahan akses") di Win7 (mungkin Vista) dan yang lebih baru Buka Explorer.

Klik Tools pada menu, lalu Folder Options. Di bagian tengah halaman itu Anda akan melihat ⃰ ⃰Folder Options ***, klik itu. Di bagian tengah, Anda akan melihat bahwa biasanya Anda dapat memilih gaya garis bawah jika Anda memilih:

⚫ Sekali klik untuk membuka item (tunjuk untuk memilih)

⚫ Garis bawahi judul ikon yang konsisten dengan browser saya

⚪ Garis bawahi judul ikon hanya jika saya menunjuk ke sana.

⚪ D̲klik dua kali untuk membuka item (klik sekali untuk memilih)

⃰ ⃰ - Sebagian besar fitur yang ada di halaman Kemudahan Akses, sudah ada di versi jendela sebelumnya - ini hanya menjadi titik pengumpulan bagi mereka untuk membuat menu baru dan memungkinkan mereka "titik" baru dalam pemasaran mereka daftar fitur.

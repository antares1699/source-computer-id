Jika Anda hanya perlu membaca gambar, 7-zip harus melakukannya, percaya atau tidak. UI pengelola file harus memungkinkan Anda melihat konten atau mengekstraknya, alat baris perintah 7z.exe akan memungkinkan Anda mengekstraknya juga.

Bukti karena ini mendapat suara negatif meskipun benar, di changelog .

Jika Anda benar-benar perlu menulis, atau membuat sistem berpikir itu adalah disk sungguhan, saya kira OSFMount mungkin satu-satunya pilihan. Ini gratis (seperti pada bir gratis, jika Anda ingin perbedaan itu dibuat).

ImDisk akan me-mount file image partisi kecuali Anda membuatnya menggunakan format "khusus" (opsi '-s') dari ntfsclone.

Jika partisi tersebut diklon menggunakan format khusus, Anda perlu mengkloningnya lagi menggunakan ntfsclone.

contoh: ntfsclone -ro newfile.img oldfile.img

Ini mungkin bukan pengaturan yang ideal, tetapi inilah yang saya gunakan:

Mempersiapkan:

  1. Tempatkan proyek Anda di direktori XAMPP Anda, mis. xampp\htdocs\<project>:

  2. Pastikan bahwa Apache sedang berjalan dan dapat diakses melalui browser Anda:

  3. Buka Run > Run Configurations... dan buat / edit browser di bawah Web Browser:

  4. Tambahkan Browser Executable, set Server ke Use base URL (mis. http://localhost/), dan pilih Append project name:

Demonstrasi:

Sekarang Anda dapat membuka halaman saat ini di server XAMPP Anda dengan mengklik Lari ikon tarik-turun dan memilih browser:

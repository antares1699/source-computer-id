Bagaimana cara mengaktifkan resolusi 1920 x 1200 untuk VGA?

saya sudah Monitor BenQ T241WA . Ini menggunakan VGA.

Dengan kabel biasa sudah OK dan saya dapat mengatur resolusi asli 1920x1200 Sekarang saya membutuhkan kabel yang lebih panjang. Ketika saya mencolokkan kabel VGA lagi, saya mendapatkan resolusi maksimal 1600x1200.

Saya ingin memaksa adaptor video untuk menggunakan 1920x1200, bukan 1600x1200. Bagaimana aku melakukan itu?

OS: Windows 7 64 bit
Video: NVidia GeForce 9500 GT
Monitor: Generic Non-PnP Monitor

Dengan monitor kabel asli didefinisikan sebagai Generic PnP Monitor. PnP tidak bekerja melalui kabel yang lebih panjang.

Saya bertanya tentang bagaimana memaksa Windows 7 menggunakan resolusi yang berbeda dari default max 1600x1200.

Di Linux saya dapat mengedit xorg.conf. Apa yang ditawarkan Windows sebagai alternatif?

Kabel standar VGA tidak memiliki batasan resolusi, tetapi kabel yang lebih panjang akan menangkap lebih banyak noise hingga batas resolusi.

Jadi satu-satunya hal yang membatasi resolusi adalah bandwidth yang dapat ditransfer kabel, jika kabel tidak dapat mentransfer bandwidth yang cukup untuk membuat layar 1920x1200 lengkap, satu-satunya hal yang dapat Anda lakukan adalah membeli kabel dengan kualitas yang lebih baik atau menggunakan yang lebih pendek.

Periksa kapan komputer dihidupkan untuk Windows

Di komputer Windows, apakah ada cara untuk melihat tanggal dan waktu komputer di-boot / dihidupkan / dihidupkan? Sesuatu seperti perintah waktu operasi Linux.

Tapi saya lebih suka ini menampilkan waktu yang tepat saat dinyalakan, bukan waktu yang telah berlalu sejak saat itu. Misalnya jika Anda menyalakannya hari ini pada pukul 14:30, itu akan menampilkan sesuatu seperti 9/16/2016 14:30.

Ringkasan:

Menarik untuk Anda:

  • systeminfo (menunjukkan waktu absolut)

    systeminfo | findstr "Waktu Boot Sistem:"

  • statistik bersih (menunjukkan waktu absolut)

    stasiun kerja statistik bersih

    atau

    server statistik bersih

    tergantung pada peran komputer Anda

  • Powershell

    Dapatkan-WinEvent -ProviderName eventlog | Dimana-Objek {$ .Id -eq 6005 -atau $ .Id -eq 6006}

  • WMI

    wmic os mendapatkan lastbootuptime

Tidak menarik untuk Anda:

  • Task Manager (menunjukkan waktu berlalu sejak itu)
  • Utilitas uptime (membutuhkan alat terpisah, acara berlalu sejak waktu itu)

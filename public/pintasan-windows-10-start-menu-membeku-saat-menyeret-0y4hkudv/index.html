Pintasan Windows 10 Start Menu membeku saat menyeret

Masalah ini dimulai baru-baru ini dan bekerja dengan sempurna sebelumnya.

Ketika saya membuka menu mulai dan saya mencoba menyeret salah satu pintasan saya (di panel kanan), ini akan meredupkan semua ikon lainnya dan kemudian ikon yang saya pilih untuk diseret akan menghilang dari bawah kursor dan menghilang. Semua ikon tetap redup dan tidak lagi dapat diseret ... Dan pintasan yang saya pilih untuk diseret hilang. Ini hampir seolah-olah fungsionalitas untuk menyeret dan melepaskan ikon macet dalam keadaan 'redup'.

Saat saya keluar dan masuk lagi, semuanya kembali normal (ikon belum bergerak).

Saya masih dapat menggunakan ikon 'redup' seperti biasa ... Saya dapat mengklik untuk meluncurkannya serta mengklik kanan untuk menampilkan opsi pintasan ... Tetapi saya tidak dapat menyeretnya ...

Edit: Berikut video yang menunjukkan apa yang terjadi: https://youtu.be/lPJ00qahu5I

Ini terlihat seperti masalah driver NVidia (375.57) terbaru. Saya mengalami masalah yang sama hari ini dan bahkan menginstal ulang Windows 10. Tidak masalah dengan driver 375.57, tetapi setelah saya memperbarui Windows ke Anniversary, saya tidak dapat lagi menyeret pintasan apa pun. Sekarang baru saja kembali ke 373.06 dengan uninstall lengkap (DDU) dan Start Menu berfungsi dengan baik lagi tanpa peredupan dan hang ini!

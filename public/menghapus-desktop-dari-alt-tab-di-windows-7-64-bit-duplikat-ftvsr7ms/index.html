Menghapus desktop dari Alt + Tab di Windows 7 64-bit [duplikat]

Saya memiliki PC Windows 7 64-bit. Ketika saya menekan Alt + Tab atau WinKey + Tab, itu menunjukkan jendela saat ini termasuk desktop, tetapi saya tidak ingin menampilkan desktop di sana. Saya mencoba solusi yang diberikan untuk Windows 7 32-bit, tetapi tidak berhasil. Apakah ada peretasan registri untuk mengatasi masalah ini?

Mencoba:

1

Buka Peninjau Suntingan Registri dengan mengetik regedit di kotak pencarian Start Menu atau tekan + R dan ketik regedit .

Di Editor Registri, navigasikan ke:

HKEY_CURRENT_USER> Perangkat Lunak> Microsoft> Windows> CurrentVersion> Penjelajah

Di folder Explorer ("subkunci"), buat nilai DWORD baru, bernama AltTabSettings .

(Klik kanan> Baru> Nilai DWORD (32-bit)).

Klik dua kali nilai DWORD yang baru dibuat, ubah nilainya menjadi 1 dari pada 0 , dan pilih Heksadesimal .

Tekan OK, dan nikmati fitur Alt-Tab klasik.

2

Pilihan lain adalah menggunakan perangkat lunak Switcher .

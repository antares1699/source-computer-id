Ubah kata sandi windows XP melalui (2) remote desktop

Saya ingin mengubah kata sandi workstation Win XP saya dari rumah, tetapi saya harus melakukannya melalui koneksi Remote Desktop (dari rumah, dan ini adalah mesin kerja saya).

Masalahnya adalah, saya membuat koneksi melalui mesin windows lain .... jadi

MESIN RUMAH -> SERVER TENGAH -> BEKERJA

Sekarang jika saya menekan CTRL+ALT+DEL (saat masuk ke WORKSTATION), saya memiliki akses ke fungsi perubahan kata sandi mesin rumah saya (yang normal). Jika saya menekan CTRL+ALT+END (karena itu adalah jalan pintas untuk pengelola tugas Remote Desktop), saya memiliki akses ke fungsi perubahan kata sandi server tengah.

(Menggunakan solusi panel kontrol yang disebutkan di bawah tidak berfungsi, lihat komentar saya tentang itu.)

Tapi bagaimana saya bisa mencapai fungsi ubah kata sandi di workstation?

Terima kasih sebelumnya!

RDP ke SERVER TENGAH, lalu ke WORKSTATION. Minimalkan WORKSTATION dan tampilkan keyboard di layar menggunakan "kemudahan akses". Pastikan ini disetel ke "selalu di atas". Sekarang maksimalkan WORKSTATION Anda dan tekan ctrl-alt-end pada keyboard di layar dari MIDDLE SERVER. Ini akan memunculkan layar opsi di mana Anda dapat mengklik ubah kata sandi

Jika Anda adalah administrator di jaringan, Anda dapat melakukannya di prompt perintah:

pengguna bersih nama_pengguna baru

Saya menguji ini ... akan menambahkan lebih banyak jika saya dapat menemukan cara untuk melakukannya untuk pengguna domain non-administratif.

Hal lain, yang mungkin tidak membantu Anda, tetapi jika Anda memiliki akses ke Outlook Web Access 2010 (2007 jika administrator Anda membuat modifikasi khusus), Anda dapat mengubah kata sandi dalam opsi.

P.S. Saya menemukan ini: Ini cukup keren untuk hanya mengunci workstation: rundll32.exe user32.dll, LockWorkStation

Catatan: Komentar di bawah ini merupakan tanggapan atas jawaban awal saya, yang telah diubah.

Bagaimana cara mengetahui file mana yang dibuka pengguna di komputer bersama?

Di Windows, katakan saya masuk ke Control Panel->Computer Management dan melihat file
Computer Management (Local)->Shared Folders->Sessions.

Saya tahu bahwa jika ada orang yang muncul dalam daftar ini, kemungkinan besar mereka membuka file di komputer saya, dilambangkan dengan
# Open Files kolom. Tapi pertanyaan saya adalah, bagaimana cara mengetahui file mana yang dibuka orang itu?

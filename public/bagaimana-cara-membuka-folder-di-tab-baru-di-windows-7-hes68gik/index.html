Bagaimana cara membuka folder di tab baru di windows 7?

Saya adalah pengguna jendela dari awal, tetapi baru-baru ini saya mulai mengerjakan Linux dan ada satu fitur yang sangat saya sukai Linux (tentu saja masih banyak lagi tetapi saya tidak menggunakan semuanya secara menyeluruh).

Fitur yang saya suka adalah tabbing di sistem yaitu kita dapat membuka folder di tab baru sehingga kita tidak mengganti jendela lagi.

Sekarang saya kembali ke jendela lagi tetapi saya merindukan tab fitur di jendela sangat buruk. Saya harus membuka setiap folder di jendela baru yang tidak saya suka.

Mungkinkah melakukan ini di windows jika memungkinkan daripada bagaimana.

Windows tidak mendukung folder tab secara asli, tetapi ada beberapa tambahan atau penggantian Explorer yang menambahkan fitur ini. Saya menggunakan QTTabBar dan bekerja dengan sangat baik.

QTTabBar tidak menggantikan Windows Explorer. Sebaliknya, itu menyuntikkan modifikasinya ke dalam prosesnya. Ada banyak opsi konfigurasi. Anda dapat, misalnya, mengonfigurasi Explorer untuk selalu bekerja di satu jendela yang menangkap semua jendela lainnya ke dalam tab.

QTTabBar asli tidak dipertahankan lagi, tapi ada garpu yang masih dalam pengembangan aktif dan menawarkan dukungan yang ditingkatkan untuk versi Windows yang lebih baru.

Sebaik QTTabBar dan Semanggi ada juga BrightExplorer .

Tidak keduanya Semanggi maupun BrightExplorer adalah open source, jadi saya akan tetap menggunakan QTTabBar .

Perhatikan bahwa kapan gronostaj.dll menyebutkan Asli dan Garpu , dia melakukannya dengan cara yang salah.

Pengembang aslinya adalah Quizo , tapi Quizo tidak terlalu cepat memperbarui tanda QTTabBar untuk bekerja dengan Windows 7, jadi Paul Accisano memulai proyek ini sourceforge pada tahun 2010. Kapan Quizo memulai devlopment aktif lagi pada tahun 2012, Paul Accisano berhenti mengembangkan, dengan versi terakhir yang dipublikasikan v1.5.0.0 Beta 2 ).

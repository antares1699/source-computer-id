Menjalankan aplikasi Android Market di simulator

(Terkait dengan, tetapi tidak sama dengan ini ).

Saya dulunya adalah pengguna Android tetapi beralih ke iPhone. Ada beberapa aplikasi yang ingin saya terus gunakan, tetapi gagasan untuk membawa ponsel kedua, atau bahkan harus mengisi satu, tidak terlalu menarik. Mereka adalah aplikasi gratis sehingga pembajakan tidak menjadi masalah di sini, dan saya telah mengunduh APK secara sah dari Market. Apakah mungkin untuk melakukan sideload APK kembali ke komputer saya dan menjalankannya di emulator Android?

  1. Unduh SDK Android dari http://developer.android.com/sdk/index.html

  2. Buka zip Android SDK dan pindahkan ke ~ / .android

  3. Di terminal, ketik ~/.android/tools/android update sdk

  4. Instal Alat Platform SDK Android, dan Platform SDK Android pilihan Anda.

  5. Pada tab Perangkat Virtual, buat AVD baru. Saya menyebut milik saya "AndroidSandbox", API Level 10, dengan kartu SD 2GB.

  6. Di Android SDK Tools, klik Start untuk meluncurkan emulator Android.

  7. Setelah emulator terbuka, instal file APK apa pun di dalamnya. Di terminal, ketik: export PATH=$PATH:~/.android/tools/:~/.android/platform-tools/; adb install myapp.apk

Anda dapat menggunakan adb di lingkungan sistem. Klik kanan "My Computer" pilih "Properties". Pilih "Pengaturan sistem lanjutan" dari menu sisi kiri. Kemudian di bawah "Variabel sistem" temukan "Jalur" lalu klik "Edit ...". Kemudian di "Nilai variabel" pergi ke bagian paling akhir dari baris, paling akhir. Dan tambahkan "; C: \" ditambah path di mana Anda memiliki adb yang terletak di sistem Anda. Lalu klik "OK".

Jika Anda hanya ingin membuat variabel lingkungan sederhana untuk Java64, cari "JAVA_HOME" dan tambahkan jalur Java64. Atau klik "Baru ..." lalu di bawah "Nama variabel" tambahkan "JAVA_HOME". Kemudian di "Nilai variabel" tambahkan jalur Java64 Anda. yaitu "C: \ Program Files \ Java \ jre7"

Sekarang temukan 1Mobile Market .apk secara online. Kemudian di terminal ketik "adb install" lalu jalur di mana Anda memiliki 1Mobile Market .apk. Setelah ini dipasang, Anda dapat melakukan semua pemasangan di masa mendatang melalui "1Mobile Market". Saya sarankan juga mengambil "Appbrain". Dan juga menginstalnya melalui "adb install" lalu path ke appbrain.apk. Mulai sekarang Anda dapat menginstal semua program dari emulator. Tidak perlu lagi menggunakan "adb install" selain beberapa waktu acak. Di mana mungkin dibutuhkan. Serta "Appbrain" akan selalu memastikan Anda menginstal yang terbaru.

Beberapa saran lain jika Anda memiliki intel dengan virtualisasi. Aktifkan itu untuk emulasi Anda. Serta jika Anda dapat mengelola lebih banyak RAM dll dll. Sekarang jangan meningkatkan segalanya dan berpikir itu keren. Uji lingkungan contoh terlebih dahulu dengan sistem Anda. Kemudian kerjakan dari sana. Jika melambat, kembalikan ke keadaan sebelumnya. Contoh di bawah ini pada sistem yang lambat membutuhkan waktu tidak lebih dari 2 menit untuk memulai. Pada sistem yang jauh lebih lambat mungkin sekitar 5 menit atau lebih. Ini akan menjadi salah satu yang tercepat untuk dipasang. Anda juga dapat mempertimbangkan untuk mendapatkan "RAM Booster" dari "1Mobile Market".

Sekarang buka terminal setelah Anda mendapatkannya

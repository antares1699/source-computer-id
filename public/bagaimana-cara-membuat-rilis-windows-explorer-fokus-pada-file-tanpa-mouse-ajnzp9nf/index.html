Bagaimana cara membuat rilis Windows Explorer fokus pada file tanpa mouse?

Saya selalu mencoba menggunakan komputer saya tanpa menyentuh mouse dan meletakkan kedua tangan di atas keyboard setiap saat. Ini benar-benar meningkatkan produktivitas saya karena saya tidak perlu memindahkan tangan saya ke mouse untuk tugas-tugas umum. Namun, saya menemukan kasus di mana saya tidak dapat melakukannya hanya dengan keyboard.

Saat menavigasi melalui Windows Explorer, katakan saya memiliki fokus pada file, tetapi saya ingin membuka menu konteks seolah-olah tidak ada file yang dipilih, seperti ketika Anda ingin membuat folder baru atau tindakan lain yang tidak terkait dengan file tersebut. se. Apa yang saat ini saya lakukan adalah menggerakkan tangan saya ke mouse, dan klik kanan untuk membuat menu itu muncul.

Bagaimana saya mencapai ini tanpa menggunakan mouse?

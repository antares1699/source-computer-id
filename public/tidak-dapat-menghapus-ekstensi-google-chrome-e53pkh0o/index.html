Tidak dapat menghapus ekstensi Google Chrome

Saya mencoba menghapus beberapa ekstensi Google Chrome dari sistem saya, buka menu Ekstensi, klik ikon Sampah, konfirmasikan bahwa saya ingin menghapus ekstensi ini, dan ekstensi tersebut dihapus untuk sementara, tetapi ekstensi muncul kembali saat saya memulai ulang sistem. Ini sudah berlangsung cukup lama, saya tidak tahu kenapa.

Berikut adalah daftar ekstensi yang saya coba hapus: Freemake Video Converter, Google Docs, Percayakan, Avast! Antivirus dan Be Limitless.

Apa yang salah? Apa yang harus saya lakukan?

Mungkin Anda telah menyinkronkan Google Chrome Anda dengan contoh Google Chrome di perangkat lain, dan sebagai tambahan untuk menyinkronkan bookmark Anda, Anda juga secara tidak sengaja memilih untuk menyinkronkan ekstensi Anda. Jadi saat pertama kali Anda membuka Google Chrome di hari yang baru, Chrome mencari data perangkat master untuk disinkronkan, lalu memulihkan ekstensi Anda dengan menyinkronkan data di komputer Anda dengan data dari perangkat master.

Periksa pengaturan sinkronisasi Anda untuk melihat apakah sudah benar. Menu Chrome ⋮ -> Pengaturan -> Orang-orang . Jika Anda menyinkronkan ekstensi, Anda dapat mengubahnya dari sana.

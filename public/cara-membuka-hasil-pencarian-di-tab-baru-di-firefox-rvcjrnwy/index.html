Cara membuka hasil pencarian di tab baru di Firefox

Ketika saya melakukan pencarian dari kanan atas kotak pencarian Firefox, hasilnya selalu terbuka di tab saat ini. Ini menjengkelkan karena memuat hasil ke tab yang saya pantau, dan kemudian saya harus kembali untuk menemukan halaman saya lagi setelah saya selesai dengan urusan pencarian. Saya ingin Firefox selalu membuka hasilnya di tab baru. Apa itu mungkin?

4
Mehper C. Palavuzlar — Sumber
  • Bukan jawaban, tapi solusi: tekan Ctrl-T / Command-T terlebih dahulu, untuk mendapatkan tab baru sebelum mencari? (Lalu tekan Ctrl-L / Command-L untuk melompat ke bidang lokasi, dan Tab untuk masuk ke bidang pencarian.)
    — Arjan
  • Arjan: Atau tekan saja Alt + Enter untuk mencari. Bekerja dengan bilah alamat juga.
    — Joey

1 Jawaban

7
7

Arahkan ke about:config di bilah alamat, filter ke browser.search.openintab dan setel ke benar . Ini akan membuka hasil pencarian dari kotak pencarian di tab baru.

Anda juga bisa menekan Alt + Memasukkan saat mencari untuk membukanya di tab baru.

Timpa Bitlocker MBR

Saya tidak yakin persis bagaimana saya menimpanya, tetapi saya menulis gambar floppy ke floppy disk (menggunakan USB floppy drive), dan entah bagaimana saya secara tidak sengaja menulis floppy disk ke hard disk utama saya yang saya miliki Windows 7 Enterprise x64 diinstal dengan Bitlocker diaktifkan. Sekarang ketika saya boot, saya mendapatkan kesalahan "Disk non-sistem atau kesalahan disk." Apakah ada cara untuk memperbaikinya?

Saya telah membuat cadangan tingkat rendah jika sesuatu yang saya coba menyebabkan kerusakan lebih lanjut, dan inilah yang telah saya coba sejauh ini ...

  • bootrec /fixmbr - tidak berfungsi, sekarang dikatakan "tabel partisi tidak valid" ketika saya mencoba untuk boot.
  • bootrec /fixboot - tidak berfungsi, perintah mengembalikan "elemen tidak ditemukan."
  • bootrec /rebuildbcd - tidak berfungsi, perintah mengembalikan "Total penginstalan Windows yang teridentifikasi: 0"

MEMPERBARUI: Setelah menjalankan bootrec /rebuildbcd saya menggunakan diskpart untuk membuat ulang partisi dan menetapkan huruf drive padanya. Saya kemudian dapat menggunakan repair-bde D: R: -rp 123456-* untuk mulai memulihkan file. Saat ini masih berjalan lancar, jadi kita akan melihat cara kerjanya besok.

Jika ada orang lain yang melakukan ini, inilah yang harus saya lakukan, langkah demi langkah, untuk memulihkan PC saya ke keadaan semula.

  1. Boot dari Windows Instal disk dan pilih Pemulihan, dan masuk ke prompt perintah.
  2. Jalankan bootrec /fixmbr (mungkin tidak diperlukan)
  3. Jalankan diskpart dan buat ulang partisi menggunakan bendera quick. Semoga Anda tahu seberapa besar partisi Anda sebenarnya.
    • Pastikan untuk menandai System Reserved (jika ada) atau partisi C: aktif.
  4. Pasang drive dengan ukuran yang sama atau lebih besar dengan jumlah partisi yang sama yang perlu Anda pulihkan.
  5. Jalankan repair-bde C: Z: -rp 111111-222222-[...] di mana C: ada di disk yang rusak dan Z: di disk baru yang akan Anda gunakan untuk menyimpan sementara file yang dipulihkan / didekripsi. Anda harus melakukan ini untuk setiap partisi. Mudah-mudahan, Anda memiliki kata sandi pemulihan BitLockers.
  6. Setelah repair-bde selesai, verifikasi file Anda telah dipulihkan. Kemudian gunakan quick untuk menyalin file kembali ke disk lama yang rusak.
    • MEMPERBARUI: Sesuatu yang tidak saya sadari saat itu adalah bendera /mir tidak menyalin kepemilikan atau keamanan. Yang seharusnya saya jalankan adalah robocopy /e /copyall Z:\ C:\ /xf hiberfil.sys /xj /xjf /xjd.
  7. Gunakan opsi Windows install disk Startup Recovery untuk memperbaiki boot loader & partisi Cadangan Sistem.
  8. Reboot PC dan Windows akan dimuat.
  9. Periksa apakah titik persimpangan Anda (misalnya C:\Documents and Settings menunjuk ke C:\Users) telah dipulihkan dengan benar sebagai titik persimpangan. Punyaku tidak. Bisa jadi di mana saya lupa bendera /xj /xjf /xjd saat pertama kali saya menjalankan robocopy atau bisa jadi repair-bde. Saya condong ke arah yang terakhir.
    • Jika Anda perlu membuat ulang titik persimpangan Anda, saya menggunakan junction /s /q C:\ > junctionpoints.txt dari SysInternals pada penginstalan lain yang berfungsi untuk mencantumkan semua titik persimpangan Windows.
    • Kemudian saya menggunakan mklink /j "C:\Documents and Settings" "C:\Users" untuk membuat ulang titik persimpangan pada PC yang rusak.
  10. Aktifkan kembali BitLocker
  11. Buka Panel Kontrol dan lakukan Perbaikan di Microsoft Office. Ini mungkin tidak diperlukan untuk Anda, tetapi Outlook tidak berfungsi sampai saya melakukan ini.

Catatan: Setelah Anda melakukan ini memulihkan PC, ID mesin unik Anda akan berubah, jadi beberapa perangkat lunak berlisensi yang menggunakan ID unik ini (misalnya DameWare Remote Support dalam kasus saya) mungkin perlu diaktifkan kembali.

Jika Anda belum pernah menggunakan Bitlocker, ini akan cukup mudah. Namun, dengan Bitlocker dalam gambar itu tidak hanya jauh lebih sulit tetapi juga berpotensi tidak mungkin.

Ada tiga masalah yang perlu Anda selesaikan.

1) Anda harus memulihkan MBR dan bootloader yang valid (atau bootloader GPT + EFI jika Anda menggunakannya). Ini cukup mudah - perintah yang telah Anda lakukan di atas akan melakukannya.

2) Anda harus membuat ulang tabel partisi Anda secara manual.

3) Anda harus mengambil salinan kunci enkripsi master Bitlocker Anda. Ini BUKAN kata sandi.

Bagian 3 adalah yang sulit. Namun, jika Anda memiliki pengaturan Windows 7 default dari partisi boot 100-300MB diikuti oleh partisi OS utama Anda, Anda mungkin beruntung.

Selanjutnya, proses di atas adalah apa yang perlu Anda lakukan jika Anda ingin memulihkan disk ke status kerja. Jika Anda dapat mengakses disk dari komputer lain, memulihkan data ke drive kedua adalah proses yang lebih mudah daripada memperbaikinya secara in situ.

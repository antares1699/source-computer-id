Bagaimana cara menemukan shortcut apa yang dijalankan melalui Start-> Run

Ketika saya menekan Start-> Run dan mengetik "Chrome", tidak ada yang terjadi (tidak ada proses "chrome.exe" baru yang diluncurkan, tidak ada kesalahan, tidak ada).

Ketika saya menekan Start (tanpa "Run"), ketik Chrome dan jalankan, itu berfungsi.

Bagaimana cara menemukan pintasan apa yang dijalankan melalui Start-> Run? Saat saya membuka prompt perintah dan menulis "Chrome.exe", ia tidak mengenalinya sebagai program yang dapat dijalankan.

Itu karena folder Chrome tidak ada dalam variabel PATH. Variabel lingkungan ini memutuskan lokasi mana yang dapat Anda rujuk tanpa jalur yang ditentukan. Alasan ia bekerja hanya dengan membuka awal adalah karena ia hanya mencari Chrome di menu program alih-alih mencari file yang dapat dieksekusi itu sendiri. Biasanya Chrome yang dapat dieksekusi terletak sebagai "C: \ Users \ username \ AppData \ Local \ Google \ Chrome \ application \ chrome.exe" ("% LOCALAPPDATA% \ Google \ Chrome \ application \ chrome.exe")

Bersama dengan jalur yang terdaftar oleh Madsen, jalan pintas yang direferensikan terdaftar di Registri.

Jalankan regedit, lalu navigasikan ke:

Jalur Komputer \ HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft Windows \ CurrentVersion \ App

(ini mungkin sedikit berbeda pada 7, tapi saya ragu).

Anda perlu memastikan ada entri (Default) dan Jalan untuk chrome.exe di sini, dengan bidang data (untuk keduanya) menjadi jalur file tempat program berada.

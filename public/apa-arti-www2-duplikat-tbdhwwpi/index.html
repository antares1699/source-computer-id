max — Sumber
  • Dari WIKI: WWW2 dan WWW3 adalah nama host atau subdomain, biasanya digunakan untuk mengidentifikasi serangkaian situs web yang terkait erat dalam suatu domain, seperti www.example.com, www2.example.com, dan www3.example.com; seri dapat dilanjutkan dengan nomor tambahan: WWW4, WWW5, WWW6 dll. Secara tradisional, situs web semacam itu adalah mirror yang digunakan untuk penyeimbangan beban server. Dalam beberapa kasus, nama host tertentu mungkin dikaburkan, menciptakan kesan bahwa pengguna melihat subdomain "www", meskipun mereka sebenarnya melihat situs mirror.
    — Rohlik

Peternakan web dan load balancing

Nama-nama seperti www1, www2, www3 sering digunakan untuk anggota dari server web penyeimbang muatan server web untuk situs web populer di mana beban kerjanya terlalu banyak untuk satu server atau di mana beberapa server digunakan untuk memberikan kontinuitas layanan di peristiwa kegagalan satu server.

Contoh cara kerjanya

Server front-end di www.example.com awalnya menerima permintaan dan mengalihkan browser ke server yang tersedia di farm, melakukannya dengan mengembalikan respons pengalihan HTTP ke browser, browser kemudian mencoba kembali permintaannya menggunakan URL yang dialihkan disediakan.

Itu hanya sebuah nama

Mungkin saja beberapa bagian dari situs web besar ditangani oleh server yang berbeda dan daripada menamai server shop.example.com pemegang saham.example.com jobs.example.com mereka menamainya www1.example.com www2.example.com dll.

Catat hanya perubahan dalam keluaran robocopy

Saya tidak dapat menemukan cara untuk mencatat hanya perubahan dalam output dari eksekusi robocopy, yaitu: file yang dianggap baru, direktori baru, file baru, file ekstra, atau direktori ekstra.

Output default yang saya miliki sekarang terdiri dari daftar semua direktori (tersentuh atau tidak tersentuh) yang dianggap dalam penyalinan. Saya ingin menekan yang tak tersentuh.

Bantuan tampaknya tidak menunjukkan cara untuk melakukan ini.

Jika Anda perhatikan dengan cermat dokumentasi atau robocopy /?, Anda akan menemukan tombol /NDL.

Menentukan bahwa nama direktori tidak akan dicatat.

Dengan ini, Anda tidak akan memiliki keluaran pada nama individu untuk direktori (yang dapat dianggap sebagai jenis penanda kemajuan). Tapi "nama file" akan menyertakan jalurnya.

Jadi misalnya Anda akan mendapatkan output berikut:

        *EXTRA Dir        -1    C:\Temp\test\
          *EXTRA File                  0        C:\Temp\test\test.txt
100%        Newer                     72        C:\Temp\test.txt
Bagikan

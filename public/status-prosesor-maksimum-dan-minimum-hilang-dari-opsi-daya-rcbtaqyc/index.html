Status Prosesor Maksimum dan Minimum hilang dari Opsi Daya

Restart laptop saya setelah torrent semalaman, dan ketika exhaust fan terasa sangat hangat, saya perhatikan pengaturannya hilang, CPU saya berjalan dengan kecepatan maksimal tidak peduli profil apa yang saya pilih (bahkan mengatur CPU untuk selalu rendah di BIOS don tidak bekerja). Saya sudah mencoba mengatur ulang profil daya, dan masih tidak ada yang terjadi. Mungkin peningkatan di tempat akan menyelesaikan masalah ini, tetapi adakah cara untuk memulihkannya tanpa menghabiskan beberapa jam menunggu prosesnya? Atau apakah kegagalan opsi BIOS untuk memaksa ke daya rendah (yang sebelumnya berfungsi dengan baik, bahkan pada distro Linux yang tidak mendukung pelambatan daya) merupakan tanda dari beberapa masalah perangkat keras?

EDIT: Saya memeriksa CPU-Z dan ternyata pelambatan CPU saya masih berfungsi normal. Bahkan pengaturan ulang tidak mengubah pengaturan status (saya mengaturnya untuk selalu menggunakan yang terendah pada penghemat daya, dan maksimum pada 80% pada keseimbangan), sehingga pengaturan tersebut entah bagaimana tidak muncul, dan tidak terpengaruh oleh pengaturan ulang juga. Pengelola tugas masih melaporkan seolah-olah CPU saya bekerja pada frekuensi maksimum.

Ini terjadi pada saya setelah saya menginstal Hyper-V di komputer saya. Rupanya, Anda tidak dapat menggunakan opsi lanjutan ini dan Hyper-V secara bersamaan. Saya pergi ke Control Panel > Programs > Programs and Features > Turn Windows features on or off dan menonaktifkan Hyper-V.

Setelah reboot, semuanya kembali normal.

dari 1 hingga 0 seperti yang disarankan dalam ini posting forum.

Saya mengalami masalah serupa di Windows 8 Pro. Frekuensi CPU saya tidak berubah selama pekerjaan normal (setidaknya ditunjukkan pada 100% di Resource Monitor dan di Task Manager) dan bahkan tidak berubah saat mengubah profil daya. Juga opsi frekuensi CPU minimum dan maksimum di profil daya tidak ada.

Saya menemukan bahwa saya menyebabkan ini dengan memaksa Windows menggunakan kedua inti saat OS dimulai. Jadi mengembalikan pengaturan ke default di msconfig , semuanya kembali normal setelah restart ( "jumlah prosesor" perlu dicentang ).

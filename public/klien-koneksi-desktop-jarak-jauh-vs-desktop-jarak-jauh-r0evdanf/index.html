Klien Koneksi Desktop Jarak Jauh Microsoft untuk Mac Versi 2.1.1

Diterbitkan 5/5/2011.

Klien Koneksi Desktop Jarak Jauh untuk Mac 2.1.1 memungkinkan Anda menghubungkan dari komputer Macintosh Anda ke komputer berbasis Windows dan kemudian bekerja dengan aplikasi dan file pada komputer berbasis Windows.

...

Versi Sistem Operasi: Mac OS X versi 10.5.8 atau versi yang lebih baru dari Mac OS.

catatan: Klien Koneksi Desktop Jarak Jauh Microsoft untuk Mac (versi 2.1.1) tidak didukung untuk digunakan dengan Mac OS X v10.7 (Lion) atau yang lebih baru .

Linux: gunakan find untuk menemukan file teks?

Apakah ada cara untuk menentukan ke find bahwa saya hanya menginginkan file teks (dan bukan file biner)? Grep memiliki opsi untuk mengecualikan file biner, jadi saya pikir find mungkin memiliki fitur serupa, tetapi saya tidak dapat menemukannya.

Itu karena grep melihat konten file.

find . -exec sh -c "file -b --mime-type {} | grep -q '^text/'" \; -print

find dapat menemukan file berdasarkan metadatanya, tetapi tidak dapat memeriksa isinya. Namun, Anda dapat memasangkannya dengan file:

find .git -type f -exec file -iN0 {} + |
awk -F '\0' '$2 ~ /^: text\// {print $1}'

(Anda dapat menghapus bagian | awk ... jika Anda juga menginginkan informasi yang lebih detail.)

Os X - Mengapa saya perlu mengaktifkan WiFi untuk menentukan lokasi terdekat

Mengapa saya perlu mengaktifkan WiFi untuk menentukan lokasi terdekat saya?

Untuk beberapa alasan, mac saya tidak akan mendapatkan waktu terbaru dari server NTP jika saya terhubung ke Ethernet. Apa yang mungkin berbeda antara nirkabel saya dan Ethernet saya?

Memperbarui : Saya membiarkan Mac saya duduk selama 30 detik dengan jendela ini terbuka, wifi mati, dan ethernet terpasang. Lokasi dan tanggal tidak akan diperbarui. Mengaktifkan wifi menyebabkan lokasi dan tanggal ditentukan dalam hitungan detik.

Apple menggunakan database lokasi jaringan WiFi untuk menentukan perkiraan lokasi Mac Anda.

Mengutip Artikel dukungan Apple tentang Layanan Lokasi :

Layanan Lokasi memungkinkan aplikasi dan situs web untuk mengumpulkan dan menggunakan informasi berdasarkan lokasi komputer Anda saat ini. Anda harus memberikan izin Anda sebelum aplikasi atau situs web dapat menggunakan data lokasi Anda.

Perkiraan lokasi Anda ditentukan menggunakan informasi dari jaringan Wi-Fi lokal, dan dikumpulkan oleh Layanan Lokasi dengan cara yang tidak mengidentifikasi Anda secara pribadi.

Mereka dulu mengandalkan Skyhook Wireless database, tetapi sejak 2010 Apple menjalankannya sendiri , berdasarkan setidaknya sebagian data yang dikumpulkan dari iPhone (yang memiliki GPS dan WiFi dan karenanya dapat menetapkan perkiraan lokasi fisik ke titik akses WiFi).

Berhenti mengirim email ke saya sendiri saat memilih Balas Semua di Outlook

Perusahaan saya telah menyiapkan setiap akun email dengan alias serveral. Akun default saya adalah firstL@mycompany.com. Saya juga memiliki:

  • first.last@mycompany.com
  • flast@mycompany.com
  • first_last@mycompany.com

Masalahnya adalah: Akun saya menggunakan default untuk masuk dan menerima email, firstL@mycompany.com. Namun, orang akan mengirim email ke salah satu alias saya dan ketika saya mengklik "balas ke semua", email tersebut dikirim kembali kepada saya .

Bagaimana cara mengonfigurasi Outlook (2010) untuk mengabaikan balasan ke semua sehingga tidak mengirim balasan saya sendiri ke akun saya sendiri?

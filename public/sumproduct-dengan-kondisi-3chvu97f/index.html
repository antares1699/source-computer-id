SUMPRODUCT dengan kondisi

Saya memiliki 2 kolom B = "Poin" dan C = "Nilai" Dan saya mencoba menemukan rumus untuk menjumlahkan produk dari semua poin * nilai untuk nilai> = 60.

Saya mencoba formula ini dengan banyak formula lainnya, tetapi tidak berhasil:

SUMPRODUCT((B2:B100)*(C2:C100)*(C2:C100=">60"))

Keluaran adalah: 0

Ada yang punya saran?

4
Orel Eraki — Sumber
  • Saran saya adalah membuat kolom baru dan meletakkan rumus ini =IF(C2>=60;C2*B2;0) di dalamnya, buatlah ini untuk semua baris dan pada akhirnya jumlahkan semuanya.

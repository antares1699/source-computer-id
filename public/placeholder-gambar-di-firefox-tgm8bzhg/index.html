Placeholder gambar di firefox

Kadang-kadang saya menggunakan koneksi seluler saat bepergian; jika saya terhubung di area yang tidak tercakup dengan baik oleh jaringan seluler, saya mengalami masalah.

Saya mencoba menonaktifkan pemuatan gambar dari panel preferensi. Masalahnya adalah Firefox tidak memuat gambar, tetapi tidak ada placeholder, dan tata letak halaman web dapat berubah.

Apakah ada cara untuk mengaktifkan placeholder untuk gambar?

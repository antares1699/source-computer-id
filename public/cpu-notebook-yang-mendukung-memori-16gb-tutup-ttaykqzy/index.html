CPU Notebook yang mendukung memori 16GB? [Tutup]

Saya berencana untuk membeli notebook baru, dan di antara persyaratan yang saya pikirkan bahwa saya menginginkan memori 8GB sekarang, dan kemungkinan untuk ditingkatkan ke 16 nanti.

Saya telah melakukan beberapa pencarian dan saya telah mengetahui bahwa merek Intel Core i3 / i5 / i7 adalah kekacauan total , dan bahwa versi seluler dari Core i3 / i5 / i7 (Arrandale atau Clarksfield) tidak memiliki atau sedikit kecocokan yang sesuai dengan versi desktop (Lynnfield, Clarkdale, Bloomfield, ...).

Ternyata cpus desktop Core i7 bisa mendukung memori 24GB, tapi itu tidak membantu saya karena tampaknya tidak ada ponsel CPU mendukung lebih dari 8GB (bahkan tidak core i7 edisi ekstrim ).

Saya menyerah untuk mencari informasi di situs web AMD; Saya tidak dapat menemukan apa pun yang disebutkan tentang keterbatasan memori sama sekali.

Apakah benar-benar hanya memori 8GB jumlah maksimum memori yang mungkin untuk notebook saat ini ???

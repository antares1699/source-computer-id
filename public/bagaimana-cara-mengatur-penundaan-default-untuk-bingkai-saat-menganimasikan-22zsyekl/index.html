Bagaimana cara mengatur penundaan default untuk bingkai saat menganimasikan?

Saya mengimpor banyak bingkai sebagai lapisan dan melakukan Filter → Animasi → Optimalkan (untuk GIF). Saya mengubah lapisan dan penundaan diterapkan ke setiap nama lapisan. Dalam kasus saya, ini adalah "(100ms)". Namun saya ingin penundaan ini menjadi 50 md, jadi saya dapat menyetel penundaan khusus untuk beberapa frame secara manual.

Anda dapat menentukan penundaan default saat mengekspor gif Anda.

Karena ini sedikit lebih rumit, inilah yang Anda lakukan, untuk mengatur ulang semua penundaan pada setiap lapisan :

  1. Ekspor animasi Anda sebagai GIF. Dalam Dialog Ekspor GIF, setel Delay between frames where unspecified hingga 50ms dan mengekspornya ke <yourname>.gif, misalnya test.gif.

  2. Centang kotak di bawah ini berlabel: Use delay entered above for all frames. Ini akan mengatur ulang penundaan pada semua lapisan ke 50ms.

  3. Impor GIF Anda lagi. Semua lapisan sekarang harus menyatakan penundaan 50ms. Anda sekarang dapat mengatur penundaan khusus pada lapisan tertentu. Ekspor GIF lagi dengan menggunakan Ekspor... .

  4. Pastikan Anda menghapus centang pengaturan sebelumnya. Kali ini Anda tidak mengatur apa pun dan mengekspornya dengan pengaturan default.

Detail tambahan apa pun, dijelaskan dalam dokumentasi di bawah Format File .

Bagaimana cara kerja AutoRecover?

Terkadang AutoRecover merusak saya dan mematikan Word dengannya, tetapi saya tetap menyimpannya setiap menit atau lebih.

Saya selalu bertanya-tanya bagaimana cara kerjanya. Apakah itu memiliki direktori sementara yang disimpan sebagai format rahasia? Atau sesuatu yang lain?

Saya tidak berpikir ada format rahasia, tetapi Word hanya menyimpan file ke lokasi sementara setiap beberapa menit.

Anda dapat mengubah frekuensi penyimpanan pemulihan otomatis - secara default adalah 10 menit.

Lokasi file pemulihan otomatis juga dapat dikonfigurasi. Secara default itu

C: \ Dokumen dan Pengaturan [pengguna] \ Data Aplikasi \ Microsoft \ Word \

di Windows XP dengan Word 2007.

Opsi diakses dari Opsi Word> Simpan (lagi di Word 2007)

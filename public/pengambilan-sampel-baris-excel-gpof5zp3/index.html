Anda dapat menambahkan kolom angka acak, menggunakan rumus =RAND( ) dengan tidak ada apa pun di antara tanda kurung:

Salin rumus ke seluruh kumpulan data sehingga semua baris terisi. Kemudian Anda dapat menggunakan 'Number Filters' dan memilih 'Top 10':

Ubah nilainya menjadi '100' dan filter akan memilih nilai acak '100 Teratas'. Hasilnya akan menjadi sampel acak dari 100 baris. Mungkin ada cara yang lebih baik atau lebih mudah, tapi itu akan berhasil.

Ini sepenuhnya didasarkan pada batesan pendekatan yang sangat baik (yang diotomatiskan oleh makro ini) .

Katakanlah kita mulai dengan satu kolom dan, mengikuti saran batesan, menambahkan kolom =RAND():

Dalam contoh sederhana saya, tabel hanya memiliki 18 entri dan saya ingin memilih 3 sembarangan.

Menjalankan makro pendek:

Sub PickThree()
    Dim sh As Worksheet, rng As Range

    Set sh = ActiveSheet
    Set rng = sh.Range("A1:B19")

    If sh.AutoFilterMode = False Then rng.AutoFilter Field:=2
    If ActiveSheet.FilterMode Then ActiveSheet.ShowAllData

    Calculate
    rng.AutoFilter Field:=2, Criteria1:="3", Operator:=xlTop10Items
End Sub

menghasilkan:

Setiap kali makro dijalankan kembali, kumpulan baru tiga baris acak dipilih!

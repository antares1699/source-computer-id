Jalankan Safari dari baris perintah dengan parameter URL

Saya mencoba membuka Safari (di OS X 10.8) dari terminal dengan menjalankan yang berikut ini:

open -a /Applications/Safari --args "http://www.example.com"

Safari terbuka dan mencoba menavigasi ke url:

file:///http:/www.example.com

Adakah cara agar Safari membuka browser di url yang ditentukan?

Catatan: Saya tidak dapat melakukan open http://www.example.com karena saya tidak dapat dijamin bahwa Safari adalah browser default. (Dan sebelum saya ditanya mengapa saya tidak menggunakan browser default - saya melakukan otomatisasi pengujian khusus browser.)

Sepertinya jawabannya adalah:

open -a safari http://www.example.com

Menurut halaman manual untuk terbuka (1) , buka -a <applicationName file> membuka file yang diberikan dengan aplikasi tertentu. Oleh karena itu, ini sepertinya sintaks yang paling tepat.

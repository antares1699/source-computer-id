Cara memindahkan folder default untuk virtualbox headless

Saya (mencoba) menggunakan Virtualbox di komputer jarak jauh - mode tanpa kepala.

Saat membuat mesin virtual, dengan

VBoxManage createvm --ostype Windows7 --name MYBOX --register

itu membuatnya dalam mengerikan $HOME/VirtualBox VMs/ saya. Sungguh menjengkelkan memiliki ruang di jalan. ; (

Bagaimana cara mengubah jalur default dari "VirtualBox VMs" ke sesuatu yang lain dengan commandline?

Anda juga dapat mencoba menggunakan mklink /J junction untuk folder di drive lain, yang mungkin menyelesaikan masalah Anda tetapi pertama-tama Anda harus memindahkan "VM VirtualBox" ke drive lain dan kemudian membuat tautan ke direktori itu.

Contoh:

mklink /J "%HOMEPATH%/VirtualBox VMs/" "%SOME_DRIVE%\%YOUR_MOVED_VMs_FOLDER%"

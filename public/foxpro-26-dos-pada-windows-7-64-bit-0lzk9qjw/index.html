FoxPro 2.6 DOS pada Windows 7 64-bit

Saya mendukung perusahaan yang memiliki aplikasi FoxPro for DOS 2.6 (FPD) yang sangat tua dan sangat penting.

Untuk berbagai alasan perusahaan tidak mengadaptasi / memigrasi aplikasi mereka, yang ironisnya, telah berjalan lebih baik di bawah Windows XP (dan 32-bit Win7) karena OS memungkinkan fitur baru seperti jaringan yang lebih andal, pencetakan terdistribusi, integrasi email. Sayangnya untuk perusahaan ini, sebagian besar mesin baru sekarang hadir dengan versi Windows 7 64-bit, yang tidak kompatibel dengan aplikasi FPD mereka.

Saya tahu kali ini tulisannya ada di dinding: satu-satunya solusi jangka panjang adalah memigrasi aplikasi mereka. Tapi saya ingin tahu apakah ada yang bisa menyarankan jalur alternatif sementara, yang tidak melibatkan:

  1. menurunkan versi Windows 64-bit ke 32-bit, atau

  2. menjalankan aplikasi di XP 32-bit virtual

Mac: Bagaimana cara memasukkan path direktori lengkap dalam dialog save As?

Misalkan saya mengedit dokumen / file di aplikasi seperti MS office, editor teks atau emacs, dll dan saya ingin menyimpan dokumen di direktori tertentu.

Di Windows / Linux, menggunakan alias saya, dll., Saya akan dengan cepat menavigasi ke direktori tempat ia harus disimpan dan menyalin jalur direktori ke clipboard dan kemudian menempelkan jalur direktori ke dalam dialog "simpan sebagai".

Di Mac, tidak ada bidang tempat saya dapat menempelkan jalur direktori lengkap di Dialog "Simpan sebagai". Jika Windows dan Linux menyediakan cara untuk melakukannya, mengapa saya tidak menemukan hal yang sama di mac. Di mac, saya harus menavigasi mouse dari direktori favorit atau sistem file dan itu membutuhkan banyak waktu.

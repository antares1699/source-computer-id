Apakah Anda menyadari di Panel Preferensi Keamanan Anda dapat meminta kata sandi untuk membangunkan dari screen saver, tetapi secara default tidak memerlukan otentikasi?

Kemudian Anda dapat mengaktifkan screen saver dengan metode Anda, atau AppleScript ini:

do shell script "open /System/Library/Frameworks/ScreenSaver.framework/Versions/A/Resources/ScreenSaverEngine.app"

Anda juga dapat menambahkan kunci Keychain dan menggunakan AppleScript yang saya temukan ini sini .

activate application "SystemUIServer"
tell application "System Events"
    tell process "SystemUIServer"
        repeat with i from 1 to number of menu bar items of menu bar 1
            tell menu bar item i of menu bar 1
                click
                try
                    if name of menu item 1 of front menu is "Lock Screen" then
                        click menu item "Lock Screen" of front menu
                        exit repeat
                    end if
                end try
            end tell
        end repeat
    end tell
end tell

Skrip ini memerlukan Akses Universal -> Aktifkan akses untuk perangkat bantu yang diperiksa.

Inilah solusinya dari Petunjuk Mac OS X, tetapi tampaknya perlu menginstal JackSMS terlebih dahulu, apa pun itu. Maka Anda bisa melakukannya

tell application "JackSMS" to set lock screen to true
tell application "JackSMS" to set lock screen to false

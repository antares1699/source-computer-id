Apakah ada cara untuk mengekstrak file dari folder c: \ windows \ csc di Windows 7 professional?

Solusi Kerja

Di baris perintah:

vssadmin List Shadows

Ini akan menampilkan daftar salinan bayangan Anda (bayangan folder CSC tidak akan dikunci). Cari dan salin alamat asli bernomor tertinggi.

Kedua, kami membuat tautan ke salinan bayangan:

mklink /D C:\LatestShadow \\?\GLOBALROOT\Device\HarddiskVolumeShadowCopy417\

Bagian paling kanan adalah alamat asli yang Anda salin sebelumnya

Selanjutnya gunakan robocopy untuk mengeluarkan semua file dari cache

mkdir C:\CSC_copy
robocopy C:\LatestShadow\Windows\CSC C:\CSC_copy /E /COPY:DT

Anda akan mendapatkan salinan dari semua file cache di C: \ CSC_Copy.

Gunakan perintah ini untuk memberikan akses penuh

cd %systemroot%
takeown /f csc /r /a /d y
icacls csc /grant Everyone:(F) /t /l /q

lalu reboot.

kemudian mengambil alih kepemilikan file di C:\CSC_Copy tersebut

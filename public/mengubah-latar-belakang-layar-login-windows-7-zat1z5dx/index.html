Mengubah latar belakang layar login Windows 7?

Apakah ada suatu tempat di dalam Windows 7 di mana saya dapat mengubah gambar latar belakang yang lumpuh untuk layar login?

Saya tahu ada banyak peretasan / utilitas / perubahan Regedit dan hal-hal seperti itu untuk mengubahnya menjadi 'cara peretas', tetapi apakah Windows 7 tidak memiliki cara bawaan untuk mengubahnya? Atau bahkan membuatnya terlihat seperti login windows XP standar (yaitu: user / pass / domain)?

Membaca hasil pencarian, mereka benar-benar membiarkan Anda tanpa peretasan, tetapi masih sedikit peretas. Dari situs ini :

Gunakan pengaturan Kebijakan Grup berikut di Windows 7

  1. Mulai Editor Kebijakan Grup (gpedit.msc)
  2. Pergi ke cabang berikut:
    Computer Configuration | Administrative Templates | System | Logon
  3. Setel opsi berikut ke Diaktifkan
    Always use custom logon background

Langkah selanjutnya, dikatakan, adalah menyalin latar belakang Anda ke C:\Windows\System32\oobe\info\backgrounds\ sebagai backgroundDefault.jpg.

Cara bawaan tidak diekspos ke pengguna melalui antarmuka yang bagus, tetapi cukup sederhana lakukan sendiri - Saya tidak akan menggambarkannya sebagai "cara peretas".

Jika Anda menginginkan gaya pengguna / sandi / domain, Anda bisa mendapatkannya dengan mengaktifkan pengaturan kebijakan grup untuk "jangan tampilkan nama pengguna terakhir". Jika Anda memiliki Windows 7 Pro, Ultimate atau Enterprise, Anda dapat mengubahnya dengan membuka gpedit.msc dari kotak Run, dan menavigasi ke Konfigurasi Komputer> Pengaturan Windows> Pengaturan Keamanan> Kebijakan Lokal> Opsi Keamanan, dan menemukan "Logon interaktif: lakukan tidak menampilkan nama pengguna terakhir ". Jika Anda memiliki Windows 7 Home, Anda harus mengubahnya di registri dengan membuka HKLM\Software\Microsoft\Windows\CurrentVersion\Policies\System dan mengubah nilainya dari 0 ke 1.

Sejauh yang saya tahu seharusnya ada "cara resmi" untuk mengubahnya, tetapi mereka masih membutuhkan file registri untuk dijalankan. Ada juga bermacam-macam program, hacks, dll. Untuk melakukannya, tetapi berikut sedikit panduannya mereka mengklaim sebagai "cara resmi" untuk mengubahnya:

  1. Unduh file registri dari sini . Ekstrak file dan klik dua kali pada file untuk mengaktifkan fitur ini (Metode alternatif: klik kanan, pilih Gabung).

  2. Sekarang, buka folder berikut:

    C:\Windows\System32\oobe † (“C” adalah drive Instalasi Windows 7 Anda)

  3. Di sini, buat folder baru bernama Info. Sekali lagi, buat folder baru folder di dalam folder Info dan ubah namanya menjadi backgrounds.

  4. Salin file gambar favorit Anda ke folder ini dan ubah namanya menjadi backgroundDefault.jpg ‡ . Perhatikan gambar itu harus dalam format JPEG dan ukurannya kurang dari 256 KB.

  5. Kamu selesai. Mulai ulang sistem Anda dan lihat perubahannya.
    __________
    † "Oobe" adalah singkatan dari "out-of-the-box experience".
    ‡ Nama file mungkin juga dalam bentuk backgroundDefaultDIMENSIONS.jpg; mis., background960×1280.jpg atau background1024×1280.jpg. Sistem akan mencari dulu file yang namanya sesuai dengan dimensi layar; jika tidak ditemukan, itu jatuh kembali ke file backgroundDefault.jpg generik. Ini sangat berguna jika Anda menyiapkan distribusi yang akan dipasang di berbagai platform (atau live CD atau flash drive yang dapat di-boot yang akan digunakan pada berbagai platform), atau jika Anda sering mengganti monitor, atau resolusi pada layar Anda.

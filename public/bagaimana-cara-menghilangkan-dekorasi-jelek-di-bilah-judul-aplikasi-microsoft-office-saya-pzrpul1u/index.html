Bagaimana cara menghilangkan dekorasi jelek di bilah judul Aplikasi Microsoft Office saya?

Saya dulu memiliki Office 2013 tetapi memutakhirkan ke Office 360. Saya ingin menyingkirkan hal-hal abu-abu yang Anda lihat di kanan atas gambar. Ada ide? Saya menemukan tema kantor, tetapi saya hanya dapat mengubah warna panel.

di bawah "Personalisasikan salinan Microsoft Office Anda"

Pilihan 2:

Tab File

Akun

"Latar Belakang Kantor" berubah menjadi "Tanpa Latar Belakang"

CATATAN

Coba dulu di Microsoft Word. Saya telah melihat beberapa bug aneh yang dilaporkan tidak muncul di beberapa aplikasi lain sampai Anda mencobanya di Word terlebih dahulu.

JIKA Tak satu pun dari yang di atas berhasil - coba perbaikan ini

Ini mengasumsikan Anda memiliki hak istimewa Admin

  1. Buka Editor Registri Anda (jalankan regedit.exe)

  2. Buka Kunci "HKEY_CURRENT_USER \ Software \ Microsoft \ Office \ 15.0 \ Common \ General".

  3. Di sisi kanan, cari DWORD "DisableBackgrounds" (Jika tidak ada, buatlah)

  4. Setel sebagai 1 untuk menonaktifkan semua latar belakang sepenuhnya.

CATATAN: Secara teori, pengaturan ke 0 akan mengembalikan menu untuk memungkinkan Anda memilih latar belakang / tanpa latar belakang di Office dan tidak memiliki kunci sama sekali membuat menu tidak muncul. Namun, tampaknya hasil dari melakukan ini tidak konsisten.

  1. Keluar dari Editor Registri

Mungkin perlu me-restart komputer Anda.

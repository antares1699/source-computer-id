PowerShell dapat mencetak file apa pun, menggunakan tindakan yang sama seperti kata kerja Print ditentukan untuk tipe file.

Perintah PowerShell untuk digunakan adalah Mulai-Proses , dan idenya hanyalah memanggil file itu sendiri dengan kata kerja Print.

Sintaksnya sangat sederhana:

Start-Process "path-to-file" -Verb print

Saya mengujinya dengan file PDF, dan file tersebut dicetak dengan benar.

Contoh yang lebih rumit dapat ditemukan di dokumentasi:

Contoh 2: Cetak file teks

Contoh ini memulai proses yang mencetak file C: \ PS-Test \ MyFile.txt.

Start-Process -FilePath "myfile.txt" -WorkingDirectory "C:\PS-Test" -Verb Print

Apakah perangkat keras grafis Modern UI Windows 8 dipercepat?

Layar mulai benar-benar licin dan halus dan saya cukup terkesan dengan betapa tajamnya segala sesuatu tampak, tetapi saya melihat sesuatu yang menarik ...

Bahkan ketika terhubung melalui RDP, animasinya tampak sama mulus dan tajam, yang tampak aneh mengingat bagaimana grafik yang dipercepat perangkat keras di-nerf melalui koneksi RDP.

Apakah perangkat keras Modern-UI dipercepat? Jika tidak, bagaimana mereka membuatnya begitu apik tanpa membebani sistem secara tidak perlu?

Pertanyaan bagus! Saya juga bertanya-tanya tentang ini.

Ternyata ada perbaikan besar-besaran ke RDP dengan Windows 8. Anda dapat membaca tentang peningkatan grafis ke RDP di blog Remote Desktop dari Microsoft.

Untuk menjawab pertanyaan Anda secara khusus, berikut adalah kutipan dari blog ini:

Di Windows Server 2012 dan Windows 8, desktop lokal lengkap adalah disusun dan ditampilkan pada host dalam sesi jarak jauh sebelum gambar yang dihasilkan dikodekan dan dikirim oleh RemoteFX. Dengan kata lain, RemoteFX didasarkan pada rendering sisi host. Hasilnya, penuh pengalaman desktop desktop Windows 8 serta semua aplikasi antarmuka pengguna (UI) secara konsisten dikirim dari jarak jauh paling tinggi kualitas dengan 32 bit per piksel (BPP). Untuk menyeimbangkan pengalaman pengguna dengan encoding dan biaya bandwidth, frame rate target dalam sesi jarak jauh adalah 30 frame per detik (FPS).

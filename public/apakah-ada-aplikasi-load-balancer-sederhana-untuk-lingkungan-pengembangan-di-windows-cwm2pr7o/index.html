Apakah ada aplikasi Load Balancer sederhana untuk lingkungan pengembangan di Windows?

Apakah ada aplikasi Load Balancer sederhana untuk pengembangan di Windows? Saya menjalankan sepasang instance JBoss 5.x dalam sebuah cluster pada satu mesin. Biasanya, konfigurasi ini memuat diimbangi oleh penyeimbang beban perangkat keras yang bagus tetapi saya bertanya-tanya apakah ada perangkat lunak sederhana untuk mengaktifkan penyeimbangan beban di lingkungan dev Eclipse saya.

Pada dasarnya, misalnya, saya ingin load balancer yang berjalan pada port 11111 yang round-robins antara 2 instance JBoss yang berkerumun pada port ssl 8443 dan 8543. (atau port http jika itu tidak memungkinkan)

Saya tahu bahwa Glassfish memiliki penyeimbang beban bawaan tetapi saya tidak dapat menggunakan Glassfish.

Satu ide yang saya miliki adalah mencoba menyiapkan instance Tomcat terpisah dengan aplikasi web "penyeimbang". Saya mencobanya sekarang ... tidak yakin apakah itu akan berhasil ... dan ini adalah pengaturan yang rumit dan saya berharap ada sesuatu yang sangat mudah.

Anda dapat menggunakan Nginx

Instal nginx dan tambahkan file konfigurasi:

http {
  upstream myproject {
    server 127.0.0.1:8000;
    server 127.0.0.1:8001;
  }

  server {
    listen 80;
    server_name localhost;
    location / {
      proxy_pass http://myproject;
    }
  }
}

Alih-alih load balancing antar port, mengapa tidak mengikat setiap instance ke alamat IP loopback tertentu. Seluruh jaringan 127.0.0.0/8 adalah loopback (bukan hanya 127.0.0.1). Jika Anda mengkonfigurasi proses Anda untuk mengikat ke 127.0.0.2 dan 127.0.0.3 (atau apa pun yang Anda inginkan) maka Anda dapat melakukan IP round robin. Ini dapat dilakukan dengan DNS jika Anda mau.

Anda juga dapat menulis pembungkus yang mengikat ke IP / port dan akan meneruskan koneksi secara acak ke salah satu server, dan hanya meneruskan data.

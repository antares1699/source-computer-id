Bagaimana cara kerja tombol "validasi" CPU-Z?

Saya menjalankan CPU-Z di Windows 7 Virtual PC di Macbook Pro. Tab "memory" dan "SPD" kosong. Namun, ada tombol validasi di sebelah tombol OK di bagian bawah.

Ketika saya mengklik tombol "validasi", akan muncul dialog di mana saya memberikan alamat email dan mengarahkan saya ke ini tautan tapi saya tidak tahu cara mengaktifkan CPU-Z. Saya telah menggunakannya di masa lalu tanpa masalah. Mengapa tidak berhasil?

Cpu-Z terjebak dalam mesin virtual dan tidak memiliki akses ke perangkat keras sebenarnya. Saya tidak yakin apa yang Anda harapkan. Virtual RAM adalah konstruksi khusus perangkat lunak dan tidak memiliki waktu siklus, latensi, dan sebagainya. Mesin Anda bahkan tidak harus memiliki RAM DDR2, itulah yang dilaporkan oleh perangkat lunak VM.

Bagikan

Mereka berwarna abu-abu karena tidak masuk akal di mesin virtual, karena ini bukan perangkat keras fisik dan karena itu tidak tahu pengaturan waktu, latensi, dll ...

Fitur "validasi" mengacu pada Validator CPU-Z fasilitas, yang memungkinkan seseorang untuk membuktikan bahwa pembacaan CPU-Z mereka asli (diperlukan saat berbagi informasi tentang overclock, stabilitas perangkat keras, dll ...) dan tidak dipalsukan.

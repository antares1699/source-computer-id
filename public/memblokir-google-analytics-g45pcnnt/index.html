Jika Anda ingin tidak terlihat oleh google analytics dari beberapa situs - cara terbaik untuk melakukannya adalah dengan menimpa URL Google Analytics di file host windows:

Buka file ini di notepad:

C:\Windows\System32\drivers\etc\hosts

Anda akan melihat string seperti itu:

# localhost name resolution is handled within DNS itself.
#   127.0.0.1       localhost
#   ::1             localhost

tambahkan string ini ke akhir file:

   209.85.225.113  localhost
   209.85.225.138  localhost    
   209.85.225.139  localhost 
   209.85.225.100  localhost 
   209.85.225.101  localhost
   209.85.225.102  localhost

yang akan membuat semua panggilan Google Analytics menjadi lokal, yaitu bukan ke server GA yang sebenarnya

Saya baru saja memperhatikan bahwa Chrome (setidaknya v.12) tidak meminta http://www.google-analytics.com/ga.js melainkan mengetahui IP langsung. memblokir www.google-analytics.com di file host berfungsi untuk Firefox, browser lain, wget langsung, tetapi Chrome masih membuka file ga.gs (Saya membersihkan cache-nya). Mungkin ada terlalu banyak nasihat di Internet tentang cara menggunakan file host, jadi mereka "meretas" metode ini untuk menghindari cengkeraman mereka, setidaknya di browser mereka sendiri

Anda memiliki beberapa opsi untuk ekstensi yang dapat Anda gunakan. Ini dia, dalam urutan yang paling mungkin berhasil, dan paling tidak mungkin:

  • Ghostery Perpanjangan. Itu akan memblokir semua web-bug terkenal - itu artinya Analytics.

  • AdBlock / Adblock Plus / uBlock Perpanjangan. Tambahkan kode berikut ke filter manual Anda:

    http://www.google-analytics.com/ga.js

  • ScriptSafe Perpanjangan. Blokir skrip.

Anda dapat menggunakan plugin browser. Tidak sesulit itu karena JS analitik Google selalu berupa URL yang sama.

ATAU

Anda dapat menyetel google-analytics.com ke 127.0.0.1 di file host Anda. Secara efektif memblokir semua google analytics js. Jika Anda mencegahnya terus melakukan ping ke server dan mencoba mendapatkan dokumen, Anda dapat mengarahkan google-analytics.com ke server yang dikenal yang akan menyajikan file js kosong.

Catatan: Saya yakin url Google Analytics adalah google-analytics.com/ga.js Meskipun saya bisa saja salah.

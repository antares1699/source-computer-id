Secara resmi dan teknis dari Microsoft - Ya.

Dari OEM Anda - Tidak Diketahui - Anda harus bertanya kepada mereka.

Ketika Anda membeli edisi OEM Windows, Anda terutama membeli kunci dan bukan medianya .... Saya membeli banyak dan sangat jarang menyentuh CD, saya biasanya hanya menempelkan stiker pada casing dan kemudian menggunakan server PXE saya untuk menginstal OS, saya kemudian memberikan medianya kepada klien.

Lisensi mengatakan edisi mis. Windows Vista Ultimate, tetapi tidak tertulis x86 atau x64, dan lisensinya valid pada kedua edisi. Seperti halnya semua komoditas, disk x86 dan x64 dibeli dan dijual oleh distributor pada waktu yang berbeda sesuai dengan tingkat stok yang berarti terkadang ada perbedaan harga yang besar, saya bertanya kepada Microsoft tentang hal ini dan diberi tahu bahwa saya dapat membeli yang termurah dan menggunakan yang mana saja. platform yang saya butuhkan selama itu adalah edisi yang benar.

Karena itu, saya masih hanya dapat memberikan klien saya disk asli yang saya beli - jika mereka memerlukan penginstalan ulang, saya biasanya mengungkapkan kepada mereka dan melakukannya dari disk saya sendiri ... Saya tidak tahu apakah OEM besar akan melakukannya membantu Anda melakukan ini.

Sekali lagi, secara teknis dan legal, Anda dapat ... Jika OEM mengizinkan Anda terserah mereka. Ini dengan asumsi Anda mendapatkan salinan OEM dari perusahaan besar seperti Dell, HP, dll. Jika Anda pergi ke perusahaan kecil seperti perusahaan saya yang membeli disk OEM mereka dari distribusi, Anda dapat menggunakan salinan dan penggunaan OEM standar lainnya. kunci MU.

Periksa semua file dari cabang, tanpa mengubah HEAD atau file yang sudah ada

Apa cara terbaik untuk melakukan checkout / menggabungkan semua file dari cabang ke direktori kerja, yang saat ini tidak ada di sana, tetapi tanpa mengubah status repositori, HEAD, atau file yang sudah ada?

Apakah ini mungkin dengan perintah satu baris?

git checkout dapat mengambil jalur sebagai argumen, yang, jika diberikan, meninggalkan HEAD saja, dan hanya memeriksa jalur tersebut ke direktori kerja dan indeks Anda, jadi Anda bisa menggunakan:

git checkout branchname -- .

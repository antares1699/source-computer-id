Bagaimana telnet melewati firewall windows?

Saya mengaktifkan klien Telnet di Windows 7 dan langsung dapat terhubung ke server meskipun Windows Firewall diaktifkan.

Bagaimana cara kerjanya? Bukankah seharusnya firewall memblokir koneksi keluar sampai saya mengizinkannya secara eksplisit?

Apakah aturan firewall default dibuat saat Anda mengaktifkan klien telnet di Win 7 dan jika demikian, aturan firewall mana yang harus saya cari?

Saya tidak dapat menemukan aturan apa pun di firewall untuk telnet saya.

Penafian: Ini sebenarnya bukan masalah yang saya coba selesaikan. Saya hanya mencoba memahami bagaimana telnet dapat berkomunikasi tanpa saya membuka firewall untuk program tersebut.

Komunikasi keluar umumnya diizinkan secara default. Namun, jika Anda menjalankan Telnet Server, koneksi masuk akan diblokir secara default. Pengecualiannya adalah jika penginstal Telnet Server membuka porta sebagai bagian dari penginstalan.

http://technet.microsoft.com/en-us/library/cc732283(WS.10).aspx

Firewall standar di Windows tidak memblokir koneksi keluar. Jika Anda ingin memblokir klien telnet (atau apa pun dengan koneksi keluar), Anda dapat menggunakan file Windows Firewall dengan Keamanan Lanjutan untuk menyiapkan aturan keluar.

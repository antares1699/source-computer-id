Linux pada Windows10 - mungkin menjalankan rasa topi merah?

Saya baru saja mengatur ubuntu di windows 10. Wow! Keren!

Kami menjalankan CentOS di tempat kerja. Saya sangat ingin memiliki Fedora, atau beberapa rasa Red Hat sebagai subsistem Linux saya.

Apakah ini mungkin?

Jawabannya adalah: Fedora seharusnya tersedia di MS Store, tetapi ditahan karena beberapa alasan.

Utas yang terkait dengan ini adalah:

  • sini: https://github.com/Microsoft/WSL/issues/2584
  • Tautan untuk memilih di reddit, twitter dan suara pengguna ada di sini: https://github.com/Microsoft/WSL/issues/3288

Kami membutuhkan dukungan fedora. Penggunaan WSL kami ditangguhkan hingga Fedora atau CentOS ada di MS Store. (Semua pengembang kami adalah pengguna Win10 Pro.)

Ada dua cara untuk memasukkan Linux ke WSL: Dapatkan dari Microsoft Store, atau buat sendiri.

Ketika distro Linux tertentu tidak tersedia di Windows Store, Anda dapat membuatnya sendiri. Untuk mengetahui caranya, dan mengunduh alat yang dibutuhkan, kunjungi:

  • "Membuat Distro Linux Kustom untuk WSL" . Docs . Microsoft (27 Maret 2018).

Tentu saja, Microsoft telah mengumumkan bahwa Fedora akan segera tersedia di Windows Store juga.

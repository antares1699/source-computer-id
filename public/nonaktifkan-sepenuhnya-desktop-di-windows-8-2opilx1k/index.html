Nonaktifkan sepenuhnya desktop di Windows 8

Saya sedang mempertimbangkan apakah akan menggunakan Windows 8 sebagai OS pertama untuk pengguna yang benar-benar tidak melek komputer (seperti dalam, tidak pernah menggunakan komputer).

Saya pikir ini mungkin benar-benar berfungsi dengan baik, kecuali saya pikir desktop akan membingungkan. Apakah ada cara untuk menonaktifkan sepenuhnya pergi ke desktop di Windows 8, sehingga satu-satunya antarmuka yang terlihat adalah antarmuka Metro baru?

Anda tidak dapat menonaktifkan desktop sepenuhnya, tetapi Anda dapat:

  • tutup tombol desktop di sisi kiri (mungkin perlu membuka aplikasi Metro sebelum Anda dapat mengklik kanan untuk menutup)

  • hapus (lepas pin) ubin Desktop

Namun, bahkan setelah melakukan ini, Anda masih dapat mengakses Desktop dengan:

  • mencarinya

  • mengklik kanan di pojok kiri bawah

  • memilih Mulailah di bilah Charms

  • Windows + D

Saya tidak percaya Anda bisa melakukannya -tertentu-. Windows 8 mungkin sangat berbeda dalam hal tampilannya saat berada di UI Modern, tetapi desktop, meskipun ditampilkan hanya sebagai aplikasi, masih merupakan dasar dari sistem. Ini membutuhkannya untuk bekerja, dan itu selalu ada, di suatu tempat, seperti yang diilustrasikan oleh fakta bahwa Skype (desktop) mulai otomatis, baik saya memulai "aplikasi" desktop terlebih dahulu atau tidak.

Alternatif yang saya sarankan:

  • Jika Anda mahir dengan hal itu, saya yakin Anda dapat merancang solusi perangkat lunak yang, pada akun tertentu, akan memunculkan orang kembali ke layar Mulai jika mereka berakhir di desktop. Jika perangkat lunak perusak dapat melakukan banyak hal yang lebih aneh, saya yakin pengembang yang baik dapat melakukannya.

  • Untuk pendekatan yang tidak terlalu agresif, buat pop-up muncul setiap kali pengguna memasuki desktop: Anda telah memasuki Desktop (mode lanjutan). Apakah anda yakin ingin melanjutkan? Anda dapat kembali kapan saja dengan tombol Windows.

  • Mungkin ada cara untuk melacak semua pintasan "Desktop" agar mengarah kembali ke layar Mulai.

  • Kosongkan desktop dari semua yang mungkin ada yang membuatnya tampak seperti desktop (ikon, bilah tugas, apa pun).

Kemudian Anda akan mengubah wallpaper menjadi sesuatu yang akan Anda buat sendiri, terlihat seperti UI Modern (font Segoe UI putih dan warna solid, sederhana), yang akan memberitahu untuk kembali ke layar Mulai. Ada beberapa cara untuk melakukannya.

-Berikan instruksi di layar seperti "tekan tombol Windows untuk kembali ke komputer" atau semacamnya.

-Minta untuk mengklik layar, dengan asumsi Anda dapat mengatur untuk membuat halaman Web Desktop Aktif (tutorial untuk Windows 7 ini mungkin hanya berfungsi: http://www.youtube.com/watch?v=7O1OAWMFBgY ), dan membuat tautan ke desktop, yang, secara teknis bukan laman web, kedengarannya sulit.

-Minta untuk mengklik ikon "Kembali ke Komputer". Untuk memastikan Anda tidak membingungkan mereka, Anda dapat membuat pintasan desktop dengan ikon khusus, bergaya UI Modern, aktifkan pembukaan dengan klik sederhana (Opsi Folder) dan buat ikon sebesar mungkin (di suatu tempat di Tampilan, di panel kontrol).

Bagaimana sebenarnya saya membuka program dari file batch? Yang dilakukannya hanyalah membuka command prompt dengan nama file itu [duplikat]

Saat ini saya mencoba mempelajari pembuatan skrip file Batch jadi mungkin saya tidak melihat apa yang saya lakukan salah tetapi saya hanya membuat file batch yang sangat sederhana yang membuka file teks.

Ini kode yang saya gunakan:

@echo off
title Opening a File Lesson
START "C:\Projects\batch\hi.txt"
pause

Alih-alih membuka textfile seperti seharusnya, yang dilakukannya hanyalah membuka command prompt Batch File dengan output yang sesuai:

Tekan tombol apa saja untuk melanjutkan . . .

Tidak apa-apa, itulah yang saya maksudkan tetapi alih-alih membuka file teks, itu hanya membuka prompt perintah LAINNYA dengan jalur file: C: \ Projects \ batch \ hi.txt sebagai keterangan dan sama sekali tidak melakukan apa pun selain menampilkan jalur default saya :

Microsoft Windows [Versi 6.1.7600] Hak Cipta (c) 2009 Microsoft Perusahaan. Seluruh hak cipta.

C: \ Proyek>

Dapatkah seseorang membimbing saya tentang kesalahan yang saya lakukan dan mengapa tidak dibuka dengan program default? Saya mencoba membuka file lain juga seperti Outlook dan Visual Studio 2010 menggunakan jalur absolut (target) dan melakukan hal yang sama jadi bukan hanya file teks. Bantuan apa pun akan dihargai, saya belum terbiasa dengan bahasa skrip dulu.

Agar sejalan dengan contoh Anda saat ini, saya akan menambahkan yang berikut ini:

START "" notepad.exe C:\Projects\batch\hi.txt

karena yang benar-benar ingin Anda lakukan adalah terbuka buku catatan dan berikan parameter, dalam hal ini nama file. Solusi Athom mungkin berhasil juga, tetapi yang ini muncul begitu saja di pikiran.

Jika Anda ingin menunggu notepad ditutup sebelum Anda selesai:

START "" /wait notepad.exe C:\Projects\batch\hi.txt

Judul jendela batch baru ada di dalam tanda kutip, dalam contoh saya, saya tidak memerlukan judul jadi saya biarkan kosong. Saya pikir perintah START mengharapkannya, jadi itu mungkin wajib.

Saya juga menyarankan belajar PowerShell ketika Anda selesai, kata-katanya kekuasaan dan kulit benar-benar menjelaskan kegunaannya. Saya tidak akan mengatakan menyerah pada batch meskipun karena saat ini beberapa hal lebih mudah dalam batch. Juga pada titik ini PowerShell tidak dapat mengembalikan kode keluar negatif yang hanya menyebalkan, tapi saya ngelantur.

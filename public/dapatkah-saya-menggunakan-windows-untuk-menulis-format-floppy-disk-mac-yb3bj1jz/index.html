Dapatkah saya menggunakan Windows untuk menulis format floppy disk Mac?

Untuk bersenang-senang, saya mencoba menggunakan Apple PowerBook 5300c (Mac OS 7.5.2) untuk melakukan beberapa pemrograman. Saya telah menemukan perangkat lunak yang ingin saya gunakan ( berkat beberapa jawaban di SO ) tetapi saya tidak terbiasa dengan format ini (saya selalu menjadi orang PC). Saya telah mencari hal-hal seperti "Windows Mac write floppy", tetapi saya hanya benar-benar melihat halaman tentang membaca format Mac di PC (bukan yang saya butuhkan).

Pikiran pertama saya adalah membeli file Drive floppy disk USB dan beberapa disket - apakah ini yang perlu saya beli? Saya benar-benar tidak tahu apa yang saya lakukan. Apakah PC dan Mac menggunakan disket yang sama? Apakah saya perlu menulis dalam format Mac? Haruskah saya menggunakan mungkin dd atau sesuatu seperti itu untuk menulis file .bin? Mungkin apakah Mac bisa membaca format PC? Saya juga melihat beberapa file .sit (Stuffit); akankah Mac mengenali ini? Bagaimana cara menggunakan file .hqx? Beberapa saran akan sangat berguna.

Windows tidak dapat menulis ke floppy disk berformat Mac (tanpa bantuan), tetapi Mac dapat membaca floppy disk berformat Windows jika diformat dalam FAT dan bukan NTFS. Sementara Mac dan PC menggunakan disket ukuran, format sistem file berbeda. Jadi yang dapat Anda lakukan adalah mengunduh file menggunakan PC Anda dan memasukkannya ke dalam floppy disk FAT16. Mac harus dapat membaca disk dan Anda dapat menyalinnya dari sana.

Pilihan lain mungkin jika Anda dapat menemukan kartu ekspansi internal Ethernet untuk 5300c (jika belum memilikinya) dan Anda dapat melampirkannya ke jaringan dan mengambil file dari drive bersama.

Di mana kata kunci pencarian toko chrome / chromium?

Saya telah menyiapkan sejumlah kata kunci pencarian khusus, seperti "yt" sehingga saya dapat mengetik "yt foo" di bilah URL dan membuatnya mencari "foo" di youtube. Tidak seperti yang lainnya, chrome dan chromium tidak menyinkronkan ini dengan akun Google Anda. Saya ingin menyiapkan sinkronisasi untuk ini sendiri, tetapi menelusuri ~ / .config / chromium Saya tidak dapat menemukan referensi apa pun ke penelusuran kata kunci yang telah saya siapkan. Saya berasumsi mereka disimpan dalam format biner di suatu tempat ... cara apa pun untuk mengekstraknya, menyimpannya, dan dapat mengimpornya di kotak lain nanti? Saya melihat database sqlite3 di sana jadi saya pikir itu mungkin bisa dilakukan.

Kata kunci pencarian Chrome disimpan dalam file SQLite Web Data di dalam folder Data Pengguna. Jalur yang tepat tergantung pada OS yang Anda gunakan:

macOS

~/Library/Application Support/Google/Chrome/Default/Web Data

Windows XP

C:\Documents and Settings\<username>\Local Settings\Application Data\Google\Chrome\User Data\Default\Web Data

Windows Vista / 7/10

C:\Users\<username>\AppData\Local\Google\Chrome\User Data\Default\Web Data

Linux

~/.config/google-chrome/Default/Web Data

Di dalam file database SQLite Web Data itu, tabel keywords memiliki daftar kata kunci pencarian.

Anda dapat mengeditnya menggunakan apa pun yang dapat menggunakan database SQLite (mis Microsoft Access dengan plugin SQLite ODBC , membuat skrip solusi dengan Modul sqlite3 Python , dll.)

Sayangnya, Chrome membuat file tetap terkunci saat sedang berjalan, jadi menurut saya solusi sinkronisasi otomatis di luar Chrome tidak akan mungkin dilakukan.

Jika Anda benar-benar menginginkan keseluruhan kurus pada database Chrome Web Data, Anda dapat menyisirnya bagian itu dari kode sumber Chromium untuk petunjuk .

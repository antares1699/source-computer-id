Ikon Google Chrome yang hilang adalah masalah umum , tetapi saya tidak dapat menemukan solusi yang jelas untuk masalah ini. Saya menemukan beberapa metode yang berhasil untuk beberapa orang, jadi saya sarankan Anda untuk mencobanya. Meskipun saya tidak dapat menjamin itu akan berhasil, ada baiknya dicoba:

  • Buka Microsoft Internet Explorer, jadikan sebagai browser default Anda, lalu buka Google Chrome dan jadikan browser default Anda.
  • Lepas pin Google Chrome dari Menu Mulai, ubah ikonnya di Properti, dan sematkan kembali.
  • Ketik di bilah pencarian Start Menu "chrome" dan cari chromesetup.exe. Klik di atasnya.

Saya memperbaiki masalah ini dengan melakukan hal berikut:

Lepas pin item dari bilah tugas Anda dan periksa pintasan di folder 'Google Chrome' di menu mulai. Jika Anda memiliki masalah yang sama dengan saya, ikon menu mulai juga akan rusak.

Untuk memperbaikinya, klik kanan pintasan dan properti yang dipilih. Kemudian klik 'Ubah Ikon ...' dan pilih ikon chrome. Klik 'OK' dan kemudian 'OK' lagi. Berikan detail administrator jika diminta.

Kemudian cukup klik kanan pada shortcut tetap dan pilih 'Pin to Taskbar'.

Bagaimana cara membuat ctrl-k menampilkan beranda Google biasa di Firefox?

Menekan ctrl-k biasanya membawa saya ke beranda Google versi vanilla biasa. Rupanya, ini telah berubah, pada Firefox 31. Menekan ctrl-k sekarang membawa saya ke semacam pembungkus bermerek Firefox di sekitar kotak pencarian Google. Adakah cara untuk mengubah perilaku baru ini kembali ke perilaku lama?

Saya menyelesaikan ini dengan mengubah about: home page itu sendiri. Ini adalah solusi "hacky" yang mungkin perlu diterapkan kembali pada setiap peningkatan, tetapi begini:

Cari file omni.ja, milik saya ada di c: \ Program Files (x86) \ Mozilla Firefox \ browser. Perhatikan bahwa ada beberapa.

Edit file chrome \ browser \ content \ browser \ abouthome \ aboutHome.xhtml. Untuk melakukan ini, Anda perlu membongkar omni.ja, mengedit, lalu mengemas ulang omni.ja lagi.

Saya meletakkan ini sebagai ganti konten file asli:

<? xml version = "1.0" encoding = "UTF-8"?>

<! DOCTYPE html>

<html xmlns = "http://www.w3.org/1999/xhtml">

<head>

<script>

window.location.href = " https://www.google.se/ ";

</script>

</head>

<body />

</html>

Dari diskusi di Forum Dukungan Mozilla, tampaknya tidak ada cara untuk membuat ctrl-k berperilaku seperti pada versi Firefox sebelumnya; logika dibangun ke dalam kode. Menekan ctrl-k telah melakukannya, dan berlanjut ke, membawa Anda ke jendela pencarian di bilah alat navigasi. Jika Anda telah menyesuaikan bilah alat navigasi Anda untuk menghapus jendela pencarian, seperti yang saya miliki, maka ctrl-k, yang sebelumnya terprogram untuk melompat ke google.com, sekarang terprogram untuk melompat ke about: home . (Inilah yang saya sebut di atas sebagai "semacam pembungkus bermerek Firefox ...")

Saya menemukan solusi yang mungkin menarik, jadi saya akan memposting ulang di sini.

Mengikuti petunjuk di Dukungan Mozilla Cara mencari IMDB, Wikipedia, dan lainnya dari bilah alamat , definisikan g untuk mengarahkan ke jendela pencarian di google.com, lalu alih-alih ctrl-k, lakukan ctrl-l g.

Mengapa saya ingin membuka google.com dan memasukkan istilah penelusuran di sana, daripada langsung menambahkannya ke kata kunci pintar? Karena terkadang saya tidak yakin dengan ejaan, atau frasa, dan saya ingin mendapatkan bantuan dari saran drop-down.

Mengapa saya lebih memilih google.com biasa untuk melakukan ini, dibandingkan dengan about: home halaman, yang juga memiliki perilaku sarankan-saat-Anda-ketik? Untuk Google Doodles. Tidak ada alasan lain.

Sepertinya Anda telah menghapus kotak pencarian di samping bilah URL. Untuk mengembalikan kotak telusur ke tempatnya semula:

Ctrl + K + Memasukkan

Ctrl + K harus memindahkan fokus ke kotak pencarian di samping bilah URL. Anda harus meletakkan kembali kotak telusur di sana (karena secara default) jika Anda telah melepasnya sebelumnya.

Lalu tekan Memasukkan ketika Anda berada di sana akan memuat halaman pencarian Google.

Untuk saya pergi dari Ctrl + K untuk Ctrl + K + Memasukkan tidak sulit untuk membiasakan diri.

Saya mencari waktu yang lama dan menemukan solusi yang paling sesuai untuk saya DAN pada dasarnya Anda dapat menjadikan URL apa pun sebagai pintasan keyboard dengan cara ini:

  1. Instal Add-on Keyconfig: http://forums.mozillazine.org/viewtopic.php?f=48&t=72994

  2. Buka Keyconfig dengan menekan Ctrl + Bergeser + F12

  3. Klik kolom Shortcut yang mengarah untuk mengurutkan menurut Pintasan dan gulir ke bawah ke Ctrl+K
  4. Klik Edit this key dan ubah teks Ctrl+K menjadi berikut: gBrowser.selectedTab = gBrowser.addTab("www.google.com");

Anda dapat membuat pintasan apa pun untuk URL apa pun dengan kode ini!

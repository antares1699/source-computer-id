Cara lebih cepat untuk menghapus file di drive jaringan

Saya memiliki drive jaringan dan saya mencoba menghapus folder besar. Sungguh menyebalkan!

Ini telah berada di fase "Menghitung" selamanya sekarang (pada 26 GB). Apakah tidak ada cara untuk hanya menghapus ledakan ! Saya tidak perlu tahu berapa banyak file dan seberapa besar mereka!

Saya menggunakan Windows 7. Saya menggunakan "shift" delete untuk melewati recycle bin.

Anda mungkin bisa menggunakan command prompt. Saya mungkin melewatkan beberapa detail prosedur, tetapi coba ini:

  1. Buka Start Menu dan di kotak teks, ketik cmd.exe dan tekan Enter (atau buka command prompt menggunakan metode pilihan Anda)
  2. Beralih ke drive jaringan dengan mengetik Z: (di mana Z adalah huruf drive jaringan)
  3. Ubah ke direktori induk dari direktori yang Anda coba hapus menggunakan cd path\to\parent\directory
  4. Hapus direktori menggunakan rmdir /S giantdir

Misalnya, jika Anda ingin menghapus direktori O:\MG\WTF\BBQ\SOMANYFILES:

C:\Documents And Settings\Me> O:

O:> cd MG\WTF\BBQ

O:\MG\WTF\BBQ> rmdir /S SOMANYFILES

Atau sekarang setelah saya memikirkannya, saya pikir Anda bisa melakukannya

C:\Documents And Settings\Me> O:

O:> rmdir /S MG\WTF\BBQ\SOMANYFILES

tetapi Anda akan kehilangan kesempatan untuk melihat Windows mengeja cmd.exe dalam font terminal ;-)

Ngomong-ngomong, rmdir (atau del) tidak memindahkan barang ke Recycle Bin, itu hanya menghapusnya, jadi berhati-hatilah untuk tidak menghapus hal-hal yang sebenarnya tidak Anda inginkan.

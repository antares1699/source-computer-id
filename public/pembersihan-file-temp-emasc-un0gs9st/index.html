pembersihan file temp emasc

Saat saya mengedit file dengan emacs, emacs membuat file dengan nama yang sama, tetapi dengan tilde di bagian akhir:

notes.txt ~

Ketika saya menyimpan perubahan saya dan keluar dari emacs, itu tidak menghapus file temp. Apakah ada pengaturan yang dapat saya buat di .emacs yang memaksanya untuk menghapus file temp sehingga berhenti mengacaukan direktori saya?

4
Dr. Watson — Sumber

8
8

Emacs memanggil file cadangan ini (sebagai lawan dari file sementara yang dimulai dengan #). Untuk tidak membuat ini, tambahkan

(setq make-backup-files nil)

ke .emacs Anda

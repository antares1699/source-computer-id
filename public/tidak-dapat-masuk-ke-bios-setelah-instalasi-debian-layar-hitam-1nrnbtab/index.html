Tidak dapat masuk ke BIOS setelah instalasi debian - layar hitam

Saya memiliki laptop Acer Aspire dengan Windows 10. Baru-baru ini saya telah menginstal Debian 9 di partisi yang berbeda dan semuanya berjalan dengan baik. Kedua kalinya saya mencoba boot ke BIOS, saya mendapat layar hitam dengan tanda hubung di atasnya.

Saya juga tidak dapat mem-boot grub karena Windows Boot Manager melakukan booting terlebih dahulu (inilah mengapa saya ingin masuk ke BIOS).

Windows 10 mem-boot dengan baik.

Coba ini: Keluarkan baterai, cabut laptop, tahan tombol start setidaknya selama 30 detik, lalu ganti baterai dan colokkan.

Jika ini tidak membantu, ikuti prosedurnya:

Masuk ke Pengaturan Firmware UEFI dari dalam Windows
Buka Pengaturan PC> Perbarui & keamanan> Pemulihan> Mulai Ulang Sekarang> Pemecahan Masalah> Opsi lanjutan> Pengaturan Firmware UEFI> Mulai Ulang.
Sistem harus memulai ulang dan membawa Anda ke BIOS, di mana Anda dapat mengatur ulang beberapa pengaturan yang mungkin merepotkan.

Dan jika gagal, gunakan senjata besar:
Jalankan Perbaikan Startup di Windows 10 .

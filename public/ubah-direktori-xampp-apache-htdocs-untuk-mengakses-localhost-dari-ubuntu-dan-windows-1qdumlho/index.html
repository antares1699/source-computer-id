Ubah direktori xampp apache htdocs untuk mengakses localhost dari ubuntu dan windows

Saya menggunakan ubuntu dan windows 7 dualboot, dan menginstal xampp pada keduanya. Saya ingin memiliki akses ke satu direktori localhost apakah saya menggunakan windows atau ubuntu. kedua sistem diinstal pada partisi yang sama.

Ini adalah pertanyaan lama tapi saya belum melihatnya dijawab dengan benar. Inilah yang perlu Anda lakukan:

  1. Instal XAMPP di Windows dan LAMPP di Ubuntu
  2. Boot ke Ubuntu dan rekam folder htdocs Windows Anda (katakanlah /media/user/OS/xampp/htdocs)
  3. Buka /opt/lampp/etc dan edit httpd.conf dan lakukan perubahan berikut (Anda memerlukan izin root untuk melakukannya)
  4. Ganti
    #DocumentRoot "/opt/lamp/htdocs"
    by
    #DocumentRoot "/media/user/OS/xampp/htdocs"
  5. Replace
    Directory "/opt/lampp/htdocs"
    dengan
    Directory "/media/user/OS/xampp/htdocs"

  6. Di bagian

    IfModule unixd_module
    ganti
    User daemon
Group daemon
    dengan
    User yourUsername
Group yourGroupname
    Biasanya nama pengguna dan nama grupAnda sama: nama pengguna Ubuntu Anda tetapi Anda dapat memeriksanya di /etc/passwd dan /etc/groups

  7. Mulai ulang layanan LAMPP dan Anda sudah siap.

Saya akan merekomendasikan memiliki folder HTDOCS "bersama" di sisi Windows Anda karena Ubuntu dapat membaca NTFS secara asli.

Anda perlu menggunakan FSTAB untuk memasang partisi Windows Anda saat Ubuntu melakukan boot agar Anda dapat mengakses file web.

Setelah Anda memiliki titik pemasangan biasa untuk partisi Windows, Anda perlu membuat tautan simbolis di Ubuntu sehingga file web Anda masih dapat diakses oleh Ubuntu Apache tetapi masih dapat membaca dari lokasi defaultnya.

Misalnya. sudo chown -R $USER /var/www; ln -s /var/www /media/Windows/xampp/htdocs

Saya hanya menggunakan kode di atas sebagai contoh. Silakan lakukan pembacaan lebih lanjut untuk mendapatkan skenario yang benar.

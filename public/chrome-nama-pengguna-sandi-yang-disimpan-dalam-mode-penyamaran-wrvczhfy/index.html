Chrome: Nama pengguna / sandi yang disimpan dalam mode penyamaran

Jika saya membuka jendela penyamaran di Google Chrome dan membuka halaman web tempat Chrome memiliki nama pengguna dan sandi yang disimpan (misalnya, formulir masuk di http://gmail.com ), Saya melihat bahwa nama pengguna dan kata sandi saya terisi secara otomatis.

Apakah itu berarti saya tidak benar-benar penyamaran ? Dapatkah situs web melihat nama pengguna saya meskipun saya tidak masuk secara eksplisit? Atau apakah ada mekanisme di balik layar yang mencegah laman web mengambil nilai yang terisi otomatis kecuali saya benar-benar masuk?

Klarifikasi: Nama pengguna yang disimpan (dan kata sandi) sangat mirip dengan cookie: pengenal unik Anda yang ditautkan ke situs tertentu, disimpan secara lokal di browser Anda, tersedia untuk situs tersebut saat Anda membukanya. Saat kamu pergi penyamaran Anda meminta browser Anda untuk tidak mengidentifikasi Anda ke situs yang Anda kunjungi. Itu dilakukan dengan (antara lain) tidak mengekspos cookie-nya. Mengekspos nama pengguna yang disimpan dalam mode ini tidak masuk akal bagi saya (tapi mungkin saya melewatkan sesuatu ...).

Pembaruan (2014-09-25): tampaknya versi Chrome terbaru tidak melakukan ini lagi.

Apakah itu berarti saya bukan penyamaran? Dapatkah situs web melihat nama pengguna saya meskipun saya tidak masuk secara eksplisit? HAI

Penyamaran berarti Chrome menghancurkan semua cookie yang dibuat & tidak merekam situs web apa pun yang Anda kunjungi. Kata sandi diisi dari pengelola kata sandi Chrome, bukan dari situs.

Penyamaran juga merupakan penerapan khusus browser di sisi klien - bukan berarti tidak ada badan yang dapat melacak Anda.

Baca pesan penyamaran Chrome:

Anda menggunakan mode penyamaran. Laman yang Anda lihat di jendela ini tidak akan muncul di riwayat browser atau riwayat penelusuran, dan tidak akan meninggalkan jejak lain, seperti cookie, di komputer Anda setelah Anda menutup jendela penyamaran. Namun, file apa pun yang Anda unduh atau bookmark apa pun yang Anda buat akan dipertahankan.

Menggunakan mode penyamaran tidak memengaruhi perilaku orang lain, server

Saat Anda menggunakan mode penyamaran, Anda meminta browser Anda untuk tidak mengidentifikasi Anda ke situs yang Anda kunjungi. Itu dilakukan dengan (antara lain) tidak mengekspos cookie-nya. Mengekspos nama pengguna yang disimpan dalam mode ini tidak masuk akal bagi saya (tapi mungkin saya melewatkan sesuatu ...).

Tidak, Anda salah paham di sini. Mode penyamaran bekerja dengan menghancurkan cookie setelah sesi selesai (yang ditandai dengan Anda menutup jendela penyamaran). Penyamaran tidak berarti tidak mengidentifikasi diri saya, itu berarti tidak melacak aktivitas saya saat saya dalam sesi penyamaran.

Hal yang Anda lewatkan adalah cara kerja penyamaran - browser tidak menampilkan cookie apa pun - bukan berarti browser melakukannya tidak tahu situs web mana yang Anda kunjungi - nama pengguna / kata sandi diambil oleh browser dengan mencocokkan situs mana yang Anda buka - bukan melalui cookie.

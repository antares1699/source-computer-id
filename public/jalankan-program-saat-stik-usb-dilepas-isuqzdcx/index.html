Jalankan program saat stik USB dilepas

Judulnya menjelaskan semuanya. Pada dasarnya, saya ingin melepas volume TrueCrypt ketika stik USB dengan keyfile di atasnya dilepas (dan idealnya, matikan komputer juga).

Dulu ada program yang melakukan itu (Truemounter http://www.truemounter.de/ ) tetapi tampaknya ditinggalkan dan tampaknya tidak berfungsi dengan baik di bawah Windows 7.

Ada ide?

Lepas USB dengan Aman memiliki fitur seperti itu: dapat memulai program saat perangkat USB dilepas.

Fitur lain USB Safely Remove itu membedakannya dari perangkat lunak serupa memulai aplikasi apa pun tidak hanya setelah menghubungkan perangkat, tetapi juga sebelum melepasnya . Autorun itu fitur memungkinkan Anda untuk mengatur data cadangan sebelum memutuskan sambungan a hard drive yang dapat dilepas, untuk menjalankan Total Komandan dengan isi file pen-drive, lepaskan secara otomatis drive TrueCrypt terenkripsi sebelumnya melepaskan media USB , dll.

USB Safely Remove berfungsi dengan baik pada Windows 7 (juga dalam 64-bit).

Bagaimana cara mencegah jendela koneksi jaringan dial up muncul di Windows 7?

Saya memiliki koneksi PPP melalui Ethernet di rumah, jadi saya memiliki koneksi dial up yang muncul setiap kali saya masuk, jadi saya dapat terhubung ke internet dengan mengkliknya. Ini berfungsi dengan baik, tetapi sekarang saya mendapat router yang melakukan koneksi secara otomatis, jadi tidak perlu jendela koneksi jaringan dial up untuk muncul.

Namun demikian itu muncul setiap kali saya masuk terlepas dari koneksi internet hidup yang sudah dibuat oleh router. Itu tidak menyebabkan kesalahan. Saya hanya perlu membatalkan dialog, tetapi ini mengganggu dan saya tidak dapat menemukan pengaturan apa pun untuk menghentikannya.

Tentu saja, saya dapat menghapus koneksi jaringan sama sekali, tetapi saya lebih suka menyimpannya, jadi saya tidak perlu membuatnya lagi jika terkadang saya perlu terhubung ke internet secara langsung tanpa router.

Apakah ada cara untuk menghentikan dialog koneksi agar tidak muncul tanpa memutuskan koneksi jaringan?

Sangat mudah untuk melakukannya hanya dengan membuka Panel Kontrol> Jaringan dan Pusat Berbagi . Sekarang pilih Ubah pengaturan Adaptor dari panel kiri. Sekarang koneksi default harus dicentang dengan tombol hijau dan klik kanan di atasnya. Sekarang klik kanan padanya dan pilih Batalkan sebagai koneksi default .

Edit sebagai tanggapan atas komentar

Jika tidak disetel default di sana maka Anda harus meluncurkan file Internet Explorer dan sekarang buka Alat> Opsi Internet dan pilih koneksi tab. Anda juga bisa pergi ke Koneksi internet melalui panel kontrol. Sekarang di sini Anda memiliki tiga opsi:

1: Jangan pernah Panggil koneksi
2: Panggil setiap kali koneksi tidak tersedia
3: Selalu panggil koneksi default saya

Anda harus memilih opsi pertama dan klik menerapkan .

Bagaimana cara memutar file FLV di Windows Media Player?

Saya ingin memutar video Flash atau file "FLV" di Windows Media Player. Saya telah melakukan ini sebelum menggunakan paket codec seperti K-Lite dan CCCP, tetapi menurut saya mereka menyebabkan terlalu banyak masalah dan saya sangat ingin menghindarinya.

Saya juga melihat PlayFLV , tetapi ini sudah mendukung Flash sebelum H.264 dan itu agak diperlukan hari ini.

Jadi, apakah ada opsi codec lain yang tersedia? Idealnya juga akan ada dukungan thumbnail dengan codec.

NB: Menggunakan pemutar video lain tidak pilihan bagi saya - Saya suka WMP, saya hanya ingin tahu codec terbaik untuk pekerjaan itu.

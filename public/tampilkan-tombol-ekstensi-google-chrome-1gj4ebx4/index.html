Tampilkan tombol ekstensi Google Chrome

Pembaruan Google Chrome terkini membuat semua tombol ekstensi disembunyikan. Saya harus mengklik tombol dropdown untuk melihat tombol ekstensi. Saya ingin tombol ekstensi ditampilkan tanpa harus mengklik tombol tarik-turun. Apakah ada cara untuk menampilkan tombol?

Menurut Forum Produk Google , Chrome versi terbaru (~ 39.0.2171) memiliki konflik dengan akselerasi perangkat keras pada sistem operasi tertentu. Kunjungi setelan Anda, gulir ke bawah dan klik "lanjutan", lalu hapus pilihan "gunakan akselerasi perangkat keras jika tersedia". Sekarang coba seret tepi kanan Omnibox / Address Bar untuk mengecilkannya dan menampilkan ekstensi tersembunyi.

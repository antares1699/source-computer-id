Mengapa Textual memiliki ikon tema "pesta" tanpa alasan yang jelas?

Saya baru saja memperhatikan bahwa klien IRC saya, Tekstual , memiliki ikon yang berbeda dari biasanya. Mengapa?

Tidak ada pemberitahuan di aplikasi yang dapat menjelaskan hal ini. Ini bukan ulang tahunku atau siapa pun yang aku kenal.

Hari ini, 23 Juli, adalah hari ulang tahun Textual, dan ikonnya berubah untuk hari itu. Dari Kode sumber :

Komitmen publik pertama Textual terjadi pada tanggal 23 Juli 2010. Ini adalah hari yang kami anggap sebagai ulang tahun aplikasi.

Kutipan kode sumber menunjukkan kondisi untuk ikon yang berbeda:

/* THIS IS A SECRET!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

 Birthday icon designed by Alex Sørlie Glomsaas. */

NSCalendar *sysCalendar = [NSCalendar currentCalendar];

NSDateComponents *breakdownInfo = [sysCalendar components:(NSDayCalendarUnit | NSMonthCalendarUnit) fromDate:[NSDate date]];

/* The first public commit of Textual occured on July, 23, 2010. This is the day
 that we consider the birthday of the application. */
if ([breakdownInfo month] == 7 && [breakdownInfo day] == 23) {
    return [NSImage imageNamed:@"birthdayIcon"];
} else {
    return [NSImage imageNamed:@"NSApplicationIcon"];
}

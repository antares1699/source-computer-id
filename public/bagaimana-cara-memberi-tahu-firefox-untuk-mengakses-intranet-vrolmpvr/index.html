Bagaimana cara memberi tahu Firefox untuk mengakses intranet

Di pekerjaan saya, kami memiliki banyak situs di intranet kami. Bagaimana cara memberi tahu Firefox untuk mengakses situs intranet?

Saya terus mendapatkan ini:

Item yang diminta tidak dapat dimuat oleh proxy.

Server proxy tidak dapat menemukan file server: myintranet. Server melakukannya tidak memiliki entri DNS. Periksa server nama di Lokasi (URL) dan coba lagi.

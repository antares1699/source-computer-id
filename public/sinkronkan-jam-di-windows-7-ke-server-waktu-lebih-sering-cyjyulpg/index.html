Sinkronkan jam di Windows 7 ke server waktu lebih sering

Pada Windows 7, masih merupakan cara untuk menyinkronkan jam ke server waktu lebih sering sama seperti sebelumnya (dengan regedit)?

Apakah ada cara lain seperti dengan menggunakan panel kontrol atau alat sistem?

Itu sama. Itu tidak berubah sejak Windows XP.

Untuk mengubah penundaan, cukup edit kunci registri berikut ini:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\w32time\TimeProviders\NtpClient

Nilai ada jumlah detik di antara pembaruan.

Saya suka metode menggunakan penjadwal tugas windows standar untuk mengatur pekerjaan yang lebih sering disinkronkan. Apa yang saya suka tentang ini adalah ini berfungsi dengan baik dan saya tidak perlu mengacaukan registri.

Lihat http://www.pretentiousname.com/timesync/ untuk tulisan yang bagus tentang ini.

Intinya adalah:

Siapkan tugas (jalankan taskschd.msc dari garis cmd) untuk berjalan pada frekuensi yang Anda inginkan.

Anda akan menambahkan dua tindakan ke tugas yang dijalankan sebagai LOCAL SERVICE pengguna yang berjalan dengan hak istimewa tertinggi.

1

program / script: %windir%\system32\sc.exe

argumen: start w32time task_started

2

program / script: %windir%\system32\w32tm.exe

argumen: /resync

Di bawah pengaturan pastikan untuk memeriksa Run task as soon as possible after a scheduled start is missed.

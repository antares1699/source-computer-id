Dimana Synergy terbaru untuk Mac OS X? [Tutup]

Saya menggunakan SynergyKM di Mac OS X 10.5.7 MacBook Pro untuk berbagi keyboard saya dengan desktop Windows XP saya.

Di Mac baru-baru ini berhenti bekerja. Ini lognya:

7/29/09 2:23:39 PM Synergyd[328] *** -[SDSynergyServerWrapper isEqualToString:]: unrecognized selector sent to instance 0x145930 
7/29/09 2:23:39 PM Synergyd[328] Exception raised during posting of notification. Ignored.  exception: '*** -[SDSynergyServerWrapper isEqualToString:]: unrecognized selector sent to instance 0x145930'  invoked observer method: '*** -[LCSStdIOTaskWrapper readCompletition:]'  observer: 0x146110  notification name: 'NSFileHandleReadCompletionNotification'

Saya kira pembaruan sistem baru-baru ini merusaknya. Saya pergi untuk meningkatkan sinergi, tetapi sepertinya SynergyKM proyek dan sinergi asli proyek keduanya mati (tidak ada yang dirilis sejak v1.3.1 pada tahun 2006!). Sinergi sangat berguna, pasti proyeknya telah dihidupkan kembali di tempat lain?

Dari manual

Protokol http memungkinkan klien untuk mengidentifikasi diri mereka sendiri menggunakan file Bidang tajuk Agen-Pengguna. Ini memungkinkan untuk membedakan perangkat lunak www, biasanya untuk keperluan statistik atau untuk penelusuran protokol pelanggaran. Wget biasanya diidentifikasikan sebagai ' Wget / versi ', Versi sedang nomor versi Wget saat ini.

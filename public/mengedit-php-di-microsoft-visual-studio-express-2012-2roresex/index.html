Mengedit PHP di Microsoft Visual Studio Express 2012

Apakah ada cara untuk menginstal sesuatu di VS Mengekspresikan jadi saya bisa mengedit PHP di VS Express Web dan meminta program mewarnai teks seperti di HTML dan JS?

Visual Studio tidak mendukung PHP secara native. Ada beberapa ekstensi berbeda yang dapat menambahkan dukungan:

  • VS.Php untuk Visual Studio (komersial dengan masa percobaan)
  • Alat PHP untuk Visual Studio (komersial dengan masa percobaan)

Namun, VS Express tidak mengizinkan Anda memasang ekstensi, jadi saya akan merekomendasikan pemasangan Visual Studio Community Edition sehingga Anda dapat menggunakan ekstensi.

Microsoft memiliki beberapa alat pengembangan lain yang mendukung PHP, jadi, Anda dapat mencoba salah satunya:

  • Kode Visual Studio
  • Matriks Web

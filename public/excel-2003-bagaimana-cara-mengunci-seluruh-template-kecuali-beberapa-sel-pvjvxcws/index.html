Excel 2003: Bagaimana cara mengunci seluruh template kecuali beberapa sel?

Saya ingin mengunci seluruh template / sheet kecuali untuk a sedikit sel yang harus diisi secara manual. Untuk mengunci template saya menggunakan "Tools / Protection / Protect Sheet"

Saya menggunakan spreadsheet Excel 2003. Bagaimana saya bisa melakukan itu?

4

7

Jawabannya adalah dengan memilih sel yang ingin Anda edit, pilih Format / Sel, tekan tab "Proteksi", dan hapus centang pada kotak yang bertuliskan "Terkunci".

Perhatikan bahwa ini harus dilakukan sebelum Anda melindungi seprai.

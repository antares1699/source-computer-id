Menggunakan proxy untuk situs tertentu saja

Ketika saya bekerja dari rumah, saya ingin membaca beberapa situs yang diblokir oleh pekerjaan saya. Karena saya terhubung melalui VPN, server mereka merutekan permintaan saya dan situs seperti reddit diblokir.

Saat ini, saya hanya RDP ke komputer lain di rumah saya dan menggunakan browser di mesin itu. Apakah ada cara untuk menyiapkan proxy di komputer lain dan menginstruksikan sistem saya untuk menggunakan proxy untuk situs tertentu?

Ini semua di win7.

Secara teoritis skrip konfigurasi otomatis proxy (PAC) dimaksudkan untuk ini (tetapi ini memerlukan beberapa pemrograman). Secara praktis, saya rasa Anda dapat menggunakan browser yang berbeda - satu untuk situs yang Anda akses melalui proxy dan yang kedua untuk semua situs lainnya. Di Firefox Anda juga dapat mengatur profil yang berbeda - satu untuk menggunakan proxy, dan yang kedua untuk akses langsung ke internet: https://support.mozilla.org/en-US/kb/Managing-profiles

Jika Anda menggunakan koneksi Windows VPN, ada kotak centang di jendela properti lanjutan pengaturan ipv4 (dari koneksi itu) yang (sejak rilis windows 7 dicentang secara default) memaksa semua koneksi untuk menggunakan 'default gateway on jaringan jarak jauh '. Hapus centang pada kotak ini dan Anda akan dapat menavigasi ke alamat mana pun. Ini mungkin mengharuskan Anda untuk menentukan domain saat Anda rdp ke mesin di kantor Anda (yaitu machine.domain.com).

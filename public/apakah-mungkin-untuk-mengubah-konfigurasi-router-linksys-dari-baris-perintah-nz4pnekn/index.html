Apakah mungkin untuk mengubah konfigurasi router Linksys dari baris perintah?

Saya terus-menerus masuk ke mesin rumah saya, yang menjalankan Ubuntu, dari berbagai lokasi terpencil melalui SSH. Kadang-kadang saya ingin mengubah pengaturan pada router Linksys rumah saya - pengaturan penerusan port atau menonaktifkan / mengaktifkan nirkabel, hal-hal seperti itu.

Ketika saya mencoba menggunakan browser teks links2 untuk mengkonfigurasinya, tidak banyak yang dapat saya lakukan, karena judul tab tidak muncul (mungkin karena itu adalah gambar).

Apakah ada cara lain untuk mengkonfigurasi router Linksys dari baris perintah? Saya kira saya dapat menyiapkan proxy di mesin rumah saya dan menggunakan browser yang terhubung ke proxy itu untuk mengkonfigurasinya, tetapi saya akan berpikir mungkin ada cara non-browser untuk melakukannya.

Beberapa firmware router pihak ketiga (seperti DD-WRT , OpenWRT , Tomat , dll) menyediakan antarmuka SSH ke router. Saya akan mengaktifkan ini di sisi LAN tetapi bukan WAN. (Anda juga dapat mengaktifkannya di sisi WAN menggunakan port yang tidak standar, dan kata sandi yang sangat kuat; Anda juga dapat mengaktifkan akses WAN ke antarmuka web; tetapi saya tidak merekomendasikan konfigurasi tersebut.)

Setelah diaktifkan, Anda dapat SSH ke router Anda dari dalam LAN. (Misalnya, jika antarmuka SSH diaktifkan di LAN tetapi tidak di WAN, Anda akan menggunakan SSH ke mesin Ubuntu Anda, dan SSH lagi dari sana ke router.) Setelah di sana Anda mendapatkan shell minimal (sering disediakan oleh BusyBox), dan memiliki akses ke file konfigurasi, skrip daemon, pengaturan firmware, dan perintah apa pun yang diinstal.

Misalnya, saya memiliki Linksys WRT54G lama yang menjalankan OpenWRT. Melalui antarmuka SSH, saya memiliki akses ke:

  • iptables (misalnya, untuk membuka port sementara);
  • itu / etc / hosts dan /etc/dnsmasq.conf file (misalnya, untuk menambahkan penggantian DNS lokal atau mengubah server DHCP); dan
  • itu reboot perintah (jika diperlukan).

Saya juga dapat memeriksa pengaturan firmware ( acara nvram ), ubah mereka ( set nvram ), dan tulis ke memori flash nonvolitile ( nvram komit ). Banyak hal yang dapat Anda akses di antarmuka web dapat langsung diakses nvram , tetapi berhati-hatilah. Ini OpenWRT HowTo lama menjelaskan dasar nvram pemakaian ; mungkin ada panduan online yang lebih baik dan lebih baru, jadi bacalah dan berhati-hatilah saat menggunakan utilitas itu.

Jika Anda menggunakan ssh, Anda dapat mengaturnya dengan cepat, proxy sementara:

ssh -L 9000:routerip:80 homemachine

Kemudian di komputer lokal Anda, arahkan browser web Anda ke http://127.0.0.1:9000/ dan Anda akan melihat layar konfigurasi router Anda. routerip adalah IP yang akan digunakan mesin ubuntu rumah Anda untuk terhubung ke router.

Jika Anda menggunakan puTTY, pengaturan ini berada di bawah Connection-> SSH-> Tunnels Source-port 9000, destination routerip:80, ketik Local (lalu klik "Add", lalu buka koneksi seperti biasanya).

Ini adalah proxy cepat dan sederhana yang dapat Anda atur kapan saja Anda perlu menghubungkan dan mengubah sesuatu.

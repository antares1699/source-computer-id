Bagaimana cara memaksa modem Huawei E3276 LTE untuk hanya dipasang sebagai modem?

Saya memiliki modem USB Huawei E3276 LTE dan saya perlu memaksa modem ini untuk hanya dipasang sebagai modem saat terhubung ke komputer. Secara default, modem juga dipasang sebagai CD-drive dan perangkat penyimpanan massal USB, yang baik-baik saja saat digunakan dengan komputer, tetapi masalah saat menghubungkannya ke router atau yang serupa.

Apa yang telah saya baca sejauh ini adalah bahwa Anda dapat menggunakan PuTTY atau yang serupa untuk menghubungkan ke modem melalui COM-port, dan kemudian mengirim perintah-AT ke sana untuk mengkonfigurasi modem agar hanya dipasang sebagai modem. Perintah AT yang disebutkan untuk sebagian besar modem Huawei adalah AT^U2DIAG=0 tetapi ketika mengirimnya ke modem saya, saya menerima ERROR sebagai tanggapan, yang tidak memberi saya banyak hal untuk dikerjakan.

Saya juga mencoba AT^U2DIAG? yang dikatakan berfungsi untuk beberapa modem Huawei untuk menampilkan nilai yang mungkin untuk properti ini, tetapi itu juga mengembalikan ERROR.

Saya sudah mencoba perintah AT lainnya, seperti ATI untuk mendapatkan informasi tentang model, IMEI, dll. Yang berfungsi dengan baik.

Apakah ada perintah lain yang dapat saya gunakan untuk melakukan ini, atau adakah hal lain yang perlu saya lakukan agar ini berfungsi?

Anda dapat menggunakan perintah AT ini:

AT^SETPORT?  - show current mode
AT^SETPORT=? - show available modes
AT^SETPORT="A1,A2;1,2,3" - set "only modem" mode.

Pastikan Anda menuliskan mode saat ini (menggunakan perintah pertama), untuk berjaga-jaga jika Anda ingin mengembalikan perubahan di lain waktu.

Setelah mengirim perintah ini, Anda harus memasukkan kembali modem Anda.

Anda harus menggunakan perintah ini

AT^SETPORT="FF;1"

Perintah ini menonaktifkan semua port kecuali port modem.

Saya kira Anda telah menyelesaikan masalah Anda tetapi saya harap ini akan membantu pengguna lain yang memiliki masalah yang sama, karena saya telah menghabiskan beberapa jam mencoba menemukan solusinya.

Saya memiliki modem HUAWEI E3131 (dikeluarkan oleh RCS-RDS di Rumania) dan sekarang berfungsi dengan router Edimax 3g-6200n

E3276 adalah modem 4G LTE.

AT^SETPORT=?

^SETPORT:1: 3G MODEM
^SETPORT:2: 3G PCUI
^SETPORT:3: 3G DIAG
^SETPORT:5: 3G GPS
^SETPORT:A: BLUE TOOTH
^SETPORT:16: NCM
^SETPORT:A1: CDROM
^SETPORT:A2: SD
^SETPORT:10: 4G MODEM
^SETPORT:12: 4G PCUI
^SETPORT:13: 4G DIAG
^SETPORT:14: 4G GPS

Anda dapat mengatur 1, 2 dan 3 tetapi mereka akan menjadi 10, 12 dan 13. Anda perlu mengirim perintah:

AT^SETPORT="FF;10,12"

Untuk Modem dan PCUI. Jika Anda menginginkan Model Kontrol Jaringan daripada Modem, Anda perlu mengirim:

AT^SETPORT="FF;12,16"

Saya telah menetapkan milik saya untuk Modem dan NCM sehingga saya dapat menggunakannya di Android (dengan PPP Widget) dan Mac (dengan panggilan driver Modem Generik * 99 #) menggunakan Modem dan pada Windows 10 menggunakan NCM, itu adalah:

AT^SETPORT="FF;10,12,16"

Kamu PERLU untuk meninggalkan port "PCUI" atau Anda akan "memblokir" modem (tidak dapat mengirim perintah lebih lanjut, dan Anda akan terjebak dengan apa yang Anda atur)

Catatan lain adalah bahwa "DIAG" digunakan untuk mode DIAGNOSTIC dan tidak diperlukan untuk konektivitas dan dapat diaktifkan jika diperlukan nanti melalui "PCUI", dengan "DIAG" dan firmware yang benar Anda dapat mengubah beberapa hal tentang modem . Ingatlah untuk TIDAK PERNAH hapus "PCUI", sangat penting!

Editor LaTeX dengan manajemen proyek untuk Mac

Saya mencoba beberapa editor LaTeX untuk OS X. TexMaker, TexWorks, TexShop dan TextMate / BBEdit menawarkan penyorotan sintaks dan beberapa fitur tambahan untuk mengedit file tex tunggal.

Di Linux saya telah menggunakan Kile untuk mengedit LaTeX. Fitur luar biasa adalah mode 'proyek', yang membuka tampilan proyek dengan panel samping. Anda dapat membuka beberapa dokumen tex dan menyimpannya dalam struktur folder hierarki dan dengan demikian membagi bab Anda menjadi beberapa file dengan file utama tempat Anda memasukkan setiap bab. Anda juga dapat menambahkan gambar, dll. Ini seperti proyek Eclipse. Itu sangat berguna untuk proyek besar seperti tesis atau makalah penelitian.

Pertanyaan: Apakah ada editor LaTeX untuk OS X yang menawarkan opsi serupa?

TextMate mendukung proyek. Ada "laci proyek" di mana Anda dapat mengatur beberapa file dalam struktur hierarki. Anda dapat menyimpan proyek Anda, membukanya nanti, dll.

Kemudian Anda dapat menggunakan, mis., karet untuk menyusun proyek Anda. Tentu saja rubber tidak tahu apa-apa tentang proyek TextMate, tetapi tidak perlu: ia akan menemukan dependensi secara otomatis dan mengkompilasi semua yang sudah ketinggalan zaman; cukup arahkan ke file Latex utama Anda. Cukup mudah untuk mengkonfigurasi TextMate sehingga Anda dapat meluncurkan rubber dengan menekan pintasan keyboard di TextMate.

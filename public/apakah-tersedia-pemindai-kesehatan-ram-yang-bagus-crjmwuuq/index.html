Apakah tersedia pemindai kesehatan RAM yang bagus?

Saya punya masalah yang saya coba cari tahu dengan MySQL Workbench 5.2.34 di Win7 64-bit Ultimate. Informasi pengecualian mengatakan

System.AccessViolationException was unhandled
  Message=Attempted to read or write protected memory. This is often an indication that other memory is corrupt.

Saya ingin memastikan ingatan saya baik-baik saja untuk berjaga-jaga. Saya telah mencari di Google untuk beberapa, tetapi hasil teratas tampak agak cerdik bagi saya. Ada Ide?

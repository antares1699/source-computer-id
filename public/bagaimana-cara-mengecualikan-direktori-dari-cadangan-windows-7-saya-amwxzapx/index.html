Bagaimana cara mengecualikan direktori dari cadangan Windows 7 saya?

Saya sedang menyiapkan cadangan untuk komputer teman ke drive USB-nya. Saya sedang membaca artikel ini tentang fitur Pencadangan dan Pemulihan Windows 7.

Semuanya terlihat baik-baik saja, tetapi demi kehidupan saya, saya tidak tahu cara membuka layar ini.

Saya perlu menyiapkan beberapa folder yang dikecualikan, karena media cadangan sedikit lebih kecil dari folder utama. Saya perlu mengecualikan direktori video 100 GB yang dapat diganti.

Anda dapat menggunakan program lain sebagai pengganti utilitas windows bawaan. Saya menggunakan GFI Backup Home Edition ( http://www.gfi.com/backup-hm/backup-hm-features.htm ) - ini adalah versi gratis yang berfungsi dengan baik dan memungkinkan Anda mengecualikan direktori yang ada di dalam direktori yang ingin Anda cadangkan. Ini memiliki sejumlah opsi yang tersedia, dan dicadangkan sebagai salinan (sehingga Anda masih dapat menelusuri cadangan) atau sebagai zip.

Jika Anda memilih opsi "Biarkan saya Pilih", Anda dapat melakukan ini. Triknya adalah dengan mengabaikan (hapus centang semua kotak di) bagian berlabel "Data Files". Kemudian, Anda dapat memilih semua folder yang Anda inginkan di bagian "Komputer".

Jika Anda awalnya mencentang kotak di bawah folder Users \ accountname \ Anda, itu akan memeriksa semua subfolder. Kemudian Anda dapat menelusuri dan menghapus centang satu per satu subfolder yang tidak Anda inginkan.

Jika Anda menggunakan metode ini, Anda tidak perlu melakukan peretasan di mana Anda meletakkan semua file yang tidak ingin Anda cadangkan di beberapa subdirektori dari salah satu "perpustakaan" yang dikenali Windows.

Ini tautan menunjukkan layar secara lebih rinci, tetapi mengabaikan metode yang saya usulkan. Tangkapan layar terpenting adalah tangkapan layar dengan pesan "Apa yang ingin Anda cadangkan?" dialog.

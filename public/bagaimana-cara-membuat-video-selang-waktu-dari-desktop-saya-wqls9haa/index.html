Ada beberapa aplikasi di luar sana yang dapat menangkap layar secara otomatis, AutoScreenShot adalah salah satu contohnya.

Kemudian ketika Anda memiliki gambar membuat film dari mereka tidak bahwa keras.

Membuat film dari file gambar menggunakan ffmpeg / mencoder

Versi pendek:

Kita juga dapat menggunakan ffmpeg secara langsung untuk menyandikan file gambar ke dalam film. Jika kita mulai dengan nama file 001.jpg, 002.jpg, ..., kita dapat menggunakan:
ffmpeg -r 10 -b 1800 -i %03d.jpg test1800.mp4

Memotong Ubuntu ke tulang untuk Virtualbox VM

Saya telah mencari-cari varian Linux yang akan diinstal hanya perangkat lunak yang saya butuhkan daripada semua yang Ubuntu (misalnya) pasang secara default. Ini untuk membuat mesin virtual di Virtualbox yang memiliki bash, apache, python, perl, SQLite, openssh dan beberapa program lain tetapi tidak ada yang lain. Saya lebih suka menggunakan Ubuntu jika memungkinkan tetapi distro modern lain juga akan melakukannya (saya suka menggunakan apt-get dan yum daripada mengunduh / kompilasi dll).

Sejauh ini, saya sudah mencoba:

  • SuseStudio.com, yang mungkin adalah yang terbaik sejauh ini.
  • Ubuntu 9.10, dengan menekan F4 untuk mendapatkan opsi boot pada disk instalasi, tetapi tidak ada instalasi minimal (oops! bergantian disk bukan server cakram).
  • Arch Linux, prosedur penginstalan yang sedikit membingungkan tetapi saya mungkin akan kembali dan mencoba lagi.
  • Gentoo, mulai dengan baik tetapi segera HD pada mesin virtual menjadi 2Gb, bahkan sebelum instalasi dimulai dengan sungguh-sungguh (saya telah mempartisi semua disk).

Saya menyadari ada berbagai Linux "kecil" di sekitar seperti Puppy, Feather, DSL, dll, tetapi tampaknya ditujukan untuk pengguna desktop atau sebagai perangkat teknisi, dan saya ingin distro server sekecil mungkin yang dapat dikelola dengan alat seperti apt atau yum atau serupa.

  1. Unduh ISO server Ubuntu.
  2. Saat boot, tekan F4 dan gunakan "Mesin virtual minimal".

Di VirtualBox Anda perlu mengubah pengaturan VM tamu terlebih dahulu jika Anda ingin menggunakan "mesin virtual minimal" (jika tidak, tamu Ubuntu akan macet):

Enable PAE/NX
Delete the SATA Controller and use SCSI Controller to add the hard disk. Enable host I/O cache.

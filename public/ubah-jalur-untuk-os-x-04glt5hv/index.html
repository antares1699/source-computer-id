Ubah Jalur untuk OS X

saya tidak menanyakan cara mengubah jalur untuk jendela terminal mac OS X.

Saya mengalami masalah ini , dan saya ingin meluncurkannya dengan mengklik. Ini berfungsi jika saya membukanya di jendela terminal (karena jalur diatur dengan benar).

Saya telah mencoba menambahkan variabel jalur ke ~/.MacOSX/environment.plist tetapi tidak berhasil.

Bagaimana cara mengubah jalur untuk pengguna saat ini di Mac OS X?

Tambahkan file dengan jalur yang Anda inginkan di dalamnya (mis. /opt/local/bin) ke direktori /etc/paths.d. Metode ini adalah bagaimana TeXLive melakukannya . Anda juga dapat menambahkan jalur man Anda ke /etc/manpaths.d dengan cara ini.

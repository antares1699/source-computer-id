Ubah trapesium menjadi persegi panjang

Saya menggunakan iPhone saya untuk menangkap lukisan. Namun, sudutnya tidak sempurna, jadi alih-alih mendapatkan persegi panjang lurus, saya mendapatkan trapesium. Saya ingin mengubah trapesium ini kembali menjadi persegi panjang (menggunakan beberapa transformasi affine). Namun, saya tidak dapat menemukan cara yang baik untuk melakukannya. Mohon saran.

Mencadangkan data dengan cara terenkripsi

Saya memiliki kasus penggunaan berikut:

  1. Ada beberapa data dari PC saya yang ingin saya cadangkan secara online
  2. Saya memiliki beberapa hosting, jadi saya ingin menggunakannya untuk backup, tidak ingin membayar ke layanan backup lain
  3. Saya ingin mengenkripsi data saya secara lokal sebelum memindahkannya ke server

Saya tidak memiliki masalah dalam menulis skrip untuk mengotomatiskan proses (katakanlah, secara berkala buat cadangan dan unggah melalui FTP ke server saya), tetapi pertanyaan utama saya adalah tentang langkah 3 - enkripsi: cara mana yang disarankan untuk mengenkripsi file saya (katakanlah, dikumpulkan ke .ZIP) sebelum mengunggah ke server?

P.S. TrueCrypt tampaknya populer tetapi itu bukan yang saya cari, karena saya tidak ingin file terus-menerus dienkripsi di sini di PC saya.

Sebenarnya, TrueCrypt mungkin sangat membantu Anda di sini. Sebagai bagian dari proses otomatis, misalnya melalui file batch, Anda dapat:

  1. Pasang volume TrueCrypt menggunakan Antarmuka baris perintah TrueCrypt (atau lakukan secara manual).

  2. Salin file yang ingin Anda cadangkan ke volume yang terpasang.

  3. Lepaskan volumenya.

  4. Unggah wadah TrueCrypt ke server dengan cara yang Anda pilih.

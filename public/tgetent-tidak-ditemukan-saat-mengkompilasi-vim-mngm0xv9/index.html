tgetent () tidak ditemukan saat mengkompilasi vim

Saat ini saya mencoba menginstal ekstensi YouCompleteMe ke pengaturan vim saya, tetapi vim saya memerlukan versi 7.3.584 atau lebih tinggi. Vim yang dikirimkan dengan Ubuntu 12.10 saya adalah 7.3.547, jadi saya harus mengkompilasi ulang vim dari sumber berikut instruksi ini .

Seperti yang dikatakan instruksi, saya mengeksekusi:

sudo apt-get install libncurses5-dev libgnome2-dev libgnomeui-dev \
libgtk2.0-dev libatk1.0-dev libbonoboui2-dev \
libcairo2-dev libx11-dev libxpm-dev libxt-dev python-dev ruby-dev mercurial

yang seharusnya memberi saya semua paket yang dibutuhkan untuk kompilasi. Namun, ketika saya telah mengkloning sumber vim dan baru saja akan mulai mengubah konfigurasi untuk kompilasi, saya mengerti

...
checking for tgetent in -lcurses... yes
termcap library is not usable
no terminal library found
checking for tgetent()... configure: error: NOT FOUND!
    You need to install a terminal library; for example ncurses.
    Or specify the name of the library with --with-tlib.

Saya melakukan beberapa penelitian, dan mendapatkan pengertian umum bahwa saya harus menetapkan ncurses sebagai perpustakaan terminal saya. Tapi saat aku melakukannya

./configure --with-tlib=ncurses

itu berteriak kembali

...
checking --with-tlib argument... ncurses
chekcing for linking with ncurses library... OK
chekcing for tgetent()... configure: error: NOT FOUND!
    You need to install a terminal library; for example ncurses.
    Or specify the name of the library with --with-tlib.

Saya tidak tahu apakah ada sesuatu yang saya lewatkan, atau apakah ada cara lain untuk mendapatkan versi terbaru dari vim? Terimakasih banyak!

Melarang windows mengganti namanya sendiri di tmux

Saya telah menginstal tmux pada sistem OS X dengan zsh yang disediakan oleh oh-my-zsh.

Saya biasanya menggunakan cukup banyak jendela tmux untuk sejumlah besar sesi ssh, dan saya suka membiarkan jendela ini dinamai sesuai nama host yang saya gunakan sehingga lebih mudah bagi saya untuk mencari tahu jendela mana yang berisi apa.

Saya sudah mencoba menambahkan baris konfigurasi seperti set-window-option -g automatic-rename off dan set-option -g allow-rename off ke ~ / .tmux.conf saya, namun jika saya memulai sesi tmux baru, ganti nama jendela dan cd ke direktori yang berbeda tmux dengan senang hati mengganti nama jendela lagi agar sesuai dengan direktori kerja saat ini .

Bagaimana saya bisa memberi nama windows tanpa mereka mengubah saya secara otomatis?

Saya sadar bahwa ini tidak menjawab pertanyaan langsung Anda tetapi dapat membantu Anda mendapatkan perilaku yang diinginkan - dan ini terlalu panjang untuk memberikan komentar

IMHO Anda sebaiknya menyembuhkan penyebabnya dan tidak mencoba meredakan gejalanya ... yang saya maksud adalah sebagai berikut: Di suatu tempat di konfigurasi shell Anda (mungkin di salah satu plugin oh-my-zsh Anda yang diaktifkan) judul jendela disetel setiap saat Anda mengubah direktori saat ini. Kemungkinan besar dalam fungsi precmd() atau chpwd(). Jadi periksa dulu outputnya

$ which precmd
$ which chpwd

dan coba temukan sumbernya di file zsh rc Anda.

Solusinya bisa, untuk mengganti fungsi ini (dengan tmux ganti nama otomatis opsi dimatikan dan izinkan-ganti nama di):

$ chpwd() { }
$ precmd() {
    if [[ "$TERM" = "screen" ]] && [[ -n "$TMUX" ]]; then
       print -Pn "\033k%m\033\\"
    fi
  }

Catatan:

  • Saya menggunakan cek jika Anda dalam sesi tmux dari jawaban itu
  • Judul jendela diubah melalui pencetakan urutan escape setiap kali sebelum prompt digambar ulang
  • print -P menggunakan perluasan yang sama seperti pada prompt (lihat PERLUASAN URUTAN PROMPT di man zshmisc), jadi %m diperluas ke nama host hingga titik pertama.

Pastikan, Anda memasukkan kode ini ke file ~/.zshrc Anda setelah oh-my-zsh dimuat.

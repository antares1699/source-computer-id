Bagaimana cara mengetahui file RAR mana yang rusak?

Saya memiliki RAR yang saya unduh yang terdiri dari banyak file. Saat mencoba mengekstraknya, ia memberi tahu saya bahwa datanya tidak lengkap, sesuatu yang buruk dengan salah satu file. Mengingat saya harus mengunduh banyak file besar, apakah ada cara untuk mengetahui dengan tepat file mana yang harus saya unduh ulang?

Saya menggunakan Mac.

Pertimbangkan untuk mengunduh UnRarX untuk menguji integritas file.

Buka UnRarX.app, klik Jelajahi (atau Cmd + B ), pilih file RAR Anda. Saya kira memilih yang pertama sudah cukup. Lalu klik Uji . Ini akan memindai setiap file RAR dan memberi Anda pesan kesalahan yang lebih spesifik tentang mana yang rusak.

Saya lebih suka UnRarX daripada Pembantu Arsip default di OS X, karena terkadang memberikan hasil yang lebih baik saat membuka file RAR yang dilindungi kata sandi.

Bagi saya, jawaban yang diterima tidak berhasil karena beberapa alasan. Mungkin UnrarX telah mengubah sesuatu di versi mereka yang lebih baru, atau ada sesuatu yang lucu tentang file saya. Itu hanya tidak menunjukkan hasil yang benar untuk file saya. Bagaimanapun, inilah cara saya menyelesaikannya:

Saya mengunduh rar alat baris perintah dari Rarlab . Unduh paket 'RAR untuk OS X'. Meskipun tertulis trial, bagian unrar sebenarnya adalah freeware. Untuk menguji arsip, jalankan

unrar t /path/to/file.part01.rar

Ini akan menunjukkan kepada Anda file mana yang rusak. Setelah mengganti file, Anda dapat mengekstrak menggunakan program gui atau dengan menjalankan

unrar x /path/to/file.part01.rar

Program apa yang Anda gunakan untuk mengekstrak file rar? Ini harus memberi tahu Anda file mana yang rusak.

Anda mengatakan bahwa Anda mengunduh beberapa file, melihatnya di Finder, Anda mungkin dapat mengetahui yang rusak dengan fakta bahwa ukuran file secara nyata lebih kecil daripada yang lain. Idealnya Anda tidak perlu mengunduh ulang seluruh file yang buruk. Poster asli RAR juga harus disertakan .Par atau file .Par2 hanya untuk situasi ini.

Sebuah program seperti MacPar Deluxe tidak hanya akan menghapus file Anda tetapi juga menggunakan file .par untuk membangun kembali data yang buruk atau terkadang hilang.

Bagaimana cara menginstal Zope di Windows?

Saya sedang mencoba menyiapkan BuildBot . BuildBot membutuhkan Memutar , yang pada gilirannya membutuhkan Zope .

Saya tidak dapat mengatur hal-hal Zope ini. Saya memiliki kode sumber lengkap untuk Zope-3.4.0, tidak ada yang berjalan seperti yang dikatakan readme. Saya juga memiliki file ZIP yang berupa telur. Apakah Telur itu?

Anda tidak memerlukan paket Zope lengkap untuk menginstal Twisted. Yang Anda butuhkan hanyalah Zope.Interface, yang merupakan komponen independen dari Zope. Anda harus dapat menginstal Twisted menggunakan binari (Zope.Interface, termasuk) yang ditemukan sini .

Untuk menjawab pertanyaan terakhir Anda:

Telur Python dijelaskan dalam: " Telur Python - Pengantar Sederhana ".

Itu juga menyebutkan Zope.

Instal Mudah yang digunakan untuk memasang telur adalah bagian dari setuptools.

Petunjuk penginstalan Windows untuk alat pengaturan (tautan ke penginstal Windows ada di akhir halaman itu).

Kutipan:

Setelah instalasi selesai, Anda akan menemukan program easy_install.exe di subdirektori Python Scripts Anda. Pastikan untuk menambahkan direktori ini ke variabel lingkungan PATH Anda, jika Anda belum melakukannya.

Memasang hard drive 3,5 "dalam ruang 2,5"?

Saya tahu pertanyaan ini tampak konyol, tetapi server saya (sebenarnya komputer Mini-ITX) memiliki tempat hard drive 2,5 "dan tempat CD-ROM kosong di bawahnya. Jika saya melepas tempat hard drive, saya mungkin dapat memasang 3,5" hard drive di tempat CD-ROM yang tidak digunakan ...

Saya tahu ini cara besar untuk melakukan sesuatu tetapi itu berarti meningkatkan hard drive ke 2TB jauh lebih murah.

Pertanyaan saya adalah: apakah persyaratan daya dari disk 3,5 "sama dengan 2,5"? Saya tahu konektornya secara fisik masih cocok, tetapi saya tidak ingin merusak server saya!

Drive 3,5 inci berputar pada 5400rpm (semua drive ramah lingkungan, dll.) Dibandingkan dengan drive 2,5 "menggunakan daya sekitar dua kali lipat saat mencari (seperti dalam 6 watt hingga 3 watt) dan empat kali saat idle (4 watt hingga 1 watt).

Hal yang harus diperhatikan adalah penggunaan power spin-up. Drive 3,5 "dapat menggunakan sebanyak 2,5 amp pada saluran 12V (30 watt) untuk detik pertama saat booting.

Biasanya tidak menjadi masalah kecuali jika Anda menggunakan picoPSU daya rendah atau konverter DC-DC terintegrasi - maka Anda harus memeriksa peringkat puncak saluran 12v Anda.

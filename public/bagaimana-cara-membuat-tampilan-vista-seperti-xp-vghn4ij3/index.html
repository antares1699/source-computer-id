Bagaimana cara membuat tampilan Vista seperti XP?

Saya merekam bagian dari pelatihan video dengan XP dan sekarang saya telah menginstal Vista.

Saya ingin tampilan jendela (tema tombol permen XP) sama di semua video.

Bagaimana saya bisa membuat Vista memiliki tombol merah biru cerah dengan kancing permen seperti yang dimiliki XP :

Saat saya masuk ke Pengaturan Penampilan di vista, saya hanya dapat memilih dari:

  • Windows Aero
  • Windows Vista Basic
  • Standar Windows
  • Windows Klasik

tapi tidak ada Windows XP.

Saya suka beberapa produk Stardock seperti di atas, tetapi tanpa memerlukan program pihak ke-3, Anda dapat membuka panel kontrol> sistem, lalu klik pengaturan sistem lanjutan.

Klik tombol kinerja dan klik "Sesuaikan untuk kinerja terbaik" dan komputer Anda akan terlihat seperti Windows XP ketika dalam mode dasar / tanpa tema.

Tidak sebagus Stardock, tapi berhasil!

Untuk sentuhan lain, edit properti start menu / task bar untuk hal-hal seperti no stacking e.t.c.

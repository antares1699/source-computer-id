Bagaimana cara kembali ke baris perintah setelah meluncurkan skrip Node.js?

Saya menggunakan Mac OS X 10.6.7.

Saya memulai skrip node.js, yang memulai server HTTP.

Masalahnya adalah saya kehilangan baris perintah, terlihat seperti ini:

$me cd directory
$me node test.js
Message from test.js: Server is running on localhost
..|

| = menunjukkan tanda sisipan

Dimungkinkan untuk menulis tetapi saya tidak dapat menjalankan kode apa pun, ini seperti menulis string biasa, bagaimana saya bisa kembali ke baris perintah?

Maaf saya benar-benar noob di terminal :)

tekan Ctrl + C untuk menghentikan program dan kembali ke prompt shell.

Jika Anda ingin program berjalan di latar belakang dan mengembalikan Anda ke prompt shell, tambahkan ampersand (&) di akhir perintah. Sebagai contoh:

node test.js &

Buka saja tab baru dengan menekan Cmd-T, atau jendela baru (menggunakan Cmd-N). Anda ingin mendapatkan peringatan / pesan kesalahan yang dikirimkan program ke terminal Anda.

Anda juga dapat menggunakan screen untuk mendapatkan beberapa ... er ... terminal dalam satu tab / jendela.

Tekan Ctrl-A Ctrl-C untuk membuat terminal baru, Ctrl-A Ctrl-N atau Ctrl-A Ctrl-P (berulang kali) untuk pergi ke terminal berikutnya dan sebelumnya. Ketik exit di semua terminal screen untuk pergi.

Mengenai node: Periksa dokumentasi - mungkin saja untuk memulainya sebagai server latar belakang.

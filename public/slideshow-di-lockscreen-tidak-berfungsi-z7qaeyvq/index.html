Saya tidak berpikir ini adalah masalah dengan tayangan slide itu sendiri, saya pikir ini lebih berkaitan dengan jendela yang tidak melihat gambar apa pun yang dapat digunakan untuk tayangan slide.

Jika itu saya, saya akan melakukan hal berikut:

  1. Buat folder bernama "testlide" (atau apa pun)
  2. Masukkan 3 atau 4 gambar ke dalamnya
  3. Pilih Add a folder dari layar screenshot Anda.
  4. Temukan folder "testlide" ini .... apakah windows menampilkan gambar di sini?
  5. Pilih Choose this folder
  6. Lihat apakah pesan You need more than one picture to show in your slide show telah hilang.

Pada titik ini, baik a) semuanya sekarang berfungsi - buat folder dengan gambar yang Anda inginkan. b) gambar Anda tidak sesuai karena beberapa alasan .. coba gambar dengan ekstensi berbeda (bmp, jpg, dll) ATAU c) ada yang tidak beres dengan pemasangan windows Anda - tetapi menurut saya tidak demikian.

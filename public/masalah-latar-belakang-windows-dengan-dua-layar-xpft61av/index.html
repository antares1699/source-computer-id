Masalah latar belakang Windows dengan dua layar

Saya menjalankan Windows Vista 64-bit pada laptop HP Compaq 6510b dengan layar eksternal terpasang.

  • Layar built-in memiliki resolusi 1400x900.
  • Layar eksternal memiliki resolusi 1920x1200.

Setiap gambar latar belakang yang saya atur ditampilkan dengan ukuran maksimum 1400x900, jadi tidak mungkin untuk mengisi latar belakang layar lebar.

Apakah ada solusi agar layar besar juga menampilkan gambar penuh atau saya dapat menyetel gambar latar belakang terpisah per layar seperti di Mac OS X?

Platipus akan melakukan pekerjaan itu dengan adil.

Namun, jika Anda tidak ingin menginstal perangkat lunak lain, Anda juga dapat mencapai hal yang sama menggunakan AppleScript:

Langkah 1: buka Apple Script Editor dari pencarian Spotlight.

Langkah 2: tulis perintah ini

to run
    do shell script "path_to_unix_executable"
end run

Langkah 3: Simpan sebagai bundel aplikasi.

Langkah 4: Lari

Alternatif 1) mktorrent untuk Windows :

Mktorrent adalah alat yang sangat mudah untuk membuat Torrents di commandline, tetapi hanya untuk Linux (hanya Kode Sumber ke mengkompilasi diri) - jadi saya mengkompilasi a Versi Win32 / Win64 melalui Cygwin.

Instalasi: Salin mktorrent.exe dan DLL di file \ Windows \ System32 \
Di Server 2008 R2 Salin mktorrent.exe di file \ Windows \ System32 \, dan DLL dalam format \ Windows \ SysWOW64 \

Penggunaan yaitu: mktorrent -l 21 -a http: //tracker.url nama folder

Alternatif 2) py3createtorrent :

py3createtorrent adalah program yang komprehensif shell / commandline utilitas untuk membuat torrents (Linux & Windows). Itu a Skrip Python v3.1 berlisensi GPL. saya mengujinya dengan Ubuntu 8.04 / rTorrent dan Windows 7 / µTorrent.

Penularan dilengkapi dengan transmission-create yang merupakan utilitas baris perintah untuk membuat .torrent file.

Usage: transmission-create [options] <file|directory>

Options:
 -h --help              Display this help page and exit
 -p --private           Allow this torrent to only be used with the specified
                        tracker(s)
 -o --outfile <file>    Save the generated .torrent to this filename
 -c --comment <comment> Add a comment
 -t --tracker <url>     Add a tracker's announce URL
 -V --version           Show version number and exit

Petunjuk untuk membuat Transmisi di Windows.

mktorrent disebutkan oleh Mehper C. Palavuzlar akan melakukan pekerjaan itu. Penggunaan sampel adalah

    mktorrent.exe -l %piece_size% -a %tracker_announce_url% %folder_or_file%

Ukuran bidak adalah pangkat 2, yaitu -l 22 akan membuat file torrent dengan kepingan 2 ^ 22 byte = 4 MB.

mktorrent saat ini dikelola oleh Rudde :
https://github.com/Rudde/mktorrent

Untuk membangunnya di Windows dari sumber, Anda memerlukan GNU Make yang dapat Anda peroleh MinGW-w64.dll , sebagai contoh.

Untuk membangun mktorrent untuk Windows 32-bit (asalkan Anda memiliki make dalam variabel lingkungan Windows PATH Anda dan direktori kerja saat ini berisi file kode sumber mktorrent):

    make CC=i686-w64-mingw32-gcc USE_LONG_OPTIONS=1 USE_LARGE_FILES=1

Untuk membuat mktorrent untuk Windows 64-bit:

    make CC=x86_64-w64-mingw32-gcc USE_LONG_OPTIONS=1 USE_LARGE_FILES=1

mktorrent.exe dan prefix.exe akan dibangun.

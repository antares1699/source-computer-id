Seberapa rendah daya yang didapat server rumah? [Tutup]

Sebenarnya saya punya pertanyaan yang cukup sederhana. Betapa hijau, rendahnya daya, dan server rumah x86 yang efisien dapat saya bangun menggunakan suku cadang konsumen dengan anggaran yang agak terbatas.

Setelah melihat-lihat beberapa hit Google, saya menemukan bahwa sistem itu didasarkan pada atom inti ganda, beberapa papan mITX sederhana (gigabit lan, audio terintegrasi dan gfx dll), satu modul RAM dan satu HDD WD 'hijau', didukung oleh picoITX PSU menggunakan sekitar 30W saat idle hingga 40 saat memuat.

Bisakah Anda lebih rendah (atau seberapa rendah) dari itu? Mungkin beberapa chip nano VIA, atau atom inti tunggal?

Server rumah saya akan menangani beberapa back-up yang dicampur dengan sedikit lalu lintas ftp / http.

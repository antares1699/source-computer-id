Bagaimana cara menjadikan bootloader Windows sebagai default?

Ketika saya mendapatkan komputer baru, saya menginstal Ubuntu dan Windows XP di dalamnya. Ketika saya menyalakannya, GRUB muncul dan memungkinkan saya memilih di antara OS. Baiklah sejauh ini.

Beberapa saat kemudian, saya memutuskan saya tidak ingin Ubuntu lagi (fanboy Linux, ulangi). Saya menghapus partisinya, tetapi saya terlambat menyadari bahwa GRUB bergantung padanya, sehingga komputer tidak bisa boot. Saya menginstal Ubuntu lagi dan berfungsi dengan baik. Jadi bagaimana cara memberi tahu komputer bahwa saya ingin boot langsung ke Windows?

Anda perlu menginstal ulang Windows NT Loader (ya, NT-nya bahkan untuk XP) ke partisi boot utama. Untungnya, melakukannya sangat mudah.

  1. Boot dalam mode Pemulihan dari file CD XP . Anda perlu boot dari CD untuk melakukan ini.
  2. Jalankan fixboot
  3. Jalankan fixmbr
  4. Mulai ulang

Seperti yang ditambahkan GeneQ pada komentar di bawah, jika fixboot dan fixmbr gagal, Anda selalu dapat menggunakan fdisk /mbr. Metode fdisk akan bekerja pada komputer lama yang memiliki diska boot MSDOS / 9x.

Kapan lagi gagal (jika tidak, gunakan metode fixboot / fixmbr) dan perlu kembali ke bootloader Windows XP Anda alih-alih yang diinstal oleh distro Linux Anda, cukup ikuti petunjuk ini:

  1. Boot dengan disk Windows XP Anda.
  2. Pilih opsi Recovery Console.
  3. Saat diminta, ketik: fdisk / mbr
  4. Hidupkan Kembali komputer Anda.

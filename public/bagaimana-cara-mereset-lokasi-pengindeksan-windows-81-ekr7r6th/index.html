Bagaimana cara mereset Lokasi Pengindeksan (Windows 8.1)?

Saya telah menghapus semua lokasi pengindeksan saya dari Windows 8.1 saya, dan sekarang saya ingin menambahkannya kembali.

Bagaimana cara memulihkannya secara otomatis? Ketika saya mencoba menambahkannya secara manual, saya tidak dapat menemukan folder dengan nama Start Menu, dan setelah saya memilih yang lain (%appdata% dan ie history) saya masih tidak dapat menemukan aplikasi Start Menu ketika saya menekan Windows + S .

Saya telah mencoba menyegarkan komputer saya satu bulan yang lalu tanpa mengatur ulang semuanya ... dan setelah itu saya masih tidak dapat menemukan aplikasi itu (Suara), dan lainnya, seperti Disk Clean.

Dan satu hal lagi: Ketika saya melakukan pencarian, penggunaan CPU saya mencapai 100% (AMD 965 x4 3.40)

Di Windows 10 aturan lokasi pengindeksan disimpan dalam registri di lokasi berikut: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search\CrawlScopeManager\Windows\SystemIndex\WorkingSetRules (di win 8 atau lebih rendah bisa jadi di lokasi tetangga, instruksi lainnya akan sama)

Untuk mengatur ulang aturan lokasi pengindeksan ke defaultnya, hapus kunci ini. (cadangan)

Namun menghapusnya mungkin tidak langsung, ini akan memberikan kesalahan "akses ditolak", karena mungkin ada beberapa layanan yang menggunakannya. Cara terbaik untuk mengedit registri ini adalah melalui lingkungan Konsol Pemulihan Windows yang dapat diakses melalui Settings > Update & Security > Recovery > Advanced Startup > Restart now kemudian Troubleshoot > Advanced options > Command Prompt

Ketika restart dan prompt perintah terbuka, ketik regedit untuk membuka Editor Registri. Namun registri dimuat di sini secara default, serta drive C:\, bukan milikmu yang sebenarnya , tetapi lingkungan Konsol Pemulihan. Drive Sistem Anda yang sebenarnya harus E:\ atau sesuatu. Pada langkah selanjutnya Anda akan dapat menelusuri drive melalui Open File Dialog, mencari tahu dengan label drive Anda, atau file E:\Users\<your account name> yang merupakan drive milik Anda. Registri di dalamnya harus dalam direktori Troubleshoot > Advanced options > Command Prompt, berisi file tertentu (atau "sarang"), yang Anda inginkan adalah SOFTWARE.

Jadi di panel kiri Editor Registri, pilih HKEY_LOCAL_MACHINE dan kemudian goto File > Load Hive dan temukan E:\Windows\System32\config\SOFTWARE itu. Ini akan meminta nama untuk menampilkan sarang yang dimuat, cukup ketik SOFTWARE_real atau semacamnya. Kemudian hapus kunci registri yang disebutkan di atas dari sarang barunya HKLM\SOFTWARE_real\...\Windows Search\...\WorkingSetRules.

Itu saja, tutup, keluar, dan mulai ulang secara normal. Ini akan membuat ulang nilai registri default di tempat yang dihapus dan Opsi Pengindeksan sekarang harus memiliki Lokasi default tersebut.

Pencarian universal Windows 8.1 hanya menampilkan file dari folder yang telah diindeks. Salah satu cara untuk memperluas jangkauannya adalah dengan menambahkan folder atau seluruh drive ke indeks. Namun, proses pengindeksan file Anda dan menjaga indeks tersebut tetap mutakhir memakan sumber daya sistem, jadi, jika Anda mengatur Windows untuk mengindeks seluruh drive sistem, Anda dapat memperlambat komputer. Anda lebih baik memilih folder mana yang ingin Anda indeks. Berikut ini cara menambahkan lebih banyak folder atau seluruh drive ke indeks Anda.

  1. Buka Panel Kontrol Windows.
  2. Klik Opsi Pengindeksan. Jika Anda tidak melihat ikon untuk Opsi Pengindeksan pilih Ikon Besar dari menu Lihat berdasarkan.
  3. Klik Ubah.
  4. Pilih folder yang ingin Anda indeks dan Klik Ok. Sistem akan mengindeks subfolder di bawahnya. Jika Anda memilih seluruh drive, itu akan mengindeks semua folder di bawah level root.
  5. Klik Tutup. Sistem memerlukan beberapa menit untuk menambahkan lokasi baru yang Anda tambahkan ke indeksnya.

P.S: Mengenai lokasi menu mulai: Jika Anda tidak melihat folder Appdata, Anda perlu mengaktifkan file tersembunyi.

Menu Mulai Per-pengguna: C: \ Users [nama pengguna] \ AppData \ Roaming \ Microsoft \ Windows \ Start Menu Menu Mulai Semua pengguna: C: \ ProgramData \ Microsoft \ Windows \ Start Menu

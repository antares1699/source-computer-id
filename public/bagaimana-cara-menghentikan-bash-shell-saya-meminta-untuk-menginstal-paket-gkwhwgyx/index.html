Bagaimana cara menghentikan bash shell saya meminta untuk menginstal paket?

Ketika saya salah mengetik perintah, shell mencoba menemukan paket yang menyediakannya. Bagaimana cara menonaktifkan fitur ini?

Contoh:

[dave@dave cpp]$ sl
bash: sl: command not found...
Install package 'sl' to provide command 'sl'? [N/y]

Saya ingin jika saya tidak diminta untuk menginstal perintah.

Masalahnya adalah PackageKitCommandNotFound . Menghapusnya

yum hapus PackageKit-command-not-found

harus melakukan trik, menurut milis .

Setelah melakukan a

yum remove PackageKit-command-not-found

jika Anda mendapatkan kesalahan,

bash: /usr/libexec/pk-command-not-found: No such file or directory

Anda perlu menambahkan baris berikut ke ~ / .bashrc

unset command_not_found_handle
Bagikan

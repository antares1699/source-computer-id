Apakah mungkin untuk membuka file ke nomor baris tertentu seperti yang diformat dalam hasil grep -n?

Saya sering menggunakan grep dengan opsi -n untuk mencantumkan nomor baris, misalnya

$ grep -n xyz .*
/ext1/acheong/foo/bar/a.cc:42:    inline bool abc(int xyz);
/ext1/acheong/foo/bar/a.cc:43:    inline bool def(int xyz);
/ext1/acheong/foo/bar/b.cc:415:            int xyz = result;
/ext1/acheong/foo/bar/b.cc:490:                xyz += result;
$

Apakah mungkin untuk alias vim sedemikian rupa

vim /ext1/acheong/foo/bar/b.cc:415

akan membuka file dan mencari baris 415 secara otomatis?

Apakah mungkin pada csh? (Ya, ya, csh memunculkan iblis dan semua yang tidak suci; tapi saya tidak punya pilihan ...) Jawaban untuk kerang lain akan berguna juga.

Nah, ini bukan alias, Anda bisa menggunakan vim seperti ini:

vim +<LineNumberHere> fileName

Jadi misalnya

vim +150 .bash_history

membuka file .bash_history Anda (untuk bash), dan menavigasi ke baris 150.

Secara kebetulan, Anda juga dapat melakukan ini dengan istilah penelusuran. Sebagai contoh

vim +/MyTerm MyFile

membuka MyFile dan menavigasi ke kemunculan pertama MyTerm dari atas.

Nikmati!

Salah satu solusi untuk ini adalah dengan menggunakan file file_line.vim plugin, yang memungkinkan Anda menentukan nama file dan nomor baris sebagai argumen untuk Vim atau pada baris perintah Vim, seperti yang Anda tunjukkan.

Lainnya adalah skrip ini,

$ cat $(which vimgrep)
#!/bin/bash

tmp=$(mktemp)

cat > $tmp
exec < /dev/tty

vim --cmd 'let &efm=&gfm' -q $tmp "$@"
rm $tmp

yang bisa digunakan seperti ini:

$ grep -Hn xyz .* | vimgrep

dan yang memuat semua kecocokan ke dalam daftar kesalahan perbaikan cepat Vim. Lihat

:help quickfix.txt

Perhatikan opsi -H untuk grep untuk memastikan bahwa nama file disertakan dalam keluaran grep bahkan jika .* diperluas menjadi hanya satu nama file.

Cara paling sederhana dan tangguh:

  1. Buat file baru ~/.bin/vim dan tempel kode di bawah ini
  2. Buat alias shell: di ~/.bashrc tambahkan baris alias vi=~/.bin/vim

Ini disebut substitusi parameter dan pada dasarnya akan memisahkan file parameter yang disediakan oleh ' : ' (usus besar).

#!/usr/bin/env bash

if [[ ${1} == *":"* ]]; then
    vim ${1/:*/} -c "${1/*:/}"
else
    vim ${1}
fi

Kemudian Anda bisa menggunakannya sebagai: vim /ext1/acheong/foo/bar/b.cc:415

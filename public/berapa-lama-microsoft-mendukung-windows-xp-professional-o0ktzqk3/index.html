Berapa lama Microsoft mendukung Windows XP Professional?

Saya menggunakan Windows XP Professional untuk beberapa program perangkat lunak tertentu. Sekarang saya membaca di CT-Magazin Jerman terbaru bahwa Microsoft akan memperbaiki masalah keamanan besar hingga 2014. Di microsoft.com dibicarakan tentang dukungan "umum" (untuk SP3) hingga April 2012. Dukungan yang diperpanjang hingga April 2017, tetapi hanya untuk "pelanggan komersial" (dan apakah saya?).

Apakah ada yang tahu lebih banyak tentang itu?

Pelanggan komersial adalah Perusahaan atau bisnis besar lainnya yang mampu membeli kontrak dukungan yang diperpanjang> (Penting dan Premier) Saya ragu sebagian besar pengguna akhir mampu membelinya.

Dari tautan yang diposting oleh David Statton http://support.microsoft.com/gp/lifepolicy

Pembaruan keamanan akan tersedia melalui akhir fase Dukungan Perpanjangan (lima tahun Dukungan Mainstream ditambah lima tahun Dukungan Perpanjangan) tanpa biaya tambahan untuk sebagian besar produk. Pembaruan keamanan akan diposting di situs Web Pembaruan Microsoft selama fase Mainstream dan Dukungan Diperpanjang.

Tanggal Akhir Dukungan yang Diperpanjang untuk XP Pro adalah 4-8-2014 , setelah itu sebagian besar pengguna XP tidak akan lagi mendapatkan pembaruan keamanan melalui pembaruan windows, sekarang saatnya untuk meninggalkan XP setelah tanggal ini untuk alasan keamanan.

Microsoft dapat mengubah kebijakan atau tanggal ini kapan saja jika mereka mau, yang mereka lakukan untuk Windows 98 tahun yang lalu karena protes Perusahaan, saya ragu ini akan terjadi untuk XP tetapi siapa yang tahu. .

  1. Open notepad as administrator = klik kanan ikon notepad lalu pilih Run as administrator.

  2. Masuk ke menu File -> Open.

  3. Arahkan ke file ini dan buka:

    C:\Windows\System32\Drivers\etc\hosts

  4. Tambahkan baris berikut:

    127.0.0.1 ads.viber.com

  5. Simpan file dan keluar dari notepad.

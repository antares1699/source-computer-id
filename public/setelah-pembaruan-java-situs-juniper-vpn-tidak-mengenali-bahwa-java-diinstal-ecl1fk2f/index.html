Setelah pembaruan Java, situs Juniper VPN tidak mengenali bahwa Java diinstal

Saya menggunakan Chrome dengan Java yang diperbarui (v. 10.51.2.13, SE7 U51). Sebelum Java yang diperbarui, saya bisa mendapatkan VPN pekerjaan saya tanpa masalah apa pun. Sekarang, setelah masuk, saya mendapatkan prompt ini:

Saya mengklik jalankan, dan kemudian saya diarahkan ke layar ini:

Ini secara otomatis mulai mengunduh JuniperSetupClientInstaller.exe . Saya tidak pernah mengalami ini sebelumnya tetapi, untuk berjaga-jaga, saya menjalankannya dan selesai dengan baik. Saya kemudian kembali dan mencoba masuk lagi tetapi hal yang persis sama terjadi.

Tidak yakin langkah apa yang harus saya ambil untuk memecahkan masalah ini. Ada ide?

Akhirnya menemukan perbaikannya:

  1. Buka Panel Kontrol Java Anda (yaitu, pintasan "Konfigurasi Java").
  2. Buka tab "Keamanan"
  3. Di bagian bawah, klik "Edit Daftar Situs ..."
  4. Tambahkan URL VPN Anda tanpa jalur apa pun, mis .: https://www.example.com
  5. Klik oke.

Setelah selesai, tab keamanan Anda akan terlihat seperti ini:

Semoga ini membantu. Dapatkan solusi melalui KB Juniper .

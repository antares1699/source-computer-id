Bagaimana cara menonaktifkan / mengaktifkan layanan TeamViewer di Linux dan Windows?

Pertanyaan ini untuk tujuan menonaktifkan layanan yang tidak perlu berjalan di latar belakang program yang disebut TeamViewer.

Biasanya, layanan latar belakang berjalan sepanjang waktu.

Saya ingin itu dinonaktifkan dari prompt Administrator PowerShell dan diaktifkan kembali atas permintaan / dan juga dari Terminal Linux.

Lebih disukai dengan satu perintah sederhana atau fungsi yang ditentukan.

Note1: Saya membutuhkan ini di Windows (10) dan Linux (Mint 18).

Note2: Saya butuh solusi CLI.

Linux (Terminal)

  • Mengaktifkan:

    sudo teamviewer daemon enable

  • Menonaktifkan:

    sudo teamviewer daemon disable

Alias ​​pribadi Anda akan terlihat persis seperti itu.

Cukup tambahkan ke file alias Bash pribadi Anda:

~/.bash_aliases

  • Mengaktifkan:

    alias tv-enable='sudo teamviewer daemon enable'

  • Menonaktifkan:

    alias tv-disable='sudo teamviewer daemon disable'

Windows (PowerShell)

  • Mengaktifkan:

    sc.exe config "TeamViewer" start= auto
net start TeamViewer

  • Menonaktifkan:

    sc.exe config "TeamViewer" start= disabled
net stop TeamViewer

Alias ​​pribadi Anda akan terlihat persis seperti itu.

Cukup tambahkan ke file profil PoweShell pribadi Anda:

$PROFILE

  • Mengaktifkan:

    function tv-enable {
    sc.exe config "TeamViewer" start= auto
    net start TeamViewer
}

  • Menonaktifkan:

    function tv-disable {
    sc.exe config "TeamViewer" start= disabled
    net stop TeamViewer
}

Membaca keluaran dari perintah shell dalam skrip awk

Saya memiliki file teks dengan daftar alamat ip dan informasi lainnya.

Saya menggunakan skrip awk untuk memproses daftar ini dan menghasilkan berbagai perhitungan. Saya ingin memanggil dig -x dari dalam skrip awk dan menggunakan nilai yang dikembalikan.

saya telah mencoba

hostname = system("dig +short -x" ip_address);

tapi yang terjadi adalah -

  1. panggilan untuk menggali mencetak garis ke shell

  2. nama host tetap nol

4

11

Sudah mengerti, tapi jangan ragu untuk menambahkan jawaban yang lebih baik

cmd = "dig +short -x " ;
cmd ip_address | getline hostname;
close(cmd)

Kemudian saya dapat menggunakan nama host di mana saja dalam skrip.

Lakukan penggantian perintah prompt perintah Windows

Bagaimana Anda dapat melakukan penggantian perintah pada prompt perintah Windows?

Substitusi perintah adalah konsep yang sangat kuat dari shell UNIX. Ini digunakan untuk memasukkan output dari satu perintah ke perintah kedua. Misalnya. dengan sebuah tugas:

$ today=$(date) # starts the "date" command, captures its output
$ echo "$today"
Mon Jul 26 13:16:02 MEST 2004

Ini juga dapat digunakan dengan perintah lain selain tugas:

$ echo "Today is $(date +%A), it's $(date +%H:%M)"
Today is Monday, it's 13:21

Ini memanggil perintah tanggal dua kali, pertama kali mencetak hari-minggu, kedua kali untuk waktu sekarang.

Anda tidak dapat melakukannya di DOS.

Jika yang Anda maksud dengan DOS adalah Prosesor Perintah Windows cmd.exe maka Anda bisa mendapatkan output dari perintah dengan for /f:

for /f %%x in ('date') do set "today=%%x"

Bergantung pada kebutuhan Anda, ini bisa menjadi sedikit lebih rumit.

Apakah ada cara untuk mencegah notepad memasukkan baris baru saat menggunakan word wrap?

Saya ingat itu tidak melakukan ini sebelumnya tetapi sekarang ketika saya menulis beberapa paragraf dan menempelkannya ke dalam posting forum, saya melihat baris saya dipecah menjadi baris baru di mana saya harus menggabungkannya secara manual.

Adakah cara untuk mencegah perilaku ini? Saya hanya ingin melihat teks saya terlihat dalam ukuran notepad saya tanpa scrollbar horizontal, tetapi tidak ada teks yang dimodifikasi menurut ini.

Larutan :

Simpan dokumen dan buka kembali di Notepad.

Penjelasan :

Carriage return (CR) dan line feed (LF) dibuat dengan baris baru di file teks Windows, dan kedua karakter diperlukan untuk membuat file teks kompatibel dengan printer dot-matrix lama yang harus diberi tahu untuk memberi makan kertas. CRLF telah menciptakan berbagai macam masalah karena kadang-kadang dua karakter yang tidak terlihat ini tidak dapat digabungkan.

Sejak XP, Notepad terkadang menambahkan CR ekstra saat editor berada dalam mode wordwrap dan dokumen disimpan; dimanapun garis membungkus, CR dimasukkan. Ini membuat garis baru di editor, yang dapat Anda lihat ketika Anda mengubah ukuran klien atau mengedit garis yang ada. Namun, masalah ini memang benar hanya visual, dan dokumen tidak disimpan dengan CR ekstra.

Untuk mengatasi masalah ini, cukup simpan dokumen lalu buka kembali di Notepad dan itu akan terlihat baik-baik saja.

Satu catatan terakhir: ini bukan hanya masalah lukisan tetapi masalah dengan teks di buffer, jadi menyalin teks ke editor lain tidak menyelesaikan masalah.

Komputer

Apakah mungkin untuk mengunduh file yang sangat besar dengan cerdas atau sebagian melalui SSH dari Linux ke Windows?

Saya memiliki file ~ 35 GB di server Ubuntu Linux jarak jauh. Secara lokal, saya menjalankan Windows XP, jadi saya menyambung ke server Linux jarak jauh menggunakan SSH (secara khusus, saya menggunakan program Windows yang disebut Klien SSH Secure Shell versi 3.3.2).

Meskipun koneksi internet broadband saya cukup bagus, unduhan saya untuk file besar sering gagal dengan pesan kesalahan Connection Lost. Saya tidak yakin, tetapi saya pikir itu gagal karena mungkin koneksi internet saya terputus selama satu atau dua detik setiap beberapa jam. Karena ukuran file sangat besar, pengunduhan mungkin memerlukan waktu 4,5 hingga 5 jam, dan mungkin koneksi internet terputus selama satu atau dua detik selama waktu yang lama itu. Saya pikir ini karena saya telah berhasil mengunduh file dengan ukuran ini menggunakan koneksi internet yang sama dan perangkat lunak SSH yang sama di komputer yang sama. Dengan kata lain, terkadang saya beruntung dan unduhan selesai sebelum koneksi internet terputus sedetik.

Adakah cara agar saya dapat mengunduh file dengan cara yang cerdas - di mana sistem operasi atau perangkat lunak "tahu" di mana ia berhenti dan dapat melanjutkan dari titik terakhir jika koneksi internet terputus?

Mungkinkah mengunduh file dalam beberapa bagian? Meskipun saya tidak tahu apakah saya dapat dengan mudah membagi file saya menjadi beberapa file - menurut saya ini akan sangat sulit, karena file tersebut biner dan tidak dapat dibaca manusia.

Seperti sekarang, jika seluruh unduhan file ~ 35 GB tidak selesai sebelum koneksi terputus, maka saya harus memulai pengunduhan dan menimpa potongan ~ 5-20 GB yang telah diunduh secara lokal sejauh ini.

Apakah Anda punya saran? Terima kasih.

Solusi 'tepat' saya adalah menemukan dan memperbaiki apa yang menyebabkan masalah, tetapi hal-hal ini mungkin berfungsi sebagai solusi:

  1. gunakan split (man split - pisahkan file menjadi beberapa bagian). Tampaknya diinstal pada sebagian besar sistem unix.
  2. Pisahkan file melalui dd (dd if = inputfilename of = file_part0 skip = 0 bs = 500MB). (ulangi dengan lewati = 1 dan nama file yang berbeda.)
  3. Gunakan program yang dapat melanjutkan pengunduhan. FTP akan berfungsi (tapi eww. Kata sandi teks biasa, kontrol terpisah dan koneksi data yang tidak cocok dengan kebanyakan firewall)
  4. Cheat dan pindahkan file ke direktori web. Kebanyakan brosers mendukung melanjutkan unduhan,

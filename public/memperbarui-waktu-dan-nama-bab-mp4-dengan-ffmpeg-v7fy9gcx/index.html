Memperbarui waktu dan nama bab mp4 dengan ffmpeg?

Saya mencoba menambahkan bab khusus dengan ffmpeg ke file mp4 dengan mengekstrak file metadata dan mengikuti panduan ini .

Namun, ketika saya mengedit file metadata untuk mengubah salah satu nama bab, perintah kedua untuk menambahkan kembali metadata mengabaikan perubahan pada bab dan membiarkannya apa adanya. Itu pasti membaca di file metadata dengan benar, seperti ketika ffmpeg mencetak data dari dua file masukan, itu mencetak aliran mp4 dan semua metadata dari file metadata, dengan semua bab diformat dan diurai dengan benar.

Segera setelah itu membuat file keluaran, ia memiliki nama bab yang lama lagi. Jika saya mengubah metadata lain yang berfungsi (seperti kunci judul atau kunci artis), tetapi bab tidak.

Diselesaikan - sepertinya -map_metadata hanya melakukan tombol atas sebelum bab, Anda perlu -map_chapters untuk mendapatkan bab juga.

ffmpeg -i INPUT -i FFMETADATAFILE -map_metadata 1 -map_chapters 1 -codec copy OUTPUT

Anda dapat menormalkan volume file suara dengan SoX cukup mudah. Ini dijuluki pisau Swiss Army untuk program pengolah suara dan memiliki banyak pilihan.

Ada juga normalize dan digunakan seperti ini:

normalize -b file1.wav file2.wav file3.wav

Saya belum pernah menggunakan normalisasi diri saya sendiri.

Namun SoX dan normalize bekerja dengan memodifikasi data audio. Jika Anda tidak mau, Anda bisa mencobanya mendapatkan kembali . Itu masih memodifikasi file audio tetapi hanya meta data.

Bagaimana saya bisa mengupgrade paket Chocolatey jika sudah terinstal, dan menginstalnya jika belum?

Saya sedang menulis satu set skrip PS untuk membantu mengatur dan memelihara lingkungan pengembang Windows. Saat ini, saya mempertahankan dua file skrip, satu untuk menginstal paket Chocolatey, dan satu lagi untuk mengupgrade paket yang sama.

Kedua skrip tersebut sangat lugas, kurang lebih terlihat seperti ini:

# choco-install.ps1
choco install git
choco install VisualStudioCode
# ...

# choco-upgrade.ps1
choco upgrade git
choco upgrade VisualStudioCode
# ...

Saya ingin menggabungkan ini menjadi satu skrip yang, untuk setiap paket, akan menginstalnya jika belum diinstal, dan mengupgrade paket jika sudah diinstal. Saya ingin dapat membagikan skrip ini, oleh karena itu skrip tersebut tidak boleh mengganggu paket Chocolatey lain yang mungkin telah diinstal oleh pengguna.

Apakah ada perintah cokelat bawaan untuk melakukannya, atau trik PowerShell lain yang bisa saya gunakan?

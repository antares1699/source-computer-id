Tidak! Protokol IRC tidak menggunakan / s. Misalnya, mengubah nama panggilan Anda dilakukan dengan perintah "NICK name", dan ucapan biasa dilakukan dengan perintah "PRIVMSG #channel: text". Terserah klien untuk menyediakan metode untuk menggunakan perintah ini, dan untuk itu, karakter kontrol bisa apa saja! Namun, ini berarti bahwa sebagian besar klien tetap menggunakan sesuatu yang konsisten, jadi kemungkinan besar semua perintah akan diawali oleh / di klien Anda.

Tidak. Itu hanya cara klien IRC berbeda dari teks 'normal'.

Anda dapat mengakses semua perintah server yang didukung dengan menggunakan / quote cmd atau / raw cmd

Ex:

/quote whois bgy

Itu akan mengirim perintah mentah ke server.

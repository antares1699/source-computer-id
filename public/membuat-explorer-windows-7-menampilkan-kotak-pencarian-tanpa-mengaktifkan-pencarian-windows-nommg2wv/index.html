Ada dua cara yang cukup sederhana untuk menonaktifkan Windows dari Pengindeksan;

  • Arahkan ke - Mulai> Cari 'Opsi Pengindeksan' dan Buka> Klik 'Ubah'> Klik 'Tampilkan Semua Lokasi' Sekarang batalkan pilihan apa pun yang Anda tidak ingin Windows mengindeks. Membatalkan pemilihan semuanya secara efektif mematikan pengindeksan karena Windows tidak memiliki apa pun untuk Diindeks.

  • Cara lain untuk melakukannya adalah dengan menonaktifkan Pengindeksan berbasis drive. tekan Tombol Windows + E , klik kanan pada drive yang ingin Anda nonaktifkan dan pilih 'Properti'. Hapus centang pada kotak seperti yang ditunjukkan di bawah ini. Ini akan menonaktifkan drive yang dipilih agar tidak diindeks.

Sejauh yang saya tahu, kedua metode ini masih memungkinkan Anda untuk mencari di Windows Explorer.

AwesomeWM mengubah ukuran firefox

Saya pemula di AwesomeWM dan tidak tahu cara mengubah ukuran jendela firefox saya. Saya dapat mengubah ukuran jendela terminal dengan Mod4 + H / L tetapi apa pun yang saya lakukan, firefox dan pembaca PDF saya macet berjalan dalam mode layar penuh. Ketika saya menambahkan terminal ke ruang kerja, mereka hanya akan muncul di depan atau di belakang jendela firefox.

Saya memiliki masalah yang sama tetapi mengubah maksimisasi tidak membantu jadi saya menemukan melalui Dokumentasi bahwa panah melambangkan pemaksimalan horizontal dan vertikal jadi saya menulis pintasan keyboard ini

  awful.key({ modkey, "Shift" }, "m",
function (c)
  c.maximized = false
  c.maximized_vertical=false
  c.maximized_horizontal=false
  c:raise()
end ,
{description = "demaximize", group = "client"})

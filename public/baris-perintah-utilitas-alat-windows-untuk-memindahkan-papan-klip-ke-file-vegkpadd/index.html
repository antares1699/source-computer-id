Baris perintah utilitas / alat Windows untuk memindahkan papan klip ke file

Saya telah melihat alat serupa CLIP di sini Perintah klip tetapi berfungsi ke papan klip bukan ke arah lain.

Saya telah melihat referensi untuk "menempelkan" tetapi tampaknya ini bukan bagian dari jendela. Saya perlu menemukan solusi yang berfungsi dengan lingkungan w7 w8 w10 standar.

Klip didistribusikan. Apakah ada alat yang bekerja dari clipboard ke file?

EDIT Sepertinya mungkin dilakukan dengan perintah PowerShell
Program C # yang baru saya tulis ini melakukan apa yang akan saya lakukan dengan Powershell

class Clip2File
{
    [STAThread()]
    static void Main(string[] args)
    {
        var fileName = @"c:\demo.png";
        if (args.Length != 0){
            fileName = args[0];
        }
        if (!Clipboard.ContainsImage()) return;
        Image img = Clipboard.GetImage();
        img?.Save(fileName, ImageFormat.Png);
    }
}

Anda dapat menggunakan cmdlet PowerShell Get-Clipboard (alias gcb).

Tulis teks clipboard ke file:

gcb > myfile.txt

Itu memang memiliki opsi untuk mendapatkan gambar dari clipboard, -Format Image, tetapi tampaknya hanya menyediakan data meta, jadi mungkin tidak membantu dalam menulis gambar ke file.

Apakah ada yang punya alat yang bekerja dari clipboard ke file?

Ini dapat dilakukan dengan menggunakan file batch yang berisi beberapa VBS yang disematkan.

GetClip.cmd:

@echo off
(
echo set objHTML = CreateObject("htmlfile"^)
echo ClipboardText = objHTML.ParentWindow.ClipboardData.GetData("text"^)
echo set objFSO = CreateObject("Scripting.FileSystemObject"^)   
echo set objFile = objFSO.OpenTextFile("clip.txt", 2, true^)
echo objFile.WriteLine ClipboardText
echo objFile.Close
) > "%temp%\clip.vbs"
"%temp%\clip.vbs"
endlocal

Catatan:

  • Isi clipboard ditulis ke clip.txt di direktori saat ini

Contoh Penggunaan:

F:\test>type LoremIpsum.txt

Cara terbaik yang saya tahu, adalah dengan menggunakan alat mandiri bernama WINCLIP .

Anda bisa mendapatkannya di sini: Mengecoh

Pemakaian:

  • Simpan clipboard ke file: winclip -p file.txt

  • Salin stdout ke clipboard: winclip -c Contoh: Sed 's/find/replace/' file | winclip -c

  • Clipboard pipa ke sed: winclip -p | Sed 's/find/replace/'

  • Gunakan output winclip (clipboard) sebagai argumen dari perintah lain:

    FOR /F "tokens=* usebackq" %%G in ('winclip -p') Do (YOUR_Command %%G )

Anda mungkin juga ingin melihatnya getclip & putclip alat: CygUtils untuk Windows tapi winclip lebih baik menurut saya.

Apakah setiap komputer memiliki alamat IP publik?

Apakah setiap komputer memiliki alamat IP publik? Atau apakah komputer hanya memiliki IP pribadi dan router internet tempat komputer terhubung memiliki IP publik?

Ini sepenuhnya tergantung pada pengaturan jaringan.

Biasanya di sebagian besar lokasi memberikan satu Alamat IP ke router yang berada di tepi jaringan, dan kemudian semua komputer di belakang router memiliki alamat IP pribadi. Ketika salah satu mesin membutuhkan koneksi ke internet, mereka berbagi satu alamat IP yang sama.

Jika mesin perlu menghosting layanan apa pun, biasanya di router, Anda mengaktifkan penerusan port yang akan mengirimkan informasi tersebut ke komputer tertentu. Teknologi ini dikenal sebagai NAT (Terjemahan Alamat Jaringan) .

Semua ini dikatakan, di lingkungan di mana ada banyak server atau layanan yang dihosting, tidak jarang untuk langsung meneruskan alamat IP ke server. Dalam situasi ini, router hanya merutekan semua lalu lintas pada alamat IP ke mesin target.

Tidak jarang melihat kantor besar yang memiliki beberapa server yang menghadap publik memiliki blok kecil alamat IP dan merutekan masing-masing ke server dan kemudian diteruskan ke router lain yang menyediakan NAT ke semua perangkat biasa yang tidak. membutuhkan IP mereka sendiri.

Tidak ada satu aturan yang cocok untuk semua. Satu IP ke router NAT sejauh ini merupakan skenario yang paling umum, tetapi bisa berupa apa saja.

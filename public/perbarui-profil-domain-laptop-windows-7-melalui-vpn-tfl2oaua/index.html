Perbarui profil domain laptop Windows 7 melalui VPN

Saya memiliki laptop perusahaan yang merupakan anggota domain kami, dan saya mengakses jaringan menggunakan klien Cisco VPN 5.0.07.0290. Saya baru saja mengubah sandi domain saya di kantor menggunakan komputer desktop saya, dan sekarang profil laptop tidak sinkron. Ini masih menggunakan kata sandi sebelumnya. Sebagai pengganti membawa laptop bekerja, bagaimana cara memperbarui profil di laptop saya melalui VPN? Laptop menjalankan Windows 7 Enterprise SP1.

Mudah:

  • Masuk di laptop tanpa jaringan dan menggunakan tua kata sandi.
  • Kemudian hubungkan jaringan dan mulai koneksi VPN Anda untuk bekerja.
  • Sekarang kunci laptop Anda. kontrol - alt - del Mengunci).
  • Kemudian buka kuncinya menggunakan baru kata sandi.

Pada saat membuka komputer terhubung ke domain (melalui terowongan VPN) dan itu akan memverifikasi kata sandi dengan domain. Sebagai efek sampingnya, ini akan memperbarui kata sandi pada laptop.

mengapa transfer file lokal lebih lambat dari 1Gbps / 100Mbps

Saya memiliki 2 komputer dengan mobil ethernet 1Gbps (PCI) dan terhubung ke Switch berkemampuan 1Gbps dengan kabel ethernet Cat 5e. Mengapa transfer file jauh lebih lambat dari 100Mbps apalagi 1Gbps. (perhatikan saya tidak bertanya tentang apa pun yang berhubungan dengan internet) Sakelar saya terhubung ke router yang setidaknya memiliki kecepatan ethernet 100Mps. Jadi asal file

satu komputer -> sakelar -> router -> sakelar -> komputer lain

atau sederhananya

satu komputer -> saklar -> komputer lain?

Untuk bagian pertama dari pertanyaan Anda:

Anda terbatas pada kecepatan hard drive dari drive TERLAMBAT Anda dalam transfer. Sakelar mega cepat Anda dan NIC dan PC baru badass dengan drive 15.000 rpm hanya dapat mengirim data ke laptop lama 10 tahun secepat itu dapat menulisnya ke drive.

Untuk bagian kedua:

Itu tergantung pada bagaimana perangkat terhubung. Jika (seperti yang saya duga) komputer Anda menggunakan skema ip yang sama, dan sakelar memiliki port uplink ke router, maka router tidak terlibat. Router hanya akan disertakan jika itu adalah dua buah pada skema ip atau subnet yang berbeda.

Ada cukup banyak alasan hal ini bisa terjadi. Tidak semua hard drive dibuat sama dan Anda akan beruntung mendapatkan 1Gbps dari drive konsumen kecuali dalam kondisi ideal seperti membaca satu file besar tanpa fragmentasi. Banyak file yang lebih kecil telah menambah overhead, dan fragmentasi meningkatkan waktu akses. Selain itu, kecuali Anda memiliki komponen yang sangat mahal, baik SATA dan NIC memerlukan interaksi dari CPU, sehingga program lain akan memperlambat segalanya. Jika Anda menjalankan perangkat lunak anti-virus, ini mungkin juga memeriksa data karena diakses semakin memperlambat proses. Terakhir, dan ini mungkin yang paling tidak penting, Cat6 direkomendasikan untuk GigE.

Perlu diingat, Windows Vista / 7 melaporkan kecepatan transfer dalam Megabyte per detik (MBps), bukan Megabit per detik (Mbps). Kecepatan tautan dinilai dalam bit, bukan byte per detik. 1000 Mbps setara dengan sekitar 125 MBps ... tetapi, banyak faktor lainnya (termasuk batasan bus PCI / kecepatan HDD / kualitas kabel) mengurangi throughput 1Gbps menjadi mendekati 400 Mbps. Kecepatan transfer 50-75 MBps tidak jarang .... pada drive 7200 RPM saya, saya hanya mendapatkan 20 MBps.

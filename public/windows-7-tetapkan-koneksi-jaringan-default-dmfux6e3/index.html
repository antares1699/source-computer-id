Windows 7 Tetapkan Koneksi Jaringan Default

Bagaimana cara mengatur koneksi jaringan default menjadi Ethernet.

Sepertinya kartu nirkabel saya di laptop baru saya menjadi aktif sebelum Ethernet dan meninggalkan saya dengan koneksi yang jauh lebih lambat daripada Ethernet.

Saya kemudian memutuskan sambungan nirkabel secara manual setiap hari, namun saya ingin ini terjadi secara otomatis jika memungkinkan.

Anda harus dapat menyesuaikan pengaturan metrik untuk setiap antarmuka, memberikan preferensi ke antarmuka berkabel. Secara default, antarmuka berkabel dengan kecepatan tautan yang lebih cepat harus diberi preferensi menggunakan pengaturan metrik otomatis. Namun menurut TechNet , metrik otomatis biasanya diaktifkan secara default.

  1. Klik kanan antarmuka jaringan, lalu klik Properti.
  2. Klik Protokol Internet (TCP / IP), lalu klik Properti.
  3. Pada tab Umum, klik Lanjutan.
  4. Untuk menentukan metrik, pada tab Pengaturan IP, klik untuk mengosongkan kotak centang Metrik otomatis, lalu masukkan metrik yang Anda inginkan di bidang Metrik Antarmuka.

(langkah di atas diadaptasi dari KB299540 )

Angka yang lebih rendah mewakili antarmuka yang disukai, karena angka tersebut mewakili nilai biaya untuk memilih antarmuka itu.

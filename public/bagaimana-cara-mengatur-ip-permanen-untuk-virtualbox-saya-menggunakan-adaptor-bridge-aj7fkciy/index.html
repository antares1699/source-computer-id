Bagaimana cara mengatur IP permanen untuk VirtualBox saya menggunakan adaptor Bridge?

Saya menjalankan server web (pengembangan) di Virtualbox (Ubuntu 11.1) saya. Saya sering berpindah-pindah dan dengan demikian saya mendapatkan berbagai alamat IP di komputer saya.

Setiap kali saya mendapatkan alamat IP baru, saya harus mengubah berbagai pengaturan konfigurasi agar situs saya aktif dan berjalan.

Apakah ada cara agar saya dapat menyetel alamat IP "permanen" di VB saya dan merutekan semua lalu lintas ke laptop saya?

Saya memiliki masalah yang sama (VirtualBox 4.1.10, Host Ubuntu 11.10, Guest Ubuntu 11.10). Begini cara saya memperbaikinya:

Setel dua antarmuka jaringan untuk VM. Yang pertama harus NAT dan yang kedua harus Host-Only. Jika Anda tidak dapat menambahkan koneksi khusus-host, pastikan Anda telah mengaturnya di Preferensi -> Jaringan.

Di OS Tamu (Ubuntu untuk saya), edit /etc/network/interfaces, tambahkan yang berikut di bagian akhir:

auto eth1 # this refers to the Host-only network interface
iface eth1 inet static
address 192.168.56.10 # Arbitrary IP address
netmask 255.255.255.0

dan reboot. Dan itu dia. Jangan memodifikasi apa pun untuk eth0, karena kita perlu mendapatkan pengaturannya secara otomatis melalui DHCP untuk mengakses internet. eth1 memungkinkan tuan rumah dan tamu berkomunikasi satu sama lain.

Anda dapat mengubah IP Tamu VB Anda menjadi statis dengan memodifikasi file-file ini.

vim /etc/network/interfaces

Tambahkan baris ini di akhir:

auto eth0
iface eth0 inet static
address xxx.xxx.xxx.xxx
netmask xxx.xxx.xxx.xxx
gateway xxx.xxx.xxx.xxx

Untuk Pengaturan DNS

vim /etc/resolv.conf

dan tambahkan:

nameserver xxx.xxx.xxx.xxx

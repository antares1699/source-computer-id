Bagaimana cara mengganti sekitar wildcard dalam pencarian regex?

Saya mencoba mengomentari semua deklarasi tipe variabel dalam skrip yang saya miliki, tetapi saya mengalami masalah ini beberapa kali. Saya ingin mencocokkan string, diikuti dengan nama variabel, diikuti dengan string dan mengganti teks persis tetapi dengan satu kutipan di depan string kedua.

Sebagai contoh:

Dim variableName As Boolean
Dim variableName2 As String

harus menjadi:

Dim variableName 'As Boolean
Dim variableName2 'As String

Sekarang, masalahnya adalah saya juga memiliki string "As" di tempat lain dalam kode yang tidak boleh diganti, jadi temukan / ganti sederhana tidak akan berfungsi.

Saya menggunakan NotePad ++ dan memiliki regex berikut yang cocok dengan deklarasi saya dengan benar:

Dim.*As

Tapi saya tidak tahu apa yang harus menggantikan .* di bidang ganti. Dim.*'As jelas menggantikan string wildcard dengan .* literal, yang tidak baik.

Ada ide?

Anda dapat menggunakan RegEx "menangkap grup" untuk menentukan bagian yang ingin Anda tangkap dan simpan, dan ganti sisanya.

Dalam Notepad ++, grup pengambilan diwakili oleh tanda kurung (( dan )), dan Anda dapat memasukkan kontennya ke dalam "Ganti" dengan menggunakan notasi seperti \1, \2, dll. Untuk setiap grup tangkapan yang Anda tambahkan (dalam urutan tampilan).

Jadi, ubah RegEx pencarian Anda ke (Dim.*)(As), yang akan menemukan hal-hal seperti pada awalnya, tetapi membuat dua grup tangkap. Satu sebelumnya di mana Anda ingin menyisipkan ', dan satu lagi setelahnya.

Untuk ini akan menemukan Dim variableName As Boolean seperti sebelumnya, dan dua kelompok penangkapan akan berisi Dim variableName dan As Boolean.

Untuk bidang Replace di Notepad ++, gunakan \1'\2; ini akan menggantikan teks yang cocok dengan grup tangkapan pertama, lalu ', diikuti oleh grup tangkapan kedua.

Lakukan penggantian, dan Dim variableName As Boolean menjadi Dim variableName 'As Boolean (dll).

Ini dapat dilakukan dengan melihat ke belakang dan ke depan. Sebagai contoh: Temukan:

(?<=Dim)(.*?)(?=As)

Ubah dengan:

\1'

Ini harus melakukan apa yang Anda coba lakukan. '\ 1' di replace cocok dengan teks antara 'Dim' dan 'As', memungkinkan Anda untuk menambahkan komentar.

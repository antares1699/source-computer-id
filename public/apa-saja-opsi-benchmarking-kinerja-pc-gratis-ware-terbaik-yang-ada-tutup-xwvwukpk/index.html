Rangkaian Uji Phoronix

Phoronix Test Suite adalah platform pengujian dan benchmarking paling komprehensif yang tersedia untuk sistem operasi Linux. Perangkat lunak ini dirancang untuk secara efektif menjalankan tolok ukur kualitatif dan kuantitatif dengan cara yang bersih, dapat direproduksi, dan mudah digunakan. Perangkat lunak ini didasarkan pada pekerjaan pembandingan Linux yang ekstensif dan alat internal yang dikembangkan oleh Phoronix.com sejak 2004 bersama dengan masukan dari vendor perangkat keras komputer tingkat satu terkemuka. Perangkat lunak ini bersumber terbuka dan berlisensi di bawah GNU GPLv3. Phoronix Test Suite terdiri dari inti pemrosesan ringan (pts-core) dengan setiap tolok ukur terdiri dari profil berbasis XML dengan skrip sumber daya terkait. Proses dari penginstalan benchmark, hingga pembandingan yang sebenarnya, hingga penguraian komponen perangkat keras dan perangkat lunak yang penting sangat otomatis dan dapat diulang sepenuhnya, hanya meminta pengguna untuk mengonfirmasi tindakan.

  • 100+ Profil Uji
  • 40+ Suite Uji
  • Arsitektur Pengujian yang Dapat Diperluas (berbasis XML)
  • Distribusi Pengujian LiveDVD / USB berbasis Linux (PTS Desktop Live)
  • Instalasi Pengujian Otomatis
  • Dukungan Manajemen Ketergantungan
  • Arsitektur Plug-In berbasis modul
  • Penampil Hasil Terintegrasi
  • PNG, JPG, Adobe SWF, Dukungan Rendering Grafik SVG
  • Dukungan Mode Batch Otonom
  • Basis Data Global Untuk Unggahan Hasil, Perbandingan Tolok Ukur
  • Dokumentasi HTML yang Meliputi Profil Uji, Kerangka Modul
  • Perangkat Lunak Terinstal, Deteksi Perangkat Keras
  • Dukungan Pemantauan Sistem
  • Berjalan di Sistem Operasi Linux, OpenSolaris, Mac OS X, & BSD

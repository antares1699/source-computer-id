Ctrl-C Tidak berfungsi di zsh

Di zsh ketika saya mencoba dan menekan CTRL-C tidak ada yang terjadi (berfungsi dengan baik di bash) ketika saya menjalankan bindkey | grep \\^C saya mendapatkan "^C" Applications jadi sesuatu pasti terjadi ... bagaimana cara mengikat ^-C ke default?

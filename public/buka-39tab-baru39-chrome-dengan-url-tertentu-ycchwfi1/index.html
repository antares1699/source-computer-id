Buka 'tab baru' Chrome dengan url tertentu

Ketika saya membuka tab baru di Chrome, saya ingin itu memuat URL tertentu.

Saya dapat melihat cara menyetel halaman beranda, dan cara memberitahukan apa yang harus dilakukan saat memulai - tetapi bukan cara memuat halaman untuk tab baru.

Bagaimana saya bisa membuatnya melakukan ini? Apa itu mungkin?

Ini berhasil untuk saya!

  1. Buat ekstensi Chrome (atau Anda dapat menggunakan yang sudah Anda kodekan).
  2. Di dalam manifest.json, termasuk:
    "chrome_url_overrides": {
        "newtab": "newTabOverride.html"
    }
  1. Di folder root ekstensi Anda, buat file newTabOverride.html yang hanya berisi baris ini: <meta http-equiv="refresh" content="0; url=http://www.example.com/" />

(Saya mengikuti instruksi dari @ PredatorIWD's jawab disini ; Saya tidak tahu mengapa itu dihapus. Mungkin karena tidak meninggalkan kursor di bidang URL untuk memungkinkan pencarian Google.)

Selain itu, Anda mungkin hanya ingin memasang ekstensi yang ada, jika Anda mempercayainya: https://github.com/jimschubert/NewTab-Redirect

File Tersembunyi di Flash Drive

Saya baru saja diberikan flash drive oleh rekan kerja. Saat dicolokkan ke mesin windows, beberapa logika pada perangkat membuka dialog run, mengetikkan URL huruf demi huruf, dan kemudian menekan enter untuk membuka situs web. Saya tidak dapat menemukan satu file pun di flash drive ini. Saya telah memasukkannya ke dalam mesin Mac, Ubuntu Linux, dan Windows - semuanya mengatakan file nol.

Tingkah laku

Mac - Membuka App store

Linux - Tidak melakukan apa pun

Windows 7 - Lihat di atas

Apakah Anda yakin ini adalah drive usb? Apa Anda sebenarnya adalah persetujuan pria. Perangkat usb memberi tahu Anda dan Anda hanya bisa berharap itu tidak berbohong kepada Anda. Kemudian saya pikir Anda dapat beralih mode dan berpura-pura menjadi orang lain. Ini berguna untuk "perangkat yang menginstal sendiri". Anda mencolokkannya, lalu perangkat lunak otomatis menginstal driver yang relevan dan Anda siap melakukannya.

Ini membutuhkan akses tingkat perangkat keras ke perangkat usb, jadi agak sulit untuk berkembang biak sebagai virus.

Bagaimana Anda memasang disk saat memasangnya di Linux?

Dengan asumsi bahwa Anda berurusan dengan file yang sebenarnya (sebagai lawan dari sesuatu yang lebih menarik, seperti infeksi yang berada di bagian kontrol sirkuit), mereka mungkin disembunyikan. Anda dapat mencoba salah satu cara berbeda untuk mengaktifkan tampilan file tersembunyi (konfigurasikan mesin Anda untuk menampilkan file tersembunyi, ekstensi, dan menampilkan file sistem, atau menjalankan sesuatu seperti berikut pada konten disk:

attrib -s -r -h *.*

Menjalankan perintah di semua file yang berisi string

Saya menemukan banyak file dengan find . -exec grep -l mypattern {} \;

Apa cara paling sederhana dan paling elegan untuk menjalankan perintah tertentu secara terpisah di setiap daftar file yang dihasilkan?

4
GJ. — Sumber

1 Jawaban

9
9

Untuk memulai, gunakan:

grep -Rl mypattern .

langsung sebagai ganti konstruksi find.

Apa perintah tertentu? Mungkin cara terbaik dan paling aman dari kegagalan adalah

grep -ZRl mypattern . | xargs -0 mycommand

Persis yang ingin Anda lakukan adalah hingga xargs dari sini.

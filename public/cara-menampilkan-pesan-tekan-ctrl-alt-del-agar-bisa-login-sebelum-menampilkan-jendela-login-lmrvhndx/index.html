Cara menampilkan pesan “Tekan Ctrl + Alt + Del agar bisa login” sebelum menampilkan jendela login

Pada beberapa komputer saya memiliki tampilan keamanan yang meminta saya untuk "menekan Ctrl + Alt + Del" sehingga saya dapat membuka jendela Login dengan kata sandi nama pengguna dan seterusnya, bahkan saat boot (saya kira begitu) ...

Tetapi di beberapa komputer saya tidak memiliki pesan itu saat boot.

Apakah Anda mengetahui parameter yang dapat saya ubah untuk beralih di antara dua mode? (Menampilkan atau tidak jendela keamanan ini).

4

Anda dapat mengubahnya melalui Akun Pengguna di Panel Kontrol

  • Masuk ke komputer sebagai administrator.
  • Klik Mulai, klik Panel Kontrol, lalu klik dua kali Akun Pengguna.
  • Klik Ubah cara pengguna masuk atau keluar.
  • Klik untuk memilih kotak centang Gunakan layar Selamat Datang jika Anda ingin untuk menonaktifkan CTRL + ALT + DELETE urutan
  • Klik Mulai, klik Jalankan, ketik Control Userpasswords2, dan kemudian tekan enter. Catatan Anda mungkin menerima kotak dialog berikut: Anda harus menjadi anggota grup Administrator di komputer ini untuk membuka Akun Pengguna. Anda masuk sebagai nama pengguna, dan akun ini bukan anggota dari Grup administrator. Di dalam syaratnya, Anda diminta untuk mengetik nama pengguna dan sandi saat sistem meminta kotak dialog untuk mengetik nama pengguna dan kata sandi administrator komputer.
  • Klik tab Advanced.
  • Di bagian logon aman, klik untuk menghapus Wajibkan pengguna untuk menekan Ctrl + Alt + Hapus kotak centang jika Anda mau untuk menonaktifkan CTRL + ALT + DELETE urutan.

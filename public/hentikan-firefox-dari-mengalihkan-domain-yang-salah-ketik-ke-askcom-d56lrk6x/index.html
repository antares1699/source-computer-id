Hentikan Firefox dari mengalihkan domain yang salah ketik ke ask.com

Ketika saya mendapatkan URL yang salah di tingkat atas (yaitu ketika pencarian DNS mengembalikan NXDOMAIN), browser Firefox saya mengarahkan saya ke ask.com. Bagaimana cara menghapus atau menyesuaikan fungsi ini? Tidak ada yang jelas dalam dialog opsi dan Google tidak membantu — mungkin karena saya tidak tahu nama yang benar untuk kesalahan tersebut!

Saya menjalankan Windows, Firefox 3.0.14. Bukan malware, saya rasa saya tidak sengaja menginstalnya dengan realplayer atau semacamnya.

  1. Ketik about: config di bilah lokasi Firefox dan tekan Enter.
  2. Ketik kata kunci di kotak teks Filter dan Anda hanya akan melihat kata kunci preferensi.URL.
  3. klik keyword.enabaled untuk menyetelnya ke false. Ini akan menonaktifkan fungsionalitas pencarian.
  4. Jika Anda ingin mempertahankan fungsionalitas pencarian tetapi menggunakan mesin pencari lain, klik dua kali pada kata kunci.URL dan ubah nilainya menjadi " http://www.google.com/search?btnI=I%27m+Feeling+Lucky&q= "yang merupakan default firefox.

Seperti yang disebutkan sebelumnya , mungkin ada program "pembantu" yang disertakan dengan Ask Toolbar yang masih aktif. Jika tidak ada dalam program Tambah / Hapus, Anda mungkin ingin melihat daftar layanan yang berjalan karena mungkin juga ditempatkan di sana.

Periksa setelan Proxy / DNS Anda untuk melihat apakah Ask Toolbar tidak menyetel Tanya sebagai layanan DNS. Ini mungkin telah mengubahnya dalam pengaturan FireFox dan / atau mungkin telah mengubah Opsi Internet global di Windows.

Dalam kebanyakan kasus, Anda dapat membersihkan banyak sampah yang tertinggal oleh aplikasi dengan menggunakan alat seperti CCleaner .

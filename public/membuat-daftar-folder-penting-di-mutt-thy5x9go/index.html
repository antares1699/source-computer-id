Membuat daftar folder penting di mutt

Ketika saya mengetik c di mutt saya mendapatkan semua folder saya ditampilkan dan ini cukup panjang.

Saya ingin mengkonfigurasinya agar hanya menampilkan folder yang saya anggap menarik. Akan dengan senang hati mengkonfigurasinya dengan menambahkan daftar di saya .muttrc

Mendesak TAB setelah c akan beralih dari folder ke tampilan daftar kotak surat. Menekannya lagi akan membawa Anda kembali.

Anda dapat menambahkan kotak surat baru dengan sesuatu seperti

mailboxes /some/local/path
mailboxes imaps://some.imap.host/INBOX

Untuk imap (setelah login) ini akan menambahkan semua folder teratas ke daftar kotak surat.

Ada mutt sidebar patch (tersedia di Debian, Ubuntu dan mungkin lainnya sebagai mutt-patched ).

Dengan itu, Anda mengatur mailboxes Anda sebagai daftar folder yang Anda minati; mereka akan muncul di kiri layar. Anda kemudian dapat menggunakan perintah seperti sidebar-next untuk menavigasi. Saya menggunakan pengaturan ini:

bind index \CN sidebar-next
bind index \CP sidebar-prev
bind index \CO sidebar-open

Bagaimana cara membuat ikon lebih kecil dalam tampilan kisi Tumpukan untuk Snow Leopard?

Sejak saya menginstal Snow Leopard, saya terjebak dengan besar ikon di tampilan tumpukan / folder saya di dok saya. Ini digunakan untuk menyesuaikan tata letak dan ukuran jendela sehingga sebanyak mungkin ikon muat di layar, tetapi sekarang ini menunjukkan ikon besar dan scrollbar sebagai gantinya. Saya ingin perilaku lama kembali jadi saya tidak perlu menggulir lagi.

Adakah yang tahu dimana setting untuk ini? Mungkin tersembunyi di beberapa .plist?

Triknya: Di Terminal, diuji di Lion dan Snow Leopard:

defaults write com.apple.dock use-new-list-stack -boolean YES
killall Dock

Kemudian Anda bisa menggunakan cmd - + dan cmd - - saat daftar / kisi berbasis Dock terbuka.

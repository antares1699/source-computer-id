Apakah termcap di cygwin

Saya mencoba membangun sesuatu yang tampaknya bergantung pada termcap (terus mengatakan cannot find - ltermcap).

Saya menjalankan setup-x86_64.exe Cygwin, berharap menginstal termcap dari sana; Namun, saya tidak menemukannya di daftar paket, atau di daftar paket di situs web Cygwin, jadi sepertinya tidak ada di Cygwin. Saya tidak dapat menerima bahwa itu tidak benar, karena googling mengisyaratkan sebaliknya.

Saya bisa mendapatkan termcap dari http://gnuwin32.sourceforge.net/packages/termcap.htm , tetapi saya tidak tahu apakah itu kompatibel dengan Cygwin, dan tampaknya jauh lebih bersih untuk menggunakan Cygwin.

Maaf jika saya terdengar seperti saya tidak tahu apa yang saya bicarakan karena saya mungkin tidak.

Saya menjalankan setup-x86_64.exe Cygwin, berharap menginstal termcap

Namun, saya tidak menemukannya di daftar paket

Catatan: Dari http://gnuwin32.sourceforge.net/packages/termcap.htm :

Namun, penggunaan termcap tidak disarankan. Termcap sedang dihapus untuk mendukung pustaka ncurses berbasis terminfo, yang berisi emulasi dari rutinitas pustaka termcap di samping implementasi kutukan yang sangat baik.

Jadi, di Cygwin Anda harus membangun menggunakan ncurses sebagai gantinya.

  • Anda mungkin perlu mengubah makefile untuk program yang Anda coba buat untuk melakukan ini (sehingga program ini terhubung dengan pustaka yang benar untuk emulasi ncurses dari termcap)

  • Paket Cygwin ncurses mencakup:

    usr/include/ncurses/termcap.h -> ../termcap.h

    Yang menyediakan antarmuka ke emulasi ncurses dari termcap

Anda selalu dapat mengetahui paket apa yang berisi perintah cygwin tertentu yang digunakan Paket Pencarian :

Saya sarankan menginstal:

  • cygwin32-ncurses-5.7-1 - cygwin32-ncurses: ncurses untuk Cygwin 32bit toolchain (binari terinstal dan file dukungan)
  • libncurses-devel-6.0-1.20151017 - libncurses-devel: Perpustakaan tampilan terminal (binari terinstal dan file pendukung)

Cygwin menggunakan ncurses , yang menyediakan istilahcap antarmuka . Ini telah melakukan ini untuk beberapa waktu (lihat catatan dari 2001 ). ncurses menyediakan antarmuka untuk termcap, terminfo dan tentu saja kutukan.

Pada beberapa sistem, paket ncurses mungkin menyediakan tautan simbolis untuk membantu aplikasi yang mengharapkan pustaka yang menyediakan antarmuka termcap diberi nama "termcap", mis.,.

libtermcap.a -> libncurses.a

tapi (lihat runtime dan pengembangan daftar), Cygwin tidak melakukan ini. Juga (seperti yang Anda catat) tidak menyediakan pustaka termcap terpisah (lihat Hasil Pencarian ).

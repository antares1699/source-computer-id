Mengompilasi pinfo di OS X

Saya ingin menginstal pinfo di Mac saya untuk membaca info file, alih-alih menggunakan kakak laki-laki yang tampak buruk info.

pinfo tidak namun disediakan oleh Homebrew . Mungkin akan segera, ketika saya mendapatkan pertanyaan ini dijawab. ( Sini masalah relatif, di GitHub.)

Situs web proyek dapat ditemukan sini .
Kode sumber sini .

Ketika saya mencoba untuk mengkompilasinya, menjalankan ./autogen.sh saya mendapatkan pesan kesalahan berikut:

atcold@AlfMAC ~/Work/Sandbox/pinfo-0.6.10 $ ./autogen.sh 
Cleaning autotools files...
find: illegal option -- t
usage: find [-H | -L | -P] [-EXdsx] [-f path] path ... [expression]
       find [-H | -L | -P] [-EXdsx] -f path [path ...] [expression]
find: illegal option -- t
usage: find [-H | -L | -P] [-EXdsx] [-f path] path ... [expression]
       find [-H | -L | -P] [-EXdsx] -f path [path ...] [expression]
Running autoreconf...
Can't exec "autopoint": No such file or directory at /usr/local/Cellar/autoconf/2.69/share/autoconf/Autom4te/FileUtils.pm line 345.
autoreconf: failed to run autopoint: No such file or directory
autoreconf: autopoint is needed because this package uses Gettext
Something is bad with find's parameters (no big deal, I guess),
autopoint is missing, and brew doesn't know anything about it.
  1. Ada yang buruk dengan parameter find (bukan masalah besar, kurasa),
  2. autopoint hilang, dan brew tidak tahu apa-apa tentang itu.

Saya juga tahu bahwa itu mungkin untuk mendapatkan pinfo dengan MacPort , jadi ada tempat untuk mengambil inspirasi, jika perlu (belum yakin bagaimana menuju ke sana).
Ketika saya mencoba untuk melihat trac.macports.org saya mendapatkan jawaban 403 Forbidden :(
Sini adalah tautan dari hampir semua Macport paket untuk pinfo, tetapi saya tidak bisa melangkah lebih jauh.

Masih mengedit!

Saya memperbarui masalah di GitHub, sini . Saya juga akan memperbarui pertanyaan ini nanti.

Saya kira orang mungkin sadar tetapi pinfo sekarang tersedia dari brew:

brew install pinfo

Jika Anda ingin tahu tentang mendapatkan autopoint di MacOS / X - ini tersedia dari formula minuman gettext - tangkapannya adalah hanya-tong yang berarti tidak terhubung ke / usr / local secara default karena dapat bertentangan dengan MacOS versi internal, namun MacOS tidak menyediakan autopoint jadi dapatkanlah sebagai berikut - instal gettext:

brew install gettext

lalu sym-link autopoint ke / usr / local / bin:

ln -s `brew ls gettext | grep bin/autopoint` /usr/local/bin

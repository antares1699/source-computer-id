Di prompt CMD, ketik:

path

Untuk menambahkan ke jalur akun pengguna Anda:

setx path "%path%;C:\foo"

Untuk jalur sistem:

setx path "%path%;C:\foo" -m

  • %PATH% berisi keduanya pengguna dan mesin komponen variabel lingkungan PATH. Berjalan mis. perintah setx path "%path%;C:\foo" -m secara efektif akan menggabungkan keduanya yang tidak diinginkan dalam banyak kasus. Selain itu, ini juga akan menimpa jalur yang tidak diperluas dengan ditto yang diperluas (%SYSTEMROOT%\System32 menjadi misalnya C:\Windows\System32).
    — mandrake
  • @mandrake: Ya, Anda benar. Akan lebih baik untuk echo %PATH% | clip memasukkan path ke clipboard, edit di Notepad dan kemudian gunakan path lengkap yang belum diperluas (tanpa duplikat) pada perintah di atas.
    — paradroid

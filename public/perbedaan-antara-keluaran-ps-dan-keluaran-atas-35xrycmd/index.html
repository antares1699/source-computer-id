Perbedaan antara keluaran ps dan keluaran atas?

Saya merasa sulit untuk memahami output yang dihasilkan oleh ps dan top?

Ini adalah keluaran dari atas:

PID    PSID USERNAME  TID  PRI NICE  SIZE  RES  STATE  TIME    CPU   COMMAND
26439 23712 soumyau  26439  15    0 7512M 5234M sleep 286:25  16.67% or_lse2 (18)
26523 23712 soumyau  26439  -2    0 7512M 5234M cpu9  143:10   8.33% or_lse2
26522 23712 soumyau  26439  -2    0 7512M 5234M cpu3  143:10   8.33% or_lse2

Ini oleh ps (ps -L -p 26439 -o pcpu,psr,pid,user,tid):

%CPU  PSR   PID   USER       TID
99.9   3   26439 soumyau    26522
99.9   9   26439 soumyau    26523
0.0    8   26439 soumyau    26439

Mengapa ada perbedaan dalam dua hasil tersebut?

Bisakah Anda menjelaskan secara singkat pentingnya dua% CPU?

ps didasarkan pada akumulasi penggunaan CPU (sejak proses dimulai), di mana% CPU adalah rata-rata (total / waktu).

top melaporkan penggunaan CPU (rata-rata) sejak terakhir kali diambil sampelnya.

Untuk referensi, lihat cuplikan ini dari man ps

Penggunaan CPU saat ini dinyatakan sebagai persentase waktu yang dihabiskan berjalan selama proses berlangsung. Ini tidak ideal, dan tidak sesuai dengan standar yang sesuai dengan ps untuk. Penggunaan CPU tidak mungkin bertambah hingga tepat 100%.

dan dari man top

Bagian tugas dari waktu CPU yang telah berlalu sejak pembaruan layar terakhir, dinyatakan sebagai persentase dari total waktu CPU. Di SMP sejati lingkungan, jika 'mode Irix' Mati, atas akan beroperasi dalam 'mode Solaris' di mana penggunaan cpu tugas akan dibagi dengan jumlah total CPU. Anda mengaktifkan mode 'Irix / Solaris' dengan perintah interaktif 'I'.

Perintah 'ps' menunjukkan semua proses berdasarkan waktu. yang mengalami semua proses. Ini tidak peduli tentang status proses (berjalan atau tidur).

Perintah 'top' menunjukkan proses yang sedang berjalan di urutan teratas. Itulah mengapa Anda dapat melihat variasi dalam keluarannya. Beberapa proses di baris pertama setelah beberapa waktu akan menjadi yang kedua atau ketiga ketergantungannya pada status proses.

LibreOffice Calc: Cara menghapus karakter dari beberapa kolom

Untuk beberapa alasan, ketika saya mengimpor data dari sumber tanpa nama tertentu (toggl.com), beberapa kolom waktu memiliki apostrof di dalamnya.

Saat saya memuat data ini di Calc, apakah ada cara cepat dan mudah untuk menghapusnya?

Kolomnya terlihat seperti:

'12:05:12
'05:05:05
'04:04:44

Dan aku ingin:

12:05:12
05:05:05
04:04:44

Saya melihat fungsi = REPLACE, tetapi saya tidak bisa membuatnya bekerja. Mungkin saya tidak mengerti langkah-langkahnya.

Jika panjang string Anda tidak sama, Anda dapat menggunakan gabungan perintah mid() dan len().

Dalam contoh Anda, perintahnya adalah: =MID(A1,2,LEN(A1))

Seret ke bawah seperti pada jawaban di atas.

Bagaimana cara kerjanya?

MID mengembalikan sub-string dari string yang ditentukan sebagai: MID([string], [start index], [end index]). Jika [end index] bervariasi, menggunakan LEN(A1) menentukan panjang string tertentu.

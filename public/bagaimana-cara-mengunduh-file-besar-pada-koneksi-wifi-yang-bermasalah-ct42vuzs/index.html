Bagaimana cara mengunduh file besar pada koneksi wifi yang bermasalah?

Saya bekerja di luar hotel dengan internet yang buruk. Saya perlu mengunduh file 5 pertunjukan besar. Sepertinya perkiraannya sekitar satu hari. Saya yakin ini mungkin akan terjadi 2 masalah, koneksi itu sendiri terputus di beberapa titik DAN bahkan jika unduhan selesai, itu mungkin rusak karena sifat http.

Selain diskusi tentang bagaimana dan mengapa ini terjadi, apakah ada solusi aktual dalam situasi saya? Saya pikir masalah terbesar saya adalah korupsi, apakah ada klien http yang dapat mencegahnya sama sekali?

Detail mesin saya: Acer Aspire "Timeline" M5 Nomor model: -581TG (Saya membelinya dari toko resmi Acer di Thailand, Pantip Plaza). Ini memiliki driver jaringan default di atasnya:

  • Broadcome 802.11n Network Adapater (versi 5.100.196.18, 3/19/2012)
  • Broadcom NetLink (TM) Gigabit Ethernet
  • Adaptor Miniport WiFi Virtual Microsoft (versi 6.1.17600.16385, 6/21/2006)

Alat CLI keriting dan Wget keduanya punya lanjut yang akan melanjutkan di mana mereka tinggalkan, dengan asumsi server HTTP itu sendiri mendukung permintaan rentang HTTP.

curl -C - http://...
wget --continue http://...

Saya telah menemukan keduanya dapat diandalkan. Dan dengan fitur pembatasan bandwidth itu berarti Anda dapat mengatur unduhan yang berjalan lama untuk menggunakan lebih sedikit dari bandwidth rata-rata Anda sehingga koneksi masih dapat digunakan untuk tujuan lain.

Tidak ada solusi HTTP untuk unduhan yang rusak, HTTPS lebih cenderung memperhatikan masalah tertentu. Satu-satunya hal yang dapat Anda lakukan adalah memverifikasi checksum atau tanda tangan di akhir unduhan.

Protokol paling sederhana yang saya ketahui yang dapat menyelesaikan masalah Anda adalah BitTorrent, dengan rsync sedetik.

Gunakan metode yang mengurangi kemungkinan file rusak, dan dapat mengatasi gangguan koneksi. Saya tahu dua:

  • Bittorrent, meskipun Anda membutuhkan host untuk menjadi seeding di Bittorrent khususnya, yang tidak selalu memungkinkan.

Jika memungkinkan, jika saya jadi Anda, saya akan mencoba menyambungkan komputer dengan kabel Ethernet. Selalu merupakan opsi paling stabil melawan wi-fi.

Sekarang, tidak ada yang sempurna, dan selalu mungkin ada masalah. Tapi saya pikir ini adalah taruhan terbaik Anda.

Cara mengatur LastPass untuk memverifikasi sebelum membuka lemari besi saya dan melihat semua kata sandi

Saya baru mengenal LastPass dan saya terkejut ketika saya menyadari bahwa, setelah saya masuk ke ekstensi browser LastPass, saya dapat membuka lemari besi saya dan melihat semua kata sandi bahkan setelah saya keluar dari browser saya dan membukanya kembali.

Bagaimana ini bahkan aman? Apakah ada cara untuk membiarkan LastPass meminta verifikasi sebelum membuka lemari besi saya atau melihat kata sandi? Jika tidak memungkinkan, apakah disarankan untuk beralih ke Pengelola Kata Sandi Google atau serupa?

Opsi Lastpass yang tersedia sangat berbeda antara ekstensi browser dan situs web.

Untuk meminta kata sandi utama untuk setiap tampilan kata sandi situs:

  • Masuk ke akun Anda di situs web ( https://lastpass.com )
  • Klik "Pengaturan Akun"
  • Buka pengaturan lanjutan (Tombol di bagian bawah popover)
  • Cari bagian "Warnings", lalu pilih yang Anda sukai di antara "Master Password Re-Entry"

Saya setuju bahwa proses ini tidak perlu berbelit-belit dan tidak praktis.

Secara pribadi, saya jarang benar-benar perlu melihat kata sandi. Saya membiarkan Lastpass mengisi kredensial untuk saya dan ini berfungsi dengan sangat baik di sebagian besar situs.

Namun, saya sarankan Anda menggunakan faktor kedua seperti TOTP (misalnya dengan Google Authenticator) untuk login akun.

Anda perlu mengedit preferensi Lastpass Anda. Di Firefox, jika saya mengklik ikon untuk ekstensi Lastpass dan pergi ke Preferensi , Saya dapat mencentang "Keluar Secara Otomatis saat semua browser ditutup", dan / atau "Keluar Secara Otomatis setelah menganggur (menit)". Anda juga dapat mengaktifkan permintaan ulang kata sandi seperti yang disarankan oleh jawaban lain.

Terserah Anda, tingkat keamanan vs. kenyamanan seperti apa yang ingin Anda buat. Default untuk tetap masuk mungkin mengasumsikan Anda menggunakan komputer pribadi yang akan dikunci oleh OS saat Anda jauh dari komputer, tetapi asumsi itu jelas tidak aman dalam semua keadaan.

Anda selalu dapat menginstruksikan LastPass untuk meminta Anda memasukkan kata sandi sebelum menunjukkan kata sandi kepada Anda.

  1. Masuk ke halaman edit kata sandi
  2. klik pada "pengaturan lanjutan" di bagian bawah
  3. centang kotak "memerlukan reprompt sandi"
  4. Simpan itu

Anda sekarang harus memasukkan kata sandi vault Anda setiap kali Anda ingin mengetahui kata sandi yang disimpan.

Peringatannya adalah bahwa LastPass tidak menawarkan pengaturan vault-wide untuk repormpt kata sandi, jadi Anda perlu mengatur setiap item secara manual.

Seperti yang disarankan oleh @Marcel bahwa ada pengaturan lebar vault di bawah Pengaturan Akun> Umum> Tampilkan Pengaturan Lanjutan, Anda dapat menginstruksikan LastPass untuk meminta kata sandi vault untuk mengakses jenis item / tindakan tertentu.

Opsi untuk mengeluarkan Anda dari Chrome ekstensi browser dapat ditemukan di halaman pengaturan ekstensi

Untuk mendapatkan keamanan yang layak dengan logout otomatis, saya merekomendasikan yang berikut ini:

Langkah 1 - Keluar otomatis ekstensi Chrome

  1. Buka ekstensi chrome

  2. Klik pada ekstensi LastPass Detail

  3. Temukan dan klik Opsi Ekstensi (gulir ke bawah jika perlu)

  4. Centang kedua kotak logout dan setel ke waktu yang singkat seperti 10 menit

Bagaimana cara mengaktifkan mode minor untuk semua file di bawah direktori?

Di Emacs (23.4.1), bagaimana cara mengaktifkan mode minor (misalnya mode pengisian otomatis) untuk semua file dalam mode utama (misalnya mode teks) di bawah direktori (secara rekursif)?

Sepertinya variabel direktori-lokal tidak dapat melakukan ini, tetapi saya mungkin salah.

4

6
6

Anda harus menggunakan hook mode teks. Di file .emacs Anda:

(defun turn-on-auto-fill-hook ()
  (cond ((string-match "^/home/foo/bar/blat/" buffer-file-name)
         (auto-fill-mode 1))))

(add-hook 'text-mode-hook 'turn-on-auto-fill-hook)

Ganti / home / foo / bar / blat dengan jalur ke direktori yang file-nya ingin Anda aktifkan mode pengisian otomatis saat dikunjungi. Jangan lepaskan tanda sisipan.

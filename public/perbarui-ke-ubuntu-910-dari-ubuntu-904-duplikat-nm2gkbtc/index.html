Pergi ke System -> Administration -> Update Manager dan cukup klik "Upgrade". Ini Sebaiknya tidak menimbulkan masalah bagi Anda.

Saya belum meningkatkan diri saya sendiri, dan saya akan menunggu setidaknya sebulan. Tentu saja seseorang perlu menjadi yang pertama, tetapi jika Anda tidak terburu-buru, dan sistem Anda saat ini berjalan dengan baik, Anda sebaiknya menunggu sebentar. Kecuali jika ada sesuatu di rilis baru yang ingin Anda gunakan, tentunya.

Perhatikan bahwa penginstalan baru 9.10 memungkinkan Anda untuk menggunakan file ext4 berkas sistem. Dengan peningkatan ini tentu saja tidak mungkin. Anda juga bisa konversikan ex3 ke ex4 di tempat , tapi ini lebih merepotkan. Ada sebuah laporan bug tentang korupsi file besar saat menggunakan ext4, jadi siapa pun yang melakukan instalasi baru saya akan merekomendasikannya tidak gunakan ext4 default, tetapi tetap dengan ext3 , sekali lagi.

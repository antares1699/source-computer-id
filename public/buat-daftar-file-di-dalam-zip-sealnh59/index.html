Unduh versi baris perintah 7-zip (7za) dan lakukan 7za -l. Ini akan mencantumkan konten file. Anda kemudian dapat, tentu saja, memasukkan ini ke dalam jenis skrip batch apa pun.

    unzip -Z1 foo.zip

    unzip -Z foo.zip
zipinfo foo.zip

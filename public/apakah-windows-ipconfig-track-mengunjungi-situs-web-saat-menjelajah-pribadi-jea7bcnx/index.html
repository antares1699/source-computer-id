Apakah Windows ipconfig track mengunjungi situs web saat menjelajah pribadi?

Saya menemukan ini sederhana tip tentang cara membuat daftar entri DNS atau situs web yang dikunjungi di Windows dengan mengetik di baris perintah:

 ipconfig /displaydns > dnslist.txt

Pertanyaan saya adalah situs web yang dikunjungi dalam mode privasi browser akan dicantumkan jika saya mengetik ulang perintah yang sama atau browser akan menghapusnya secara otomatis?

TL; DR: Ya, itu (mungkin) akan.

Ini bukan fungsi dari browser web, ini adalah fungsi dari sistem DNS.

Saat komputer terhubung ke komputer lain, pertama kali menggunakan DNS (sistem nama domain) untuk mencari alamat ip. Anda dapat menganggap ini sebagai buku telepon. Cari superuser.com di "buku telepon" DNS:

C:\> nslookup superuser.com
Server:     10.10.10.10
Address:    10.10.10.10#53

Non-authoritative answer:
Name:   superuser.com
Address: 190.93.247.58
Name:   superuser.com
Address: 190.93.246.58
Name:   superuser.com
Address: 141.101.114.59
Name:   superuser.com
Address: 190.93.244.58
Name:   superuser.com
Address: 190.93.245.58

(ini dijalankan di server linux, kemudian diubah sedikit. Hasil Anda akan bervariasi).

Ini menunjukkan bahwa, untuk menghubungi superuser.com, Anda dapat "menghubungi" salah satu "nomor telepon" (alamat IP) - yang pertama tercantum di sini adalah 190.93.247.58.

Karena ini memerlukan beberapa saat (0,2 detik di komputer saya!), Informasi ini disimpan secara lokal; sering kali, orang akan membuat beberapa koneksi ke komputer yang sama secara berurutan (misalnya untuk mengunduh halaman web, dan beberapa gambar - ini sangat disederhanakan!). Aturan menentukan berapa lama informasi diperbolehkan untuk disimpan, sebelum harus dicari lagi - seringkali 24 jam. Data mungkin disimpan di memori atau di disk. Memulai ulang komputer Anda harus mengatur ulang informasi ini (meskipun tidak perlu).

Tentu saja, jika Anda ingin mengonfirmasi ini, cobalah - Anda memiliki semua yang Anda butuhkan untuk menjalankan eksperimen ini.

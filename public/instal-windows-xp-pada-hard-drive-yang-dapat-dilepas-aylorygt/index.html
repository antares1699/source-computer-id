Ekstrak file .wim ke disk. Ini mungkin akan memakan waktu sekitar 15-20 menit. atau lebih lama tergantung pada kecepatan transfer untuk HDD dan CD / DVD. Saya tahu DVD Windows 7 memiliki install.wim yang menggunakan pengaturan default. Windows XP tidak memilikinya, tetapi jika Anda memiliki file wim yang Anda buat dengan ImageX, itu mungkin cara yang paling nyaman untuk mengaturnya. Namun, Anda harus reboot setelah selesai mengekstrak file, tetapi setidaknya Anda dapat terus bekerja selama bagian menyalin / mengekstrak file.

Jika Anda belum memiliki file wim, Anda harus membuatnya (Anda dapat menyiapkan VM, menjalankan sysprep, dan membuat file wim). Sayangnya hal ini tidak akan menghemat waktu kecuali Anda harus menginstalnya berkali-kali.

Anda juga dapat masuk ke Device Manager dan memilih Action > Scan for hardware changes, tetapi Anda mungkin harus melakukan boot ulang beberapa kali. Menjalankan sysprep dengan opsi generalisasi akan menghapus pengaturan dan driver khusus perangkat keras, tetapi dapat menyebabkan masalah dengan aktivasi.

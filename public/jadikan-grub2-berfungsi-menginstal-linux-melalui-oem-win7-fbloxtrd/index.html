Jadikan GRUB2 berfungsi menginstal Linux melalui OEM Win7

Saya membeli laptop Acer dengan Win7, dan menginstal Linux Xubuntu sebagai OS kedua. Dikatakan GRUB telah diinstal juga, tetapi saat boot itu terus memuat hanya jendela. Tidak ada layar grub sama sekali.

Apa yang harus dilakukan?

4
arden — Sumber

2 Jawaban

5
5

  1. Jalankan Xubuntu dari LiveCD atau LiveUSB dan instal Perbaikan Boot .

  2. Untuk menginstal Boot-Repair, buka terminal dengan menekan Ctrl + Alt + T dan ketik perintah berikut:

    sudo add-apt-repository ppa:yannubuntu/boot-repair && sudo apt-get update
sudo apt-get install -y boot-repair && boot-repair

  3. Setelah instalasi, Boot-Repair akan diluncurkan secara otomatis. Sekarang pilih "perbaikan yang disarankan" untuk memperbaiki grub.

  4. Mulai ulang dan hanya itu.

Anda sekarang harus memiliki menu Grub saat boot yang memungkinkan Anda memilih OS mana yang akan di-boot.

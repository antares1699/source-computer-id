Cara memutar file MP4 dengan AAC 5.1 melalui S / PDIF

Saya memiliki laptop dengan Windows 7 x64 dan a Sound Blaster X-Fi Surround 5.1 Pro . Saya menghubungkan output HDMI dari notebook ke LCD-TV saya dan soundcard eksternal ke receiver home-theater saya melalui kabel audio optik (S / PDIF).

Saya tidak punya masalah melihat video .mkv dengan 6 saluran DTS atau Dolby Digital. Tapi file ini adalah .mp4 dengan AAC 6 Channels.

Masalahnya adalah saya hanya mendapatkan 2 saluran / stereo di receiver saya (ini menunjukkan PCM 48 dan hanya speaker kiri dan kanan yang ditampilkan di layar). Ketika saya mendapatkan 6 saluran, semua speaker ditampilkan di layar dan juga tertulis DTS atau Dolby Digital.

Ada ide bagaimana membuatnya bekerja? Saya tidak keberatan menginstal pemain lain atau codec eksternal lainnya untuk membuatnya berfungsi.

Saya akhirnya melakukan ini:

Instal ffdshow (32-bit), tambahkan ffdshow Audio Decoder ke "Filter eksternal" dari MPC HomeCinema, setel ke "Lebih suka". Sekarang ffdshow Audio Decoder akan digunakan untuk memecahkan kode format audio apa pun yang didukungnya (misalnya AAC). Dalam konfigurasi Audio Decoder ffdshow> Output, pastikan untuk memeriksa "AC3 (mode enkode S / PDIF)".

Saya bisa menjelaskan jika seseorang membutuhkan, tanyakan saja.

Bagikan

Tidak, itu tidak mungkin, Anda memerlukan WinPE 64-bit untuk menginstal Windows 64-bit.

Dari Microsoft Technet:

Untuk menginstal Windows versi 64-bit, Anda harus menggunakan Windows PE versi 64-bit. Demikian pula, untuk menginstal Windows versi 32-bit, Anda harus menggunakan Windows PE versi 32-bit.

Sangat sulit untuk mengubah sidik jari browser Anda secara meyakinkan tanpa meninggalkan jejak sidik jari tambahan. Tes Panopticlick EFF adalah bukti konsep yang masuk akal, tetapi ada banyak karakteristik lain yang tidak diuji di situs tersebut yang juga dapat digunakan untuk sidik jari Anda. Tidak semua sidik jari akan menggunakan setiap metode, dan beberapa hanya akan menggunakan sidik jari dasar, jadi strategi yang berbeda akan memiliki efek yang bervariasi untuk server yang berbeda. Pertahanan terbaik adalah:

Tangani item dengan entropi tertinggi terlebih dahulu (mungkin Flash dan Java, dan font yang ditampilkan, dan agen pengguna. Spoofing agen pengguna sulit dilakukan secara meyakinkan. JavaScript juga merupakan vektor sidik jari kunci (ukuran layar, ukuran dan posisi browser, dll.) , matikan atau gunakan sesuatu seperti NoScript.

Browser Tor juga merupakan pertahanan yang baik terhadap sidik jari perangkat. Ini memiliki beberapa perlindungan bawaan terhadap sidik jari.

Sebaiknya Anda memilih perangkat yang bersih dan dikonfigurasi default pabrik (atau mesin virtual) dengan agen pengguna umum dan karakteristik umum lainnya yang dapat ditulis dengan jari, dan menggunakan beberapa perangkat atau VM tersebut untuk mencegah korelasi aktivitas terpisah.

Saya tidak tahu browser apa yang Anda gunakan, jadi saya akan memberi Anda beberapa tip untuk masing-masing.

Chrome:

  • Nonaktifkan plugin yang tidak perlu alias Flash: chrome: // plugins
  • Spoof agen pengguna Anda dengan ini: Ekstensi Chrome
  • Matikan Do Not Track Requests: chrome: // settings
  • Nonaktifkan WebGL: chrome: // flags / # disable-webgl

Firefox:

  • Ubah agen pengguna Anda menjadi agen umum di: about: config
  • Nonaktifkan Perujuk HTTP di: about: config
  • Gunakan NoScript: http://noscript.net/

Jika Anda membutuhkan lebih banyak, klarifikasi browser mana yang Anda gunakan dan saya dapat menambahkan ke daftar.

Mengganti pola multi-garis dalam file HTML

Saya memiliki serangkaian file HTML yang berisi dua baris seperti ini:

<body>
<h1>Title</h1><p>
<a href="url">Description</a><br>

Saya ingin mengganti teks ini dengan yang lain menggunakan skrip bash. Saya mencoba

sed -i -r 's/<h1>Title.*?$\/^.*?<br>/Replacement text/1' filename.html

tapi tidak berhasil. Saya curiga itu macet di jalur baru dan tidak tahu bagaimana mengatasi masalah tersebut.

Setiap bantuan dihargai. Jangan ragu untuk menyarankan alat Linux lain selain sed selama masih berfungsi!

Saya akan menggunakan Perl untuk ini:

perl -0pe 's/<h1>Title.*\n.*<br>/replacement/' filename.html

Di sini, -0 membuat Perl memisahkan rekaman pada karakter NUL alih-alih membaca baris demi baris, yang merupakan default saat menggunakan opsi -p.

Dengan ekspresi reguler Perl Anda memerlukan .* untuk mencocokkan karakter apa pun beberapa kali, dan Anda mencocokkan baris baru dengan \n.

Contoh:

$ echo '<body>
<h1>Title</h1><p>
<a href="url">Description</a><br>' | perl -0pe 's/<h1>Title.*\n.*<br>/replacement/'
<body>
replacement

sed tidak dapat mencocokkan lebih dari satu baris secara langsung. Saat pola multiline diperlukan, gunakan alat yang lebih canggih seperti Perl:

perl -i~ -ne 'if (/^<h1>Title/) {
                  $n = <>;
                  if ($n =~ /<br>$/) { print "Replacement\n" }
                  else { print "$_$n" }
              } else { print }'

Ini bisa dilakukan dengan sed.

sed -nf repl.sed filename.html

dimana repl.sed berisi:

# Must have one line loaded up before branching to rep.
# Processing will start this way.
:rep
# Load extra line into pattern space
N
# Test for title
/<h1>.*<\/h1><p>\n<a href=".*">.*<\/a><br>/{
  #Substitute and print
  s/<h1>\(.*\)<\/h1><p>\n<a href=".*">.*<\/a><br>/Title: \1/p
  #append next line without cycling
  N
  # everything but the last line
  s/.*\n\([.\n]*\)/\1/
  #test for last line
  ${
    p
    # this will effectively end the program
    n
  }
  b rep
}
${
  # will print pattern space (both lines)
  p
  # this will effectively end the program
  n
}
#Print first line in pattern space
P;
#Remove first line in pattern space with newline
s/.*\n\([.\n]*\)/\1/
b rep

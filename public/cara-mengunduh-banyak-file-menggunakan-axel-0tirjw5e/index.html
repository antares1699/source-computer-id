cara mengunduh banyak file menggunakan axel

Saya memiliki 5 file (mis. A.zip, b.zip. C.zip dll ..) yang ingin saya unduh menggunakan Axel . Karena dokumentasi terbatas, apakah ada yang tahu perintah yang benar?

Tidak yakin apakah Axel sudah memiliki fungsi untuk membaca daftar, tetapi jika tidak, Anda dapat menggunakan loop sementara. Dengan asumsi Anda memiliki file bernama 'mylinks.txt' dengan URL Anda dipisahkan oleh kembalian, yaitu

http://www.example.com/1.jpg

http://www.example.com/2.mp3

http://www.example.com/bobsparty.avi

Kemudian ketik terminal:

while read url; do axel $url; done < mylinks.txt

Anda dapat menggunakan axel untuk mengunduh banyak file dengan satu perintah! Anda cukup membuat skrip shell dengan semua tautan dan menjalankan skrip. Yang ini mengunduh setiap tautan satu per satu, dan berhenti selama 10 detik dan mematikan komputer Anda setelah semua tautan selesai diunduh, misalnya:

#!/bin/bash  
axel link1 -a  
axel link2 -a  
axel link3 -a  
sleep 10  
poweroff

Anda tidak bisa mendapatkan banyak file dengan satu perintah, meskipun Anda bisa mengulang perintah untuk setiap file. Perintahnya adalah

Dari beberapa URL

axel -o file.ext <url1> <url2> <url3>

Dari satu URL (Banyak koneksi - x = jumlah koneksi)

axel -o file.ext -n x <url1>

Ini adalah referensi OK (ish) http://www.linuxjournal.com/content/speed-your-downloads-axel

Bagikan

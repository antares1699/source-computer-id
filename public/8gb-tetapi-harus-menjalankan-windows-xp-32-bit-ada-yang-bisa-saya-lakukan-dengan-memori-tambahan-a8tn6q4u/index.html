Komputer

8GB, tetapi harus menjalankan Windows Xp (32 Bit). Ada yang bisa saya lakukan dengan memori tambahan?

Saya memesan komputer baru dengan RAM 8GB dengan rencana menjalankan Windows 7, 64Bit. Ternyata sekarang, saya perlu menjalankan OS 32Bit (XP atau 7) karena beberapa perangkat lunak yang belum berjalan di 64Bit (bahkan dengan pengaturan / mode kompatibilitas yang tersedia).

Apakah ada yang dapat saya lakukan dengan memori di atas 4GB dalam skenario ini? Saya bersedia mempertimbangkan solusi kreatif seperti menjalankan hypervisor di bawah XP yang menawarkan memori sebagai RAM-Disk untuk bertukar dll.?

Perangkat lunak yang tidak berjalan pada 64Bit adalah CISCO VPN (tampaknya ada solusi yang setengah berfungsi untuk itu) dan integrasi telepon / webcam IP CISCO "CISCO Unified Video Advantage" (tampaknya tidak ada solusi untuk itu).

Jika Anda memiliki lisensi untuk mendukungnya, saya sangat menyarankan Anda menjalankan windows 7 x64 atau Linux x64, menginstal virtualbox, dan menjalankan OS 32 bit Anda sebagai tamu. Setelah perangkat lunak menjadi kompatibel 64 bit Anda dapat membuang tamu dan menginstalnya di host utama.

Jika Anda sudah mati-matian untuk membuat ini bekerja, dan Anda adalah seorang badass, Anda dapat hexedit kernel windows 32 bit untuk mengaktifkan " ekstensi alamat fisik ". Ini akan memungkinkan Anda untuk menggunakan semua RAM Anda. Ya, semudah itu.

Ada panduan yang sangat rinci di sini di forum evga .

Pengeditan hex tidak sulit. Panduan ini sepertinya membutuhkan waktu 45 menit. Seseorang juga mengklaim Anda dapat membuat ramdisk dengan ram tambahan dan meletakkan file halaman di atasnya, yang pada dasarnya memberikan ram tambahan yang digunakan melalui lapisan paging.

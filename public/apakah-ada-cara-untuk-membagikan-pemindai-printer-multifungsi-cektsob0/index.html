Apakah ada cara untuk membagikan pemindai printer multifungsi?

Saya memiliki printer multifungsi di rumah (HP Photosmart C4280) dan saya ingin dapat menggunakan pemindai dari komputer mana pun (semuanya Windows XP) di jaringan saya. Apakah ini bisa dilakukan?

Dengan pengecualian perangkat lunak seperti yang disebutkan Robert, sebagian besar perangkat multifungsi tidak akan berfungsi dengan baik kecuali jika dihubungkan ke komputer yang tepat dan dibagikan.

Jadi jika Anda berpikir Anda dapat menggunakan salah satu perangkat berbagi cetak USB yang mewah - mungkin tidak akan berfungsi. Ini berlaku untuk sebagian besar server cetak - kecuali server cetak secara spesifik dan jelas menyatakan bahwa ia akan berbagi pemindai melalui IP.

Printer harus disambungkan ke komputer, karena komputer menyediakan sebagian daya pemrosesan dan informasi yang diperlukan untuk menyelesaikan tugas seperti pencetakan dan pemindaian. Seringkali, saya telah melihat printer dalam pengaturan ini menolak untuk memindai, dan menghasilkan cetakan dengan kualitas setengah.

Jadi, itu berarti Anda harus menghubungkan printer Anda ke komputer secara langsung. Jika Anda tidak ingin membeli perangkat lunak (atau tidak melakukan apa yang Anda inginkan), Anda tidak memiliki pilihan lain untuk berbagi pemindai, kecuali untuk membeli perangkat yang mendukung pemindaian berbasis jaringan, seperti HP OfficeJet. Sayangnya, itu bisa membuat Anda mendapatkan banyak uang.

Tidak ada cara mudah untuk melakukan ini. Anda juga membeli perangkat lunak yang mahal, atau mengganti perangkat dengan printer yang akan memindai melalui IP.

Sudahkah Anda melihat ke Server Cetak? Saya membeli satu baru-baru ini dan saat memeriksa fitur, saya perhatikan bahwa beberapa perangkat mendukung multi-fungsi. Posting forum ini (dari tahun 2006 memang) menyatakan:

Biasanya, TIDAK Anda tidak akan bisa gunakan opsi pemindai. Namun, jika server cetak dan printer / scanner keduanya dari HP, lalu jawabannya adalah "mungkin?"

Server cetak hanya menyediakan sarana mengarahkan kembali permintaan cetak ke a port jaringan (nirkabel dalam kasus Anda.) Mereka tidak menyediakan USB "jarak jauh" port, yang merupakan bagian pemindai membutuhkan. Namun, HP memiliki beberapa printer / pemindai yang "dapat" bekerja perangkat server cetak dan juga memberi Anda opsi pemindaian.

Jadi mengingat Anda memiliki perangkat HP, Anda dapat mempertimbangkan untuk mendapatkan server cetak HP.

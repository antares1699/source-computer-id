Di winxp, saya memiliki 4GB RAM, saya menggunakan 1,77GB, tetapi program terus berpindah ke memori. Mengapa?

Semua dalam judul. Apa gunanya lebih banyak RAM jika program saya keluar halaman daripada menggunakannya?

Untuk lebih spesifik, berikut contohnya: Saya membuka 12 tab di browser. Saya melihat salah satunya saat menggunakan program lain selama beberapa jam. Total RAM saya yang digunakan tidak pernah melebihi memori fisik saya. Namun, ketika saya membuka 11 tab lainnya beberapa jam kemudian, masing-masing membutuhkan waktu 30 detik untuk memuat saat disk saya meronta-ronta. Mengapa windows menghapus tab yang tidak digunakan dari RAM alih-alih mengalokasikan RAM baru untuk program yang saya gunakan?

Memang saya tidak benar-benar menggunakan tab yang belum dibuka itu, tetapi mengganti beberapa jam dengan 30 menit dan itu mengganggu.

Ini adalah keluhan umum tetapi ini didasarkan pada sejumlah kesalahpahaman tentang manajemen memori Windows dan apa yang dikatakan oleh Task Manager kepada kami. Dalam kebanyakan kasus Windows melakukan pekerjaan yang sangat baik dalam mengelola memori. Tapi sayangnya Task Manager tidak melakukan pekerjaan yang sangat baik untuk menunjukkan ini. Sebagian dari masalahnya adalah bahwa manajemen memori jauh lebih kompleks daripada yang dibayangkan kebanyakan orang dan mengikuti prinsip-prinsip yang sangat berbeda dari yang dibayangkan secara populer.

Hanya beberapa hal spesifik yang perlu diingat:

  1. Pengukur memori di Vista tidak menunjukkan penggunaan memori penuh. Sebagian besar bagian atas pengukur benar-benar digunakan. Demikian pula memori yang ditampilkan sebagai "Tersedia" di XP sebagian besar sedang digunakan.
  2. Tidak ada versi Task Manager yang pernah menunjukkan penggunaan pagefile yang sebenarnya. Apa yang diberi label sebagai "PF Usage" di XP dan "Pagefile" di Vista sebenarnya adalah commit charge. Penggunaan pagefile yang sebenarnya biasanya jauh lebih rendah.
  3. Sebagian besar data yang ada di file halaman juga ada di RAM.
  4. Paging tidak eksklusif untuk pagefile. Biasanya hanya sekitar 10% dari semua paging keras yang melibatkan file halaman. Tidak ada di Task Manager yang mengisyaratkan hal ini.

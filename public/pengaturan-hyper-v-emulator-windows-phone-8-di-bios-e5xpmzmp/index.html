Pengaturan hyper-v emulator Windows Phone 8 di BIOS

Saya menginstal Windows Phone 8 SDK, dan mencoba menyiapkan emulator. Tetapi mengeluh tentang pengaturan virtualisasi sistem saya tidak diaktifkan.

Saya telah menginstal Hyper-V dari "Tambah Program dan Fitur". Ketika saya pergi ke pengaturan BIOS saya, saya tidak melihat opsi seperti itu dari mana saya dapat mengaktifkan pengaturan virutalization.

Motherboard saya adalah ASUS M5A88. Apakah saya perlu memutakhirkan BIOS saya?

Memperbarui: Sesuai saran, saya memperbarui BIOS, dan telah mengaktifkan pengaturan itu, tetapi tetap saja, Visual Studio gagal memulai Emulator dengan kesalahan ini. BIOS yang lebih lama tidak memiliki pengaturan itu.

Memperbarui: Saya menemukan pengaturan di bawah "Advanced -> CPU Configuration". "Intel Virtualization Technology" tidak ada, saya kira, karena, ini untuk prosesor AMD.

Tetap saja, tidak berhasil. Saya melakukan pemeriksaan dengan mengunduh utilitas kecil, yang memeriksa apakah Virtualisasi didukung di sistem saya, dan dikatakan sistem saya mendukung Virtualisasi.

Prosesor saya adalah AMD-FX8150, jika itu penting di sini.

Dari halaman persyaratan sistem situs pengembang, saya mencoba menggunakan alat CoreInfo, dan alat itu juga mengatakan bahwa sistem saya mendukung virtualisasi yang dibantu perangkat keras.

Saya juga telah mengaktifkan DEP untuk semua program dari panel Kontrol.

Terpecahkan: Setelah mengaktifkan virtualisasi dari BIOS, saya harus menginstal ulang paket Hyper-V dari program "Aktifkan atau nonaktifkan fitur Windows", mulai ulang sistem saya, dan mulai berfungsi.

Berdasarkan posting ini , Anda harus memiliki BIOS versi terbaru untuk bekerja dengan Hyper-V dan mengaktifkan Virtualisasi:

Versi bios yang lebih lama tidak memiliki opsi itu, tetapi itu diaktifkan secara default. Rilis terbaru versi 1202, 1305, memiliki opsi untuk mengaktifkan atau menonaktifkan, fitur virtualisasi.

Ada dua tempat yang berpotensi menjadi:

  1. Posting SU ini menyarankan itu ditemukan di bawah "Intel Virtualization Technology"
  2. Posting SU ini menyarankan opsi di bawah "Mode Mesin Virtual Aman" di Lanjutan -> Konfigurasi CPU

Saya akan memeriksa keduanya.

Karena Hyper-V menggunakan Hypervisor, ini diperlukan untuk mengaktifkan emulator Win Phone 8.

Pada motherboard Anda, Anda perlu mengaktifkan pengaturan yang disebut "Secure Virtual Machine Mode" di BIOS Anda. Ini mengaktifkan teknologi virtualisasi (AMD-V) pada CPU Anda, yang diperlukan Hyper-V dan Windows Phone SDK.

Windows 7 batch files: Bagaimana cara menulis string ke file teks tanpa carriage return AND trailing space?

Saya mencoba agar file batch saya menulis string teks ke file teks. Pada awalnya, perintah yang saya gunakan adalah menulis kereta ekstra yang kembali ke akhir string, tetapi saya menemukan perintah ini yang mencegahnya:

echo|set /p=hello>hello.txt

Namun, sekarang menempatkan spasi di bagian akhir. saya butuh hanya string yang saya tentukan untuk ditulis tanpa karakter tambahan. Apakah ini mungkin?

Perintah ini harus bekerja persis seperti yang Anda inginkan - hanya tulis "halo", tanpa simbol tambahan apa pun.

> echo|set /P ="hello" > foo

> dir foo
...
09.07.2012  19:25                 5 foo
               1 File(s)              5 bytes

Sepertinya tepat 5 simbol tanpa CR + LF bagi saya.

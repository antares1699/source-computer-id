Izinkan lalu lintas jaringan hanya untuk program tertentu di windows 7

Terkadang saya menggunakan ponsel saya sebagai router wi-fi. Ketika windows 7 home premium saya terhubung ke jaringan ini, ia menghabiskan banyak data dengan segera. Saya perlu menonaktifkan semua lalu lintas internet kecuali browser dan skype dalam kasus tersebut, dan dengan mudah beralih kembali ketika saya mendapatkan wi-fi rumah biasa. Saya pikir firewall windows dapat dengan mudah melakukannya, tetapi tampaknya tidak begitu.

Jika Anda ingin memblokir semua koneksi keluar, lakukan hal berikut:

Ketik Firewall di kotak pencarian di menu mulai dan pilih "Windows Firewall dengan keamanan lanjutan", di bagian tengah jendela klik `Windows Firewall Properties '

Dari sana, apa pun jaringan yang dianggap modem ponsel Anda (Publik, Domain, atau Pribadi), buka tab itu dan ubah koneksi keluar dari Allow ke Block.

Jika Anda memerlukan beberapa program untuk dapat terhubung, Anda harus secara eksplisit membiarkan mereka terhubung dengan menambahkan aturan ke Aturan Keluar.

Jika Anda membuat koneksi ponsel Anda termasuk dalam profil "Publik" dan koneksi rumah Anda sebagai profil "Pribadi" Anda tidak perlu membuat perubahan apa pun saat Anda beralih di antara dua koneksi jika Anda hanya memblokir koneksi "publik" keluar .

Jika Anda dapat mengidentifikasi program yang menggunakan banyak lalu lintas, Anda dapat melewati semua langkah di atas dan hanya membuat aturan keluar baru dan menyetelnya untuk memblokir aplikasi itu.

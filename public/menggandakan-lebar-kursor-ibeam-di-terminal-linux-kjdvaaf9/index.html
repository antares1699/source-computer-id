Menggandakan lebar kursor ibeam di Terminal Linux?

Saya ingin menggunakan kursor ibeam | di terminal saya, tetapi sangat sempit dan sulit dikenali terutama di editor vim. Apakah mungkin untuk menambah lebar atau mungkin menentukan karakter Unicode sebagai gantinya?

Saya menggunakan emulator terminal termite yang didasarkan pada VTE.

Misalnya di editor Sublime Text, ukurannya hampir dua kali lipat dan menurut saya itu ideal untuk saya. Perlu mencobanya.

Untuk emulator terminal berdasarkan versi terbaru VTE (misalnya gnome-terminal, mate-terminal, xfce4-terminal, terminator, tilix dan beberapa lainnya), letakkan sesuatu seperti ini di file .config/gtk-3.0/gtk.css Anda:

vte-terminal, VteTerminal {
    -GtkWidget-cursor-aspect-ratio: 0.2;
}

Anda harus keluar dan memulai ulang proses emulator terminal (qiut semua jendelanya sekaligus).

Situs web OCR online untuk memproses seluruh file pdf sekaligus? [Tutup]

Saya mencari situs web OCR online untuk memproses file pdf multi-halaman sekaligus. Lebih disukai gratis.

aku tahu http://www.newocr.com/ . Jika saya benar, itu hanya dapat OCR satu halaman pada satu waktu, dengan mengklik "Pratinjau" secara manual dan kemudian mengklik "OCR" untuk setiap halaman. Setelah setiap halaman di-OCR, saya juga harus menyalin hasil teks secara manual.

Jika file pdf saya memiliki 30 ~ halaman, akan membosankan untuk mengulangi proses di atas untuk setiap halaman. Saya ingin tahu apakah ada beberapa situs online lain yang OCR seluruh file pdf, tanpa meminta saya untuk operasi manual?

Terima kasih!

Bagaimana cara memperbesar 100% (status default) di Windows 7?

Saya dapat menggunakan control + scroll untuk memperbesar dan memperkecil layar di Windows 7. Bagaimana saya bisa membuatnya 100%?

Dengan MS Word, ada indikator untuk menunjukkan persentase, tetapi saya tidak tahu bagaimana cara melihat persentase di aplikasi lain seperti IE.

Tergantung aplikasinya. Di browser seperti IE, Firefox, dan Chrome, Anda dapat menekan CTRL + 0 untuk mengatur ulang zoom. Untuk memperbesar, CTRL + + akan bekerja dengan baik dan CTRL + - untuk memperkecil. Opsi yang tepat akan selalu tersedia di bawah menu 'Tampilan', atau serupa.

Jika Anda mencoba mengubah resolusi layar Anda di Windows 7, ini adalah cerita yang berbeda. Berikut ini tautan ke halaman Microsoft yang memberi tahu Anda cara melakukannya . Atau Anda dapat mengklik kanan desktop Anda dan membuka Screen Resolution.

Cara menghubungkan output headphone ke input mikrofon laptop

Saya telah berjuang untuk menemukan cara terbaik untuk menghubungkan output headphone stereo (soket tempat Anda biasanya mencolokkan headphone Anda) ke input mikrofon laptop Lenovo G575 untuk merekam. Saya telah mencoba berbagai metode dan mendapatkan distorsi bergelombang yang mengerikan meskipun levelnya disetel dengan benar dan tidak ada pemotongan. Kartu suara laptop tidak memiliki opsi untuk mengubah input mikrofon menjadi input tambahan. Saya mencoba saran untuk menghubungkan setiap saluran stereo ke resister 330 ohm secara seri dengan kapasitor 1 mikro farad yang disambungkan ke input mikrofon laptop (cincin dan ujung disambungkan) tetapi saya mendapatkan distorsi bergelombang.

Video

Saya juga mencoba menambahkan resistansi 1k untuk ini setelah kapasitor dan juga menguji dengan 22K tetapi mendapatkan distorsi bergelombang yang sama bahkan ketika levelnya disetel dengan benar. Distorsinya sulit untuk dijelaskan. Ini bukan kliping. Ini lebih seperti semakin keras dan lembut seolah-olah seseorang melambaikan mikrofon.

Saya perhatikan resistansi mikrofon laptop tipikal adalah 1K. Saya pikir solusinya harus terletak pada pemasangan jembatan resistensi yang benar. Nasihat Anda akan dihargai.

Level speaker rata-rata VOLT dan level Mikrofon berada di urutan milivolt. Hanya ada beberapa pengecualian untuk generalisasi ini yang tidak mungkin terjadi pada laptop. Jadi, apa pun yang Anda lakukan dengan resistor, banyak redaman diperlukan, mengapa tidak memulai dengan faktor atenuasi 100 dan menggunakan kapasitor Anda untuk memblokir DC menjaga respons frekuensi tetap datar. Jika semua ini tidak bekerja dengan baik setelah mengoptimalkan faktor atenuasi, Anda dapat menggunakan lebih banyak topi untuk filter lowpass katakanlah 16KHZ kalau-kalau output speaker laptop penuh dengan HF kasar sehingga Anda tidak dapat mendengar tetapi dapat merusak input mikrofon.Kemungkinan lain adalah loop tanah, masalah kebisingan.Jika Anda menempatkan trafo audio di sirkuit Anda dan masalahnya hilang maka Anda tahu. membuktikan ini dapat dilakukan dengan audio junkbox apa pun trafo. Kemudian Anda dapat melakukannya dengan baik dengan baik trafo ATAU tahap amp diff.

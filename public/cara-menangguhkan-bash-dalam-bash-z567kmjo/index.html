Cara menangguhkan bash dalam bash

Apakah mungkin untuk menangguhkan bash yang dipicu oleh bash lain? Misalnya, jika saya menggunakan su untuk menjadi pengguna yang berbeda tetapi ingin beralih kembali ke pengguna saya sendiri sejenak. Untuk contoh lain, dalam sesi SSH, saya dapat menggunakan ~ shortcut (~, ^ Z) untuk menangguhkan shell jarak jauh dan kembali ke shell lokal saya (tapi itu jelas merupakan fitur SSH, bukan shell).

bash memotong ^ Z untuk melakukan kontrol prosesnya sendiri, jadi itu harus menjadi perintah terpisah (bahkan jika didukung). Tetapi mencoba menemukannya hanya menghasilkan informasi tentang kontrol proses normal.

Bash memiliki suspend bawaan:

suspend: suspend [-f]
    Suspend shell execution.

    Suspend the execution of this shell until it receives a SIGCONT signal.
    Unless forced, login shells cannot be suspended.

    Options:
      -f    force the suspend, even if the shell is a login shell

    Exit Status:
    Returns success unless job control is not enabled or an error occurs.

Coba dengan:

kill -STOP $$

di mana $$ adalah PID dari cangkang saat ini.

Anda bisa mengikatnya ke Alt + z misalnya dengan menambahkan ke ~/.inputrc:

"\ez": "kill -STOP $$\n"

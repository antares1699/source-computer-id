Tidak, Windows 7 tidak menyediakan dukungan bawaan untuk menjalankan mesin virtual. Itu memang memiliki dukungan bawaan untuk memasang file disk PC Virtual (.vhd) tetapi itu hanya memungkinkan Anda untuk memperlakukannya sebagai drive data.

Apa yang Anda cari adalah Windows Virtual PC persembahan. Virtual PC 2007 masih akan berjalan di Windows 7, tetapi tidak memiliki semua ekstensi fitur dari Windows Virtual PC.

Windows Virtual PC melakukannya memerlukan dukungan virtualisasi perangkat keras, yang seharusnya Anda miliki, dan memungkinkan Anda membuat dan menjalankan gambar virtual. Mode Windows XP adalah unduhan produk terpisah (dan memerlukan Windows Virtual PC) yang memungkinkan Anda menjalankan lingkungan Windows XP virtual dan tanpa hambatan untuk aplikasi lawas. Anda tidak perlu menginstal mode Windows XP jika Anda hanya ingin membuat dan menjalankan mesin virutal.

Anda juga dapat menggunakan VMWare (yang tidak gratis) dan Sun VirtualBox (yang gratis). Keduanya akan menggunakan virtualisasi perangkat keras jika ada, tetapi tidak memerlukannya seperti Windows Virtual PC. Saat ini saya menggunakan VirtualBox karena mesin saya saat ini tidak mendukung virtualisasi perangkat keras.

Tidak, Windows 7 tidak dilengkapi dengan Mesin Virtual bawaan. Masih ada banyak VM gratis dan bagus untuk diunduh, seperti Virtual PC atau VirtualBox.

Ada juga Mode XP Virtual yang berjalan pada backend yang sama dengan Virtual PC dengan beberapa kenyamanan tambahan. Namun, itu harus diunduh secara terpisah.

Juga Windows 7 menyertakan opsi untuk langsung boot dari file VHD (yang digunakan Virtual PC dan VirtualBox dapat digunakan sebagai format untuk hard disk virtual).

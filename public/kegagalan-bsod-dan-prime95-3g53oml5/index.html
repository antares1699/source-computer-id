Kegagalan BSOD dan Prime95

Komputer saya terkenal tidak stabil. Itu layar biru sepanjang waktu. Saya menjalankan Windows 7. Inilah yang ada di dalam kotak:

  • Intel Core i7 920 (Pendingin stok, bukan overclock)
  • Gigabyte EX58-UD3R motherboard
  • Memori OCZ Gold 6 GB (3x2 GB) (disetel ke 1333MHz, memiliki masalah boot jika saya tinggalkan pada 1066)
  • GeForce 9500 GT
  • Catu daya Antec 650W

Saat idle, suhu berkisar antara 40 dan 50 derajat Celcius, menurut SpeedFan. Saya telah menjalankan banyak tes memori, dan tidak satupun yang menghasilkan masalah.

Sekarang saya telah menerima beberapa pesan ketika itu Layar Biru:

  • IRQL_NOT_LESS_OR_EQUAL
  • Kesalahan halaman saat tidak paging (atau semacamnya)
  • Alamat acak / terdaftar

Sayangnya, mereka pergi terlalu cepat bagi saya untuk mengambil banyak dari mereka.

Saya baru saja menjalankan hot fix Microsoft untuk yang pertama (meskipun saya tidak yakin bahwa kesalahan saya 100% sama dengan kesalahan mereka, saya tidak tahu apakah saya mendapatkan bagian 0x0000000A), jadi saya tidak tahu apakah itu akan membantu atau tidak, tetapi jika Prim95 adalah indikasi apa pun, itu tidak akan membantu, karena alasan berikut:

Ketika saya menjalankan Prime95, 8 utas dimulai, dan semuanya berhenti dengan sangat cepat. Saya mendapatkan kesalahan berikut di file results.txt:

Ketika saya melihat file stress.txt, itu menyarankan memori mungkin menjadi masalah saya, tetapi seperti yang saya katakan, saya telah menjalankan beberapa tes memori (menurut saya MemTest86? Itu beberapa waktu yang lalu), dan tidak ada masalah yang terdeteksi.

Setelah menjalankan hotfix, utas pengujian berhasil tetap berjalan sedikit lebih lama, dan sementara suhu saya pasti naik, mereka tidak pernah benar-benar mencapai suhu 60C.

Jadi, pada dasarnya saya melihat tiga masalah:

  1. Aku kepanasan. Dengan stok pendingin, saya menganggur mendekati 50 beberapa inti dengan sisi kasing saya mati. Menempatkan tangan saya di depan Kipas CPU, saya tidak terlalu merasakannya angin sepoi-sepoi. Apakah ini normal untuk 920 stok pendingin?
  2. Saya layar biru sepanjang waktu (seperti 1-4 kali per hari).
  3. Sepertinya saya tidak bisa menjalankan Prime95 lebih dari beberapa detik.

Adakah yang bisa mengarahkan saya ke arah apa yang mungkin salah di sini, dan mungkin apa yang harus dilakukan untuk mengonfirmasi / memperbaiki masalah?

Hal pertama yang pertama - Buka Panel Kontrol> Sistem ( Tombol Windows + Jeda istirahat ) dan kemudian di bawah Lanjutan, Anda akan melihat "Startup dan pemulihan", klik Pengaturan dan Anda dapat menonaktifkan Mulai ulang otomatis pada kegagalan sistem.

Lain kali jika BSOD terjadi, Anda dapat melihat apa penyebabnya.

Juga, Anda mungkin ingin melihat Tampilan Layar Biru , Alat yang sangat bagus untuk membantu Anda melihat kesalahan layar Biru sebelumnya.

Sekarang, As ~ quack berkata, hanya karena lulus beberapa tes, tidak berarti itu bagus. Jika Anda menjalankannya selama beberapa jam, menukar modul dan menjalankannya kembali dapat membuatnya dengan cepat menyentuh beberapa tempat yang sebelumnya tidak - tetapi sungguh, kecuali Anda menjalankannya memtest86 + selama sekitar (atau idealnya lebih) 48 jam, Anda tidak akan mendapatkan hasil yang baik.

Selanjutnya, kesalahan yang Anda katakan paling sering disebabkan oleh memori yang rusak / rusak / buruk, tetapi sebenarnya bisa apa saja - alasan yang paling mungkin adalah driver perangkat yang buruk / cerdik.

Jika Anda mendapatkan ini setiap kali Anda menjalankan Prime95, saya sangat menyarankan Anda untuk mencoba mencabutnya SEGALA SESUATU dari mesin Anda selain daya, video dan keyboard (dan mouse, kecuali Anda yakin dapat menggunakan mesin tanpa salah satunya). Sekarang, masuk ke mode aman dan coba jalankan Prime 95 lagi. Ini adalah cara terbaik untuk menguji apakah ini masalah driver - selain benar-benar menginstal ulang Windows dari awal dan tidak menginstal driver apa pun!

Jika Anda masih melihat masalah acak dan Memtest86 + benar-benar tidak menunjukkan kesalahan, kemungkinan besar ada masalah dengan motherboard atau bahkan CPU, namun ini bisa terjadi. sangat sulit untuk didiagnosis.

Sedangkan untuk suhu - semakin rendah suhunya, semakin lambat kecepatan kipas - CPU Anda sangat dingin dan tidak ada yang perlu dikhawatirkan.

Recuva adalah utilitas Windows freeware untuk memulihkan file yang telah terhapus secara tidak sengaja dari komputer Anda. Ini termasuk file yang dikosongkan dari Recycle bin serta gambar dan file lain yang juga telah dihapus dari lokasi lain.

Anda juga dapat mencoba Undelete360 yang merupakan program pemulihan file sederhana yang dapat membatalkan penghapusan file yang baru saja terhapus secara tidak sengaja. Ada juga versi portabel yang tersedia, jika Anda tidak mau Install perangkat lunak apa pun di PC Anda.

Apakah Anda mengaktifkan Shadow Copy?

Jika demikian, Anda berpotensi dapat memulihkannya dari snapshot Salinan Bayangan terakhir sebelum penghapusan.

Klik kanan di folder tempat file yang dihapus dulu ada, dan lihat apakah ada opsi "Pulihkan versi sebelumnya" di menu konteks.

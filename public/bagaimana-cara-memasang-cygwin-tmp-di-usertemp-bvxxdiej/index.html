Bagaimana cara memasang Cygwin / tmp di usertemp?

Saya telah menambahkan baris ini ke /etc/fstab saya, diambil dari Panduan pengguna Cygwin, bagian 3.1.5 :

none /tmp usertemp binary,posix=0 0 0

Namun, saya mendapatkan kesalahan saat mencoba memasangnya:

# mount /tmp
mount: /tmp: Invalid argument

Baik TMP dan TEMP env vars mengarah ke / cygdrive / r / tmp (yang ada di drive RAM) Saya dapat melakukan mount r:/tmp /tmp tanpa masalah, saya hanya berpikir menggunakan usertemp akan diinginkan untuk menghindari hardcode dir temp.

Untuk saat ini, saya menggunakan baris ini di fstab saya:

R:/tmp /tmp ntfs binary,posix=0  0 0

Apa alasannya tidak berfungsi?

    — Gene Pavlovsky

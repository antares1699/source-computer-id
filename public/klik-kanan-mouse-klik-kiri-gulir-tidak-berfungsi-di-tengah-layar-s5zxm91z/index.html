klik kanan mouse / klik kiri / gulir tidak berfungsi di tengah layar

Saya pikir ini mungkin hal teraneh yang pernah saya lihat dilakukan oleh komputer. Ini adalah mesin Windows 7 SP1. Ketika saya memposisikan panah di bagian tengah layar, tidak ada yang saya lakukan dengan mouse, kecuali bergerak, yang berpengaruh. Artinya jika kotak dialog kecil muncul, di tengah layar, seperti halnya, saya hanya dapat berkomunikasi dengannya dengan keyboard. Apa yang bisa menjadi penyebabnya, dan apa yang harus saya lakukan untuk menghilangkannya?

Memperbarui: Berikut adalah petunjuk yang disebutkan di komentar. Mereka tidak ada urusannya di sana.

Pertama, dua pop-up pertama, tepat setelah sistem dimulai. Dua lainnya muncul setelah antivirus mengunduh pembaruan. Yang di kanan muncul di atas yang di kiri. Saya telah menyeretnya menggunakan keyboard (mouse tidak akan berfungsi di sana). Yang di kiri tidak responsif. Saat saya mematikan yang pertama, masalah tetap ada. Saya yakin masalahnya hilang saat yang kedua muncul atau saat saya mematikannya.

Memperbarui: Menghapus Avira menghilangkan masalah, tetapi sayangnya menginstalnya lagi membuatnya kembali. Saya akan menambahkan tag avira.

Jelas ini adalah utas lama jadi Anda mungkin sudah menemukan solusi atau cara untuk mengatasinya. Namun, karena ini adalah salah satu google resluts pertama untuk masalah ini, saya pikir saya akan membagikan solusi saya di sini-demi orang lain yang masih mencarinya.

Jika Anda mengalami masalah yang sama persis dengan OP (avira diinstal, pop-up unduhan aneh) - matikan proses ipmgui.exe dan tolak akses baca / tulis padanya. Bagian layar yang tidak dapat diklik adalah jendela bayangan yang selalu ada di atas yang dibuat oleh proses notifikasi avira.

Jika Anda tidak memiliki avira, dapatkan file Sysinternals Process Explorer dan gunakan untuk mengidentifikasi prosesnya. ("tahan dan seret ikon 'target' dari procexp ke jendela 'hantu' dan lepaskan.") 'atkosd2.exe' misalnya, diketahui menyebabkan masalah yang sama.

Hari ini masalah yang sama muncul di PC Win7-SP1 saya. Saya tidak pernah memilikinya sebelumnya meskipun menggunakan Avira sejak bertahun-tahun. Dengan Process Explorer saya menemukan bahwa jendela misterius berasal dari proses "c: \ Program Files \ Avira \ Launcher \ Avira.ServiceHost.exe". Setelah membunuhnya, masalah terus berlanjut. Berharap Avira segera menyelesaikan ini.

Ini bisa menjadi jendela tak terlihat yang terjebak di bagian tertentu dari layar.

Dalam kasus saya, itu di sisi kanan. Itu adalah pemberitahuan Obrolan dari aplikasi Slack Windows yang tidak terlihat.

Untuk menutup jendela itu, cara yang sangat sederhana hanya dengan mengklik Tombol Windows + Tombol Tab atau Tombol ALT + Tombol Tab .

Saat Anda mengklik tombol-tombol ini, ini akan menampilkan semua jendela yang terbuka.

Tekan tombol Tabs Menahan Tombol Windows dan pergi ke jendela yang tidak terlihat, tapi Anda bisa melihatnya seperti ini.

Setelah membuka jendela itu, tekan Tombol ALT + F4 . Ini akan menutup jendela itu dan Anda akan dapat menggunakan area khusus itu juga.

Meskipun ini adalah pertanyaan lama, masih banyak pengguna yang menjalankan antivirus gratis Avira dan masalah ipmgui.exe benar-benar menjengkelkan.

Karena terlepas dari mengatur aturan firewall windows untuk ipmgui.exe, proses akan dimulai secara berkala, saya menemukan itu adalah solusi yang berfungsi sempurna untuk menginstal Manajer Proses Bill2 ( https://www.bill2-software.com/processmanager/download-en.shtml ) dan di sana Anda dapat dengan mudah membuat aturan untuk ipmgui.exe untuk menghentikan proses dengan cepat saat dimulai. Selain itu Anda dapat mengkonfigurasi tooltip untuk ditampilkan di systray ketika ini terjadi dan Anda dapat melihat statistik untuk eksekusi proses dan banyak lagi.

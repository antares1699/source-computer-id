MS Word 2013 Nonaktifkan Penyimpanan Template

Di MS Word 2013, ketika saya membuat dokumen baru dengan template pribadi, setiap kali saya ingin menyimpan dokumen, Word meminta saya untuk menyimpan perubahan ke template juga. Bagaimana cara menonaktifkan prompt ini?

Catatan: Saya TIDAK ingin menyimpan perubahan pada template.

Berikut adalah langkah-langkah sebenarnya yang harus saya ikuti untuk menyelesaikan ini. Saya menggunakan opsi yang disebutkan di atas serta bantuan dari forum lain untuk mencari tahu. Saya menggunakan Office 2013.

  1. Anda harus membuka Word sebagai administrator.
  2. Untuk melakukan ini, klik kanan pada pintasan Microsoft Office Word 2013 Anda dan pilih properti (Jika Anda mengklik pintasan bilah tugas, Anda harus mengklik kanan pada ikon bilah tugas, lalu klik kanan pada pintasan Word 2013 tepat di atas bilah tugas).
  3. Dari menu properti klik Buka Lokasi File.
  4. Dari direktori \ root \ office15, klik kanan pada winword.exe dan pilih Run as administrator
  5. Buka File> Opsi> Add-In
  6. Di bagian bawah, klik tombol Go ... tepat di sebelah menu drop-down COM Add-in.
  7. Hapus centang dari kotak Send to Bluetooth Add-Ins.
  8. Klik OK

Sekarang Anda seharusnya dapat membuka dan mengerjakan file template apa pun tanpa diminta untuk menyimpan perubahan ke template asli Anda. Saya menemukan bahwa plugin Bluetooth telah rusak setidaknya selama satu atau dua tahun sekarang dengan referensi di artikel ini , namun tidak ada referensi ke mana mereka menemukan informasi itu ... Semoga ini bisa membantu.

Saya baru saja mencari hal yang sama ini dan menemukan solusinya sebagai berikut:

Di Word, buka Options > Add-Ins, klik tombol Go di samping Manage COM Add-ins.

Nonaktifkan Add-ins dan ini akan memperbaiki masalah.

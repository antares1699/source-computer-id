Bagaimana SEBENARNYA menginstal Java di Linux?

Saya memiliki Server Ubuntu.

Dari terminal, bagaimana cara menginstal JDK?

Dalam panduan ini dikatakan untuk menggunakan perintah ini:

sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-jdk

Tapi di situs Suns, dikatakan JDK termasuk JRE, jadi kenapa JRE di baris di atas?

Ada yang tahu cara menginstal Java?

Setiap panduan dan setiap forum menunjukkan cara yang berbeda untuk melakukannya.

Inilah yang Anda dapatkan dari setiap paket:

sun-java6-bin: Basefiles untuk menjalankan program Java

sun-java6-jre: File pelokalan dan semua yang Anda butuhkan untuk memenuhi persyaratan Runtime "lengkap". Tergantung pada bin.

sun-java6-jdk: Semua yang Anda butuhkan untuk mengkompilasi sesuatu. Ini tidak perlu Runtime untuk menjalankan program java. Tergantung pada bin.

Jadi Anda dapat menulis perintah Anda lebih pendek seperti:

sudo apt-get install sun-java6-jre sun-java6-jdk

Oke, ini adalah jawaban kedua saya untuk pertanyaan ini, dan ini adalah cara Anda menginstal JDK saat ini di linux! (Ini untuk Debian, tetapi pengguna Ubuntu seharusnya tidak melihat banyak perbedaan).

Alat yang Anda butuhkan:

apt-get install fakeroot zip unzip mkisofs

Sekarang pergi ke java.sun.com dan unduh JDK6 terbaru. Seharusnya namanya kira-kira seperti jdk-6u20-linux-i586.bin!

Kami tidak ingin menginstalnya secara langsung untuk membuatnya berperilaku seperti paket yang baik. Jadi kami menggunakan fakeroot untuk menginstalnya sebagai pengguna.

mv jdk-6u20-linux-i586.bin ~someuser
su - someuser
fakeroot
chmod a+x jdk-6u20-linux-i586.bin
sh ./jdk-6u20-linux-i586.bin
cd jdk1.6.0_20/man
for i in `find -type f`; do gzip $i; done;
cd ../..
# Tar the results to have them handy for the real installation
tar cfz jdk1.6.0_20.tgz jdk1.6.0_20/

Sekarang kita tinggalkan fakeroot (Ctrl-D) dan someuser-shell (Ctrl-D lagi), dan instal tar kita ke / usr / local:

cd /usr/local
tar xfvz ~someuser/jdk1.6.0_20.tgz
ln -s /usr/local/jdk1.6.0_20/ /usr/local/jdk
ln -s /usr/local/jdk1.6.0_20/jre/ /usr/local/jre

for program in appletviewer apt extcheck idlj jar jarsigner java javac \
           javadoc javah javap javaws jconsole jdb jhat jinfo jmap \
           jps jrunscript jsadebugd jstack jstat jstatd jvisualvm \
           keytool native2ascii orbd pack200 policytool rmic rmid \
           rmiregistry schemagen serialver servertool tnameserv \
           unpack200 wsgen wsimport xjc ; do \
    update-alternatives --install "/usr/bin/$program" "$program" \
    "/usr/local/jdk1.6.0_20/bin/$program" 1130 \
    --slave "/usr/share/man/man1/$program.1.gz" "$program.1.gz" \
    "/usr/local/jdk1.6.0_20/man/man1/$program.1.gz"; done;

Biarawati testen ob alles ok ist:

java -version
man java

Selesai.

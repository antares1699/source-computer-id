Tidak dapat berbagi layar pada Zoom di Ubuntu 17.10

Ketika saya mencoba berbagi layar di Zoom, saya mendapatkan kesalahan berikut:

tidak bisa mulai berbagi, wayland belum didukung, silakan gunakan x11 sebagai gantinya

  • Zoom versi 2.0.106600.0904

  • Ubuntu 17.10

Bagaimana cara mengatasi masalah tersebut?

Saya mengalami masalah yang sama dengan Fedora 26. Sepertinya wayland adalah window manager yang digunakan beberapa distro sebagai pengganti X11 (Xorg). Untuk melihat window manager apa yang Anda gunakan, Anda dapat menggunakan yang berikut ini:

echo $XDG_SESSION_TYPE

Di Fedora, saya dapat memilih lingkungan desktop (gnome dengan Xorg) di layar login yang akan mengaktifkan X11 alih-alih Wayland - sepertinya Anda dapat melakukan hal yang sama dengan Ubuntu.

Tiket ini memiliki lebih banyak informasi jika Anda tertarik: https://unix.stackexchange.com/questions/336219/gnome-defaults-to-wayland-how-can-i-go-to-back-to-x11

edit file /etc/gdm3/custom.conf dan hapus komentar pada baris berikut:

#WaylandEnable=false

Saya menggunakan perintah ini dan itu berfungsi dengan baik untuk saya tetapi saya pikir lightdm tidak akan digunakan lagi tetapi ini mungkin dapat membantu seseorang:

sudo dpkg-reconfigure lightdm

Jika lightdm tidak diinstal, jalankan saja sudo apt-get install lightdm dan layar dengan 2 opsi akan muncul, pilih lightdm dan restart.

Dan btw ini juga harus berfungsi:

sudo vi /etc/gdm3/custom.conf
uncomment #WaylandEnable=false
restart

Bagi saya ini bekerja dengan baik:

langkah 1: sudo chmod 777 /etc/gdm3/custom.conf
langkah 2: gedit /etc/gdm3/custom.conf
langkah 3: tanpa komentar WaylandEnable=false
langkah 4: simpan file dan tutup
langkah 5: restart komputer
setp 6: sekarang, dapat berbagi layar di skype, hangouts dan zoom

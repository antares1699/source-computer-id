Bagaimana cara menginstal AmigaOS 4.1 pada sistem x86 standar?

Itu AmigaOS mungkin hanya diingat sebagai sistem operasi dari tahun 80-an dan 90-an, tetapi masih hidup, dan telah berkembang menjadi OS modern, rilis terbaru adalah 4.1 pada Desember 2014, dengan Meja Kerja UI modern yang mendukung banyak perangkat lunak modern.

Namun, keluarga perangkat keras utama tampaknya masih PowerPC, dan tidak jelas bagi saya apakah ini dapat berjalan secara native pada arsitektur x86. Tentu saja, dalam kasus terburuk, saya dapat mengejar perangkat keras tertentu, seperti dari daftar ini , tetapi apakah ada yang memiliki pendekatan langsung untuk melakukan ini pada x86 standar?

saya tidak mencari beberapa emulator retro, karena itu tidak ada hubungannya dengan Amiga modern, tetapi pendekatan VM, Linux lebih disukai, mungkin dapat dilakukan jika x86 tidak.

Adakah cara yang baik untuk menjalankan AmigaOS 4.1 di x86?

Edit: Sepertinya ada tutorial untuk menginstal versi sebelumnya, AmigaOS XL, dari 2002, di sistem x86. Emulasi perangkat keras tampaknya cukup lurus ke depan, jadi jika sesuatu yang mirip untuk seri 4.x ada, saya akan sangat berterima kasih.

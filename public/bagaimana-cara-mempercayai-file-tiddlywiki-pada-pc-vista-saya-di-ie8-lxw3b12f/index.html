Bagaimana cara mempercayai file TiddlyWiki pada PC Vista saya di IE8?

Setiap kali saya membuka file tiddlywiki.html saya di IE8, saya mendapatkan bilah peringatan:

Untuk membantu melindungi keamanan Anda, Internet Explorer telah membatasi halaman web ini dari menjalankan skrip atau kontrol ActiveX yang dapat mengakses komputer Anda. Klik di sini untuk opsi.

dan tiddlywiki tidak ditampilkan. Jika saya memilih "Izinkan Konten yang Diblokir ..." tidak apa-apa.

Saya ingin mengubah pengaturan IE8 apa pun yang diperlukan sehingga saya tidak perlu melakukan ini. Ada yang tahu pengaturan apa itu?

Saya mencoba menambahkan file c:\users\me\tiddlywiki.html saya ke "Situs Tepercaya" di IE-> Alat-> Opsi-> Keamanan tetapi dikatakan bahwa URL tidak cocok dengan beberapa validasi ekspresi reguler.

TiddlyWiki menggunakan ActiveX FileSystemObject untuk mengakses sistem berkas di bawah penjelajah internet. Untuk menggunakan objek ActiveX ini, Anda tidak perlu menyetel tingkat keamanan browser ke rendah seperti yang dijelaskan di beberapa halaman. Anda hanya boleh menyetel satu setelan keamanan "Inisialisasi dan skrip kontrol ActiveX tidak ditandai sebagai aman untuk skrip" ke "Prompt" atau "Aktifkan". Itu dia.

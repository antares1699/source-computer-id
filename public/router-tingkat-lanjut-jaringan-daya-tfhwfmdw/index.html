Router Tingkat Lanjut - Jaringan daya

Saya sedang mempertimbangkan untuk memutakhirkan router yang dilengkapi mikro DD-WRT saya saat ini ke sesuatu yang lebih kuat. Saya tertarik pada router daripada menangani setidaknya dua kali lipat jumlah koneksi yang dapat digunakan router saya saat ini (4096) untuk menangani lalu lintas BitTorrent dengan lebih baik saat memaksimalkan bandwidth koneksi saya (16mbps / 2mbps). Selain itu, saya ingin membuat gigabit lan untuk streaming video 1080P ke seluruh rumah saya.

Untuk jenis situasi ini, apakah router kelas konsumen akhir yang lebih tinggi dapat menyesuaikan tagihan ini dengan DD-WRT di atasnya untuk pengaturan tweaker, apakah saya akan lebih baik melihat router kelas komersial, atau bagaimana membangun mesin linux untuk bertindak sebagai router dan menggunakan sakelar dalam kombinasi dengan itu, bukan?

Saya mencari yang terbaik untuk uang saya, karena saya yakin ada banyak $ 1000 + router komersial yang dapat melakukan semua yang saya inginkan tetapi berlebihan untuk aplikasi saya.

Semua ini akan menangani a banyak koneksi, dan dapat melakukan hampir semua hal yang Anda inginkan dari sistem seperti itu (multi-koneksi dengan failover, QoS, deteksi intrusi, host / koneksi ke VPN, dll.)

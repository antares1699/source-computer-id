Untuk mengimpor data dari Excel ke SQL Azure, Anda harus menyambungkan ke server lokal dan menggunakan panduan Impor / Ekspor, karena tidak terlihat dari koneksi Azure.

saat Anda menyambungkan ke server lokal, buka tugas -> Impor data dan sambungkan ke Excel dan file Anda:

Klik Berikutnya dan sambungkan ke db SQL Azure Anda:

Dan ikuti wizard.

Berikut ini juga video yang menjelaskan keseluruhan proses secara detail dengan beberapa tip: http://www.sqltrainingonline.com/microsoft-excel-to-windows-azure-sql-in-10-minutes/

Anda dapat membuka hanya Import-Wizard secara langsung karena ini adalah Program yang Berdiri Sendiri. Cukup cari "SQL Server 2014 Impor dan Ekspor Data" di Start Menu Anda atau buka DTSWizard.exe yang seharusnya terletak di direktori instalasi Anda di bawah \ DTS \ Binn \

Tidak perlu repot mengotak-atik database lokal. Juga tidak perlu memulai Management-Studio sama sekali.

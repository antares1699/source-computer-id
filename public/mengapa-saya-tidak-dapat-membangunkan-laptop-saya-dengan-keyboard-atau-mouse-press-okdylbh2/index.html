Mengapa saya tidak dapat membangunkan laptop saya dengan keyboard atau mouse press?

Saya punya Asus n56vv laptop yang menjalankan Windows 8.1 profesional yang memberi saya masalah yang cukup menjengkelkan: Saya tidak dapat membangunkannya dari tidur dengan menekan tombol pada mouse atau keyboard. Ini terjadi pada keyboard laptop terintegrasi dan USB yang saya sambungkan (saya tertarik untuk membangunkannya dengan keyboard USB atau mouse btw). Satu-satunya cara untuk membangunkannya tampaknya menjadi tombol Power di laptop itu sendiri.

Saya bisa melakukan ini dengan laptop lama saya (5 tahun), saya menolak untuk percaya yang baru seperti ini tidak mendukung fungsi dasar ini.

Saya sudah mengaktifkan opsi "izinkan perangkat ini untuk membangunkan komputer" di setiap port usb / perangkat input yang saya bisa di Device Manager, beberapa di antaranya memiliki opsi yang berwarna abu-abu, jadi saya tidak dapat melakukannya untuk semuanya . Tapi sepertinya tidak ada efeknya.

Setiap ide dihargai.

DI SINI ADALAH PERBAIKAN:

Opsi Daya (pilih paket daya Anda. Balanced, high, dll)> Pengaturan USB> Nonaktifkan "Pengaturan penangguhan selektif USB"

SELESAI.

Ini sudah dijawab oleh pengguna280074.

Untuk berjaga-jaga, jika masih tidak berhasil, coba ini:

  1. Periksa BIOS dan nonaktifkan tidur nyenyak.
  2. Ubah port USB untuk keyboard dan mouse.
  3. Pulihkan default paket untuk semua paket daya.

Inilah cara saya menyelesaikannya:

Awalnya, tab "Manajemen Daya" yang berisi opsi "Izinkan perangkat ini untuk membangunkan komputer" bahkan tidak ada.

Untuk memperbaikinya, saya pergi ke pengaturan BIOS dan mengaktifkan opsi ini "Aktifkan Dukungan USB Wake" . (Tidak yakin apakah opsi ini tersedia di semua laptop. Milik saya adalah Dell Inspiron).

(Gambar dikreditkan ke pengguna8024277 )

Setelah saya mengaktifkan opsi ini di BIOS dan mem-boot ke Windows, tab Manajemen Daya muncul di perangkat Keyboard USB dan Mouse Eksternal saya. Saya mengaktifkan "Izinkan perangkat ini untuk membangunkan komputer" pilihan dan sekarang saya bisa membangunkan laptop saya dengan keyboard dan mouse eksternal.

Catatan: "Izinkan perangkat ini untuk membangunkan komputer" - Opsi ini hanya tersedia di keyboard dan mouse eksternal saya. Itu masih tidak tersedia untuk keyboard dan touchpad bawaan laptop saya.

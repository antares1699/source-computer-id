memaksimalkan jendela Command prompt

Mengklik tombol maksimalkan pada prompt perintah Windows tidak membantu karena ukurannya tidak bertambah. Pertama-tama, mengapa demikian?

Selain itu, jika Anda menjalankan ini,

C:\Windows\System32\cmd.exe /k "mode 200,70"

Anda mendapatkan Jendela yang lebih besar. Bagaimana itu bisa terjadi? Apa /k "mode 200,70" artinya?

Ada dua bagian untuk menjawabnya.

Itu opsi shell perintah / k memulai prompt perintah baru, menjalankan beberapa perintah (atau program, skrip batch, atau sekelompok perintah) dan prompt tetap daripada berhenti.

Dalam hal ini, perintahnya adalah mode, yang dapat digunakan untuk menyetel file ukuran layar prompt perintah / penyangga jendela jadi lebar 200 kolom dan tinggi 70 baris.

Terakhir, jika Anda ingin mengubah ukuran default dari command prompt, klik kanan pada bilah judul, pilih 'Properties', dan pilih tab 'Layout'. Anda dapat mengatur ukuran buffer layar dan jendela secara terpisah. Lebarnya harus sama untuk keduanya. Pengaturan ketinggian penyangga layar menentukan seberapa jauh Anda dapat menggulir ke belakang.

/K | /C Digunakan untuk menjalankan perintah setelah meluncurkan cmd.exe. Dengan demikian Anda menjalankan perintah / program mode. Dengan mode ada beberapa opsi, 200,70 adalah mengubah tampilan, tetapi ada beberapa opsi lain yang dapat digunakan dengan mode.

Adapun alasan Anda tidak dapat memaksimalkan cmd.exe adalah buffer layar. Anda harus memeriksanya ini situs yang akan menunjukkan cara mengubahnya.

Windows menjalankan cmd.exe seperti kebanyakan aplikasi Windows lainnya, di jendela. Mengklik pada tombol maksimalkan "memaksimalkan jendela, tetapi ini masih sebuah jendela.

Di bawah versi Windows sebelumnya, prompt perintah dapat dialihkan ke layar penuh (sebagai lawan dari jendela) dengan menekan ALT / Enter dari dalam jendela perintah. ALT / Enter kembali ke mode berjendela. Saya baru saja mencobanya di desktop Vista saya dan mendapat kesalahan yang mengatakan bahwa sistem saya tidak mendukung mode layar penuh, bagaimanapun, itu mungkin karena saya memiliki banyak monitor. Laptop Windows 7 saya tampaknya mengabaikan ALT / Enter.

Sedangkan untuk sakelar cmd.exe, buka jendela perintah dan ketik "cmd /?" atau "bantu cmd" dan Anda akan mendapatkan banyak detail.

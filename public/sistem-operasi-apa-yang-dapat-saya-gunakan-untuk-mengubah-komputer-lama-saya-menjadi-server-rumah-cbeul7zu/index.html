Saya menjalankan server web / file pada PIII 450 dengan 640mb ram. Untuk fileserver 'murni', freenas bagus, jika tidak menggunakan cli hanya menginstal ubuntu bekerja dengan baik. Demikian juga, Anda memerlukan setidaknya satu port jaringan, secara alami, dan Anda dapat mengatur kotak tanpa kepala, melakukan tugas admin melalui openssh. Saya menggunakan kurang dari 100 mb ram sekarang dengan file / server web berbasis ubuntu jadi ... ya, itu mungkin akan berhasil

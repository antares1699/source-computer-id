Bagaimana cara menggunakan internet di ubuntu menggunakan nokia mobile?

Saya memiliki ponsel Nokia tetapi saya tidak memiliki Nokia PC Suite untuk Ubuntu Tapi saya punya setelan PC untuk windows. Bagaimana saya bisa menggunakan pengaturan itu untuk ubuntu atau memberi saya cara lain untuk menggunakan internet menggunakan ponsel Nokia

Tidak terlalu sulit, saya melakukannya dengan milik saya. (Nokia N95 / Ubuntu 9.10)

  • Pertama, pasang telepon dengan kabel USB (tidak tahu bagaimana melakukannya melalui Bluetooth),
  • di telepon, pilih "PC Suite"
  • di Ubuntu, klik kanan manajer jaringan (ikon di kanan atas) dan pilih 'edit koneksi'
  • Klik tab 'Mobile Broadband'
  • Klik 'Tambahkan' di kanan atas
  • Pilih ponsel Anda dari daftar (atau mana saja yang paling mirip)
  • Ikuti petunjuk untuk mengisi nama pengguna / kata sandi Anda (Jika Anda tidak melakukan ini di ponsel Anda, Anda tidak perlu melakukannya sekarang - default harus bagus, tetapi itu tergantung pada penyedia layanan Anda)

Anda dapat mengatur koneksi itu menjadi 'otomatis', kemudian setiap kali Anda memasang plugin telepon dan mengklik 'pc suite' di masa mendatang, internet akan terjadi begitu saja.

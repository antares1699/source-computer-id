Jika Anda hanya ingin menyorot sumber, Anda membuangnya ke terminal dengan gaya cat coba luar biasa pigmen . Ini sebenarnya mendukung banyak bahasa. Jalankan dengan

pygmentize source.py

Jika sebaliknya Anda berbicara tentang mengedit file dengan penyorotan sintaks di dalam terminal mac, ini sangat tergantung pada editor.

  • untuk vi atau vim paling tidak Anda menginginkan opsi ini pada ~/.vimrc

    syntax on

  • emacs bekerja langsung di luar kotak

  • jangan tanya saya tentang nano.

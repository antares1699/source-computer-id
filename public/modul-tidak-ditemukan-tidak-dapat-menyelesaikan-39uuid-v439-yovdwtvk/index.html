Modul tidak ditemukan: Tidak dapat menyelesaikan 'uuid / v4'

Saya seorang pemula dalam pemrograman React dan saya mengikuti tutorial ini di Youtube: https://youtu.be/hQAHSlTtcmY dan semuanya berjalan dengan baik sampai diperlukan untuk menginstal pustaka uuid dan setelah itu ketika saya menjalankan halaman saya mendapatkan pesan kesalahan yang mengatakan: "Gagal mengkompilasi.

./src/App.js

Modul tidak ditemukan: Tidak dapat menyelesaikan 'uuid / v4' di 'C: \ Users \ username \ reactnative \ src' "jadi adakah yang bisa membantu saya memperbaiki kesalahan ini? Dan juga tidak ada kesalahan lain dalam kode dan itu persis sama seperti yang ditunjukkan di video!

Instal pustaka uuid melalui manajer paket node, yaitu npm

npm install uuid

Kemudian impor uuidv4 dari pustaka uuid

   const { v4: uuidv4 } = require('uuid');

Fungsi di dalam:

  setTodo(prevTodo => {

    return [...prevTodo,{id:uuidv4(), name: name, complete:false}]
  })

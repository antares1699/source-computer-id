Pengguna domain tidak memiliki hak admin di komputer lokal

Saya menggunakan server Windows 2003 sebagai direktori aktif dan Windows 7 sebagai stasiun. Saya harus memiliki pengguna domain biasa yang memiliki hak admin di mesin lokal. Saya menambahkan pengguna domain ke grup admin lokal tetapi mereka masih tidak memiliki akses penuh sebagai administrator lokal.

8

Di workstation, masuk sebagai admin domain dan masalah

net localgroup Administrators /add DOMAIN\USER

Itu akan membuat pengguna domain menjadi Administrator di workstation itu saja.

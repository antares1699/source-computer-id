Anda dapat mencoba menginstal MacPorts dan instal paket renameutils:

ganti namautils @ 0.10.0 (sysutils)

renameutils adalah sekumpulan program yang dirancang untuk membuat penggantian nama file lebih cepat dan tidak terlalu merepotkan

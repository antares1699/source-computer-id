Jika Anda membuka regedit.exe dan menambahkan "Start_NotifyNewApps" sebagai Nilai DWORD (32-bit) dengan nilai "0" ke jalur berikut dan melakukan boot ulang, penyorotan akan dinonaktifkan:

 HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

Karena ini adalah pratinjau pertama dengan Start Menu baru, saya tidak akan terkejut jika banyak fitur dari Start Menu yang lama belum memiliki analog.

Karena itu, aplikasi / program terbaru yang diinstal akan disorot secara otomatis secara default, tetapi saat ini tidak ada cara untuk menonaktifkannya. Ini tangkapan layar, saya baru saja menginstal aplikasi Netflix ...

Kelemahan: Ini sepertinya hanya menampilkan satu atau dua aplikasi terakhir yang diinstal, dan saat Anda melihat Semua Aplikasi, tidak ada penyorotan.

Jadi ini jelas masih membutuhkan beberapa pekerjaan. :) Mungkin melaporkannya ke MS (melalui aplikasi Windows Feedback di Win10) sehingga mereka tahu orang-orang menginginkannya (kembali).

Alternatif untuk saat ini adalah kembali menggunakan Start Layar dan, seperti di Windows 8.1, klik "Semua Aplikasi" di mana aplikasi baru akan memiliki tag "Baru", dan / atau Anda dapat mengurutkannya "berdasarkan tanggal terpasang" untuk mendapatkan yang terbaru di awal.

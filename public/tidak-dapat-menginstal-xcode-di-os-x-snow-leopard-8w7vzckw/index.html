Tidak Dapat Menginstal Xcode di OS X Snow Leopard

Saya telah mencoba 3.2.5, 3.2.6, 4.0.2, 4.2 dll dan saya mendapatkan kesalahan penginstalan segera setelah memberikan kredensial. Saya memberikan nama pengguna, kata sandi. Saya adalah admin. Kemudian saya mendapatkan pesan kesalahan "Penginstalan Gagal":

Terjadi kesalahan penginstalan yang tidak diketahui. Penginstal mengalami kesalahan yang menyebabkan penginstalan gagal. Hubungi produsen perangkat lunak untuk bantuan.

Bagaimana saya bisa mengetahui apa masalahnya?

bagaimana cara membuat emacs menggunakan css lain untuk mempublikasikan orgmode

Saat menggunakan orgmode di Emacs, saat saya memublikasikan ke HTML, ia menggunakan CSS default yang tidak terlihat bagus.

Setelah menerbitkan, saya secara manual membuang CSS lama untuk yang khusus.

Apakah ada cara di Emacs untuk membuatnya menggunakan file CSS saya yang lain secara default?

4
reinier — Sumber

1 Jawaban

6
6

Gunakan parameter :style dalam proyek penerbitan Anda. Sebagai contoh:

(setq org-publish-project-alist
      '(("Homepage"
     :base-directory "~/git/homepage/"
     :base-extension "org"
     :publishing-directory "~/public_html/"
     :publishing-function org-publish-org-to-html
     :style "<link rel=\"stylesheet\" href=\"style.css\" type=\"text/css\" />")))

Anda juga dapat menentukan opsi per file seperti ini:

#+STYLE: <link rel="stylesheet" type="text/css" href="stylesheet.css" />

Melakukan C-h v org-export-html-style TAB TAB untuk membuat daftar berbagai opsi yang terkait dengan gaya untuk ekspor HTML.

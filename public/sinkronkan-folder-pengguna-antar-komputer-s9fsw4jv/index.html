Sinkronkan folder pengguna antar komputer

Adakah cara agar saya dapat menyinkronkan seluruh folder pengguna (Dokumen, Gambar, Musik, Video, dll.) Antara 2 komputer? Saya tidak ingin menggunakan Dropbox (atau layanan sinkronisasi folder tunggal serupa) karena saya memiliki banyak file dan terpaksa memindahkan semua yang ada di Dropbox.

Idealnya saya ingin mempertahankan struktur di desktop saya dan memiliki file-file itu tersedia di laptop saya.

Kedua komputer saya menjalankan Windows 8.1 Pro.

Jika kedua komputer ini ada dalam jaringan yang sama, Anda dapat mengatur jaringan berbagi dan mengubah lokasi yang folder ini tunjuk di setiap komputer (baik secara manual dengan masuk ke direktori pengguna (Di bilah Alamat Explorer, saat masuk ke pengguna yang dimaksud, ketik% USERPROFILE%) atau melalui GPO).

Perlu dicatat bahwa berikut ini hanya akan berfungsi jika kedua komputer tetap berada di jaringan yang sama (baik melalui VPN atau secara lokal tidak masalah). Saat Anda mencoba mengakses dokumen Anda di laptop jauh dari jaringan tempat desktop Anda aktif, Anda akan mengalami masalah.

Di desktop Anda:

  1. Klik Mulai
  2. Buka Panel Kontrol
  3. Buka Jaringan dan Pusat Berbagi
  4. Buka Pengaturan Berbagi Lanjutan (Di bilah sisi kiri)

Di bawah Rumah atau Kantor

  1. Aktifkan Penemuan Jaringan
  2. Aktifkan Berbagi File dan Printer
  3. Tekan Simpan Perubahan

Tutup jendela yang sedang terbuka.

  1. Buka Explorer (Komputer)
  2. Di bilah alamat, ketik% USERPROFILE%
  3. Tekan enter
  4. Klik kanan Dokumen Saya
  5. Buka properti
  6. Klik pada Tab Berbagi
  7. Klik tombol bagikan
  8. Tentukan bagaimana Anda ingin mengatur izin berbagi Anda (Di luar ruang lingkup pertanyaan ini)
  9. Klik Bagikan
  10. Klik Tutup

Di laptop Anda:

  1. Klik Mulai
  2. Buka Panel Kontrol
  3. Buka Jaringan dan Pusat Berbagi
  4. Buka Pengaturan Berbagi Lanjutan (Di bilah sisi kiri)

Di bawah Rumah atau Kantor

  1. Aktifkan Penemuan Jaringan
  2. Aktifkan Berbagi File dan Printer
  3. Tekan Simpan Perubahan

Tutup jendela yang sedang terbuka.

  1. Buka Explorer (Komputer)
  2. Di daftar pohon di sisi kiri, navigasikan ke Komputer
  3. Klik Map Network Drive di bilah tindakan di bawah bilah alamat
  4. Tekan telusuri
  5. Pilih share yang ingin Anda petakan dari desktop Anda
  6. Hit Ok
  7. Tekan Selesai

Uji untuk memastikan Anda memiliki akses ke drive baru

  1. Di bilah alamat, ketik% USERPROFILE%
  2. Tekan enter
  3. Klik kanan Dokumen Saya
  4. Buka properti
  5. Klik Tab Lokasi
  6. Klik tombol Pindah
  7. Arahkan ke drive yang baru Anda buat
  8. Klik Pilih Folder

Ini akan meminta Anda menanyakan apakah Anda ingin menggabungkan folder Anda, saya akan merekomendasikannya.

Ulangi petunjuk ini untuk folder pengguna lain yang ingin Anda petakan ulang.

Informasi Lebih Lanjut:

https://technet.microsoft.com/en-us/library/cc732275.aspx

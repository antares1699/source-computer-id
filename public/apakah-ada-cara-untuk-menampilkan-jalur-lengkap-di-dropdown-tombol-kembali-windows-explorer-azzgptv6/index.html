Apakah ada cara untuk menampilkan jalur lengkap di dropdown tombol kembali Windows Explorer?

Tahukah Anda jika ada cara agar Windows Explorer menampilkan jalur lengkap di tarik-turun tombol kembali? (bukan hanya menampilkan bagian terakhir folder).

Pada tangkapan layar di atas, yang ingin saya lihat di tarik-turun yang menunjukkan riwayatnya adalah:

  • C: \ Windows \ Microsoft.NET
  • C: \ Windows
  • C: \
  • Komputer

Ini akan sangat berguna ketika seseorang telah menavigasi semua tempat lainnya, dan ketika ada beberapa nama folder serupa yang berasal dari tempat berbeda yang tidak terkait ...

Dengan menggunakan RegShot Saya menemukan kunci Registry yang akan mengubah beberapa pengaturan Windows Defender.

Untuk mengaktifkan pemindaian drive yang dapat dilepas, navigasikan ke

HKLM \ SOFTWARE \ Microsoft \ Windows Defender \ Scan \ DisableRemovableDriveScanning

Untuk mengaktifkan setel nilai ke 0 - untuk menonaktifkan setel ke 1

Tidak seperti Microsoft Security Essentials, Windows Defender tidak memiliki opsi konfigurasi untuk pemindaian terjadwal di GUI-nya (Antarmuka Pengguna Grafis), tetapi Anda mungkin masih ingin mengotomatiskan pemindaian cepat atau lengkap. Untuk melakukannya, buka Pencarian Pengaturan menggunakan pintasan keyboard Tombol Windows + W, ketik "jadwal" ke dalam kotak Pencarian dan klik Jadwalkan tugas. Pemilik layar sentuh harus terlebih dahulu menggesek dari tepi kanan layar - ini akan membuka bilah Charms di mana Anda dapat mengklik ikon Pencarian. Windows 8, layar Mulai, Pencarian pengaturan. Untuk menjadwalkan tugas Windows Defender, ketik "jadwal" ke dalam kotak Pencarian. Kemudian klik "Jadwalkan tugas".

Klik kanan Penjadwal Tugas (Lokal) di sisi kiri dan pilih Buat Tugas Dasar.

Buat Wisaya Tugas Dasar terbuka. Ketikkan nama deskriptif untuk tugas pemindaian dan klik Next.

Jika Anda ingin menjalankan pemindaian mingguan cepat, setel frekuensinya ke Mingguan. Karena pemindaian penuh membutuhkan waktu lama untuk diselesaikan, Anda harus menggunakan Monthly untuk ini.

Tetapkan hari kerja dan waktu untuk pemindaian cepat, atau semua bulan, hari dan waktu untuk pemindaian lengkap. Karena Anda tidak dapat membatasi penggunaan CPU, pilih waktu di mana PC Anda kemungkinan besar dihidupkan, tetapi tidak dalam penggunaan yang sangat aktif - selama pemindaian, komputer Anda melambat.

Dalam pemilihan tindakan, default Mulai program baik-baik saja.

Klik Jelajahi.

Arahkan ke folder C: \ Program Files \ Windows Defender dan klik dua kali MpCmdRun.exe. Ini adalah file yang dapat dieksekusi yang memungkinkan melakukan tugas-tugas umum di Windows Defender.

Untuk menjadwalkan pemindaian cepat mingguan, ketik "-Scan -ScanType 1" ke dalam kolom Tambahkan argumen (opsional). Untuk melakukan pemindaian penuh, ketik "-Scan -ScanType 2" sebagai gantinya.

Kami hampir selesai di sini. Aktifkan dialog Buka Properti untuk tugas ini saat saya mengklik opsi Selesai sebelum mengklik Selesai.

Di tab Umum dari Tugas yang baru Anda buat, centang kotak Jalankan dengan hak istimewa tertinggi. Ini memungkinkan Windows Defender untuk berjalan dengan hak yang lebih tinggi dan memastikan semua malware benar-benar terhapus.

Buka tab Pengaturan dan nyalakan tugas Jalankan sesegera mungkin setelah jadwal mulai terlewat. Jika komputer Anda dimatikan atau Anda tidak masuk pada waktu yang dijadwalkan, pemindaian akan dimulai setelah Anda masuk ke Windows di waktu berikutnya. Klik OK untuk menutup jendela Properti Tugas.

Pada waktu yang dijadwalkan, jendela Command Prompt hitam muncul. Ini akan menutup secara otomatis setelah pemindaian selesai.

Untuk James dan orang lain yang ingin mengotomatiskan proses penyebaran, PowerShell memiliki alat yang bagus khusus untuk skenario itu, dimulai dengan Windows Server 2012 R2 dan Windows 8.1.

Cmdlet Set-MpPreference mengkonfigurasi preferensi untuk pemindaian dan pembaruan Windows Defender. Lihat artikel TechNet Set-MpPreference untuk rangkaian opsi lengkap dan detail teknis dan platform khusus lainnya.

Untuk keterangan Yakobus ...

Memindai drive yang dapat dilepas

-DisableRemovableDriveScanning<Boolean>

Menunjukkan apakah akan memindai perangkat lunak berbahaya dan tidak diinginkan di drive yang dapat dilepas, seperti flash drive, selama pemindaian penuh. Jika kamu tentukan nilai $ True, Windows Defender memindai drive yang dapat dilepas selama jenis pemindaian apa pun. Jika Anda menentukan nilai $ False atau tidak tentukan nilai, Windows Defender tidak memindai drive yang dapat dilepas selama pemindaian penuh. Windows Defender masih dapat memindai drive yang dapat dilepas selama pemindaian cepat atau pemindaian khusus.

Waktu pemindaian harian

-RemediationScheduleTime<DateTime>

Menentukan waktu hari, sebagai jumlah menit setelah tengah malam, sampai melakukan pemindaian terjadwal. Waktu mengacu pada waktu lokal di komputer. Jika Anda tidak menentukan nilai untuk parameter ini, a pemindaian terjadwal berjalan pada waktu default dua jam setelah tengah malam.

Perlindungan waktu nyata

-DisableRealtimeMonitoring<Boolean>

Menunjukkan apakah akan menggunakan perlindungan waktu nyata. Jika Anda menentukan nilai dari $ True atau tidak menentukan nilai, Windows Defender menggunakan waktu nyata perlindungan. Kami menganjurkan agar Anda mengaktifkan Windows Defender untuk digunakan perlindungan waktu nyata.

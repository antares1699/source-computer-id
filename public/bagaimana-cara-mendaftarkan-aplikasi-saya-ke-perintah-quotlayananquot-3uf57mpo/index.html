Bagaimana cara mendaftarkan aplikasi saya ke perintah "layanan"?

Di RedHat EL 6, banyak layanan dapat dimulai dan dihentikan pada baris perintah:

service some_service start

dan

service some_service stop

Saya telah mengemas aplikasi server Node.JS dalam RPM. Bagaimana cara mendaftarkan aplikasi saya ke service, untuk kemudahan start dan stop?

Lihat Pedoman RPM untuk Fedora (Fedora adalah upstream untuk RHEL, jadi mereka harus dapat diterapkan dengan beberapa futz untuk versi yang lebih baru). Anda masih harus menulis skrip init, yang lebih merupakan seni hitam. Filch file /etc/init.d untuk sesuatu yang mirip sebagai titik awal, bandingkan dengan yang lain.

Perhatikan bahwa Fedora beralih ke systemd , dan RHEL 7 juga akan menggunakannya. Itu berarti bahwa startup berubah secara radikal (dan penulisan file startup menjadi lebih sederhana). Mungkin Anda harus menginstal Fedora terbaru dan mulai bereksperimen.

Bagaimana cara mengubah font dan warna teks di bawah ikon di desktop?

Saya mencoba menggunakan font dan warna yang berbeda untuk teks pada ikon di desktop. Di bawah Personalize saya telah dapat mengubah jarak ikon (baik horizontal maupun vertikal) dan ukuran teks. Tapi saya tidak bisa mengubah font atau warna teks yang ditampilkan. Font mengklaim telah berubah tetapi tidak dan menu tarik-turun warna dinonaktifkan. Saya menggunakan non-Aero Windows 7 Basic tetapi saya tidak dapat mengubah font atau warnanya.

Saya memiliki Windows 7 Ultimate dan saya mengikuti posting di atas dari PersonalNexus dengan sedikit modifikasi untuk mengubah warna font desktop (dari putih menjadi hitam).

Untuk mengubah warna font ikon desktop Anda:

Klik kanan pada desktop dan pilih Personalize | klik Warna Jendela | Pengaturan Tampilan Lanjut | Pilih 'Desktop' di bawah Item (bukan Ikon) | ubah 'Warna 1' dari Hitam ke Putih untuk warna font ikon desktop hitam.

Dan, warna font desktop Anda akan berubah menjadi Hitam sehingga Anda dapat membaca ikon Anda! Jika Anda ingin warna font desktop putih, buat saja 'desktop' Warna item 1 Hitam, bukan putih!

Semoga ini bisa membantu semua orang!

Anda dapat mengubah fonta teks ikon desktop Anda bahkan saat Anda tidak menggunakan tema Windows 7 Basic.

Untuk melakukannya, klik kanan pada desktop dan pilih Personalize. Klik Window Color di bagian bawah layar dan kemudian Advanced appearance settings... di layar berikutnya. Pilih Icon dari dropdown Item dan pilih font yang Anda suka (untuk melihatnya benar-benar berubah, pilih sesuatu yang tidak biasa seperti Comic Sans ). Saat Anda mengklik tombol Apply, keterangan di bawah ikon di desktop Anda sekarang harus ditampilkan dalam font yang Anda pilih.

Namun perhatikan, warna judul ikon tidak dapat diubah. Ini akan dipilih secara otomatis berdasarkan warna wallpaper desktop Anda, menurut posting ini di Microsoft Answers :

Anda tidak dapat mengubah warna font desktop.

Karena gambar wallpaper desktop memiliki warna yang bervariasi, itu mudah untuk warna font tidak dapat dibaca jika berada di atas warna yang sama sebagai gambar wallpaper. Misalnya font hitam di atas bagian yang gelap gambar latar belakang atau font putih di bagian putih a gambar latar belakang.

Font ikon desktop dinamis dan akan terlihat putih dalam gelap latar belakang dan ubah menjadi putih dengan efek bayangan hitam pada cahaya Latar Belakang.

Akhirnya diselesaikan dengan Tema Aero:

  1. Jalankan sysdm.cpl untuk masuk ke Properti Sistem.
  2. Pilih tab Advanced dan Performance Setting untuk masuk ke jendela Performance Options.
  3. Pilih tombol "Sesuaikan untuk kinerja terbaik" untuk mengosongkan semua kotak centang.
  4. Menggunakan Ikonoid untuk mengubah warna font ikon.

Jika Anda lebih suka untuk tetap memiliki pratinjau sekilas jendela sambil mengarahkan kursor ke ikon bilah tugas, Anda dapat memilih kotak centang "Aktifkan komposisi desktop" dan "Gunakan gaya visual pada jendela dan tombol".

Selamat bersenang-senang!

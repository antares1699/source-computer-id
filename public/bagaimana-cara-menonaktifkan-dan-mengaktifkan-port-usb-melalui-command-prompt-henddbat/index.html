Unduh utilitas devcon dari Microsoft.

Kemudian, gunakan sebagai berikut: Buat daftar semua perangkat USB yang dikenal:

devcon find USB*

Nonaktifkan perangkat USB:

devcon disable "USB\VID_xxxx&PID_yyyy"

Aktifkan perangkat USB:

devcon enable "USB\VID_xxxx&PID_yyyy"

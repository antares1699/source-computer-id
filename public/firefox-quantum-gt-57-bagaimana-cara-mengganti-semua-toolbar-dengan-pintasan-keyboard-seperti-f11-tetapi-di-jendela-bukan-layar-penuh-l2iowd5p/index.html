Firefox Quantum (> = 57) Bagaimana cara mengganti semua toolbar? (dengan pintasan keyboard, seperti F11 tetapi di jendela, bukan layar penuh)

F11 / Layar penuh terkadang tidak praktis *. Saya hanya ingin mengaktifkan toolbar.
Di versi Firefox yang lebih lama, hal ini dimungkinkan dengan addon "Dorando keyconfig":

1. tentukan skrip:

var s = document.getElementById('status-bar');
s.hidden = !s.hidden;
var b = document.getElementById('PersonalToolbar');
b.collapsed = !b.collapsed;
var nb = document.getElementById('nav-bar');
nb.collapsed =!nb.collapsed
var tb = document.getElementById('TabsToolbar');
tb.collapsed =!tb.collapsed

2. memetakannya ke pintasan keyboard melalui antarmuka addon Dorando

Addon ini tidak tersedia lagi di Firefox 57 ke atas.

Berdasarkan jawaban ClairelyClaire.msft (yang hebat!), Berikut adalah beberapa perbaikan (tambahkan ke userChrome.css Anda):

  • Pertahankan agar bilah alat tetap terlihat saat memiliki fokus (sehingga tidak hilang saat Anda mengetik URL).
  • Sembunyikan bilah alat sepenuhnya tetapi tetap menerima peristiwa tetikus, sehingga bilah judul peramban tidak lagi diperlukan (tidak perlu mengaktifkan "tampilkan bilah judul" di Sesuaikan).
  • Tidak perlu menentukan tinggi bilah alat secara manual dalam kode (yang juga lebih kuat jika berubah).
#navigator-toolbox {
    position: relative;
    z-index: 1;
    max-height: 7px;
    margin-top: 0px;
    opacity: 0;
    overflow: hidden;
    transition: max-height 2s ease 1s, opacity 3s step-end;
}

#navigator-toolbox:hover,#navigator-toolbox:focus-within {
    max-height: 100px;  /* must be somewhat larger than actual bar height */
    opacity: 1;
    overflow: inherit;
    transition: max-height 0.3s ease 0s, opacity 0s step-end;
}

Karena semua jawaban di sini bergantung pada file Firefox .css, perlu dicatat yang dimulai pada Firefox 69, userChrome.css dan userContent.css dinonaktifkan secara default "untuk meningkatkan kinerja".

Untuk mengaktifkannya kembali, lakukan hal berikut:

  • Masukkan about:config di bilah alamat Firefox
  • Jika diminta, klik "Tampilkan semua"
  • Masuk di bilah pencarian: toolkit.legacyUserProfileCustomizations.stylesheets
  • Alihkan preferensi. Benar berarti Firefox mendukung file CSS, Salah bahwa itu mengabaikan mereka.

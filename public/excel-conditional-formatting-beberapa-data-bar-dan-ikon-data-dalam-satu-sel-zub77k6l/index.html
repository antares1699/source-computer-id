Excel Conditional Formatting Beberapa Data Bar dan Ikon Data dalam satu sel

Saya menggunakan Excel 2007 di mesin Windows.

Saya mencoba untuk menempatkan satu bilah data dan satu ikon data ke dalam sel di bawah pemformatan bersyarat. Masalahnya adalah saya tidak benar-benar ingin memiliki ikon data atau bilah data untuk sel yang memiliki tanggal di masa depan dan saya hanya ingin memiliki ikon data untuk tanggal setidaknya dalam satu bulan yang lalu.

Inilah yang saya miliki:

Ini yang aku inginkan:

Saya menggunakan fungsi EOMONTH untuk menentukan hari terakhir setiap bulan untuk penghitungan pemformatan bersyarat. Untuk bilah data rumusnya adalah =EOMONTH(Now(), 4) dan =EOMONTH(Now(), -1). Rumus ikon data adalah =EOMONTH(Now(), -1) dan =EOMONTH(Now(), -2).

Apakah ada cara di Excel 2007 untuk menyingkirkan ikon data untuk semua tanggal di masa mendatang dan kehilangan bilah data saat tanggal telah lewat?

Nah, setelah berjam-jam mencari di internet, saya dapat menemukan Blog Office Excel yang memiliki jawaban saya. http://blogs.office.com/b/microsoft-excel/archive/2006/02/24/conditional-formatting-trick-1-multi-coloured-data-bars.aspx

Idenya adalah untuk memberikan rumus untuk setiap format bersyarat untuk memberi tahu sel mana yang akan diformat dan sel mana yang tidak diformat. Tidak ada antarmuka untuk bawaan ini agar unggul jadi kita perlu beralih ke VBA untuk menerapkan rumus.

selection.FormatConditions(1).formula = "enter formula here"  
    -- Quotes are mandatory and the Conditional Formats start with "1"
       at the top of the list   
    -- If you make a mistake selection.FormatConditions(1).formula = "=TRUE"

Saya tidak yakin apakah format ini akan bertahan dari satu sesi ke sesi berikutnya. Saya memutuskan untuk membuka file Excel dan melihat konten XML-nya. Di folder lembar kerja (setelah Anda menambahkan ekstensi .zip ke file), klik pada spreadsheet "Sheet1" dan di bagian bawah adalah informasi untuk format bersyarat. XML di sebelah kiri adalah dari sebelum saya menerapkan rumus format bersyarat dan XML di sebelah kanan adalah setelahnya.

Saya menghabiskan beberapa saat mencoba berbagai hal, tetapi jawabannya adalah menambahkan kondisi di tengah yang membuat mengisi sel dengan warna putih jika kurang dari EOMONTH(Now(), -1). Letakkan di posisi ke-2 seperti yang ditunjukkan dengan Stop jika True dicentang. Terakhir, ganti tanda centang hijau dengan "Tidak Ada Ikon".

EDIT: untuk Excel 2007

Satu-satunya cara yang bisa saya pikirkan di XL 2007 adalah dengan kolom pembantu. Saya memasukkan kolom A dengan rumus ini:

=IF(B3<EOMONTH(NOW(),-1),"X",IF(B3<EOMONTH(NOW(),0),"!",""))

Kemudian kolom A diformat bersyarat untuk font merah jika sel sama dengan "X" dan oranye jika sama dengan "!".

Anda bisa mengutak-atik warna latar belakang dan batas di kedua set sel untuk membuat aliran merah melintas. Kupikir. Itu kesempatan terbaikku!

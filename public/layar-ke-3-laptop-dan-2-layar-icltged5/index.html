Layar ke-3: laptop dan 2 layar

Sekarang desktop saya diperluas menjadi 2 monitor: laptop dan layar terhubung ke laptop melalui hmdi.

Saya memutuskan untuk menambahkan monitor ke-3 dan menghubungkannya melalui d-sub ke laptop.

Windows 7 mendeteksinya pada Control Panel\Appearance and Personalization\Display\Screen Resolution.

Saya memilih monitor ke-3 dan memilih Extend desktop to this dislplay. Klik Apply, Save Changes. Tetapi bidang Multiple displays mengembalikannya ke Disconnect this display. Dan tidak ada apa-apa di monitor ke-3.

Kartu video laptop saya adalah Intel(R) HD Graphics 4000. Model monitor ketiga BenQ FP71G+. Nama model Lapton adalah samsung NP300E5X-A07RU Sistem operasi: windows 7 ultimate x64

Tampaknya itu mungkin dengan kartu video saya , tapi pertanyaannya adalah bagaimana melakukannya?

Dapatkan kartu video eksternal UGA, ternyata port USB 2.0 / 3.0 Anda ke port HDMI atau dvi atau vga yang dapat Anda gunakan untuk menghubungkan monitor ke-3 Anda. Anda perlu menginstal driver yang disertakan dengan kartu uga. Ini solusi murah untuk banyak monitor. Anda dapat menghubungkan hingga 6 monitor dengan kartu UGA. Setiap kartu UGA hanya berharga $ 50.

Pengaturan saya: satu monitor terhubung ke port VGA di laptop saya, monitor ke-2 terhubung ke port HDMI di laptop saya, (laptop saya memiliki port VGA dan HDMI), monitor ke-3 terhubung ke USB melalui kartu UGA eksternal)

Menurut Halaman dukungan tiga monitor HD Graphics 4000 ,

Dalam kebanyakan kasus, tiga tampilan independen didukung di konfigurasi berikut:

  • Jika dua tampilan adalah monitor DisplayPort
  • Jika salah satu tampilan adalah Intel® Wireless Display (Intel® WiDi)
  • Jika layar internal adalah Embedded DisplayPort (eDP)

Karena Anda tidak memiliki output DisplayPort di mesin Anda, pengaturan yang Anda inginkan hanya berfungsi jika monitor internal Anda terhubung melalui eDP, bukan LVDS.

Untuk memeriksa apakah layar internal komputer Anda menggunakan tertanam Jenis konektor DisplayPort (eDP) atau LVDS: Dari desktop Windows, tekan kombinasi tombol Ctrl + Alt + F12. Jika diminta pilih Advanced Mode mode aplikasi. Pilih Opsi dan Dukungan, ubah Sistem Informasi ke Tampilan Bawaan. Periksa bidang Jenis Konektor.

Jika ternyata LVDS, sayangnya pengaturan ini tidak kompatibel dengan perangkat keras Anda.

Perhatikan bahwa, sejauh yang saya tahu, sebagian besar laptop saat ini tidak menggunakan eDP karena ini adalah standar baru (kecuali beberapa laptop 3D), jadi umumnya tidak mungkin menjalankan 3 monitor dari GPU HD 4000 kecuali Anda laptop memiliki DisplayPort. (Atau jika Anda menjalankan satu monitor dari WiDi.)

Untuk mengetahui apakah monitor laptop Intel Anda eDP atau LVDS, tekan CTRL + ALT + F12 , pilih "opsi dan dukungan", "Pusat informasi" di tengah atas layar akan menjadi tab drop-down yang bertuliskan "Informasi Sistem". Kemudian, di sebelahnya ada tombol simpan. Pilih tab drop-down "Sustem Information" lalu pilih "Built-in Display". Di bawah general information akan muncul jenis konektor. Sayangnya saya adalah LVDS (dan saya tidak punya laptop "murah").

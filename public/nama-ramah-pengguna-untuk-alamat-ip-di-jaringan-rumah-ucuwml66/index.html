Gunakan file host. Wikipedia memiliki artikel .

File host berisi baris teks terdiri dari satu alamat IP dan satu atau lebih banyak nama host, setiap bidang dipisahkan dengan spasi putih (kosong atau tabulasi karakter). Baris komentar mungkin termasuk; mereka ditunjukkan dengan hash karakter (#) di posisi pertama garis seperti itu. Sebagai contoh,

#This is an example of the hosts file
127.0.0.1  localhost loopback

mungkin entri untuk loopback alamat dan nama host, tipikal konten default dari file host.

Itu datang dengan dokumentasi, tetapi Anda mungkin juga membutuhkan buku tentang bind (mis. "DNS dan BIND").

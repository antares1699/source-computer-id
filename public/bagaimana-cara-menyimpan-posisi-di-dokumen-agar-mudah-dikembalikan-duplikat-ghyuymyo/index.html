Bagaimana cara menyimpan posisi di dokumen agar mudah dikembalikan? [duplikat]

Kemungkinan Duplikat:
Apakah ada cara untuk menandai file kata atau file kata yang Anda tinggalkan?
Apakah ada cara untuk memasukkan jangkar / penanda internal ke dalam dokumen MS Word dan kemudian mereferensikannya?

Bagaimana cara menyimpan posisi di dokumen sehingga saya dapat kembali ke posisi yang tepat dengan mudah nanti?

Misalnya saat saya ctrl -klik pada tautan internal di dokumen Word, itu membawa saya ke target tautan itu, kehilangan posisi semula. Atau ketika saya mencari sesuatu yang indah F3 dari posisi tertentu, itu melompat ke kemunculan berikutnya dari istilah tersebut. Setiap kali saya merasa agak sakit untuk kembali ke posisi di dokumen tempat saya memulai. Tentu saja ini bukan masalah untuk dokumen kecil, tetapi untuk ratusan halaman, memindai kembali ke halaman yang diinginkan dapat memakan waktu lama. Sekarang saya menyadari fakta bahwa saya dapat memeriksa / memasukkan nomor halaman sebenarnya di Word, tetapi saya mencari solusi umum untuk dokumen lain dan solusi yang dapat diaktifkan dengan penekanan tombol sederhana.

Apakah ada metode umum untuk menangani gangguan seperti itu misalnya, menyimpan posisi tertentu dalam dokumen dengan penekanan tombol yang dapat saya gunakan nanti untuk mengembalikan posisi itu? Mungkin banyak posisi disimpan dengan kunci yang berbeda? Apakah ada metode umum untuk menangani .doc, .pdf, .txt, atau file lainnya, bahkan mungkin halaman web yang ditampilkan oleh browser? Jika tidak ada cara umum untuk melakukan ini, mungkin hanya di Microsoft Word?

Alt + Panah kiri - Kembali setelah mengklik hyperlink. ("Web Toolbar" telah hilang di Word 2007.)

Alt + Panah kanan - Batalkan Alt + Panah kiri .

Alt + Ctrl + Z - Kembali ke lokasi yang sebelumnya diedit dalam dokumen (hingga 4 tempat)

Bergeser + F5 - Pergi ke perubahan atau revisi terakhir. Juga berfungsi setelah membuka dokumen

Anda juga dapat menyimpan posisi di dokumen dengan file penanda buku :

Ctrl + Bergeser + F5 - Setel, buka dan edit bookmark

Ctrl + G atau F5 - Pergi ke halaman, bookmark, catatan kaki, tabel, komentar, grafik, atau lokasi lain

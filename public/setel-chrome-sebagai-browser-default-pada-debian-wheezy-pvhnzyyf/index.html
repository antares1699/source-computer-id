Aplikasi default ditentukan dalam file ~/.local/share/applications/mimeapps.list. Buka file di editor teks favorit Anda dan ubah

x-scheme-handler/http=firefox.desktop
x-scheme-handler/https=firefox.desktop

untuk

x-scheme-handler/http=chrome.desktop
x-scheme-handler/https=chrome.desktop

Itu akan bekerja selama Anda memiliki chrome.desktop di /usr/share/applications/.

MEMPERBARUI:

Jika Anda tidak memiliki file chrome.desktop di /usr/share/applications/ buat saja dengan baris berikut:

Name=Google Chrome
Terminal=false
Icon=google-chrome
Type=Application
Categories=Network;WebBrowser;
MimeType=text/html;text/xml;application/xhtml_xml;x-scheme-handler/http;x-scheme-handler/https;x-scheme-handler/ftp;
X-Ayatana-Desktop-Shortcuts=NewWindow;NewIncognito
Exec=/opt/google/chrome/google-chrome %U

Ubah /opt/google/chrome/google-chrome menjadi apa pun yang dikembalikan oleh which google-chrome.

Standarnya adalah /usr/share/applications/mimeinfo.cache, dia dibaca di /usr/share/applications/*.desktop

grep -R x-scheme-handler/http /usr/share/applications/*.desktop

/usr/share/applications/chromium.desktop:MimeType=text/html;text/xml;application/xhtml+xml;text/mml;x-scheme-handler/http;x-scheme-handler/https; /usr/share/applications/firefox.desktop:MimeType=text/html;text/xml;application/xhtml+xml;application/vnd.mozilla.xul+xml;text/mml;x-scheme-handler/http;x -pengendali skema / https;

Pendekatan saya adalah mengubah ~ / .local / share / application / mimeinfo.cache pengguna akan x-scheme-handler / http (protokol http) 、 x-scheme-handler / https (protokol https) yang ditentukan oleh chrome.desktop terbuka.

simpan kode ini ke foo; dan sh ./foo

#!/bin/bash
cd ~/.local/share/applications/
sed -i '/http=\|https=/d' mimeinfo.cache
cat >>mimeinfo.cache<<EOF
x-scheme-handler/http=chrome.desktop;
x-scheme-handler/https=chrome.desktop;
EOF

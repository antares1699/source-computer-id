Untuk menonaktifkan kotak dialog ini untuk SEMUA file .exe, Anda dapat menambahkan kunci berikut ke registri Anda:

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Associations]
"LowRiskFileTypes"=".exe

Catatan: ini tidak disarankan, karena membuka komputer Anda untuk risiko lain.

Selain daripada itu, ClickOff adalah teman Anda (pendekatan yang lebih umum untuk gangguan seperti itu :)

ClickOff telah dihapus dari file kebutuhan pribadi - seperti kebanyakan program sini. Pertanyaan seperti "Anda yakin kamu ingin keluar dari program? "membuatku menjadi gila setelah melihat mereka a belasan kali.

ClickOff dikembangkan menjadi Swiss Pisau tentara untuk menangani jendela, bukan hanya pemblokir popup sederhana.

Berikut beberapa fitur utama:

  • Secara otomatis mengklik tombol pada pesan kesalahan yang muncul setiap saat.
  • Antarmuka pengguna yang intuitif.
  • Menutup popup browser
  • Meminimalkan atau memaksimalkan jendela
  • Waktu tunda dapat diatur untuk memungkinkan intervensi pengguna
  • Tombol tunggal dapat dinonaktifkan sementara
  • Klik salah dihindari dengan memasukkan teks pesan ke dalam proses pendeteksian.
  • Untuk menutup jendela yang berisi kata-kata tertentu, karakter pengganti dapat digunakan
  • Kecepatan pemindaian dapat diatur
  • Dapat dinonaktifkan untuk waktu yang singkat atau permanen tanpa harus keluar.
  • Memiliki kebutuhan sumber daya sistem yang sangat rendah, membutuhkan waktu sekitar 1 detik. untuk memuat dan tidak memperlambat sistem saat berjalan.

ClickOff adalah freeware.

Saat saya mendownload EXE dari internet, saya akan ditanya seperti yang Anda tunjukkan. Namun, jika saya meletakkan EXE di desktop (misalnya) dan menjalankannya, ada kotak centang yang memungkinkan saya untuk mengatakan "jangan minta saya untuk ini lagi". Anda mungkin tidak mendapatkan kotak centang tambahan jika Anda menjalankan program dari direktori file program. Dalam hal ini, saya yakin Anda dapat mematikan UAC, reboot, jalankan program, setel kotak centang, dan kemudian hidupkan kembali UAC.

Jika EXE diekstrak dari file ZIP yang diunduh dari Internet, lakukan tidak gunakan folder zip (antarmuka File Explorer) tetapi alat pihak ketiga seperti 7-Zip. (Jika Anda sudah mengekstrak EXE dan masih memiliki file ZIP, Anda cukup menghapus EXE dan mengekstraknya lagi.)

Folder Zipped akan mempertahankan Zone.Identifier ADS (yang menyimpan informasi ini), 7-Zip (mulai 16.04) tidak.

Sudahkah Anda mempertimbangkan untuk menggunakan IDE? Saya menemukan bahwa untuk pengembangan menggunakan IDE eclipse dengan pyDev sangat bagus. Entah itu atau pyscripter cukup terkenal - http://code.google.com/p/pyscripter/downloads/list

Jika Anda ingin melanjutkan seperti sebelumnya, coba instal ulang python, setiap versi dilengkapi bundel dengan IDLE Python yang saya harapkan adalah apa yang Anda gunakan. - http://www.python.org/getit/

Maka itu harus kembali. Jika gagal:

  1. Cari di regedit untuk kunci: HKEY_CLASSES_ROOT * \ shell
  2. Klik kanan pada "shell" dan pilih 'buat kunci baru'
  3. Beri nama "Edit dengan IDLE"
  4. Buat kunci di bawahnya dan beri nama "perintah".
  5. Klik dua kali pada nilai (Default) yang akan Anda temukan di panel jendela sebelah kanan, lalu ketik berikut ini: "python.exe% 1" (atau jika Anda tidak memilikinya di PATH, masukkan python absolut direktori, misalnya: C: \ Python27 \ bin \ python.exe)

Sepertinya Anda menggunakan Windows. Anda perlu mengasosiasikan IDLE sebagai editor untuk file .py. Saya tidak tahu tentang Windows 7 tetapi dengan XP:

  • Buka Windows Explorer, klik menu 'Tools', pilih 'Folder Options'
  • Pilih tab 'Jenis file'.
  • Gulir ke bawah dan pilih "py Python File" dalam daftar jenis file Terdaftar.
  • Klik tombol Advanced untuk membuka kotak dialog Edit File Type.
  • Jika Anda melihat "Edit dengan IDLE", klik tombol Edit dan pastikan kotak "Aplikasi yang digunakan" bertuliskan "C: \ PythonXX \ pythonw.exe" "C: \ PythonXX \ Lib \ idlelib \ idle.pyw" - e "% 1" (menggantikan XX dengan versi Python Anda.
  • Jika Anda tidak melihat "Edit dengan IDLE", klik tombol New, masukkan "Edit dengan IDLE" di kotak Tindakan dan string "Aplikasi yang digunakan" yang sama seperti di atas, lalu centang kotak DDE.

Klik kanan pada file tersebut, pilih Open with, pilih pilih program default, dan pilih python.

Peringatan, ini akan mencoba menjalankan skrip apa pun yang Anda coba buka dari command prompt (tetapi Anda bisa x ini keluar).

Setelah itu, komputer akan mengenali file tersebut sebagai file python dan memberi Anda opsi "Edit with Idle" lagi.

Program yang baru saja Anda instal atau luncurkan mungkin telah mengubah asosiasi file. Inilah cara saya memperbaiki milik saya di Windows 7:

  1. Di Windows Explorer, klik kanan pada file .py Anda dan pilih "Properti".
  2. Di tab "Umum" di bawah "Jenis file:" klik Tombol "Ubah ...".
  3. Di bawah "Program yang Direkomendasikan" pilih "Python Peluncur untuk Windows (Konsol) ".

Simpan teks di bawah ini ke dalam file .reg menggunakan notepad dan klik dua kali di atasnya.

Windows Registry Editor Versi 5.00

[HKEY_CLASSES_ROOT \ Python.File \ shell \ Edit dengan IDLE \ perintah] @ = "\" C: \ Python34 \ pythonw.exe \ "\" C: \ Python34 \ Lib \ idlelib \ idle.pyw \ "-e \"% 1 \ ""

[HKEY_CLASSES_ROOT \ Python.NoConFile \ shell \ Edit dengan perintah IDLE \] @ = "\" C: \ Python34 \ pythonw.exe \ "\" C: \ Python34 \ Lib \ idlelib \ idle.pyw \ "-e \"% 1 \ ""

Jika Anda menggunakan Python27, ubah "34" di atas menjadi "27". Hidupkan Kembali komputer Anda.

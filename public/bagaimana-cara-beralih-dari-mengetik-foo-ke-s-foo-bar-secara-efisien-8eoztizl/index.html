bagaimana cara beralih dari mengetik `/ foo` ke`:% s / foo / bar` secara efisien

Pada vim, saya mengetik /foo Memasukkan dan menemukan beberapa kecocokan.

Sekarang saya ingin mengganti semua contoh foo dengan bar (yaitu :%s/foo/bar).

Bisakah ini dilakukan tanpa mengetik ulang foo? (misalnya, ke dalam baris perintah setelah :)

Ya, Anda dapat melakukannya dengan :%s//bar.

Penjelasan: pada perintah substitusi ketika pola pencarian kosong, Vim menggunakan pola pencarian terakhir.

Alternatifnya, Anda dapat secara eksplisit menempelkan pola pencarian terakhir, yang disimpan dalam register khusus. Anda dapat dengan cepat memasukkan isinya dengan mengetik Ctrl-R + / dalam mode sisip atau perintah.

Jadi masuk ke mode perintah, ketik %s/, lalu Ctrl-R + / untuk menempelkan pola terakhir, dan kemudian sisa /bar.

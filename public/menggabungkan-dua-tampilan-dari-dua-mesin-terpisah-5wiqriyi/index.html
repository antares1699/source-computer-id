menggabungkan dua tampilan dari dua mesin terpisah

Saya memiliki dua desktop masing-masing dengan output monitor vga tunggal. Saya tertarik untuk memiliki pengaturan monitor yang diperluas, di mana saya dapat memperluas desktop salah satu komputer saya dan dengan begitu saya juga dapat bekerja di komputer lain menggunakan desktop jarak jauh. Tahu bagaimana saya bisa melakukan itu? Saya menjalankan Windows 7 di satu dan Win Server 2k8 di lain.

Melihat Sinergi .

Synergy adalah Perangkat Lunak Bebas dan Sumber Terbuka yang memungkinkan Anda berbagi dengan mudah mouse dan keyboard Anda di antara beberapa komputer, di mana masing-masing komputer memiliki layarnya sendiri. Tidak diperlukan perangkat keras khusus, Anda semua kebutuhan adalah jaringan area lokal. Synergy didukung di Windows, Mac OS X dan Linux. Mengarahkan mouse dan keyboard semudah menggerakkan mouse dari tepi layar Anda.

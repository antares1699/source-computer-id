Cara melihat / mengedit nilai registri x86 di windows 7 x64

Saya memiliki aplikasi yang dikompilasi dengan prosesor x86 yang mendaftarkan sejumlah nilai di registri mesin target. Saya telah menginstal aplikasi ini di windows 7 x64.
Sekarang ketika saya mencoba untuk melihat kunci terdaftar menggunakan regedit saya perhatikan mereka tidak ada di sana. Pencarian di sekitar saya menemukan bahwa ada dua versi registri pada windows 7 x64. Satu untuk aplikasi x86, dan satu lagi untuk aplikasi x64. Rupanya dalam situasi ini regedit menunjukkan registri x64.
Bagaimana cara melihat / mengedit nilai registri x86?

Jika Anda melihat registri dari aplikasi 64-bit, Anda dapat melihat nilai untuk aplikasi 32-bit di bawah kunci Wow6432Node (mis., HKEY_LOCAL_MACHINE\Software\Wow6432Node berisi pohon spesifik 32-bit yang terlihat sebagai HKEY_LOCAL_MACHINE\Software dari aplikasi 32-bit). Lihat Dokumentasi Microsoft untuk informasi lebih detail.

Baru-baru ini di Windows Explorer, sebuah Drive "Local Disk (F :)" muncul dan menunjukkan 34,2MB bebas dari 449MB, tetapi ketika saya klik dua kali padanya, Explorer tidak menunjukkan file pada drive ini. Saya tidak tahu apa ini dan saya belum memasang drive USB apa pun belakangan ini, dan saya tidak memiliki apa pun yang dicolokkan sekarang. Saya memindai virus dan tidak apa-apa. Windows juga terus memberi tahu saya bahwa ruang disk drive ini hampir habis.

Apakah ada yang tahu apa drive ini?

Ini adalah masalah yang diketahui dengan pembaruan Windows 10 April. Dengan pembaruan ini, Windows terkadang secara keliru menetapkan huruf drive ke salah satu partisi OEM atau pemulihan yang biasanya tersembunyi dan kemudian mengeluh tentang kurangnya ruang kosong di partisi itu. Solusinya cukup dengan menghapus huruf drive itu lagi.

Namun terkadang ini tidak begitu mudah karena aplikasi Manajemen Disk standar menolak untuk menyentuh partisi. Gejala yang muncul adalah menu konteks untuk partisi di Manajemen Disk tidak menunjukkan apa-apa selain "Bantuan". Dalam hal ini Anda harus menjalankan alat baris perintah diskpart dari prompt perintah dengan hak administrator dan kemudian menggunakan perintah:

list volume
select volume <number of offending volume>
remove letter=<offending drive letter>
exit

untuk mengidentifikasi dan menghapus huruf drive yang melanggar.

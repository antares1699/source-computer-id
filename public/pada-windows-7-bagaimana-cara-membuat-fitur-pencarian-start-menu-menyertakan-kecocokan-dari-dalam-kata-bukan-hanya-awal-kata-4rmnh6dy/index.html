Komputer

Pada Windows 7, bagaimana cara membuat fitur pencarian Start Menu menyertakan kecocokan dari dalam kata, bukan hanya awal kata?

Saya memiliki program terinstal yang disebut WinSCP. Saat saya menekan tombol Windows dan mengetik "SCP", saya mendapatkan "Tidak ada item yang cocok dengan pencarian Anda." Apakah ada opsi konfigurasi yang dapat saya atur di suatu tempat, sehingga item ini akan ditemukan?

Saya tidak mencari solusi khusus untuk program khusus ini, tetapi sesuatu yang umum, sehingga jika ada program bernama XYZ, saya dapat menemukannya melalui pencarian Start Menu dengan memasukkan YZ.

EDIT UNTUK MENAMBAH : Saya mencari jenis perubahan konfigurasi "set-it and forget-it", sehingga pencarian dalam kata selalu terjadi, secara otomatis. Saya tidak ingin mengetik * sebelum setiap kueri. Rupanya ini tidak jelas dari apa yang saya tulis di atas.

Apakah `ip` menggantikan` ifconfig`

Seorang teman saya mengatakan itu ip seharusnya menjadi penerus ifconfig dan iwconfig - namun saya tidak menemukan artikel Wikipedia tentangnya dan tidak pernah melihat ada orang yang menyebutkan atau merekomendasikannya secara online meskipun dari tahun 2002 menurut halaman manualnya. Itu menandakan bahwa itu tidak banyak digunakan.

Nama ip sulit untuk Google jadi saya memutuskan untuk bertanya di sini.

Perintah ip adalah bagian dari kumpulan utilitas iproute2.

Menurut Linux Foundation perintah ifconfig seharusnya tidak digunakan lagi, tetapi kebanyakan distro masih menyertakannya. iproute2 juga berisi perintah tc (traffic control). Lihat informasi penggunaan dengan menggunakan perintah $ man ip.

Ada diskusi tentang ini serverfault tahun lalu.

IP adalah bagian baru dari perintah yang dimaksudkan untuk menggantikan ifconfig, beberapa contohnya:

ip link show    List network interfaces
ip link set dev eth0 name wan   Rename interface eth0 to wan
ip link set dev eth0 up Bring interface eth0 up (or down)
ip addr show    List addresses for interfaces
ip addr add 1.2.3.4/24 brd + dev eth0   Add (or del) ip and mask (255.255.255.0)
ip route show   List routing table
ip route add default via 1.2.3.254  Set default gateway to 1.2.3.254

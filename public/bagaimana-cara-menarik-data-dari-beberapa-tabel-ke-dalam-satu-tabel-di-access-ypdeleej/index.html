Bagaimana cara menarik data dari beberapa tabel ke dalam satu tabel di Access?

Saya memiliki beberapa tabel yang berisi tugas sesuai dengan kategorinya. Setiap tabel mewakili kategori itu. Yang perlu saya lakukan adalah membuat tabel 'Master' yang kemudian dapat menarik data dari SEMUA tabel lain dalam kotak dropdown. Sebagai contoh:

seperti ini, namun ini hanya satu tabel yang dapat saya capai melalui penggunaan wizard 'pencarian dan hubungan'. Bagaimana cara mengaktifkan ini untuk data dari beberapa tabel?

Sedikit Lebih Banyak Klarifikasi:

Setiap subtabel memiliki 'ID Tugas' unik:

Kategori 1 = 1000 seri

Kategori 2 = 2000 seri

Kategori 3 = 3000 seri

....

Saya membutuhkan kemampuan untuk membuat ID unik untuk masing-masing kategori tersebut yang kemudian diperbarui, dan dapat dipilih dari tabel yang berbeda.

Sekali lagi, jawaban untuk masalah Anda adalah pertanyaan! Dan berikut ini yang Anda butuhkan:

SELECT Name FROM FirstTable
UNION
SELECT Name FROM SecondTable
UNION
SELECT Name FROM ThirdTable

dll.

Ini akan memberi Anda sekumpulan rekaman dari kolom name yang dikumpulkan oleh kueri dari FirstTable, SecondTable dan ThirdTable.

Anda kemudian dapat menggunakan panduan pencarian dalam desain tabel untuk memilih kueri setelah Anda menyimpannya untuk mengisi menu drop down Anda.

Anda bisa menjalankan kueri di Access dengan melakukan hal berikut:

  1. Buka tab "Buat" dan klik "Desain Kueri"

  2. Tutup saja jendela yang muncul yang meminta Anda untuk memilih tabel, kami tidak membutuhkannya.

  3. Pergi ke tab "Design" dan klik tombol dengan panah sampai Anda mendapatkan layar input tekstual. (Secara default, dikatakan SELECT;).

  4. Hapus teks default dan tempel kueri di atas.

  5. Klik "Jalankan".

  6. Simpan dengan menekan Ctrl + S sedangkan tab Query adalah tab aktif.

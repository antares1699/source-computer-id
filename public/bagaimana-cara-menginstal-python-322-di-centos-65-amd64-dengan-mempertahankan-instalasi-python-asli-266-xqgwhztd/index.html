Instal Python 3.2.2 (CentOS 6.5 AMD64):

Untuk mengkompilasi Python, Anda harus terlebih dahulu menginstal alat pengembangan dan beberapa lib ekstra. Lib ekstra tidak benar-benar diperlukan untuk mengkompilasi Python tetapi tanpa mereka penerjemah Python baru Anda akan sangat tidak berguna

sudo yum groupinstall "Development tools"
sudo yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel

Berikut adalah perintah untuk mengunduh, mengkompilasi dan menginstal Python

cd /usr/local/src
sudo wget http://python.org/ftp/python/3.2.2/Python-3.2.2.tar.xz --no-check-certificate
sudo tar xf Python-3.2.2.tar.xz
cd Python-3.2.2
sudo ./configure --prefix=/usr/local --enable-shared LDFLAGS="-Wl,-rpath /usr/local/lib"
sudo make && sudo make altinstall

Setelah menjalankan perintah di atas, interpreter Python yang baru Anda instal akan tersedia sebagai /usr/local/bin/python3.2

/usr/local/bin/python3.2

Unduh dan instal Setuptools + pip

cd /usr/local/src
sudo wget https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py
sudo /usr/local/bin/python3.2 ez_setup.py
sudo /usr/local/bin/easy_install-3.2 pip

Buat lingkungan Python 3.2 Anda yang terisolasi

sudo /usr/local/bin/pip3.2 install virtualenv
sudo /usr/local/bin/virtualenv /usr/local/virtenv3.2
cd /usr/local/virtenv3.2
source bin/activate
python --version # (To check version)

Skrip <SCRIPT_NAME> rusak: akhir komentar LSB tidak ada

Saya mendapatkan kesalahan ini saat menjalankan:

  • insserv -r udo-startup.sh
    insserv: Script udo-startup.sh rusak: akhir komentar LSB tidak ada.
    insserv: keluar sekarang!

Isi dari udo-startup.sh adalah ini:

#!/bin/bash
### BEGIN INIT INFO
# Provides:          udo-startup.sh
# Required-Start:    $local_fs $remote_fs $network $syslog
# Required-Stop:     $local_fs $remote_fs $network $syslog
# Default-Start:     2 3 4 5
# Default-Stop:      0 1 6 
# Short-Description: -
# Description:       -
### END INIT INF

ID=$(xinput list | grep -i touchpad | sed '/TouchPad/s/^.*id=\([0-9]*\).*$/\1/')
xinput set-prop $ID "Device Enabled" 0

exit 0

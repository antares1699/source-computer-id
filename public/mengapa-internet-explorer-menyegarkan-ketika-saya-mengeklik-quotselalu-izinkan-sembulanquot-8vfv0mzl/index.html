Mengapa Internet Explorer selalu menyegarkan halaman saat saya mengklik "selalu izinkan popup"?

Saat Internet Explorer memblokir sembulan dari situs web tertentu, ia menanyakan saya apakah saya ingin mengizinkan sembulan sementara, atau selalu mengizinkannya dari situs ini.

Itu selalu menyegarkan sendiri tidak peduli yang mana yang saya klik.

Bagaimana cara mencegah hal ini terjadi?

Tidak boleh. Dan bahkan jika Anda bisa, Anda seharusnya tidak melakukannya.

Setelah popup diblokir, itu diblokir. Seluruh halaman harus dimuat ulang untuk memastikan bahwa situs akan berfungsi dengan baik.

Contoh:

  1. URL popup ditautkan ke beberapa variabel. Variabel ini mungkin telah berubah.

  2. Bagian dari kode hanya dijalankan jika popup bisa dibuka.

EDIT: Jika link membuka popup, tekan Ctrl saat mengklik link tersebut akan membuka popup tanpa perlu menyegarkan halaman.

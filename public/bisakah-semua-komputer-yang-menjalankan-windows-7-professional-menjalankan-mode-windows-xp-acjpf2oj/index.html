Komputer

Bisakah semua komputer yang menjalankan Windows 7 Professional menjalankan mode Windows XP?

Saya sedikit bingung tentang apa itu mode XP di Windows 7 (Professional). Mode XP tampaknya menjadi mode di mana Anda dapat menjalankan aplikasi yang kompatibel dengan XP yang sebaliknya tidak kompatibel dengan Windows 7. Web tampaknya mengatakan bahwa mode XP adalah add-on yang dapat diunduh yang, secara teori, berfungsi dengan versi Windows 7 apa pun - asalkan Anda memiliki prosesor yang mendukung visualisasi perangkat keras.

Apa yang saya tidak mengerti ketika saya memberi harga pada berbagai laptop Windows 7 Professional (pertama saya) adalah mengapa beberapa model tampaknya datang dengan mode XP, sedangkan yang lain tidak. Misalnya, lini XPS laptop Dell tampaknya tidak datang dengan mode XP (atau kompatibel dengan?) - lihat, misalnya, halaman ini di situs Dell . Jika Anda mengkonfigurasi salah satu dari mesin ini, tidak disebutkan di mana pun saya dapat melihat mode XP. Di sisi lain, garis Latitude dari laptop Dell tidak sebutkan mode XP - lihat, misalnya, halaman ini . Anda dapat memilih untuk menyertakan mode XP, dan Anda bahkan memiliki pilihan OS 32 atau 64-bit.

Apakah ada alasan untuk perbedaan ini? Prosesor dan kartu video yang tersedia untuk sistem ini tampaknya sangat sebanding. Apakah menurut Anda akan bijaksana untuk memilih sistem dengan mode XP yang telah diinstal sebelumnya, atau apakah menurut Anda menginstal mode XP sendiri, dari web, mungkin sulit karena kemungkinan ketidaksesuaian dengan prosesor dan / atau kartu video?

Terima kasih atas waktunya!

3 persyaratan dasar:

  • Mode XP hanya akan diinstal dan dijalankan Windows 7 Professional atau lebih tinggi .
  • Itu prosesor harus mendukung virtualisasi perangkat keras (catatan: ini bukan lagi persyaratan )
  • Setidaknya Memori terpasang 2 GB .

Secara pribadi saya tidak akan membayar untuk memilikinya sudah diinstal sebelumnya pada PC baru.

Nah, mode Windows XP dimaksudkan untuk pengguna bisnis untuk menjalankan aplikasi lama di bawah Windows 7. Ini kurang uesful untuk pengguna rumahan dan game karena hal tersebut menggunakan protokol desktop jarak jauh untuk terhubung ke VM dan tidak dapat merender grafik 3D atau memutar video secara berguna.

Jadi wajar saja jika garis Latitude berorientasi bisnis untuk mempromosikannya dan bahkan sudah diinstal sebelumnya pada gambar sistem mereka. Sebagian besar pengguna XPS mungkin tidak akan pernah menggunakannya, jadi mengapa membuang disk dan ruang iklan?

Selama Anda memiliki lisensi Windows 7 Pro / Enterprise / Ultimate. Mode XP hanyalah unduhan 400MB.

Mengapa torrent memperlambat internet saya? [duplikat]

Kemungkinan Duplikat:
Bagaimana cara menghindari bit torrent untuk menurunkan kinerja koneksi saya tanpa membatasi penggunaan bandwidth?

Saya bukan "torrenter" biasa, tetapi saya menggunakan µTorrent sesekali untuk mengunduh file besar dengan cepat (misalnya CD Ubuntu). Saat µTorrent berjalan, internet saya sepertinya melambat hingga merangkak. Saya mendapat penundaan sekitar setengah detik dengan permainan, memuat halaman web membutuhkan waktu lebih lama, dan percakapan Skype kadang-kadang terputus. Kenapa ini? Ini bukan masalah besar karena saya tidak menjalankannya 24/7, tetapi saya hanya ingin tahu mengapa saya bisa mendapatkan unduhan super cepat di µTorrent, namun aplikasi berbasis internet lainnya memiliki kinerja yang signifikan memukul.

Juga, jika ini mungkin sesuatu yang spesifik untuk router saya atau koneksi internet saya, saya ingin mencari solusi atau solusi.

Protokol torrent sangat efisien dalam memaksimalkan koneksi Anda, sehingga menggunakan semua sumber daya yang tersedia untuk menyelesaikan pekerjaan lebih cepat.

Selain hanya menggunakan semua bandwidth yang tersedia, ini dapat memperlambat koneksi Anda hanya dengan membuka koneksi. Biasanya merupakan ide yang baik untuk membatasi jumlah total koneksi / koneksi per torrent, karena ini akan membantu agar koneksi Anda tidak jenuh.

Ini juga dapat membantu mengaktifkan antrian torrent, sehingga jika Anda menambahkan 100 torrent, tidak semuanya mencoba bersaing untuk bandwidth Anda dan masing-masing berjalan sama lambatnya. Mungkin sesuatu seperti <10 torrent aktif adalah praktik yang baik.

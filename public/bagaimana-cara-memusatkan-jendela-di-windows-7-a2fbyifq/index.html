Sizer akan melakukannya untuk Anda: http://www.brianapps.net/sizer.html

EDIT: Berikut beberapa petunjuknya:

  1. Instal dan Jalankan.
  2. Sizer menambahkan ikon ke Notification Area / System Tray. Klik dua kali.
  3. Jendela Sizer Configuration akan muncul.
  4. Klik tombol Add.
  5. Item berukuran 640x480 lainnya ditambahkan ke bagian bawah daftar. Klik di atasnya.
  6. Ubah Deskripsi menjadi "Center". Pilih "Jangan ubah ukuran jendela" dan Pindahkan ke: "Tengah".
  7. Klik OK.
  8. Buka jendela Explorer. Klik kanan tombol Maksimalkan. Klik Tengah.

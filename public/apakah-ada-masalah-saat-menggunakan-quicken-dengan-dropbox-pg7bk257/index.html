Saya ingin mulai menggunakan Quicken 2011 dan berpikir akan menjadi ide yang bagus untuk menggunakannya dengan Dropbox sehingga istri saya dapat mengakses file dari komputer lain dan kami juga akan selalu memiliki cadangan online. Saya tidak peduli tentang penggunaan simultan (toh hanya satu dari kami yang akan membayar tagihan pada waktu itu). Apakah ada yang punya pengalaman dengan ini?

Ada cara untuk menggunakan dropbox dengan Quicken. Saya telah melakukannya selama sebulan atau lebih sekarang dan saya telah menginstalnya di empat komputer. Saya ingin menemukan cara untuk mengakses data saya yang lebih cepat di mana pun saya berada. Berikut cara melakukannya. Di komputer pertama Anda, tempatkan semua file quicken Anda di folder dropbox. Biarkan tersinkron ke cloud (akun dropbox Anda). Tutup quicken di mesin itu.

Kemudian instal dropbox di komputer berikutnya dan biarkan tersinkron. Kemudian instal quicken di komputer itu dan arahkan ke folder dropbox untuk file data. Anda dapat melanjutkan ini pada sejumlah komputer yang tidak terbatas jika Anda mau. Kunci untuk diingat - Anda hanya dapat menjalankan Quicken di satu mesin dalam satu waktu. Hal berikutnya yang perlu diingat untuk membuat ini berfungsi adalah sebelum Anda membuka Quicken (di komputer mana pun), Anda harus menjeda sinkronisasi. Caranya dengan "mengklik kanan" pada ikon dropbox di tray.

Saat Anda menutup Quicken, ingatlah untuk melanjutkan sinkronisasi. Proses ini akan memungkinkan Quicken menulis ke file data lokalnya tanpa interupsi apa pun dari Dropbox, dan kapan pun Anda selesai dan melanjutkan sinkronisasi, itu akan memperbarui semua komputer lain. Ini bekerja dengan baik untuk saya. Ingat saja, satu orang dalam satu waktu di Quicken, jeda sinkronisasi sebelum Anda membuka Quicken, dan lanjutkan sinkronisasi setelah Anda menutupnya.

Saya menyukai Dropbox. Saya suka Quicken untuk Mac 2007. Saya tidak suka mereka berdua. Semuanya berfungsi dengan baik selama 3 bulan tetapi hari ini Quicken macet ketika saya mencoba untuk merekonsiliasi akun saya. File data saya rusak dari Dropbox.

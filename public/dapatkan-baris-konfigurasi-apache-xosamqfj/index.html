Dapatkan baris konfigurasi Apache

Apakah ada cara untuk mendapatkan baris konfigurasi yang digunakan untuk membangun instalasi Apache dari sumber?

Saya mengkompilasi Apache 2.2.14 beberapa waktu yang lalu di Ubuntu dan saya tidak dapat mengingat baris ./configure yang saya gunakan untuk membuatnya. Apakah ada cara saya dapat mengambilnya kembali?

Jika direktori sumber yang Anda kompilasi tersedia, cari file config.nice, yang akan berisi daftar konfigurasi yang bagus.

#! /bin/sh
#
# Created by configure

"./configure" \
"--prefix=/opt/apache" \
"--enable-rewrite" \
"--enable-so" \
"--enable-mime-magic" \
"--enable-cgi" \
"--enable-cache" \
"--enable-ssl" \
"--enable-proxy" \
"--enable-proxy-http" \
"--enable-proxy-balancer" \
"$@"

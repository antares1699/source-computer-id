mpg123 dan mpg321 adalah pemutar MP3 baris perintah. Anda dapat mengaitkannya dengan file MP3 di Thunar / Nautilus / file-manager-of-choice. Mereka mengambil daftar file / URL untuk dimainkan sebagai argumen:

$ mpg123 /path/to/mp3s/*.mp3
# plays all mp3 files in directory /pat/to/mp3s (generally in order of filename)

$ find /path/to/mp3s -name '*.mp3' -print0 | xargs -0 mpg123
# plays all mp3 files in /path/to/mp3s and subdirectories

Lihat man mpg123 ​​atau man mpg321 untuk detailnya.

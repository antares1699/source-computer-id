Bagaimana Anda menginstal ulang nano di Mac OS X?

Saya tidak sengaja menimpa aplikasi nano saya dengan versi biner unix, dan sekarang nano tidak berfungsi di Mac OS (Maverick) saya.

Bagaimana cara memulihkan nano atau mendapatkannya kembali?

Saya tidak bisa menemukannya di homebrew dan saya tidak tahu kacang tentang kompilasi file src.

Kesalahan nano saat mencoba menggunakan git

/Applications/Xcode.app/Contents/Developer/usr/libexec/git-core/git-rebase--interactive: line 225: /usr/bin/nano: cannot execute binary file
Could not execute editor

  1. Unduh sumber nano (Saya merekomendasikan yang tar.xz) ( nano-editor.org/download.php )
  2. Buka folder tempat unduhan itu (Secara default diunduh ke ~/Downloads tetapi jika Anda mengubah direktori tempat unduhan file, buka saja).
  3. Ekstrak sumber nano yang Anda unduh.
  4. Buka terminal Anda
  5. Ketik "cd" tanpa "" lalu seret dan jatuhkan folder yang Anda ekstrak dan klik enter.
  6. jalankan "./configure" tanpa "".
  7. jalankan "make" tanpa "".
  8. jalankan "sudo make install" tanpa "".]

Sekarang Anda telah berhasil menginstal nano!

Pembaruan: Jawaban di bawah ini tidak lagi akurat. Mulai 4 Apr 2017, rumus dipindahkan ke homebrew / core. Lihat https://github.com/Homebrew/homebrew-dupes/commits/master/README.md

Alasan Anda tidak dapat menemukannya di homebrew adalah karena repo utama homebrew tidak menyertakan program yang disertakan dengan OS. Anda harus menambahkan repo tambahan:

brew tap homebrew/dupes
brew install nano

https://github.com/Homebrew/homebrew-dupes

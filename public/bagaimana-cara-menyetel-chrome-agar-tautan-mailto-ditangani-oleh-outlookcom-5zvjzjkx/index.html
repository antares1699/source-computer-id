Bagaimana cara menyetel Chrome agar tautan mailto: ditangani oleh Outlook.com?

Saat Anda membuka gmail.com, Chrome meminta Anda untuk membiarkan mailto: tautan ditangani oleh gmail. Di Outlook.com, Microsoft lupa memberikan fitur ini kepada penggunanya. Bagaimana cara melakukannya sendiri?

Ini sedikit proses yang berbelit-belit, tetapi Anda dapat melakukannya dengan menggunakan navigator.registerProtocolHandler dan situs web yang sedikit menulis ulang string kueri URL.

Pergi ke http://terrific-charger.glitch.me . Ini adalah proyek Glitch kecil yang saya tulis dengan kodenya di bawah ini, tetapi Anda dapat memverifikasi bahwa tidak ada saputangan di sini: https://glitch.com/edit/#!/terrific-charger?path=script.js:1 E5E5E5 .

Buka konsol Chrome ( ctrl + bergeser + j ), lalu tempel kode berikut ke dalamnya dan tekan enter:

navigator.registerProtocolHandler('mailto','http://terrific-charger.glitch.me/?mailto=%s','Outlook.com');

Chrome sekarang akan memberi tahu Anda bahwa " http://terrific-charger.glitch.me/ ingin Membuka tautan email ". Klik" Izinkan ". Selesai!

Untuk membatalkan perilaku baru, buka: chrome://settings/handlers, dan hapus penangan yang terdaftar menggunakan menu tiga titik.

Kode kesalahan:

if (location.search.length)
   location = 'https://outlook.live.com/default.aspx?rru=compose&to=' + 
       decodeURIComponent(location.search).slice('?mailto=mailto:'.length).replace('?','&');

Perusahaan saya memiliki Office 365 dan Outlook online dihosting di https://outlook.office.com/ . Untuk menyetel situs ini sebagai penangan email di Chrome:

  1. Buka situs surat web
  2. Buka konsol Chrome ( ctrl + bergeser + j )
  3. Salin baris berikut di konsol dan tekan Memasukkan :
navigator.registerProtocolHandler("mailto", "https://outlook.office.com/?path=/mail/action/compose&to=%s", "Office 365");

Jika Anda berubah pikiran, buka chrome://settings/handlers dan hapus penangan yang terdaftar menggunakan menu tiga titik.

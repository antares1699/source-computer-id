Bagaimana melindungi partisi agar tidak menulis hanya untuk satu OS

Saya memiliki skenario & masalah yang tidak biasa:

  • Saya memiliki tiga sistem operasi yang diinstal pada laptop: Windows 7, Pratinjau Pengembang Windows 8, dan Windows Vista.
  • Saya memiliki partisi untuk setiap sistem operasi dan ke-4 tempat penyimpanan data, sehingga dapat diakses dari semua sistem operasi.

Sekarang, saya memiliki kebutuhan berikut:

  • Saya ingin Windows 8 dan Windows Vista mengakses semua partisi kecuali yang Windows 7.
  • Saya ingin Windows 7 TIDAK dapat mengakses semua file partisi lain kecuali yang diinstal.

Bagaimana saya bisa "mengunci" Windows 7 dari menulis atau mengakses partisi lain? Selain itu, bagaimana cara "mengunci" OS lain agar tidak menulis atau mengakses partisi Windows 7?

Boot setiap Microsoft OS, dan mulai utilitas Manajemen Disk (Mulai / Alat Administratif / Manajemen Komputer).

Untuk setiap volume / partisi yang Anda tidak ingin mengakses versi Windows, pilih menggunakan klik kanan dan pilih "Ubah Huruf dan Jalur Drive". Pilih "Hapus", dan partisi tidak lagi memiliki huruf drive. Anda akan secara efektif menonaktifkan akses ke partisi itu untuk versi Windows itu.

Anda dapat mengunci partisi lain hanya dengan melepasnya. OS hanya melihat partisi seperti disk. Bagaimana Anda mencegah OS menulis ke flash drive Anda? Jangan mencolokkannya sejak awal.

Plus, setiap OS akan mengingat disk mana yang Anda putuskan untuk tidak dipasang.

Tidak ada OS yang dapat mengunci OS lain dari sebuah partisi. Ini adalah pelanggaran alam yang sederhana karena setiap OS adalah entitas yang berbeda itu tidak mengetahui OS lain . Dan seharusnya seperti itu.

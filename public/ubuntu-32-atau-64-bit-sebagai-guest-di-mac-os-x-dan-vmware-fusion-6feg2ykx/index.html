Ubuntu 32 atau 64 bit sebagai guest di Mac OS X dan VMWare Fusion?

Saya akan menggunakan VM Ubuntu di Mac OS X dengan VMWare Fusion tetapi saya tidak yakin versi mana yang akan memberi saya kinerja yang lebih baik (32 atau 64 bit).

Saya yakin versi 32 bit harus menggunakan lebih sedikit memori, tetapi saya tidak yakin. Apa yang kamu sarankan?

4

Kecuali Anda berencana memberikan VM lebih dari 4 giga memori, tidak perlu menggunakan versi 64-bit. Sistem operasi 64-bit dapat menangani lebih banyak memori daripada sistem operasi 32-bit, tetapi jika Anda memiliki kurang dari 4 pertunjukan yang tersedia, maka tidak masalah, karena OS 32-bit dapat menangani sebanyak itu. Keduanya harus memberi Anda kinerja yang sama, tetapi beberapa perangkat lunak tidak tersedia untuk linux 64-bit dan untuk linux 32-bit, jadi Anda mungkin lebih baik menggunakan versi 32-bit.

Memori adalah apa pun yang Anda berikan pada VM, bitness tidak ada hubungannya dengan itu di sini (kecuali bahwa 32-bit dibatasi hingga maksimum 4 gigs).

Masalah sebenarnya adalah kecepatan perintah dan virtualisasi, di mana 64-bit akan memberi Anda keuntungan. Menggunakan bitness tertinggi yang tersedia pada prosesor Anda untuk proses tersebut tidak hanya akan memberikan fleksibilitas untuk menjalankan Ubuntu 64-bit, tetapi juga memungkinkan Anda untuk menjalankannya menggunakan instruksi virtualisasi perangkat keras. Saya tidak yakin apakah versi 32-bit memanfaatkannya (saya tahu itu tidak memerlukannya).

Perbedaan kinerja tidak akan besar kecuali pada operasi besar atau memori besar, tetapi secara umum, jika Anda akan membeli perangkat keras 64-bit (yang setara dengan VMWare di sini), Anda harus menggunakan aplikasi 64-bit.

Tidak ada kerugian menggunakan host 64-bit dan setidaknya beberapa keuntungan.

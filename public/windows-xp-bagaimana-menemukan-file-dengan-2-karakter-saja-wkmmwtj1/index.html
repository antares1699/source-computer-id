Windows XP: Bagaimana menemukan file dengan 2 karakter saja?

baru-baru ini saya menemukan pintu belakang di laptop saya. Tadi menghasilkan banyak file dengan nama acak hanya 2-3 karakter saja. Sebagian besar kosong. Bagaimana saya dapat menemukan file hanya dengan 1 hingga 3 karakter yang dibuat dalam 30 hari terakhir? Terima kasih Arman.

Gunakan fasilitas pencarian di Windows Explorer dan gunakan '?' karakter wildcard.

Cari nama file yang cocok dengan ?, ?? dan ???. Jika file memiliki ekstensi, coba ?.*, ??.* dan ???.*.

Anda juga dapat menentukan tanggal pembuatan.

Jika laptop Anda memiliki / memiliki pintu belakang, Anda seharusnya tidak terlalu peduli dengan file di sistem dan lebih peduli dengan menginstal ulang laptop Anda, karena ini adalah satu-satunya cara yang pasti untuk menutup pintu belakang dan menghapus kit root apa pun.

Mungkin lebih mudah menemukan file kosong yang dibuat dalam 30 hari terakhir.

Buka menu Start - Search

Klik When was it modified pilih Past month atau pilih tanggal secara manual.

Klik What size is it pilih Specify size at most 1KB

Menggunakan Powershell:

$date = get-date

$olddate = $date.adddays(-30)

get-childitem -path c:\ -Recurse | where-object {$_.basename.length -lt "4" -and $_.creationtime -gt $olddate}

secara otomatis membuka perangkat yang dapat dilepas di penjelajah

apakah mungkin menyetel putar otomatis untuk selalu membuka perangkat yang dapat dilepas di windows explorer? Saya menggunakan pengaturan yang ditawarkan dalam dialog putar otomatis dan mengatur pembukaan di Explorer untuk hampir semua jenis konten, tetapi setelah memasukkan flash disk, saya mendapatkan dialog lagi.

Saya memerlukan ini untuk saudara perempuan saya yang mengalami gangguan penglihatan - baginya tidak selalu nyaman untuk selalu memilih item yang sesuai di menu atau selalu menutupnya dan membuka Komputer Saya.

Dia menggunakan Windows 7 Pro 64 bit.

Buka Start - Control Panel - Autoplay dan pilih tindakan default untuk setiap jenis media. Secara default, Windows 7 telah autorun.ini dinonaktifkan untuk perangkat USB untuk melindungi dari malware. Jika ini tidak menyelesaikan masalah Anda, Anda mungkin perlu mengaktifkan autorun.ini dan membuatnya di media untuk saudara perempuan Anda.

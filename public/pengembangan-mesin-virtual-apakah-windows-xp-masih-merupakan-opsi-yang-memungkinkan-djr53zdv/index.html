Pengembangan Mesin Virtual - Apakah Windows XP masih merupakan opsi yang memungkinkan

Saya ingin memisahkan pekerjaan pengembangan saya dari pekerjaan rutin saya. OS host saya adalah Windows 7. Saya bertanya-tanya apakah Windows XP akan menjadi pilihan yang layak untuk OS tamu. Saya tidak memiliki salinan Windows XP sekarang. Melakukan beberapa pencarian cepat di situs seperti Futureshop, saya melihat bahwa mereka tidak lagi menjual Windows XP.

Saya tidak ingin menginstal Windows 7 sebagai OS tamu (karena akan memakan banyak sumber daya sistem).

Apakah saya masih dapat membeli Windows XP dari sumber yang memiliki reputasi baik? (Saya ingin itu legal). Apakah ada alasan mengapa itu tidak tersedia lagi (saya pikir itu akan tersedia lebih banyak setidaknya untuk keperluan mesin Virtual - mengingat itu jauh lebih ringan dan menyediakan basis yang diperlukan untuk Visual Studio, SQL Server dll.).

Bagaimana cara menyalin teks dari editor teks jendela ke nano / vi di jendela Putty

Saya mengikuti beberapa panduan untuk membuat pasangan kunci untuk sistem linux jarak jauh saya, salah satu langkahnya adalah menyalin kunci publik yang dihasilkan dan disimpan di komputer lokal saya ke file authorized-keys di komputer jarak jauh. Saya mencoba ctrl + c lalu ctrl + v ke editor nano / vi di jendela "putty", tetapi tidak berhasil (tidak ada yang muncul di editor). Apakah saya benar-benar harus mengetikkan kunci secara manual atau ada cara lain untuk menyalin dan menempelkannya?

Anda tentu saja dapat menggunakan mouse, tetapi itu berisiko memilih / menempel secara tidak sengaja karena gerakan mouse.

Putty mengakui Bergeser Memasukkan untuk menempelkan dari papan klip. Ini bekerja dengan andal (pemantulan kunci jarang terjadi). Anda juga dapat mengubah perilaku klik kanan untuk mendapatkan menu, seperti yang disebutkan di halaman Greg Ferro.

Bacaan lebih lanjut:

  • Putty - Memperbaiki Tempel Klik Kanan
  • Cara Menyalin dan Menempel Dengan Putty

Jika Anda memiliki konten dari aplikasi lain yang disalin ke clipboard di mesin lokal Anda, klik kanan di mana saja di dalam jendela PuTTY untuk menempelkannya ke shell jarak jauh.

Catatan: alih-alih menempelkan kunci ke editor teks, Anda dapat menggunakan echo dan mengalihkan keluarannya ke authorized-keys, seperti: echo [paste your public key here] >> ~/.ssh/authorized_keys. Dulu, saya mengalami masalah saat menempelkan ke nano / vim dengan PuTTY (meskipun mungkin telah diperbaiki perilaku itu).

Coba klik shift-right sebagai gantinya.

Saya juga menemukan bahwa klik kanan saja tidak akan berfungsi (dalam nano) - meskipun mengklik kanan di terminal AKAN menempel clipboard. Dan saya menemukan segala macam saran untuk dicoba (ctrl-shift-v, shift-insert), tidak ada yang berhasil. Akhirnya, saya menemukan seseorang yang menyarankan klik kanan shift, dan itu berfungsi tanpa mod yang diperlukan. Keselamatan ketika seseorang harus memasukkan baris panjang atau banyak dari luar editor!

Karena itu, saya menemukan beberapa teks multi-baris akan ditempel sementara beberapa tidak. Dalam hal ini, jika saya menempelkannya beberapa baris sekaligus, itu berhasil. Aneh. Membagikan hal itu seandainya itu dapat membantu orang lain, atau mereka mungkin tahu apa yang terjadi.

Ikon untuk Chrome hilang setelah memperbarui Windows-10 hingga Pembaruan November 1511

Setelah memperbarui Windows-10 hingga Pembaruan November 1511, ikon untuk Chrome hilang seperti pada tangkapan layar, juga ikon Chrome dan Firefox menampilkan latar belakang abu-abu alih-alih warna tema. Ikon Chrome muncul ketika ukurannya diubah menjadi Sedang tetapi tidak terlihat bagus. Di build sebelumnya 10240 itu menunjukkannya dengan benar. Apakah ini bug di pembaruan Windows atau chrome?

Saya memiliki "masalah" yang sama. Berhasil menyelesaikannya untuk Chrome, menggali untuk Firefox. Di folder Chrome (tempat chrome.exe berada) ada file visualelementsmanifest.xml. Tujuannya adalah untuk menyesuaikan tampilan ubin layar awal sejak Windows 8 hari. Ini dibuat secara default oleh pemasang Chrome. Untuk beberapa alasan, 10240 mengabaikannya, sekarang 10586 sedang memuatnya dan itu menentukan warna abu-abu - # 323232. Hapus file. Mungkin perlu beberapa saat bagi ubin untuk memperbarui, melepas pin, dan memasangnya ulang, dll. Sebenarnya, opsi terbaik adalah mengubah ikon pintasan ke yang lain, daripada kembali ke yang default.

Adapun Firefox - tidak yakin - tidak ada file seperti itu untuk itu, jadi saya belum tahu apa yang menyebabkan warna keabu-abuan yang disesuaikan.

R. Klik pintasan, buka properti, klik tombol "Ubah Ikon". (*) Ini akan mengeluh bahwa "jendela tidak dapat menemukan file ...". Masukkan "% ProgramFiles% \ Google \ Chrome \ Application \ chrome.exe" dan ikon Anda akan kembali. Anda juga dapat mencoba "% ProgramFiles (x86)% daripada% ProgramFiles%. Jika masih belum kembali, telusuri secara manual ke tempat chrome.exe berada di kotak dialog untuk mengubah ikon.

(*) Jika itu pin, Anda harus r. Klik pin, menu konteksnya adalah pin. Pada menu konteks pin, r.klik entitas "google chrome". Itu jalan pintas yang sebenarnya.

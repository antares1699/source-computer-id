Saya ingin mengubah paket daya laptop saya melalui Remote Desktop Connection, tetapi opsinya berwarna abu-abu.

Saya masuk dengan akun admin dan kedua PC menjalankan Windows 7.

Apa masalahnya?

Ini harus dimungkinkan jika Anda terhubung ke console melalui RDP. Untuk melakukan ini, buka command prompt dan sambungkan ke server RDP seperti ini:

mstsc /v:server /console

Ganti server dengan nama host atau IP server Anda.

Konsol adalah jenis sesi jarak jauh khusus yang hanya dapat menghubungkan satu koneksi pada saat itu. Anda dapat menemukan informasi lebih lanjut di Microsoft TechNet: http://technet.microsoft.com/en-us/library/cc740144%28v=WS.10%29.aspx

Biasanya, Anda dapat melakukan dari baris perintah, dalam sesi jarak jauh, apa yang tidak dapat Anda lakukan melalui UI. Dalam hal ini, Anda harus dapat menjalankan POWERCFG dengan opsi yang sesuai, meskipun Anda tidak dapat menjalankan melalui baterai atau UI panel kontrol.

Opsi untuk perintah dirinci di http://technet.microsoft.com/en-us/magazine/ff700230.aspx

Bagaimana cara mengatur ukuran font default di Windows 7 tanpa menskalakan sisanya?

Ada opsi untuk "Membuat teks dan item layar lainnya lebih besar atau lebih kecil ". Bagaimana cara mengatur hanya ukuran font , tapi biarkan item layar lainnya tidak berubah?

Motivasi Saya memiliki netbook, layarnya kecil dan tidak ada ruang untuk memperbesar elemen non-font atau mereka akan keluar dari layar.

Di sisi lain, font terlalu kecil untuk dibaca dengan mudah.

Prioritas saya adalah:

  • Ukuran font file menu mulai
  • Pengaturan seluruh sistem, jika memungkinkan
  • Daftar entri registri akan menjadi yang terbaik :)

Saya mencari jawaban lain, tetapi saya pikir mereka akan meningkatkan segalanya, bukan hanya fontnya.

Pergi ke...
Control Panel> Personalization> Window Color> Advanced appearance settings... ...atau...
Klik kanan desktop, pilih Personalize, dan lanjutkan dengan urutan di atas.

Dari sini, Anda dapat mengubah ukuran font di seluruh sistem (dengan sendirinya) untuk berbagai hal. Mengubah Ikon ukuran font adalah salah satu yang mempengaruhi teks Start Menu.

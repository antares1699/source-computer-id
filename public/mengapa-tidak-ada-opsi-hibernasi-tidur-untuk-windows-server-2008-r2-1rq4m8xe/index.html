Mengapa tidak ada opsi hibernasi / tidur untuk Windows Server 2008 R2?

Saya menggunakan Windows Server 2008 R2 Box untuk tujuan pengembangan saya. Saya tidak bisa melihat opsi hibernasi / tidur di opsi mematikan. Apa yang bisa menjadi alasan untuk menyembunyikan opsi ini untuk mesin server?

4
sarat — Sumber
  • Mungkin karena tidak ada gunanya opsi tersebut di server? Mengapa server perlu hibernasi dan tidak dimatikan atau dimulai ulang?
    — AndrejaKo
  • ya saya setuju. hibernasi / tidur tidak terlalu cocok untuk lingkungan server. Saya hanya ingin memastikan apakah itu satu-satunya alasan.
    — sarat
  • Tidak semua server diperlukan 24/7. Jika server Anda menjalankan operasi LOB untuk satu zona waktu, maka masuk akal untuk meminta mereka mengurangi penggunaan daya saat tidak diperlukan. Banyak perusahaan memiliki sekitar 40 jam kerja dalam seminggu dan ada 168 jam dalam seminggu.
    — Colin Pickard
  • Benar. Tetapi beberapa orang datang lebih awal (sebelum jam 7 pagi), beberapa pergi terlambat (katakanlah jam 8 malam). Beberapa bekerja di server dari rumah (misalnya saya, biasanya 8 hingga 10 jam per akhir pekan). Jadi a Server sering kali harus menyala lebih dari 40 jam.
    — Hennes

Sudahkah Anda menginstal peran Hype-v? Jika ya, maka Windows menonaktifkan opsi Tidur dan Hibernasi. Anda harus mengaktifkan layanan Hyper-V saat Diminta untuk Tidur dan opsi Hibernasi untuk diaktifkan kembali.

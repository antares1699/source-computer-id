Oke, tidak ada keraguan bahwa diskpart adalah alat yang bagus untuk partisi windows, tetapi sepertinya itu hanya dapat membaca atau mengedit disk lokal windows dan patisinya, tidak dapat mengenali file disk seperti file partisi MBR, recovery, atau EBR1 yang Anda lihat di filesystem linux; Jadi benar-benar ada alat yang cocok dengan fdisk banyak di sistem Windows, itu alat cmd, ini namanya dan situs web unduhannya:

Unduhan GPT fdisk

dan ini adalah gambarannya tentang kenali saya MBR mengajukan:

Tangkapan layar GPT fdisk

Ini memiliki sintaks yang mirip dengan fdisk -l filename, Anda dapat menggunakan gdisk64.exe -l filename.

Halaman unduhan juga menyertakan situs web resminya, Anda bisa mendapatkan lebih banyak penggunaan dari itu! atau cukup gunakan gdisk64.exe /? dalam cmd untuk informasi lebih lanjut _

Cara membuat pekerjaan cron untuk mengunggah file ke server FTP

Saya ingin membuat pekerjaan cron yang mengunggah file dari direktori di komputer saya ke server FTP saya. Saya ingin melakukannya setiap hari pada tengah malam. Saya tidak tahu apa-apa tentang cron, jadi saya minta maaf jika saya terdengar bodoh!

Ini adalah contoh skrip FTP untuk mentransfer satu file: (Perhatikan, Anda dapat menggunakan FQDN alih-alih IP)

#!/bin/bash

# $1 is the file name for the you want to tranfer
# usage: this_script  <filename>
IP_address="xx.xxx.xx.xx"
username="remote_ftp_username"
domain = sample.domain.ftp
password= password

ftp -n > ftp_$$.log <<EOF
 verbose
 open $IP_address
 USER $username $password
 put $1
 bye
EOF

Tambahkan > ftp_$$.log hanya jika Anda perlu logging. Kemudian Anda dapat menggunakan

crontab -e

perintah untuk mengedit tabel cronjob dan menambahkan skrip Anda.

Ini adalah Contoh:

Jika Anda suka memiliki skrip di atas, (asumsikan Anda memilikinya di rumah dan namanya myscript.sh) /home/myscript.sh, jalankan setiap hari pada jam 2 pagi, Anda harus melakukan:

# crontab -e

dan kemudian Anda harus menambahkan entri berikut:

0 2 * * * /home/myscript.sh

Sebagai referensi, di sini Anda memiliki parameter entri crontab yang artinya:

* * * * * command to be executed
- - - - -
| | | | |
| | | | ----- Day of week (0 - 7) (Sunday=0 or 7)
| | | ------- Month (1 - 12)
| | --------- Day of month (1 - 31)
| ----------- Hour (0 - 23)
------------- Minute (0 - 59)

Tutorial ini juga bisa membantu Anda.

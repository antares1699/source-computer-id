Ubuntu Automount berdasarkan Label?

Server Ubuntu 9.10

Saya tahu bahwa menggunakan perintah mount, Anda dapat menggunakan -L untuk memasang dengan label seperti:

mount -L thelabel /media/themount

Apakah ada cara serupa untuk mengatur Automount / Autofs untuk dipasang berdasarkan nama label?

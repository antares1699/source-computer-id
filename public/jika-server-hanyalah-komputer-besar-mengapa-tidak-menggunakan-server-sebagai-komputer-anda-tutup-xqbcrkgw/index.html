Itu adalah server bekas, dan Anda bisa gunakan itu. Tapi Anda juga bisa membeli PC bekas yang murah.

Perhatikan juga, spesifikasi tersebut memberi tahu Anda sangat sedikit.

Beberapa hal yang perlu diperhatikan:

Apakah ada 2 Pentium era 3,2GHz Xeon? Atau apakah mereka Itanium (Tidak berguna untuk komputasi desktop).

Apakah Power Supply sudah termasuk? Apakah ini membutuhkan 2 catu daya atau jika tidak, bip terus-menerus karena catu daya gagal?

RAM apa itu? MEE? EEC lebih lambat dari RAM desktop biasa.

Apakah Anda dapat menemukan driver RAID yang benar dengan mudah untuk OS desktop?

Dalam bentuk apa hard drive itu?

Jika sudah berlangsung selama 2-5 tahun atau lebih di perusahaan 24/7, Anda dapat mengharapkan PSU, motherboard, dan hard drive mendekati akhir masa pakainya.

Ada kemungkinan besar tidak akan ada kemampuan grafis.

Beberapa lisensi sistem operasi mengizinkan banyak prosesor logis (dual, quad core, dll.) Tetapi hanya satu prosesor fisik.

Ruang, tergantung pada server, mungkin sulit untuk memasukkannya ke suatu tempat. Apakah ini 1u, 2u, 4u? Bagaimana Anda memasang server rak di rumah?

Seperti yang Anda katakan, server adalah komputer "besar". Apakah Anda benar-benar ingin memiliki lempengan logam yang berukuran setengah dari ukuran orang dewasa seperti PC Anda? Di mana Anda akan menaruhnya?

Masalah kebisingan pasti akan menjadi masalah juga, yang akan merusak pengalaman bermain game dan / atau pusat data Anda.

Alasan mengapa hal-hal ini sangat murah adalah karena mereka sudah usang dibandingkan dengan penawaran server modern. Anda bisa mendapatkan server baru dengan Ivy Bridge Xeons ganda dengan 16 core pada hampir 3GHz, RAM lebih dari 100GB, dan penyimpanan lebih dari satu TB, untuk memberikan contoh tipikal. Ingatlah, server $ 200 ini mungkin Dell 1950 / 2950s atau HP Gen5s. Ini adalah mesin Core 2-era. Tentu tidak buruk untuk saat ini (mempertimbangkan prosesor 3,2GHz), tetapi apakah itu BENAR-BENAR akan memberi Anda tenaga kuda sebanyak itu untuk bermain game di judul hari ini dibandingkan dengan sistem anggaran saat ini?

Lihat Maraton Pembuat Sistem triwulanan Perangkat Keras Tom, tempat Anda dapat membuat PC game seharga ~ $ 600 menggunakan suku cadang baru. Ini adalah penggunaan uang Anda yang jauh lebih baik, belum lagi energi yang dibutuhkan untuk menyeret pembelian baru Anda menaiki tangga.

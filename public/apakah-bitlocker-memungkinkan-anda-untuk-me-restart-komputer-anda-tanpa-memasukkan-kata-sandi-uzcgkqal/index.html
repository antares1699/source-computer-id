Apakah Bitlocker memungkinkan Anda untuk me-restart komputer Anda tanpa memasukkan kata sandi?

Saya mengaktifkan Bitlocker di komputer Windows 10 saya, tanpa TPM, hanya kata sandi. Biasanya ketika saya me-restart komputer saya, itu memberi saya halaman biru biasa yang meminta kata sandi saya. Setelah memasukkannya, itu melakukan boot secara normal, memberi saya layar masuk normal dengan nama pengguna dan gambar profil saya. Proses yang sama ini biasanya terjadi pada restart otomatis karena pembaruan Windows.

Selama beberapa hari saya menerima pemberitahuan bahwa saya perlu memulai ulang untuk pembaruan besar yang baru. Pagi ini otomatis restart (rupanya, saya tertidur) dan menginstal pembaruan. Tetapi ketika saya bangun dan menggunakan komputer, saya tidak mendapatkan halaman kata sandi biru biasa. Saya mendapat halaman login dengan nama pengguna dan gambar saya, meskipun gambar latar belakangnya berbeda. Saat masuk, dikatakan selamat datang di pembaruan baru, dan semua jendela yang saya buka ditutup (sama seperti restart).

Jadi pertanyaan saya adalah, bagaimana cara melakukan restart tanpa halaman biru meminta kata sandi saya? Bagaimana saya bisa mengetahui nama pengguna dan gambar profil saya tanpa saya memasukkan kata sandi? Apakah Bitlocker benar-benar mengenkripsi data saya atau tidak?

Dalam log acara saya tertulis "Sistem operasi dimulai pada waktu sistem 2018 - 02 - 04T12: 19" [...]. Jadi itu sepertinya menandakan itu reboot. Anehnya semua log acara saya sebelum waktu ini tampaknya telah dihapus.

Anda dapat menangguhkan bitlocker hingga 15 Reboot sekaligus dengan Powershell.

Mulai Powershell Shell dengan Hak Admin dan masuk

Suspend-BitLocker -MountPoint "Drive" -RebootCount X

Di mana Drive adalah Driveletter Anda (yaitu C :) dan X adalah Jumlah Reboot yang ingin Anda tangguhkan Bitlocker (0-15)

Kemungkinan besar Program Pembaruan menggunakan ini atau variasinya.

Sumber: https://technet.microsoft.com/de-de/library/jj649830(v=wps.630).aspx

Bagaimana cara cepat mengganti monitor pada pengaturan multi-monitor?

Di PC desktop saya, saya memiliki 3 layar yang terhubung ke Palit GeForce GTX 480 saya untuk memperluas desktop saya:

  • DVI: Samsung SyncMaster T220 (22 "16:10, 1680x1050, sebagai komputer utama saya)
  • DVI: Samsung SyncMaster 940BF (19 "4: 3, 1280x1024)
  • HDMI: Sony Bravia KDL-32D3000 (32 "16: 9, 1360x768)

Seperti yang diharapkan, kartu GFX hanya dapat menggunakan 2 tampilan pada waktu tertentu, jadi saya harus beralih secara manual menggunakan dialog "Properti Tampilan" Windows, setiap kali saya ingin beralih di antara keduanya.

Saya sudah mencoba UltraMon, DisplayFusion dan PowerStrip (jadi tolong jangan rujuk saya ke pertanyaan di mana itu diberikan sebagai solusi) tetapi mereka tidak dapat mengganti monitor. Sebaliknya mereka hanya dapat mengubah resolusi tampilan dan pengaturan pada monitor aktif yang ada, dengan hasil yang aneh khususnya ketika saya menyimpan status monitor dan kemudian mencoba untuk beralih monitor menggunakan status tersimpan!

Adakah ide tentang cara melewati dialog "Properti Tampilan" yang buruk dan pengatur waktu "konfirmasi perubahan Anda"?

Ada fungsi bawaan di Windows 7: Kunci Windows + P. .

Ini menunjukkan empat opsi: tunggal (Putuskan Sambungan Proyektor), cermin (Duplikat), perpanjang (Perpanjang), dan proyektor (hanya Proyektor). Salah satu monitor Anda akan menjadi proyektor, yang lainnya adalah layar yang diperpanjang sehingga Anda dapat beralih antara proyektor dan layar.

displaywitch.exe adalah alat di Windows 7 yang memungkinkan Anda beralih mode dengan cepat.

Ini memiliki parameter baris perintah:

  • / internal (Putuskan sambungan proyektor)
  • / clone (Layar duplikat)
  • / perpanjang (Perpanjang layar)
  • / eksternal (hanya Proyektor)

Informasi lebih lanjut sini .

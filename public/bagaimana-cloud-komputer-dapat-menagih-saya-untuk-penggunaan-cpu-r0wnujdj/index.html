Komputer

Bagaimana cloud komputer dapat menagih saya untuk penggunaan CPU?

Saya sedang mempertimbangkan untuk memindahkan blog saya ke MS Azure atau Google App Engine, menggunakan mesin blog ASP.NET atau Python, tetapi saya tidak mengerti bagaimana mereka akan menagih saya untuk penggunaan CPU. Bandwidth dan biaya penyimpanan tampak mudah, tetapi bukan CPU ...

Menurut dokumentasi masing-masing layanan, saya akan membayar $ 0,12 atau $ 0,10 untuk satu jam penggunaan CPU. Apakah itu berarti saya akan dikenakan biaya sekitar 30 days * 24h * $0.12 = $86.40 setiap bulan terlepas jumlah pengunjung dan kompleksitas kode saya? Apakah saya akan membayar harga CPU yang sama ketika saya memiliki 2 pengunjung sehari seperti ketika saya memiliki seribu?

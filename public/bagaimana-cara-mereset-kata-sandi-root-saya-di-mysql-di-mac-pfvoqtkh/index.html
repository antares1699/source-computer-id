Bagaimana Cara Mereset Kata Sandi Root Saya di MySQL di Mac?

Saya menggunakan SequelPro ( http://www.sequelpro.com/ ) dan saya ingin tahu cara mengatur ulang kata sandi root saya. Saya mencoba masuk sebagai

host: localhost
nama pengguna: root
kata sandi: (KOSONG)

Tapi saya terus mendapatkan kesalahan ini:

Tidak dapat menyambung ke host karena akses ditolak.
Periksa ulang nama pengguna dan kata sandi Anda dan pastikan bahwa akses dari lokasi Anda saat ini diizinkan.
MySQL berkata: Access denied for user 'root'@'localhost' (using password: NO)

Ada saran ?!

Bergantung pada apa yang Anda gunakan untuk menjalankan MySQL, Anda dapat melihatnya instruksi resmi .

Jika Anda menggunakan MAMP atau sesuatu yang serupa Anda perlu mencari di mana mysqladmin berada.

/Applications/MAMP/Library/bin/mysqladmin -u root -p password ’somepassword’

Jika Anda menggunakan instalasi mysql lain, Anda mungkin dapat mengetahui di mana lokasinya dengan:

$ which mysqladmin

Berikut adalah prosedur untuk mereset kata sandi pengguna root.

1) Hentikan mysql (Bunuh proses mysql atau jalankan perintah berikut)

sudo /usr/local/mysql/support-files/mysql.server stop

2) Mulailah dalam mode aman

sudo mysqld_safe --skip-grant-tables

3) Buka terminal lain dan jalankan perintah berikut (Biarkan terminal terakhir tetap terbuka)

mysql -u root

4) Jalankan perintah berikut dengan kata sandi baru yang sesuai di konsol mysql

mysql > UPDATE mysql.user SET Password=PASSWORD('password') WHERE User='root';

5) mysql > FLUSH PRIVILEGES;

6) Keluar dari kedua terminal dan buka terminal baru dan hubungkan ke mysql dengan pengguna root dan kata sandi baru

mysql -uroot -p

Daftar Langsung Bilah Tugas Putty Rusak di Windows 10

Saya memiliki Windows 8 di laptop saya dan jumplist untuk dempul selalu berfungsi dengan baik. Setelah menginstal Putty di desktop saya (Win 10), jumplist tidak muncul saat saya mengklik kanan ikon taskbar, Putty v0.65.

Setelah menjelajahi internet, ternyata perbaikan disediakan di versi 0.66. Jadi saya mencopot pemasangan 0,65 dan memasang 0,66 ... tidak ada bantuan, masalah yang sama. beberapa komentar tentang pertanyaan serupa dengan ini sebenarnya mengatakan bahwa mereka lebih beruntung dengan versi pengembangan, jadi saya mencopot pemasangan 0,66 dan menginstal Putty versi 11/30/2015. Saya telah melepas pin dan menyematkan ulang serta menghapus sesi yang saya simpan dan membuatnya kembali. Saya bahkan mencoba membersihkan entri registri dengan CCleaner setelah mencopot pemasangan dan sebelum memasang. Saat ini saya memiliki 2 sesi yang disimpan dan seperti inilah tampilan menu konteks bilah tugas klik kanan saya (daftar lompat menu mulai persis sama):

P.S. aku sudah mengunjungi pertanyaan ini dan mencoba solusinya. Namun, saya tidak dapat berkomentar karena reputasi saya yang terbatas, jadi saya harus membuat pertanyaan sendiri.

Solusi yang diusulkan yang saya lihat tidak berhasil untuk saya. Tapi ini berhasil:

Dengan asumsi Anda sudah memasang PuTTY yang disematkan ke bilah tugas:

  1. Mulailah sesi PuTTY.
  2. Anda sekarang akan memiliki ikon Putty kedua di bilah tugas Anda. Lepas pin yang asli.
  3. Pada ikon baru, klik kanan dan pilih Sematkan ke bilah tugas. Daftar lompat akan ada di ikon baru ini.

Jika penting, saya menjalankan PuTTY 0.68, 64-bit.

Jika Anda berada di perahu yang sama dengan saya, Anda dapat menggunakan solusi berikut yang saya temukan di sini:

http://www.thegeekstuff.com/2009/07/10-praktis-putty-tips-and-tricks-you-probably-didnt-know/ .

  • Buat pintasan windows PuTTY baru
  • Ubah nama pintasan ini menjadi nama favorit Anda.
    • contoh: dev-db.
  • Klik kanan pada pintasan ini dan pilih properti.
  • Di kolom Target, di bagian akhir tambahkan "-load putty-session-name"
    • contoh: "C: \ Program Files (x86) \ PuTTY \ putty.exe" -load dev-db
  • Klik kanan pintasan lagi dan pilih "Sematkan ke Bilah Tugas"
  • Anda sekarang dapat menghapus atau memindahkan pintasan, jika Anda mau

Ini akan menjadi item taskbar dempul tambahan jika Anda sudah memiliki dempul standar yang disematkan ke taskbar. Karena saya hanya menggunakan satu sesi dempul setiap kali, saya menghapus pin bilah tugas dempul asli. Jika Anda memiliki beberapa koneksi yang ingin disematkan, maka Anda akan mengulangi langkah-langkah di atas untuk masing-masing koneksi dan mungkin ingin mengubah ikon pintasan untuk masing-masing koneksi.

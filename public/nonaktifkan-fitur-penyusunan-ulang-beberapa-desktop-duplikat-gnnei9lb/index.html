Nonaktifkan Fitur Penyusunan Ulang Beberapa Desktop? [duplikat]

Saya memiliki MBP baru dengan 10.9 terinstal di dalamnya. Saya biasanya bekerja dengan bertukar antara desktop yang memiliki fungsi tertentu (seperti yang menjalankan Mail.app dan Cal.app di atasnya). Setelah saya mulai menggunakan mavericks, saya perhatikan bahwa ketika saya mengklik ikon dok, urutan desktop diatur ulang. Saya tidak berpikir ini adalah cara kerja 10.7, tetapi karena saya tidak lagi memiliki mesin itu, saya tidak dapat mengonfirmasi.

Adakah cara untuk menonaktifkan perilaku ini karena sangat mengganggu.

Lihat apakah ini berhasil untuk Anda.

  1. Luncurkan System Preferences dan klik Mission Control.
  2. Hapus centang pada kotak di samping Automatically rearrange spaces based on most recent use.
  3. Tutup Preferensi Sistem .

Dengan penataan ulang otomatis dinonaktifkan, Mission Control akan mengingat kembali aplikasi dan penempatan desktop Anda dan tidak mengatur ulang sendiri.

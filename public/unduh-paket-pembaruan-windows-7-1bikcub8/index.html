Tidak ada produk seperti itu dari Microsoft, tetapi ada alternatifnya.

Lihat ini : Unduh Pembaruan Windows 7 Offline dengan Pembaruan Offline WSUS .

Pembaruan Offline WSUS memungkinkan Anda untuk dengan mudah unduh tambalan dan pembaruan untuk Anda Windows dan Office tanpa perlu pergi melalui seluruh prosedur Situs web Microsoft Windows Update.

Motherboard Anda dirancang dengan mempertimbangkan beberapa perusahaan. Itu tidak pernah benar-benar dibuat untuk mendukung semua. Jadi, jika Anda mengetahui dengan RAM mana motherboard bekerja paling baik, tidak akan ada bedanya jika Anda memasangkan 4g dengan 1g selama Anda memasangkannya dengan benar.

Dalam papan 2 slot, dimm0 adalah 4g dan dimm1 adalah 1g. Tapi di papan 4 slot, itu akan menjadi dimm0 4g, dimm2 1g, dimm1 4g, dan dimm3 1g. Jumlahnya mungkin berbeda-beda, tetapi slot RAM diberi kode warna bersama-sama, katakanlah kuning dan biru kuning akan cocok dengan set dan biru cocok selama Anda selalu keluar dari kecepatan dan ukuran RAM yang lebih tinggi di set pertama mulai dari dimm0. Terakhir, perhatikan bahwa pada papan 8 slot, itu berubah sangat banyak kecuali jika mendukung 4 saluran ganda. Akan ada dua set empat dengan empat terkait bersama, jadi hati-hati untuk itu.

Bagaimana daftar file berdasarkan rentang tanggal?

Saya ingin membuat daftar file dengan umur 3 hari tahun. Saya menemukan yang ini di stackoverflow :

find . -type f -printf "%-.22T+ %M %n %-8u %-8g %8s %Tx %.8TX %p\n" | sort | cut -f 2- -d ' ' | grep 2012

Tapi saya agak tidak mengerti apa arti keseluruhan perintah. Saya ingin tahu apakah ada sesuatu yang singkat dan sederhana untuk dipahami.

Ini seharusnya berhasil

find . -type f -mtime -3

Penjelasan

find         find files
.            starting in the current directory (and it's subdirectories)
-type f      which are plain files (not directories, or devices etc)
-mtime -3    modified less than 3 days ago

Lihat man find untuk detailnya

Memperbarui

Untuk menemukan file yang terakhir diubah sebelum tanggal dan waktu tertentu (mis. 08:15 pada 20 Februari 2013) Anda dapat melakukan sesuatu seperti

  touch -t 201302200815 freds_accident
  find . -type f ! -newer freds_accident
  rm freds_accident

Lihat man touch (atau info touch - ugh!)

Ini cukup mengerikan dan mungkin ada cara yang lebih baik. Pendekatan di atas bekerja pada Unix kuno dan non-GNU serta Linux saat ini.

Temukan mendukung interval dengan argumen -ctime dan -mtime +/-.

misalnya

$ for y in {07..14};do \
  for m in {01..12};do \
  for d in {01..30};do \
    touch -t 20$y$m${d}0101 $y$m$d.file ;done;done;done

$ find . -mtime +0 -mtime -$(( 3 * 365 + 3 )) |sort 
./100304.file
./100305.file
./100306.file
(...)
./130302.file
./130303.file
./130304.file

Jika Anda ingin file dibuat dalam interval antara 3 tahun dan 3 hari yang lalu hingga seminggu yang lalu Anda akan menggunakan -mtime +7 -mtime -1098.

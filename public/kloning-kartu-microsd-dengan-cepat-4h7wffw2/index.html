Kloning kartu microSD dengan cepat

apakah ada cara mudah dan cepat untuk mengkloning semua data dari satu kartu microSD ke yang lain? Saya ingin secara berkala mencadangkan konten kartu telepon saya ke cadangan, dan seperti yang saya miliki banyak file kecil di kartu, penyalinan biasa memiliki banyak biaya tambahan.

Saya rasa saya dapat menggunakan beberapa jenis perangkat lunak gambar disk, tetapi saya tidak tahu apakah mereka dapat digunakan untuk media memori flash.

Apakah ada perangkat lunak yang dapat melakukan ini dengan mudah, seperti Ghost untuk hard drive?

Karena Anda tidak menentukan OS, berikut adalah cara saya melakukannya di linux.

colokkan kartu ke pembaca kartu Anda. Pasang itu (itu harus dipasang sebagai sdX lama #) dd if=/dev/sdX# of=/path/to/save/location/image.dd sync=noerror,notrunc bs=1k

Ini akan memberi Anda salinan persis dari 'parisi' pertama (jika dipartisi .. jika tidak, jangan khawatir). Anda dapat memasang ini dengan loopback lokal juga untuk mengambil file dari gambar.

Jika Anda khawatir tentang ukuran file, Anda selalu dapat melakukannya seperti halaman ini menyarankan untuk membuatnya lebih kecil dan lebih ringkas.

Bagikan

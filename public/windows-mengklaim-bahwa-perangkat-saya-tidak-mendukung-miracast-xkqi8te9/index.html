Windows mengklaim bahwa perangkat saya tidak mendukung Miracast

Saya baru saja membeli adaptor Microsoft Wireless Display dan mesin Windows 10 saya mengklaim bahwa perangkat saya tidak mendukung Miracast. Menggunakan panduan sini , Saya mengonfirmasi bahwa saya memiliki NdisVersion 6.4 pada kartu Wifi saya dan bahwa driver tampilan saya memiliki WDDM 2.0. Bahkan produknya deskripsi untuk kartu WiFi saya mengatakan bahwa itu mendukung WiDi.

Perangkat keras saya:

  • Nvidia GTX 770M menjalankan driver stabil terbaru
  • Intel i7 4710MQ @ 2.5GHz
  • Intel Duel Band AC 7260 diperbarui ke driver terbaru

Sejauh saya sadar , ini semua yang diperlukan untuk menggunakan Miracast, namun saat saya membuka pusat tindakan dan memilih "sambungkan", Windows memberi tahu saya bahwa "Perangkat Anda tidak mendukung Miracast, jadi tidak dapat diproyeksikan secara nirkabel". Apakah saya kehilangan persyaratan untuk Miracast atau apakah Windows sudah gila?

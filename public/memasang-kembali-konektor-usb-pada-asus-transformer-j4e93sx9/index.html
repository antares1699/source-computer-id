Memasang kembali konektor USB pada Asus Transformer

Port USB Asus Transformer saya rusak beberapa kali, jadi konektornya sendiri sekarang rusak karena penyolderan. Satu-satunya pilihan saya adalah memotong kabel dan menghubungkannya ke salah satu USB saya yang tidak terpakai, tetapi kabelnya berbeda.

Bagaimana cara menyambungkan konektor USB berkabel merah, oranye, kuning, hijau, dan putih ke konektor USB berkabel kuning, biru, coklat, dan merah?

inilah kode warna pinout standar untuk konektor 1.x / 2.0 (dipinjam dari Wikipedia)

Pin    Name


1  VBUS    Red +5 V
2  D−  White(Gold *)   Data −
3  D+  Green   Data +
4 GND Black(blue*) Ground

.- D + D- +

.-.-.-.-.-.-.-.

| ======== |

4 3 2 1

NB ..tapi Anda akan kehilangan jalur pin USB ke-5 dari kabel pengisi daya asus untuk memberitahukannya untuk beralih ke pengisian daya 15V, bukan 5V. Meskipun perbaikan tampaknya dapat dilakukan dengan membumikan kabel ke-5 (pin7) dari konektor Asus USB 3 menggunakan pengganti USB 2.0 (dijembatani / disolder ke pin 4 seperti di atas), Anda sebaiknya menggunakan konektor USB 3 / kabel jika Anda memilikinya. Lihat tautan ini http://forum.xda-developers.com/showthread.php?t=1087321

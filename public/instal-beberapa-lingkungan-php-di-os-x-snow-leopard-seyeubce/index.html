Instal beberapa lingkungan PHP di OS X Snow Leopard

Saya baru saja memutakhirkan MBP saya ke Snow Leopard (OS X 10.6), yang membawa PHP ke 5.3

Ini bagus, kecuali saya menggunakan MBP saya sebagai mesin pengembangan saya dan saya menggunakan banyak libs PHP dan framework (yaitu CakePHP 1.2) yang saat ini tidak kompatibel dengan PHP 5.3. CakePHP khususnya tidak memiliki versi stabil untuk PHP 5.3 jadi bukan masalah mengupgrade kerangka kerja (dan server produksi di bawah PHP 5.2).

Apakah ada cara untuk menginstal PHP 5.2.9 bersama PHP 5.3 dan kemudian menggunakan httpd.conf atau .htaccess memberi tahu Apache versi PHP mana yang akan digunakan untuk direktori tertentu?

Atau apakah ada cara untuk melakukan ini dengan MacPorts?

Terima kasih!

Saya sarankan mengunduh distribusi PHP yang Anda inginkan sebagai sumber, dan kemudian mengatur opsi parameter ./configure (lihat ./configure --help) sehingga perangkat lunak diinstal baik di lokasi yang unik atau dengan nomor versi yang ditambahkan ke namanya. Dengan cara ini ketika Anda menjalankan make install penginstalan PHP 5.3 tidak ditimpa. Terakhir, siapkan pemetaan sufiks terpisah di Apache yang merutekan ke versi yang Anda inginkan, atau ganti pemetaan PHP default untuk stanza VirtualHost atau Directory tempat skrip Anda berada.

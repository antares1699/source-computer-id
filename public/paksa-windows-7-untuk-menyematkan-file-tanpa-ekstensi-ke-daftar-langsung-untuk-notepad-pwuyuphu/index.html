OK saya bermain sedikit.

Jika Anda memberi pengguna Anda akses tulis ke file HOSTS (berbahaya jika malware berjalan seperti Anda), Anda kemudian dapat membuat tautan keras ke file (sebagai admin) menggunakan ekstensi .txt yang dapat Anda edit.

Buka prompt cmd sebagai Administrator dan gunakan FSUtil untuk membuat tautan keras di folder yang Anda miliki akses penuhnya (seperti desktop Anda):

fsutil hardlink create C:\Users\%USERNAME%\Desktop\PinnableHosts.txt C:\Windows\System32\drivers\etc\hosts.

Anda kemudian dapat membuka "PinnableHosts.txt" di Notepad dan ketika Anda mengeditnya, perubahan akan tercermin dalam file HOSTS Anda (dan sebaliknya, karena mereka ditautkan).

Ini akan (harus) kemudian berada di 'daftar terbaru' untuk Notepad, dan oleh karena itu tersedia untuk disematkan.

HTH.

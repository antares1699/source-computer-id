Bagaimana cara beralih di antara beberapa rangkaian konfigurasi jaringan pada Windows XP?

Saya melakukan konsultasi dan membawa laptop Windows XP saya dari majikan ke majikan. Di beberapa tempat yang saya kunjungi, mereka memiliki pengaturan jaringan yang unik, jadi terkadang saya harus duduk di sana dan mengubah pengaturan jaringan saya (misalnya ketik server DNS, nilai IP, dll ...).

Tentu saja, ketika saya bekerja di perusahaan lain, saya harus mengatur ulang pengaturan kembali ke aslinya.

Adakah cara untuk menyimpan berbagai pengaturan ini dan mengubahnya sesuai kebutuhan dalam satu gerakan?

Menggunakan netsh.exe bersama dengan beberapa file batch, dan Anda siap melakukannya! Bukan hanya itu Gratis , itu disertakan dengan Windows , dan Anda tidak memerlukan apa pun yang terus berjalan di latar belakang. Cukup tetapkan beberapa file konfigurasi sebagai file berbeda, dan jalankan file lain untuk mengganti pengaturan jaringan Anda!

Misalnya, jika saya ingin mengubah "Local Area Connection" saya (nama yang ditampilkan di Network Connections), maka saya akan menyalin dan menempel yang berikut ini ke Notepad, dan menyimpannya sebagai "sesuatu.bat":

netsh interface ip set address name="Local Area Connection" static 192.168.0.100 255.255.255.0 192.168.0.1 1

Ini akan mengatur konektor jaringan saya bernama "Local Area Connection" untuk memiliki IP statis 192.168.0.100, dengan netmask 255.255.255.0, dan gateway default 192.168.0.1.

Untuk beberapa contoh lanjutan, lihat artikel ini , atau dokumentasi dari Windows XP Professional .

Dan ya, Anda juga dapat mengatur DNS melalui alat ini:

netsh interface ip set dns "Local Area Connection" static 192.168.0.200

Kenapa saya tidak bisa mengakses video Youtube di Chrome di Ubuntu?

Ketika saya mencoba menonton video Youtube di Google Chrome pada Ubuntu Quantal Quetzal, saya mendapatkan pesan "Tidak dapat memuat plugin" berikut. pesan.

Saya mencoba membuka "Wrench" -> Tools -> Task Manager di Google Chrome, menutup proses Shockwave, dan memuat ulang halaman dengan video Youtube, tetapi masalah tetap ada.

Apakah saya kehilangan plugin? Apa yang salah?

=====

Memperbarui :

Terimakasih semuanya! Kamu luar biasa.

Setelah beberapa saat saya akhirnya memecahkan masalah tersebut. Setelah membuka chrome://plugins, saya menemukan bahwa saya memiliki plugin misterius bernama libpepflashplayer. Saya menonaktifkannya, dan sekarang, saya dapat memutar video Youtube. Beginilah tampilan chrome://plugins saya sekarang.

Namun, apakah ini cara yang baik untuk menyelesaikan masalah? Apakah ada masalah yang lebih dalam yang mungkin saya alami? Seperti browser Chrome saya tidak diperbarui dengan benar?

Saya menggunakan versi yang berbeda tapi mungkin ini membantu Anda.

Saya mengalami masalah ini dengan saluran dev terbaru. Version 26.0.1397.2 dev

Saya memperbaikinya dengan menghapus folder PepperFlash di .config/google-chrome/ dan .config/google-chrome/Default/Pepper\ Data/, entah kenapa ada salinan file .so untuk plugin pepper flash, yang tidak terlalu dibutuhkan karena ada salinan asli di /opt/google/chrome/PepperFlash/ yang harus dimuat juga yang mungkin menyebabkan konflik atau sesuatu.

Pastikan untuk mencadangkan seluruh folder .config/google-chrome/ Anda untuk berjaga-jaga.

Saya tidak lagi menggunakan paket flash mandiri, hanya versi flash lada yang disertakan dengan Chrome, kalau-kalau Anda bertanya-tanya.

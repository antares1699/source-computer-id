Saya akan merekomendasikan untuk menggunakan yang berikut ini:

untuk sftp:

netstat -nap | grep -e ":22"

untuk ftp

netstat -nap | grep -e ":21"

Kedua perintah ini akan memberi Anda program yang mendengarkan pada port 22 (ssh) dan 21 (ftp)

Tebakan terbaik saya adalah dalam kasus Anda ini akan menjadi dropbear atau openssh mendengarkan pada port 22

Jika Anda mencari layanan sftp, maka vsftpd kemungkinan besar tidak ada hubungannya dengan itu

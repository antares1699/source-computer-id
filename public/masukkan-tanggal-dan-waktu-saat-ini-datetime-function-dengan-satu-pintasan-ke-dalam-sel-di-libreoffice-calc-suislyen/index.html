Masukkan tanggal dan waktu saat ini (DATETIME-function) dengan satu pintasan ke dalam sel di LibreOffice Calc

Saya tahu ada CTRL + ; pintasan untuk memasukkan tanggal saat ini dan CTRL + BERGESER + ; untuk memasukkan waktu saat ini.

Namun saya memiliki dua masalah dengan itu:

1) Saya ingin memiliki satu jalan pintas untuk keduanya

2) Saya ingin dapat menggunakan format tanggal-waktu khusus (YYYY-MM-DD HH:MM:SS)

Format default bahasa saya adalah MM/DD/YY HH:MM:SS pm/am - saya lakukan tidak ingin mengubah ini. Saya ingin menggunakan format ubahsuaian khusus untuk pintasan itu, sebaiknya tanpa solusi yang melibatkan xdotool atau perangkat lunak makro eksternal serupa atau pintasan seluruh sistem global.

Fungsi-fungsi dalam Tools -> Customize -> Keyboard tampaknya tidak menawarkan bantuan apa pun.

( Mengapa saya tidak ingin menggunakan xdotool ; Solusi langsung di dalam LibreOffice akan menjadi yang terbaik.)

Saya menemukan kode makro OpenOffice berikut sini , tetapi dikatakan hanya berfungsi dengan dokumen Penulis. Bagaimana saya memodifikasi makro ini untuk memasukkan DATE-TIME yang telah diformat ke dalam sel yang saat ini dipilih di Calc?

'Author: Andrew Pitonyak
'email:   andrew@pitonyak.org 
'uses:  FindCreateNumberFormatStyle
Sub InsertDateField
  Dim oDoc
  Dim oText
  Dim oVCurs
  Dim oTCurs
  Dim oDateTime
  Dim s$

  oDoc = ThisComponent
  If oDoc.SupportsService("com.sun.star.text.TextDocument") Then
    oText = oDoc.Text
    oVCurs = oDoc.CurrentController.getViewCursor()
    oTCurs = oText.createTextCursorByRange(oVCurs.getStart())
    oText.insertString(oTCurs, "Today is ", FALSE)
    ' Create the DateTime type.
    s = "com.sun.star.text.TextField.DateTime"
    ODateTime = oDoc.createInstance(s)
    oDateTime.IsFixed = TRUE
    oDateTime.NumberFormat = FindCreateNumberFormatStyle(_
      "DD. MMMM YYYY", oDoc)

    oText.insertTextContent(oTCurs,oDateTime,FALSE)
    oText.insertString(oTCurs," ",FALSE)
  Else
    MsgBox "Sorry, this macro requires a TextDocument"
  End If
End Sub

Ubah format sel sebelum pintasan adalah solusi paling sederhana?

Format sel setelah pintasan tidak berfungsi.

Cara makro ...

Sub Main

    Dim Doc As Object
    Dim Sheet As Object
    Dim Cell As Object
    Dim NumberFormats As Object
    Dim NumberFormatString As String
    Dim NumberFormatId As Long
    Dim LocalSettings As New com.sun.star.lang.Locale

    Doc = ThisComponent
    Sheet = Doc.Sheets(0)
    Cell = Doc.getCurrentSelection
    Column = Cell.CellAddress.Column
    Row = Cell.CellAddress.Row

    Cell.Value = Now()

    LocalSettings.Language = "en"
    LocalSettings.Country = "us"

    NumberFormats = Doc.NumberFormats
    NumberFormatString = "YYYY-MM-DD HH:MM:SS"

    NumberFormatId = NumberFormats.queryKey(NumberFormatString, LocalSettings, True)
    If NumberFormatId = -1 Then
    NumberFormatId = NumberFormats.addNew(NumberFormatString, LocalSettings)
    End If

    MsgBox NumberFormatId
    Cell.NumberFormat = NumberFormatId

End Sub

Untuk Excel (diuji untuk MS Excel 2010 untuk Windows): memasukkannya ke dalam satu sel tanpa makro atau perangkat lunak tambahan, satu-satunya solusi adalah menggunakan 3 pintasan setelah yang lain:

  • CTRL + ;
  • ruang
  • CTRL + BERGESER + ;

Setidaknya ini akan memberi Anda DD-MM-YYYY HH:MM

Dari sana Anda hanya perlu mengubah format yang ditentukan pengguna ke pilihan Anda: YYYY-MM-DD HH:MM:SS

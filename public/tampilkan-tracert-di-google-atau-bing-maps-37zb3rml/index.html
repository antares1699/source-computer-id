Tampilkan tracert di Google atau Bing Maps

Apakah ada program dari situs web yang dapat menampilkan daftar alamat ip di peta Google atau Bing?

Misalnya, jika saya melakukan tracert -d www.youtube.com, saya akan mendapatkan banyak alamat IP, menampilkan jalur yang ditempuh permintaan saya.

Menurut saya lucu melihat ini di peta, tetapi saya merasa sulit untuk memasukkan semua alamat IP ini secara manual. Adakah yang tahu program yang bisa melakukan ini?

Jejak Visual bekerja juga:

Alat penelusuran visual menampilkan jalur yang dilintasi paket Internet mencapai tujuan yang ditentukan. Alat tersebut bekerja dengan mengidentifikasi IP alamat setiap hop di sepanjang jalan ke jaringan tujuan alamat. Untuk mendapatkan info lebih spesifik tentang alamat IP yang bisa Anda gunakan alat pelacak IP ini dari IP-Tracker.org. Estimasi geofisika lokasi setiap hop diidentifikasi menggunakan database GeoIP MaxMind. Setelah semua lokasi hop 'diidentifikasi, jalur ke tujuan diplot di Google Map.

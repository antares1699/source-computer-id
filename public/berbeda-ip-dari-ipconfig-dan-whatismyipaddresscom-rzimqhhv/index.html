Ini bukan pertanyaan untuk http://security.stackexchange.com .

Jika Anda melakukan sedikit riset tentang IP, Anda akan segera melihat bahwa rentang 192.168.0.0/16 dicadangkan untuk jaringan pribadi . Artinya ini bukan alamat yang bisa dilihat dari "luar".

IP yang ditampilkan oleh situs web sesuai dengan IP titik akhir koneksi Anda. Biasanya ini adalah IP yang diberikan oleh ISP Anda ke modem / router Anda saat terhubung ke internet.

Router kemudian bebas untuk mengirimkan lalu lintas ke satu set alamat IP pribadi untuk memungkinkan banyak komputer menggunakan IP publik yang sama.

Alamat 192.168 adalah alamat IP mesin Anda terkait dengan Anda lokal jaringan. Misalnya, jika saya memiliki PC, perangkat seluler, dan konsol yang terhubung ke router saya, PC saya mungkin 192.168.0.1, perangkat seluler saya mungkin 192.168.0.2 dan konsol saya menjadi 192.168.0.3 dan seterusnya.

Alamat IP lokal Anda hanya penting untuk router Anda sendiri. Jika Anda pergi ke rumah orang lain, Anda mungkin akan menemukan bahwa alamat IP lokal mereka juga dimulai dengan 192.168 dimulai dengan 192.168.0.1 dan naik satu per satu untuk setiap perangkat yang terhubung ke router.

Di sisi lain, alamat IP lain yang Anda berikan mungkin jauh lebih besar dari 0,1 atau 0,2, karena itulah alamat IP yang terkait dengan ISP Anda. Beginilah cara situs web lain melihat alamat IP Anda - mereka tidak peduli dengan alamat IP lokal Anda yang hanya penting untuk router Anda, mereka ingin tahu ke ISP mana konten harus dikirimkan, dan pengguna ISP mana, secara lebih spesifik.

Tidak dapat mengatur atau menghapus PIN masuk pada Windows 10

Setelah komputer saya memutuskan untuk menjalankan beberapa pembaruan, saya tidak dapat masuk dengan PIN saya. Saya harus melakukan reset kata sandi. Saya mencoba mengaktifkan kembali PIN, tetapi saya mengalami kesulitan.

Saya berada di Pengaturan> Akun? Opsi masuk. Di bawah bagian PIN, saya akan melihat opsi untuk mengubah atau mengatur ulang PIN saya. Namun, opsi itu tidak muncul. Sebagai gantinya, ada tombol berlabel Tambah. Lihat tangkapan layar ini:

Saat mengklik tombol Add, saya mendapatkan dialog berikut:

Tidak peduli apa yang saya masukkan ke dalam kotak, itu menghilang setelah saya memasukkan jumlah karakter yang sama dengan panjang PIN yang saya gunakan, terlepas dari karakter mana yang saya masukkan. Saat dialog ditutup, tidak ada lagi yang terjadi.

Bagaimana cara mengatur PIN?

Setelah tidak menerima balasan di sini, saya mengajukan pertanyaan ini terus menerus Situs dukungan Microsoft . Saya menyalin balasan yang saya dapatkan di sana, dengan satu modifikasi yang harus saya buat agar berfungsi.

Langkah # 1: Hapus folder Ngc.

  1. Gunakan kata sandi Anda untuk masuk ke Windows 10.
  2. Buka File Explorer dan tempel ini di kotak URL: C:\Windows\ServiceProfiles\LocalService\AppData\Local\Microsoft.
  3. Klik kanan folder Ngc dan pilih Properties.
  4. Arahkan ke tab Keamanan dan klik tombol Lanjutan.
  5. Cari bagian Pemilik di atas dan klik Ubah tautan.
  6. Masukkan nama objek untuk memilih masukkan Administrator, jika Anda menggunakan akun administrator, atau nama pengguna Anda dan klik Periksa Nama dan klik Oke.
  7. Centang Ganti pemilik pada subkontainer dan objek dan klik Terapkan dan OK. Sekarang Anda dapat mengakses folder Ngc.
  8. Buka folder Ngc dan lakukan langkah 3-7 di setiap subfolder.
  9. Buka folder Ngc, pilih semua file dan folder di dalamnya dan hapus.

Langkah # 2: Mulai buat PIN baru.

Setelah Anda menghapus semuanya dari folder Ngc, Anda dapat membuat PIN baru untuk perangkat Windows 10 Anda dengan melakukan langkah-langkah berikut:

  1. Buka aplikasi Pengaturan, lalu Pengaturan Akun.
  2. Pilih Opsi Masuk dan klik Tambahkan PIN.
  3. Ikuti instruksi untuk menambahkan PIN baru.
  4. Uji PIN Anda menggunakan Command Prompt (Admin).
  5. Jika Anda tidak dapat menemukan Admin Command Prompt, Anda dapat mengaktifkannya menggunakan Command Prompt dan mengetik net user administrator /active:yes
  6. Setelah mengaktifkan akun administrator internal, Anda dapat masuk dan mencoba mengatur ulang kata sandi akun yang tidak berfungsi. Anda dapat menonaktifkannya nanti setelah selesai dengan memasukkan net user administrator /active:no di Command Prompt. Anda juga dapat menggunakan ini jika Anda ingin mengubah kata sandi atau PIN di Akun Pengguna lain.

Catatan: Saya tidak tahu apakah boleh menyalin jawaban ini seperti yang saya lakukan, bahkan dengan satu tambahan saya. Jika tidak, silakan edit jawaban ini dan cukup tautkan ke solusinya.

berikut adalah versi PowerShell dari jawaban Scott yang berhasil untuk saya.

setelah menjalankan ini sebagai admin, saya tidak dapat membuat PIN sampai komputer pertama kali dihidupkan ulang

$passportFolder = 'C:\Windows\ServiceProfiles\LocalService\AppData\Local\Microsoft\Ngc'

if (Test-Path $passportFolder)
{
    takeown /f $passportFolder /r /d y
    icacls $passportFolder /reset /t /c /l /q
    
    Remove-Item $passportFolder –recurse -force
}

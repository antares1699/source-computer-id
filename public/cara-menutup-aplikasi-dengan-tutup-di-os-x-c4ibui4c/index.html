Cara menutup aplikasi dengan tutup di OS X

Saya memiliki MacBook Air. Saya sering menggunakan klien RDP Microsoft untuk menyambung ke workstation virtual. Saya perlu sesi RDP ini ditutup secara otomatis (keluar dari RDP) ketika saya menutup penutupnya, sebelum OS X tidur.

Bagaimana fungsi ini dapat dicapai?

1 Jawaban

8
8

Ada beberapa alat yang memungkinkan Anda untuk menghalangi perubahan status daya OS X:

  • Skenario dan Manajer Daya , baik alat komersial maupun GUI.
  • Sleep Watcher , gratis dan open source, tetapi baris perintah.

Instalasi Sleep Watcher

Mari kita coba dengan Sleep Watcher. Unduh alat dari beranda Bernhard Baehr dan ikuti petunjuk instalasi: dengan asumsi Anda membongkar unduhan ke Desktop Anda, panggil perintah berikut dari jendela Terminal:

sudo mkdir -p /usr/local/sbin /usr/local/share/man/man8
sudo cp ~/Desktop/sleepwatcher_2.2/sleepwatcher /usr/local/sbin
sudo cp ~/Desktop/sleepwatcher_2.2/sleepwatcher.8 /usr/local/share/man/man8

Ini akan menginstal biner sleepwatcher dan halaman manualnya. Anda akan dimintai kata sandi administrator.

Membuat skrip tidur

Sekarang, mari kita buat skrip tidur bernama ~/.sleep. Ini akan berisi perintah yang dijalankan saat mesin dalam mode tidur (yaitu saat tutupnya tertutup).

touch ~/.sleep
open -e !$

Di file ini, tambahkan yang berikut ini:

osascript -e 'tell application "Remote Desktop Connection" to quit'

Ini akan keluar dari aplikasi Microsoft RDP melalui perintah AppleScript. Anda dapat melakukan ini dengan aplikasi apa pun selama Anda mengubah namanya sesuai dengan judulnya. Simpan file. Sekarang kita perlu membuatnya dapat dieksekusi:

chmod +x ~/.sleep

Menguji skrip tidur

Sekarang uji skrip Anda dengan terlebih dahulu memulai program RDP, kemudian jalankan yang berikut ini di Terminal:

/usr/local/sbin/sleepwatcher --verbose --sleep ~/.sleep

Ini tidak akan menghasilkan apa pun, jadi biarkan RDP dan Sleep Watcher berjalan, lalu tutup MacBook Anda. Tunggu beberapa detik hingga benar-benar tertidur. Lalu bangunkan kembali. Apakah RDP ditutup? Bagus! Apakah ada yang salah? Lihat keluaran dari perintah sleepwatcher untuk mendapatkan petunjuk tentang di mana ia gagal.

Sekarang Anda dapat membatalkan Sleep Watcher dengan menekan Ctrl C di Terminal.

Menjalankan skrip di latar belakang

Untuk menjalankan Sleep Watcher setiap saat, Anda harus menjadikannya LaunchDaemon. Buat file konfigurasi baru dengan menyalin konfigurasi contoh dan skrip rc.

sudo cp ~/Desktop/sleepwatcher_2.2/config/de.bernhard-baehr.sleepwatcher-20compatibility.plist /Library/LaunchDaemons/de.bernhard-baehr.sleepwatcher.plist
sudo cp ~/Desktop/sleepwatcher_2.2/config/rc.* /etc

Sekarang muat file konfigurasi dengan launchd:

sudo launchctl load /Library/LaunchDaemons/de.bernhard-baehr.sleepwatcher.plist

Hanya itu yang Anda butuhkan. Sekarang file .sleep akan selalu dijalankan setiap kali Anda tidur di Mac Anda.

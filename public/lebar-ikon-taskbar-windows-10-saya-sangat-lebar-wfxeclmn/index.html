Lebar ikon taskbar windows 10 saya sangat lebar

Saya tidak tahu mengapa lebar ikon pada taskbar windows 10 saya menjadi sangat lebar.

Saya mencoba "Gunakan tombol bilah tugas kecil".

Dan "Skala dan tata letak" saya 100%, saya tidak dapat menyetel ulang ke normal.

Ada yang tahu cara memperbaikinya?

Memperbarui Saya telah menemukan cara mematikan ini!

  1. Buka Pengaturan (mis. [Win] + I), lalu Sistem, lalu Tablet; atau tekan [Win] dan ketik "Pengaturan tablet" untuk langsung ke sana.
  2. Klik "Ubah setelan tablet tambahan"
  3. Di bawah "Saat saya tidak dalam mode tablet", setel "Jadikan ikon aplikasi di bilah tugas lebih mudah disentuh" ​​ke nonaktif

Jawaban asli

Apakah Anda menggunakan PC 2-in-1 dalam mode "tenda" (mungkin dengan keyboard dan mouse eksternal)?

Menurut WinCentral , Tampaknya ini adalah "fitur" baru di versi 2004 (alias 20H1) untuk PC 2-in-1 agar lebih mudah menggunakan layar sentuh saat tidak digunakan sebagai laptop "normal".

Saya khawatir saya belum menemukan cara mematikannya, tetapi SANGAT mengganggu!

Satu-satunya solusi yang dapat saya temukan adalah membalik laptop Anda dan menggunakannya sebagai laptop konvensional - ikon kembali ke ukuran normal.

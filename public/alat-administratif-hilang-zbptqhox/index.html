Saya belum pernah melihat mereka dihapus. Buka panel kontrol dan cari "Alat administratif".

Jika Anda tidak dapat menemukannya, Anda dapat membuat ulang pintasan melalui berikut ini (ini adalah daftar lengkap semuanya, bukan hanya Windows XP):

Jika Anda hanya menginginkan Windows XP default yang, ini dia (saya hanya menyertakan yang valid dalam alat administratif, misalnya, sementara "Pengelola Perangkat" dapat diakses sendiri, ini adalah bagian dari "Manajemen Komputer", jadi saya tidak akan mencantumkannya):

Event Viewer  eventvwr.msc
Internet Information Services  iis.msc (If you have it installed)
Performance  perfmon.msc

Task Scheduler tasksched.msc
Printer Management printmanagement.msc

Ekstra:

System Configuration - %windir%\system32\msconfig.exe
Windows Memory Diagnostics (Vista + 7?) - %windir%\system32\MdSched.exe
iSCSI Initiator - %windir%\system32\iscsicpl.exe
Data Sources (ODBC) - %windir%\system32\odbcad32.exe

Selain itu, jika tidak ditampilkan, Anda dapat melihat properti menu Mulai (klik kanan pada menu mulai, dan klik properti), lalu -

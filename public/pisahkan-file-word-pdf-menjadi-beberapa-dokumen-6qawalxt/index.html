Pisahkan file Word / PDF menjadi beberapa dokumen?

Saya membuat desain kartu nama di Word, setiap sisi kartu sebagai satu halaman 85x55mm. Sebagian besar printer tampaknya menginginkannya dalam file PDF yang mudah, tetapi terkadang mereka menginginkan setiap sisi sebagai dokumen yang berbeda.

Apakah mungkin untuk membagi file kata atau PDF menjadi beberapa dokumen, mis., Dipisahkan pada jeda halaman? Saya tidak dapat menemukan opsi di Word 2007 dan tidak tahu tentang Acrobat tetapi saya meragukannya.

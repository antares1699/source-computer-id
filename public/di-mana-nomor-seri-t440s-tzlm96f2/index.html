Di mana nomor seri T440s?

Saya membutuhkan jenis mesin dan nomor seri T440s saya. Saya tidak dapat menemukannya di bagian bawah laptop. Apakah ini normal? Saya ingat itu terlihat jelas di T420 lama saya.

Dalam hal ini Anda mungkin menghargai pendekatan ini (dijalankan dari baris perintah):

wmic.exe csproduct list full

Ini memberi saya output ini pada X200:

Description=Computer System Product 
IdentifyingNumber=S3R1AL
Name=xxxxxx
SKUNumber=
UUID=-removed-
Vendor=LENOVO 
Version=ThinkPad X200

Pertanyaan forumula spreadsheet untuk memilih DVD acak

Saya memiliki daftar (279) DVD saya di OpenOffice Spreadsheet (versi 3.3.0) di laptop Windows 7 saya dan saya ingin membuat rumus untuk memilih film secara acak untuk ditonton (untuk saat-saat ketika saya tidak bisa pikiran). Ini bekerja:

=RANDBETWEEN(1;279)

Tetapi yang diberikan hanyalah angka yang harus saya gulir ke atas dan ke bawah untuk memeriksa dan juga memiliki jumlah total 'kode keras' DVD. Saya ingin menampilkan nama filmnya (ada di kolom A) sehingga jika, misalnya, angkanya 277, nama filmnya akan muncul, dalam hal ini 'X-Men 2'. Saya berasumsi bahwa hanya menambahkan huruf kolom di depan seperti ini:

=A(RANDBETWEEN(1;279))

akan berhasil tetapi memunculkan kesalahan: #NAME?

Jadi, bagaimana saya bisa ...

  1. Buatlah agar menampilkan nama filmnya
  2. Deteksi nomor teratas secara otomatis saat saya menambahkan DVD baru, jadi saya tidak punya kode keras ke dalam rumus?
  3. Bagaimana saya bisa membuatnya jadi saya adil klik sel untuk membuat yang baru angka acak? (yaitu tidak harus hitung ulang secara manual dengan menekan Shift + Ctrl + F9)

Terima kasih.

  1. Untuk mengembalikan nilai dari kolom A, baris R , Anda dapat menggunakan =INDIRECT("A"& R )

  2. Untuk menghitung jumlah nilai di kolom A, Anda dapat menggunakan =COUNTA(A1:A65536)

    Jadi untuk kasus Anda, Anda dapat menggunakan =INDIRECT("A"&RANDBETWEEN(1;COUNTA(A1:A65536)))

  3. Anda mungkin bisa membuat tombol makro yang akan menghitung ulang untuk Anda. Saya tidak cukup tahu tentang bahasa makro OpenOffice.org Calc untuk memberikan lebih banyak bantuan di sana.

Saya tidak cukup paham dengan internal OpenOffice untuk membantu permintaan ketiga Anda, selain mengatakan bahwa F9 saja yang berfungsi untuk saya, tidak perlu shift atau kontrol. Adapun dua lainnya, meskipun:

=INDIRECT(ADDRESS(RANDBETWEEN(1,MATCH("*",$A$1:$A$50000,-1)),1))

Gantilah 50000 dengan sesuatu yang lebih besar jika Anda memiliki atau berpikir Anda akan memiliki lebih dari 50000 DVD, dan ganti 1 dalam fungsi ADDRESS() jika berada di kolom selain A.

  • Letakkan rumus dari jawaban Bavi_H di sel B1 (=INDIRECT("A"&RANDBETWEEN(1;COUNTA(A1:A65536))))
  • Buat lebar dan tinggi B1 bagus dan besar
  • Format B1 dengan font yang menarik dalam ukuran besar dipilih agar judul film yang panjang tetap muat
  • Pilih sel mana saja di kolom A dan pilih item menu Format > Column > Hide
  • Pilih item menu Tools > Macros > Record Macros
  • Tekan F9 (atau Ctrl-Shift-F9)
  • Klik pada tombol Stop Recording
  • Di Basic Macros Dialog yang muncul, ketik nama seperti "refresh" di kolom Macro name, pilih spreadsheet di kotak Save macro in, klik Save dan masukkan nama modul seperti "film", klik OK
  • Pilih item menu View > Toolbars > Form Controls
  • Di toolbar tersebut, pastikan tombol Design Mode dihidupkan
  • Tekan tombol Form Design untuk membuat toolbar Desain Formulir terlihat
  • Dalam toolbar Stop Recording, matikan tombol Open in Design Mode
  • Pada toolbar Form Controls, klik tombol Label Field
  • Seret kursor dari sudut kiri atas sel B1 ke sudut kanan bawah untuk membuat bidang label yang ukurannya sama dengan sel
  • Klik kanan pada sel / label dan pilih item Control... dari menu konteks
  • Pilih tab General, lalu hapus teks dari bidang Label
  • Pilih tab Events, klik tombol ...
  • Di kotak dialog Assign Action, klik tombol Assign Action
  • Di kotak dialog Macro Selector, perluas entri di bawah spreadsheet Anda di kotak Library sampai Anda melihat nama modul yang Anda gunakan sebelumnya (mis. "Film") dan klik di atasnya
  • Di kotak Macro name, pilih makro yang Anda buat sebelumnya (mis. "Segarkan")
  • Klik OK, klik OK lalu tutup kotak Properties dengan mengklik tombol X di pojok kanan atas
  • Di toolbar Form Controls, matikan Design Mode
  • Sekarang ketika Anda mengklik sel B1, nama film yang ditampilkan di sana akan berubah
  • Tutup bilah alat jika Anda mau
  • Sebelum Anda menyimpan spreadsheet, Anda dapat masuk ke Tools > Options > OpenOffice.org Calc > View dan mematikan Window: Column Headers and Rows, Horizontal Scroll Bars, Vertical Scroll Bars dan Sheet tabs untuk tampilan yang sangat bersih (Anda mungkin perlu mengaktifkannya kembali saat Anda membuat spreadsheet kosong baru)

Untuk memudahkan, beri nomor DVD di kolom A dan cantumkan nama DVD di kolom B (Anda dapat menggunakan rumus A2 = A1 + 1 menjadi nomor jika ingin lebih mudah).

Asumsikan n = jumlah baris di atas tempat DVD dimulai, seperti jika Anda memiliki label kolom.

=INDEX(B:B, RANDBETWEEN(1, MAX(A:A)) + n, 1)

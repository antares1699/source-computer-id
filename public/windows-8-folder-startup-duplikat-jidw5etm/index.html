Windows 8: Folder startup? [duplikat]

Kemungkinan Duplikat:
Di mana folder startup (autostart) di Windows 8

Di versi Windows sebelumnya, ada folder bernama "Startup" yang terletak di menu Start tempat Anda dapat menyeret pintasan ke program yang ingin Anda mulai saat Windows pertama kali melakukan booting.

Apakah ada yang setara dengan ini di Windows 8?

4
Nick — Sumber

1 Jawaban

8
8

Ini dapat ditemukan per pengguna di bawah:

C:\users\%username%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

atau secara global di bawah:

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp

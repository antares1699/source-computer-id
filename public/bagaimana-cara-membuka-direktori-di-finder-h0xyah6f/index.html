Bagaimana cara membuka direktori di Finder?

Saya datang dari jendela di mana ada tombol yang ditunjuk untuk melakukan ini. Cukup klik dan Anda dapat membuka direktori "ke atas".

Saya tidak dapat menemukan cara untuk melakukan ini di macOS menggunakan Finder.

Saya menjalankan macOS - Mojave.

Saya menggunakan Finder 10.14.4.

Jika Anda membuka jendela Finder, di dekat bagian bawah adalah daftar struktur direktori jendela. Lihat gambar di bawah ini.

Klik salah satu folder / direktori dalam daftar dan Finder akan pergi ke lokasi itu.

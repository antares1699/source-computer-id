Bergantung pada kebutuhan Anda, Anda dapat mempertimbangkan Google Sketchup, tampaknya beberapa orang telah berhasil menjalankannya menggunakan Wine. http://wiki.winehq.org/GoogleSketchup

Sudah beberapa tahun sejak saya menulis perangkat lunak untuk AutoCAD tetapi saya selalu mendapat kesan bahwa itu adalah standar pasar, dengan kemungkinan pengecualian Pekerjaan Padat yang juga memiliki pangsa pasar.

Jika Anda harus mengirim gambar ke orang lain atau menerima gambar mereka, kemungkinan besar Anda akan menemukan bahwa mereka akan menggunakan file DWG (AutoCAD) dan Anda memerlukan fasilitas untuk melihat jika tidak mengeditnya.

Itu Aliansi Desain Terbuka memiliki alat untuk melihat dan mengedit file DWG, tetapi saya tidak yakin tentang tingkat dukungan Linux mereka.

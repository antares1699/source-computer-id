Coba boot ke Safe Mode. Apa masih disana? Pindai komputer Anda dengan file bersih salinan perangkat lunak antivirus. (sebaiknya diunduh dari internet saat Anda berada dalam Mode Aman).
Jika mendeteksi itu sebagai malware yang dikenal, maka itu harus membereskannya. Jika tidak, bahkan jika itu tidak mendeteksinya, coba hapus kembali file tersebut dan kemudian restart dan lihat apakah itu masih berjalan.

EDIT: sudahlah, jika mengaitkan OS bahkan Safe Mode tidak akan melakukan apa-apa. Satu-satunya hal yang harus dilakukan adalah menggunakan alat pemindaian rootkit (seperti Wakil ) atau gambar ulang sepenuhnya dengan salinan bersih Windows.

Hal lain yang dapat Anda coba adalah mem-boot ke Live CD salah satu dari berbagai versi Linux untuk menghapus file, karena alasan Anda tidak dapat menghapusnya sekarang adalah (kemungkinan besar) karena kait OS, sehingga dapat mendeteksi saat dihapus dan pulihkan sendiri.

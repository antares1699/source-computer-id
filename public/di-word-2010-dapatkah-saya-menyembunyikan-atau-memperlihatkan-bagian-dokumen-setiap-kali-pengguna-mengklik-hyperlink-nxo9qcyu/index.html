Di Word 2010, dapatkah saya menyembunyikan atau memperlihatkan bagian dokumen setiap kali pengguna mengklik hyperlink?

Saya mencoba membuat dokumen yang sebagian besar terdiri dari daftar.

Setiap item daftar meringkas isi dari spreadsheet Excel yang ditautkan yang saat ini dilampirkan di akhir dokumen; yang ingin saya lakukan adalah menyalin setiap bagian spreadsheet dan menempelkannya di bawah item daftar yang relevan dan menyembunyikan atau menampilkannya berdasarkan klik hyperlink, sehingga saya dapat mendistribusikan dokumen ini ke pengguna individu yang hanya dapat menampilkan bagian yang relevan dengan departemen mereka dan mencetak versi terkustomisasi dari total laporan.

Apakah ini mungkin, atau apakah saya mencoba menggunakan Word untuk sesuatu yang benar-benar saya perlukan untuk menggunakan HTML / CSS / Javascript?

Saya telah melakukan dokumen jenis ini untuk perusahaan asuransi besar. Kami melakukannya dengan membuat formulir Word kustom.

Pada formulir Anda membuat beberapa mekanisme bagi pengguna untuk memilih bagian dokumen yang diinginkan. Kotak daftar atau serangkaian kotak centang akan dilakukan. Saat pengguna mengisi pilihan mereka, mereka mengklik tombol di formulir.

Selanjutnya Anda harus menulis beberapa kode VBA. Kode tersebut, membuat dokumen secara dinamis berdasarkan pilihan mereka atau menyembunyikan bagian yang tidak diinginkan. Anda dapat menambahkan bookmark ke dokumen untuk setiap segmen, lalu menemukan bookmark di dokumen, lalu mengatur font menjadi tersembunyi.

Setidaknya 8 tahun yang lalu saya melakukan ini. Hari ini saya mungkin akan menggunakan Visual Studio 2010 dan membuat dokumen Excel menggunakan Office tmplates.

Saya kira Anda harus menulis beberapa kode VBA atau .NET untuk memecahkan masalah ini.

Ini mungkin mungkin dengan VBA, namun saya meragukannya.

MS Word adalah editor dokumen WYSIWYG dan tidak dimaksudkan untuk memiliki tampilan data yang dapat diubah secara dinamis. Anda benar bahwa solusi berbasis web lebih cocok untuk ini. Anda dapat membuat halaman web / situs kustom menggunakan HTML, JS, dan CSS untuk melakukannya. Namun, ada banyak aplikasi web bawaan untuk ini. Ada banyak aplikasi wiki, basis pengetahuan, dll. Yang ada di luar sana yang gratis untuk digunakan. Situs ini gratis dan open source dan saya pikir akan lebih cocok untuk tugas daripada kata.

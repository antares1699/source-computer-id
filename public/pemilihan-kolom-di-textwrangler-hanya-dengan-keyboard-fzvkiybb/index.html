Pemilihan kolom di TextWrangler hanya dengan keyboard

Saya sadar bahwa saya dapat memilih kolom (atau blok, atau persegi panjang) teks di TextWrangler dengan Alt + seret mouse atau Bergeser + Alt + klik mouse. Kedua tindakan ini membutuhkan penggunaan mouse. Saya ingin tahu apakah mungkin melakukan ini hanya dengan keyboard.

Di MacOS

  1. Dengan Bergeser kunci ditekan,
  2. gunakan kiri / kanan arrow untuk menyorot beberapa teks pada baris saat ini.
  3. Lalu, masih memegang Bergeser kunci,
  4. tekan dan tahan Kontrol kunci dan gunakan atas / bawah arrow untuk memilih garis untuk persegi panjang.

catatan: jika atas / bawah Anda mencapai garis yang lebih pendek dari tepi kiri pilihan Anda, hal-hal aneh terjadi.

Untuk Mac, saya telah menggunakan Shift + alt / opsi .

Tempatkan kursor di sebelah kiri Karakter / kolom pertama yang ingin Anda pilih. tekan terus Shift + alt dan letakkan kursor di sebelah kanan karakter terakhir yang ingin Anda pilih (Anda dapat menggulir semua halaman ke bawah tetapi terus menahannya Shift + alt ). Selama Anda terus menahan Shift + alt , Anda dapat menyesuaikan teks yang dipilih.

Ini telah berhasil untuk saya sejauh ini.

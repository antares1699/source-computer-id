Berapa banyak sensor suhu yang dimiliki laptop? [Tutup]

Laptop teman saya tiba-tiba membeku, setelah dia reboot (mesin diberi kesempatan untuk sedikit dingin) dia meluncurkan aplikasi bernama SpeedFan dan itu memberi yang berikut:

Saya terkejut dengan jumlah pengukuran suhu yang ditunjukkan program, terakhir kali saya memeriksa (4-5 tahun yang lalu) saya hanya dapat melihat suhu CPU dengan aplikasi serupa.

Sesuatu yang dia katakan sangat menarik "Versi sebelumnya dari SpeedFan ini hanya menunjukkan 3 pengukuran suhu"

Jadi pertanyaannya adalah , apakah SpeedFan meningkatkan nilainya? Berapa banyak sensor suhu yang biasanya dimiliki komputer modern?

Laptop modern memiliki banyak sensor suhu bawaan.

Biasanya ditemukan:

  • 1 di CPU
  • 1 ekstra di setiap inti CPU (jadi untuk laptop biasa di zaman kita, hingga 4)
  • 1-2 di GPU (jika grafik eksternal)
  • 1-2 dalam chipset motherboard
  • 1 di setiap (hard) drive

Beberapa laptop menyertakan sensor tambahan:

  • di pipa panas
  • baterai
  • beberapa tempat dalam kasus ini
  • slot memori
  • pengendali serangan
  • Macbook Pro saya (model awal 08) bahkan mendapatkannya di kartu nirkabel

Versi Speedfan yang lebih baru (berlaku untuk perangkat lunak lain, juga) sering kali mempelajari cara membaca lebih banyak dan sensor baru, jadi menemukan lebih banyak sensor dengan pembaruan tidaklah mengherankan.

temp1 kemungkinan adalah sensor motherboard / case-temp. dan CPU biasanya rata-rata dari core0 + core1 atau mungkin sensor ketiga yang memeriksa seluruh suhu paket CPU.

Laptop saya (MacBook Pro 2011) memiliki sekitar 10 sensor suhu yang berbeda.

CPU Diode, CPU Digital Core 0, CPU Digital Core 1, CPU Proximity, Integrated Graphics chip, RAM, disk bay, (disk sebenarnya juga memiliki sensor), baterai, dan ada juga beberapa lagi yang tidak berhubungan dengan komponen tertentu (sensor case)

Suhu CPU secara rutin sekitar 90C, karena saya sering bermain game. Pemalasan, di OS X CPU Diode sekitar 40C biasanya, dan di bawah Windows sekitar 50C. (Saya tidak yakin mengapa, ini hanya berjalan lebih hangat di Windows. Saya bisa merasakan casingnya juga lebih hangat, jadi tidak mengukur sensor yang berbeda)

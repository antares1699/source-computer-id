Cara membatasi akses ke jaringan nirkabel tertentu hanya untuk 1 profil pengguna di Windows 7.

Halo semua,

Saya menggunakan Win7 SP1.

Saya memiliki banyak pengguna di laptop yang dapat / harus terhubung ke jaringan nirkabel, sebut saja Wireless1.

Saya punya jaringan nirkabel kedua, (sebut saja Wireless2), yang ingin saya batasi aksesnya hanya untuk pengguna admin laptop.

Sekarang saya dapat menghapus Wireless2 di manajer jaringan setiap saat, tetapi saya menginginkan solusi yang lebih ramah pengguna, sehingga hanya admin yang dapat terhubung ke Wireless2, dan semua pengguna lain tidak dapat (mereka mungkin melihat jaringan, tetapi harus memasukkan kata sandi, seperti semua jaringan lainnya).

Iya, Anda dapat membatasi koneksi jaringan nirkabel ke profil pengguna di Windows 7 . Ikuti langkah ini:

1) Hal pertama yang perlu Anda lakukan adalah menuju ke Jaringan dan Berbagi Pusat, yang dapat diakses melalui ikon baki ... Setelah Anda berada di sana, klik Kelola Jaringan Nirkabel. Sekali di sana, klik tombol Jenis profil.

2) Dan sekarang Anda dapat memilih "Gunakan profil semua pengguna dan per pengguna" pilihan.

3) Pada titik ini, sistem Anda sekarang dapat menggunakan profil per pengguna — tetapi apa saja profil saat ini masih disetel untuk semua pengguna. Teruskan membaca ...

4) Cara termudah untuk membuat nirkabel khusus pengguna baru koneksi hanya akan menyambung ke jaringan — defaultnya adalah per pengguna sekarang. Jika Anda ingin mengonversi yang sekarang, Anda harus terlebih dahulu Singkirkan. Cukup pilih dalam daftar di Kelola Jaringan Nirkabel layar, lalu klik tombol Hapus.

5) Sekarang Anda dapat menggunakan ikon nirkabel baki sistem untuk memilih jaringan, sambungkan, lalu masukkan kata sandi.

6) Jika Anda mengklik kanan entri dalam daftar dan memilih Properties, Anda akan melihat bahwa opsi "Ketersediaan jaringan" sekarang disetel ke "Saya hanya". Keberhasilan! Hal yang sama akan terjadi untuk jaringan apa pun yang Anda sambungkan untuk.

Apakah ada keuntungan menjalankan OS 64 bit dengan RAM 4GB? [duplikat]

Laptop saya memiliki 4 gigs RAM DDR2 dan saat ini saya menggunakan OS windows 7 32bit. Apakah saya akan melihat perbedaan kinerja jika saya meningkatkan ke 64bit atau apakah semua keunggulan X64 mulai dari RAM 5GB?

Anda akan mendapatkan tambahan memori 0,3-.7 ish GB tergantung pada perangkat keras lain seperti grafik.

Namun, sejujurnya, tidak banyak peningkatan pada x64 windows melalui x86 kecuali Anda benar-benar menggunakan aplikasi yang hanya tersedia dalam edisi 64-bit atau selalu menggunakan sekitar 90% penggunaan memori.

Ada banyak pembicaraan tentang 64 bit menjadi lebih stabil, namun, saya tidak pernah benar-benar memiliki masalah pada sistem 32-bit yang menurut saya adalah karena itu 32-bit.

Saya terkejut saat mengetahui bahwa Windows 32-bit pada CPU T9300 saya memuat 4 detik lebih cepat daripada Windows 64-bit dan saya melihat penurunan kecil lainnya di beberapa area ... Sebenarnya menggunakan Windows, saya menemukan sedikit perbedaan.

Pada akhirnya, sejujurnya saya tidak menganggap x64 itu buruk, perangkat lunak 32-bit berjalan dengan kecepatan yang hampir sama pada keduanya dan perangkat lunak yang dikompilasi 64-bit umumnya lebih cepat.

Saya pribadi menggunakan Windows x64, saya akan mengatakan jika Anda memiliki 4GB, itu adalah pilihan yang sulit (saya pribadi akan melakukannya), Jika Anda memiliki lebih dari 4GB, tidak perlu berpikir untuk menggunakan Windows x64.

Pertama-tama, ada lebih banyak manfaat untuk OS 64bit daripada hanya RAM. Sistem 64bit juga memiliki lebih banyak register cpu yang dapat dimanfaatkan oleh OS 64bit. Ada juga pengoptimalan lain.

Adapun RAM, jika Anda memeriksa sistem Anda, saya pikir Anda akan menemukan Anda tidak menggunakan semua RAM Anda sekarang. Perangkat lain di sistem Anda berbagi ruang alamat 4GB dengan RAM Anda.

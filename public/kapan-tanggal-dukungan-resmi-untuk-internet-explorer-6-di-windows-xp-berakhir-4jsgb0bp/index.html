Kapan tanggal dukungan resmi untuk Internet Explorer 6 di Windows XP berakhir?

Jika saya membaca dokumen tentang Kebijakan dukungan Windows Service Pack , dan halaman dukungan siklus hidup Internet Explorer tertentu serta Halaman Wikipedia Saya telah menyimpulkan bahwa:

Dukungan IE6 berakhir / diakhiri pada:

  • Windows 2000
    • Berakhir (tanggal tidak diketahui)
  • Windows XP
    • SP0 (RTM) Berakhir
      • Rumah: 30-Agustus-2003
      • Pro: 30-Sep-2004
    • SP1 Berakhir
      • Rumah: 11-Jul-2004
      • Pro: 11-Juli-2004
    • SP2
      • Beranda: 13-Jul-2010
      • Pro: 13 Juli 2010
    • SP3 (dirilis: 21 April 2008)
      • Rumah: ???
      • Pro: ???

Yang tidak jelas adalah Windows XP SP3 skenario.

Dalam istilah "manusia", kapan akhir dari dukungan untuk IE6 pada Windows XP SP3?

Saya menyadari ini tidak memaksa orang untuk mengupgrade ke browser yang lebih baru namun saya mencoba untuk mendapatkan kata "semi" resmi ketika IE6 pindah ke kategori "tidak didukung".

Saya tidak tertarik dengan jawaban filosofis, mis. "perusahaan besar tidak akan meningkatkan tetapi mereka akan mengharapkan dukungan hingga tahun 2017" ... Saya hanya ingin "jawaban yang jelas" dalam hal dukungan resmi Microsoft.

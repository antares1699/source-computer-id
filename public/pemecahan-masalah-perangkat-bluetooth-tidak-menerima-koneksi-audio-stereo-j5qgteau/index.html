Pemecahan masalah perangkat Bluetooth tidak menerima koneksi audio stereo

Saya baru saja membeli headphone berkemampuan Bluetooth "Hesh 2 Wireless" dari Skullcandy. Saya menghubungkannya ke komputer Windows saya dengan menambahkan perangkat bluetooth dan semuanya bekerja dengan baik.

Setelah memutuskan dan menghubungkan kembali saya mendapat pesan yang mengatakan perangkat bluetooth tidak menerima koneksi audio stereo.

Sekarang semuanya terdengar seperti omong kosong. Saya dapat memasangkan headphone ke ponsel Android saya tanpa masalah dan musiknya terdengar bagus.

Saya sudah mencoba melepas dan memasangkan kembali perangkat, tetapi kualitas suaranya masih jelek. Ada ide?

Saya melakukan beberapa langkah (mulai dari instalasi). Saya memiliki bluetooth USB inline untuk Windows 7 x64.

  1. menginstal perangkat lunak generik untuk bluetooth
  2. dicolokkan ke usb dongle dan menunggu sampai Windows menginstal driver yang hilang
  3. mengganti ponsel saya dalam mode berpasangan dan menambahkan perangkat di windows bluetooth manager .
  4. diklik kanan dan Suara / Perangkat yang dipilih. Jendela baru terbuka dan saya memilih tab pertama. Menemukan headphone saya dengan teks ringkasan (handsfree) dan mematikannya.

  1. restart PC dan mematikan PC
  2. ketika Windows mencapai desktop (dengan ikon) saya mengaktifkan headphone saya (Windows akan secara otomatis terhubung dengan telepon dengan STEREO). Jika tidak ada yang berubah (tetapi seharusnya terjadi) coba aktifkan STEREO secara manual di jendela Sounds / Devices.

Saya baru saja mengalami masalah ini sendiri di mana Windows hanya akan terhubung ke earbud bluetooth saya dalam bentuk mono, memanggil perangkat "[Nama perangkat] Hands-free". Saya tidak dapat menemukan cara untuk mengaktifkan stereo jenis apa pun, tetapi tampaknya itu mendeteksi bahwa stereo didukung, tetapi hanya memilikinya sebagai default. Oleh karena itu, versi stereo disembunyikan karena saya terputus darinya.

Jadi saya membuka Sound di Control Panel, mengklik kanan pada daftar di tab Playback, dan mencentang "Show Disconnected Devices". "[Nama perangkat] Stereo" kemudian muncul di daftar, yang saya klik kanan dan tekan "Hubungkan". Ini kemudian mengaturnya sebagai perangkat default dan berfungsi dengan baik sekarang.

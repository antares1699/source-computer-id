Tidak dapat mengubah pengaturan layar di Linux Mint (SettingsDaemon tidak disediakan oleh file .service apa pun)

Saya mencoba menyiapkan layar kedua di laptop saya. Saya memasang layar menggunakan port VGA dan pergi ke Preferensi -> Tampilan.

Layar terdeteksi dengan benar, tetapi ketika saya mencoba mengaktifkannya, kesalahan muncul:

Kegagalan aplikasi konfigurasi:% s (diterjemahkan dari kalimat Prancis dari tangkapan layar di bawah)

GDBus.Error: org.freedesktop.DBus.Error.ServiceUnknown: Nama org.cinnamon.SettingsDaemon tidak disediakan oleh file .service mana pun

Saya tidak tahu apakah itu dapat membantu:

~$ lspci | grep VGA
00:02.0 VGA compatible controller: Intel Corporation Haswell-ULT Integrated Graphics Controller (rev 09)

Saya tidak menemukan apa pun di Internet tentang pengaturan khusus ini ... Adakah yang tahu cara memperbaikinya? Terima kasih.

Saya menerima kesalahan ini karena cinnamon-settings-daemon tidak berjalan. Anda dapat memeriksa apakah itu berjalan karena perintah berikut akan mencantumkannya:

pgrep -fa cinnamon-settings-daemon

Jika cinnamon-settings-daemon muncul pada output dari perintah di atas, maka masalah Anda berbeda dengan saya. Jika tidak, jalankan dari terminal sebagai berikut:

cinnamon-settings-daemon &

Ini memungkinkan Anda untuk mengubah pengaturan layar.

Karena minat, cinnamon-settings-daemon tidak mulai masuk karena masalah ini . Melakukan hal berikut dan kemudian menjalankan cinnamon-settings-daemon memperbaikinya untuk saya:

rm -f ~/.config/monitors.xml{,.backup}

Utas lama tetapi saya memiliki masalah yang sama persis seperti yang tertulis. Saya memecahkan masalah dengan menghapus cinnamon-settings-daemon-xrandr.desktop dari .config/autostart dan memulai ulang.

rm -f ~/.config/autostart/cinnamon-settings-daemon-xrandr.desktop

Menghapus ~/.config/monitors.xml tidak berfungsi dalam contoh saya.

Linux Mint 19.2 Cinnamon (v4.2.4), Kernal 4.15.0-70-generik

Di Linux Mint deamon ini seharusnya dimulai secara otomatis saat startup.

Pergi ke Menu> Startup Applications.

Pastikan Cinnamon Settings Daemon - xrandr diaktifkan.

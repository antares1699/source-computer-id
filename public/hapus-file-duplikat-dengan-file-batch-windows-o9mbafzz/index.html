Hapus file duplikat dengan file batch Windows

Saya memiliki program yang secara otomatis menyalin file ke direktori, dan jika itu membuat duplikat, itu akan menamainya seperti ini:

file with duplicate.xxx
file with duplicate - 1.xxx
file with duplicate - 2.xxx

Saya perlu cara untuk menghapus semua duplikat dengan file batch Windows.

Sesuatu di sepanjang baris:

FOR %f IN (C:\files\*.*) DO del "%f - 1"

Namun, itu tidak akan berhasil karena itu akan menyelesaikan hingga file with duplicate - 1.xxx - 1

Jika Anda ingin menghapus semua file dengan nama yang berakhiran - ?, maka Anda sudah dekat.

Coba ini:

FOR %%f IN ("C:\files\* - ?.*") DO del "%%f"

Anda perlu menggandakan tanda persen jika Anda menjalankannya dari file batch. Dan uji dulu dengan echo, bukan del.

Tidak perlu FOR, del dapat menerima wildcard. Mencoba:

del "c:\files\* - 1.xxx"

Akan cocok dengan apa pun dengan akhiran "- 1.xxx" di direktori.

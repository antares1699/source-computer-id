Diff untuk benar-benar mengabaikan spasi kosong

Saya mencari aplikasi yang akan membedakan dua file, dan benar-benar mengabaikan semua spasi, jadi misalnya:

class foo { 
  bar
  spaz 
}

sama sama dengan

class foo{bar spaz}

atau, juga

classfoo { 
  barspaz}

tapi tidak

classfoo { 
  spazspaz
}

yaitu akan menunjukkan kepada saya bahwa spaz pada contoh sebelumnya telah menggantikan bar dalam contoh lainnya. Itu hanya perlu membandingkan 2 file.

  • Ini bisa berupa utilitas yang kompatibel dengan windows atau linux / unix / posix
  • Saya sudah mencoba perintah lin / unix diff -w, hanya mengabaikan spasi jika ada perbedaan per baris adalah spasi. Saya tidak melihat opsi untuk sama sekali abaikan spasi.
  • Saya juga mencoba UECompare atau Ultracompare, utilitas perbandingan non-gratis untuk windows.

Apakah Anda mencari sesuatu seperti perintah tr? Ini dia manpages . Ini disertakan dengan msysgit, cygwin, dan gnuwin32 tools sejauh yang saya tahu.

Jadi Anda dapat menghapus semua spasi sebelum diffing dengan melakukan sesuatu seperti:

tr --delete '[:space:]' <filename.txt

Anda kemudian dapat memberi makan output dari perintah itu ke diff dan membuatnya berfungsi tanpa spasi.

Misalnya, saya memiliki file bernama HelloWorldApp.java. Izinkan saya menunjukkan kepada Anda bagaimana cygwin memprosesnya:

C:\temp>cat HelloWorldApp.java
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}
C:\temp>tr -d '[:space:]' <HelloWorldApp.java
classHelloWorldApp{publicstaticvoidmain(String[]args){System.out.println("HelloWorld!");//Displaythestring.}}

Meskipun berorientasi pada garis, diff bisa dibuat untuk melakukan apa yang ada diminta. Secara kasar, Anda dapat melakukan diff -w dan kemudian memposting proses keluaran diff. Itu bisa melihat keluaran diff dan menggabungkan berbagai pasangan garis untuk melihat apakah mereka (sekarang) cocok dengan baris di file lain.

Sepertinya itu mungkin O (n ^ 2) atau hal buruk lainnya, tapi tetap saja sangat membantu jika membatasi diri pada (katakanlah) tidak bergabung, gabung 2 baris, gabung 3 baris pada file 1 X no join, join 2 lines, join 3 lines on file 2 on a moving "window" of the diff output. (dengan X adalah ~ "produk silang")

Nyatanya, sepertinya pekerjaan yang dapat dilakukan oleh skrip pasca-pemrosesan keluaran diff Perl beberapa jam kerja (tergantung pada keterampilan pemrograman (Perl) Anda)

Mungkin ketika saya memiliki beberapa tugas yang saya tidak benar-benar ingin lakukan di tempat kerja ... :-)

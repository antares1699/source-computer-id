Microsoft Excel 2010 Bagaimana mereferensikan tabel ke lembar lain

Saya menggunakan Excel 2010.

Saya mendapat nama sheet "Maret"

Saya ingin menduplikasinya [Ini adalah meja ] (dengan referensi / tautan) ke sheet lain yang disebut "Ringkasan" yang meliputi Januari, Feb, dan Maret.

Di tab ringkasan saya akan menambahkan kolom lain yang disebut "Bulan" dan akan menduplikasi nilai "Maret" atau bulan tab mereka.

Saya melakukan beberapa penelitian tentang itu, mereka mengatakan salin dan tempel khusus dengan tautan, tetapi saya tidak dapat menemukannya.

Saya perlu mereferensikan seluruh tabel, kalau-kalau ada perubahan menambah / menghapus baris tabel, ringkasan masih akan memperbaruinya. Apakah ada solusi untuk ini?

Terima kasih!

Microsoft Excel tidak dapat mereferensikan Table dengan referensi / tautan ke lembar lain. Ini hanya bekerja dengan Range.

Dengan range Anda bisa mereferensikan seperti ini. Sheet1! $ A $ 1

Tetapi tidak mungkin dengan Table.

Ada solusi untuk ini ... meskipun yang jelek.

Anda dapat mencapai efek yang diinginkan dengan mengimpor Tabel dari File Sumber Excel ke Access DB (menggunakan tabel tertaut misalnya), lalu mengimpor Tabel Access tersebut ke file Tujuan Excel terakhir Anda.

Namun, jika Anda ingin memperbarui secara otomatis, Anda harus menulis kode kecil untuk membuka Access DB dan memperbarui file di sana, sebelum datanya benar di file tujuan Anda.

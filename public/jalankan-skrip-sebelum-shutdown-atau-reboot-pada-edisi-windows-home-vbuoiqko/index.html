Cara termudah untuk melakukannya adalah dengan kebijakan grup lokal.

Untuk melakukan ini, buka editor kebijakan grup lokal Anda (Klik Mulai, ketik gpedit.msc di kotak Mulai Pencarian, lalu tekan ENTER.)

Setelah ini dimuat, pergi ke jalur berikut: Konfigurasi Komputer> Pengaturan Windows> Skrip (Startup / Shutdown)> Shutdown (Lihat Di Bawah)

Di sini, Anda dapat memilih untuk memilih skrip yang akan dijalankan di komputer Anda sebagai bagian dari startup / shutdown. Anda juga dapat mengikuti Konfigurasi Pengguna> Pengaturan Windows> Skrip (Logon / Logoff) untuk mengatur skrip yang berjalan saat logon dan logoff.

Perbedaannya adalah:

Memulai Berjalan sebelum pengguna logon. Skrip startup dijalankan sebagai akun sistem lokal

Logon Berjalan segera setelah pengguna memasukkan sandi mereka. Ini dijalankan sebagai pengguna yang logon.

Keluar Berjalan saat sesi pengguna keluar (baik mematikan atau keluar untuk mengubah pengguna) dan dijalankan saat pengguna keluar

Mematikan Berjalan setelah logoff dan hanya sebagai bagian dari mesin yang dimatikan. Ini berjalan sebagai akun sistem lokal.

Semoga ini membantu.

Ini jalur registri

HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ Run

atau sebagai alternatif

Luncurkan Editor Kebijakan Grup dengan mengklik Mulai, lalu ketik gpedit.msc dan tekan Enter. Sekarang arahkan ke

Konfigurasi Pengguna> Pengaturan Windows> Skrip (Masuk / Keluar)

pilihan.

Tambahkan skrip Anda di sana.

Periksa ini untuk edisi rumah:

Untuk membuka Editor Kebijakan Grup Lokal sebagai snap-in MMC

Open MMC. (Click Start , click in the Start Search box, type mmc , and then press ENTER .)
On the File menu, click Add/Remove Snap-in .
In the Add or Remove Snap-ins dialog box, click Group Policy Object Editor , and then click Add .
In the Select Group Policy Object dialog box, click Browse .
Click This computer to edit the Local Group Policy object, or click Users to edit Administrator, Non-Administrator, or per-user Local Group Policy objects.
Click Finish .

Bagaimana cara mendapatkan screensaver di layar login Windows 7?

Saya menggunakan Windows 7 Ultimate 64-bit. Sejak beberapa minggu, saya tidak lagi memiliki screensaver di layar login Windows. Jika pengguna mencatat dan kemudian mengunci sesinya, screensaver akan dimulai pada akhirnya tetapi jika tidak ada pengguna yang masuk, layar login tidak akan pernah memulai screensaver.

Ada yang tahu cara memasang screensaver di sana?

Saya tidak 100% yakin apakah langkah-langkah ini akan berhasil jika seseorang tidak masuk!

Untuk mengaktifkan screensaver, pertama pilih screensaver yang Anda inginkan. Lihat di C: \ WINDOWS \ SYSTEM32 untuk file dengan ekstensi .SCR - biasanya ada adalah sebagai berikut (setidaknya di saya Instalasi Perusahaan) ...

  • SSTEXT3D - Teks 3D
  • BUBBLES.SCR - Gelembung
  • MYSTIFY.SCR - Membingungkan
  • PHOTOSCREENSAVER.SCR - Foto
  • RIBBONS.SCR - Pita

Salah satu hal yang harus diperhatikan adalah bahwa Anda tidak memiliki Direct3D dukungan di layar masuk sehingga screensaver bawaan di atas tidak akan berfungsi. Kecuali Anda memiliki file lain ingin mengambil file LOGON.SCR dari sistem operasi sebelumnya.

Setelah Anda memiliki screensaver (di file case LOGON.SCR), buka registri Anda editor dan arahkan ke berikut ini kunci -

HKEY_USERS \ .Default \ Control Panel \ Desktop

Di bawah kunci ini buat yang berikut ini Nilai REG_SZ ...

SCRNSAVE.EXE = C:\WINDOWS\SYSTEM32\LOGON.SCR
ScreenSaveActive = 1
ScreenSaveIsSecure = 0
ScreenSaveTimeOut = 60 (or whatever timeout you want in seconds)

Itu saja - Anda akan mendapatkan file screensaver di layar masuk lalu.

MacBook tidak masuk mode tidur saat dicolokkan sejak memutakhirkan ke Lion

MacBook saya tidak mati saat saya mengalihkannya ke mode tidur, saat dicolokkan. Saya meletakkan laptop dalam mode "tidur" atau menutup tutupnya, tetapi komputer tetap menyala dan tetap panas! MacBook saya menjadi tidur nyenyak (dimatikan dan memiliki lampu putih yang berdenyut saat menggunakan baterai). Jika dicolokkan & saya menidurkannya, layar akan mati, tetapi komputer tetap menyala (lampu putih solid)!

Ini telah terjadi sejak peningkatan versi ke Lion! Apakah ada cara untuk memperbaikinya?

Ini mati dengan baik & itu pergi tidur dengan baik saat menggunakan baterai. Itu tidak pergi tidur saat dicolokkan, itulah masalahnya! Saya sudah mencoba mengatur ulang SMC & PRAM, memperbaiki izin, dan menonaktifkan aplikasi startup!

Teman saya macbook pro tidur nyenyak dengan Lion! Apa yang salah! Ini tidak terjadi di Snow Leopard!

  1. Buka Utilities di folder Applications dan buka Terminal aplikasi.
  2. Ketik "pmset -g" tanpa tanda kutip dan tekan enter.
  3. Jika "tidur" memiliki 0 di sebelahnya dan kata "ditentukan oleh"; Nomor di sebelah yang ditentukan oleh adalah ID proses. Tuliskan angka itu.
  4. Ketik "ps -e" dan tekan enter. Ini akan mencantumkan semua proses yang berjalan.
  5. Temukan proses yang diterapkan pada mode tidur dan ambil tindakan yang sesuai.

Dalam kasus saya itu adalah proses id 19 yang ternyata dalam hal ini adalah proses server cetak.

Bagaimana cara menggeser subtitle secara akurat

Saya mencoba untuk men-hardsub file avi. Saya membuat subtitle untuk itu menggunakan file avi yang berbeda. Ada sedikit perbedaan waktu antara kedua file tersebut, jadi saya perlu sedikit menggeser waktu teks film. Saya telah melakukan ini sebelum menggunakan Subtitle Workshop, tetapi metode yang saya gunakan tidak terlalu akurat. Ini membutuhkan penggunaan slider dan cukup menyakitkan. Adakah cara yang lebih baik untuk melakukan ini?

Ada beberapa cara untuk menyesuaikan pengaturan waktu untuk semua subtitle dalam satu file dengan Lokakarya Subtitle:

1: Edit> pengaturan waktu> Atur Penundaan, atur waktu tunda positif atau negatif, pilih tombol radio "Semua teks film", Terapkan.

2: Pilih semua subtitle: Ctrl + A, untuk menambahkan penundaan 100 milidetik, tekan Shift + Ctrl + H, untuk mengurangi 100 milidetik, tekan Shift + Ctrl + N. Anda dapat menahan setiap kombinasi tombol ini untuk menambah atau mengurangi waktu dengan cepat.

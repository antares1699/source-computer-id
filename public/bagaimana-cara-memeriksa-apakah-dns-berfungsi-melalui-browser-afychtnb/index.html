Bagaimana cara memeriksa apakah DNS berfungsi melalui browser?

Bagaimana cara memeriksa melalui browser saya apakah DNS saya berfungsi atau tidak? Konteksnya adalah server DNS ISP saya terkadang mati selama beberapa menit. Jadi terkadang masalah konektivitas jaringan disebabkan oleh kesalahan DNS. Saya bisa pergi ke baris perintah dan melakukan nslookup, tetapi itu tidak praktis. Jadi apa cara mudah untuk memeriksa apakah DNS berfungsi melalui browser? Saya menggunakan firefox-4.

Bukan jawaban langsung untuk Anda pertanyaan tapi mungkin untuk masalahmu. Jika Anda terhubung ke Internet menggunakan router, coba aktifkan fitur DNS-nya dan gunakan sebagai server DNS PC Anda. Sekarang di router, setel default ke server DNS ISP Anda, tetapi miliki opsi kedua (ketiga) 8.8.8.8 dan / atau 8.8.4.4. Dengan begitu, jika server ISP tidak menjawab, router akan mencoba DNS cadangan.

Tidak semua perute konsumen mengizinkan ini, tetapi jika perute Anda mengizinkannya, itu akan mengatasi masalah.

Anda juga dapat membuat 8.8.4.4 atau apa pun DNS sekunder PC Anda, hanya digunakan jika yang utama tidak tersedia. Di bawah * OS tipe nix ini membutuhkan langkah yang sangat sederhana: tambahkan satu baris ke file /etc/resolve.conf. Saya tidak lepas dari kepala saya tahu bagaimana melakukannya di bawah Windows.

Sebenarnya solusi elegan adalah menghubungi saluran bantuan ISP Anda setiap kali DNS mereka turun, sampai mereka memperbaiki masalah.

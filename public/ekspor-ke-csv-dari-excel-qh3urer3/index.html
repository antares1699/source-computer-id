Ekspor ke CSV Dari Excel

Saya sering menggunakan Excel untuk menyiapkan data untuk dimuat ke dalam tabel. Saya membuat file CSV darinya dan menggunakan file itu untuk memuat data. Saya biasanya menemukan diri saya membuat file CSV beberapa kali.

Alur kerja saya saat ini:

  1. Edit spreadsheet
  2. File -> Simpan Sebagai -> * .csv
  3. Singkirkan dialog "... tipe file yang dipilih tidak mendukung buku kerja yang berisi beberapa lembar ..."
  4. Singkirkan dialog "... mungkin berisi fitur yang tidak kompatibel dengan CSV ..."
  5. Tutup file CSV dan klik "Jangan Simpan" pada dialog peringatan tentang menyimpan perubahan yang dibuat
  6. Buka kembali file spreadsheet asli
  7. Ulangi proses tersebut hingga selesai

Apa yang ingin saya lakukan:

  1. Edit spreadsheet
  2. File -> Ekspor -> * .csv
  3. Ulangi proses tersebut hingga selesai

Saya tidak dapat menemukan cara untuk menyelesaikan alur kerja yang saya inginkan.

Jawaban Bepergian Tech Guy tentu benar sejauh ini. Namun beberapa poin - jika Anda menyimpan makro di buku kerja saat ini, itu tidak akan tersedia saat Anda memulai buku kerja lain. Anda mungkin ingin menyimpannya di Personal Workbook - ini adalah file Excel yang dimuat secara otomatis setiap kali Anda memulai Excel; itu adalah lembar kerja tetapi tersembunyi. Itu membuat makro yang disimpan di dalamnya tersedia untuk buku kerja lain di sesi Excel yang sama.

Masalah lainnya adalah makro mungkin akan merekam folder dan nama file yang Anda simpan. Jika Anda tidak keberatan menulis ke file yang sama setiap kali (tidak hanya di sesi ini, tetapi di semua sesi mendatang) itu keren, tetapi jika Anda ingin dapat menyimpan lembar kerja saat ini ke CSV tanpa menulis ulang file CSV Anda dibuat di sesi sebelumnya, Anda perlu mengedit makro dan menghapus parameter Nama file. Makro tanpa nama file terlihat seperti ini setelah Anda mengeditnya ...

Setelah ini selesai, Anda dapat menyimpan file Excel apa pun dalam format CSV hanya dengan menekan pintasan yang ditetapkan, Ctrl-x dalam contoh ini. Ini akan disimpan ke folder dan nama yang sama dengan file XLS asli tetapi dengan ekstensi CSV - Anda tidak perlu memilih jalur atau nama file dan tidak akan menimpa file CSV lain dari lembar kerja lain. Tidak ada pesan peringatan, bahkan dialog konfirmasi - cukup berkedip dan selesai.

Satu detail terakhir - saat Anda masuk untuk mengedit Makro yang disimpan di buku kerja Pribadi, Excel mengatakan Anda tidak bisa karena "tersembunyi". Ini mengacu pada perintah Hide / Unhide pada toolbar di bawah menu View - ini tidak berarti Anda harus menemukan file di Explorer dan menghapus atribut file "tersembunyi". Itu menyia-nyiakan beberapa menit waktuku!

Semoga berhasil...

Yang ingin Anda lakukan adalah merekam makro yang menyimpan file Anda ke CSV:

  1. Buka menu Alat
  2. Pilih Makro> Rekam makro baru
  3. Beri nama seperti 'ToCSV'
  4. Anda dapat memilih untuk menyimpannya di buku kerja saat ini, atau secara global. Dalam kasus Anda, buku kerja saat ini sudah cukup
  5. Dari menu File, pilih Save As ..., Pilih CSV dan melalui semua dialog yang mengganggu
  6. Setelah disimpan, buka Alat> Makro> Berhenti Merekam Makro
  7. Untuk menyimpan ke CSV, pilih Tools> Macro> Macross ... Pilih Macro Anda dan klik run
  8. Untuk menyederhanakan proses, Anda dapat mengklik tombol Opsi ... dan menetapkan tombol pintasan ke makro Anda

Dan sekarang Anda dapat menyimpan ke CSV dengan sekali klik. Semoga berhasil!

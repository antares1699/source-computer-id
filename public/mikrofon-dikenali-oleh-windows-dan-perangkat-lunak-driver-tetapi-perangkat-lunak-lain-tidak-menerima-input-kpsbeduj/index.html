Mikrofon dikenali oleh windows dan perangkat lunak driver tetapi perangkat lunak lain tidak menerima input

Tiba-tiba mikrofon saya berhenti berfungsi pada semua Perangkat Lunak Desktop di Windows 10. Saya telah menginstal ulang semua driver terkait dan memperbarui semua yang diperlukan. Ketika saya menggunakan perangkat lunak driver (Realtek HD Audio Manager) saya dapat mendengar input mikrofon ketika saya mengatur volume output ke tinggi sehingga saya tahu ini bukan masalah perangkat keras. Tetapi ketika saya mencoba mengaksesnya di perangkat lunak lain, itu hanya diam dan tidak ada input yang direkam meskipun mikrofon yang benar muncul di pengaturan. Adakah yang tahu apa masalahnya

Anda dapat membuat sistem file yang tidak peka huruf besar / kecil dengan ZFS:

# zfs create -o casesensitivity=insensitive filesystem

jlliagre — Sumber
  • ZFS didistribusikan dengan lisensi Sun CDDL, yang tidak kompatibel dengan lisensi GPL yang digunakan oleh Kernel Linux. Jadi, ZFS tidak dapat (secara legal) digunakan dengan Linux. Satu-satunya pilihan adalah menggunakan ZFS dengan filesystem di userspace (zfs-fuse).
    — Vinay
  • ZFS pada sekring cukup berguna. Saya berbagi kolam antara ubuntu dan Solaris tanpa masalah. Ada juga proyek ZFS asli: zfsonlinux.org github.com/zfs-linux
    — jlliagre

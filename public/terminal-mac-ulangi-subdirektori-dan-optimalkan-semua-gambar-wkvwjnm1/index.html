Terminal Mac: Ulangi subdirektori dan optimalkan semua gambar

Saya memiliki folder yang berisi banyak subfolder yang penuh dengan gambar. Lihat gambar terlampir misalnya.

Pada dasarnya saya ingin mengulang semuanya dan memperkecil ukuran gambar sehingga tidak ada yang lebih lebar dari 1024 piksel. Semuanya jpeg.

Saya mengetahui perintah SIPS yaitu

sips -Z 1024 *.png

Namun, itu hanya berfungsi jika semua gambar ada di direktori saat ini.

Bagaimana cara menyiapkannya untuk melintasi semua subdirektori?

Anda dapat menjalankannya dengan find: find images/path -type f -name '*.png' -exec sips -Z 1024 {} \;

Find akan mencari file (-type f), dengan ekstensi png (-name '* .png') di dalam direktori 'images / path' dan mengeksekusi perintah dalam parameter, mengganti "{}" dengan nama file, Anda harus mengakhiri perintah dengan "\;".

Saya berhasil mengubah skrip bash kecil yang berhasil untuk saya

#!/bin/bash

find "foldername" -type f | \
while read file ; do
    echo "processing ${file}"
    sips -Z 2000 ${file}
done

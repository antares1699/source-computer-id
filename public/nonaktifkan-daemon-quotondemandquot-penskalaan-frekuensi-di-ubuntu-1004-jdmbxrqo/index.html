sudo apt-get install cpufrequtils sysfsutils

kemudian

echo performance | sudo tee /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor

dan ulangi untuk setiap core / CPU yang Anda miliki.

Anda dapat memverifikasi hasilnya dengan cpufreq-info.

Untuk membuat perubahan permanen: tambahkan baris berikut (atau baris - untuk beberapa core / CPU)

devices/system/cpu/cpu0/cpufreq/scaling_governor = performance

ke /etc/sysfs.conf (atau edit baris yang sudah ada).

Sebagian besar solusi yang disediakan di internet rumit dan tidak efisien karena frekuensi CPU disetel ulang setelah beberapa saat.

Solusi paling sederhana dan 100% untuk menonaktifkan penskalaan CPU adalah:

  1. sudo rcconf

  2. Hapus centang cpufrequtils, loadcpufreq, ondemand, powernowd, powernowd early.

  3. Tambahkan monitor penskalaan frekuensi CPU ke panel Anda dan atur core Anda ke frekuensi yang diinginkan.

  4. Mulai ulang.

  5. Periksa dengan cpufreq-info bahwa frekuensi yang dipilih aktif sepanjang waktu (100%).

  6. Semua obat Anda adalah milik kami! : D

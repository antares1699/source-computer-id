Mulai Windows 10 versi 1511, Windows Store tidak lagi mengikuti pengaturan ini kecuali Anda memiliki Windows 10 Perusahaan . Dari Darrell Gorter, karyawan Microsoft, masuk utas ini :

Halo,

Kebijakan grup untuk menonaktifkan Windows Store diubah menjadi hanya bekerja dengan edisi Windows 10 Enterprise

Jika Anda hanya memiliki Windows 10 Pro, Anda harus menghapus aplikasi Windows Store. Perhatikan bahwa menggunakan Remove-AppxPackage hanya berlaku untuk pengguna saat ini. Anda dapat membuat skrip logon yang menjalankannya ketika siapa pun masuk, atau Anda dapat menggunakan perintah ini (memerlukan admin) untuk menghapus Bursa dari daftar aplikasi yang diinstal saat pengguna baru masuk ( sumber ):

Remove-AppXProvisionedPackage -Online -PackageName Microsoft.WindowsStore_2015.701.14.0_neutral_~_8wekyb3d8bbwe

Perhatikan bahwa ini tidak akan menghapus Store untuk pengguna yang sudah masuk; mereka harus menjalankan perintah Remove-AppxPackage. Oleh karena itu, waktu terbaik untuk melakukannya Remove-AppxProvisionedPackage adalah sebelum pengguna akhir mendapatkan mesin (yaitu pada waktu penerapan). Namun begitu Store hilang, tidak dapat dengan mudah diinstal ulang karena tidak ada toko tempat menginstal aplikasi "modern".

Ini harus menghapus instalan Store, daripada menonaktifkannya (itu akan hilang dan Anda harus mengetikkan perintah lain untuk memulihkannya).

  1. Mulai -> ketik windows powershell

  2. Klik kanan aplikasi yang ditampilkan dan lakukan ** run as administrator * *

  3. Tunggu hingga kursor siap, lalu tempel:

    Get-AppxPackage *windowsstore* | Remove-AppxPackage

  4. Kunci dalam Memasukkan dan tunggu. Ini harus menghapus ikon dari Start setidaknya.

Baru saja diuji pada Windows 10 x64 saya.

Berikut beberapa info tentang Get-AppxPackage

firefox font alias / bergerigi di xfce

Saya telah menggunakan linux mint 7 selama beberapa minggu sekarang dan saya cukup senang dengannya, tetapi saya ingin mencoba desktop lain, mis. KDE / Xfce

Saya tidak yakin apakah itu kesalahan KDE dari Xfce, tetapi rendering font firefox payah sekarang, ini membuat font bergerigi / alias.

Saya menggunakan xfce sekarang,

Pengaturan Xfce settings Manager > appearance > fonts saya secara kasar terlihat seperti ini:

Default Font:

     Sans | 9  

Rendring :

 [x] Enable anti-aliasing  

           Hinting: None  
   Sub-pixel Order: None

Tetapi firefox seolah-olah mengabaikan pengaturan ini!

Coba edit file .fonts.conf Anda di bawah direktori home Anda (~/.fonts.conf).

Ini bekerja dengan baik untuk saya:

 <?xml version='1.0'?>
 <!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
 <fontconfig>
  <match target="font" >
   <edit mode="assign" name="hinting">
    <bool>false</bool>
   </edit>
  </match>
 </fontconfig>

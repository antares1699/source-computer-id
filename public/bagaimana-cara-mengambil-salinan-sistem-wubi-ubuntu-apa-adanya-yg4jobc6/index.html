Bagaimana cara mengambil salinan sistem wubi ubuntu apa adanya?

Saya telah menginstal ubuntu (sebagai dual boot) menggunakan penginstal wubi (Penginstal UBuntu berbasis Windows) untuk windows, dan telah bekerja di linux sejak saat itu. Sekarang saya memiliki banyak proyek dengan banyak dependensi, saya ingin menginstal ubuntu yang sama ke mesin lain, sehingga saya tidak perlu menginstal Ubuntu terlebih dahulu, lalu setiap proyek dan dependensinya. Ada folder bernama ubuntu di drive windows saya, yang dibuat oleh wubi dan berisi semua hal tentang ubuntu.

Mesin lain hanya menginstal windows 7 dan memiliki konfigurasi yang sama.

Apakah ada cara untuk menginstal ubuntu yang sama yang saya gunakan di komputer lain?

ini adalah metode yang saya gunakan ...

  1. Install Ubuntu menggunakan WUBI pada PC target dan salin Wubi Anda ROOT.DISK file ke lokasi yang sama di PC lain dan menggantinya.

  2. selanjutnya Anda harus menggunakan file ubuntu live cd .boot dari live cd lalu pasang partisi windows (jika instalasi Ubuntu Anda ada di partisi windows)

    sudo mkdir / win sudo mount / dev / sda1 / win

    Ganti sda1 dengan perangkat yang sesuai (a = disk, 1 = nomor partisi), lalu pasang disk virtual di dalamnya (gunakan fdisk -l" atau utilitas Disk untuk nomor perangkat)

    sudo mkdir / vdisk sudo mount -o loop /win/ubuntu/disks/root.disk / vdisk

  3. Sekarang konten disk virtual akan terlihat di bawah / vdisk

  4. Sekarang edit file grub.cfg gksu gedit /vdisk/boot/grub/grub.cfg mengajukan dan menghapus semua baris di atas " menuentry ".

  5. Selesai !!!

Dari Bisakah saya menyalin instalasi wubi saya antar mesin? :

  1. Pastikan Anda belum menginstal driver khusus (atau hapus jika Anda punya)
  2. Pastikan Anda menginstal rilis Ubuntu yang sama di mesin baru

  3. Muat instal pada mesin baru, ketika Anda melihat menu grub tekan 'e' pada entri pertama dan catat xxx dan yyy:

    set root=(xxx) misalnya (hd0, msdos2)

    linux /boot/vmlinuz.... root=yyy (misalnya / dev / sda2 atau UUID = nnnnnnn)

  4. Salin root.disk ke mesin baru

  5. Boot ke Ubuntu tetapi hanya sejauh menu grub

    Tekan 'e' pada entri pertama dan ubah nilai yang Anda lihat dengan xxx dan yyy dari pemasangan baru Anda. Hapus juga baris mulai search --no-floppy .... Kemudian tekan Ctrl - X untuk boot.

  6. Setelah booting, jatuhkan ke terminal ( Ctrl - Alt - T ) dan jalankan sudo update-grub untuk memperbaiki menu grub.

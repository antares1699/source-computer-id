Mac OS X: Ubah tombol F biasa menjadi tombol media

Saya baru-baru ini membeli keyboard biasa untuk MacBook saya untuk menyelesaikan set DJ stasioner saya. Tetapi ketika saya melakukannya, saya tidak menemukan cara untuk memetakan ulang kunci F [1-12] ke kunci media.

Pada dasarnya yang saya cari adalah kemampuan untuk membuatnya F1 , misalnya, jeda iTunes. Tentu saja, saya juga ingin berpaling F -keys ke tombol media untuk mundur dan melewati iTunes saya, karena saya merasa ini sangat nyaman.

Adakah alat atau cara yang memungkinkan saya melakukan itu? Saya di 10.9.

Unduh skrip berikut: http://gskinner.com/blog/assets/MediaControlScripts.zip

Kemudian ikuti saja petunjuk ini:

  1. Klik dua kali pada setiap file skrip untuk perintah yang ingin Anda instal, dan klik tombol "Instal" di dialog

  2. Buka Preferensi Sistem> Keyboard> Pintasan Keyboard

  3. Pilih Layanan di panel kiri, lalu gulir ke bawah ke skrip yang ingin Anda tetapkan pintasannya

  4. Klik dua kali di sisi kanan baris untuk mengedit pintasan

  5. Tekan kombinasi tombol yang ingin Anda gunakan, dan pastikan kotak centang di sebelah kiri dicentang (ini suka tidak dicentang saat Anda mengedit kombo tombol)

Ref: http://blog.gskinner.com/archives/2011/10/media-keys-in-osx-for-any-keyboard.html

Install Karabiner dan simpan file seperti ini sebagai ~/Library/Application Support/Karabiner/private.xml:

<?xml version="1.0"?>
<root>
  <item>
    <name>ctrl+f-keys to media keys</name>
    <identifier>mediakeys</identifier>
    <autogen>__KeyToKey__ KeyCode::F1, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_BRIGHTNESS_DOWN</autogen>
    <autogen>__KeyToKey__ KeyCode::F2, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_BRIGHTNESS_UP</autogen>
    <autogen>__KeyToKey__ KeyCode::F8, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_PREV</autogen>
    <autogen>__KeyToKey__ KeyCode::F9, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_PLAY</autogen>
    <autogen>__KeyToKey__ KeyCode::F10, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_MUSIC_NEXT</autogen>
    <autogen>__KeyToKey__ KeyCode::F11, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_VOLUME_DOWN, ModifierFlag::OPTION_L | ModifierFlag::SHIFT_L</autogen>
    <autogen>__KeyToKey__ KeyCode::F12, VK_CONTROL | ModifierFlag::NONE, KeyCode::VK_CONSUMERKEY_VOLUME_UP, ModifierFlag::OPTION_L | ModifierFlag::SHIFT_L</autogen>
  </item>
</root>

Kemudian buka aplikasi Karabiner, tekan tombol ReloadXML, dan aktifkan pengaturan.

Anda juga dapat menggunakan GUI untuk mengkonfigurasinya seperti ini:

Karabiner memiliki beberapa opsi untuk kumpulan kunci F mana yang ingin Anda gunakan, jadi lihat opsi mereka untuk mengetahui mana yang paling cocok untuk Anda.

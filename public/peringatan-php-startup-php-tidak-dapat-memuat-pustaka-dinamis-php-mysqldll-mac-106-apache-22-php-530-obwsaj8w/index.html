Di Mac Anda tidak boleh berurusan dengan file dll karena ini adalah pustaka Windows. Anda harus memeriksa php.ini file untuk setiap baris tanpa komentar (tidak ada titik koma di depan) seperti ekstensi = di bawah bagian Ekstensi Dinamis dan pastikan bahwa semua baris itu diberi komentar. Btw untuk menemukan lokasi tipe file php.ini di Terminal:

$php --ini

Dan itu akan memberi Anda sesuatu yang cukup jelas seperti:

Configuration File (php.ini) Path: /etc
Loaded Configuration File: /private/etc/php.ini

Aku ingin tahu apa Mulai Ulang Nanti cara untuk Ubuntu.

Windows memungkinkan kita memilih untuk diingatkan 10 menit, 1 Jam atau 4 Jam

Begitu juga dengan Macintosh

Setelah saya mengklik Restart Nanti kadang-kadang saya melihat ini beberapa kali dalam sehari dan kadang-kadang saya melihatnya di hari berikutnya. Ubuntu Wiki juga tidak menjelaskan opsi ini dengan baik.

Kapanpun Anda merasa seperti itu. Ada baiknya mempertimbangkan beberapa hal saat mengatakan ini.

Manajer paket yang mendasari melakukannya tidak memaksa Anda untuk reboot - meskipun /var/run/reboot-required akan memberi tahu Anda jika diperlukan .

Linux akan berpegang pada th e versi lama file tersebut sampai tidak ada yang menggunakannya , jadi khas restart adalah untuk peningkatan kernel, dan mungkin driver / modul.

Akhirnya ada filosofis perbedaan antara linux (dirancang untuk pengguna listrik, dan orang-orang yang membanggakan uptime dan keandalan) dan OS berorientasi desktop - server windows mungkin juga tidak akan memaksa reboot.

Karena itu, di linux, peringatan booting ulang ini dipertimbangkan direkomendasikan tidak penting dan informasinya. Tidak ada yang menghentikan seseorang untuk menulis sesuatu yang menjalankan sudo reboot untuk Anda.

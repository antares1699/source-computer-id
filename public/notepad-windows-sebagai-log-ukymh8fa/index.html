Notepad Windows sebagai log

Jika saya menyimpan notepad sebagai .LOG, ini berfungsi sebagai file log.

Pertanyaan saya adalah ketika saya mengubah entri di tengah file yang sudah ada - tidak menambahkan tanggal hari ini untuk itu.

Apakah ada cara untuk membuatnya bekerja?

Seperti yang Anda ketahui, saat Anda membuka file teks yang dimulai dengan .LOG, Notepad akan menggulir ke bagian bawah file dan memasukkan waktu dan tanggal.

Untuk memasukkan waktu dan tanggal secara manual di tempat lain, Anda dapat menggunakan Sunting → Waktu / Tanggal atau tekan F5 .

Dari FAQ tentang menginstal Eclipse :

Untuk menginstal Eclipse, yang Anda lakukan hanyalah mengekstrak unduhan file zip di direktori yang diinginkan. Tidak ada pekerjaan lebih lanjut yang diperlukan (selain memastikan Anda telah menginstal Java Runtime Engine). Saat Anda mengekstrak file, itu membuat subdirektori yang disebut "eclipse", dengan beberapa subdirektori di bawahnya. Jadi, misalnya, di Windows Anda dapat membongkar file zip di direktori root (mis., C :) dan Eclipse akan diinstal di C: \ eclipse. Perhatikan bahwa menginstal Eclipse tidak mengubah registri Windows.

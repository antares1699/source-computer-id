Melihat apa yang saya anggap sebagai copy paste:

chown -R www–data:www-data /somefolder

Perlu diperhatikan bahwa tanda hubung di nama pengguna bukanlah tanda hubung, tetapi ASCII 150, atau tanda hubung en. Ini dapat diverifikasi jika Anda menyalin dan menempelkan string ke alat seperti http://www.asciivalue.com/ . Anda akan melihat tanda hubung pertama adalah ASCII 150, sedangkan yang kedua adalah ASCII 45. ASCII 45 adalah tanda hubung standar yang diharapkan.

Perbaiki itu, dan kamu akan baik-baik saja.

Jika Anda terbiasa menggunakan distribusi Debian / Ubuntu dan Anda beralih ke distribusi CentOS / Red Hat, Anda mungkin mengalami masalah karena pengguna yang benar adalah "apache" di CentOS / Red Hat. Stringnya menjadi:

chown -R apache:apache /somefolder

Menambahkan file ke playlist foobar tanpa beralih ke playlist itu

Proses pembuatan playlist saya terlihat seperti itu:

  1. Buka playlist baru
  2. Filter musik bagus dari perpustakaan saya (daftar putar 'Filter Hasil')
  3. Seret file dari daftar yang difilter ke playlist baru
  4. Beralih kembali ke Hasil yang Difilter
  5. Pergi ke 3 dan lanjutkan sampai playlist dikompilasi

Saya ingin menyingkirkan tahap ke-4 (yaitu tidak kehilangan fokus hasil yang difilter) yang sangat melelahkan. Lebih baik lagi: gunakan pintasan keyboard untuk menambahkan file ke daftar putar.

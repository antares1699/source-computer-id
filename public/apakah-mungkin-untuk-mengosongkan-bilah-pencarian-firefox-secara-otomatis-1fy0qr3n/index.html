Apakah mungkin untuk mengosongkan bilah pencarian firefox secara otomatis?

Saya sering memilih sesuatu yang saya inginkan ke google dengan mouse (tidak [ctrl-c], saya menggunakan X11 / Linux), jadi saya dapat menempelkannya ke kotak mesin pencari di firefox. Namun, biasanya sudah ada sesuatu di sana dari pencarian sebelumnya, jadi saya tidak bisa menempelkannya begitu saja. Ini selalu menghalangi saya, jadi inilah pertanyaan saya:

Apakah ada cara untuk menghapus formulir pencarian segera setelah saya menekan [enter]? Jadi akan kosong untuk lain kali saya mencari sesuatu?

Di Linux, saya menemukan bahwa [ctrl-c] ... [ctrl-v] seharusnya tidak diperlukan dalam kasus itu, karena memaksa saya untuk memindahkan tangan kembali ke keyboard (saya kidal).

Ini juga akan membantu jika saya melakukan sesuatu di Google yang saya tidak ingin dilihat orang lain. Sangat mudah untuk melupakan bahwa pencarian terakhir Anda masih ada, meskipun Anda sudah menutup tab.

Karena penghapusan dukungan untuk Pengaya gaya lama (XUL) sejak FF-57 + "Quantum", semua plugin yang disebutkan di atas tidak lagi kompatibel dengan versi FF saat ini.

Sayangnya permintaan tersebut telah ditolak (dalam Bugzilla 1418024 ) untuk menambahkan fungsionalitas yang diperlukan untuk implementasi fitur semacam itu melalui API "ekstensi web" baru dari FF (lihat penjelasan di StackOverflow ).

Namun, sedang berlangsung diskusi apakah fitur yang diminta harus ditambahkan sebagai opsi yang dapat dikonfigurasi dari FF di Bugzilla 253331 .

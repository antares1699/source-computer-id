Perhatikan bahwa sebagian besar browser menampilkan FTP dengan cara yang cukup transparan.

Jika Anda memberikan tautan langsung ke file Anda, seperti file ftp: // user: password@yoursite.home (perhatikan sepenuhnya tentang sintaks, meskipun), dia bahkan tidak perlu repot-repot menggunakan klien FTP. Ini akan seperti mengunduh file biasa dari Internet. Ini bisa menjadi cara untuk membuatnya lebih mudah, menggunakan apa yang sudah Anda persiapkan.

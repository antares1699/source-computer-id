Bagaimana cara meniru program ARM64 di Windows X64 dengan Qemu?

Saya telah mengunduh ISO Windows ARM64 dari https://uup.rg-adguard.net/ dan buka dengan Qemu seperti pada artikel ini . Tapi Qemu sangat lambat untuk meniru seluruh OS Windows ARM64 (saya tahu emulasi << hypervisor). Oleh karena itu saya berpikir apakah mungkin untuk meniru satu executable (seperti arp, ping dll) di Qemu (atau dengan program lain). Jika ya, bagaimana saya bisa melakukannya? Jika tidak, lalu mengapa?

Jika emulator QEMU untuk komputer ARM terlalu lambat, Anda bisa mencoba Microsoft Device Emulator 3.0 - Rilis Mandiri . Anda dapat menemukan beberapa informasi di artikel Panduan Pengembang untuk Emulator ARM , meskipun itu berasal dari tahun 2005 dan tentang Microsoft Device Emulator 2.0.

Itu Proyek Xen juga telah diumumkan Dukungan untuk CPU ARM , meskipun saya tidak tahu seberapa maju perkembangan itu dan tidak jelas apakah itu dapat meniru ARM atau hanya berjalan di CPU fisik ARM. Ini adalah emulator lain yang bisa Anda coba.

Jika Anda ingin menggunakan arsitektur ARM dengan kecepatan asli, satu-satunya solusi adalah mendapatkan perangkat ARM. Raspberry Pi adalah kemungkinan murah di sini.

Untuk mendapatkan gambar siap pakai dengan Raspberry Pi di QEMU, lihat proyeknya Emulasi Raspberry Pi untuk Windows . Itu datang dengan file run.bat dan Anda harus menjalankannya dan mengikuti instruksi.

Jika semua yang Anda inginkan adalah bermain dengan program assembler ARM, Anda dapat menggunakan CEMU , mendukung Arm / AArch64, dan dijelaskan sebagai:

aplikasi GUI yang dibundel yang memberdayakan pengguna untuk menulis assembly dan mengujinya dengan mengompilasinya menjadi bytecode dan mengeksekusinya dalam emulator berbasis QEMU.

Petunjuk untuk membuat dan juga gambar virtual yang sudah jadi dapat ditemukan dalam Lingkungan Lab Azeria ARM . Mereka juga memberikan Cheatsheet Dasar-dasar Perakitan ARM

Beberapa informasi yang lebih berguna dapat ditemukan di sumber berikut:

  • Menyiapkan mesin virtual ARM
  • Cara Mengemulasi Raspberry Pi di PC Anda

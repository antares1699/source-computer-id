Apa sebenarnya proxy itu?

Apa yang dilakukan proxy dan apa hubungannya dengan keluar dari jaringan saya ke jaringan luar?

Proxy berarti sesuatu seperti "sesuatu yang bertindak atas nama hal lain". Ini adalah kata bahasa Inggris, bukan kata IT. Misalnya, memberikan suara dengan proxy adalah ketika Anda meminta orang lain untuk memberikan suara menggantikan Anda, biasanya karena Anda mempercayai keputusan mereka.

Dalam kasus web proxy, ini seperti server web lokal yang dapat dimintai halaman, tetapi akan mendapatkannya dari server web asli dan kemudian memberikannya kepada Anda. Demikian juga, dalam kasus server basis data, proxy DB akan mengambil kueri SQL, mengirimkannya ke server basis data jarak jauh, mendapatkan jawabannya, dan meneruskannya kembali kepada Anda. Dalam kode, proxy dapat berupa instance (atau objek, seperti dalam pemrograman berorientasi objek) yang berjalan di mesin lokal dan menerima perintah, tetapi sebenarnya meneruskan perintah tersebut ke bagian kode serupa yang dijalankan di komputer lain.

Sekarang, sebagian besar kedengarannya tidak terlalu berguna. Apa yang membuatnya berguna adalah kebanyakan proxy mengabstraksikan komunikasi jarak jauh, atau melakukan hal-hal perantara yang berguna. Misalnya, proxy web sering kali mendapatkan halaman dari server jarak jauh, menghapus iklan atau porno, menyimpan salinan, dan KEMUDIAN memberikan halaman web yang dimodifikasi kepada pemohon. Jika orang kedua menginginkan halaman tersebut, mereka mendapatkan salinan yang disimpan. Jadi itu pekerjaan caching / modifikasi yang cukup berguna, dan cukup transparan bagi pengguna dengan browser mereka yang hanya meminta halaman web seperti biasa.

Demikian juga, proxy DNS mungkin menerima permintaan untuk alamat di internet, mengirimkan permintaan tersebut ke internet seperti biasa, tetapi mencegat permintaan untuk alamat di dalam perusahaan dan menjawabnya sendiri dengan menggunakan informasi internal.

Dalam istilah "keluar dari jaringan Anda" ... itu adalah masalah pemfilteran. Perusahaan waras mengunci akses internet, dan memaksa Anda untuk meminta halaman melalui proxy web. Proksi ini kemudian dapat memfilter dan memantau permintaan situs web, memeriksa pornografi, menghapus virus yang diunduh, dll.

Jika Anda berbicara tentang server proxy untuk web, itu adalah server tempat Anda membuat permintaan untuk halaman web, kemudian berbalik dan mendapatkan halaman dan menyerahkannya kembali ke mesin Anda.

Biasanya itu cache halaman dalam prosesnya. Kemudian jika menangani sebuah kantor, misalnya, dan Anda pergi ke Microsoft.com, dan tiga orang lainnya pergi ke microsoft.com, permintaan pertama membuat proxy keluar dan mendapatkan halaman, lalu permintaan berikutnya ditarik dari cache, membuatnya lebih cepat.

Proksi juga dapat memeriksa untuk memastikan konten tidak dicampur dengan malware, mencatat aktivitas penjelajahan web Anda, atau memblokir Anda untuk membuka situs porno atau jenis situs lain yang tidak ingin Anda gunakan oleh perusahaan Anda pada waktu perusahaan.

Pada dasarnya proxy adalah agen perantara antara intranet Anda dan Internet.

Proksi lain dapat dikonfigurasi untuk layanan seperti FTP, dll. Tetapi yang paling umum adalah untuk penjelajahan web.

Bagaimana saya bisa meyakinkan OneDrive bahwa itu harus menggunakan folder di SD-Card saya?

Saya baru saja meningkatkan ke Windows 10 dari Windows 8.1 di Microsoft Surface Pro saya. OneDrive memberi tahu saya bahwa itu tidak dapat menggunakan lokasi saat ini yang telah saya konfigurasikan di Windows 8.1, yang terletak di kartu SD 128GB saya.

Saya sudah mencoba tersangka biasa dengan utilitas baris perintah mklink untuk membuat tautan direktori simbolis dari C: \ Users \ xxx \ OneDrive ke D: \ OneDrive seperti:

mklink /d C:\Users\xxx\OneDrive D:\OneDrive

dan

mklink /j C:\Users\xxx\OneDrive D:\OneDrive

dan bahkan:

mklink /j C:\Users\xxx\OneDrive2 D:\OneDrive
mklink /d C:\Users\xxx\OneDrive C:\Users\xxx\OneDrive2

Tidak ada yang berhasil. Saat saya membuka OneDrive, ia menanyakan lokasi folder OneDrive saya, yang saya arahkan ke C:\Users\xxx\OneDrive, namun, ia terus mengatakan:

Folder OneDrive Anda tidak dapat dibuat di lokasi yang Anda pilih.

Saya sekarang sedang membuat .VHDX pada SD-Card, dan memasangnya pada C:\Users\xxx\OneDrive, yang saya anggap akan berhasil; Namun apakah ada cara yang lebih baik? Mungkin beberapa tweak registry? Mungkin palu besar?

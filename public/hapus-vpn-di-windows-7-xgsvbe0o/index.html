Hapus VPN di Windows 7

Bagaimana cara menghapus VPN yang tidak lagi saya perlukan dari daftar Sambungan Jaringan di Windows 7?

4
Liam — Sumber

1 Jawaban

7
7

Cukup klik Windows Orb, kemudian mulailah mengetik Network Connections untuk mencari dan klik View Network Connections ketika sudah terlihat.

Cukup hapus koneksi VPN yang tidak Anda perlukan.

William Hilsum — Sumber
  • Saya mencobanya sebelumnya dan tidak muncul. Sekarang, jika saya mengetik 'Sambungan Jaringan', ini menunjukkan 5 dari 17 kecocokan dalam Panel Kontrol, tidak ada yang saya inginkan. Tetapi jika saya mengklik 'Control Panel' untuk mendapatkan semua hasil pencarian, atau jika saya mengetik 'View Network Connections', maka saya bisa membukanya. Terima kasih!

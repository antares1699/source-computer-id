Bagaimana cara menyalin semua folder pada tingkat tertentu, tetapi tidak ada file atau subfolder?

Saya ingin tahu cara menyalin semua folder ke lokasi baru, tetapi saya hanya ingin memasukkan folder itu sendiri - tidak ada yang ada di dalamnya (jadi, tidak ada subfolder, dan tidak ada file). Apakah ini mungkin?

Selain itu, terkait dengan ini, apakah ada juga cara untuk menyalin semua folder, dan juga subfolder di dalam folder tersebut, tetapi hanya sedalam 1 tingkat (jadi hanya folder anak langsung, bukan anak di dalam anak), dan tetap tanpa ada file yang disalin.

Saya terbuka untuk melakukan ini dengan cara apa pun - baris perintah, atau melalui utilitas.

Gunakan utilitas Microsoft robocopy.

robocopy SOURCE_FOLDER DEST_FOLDER /E /LEV:1 /XF *

Pilihan:

  • /E Salin Subfolder, termasuk Subfolder Kosong
  • /LEV:1 Hanya salin n LEVel teratas dari pohon sumber.
  • /XF * eXclude File yang cocok dengan nama / jalur / wildcard yang diberikan.

Sumber: http://ss64.com/nt/robocopy.html

Menggunakan baris perintah, jika Anda hanya ingin menyalin folder kosong, itu membuat folder di lokasi baru katakanlah di suatu tempat di c: ...:

set newlocation=c:\...
@for /f "tokens=3,4" %i in ('dir') do @if "%i" equ "<DIR>" if "%j" neq "." if "%j" neq ".." mkdir %newlocation%\%j

Jika Anda ingin melangkah lebih dalam, ulangi perulangan for sekali lagi:

@for /f "tokens=3,4" %i in ('dir') do @if "%i" equ "<DIR>" if "%j" neq "." if "%j" neq ".." mkdir %newlocation%\%j && for /f "tokens=3,4" %u in ('dir %j') do @if "%u" equ "<DIR>" if "%v" neq "." if "%v" neq ".." mkdir %newlocation%\%j\%v

Perhatikan bahwa jika Anda ingin memasukkan ini ke dalam file batch, ganti semua % dengan %%.

Kamu bertanya,

Apakah ada masalah untuk mengonversi menu.vim dan _vimrc dari latin1 ke utf8?

Seharusnya tidak menjadi masalah, dan jawaban FvD menunjukkan betapa mudahnya hal itu.

Tapi Anda juga tidak perlu melakukannya.

Skrip yang disertakan dengan Vim, tidak boleh menggunakan karakter mewah yang representasi byte berubah ketika beralih dari latin1 ke utf-8. Jika salah satu skrip ubahsuaian Anda bergantung pada pengkodean file tertentu, Anda dapat mengonversinya, ATAU, Anda dapat menambahkan perintah "skrip pengodean apa punEncodingName" ke bagian atas skrip, dan Vim akan mengonversi dan menafsirkannya dengan benar seperti yang terbaca di data file (file itu sendiri tetap dalam pengkodean lama).

Bagaimana cara mendapatkan "eject" / "remove safe" untuk USB di Windows 8?

Saya memiliki masalah yang sangat serius yang juga dialami oleh banyak pengguna lain: tidak ada cara lagi untuk "mengeluarkan" / "mengeluarkan dengan aman" drive USB dari Windows 8 / Windows 8.1.

Dengan googling saya menyadari bahwa banyak orang memiliki masalah dan hanya "menarik" drive USB tanpa mengeluarkannya dengan benar.

Sekarang jika saya kemudian mencoba memasang drive ini dari, katakanlah, Linux, inilah yang saya dapatkan:

The disk contains an unclean file system (0, 0).
The file system wasn't safely closed on Windows

Dan saya tidak ingin mengambil risiko apa pun dengan data saya: terutama karena saya telah membaca laporan dari orang-orang yang tidak mengeluarkan data dengan benar dan mengalami masalah kerusakan data. Tujuan utama dari drive USB yang saya sambungkan adalah untuk membuat cadangan jadi ...

Pada Windows 7 saya dapat "mengeluarkan dengan aman" dengan mengklik kanan pada sembarang partisi dari drive USB dan kemudian mengklik "keluarkan" / "eject dengan aman" (sistem saya dalam bahasa Prancis, jadi saya tidak tahu nama pasti dari pilihan dalam bahasa Inggris). Opsi itu tepat di bawah "Format".

Saya melihat beberapa pesan berbicara tentang "opsi tersembunyi" yang dapat disembunyikan dengan mengklik "panah atas" di "area pemberitahuan".

Saya sama sekali tidak tahu apa artinya ini dan saya tidak melihat "panah atas" di mana pun.

Saya mencoba banyak hal dan tidak bisa lagi "mengeluarkan dengan aman" drive USB dari Windows 8.

Bagaimana saya bisa mendapatkan kembali opsi itu? (gambar / tangkapan layar mungkin bernilai sepuluh ribu kata di sini)

Ikon Safely Remove Hardware tidak selalu muncul di dok. Untuk mendapatkannya kembali, (dengan perangkat USB dicolokkan) klik kanan waktu dan pilih Sesuaikan Ikon Pemberitahuan. Gulir ke bawah dan temukan Windows Explorer, Ini akan memiliki ikon USB yang sudah dikenal. Ubah dari Hide Notifications. Perubahannya langsung.

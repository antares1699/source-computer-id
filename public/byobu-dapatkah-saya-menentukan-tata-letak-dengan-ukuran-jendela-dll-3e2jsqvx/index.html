byobu: dapatkah saya menentukan tata letak dengan ukuran jendela, dll?

Saya baru saja memulai dengan byobu dan tidak dapat memahami hal mendasar ini. Saya perlu mengatur tata letak yang telah ditentukan untuk windows saya di byobu jadi ketika saya memulainya, saya mendapatkan tata letak yang saya tentukan dan program saya mulai di jendela. Apakah ini mungkin?

Tentu, ini cukup mudah digunakan Byobu dibangun di hotkeys.

Buat tata letak sesuka Anda, menggunakan:

  • Ctrl - F2 untuk membelah secara vertikal
  • Bergeser - F2 untuk membelah secara horizontal
  • Bergeser - Atas bawah kiri kanan untuk bergerak di sekitar perpecahan
  • Bergeser - Alt - Atas bawah kiri kanan untuk mengubah ukuran setiap bagian

Sampai Anda puas dengan tata letaknya.

Kemudian, Anda dapat menyimpan tata letak menggunakan: Ctrl - Bergeser - F8 .

Dan Anda dapat memulihkan tata letak menggunakan: Alt - Bergeser - F8 .

Perhatikan juga bahwa Anda dapat melihat semua hotkey Byobu menggunakan Bergeser - F1 .

Pengungkapan penuh: Saya adalah penulis dan pengelola Byobu .

Ini tidak mungkin.

Waktu file digunakan di Windows API melalui Struktur FILETIME , dijelaskan sebagai:

Berisi mewakili nilai 64-bit jumlah interval 100 nanodetik sejak 1 Januari 1601 (UTC).

Karena nilai negatif tidak didukung, tanggal di atas adalah batas awal.

Kesimpulan: Kode Anda yang menguji tanggal sebelumnya tidak akan pernah dipertanyakan.

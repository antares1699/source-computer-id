Apa cara paling efisien untuk mentransfer data dari desktop ke laptop?

Saya telah membeli laptop pengganti desktop dan saya masih memiliki sekitar 20-30 GB data untuk ditransfer dari komputer saya saat ini. Apa cara paling efisien untuk melakukannya?

Perhatikan bahwa desktop adalah komputer tahun 2001 dengan USB 1.0, jadi USB bukanlah pertanyaan.

Cara / cara tercepat yang harus dilakukan! - Jika Windows (Vista +) ke Windows (Vista +), Bung in cross over cable dan aktifkan file sharing. Setel lokasi ke rumah dan. Di mesin lama Anda, atur file yang ingin Anda bagikan dan di yang baru, buka peta jaringan dan klik pada mesin lama dan Anda akan melihat semua file.

Jika lebih lama, cukup atur jaringan dan berbagi mengikuti panduan / wizard apa pun dan salin file dengan cara itu.

Namun terkadang, jika mengalami kesulitan dengan izin jaringan, saya menemukan hal termudah untuk dilakukan adalah menginstal Filezilla Server di mesin lama dan klien di mesin baru. Di yang lama, atur root hard drive untuk dibagikan dan cukup navigasikan dari klien melalui IP di alamat ftp. Ini berantakan dibandingkan dengan berbagi folder Windows, tetapi biasanya hanya berfungsi ketika izin atau pengaturan menghalangi.

Terakhir, jika Desktop ke Desktop, saya akan (dan ada di posting lain) merekomendasikan agar Anda merobek hard drive dan meletakkannya di mesin baru untuk melakukan transfer. Anda selalu dapat membeli Enklosur dari eBay dan memilikinya untuk digunakan di masa mendatang. Selain itu, cara tercepat adalah - Jika laptop Anda adalah SATA, keluarkan hard drive, letakkan di mesin LAMA Anda karena laptop SATA berfungsi dengan SATA desktop, dan cukup salin file lalu masukkan kembali.

Opsi tercepat yang dapat saya pikirkan (ketika saya mengalami masalah ini minggu lalu) adalah koneksi Gigabit Ethernet langsung antara desktop dan laptop. Setelah diatur, dan dengan bingkai jumbo diaktifkan, ini akan lebih cepat daripada menyalin ke hard drive eksternal (USB), dan kemudian menyalin lagi ke laptop.

Tetapi pada kenyataannya, ini bisa menjadi menyakitkan untuk diatur dan sedikit temperamental; jika kedengarannya rumit maka menggunakan hard-drive eksternal mungkin akan lebih cepat untuk Anda.

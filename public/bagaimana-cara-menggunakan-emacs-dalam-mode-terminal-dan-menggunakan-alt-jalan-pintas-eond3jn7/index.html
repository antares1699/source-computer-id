Bagaimana cara menggunakan Emacs dalam mode terminal dan menggunakan Alt +? jalan pintas?

Saya ingin menggunakan emacs di terminal, tetapi di Ubuntu 11.10 terminal bash sudah memiliki banyak jalan pintas Alt, bagaimana saya bisa mengatasinya?

5

Anda mungkin ingin mempertimbangkan untuk menggunakan emulator terminal yang tepat sebagai ganti dari monster raksasa berukuran kraken yang dikirimkan bersama Gnome. Saya sarankan urxvt, atau sakura jika gtk adalah suatu keharusan.

Karena itu, Emacs secara teknis tidak menggunakan tombol Alt. Ini menggunakan kunci "meta", yang kebetulan dipetakan ke tombol alt pada sistem modern, tetapi itu tidak wajib. Pertama, Anda dapat menggunakan kombinasi A-<whatever> dengan menekan ESC dan kemudian <whatever>. Ini bekerja cukup banyak pada semua Unix.

Atau, petakan kunci meta ke sesuatu yang lain. Ini dijelaskan sini .

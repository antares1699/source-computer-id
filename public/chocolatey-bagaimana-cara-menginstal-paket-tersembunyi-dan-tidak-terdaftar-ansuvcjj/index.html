Chocolatey - Bagaimana cara menginstal paket tersembunyi dan tidak terdaftar?

Saya ingin menginstal paket yang tersedia di chocolatey, tetapi terdaftar sebagai tersembunyi \ tidak terdaftar:

Paket ini tidak terdaftar dan disembunyikan dari daftar paket.

Contoh paket tersebut:

https://chocolatey.org/packages/napiprojekt

Baris perintah Chocolatey cinst napiprojekt menghasilkan kesalahan:

Kegagalan - napiprojekt - napiprojekt tidak terpasang. Paket tidak ditemukan dengan sumber yang terdaftar. Jika Anda menentukan versi tertentu dan menerima pesan ini, ada kemungkinan bahwa nama paket ada tetapi versinya tidak. Versi: "" Sumber: " https://chocolatey.org/api/v2/ "

Bagaimana saya bisa memaksa Chocolatey untuk menginstal paket tersembunyi dan tidak terdaftar?

Untuk menginstal paket tidak terdaftar, Anda perlu mengetahui versi yang Anda coba instal dan meneruskan --version ### dengan choco install pkgname.

Namun jika Anda membaca komentar, paket ini tidak terdaftar karena hasil pemindaian VirusTotal - https://www.virustotal.com/en/file/d1baa923cf2a8ed8c93393b74df3a29907da85ccb2eb82795d431f6493f87fca/analysis/1454729257/

Mungkin lebih baik untuk memahami mengapa ada sesuatu yang tidak terdaftar terlebih dahulu sebelum mencoba menginstalnya. Biasanya paket yang aman atau dikenal baik tidak akan dicantumkan.

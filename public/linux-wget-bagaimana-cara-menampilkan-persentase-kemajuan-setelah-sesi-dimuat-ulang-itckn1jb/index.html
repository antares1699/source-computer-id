Linux wget, bagaimana cara menampilkan persentase kemajuan setelah sesi dimuat ulang?

Saya menjalankan debian Squeeze dalam mode konsol di komputer steker. Saya mengontrolnya membuka sesi SSH dari mesin Windows, di jaringan lokal yang sama.

Saya mulai mengunduh file besar menggunakan wget.

Yang saya dapatkan adalah progress bar konsol yang menunjukkan persentase data yang diunduh, ukuran file, dan kecepatan unduhan.

Saat saya menutup sesi, debian masih berjalan dan mengunduh. Baik.

Tetapi ketika saya menutup dan membuka kembali sesi, bagaimana cara melihat jumlah data yang diunduh, menggunakan perintah linux?

Terima kasih.

Untuk menjawab "bagaimana cara melihat jumlah data yang diunduh, menggunakan perintah linux?" hanya, di direktori unduhan gunakan salah satu dari yang berikut ini

ls -l
ls -s
ls -sh <download filename>

Itu akan menunjukkan kepada Anda ukuran file yang sudah diunduh / ditulis ke disk Anda.

wget biasanya mencatat outputnya ke file wget-log di direktori unduhan. Untuk menampilkan file ini dan perubahannya, jalankan perintah berikut di direktori unduhan:

$ tail -f wget-log

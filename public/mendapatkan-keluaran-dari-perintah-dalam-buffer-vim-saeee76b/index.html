mendapatkan keluaran dari perintah dalam buffer vim

apa yang ingin saya lakukan

  1. mengedit satu file
  2. run '! svn diff%' (atau perintah lain yang menggunakan nama file saat ini )
  3. letakkan output di buffer (atau tab) baru

bonus: tidak menggunakan skrip, karena saya sering bekerja di host, saya tidak berhasil sesuai keinginan saya.

Percobaan pertama saya adalah: :tabnew | r !svn diff % yang jelas tidak berfungsi karena% akan kosong seperti yang diurai setelah perintah tabnew.

Ini harus bekerja:

:tabnew | r !svn diff #

# mereferensikan buffer sebelumnya, yaitu buffer yang telah Anda buka sebelum :tabnew dijalankan.

Bahkan lebih baik

:tabnew | set buftype=nowrite | r !svn diff #

karena vim tidak akan mengeluh ketika Anda menutup tab, tetapi ini sudah terlalu banyak untuk diketik setiap saat.

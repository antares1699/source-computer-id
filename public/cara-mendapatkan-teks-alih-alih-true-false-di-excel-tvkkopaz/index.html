Sebenarnya, ya, itu bisa dilakukan melalui pemformatan bersyarat ... Semacam. Gunakan tangkapan layar di bawah ini untuk menyiapkan format bersyarat yang menggunakan format angka kustom untuk sel yang TRUE. Anda harus dapat memperkirakan penyiapan untuk False = No.

Penting untuk diperhatikan bahwa ini hanya akan mengubah semu nilai di sel dan bukan nilai sebenarnya. Jika Anda mencoba melakukan kalkulasi lain berdasarkan sel ini, itu akan menggunakan nilai TRUE / FALSE dan bukan nilai YES / NO. Oleh karena itu, metode ini hanya berguna jika Anda khawatir tentang apa yang disajikan kepada pengguna dan bukan apa yang digunakan dalam penghitungan yang mendasarinya.

Lihat bagaimana nilainya tetap menjadi TRUE tetapi justru menampilkan "Ya".

Biarkan saya memberikan dua sen saya.

Misalkan Anda hanya tertarik dengan nilai yang ditampilkan, Anda dapat menggunakan trik berikut.

Di C3 masukkan rumus:

=AND(A1=B1)+0

Mungkin, singkirkan bagian DAN:

=(A1=B1)+0

Sekarang, Anda dapat mengatur format kustom untuk C3 menjadi "Yes";;"No" .

Itu +0 bagian memaksa Excel untuk memperlakukan hasil sebagai numerik, memberi Anda opsi untuk menggunakan format kustom.

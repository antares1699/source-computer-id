1 Jawaban

10
10
  1. Mulai browser Chrome / Chromium
  2. Arahkan ke "chrome: // flags"
  3. Temukan entri "Simpan Halaman sebagai MHTML"
  4. Klik "aktifkan"
  5. Mulai ulang browser Anda
  6. Itu saja, sekarang halaman web akan disimpan secara default sebagai MHTML

Di Google Chrome Versi 43.0.2357.130 m

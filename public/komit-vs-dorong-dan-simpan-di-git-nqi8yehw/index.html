Dorong dan tarik khusus untuk Sistem Kontrol Versi yang terdesentralisasi.

Anda harus membaca buku ini http://book.git-scm.com/ Ini menjelaskan semua yang perlu Anda ketahui

Dengan DCVS, repositori bersifat lokal. Jika Anda hanya berkomitmen, itu tidak akan dibagikan di server jarak jauh.

Perintah push mengirimkan komit lokal Anda ke server jarak jauh, dan menggabungkannya.

Pull = Ambil + Gabung Git mendapatkan komit dari server jarak jauh (ambil) sebagai cabang pelacakan jarak jauh, lalu menggabungkannya di cabang Anda.

Menyimpan adalah fitur khusus. Ini memungkinkan Anda untuk menyimpan modifikasi yang tidak terikat ke dalam simpanan, dan menghapusnya dari cabang yang sedang Anda kerjakan. Anda dapat mengambilnya nanti dengan menerapkan simpanan.

Ini sangat berguna saat Anda berada di tengah perubahan besar, dan Anda hanya ingin melakukan perbaikan cepat. Anda hanya perlu menyimpan perubahan, menulis perbaikan, mengkomitnya dan kemudian menerapkan simpanan yang disimpan untuk kembali ke pekerjaan Anda.

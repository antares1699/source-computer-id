Irfanview dengan plugin Save-for-Web melakukan ini.

  1. Pasang irfanview .
  2. Instal dan unduh plugin. Cara yang disarankan untuk melakukannya adalah dengan mengunduh dan menginstal Paket "Semua plugin" dari situs web mereka.

Di irfanView, plugin dapat diakses melalui "File", "Save for web (plugin)". Ada opsi "Kompres untuk ukuran" yang memungkinkan Anda memasukkan ukuran file yang diinginkan dan kemudian membuat perkiraan dari apa yang Anda masukkan.

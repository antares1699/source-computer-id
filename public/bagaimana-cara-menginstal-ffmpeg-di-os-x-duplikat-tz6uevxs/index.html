Bagaimana cara menginstal ffmpeg di OS X? [duplikat]

Anda bisa mendapatkan biner di sini http://ffmpegmac.net/ Bagaimana cara menginstal biner tersebut untuk menggunakannya sebagai alat baris perintah? Saya punya mac, dengan mavericks. Jika memungkinkan, alangkah baiknya menginstalnya tanpa menggunakan kata sandi admin.

Sepertinya Anda dapat mengunduh zip untuk "Snow Leopard dan Di Atas 64bit", unzip file setelah selesai diunduh (dengan mengklik dua kali atau menggunakan alat baris perintah unzip), dan kemudian Anda memiliki binari.

Kemudian Anda hanya perlu memutuskan di mana meletakkan binari. Karena Anda tidak menginginkan kata sandi admin, mungkin yang terbaik adalah menyimpannya di direktori home Anda. Karena itu, saya sarankan melakukan hal berikut di terminal Anda:

mkdir -p ~/.local/bin
cd ~/Downloads
curl -L 'http://www.ffmpegmac.net/resources/SnowLeopard_Lion_Mountain_Lion_Mavericks_Yosemite_El-Captain_25.02.2016.zip' | tar -xf-
mv ffmpeg ffprobe ffserver ~/.local/bin
echo 'export PATH=$PATH:~/.local/bin' >> ~/.bash_profile
source ~/.bash_profile

Skrip di atas akan membuat direktori untuk binari baru, lalu unduh zip ffmpeg ke direktori Unduhan Anda, lalu unzip dan pindahkan binari ke direktori biner yang baru dibuat, dan terakhir menambahkan baris ke lingkungan bash Anda sehingga Anda terminal sumber biner dengan benar.

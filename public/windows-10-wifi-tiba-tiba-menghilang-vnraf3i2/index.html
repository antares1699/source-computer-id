windows 10 wifi tiba-tiba menghilang

Saya sedang membaca berita di internet ketika saya menyadari bahwa saya kehilangan koneksi. Tidak hanya saya terputus dari jaringan wifi, tetapi saya tidak dapat melihat jaringan sama sekali (Tidak terhubung - Tidak ada koneksi yang tersedia). Jaringan terlihat di perangkat lain.

Ini adalah Laptop dari 3 bulan yang lalu, Asus, tidak ada port Ethernet. Itu datang dengan windows 10. Saya tidak tahu apa yang harus dilakukan karena saya lebih seorang pria ubuntu, jadi tolong beri tahu saya jika Anda memerlukan info lebih lanjut.

Layar "Pengaturan jaringan & Internet" bertuliskan "Tidak tersambung" "Anda tidak tersambung ke jaringan mana pun".

Di layar "Ubah opsi adaptor" hanya ada satu ikon "Ethernet 2, kabel Jaringan dicabut".

Pemecahan masalah meminta saya untuk menyambungkan kabel Ethernet, yang saya tidak bisa, jadi tidak berguna.

Layar "Lihat properti jaringan Anda" memiliki properti Wifi, dengan status "Tidak Ada" dan Konektivitas "Terputus".

Di layar "Device Manager", ada tanda seru di dalam segitiga kuning di Network adapter "Intel Dual Band Wireless-AC 8265". Jelas disinilah masalahnya, tapi saya tidak tahu bagaimana menyelesaikannya. Status perangkat adalah "Windows masih menyiapkan konfigurasi kelas untuk perangkat ini. (Kode 56)"

Ada kemungkinan bahwa Pembaruan Windows telah menginstal driver baru saat Anda sebelumnya browsing. Beberapa pembaruan driver tidak menyebabkan permintaan untuk reboot.

Untuk menguji idenya, masuk ke Device Manager, klik kanan pada perangkat, pilih Properties, buka tab Driver dan lihat apakah tombol "Roll Back Driver" dinonaktifkan. Itu tidak dinonaktifkan, klik, dan kemudian verifikasi bahwa WiFi berfungsi dengan baik.

Anda juga dapat mengunduh dan menginstal driver terbaru dari halaman Intel Download untuk Intel® Dual Band Wireless-AC 8265 . Pengemudi di sana cukup baru dan berasal dari sebulan yang lalu. Saya akan menasihati ini bahkan jika kemunduran di atas berhasil, karena Anda bisa selalu kembalikan lagi.

Beri tahu saya bagaimana kelanjutannya.

Inilah cara saya memperbaikinya tanpa harus memulai ulang.

Sebagian besar masalah adalah bahwa adaptor nirkabel mengalami beberapa kesalahan dan Windows mematikan perangkat.

Arahkan ke Device Manager (Win + X, M). Di bawah "Network Adapters" Anda kemungkinan besar akan melihat bahwa adaptor nirkabel Anda memiliki ikon peringatan (segitiga kuning dengan tanda seru). Jika Anda mengarahkan kursor ke nama adaptor, Anda akan melihat keterangan alat yang mengatakan "Windows mengalami beberapa masalah dengan perangkat ini dan telah menghentikannya" atau hal lain yang menyebabkan itu.

Klik kanan adaptor jaringan nirkabel dan pilih Disable device. Setelah perangkat dinonaktifkan sepenuhnya dan Pengelola Perangkat telah menyegarkan daftar, klik kanan lagi dan klik Enable device.

Itu saja, jaringan wifi Anda akan mulai muncul lagi.

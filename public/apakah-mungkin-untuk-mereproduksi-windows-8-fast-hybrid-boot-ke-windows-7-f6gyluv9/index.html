Apakah mungkin untuk mereproduksi Windows 8 Fast Hybrid Boot ke Windows 7?

Salah satu fitur baru terbaik yang diperkenalkan oleh Microsoft di Windows 8 adalah Boot Hybrid Cepat , yang meningkatkan waktu boot Windows 8 hingga 70% dibandingkan dengan Windows 7:

Pada dasarnya, apa yang dilakukan teknik baru ini adalah melanjutkan file hibernasi yang disimpan dari boot Windows sebelumnya. Berdasarkan fakta bahwa file hibernasi ini sudah ada di Windows 7, dapatkah teknik yang sama digunakan pada sistem Windows 7? Bagaimana saya bisa melakukan hal seperti itu?

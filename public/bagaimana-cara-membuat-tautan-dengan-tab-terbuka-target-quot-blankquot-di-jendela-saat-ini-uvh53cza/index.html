Bagaimana cara membuat tautan dengan tab terbuka target = "_ blank" di jendela saat ini?

Menggunakan Firefox 3.6.8, jika saya memiliki dua atau lebih jendela yang terbuka dan saya mengeklik tautan dengan target = "_ blank", itu selalu membuka tautan di tab baru di pertama jendela.
Bagaimana cara membuatnya agar tautan ini selalu terbuka di tab baru di jendela tempat mereka diluncurkan ?

Saya telah mencoba addon "Tab Mix Plus", dan tidak dapat menemukan cara untuk melakukannya.

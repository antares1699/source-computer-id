Tidak dapat menginstal perangkat lunak antivirus apa pun di Windows 7 64 bit

Saya memiliki sistem Windows 7 64 bit dengan RAM 4 GB dan prosesor Intel i3. Saya mencoba menginstal antivirus gratis Avast 9 dan AVG di dalamnya beberapa kali, tetapi saya tidak bisa. Saya bahkan mencoba menginstalnya sebagai admin, itu juga tidak berhasil.

Setiap kali saya mencoba menginstal keduanya, prompt Windows muncul (saya mengklik 'Ya' untuk itu). Ikon ekstraksi mandiri dari program antivirus muncul dan tidak ada yang terjadi setelahnya. Mengapa saya tidak dapat menginstal antivirus? Bagaimana masalah ini bisa diselesaikan?

Ini adalah tangkapan layar dari pesan kesalahan ketika saya mencoba menginstal Avast 9 dalam mode aman.

Tampaknya masalahnya adalah ketika komputer Anda mengekstrak file karena alasan tertentu itu tidak menjalankan file. Ada dua cara untuk menyelesaikan ini.

Cara pertama dan termudah adalah mencoba perangkat lunak anti virus lain (saya akan mencoba ini dulu).

Cara kedua yang lebih rumit adalah pergi ke windows explore dan ketik %temp% di bilah navigasi dan tekan enter. Maka Anda harus menghapus semua file di folder itu. Jika tertulis This file is in use by anther process maka cukup klik lewati. Kemudian coba instal perangkat lunak anti virus lagi dan itu akan mengekstrak file ke dalam folder %temp%. Kemudian buka folder yang dibuatnya dan coba jalankan beberapa file exe yang disebut sesuatu baris setup.exe atau install.exe atau sesuatu seperti itu.

Semoga membantu. Beri tahu saya jika ini berhasil untuk Anda.

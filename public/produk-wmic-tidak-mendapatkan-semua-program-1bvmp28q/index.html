Saya mendapatkan daftar di mana ditampilkan semua program yang diinstal pada PC saya, tetapi masih tidak semua program terdaftar. Ketika saya pergi ke panel kontrol> Hapus program, saya melihat lebih banyak program

Per Microsoft: Kelas Win32_Product WMI mewakili produk yang diinstal oleh Penginstal Windows.

Karena ada penginstal / pemaket perangkat lunak lain (InstallShield, Wise, NSIS [Nullsoft Scriptable Install System]) yang ternyata tidak mendaftarkan diri mereka sendiri di WMI (karena mereka mungkin mendahului) Saya kira kita harus bertanya di beberapa lokasi :(. Standar berubah, seperti biasanya TI jadi inilah kenyataan yang harus kita tangani. Seperti Anda 've sudah disebutkan dalam komentar Anda dapat meminta registri di lokasi ini untuk semua aplikasi lain: HKLM\Software\Microsoft\Windows\CurrentVersion\Uninstall & HKLM\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall

Saya kira WMI meminta registri di lokasi ini: HKEY_CLASSES_ROOT\Installer\Products

Akhirnya!!!!!

Saya telah mencoba memperbaiki masalah ini selama berhari-hari.

Saya memiliki Office 2013 dan Visio 2013. Saya mencopot pemasangan Office dan Visio untuk penginstalan ulang guna memperbaiki masalah lain yang saya alami dengan Outlook. Selama penginstalan Visio, laptop saya kehabisan baterai dan hibernasi. Sejak itu, penginstalan Visio akan macet. Tidak ada peringatan, tidak ada pesan kesalahan, tidak ada. Jendela penginstalan akan ditutup. Itu menunjukkan dalam daftar program, bahwa Visio masih diinstal, tetapi ketika mencoba menjalankannya, saya menerima pesan yang mengatakan "sistem operasi saat ini tidak dikonfigurasi untuk menjalankan aplikasi ini".

Setelah banyak upaya untuk memperbaiki registri dan menggunakan aplikasi microsoft fixit tertentu, saya mencoba menggunakan kedua perbaikan ini:

http://answers.microsoft.com/en-us/office/forum/office_2013_release-office_install/office-2013-visio-2013-and-ms-project-2013-will/a2dbb2c9-1cff-4afc-b619-a04e18df1289

dan menjalankan perbaikan 2010 di sini:

http://support.microsoft.com/kb/971179

akhirnya penginstalan Visio berhenti mogok dan berhasil diselesaikan.

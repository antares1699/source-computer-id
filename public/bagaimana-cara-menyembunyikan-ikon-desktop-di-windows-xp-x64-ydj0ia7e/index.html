ke nilai DWORD 1

saya menemukan sedikit informasi ini :

Saat Anda mengklik kanan Desktop dan memilih Atur Ikon Menurut sub-menu, tiga opsi ini mungkin hilang:

Tampilkan ikon Desktop / Kunci item web aktif Desktop / Jalankan Pembersihan Desktop Penyihir

Ini terjadi jika kebijakan NoActiveDesktopChanges atau NoActiveDesktop diaktifkan melalui Editor Kebijakan Grup atau melalui Registry. Ikuti langkah-langkah yang diberikan di bawah ini untuk mendapatkan kembali opsi yang hilang:

Klik Start, Run dan ketik REGEDIT. Arahkan ke kunci berikut:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

Klik dua kali NoActiveDesktopChanges dan setel ke 0 Klik dua kali NoActiveDesktop dan setel ke 0 (jika ada)

Tutup Peninjau Suntingan Registri dan mulai ulang Windows.

Ini cara yang mudah

  1. Pergi ke ui tweak pada Control Panel / Performance and Maintenance dan cari desktop selanjutnya UN-check semuanya klik ok.

  2. Ctrl + klik kiri dan dapatkan semua ikon yang tidak ingin Anda tampilkan selanjutnya klik kanan pada ikon apa saja lalu klik properti

    Atau: Klik kanan dan seret pada semua ikon yang tidak ingin Anda tampilkan Anda akan diminta dengan properti di sini klik itu.

  3. Lihat atributnya, klik tersembunyi lalu terapkan, lalu okelah!

Jika Anda masih melihatnya, buka Control Panel / Appearance and Themes dan opsi folder klik pada tampilan mencari file dan folder tersembunyi klik jangan tampilkan file dan folder tersembunyi.

Jika karena alasan tertentu Anda ingin ikon kembali ke Control Panel / Appearance dan Tema dan opsi folder klik pada tampilan mencari file dan folder tersembunyi klik tunjukkan file dan folder tersembunyi lalu klik ok selanjutnya pergi ke desktop lihat ikon lalu ikuti langkah 2 dan 3 dan hanya itu!

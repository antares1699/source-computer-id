Cara menginstal skype di CentOS tanpa akses root

Jadi saya mencoba menginstal Skype di komputer kantor saya yang memiliki CentOS6.4 dan di mana saya tidak memiliki akses root. http://www.skype.com/en/download-skype/skype-for-computer/ hanya menawarkan binari untuk Ubuntu, Debian, Fedora dan OpenSUSE. Saya mencoba yang dinamis, tetapi mengeluh:

skype: error while loading shared libraries: libasound.so.2: cannot open shared object file: No such file or directory

README merekomendasikan untuk menyalin biner skype ke / usr / bin dan menginstal sounds /, lang / dan avatars / ke direktori / usr / share / skype. Jelas saya tidak bisa melakukan ini, karena saya tidak memiliki akses root.

Panduan instalasi CentOS online lainnya selalu mengasumsikan akses root.

Saya bertanya-tanya: bisakah itu dilakukan sama sekali?

Anda tidak membutuhkan pustaka 32-bit. Jika Anda menginstal x86_64 arch CentOS, Anda mungkin harus menginstal beberapa dependensi (paket i686) untuk Skype menggunakan YUM.

Coba cari paket yang berisi pustaka itu libasound.so.2.

Contoh: yum whatprovides libasound.so.2

Keluaran:

alsa-lib-1.0.22-3.el6.i686 : The Advanced Linux Sound Architecture (ALSA)
                           : library
Repo        : installed
Matched from:
Other       : Provides-match: libasound.so.2

Gunakan informasi tersebut untuk mencari dependensi dan kemudian instal.

Contoh: sudo yum install alsa-lib.i686

Akan ada beberapa dependensi i686 untuk skype sehingga Anda harus menjalankan Skype dan menyelesaikan dependensi beberapa kali sebelum Anda membuatnya berfungsi. Semoga berhasil!

PS. Tanpa akses root , Saya tidak berpikir Anda punya kesempatan

Jika Anda mengalami masalah saat menjalankannya, coba versi lain (mis. Anda dapat memiliki versi lain dari yang diperlukan).

Hapus Perlindungan Tulis dari USB

USB flash saya dilindungi dari penulisan dan saya telah mencoba setiap metode yang mungkin saya temui melalui internet, untuk menyingkirkan ini.

Mengakses usb tidak berfungsi. Format dari komputer saya, tidak berfungsi. Hapus atribut dengan DiskPart tidak berfungsi (tidak dapat dibersihkan, Current Read-only State adalah Yes tetapi Readonly disetel ke false ...

Dengan Manajemen Disk, saya tidak dapat menghapus partisi ...

Apakah ada metode lain ... Dengan perangkat lunak tertentu atau entah bagaimana?

Sunting :

USB datang tanpa driver tertentu, ini adalah yang umum yang saya percaya, yang menginstal dan hanya itu. Saya mencoba ini di Windows 8.1, tetapi tidak berfungsi dengan baik di Windows 7.

Ini adalah detail yang diekstrak dengan Flash Drive Information Extractor:

Anda perlu menemukan VID PID flash drive Anda. Anda dapat melakukan ini dengan bantuan FlashGenius atau CheckUdisk dll. Berikut adalah halaman dengan semua utilitas ini http://flashboot.ru/files/vidpid/

Lalu pergi ke http://flashboot.ru/iflash/ Dan cari utilitas khusus untuk flashdisk Anda. Cukup Anda perlu mem-flash ulang pengontrol flashdrive USB Anda.

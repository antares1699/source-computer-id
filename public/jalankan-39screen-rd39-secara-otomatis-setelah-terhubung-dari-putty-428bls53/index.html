Jalankan 'screen -RD' secara otomatis setelah terhubung dari PuTTY

Saya ingin PuTTY secara otomatis menjalankan screen -RD ketika saya masuk, namun saya ingin jika memungkinkan, agar ini dikonfigurasi sebagai bagian dari Putty, atau agar server mengidentifikasi kapan pengguna Putty terhubung dan memaksa screen -RD.

Saya tidak ingin ini terjadi dari kotak lain, karena saya tidak ingin ini mempengaruhi proses otomatis tertentu karena perlu khawatir tentang apakah screen -RD sudah sibuk dan tidak dapat menerima perintah.

Terima kasih!

Opsinya ada di bawah Connection-> SSH di opsi Putty. Cukup letakkan screen -RD di bidang perintah jarak jauh dan itu akan dieksekusi.

Sebuah kata peringatan: pastikan itu benar-benar kombinasi opsi yang Anda inginkan; karena itu akan menutup sesi apa pun yang sudah Anda buka. Anda mungkin ingin menggunakan -Rd atau memiliki beberapa logika berdasarkan keluaran grepping screen -list.

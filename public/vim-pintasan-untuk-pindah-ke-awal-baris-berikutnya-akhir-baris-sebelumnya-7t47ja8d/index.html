Vim: Pintasan untuk pindah ke awal baris berikutnya / akhir baris sebelumnya?

Apakah ada jalan pintas dalam vim untuk pindah ke awal baris berikutnya atau ke akhir baris sebelumnya? Apakah ada cara untuk melakukannya tanpa harus menggunakan $ atau ^?

Juga, adakah cara untuk membuatnya sehingga jika saya menekan right atau l di akhir baris, itu pindah ke awal baris berikutnya? Demikian pula untuk left atau h di awal baris pindah ke akhir baris sebelumnya? Saya menggunakan MacVim jika itu membuat perbedaan.

Untuk pindah ke awal baris berikutnya, tekan Enter. Lihat :help <CR>.

Untuk membuat right dan l membungkus awal baris berikutnya, tambahkan > dan l ke opsi 'whichwrap', mis.,

:set whichwrap+=>,l

Demikian pula, untuk membuat left dan h membungkus ke akhir baris sebelumnya, tambahkan < dan h ke opsi 'whowrap', mis.,

:set whichwrap+=<,h

Lihat :help 'whichwrap'.

Saya tidak tahu pintasan satu tombol untuk pindah ke akhir baris sebelumnya, tetapi dengan pengaturan 'yang membongkar' itu, 0h atau 0left akan berfungsi, dan Anda mungkin dapat memetakan salah satunya ke beberapa kunci yang tidak Anda miliki. sering digunakan.

Untuk pertanyaan pertama Anda, saya tidak mengetahui jalan pintas yang akan melakukan apa yang Anda inginkan, tetapi Anda dapat memetakannya. Sebagai contoh:

:nnoremap <c-p> k$
:nnoremap <c-n> j0

Untuk pertanyaan kedua Anda, lihat ":help 'whichwrap'" (dengan tanda kutip tunggal sebagai bagian dari perintah).

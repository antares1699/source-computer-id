Pencarian berulang untuk file yang menyimpan beberapa teks tertentu

Bagaimana cara membuat file teks secara rekursif yang memiliki daftar semua file di server saya yang berisi string tertentu di mana saja di file tersebut?

Saya tahu perintah berikut dapat digunakan untuk mengganti string secara rekursif

find /var/www -type f -print0 | xargs -0 sed -i 's/old string/new string/g'

Saya tidak ingin mengganti string, saya hanya ingin daftar semua file yang berisi string tersebut.

Gunakan grep alih-alih sed:

find /var/www -type f -print0 | xargs -0 grep -i 'old string'

Dari cara Anda menyusun pertanyaan, sepertinya Anda belum terlalu familiar dengan grep. Baca lebih lanjut tentang opsinya dalam tipe halaman manual: man grep pada baris perintah Anda.

update untuk menjawab komentar - coba tambahkan opsi -l untuk menampilkan nama file saja. -i membuat kasus pencarian tidak peka huruf besar / kecil. Keduanya yang mudah digunakan adalah dengan satu tanda hubung: grep -il

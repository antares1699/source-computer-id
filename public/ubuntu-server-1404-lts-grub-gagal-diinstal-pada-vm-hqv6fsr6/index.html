Ubuntu Server 14.04 LTS - GRUB gagal diinstal pada VM

Saya mencoba menginstal dan menjalankan Ubuntu Server 14.04.1 LTS di VirtualBox untuk tujuan pengujian. Namun, saya bahkan tidak dapat melewati proses instalasi tanpa kesalahan, dan itu akan gagal untuk menginstal boot loader GRUB setiap saat, tidak peduli bagaimana saya memilih untuk mengkonfigurasi sistem.

Saya menjalankan instalasi meninggalkan semuanya dengan nilai defaultnya:

Partitioning Method: Guided - use entire disk and set up LVM
Disk: SCSI3 (0,0,0) (sda) - 2.1 GB ATA VBOX HARDDISK
Amount of volume group to use for guided partitioning: 1.9 GB

The partition tables of the following devices are changed:
   LVM VG dev-vg, LV root
   LVM VG dev-vg, LV swap_1
   SCSI3 (0,0,0) (sda)

The following partitions are going to be formatted:
   LVM VG dev-vg, LV root as ext4
   LVM VG dev-vg, LV swap_1 as swap
   partition #1 of SCSI3 (0,0,0) (sda) as ext2

Write the changes to disk (y/N): y

Automatic updates:
   [*] No automatic updates

Additional packages to install:
   [*] LAMP server
   [*] PostgreSQL server

Semuanya berjalan mulus, hingga:

[!!] Install the GRUB boot loader on a hard disk
   Installation step failed
   An installation step failed. You can try to run the failing item again
   from the menu, or skip it and choose something else. The failing step is:
   Install the GRUB boot loader on a hard disk

Setelah memulai ulang, saya dibiarkan dengan layar hitam tanpa kemungkinan untuk mem-boot-nya secara manual.

Saya tidak bisa benar-benar melihat sumber kesalahan, jadi saya akan senang untuk memikirkan hal ini.

Saya akhirnya menyingkirkan kesalahan itu. Dan sejujurnya, itu bodoh.

Masalahnya sebenarnya adalah, bahwa partisi /boot tidak dibuat, meskipun ditampilkan di UI. Masih mungkin untuk menjalankan semuanya dengan partisi terpandu, tetapi ketika Anda diminta ke Write the changes to the disk?, Anda harus menuju <Back> untuk melihat daftar terperinci dari semua partisi. Saat mengklik <Next>, penginstalan akhirnya akan mengenali dan mempartisi semuanya dengan benar dan seperti yang diharapkan.

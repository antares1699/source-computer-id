Dapatkan umpan RSS / XML untuk Podcast iTunes U untuk digunakan di podcatcher lain

Ada banyak cara untuk mendapatkan umpan podcast untuk podcast iTunes standar seperti ini atau ini , namun, tidak satu pun dari metode ini yang berfungsi pada umpan podcast di iTunes U.

Saya tidak ingin menggunakan iTunes, bagaimana saya dapat menemukan umpan podcast xml alternatif? Ini salah satu contohnya: http://deimos3.apple.com/WebObjects/Core.woa/Feed/fora.tv.1901773207.01901773213

Bagaimana saya bisa berlangganan feed ini di luar iTunes? Saya telah mencoba mengirim email ke penerbit (Fora.tv) berkali-kali tetapi mereka tidak pernah menanggapi.

Anda bisa mendapatkan feed rss menggunakan situs alat php ini

http://itunes.duckdns.org

situs ini adalah solusi berbasis web browser

EX: https://itunes.apple.com/podcast/state-trance-official-podcast/id260190086 ?

muat tautan ini dengan ID https://itunes.apple.com/lookup?id=260190086&entity=podcast

Dapatkan "feedUrl" dari file json ini

situs solusi mempermudah proses ini. Saya menggunakan podcast dengan baik dengan alamat rss yang saya dapatkan.

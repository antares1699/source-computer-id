Edisi Enterprise memiliki beberapa fitur yang lebih kecil yang tidak dimiliki Windows 8 Pro, termasuk Windows To Go, yang memungkinkan Anda membuat versi portabel Windows dan menjalankannya dari flash drive USB. Namun, seperti yang dikatakan Indrek di atas, Enterprise tidak bisa mendapatkan Media Center. Dengan kata lain, tidak ada edisi "lengkap" Windows 8 yang memiliki semua fitur yang memungkinkan.

Jika Anda menganggap Media Center penting, gunakan Windows 8 Pro. Jika tidak, saya akan menganggap Windows 8 Enterprise sebagai paket paling "lengkap".

Anda harus terlebih dahulu mengetahui kebutuhan Anda. Sebelumnya dengan Windows Vista dan bahkan Windows 7 (jika saya tidak salah), Microsoft memasukkan beberapa fitur konsumen yang sangat bagus ke dalam versi Premium & Ultimate saja. Namun tidak demikian halnya dengan Windows 8. Jika Anda ingin merasakan semua fitur konsumen Windows 8, Anda cukup tetap menggunakan Windows 8.

Jika Anda seorang pengembang atau mencari beberapa fitur canggih seperti Hyper-V, Encrypting File System, Remote Desktop server, dll .. Anda harus menggunakan Windows 8 Pro.

Saya pribadi merasa Anda dapat meninggalkan Windows 8 Enterprise. Ini ditargetkan untuk organisasi TI.

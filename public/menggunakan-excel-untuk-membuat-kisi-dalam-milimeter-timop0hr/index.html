Menggunakan Excel untuk membuat kisi dalam milimeter

Mencoba mencari cara untuk membuat grid di Excel yang akan dicetak dengan jarak mm. Belum menemukan cara untuk melakukan ini.

Saya melakukan beberapa pekerjaan dengan Nanoblocks (yang 1x1 tunggal adalah 4mm x 4mm), dan saya ingin memiliki kisi untuk dikerjakan, saya bahkan mungkin membuat tata letak desain di Excel (hanya mengisi sel), tetapi bagian selanjutnya ini akan menjadi manual, hanya benar-benar membutuhkan grid berukuran benar (dicetak).

Untuk mengubah ukuran default menjadi mm:

  • Mengajukan Tab -> Pilihan -> Maju Tab -> Tampilan Bagian -> Unit Penggaris
  • Pilih Millimeters

Untuk mengubah lebar / tinggi kolom / baris (dalam mm):

  • Melihat tab -> Tata letak halaman
  • Pilih semua sel
  • Klik kanan pada Kolom -> Lebar kolom...
  • Klik kanan pada Rows -> Tinggi Baris ...

NAS vs. Windows Home Server [ditutup]

Saya akan berinvestasi sesuatu yang akan membantu saya dengan backup, dll.

Jadi, NAS vs Windows Home Server? Rasanya saya mendapatkan sedikit lebih banyak dengan Windows Home Server - tetapi saya benar-benar tidak yakin.

Dapatkah saya menggunakan klien Mac dengan Windows Home Server atau NAS apa pun?

Banyak hal tergantung pada cakupan proyek Anda.

Jika Anda hanya mencadangkan satu sistem dan hanya itu yang ingin Anda lakukan, NAS mungkin adalah solusi yang tepat.

Jika Anda memiliki banyak komputer, saya sangat merekomendasikan WHS. Sudah menjadi GODSEND di rumah saya. Inilah yang dilakukannya untuk saya:

  • Mencadangkan SEMUA PC saya secara otomatis (1 laptop, 2 desktop, kadang-kadang lebih) tanpa redundansi (yaitu file yang sama pada 3 PCS berarti satu salinan file dan 3 pointer - menghemat banyak ruang)

  • Streaming video ke semua PC dan extender (XBox 360 dan pemutar Blu-Ray yang mendukung MKV)

  • Dengan mudah menggabungkan semua drive menjadi satu 'huruf' sehingga saya tidak perlu mengingat drive mana saya meletakkan file.

  • Memberi saya akses jarak jauh ke PC APAPUN di jaringan. Saya masuk ke desktop saya melalui kotak WHS saya saat berlibur di Florida menggunakan laptop dan wifi hotel.

  • Memberi saya situs web dasar tempat saya dapat mengelola apa pun. Juga memungkinkan saya untuk meletakkan lebih banyak situs web di kotak WHS saya tanpa harus membayar perusahaan hosting.

  • Memberi saya 'pemulihan logam murni'. HD di laptop saya memburuk. Mendapat HD baru. Terpasang. Di-boot melalui CD yang disediakan WHS. Mengidentifikasi kotak WHS saya melalui LAN. Sedikit lebih dari satu jam kemudian, saya memiliki PC yang sepenuhnya pulih tanpa harus menginstal ulang apa pun.

Ada juga add-in lain yang belum sempat saya coba - untuk hal-hal seperti berbagi koleksi iTunes saya di rumah, mengontrol perangkat melalui X10, dll.

Secara pribadi saya akan memilih NAS, sistem NAS modern bisa sangat kuat, yang berarti mereka bahkan dapat mengunduh torrent favorit Anda. Dan host situs web lengkap dengan PHP / mysql dll.

Semua sistem NAS modern akan mendukung MAC / Windows / Linux dll (baik melalui folder bersama atau FTP dll)

Offcourse itu benar-benar tergantung pada anggaran Anda dan apa yang benar-benar ingin Anda lakukan dengannya. Tetapi untuk penggunaan cadangan / penyimpanan, saya pasti akan menggunakan NAS dengan pengaturan RAID untuk keandalan maksimum.

Jika Anda menginginkan server media / streaming dan ekstra lainnya, saya akan lebih condong ke Windows Home Server karena pada akhirnya ini akan menawarkan lebih banyak fleksibilitas dan kekuatan.

Itu tergantung pada firmware di NAS itu. Jika NAS komersial, mereka mendukung semua klien: linux, windows, Mac. Dari linux, Anda dapat menggunakan mount.cifs; dari jendela, itu seperti bagian apa pun; dari Mac, samba share.

Anda dapat membuat NAS Anda sendiri: beli motherboard yang mendukung RAID dan yang memiliki> 4 slot SATA. Beli beberapa HD dari frys atau newegg (Saya baru saja membeli 1 TB 7200rpm, 32mb, barracuda hd seharga $ 60 + pajak dari toko fry). Instal freenas, Anda selesai dengannya.

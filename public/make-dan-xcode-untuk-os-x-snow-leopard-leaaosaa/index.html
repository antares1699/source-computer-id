Make dan Xcode untuk OS X Snow Leopard

Saya memiliki OS X Snow Leopard dan saya perlu menginstal make (ini ada di Xcode). Saya tidak memiliki DVD untuk dipasang.

Di Halaman Xcode Saya tidak menemukan Xcode versi lama.

Bagaimana saya bisa mendapatkan Xcode versi lama?

Xcode 3

Terbaru Xcode 3.2.6 (dengan iOS SDK) tersedia di bawah tautan ini:

https://developer.apple.com/devcenter/download.action?path=/Developer_Tools/xcode_3.2.6_and_ios_sdk_4.3__final/xcode_3.2.6_and_ios_sdk_4.3.dmg

Anda harus mendaftar ke Program Pengembang Apple gratis agar ini berhasil, kurasa. Perhatikan bahwa versi ini tidak akan diperbarui lagi. Jika Anda mengalami masalah saat menginstalnya, lihat pertanyaan ini: Tidak Dapat Menginstal Xcode di OS X Snow Leopard

Daftar semua unduhan juga tersedia. Cukup masuk ke situs web Pengembang Apple dan kunjungi tautan berikut:

http://connect.apple.com/cgi-bin/WebObjects/MemberSite.woa/wa/downloads

Di sini, di bawah "Alat Pengembang" (di bilah sisi kanan), Anda akan menemukan Xcode versi lama dan alat tambahan.

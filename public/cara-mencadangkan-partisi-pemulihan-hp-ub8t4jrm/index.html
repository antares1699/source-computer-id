Cara mencadangkan partisi pemulihan HP

Saya memiliki HP Pavilion dm1, laptop ringan tanpa drive optik.

Ini memiliki partisi pemulihan tempat saya dapat memulihkan edisi Windows 7 Home yang diinstal. Sepertinya ini adalah folder khusus; Saya tidak dapat mengaksesnya dari dalam Windows 7.

Apakah ada cara bagi saya untuk mencadangkan partisi ini (melalui jaringan, atau ke perangkat USB) sehingga saya dapat memulihkannya jika hard drive gagal?

Solusi termudah adalah membuat media pemulihan menggunakan alat pembuatan media pemulihan internal. Ketik recovery media ke dalam menu start dan Anda akan mendapatkan opsi untuk Recovery Media Creation. Jalankan program ini. Ini akan memberi Anda opsi untuk membuat media pemulihan menggunakan DVD kosong atau USB flash drive. Dalam kasus Anda, karena dm1 tidak memiliki drive DVD, Anda dapat menggunakan flash drive. Sayangnya, ini hanya dapat dilakukan sekali per komputer, jadi pastikan Anda melakukannya dengan benar dan tidak kehilangan flash drive.

Jika perlu menginstal ulang komputer Anda dengan image default muncul, Anda akan boot off dari USB flash drive dan dipandu melalui proses reimaging. Komputer Anda akan menjadi seperti saat Anda pertama kali menerimanya.

Anda mungkin juga tertarik untuk menggunakan Windows Backup built-in untuk menjalankan backup biasa. Cadangan ini membuat gambar disk penuh, dan dapat digunakan untuk memulihkan komputer Anda ke kondisi persis seperti saat pencadangan dibuat, bahkan jika Anda harus memulihkan ke disk yang benar-benar baru.

Jika Anda ingin melakukan instalasi ulang yang bersih, Anda juga harus mencadangkan folder C:\swsetup. Di sinilah semua driver untuk komputer Anda disimpan. Banyak dari mereka tersedia secara online, tetapi beberapa mungkin sulit ditemukan, seperti driver WLAN pada beberapa model Pavillion.

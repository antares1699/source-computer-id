Kunci Windows otomatis pada waktu tertentu

Saya mencari perangkat lunak untuk Windows XP atau mekanisme dalam Windows XP yang memungkinkan saya mengunci Windows secara otomatis (efek yang sama seperti WinKey + L) pada waktu tertentu (misalnya pada pukul 3:30 sore) atau setelah pengatur waktu tertentu kedaluwarsa ( misalnya 15 menit).

Saya akan melakukan wawancara dan orang yang diwawancarai akan diberi waktu tetap untuk menyelesaikan beberapa masalah pemrograman dengan PC. Saya suka PC dikunci setelah waktu itu berlalu.

Terima kasih.

Saat masuk sebagai administrator, buka prompt perintah. Gunakan perintah at untuk membuat tugas terjadwal untuk mengunci komputer. Perintah untuk mengunci komputer adalah rundll32.exe user32.dll, LockWorkStation.

Misalnya, at 13:56 /next:monday "rundll32.exe user32.dll, LockWorkStation" akan menyebabkan workstation terkunci pada jam 1:56 Senin depan.

Jika Anda tidak memiliki akses administratif yang diperlukan untuk perintah at, Anda dapat menambahkan tugas terjadwal di panel kontrol untuk melakukan hal yang sama.

Alat baris perintah Windows memiliki perintah yang disebut "SHUTDOWN" yang pada Windows 7 setidaknya memiliki opsi untuk log off daripada mematikan komputer. Untungnya sepertinya ini juga ada di Windows XP!

Perintah shutdown

Anda perlu memiliki sesuatu yang menjadwalkan mesin untuk dimatikan setelah 15 menit, saya tidak akan merekomendasikan menggunakan Penjadwal Windows atau baris perintah "AT" karena menurut pengalaman saya ini tidak dapat diandalkan.

Jika Anda memiliki keahlian pemrograman, buat program sederhana yang menentukan kapan 15 menit telah berlalu dan mengeluarkan perintah SHUTDOWN baris perintah.

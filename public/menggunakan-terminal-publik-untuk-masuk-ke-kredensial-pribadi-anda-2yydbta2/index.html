Menggunakan terminal publik untuk masuk ke kredensial pribadi Anda

Apakah Anda mempercayai terminal publik?

Jika Anda perlu memeriksa email Anda, dan Anda tidak memiliki pilihan lain maka untuk menggunakan terminal publik di dekat Anda, apakah Anda masih akan menggunakannya?

Adakah cara untuk melindungi diri Anda dari virus, keylogger, "evilware" untuk tidak mencuri data pribadi Anda di dalamnya, yaitu username dan password yang Anda masukkan?

Kebanyakan keylogger hanya mencatat kunci, bukan klik mouse, jadi saya memasukkan nama pengguna dan kata sandi saya secara bersamaan sambil juga mengklik di area kosong halaman web dan mengetik charicters sewenang-wenang atau mengklik di mana saya sudah berada. Ini akan berfungsi jika keylogger melacak klik mouse tetapi bukan lokasi atau jika keylogger tidak mengetahui lokasi kontrol login di halaman web. Ini mungkin salah satu cara login yang paling aman, karena sulit untuk dilacak bahkan jika keylogger mengharapkannya.

Misalnya, nama pengguna saya adalah Pengguna dan kata sandi saya Pass. Kemudian, saya akan mengetikkan U atau AS di pengguna, lalu klik lepas kontrol dan ketik sesuatu yang sewenang-wenang, lalu saya mulai dengan kata sandi saya, lalu saya klik di kotak nama pengguna dan ketik huruf lain, lalu saya klik lagi dan selesaikan nama pengguna saya, dll.

Saya sarankan menjalankan keyboard di layar. Jika ini adalah Kios Windows 7, lihat apakah Anda dapat mengaksesnya:

  1. Pilih Mulai
  2. Panel kendali
  3. Akses mudah
  4. Pusat Kemudahan Akses.

Saya tidak akan khawatir tentang malware yang memaku komputer Anda, karena sebagian besar kios dirancang untuk berjalan dalam mode memori terlindungi. Namun saya khawatir tentang keylogger, karena Anda tidak pernah tahu apakah seseorang telah menempatkannya di kios. Jika Anda tidak dapat mengaksesnya, pastikan Anda sering mengubah kata sandi dan menggunakan karakter non alfanumerik sebagai bagian dari kata sandi Anda.

Sedangkan untuk Evilware, kami lebih tertarik pada berita daripada file Anda.

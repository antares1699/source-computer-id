Utilitas baris perintah resmi bzip2 tersedia untuk Windows.

Instalasi:

  1. Unduh dan ekstrak file bzip2-1.x.x-win-x64.zip untuk Windows 64 bit, atau bzip2-1.x.x-win-x86.zip untuk Windows 32 bit.

  2. Instal msvcr120.dll menurut instruksi :

    Semua biner bergantung pada Visual Studio 2013 C Runtime Library (msvcr120.dll). Ini dapat diinstal menggunakan Paket Visual C ++ yang Dapat Didistribusikan Ulang untuk Visual Studio 2013 pemasang.

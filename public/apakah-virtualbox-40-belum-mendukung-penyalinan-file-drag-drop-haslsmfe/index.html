Apakah VirtualBox 4.0 belum mendukung penyalinan file drag-drop?

Versi 4.0.0 akan menjadi rilis besar baru. Fitur baru utama berikut telah ditambahkan:
Pengaturan -New / tata letak file disk untuk portabilitas VM; lihat manual untuk informasi lebih lanjut.
-Buka Dukungan Virtualization Format Archive (OVA); lihat manual untuk informasi lebih lanjut.
-VMM: mendukung lebih dari 1,5 / 2 GB RAM tamu pada host 32-bit
-Bahasa binding: binding Java seragam untuk API lokal (COM / XPCOM) dan jarak jauh (SOAP) -invocation
-Chipset: dukungan tambahan untuk chipset Intel ICH9 dengan 3 bus PCI, PCI express dan -Message Signaled Interrupts (MSI)
-Audio: Intel HD Audio sekarang tersedia sebagai perangkat keras tamu, untuk dukungan yang lebih baik dengan sistem operasi tamu modern (misalnya Windows 64-bit; bug # 2785).
-GUI: antarmuka pengguna yang didesain ulang dengan pratinjau jendela tamu
-GUI: mode tampilan baru dengan tampilan tamu yang diturunkan skalanya
Kontrol -Resource: menambahkan dukungan untuk membatasi waktu CPU VM dan bandwidth IO.
-Storage: mendukung asynchronous I / O untuk iSCSI, VMDK, VHD dan gambar Parallels
-Penyimpanan: dukungan untuk mengubah ukuran gambar VDI dan VHD
-Windows Additions: dukungan untuk secara otomatis memperbarui Guest Additions (memerlukan Windows Guest Additions 4.0 atau yang lebih baru)
-Tambahan Tamu: dukungan untuk menyalin file ke sistem file tamu

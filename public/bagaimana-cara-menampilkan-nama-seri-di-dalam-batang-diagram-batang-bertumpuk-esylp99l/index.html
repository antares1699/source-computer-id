Bagaimana cara menampilkan nama seri di dalam batang diagram batang bertumpuk?

Saya memiliki diagram batang bertumpuk yang sangat sibuk, sekitar 20 seri atau lebih dari waktu ke waktu

Saya tidak tertarik dengan angka sebenarnya, saya hanya ingin menunjukkan tren keseluruhan dari waktu ke waktu.

Warna default di Excel 2010 sangat mirip sehingga hampir tidak mungkin untuk memilih rangkaian individu

Satu-satunya pilihan tampaknya adalah menambahkan 'label data' yaitu nilai numerik yang mendasari - ini bukan yang saya inginkan

Saya ingin menunjukkan nama seri (kode 3 digit pendek) yang harus dengan mudah masuk ke dalam (atau di sebelah) bagian dari batang bertumpuk

Ini juga berarti saya akan dapat menghapus legenda, membebaskan lebih banyak ruang untuk area plot

Dengan asumsi data Anda disiapkan dengan benar, Anda harus dapat menambahkan pencapaian ini dengan:

  1. Tambahkan Label Data
  2. Format Label Data
    • Periksa Label Contains ke Nama Seri
    • Hapus centang Label Contains Value
    • Periksa Label Position ke Inside End
  3. Ulangi untuk setiap seri
  4. Hapus legenda (yang hampir selalu merupakan praktik yang baik).

Tapi sepertinya bagan Anda mencoba melakukan terlalu banyak hal tentang bagaimana pembaca akan dapat memahaminya. Sudahkah Anda mempertimbangkan alternatif? Untuk memperlihatkan tren dari waktu ke waktu, seringkali Bagan Garis lebih cocok daripada Bagan Kolom. Anda juga dapat mempertimbangkan untuk menggunakan Penumpukan di bagan Anda (mis. Tumpukan menurut tanggal, bukan seri). Anda juga dapat mempertimbangkan kelipatan kecil / bagan panel, atau membuat sebagian besar rangkaian memiliki format yang serupa (mis. Skala abu-abu), dengan bidang minat yang disorot dalam warna.

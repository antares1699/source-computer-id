Seberapa efisien menggunakan WinRAR untuk melindungi file dengan kata sandi?

Saya memiliki file dengan data penting yang ingin saya lindungi dengan kata sandi. Saya pada dasarnya ingin dimintai kata sandi setiap kali saya mencoba membuka file. Saya mencoba TrueCrypt tetapi sepertinya tidak meminta kata sandi setiap kali Anda mencoba membuka file (yah, kecuali saya melewatkan sesuatu).

Sebagai solusi sederhana, saya memutuskan untuk menggunakan WinRAR untuk tugas ini (dengan menambahkan kata sandi ke arsip). Ini bekerja dengan baik dan semuanya tetapi saya memiliki sedikit perhatian. Setiap kali saya membuka file dengan WinRAR itu mendekompresi file tersebut ke folder sementara. Sekarang saya khawatir datanya bisa diakses di folder sementara. Saya tidak tahu apakah WinRAR menghapus konten folder sementara nanti, tetapi meskipun demikian, konten masih dapat diakses oleh beberapa utilitas (salah satu utilitas yang memungkinkan Anda untuk melihat / menyimpan file yang dihapus) jadi tidak ' Bagi saya, sepertinya bukan pilihan yang aman.

Pertanyaan saya adalah, apakah pemahaman saya tentang cara kerja WinRAR sudah benar? Jika ya, apakah ada cara sederhana (dan aman) lain untuk melindungi file dengan kata sandi?

Truecrypt sebenarnya adalah pendekatan yang cukup bagus.

Ia memiliki opsi "cache recent used password", sehingga jika Anda me-mount file Anda, kemudian melepasnya, mount lagi - itu tidak akan meminta password kedua kalinya untuk menghemat waktu Anda. Anda dapat (dan dengan suara hal-hal yang Anda inginkan) mematikan ini, dan titik mana yang mungkin melakukan apa yang Anda inginkan.

Anda benar, jika keamanan menjadi perhatian, maka WinRAR bukanlah pilihan yang aman. Ini dirancang lebih untuk mengarsipkan atau mengirim file daripada penyimpanan yang aman. RAR adalah format yang dikenal dengan banyak implementasi - Anda perlu mengetahui detail implementasi spesifik yang Anda gunakan untuk menjamin keamanan.

Anda masih bisa menggunakan TrueCrypt jika Anda ingin mengetikkan kata sandi setiap kali Anda menggunakan file. Cukup buat volume dengan file rahasia, pasang ketika Anda perlu menggunakannya, dan lepaskan setelah selesai.

Ada juga opsi untuk Auto-turun setelah tidak ada data yang dibaca / ditulis selama X menit , yang dapat Anda gunakan untuk menghilangkan langkah turun.

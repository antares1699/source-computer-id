Bagaimana cara menggunakan GUI 7-Zip untuk Windows untuk membuat file .tar.gz?

Bagaimana cara menggunakan GUI 7-Zip di Windows untuk membuat file .tar.gz?

4
Dherik — Sumber

    — Dherik

1 Jawaban

7
7

Seperti yang dijelaskan sini :

Karena arsip terkompresi GZIP hanya dapat berisi satu file, 7-Zip tidak menawarkan kompresi GZIP jika Anda mencoba mengompresi satu atau beberapa folder file sekaligus. Saat Anda memilih TAR, Anda tidak dapat memilih kompresi apa pun algoritma.

Caranya, Anda harus membuat arsip TAR terlebih dahulu. Anda kemudian dapat memasukkan file TAR tunggal itu ke dalam 7-Zip dan akhirnya akan menawarkan Anda kemampuan untuk mengompres file itu menggunakan GZIP.

Mulai Outlook dengan skrip .vbs berikut ini:

Set WshShell=Wscript.CreateObject("Wscript.Shell") 
WshShell.Run """C:\Program Files (x86)\Microsoft Office\Office14\OUTLOOK.EXE""",7,FALSE

Untuk membuat skrip:

  1. Klik Mulai, ketik 'Notepad' ke dalam kotak pencarian dan tekan Enter.
  2. Salin dan tempel kode di atas ke Notepad.
  3. Klik Mengajukan , Menyimpan
  4. Beri nama file (termasuk tanda kutip) "Start Outlook.vbs"

Kamu bisa gunakan SplashKiller

Ini berfungsi untuk Office XP dan 2003. Saya harap ini berfungsi untuk Office 2007!

SplashKiller memungkinkan Anda menyembunyikan layar splash yang muncul saat aplikasi sedang dimuat.

Ini menyelamatkan Anda dari keharusan untuk:

  1. Temukan opsi 'tanpa splash screen' (jika ada) dalam aplikasi.
  2. Ubah registri Anda untuk mematikan layar splash.
  3. Retas aplikasi itu sendiri untuk menghapus layar splash.

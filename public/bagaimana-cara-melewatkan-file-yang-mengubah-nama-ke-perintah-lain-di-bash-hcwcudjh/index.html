Bagaimana cara melewatkan file yang mengubah nama ke perintah lain di Bash?

Saya sering menggunakan wget untuk mengunduh file tarball dan zip dari web, lalu untar lalu atau gunzip. Saya melakukan:

wget google.com/somefile.zip
unzip somefile.zip
rm somefile.zip

Apakah ada cara bagi saya untuk secara otomatis meneruskan file zip ke tar atau unzip SAAT wget-ting? Dalam pseudocode:

wget google.com/somfile.zip && unzip

Kedua jawaban di sini berfungsi, dan jawaban @RedGrittyBrick menghilangkan file temporer yang tetap Anda hapus (saya akan menerimanya). Saya akan menambahkan cara lain.

Anda dapat melakukan manipulasi string pada variabel yang Anda gunakan untuk URL untuk mendapatkan file keluaran.

URL=google.com/somefile.zip
wget $URL
FILE=${URL##*/} # strip everything up to last /
unzip $FILE
rm -f $FILE

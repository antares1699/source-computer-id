Enkripsi Profil Thunderbird

Adakah cara untuk mengenkripsi profil Mozilla Thunderbird saya (email yang disimpan dan semacamnya) untuk melindunginya dengan kata sandi?

Solusi seperti menggunakan enkripsi Windows atau TrueCrypt tidak akan berfungsi karena saya hanya ingin mengenkripsi konten file, tidak menggunakan fitur khusus sistem file.

Dengan TrueCrypt:

  1. Buat penampung berbasis file baru.
    1. Di jendela utama, Buat volume
    2. Buat penampung file terenkripsi → Volume standar
    3. Pilih tempat Anda ingin menyimpannya. (Saya memiliki AppData.tc di direktori pengguna saya.)
    4. Terima algoritma enkripsi default.
    5. Pilih seberapa besar volume yang Anda inginkan.
    6. Masukkan kata sandi, atau pilih file kunci, atau keduanya.
    7. Format volumenya. (Saya pribadi memilih NTFS sebagai filesystem, untuk beberapa keandalan.)
      • Meskipun berbasis file, containernya masih memiliki filesystem standar.
      • Istilah Linux adalah "pemasangan loop".
    8. Klik Keluar .
  2. Di jendela TrueCrypt utama, buka volume yang baru dibuat.
    1. Menggunakan Pilih file
    2. Pilih huruf drive kosong dari daftar besar
    3. Klik Meningkat
    4. Anda dapat melakukan langkah ini sebagian besar secara otomatis Favorit → Tambahkan Volume Terpasang ke Favorit .
  3. Pindahkan profil Thunderbird Anda.
    1. Salinan profil saat ini dari folder AppData Anda ke drive yang Anda pilih di 2.2
      • Biasanya pada %APPDATA%\Thunderbird\Profiles dan memiliki nama yang mirip dengan mbqbp1tq.default
      • Setelah menyalin, ganti nama menjadi Thunderbird profile atau semacamnya, untuk menghindari kebingungan nanti.
    2. Hapus profil lama dengan aman.
      • saya dulu seperti Penghapus , sampai menerima penulisan ulang lengkap dan menjadi tidak nyaman untuk digunakan "tapi sekarang .NET!"
      • Sekarang saya tetap dengan sdelete .
    3. Beri tahu Thunderbird tentang lokasi baru. Itu disimpan di %APPDATA%\Thunderbird\Profiles.ini, tetapi ada cara yang lebih mudah untuk memperbaruinya:
      1. Mulailah → Lari → masukkan thunderbird -profilemanager
      2. Hapus profil Anda saat ini. Klik Jangan hapus file ; Anda sudah menukarnya di langkah 3.2.
      3. Klik Buat profil , masukkan nama apa saja (seperti default), dan klik Pilih folder .
      4. Pilih lokasi profil terenkripsi Anda dari langkah 3.1.
  4. Mulai Thunderbird.

Jika Anda memutuskan tidak menyukai TrueCrypt, ada FreeOTFE, yang sebagian besar bekerja dengan cara yang sama.

Dengan bawaan Windows Mengenkripsi Sistem File : Jangan bingung dengan BitLocker.

Anda menyebutkan bahwa Anda tidak ingin menggunakan fitur khusus sistem file, tetapi fitur tersebut terkadang berguna.

  1. Jelajahi folder pengaturan Thunderbird Anda. Biasanya %APPDATA%\Thunderbird.
  2. Klik kanan pada Profiles, pilih Properti .
  3. Maju → Enkripsi konten → baik → baik
  4. Mulai Thunderbird.
  5. Cadangkan kunci enkripsi . Anda hanya perlu melakukannya sekali untuk akun Windows Anda.
    1. Mulai → Lari → certmgr.msc
    2. Pribadi → Sertifikat
    3. Temukan satu dengan "Encrypting File System" di kolom "Intended Purposes" nya.
    4. Klik kanan, Semua tugas → Ekspor
    5. Klik Ya, ekspor kunci privat
    6. Masukkan kata sandi enkripsi untuk kunci yang diekspor, dan pilih tempat untuk meletakkannya.
  6. Oh, satu hal lagi. Anda harus menghapus data lama yang tidak terenkripsi. Saya menggunakan cipher /w:C: untuk menghapus semua ruang yang tidak terpakai, tetapi bahkan satu kali berlalu membutuhkan waktu lama ...

Sisi negatifnya - EFS hanya tersedia di Windows * Professional dan yang lebih baru.

Dari komentar:

Itu hanya Cara lain (selain enkripsi transparan, seperti di atas) adalah dengan membangun kemampuan crypto ke dalam Thunderbird itu sendiri. Dan mengingat kompleksitas programnya, memang begitu tidak sebuah solusi.

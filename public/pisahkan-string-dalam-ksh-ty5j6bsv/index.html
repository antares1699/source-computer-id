Pisahkan string dalam ksh

Saya mencari cara cepat / mudah untuk membagi string di ksh.

Ini memiliki pemisah angka (spasi) yang bervariasi di antara setiap item.

Contoh:

value1 value2                  value3

Ada saran / saran?

Menggunakan for loop dengan string input akan terbagi pada spasi kosong.

LIST="value1  value2 value3"
for x in $LIST ; do
    echo $x
done

Hasil

value1
value2
value3

atau

LIST="value1  value2 value3"
set -A STRING "$LIST"
for x in $STRING ; do
    echo $x
done

Hasil

value1
value2
value3

Perpipaan melalui sed dan grep opsional juga berfungsi

LIST="value1  value2 value3" ;
echo "$LIST" | sed 's/[[:space:]]/\n/g' | grep .

Anda dapat menghapus grep jika Anda juga menghapus tanda kutip yang menghapus banyak spasi meskipun pemformatan lain mungkin dilakukan.

LIST="value1  value2 value3" ;
echo $LIST | sed 's/[[:space:]]/\n/g'

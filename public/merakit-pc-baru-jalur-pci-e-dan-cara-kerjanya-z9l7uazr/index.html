Merakit PC baru - jalur PCI-E dan cara kerjanya

Saya ingin membuat PC game baru yang dapat bertahan lama. Saya ingin menggunakan PCI-E x8 SSD + GTX 780 SLI dalam build ini.

Sekarang saya mencari di web untuk melihat apa yang saya butuhkan untuk mendukung ini dalam hal motherboard dan CPU.

Penyiapan pertama saya adalah:

  • CPU - i7-5820K (Mendukung PCIe 3.0, dengan 28 jalur)
  • Motherboard - ASRock X99 Extreme4 (Dukungan untuk 36 jalur, dengan cpu 28 jalur, slotnya akan menjadi 16x-8x-4x)

Jadi di sini saya akan berpikir bahwa pengaturan saya dengan pci-e x8 ssd tidak akan berfungsi karena slot ketiga hanya pci-e x4.

penyiapan kedua:

  • CPU - i7-5930K (Mendukung PCIe 3.0, dengan 40 jalur)
  • Motherboard - ASRock X99 Extreme6 (Dukungan untuk 40 jalur, dengan cpu 40 jalur, slotnya akan menjadi 16x-16x-8x)

di sini saya akan berpikir ini tidak masalah dengan pci-e x8 ssd. tapi juga jauh lebih mahal. dan saya membaca berbagai hal tentang cara kerja jalur pci-e.

jadi tolong bisakah seseorang membantu saya di sini dan memberi tahu saya bagaimana hal-hal jalur pci-e ini benar-benar berfungsi dan jika saya benar dalam pengaturan saya.

-EDIT- Terima kasih untuk semua komentarnya, benar-benar membantu saya memahami sedikit cara kerja. dan saya pikir saya telah memutuskan apa yang akan saya beli untuk saat ini. Terima kasih!

Slot PCI-E dapat memiliki 1, 4, 8, atau 16 jalur. CPU mendukung sejumlah jalur tertentu.

Istilah "jalur" digunakan karena setiap jalur tidak digunakan bersama dengan perangkat lain, dan inilah mengapa lebih cepat daripada PCI standar. Bandwidth I / O pada standar PCI yang lebih lama dibagikan di antara semua perangkat di bus. Perangkat seperti adaptor grafis PCI atau adaptor Ethernet yang banyak digunakan dapat memenuhi bus, yang menyebabkan perselisihan dan perlambatan. Salah satu alasan AGP dikembangkan adalah untuk meletakkan kartu grafis pada bus 1 perangkatnya sendiri.

Jelas, lebih banyak jalur = lebih banyak kecepatan. Kartu grafis biasanya berukuran 16x.

Kartu dapat dimasukkan ke dalam slot dengan jalur yang lebih sedikit dari yang dibutuhkan (jika komponen motherboard tidak menghalangi), tetapi akan beroperasi lebih lambat. Kartu dapat dimasukkan ke dalam slot dengan lebih banyak jalur daripada yang dibutuhkannya, tetapi Anda kehilangan jalur ekstra. Itu tidak akan membuat perangkat lebih cepat.

Beberapa chipset melakukan hal-hal aneh dengan jalur PCI-E. Saya pernah mendengar tentang beberapa motherboard (akan diperbarui jika saya menemukan info) tergantung pada jenis kartu yang Anda miliki di slot lain, slot 16x dapat menjadi slot 8x.

Jalur pada dasarnya membawa bandwidth dalam satu arah atau keduanya (dua arah). Kinerja untuk bermain game antara 8x dan 16x dapat dinegosiasikan dengan kartu grafis saat ini, mungkin beberapa keuntungan FPS dalam situasi tertentu.

Adapun pengaturan Anda, yang pertama seharusnya baik-baik saja menurut perhitungan saya, slot 16x dan 8x tidak akan mengalami masalah dalam menjalankan kartu.

Ini membuat Anda memiliki 4x untuk SSD Anda yang memberi Anda kecepatan teoretis bandwidth 2000MB / dtk, sekarang dari pemahaman saya ini adalah tentang batas kartu PCI-e SSD terbaru jadi saya pikir Anda mungkin bisa lolos begitu saja di hal kinerja.

Namun, Anda harus memikirkan tentang pemeriksaan di masa mendatang, bagaimana jika Anda menginginkan SSD yang lebih cepat di masa mendatang, maka penting untuk memiliki 8x ekstra.

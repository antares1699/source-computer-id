Sembunyikan layar masuk windows 10 IOT

Saya ingin sistem saya melakukan boot langsung ke aplikasi tanpa elemen UI lainnya.

Saya telah mengatur sistem untuk boot menggunakan Antarmuka Pengguna Kustom dan langsung boot ke aplikasi tetapi selama proses login itu menunjukkan nama pengguna, gambar, memuat "titik" dan latar belakang.

Apakah ada cara untuk menjadikannya sebagai layar kosong?

Saya tidak memiliki pengalaman dengan windows 10 IoT, tetapi metode biasa untuk melewatkan layar login dan untuk login langsung saat boot ke akun pengguna adalah untuk masuk ke Panel Kontrol -> Akun Pengguna, klik akun pengguna Anda untuk memilihnya dan hapus centang pada kotak berlabel "Pengguna harus memasukkan nama pengguna dan sandi untuk menggunakan komputer ini".

Anda akan diminta memasukkan sandi akun pengguna - masukkan sandi dan klik OK.

Jika opsi ini tidak ada di Windows Iot, berikut ini cara melakukan hal yang sama melalui registri (regedit):

  • Navigasi ke HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon
  • Masukkan 1 sebagai nilai AutoAdminLogon
  • Masukkan nama komputer (atau domain) sebagai nilai DefaultDomainName
  • Masukkan nama akun dengan nilai DefaultUserName
  • Masukkan kata sandi akun dengan nilai DefaultPassword

Info lebih lanjut dapat ditemukan sini .

Untuk menonaktifkan Layar Selamat Datang Windows 10 di Editor Kebijakan Grup:

  • Jalankan gpedit.ms
  • Arahkan ke: Konfigurasi Komputer> Template Administratif> Sistem> Logon
  • Klik dua kali "Jangan tampilkan layar pembuka Memulai saat logon"
  • Pilih "Nonaktifkan"
  • Klik OK

Lain kali Anda me-restart komputer, komputer Anda akan secara otomatis melewati layar login Windows 10.

Buka halaman Proses pada halaman IoT Utilities dan ketik perintah ini. Ini harus menonaktifkan logo Windows.

  1. bcdedit /set quietboot on
  2. bcdedit /set bootux Disabled
  3. bcdedit /set bootuxdisabled on

Ini adalah satu-satunya pengaturan yang berhasil bagi saya untuk menyembunyikan layar selamat datang secara visual di Windows 10:

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Embedded\EmbeddedLogon" /v "HideAutoLogonUI" /t REG_DWORD /d 1 /f

Saya menggunakan pengaturan ini bersama dengan AutoAdminLogon dari sini .

Informasi selengkapnya tentang kunci registri ini dapat ditemukan di MSDN: https://docs.microsoft.com/en-us/windows-hardware/customize/desktop/unattend/microsoft-windows-embedded-embeddedlogon

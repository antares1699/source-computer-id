Bagaimana cara melihat lebar / tinggi piksel dari sebuah lapisan di CS5

Saya ingin dapat memilih sebuah lapisan, dan melihat lebar / tinggi dari lapisan yang dipilih di Photoshop CS5. Saya yakin ada cara mudah untuk melakukan ini, tetapi untuk kehidupan saya, saya tidak dapat menemukannya. Saya sudah mencoba memasukkannya ke mode transformasi, tetapi itu hanya menunjukkan 100% untuk membiarkan saya menyesuaikannya dengan persentase.

Jika kamu Ctrl -klik ( Cmd -klik di OS X) lapisan di Lapisan jendela, itu akan memilih piksel yang terlihat di lapisan ini.

Kemudian, periksa Informasi Anda ( INFO ) jendela. Seperti biasa, lebar dan tinggi pilihan saat ini akan ditampilkan.

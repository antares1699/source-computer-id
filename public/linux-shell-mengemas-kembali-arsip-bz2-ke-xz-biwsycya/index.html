Linux shell: mengemas kembali arsip bz2 ke xz

Bagaimana cara mengemas ulang arsip .bz2 ke format .xz, jika saya ingin menyimpan stempel tanggal file dan pengaturan chmod-chownnya?

Saya dapat menggunakan alat standar apa saja dan xz, lzma, p7zip

Anda perlu memecahnya menjadi dua bagian

1) Konversi dari bz2 ke xz, yang dijawab oleh billc.cn

2) Perbaiki kepemilikan, izin, dan stempel waktu file xz agar sesuai dengan bz2.

chown "`stat --printf '%u:%g' file.bz2`" file.xz
chmod "`stat -c '%a' file.bz2`" file.xz
touch -a -d "`stat -c '%x' file.bz2`" file.xz
touch -m -d "`stat -c '%y' file.bz2`" file.xz

Anda cukup menggunakan bzcat untuk menghapus kompresi bzip2 tetapi tetap menggunakan format tar dan kemudian menyalurkan hasilnya ke xz:

bzcat file.tar.bz | xz > file.tar.xz

Anda dapat menambahkan opsi kompresi Anda ke xz seperti biasa, tetapi tinggalkan nama file sehingga input standar yang digunakan.

  • dan bagaimana "menyimpan stempel tanggal file dan pengaturan chmod-chownnya?"
    — osgx
  • Info meta file disimpan dalam file tar dan tidak ditangani oleh algoritma kompresi. Jika Anda tidak menyentuh tar, maka tidak akan ada yang berubah. Jika Anda mengompresi file tanpa tar, Anda sudah kehilangan semua info meta.
    — billc.cn
  • maaf, saya tidak bertanya tentang "tar.bz2" tapi tentang "bz2". Tar ditambahkan dalam jawaban Anda oleh Anda. Saya ingin menyimpan stempel File.bz2 (File.xz harus memiliki stempel yang sama).
    — osgx
  • Dalam hal ini pertanyaannya tidak ada hubungannya dengan bzip atau xz karena format ini TIDAK menyimpan cap waktu di file arsip. Saat Anda mendekompresi, stempel waktu dari file yang dikompresi akan digunakan untuk keluaran. Untuk menyetel stempel waktu file, Anda dapat menggunakan touch. ls memiliki opsi tambahan untuk menampilkan waktu {pembuatan, akses, modifikasi} lengkap dari file asli Anda.
    — billc.cn
  • Bukan di dalam file, tapi dari arsip itu sendiri. Saya ingin mengemas kembali banyak bzip2 backup dengan xz dan timestamp arsip adalah informasi berharga dalam kasus saya.
    — osgx
  • Kemudian Anda dapat menggunakan ls (atau stat) untuk mendapatkan tanggal dari bzip2 dan menggunakan touch untuk mengaturnya pada xz baru. Anda mungkin harus menulis skrip shell untuk ini.
    — billc.cn
  • sulit untuk mengurai keluaran ls karena dalam format yang berbeda. Saya benar-benar memiliki banyak file dan tidak dapat mengonversi secara manual; file terlama 4+ tahun yang lalu, lebih baru beberapa minggu.
    — osgx
  • Dalam hal ini gunakan stat -c %X untuk mendapatkan nilai mentahnya. Ubah X untuk Y dan Z, untuk mendapatkan modifikasi dan mengubah waktu. Lihat halaman manual untuk stat.
    — billc.cn

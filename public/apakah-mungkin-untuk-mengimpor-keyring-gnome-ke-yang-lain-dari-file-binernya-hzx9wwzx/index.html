Apakah mungkin untuk mengimpor keyring (gnome) ke yang lain dari file binernya

(Pertanyaan ini awalnya diposting di AskUbuntu, tetapi tidak mendapat jawaban yang bagus: https://askubuntu.com/questions/217709 )

Saya baru-baru ini menyiapkan kotak Ubuntu menggunakan desktop Gnome, dan ingin mengimpor data yang telah saya simpan dari komputer Ubuntu lama. Data ini termasuk gnome keyring, yang saya (agak bodoh) menyalin file yang dienkripsi alih-alih mengekspor.

Apa yang saya lakukan

cp -R ~ / .gnome2 / keyring / backup / gnome2_keyring

Mengingat saya masih mengetahui sandi sesi yang saya gunakan untuk akun pengguna terkait di komputer lama, apa yang dapat saya lakukan untuk memulihkan / mengimpor sandi / data dari keyring?

Ini adalah pertanyaan pengguna rumahan, dan harus dilakukan hanya sekali: Saya tidak peduli dengan masalah keamanan: Saya akan menghapus ruang kosong saya setelahnya dan itu sudah cukup.

Itu tergantung pada seberapa terkini kuda laut itu. Untuk kuda laut yang cukup lama, Anda cukup menyalin file keyring yang diberikan ke direktori ~ / .gnome2 / keyring yang baru. Untuk kuda laut yang lebih baru, Anda harus menyalin file keyring ke ~ / .local / share / keyrings.

Versi terbaru kuda laut menjengkelkan karena Anda harus secara eksplisit membuka menu tampilan dan menyatakan bahwa Anda ingin melihat dengan keyring atau hanya akan menampilkan kata sandi untuk keyring yang sudah terbuka, tetapi setelah Anda melakukannya, perubahan itu seharusnya berfungsi. Sebelum saya menemukan ini, sepertinya kuda laut tidak melihat gantungan kunci.

Menemukan file pada disk yang terpasang dengan lokasi

Saya memiliki disk SSD kecil dengan sistem linux saya, dan disk IDE besar dengan sebagian besar pekerjaan saya, yang dipasang saat sistem dimulai. Untuk beberapa alasan saya tidak dapat menemukan file pada disk IDE menggunakan locate. Saya tidak tahu apakah kesalahan terletak pada saya tidak memberikan tanda yang benar ke locate atau jika disk IDE diabaikan oleh updatedb ( yaitu tidak diindeks sehingga tidak dapat dicari). Bagaimana saya bisa melakukannya?

Saya menemukan file jika saya pergi ke folder mount dan menjalankan

$ find . -name "my_pattern"

tapi ini cukup lambat.

  • Lihat /etc/updatedb.conf Anda. Ini berisi PRUNEFS dan PRUNEPATHS yang mungkin mengecualikan drive IDE Anda. Jika ya, ubah file itu sehingga tidak mengecualikan drive Anda. Dan kemudian jalankan updatedb. Jika ini tidak berhasil atau Anda tidak tahu apa yang harus diubah dalam file itu, harap sertakan baris dari mount yang memiliki drive IDE Anda, dan konten /etc/updatedb.conf.

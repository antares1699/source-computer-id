Windows: mengunduh file dari baris perintah

Saya ingin mengunduh file dari tautan yang diberikan. Bagaimana cara melakukannya dari baris perintah di Windows? Ini adalah file zip ware kursus terbuka MIT yang ukurannya lebih besar dari yang diizinkan di jaringan kami. Atau apakah ada situs resmi tempat saya dapat mengunduh file ini?

Kamu bisa mendapatkan Wget untuk Windows dan gunakan dari baris perintah.

Tapi, apa yang Anda maksud dengan 'lebih besar dari yang diizinkan di jaringan Anda'?
Bagaimana penerapannya? (bagaimana Anda melihatnya membatasi unduhan ini misalnya).

Lebih lanjut tentang komentar Anda:
Jika Anda mendapat balasan terlalu besar, Anda dapat mempertimbangkan unduhan multi-bagian.
Banyak akselerator mendukung itu - DownThemAll adalah salah satunya untuk Firefox.

Saya juga menyarankan untuk berbicara dengan admin squid Anda untuk menjelaskan kebutuhan Anda untuk mengunduh.
Mereka mungkin membuat pengecualian, atau (jarang)
bahkan mengunduhnya dan menyimpannya di area bersama lokal;
jika unduhan itu legal seperti yang Anda sarankan.

Terakhir, Anda tidak terlalu membutuhkan baris perintah untuk ini.
MultiGet membungkus fitur wget di GUI.

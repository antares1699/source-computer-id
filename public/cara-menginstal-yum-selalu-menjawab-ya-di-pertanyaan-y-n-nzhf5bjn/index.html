Cara menginstal yum selalu menjawab ya di pertanyaan y / n

Di Linux Centos 6-64 bits, ketika kita menginstal sebuah paket, ia meminta kita untuk menekan y atau n kadang-kadang ditambah enter seperti ini: "Bolehkah? [Y / n]".

Apa yang harus kita tambahkan dalam perintah "yum install sesuatu" sehingga selalu ketika akan menghentikan instalasi untuk menanyakan y / n itu selalu berjalan secara otomatis pada "y" (ya) selalu, tanpa berhenti bertanya?

Bagaimana cara menjalankan yum install dan selalu menjawab yes?

Gunakan opsi baris perintah -y atau --assumeyes:

-y, --assumeyes
    Assume yes; assume that the answer to any question which would be asked is yes.
    Configuration Option: assumeyes

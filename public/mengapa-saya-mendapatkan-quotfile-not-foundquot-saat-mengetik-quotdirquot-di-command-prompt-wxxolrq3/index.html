Mengapa saya mendapatkan "File Not Found" saat mengetik "dir" di command prompt?

Di mesin Windows XP saya, saya mengetik dir ke dalam command prompt di direktori tertentu. Itu mulai mendaftar banyak file tetapi kemudian berhenti dan kemudian ditampilkan Berkas tidak ditemukan .

Adakah yang tahu mengapa itu terjadi dan bagaimana cara memperbaikinya?

Jika ada entri yang rusak dalam daftar direktori, kesalahan mungkin terjadi. Coba lagi dan coba temukan di mana iklan tersebut dibatalkan. Gunakan Explorer sebagai ganti command prompt dan periksa daftar file. Terkadang file yang dilindungi -virus / antivirus dan Sony melakukannya- tidak dapat dikenali oleh sistem file.

Mungkin Anda tidak dapat menyalin atau menghapus SEMUA file di direktori itu. Jika tidak penting, coba hapus semua file. Pada akhirnya Anda mungkin menemukan entri "kotor", setelah menghapus semua file / dirs yang jelas.

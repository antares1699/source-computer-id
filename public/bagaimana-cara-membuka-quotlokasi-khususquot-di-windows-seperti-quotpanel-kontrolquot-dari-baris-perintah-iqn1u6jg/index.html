Bagaimana cara membuka "lokasi khusus" di Windows, seperti "Panel Kontrol" dari baris perintah?

Jika salah satu membuka Windows Explorer dan kemudian memasukkan "Control Panel" di bilah lokasi, maka Panel Kontrol Windows akan mulai. Demikian pula, string lokasi "Panel Kontrol \ Panel Kontrol NVIDIA" memulai Panel Kontrol NVIDIA.

Apakah ada cara untuk mengakses ini dari baris perintah? Berikut ini tidak berhasil:

explorer "Control Panel\NVIDIA Control Panel"

Ini lebih mudah dari yang Anda kira, Anda hanya perlu mengetahui nama yang benar dari aplikasi panel kontrol.

Aplikasi panel kontrol biasanya berupa file dengan ekstensi .CPL di direktori Windows.

Jumlahnya terlalu banyak, dan mereka dapat bervariasi tergantung pada apa yang dipasang.

Wikipedia memiliki daftar yang bagus dari aplikasi Windows resmi dan banyak aplikasi bagian ketiga yang lebih besar.

Untuk menemukan semuanya, Anda dapat pergi ke command prompt dan pastikan berada di root drive sistem dan ketik

dir *.cpl /s

Ini akan mencantumkan semua file panel kontrol pada mesin Anda.

Jadi untuk menjalankan panel kontrol Nvidia, menurut tautan Wikipedia, Anda harus mengetik nvidia.cpl selama lokasi file tersebut dilalui. (Saya tidak dapat menguji ini karena saya hanya memiliki kartu ATI di sini saat ini)

Item Panel Kontrol biasanya memiliki nama applet terdaftar yang terkait dengannya. Anda harus membaca dokumentasi untuk item pihak ketiga atau melakukan pencarian untuk mereka.

Sini Anda dapat menemukan daftar pintasan applet panel kontrol yang lebih umum. Untuk NVIDIA, daftar nvidia.cpl untuk mengakses panel kontrol nvidia, yang dapat saya konfirmasi berfungsi pada sistem Windows 7 saya. Setidaknya dalam beberapa sistem, ini adalah nvcpl.cpl. Yang berguna lainnya untuk dijalankan dari baris perintah adalah sysdm.cpl yang menampilkan properti sistem.

nvcplui.exe

adalah panel kontrol nVidia "asli". Itu dapat ditemukan di:

"C:\Program Files\NVIDIA Corporation\Control Panel Client\nvcplui.exe"

Dan gunakan

Show-ControlPanelItem

Untuk membuka item Panel Kontrol.

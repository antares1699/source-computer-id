Cara mengedit file Windows .url

Saya memiliki beberapa file .url. Saya tahu itu adalah file teks biasa sederhana (cfr. http://www.fmtz.com/formats/url-file-format/article )

Tetapi saya tidak dapat menemukan cara untuk mengeditnya.

Misalnya, menggunakan Notepad ++ di Windows 8: mengklik Buka file, masuk ke folder tempat file .url berada dan memilihnya terjadi hal yang aneh: mulai "unduh" dari (saya kira) folder temp lokal dan kemudian saya mendapatkan ini kesalahan:

Membingungkan ...

Ada ide?

Berikut ini cara mudahnya:

  1. Buka Windows "Command Prompt"
  2. Arahkan ke folder yang berisi file ".URL" file yang ingin Anda edit.
    Untuk contoh ini, katakanlah Anda ".URL" file bernama "Example.url"
  3. Ubah nama file menjadi Menambahkan Sebuah ".txt" perpanjangan.
    C:\>ren "Example.url" "Example.url.txt"
  4. Biarkan jendela "Command Prompt" terbuka untuk nanti.
  5. Edit file seperti biasa Anda mengedit file ".txt" file, menggunakan editor teks favorit Anda.
  6. Kembali ke jendela "Command Prompt" yang terbuka.
  7. Ubah nama file kembali seperti semula, dengan menghapus ".txt" perpanjangan.
    C:\>ren "Example.url.txt" "Example.url"


Jika Anda akan mengedit .URL file lebih sering, Anda dapat menambahkan "Notepad.exe" , atau editor teks favorit Anda, ke "context-menu" Anda lihat ketika Anda "Right Click" pada salah satu ".URL" mengajukan.

Untuk melakukan ini, Anda perlu melakukan beberapa modifikasi sederhana pada registri Anda. Ada beberapa cara untuk melakukan ini. Ini 2 di antaranya ...

Metode 1:

Membuat ".reg" mengajukan dengan informasi dan penggunaan yang diperlukan "regedit.exe" untuk "import" itu ".URL" mengajukan.

Pertama, buat dokumen teks baru dan edit dengan editor teks favorit Anda. Salin dan tempel berikut ini ke dalam file:

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\IE.AssocFile.URL\Shell\Open URL as Text]
@="Open with Notepad"

[HKEY_CLASSES_ROOT\IE.AssocFile.URL\Shell\Open URL as Text\Command]
@="C:\\Windows\\System32\\Notepad.exe %1"

Di atas dikonfigurasi untuk digunakan "Notepad.exe" sebagai editor yang akan dijalankan saat Anda menggunakan file "context-menu" item untuk mengedit a ".URL" mengajukan. Jika Anda ingin menggunakan editor teks yang berbeda, lakukan perubahan ini pada dokumen teks baru di atas sebelum Anda menyimpannya:

  1. Bagian yang mengatakan @="Open with Notepad" mendefinisikan apa yang akan muncul di "context-menu" bila Anda "Right Click" pada a "Notepad.exe" mengajukan. Anda dapat mengubahnya menjadi "judul" apa pun yang Anda inginkan dengan mengubahnya "Open with Notepad" untuk "Whatever you want" .
  2. Bagian yang mengatakan @="C:\\Windows\\System32\\Notepad.exe %1" mendefinisikan program yang akan dijalankan saat Anda menggunakan yang baru "context-menu" item untuk mengedit a ".URL" mengajukan. Anda dapat mengubahnya untuk menggunakan editor teks favorit Anda dengan mengubahnya "C:\\Windows\\System32\\Notepad.exe" ke jalur dan nama editor teks yang Anda inginkan. Perhatikan bahwa Anda perlu menggandakan garis miring terbalik ( "\" ) di jalan Anda seperti \\ seperti yang ditunjukkan. Jangan lupa tinggalkan "ruang" dan "%1" pada akhirnya.

Jika menggunakan "Notepad" tidak apa-apa dengan Anda, maka Anda dapat meninggalkan file seperti yang ditunjukkan di atas.

Dari editor teks Anda, simpan yang baru ".reg" file yang Anda buat. Sebagai contoh, Anda dapat menyimpannya sebagai file "EditURLfile.reg" .

Selanjutnya, lari "Regedit" . Biasanya akan ditempatkan di sini: "C:\Windows\regedit.exe" .

Dari dalam "Regedit" , klik item menu: File -> Import , dan navigasi untuk memilih yang baru ".reg" file yang Anda buat ( ".URL" ) dan klik "Regedit" .

Sekarang Anda dapat menutupnya "Regedit" . Sekarang, ketika Anda "Right Click" di mana saja ".URL" file, Anda akan melihat item baru di "context-menu" membuka ( edit ) berkas.


Metode 2:

Jika Anda tidak nyaman menggunakan "Regedit" untuk menambahkan "context-menu" , Anda dapat melakukannya langsung dari Windows "Command Prompt". Anda akan menempelkan 2 baris berikut ke Windows "Command Prompt":

reg add "HKCR\IE.AssocFile.URL\Shell\Open URL as Text" /t REG_SZ /d "Open with Notepad"
reg add "HKCR\IE.AssocFile.URL\Shell\Open URL as Text\Command" /t REG_SZ /d "C:\Windows\System32\Notepad.exe %1"

Di atas dikonfigurasi untuk digunakan "Notepad.exe" . Jika Anda ingin menggunakan editor teks yang berbeda, lakukan perubahan ini pada 2 baris di atas.

  1. Bagian yang mengatakan "Open with Notepad" : Ubah ini menjadi "judul" apa pun yang Anda inginkan untuk yang baru "context-menu" barang.
  2. Bagian yang mengatakan "C:\Windows\System32\Notepad.exe %1" : Ubah ini untuk menentukan jalur dan nama editor teks yang Anda inginkan. Perhatikan bahwa untuk metode ini, Anda memerlukan satu garis miring terbalik ( "\" ) di jalur Anda seperti yang ditunjukkan. Seperti di atas, jangan lupa tinggalkan "ruang" dan "%1" pada akhirnya.

Jika menggunakan "Notepad" tidak masalah bagi Anda, maka Anda dapat meninggalkan 2 baris seperti yang ditunjukkan di atas.

  1. Buka Windows "Regedit"
    Bergantung pada penyiapan "Policies" di komputer Anda, Anda mungkin harus membuka "Command Prompt" sebagai "Administrator" ("Right Click" pada "cmd.exe" dan pilih. "Run as Administrator").
  2. Tempel 2 baris di atas (satu per satu) ke jendela "Command Prompt".
    Untuk setiap baris yang Anda tempel, Anda akan mendapatkan pesan konfirmasi yang ditampilkan:
    The operation completed successfully.

Sekarang, ketika Anda "Right Click" pada salah satu ".URL" file, Anda akan melihat item baru di "context-menu" membuka ( edit ) berkas.

Untuk file URL Anda dapat mengklik kanan dan memilih "Properties", dan di jendela pilih tab "Web Document" dan di sini Anda dapat mengedit URL.

EDIT: Jika Anda ingin membuka file sebenarnya untuk mengubah hal lain selain URL sebenarnya, Anda cukup drag dan drop file .URL ke notepad atau notepad ++.

Jika file URL menunjuk ke file yang valid (seperti file teks) membuka langsung dengan notepad akan membuka file shortcut yang ditunjuk. Untuk mengedit file url yang sebenarnya, geser + klik kanan pada beberapa spasi di folder tempat file 'url' berada (bukan pada file sebenarnya, Anda menginginkan spasi kosong). Klik 'buka lokasi file di [command prompt / powershell]'.

Setelah command prompt (atau PowerShell) terbuka, jalankan

notepad.exe filenamehere.url

Anda dapat menggunakan dir terlebih dahulu untuk melihat nama file lengkap dengan ekstensi.

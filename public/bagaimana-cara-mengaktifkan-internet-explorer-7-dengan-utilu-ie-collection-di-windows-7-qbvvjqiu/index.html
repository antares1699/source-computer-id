Bagaimana cara mengaktifkan Internet Explorer 7 dengan Utilu IE Collection di Windows 7?

Saya tidak dapat mengaktifkan Internet Explorer 7 dengan Koleksi Utilu IE . Saya memiliki Windows 7 64-bit. Apa yang dapat saya?

Saya yakin Anda harus menggunakan Virtual PC atau simulasi Sistem Operasi lainnya. Koleksi IE ini juga merupakan harapan terakhir saya.

Modus pengembang dari IE9 rendering IE7 adalah tidak sama sebagai IE7 yang sebenarnya! Hal yang sama berlaku untuk IE8. Anda hanya dapat mengujinya untuk pemosisian sederhana dari elemen (padding, margin, dll. Ditampilkan dengan baik dalam mode pengembang). Tetapi hal-hal lain ...

Saya akan menamai Anda satu contoh sederhana sebagai bukti, tetapi saya telah menemukan perbedaan yang jauh lebih banyak, yang akan gagal di IE7 dan IE8 tetapi tidak akan ditampilkan di versi IE9 dari browser tersebut:

Css sebaris: background-image: url(testimage.png);filter: alpha(opacity=50); akan gagal di semua versi <IE9.

Dengan demikian, Anda pasti tidak bisa mengandalkan mode pengembang.

Memulai Emacs hanya sekali dengan file berbeda di Windows

Saya ingin emacs terdaftar dengan ekstensi file di windows, tetapi hanya satu contoh emacs yang harus dibuka, bahkan jika saya membuka beberapa file melalui ekstensinya.

Saya telah menginstal emacs dari emacs-22.3-bin-i386.zip dengan membongkar ke c: \ bin \ emacs pada sistem windows server 2003 (32 bit)

Saya membuat skrip c: \ bin \ emacs \ bin \ emacs-filetarget.cmd

@echo off
"%~dp0emacsclientw.exe" -na "%~dp0runemacs.exe" "%1"

Ketika saya membuka file pertama * .el dengan c: \ bin \ emacs \ bin \ emacs-filetarget.cmd itu akan dibuka dalam contoh emacs baru yang baik-baik saja.

Tetapi file kedua * .el dibuka dalam contoh emacs kedua. Saya mencari cara agar hanya memiliki satu contoh emacs di mana semua file berikutnya dibuka.

Anda perlu memastikan server emacs telah dimulai. Tambahkan berikut ini ke file .emacs Anda:

(server-start)

Dan kemudian selalu gunakan emacsclientw.exe untuk memulai emacs.

Sebagai referensi, saya menggunakan file emacs.cmd yang terlihat seperti ini:

@echo off
if "%1"=="" goto nofile

start "" "c:\emacs\bin\emacsclientw.exe" -n %*
goto end

:nofile
start "" "c:\emacs\bin\runemacs.exe

:end

Instance "runemacs" adalah karena emacsclientw mengharuskan Anda menentukan nama file.

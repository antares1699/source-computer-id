Skrip Bash yang menjalankan perintah hanya jika port bebas

Hai, saya memiliki skrip bash yang memerlukan eksekusi bersyarat dari beberapa baris kode berdasarkan apakah port 80 sudah digunakan:

sudo git fetch origin;
sudo git checkout master;
sudo git pull;

--- jika port 80 terbuka

echo Starting Meteor;
export LC_ALL=C;
export ROOT_URL=$ROOT_URL;
sudo meteor --port 80;

--- lain

echo Meteor already running;

Anda bisa menggunakan:

netstat -ln | grep ": 80"

Jika kode yang dikembalikan ($?) Adalah 0 maka ada sesuatu di port: 80, jika tidak, tidak. Jadi contohnya:

netstat -ln | grep ":80 " 2>&1 > /dev/null 
if [ $? -eq 1 ]; then   
     ... your code here 
fi

Bagaimana cara memuat aturan iptables saat memulai?

Baru saja menginstal Arch Linux dan menyimpan beberapa aturan iptables menggunakan iptables-save tetapi tidak dimuat saat memulai. Apa artinya ini?

File unit systemd menunjuk ke lokasi di mana konfigurasi aturan akan disimpan

Kata pengantar: Membuat IPTables menjadi persisten antara reboot dapat menjengkelkan karena proses / prosedur dasar tampaknya sangat bervariasi antara distro Linux. Jadi jangan khawatir jika ini tampak membingungkan; ini. </em>

Menurut apa yang saya ketahui tentang IPTables dan Arch Linux — dan sejujurnya, terutama dari wiki Arch Linux resmi —Aturan IPv4 default harus disimpan di sini:

/etc/iptables/iptables.rules

Jadi jika Anda menyimpan file iptables.rules di direktori home pengguna Anda cukup salin dari direktori home ke tujuan tersebut seperti ini:

sudo cp ~/iptables.rules /etc/iptables/iptables.rules

Dan setelah selesai, restart IPTables melalui systemctl seperti ini:

sudo systemctl restart iptables

Itu hanya akan segera memulai ulang IPTables, tetapi Anda mungkin ingin mem-boot ulang mesin untuk menguji apakah ini berfungsi.

Jika tidak berhasil, mungkin karena IPTables tidak diaktifkan sebagai layanan startup di sistem Anda. Untuk mengaktifkannya sebagai layanan startup, jalankan saja perintah ini:

sudo systemctl enable iptables

Dan kemudian restart dan IPTables Sebaiknya aktif dan berjalan dengan aturan Anda di /etc/iptables/iptables.rules dimuat dengan benar.

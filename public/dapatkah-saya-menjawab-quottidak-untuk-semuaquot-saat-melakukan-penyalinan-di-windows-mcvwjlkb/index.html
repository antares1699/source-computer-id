Dapatkah saya menjawab "Tidak untuk Semua" saat melakukan penyalinan di Windows?

Saat menyalin / menggabungkan struktur direktori besar di bawah Windows, apakah mungkin untuk melakukan sesuatu seperti "Ya untuk semua" tetapi secara terbalik sehingga semua file yang ada tidak ditimpa?

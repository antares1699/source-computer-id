Bagaimana cara menggunakan Dropbox dengan Truecrypt?

Saya memiliki beberapa file yang harus dapat diakses dari banyak komputer, tetapi dienkripsi di setiap komputer. Sesuatu seperti fitur 'folder bersama' di Dropbox, tetapi dengan folder yang dienkripsi secara lokal di setiap komputer.

Apa yang saya cari:

  • Folder yang dibagikan antara beberapa komputer yang menjalankan Vista atau Mac OS X
  • Folder ini dienkripsi dari setiap komputer
  • File yang tidak dienkripsi dapat diakses saat masuk ke beranda Dropbox.

Pada dasarnya, saya kira saya ingin mengenkripsi folder Dropbox di masing-masing komputer, tetapi hanya Dropbox yang menyinkronkan konten yang tidak terenkripsi.

Apakah ini mungkin? Dan tolong tanyakan jika di atas tidak jelas.

Edit untuk menambahkan : Ini tidak memungkinkan akses file melalui situs web. / Edit

Inilah yang saya lakukan:

  1. Buat TrueCrypt wadah volume file dan taruh di file folder dropbox.
  2. Minta TrueCrypt.exe dapat diakses di masing-masing file menghitung (bisa juga di folder DropBox Anda).
  3. Saat Anda perlu mengaksesnya, pasang file sebagai drive di komputer yang dihosting.

Beberapa hal yang perlu diketahui:

  • DropBox tidak akan memperbarui file sampai TrueCrypt telah melepaskannya.
  • jika Anda akhirnya memasangnya di dua lokasi, DropBox akan menyebutnya "versi yang berkonflik" dan Anda harus menyelesaikan perbedaannya. (Ketika saya melakukan ini, saya memasang kedua file pada saat yang sama dan membedakannya menggunakan BeyondCompare3.)
  • Fakta bahwa DropBox menyimpan versi historis dari file kontainer secara teoritis dapat membahayakan enkripsi (meskipun saya tidak dapat mendukungnya dengan kutipan sekarang).

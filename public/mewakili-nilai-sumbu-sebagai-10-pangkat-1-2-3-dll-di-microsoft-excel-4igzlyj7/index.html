Mewakili nilai sumbu sebagai 10 pangkat 1, 2, 3, dll… di Microsoft Excel

Saya memiliki bagan berikut di tangan saya:

Saya ingin itu mengganti nilai sumbu X dan Y yang saat ini dimilikinya dengan 10 pangkat notasi bilangan bulat. Saya tidak ingin notasi ilmiah (eksponensial). Terima kasih sebelumnya.

Anda harus melakukan sedikit pekerjaan pemformatan manual, karena Excel tidak memiliki format angka yang Anda inginkan.

Anda bisa membuat grafik terlihat seperti ini dengan sedikit usaha:

1 - Format angka khusus

Klik kanan pada nomor sumbu, pilih "format sumbu", pergi ke bagian "angka", dan masukkan format Kustom berikut:

"10^"#

Buat sumbu Anda menggunakan format kustom ini.

Ini akan menambahkan teks "10 ^" ke depan nomor yang ditampilkan.

2 - Loggifikasi data Anda

Format kustom dari 1) mengasumsikan bahwa data Anda sudah dicatat. Jadi kami perlu mencatat data Anda, dan membuat grafik sebagai gantinya. Temukan log dari setiap titik data:

3 - Perbaiki garis log

Plot log Excel mengasumsikan bahwa baris log utama harus di 1, 10, 100, 1000 dll. Tetapi Anda ingin baris utama di 1, 2, 3 karena Anda sudah mencatat data Anda.

Jadi kita perlu membuat garis kita sendiri, dan memformatnya menjadi garis abu-abu tipis.

Pertama, hapus garis kisi dari plot saat kita membuatnya sendiri.

Kemudian tambahkan pasangan titik data yang menentukan awal dan akhir setiap baris setiap 10, 100, 1000 dll, lalu catat datanya. Letakkan spasi di antara setiap pasangan titik untuk memutus garis dan juga memudahkan untuk melihat apa yang terjadi. Anda harus mendapatkan tabel seperti:

Jadi pendekatannya adalah menambahkan deret dummy di sepanjang setiap sumbu, di tempat yang Anda inginkan label sumbu. Sembunyikan titik-titik ini, dan tambahkan label data, masukkan 101, 102, dll (untuk 10 ^ 1, 10 ^ 2, dll), dan format eksponen menjadi superskrip. Dan ini sulit dilakukan dengan tangan, karena sulit untuk memilih eksponen dan menerapkan pemformatan, di antara hal-hal sulit lainnya.

Jadi saya rutin menulis sedikit. Pilih plot log-log dengan sumbu di tepi kiri dan bawahnya, dan jalankan kode di bawah ini.

Sub NiceExponentialAxisLabels()
  Dim cht As Chart
  Dim iPt As Long, iLog As Long, iMin As Long, iMax As Long
  Dim vXVals As Variant, vYVals As Variant
  Dim dFont As Double

  Set cht = ActiveChart

  ' HORIZONTAL AXIS ------------------------------------
  cht.Axes(xlCategory).TickLabels.NumberFormat = ";;;" ' hide tick labels

  ' build arrays of X and Y values
  iMin = WorksheetFunction.Log10(cht.Axes(xlCategory).MinimumScale)
  iMax = WorksheetFunction.Log10(cht.Axes(xlCategory).MaximumScale)
  ReDim vXVals(1 To 1)
  ReDim vYVals(1 To 1)
  iPt = 0
  For iLog = iMin To iMax
    iPt = iPt + 1
    ReDim Preserve vXVals(1 To iPt)
    ReDim Preserve vYVals(1 To iPt)
    vXVals(iPt) = 10 ^ iLog
    vYVals(iPt) = cht.Axes(xlValue).MinimumScale
  Next

  ' add series, hide points, add and format labels
  With cht.SeriesCollection.NewSeries
    .Name = "horizontal"
    .XValues = vXVals
    .Values = vYVals
    .Format.Line.Visible = False
    .MarkerStyle = xlMarkerStyleNone
    .HasDataLabels = True
    .DataLabels.Position = xlLabelPositionBelow
    For iPt = 1 To .Points.Count
      With .DataLabels(iPt)
        dFont = .Font.Size
        .Text = 10 & WorksheetFunction.Log10(vXVals(iPt))
        With .Characters(3, Len(.Text) - 2)
          .Font.Superscript = True
          .Font.Size = dFont + 2
        End With
        With .Characters(1, 2)
          .Font.Size = dFont
        End With
      End With
    Next
  End With

  ' VERTICAL AXIS ------------------------------------
  cht.Axes(xlValue).TickLabels.NumberFormat = "_0_0_0_0_0_0_0" ' hide but maintain margin

  ' build arrays of X and Y values
  iMin = WorksheetFunction.Log10(cht.Axes(xlValue).MinimumScale)
  iMax = WorksheetFunction.Log10(cht.Axes(xlValue).MaximumScale)
  ReDim vXVals(1 To 1)
  ReDim vYVals(1 To 1)
  iPt = 0
  For iLog = iMin To iMax
    iPt = iPt + 1
    ReDim Preserve vXVals(1 To iPt)
    ReDim Preserve vYVals(1 To iPt)
    vXVals(iPt) = cht.Axes(xlCategory).MinimumScale
    vYVals(iPt) = 10 ^ iLog
  Next

  ' add series, hide points, add and format labels
  With cht.SeriesCollection.NewSeries
    .Name = "vertical"
    .XValues = vXVals
    .Values = vYVals
    .Format.Line.Visible = False
    .MarkerStyle = xlMarkerStyleNone
    .HasDataLabels = True
    .DataLabels.Position = xlLabelPositionLeft
    For iPt = 1 To .Points.Count
      With .DataLabels(iPt)
        dFont = .Font.Size
        .Text = 10 & WorksheetFunction.Log10(vYVals(iPt))
        With .Characters(3, Len(.Text) - 2)
          .Font.Superscript = True
          .Font.Size = dFont + 2
        End With
        With .Characters(1, 2)
          .Font.Size = dFont
        End With
      End With
    Next
  End With
End Sub

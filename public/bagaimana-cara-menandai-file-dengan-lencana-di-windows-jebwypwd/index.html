Saya sedang mencari solusi serupa ketika saya menemukan utas ini. Saya menemukan cara yang cukup sederhana untuk mencapai apa yang ingin Anda lakukan (setidaknya dalam kasus khusus ini.) Mungkin itu akan membantu orang lain yang melakukan ini.

Jika Anda memiliki izin yang sesuai, Anda dapat melihat file "Tersembunyi" di Windows. Mereka muncul sebagai ikon / thumbnail berwarna abu-abu / semi-transparan dari dokumen asli.

Pertama, pastikan Anda dapat melihat File Tersembunyi.

  1. Buka Opsi Folder dengan mengklik tombol Mulai, mengklik Panel Kontrol, mengklik Penampilan dan Personalisasi, lalu mengklik Opsi Folder.
  2. Klik tab Tampilan.
  3. Di bawah Pengaturan tingkat lanjut, klik Tampilkan file dan folder tersembunyi, lalu klik OK.

Sekarang yang perlu Anda lakukan adalah mengklik kanan setiap episode setelah melihatnya, pergi ke Properties, dan centang kotak "Hidden". Itu akan tetap di sana tetapi berwarna abu-abu, memungkinkan Anda untuk mempertahankan tempat Anda!

Bersulang!

Bagaimana cara membuat tautan default Google Chrome di Layar Mulai Windows 8 membuka Chrome di desktop?

Secara default, penginstalan baru Google Chrome akan menempatkan entri di Layar Mulai yang ketika diklik akan membuka Chrome versi UI / Metro Modern. Ini membuat frustrasi, karena saya ingin membukanya di desktop saya. Apakah ada cara untuk memperbaikinya?

Ah, saya melacaknya. Buka regedit dan atur semua kunci ini menjadi kosong:

HKEY_CLASSES_ROOT\ChromeHTML\shell\open\command\DelegateExecute

HKEY_CLASSES_ROOT\Chrome\.exe\shell\open\command\DelegateExecute

HKEY_CLASSES_ROOT\Chrome\.exe\shell\opennewwindow\command\DelegateExecute

HKEY_CLASSES_ROOT\Chrome\.exe\shell\run\command\DelegateExecute

Setelah Anda menutup regedit, saat Anda membuka Chrome dari layar awal, Chrome akan terbuka di desktop Anda.

Ini mungkin permulaan sampai menjadi lebih jelas apa yang Anda minta. Acrobat 8 ​​(dan dengan demikian, versi selanjutnya) menyertakan kemampuan untuk mengemas file dari berbagai jenis dalam file pdf: http://help.adobe.com/en_US/Reader/8.0/help.html?content=WSE034CA46-D08F-4fff-AA3C-FF04510DAEF0.html

DIPERBARUI untuk menanggapi komentar: Saya melakukan pencarian google untuk "membuat paket pdf" dan tampaknya ada beberapa aplikasi yang harganya lebih murah dari Acrobat yang menawarkan kemampuan ini. Saya tidak dapat menemukan sumber terbuka atau opsi gratis untuk melakukan ini. Ghostscript mencoba menjadi sistem berfitur lengkap untuk membuat PDF dan dengan lebih banyak penelitian saya yakin ini adalah tempat terbaik untuk mencari pembuatan paket PDF open source.

Utilitas baris perintah Gratis (seperti dalam kebebasan) dan gratis (seperti dalam bir) pdftk tersedia untuk semua platform OS utama (Unix, Linux, Windows, Mac OS X). Ini memiliki sub-perintah untuk menyematkan file eksternal ke dalam PDF. Coba ini:

 pdftk.exe ^
   original.pdf ^
   attach_files file1.docx file2.xls file3.html ^
   to_page 3 ^
   output file-with-attachments.pdf

File terlampir akan ditunjukkan sebagai milik halaman 3 dari PDF keluaran. Untuk menghapus file yang disematkan dalam PDF, coba ini:

pdftk.exe ^
  file-with-attachments.pdf ^
  unpack_files ^
  output c:\path\to\directory\

Semua lampiran kemudian akan disimpan di direktori keluaran bernama.

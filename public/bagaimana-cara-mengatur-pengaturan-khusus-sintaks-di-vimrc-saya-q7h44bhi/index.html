Bagaimana cara mengatur pengaturan khusus sintaks di VIMRC saya?

Saya bekerja dengan beberapa bahasa dan markup di vim setiap hari (ruby, python, javascript, CSS, HTML, dll), dan ingin memiliki pengaturan yang berbeda untuk setiap buffer ketika saya menjalankan editor saya. Bagaimana cara mendeteksi sintaks mana yang dimuat di buffer saat ini?

Secara khusus, saya lebih suka python indent menjadi 4 spasi, sementara bahasa lain ditemukan dengan 2. Saya membayangkan sesuatu seperti ini di file .vimrc saya:

if syntax == 'python'
  set softtabstop=4
  set shiftwidth=4
else if syntax == 'html'
  " ...
endif

Ada yang seperti itu di vim? Terima kasih.

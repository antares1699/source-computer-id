Apakah ada Pendengar Registri Windows gratis? [Tutup]

Saya mengalami masalah dengan aplikasi yang memiliki banyak tempat bisa simpan pengaturan tertentu tetapi tampaknya pengaturan tersebut terkadang tidak melekat.

Saya cukup yakin mereka disimpan di Windows Registry dan oleh karena itu saya ingin tahu apakah ada file Gratis alat yang "memonitor" / mendengarkan penulisan / pengeditan registri.

Idealnya menggunakan alat seperti itu akan membantu saya mengidentifikasi bagian mana dari aplikasi yang menyimpan pengaturan dengan benar dan mana yang tidak (untuk alasan apa pun).

Sebagai catatan, saya tidak perlu mencegat / memblokir atau mengubah pengeditan, saya hanya ingin tahu kapan dan di mana pengeditan itu terjadi.

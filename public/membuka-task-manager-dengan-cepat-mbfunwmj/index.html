Membuka Task Manager dengan cepat

Untuk membuka Task Manager, saya tekan Ctrl+Alt+Del dan kemudian pilih Mulai Manajer Tugas .

Saya lebih suka memiliki pintasan di desktop saya dan menghindari Ctrl+Alt+Del. Bagaimana cara melakukannya?

4
4
  1. Klik kanan di desktop Anda
  2. Pilih "Baru" dan "Pintasan"
  3. Untuk lokasi, ketik "taskmgr.exe" dan klik "Next"
  4. Ketik nama untuk pintasan tersebut, seperti "Pengelola Tugas"
  5. Klik "Selesai"

Nikmati pintasan desktop baru Anda yang luar biasa ke Task Manager!

  • Tekan Ctrl-Shift-Escape untuk membuka Task Manager.

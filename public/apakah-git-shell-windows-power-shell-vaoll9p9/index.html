Jika Anda menggunakan msysgit maka git shell sebenarnya adalah shell msys yang menggunakan MinGW dan menyediakan implementasi bash shell untuk windows.

Karena msys memanfaatkan MinGW, ia juga menyediakan implementasi ulang beberapa program GNU standar yang ditulis untuk windows yang berguna saat mengoperasikan git.

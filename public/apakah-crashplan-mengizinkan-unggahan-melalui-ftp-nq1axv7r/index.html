Apakah CrashPlan mengizinkan unggahan melalui FTP?

Saya mendapatkan versi gratis dari CrashPlan dan saya mencadangkan komputer saya satu sama lain. Saya penyedia web hosting memungkinkan saya menggunakan penyimpanan online 50 GB untuk pencadangan, dan saya ingin mengintegrasikannya dengan CrashPlan tetapi saya tidak melihat opsi untuk melakukannya.

Apakah saya baru saja melewatkan pengaturannya, atau apakah CrashPlan benar-benar tidak menawarkan untuk menyimpan cadangan di lokasi FTP?

Sesuai FAQ mereka , ini tidak didukung:

Bisakah saya mencadangkan menggunakan FTP?

Tidak. Saat ini, CrashPlan harus diinstal dan dijalankan di tujuan. Kami mungkin mendukung FTP di beberapa titik, tetapi itu akan jauh lebih lambat dan kurang dari penyimpanan terjamin daripada menjalankan CrashPlan. Dengan CrashPlan berjalan di kedua ujungnya, perangkat lunak tujuan akan mensimulasikan pemulihan dan memvalidasi file saat tujuan menganggur dan tidak digunakan. Ini masalah besar - sesuatu yang tidak dapat Anda lakukan dengan FTP secara efisien - dan memastikan bahwa file Anda dalam kondisi yang baik saat Anda membutuhkannya

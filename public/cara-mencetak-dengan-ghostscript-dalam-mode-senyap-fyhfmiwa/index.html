Cara mencetak dengan ghostscript dalam mode senyap

Saya menggunakan perintah Ghostscript berikut untuk mencetak dokumen postscript:

C:\Program Files\gs\gs9.14\bin>gswin64c.exe -sDEVICE=mswinpr2 testprinter.ps

Perintah ini membuka dialog Cetak di mana saya dapat mengubah Printer, menentukan Rentang Cetak dan Salinan, dan dengan tombol Ok dan Batal.

Namun persyaratan saya adalah membuat Ghostscript hanya mencetak diam-diam menekan dialog Print.

Apakah ada tombol perintah yang memungkinkan saya mencetak secara diam-diam? Jadi saya bisa mengeluarkan perintah dan printer mencetaknya.

Saya tidak bisa mendapatkan informasi apa pun dari Google atau berbagai dokumen Ghostscript lainnya yang tersedia di internet. Jadi saya ingin memeriksa dengan ramah dan berpengetahuan pengguna super masyarakat.

Diekstrak dari dokumentasi ghostscript , Pencetak MS Windows

Jika tidak ada nama printer Windows yang ditentukan di -sOutputFile, Ghostscript prompt untuk printer Windows menggunakan dialog Print Setup standar kotak. ...

Jika nama printer Windows ditentukan di -sOutputFile menggunakan format "% printer% printer_name", misalnya

gs ... -sOutputFile="%printer%Apple LaserWriter II NT"

kemudian Ghostscript mencoba membuka printer Windows tanpa prompt (kecuali, tentu saja, jika printer terhubung ke FILE :)

Bergantung pada bagaimana proses terakhir seharusnya bekerja, mungkin Anda juga memerlukan sakelar -dBATCH dan -dNOPAUSE

catatan : %printer% adalah literal ke Ghostscript tetapi sintaksnya akan bertabrakan dengan parser cmd yang melihat %printer% sebagai operasi baca variabel.

Di dalam file batch, tanda persen bisa di-escape dengan menggandakannya (%%printer%%) tetapi dalam mode baris perintah tidak ada cara untuk keluar dari tanda persen jika berada di dalam string yang dikutip (seperti yang ditunjukkan dalam dokumentasi, mereka diperlukan untuk menangani spasi dalam argumen).

Dalam mode baris perintah ada dua alternatif:

  • Pastikan variabel lingkungan printer tidak ada (jika tidak ada, maka operasi baca tidak dijalankan dan literal %printer% disimpan dalam perintah akhir yang dieksekusi).
  • Tentukan beberapa variabel lingkungan sehingga ketika operasi baca dijalankan, perintah yang dihasilkan akan benar.

Hanya beberapa contoh baris perintah:

rem ensure the variable does not exist
set "printer="
gs ... -sOutputFile="%printer%Apple LaserWriter II NT"

rem store the full literal
set p=%^printer%
gs ... -sOutputFile="%p%Apple LaserWriter II NT" 

rem store the percent sign
set "p=%"
gs ... -sOutputFile="%p%printer%p%Apple LaserWriter II NT"

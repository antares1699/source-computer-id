Jalankan browser dalam layar penuh tanpa elemen UI lainnya (seperti menerjemahkan, memulihkan sesi, dll.)

Saat ini saya mencoba menyiapkan Raspberry-Pi sebagai solusi pemantauan yang sangat ringan / sederhana. Yang harus dilakukan hanyalah membuka halaman web saat boot.

Membuka halaman web itu cukup mudah. Saya cukup memasukkan file desktop ke ~/.config/autostart dengan perintah seperti:

chromium-browser --kiosk http://192.168.1.23/monitor.html

Sejauh ini bagus. Namun ada dua masalah:

  1. Saya melihat bilah alat yang menyatakan "Halaman ini dalam bahasa Jerman. Apakah Anda ingin menerjemahkannya?
  2. Ketika saya mencabut R-Pi dan memasangnya kembali, saya mendapatkan toolbar kuning "sesi pemulihan".

Saya tidak tahu cara untuk memperbaikinya. Saya mencoba dengan --temp-session berharap "sesi pemulihan" akan hilang. Sayangnya tidak :(

Saya juga mencoba dengan --app yang tidak mengubah apa pun.

Tahu bagaimana cara memperbaikinya? Perhatikan bahwa saya tidak terpaku pada kromium. Ini kebetulan menjadi satu-satunya browser yang saya temukan yang membuatnya mudah diluncurkan dalam layar penuh dengan URL dalam sekali perintah sederhana.

Anda harus bisa menambahkan dua bendera --disable-translate and --incognito

chromium-browser --disable-translate --kiosk --incognito http://192.168.1.23/monitor.html

Mode penyamaran adalah cara yang lucu untuk membuat Chromium tidak mengkhawatirkan sesi terakhir saat pulih dari kerusakan.

Referensi: Nonaktifkan fungsi pemulihan sesi Google Chrome

Baris perintah Chrome beralih dari percakapan di Stack Overflow

Mulai chromium versi 60.0.3112.89, --disable-translate tidak berfungsi lagi.

Solusinya, jika Anda memiliki akses, adalah menghapus lang dari tag html atau menyetelnya ke "en".

MEMPERBARUI :

Pada versi 69, solusi sebelumnya tidak berfungsi lagi, solusinya adalah menambahkan tag ini ke kepala html: <meta name="google" content="notranslate">

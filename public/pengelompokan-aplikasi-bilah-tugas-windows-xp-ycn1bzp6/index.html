Pengelompokan aplikasi bilah tugas Windows XP

Di windows XP sp2 normal Anda dapat mengelompokkan program serupa di taskbar. Biasanya pengelompokan ini hanya terjadi jika jumlah apps> x.

Adakah yang tahu jika x bisa diubah? Saya mencoba mengaktifkan pengelompokan jika x> 1.

Pengelompokan terjadi saat taskbar menjadi terlalu ramai. Ini bisa jadi diubah dengan mengedit registri. Tipe regedit di kotak dialog Jalankan untuk membuka editor registri.

Buka HKEY_CURRENT_USER \ Software \ Microsoft \ Windows \ CurrentVersion \ Explorer \ Advanced.

Jika belum ada, buat file nilai DWORD baru dipanggil TaskbarGroupSize

Masukkan nilai 2 atau lebih besar. Nilai dari 2 akan mengelompokkan tombol yang mirip dengan segera setelah jendela kedua dari aplikasi dibuat. Nilai yang lebih tinggi akan memungkinkan banyak jendela aplikasi yang sama sebelum pengelompokan terjadi.

Oleh: http://thismatter.com/tutorials/software/windows-xp/managing-windows-with-taskbar.htm

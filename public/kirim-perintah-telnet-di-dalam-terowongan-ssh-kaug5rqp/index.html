Kirim perintah Telnet di dalam Terowongan SSH

Saya ingin menghubungkan komputer pribadi saya ke beberapa router melalui Telnet, untuk mengelolanya.

Sebenarnya, router saya ada di jaringan lain. Saya membuka konsol, membuat koneksi SSH ke Main Mahcine, dan kemudian membuat Telnet.

Saya ingin menutupi koneksi perantara itu dengan terowongan SSH, dan secara langsung membuat pengikatan telnet misalnya, port 2222 ke server jarak jauh: 22

Menemukan di Google, saya melihat cara mudah untuk membuat Terowongan SSH

ssh -f user@server -L 2222:server:22

-L Mengikat port lokal ke mesin jarak jauh: port -f memaksa untuk tetap berada di latar belakang

Tapi sekarang, saya tidak tahu bagaimana saya bisa mengirim telnet melalui terowongan SSH, bagaimana saya bisa mereferensikan sumbernya, menggunakan terowongan?

Terima kasih.

Karena ini port lokal, Anda dapat mereferensikannya menggunakan localhost.

Mari kita asumsikan bahwa mesin utama bernama main, dan router di belakang mesin ini disebut router. Urutan perintah berikut seharusnya berfungsi:

ssh -f user@main -L 2222:router:23 -N
telnet localhost 2222

Mengingat bahwa alamat IP main adalah 192.168.1.2, dan alamat IP router adalah 10.0.1.2, ini adalah urutan perintah yang akan digunakan:

ssh -f user@192.168.1.2 -L 2222:10.0.1.2:23 -N
telnet localhost 2222

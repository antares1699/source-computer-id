Saya memiliki ribuan gambar (90 GB) kebanyakan dokumen, dan hal-hal terkait karyawan seperti salinan paspor .. dll. Sebagian besar dokumen ini berukuran lebih besar dan dipindai dalam DPI tinggi (300 atau 200). Adakah software yang akan membantu saya dalam melakukan pekerjaan ini dengan cara yang lebih mudah karena hampir tidak mungkin untuk membukanya satu per satu dan mengubah ukurannya.

Imagemagick bisa ubah ukuran gambar .

convert input.jpg -resize 600x600 output.jpg

Perhatikan bahwa ini akan menyetel ukuran maksimum untuk setiap sisi, tetapi akan mempertahankan rasio aspek. Jadi gambar 1200x1000 yang diubah ukurannya dengan perintah di atas akan diskalakan menjadi 600x500 piksel, sedangkan gambar 1000x1200 akan diubah ukurannya menjadi 500x600.

Anda dapat, tentu saja, memasukkan ini ke dalam skrip. Di bash, untuk mengonversi setiap *.jpg dalam direktori menjadi 600x600 jpg:

for f in ./*.jpg; do convert "$f" -resize 600x600 "${f%.jpg}-resized.jpg"; done

Untuk melakukan hal yang sama secara rekursif:

find . -type f -name *.jpg -exec bash -c 'convert "$0" -resize 600x600 "${0%.jpg}-resized.jpg"' {} \;

Saya tidak tahu cara melakukannya di skrip baris perintah / batch Windows, tetapi saya tahu itu mungkin.

Jika Anda ingin menimpa file asli Anda, Anda dapat menggunakan ImageMagick's Mogrify perintah:

mogrify input.jpg -resize 600x600

Ini memiliki keuntungan ekstra karena dapat menggunakan perluasan shell, sehingga Anda dapat mengubah ukuran setiap * .jpg dalam direktori tanpa menggunakan for loop:

mogrify ./*.jpg -resize 600x600

Untuk menggunakannya secara rekursif, Anda harus memasukkannya ke dalam perintah find, seperti pada convert:

find . -type f -name *.jpg -exec mogrify '{}' -resize 600x600 \;

IrfanView akan melakukan pekerjaan itu untuk Anda. Ini memiliki alat "Konversi Batch / Ganti Nama" yang dapat Anda lakukan banyak hal, termasuk mengubah ukuran.

Dari tutorial ini tentang mengubah ukuran gambar dengan IrfanView:

  1. Luncurkan IrfanView dan tekan B kunci. Ini membawa Anda ke mode batch.

  2. Tambahkan semua file atau seluruh folder ke file input

  3. Ubah format keluaran dan ubah persentase kualitas

  4. gunakan opsi Lanjutan untuk mengubah ukuran semua file

Perhatikan bahwa IrfanView gratis untuk penggunaan non-komersial saja .

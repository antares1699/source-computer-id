Bagaimana saya bisa memasukkan angka nol dalam loop Batch FOR?

Saya menulis file batch untuk melakukan beberapa operasi pada komputer jarak jauh di gedung saya. Mereka semua diberi nama dengan awalan yang sama, tetapi nomor di akhir berubah (01–65). Saya mencoba mengikuti tutorial online untuk membuat loop file batch berdasarkan angka, dan menghasilkan ini:

FOR /l %%N in (1,1,65) do (
    set HOSTNAMETOUSE=prefix-%%N
    ECHO %HOSTNAMETOUSE%
)

Ini berfungsi dengan baik setelah mencapai 10, karena kemudian menjadi dua digit. Yang saya butuhkan adalah angka 1-9 muncul sebagai 01-09, karena itu cocok dengan skema penamaan kami. Pada dasarnya, saya membutuhkan satu digit dari nol di depan untuk 9 iterasi pertama. Bagaimana saya bisa melakukannya dengan Windows Batch Files?

Kode yang Anda posting tidak dapat berfungsi karena %HOSTNAMETOUSE% diperluas ketika pernyataan diurai, dan seluruh blok yang diberi tanda kurung diurai dalam satu lintasan, sebelum loop dijalankan. Jadi nilai yang diperluas konstan untuk semua iterasi.

Anda harus menggunakan ekspansi tertunda untuk mendapatkan nilai pada waktu eksekusi.

@echo off
setlocal enableDelayedExpansion
FOR /l %%N in (1,1,65) do (
    set HOSTNAMETOUSE=prefix-%%N
    ECHO !HOSTNAMETOUSE!
)

Sekarang awali dengan 0 sesuai kebutuhan.

Karena Anda tidak pernah membutuhkan lebih dari satu 0, Anda dapat menggunakan pernyataan IF.

@echo off
setlocal enableDelayedExpansion
FOR /l %%N in (1,1,65) do (
    if %%N lss 10 (set HOSTNAMETOUSE=prefix-0%%N) else set HOSTNAMETOUSE=prefix-%%N
    ECHO !HOSTNAMETOUSE!
)

Tetapi itu tidak nyaman jika lebar nomor Anda lebih dari 2 digit. Solusi yang lebih umum adalah menambahkan string n nol dengan panjang yang Anda inginkan, lalu gunakan operasi substring untuk mempertahankan n digit paling kanan.

@echo off
setlocal enableDelayedExpansion
FOR /l %%N in (1,1,65) do (
    set "n=00%%N"
    set "HOSTNAMETOUSE=!n:~-2!
    ECHO !HOSTNAMETOUSE!
)

Anda dapat dengan mudah memperluas solusi ini ke hampir semua angka nol di depan.

foo-2-bar.jpg -> foo-02-bar.jpg
...

Dalam contoh di atas, ???? mewakili 4 karakter pertama (foo-), <0> menyisipkan nol di posisi ke-5, dan *.* mewakili sisa nama file (-bar.jpg) yang tidak akan berubah.

set first=%1
set last=%2
set len=%3
setlocal EnableDelayedExpansion
FOR /l %%N in (%first%, 1, %last%) do (
    set "n=0000000000000%%N"
    set "padded=!n:~-%len%!
    echo !padded!
)

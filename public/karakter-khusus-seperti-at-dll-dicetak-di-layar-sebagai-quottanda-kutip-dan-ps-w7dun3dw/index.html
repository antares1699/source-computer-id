Karakter khusus seperti @, # dll dicetak di layar sebagai "(tanda kutip) dan £

Setelah menginstal versi percobaan Windows 8, beberapa kunci laptop saya tidak berfungsi seperti yang diharapkan. Masalahnya seperti ketika saya mencoba mengetik @ dengan menekan Shift + 2 itu menunjukkan " di layar & sebaliknya.

Demikian pula -

shift + 3      types     £    instead of     #
shift + `      types     ¬    instead of     ~
shift + '      types     @    instead of     "
shift + 2      types     "    instead of     @

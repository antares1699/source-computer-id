Bagaimana cara menampilkan ikon kunci khusus di Terminal.app?

Saat mengirim permintaan POST ke server yang membutuhkan otentikasi pengguna dan kata sandi, Terminal.app menampilkan ikon kunci berkedip yang saya percaya bukan Unicode.

Saat meminta pengguna untuk memasukkan informasi yang masuk akal dalam skrip saya sendiri, saya dapat memeriksa apakah TERM_PROGRAM adalah Apple_Terminal, tetapi saya tidak tahu cara menampilkan ikon berkedip.

Gambar "kunci" bukanlah mesin terbang karakter, ini adalah bentuk kursor kustom yang digambar Terminal.

Terminal menampilkan bentuk kursor khusus ini setiap kali pengguna terlihat seperti diminta untuk memasukkan kata sandi atau data sensitif serupa. Selain menampilkan kursor ini, ini membatasi keyboard ke metode masukan Romawi untuk menghindari metode masukan secara tidak sengaja menampilkan informasi sensitif sebagai teks sebaris atau di jendela lain. Ini berperilaku seperti bidang teks kata sandi OS X standar.

Terminal memasuki mode ini ketika perangkat TTY dikonfigurasikan untuk mode kanonik (alias "dimasak") dan gema karakter tidak aktif. Dalam konfigurasi ini, input pengguna tidak di-echo dan hanya buffer input perangkat TTY yang akan melihat karakter sampai Anda mengetik Return untuk mengirimkannya ke program aplikasi. Ini memastikan bahwa baik perangkat TTY maupun program aplikasi tidak dapat menggemakan masukan pengguna saat Anda memasukkan teks, jadi konfigurasi ini biasanya hanya digunakan untuk membaca informasi sensitif.

Anda dapat melihatnya beraksi di Bash, dengan menjalankan

stty -echo && echo Tell me your secrets: && cat; stty echo

stty -echo mematikan gema karakter TTY, dan saat Bash menjalankan perintah, Bash mengaktifkan mode kanonik, sehingga perintah cat berikut akan dijalankan dalam mode “sandi”. cat akan menggemakan setiap baris masukan sampai Anda mengetik Kontrol - D . stty echo menyalakannya kembali setelah cat keluar.

Perhatikan bahwa skrip shell dapat dengan mudah menggunakan read -s untuk membaca dengan TTY dalam mode ini, mis.,

while read -sp 'Tell me your secrets: '; do echo; echo $REPLY | vis; done

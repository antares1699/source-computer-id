Menyetel opsi tampilan folder tunggal vs. menyetel opsi tampilan default untuk semua folder di Mac OS X 10.6

Tujuan

  • Atur opsi tampilan folder default dalam format daftar, dan kemudian atur beberapa folder khusus ke default ke tampilan ikon.

Proses (menurut saya)

  • Buka Finder dan konfigurasikan opsi tampilan sesuai keinginan untuk semua folder — tampilan daftar dengan font 10pt
  • Klik "Gunakan sebagai Default"
  • Buka salah satu folder yang ingin saya lihat secara default berbeda dari folder lain (mis., Aplikasi)
  • Setel opsi tampilan untuk folder itu saja — tampilan ikon, ukuran ikon 40 x 40, font 10pt
  • Pilih "Selalu buka dalam tampilan ikon"
  • Haruskah saya mengklik "Gunakan sebagai Default"?

Pertanyaan

  • Apa dampak "Gunakan sebagai Default"?
    1. Setelan default untuk gaya tampilan tertentu — ikon, daftar, kolom, dan aliran sampul?
    2. Pengaturan default untuk folder tertentu itu?
    3. Pengaturan default untuk semua folder?

Sistem konfigurasi

  • OS: Mac OS X 10.6.2

Penelitian Pendahuluan / Latar Belakang

  • Saya melihat Pertanyaan StackOverflow Setel kebijakan default di Finder "Opsi Tampilan" , tapi saya masih tidak yakin tentang cakupan dampak saat menggunakan "Gunakan sebagai Default".
  • Di bawah Mac OS X 10.4 Tiger, palet View Options memungkinkan Anda untuk memilih salah satu, tetapi tidak keduanya, Jendela ini saja atau Semua jendela . Sekarang di OS X 10.5 Leopard dan 10.6 Snow Leopard Anda dapat memilih centang Selalu buka dalam tampilan xxx dan Anda dapat mendorong Gunakan sebagai default tombol (secara terpisah atau keduanya). Metode lama tampaknya lebih masuk akal bagi saya, tetapi saya rasa saya hanya perlu menemukan dokumentasi yang menyatakan cakupan dari dua opsi tampilan baru.

Menurut Diskusi Bantuan Sederhana Ringkasan: Penemu OS X 10.5 (Leopard) , cakupan dari dua perintah opsi tampilan adalah sebagai berikut:

  • Selalu Buka di Tampilan xxx : Berlaku hanya untuk folder tertentu itu.
  • Ditetapkan sebagai default : Berlaku untuk semua folder yang dibuka dalam tampilan tertentu itu, artinya ini menetapkan default untuk tampilan itu sebagai lawan dari default untuk folder itu. (Dalam pertanyaan awal saya, jawabannya adalah # 1.)

Berbekal pengetahuan ini, masalah saya harus berasal dari file .DS_Store lama di beberapa folder mengapa di folder tertentu ukuran teks menyimpang dari default 10 poin default saya.

Menghapus .DS_Store Secara Rekursif di Folder

Rasa frustrasi dan kebingungan saya karena melihat folder terbuka tidak menggunakan default saya tampaknya berasal dari file .DS_Store lama dari folder yang telah saya salin dari mesin yang lebih lama. Untuk mengatasinya, saya secara rekursif menghapus folder sebagai berikut menggunakan Terminal:

cd ~/Folder with bad views
sudo find . -name ".DS_Store" -depth -exec rm {} \;

Kudos to the Adobe TechNote Menghapus .DS_Store file di Macintosh OS X? untuk memberikan perintah ini untuk menghapus secara rekursif.

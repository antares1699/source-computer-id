Tentukan jenis pantomim saat mengunduh file

Saya ingin melihat jenis pantomim saat mengunduh file. Bagaimana saya bisa melakukan ini? (Saya menggunakan Firefox saat ini, tetapi saya dapat mengunduh browser lain jika perlu)

Jenis pantomim dikomunikasikan dalam Header HTTP "Jenis Konten" yang dikirim sebagai tanggapan dari server web. Ada banyak cara untuk mengambil HTTP Header (dan semua bagian dari proses permintaan / tanggapan).

Addon favorit saya dan yang sangat populer untuk Firefox adalah Pembakar . Lihat di panel "Net" Firebug saat memuat halaman. Anda akan melihat entri untuk setiap aset yang dimuat Firefox (halaman web itu sendiri, gambar, file CSS, dll). Klik entri tersebut untuk meluaskannya dan lihat di bawah "Header Respons".

Addon lainnya adalah LiveHTTPHeaders . Di mana Firebug mungkin melewatkan unduhan file yang sebenarnya, yang ini harus menangkapnya karena itu membuang semua header ke jendela log. Mengingat berapa banyak aset yang dapat dimuat untuk satu halaman, jendela log ini bisa membuat Anda kewalahan.

HttpFox memperbaiki ini dengan tampilan jendela log yang mencantumkan setiap permintaan aset dalam satu baris; klik garis untuk detail lengkapnya. Jendela masih terisi dengan cepat tetapi ada tiga kolom yang sangat membantu - jenis, ukuran, url. Penyortiran kolom akan menjadi fitur yang bagus; tanpa itu sedikit berburu-n-mematuk untuk menemukan baris yang sesuai dengan unduhan sebenarnya tetapi masih tidak terlalu buruk. Meskipun Firebug luar biasa untuk banyak hal, coba HttpFox terlebih dahulu.

Untuk menjadi teknis dan rewel, 'mengunduh' adalah cara khusus yang digunakan browser web untuk menangani respons server. Firefox sedang 'mengunduh' HTML dari sebuah halaman web, sepertinya tidak seperti itu karena Firefox membuat dan menampilkannya untuk Anda. Saat Anda mengeklik kanan "Simpan Tautan Sebagai", Anda secara khusus memberi tahu browser untuk mengunduh apa pun yang ada di sana. Header Jenis Konten tertentu biasanya akan memicu browser ke mode 'unduh' (yaitu memberi Anda kotak dialog Simpan). Berikut ini dua yang umum: application/download dan application/octet-stream.

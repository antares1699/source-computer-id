Cari teks dalam daftar file

Saya mencoba menjalankan pencarian ganda di dalam file dan mengembalikan nama file.

saya menggunakan

find ./ -iname '*txt' | xargs grep "searchtext" -sl

untuk menemukan nama file dengan 'searchtext' di dalamnya.

Perintah mengembalikan daftar file.

Bagaimana saya bisa menemukan "teks pencarian orang lain" dalam file yang sudah ditemukan itu dan menampilkannya dengan cara yang sama?

Berikan hasil ke grep lainnya dengan menggunakan backticks atau setara $ ().

Jika Anda ingin mempertahankan daftar perantara, gunakan tee

grep -l oranges $(find . -name "*txt" | xargs grep -l apples | tee apples.txt)

Anda dapat merantai xargs grep …:

find . -iname '*txt' | \
    xargs grep -l "searchtext" | \
    xargs grep -l "othersearchtext" | \
    xargs grep -l "yetanothersearchtext"

Bash ini akan mencari beberapa string di folder tertentu secara rekursif dan mengembalikan nama file

Pemakaian:

findfiles /var/www/ php searchText1 searchText2 searchText3 searchText4

bash: findfiles

#!/bin/sh

DIR=$1
EXT=$2

CMD="find $DIR -iname '*.$EXT' | xargs grep -l '$3' | xargs grep -l '$4' | xargs grep -l '$5' | xargs grep -l '$6' | xargs grep -l '$7' | xargs grep -l '$8' | xargs grep -l '$9'"

eval $CMD

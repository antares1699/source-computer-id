Cara batch kompres gambar dengan Imagemagick atau perintah atau program serupa di Linux

Saya baru-baru ini belajar tentang perintah convert dari imagemagick yang telah saya gunakan untuk mengompres banyak gambar yang saya miliki untuk penggunaan pribadi, dan untuk blog yang saya miliki. Menurut pengalaman saya, Imagemagick adalah program kompresor gambar terbaik, dan ini memberikan kontrol paling besar atas cara mengompres gambar. Saya memiliki lebih banyak gambar yang ingin saya kompres, tetapi tidak ingin menggunakan perintah convert berkali-kali, satu per satu untuk setiap file. Masalah utamanya adalah sangat memakan waktu bagi saya untuk melakukannya. Saya ingin dapat melakukan satu cara besar untuk mengompres gambar saya. Saya baik-baik saja dengan melakukan ini melalui baris perintah, tetapi GUI mungkin membuat ini sedikit lebih intuitif.

Berikut ini contoh perintah yang saya gunakan untuk membuat gambar asli 20% dari ukuran aslinya. mengonversi -ukuran 20% 20140322_102113.jpg 20140322_102113opt.jpg

Jika saya mengatakan memiliki 100 gambar, dan semuanya ada dalam folder yang sama, saya ingin dapat melakukan sesuatu seperti berikut

Untuk semua gambar ubah -ukuran 20% imagename.jpg imagename_optimized.jpg

Saya tidak tahu apakah ada perintah yang sudah bisa melakukan ini, tetapi jika tidak, saya berpikir untuk membuat perintah bash, tetapi saya tidak begitu paham dengan bash. Membantu membuat skrip bash sederhana ini, atau saran tentang cara mengatasi dilema saya sangat kami hargai. Saya menggunakan Linux, dan hanya menginginkan solusi khusus untuk Linux. Terima kasih

Menggunakan loop 'untuk' pasti akan berhasil - dan merupakan teknik umum yang baik - tetapi Anda hampir pasti memiliki lebih dari 1 prosesor pada mesin Anda, jadi mengapa hanya melakukan satu konversi pada satu waktu?

Anda bisa membuat segala sesuatunya bergerak lebih cepat jika Anda melakukan:

find *.jpg | xargs -n1 -P8 -I{} convert -resize 20% "{}" "opt-{}"

Argumen untuk xargs adalah:

n1  - Only give 'convert' one file at a time
P8  - Use 8 processes
I{} - Replace {} with the filename given by 'find'

Dan kemudian perintah convert diberikan setelahnya.

Saya pikir skrip bash sederhana dapat membantu Anda.

for i in `ls /path/to/img/dir`; do convert -resize 20% "/path/to/img/dir/${i:0:-4}" "/path/to/new/img/dir/${i:0:-4}_optimized.jpg"; done

Catatan : ${i:0:-4} adalah menghapus ekstensi .jpg. Jika file Anda memiliki panjang ekstensi yang berbeda, Anda mungkin ingin mengubahnya -4 untuk sesuatu yang lain :)

Pembaruan Windows 10 macet pada instalasi terpisah

Saya baru-baru ini memutakhirkan komputer saya ke Windows 10 dan kemarin saya mendapat pemberitahuan yang mengatakan bahwa beberapa pembaruan memerlukan instalasi terpisah untuk diselesaikan.

Namun, ketika saya mengklik tombol Install Now tidak terjadi apa-apa, lingkaran progrees terus berputar dan jendela terkunci.

Daftar pembaruan mencakup item berikut ini:

  • Microsoft - Kamera - Microsoft LifeCam HD-3000
  • Microsoft - Gambar Penunjuk - Mouse USB Perangkat Keras Microsoft
  • Microsoft - Keyboard - Keyboard USB Perangkat Keras Microsoft.

Ada ide untuk mengatasi ini?

Hal ini disebabkan oleh kesalahan antara Kontrol Akses Pengguna (UAC) dan Pembaruan Windows (WU), di mana WU mencoba memuat dialog autentikasi UAC di dalam jendela WU, bahkan jendela mengharuskan dialog UAC dibuka dalam mode terlindungi (di mana sisa desktop Anda menjadi abu-abu sementara UAC menunggu tanggapan). Meskipun Anda dapat menonaktifkan UAC untuk melewatinya, ini membuka PC Anda terhadap risiko, meskipun hanya dinonaktifkan cukup lama untuk menginstal pembaruan. (Bagaimana jika Anda memiliki beberapa malware yang menunggu kesempatan untuk menyerang !?)

Ini adalah cara teraman dan termudah untuk mengatasi masalah tersebut: Setelah mengklik tombol Instal Sekarang, yang mengunci jendela WU dan secara opsional memainkan bunyi lonceng, cukup tekan Alt + Y pada keyboard Anda. Ini akan memilih "Ya" dari jendela UAC yang ada, tetapi tidak terlihat (karena tidak memuat dalam mode terlindungi). Jendela WU Anda sekarang akan terbuka dan mulai menginstal pembaruan.

Jika Anda menekan Alt + Y dan tidak ada yang terjadi, klik sekali di jendela WU yang terkunci untuk memastikannya aktif, dan tekan Alt + Y lagi.

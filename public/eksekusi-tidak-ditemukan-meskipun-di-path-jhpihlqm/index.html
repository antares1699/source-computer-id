Eksekusi tidak ditemukan meskipun di PATH

$ PATH saya diatur ke banyak direktori termasuk ~/bin dan ~/target/bin

Saya ingin mengeksekusi script.sh yang ada di ~/target/bin tetapi saya mendapatkan kesalahan bash: ~/bin/script.sh: No such file or directory

Ini membingungkan saya karena which menemukan direktori yang benar ~ / target / bin dan ~ / target / bin / script.sh dijalankan dengan baik.

Apakah ada pengaturan khusus dalam bash yang dapat menyebabkan penanganan jalur yang salah ini?

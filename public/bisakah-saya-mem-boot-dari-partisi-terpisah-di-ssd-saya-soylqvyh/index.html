Bisakah saya mem-boot dari partisi terpisah di SSD saya?

Saya memiliki drive 3,5 "yang gagal, yang ingin saya coba selidiki dengan SpinRite. SpinRite ingin diinstal pada media yang dapat di-boot, tetapi saya tidak memiliki memory stick. Saya perhatikan melalui Windows Computer Management, bagaimanapun, itu SSD saya memiliki cadangan 450MB yang tidak terisi. Saya jelas dapat membuat parisi baru di sini, dan meletakkan SpinRite di dalamnya .... tetapi apakah mungkin memberi tahu PC saya untuk boot dari partisi itu?

Saya tahu di BIOS ada opsi untuk memilih drive mana yang akan di-boot, tetapi saya merasa belum pernah melihat opsi untuk membedakannya partisi untuk boot dari?

Beberapa UEFI BIOS modern mungkin memiliki opsi untuk boot dari partisi tertentu selain dari drive, cari di bagian "boot", biasanya ada bagian yang disebut "boot override" yang memungkinkan Anda untuk memilih dari mana Anda akan boot. Periksa apakah ada lebih dari satu partisi per drive yang ditampilkan (terlepas dari dua entri, satu untuk UEFI dan satu lagi untuk gaya MBR), Jika demikian, Anda kemungkinan besar dapat mempartisi ruang yang tidak diketahui dan memaksa PC untuk boot dari partisi itu menggunakan BIOS Tidak bisa.

Secara umum, Anda tidak bisa melakukan itu. Setelah Anda menggunakan drive untuk boot, komputer Anda mulai pada MBR disk itu, dan mem-boot partisi apa pun yang dipilih. Untuk dapat memilih dari beberapa partisi, program yang harus dilakukan harus diinstal di bagian drive tersebut. Program itu disebut bootloader. GRUB mungkin yang paling umum (dan kemungkinan besar yang digunakan BIOS jika dapat melakukan boot dari partisi beton), dan digunakan secara luas untuk sistem boot ganda. NAMUN, meskipun Anda dapat melakukannya secara teknis (partisi ruang yang tersedia, instal grub pada MBR, sehingga Anda dapat memilih antara windows dan SpinRite) Saya akan mengatakan itu tidak disarankan, Ini akan memakan waktu, berisiko untuk data di Anda SSD dalam kasus yang mungkin terjadi, detail kecil salah, dan pasti berlebihan. Beli saja drive USB 8GB termurah yang dapat Anda temukan di toko lokal, Anda akan menghemat banyak sakit kepala.

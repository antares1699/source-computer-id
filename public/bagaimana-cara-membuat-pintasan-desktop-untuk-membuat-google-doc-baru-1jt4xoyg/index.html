Bagaimana cara membuat pintasan desktop untuk membuat Google Doc baru?

Melelahkan untuk membuka Google Drive di browser dan dari sana mengklik "Dokumen baru ...". Jadi bagaimana cara menemukan dan menyimpan URL untuk membuat dokumen baru sehingga saya dapat meletakkan pintasan di desktop saya?

Ini adalah URL (pada saat penulisan): https://docs.google.com/document/u/0/create?usp=docs_home

Anda dapat menemukannya dengan memantau lalu lintas jaringan di docs.google.com saat mengklik tombol "Baru".

Sunting: Memiliki URL ini Anda jelas dapat membuat pintasan desktop atau (seperti yang dinyatakan responden lain), dengan asumsi Anda menjalankan windows, file bat kustom memulai browser dengan URL sebagai argumen.

Anda dapat membuat file batch (dengan asumsi Anda memiliki windows).

  1. Buat dokumen teks baru
  2. Ubah nama semuanya menjadi "googleDocs.bat" - Anda dapat mengganti googleDocs dengan milik Anda sendiri
  3. Edit file batch dengan klik kanan dan 'edit'
  4. Salin dan tempel ini:

mulai "chrome.exe" " https://docs.google.com/document/u/0/create?usp=docs_home "

  1. Simpan dan biola!

Google sekarang telah menambahkan pintasan ringkas yang indah untuk membuat dokumen baru. Jadikan pintasan Anda mengarah ke:

  • https://doc.new

(Saking pendeknya, Anda mungkin bahkan tidak memerlukan pintasan lagi. Cukup ketik "doc.new" di bilah url Anda.)

Ini juga berfungsi untuk spreadsheet, dek slide, formulir, dan situs:

  • https://sheet.new
  • https://slide.new
  • https://form.new
  • https://site.new

Ubah PDF berwarna menjadi putih / hitam

Di Debian Sid, saya memiliki PDF dengan latar belakang biru dan font kuning. Saya sudah banyak mencari di Super User tetapi saya belum menemukan sesuatu yang berguna untuk saya.

Saya telah mencoba mengonversi PDF menjadi skala abu-abu dengan:

gs -o grayscale.pdf -sDEVICE=pdfwrite -sColorConversionStrategy=Gray -sProcessColorModel=DeviceGray -dCompatibilityLevel=1.4 colored.pdf

Masalahnya adalah saya mendapatkan PDF dengan font putih dan latar belakang abu-abu tua sehingga saya tidak dapat mencetaknya.

Setelah itu saya coba:

convert -density 96x96 gs2.pdf -density 96x96 -negate -compress zip inv.pdf

Saya mendapat PDF dengan font hitam (dan ini tidak masalah) dan latar belakang abu-abu (dan ini tidak oke).

Apa yang dapat saya lakukan untuk mendapatkan PDF dengan latar belakang putih dan font hitam?

PERINGATAN UMUM !!! bekerja pada SALINAN FILE Anda !!!

(sehingga Anda dapat memiliki kesempatan kedua jika Anda melakukan kesalahan)

vektor background pdf (artinya tidak gambar raster ) dalam file pdf bisa mudah diubah dalam beberapa langkah (lihat juga jawaban stackoverflow saya itu sekarang saya akan memperluas dan meningkatkan

  • https://stackoverflow.com/questions/10856521/change-background-color-of-pdf10869208#10869208 )
  • PERIKSA AWAL :

buka file pdf Anda dengan editor yang dapat menampilkan struktur pdf internal, Suka

notepad ++

  • http://notepad-plus-plus.org/download/v6.1.8.html (versi portabel dalam file zip bekerja dengan baik dengan wine di Linux atau Anda dapat menggunakan vim demikian juga)

dan verifikasi apakah Anda dapat melihat cuplikan kode seperti

1.000 1.000 0.000 rg (itu berarti kuning )

0.000 0.000 1.000 rg (itu berarti biru jika Anda biru adalah biru murni memiliki RGB triplet 0, 0, 255, jika tidak baca sisa jawaban untuk mengidentifikasi triplet yang tepat ke dalam kode pdf )

dan seterusnya...

(potongan kode dapat berubah, misalnya, dalam pdf yang dihasilkan oleh openoffice fitur ekspor pdf internal, potongan kode yang sama ada dalam bentuk ini:

0 0 0 rg (it means *black*)
1 1 1 rg (it means *white*)

dan seterusnya...

jika Anda dapat melihat potongan kode ini, maka Anda dapat mulai mengubahnya nilai, jika tidak, Anda perlu mendekompresi aliran teks

Anda dapat melakukan tugas ini dengan

pdftk

http://www.pdflabs.com/docs/install-pdftk/

pdftk file.pdf output uncompressed.pdf uncompress

dan kompres ulang setelah perubahan selesai

pdftk uncompressed.pdf output recompressed.pdf compress

sekarang, jika Anda melihat cuplikan kode ini, Anda dapat mengubah nilai

LANGKAH 1 (untuk mengedit pdf) -

hal pertama yang Anda butuhkan adalah menemukan kesetaraan yang tepat antara RGB nilai warna teks dan latar belakang dan representasi pdf internal dengan warna yang sama

Anda dapat menggunakan alat pilih warna gratis seperti ini

untuk mengidentifikasi nilai rgb dari teks dan warna latar belakang

  • http://freecode.com/projects/grabc
  • http://gcolor2.sourceforge.net/

setelah Anda memiliki nilai-nilai ini, Anda perlu mengubahnya menjadi pdf internal khusus perwakilan

untuk melakukan ini, perhatikan proporsi ini:

1: 255 = x: warna yang Anda pilih

misalnya: katakanlah Anda memiliki triplet RGB ini untuk latar belakang: 30.144.255

untuk mengetahui nilai koresponden dalam pdf untuk dimasukkan dalam potongan kode ke ubah warna latar belakang pdf, Anda lakukan: (Anda dapat menggunakan http://www.wolframalpha.com/ untuk menghitung dengan presisi)

1:255=x:30 = 30/255 = 0.117 (approximated to first three decimals)

1:255=x:144 = 144/255 = 0.564 (approximated to first three decimals)

1:255=x:255 = 255/255 = 1

jadi, seluruh triplet dalam pdf, sesuai dengan RGB 30.144.255 , akan:

0.117 0.564 1.000

# LANGKAH 2 (untuk pengeditan pdf)

kami mencari 0.117 0.564 1.000 dalam file pdf dengan notepad ++ ( membungkus dan cocokkan satu kata saja perlu diperiksa) dan kami menemukan internal representasi pdf dari latar belakang dan kita dapat mengubah dari biru menjadi, katakanlah, putih

1.000 1.000 1.000

atau

1 1 1

tetapi, karena Anda menulis tentang latar belakang biru , lebih tepatnya, saya membuat contoh pdf dengan latar belakang biru ( biru murni 0,0,255 RGB - jika Anda biru memiliki nada yang berbeda memodifikasi sesuai kebutuhan tip saya) dan teks kuning

  • http://ge.tt/3mFIRfP

karena kita tahu bahwa 0,000 0,000 1.000 rg artinya biru , kami mencari ini dan kami dapat mengubah dari 0.000 0.000 1.000 rg, menjadi 1.000 1.000 1.000 rg (putih) TAPI ...

pada saat yang sama, Anda juga mengubah teks dari kuning menjadi hitam

mencari 1.000 1.000 0.000 (teks kuning) dan ubah menjadi hitam 0,000 0,000 0,000

dan kami sekarang memiliki pdf vektor dengan hitam teks dan putih Latar Belakang

tolong, ingatlah untuk

    • kompres lagi pdf ini Anda dapat diubah jika Anda tidak terkompresi dengan pdftk
    • perbaikan
pdftk file.pdf output fixed.pdf

ada cara lain, mulai dari postscript, untuk melakukan tugas yang sama, tetapi karena Anda sudah memiliki ekstensi File pdf , mengonversi ke postscript untuk melakukan tugas serupa akan menjadi langkah yang tidak berguna

tolong beri umpan balik, dan jangan ragu untuk bertanya lebih lanjut

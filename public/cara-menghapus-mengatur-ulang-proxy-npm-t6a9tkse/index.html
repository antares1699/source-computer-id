Cara menghapus / mengatur ulang proxy npm

Saya baru mengenal nodejs dan npm, terminal saya menunjukkan satu kesalahan ketika saya mencoba menginstal apa pun dengan npm, ini tangkapan layarnya

Saya ingat saya bermain dengan konfigurasi proxy seperti ini:

npm config set proxy http://localhost:8080/
npm config set https-proxy http://localhost:8080/
npm config set strict-ssl false

dan itu untuk memecahkan masalah lain dengan npm. ini kesalahan sebelumnya

tolong bantu.. Terima kasih sebelumnya.

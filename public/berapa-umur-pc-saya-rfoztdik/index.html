Berapa umur PC saya?

Adakah cara mudah untuk mengetahui berapa umur mesin Windows XP?

Anda dapat mengetahui kapan Windows XP pertama kali diinstal, tetapi ini tidak akan berguna bagi Anda jika komputer Anda diinstal dari gambar di pabrik seperti Dell atau HP karena gambar tersebut dibuat jauh sebelum komputer Anda.

Jika Anda menginstal Windows sendiri: Mulai> Jalankan> Ketik "cmd" dan tekan enter. Kemudian ketik "systeminfo" dan tekan enter. Sekelompok data akan berkedip di layar. Gulir ke atas untuk menemukan "Tanggal Pemasangan Asli".

Sebagai alternatif, sebagian besar pabrikan mencantumkan nomor seri di komputer. Misalnya, Dell memasang Tag Servis di setiap PC. Anda dapat memasukkan Tag Servis tersebut di situs web produsen untuk mendapatkan informasi terperinci, termasuk konfigurasi perangkat keras asli dan tanggal pembelian.

Lihat casing mesin, hard disk, PSU, dan berbagai komponen lain untuk stiker pabrikan. Beberapa di antaranya mungkin memiliki tanggal yang akan menjadi indikator yang cukup baik untuk usia mesin.

Juga, saat boot, Anda mungkin melihat tanggal hak cipta untuk BIOS. Kecuali BIOS telah di-flash, ini juga dapat memberikan beberapa indikasi.

  1. Periksa tanggal pembuatan file data Anda (bukan file program karena dapat diatur oleh penginstal). Kekurangannya, Anda dapat memulihkan file dari mesin sebelumnya menggunakan perangkat lunak yang menyimpan stempel tanggal / waktu.

  2. Buka casing dan cari stiker pada komponen yang mencantumkan tanggal pembuatannya. Tanggal terakhir adalah tanggal paling awal mesin Anda dapat dibuat. Kekurangannya, pembangun mungkin telah menggunakan komponen yang sangat tua atau bekas.

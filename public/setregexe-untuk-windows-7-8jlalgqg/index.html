setreg.exe untuk Windows 7

Saya ingin menggunakan setreg.exe ( http://msdn.microsoft.com/en-us/library/aa387700(VS.85).aspx ) untuk menonaktifkan pemeriksaan pencabutan sertifikat. Namun, ini didasarkan pada versi .NET yang lebih lama daripada yang saya miliki. Microsoft mengatakan itu dikirimkan dengan .net SDK yang lebih lama, dan ketika saya mendownloadnya, coba instal, dikatakan sesuatu seperti 'tidak dapat menginstal SDK tanpa .net 1.1'.

Dalam artikel yang ditautkan, dikatakan untuk versi yang lebih baru saya harus menggunakan SignTool, tetapi SignTool tampaknya tidak memiliki fungsi yang saya butuhkan.

Apakah mungkin untuk -

a) temukan alat yang dapat menjalankan fungsi yang saya perlukan yang tidak memerlukan saya untuk menginstal .NET yang lebih lama

b) dapatkan setreg.exe tanpa mengunduh SDK

atau apakah saya perlu menginstal .NET yang lebih lama di komputer saya? Dan jika yang terakhir adalah kasusnya, apa yang harus saya lakukan untuk menginstal .NET yang lebih lama? Apakah itu akan menimpa .NET saya saat ini?

Apa pun alasannya, mungkin ada peringatan dengan aplikasi .NET 1.1 karena sebelum kerangka kerja .NET 2 didistribusikan secara luas.

Kerangka .NET 1.1 dapat diunduh sini dan tidak akan menimpa framework .NET lainnya yang mungkin telah Anda instal.

Jika Anda mengunduh '.NET Framework SDK Version 1.1', Anda dapat mengekstrak isinya. Di dalam Anda akan menemukan netfxsd1.cab. Di dalamnya Anda akan menemukan 'FL_setreg_exe _____ X86.3643236F_FC70_11D3_A536_0090278A1BB8' yang merupakan setreg.exe yang Anda cari. (Cukup ganti nama file. Ini persis sama dengan versi yang disertakan dengan W2K.)

'setup.exe' adalah file SFX .CAB.

Jadi jika Anda memiliki metode untuk membuka file .CAB (sekali lagi ganti nama dari .exe menjadi .exe.cab) cukup mudah untuk membukanya atau Anda bisa menggunakan alat, uniextract.

http://www.legroom.net/software/uniextract

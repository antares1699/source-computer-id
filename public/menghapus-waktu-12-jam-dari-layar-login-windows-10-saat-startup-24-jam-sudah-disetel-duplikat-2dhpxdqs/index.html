Menghapus waktu 12 jam dari layar login Windows 10 saat startup (24 jam sudah disetel) [duplikat]

Jadi masalah saya agak kecil, dan masalah utama telah diatasi sini , tapi saat pertama kali menyalakan komputer, format 12 jam selalu ditampilkan. Akun yang saya masuki adalah akun admin, dan setelah saya masuk (karena pengaturan saya), layar kunci menunjukkan 24 jam. Tapi sebelum saya melakukannya, selalu menunjukkan 12h.

Ini bukan masalah besar, saya hanya merasa menjengkelkan karena menurut saya sistem 12h itu menjengkelkan untuk memulai jika saya tidak membaca jam analog. Saya kira ada beberapa nilai "default" yang mengikutinya (mungkin yang di Amerika umumnya diasumsikan oleh Windows secara default) - adakah cara untuk mengubahnya?

Pengaturan internasionalisasi sistem ".DEFAULT" dapat ditemukan di kunci registri berikut:

HKEY_USERS\.DEFAULT\Control Panel\International

Perbarui nilai berikut agar sesuai dengan pengaturan pengguna Anda:

  • sTimeFormat (kemungkinan: H: mm: ss)
  • sShortTime (kemungkinan: H: mm)

Catatan: Pengaturan pengguna Anda dapat ditemukan di:

HKEY_CURRENT_USER\Control Panel\International

Metode PowerShell:

New-PSDrive -Name HKU -PSProvider Registry -Root HKEY_USERS | Out-Null 
$internationalPaths = @("HKU:\.DEFAULT\Control Panel\International","HKCU:\Control Panel\International")
$hourFormat = "h"
IF($TimeFormat -eq '24h')
{
    $hourFormat = "H"
}       
FOREACH ($path in $internationalPaths)
{
    IF((Get-ItemProperty $path).'sTimeFormat')
    {
        #Windows 10 default time format h:mm:ss tt
        Set-ItemProperty -Path $path -Name "sTimeFormat" -Value "$hourFormat`:mm:ss tt"
    }
    IF((Get-ItemProperty $path).'sShortTime')
    {
        #Windows 10 default time format h:mm tt
        Set-ItemProperty -Path $path -Name "sShortTime" -Value "$hourFormat`:mm tt"
    }
}

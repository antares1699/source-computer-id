Bagaimana cara mengubah warna font MS-DOS prompt?

MS-DOS sudah sangat tua, dan menjadi kakek dari komputer sangat sulit untuk mendapatkan dukungan untuk itu.

Saya sudah mencoba saran ini dari situs web:
http://www.easydos.com/menucolor.html
https://support.microsoft.com/en-us/kb/95099
http://www.computerhope.com/color.htm

Saya telah mengedit: Config.sys (Sekarang tertulis 'Menucolor = 2,0') C: \ Windows \ color.txt (Sekarang tertulis 'hijau')

Masih tidak bekerja.
Adakah yang punya ide tentang bagaimana melakukan ini?

Anda dapat menggunakan debug untuk menulis file COM pendek. Ketika Anda menjalankan debug Anda akan mendapatkan satu prompt tanda hubung, masukkan yang berikut ini (termasuk baris kosong) dan Anda akan mendapatkan file color.com di direktori kerja Anda saat ini:

a 100
mov ah, 06
xor al, al
xor cx, cx
mov dx, 184f
mov bh, 07
int 10
mov ah, 4c
xor al, al
int 21

rcx
e
ncolor.com
w
q

Ia memanggil int 10 ah=06 untuk membersihkan layar, mengatur kursor di bagian bawah dan mengisi dengan atribut di bh. Gigitan tinggi adalah latar belakang, rendah latar depan, warna:

  • 0 = hitam
  • 1 = biru
  • 2 = hijau
  • 3 = cyan
  • 4 = merah
  • 5 = ungu
  • 6 = kuning
  • 7 = putih
  • 8 = hitam muda
  • 9 = biru muda
  • A = hijau muda
  • B = sian muda
  • C = merah muda
  • D = ungu muda
  • E = kuning muda
  • F = putih muda

(Jadi bit 4 adalah intensitas tinggi.) Untuk mendapatkan latar belakang merah dengan latar depan hitam, ubah 07 menjadi 40.

Mengurai argumen baris perintah untuk mengatur warna bukanlah hal yang sepele, jadi cukup gunakan kode favorit Anda. Atau lakukan ini di debug untuk memperbaruinya:

debug color.com
e 10a
40
w
q

Atribut disimpan di lokasi 010A, memperbaruinya secara langsung lebih mudah daripada mengetik semuanya lagi.

Jika Anda memuat ANSI.SYS lalu tentukan prompt DOS (misalnya):

c:> prompt $ e [1; 33m $ p $ g

Anda dapat mengubah warna prompt dan prompt itu sendiri.

Mengubah prompt perintah cmd.exe.

PROMPT [text]

  text    Specifies a new command prompt.

Prompt can be made up of normal characters and the following special codes:

  $A   & (Ampersand)
  $B   | (pipe)
  $C   ( (Left parenthesis)
  $D   Current date
  $E   Escape code (ASCII code 27)
  $F   ) (Right parenthesis)
  $G   > (greater-than sign)
  $H   Backspace (erases previous character)
  $L   < (less-than sign)
  $N   Current drive
  $P   Current drive and path
  $Q   = (equal sign)
  $S     (space)
  $T   Current time
  $V   Windows version number
  $_   Carriage return and linefeed
  $$   $ (dollar sign)

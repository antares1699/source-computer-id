Harap dicatat, ini tampaknya hanya berfungsi di Notepad ++ 6.0 (atau lebih tinggi).
Versi 6.0 disertakan dukungan untuk PCRE , yang diperlukan agar ini berfungsi.

Cari \z (Itu spasi, diikuti dengan garis miring terbalik, diikuti oleh huruf z). Itu harus dilakukan :)

\z menandai akhir file .

Jika Anda ingin mencari spasi di akhir file, gunakan \s+\z. \s artinya karakter spasi apa pun :

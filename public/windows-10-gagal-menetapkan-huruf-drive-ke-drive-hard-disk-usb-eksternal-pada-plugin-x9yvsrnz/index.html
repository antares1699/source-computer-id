Windows 10 gagal menetapkan huruf drive ke drive hard disk USB eksternal pada plugin

Saya memiliki drive 500 GB SATA 2,5 inci yang diformat dengan benar (Master Boot Record, FAT32) di dalam penutup hard drive eksternal, dihubungkan dengan kabel USB.

Drive ini secara otomatis terpasang dengan sempurna ke laptop Ubuntu saya, tetapi ketika dicolokkan ke laptop Windows 10 saya, itu hanya mencatat keberadaan perangkat USB yang dapat "dilepas dengan aman", itu tidak menetapkan huruf drive.

Ini sangat merepotkan, seperti yang bisa Anda bayangkan.

Bagaimana cara mengetahui jika ini mungkin, dan jika ya, bagaimana cara memperbaikinya?

Kedengarannya seperti bendera volume disetel. Coba prosedur ini agar Windows menetapkan surat secara otomatis.
1. Buka prompt CMD dengan hak Administrator .
2. Ketik DiskPart dan tekan Enter.
3. Jalankan perintah berikut satu per satu. Tidak peka huruf besar / kecil.

List Disk

Identifikasi drive eksternal Anda dari daftar yang diberikan secara manual. Asumsikan itu disk # (angka), dan jalankan

Select Disk #
Select Partition 1
Attribute Volume Clear NoDefaultDriveLetter

Selesai. Sekarang di setiap sistem Windows itu harus ditetapkan dengan surat secara otomatis dan muncul di Explorer.

Masalah ini membuat saya gila selama setengah tahun. Saya mencoba semuanya termasuk jawaban di atas, yang mendekati tetapi tidak menyelesaikannya. Pada akhirnya ternyata volume itu ditandai tersembunyi. Beginilah cara saya menyelesaikan masalah:

  1. Klik menu start atau tekan tombol windows
  2. Ketik diskpart dan tekan tombol enter
  3. Klik ya di dialog yang muncul
  4. Ketik list volume ke dalam jendela diskpart dan tekan tombol enter
  5. Identifikasi volume yang menyebabkan masalah dan catat nomor volume

  6. Ketik berikut ini ke dalam jendela diskpart (ganti X dengan nomor volume dari langkah sebelumnya):

    select volume X
attributes volume

  7. Jika atribut tersembunyi adalah yes maka ini adalah masalah Anda. Cukup ketik:

    attributes volume clear hidden

  8. Kemudian tekan tombol enter dan windows secara otomatis akan menetapkan huruf drive.

  9. Jika tidak, mungkin automount dinonaktifkan atau macet, Anda dapat mencoba memperbaikinya dengan:

    automount disable
automount enable

di mana variabel% username% didefinisikan?

Saya baru saja menginstal Windows XP di virtualbox saya pada host windows 7. Saya tidak dapat mengubah variabel lingkungan nama pengguna:

USERNAME=Owner

Saya berhasil mengubah variabel USERDOMAIN karena ditentukan di properti sistem di panel kontrol. Tapi saya tidak tahu di mana USERNAME didefinisikan. Saya ingin tahu apakah memodifikasinya akan berdampak pada variabel:

USERPROFILE=C:\Documents and Settings\Owner

Saya telah mengubah nama pengguna saya di akun pengguna saya tetapi tidak berpengaruh pada variabel sistem USERNAME.

Variabel lingkungan USERNAME ditentukan dalam kunci registri HKEY_CURRENT_USER\Volatile Environment.

Namun perlu dicatat bahwa seperti yang tersirat pada keyname, variabel dalam kunci ini adalah lincah , artinya meskipun pengguna dapat mengubahnya, mereka tidak akan mempertahankan nilai barunya dan akan ditimpa oleh sistem dengan nilai turunan (semacam RAM registri). Variabel lingkungan username adalah salah satu dari sedikit variabel yang tidak dapat disetel, atau bahkan jika ada, mereka tidak akan "melekat", seperti bagaimana menyetel variabel dalam sesi konsol hilang ketika Anda membuka yang baru.

Itu harus memperbarui variabel dengan nama pengguna baru. Sudahkah Anda mencoba keluar dan masuk kembali atau me-reboot sistem?


Di mana / bagaimana Anda mengubah nama pengguna, file Akun Pengguna Applet Panel Kontrol? Itu Akun Pengguna applet hanya mengubah nama pengguna , bukan yang sebenarnya nama pengguna . Gunakan lusrmgr.msc untuk mengubah nama pengguna dengan mengkliknya, menekan F2 dan mengubahnya (lalu log out / in atau reboot).

Jika tidak berhasil, cari nama pengguna lama / asli di registri dan ubah di sana. Periksa kunci berikut:

HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList

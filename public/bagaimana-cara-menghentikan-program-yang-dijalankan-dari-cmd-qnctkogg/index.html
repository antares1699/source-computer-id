Bagaimana cara menghentikan program yang dijalankan dari CMD?

Saya mencoba masuk ke pemrograman C, tetapi semakin besar proyek, semakin sering saya ingin mengganggu program yang saya jalankan dari CMD. Biasanya ketika menjalankan misalnya Python dari CMD, saya dapat menginterupsi skrip menggunakan CTRL + C. Namun ini tidak berhasil. Saya telah melihat orang mengatakan Anda dapat menggunakan CTRL + BREAK / PAUSE, tetapi keyboard saya tidak memiliki tombol tersebut.

Apakah ada cara untuk mengganggu program C yang saya jalankan dari CMD? Mungkin menggunakan keybind khusus (untuk memberi tahu CMD bahwa CTRL + C sama dengan CTRL + BREAK)?

Kamu bisa gunakan AutoHotkey untuk membuat Ctrl + C Kirim Ctrl + Istirahat sebagai gantinya - atau mengirim keduanya - hanya di Command Prompt.

Berikut adalah skrip AutoHotkey untuk keduanya:

  • Ctrl + C hanya mengirim Ctrl + Istirahat :

    #IfWinActive, Command Prompt
^c::^CtrlBreak

  • Ctrl + C mengirimkan keduanya Ctrl + Istirahat dan Ctrl + C :

    #IfWinActive, Command Prompt
~^c::^CtrlBreak

Tilde (~) membuat kunci juga mempertahankan fungsi aslinya. Untuk menggunakan, cukup instal program, masukkan skrip yang Anda inginkan ke dalam file dan jalankan.

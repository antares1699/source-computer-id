Bagaimana cara menonaktifkan pesan "Perangkat keras baru ditemukan" di Windows?

Di netbook saya, saya memiliki perangkat Bluetooth yang tidak diinstal - tidak ada driver yang diinstal.

Setiap kali saya memulai Windows, sebuah pesan muncul

Perangkat keras baru ditemukan, apakah Anda ingin menginstal driver?

Bisakah pesan ini dinonaktifkan?

EDIT: Saya tidak menggunakannya dan saya tidak ingin memilikinya! (alasan keamanan)

Coba gunakan Layanan Pembaruan Windows untuk menemukan driver untuk perangkat Anda yang tidak dikenal.

Jika masih berlanjut, Anda dapat pergi ke manajer perangkat Anda (Tombol Mulai / klik kanan pada Komputer / Properti / Manajer Perangkat), temukan perangkat yang tidak memiliki driver yang dikenal, klik kanan padanya dan pilih opsi untuk menonaktifkannya.

Berikut adalah cara "cepat & kotor" untuk menghilangkan "Found New Hardware Wizard" jika muncul di setiap boot ulang. Saya tersandung ke dalam solusi ini, meskipun ketika saya mencari sedikit bantuan, saya tidak pernah melihatnya.

[Sekarang tentu saja ini mengasumsikan tidak ada solusi yang lebih tepat, seperti benar-benar menginstal driver perangkat keras, mengubah perangkat lunak atau pengaturan BIOS, dll. Tetapi jika Anda menghapus instalasi sesuatu dan meninggalkan sisa, memutuskan untuk berhenti untuk selamanya menginstal mid-stream, hal-hal seperti itu, maka ini adalah cara non-teknis yang mudah untuk menyelesaikannya.]

Lain kali Wizard muncul, lakukan gerakan SEBAGAI JIKA Anda akan menginstal driver.

Di layar pertama, pilih "Tidak, tidak kali ini" (karena jangan periksa drivernya dengan Microsoft).

Di layar kedua, pilih "Instal dari lokasi tertentu (karena jangan mulai mencari driver di komputer).

Pada layar ketiga, DE-pilih "Cari media yang dapat dilepas" dan tepat di bawahnya, tentukan folder di komputer yang KOSONG. Ini akan menyebabkan penginstalan segera GAGAL.

Layar keempat / kegagalan - berjudul "Tidak Dapat Menginstal Perangkat Lunak Ini" - adalah hasilnya. Di bagian bawah layar itu, centang kotak yang bertuliskan "Jangan minta saya lagi untuk menginstal perangkat lunak ini." Klik tombol Selesai.

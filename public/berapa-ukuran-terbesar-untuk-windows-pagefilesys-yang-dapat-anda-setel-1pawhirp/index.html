Batas pada Windows 7 adalah 16 Terabyte.

Pelajari Praktik Terbaik untuk Mengoptimalkan Konfigurasi Memori Virtual - Microsoft TechNet

Di PC dengan prosesor yang mendukung Ekstensi Alamat Fisik (PAE) —yaitu, pada PC mana pun yang mampu menjalankan Windows 7 — ukuran maksimum file halaman adalah 16 TB.

Ketika mencoba untuk mengatur jumlah yang lebih besar, Windows menampilkan kesalahan:

Properti sistem

Masukkan ukuran file halaman maksimum yang lebih besar dari atau sama dengan file ukuran file halaman awal, dan kurang dari 16777216 MB.

Catatan: Batas ini juga merupakan ukuran file maksimal untuk file di Windows 7 NTFS (Sumber: NTFS - Wikipedia ). Ukuran file maksimal untuk NTFS pada Windows 8 dan Windows 10 lebih besar, tetapi tidak jelas apakah file halaman yang lebih besar diperbolehkan.

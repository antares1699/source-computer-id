Bagaimana cara menambahkan beberapa alamat IPv6 untuk satu host di / etc / hosts?

Saya ingin sebuah nama harus diselesaikan dengan beberapa alamat IPv6. Saya tidak memiliki akses ke server DNS dan saya tidak ingin menjalankan setan untuk tujuan itu.

Saya ingin menambahkan entri pada /etc/hosts dan harus mengembalikan dua alamat IPv6 untuk satu nama.

Nah, jika Anda ingin melakukannya, lakukan saja :-)

Jika Anda menambahkan ini ke / etc / hosts:

2001:db8::1 testing
2001:db8::2 testing
192.168.0.1 testing

maka nama host akan memiliki tiga alamat:

$ telnet testing
Trying 2001:db8::1...
Trying 2001:db8::2...
Trying 192.168.0.1...
telnet: Unable to connect to remote host: No route to host

Dalam hal ini alamat tidak ada sehingga koneksi gagal, tetapi Anda dapat melihat bahwa itu berfungsi.

Ketika saya membutuhkan ini saya telah menemukan HTTrack menjadi efektif, mudah digunakan, dan opsi yang cukup lengkap.

HTTrack adalah utilitas browser offline gratis (GPL, gratis / gratis) dan mudah digunakan.

Ini memungkinkan Anda untuk mengunduh situs World Wide Web dari Internet ke direktori lokal, membangun secara rekursif semua direktori, mendapatkan HTML, gambar, dan file lain dari server ke komputer Anda. HTTrack mengatur struktur tautan relatif situs asli. Cukup buka halaman situs web "cermin" di browser Anda, dan Anda dapat menjelajahi situs dari link ke link, seolah-olah Anda sedang melihatnya online. HTTrack juga dapat memperbarui situs cermin yang ada, dan melanjutkan unduhan yang terputus. HTTrack dapat dikonfigurasi sepenuhnya, dan memiliki sistem bantuan terintegrasi.

WinHTTrack adalah rilis HTTrack Windows 2000 / XP / Vista / Seven, dan rilis WebHTTrack Linux / Unix / BSD.

-m --mirror Turn on options suitable for mirroring.
            This option turns on recursion and time-stamping,
            sets infinite recursion depth and keeps FTP directory listings.
            It is currently equivalent to -r -N -l inf --no-remove-listing.

Bagaimana cara mendapatkan lokal sistem default di Windows 7?

Saya ingin mendapatkan lokal sistem default di Windows 7, saya menemukan bahwa layar login menggunakan ini. Jadi saya ingin memulai satu aplikasi selama startup sistem, sebelum layar login muncul. Sehingga hasil yang saya dapatkan adalah user independent. Tolong beri tahu saya jika ada petunjuk ...

Anda tidak perlu menjalankan ini dari layar login, contoh:

systeminfo /FO LIST | find "System Locale"

Alternatif Pengembang dapat ditemukan di sini: MSDN - GetSystemDefaultLocaleName

Hasil ini akan menjadi pengguna independen!

Jangan gunakan layanan hanya untuk mendapatkan nilai itu, layanan dimaksudkan sebagai tugas yang berjalan lama.

Jangan gunakan penyedia kredensial baik, mereka digunakan sebagai metode otentikasi.

Perbedaannya mungkin kecil dalam contoh cd yang Anda berikan, tetapi berhati-hatilah ketika Anda mv file.

Misalnya jika Anda ingin memindahkan beberapa file ke direktori lain, dan Anda mengetik

for i in *.mp3 ; do mv $i dest ;  done

Lupakan garis miring, lalu setiap file yang dipindahkan akan menimpa file bernama dest (yang pertama sebenarnya membuat file bernama dest jika belum ada). Hasilnya adalah semua file kecuali yang terakhir ditimpa dengan file terakhir, dan sekarang disebut dest.

Untuk pindah ke direktori, Anda harus menentukan /

for i in *.mp3 ; do mv $i dest/ ;  done

Salah satu efek samping dari garis miring adalah jika direktori tidak ada, Anda akan mendapatkan pesan kesalahan.

cd abc/xyz

Benar, karena "abc / xyz" mengacu pada ' entitas "xyz" di direktori "abc" '. Saat Anda mengetik

cd abc/xyz/

itu mengacu pada ' Entitas "" di direktori "xyz" di direktori "abc" '. Entitas "" (string kosong) secara otomatis diterjemahkan ke ".", Yang merupakan direktori "saat ini", yang dalam hal ini adalah direktori "xyz". Jadi semuanya bekerja untuk hal yang sama.

(Beberapa orang merasa bahwa direktori " harus "memiliki garis miring ditambahkan. Mereka salah.)

Mereka identik. String diperlakukan oleh operator nama file sebagai "jalur" yang merupakan abstraksi, dan setiap jalur dikaitkan dengan nama dasar dan jalur lain, nama dir. Dua jalur identik jika masing-masing adalah string yang sama. Sebagai contoh Anda:

cas var$ dirname abc/xyz; basename abc/xyz
abc
xyz
cas var$ dirname abc/xyz/; basename abc/xyz/
abc
xyz

Perhatikan bahwa, sementara beberapa program memperlakukan string secara identik, termasuk semua utilitas UNIX biasa, jika mereka berada di jalur yang sama, ada program lain yang peduli tentang garis miring tertinggal. Yang terpenting dari ini adalah rsync.

Nota bene Dari spesifikasi POSIX, Resolusi Nama Jalur :

Nama jalur yang berisi setidaknya satu karakter non-garis miring dan yang diakhiri dengan satu atau lebih garis miring harus diselesaikan seolah-olah satu karakter titik ('.') Ditambahkan ke nama jalur.

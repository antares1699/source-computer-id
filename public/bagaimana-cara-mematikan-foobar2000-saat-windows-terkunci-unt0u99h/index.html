Anda bisa menggunakan Mute On Lock:

Bisukan Saat Terkunci adalah utilitas penduduk kecil yang berada di baki sistem dan memantau sesi Anda: saat Anda mengunci Windows Anda workstation, ini mematikan suara speaker Anda. Dengan bantuan ini program Anda dapat dengan cepat menonaktifkan atau mengaktifkan suara sistem dengan mengklik dua kali ikon baki atau menggunakan hot opsional kunci.

Ini gratis dan dijalankan dari Windows XP ke Windows 7 (termasuk edisi 64-bit).

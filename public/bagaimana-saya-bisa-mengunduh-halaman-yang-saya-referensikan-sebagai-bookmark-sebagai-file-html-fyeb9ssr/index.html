Ekspor daftar bookmark Anda sebagai HTML, lalu gunakan wget untuk mendownload setiap halaman yang ditautkan, dan semua gambar serta skrip yang diperlukan untuk menampilkan halaman tersebut:

wget --recursive --level 1 --page-requisites --convert-links --force-html --input-file bookmarks_7_7_14.html

--level 1 membatasi pengunduhan rekursif untuk mengikuti satu tautan, dari halaman bookmark Anda ke situs web yang di-bookmark.

--page-requisites artinya mengunduh file yang diperlukan untuk menampilkan setiap halaman (gambar, gaya, javascript, dll).

--convert-links akan mengubah tautan ke tautan relatif yang berfungsi secara lokal jika file diunduh (gambar dll) dan akan mengubah tautan menjadi tautan absolut (tautan ke halaman lain) jika tidak.

--force-html akan memberi tahu wget untuk memperlakukan file input sebagai halaman html daripada daftar URL.

Jika Anda melakukan ini berulang kali, beberapa perawatan lebih lanjut perlu dilakukan untuk menghindari penimpaan unduhan lama dengan unduhan baru.

CATATAN: Ini hanya akan mengunduh halaman yang telah Anda tandai, bukan halaman lain di situs yang sama. Solusi yang lebih kompleks akan diperlukan jika Anda ingin mencerminkan beberapa atau semua situs target, dan lebih kompleks lagi jika Anda ingin pilih-pilih tentang mana yang dicerminkan seberapa banyak.

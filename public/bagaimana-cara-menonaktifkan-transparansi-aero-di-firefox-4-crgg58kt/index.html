Bagaimana cara menonaktifkan transparansi Aero di Firefox 4?

Saya perhatikan bahwa antarmuka transparan di Firefox 4 untuk Windows 7 benar-benar membatasi kinerja. Setelah mencari metode untuk menonaktifkan Aero untuk Firefox di Google, saya tidak mendapatkan hasil. Sepertinya tidak ada metode langsung (seperti Boolean sederhana untuk mematikannya di about:config) metode untuk menghilangkan efek Aero.

Anda dapat menonaktifkan Aero saat menjalankan FireFox dengan menyetel opsi kompatibilitas Windows dan menandai "Nonaktifkan Komposisi Desktop". Tapi ini akan mengubah seluruh sistem Aero saat Anda menjalankan FF.

Ada orang lain yang menemukan masalah ini juga, dan tampaknya terkait dengan akselerasi perangkat keras baru di FF4, seperti yang disaksikan di forum mozillazine.org .

Menguji saran dari sini :

Untuk menguji apakah masalah yang Anda lihat disebabkan oleh akselerasi perangkat keras, Anda dapat menonaktifkannya. Ada dua cara untuk melakukannya:

• Jalankan Firefox dalam mode aman. (Semua akselerasi perangkat keras selalu dinonaktifkan dalam mode aman.)

• Hapus centang "Gunakan akselerasi perangkat keras bila tersedia" di bagian Lanjutan dari dialog Preferensi / Opsi.

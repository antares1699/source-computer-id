Temukan jendela di ruang kerja saat ini dan terapkan beberapa tindakan padanya

Saya pikir ini seharusnya sangat mudah bagi seseorang yang pandai Haskell tetapi saya tidak bisa mengetahuinya karena saya buruk dalam hal itu. Pada dasarnya saya ingin mencari melalui daftar jendela di ruang kerja saat ini, menemukan satu yang cocok dengan beberapa kondisi dan menerapkan beberapa tindakan padanya; jika tidak ditemukan maka buat jendela ini. Saya memiliki kode kerangka seperti ini

import qualified XMonad.StackSet as W
import XMonad.Actions.WindowGo

findWindow condition actionIfFound actionIfNotFound = 
    ifWindow (condition <&&> member W.current) windowFound actionIfNotFound
        where windowFound = ask >>= w -> do actionIfFound w

Posting lama, tetapi untuk siapa pun yang masih menginginkan jawaban:

Pada dasarnya, dengan XMonad.StackSet.index, Anda bisa mendapatkan daftar jendela di ruang kerja saat ini seperti ini:

do
    windows <- gets (W.index . windowset)
    -- something...

Jadi findWindow dapat diimplementasikan seperti ini:

import qualified XMonad.StackSet as W
import Data.List (find)

findWindow :: (Window -> Bool) -> (Window -> X()) -> X() -> X()
findWindow condition actionIfFound actionIfNotFound = do
    windows <- gets (W.index . windowset)
    let found = find condition windows  -- found has type Maybe Window
    case found of
        Nothing -> actionIfNotFound
        Just w -> actionIfFound w

PS: Tiga baris terakhir bisa disingkat menjadi maybe actionIfNotFound actionIfFound found

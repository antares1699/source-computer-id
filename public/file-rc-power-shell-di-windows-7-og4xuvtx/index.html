File rc Power Shell di Windows 7

Sebagai contoh di Linux, ada file .bashrc atau .cshrc yang bersumber di bash atau shell untuk mengkonfigurasinya saat startup. Apa file analog untuk Power Shell di Windows 7. Dan juga apa perintah analog dari sumber di Power Shell.

P.S. Saya ingin memberikan konfigurasi ke power shell dengan file itu dan mengambilnya!

Profil disimpan dalam variabel otomatis $PROFILE. Ada profil untuk setiap host (konsol dan Lingkungan Skrip Terintegrasi) untuk pengguna saat ini atau semua pengguna (empat di semua). Variabel otomatis $profile menunjuk ke profil host pengguna saat ini.

Anda dapat melihat jalur ke setiap profil dengan menjalankan:

$profile | Select-Object *user* | Format-List

Saya lebih suka profil $profile.CurrentUserAllHosts, dengan begitu saya mendapatkan semua fungsi pembantu saya di konsol dan ISE.

ise $profile.CurrentUserAllHosts

Anda dapat memuat ulang profil Anda dengan dot-sourcing file:

. $profile

Semoga berhasil!

Ubah kata sandi image disk terenkripsi di OS X dengan Terminal

Saya ingin program yang akan ada di flash drive saya yang, ketika dijalankan, akan melakukannya

  • buka gambar disk terenkripsi dengan kata sandi yang disimpan
  • kemudian ubah kata sandi menjadi nomor acak
  • simpan untuk digunakan lain kali

Bisakah ini dilakukan?

7
7

Jika Anda mengetahui skrip shell, Anda dapat menulis program sederhana yang dapat melakukannya. Gunakan hdiutil untuk mengubah kata sandi disk terenkripsi:

hdiutil chpass [image_name]

Ini akan secara interaktif meminta Anda untuk kata sandi sebelumnya dan kata sandi baru sehingga Anda harus menggunakan beberapa pipa atau mungkin mengharapkan . Untuk menghasilkan nomor acak, echo $RANDOM harus dilakukan. Untuk menyimpan kata sandi, Anda dapat menggunakan file teks sederhana, jika Anda tidak keberatan menyimpannya tanpa perlindungan.

file host dan subdomain yang tidak diblokir

Saya memodifikasi file host untuk memblokir google.com dan subdomainnya tetapi hanya memblokir nama domain utama. Itu tidak memblokir subdomain. Windows 7 Starter

127.0.0.1 google.com

Itu memblokir google.com tetapi tidak memblokir www.google.com

Lalu saya mencoba yang ini: 127.0.0.1 * google.com Itu tidak membantu.

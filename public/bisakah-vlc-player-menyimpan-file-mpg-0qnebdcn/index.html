Bisakah VLC Player menyimpan file `.mpg`?

Saya memiliki video keluarga pendek di DVD. Saya telah berhasil menggunakan pemutar VLC untuk menyimpannya sebagai file, tetapi memiliki ekstensi file .ps.

Saya ingin memasukkannya ke dalam album Picasa, tetapi Picasa hanya mengenali video dengan ekstensi file berikut: .avi, .mpg, .asf, .wmv.

Inilah yang telah saya lakukan sejauh ini:

  • Buka videonya
  • Pilih Media> Konversi / Simpan
  • Tambahkan file yang sama dan klik tombol Convert / Save
  • Klik ikon kunci inggris untuk opsi lainnya
  • Pilih enkapsulasi MPEG-TS, codec video MPEG-4, dan hapus centang audio di bawah bagian codec itu (tidak ada di video ini).
  • Klik "Jelajahi" dan berikan lokasi dan nama file (tanpa ekstensi)
  • Klik "Mulai"

Terlepas dari opsi pengkodean, ekstensi file masih .ps.

Dalam konteks video, ekstensi file .ps biasanya mewakili Aliran Program MPEG , yang merupakan jenis file yang sama yang diwakili oleh ekstensi .mpg atau .mpeg. Anda harus dapat dengan mudah mengganti nama file.

Karena Anda telah memilih MPEG TS (Transport Stream, yang benar-benar lebih berguna untuk penyiaran daripada duduk di file di komputer Anda) sebagai wadah, VLC mungkin melakukan sesuatu yang konyol ... tetapi jika itu dulu aliran transportasi, saya pikir itu akan menggunakan ekstensi file .ts. Jadi kemungkinan besar, Anda cukup mengganti nama file yang sudah Anda miliki.

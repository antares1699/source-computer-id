Dapatkan jumlah tiff dalam multi-tiff dengan baris perintah ImageMagick?

Apakah ada cara untuk mendapatkan jumlah tiff di multi-tiff dengan utilitas baris perintah di ImageMagick?

Yang ingin saya lakukan adalah mengekstrak (membagi) multi-tiff menjadi satu file. Namun jika tiff hanya berisi satu tiff (bukan multi-tiff), saya tidak ingin melakukan apa pun ... Juga jika saya menggunakan opsi "% d" untuk memberi nama file keluaran, saya mendapatkan "% d "di nama file jika hanya satu.

... atau apakah ada cara lain untuk menggunakan ImageMagick untuk menghindari nama file yang aneh saat mengonversi?

Anda dapat mengembalikan jumlah halaman dalam tiff menggunakan identify -format "%p" multipage.tif.

Anda dapat menggunakan yang berikut ini untuk menghitung jumlah halaman dalam file TIFF:

tiffinfo output_file_name.tif | grep "Page Number" | grep -c "P"

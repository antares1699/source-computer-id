Komputer

Saya memiliki dua komputer dengan 2 bahasa keyboard yang berbeda. Windows 10 terus mencampurkannya. Bagaimana cara mengubah ini?

Beberapa masalah keyboard pada 2 PC dengan windows 10 dan akun pengguna yang sama

Pada dasarnya, saya memiliki dua PC yang berbeda (satu desktop dan satu laptop) dengan windows 10 diatur pada masing-masing dan saya menggunakan akun pengguna yang SAMA di kedua PC yang ditautkan ke akun Microsoft saya.

Laptop memiliki keyboard gaya Inggris sedangkan desktop memiliki keyboard Kroasia (saya membelinya saat saya diposting di sana dan sangat jarang menggunakan desktop sehingga saya tidak pernah membeli keyboard Inggris / internasional).

Bagaimanapun, ketika saya mengatur desktop untuk menggunakan keyboard Kroasia dan kemudian masuk ke laptop saya memiliki ikon pemberitahuan bilah bahasa (ikon perubahan input keyboard) muncul di bilah tugas, dan keyboard secara otomatis diatur ke Kroasia. Sebaliknya, ketika saya mengubah keyboard laptop ke bahasa Inggris dan mematikan, PC desktop tampaknya disetel ke bahasa Inggris saat saya mem-boot-nya.

Pertama kali ini terjadi sangat sulit untuk diurutkan karena kata sandi windows saya pada saat itu berisi y (atau z, tidak ingat yang mana) dan tombol-tombol ini dibalik pada keyboard Kroasia, jadi saya tidak bisa masuk ke sistem.

Adakah yang tahu cara mengatur satu keyboard di desktop dan satu di laptop tanpa harus menautkan melalui akun pengguna ini?

p.s. Saya TIDAK PERNAH menyiapkan keyboard Kroasia di laptop karena saya tidak tahu bahasanya dengan cukup baik sehingga dapat berguna dan ketika saya melepas keyboard dari salah satu PC, keyboard itu akan dihapus dari keduanya.

Pengaturan Anda disinkronkan ke semua perangkat Anda karena Anda masuk ke perangkat tersebut dengan akun Microsoft yang sama. Untuk mengubah pengaturan khusus ini, buka aplikasi Pengaturan dan arahkan ke Akun → Sinkronkan pengaturan Anda (di kiri). Anda kemudian dapat menonaktifkan Preferensi bahasa slider. Itu akan menghentikan sinkronisasi tata letak keyboard, bahasa tampilan, dan format tanggal / waktu.

Anda dapat mengatur tata letak keyboard dengan kebijakan grup. Silakan lihat ini Artikel TechNet . Ini harus mengesampingkan pengaturan khusus pengguna.

Buat file .adm dengan konten berikut:

CLASS USER
  CATEGORY "Keyboard Layout"
     POLICY "Keyboard layout"
     KEYNAME "Keyboard Layout\Preload"
         PART "Enable English" CHECKBOX
           DEFCHECKED
          VALUENAME "1"
           VALUEON "00000409"
           VALUEOFF DELETE
         END PART
     END POLICY

  END CATEGORY

Berbagi folder dengan VirtualBox, Windows 7 Host dan Ubuntu 9.10 Guest

Saya memiliki pengaturan pengembangan dari tutorial ini: Bangun Server Pengembang Anda Sendiri dengan VirtualBox .

Tapi yang tidak saya ketahui adalah bagaimana cara berbagi folder di mesin virtual Ubuntu saya dengan host Windows 7. Saya ingin menggunakan editor teks Windows untuk mengedit kode yang ada di server Ubuntu saya.

Saya sudah mencoba menggunakan pengaturan Folder Bersama, menambahkan "/ var / www" tetapi dikatakan bahwa jalurnya tidak mutlak. Ketika saya mengklik "lainnya", ini hanya memungkinkan saya untuk menelusuri folder di host Windows 7 saya.

Baik tuan rumah dan tamu adalah OS 64-bit.

Saya mengalami banyak masalah dengan ini, tetapi menemukan solusi yang cukup mudah. Ini melibatkan pengaturan adaptor jaringan kedua untuk os tamu. Memiliki 2 adapter pada sistem: 1> NAT dan 2> Host-Only Adapter terhubung ke VirtualBox Host hanya adapter jaringan (ini dipilih dari daftar dropdown). Hal ini memungkinkan dua hal ... 1 memelihara koneksi ke internet untuk pembaruan, dll., Dan 2 memungkinkan tamu untuk terlihat di workgroup di OS host win7. Sekarang Anda dapat berbagi folder dari OS tamu ke windows => karena saya tidak bisa membuatnya bekerja sebaliknya. Ini berguna bagi saya karena saya jarang mengangkut barang dari vista ke ubuntu. Dengan mindterm diinstal pada Windows dan opensh-server diinstal pada OS tamu, saya kemudian dapat mengaktifkan transfer file dua arah antara win7 dan ubuntu, yang sangat ideal, menggunakan plugin SFTP.

Inilah solusi sempurna Anda hanya dengan "NAT".

Dikutip kata demi kata dari http://blogs.oracle.com/tao/entry/virtual_box_shared_folder_between :

  1. Tambahkan folder ke menu folder bersama.

    Penting untuk menuliskan nama folder pada langkah ini.

  2. Boot sistem tamu, buka terminal dan ketik perintah berikut

    sudo mkdir /mnt/tao_xp
sudo mount.vboxsf TAO /mnt/tao_xp

Folder bersama akan dapat diakses sekarang: baca & tulis!

Apa yang berhasil bagi saya adalah mengalihkan "Network Adapter" saya di VirtualBox dari "NAT" ke "Bridged". Ini dapat dilakukan di menu VirtualBox, "Perangkat" -> "Adaptor Jaringan". Semua ini perlu dilakukan setelah menginstal "Edisi Tamu". Untuk beberapa alasan, penginstalan saya default ke "NAT", yang tidak pernah bisa saya lihat di komputer jaringan saya.

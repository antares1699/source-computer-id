Bagaimana cara mengetahui jumlah baris dokumen dalam folder menggunakan skrip shell?

Saya memiliki folder yang penuh dengan kode saya, skrip shell apa yang dapat saya gunakan untuk mengetahui jumlah total baris kode?

4

7

Solusi yang sangat sederhana adalah wc -l ./*, yang menghitung jumlah baris di semua file, serta setiap file satu per satu.

Jika Anda menginginkan sesuatu yang lebih canggih, gunakan sloccount . Anda dapat menginstalnya mis. menggunakan Homebrew :

sudo brew install sloccount

Jalankan di folder yang berisi kode, seperti ini:

sloccount .

Ini akan memberi Anda baris berdasarkan bahasa pemrograman dan direktori.

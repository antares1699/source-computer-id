Dapatkan daftar subtitle menggunakan mediainfo di Windows

Saya menggunakan mediainfo untuk melihat informasi media tentang file mkv.

Saya melihat bahwa itu memberi saya detail tentang subtitle seperti:

Text #1
ID                                       : 3
Format                                   : PGS
Muxing mode                              : zlib
Codec ID                                 : S_HDMV/PGS
Codec ID/Info                            : The same subtitle format used on BDs/HD-DVDs
Language                                 : English
Default                                  : Yes
Forced                                   : No

dan seterusnya...

Dimungkinkan, untuk template khusus memiliki seperti:

SUBTiTLE: .............. English
SUBTiTLE: .............. French

dan seterusnya ?

Jika ya, bagaimana caranya?

Saya mencoba menempatkan hanya untuk Text:

$if(%Language%,Language : %Language%)

tapi sepertinya itu tidak berhasil

4

MediaInfo.exe "--Output=Text;%ID%: %Format%$if(%Language/String%, SUBTiTLE: .............. %Language/String%)\r\n" File.mkv

7: VobSub SUBTiTLE: .............. English
8: VobSub SUBTiTLE: .............. French

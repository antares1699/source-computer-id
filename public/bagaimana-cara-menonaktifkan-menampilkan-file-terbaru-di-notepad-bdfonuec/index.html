Anda dapat menonaktifkan daftar dokumen terbaru dengan mengubah pengaturan Riwayat File Terbaru di bawah

Pengaturan> Preferensi> Riwayat File Terbaru

Meskipun tidak ada fitur "nonaktifkan" yang eksplisit, Anda dapat menyetel "Jumlah entri maks." Ke 0.

(Saat ini saya menggunakan Notepad ++ v6.8.3)

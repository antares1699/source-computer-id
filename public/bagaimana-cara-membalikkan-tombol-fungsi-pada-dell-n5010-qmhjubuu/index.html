Saya menemukan solusi untuk membalik tombol Fn di Dell N5010 dari manual Pengguna. Ini dapat dilakukan baik dari Bios atau dari program Windows Mobility Center - Saya menggunakan yang terakhir: Tekan tombol Windows + "X". Saat membuka Win Mob Center, di baris tombol Fungsi, pilih 'Tombol fungsi' sebagai perilaku Tombol Fn. Voila - kembali ke bisnis.

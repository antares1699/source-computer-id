Manfaatkan akun dropbox dari solusi baris perintah murni?

Sudahkah kamu mencoba dbxcli ?

dbxcli: Alat baris perintah untuk pengguna Dropbox dan admin tim

fitur

  • Mendukung operasi file dasar seperti ls, cp, mkdir, mv (melalui Files API)
  • Mendukung pencarian
  • Mendukung revisi file dan pemulihan file
  • Unggahan terpotong untuk file besar, daftar halaman untuk direktori besar
  • Mendukung serangkaian operasi Tim yang berkembang

Instalasi

Unduh binari yang telah dikompilasi untuk Mac, Windows dan Linux dari halaman rilis.

...

$ dbxcli --help

Dropbox mendukung di Linux antarmuka baris perintah resmi melalui perintah dropbox . Seseorang perlu menavigasi ke root folder Dropbox yang sebenarnya sebelum menjalankannya perintah di bawah ini, yang biasanya ~/Dropbox.

dropbox update
dropbox filestatus [-l] [-a] [FILE]...
dropbox ls [FILE]...
dropbox puburl FILE
dropbox sharelink FILE
dropbox proxy MODE [TYPE] [HOST] [PORT] [USERNAME] [PASSWORD]
dropbox throttle DOWNLOAD UPLOAD
dropbox status
dropbox running
dropbox stop
dropbox lansync [y/n]
dropbox exclude [list]
dropbox start [-i]
dropbox autostart [y/n]
dropbox version
dropbox help [COMMAND]

Untuk informasi lebih lanjut lihat artikel bantuan Dropbox

TN3270 bagus untuk MacOS? (Mac OS X Snow Leopard)

Saya membutuhkan emulator TN3270 yang bagus untuk Mac yang mendukung transfer file ke TSO / ISPF (IND $ FILE) dan layar lebar 132 karakter. Min 132x27 (3278-model-5), lebih baik bahkan ukuran yang ditentukan pengguna seperti 132x43.

Sejauh ini saya sudah mencoba TN3270 (tanpa transfer file) dan X3270. Selain sebagai aplikasi X11, sayangnya x3270 tidak menangani layar berukuran besar dengan benar dan Mocha hanya memiliki TN5250.

Jadi, apa TN3270 pilihan di Mac (atau bagaimana setidaknya saya mendapatkan x3270 untuk menangani 132xSomething di ISPF dengan benar, misalnya daftar Kelas-Pekerjaan SDSF)?

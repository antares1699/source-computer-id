Ada trik untuk ini: Saya telah beralih sejak dari penyedia Outlook Exchange ke penyedia email IMAP / POP3 biasa. Karena saya tidak memerlukan email lama pada saat itu, saya baru saja menyimpan file .ost - Anda dapat menebaknya - saya memerlukan beberapa email lama.

.ost hanyalah cache lokal, ini tidak dimaksudkan untuk pengambilan data, terutama jika akun pertukaran Anda dihapus dari mesin Anda atau akun dihapus, jadi biasanya orang akan berkata "maaf" jika Anda tidak mengekspor email Anda ke. pst, file cadangan atau yang serupa dan host pertukaran tidak memiliki salinan cadangan untuk Anda.

Ada alat yang dapat memulihkan item darinya meskipun tidak ada jaminan. Saya telah menggunakan Recovery Toolbox untuk Outlook dengan beberapa keberhasilan untuk beberapa klien pertama kali yang malang. Versi terakhir yang saya beli mengalami masalah saat menyimpan ke file .pst dengan benar sehingga hasil akhirnya adalah banyak file datar untuk diimpor, tetapi berfungsi untuk email, kontak, dan item kalender. Saya tidak pernah mendapatkan hasil yang berguna dari alat gratis yang saya coba pertama kali, meskipun ini satu atau dua tahun yang lalu.

Anda dapat melakukan apa saja dengan file .ost lama Anda, meskipun file tersebut rusak. Jika Anda mencarinya di Google, Anda akan menemukan ratusan metode dan perangkat lunak yang mengklaim mereka akan melakukan trik untuk Anda. Tapi bagi saya, selalu lebih baik mencoba cara manual dulu. Selalu disarankan untuk menggunakan metode manual.

Terutama ada tiga metode manual yang tersedia saat bekerja dengan file OST. Yang pertama disebut Ekspor , lalu yang kedua adalah Arsip dan yang terakhir dapat dicapai dengan membuat file PST baru mengajukan. Metode langkah demi langkah dengan gambar dapat ditemukan di sini Technet tautan.

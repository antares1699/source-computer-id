EDIT: lupakan apa yang saya tulis sebelumnya. Setelah penyelidikan lebih lanjut, saya menemukan bahwa Soundrecorder secara native mendukung ini.

Buat pintasan atau batchfile atau jalankan langsung dari baris perintah dan tentukan file serta durasi dan itu akan berjalan secara diam-diam tanpa jendela. Ini akan terlihat di taskmanager di bawah proses untuk pengguna itu, tetapi solusi lama bahkan dapat menyembunyikannya hingga tingkat tertentu.

Di jalan pintas tulis yang berikut ini:

soundrecorder /FILE <filename> /DURATION <hh:mm:ss>

sebagai contoh:

soundrecorder /FILE c:\temp\output.wav /DURATION 0:0:10

Berikut adalah instruksi langkah demi langkah untuk membuat sesuatu yang akan memulai perekam suara saat windows dimulai.

  1. Tekan bola awal untuk membuka start menu.
  2. Tekan All Programs untuk menampilkan semua program.
  3. Gulir ke bawah ke tempat yang bertuliskan Startup.
  4. Klik kanan item Startup dan pilih open

  1. Klik kanan di area kosong.
  2. Dari menu popup, masuk ke menu New, lalu pilih Shortcut

  1. salin teks yang saya tulis di atas untuk pintasan dan tempel di bidang lokasi.
  2. Ubah durasi waktu sesuai keinginan Anda.

  1. tekan Lanjut
  2. tekan Selesai

Dan Anda sudah selesai. Perhatikan bahwa ini akan menimpa file gelombang setiap kali ini dimulai. Itu dapat diubah sehingga setiap kali dijalankan itu akan membuat file baru, tetapi itu akan dengan cepat membuat file besar di komputer.

Untuk mengeluarkan ke file acak, Anda dapat mengatur file ke c: \ temp \ output_% random %% random% .wav atau mungkin lebih baik, lakukan ini: c: \ temp \ output_% date% _% random% .wav

Suka baru akan terlihat seperti ini:

soundrecorder /FILE c:\temp\output_%date%_%random%.wav /DURATION 0:0:10

--- [solusi lama] ------

Memulai program sebagai pengguna yang berbeda tanpa interaksi desktop akan sepenuhnya menyembunyikannya dari windows. Di pengelola tugas Anda hanya dapat melihatnya jika Anda memilih "tampilkan proses dari semua pengguna".

Untuk melakukannya, buat akun kedua di komputer jika tidak memiliki minimal 2 akun, lalu gunakan runas / user: [username] / savecred "% SystemRoot% \ system32 \ SoundRecorder.exe / file c: \ output.wav / durasi 1:00:00 "

Pertama kali Anda harus memasukkan kata sandi, tetapi kemudian kata sandi akan disimpan dan kali berikutnya kata sandi ini digunakan. Anda dapat menghapus ini dari Manajer Kredensial di tahap selanjutnya jika Anda menginginkannya.

Ini kemudian akan merekam ke c: \ output.wav selama 1 jam. </i>

Referensi Taskbar Hide - menyembunyikan program windows 1.8

  • Sembunyikan program windows (aplikasi) atau / Show / Close //

    Menggunakan Taskbar Hide Anda dapat menyembunyikan jendela aplikasi yang sedang berjalan. Yang perlu Anda lakukan adalah memilih aplikasi di Taskbar Hide daftar dan klik Sembunyikan ikon atau klik tombol kanan mouse untuk memunculkan menu dan pilih sembunyikan item agar perubahan diterapkan. Taskbar Hide adalah juga dapat menyembunyikan / menampilkan / menutup jendela program dengan cepat menekan tombol pintas.

  • Sembunyikan / Tampilkan ikon dari baki sistem /

    Anda dapat menyembunyikan beberapa ikon yang jarang digunakan dari baki sistem. Pengoperasiannya sama dengan Hide / Show jendela aplikasi. Sekarang ini fungsi mendukung windows me / 2000 / xp / vista / 7.

  • Ubah urutan jendela aplikasi di bilah tugas

    Anda dapat mengatur jendela di bilah tugas sesuai dengan preferensi Anda. Pilih aplikasi di daftar Hide Taskbar dan gunakan Up dan panah Bawah untuk menyusun ulang jendela aplikasi. Fitur ini menjadi berguna ketika Anda ingin meletakkan jendela aplikasi terkait bersama.

  • Tetap di jendela atas

    Anda dapat memilih Selalu di atas (paling atas) atau Normal (Non-teratas). Yang pertama menempatkan jendela yang dipilih di atas semua jendela non-teratas, dan jendela ini akan mempertahankan posisi teratasnya bahkan pada saat itu dinonaktifkan. Yang terakhir menempatkan jendela yang dipilih di atas yang lainnya jendela bukan paling atas tetapi di bawah semua jendela paling atas.

  • Minimalkan ke baki

    Anda dapat meminimalkan jendela aplikasi yang sedang berjalan ke baki sistem alih-alih ikon di bilah tugas Anda. Anda bisa membawa aplikasinya kembali normal dengan menggunakan Show, atau klik dua kali tombol kiri mouse pada ikon atau klik tombol kanan mouse dari menu.

  • Sembunyikan bilah tugas

    Anda dapat memilih untuk menyembunyikan bilah tugas atau Sembunyikan / Tampilkan salah satu dari berikut ini: bilah tugas, tombol mulai, jendela tugas, baki sistem, dan jam waktu.

  • Properti Jendela

    Anda dapat mengetahui lebih banyak informasi dari jendela yang dipilih dengan Kotak dialog Properti Jendela. Seperti pegangan jendela, nama kelas, id proses, id utas, gaya jendela dan exstyle.

  • Maksimalkan semua jendela

    Anda bisa menggunakan Taskbar Hide untuk memaksimalkan semua windows atau hanya jendela IE saja bahwa mereka menutupi seluruh layar. Anda juga dapat meminimalkan semua jendela ke bilah tugas.

Taskbar Hide merupakan produk freeware, artinya Anda dapat mendownload dan menggunakannya secara gratis.

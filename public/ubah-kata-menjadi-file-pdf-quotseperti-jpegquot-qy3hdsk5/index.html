Ubah kata menjadi file pdf "seperti JPEG"

Saya memiliki dokumen kata yang saya coba simpan ke file PDF yang tidak dapat diedit dan tidak dapat dipilih. Pada dasarnya, saya ingin ini terlihat seperti JPEG, tetapi dalam format PDF. Saya mencoba untuk menghindari "mencetak ke tiff, KEMUDIAN mencetak ke PDF." Saya lebih suka membuka langsung dari Word ke PDF.

Selain itu, saya tidak ingin menambahkan Kata Sandi atau apa pun.

Apakah ada pilihan untuk ini? Terima kasih!

Mencetak sebagai file PDF vektor (atau raster) tidak akan meningkatkan keamanan file PDF Anda. File PDF itu dapat dengan mudah diimpor ke Photoshop, dimodifikasi dan diekspor kembali.

Mensertifikasi PDF Anda secara digital akan memastikannya tidak diubah tanpa otorisasi Anda. Jika ya, sertifikasi dianggap tidak valid dan Anda dapat mengkonfrontasi klien / pemasok / apapun tentang itu. Setelah disertifikasi, Anda benar-benar dapat membatasi file PDF Anda.

Lihat http://help.adobe.com/en_US/Acrobat/8.0w/Professional/help.html?content=WS58a04a822e3e50102bd615109794195ff-7d3d.html

Anda dapat melakukannya dalam 2 langkah. Pertama, cetak dokumen menggunakan driver "Microsoft Office Document Image Writer". Ini membuat file TIFF. Kemudian, buka TIFF dengan program pencitraan (Saya suka Irfanview) dan cetak menggunakan PDF writer. Atau, dengan Irfanview Anda bisa menyimpannya sebagai PDF. Berikut cara menginstal driver MODI:

http://support.microsoft.com/kb/982760

Dan Irfanview dapat diunduh dari http://www.irfanview.com/main_download_engl.htm

Langkah 1: Ubah PDF asli menjadi gambar

Menggunakan jawaban ini: https://askubuntu.com/a/50180/400409

pdftoppm input.pdf pageAAA -png

Langkah 2: Konversikan gambar ke PDF yang tidak dapat diponsel

Sekali lagi, gunakan jawaban lain: https://stackoverflow.com/a/8955465/1115187

convert pageAAA*.png ro-document.pdf

Kedua utilitas pdftoppm, convert memiliki banyak pilihan, Anda dapat memilih nomor halaman / kualitas dan hal lainnya. Dan keduanya telah dipasang sebelumnya di Ubuntu saya.

Sistem desktop tidak selalu menekan BIOS

Saya memiliki sistem desktop yang berjalan dengan baik selama berbulan-bulan, tetapi sekarang tidak lagi masuk ke BIOS.

Saya telah mengganti baterai pada motherboard (CR2032) dan catu daya, dan perilakunya tidak berubah.

Jika saya memutuskan sambungan kotak dari daya untuk waktu yang lama (beberapa jam) dan kemudian menyambungkan sistem, terkadang saya akan berhasil melewati BIOS. Suatu kali saya berhasil masuk ke layar login dan cara bisa login. Setelah beberapa menit, OS membeku dan saya harus melakukan boot ulang dengan keras. Kadang-kadang urutan boot akan membuatnya sejauh layar yang mengatakan "tekan delete to setup BIOS", tetapi setelah beberapa reboot, saya bahkan tidak akan menekan layar itu lagi, dan tidak ada yang akan ditampilkan ke layar.

detil tambahan: motherboard dan kartu video lebih tua dan diberikan kepada saya oleh seorang teman. Semua perangkat keras tidak bergaransi.

tolong beri tahu saya jika ada detail yang ingin Anda ketahui.

Bagaimana saya bisa mengedit banyak preferensi LastPass sekaligus?

Tetapi untuk mengedit daftar kata sandi LastPass saya, saya harus melakukannya cara terlalu banyak klik. Terutama ketika ia telah mengingat banyak kata sandi untuk server yang berbeda di domain yang sama, tetapi ia hanya membedakannya dengan (2) (3) (4) dll., Bukan dengan nama server atau URL.

Jadi ... apakah ada cara untuk mendapatkan "tabel" dari kata sandi LastPass saya yang menunjukkan penuh URL masing-masing terkait dengan, nama pengguna, dan kata sandi, semuanya dalam satu tampilan, jadi saya bisa tahu apa itu (2) (3) (4) tanpa mengklik?

Bahkan jika saya tidak dapat mengedit dari tabel seperti itu, itu akan sangat bagus. Tetapi jika dimungkinkan juga untuk menerapkan pengaturan secara massal, mis. IsiOtomatis, masuk otomatis, dll., Itu juga akan sangat membantu. Terima kasih! NB Saya menggunakan LastPass di Google Chrome 6 di Mac OS X - tidak yakin apakah itu penting.

Ekstensi LastPass Chrome hadir dengan fitur ekspor, Anda dapat menemukannya di LastPass -> Alat -> Ekspor Ke. Pilih CSV "LastPass".

Baris pertama menunjukkan urutan kolom

url,username,password,extra,name,grouping,fav

Setelah Anda mengekspor CSV - Anda membukanya di Excel, edit ke apa pun yang Anda inginkan & impor kembali ke LastPass

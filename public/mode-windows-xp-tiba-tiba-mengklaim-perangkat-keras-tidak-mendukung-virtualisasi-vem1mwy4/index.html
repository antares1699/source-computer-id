Mode Windows XP tiba-tiba mengklaim perangkat keras tidak mendukung virtualisasi

Saya memiliki sistem PC dengan prosesor dual-core AMD X2 dan Windows 7 profesional.

Saya menggunakan mode Windows XP untuk menggunakan pemindai yang tidak memiliki driver Windows 7. Ini dulu bekerja dengan baik.

Tetapi tiba-tiba, Microsoft Virtual PC menolak untuk memulai dengan pesan yang kira-kira mengatakan (diterjemahkan dari bahasa Jerman jadi bukan kata demi kata):

Mode Windows XP tidak dapat dimulai karena ketidakcocokan dalam jenis prosesor host, atau karena virtualisasi perangkat keras tidak didukung.

Saya belum mengubah konfigurasi prosesor saya, atau apa pun di sistem.

Saya telah memeriksa BIOS dan tombol "Dukungan virtualisasi" dimatikan. Saya menyalakannya, tetapi tidak berhasil.

Apa alasannya?

Masalah ini disebabkan oleh file hibernasi yang rusak.

Saya memiliki masalah yang sama dan dapat memperbaikinya menggunakan artikel ini .

Ini adalah langkah-langkah dari artikel yang saya gunakan untuk memperbaiki masalah saya.

  1. Klik Mulai dan klik komputer.
  2. Pergi ke lokasi berikut: C: \ Users \ (Nama Pengguna) \ AppData \ Local \ Microsoft \ Windows Direktori Virtual PC \ Mesin Virtual
  3. Anda akan menemukan file .VSV di Direktori Mesin Virtual.
  4. Ubah nama file. Misalnya: Jika nama file adalah VM.vsv, ganti namanya menjadiVM.old
  5. Tutup semua jendela dan mulai ulang Mesin Virtual. kamu harus bisa menjalankannya tanpa masalah.

Mengubah FLAC ke MP3 menggunakan FFMPEG?

Jadi saya menggunakan ffmpeg untuk mengonversi file flac dari folder tertentu ke mp3 pada kecepatan bit 320k.

Saya menggunakan jawaban ini sebagai referensi.

https://stackoverflow.com/a/41096234/9259505

Jadi perintah yang saya gunakan di shell perintah Windows adalah

ffmpeg  -i infile.flac  -c:v copy  -b:a 320k  outfile.mp3

tapi itu hanya untuk satu lagu. Bagaimana cara mengubah perintah tersebut untuk menelusuri semua file .flac di folder direktori kerja?

Saya sudah mencoba 2 ini sendiri tetapi tidak berhasil.

for file in 'C:\Users\...' *.flac; do ffmpeg -i "$file" -c:v copy -q:a 0 "${file%.flac}".mp3; done

ffmpeg  -i "*.flac"  -c:v copy  -q:a 0  outfile.mp3

Catatan: Saya menggunakan salinan -c: v karena sampul album akhirnya di-transcoding sehingga menghasilkan file yang jauh lebih besar yang saya coba hindari. Jadi perintahnya harus menyalin aliran.

Sistem Operasi: Windows 10

Anda mengatakan Anda menggunakan shell perintah Windows, tetapi baris perintah pertama yang Anda tunjukkan (yang dimulai "untuk file dalam") tidak terlihat seperti Windows, lebih seperti beberapa jenis perintah shell Linux. Yang kedua tidak akan berfungsi karena ffmpeg tidak akan menerima wildcard sebagai spesifikasi file input.

Perintah satu baris Windows ini melakukan apa yang Anda inginkan, untuk membuat ffmpeg (menggunakan opsi Anda di atas) mengambil masukannya, pada gilirannya, setiap file flac di folder dan output saat ini, di folder yang sama, file mp3 dengan file yang sama. nama sebelum ekstensi:

Pada prompt:

for %A in (*.flac) do ffmpeg  -i "%~nA.flac"  -c:v copy  -b:a 320k  "%~nA.mp3"

Jika file mp3 sudah ada, ffmpeg akan menanyakan apakah Anda ingin menimpanya.

Catatan: perintah di atas dibuat untuk command prompt. Ini tidak akan berfungsi dalam skrip batch. Untuk itu, Anda perlu menggandakan semua tanda persen (%) seperti ini

Dalam skrip batch (cmd):

for %%A in (*.flac) do ffmpeg  -i "%%~nA.flac"  -c:v copy  -b:a 320k  "%%~nA.mp3"

Hapus pengguna di Windows XP, tidak dapat menemukan tombol hapus atau tidak muncul

Saya menambahkan akun percobaan ke mesin XP virtual saya minggu lalu dan sekarang saya ingin menghapusnya.

Tombol hapus pengguna tidak terlihat di halaman mereka. Pada halaman akun pengguna, keterangan alat mengatakan bahwa mengklik pengguna akan memungkinkan saya untuk menghapusnya - tetapi tidak ada tombol untuk ini.

Saya memiliki penyetelan mesin untuk melakukan logon otomatis - mungkin itu menghapus fasilitas hapus. Tetapi itu masuk seperti biasanya (akun lain) saya sehingga Anda akan berpikir bahwa itu tidak akan mempengaruhi banyak hal.

Apakah ada izin yang perlu saya aktifkan?

Coba baris perintah ini

net user yourUserNameHere /DELETE

Misalnya, jika nama pengguna uji Anda adalah "ABCD", Anda akan menjalankan ini:

net user ABCD /DELETE

Pastikan Anda masuk sebagai administrator mesin.

Jika Anda ingin mencoba GUI yang berbeda, klik kanan pada My Computer dan pilih Manage. Pada jendela Computer Management pilih Local Users and Groups > Users. Dalam tampilan tersebut, klik kanan User dan pilih Delete.

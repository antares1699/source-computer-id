Bagaimana cara mendapatkan akses melalui C: \ Windows \ Logs \ CBS \ CBS.log?

Saya tidak dapat mengakses file log ini karena beberapa alasan. Inilah yang telah saya coba sejauh ini:

  • jalankan prompt: C: \ Windows \ Logs \ CBS \ CBS.log
  • notepad: C: \ Windows \ Logs \ CBS \ CBS.log
  • cmd yang ditinggikan: C: \ Windows \ Logs \ CBS \ CBS.log
  • notepad ++

Microsoft Windows [Version 6.0.6002]

C:\Windows\system32>C:\Windows\Logs\CBS\CBS.log
Access is denied.

C:\Windows\system32>

Apa yang saya lewatkan? Bagaimana cara mengakses file ini?

Saya mencoba melihat log ini sebagai bagian dari operasi pemeriksa file sistem. Itu telah menemukan file yang rusak dan itu dengan jelas menunjukkan bahwa saya dapat menemukan log di lokasi ini.

Windows Resource Protection found corrupt files and successfully repaired
them. Details are included in the CBS.Log windir\Logs\CBS\CBS.log. For
example C:\Windows\Logs\CBS\CBS.log

Tetapi untuk beberapa alasan tidak berdasar Windows tidak mengizinkan saya untuk melihat file ini. Mengapa?...

Itu Artikel Microsoft di SFC menyarankan menggunakan perintah berikut untuk menyalin entri SFC dari log ke desktop Anda. Anda kemudian dapat membukanya di Notepad.

1. Buka prompt perintah yang ditinggikan seperti yang dijelaskan di langkah 1 sebelumnya.

2. Di prompt perintah, ketik perintah berikut ini, dan kemudian tekan ENTER:

findstr / c: "[SR]"% windir% \ Logs \ CBS \ CBS.log> "% userprofile% \ Desktop \ sfcdetails.txt"

Bagikan

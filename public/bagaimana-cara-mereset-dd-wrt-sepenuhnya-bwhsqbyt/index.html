Bagaimana cara mereset DD-WRT sepenuhnya?

Saya memiliki router nirkabel Linksys WRT54GS yang di-flash dengan DD-WRT, dan saya lupa nama pengguna dan kata sandi untuk itu. Jadi, saya tidak dapat mengatur ulang kata sandi dari antarmuka web.

Tanpa akses ke antarmuka web, bagaimana saya dapat mengatur ulang semuanya?

Untuk mengatur ulang sepenuhnya router yang di-flash dengan DD-WRT, lakukan a 30-30-30 ulang :

Prosedur berikut akan menghapus NVRAM dan mengembalikan dd-wrt ke nilai default:

  • Dengan unit menyala, tekan dan tahan tombol reset di belakang unit selama 30 detik.
  • Tanpa melepaskan tombol reset, cabut unit dan tahan reset selama 30 detik.
  • Colokkan unit kembali MASIH menahan tombol reset 30 detik terakhir (harap dicatat bahwa langkah ini dapat menempatkan perangkat Asus ke mode pemulihan)

Setelah Anda melakukan ini, Anda harus masuk dengan nama pengguna default ("root") dan kata sandi ("admin").

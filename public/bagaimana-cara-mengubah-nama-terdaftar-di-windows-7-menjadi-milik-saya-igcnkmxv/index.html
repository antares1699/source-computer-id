Bagaimana cara mengubah nama terdaftar di Windows 7 menjadi milik saya?

bagaimana cara mengubah nama terdaftar di Windows 7 menjadi nama saya sendiri?

7

Melalui perubahan registri:

  • tekan Tombol Windows + R dan ketik regedit. Kemudian tekan Memasukkan .
  • Navigasi ke:

    HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows NT \ CurrentVersion

Seperti terlihat di atas, Anda dapat melihat RegisteredOrganization dan RegisteredOwner. Klik dua kali untuk mengubah pemilik / organisasi terdaftar!

Untuk mengonfirmasi perubahan, cukup ketik winver.exe di kotak pencarian Mulai atau di kotak dialog Jalankan Perintah, dan Anda seharusnya dapat melihat perubahan Anda tanpa memulai ulang.

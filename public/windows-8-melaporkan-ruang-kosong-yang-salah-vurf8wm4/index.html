Windows 8 melaporkan ruang kosong yang salah

Saya memiliki laptop Windows 8, dengan SSD Samsung 64GB. Ketika saya mengklik kanan disk saya dan mengklik properti, dikatakan, bahwa saya menggunakan 58,7GB, tetapi jika saya hanya menandai semua file (saya telah mengaktifkan menampilkan file tersembunyi dan file yang dilindungi sistem) dikatakan mereka menggunakan 49,9GB ( windirstat mengatakan hal yang sama). Kemana perginya 10GB saya? Saya telah menonaktifkan file SWAP, saya telah menonaktifkan perlindungan sistem dan salinan bayangan. Saya mencoba chkdsk (mengatakan saya tidak memiliki kesalahan). Apa yang saya lewatkan?

Salah satu kemungkinannya adalah folder "WindowsApps" di bawah Program Files, yang menyimpan aplikasi Metro UI. Karena mereka berjalan di kotak pasir yang aman, ada banyak duplikasi file dan semuanya bisa menjadi relatif besar dengan cepat jika Anda menginstal beberapa aplikasi toko.

Folder ini tidak hanya disembunyikan, tetapi Anda sebagai pengguna (atau bahkan Anda sebagai administrator) tidak memiliki izin untuk membacanya dan Anda juga bukan pemiliknya. Hal ini dapat mencegah pencarian atau klik kanan + properti pada folder untuk menghitungnya. Mungkin ada file lain dalam keadaan diblokir seperti itu di drive C, tetapi ini akan menjadi potongan terbesar yang dapat saya pikirkan.

Lihat jawaban ini untuk detail lebih lanjut tentang cara mengakses folder WinApps.

Saya tidak akan menyarankan untuk menghapus semua yang ada di sana, tetapi saya tidak mengalami masalah (sejauh ini!) Menghapus folder yang saya tahu tidak saya gunakan.

Edit: Coba Utilitas gratis TreeSize dan lihat hasil apa yang Anda dapatkan. Saya pertama kali memperhatikan folder WindowsApps yang menggunakannya, jadi sepertinya crawler HDD yang dapat dipercaya.

Laptop tidak dilengkapi dengan DVD instalasi tetapi memiliki partisi tersembunyi sekitar 10GB yang memungkinkan Anda menginstal OS setelah memformat PC Anda.

Tekan tombol Windows lalu ketik "Manajemen Komputer" dan jalankan. Pilih Storage lalu Disk Management dari kiri. Anda dapat melihat partisi tersembunyi di sana.

Perhatikan bahwa jika Anda menghapusnya, Anda tidak dapat menginstal OS Anda tanpa DVD instalasi lagi.

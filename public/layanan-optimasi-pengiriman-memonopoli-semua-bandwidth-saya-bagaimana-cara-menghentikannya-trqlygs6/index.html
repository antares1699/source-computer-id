Layanan Optimasi Pengiriman memonopoli semua bandwidth saya, bagaimana cara menghentikannya?

Saya menginstal Windows 10 Home di laptop saya. Saya perhatikan bahwa ada proses 'Service Host: Delivery Optimization' yang memonopoli semua bandwidth. Saya tidak dapat menggunakan internet saya untuk menjelajah.

Dari ini tautan Saya menyimpulkan bahwa Anda dapat menonaktifkannya dengan mematikan pengaturan 'Pembaruan dari lebih dari satu tempat'. Saya melakukannya dan memulai ulang PC saya dan masih tidak berhasil. Saya dapat melihat layanan ini pada services.msc dan disetel ke 'Otomatis (Start Tertunda)'. Bukankah layanan ini harus dinonaktifkan ketika saya mematikan 'Pembaruan dari lebih dari satu tempat'?

Bagaimana saya bisa membuat layanan ini tidak berjalan? Apakah aman jika saya menonaktifkan layanan ini dari 'services.msc'?

Oke, jadi ada pengaturan lain yang perlu dimatikan. Kutipan dari ini tautan (ini juga menjelaskan beberapa hal lain).

Saya membawa saya sampai beberapa hari yang lalu untuk menyadari itu bukan Pembaruan Windows lagipula sebenarnya aplikasi (game) yang baru-baru ini saya instal Windows Store sedang memperbarui dirinya sendiri. Jadi saya mencoba langkah-langkah ini:

  1. Buka aplikasi Windows Store

  2. Buka Pengaturan (klik gambar profil Anda untuk mengakses menu)

  3. Matikan "Perbarui aplikasi secara otomatis"

Ini memecahkan masalah saya untuk saat ini tetapi ini hanya solusi dan Anda harus memperbarui aplikasi Windows Store secara manual mulai dari sini.

Anda mungkin ingin mencopot pemasangan aplikasi yang mungkin tidak Anda perlukan karena perlu diperbarui tetapi Anda mungkin tidak pernah menggunakannya.

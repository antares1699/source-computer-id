Minta file muncul di beberapa folder

Saya memiliki 50+ folder, yang masing-masing berisi banyak data yang perlu diproses. Semuanya diproses menggunakan kode yang sama persis, memanfaatkan os.path.dirname (os.path.realpath ( mengajukan )) untuk mendapatkan direktori tempat skrip python berada sehingga tidak diperlukan pengeditan manual oleh pengguna, mereka hanya perlu mengklik dua kali.

Saya membutuhkan skrip untuk muncul di setiap folder sementara sebenarnya hanya berada di 1 tempat, jadi saya dapat mengeditnya sekali dan kemudian ketika dijalankan dari salah satu lokasi ini, jalur foldernya sudah benar. Alternatifnya adalah mengedit master dan kemudian menempelkan satu folder pada satu waktu meskipun semua 50+ folder setiap kali saya memperbarui kode, yang sangat membosankan dan rawan kesalahan. Di Linux saya dapat mengatur ini dengan tautan simbolis, tetapi saya tidak dapat menemukan cara untuk melakukannya di windows.

Bergantian, cara untuk menempelkan file ke semua direktori target sekaligus, bukan satu per satu, akan mencapai tujuan yang sama.

Anda membutuhkan tautan simbolis atau tautan keras.

Tautan simbolis (atau disingkat symlink) sangat mirip dengan pintasan: ada satu file aktual dan banyak referensi (symlink) ke sana. Mereka bahkan memiliki ikon panah kecil di atasnya. Tidak seperti pintasan, tautan simbolis dapat memiliki ekstensi apa pun.

Tautan keras bind file pada hard drive ke lokasi di pohon direktori. Setiap file memiliki setidaknya satu tautan keras, jika tidak maka tidak akan ada di direktori mana pun. Jika sebuah file memiliki beberapa hard link, yang asli tidak dapat dibedakan dan file tersebut secara fisik hanya ada di satu lokasi.

Keduanya memiliki keterbatasan:

  • Beberapa perangkat lunak tidak bekerja dengan baik dengan symlink
  • Menghapus file asli membuat semua symlinknya rusak
  • Anda tidak dapat melakukan hard link folder (tetapi Anda dapat membuat file persimpangan direktori jika symlink tidak cukup)
  • Membuat hard link lintas-partisi tidak mungkin dilakukan

Symlinks biasanya cukup.

Untuk membuat symlink atau hard link:

  1. Luncurkan baris perintah dengan hak istimewa: tekan Menang , ketik cmd, tekan Ctrl + Bergeser + Memasukkan

  2. Masalah mklink perintah:

    • mklink link_name link_target untuk file symlink
    • mklink /d link_name link_target untuk symlink folder
    • mklink /h link_name link_target untuk file hard link
    • mklink /j link_name link_target untuk sambungan direktori

Di Win Vista dan yang lebih baru, Anda dapat membuat tautan sym atau tautan keras, yang berperilaku seperti yang Anda harapkan di Linux. Anda dapat membuat tautan keras atau tautan sym dengan menggunakan perintah MKLINK bawaan Windows Visa dan yang lebih baru. (Membuat tautan sym membutuhkan hak istimewa admin.)

Atau, untuk alternatif yang Anda sebutkan, jika folder hanyalah subfolder langsung dari folder tempat skrip berada, Anda dapat melakukannya di prompt perintah di folder skrip:

for /d %D in (*) do (copy /y "myscript.py" "%D")

untuk memperbaruinya sekaligus. Jika menulis perintah ini secara batch, Anda perlu menggunakan kode berikut:

for /d %%D in (*) do (copy /y "myscript.py" "%%D")

Firefox dan Chrome lambat di localhost; perbaikan yang diketahui tidak berfungsi di Windows 7

Firefox dan Chrome diketahui lambat di localhost saat IP6 diaktifkan. Di versi Windows sebelumnya, perbaikan paling sederhana adalah mengomentari baris ini dari file host, seperti yang dijelaskan di jawaban atas pertanyaan ini .

::1 localhost

Namun, seperti yang disebutkan dalam pertanyaan ini , di Windows 7 baris ini sudah diberi komentar:

# localhost name resolution is handled within DNS itself.
#   127.0.0.1 localhost
#   ::1 localhost

Apakah ada cara alternatif untuk menonaktifkan referensi ::1 localhost di Windows 7?

Ternyata jika Anda menghapus komentar pada baris 127.0.0.1 di file host, Chrome kembali ke host tajamnya di URL localhost.

# localhost name resolution is handled within DNS itself.
    127.0.0.1 localhost

File host biasanya berukuran C:\WINDOWS\system32\drivers\etc\hosts. Untuk mengeditnya di Win7, Anda harus menjalankan Notepad sebagai administrator.

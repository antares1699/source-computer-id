Mulai (satu kali) dengan hak akses administratif dan aktifkan. Jika Anda keluar dari Notepad ++, itu akan disimpan.

Saya menganalisis kode sumber plugin ini dan saya telah melihat bahwa itu dinonaktifkan secara default. Jadi saya mengubah pengaturan default ini dan memasukkannya ke github . Anda dapat mengunduh plugin baru langsung di sini . Cukup salin ke direktori Notepad ++ \ plugins Anda.

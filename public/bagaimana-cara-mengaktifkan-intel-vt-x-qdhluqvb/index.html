Baiklah, jadi setelah berjam-jam mencari, saya rasa saya tahu jawabannya. Saya akan memposting apa yang saya ketahui di sini jika ada orang lain yang membutuhkannya suatu hari nanti.

Untuk mengaktifkan teknologi Intel VT ada 3 hal yang dibutuhkan:

  • Prosesor yang mendukung VT-x
  • Motherboard dengan chipset yang mendukung VT-d
  • Bios yang mendukung VT-d juga

Ternyata, alasan saya tidak dapat mengaktifkan VT, itu karena motherboard saya tidak mendukung VT-d, dan meskipun prosesor saya mendukung VT-x, itu tidak masalah. Itu semua atau tidak sama sekali.

Bagaimana cara mengetahui apakah motherboard mendukung VT-d? Itu sangat tergantung chipset yang dimilikinya. Dalam kasus Intel, posting ini sangat membantu:

  • Apakah produk saya mendukung Intel VT?
  • Papan Desktop

Jadi setelah berjam-jam mencari spesifikasi motherboard Clevo W150ER ( Tinjau Clevo W150ER Barebone Buku catatan ), Saya menemukan bahwa itu memegang Model Panther Point Intel HM76 yang tidak mendukung Intel VT-d.

Harap perhatikan bahwa mendukung VT-d tidak cukup untuk mendukung Intel VT. Prosesor harus mendukung Intel VT-x, dan Bios juga harus mendukung saya t. Jika Anda memerlukan penggunaan jaringan khusus, tanda VT khusus juga diperlukan, meskipun saya tidak dapat mengingatnya.

Saya melihat ini untuk mengetahui apakah mesin akan mendukung pass-through atau tidak sebelum saya kehilangan waktu untuk menginstalnya kembali. Sangat sulit untuk menemukan bagaimana tepatnya untuk mengetahuinya sebelum mencoba untuk melewati perangkat PCI.

dmesg | grep -e IOMMU -e DMAR -e PCI-DMA

Bergantung pada versi kernel Anda, Anda perlu melihat

DMAR: Intel(R) Virtualization Technology for Directed I/O

atau

PCI-DMA: Intel(R) Virtualization Technology for Directed I/O

Hanya Intel-IOMMU: enabled atau DMAR: IOMMU enabled tidak cukup untuk mengetahui semuanya baik-baik saja.

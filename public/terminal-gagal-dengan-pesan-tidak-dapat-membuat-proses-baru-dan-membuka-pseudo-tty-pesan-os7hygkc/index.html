Kemarin, saya meningkatkan Mac saya ke Mac OS X 10.9 (Mavericks), dan sejak itu saya tidak dapat membuka "Terminal", setiap kali saya membuka sesi, kesalahan berikut ditampilkan:

[Tidak dapat membuat proses baru dan membuka pseudo-tty.]

[Perintah tidak ditemukan: tmux]

Saya tidak tahu apa yang menyebabkan masalah ini dan memulai ulang baru serta menghentikan proses yang relevan di monitor aktivitas tidak membantu. Ada pemikiran?

Prioritas CPU Windows 10 Bluetooth

Bagaimana cara mengatur prioritas tinggi untuk proses / layanan bluetooth di Windows 10? Di Windows 7 ralink memiliki proses yang memiliki default tinggi, tetapi di sini tidak ada proses yang terhubung dengan BT, tanpa itu saya mengalami penundaan pada headset BT saya :(

"svchost.exe - k bthsvcs" adalah proses tepat yang perlu Anda lakukan jika Anda ingin memastikan Prioritas CPU Tinggi terutama untuk hal-hal seperti A2DP / AptX dan memastikan latensi rendah serta kualitas audio yang tinggi. Inilah alasan saya harus mencari tahu, sebagai Game di mana membuat Headphone Bluetooth saya melompat dan menjatuhkan kualitas meskipun sinyalnya sangat bagus.

Saya menggunakan program Gratis (Nag sedikit saja), Proses Lasso, untuk mengotomatiskan pengaturan proses ke Prioritas Tinggi. Masalah Terpecahkan untuk saya :)

Harap perhatikan beberapa program tidak memisahkan setiap Layanan dan hanya menampilkan "svchost.exe"

Lihat tab detail pengelola tugas.

  1. Tekan CTRL + SHIFT + ESC
  2. Di jendela yang muncul, klik tab detail.
  3. Klik kanan layanan yang Anda inginkan.
  4. Arahkan mouse ke Atur Prioritas.

Di Command Prompt, jalankan tasklist /svc /fi “imagename eq svchost.exe” untuk melihat daftar ID proses.

Temukan layanan 'bthserv' dan perhatikan PID-nya. Kemudian Tekan CTRL + SHIFT + ESC, pilih PID yang sesuai dan atur ke prioritas tinggi.

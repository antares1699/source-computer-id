Bagaimana saya bisa mengubah PDF untuk mendapatkan PDF yang sama tetapi dengan empat Halaman dalam satu?

Di universitas saya, kami memiliki layanan web yang memungkinkan kami mengunggah file PDF dan mencetaknya. Terkadang saya ingin mencetak dua atau bahkan empat halaman dokumen pada satu halaman kertas. Karena layanan online tidak mengizinkan saya untuk memilih opsi ini (seperti yang dilakukan Adobe Reader), saya harus mengubah PDF.

Bagaimana saya bisa mengubah PDF dengan cara ini (di mesin Ubuntu Linux)?

  • Buka File> Cetak
  • Pilih Print to File dengan tipe keluaran PDF
  • Pada tab penyetelan halaman, cari Pages per side
  • Di sini pilih 4 dan tekan mencetak .

Ini akan membuat file baru dengan 4 halaman di setiap halaman.

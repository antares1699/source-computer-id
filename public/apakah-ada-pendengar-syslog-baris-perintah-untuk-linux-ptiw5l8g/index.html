Apakah ada pendengar syslog baris perintah untuk Linux?

Kami memiliki beberapa sistem tertanam yang dapat dikonfigurasi untuk mengirim data log ke server jauh untuk keperluan debugging. Namun demikian kerumitan mengkonfigurasi mesin pengembang untuk mendengarkan dan memfilter data ini kebanyakan orang cenderung hanya menonton lalu lintas syslog dengan sesuatu seperti Wireshark. Apakah ada alat syslog baris perintah yang dapat dikonfigurasi untuk mendengarkan penggunaan port sementara dalam skenario debugging ini?

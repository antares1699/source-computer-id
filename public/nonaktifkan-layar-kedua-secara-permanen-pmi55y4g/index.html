Nonaktifkan layar kedua secara permanen

Monitor laptop saya tidak berfungsi lagi dan karena itu saya telah memasang monitor eksternal.

Saat boot, GDM masih mengenali layar laptop utama dan menampilkan jendela loginnya. Yang kemudian tidak bisa saya lihat.

Saya dapat menonaktifkan layar laptop dengan menjalankan xrandr --output LVDS1 --off setelah masuk. Saya mencoba memasukkan perintah ini ke ~\.xinitrc dan menjadi ./40-xrandr.sh pada /etc/X11/xinit/xinitrc.d.

Keduanya tidak berpengaruh sama sekali. Saya berharap setidaknya akan mematikan layar laptop saat masuk.

Bagaimana cara menonaktifkan layar laptop sebelum GDM dimulai?

Ini membuat linux mengabaikan output LVDS sama sekali. Dengan begitu, ini tidak akan digunakan oleh semua pengelola login, dan tidak akan muncul di dialog konfigurasi tampilan.

Buat file /usr/share/X11/xorg.conf.d/50-disable-lvds.conf dan tulis di sana:

Section "Monitor"
        Identifier      "lvds monitor"
        Option  "ignore"        "true"
EndSection
Section "Device"
        Identifier      "onboard"
        Option  "Monitor-LVDS1" "lvds monitor"
EndSection
Bagikan

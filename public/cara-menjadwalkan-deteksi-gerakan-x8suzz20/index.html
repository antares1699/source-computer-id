Cara menjadwalkan deteksi Gerakan

Saya baru saja menemukan Gerakan dan bekerja seperti pesona .. Satu-satunya masalah saya adalah saya ingin itu aktif hanya pada malam hari, tetapi tidak ada parameter untuk penjadwalan ..

Saya rasa saya bisa menulis program yang akan mengontrol Motion melalui programnya "HTTP API" , tetapi saya bertanya-tanya apakah seseorang telah melakukannya, atau apakah ada solusi alternatif.

Terima kasih sebelumnya :)

Untuk tugas sederhana ini, API HTTP Motion dapat digunakan langsung dengan wget atau yang serupa ..
Langsung dari FAQ :

0 9 * * * root /usr/bin/lwp-request http://localhost:8080/0/detection/start > /dev/null 
0 18 * * * root /usr/bin/lwp-request http://localhost:8080/0/detection/pause > /dev/null

Gunakan cron:

crontab -e

lalu taruh di dalamnya:

DISPLAY=:0.0
0 0 * * * /usr/bin/motion 
0 8 * * * /usr/bin/killall motion

Saya berasumsi bahwa Anda ingin membuka "gerakan" pada tengah malam setiap hari, dan jalur gerakan itu adalah / usr / bin / motion: sesuaikan crontab ini dengan kebutuhan Anda dan itu akan berhasil.

Tampilkan konten saat mengubah ukuran di chrome?

Sebagai pengembang web - Saya perlu menguji bagaimana hal-hal (tata letak) berubah secara real time Saat saya mengubah ukuran lebar / tinggi layar

Namun ini tidak berfungsi seperti yang diinginkan:

Apakah ada solusi / plugin untuk melakukan itu?

Hasil yang diinginkan: saat mengubah ukuran - lihat konten. (tidak hanya saat melepaskan mouse)

Ada efek di "System Properties" Windows bernama "Show content when moving or resizing windows". Jika itu dinonaktifkan, Anda tidak dapat melihat konten saat mengubah ukuran atau memindahkan jendela. Terkadang itu akan dinonaktifkan untuk meningkatkan kinerja sistem. Periksa.

Dengan asumsi Windows 7:

  1. Klik "Mulai"
  2. Klik kanan "Komputer"
  3. Klik "Properties"
  4. Klik "Pengaturan sistem lanjutan"
  5. Dari tab "Advanced", klik tombol "Settings" di bawah "Performance"
  6. Pilih "Sesuaikan untuk performa terbaik" dan klik "Terapkan" (layar Anda akan menyesuaikan, bersabarlah)
  7. Pilih "Biarkan Windows memilih yang terbaik untuk komputer saya" dan klik "Terapkan" (layar Anda akan menyesuaikan ulang, bersabarlah)
  8. Klik "OK" untuk menutup jendela System Properties
  9. Klik "OK" untuk menutup jendela Properti Sistem induk dan kembali ke desktop Anda
  10. Tutup semua jendela browser Chrome Anda
  11. Buka kembali jendela browser Chrome Anda dan uji untuk mengonfirmasi bahwa masalah telah diperbaiki.

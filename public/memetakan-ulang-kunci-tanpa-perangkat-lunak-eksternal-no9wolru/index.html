Alat terbaik untuk pekerjaan itu adalah milik Microsoft remapkey.exe, tersedia sebagai bagian dari Kit Sumber Daya Windows . (Tautan itu untuk Windows 2003; Saya telah menggunakannya dengan sukses selama bertahun-tahun pada banyak versi Windows, termasuk Windows 7; belum menguji Windows 8).

remapkey.exe akan menulis pemetaan langsung ke registri, saya yakin, dan tidak memerlukan proses yang sedang berjalan. Anda cukup menyeret dari kunci sebenarnya ke kunci yang diinginkan, dan simpan.

Ada alat gratis yang saya gunakan di Windows 7 dan Windows 8 ketika saya tidak dapat lagi menggunakan utilitas remapkey Windows Server 2003. Ini disebut SharpKeys. Itu melakukan hal yang sama (dengan UI yang berbeda.

TAPI, ini tidak berfungsi di Windows 8.1, dan saya belum menemukan apa pun yang berfungsi. Itu membuat saya sedih, karena MacBook Pro tidak memiliki file Del key dan saya biasa memetakannya kembali ke Right-Windows. Sekarang saya harus menggunakan On Screen Keyboard untuk login.

Bagaimanapun, jika Anda tidak meningkatkan ke Windows 8.1, maka SharpKeys pasti bagus untuk Anda.

Area utama Dock (yaitu sisi kiri jika Dock berada di bagian bawah layar) adalah hanya untuk aplikasi . Oleh karena itu, Anda tidak dapat meletakkan alias non-aplikasi Anda di sini. Ini terdengar seperti apa yang Anda coba lakukan.

Namun , sejak OS X 10.5, sisi kanan memiliki sisi Dock (sekali lagi jika ada di bagian bawah layar) di sebelah Sampah memungkinkan Anda untuk meletakkan file, folder, dan alias di sana. Fitur ini disebut Tumpukan . Ketika Anda pertama kali menginstal OS X, Anda mungkin memiliki tumpukan untuk folder ~/Downloads Anda di sana. File atau folder lebih lanjut dapat ditambahkan sesuai keinginan pengguna.

Coba seret file alias Anda ke sana dan Anda akan berada dalam kondisi yang baik.

Pengganti alias atau symlink di dok (berfungsi di High Sierra 10.13.6).

Contoh: Saya ingin menjalankan iTunes versi lama. OS (atau mungkin iTunes itu sendiri) bersikeras bahwa itu dipanggil sebagai "iTunes.app". Setelah menghapus versi iTunes yang tidak diinginkan (yang merupakan prosedur lain):

  1. Dalam / Applications / iTunesVersions: iTunes12.6.5.3.app [direktori saya dan versi lama iTunes]

  2. Di / Aplikasi, buat symlink: iTunes.app -> / Applications / iTunesVersions /iTunes12.6.5.3.app

  3. Symlink dapat diklik. Klik dan aktifkan iTunes melalui symlink. Ikon iTunes akan muncul di sisi kanan dok.

  4. Klik pada "options-dot" di bawah ikon iTunes di dock dan pilih Options-> Keep In Doc. Keluar dari iTunes.

  5. Untuk memindahkan ikon iTunes ke posisi dok pilihan Anda, tahan tombol perintah dan opsi sambil menahan tombol mouse [kiri] atau tombol trackpad dan pindahkan ke tempat yang Anda inginkan di dok.

Ini bukan symlink atau alias, tetapi IS ikon dok yang berfungsi dan di mana Anda menginginkannya.

Chmod 755 dan 751

Melakukan chmod 755 pada direktori memberi saya drwxr-xr-x. Haruskah itu tidak memberi saya drwxr-xr-xr karena 5 dibaca, jalankan?

Dan Melakukan chmod 751 pada direktori memberi saya drwxr-x - x. Bukankah seharusnya itu memberi saya drwxr-xr-x?

Mohon jelaskan.

Bit adalah kelompok tiga hasilnya benar, saya tidak benar-benar mengerti maksud Anda salah paham. 5 dibaca, dieksekusi tetapi urutan rwx tidak berubah. Anggap mereka sebagai sakelar, mereka mungkin hidup (r / w / x) atau mati (-). Di sini sekali lagi ikhtisar nilai:

7(8) = 111(2) => rwx
5(8) = 101(2) => r-x
1(8) = 001(2) => --x

Jawaban: Ini rwx. Baca baca. Menulis. Menjalankan.

Bukan wxr atau xrw atau rxw. Ini hanya salah satu dari 3! berbagai cara menyusun huruf. Ini mungkin menghilangkan beberapa kebingungan Anda.

Juga, chmod bekerja dengan sistem posisi oktal. Oktal mewakili basis 8. Yang dapat direpresentasikan menggunakan 1 bit dengan nilai dari 0-7 inklusif - total 8, itulah mengapa disebut oktal.

Tidak terkait: Dalam biner, tiga 1 mewakili 1 + 2 + 4 = 7.

Anggap saja seperti ini.

rwx
111  where 111 is a binary value

1 1 1      (binary)
4+2+1 -> 7 (decimal by position)

So 751 would be rwx r_x __x
                111 101 001 (binary)
                421 401 001 (decimal by position)
                 7   5   1  (decimal equivalent)

Saya juga mengalami kebingungan ini. Tanda hubung adalah bagian yang membingungkan. Orang akan berasumsi bahwa tanda hubung digunakan untuk memisahkan kelompok perizinan (7-5-1), tetapi tanda hubung tidak digunakan dengan cara itu dalam kasus ini; tidak ada spasi antara grup izin dan tanda hubung digunakan di 'tengah' grup izin, sehingga bisa dikatakan, sebagai placeholder untuk tidak ada.

Singkatnya, tanda hubung tidak digunakan untuk memisahkan grup izin.

Seperti yang Octavian katakan di atas, tanda hubung tidak digunakan untuk memisahkan pengguna / grup / semua tetapi digunakan untuk 0 izin. Dalam kasus Anda, jika Anda ingin melihat pemisahan pengguna / grup / semua akan terlihat seperti ini:

755 would be drwxr-xr-x where you can separate like this: drwx|r-x|r-x
751 would be drwxr-x--x where you can separate like this: drwx|r-x|--x

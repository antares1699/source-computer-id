Pulihkan bookmark yang hilang oleh FireFox Sync

Saya pikir sinkronisasi FireFox mungkin telah menghapus semua bookmark saya. Saya menginstalnya di komputer baru dan memilih opsi "Ganti semua data di komputer ini dengan data sinkronisasi saya". Namun, tidak ada bookmark yang muncul, membuat saya bertanya-tanya apakah yang terjadi sebaliknya dan menghapus bookmark saya di server.

Saat ini, saya berasumsi bahwa bookmark saya masih tersimpan di komputer saya yang lain. Adakah cara agar saya dapat membuat cadangan sebelum Sync menghapusnya juga?

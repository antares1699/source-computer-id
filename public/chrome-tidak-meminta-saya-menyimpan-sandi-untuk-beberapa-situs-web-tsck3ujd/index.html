Chrome tidak meminta saya menyimpan sandi untuk beberapa situs web

Mengapa Chrome meminta untuk menyimpan beberapa sandi situs web, tetapi tidak untuk situs web lain?

Terkadang, ini karena beberapa situs menggunakan pengalihan setelah login. Halaman tempat Anda login menghilang begitu cepat, sehingga dialog "simpan sandi" tidak memiliki waktu untuk muncul.

Terkadang, menonaktifkan skrip dapat membantu (jika situs dialihkan dengan bantuan JavaScript), tetapi tidak di setiap situs.

Beberapa situs web memiliki autocomplete=”off” yang meminta browser untuk tidak menawarkan untuk menyimpan kata sandi. Begini cara kerjanya di Firefox, mungkin sama dengan Chrome. Anda dapat melihat kode sumber halaman web dan memeriksa apakah opsi itu ada.

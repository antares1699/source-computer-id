Saya mendapatkan bash: kesalahan sintaks dekat token yang tidak diharapkan '('

Saya sedang menyimpan skor tinggi untuk papan Game Elf JAMMA saya (412-in-1). Saya sedang mengikuti tutorial ini . Saya mencoba menjalankan perintah ini

mv hiscore(pre_mame0133u1).dat /mnt/three/usr/local/share/xmame/hiscore.dat

tapi seperti yang bisa Anda lihat tangkapan layar saya , itu mengembalikan kesalahan

bash: kesalahan sintaks dekat token yang tidak terduga '('

bash: syntax error near unexpected token '('

Kamu butuh melarikan diri tanda kurung:

mv hiscore\(pre_mame0133u1\).dat /mnt/three/usr/local/share/xmame/hiscore.dat

catatan:

Untuk referensi di masa mendatang, Anda dapat menggunakan ShellCheck untuk menemukan bug di kode bash Anda. Memasukkan skrip yang tidak dikoreksi memberikan yang berikut:

$ shellcheck myscript

Line 1:
mv hiscore(pre_mame0133u1).dat /mnt/three/usr/local/share/xmame/hiscore.dat
^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.
          ^-- SC1036: '(' is invalid here. Did you forget to escape it?
          ^-- SC1088: Parsing stopped here. Invalid use of parentheses?

Memperbaiki kesalahan pertama:

$ shellcheck myscript

Line 1:
mv hiscore\(pre_mame0133u1).dat /mnt/three/usr/local/share/xmame/hiscore.dat
^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.
                          ^-- SC1089: Parsing stopped here. Is this keyword correctly matched up?

Dan mengoreksi kesalahan kedua:

$ shellcheck myscript

Line 1:
mv hiscore\(pre_mame0133u1\).dat /mnt/three/usr/local/share/xmame/hiscore.dat
^-- SC2148: Tips depend on target shell and yours is unknown. Add a shebang.

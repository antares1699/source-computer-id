Jika Anda menjalankan Arch, Anda ingin menginstal bluez dan bluez-utils. Keduanya tersedia dari repositori resmi. Jadi jalankan saja:

sudo pacman -S bluez bluez-utils

Kemudian mulai layanan bluetooth dengan:

sudo systemctl start bluetooth

Dan jika Anda juga ingin bluetooth dimulai saat boot:

sudo systemctl enable bluetooth

Ikuti instruksi lengkap di Arch wiki sini . Untuk bantuan menggunakan perintah bluetoothctl.

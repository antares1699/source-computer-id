Bagaimana cara membuat ulang sinergi applescript / quicksilver OS X di Ubuntu?

Saya baru-baru ini mengonversi OS X-ke-Ubuntu, tetapi satu fitur yang sangat saya lewatkan adalah kombo quicksilver / applescript. Singkatnya: Saya akan menggunakan applescript untuk menulis skrip fungsional yang sangat sederhana (misalnya, "ubah ukuran jendela saat ini untuk mengambil tepat separuh kiri layar", atau "tambahkan tanggal sekarang ke clipboard dalam format tertentu") quicksilver untuk ditetapkan ke hotkey (misalnya Ctrl-Alt-j untuk yang pertama).

Saya ingin membuat ulang ini di ubuntu; di mana saya memulai?

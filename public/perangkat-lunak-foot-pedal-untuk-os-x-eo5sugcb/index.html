Konfigurasi yang saya aktifkan untuk memetakan tombol 2 dan 3 ke CMD + C dan tombol 1 sampai CMD + V. untuk membuat separuh kiri melakukan perintah Salin dan perintah Tempel kanan terlihat seperti ini ...

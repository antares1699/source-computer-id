Mengapa hard disk internal baru saya tidak muncul setelah menginstal ulang Windows 7?

Saya memiliki desktop dengan motherboard Intel yang menjalankan Windows 7. Beberapa hari yang lalu saya membeli 2 nd hard disk internal (SATA Seagate 500GB). Saya menginstalnya dan itu berjalan normal. Saya menyalin beberapa data saya yang sangat penting pada hard disk baru itu dari hard disk internal lama saya yang datanya telah dihapus dari hard disk lama.

Sekarang untuk beberapa masalah OS, saya harus menginstal ulang Windows 7 di hard disk lama seperti sebelumnya. Hard disk baru tidak lagi ditampilkan di Komputer Saya. Bagaimana cara memulihkan data dari HDD baru?

6

Anda perlu menetapkan (memasang) drive, dengan menetapkan huruf drive. Buka konsol Manajemen Komputer (Mulai -> Jalankan -> compmgmt.msc), dan di bawah Penyimpanan, buka Manajemen Disk. Klik kanan pada drive / partisi yang ingin Anda pasang, dan klik Change Drive Letter and Paths.

Klik tombol Add untuk menambahkan titik pemasangan baru, lalu Anda dapat memasangnya sebagai folder atau drive baru (Anda dapat menentukan huruf drive mana yang harus digunakan).

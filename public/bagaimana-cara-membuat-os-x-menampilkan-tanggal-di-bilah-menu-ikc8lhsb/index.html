Bagaimana cara membuat OS X menampilkan tanggal di bilah menu?

Saya ingin tanggal hari ini ditampilkan di bilah menu di samping hari dalam seminggu dan waktu (yaitu Sen, 31 Des, 21.59). Apakah ada cara untuk melakukan ini? Saya menggunakan OS X 10.5.

EDIT: Saya menemukan satu saran , tapi tidak bisa membuatnya berhasil ...

Kamu bisa gunakan Gaib , atau lakukan sedikit konfigurasi Preferensi Sistem .

  • Di System Preferences, buka Internasional dan buka tab Format.
  • Di samping Tanggal, klik tombol Kustomisasi. Atur tanggal Anda dalam format yang Anda inginkan untuk muncul di bilah menu Anda. Gunakan panah bawah pada setiap elemen untuk memilih di antara format (Januari, Jan, 01, 1, dll.):

  • Sekarang, pilih seluruh format tanggal dan salin ke clipboard Anda (Cmd + A, Cmd + C). Tekan OK.
  • Tekan tombol Kustomisasi di sebelah Times. Pilih format "Sedang".
  • Tempel format tanggal di clipboard Anda di sebelah format waktu, dan tekan OK. Kamu sudah selesai!

Perlu memulihkan data dari laptop sebelum mengirimnya untuk diperbaiki

Inilah masalahnya: Air tumpah di laptop saya dan setelah mencoba semua yang kami bisa untuk mengeringkannya, laptop tetap tidak mau hidup. Menghubungi dukungan Toshiba dan mereka akan memeriksanya, namun, kami diperingatkan tentang kemungkinan kehilangan data jika mereka memperbaikinya. Bagaimana saya bisa menghapus data tanpa melakukan sesuatu untuk membatalkan garansi (perbaikan ini mungkin tercakup dalam garansi mereka)?

Laptop ini memiliki port e-SATA dan port HDMI. Saya ingin tahu apakah mungkin menggunakan ini untuk menghapus data kami sebelum saya membeli kabel untuk melakukannya. Terima kasih sebelumnya.

Hal terbaik yang harus dilakukan adalah melepas HDD dan menghubungkannya ke HDD Caddy. Kemudian Anda bisa mendapatkan data dari HDD seolah-olah itu adalah drive yang dapat dilepas di PC lain.

Cara lain untuk melakukan ini tanpa membeli peralatan adalah menginstal HDD di komputer lain dan mencadangkan file dengan cara itu. Cara termudah untuk melakukannya adalah sebagai HDD kedua di PC untuk mengatasi masalah driver. Ini Panduan ini cukup lengkap untuk meletakkan HDD kedua ke komputer lain. Tentu saja jika Anda tidak memiliki kabel SATA cadangan untuk meletakkan HDD di mesin, Anda juga akan mengalami masalah di sini.

Drive Caddy di bawah £ 5, untuk menjamin data Anda baik-baik saja, itu IMHO harga murah.

(Alasan mereka mengatakan hal semacam itu adalah jika ternyata itu masalah perangkat lunak, resolusi mereka adalah menghapus drive dan menginstal ulang Windows, tetapi jelas ini bukan masalah perangkat lunak ...)

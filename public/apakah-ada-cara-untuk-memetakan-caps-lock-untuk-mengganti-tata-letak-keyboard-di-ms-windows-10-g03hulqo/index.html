Pertama, Anda dapat membaca info tentang cara mengatur pintasan default: Pintasan untuk mengganti tata letak keyboard

Di windows 10 ada fitur baru: Anda bisa beralih dengan Menang + Ruang (coba saja dan lihat)

Berikut ini pendekatan dengan Capslock untuk Windows 7: Capslock untuk mengganti tata letak
Skrip AHK serupa juga berfungsi pada Windows 10.

Untuk meringkas pendekatan Autohotkey yang diuji dan berfungsi:

Opsi 1. Ikat Capslock untuk mensimulasikan Alt + Shift

Pertama, pastikan itu Alt + Bergeser adalah kombinasi tombol default. Gunakan skrip AHK ini:

capslock::
    send {Lalt down}{Shift}{Lalt up}
return

Opsi 2. Ikat Capslock ke kombo Win + Space baru

Saya lebih suka ini karena memberikan umpan balik visual yang baik saat beralih.

Berikut script untuk mengikatnya ke Capslock:

sel := 0

#if (sel=0)
capslock::
    send {lwin down}{Space}
    sel := 1
return 
#if

capslock up::
    send {lwin up}
    sel := 0
return

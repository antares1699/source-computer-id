Mengapa saya tidak dapat menggunakan VPN saat saya terhubung melalui broadband seluler?

Saya memiliki stik data HUAWEI (tidak terkunci) yang saya gunakan untuk menyambungkan laptop saya ke penyedia internet seluler UE. Karena saya yakin akan lebih aman, saya berniat menggunakan VPN

Jadi, saya menyiapkan VPN yang biasanya saya gunakan di laptop saya hanya untuk mengetahui bahwa itu tidak berfungsi. Ia mencoba untuk membuka port, tapi kemudian hang dan gagal. Apakah saya memerlukan VPN khusus untuk penyiapan ini?

Apa saran Anda tentang cara melindungi diri Anda sendiri saat menggunakan koneksi broadband seluler? Apakah VPN diperlukan?

Jika Anda menggunakan VPN PPTP, kemungkinan besar penyedia broadband seluler Anda memblokir port PPTP standar 1723, dan tidak terlalu berlebihan untuk berasumsi bahwa mereka juga akan memblokir port VPN IPSec.

Terkadang penyedia semacam itu menawarkan layanan "kelas bisnis" yang lebih mahal yang memungkinkan lalu lintas seperti itu.

Beberapa solusi:

  • Temukan penyedia VPN yang menawarkan layanan di port alternatif
  • Apakah ponsel Anda ponsel Android yang dapat menerima ROM khusus? Banyak ROM khusus memiliki dukungan OpenVPN bawaan dan Anda dapat mencoba penyedia OpenVPN.
  • Jika ponsel Anda adalah ponsel Android, Anda dapat mencoba menginstal OrBot dan OrWeb - ini akan mengaktifkan penggunaan Tor di ponsel Anda dan mungkin memberikan perlindungan serupa.
  • Jika Anda memiliki sistem Linux di rumah, Anda dapat mengatur server PPTP / IPSec / OpenVPN Anda sendiri yang mendengarkan pada port nonstandar dan lalu lintas VPN melalui koneksi broadband rumah Anda. Bahkan mungkin saja sistem Linux tersebut menggunakan layanan VPN lain karena itu adalah koneksi Internet "keluar" (tetapi ini bukan pengaturan yang sederhana).

Saya sarankan Anda menggunakan OpenVPN, bukan PPTP.

Ini lebih kuat dan ketika menggunakan protokol VPN-nya sendiri juga memiliki fitur penjelajahan NAT, jadi Anda dapat menggunakannya di belakang penyedia NAT.

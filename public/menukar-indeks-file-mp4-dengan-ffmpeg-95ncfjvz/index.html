Menukar indeks file MP4 dengan FFMpeg

Saya menjalankan aplikasi web yang menyediakan beberapa video di dalamnya. Video tersebut sedang ditampilkan dengan flash-video-player.

Sayangnya, indeks file kontainer tersebut berada di akhir file, jadi seluruh video 2GB harus dimuat hingga pemutaran dapat dimulai.

T: Apakah ada cara dengan FFMpeg untuk menukar MP4-Container-index ke depan file?

Anda ingin memindahkan atom moov ke awal. Saat encoding, Anda dapat menggunakan:

ffmpeg -i inputfile [parameter lain] -movflags + faststart outputfile.mp4

Jika sudah dienkode, Anda cukup menyalin streaming tanpa encoding ulang:

ffmpeg -i input.mp4 -codec copy -map 0 -movflags + faststart output.mp4

Anda juga dapat menggunakan alat qt-fasstart. Ini perlu dikompilasi di direktori alat ffmpeg. Perintahnya sederhana:

qt-faststart inputfile.mp4 outputfile.mp4

Ada binari windows yang tersedia.

Bisakah saya mendapatkan pemindai Canon D1250D2 untuk bekerja dengan Windows 7?

Setelah baru-baru ini memutakhirkan dari Windows XP ke Windows 7 Canon D1250D2 saya tidak lagi berfungsi karena tidak ada driver Windows 7 untuk itu. Selain itu, itu dalam urutan kerja yang sempurna.

Sebelum saya menghabiskan uang hasil jerih payah saya untuk pemindai baru, adakah yang dapat saya lakukan untuk membuatnya berfungsi? Apakah ada driver pihak ketiga yang akan berfungsi? Bisakah saya membuatnya bekerja melalui mesin virtual?

Sistem operasi virtual akan selesaikan masalah ini. Anda hanya memerlukan platform virtualisasi yang memungkinkan Anda mengalihkan perangkat USB dari mesin fisik ke mesin virtual.

VMware Workstation 7 akan melakukan ini, misalnya. Ini akan mencantumkan perangkat USB Anda di Windows 7, Pemindai Canon Anda akan ada dalam daftar itu. Pilih dan beri tahu VMware untuk mengarahkan perangkat USB itu ke mesin virtual. Jika Anda memiliki driver yang berfungsi untuk Windows XP, jalankan saja mesin virtual Windows XP. Pemasangan driver akan berjalan persis seperti yang mereka lakukan pada mesin 'nyata'.

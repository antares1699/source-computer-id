Di manakah letak sgen.exe? (Yang digunakan oleh Visual Studio)

Visual Studio 2015 saya menggunakan " sgen.exe ".

Saya tidak menemukannya di mana pun di "C: \ Program Files (x86) \ Microsoft Visual Studio 14.0"

Darimana ini diambil?

Saya perlu tahu ini karena Studio visual tidak mengungkapkan detail kesalahan "SGEN exited with code 1". Saya ingin melihat detail ini untuk mengetahui alasan sebenarnya dari SGEN kesalahan.

Bos saya menyarankan saya untuk lari SGEN manual, karena dalam kasus ini SGEN memiliki beberapa keluaran, misalnya, "... Cannot deserialize type 'TypeName' because...". Dan keluaran ini persis seperti yang saya butuhkan.

Terima kasih!

PS: Memindahkan pertanyaan ini dari SO karena komentar: "Ini sebenarnya bukan pertanyaan pemrograman karena Anda dapat dengan mudah mengajukan pertanyaan yang sama tentang msword.exe - mencari file di hard disk Anda mungkin harus ditanyakan di SuperUser."

Memang, jawabannya langsung setelah mencari di hard disk.

Itu tergantung proyek .Bersih Versi: kapan.

  • .Net 4.6.1: C:\Program Files (x86)\Microsoft SDKs\Windows\v10.0A\bin\NETFX 4.6.1 Tools
  • Bersih 4.6: C:\Program Files (x86)\Microsoft SDKs\Windows\v10.0A\bin\NETFX 4.6 Tools
  • .Net 4.5.1: C:\Program Files (x86)\Microsoft SDKs\Windows\v8.1A\bin\NETFX 4.5.1 Tools
  • .Net 4.0: C:\Program Files (x86)\Microsoft SDKs\Windows\v8.0A\bin\NETFX 4.0 Tools
  • Versi yang lebih rendah: lihat folder lain di Program Files (x86)\Microsoft SDKs\Windows satu.

Ini membantu saya men-debug, dan, karenanya, memprogram. Jadi, mungkin, pertanyaan ini terkait dengan pemrograman. Namun, karena fakta menerima suara negatif, memindahkan pertanyaan dan jawaban ke Pengguna Super.

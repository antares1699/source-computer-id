Apa itu winlogon.exe -SpecialSession?

Saya menjalankan Windows 8 Enterprise. Beberapa kali sekarang, saya memperhatikan beberapa winlogon.exe proses berjalan (disertai dengan csrss.exe dan dwm.exe tambahan). Hari ini saya melihat satu ekstra, tetapi pertama kali saya menyadarinya, ada lima atau enam tambahan.

Process Explorer menunjukkan bahwa semuanya memang C:\Windows\System32\winlogon.exe, tetapi yang tambahan telah dimulai dengan bendera -SpecialSession, dan bahwa proses induknya tidak ada lagi.

Pemindaian malware dengan Malwarebytes dan Spybot S & D belum menunjukkan apa-apa, dan Windows Defender tetap diam.

Apa opsi -SpecialSession ini dan dari mana contoh tambahan ini bisa berasal? Google sepertinya tidak tahu apa-apa tentang itu.

Saya juga melihat perilaku ini sejak saya mengupgrade ke Windows 8. Saya melakukan upgrade di tempat melalui windows 7, FWIW. Saya tidak memiliki cukup poin untuk dikomentari, jadi saya menulis ini sebagai jawaban.

Ada pola yang jelas bahwa setiap set winlogon.exe yatim piatu dan proses turunannya kemungkinan besar merupakan artefak dari fitur "Fast boot / hybrid sleep" baru pada windows 8. Secara default, Windows 8 tidak mati sepenuhnya. Ketika Anda memilih "Shutdown" dari bilah pesona, yang sebenarnya terjadi adalah Anda keluar, tetapi alih-alih mematikan, Windows akan hibernasi. Dengan cara ini ketika Anda menyalakannya kembali, itu melanjutkan dari hibernasi pada prompt login Anda, bukan boot penuh.

Saat ini rangkaian proses tertua saya berumur 12 hari, tetapi saya mematikan komputer saya setiap hari. Saya memiliki sekitar 12 grup proses ini.

Saya belum melihat efek samping yang merugikan, tetapi saya akan mencoba menonaktifkan "startup cepat". Saya tidak yakin bahwa ini lebih cepat.

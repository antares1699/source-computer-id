Tidak sepenuhnya benar bahwa Anda tidak dapat mengirim SIGTERM dari baris perintah. Anda tidak dapat mengirimnya dari pintasan keyboard , tapi kamu bisa kirimkan dari baris perintah.

Berdasarkan halaman manual untuk membunuh , Anda dapat mengirim SIGTERM ke proses apa pun. Anda akan melakukannya dengan menemukan proses Anda di tabel proses (ketik ps) dan kemudian ketik kill -15 [pid].

Berikut adalah daftar pintasan keyboard yang dapat Anda gunakan di terminal untuk menangani proses:

  • Ctrl + C mengirimkan SIGINT yang meminta untuk menghentikan program tetapi dapat ditangkap atau diabaikan.
  • Ctrl + Z mengirimkan SIGTSTP yang meminta untuk menjeda program tetapi dapat ditangkap atau diabaikan. Proses ini bisa dilanjutkan nanti.
  • Ctrl + \ mengirimkan SIGQUIT yang sama dengan SIGINT kecuali itu juga menghasilkan dump inti.

Apakah ada font siap unicode 6.x?

Saya masih mencari font TTF / OTF dengan dukungan Unicode 5.x atau 6.x yang hampir penuh, yaitu saya tertarik dengan Transport and Map Symbols dari Unicode 6.2 jarak 1F680-1F6FF . Apakah font ini ada?

Tidak, secara teknis tidak mungkin untuk mencakup sekitar 100.000 karakter ketika tekonologi font menetapkan batas atas 65.536 mesin terbang.

Untuk memeriksa dukungan untuk beberapa karakter tertentu dalam font, titik awal terbaik biasanya adalah fileformat.info. Tidak sempurna, tapi sangat berguna. Misalnya, informasinya tentang mendukung U + 1F680 adalah bahwa itu didukung oleh Symbola font. (Font LastResort yang disebutkan dalam daftar bukanlah font yang sebenarnya.) Namun, ini juga didukung oleh Segoe UI Symbol. (Saya menemukan ini menggunakan utilitas hanya untuk IE saya yang sederhana yang memeriksa font di komputer saya.) Tampaknya dikirimkan dengan Windows 7, tetapi file Halaman MS tentang itu menunjuk ke halaman pembelian yang sebenarnya tidak menjualnya.

Selain dukungan font, Anda mungkin mengalami masalah perangkat lunak lainnya. Banyak program tidak dapat menangani karakter yang melewati BMP (nomor Unicode 0 hingga FFFF hex.) Sama sekali, atau memerlukan tindakan ekstra untuk mengatasinya; bahkan mengetiknya mungkin menjadi masalah.

Solusi yang diberikan Terobosan tidak berguna jika Anda memiliki akun terbatas. Inilah yang Anda dapatkan untuk mendapatkan hak istimewa admin: -

Langkah pertama: Pertama, buka Command Prompt (baik; Mulai-> Jalankan-> Ketik cmd.exe-> OK ATAU; Klik dua kali C: \ Windows \ System32 \ cmd.exe

Langkah Kedua: Buat akun, jika Anda tidak ingin membuat akun baru, abaikan langkah ini. Ketik yang berikut ini ke dalam kotak hitam yang muncul (prompt perintah)

pengguna bersih / tambahkan

Contoh: pengguna bersih maks 123456 / tambahkan

Ini akan membuat akun yang dilindungi kata sandi. Nama pengguna akan menjadi maks Kata sandi akan menjadi 123456

Langkah ketiga: Temukan semua grup pengguna yang tersedia di jaringan

grup lokal bersih

Ini akan memunculkan daftar seperti ini * Administrator * Pengguna Debugger * Tamu * HelpServicesGroup * Pengguna Perintah berhasil diselesaikan. (bisa berbeda menurut sistem yang berbeda)

Langkah Keempat: Masukkan perintah berikut ke Command Prompt

localgroup bersih / tambahkan

Contoh:

administrator grup lokal bersih maks / tambahkan

Ini akan menambahkan nama pengguna "maks" bergabung dengan grup "administrator" dan memiliki semua hak istimewa yang terkait dengan grup itu.

Ini akan bekerja pada mesin lokal. dan jika Anda cukup beruntung dan admin tidak terlalu pandai dalam apa yang dia lakukan, Anda dapat membuat akun admin Anda sendiri di lingkungan jaringan juga :) :) tetapi jika prompt perintah cmd dibatasi atau hak akses manual ditempatkan maka tidak ada cara untuk membuat ini berfungsi .. Anda mungkin akan mendapat masalah jika ketahuan melakukan ini .. Saya hanya memberikan ini sebagai contoh tentang apa yang dapat Anda lakukan dengan command prompt dan oleh karena itu mengapa itu harus selalu dinonaktifkan pada akun terbatas , Saya tidak bertanggung jawab atas apa pun yang Anda lakukan dengan ini. perhatikan juga bahwa ini hanya berfungsi di rumah atau di jaringan yang tidak dikelola.

Selamat bersenang-senang

Namun, jika Anda telah memangkas instalasi Windows Anda, program tersebut mungkin tidak tersedia (ini biasanya hanya terjadi jika Anda menggunakan nLite / vLite / RT7Lite dan menghapus sesuatu yang seharusnya tidak Anda miliki).

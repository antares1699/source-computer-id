Dapatkah saya melarang aplikasi mengeluarkan suara di Windows?

Saya terpaksa menggunakan Lotus Notes di tempat kerja. Saya dapat menonaktifkan sebagian besar suara yang dihasilkannya melalui manajemen preferensi yang sangat buruk, tetapi ada beberapa suara yang tampaknya tidak mungkin dinonaktifkan.

Itu atau mereka dikuburkan. Karena fakta bahwa saya tidak dapat menemukan opsi global "Lotus tidak mengintip" di mana pun, saya pikir saya terpaksa melakukan tindakan yang lebih drastis.

Apakah ada cara untuk memberi tahu Windows 7 agar tidak mengizinkan akses aplikasi ke subsistem suara? Sebaliknya, secara teknis Notes adalah aplikasi java, jadi mungkin saya bisa memaksa JVM yang digunakannya untuk mengirim semua suara ke void.

Atau apakah saya hanya harus menderita dengan kekejian klien email?

(Catatan, setiap 'acara' di Lotus disetel untuk tidak memutar suara, jadi ada beberapa acara yang memutar suara yang bukan acara. Atau semacamnya.)

Saya tidak tahu bahwa setiap aplikasi di akan muncul di daftar itu seperti di jawaban lainnya. Jika tidak, buat file gelombang yang sangat pendek, tetapi diam (Anda mungkin juga dapat mengunduhnya dari suatu tempat). Cobalah untuk mencari tahu di mana lokasi file tersebut, yang membuat suara yang tidak Anda sukai, dan ganti file yang tidak diinginkan (ganti namanya .old) dengan salinan yang diubah namanya dari file gelombang diam Anda.

Mungkin membutuhkan sedikit perburuan, tetapi Anda harus dapat menemukannya. Program yang lebih lama seperti ini biasanya memiliki file gelombang sendiri di sub-folder instalnya sendiri.

Tentu saja, pastikan tidak hanya di applet Panel Kontrol Suara, di mana Anda dapat menonaktifkannya. Saya meletakkan ini yang terakhir karena kedengarannya Anda mungkin sudah memeriksanya di sana.

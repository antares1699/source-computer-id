Anda harus keluar dari karakter garis miring terbalik agar berfungsi:

cd abcd\\fff\\ggg\\ddd/

Melarikan diri berarti mengawali garis miring terbalik (juga harus melakukan ini jika direktori berisi spasi) dengan, lucunya, garis miring terbalik. Linux mengartikan satu garis miring terbalik sebagai karakter pelarian, memungkinkan Anda untuk mengakses direktori dan file yang berisi spasi, garis miring terbalik, dan karakter khusus lainnya.

Itu adalah praktik terbaik; Sebaiknya biasakan untuk memahami bagaimana dan kapan menggunakan karakter escape jika Anda akan menggunakan baris perintah Linux. Jawaban lain disediakan:

cd 'abcd\fff\ggg\ddd'

juga akan bekerja. Saya memberikan jawaban saya sebagai jawaban yang lebih lengkap dan 'benar', karena memahami konsep karakter melarikan diri sangat penting untuk interaksi CLI yang tepat.

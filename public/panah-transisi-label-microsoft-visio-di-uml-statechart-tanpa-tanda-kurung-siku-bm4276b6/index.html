Panah transisi Label Microsoft Visio di UML Statechart (tanpa tanda kurung siku)

Saya tidak tahu bagaimana memberi label transisi (panah) di Statechart UML. Dengan sebagian besar item, saya baru mulai mengetik, tetapi dengan transisi dikatakan:

Perlindungan bentuk dan / atau properti lapisan mencegah eksekusi lengkap dari perintah ini.

Saya menemukan Format -> Protection dan tidak mencentang kotak "Text", dan sekarang saya dapat mulai mengetik, tetapi teks tersebut akan hilang begitu saya selesai. Ada apa dengan itu?

Satu hal yang berhasil adalah "Ekspresi penjaga", tapi saya rasa bukan itu yang saya inginkan. Tidak seperti contoh MSDN ini , ada tanda kurung siku yang aneh di sekelilingnya. Adakah cara untuk mendapatkan label transisi yang tampak normal?

Klik dua kali pada transisi untuk memunculkan dialog properti transisi. Buat Acara (tombol acara), Buat Acara baru. Kembali ke dialog Properti Transisi UML pilih acara baru Anda dari acara: drop-down. sekarang akan muncul sebagai label. Sakit bukan

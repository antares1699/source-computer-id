Bagaimana cara membuat Chrome memblokir semua pop-up

Saya harus menyarankan Anda menggunakan Pemblokir Popup (ketat) Ekstensi Chrome karena Anda bertanya secara khusus " Bagaimana cara membuat Chrome memblokir semua pop-up "dan di bawah ini adalah petunjuk khusus tentang mengonfigurasi ini untuk mewujudkannya dan bahkan tidak pernah meminta penggantian di tempat pertama dari hasil pengujian yang saya simpulkan untuk saya.

Terlepas dari pengawasan apa pun yang mungkin ditimbulkan oleh jawaban ini dari orang lain, cobalah sederhana dan konfigurasikan opsi seperti yang diuraikan di bawah ini dan Anda akan melihatnya berfungsi seperti yang diharapkan seperti yang saya lakukan. Saya juga akan memberikan metode agar Anda dapat menguji dengan mudah sebelum dan sesudah di bagian bawah petunjuk konfigurasi terperinci.

Petunjuk Konfigurasi Terperinci

  1. Pasang nilai bintang 4,5 Pemblokir Popup (ketat) Ekstensi Chrome

    Blokir dengan ketat semua permintaan popup dari situs web mana pun

    Supported blocking methods:
 1. window.open

  2. Di bilah alamat Chrome, tempelkan chrome://extensions/ dan tekan Memasukkan

  3. Pastikan sudah diputar Di dan kemudian klik Detail tombol

  4. Klik pada Opsi ekstensi ikon konfigurasi

  5. Anda akan ingin memastikan pengaturan ini ditetapkan dengan nilai-nilai ini — dan pastikan untuk Save :

    • Jumlah maksimum permintaan popup untuk satu halaman: 1
    • Batas waktu untuk menampilkan setiap popup (dalam detik): 1
    • Hitung mundur untuk melakukan tindakan terakhir (dalam detik): 1
    • Tindakan default saat popup diminta: deny request
    • Tampilkan jumlah total permintaan yang diblokir di ikon toolbar : unchecked
    • Cegah pengalihan halaman selama 2 detik setelah popup diminta : checked

    • Abaikan pemberitahuan popup jika memungkinkan (tidak disarankan) : checked

    • Tujuan daftar putih: Blank/Delete All

    • Sumber daftar putih: Blank/Delete All
    • Menyimpan ( Sangat penting )

Pengujian Sebelum dan Sesudah

  • Uji pemblokir popup Anda
  • Uji perangkat lunak pemblokir popup Anda

Untuk melakukan lebih baik daripada pemblokir pop-up yang ada di dalam Chrome, Anda perlu menggunakan ekstensi pihak ketiga.

Ada beberapa ekstensi yang tersedia. Saya telah membuat daftar beberapa dari mereka di bawah ini dalam urutan catatan mereka di toko Chrome.

  • uBlock Origin
    Tercatat bintang 5 oleh 19197 pengguna. Ekstensi ini paling direkomendasikan.

  • Pemblokir Popup Pro
    Tercatat bintang 4 oleh 6984 pengguna.

  • Keamanan JustBlock
    Tercatat 3,5 bintang hanya oleh 2060 pengguna, jadi yang paling tidak direkomendasikan.

  • ScriptSafe
    Meskipun ini bukan hanya pemblokir pop-up, saya juga merekomendasikan ekstensi ini yang dengannya seseorang dapat memblokir eksekusi JavaScript sumber tidak sah, sehingga dapat menonaktifkan skrip sebelum diluncurkan pop-up dengan cara tidak terjebak oleh pemblokir. Perpanjangan ini tidak sesuai dengan gaya kerja semua orang tetapi merek berselancar lebih aman.

Apakah ada add-on untuk Excel untuk memungkinkan saya mengurutkan secara sewenang-wenang kolom dengan satu klik?

Excel di sisi lain memaksa Anda untuk melalui Sort-> Custom Sort jendela untuk menyelesaikan pekerjaan yang sama jauh lebih efisien.

Sebenarnya, Anda dapat menyisipkan tabel seperti itu dengan data Anda di Excel 2007. Seharusnya tidak perlu add-on khusus untuk mendapatkan fungsionalitas ini dengan Excel 2007 (atau 2010 dalam hal ini juga).

  1. Pilih semua kolom dan baris Anda di lembar kerja Excel Anda, buka tab Insert, dan pilih opsi Table.
  2. Di jendela dialog Create Table yang muncul, pilih apakah kolom data Anda memiliki header atau tidak, dan tekan OK.
  3. Sekarang lembar kerja data Anda memiliki panah filter yang dapat dipilih di bagian atas setiap tajuk kolom yang akan digunakan untuk mengurutkan nilai setiap kolom.

TANPA KEPALA KOLOM (masih memiliki fungsi filter satu klik per kolom)

kamu bisa

  1. Letakkan kancing Anda sendiri di baris ekstra itu. Anda harus melakukannya untuk setiap kolom, tetapi efeknya adalah satu klik nyata. Lihat artikel Microsoft ini tentang cara melakukannya: https://support.office.com/en-in/article/Add-a-button-and-assign-a-macro-to-it-in-a-worksheet-d58edd7d-cb04-4964-bead-9c72c843a283

  2. Gunakan Filter Otomatis - mereka datang dengan 'Sortir': Pilih semua data Anda, di bawah Data, klik Filter. Anda akan mendapatkan segitiga kecil di baris atas di atas setiap kolom, yang memungkinkan Anda untuk mengurutkan (dan memfilter) dengan cepat (tetapi dengan dua klik).

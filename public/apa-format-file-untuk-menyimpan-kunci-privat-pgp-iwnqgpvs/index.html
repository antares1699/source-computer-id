Saya mencoba memahaminya dari RFC dan dari kode sumber (terutama dari gnupg dan gpgme) dan saya tidak mengerti cara penyimpanan data ketika seseorang mengekspor kunci rahasia.

Saya tahu seharusnya ada perbedaan antara penyimpanan RSA dan DSA / ElGamal, tetapi harus ada cara yang umum.

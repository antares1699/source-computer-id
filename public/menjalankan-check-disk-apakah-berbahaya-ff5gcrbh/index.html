Menjalankan Check-Disk (Apakah berbahaya)

Saya memiliki lap top yang seorang teman ingin saya perbaiki. Ini memberikan layar biru saat boot. Ketika saya mencari pesan kesalahan itu menunjukkan bahwa saya harus menjalankan disk pemeriksaan.

Apakah ini berbahaya? Haruskah saya mencoba untuk melepaskan barang dari hard drive terlebih dahulu?

Saya bertanya karena saya memiliki hard drive sendiri sehingga ketika saya menjalankan check disk itu menghapus sekumpulan "bad sector" dan saya kehilangan sebagian besar info di dalamnya (tetapi sudah menjadi buruk).

Dalam keadaan normal, ini sangat aman selama Anda menggunakan yang dirancang untuk versi Windows Anda.

Namun, ada kemungkinan bahwa ada file penting di cluster yang buruk, yang belum ditandai sebagai buruk, dan cluster tersebut mungkin bekerja sesekali. Dengan "memperbaiki" mereka, drive mengabaikan cluster tersebut, dan oleh karena itu file tersebut tidak dapat diakses.

Tapi kemudian Anda pergi dan mencoba memulihkan cluster yang buruk dan terkadang Anda bisa mendapatkannya kembali. Biasanya Anda akan mendapatkan file seperti "? Utoexec.bat" dan semacamnya; tidak yakin cara kerjanya baru-baru ini.

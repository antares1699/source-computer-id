Dapatkah administrator domain Windows melihat kata sandi pengguna?

Saya berada di domain perusahaan. Komputer saya memiliki Windows 7 dan Office 2007, jadi saya Tebak bahwa server menjalankan Windows Server 2008 atau yang serupa. Saya diizinkan untuk mengubah kata sandi komputer saya dan saya melakukannya. Pertanyaannya adalah:

  1. Dapatkah administrator domain melihat kata sandi saya?
  2. (Saya kira jawaban yang satu ini sama dengan yang sebelumnya, tapi ...) Dapatkah admin melihat folder terenkripsi menggunakan fitur Sistem File Enkripsi pra-bitlocker di Windows Explorer?

Pertama - Anda menjalankan windows 7 - tetapi itu tidak berarti DC 2008 (Pengontrol Domain). Bisa apa saja dari server windows 2000 hingga 2012R2! - mereka semua dapat mengelola domain Anda sampai batas tertentu! Saran bahwa Anda menggunakan EFS daripada bitlocker sebenarnya menyiratkan pengontrol domain yang sedikit lebih tua.

Admin domain tidak dapat melihat atau mengambil kata sandi, tetapi dapat mengatur yang baru dengan menggunakan konsol yang disebut "Pengguna Direktori Aktif dan Snap-in Komputer" atau Pusat Administratif AD .. mereka juga dapat menggunakan VBScript, Powershell atau nomor lain metode untuk menyetel kata sandi, tetapi tidak dapat mengungkapkannya setelah disetel!

Re: admin membaca folder terenkripsi Artikel Technet ini menyatakan bahwa jika "kunci agen pemulihan" telah ditetapkan - maka ya, mereka bisa, jika tidak - tidak, mereka tidak bisa.

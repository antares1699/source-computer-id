Boot USB dari paralel VM (trik dengan ISO / floppy)

Beberapa tahun yang lalu tidak mungkin untuk boot dari USB di setiap komputer (di mana BIOS tidak mendukung ini). Ada gambar ISO kecil yang memungkinkan untuk boot dari stik USB.

Sekarang kami memiliki situasi yang sama dengan perangkat lunak untuk mesin virtual seperti VMWare, Parallels Desktop dan ISO ini berguna lagi. Saya menggunakan Parallels Desktop 7 untuk Mac OS X yang tidak memiliki opsi untuk boot dari USB atau stik USB.

Saya memiliki pengikut dunia nyata situasi: Saya memiliki HDD dengan Linux diinstal. Ini tidak seperti Live CD, ini sudah saya konfigurasikan. Saya menggunakannya di komputer saya di tempat kerja untuk mem-boot lingkungan yang saya suka dan saya menggunakannya di rumah untuk memiliki lingkungan yang sama. Kadang-kadang saya bekerja dengan Mac OS X dan ingin mengunduh sesuatu untuk Linux saya (seperti pembaruan dan perangkat lunak baru) dan mengkonfigurasi sesuatu jika saya perlu.

Saya tidak bisa boot dari itu menggunakan Parallels Desktop atau VMWare karena BIOS di VM ini tidak mengenali USB saya atau tidak bisa dipasang sebelum saya menjalankan VM saya.

Di mana saya dapat menemukan ISO yang memungkinkan saya untuk boot dari stik USB?

Menggunakan Manajer boot Plop :

  1. Dari Unduh Halaman , unduh 5.0.13.zip
  2. Buka zip file
  3. Jadikan drive DVD / CD virtual VM Anda menggunakan plpbt.iso sebagai image
  4. Hubungkan drive USB Anda ke perangkat keras virtual VM
  5. Boot VM Anda dari ISO
  6. Di layar boot pilih Anda ingin boot dari USB
  7. Lanjutkan boot dari USB

Apakah ada distribusi linux yang mirip dengan RedHat 5 Enterprise?

Saya ingin menginstal server RedHat 5 Enterprise Edition di PC rumah saya. Saya akan menginstal server dan klien pada mesin virtual menggunakan kotak Virtual dan mempelajari struktur server-klien di rumah. Tapi RedHat EE tidak gratis. Sebagai gantinya, saya akan menginstal distribusi server linux yang mirip dengan RedHat 5 EE. Tapi saya tidak tahu mana yang paling mirip.

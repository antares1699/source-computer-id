Dekompresi lrzip di Windows

Saya telah terjadi lrzip sebagai alat kompresi, dibandingkan dengan gzip, kompresinya tidak masuk akal.

lrzip memampatkan file teks 300MB menjadi 7MB sedangkan gzip hanya berhasil membuatnya menjadi 100MB.

Saya telah mentransfer file dari mesin ubuntu ke Windows tetapi sekarang tidak yakin bagaimana saya mendekompresi file lrzip di Windows?

3 cara dimungkinkan:

  1. Kompilasi versi Windows Anda sendiri. Sumber lrzip adalah tersedia di GitHub.
  2. Gunakan sesuatu yang bisa ditemukan di Google. Saya menemukan versi lama , itu bahkan mungkin tidak berhasil. Dan: selalu berhati-hatilah saat mengunduh binari yang telah dikompilasi dari Internet. Beberapa di antaranya mungkin mengandung efek yang tidak diinginkan (baca: virus, atau sejenisnya).
  3. Gunakan format yang berbeda. Mereka mungkin bukan solusi optimal untuk masalah Anda, tetapi mereka masih akan mencapai arsip yang dikompresi sedikit lebih sedikit. Beberapa bahkan mungkin sama-sama dikompresi. 7-Zip adalah alat gratis (salah satu dari banyak. sekali lagi: google untuk opsi lain.) untuk Windows yang mendukung banyak format kompresi yang tersedia di Linux.

Cygwin hadir dengan lrzip versi Windows. Anda mungkin menginginkan versi 64-bit karena menurut manual lrzip, ia menggunakan pemetaan memori untuk akses file, yang berarti build 32-bit memiliki batasan yang parah pada ukuran file.

  1. Jalankan penginstal cygwin ( x64 terbaru )
  2. Klik Next beberapa kali
  3. Ketik "lrzip" dalam pencarian, kemudian luaskan Arsip dan klik "Lewati":

Langkah opsional 4: buka direktori bin penginstalan Anda, temukan lrzip.exe dan file * .dll, pindahkan ke lokasi pilihan Anda dan hapus Cygwin.

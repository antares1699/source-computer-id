Bagaimana saya bisa melihat proses apa yang memiliki kunci pada file teks?

Saya kesulitan mengedit file konfigurasi untuk server web. File tersebut adalah file teks, dan saya dapat membukanya di editor teks, tetapi tidak dapat menyimpannya. Ketika saya mencoba menyimpannya, dikatakan bahwa aplikasi lain sedang menggunakannya, tetapi saya belum memulai server web.

Adakah cara agar saya dapat melihat proses apa yang memiliki kunci pada file teks, atau mengapa saya tidak dapat menyimpan file? Saya masuk sebagai administrator, dan OS-nya adalah Windows Vista.

Menggunakan LockHunter . Ini mirip dengan Unlocker yang merupakan favorit saya untuk waktu yang lama, tetapi LockHunter bekerja untuk OS 32bit dan 64bit tidak seperti Unlocker yang hanya berfungsi pada 32bit.

Mulai 5/7/2010, versi 1.9.0, dukungan 64-bit penuh telah ditambahkan ke Unlocker.

Nah, Oracle VirtualBox memiliki versi gratis yang menjalankan Windows, Linux, dll. dan gratis untuk penggunaan non-komersial. Sangat mudah untuk memulainya juga.

PEMBARUAN 2014: Sayangnya, satu pengembang inti di tim VirtualBox memutuskan untuk mengacaukan kompatibilitas dan fungsionalitas dasar untuk mencoba mengamankan OS tamu dari malware di host (yang sama sekali tidak mungkin), dan sejak versi 4.3.14 itu tidak dapat digunakan (berfungsi sangat lambat, menyebabkan layar biru pada mesin host atau tidak berjalan sama sekali)

Gunakan VirtualBox 4.3.12 atau gunakan Hyper-V yang disertakan gratis dengan Windows 8 (Anda harus mengaktifkannya di kotak Fitur Windows).

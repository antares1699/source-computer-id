Cara tercepat untuk berbagi file dari shell

Saya menggunakan instalasi minimal Ubuntu (Tanpa pengelola jendela). Apa cara tercepat dan termudah untuk mengupload file di suatu tempat? Sesuatu seperti skrip ke pastebin akan berfungsi.

8

Ubuntu memiliki paket yang disebut "pastebinit", yaitu skrip yang meletakkan file di pastebin dan kemudian mencetak URL-nya.

Instal dengan:

$ sudo apt-get install pastebinit

Menggunakannya sangat mudah:

$ pastebinit your_file.txt
http://pastebin.com/whatever

mengonversi file avi ke mp4 di linux [duplikat]

Saya memiliki banyak avi file. Saya ingin mengonversinya ke format mp4 dengan tetap mempertahankan ukuran file yang kecil. Saya menggunakan linux. Bisakah seseorang menunjukkan langkah demi langkah bagaimana membuat konversi, sebaiknya menggunakan perangkat lunak gratis di linux?

Saya mencoba menggunakan encoder media dari adobe creative cloud, tetapi akhirnya membesar-besarkan ukuran file menjadi lima atau enam kali ukuran file asli avi.

Saya ingin menggunakan file mp4 di situs web menggunakan tag video html5 untuk memberikan kontrol pemutaran.

Anda dapat menggunakan ffmpeg. Periksa jawaban ini: Konversi AVI ke MP4 dengan kualitas yang sama

Selain itu, halaman ini juga bagus: http://andrebluehs.net/blog/converting-avi-to-mp4-with-ffmpeg/

ffmpeg -i input.avi -acodec libfaac -b:a 128k -vcodec mpeg4 -b:v 1200k -flags +aic+mv4 output.mp4

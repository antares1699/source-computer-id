Bagaimana cara memperbarui podcast di iPhone saya yang tidak terdaftar di direktori iTunes?

Saya mendengarkan sejumlah podcast yang tidak terdaftar di direktori iTunes (dan tidak akan pernah ada) - apakah mungkin memperbarui podcast ini dari iPhone untuk mendapatkan episode terbaru saat saya sedang bepergian?

Sekarang, satu hal yang menarik: Saya tahu saya dapat menggunakan aplikasi terpisah atau situs web seluler untuk mendengarkan podcast, tetapi saya ingin dapat melakukan ini sehingga saya masih dapat menggunakan perangkat lunak iPod bawaan untuk mendengarkan podcast.

Jika saat ini tidak memungkinkan, ada ide apakah secara teknis memungkinkan untuk membangun aplikasi dengan SDK baru yang akan memungkinkan untuk ini?

Memperbarui:
saya menemukan posting ini di atas StackOverflow yang menjelaskan API Akses Perpustakaan Musik SDK, dan sepertinya permintaan saya juga tidak dapat diselesaikan melalui aplikasi terpisah.

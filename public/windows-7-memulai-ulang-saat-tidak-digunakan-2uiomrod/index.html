Windows 7 memulai ulang saat tidak digunakan

Windows 7 saya hampir selalu restart ketika saya tetap diam selama ~ 20-30 menit. Itu terjadi secara acak sebelumnya, tetapi akhir-akhir ini, jika saya meninggalkan komputer, saya yakin komputer akan restart setelah 30 menit tersebut. Itu tidak pernah terjadi ketika saya bermain game atau bekerja, hanya ketika menganggur.

Ini adalah penginstalan baru Windows 7 64bit. Saya juga mengalami masalah saat menginstalnya, selalu macet saat menyelesaikan penginstalan dan saya harus menginstal ulang lagi. Akhirnya terinstal pada percobaan ke-3 atau ke-4 setelah saya menghapus semua partisi saya dan menambahkannya lagi.

Saya pikir itu mungkin masalah perangkat keras, tetapi suhu tampaknya baik-baik saja dan saya tidak tahu bagaimana melacak apa yang mungkin menyebabkannya.

Ada ide?

Saya menjalankan Windows 7 64bit pada:

  • Gigabyte EX58-UD4P
  • Intel (R) Core (TM) i7 CPU 920 @ 2.67GHz
  • NVIDIA GeForce GTX 260
  • 6GB RAM DDR3 1066Mhz
  • WDC WD1001FALS-00J7B0 1TB SATA II

Saya merasa sangat tidak enak mungkin ada sesuatu dengan HDD dan kompatibilitasnya dengan Windows 7 karena saya tidak mengalami masalah tersebut selama 1 tahun ketika saya menggunakan Vista.

Edit:

Saya memeriksa kesalahan kritis Peraga Peristiwa dari malam ini. PC dimulai ulang pertama kali pada pukul 11:12, kemudian pada pukul 3:06 dan sejak itu setiap ~ 20 menit hingga saya kembali lagi.

Pesan kesalahan adalah:

Sistem telah melakukan boot ulang tanpa menutup dengan bersih dulu. Ini kesalahan bisa terjadi jika sistem berhenti merespons, jatuh, atau hilang kekuatan tiba-tiba.

Saya memiliki masalah yang sama, dan saya memperbaikinya dengan melakukan langkah-langkah berikut:

  1. Buka "Control Panel" → "Power Options".
  2. Di sisi kiri layar, tertulis "Ubah saat komputer tidur". Klik itu.
  3. Klik "Ubah pengaturan daya lanjutan". Jendela "Power Options" kecil akan muncul.
  4. Perluas bagian "Hard disk" dengan mengklik + di sebelah kiri. Demikian pula, perluas "Matikan hard disk setelah".
  5. Setel nilai "On battery" dan "Plugged in" ke "Never" (dengan mengklik nilainya lalu mengklik panah bawah berulang kali).

VM dikonfigurasi untuk tidak menjalankan screen saver apa pun, tetapi ini tepat 20 menit.

Berikut beberapa perintah untuk mengetahui apa yang terjadi:

1) Jalankan powercfg -list untuk menunjukkan skema daya apa yang aktif. Salin GUID dari skema aktif (ditandai dengan awal)

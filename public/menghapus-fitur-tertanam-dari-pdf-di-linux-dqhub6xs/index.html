Menghapus fitur tertanam dari PDF di Linux

Saya mencari cara untuk menghapus fitur yang disematkan dari dokumen PDF. Saya baru saja membeli Kindle DX. Ketika saya mencoba membuka beberapa file PDF saya di perangkat, saya mendapatkan pesan yang mengatakan:

pdf ini tidak dapat dibuka karena fitur yang disematkan belum didukung dengan kindle

Saya mencari cara untuk menghapus penggunaan fitur-fitur ini dari dokumen sehingga saya dapat melihatnya di Kindle saya. Saya adalah pengguna Ubuntu Linux. (Saya mencoba membuka PDF di Document Viewer dan mencetaknya ke file PDF dengan membuka File-> Print. Ini membuatnya jadi saya bisa membukanya di Kindle DX, tetapi fontnya kacau dan sangat sulit dibaca. Ini tidak akan melakukannya.)

Itu akan berhasil.

Jawaban singkat @ barbaz berhasil untuk saya. Berikut detail bagaimana saya mencapai kesuksesan:

Saya mengunduh qpdf-2.2.4-bin-mingw.zip dari situs SourceForge dan mengekstrak folder utama di zip ke c:\qpdf di kotak Windows 7 saya. Saya membuka Command Prompt di direktori bin dan menjalankan:

qpdf --decrypt infile.pdf outfile.pdf

Infile.pdf saya lebih dari 4MB, penuh dengan format dan gambar yang keren. Properti keamanan dokumen menampilkan 'keamanan kata sandi', bukan untuk membuka tetapi untuk mengubah, berkomentar, mengekstraksi. Dokumen tidak akan terbuka di Kindle standar saya.

Outfile.pdf baru dibuka di Acrobat dan terlihat sama. Properti keamanan dokumen menampilkan 'No Security', dan yang lebih penting, PDF baru terbuka di Kindle saya setelah memasang kabel USB dan menyalin PDF ke direktori Kindle document.

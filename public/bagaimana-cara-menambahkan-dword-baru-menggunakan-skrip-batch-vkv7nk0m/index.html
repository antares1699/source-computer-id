Ini adalah skrip yang saya miliki tetapi tidak berfungsi.

REG ADD HKLM\Software\Microsoft\windows\CurrentVersion\Policies\system] [/v LocalAccountTokenFilterPolicy] [/t REG_DWORD] [/d 1][/f]

Itu [ dan ] tidak dimaksudkan sebagai bagian dari perintah (sepertinya Anda menyalin beberapa teks dari halaman yang menjelaskan perintah tersebut dan tidak menghapusnya). Karakter ini digunakan sebagai pembatas (pengelompokan karakter) saat menjelaskan cara menggunakan perintah reg.

Coba yang berikut ini:

REG ADD HKLM\Software\Microsoft\windows\CurrentVersion\Policies\system /v LocalAccountTokenFilterPolicy /t REG_DWORD /d 1 /f

Lihat REG.exe untuk informasi lebih lanjut tentang sintaks reg. Ini juga memiliki beberapa contoh.

Bacaan lebih lanjut

  • Indeks A-Z dari baris perintah CMD Windows - Referensi yang sangat baik untuk semua hal yang berhubungan dengan garis cmd Windows.
  • reg - Baca, Tetapkan atau Hapus kunci dan nilai registri, simpan dan pulihkan dari file .REG.

Bagaimana cara menyesuaikan GUI Firefox menggunakan CSS?

Ekstensi Firefox Stylish memungkinkan kustomisasi CSS situs. Saya ingin menggunakan ekstensi yang sama untuk membuat beberapa perubahan pada GUI Firefox. Bagaimana saya bisa melakukan ini?

4

1 Jawaban

8
8
  1. Buka url ini: chrome: //browser/content/browser.xul Ini memuat halaman yang diisi dengan elemen GUI Firefox.
  2. Buka Alat-> Pengembang Web-> Inspektur.
  3. Periksa elemen yang ingin Anda sesuaikan. Ini kemudian memungkinkan Anda mendapatkan ID dan kelas dari berbagai elemen.
  4. Buat gaya Stylish yang memodifikasi elemen tersebut.

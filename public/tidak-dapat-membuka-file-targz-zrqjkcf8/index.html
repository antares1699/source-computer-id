Tidak dapat membuka file tar.gz

Apakah ada cara alternatif untuk membuka file ".tar.gz" selain -xzf atau -zxvf atau xzvf? Saya menjalankan Mac OS X 10.7.

Baris perintah saya kembali:

tar: Unrecognized archive format
tar: Error exit delayed from previous errors.

Saya melakukan ini untuk membuka mysql-5.1.33.tar.gz dari metode situs untuk menginstal MYSQL. Saya telah mencoba banyak metode untuk membuka tar dan ada yang salah. Apa yang tidak saya lihat?

Saya kira arsipnya rusak. Coba unduh lagi, dan bahkan kendalikan checksum jika nilai yang diinginkan diberikan di situs unduhan jika tidak berfungsi.

Jika masih tidak berhasil, coba

gunzip mysql-5.1.33.tar.gz

untuk melihat apakah kompresi telah dilakukan dengan benar, lalu

tar xvf mysql-5.1.33.tar

untuk mencoba membuka gulungan arsip. Ini akan menunjukkan masalah dengan lebih baik.

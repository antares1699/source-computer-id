Linux sesuai perintah dengan presisi detik

Saya mencari beberapa perintah Linux yang dapat meluncurkan perintah lain pada waktu tertentu.

Saya tahu tentang perintah at, tetapi perintah itu hanya memberi saya ketepatan beberapa menit, dan saya perlu ketepatan detik. Apakah ada opsi dengan perintah yang tidak saya sadari? Atau apakah ada perintah lain yang harus saya gunakan?

Orientasi apa pun akan berguna.

Buka Windows Explorer (Win Key+E), pilih bilah alamat (Alt+D), ketik "Recycle Bin", tekan enter, masuk.

Atau, buka jendela Jalankan (Win Key+R) dan ketik shell:recyclebinfolder

Anda dapat mencoba PowerShell:

@ECHO OFF start /b /wait powershell.exe -command "$Shell = New-Object -ComObject Shell.Application;$RecycleBin = $Shell.Namespace(0xA);$RecycleBin.Items() | foreach{Remove-Item $_.Path -Recurse -Confirm:$false}"

(dari: https://stackoverflow.com/a/21111992 )

Anda dapat mengklik kiri pada ruang kosong di desktop Anda, mulai mengetik "daur ulang", lalu tekan enter.

Ini akan membuka tempat sampah dan menampilkan isinya. Kemudian Anda akan memiliki opsi di bagian atas untuk "memulihkan file" atau "Kosongkan tempat sampah".

Sepertinya Anda memiliki program pihak ketiga atau produsen mouse yang telah memetakan ulang klik mouse kombo ke Windows Flip 3D ( Tombol Windows + Tab ). Untuk mengatasi masalah tersebut, Anda perlu mencari program dan menghapus pemetaan.

Jika Anda ingin menonaktifkan Flip 3D sepenuhnya, ikuti petunjuk di bawah ini. Perhatikan bahwa ini tidak "secara teknis" menyelesaikan masalah pemetaan. Namun, karena itu menonaktifkan fitur Windows, itu akan mencegah permainan Anda diminimalkan ketika mengklik dua tombol mouse pada saat yang bersamaan.

Cara Menonaktifkan Sepenuhnya Windows Flip 3D melalui Registry - Windows 7

  1. Klik Mulailah ikon
  2. Ketik regedit
  3. Pilih regedit dari atas hasil
  4. Pergi ke: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DWM
  5. Langkah untuk menambahkan DisallowFlip3d ke registri
    1. Pastikan DWM dipilih
    2. Klik kanan pada area kosong di sisi kanan file Regedit program
    3. Arahkan kursor ke atas Baru
    4. Pilih Nilai DWORD (32-bit) dari daftar
    5. Ubah nama menjadi: DisallowFlip3d
    6. Klik dua kali DisallowFlip3d yang baru saja dibuat
    7. Perubahan Data nilai sampai 1
    8. Klik baik
  6. Mulai ulang

Ini adalah cara melakukannya tanpa mengunduh apa pun. Langkah-langkahnya adalah langkah-langkah detail untuk membantu pengguna berpengalaman dan tidak berpengalaman.

Penafian: Anda Sebaiknya selalu buat cadangan registri Anda sebelum mengedit registri Anda.

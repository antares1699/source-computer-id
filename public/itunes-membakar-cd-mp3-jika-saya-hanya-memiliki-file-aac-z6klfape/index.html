iTunes: Membakar CD MP3 jika saya hanya memiliki file AAC?

Apakah ada cara untuk membakar CD MP3 saat iTunes hanya berisi file AAC (tidak dilindungi) yang dibeli dari toko iTunes?

Tidak ada pilihan langsung di iTunes untuk mengonversi AAC -> MP3 seperti untuk MP3 -> AAC, dan harus mengonversi file AAC ke MP3 dan menggunakan alat eksternal untuk membakar disk akan memakan waktu. Apakah ada plugin untuk iTunes yang dapat melakukannya? Atau program yang dapat membaca playlist iTunes, mengonversi trek ke MP3 "dengan cepat" sambil membakar CD?

Iya , tetapi Anda perlu membuat versi MP3 dari file Anda sebelum mencoba membakar CD MP3 Anda.

Sayangnya, iTunes tidak dapat mengonversi lagu ke file MP3 dengan cepat. Untuk membakar CD MP3, Anda perlu menambahkan lagu MP3 ke daftar putar Anda. Ini tidak akan berfungsi untuk lagu AAC yang ditambahkan ke dalamnya, dilindungi atau tidak.

Tapi tunggu dulu, ada kabar baik: Anda bisa dengan mudah membuat versi MP3 dari lagu Anda. Tanpa perangkat lunak tambahan, langsung dari iTunes.

Item menu untuk "Membuat Versi AAC" dari lagu Anda, seperti yang Anda sebutkan, sebenarnya terkait dengan pengaturan impor Anda. Jika Anda mengubah format pengkodean ke MP3, item menu untuk mengonversi lagu Anda akan mencerminkan pilihan itu, memungkinkan Anda untuk membuat versi MP3 dari lagu Anda.

Berikut cara melakukannya:

  1. Buka preferensi iTunes ("iTunes" Menu> "Preferences ...")
  2. Pilih tab "Umum"
  3. Klik pada tombol "Import Setting ..."
  4. Pilih "MP3 Encoder" untuk opsi "Import Using:"

Anda kemudian dapat melihat " Buat Versi MP3 "item dari menu" Advanced ".

Langkah selanjutnya sederhana:

  1. Pilih lagu yang ingin Anda bakar sebagai CD MP3
  2. Pilih item "Buat Versi MP3" dari menu "Lanjutan"
  3. Buat playlist baru
  4. Tambahkan lagu MP3 yang baru Anda buat ke dalamnya
  5. Klik tombol "Burn Disc" yang terletak di kanan bawah jendela
  6. Saat ditanya, pilih "CD MP3" untuk opsi "Format disk:"
  7. Klik "Bakar"
  8. Nikmati CD MP3 Anda.

Jika jawaban yang diberikan oleh Arko tidak berhasil, atau jika Anda hanya ingin mengonversi beberapa lagu:

Masuk ke Preferensi dan ubah pengaturan impor untuk menggunakan encoder MP3 (tab Umum -> Pengaturan Impor), setelah melakukan itu Anda akan memiliki opsi untuk Membuat versi MP3 dari lagu daripada Buat versi AAC (Dari menu Lanjutan atau konteks- Tidak bisa)

Untuk membakar daftar putar yang berisi non-MP3 sebagai MP3:

  1. Pilih playlist
  2. Pilih semua lagu di playlist
  3. Klik kanan pada lagu yang dipilih
  4. Pilih "Ubah ke MP3"
  5. Buat playlist baru dengan versi MP3 dari lagu-lagunya
  6. Bakar playlist itu ke disk

Kesalahan “Tidak ada dukungan mouse: mouse = a” di vim

Saya menjalankan skrip tertentu dan mendapatkan kesalahan berikut:

  Error detected while processing /home/subhrcho/.vimrc:
  line   42:
   E538: No mouse support: mouse=a

Ini adalah bagian yang relevan dari file .vimrc:

" In an xterm the mouse should work quite well, thus enable it.
 set mouse=a

Ada gagasan tentang apa yang salah?

Red Hat Enterprise Linux Server rilis 5.8 (Tikanga)

Mencari kesalahan (:help E538) membawa Anda ke bantuan untuk pengaturan 'mouse':

Hanya berfungsi untuk terminal tertentu (xterm, MS-DOS, Win32, QNX pterm, konsol BSD dengan sysmouse dan konsol Linux dengan gpm).

Juga, periksa apakah Vim Anda (Anda tidak menyatakan baik platform maupun versi) dikompilasi dengan dukungan mouse:

:echo has('mouse')

Jika ya, itu karena terminal; gunakan yang lain. Jika tidak, pertama-tama Anda perlu mencari atau mengompilasi biner Vim yang memiliki dukungan mouse.

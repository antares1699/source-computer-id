Bagaimana cara membersihkan komputer saya dari spyware dan adware

Saya punya sedikit masalah spyware.

Saya mencoba menggunakan Cara Membersihkan Infestasi Spyware Windows untuk membantu menghilangkan spyware / adware di komputer saya. Saya memiliki autoruns dan proses explorer dan menyingkirkan file yang tidak memiliki nama penerbit atau perusahaan. Saya me-restart komputer dan file yang sama kembali. Saya benar-benar tersesat menjelang akhir artikel tentang kait winlogon dan mencoba menemukan pegangan yang buruk dan file DLL.

Saya tidak ingin menghapus file winlogon karena saya tidak dapat masuk (sesuai dengan peringatan) dan tidak ada nama penerbit seperti perusahaan Microsoft.

Sepertinya Anda telah menghapus semua file tanpa menggunakan penerbit yang valid Proses Explorer .

Ini adalah bagian yang mungkin membuat Anda terjebak:

Aplikasi jahat suka menempelkan dirinya pada proses sistem yang tidak dapat dibunuh. Dengan begitu, mereka tidak dapat dengan mudah dihapus dan secara ajaib akan "muncul kembali".

Seperti sebelumnya, penjelajah proses pemindaian mencari proses tanpa penerbit yang valid. Catat nama file dari proses ini. Anda sekarang perlu bunuh semua utas aktif dalam proses sistem tak terbunuh yang mereferensikan file jahat ini .

Gunakan fungsi find di Process Explorer untuk mencari referensi langsung ke file jahat. Dialog properti proses adalah tempat yang Anda inginkan, lalu pilih tab utas dan klik tombol Bunuh untuk setiap utas jahat.

Jalankan Process Explorer dan gunakan Find | Temukan menu Handle atau DLL untuk menemukan semua contoh DLL ini berdasarkan nama. (Lihat, saya katakan opsi ini sangat kuat.) Bunuh semua pegangan yang terbuka ke file ini yang Anda temukan, persis seperti yang kita lakukan sebelumnya. Tetapi Anda harus melangkah lebih jauh. Kami tahu dari Autoruns bahwa DLL ini kemungkinan besar akan dilampirkan ke proses Explorer dan Winlogon, tetapi biarkan hasil pencarian menjadi panduan Anda. Klik dua kali pada proses yang Anda temukan yang mereferensikan DLL ini. Dalam dialog properti proses, pilih tab Threads. Gulir melalui utas dan bunuh setiap orang yang memiliki DLL nakal dimuat.

Setelah Anda membunuh semua utas, Anda akhirnya dapat menghapus entri di Autoruns tanpa mereka kembali. Nyalakan ulang, dan mesin Anda sekarang benar-benar bebas dari spyware. Saya menghitung 17 entri di Task Manager, jumlah yang persis sama seperti saat saya memulai.

Sampai Anda melakukan ini, file tidak akan dapat dihapus!

Cara yang paling efektif, namun mengganggu, untuk membersihkan komputer dari spyware dan adware adalah dengan memformat dan menginstal ulang.

Selanjutnya, saya akan mencoba menggunakan solusi dan alat gratis yang sudah ada di sebagian besar mesin Windows.

Pergi ke Start> Run dan ketik "mrt.exe" untuk menjalankan file Alat Penghapus Perangkat Lunak Berbahaya Microsoft Windows . Bilah judul untuk program harus memiliki bulan dan tahun terakhir di dalamnya untuk memberi tahu Anda bahwa program tersebut baru saja diperbarui. Pilih "pemindaian penuh" dan biarkan melakukan tugasnya.

Anda mungkin juga punya Windows Defender terpasang. Menjalankannya.

Anda mungkin pernah keamanan penting microsoft , alat antivirus gratis dari Microsoft. Jika tidak, Anda dapat mencari dan mendownloadnya.

Terakhir, kunjungi onecare.live.com dan klik "Safety Scanner." Program Active-X dari Microsoft akan memindai komputer Anda dari malware.

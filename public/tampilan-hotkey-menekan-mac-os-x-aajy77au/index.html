Tampilan hotkey menekan Mac OS X

Apakah ada cara untuk menampilkan penekanan tombol cepat saat terjadi di layar dalam teks besar di Mac OS X Yosemite? Atau apakah ada aplikasi yang dapat melakukan ini?

Saya telah mencoba "Sticky Keys." Mereka hampir melakukan ini, tetapi ketika penekanan tombol pintas selesai, tombol itu segera menghilang. Saya ingin tombol hotkey tetap berada di layar sampai tombol hotkey berikutnya ditekan. Tombol lengket juga tidak menampilkan karakter yang terlibat dalam penekanan tombol pintas.

Saya juga telah mencoba "Tampilkan Penampil Keyboard". Jenis ini berfungsi, tetapi sulit untuk mengetahui dengan tepat tombol pintas mana yang ditekan karena ada begitu banyak informasi visual di keyboard. Penekanan tombol juga tidak akan bertahan hingga tombol berikutnya ditekan.

Bagaimana cara menginstalnya :

  1. Unduh aplikasinya: Tautan .
  2. Buka kemasan arsip keystroke-pro.zip yang baru saja Anda unduh.
  3. Seret aplikasi yang telah dibuka ke direktori aplikasi Anda.
  4. Mulai aplikasi "Keystroke Pro".

Bagaimana cara menggunakannya :

  • Untuk mendeteksi penekanan tombol, aplikasi perlu ditambahkan ke tab Aksesibilitas macOS: System Preferences...> Security & Privacy> Accessibility

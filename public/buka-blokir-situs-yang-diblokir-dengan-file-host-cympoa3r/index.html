Buka blokir situs yang diblokir dengan file host

Saya memiliki www.somesite.com yang diblokir di negara saya. Tetapi saya memiliki domain lain www.another.com yang mengarah ke server yang sama dan tidak diblokir (Apache Virtual Host) dan berfungsi dengan baik.

Saya memasukkan entri di file host komputer saya sebagai berikut.

www.another.com          www.somesite.com

Saya masih tidak bisa mengakses www.somesite.com. Apakah saya mendapatkan semua file host yang salah?

Ya, ini tidak benar. File host dirancang untuk memetakan satu host atau daftar host ke satu alamat IP. Sebaliknya, Anda telah mencoba memetakan sebuah host menjadi sebuah host.

Dengan asumsi www.another.com dan www.somesite.com mengarah ke alamat IP yang sama, Anda dapat mencoba yang berikut ini.

1 - Ping alamat yang dapat Anda selesaikan, yaitu www.another.com.

ping www.another.com

2 - Dengan asumsi IP 1.2.3.4, salin ke file host, masukkan spasi, lalu masukkan domain yang ingin Anda petakan.

1.2.3.4   www.somesite.com

3 - Di Windows, Anda mungkin perlu membersihkan cache browser Anda. Buka prompt perintah dan ketik:

ipconfig /flushdns

4 - Anda mungkin masih perlu memulai ulang browser Anda. Pada sistem berbasis Mac dan Unix, perubahan harus segera dilakukan.

Perhatikan bahwa, tergantung bagaimana situs diblokir, Anda mungkin perlu menggunakan metode lain untuk melewatinya. Jika diblokir oleh ISP Anda dan bukan DNS, Anda mungkin perlu mencari proxy Anonim untuk mengambil konten dari proxy di negara lain. Anda dapat menemukan beberapa proxy sini , serta di Google. Semoga berhasil!

Catatan: Anda dapat menambahkan lebih dari satu host ke satu IP dengan memberi jarak pada alamat:

1.2.3.4  my.domain.com  www.somewhere-else.com  anotheraddress.net

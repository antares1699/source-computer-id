Pengaturan Firefox “tabMinWidth” tidak berfungsi, bagaimana cara saya mengkonfigurasi lebar tab minimum?

Ketika saya menginstal komputer saya, saya ingin mengatur lebar minimum tab ke nilai yang lebih kecil melalui halaman about:config. Saya perhatikan bahwa ini tidak lagi berfungsi (saya menggunakan 8.0), meskipun pengaturannya masih ada. Bukankah ini masih harus bekerja?

Bagaimana cara mempersempit tab daripada saat ini?

Untuk userChrome.css, tambahkan:

.tabbrowser-tab:not([pinned])
{
  max-width: 150px !important;
  min-width: 10px !important;
}
.tabbrowser-tab:not([fadein])
{
 max-width: 1px !important;
 min-width: 1px !important;
 max-width: 1px;
 min-width: 1px;
}

Pada Firefox 71.0 (dari Januari 2020), pengaturan konfigurasi browser.tabs.tabMinWidth masih ada dan masih berfungsi. Anda dapat mengeditnya setelah memasukkan about:config ke dalam bilah URL.

Jadi mungkin saja peringatan depresiasi tentang pengaturan konfigurasi ini sendiri sudah usang.

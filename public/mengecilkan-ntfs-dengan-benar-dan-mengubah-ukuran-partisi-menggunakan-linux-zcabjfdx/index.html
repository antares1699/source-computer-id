Mengecilkan NTFS dengan benar dan mengubah ukuran partisi menggunakan linux

Saya mencoba mengubah ukuran disk 160GB agar sesuai dengan disk 80GB,

Saya telah mencoba 3 kali untuk mengecilkan NTFS saya dan mengubah ukuran partisi. Saya menggunakan backtrack sebagai live CD untuk melakukan ini,

saya menggunakan ntfsresize ke sistem file hingga 60 GB Lalu saya gunakan fdisk untuk menghapus partisi dan kemudian membuat partisi baru mulai dari titik yang sama tetapi berakhir pada 60GB, saya menandai partisi tersebut dapat di-boot dan memberinya ID 7 yang untuk NTFS kemudian saya menuliskannya ke disk dan reboot, saya kemudian mendapatkan,

"Sistem operasi tidak ditemukan"

Jadi saya menggunakan ms-sys untuk memperbaiki MBR yang dikatakan berhasil, Tapi sekarang ketika saya boot saya dapatkan

"Terjadi kesalahan saat memuat sistem operasi"

Ada Ide, untungnya saya memiliki cadangan clonezilla dari disk 160GB dan saya terus menggambar ulang dan kemudian mencoba lagi tetapi saya terjebak bagaimana cara membuatnya berfungsi,

Mungkin layak untuk menginstal ulang gambar Anda dan mencobanya.

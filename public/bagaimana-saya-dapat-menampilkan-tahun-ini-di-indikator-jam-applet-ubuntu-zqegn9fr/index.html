Bagaimana saya dapat menampilkan tahun ini di Indikator-jam / applet Ubuntu?

Untuk waktu yang lama, saya ingin membuat jam Ubuntu 12.04 menunjukkan tahun ini; Saya suka itu muncul di tangkapan layar saya.

Namun, tidak ada konfigurasi gui untuk melakukan ini: https://bugs.launchpad.net/ubuntu/+source/gnome-panel/+bug/465689

Adakah yang tahu bagaimana saya bisa mengubah ini di beberapa file konfigurasi atau sesuatu?

Sehubungan dengan itu menjawab bantu saya mencari tahu, mari kita sesuaikan indikator-datetime.

gsettings set com.canonical.indicator.datetime time-format "'custom'"
gsettings set com.canonical.indicator.datetime custom-time-format "'%a, %d. %h  %H:%M %Y'"

Itu akan memberikan output "Sen, 02, Jul 21:00 2012".

Anda dapat menyesuaikan format asalkan dipahami oleh strftime. Untuk format lain, Anda dapat melihat halaman manual untuk perintah strftime dan sesuaikan.

Fungsi Excel-MATCH tidak berfungsi pada nilai yang ditempelkan

Saya mencoba menemukan posisi nilai (teks) dalam satu kolom di kolom lain. Saya menjalankan fungsi: =MATCH(B1, A:A, 0) dan saya mendapatkan hasil #N/A. Tetapi hasil ini salah ... karena saya melihat dengan jelas nilai B1 di kolom A.

Saya pikir masalahnya mungkin dengan fakta bahwa saya menempelkan sel ke dalam lembaran. Jadi saya melakukan uji coba di mana saya memasukkan nilai secara manual ke dalam sel dan kemudian menjalankan fungsinya. Hasil: Berhasil .... Tapi saya yakin sekali tidak ingin memasukkan semua data saya secara manual.

Jadi pertanyaan saya adalah ... bagaimana cara mengatasinya? Saya sudah mencoba menempelkan nilai dalam semua jenis format dan masih belum berhasil. Mungkin bukan ini masalahnya? Saya tidak tahu. Saran akan sangat dihargai.

Jika Anda mencocokkan angka, coba gunakan fungsi VALUE.

Misalnya =VALUE(A1) akan mengembalikan (Angka) 100 jika sel A1 diformat TEXT, dan berisi 100, atau 100 dengan spasi di belakang atau di depan (mungkin banyak, saya tidak mencoba)

Ini bisa sangat membantu ketika, seperti disebutkan di atas, pemformatan menghentikan pencocokan atau pencarian.

Inilah yang dikatakan bantuan Excel 2007 tentangnya:

Mengonversi string teks yang mewakili angka, menjadi angka

Teks bisa dalam format angka, tanggal, atau waktu konstan yang dikenali oleh Microsoft Excel. Jika teks tidak dalam salah satu format ini, VALUE mengembalikan #VALUE! nilai kesalahan.

Kemungkinan besar Anda memiliki spasi atau karakter khusus yang tidak dapat Anda lihat. (Masalah Format)

Daftar / Kolom tempat Anda mencari data Anda di (A: A) -Dalam Skenario ini

  1. Salin kolom (A: A) dan Tempel ke Notepad
  2. Setelah Disisipkan ke Notepad, Ctrl + A dan Ctrl + X

Kembali ke Excel

  1. Ctrl + V ke pada Kolom

Ini menghilangkan semua spasi di antara karakter / memperbaiki masalah pemformatan.

-Bukan cara terbaik untuk memecahkan masalah ini, tetapi ketika Anda memiliki ribuan baris untuk dicari, itu adalah solusi termudah.

Saya memiliki masalah yang sama dan saya baru saja menyelesaikannya.

Sebelum membandingkan sel atau menggunakan Application.Match di VBA, Anda harus mengonversi KEDUA sel ke format yang sama.

Salah satu rentang saya dapat memiliki string alfa-numerik ATAU nilai integer, jadi saya mengonversi semuanya menjadi Teks jenis format menggunakan kode berikut:

Range("A1").NumberFormat = "@" 'The @ sign converts to Text type

Untuk semua jenis pemformatan sel lainnya, halaman web ini adalah referensi yang bagus:

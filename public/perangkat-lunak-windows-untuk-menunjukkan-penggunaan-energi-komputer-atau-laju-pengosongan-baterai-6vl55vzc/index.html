Komputer

Perangkat lunak Windows untuk menunjukkan penggunaan energi komputer atau laju pengosongan baterai

Apakah ada software untuk Windows 7 yang dapat menunjukkan penggunaan energi komputer secara real-time? Saat ini saya menggunakan Battery Bar, yang memberikan perkiraan laju pengosongan, tetapi ini pada dasarnya adalah rata-rata penurunan daya baterai selama periode waktu tertentu, dan tidak seketika - tampaknya merespons perubahan dengan sangat lambat. Saya ingin sesuatu yang menunjukkan berapa watt yang digunakan komputer pada waktu tertentu. Selain itu, bilah baterai tidak berfungsi untuk komputer desktop, dan saya ingin dapat mengukur penggunaan energi sistem untuk desktop juga.

Kalau software juga bisa catatan penggunaan daya dari waktu ke waktu, itu akan menjadi lebih baik.

Akhirnya, saya menemukan solusinya! Microsoft memiliki program Lab Penelitian gratis bernama Joulemeter yang memperkirakan penggunaan daya dari keseluruhan sistem dan berbagai komponen:

Joulemeter adalah alat perangkat lunak yang memperkirakan konsumsi daya komputer Anda. Ini melacak sumber daya komputer, seperti pemanfaatan CPU dan kecerahan layar, dan memperkirakan penggunaan daya.

Pada laptop menggunakan sirkuit pengukur daya internal yang saya anggap ada di sebagian besar komputer. Untuk komputer desktop, dapat membuat perkiraan atau menggunakan alat pengukur daya eksternal untuk mengkalibrasi sendiri.

Sayangnya, saya mengalami beberapa kesalahan dengan kalibrasi otomatis di laptop saya (mungkin terkait dengan fakta bahwa baterai HP tidak menyediakan data penggunaan baterai sebanyak laptop lain), tetapi dapat mengkalibrasi secara manual.

Saya baru saja menemukan ini dan belum punya banyak waktu untuk bermain dengannya, tetapi sepertinya itu akan berhasil!

Kode (dalam C):

#include <stdio.h>
#include <windows.h>
#include <powrprof.h>
#pragma comment(lib, "powrprof.lib")

int main(void)
{
    SYSTEM_BATTERY_STATE sbs = {0};
    CallNtPowerInformation(SystemBatteryState, NULL, 0, &sbs, sizeof(sbs));
    printf("Charge rate: %.1f W\n", (int)sbs.Rate / 1000.0);
    return 0;
}

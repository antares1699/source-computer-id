Bagaimana saya hanya menampilkan dir basis di Terminal Ubuntu saya?

Di Terminal Ubuntu 10.04 ketika saya melintasi beberapa struktur direktori yang panjang, seluruh jalur akan ditampilkan, ini menyebabkan banyak pemborosan ruang di terminal saya, di mana kursor datang ke kanan. Itu menyebabkan banyak Iritasi. Cara apa pun untuk hanya menampilkan dir basis.

Saat ini - ops@ops:/media/Main320/final_code/code/office/c/practice/parsers/proj1$

Perlu - ops@ops:~/proj1$

Saya yakin itu beberapa perubahan konfigurasi atau beberapa preferensi profil berubah. Siapapun ...

Mengubah Bash prompt itu mudah. Cukup tetapkan nilai baru ke PS1:

PS1="myprompt : "

Sekarang prompt baru akan terlihat seperti

myprompt:

Bash memungkinkan string prompt dikustomisasi dengan memasukkan sejumlah karakter khusus yang diloloskan dengan garis miring terbalik yang diterjemahkan sebagai berikut:

* \a : an ASCII bell character (07)
* \d : the date in "Weekday Month Date" format (e.g., "Tue May 26")
* \D{format} : the format is passed to strftime(3) and the result is inserted into the prompt string; an empty format results in a locale-specific time representation. The braces are required
* \e : an ASCII escape character (033)
* \h : the hostname up to the first '.'
* \H : the hostname
* \j : the number of jobs currently managed by the shell
* \l : the basename of the shell’s terminal device name
* \n : newline
* \r : carriage return
* \s : the name of the shell, the basename of $0 (the portion following the final slash)
* \t : the current time in 24-hour HH:MM:SS format
* \T : the current time in 12-hour HH:MM:SS format
* \@ : the current time in 12-hour am/pm format
* \A : the current time in 24-hour HH:MM format
* \u : the username of the current user
* \v : the version of bash (e.g., 2.00)
* \V : the release of bash, version + patch level (e.g., 2.00.0)
* \w : the current working directory, with $HOME abbreviated with a tilde
* \W : the basename of the current working directory, with $HOME abbreviated with a tilde
* \! : the history number of this command
* \# : the command number of this command
* \$ : if the effective UID is 0, a #, otherwise a $
* \nnn : the character corresponding to the octal number nnn
* \\ : a backslash
* \[ : begin a sequence of non-printing characters, which could be used to embed a terminal control sequence into the prompt
* \] : end a sequence of non-printing characters

Sebagai contoh, mari buat string prompt yang menampilkan tanggal hari ini dan nama host:

PS1="\d \h $ "

Outputnya seperti

Minggu 04 Sep ubuntu $

Ketika Anda puas dengan string prompt Anda, Anda dapat menjadikannya prompt default, bahkan setelah reboot dengan mengatur var PS1 di .bashrc

Seperti yang dikatakan @ nkr1pt, Anda ingin menyetel PS1

PS1 = '\ u @ \ h \ W \ $'

Juga periksa pengaturan variabel env PROMPT_COMMAND, yang merupakan perintah yang dijalankan setiap kali bash mencetak prompt. Saya telah menetapkan milik saya ke:

PROMPT_COMMAND='echo -ne "\e]0;$USER@${HOSTNAME%%.*}: $(pwd -P)\a"'

Jelas dengan USER dan HOSTNAME yang disetel di tempat lain. Ini menempatkan jalur lengkap dalam judul istilah Anda.

Bash 4 memiliki variabel yang disebut PROMPT_DIRTRIM

user@host:~$ echo $PS1
\u@\h:\w\$
user@host:~$ cd /usr/share/doc/bash/examples
dennis@emperor:/usr/share/doc/bash/examples$ PROMPT_DIRTRIM=2
dennis@emperor:.../bash/examples$

yang menetapkan jumlah minimum elemen direktori untuk ditampilkan.

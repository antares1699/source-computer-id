Format apa yang digunakan saat merekam musik ke CD?

Saya memiliki banyak file mp3 yang ingin saya rekam ke CD. Namun, saya tidak yakin apakah pemutar CD dapat membaca MP3. Format musik apa yang harus digunakan untuk memastikan terbaca dengan benar? Dan apakah itu penting atau apakah CD menggunakan format "internal"?

4
Javier — Sumber
  • CD memang menggunakan format internal, yang disebut "Redbook Audio". Banyak program pembakar CD memiliki tujuan untuk membakar dengan nama "CD Audio" atau yang serupa, berbeda dengan "CD data" yang akan Anda bakar seolah-olah pemutar Anda dapat membaca file .mp3 dari disk. Redbook Audio membatasi Anda hingga 80 menit musik yang dikodekan pada PCM 16-bit pada 44.1KHz.
    — LawrenceC

Musiknya harus masuk Format WAV agar dapat dibaca dari sebagian besar pemutar CD.
Pembuat CD MP3 adalah program mudah yang dapat Anda gunakan untuk meletakkan MP3 sebagai WAV di CD.

Memperbarui
Seperti yang ditunjukkan oleh @Vdex di komentar, formatnya sebenarnya Format audio RAW yang sebanding dengan WAV tanpa informasi header.
Bahkan lebih spesifik CDDA , sebagai ditunjukkan oleh @Joakim .
Itulah mengapa Anda membutuhkan perangkat lunak untuk membuat CD audio dari file audio tunggal.
Saya rasa jika Anda sudah memiliki file WAV, Anda dapat melakukannya dengan sebagian besar perangkat lunak perekam CD (pilih untuk membuat CD audio atau musik pada langkah pertama sebagian besar wizard, lalu pilih file WAV Anda), tetapi MP3 perlu dikonversi ke WAV dulu. Di situlah MP3 CD Maker berperan.

Dapat dimainkan di apa saja Pemutar CD yang perlu dibakar Format CD Audio (CD-DA) . Ini tidak sama dengan CD data dengan sistem file (iso9660) di mana musik (mp3 atau apapun) dapat direpresentasikan sebagai file. Format CD Audio tidak memiliki sistem file atau format file, musik hanya diletakkan langsung di CD dalam format RAW trek demi trek.

apa saja tidak sepenuhnya benar karena pemutar CD Audio lama (sebenarnya) mungkin bermasalah dengan media yang dibakar.

Beberapa pemutar CD dapat memutar file mp3 (kebanyakan yang baru melakukannya), biasanya ada stiker yang bertuliskan "kompatibel dengan mp3" atau yang serupa, dan jika demikian Anda tidak perlu mengonversinya, tidak semua pemutar cd mp3 dapat menangani sub-folder, jadi Anda mungkin perlu menempatkan semua mp3 di folder root. Jika tidak dapat menangani mp3 harus menulis CD dengan cara khusus (format cd audio), untuk itu Anda memerlukan perangkat lunak khusus, saya telah menggunakan Nero di masa lalu - Anda cukup menyeret dan melepaskan mp3 dan mengubahnya untuk Anda.

Bagi saya, pemutar CD yang tidak dapat menangani mp3 akan menjadi pemecah masalah, mengingat CD audio mentah menyimpan sekitar 75 menit musik per disk, dibandingkan dengan ~ 700 menit yang dapat Anda capai dengan mp3.

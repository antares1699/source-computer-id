Bagaimana cara menambahkan tombol 'Matikan Komputer' ke RocketDock?

Saya menjalankan Windows XP dengan RocketDock .

Saya ingin menambahkan tombol / ikon ke dok saya yang bertindak dengan cara yang sama seperti mengklik tombol 'Matikan Komputer' di menu Mulai yaitu sehingga menampilkan kotak dialog 'Siaga, Matikan, Mulai Ulang'.

Saat ini saya memiliki tombol di dok saya yang mematikan komputer saya dan tombol lain yang memulai ulang tetapi saya hanya ingin satu tombol menampilkan kotak 'Siaga, Matikan, Mulai Ulang' sehingga saya memiliki pilihan biasa (dan dapat menyimpan tombol di waktu yang sama).

  1. Buka Notepad atau editor teks apa pun yang Anda gunakan.
  2. Salin satu baris berikut ke teks: (new ActiveXObject("Shell.Application")).ShutdownWindows();
  3. Simpan sebagai "Shutdown.js" (tanpa tanda kutip) di mana pun Anda suka di komputer Anda.
  4. Buat pintasan baru yang mengarah ke Shutdown.js.

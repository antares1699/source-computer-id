Buka file dengan instance Notepad ++ portabel yang ada

Saya memiliki Notepad ++ portable "diinstal" dan jika saya mencoba membuka file dari Windows Explorer yang seharusnya terbuka dengan Notepadd ++ portable saya mendapatkan kesalahan

Contoh lain dari Notepad ++ sudah berjalan. Harap tutup instance Notepad ++ lainnya sebelum meluncurkan Notepad ++ portable.

Masalah saya adalah saya tidak ingin membuka contoh terpisah, saya ingin file dibuka dengan contoh yang ada.

Saya menemukan entri blog tentang cara mengatasi masalah ini dengan membuka banyak contoh http://mattrefghi.com/blog/2010/11/how-to-launch-multiple-instances-of-notepad/ tetapi seperti yang saya katakan, saya lebih suka file tersebut dibuka di jendela Notepad ++ saya yang sedang berjalan

Saya dapat menarik & melepas file di jendela yang ada dan berhasil, tetapi mengklik dua kali tidak akan membuka file.

Adakah yang tahu trik untuk membuat ini berhasil?

Anda perlu mengarahkan ke aplikasi yang dapat dijalankan, bukan ke peluncur. Jika Anda menunjuk ke:

X:\PortableApps\Notepad++Portable\Notepad++Portable.exe

itu tidak berfungsi dan memberi tahu Anda Notepad ++ sudah berjalan.

Tetapi jika Anda menunjuk ke:

X:\PortableApps\Notepad++Portable\App\Notepad++\notepad++.exe

maka semuanya berjalan seperti yang diharapkan.

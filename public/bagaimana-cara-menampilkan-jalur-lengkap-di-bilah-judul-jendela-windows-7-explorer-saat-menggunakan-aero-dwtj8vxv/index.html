Bagaimana jendela Windows 7 Explorer menampilkan jalur lengkap di bilah judul saat menggunakan Aero (bukan Tema Klasik)?

Ini tidak dapat dilakukan menggunakan solusi Windows 7 asli apa pun.

Namun, ada beberapa utilitas pihak ketiga yang dapat mencapai efek ini. Mereka -

  • AeroBar .
  • ShellFolderFix .
  • Explorer Klasik dari Shell Klasik .

Untuk ketiga opsi untuk mendapatkan jalur lengkap (dan bukan hanya nama folder) pastikan "Tampilkan jalur lengkap di bilah judul (Hanya tema klasik)" dipilih di bawah 'Alat, Opsi Folder, Tampilan' dari jendela Explorer mana pun.

AeroBar

Yang paling sederhana dari ketiganya, misi utamanya adalah menampilkan nama / jalur folder di jendela Explorer.

Cukup unduh file yang dapat dieksekusi dari tautan, dan jalankan. Jalur folder akan segera muncul. Jalankan lagi (atau matikan yang dapat dieksekusi dari pengelola tugas) untuk menonaktifkan.

Untuk mendapatkan ikon serta nama / jalur folder - ganti nama AeroBar.exe menjadi AeroBarIco.exe sebelum menjalankannya.

ShellFolderFix

Bertujuan untuk mengingat posisi dan ukuran jendela Explorer (mirip dengan perilaku Windows XP). Ini juga mendukung penyimpanan dan pemulihan posisi secara eksplisit.

Sebagai fitur samping, ia juga memiliki opsi "Tampilkan judul jendela di bawah Aero" dan "Tampilkan ikon judul jendela di bawah Aero".

Jika diaktifkan, jendela Explorer akan memiliki judul dan ikon.

Tautan yang disediakan memiliki versi "portabel" dan penginstal. Ambil pilihanmu.

Penjelajah Klasik dari Shell Klasik

Aplikasi yang dirancang untuk menghidupkan kembali berbagai fitur yang hilang selama bertahun-tahun di Vista dan Windows 7.

Di antara fitur berguna lainnya (seperti menambahkan ikon "Naik" dan menonaktifkan bilah alamat runut tautan), fitur ini memiliki dua opsi untuk "Tampilkan teks di bilah judul" dan "Tampilkan ikon di bilah judul".

Aktifkan keduanya untuk mendapatkan efek yang diinginkan.

Untuk detail tentang menggunakan Classic Explorer, lihat tautan yang disediakan.

Untuk menghapus semua cookie, klik roda gigi di pojok kanan atas dan buka opsi internet.

Pada tab umum, di bawah riwayat penjelajahan, klik delete....

Di jendela baru Anda dapat mencentang kotak untuk menghapus semua cookie Anda.

Jika Anda ingin "folder Temp Internet Files" secara default itu

%localappdata%\Microsoft\Windows\Temporary Internet Files

tetapi Anda bisa memindahkannya. Anda dapat menemukan folder saat ini dengan mengklik pengaturan di bagian riwayat penelusuran dari bagian pertama jawaban ini dan klik View Files. Anda dapat menghapus cookie satu per satu dari folder itu jika Anda mau.

Jawaban Vima dan Scott tidak lengkap. Saya telah menghapus cookie dengan cara resmi, dan terkejut melihat bahwa Gmail masih ingat sebagai pengguna yang terakhir saya masuki. Itu karena Anda harus ingat untuk un periksa tombol "Pertahankan data situs web Favorit" yang tampak menipu di dialog Hapus Riwayat Peramban:

Untuk sepenuhnya menghapus cookie IE9, lakukan ini:

  1. Hapus %appdata%\Microsoft\Windows\Cookies dan subfoldernya Low
  2. Hapus %localappdata%\Microsoft\Windows\Temporary Internet Files

Jika secara kebetulan sebuah situs tampaknya masih memiliki akses ke cookie, tekan F12 untuk alat pengembang, lalu dari menu Cache, pilih "Hapus cookie untuk domain". Untuk rasa penasaran Anda, Anda dapat memilih "Lihat informasi cookie", untuk melihat di mana mereka berada, lalu hapus direktori itu juga.

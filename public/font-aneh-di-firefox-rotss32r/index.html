Font aneh di Firefox

Saya mengalami masalah ini di mana font saya di Firefox berubah menjadi huruf tebal jelek yang aneh ini dan sepertinya saya tidak dapat mengubahnya. Pengaturan font saya persis sama dengan yang ada di komputer saya yang lain dan font yang satu itu normal. Saya menjalankan Windows Vista.

Gambar paling atas adalah gambar abnormal dan gambar paling bawah adalah gambar normal untuk perbandingan.

Sudahkah Anda memeriksa apakah hal yang sama terjadi dengan IE? Jika ya, beberapa font Anda rusak. Anda dapat memperbaikinya dengan mengekstrak fonta dari disk instalasi Windows Vista (jika Anda memilikinya) atau menggunakan mesin pencari favorit Anda dan mencari Font Inti Microsoft dan Koleksi Font Microsoft ClearType . Saya berasumsi Anda sudah tahu cara menginstal font.

Namun, jika IE tidak menunjukkan masalah yang sama, Anda dapat mencoba mengubah font default di Firefox ( Alat> Opsi ...> Konten> Font & Warna> Lanjutan ... ) ke font lain, restart, ubah font kembali seperti sebelumnya, dan restart lagi. Jika ini tidak berhasil, Anda harus menghapus profil Firefox Anda, dan membuat yang baru. Lihat sini untuk informasi lebih lanjut tentang itu.

Pengakuan Vim Gemfile

Ketika saya membuka Gemfile di vim, saya tidak secara otomatis mendapatkan warna untuk kode tersebut. Saya harus melakukannya secara manual :set syntax=ruby.

Apakah ada cara sederhana untuk mengotomatiskan ini?

4

  • Dapatkah Anda membuat ~ / .vim / filetype.vim yang menyetel tipe file menjadi ruby ​​untuk file GEM?
    — Scott C Wilson

1 Jawaban

6

Jika Anda menjalankan Vim 7.3, atau jika Anda telah menginstal versi terbaru dari vim-ruby plugin (versi apa pun yang lebih baru dari Desember 2009), maka ini seharusnya sudah terjadi — keduanya mengatur untuk menyetel filetype=ruby untuk file bernama Gemfile.

Jika tidak ingin menginstal vim-ruby, maka Anda masih dapat secara otomatis menggunakan dukungan Ruby yang ada pada file bernama Gemfile dengan meletakkan ini di ~/.vimrc Anda:

autocmd BufNewFile,BufRead Gemfile set filetype=ruby

Anda dapat melewati baris ini jika Anda memutuskan untuk menginstal vim-ruby untuk mendapatkan manfaat dari indentasi / sorotan / penyelesaian yang diperbarui (vim-ruby secara otomatis melakukan autocmd di atas).

Tidak ada pesan baru di Inbox yang menggunakan Outlook 2013, IMAP, tetapi pesan muncul di Terkirim

Saya mengalami masalah yang mirip dengan ini tetapi jawaban yang diberikan tidak menyelesaikannya.

Gejala: Menggunakan layanan email non-gmail, saya telah mengonfigurasi IMAP di beberapa perangkat menggunakan pengaturan yang sama dan satu-satunya contoh Outlook ini adalah satu-satunya yang tidak berfungsi dengan benar. Saya melihat setiap email yang saya kirim di folder Terkirim saya, dan saya melihat spam baru di folder Spam saya, tetapi tidak ada yang baru yang muncul di Kotak Masuk saya. Kesalahan berikut muncul dalam masalah Sinkronisasi beberapa kali setiap kali saya meluncurkan Outlook:

20:44:54 Synchronizer Version 15.0.4551.1004
20:44:54 Synchronizing Mailbox 'xxx@yyy.zzz'
20:44:54 Synchronizing Hierarchy
20:44:57 Error in folder 'spam'
20:44:57         [800CCCD2-0-0-733]
20:44:57 Terminated in error
20:44:57         [800CCCD2-0-0-733]

... dengan 'spam', 'terkirim', dan 'sampah' dicantumkan sebagai folder kesalahan. Itu adalah folder yang sepertinya berfungsi. Ukuran folder untuk Kotak Masuk tidak berubah, jadi menurut saya folder tersebut tidak sedang diunduh.

Saya sudah mencoba:

  • mengganti nama .ost, memaksa Outlook untuk membuat ulang ost
  • mengubah opsi "email agar tetap offline" dari dan menjadi "semua"
  • memulai Outlook dalam mode aman
  • menghapus dan membuat ulang profil email saya
  • memperbaiki Office
  • menghapus instalan Office (dengan ini alat) dan menginstalnya kembali
  • booting boot bersih
  • menonaktifkan firewall
  • mengatur ulang tampilan di Outlook

Tidak ada opsi yang tersedia melalui antarmuka web cpanel penyedia email saya yang tampaknya terkait dengan ini.

Saya akan mengawasi utas ini, dan dengan senang hati memberikan detail lebih lanjut atau melakukan langkah apa pun yang disarankan.

Saya melihat ini beberapa kali seminggu dengan berbagai klien di Outlook 2013.

(PALING UMUM)
1. Nonaktifkan atau hapus instalan plugin pihak ketiga (terutama AntiSpam)

(KURANG UMUM)
2. Ubah Pengaturan Kirim / Terima untuk hanya memuat folder Kotak Masuk

(BAHKAN KURANG UMUM)
3. Pastikan Anda berlangganan folder Kotak Masuk

(SOLUSI YANG SELALU BERFUNGSI)
4. Jika semua di atas gagal, tambahkan akun IMAP lagi sehingga Anda memiliki dua contoh di profil yang sama. Setel instance IMAP yang baru ditambahkan sebagai default (ini juga mengubah penyimpanan data default). Izinkan waktu Outlook untuk memindahkan data ke contoh baru, lalu hapus contoh akun lama. Ini adalah solusi tetapi berhasil. Ini lebih baik daripada membuat profil baru jika Anda membutuhkan kontak, kalender, dan tugas. Alternatifnya, Anda dapat mencadangkan ke .pst, membuat profil baru, lalu mengimpor, tetapi itu lebih berfungsi kecuali Anda hanya menggunakan Outlook untuk email.

Peringatan!!! : Outlook 2013 melampirkan semua data ke akun Anda yang dimuat di Outlook. Jika Anda menghapus file data / pst sebelum menyiapkannya, Anda tidak akan dapat membukanya sebagai .pst seperti versi Outlook sebelumnya! Anda akan kehilangan semua Kontak, Kalender, Tugas jika Anda menghapus akun sebelum mengaturnya lagi. </em>

Sudahkah Anda mencoba mengubah nomor porta?

Saya mengalami masalah dengan Outlook 2013 saat menyambungkan ke imap di mail.com. Saya menggunakan nomor port "465" di kotak "Server Keluar (SMTP)" di tab "Lanjutan" dari jendela "Pengaturan Email Internet". Saya mengubah porta menjadi "587", memilih "TLS" dari menu drop-down. Mengklik "OK" dan Outlook mulai mengunduh email.

Saya meninggalkan "Server masuk (IMAP) sebagai" 993 "sebagai nomor port dan" SSL "sebagai jenis enkripsi dan itu berhasil untuk saya.

Utilitas penyalinan file besar TERBAIK di Windows adalah ESEUtil. Gunakan sakelar / Y. Info lebih lanjut di blog Technet di sini: http://blogs.technet.com/b/askperf/archive/2007/05/08/slow-large-file-copy-issues.aspx

Berikut adalah beberapa metode penyalinan yang lebih cepat daripada Explorer untuk dicoba (berbasis baris perintah kecuali disebutkan), agak dalam urutan kecepatan dari lebih lambat ke lebih cepat

  • Salin (Dilengkapi dengan Windows)
  • XCopy (Dilengkapi dengan Windows)
  • Robocopy ( http://www.microsoft.com/downloads/en/details.aspx?FamilyID=9d467a69-57ff-4ae7-96ee-b18c4790cffd&displaylang=en )
  • FastCopy (GUI, http://ipmsg.org/tools/fastcopy.html.en )
  • TeraCopy (GUI, http://www.codesector.com/teracopy.php )
  • EMCopy (Dilengkapi dengan alat EMC)
  • ESEUtil (Dilengkapi dengan Exchange).

Saya tidak tahu tentang yang satu itu tetapi yang memang bekerja lebih cepat tergantung pada apa yang Anda salin dan memiliki dukungan stop / resume TeraCopy :

TeraCopy dirancang untuk menyalin dan memindahkan file semaksimal mungkin kecepatan. Ini melewatkan file buruk selama proses penyalinan, dan kemudian menampilkannya di akhir transfer sehingga Anda dapat melihat yang mana yang membutuhkan perhatian. TeraCopy dapat secara otomatis memeriksa file yang disalin untuk kesalahan dengan menghitung nilai checksum CRC mereka. Ini juga menyediakan lebih banyak informasi tentang file yang sedang disalin daripada Windows-nya rekan. TeraCopy terintegrasi dengan klik kanan Windows Explorer menu dan dapat disetel sebagai penangan salinan default.

Saya sarankan menggunakan robocopy dengan opsi / MT: 16 (multithreading). Ini akan banyak membantu dengan menyalin sejumlah besar file kecil, tetapi tidak akan banyak membantu dengan file besar.

Ini dibangun ke dalam Windows secara default, tetapi membutuhkan pengetahuan baris perintah dan memiliki banyak sakelar - yang tampaknya menghalangi sebagian besar pengguna.

Jika Anda ingin menggunakan robocopy dengan cara yang sederhana dan Anda ingin berperilaku seperti salinan biasa, Anda dapat melihat RCWM project - Saya telah membuat robocopy ke menu klik kanan untuk mempercepat alur kerja saya. Semoga rangkaian alat ini dapat membantu orang lain juga.

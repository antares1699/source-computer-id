Bagaimana cara menonaktifkan Tampilan Slide Latar Belakang Desktop di Windows 7?

Di Windows 7 RTM baru, saya tidak dapat melihat tempat di Panel Kontrol-> Penampilan dan Personalisasi-> Personalisasi untuk menonaktifkan fitur tampilan slide latar belakang desktop. Anda dapat mengubah apa yang disertakan, jadi saya kira jika Anda memilih satu latar belakang yang akan melakukannya.

Adakah cara untuk mematikan fitur ini?

Batalkan pilihan semua gambar tetapi salah satunya harus berhasil.

A lebih cara yang canggih adalah:

Jeda Tampilan Slide Latar Belakang Desktop

  • Buka Control Panel di menu Start.
  • Buka Opsi Daya di Panel Kontrol.

Catatan: Jika Opsi Daya tidak tersedia, ubah Tampilkan oleh menjadi ikon Besar di kanan atas Panel Kontrol.

  • Klik Change plan settings di sebelah power plan yang ingin Anda edit.
  • Klik Ubah pengaturan daya lanjutan.
  • Klik Ubah pengaturan yang saat ini tidak tersedia untuk membuka kunci pengaturan daya lanjutan.

  • Setel Tayangan slide di bawah Pengaturan latar belakang Desktop ke Dijeda, lalu klik OK untuk menyimpan perubahan.

Huruf kandar seperti C: hanya ditetapkan selama Runtime penginstalan Windows, jadi biasanya keduanya C: untuk kedua penginstalan Anda.

Buka cmd.exe atau Powershell.exe yang ditinggikan (sebagai Administrator), lalu ketik:

diskpart

setelah beberapa saat, ketik:

list volume

Anda mendapatkan sesuatu seperti ini:

 Volume ###  Ltr  Label        Fs     Type        Size     Status     Info
 ----------  ---  -----------  -----  ----------  -------  ---------  --------
 Volume 1         WinBOOT      FAT32  Partition    350 MB  Healthy    System
 Volume 2     D   Win1         NTFS   Partition     31 GB  Healthy
 Volume 3     C   Win2         NTFS   Partition     33 GB  Healthy    Boot
 Volume 4     S   Data         NTFS   Partition    380 GB  Healthy

Dalam hal ini Anda dapat melihat bahwa Windows pada Volume 3 sedang berjalan, ada satu lagi pada Volume 2, Boot di kolom Info di paling kanan menunjukkan bahwa ini adalah Drive Windows saat ini.

Anda dapat melihat hal yang sama dengan membuka Manajemen Disk:

 diskmgmt.msc

dan cari Boot di kolom Status volume.

Saat menyiapkan Windows dual boot, Anda harus menggunakan nama deskriptif untuk setiap entri boot.

Beberapa cara lain

  • Buka dialog jalankan dengan Menang + R , ketik \ lalu tekan Enter. Itu akan menunjukkan drive Windows Anda. (Tip: menjalankan . akan menampilkan folder utama Anda dan .. akan menampilkan folder pengguna Anda)
  • Buka menu start, ketik "info sistem" dan buka Sistem Informasi . Anda akan melihat Direktori Windows dibawah Ringkasan Sistem
  • Buka menu start, ketik "manajemen disk" atau tekan Menang + R > diskmgmt.msc> Masuk. Itu akan mencantumkan semua drive di PC Anda. Drive sistem akan terdaftar dengan Boot bendera, kemungkinan besar bersama dengan "Crash Dump" dan "Page File"

  • Cukup tekan Menang + R dan jalankan cmd. Path ke cmd.exe sering ditampilkan di bilah judul secara default. Pada beberapa sistem, ini hanya menampilkan "Command Prompt" tetapi dalam hal ini Anda dapat melihat string prompt yang sering menunjuk ke folder pengguna Anda di drive sistem

    Ini cepat tetapi juga tidak dapat diandalkan, jadi saat Anda berada di command prompt, jalankan saja perintah di jawaban lain atau Anda dapat menjalankan echo %systemroot% atau set sys

    C:\>set sys
SystemDrive=C:
SystemRoot=C:\WINDOWS

  • tekan Menang + Berhenti sebentar atau klik kanan My Computer> Properties> System Protection. Drive sistem Anda akan terdaftar dengan kata "Sistem"

Yang mengatakan, saat ini Windows sering menunjukkan drive sistemnya selalu sebagai C: terlepas dari posisi volume sistemnya di drive, jadi label drive atau nama perangkat lebih berguna daripada huruf drive

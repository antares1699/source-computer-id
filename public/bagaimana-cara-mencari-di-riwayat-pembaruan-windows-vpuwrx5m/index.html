Bagaimana cara mencari di Riwayat Pembaruan Windows?

Jika saya mencoba mencari Nomor KB di Control Panel> Windows Update> View Update History, mengetik di pojok kanan atas kotak pencarian, pencarian tidak dilakukan di History tetapi di Control Panel itu sendiri.

Bagaimana cara mencari pembaruan yang selesai / gagal?

Gunakan Powershell untuk mencari pembaruan Windows yang diinstal

Get-Hotfix mengambil semua pembaruan yang terinstal saat ini di mesin, gunakan sakelar -id untuk menentukan nomor KB.

Contoh:

PS C:\Users\User> Get-HotFix -Id kb974322
Get-HotFix : Cannot find the requested hotfix on the 'localhost' computer. Verify the input and run the command again.
At line:1 char:1
+ Get-HotFix -Id kb974322
+ ~~~~~~~~~~~~~~~~~~~~~~~
    + CategoryInfo          : ObjectNotFound: (:) [Get-HotFix], ArgumentException
    + FullyQualifiedErrorId : GetHotFixNoEntriesFound,Microsoft.PowerShell.Commands.GetHotFixCommand

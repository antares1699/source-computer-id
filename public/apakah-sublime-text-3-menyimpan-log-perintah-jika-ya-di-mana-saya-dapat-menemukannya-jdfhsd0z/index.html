Saya yakin teks log dari jendela log Sublime juga ada di suatu tempat di sistem file. Jika saya dapat menemukan file log itu, saya dapat membukanya dengan vim (atau yang serupa) dan mencari di sana.

Ya, benar.

Anda dapat melihat riwayat konsol file ini:

/Users/maks/Library/Application Support/Sublime Text 3/Data/Local/Session.sublime_session

Ini adalah file JSON dan perintah konsol disimpan pada windows.console.history.

Satu catatan, Sublime Text tidak memperbarui file sampai Anda menutup program.

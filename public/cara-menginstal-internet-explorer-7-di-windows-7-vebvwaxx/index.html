Sudahkah Anda mencoba IE 8 dalam mode Kompatibilitas IE 7 penuh?

Anda dapat mencoba mengklik ikon kertas pecah di bilah URL di bagian atas (akan menempatkan Anda dalam mode Kompatibilitas) atau melakukan mode Browser IE 7 lengkap jika tidak berhasil:

Dengan IE 8, buka Alat Pengembang F12 , dan pastikan Anda menetapkan "Internet Explorer 7" sebagai Mode Browser dan "Standar Internet Explorer" sebagai Mode Dokumen Anda. Jika berhasil, Anda dapat mengubah pengaturan tampilan kompatibilitas untuk selalu menjalankan situs web tertentu dalam mode kompatibilitas.

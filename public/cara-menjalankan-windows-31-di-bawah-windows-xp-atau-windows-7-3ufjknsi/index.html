Cara menjalankan Windows 3.1 di bawah Windows XP (atau Windows 7)

Kami mengalami yang buruk di tempat kerja.

Kita harus menghidupkan kembali dan memodifikasi sepotong kode yang terakhir dibangun sekitar akhir 1990-an. Toolset berjalan pada Windows 3.1 dan 3.11. Itu tidak berjalan di bawah Windows XP: NTVDM melaporkan pengecualian yang tidak tertangani. Kami telah mencoba semua yang dapat kami pikirkan untuk kompatibilitas. (Jika ada yang memiliki daftar periksa serius untuk hal semacam ini, saya akan menerimanya.)

Adakah cara, menggunakan sistem mesin virtual pihak ketiga, atau SESUATU, untuk menjalankan sistem operasi Windows 3.1 yang nyata, langsung, dan jujur, mungkin pada mesin simulasi, di Windows XP atau Windows 7? Saya pikir kita kemudian dapat menginstal toolset lama pada mesin yang disimulasikan dan melakukan apa yang perlu kita lakukan.

Anda dapat menjalankan Windows 3.1 di VmWare, VirtualBox, Bochs atau mesin virtual lainnya dengan mode emulasi perangkat keras PC lengkap. Saya juga mendengar Windows 3.11 dapat berjalan di bawah DosBox, tetapi belum mencobanya sendiri.

Anda juga dapat mencoba menginstal Windows 95 di VM dan menjalankan toolchain Anda di atasnya, subsistem 16-bit Win95 sangat mirip dengan Win3.x, tetapi antarmuka pengguna lebih baik.

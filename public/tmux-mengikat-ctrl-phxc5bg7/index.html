Tmux: mengikat Ctrl-`

Saya ingin mengikat Ctrl-`di tmux ke panel-pilih.

Saya sudah mencoba: mengikat -n C-`pilih-panel -t:. + Yang memberi saya: kunci C-tidak diketahui

Saya juga sudah mencoba: bind -n 'C-' 'pilih-panel -t:. + Yang memberi saya kesalahan yang sama.

Saya kemudian mencoba sekadar ': bind -n `pilih-panel -t:. + Itu bekerja dengan baik (tapi terikat `bukan C-`.

Yang paling membingungkan, saya mencoba: bind -n `-C pilih-panel -t:. + Yang ternyata berhasil (saya harus menekan backtick dulu, lalu ctrl). Apa di dalam Ctrl-`yang tidak disukai tmux?

tmux menggunakan logika ini untuk menangani file ctrl pengubah di key-string.c (backtick adalah kode 96):

    /* Convert the standard control keys. */                               
    if (key < KEYC_BASE && (modifiers & KEYC_CTRL) && !strchr(other, key)) {
            if (key >= 97 && key <= 122)
                    key -= 96;
            else if (key >= 64 && key <= 95)
                    key -= 64;
            else if (key == 32)
                    key = 0;
            else if (key == 63)
                    key = KEYC_BSPACE;
            else
                    return (KEYC_NONE);
            modifiers &= ~KEYC_CTRL;
    }

Meskipun beberapa aplikasi mungkin memperlakukan seluruh rentang dari 64 hingga 126 sebagai valid untuk ctrl , tmux tidak termasuk backtic, curly braces, vertical bar dan tilde. Kemungkinan alasannya adalah bahwa mereka kurang berguna untuk pengikatan kunci karena kunci yang tidak dimodifikasi sering digunakan dalam skrip shell.

Di sisi lain, ctrl-@ akan memiliki interpretasi yang sama, dan Anda dapat menggunakannya sebagai gantinya.

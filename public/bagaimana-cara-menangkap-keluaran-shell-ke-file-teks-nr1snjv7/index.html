Bagaimana cara menangkap keluaran shell ke file teks?

Di linux saya bisa mengetik echo "hello" >& text.output dan mendapatkan output teks dalam sebuah file.

Bagaimana saya bisa mendapatkan keluaran semacam ini di Windows 8?

Apakah ada cara untuk membedakan kesalahan dari keluaran?

Motivasi saya mencoba mencatat keluaran tracert dan menyelidiki perbedaan antara Windows 8 dan Windows 7.

Cara yang sama seperti di Linux:

echo "Hello" >text.output

Kesalahan biasanya dicetak ke aliran stderr (ini # 2). Anda dapat merekam streaming stderr:

del 1.txt 2>text.output

Jika file 1.txt tidak ada, text.output akan berisi Could not find 1.txt.

Anda dapat mengalihkan stdout ke stderr:

echo "Hello" 2>&1

Atau stderr hingga stdout:

echo "Hello" 1>&2

P.S. Saya tidak yakin perintah Anda akan berfungsi di Linux. Pokoknya echo "hello" >& text.output menghasilkan kesalahan sintaks: >& was unexpected at this time.

Shell di Windows sekarang sangat mirip dengan BASH di Linux. Jadi Anda bisa melakukannya

echo "Hello" > text.txt

Atau, jalankan program Anda, dan biarkan keluaran sistem menulis ke text.txt:

run_program > text.txt

Jika Anda perlu melacak kesalahan, gunakan yang berikut ini:

run_program 2> error.txt

Sunting : truecrypt dihentikan mulai 28 Mei 2014
Anda bisa melihat VeraCrypt untuk fungsionalitas serupa.

Cara teraman adalah truecrypt .
Ini memiliki mode portabel jadi Anda tidak perlu memasang perangkat lunak.

Ini dia instruksi agar Truecrypt portabel untuk Mac. (dengan menyeret ".app" ke cd / usb Anda).

Jika tidak, Anda harus menggunakan perangkat lunak yang standar di kedua komputer.

Apakah "zip" dengan kata sandi bukanlah suatu pilihan?

Jika Anda akan menggunakan zip dengan kata sandi, pastikan untuk menggunakan enkripsi terkuat yang tersedia untuk kedua sistem. Metode enkripsi yang lebih lama relatif lemah.

Satu-satunya dokumen terenkripsi lintas platform yang didukung langsung di macOS dan Windows adalah file ZIP terenkripsi. Namun, hanya enkripsi ZipCrypto yang sangat lemah didukung, yang mudah retak. File ZIP terenkripsi AES tidak didukung dan memerlukan perangkat lunak tambahan seperti 7Zip.

Opsi lain yang perlu dipertimbangkan adalah PDF yang didukung oleh Chrome, Edge, dan Safari. Namun, Anda perlu memperhatikan pemilihan algoritma enkripsi yang cermat dalam PDF (RC4, AES128, AES256v1, AES256v2) karena beberapa di antaranya adalah rentan terhadap kekerasan seperti RC4 dan AES256v1. Juga, ingatlah pemeriksaan integritas yang hilang dalam file PDF , yang mungkin memungkinkan melewati enkripsi dalam beberapa kasus jika man-in-the-middle dapat mengganti dokumen sebelum pengguna akhir membukanya.

Ada opsi lain yang menggunakan kode yang dapat dieksekusi tetapi kemungkinan besar akan diblokir oleh sistem anti-spam / antivirus.

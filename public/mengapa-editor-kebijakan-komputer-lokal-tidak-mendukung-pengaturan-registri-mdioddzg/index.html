Kata kunci dalam gambar Anda dari antarmuka Server 2008 R2 adalah Preferensi :

Ini adalah Preferensi Kebijakan Grup Kegunaan:

Di Windows Server 2008 Microsoft memperkenalkan ekstensi Kebijakan Grup - Preferensi Kebijakan Grup (GPP). GPP termasuk pengaturan registri, yang memungkinkan Anda untuk menambah, menghapus, atau mengubah nilai kunci.

Pada akhirnya, Anda memiliki kemampuan untuk menggunakan pengaturan Preferensi Kebijakan Grup dalam Editor Manajemen Kebijakan Grup Windows Server 2008 R2 untuk menyebarkan modifikasi registri di seluruh domain Windows Anda. Namun, tidak ada kemampuan setara yang ada di dalam gpedit.msc editor kebijakan grup lokal pada workstation Windows 7 Enterprise.

(Sumber: Tambahkan, ubah, dan hapus kunci Registri menggunakan Kebijakan Grup )

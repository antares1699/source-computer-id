Apa nama file (termasuk jalur) dari database WordPress?

Saya memiliki database Wordpress lokal yang rusak akibat pemadaman listrik mendadak (saya bahkan tidak mengeditnya!).

Karena sistem yang saya hadapi cukup statis (dan lama), saya hanya ingin memulihkan file database (atau kumpulan file) dari backup malam.

Tapi saya tidak tahu yang file yang harus saya pulihkan. Blog WordPress yang dimaksud dipasang di bawah akun pengguna lokal di server Linux lama (FC4).

Saya memeriksa direktori wp-content, tetapi ternyata ini bukan tempat kontennya:

~/public_html/wordpress/wp-content> ls -l
total 24
-rw-r--r--  1 ww4s ww4s   32 Nov 18  2005 index.php
drwxr-xr-x  3 ww4s ww4s 4096 Jun  2  2006 plugins
drwxr-xr-x  4 ww4s ww4s 4096 Mar 10  2006 themes

Tahu file mana yang harus saya pulihkan dan apa jalurnya?

WordPress hampir pasti menggunakan database MySQL untuk menyimpan kontennya, dan itu biasanya disimpan di tempat lain di sistem, sering kali /var/lib/mysql/some_db_name. Buka file wp-config.php Anda dan mulailah melihat pengaturan MySQL Anda. Ini akan membantu Anda mengetahui lingkungan Anda dan ke mana harus mencari selanjutnya. Anda dapat melihat contoh file wp-config.php sini .

  • Memang, versi WordPress saya menggunakan MySQL, dan database disimpan di direktori /var/lib/mysql - jalur yang ditunjukkan oleh variabel datadir di /etc/my.conf (satu direktori per database). Terima kasih +1 + Menerima.

Bagaimana saya bisa mendapatkan partisi di dalam partisi?

Saya dulu memiliki Vista terinstal, tetapi ingin meningkatkan ke 7. Saya mempartisi drive saya, menginstal 7 di partisi baru kemudian menyalin semua data saya dari partisi Vista. Kemudian, saya pergi untuk menghapus partisi Vista, tetapi sekarang sistem saya tidak bisa boot. Saya menggunakan Gparted tetapi sepertinya saya memiliki partisi di dalam partisi. Saya melihat saya memiliki / dev / sda2, kemudian saya dapat membuka segitiga dan di dalamnya ada partisi Windows 7 / dev / sda5 saya. Bagaimana saya bisa mengeluarkan partisi Windows 7 saya dari sana?

Terima kasih sebelumnya!

Saya pikir Anda memiliki partisi logis di dalam partisi extended.
Itu ditunjukkan dengan 'sda5` (angka lebih besar dari 4; partisi primer maks).

Beberapa referensi,

  1. Apa perbedaan antara partisi "diperpanjang" dan "partisi" logis
    di unix.StackExchange.com
  2. Partisi / Diperpanjang: Drive Logis di SevenForums
  3. Partisi Disk di Wikipedia

Pembaruan atas komentar Anda,

Membuat Win7 Anda di partisi logis pindah ke primer tidak semudah itu :-)
Namun, jika Anda baik-baik saja untuk menghapus semua partisi logis (terlihat di dalam segitiga terbalik, seperti yang Anda menyebutnya), Anda kemudian dapat menghapus partisi perluasan itu sendiri (perhatikan bahwa Anda akan kehilangan Win7 dan partisi logis lainnya bersama dengan datanya saat Anda melakukan ini). Setelah Anda melakukan ini, Anda dapat membuat partisi primer baru dengan ukuran yang sama seperti 'sda5' sekarang dan kemudian menginstal Win7 baru ke dalamnya.

Ini akan memberi Anda partisi utama Win7 (mungkin terlihat sebagai 'sda2' di GParted).

Beberapa fakta dasar lainnya: Anda dapat memiliki paling banyak 4 partisi utama. Itulah sebabnya, partisi perluasan Anda menampilkan partisi logis mulai dari 'sda5'. Diberikan penjelasan Anda, saya melihat bahwa Anda memiliki satu partisi primer (sda1) dengan Vista dan ingin memiliki partisi primer lain (sda2) dengan Win7. Jika Anda memiliki ruang disk lebih banyak setelah dua partisi ini, Anda dapat membuat beberapa (paling banyak) partisi primer atau hanya membuat partisi tambahan untuk lebih banyak partisi logis untuk data Anda.

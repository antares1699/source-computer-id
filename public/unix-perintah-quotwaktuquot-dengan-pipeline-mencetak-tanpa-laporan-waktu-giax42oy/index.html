Unix perintah "waktu" dengan pipeline mencetak tanpa laporan waktu

Dari dua alias C-shell di bawah, hanya yang pertama yang mencetak waktu laporan perintah waktu yang telah berlalu. Bagaimana saya bisa mendapatkan alias dengan pipeline untuk juga mencetak laporan waktu?

alias make1 'time make'

alias make2 'time make |& tee make.log'

Itu waktu perintah bawaan shell C (csh atau tcsh) tidak berfungsi dengan pipeline. Untuk menghindari batasan ini, gunakan standalone waktu sebagai gantinya, yang biasanya ditemukan di / usr / bin / time (jika tidak, coba whereis time untuk menemukannya).

Ubah time di baris perintah Anda ke /usr/bin/time (atau /usr/bin/time -p) atau apa pun jalur ke program waktu, dan itu akan berfungsi.

Mengapa tidak berhasil:

Shell C (seperti beberapa shell lainnya) memiliki built-in waktu perintah (lihat builtin halaman manual), yang digunakan dalam preferensi program waktu non-builtin (mandiri):

% which time
time: shell built-in command.
%

dan halaman manual untuk csh (sebenarnya tcsh di sistem saya) menyatakan:

time [command]
   Executes command (which must be a simple command, not an alias,
   a pipeline, a command list or a parenthesized command list) ...

Perintah / usr / bin / time tidak memiliki batasan ini, dan begitu juga dengan perintah bawaan waktu perintah dari sebagian besar shell lain, misalnya bash atau zsh.

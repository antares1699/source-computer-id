Bagaimana cara memperbarui tanggal akses terakhir file dan folder di windows 7?

Saya telah mengamati bahwa "tanggal akses" terakhir dari file atau folder di windows 7 tidak diperbarui secara real time. Itu tidak memperbarui bahkan setelah menyegarkan folder, navigasi kembali ke folder dan mengakses kontennya dan memulai ulang OS. Bagaimana cara memperbarui "tanggal terakhir diakses"?

Menurut dokumentasi Microsoft dari Win32 GetFileTime api:

NTFS menunda pembaruan ke waktu akses terakhir untuk file hingga satu jam setelah akses terakhir. NTFS juga mengizinkan pembaruan waktu akses terakhir untuk dinonaktifkan. Waktu akses terakhir tidak diperbarui pada volume NTFS oleh default.

JanSchejbal benar bahwa masalah dengan waktu akses terakhir adalah kinerja, yang mengubah setiap pembacaan menjadi tulisan tambahan. Microsoft menjelaskan bagaimana file fsutil perintah dapat digunakan untuk mengubah perilaku sistem file.

Tanggal akses terakhir telah dinonaktifkan untuk tujuan kinerja, karena mengubah setiap pembacaan menjadi operasi tulis (sumber) . Anda dapat menggunakan fsutil behavior set disablelastaccess 0 untuk mengaktifkannya, tetapi ini akan merusak kinerja.

  1. Ekspor pekerjaan dari penjadwal tugas
  2. Edit file XML tugas tersebut, tambahkan node ValueQueries di dalamnya. Tambahkan value node di dalamnya. Misalnya: <EventTrigger> ... <ValueQueries> <Value name="EventData">Event/EventData/Data</Value> </ValueQueries> ... </EventTrigger>
  3. Impor pekerjaan lagi dan edit detail email. Anda dapat menggunakan $(EventData) untuk memasukkan detail acara

Harap dicatat bahwa tidak semua versi Windows Server menerima skema XML ini, tapi saya rasa bahwa versi terbaru (WS2012 dan seterusnya) akan baik-baik saja. Misalnya: WS2008 (non R2), yang menggunakan http://schemas.microsoft.com/windows/2004/02/mit/task skema, tidak menyukainya.

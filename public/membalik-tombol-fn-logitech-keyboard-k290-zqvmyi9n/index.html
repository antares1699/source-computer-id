Membalik tombol Fn - Logitech Keyboard K290

Pertanyaan saya mirip dengan ini pertanyaan tetapi situasi di sini adalah di keyboard eksternal (Logitech Comfort Keyboard K290). Apakah saya tetap dapat mengembalikan tombol Fn agar berfungsi seperti yang ditekan sepanjang waktu?

catatan: ini bukan masalah, ini adalah keyboard default. semua tombol F tidak dapat ditekan tanpa tombol Fn karena yang default adalah pintasan (kecerahan - volume - media - ..etc)

Saya menghubungkan keyboard ke Asus K55VD - Windows 8

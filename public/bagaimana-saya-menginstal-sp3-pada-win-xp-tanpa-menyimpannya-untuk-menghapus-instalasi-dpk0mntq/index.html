Bagaimana saya menginstal SP3 pada Win XP tanpa menyimpannya untuk menghapus instalasi?

Saya memiliki file penginstalan jaringan SP3, menginstal melalui penginstalan Windows XP asli yang asli.

Ketika saya menjalankan penginstalan SP3, ia menghabiskan ribuan tahun untuk memperkirakan ruang untuk penginstalan dan yang lainnya. Saya tidak peduli tentang ini dan saya hanya ingin menyelesaikannya dan kembali menjadi sombong mac.

Saya menonaktifkan Pemulihan Sistem.

Bagaimana saya bisa membuatnya menginstal tanpa menyimpan semua omong kosong pencopotan pemasangan ini?

(Saya memiliki file taksi yang diekstrak secara manual, jika itu membantu sama sekali.)

penginstalan SP3 memiliki tombol baris perintah untuk / nobackup dan tidak akan menyimpan info pencopotan pemasangan jika Anda menggunakannya. Ada beberapa opsi praktis lainnya juga, tetapi saya pikir ini yang Anda inginkan. Contoh baris perintah lengkap ada di bawah ini. Cukup lakukan run -> cmd dan kemudian pindah ke direktori tempat distro xp sp3 Anda berada. Kemudian jalankan perintah di bawah ini (tentu saja menyesuaikan dengan patch) dan Anda harus melakukannya dengan baik!

C:\Desktop\WindowsXP-KB936929-SP3-x86-ENU /nobackup

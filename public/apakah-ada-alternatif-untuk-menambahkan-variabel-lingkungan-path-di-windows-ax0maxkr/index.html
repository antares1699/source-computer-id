Pengaturan registri App Paths dapat digunakan untuk mendaftarkan aplikasi yang dapat dieksekusi di seluruh sistem atau untuk pengguna individu seperti menambahkan direktori aplikasi ke PATH. Jadi aplikasi dapat dimulai dengan hanya menulis nama yang dapat dieksekusi di baris perintah.
Untuk detail lebih lanjut tentang App Paths periksa artikel ini http://msdn.microsoft.com/en-us/library/windows/desktop/ee872121.aspx atau kunci registri HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths

Setiap aplikasi dapat dibuat portabel saat dikembangkan. Tidak perlu menggunakan variabel PATH.

Tekniknya adalah mendapatkan jalur peluncuran aplikasi dan menggunakannya untuk meluncurkan modul aplikasi lain yang relatif ke jalur peluncuran ini. dll jalur pencarian sudah termasuk jalur aplikasi jadi ini berlaku untuk executable lain atau modul / data yang membuat aplikasi.

Satu-satunya solusi lain adalah meletakkan semua file di direktori yang sama dengan file yang dapat dieksekusi, sehingga jalur tidak diperlukan untuk mengakses file. Atau tentukan pintasan untuk eksekusi dengan nama jalur default.

Solusi ini digunakan oleh banyak aplikasi, untuk alasan ini: hindari menambahkan dirinya sendiri ke PATH.

dapatkah perangkat lunak dikembangkan dengan cara yang berbeda dan mengelolanya dengan teknik lain?

Iya. Apa yang akan saya lakukan dalam kasus ini, (jika Anda memiliki program untuk diinstal atau yang lainnya, tetapi Anda tidak dapat mengakses variabel Path) adalah membuatnya agar biner (program yang dapat dieksekusi) diinstal ke folder C:\Windows atau C:\Windows\System32. Kemudian Anda dapat memanggilnya dari baris perintah atau secara terprogram, hanya dengan program, program /option, atau program -o, atau apa pun itu.

Akan lebih optimal untuk menambahkannya ke variabel sistem %Path%, dan jika menargetkan Windows Vista, 7, atau 8, Anda dapat membuatnya meminta pengguna untuk meningkatkan hak istimewa dan mengizinkannya untuk mengaksesnya. Tetapi jika itu bukan pilihan, maka Anda setidaknya harus bisa melakukan apa yang saya katakan di atas.

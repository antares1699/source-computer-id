Bagaimana cara menonaktifkan "ThinkPad Zoom"?

Saya memiliki ThinkPad T61p yang menjalankan Vista 32-bit SP2. Saya banyak memprogram di Eclipse, dan saya menemukan bahwa ketika saya mencoba mengetikkan hotkey penyelesaian kode (Ctrl-Space), saya sering secara tidak sengaja menekan kombinasi ThinkPad Zoom (Fn-Space). Ini tidak hanya menyebabkan layar beralih ke resolusi 800x600, tetapi juga menukar SEMUA aplikasi saya yang sedang berjalan kembali, dan menggagalkan alur pemikiran apa pun yang saya miliki.

Adakah yang tahu cara untuk menonaktifkan kombinasi tombol Fn khusus ini?

Jika fitur ini sama sekali tidak berguna dan Anda ingin menghapusnya secara permanen, Anda selalu dapat mengunjungi Vista Panel Kontrol> Program dan Fitur , dan hapus instalan Kaca Pembesar Layar Penuh Thinkpad utilitas.

Itu akan menonaktifkan kombo keyboard Fn + Space.

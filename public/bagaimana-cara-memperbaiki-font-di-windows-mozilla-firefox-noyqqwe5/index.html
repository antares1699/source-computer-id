Bagaimana cara memperbaiki font di Windows Mozilla Firefox?

Setiap kali saya memformat komputer saya, saya biasanya hanya menyalin / menempel semua file .ttf dan .fon ke dalam folder sistem Fonts.

Saya tidak dapat memastikan 100%, tetapi saya pikir setelah saya melakukannya, Firefox berhenti menampilkan beberapa font situs web dengan benar, dan maksud saya font default biasa, seperti Arial, Helvetica, dll.

Saya tidak mengalami masalah ini di program lain, selain Firefox.

Berikut adalah screenshot dari apa Digg lihat di laptop saya sekarang:

Kelihatannya aneh, fontnya entah bagaimana kacau.

Bagaimana saya memperbaikinya? Saya menggunakan Windows 7, Firefox 3.6.10.

Masalah khusus ini disebabkan oleh penginstalan versi font Helvetica (sangat umum digunakan di web) yang tidak dapat ditampilkan oleh Windows atau Firefox. Untuk mengonfirmasi, yang harus Anda lakukan adalah menempelkan teks berikut di bilah alamat Firefox Anda dan tekan enter:

data:text/html,
Helvetica
   <font face="Helvetica" size="25">abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ</font>

Helvetica Neue
  <font face="Helvetica Neue" size="25">abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ</font>

Jika ditampilkan sebagai simbol aneh, maka Anda bisa yakin itu masalah Anda.

Untuk memperbaikinya, yang harus Anda lakukan adalah pergi ke% WINDIR% \ Fonts dan hapus semua font bernama Helvetica.

Jika karena alasan tertentu, Windows tidak mengizinkan Anda, Anda harus pergi ke registri, arahkan ke HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts dan kemudian hapus semua entri yang memiliki Helvetica pada namanya.

Jika Anda menggunakan Windows 7 64-bit, Anda juga harus menghapus entri Helvetica dari kunci registri berikut.

HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows NT\CurrentVersion\Fonts

Setelah ini, Anda hanya perlu me-reboot komputer Anda dan font akan ditampilkan dengan benar lagi. Setidaknya itu memperbaikinya untukku.

Saya baru saja berjuang dengan masalah yang sama (meskipun dengan Google Chrome) dan menemukan topik ini. Karena saya menggunakan PC di tempat kerja, saya tidak memiliki hak istimewa untuk mengedit registri. Saya menemukan solusi lain, yang mungkin hanya berfungsi di Chrome, tetapi setidaknya itu memperbaiki masalah untuk saya. Jadi meskipun ini bukan tentang Firefox, saya masih ingin membagikannya, jika ada orang lain yang juga membahas topik ini.

  1. Buka Google Chrome
  2. Buka tab baru dan masukkan berikut ini di bilah alamat:

chrome: // flags / # disable-direct-write

  1. Klik "aktifkan" untuk mengaktifkan sakelar ini, menonaktifkan penggunaan Microsoft DirectWrite oleh Google Chrome.
  2. Tutup dan buka kembali Chrome.

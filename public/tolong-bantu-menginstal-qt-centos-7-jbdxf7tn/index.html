Tolong Bantu menginstal Qt centos 7

Saya memiliki 2 versi python, python3.6 diinstal pada virtualenv untuk penggunaan pengembangan, saya mencoba menginstal paket PyQt dan Qt untuk digunakan dengan python3.6 saja,

Saya telah menginstal PyQt5 menggunakan sudo pip3.6 install pyqt5, Instruksi yang saya ikuti mengatakan bahwa Qt disertakan dengan PyQt5, Juga dalam paket pyqt5 di /usr/lib64/python3.6/site-packages/PyQt5 Saya melihat folder Qt di dalamnya , Tetapi ketika saya meletakkan qmake --version untuk memeriksa versi qt, saya mendapat kesalahan berikut:

qmake --version

bash: qmake: command not found...

Similar command is: 'make'

Ketika menggunakan:

make --version

Ini memberikan yang berikut:

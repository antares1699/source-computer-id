Di manakah "Item Terbaru" dan "Aplikasi yang Paling Sering Digunakan" di Windows 8?

Bukankah Windows 8 melacak aplikasi yang saya gunakan seperti versi Windows sebelumnya yang secara otomatis mencantumkan perangkat lunak yang paling sering digunakan di menu Mulai?

Saya juga tidak bisa menemukan fitur "Dokumen Terbaru". Apakah ini tidak tersedia di Windows 8 atau saya melewatkan sesuatu?

Fitur tersebut telah dihapus, karena tidak ada lagi menu mulai.

Pilihan Anda adalah:

  • Daftar lompatan dokumen / file terbaru aplikasi masih tersedia, tetapi hanya jika disematkan ke bilah tugas (karena tidak ada lagi menu mulai)
  • Anda dapat mengatur pintasan ke "Terbaru" di folder pengguna Anda, yang mencantumkan semua yang telah Anda akses.
  • Pulihkan menu mulai, menggunakan aplikasi pihak ke-3, yang menyertakan dokumen terbaru, seperti di Windows 7. Pengganti menu Mulai meliputi:
    • ViStart (Gratis!)
    • Shell Klasik (Gratis!)
    • Pokki (Gratis!)
    • StartMenu7 (gratis dengan versi pro untuk dijual)
    • Mulai8 ($ 5)

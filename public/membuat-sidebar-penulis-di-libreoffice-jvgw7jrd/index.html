Membuat sidebar penulis di LibreOffice

Saya mencoba membuat sidebar di dokumen penulis Libreoffice, persis seperti berikut ini template .

Bilah sisi harus memiliki warna latar belakang yang berbeda dan bagian teks yang terpisah, tetapi disejajarkan secara vertikal dengan konten di sebelah kanan (bilah sisi akan digunakan untuk judul, sedangkan bagian kanan akan digunakan untuk konten yang sebenarnya).

Tabel dua kolom tampaknya cocok untuk ini, kolom kiri menampung tajuk dan yang kanan memegang konten, tetapi warna latar belakang sidebar harus mengisi seluruh bagian kiri dokumen (strip vertikal) tanpa margin, jadi saya harus mengatur margin menjadi nol. Masalahnya adalah, saya membutuhkan margin untuk teks, dan saya membutuhkan warna latar belakang tabel untuk mengabaikannya.

Saya mengerti bahwa ini mungkin tidak cocok untuk Writer (dan saya dapat membuat dokumen yang sama tanpa masalah di HTML / CSS atau editor grafis seperti Photoshop), tetapi saya sangat membutuhkannya di writer (dokumen ini untuk orang lain yang menang) menerima versi HTML atau Photoshop).

Berikut solusi untuk LibreOffice:

Saya merekomendasikan untuk meletakkan bingkai teks (LibreOffice 4.4) atau objek LibreOffice Draw OLE (persegi panjang) ke dalam header dari halaman pertama. Anda dapat mengubah ukuran persegi panjang atau bingkai teks sehingga menutupi seluruh area margin halaman kiri, dan Anda dapat menetapkan warna kustom untuk itu. Karena bingkai persegi panjang / teks terletak di tajuk, itu akan muncul secara otomatis di setiap halaman baru.

Untuk memasukkan konten ke dalam area margin itu, saya sarankan masukkan bingkai teks ke dalam area konten utama , berlabuh ke paragraf tertentu (agar tetap sejajar dengan konten), dan pindahkan bingkai teks ke kiri ke dalam area margin . Anda bahkan dapat membuat gaya bingkai berdasarkan bingkai teks pertama yang dibuat secara manual, sehingga Anda menambahkan gaya ke semua bingkai dalam satu langkah.

Saya baru saja memperhatikan bahwa LO 4.4 tampaknya berperilaku buggy saat mencoba menyetel warna latar belakang untuk objek Draw OLE; sebagai solusinya, Anda dapat menggunakan bingkai teks sebagai gantinya.

  1. Di dokumen baru, masukkan header default;

  2. Sisipkan "objek margin":

    • Untuk LibreOffice 4.4:

      • klik di header, pilih Insert -> Frame...;

      • terapkan pengaturan berikut (yang paling penting adalah menambatkan bingkai ke paragraf, sehingga muncul di setiap halaman; juga, berhati-hatilah agar Ukuran Otomatis dinonaktifkan; Anda dapat mengatur lebar / tinggi sesuai dengan kebutuhan Anda atau mengubah ukuran margin setelahnya menggunakan mouse ):

    • untuk LibreOffice dengan versi <4.4:

      • klik di header, pilih Insert -> Object -> OLE Object -> LibreOffice 4.4 Drawing, klik baik ;

      • Jendela menggambar muncul; jangan mengeditnya, biarkan saja menggunakan ESC kunci.

        Sekarang, layarnya terlihat seperti ini:

      • Pindahkan area gambar menggunakan persegi panjang hijau kecil sehingga menutupi margin kiri lengkap (pertama, atur lebarnya; setelah itu, pindahkan ke margin halaman):

    Perhatikan bahwa di area utama, konten akan secara otomatis mengenali area gambar header, jadi Anda bahkan tidak perlu menyesuaikan margin halaman (Anda dapat melakukannya untuk memiliki beberapa padding antara area margin dan konten).

  3. Isi area margin dengan warna apa saja

    • untuk bingkai teks:
      • pilih bingkai
      • aktifkan toolbar Drawing Object properties (menggunakan Menu View -> Toolbars) atau klik kanan -> Frame... -> Area;
      • pilih warna latar belakang;
    • untuk objek Draw OLE:
      • pilih objek OLE sehingga persegi panjang hijau kecil terlihat;
      • klik kanan di mana saja dalam objek OLE
      • pilih Object... dari menu konteks, pilih Background;

  4. Klik ke area konten utama, tambahkan beberapa teks, dan sisipkan bingkai teks (saya telah mengatur margin halaman kiri menjadi 8 cm untuk memisahkan konten dari margin - Anda harus menyesuaikan posisi margin karena bergerak saat mengubah margin halaman):

  5. Sekarang, jika Anda memasukkan konten tambahan sebelum bingkai teks, itu akan tetap berada di sebelah paragraf tempatnya berlabuh:

Anda dapat mengatur warna latar belakang sel individu.

LibreOffice

  1. Sisipkan tabel dengan 1 baris dan 2 kolom
  2. Pilih kolom pertama
  3. Tabel -> Properti Tabel -> Latar Belakang -> Sebagai "Warna" -> Untuk "Sel"
  4. Pilih Warna
  5. Tekan OK"

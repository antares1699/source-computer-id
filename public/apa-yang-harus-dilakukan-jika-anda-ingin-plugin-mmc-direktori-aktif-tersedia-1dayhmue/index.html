Apa yang harus dilakukan jika Anda ingin plugin MMC Direktori Aktif tersedia?

Saya tidak mengerti atau terlalu memikirkan tentang direktori aktif. Saat ini saya hanya ingin tahu bagaimana ikon berikut muncul di desktop saya. Apa yang perlu diinstal di mesin saya untuk mendapatkannya.

Info lebih lanjut: Ikon khusus itu memanggil dsa.msc yang berada di system32 direktori windows.

Untuk mendapatkan Plugin MMC Direktori Aktif (dsa.msc), Anda memerlukan yang berikut ini:

Windows XP SP1 dan yang lebih baru atau Windows Server 2003: adminpak.msi

Windows Vista: Alat RSAT untuk Windows Vista

Windows 7: Alat RSAT untuk Windows 7

Windows 10: Alat RSAT untuk Windows 10 v1.2

Anda perlu menginstal Alat Admin Server Jarak Jauh (RSAT) dari Microsoft. Pemasangannya miring. Anda mendownload dan menginstal perangkat lunak, LALU, masuk ke Add Remove Programs, Turn Windows Features On / Off dan periksa alat yang ingin Anda aktifkan pada PC Anda.

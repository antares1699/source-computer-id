Buat Windows 7 masuk mode tidur setelah x detik

Ketika saya memulai pekerjaan pemrosesan lama sebelum tidur sendiri, saya sering lari

shutdown -s -t 3600

sehingga komputer mati secara otomatis setelah 3600 detik.

Bagaimana melakukan hal yang sama untuk meminta komputer tidur setelah x detik?

Saya mencoba

shutdown -h -t 60

tetapi 1) itu bahkan tidak berhasil dan bagaimanapun 2) ini akan membuat komputer pergi ke hibernasi, yang tidak saya inginkan (saya lebih suka mode tidur).

catatan: jawaban ini tidak memecahkan masalah karena tidak memungkinkan untuk menentukan waktu sebelum tidur.

Bagaimana cara membuat Windows 7 masuk mode tidur setelah x detik?

Gunakan file batch berikut, dan jalankan sebagai Administrator

@echo off
rem disable hibernate
powercfg -hibernate off
rem wait x seconds, eg 1 hour
timeout 3600 /nobreak
rem sleep
%windir%\System32\rundll32.exe powrprof.dll,SetSuspendState 0,1,0

Bacaan lebih lanjut

  • Indeks A-Z dari baris perintah CMD Windows
  • Daftar kategori perintah CMD Windows
  • powercfg.dll - Kontrol pengaturan daya, konfigurasikan mode Hibernasi / Siaga.
  • waktu habis - Penundaan eksekusi selama beberapa detik atau menit, untuk digunakan dalam file batch.

Satu baris (berdasarkan jawaban yang diterima DavidPostill):

timeout /T 3600 & rundll32.exe powrprof.dll,SetSuspendState 0,1,0

Untuk memastikannya berhasil, Anda bisa mencobanya seperti ini:

timeout /t 10 & notepad

Notepad akan terbuka setelah 10 detik.

Catatan penting: Jika Anda menginstal git di komputer Anda, folder bin-nya mungkin ada di PATH, dan melakukan timeout di baris perintah akan berjalan C:\Program Files\Git\usr\bin\timeout.exe bukan default C:\Windows\System32\timeout.exe! Dalam hal ini Anda harus menyertakan jalur lengkap timeout, untuk menggunakan alat batas waktu Windows.

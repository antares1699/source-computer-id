Apa keuntungan menjalankan server DNS di rumah?

Saya berencana menginstal fedora 15 dan beberapa server (http, ftp, svn, mail ++) tetapi saya bertanya-tanya apa keuntungannya dengan menginstal dan menjalankan server DNS di kotak yang sama?

Beberapa keunggulannya antara lain:

  • Server DNS yang dilokalkan dapat memberikan kinerja yang lebih baik karena berhubungan langsung dengan server root

  • Sumber daya hanya didedikasikan untuk komputer yang Anda izinkan untuk menggunakan server DNS (tidak seperti server DNS ISP yang akan melayani sebagian besar pelanggannya dan mungkin membuang catatan lama lebih cepat {re-querying a zone that not cached meningkatkan waktu respons})

  • Anda dapat menambahkan zona khusus untuk tujuan lain (mis., Intranet)

  • Keamanan bisa lebih baik karena orang lain tidak dapat merusak cache Anda (dengan asumsi Anda mengambil langkah yang wajar untuk mengamankan sistem Anda)

  • Jika server DNS ISP mati (biasanya sangat jarang), itu tidak memengaruhi Anda

Saya menjalankan server DNS saya sendiri di rumah, dan juga mengaturnya untuk klien saya yang menggunakan server Unix.

Bagaimana cara mengisi seri besar (lebih dari 10.000 baris) di Microsoft Excel tanpa menyeret atau memilih sel?

Saya memiliki lebih dari 10.000 baris data. Saya perlu melakukan beberapa penghitungan pada satu kolom data. Saya telah melakukan penghitungan dalam satu sel dan sekarang perlu mengisi rumus untuk penghitungan dan semua sel.

Biasanya saya hanya memilih sel menggunakan keyboard dan kemudian menekan Ctrl + D untuk mengisi sel. Namun, sekarang saya harus melakukan ini berulang kali, dan sulit untuk terus memilih 5000 baris setiap file. Apakah ada cara yang lebih sederhana di mana saya bisa menentukan rentang menggunakan keyboard? Bahkan mungkin memiliki makro karena semua file memiliki panjang tetap

Juga karena saya mengalami kesulitan dalam menggunakan mouse karena alasan medis dan oleh karena itu saya belum melakukan tes solusi khusus ini yang melibatkan mengklik gagang isian yang sangat kecil. Mungkin ada solusi paralel untuk ini menggunakan keyboard atau mungkin saya dapat membuat makro yang secara otomatis akan mengklik pegangan isi sel yang dipilih - mohon saran jika ada cara.

Dengan asumsi data ada di kolom A dan rumus di kolom B:

  1. Masukkan rumus pada B1
  2. Salin sel B1
  3. Menavigasi dengan tombol panah ke sel mana pun di Kolom A
  4. tekan Ctrl + Panah Bawah
  5. tekan Panah Kanan (Anda sekarang harus berada di sel kosong di bagian bawah kolom B)
  6. tekan Ctrl + Bergeser + Panah Atas
  7. Tempel ( Ctrl + V. )

Memperbarui:

Versi Excel yang lebih baru akan secara otomatis menggunakan rumus array untuk diisi ketika ada baris data baru sehingga Anda tidak perlu melakukannya secara manual

Dimulai dengan pembaruan September 2018 untuk Office 365, rumus apa pun yang dapat mengembalikan beberapa hasil akan secara otomatis menumpahkannya ke bawah, atau ke sel tetangga. Perubahan perilaku ini juga disertai dengan beberapa fungsi array dinamis baru. Rumus larik dinamis, baik menggunakan fungsi yang sudah ada maupun fungsi larik dinamis, hanya perlu dimasukkan ke dalam satu sel, lalu dikonfirmasi dengan menekan Enter. Sebelumnya, rumus larik warisan mengharuskan terlebih dahulu memilih seluruh rentang keluaran, lalu mengonfirmasi rumus dengan Ctrl + Bergeser + Memasukkan . Mereka biasanya disebut sebagai rumus CSE.

Panduan dan contoh rumus array

Rumus array memiliki banyak keunggulan:

Mengapa menggunakan rumus array?

Jika Anda memiliki pengalaman menggunakan rumus di Excel, Anda tahu bahwa Anda dapat melakukan beberapa operasi yang cukup canggih. Misalnya, Anda dapat menghitung total biaya pinjaman selama beberapa tahun tertentu. Anda bisa menggunakan rumus array untuk melakukan tugas-tugas kompleks, seperti:

  • Hitung jumlah karakter yang terdapat dalam rentang sel.
  • Menjumlahkan hanya angka yang memenuhi kondisi tertentu, seperti nilai terendah dalam rentang atau angka yang berada di antara batas atas dan bawah.
  • Jumlahkan setiap nilai ke-n dalam satu rentang nilai.

Rumus array juga menawarkan keuntungan berikut:

  • Konsistensi: Jika Anda mengklik salah satu sel dari E2 ke bawah, Anda akan melihat rumus yang sama. Konsistensi tersebut dapat membantu memastikan akurasi yang lebih baik.
  • Keamanan: Anda tidak dapat menimpa komponen rumus array multisel. Misalnya, klik sel E3 dan tekan Delete. Anda harus memilih seluruh rentang sel (E2 hingga E11) dan mengubah rumus untuk seluruh larik, atau membiarkan larik apa adanya. Sebagai tindakan keamanan tambahan, Anda harus menekan Ctrl + Bergeser + Memasukkan untuk mengonfirmasi perubahan rumus.
  • Ukuran file lebih kecil: Anda sering kali dapat menggunakan rumus array tunggal daripada beberapa rumus perantara. Misalnya, buku kerja menggunakan satu rumus array untuk menghitung hasil di kolom E. Jika Anda telah menggunakan rumus standar (seperti = C2 * D2, C3 * D3, C4 * D4…), Anda akan menggunakan 11 rumus berbeda untuk menghitung hasil yang sama.

Panduan dan contoh rumus array

Rumus array juga lebih cepat karena pola aksesnya sudah diketahui. Sekarang alih-alih melakukan 11 perhitungan berbeda secara terpisah, mereka dapat vektorisasi dan dilakukan secara paralel, menggunakan beberapa inti dan unit SIMD di CPU

Misalnya jika Anda ingin sel di kolom D berisi produk sel di B dan C, dan kolom E berisi jumlah B dan C, alih-alih menggunakan D1 = B1*C1 dan E1 = B1 + C1 dan menyeret ke bawah, Anda cukup meletakkan rumus di bawah ini di D1 dan E1 masing-masing dan tekan Ctrl + Bergeser + Memasukkan

=B:B*C:C
=B:B + C:C

Di lembar Google Anda harus menggunakan ARRAYFORMULA alih-alih pintasan seperti di Excel

=ARRAYFORMULA(B:B*C:C)
=ARRAYFORMULA(B:B + C:C)

Setelah itu rumusnya akan otomatis diterapkan ke seluruh kolom segera tanpa menyeret apapun. X:X mengacu pada kolom X. Jika Anda ingin memasukkan hanya dari baris 3 sampai akhir gunakan X3:X. Atau jika Anda ingin menghitung hasil hanya untuk sel dari X3 ke X10003 maka gunakan X3:X10003. Sintaks yang sama dapat digunakan untuk baris, yaitu 1: 1 untuk baris 1. Berikut adalah contoh lain dari dokumentasi Excel yang menghitung produk di setiap kolom

Rumus array adalah alat yang sangat ampuh. Namun gunakan dengan hati-hati. Setiap kali Anda perlu mengeditnya Anda tidak boleh lupa untuk menekan Ctrl + Bergeser + Memasukkan

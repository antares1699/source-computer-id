Cadangkan ke folder tertentu dengan Windows Backup

Saat mengatur Windows Backup saya mendapatkan layar berikut:

Seperti yang Anda lihat, tidak ada opsi untuk mencadangkan ke folder tertentu di salah satu drive ini. Apakah ada cara untuk memaksa backup masuk ke folder tertentu pada drive yang saya pilih?

Memperbarui: Saya baru saja membuat file Cadangan partisi eksternal saya sehingga saya dapat mempertahankan pemisahan. Saya lebih suka folder sederhana karena lebih fleksibel tetapi bisa digunakan.

Memaksakan ke dalam folder bersarang tidak dimungkinkan dengan UI default.

Satu-satunya pilihan adalah memindahkan folder setelah Anda membuat cadangan. Semua informasi ada di folder itu sendiri.

Faktanya, ini adalah rekomendasi Microsoft untuk membuat cadangan untuk berbagi file jaringan. Cadangan hanya akan memungkinkan ada satu gambar Cadangan per berbagi file. Mengganti nama folder setelah fakta memungkinkan Anda menyimpan beberapa versi gambar per berbagi file. Ada entri blog tentang membuat skrip ini menggunakan wbadmin, yang merupakan baris perintah Cadangan di blog Penyimpanan Windows beberapa waktu lalu. Ini juga yang saya lakukan dengan mesin saya.

Anda dapat melakukannya dengan

  • memilih "Simpan di jaringan"
  • menyediakan link seperti ini misalnya: \\YourPcName\Backup (lakukan ini melalui opsi Browse)
  • memastikan folder Cadangan dibagikan Baca / Tulis

Perhatikan bahwa Anda memang memerlukan kata sandi untuk akun Windows Anda karena Anda harus memberikannya sebelum dapat berfungsi.

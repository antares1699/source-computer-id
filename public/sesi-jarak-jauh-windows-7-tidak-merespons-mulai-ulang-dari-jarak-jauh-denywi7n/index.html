Sesi jarak jauh Windows 7 tidak merespons, mulai ulang dari jarak jauh?

Saya memiliki desktop Windows 7 yang saya masuki dari jarak jauh. Sesuatu terjadi dan salah satu program terhenti, dan menyebabkan sesi terhenti. Saya dapat melakukan remote ke sistem dan masuk ke layar login, tetapi setelah mengetikkan kata sandi, saya hanya melihat layar login berwarna biru dengan pinwheel "harap tunggu".

Apakah ada cara untuk merestart komputer atau sesi pengguna dari jarak jauh? Sistem hanya memiliki satu pengguna, yang login. Saya sedang berlibur sejauh 200 mil, jadi solusi jarak jauh adalah yang terbaik. Terima kasih untuk bantuannya.

Anda dapat terhubung ke prompt perintah dari jarak jauh dengan psexec . jalankan command prompt sebagai admin di localhost Anda, dan masukkan:

psexec \\remotecomputername -u username -p password cmd

psexec akan terhubung ke komputer jarak jauh dan menampilkan prompt perintah yang dijalankan di atasnya. lalu masuk

shutdown -r -t 1

untuk mem-boot ulang mesin. koneksi psexec Anda akan berhenti dalam prosesnya.

Saya biasanya mulai melakukan ping ke host (dengan -t99 di windows) sebelum mengeluarkan shutdown, jadi saya bisa melihatnya turun, dan kembali naik.

Dengan asumsi Anda menggunakan layanan Windows Remote Desktop yang disertakan dengan hampir semua komputer windows, secara bersamaan tekan tombol "Ctrl", "Alt", dan "End". Ini akan memunculkan dialog yang akan memberi Anda opsi shutdown dan restart.

Metode lain untuk melakukan hal seperti itu (meskipun saya tidak yakin ini dapat diselesaikan dari layar masuk) adalah membuka prompt Jalankan (dengan menekan tombol "Windows" dan "R") dan kemudian memasukkan "shutdown -f -r -t 1 "ke dalam prompt.

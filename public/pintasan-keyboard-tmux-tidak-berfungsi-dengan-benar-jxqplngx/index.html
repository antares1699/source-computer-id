Pintasan keyboard tmux tidak berfungsi dengan benar

Saya sudah mulai menggunakan tmux baru-baru ini, dan mengubah binding sehingga itu akan merespons C-a, bukan C-b. Ini adalah baris di file .tmux.conf untuk melakukan ini:

unbind C-b
set -g prefix C-a

Tapi, saya tidak bisa membuat C-a bekerja untuk pergi ke awal baris. C-a, C-a tidak berfungsi, begitu pula C-a, a. Selain itu, pintasan lain seperti C-a, <beberapa tombol panah> untuk berpindah antar panel tidak berfungsi. Apa yang saya lewatkan?

Terima kasih!

Anda adalah bagian dari perjalanan ke sana. unbind C-b menonaktifkan awalan default, dan set -g prefix C-a menyiapkan C-a sebagai awalan perintah baru seperti layar. Untuk mendapatkan perilaku layar lainnya, tambahkan beberapa baris lagi:

set -g prefix C-a
unbind C-b

# C-a a should send ctrl-a to the underlying shell (move to start of line)
bind-key a send-prefix    

# C-a C-a
bind-key C-a last-window

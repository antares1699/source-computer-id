Bisakah saya menghapus opt / local / bin?

Saya telah menghapus Macports karena konflik perpustakaan dan menggunakan Homebrew sekarang. Dan saya mengalami masalah dengan Openssl dan di folder opt/local/bin saya, ada openssl, oldOpenssl dan oldOpenssl2 yang ingin saya hapus, tetapi saya telah membaca bahwa opt/local/bin adalah direktori macports saya. Hal buruk apa yang akan terjadi jika saya menghapusnya?

Anda bisa menghapusnya.

/opt tidak ada pada penginstalan bersih OS X. Jika Anda ingin menghapus MacPorts sepenuhnya, Anda tetap harus menghapus penginstalan port terlebih dahulu. Anda nanti dapat menginstal sebagian besar aplikasi lagi, melalui Homebrew, yang juga memiliki OpenSSL.

Untuk menghapus semua port yang saat ini diinstal - yang mungkin tidak dapat Anda lakukan karena Anda telah menghapus MacPorts:

sudo port -fp uninstall --follow-dependents installed

Ini juga akan menghapus binari pada /opt/local/bin. Sisanya adalah pembersihan direktori tambahan, termasuk /opt/local, yang hanya merupakan awalan MacPorts. Tidak mungkin ada orang lain yang akan menulis file di sana.

Ini semua dijelaskan di instruksi pencopotan resmi demikian juga. Penghapusan penuh termasuk menghapus beberapa direktori lain juga, seperti:

/Applications/DarwinPorts
/Applications/MacPorts
/Library/LaunchDaemons/org.macports.*
/Library/Receipts/DarwinPorts*.pkg
/Library/Receipts/MacPorts*.pkg
/Library/StartupItems/DarwinPortsStartup
/Library/Tcl/darwinports1.0
/Library/Tcl/macports1.0
~/.macports

Memulai MacBook Pro dalam mode 32 bit

Mengikuti dari Apakah Macbook Pro 64-bit atau 32-bit baru? . MacBook Pro baru (Maret 2011) sekarang tampaknya mulai dalam mode 64bit daripada mode 32bit. Terima kasih telah memberi tahu kami tentang Apple!

Ini menyebabkan kami bermasalah dengan klien Cisco VPN kami karena menggunakan ekstensi kernel 32-bit yang secara alami tidak berfungsi saat di-boot ke mode 64-bit.

Menahan 3 & 2 akan memulai mesin ke mode 32bit dan klien VPN berfungsi dengan baik, tetapi saat kami mem-boot ulang Mac, mesin akan mulai lagi ke mode 64bit.

Bagaimana kami bisa memastikan bahwa Mac mulai dalam mode 32bit.

(Tentu mendapatkan klien Cisco VPN 64bit adalah solusi yang lebih baik ... suatu hari nanti)

Saya berharap sudo nvram boot-args=arch=x86 akan berhasil. Anda juga harus dapat mengedit /Library/Preferences/SystemConfiguration/com.apple.Boot.plist (yang merupakan daftar properti format XML secara normal; jika tidak, lihat /Library/Preferences/SystemConfiguration/com.apple.Boot.plist) dan mengatur kunci Kernel Flags ke arch=x86. Perhatikan bahwa kehilangan konten NVRAM akan menghapus sebelumnya dan pembaruan sistem Apple mungkin akan menimpa terakhir.

http://osxbook.com/blog/2009/08/31/is-your-machine-good-enough-for-snow-leopard-k64/ mendekatinya dari arah lain (menjadikan Mac boot dalam mode 64-bit secara default); cukup gunakan x86 untuk mode 32-bit.

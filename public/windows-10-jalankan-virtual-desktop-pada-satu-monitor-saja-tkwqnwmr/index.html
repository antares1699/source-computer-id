Windows 10: Jalankan virtual desktop pada satu monitor saja?

Saat ini saya memiliki monitor eksternal yang terhubung ke laptop saya. Saya memiliki empat desktop virtual yang menjangkau kedua layar.

Saya ingin monitor eksternal saya memiliki empat desktop virtual, tetapi hanya memiliki satu untuk laptop saya. Dengan cara ini saya dapat mengubah desktop virtual di monitor eksternal saya, tetapi laptop saya tetap di layar yang sama.

Apakah saat ini mungkin di Windows 10?

EDIT: Saat ini saya menggunakan desktop virtual Windows 10 bawaan. Misalnya, saya ingin pergi dari ini ...

  • Laptop: Desktop 1
  • Monitor: Desktop 1

untuk ini...

  • Laptop: Desktop 1
  • Monitor: Desktop 2 atau Desktop 3 atau Desktop 4

Bagaimana cara mengubah XP VM pada virtualbox agar memiliki dua CPU tanpa harus menginstal ulang XP? [duplikat]

Saya menginstal Windows XP di Virtualbox di bawah Ubuntu.

Setelah menginstal XP saya ingin mengubah VM menjadi dua cpu.

Bagaimana melakukannya tanpa menginstal ulang XP?

Anda hanya perlu mengubah HAL Anda. Ada cara yang "lebih aman" dengan meningkatkan perangkat "komputer" di pengelola perangkat atau Anda dapat mengganti file secara manual.

Sebenarnya, cukup ubah jumlah CPU dan HAL uni-prosesor harus dapat mengenali inti yang ditambahkan dan memicu deteksi perangkat keras baru dan dengan demikian mengubah HAL.

Lihat KB309283 .

Edit:

Untuk benar-benar menggunakan 2 inti, Anda perlu mengaktifkan opsi lain, seperti "Aktifkan I / O APIC", untuk menggunakan lebih dari satu inti. Lihat manual .

HAL atau lapisan abstraksi perangkat keras adalah beberapa kode yang memungkinkan kernel windows berjalan pada arsitektur yang berbeda. Biasanya penginstal akan mendeteksi konfigurasi perangkat keras Anda dan menginstal file hal.dll yang benar ke direktori system32 Anda. Jika Anda "memutakhirkan" CPU Anda, karena kompatibilitas yang luar biasa di keluarga X86, HAL lama Anda akan tetap berfungsi. Menurut artikel KB di atas, HAL lama juga harus mendeteksi perubahan pada perangkat keras dan memicu proses deteksi perangkat keras untuk memasang HAL baru. Jika itu tidak terjadi, Anda dapat melakukannya secara manual dengan mengikuti petunjuk di KB.

Berdasarkan posting blog ini yang harus Anda lakukan adalah

  1. Klik kotak centang "Aktifkan IO APIC" di pengaturan Sistem mesin tamu kotak virtual Anda.

  2. Jalankan perintah berikut di prompt perintah di dalam tamu windows:

    rundll32 syssetup,SetupInfObjectInstallAction ACPIAPIC_MP_HAL 128 %windir%\inf\hal.inf

  3. Nyalakan ulang tamu dua kali seperti yang diminta.

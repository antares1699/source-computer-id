Apa URL baru untuk bookmark "Pencarian Cepat" Gambar Google di Firefox?

Bookmark di atas digunakan untuk memungkinkan saya mencari Gambar Google dengan mengetik goimg mysearchterms di bilah alamat, di Firefox.

Sekarang, saya mengerti.

Jelas sekali, Google telah mengubah sesuatu. Tapi, saya tidak tahu apa. Selain itu, dari format URL saat ini, saya tidak dapat menguraikan bit mana yang diperlukan untuk melakukan pencarian dari bilah alamat dan mana yang halus.

URL di bawah ini adalah untuk istilah pencarian berikut, dengan urutan: "google", "myspace", "twitter".

http://images.google.com/search?tbm=isch&hl=en&source=hp&q=google&gbv=2&oq=google&aq=f&aqi=g10&aql=&gs_sm=e&gs_upl=3078l3593l0l4000l6l4l0l0l0l0l203l328l0.1.1l2l0

http://images.google.com/search?q=myspace&tbm=isch&hl=en&source=hp&gbv=2&gs_sm=e&gs_upl=3078l3593l0l4000l6l4l0l0l0l0l203l328l0.1.1l2l0&oq=mys&aq=0&aqi=g10&aql=

http://images.google.com/search?q=twitter&tbm=isch&hl=en&source=hp&gbv=2&gs_sm=e&gs_upl=3078l3593l0l4000l6l4l0l0l0l0l203l328l0.1.1l2l0&oq=twitter&aq=f&aqi=g10&aql=

Bagian apa yang perlu saya tambahkan ke "Lokasi" Bookmark, dan bit mana yang dapat ditinggalkan?

Pindahkan semua jenis file ke folder baru di windows 7

Adakah yang bisa membantu dalam hal ini.

Saya memiliki file mp3 di seluruh toko dan karena itu banyak duplikat.

Yang saya butuhkan adalah fungsi baris perintah yang akan memindahkan semua file jenis mp3 ke folder baru. Saya tidak khawatir jika struktur file di folder baru dipertahankan atau tidak.

Saya dapat dengan mudah menyalin menggunakan salinan c: *. Mp3 / s c: \ mp3 misalnya, tetapi karena jumlah file yang saya miliki saya akan kehabisan ruang disk untuk menyalinnya, oleh karena itu harus dipindahkan.

Coba perintah ini dari folder tingkat atas tempat Anda memiliki file mp3.

forfiles /M *.mp3 /C "cmd /c move @file C:\music"

Perhatikan bahwa jika Anda memiliki file dengan nama duplikat, memindahkan semuanya ke dalam satu folder bukanlah hal yang benar untuk dilakukan. Ini akan menyebabkan kehilangan data.

Ada juga cara untuk memindahkan semua file yang cocok dengan kriteria di dalam sub direktori dengan flag /S:

forfiles /S /M *mp3 /C "cmd /c move @files C:\music

Ini akan secara rekursif mencari semua folder di direktori Anda saat ini

Seperti yang dieksekusi dari baris perintah:

for /r %M in (*.mp3) do @if /I "%~dpM" neq "c:\temp\mp3 files\" @echo move /-Y "%M" "c:\temp\mp3 files\"

Ini hanya menggemakan perintah ke layar sehingga aman untuk dijalankan dan lihat apakah itu sesuai dengan kebutuhan Anda. Ini akan mulai dari direktori saat ini, jadi jika Anda ingin menghitung semuanya, jalankan dari c:\.
Penamaan direktori tujuan Anda tentu saja terserah Anda - tetapi pastikan jika ada! Anda mungkin juga ingin memasukkannya ke dalam variabel, atau sebagai parameter batch untuk menghindari kesalahan pengetikan. Ketika Anda 100% yakin hasilnya benar, hapus saja @echo, dan kemudian uji lagi dengan sedikit file / jalur (pastikan untuk menyertakan yang memiliki spasi dan beberapa karakter yang tidak biasa)

Beberapa kata peringatan:
1. Anda bilang Anda punya dups. Saya telah menempatkan / -Y sebagai pengaman, jadi memindahkan file dengan nama yang sama seperti yang sudah dipindahkan akan menghentikan skrip ini menunggu Anda untuk mengonfirmasi. Anda dapat menimpa ini dengan tombol / Y, tetapi Anda berisiko menimpa lebih dari yang Anda inginkan!
2. Ini hampir tidak dapat dibalik atau tidak dapat diubah (jika Anda memiliki duplikat 'palsu' - file berbeda dengan nama yang sama). Buat cadangan sebelum Anda mulai!
3. Lihat apa yang Anda miliki sebelum Anda pindah - ini akan memberi Anda daftar semua mp3 Anda, diformat dengan baik dengan nama, ukuran dan direktori:
for /r %M in (*.mp3) do @echo "%~nxM", "%~zM", "%~dpM" >>filelist.txt
Gunakan spreadsheet untuk menyortir dan memeriksanya.

Catatan:
a) Sebenarnya Anda dapat menghapus bagian if itu sama sekali, karena memindahkan file ke dirinya sendiri tidak dicatat, tetapi saya merasa itu berantakan :-)
b) jika Anda ingin memasukkannya ke dalam batch, ganti % tunggal dengan %% ganda

Saya berasumsi ini untuk membuat cadangan atau serupa. Jika demikian, saya membuat file bat dan tugas Terjadwal menjalankannya seminggu sekali.

Coba buat file bat.

Buka notepad dan ketik

xcopy "file location" "file destination" /i /e /y /z

misalnya

xcopy "c:\allmusic\mp3\" "e:\music\" /i /e /y /z

Simpan file sebagai .bat (misalnya "copyFiles.bat")

Anda dapat memiliki banyak baris tetapi satu baris per baris. MISALNYA

xcopy "c:\allmusic\mp3\" "e:\music\" /i /e /y /z
xcopy "c:\allmusic\wav\" "e:\music\" /i /e /y /z
xcopy "c:\allmusic\wmv\" "e:\music\" /i /e /y /z

Untuk menyalin berbagai folder dengan mengulang, coba

for /f %%f in ('dir /b c:\')  do (
cd\
cd %%f
copy *.mp3 "e:\music"

for /f %%g in ('dir /b %%f')  do (
cd\
cd %%g
copy *.mp3 "e:\music"


for /f %%h in ('dir /b %%g')  do (
cd\
cd %%h
copy *.mp3 "e:\music"
pause
)
)
)

Perhatikan, ini melakukan 3 level subfolder

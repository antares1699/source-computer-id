Cara meminta informasi saat memulai dokumen baru dari template Microsoft Word

Saya ingin membuat bidang dalam templat (dotx) yang, ketika memulai dokumen baru dari templat itu, akan meminta informasi. Bidang adalah sesuatu yang berulang-ulang di dalam dokumen. Saya ingin dapat memasukkannya sekali dan itu akan memperbarui dirinya sendiri di seluruh dokumen.

Cara termudah untuk satu item data adalah sebagai berikut.

Pertama, sisipkan bidang FILLIN di template

(Di Windows Word, gunakan ctrl-F9 untuk menyisipkan sepasang kurung kurawal kode bidang khusus {} lalu ketik

FILLIN "my prompt text"

diantara { }

jadi kamu lihat

{ FILLIN "my prompt text" }

Jika Anda memilih bidang dan menekan F9, Anda akan melihat kotak dialog tempat Anda dapat mengetikkan nilai yang Anda butuhkan. Ketika Anda telah memasukkan nilai, Word akan menampilkan nilai di lokasi di mana bidang FILLIN berada (tekan Alt-F9 untuk beralih antara tampilan "kode bidang" dan "hasil bidang".

Untuk membuat Word menampilkan hal yang sama di beberapa tempat di dokumen Anda, Pilih bidang FILLIN Anda dan tekan ctrl-F9 lagi untuk membungkusnya dengan pasangan lain {} Kemudian, setelah inisial "{", ketik

SET myfield

Sehingga semuanya terlihat seperti

{ SET myfield { FILLIN "my prompt text" } }

Kemudian, di setiap titik di mana Anda membutuhkan nilainya, masukkan bidang lain seperti ini:

{ myfield }

yang merupakan singkatan dari

{ REF myfield }

Saat Anda membuat dokumen baru berdasarkan template, Word harus menjalankan FILLIN, kemudian memperbarui semua bidang {myfield}. Tetapi jika pengguna perlu mengubah nilainya nanti, mereka harus memilih seluruh dokumen (mis. Ctrl-A) dan tekan F9 untuk melihat prompt lagi. Dalam hal ini, saya tidak yakin bidang akan diperbarui di semua tempat.

Ada jenis bidang lain yang disebut ASK yang, pada dasarnya, menggabungkan SET dan FILLIN, tetapi tidak secara otomatis dijalankan saat Anda membuat dokumen baru berdasarkan templat sehingga bukan pilihan yang baik untuk tugas ini.

Ada sedikit lebih banyak di bidang FILLIN daripada itu, tapi saya biarkan Anda mencarinya.

Saya hanya mencoba membuat bidang gabungan FILLIN bekerja sesuai instruksi dan tidak bisa membuatnya bekerja. Cara yang tidak terlalu rumit adalah dengan menggunakan bidang gabungan STYLEREF. Untuk melakukan ini, buat nama kolom generik untuk data contoh pertama (mis. "Organisasi" atau "tanggal") dan buat gaya unik untuknya (untuk keperluan petunjuk ini, saya akan menyebutnya "Style1") . Di setiap tempat dalam dokumen tempat Anda ingin data berulang, pilih ctl + F9 dan sisipkan STYLEREF "Style1" di antara tanda kurung (akan terlihat seperti ini: {STYLEREF "Style1"}. Selama Anda tidak memisahkan gaya dari entri pertama. Catatan, jangan gunakan gaya yang Anda gunakan untuk apa pun selain untuk bidang gabungan itu. Word mengisi bidang gabungan ini dari entri terakhir yang diketahui untuk gaya.

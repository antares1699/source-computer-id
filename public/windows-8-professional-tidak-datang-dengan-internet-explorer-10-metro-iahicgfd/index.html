Jika Anda mencari versi "Metro", versi itu mungkin tidak tersedia karena Anda memiliki browser lain yang disetel sebagai browser default. Jika Internet Explorer bukan browser default Anda, maka aplikasi IE10 Metro tidak tersedia. Ini harus tetap tersedia di desktop.

Anda dapat mencari "Program default" dan menjadikan IE sebagai browser default dan aplikasi Metro IE akan muncul secara ajaib.

Periksa untuk melihat apakah itu dinonaktifkan, Buka Panel Kontrol (tampilan ikon)

Klik "Program dan Fitur", ketika terbuka klik "hidupkan atau matikan fitur windows" di sisi kiri

.

Sebuah jendela kecil akan terbuka dan mengisi daftar, pastikan Internet Explorer 10 dicentang, jika tidak dicentang, centang dan tekan ok.

.

Jika ini tidak membuat IE10 tersedia, buka desktop dan lakukan a Menang + E

Catatan: Saat Anda berada di desktop, apakah ada ikon IE di bilah tugas?

ini akan membuka browser file, sekarang arahkan ke

C: \ Program Files (x86) \ Internet Explorer

apakah folder itu ada? apakah ada "iexplore" .exe di sana?

jika ada, Anda dapat menyeret pintasan ke desktop dan menggunakannya untuk sementara, IE di layar mulai mungkin memerlukan lebih banyak pekerjaan untuk membuatnya berfungsi.

Penggunaan praktis untuk PS3 linux? [Tutup]

Saya punya PS3 yang tidak saya gunakan banyak karena saya

Sebuah. tidak punya waktu untuk bermain game b. tidak punya waktu untuk film (bluray, dvd, dan lainnya)

Jadi saya sedang mempertimbangkan untuk memuat beberapa rasa linux ke dalamnya. Selain faktor gee-jagoan-tapi-apakah-itu-run-linux, adakah yang bisa menyarankan penggunaan praktis untuk melakukannya?

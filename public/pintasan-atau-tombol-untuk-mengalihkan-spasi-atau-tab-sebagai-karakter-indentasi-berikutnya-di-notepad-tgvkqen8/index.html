Saat mengganti spasi / tab untuk indentasi, apakah Anda ingin perubahan hanya berlaku untuk indentasi di masa mendatang atau semua indentasi, termasuk yang sudah ada?

Untuk Mengonversi Semua Spasi yang Ada ke Tab dan Sebaliknya

Hotkey : Pengaturan -> Pemeta Pintasan -> Menu Utama dan setel tombol pintas untuk TAB ke opsi Ruang dan wakil

Tombol Toolbar : Instal plugin "Sesuaikan Toolbar" dari Plugins -> Pengelola Plugin. Setelah instalasi, Plugins -> Customize Toolbar -> Customize akan memungkinkan Anda untuk menambahkan tombol "Tab to Space" dan "Space to Tab"

Alihkan karakter indentasi (spasi atau tab) untuk indentasi berikutnya

  • Instal Skrip Python plugin dan dari menunya, Buat skrip baru .
  • Pilih nama file dan simpan di direktori yang diberikan
  • Dalam file skrip kosong tambahkan: editor.setUseTabs(not editor.getUseTabs()) dan simpan

Dari Python Script -> menu Konfigurasi:

Hotkey : Tambahkan item menu dan mulai ulang N ++. Dari Shortcut Mapper -> Plugin Commands, atur pintasan untuk skrip toggle Anda.

Tombol : Tambahkan tombol. Mulai ulang N ++. Jika tombol toolbar tidak dapat diklik, konflik dengan file Ubahsuaikan Toolbar plugin sedang terjadi, jadi Anda mungkin harus menghapus plugin itu. Jika ada cara untuk mendapatkannya Ubahsuaikan Toolbar dan Skrip Python untuk menambahkan tombol ke toolbar secara bersamaan, saya belum menemukannya.

Mengalihkan Visibilitas Karakter Spasi Putih

Hotkey : Pemeta Pintasan -> Menu Utama -> "Tampilkan Ruang Putih dan Tab" atau

Set Hotkey dan / atau Tombol Toolbar untuk N ++ Python Script berikut:

# Toggle whitespace visibilty through the following modes
# 0: The normal display mode with white space displayed as an empty background colour
# 1: White space characters are drawn as dots and arrows
# 2: White space used for indentation is displayed normally but after the first visible character, it is shown as dots and arrows
# 
# Mode descriptions from http://www.scintilla.org/ScintillaDoc.html#SCI_GETVIEWWS

editor.setViewWS( (editor.getViewWS()+1)%3 )

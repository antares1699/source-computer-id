Bagaimana cara memaksa proses untuk berjalan di satu utas hanya dengan numactl?

Mesin lokal saya memiliki banyak CPU, masing-masing memiliki banyak inti, dan masing-masing dapat mendukung banyak utas. Saya memiliki proses multithreading yang ingin saya paksa berjalan hanya pada satu utas. Saya mengerti numactrl dapat melakukan itu.

Bagaimana saya bisa menggunakan numactl untuk memaksa proses berjalan hanya pada satu utas? Atau, apakah ada utilitas bawaan yang lebih sederhana yang dapat melakukan itu?

4

1 Jawaban

8
8
numactl --physcpubind=+1 /path/to/your/executable

ini harus menjalankan proses Anda pada inti / cpu kedua yang ditetapkan ke chipset Anda (indeks 1). lihat bagian Contoh halaman manual untuk lebih jelasnya: http://linux.die.net/man/8/numactl

Edit: Saya harus menunjukkan, bahwa ini berarti program akan berjalan paling banyak satu utas secara bersamaan, tetapi itu tidak berarti bahwa seluruh proses hanya akan terdiri dari satu utas. jika program ditulis untuk menelurkan utas baru, program akan melakukannya, tetapi akan dijalankan pada inti / cpu yang sama dengan proses lainnya. perbedaan tipis, tapi tetap berpotensi penting.

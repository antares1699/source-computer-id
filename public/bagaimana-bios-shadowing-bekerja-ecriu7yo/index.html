Bagaimana BIOS shadowing bekerja

Saya mendengar bahwa BIOS shadowing bekerja dengan menyalin program BIOS ke dalam memori yang lebih cepat dan menjalankannya dari sana. Apakah itu benar? BIOS bertanggung jawab untuk tugas-tugas startup tertentu, jadi setelah startup, apakah itu diturunkan dari memori?

Anda benar BIOS shadowing cukup memuat BIOS ke dalam Memori untuk referensi lebih cepat. Saya tidak dapat menemukan banyak kegunaannya setelah startup, namun menurut Microsoft itu akan tetap berada di memori setelah startup. Rupanya DOS menggunakan BIOS untuk menulis ke layar. Dari artikel MS:

ROM BIOS shadowing adalah proses menyalin BIOS dari ROM ke dalam RAM dan menggunakan perangkat keras atau mode yang ditingkatkan 386 untuk memetakan ulang RAM ke dalam ruang alamat normal BIOS. Karena membaca RAM jauh lebih cepat daripada membaca ROM, operasi intensif BIOS jauh lebih cepat. Misalnya, MS-DOS menggunakan BIOS untuk menulis ke layar; oleh karena itu, dengan bayangan ROM BIOS, daftar direktori berjalan lebih cepat.

Windows NT tidak menggunakan BIOS (kecuali selama startup); oleh karena itu, tidak ada kinerja yang diperoleh dengan membayangi. Jika bayangan ROM BIOS tidak digunakan, lebih banyak RAM tersedia. Dengan Windows NT, ada keuntungan untuk menonaktifkan opsi bayangan ROM BIOS.

Artikel ini agak lama tetapi masih relevan karena versi Windows yang lebih baru tidak mereferensikan BIOS setelah boot. Sumber

Mengatur tanggal pembuatan file di Mac OS X

Saya memerlukan skrip untuk menyetel tanggal pembuatan ke tanggal modifikasi untuk sekumpulan file yang dibuat oleh perekam suara saya - ada cara untuk melakukan ini? Menjalankan Mac OS X 10.4.11

4
Red Miller — Sumber
  • Hanya untuk membantu Anda menemukan solusi: ini disimpan di "HFS meta data", pada kMDItemFSCreationDate. (UNIX sendiri tidak memiliki gagasan tentang tanggal pembuatan. Lihat superuser.com/questions/43310/… untuk beberapa detail, tetapi itu tidak menjelaskan cara mengubahnya.)
    — Arjan

8

Anda dapat menggunakan SetFile pada baris perintah untuk melakukan ini namun ini tidak disertakan secara default di Mac OS X 10.4.x (Tiger). Jika Anda telah menginstal Alat Pengembang atau sebagian besar pembaruan Combo itu diinstal tetapi di lokasi non standar. SetFile disertakan dalam /usr/bin/ untuk Mac OS X 10.5 dan yang lebih baru.

Untuk menemukan perintah, Anda dapat mencoba menggunakan locate untuk menemukan lokasinya.

Dengan asumsi Anda menginstal Alat Pengembang:

/Developer/usr/bin/SetFile -d '12/31/2008 12:00:00 PM' nameoffile.txt

Anda dapat menggunakan program lain bernama GetFileInfo yang diinstal dengan SetFile untuk mendapatkan tanggal modifikasi.

/Developer/usr/bin/GetFileInfo nameoffile.txt

Contoh skrip untuk satu file:

#! /bin/bash
# Usage: nameOfThisScript.sh nameOfFile.txt

modifiedDate=`/usr/bin/GetFileInfo -m $1`
/usr/bin/SetFile -d "$modifiedDate" $1

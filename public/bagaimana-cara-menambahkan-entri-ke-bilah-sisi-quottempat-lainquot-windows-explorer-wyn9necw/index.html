Bagaimana cara menambahkan entri ke bilah sisi "Tempat Lain" Windows Explorer?

Di Windows XP, Windows Explorer memiliki bilah sisi "tugas umum" di sisi kiri jendela Explorer yang menampilkan item yang dikelompokkan berdasarkan bagian sistem file yang sedang dieksplorasi. Saat melihat grup "Komputer Saya" yang terdaftar adalah "Tugas Sistem", "Tempat Lain" dan "Detail". Adakah cara untuk menambahkan direktori yang saya pilih ke grup "Tempat Lain"?

Tanpa TweakUI, Pertama buka kunci registri HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\comdlg32\PlacesBar dan kemudian Anda akan melihat lima nilai DWORD, They are Places0 to Places4. Nilainya dapat berupa:

  • Desktop - 0x00
  • Lingkungan Jaringan - 0x12
  • Favorit - 0x06
  • Sejarah - 0x22
  • Dokumen Saya - 0x05
  • Foto Saya - 0x27
  • Musik Saya - 0x0D
  • Dokumen Terbaru - 0x08
  • Komputer Saya - 0x11

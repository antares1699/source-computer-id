SED: Bagaimana saya bisa mencetak setiap baris setelah string pertama menggunakan Sed?

Saya memiliki file dengan format serupa ...

16:28 asdfasdf
16:29 4398upte
16:30 34liuthr
16:31 34tertio

Bagaimana saya bisa menggunakan SED untuk mencetak setiap baris termasuk dan setelah baris dengan "16:30"?

Hasilnya adalah ...

16:30 34liuthr
16:31 34tertio

Saat ini, saya menggunakan sed sebagai berikut, tetapi saya harus mencari nomor baris pertama secara manual, mis. "562697":

sed -n '562697,$p'

Alamat di sed bisa berupa nomor baris atau pola. Coba ini:

sed -n '/16:30/,$p'

Jika polanya berisi /, Anda dapat menghindarinya dengan \. Misalnya, untuk menelusuri 16/30 alih-alih 16:30, coba ini:

sed -n '/16\/30/,$p'

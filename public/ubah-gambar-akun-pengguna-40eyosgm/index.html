Laptop Windows 7 (Enterprise) saya yang baru tidak mengizinkan saya mengubah gambar Akun Pengguna saya. Bagaimana cara mengaktifkan ini?

Saya harap tidak apa-apa untuk menjawab pertanyaan lama ini, tetapi itu adalah hal pertama yang saya temukan saat menelusurinya, jadi saya pikir mungkin akan berguna bagi orang lain untuk mencoba apa yang berhasil untuk saya.

Hal lain yang dapat Anda coba jika Anda merasa nyaman menggunakan RegEdit. (Ini tentu saja dapat mengacaukan komputer Anda, jadi gunakan dengan risiko Anda sendiri).

  1. Buka Regedit . (cari di startmenu atau pilih Run ... dan ketik regedit)
  2. Telusuri setiap kemunculan UseDefaultTile ( seharusnya: HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion‌ \ Policies \ Explorer \ U‌ seDefaultTile )
  3. Untuk masing-masing, pastikan sudah disetel ke 0 dari pada 1 .
  4. Tutup Registry Editor dan periksa apakah Anda dapat mengubah ikon Anda.

Ini jelas akan mengubahnya untuk kebijakan lokal apa pun yang tersedia di komputer Anda. Jika Anda ingin mengubahnya hanya untuk kebijakan tertentu, Anda harus memeriksa kebijakan mana yang Anda ubah terlebih dahulu.

Catatan: ini hanya akan mempengaruhi akun lokal. Untuk kebijakan Grup / Domain Anda kemungkinan besar perlu mengubah pengaturan domain di server.

ALTERNATIF: Ubah gambar akun pengguna default

Anda dapat mengubah gambar pengguna default, tetapi itu berarti setiap pengguna akan memiliki gambar yang sama , yang mungkin bukan yang Anda inginkan.

  1. Buat atau ubah ukuran gambar menjadi 128 * 128 piksel dan simpan sebagai .bmp untuk menghindari pemotongan
  2. Simpan gambar sebagai "user.bmp"
  3. Navigasi ke Gambar \ ProgramData \ Microsoft \ User Account (ProgramData mungkin disembunyikan, tergantung pada Pengaturan Anda, harus ada di partisi boot utama Anda.)
  4. Ubah nama user.bmp menjadi user.bmp.bak sehingga Anda memiliki cadangan.
  5. Salin gambar yang dibuat sebelumnya ke folder (memerlukan hak istimewa admin)
  6. Tekan Windows + L pada Keyboard untuk menguji apakah itu berhasil.

Semoga ini membantu.

Bagaimana cara membuat pintasan ke VMWare Workstation Player 15 VM?

Saya ingin memulai Mesin Virtual khusus VMWare Workstation Player dari tautan, daripada memulai VMWare, lalu mengklik dua kali pada VM.

VMWare menyediakan file penjelasan tentang prosedur ini untuk produk Pro, tetapi tidak berfungsi dengan Pemutar.

Apakah ini mungkin?

tl; dr: Perintah yang diperlukan vmplayer.exe HelloWorld.vmx

Pertama temukan jalur lengkap vmplayer.exe yang dapat dieksekusi. Jalur default adalah:

"C:\Program Files (x86)\VMware\VMware Player\vmplayer.exe"

Jalankan jalur itu dengan opsi --help di Command Prompt untuk jendela bantuan:

Penggunaan: vmplayer.exe [OPTION ...] [-] [file konfigurasi]

dimana OPTIONS adalah:

-v          Show program version
-X          Enter full screen mode
            when a virtual machine is powered on
--unity     Enter Unity
            when a virtual machine is powered on

Sekarang temukan file konfigurasi mesin virtual VMware yang diperlukan dengan .VMX ekstensi file. Sebagai contoh:

"%UserProfile%\Desktop\Virtual Machines\Windows 10 x64\Windows 10 x64.vmx"

Karenanya perintah lengkapnya akan seperti ini:

"C:\Program Files (x86)\VMware\VMware Player\vmplayer.exe" "%UserProfile%\Desktop\Virtual Machines\Windows 10 x64\Windows 10 x64.vmx"

Terlalu lama, bukan? Buat pintasan vmplayer.exe dengan perintah itu. Ini juga berfungsi di Command Prompt. Pastikan untuk pengguna mengutip ganda jalur karena jalur tersebut berisi spasi. Ini mungkin membantu.

Anda seharusnya bisa mengklik dua kali pada file VM .vmx. Jika Anda hanya menginstal VMware (Workstation) Player (dan bukan VMware Workstation Pro), maka VM akan terbuka secara otomatis di Player.

Jika Anda ingin mengklik dua kali pada ikon dari lokasi yang lebih nyaman, Anda dapat melakukan seret tombol kanan mouse pada file .vmx untuk membuat pintasan ke sana dan kemudian memindahkan pintasan ke mana pun Anda mau.

Pintasan mode VMware KVM adalah cara yang lebih baik!

Buat pintasan desktop dengan target ini:

"C: \ Program Files (x86) \ VMware \ VMware Workstation \ vmware-kvm.exe" "% UserProfile% \ Documents \ Mesin Virtual \ Windows 10 x64 \ Windows 10 x64.vmx"

Baca artikel ini untuk informasi lebih lanjut:

https://kb.vmware.com/s/article/2057914

Mengapa hard drive saya masih terfragmentasi setelah saya menjalankan defraggler?

Saya menjalankan defraggler tadi malam karena dan saya membiarkannya berjalan saat saya tidur. Saya bangun pagi ini dan masih memberi tahu saya bahwa hard drive saya 20% terfragmentasi. Jadi saya menjalankannya lagi sebelum saya meninggalkan rumah selama beberapa jam. Saya kembali, dan masih tertulis 19% terfragmentasi.

Apa alasannya ini? Apakah normal atau ada masalah dengan hard drive saya? Saya sedang mempersiapkan untuk mengatur ulang partisi saya, jadi saya pikir akan lebih bijaksana untuk melakukan defrag terlebih dahulu.

Sunting: Setelah analisis ulang ketiga, Puran Defrag dan Defraggler keduanya memberi tahu saya bahwa saya hanya memiliki fragmentasi 125mb. Sepertinya hanya kebetulan.

Penginstal mandiri flash untuk Win 2008/64 / IE8

Saya bekerja pada server x64 Win2008 yang tidak memiliki akses jaringan eksternal. Perangkat lunak yang saya gunakan menggunakan flash, tetapi tidak menginstalnya.

Saya perlu mencari penginstal flash mandiri yang dapat saya unduh dan masukkan ke kotak ini.

Jika Anda pergi ke situs adobe, Anda dapat memilih os / arsitektur, tetapi itu ingin menginstal adobe-downloader untuk mengunduh dan menginstalnya daripada hanya mengizinkan saya untuk mengunduh.

Benang ini menunjuk ke penginstal (sebagai bagian dari sub komentar), tetapi tautannya mati. Lebih lanjut, satu-satunya penginstal offline yang tersedia tampaknya hanya menginstal versi 32-bit, bahkan pada sistem 64-bit seperti yang terlihat di Panel kendali (di bawah).

Adakah yang tahu di mana saya dapat menemukan penginstal mandiri untuk 2008 / x64 / IE8?

Saya telah menemukan bahwa melihat situs adobe menggunakan Chrome biasanya memungkinkan Anda untuk mengunduh file tanpa menggunakan Adobe Download Manager yang sangat mengganggu.

Cara lainnya adalah dengan meminta izin untuk mendistribusikan produk Adobe. Anda harus mengirimkan permintaan dan biasanya perlu satu atau dua minggu untuk mendapatkan persetujuan, tetapi kemudian mereka memberi Anda tautan untuk mengunduh langsung.

Metode terakhir adalah menggunakan tautan yang saya dapatkan dari sistem izin distribusi Adobe: http://www.adobe.com/products/flashplayer/fp_distribution3.html

Tautan tidak diberi kode untuk akses atau akun tertentu, jadi Anda tidak mendapatkan aplikasi versi "saya".

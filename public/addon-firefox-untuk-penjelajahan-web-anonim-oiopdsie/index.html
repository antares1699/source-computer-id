Aku s Jaringan Tor (The Onion Router) sesuatu untuk mu? Dengan Tor, Anda anonim untuk semua orang di jaringan, tetapi Tor perlu menginstal perangkat lunak pada PC. Ada beberapa peringatan di halaman FAQ tentang penggunaan Tor.

Torbutton adalah ekstensi yang akan mengalihkan penggunaan Tor untuk Anda.

Jika Anda hanya membutuhkan browser untuk tidak merekam data penggunaan (cookie, history, cache), maka Anda dapat menggunakan Mode Privasi di Firefox.

Ada juga ekstensi yang disebut QuickProxy yang memungkinkan Anda untuk mengganti status Proksi dengan cepat. Anda dapat menggunakan proxy anonim di suatu tempat di luar negeri untuk menyembunyikan kunjungan Anda.

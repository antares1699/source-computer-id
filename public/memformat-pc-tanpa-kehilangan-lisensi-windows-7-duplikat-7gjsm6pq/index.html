Memformat PC tanpa kehilangan lisensi Windows 7 [duplikat]

Kemungkinan Duplikat:
Aktivasi Windows dengan instal ulang terencana

Saya memiliki instalasi windows 7 yang saya dapatkan dari MSDN. Masalahnya adalah saya ingin memformat PC saya dan menginstal ulang windows 7.

Awalnya saya pikir saya hanya perlu menulis kunci saat ini di suatu tempat, menginstal ulang windows 7 secara normal dan kemudian mengaktifkannya dengan kunci lama saya, tetapi seorang teman melakukan ini dan pada langkah terakhir aktivasi tidak akan berfungsi karena dikatakan bahwa kuncinya adalah digunakan.

Apa yang dapat saya?

Dalam situasi ini di bawah Windows XP, saya hanya akan melanjutkan dan menginstal ulang dari awal menggunakan kunci asli. Jika aktivasi gagal, ada pilihan untuk mengaktifkan melalui telepon. Anda menelepon nomor tersebut, memberi tahu mereka bahwa ini adalah penginstalan ulang di komputer yang sama dan lisensinya tidak digunakan di tempat lain, lalu mereka akan memberi tahu Anda kode 50 digit. Masukkan kode itu dan Anda siap melakukannya.

Saya pikir ini masih berlaku untuk Windows 7, tetapi saya tidak yakin, karena saya belum harus melakukannya sendiri di mesin Windows 7.

Ini berhasil untuk saya:

  1. Device Manager> USB Root Hub> Properties> Power Management (tab)
  2. Hapus centang Allow computer to turn off this device to save power.
  3. Anda dapat menerapkan ini ke perangkat lain di bawah Device Manager> Universal Serial Bus Controllers
  4. Anda mungkin perlu memulai ulang komputer agar ini berlaku.

Bagi saya, efeknya tidak langsung, tetapi setelah 1 minggu atau lebih, kecepatan sakelar meningkat secara dramatis (2-4sec).

Kadang-kadang, saya masih menunggu seperti 13 detik, tetapi setelah restart atau cabut, semuanya bekerja kembali dengan cepat.

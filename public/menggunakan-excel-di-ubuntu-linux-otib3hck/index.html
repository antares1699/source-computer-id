Menggunakan Excel di Ubuntu (Linux)?

Saya pernah mendengar tentang orang yang menjalankan Excel di Ubuntu menggunakan Anggur , tapi kesan saya adalah itu ini bisa lebih membuat frustrasi daripada nilainya . Saya kira alternatif teratas adalah menjalankan Windows atau menggunakan sesuatu seperti OpenOffice.

Adakah yang berhasil menggunakan Excel di Ubuntu, dan jika demikian, apa kesimpulan Anda?

Sebagai referensi, saya saat ini menggunakan Excel 2007 dan baru saja memutakhirkan ke Ubuntu 10.04, meskipun saya ingin tahu tentang pengalaman apa pun dengan ini.

Saya benar-benar menemukan bahwa Microsoft Office 2003 telah bekerja cukup baik dengan Wine selama beberapa tahun terakhir. Bergantung pada versi Excel Anda, Anda mungkin mencoba CrossOver Linux

CrossOver Linux memungkinkan Anda menginstal banyak aplikasi, plugin, dan game produktivitas Windows yang populer di Linux, tanpa memerlukan lisensi Sistem Operasi Microsoft. Setelah terinstal, aplikasi Anda terintegrasi secara mulus dengan lingkungan Gnome atau KDE Anda. Cukup klik dan jalankan aplikasi Anda, persis seperti yang Anda lakukan di Windows, tetapi dengan kebebasan penuh di Linux.

Saya memiliki pengalaman yang baik dengan orang-orang ini di masa lalu (sekitar 2002) tetapi saya rasa hal itu tidak menjadi lebih buruk sejak saat itu.

Belum mencoba Excel tetapi saya menggunakan VirtualBox (perangkat lunak gratis) di bawah Ubuntu 10.4 untuk menjalankan kompiler C ++ 6.0 Microsoft dengan hasil yang sangat baik. VirtualBox didukung oleh Oracle Corp (www.virtualbox.org). Saya menggunakan mesin virtual dengan Windows 2000 untuk meng-host C ++ tetapi seharusnya bekerja sama dengan XP atau Win7. Jika Anda memilih OS yang lebih baru, tentu saja akan ada persyaratan RAM host yang lebih besar. Setup berlangsung sekitar 30 menit, termasuk menginstal Win2K. Pantas untuk dicoba jika Anda memiliki CD penginstalan MS Win2K / XP / Vista / Win7.

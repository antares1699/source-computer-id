Posting ini menguraikan cara membuat direktori AppData muncul pada prompt DOS, tetapi bagaimana cara membuatnya muncul di File Explorer (pada Windows 7 Home Premium 64-bit)? Apakah ada pengaturan global? Juga, apa izin file di bawah properti yang membuatnya tersembunyi atau tidak? Terima kasih.

Bagaimana cara membuat AppData muncul di File Explorer?

  1. Menu "Alat"> "Opsi folder ..."

  2. Klik tab "Tampilan"

  3. Gulir ke bawah hingga Anda melihat "File dan folder tersembunyi"

  4. Aktifkan "Tampilkan file, folder, dan drive tersembunyi"

  5. Klik "OK"

Apa izin file di bawah properti yang membuatnya tersembunyi atau tidak?

  1. Klik kanan folder AppData

  2. Pilih "Properties"

  3. Klik tab "Umum"

  4. Aktifkan / Nonaktifkan "Tersembunyi" sebagaimana mestinya

Bagaimana cara membalikkan arah zoom roda gulir mouse di Firefox?

Saat menggulir ke bawah roda mouse dengan CTRL , zoom menurun. Saya ingin meningkatkannya saat saya menggulir ke bawah, dan sebaliknya.

Apakah ada cara untuk mengubahnya?

4

1 Jawaban

6
6

Buka about:config dan atur preferensinya:

Untuk Firefox 17+:

mousewheel.with_control.delta_multiplier_y = -100 // default 100

Untuk Firefox <= 16:

mousewheel.withcontrolkey.numlines = -1 // default 1

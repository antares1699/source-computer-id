Luncurkan program dengan Wine menggunakan argumen .exe

Saya memiliki program ini yang dapat saya luncurkan melalui command prompt di Windows atau melalui .lnk yang dibuat saat menambahkan argumen yang ditentukan seperti ini:

"C:\My\program.exe" /ID:"MyID" /CompanyID:"123" /TheAddr:"my.address.com"

Dengan membaca dokumentasi Tampaknya Wine menafsirkan argumen dengan cara yang sama seperti Windows, dan bahkan ketika saya yakin itu benar untuk argumen yang disajikan dengan cara ini:

wine program.exe -my -arguments

Ini tidak berfungsi untuk argumen yang dijelaskan di atas, mungkin saya kehilangan semacam pelarian untuk beberapa karakter?

Saya sudah mencoba meluncurkan aplikasi dengan cara ini:

wine program.exe /ID:"MyID" /CompanyID:"12" /TheAddr:"my.address.com"

Tetapi tidak berhasil, yang saya dapatkan hanyalah pesan kesalahan umum dari program saya.

Saya percaya itu karena saya tidak menyampaikan argumen dengan cara yang benar, karena jika saya membuat file .lnk di Windows memperbarui lokasinya agar sesuai dengan instalasi awalan Wine saya dan kemudian menavigasi ke sana melalui Wine's explorer.exe dan klik dua kali pada itu, program saya berjalan dengan sempurna.

Coba ini

exec wine "~/.wine/drive_c/My/program.exe" "-my"
  • Perintah exec memberi tahu bash untuk berubah menjadi wine dengan argumen berikut, jadi ini bukan lagi bash running wine, tetapi proses bash menjadi wine. PID tetap ada. Anda tidak memiliki dua proses yang sedang berjalan.

Solusi yang diberikan untuk pertanyaan ini dapat ditemukan di Unix.SE.

Windows 10 Meminta saya untuk Memasukkan Kata Sandi

Saya menggunakan Windows 10 Pro x64 di laptop pribadi saya di mana saya memiliki satu akun pengguna tanpa kata sandi. Namun untuk beberapa alasan, Windows 10 mulai meminta saya memasukkan kata sandi setiap kali saya mem-boot mesin saya. Yang harus saya lakukan adalah menekan tombol tanpa memasukkan apa pun, dan saya masuk, tetapi saya lebih suka melewatkan langkah ini demi masuk otomatis. Mengapa ini baru mulai terjadi sekarang? Apa yang dapat saya lakukan untuk menghentikan perilaku ini?

Berikut adalah cara menghentikan komputer meminta kata sandi setiap saat:

  1. Klik kanan Logo Windows (Tombol Mulai) dan klik "Jalankan"
  2. Ketik netplwiz dan klik "OK"
  3. Hapus centang "Pengguna harus memasukkan nama pengguna dan sandi untuk menggunakan komputer ini."

Cara menutup jendela Peringatan Javascript besar di Chrome / OSX

Saya menggunakan Chrome 14 dan OSX 10.5.8.

Jika situs web menghasilkan jendela peringatan Javascript dengan banyak konten (seperti informasi debug), bagian bawah jendela menghilang dari layar.

Ini menempatkan tombol di luar jangkauan dan berarti tidak ada cara untuk menutup jendela peringatan karena "lampu lalu lintas" tombol jendela tidak muncul di sudut kiri atas.

Membuka jendela peringatan menonaktifkan berbagai tindakan di Chrome dan membuatnya tidak dapat digunakan. Satu-satunya cara saya dapat menutup jendela peringatan adalah menggunakan "Force Quit" untuk menutup Chrome sepenuhnya.

Apakah ada cara untuk menutup jendela peringatan Javascript Chrome yang terlalu besar untuk layar di OSX?

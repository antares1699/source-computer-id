Tidak dapat terhubung ke titik akses nirkabel: waktu asosiasi habis

Saya mencoba untuk memperluas jangkauan nirkabel saya menggunakan VersaLink 327W sebagai wireless range extender dan AP .

Saya mengikuti instruksi:

  1. DSL dinonaktifkan.
  2. Akses nirkabel yang diaktifkan dengan kunci WEP (perangkat tidak mendukung WPA dan WPA2).
  3. DHCP dinonaktifkan.
  4. Setel alamat IP ke 192.168.7.2 (karena router utama yang akan dihubungkan melalui ethernet ini adalah 192.168.7.1).

Sayangnya, saya tidak dapat terhubung ke perangkat melalui WiFi dari kotak linux:

Yaitu., autentikasi bekerja, tapi asosiasi (apapun itu) tidak.

Ini tidak mungkin menjadi masalah dengan linux karena saya tidak dapat terhubung dari file Mac dan ponsel Android juga.

Saya mengalami masalah yang sangat mirip hari ini. Apa yang memperbaikinya untuk saya adalah mengubah konfigurasi NetworkManager:

  1. Edit file /etc/NetworkManager/NetworkManager.conf (lokasi mungkin berbeda-beda, tergantung distro Anda)

  2. Tambahkan baris berikut ke file itu:

    [device]
wifi.scan-rand-mac-address=no
  3. Mulai ulang layanan NetworkManager: sudo service network-manager restart

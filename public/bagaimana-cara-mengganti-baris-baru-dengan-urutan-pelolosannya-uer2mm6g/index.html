Bagaimana cara mengganti baris baru dengan urutan pelolosannya?

Menggunakan sed untuk membuat teks yang berisi karakter tertentu yang cocok untuk digunakan dalam literal string sangatlah mudah:

sed "s/\\\\/\\\\\\\\/g"
sed "s/\\\"/\\\\\\\"/g"

Tetapi bagaimana saya bisa melakukan sesuatu yang serupa dengan file teks yang berisi karakter baris baru?

sed berbasis garis, dan ini dapat menyebabkan masalah saat mencoba mengganti karakter baris baru.

dokumentasi resmi untuk sed membuat referensi khusus ke karakter baris baru dan menyatakan bahwa mereka dilucuti sebelum diteruskan ke sed.

Saya akan menyarankan bahwa 'tr' mungkin lebih cocok di sini.

sebagai contoh, untuk mengganti karakter baris baru dengan spasi:

tr '\n' ' ' < inputfile

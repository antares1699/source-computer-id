Bagaimana cara menonaktifkan driver touchpad laptop?

Saya ingin menonaktifkan driver untuk touchpad saya. Ini disebut "Mouse yang Kompatibel dengan PS / 2". Jika saya membuka properti, tombol nonaktifkan berwarna abu-abu. Juga tidak ada opsi untuk menonaktifkan di menu konteks jika saya mengklik kanan di pengelola perangkat. Lampirkan tangkapan layar dari masalah di bawah. Saya menggunakan Windows 8 tetapi saya menduga masalahnya akan sama di Windows 7.

Microsoft Wireless Display: batas hitam di sekitar gambar (gambar layar diperkecil)

Saya telah terhubung ke Microsoft Wireless Display Adapter (versi firmware 2.0.8381) selama beberapa tahun terakhir, dari laptop Windows 10 (64-bit) saya dengan chip grafis Intel HD 620. Saya telah memperluas tampilan ke tampilan eksternal ini tanpa masalah.

Namun, dalam seminggu terakhir, batas hitam dengan lebar sekitar 1,5 "telah muncul di sekitar gambar pada layar eksternal, dan tidak ada yang saya coba yang dapat menghilangkannya. Dengan kata lain, gambar pada tampilan eksternal diperkecil sekitar 15 %, dan di tengah tampilan dengan garis tepi hitam di sekitar gambar.

Resolusi layar eksternal: 1920 x 1080. Layar ini melaporkan input HDMI - 1080P, 67KHz, 60Hz.

Windows 10 versi 1803 membangun 17134.228

Saya telah mencoba: 0. Power bersepeda semuanya. 1. Perbarui driver Intel Graphics 620 saya ke versi terbaru dari situs web intel.com 2. Sesuaikan pengaturan Pemindaian Berlebih di aplikasi Adaptor Tampilan Nirkabel 3. Perbarui firmware Adaptor Tampilan Nirkabel ke versi terbaru yang tersedia. 4. Kembalikan Pembaruan Windows ke tanggal sebelum pembaruan versi terakhir 5. Setelah memutar kembali, instal ulang driver grafis Intel terbaru

Ada ide? Apakah ada opsi tersembunyi di suatu tempat yang memungkinkan gambar diskalakan?

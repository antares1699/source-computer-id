Atur Pembaruan Windows untuk menyertakan Visual Studio

Dapatkah saya mengatur Pembaruan Windows untuk menyertakan juga pembaruan ke Visual Studio Beta?

Misalnya, Microsoft baru saja mengumumkan pembaruan April untuk Visual Studio 11 Beta. Akan sangat bagus, jika saya dapat menginstalnya dengan cara yang sama seperti saya menginstal pembaruan untuk Windows, Office, dll.

Untuk menyiapkan pembaruan Windows untuk memindai Microsoft atau produk lain yang dapat diperbarui, Anda perlu melakukan hal berikut:

  1. Menu Mulai -> Pembaruan Windows

  2. dekat bagian tengah / bawah Anda akan melihat teks yang berbicara tentang memperbarui produk lain di sistem Anda, klik pada teks biru dan itu akan memunculkan situs web di Internet explorer, atau menanyakan apakah Anda ingin mengizinkannya.

  3. Jika itu meminta Anda untuk mengizinkan / menolak, mengklik izinkan, dan kemudian memeriksa pembaruan sekarang akan memberi Anda pembaruan tambahan yang Anda cari

    Jika muncul situs web, Anda harus mengikuti panduan di atasnya, lalu centang "saya terima" dan tekan tombol terapkan yang setara dengan perisai di atasnya

  4. Sekarang seharusnya membawa Anda kembali ke pembaruan Windows Anda, dan teksnya akan tetap ada di sana, Anda perlu mengklik tautan biru lagi, dan kembali ke 3a.

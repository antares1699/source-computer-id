Komputer

Apakah mungkin untuk membuat kode pada dua komputer yang berbeda secara bersamaan?

Saya ingin bekerja dengan programmer lain dan saya ingin kode sumber ditayangkan secara real-time di kedua layar kami. Apakah ini mungkin di Mac OS x atau Linux? Kami akan menggunakan OS X tetapi kadang-kadang kami mungkin perlu menambahkan komputer Ubuntu juga.

Apakah ada cara saya dapat melakukan ini menggunakan ssh, program berbasis shell, atau bahkan GUI yang bagus? Saya pikir Coda mungkin mampu melakukan ini tetapi tidak benar-benar berfungsi.

Ada yang pernah melakukan ini? Saya tidak mencari git / svn / atau sistem kontrol versi lainnya. Ini lebih merupakan sesi pengkodean langsung. :)

EDIT: Untuk menjelaskan situasi ini dengan lebih baik, katakanlah teman saya Emily dan saya ingin berkolaborasi, haruskah saya memiliki SSH yang dihosting melalui internet?
... terkadang saya menemukan alamat SSH di pengaturan berbagi saya di OS X. Dapatkah saya mengambil alamat SSH ini dan memberikannya kepada Emily dan apakah dia dapat menggunakannya dan setelah ini, apa yang kita perlukan untuk terhubung secara bersamaan?

EDIT 2: Adakah yang bisa melakukan ini menggunakan Coda melalui internet?

Mengapa saya memilih jawaban yang saya lakukan ... Meskipun saya menemukan SubEthaEdit sangat berguna, saya tidak ingin menghabiskan $ 60 + ($ 30 untuk saya dan $ 30 untuk teman saya) karena kami sudah menggunakan banyak alat untuk pengeditan sumber. Saya menyadari bahwa teman saya adalah pengguna emacs yang besar dan saya adalah pengguna vim, juga SSH akan mempersulit diri saya untuk melakukan penyiapan. Meskipun di masa depan saya mungkin menganggap yang terakhir sebagai solusi terbaik. Saya memilih untuk pergi bersama DocShare karena ini adalah tambahan untuk alat yang kami berdua tahu cara menggunakannya, gratis, dan saya merasa saya akan dapat bekerja secara bersamaan tanpa banyak masalah.

TL; DR Saya memerlukan alat yang membantu saya berkolaborasi kode sumber dengan teman saya, itu harus hanya Mac atau multiplatform. Saya memilih untuk pergi bersama DocShare .

Untuk SSH atau solusi berbasis shell, lihat "layar" untuk Linux. Pertama-tama hubungkan melalui SSH ke server (server tersebut bisa menjadi salah satu dari dua mesin, dalam hal ini Anda hanya perlu terhubung dengan yang lain) dan menjalankan perintah berikut.

Di komputer pertama:

$ screen -S my_screen
$ nano file.php

Di komputer kedua:

$ screen -x my_screen

Anda dapat menggunakan X-forwarding dan Emacs, seperti di masa lalu yang indah.

Mulai Emacs di satu mesin. Di komputer lain, pastikan Anda menjalankan X-server dan mengizinkan xhost <name> di mana <name> adalah mesin pertama atau nama pengguna orang yang mengerjakan mesin pertama.

Kemudian di Emacs yang berjalan di mesin pertama, ketik M-x make-frame-on-display et voila, Anda akan mendapatkan jendela Emacs di mesin kedua yang berbagi konten dengan Emacs yang berjalan di mesin pertama.

Namun, ini mungkin bukan yang Anda inginkan, karena kedua bingkai akan disinkronkan sepanjang waktu: ketika satu pengguna mengubah buffer, itu juga terjadi di jendela lain.

apakah ada wadah plugin seperti firefox untuk chrome?

Saya baru saja beralih ke chrome beberapa hari yang lalu dan saya sangat menyukainya, tetapi ada satu hal yang sangat saya sukai dengan Firefox: Wadah plugin. Dari waktu ke waktu flash player menutup telepon dan dengan Firefox saya sangat senang karena saya harus mematikan penampung plugin, memuat ulang halaman dan semuanya baik-baik saja.

Sekarang, dengan chrome, browser itu sendiri menutup, jadi saya harus memulai ulang chrome - dan dengan ini semua 5 jendela dan 40 tab saya.

Jadi, apakah ada ekstensi untuk chrome, yang menambahkan lapisan baru antara browser dan pemutar flash, sehingga saya hanya perlu mematikan pemutar flash itu sendiri?

Chrome juga multi-proses. Jika Anda menekan Bergeser + ESC Anda dapat membuka pengelola tugas Chrome dan mematikan instance atau tab Flash yang tidak merespons tanpa mematikan seluruh browser.

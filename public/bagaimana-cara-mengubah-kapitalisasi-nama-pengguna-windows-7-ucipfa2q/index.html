Bagaimana cara mengubah kapitalisasi nama pengguna Windows 7?

Saat saya menginstal Windows kali ini, saya salah memasukkan nama saya sebagai "myname". Melihat ini mengganggu saya setiap kali saya logon, dan saya ingin mengubahnya menjadi "Nama Saya".

Sayangnya, saat saya mencoba melakukan ini dengan panel kontrol "Akun Pengguna", saya diberi tahu bahwa "sudah ada akun dengan nama itu". Saya membayangkan bahwa nama pengguna tidak peka huruf besar-kecil untuk tujuan praktis - karenanya tidak mengizinkan saya untuk mengubahnya - tetapi bagaimana saya dapat mengubah kapitalisasi nama yang ditampilkan di layar Selamat Datang?

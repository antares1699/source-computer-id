Synergy terhubung, tetapi mouse tidak mau beralih layar (dan tidak ada keluaran debug yang menunjukkan masalah)

Mencoba mengatur Synergy, dan sepertinya tidak bisa melakukannya dengan benar. Inilah konfigurasinya:

  • Server: WinXP dengan layar bernama TV
  • Klien: Ubuntu 11.04 bernama LAPTOP

Konfigurasi server:

section: screens
    LAPTOP:
        halfDuplexCapsLock = false
        halfDuplexNumLock = false
        halfDuplexScrollLock = false
        xtestIsXineramaUnaware = false
        switchCorners = none +top-left +bottom-left 
        switchCornerSize = 0
    TV:
        halfDuplexCapsLock = false
        halfDuplexNumLock = false
        halfDuplexScrollLock = false
        xtestIsXineramaUnaware = false
        switchCorners = none +top-right +bottom-right 
        switchCornerSize = 0
end

section: links
    LAPTOP:
        right = TV
    TV:
        left = LAPTOP
end

section: options
    relativeMouseMoves = false
    screenSaverSync = false
    win32KeepForeground = false
    switchCorners = none 
    switchCornerSize = 0
    keystroke(F12) = lockCursorToScreen(toggle)
end

Dapatkah BitTorrent Sync CLI digunakan untuk membuat saham?

BitTorrent Sync dapat dijalankan dari baris perintah, tetapi hanya ada beberapa opsi yang ditampilkan oleh parameter --help. Menjalankan btsync membuat UI web tersedia (di port 8888) yang dapat digunakan untuk mengkonfigurasi.

BitTorrent Sync 1.2.73
Usage:
      btsync [--config <path>] [--nodaemon] [--generate-secret] [--dump-sample-config] [--help] [--get-ro-secret <secret>]
Options:
    --config - location and name of configuration file
    --nodaemon - do not use daemon mode
    --generate-secret - generate shared secret
    --get-ro-secret - get read only secret for existing master secret
    --dump-sample-config - dump sample config file
    --help - print this message and exit

Opsi ini sepertinya tidak termasuk membuat folder bersama baru.

Dapatkah saya membuat share baru dengan BitTorrent Sync hanya menggunakan baris perintah?

Saya juga tertarik dengan solusi yang sangat hacky, seperti mengedit file konfigurasi dan memulai ulang layanan, atau memposting permintaan HTTP tertentu ke localhost: 8888.

Anda perlu menggunakan file konfigurasi untuk ini.

Anda dapat menggunakan btsync --dump-sample-config (atau --get-sample-config jika Anda menggunakan versi yang lebih lama) untuk mendapatkan sampel, memasukkannya ke jalur yang diperlukan di bawah "folder bersama" dan rahasia yang dihasilkan oleh --generate sebuah rahasia.

Kemudian simpan file konfigurasi lalu beri tahu btsync untuk menggunakan file konfigurasi itu dengan meluncurkannya dengan --config.

Anda dapat menemukan semua ini di halaman 10 dari versi manual ini . Perhatikan bahwa ini akan menonaktifkan UI web.

Saya membiarkannya sebagai latihan bagi pengguna untuk membuat skrip semua ini dengan cara apa pun yang diinginkan hatinya.

Bagaimana cara menghapus ekstensi bat dari nama file di parameter?

Saya membuat file bat dan selama eksekusi, ini menampilkan judul di jendela perintah.

Judul termasuk nama file kelelawar, dan itu ditampilkan secara penuh, termasuk ekstensi .bat.

Kode yang saya gunakan untuk melakukan ini ditunjukkan di bawah ini (lihat perintah Title pada baris ke-5).

Nama-nama yang saya gunakan:

  • Nama file: Facebook.bat
  • Judul (baris 5): Hacking %~nx1 Called Domain yang menghasilkan Hacking Facebook.bat Called Domain di bilah judul.

Saya ingin bilah judul membaca Hacking Facebook Called Domain.

Bagaimana saya melakukan ini?

File bat saya:

@echo off
color 0d
call :sub *.bat
:sub
Title Hacking %~nx1 Called Domain
echo Test 1
timeout /t 2
echo Test 2
timeout /t 2
echo Completed
timeout /t 2
exit

Tangkapan layar tersebut membuat pertanyaannya jauh lebih jelas.

Saya melihat Anda menggunakan %~nx1 untuk menampilkan nama file dalam judul prompt perintah yang muncul.

Untuk menggunakan nama file tanpa ekstensi , alih-alih menggunakan %~nx1, gunakan %~n0.

yaitu:

@echo off
color 0d
call :sub *.bat
:sub
Title Hacking %~n0 Called Domain
echo Test 1
timeout /t 2
echo Test 2
timeout /t 2
echo Completed
timeout /t 2
exit

Saya mendapat %~n0 dari sini .

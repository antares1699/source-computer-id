"Lampu" stik USB di bawah Windows 7

Banyak USB stick (thumb drive) memiliki lampu kecil yang menunjukkan bahwa mereka aktif (misalnya banyak thumb drive SanDisk).

Di bawah Windows XP, saat Anda "Melepas Perangkat Penyimpanan Massal USB dengan Aman" XP dengan mudah mematikan lampu.

Di bawah Windows 7, lampu tetap menyala. Saya menyadari bahwa ini adalah "bagatelle belaka" (tetapi ternyata bermanfaat bagi saya), tetapi apakah mungkin untuk memulihkan fungsi itu? Registry? Masa bodo?

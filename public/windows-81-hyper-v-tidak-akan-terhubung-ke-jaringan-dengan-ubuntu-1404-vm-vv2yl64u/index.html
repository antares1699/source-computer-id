Windows 8.1- Hyper-V tidak akan terhubung ke jaringan dengan Ubuntu 14.04 VM

Yah, secara resmi saya tidak sabar untuk hal semacam ini, dan sekarang jam 12:23.

Saya mencoba menjalankan Ubuntu 14.04 dengan Hyper-V pada Lenovo Yoga 2 Pro yang menjalankan Windows 8.1. Saya tidak dapat menghubungkan mesin virtual ke internet - baik melalui adaptor nirkabel mesin saya atau melalui adaptor port Ethernet ke USB yang saya miliki. Saya telah melihat beberapa panduan tentang bagaimana ini dilakukan, tetapi tidak ada yang masuk akal, dan saya pikir saya hanya memperburuk keadaan.

Langkah demi langkah, bagaimana cara membuat koneksi menggunakan adaptor WiFi saya sehingga VM dapat mengakses web?

Ini bersifat konseptual, bukan langkah demi langkah, tetapi harus cukup menjelaskan untuk membantu Anda mengetahui apa yang perlu Anda lakukan. Sakelar Hyper-V yang digunakan VM harus berupa Eksternal atau Internal dan bukan Pribadi. Perlu diingat bahwa sakelar Eksternal akan terlihat seperti koneksi LAN ke VM, bukan koneksi nirkabel meskipun pada perangkat keras sebenarnya itu terkait dengan adaptor nirkabel. Selain itu, sakelar Eksternal mungkin tidak berfungsi dengan semua router nirkabel karena beberapa mengenali bahwa permintaan DHCP yang berasal dari VM Anda benar-benar berasal dari mesin yang sama dan tidak akan memberikan alamat.

Cara mengatasinya (router nirkabel terdengar seperti masalah dari deskripsi Anda) adalah mengatur sakelar Hyper-V untuk VM Anda menjadi Internal dan kemudian menggunakan ICS (Berbagi Koneksi Internet) pada adaptor nirkabel perangkat keras yang sebenarnya, berbagi dengan Hyper -V beralih. ICS ada di tab saat Anda membuka properti di adaptor yang hanya bertuliskan "Berbagi". Itu membuat adaptor Anda berperilaku seperti itu adalah router NAT ke adaptor lain di mesin Anda. Ini menanggapi permintaan DHCP VM dan memungkinkan VM untuk terhubung dengannya sehingga untuk berbicara.

Saya menggunakan sakelar eksternal yang saya buat di manajer Hyper-v.

Kemudian, pergi ke panel kontrol> Jaringan & berbagi> temukan sakelar eksternal yang saya buat di Hyper-V. Memeriksa propertinya. Di bawah IP v4 menetapkannya sebagai IP manual karena tidak mendapatkannya dari adaptor WiFi saya. Memberinya DNS, dan info gateway juga. BTW, saya menjalankan lingkungan ini pada laptop Windows 8.1 yang berjalan di Lenovo Flex. Saya tidak menggunakan koneksi kabel. Saya menggunakan koneksi WiFi saya.

Masalah ini tampaknya terjadi saat komputer Anda tidak tersambung ke Internet saat penginstalan sedang berjalan. Penginstal tidak akan dapat memverifikasi sumber apt-get yang ada, dan akan menonaktifkannya.

Dalam kasus saya, server DNS tidak diatur (jadi ping domain apa pun tidak akan berfungsi), jadi saya pertama-tama menjalankan sudo nano /etc/resolv.conf, dan menambahkan beberapa server DNS:

nameserver 8.8.8.8

nameserver 8.8.4.4

pinging server sekarang berhasil.

Kemudian, untuk mengaktifkan sumber apt-get, saya menjalankan sudo nano /etc/apt/sources.list, dan menambahkan repositori bergulir Kali (dari http: //docs.kali.org/general-use/kali-linux-sources-list-repositories):

deb http://http.kali.org/kali kontribusi utama kali-rolling tidak gratis

#Untuk akses paket sumber, hapus komentar pada baris berikut

deb-src http://http.kali.org/kali kontribusi utama kali-rolling tidak gratis

Tutup nano, dan keluar dari cangkangnya. Jalankan apt-get update, lalu apt-get install grub-pc. Keduanya berhasil, dan saya keluar dari shell untuk menyelesaikan instalasi GRUB melalui GUI.

Untuk beberapa alasan, Linux tidak dapat mendeteksi koneksi jaringan dengan benar dan akibatnya grub tidak dapat diinstal di akhir penginstalan.

Terlepas dari apa yang dikatakan beberapa orang di atas, saya tidak berhasil dengan mengubah antarmuka jaringan utama ke adaptor ethernet kabel lain, adaptor nirkabel dan bahkan beralih di antara berbagai hotspot untuk nirkabel seperti hotspot pribadi di ponsel saya ...

Dalam kasus saya, saya dapat melewati masalah dengan mengklik kembali dan pergi ke manajer paket dan mengubah mirror jaringan dari YA ke TIDAK. Kemudian klik berikutnya dan selesai instalasi dengan sempurna! Namun setelah instalasi, saya harus boot dan mengaktifkan kembali mirror jaringan agar dapat menginstal aplikasi dari repositori online. Cukup keluarkan perintah berikut:

gedit /etc/apt/sources.list

dan hapus komentar pada deb http://http.kali.org/kali kali-rolling main contrib non-free

lalu ketik

apt-get update

Semoga ini membantu!

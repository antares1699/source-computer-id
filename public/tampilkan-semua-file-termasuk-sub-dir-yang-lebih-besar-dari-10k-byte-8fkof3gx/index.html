Tampilkan semua file (termasuk sub-dir) yang lebih besar dari 10K byte

Saya tahu saya seharusnya menggunakan ls, tetapi ada banyak cara untuk melakukan hal ini, bukan? Saya mencoba ls -S +10 k, tapi tidak ada. Apa yang harus saya lakukan?

Anda dapat menggunakan perintah find sebagai gantinya:

find . -type f -size +10k

Jika Anda ingin membatasi hanya pada tingkat direktori saat ini:

find . -maxdepth 1 -type f -size +10k

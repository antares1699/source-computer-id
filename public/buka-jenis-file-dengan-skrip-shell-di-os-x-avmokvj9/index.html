Satu-satunya cara adalah mengemas skrip sebagai aplikasi.

Untuk portabilitas, Anda cukup membuat penampung AppleScript yang akan menjalankan skrip shell yang Anda buat.

Periksa do shell script untuk informasi lebih lanjut.

Saat Anda pergi untuk menyimpan skrip, pastikan untuk menyimpannya sebagai "Application" atau "Application Bundle".

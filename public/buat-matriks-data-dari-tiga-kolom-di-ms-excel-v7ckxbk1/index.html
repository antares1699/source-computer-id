Buat matriks data dari tiga kolom di MS Excel

Saya memiliki tiga kolom dengan data, misalnya:

 A     B            C          D
 1   [From]       [To]      [Total]
 2   Vancouver   Shanghai    100,000
 3   Vancouver   Melbourne    50,000
 4   Shanghai    Melbourne    26,000
 5   Vancouver   Shanghai    150,000
 6   Cape town   Vancouver     5,000
 7   Cape town   Vancouver    50,000
 8   etc         etc          etc

Dan ini berlanjut hingga 350 baris, dengan beberapa tujuan [Dari] dan [Ke] sering berulang untuk Total yang berbeda. Saya juga sering harus menambahkan tujuan baru dan mengubah totalnya. (artinya saya tidak bisa benar-benar mengurutkan atau mengubah data asli)

Saya ingin membuat matriks keluaran di tempat lain di spreadsheet, yang menemukan dan mencantumkan tujuan [Dari] vs [Ke] dan menjumlahkan total keseluruhan untuk setiap "kaki", agar terlihat seperti ini:

 A     B            C          D           E           F    
 1               Shanghai     Melbourne  Vancouver    etc   
 2   Vancouver   250,000       50,000        0         
 3   Melbourne      0            0           0
 4   Shanghai       0          26,000        0
 5   Cape town      0            0          55,000
 6   etc

Pada dasarnya kebalikan dari ini: Ubah urutan tabel bergaya matriks menjadi 3 kolom di Excel , tetapi dapat menambahkan baris / tujuan baru dan memperbarui matriksnya sendiri.

Setiap panduan sangat dihargai!

Gunakan Tabel Pivot. Letakkan [Dari] di Baris Label, [Ke] di Label Kolom dan [Total] di Nilai. Bergantung pada bagaimana Anda ingin memperlakukan rute duplikat, Anda mungkin ingin menggunakan Jumlah, Rata-rata, atau Maks pada agregasi Nilai.

Jika Anda mengubah rentang data Anda menjadi Daftar sebelum Anda membuat tabel pivot maka Anda harus dapat menambahkan data ke daftar dan kemudian menyegarkan tabel pivot untuk memasukkan data baru.

Anda mungkin ingin menonaktifkan total baris dan kolom di tabel pivot juga untuk kejelasan.

Notepad ++ Cara memotong semua teks setelah karakter tertentu

Saya memiliki beberapa teks yang ingin saya edit Notepad ++ .

Teks mengikuti pola seperti test 1 - test1, test 2 - test2, test 3 - test3 dan seterusnya, satu per baris. Saya ingin 'memotong' semua karakter di setiap baris setelah karakter / simbol '-'. Saya membutuhkan semua yang telah dipotong untuk dimasukkan ke dalam clipboard sehingga saya dapat menempelkannya di tempat lain. (Jika awalan tidak dapat diletakkan di papan klip, maka cukup cara menghapus semua karakter setelah '-' sudah cukup).

Dengan mencari \ ganti di notepad menggunakan -. * Akan menghapus semuanya setelah '-'. Saya perlu tahu cara memotong dan menyalinnya. Jika saya dapat mengetahui cara menghapus semuanya SEBELUM '-', seperti yang dilakukan di atas, saya dapat menyalin semuanya. Ada ide? Akan jauh lebih mudah.

Untuk mendapatkan awalan (teks sebelum tanda hubung), tekan Ctrl + H dan isi bidang sebagai berikut:

Untuk mendapatkan sufiks (teks setelah tanda hubung), tekan Ctrl + H dan isi bidang sebagai berikut:

Ini rusak jika ada tanda hubung di awalan atau akhiran.

Jika Anda perlu mendapatkan keduanya secara terpisah, cukup tangkap hasil dari penggantian pertama ke file yang berbeda (atau clipboard), lalu tekan Ctrl + z dan lakukan penggantian kedua.

Jika Anda perlu menyertakan tanda hubung dan / atau spasi, Anda dapat menambahkannya sesuai kebutuhan di kotak teks "Ganti dengan:" sebelum atau setelah placeholder pengganti.

Buka file di MS Excel dan ikuti prosedur di bawah ini:

  1. Pilih opsi 'Dibatasi'

  2. Tetapkan pembatas sebagai -

  3. Klik 'Selesai' dan selesai!

Untuk menghapus semuanya sebelum - dan simbol itu sendiri, gunakan regexp [^\-\n\r]+- (tanpa tanda hubung dan garis putus-putus diikuti dengan tanda hubung).

Ganti teks yang ditemukan dengan string kosong dan pilih yang tersisa.

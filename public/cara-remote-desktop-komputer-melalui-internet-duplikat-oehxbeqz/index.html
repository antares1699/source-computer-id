Cara remote desktop komputer melalui internet [duplikat]

Kemungkinan Duplikat:
Koneksi jarak jauh ke PC melalui internet
Aplikasi desktop jarak jauh gratis bagus untuk bekerja dengan komputer seseorang yang buta huruf?

Kakak saya sudah mulai menggunakan komputer / windows XP dan sudah mendapat koneksi internet. Ini adalah PC rumahnya. Sekarang dia memiliki banyak masalah yang melibatkan penggunaannya, penginstalan anti-virus dan perangkat lunak lain, beberapa atau masalah lainnya. dan sulit bagi saya untuk memberi tahu dia melalui telepon cara memperbaikinya, memahami pesan kesalahan, dll. Karena dia memiliki koneksi internet yang cukup cepat, saya bertanya-tanya berapa banyak cara untuk mencapainya sehingga saya dapat melakukan desktop jarak jauh pada mesinnya dari mesin saya dan memperbaiki sistemnya, jika dia mengalami masalah.

Sistem saya juga menjalankan windows XP.

Perangkat lunak klien / server desktop jarak jauh pihak ketiga mana pun yang dapat saya instal di mesin saya dan memandu dia untuk menginstal di mesinnya dan sejak saat itu mengendalikan sistemnya, kapan pun saya mau.

Idealnya, saya mencari sesuatu yang sederhana, ringan, antarmuka pengguna dasar, yang tidak bergantung pada bagaimana dia terhubung ke internet dan bagaimana saya terhubung ke internet, versi windows, bahkan OS jadi apakah ada solusi di mana saya dapat memberikan dukungan dari Mac saya sebagai baiklah .. sedangkan dia tidak perlu mengubah apapun di mesinnya ..

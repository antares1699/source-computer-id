zsh cdpath dan pelengkapan otomatis

CDPATH saya di zsh terlihat seperti ini:

export CDPATH='.:..:../..'

Sayangnya, ini berarti bahwa ketika saya mencoba melengkapi otomatis setelah mengetik 'cd', ada terlalu banyak opsi pelengkapan otomatis yang tidak relevan, tergantung di mana saya berada. Misalnya, jika saya berada di direktori home saya, mengetik cd akan mencoba melengkapi otomatis setiap pengguna di sistem karena CDPATH memiliki ".."

Apakah ada cara untuk menyimpan CDPATH saya seperti yang ditunjukkan di atas, tetapi apakah pelengkapan otomatis untuk cd, pushd, dll. Mengabaikan nilai CDPATH, dan pelengkapan otomatis hanya berdasarkan direktori saat ini?

path-directories

Salah satu caranya adalah dengan menambahkan pengaturan penyelesaian berikut di .zshrc Anda untuk menghapus path-directories dari sumber saran.

zstyle ':completion:*:complete:(cd|pushd):*' tag-order \
    'local-directories named-directories'

nama grup

Sebagai alternatif atau tambahan, pengaturan berikut akan menampilkan judul untuk semua grup saran penyelesaian sehingga Anda dapat melihat direktori mana yang merupakan direktori lokal dan mana yang merupakan saran dari cdpath Anda.

zstyle ':completion:*' group-name ''
zstyle ':completion:*:descriptions' format %d

Anda dapat menerapkan format perintah standar ke deskripsi berikut untuk membuatnya menonjol:

zstyle ':completion:*:descriptions' format %B%d%b        # bold
# zstyle ':completion:*:descriptions' format %S%d%s        # invert/standout
# zstyle ':completion:*:descriptions' format %U%d%u        # underline
# zstyle ':completion:*:descriptions' format %F{green}%d%f # green foreground
# zstyle ':completion:*:descriptions' format %K{blue}%d%k  # blue background
# etc.

Itu sedikit membantu memahami sumber yang berbeda dalam pengalaman saya.

Catatan 1 : zsh memiliki dua representasi untuk variabel larik seperti PATH dan CDPATH, yang varian huruf kecilnya adalah larik sebenarnya. Artinya, Anda dapat:

cdpath=(path/to/dir /path/to/another/dir)

Atau, untuk mendapatkan hasil yang Anda inginkan:

cdpath=(.. ../..)

Saya pribadi merasa itu sedikit lebih mudah dibaca daripada pseudo array yang dipisahkan titik dua.

Catatan 2 : Mengekspor CDPATH tidak diperlukan kecuali Anda memiliki beberapa program yang ingin disetel.

Bagaimana cara mengubah kunci produk untuk Microsoft Office 2007?

Saya telah diminta untuk "memasukkan kembali" salinan MS Office 2007 saya untuk menggunakan kunci produk yang berbeda. Apakah ini mungkin? Jika ya, bagaimana saya melakukannya?

Microsoft menawarkan jawaban untuk pertanyaan ini sini . Ini juga berlaku untuk versi Microsoft Office sebelumnya.

  • Tutup semua program Office.
  • Klik Mulai, klik Jalankan, ketik regedit, lalu klik OK.
  • Temukan dan kemudian klik subkunci berikut, tergantung pada versi Anda Windows dan Office:

Versi 32-bit

2007 Office system
HKEY_LOCAL_MACHINE \Software\Microsoft\Office\12.0\Registration

Office 2003 
HKEY_LOCAL_MACHINE \Software\Microsoft\Office\11.0\Registration

Office XP
HKEY_LOCAL_MACHINE \Software\Microsoft\Office\10.0\Registration

Versi 64-bit

2007 Office system
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Office\12.0\Registration

Office 2003
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Office\11.0\Registration

Office XP
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Office\10.0\Registration

  • Ekspor subkunci Registrasi

    • Anda dapat membuat cadangan nilai di bawah subkunci Registrasi jika kunci produk baru tidak berfungsi. Untuk melakukannya, ikuti langkah-langkah berikut:
      • Klik kanan subkunci Pendaftaran, lalu klik Ekspor.
      • Dalam daftar Simpan di, pilih lokasi untuk menyimpan pendaftaran entri.
      • Di kotak Nama file, ketikkan nama untuk file .reg yang Anda inginkan untuk mengekspor, lalu klik Simpan.

  • Di bawah subkunci Pendaftaran, mungkin ada beberapa Unik Secara Global Pengidentifikasi (GUID) yang berisi file kombinasi alfanumerik karakter. Setiap GUID dikhususkan untuk a program yang diinstal pada Anda komputer.

    • Klik sebuah GUID. Lalu, lihat versi Office yang tercantum di panel kanan untuk Productname entri registri. Setelah Anda menemukan file GUID yang berisi versi Anda Program Office, ikuti langkah-langkah berikut:

      • Klik kanan entri registri DigitalProductID, klik Ya Hapus, lalu klik Ya.
      • Klik kanan entri registri ProductID, klik Hapus, lalu klik Yes.

  • Buka program Office, seperti Microsoft Word. Saat Anda menerima file pesan yang meminta Anda untuk kunci produk, ketik produk yang benar kunci, dan kemudian klik OK.

Bagikan

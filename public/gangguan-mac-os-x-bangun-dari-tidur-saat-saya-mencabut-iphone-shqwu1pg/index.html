Gangguan Mac OS X: Bangun dari tidur saat saya mencabut iPhone

Saat Mac saya ditangguhkan (mode tidur), dan saya mencabut iPhone saya, komputer selalu aktif. Ini menjengkelkan karena saya tidak ingin menyalakan komputer; Saya hanya ingin iPhone saya.

Adakah cara untuk menonaktifkan perilaku ini?

Saya tidak berpikir demikian karena mesin perlu menjalankan beberapa rutinitas untuk menangani Anda mencabut perangkat.

EDIT: Biar saya jelaskan sedikit lebih jelas :)

Jika Anda menulis aplikasi yang berlangganan acara mouse, tidakkah Anda ingin tahu apakah mouse telah menghilang sepenuhnya dari sistem? Contoh yang lebih penting adalah subsistem OS X yang mendeteksi ketika pengguna memasukkan informasi sehingga ia dapat terjaga dari tidurnya ... jika mesin tidak bangun ketika Anda mencabut sesuatu, ia tidak akan pernah tahu bahwa itu ada lagi ketika Anda mencolokkannya kembali, jadi jika Anda mencabut keyboard atau mouse dan memasangnya kembali, menekan tombol atau mengklik mouse, sistem tidak akan bangun lagi karena tidak menjalankan rutinitas untuk menanganinya. Saya harap itu masuk akal bagi seseorang :)

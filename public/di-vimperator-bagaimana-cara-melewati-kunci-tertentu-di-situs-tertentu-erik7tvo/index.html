Di vimperator, bagaimana cara melewati kunci tertentu di situs tertentu?

Saya ingin menyetel kunci tertentu untuk diteruskan di situs tertentu. Jadi, misalnya, j / k akan diteruskan pada *duckduckgo.com* tetapi berfungsi di tempat lain.

Apakah ada cara mudah untuk melakukan ini?

Saya beralih kembali ke vimperator dari pentadactyl (terlalu lambat) dan saya membutuhkan fitur ini. Ini disediakan oleh perintah :ignorekeys, seperti:

:ignorekeys add http://example.com/

Ini akan mengabaikan semua kunci, termasuk : dan / jadi Anda mungkin menginginkan setidaknya sesuatu seperti:

:ignorekeys add -except=:,/ http://example.com/

Lihat :help :ignorekeys untuk lebih lanjut!

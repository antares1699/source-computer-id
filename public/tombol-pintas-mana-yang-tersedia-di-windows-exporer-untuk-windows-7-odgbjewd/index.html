Microsoft memiliki referensi yang bagus untuk semua pintasan keyboard di Windows 7. Mengenai Windows Explorer, mereka mencantumkan yang berikut ini, antara lain:

  • Ctrl + N = Buka jendela baru
  • Ctrl + W = Tutup jendela saat ini
  • Ctrl + Bergeser + N = Buat folder baru
  • Akhir = Tampilkan bagian bawah jendela aktif
  • Rumah = Tampilkan bagian atas jendela aktif
  • F11 = Maksimalkan atau minimalkan jendela aktif
  • Ctrl + . = Putar gambar searah jarum jam
  • Ctrl + , = Putar gambar berlawanan arah jarum jam
  • Num Lock + Num * = Tampilkan semua subfolder di bawah folder yang dipilih
  • Num Lock + Num + = Tampilkan isi folder yang dipilih
  • Num Lock + Jumlah - = Tutup folder yang dipilih
  • Kiri = Tutup pilihan saat ini (jika diperluas), atau pilih folder induk
  • Alt + Memasukkan = Buka kotak dialog Properti untuk item yang dipilih
  • Alt + P. = Tampilkan panel pratinjau
  • Alt + Kiri = Lihat folder sebelumnya
  • Menghapus = Lihat folder sebelumnya
  • Baik = Tampilkan pilihan saat ini (jika diciutkan), atau pilih subfolder pertama
  • Alt + Baik = Lihat folder berikutnya
  • Alt + Naik = Lihat folder induk
  • Ctrl + Bergeser + E = Tampilkan semua folder di atas folder yang dipilih
  • Ctrl + Roda gulir mouse = Ubah ukuran dan tampilan ikon file dan folder
  • Alt + D = Pilih bilah alamat
  • F4 = Tampilkan dropdown bilah Alamat dari riwayat terkini
  • Ctrl + E = Pilih kotak pencarian
  • Ctrl + F = Pilih kotak pencarian
  • F3 = Pilih kotak pencarian

Windows 10 - "Penulis Dokumen Microsoft XPS" dan "Microsoft Print To PDF" tidak lagi berfungsi

Untuk beberapa waktu di PC desktop Windows 10 saya, driver cetak virtual "Microsoft XPS Document Writer" atau "Microsoft Print To PDF" juga tidak berfungsi.

Setiap kali saya mencoba untuk mencetak dokumen menggunakan salah satunya, itu meminta dialog file bagi saya untuk memilih nama file dan lokasi, dan ketika saya mengklik OK untuk mencetak, itu tidak melakukan apa-apa. Jendela dialog menghilang, tidak ada kesalahan yang ditampilkan, dan tidak ada lagi yang terjadi. Ketika saya melihat direktori tempat file seharusnya dibuat, tidak ada apa-apa, bahkan file nol byte. Direktori tempat saya menyimpan file tidak memiliki karakter khusus di dalamnya (tanpa koma), dan saya memiliki akses admin penuh ke sana. Windows Event Viewer juga tidak menampilkan kesalahan apa pun.

Jika saya mencoba mencetak halaman uji menggunakan panel properti Printer dari setiap printer virtual, saya mendapat pesan kesalahan yang mengatakan "Halaman uji gagal dicetak." dan mengusulkan saya untuk menggunakan pemecah masalah (yang saya lakukan dan tidak menemukan apa pun).

Saya juga mencoba mencopot dan menginstal ulang kedua printer (menggunakan panel "Aktifkan atau nonaktifkan fitur Windows"), dengan reboot di antaranya, tetapi itu tidak memperbaiki masalah.

Saya memiliki printer jaringan yang terpasang yang berfungsi dengan baik. Kontrol akses pengguna juga dinonaktifkan (geser sepenuhnya ke bawah) dan jenis akun saya adalah administrator.

Ini digunakan untuk bekerja sebelumnya (meskipun saya tidak bisa menentukan kapan tepatnya saya tidak cukup sering menggunakannya). Saya menduga itu disebabkan oleh pembaruan Windows, tetapi karena pembaruan Windows 10 bersifat wajib dan tidak dapat dengan mudah dikecualikan, bahkan jika saya menghapus pembaruan terakhir dalam daftar Pembaruan yang Terinstal, pembaruan tersebut akan kembali begitu saya reboot, dan saya bisa ' t menyembunyikannya seperti yang saya bisa di Windows 7.

Ini adalah penginstalan baru Windows 10 Pro, dan bukan peningkatan.

Saya telah menghabiskan cukup banyak waktu mencari Google untuk masalah ini tetapi satu-satunya kasus yang saya temukan adalah tentang koma di nama file atau jalur dan hanya memengaruhi pencetakan dari Edge, yang bukan kasus saya. Bantuan apa pun akan dihargai. Terima kasih!

Ketika CEO Microsoft Satya Nadella memutuskan untuk melakukannya memotong 5.500 karyawan Microsoft , dengan Windows yang paling terpukul, banyak yang telah berubah di dunia Windows. Nadella telah menghentikan banyak teknisi pengujian, menambahkan peran pengujian ke manajer produk, dan lebih mengandalkan Program Windows Insider .

Sayangnya, itu bertepatan dengan rilis Windows 10, sistem operasi yang terlalu terburu-buru dan tidak cukup menguji, mengandalkan ritme yang sangat cepat agar pembaruan diperbaiki masalah yang diharapkan, yang tiba segera setelahnya.

Sayangnya lagi, karena kurangnya pengujian dan jaminan kualitas, perbaikan yang sekarang sedang dilakukan juga cenderung menimbulkan masalah dalam giliran mereka.

Dengan sangat sedih saya menyaksikan di forum ini, setelah setiap pembaruan besar Windows 10, datangnya masalah baru. Kebanyakan dari mereka tidak memiliki solusi kecuali menunggu perbaikan dari Microsoft atau menurunkan versi kembali ke Windows 7 yang sangat stabil.

Jadi ada sedikit solusi yang bisa saya tawarkan untuk masalah Anda:

  1. Lari sfc / scannow untuk memverifikasi bahwa, untuk Windows, keadaan rusak sebenarnya adalah "normal".

  2. Periksa Peraga Peristiwa untuk pesan kesalahan yang berkaitan dengan masalah Anda.

  3. Laporkan bug ke Microsoft menggunakan aplikasi "Windows Feedback" dan tunggu sampai diperbaiki.

  4. Gunakan produk pihak ketiga, seperti CutePDF Writer yang sudah Anda miliki mencoba dan terbukti berfungsi dengan baik, sebagai solusi sementara hingga masalah diperbaiki oleh Microsoft.

Sebagai tindakan putus asa, dalam hal pembaruan Windows 10 1511 menyebabkan masalah, Anda mungkin dapat menemukan ISO pasca-1511 dan melakukannya instalasi bersih Windows 10. Ini adalah sedikit harapan bahwa beberapa Komponen Windows yang sebelumnya tidak sepenuhnya diperbarui sekarang akan berfungsi dengan benar.

Pada pertanyaan Anda, Anda mengatakan bahwa Anda mencoba mencopot pemasangan fitur menggunakan 'Mengaktifkan atau menonaktifkan fitur Windows', mari coba pendekatan lain:

  • Cari Device Manager, atau lakukan Win + R lalu DEVMGMT.MSC.
  • Buka 'Antrean cetak' lalu hapus instalan antrean cetak PDF dan XPS.
  • Cari 'Advanced Printer Setup', atau lihat opsi ini di Control panel.
  • Pilih 'Printer yang saya inginkan tidak terdaftar', di jendela berikutnya pilih 'Tambahkan printer lokal atau printer jaringan dengan pengaturan manual', pilih 'Gunakan port yang ada:' lalu di dropdown pilih 'FILE: (Cetak ke Mengajukan)'.
  • Di jendela berikutnya, akan muncul daftar. Pilih Microsoft lalu di daftar kedua pilih 'Microsoft Print to PDF' dan selanjutnya.
  • Pilih 'Ganti driver saat ini' dan kemudian berikutnya
  • Beri nama printer (saya sarankan nama aslinya) kemudian berikutnya dan itu akan menginstal ulang printer dan drivernya.
  • Ulangi prosesnya tetapi sekarang alih-alih memilih 'Microsoft Print to PDF' dalam daftar, pilih yang XPS.
  • Mulai ulang dan lihat apakah sudah dicetak sekarang.

Setting monitor utama di ubuntu tanpa xrandr

Saya menjalankan Kubuntu 12.10. Saya memiliki 3 monitor yang dicolokkan ke AMD Radeon 7970. Saya telah mencoba mengganti urutan output, dan mengkonfigurasinya berulang kali dengan amdcccle tetapi kubuntu sepertinya selalu membuat monitor terjauh di sebelah kiri sebagai monitor utama. Solusinya sejauh ini adalah melakukan hal berikut.

xrandr --query

Yang mengembalikan daftar monitor saya. Kemudian secara eksplisit mengatur primer dengan xrandr dengan

xrandr --output <monitor identifier> --primary

Saya menambahkan perintah ini ke skrip start up, tetapi masih tampak hacky. Sistem masih melakukan booting dengan monitor paling kiri sebagai yang utama, dan kemudian beralih ke monitor tengah saya setelah penyalaan. Beberapa aplikasi saya masih memperlakukan monitor kiri sebagai yang utama dan akan diluncurkan di sana, bukan di monitor yang saya inginkan sebagai yang utama.

Saya menemukan bahwa mengedit xorg.conf bekerja paling baik untuk saya (saya menggunakan KDE).

Saya menambahkan Opsi "Utama" "benar" di bawah monitor yang saya inginkan sebagai yang utama dan sekarang hampir semuanya bekerja seperti yang diharapkan.

Bagian dari xorg.conf saya yang saya edit terlihat sebagai berikut

Section "Monitor"
  Identifier   "0-DFP10"
  Option      "VendorName" "ATI Proprietary Driver"
  Option      "ModelName" "Generic Autodetecting Monitor"
  Option      "DPMS" "true"
  Option      "PreferredMode" "1920x1080"
  Option      "TargetRefresh" "60"
  Option      "Position" "1680 0"
  Option      "Rotate" "normal"
  Option      "Disable" "false"
  Option      "Primary" "true"
EndSection

Beberapa aplikasi masih terbuka di layar yang berbeda dari layar utama saya, saya tidak yakin bagaimana cara memperbaikinya, atau apakah itu sesuatu yang dapat saya perbaiki. Tapi sejauh yang saya ketahui, masalah ini sudah terpecahkan.

Jika saya tidak salah xrandr hanya mengubah pengaturan dengan cepat, jadi jika Anda ingin membuatnya secara permanen, Anda harus menyimpannya menggunakan metode lain.

Cara termudah di Gnome adalah mengedit ~/.config/monitors.xml. Di sana Anda dapat mengatur monitor mana yang utama. Contoh:

<monitors version="1">
  <configuration>
      <clone>no</clone>
      <output name="LVDS1">
          <vendor>AUO</vendor>
          /* (...) */
          <primary>yes</primary>
      </output>
      <output name="VGA1">
      </output>
      <output name="HDMI1">
          <vendor>SAM</vendor>
          /* (...) */
          <primary>no</primary>
      </output>
      <output name="DP1">
      </output>
  </configuration>
</monitors>

Ini berfungsi di Ubuntu (Gnome / Unity), tetapi saya belum pernah menggunakan Kubuntu sehingga mungkin menggunakan lokasi yang berbeda, atau bahkan menggunakan file lain. Selain itu, ini hanya akan diterapkan setelah Anda masuk.

Jika itu tidak berhasil, Anda harus mengedit xorg.conf. Saya tidak dapat banyak membantu Anda dalam hal itu, tetapi dalam pencarian cepat saya menemukan ini posting yang akan membantu. Lihat juga ini Halaman Wiki Ubuntu.

Semoga ini membantu.

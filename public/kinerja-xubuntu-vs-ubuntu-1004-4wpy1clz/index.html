Kinerja Xubuntu vs Ubuntu 10.04

Saya tahu itu baru saja keluar hari ini, tetapi apakah ada statistik dengan persyaratan memori, sumber daya sistem, penggunaan daya, dan kinerja untuk memutuskan mana yang lebih baik Xubuntu atau Ubuntu (XFCE vs Gnome)?

Perhatian utama saya menjalankannya sebagai OS sekunder di laptop Lenovo T400 saya untuk online dengan cepat dan menggunakan SSH dari terminal untuk terhubung ke server web jarak jauh.

Masih terlalu dini untuk statistik. Tapi inilah perbandingan kinerja Ubuntu-vs-Xubuntu umum, dicampur dengan pengalaman saya tentang kandidat rilis 10.04. Laptop lama saya (~ 2005) menjalankan Xubuntu 9.04 dengan cukup nyaman untuk waktu yang lama. Saya rasa saya tidak pernah mencoba Ubuntu 9.04 standar di atasnya.

  • Ubuntu menggunakan lingkungan desktop Gnome yang banyak sumber daya secara default. Di laptop saya, Ubuntu 10.04 RC cukup lamban, bahkan dengan efek desktop dimatikan. Anehnya, LiveCD jauh lebih responsif daripada desktop default setelah penginstalan.

  • Xubuntu adalah Ubuntu dengan lingkungan desktop XFCE yang lebih ringan. Secara umum, ini lebih cocok untuk mesin lama dengan memori lebih sedikit dan perangkat keras yang kurang berkemampuan (CPU / GPU / dll). XFCE dikenal sangat ringan pada sumber daya, tetapi build XFCE default Ubuntu dianggap lebih berat daripada XFCE normal.

  • Lubuntu adalah desktop Ubuntu lain yang didasarkan pada lingkungan desktop LXDE. Ini dirancang agar lebih ringan dari XFCE.

Sebelum harddisk laptop saya mati, saya menginstal lubuntu-desktop dan xubuntu-desktop meta-paket. Setelah mereka diinstal, saya menghabiskan waktu untuk mencoba sesi Gnome / Openbox di pemilih sesi jendela login. (Openbox adalah manajer jendela default di desktop Lubuntu.) Sesi Gnome / Openbox jauh lebih responsif daripada sesi Gnome standar, tetapi ada beberapa applet panel yang buggier daripada di bawah Gnome normal.

Saya belum sempat mencoba sesi Lubuntu atau Xubuntu.

Secara umum ada perbedaan sekitar 40-50MB di edisi Linuxmint Xfce vs Gnome (120 vs 170). (Linux mint adalah turunan langsung dari Ubuntu).

Namun, ada satu hal lucu yang saya amati - setelah mengaktifkan Compiz (tema efek visual) - ada peningkatan konsumsi RAM (sekitar 30-40MB) di edisi Xfce - namun, hampir tidak ada peningkatan pada edisi Gnome.

Gambar di atas berasal dari Live CD.

Setelah dipasang, konsumsi RAM semakin berkurang 10-20MB.

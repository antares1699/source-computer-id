Xbox 360 benar-benar lambat sebagai Windows Media Extender

Ketika saya mengatur XBox 360 saya sebagai Windows Media Extender meskipun saya dapat melihat semua media yang saya miliki di PC Windows Vista, sangat lambat untuk berpindah antar album, dll.

Adakah yang dapat saya lakukan untuk memperbaikinya atau hanya karena jumlah file media yang saya miliki - sekitar 5GB?

Kebetulan, apakah PC Anda menggunakan koneksi nirkabel? Saya memiliki pengaturan itu dan Pusat Media memberi saya masalah yang persis sama. Ketika saya benar-benar menjalankan file audio atau video, streaming baik-baik saja, tetapi memilih item menu sangat lambat - perlu waktu 30 detik atau lebih hanya untuk berpindah dari satu item menu ke item menu lainnya. Itu a masalah yang sering dilaporkan .

Microsoft merekomendasikan menghubungkan PC langsung ke router, tetapi mereka mengatakan bahwa masalah kinerja terjadi jika kedua perangkat nirkabel, dan 360 saya terhubung langsung ke router.

Bagaimana cara mengubah port SSH2 default di ECLIPSE?

Saya ingin mengubah port SSH2 default di Eclipse dari 22 ke port lain. Saya mencoba mendefinisikan host seperti host: 55 (jika menggunakan 55 untuk SSH2) dan tidak berhasil.

Ada tip?

  • Saya mencoba untuk terhubung dengan plugin Remote System Explorer. kembali ke pengkodean dengan SmartFTP

Setelah Anda membuat koneksi, pergi ke Remote Systems Explorer (Window-> Views-> Remote Systems) klik kanan pada salah satu item anak koneksi dan pilih Properti . Di jendela properti pilih Subsistem kelompok. Port dapat dikonfigurasi dari layar ini.

Menghubungkan catu daya 20 pin ke papan 24 pin

Saya telah menjelajahi web dan tidak dapat menemukan manual untuk papan saya yang memberi tahu saya bagaimana saya harus menghubungkan catu daya baru saya.

  • PC: Gateway GT5628
  • Naik: Papan Desktop Intel DG33SXG2
  • Catu Daya Asli: Konektor generik 400w w / 24 pin
  • Pengganti Catu Daya: Antec Basiq 500w w / konektor 20 pin + 4 pin

Saya membaca di forum yang harus ditentukan oleh produsen papan di manual apakah akan menggunakan +4 atau tidak, namun saya tidak dapat menemukan dokumentasi di portal dukungan Intel atau Gateway untuk papan ini. (Gateway memiliki dokumen dukungan tetapi tidak ada yang mengacu pada catu daya)

Karena aslinya adalah 24, haruskah saya mencolokkan 20 terjauh ke kanan dan menambahkan +4 di kiri, atau haruskah saya membiarkan +4 dicabut?

Memperbarui

Saya dapat menemukan file Manual Antec jadi saya bisa melihat ke sisi mana harus menambahkan +4, saya hanya ingin memastikannya karena sepertinya bisa masuk ke kedua sisi. Pada titik ini saya berasumsi karena ada 24 sebelum saya harus menghubungkan ini sesuai.

Daftar catu daya baru sebagai 20 + 4 karena bagian belakangnya kompatibel dengan papan 20 pin. Jika Anda perhatikan, konektor 4 pin akan ditempatkan di 20 pin untuk membuat konektor 24 pin yang kokoh.

Selain itu, pin diberi kunci sehingga hanya dapat dipasang dengan mudah ke satu arah. Jika tampaknya tidak sesuai dengan konfigurasi yang diberikan, ubahlah hingga bergeser dengan mudah. perhatikan bentuk pada beberapa pin yang berbeda. Atur saja.

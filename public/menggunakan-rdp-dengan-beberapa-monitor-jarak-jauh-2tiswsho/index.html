Menggunakan RDP dengan beberapa monitor jarak jauh

Saya memiliki mesin windows 7 lokal dengan 1 monitor, dan mesin window 7 remote dengan 2 monitor.

Saya ingin RDP ke mesin jarak jauh, tetapi saya ingin dapat menduplikasi pengaturan layar ganda mesin jarak jauh saya, di mana saya dapat beralih di antara 2 monitor jarak jauh dari komputer lokal saya (fungsi ini tersedia di Teamviewer dan GOTOMYPC). Saat saya masuk, saat ini saya mendapatkan satu jendela yang menunjukkan ukuran ulang konten kedua layar jarak jauh saya ke satu layar.

Jadi saya ingin satu jendela di sisi lokal dengan sakelar / sakelar / tombol untuk beralih antara melihat layar jarak jauh saya 1 atau 2.

Apakah ini mungkin?

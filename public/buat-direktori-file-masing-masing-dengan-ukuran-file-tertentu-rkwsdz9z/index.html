Buat direktori file masing-masing dengan ukuran file tertentu

Saya ingin membuat 100 file masing-masing dengan ukuran file 10Mb. Apakah ada perintah unix sederhana yang dapat membuat direktori dengan 100 file ini?

for x in {1..100}; do dd if=/dev/zero of=file$x bs=10485760 count=1; done

if=FILE (read from FILE instead of stdin)
of=FILE (write to FILE instead of stdout)
bs=BYTES (force ibs=BYTES and obs=BYTES)
ibs=BYTES (read BYTES bytes at a time)
obs=BYTES (write BYTES bytes at a time)
count=1 (copy only 1 input block)

Untuk melengkapi jawaban Ochi, berikut adalah cara membuat 100 file 10 MB tanpa menggunakan ruang disk 1 GB dan berjalan secepat kilat:

size=$((10*1024*1024))
nb_files=100
for n in $(seq -f "%03.0f" 1 $nb_files); do
  dd if=/dev/zero of=file$n bs=1 count=1 seek=$((size-1))
done

(Lihat file jarang untuk lebih jelasnya.)

Atau dengan cara lain (tetapi di sini, jika Anda mengedit salah satu file, Anda mengedit semuanya):

size=$((10*1024*1024))
n=$(seq -f "%03.0f" 1 1)
dd if=/dev/zero of=file$n bs=1 count=1 seek=$((size-1))
nb_files=100
for m in $(seq -f "%03.0f" 2 $nb_files); do
  ln file$n file$m
done

(Lihat tautan keras untuk lebih jelasnya.)

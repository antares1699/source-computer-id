Tombol pintas Shift + Ctrl + End dan Shift + Ctrl + Home berhenti bekerja

Kunci pintas Bergeser + Ctrl + Akhir atau Bergeser + Ctrl + Rumah berhenti bekerja di editor teks apa pun seperti Microsoft Word 2016, Notepad ++ atau Notepad.

OS saya adalah Windows 10. Sangat berguna karena saya dapat memilih teks tanpa menggulir seluruh dokumen.

Apa yang dapat dilakukan agar tombol pintas ini berfungsi?

Beberapa program mungkin telah mengambil alih pintasan ini.

Coba boot dalam Safe Mode dan periksa apakah ini masih terjadi, lalu reboot dalam mode normal.

Jika masih terjadi, maka Windows rusak. Coba periksa integritasnya menggunakan SFC dan DISM .

Jika tidak terjadi lagi, maka beberapa program yang diinstal menyabotase ini tombol pintas. Unduh dan pasang Autoruns dan jalankan untuk melihat semua program startup. Di sini Anda dapat menonaktifkan dan mengaktifkan program dengan tanda centang mereka. Jika Anda tidak dapat menebak yang mana, Anda harus melakukannya cara yang sulit dengan menghapus centang mereka dalam tandan dan reboot. Aktifkan kembali yang tidak menimbulkan masalah.

Untuk menempatkan Komentar Bruno menjadi jawaban yang tepat, berikut cara memperbaikinya, diambil dari forum Opera:

https://forums.opera.com/topic/34954/opera-shortcut-key-delete-history/5

Tutup Opera, lalu buka folder Opera di dalam folder appdata Anda. Ini mungkin terlihat seperti:

C:\Users\USERNAME\AppData\Roaming\Opera Software\Opera Stable

Di dalam folder itu ada file Preferensi (tanpa ekstensi file). Buka dan hapus tombol Control-Shift-End dan Control-Shift-Home di JSON. Bagi saya, di Notepad, ini terlihat seperti:

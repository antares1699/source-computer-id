Apakah ada daftar perangkat lunak sumber terbuka yang diinstal dengan Snow Leopard?

adakah daftar yang tersedia dari alat pengembangan apa yang diinstal dengan Snow Leopard?

Saya hanya bisa melihat-lihat dan melihat apa yang diinstal dengan sembarangan, tetapi akan sangat membantu jika memiliki dokumen yang mengatakan "Ruby 1.8.7 - (lokasi), PHP 5.3 - (lokasi), Python (dll.)."

Ini akan mencakup hal-hal seperti Web dan server database, juga (saya melihat Apache ada di sana, dan SQLite, tetapi ingin memiliki daftar lengkap).

Saya telah memeriksa Catatan Rilis dan Panduan Instalasi XCode 3.2 tetapi tidak dapat menemukan daftar seperti itu, selain "Dukungan Pengembangan UNIX" yang diinstal di / usr.

Saya melakukan ini sebelum menginstal hal-hal khas di lokasi biasa (alih-alih pengaturan default, berpemilik tanpa indikasi apa yang dipasang di mana).

Ini a tautan dari situs Apple. Inilah yang dikatakannya, antara lain:

Mac OS X menyediakan yang paling populer alat pengembangan sumber terbuka, termasuk Koleksi Kompilator GNU (GCC), GNU Debugger (GDB), make, autoconf, dan banyak lagi.

Selain alat kompilator, Mac OS X juga menyediakan sejumlah besar file bahasa skrip.

Banyak dari alat kompilator adalah tersedia di bawah lisensi GPL.

Proyek open source berikut adalah tersedia melalui CVS: cygnus gcc gcc2 gcc3 gdb keymgr

Dan berikut adalah bahasa skrip yang didukung:

Bahasa skrip memanfaatkan kekuatannya dari baris perintah Unix. Mac OS X termasuk sumber terbuka paling populer bahasa scripting siap digunakan dari kotak.

Kerang dan skrip berikut bahasa tersedia sebagai terbuka sumber:

awk bash ksh perl PHP (apache_mod_php) python ruby ​​sed (text_cmds) tcl tcsh XSLT (libxslt) zsh

Beberapa dari hal-hal ini telah diinstal dengan XCode. Anda dapat menemukan informasi lebih lanjut pada tautan di atas.

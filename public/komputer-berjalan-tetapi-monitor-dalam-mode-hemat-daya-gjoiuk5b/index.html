Komputer berjalan tetapi monitor dalam mode hemat daya?

Hal-hal ini jarang terjadi, tetapi ketika itu terjadi saya cukup me-restart PC saya maka itu berfungsi dengan baik. Tetapi hari ini setelah beberapa upaya restart tampaknya macet.

Saya mencoba mengeluarkan kartu video kemudian beralih ke video onboard, hal yang sama: PC berjalan, (ada daya dan kipas bekerja) tetapi monitor macet pada mode hemat daya (layar kosong).

Apakah ini masalah monitor atau PC? Ada ide apa yang mungkin menjadi masalah?

Saya Windows 7 btw.

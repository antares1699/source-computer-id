Cara menentukan apakah hard disk benar-benar gagal atau tidak

Linux mengatakan bahwa hard disk saya rusak. Kemudian saya mencoba menggunakan chkdsk / r di windows. Kemudian bersihkan menggunakan boot dan nuke (dban). Laptop saya berjalan lambat saat itu. Setelah 2 minggu tidak menggunakan laptop saya. Saya memutuskan untuk menggunakan pemeriksa startup bawaan di bios. Ia memeriksa memori dan hard disk. Setelah sekitar 1 jam. Pemeriksa startup mengatakan bahwa pemeriksaan startup lulus. Kemudian saya mencoba boot laptop yang saya instal linux mint. Lalu tiba-tiba menjadi cepat lagi. Perhatikan bahwa laptop saya berjalan sangat lambat saat saya menggunakannya 2 minggu yang lalu. Mungkinkah pemeriksa startup bawaan memperbaiki sesuatu? Haruskah saya percaya apa yang dikatakan Linux? Apakah saya benar-benar perlu mengganti hard disk? Tangkapan layar di bawah ini diambil setelah pemeriksaan startup.

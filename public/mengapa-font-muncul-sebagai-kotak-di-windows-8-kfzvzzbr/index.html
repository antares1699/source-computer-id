Mengapa font muncul sebagai kotak di Windows 8?

Saya menemukan bahwa font "MS Outlook" muncul sebagai kotak di komputer saya. Bagaimana cara menampilkan karakter MS Outlook dengan benar? Ada font lain yang muncul sebagai kotak di Windows 8 juga.

"Menampilkan kotak" adalah tanda bahwa font yang diberikan tidak mendukung semua mesin terbang yang ingin Anda render.

Menebak lokal Anda adalah sesuatu seperti 'Cina' atau 'Jepang' atau apapun. Dan teks contoh dari aplikasi pratinjau font memilih beberapa teks dalam bahasa Anda sendiri. Maka mungkin kasusnya memilih teks yang membutuhkan mesin terbang yang tidak tersedia dalam font yang ingin Anda gunakan.

Mungkin lebih baik menggunakan alat charmap (charmap.exe). Di sana Anda bisa melihat semua mesin terbang dari font tertentu, bukan hanya panagram.

Ini adalah font khusus yang digunakan oleh Microsoft Outlook untuk membuat beberapa elemen antarmuka.

Menjadi font tujuan khusus, itu tidak berisi karakter biasa, dan Anda melihat kotak, bukan teks percobaan.

Jumlah font tersebut tergantung pada perangkat lunak yang diinstal.

Windows memiliki font khusus Marlett yang digunakan untuk merender elemen antarmuka Windows (dalam tema klasik), seperti tombol window meminimalkan, memaksimalkan, dan menutup.

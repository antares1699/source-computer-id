Cara menghapus titik pertama dari daftar jalur file

Saya perlu menghitung md5sum dari satu string per baris di dump ls saya:

directory_listing.txt:

./r/g4/f1.JPG
./r/g4/f2.JPG
./r/g4/f3.JPG
./r/g4/f4.JPG

Namun md5sum harus dihitung tanpa 'titik pertama'. Sejauh ini saya telah menulis skrip sederhana:

while read line
do
echo $line | exec 'md5sum'
done

./g.sh < directory_listing.txt

Bagaimana cara menghapus titik pertama dari setiap baris?

Bagaimana cara memblokir spam messenger di Windows 8?

Saya menggunakan Windows 8 dengan aplikasi Metro Messaging. Saya tidak tahu cara memblokir spam messenger, sepertinya tidak ada opsi baik di dalam aplikasi atau di dalam pengaturan akun live online. Saya mendapatkan setidaknya 4 hari dari orang acak yang tidak ada di daftar kontak saya.

Setiap bantuan dihargai! Saya hanya dapat menemukan panduan online untuk versi perpesanan aplikasi desktop.

EDIT: Hanya untuk memperjelas, saya menggunakan Windows 8 di Laptop - saya pikir versi Windows 8 Home atau Professional.

Saya tahu itu jawaban yang tidak disukai siapa pun, tetapi salah satu cara termudah dan termudah untuk menangani segala jenis spam adalah mendapatkan alamat email, nama pengguna, nomor telepon, apa pun.

Jika Anda mendapatkan pesan sebanyak itu setiap hari, beberapa orang aneh memiliki info Anda, sebaiknya tinggalkan akun tersebut dan mulai lagi dengan yang tidak mereka ketahui.

Satu-satunya alternatif lain adalah untuk melihat apakah ini adalah cara Anda bisa mendapatkan filter untuk memungkinkan HANYA orang yang Anda setujui untuk menghubungi Anda, saya tidak tahu apakah perpesanan windows 8 memiliki fitur itu atau tidak.

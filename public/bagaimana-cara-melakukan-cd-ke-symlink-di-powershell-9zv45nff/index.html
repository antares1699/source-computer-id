Bagaimana cara melakukan cd ke symlink di PowerShell?

Saya telah membuat tautan simbolis seperti ini:

PS C:\dev\bin> cmd /c mklink "wd" c:\dev\code
symbolic link created for wd <<===>> c:\dev\code

Oke, sederhana. Jadi saya mencoba untuk cd itu:

PS C:\dev\bin> cd wd
Set-Location : Cannot find path 'wd' because it does not exist.

PS C:\dev\bin> cd .\wd
Set-Location : Cannot find path '.\wd' because it does not exist.

PS C:\dev\bin> cd \wd
Set-Location : Cannot find path 'C:\wd' because it does not exist.

Jauh lebih mudah untuk melakukan ini di Linux. Bagaimana cara melakukan cd ke symlink di PowerShell?

(Sepertinya pertanyaan sederhana, tetapi 3 halaman pertama hasil Google tidak memiliki jawaban.)

Perintah yang benar untuk menautkan direktori adalah

cmd /c mklink /D link target

Lihat 'mklink /?' untuk opsi lain.

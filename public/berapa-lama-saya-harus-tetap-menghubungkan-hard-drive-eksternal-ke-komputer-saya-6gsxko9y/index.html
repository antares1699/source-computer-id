Berapa lama saya harus tetap menghubungkan hard drive eksternal ke komputer saya

Saya baru saja mendapatkan hard drive eksternal Toshiba 500GB melalui pos hari ini dan saya sangat bersemangat. Jadi tentu saja saya tidak ingin merusaknya. Berapa lama saya harus tetap menghubungkannya?

Untuk flash drive saya akan tetap menghubungkan saya sepanjang hari, tetapi jelas saya tidak dapat melakukan ini dengan HD eksternal karena disk di dalamnya benar-benar berputar terus-menerus. Saya ingin milik saya bertahan lama jadi apa yang kalian rekomendasikan?

EDIT: Saya idiot, saya lupa menyebutkan ini adalah hard drive eksternal PORTABLE, bukan yang desktop yang dimaksudkan untuk disimpan setiap saat.

Mengapa Anda tidak bisa terus menghubungkannya sepanjang waktu? Sebagian besar hard drive modern akan memasuki mode 'tidur' jika tidak sering digunakan untuk mengurangi konsumsi daya dan meningkatkan masa pakai drive.

Hard drive modern dirancang untuk tidur setelah periode tidak aktif tertentu. Ini untuk menghemat energi dan juga dianggap oleh beberapa orang untuk meningkatkan masa manfaat penggerak dengan mengurangi keausan. Dalam mode siaga, hard drive dalam keadaan konsumsi energi rendah dan piringannya berhenti tidur

Bergantung pada pengaturan sistem operasi, hard drive secara otomatis akan masuk mode tidur setelah beberapa saat tidak aktif, dan dengan demikian meningkatkan masa pakai drive.

Bagaimanapun, hard drive di dalam komputer Anda tetap terpasang sepanjang waktu, jadi mengapa yang eksternal harus berbeda?

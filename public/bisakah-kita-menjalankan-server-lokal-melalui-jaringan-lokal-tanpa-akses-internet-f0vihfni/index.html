Alasan (kemungkinan) Anda tidak dapat mengakses server melalui laptop Anda adalah karena DNS tidak menyelesaikan. Itu mencoba untuk terhubung ke server melalui IP eksternal (yang terhubung ke internet) dan tentu saja, itu tidak akan berhasil.

Jika server Anda terhubung ke Internet, kemungkinan besar server tersebut ditetapkan dan diakses melalui nama Domain, yaitu: server.mycompany.com.

Server ini harus (biasanya) dapat diakses dengan 3 metode: nama DNS, ip eksternal, atau ip internal . Metode terakhir inilah yang kita butuhkan untuk mengakses secara lokal.

Jika ini adalah aplikasi web yang perlu Anda akses, cukup menggunakan ip internal server sudah cukup untuk mengaksesnya:

Alih-alih server.mycompany.com, seseorang akan menggunakan ip, yaitu: http://192.168.0.23

Untuk menghindari laptop Anda terputus dari server saat internet mati, Anda dapat menambahkan entri di file host Anda yang mengarah ke ip lokal, IE:

server.company.com    182.168.0.23

Tentu saja, ini berarti ketika berada di luar jaringan lokal Anda, Anda perlu menghapus entri ini untuk memungkinkan DNS menyelesaikan melalui ip / antarmuka eksternal.

Semoga ini bisa membantu, semoga berhasil!

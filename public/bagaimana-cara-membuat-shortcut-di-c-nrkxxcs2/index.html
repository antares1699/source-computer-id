Bagaimana cara membuat shortcut di C: \?

Apakah ada cara untuk membuat pintasan yang aktif di C:\ (kebalikan dari desktop saya), dan menunjuk ke folder juga di C:\?

(Saya tidak yakin apakah alat shortcut.exe dapat melakukan ini sama sekali, tetapi bagaimanapun, saya tidak dapat menemukan versi apa pun untuk Windows 7.)

  • Sangat buruk untuk menggunakan file dan folder langsung di folder root C: \ di Windows 7 (benar-benar: Vista dan yang lebih baru). Pengguna standar tidak memiliki hak menulis ke folder ini sejak Windows 2000, tetapi dimulai dengan Vista, sistem operasi mulai menjadi sangat ketat dalam menegakkan ini, bahkan jika Anda masuk sebagai pengguna dengan hak admin.

  • Saya mengunduh kursor mouse saya sendiri, dan menyimpannya di C:\data_me\app_settings\windows_cursors`, which is a folder I automatically back up. Saya perlu (ingin) pintasan dari folder itu, kembali ke milik saya, jadi saya bisa bolak-balik dengan cepat di antara mereka. Itu kasus penggunaan saya.

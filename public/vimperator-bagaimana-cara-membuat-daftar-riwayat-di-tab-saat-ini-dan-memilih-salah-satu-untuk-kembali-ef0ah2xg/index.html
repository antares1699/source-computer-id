Vimperator: Bagaimana cara membuat daftar riwayat di tab saat ini dan memilih salah satu untuk kembali?

Vimperator memiliki kunci "H" untuk kembali ke halaman, tetapi bagaimana cara melihat daftar riwayat untuk tab saat ini dan kembali ke tab yang dipilih?

Perintah jumps

Anda dapat menggunakan perintah :jumps (atau hanya :ju). Ini menampilkan "daftar lompat" bernomor dengan riwayat tab. Anda kemudian dapat menavigasi ke url menggunakan petunjuk atau mouse.

Jika kunci petunjuk f dan F tidak bekerja

Ketika saya baru saja menguji ini pada versi Vimperator di laptop saya, petunjuk normal menggunakan f atau F tidak bekerja karena suatu alasan. Saya yakin mereka harus melakukannya (dan mereka melakukannya dengan Pentadactyl), jadi mudah-mudahan mereka akan bekerja untuk Anda. Jika tidak, Anda selalu dapat menggunakan ; keluarga binding petunjuk tambahan sebagai gantinya:

  • ; Hai buka petunjuk di tab saat ini
  • ; t buka petunjuk di tab baru
  • ; b buka petunjuk di tab latar belakang baru
  • ; w buka petunjuk di jendela baru

Dan seterusnya...

Anda juga dapat menggunakan angka sebagai argumen untuk Ctrl-o dan Ctrl-i . Jadi jika Anda ingin melompat ke # 3 pada daftar lompat, Anda cukup mengetik 3 Ctrl-o .

Membuat pengikatan kunci

Jika Anda menginginkan pengikatan kunci untuk daftar lompat, Anda dapat menentukannya dengan menambahkan baris berikut ke file $HOME/.vimperatorrc Anda:

map <C-j> :jumps<CR>

Ini mendefinisikan Ctrl-j sebagai jalan pintas untuk daftar lompat. Ini hanya bekerja dalam mode normal / visual, mis. tidak saat Anda berada di area teks, membuka menu, dll.

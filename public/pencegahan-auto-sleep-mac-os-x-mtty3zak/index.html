Pencegahan Auto Sleep Mac OS X.

Saya memiliki masalah dengan Mac OS X (Leopard) di MacBook Pro saya: ketika saya pergi selama beberapa menit laptop saya secara otomatis tidur dan semua aplikasi saya yang sedang berjalan berhenti bekerja (seperti unduhan saya). Apa yang harus saya atur agar berfungsi seperti Windows dan menampilkan layar kunci tetapi aplikasi terus melakukan tugasnya, seperti mengunduh.

P.S: Saya mengunduh widget layar kunci dan ketika saya menggunakannya aplikasi saya tidak menutup tetapi saya kehilangan koneksi internet sampai saya membuka kunci laptop saya.

[Jawaban ini berdasarkan komentar OP atas jawaban deddebme.]

  1. Ikuti petunjuk deddebme.
  2. Buka System Preferences-> Desktop & Screen Saver-> Screen Saver, lalu pilih screen saver yang Anda inginkan dan waktu yang Anda inginkan sebelum muncul. (Secara opsional, atur juga sudut panas untuk screen saver sehingga Anda dapat mengunci sesuai permintaan.)
  3. Pergi ke System Preferences-> Security-> General, dan pastikan Wajibkan kata sandi untuk membangunkan komputer ini dari mode tidur atau screen saver diperiksa

Sekarang Anda memiliki segalanya - tanpa tidur, screen saver otomatis dengan kunci, dan kata sandi yang diperlukan untuk kembali dari itu. Saat berada dalam mode screen saver, itu akan terus memiliki koneksi internet.

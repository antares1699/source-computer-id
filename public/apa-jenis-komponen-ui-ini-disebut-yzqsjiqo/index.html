Apa yang disebut komponen UI semacam ini? Saya pikir itu lebih masuk akal dalam kasus tertentu daripada satu set tombol radio dan saya ingin membuat komponen serupa untuk aplikasi web yang saya kembangkan. Saya hanya ingin tahu apa namanya jika seseorang di luar sana telah membuat cara JavaScript yang bagus untuk mengimplementasikan komponen semacam ini. Namun, saat ini saya tidak tahu harus mencari apa.

Mirip dengan a tombol radio , ada juga komponen GUI yang disebut a tombol sakelar . Kedua jenis tombol dapat ditambahkan ke a grup tombol jadi hanya satu tombol dalam grup yang dapat dipilih dalam satu waktu. Ini berbeda dengan a tab , yang dilampirkan ke a panel .

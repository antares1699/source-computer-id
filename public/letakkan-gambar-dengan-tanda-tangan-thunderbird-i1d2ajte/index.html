Cara menggunakan gambar sebagai signature Thunderbird

  1. Letakkan file gambar di C: \ dan ganti namanya menjadi gambar (untuk kenyamanan)
  2. Buka Notepad

  3. Ketik kode HTML ini

    <b>Jhon Cena</b>
<br><img moz-do-not-send="false" src="file:///C:/image.gif" alt="Signature">
  4. Simpan ke C: \ as Test.html
  5. Anda harus memiliki 2 file di C: \ drive: image.gif dan Test.html
  6. Buka Mozilla Thunderbird dan login ke akun Anda
  7. Pergi ke Alat Menu dan pilih Pengaturan akun
  8. Sorot akun Anda
  9. Centang opsi Lampirkan tanda tangan dari file sebagai gantinya (teks, html atau gambar)
  10. Klik Memilih , navigasikan ke C :, buka Test.html dan klik OK
  11. Untuk Menguji apakah ini berfungsi, cukup tulis email dengan mengklik Menulis Tombol

diuji pada Windows XP Pro SP3

  • Perhatikan di sini saya telah menggunakan file image.gif, jika Anda menggunakan .jpeg atau .bmp atau file gambar lainnya, silakan ubah ekstensi dengan benar di file html. Ngomong-ngomong, baris pertama dalam kode html adalah memasukkan nama Anda dalam teks dan baris kedua untuk gambar. Jika Anda tidak ingin nama Anda ada dalam teks, lewati baris pertama.

Bagaimana cara menghapus snapshot apt-btrfs-saat error "Direktori tidak kosong?"

Ada satu snapshot yang dibuat oleh apt-btrfs-snapshot yang tidak dapat dihapus

  Delete subvolume '/tmp/foo/@apt-snapshot-2013-04-XX_1X:3X:XX'
  ERROR: cannot delete '/tmp/foo/@apt-snapshot-2013-04-XX_1X:3X:XX' - Directory not empty

Bagaimana cara menghapusnya?

4

7

Menjawab pertanyaan saya sendiri:

Masalahnya adalah ada subvolume di dalam subvolume. Anda perlu menghapus semua subvolt dalam sebelum menghapus vol yang berisi.

btrfs subvol delete / tmp / foo / @ apt-snapshot-2013-04-XX_1X: 3X: XX / @ btrfs subvol delete / tmp / foo / @ apt-snapshot-2013-04-XX_1X: 3X: XX

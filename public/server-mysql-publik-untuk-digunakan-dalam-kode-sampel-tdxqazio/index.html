Server MySQL publik untuk digunakan dalam kode sampel

Mirip dengan bagaimana layanan web dan email domain example.com IETF dapat digunakan dengan aman dalam sampel kode, apakah ada server MySQL publik yang dapat saya gunakan dengan aman (*) dalam beberapa kode sampel?

("Aman" karena tidak mempermasalahkan jumlah kecil lalu lintas yang akan dihasilkan pengguna kode contoh saya, dan kemungkinan akan tetap dapat diakses selama beberapa tahun mendatang.)

example.com tidak dimaksudkan untuk berfungsi dengan cara apa pun, itu hanya dimaksudkan agar aman untuk digunakan dalam dokumentasi. Perangkat lunak apa pun yang menggunakannya kemungkinan besar akan menghasilkan pesan kesalahan kepada pengguna akhir.

Anda dapat menggunakan mysql.example.com di file dokumentasi Kode sampel . Pembaca itu dokumentasi Kode sampel TIDAK boleh berharap akan ada layanan MySQL yang berfungsi di alamat itu. Pembaca berharap mereka harus mengganti mysql.example.com dengan nama server MySQL mereka sendiri.

Nah, Anda bisa menggunakan situsnya: http://www.db4free.net/ yang memungkinkan Anda masuk untuk mendapatkan database gratis.

Ini bukan untuk produksi tetapi bagus untuk pengujian dan saya membuat database bertahun-tahun yang lalu dan masih berfungsi.

Catatan : Saya tidak tahu apakah itu akan menangani koneksi dari pengguna yang sama dari URL yang berbeda tanpa memblokir akun atau tidak.

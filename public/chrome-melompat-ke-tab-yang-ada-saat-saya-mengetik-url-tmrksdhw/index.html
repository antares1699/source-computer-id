Chrome melompat ke tab yang ada saat saya mengetik URL

Saat saya mengetik url (katakanlah, superuser.com), Chrome melompat untuk menggunakan kembali tab yang ada yang sudah terbuka di tab itu. Saya pernah menemukan ini berguna (memang, saya mengajukan pertanyaan memintanya!) Sekarang saya ingin menyingkirkannya. Seringkali saya benar-benar ingin dua tab terbuka ke situs yang sama - jadi saat ini satu-satunya solusi adalah menggunakan "tab duplikat".

Sekarang, saya tidak menggunakan ekstensi apa pun yang bisa menjelaskannya (tentu saja bukan Hanya Satu). Apakah ada pilihan lain di suatu tempat yang dapat menyebabkan perilaku ini?

Saya rasa Anda telah mengaktifkan fitur eksperimental "Fokus tab yang ada saat terbuka" di Chrome:

Jika demikian, buka chrome: // flags /, temukan fitur "Fokuskan tab yang ada saat buka" dan nonaktifkan. Luncurkan ulang browser agar efek tersebut terjadi.

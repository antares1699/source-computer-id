Adakah cara untuk langsung membuka tab paling kiri dan paling kanan di Firefox pada Vista / Windows 7?

Di Windows, Ctrl + 1 memilih tab paling kiri.

Ctrl + nomor 2 - 8 memilih tab 2 - 8 .

Ctrl + 9 memilih tab paling kanan.

Di Linux, menahan Alt dan menekan 1 akan mengalihkan ke tab paling kiri.

Di Windows, gunakan Ctrl sebagai ganti Alt dan di Mac gunakan Cmd.

Sini adalah daftar pintasan keyboard untuk Firefox.

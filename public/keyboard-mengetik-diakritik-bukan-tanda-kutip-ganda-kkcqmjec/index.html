Keyboard mengetik diakritik (¨), bukan tanda kutip ganda

Saya mengalami masalah dengan tanda kutip ganda pada keyboard saya saat bekerja di Linux.

  • Kutipan ganda jenis keyboard saya adalah: ¨¨
  • dan yang benar-benar saya harapkan adalah: ""

Anda melihat perbedaannya di sana? Yang pertama yang jenis keyboard saya tidak diterima ketika saya bekerja dengan R, jadi saya harus menyalin-tempel tanda kutip ganda yang benar dari internet setiap saat.

Ini hanya terjadi ketika saya bekerja dengan Linux, dan bukan Windows. Adakah yang tahu bagaimana mengubahnya atau melakukannya dengan benar?

Kemungkinan penyebabnya adalah Anda memiliki tata letak keyboard yang disetel ke Inggris (Internasional AS) , seperti yang saya miliki. Pengaturan ini memungkinkan Anda untuk memasuki diakritik internasional, tetapi dilengkapi dengan beberapa gotcha.

Jika Anda ingin menggunakan " daripada ¨ Alt + Bergeser + " (yang di sebelah tombol Enter).

Jika Anda ingin menggunakan ', bukan ´ Alt + " (yang di sebelah tombol Enter).

Bagikan

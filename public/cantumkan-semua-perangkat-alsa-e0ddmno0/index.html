Cantumkan semua perangkat ALSA

Saya menggunakan PulseAudio di sistem saya, dan saya mengikuti petunjuk untuk menambahkan file perangkat monitor sesuai wiki PulseAudio. Perangkat monitor itu sendiri berfungsi dan mendaftar / berfungsi dengan baik di aplikasi seperti Audacity, tetapi saya tidak bisa membuat perangkat menampilkan dirinya sendiri melalui alat baris perintah mana pun. aplay -L mengembalikan daftar yang kira-kira sama dengan Audacity, tetapi perangkat monitornya tidak ada.

Tujuan akhir saya adalah mengurai output dalam aplikasi Java dan akhirnya meneruskannya ke FFmpeg. Saya dapat memberi tahu perangkat itu ada melalui Java audio API saat 2 "Port sumber pengambilan" generik muncul, tetapi tidak memberikan nama untuk perangkat tersebut, dan tujuannya adalah agar dapat dipilih pengguna di sikap yang agak ramah.

Adakah cara agar perangkat monitor muncul di aplikasi baris perintah atau melalui API / pustaka Java?

Terima kasih sebelumnya!

Gunakan perintah pacmd list-sources atau pactl list untuk menampilkan perangkat PulseAudio. Untuk menampilkan hanya nama, Anda dapat melakukan grep pada outputnya. Ini adalah contoh dari FAQ PulseAudio

$ LANG=C pactl list | grep -A2 'Source #' | grep 'Name: ' | cut -d" " -f2

Dengan grep tambahan pada ".monitor" Anda hanya bisa mendapatkan perangkat monitor.

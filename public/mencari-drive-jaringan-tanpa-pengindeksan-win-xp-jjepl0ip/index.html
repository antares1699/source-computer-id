Ketika saya perlu mencari lokasi jaringan, saya hanya keluar ke jendela perintah dan menggunakan DIR /S. Ini tidak menggunakan indeks, hanya mencari nama file (bukan isinya) dan saya merasa cukup cepat. Sakelar /S akan mencari subfolder untuk Anda. Anda dapat menggunakan beberapa wildcard jika diperlukan untuk memperluas atau mempersempit pencarian Anda (* untuk sejumlah karakter, ? untuk satu karakter) dan tidak mengalami masalah dengan mencoba memasukkan atau mengecualikan hal-hal seperti tanda kurung, seperti yang dilakukan pencarian Windows. Saya juga menggunakan sakelar /B untuk mendapatkan daftar yang lebih ringkas (tidak ada ringkasan antar subfolder).

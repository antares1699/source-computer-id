Alat hidup untuk pemeriksaan hard disk, perbaikan dan format ulang

Di masa lalu, saya memiliki floppy dengan CHCKDSK dan alat lain untuk memeriksa kegagalan hard disk, memperbaikinya dan, jika perlu, memformatnya ulang. Saya baru-baru ini mencoba menginstal ulang XP di komputer lama, tetapi mendapat kesalahan terkait dengan hard disk. CD instalasi XP tidak dapat menanganinya, jadi saya membutuhkan sesuatu yang lebih bertenaga.

Siapa pun dapat memberi nasihat tentang seperangkat alat langsung pada CD-ROM yang dapat di-boot untuk manajemen disk tingkat lanjut (memeriksa, memperbaiki, memformat ulang di NTFS, mempartisi, ...) Terutama NTFS mengganggu saya.

Saya sudah memeriksanya pertanyaan ini , tapi jawabannya bukanlah yang saya cari.

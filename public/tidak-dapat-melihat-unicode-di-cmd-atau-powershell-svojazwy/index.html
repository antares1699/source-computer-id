cmd menampilkan karakter Unicode sebagai?

Anda perlu menyetel halaman kode yang mendukung karakter Unicode yang Anda gunakan:

  • Di shell cmd, gunakan chcp
  • Pada powershell, gunakan $OutputEncoding = [System.Text.Encoding]::Unicode

Bacaan lebih lanjut

  • Indeks A-Z dari baris perintah CMD Windows | SS64.com
  • Perintah CMD Windows (dikategorikan) - Windows CMD - SS64.com
  • Chcp - Halaman Ubah Kode - Windows CMD - SS64.com
  • Memahami pengkodean default dan Ubah yang sama di PowerShell - mohitgoyal.co

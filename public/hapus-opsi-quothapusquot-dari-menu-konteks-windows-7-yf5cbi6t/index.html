Saya pikir menu yang dimaksud setidaknya sebagian di file shell32.dll.mui file, terletak @

windows \ system32 \ en-us (tergantung pada wilayah / bahasa OS Anda)

Menggunakan peretas sumber daya , kamu butuh ubah menu 210 beberapa menu! (Ctrl + F ind "& Hapus", F3 ke Temukan Berikutnya :)

MENUITEM "Cu&t", 24, MFT_STRING, MFS_ENABLED

MENUITEM "", 65535, MFT_SEPARATOR, MFS_ENABLED
MENUITEM "Create &shortcut", 16, MFT_STRING, MFS_ENABLED
MENUITEM "Rena&me", 18, MFT_STRING, MFS_ENABLED
MENUITEM "", 65535, MFT_SEPARATOR, MFS_ENABLED

pertanyaannya adalah, JIKA Saya menghapus menu konteks hapus entri # 17 entri, apakah fungsi hapus (melalui tombol why-are-you-so-close-to-my-home-key-DELETE) juga dinonaktifkan? Ayo lihat.

NB: pengguna 64-bit juga perlu memodifikasi file @ windows \ syswow64 ...

Anda mungkin perlu mengambil kepemilikan file terlebih dahulu (saya sarankan googling 'add mengambil kepemilikan ke menu konteks ') salin file (shell32.dll.mui), kemudian ubah dan simpan perubahan ke salinan (setelah kompilasi di sumber hacker), LALU ganti file sys (cadangan dengan mengganti nama asli ke shell32.dl.mui.whateveryouwant ) dengan salinannya.

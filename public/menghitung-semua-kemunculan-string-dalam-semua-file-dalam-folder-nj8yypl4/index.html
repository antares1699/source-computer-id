Menghitung semua kemunculan string dalam semua file dalam folder

Bagaimana cara menghitung semua kemunculan string tertentu di semua file dalam folder?

Sebagai contoh,

1.txt: 'hahaha hehe'
2. txt: 'ha hee'

Saya ingin menghitung semua kemunculan 'ha' dalam file ini.

Saya mencoba:

grep "ha" * | wc-l

Tapi itu hanya memberi saya kejadian individu dalam output file

Anda sudah dekat. Untuk mendapatkan jumlah total semua kemunculan "ha" dalam semua file .txt dalam sebuah folder:

grep -o "ha" *.txt | wc -l

Dari man grep:

-o, --only-matching
       Print only the matched (non-empty) parts of a matching line, with
       each such part on a separate output line.

Ini berhasil karena setiap pertandingan dicetak pada baris terpisah, sehingga memungkinkan wc -l untuk menghitung semuanya.

Secara default, bagaimanapun, grep hanya menemukan kejadian pertama pada sebuah baris dan mengeluarkan keseluruhan baris. Demikian juga, opsi -c hanya menemukan kemunculan pertama di semua baris, lalu menampilkan berapa banyak baris yang memiliki 1 (atau lebih) kecocokan.

EDIT:

Berikut adalah cara untuk mencetak jumlah total kejadian dalam setiap file individu (dengan nama file):

find *.txt -printf 'echo "$(grep -o "ha" %p | wc -l) %p";' | sh

#Example output
3 file1.txt
1 file2.txt

Penjelasan:

find *.txt - menemukan file .txt

-printf - mencetak semuanya antara tanda kutip tunggal (diformat) ke keluaran standar, menggantikan kejadian %p dengan keluaran find (nama file)

$(grep -o "ha" %p | wc -l) - bekerja seperti di atas

| sh - output dari -printf (yang merupakan perintah) disalurkan ke shell dan dieksekusi

Perhatikan bahwa printf dipanggil sekali per nama file.

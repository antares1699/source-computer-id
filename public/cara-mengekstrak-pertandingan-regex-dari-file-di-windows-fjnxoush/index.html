Saya menggunakan banyak ekspresi reguler. Saya menggunakan EditPadpro untuk semua tujuan pengeditan saya. Sebagian besar waktu saya harus mengekstrak semua pencocokan ekspresi reguler dalam file ke file lain. Apakah ada software yang bisa melakukan ini?

RegexBuddy , produk saudara untuk EditPad Pro melakukan ini dengan sangat baik melalui tab Grep itu. Akan berulang melalui folder dan mengekstrak kecocokan ke file. Ini juga bagus untuk membuat dan menguji ekspresi reguler.

Gunakan pengaturan seperti di bawah ini:

  • Anda dapat menambahkan regex email yang telah dibuat sebelumnya dari tab 'Perpustakaan'
  • Pastikan Anda memilih tombol 'Cocokkan' di toolbar bagian atas
  • Masukkan detail file pada tab 'Grep' dan pilih 'Simpan hasil ke dalam satu file'

Anda dapat menguji regex dengan menempelkan bagian file Anda ke tab 'Test'.

  • Tekan 'Execute' dari drop-down 'Grep' untuk menjalankan.

Install Cygwin , ini akan memberi Anda lingkungan seperti unix yang sangat fungsional di windows. Kemudian pelajari grep dasar, egrep, awk, sed atau perl. Waktu diinvestasikan dengan sangat baik. Hanya beberapa contoh:

# extract email addresses from text.txt to emails.txt
egrep -o '[[:alnum:].]+@[[:alnum:].]+' < text.txt > emails.txt
# extract third whitespace separated column from lines starting with "DEBUG"
awk '/^DEBUG/ { print $3 }' < text.txt > debug.txt
# replace multiple whitespace characters with one space
perl -pe 's/[\t ]+/ /g' < text.txt > clean.txt

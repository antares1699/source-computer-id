Apakah ada ekstensi Chrome untuk menukar CSS? [Tutup]

Saya mencoba untuk sepenuhnya mengganti CSS untuk domain dengan CSS yang berbeda (yaitu menukar CSS di gaming.SE untuk Stack Overflow). Idealnya saya ingin ekstensi yang memungkinkan saya melakukan ini. Saya telah menggunakan Web yang Dipersonalisasi , yang memungkinkan pemuatan CSS untuk domain tertentu, tetapi masih memuat CSS asli (Anda dapat menentukan CSS untuk diabaikan, tetapi Anda harus melakukan ini aturan demi aturan !

Adakah yang tahu cara melakukan ini?

Menggunakan Ekstensi WebDeveloper , Anda bisa melakukan itu!

Setelah diinstal, buka ekstensi, tekan tab CSS dan pilih "edit css". Anda kemudian dapat mengganti setiap CSS yang dimuat untuk halaman web tersebut.

Semoga ini membantu!

-EDIT- Untuk perubahan yang terus-menerus, Anda dapat mencoba Stylebot . Mereka mengatakan "Stylebot adalah ekstensi Google Chrome yang memungkinkan Anda dengan cepat membuat dan menyimpan CSS khusus yang persisten untuk situs".

Dengan Gaya hidup Anda memiliki pemuatan ulang css waktu nyata saat penyimpanan (ekstensi SublimeText), Anda dapat mengaitkan gaya css jarak jauh dengan gaya lokal dan bahkan menambahkan file css tambahan untuk menimpa gaya jarak jauh.

Edit: Saya memutuskan CSSInject extension - LiveStyle sepertinya mengatur ulang perubahan css saat refresh, jadi saya menyimpannya hanya untuk pengeditan langsung. Jika Anda membutuhkan CSSInject untuk menghapus gaya, tambahkan kode ini pada baris 49 di loadcss.js

for(var elem in document.getElementsByTagName("link") ) { 
   var node = document.getElementsByTagName("link")[elem];
   if(node.getAttribute("href") == "/path/to/css-style/to-remove.css") {       
     node && node.parentNode.removeChild(node);
  } 
}

Memecahkan sistem persamaan linier di Excel tanpa menggunakan VBA

Misalkan Anda memiliki sistem persamaan linier berikut dan ingin menyelesaikannya x , y , dan z :

  • 2x + y - z = -1
  • -4x + 11y = 18
  • x - y + 2z = 9

Apakah ada cara mudah untuk melakukan ini dengan Excel menggunakan fungsi lembar kerja?

Ini dapat dilakukan dengan menggunakan sedikit aljabar linier dan fungsi matriks Excel.

Pertama, buat matriks tambahan yang mewakili persamaan Anda di lembar kerja.

Karena Anda ingin menyelesaikan tiga variabel, pilih 3 sel di kolom tempat Anda ingin hasilnya dikembalikan. Tanpa mengubah pilihan, ketikkan rumus berikut:

=MMULT(MINVERSE(A2:C4),D2:D4)

tekan Ctrl + Bergeser + Memasukkan untuk memasukkan rumus dalam larik yang dipilih. Dan itu akan mengembalikan solusi ke sistem persamaan Anda.

Ini akan bekerja untuk sistem persamaan yang lebih besar dengan lebih banyak variabel juga. Rumus ini bekerja berdasarkan pengamatan bahwa ruas kiri sistem persamaan linier dapat direpresentasikan sebagai hasil perkalian dari sebuah sistem persamaan linier. n oleh n matriks, SEBUAH , dan n dengan 1 matriks variabel, x , dan sisi kanan dapat direpresentasikan sebagai file n dengan 1 matriks, B . Untuk menyelesaikan persamaan matriks ini, kedua sisi harus dikalikan dengan kebalikan dari SEBUAH .

Kapak = B
terbalik( SEBUAH ) Kapak = terbalik ( SEBUAH ) B
x = terbalik ( SEBUAH ) B

Jadi, solusinya hanyalah hasil kali matriks dari kebalikan dari matriks kuadrat dan matriks konstanta di sisi kanan persamaan.

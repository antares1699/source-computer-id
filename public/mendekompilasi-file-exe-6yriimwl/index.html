Mendekompilasi file EXE

Saya membantu teman dalam menangani masalah perangkat lunak. Dia memiliki mesin Windows yang saat startup menjalankan file .exe yang telah dapat kami dekonstruksi menjadi file .bat yang sesuai. Eksekusi file itu diharapkan.

Dalam file .bat, kami melihat bahwa ia memanggil dua file .exe lainnya. Mereka disebut myScript5.exe dan f11.exe. Kami juga tidak dapat mendekompilasi file .exe karena kami tidak tahu bahasa apa aslinya, dan tidak ada file .bat yang muncul di folder temp kami saat kami menjalankannya.

Kami tahu apa yang dilakukan f11.exe, setidaknya di permukaan. Kami cukup yakin bahwa ini mensimulasikan penekanan tombol F11, yang pada dasarnya meledakkan jendela saat ini ke mode layar penuh.

Kami tidak dapat mengetahui apa yang myScript5.exe lakukan, kecuali bahwa itu mengubah penunjuk tetikusnya menjadi ikon "memuat"; mengklik di mana saja akan mengembalikan mouse ke ikon biasa.

Kedua file .exe memiliki ikon kotak hijau dengan Comic Sans "H" putih. Adakah yang tahu bagaimana kami dapat mencoba mendekompilasi myScript5.exe, atau jika ada yang akrab dengan ikon yang saya jelaskan? Saya pribadi berpikir bahwa kedua file .exe diunduh dari situs web yang menerbitkan aplikasi utilitas (seperti meledakkan jendela ke mode layar penuh).

Bantuan apa pun sangat dihargai. Terima kasih!

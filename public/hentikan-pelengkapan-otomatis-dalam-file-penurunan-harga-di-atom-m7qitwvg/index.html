Hentikan pelengkapan otomatis dalam file penurunan harga di Atom

Saya mengalami masalah saat menulis penurunan harga saya di Atom. Karena ini melengkapi otomatis dari file sumber dalam proyek. Perhatikan bahwa saya punya https://atom.io/packages/language-markdown terpasang.

Adakah cara untuk menonaktifkan pelengkapan otomatis untuk jenis file? Atau setidaknya satu file?

10

Anda dapat mencoba memasukkan semua file .md ke daftar hitam ini dari paket pelengkapan otomatis.

Buka Pengaturan > Paket dan temukan paket bernama pelengkapan otomatis-plus . Pergi ke itu Pengaturan dan tambahkan *.md ke file File Daftar Hitam bidang.

Mudah-mudahan, Anda tidak akan mengalami pelengkapan otomatis di file penurunan harga mulai sekarang.

Apa rasa terbaik Linux untuk server file rumah (media)? [Tutup]

Saya ingin menyiapkan server Linux untuk menyajikan file media (video dan musik) ke komputer saya yang lain di rumah.

Beberapa persyaratan:

  1. Harus dapat terhubung ke jaringan Komputer Windows di rumah
  2. Harus bisa dimatikan pada jam tertentu waktu, atau saat saya menekan bagian depan tombol komputer
  3. Semakin banyak fitur daya, semakin baik: Bangun, tutup sebagian besar
  4. Tidak ingin memasukkan kredensial masing-masing kali saya mencoba mengakses file

Dan mungkin hal lain yang tidak saya pikirkan.

Jika Anda ingin menggunakan komputer Anda sebagai server NAS, mengapa tidak mencobanya FreeNAS ?

Wikipedia:

FreeNAS adalah jaringan gratis yang terpasang server penyimpanan, mendukung: CIFS (Samba), FTP, NFS, rsync, AFP protokol, iSCSI, S.M.A.R.T., lokal otentikasi pengguna, dan perangkat lunak RAID (0,1,5), dengan berbasis web antarmuka konfigurasi. FreeNAS membutuhkan kurang dari 64 MB setelah diinstal CompactFlash, hard drive atau USB flash mendorong. FreeNAS saat ini didistribusikan sebagai citra ISO dan dalam format bentuk sumber. Itu mungkin untuk dijalankan FreeNAS dari Live CD, dengan ekstensi file konfigurasi yang disimpan di file Floppy disk berformat MS-DOS. Ada juga tersedia image disk VMware.

Jika bukan ini, saya hanya akan menggunakan rasa linux mana pun yang Anda sukai.

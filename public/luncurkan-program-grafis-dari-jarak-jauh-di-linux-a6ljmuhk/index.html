Luncurkan program grafis dari jarak jauh di Linux

Apakah ada cara untuk meluncurkan program grafis dari jarak jauh (misalnya, melalui SSH) pada mesin Linux? Saat menggunakan Putty, jika saya menjalankan perintah yang membutuhkan X, saya "gagal membuka tampilan". Perhatikan bahwa saya ingin program diluncurkan pada mesin jarak jauh - jadi saya TIDAK ingin meneruskan output ke server X lokal atau semacamnya.

Saya ingin ini khusus untuk membuat aplikasi remote control di Android - yaitu saya menekan tombol di ponsel saya, ini membuka tayangan slide gambar di mesin Linux.

Saya mulai merasa seperti saya mungkin perlu menulis proses server saya sendiri untuk mendengarkan acara ini ... tetapi jika saya dapat mendorongnya melalui SSH, mungkin akan lebih sederhana.

Anda harus mengatur variabel lingkungan DISPLAY ke server X yang Anda inginkan untuk dihubungkan dengan program Anda.

Misalnya xterm dapat dimulai dengan: DISPLAY=:0 xterm

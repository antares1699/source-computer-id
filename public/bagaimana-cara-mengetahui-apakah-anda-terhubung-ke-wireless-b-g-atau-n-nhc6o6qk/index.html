Bagaimana cara mengetahui apakah Anda terhubung ke Wireless B, G atau N?

Saya menggunakan Windows 7 di semua desktop berkabel dan laptop nirkabel di jaringan rumah saya. Saya baru-baru ini meningkatkan sakelar Ethernet saya ke Gigabit dan langsung melihat peningkatan throughput pada perangkat berkabel. Saya juga membeli WAP Wireless-N tetapi dengan penurunan kecepatan transfer file nirkabel.

Saya telah diberi tahu bahwa sejumlah alasan dapat mempengaruhi kecepatan nirkabel termasuk WAP mana yang digunakan, berapa banyak perangkat nirkabel yang terhubung, mode keamanan mana yang digunakan, dll. Namun, itu tetap tidak relevan dengan pertanyaan saya.

Setiap laptop saya mengklaim mendukung Wireless-N tetapi saya tidak dapat menemukan cara untuk menentukan apakah laptop tersebut benar-benar menjalankan Wireless-N atau terhubung ke WAP melalui semacam mode campuran. Saya tidak memiliki kendali atas perangkat WAP jadi tidak dapat membedakan mode apa yang sedang dijalankan.

Adakah cara untuk mengetahui mode mana yang sedang digunakan dan berapa throughput untuk setiap perangkat yang terhubung tanpa memiliki akses ke antarmuka WAP?

Saya menggunakan Windows 8, tapi saya yakin hal yang sama harus bekerja untuk 7. Ini tidak secara tepat memberitahu Anda protokol, tetapi memberitahu Anda kecepatan jaringan saat ini.

  1. Buka Jaringan dan Pusat Berbagi
  2. Klik kiri pada koneksi Wi-Fi
  3. Perhatikan level "Kecepatan" untuk sementara waktu

Saya melakukannya dengan milik saya sendiri, dan biasanya dipatok pada 54,0 Mbps, menunjukkan bahwa ini berfungsi pada 802.11g. Jika Anda melihatnya di atas 54 Mbps, Anda menggunakan 802.11n, karena 54 setinggi g.

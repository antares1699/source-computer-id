Untuk menghapus Funmoods dari Google Chrome

  1. Pertama-tama pergi ke Control Panel, lalu Programs and Features for Windows 7. Cari "Funmoods" dan hapus instalannya.
  2. Buka jendela browser Chrome, Klik kiri kunci pas (atau kunci inggris jika Anda menggunakan nama Amerika), kanan atas jendela browser.
  3. Pindahkan kursor ke bawah ke "Alat". Klik kiri, "Ekstensi". Di kiri bawah dasar-dasar klik kiri "Dasar", di samping bagian "Beranda", jika Anda menemukan "Funmoods" di sini, hapus, dan untuk saat ini, di atasnya, kiri klik, Halaman "Gunakan Tab Baru" dan Anda selesai pada bagian ini.
  4. Sekarang lihat ke bawah halaman "Dasar-dasar" untuk bagian "Pencarian" dan klik kiri, "Kelola Mesin Telusur". Jika Anda tidak menemukan bagian "Pencarian", cari bagian "pengaturan" dan itu akan membawa Anda ke bagian "Kelola Mesin Pencari".
  5. Di bawah "Opsi pencarian default", arahkan kursor Anda pada entri "Funmoods" dan tanda x akan muncul di tepi kanannya. Klik kiri x dan itu akan menghapusnya dari daftar.
  6. Terakhir, Periksa saja bahwa hasil penelusuran Anda tidak lagi diberikan oleh "Funmood".

Ini adalah satu-satunya metode yang saya temukan untuk menghapus Funmoods saat memulai.

  • Klik pada "kunci inggris" di kanan atas. Buka "Pengaturan".
  • Temukan "Saat Memulai". Anda akan melihat "Set Pages" berwarna biru, klik.
  • Daftar item start up akan ditampilkan.
  • Di sebelah kanan Funmoods sebuah "X" akan muncul saat mouse bergerak di atasnya.
  • Klik "X" untuk menghapus Funmoods.
  • Lalu klik OK. Itu hilang.

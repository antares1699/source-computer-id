Menjalankan chrome di Backtrack Linux

Saya menggunakan distribusi linux Backtrack 5, dan saya ingin menginstal chrome, tetapi secara default tidak akan berjalan sebagai root.

Saya tahu bahwa menggunakan --user-data-dir akan membiarkan saya menjalankannya sebagai root.

Tetapi apakah ada cara yang lebih aman untuk menggunakan chrome di distribusi linux ini tanpa harus menjalankannya sebagai root?

pengguna default di Backtrack adalah root, tetapi Anda dapat mencoba adduser

# type adduser

adduser is /usr/sbin/adduser

lalu tambahkan pengguna tanpa hak istimewa, beri dia hak grup

# adduser <name>
# passwd <name>

masukkan password 2 kali, dan logout / reboot.

login sebagai pengguna tanpa hak istimewa harus bekerja.

1) buat pengguna biasa

$ sudo useradd knoim
$ sudo passwd knoim

2) login sebagai pengguna biasa

$ su knoim
<knoims password>

3) Jalankan firefox $ $ firefox atau menelurkan proses $ firefox &

PS: solusi ini seharusnya berfungsi untuk sebagian besar distro linux termasuk backtrack. jika Anda sudah memiliki hak akses root (Anda telah mengatakannya tetapi saya bermaksud menulis solusi umum) Anda tidak perlu menggunakan sudo

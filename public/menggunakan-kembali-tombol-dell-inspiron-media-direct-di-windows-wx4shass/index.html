Lebih baik terlambat daripada tidak sama sekali...

Arahkan ke kunci berikut di registri:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\MobilePC\HotStartButtons\0

dan ubah ApplicationPath nilai ke jalur program apa pun yang ingin Anda jalankan sebagai gantinya ketika Anda menekan tombol Media Direct.

Untuk hibernasi, gunakan rundll32.exe PowrProf.dll, SetSuspendState

Saya yakin Anda perlu menginstal QuickSet agar ini berfungsi, yang merupakan standar dengan laptop Dell baru. Anda selalu dapat mengunduh dari Situs web Dell jika Anda belum memilikinya.

Metode 1: Tombol pintas atau pemetaan ulang keyboard

Faktanya adalah bahwa Tombol Langsung Media ini hanya mengemulasi tombol keyboard. Ini berarti Anda dapat menggunakannya sebagai kunci untuk pintasan apa pun. Cukup klik kanan pintasan apa pun di desktop, pilih Properti, klik di dalam "Tombol pintasan", tekan tombol Media Direct, lalu OK. Ini mengasumsikan bahwa program Dell Quick Set tidak diluncurkan untuk mengacaukan masalah.

Hal lain yang dapat Anda lakukan dengan Tombol Langsung Media memetakannya ke kunci lain, hanya dengan menggunakan teknik pemetaan keyboard.

Yang perlu Anda lakukan hanyalah mengunduh program seperti Keytweak , tetapkan hotkey ke pintasan apa pun di desktop, dan gunakan Keytweak (atau program pemetaan keyboard lainnya) untuk memetakan tombol Media Direct ke hotkey ini. Mulai sekarang, menekan Tombol Media Direct akan meluncurkan pintasan.

Detail lebih lanjut tersedia di Bagaimana cara ... memetakan ulang keyboard saya dengan KeyTweak? .

Pemetaan yang lebih rumit dari Tombol Langsung Media juga memerlukan Perangkat Lunak Microsoft Intellitype Keyboard . Lihat artikelnya Cara Menyesuaikan Tombol Dell Media Direct untuk Hibernasi , untuk contoh yang mengatur tombol Media Direct untuk menghibernasi mesin.

Metode 2: Ganti MDdirct.exe

Satu lagi metode untuk menyesuaikan tombol Dell Media Direct menggunakan fakta bahwa semua yang dilakukannya adalah meluncurkan program C:\Program Files\Dell\MediaDirect\MDdirct.exe. Jadi yang harus Anda lakukan adalah menggantinya dengan program lain atau skrip terkompilasi (menyimpan file yang ada, untuk berjaga-jaga).

Lihat artikelnya Ubah tombol Dell Media Direct misalnya, di mana MDdirct.exe diganti dengan Media Center. Ini juga memerlukan program Dell Quick Set (secara default adalah program startup yang diinstal Dell).

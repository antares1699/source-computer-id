Anda bisa mendapatkan 'Open With' di menu konteks untuk file apa pun hanya dengan menahan Bergeser kunci saat Anda mengklik kanan.

Jika Bergeser + klik kanan tidak berfungsi, Bergeser + F10 mungkin berhasil.

Untuk mendapatkan 'Edit' di menu konteks, Anda perlu mengedit registri. Buka regedit dan telusuri ke HKEY_CLASSES_ROOT\bat\shell. Buat kunci baru bernama Edit, lalu buat kunci baru di bawah yang disebut command. Edit nilai default dari kunci perintah menjadi "c:\windows\notepad.exe" "%1".

Bagi saya, open with tidak tampak bahkan untuk shift. Namun, Mengedit registri seperti yang ditentukan di jawaban lain tidak berfungsi. Dengan satu modifikasi:

C:\> reg.exe add HKEY_CLASSES_ROOT\batfile\Shell\Edit\Command /ve /t REG_EXPAND_SZ /d "xxx"

Anda perlu menggunakan di atas dalam cmd untuk membuat kunci dengan tipe default string yang dapat diperluas.

Juga, Ketika saya mengacaukan perintah buka, saya harus memperbaikinya dengan menghapus:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FileExts\.bat

dan semua sub kunci.

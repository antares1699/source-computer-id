Mengatur tombol Taskbar untuk digabungkan saat penuh melalui PowerShell

Apakah mungkin untuk mengatur opsi tombol Taskbar untuk Menggabungkan saat taskbar penuh melalui perintah PowerShell? Saya telah melihat Setel "tidak pernah menggabungkan" di Windows 7 menggunakan registri? tapi saya mencari pendekatan PowerShell yang bekerja tanpa logout.

4
Joshka — Sumber
  • Saya cukup yakin setiap perintah Powershell masih perlu mengubah nilai registri itu. Alih-alih keluar, coba mulai ulang proses penjelajah.
    — Qwilson

1 Jawaban

7
7

Ini berfungsi seperti yang dinyatakan @Qwilson. Dalam kode Powershell:

1. Atur nilai registy

Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced -Name TaskbarGlomLevel -Value 1

2. Hentikan Explorer karena akan otomatis restart sendiri.

Stop-Process -Name "Explorer"
Bagikan
MFT — Sumber
  • Ini membutuhkan satu edit, Anda perlu mengatur nilai "TaskbarGlomLevel" sama dengan "1" bukan "0". 0 untuk perilaku default yaitu "Selalu Gabungkan, sembunyikan label", 1 untuk "Gabungkan saat bilah tugas penuh" seperti yang ditanyakan dalam pertanyaan awal.
    — Always Learning
  • Atur nilainya ke 2 jika Anda ingin tombol bilah tugas tidak pernah digabungkan.
    — bizna

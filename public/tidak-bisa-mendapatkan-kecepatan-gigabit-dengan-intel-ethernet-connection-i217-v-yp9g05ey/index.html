Tidak bisa mendapatkan kecepatan gigabit dengan Intel Ethernet Connection I217-V

Saya memiliki paket gigabit dengan penyedia internet lokal. Saya menggunakan kabel cat 5e untuk terhubung ke router gigabit (TL-WR1043ND) atau langsung ke internet menggunakan koneksi pppoe.

Adaptor jaringan saya adalah Intel Ethernet Connection I217-V, saya menggunakan Windows 7 64 bit. Saya telah memperbarui driver kartu jaringan saya.

Masalah saya adalah saya tidak dapat mengunduh dengan lebih dari 10 MB / s dari internet (sama untuk tes kecepatan)

Saya membuat ftp lokal menggunakan router dan flash drive / hdd eksternal - kecepatan unduh maksimal 10 MB / s jadi saya pikir ada yang salah dengan kartu ethernet saya.

Jika saya pergi ke status Local Area Connection, saya dapat melihat bahwa kecepatannya 1,0 Gbps.

Namun, jika saya membuka properti -> konfigurasi -> Kecepatan tautan saya mendapatkan pesan aneh yang mengatakan bahwa 1,0 Gbps tidak tersedia.

Apakah saya di jalur yang benar? Haruskah saya mendapatkan kartu jaringan baru atau haruskah saya menyelidiki lebih lanjut (kabel / koneksi rusak, dll)?

Coba nonaktifkan "Ethernet Hemat Energi". Ini menonaktifkan beberapa masalah konektivitas untuk saya. Masuk ke Device Manager (klik kanan pada ikon windows di start bar) dan edit properti untuk Koneksi Ethernet. Di bawah tab "Advanced", ubah "Value" untuk "Energy Efficient Ethernet" menjadi mati. Beri tahu kami jika ini berhasil!

Tampaknya kartu jaringan Gigabit Anda berpikir bahwa ada masalah dan menurunkan kecepatan hingga 10 mb / s.
Kedengarannya seperti kabel UTP yang buruk.
Seolah-olah 1 atau 2 dari 8 kabel internal putus atau jika kabel mengalami tekanan yang parah (terlipat dalam tikungan yang sangat tajam atau telah dijepit di suatu tempat).
Jika kabel adalah buatan rumah, konektor RJ45 mungkin macet atau salah kabel.

Dapatkan sendiri kabel Cat 5e lainnya. Itu seharusnya memperbaiki masalah.

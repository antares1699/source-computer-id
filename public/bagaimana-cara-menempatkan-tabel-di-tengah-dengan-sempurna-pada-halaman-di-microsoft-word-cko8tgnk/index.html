Bagaimana cara menempatkan tabel di tengah dengan sempurna pada halaman di Microsoft Word?

Saya mencoba memasukkan tabel di Microsoft Word. Seluruh halaman akan terdiri dari hanya tabel ini, dan saya ingin tabel ini berada tepat di tengah (baik secara horizontal maupun vertikal) pada halaman.

Saya tidak mengalami kesulitan untuk membuatnya berada tepat di tengah halaman secara horizontal, tetapi saya tidak bisa membuatnya berada di tengah halaman secara vertikal. Saya memiliki tata letak halaman yang diatur ke tengah secara vertikal, tetapi ada baris tambahan di bawah tabel yang tidak akan saya hapus, yang menyebabkan spasi ekstra di bawahnya.

Saya menggunakan Microsoft Word 2010. Bagaimana saya bisa membuat tabel ini berada di tengah sempurna secara vertikal dan mengabaikan garis konyol yang bersikeras ada di sana?

Ini seharusnya cukup mudah menggunakan properti Tabel. Coba ini:

  1. Pilih tabel Anda, lalu klik kanan.
  2. Pada Tab Tabel, Pembungkusan Teks, pilih Sekitar. Ini harus mengaktifkan opsi Positioning.
  3. Pilih Positioning, dan dalam kotak dialog Table Positioning, atur:
    • Posisi Horizontal = Tengah, Sehubungan dengan Halaman
    • Posisi Vertikal = Tengah, Relatif dengan Halaman
    • Pindah dengan Teks = tidak dicentang
  4. OK keluar dari kotak dialog dan Tabel Anda harus berada di tengah. Dan itu harus tetap di tengah terlepas dari perubahan pada Tabel (mis. Menambahkan baris) atau perubahan pada teks di sekitarnya (termasuk tambahan, "garis konyol".

FWIW, Anda bahkan tidak perlu mengubah Properti Tata Letak Halaman, solusi ini menangani lokasi Tabel terlepas dari properti Halaman, jadi teks apa pun yang mungkin ingin Anda tambahkan masih dapat dibenarkan.

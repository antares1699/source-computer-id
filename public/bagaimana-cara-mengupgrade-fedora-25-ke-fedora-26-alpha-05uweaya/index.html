Bagaimana cara mengupgrade fedora 25 ke fedora 26 alpha?

Saya ingin menguji sesuatu di fedora 26 alpha, tetapi saya tidak ingin menginstal ulang seluruh OS. Saat ini saya menggunakan fedora 25, bagaimana cara mengupgrade langsung ke fedora 26 alpha?

  • Fed26 alpha tidak akan dirilis hingga awal Maret. Cabang pengembangan fedora disebut kulit mentah, dan inilah yang akan menjadi pelepasan alfa itu. Anda dapat beralih ke cabang kulit mentah dengan mengikuti petunjuk di tautan berikut. Ini akan memperbarui Anda ke fed26 tetapi ini adalah rilis pra-alfa. Peringatan biasa- jangan lakukan di mesin utama, buat cadangan apa pun yang ingin Anda simpan, bersiaplah untuk sistem yang tidak stabil. fedoraproject.org/wiki/Releases/Rawhide

Tujuan saya adalah untuk mengoreksi jawaban linuxdev2013 tetapi saya tidak memiliki reputasi yang cukup untuk mengomentarinya.

Saya perlu melakukan (perhatikan bahwa tidak ada tanda hubung sebelum mengunduh):

sudo dnf system-upgrade download --refresh --releasever=26

Saya juga perlu menggunakan --nogpgcheck untuk mengatasi masalah gpg yang tampak jelas dengan paket alpha kernel, tetapi saya berharap itu adalah kesalahan sementara.

Ngomong-ngomong, file dokumentasi aktual menjelaskan proses ini.

Cara yang jauh lebih mudah dan waras untuk ini.

Jalankan perintah berikut di terminal satu per satu:

sudo dnf install dnf-plugin-system-upgrade
sudo dnf system-upgrade download --refresh --releasever=26
sudo dnf system-upgrade reboot

Setelah bagian penginstalan selesai dan menjalankan boot ulang:

sudo dnf system-upgrade clean

... dan nikmati penginstalan f26 alpha (ditingkatkan di tempat)!

Untuk informasi lebih lanjut tentang plugin peningkatan sistem dnf lihat, https://fedoraproject.org/wiki/DNF_system_upgrade .

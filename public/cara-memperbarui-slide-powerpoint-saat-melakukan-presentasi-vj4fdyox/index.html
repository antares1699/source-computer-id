Cara memperbarui slide PowerPoint saat melakukan presentasi

Saya perlu menemukan cara untuk memperbarui slide powerpoint di tengah presentasi dengan gambar berdasarkan umpan balik audiens (mis. Audiens memilih gambar dan kemudian gambar itu diletakkan di slide nanti).

Saya khawatir tentang risiko pengeditan dalam mode presentasi dan editor secara tidak sengaja keluar dari tayangan slide di tengah-tengah presentasi.

Apakah ada add-on dll. Yang dapat digunakan di PowerPoint untuk mengedit / memodifikasi slide selama pertengahan presentasi sambil mencegah risiko mengacaukan presentasi langsung jika memungkinkan?

Ide sejauh ini:

  • Menggunakan plugin LiveWeb dan situs FTP / Dropbox untuk mengarahkan ke gambar dan menimpa gambar di slide LiveWeb berdasarkan umpan balik audiens (terlalu rumit untuk editor yang akan melakukan ini secara langsung)
  • Perbarui presentasi dari jarak jauh saat berada di jaringan bersama (tidak yakin apakah PPT akan memperbaruinya secara langsung.).
  • Memperbarui: Selama presentasi menggunakan "Presenter View" dan monitor tambahan, Anda dapat menavigasi seluruh komputer dan mengedit slide apa pun di PowerPoint. Slide yang diedit akan menampilkan konten / gambar yang diperbarui dalam presentasi langsung. Namun, melakukan ini akan menjeda presentasi langsung. Jadi jika orang yang mempresentasikan perlu pergi ke slide berikutnya atau mengaktifkan animasi saat Anda mengedit, itu tidak akan terjadi sampai presentasi dilanjutkan. Sepertinya ini akan berfungsi selama penyaji tidak mencoba pindah ke slide berikutnya sebelum editor selesai mengedit. Adakah add-in yang memungkinkan pengeditan DAN navigasi presentasi secara bersamaan?

Adakah yang pernah melakukan hal seperti ini sebelumnya? Apa yang Anda lakukan untuk membuatnya berfungsi dalam presentasi langsung?

Di PowerPoint 2013, ini sangat sederhana:

  1. Di Presenter View, klik Show Taskbar (pojok kiri atas)

  2. Ketika taskbar terbuka di bagian bawah layar, klik pada tab PowerPoint,

  3. Dalam tiga tampilan yang diperlihatkan (Editor, Presenter dan SlideShow), klik pada tampilan Editor.

  4. Jika Anda ingin audiens melihat perubahan Anda saat Anda membuatnya, tampilkan slide yang sama di semua 3 tampilan.

  5. Jika Anda tidak ingin audiens melihat perubahan hingga Anda selesai, klik ikon layar Hitam di Tampilan Penyaji (Monitor dengan garis diagonal melintasi), sebelum Anda memulai perubahan, ATAU ubah ke slide lain dalam tampilan Penyaji.

  6. Tayangan slide akan menampilkan slide Anda yang diubah di layar besar, saat Anda membukanya.

Selamat PowerPoint!

Ini sangat sederhana, terutama karena Anda mengatakan Anda ingin mengubah slide yang saat ini tidak terlihat (yang bisa menjadi rumit karena bug di beberapa versi PPT).

Tambahkan ini ke modul VBA di presentasi Anda.
Anda harus menyimpan presentasi sebagai PPTM atau PPSM daripada PPTX / PPTX.

Ikuti instrux yang disertakan sebagai komentar:

Option Explicit

' We'll modify slide #4 ... change as needed
' Make sure that the slide has no empty content or picture placeholders on it
Const lSlideNum As Long = 4

Sub AddAnImage()
    ' add a shape to any slide you like
    ' assign the shape an Action Setting of Run Macro: AddAnImage

    Dim oSl As Slide
    Dim oSh As Shape

    Set oSl = ActivePresentation.Slides(lSlideNum)

    ' bring in the image; setting width/height to -1 ensures that you
    ' don't distort it
    Set oSh = oSl.Shapes.AddPicture("c:\temp\photo.jpg", msoFalse, msoTrue, 0, 0, -1, -1)

    With oSh
        .LockAspectRatio = msoTrue ' to make sure it stays undistorted
        ' change its position/size as you wish
        ' for example, let's make it the full width of the slide:
        .Width = ActivePresentation.PageSetup.SlideWidth
    End With

End Sub

Option Explicit

' We'll modify slide #4 ... change as needed ' Make sure that the slide has no empty content or picture placeholders on it Const lSlideNum As Long = 4

Sub AddAnImage()
    ' add a shape to any slide you like
    ' assign the shape an Action Setting of Run Macro: AddAnImage

    Dim oSl As Slide
    Dim oSh As Shape

    Set oSl = ActivePresentation.Slides(lSlideNum)

    ' NEW:  Delete any existing image we may have added previously
    On Error Resume Next
    Set oSh = oSl.Shapes("MagicImage")
    If Err.Number = 0 Then
       oSh.Delete
    End If

    ' bring in the image; setting width/height to -1 ensures that you
    ' don't distort it
    Set oSh = oSl.Shapes.AddPicture("c:\temp\photo.jpg", msoFalse, msoTrue, 0, 0, -1, -1)

    With oSh
        ' NEW:  Name it so we can locate it easily later:
        .Name = "MagicImage"
        .LockAspectRatio = msoTrue ' to make sure it stays undistorted
        ' change its position/size as you wish
        ' for example, let's make it the full width of the slide:
        .Width = ActivePresentation.PageSetup.SlideWidth
    End With

End Sub

Pencadangan Time Machine yang setara untuk Windows [ditutup]

Apa perangkat lunak cadangan yang paling dekat dengan Time Machine (utilitas cadangan hebat yang terpasang di Mac OS X) untuk Windows (terutama Windows 7).

Bagi mereka yang bertanya-tanya, fitur terbaik dari Time Machine adalah:

  1. tampilan mudah dari berbagai versi file yang dicadangkan di masa lalu
  2. pilihan untuk memulihkan komputer Anda ke tanggal tertentu dari cadangan Anda
  3. kelancaran penimpaan versi lama file Anda (itu hanya menipiskannya menjadi mingguan, bukan menghapus versi terlama)

Apakah ada Windows yang setara yang memiliki beberapa / semua fitur tersebut?

Ini sedikit jawaban tangensial untuk pertanyaan Anda, tetapi jika Anda memiliki banyak komputer, dan Anda menginginkan kombinasi cadangan pusat dan server media, lihat Windows Home Server .

Ini secara otomatis mengambil backup setiap malam (inkremental) dari semua PC yang terhubung, tetapi jika dilihat, backup tersebut tampak seperti snapshot lengkap yang diberi stempel waktu dari seluruh konten PC yang terhubung. Secara default, snapshot malam disimpan selama 3 hari, kemudian mingguan selama 3 minggu, dan bulanan selama tiga bulan. Ini dapat diubah sesuai dengan kebutuhan Anda.

Anda dapat melihat konten snapshot, dan memilih satu untuk memulihkan semua atau beberapa file dari tanggal tertentu.

Itu menyimpan bacon saya beberapa kali, baik ketika saya secara tidak sengaja menghapus folder foto digital, dan ketika saya perlu mengembalikan ke versi file sebelumnya.

Alat Cadangan Windows bawaan di Windows 7 cukup bagus. Produk XP dapat diterapkan tetapi sulit digunakan, Vista tidak menyediakan kemampuan yang cukup untuk menentukan folder tertentu atau mengecualikan folder tertentu, tetapi alat cadangan Windows 7 sebenarnya cukup bagus.

Dikombinasikan dengan Versi Sebelumnya, dan beberapa file batch (file .cmd) yang menggunakan 7zip untuk memampatkan folder kode sumber tertentu menjadi snapshot malam (saya seorang programmer), ditambah sistem kontrol versi saya. Jadi, saya ingin melihat BERAPA BANYAK jenis backup dan redundansi yang dapat saya terapkan yang tidak mengambil masukan harian dari saya.

Saya tidak keberatan bahwa Klien Versi Sebelumnya menggunakan satu ton ruang pada hard disk lokal saya, pada kenyataannya, saya lebih suka seperti itu. Saya juga tidak keberatan System Restore memungkinkan saya mengembalikan perubahan driver secara terpisah, dan saya lebih suka seperti itu.

Berikan fakta berikut: (a) bahwa hard drive besar dan murah, dan (b) bahwa hard drive eksternal mungkin yang paling penting, tetapi bukan satu-satunya jenis cadangan yang harus Anda pertimbangkan, tidak ada alasan untuk memilih HANYA satu solusinya, (c) pencadangan online semakin penting, tetapi saya belum menganggapnya berguna karena saya memerlukan penyimpanan 100+ gigabyte, dan tidak ingin membayar ISP saya sebanyak itu untuk mengunggah 100 gb data dengan sangat cepat.

Apa yang belum saya temukan, adalah perangkat lunak pihak ketiga yang tidak menyebabkan saya lebih sakit atau bermasalah daripada nilainya.

Saya menggunakan mac di rumah, dan meskipun menurut saya Time Machine mencukupi, sebenarnya ada pemulihan kesalahan yang LEBIH KURANG dan tingkat keamanan yang lebih rendah di sana, daripada yang saya gunakan di Windows. Anda dapat mengatakan, bagaimanapun, bahwa saya merasa saya membutuhkan lebih banyak tingkat keselamatan dan keamanan mengingat bahwa Windows memiliki sejarah ketidakstabilan yang panjang. Namun, dengan Windows 7, saya memiliki lebih sedikit masalah dibandingkan dengan versi Windows sebelumnya. (Meskipun demikian, sistem operasi tidak dapat mencegah kegagalan perangkat keras, dan saya telah kehilangan hard drive utama saya sekali dalam 7 tahun terakhir, jadi pencadangan HARUS.)

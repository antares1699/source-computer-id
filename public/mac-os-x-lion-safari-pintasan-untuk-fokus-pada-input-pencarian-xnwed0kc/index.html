Mac OS X (Lion) Safari: pintasan untuk fokus pada input pencarian

Setiap kali saya perlu mencari di Safari, saya harus fokus pada bilah pencarian dengan mouse dan kemudian mengetik, yang menurut saya tidak nyaman.

Apa pintasan untuk Mac OS X (Lion) Safari agar fokus pada input pencarian?

4
bakytn — Sumber

2 Jawaban

6
6

Pintasan untuk mengakses kotak pencarian adalah ⌘ + ⌥ + F .

Anda juga dapat memfokuskan bilah alamat dengan ⌘ + L dan tekan Tab .

Jika Anda tidak puas dengan pintasan tiga tombol dan lebih memilih yang lain, buka Preferensi Sistem , Pilih Papan ketik , Pilih Pintasan Keyboard tab dan masuk Pintasan Aplikasi buat pintasan baru:

  • Judul menu adalah Pencarian Google… (bukan tiga titik tapi elipsis, tekan ⌥ + ; untuk mendapatkan salah satunya)
  • Aplikasi adalah Safari .
  • Pintasan keyboard terserah Anda, tetapi saya lebih suka ⌘ + K .

Tutup preferensi dan nikmati pintasan baru Anda.

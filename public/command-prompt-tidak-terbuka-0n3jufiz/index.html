Command Prompt Tidak Terbuka

Hai, saya baru saja menemukan masalah aneh dan menghabiskan satu jam mencoba menemukan masalah yang sama di internet tetapi tidak dapat ditemukan di mana pun. Saya menggunakan windows 10. Inilah masalahnya.

Ketika saya memulai mesin saya, prompt perintah terbuka untuk sepersekian detik dan menutup setelah itu komputer saya terbuka (Folder]. Ketika saya mencoba membuka sendiri command prompt itu melakukan hal yang sama. Saya memiliki plugin yang saya gunakan di atom yang menyematkan command prompt di dalamnya. Saya mencoba menggunakan command prompt dari sana dan lagi saya tidak dapat menggunakannya tetapi kali ini saya dapat melihat kesalahan yang disajikan. Berikut adalah kesalahan yang merupakan satu-satunya petunjuk yang dapat saya berikan masalah ini.

The screen cannot be set to the number of lines and columns specified.

Ada ide tentang cara memperbaikinya?

Saya memiliki masalah yang sama dan itu terkait dengan malware penambangan yang diinstal di mesin saya. Untuk menyelesaikan:

  1. Hapus file dan folder SoundMixer di data aplikasi Anda:% appdata% \ Microsoft \ Microsoft \ SoundMixer \ SoundMixer.exe
  2. Buka registri windows dan buka HKCU \ Software \ Microsoft \ Command Prosesor dan hapus kunci AutoRun.

sumber: https://forums.windowscentral.com/windows-10/466299-unable-launch-command-prompt.html

Rupanya Anda telah menyetel lebar dan / tinggi ke angka yang salah.

Untuk memperbaikinya, buka startmenu dan cari Command Prompt. Tapi jangan dibuka. Sebaliknya, klik kanan dan pilih properti.

Buka tab tata letak dan setel ulang pengaturannya ke default: lebar: 80, tinggi: 25 dan tekan ok.

Masalahnya sekarang harus diselesaikan.

Cara mengarahkan output program ke perintah Linux diff

Saya memiliki program yang menulis ke stdout . Apakah ada cara agar saya dapat mengarahkan output ke Linux beda perintah atau apakah saya harus menulis output ke file dan kemudian membandingkannya?

Sebagai contoh, saya memiliki banyak file input pengujian untuk suatu program dan output yang diharapkan sesuai di kumpulan file lain. Dan saya ingin melakukan sesuatu seperti ./program < t1.input | diff t1.expected.

Atau cara sederhana menggunakan bash:

diff <(./program) <(cat t1.expected)

Saya sering menggunakan kasus yang lebih umum

diff <(command1) <(command2)

di mana dua perintah mungkin serupa kecuali untuk mis. file input yang berbeda, atau versi program yang berbeda yang beroperasi pada file yang sama. Juga bekerja dengan gvimdiff.

Bagaimana cara membuka tab folder yang sebelumnya ditutup oleh folder virtual bernama "Desktop"?

Saya mencoba mengubah folder desktop saya dari D:\test4 menjadi D:\test5:

Pada langkah terakhir, saya tidak memindahkan file dari D:\test4 ke D:\test5.

Jadi yang terjadi adalah pada D:\, saya memiliki total 2 folder virtual, keduanya disebut Desktop:

  1. 1 folder virtual Desktop yang menutupi (dan menunjuk ke) D:\test5

  2. 1 folder virtual Desktop yang menutupi (dan menunjuk ke) D:\test4

Sekarang konten D:\test4 tidak lagi ditampilkan di desktop saya, namun masih ditutupi oleh folder virtual.

Jelas sejak D:\test4 sekarang bukan lagi milik saya Desktop folder, saya mencoba membuatnya kembali menjadi "folder normal" (yaitu tidak bertopeng).

Saya bertanya-tanya bagaimana cara membuka kedoknya?

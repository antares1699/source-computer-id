Setel ulang router gargoyle ke default

Saya punya Linksys WRT54GL router dengan gargoyle terpasang.

Sayangnya, saya lupa kata sandi saya, jadi sekarang saya mencoba menyetel ulang. Saya sudah mencoba metode hard reset 30/30/30, tetapi tidak berfungsi seperti yang diharapkan.

Setelah mem-boot router, nyalakan LED dalam urutan ini: yang pertama, lalu DMZ, dan kemudian Wifi. Setelah ketiga LED ini menyala, saya dapat melakukan ping ke router saya; balasan pertama memiliki latensi> 1 ms dan kemudian latensi tetap di = 1ms.

Saya melakukan ping menggunakan perintah ping -t pada Windows melalui kartu LAN dengan lampu LED LAN menyala.

Ketika saya Ctrl+C melakukan proses ping dan kemudian restart, saya mendapatkan hasil yang sama seperti sebelumnya: balasan pertama selalu > 1ms dan kemudian = 1ms untuk sisanya.

saya tidak bisa akses antarmuka web gargoyle, meskipun router ini membalas ping saya. Setelah menunggu setidaknya 30 menit, LED DMZ mati dengan sendirinya dan saya akhirnya dapat mengakses antarmuka web gargoyle. Kemudian saya mencoba cara hard reset 30/30/30 tetapi sekali lagi, itu tidak menghasilkan apa-apa.

Apa yang saya harapkan menjadi solusinya adalah cara mengatur ulang kata sandi atau mem-reflash firmware dengan yang lain.

Saat Anda menyalakan WRT54GL N, LED daya mulai berkedip. Anda harus menekan tombol SES berkali-kali sehingga Anda dapat boot ke mode gagal aman (LED Daya dan DMZ berkedip, antarmuka web dinonaktifkan).

telnet untuk itu tanpa kata sandi dan jalankan

mtd -r erase rootfs_data

Akhirnya, WRT54GL akan diatur ulang.

Sekarang Anda harus melanjutkan dengan hal yang sama:

Mengutip @iroel

Saat Anda menyalakan WRT54GL N, LED daya mulai berkedip. Anda harus menekan tombol SES berkali-kali sehingga Anda dapat boot ke mode gagal aman (Daya dan LED DMZ berkedip, antarmuka web dinonaktifkan).

Tetapi sekarang, ssh hingga 192.168.1.1, pengguna adalah root dan kata sandinya adalah: password

Jika Anda menggunakan Linux atau macOS:

ssh -l root 192.168.1.1

lalu ketik: firstboot

Ingat saja, Anda harus mengatur jaringan Anda ke jaringan 192.168.1.0/24 jika Anda mengatur ke jaringan lain sebelumnya. Perlu diingat bahwa dalam mode ini, router tidak lagi memiliki DHCP Server.

  1. Buka browser Web Anda dan ketik alamat IP router Anda, masukkan ID pengguna dan kata sandi Anda saat diminta

  2. Di bagian atas halaman Web, klik "Administrasi" dan kemudian "Default Pabrik"

  3. Klik tombol "Kembalikan Default Pabrik" dan kemudian klik "Ya" ketika Anda ditanya apakah Anda yakin ingin mereset Linksys WRT54G. Sekarang klik tombol "Simpan Pengaturan" di bagian bawah halaman Web.

Sumber: http://linksys-wrt54g-firmware.blogspot.com/

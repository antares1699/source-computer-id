Bagaimana cara menggabungkan beberapa file teks menjadi satu file besar?

Saya akan bekerja dengan banyak file teks dengan teman-teman saya. Saya butuh sesuatu yang akan menggabungkan beberapa file teks menjadi file master. Metode yang saya kejar adalah sesuatu yang sederhana dan akan membuat file master dengan satu klik tombol, lebih baik jika saya tidak perlu menginstal perangkat lunak apa pun (saya menggunakan Windows 7 64bit Professional), kesederhanaan adalah yang saya butuhkan .

Misalnya di: C:\mydocs\blah\blah

Saya sudah:

  • Halo .txt
  • Iloveyou.txt
  • Selamat tinggal.txt

Saya ingin file master menggabungkan teks di dalamnya dalam urutan itu. Bagaimana saya bisa melakukan ini?

Buka shell cmd, lalu

type Hello.txt Iloveyou.txt Goodbye.txt >masterfile.txt

atau

copy Hello.txt+Iloveyou.txt+Goodbye.txt masterfile.txt

Saya rasa akan lebih mudah jika Anda menulis .bat untuk melakukannya.

Seberapa penting file agregat master diperbarui sepanjang waktu? Jika konektivitas internet tidak menjadi masalah, bagaimana dengan penggunaan Google Docs ? Saya pikir Anda semua bahkan dapat mengedit file master dan bagian yang sesuai akan digabungkan.

Saya berpikir di sepanjang baris skrip tambahan batch sederhana, tetapi mungkin ada cara yang lebih modern untuk melakukan ini.

EDIT: Saya memiliki hal yang luar biasa. Itu disebut MobWrite , dan dikembangkan oleh Neal Fraser, GE . Situs Google Code memiliki versi mandiri yang dapat diunduh, dan Anda juga dapat membuat dokumen online di sini ala PasteBin.

Dalam kata-kata penulis,

MobWrite sekarang memiliki apa yang saya yakini sebagai algoritma pelestarian kursor paling canggih yang tersedia.

Jika Anda tertarik dengan rincian teknis (dan bagaimana perbandingannya dengan Google Docs), silakan lihat tulisannya di sini:

http://neil.fraser.name/writing/cursor/

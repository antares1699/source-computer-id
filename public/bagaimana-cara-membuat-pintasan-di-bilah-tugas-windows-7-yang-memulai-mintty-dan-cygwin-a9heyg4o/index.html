Bagaimana cara membuat pintasan di bilah tugas Windows 7 yang memulai mintty dan cygwin?

Saya ingin memiliki starter di taskbar Windows 7 yang memulai Cygwin, lalu mulai mintty dan idealnya menavigasi ke /cygdrive/c/Users/myName/

Saya kira ini harus dimungkinkan dengan menggunakan beberapa kombinasi file batch dan skrip shell yang tidak suci. Bisakah Anda membantu saya di sana?

Anda bisa membuat pintasan mintty, dengan pengaturan ini:

Target: C:\cygwin\bin\mintty /bin/env CHERE_INVOKING=1 /bin/bash -l
Start in: C:\Users\yourName

Tidak ada skrip yang dibutuhkan. Pengaturan CHERE_INVOKING menghentikan / etc / profile dari mengubah direktori ke $ HOME. (Variabel itu berasal dari paket 'chere'.)

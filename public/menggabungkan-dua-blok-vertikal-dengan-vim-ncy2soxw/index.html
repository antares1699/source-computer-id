Menggabungkan dua blok vertikal dengan vim

Saya memiliki 2 file

file1.txt
a =
b =

file2.txt
1
2

Bisakah saya menggabungkannya menggunakan gvim sehingga hasilnya terlihat seperti:
a = 1
b = 2

Tarik blok vertikal: Pilih mis. garis pada file2.txt dengan CTRL v , menyentak dengan y dan tempelkan setelah pindah ke akhir baris pertama pada file1.txt dengan p .

Jika Anda menjalankan vim pada GNU stack, Anda juga dapat menggunakan program paste untuk melakukan hal yang sama. Gunakan -d untuk menentukan pembatas apa yang akan digunakan antara bidang dari file.

paste -d "" 1 2 > merged.txt

bekerja dengan vim murni juga:

 % vim -O file1.txt file2.txt  # open both files vertically splitted

lalu di file1.txt yank blok ke register bernama, katakanlah 'a':

  • gg pergi ke awal file
  • ctrl v pergi ke pilih mode blok
  • G pergi ke baris terakhir
  • $ pergi ke karakter terakhir di baris terakhir
  • " Sebuah y yank untuk mendaftarkan 'a'

kemudian alihkan ke file2.txt dan lakukan:

  • gg pergi ke awal file
  • " Sebuah P. tempel konten register 'a' sebelum teks

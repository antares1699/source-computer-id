Ini adalah test perintah.

Misalnya, cari tahu apakah /etc/passwd ada atau tidak:

[ -f /etc/passwd ] && echo "Yes" || echo "No"

Secara teratur Anda menulisnya sebagai berikut:

if test -f /etc/passwd
then
  echo "Yes"
else
  echo "No"
fi

Pernahkah Anda melihat / memperhatikan jika kondisi di skrip shell? Bentuknya adalah:
[ -e $FILE ]

Dalam kasus tersebut, fungsionalitas pemeriksaan kondisi biasanya ada di dalam shell yang menafsirkan skrip tersebut, biasanya bash atau sh-aliased-bash.

Biner yang Anda lihat memiliki tujuan yang sama, tetapi ada sehingga pemeriksaan kondisi dapat digunakan di shell tanpa fungsionalitas seperti itu ada di dalamnya.

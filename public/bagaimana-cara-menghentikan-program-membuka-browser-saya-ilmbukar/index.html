Saya menyadari utas ini sudah tua, tetapi bagi mereka yang masih bertanya-tanya:

Buka Registry Editor (cari regedit di start menu), lalu buka HKEY_CLASSES_ROOT \ http \ shell \ open \ command

Klik dua kali (Default) di panel kanan, lalu hapus apa yang ada di kolom "data nilai" dan ganti dengan "" lalu klik ok.

Lalu pergi ke perintah HKEY_CLASSES_ROOT \ https \ shell \ open \, klik dua kali (Default) dan ganti apapun yang ada di dalam kotak dengan "" lagi dan klik ok.

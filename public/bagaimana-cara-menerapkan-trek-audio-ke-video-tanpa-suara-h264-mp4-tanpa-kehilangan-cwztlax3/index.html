Bagaimana cara menerapkan trek audio ke video tanpa suara h264 / mp4 tanpa kehilangan?

Saya memiliki video MP4 / h.264 tanpa suara. Dengan ffmpeg atau yang serupa, bagaimana saya dapat menerapkan trek audio ke file MP4 tersebut tanpa menurunkan kualitas videonya (yaitu tanpa mengode ulang trek video)?

Saya lebih suka solusi yang melibatkan perangkat lunak gratis, tersedia untuk Win7, Ubuntu, atau FreeBSD.

Perintah FFmpeg yang benar adalah:

ffmpeg -i video.mp4 -i audio.mp4 -map 0:0 -map 1:0 -c copy output.mp4

Ini akan menyediakan dua aliran input, video yang sudah Anda miliki dan audio, mis. dari file MP4 dengan audio AAC, dan gabungkan keduanya menggunakan opsi -map.

Di sini, angka pertama pada 0:0 adalah file masukan (0 untuk file video dan 1 untuk file audio), dan angka kedua adalah aliran dari file itu (0 karena masing-masing hanya ada satu aliran, video atau audio). Kedua aliran akan dipetakan ke satu file keluaran, jadi video pertama, lalu audio.

Bitstream akan disalin dan tidak dikodekan ulang menggunakan opsi -c copy. Anda dapat mengamati ini di keluaran FFmpeg:

Stream mapping:
  Stream #0:0 -> #0:0 (copy)
  Stream #1:0 -> #0:1 (copy)

Anda dapat menggunakan ffmpeg untuk melakukannya. Saya hanya melakukannya dari bagian belakang otak saya yang lemah tetapi mungkin sesuatu seperti ini akan berhasil:

ffmpeg -i movie-file.mp4 -i audio-file.mp3 -c:v copy -c:a copy output.mp4

Bagaimana cara agar iChat terhubung kembali secara otomatis?

Koneksi Internet saya cukup sering terputus. Dan setiap kali iChat menunjukkan pesan kesalahan "koneksi terputus". Saya harus mengklik "Oke" dan mengalihkan status saya dari Offline ke Tersedia. Bagaimana cara membuatnya melakukannya secara otomatis?

Saya baru saja mencari di Google jawaban ini:

http://blogs.oracle.com/safron/entry/make_ichat_reconnect_automatically

Pada dasarnya, ini mengatakan untuk mengatur tab cron untuk memberi tahu iChat agar terhubung kembali setiap 5 menit. Saya telah menyetelnya untuk berulang setiap menit:

$ crontab -e
* * * * *       osascript -e 'tell application "iChat" to log in'

Versi ini memeriksa bahwa aplikasi sedang berjalan, jadi jika Anda menutupnya, skrip tidak akan membukanya kembali:

*/5 * * * * osascript -e 'tell application "System Events" to if (processes whose name is "iChat") exists then tell application "iChat" to log in'

Ditemukan di sini, dengan cara: http://www.mikeperham.com/2011/12/30/getting-ichat-to-automatically-reconnect/

Bagaimana cara pergi ke jalur tertentu (folder yang berisi file tertentu) di Total Commander?

Katakanlah saya memiliki path tertentu - sangat panjang - ke sebuah file (katakanlah: c:\Documents and Settings\Username\Application Data\Mozilla\Firefox\Profiles\) dan saya ingin mengakses direktori tersebut dengan cepat di Total Commander. Apakah ada kotak dialog / pintasan keyboard yang memungkinkan saya untuk menempelkan jalur di sana dan memasuki lokasi itu, alih-alih proses membosankan untuk menggali ke dalam pohon direktori?

Jika Anda mengklik dua kali di bagian atas, di mana jalur saat ini ditampilkan, itu akan berubah menjadi bidang edit, di mana Anda bisa masuk ke jalur baru dan tekan Enter.

Anda dapat beralih ke mode edit jika kursor berada di [..] dan Anda menekan Shift + F6.

Pilihan lain adalah menulis jalur ke baris perintah, mendahului dengan cd seperti ini:

cd path_to_the_directory

lalu tekan enter.

Buat tab (tekan Ctrl + T ) untuk folder yang sering Anda gunakan. Jelajahi folder tersebut lalu klik kanan pada tab dan kunci tab tersebut.

Tab selalu ditampilkan saat Anda memulai komandan total, ini menghemat banyak penjelajahan ke folder saat Anda telah membuat tab untuk folder yang paling sering digunakan.

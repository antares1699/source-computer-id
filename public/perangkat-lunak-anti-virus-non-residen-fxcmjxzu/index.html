Perangkat lunak anti-virus non-residen?

Sejak 6 atau 8 tahun yang lalu saya telah berhenti menggunakan antivirus di komputer utama saya. Setiap kali saya mengunduh file yang mencurigakan, saya akan mengunggahnya ke VirusTotal atau menggunakan pemindai online lain untuk memeriksa apakah file tersebut aman. Namun, kecepatan unggah saya tidak terlalu spektakuler, jadi mengunggah file besar ke VirusTotal membutuhkan waktu lama. Mereka mendukung pemeriksaan menurut hash file, yang merupakan fitur hebat, tetapi terkadang saya memiliki file yang tidak mereka rekam. Di sisi lain, pemindai online membutuhkan waktu untuk memuat dan saya lebih suka pemindai yang dimuat secara instan.

Apakah ada perangkat lunak anti-virus non-residen yang baik yang kadang-kadang dapat saya gunakan untuk memindai file yang mencurigakan? Lebih disukai yang tidak membuat perubahan pada file sistem dan tidak menambahkan kaitan ke panggilan sistem. (Apakah saya terlalu menuntut? :-)

Jika Anda sangat lucu tentang menginstal perangkat lunak di komputer Anda, maka Anda selalu dapat membuat Mesin Virtual di komputer Anda. (VirtualBox adalah Perangkat Lunak Mesin Virtual gratis yang hebat) Anda dapat menginstal program antivirus di VM dan melakukan semua pengunduhan Anda di sana. Anda kemudian dapat menggunakan fitur jaringan bawaannya untuk mentransfer file ke host Anda. Anda juga dapat membuat cadangan VM jika Anda mengunduh file yang buruk dan perlu memformat VM. Anda cukup menghapus HDD Virtual dan memulihkan dari cadangan Anda.

Ini adalah kait kernel yang saya bicarakan. Alasan mereka menjadi lebih menonjol adalah karena keunggulan prosesor multicore. Mereka mencoba menyelinap di antara sinkronisasi utas dan pada dasarnya menimpa data. Ini adalah teknik yang sulit untuk menulis tetapi tetap ada. Mungkin bagian paling pahit dari artikel ini adalah Microsoft telah mengetahui tentang kerentanan ini sejak XP64. Ketika Microsoft memutuskan untuk memblokir "kerentanan" ini, perusahaan keamanan menolak keras dan mengancam akan menuntut. Rupanya, kait kernel yang sama ini sangat penting untuk fungsionalitas banyak mesin anti-virus. Karena semua orang suka menuntut Microsoft, mereka memutuskan untuk mundur. Dan dengan demikian tak terelakkan. . .

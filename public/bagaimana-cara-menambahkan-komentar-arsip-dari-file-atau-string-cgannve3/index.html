Bagaimana cara menambahkan komentar arsip dari file atau string?

Saya sedang menulis skrip BASH. Saya menggunakan RAR 3.93 di Ubuntu 10.08. Saya ingin menambahkan komentar ke arsip example.rar. Komentarnya adalah "Ini adalah komentar".

rar c example.rar
Processing archive rarcomment.rar
Reading comment from stdin

Tetapi saya ingin menambahkan komentar dari variale daripada mengetiknya dari stdin.

$Comment="This is a comment"

Bagaimana saya bisa melakukan itu?

4
DocWiki — Sumber

1 Jawaban

6
6
    export COMMENT="This is a comment
with multiple lines"
    echo "$COMMENT" | rar c example.rar

Untuk menulis beberapa variabel baris, jangan tutup kutipan sampai Anda selesai. Untuk membuatnya ditambahkan ke gema, gunakan tanda kutip.

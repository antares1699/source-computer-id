Gunakan program pengelola tugas SysInternals: Proses Explorer .

Di antarmuka Process Explorer ada perintah pencarian yang dapat Anda gunakan untuk mencari jalur file ini. Process Explorer kemudian akan menunjukkan kepada Anda executable yang bekerja di direktori ini dan membuat file-file ini.

Kemudian jalankan pencarian di internet untuk nama proses yang bekerja di direktori ini untuk menentukan apakah mereka malware atau bukan.

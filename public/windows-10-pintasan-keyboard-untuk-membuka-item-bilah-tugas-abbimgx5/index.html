Windows 10 - Pintasan Keyboard untuk Membuka Item Bilah Tugas

Di Windows 7, saya dapat menekan tombol Windows lalu menekan escape dan kemudian tab dan saya akan fokus pada item bilah tugas saya. Dari sana, saya dapat menggunakan tombol panah kanan dan kiri untuk melihat daftar item yang saya miliki di bilah tugas saya.

Bagaimana cara melakukan ini di Windows 10? Saya tidak ingin langsung membuka program tertentu (yang dapat saya lakukan dengan menggunakan tombol Windows + 1, 2, 3, dll ...).

Saya tidak berpikir ada pintasan langsung untuk ini, tetapi Anda dapat menggunakan solusi seperti yang Anda lakukan untuk Windows 7. Ada 2 yang dapat saya pikirkan:

  1. Anda dapat menggunakan "tombol Windows + B" untuk mengatur fokus ke area notifikasi (kanan bawah layar) dan kemudian tekan "shift + tab" dan fokus harus diarahkan ke bilah tugas Anda. Anda kemudian dapat menggunakan panah (kiri dan kanan) untuk menelusuri aplikasi yang terbuka.
  2. Anda dapat menekan tombol Windows dua kali, lalu menekan "tab" dua kali dan Anda akan berada di bilah tugas. Sekarang Anda dapat menggunakan panah (kiri dan kanan) untuk menelusuri aplikasi yang terbuka.

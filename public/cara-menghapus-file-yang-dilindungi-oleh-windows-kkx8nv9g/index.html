Cara menghapus file yang dilindungi oleh Windows

Saya memiliki hard disk portabel yang pernah diinstal di PC dan windows XP diinstal di atasnya. Sekarang saya menggunakannya sebagai cadangan dan saya ingin menghapus direktori Windows darinya, tetapi ketika saya mencoba menghapus direktori Windows, saya mendapatkan pesan kesalahan sehingga saya tidak dapat menghapusnya. Hard disk terhubung ke PC saya melalui USB. PC saya melakukan booting dari hard disknya sendiri dan memiliki Windows Vista Business.

Bagaimana cara menghapus file-file ini dari hard disk portabel?

Cara termudah untuk membersihkan hard drive adalah dengan memformat ulang. Pastikan Anda menarik file yang ingin Anda simpan, lalu lakukan pembersihan menyeluruh. Anda dapat menyimpan kembali file yang Anda tarik nanti.

Kemungkinannya, Pencarian Windows hanya memindai folder.

Ada banyak cara rumit untuk melakukannya, dari Boot Aman hingga menonaktifkan Layanan Windows hingga menjalankan Linux.

Sebenarnya, cara paling sederhana adalah dengan memformat ulang.

FileAssassin melakukan pekerjaan dengan cukup baik. Ini membuka kunci file dan (opsional) menghapus file. Ini juga dapat memicu pekerjaan "hapus saat restart" yang merupakan fungsi dari Windows itu sendiri.

  1. Unduh perangkat lunak dan buka. Anda akan melihat ini:

  2. Pilih file dengan menyeret file itu sendiri (dari explorer!) Ke dalam area teks FileAssassin atau menggunakan tombol pilih yang dilampirkan.

  3. Sekarang Anda dapat memilih apa yang harus dilakukan FileAssassin.

    • The "FileAssassin method": Opsi pertama ini memungkinkan Anda melakukan beberapa hal sekaligus, yaitu:
      • Buka kunci pegangan file yang terkunci (ini akan mencoba untuk menghapus semua pegangan Windows yang memblokir file agar tidak dihapus)
      • Bongkar modul (Saya bukan 100% dalam hal ini, tetapi jika itu perpustakaan (DLL), itu akan mencoba membongkar di mana pun itu dimuat)
      • Hentikan proses file (Ini akan menghentikan semua proses yang menggunakan nama file ini. Bermanfaat jika itu adalah exe, meskipun Anda bisa melakukannya sendiri menggunakan pengelola tugas!)
      • Hapus file (langsung)
    • Hapus saat boot ulang: Menggunakan fungsi hapus bawaan Windows saat boot ulang
    • Gunakan fungsi hapus normal: Hampir tidak ada gunanya karena itulah yang sebenarnya sudah Anda lakukan.

Jika Anda yakin Anda tidak menghapus file yang penting untuk sistem, pilih opsi pertama dan semua kotak centang di bawahnya. Kotak centang kedua dan ketiga hanya memicu untuk modul dan executable, jadi tidak masalah apakah mereka dipilih atau tidak.

Untuk menghapus file MS Windows, Anda harus mengklik kanan, klik Properties, pergi ke Security, Advanced, Change Permissions, dan pastikan Anda adalah admin. Klik nama pengguna Anda dan klik Edit. Kemudian klik "Kontrol Penuh" dan keluar dari semua jendela yang Anda buka. Ini harus bekerja untuk versi dari windows 2000 ke windows 7.

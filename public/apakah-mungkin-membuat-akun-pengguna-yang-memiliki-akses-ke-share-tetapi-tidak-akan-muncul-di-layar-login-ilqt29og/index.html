Akun pengguna untuk akses ke share

Membuat akun pengguna tidaklah istimewa. Buat saja akun pengguna di mesin host seperti biasa (menurut komentar Anda, Anda telah melakukan ini). Pastikan akun pengguna yang Anda buat memiliki akses ke share. Mesin tamu (mesin teman sekamar Anda) tidak harus memiliki akun dengan nama itu. Satu-satunya hal yang dia butuhkan untuk masuk, dari segi akses, adalah mengetahui nama pengguna dan kata sandi.

Untuk membantu Anda memahaminya dengan lebih baik, dia bahkan tidak perlu memiliki mesin jendela. Dia bisa menggunakan tablet, ponsel pintar atau perangkat apa pun termasuk pemindai ( kebanyakan printer memungkinkan Anda memindai langsung ke folder bersama ).

Jadi kriteria untuk akses ke folder bersama adalah nama pengguna dan kata sandi yang Anda siapkan.

Dalam hal ini nama pengguna dan kata sandi yang telah Anda siapkan. Nama pengguna Anda untuk administrator, baca / tulis, dan nama pengguna lainnya untuk hanya baca.

Menyembunyikan nama pengguna akun khusus

Edit Registry dengan menambahkan nilai dword dari nama pengguna dengan nilai "0".

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList

Dalam kunci ini buat entri dword dari nama pengguna. Klik kanan pada nama pengguna dan tentukan nilai "0". Ngomong-ngomong, nol adalah nilai default dari entri dword baru.

Anda juga dapat menyimpan ini sebagai file dengan ekstensi .reg dan mengkliknya untuk membuat entri.

File bernama myreg.reg:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList]
"username"=dword:00000000

Pastikan ekstensi file adalah * .reg "dan bukan * .txt". Anda mungkin harus memeriksa ekstensi menggunakan konsol cmd.

Anda telah melakukannya ... kurang tepat.

Anda harus:

  • Jalankan lusrmgr (Win+r, ketik lusrmgr.msc).
  • Jika Anda tidak memiliki kata sandi untuk pengguna Anda, Anda harus menambahkannya sekarang ( keamanan! ) - pilih pengguna Anda dari Users dan tambahkan kata sandi ('RMB', 'Password setup ...').
  • Pilih grup Guests dari Groups.
  • Tambahkan pengguna baru ke Guests (tombol Enter, Add..., tombol Localizations...) dari mesin mereka dengan Network Neighborhood.

Anda harus mempertimbangkan penyesuaian Guests kebijakan grup.

Anda tidak perlu benar-benar menyembunyikan akun Guests yang ditambahkan, tetapi autologon pengguna default Anda:

  • Jalankan netplwiz (Win+r, ketik netplwiz).
  • Pilih pengguna Anda dari daftar pengguna.
  • Hapus centang Users must enter user name and password to use this computer.
  • Apply dan masukkan kata sandi Anda - kata sandi tidak divalidasi di sini, jadi Anda harus memasukkan kata sandi yang benar untuk autologon berhasil.

Dan tentu saja Anda harus:

  • Bagikan folder yang diinginkan dengan Guests grup pengguna.

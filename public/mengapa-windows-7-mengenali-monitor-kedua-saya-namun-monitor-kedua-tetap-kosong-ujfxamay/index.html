Mengapa Windows 7 mengenali monitor kedua saya namun monitor kedua tetap kosong?

saya membeli sebuah monitor baru (Acer X203H) dan ingin menyambungkannya ke kartu grafis saya tetapi pada kartu grafis saya ada satu steker CRT yang sudah diisi dengan monitor pertama dan kata kedua DVI , yang tidak saya miliki kabelnya di monitor kedua saya.

Saya menemukan apa yang tampak seperti file VGA outlet di saya motherboard dan memasangnya di:

Saya Windows 7 mengenali monitor dan saya mouse melampaui monitor pertama saya tampaknya ke monitor kedua tetapi monitor kedua saya tetap kosong dan ketika saya mematikannya dan mengatakannya tidak ada sinyal .

Bagaimana cara mengaktifkan monitor kedua saya? Apakah saya perlu mendapatkan adaptor atau kabel untuk menyambungkannya ke kartu grafis saya dan bukan ke port VGA?

Info lebih lanjut berdasarkan jawaban:

Saya juga sudah membaca beberapa ulasan bahwa Acer X203H seharusnya memiliki dukungan DVI, tetapi berikut adalah gambar bagian belakang monitor saya tempat saya mencolokkan kabel, sepertinya ada tempat untuk kabel DVI tetapi model khusus ini tidak memilikinya. Ergo, saya pikir taruhan terbaik saya adalah mendapatkan konverter VGA-ke-DVI seperti yang disarankan Will.

Masalahnya adalah Anda menggunakan kartu video, jadi port VGA pada motherboard Anda dinonaktifkan. Saya tidak berharap monitor yang dicolokkan ke motherboard akan dikenali, tetapi saya kira itu mungkin.

Saya merekomendasikan untuk menghubungkan ke monitor Anda menggunakan slot DVI kartu video (port putih di sebelah koneksi VGA). Jika monitor Anda tidak mendukung DVI, Anda dapat memasang file DVI -> Konverter VGA ke dalam slot.

DVI biasanya menyediakan koneksi berkualitas lebih tinggi, jadi Anda harus mencolokkan monitor baru Anda ke slot DVI jika memungkinkan, dan monitor CRT lama Anda, ke slot VGA terdekat.

Menurut spesifikasi Acer X203H memang memiliki DVI, tampilan sekilas akan mengkonfirmasi hal itu. Dalam hal ini, solusi terbaik adalah kabel DVI-DVI.

Opsi kedua adalah konektor DVI-VGA, tetapi pertama-tama pastikan bahwa Kartu Grafis Anda mendukung 2xVGA. Porta DVI memiliki rasa yang berbeda, VGA analog menjadi pilihan. Dari gambar saya akan mengatakan Anda baik-baik saja, tetapi kabel DVI adalah taruhan yang lebih aman.

Dan Acer sedang murah, tidak termasuk kabel ke-2. Apakah Anda sudah mencentang kotak itu lagi?

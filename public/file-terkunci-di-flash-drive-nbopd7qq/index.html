File Terkunci di Flash Drive

Saya benar-benar frustrasi. Saya adalah pemilik OCZ 8GB Rally2 yang telah saya gunakan dengan sempurna selama sekitar 4 bulan sekarang, dan tiba-tiba minggu ini saya mulai mendapatkan kesalahan yang memberi tahu saya bahwa saya tidak dapat menghapus file karena "pastikan disk tidak dilindungi dari penulisan, penuh atau dan file tidak sedang digunakan "dialog. Inilah yang saya coba:

  • Menghapus file menggunakan FileASSASSIN (ini bekerja pada yang lainnya kecuali file pada flash drive sialan ini)
  • Memformat flash drive dan kemudian memulihkan informasi yang dicadangkan (format I ke NTFS, FYI)

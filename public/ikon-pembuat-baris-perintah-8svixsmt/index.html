Ikon pembuat baris perintah

Apakah ada utilitas baris perintah untuk Linux (atau OS mirip Unix lainnya) yang dapat:

  • Buat file .ico dengan banyak ikon di dalamnya
  • Lakukan hanya dari 1 ( satu ) png atau gambar jpeg

Sebagai contoh:

% cool_icon_maker myimage_128x128.png file.ico

Di file.ico seharusnya otomatis ada semua ukuran ikon seperti 128x128, 64x64, 32x32, 16x16, dll.

Saya tidak tahu solusi all-in-one, tapi saya tahu dua bagian yang bisa disatukan:

  • icoutils memiliki icotool, yang dapat membuat / mengekstrak file .ico.
  • ImageMagick memiliki convert, yang dapat mengkonversi dan mengubah ukuran file ke ukuran yang diinginkan.

Jadi, sesuatu seperti ini akan bekerja (mungkin hanya bekerja untuk file dengan ekstensi ".png"):

#!/bin/bash

# Just pass the original .png image as the only parameter to this script.
SOURCE="$1"
BASE=`basename "${SOURCE}" .png`

convert "${SOURCE}" -thumbnail 16x16 "${BASE}_16.png"
convert "${SOURCE}" -thumbnail 32x32 "${BASE}_32.png"
convert "${SOURCE}" -thumbnail 48x48 "${BASE}_48.png"
convert "${SOURCE}" -thumbnail 64x64 "${BASE}_64.png"

icotool -c -o "${BASE}.ico" "${BASE}"_{16,32,48,64}.png

rm -f  "${BASE}"_{16,32,48,64}.png

Itu sangat jelek, aku tahu. Tapi itu sederhana untuk dipahami, dan berhasil (saya mengujinya!). Berhati-hatilah, karena ini akan membuat file sementara di direktori saat ini, dan kemudian menghapusnya.

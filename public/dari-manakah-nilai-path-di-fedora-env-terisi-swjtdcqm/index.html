  • @Sen: secara default / sbin tidak berada di jalur pengguna biasa. Hal-hal di / sbin dimaksudkan untuk penggunaan root dan superuser saja. Cara tipikal untuk mendapatkan akses ke / sbin adalah dengan beralih ke akun superuser. Baik su - untuk melakukan root atau menggunakan sudo untuk mengakses hal-hal seperti sudo /sbin/service.

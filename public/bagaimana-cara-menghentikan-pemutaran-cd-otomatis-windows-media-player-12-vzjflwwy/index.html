Bagaimana cara menghentikan pemutaran CD otomatis Windows Media Player 12?

Saya telah menyetel putar otomatis ke "Tidak melakukan tindakan apa pun" di semua perangkat dan jenis file di Windows 7, tetapi Windows Media Player 12 tampaknya mengabaikan ini sepenuhnya. Saya juga tidak dapat menemukan pengaturan di opsi pemutar media mana pun.

Bagaimana cara mencegah CD memutar otomatis Windows Media Player 12?

Inilah yang saya temukan ketika saya mencari di Google untuk masalah ini, semoga ini berfungsi untuk Anda meskipun pengeditan registri ini dimaksudkan untuk Windows Media Player 10:

Saya memeriksa mesin saya yang lain dan menemukan perilaku yang sama dijelaskan di atas ketika sebelumnya tidak memutar CD secara otomatis saat Media Player 10 terbuka. Jadi saya mengaduk-aduk registri dan ditemukan bahwa kunci CDAutoPlay disetel ke 1 di kedua mesin di bawah [HKEY_CURRENT_USER \ Software \ Microsoft \ MediaPlayer \ Preferences]. Saya mengubah CDAutoPlay menjadi 0 dan banyak hal seperti sebelumnya. Berharap ini membantu pengguna lain mendapatkan hal yang sama isu.

Tautan ke postingan asli

Jika Anda harus menambahkannya, CDAutoPlay adalah nilai DWORD 32-bit.

Ini menyelesaikan sebagian masalah.

Buat tambahan berikut di registri dan pemutarannya tidak "tidak dapat dipulihkan"

  1. Buka Regedit

  2. Buka HKEY_CURRENT_USER \ Software \ Microsoft \ MediaPlayer \ Preferences

  3. Buat DWORD dengan nama "MediaFolderDetectionDepth" dan setel nilainya menjadi "0"

  4. Buat DWORD dengan nama "NonMediaCountToAbortMediaFolderDetection" dan setel nilainya menjadi "0"

Sekarang, saya memasukkan DVD dengan beberapa file media, dan pemutaran mp3 tidak berhenti. (sebelumnya ini menghentikan pemutaran)

Dalam hal memasukkan DVD, maka saya harus menghentikan pemutaran dan menekan "Putar Daftar Sebelumnya" di "Jendela Putar Sekarang"

Restart tidak diperlukan. (Saya melihat bahwa WMP12 membaca kunci setidaknya satu kali saat drive dimasukkan). Mungkin jawaban saya kurang lengkap. Jika demikian, seseorang tolong bantu saya (saya juga akan mengerjakan ini)

Saya harap ini berguna untuk beberapa level.

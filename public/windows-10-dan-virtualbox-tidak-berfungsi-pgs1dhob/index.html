Windows 10 dan Virtualbox tidak berfungsi

Saya memperbarui mesin saya ke Windows 10 tetapi saya memiliki masalah untuk membuatnya berfungsi. Kesalahan yang saya dapatkan ketika saya menginstal Virtualbox (versi 5.0.3) adalah:

Callee RC: 
REGDB_E_CLASSNOTREG (0x80040154)

Saya telah mencari-cari di Google dan saya menemukan masalah ini dan telah mencoba opsi yang dinyatakan dalam jawaban yang diterima tetapi ini tidak berhasil.

Ketika saya melakukan hal berikut:

regsvr32 /u VBoxC.dll

Dikatakan VBoxC.dll tidak dapat dimuat. Tidak dapat menemukan modul.

Ada saran?

Anda mungkin ingin mempertimbangkan ini.

Saya telah menginstal Windows 10 Enterprise, memutakhirkan dari Windows 8.1 Enterprise dan mengalami masalah penguncian Windows 10 cukup sering.

Setelah beberapa penggalian, saya menemukan bahwa manajer Hyper-V untuk Windows 10 masih membaca konfigurasi manajer Hyper-V untuk Windows 8.1. Tidak sampai saya melakukan reset lengkap Windows 10 (untuk menghapus konfigurasi versi sebelumnya), itu berfungsi dengan baik.

Meskipun tampaknya tidak terkait langsung dengan masalah Anda, maksud saya adalah, Windows Hyper-V, terutama di Windows 10, mungkin memperebutkan sumber daya dengan instalasi VitualBox Anda.

Saya akan menyarankan dari Program Files -> fitur untuk menjalankan, menghapus Hyper-V, reboot dan jalankan dari command prompt (dalam mode admin) sfc / scannow

Kemudian reboot, dan coba jalankan VirtualBox Anda (setelah Anda menginstal ulang VBox Anda seperti yang disarankan oleh poster sebelumnya).

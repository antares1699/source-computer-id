gunakan ffmpeg filter_complex trim dengan durasi waktu termasuk jam dan menit

Saya mencoba menyiapkan skrip untuk mengotomatiskan beberapa manipulasi video, termasuk memotong video. untuk itu saya menggunakan ffmpeg filter_complex.

Saya ingin menentukan posisi awal dan akhir menggunakan format HH: MM: SS.m, tetapi saya tidak dapat melakukannya, meskipun dokumentasi ffmpeg menyatakan bahwa hal itu dapat dilakukan. Berikut adalah versi kode yang disederhanakan:

ffmpeg -i input.mp4 -filter_complex \
"[0:v]trim=1.40.1:1.59.3,setpts=PTS-STARTPTS[v]" \
-map "[v]" -pix_fmt yuv420p -c:v libx264 -preset fast -y output.mp4

Saya pikir ini karena fakta bahwa ":" digunakan untuk memisahkan argumen dalam opsi filter.
Ada kesempatan untuk membuatnya berhasil?

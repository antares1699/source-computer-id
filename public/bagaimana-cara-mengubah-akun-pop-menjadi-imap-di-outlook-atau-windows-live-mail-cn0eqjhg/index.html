Bagaimana cara mengubah akun POP menjadi IMAP di Outlook atau Windows Live Mail?

Saya memiliki beberapa akun email yang disetel ke POP dan sekarang kami memiliki layanan IMAP, jadi kami ingin mengubah ke IMAP karena manfaat yang menyertainya.

Tetapi saya mencoba melakukannya di Outlook (2007) dan Windows Live Mail, tetapi tampaknya saya tidak dapat mengubah pengaturan akun, layanan seperti apa yang saya inginkan secara khusus POP atau IMAP, hanya jika saya membuat yang baru. akun mungkin, tetapi saya tidak ingin melakukan itu, apakah itu benar? Satu-satunya cara untuk mengubah POP ke IMAP adalah membuat akun baru di kedua program?

Tahu bagaimana saya bisa melakukan itu?

Anda tidak dapat beralih antara POP dan IMAP

  • IMAP - tetap selaras dengan akun tempat Anda masuk dengan semua perangkat
  • POP - selalu mendapat pesan baru untuk perangkat yang meminta

Hal terbaik untuk dilakukan adalah,

  1. ekspor email yang Anda inginkan dari akun POP
  2. nonaktifkan akun POP Anda
  3. buat akun IMAP baru Anda
  4. Impor POP-EMAILS ke akun IMAP Anda

Mengimpornya kembali akan mengirimnya kembali ke server, sehingga perangkat lain akan tetap di SYNCH, mis. IPhone, komputer rumah .. dll. Anda menghapusnya di salah satu perangkat, maka perangkat lainnya akan tetap aktif.

  • Server IMAP akan menjadi titik SYNCH terpusat, jadi semua email harus ada di sana, dan kotak surat di server dapat bertambah besar (mis. GMAIL menawarkan 7GB gratis, atau untuk 5USD per tahun Anda mendapatkan tambahan 20GB, yay! )

  • Dengan POP semuanya terdesentralisasi dan menyimpan data di SYNCH hampir tidak mungkin, tetapi Anda tidak memerlukan server untuk menyimpan sejumlah data.

Jika Anda tidak ingin mengimpornya kembali, pertahankan akun offline dengan email yang diarsipkan.

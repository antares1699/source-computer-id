Ambil / Ekspor kontak dari versi Viber PC

Pertanyaan:

Saya menggunakan Viber di PC (bukan aplikasi bergaya metro windows 8) dan ponsel. Sayangnya saya kehilangan ponsel saya. Masih saya bisa menggunakan versi desktop Viber. Saya dapat melihat semua kontak saya di sana (Viber + non-Viber). Saya perlu mengambil kontak tersebut sebagai .vcf atau format standar lainnya yang kami gunakan untuk menyimpan kontak.

Tolong beri tahu saya jika ada jalan.

Apa yang telah saya coba sejauh ini:

Saya membuka file di bawah "C: \ Users [userName] \ AppData \ Roaming \ ViberPC" menggunakan buku catatan dan memeriksa informasi kontak. Namun tidak berhasil.

Anda dapat membuka viber.db di C:\Users\USERNAME\AppData\Roaming\ViberPC\YourNumber dengan WordPad dan di suatu tempat di awal file Anda akan menemukan nomor telepon kontak. Cukup masukkan secara manual di telepon baru Anda dan mereka akan muncul di daftar kontak Anda di Viber.

Atau unduh shell baris perintah untuk mengakses dan memodifikasi database SQLite dan salin sqlite3.exe, viber.db dan data.db ke C:\. Kemudian di CMD (start -> run -> cmd.exe) posisikan diri Anda di C:\ dan masukkan sqlite3.exe viber.db
Lalu masukkan:

 SELECT ContactRelation.Number, Contact.FirstName, Contact.SecondName FROM Contact INNER JOIN ContactRelation ON Contact.ContactID = ContactRelation.ContactID ORDER BY Contact.FirstName;

Ini dia! Anda mendapatkan semua kontak yang terdaftar, nomor telepon terlebih dahulu dan kemudian nama!

Mungkin struktur basis data telah berubah, dalam kasus saya, saya harus sedikit mengubah instruksi @ Davidenko.

Instal shell baris perintah SQLite, mungkin dari sini sqlite.org/download.html . Itu dibundel dalam sqlite-tools.
Salin file

C:\Users\USERNAME\AppData\Roaming\ViberPC\YourNumber\viber.db

di suatu tempat. Sekarang di direktori PowerShell atau CMD cd hingga viber.db dan buka dengan:

sqlite3 .\viber.db

Itu membuka viber.db di shell SQLite.
Untuk mengekspor kontak sebagai CSV file, tulis di shell SQL:

.mode csv
.output contacts.csv
SELECT Contact.Name, Contact.Number, Contact.ViberContact FROM Contact;
.output stdout

Untuk mengekspor pesan teks sebagai CSV termasuk kontak terkait, tulis:

.mode csv
.output messages.csv
SELECT Contact.Name, Contact.Number, Contact.ViberContact, Events.TimeStamp, Messages.Body  FROM Contact INNER JOIN Events ON Contact.ContactID = Events.ContactID INNER JOIN Messages ON Events.EventID = Messages.EventID  ORDER BY Contact.Name;
.output stdout

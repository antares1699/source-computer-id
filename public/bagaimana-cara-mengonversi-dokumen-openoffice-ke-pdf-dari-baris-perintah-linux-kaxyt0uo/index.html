Bagaimana cara mengonversi dokumen OpenOffice ke PDF dari baris perintah Linux?

Saya memiliki siswa yang, ketika dimintai PDF, terkadang memberikan saya dokumen atau spreadsheet OpenOffice. file(1) dapat mengidentifikasi dokumen-dokumen ini, tetapi saya tidak dapat menemukan cara mengonversinya menjadi PDF menggunakan baris perintah. (Halaman manual untuk ooffice(1) mencantumkan opsi untuk mencetak dokumen tetapi tidak untuk dikonversi ke PDF.) Google tidak membantu, kecuali untuk memberi saya perasaan tidak nyaman bahwa ini tidak dapat dilakukan tanpa skrip bagus dalam bahasa yang tidak saya pahami dengan API yang dokumentasinya tidak dapat saya temukan .

Adakah yang bisa membantu saya memecahkan masalah mengonversi OpenDocument ke PDF hanya dengan menggunakan baris perintah Unix?

Saya melakukan ini dengan menggunakan cangkir-pdf . Anda harus menginstal OpenOffice, dan Anda harus menjalankan ooffice dalam mode tanpa kepala agar aplikasi tidak meluncurkan GUI.

printer PDF biasanya di repositori, jadi cukup sudo apt-get install cups-pdf atau yum install cups-pdf.

Selanjutnya Anda harus menginstal printer. Jika mau, Anda dapat menginstal printer melalui baris perintah .

Jika Anda menginstal printer melalui GUI, (di Ubuntu) semudah itu Sistem -> Administrasi -> Pencetakan . Klik panah dan pilih Printer Baru :

Pergi melalui wizard dan catat apa yang Anda beri nama printer Anda, Anda akan membutuhkannya dari baris perintah. Seperti yang Anda lihat, saya telah menggunakan cups-pdf:

Selesaikan wizard dan Anda siap untuk mengonversi. Buka terminal dan jalankan:

ooffice -headless -nologo -pt cups-pdf doc.odt

Di bawah direktori home Anda, Anda akan memiliki folder PDF. Di dalamnya akan ada dokumen Anda yang telah dikonversi.

Dengan libreoffice yang disertakan dengan CentOS 7 (4.1.4):

soffice --headless --convert-to pdf  file.odt 
soffice --headless --convert-to doc  file.odt 
soffice --headless --convert-to html file.odt

soffice dan libreoffice adalah biner yang sama suka satu sama lain. oowriter adalah script yang memanggil libreoffice --writer. Jadi hal di atas harus berfungsi untuk semua turunan staroffice (libreoffice, openoffice).

Saya menggunakannya untuk mengonversi ratusan dokumen / hari, berfungsi dengan baik.

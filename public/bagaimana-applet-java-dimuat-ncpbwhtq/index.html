Bagaimana applet Java dimuat?

Saya melakukan percakapan ini kemarin dengan teman saya dan saya mencoba menjelaskan kepadanya bahwa applet adalah kode yang telah dikompilasi sebelumnya yang dimuat oleh JVM melalui plugin browser Java. Saya mencoba menemukan beberapa dokumentasi atau penjelasan rinci tentang apa yang terjadi di browser saat situs dengan applet dikunjungi dan akhirnya bagaimana situs itu dimuat, tetapi tidak bisa.

Jawaban cepat: Kode sumber Java dalam file .java dikompilasi ke file .class yang berisi kode byte Java. Ini dikemas menjadi satu file .jar. HTML halaman web menyertakan referensi ke file jar ini. Browser mengambil jar, menjalankan Java Virtual Machine (JVM) yang diinstal secara lokal dan menyerahkannya ke data file jar. JVM kemudian melakukan kompilasi Just-In-Time (JIT) dari byte-code ke kode native seperlunya.

Bagaimana cara saya mencadangkan data saya sebelum menginstal ulang windows? File yang dihilangkan saat menyalin drive C.

Saya bertanya-tanya bagaimana cara memastikan saya mencadangkan semua file sebelum mengeluarkan C dan menginstal ulang Windows (reset pabrik).

Saya sangat berhati-hati saat menyalin dokumen, foto, dll. Ke drive eksternal. Namun tidak ada cara saya dapat mengecualikan kemungkinan bahwa mungkin ada di suatu tempat beberapa file yang mungkin saya perlukan. Misalnya, tepat setelah saya pikir saya selesai, saya menemukan folder di C di mana beberapa game telah meletakkan file game yang disimpan. Jadi saya mungkin kehilangan ini jika saya tidak melihatnya karena mereka berada di lokasi yang tidak biasa dalam perjalanan.

Satu opsi yang jelas tampaknya cukup menyalin seluruh drive sistem (dan partisi lain). Namun, ketika saya melakukannya dengan ctrl + c dan + v (salin dan tempel), Windows menyalin volume data yang jauh lebih sedikit (dalam GB) daripada yang ada di drive. Tampaknya beberapa file dihapus secara sistematis. Mungkin ini adalah file sistem yang tidak penting, tetapi saya tidak yakin.

Karenanya pertanyaan saya: bagaimana saya harus mencadangkan data saya sebelum reset pabrik untuk memastikan saya mendapatkan semuanya (di luar hal-hal yang dapat saya salin dengan mudah secara manual)?

Edit sebagai tanggapan atas @JourneymanGeek: Saya menemukan ini Tautan Technet di mana penulis menyatakan sangat bermasalah untuk meluncurkan vhd pada sistem yang sama dengan gambarnya. Jika itu benar, image disk tampaknya bukan solusi yang baik.

Cukup gambar seluruh drive - itu akan mencadangkan semuanya, dan perangkat lunak pencitraan yang paling layak akan membiarkan Anda melakukan kompresi. Anda kemudian dapat memasang drive secara langsung dan menghapus file.

Ada banyak pilihan yang berbeda tetapi disk2vhd Sepertinya opsi 'paling sederhana' karena versi modern windows mendukung VHD secara native. Gambar akan mengambil ruang yang digunakan oleh konten drive dan Anda dapat mengambil file darinya dengan santai. Saya tidak berpikir ada opsi kompresi apa pun tetapi kemudahan pemasangan menebusnya.

Demikian juga, sebagian besar perangkat lunak cadangan yang layak akan memungkinkan Anda memasang cadangan. Saya telah menggunakan macrium mencerminkan gratis untuk ini, tetapi acronis atau perangkat lunak cadangan umum lainnya yang melakukan gambar akan memungkinkan Anda melakukan ini juga.

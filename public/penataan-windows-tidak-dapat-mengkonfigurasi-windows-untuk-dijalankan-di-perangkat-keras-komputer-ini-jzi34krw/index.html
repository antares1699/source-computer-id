Saya mencoba menyerang kesalahan "stop: c000021a".

Dari utas stop: c000021a {Fatal System Error} :

Perbaikan untuk masalah ini, adalah tempat file Cable Select Jumper digunakan (kiri menyala oleh pabrik), pada Hard Drive dan di mana BIOS tidak mendukung file fitur.

Lepaskan jumper untuk Cable Select, dan tergantung pada pembuatan HDD Anda mungkin ingin mencoba Drive 0 (master) pengaturan jumper.

Ini memperbaiki Instalasi Windows 7 setelah menggunakan instal Windows 7 lingkungan untuk menghapus semua partisi dari instalasi XP yang ada, dan membuat format baru dan cepat NTFS .. Instalasi berhenti pada 99%, dan menyebabkan semua jenis Kesalahan Fatal dan Blue Screens mencoba menampilkannya boot.

Melepas Jumper Cable Select on HDD segera memperbaiki masalahnya.

Anda juga dapat mencoba ide-ide di artikel Microsoft ini (jika berlaku):
Cara memecahkan masalah kesalahan "STOP 0xC000021A"

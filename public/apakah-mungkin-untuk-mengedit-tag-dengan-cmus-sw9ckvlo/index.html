Apakah mungkin untuk mengedit tag dengan cmus?

Apakah mungkin untuk mengedit tag untuk .ogg file dengan cmus ?

4
Wuffers — Sumber

1 Jawaban

6
6

Tidak tetapi - mengutip dari daftar fitur - [itu] Can run external commands for the currently selected files (tag-editor for example).

Anda dapat mengikat misalnya:

bind common ^E run your-tag-editor {}

Jadi saat Anda menekan Ctrl + e dalam konteks common, cmus akan membuka your-tag-editor memperluas {} ke file yang sedang dipilih.

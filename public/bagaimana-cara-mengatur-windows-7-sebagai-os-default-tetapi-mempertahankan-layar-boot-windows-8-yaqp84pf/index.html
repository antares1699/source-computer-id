Bagaimana cara mengatur Windows 7 sebagai OS default tetapi mempertahankan layar Boot Windows 8?

Saya melakukan dual-boot antara Windows 7 dan Windows 8 pada workstation uji, dan biasanya melakukan booting ulang 3-4 kali per hari.

Jika saya menetapkan Windows 8 sebagai OS default, saya mendapatkan layar boot grafis Windows 8, yang mudah 'dilihat' selama proses boot, tetapi jika saya menetapkan Windows 7 sebagai OS default, saya hanya mendapatkan teks Windows 7- mode layar boot. Sementara saya kebanyakan menginginkan Windows 7 (saat ini), pada kesempatan saya restart untuk masuk ke Windows 8, saya sering 'ketinggalan' melihat boot hanya teks dan harus restart dua kali.

Apakah mungkin untuk (dan jika demikian, bagaimana saya) mengkonfigurasi ini sehingga Windows 7 adalah OS default, tetapi layar boot Windows 8 masih muncul?

Sunting: Asal kalian tahu, saya sudah mencoba menyetel Windows 7 sebagai default baik dari layar Boot Windows 8 itu sendiri, dan dari dalam Windows 8 - tidak satu pun dari ini memiliki efek yang diinginkan.

Saya khawatir itu tidak mungkin:

Selama file boot Windows 8 digunakan, menu boot yang Anda lihat bergantung pada setidaknya 3 hal:

  • Entri Windows default diatur di BCD: jika itu adalah Windows 7, Anda akan melihat menu tekstual.
  • Jika untuk Windows 8 memiliki bootmenupolicy disetel ke Standard, karena jika disetel ke Legacy atau bahkan tidak disetel, Anda akan melihat menu tekstual meskipun entri Windows 8 disetel menjadi yang default.
  • Jika entri boot manager displaybootmenu disetel ke Yes, menu tekstual akan digunakan terlalu terpisah dari entri default.

Jadi, Anda dapat memilih untuk memiliki menu tekstual dalam konfigurasi apa pun, tetapi tidak sebaliknya.

Saya agak mengerti mengapa itu: dengan Windows 7 Anda pertama kali disajikan dengan menu boot, dan kemudian semuanya mulai dimuat tergantung pada apa yang Anda pilih. Di Windows 8, menu boot tidak muncul sampai Anda telah memuat banyak hal, dan jika Anda kebetulan tidak ingin mem-boot entri default, mesin akan melakukan boot ulang untuk memuat apa yang Anda inginkan sesudahnya.

Anda bahkan dapat menduplikasi entri untuk Windows 8 namun Anda mengalami perilaku yang sama, memiliki 2 entri yang sama persis untuk memuat hasil Windows 8 di mesin reboot jika Anda tidak memilih yang default meskipun apa yang akan Anda boot adalah sebenarnya sama.

Jadi jika sesuatu selain Windows 8 diatur menjadi default di toko BCD, pramuat barang-barang Windows 8 sebelum pemilihan tidak masuk akal, maka Anda disajikan dengan menu tekstual dan kemudian hal-hal dimuat. Saya kira pramuat semacam itu tidak tersedia atau mungkin dilakukan dengan versi Windows lainnya.

Anda tidak dapat menggunakan Windows 7 sebagai entri boot default dan "metro" atau menu boot UI baru.

Alasannya adalah bahwa menu boot UI ditampilkan oleh kode dalam file .dll yang tidak tersedia di Windows 7.

Juga pengaturan kebijakan menu boot untuk entri pemuat Windows 7 (disetel sebagai default) memiliki efek samping yang aneh - menu booting dilewati dan Windows 7 di-boot secara langsung bahkan ketika ada beberapa entri menu boot.

Tidak ada dokumentasi tentang subjek dari Microsoft. Elemen BCD lainnya juga memiliki efek samping pada tampilan menu boot: DisplayBootMenu (untuk bootmgr) dan emssettings.

Lihat Kebijakan Menu Booting untuk penjelasan rinci.

Lihat juga deskripsi singkat tentang elemen boot manager dan item menu boot.

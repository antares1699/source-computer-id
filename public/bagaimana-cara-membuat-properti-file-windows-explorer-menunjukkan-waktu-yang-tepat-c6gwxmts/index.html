Mengapa properti file Windows Explorer tidak menunjukkan waktu yang tepat?

Saat saya membuka properti file, saya tidak ingin OS saya membulatkan nilai untuk saya

Perilaku ini disebabkan karena format data yang panjang disetel ke Day of the Week, bukan Day.

Bagaimana cara mengatasinya?

  • Buka Panel Kontrol> Wilayah dan Bahasa> Pengaturan Tambahan> Tanggal

  • Ubah format “Tanggal panjang” menjadi “hhh, dd MMMM, tttt” (hapus salah satu 'd').

Ini akan mengubahnya dari format "Hari Panjang" menjadi "Disingkat tanggal "format (yaitu" Kamis "menjadi" Kam ").

Rupanya si "Kencan panjang" (istilah saya) juga mengerti sesuatu tentang "Hari ini", tetapi "Tanggal singkat" (sekali lagi, terminologi saya) tidak, jadi bukannya "Hari ini" yang tertulis "Kam".

Karena tidak tahu ini adalah "Hari Ini", ia mencantumkan waktu penuh.

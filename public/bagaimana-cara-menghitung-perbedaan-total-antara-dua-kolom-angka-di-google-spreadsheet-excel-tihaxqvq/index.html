Bagaimana cara menghitung perbedaan total antara dua kolom angka di Google spreadsheet / Excel?

Biasanya saya hanya akan menghitung selisih setiap baris seperti ini: =B1-A1

Kemudian saya akan menjumlahkan nilai kolom seperti ini =SUM(C1:C10) dan mendapatkan nilai yang saya inginkan.

Saya tidak ingin kolom C. Apakah mungkin untuk segera menghitungnya?

Misalnya saya mencoba: =SUM((B1-A1):(B10-A10)) (yang tidak berfungsi)

Biasanya saya dapat menemukan jawabannya di suatu tempat di dokumentasi, tetapi kosakata Matematika-Bahasa Inggris saya agak tidak stabil dan dengan dokumentasi bahasa Jerman saya belum menemukan apa yang saya cari.

Saya tahu saya bisa memasukkan 10 nilai secara manual ke dalam penjumlahan. Tetapi ini tidak berskala dengan anggun, saya ingin dapat mengubah hanya beberapa nilai dalam rumus daripada menambahkan 1000.

Jika Anda ingin menghitung setiap nilai secara bergantian, evaluasi sebagai angka positif, KEMUDIAN cari tahu apa yang ditambahkan maka taruhan terbaik Anda adalah rumus array.
Ini dimasukkan bukan dengan menekan ENTER di akhir, tetapi CTRL + SHIFT + ENTER. Setelah selesai, Anda akan melihat sepasang kawat gigi di kedua sisinya ({ }).
Catatan - Anda harus menggunakan metode ini, Anda tidak dapat mengetikkan tanda kurung dan CTRL + SHIFT + ENTER dll

=sum ( abs(b1:b10 - a1:a10) )

Jelas, Anda dapat mengganti jumlah dengan rata-rata jika itu lebih sesuai dengan kebutuhan Anda.

Utas lain menyarankan SumProduct, jika salah satu sel mungkin kosong. Trik digunakan untuk satu argumen, dan yang lainnya adalah perbedaan yang diinginkan, misalnya: = SumProduct (- (b1: b10 <> ""), (b1: b10 - a1: a10)) di mana triknya menggunakan - untuk mengubah nilai logika / boolean menjadi 1 atau 0. Namun, Excel 2010 menampilkan kesalahan #VALUE! melawan argumen kedua itu. Sebuah alternatif: = (Jumlah (b1: b10) -SumProduct (- (b1: b10 <> ""), a1: a10)) / Hitung (b1: b10) Tapi itu akan menghasilkan # DIV / 0 !, jika semua sel dalam rentang B kosong.

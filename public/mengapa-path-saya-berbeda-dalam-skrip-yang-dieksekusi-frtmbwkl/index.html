Skrip dijalankan menggunakan /bin/zsh, yang bukan shell interaktif atau login dan tidak memuat file ini. Dari man zsh, tekankan milik saya:

Perintah pertama kali dibaca dari / etc / zshenv; ini tidak bisa diganti. Perilaku selanjutnya diubah oleh opsi RCS dan GLOBAL_RCS; yang pertama memengaruhi semua file startup, sedangkan yang kedua hanya memengaruhi file startup global (yang ditampilkan di sini dengan jalur yang dimulai dengan a /). Jika salah satu opsi tidak disetel, file startup berikutnya dari jenis yang sesuai tidak akan dibaca. File dalam $ ZDOTDIR juga dapat mengaktifkan kembali GLOBAL_RCS. RCS dan GLOBAL_RCS keduanya disetel secara default.

Perintah kemudian dibaca dari $ ZDOTDIR / .zshenv. Jika shell adalah shell login, perintah dibaca dari / etc / zprofile dan kemudian $ ZDOTDIR / .zprofile. Kemudian, jika shell bersifat interaktif, perintah dibaca dari / etc / zshrc dan kemudian $ ZDOTDIR / .zshrc . Terakhir, jika shell adalah shell login, / etc / zlogin dan $ ZDOTDIR / .zlogin akan dibaca.

Skrip mewarisi lingkungan tempat ia dipanggil, dan jika ini bukan shell (interaktif) lain, ia tidak akan berisi preferensi yang Anda setel pada .zshrc.

Anda dapat mengatur PATH di mana itu berlaku secara global (misalnya /etc/zshenv), mengaturnya secara eksplisit di skrip secara langsung, atau mengubah tajuk skrip shebang untuk menjalankan /bin/zsh -i sebagai gantinya, membuatnya memuat .zshrc (mengutip man zsh: Paksa shell menjadi interaktif. Masih memungkinkan untuk menentukan skrip yang akan dieksekusi. ).

Atau, cukup tentukan jalur lengkap ke program yang tidak ada di default PATH, mis. /home/pc/less.js/bin/lessc.

file startup zsh (file a.k.a. rc)

Nama file di bawah ini yang bukan path lengkap secara implisit diawali dengan "$ ZDOTDIR /", yang biasanya merupakan direktori home Anda.

Urutan file skrip startup sumber zsh adalah sebagai berikut.

/ etc / zshenv - Pertama, perintah dibaca dari sini; opsi tidak dapat menimpa ini.

.zshenv

/ etc / zprofile - shell login

.zprofile - shell login

/ etc / zshrc - shell interaktif

.zshrc - shell interaktif

/ etc / zlogin - shell login

.zlogin - shell login

Dua opsi zsh mempengaruhi apakah sumber zsh tertentu dari file di atas. ("RCS" adalah bentuk jamak dari "rc", dalam huruf besar.)

RCS dan GLOBAL_RCS zsh opsi disetel secara default.

  • RCS - mempengaruhi semua file startup
  • GLOBAL_RCS - hanya memengaruhi file startup global (nama jalur yang dimulai dengan “/”)

Jika file startup membatalkan salah satu opsi ini, zsh melompati file startup berikutnya dari jenis itu.

Jika file startup menyetel opsi GLOBAL_RCS, zsh mengambil file startup global berikutnya.

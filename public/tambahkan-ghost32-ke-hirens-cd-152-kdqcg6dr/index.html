Tambahkan Ghost32 ke Hiren’s CD 15.2

Hai, saya perlu menambahkan ghost32 ke Hiren's CD v15.2 Windows Mini XP.

aku menemukan ini tutorial .

Langkah saya:

  • Dengan HBCDCustomizer saya mengekstrak ISO

  • Buat folder C: \ Ghost

  • Salin Ghost32.exe dan uharc.exe ke folder C: \ Ghost

  • uharc buat file Ghost32.uha

  • Ghost32.uha perlu disalin ..CD\HBCD\WinTools\Files. (dari tutorial)

  • Tetapi folder HBCD berisi folder-folder ini:

  • Boot
  • Program
    • File

  • XP

  • Saya tidak tahu di folder mana saya perlu menyalin file Ghost32.uha atau apakah saya perlu menyalin Ghost32.exe atau file lain di suatu tempat.

  • Pertama saya copy hanya Ghost32.uha ke ..\CD\HBCD\Programs\Files. Hasilnya di Windows Mini Xp Ghost32 tidak ditemukan.

  • Usaha kedua. Folder Programs\Files berisi file 7z jadi saya coba salin ke folder ini

    • Ghost32.uha
    • Ghost32.7z

  • Ghost32 ditemukan tetapi tidak berfungsi.

Adakah saran bagaimana menambahkan Ghost32 ke Windows XP di Hiren's CD v15.2?

  1. Unduh Hirens.BootCD.15.2.zip dan ekstrak ke folder (untuk contoh, D: \ Hirens.BootCD.15.2 ).
  2. Lari D: \ Hirens.BootCD.15.2 \ HBCDCustomizer.exe , ikuti langkah-langkahnya IKLAN seperti yang ditunjukkan pada gambar dan tunggu hingga ekstraksi selesai.
  3. Unduh Hantu 11.5.1.2269 atau Hantu 12.0.0.4112 dan salin ke D: \ MyBootCD \ CD \ HBCD \ Program \ Files .
  4. Ikuti langkah-langkahnya E-F seperti yang tertera pada gambar dan tunggu sampai muncul gambar pembuatan selesai.
  5. Lari D: \ Hirens.BootCD.15.2 \ BurnToCD.cmd dan membakar file yang baru dibuat gambar.
  6. D: \ MyBootCD \ MyHBCD.iso ke CD kosong.

Tukar urutan teks menggunakan notepad ++

Saya tidak yakin bagaimana menjelaskan apa yang saya tanyakan hanya dalam judul.

Ok jadi saya memiliki daftar BESAR yang saya masukkan ke SQL DB dengan program kustom. Ini mengimpor item dari pastebin. Namun, saya benar-benar memberi jarak bagaimana info perlu dipesan.

Jadi, inilah contoh daftar kecil

M9; 15002
M9 Mag; 15101
FiveSeven; 15003
FiveSeven Mag; 15110
MP443; 15004
MP443 Mag; 15109

Apa yang saya ingin tahu apakah itu mungkin adalah meminta notepad ++ mengambil teks dan membuatnya seperti ini

15002; M9; 
15101; M9 Mag; 
15003; FiveSeven; 
15110; FiveSeven Mag; 
15004; MP443; 
15109; MP443 Mag;
Saya dapat secara manual jika perlu menghapus ekstra trailing; di akhir setiap baris.

Di Notepad ++ tekan CTRL + H

  • Temukan apa: (^.*); (.*?$)
  • Ganti dengan: \2; \1;
  • Memeriksa Ekspresi reguler pilihan

Before                                       After

Triknya adalah dengan menggunakan dua grup penangkapan melalui ( ) dan mengaksesnya dengan \1 dan \2. Tetapi dalam urutan terbalik

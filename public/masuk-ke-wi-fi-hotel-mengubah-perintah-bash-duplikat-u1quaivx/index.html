Masuk ke Wi-Fi hotel mengubah perintah bash? [duplikat]

Sejak mengklik pop-up "Saya setuju" saat masuk ke Wi-Fi hotel, prompt bash saya berisi semacam hash di dalamnya. Formatnya sekarang:

<strange hash>:<working dir> <username>

Apa yang sedang terjadi? Apakah ini berbahaya?

"Aneh hash" adalah nama komputer Anda saat ini, yang ditetapkan oleh server DHCP hotel, dan diterima oleh klien DHCP apa pun yang Anda gunakan. Jalankan perintah hostname untuk memeriksa apakah tebakan saya benar.

Untuk menghindari hal ini di masa mendatang:

  • Untuk dhcpcd , tambahkan nohook hostname ke file konfigurasinya. Temukan juga pengaturan option ..., dan pastikan tidak ada yang mencantumkan opsi host_name.

  • Untuk dhclient.dll , pastikan pengaturan request ... tidak mencantumkan opsi host-name.

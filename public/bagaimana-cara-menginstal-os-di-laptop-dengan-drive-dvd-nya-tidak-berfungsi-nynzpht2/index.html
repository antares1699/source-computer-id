Bagaimana cara menginstal OS di laptop dengan drive DVD-nya tidak berfungsi?

Saya memiliki Laptop Acer yang sangat tua dan tidak membaca CD lagi. Oleh karena itu, saya tidak dapat melakukan booting dari CD.

Saya membeli penulis DVD eksternal, tetapi tidak ada opsi di BIOS untuk boot dari itu.

Apakah ada cara agar saya dapat menginstal sistem operasi di laptop?

Saya memiliki model Acer Aspire 2004 yang menjalankan Windows XP.

Bisakah saya menghubungkan hard drive laptop ke desktop?

Anda dapat membuat image ISO dari CD menggunakan komputer lain. Salin itu ke hard drive di laptop Anda, lalu pasang menggunakan file Utilitas ISO . Ini kemudian akan bertindak seperti CD biasa.

Peringatan: Satu hal yang saya tidak yakin dan belum harus diuji, adalah apakah image ISO dapat diatur untuk boot dengan benar.

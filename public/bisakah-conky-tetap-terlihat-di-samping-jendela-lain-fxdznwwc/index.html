Bisakah Conky tetap terlihat di samping jendela lain?

Saya ingin conky terlihat setiap saat dan semua jendela menghormati batas conky. Mereka tidak boleh tumpang tindih dengan conky. Sekarang ketika saya membuka aplikasi baru itu tumpang tindih dengan conky dan saya harus menyeretnya secara manual ke kiri. Saya ingin aplikasi tidak pernah tampil terlalu conky. Bisakah itu dilakukan?

Tambahkan baris ini ke ~/.conkyrc Anda:

own_window yes
own_window_type panel
own_window_hints above

Menurut conky's dokumentasi (penekanan saya):

jika own_window adalah yes, Anda dapat menentukan tipe normal, desktop, dock, panel atau override (default: normal). Jendela desktop adalah jendela khusus yang tidak memiliki dekorasi jendela; selalu terlihat di desktop Anda; tidak muncul di pager atau taskbar; dan melekat di semua ruang kerja. Jendela panel menyediakan ruang di sepanjang tepi desktop, seperti panel dan bilah tugas, mencegah jendela yang dimaksimalkan agar tidak tumpang tindih. Tepi dipilih berdasarkan opsi perataan. Override windows tidak berada di bawah kendali window manager. Petunjuk diabaikan. Jenis jendela ini dapat berguna untuk situasi tertentu.

Saya memiliki pengaturan conky dengan cara ini, tetapi ini berjalan di bagian kanan layar. Saya menggunakan fbpanel untuk mempengaruhi tidak adanya tumpang tindih.

~ / .conkyrc:

own_window yes
own_window_type dock
own_window_transparent yes

maximum_width 160
gap_x 1755
gap_y 1

~ / .config / fbpanel / default:

Global {
    edge = right
    allign = center
    margin = 0
    widthtype = pixel
    width = 0
    height = 172
    transparent = true
    tintcolor = #000000
    alpha = 0
    setdocktype = true
    setpartialstrut = true
    autohide = false
    heightWhenHidden = 2
    roundcorners = true
    roundcornersradius = 7
    layer = below
    MaxElemHeight = 32
    setlayer = true
}

Bagaimana cara kerja folder khusus di Windows 7?

Saya melakukan sesuatu yang konyol dan secara tidak sengaja memindahkan folder khusus saya (Dokumen Saya, Musik Saya, dll.) Ke Desktop. Begitu saya menyadari kesalahan saya, saya segera memindahkannya kembali. Tapi sekarang Sistem saya bertindak tidak menentu.

Ini terus menerus membuat folder kosong baru di Desktop saya. Dan sekarang saya memiliki dua folder bernama "My Pictures" di direktori home saya. Salah satunya secara ajaib ditautkan ke yang ada di Desktop sementara di baris perintah yang satu benar-benar terlihat.

Jika saya menghapus folder yang tidak berguna di Desktop, maka penjelajah macet. Semuanya berantakan besar.

Tentu saja saya memeriksa HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\explorer\User Shell Folders dan semuanya terlihat baik-baik saja di sana.

Jadi di sini pertanyaannya: Apakah ada yang mendapat informasi tentang bagaimana Windows 7 menangani folder khusus yang mungkin membantu membersihkan kekacauan?

Saya sangat ahli dalam menggunakan baris perintah atau editor registri.

Untuk kepentingan mereka yang mungkin menemukan pertanyaan:

Di dan saya memperbaikinya dengan menggunakan kombinasi baris perintah dan penjelajah:

  1. Saya membiarkan Windows membuat ulang folder kosong baru di desktop
  2. Saya mengganti nama direktori lengkap di direktori home saya menggunakan TakeCommand
  3. Saya memindahkan folder dari desktop kembali ke tempatnya menggunakan Explorer
  4. Saya menghapus direktori kosong menggunakan TakeCommand .
  5. Saya mengganti nama direktori lengkap menggunakan TakeCommand .

Saya membiarkan pertanyaan terbuka karena masih belum ada jawaban tentang bagaimana penjelajah mengelola folder khusus tersebut dan nama alias yang muncul di penjelajah tetapi tidak memiliki representasi sama sekali dalam sistem file (Mengingatkan saya pada tautan bayangan OS / 2).

Banyak dari 'folder khusus' yang Anda lihat di explorer di Windows 7 (yang 'terhubung secara ajaib') adalah titik persimpangan. Buka prompt perintah dan masuk ke direktori pengguna Anda. ketik

DIR / A: L

dan Anda akan melihat semua poin reparse. Opsi atribut "L" adalah opsi baru yang belum saya lihat hingga windows 7 (saya melewatkan vista).

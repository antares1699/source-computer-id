Bagaimana cara mengatur font default untuk gvim di Windows?

Saya menggunakan gvim di Windows. Secara default menggunakan font Fixedsys, tetapi saya ingin menggunakan Consolas sebagai font default.

Saya mencoba mengaturnya menggunakan opsi menu; itu berhasil tetapi diatur ulang saat berikutnya saya memulai gvim.

Bagaimana cara mengatur font default di gvim? Dapatkah saya menambahkan sesuatu ke .vimrc?

7

saya akan menambahkannya ke _gvimrc, bagian yang relevan dari saya terlihat seperti ini:

if has("gui_kde")
    set guifont=Consolas/13/-1/5/50/0/0/0/0/0
elseif has("gui_gtk")
    set guifont=Consolas\ 13
elseif has("gui_running")
    if has("win32") || has("win64")
        set guifont=Consolas:h12
    else
        set guifont=-xos4-terminus-medium-r-normal--14-140-72-72-c-80-iso8859-1
    endif
endif

Hapus daftar Direktori yang panjang di prompt shell

Saya ingin menghapus daftar panjang direktori saat ini di prompt shell.

Terkadang mengganggu ketika Anda menggunakan jendela shell kecil dan itu menempati seluruh ruang di jendela

Saya lebih suka hanya direktori saat ini pada prompt.

root @ Kiran: ~ / Downloads / Director1 / Director2 / Director3 / Director4 # Saya lebih suka sesuatu seperti: root @ Kiran: .. Directory4 #

atau bahkan lebih baik,

root: Directory4 #

Terima kasih

Dengan Bash versi 4+ Anda dapat mengatur

export PROMPT_DIRTRIM=1

di .bashrc Anda. Perintah untuk pengguna non-root:

user@host:~/.../dir>

dan untuk root

host:.../dir #

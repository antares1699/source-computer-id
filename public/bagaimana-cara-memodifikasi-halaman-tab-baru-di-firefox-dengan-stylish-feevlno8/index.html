Bagaimana cara memodifikasi halaman Tab Baru di Firefox dengan Stylish?

Saya baru saja meningkatkan ke Firefox 58 setelah menggunakan Chrome untuk waktu yang sangat lama. Saya telah mengunduh Stylish dan membuat custom style berikut:

@namespace url(http://www.w3.org/1999/xhtml);
@-moz-document url-prefix("about:newtab") {
    body {
        background-color: #111111 !important;
    }
}

Ini disimpan dan diaktifkan, tetapi ketika saya membuka tab baru Bergaya tidak menampilkan gaya yang dipasang:

Namun, ketika saya beralih ke halaman lain, Stylish memang menunjukkan gaya yang saya sertakan di atas:

Bahkan jika saya menavigasi secara manual ke about:newtab, gayanya masih tidak aktif.

Lantas, bagaimana cara mengubah warna background halaman tab baru di Firefox dengan Stylish?

Solusi di Youtube

Jika pembuat vid akan menghapusnya, unggah kembali vid ini ke Bitchute.com (: D) atau baca instruksi versi teks:

  1. Ketik di bilah alamat about:profiles
  2. Bukalah Direktori ROOT dengan mengklik "tombol Buka direktori". Tidak Direktori Lokal.
  3. Buat direktori "chrome" di sini dengan perintah mkdir chrome di terminal atau melalui tombol klik kanan mouse Anda.
  4. Buka folder yang Anda buat dan buat file css di sini dengan menamainya "userContent.css"

  5. Tempel kode ini ke file "userContent.css" Anda:

    @ -moz-document url ("about: newtab") { tubuh { background-color: # 000000! penting; } }

  6. Restart browser Anda secara manual atau dengan mengklik tombol "Restart normal ..." pada alamat about:profiles.

Anda akan mendapatkan hasil yang Anda butuhkan! :)

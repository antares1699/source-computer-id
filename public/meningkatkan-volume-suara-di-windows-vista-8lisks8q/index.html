Meningkatkan volume suara di Windows Vista

Saya memiliki Windows XP dan saya menginstal Vista. Saya merasa suaranya sekarang lebih rendah daripada di XP.

Bagaimana cara meningkatkan volume di Windows Vista?

Catatan:
Saya memeriksa volume speaker, dan volume di taskbar dan semuanya secara maksimal.

Di Vista, setiap aplikasi memiliki volume sendiri serta volume master. Klik kanan pada ikon suara di bilah tugas saat aplikasi yang ingin Anda tingkatkan volumenya sedang berjalan. Kemudian klik Volume Mixer. Jika volume aplikasi tidak maksimal, coba naikkan juga.

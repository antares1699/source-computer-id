Windows 7 Easter Egg Menampilkan XP Alt-Tab Prompt

Untuk berjaga-jaga jika Anda belum menggunakan Windows 7, jendela Alt-Tab normal di Win7 besar, transparan, dan benar-benar merupakan peningkatan besar pada versi XP yang jelek itu.

Untuk memicu XP gaya yang lama, Anda perlu melakukan hal berikut:

Tahan tombol Alt Kiri (atau Kanan). Tekan dan lepaskan tombol Alt lainnya. Terus tahan tombol Alt pertama itu, lalu tekan Tab. Dan Voila! XP-style yang jelek akan muncul.

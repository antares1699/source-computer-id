“X tidak dapat dilakukan karena file terbuka di program lain” Mungkin untuk mencari tahu yang mana? [duplikat]

Kemungkinan Duplikat:
Bagaimana cara mengidentifikasi aplikasi apa yang menggunakan file tertentu?

Sesekali saya diberitahu oleh Windows bahwa saya tidak dapat melakukan tindakan tertentu (seperti menghapus file) karena "file terbuka di program lain" . Namun, itu tidak pernah memberi tahu saya yang program.

Apakah ada cara untuk mengetahuinya? Jika tidak, apakah ada cara saya dapat melakukan rm -f mirip unix pada file tersebut? Biasanya saya bisa me-reboot mesin saya, mencoba tindakan pertama setelah startup dan kemudian saya bisa melakukannya, tetapi tidak selalu, jadi akan berguna untuk mengetahui solusi alternatif ...

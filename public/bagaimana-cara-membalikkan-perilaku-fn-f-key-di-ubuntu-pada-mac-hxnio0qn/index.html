Saya memiliki konfigurasi yang sama. Secara default, di Ubuntu, fungsinya adalah default.

Agar Anda dapat menggunakan tombol fungsi sebagai F1,F2:

jalankan di bawah ini di terminal:

sudo su -c "echo -n 0x02 > /sys/module/hid_apple/parameters/fnmode"

PS: Saya sudah pakai jawaban dari sini:

Jawaban yang ditulis dengan sangat jelas ada di sini: https://help.ubuntu.com/community/AppleKeyboard#Change_Function_Key_behavior

Saya telah mengujinya, dan berfungsi seperti yang saya harapkan (Menekan F2 sekarang membuat saya F2, bukan "meningkatkan kecerahan layar")

Saya menemukan bahwa reboot bukanlah opsional - untuk melihat perubahannya, saya harus reboot.

  1. Jalankan perintah berikut untuk menambahkan baris konfigurasi ke file /etc/modprobe.d/hid_apple.conf membuatnya jika perlu:

$ echo options hid_apple fnmode = 2 | sudo tee -a /etc/modprobe.d/hid_apple.conf

  1. Pemicu menyalin konfigurasi ke dalam file boot initramfs.

$ sudo update-initramfs -u -k all

  1. Secara opsional, reboot

$ sudo reboot

Mac Anda mungkin memiliki yang serupa Buka Firmware pilihan. Saat mem-boot, tahan Perintah , Pilihan , 0 dan F kunci secara bersamaan. Quora memiliki beberapa info lebih lanjut tentang opsinya.

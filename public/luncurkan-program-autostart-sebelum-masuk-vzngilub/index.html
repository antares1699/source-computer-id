Luncurkan program AutoStart sebelum masuk

Ketika saya menyalakan komputer saya, saya biasanya membiarkannya boot selama beberapa menit sebelum saya kembali. Namun, ketika saya kembali, saya harus log in dan kemudian menunggu semua program start up saya dimuat. Apakah ada cara agar program tersebut dimuat di layar masuk (saya hanya memiliki satu akun)? Saya menjalankan Win7 x64.

Satu-satunya cara untuk memuat program start up Anda adalah masuk dengan akun pengguna Anda. Jika Anda ingin masuk secara otomatis saat komputer Anda mulai, ada instruksi di sini:

Cara mengaktifkan autologon di Windows 7

Jika Anda khawatir dengan keamanan (saya akan), Anda dapat menambahkan program startup yang akan mengunci layar:

Perintah baris perintah (cmd) untuk mengunci mesin windows

Dengan begitu, komputer Anda akan mulai, akun pengguna Anda akan masuk, dan layar akan terkunci. Anda masih harus memasukkan kata sandi Anda ketika Anda kembali ke komputer Anda, tetapi item start up Anda akan dimuat.

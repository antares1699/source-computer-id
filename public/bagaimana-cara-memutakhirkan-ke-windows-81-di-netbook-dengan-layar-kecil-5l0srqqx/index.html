Bagaimana cara memutakhirkan ke Windows 8.1 di netbook dengan layar kecil?

Saya mencoba mengupdate netbook ke Windows 8.1 dari Windows 8. Sayangnya, mengunjungi Windows Store di Internet Explorer dan mengklik link "Dapatkan update gratis sekarang" menyebabkan error berikut:

Bagaimana cara mendapatkan pembaruan gratis jika saya bahkan tidak dapat membuka Windows Store?

4

1

Anda dapat mengatur laptop Anda untuk menggunakan EnableDynamicScaling dengan membuka regedit dan mengubah EnableDynamicScaling dari 0 ke 1.

Berita buruk: Microsoft menjadi licik dan sekarang telah menonaktifkan fungsi itu di Windows 8.1. Setelah Anda mengatur skala ke 1 dan mengubah resolusi Anda menjadi 1024x768 dan menginstal Windows 8.1, Anda tidak akan lagi memiliki 1024x768 atau DynamicScaling untuk netbook dengan resolusi maksimal 1024x600. Setelah Anda meningkatkan, Anda terjebak.

Di regedit, buka Edit - Temukan, dan masukkan Display1_DownScalingSupported. Pada semua kejadian, ubah nilainya menjadi 1. F3 menemukan kejadian berikutnya. Setelah reboot resolusi lain harus tersedia.

Saya menanyakan Microsoft ini awal minggu ini karena saya mengharapkan pertanyaan seperti itu dan jawabannya adalah keberuntungan Anda. Satu-satunya cara untuk memutakhirkan adalah dengan mendapatkan MSDN / TEchnet Windows 8.1 ISO, pasang di dalam Windows 8, jalankan setup.exe dari DVD yang terpasang, masukkan kunci Win8.1 generik, pilih Tingkatkan, instal 8.1 dan masukkan kembali Kunci Anda di Windows 8.1 dan aktifkan.

Layar Anda berukuran 1024x600, yang tidak memenuhi persyaratan minimum Windows Store, yaitu 1024x768.

Jika perangkat keras Anda mengizinkannya, sambungkan sementara monitor eksternal dengan resolusi minimal 1024x768. Anda kemudian dapat menggunakan Store untuk mengunduh pemutakhiran Windows.

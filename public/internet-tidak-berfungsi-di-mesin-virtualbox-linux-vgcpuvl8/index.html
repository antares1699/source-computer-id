Internet tidak berfungsi di mesin VirtualBox Linux

Saya menginstal VirtualBox pada PC Windows 7 dan menginstal Xubuntu (Ubuntu dengan XFCE) di atasnya. Sekarang di Windows, saya terhubung ke internet melalui LAN yang secara otomatis mengautentikasi untuk terhubung ke jaringan (karena ini adalah komputer kerja saya). Saya juga telah menginstal tambahan tamu di mesin virtual ini (Xubuntu). Saya mencoba NAT dan bridging untuk mengatur koneksi internet tetapi saya masih tidak dapat terhubung ke internet.

Untuk NAT: Browser tidak mengatakan situs web tidak ditemukan tetapi terus mencoba memuatnya selamanya dan monitor status jaringan (yang disediakan oleh VirtualBox) tidak menunjukkan lampu hijau (tidak ada paket masuk).

Untuk Menjembatani: Sama seperti untuk NAT tetapi monitor status jaringan menunjukkan paket masuk dan paket keluar. Tapi internet masih belum berfungsi.

Saya mencoba banyak hal berbeda tetapi tidak ada yang berhasil.

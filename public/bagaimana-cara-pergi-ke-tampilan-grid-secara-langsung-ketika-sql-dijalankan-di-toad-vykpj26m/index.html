Saya memiliki masalah ini yang sangat mengganggu.

Ketika saya menjalankan kueri di TOAD (versi 11.5.1.2), itu menunjukkan hasilnya dalam Keluaran Skrip tab, dan kemudian dalam anaknya Keluaran tab.

F5 adalah Jalankan sebagai Pernyataan - Output Mentah F9 adalah Execute as Script - Grid Output (Alternate: CTRL-ENTER)

Saat menjalankan sebagai Pernyataan dan beralih ke kisi, Anda akan berada di tab Output Skrip yang memiliki tindakan yang tersedia (salin / pilih semua, dll). Saat menjalankan sebagai Script Anda memiliki tampilan Data Grid Anda hanya akan mengekspor tindakan. Anda dapat beralih tampilan di salah satu keluaran.

Anda dapat memetakan ulang tombol fungsi jika Anda terbiasa dengan F5.

EDIT: Versi saya berdasarkan tanggapan saya adalah untuk Toad untuk Oracle 11.6

Tombol yang mengirimkan output langsung ke grid terletak di toolbar Execute dan terlihat seperti ini:

Anda dapat mengaturnya agar dieksekusi melalui F5 dengan melakukan hal berikut:

  • Pilih View-> ToadOptions
  • Gulir sisi kiri dan di bawah Bilah Alat / Menu pilih Pintasan
  • Klik sisi kanan pada 'Execute Statement' (F9 ditampilkan sebagai pintasan pada saya)
  • Tekan F5
  • Klik 'Execute as Script' (F5 masih ditampilkan sebagai pintasan di sini)
  • Tekan F9

atom: secara otomatis menghapus spasi

Saya baru-baru ini mencoba atom dan ada satu hal yang menentangnya: ia mencoba melakukan pintar hal dan menghapus semua spasi tambahan saat saya menyimpan file - tanpa memberi tahu saya. Beberapa editor melakukan ini tetapi ini buruk ketika harus mengubah kendali. Tidak yakin mengapa programmer menjadikan ini perilaku default dan saya tidak dapat menemukan opsi untuk menonaktifkannya.

Apakah ada cara untuk menonaktifkan ini di Atom?

4
lang2 — Sumber
  • Sebagai catatan: Anda tidak ingin kebisingan komit spasi putih ini muncul setiap hari di komit Anda. Jika Anda bekerja dalam tim, sebaiknya 1) hapus semua spasi kosong dalam satu gerakan 2) biarkan semua orang mengonfigurasi editor mereka untuk menghapus spasi kosong yang tertinggal secara otomatis. Dan kita akan memiliki kedamaian.
    — Niels Bom
  • Tentu saja semua spasi tambahan harus hilang dalam kode. Tapi menurutku itu bukan pekerjaan editor. Lagipula tidak secara default - tidak ada yang menyentuh kode saya tanpa melalui saya.
    — lang2
  • Sepertinya tidak semua orang setuju dengan sentimen terakhir Anda: golang.org/cmd/gofmt :-)
    — Niels Bom
  • Tidak masalah untuk memiliki alat pemformatan otomatis yang berfungsi dengan baik. Dan saya suka cara kode semua orang terlihat sama. Namun tidak secara default tanpa memberitahu. Mungkin saya hanya kuno. :)
    — lang2

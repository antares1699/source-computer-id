Membuat keluaran ls seperti dir / b / s

Bagaimana cara mengubah output ls di Linux sehingga terlihat seperti output dari perintah dir /b /s di Windows?

dir /b /s keluaran sebagai berikut:

C:\MinGW>dir /s /b 
C:\MinGW\COPYING 
C:\MinGW\COPYING.LIB
C:\MinGW\doc 
C:\MinGW\include

Ini akan menghasilkan keluaran yang sama dengan dir /s/b

ls -R mendaftar semua file dan subdirektori secara rekursif. Dengan ls -R1 Anda mendapatkan hal yang sama, tetapi hanya satu file per baris.

Tidak ada yang membuat output terlihat persis seperti dir /b /s, tetapi harus cukup dekat.

Itu tergantung, seberapa sulit persyaratannya untuk mendapatkan "ekuivalensi" ke dir /b /s ... Yang ini cukup mendekati (mereka tidak akan memiliki garis miring terbalik sebagai pemisah direktori, dan urutan keluaran akan sedikit berbeda [dir /b /s outputs 'sibling' direktori terlebih dahulu, sebelum menyelami masing-masing untuk menampilkan kontennya]). Yang kedua hanyalah fallback jika versi find Anda tidak default ke tindakan -print (kebanyakan melakukannya):

find $(pwd)
find $(pwd) -print

MEMPERBARUI: Saya memiliki kesalahan ketik pada perintah di atas awalnya. Saya mengetik tanda kurung kurawal "{}" daripada yang bulat "()" sebagaimana mestinya. Terimakasih untuk grawity untuk melihat ini.

Dapatkah saya mengganti input monitor dengan hotkey? [Tutup]

Saat ini saya memiliki pengaturan di mana saya menggunakan tiga layar di dua monitor ganda, mesin Windows 7. Salah satu monitor (ViewSonic VG2030WM) digunakan bersama di kedua mesin melalui input ganda. Saya menggunakan Synergy untuk berbagi keyboard dan mouse di antara mesin, tetapi saya harus mengganti monitor secara manual ketika saya beralih antar mesin.

Apakah ada utilitas / alat untuk mengaktifkan input pada monitor bersama?

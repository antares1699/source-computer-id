Favorit menghilang di windows 8

Tautan favorit di sisi kiri semua folder telah hilang di komputer saya. Biasanya saya memiliki tautan ke Unduhan, Desktop, dan lainnya. Tapi suatu hari, mereka semua pergi. Dan saya tidak dapat menemukan cara untuk mendapatkannya kembali. Mengapa tidak berhasil dengan mengklik kanan padanya dan tekan "Pulihkan tautan favorit" (diterjemahkan dari bahasa swedia, tidak tahu nama uk / us)

Saya pikir mereka mungkin telah menghilang karena saya menghapus folder bernama "Tautan saya" di folder "Alfred". Tapi ini aneh karena saya belum menyentuh folder "favorit".

Saya telah mencoba banyak untuk membuatnya berfungsi tetapi tidak berhasil.

Biasanya, Pulihkan tautan favorit Sebaiknya telah bekerja; untuk itulah itu ada. Saya baru saja menghapus Tautan folder di Win7, kemudian menggunakan opsi yang sama untuk membuat ulang mereka jadi Favorit telah dikembalikan ke default.

Pada titik ini, IMO Anda memiliki dua opsi:

  1. Buat profil pengguna sementara baru (atau gunakan profil terpisah yang ada) dan salin pekerjaan Tautan folder ke profil Anda. Biasanya, item kustom yang Anda tambahkan sebelumnya tidak akan dikembalikan.

  2. Sejak Tautan hanyalah folder biasa dengan sistem + tersembunyi Dekstop.ini yang bertanggung jawab atas keajaiban, Anda cukup membuat folder baru dengan nama itu dan menyalin / membuat ulang Dekstop.ini dari contoh folder yang berfungsi (dari profil berbeda di PC Anda atau PC lain). Seperti inilah tampilannya di Win7, tidak yakin apakah itu sama di WIn8 karena saya tidak memiliki akses ke sana sekarang:

    [.ShellClassInfo]
LocalizedResourceName=@%SystemRoot%\system32\shell32.dll,-21810
[LocalizedFileNames]
RecentPlaces.lnk=@shell32.dll,-37217
Desktop.lnk=@shell32.dll,-21769
Downloads.lnk=@shell32.dll,-21798

Setelah melakukan salah satu dari ini Pulihkan tautan favorit seharusnya berfungsi (jika tidak, reboot dan coba lagi).

Ini berhasil untuk saya:

  1. Buka file explorer dan klik This PC.
  2. Pergi ke tab View dan pilih dropdown Options.
  3. Pilih Ubah folder dan opsi pencarian dari dropdown.
  4. Klik pada kotak centang untuk Tampilkan Favorit di panel Navigasi.

Saya mengalami masalah yang sama, jadi saya cukup mengikuti langkah-langkah ini untuk mendapatkan kembali tautan favorit saya:

  1. Pertama, Anda membuka file explorer

  2. Kemudian Anda memilih folder apa saja (seperti musik, dokumen, dll.)

  3. Setelah itu, Anda menyeret folder ke favorit hingga muncul pesan yang menanyakan apakah Anda ingin membuat tautan ke favorit

  4. nanti cari di folder yang baru saja Anda seret, dan harus ada folder bernama "tautan"

  5. Setelah ini, Anda dapat membuka favorit di sisi kiri jendela dan menggunakan klik kanan dan pilih pulihkan favorit

  6. Terakhir tutup jendela dan buka file explorer lagi.

Ini harus melakukan trik, semoga berfungsi untuk Anda

Menyalin teks dari Vim ke sesuatu yang lain tanpa dukungan clipboard dari Vim

Saya sudah mencoba menyalin teks dari jendela Vim ke suatu tempat tetapi tidak berhasil. Keputusan yang saya temukan dengan Google tidak membantu juga. Misalnya, menggunakan register "* dan" +.

Versi perintah vim adalah (Ubuntu 12.04. Vim 7.3):

vim --version | grep clipboard
-clientserver -clipboard +cmdline_compl +cmdline_hist +cmdline_info +comments 
 -xterm_clipboard -xterm_save

Bisakah Anda membantu saya menyelesaikannya? Haruskah saya menggunakan versi vim lain atau adakah metode untuk menyalin teks dari Vim ke yang lain?

Anda mungkin menggunakan vim-tiny.

Melakukan

$ vim --version | grep version

dan laporkan kembali hasilnya.

Jika Anda menggunakan "tiny", Anda harus menginstal versi yang sesuai.

Mencoba

$ sudo apt-get install vim-nox

dan / atau

$ sudo apt-get install vim-gnome

Lebih baik jika Anda hanya menginstal versi vim yang lebih lengkap dengan semua hal yang diperlukan dikompilasi. Tetapi jika Anda tidak dapat melakukan ini karena alasan tertentu maka Anda memiliki setidaknya dua opsi: buka file di sesuatu yang lain (seperti gedit) dan salin dari sana atau gunakan xclip atau xsel untuk meletakkan teks yang diperlukan ke clipboard. Contoh penggunaan:

:'<,'>w !xclip -i

untuk mengirim rentang baris yang dipilih secara visual ke clipboard (secara default: buffer primer / mouse disebut * di vim, gunakan argumen tambahan -sel clip untuk mengubahnya ke clipboard yang disebut +) dan

:call system('xclip -i', @")

untuk mengirim isi register default (") ke clipboard.

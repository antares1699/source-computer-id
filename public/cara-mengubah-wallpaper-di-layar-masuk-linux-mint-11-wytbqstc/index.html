Cara mengubah wallpaper di layar masuk linux mint 11

Saya mengalami sedikit masalah dalam menemukan cara mengubah wallpaper untuk layar masuk Linux Mint 11.

Apakah kalian tahu bagaimana melakukan ini?

Linux Mint, saya tahu berbasis Ubuntu, jadi saya akan memberikan jawaban Ubuntu untuk pertanyaan itu.

Di bawah menu Sistem (yang saya tahu, Mint tidak memiliki tetapi seharusnya memilikinya sebagai sub menu, jika tidak, cari sisa dari apa yang saya katakan di sini), lihat submenu Administrasi, lalu di bawah Masuk Layar. Di bawah itu, saya yakin Anda bisa mengubahnya. Jika tidak, Anda harus mencari di repo untuk sesuatu yang disebut 'Startup Manager' dan menginstalnya, yang memungkinkan Anda untuk mengubah layar login Anda.

Jika gagal .... solusi ini akan berhasil, ditautkan di sini . Saya tidak tahu solusi apa pun di luar ini ... Penjelasan panjang singkatnya: Ubah tema, ubah tema, atau Anda dapat mengunduh tema yang sesuai dengan Anda ... semoga ini membantu

Sebagai alternatif, jika Anda tidak ingin menginstal paket tweak ubuntu, Anda dapat mengubah latar belakang login dari baris perintah seperti ini:

Pertama-tama ubah latar belakang default, karena gdm telah dikonfigurasi untuk menggunakannya sebagai latar belakang login.

$ cd /usr/share/backgrounds/linuxmint
$ sudo ln -sf some-other-background.jpg default_background.jpg

Kemudian kosongkan cache gdm untuk memaksanya membuat ulang gambar latar belakangnya.

$ sudo rm /var/lib/gdm/.cache/wallpaper/*

Dan biola! Lain kali Anda harus masuk atau membuka kunci layar, gdm akan menampilkan latar belakang baru.

Bagaimana cara meletakkan kode HTML di Outlook 2010?

Bagaimana cara meletakkan kode HTML di Outlook 2010? Kodenya adalah:

<p style="color:red;">Hello</p>

Bagaimana saya bisa memasukkannya ke dalam pesan menggunakan Outlook 2010?

Mungkin perlu mencari klien / editor email yang berbeda jika Anda ingin mengirim tag HTML di email Anda: http://office.microsoft.com/en-us/outlook-help/i-can-t-use-html-tags-in-html-e-mail-messages-HA010199144.aspx

Microsoft Office Outlook 2007 menggunakan file editor pesan email berdasarkan Microsoft Office Word 2007. Kantor Outlook 2007 menawarkan WYSIWYG (apa Anda lihat adalah apa yang Anda dapatkan) pengeditan lingkungan Hidup. Pesan bahwa Anda tulis diubah ke format HTML untuk dikirim.

Office Outlook 2007 tidak dirancang bagi Anda untuk langsung masuk atau melihat HTML tag saat membuat pesan email.

Anda perlu menginstal Editor Kode HTML untuk Outlook, yang merupakan add-in Outlook untuk memungkinkan Anda mengedit HTML asli langsung di Outlook.

http://outlookhtmleditor.codeplex.com/

Anda dapat mengunduh penginstal di tab unduhan. Ini akan memungkinkan Anda untuk mengimpor file HTML atau mengedit kode sumber HTML dari email Anda.

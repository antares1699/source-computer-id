Bagaimana cara menambahkan kolom deskripsi ke Tabel Pivot

Saya membuat tabel pivot dari daftar item yang perlu dipesan. Setiap Item memiliki nomor model dan deskripsi yang tetap konsisten. Data disalin dari sheet lain dan diformat seperti ini:

|Model Number     |Description      |
=====================================
|                 |                 |
|                 |                 |
|                 |                 |
|1234-5678-90     |Model A          |
|                 |                 |
|1234-5555-90     |Model B          |
|1234-5555-90     |Model B          |
|1234-5678-90     |Model A          |
|1313-5678-90     |Model D          |
|                 |                 |
|                 |                 |
|1234-5678-90     |Model A          |

Hasil akhir yang saya coba capai adalah tabel pivot yang menyerupai ini

|Model Number     |Description      |Quantity         |
=======================================================
|1234-5678-90     |Model A          |3                |
|1234-5555-90     |Model B          |2                |
|1313-5678-90     |Model D          |1                |

Tetapi saya tidak dapat menemukan cara untuk melakukan ini tanpa membuat tabel bertingkat.

4

Dengan asumsi gambar terlampir cocok untuk format Anda, lakukan hal berikut:

  1. Ubah data Anda menjadi 2 kolom (dengan nama kolom di atas).
  2. Buat tabel pivot yang berisi 2 kolom data Anda.
  3. Seret nama bidang data ke bagian label baris.
  4. Klik kanan pada tabel pivot dan pilih Opsi Tabel Pivot , pilih Tampilan tab, pilih kotak centang Tabel Pivot Klasik tata letak, klik OK
  5. Klik kanan pada kolom pertama dan hapus pilihan Nomor Model Subtotal (atau apa pun nama kolom pertama Anda

Saya tidak memilih nilai kosong di daftar kolom pertama untuk membersihkan tabel

Bagikan

Tidak diragukan lagi solusi tata letak tabel Pivot Klasik berfungsi, saya juga menggunakannya sampai saya menemukan solusi ini yang mempertahankan gaya tabel pivot baru, yang merupakan aspek yang sangat penting untuk visualisasi data.

  1. Buat tabel pivot dengan Nomor Model diikuti dengan Deskripsi di Area Baris , PT ingin

  1. Di tab desain, ubah Tata Letak Laporan menjadi Tampilkan dalam Bentuk Tabular seperti di bawah ini

  1. Di tab desain lagi, ubah subtotal menjadi tidak memperlihatkan subtotal

** 4. hasil akhir seperti yang diinginkan adalah **

Keuntungan utama dari ini dibandingkan PT klasik adalah mempertahankan estetika desain Excel 2016, terlebih lagi Anda dapat menambahkan lebih banyak kolom untuk nilai unik seperti Item Unit Cost, Item Unit Price dan yang lainnya.

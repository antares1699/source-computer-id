Layar kunci "sorotan" Win10 macet pada satu gambar, sekarang kosong dan tidak ada gambar yang ditampilkan

Saya memiliki gambar yang sama di laptop kerja Win10 Enterprise saya selamanya (versi 1909, Build 18363.657). Saya muak dengan itu, jadi saya meneliti tentang di mana gambar-gambar ini berada, kemudian melanjutkan untuk menghapus semua gambar dan data yang ditemukan. Ini melakukan hapus gambar yang macet ... tapi sekarang saya tidak punya apa-apa untuk layar kunci, yang lebih buruk! Sekarang, apa pun yang saya pilih (Gambar, Slideshow, atau Windows Spotlight) tidak ada gambar yang ditampilkan di layar kunci. Mungkin perlu dicatat bahwa info "Spotlight" tidak masih bisa ditampilkan. Sebagai contoh, " Kota pegunungan kecil apa ini? "

Saya telah berkecimpung di bidang TI selama satu dekade dan lebih dari setengahnya adalah Dukungan Desktop dan / atau pekerjaan Win SysAdmin. Intinya adalah, saya sangat lebih dari mampu, tetapi masalah ini masih belum saya miliki. Saya sudah mencoba semua yang saya temukan secara online, terakhir dengan posting SU ini , tapi tidak ada yang berhasil.

Pembaruan 1: Saya menjalankan chkdsk /f /r dan reboot. Ia mengatakan itu "Memperbaiki (C :) Tahap 4: 10% ..." jadi mudah-mudahan ini menyelesaikan masalah.

Pembaruan 2: Negatif, chkdsk tidak memperbaiki masalah. Saya juga menjalankan DISM /Online /Cleanup-Image /RestoreHealth - selesai dan tidak menemukan apa pun.

Pembaruan 3: Gambar "pratinjau" yang ditampilkan di Setelan -> Personalisasi -> Layar kunci -> Pratinjau: latar belakang (Windows Spotlight) aku s berubah! Sesuatu antara pengaturan dan apa yang membuat gambar itu sebenarnya muncul di layar kunci rusak.

Pembaruan 4: Saya berharap bahwa kunci registri ini dapat memperbaiki masalah: HKLM\SOFTWARE\Policies\Microsoft\Windows\System\DisableLogonBackgroundImage -> kunci DWORD dengan nilai "0" - namun ternyata tidak. Perlu dicatat, kuncinya adalah tidak hadir di mesin, jadi saya mencoba membuatnya dan mem-boot ulang secara manual, tetapi tidak ada perubahan. Mungkin membalik pengaturan yang setara di Kebijakan Grup lokal mungkin berhasil ...?

Pembaruan terakhir - TETAP ..... sepertinya sudah diperbaiki untuk sebagian besar hari ini, tapi tidak. Masalahnya telah kembali!

C:\WINDOWS\system32>SFC /Scannow

Beginning system scan.  This process will take some time.

Beginning verification phase of system scan.
Verification 100% complete.

Windows Resource Protection found corrupt files and successfully repaired them.
For online repairs, details are included in the CBS log file located at
windir\Logs\CBS\CBS.log. For example C:\Windows\Logs\CBS\CBS.log. For offline
repairs, details are included in the log file provided by the /OFFLOGFILE flag.

Perbarui 5: - Saya telah menemukan masalahnya! Izin pada folder ini rusak / salah: C:\ProgramData\Microsoft\Windows\SystemData Menambahkan kembali SISTEM dengan izin penuh tampaknya telah memperbaiki masalah! :)

Untuk memeriksa kerusakan Windows terlebih dahulu, lakukan hal berikut:

  • Di Pengaturan> Privasi> Aplikasi latar belakang , Microsoft Edge harus Diaktifkan
  • Lari sfc / scannow
  • Buat akun pengguna baru dan periksa apakah ada masalah di dalamnya. Jika tidak, ada korupsi profil. Anda perlu mentransfer semua file barang pribadi ke akun baru dan bekerja dengannya, seperti menemukan penyebabnya korupsi seperti itu sangat keras.

Jika tidak ditemukan kerusakan, daftar di bawah ini berisi perbaikan yang diketahui untuk Spotlight. Jika tidak ada yang berfungsi untuk Anda, maka Windows rusak dan akan membutuhkan Perbaiki Penginstalan dengan Peningkatan Di Tempat .

Metode 1

  1. Di Mulai> Personalisasi> Layar Kunci , ubah Background menjadi Picture, tapi jangan keluar dari Pengaturan
  2. Hapus semua yang ada di dalam folder %LocalAppData%\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets (tetapi bukan folder itu sendiri)
  3. Buka di Explorer folder %LocalAppData%\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\Settings
  4. Di dalam folder, ganti nama file settings.dat dan roaming.lock hingga settings.dat.bak dan roaming.lock.bak
  5. Mulai ulang
  6. Di Mulai> Personalisasi> Layar Kunci , ubah Background kembali ke Windows Spotlight

Metode 2

Ini adalah versi yang lebih kuat dari metode pertama.

  1. Di Mulai> Personalisasi> Layar Kunci , ubah Background menjadi Picture, tapi jangan keluar dari Pengaturan
  2. Hapus semuanya dari dalam folder %LocalAppData%\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy (ambil cadangan)
  3. Dalam pengaturan Lock Screen, ubah Background kembali ke Windows Spotlight
  4. Mulai ulang

Metode 3: Menggunakan PowerShell

  1. Di Mulai> Personalisasi> Layar Kunci , ubah Background menjadi Picture

  2. Jalankan PowerShell sebagai administrator

  3. Ketik perintah berikut:

    Get-AppxPackage -allusers *ContentDeliveryManager* | foreach {Add-AppxPackage "$($_.InstallLocation)\appxmanifest.xml" -DisableDevelopmentMode -register }

  4. Mulai ulang

  5. Di Mulai> Personalisasi> Layar Kunci , ubah Background kembali ke Windows Spotlight

Saya telah menemukan masalahnya! Izin pada folder ini rusak / salah (ini adalah kesalahan saya karena di sinilah saya menghapus gambar yang macet): C:\ProgramData\Microsoft\Windows\SystemData

Menghapus secara manual semua sub-folder yang rusak di bawah folder SystemData, lalu menambahkannya kembali SISTEM dengan izin penuh di folder SystemData. Ini memungkinkan gambar layar kunci (gambar tunggal, tayangan slide, atau Spotlight) berfungsi kembali secara normal. Jika Anda mengalami masalah ini, Anda mungkin juga perlu melakukan langkah-langkah @HarryMC yang tercantum di atas.

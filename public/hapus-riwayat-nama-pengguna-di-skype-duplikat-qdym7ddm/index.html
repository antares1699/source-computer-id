Hapus riwayat nama pengguna di Skype [duplikat]

Kemungkinan Duplikat:
Hapus fungsi saran nama pengguna di Skype 4.2, Windows Vista?

Saya mencari cara untuk menghapus riwayat nama masuk di Skype (Windows 7) saya.

Saya tidak tahu di mana itu disembunyikan. Bahkan mencoba menghapus semua Skype dan menginstal ulang lagi tetapi masih menunjukkan riwayat nama yang masuk sebelumnya.

Bagaimana cara menghapusnya dari login drop-down yang disarankan?

Selangkah demi selangkah:

  • Klik menu Start
  • Klik Run (atau cukup tekan WinKey + R )
  • Ketik %AppData%/Skype
  • tekan Memasukkan di keyboard Anda
  • Di folder yang muncul, cari folder dengan nama pengguna yang ingin Anda hapus
  • Hapus folder itu.

Itu harus menghapus nama pengguna dari bidang login Anda.

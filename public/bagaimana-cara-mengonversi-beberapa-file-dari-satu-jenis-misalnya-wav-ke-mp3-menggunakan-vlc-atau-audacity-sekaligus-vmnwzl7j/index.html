Bagaimana cara mengonversi beberapa file dari satu jenis (misalnya wav ke mp3) menggunakan VLC atau Audacity sekaligus?

Saya memiliki 10 file wav. Saya ingin mengubah semuanya menjadi file mp3. Saya berhasil melakukannya untuk satu file (seperti yang dijelaskan dalam Bagaimana cara mengonversi media menggunakan VLC? ), tetapi saya tidak ingin melakukan ini untuk setiap file. Apakah ada cara untuk mengonversi 10 file sekaligus? Saya menggunakan windows 7.

Saya mengikuti saran nixda dan menggunakan skrip batch. Saya menemukan cara memprogram batch untuk windows 7. Ini adalah solusi saya:

@echo off
for /f "delims=|" %%f in ('dir /b C:\Any_Directory\*.wav') do (
echo converting      %%f
CALL "C:\Program Files\VideoLAN\VLC\vlc.exe" --intf dummy --dummy-quiet "%%f" --sout="#transcode{acodec=mp3,ab=256,vcodec=dummy}:standard{access=file,mux=raw,dst=MP3_Files/"%%f"}"  vlc://quit
move "%%f" WAV_converted/"%%f"
)

cd MP3_Files
ren *.wav *.mp3
cd ..

echo .
echo .
echo .
echo conversion finished
pause

Bagaimana program bekerja:

  • Semua file dengan ekstensi .wav dalam folder tertentu "C: \ Any_Directory \" diubah menjadi mp3 (nama file tidak boleh mengandung spasi)
  • File mp3 disimpan di folder "MP3_Files" (yang pasti sudah ada)
  • File wav didorong ke folder "WAV_converted" (yang pasti sudah ada)
  • "ab = 256" mengatur kecepatan bit. Untuk mengonversi dengan mis. bit rate 128, ganti dengan "ab = 128".
  • File batch harus dijalankan dari folder yang sama dengan file yang ingin Anda konversi (seperti yang disebutkan oleh Kevin Heidt)

Petunjuk - Hapus "--intf dummy --dummy-quiet" dari skrip jika Anda ingin melihat GUI VLC saat mengonversi. Bisa berguna untuk debugging.

Memperbarui Masih ada satu bug: Jika nama file wav berisi titik atau koma, namanya terpotong (dan tidak ada ekstensi yang ditambahkan!). File akan dikonversi meskipun demikian. Diuji dengan Versi 1.1.11. Versi yang lebih baru mungkin tidak berfungsi!

Menambahkan tanda kutip: --sout = "...". Berfungsi untuk Versi yang lebih baru juga. Diuji dengan Versi 2.1.3 64-bit di Windows (lihat Halaman wiki VLC untuk perbedaan baris perintah untuk Linux dan Windows serta contoh dan opsi tambahan).

Saya telah menggunakan VLC untuk mengonversi audio untuk satu set file video tetapi pendekatan yang sama dapat membantu dalam tugas transcoding apa pun. Anda harus membuat file .bat sederhana:

for %%a in (*.avi) do "C:\Program Files (x86)\VideoLAN\VLC\vlc.exe" -I dummy -vvv "%%a" --sout=#transcode{acodec=mp3,ab=128,channels=2,samplerate=22050}:standard{access=file,mux=ts,dst="%%a_conv.avi"} --sout-keep vlc://quit
  • File batch ini akan memindai semua file * .avi di folder yang sama dan transkode mereka dengan parameter tertentu yang dijelaskan dalam sout = # transcode { params }
  • Untuk mengetahui parameter apa yang harus Anda gunakan, Anda perlu memilih Menu-> Media-> Stream ... dan pilih semua params di GUI wizard. Pada langkah wizard terakhir, Anda akan melihat params-string yang dihasilkan yang harus Anda gunakan dalam file batch Anda.
  • File keluaran akan memiliki _conv.avi sufiks, ubah jika Anda membutuhkan ekstensi lain

Untuk menambahkan pendekatan serupa yang ditemukan di SO:

Perubahan yang saya buat, memungkinkan spasi dalam nama direktori dan nama lagu seperti yang sering (selalu) terjadi.

@echo off
chcp 65001
SETLOCAL ENABLEDELAYEDEXPANSION

for /f "delims=" %%f IN ('dir /b /s "YOUR_DISK:\Path\To\Your Music\That May contain Spaces\*.wav"') do (
set file1=%%~nf.mp3
echo "file :" !file1!
set fic1=%%f
echo "file : " !fic1!

CALL "C:\Program Files (x86)\VideoLAN\VLC\vlc.exe"  "!fic1!" --sout="#transcode{vcodec=none,acodec=mp3,ab=320,channels=2,samplerate=48000}:std{access=file{no-overwrite},mux=mp3,dst="""!file1!"""}" vlc://quit
)

echo .
echo conversion finished
pause

Apakah format mengubah nomor seri USB?

Saya telah membaca pertanyaan / jawaban ini dan menemukan bahwa stik USB dapat memiliki nomor seri yang terkait dengannya. Ini adalah info menarik untuk proyek saya, tetapi saya ingin tahu apakah format ulang stik USB dapat menghapus nomor seri dan menghasilkan yang baru.
Baiklah, saya dapat mencobanya sendiri tetapi saat ini saya tidak memiliki stik usb untuk memformat.

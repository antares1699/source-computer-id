Saya mengalami masalah yang sama setelah memperbarui ke Windows 8.1 dengan PCE-N53 saya. Tampaknya driver yang digunakan di windows 8.1 tidak menyukai kartu wifi. Anda memerlukan driver pabrikan asli yang disertakan dengan CD instal kartu - Inilah yang saya lakukan untuk memperbaiki masalah:

  1. Windows 8.1 akan meminta Anda untuk mengizinkan Anda menggunakan opsi pengaktifan lanjutan setelah pasangan pertama gagal melakukan boot - boot ke mode aman tanpa jaringan.
  2. Hapus driver dari Driver Store - Untuk menemukan lokasi driver (Buka Device Manager temukan kartu Anda di daftar -> klik kanan ke properti -> Driver -> Detail driver tulis lokasi driver.) Lalu hapus mereka.
  3. Restart PC Anda dalam start-up normal seharusnya mulai OK - Jika tidak, Anda mungkin melewatkan driver.
  4. Kembali ke Device Manager, temukan kartu Anda dan klik kanan -> Perbarui driver -> Arahkan ke drive CD Anda dengan driver Produsen, temukan folder untuk sistem operasi Anda Win8x86 atau Win8x64 dan perbarui driver.
  5. Setelah itu Anda harus baik- (Jika Anda menjalankan pembaruan win dan melihat pembaruan untuk driver, JANGAN memperbaruinya) Inilah yang mengacaukan segalanya bagi saya di tempat pertama.

Semoga berhasil!

Saya mengalami masalah ini semalaman ketika menginstal / memperbarui Windows ke 8.1.

  • Pastikan Anda telah mencoba menginstal 8.1 sehingga pembaruan sudah diunduh sebelumnya.
  • Matikan PC dan lepaskan Kartu ASUS PCE dari slotnya.
  • Instal pembaruan (tanpa kesalahan).
  • Matikan PC dan masukkan kembali Kartu PCE, lalu hidupkan PC.
  • Instal driver dari CD ( Tidak reboot sesuai kebutuhan)
  • Pergi ke ' Panel kendali ' kemudian ' Perangkat keras dan Suara '
  • Dibawah ' Perangkat dan Printer ' pergi ke ' Pengaturan perangkat '.
  • Dibawah ' Adaptor jaringan 'klik kanan' ASUS-PCE ... 'pilih' Update Driver Software ... '
  • Pergi ke ' Jelajahi komputer saya untuk perangkat lunak driver ' kemudian ' Izinkan saya memilih dari daftar driver perangkat di komputer saya ' Pilih ' Kartu LAN Nirkabel 802.11n (Ralink Technology, Corp.) ' Tidak ' ... (Microsoft) '- Yang ini menyebabkan kesalahan.
  • Mulai ulang PC.
  • Koneksi seharusnya berfungsi sekarang. Tidak instal pembaruan untuk perangkat karena ini akan menyebabkan kesalahan yang menyakitkan; klik kanan dan sembunyikan pembaruan itu agar tidak terinstal.

Ini berhasil untuk saya, saya mencoba segalanya dan ini adalah satu-satunya cara, semoga pembaruan memperbaikinya. Jika tidak berhasil untuk Anda, mohon jangan mengeluh. Saya bukan pembuat keajaiban. Saya hanya mencoba membantu.

Terima kasih.

  • Itu BUKAN Malware

  • Ini adalah nama folder konvensional, yang digunakan Microsoft untuk menempatkan file. Angka (e849ad3686d570a415) mewakili Pengkodean heksadesimal di pasang (e8 49 ad 36 86 d5 70 a4 15) sehingga tidak akan dikenali oleh software.

  • UNTUK MENGHAPUS

    1) Buka menu "alat" di bagian atas

    2) Pilih "opsi folder"

    3) Di bawah Tab Tampilan, nonaktifkan opsi "gunakan berbagi file sederhana"

    4) Sekarang Pilih Folder yang Anda inginkan menghapus

    5) Klik kanan padanya dan pilih "Berbagi dan keamanan"

    6) Di bawah Tab Keamanan

    7) Tambahkan Pengguna ke Daftar

    8) Kemudian Di Bawah di Izin

    9) Pilih "Kontrol Penuh"

    10) Terapkan dan OK

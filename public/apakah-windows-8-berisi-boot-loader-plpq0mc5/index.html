Apakah Windows 8 berisi boot loader?

Apakah Windows 8 dilengkapi dengan boot loader? Seperti Lilo atau Grub? Atau apakah kita masih harus menggunakan pihak ketiga?

9
9

Windows 8 memiliki pemuat boot khusus yang mengunci lingkungan pra-OS menggunakan UEFI. LILO dan GRUB adalah menjalankan FOSS dan Microsoft menggunakan boot loader kustom (bukan pihak ketiga).

Boot loader Windows 8 menggunakan UEFI sementara grub dan lilo akan menjalankan OS apa pun. Bootloader Windows 8 juga jauh lebih aman.

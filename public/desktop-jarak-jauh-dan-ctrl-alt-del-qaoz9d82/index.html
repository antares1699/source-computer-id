Desktop Jarak Jauh dan Ctrl + Alt + Del

Saya baru saja mulai menggunakan koneksi desktop jarak jauh dari laptop saya untuk pertama kalinya untuk mengontrol PC yang menjalankan dinding video 8 layar. Ketika saya meremote ke PC, layar pada PC jarak jauh tetap pada layar Ctrl + Alt + Delete dan saya memiliki "pseudo login" yang berjalan di jendela jarak jauh. Apakah ini cara kerjanya? Bisakah saya tidak menghapus layar itu saat saya masuk dan melihat tindakan saya di layar secara nyata, bukan di jendela laptop saya?

Seperti sekarang, saya tidak memiliki visibilitas apa pun di layar fisik (tampaknya RDC hanya mengizinkan visibilitas satu layar) dan mesin jarak jauh tetap terkunci ketika saya selesai.

emacs tidak akan membuat file konfigurasi untuk Anda. Jika Anda ingin membuat konfigurasi, Anda dapat membuat file secara manual dan memulai konfigurasi Anda. Anda harus meletakkannya di direktori home Anda.

Anda juga bisa menggunakan M-x customize untuk menggunakan alat kustomisasi bawaan emacs. Setiap perubahan yang disimpan di customize akan disimpan dalam file .emacs Anda (yang baru akan dibuat jika belum ada).

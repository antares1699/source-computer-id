GUI yang benar-benar dilucuti akan menyenangkan tetapi saya dapat hidup dengan terminal sistem. Idealnya, ISO di bawah 50MB jadi saya bisa meletakkannya di CD berukuran kartu kredit.

SystemRescueCd (www.sysresccd.org) adalah pilihan yang baik di sini. Ini cukup kecil (<300MB) tetapi juga penuh fitur dan salah satu CD "utilitas" yang lebih baik. Segala jenis kegagalan boot yang melibatkan mount yang gagal, kerusakan sistem file dan sejenisnya dapat diperbaiki dengan ini (kecuali harddisk Anda mati tentunya). Ini juga mendapat GUI xfce indah yang relatif ringan pada sumber daya.

Ups, saya membaca 50MB sebagai 500MB. Itu terlalu besar. Maaf. Saya sedang menyelidiki berapa banyak yang bisa dihilangkan saat ini, yang bisa Anda gunakan instruksi ini untuk mengkompilasi ulang dengan lebih sedikit barang. Saya membayangkan Anda tidak akan menginginkan GNU toolchain, xfce / x11, atau portage / emerge juga. Kalau dipikir-pikir, mungkin lebih mudah menggunakan DSL / Puppy. Saya bersumpah dengan ISO penuh untuk sebagian besar CD utilitas portabel.

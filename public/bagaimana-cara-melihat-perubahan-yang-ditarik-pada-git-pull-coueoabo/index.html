Bagaimana cara melihat perubahan yang ditarik pada `git pull`?

Setelah (atau sebelum) melakukan git pull, apakah ada cara mudah untuk melihat pembaruan apa yang telah didorong oleh orang lain ke cabang yang saya kerjakan?

Saya pikir Anda pertama kali akan melakukan

git fetch

untuk menarik komit baru, tetapi belum menggabungkannya. Kemudian Anda bisa mencatat perubahan antara repositori Anda saat ini dan asal jarak jauh:

git log HEAD..origin

Terakhir, gabungkan perubahan:

git log HEAD..origin

Sebagai contoh:

$ git fetch

[...]
From github.com:blah
   00f82fa..38e4017  master     -> origin/master

dan:

$ git log HEAD..origin
commit 38e4017bc89a6eb41252465cbde68f7d897377cb
Author: Werner Robitza
Date:   Tue Jul 12 10:56:49 2011 -0700

    Edited Gemfile via GitHub

Daripada menarik cabang lain ke cabang Anda, Anda dapat memeriksa dulu komit yang akan ditarik dengan git log MINE..THEIRS. Jika ini adalah cabang jarak jauh, Anda dapat mengambil perubahan terbaru mereka untuk perbandingan dengan git remote update.

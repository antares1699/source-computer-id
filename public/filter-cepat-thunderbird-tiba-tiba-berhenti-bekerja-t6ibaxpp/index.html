Filter Cepat Thunderbird tiba-tiba berhenti bekerja

Saya menggunakan Thunderbird, versi yang saya jalankan saat ini adalah 24.1.0.

Saya biasa mencari pesan yang berisi ID atau alamat tertentu dengan Filter Cepat, tetapi sejak kemarin entah bagaimana itu berhenti berfungsi. Saya memasukkan ID yang saya cari dan ada "Tidak ada hasil" padahal sebenarnya ada pesan yang berisi frase pencarian di kotak masuk itu.

Saya belum melakukan perubahan apa pun yang saya ketahui, tidak ada pembaruan atau perubahan pengaturan atau apa pun. Saya juga memeriksa bahwa keempat tab "Pengirim", "Penerima", "Subjek", dan "Isi" aktif.

Apa yang saya lakukan salah? Ada ide?

DIPERBAIKI: Saya pernah mengalami masalah ini di masa lalu dengan folder IMAP tertentu yang tidak menampilkan item secara akurat menggunakan alat filter cepat (item akan muncul dalam pencarian thunderbird global).

  1. Klik kanan folder | Perbaiki Folder
  2. Menu file | Folder Ringkas

Dari sana, Anda akan melihat di Manajer aktivitas (menu alat | Manajer Aktivitas) folder mungkin diunduh ulang, dan pengindeksan ulang semua pesan akan terjadi. Setelah itu, Filter Cepat Anda akan merefleksikan item secara akurat.

Sudah lama sejak saya melakukan pemeliharaan pada struktur folder thunderbirds :)

Sejak masalah ini terjadi, sepertinya saya telah menemukan jawabannya. Pada titik tertentu, ketika kotak surat mencapai ~ 25.000 pesan, filter (seperti dijelaskan di atas) berhenti bekerja. Setelah saya menghapus beberapa pesan, pesan itu mulai berfungsi lagi, dan begitu saya mendapat pesan baru yang mendapat jumlah pesan kotak masuk "di atas batas", pesan itu berhenti bekerja.

Saya mencoba memadatkan folder seperti yang dijelaskan oleh David T, tetapi itu tidak terlalu membantu.

Saat ini satu-satunya penanganan tampaknya tidak mengumpulkan terlalu banyak pesan.

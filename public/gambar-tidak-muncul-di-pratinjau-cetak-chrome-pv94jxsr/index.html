Gambar tidak muncul di pratinjau cetak - Chrome

Menggunakan Windows 7. Foto (gambar) tidak muncul di Pratinjau cetak di browser chrome - versi 36.0.1985.125m dan seterusnya. Peta berfungsi dengan baik. Saya telah melampirkan sampel. Ini terjadi di Chrome saja.

Catatan: Bekerja dengan baik dengan CTRL + P.

Mohon saran, terima kasih.

Pastikan gambar di-cache sebelum mengeluarkan perintah cetak, terutama saat menggunakan document.write

Cara termudah saya mengatasi masalah di Chrome adalah dengan menambahkan ini ke halaman yang ingin saya cetak:

<div><img src="image_that_does_not_show_in_preview.png" style="display:none;"/></div>

Ini akan menyimpan gambar tersebut dan memastikannya muncul di dialog pratinjau cetak Chrome.

Solusi yang sangat sederhana adalah yang ini (dan bekerja dengan firefox dan Chrome / chromium):

Buka tab baru. Tipe

data:text/html, <html contenteditable>

di bilah alamat.

Dengan demikian, Anda dapat menggunakan browser sebagai editor teks. Sebaiknya simpan tab sebagai penanda (lihat https://www.schieb.de/720201/firefox-oder-chrome-als-text-editor-benutzen ).

Kemudian salin halaman lengkap yang ingin Anda simpan ke halaman editor. Ini menyalin semua gambar bersama dengan teks. Maka sangat mudah untuk mencetak (sebagai .pdf-file) halaman dari halaman-editor.

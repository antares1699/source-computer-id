Mengapa kartu microSD saya sangat lambat?

Baru-baru ini saya membeli ASUS Transformer Book T200TA dengan memori internal 32GB dan RAM 4GB. Untuk menambah penyimpanan, saya telah menambahkan kartu microSD 64GB melalui slotnya. Kartu tersebut diidentifikasikan sebagai bagian terpisah, sebagaimana yang seharusnya dilakukan, dan semuanya baik-baik saja.

Namun, saya telah memutuskan untuk menginstal perangkat lunak pada kartu microSD. Penulisannya sangat lambat, butuh 3 sampai 4 jam untuk menginstal TeX Live 2012 !!! Instalasi lain juga memakan waktu lama!

Mengapa hal ini bisa terjadi?

Merek kartu microSD adalah Silicon Power Elite Color microSDXC 64GB UHS-I U1 85MBs With Adapter dan diformat dengan sistem file extFAT.

ASUS T200TA menggunakan Intel Bay Trail-T Z3775 SoC (System-on-Chip). Pembaca kartu SD dikendalikan oleh SoC tersebut.

Sayangnya ada beberapa bug, khususnya yang memengaruhi kartu UHS-1 SD. Tolong lihat: [BERGUNA] Daftar bug menarik Bay Trail. Kemungkinan masalah Wi-Fi / kartu SD ... diketahui?

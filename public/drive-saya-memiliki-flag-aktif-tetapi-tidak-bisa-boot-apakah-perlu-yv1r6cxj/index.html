Drive saya memiliki flag aktif, tetapi tidak bisa boot - apakah perlu?

Saya menginstal Windows ke drive baru untuk membantu memperbaiki drive saya yang rusak, tetapi saya melihat sesuatu yang aneh:

Drive baru (Disk 0) mengaktifkan tanda Active dan System, dan Boot-nya ada di partisi data, C :. Disk 1, tempat Windows diinstal pada drive lama saya, memiliki flag Active pada partisi data, dan semua flag lain yang dimiliki Disk 0 tidak ada. Disk 2 memiliki partisi Cadangan Sistem yang mengaktifkan flag aktif, tetapi tidak memiliki flag sistem dan partisi datanya tidak memiliki flag boot / etc seperti yang dimiliki Disk 0.

Jika Disk 1 dan Disk 2 memiliki flag yang sama dengan Disk 0 - dan tidak memilikinya terkait dengan mereka tidak berfungsi - atau apakah flag tersebut tidak sepenuhnya diperlukan, dan Disk 1 dan Disk 2 memiliki flag / partisi yang sesuai yang diperlukan untuk boot ?

Ah, pertanyaan yang sangat jeli. Sayangnya, kamu telah jatuh ke dalam perangkap terminologi aneh Microsoft .

Pegang celana Anda untuk kejutan besar, karena inilah cara Microsoft mendefinisikan partisi "Boot" dan "Sistem":

  • Partisi "Boot" BUKAN partisi tempat file boot (bootloader) berada! Ini adalah partisi tempat root sistem (% systemroot%) berada!
  • Partisi "Sistem" BUKAN partisi tempat root sistem (% systemroot%) berada! Ini adalah partisi tempat file boot (bootloader) berada!

Jika Anda pernah menemukan mengapa Microsoft telah memberikan definisi yang salah ke istilah yang salah, beri tahu saya juga. Lihat juga: Partisi sistem dan partisi boot di Wikipedia bahasa Inggris

Sekarang, tentang flag "Active": BIOS atau UEFI cari file boot pada partisi yang ditandai sebagai aktif. Hanya satu partisi pada satu waktu yang dapat ditandai sebagai aktif. Oleh karena itu, masuk akal untuk menandai partisi "Sistem" Anda sebagai aktif.

Saya yakin Anda telah menyimpulkannya sekarang, tetapi saya akan tetap mengatakannya: "Boot" dan "Sistem" bukanlah flag yang sebenarnya. Mereka hanyalah label yang hanya muncul di konsol Manajemen Disk. Tapi "Active" adalah flag yang sebenarnya; ada area disk tertentu untuk menyimpan bendera ini sebagai informasi. Tidak ada partisi Anda di Disk 2 yang memiliki label ini karena keduanya tidak digunakan untuk mem-boot Windows atau berisi salinan Windows yang sedang berjalan. Mungkin ada file bootloader di F: dan Windows yang berfungsi di G :, tetapi konsol Manajemen Disk tidak memeriksa dan tidak peduli.

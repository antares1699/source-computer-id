Pemeliharaan Komputer Windows 7 - Kembalikan Pintasan Desktop yang dihapus

Saya mengetahui tentang 'alat' teman tidur bawaan Microsoft yang membuat setiap pekerjaan administrator jaringan menjadi mimpi buruk ( jika kiri dihidupkan).

Singkatnya, ini akan dengan senang hati menghapus setiap pintasan desktop ke sumber daya jaringan. Kami telah menonaktifkannya secara default.

Pertanyaan saya terkait dengan memulihkan pintasan ini, Windows tampaknya tidak membuangnya ke Recycle Bin dan meskipun Anda dapat 'memulihkan versi sebelumnya' dari folder Desktop, Anda tidak dapat melakukannya dengan pengguna yang masuk ... Melainkan masalah jika Anda hanya memiliki 1 akun pengguna.

Kami memiliki pengguna yang (meskipun alat 'Pemeliharaan Komputer' dimatikan) telah muncul di 'Pusat Tindakan' dan memilih untuk menjalankannya. Saya tidak dapat mereproduksinya, tetapi alat tersebut pasti berjalan (dan menghapus 24 pintasan dari desktopnya) meskipun masih mengatakan 'Windows tidak memeriksa sistem Anda untuk masalah pemeliharaan'. ?! Sangat fustrating!

Adakah ide jika pintasan ini dicadangkan / dipindahkan ke tempat lain sehingga saya dapat dengan mudah menemukan dan memulihkannya?

Ikon mungkin masih ditemukan di beberapa titik pemulihan sistem, dan dapat diekstraksi tanpa memulihkan kembali ke titik itu.

Dari Cara Mengekstrak File dan Folder dari Titik Pemulihan Sistem di Windows 7 dan Vista :

Di Penjelajah Pemulihan Sistem, pilih titik pemulihan tertentu yang ingin Anda pasang, dan klik tombol Pasang.

Ini sekarang akan membuka titik pemulihan yang dipasang di jendela Windows Explorer di mana Anda sekarang dapat menelusuri dan menyalin file dan folder apa pun yang Anda suka.


(sumber: sevenforums.com )

Pintasan dari titik pemulihan yang dipasang akan tetap berada di direktori akar drive Windows C: sampai Anda melepasnya di Penjelajah Pemulihan Sistem, memilih titik pemulihan yang dipasang dan mengklik tombol Lepas.

Transitif: Ini juga tidak ada di Windows 7

http://social.technet.microsoft.com/Forums/en/w7itprogeneral/thread/ada5d392-2af7-43c3-9191-08baf63f0098

Masalahnya dapat disebabkan oleh beberapa tugas terjadwal yang membantu Anda hapus pintasan. Utilitas Pembersihan Desktop tidak ada di Windows 7. Ini dapat disebabkan oleh beberapa program pihak ketiga. Harap perhatikan secara khusus program pengoptimalan sistem apa pun.

Metode lainnya adalah mengaktifkan audit objek untuk folder Desktop.

  1. Klik Mulai, masukkan GPedit.msc di kotak Mulai Pencarian.
  2. Buka cabang berikut.

Konfigurasi Komputer \ Pengaturan Windows \ Pengaturan Keamanan \ Lokal Kebijakan \ Keamanan Audit

  1. Aktifkan kebijakan berikut:

Peristiwa masuk akun audit Mengaudit akses objek

  1. Buka Windows Explorer, buka folder C: \ Users [username].
  2. Klik kanan folder "Desktop", pilih Properties-> Security. Klik tombol Advanced.
  3. Klik tab Auditing. Klik Lanjutkan.
  4. Klik Add. Kemudian klik Advanced.
  5. Klik tombol Temukan Sekarang.
  6. Tunggu prosesnya selesai. Kemudian dari daftar pengguna tambahkan pengguna berikut.

ANONIM LOGON BATCH CREATOR PEMILIK Semua Orang Tamu LAYANAN LOKAL SISTEM LAYANAN JARINGAN JARINGAN

  1. Setelah memilih setiap pengguna, pilih "Hapus" dan "Hapus subfolder dan file" sebagai entri audit.

Jika Anda ingin memeriksa akun mana yang mencoba menghapus item di folder, buka Event Viewer, periksa Windows Logs \ Security Log untuk informasi detail.

Karena masalah ini terjadi setiap hari Senin, saya sarankan Anda mengaktifkannya kebijakan pada Minggu malam. Jika tidak, akan ada banyak acara. Jika Anda menemukan bahwa pintasan tersebut hilang, harap nonaktifkan pengauditan sekaligus dan periksa Log Peristiwa.

Arthur Xie - MSFT

Omong-omong, Anda dapat menonaktifkan balon ikon yang tidak digunakan dari Kontrol Panel. Buka properti Tampilan Anda, klik tab Desktop, lalu klik Kustomisasi Desktop. Hapus centang Run Desktop Cleanup Wizard setiap 60 hari. Untuk menonaktifkannya berdasarkan kebijakan, aktifkan Hapus Desktop Kebijakan Wizard Pembersihan di bawah node Desktop. Anda akan merasa lega Ketahuilah bahwa Wisaya Pembersihan Desktop benar-benar hilang di Windows Vista dan seterusnya.

Sejauh yang saya tahu mereka melewati tempat sampah daur ulang, dan langsung dihapus (tidak seperti pembersihan XP yang menempelkannya di folder Ikon yang Tidak Digunakan). Atau mungkin mereka didaur ulang, dan kemudian perawatan mengosongkan Tempat Sampah Daur Ulang? Tidak yakin. :)

Saya telah menghindari masalah ini sejak XP dengan menggunakan skrip login GP (atau beberapa, berdasarkan OU, dll.) Itu secara terprogram menghancurkan dan membuat ulang pintasan desktop wajib. Pilihan lain adalah memiliki penyimpanan sentral dari pintasan, dan menyalinnya setiap kali, tetapi saya kurang beruntung dengan ini.

Manfaat tambahan untuk metode ini adalah memperbaiki pintasan yang dimodifikasi oleh pengguna pada login berikutnya, dan dapat diubah secara terpusat, jika jalur target berubah (dll.).

Ini juga memungkinkan kami untuk membiarkan pengaturan GPO pembersihan itu sendiri sehingga kami tidak kehilangan beberapa potensi manfaatnya, seperti membersihkan sampah ikon pribadi pengguna. ;)

Sebagai referensi:

XP / 2003: User Configuration -> Administrative Templates -> Desktop -> Remove the Desktop Cleanup Wizard

Win7 / 2008R2: Computer Configuration -> Administrative Templates -> System -> Troubleshooting and Diagnostics -> Scheduled Maintenance

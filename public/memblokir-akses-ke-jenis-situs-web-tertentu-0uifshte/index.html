memblokir akses ke jenis situs web tertentu

Perlu memblokir akses ke jenis situs web tertentu tanpa mencantumkan setiap URL untuk diblokir. Siswa di sekolah menengah pergi ke situs porno. Harus dapat memblokir semua akses tersebut tanpa harus mencantumkan setiap URL situs yang memungkinkan. Memiliki tab Konten -> Peringkat disetel ke Tidak Ada untuk semua kategori pada file peringkat yang terdaftar di komputer saya tidak mencegah akses.

Menghapus centang pengguna dapat mengakses situs tanpa peringkat, bahkan dengan pengaturan keamanan yang disetel ke Tinggi, masih memungkinkan situs porno muncul. Jika dicentang, maka HANYA situs yang terdaftar yang dapat dibuka dan siswa tidak akan dapat melakukan penelitian apa pun melalui google, misalnya. Saya lebih suka tidak perlu terus memeriksa setiap komputer dan memblokir situs saat mereka menemukannya.

Bisakah vm menjadi mesin uji yang andal?

Aplikasi saya harus berjalan di Windows lawas hingga 95. Saya tidak memiliki perangkat keras yang dapat menjalankan OS usang seperti itu, tetapi klien saya memilikinya. Saya mengembangkan aplikasi saya di Windows 7, tetapi semua menggunakan C ++ dan stl lama biasa bersama dengan API Windows yang kompatibel.

Bagaimana cara menguji apakah aplikasi saya berjalan sebagaimana mestinya di semua sistem?

VM adalah pilihan yang jelas, tetapi seberapa andal mereka dalam meniru perangkat keras nyata yang berusia lebih dari satu dekade? Saya ingin menguji kecepatan dan fungsionalitas.

VM adalah mesin uji yang bagus. Faktanya, kecuali Anda menggunakan / menguji perangkat keras khusus, itu mungkin mesin uji yang lebih baik daripada kotak fisik.

Dari segi fungsionalitas, VM berfungsi penuh dan dapat digunakan sebagai mesin pengujian, kecuali Anda menggunakan perangkat keras khusus. Beberapa perangkat yang terhubung dengan USB atau serial didukung, tetapi ada yang bermasalah.

Namun, dari segi kecepatan, Anda perlu memperhitungkan bahwa mesin virtual Anda dapat menjadi urutan besarnya lebih cepat daripada perangkat keras sebenarnya yang berusia 10 tahun. Jadi, fakta bahwa sesuatu yang berjalan sangat cepat pada VM Anda tidak berarti bahwa itu akan cukup cepat pada perangkat keras sebenarnya. Saya tidak mengetahui adanya platform virtualisasi yang dapat menghambat kinerja CPU atau I / O secara manual.

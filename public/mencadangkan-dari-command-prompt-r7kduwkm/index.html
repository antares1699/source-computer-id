Anda perlu meletakkan sumber dalam tanda kutip karena ada spasi di nama jalur:

xcopy "c: \ documents dan settings \ *. doc" f: \ backup

Yang setara dengan Robocopy adalah:

robocopy / s "c: \ documents and settings" f: \ Backup * .doc / r: 0 / dcopy: t

"/ s" memastikannya berfungsi pada semua subfolder- bukan hanya direktori root (tetapi mengabaikan subfolder kosong). Sakelar "/ r: 0" menghindarinya macet jika file terkunci- Anda selalu dapat menjalankan tugas lagi jika melaporkan salinan "gagal" di ringkasan di bagian akhir. Sakelar "/ dcopy: t" akan mempertahankan berbagai stempel waktu file.

Bekerja dengan baik- Saya baru saja mengujinya.

Saya baru menyadari bahwa kami membuat asumsi bahwa semua dokumen Anda dalam format Word lama (.doc)

Jika Anda menginginkan SEMUA dokumen, Anda harus menambahkan ekstensi file ini (dan mungkin yang lainnya) juga:

* .docx * .pdf

Ini adalah masalah yang aneh tetapi telah terjadi cukup banyak di Windows 7 belakangan ini (patch yang buruk di beberapa titik?).

Karena Anda SUDAH dapat menyalin dari satu sistem ke sistem lain, sepertinya ada masalah manajemen memori di sistem Anda, dan komputer tidak menyalin dokumen / folder / file ke memori sementara sebelum mentransfernya ke lokasi baru dengan benar. Sementara itu, saat menyalin ke / dari sistem jarak jauh, ada kemungkinan file-file sementara disimpan secara berbeda.

Mungkin ada file sistem "rusak" yang menyebabkan masalah; coba lari:

sfc /scannow

Ingatlah bahwa Anda harus menjadi administrator untuk menjalankan alat ini.

Jika itu masih tidak memberi Anda apa-apa, coba nonaktifkan Aero dan kemudian coba salin file. Untuk menonaktifkan aero, buka Control Panel dan ketik Aero di kotak pencarian pojok kanan atas. Ini akan muncul dengan opsi untuk "mengaktifkan / menonaktifkan kaca di windows".

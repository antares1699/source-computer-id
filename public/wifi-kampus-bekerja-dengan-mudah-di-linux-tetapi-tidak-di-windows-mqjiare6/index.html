Wifi kampus bekerja dengan mudah di Linux, tetapi tidak di Windows

Di Linux:

Setelah terhubung ke wifi kampus, masuk ke halaman login jaringan, internet berfungsi dengan sempurna sebagaimana mestinya.

Di Windows:

Setelah terhubung ke wifi kampus, masuk ke halaman login jaringan, login, Windows menampilkan "Akses Internet" dan ikon nirkabel berubah menjadi putih. Namun tetap setelah itu, terlepas dari browser yang digunakan, mencoba mengakses halaman mana pun hanya menunjukkan " Mengirim permintaan ". Itu berhasil meskipun setelah banyak percobaan, tetapi hanya dalam interval.

Tetapi ketika menjalankan Ubuntu 11.10 di VirtualBox, ini berfungsi dengan baik seperti boot di Ubuntu, bahkan jika tidak berfungsi di Windows.

Layanan wifi kampus benar-benar jelek dan tidak dapat memecahkan masalah ini. Saya cukup yakin harus ada solusi untuk ini, tapi apa? Apa yang Ubuntu lakukan dengan benar dan Windows tidak?

Pengaturan Windows diatur ke " Deteksi pengaturan secara otomatis "dan tidak ada server proxy yang digunakan.

Pengalihan semacam ini biasanya dari jaringan yang memaksa semua pengguna melalui server proxy mereka. Mungkin Anda tidak memiliki penemuan otomatis proxy yang berfungsi dengan baik di Windows, lihat WPAD di Wikipedia . Saya pernah mengalami masalah dengan Firefox dan WPAD sebelumnya, menemukan alamat proxy yang benar dan mengaturnya secara manual. Mungkin Anda dapat bertanya kepada departemen TI perguruan tinggi Anda tentang informasi itu (alamat proxy)?

Traceroute Ubuntu Anda menunjukkan banyak waktu habis .. Traceroute Windows Anda yang 'tidak berfungsi' menunjukkan bahwa ia tidak dapat mengakses server DNS.

Periksa server DNS Anda di Ubuntu (cat /etc/resolv.conf)

Periksa Windows satu Anda (ipconfig /all)

Apakah mereka sama? Jika tidak, apakah itu membantu membuatnya sama? Untuk mengatur server DNS di Windows:

  1. Ketik network connections di pencarian menu mulai. Pergi ke View Network Connections.

  2. Klik kanan adaptor yang sesuai, pilih Properties

  3. Pilih TCP / IP (mungkin IPv4), klik Properties

Saya menggunakan teamvier untuk akses jarak jauh ke komputer saya. Jadi itu berjalan di komputer itu secara konstan.

Tapi saya tidak suka banyak ikon di toolbar. Biasanya ketika saya memiliki lebih dari 4 - Saya mulai jengkel :-). Saya ingin memindahkan Teamvierer ke system tray. Masalahnya adalah saya tidak dapat menemukan opsi seperti itu di pengaturan.

1) Klik kanan pada taskbar

3) Sesuaikan ...

4) Pada ikon Penampil Tim pilih Sembunyikan ikon dan pemberitahuan

5) Klik OK dan Selesai!

Dengan versi yang lebih baru, komputer dan kontak peninjau tim tetap berada di toolbar. Bunuh saja layanan Teamviewer di bawah pengelola tugas. Ini memuat ulang secara otomatis, tetapi komputer dan kontak Anda akan hilang di bilah alat. Sekarang cukup klik pada Teamviewer di systemtray dan Anda melihat program jarak jauh dan komputer dan program kontak bersama-sama tanpa barang di toolbar.

Memberitahu Teamviewer ini sudah berkali-kali, tetapi mereka masih belum menyelesaikan bug ini. Itu sudah ada sejak Teamviewer 7.

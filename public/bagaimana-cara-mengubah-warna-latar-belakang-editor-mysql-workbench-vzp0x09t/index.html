Editor menggunakan Scintilla dan dikonfigurasi melalui file data / code_editor.xml seperti yang Anda tebak.

File ini terdiri dari bahasa dan untuk setiap bahasa gaya harus ditentukan.

Untuk mengubah latar belakang teks gunakan id 32 (STYLE_DEFAULT)

Contoh: <style id="32" back-color="#000000" /> <!-- STYLE_DEFAULT -->

Latar belakang nomor baris, di sebelah kiri, dapat disesuaikan melalui id 33 (STYLE_LINENUMBER)

Id gaya lainnya adalah untuk kata kunci yang berbeda (dijelaskan pada komentar code_editor.xml). Contoh: Latar belakang komentar dapat diubah dengan menambahkan back-color ke id 1 dan 2 pada bahasa SCLEX_MYSQL.

Ini harus dilakukan untuk setiap bahasa yang ingin Anda sesuaikan.

Sejauh yang saya tahu tidak ada cara untuk mengubah default secara global.

Anda dapat melihat semua alamat IP yang dikonfigurasi menggunakan perintah ini:

netsh interface ip show address | findstr "IP Address"

Anda juga dapat menambahkan nama adaptor untuk mendapatkan alamat IP dari antarmuka jaringan tertentu.

netsh interface ip show address "Ethernet" | findstr "IP Address"

Ini harus bekerja setidaknya di versi terbaru Windows.

Cara valid lainnya adalah melalui WMIC:

wmic NICCONFIG WHERE IPEnabled=true GET IPAddress

Ini akan menunjukkan alamat IP jika ada dan adaptor yang telah dikonfigurasi diaktifkan. Cukup berguna dalam banyak situasi.

Saya baru saja membangun jawaban @ Asbak,
tapi untukku aku butuh alamat IP sebenarnya saja , jadi saya akan membagikannya di sini jika ada orang lain yang juga perlu mendapatkan alamat yang sama:

  1. Temukan nama antarmuka yang ingin Anda ketahui
    Bagi saya, itu Configuration for interface "Wi-Fi",
    jadi bagi saya namanya Wi-Fi.
    Ganti Wi-Fi pada perintah di bawah ini dengan nama antarmuka Anda

  2. PowerShell:

    netsh interface ip show address "Wi-Fi" `
  | where { $_ -match "IP Address"} `
  | %{ $_ -replace "^.*IP Address:\W*", ""}

    Keluaran: 192.168.1.10

  3. Atau, kasus tepi saya, menjalankan perintah di WSL2:

    netsh.exe interface ip show address "Wi-Fi" \
   | grep 'IP Address' \
   | sed -r 's/^.*IP Address:\W*//'

# e.g.
export REACT_NATIVE_PACKAGER_HOSTNAME=$(netsh.exe interface ip show address "Wi-Fi" \
   | grep 'IP Address' \
   | sed -r 's/^.*IP Address:\W*//')

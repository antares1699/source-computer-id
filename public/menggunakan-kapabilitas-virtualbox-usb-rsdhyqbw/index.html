Menggunakan kapabilitas VirtualBox USB

Dengan menggunakan kemampuan USB dari VirtualBox, saya telah mencoba beberapa komputer yang berbeda dan terus menerima pesan bahwa perangkat USB sedang sibuk dengan permintaan sebelumnya. Saya mencoba mengakses barang-barang USB sejak versi pertama (sekarang menggunakan 3.0.4), tetapi saya tidak pernah berhasil.

Apakah ada konfigurasi atau penyesuaian khusus yang harus dibuat?

Saya mengalami masalah yang sama setelah memutakhirkan dari VirtualBox 2.2.2 ke 3.1.6, dan sayangnya mem-boot ulang tidak berpengaruh. Namun, setelah lama mencari, saya menemukan solusi pada file Forum VirtualBox :

Dengan VirtualBox dimatikan, lepaskan perangkat USB dari host Anda. Kemudian mulai ulang VirtualBox, mulai mesin virtual, lalu sambungkan kembali perangkat USB. Sekarang, VirtualBox mengenali perangkat tersebut.

Catatan : Saya melakukan pengujian dengan Windows Vista dan Windows 7.

Anda akan ingin mengatur filter USB dan juga mengaktifkan USB untuk mesin virtual. Di bawah pengaturan mesin virtual Anda, buka tab USB. Anda akan ingin memastikan Pengontrol USB diaktifkan terlebih dahulu dan terutama. Kemudian (saat perangkat usb Anda dicolokkan) tambahkan filter untuk itu, prosesnya cukup mudah.

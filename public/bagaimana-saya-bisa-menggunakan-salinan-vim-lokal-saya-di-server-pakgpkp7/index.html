Bagaimana saya bisa menggunakan salinan Vim lokal saya di server?

Seperti setiap pengguna Vim, saya telah menghabiskan bertahun-tahun mengutak-atik .vimrc saya. Tetapi ketika saya terhubung ke server di tempat lain, saya kembali ke polos, vanilla vim.

Ini masih editor yang hebat, dan saya masih bisa menyelesaikan banyak hal, tetapi saya bertanya-tanya: apakah ada cara untuk mengedit file jarak jauh menggunakan salinan lokal Vim saya?

Salin .vimrc Anda ke server.

Banyak orang menyimpan dotfiles mereka (.vimrc, .bashrc, .muttrc ...) di repositori Git / Hg, diperbarui oleh cron. Dengan cara ini semua server selalu memiliki perubahan terbaru.

Anda dapat menyelidiki sekering (Sistem File di Ruang Pengguna) dan fusefs-ssh - memasang direktori jarak jauh secara lokal menggunakan ssh.

Kemudian Anda dapat memasang direktori server jarak jauh secara lokal dan mengedit file menggunakan komputer lokal Anda.

Karena Ctrl + A dan Ctrl + F9 tidak akan membahas header / footer, dan mungkin cerita lain, VBA atau kode lain akan diperlukan, yang mungkin merupakan pertanyaan yang paling cocok untuk Stack Overflow.

Kode VBA mungkin terlihat seperti:

Sub updateFieldsIncludeHeadersFooters()
    Dim sec As Section
    Dim hdrftr As HeaderFooter

    ActiveDocument.Fields.Update 'address the fields in the main text story

    'now go through headers/footers for each section, update fields per range
    For Each sec In ActiveDocument.Sections
        For Each hdrftr In sec.Headers
            hdrftr.Range.Fields.Update
        Next
        For Each hdrftr In sec.Footers
            hdrftr.Range.Fields.Update
        Next
    Next
End Sub

Saya tidak akan merekomendasikan mengambil alih acara Word's Save untuk menjalankan ini secara otomatis, melainkan mengaitkannya ke tombol atau memiliki cara lain bagi pengguna untuk secara eksplisit menyebutnya.

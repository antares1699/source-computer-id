Bagaimana cara menghentikan orang lain melihat semua folder saya di jaringan?

Saya menjalankan Windows 7 Professional. Ketika pengguna lain melihat "Tempat Jaringan" dan memperluas sistem Windows 7 saya, semua folder dapat dilihat. Bagaimana cara menyembunyikannya?

Buka menu awal Anda dan klik kanan Computer. Pilih Manage. Kemudian lihat di bawah Computer Management > System Tools > Shared Folders > Shares. Anda dapat mengklik kanan salah satu saham di sini dan memilih Stop Sharing.

Sepertinya root drive Anda dibagikan, yang selalu berbahaya. Anda dapat mengikuti instruksi Mike untuk melihat apa yang dibagikan dan dinonaktifkan.

Sebagai aturan umum, sebaiknya konfigurasikan komputer Anda untuk tidak mengizinkan pengguna anonim mengakses berbagi file, karena ini adalah kerentanan keamanan yang besar. Klik ikon jaringan di dok pemberitahuan, lalu pilih Open Network and Sharing Center.

Di panel kiri jaringan dan pusat berbagi, klik Change advanced sharing settings. Di bawah Public folder sharing, pilih:

Matikan Berbagi folder publik (orang yang masuk ke komputer ini masih dapat mengakses folder ini)

Ini akan mencegah pengguna anonim mengakses file yang Anda bagikan. Pengguna di homegroup yang sama masih dapat mengakses beberapa file, jadi jika Anda menginginkan keamanan ekstra, pilih juga di bawah HomeGroup connections untuk

Gunakan akun pengguna dan kata sandi untuk terhubung ke komputer lain

Setelah Anda mengatur ini, tak seorang pun tanpa nama pengguna dan kata sandi Anda akan dapat mengakses share Anda.

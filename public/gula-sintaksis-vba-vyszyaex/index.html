Gula Sintaksis VBA

Saya menggunakan VBA dalam pekerjaan saya setiap hari, tetapi saya tidak tahu tentang gula sintaksis Range. Saya baru tahu bulan lalu ketika saya melihat halaman Rentang MSDN. Saya mencoba mencari artikel dengan fitur ini, tetapi saya tidak menemukan satu pun.

Apakah seseorang tahu jika VBA memiliki gula sintaksis lebih dari ini

Rentang: Range("A4:C100") hingga [A4:C100]

atau

Rentang: Range("MY_DATE") hingga [MY_DATE]

Untuk rentang

Ini semua melakukan hal yang sama:

Range(Cells(1, 2), Cells(2, 2)).Select

Range("B1:B2").Select

Dim rngB As Range
Set rngB = Range("B1:B2")
rngB.Select

[B1:B2].Select

Untuk string

Anda tidak punya banyak pilihan

Dim strA As String
strA = "hello"

strA = strA + "world" dan strA = strA & "world" melakukan hal yang sama (ampersand lebih disukai)

strA &= "world" dan strA += "world" tidak berfungsi.

Untuk lembar kerja

Anda biasanya dapat bekerja dengan lembar default aktual dengan angka, bukan dengan nama lembar itu:

Worksheets("Name").Activate
Sheets("Name").Activate
'Worksheet "Name" is Sheet1 object
Sheet1.Activate

Untuk rumus / fungsi lembar kerja

Dari Jawaban Excellll untuk kelengkapan:

Pintasan sintaksis lainnya adalah untuk mengakses dan mengevaluasi lembar kerja fungsi. Anda dapat menggunakan tanda kurung untuk mengevaluasi rumus seolah-olah ada lembar kerja daripada merangkai VBA yang rumit pernyataan.

Sub sugartest()
'long version
MsgBox Application.WorksheetFunction.Average(ActiveSheet.Range("A1:D1"))
'short version
MsgBox [AVERAGE(A1:D1)]
End Sub

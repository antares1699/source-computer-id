Konsekuensi mematikan komputer selama pembaruan Windows Vista ("Jangan matikan komputer Anda")?

Sistem saya macet di "Mengonfigurasi pembaruan" selama sekitar satu jam dengan persentase macet di 25%. Jika saya melakukan hard reboot, apa kemungkinan dan konsekuensi terburuknya?

Kemungkinan, pembaruan yang saat ini sedang aktif akan gagal, tetapi Anda dapat terus menggunakan komputer Anda seperti tidak terjadi apa-apa - Anda mungkin ingin menjalankan chkdsk pada hard drive Anda hanya untuk memastikan tidak ada masalah.

Skenario biasa kasus terburuk - Ini adalah di tengah-tengah memperbarui file sistem / setengah jalan melalui proses dan restart berarti itu hilang karena tidak ada saat ini. Namun, Windows Vista cukup baik dalam memperbaiki dirinya sendiri dan dapat boot ke mode pemulihan dan mengembalikan pembaruan untuk Anda, jadi Anda akan keluar dari tindakan untuk sementara waktu (mungkin di bawah 30 menit)

Mengucapkan skenario kasus terburuk yang tidak biasa, itu memperbarui file boot (tetapi tidak yakin berapa banyak pembaruan yang melakukan ini) dan memulai ulang akan memberikan pesan "Sistem Operasi Tidak Ditemukan" atau serupa. Dalam hal ini Anda harus memasukkan disk asli dan melakukan perbaikan startup atau yang paling buruk, menginstal ulang Windows.

Kasus terburuk : Komputer Anda tidak lagi bisa boot, dan Anda harus melakukan instalasi perbaikan dari disk OS. (Atau, SANGAT tidak mungkin, disk Anda akan sedikit rusak, jadi pastikan untuk menjalankan chkdisk / f setelah melakukan boot ulang)

Kasus terbaik : Komputer Anda akan boot dengan baik, dan menginstal pembaruan lagi dengan sempurna.

edit: Hanya untuk lebih spesifik, instalasi perbaikan akan tidak hapus semua file atau pengaturan atau program Anda.

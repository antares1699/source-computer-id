Bagaimana cara menambahkan file .el custome ke jalur saya?

Saya mencoba membuat jalur custome ".el" untuk Emacs. saya mengikuti emacs wiki petunjuk tentang mode generik. Saat saya mencoba memuat mode Emacs tidak menemukannya. (Saya telah menempatkan file di folder .emacs.d). Jadi saya mencoba apa yang dijelaskan sini yang hanya menyebabkan kesalahan saat Emacs dimuat.

Saat melakukannya. Apakah ada cara untuk memuat ulang file init saya saat Emacs dijalankan?

Saya mencoba ini di arch-linux. Ketika saya mencoba memuat file menggunakan require saya dapat kesalahan bahwa file tersebut tidak ada. Saya mencoba memuat sebagai gantinya dan pekerjaan ini.

Subdirektori di bawah instalasi emacs Anda bernama

"situs-lisp"

adalah tempat perpustakaan kustom disimpan. Salin file .el Anda di sana dan di file .emacs Anda, Anda dapat memuat file tanpa menentukan jalurnya. seperti ini:

(load-library "my-file")

di mana file-saya adalah file-saya.el

Haruskah saya khawatir dengan menghapus akun pengguna

Entah bagaimana akun pengguna rusak jadi mengikuti saran, saya membuat akun pengguna baru. Sekarang saya ingin menghapus akun pengguna lama tetapi saya sedikit gugup. Beberapa poin utama:

  1. Baik titik akun baru dan lama dokumen-dokumen saya ke folder yang sama di drive lain jadi saya ingin memastikan bahwa ketika saya menghapus akun pengguna lama, itu tidak menghapus folder dokumen saya (karena akun baru juga menggunakannya.

  2. Saya melihat sejumlah direktori di bawah C: \ Documents and Settings [Old User Id], yang menghabiskan hampir 3GB ruang, seperti:

    • C: \Documents and Settings\[Old User Id]\Local Settings
    • C: \Documents and Settings\[Old User Id]\Application Data

Misalnya: Saya punya: C: \Documents and Settings\[Old User Id]\Local Settings\Application Data\Microsoft\Office

Saat saya menghapusnya, apakah itu akan mengacaukan penginstalan Office saya?

Satu lagi yang mengkhawatirkan saya adalah: C: \Documents and Settings\[Old User Id]\Local Settings\Temp\Microsoft Visual Studio 2010 Service Pack 1_10.0.40219

Adakah saran yang akan dihargai tentang apa yang perlu saya khawatirkan?

CATATAN: akun lama ADALAH Administrator

1. Baik akun baru maupun lama mengarahkan dokumen saya ke folder yang sama drive lain jadi saya ingin memastikan itu ketika saya menghapus pengguna lama akun, itu tidak menghapus my folder dokumen (sebagai akun baru sedang menggunakannya juga.

Jika itu adalah drive jaringan, itu tidak akan menyentuh dokumen di "Dokumen saya", tetapi saya akan membuat cadangannya untuk berjaga-jaga di mana pun mereka berada.

2. Saya melihat sejumlah direktori di bawah C: \ Documents and Settings [Old User Id], yang memakan ruang hampir 3 GB, seperti: • C: \ Documents dan Pengaturan [ID Pengguna Lama] \ Pengaturan Lokal • C: \ Dokumen dan Pengaturan [Pengguna Lama Id] \ Data Aplikasi

Tidak ada salahnya menghapus ini. Pengaturan aplikasi adalah hal-hal seperti preferensi pengguna, yang dapat diatur ulang, dan akan dibuat ulang ketika aplikasi dibuka lagi untuk pertama kalinya. Ini tidak berpengaruh pada penginstalan yang sebenarnya. Saya masih akan menggunakan windirstat untuk melihat apa yang saya hapus karena ukurannya sangat besar. Saya curiga banyak file temp dan cache Internet.

misalnya: saya punya: C: \ Documents dan Pengaturan [Old User Id] \ Local Pengaturan \ Aplikasi Data \ Microsoft \ Office

ketika saya menghapusnya, apakah itu akan terjadi mengacaukan penginstalan Office saya?

Sekali lagi, ini hanyalah pengaturan pribadi pengguna, dan semacamnya ... itu akan dibuat ulang.

satu lagi yang mengkhawatirkan saya adalah: C: \ Dokumen dan Pengaturan [Pengguna Lama Id] \ Pengaturan Lokal \ Temp \ Microsoft Visual Studio 2010 Paket Layanan 1_10.0.40219

Direktori temp ini dapat dikosongkan kapan saja. Itu hanya paket layanan yang diinstal ke lokasi lain. File itu tidak lagi digunakan.

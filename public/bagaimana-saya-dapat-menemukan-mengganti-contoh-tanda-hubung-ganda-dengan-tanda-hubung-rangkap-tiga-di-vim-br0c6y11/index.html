Bagaimana saya dapat menemukan / mengganti contoh tanda hubung ganda (-) dengan tanda hubung rangkap tiga (---) di vim?

Saya tahu mungkin ada cara untuk melakukannya dengan menggunakan regex, tapi sepertinya saya tidak bisa memahami regex di vim. Saya mencoba mencari: /-\{2} tetapi juga menangkap tiga tanda hubung.

Saya juga tidak bisa hanya melakukan pencarian dan penggantian tradisional, karena saya memiliki tanda hubung tiga dalam file saya. Mencari -- mengembalikan tanda hubung rangkap tiga selain tanda hubung ganda.

  • @Jon Koma adalah pemisah — karakter apa pun yang mengikutinya digunakan untuk memisahkan ekspresi pencarian, ekspresi ganti, dan opsi apa pun (dalam hal ini g). (Biasanya / digunakan, tapi tidak terlalu bagus untuk dilihat ketika Anda memiliki banyak garis miring terbalik.) -\@<! mencegah pola agar tidak cocok segera setelah '-', dan '- \ @!' menjaganya agar tidak langsung cocok sebelum '-'.

Contoh Anda berfungsi untuk saya, jadi saya curiga Anda memiliki karakter berbeda di file Anda yang hanya terlihat seperti tanda hubung.

Saya sarankan untuk menempatkan kursor Anda di salah satu mode normal, ketik yl untuk menarik karakter, lalu ketik / diikuti dengan menekan control-r lalu "(tanda kutip ganda). Jika berhasil, Anda dapat menggunakan trik yang sama untuk membuat perintah pengganti. .

Bagaimana cara menghentikan komputer saya dari mode tidur atau hibernasi?

Saya perhatikan bahwa ketika saya menjalankan Windows Media Player di laptop saya, screen saver tidak akan mulai dan yang lebih penting komputer saya tidak akan masuk ke mode tidur atau hibernasi. Namun, saat menjalankan iTunes atau aplikasi lain, kecuali Anda menggerakkan mouse atau menekan tombol di keyboard, komputer Anda pada akhirnya akan masuk ke mode tidur atau hibernasi. Saya ingin menulis sebuah aplikasi kecil yang akan mencegah komputer saya masuk ke mode tidur atau hibernasi. Adakah yang punya ide tentang bagaimana ini bisa dilakukan?

Anda punya beberapa acara untuk ditangani untuk mencapai ini. Pertama, proses pesan WM_SYSCOMMAND, khususnya opsi SC_SCREENSAVE dan SC_MONITORPOWER. Kedua, tangani pesan WM_POWERBROADCAST dan kembalikan BROADCAST_QUERY_DENY.

Namun, MS memutuskan untuk mengizinkan BROADCAST_QUERY_DENY diabaikan oleh sistem sehingga hanya akan bekerja di tempat yang diizinkan untuk bekerja, biasanya pengaturan pengguna. Terlalu banyak aplikasi yang mencegah pemadaman listrik sehingga fitur tersebut menjadi mubazir secara efektif.

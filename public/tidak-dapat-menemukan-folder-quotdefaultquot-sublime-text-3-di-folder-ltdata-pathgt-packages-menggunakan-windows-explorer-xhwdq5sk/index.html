Tidak dapat menemukan folder "Default" Sublime Text 3 di folder <data_path> / Packages / menggunakan Windows Explorer?

Saya memiliki Sublime Text 3 (ST3) yang diinstal di Windows 7.

Saya klik "Preferensi> Pengaturan - Default" di menu ST3.

File bernama Preferences.sublime-settings terbuka di ST3. Baik. Di bilah judul ST3, ini menunjukkan bahwa file preferensi ini terletak di C: \ Users \ Myname \ AppData \ Roaming \ Sublime Text 3 \ Packages \ Default \

Sekarang, masalahnya adalah, jika saya membuka Windows Explorer dan menavigasi ke C: \ Users \ Myname \ AppData \ Roaming \ Sublime Text 3 \ Packages, tidak ada direktori bernama "Default"

Jadi mengapa saya tidak dapat melihat direktori ini? Dan di mana sebenarnya letak file "Preferences> Settings - Default" dan jika sebenarnya ada di folder lain, mengapa ia menunjukkan jalur yang berbeda di bilah judul ST3'a?

Tidak seperti ST2, yang berisi semua yang ada di direktori Packages, paket di ST3 terletak di file zip .sublime-package di direktori Installed Packages (di AppData/Roaming/Sublime Text 3). Sublime dikonfigurasi secara internal untuk menampilkan file yang terdapat dalam arsip .sublime-package yang berasal dari Packages, meskipun secara fisik mungkin tidak ada di disk.

Jika Anda ingin mengakses file dalam file .sublime-package (yang tidak akan saya rekomendasikan kecuali Anda benar-benar yakin dengan apa yang Anda lakukan), instal PackageResourceViewer paket melalui Kontrol Paket, dan gunakan perintah di Palet Perintah.

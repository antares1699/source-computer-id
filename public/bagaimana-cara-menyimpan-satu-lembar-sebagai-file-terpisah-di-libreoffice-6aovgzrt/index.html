Bagaimana cara menyimpan satu lembar, sebagai file terpisah di LibreOffice?

Klik kanan nama Sheet dan pilih Pindahkan / Salin Lembar ...

Ini akan membuka jendela dialog di mana Anda dapat menentukan Lokasi di mana Anda ingin Lembar dipindahkan / disalin. Anda juga dapat memindahkan lembar ke dokumen lain yang sudah ada - untuk melakukannya, pastikan untuk membuka dokumen yang sudah ada secara bersamaan juga.

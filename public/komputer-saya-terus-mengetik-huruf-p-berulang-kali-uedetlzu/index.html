Komputer saya terus mengetik huruf p berulang kali

Saya menggunakan laptop HP yang menjalankan Windows 7 Home Edition.

Saat saya mencapai layar login, komputer secara otomatis mengetik "ppppppppp" di tempat kata sandi, untungnya saya memiliki login sidik jari untuk mengatasinya.

Saat saya masuk, itu terus bertindak seolah-olah saya mengetik 'p' berulang kali. Saya mencoba melepas tombol dan membersihkan di bawahnya tetapi tidak ada gunanya. Saya sudah mencoba memulai ulang, mencopot driver keyboard, tetapi tidak ada gunanya.

Saya tidak dapat membuka Control Panel karena saat saya tiba di sana, ia mulai mengetik "pppppp" di kolom pencarian dan saya tidak dapat menemukan apa pun.

Saya mengetik ini dari ponsel saya karena pergi ke mana pun di laptop saya sepertinya tidak mungkin.

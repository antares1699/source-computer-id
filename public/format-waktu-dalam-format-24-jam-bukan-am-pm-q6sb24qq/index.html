Jika Anda hanya ingin mengganti nilai JAM dengan NOL untuk WAKTU di Sel, maka terapkan ini sebagai Format Sel.

"00:"mm:ss

Peringatan

  • Secara sengaja saya telah menulis rumus di Cell R32 untuk membenarkan, bahwa Cell Q32 masih memiliki Nilai WAKTU.

=TIME(HOUR(Q32)+1,MINUTE(Q32),SECOND(Q32))

  • Anda dapat melewati rumus di sel R32.

Jika Anda hanya ingin mengganti 12 Hrs dengan NOL maka Anda dapat menggunakan Rumus ini.

=IF(HOUR(Q32)=12,"00:"&TEXT(Q32,"mm:ss"),Q32)

N.B. Sel Q36 memiliki format GENERAL.

Anda dapat menyesuaikan referensi sel di Formula sesuai kebutuhan.

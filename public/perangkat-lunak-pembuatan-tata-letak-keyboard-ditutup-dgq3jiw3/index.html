Untuk Windows, Keytweak dan Pembuat Tata Letak Keyboard Microsoft lakukan apa yang saya butuhkan (Keytweak cepat dan mudah dengan UI yang bagus, MSKLC lebih terlibat dan membutuhkan kompilasi tata letak keyboard, dan terkadang tidak stabil) ...

Untuk OS X, saya sangat merekomendasikan Ukelele :

Ukelele adalah a Unicode Editor Tata Letak Keyboard untuk Mac OS X versi 10.2 dan yang lebih baru. Ukelele bertujuan untuk menyederhanakan pengeditan tata letak keyboard dengan menyediakan antarmuka grafis ke file .keylayout, di mana karakter yang diinginkan dapat diseret ke tombol sesuai kebutuhan. (Palet Karakter, tersedia di menu Input jika telah diaktifkan di Preferensi Sistem, adalah tempat yang tepat untuk menemukan karakter.)

Selain penetapan sederhana kode karakter tunggal ke kunci, Ukelele dapat menetapkan string dengan banyak karakter dan dapat membuat "kunci mati", di mana penekanan tombol menetapkan status baru yang mengubah keluaran dari penekanan tombol berikut.

Untuk penetapan keyboard dengan daya penuh di OS X, Pemicu Quicksilver dan QS secara umum.

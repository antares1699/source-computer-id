Bagaimana cara mengalihkan keluaran dari stdout ke vim? [duplikat]

Saya ingin melihat output echo di vim, dan menyimpan ke file setelah melihatnya. Saya sudah mencoba echo $PATH | vim, tetapi saya mendapatkan error berikut:

Vim: Warning: Input is not from a terminal
Vim: Error reading input, exiting...

Vim: Finished.

Apa yang dapat saya?

Anda kehilangan argumen nama file - yang menginstruksikan Vim untuk mengisi buffer dari stdin; cp. :help --

echo $PATH | vim -

Sebagai alternatif, Anda dapat menggunakan cangkang Anda proses substitusi untuk membuat deskriptor file sementara dan meminta Vim mengedit file "virtual" itu.

vim <(echo $PATH)

Bagaimana cara mengalihkan keluaran dari stout ke vim?

Pertanyaan Anda telah terjawab di AskUbuntu.

Solusi paling sederhana adalah menambahkan - ke perintah Anda:

echo $PATH | vim -

Kamu bisa gunakan proses substitusi (ini juga berfungsi dengan aplikasi yang tidak dapat membaca dari STDIN):

vim <(ls -la)

Atau gunakan fungsi vim untuk membaca dari STDIN:

ls -la | vim -

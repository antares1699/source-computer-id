Mengubah folder instalasi GIMP 2 di Windows

Saya baru saja menginstal GIMP 2.8.16 di Windows XP (32-bit). Penginstal meminta saya untuk semuanya - komponen mana yang akan diinstal, asosiasi file, dll. - kecuali untuk folder instalasi (yang dipilih sebagai %ProgramFiles%\GIMP 2).

Bagaimana cara menginstal GIMP ke folder yang berbeda? Ini menempati ~ 400 MB, dan ruang disk saya hampir habis pada drive C: saya.

Setidaknya dengan penginstal GIMP 2.10.x, menggunakan CMD untuk melakukan gimp-<VERSION>-setup.exe /? mengungkapkan bahwa penginstal dapat dimulai dengan direktori instalasi yang telah ditentukan sebelumnya. Agaknya ini untuk orang-orang yang melakukan pengujian regresi atau semacamnya dan menginginkan beberapa versi, tetapi ini juga dapat menyelesaikan masalah Anda.

Untuk penginstal yang baru saya unduh, ini berfungsi untuk saya, yang saya ketikkan ke CMD setelah cd /d ke folder tempat saya mengunduh file penginstal:

gimp-<VERSION>-setup.exe "/DIR=C:\Program Files\GIMP 2.10"

Dalam kasus saya, ia mencoba menginstal ke "GIMP 2.8", yang bukan itu yang saya inginkan, terutama karena saya mencoba menginstal 2.10.x dan namanya akan menyesatkan.

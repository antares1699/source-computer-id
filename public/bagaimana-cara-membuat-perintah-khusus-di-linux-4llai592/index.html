Bagaimana cara membuat perintah khusus di Linux?

Jadi saya di sini berpikir bagaimana membuat perintah khusus di Linux Debian saya? Maksud saya agar saya bisa membuka terminal dan cukup mengetik nama skrip (dan mungkin beberapa argumen) dan itu akan merujuk ke skrip di direktori home saya. Jika tidak, saya telah menulis seluruh jalur ke skrip setiap kali dan itu mengganggu.

Anda harus menambahkan jalur skrip Anda ke PATH, masukkan saja ~/.bashrc Anda:

PATH=$PATH:<script-directory-here>

dimana <script-directory-here> sama dengan /home/you/scripts/.

Menempatkan skrip Anda di direktorinya sendiri, mis., ~ / Bin, seperti yang dijawab cYrus, mungkin merupakan solusi terbaik. Cara lain untuk menghindari keharusan "menulis seluruh jalur ke skrip setiap kali", adalah dengan menggunakan pintasan untuk merujuk ke direktori tertentu. Misalnya, mengeksekusi

./yourscript

akan menjalankan skrip bernama skrip Anda di direktori saat ini, dan

~/yourscript

akan menjalankan skrip di direktori HOME Anda.

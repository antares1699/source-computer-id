Bagaimana cara mengekstrak file ZIP multipart di Windows yang dibuat di Mac

Saya membuat file zip besar dalam tiga bagian di Mac saya:

stuff.z01 barang.z02 stuff.z03

Saya menggunakan perintah zip terminal untuk ini.

Bagaimana cara mengekstraknya di mesin Windows saya?

Anda dapat menggunakan aplikasi gratis seperti 7Zip untuk membuka ritsletingnya. Pertama, instal aplikasi kemudian Right click pada file -> 7-zip -> Open Archive.

Kemudian klik dua kali pada file di dalam kotak dialog dan tunggu sampai pemuatan selesai lalu tekan extract. Seharusnya bekerja seperti pesona.

Anda dapat menggunakan perintah copy untuk menggabungkannya di Windows:

Prompt Perintah

copy /b stuff.z* stuff.zip

PowerShell

& cmd.exe /c copy /b .\stuff.z* .\stuff.zip

Cara termudah untuk mengekstrak arsip yang dihasilkan dengan menggunakan file Ekstrak semua opsi menu konteks di UI Explorer. Bergantian, Anda dapat menggunakan PowerShell seperti di contoh ini , Otomatisasi shell melalui command prompt seperti pada contoh ini , atau solusi pihak ketiga seperti 7-Zip . Yang terakhir ini juga akan menggabungkan kembali arsip untuk ekstraksi IIRC.

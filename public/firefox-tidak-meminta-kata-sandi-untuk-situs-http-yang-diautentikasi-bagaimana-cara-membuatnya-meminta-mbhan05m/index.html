Firefox tidak meminta kata sandi untuk situs http yang diautentikasi, bagaimana cara membuatnya meminta?

Saya mencoba masuk ke intranet perusahaan. Dulu saya meminta kata sandi di Firefox.

Tapi tiba-tiba berhenti untuk melakukannya, dan hanya menampilkan pesan kesalahan HTTP 401 Tidak Sah, sama seperti saat Anda membatalkan permintaan.

Saya mencoba menggunakan livehttpheaders untuk melihat apakah ia mencoba mengirim kredensial yang salah tetapi tidak mengirim apa pun di header.

Mencoba melakukan hal yang sama di komputer lain. Saya mendapat header yang sama tetapi itu menunjukkan prompt otentikasi. Jadi saya bisa mengembangkan otentikasi.

Saya mencoba menghapus semua riwayat, membuat profil baru, tetapi tidak berhasil.

Ia bekerja dengan browser lain dan bekerja di komputer lain pada sistem operasi yang sama dengan versi FF yang sama juga.

Jadi sepertinya beberapa pengaturan global kacau di FF saya. Adakah yang bisa bagaimana saya bisa mendapatkan prompt saya kembali?

Otentikasi NTLMv1 telah dinonaktifkan secara default di Firefox 30. Untuk mengaktifkan, file pref network.negotiate-auth.allow-insecure-ntlm-v1 harus disetel ke True. Lihat: https://developer.mozilla.org/en-US/Firefox/Releases/30/Site_Compatibility (turun halaman di bawah "Keamanan").

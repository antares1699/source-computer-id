Saya mencari di Google dan menemukan trik ini tapi itu bekerja pada jumlah sel yang konstan dari setiap grup.

Ini adalah cara paling sederhana yang saya lakukan hanya dengan rumus .... jika nilainya ada di kolom A dan B, maka rumus ini di C1, lalu disalin sejauh yang Anda inginkan, itu akan menunjukkan kosong saat nilai berjalan di luar:

= IF (ROW () <= COUNTA (A: A), INDEX (A: A, ROW ()), IF (ROW ()> COUNTA (A: B), "", INDEX (B: B, ROW ( ) -COUNTA (A: A))))

Bagaimana cara membatalkan tautan semua symlink di bawah direktori?

Saya ingin membatalkan tautan simbolis seperti ini:

unlink *

Namun demikian dikatakan unlink: extra operand

Saya hanya ingin menghapus file tautan, menyimpan file asli.

Menggunakan "*" Anda mencoba untuk memutuskan semua file dan direktori di bawah lokasi Anda saat ini.

unlink tidak dapat menangani file atau direktori, itu hanya dapat menangani tautan, gunakan perintah find untuk menemukan semua tautan dan membatalkan tautannya, seperti ini:

find . -type l -exec unlink {} \;

Saya akan menghindari pemijahan proses lain dengan find … -exec …. Bila Anda perlu unlink atau rm hasil find, gunakan opsi -delete sebagai gantinya.

find -maxdepth 1 -type l -delete

Perintah ini akan menemukan semua symlink (-type l) di direktori saat ini (bukan di subdirektori; * Anda tidak turun ke subdirektori jadi saya menganggap ini yang Anda inginkan) dan menghapusnya. Perilaku defaultnya adalah tidak mengikuti symlink, jadi file asli tidak akan terpengaruh.

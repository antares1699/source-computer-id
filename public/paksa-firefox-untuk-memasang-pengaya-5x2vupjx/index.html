Paksa Firefox untuk memasang pengaya

Saya ingin menginstal add-on baru di Firefox - Firesheep, tetapi ketika saya mencoba melakukannya (dari file xpi), saya diberi tahu bahwa add-on tersebut dapat diinstal karena tidak kompatibel untuk 4.0.1.

Namun, pemeriksaan kompatibilitas dinonaktifkan. Apakah ada cara untuk memaksa pengaya agar dipasang?

Kamu bisa gunakan Reporter Kompatibilitas Add-on . Dengan ekstensi ini Firefox akan menginstal semua ekstensi baik yang kompatibel maupun tidak. Sejauh yang saya ketahui, Anda dapat "meretas" file xpi untuk mengubah versi maksimum yang diperlukan.

  1. Ubah nama xpi menjadi zip dan ekstrak dengan perangkat lunak pengarsipan favorit Anda
  2. Temukan file install.rdf dan buka di editor teks

  3. Ubah baris berikut (jika Anda berbicara tentang Firesheep ini ):

    <em:maxVersion>3.6.*</em:maxVersion>

  4. untuk yang itu:

    <em:maxVersion>4.0.1</em:maxVersion>

  5. Simpan file dan zip seluruh folder lagi.

  6. Ubah nama zip menjadi xpi dan coba instal add-on lagi.

Perhatikan bahwa ini tidak serta merta membuat add-on kerja , tetapi setidaknya harus dipasang.

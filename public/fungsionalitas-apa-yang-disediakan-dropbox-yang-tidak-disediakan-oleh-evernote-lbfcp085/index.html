Fungsionalitas apa yang disediakan Dropbox yang tidak disediakan oleh Evernote?

Saat ini saya menggunakan Evernote sebagai sistem penyimpanan penampung-semua, aplikasi pencatatan, dan sistem transfer file.

Apa, jika ada, yang akan ditambahkan Dropbox ke kemampuan saya?

(Jika penting, saya adalah pengguna OSX / iphone / ipad.)

Keunggulan Evernote adalah menangkap catatan, ide, cuplikan web, dan semacamnya, lalu membuatnya mudah untuk menyinkronkan dan mencari hal-hal itu.

Dropbox adalah tentang menyinkronkan satu direktori / folder secara otomatis. Anda menyimpan barang ke folder Dropbox di bawah direktori home Anda - file apa saja. Aplikasi Dropbox menyinkronkan file dengan servernya dan kemudian dengan perangkat lain yang menjalankan Dropbox dengan akun Anda.

Dropbox tidak dapat mengambil cuplikan atau catatan sembarangan. Evernote tidak berinteraksi langsung dengan folder di disk Anda.

Mereka adalah aplikasi pelengkap.

Saya belum pernah menggunakan Evernote jadi saya tidak tahu apakah Evernote dapat melakukan ini, tetapi fitur penting dari DropBox adalah berbagi. Anda dapat berbagi file dan folder secara selektif dengan pengguna DropBox lain. Ini berguna untuk berkolaborasi dalam dokumen dengan orang lain, terutama secara ad-hoc hanya untuk satu proyek.

DropBox juga menyimpan riwayat file Anda, sehingga Anda dapat kembali ke versi lama atau mendapatkan kembali file yang dihapus.

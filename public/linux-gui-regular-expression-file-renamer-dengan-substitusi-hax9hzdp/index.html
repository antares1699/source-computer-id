Linux GUI Regular Expression File Renamer (dengan substitusi)

Saya memiliki folder dengan beberapa ratus file dengan nama seperti ini:

010203.txt

Saya ingin mengatur ulang nama file sehingga dua digit terakhir dipindahkan ke depan seperti ini:

030102.txt

Saya ingin menghindari menulis naskah.

Sebaliknya, saya mencari "ganti nama file ekspresi reguler linux gui" yang dapat mengevaluasi secara rekursif semua file dalam folder (dan sub-foldernya).

Saya ingin kemampuan untuk menentukan ekspresi reguler untuk mencocokkan file:

(\ d \ d) (\ d \ d) (\ d \ d) (. *)

Dan saya ingin kemampuan untuk melakukan substitusi ekspresi reguler untuk mengganti nama file:

$ 3 $ 1 $ 2 $ 4

Ada saran?

Saya menyadari ini sebenarnya bukan "GUI", tetapi Anda tidak mengatakan apa-apa Mengapa itu adalah persyaratan, dan ada sudah tertulis alat baris perintah yang melakukan ini; misalnya perl-rename:

find ~/dir -type f -exec prename -n 's/^(\d\d)(\d\d)(\d\d)(.*)$/$3$1$2$4/' {} +

(-n mengaktifkan mode "uji". Jika Anda puas dengan hasilnya, jalankan lagi dengan -v atau tanpa opsi.)

Saya menghargai sarannya, tetapi saya menemukan cara yang lebih mudah, menggunakan antarmuka pengguna grafis.

Di Ubuntu, instal thunar:

sudo apt-get install thunar

Kemudian yang harus Anda lakukan adalah menavigasi ke folder (dengan thunar) dan tekan ctrl-a untuk memilih semua file di folder itu.

Setelah ini, klik kanan pada salah satu file (yang telah Anda sorot), dan pilih "rename" dari menu konteks. Kemudian, dialog ini muncul, memungkinkan Anda melakukan substitusi ekspresi reguler untuk penggantian nama.

Ini tidak memenuhi persyaratan rekursif yang saya sebutkan, tetapi untuk mengubah nama semua file dalam satu folder, itu jauh lebih sederhana daripada baris perintah, tetapi sebenarnya, saya menghargai kedua metode tersebut.

XFCE Thunar File manager memiliki fitur pengubah nama massal yang bagus.

Cukup pilih lebih dari satu file dan tekan F2 untuk menampilkan dialog Ubah Nama Beberapa File. Dari menu drop down pilih opsi Search & Replace, centang kotak [x] Regular Expression dan ketik kriteria Search dan Replace.

Misalnya saya punya beberapa file dari grabber DVB-T bernama

20190813 Okno do New Yorku.avi
20190814 Okno do Rima.avi
20190815 Okno do Bankoku.avi
20190820 Okno do Hamburgu.avi

Menggunakan pencarian ([0-9]+) (.*) dan mengganti DOC \2 \1 transformasi menciptakan

DOC Okno do New Yorku 20190813.avi
DOC Okno do Rima 20190814.avi
DOC Okno do Bankoku 20190815.avi
DOC Okno do Hamburgu 20190820.avi

yang sangat keren.

Pengganti nama massal juga memiliki pratinjau langsung sehingga Anda dapat melihat garis yang cocok dan transformasinya di sebelah nama file sumber yang bagus untuk men-debug regexp. Nama file yang tidak cocok tidak berubah.

Kombinasi dari renameutils dan milikmu editor ekspresi reguler favorit (gedit, sublime, atom) mungkin cara termudah untuk melakukannya.

  1. Instal ganti nama utils:
sudo apt install renameutils

atau

sudo pacman -Syu renameutils

  1. Jalankan qmv dengan editor favorit Anda:

    2.1 Luhur:

    qmv -f do -e "subl -w" ./PATH/TO/FILES

    2.2 Atom:

    qmv -f do -e "atom -w" ./PATH/TO/FILES

    2.3 Gedit:

    qmv -f do -e "gedit" ./PATH/TO/FILES

  2. Gunakan kemampuan regex lengkap editor (termasuk multiselection / dan Ctrl+D) untuk membuat daftar nama baru.

  3. Simpan & Tutup

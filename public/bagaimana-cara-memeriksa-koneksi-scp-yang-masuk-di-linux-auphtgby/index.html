Bagaimana cara memeriksa koneksi SCP yang masuk di Linux?

Saya ingin melihat apakah ada koneksi SCP yang masuk ke mesin Linux saya. Bagaimana saya bisa melihatnya?

Ya, Anda dapat memeriksa koneksi masuk dengan menggunakan perintah di bawah ini.

ps aux | grep scp

Anda juga dapat memeriksa IP dengan menggunakan perintah di bawah ini. scp menggunakan ssh untuk mentransfer file.

netstat -plant | grep sshd

scp menggunakan protokol ssh, jadi scp apa pun juga akan masuk / var / log / secure sebagai koneksi ssh.

sudo grep sshd /var/log/secure |tail

atau

journalctl /usr/sbin/sshd |tail

Namun, Anda tidak akan membedakan koneksi ini dari sesi SSH di akun yang sama.

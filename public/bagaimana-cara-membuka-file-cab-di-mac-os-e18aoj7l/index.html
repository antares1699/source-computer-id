Dalam kasus saya, tidak juga The Unarchiver atau cabextract dapat mengekstrak file .cab, tetapi yang terakhir mengarahkan saya ke unshield ( https://github.com/twogood/unshield ), yang berhasil membongkar file.

Kedua cabextract dan unshield tersedia di Homebrew (brew install unshield).

Sintaks untuk membongkar file bernama data.cab dengan unshield aku s:

unshield x data.cab

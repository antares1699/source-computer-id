aview: Format keluarga magic-not p? m tidak valid

Saya hanya mencoba menggunakan aview untuk menampilkan gambar JPEG sederhana menggunakan terminal, namun setiap kali mengembalikan kesalahan ini:

Invalid magic-not p?m family format

Itu terjadi dengan file lain dari format lain juga; Saya tidak yakin apa masalahnya, dan mencari kesalahan tidak banyak membantu.

Saya menjalankan Xubuntu 12.04 dan menggunakan zsh sebagai shell saya.

Anda bisa menggunakan asciiview perintah, dari pria aview :

aview hanya dapat menampilkan file dalam format file PNM. Tapi asciiview secara otomatis mengubah format file ke PNM (dengan program eksternal seperti NetPBM dan ImageMagic) dan kemudian menjalankan tampilan.

Jadi gunakan saja:

$ asciiview file.jpg

Jika tidak berhasil, Anda mungkin perlu menggunakan:

$ sudo apt-get install imagemagick

Pilihan 1) Ubah menjadi pgm.

convert file.jpg file.pgm

Tentu saja konversi harus dipasang.

Kemudian:

aview file.pgm

Pilihan 2) Anda cukup melakukan:

asciiview file.jpg

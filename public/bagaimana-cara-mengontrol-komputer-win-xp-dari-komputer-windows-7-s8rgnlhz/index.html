Bagaimana cara mengontrol komputer Win XP dari komputer Windows 7?

Apa cara terbaik (gratis) untuk mengontrol komputer Windows XP dari jarak jauh dari komputer Windows 7 (Pro)?

yaitu saya ingin melihat desktop Win XP di pooter Win 7 saya dan dapat menjalankan perangkat lunak dari jarak jauh (meskipun itu akan berada di LAN yang sama).

Saya telah melihat sekilas Remote Desktop di Windows 7 tetapi tampaknya itu tidak mengizinkannya.

Jika Anda tidak dapat menggunakan Remote Desktop, TightVNC ( http://www.tightvnc.com/ ) berfungsi cukup baik - ini juga memungkinkan Anda mentransfer file dengan mudah. Ada klien Windows VNC lainnya, seperti UltraVNC, tetapi saya tidak memiliki banyak pengalaman langsung dengan mereka.

Semua klien VNC cenderung kompatibel satu sama lain, sehingga Anda bahkan dapat masuk ke mesin XP dari kotak Mac atau Linux.

Saya kira Anda memiliki XP Home yang tidak memiliki Server Desktop Jarak Jauh yang diinstal secara default.

Meskipun ada cara untuk memaksa masalah, ini adalah pelanggaran EULA Microsoft, jadi saya tidak dapat memasukkannya di sini.

Cara legal adalah dengan mengganti Remote Desktop Microsoft dengan Komputasi Jaringan Virtual (VNC) , yang memiliki banyak varian, karena merupakan sumber terbuka. Daftar pendek dari mereka yang layak dipertimbangkan akan mencakup RealVNC, TightVNC, EsVNC, dan UltraVNC. Varian ini dapat digunakan di LAN.

UltraVNC adalah distribusi VNC yang ditingkatkan, hanya untuk platform Win32. Klaim-untuk-ketenaran terbesarnya adalah pengait driver video khusus, yang sangat meningkatkan kecepatan tampilan jarak jauh.

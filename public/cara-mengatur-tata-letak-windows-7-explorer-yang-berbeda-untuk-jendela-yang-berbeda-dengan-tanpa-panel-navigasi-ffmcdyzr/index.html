Cara mengatur tata letak Windows 7 Explorer yang berbeda untuk jendela yang berbeda (dengan / tanpa panel navigasi)

Seringkali saya ingin membuka jendela Windows Explorer dan mematikan panel Navigasi sehingga bagus dan ringkas. Tetapi di jendela Windows Explorer saya yang lain, saya ingin panel Navigasi tetap terlihat. Jika saya menggunakan opsi Atur> Tata Letak> Panel Navigasi untuk menghapus panel navigasi dari jendela A dan kemudian di jendela B saya menavigasi ke jendela folder yang berbeda, panel navigasi B menghilang. yaitu perubahan tata letak yang dibuat dalam satu jendela berlaku untuk semua jendela.

Catatan Saya tidak ingin tata letak diterapkan ke folder tertentu, melainkan untuk jendela Explorer tertentu. Saya tidak khawatir tentang perubahan ini yang bertahan saat jendela dimulai ulang.

Karena kami adalah nama kembar, saya merasa seperti saya benar-benar harus mencoba dan menjawab yang ini.

Ini bukan solusi sempurna, tapi saya sarankan menginstal AutoHotKey dan membuat tombol pintas yang mengaktifkan dan menonaktifkan panel navigasi. Pada Windows 8 Hal ini dapat dilakukan dengan tombol Alt,V,N,Space. Pada Win7 seharusnya sangat mirip.

Setelah menginstal AutoHotKey, Anda dapat membuat skrip berikut

F12::
ID := WinExist("A")
WinGetClass,Class, ahk_id %ID%
WClasses := "CabinetWClass ExploreWClass"
IfInString, WClasses, %Class%
{
    Send !vn{Space}
}
return

Ini memetakan F12 untuk mengaktifkan Bilah Navigasi, hanya saat di Explorer. Di aplikasi lain, F12 akan terus melakukan apa pun yang dilakukannya di aplikasi itu.

Jika ternyata tombol pintas di Win7 berbeda maka cukup mudah untuk menemukan dan mengubahnya. Cukup tekan Alt di penjelajah dan lihat apa yang muncul!

Ini tentu saja bukan solusi yang sempurna. Saat Anda beralih antar jendela, Anda mungkin perlu terus mengaktifkan dan menonaktifkannya, tetapi ini jauh lebih cepat daripada menjalankan file .bat dari desktop.

Alternatif lain, seperti yang disebutkan beberapa orang, adalah menggunakan penjelajah alternatif. Saya tidak bisa bekerja tanpa xplorer2 , Saya sangat merekomendasikannya.

Menurut penelitian saya, tata letak Explorer ditentukan oleh entri registri PageSpaceControlSizer ditemukan di:
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Modules\GlobalSettings\Sizer.

Kunci registri ini dikonsultasikan setiap kali jendela Explorer baru dibuka atau kapan pun folder yang ditampilkan berubah.

Oleh karena itu, idenya adalah membuat dua file .reg dengan konten registri ini masuk saat panel navigasi aktif atau nonaktif. Isinya bisa diekspor dari dalam regedit. Berikut ini berasal dari tata letak saya.

nav_on.reg

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Modules\GlobalSettings\Sizer]
"PageSpaceControlSizer"=hex:ce,00,00,00,01,00,00,00,00,00,00,00,10,04,00,00

nav_off.reg

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Modules\GlobalSettings\Sizer]
"PageSpaceControlSizer"=hex:ce,00,00,00,00,00,00,00,00,00,00,00,10,04,00,00

Anda kemudian dapat membuat dua file batch (.bat) yang akan menggunakan file perintah reg atau cukup lakukan impor file .reg secara diam-diam:

regedit /s nav_on.reg

Membuat dua pintasan di desktop untuk dua file batch akan memberi Anda cara untuk mengaktifkan atau menonaktifkan panel navigasi untuk Explorer berikutnya jendela atau folder.

Jika Anda benar-benar ingin menjadi mewah, Anda juga dapat membuat dua ekstensi shell Explorer yang akan menambah ke menu klik kanan opsi "Buka dengan panel navigasi" dan "tanpa".

Untuk titik awal lihat artikel ini:

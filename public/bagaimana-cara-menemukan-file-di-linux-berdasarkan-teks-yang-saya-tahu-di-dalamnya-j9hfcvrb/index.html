Bagaimana cara menemukan file di Linux berdasarkan teks yang saya tahu di dalamnya?

Bagaimana saya bisa menemukan semua file di direktori / home / user / example yang berisi "beberapa teks" di Linux. Saya menggunakan Ubuntu 9.10.

Saya biasanya menggunakan kombinasi find dan grep sehingga mengambil semua file tersembunyi juga:

find /home/user/example -type f -exec grep -i "some text" {} +

Sakelar -i ke grep akan menggunakan metode pencarian tidak peka huruf besar / kecil. Ini akan lebih lambat, tetapi berguna jika Anda tidak mengetahui kasus persisnya dari kata-katanya. Anda dapat melepas sakelar jika Anda tahu kasus yang tepat untuk mempercepat.

Jika Anda tidak khawatir akan berulang ke subdirektori,

grep -l "some text" *

Hanya akan menampilkan nama file, dan bukan konteks tempat kecocokan muncul.

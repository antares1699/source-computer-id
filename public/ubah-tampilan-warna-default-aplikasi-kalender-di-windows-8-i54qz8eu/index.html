Saat membuka Aplikasi Kalender, buka menu Mantra (arahkan ke kanan atas / kanan bawah layar). Klik Pengaturan. Di sana Anda bisa memilih warna yang Anda inginkan. Namun, ini berfungsi berbeda dari Google Kalender di mana Anda dapat memilih warna untuk setiap acara. Di sini, jika Anda menyelaraskan beberapa kalender (Hotmail, Google, Facebook), Anda dapat menetapkan warna berbeda untuk masing-masing kalender.

Jika Anda masuk ke Outlook.com lalu ke Aplikasi Web Kalender Anda bisa membuat kalender sebanyak yang Anda inginkan, setiap orang dengan warnanya sendiri. Kemudian di Aplikasi Kalender Windows 8 kalender ini akan tersedia, dan Anda dapat membuat acara dan menentukan kategori / warna yang Anda inginkan.

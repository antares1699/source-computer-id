Excel - masukan pengguna tetapi tetap menggunakan rumus untuk dua sel melingkar

Saya mencoba membuat dua sel mempertahankan rumus latar belakang setelah input pengguna dan memiliki rumus melingkar.

A1 = Lbs
A2 = Kgs

Formula
A1 = A2*2.2
A2 = A1/2.2

Saya tahu tentang utas di bawah ini tetapi saya tidak bisa membuatnya berfungsi untuk rumus melingkar. Saya telah mengaktifkan gunakan kalkulasi berulang dalam preferensi. Excel - bagaimana memiliki input data langsung atau nilai sel dari rumus

Excel tidak bekerja seperti ini.

Jika kedua sel memiliki rumus maka tidak ada tempat untuk mendapatkan nilai input. Itulah mengapa ini tidak akan berfungsi dengan referensi melingkar.

Sebuah sel bisa memiliki rumus atau nilai. Jika Anda memasukkan nilai ke dalam sel yang memiliki rumus, rumus tersebut akan ditimpa. Apa yang Anda coba capai akan membutuhkan setidaknya 3 sel: sel entri untuk nilai, sel untuk unit dan sel ketiga untuk hasil konversi. Anda bahkan mungkin ingin menambahkan sel keempat untuk satuan hasil. Pertimbangkan tangkapan layar ini:

Nilai dimasukkan dalam B2, unit nilai ditentukan di C2. Konversi tersebut dihitung dalam B3

=IF(C2="lbs",B2/2.2,B2*2.2)

Satuan hasil dihitung di C3 dengan =IF(C2="Kgs","Lbs","Kgs"), tetapi bit itu opsional.

Edit: Anda dapat menggunakan VBA untuk mengonversi nilai dan meletakkannya di sel lain. Ini dapat dilakukan tanpa menulis rumus ke dalam sel. Jika Anda sudah menggunakan VBA, Anda sebaiknya melakukan penghitungan di VBA daripada menggunakan VBA untuk menulis rumus ke dalam sel. Berikut adalah VBA untuk itu:

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
If Not Intersect(Target, Range("A1:A2")) Is Nothing Then
    If Target.Count = 1 Then
        Application.EnableEvents = False
        If Target.Row = 1 Then
            Target.Offset(1, 0) = Target / 2.2
        Else
            Target.Offset(-1, 0) = Target * 2.2
        End If
        Application.EnableEvents = True
    End If

End If
End Sub

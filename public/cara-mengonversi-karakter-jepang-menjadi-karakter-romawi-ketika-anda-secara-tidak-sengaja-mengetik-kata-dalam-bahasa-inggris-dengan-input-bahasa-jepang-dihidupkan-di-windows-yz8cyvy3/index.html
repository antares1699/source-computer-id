Cara mengonversi karakter Jepang menjadi karakter romawi ketika Anda secara tidak sengaja mengetik kata dalam bahasa Inggris dengan input bahasa Jepang dihidupkan (di Windows)

Di Windows saya sering menemukan diri saya mencoba menulis kata dalam bahasa Inggris, tetapi dengan metode Masukan Jepang dipilih (secara tidak sengaja). Jadi yang saya dapatkan adalah sesuatu seperti ini えぁmpれ. Saat ini, saya hanya menekan melarikan diri lalu mulai menulis lagi. Idealnya akan ada jalan pintas yang bisa saya gunakan untuk mengubahnya secara otomatis. Ada di OSX:

Tombol fungsi F6 - F10 digunakan untuk mengubah input ke berbagai format:

  • F6 : ubah ke Hiragana

    ホ ワ イ ト → ほ わ い と

  • F7 : ubah ke Katakana lebar penuh

    ほ わ い と → ホ ワ イ ト

  • F8 : ubah ke setengah lebar Katakana . Mungkin tidak diinginkan

    ホ ワ イ ト → ﾎ ﾜ ｲ ﾄ

  • F9 : ubah input saat ini menjadi romaji lebar penuh . Mungkin tidak diinginkan juga. Tekan lagi untuk huruf kapital semua, lalu sekali lagi untuk penggunaan huruf besar kata benda yang benar:

    ホ ワ イ ト → ｈｏｗａｉｔｏ → ＨＯＷＡＩＴＯ → Ｈｏｗａｉｔｏ

  • F10 : ubah ke romaji setengah lebar . Tekan lagi untuk huruf kapital semua, dan sekali lagi untuk penggunaan huruf besar kata benda yang benar:

    ホ ワ イ ト → howaito → HOWAITO → Howaito

Jika Anda lupa di atas maka gunakan saja Windows + garis miring (/) yang merupakan jalan pintas untuk "Mulai pengubahan IME".

Jika Anda hanya ingin mengubah input berikutnya ke romaji Anda cukup mengetikkan karakter kapital.

Beberapa pintasan berguna lainnya untuk mengetik bahasa Jepang di keyboard bahasa Inggris

  • Ctrl + Kunci huruf kapital : beralih ke Hiragana
  • Alt + Kunci huruf kapital : jika dalam mode alfanumerik ubah ke Hiragana, lalu alihkan ke Katakana
  • Bergeser + Kunci huruf kapital : beralih antara Hiragana lebar penuh ↔ alfanumerik lebar penuh (romaji)
  • Alt + `(Aksen Kuburan / Backtick) : beralih antara kana ↔ alfanumerik lebar setengah (romaji)
  • Alt + Bergeser : beralih antar bahasa (IME)

Cara mendapatkan jalur setelah terbuka di jendela baru dari hasil pencarian

Langkah-langkah untuk mereproduksi:

  • Cari folder, dapatkan sekitar 100 hasil dan 8 kemungkinan folder kanan dengan nama yang sama tetapi di lokasi berbeda ...
  • Klik kanan semuanya dan buka di jendela baru untuk membandingkannya secara berdampingan
  • Luangkan 39 menit untuk mencari tahu proyek mana yang memiliki perubahan yang Anda inginkan
  • Temukan bahwa tidak ada cara yang jelas untuk mendapatkan jalur sebenarnya ke folder yang Anda lihat. Hanya beberapa omong kosong microsoft yang mengingatkan Anda betapa fantastisnya pencarian mereka.
    • Bilah lokasi mengatakan: > Search Results in Dropbox > ToolStatus >
    • Klik mungkin? Nggak. Kemudian tertulis: search-ms:displayname=Search%20Results%20in%20Dropbox&crumb=location:D%3A%5CDropbox\ToolStatus

Sekarang setelah saya menempelkan ini, saya akhirnya melihat tertanam "crumb = lokasi: [jalur yang kacau]" setelah mengklik di bilah lokasi.

Mengklik kanan hasil pencarian dan mengklik "Open Folder Location" tidak membuka alamat sebenarnya, tetapi sebenarnya itu adalah jelas bukan solusi karena tidak memungkinkan Anda membuka semua 8 kemungkinan hasil sekaligus untuk dibandingkan.

Solusi yang lebih baik dari sevenforums.com :

Tahan tombol ctrl saat Anda mengklik Buka Lokasi Folder / File Ini akan memberi Anda jendela baru dengan hasil, tanpa kode jendela lucu. Meskipun itu bukan solusi sempurna yang dimaksudkan, itu tetap bagus.

Apakah ada solusi yang lebih baik tersedia?

Sudahkah Anda mengalihkan mode tampilan folder Anda ke detail? Saat Anda melakukannya, Anda dapat melihat jalur sebenarnya dari semua item hasil pencarian Anda di kolom "jalur folder". Jika kolom ini tidak terlihat, Anda dapat mengklik kanan di header setiap kolom yang terlihat dan memilih "Lainnya ..." di bagian bawah. Anda kemudian dapat mencentang opsi "jalur folder" di jendela "pilih detail" yang muncul. Lihat snapshot saya di bawah ini.

Oke - inilah cara yang benar-benar mundur untuk mengumpulkan lokasi file dari jendela Hasil Pencarian yang rusak:

Pilih file atau folder Anda dari hasil pencarian

Klik kanan dan pilih Buka di Jendela Baru (ya - akan melihat alamat Hasil Pencarian itu)

Pilih folder di Jendela Baru, klik kanan dan pilih Buka di Jendela Baru (Sekarang dua jendela baru dengan alamat Hasil Pencarian)

Klik di kotak alamat jendela baru ke-2 - Anda akan mendapatkan nama jalur lengkap di sana. Salin-tempel kembali ke kotak alamat jendela baru ke-2.

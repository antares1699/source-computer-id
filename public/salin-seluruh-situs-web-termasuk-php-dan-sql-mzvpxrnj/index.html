Salin seluruh situs web termasuk PHP dan SQL?

Apakah mungkin saat menyalin seluruh situs web dan templat menggunakan wget (httrack atau program lain) untuk juga mendapatkan kode PHP dan database SQL?

Perangkat lunak apa yang saya perlukan untuk menyalin / mencerminkan situs seperti ini? Dan apakah "Internet Download Manager" merupakan program yang dapat digunakan untuk ini?

Tidak. Server web harus memperlihatkan file-file ini kepada Anda secara eksplisit.

Saat Anda memuat situs web yang menggunakan PHP / SQL, itu akan menjalankan PHP di server itu sendiri, dan mengembalikan output dari program PHP (biasanya campuran HTML, CSS, dan JS). File-file ini sebenarnya tidak pernah disimpan dalam status tersebut di server web, tetapi dihasilkan secara prosedural. Apa yang Anda lakukan saat menggunakan wget adalah Anda mengunduh output dari server web (respons terhadap kueri HTTP Anda), bukan menyalin file dari server.

Server SQL hanya dapat diakses dari akun tepercaya, dan biasanya diakses dengan kode PHP tersebut. Kecuali Anda memiliki kredensial untuk masuk ke server itu sendiri menggunakan ssh atau sftp, Anda tidak akan bisa mendapatkan akses ke kode PHP mentah atau database SQL.

Server dapat dikonfigurasi untuk memungkinkan unduhan langsung file dari komputer, termasuk file PHP, tetapi ini tidak umum karena sama sekali tidak sesuai dengan tujuan penggunaan PHP. Admin yang kompeten tidak akan mengizinkan akses langsung ke kode PHP, dan terutama akan sangat membatasi akses ke server SQL.

Itu tidak berarti Anda tidak dapat melakukan ini menggunakan eksploitasi keamanan. Tetapi melakukan hal itu umumnya tidak bermoral kecuali Anda memiliki file sangat alasan yang baik untuk melakukannya (misalnya, Anda dipekerjakan oleh firma keamanan untuk melakukan audit keamanan di situs web klien Anda).

Tidak, ini tidak mungkin kecuali situs webnya memilikinya serius lubang keamanan.

PHP dijalankan di server web dan tidak pernah dikirim mentah ke klien. Sebaliknya, klien menerima output dari juru bahasa PHP setelah menjalankan kode PHP.

Demikian pula, backend database tidak dapat diakses oleh klien, klien juga tidak dapat membuat kueri langsung terhadapnya.

Sekali lagi tidak. Anda hanya akan mendapatkan halaman web yang dikompilasi, bukan data back-end yang disimpan dalam database.

Anda memerlukan akses FTP ke root situs web, dari sana Anda dapat mengambil cermin termasuk bagian belakang - tetapi hanya jika database disertakan dalam jalur situs web. Jika tidak, Anda perlu membuka port untuk meminta database dan membuat mirror atau meminta penghosting secara khusus untuk memberi Anda salinan database.

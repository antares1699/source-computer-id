Hard disk 3TB ditampilkan sebagai 746GB

di BIOS, UEFI Boot diaktifkan, Mode ACHI dipilih, dan Hard disk 3TB (Seagate ST30000DM001) diinstal pada SATA Port 3.

Namun, di Windows 7 64-bit, hard disk 3TB ditampilkan sebagai 746GB, tidak peduli bagaimana saya memformat atau mengatur pengaturan. Saya menggunakan GPT sebagai tabel partisi (seperti yang disarankan oleh Windows,> 2TB harus menggunakan GPT).

Bagaimana saya bisa mendapatkan kembali penyimpanan 3TB penuh?

p.s. Motherboard adalah Intel DH55HC dan software Intel Rapid Storage Technology menunjukkan kapasitas disk 2.862GB.

p.s. Sistem Operasi adalah Windows 7 Ultimate, 64-bit

MEMPERBARUI Seseorang menandai pertanyaan ini sebagai duplikat dari masalah serupa dengan penutupan USB. Tetapi pertanyaan ini tentang hard disk yang langsung dipasang ke motherboard melalui kabel SATA.

Masalah ini teratasi di desktop HP saya yang berusia 10 tahun (4 core), dengan drive: 3TB Western Digital Green, tetapi itu juga harus sama dengan Seagate.

  • Saya menambahkan solusi ke blog ini: Posting Apa Saja Di Sini .

Singkatnya, Anda perlu melakukan 2 hal:

  • untuk menginstal driver Intel RAID (gratis).
  • menggunakan alat yang bagus untuk memperluas partisi ke ukuran maksimum, seperti AOMEI Partition Assistant (freeware).

Bersulang!

PS1 : Setelah semua adalah properti setup, Anda dapat menghapus kedua alat yang disebutkan di atas.

PS2 : Jika Anda mengalami konflik antara disk 2TB atau 3TB baru dan DVD-ROM lama, putuskan sambungan DVD, dan dapatkan yang eksternal (kasus saya).

Bagaimana cara melindungi file hard disk eksternal dari penghapusan yang tidak disengaja?

Saya mengalami sedikit masalah dalam beberapa bulan terakhir dengan data yang saya miliki di hard disk Eksternal saya. Saya memiliki konten besar di setiap folder.

Jika Anda memikirkannya, hanya dibutuhkan dua tombol untuk menghapus folder - del dan enter. Apakah ada cara untuk melindungi file dari penghapusan yang tidak disengaja?

Mungkin cara paling sederhana adalah dengan menolak diri Anda menghapus izin ke folder di drive, dan sebaliknya memaksa diri Anda untuk meningkatkan kredensial untuk menghapus file. Di windows, hapus saja izin modifikasi dari properti keamanan folder.

Pastikan properti juga diwarisi oleh semua subfolder.

Untuk 'melindungi' drive

  1. Klik kanan drive di Windows Explorer, dan pilih Properties
  2. Buka tab 'Keamanan' dan klik 'Lanjutan' di bagian bawah
  3. Klik 'Izin Berubah'
  4. Temukan pengguna yang ingin Anda ubah di daftar pengguna dan klik edit
  5. Hapus centang 'Hapus' dan 'Hapus Subfolder dan File' dan klik OK
    • Ulangi pada pengguna lain jika perlu. (Petunjuk: Memilih Pengguna harus menerapkannya ke semua akun non-administratif, dan memilih Administrator harus menerapkannya ke semua akun administrator)
  6. Pilih 'ganti semua izin objek anak dengan izin yang diwariskan dari objek ini' dan klik OK lagi dan pada Windows yang tersisa.

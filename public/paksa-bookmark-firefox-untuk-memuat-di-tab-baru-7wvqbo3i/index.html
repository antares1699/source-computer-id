Paksa bookmark Firefox untuk memuat di tab baru

Jadi masalahnya adalah saya lebih suka bookmark saya di bilah alat bookmark untuk dimuat ke tab baru. Tetapi saya tidak dapat menemukan opsi itu di mana pun. Apakah ini mungkin?

Klik tengah atau control-klik pada bookmark untuk membukanya di tab baru.

Atau, jika Anda benar-benar ingin klik biasa untuk membuka bookmark di tab baru, Anda dapat menginstal Tab Mix Plus ekstensi dan aktifkan ini. Begini caranya:

  1. Buka jendela Tab Mix Plus Options
  2. Klik pada Acara ikon
  3. Klik pada Pembukaan Tab tab
  4. Dalam Buka tab dari: bagian, aktifkan Bookmark

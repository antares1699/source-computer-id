Transfer file FTP atau SCP tanpa file .DS_Store

Saya mencari cara untuk tidak mentransfer file .DS_Store saat meluncurkan transfer FTP atau SCP untuk direktori.

Apakah mungkin untuk memblokir file .DS_Store agar tidak ditransfer bersama dengan file lain yang saya pindahkan?

Saya sebenarnya menggunakan aplikasi FileZilla

Di bawah Filezilla client, Anda dapat mengkonfigurasinya untuk memblokir beberapa ekstensi file.

Di Melihat >> Filter nama file ... Anda harus membuat filter baru. Pergi ke Edit aturan filter ... dan Baru . Lihat filter yang ada bernama " File Explorer yang tidak berguna "untuk beradaptasi dengan file Mac OSx seperti .DS_Store

Jika Anda menggunakan Cyberduck sebagai klien, secara otomatis akan melewatkan file-file itu (dan beberapa lainnya) selama transmisi.

Ini berlaku untuk koneksi FTP dan SFTP.

Jika Anda menggunakan baris perintah, Anda harus menggunakan rsync daripada scp. Dengan rsync Anda dapat menambahkan opsi --exclude='.DS_Store' untuk melewati itu.

Nonaktifkan shutdown otomatis pada daya rendah Win 8.1

Saya mengalami masalah sejak saya memutakhirkan ke Windows 8.1. Baterai saya mengalami masalah karena semakin tua, dan selalu mengatakan pada 0%, padahal saya masih dapat menggunakannya selama sekitar 2 jam.

Ini bukan masalah di Windows 8, tetapi sejak saya memutakhirkan, laptop saya mati secara otomatis ketika saya melepas kabel daya karena dianggap baterainya kosong.

Adakah yang bisa memberi tahu saya bagaimana saya bisa menonaktifkan ini? Saya telah mencari cukup lama, tetapi di setiap forum ini telah diposting, satu-satunya tanggapan yang didapat orang-orang itu adalah sesuatu seperti, "Mengapa Anda ingin melakukan ini?".

Ini telah berlangsung selama beberapa minggu terakhir dan itu sangat membuat frustrasi, jadi kami menghargai bantuan apa pun.

Saatnya necromany, karena ADA cara untuk mengatur tindakan baterai kritis ke "Tidak melakukan apa-apa".

Ini tidak sepenuhnya aman, status baterai kritis ada untuk mencegah kehilangan data saat dimatikan, tetapi jika Anda tidak melakukan apa pun yang penting untuk data, jendela menangani ini dengan cukup baik.

Triknya adalah dengan menggunakan utilitas baris perintah powercfg.

  1. Buka konsol administrator (cmd.exe dengan hak administrator).

  2. Jenis: powercfg -setdcvalueindex SCHEME_CURRENT SUB_BATTERY BATACTIONCRIT 0

    (Ini mengubah skema daya yang saat ini aktif, jadi jika Anda perlu mengubah skema lain, alihkan skema.)

  3. Buka "Opsi Daya" seperti yang dijelaskan dalam jawaban @ Synetech dan lompat-lompat dengan gembira melihat bahwa tindakan baterai Kritis memang "Tidak melakukan apa-apa" saat "Dengan baterai".

lakukan apa yang dikatakan @SoonDead selain untuk penggunaan tidur
powercfg -setdcvalueindex SCHEME_CURRENT SUB_BATTERY BATACTIONCRIT 1

untuk penggunaan hibernasi
powercfg -setdcvalueindex SCHEME_CURRENT SUB_BATTERY BATACTIONCRIT 2

untuk penggunaan shutdown
powercfg -setdcvalueindex SCHEME_CURRENT SUB_BATTERY BATACTIONCRIT 3

perhatikan bahwa nilai untuk pengaturan ini adalah sebagai berikut:
0 adalah tidak melakukan apa-apa
1 adalah tidur
2 sedang hibernasi
3 adalah shutdown

Sudahkah kamu mencoba mengonfigurasi kebijakan daya Anda ?

  1. Buka Opsi Daya halaman:
    • Panel Kontrol → Perangkat Keras dan Suara → Opsi Daya (Gambar 1) atau
    • Search for power (gambar 2) atau
    • Dari area notifikasi (gambar 3)
  2. Untuk mengedit kebijakan daya saat ini, klik Ubah pengaturan paket (Anda juga dapat mengedit kebijakan lain atau buat yang baru ) (gambar 4)
  3. Klik Ubah pengaturan daya lanjutan (gambar 5)
  4. Perluas file Baterai cabang
  5. Edit tindakan untuk berbagai status baterai (gambar 6)
  6. Klik [OK]

Memperbarui:

Saya juga telah memeriksa tab "Tindakan baterai kritis", tetapi di sana saya tidak dapat mengkonfigurasinya untuk tidak melakukan apa-apa, sementara saya dapat melakukannya untuk "Tindakan baterai lemah"

Saya kira alasannya adalah sekali baterai sangat lemah sehingga tidak ada yang tersisa, Anda memiliki melakukan sesuatu. Anda dapat mencoba menyesuaikan level dalam dialog itu (gambar 6) untuk mendefinisikan apa arti "rendah" dan "kritis" (mis., Ubah rendah menjadi 50% atau 1%, dll.) Jika level baterai Anda dilaporkan rendah, tetapi bukan nol (misalnya, terkadang 5%, terkadang 1%), maka Anda dapat menetapkan kritis menjadi 0%.

Namun, jika baterai Anda benar-benar ada selalu melaporkan 0% (dan bahkan tidak 1%), maka faktanya adalah bahwa pengukur di baterai rusak dan saya khawatir satu-satunya cara adalah tetap terpasang atau mendapatkan baterai baru. Satu-satunya cara yang mungkin untuk mencegah Windows dari tidur ketika baterai mencapai 0% adalah dengan meretas sistem karena tidak ada metode (baik bawaan atau melalui program pihak ketiga) untuk mencegahnya karena baterai 0% seperti menarik kabel dari sistem desktop.

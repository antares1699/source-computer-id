Bagaimana cara menampilkan ukuran folder di emacs dired?

Saya telah dired dan dired + setup untuk melihat daftar direktori dan menggunakannya sebagai manajer file saya untuk memindahkan / menyalin file. Daftar direktori saat ini '0' sebagai ukuran untuk semua subdirektori di direktori manapun.

Saya ingin menghitung dired dan menampilkan ukuran folder. Saya ingin ini terikat pada penekanan tombol / klik mouse karena saya tidak ingin dired melakukan ini pada setiap folder di setiap buffer.

Sampai sekarang, saya memiliki "open in explorer" (w32-shell-execution di windows dan gnome-open di gnome) yang terikat ke "O" dari mana saya menggunakan Alt + Enter untuk menghitung ukuran subdirektori.

Ini Halaman wiki Emacs menyarankan menggunakan cuplikan berikut untuk menghitung ukuran file dan direktori yang dipilih:

 (defun dired-get-size ()
  (interactive)
  (let ((files (dired-get-marked-files)))
    (with-temp-buffer
      (apply 'call-process "/usr/bin/du" nil t nil "-sch" files)
      (message "Size of all marked files: %s"
               (progn 
                 (re-search-backward "\\(^[0-9.,]+[A-Za-z]+\\).*total$")
                  (match-string 1))))))

 (define-key dired-mode-map (kbd "?") 'dired-get-size)

Jika saya memahami Anda dengan benar, Anda ingin memperbarui nilai buffer Dired secara langsung. Saya tidak cukup akrab dengan internal Dired, tapi saya rasa seharusnya tidak terlalu sulit.

Jika versi Emacs Anda> = 24.4, maka Anda bisa coba dired-du Perpustakaan. Ini tersedia dari repositori ELPA.

Setelah Anda menginstal lib ini: Jika buffer Anda saat ini dalam mode Dired, maka Anda dapat melakukan:

C-x M-r

yang mematikan `dired-du-mode 'dan menampilkan rekursif ukuran direktori 'di tempat' di buffer Dired. Jika kamu kunjungi buffer Dired baru, maka buffer rekursif akan ditampilkan juga, hingga Anda mematikan mode.

Tip lain:

C-x C-h

mengubah format ukuran. Ada 3 format:

  1. Default satu dari perintah `ls '.

  2. Format yang dapat dibaca manusia.

  3. Format numerik dengan pemisah ribuan koma.

Anda dapat menyesuaikan opsi `dired-du-size-format 'menjadi buatlah ukuran Anda menjadi pilihan yang persisten.

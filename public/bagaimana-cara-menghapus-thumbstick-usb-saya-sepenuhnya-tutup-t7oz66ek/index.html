Bagaimana cara menghapus thumbstick USB saya sepenuhnya? [Tutup]

Saya memiliki thumb-stick Kingston. Saya telah memasang gambar Google Chrome OS untuk itu, jadi saya bisa mencobanya. Ketika melakukannya, itu membuat 3 partisi (saya mencadangkan semuanya dan perangkat lunak yang menginstal Chrome OS menghapus thumb-stick saya sepenuhnya dan membuat tiga partisi).

Sekarang saya ingin mengembalikannya ke satu partisi, tetapi ketika saya masuk ke Manajemen Disk, itu tidak akan membiarkan saya "memperluas" partisi. Saya telah mencoba alat partisi yang berbeda tetapi mereka tetap tidak mengenali thumb-stick saya. Saya juga sudah mencoba diskpart sia-sia. Apakah ada perangkat lunak yang benar-benar akan memformat thumb-stick saya sehingga saya hanya memiliki satu partisi?

MENGHAPUS GAMBAR

  2. buka / jalankan DISKMGMT.MSC untuk membuat partisi primer, Sistem File apa pun akan baik-baik saja untuk saat ini

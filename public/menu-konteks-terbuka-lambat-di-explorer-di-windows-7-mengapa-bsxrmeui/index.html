Menu konteks terbuka lambat di Explorer di Windows 7, mengapa?

Saya menjalankan Windows 7 di laptop saya yang cukup modern, ketika saya membuka menu konteks di Windows Explorer, perlu waktu untuk muncul (~ 10 detik). Ada beberapa program yang perintahnya ditambahkan ke dalamnya (utilitas manipulasi arsip, antivirus, sistem kontrol versi, dan semacamnya). Saya pikir salah satu dari mereka membekukan operasi.

Apakah ada alat patokan untuk mengukurnya entah bagaimana atau alat untuk mematikannya di Explorer tanpa mencopotnya (yang akan menjadi ukuran kedua dari belakang, karena menggunakannya)?

Itu tergantung pada kapan "menu klik-kanan" melambat seperti "Penangan Menu Konteks" yang harus disalahkan. Seperti yang mungkin Anda ketahui, menu klik kanan Anda berubah berdasarkan apa yang Anda klik kanan. Daftar berikut sesuai dengan apa yang Anda klik dan apa yang muncul di menu.

Deskripsi - Kunci Registri
File - HKCR * \ shellex \ contextmenuhandlers
File dan folder file - HKCR \ AllFileSystemObjects \ shellex \ contextmenuhandlers
Folder - HKCR \ Folder \ shellex \ contextmenuhandlers
Folder File - HKCR \ Directory \ shellex \ contextmenuhandlers
Kelas file - HKCR \\ shellex \ contextmenuhandlers
Desktop - HKCR \ Directory \ Background \ shellex \ ContextMenuHandlers

Tetapi Anda mungkin akan lebih mudah menggunakan editor yang dirancang untuk mengubah pengaturan ini daripada mengedit registri itu sendiri. Salah satu editor yang berguna adalah shexview , oleh Nirsoft.

http://www.nirsoft.net/utils/shexview.html

Anda akan ingin mengurutkan kolom "Jenis" atau "Ekstensi file".

Kalau tidak, autoruns.exe akan memungkinkan Anda mengedit informasi ini juga. http://technet.microsoft.com/en-us/sysinternals/bb963902.aspx

Aplikasi ini lebih baik diatur karena akan mengelompokkan penangan berdasarkan lokasi registri (tabel yang diposting di atas).

Ini memungkinkan Anda untuk mengedit lebih dari sekadar ContextMenuHandlers. Tab terkait yang Anda minati adalah "Explorer". Gulir ke bawah hingga Anda melihat lokasi registri berperilaku buruk yang benar, dan batalkan pilihan satu per satu hingga masalahnya hilang.

Catatan: Kedua program itu portabel (tidak perlu diinstal). Simpan keduanya dalam flash drive dan bantu orang lain.

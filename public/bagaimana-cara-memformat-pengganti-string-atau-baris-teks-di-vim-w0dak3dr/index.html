Bagaimana cara memformat pengganti string, atau baris teks, di Vim?

Apakah mungkin untuk memformat string pengganti saat mengganti pertandingan dengan Vim?

Katakanlah saya memiliki file dengan tabel 2 kolom, di mana entri di setiap baris dipisahkan dengan spasi. Entri (dan garis) tidak memiliki lebar tetap. Saya ingin memformat tabel sedemikian rupa sehingga item kolom ke-2 semuanya rata, dengan menambahkan jumlah spasi yang sesuai.

Misalnya, tabel yang diberikan:

A aa
BBBB bbb
CC c

Apa yang saya inginkan:

A      aa
BBBB   bbb
CC     c

(Harap perhatikan bahwa tab bukanlah solusi di sini.)

Perintah Vim mungkin terlihat seperti ini:

:0,$ s/\(.*\) \(.*\)/\1(******N)\2/

dan (******N) harus diganti dengan kode pemformatan untuk meletakkan elemen ke-2 pada posisi karakter ke-N.

PEMBARUAN 1 : Jadi, Vim memiliki fungsi printf(). Bisakah itu digunakan langsung dari command prompt untuk tujuan itu, atau apakah itu harus disertakan dalam skrip Vim, dengan loop eksplisit berjalan melalui semua baris?

Anda mungkin ingin mencoba salah satu plugin align di vim.org. Saya menggunakan yang paling populer oleh Dr.Campbell: http://www.vim.org/scripts/script.php?script_id=294

Selain itu, ya, Anda dapat memanggil fungsi Vim dari perintah pengganti, tetapi kekurangannya adalah ekstensi seluruh pola pengganti harus berupa ekspresi. Lihat ":help sub-replace-expression" dan ":help submatch()". Misalnya, jika kita ingin mengganti pola yang cocok dengan jumlah huruf:

:%s/pattern/\=len(submatch(0))/g

Jadi ide Anda menggunakan printf () akan berhasil:

:%s/\(.*\) \(.*\)/\=printf('%-7s %-7s',submatch(1),submatch(2))/

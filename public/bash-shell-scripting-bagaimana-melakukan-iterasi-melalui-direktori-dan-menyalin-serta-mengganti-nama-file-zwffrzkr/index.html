Bash Shell Scripting - Bagaimana melakukan iterasi melalui direktori, dan menyalin serta mengganti nama file?

Saya memiliki pengaturan direktori sebagai berikut:

/hosted/partner1/logo.png
/hosted/partner2/logo.png
/hosted/partner3/logo.png
/hosted/partner4/logo.png
/hosted/partner5/logo.png
..etc.

Saya ingin menulis skrip yang dapat MENYALIN file-file itu ke lokasi yang berbeda, dengan nama file yang berbeda, seperti ini:

/partners/partner1.png
/partners/partner2.png
/partners/partner3.png
..etc.

Ada ide? Saya tidak begitu hebat dengan skrip shell dan ada banyak file yang perlu saya pindahkan ke satu direktori ...

1 Jawaban

7
7
find /hosted -maxdepth 1 -name "partner*" -type d | while read -r dir
do
    cp "${dir}/logo.png" "/partners/$(basename ${dir}).png"
done

Atau

find /hosted -maxdepth 1 -name "partner*" -type d | while read -r dir
do
    cp "${dir}/logo.png" "/partners/${dir##*/}.png"
done

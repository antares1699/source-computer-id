Cara mengidentifikasi tab horizontal di Excel dan menggantinya

Saya memiliki ribuan data di file excel saya, itu adalah data scrap dari berbagai situs web, awalnya disimpan di file CSV kemudian dimigrasi ke Excel. Data ini untuk penggunaan server web saya.

Sekarang masalah saya adalah hal itu horizontal tabs tersembunyi di suatu tempat di antara banyak orang spaces , dan menyebabkan masalah kritis untuk server web saya.

Saya akan memberikan contoh untuk diperjelas di sini:

Data di Excel : "Ini adalah sel"

Data saat dikodekan untuk digunakan sebagai parameter url : "Ini% 20is% 09a% 20cell"

Anda dapat dengan jelas melihat bahwa salah satu spasi di antaranya telah dikodekan menjadi% 09, yang mewakili a tab horizontal . Ini menyebabkan masalah yang parah dari server saya dan saya PERLU mengidentifikasi dan menggantinya.

Jadi, bagaimana saya akan mencari file tab horizontal di Excel?

Jawaban Burgi berhasil, tetapi alih-alih harus menduplikasi seluruh dataset dengan lembar kerja yang penuh dengan rumus, Anda dapat mencapai hal yang sama melalui beberapa baris skrip.

Sub ReplaceTabs()
   Dim c As Range

   For Each c In ActiveCell.CurrentRegion
      c.Value = Replace(c.Value2, Chr(9), Chr(32))
   Next c
End Sub

Ada beberapa metode untuk melangkah melalui semua sel yang relevan (misalnya Anda dapat menargetkan kolom tertentu dengan ActiveSheet.Range("A:A") dll., Atau memilih sel yang dimaksud dan menggunakan Selection.Cells) tetapi metode ini akan berfungsi jika selnya berdekatan dan tabnya mungkin di sel mana pun.

Ini tidak super cepat (200.000 sel membutuhkan waktu sekitar 10 detik pada mesin saya) tetapi seharusnya cukup cepat.

Jika ini adalah tugas umum, Anda dapat membuatnya menjadi skrip impor.

Kode ini seharusnya bekerja untuk Anda:

=SUBSTITUTE(A1,CHAR(9),CHAR(32))

Rumus di atas menggantikan semua contoh CHAR(9) dengan CHAR(32) dalam string yang diberikan.

  • CHAR(9) - Karakter tab
  • CHAR(32) - Karakter spasi

Lihat ini artikel wiki untuk tabel pencarian yang digunakan Windows.

Layar GNU: jalankan perintah sewenang-wenang di bilah status?

Apakah mungkin membuat Layar GNU menampilkan output dari perintah sewenang-wenang, misalnya who | wc -l, di bilah status, menyegarkannya, katakanlah, setiap menit?

Ya, Anda dapat menggunakan fitur backtick. Backtick

Anda meletakkan baris di file screenrc Anda:

backtick 1 0 5 /bin/date

Jika saya mengerti dengan benar, itu berarti backtick id 1, yang menjalankan perintah /bin/date, menghasilkan output yang valid selama 0 detik dan caption / hardstatus harus diperbarui setiap 5 detik.

Sekarang, Anda dapat menggunakan string %1` di dalam status keras Anda untuk substitusi, misalnya

hardstatus alwayslastline "date output: %1`"

Dalam kasus Anda, saya pikir Anda harus menulis skrip. Misalnya, $HOME/my_script.sh:

#!/bin/bash
who | wc -l

Di ~/.screenrc Anda:

backtick 1 0 60 $HOME/my_script.sh
hardstatus alwayslastline "who|wc -l output: %1`"

Jelas Anda tidak akan menginginkan hal itu, tetapi tambahkan saja %1` ke apa pun yang Anda miliki. Perhatikan bahwa keluaran skrip Anda tidak dapat menyertakan pelarian string untuk layar yang akan diperluas, mis. jika skrip Anda echo "%{bg}blue on green%{dd}" itu tidak akan diwarnai, Anda hanya akan melihat ${bg} dan ${dd} di baris status Anda.

navigasi vim dengan penggalan baris dengan membungkus aktif

Bagaimana saya bisa menavigasi dengan jeda baris daripada dengan jeda keras saat bungkus diaktifkan, sehingga ketika saya turun baris saya benar-benar turun baris yang saya lihat daripada baris sumber di file? Ini akan sangat berguna saat mengedit baris yang sangat panjang, seperti yang keluar saat saya mengedit file TeX.

Anda dapat mengubah perilaku default j dan k menambahkan baris berikut ke file .vimrc Anda:

nmap <silent> j gj
nmap <silent> k gk

Lihat juga vim tip 308

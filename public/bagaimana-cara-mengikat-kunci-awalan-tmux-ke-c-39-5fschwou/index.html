Bagaimana cara mengikat kunci awalan tmux ke C- '?

Saya ingin menyetel kunci awalan tmux saya ke Ctrl-apostrof, tetapi ketika saya menggunakan set-option prefix "C-'" atau set-option prefix C-"'", tmux mengeluh bahwa "kunci tidak dikenal: C- '". Demikian pula untuk `dan:. Apakah ada cara untuk mengikat kunci ini?

Sebagian besar emulator terminal hanya menghasilkan keluaran khusus untuk tombol set yang dibatasi bila digunakan bersama dengan tombol pengubah Kontrol. Kunci apostrof adalah salah satu kunci yang berfungsi tidak menghasilkan urutan keluaran khusus (itu eter tidak melakukan apa pun atau hanya mengeluarkan tanda kutip biasa).

Lihat " Karakter kontrol ASCII "Dan" Bagaimana karakter kontrol dipetakan ke keyboard ".

Anda dapat memeriksa untuk melihat apakah emulator terminal Anda menghasilkan keluaran untuk Control-', dengan mengetikkannya ke cat -v, atau pada prompt shell (di shell paling umum) setelah mengetik Control-V.

Jika itu menghasilkan beberapa keluaran khusus (tidak hanya '), maka Anda mungkin dapat mengikatnya tmux jika kode / urutan kontrol diketahui tmux . Daftar nama kunci itu tmux mengenali di halaman manual di bawah bagian "Key Bindings". tmux hanya akan mengenali kunci khusus jika entri terminfo untuk TERM Anda di luar tmux mendefinisikannya (lihat man terminfo untuk nama “kemampuan” dan infocmp untuk nama yang ditentukan JANGKA Anda); jika perlu, Anda dapat memperpanjang entri terminfo sistem Anda hanya untuk tmux dengan variabel konfigurasi terminal-overrides (atau kompilasi ulang entri kustom Anda sendiri dengan tic).

Jika tidak menghasilkan keluaran khusus maka Anda harus mengkonfigurasi ulang untuk melakukannya (jika memungkinkan); Anda mungkin membuatnya menghasilkan kode yang sama seperti ^], atau urutan untuk "F20" (jika ditentukan dalam entri TERM Anda saat di luar tmux ).

  • Saya akhirnya beralih dari gnome-terminal ke xterm, jadi sekarang saya tahu bagaimana untuk memetakan ulang Ctrl + '(sumber daya terjemahan). Namun, dengan menyetel sumber daya XTerm * modifyOtherKeys: 1, saya bisa mendapatkan Ctrl- 'untuk menghasilkan "^ [[27; 5; 39 ~" alih-alih apostrof. Sekarang saya bertanya-tanya apakah ada cara agar tmux mengenali itu, meskipun tidak muncul di bawah infocmp -L xterm-256color ...
    — Jeffrey Yasskin
  • Anda dapat menggunakan terminal-overrides untuk mengubah bit entri terminfo itu tmux kegunaan. Misalnya: set -g terminal-overrides "*:kf20=\e[27;5;39~" dan set -g prefix F20
    — Chris Johnsen

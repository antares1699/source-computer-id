Apakah GPU aktif jika tidak ada monitor?

Hari ini saya mengubah PC lama menjadi semacam "server", yang berarti saya ingin membiarkannya berjalan 24/7. Saya tidak memerlukan tampilan apa pun karena saya akan menggunakan mesin melalui ssh jadi ketika semuanya sudah diatur, saya melepas monitor saya. Setelah satu jam saya memeriksa sistem pendingin kartu grafis saya dan masih panas.

Kartu grafis saya adalah GeForce 8600 (dengan konektor DVI), OS adalah Debian Linux.

Apakah GPU merender sesuatu saat tidak ada monitor yang dicolokkan?

Apa solusi terbaik dalam situasi ini (server mandiri) jika GPU aktif dan saya tidak ingin membuang-buang daya?

GPU akan tetap aktif dan kemungkinan besar merender konsol atau (kemungkinan besar) tampilan X. Jika pengelola jendela pengomposisian seperti Compiz sedang digunakan, GPU akan berada dalam mode 3D, yang mungkin menjadi alasan mengapa GPU menjadi panas.

Pastikan sistem tidak disetel untuk memulai X server, yang merupakan platform tempat sebagian besar aplikasi grafis berjalan. Coba boot komputer ke runlevel 3, sehingga X tidak mulai; Lihat artikel Wikipedia ini untuk lebih jelasnya.

Jika ini tidak berhasil, Anda harus menggunakan kartu video berdaya rendah, seperti file GeForce GT 610 .

Edit: Anda mengatakan bahwa tidak ada GUI yang berjalan. Nonaktifkan pengaturan mode kernel dan setel resolusi sistem ke VGA dengan menggunakan parameter boot kernel nomodeset vga=0x101, sehingga kernel tidak menyebabkan GPU untuk berjalan pada tingkat daya yang lebih tinggi dari yang diperlukan dengan menggunakan resolusi konsol yang lebih tinggi dari yang diperlukan. Namun, mengingat usia 8600 GT, ini mungkin tidak berjalan efisien bahkan dalam mode teks; jika demikian, Anda harus mengganti kartu video.

Boot 'server' hanya dalam mode teks - tidak memiliki sesi GUI apa pun akan mencegah driver GPU dimuat dan menginitalisasi mesin 3D yang haus daya.

Atau, dan sepenuhnya bertentangan, boot dalam mode GUI terlepas dari beberapa konfigurasi monitor palsu, dan driver nvidia resmi - driver mungkin menginisialisasi gpu dan menurunkannya ke penghematan daya.

Cobalah kedua cara dengan mematikan-a-watt atau pengukur daya serupa.

Setel opsi daya ke 'matikan monitor' setelah jangka waktu tertentu. Hanya itu yang dapat Anda lakukan selain menghapus subsistem grafis (untuk konsol). Selama ada adaptor grafis di sana, itu akan memakan beberapa kekuatan hanya untuk tetap aktif, dan mungkin akan menjadi 'panas' bahkan saat idle.

Ini akan tergantung pada BIOS motherboard spesifik Anda apakah Anda dapat sepenuhnya melepaskan adaptor grafis dan masih bisa boot. Ini BUKAN opsi biasa untuk motherboard bergaya desktop.

Info lebih lanjut

Anda mungkin dapat mencoba dan mendapatkan adaptor video yang kurang kaya fitur untuk digunakan, tetapi tidak ada yang tahu apakah itu benar-benar akan lebih efisien atau tidak, kecuali diuji.

Output konversi Imagemagick kecil ketika format halaman ditentukan

Saya memiliki 2 JPEG ini dan saya menggabungkannya menjadi satu pdf dengan perintah berikut:

convert -compress JPEG -quality 90 -page A4 1.jpg 2.jpg doc.pdf

Namun ukuran dokumen yang dihasilkan hanya 50x71 mm, Anda bisa melihatnya pada gambar berikut. Ini terjadi dengan setiap format -page. Ini berfungsi dengan baik jika saya tidak menentukan -page.

(Tidak, tangkapan ini tidak diperkecil agar sesuai dengan pertanyaan).

Saya tidak tahu mengapa ini terjadi, tetapi ternyata saya juga harus menentukan kepadatan piksel (DPI) menggunakan perintah -density:

convert -compress JPEG -quality 90 -page A4 -density 72 1.jpg 2.jpg doc.pdf

Saya harus memilih 72 karena itulah satu-satunya kerapatan yang memberi saya ukuran A4 yang tepat (210x297). Menentukan 300 secara manual akan memberi saya dokumen kecil yang sama: 300 adalah DPI yang saya gunakan untuk memindai gambar saya.

  • Bekerja untuk file masukan jpg, tetapi tidak untuk png. Untuk png saya juga harus menentukan -units PixelsPerInch.
    — Socowi
  • PDF menggunakan 1/72 inci sebagai unit internal. Menentukan -density 72 tidak berarti Anda mendapatkan PDF resolusi sangat rendah. Resolusi hanya diperlukan untuk menghitung ukuran A4 yang benar dalam unit PDF. Gambar di dalam PDF dapat memiliki resolusi yang lebih tinggi.
    — Socowi

Tetapkan klik kiri mouse ke tombol keyboard

Saya ingin dapat menetapkan tombol pada keyboard saya yang setara dengan klik kiri mouse.

Idealnya, tombol ini perlu bertindak sedemikian rupa sehingga menahan tombol sama dengan menahan tombol kiri mouse.

Bagaimana saya bisa melakukan ini?

Anda dapat menggunakan fitur Tombol Mouse OS X untuk menggunakan keyboard untuk mengklik mouse. (Ini adalah fitur aksesibilitas.) Anda dapat mengaktifkan pintasan keyboard untuk menyalakan dan mematikannya jika Anda tidak ingin menggunakan fitur tersebut sepanjang waktu.

Dengan Tombol Mouse, cukup gunakan angka 5 (atau tombol huruf setara I) untuk mengklik. Menekan 0 atau M akan berfungsi untuk menahan tombol mouse, dan titik akan terlepas.

https://support.apple.com/en-us/HT204434#mousekeys

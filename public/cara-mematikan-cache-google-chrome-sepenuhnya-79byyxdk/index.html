Bagian Alat Pengembang memiliki opsi untuk menonaktifkan cache. Di jendela alat pengembang, klik ikon roda gigi di pojok kanan bawah, dan di bawah Jaringan, centang "nonaktifkan cache".

Saya menemukan ini di versi 14.0.835.35 beta.

Menonaktifkan cache di Alat Pengembang hanya menonaktifkan cache saat DevTools terbuka. Meskipun ini mungkin yang dicari sebagian orang, Anda dapat menonaktifkan cache sepenuhnya dengan:

1) memasuki prompt perintah
2) masukkan "cd C: \ Program Files (x86) \ Google \ Chrome \ Application" (untuk mengubah direktori tempat Google Chrome disimpan)
3) masukkan "chrome.exe --media-cache-size = 1 --disk-cache-size = 1" untuk memperkecil ukuran kedua cache, secara efektif menonaktifkan cache

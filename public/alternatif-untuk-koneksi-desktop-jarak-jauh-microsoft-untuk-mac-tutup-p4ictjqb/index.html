Alternatif untuk Koneksi Desktop Jarak Jauh Microsoft untuk Mac? [Tutup]

Saat bekerja dari Windows, saya lebih suka menggunakan RoyalTS alih-alih klien Microsoft RDP default, karena memungkinkan saya untuk menyimpan banyak koneksi, kredensial koneksi, dll; serta memiliki lebih dari 1 koneksi yang terbuka pada satu waktu.

Apakah ada alternatif yang serupa (atau apa saja) untuk file Klien Microsoft RDP untuk menghubungkan dari OSX ke Windows? Selain VNC, yaitu; pada titik ini saya berharap untuk terus menggunakan protokol RDP.

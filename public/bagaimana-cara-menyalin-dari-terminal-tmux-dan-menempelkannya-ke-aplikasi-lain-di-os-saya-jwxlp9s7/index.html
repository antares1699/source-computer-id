Tmux secara default akan "mengambil alih" pemilihan teks dengan mouse, dan memicu seleksi internalnya sendiri untuk digunakan dengan cut and paste-nya sendiri.

Jika Anda ingin menggunakan pemilihan OS, sehingga Anda dapat menggunakan OS potong dan tempel, tahan bergeser saat memilih.

Kemudian potong dan tempel akan bekerja secara normal.

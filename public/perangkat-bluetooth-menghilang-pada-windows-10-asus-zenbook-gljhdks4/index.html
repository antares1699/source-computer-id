Perangkat Bluetooth menghilang pada Windows 10 Asus ZenBook

Saya punya Asus Zenbook UX331FAL menjalankan Windows 10.

Hingga sekitar beberapa minggu yang lalu, saya dapat menghubungkan perangkat sebanyak yang saya inginkan menggunakan Bluetooth (mouse, headset, keyboard), dan tidak ada masalah.

Namun, selama 2 minggu terakhir setiap kali saya mencoba menambahkan perangkat Bluetooth baru, inilah yang saya lihat:

Ini kira-kira pada waktu yang sama setiap kali saya mencolokkan jack headphone saya, saya tidak mendapatkan audio di headphone saya (driver audio RealTek), meskipun speaker tampaknya berfungsi dengan baik. Tidak akan membahas masalah ini pada pertanyaan ini, tetapi keduanya mungkin berhubungan.

Memperbarui : Masalah audio telah teratasi dengan sendirinya, dan sekarang saya bisa mendapatkan audio ke headphone saya, setelah semua memperbarui dan memulai ulang. Jadi kemenangan kecil di sana.

Jika itu adalah pembaruan Windows yang menyebabkan driver Bluetooth Anda berhenti berfungsi, maka Anda harus mempertimbangkan untuk memulihkan Windows ke versi sebelumnya di mana Anda tahu pasti bahwa Bluetooth berfungsi normal. Anda dapat melihat opsi pemulihan untuk memulihkan versi Windows sebelumnya sini , dan jika Anda akan mengikuti rute ini, pastikan Anda mencadangkan file penting yang mungkin telah berubah sejak titik ini.

Jika itu berhasil memperbaiki masalah Bluetooth dan semuanya berfungsi normal, maka Anda bisa sementara nonaktifkan pembaruan Windows otomatis melalui registri hingga Anda yakin bahwa memperbarui Windows lagi tidak akan merusak Bluetooth Anda. Jika Anda akan memperbarui lagi saat ini, Anda mungkin ingin memeriksanya membuat titik pemulihan sistem untuk lebih mudah mengembalikan status Windows sebelumnya, jika ada yang rusak lagi di masa mendatang setelah memperbarui Windows.

Semoga Anda bisa memperbaikinya, karena ini sepertinya masalah yang menjengkelkan untuk ditangani!

Saya memiliki masalah yang sama persis dengan Pembaruan Windows 20H2 terbaru. Sayangnya solusi yang disarankan tidak berfungsi untuk saya karena windows tidak / tidak memberikan perbaikan untuk masalah ini pada saat penulisan ini. Untungnya saya dapat memperbaiki masalah ini berdasarkan video ini di YouTube :

  • Buka pengelola perangkat
  • Copot pemasangan seluruh "Intel (R) USB 3.1 eXtensible Host Controller"
    • Ini juga akan menghapus perangkat lain (termasuk "Perangkat USB Tidak Dikenal (Permintaan Deskriptor Perangkat Gagal)")
  • Buka menu start windows dan mematikan komputer (a "restart" tidak berhasil untuk saya dan orang lain yang mengomentari video YouTube)
  • Mulai komputer dengan tombol daya
    • Komputer sekarang akan menginstal eXtensible Host Controller dan akhirnya akan mengenali dan menginstal driver Bluetooth

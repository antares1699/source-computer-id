Bagaimana cara membuka Internet Explorer untuk desktop dari baris perintah di Windows 8?

Ada dua versi Internet Explorer yang berbeda di Windows 8. Saya ingin membuka Internet Explorer untuk desktop dari baris perintah.

Kita bisa menjalankannya melalui Menang + R → iexplore, tetapi saya ingin menjalankannya dari command prompt atau PowerShell. Bisakah itu dilakukan?

4
Atilla Ozgur — Sumber
  • Untuk baris perintah normal cukup gunakan "C: \ Program Files \ Internet Explorer \ iexplore.exe" untuk 64bit atau "C: \ Program Files (x86) \ Internet Explorer \ iexplore.exe" untuk 32bit
    — nixda

Menggunakan PowerShell, saya menemukannya Invoke-Item adalah perintah kunci:

Invoke-Item "C:\Program Files\Internet Explorer\iexplore.exe"

Faktor lain:

  1. Perintah hanya berfungsi untuk saya ketika saya memberikan path lengkap ke iexplore.exe.

  2. Anda mungkin mengalami masalah eksekusi skrip jika Anda menyimpan instruksi dan menjalankan dari file .ps1. Dalam kasus tersebut, jalankan yang berikut ini dari prompt PowerShell yang ditinggikan:

    Set-ExecutionPolicy RemoteSigned
Bagikan

  • Perhatikan yakin jika Start-Process tidak akan lebih baik; sekali lagi kita membutuhkan path lengkap: Start-Process "C: \ Program Files \ Internet Explorer \ iexplore.exe"

Metode yang dicoba dan benar di setiap versi Windows yang relevan:

iexplore.exe google.com

akan membuka IE dan menavigasi ke google. Saya menggunakan invoke-item untuk beberapa hal, tetapi ini dapat dilakukan dari kotak run (yang dinyatakan OP) yang menghemat penekanan tombol. Ia juga bekerja di cmd dan PowerShell, jalur b / c iexplore ada di variabel env PATH.

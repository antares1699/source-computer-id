Bagaimana cara mengaktifkan waktu tunggu boot di Grub2?

Saya mencoba mengaktifkan pilihan batas waktu boot di grub2 pada Ubuntu 9.10.

Saya memodifikasi / etc / default / grub:

GRUB_DEFAULT=0
#GRUB_HIDDEN_TIMEOUT=0
GRUB_HIDDEN_TIMEOUT_QUIET=false
GRUB_TIMEOUT=2
GRUB_DISTRIBUTOR=`lsb_release -i -s 2> /dev/null || echo Debian`
GRUB_CMDLINE_LINUX_DEFAULT="splash quiet"
GRUB_CMDLINE_LINUX=""

dan menjalankan update-grub, tetapi saya masih tidak memiliki penghitung batas waktu booting. Apa lagi yang dapat Anda lakukan untuk mengaktifkan ini?

Jawaban diambil dari http://ubuntuforums.org/showthread.php?t=1373965

di akhir file /etc/grub.d/00_header Anda, beri komentar jika kondisi kecuali untuk baris batas waktu yang ditetapkan seperti ini:

#if [ \${recordfail} = 1 ]; then
#  set timeout=-1
#else
  set timeout=${GRUB_TIMEOUT}
#fi

jalankan update-grub untuk meregenerasi grub.cfg dengan pengaturan baru:

sudo update-grub

reboot, sekarang harus selalu boot opsi default setelah batas waktu, bahkan jika boot sebelumnya gagal.

Masalahnya terkait dengan nilai grub env: recordfail. Saya dapat menyelesaikannya dengan perintah berikut:

grub-editenv /boot/grub/grubenv unset recordfail

Opsi lain yang akan menjamin batas waktu akan selalu berfungsi:

rm /boot/grub/grubenv

Jika Anda tidak melakukan ini, grub mungkin akan menulis ulang nilai recordfail. Saya membutuhkan ini untuk sistem tertanam.

Opsi yang sedikit kurang invasif untuk mendapatkan hasil yang sangat mirip dengan jawaban sebelumnya adalah dengan menambahkan baris berikut ke /etc/default/grub Anda:

GRUB_RECORDFAIL_TIMEOUT=5

... atau berapa detik yang Anda inginkan. Mungkin berbahaya untuk menyetelnya ke 0, saya tidak yakin.

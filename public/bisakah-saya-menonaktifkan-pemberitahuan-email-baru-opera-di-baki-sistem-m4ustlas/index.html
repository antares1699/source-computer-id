Bisakah saya menonaktifkan pemberitahuan email baru Opera di baki sistem?

Saya menggunakan Opera sebagai klien email saya. Ketika Opera menemukan beberapa email baru, Opera menambahkan amplop kecil ke ikon baki sistemnya.

Itu menggangguku. Bagaimana cara menonaktifkan pemberitahuan ini?

Ini adalah pengaturan dalam preferensi:

  1. Ctrl + F12 atau Alt + P.

  2. Pergi ke Maju tab, lalu Notifikasi .

  3. Hapus centang Tampilkan pemberitahuan untuk pesan baru

Jika Anda menggunakan Opera sebagai pembaca RSS, pemberitahuan tentang umpan baru yang segar juga tidak akan ditampilkan.

Di tempat lain...

Anda mungkin juga perlu mengedit pintasan yang digunakan untuk meluncurkan Opera.

Untuk Linux , temukan peluncur dan edit propertinya untuk menambahkan -notrayicon, seperti ini:

opera% u -notrayicon

Windows , cari pintasannya, klik kanan pada Properties dan tambahkan baris target dengan /notrayicon sehingga akan terbaca seperti ini:

"C: \ Program Files \ Opera \ opera.exe" / notrayicon

Selain itu, Anda mungkin perlu melakukan hal berikut:

  • Klik kanan area kosong di bilah tugas
  • Pilih Properties
  • Pilih tab Taskbar jika belum dipilih
  • Sesuaikan ... dari Area Pemberitahuan
  • Temukan ikon Opera dan pilih "Selalu Sembunyikan" untuk perilakunya

Setelah selesai dengan yang di atas, tutup Opera dan mulai lagi.

Ikon Opera sekarang tidak akan muncul di system tray.

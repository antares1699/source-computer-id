Kecualikan situs dari kebijakan Tampilan Kesesuaian

Saya memiliki klien yang memberlakukan kebijakan di IE bahwa semua situs intranet lokal dimuat di Tampilan Kompatibilitas.

Sayangnya aplikasi web kami tidak mendukung ini dan memberikan banyak masalah di comp. melihat. Adakah cara untuk secara khusus mengecualikan situs dari kebijakan ini tanpa harus memasukkannya ke dalam kode kami?

Untuk memberi tahu Internet Explorer agar tidak menggunakan Mode Kompatibilitas di situs Anda, gunakan tag meta berikut (atau variasinya) di bagian head situs Anda:

<meta http-equiv="X-UA-Compatible" content="IE=9; IE=8; IE=EDGE" />

Perhatikan bahwa tag ini harus jadilah tag pertama di dalam tag head. Untuk informasi lebih lanjut, kunjungi: Menentukan mode dokumen warisan

Di ASP.Net Anda juga dapat memasukkan yang berikut ini ke file web.config situs web:

<system.webServer>
<httpProtocol>
    <customHeaders>
        <clear/>
        <add name="X-UA-Compatible" value="IE=8; IE=9; IE=EDGE" />
    </customHeaders>
</httpProtocol>

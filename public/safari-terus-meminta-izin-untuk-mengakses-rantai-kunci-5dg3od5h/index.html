Safari terus meminta izin untuk mengakses rantai kunci

Biasanya ketika saya menyimpan kata sandi di Safari, kata sandi itu akan ditambahkan ke rantai kunci masuk saya tanpa keributan (dengan asumsi bahwa itu sudah dibuka kuncinya). Tetapi setelah saya menyetel kata sandi master, rantai kunci default diubah menjadi FileVaultMaster. Ketika saya menyetelnya kembali ke login Safari kemudian mulai selalu meminta izin untuk mengakses rantai kunci.

Untuk mengembalikannya ke perilaku default, haruskah saya memilih always allow atau apakah ada hal lain yang harus saya lakukan?

Anda memiliki dua masalah potensial. Yang pertama adalah rantai kunci default Anda tidak lagi disetel ke "login". Buka Akses Rantai Kunci (/Applications/Utilities/Keychain Access.app), pilih rantai kunci masuk, dan pilih File »Make Keychain "login" Default.

Masalah potensial kedua adalah gantungan kunci yang rusak. Dalam Akses Rantai Kunci, masih dengan rantai kunci masuk yang dipilih, pilih Keychain Access »Keychain First Aid. Masukkan kata sandi Anda dan tekan perbaiki.

codesign --verify --verbose /Applications/Safari.app

... yang seharusnya memberi Anda:

/Applications/Safari.app: valid on disk
/Applications/Safari.app: satisfies its Designated Requirement

Saya mengalami masalah ini dengan iMac 2010 Bibi saya; dia lupa kata sandi utama dan saya melakukan reset menggunakan disk instalasi. Namun, sejak itu dia diganggu oleh masalah yang sama yang dijelaskan di atas, permintaan kata sandi yang berulang-ulang. Saya akhirnya memiliki kesempatan untuk pergi ke bar Apple Genius setelah menghabiskan semua forum diskusi tanpa hasil dan diberi solusi yang belum terbukti. Pria itu mengatakan bahwa dia mendapatkan masalah ini sepanjang waktu dan dia cukup yakin itu akan berhasil.

Sebagai berikut:

  1. Tutup semua program kecuali Finder dan masuk ke Home / Library (bisa juga Username / Library)
  2. Temukan folder Gantungan Kunci di sini; Hapus
  3. Keluar / Mulai ulang segera
  4. Keychain akan membuat ulang folder tersebut dan Anda sebaiknya melakukannya

Saya akan memperbarui posting ini saat mencoba memperbaiki ini; kedengarannya logis. Siapapun yang telah sukses, jangan ragu untuk ikut campur. Saya juga harus menunjukkan bahwa ini jelas akan menghapus semua aplikasi / sandi web yang telah Anda simpan tetapi menurut saya itu tidak akan menjadi masalah bagi kebanyakan orang.

Pembaruan: Itu berhasil, sejauh ini bagus! Tidak percaya solusinya sangat sederhana!

Mungkin juga tidak sinkron dengan akun login Anda. Secara default, pertolongan pertama rantai kunci biasa tidak akan memperbaikinya. Untuk memeriksanya dan memperbaikinya, buka Keychain Access.app, klik menu Keychain Access, dan buka preferensi. Tekan tab 'Pertolongan Pertama' dan centang kotak 'Sinkronkan kata sandi rantai kunci masuk dengan akun'.

Kemudian jalankan pertolongan pertama gantungan kunci lagi.

Ekstensi Chrome untuk memfokuskan bidang masukan pertama?

Saya mencari ini Ekstensi Firefox disebut FirstField, tetapi untuk Chrome. Pada dasarnya, Anda menekan pintasan keyboard dan ini memfokuskan bidang input teks pertama pada halaman web (misalnya, pencarian Wikipedia). Seberapa sulit membuat ini?

Pencarian cepat untuk fokus di Toko web Chrome diangkat Fokus Input . Sepertinya itu akan melakukan apa yang Anda inginkan, tekan Ctrl + Alt + L untuk mengatur fokus pada bidang teks pertama halaman dan kemudian tekan kombo lagi untuk mulai berputar melalui bidang teks lain yang ada di halaman.

Jika Anda mencari sesuatu yang lebih komprehensif, saya sangat senang dengan Ekstensi Chrome Vimium yang sangat cair jika Anda mengetahui sedikit tentang VI (M), tetapi berguna meskipun tidak.

Jika Anda menginstal Vimium, Anda dapat mengambil kolom input pertama dengan mengetik gi

Anda kemudian dapat membuka tab ke bidang entri teks lainnya.

Alternatifnya, Anda dapat mengawali gi dengan angka, seperti pada 3gi, yang akan memulai Anda pada bidang entri ketiga yang terlihat.

Sebagai alternatif tambahan, Anda dapat mengetik f yang akan memberikan semua tautan & elemen yang dapat diklik (termasuk bidang entri teks) pada halaman perintah keyboard singkat (mis. MN). Anda kemudian dapat mengetikkan pukulan pendek ini untuk melompati fokus di sana.

Bagaimana cara menghapus pesan "Lisensi Kedaluwarsa" di VMWare Player?

Saya menggunakan VMWare Player untuk menjalankan beberapa mesin virtual. Baru-baru ini saya perlu menggunakan utilitas VMWare Virtual Disk Manager (untuk mengubah beberapa disk dinamis menjadi tetap), dan karena tidak disertakan dengan Player, saya menginstal versi evaluasi VMWare Workstation. Setelah saya selesai, saya menghapus Workstation dan menginstal ulang Player, tetapi ternyata Player mengira saya masih menggunakan lisensi evaluasi karena menunjukkan "Lisensi kedaluwarsa" di bilah judul dan pesan berikut di jendela utama:

Lisensi evaluasi Anda telah kedaluwarsa. Untuk melanjutkan penggunaan komersial produk ini, beli lisensi.

Bagaimana saya bisa kembali ke versi gratis-untuk-penggunaan pribadi dari VMWare Player? Saya sudah mencoba mencopot sepenuhnya semua produk VMWare dan menginstal ulang Player, tetapi itu tidak berhasil, pesan "Lisensi kedaluwarsa" tetap ada.

Larutan

Setelah mencopot pemasangan uji coba VMware Workstation, kunci lisensi lama mungkin tertinggal. Menghapus kunci registri terkait akan mengembalikan VMware Player kembali ke status asli, bebas-untuk-penggunaan pribadi.

Nama kunci registri yang sebenarnya bergantung pada nama produk dan versinya. Misalnya, VMware Workstation versi 10.0.1 menggunakan kunci yang disebut License.ws.10.0.e1.201303. Formatnya adalah sebagai berikut:

License.<product>.<major>.<minor>.<revision>.<date>

Lokasi registri akan berbeda tergantung pada apakah Windows itu 32-bit atau 64-bit . Masing-masing:

HKEY_LOCAL_MACHINE\SOFTWARE\VMware, Inc.\VMware Workstation
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\VMware, Inc.\VMware Workstation
  1. Tutup VMware Player.
  2. tekan Menang + R , ketik regedit.exe di kotak teks, dan tekan Memasukkan .
  3. Arahkan ke jalur yang benar yang dijelaskan di atas.
  4. Temukan kunci registri lisensi, klik kanan dan pilih Menghapus dari menu konteks.

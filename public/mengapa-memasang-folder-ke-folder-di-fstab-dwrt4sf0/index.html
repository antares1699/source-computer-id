Mengapa memasang folder ke folder di fstab?

Saya menggunakan server Ubuntu dengan fstab seperti ini:

/dev/xvds /home/ubuntu/stuff auto  defaults,nofail,nobootwait 0 2
/home/ubuntu/stuff /home/bob/things auto  defaults,nofail,nobootwait,bind 0 2

Saya pikir fstab adalah untuk memasang perangkat ke folder. Apa entri kedua? Sepertinya memasang folder ke folder, lebih seperti symlink.

Ya kamu benar. Catatan Fstab

/home/ubuntu/stuff /home/bob/things auto  defaults,nofail,nobootwait,bind 0 2

sama dengan perintah yang dikeluarkan dari shell Anda:

mount --bind /home/ubuntu/stuff /home/bob/things

Artinya Anda dapat mengakses ubuntu / staff konten dari bob / hal map.

Anda dapat membaca tentang perbedaan antara symbolic link dan bind mount di sini: https://askubuntu.com/questions/557733/ln-s-vs-mount-bind

  • @LonnieBest Anda dapat mengatur profil yang berbeda dengan skema berbeda yang terkait dengan masing-masing. Misalnya saya memiliki profil "biasa" dengan skema warna pribadi saya, profil "kerja" dengan sorotan ekstra pada keluaran penting, dan profil "root" yang dengan jelas mengingatkan saya bahwa saya menggunakan root. Saya pikir cara mengaturnya cukup intuitif, Preferences >> Profile >> add

Jika niat Anda adalah untuk instal file yang dapat dieksekusi permata ke ~/.gem/bin , Anda dapat mengatur di ~/.gemrc:

gem:
 --bindir ~/.gem/bin 
 # any other gem settings like: 
 --no-ri --no-rdoc

Ini TIDAK akan mengubah EXECUTABLE DIRECTORY, tetapi file executable permata akan diinstal ke ~/.gem/bin. Anda dapat memeriksa apakah sudah diatur oleh:

$ gem env
- GEM CONFIGURATION:
     - "gem" => "--bindir ~/.gem/bin --no-ri --no-rdoc"

Untuk dapat memanggilnya, tambahkan direktori itu di awal PATH Anda.

Anda dapat mencoba

$ export GEM_HOME = ~/.gem

Ini akan mengubah direktori gem Anda yang dapat dieksekusi menjadi ~/.gem/bin

Anda dapat mengonfirmasi itu dengan menjalankan $ gem env lagi

Jika berhasil, Anda dapat membuatnya permanen dengan menambahkan export GEM_HOME=$HOME/.gem di akhir file .bashrc atau .zshrc

Semoga bantuan itu!

Ini garis untuk .bashrc / .zshrc:

#Ruby + GEM
export GEM_HOME=$HOME/.gem
export PATH=$GEM_HOME/bin:$PATH

Setelah ini semua permata saya mulai bekerja.

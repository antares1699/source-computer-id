Ya, ada ekstensi untuk browser web Google Chrome yang memungkinkan Anda melakukan ini. Itu disebut Chrome Desktop Jarak Jauh . Anda dapat memasang ekstensi dari Toko Web Chrome.

https://chrome.google.com/webstore/detail/chrome-remote-desktop/gbchcmhmhahfdphkhkmpfmihenigjmpp?utm_source=chrome-ntp-icon

Ini akan memungkinkan Anda mengakses komputer lain dari jarak jauh melalui browser Chrome atau Chromebook. Komputer dapat tersedia untuk jangka pendek untuk skenario seperti dukungan jarak jauh ad hoc, atau dalam jangka panjang untuk akses jarak jauh ke aplikasi dan file Anda. Semua koneksi diamankan sepenuhnya.

Chrome Desktop Jarak Jauh sepenuhnya lintas platform. Memberikan bantuan jarak jauh untuk pengguna Windows, Mac dan Linux, atau mengakses desktop Windows (XP dan yang lebih baru) dan Mac (OS X 10.6 dan yang lebih baru) kapan saja, semuanya dari browser Chrome di hampir semua perangkat, termasuk Chromebook.

Diperbarui : Jika sekolah Anda tidak menggunakan peramban web Google Chrome, cara termudah adalah memasang versi portabel Google Chrome pada Stik USB di rumah http://portableapps.com/apps/internet/google_chrome_portable

Kemudian instal Ekstensi Chrome Desktop Jarak Jauh ke versi portabel

Sejujurnya, jika Anda tidak dapat menjalankan apa pun atau menginstal apa pun, Anda mungkin lebih beruntung menggunakan Remote Desktop yang sudah ada di dalam Windows.

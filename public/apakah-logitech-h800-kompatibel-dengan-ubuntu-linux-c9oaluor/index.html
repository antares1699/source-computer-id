Apakah Logitech H800 kompatibel dengan Ubuntu Linux?

Saya tertarik untuk membeli Logitech H800 headset nirkabel. Tetapi layanan pelanggan memberi tahu saya bahwa itu tidak kompatibel dengan Linux. Mereka bilang itu hanya kompatibel dengan Windows & Mac OS X.

Tapi menurut satu forum , saya t aku s kompatibel dengan Ubuntu.

Jadi, apakah Logitech H800 ini kompatibel dengan Linux? Apa prasyarat untuk menggunakannya di sistem saya? Jika kompatibel, apakah kinerjanya akan sebagus di Windows?

Bekerja dengan dongle USB dua arah: sebagai input dan output, yaitu speaker dan mikrofon. Jika menggunakan fitur bluetooth hanya berfungsi sebagai output (speaker) tetapi mic tidak terdeteksi.

Ketika saya mencolokkannya sebagai bluetooth, ia akan mendeteksinya sebagai H800 dengan benar, jika saya menggunakan dongle itu akan mendeteksi sebagai H600.

Fitur tombol (putar / jeda, bisu, volume) berfungsi, setidaknya menggunakan gnome.

Ini ada di OpenSuSE 13.1 dengan kernel diperbarui ke 3.17.

Jawaban singkatnya: berhasil.

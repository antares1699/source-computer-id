Windows 8: Pilih Apa yang Harus Dipertahankan

Saya telah mengunduh "Windows 8 Upgrade Assistant" dari Situs Web Microsoft Windows 8. Saya saat ini menggunakan Windows 7 sekarang.

Ketika saya menjalankan aplikasi, saya harus memilih salah satu dari opsi berikut yang ingin saya simpan:

  • Pengaturan Windows, File Pribadi, dan aplikasi
  • Hanya File Pribadi
  • Tidak ada

Jika saya memilih Tidak ada , apakah Gambar dan Dokumen akan dihapus? Mereka berada di drive yang berbeda (Misalnya: E:\Pictures\) dan saya telah mengatur jalurnya Properti Gambar dan Properti Dokumen

Penginstal Windows tidak akan menyentuh drive apa pun selain yang Anda pasang. Jadi jika Anda menginstal Windows di drive C, semua drive lain di sistem tidak akan terpengaruh.

Jadi, TIDAK, bahkan jika Anda memilih Nothing, Gambar dan Dokumen Anda yang terletak di drive selain yang Anda instal tidak akan dihapus.

Bagaimana cara meregangkan OS tamu di VMware?

Saya menjalankan beberapa OS di VMware. Saya ingin meregangkannya sehingga saya dapat menjalankannya pada layar penuh, tetapi hanya muncul di tengah. Hanya satu yang berfungsi dan pas dengan seluruh layar. Apa masalahnya? Mengapa ini berhasil untuk satu dan bukan yang lain?

Yang itu berfungsi karena OS tamunya telah menginstal VMware Tools dan merupakan OS yang didukung oleh fitur penyesuaian resolusi tamu otomatis VMware Player. Jika Anda menginstal VMware Tools di VM lain, VM tersebut akan dapat memenuhi seluruh layar. Ini dapat dilakukan dengan memilih "Instal VMware Tools" dari menu VM; gambar CD dari disk instalasi VMware Tools akan dimasukkan ke dalam drive CD virtual.

Jika OS tamu Anda tidak didukung oleh fitur VMware Player ini, tidak ada cara pihak pertama untuk memaksa layar tamu diskalakan ke host. (VMware Workstation memang memiliki fitur seperti itu, tetapi tidak mempertahankan rasio aspek tamu.) Lihat postingan Komunitas VMware ini untuk panduan dalam menggunakan Pengubah Tampilan untuk mencapai efek yang serupa dengan yang Anda inginkan.

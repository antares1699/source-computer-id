Vim dapat dengan sempurna membuka file yang di-gzip secara default, lihat :help gzip.

Jika tidak memungkinkan dalam kasus Anda, Anda harus melihat:

  • jika mesin memiliki program gzip,
  • jika Anda memiliki izin yang tepat untuk menggunakannya,
  • jika Vim Anda dilengkapi dengan plugin standar gzip yang seharusnya berada di /usr/share/vim/vim*/plugin/gzip.vim dan pendampingnya di /usr/share/vim/vim*/autoload/gzip.vim .

Membaca file terkompresi secara transparan ditangani oleh plugin gzip vim. Plugin ini hanya diaktifkan jika vim tidak dalam mode "kompatibel" (lihat : bantuan yang kompatibel untuk lebih jelasnya). Cara mudah untuk memastikan "kompatibel" tidak disetel, adalah dengan membuat file .vimrc kosong di direktori home Anda.

Vim digunakan untuk mengedit teks. Itu tidak dapat membuka file zip.

Saya ingin melihat beberapa file log zip .gz di unix melalui dempul.

Untuk melihat konten file .gz di baris perintah, gunakan perintah ini.

 zless /path/to/file.gz

Jika zless tidak tersedia, gunakan gunzip dan less dengan pipa

gunzip -c /path/to/file.gz | less

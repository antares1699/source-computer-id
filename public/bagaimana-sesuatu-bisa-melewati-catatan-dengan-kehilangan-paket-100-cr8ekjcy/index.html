Ini bukan karena mesin kehilangan paket, melainkan karena tidak kembali bahwa paket. Traceroute bekerja dengan membatasi TTL pada paket UDP (kebanyakan OS 'traceroute) atau ICMP (dalam kasus Windows tracert), dan jika mesin dikonfigurasi untuk tidak mengembalikan pesan ICMP Time Exceeded, mesin Anda akan melihat ini sebagai paket kerugian. Jika tidak, mesin akan berfungsi seperti biasa.

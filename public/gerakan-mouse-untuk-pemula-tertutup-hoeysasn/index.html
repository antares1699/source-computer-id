Gerakan mouse untuk pemula [tertutup]

Pada Windows 3.11 memiliki tutorial yang bagus untuk pemula menggunakan mouse. Sejak itu saya menganggap tutor itu sebagai yang terbaik dari semua tutorial untuk mempelajari gerakan mouse, klik, klik dua kali, dll.

Apakah Anda tahu beberapa tutorial bagus (aplikasi) untuk pemula menggunakan mouse? Saya membutuhkannya untuk mengajar banyak orang yang membeli PC dengan saya ...

Preferensi saya adalah:

1) Multibahasa ATAU rendahnya penggunaan bahasa asli, sehingga aplikasi dapat bersifat universal;
2) Kemungkinan ketergantungan seminimal mungkin

Terima kasih untuk semua!

Saya tidak tahu aplikasi modern mana pun yang melakukannya, tetapi, untuk gerakan mouse..Saya ingin memberi Anda petunjuk: Saya ingat seseorang pernah mengatakan kepada saya suatu kali game yang terdapat dalam Windows dibuat untuk membantu pengguna memahami penggunaannya mouse. Ambil contoh solitarie. Ini mengajarkan Anda drag and drop dan satu klik. Untuk mengajarkan klik dua kali, panel kontrol, mouse, bagian tempat Anda mengatur waktu, ada kotak reaktif tempat Anda melatih klik dua kali.

Pengajaran cara menggunakan mouse bisa memakan waktu maksimal 4 jam, lalu Anda harus membiarkan pengguna berlatih sendiri.

Cara mengajar menggulir: jari di roda, dan putar sedikit ke bawah, sedikit lagi di sana, ini dia, sekarang Anda tahu cara menggulir ke bawah. Jika ingin naik, lakukan sebaliknya.

Semoga membantu.

Pergi ke Control Panel -> Power Options dan pilih Power saver

Itu seharusnya membatasi prosesor ke satu set kecepatan CPU bertenaga lebih rendah.

Tentu saja Anda selalu dapat menyesuaikan pengaturan dari rencana Power saver untuk hanya menggunakan keadaan minimal dengan memilih Edit Plan Settings lalu Change advanced power settings. Pada jendela yang terbuka cari Processor power management dan setel Maximum processor state ke 5% atau yang lebih rendah.

Di laptop, Anda harus dapat mengakses opsi daya dengan cepat dengan mengklik kanan ikon baterai di kiri bawah bilah tugas.

Saya biasa mengontrol pengaturan frekuensi CPU di W7 menggunakan ASUS AI Suite, karena saya memiliki ASUS mobo. Cari software serupa di situs resmi pabrikan mobo Anda. Alat ini memungkinkan untuk mengubah frekuensi atau pengali.

Selain itu, semua prosesor Intel Core i5 baru mendukung Teknologi Turbo Boost, yang secara dinamis meningkatkan frekuensi prosesor sesuai kebutuhan dengan memanfaatkan ruang kepala termal dan daya untuk memberi Anda semburan kecepatan saat Anda membutuhkannya, dan meningkatkan efisiensi energi saat Anda tidak membutuhkannya. .

Pengaturan Turbo Boost dikontrol dari dalam BIOS. Intel® Turbo Boost Technology adalah teknologi prosesor dan sistem operasi agnostik.

Di sini Anda dapat membaca FAQ tentang teknologi ini di situs Intel.

Berikut spesifikasi CPU Core i5 generasi ke-3 dengan teknologi Turbo Boost.

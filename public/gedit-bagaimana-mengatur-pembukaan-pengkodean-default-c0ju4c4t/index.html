Gedit - bagaimana mengatur pembukaan pengkodean default?

Saya ingin membuat Gedit di laptop saudara perempuan saya untuk membuka file teks dalam pengkodean Windows1250 (CP1250-Winlatin2) secara default.

Dia memiliki banyak ebook dalam format txt dan membukanya di LibreOffice tidak terlalu nyaman.

Saya perlu menggunakan Gedit atau aplikasi GTK serupa, yang sepenuhnya dapat diakses oleh Orca Screenreader.

PS: menjalankan Vinux (berdasarkan Ubuntu Lucid Lynx) - info lebih lanjut di sini:

buat skrip yang dapat dieksekusi /usr/local/bin/geditwin

#!/bin/sh
gedit --encoding=WINDOWS-1250 "$@"

dan setel sebagai editor teks default di gnome. (klik kanan pada file teks apa pun, buka dengan, ingat ...)

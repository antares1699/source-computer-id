Bagaimana cara meneruskan daftar file ke montase ImageMagick?

Saya memiliki banyak ubin, dengan konvensi penamaan berikut:

A-x-y.jpg

dimana

0<=x<=16 &
0<=y<=14

Sekarang saya tahu itu kita dapat menggunakan alat Montase di ImageMagick, untuk menggabungkannya menjadi satu gambar utuh.

Idealnya saya bisa menggunakan baris perintah berikut:

montage *.jpg -geometry +0+0 -tile 17x15 all.jpg

tetapi file saya sedemikian rupa sehingga A-0-0.jpg tepat di atas A-0-1.jpg, yaitu di atas A-0-2.jpg dan A-1-0.jpg ada di sebelah kanan A-0 -0.jpg.

Jadi ketika saya menggunakan perintah di atas urutannya salah. Saya memiliki file teks dengan urutan yang benar. Dapatkah saya menggunakan file teks sebagai masukan untuk montase. Jika ya, bagaimana cara melakukannya?

Di versi ImagMagick yang lebih baru (yaitu> v6.5.2-1) @ operator bisa digunakan seperti ini:

montage @list.txt -geometry +0+0 -tile 17x15 all.jpg

Saya t terlihat seperti montase tidak mendukung argumen "file dengan nama file".

Edit: @Devdatta menemukan sintaks yang benar (lihat juga sini ): @filenames.txt. Jadi, metode di bawah ini hanya diperlukan untuk versi alat ImageMagick yang sangat lama.

Tetapi jika Anda menggunakan Linux atau sistem mirip Unix, Anda dapat mengganti argumen nama file dengan output dari sebuah perintah. Dengan cara ini, Anda dapat menggunakan file Anda dengan nama file yang dipesan:

montage `cat filenames.txt` -geometry +0+0 -tile 17x15 all.jpg

(Harap perhatikan backticks di sekitar perintah cat.)

Jeda baris antara nama file di filename.txt tidak membahayakan pemanggilan perintah. Dan Anda bahkan dapat menggunakan wildcard pada filename.txt, karena penggantian perintah mendahului perluasan nama file.

Tentu saja, Anda juga dapat mengganti nama file sumber terlebih dahulu untuk mendapatkan pesanan yang diinginkan.

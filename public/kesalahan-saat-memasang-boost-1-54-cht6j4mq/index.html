Ini adalah bug dalam peningkatan. Untuk memperbaikinya buka file lib/coroutine/build/Jamfile.v2 dan ganti baris ini:

explicit yield_sources ;

Dengan yang ini:

explicit allocator_sources ;

atau tunggu rilis perbaikan bug.

Bagaimana uninstall / menghapus "Snipping Tool" dari menu start

Apakah ada cara untuk mencopot / menghapus Snipping Tool sehingga hanya Snip dan Sketch yang muncul saat saya mencari "snip" di Start Menu?

Microsoft membuat "Snip and Sketch" untuk menggantikan "Snipping Tool." Jadi, ketika Anda memulai "Snipping Tool", ia menanyakan apakah Anda ingin memulai "Snip and Sketch." Sebenarnya saya lebih suka menggunakan "Snip and Sketch", namun saya sering lupa yang mana setelah mencari "snip" di Start Menu. Saya tidak membutuhkan Snipping Tool yang lama.

Saya tidak dapat menemukan opsi untuk menghapus instalannya. Saya mencoba:

  • Aplikasi Pengaturan> Aplikasi & Fitur
  • Panel Kontrol \ Semua Item Panel Kontrol \ Program dan Fitur
  • Microsoft Store

Ada juga pengaturan registri / kebijakan grup tetapi itu hanya menonaktifkan kemampuan untuk menjalankan Snipping Tool; itu masih ada di Start Menu.

memperbarui: Ada juga dua pintasan keyboard yang mudah digunakan untuk memulai Snip & Sketch:

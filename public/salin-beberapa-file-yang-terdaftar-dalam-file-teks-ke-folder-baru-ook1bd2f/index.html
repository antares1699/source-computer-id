Salin beberapa file yang terdaftar dalam file teks ke folder baru

Saya perlu menyalin beberapa file yang tercantum dalam file teks ke folder baru, saya bukan ahli komputer, jadi harap lembut dan sederhanakan :)

(Nama file seperti 201609-ABCD.pdf)

Saya pikir di bawah ini akan melakukannya, tetapi tidak tahu cara menjalankannya.

untuk / f "delims =" %% L in (foo.txt) lakukan Salin file:

untuk / f "delims =" %% L in (foo.txt) lakukan salin "%% L" new_folder }

  • Jika Folder Sumber , Folder tujuan , dan Daftar File berada di jalur yang sama:
FileList:          D:\Files\FileList.txt
SourceFolder:      D:\Files\SourceFolder
DestinationFolder: D:\Files\DestinationFolder

Dan FileList hanya berisi nama dan ekstensi file:

File1.txt
File2.pdf
File3.docx
File4.jpg

Maka skrip Anda adalah:

@echo off
pushd D:\Files
for /f "tokens=* delims=" %%a in ('type FileList.txt') do xcopy /hrkvy ".\SourceFolder\%%a" ".\DestinationFolder"
popd
pause

@echo off

  • Jika Folder tujuan , dan Daftar File berada di bawah jalur yang sama dan SourceFolder adalah variabel:
FileList:          D:\Files\FileList.txt
SourceFolder:      %SourceFolder%
DestinationFolder: D:\Files\DestinationFolder

Dan FileList hanya berisi nama dan ekstensi file:

File1.txt
File2.jpg
File3.docx

Maka skrip Anda adalah:

@echo off
pushd D:\Files
set /p SourceFolder=Enter the path of source folder and press 'Enter':
cls
for /f "tokens=* delims=" %%a in ('type FileList.txt') do xcopy /hrkvy "%SourceFolder%\%%a" ".\DestinationFolder"
popd
pause
  • Jika Folder tujuan , dan Daftar File berada di bawah jalur yang berbeda:
FileList:          D:\Files\FileList.txt
DestinationFolder: C:\DestinationFolder

Dan FileList menunjuk ke file di lokasi berbeda:

D:\Files\File1.txt
C:\WorkFiles\File2.jpg
E:\File3.docx

Maka skrip Anda adalah:

@echo off
for /f "tokens=* delims=" %%a in ('type "D:\Files\FileList.txt"') do xcopy /hrkvy "%%a" "C:\DestinationFolder"
pause
  • Jika Folder tujuan , dan Daftar File berada di bawah jalur yang berbeda dan SourceFolder adalah variabel:

Memperbarui:

Untuk berjalan secara rekursif di subfolder OneDrive untuk menemukan file sumber:

@echo off
pushd D:\Missing Papers
for /f "tokens=* delims=" %%a in ('type MissingPapers.txt"') do (
 for /f "tokens=* delims=" %%b in ('dir /b /s "%OneDrive%" ^| find /i "%%a"') do
  xcopy /hrkvy "%%b" ".\Searched out"
 )
)
pause

  • Tempel di file

    FileList:          D:\Files\FileList.txt
SourceFolder:      D:\Files\SourceFolder
DestinationFolder: D:\Files\DestinationFolder
    dan ganti nama ekstensinya menjadi .cmd dan jalankan.

  • Jika Anda tidak memiliki izin baca dari direktori sumber, atau izin tulis ke direktori tujuan, jalankan skrip sebagai administrator.

  • Jangan lupa untuk mengganti contoh dengan jalur file yang sebenarnya.

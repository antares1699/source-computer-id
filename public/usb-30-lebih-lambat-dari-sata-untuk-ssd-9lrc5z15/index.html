USB 3.0 lebih lambat dari SATA untuk SSD

Saya memiliki SSD Kingston V200 256GB. Saat terhubung ke port SATA 6.0gbps saya di motherboard, saya melihat kecepatan baca berurutan 300 megabyte / detik.

Ketika saya memasukkan drive ke dalam kandang sata ke USB 3.0 dan menghubungkannya ke port motherboard USB 3.0 saya, kecepatan baca sekuensial saya adalah maksimum 200 megabyte / detik dan rata-rata 120 megabyte / detik. Saya mencoba enklosur di komputer lain dan mendapatkan 120 megabyte / detik juga.

Saya menduga kandang itu sendiri adalah penghambatnya. Adakah yang tahu tentang ulasan / tolok ukur pada chipset penutup sendiri sehingga saya dapat membeli sesuatu yang tidak akan memperlambat saya?

USB lebih lambat dari SATA. Standar SATA 3 baru beroperasi pada kecepatan maksimum teoritis 6,0 Gbps, sedangkan USB 3 beroperasi pada maksimum teoritis dua arah 4,8 Gbps. Jika kita menghitung angka-angka ini ...

      USB 3.0                    SATA 3
      4800Mbps            6000Mbps each way
      600 MBps            750 MBps each way

      USB 2.0                    SATA 2
480Mbps divided by 2      3000Mbps each way
  240Mbps or 30MBps       375 MBps each way

EDIT : Sesuai @MrAlpha, saya berdiri dikoreksi pada USB 3; itu dapat memanfaatkan Full Duplex (seperti yang dijelaskan sini )

  • -1 Ini tidak benar. USB 2.0 adalah half duplex, artinya dapat melakukan 480Mbps penuh (tidak memperhitungkan encoding) di kedua arah. Itu tidak bisa melakukannya di kedua arah pada saat yang bersamaan. USB 3.0 mendukung full duplex, sama seperti SATA sehingga dapat melakukan kecepatan penuh di kedua arah. Selain itu, kecepatan aktual SATA 3 setelah Anda memperhitungkan pengkodean 8b / 10b adalah dan kekacauan awalan ~ 572MiBps. Ini sebanding dengan USB 3.0s ~ 458MiBps.

Enklosur tidak memiliki mode UASP yang lebih cepat. Throughput USB 3.0 dunia nyata memiliki maksimum 400MB / detik (USB 3.0 memiliki overhead 36% dalam skenario kasus terbaik).

Perangkat USB 3.0 tipikal tampaknya hanya BOTS yang diaktifkan membatasi mereka hingga maksimum 250MB / detik meskipun dalam kasus saya ini tampaknya menjadi penutup drive yang lebih lambat.

eSata 3.0gbps masih lebih cepat dari USB 3.0 dalam kasus rata-rata pada 300MB / detik dunia nyata. Mengecewakan.

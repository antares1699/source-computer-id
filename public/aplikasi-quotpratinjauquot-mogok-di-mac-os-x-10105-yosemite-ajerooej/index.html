Aplikasi "Pratinjau" mogok di Mac OS X 10.10.5 (Yosemite)

Jadi pratinjau saya rusak, saya punya file laporan kecelakaan di sini .

Saya sudah mencoba beberapa hal:

  1. Menghapus semua preferensi Pratinjau
  2. Sama sekali menginstal ulang Preview dari Pacifist

Setiap kali saya terbuka apa pun di "Pratinjau" itu macet.

Saya tidak yakin apa penyebabnya, sepertinya tidak ada yang dijelaskan untuk memperbaikinya, saya sudah mencoba:

  • Preview.app selalu macet.
  • Perbaikan sederhana untuk Preview dan TextEdit Crash setelah Pembaruan Lion OS X.
  • Pratinjau Kerusakan Berulang Kali di Yosemite 10.10.1.

Jika Anda ingin informasi lebih lanjut atau ingin saya menjelaskannya, beri tahu saya.

Saya memiliki situasi yang sama dengan Anda. - Pratinjau tidak hanya macet setiap kali saya membuka file, file Informasi Khusus Aplikasi bagian Laporan Kerusakan sama dengan milik Anda. Kemudian saya menemukan postingan yang memperbaiki masalah crash! Ini menyarankan:

  • Instal ulang font yang baru dihapus

Untuk orang ini, itu menginstal ulang font Office 2011. Karena saya baru saja memutakhirkan dari MS Office 2011 ke Office 2016, menginstal ulang font Office 2011 juga merupakan perbaikan saya. Jelas, proses peningkatan telah menghapus font kunci.

Jika Anda baru saja membuat perubahan pada Instalasi MS Office atau dihapus apa saja font, coba instal ulang font yang terpengaruh berikutnya.

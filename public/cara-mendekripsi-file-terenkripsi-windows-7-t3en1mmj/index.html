Cara mendekripsi file terenkripsi Windows 7

Saya memiliki flash drive dan pada flash drive ini saya memiliki folder. Di Windows explorer, saya mengklik kanan pada folder dan membuka properti. Saya kemudian mengklik tombol lanjutan dan ada kotak centang di sana yang bertuliskan "Enkripsi Isi untuk Mengamankan data". Saya mengklik kotak ini dan menekan OK. Windows kemudian mengenkripsi folder untuk saya. Saya dapat membuka file di folder itu dengan baik di komputer saya, tetapi tidak di komputer lain.

Pertanyaan saya adalah, saya memiliki komputer lain tempat saya ingin membuka file-file ini. Apa yang harus saya lakukan untuk mengizinkan komputer ini mengakses folder ini? Saya berpikir saya perlu beberapa cara mengekspor kunci tetapi saya tidak yakin kunci mana yang akan diekspor atau bagaimana melakukan ini. Bantuan apa pun akan dihargai.

Kamu butuh cadangkan kunci enkripsi file Anda dan memulihkannya di komputer lain.

Karena cara kerja sistem, Anda mungkin harus mengembalikannya ke pengguna yang memiliki log-in dan kata sandi yang sama persis. Jika ini tidak nyaman, Anda harus menggunakan cara enkripsi lain.

Jawaban BillC.cn hanya sebagian benar. Meskipun Anda dapat mengekspor kunci privat, Anda tidak memerlukan kata sandi atau nama pengguna yang sama. Semacam kekalahan definisi enkripsi kunci publik yang terjadi.

Pastikan untuk mengimpor sertifikat ke toko pribadi atau pengguna Tepercaya.

Anda bahkan dapat mengenkripsi file dengan lebih dari satu kunci publik, sehingga Anda tidak harus selalu memberikan kunci pribadi Anda. Namun dalam kasus Anda, Anda ingin sertifikat EFS disimpan di semua komputer Anda.

http://technet.microsoft.com/en-us/library/bb457065.aspx

Tombol fisik untuk menjalankan skrip pada mesin Windows

Saya sering berhubungan dengan orang-orang yang sama sekali tidak dapat mengoperasikan PC, tidak peduli berapa banyak pelatihan yang mereka dapatkan.

Saya memerlukan semacam tombol fisik yang dapat diprogram atau remote control yang menjalankan skrip pada mesin Windows saat ditekan. (Dan tidak, wiseguys, tikus tidak akan melakukannya.)

Apakah hal seperti itu ada? Jika ya, bagaimana cara menyiapkannya?

Ini agak out-of-the-box, tetapi Anda dapat membeli tombol yang dirancang untuk mesin arcade. Tombol semacam itu akan membutuhkan penyolderan dan pemrograman, atau Anda dapat menyambungkannya ke keyboard.

Saya Keyboard Nirkabel Microsoft 4000 memiliki 5 tombol yang dapat diprogram di bagian atas dengan label angka. Saya meletakkan label kecil yang menempel di atasnya untuk mengingat mana yang melakukan apa.

Anda dapat menetapkan masing-masing tombol ini untuk membuka file yang dapat dieksekusi. Jadi Anda bisa menulis skrip dalam bahasa favorit Anda dan meminta tombol memanggilnya (panggil juru bahasa untuk menjalankan skrip, mungkin).

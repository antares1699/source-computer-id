Tombol Alt kanan tidak berfungsi dengan benar

Saya memiliki laptop Dell Inspiron 15 3537. Seperti yang Anda ketahui, kami menggunakan 'alt + tombol panah' untuk menavigasi maju atau mundur di browser web, penjelajah file, dll. Sekarang tombol 'alt' kiri saya berfungsi seperti ini. Tapi tombol 'alt' kanan saya akan memutar tampilan saya saat digunakan dengan tombol panah. Jadi saya mencari di internet dan menonaktifkan 'hotkeys' dari 'Graphics Options' di bawah 'Intel HD Graphics Control Panel'.

Sekarang semuanya baik-baik saja selain itu tombol 'alt' yang tepat tidak berfungsi. Itu bahkan tidak akan menyorot bilah menu seperti tombol kiri. Juga tidak akan menavigasi seperti yang disebutkan di atas.

Saya dapat meyakinkan Anda bahwa tombol 'alt' yang tepat berfungsi (seperti halnya memutar layar). Adakah yang tahu bagaimana membuat kunci 'alt' yang benar yang telah dianggap tidak berguna ini, berfungsi seperti biasa?

Terima kasih!

AutoHotKey adalah program yang dapat diunduh yang dirancang untuk mengotomatiskan komputer Anda melalui skrip. Ini pada dasarnya memungkinkan Anda membuat pintasan keyboard untuk apa pun. Itu juga dapat memetakan ulang kunci. Dalam kasus Anda, kami akan memetakan ulang file Alt Kanan kunci untuk menjadi a Alt Kiri kunci.

Pertama, unduh dan instal program dari tautan di atas. Setelah terinstal, temukan tempat di mana Anda ingin menyimpan skrip (C: \ Users \ {user} \ AppData \ Roaming \ Microsoft \ Windows \ Start Menu \ Programs \ Startup adalah tempat yang baik, karena seharusnya dimulai dengan Windows) dan klik kanan pada ruang kosong di Windows Explorer -> New -> AutoHotKey Script. Beri nama apa pun yang Anda suka. Klik kanan file baru dan tekan Edit Script dan tempel yang berikut ini:

RAlt::LAlt

Simpan file, tutup, dan klik dua kali file untuk menjalankan. Ini mencari penekanan tombol alt kanan dan mengirimkan penekanan tombol alt kiri.

Anda mungkin juga ingin memeriksa pertanyaan yang diberi tag dengan AutoHotKey aktif Pengguna Super dan Stack Overflow .

Anda mungkin tidak memerlukan AutoHotkey untuk mengatasi masalah ini. Saya memiliki masalah yang sama di Windows8.1 - alt kanan saya berfungsi (menyebabkan rotasi layar), tetapi tidak berfungsi seperti tombol alt kiri. Dalam kasus saya, saya harus mengubah pengaturan Windows sederhana untuk memperbaikinya.

Ini karena tata letak keyboard non-standar seperti yang dijelaskan di sini: https://support.microsoft.com/en-us/kb/82202

Saat Anda menjalankan Microsoft Windows dalam mode real (3.0 saja), standar, atau 386 yang disempurnakan dengan tata letak keyboard AS-Internasional, tombol ALT kanan berfungsi secara berbeda atau tidak sama sekali.

Dengan tata letak keyboard AS-Internasional, tombol ALT kanan dipetakan ulang untuk fungsionalitas internasional. Tombol ALT kiri dapat (dan harus) digunakan untuk perintah Windows.

Rupanya Windows melakukan ini dengan sengaja jika Anda mengatur wilayah Anda ke apa pun di luar AS. Dalam kasus saya, saya telah menyetelnya ke India, jadi pilihan bahasa dan wilayah saya dan yang terpenting, tata letak keyboard telah ditetapkan ke English (India).

Menggunakan ini sebagai panduan , Saya hanya perlu mengubah preferensi bahasa saya untuk menggunakan English (United States), bukan English (India). Ini mengatur tata letak keyboard saya ke "AS" dan ini segera membuat tombol alt kiri dan kanan bekerja sama. Masalah terpecahkan.

Anda dapat menggunakan Autohotkey untuk memetakan ulang tombol Alt ke LAlt:

#SingleInstance force
^>!*::LAlt

Jangan lupa untuk memberikan Script tersebut di folder autostart Anda

Deskripsi:

  • ^ = saat tombol ditekan bukan rilis
  • > = sisi kanan
  • ^ = saat tombol ditekan bukan rilis
  • ! = tombol alt
  • * = dan semua kunci lainnya
  • :: = peta ulang ke
  • LAlt = tombol Alt kiri

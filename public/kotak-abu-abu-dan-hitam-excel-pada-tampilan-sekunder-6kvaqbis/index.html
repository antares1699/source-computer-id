Instal ulang Office.

Saya memiliki masalah yang sama - hanya terlihat dengan MS Excel dan tidak ada program lain. Area yang digelapkan akan bergeser saat bergulir tetapi tidak akan pernah hilang kecuali jendela diperkecil dan diperbesar. Hal ini tidak memungkinkan untuk bekerja pada spreadsheet yang besar.

Perangkat keras saya mencakup Dell XPS15 9550 dengan dok TB15 thunderbolt 3 dan monitor dell 4k 34 ". Sejak pemasangan pertama, masalah terjadi (lihat tangkapan layar di bawah) dan bantuan Microsoft tidak membantu sama sekali - tidak ada saran setelah satu jam mengobrol dengan akses desktop jarak jauh dengan bantuan dept. Sangat membuat frustasi!

Selanjutnya bekerja dengan Office 365 Std juga. Saya hanya bisa menyimpulkan bahwa itu pasti beberapa bentuk file yang rusak selama proses unduh / instal? Sangat aneh dan membutuhkan banyak waktu untuk menemukan solusinya.

Memiliki Lenovo 910 512GB dengan 16GB, monitor 4K

Kami telah mencoba segalanya untuk memperbaiki masalah ini, 2 orang TI dan Dukungan Teknis MS. Kami percaya bahwa MS Excel tidak bekerja dengan baik dengan monitor 4K.

Saat menjalankan Excel dalam Mode Aman, kami menemukan bahwa masalah telah teratasi. Ini adalah pemahaman kami bahwa Safe Mode menonaktifkan add-on dan berjalan dalam mode VGA. Menonaktifkan Add-On sedikit membantu. Namun, menjalankan seluruh sistem dalam VGA (Safe Mode) memperbaiki masalah tersebut.

Jelas, kami tidak akan menjalankan Safe Mode. Saya telah memutuskan untuk menerima pengumuman ini sampai MS memperbarui perangkat lunak mereka untuk bekerja dengan monitor 4K dengan benar.

Saya harap ini membantu.

Saya mengalami masalah ini juga dan memposting ke / r / Excel .

Saya akhirnya menghubungi dukungan Microsoft dan diberitahu untuk melakukan hal berikut. Langkah-langkah ini memperbaiki masalah saya.

Buka Panel Kontrol dan jalankan Perbaikan Online:

  1. Tombol Windows + X
  2. Klik Program dan Fitur (opsi pertama)
  3. Klik kanan Microsoft Office> Ubah> Perbaikan Online

Penjelasan teknis dukungan adalah bahwa MS Office (Khususnya Excel) rusak dan menyebabkan masalah rendering.

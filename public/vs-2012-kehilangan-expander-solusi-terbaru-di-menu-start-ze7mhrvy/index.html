VS 2012 kehilangan expander solusi terbaru di menu Start

Saya sangat menyukai fitur Windows 7 yang memiliki item terbaru / tersemat untuk aplikasi yang langsung tersedia di menu Start melalui expander di sisi kanan item aplikasi. Namun, untuk VS 2012 saya tidak mendapatkan expander! Ini berfungsi dengan baik untuk semua aplikasi saya yang lain (VS 2010, Office, dll.), Dan juga berfungsi pada PC lain. Ada ide?

Sunting: Dengan "expander", maksud saya panah segitiga kecil menunjuk ke kanan yang ditampilkan di sebelah kanan nama aplikasi. Saat diklik, atau saat mouse diarahkan ke item menu, sisi kanan menu Start akan menampilkan daftar item terbaru dan yang disematkan untuk aplikasi itu. Ini adalah daftar yang sama yang tersedia saat mengklik kanan ikon aplikasi di bilah tugas.

Saya ingin memposting gambar, tetapi saya tidak diizinkan oleh "mekanisme pencegahan spam".

Saya mencoba banyak hal berbeda. Seperti itulah saya

Seperti yang Anda lihat, LibreOffice menampilkan file terbarunya, tetapi tidak Visual Studio 2103. Bagi saya masalah ini dimulai setelah membuka salinan proyek, dan ketika menghapus salinan itu saya biarkan Visual Studio menghapus pintasan dari daftar terbaru, ketika saya mencoba menggunakannya. Itu membuat pintasan terbaru untuk proyek asli (dengan nama yang sama) menghilang juga, dan bahkan setelah beberapa trial and error, seluruh daftar terbaru VS di startmenu.

Hal-hal mencoba tidak berhasil: Saya menghapus pengaturan "Simpan dan tampilkan item yang baru dibuka" di menu Mulai dan bilah tugas, dan memeriksanya lagi.

Saya melepas pin VS di menu awal dan menyematkan lagi, menutup VS dan membuka proyek baru dan menutup VS. Masih belum ada panah untuk proyek terbaru.

Tidak sampai saya berubah Alat / Opsi / Umum / Item yang ditampilkan dalam daftar yang baru-baru ini digunakan di dalam Visual Studio, dari 10 hingga 11, panah itu muncul, dan proyek yang baru dibuka ditambahkan.

Bagaimana cara mengaktifkan kembali mengingat kata sandi di Firefox?

Saya memberi tahu Firefox untuk berhenti meminta untuk mengingat kata sandi di suatu situs, tetapi sekarang saya ingin mengaktifkannya kembali. Saya melihat sekeliling di FF dan tidak dapat menemukan cara melakukannya.

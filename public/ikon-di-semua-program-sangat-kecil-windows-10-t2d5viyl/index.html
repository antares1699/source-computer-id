Ketika saya membuka program apa pun di laptop saya, ikon bilah tugas sangat kecil

Gunakan prosedur berikut untuk mengubah ukuran ikon bilah tugas:

  1. Klik kanan pada ruang kosong di desktop.

  2. Pilih Pengaturan tampilan dari menu kontekstual.

  3. Pindahkan penggeser di bawah "Ubah ukuran teks, aplikasi, dan item lainnya" ke 100%, 125%, 150%, atau 175%.

  4. Tekan Apply di bagian bawah jendela pengaturan. Windows mungkin meminta Anda untuk keluar dan masuk kembali untuk mendapatkan pengalaman yang lebih konsisten.

Ini tidak hanya akan membuat ikon di bilah tugas sedikit lebih besar, tetapi juga juga akan membuat teks lebih besar di aplikasi seperti Microsoft Edge dan Kalender, serta di jendela notifikasi Windows 10.

Berdasarkan solusi yang ditemukan di Forum Lenovo , Saya memecahkan sebagian masalah ini dengan:

  1. Klik kanan pada pintasan aplikasi dan pilih Properti .
  2. Pilih Kesesuaian tab.
  3. tekan Ubah pengaturan DPI tinggi tombol.
  4. Memeriksa Mengesampingkan perilaku penskalaan DPI tinggi dan pilih Sistem atau Sistem (Ditingkatkan) .

Seperti yang disebutkan di solusi asli , masih ada beberapa masalah kecil (bagi saya - kursor mouse kecil di beberapa area) tetapi ini menyelesaikan masalah saya dengan ikon kecil.

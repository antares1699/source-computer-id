firefox 5.0: bilah status menghilang meskipun ekstensi "status 4 evar"

Bilah status saya entah bagaimana menghilang dalam beberapa hari terakhir. Saya telah menginstal Status 4 Evar.

Saya mencoba menonaktifkan semua add-on saya kecuali Status 4 Evar, dan status barnya adalah masih hilang.

Tolong! Bagaimana cara memperbaikinya?

Bar yang biasa kita sebut "Status Bar", menurut pemahaman saya, sekarang dikenal sebagai "Add-On Bar". Ini dapat diubah dengan salah satu dari dua cara:

  • Melalui Menu: Firefox -> Options -> Add-On Bar
  • Melalui Pintasan: Ctrl + /

Bagaimana menemukan file dalam berbagai ukuran?

Tolong sarankan saya cara untuk menemukan semua file zip yang lebih dari 60 MB tetapi ukurannya kurang dari 70 MB menggunakan perintah find.

6
find -iname "*.zip" -size +60M -size -70M

Anda juga dapat menggunakan -o untuk mendapatkan disjungsi, dan \( … \) untuk mengelompokkan berbagai kecocokan.

find -iname "*.zip" -size +$((60*1024*1024))c -size -$((70*1024*1024))c

JANGAN gunakan abreviations 60M dan 70M karena ini juga akan mengecualikan semua file dengan ukuran lebih besar dari 69MB termasuk 69.001MB !!!

dari bagian dokumentasi info 2.4 Ukuran

-- Test: -size n[bckwMG]
    True if the file uses N units of space, rounding up.
    ...

jadi 69.001 mendapat tercakup menjadi 70 dan dengan demikian mendapat pengecualian !

contoh sempurna adalah find . -size -1M yang mana hanya cocokkan ukuran file nol .

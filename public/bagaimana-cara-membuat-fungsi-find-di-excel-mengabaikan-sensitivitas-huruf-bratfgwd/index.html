Bagaimana cara membuat fungsi FIND di Excel mengabaikan sensitivitas huruf?

Katakanlah saya memiliki lembar Excel 100 catatan di mana ada 3 kolom A, B dan C.

Syaratnya saya harus mendapatkan semua baris dari kolom C jika teks pada kolom C cocok dengan SQL Server.

Di sini teks SQL Server dalam format yang berbeda SQL Server,sql Server,sql server

Saya menggunakan perintah =IF(ISERROR(FIND("SQL Server", C2)), 1, 0) untuk menemukan baris yang cocok di kolom C.

Jika kecocokan ditemukan, ia mengembalikan 0 jika tidak ia mengembalikan 1.

Sekarang masalahnya adalah mengembalikan 1 untuk string seperti sql server dan sql Server.

Bagaimana cara membuat perintah FIND tidak peka huruf besar / kecil?

Excel 2013: “Gambar terlalu besar dan akan dipotong.”

Pesan kesalahan ini mulai muncul di Excel segera setelah saya menekan tombol potong atau salin. Saya tidak tahu kapan itu dimulai atau mengapa dan itu terjadi setiap saat. Menjalankan Windows 7, dengan Office 2013 Pro.

Saat meneliti masalah ini, saya menemukan banyak solusi yang dapat saya perbarui di sini nanti.

Namun tidak ada , bekerja untuk saya, secara pribadi.

Pada akhirnya, pelakunya adalah Manajer Windows yang sebenarnya / Beberapa Monitor Aktual yang memiliki opsi untuk "mengaktifkan riwayat papan klip [seluruh sistem]" yang dapat Anda atur sedalam 25 entri.

Setelah penelitian lebih lanjut, saya sampai pada kesimpulan itu lain / program pihak ke-3 yang terhubung ke clipboard windows dapat menyebabkan masalah yang sama. Jadi, saya sarankan Anda melihat opsi ini sebelum yang lain, karena ini adalah tes / perbaikan cepat:

  • Saya menonaktifkan opsi riwayat clipboard di Manajer Windows Aktual
  • lalu coba potong atau salin lagi di Excel, kesalahan telah hilang
  • Dalam kasus saya, saya tidak perlu memulai ulang Excel atau AWM, tetapi saya akan menyarankan untuk mencoba keduanya jika solusi ini tidak langsung berhasil untuk Anda.
  • Jika tidak ada opsi sederhana untuk menonaktifkan, coba tutup atau matikan (di pengelola tugas) proses apa pun yang terkait dengan program pihak ketiga yang dicurigai
  • Jelas, Anda dapat mengaktifkan kembali program / opsi pihak ketiga lagi ketika Anda tidak menggunakan excel, atau hanya hidup dengan kesalahan.

Info lebih lanjut: Saya juga membaca tentang hal ini terjadi dengan TeamViewer , saat mengaktifkan opsi "sinkronisasi papan klip antara host lokal dan sesi jarak jauh"

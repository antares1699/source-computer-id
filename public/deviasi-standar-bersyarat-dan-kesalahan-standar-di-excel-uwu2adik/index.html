Deviasi standar bersyarat dan kesalahan standar di Excel

Saya memiliki data yang ingin saya grafikkan dalam spreadsheet Excel 2013. Berikut adalah fragmen datanya

-------------------------------------------
| intervalMode  | keyboardEventsPerSecond |
-------------------------------------------
| orienting     | 0                       |
| orienting     | 0                       |
| orienting     | 0.171115674             |
| orienting     | 0                       |
| orienting     | 0                       |
| opportunistic | 0                       |
| opportunistic | 0.016913605             |
| opportunistic | 0                       |
| opportunistic | 0                       |
-------------------------------------------

Saat saya memetakan data, saya ingin menempatkan rata-rata untuk setiap mode. Ini mudah dicapai dengan tabel yang berisi rumus

=AVERAGEIF(AllDataFromSQL!A2:A14057,Graphs!A3,AllDataFromSQL!B2:B14057)

yang menggunakan kolom pertama dan kedua dari fragmen yang ditampilkan sebagai argumen pertama dan ketiga. Menggunakan rumus itu saya mendapatkan tabel ini:

--------------------------------------------------
| intervalMode  | average keyboardEventsPerSecond |
--------------------------------------------------
| lean-back     | 0.009044655                     |
| opportunistic | 0.01058782                      |
| orienting     | 0.036665215                     |
| purposeful    | 0.03851359                      |
| respite       | 0.120037091                     |
--------------------------------------------------

Itu hebat. Jika saya memetakan tabel itu, saya mendapatkan ini

Saya ingin menambahkan bilah kesalahan, yang dapat saya lakukan di alat bagan Excel. Namun saya berakhir dengan ini

Bilah kesalahan untuk dua bilah pertama meluas ke bawah sumbu, dan seharusnya tidak.

Untuk memperbaikinya, saya akan beralih ke bilah kesalahan khusus dan menambahkan dua kolom ke tabel saya untuk bilah kesalahan positif dan negatif. Karenanya saya berharap untuk menggunakan rumus seperti

=STERRIF(AllDataFromSQL!A2:A14057,Graphs!A3,AllDataFromSQL!B2:B14057)

Tapi sepertinya tidak ada rumus yang dibuat untuk kesalahan standar. Jadi saya bisa menggulungnya sendiri menggunakan sesuatu seperti

=STDEVIF(AllDataFromSQL!A2:A14057,Graphs!A3,AllDataFromSQL!B2:B14057)/SQRT(COUNTIF(AllDataFromSQL!A2:A14057,Graphs!A3))

Tetapi STDEVIF tampaknya tidak ada. Tentu saja saya dapat menambahkan kolom tambahan dan menghitungnya dengan tangan, tetapi adakah cara yang lebih baik untuk menghitung kesalahan standar dengan cara yang sama saya menggunakan AVERAGEIF untuk menghitung rata-rata bersyarat?

Anda dapat menghitung simpangan baku bersyarat dengan menggunakan "rumus larik" - sintaksnya akan seperti ini

=STDEV(IF(AllDataFromSQL!A$2:A$14057=Graphs!A3,AllDataFromSQL!B$2:B$14057))

Itu adalah rumus array dan karena itu Anda perlu mengonfirmasi CTRL + BERGESER + MEMASUKKAN . Untuk melakukan itu pilih sel dengan rumus, tekan F2 untuk memilih rumus lalu tahan CTRL dan BERGESER sambil menekan MEMASUKKAN . Jika dilakukan dengan benar maka tanda kurung kurawal seperti {dan} akan muncul di sekitar rumus pada bilah rumus.

Anda perlu melakukan itu untuk rumus pertama .... lalu Anda dapat menyalin / mengisi

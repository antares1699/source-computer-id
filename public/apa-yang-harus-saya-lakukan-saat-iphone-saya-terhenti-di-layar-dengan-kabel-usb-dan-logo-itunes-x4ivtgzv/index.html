Saya melihat layar itu beberapa hari yang lalu .. Saya menggunakan laptop saya untuk mencabut dan menyuruhnya untuk terus mengupgrade perangkat lunak iPhone terbaru. Butuh beberapa saat, saya tidak memperhatikan, dan BANG .. laptop tidur karena kehabisan jus. Ketika saya memasangnya di dalamnya, duduk selama beberapa menit lalu bertanya apakah saya ingin memulihkan. Saya melakukannya, dan itu berhasil.

Saya berasumsi bahwa Anda telah mengikuti saran di halaman tertaut, dan itu tidak berhasil. Satu-satunya hal lain yang akan saya coba (sebelum mengunjungi Genius Bar) adalah:

  1. Hard reset iPhone (tahan kedua tombol sampai apel muncul). Ini kemungkinan besar tidak akan menyelesaikan masalah, tetapi akan membersihkan sisa-sisa makanan.
  2. Dengan menjalankan iTunes, sambungkan iPhone dan diamkan sebentar.
  3. Jika iTunes mengenali iPhone, coba opsi pulihkan.

Semoga berhasil.

Bagaimana cara memasang hard drive secara manual di mac?

Mac saya baru saja menyerah dan saya mencoba untuk menyelamatkan data apa pun yang saya bisa dari hard drive. Saya dapat boot menggunakan CD Instalasi OSX dan menjalankan Disk Utility yang pada dasarnya memberi tahu saya bahwa drive tidak dapat diperbaiki dan saya perlu memformatnya. Pertanyaan saya - bagaimana saya bisa memasangnya secara manual (dari Terminal) dan mencoba memulihkan beberapa data saya? Saya pikir jika Disk Utility dapat melihatnya, mungkin saya juga bisa. Terima kasih.

Untuk memasang disk:

daftar diskutil

lihat drive dan note id #, lalu:

diskutil mountDisk / dev / diskn

di mana n adalah # yang Anda id'ed dalam daftar diskutil

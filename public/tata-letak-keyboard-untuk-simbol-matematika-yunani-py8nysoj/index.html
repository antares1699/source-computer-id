Tata letak keyboard untuk simbol matematika / Yunani

Saya sudah lama bertanya-tanya tentang hal ini tetapi tidak pernah terpikir untuk bertanya: Saya melakukan banyak pekerjaan ilmiah sehingga sering kali akan sangat berguna untuk dapat mengetik simbol matematika atau huruf Yunani yang, sebagian besar , bukan bagian dari kumpulan karakter ASCII. Seperti "∞ µ ρ σ τ ω ∑… √ ∫ ≤ ≥ ±" dan seterusnya. Apakah ada tata letak keyboard (untuk Linux) yang memetakan kombinasi tombol sederhana ke jenis karakter ini? (Dengan asumsi semua masalah pengkodean dan font diselesaikan dengan benar) Saya tahu saya dapat membuatnya sendiri tetapi akan jauh lebih mudah jika seseorang sudah menyelesaikan pekerjaannya, atau setidaknya jika ada solusi parsial yang dapat saya modifikasi.

Bagaimana cara mengikat numpad ke pengalih tag di WM yang mengagumkan?

Di bawah WM yang mengagumkan, saya menemukan Mod4 + numpad <1-9> tidak berfungsi sama dengan Mod4 + 1-9 meskipun NumLock aktif. Saya pikir akan lebih mudah menggunakan Mod4 + numpad <1-9> untuk beralih antar tag. Bagaimana cara mengikat ini?

Saya mendapatkan peta kunci numpad dengan menggunakan xev. Jadi, mungkin mudah untuk memetakan numpad ke pengalih tag sebagai berikut:

-- Numpad: [0-9] = [#90, #87-#89, #83-#85, #79-#81]
local np_map = { 87, 88, 89, 83, 84, 85, 79, 80, 81 }
for i = 1, keynumber do
   globalkeys = awful.util.table.join(
      globalkeys,
      awful.key({ modkey }, "#" .. np_map[i],
        function ()
           local screen = mouse.screen
           if tags[screen][i] then
              awful.tag.viewonly(tags[screen][i])
           end
        end),
      awful.key({ modkey, "Control" }, "#" .. np_map[i],
        function ()
           local screen = mouse.screen
           if tags[screen][i] then
              awful.tag.viewtoggle(tags[screen][i])
           end
        end),
      awful.key({ modkey, "Shift" }, "#" .. np_map[i],
        function ()
           if client.focus and tags[client.focus.screen][i] then
              awful.client.movetotag(tags[client.focus.screen][i])
           end
        end),
      awful.key({ modkey, "Control", "Shift" }, "#" .. np_map[i],
        function ()
           if client.focus and tags[client.focus.screen][i] then
              awful.client.toggletag(tags[client.focus.screen][i])
           end
        end))
end

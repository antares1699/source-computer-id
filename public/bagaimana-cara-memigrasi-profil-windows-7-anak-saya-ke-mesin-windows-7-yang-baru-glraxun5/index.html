Transfer Mudah Windows , hadir dengan Windows 7. Anda dapat menemukannya dengan mengetikkan namanya di Start Menu.

Itu adalah Wizard yang cukup mudah. Pilih "ini adalah PC lama saya" pada PC saat ini dan itu akan memindai profil. Batalkan pilihan yang tidak ingin Anda transfer (Anda juga dapat memilih file misc yang ingin Anda transfer dengan profil.)

Simpan profil (saya pikir itu menyimpannya sebagai file .mig) pada drive eksternal lalu cukup klik dua kali file itu di PC baru untuk memulai transfer (atau buka transfer mudah dan temukan file.)

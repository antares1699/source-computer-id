Saya baru saja membeli Dell Studio XPS 16, dengan Windows 7 Home Premium. Itu datang dengan semua jenis bloatware, jadi saya memutuskan untuk menginstal salinan Windows yang baru. Namun, saya tidak dapat menemukan cara untuk mengunduh salinan Windows 7 dari situs web Microsoft. Apakah ada cara untuk melakukan ini (selama saya memberikan kunci produk)?

Untuk seri Dell Studio biasanya terdapat partisi tersembunyi pada disk untuk memungkinkan penginstalan ulang atau pemulihan. Ini termasuk file penginstalan Windows (sebenarnya biasanya hanya gambar hard drive dalam keadaan pabrik). Saya akan mulai dengan melihat dokumentasi. Lebih sering daripada tidak saya pergi begitu saja Situs Dell dan masukkan tag servis vs. rooting melalui kertas.

Masalahnya tentu saja, partisi penginstalan ini biasanya disetel ulang ke status pabrik, yang menyertakan bloatware yang sangat mengganggu. Seringkali jika disk disertakan dengan dokumentasi, akan ada disk Windows dan kemudian disk Dell dengan program / bloatware-nya. Jika Anda menginginkan disk dan tidak menerima, OEM terkadang akan mengirimkan cakram dengan sedikit biaya, tetapi berbeda-beda menurut pabrikan, saya tahu Lenovo telah menawari saya ini di masa lalu, tetapi saya tidak yakin tentang Dell.

Ingatlah bahwa Anda memiliki lisensi untuk perangkat lunak tersebut, Anda mencoba mencari medianya. Menghubungi Microsoft hampir pasti akan mengakibatkan mereka mendorong Anda ke Dell. Juga, sekarang mungkin saat yang tepat untuk menuliskan kunci Windows Anda, apakah itu di bagian bawah laptop atau jika Anda menjalankan utilitas ekstrak kunci.

Terakhir, Anda selalu dapat meminjam disk dari teman, asalkan Anda memiliki kunci, dan menginstal sepenuhnya bersih. Saya akan merekomendasikan mengunduh driver Dell terlebih dahulu, karena partisi instal ulang / pemulihan biasanya memiliki ini yang dimuat sebelumnya, tetapi instalasi bersih melalui disk Windows yang berbeda tidak akan.

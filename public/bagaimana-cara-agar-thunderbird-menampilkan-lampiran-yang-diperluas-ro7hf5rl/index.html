Bagaimana cara agar Thunderbird menampilkan lampiran yang diperluas

Pembaruan terbaru untuk klien email Thunderbird menunjukkan lampiran di bawah panel terpisah. Saya ingin panel yang ditampilkan diperluas secara otomatis.

Bagaimana aku melakukan itu? Tampaknya tidak ada di file bantuan atau catatan rilis.

4
Brad Bruce — Sumber

4 Jawaban

2
2

Sesuai artikel ini di Desain Ulang Panel Lampiran di MozillaZine, Anda dapat menambahkan berikut ini ke userChrome.css untuk menghapus matikan lampiran:

#attachmentView > [collapsed="true"] {
  visibility: visible !important;
}

#attachmentToggle {
  display: none !important;
}

Edit: Nama file dikoreksi menjadi userChrome.css

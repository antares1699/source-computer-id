Pintasan keyboard untuk beralih ke mode bahasa Rusia di Windows

Jika Anda memiliki beberapa bahasa input yang diinstal di Windows (di bawah item panel kontrol Wilayah dan Bahasa → Keyboard dan Bahasa), apakah ada cara cepat untuk beralih antar bahasa (misalnya, beralih antara keyboard bahasa Inggris dan Rusia)?

4
Derek Morrison — Sumber

10

Pintasan default untuk memutar bahasa adalah Alt Kiri + Bergeser .

Jika Anda memiliki lebih dari dua bahasa, saya sarankan Anda menyetel pintasan terpisah untuk bahasa yang berbeda di Preferensi bahasa → Pengaturan lanjutan → Ubah tombol pintas bilah bahasa

Misalnya, saya punya Ctrl + Bergeser + 1 diatur ke Rusia dan Ctrl + Bergeser + 2 disetel ke IME Jepang, karena saya memiliki 8 tata letak lagi yang terpasang dan akan memakan waktu terlalu lama untuk merotasi semuanya.

Perbarui 17 Agustus: Pada Windows 10, ada juga yang baru Menang + Ruang pintasan tersedia.

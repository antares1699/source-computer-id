Membuat alias untuk NPM menggunakan cmder di Windows

Saya menggunakan cmder di windows: http://bliker.github.io/cmder/

Saya telah menginstal node.js dari nodejs.org.

Saya dapat menjalankan npm di cmder dengan menjalankan perintah ini:

C:\Program Files\nodejs\npm.cmd

Sekarang bagaimana cara membuat alias untuk ini sehingga saya bisa menjalankannya saja, misalnya npm install -g bower?

Saya mencoba ini:

alias npm="C:\Program Files\nodejs\npm.cmd"

Tapi itu melaporkan kembali:

alias npm="C:\Program Files\nodejs\npm.cmd" Files\nodejs\npm.cmd""] was unexpected at this time.

Jika nama file menyertakan spasi, maka perlu menggunakan nama versi 8.3, seperti ini:

alias npm=C:\Progra~1\nodejs\npm.cmd $*

Anda bisa mendapatkan nama pendek dengan menggunakan perintah: dir /x c:\.

Anda mungkin ingin menambahkan folder yang berisi npm ke variabel sistem "Path".

Untuk melakukannya :

  • Buka "Komputer" -> Klik kanan pada ruang kosong -> Properti -> Klik "Parameter sistem lanjutan" yang terletak di panel kiri

  • Jendela baru seharusnya terbuka -> Pilih tab "Parameter sistem lanjutan" -> Klik "Variabel lingkungan"

  • Kemudian pilih "Path" di bawah "Variabel sistem" dan klik ubah.

  • Anda harus memiliki sesuatu seperti:

C: \ ProgramData \ Oracle \ Java \ javapath;% SystemRoot% \ system32;% SystemRoot%;% SystemRoot% \ System32 \ Wbem;% SYSTEMROOT% \ System32 \ WindowsPowerShell \ v1.0 \; C: \ wamp \ bin \ php \ php5.5.12; C: \ Program Berkas (x86) \ Git \ cmd; C: \ Program Files (x86) \ GNU \ GnuPG \ pub

  • Di akhir, tambahkan saja "; PATH_TO_NPM" (Jangan lupa ";" di awal "

  • Nyalakan ulang komputer Anda dan seharusnya berfungsi.

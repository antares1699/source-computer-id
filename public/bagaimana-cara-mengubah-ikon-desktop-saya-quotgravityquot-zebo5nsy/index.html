Bagaimana cara mengubah Ikon Desktop saya "Gravity"?

Ikon pada desktop XP saya memiliki "gravitasi" kiri normal. Artinya, mereka mengisi dari sisi kiri ke sisi kanan. Namun, wallpaper yang baru saja saya alihkan membuat sisi kiri "sibuk" dan ikon sulit dilihat di sana, jadi saya sangat ingin mengalihkannya ke gravitasi sisi kanan. Tidak ada yang mewah, seperti Pagar, saya hanya ingin mengubah arah pengisian defaultnya.

Saya berpikir pasti bahwa saya dulu dapat melakukan ini di Windows, tetapi saya yakin tidak dapat menemukan cara untuk melakukannya di XP. Ada yang tahu bagaimana melakukan ini?

Googling kecil yang menganggur memunculkan beberapa kemungkinan:

WinTidy 2 memiliki tata letak / templat untuk menyelaraskan ikon. Masalah dengan XP SP2, YMMV.

Ikonoid memungkinkan Anda untuk menyimpan posisi ikon untuk berbagai resolusi, sehingga dapat membantu dengan masalah pengantian / pengubahan resolusi.

Sejauh yang saya lihat tidak ada fitur bawaan untuk menyelaraskan ikon di sebelah kanan desktop.

Tidak tahu solusi mudah apa pun kecuali menonaktifkan opsi "Atur Otomatis". Kemudian Anda dapat mengatur ikon sesuka Anda. Jika Anda menginginkan lebih banyak kebebasan dalam menempatkannya, Anda bahkan dapat menonaktifkan "Sejajarkan dengan kisi".

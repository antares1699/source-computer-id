Menjalankan pencarian dan xargs di latar belakang

Saya ingin menjalankan perintah ini di latar belakang karena prosesnya bisa memakan waktu lama. Bagaimana cara mengirimnya ke proses latar belakang?

find /tmp/ -type f -mtime +3 | xargs rm -Rf

Ini tidak bekerja:

find /tmp/ -type f -mtime +3 | xargs rm -Rf &

Bagaimana saya bisa melakukan ini?

Jika Anda ingin menjalankan keduanya di latar belakang, letakkan di subkulit:

(find /tmp/ -type f -mtime +3 | xargs rm -Rf) &

Tapi tolong, jangan lakukan ini . Mengirimkan keluaran find ke xargs tidak aman kecuali Anda menggunakan opsi berikut, yang didukung di GNU dan BSD find dan xargs:

find … -print0 | xargs -0 …

Jika find mengembalikan file dengan spasi dalam namanya, Anda dapat - bahkan tanpa menyadarinya - menghapus folder yang salah secara permanen. Bacalah dengan seksama manual find dan bagian tentang menghapus file untuk info lebih lanjut.

Cara teraman, dalam kasus Anda, adalah:

find /tmp/ -type f -mtime +3 -delete &

Bergantung pada perangkat keras tempat sistem file diletakkan, mungkin lebih cepat untuk menghapus secara paralel:

( find . | parallel -X rm ) &

Berbeda dengan xargs GNU Parallel menangani spasi dengan benar, "dan 'dalam nama file. Hanya jika Anda memiliki nama file yang berisi baris baru, Anda memerlukan -print0 / -0.

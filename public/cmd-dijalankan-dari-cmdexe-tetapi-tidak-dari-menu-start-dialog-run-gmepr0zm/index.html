.cmd dijalankan dari cmd.exe tetapi tidak dari menu start / dialog Run

masalah

Jika saya menjalankan file .cmd dari cmd.exe itu berfungsi dengan baik. Namun jika saya menjalankannya dari baris Start menu / Run, itu terbuka di Notepad ++, editor saya. ( edit Saya sudah menginstalnya selama bertahun-tahun, tidak memperbaruinya baru-baru ini. Namun, fenomena tahun .cmd ini baru dimulai baru-baru ini.)

Kedua kali, saya menggunakan jalur absolut penuh. Saya menggunakan windows 7 x64.

registri

HKLM/Software/Classes/cmdfile/shell/open/command saya (Default) disetel ke "%1" %* dan .cmd (Default) adalah cmdfile di HKLM dan HKCU dan tidak ada cmdfile di HKCU.

Namun kunci lain yang tidak saya periksa / tidak mengerti artinya atau signifikansinya, khususnya saya tidak tahu cara mudah untuk mengikuti nilai clsid, dan tahu apa artinya sebagai kunci yang diberikan.

menu konteks

Jika saya klik kanan .cmd teks tebal adalah namanya ( edit ) mengatakan " Buka Tapi bahkan dari sini jika saya mengklik buka, itu terbuka di notepad ++, editor teks saya.

ikon (diedit) Jika saya membuka folder di explorer, ikonnya adalah ikon Notepad ++.

pertanyaan

Mengapa baris Jalankan berperilaku berbeda dari baris perintah?

Bagaimana cara memulihkan .cmds saya untuk dijalankan dari dialog Jalankan juga?

Buat cadangan registri.

Setelah itu Anda harus menghapus nilai apa pun dengan "Notepad ++" di bawah kunci berikut:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FileExts\.cmd

Satu-satunya kunci yang saya miliki di bawah sana adalah:

Di Windows 7, jika Anda tidak ingin mengedit registri, itu Mulai> Panel Kontrol> Program Default> Mengaitkan jenis file atau protokol dengan program tertentu. (Seringkali membutuhkan waktu lama untuk memuat!) ".Cmd" harus dikaitkan dengan "Windows Command Script". Karena jelas tidak, klik Ubah Program, Jelajahi \ windows \ system32 \ cmd.exe dan pilih, centang kotak untuk "selalu gunakan", dan itu akan memperbaiki pengaitan.

Atau Anda dapat melakukannya langsung di registri, tetapi lokasinya berada di bawah HKEY_CLASSES_ROOT, tempat asosiasi file ditetapkan. File CMD secara default dikaitkan dengan "penangan persisten teks biasa". Defaultnya adalah:

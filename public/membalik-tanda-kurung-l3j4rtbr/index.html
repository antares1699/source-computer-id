Membalik Tanda Kurung

Saya memiliki beberapa teks dengan tanda kurung terbalik. Misalnya, alih-alih (text-in-parentheses), saya memiliki )text-in-parentheses(

Apakah ada cara mudah untuk membalik semua tanda kurung di teks? Jika ya, bagaimana caranya?

Memperbarui: Ini adalah file teks biasa, setiap tanda kurung dibalik. Saya menggunakan Windows 7.

Powershell

Versi pendek

(Get-Content "in.txt" -raw) -replace "\(", "`0)" -replace "\)", "(" -replace "(`0\()", ")" >> "out.txt"

Mungkin juga:

Set-Content "out.txt" ((Get-Content "in.txt" -raw) -replace "\(", "`0)" -replace "\)", "(" -replace "(`0\()", ")")
  • Bacaan in.txt
  • Menulis ke out.txt (gulir secara horizontal di contoh kode pertama untuk melihat akhir baris)

Versi panjang

$in = "in.txt";
$out = "out.txt";

(Get-Content $in -raw) -replace "\(", "`0)" -replace "\)", "(" -replace "(`0\()", ")" >> $out

Mungkin juga:

Set-Content $out ((Get-Content $in -raw) -replace "\(", "`0)" -replace "\)", "(" -replace "(`0\()", ")")

Penjelasan

Harap abaikan sebentar garis miring terbalik - ini hanya bagus untuk mengawal tanda kurung yang memang memiliki arti khusus dalam ekspresi reguler.

Pertama-tama kami mengganti setiap kemunculan ( dengan NUL) (lihat Karakter NUL ). Kemudian, kami mengganti setiap ) dengan (. Sebagai langkah terakhir, NUL( diganti kembali oleh ).

Dengan demikian, kami memastikan bahwa kami tidak melakukan "penggantian ganda", misalnya:

Initial string value:
(test)

After replacing "(" by ")"
)test)

After replacing ")" by "("
(test(

Peringatan

File itu sendiri tidak boleh berisi urutan NUL). Ini akan menyebabkan salah tafsir skrip Powershell saya karena saya menggunakan karakter NUL sebagai indikator khusus (misalnya, situasi yang sama muncul saat menggunakan layar hijau dan memakai baju hijau).

Saya yakin ada cara satu langkah untuk melakukan ini, tetapi cara cepat adalah dengan mengganti semua satu paren ke karakter unik, seperti ^, lalu alihkan semua paren lain ke paren yang tepat, lalu ganti karakter khusus ke paren lawan.

Misalnya di notepad:

  1. Ganti semua (dengan ^
  2. Ganti semua) dengan (
  3. Ganti semua ^ dengan)

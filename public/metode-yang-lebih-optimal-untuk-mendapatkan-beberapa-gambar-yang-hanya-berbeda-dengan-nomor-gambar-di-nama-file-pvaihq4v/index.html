Jika Anda menggunakan bash yang cukup modern, Anda dapat menggunakan ekspansi shell seperti ini:

wget http://www.example.com/file/image{1..99}.png

Di atas akan mencoba untuk mendapatkan image1.png hingga image99.png. Jelas, pilih kisaran yang sesuai. File-file yang tidak ada itu seharusnya memunculkan kesalahan, tetapi tidak menyebabkan masalah sebenarnya. Jika mau, Anda mungkin dapat menekan kesalahan dengan beberapa opsi wget, atau dengan mengarahkan STDERR ke / dev / null:

wget http://www.example.com/file/image{1..99}.png 2> /dev/null

Untuk jumlah file yang sangat besar, Anda mungkin menemui batasan baris perintah - ada jumlah maksimum argumen yang dapat berada di baris perintah. Anda sangat tidak mungkin mengalami masalah ini, tetapi jika Anda berurusan dengan file dalam jumlah yang sangat besar, loop for mungkin lebih aman:

for f in {1..1000}; do wget http://www.example.com/file/image$f.png; done
##  and the version sending STDERR to /dev/null:
for f in {1..1000}; do wget http://www.example.com/file/image$f.png 2> /dev/null; done

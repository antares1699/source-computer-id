IP Lokal Statis atau Dinamis berdasarkan Lokasi Fisik

Di jaringan lokal saya (di rumah), saya memiliki mesin Linux bernama A, yang terhubung ke Internet melalui modem. Saya ingin memberikan alamat IP lokal statis saat berada di rumah, tetapi dialokasikan secara dinamis jika tidak. Bagaimana saya mencapai ini?

Saya pikir salah satu cara saya dapat melakukan ini adalah dengan memeriksa alamat IP publik saya dan melihat apakah itu dalam rentang alokasi ISP saya, tetapi ini tampaknya tidak elegan dan saya tidak tahu bagaimana melakukannya. Saya juga dapat memeriksa apakah modem saya ada antara saya dan Internet, tetapi sekali lagi, saya tidak tahu caranya.

Saya sangat menghargai saran apapun. Terima kasih atas perhatiannya!

Lebih mudah untuk mengkonfigurasi router Anda daripada komputer Anda. Untuk bekerja, Anda memerlukan alamat mac dari kartu nirkabel Anda. Ini biasanya disebut 'Alamat perangkat keras' dan Anda dapat melihatnya di panel kontrol dengan mengetikkan ifconfig di konsol Anda.

Ketika Anda mendapatkan alamat MAC Anda (akan terlihat seperti AA: BB: CC: DD: EE: FF) Anda kemudian dapat mengkonfigurasinya di router Anda. Opsinya mungkin di bawah konfigurasi DHCP yang disebut sewa statis.

Di sana Anda dapat menetapkan ip tetap ke alamat mac. Jadi, untuk alamat mac Anda memasukkan yang baru saja Anda peroleh, dan untuk IP Anda memasukkan IP yang diinginkan.

Setelah itu, meskipun komputer Anda disetel ke konfigurasi otomatis, ia akan selalu mendapatkan IP tetap dari router.

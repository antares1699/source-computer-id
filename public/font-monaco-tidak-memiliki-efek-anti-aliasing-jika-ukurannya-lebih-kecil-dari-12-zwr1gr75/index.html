Font Monaco tidak memiliki efek anti-aliasing jika ukurannya lebih kecil dari 12

Saya ingin menyetel font editor saya sebagai Monaco-11, tetapi tidak memiliki efek anti-aliasing. Dan ketika saya menyetelnya ke Monaco-12, efek anti-aliasing berfungsi. Bagaimana saya bisa membuatnya juga bekerja untuk Monaco-11? Saya menggunakan Ubuntu 9.10

Saya menduga bahwa anti-aliasing sengaja dinonaktifkan untuk ukuran font kecil. Hal ini sering dilakukan karena pada ukuran kecil anti-aliasing membuat font menjadi sangat buram.

Anda mungkin dapat mengkonfigurasi perilaku ini. Saya percaya di Ubuntu ini dikontrol melalui Fontconfig . Fontconfig memiliki file konfigurasi di tempat standar (/ etc / font, dan ~ / .fonts.conf). Lihat panduan pengguna untuk detailnya.

Anda mungkin ingin menambahkan perintah edit untuk mengaktifkan anti-aliasing. Lihat mis. entri blog ini . Anda mungkin harus sedikit bereksperimen ...

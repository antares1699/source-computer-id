Bagaimana cara memindahkan trek dari satu daftar putar foobar2000 ke yang lain?

Saya suka mempertahankan beberapa playlist di foobar2000, memindahkan seluruh album dari satu playlist ke playlist lainnya. Saya biasanya melakukan ini dengan memilih album dan menggunakan Utilities -> Send to playlist... untuk memilih playlist baru. Saya kemudian harus kembali ke playlist lama, dan menghapus album dari sana.

Saya ingin membuang langkah terakhir ini, "memotong dan menempel" trek daripada "menyalin" mereka. Apakah ada item menu konteks atau komponen yang memungkinkan saya melakukan ini?

Jika Anda memilih sekelompok lagu, Anda dapat mengklik dan menyeret lagu ke playlist baru. Ini akan memiliki perilaku yang telah Anda lihat.

Alih-alih menggunakan tombol kiri mouse untuk mengklik dan menyeret lagu, Anda harus menggunakan tombol mouse kanan untuk menyeret lagu baru ke playlist baru. Jika Anda menjatuhkannya pada nama daftar putar, Anda akan mendapatkan menu konteks dengan opsi Copy here + remove from the source playlist

Triknya di sini adalah dengan melepaskan tombol mouse sebelah kanan sebelum itu mengubah playlist. Jika Anda tidak segera melepaskan tombol mouse maka itu tidak akan menghapus file dari playlist sumber.

Mengklik kanan dan menyeret ke tempat tertentu dalam daftar putar baru akan memberi Anda menu konteks dengan opsi Move here, tetapi ini akan tidak hapus dari playlist lama.

Pencetak Direktori Karen dapat mencetak nama setiap file pada drive, bersama dengan ukuran file, tanggal dan waktu modifikasi terakhir, dan atribut (Hanya-Baca, Tersembunyi, Sistem, dan Arsip)! Dan sekarang, daftar file dapat diurutkan berdasarkan nama, ukuran, tanggal dibuat, tanggal terakhir diubah, atau tanggal akses terakhir. Tentu saja Anda dapat menyimpan ke file daripada mencetak.

Karen's Directory Printer adalah freeware.

Atau jalankan perintah DIR dari ColorConsole lalu tekan tombol HTML atau RTF

ColorConsole adalah freeware dan portabel.

Itu Pencetak Direktori JR utilitas (freebie portabel lainnya) memungkinkan Anda untuk mencetak daftar setiap subdirektori dan / atau file yang terdapat dalam direktori dan / atau subdirektori. Ini mendukung nama file gaya huruf kecil / dos, ukuran file dalam KB / byte, atribut, panjang file khusus, dll.

Buka prompt perintah di folder yang ingin Anda daftarkan dan ketik "dir> output.txt".

Ada XP Powertoy yang menambahkan "Buka Jendela Perintah di Sini" ke menu konteks saat Anda mengklik kanan pada nama folder.

Ini tidak akan bagus, tetapi Anda dapat mengontrol hasilnya. Ketik "dir /?" untuk daftar opsi.

Displays a list of files and subdirectories in a directory.

DIR [drive:][path][filename] [/A[[:]attributes]] [/B] [/C] [/D] [/L] [/N]
  [/O[[:]sortorder]] [/P] [/Q] [/S] [/T[[:]timefield]] [/W] [/X] [/4]

  [drive:][path][filename]
              Specifies drive, directory, and/or files to list.

  /A          Displays files with specified attributes.
  attributes   D  Directories                R  Read-only files
               H  Hidden files               A  Files ready for archiving
               S  System files               -  Prefix meaning not
  /B          Uses bare format (no heading information or summary).
  /C          Display the thousand separator in file sizes.  This is the
              default.  Use /-C to disable display of separator.
  /D          Same as wide but files are list sorted by column.
  /L          Uses lowercase.
  /N          New long list format where filenames are on the far right.
  /O          List by files in sorted order.
  sortorder    N  By name (alphabetic)       S  By size (smallest first)
               E  By extension (alphabetic)  D  By date/time (oldest first)
               G  Group directories first    -  Prefix to reverse order
  /P          Pauses after each screenful of information.
  /Q          Display the owner of the file.
  /S          Displays files in specified directory and all subdirectories.
  /T          Controls which time field displayed or used for sorting
  timefield   C  Creation
              A  Last Access
              W  Last Written
  /W          Uses wide list format.
  /X          This displays the short names generated for non-8dot3 file
              names.  The format is that of /N with the short name inserted
              before the long name. If no short name is present, blanks are
              displayed in its place.
  /4          Displays four-digit years

Switches may be preset in the DIRCMD environment variable.  Override
preset switches by prefixing any switch with - (hyphen)--for example, /-W.

Anda bisa mendapatkan tangkapan layar dan mungkin mengubahnya menjadi teks menggunakan Snagit, atau cukup buka Command Prompt dan ketik:

DIR > somefilename.txt

Ini akan menghasilkan daftar direktori (perintah DIR) dan mengarahkan outputnya ke file teks bernama somefilename.txt. Jelas Anda dapat menamai file itu apa pun yang Anda inginkan.

Anda kemudian dapat membuka file teks di Excel, dan menggunakan Alat -> Data -> Teks ke Kolom untuk mengubahnya menjadi format yang dapat dibaca spreadsheet.

Jawaban sudah memberikan daftar apa yang akan saya sarankan pada awalnya, semoga jawaban ini hanya memperkuatnya:

tree /f /a

... yang mencetak pohon rekursif dari daftar direktori, termasuk file (/ f), dan menggunakan keluaran ASCII (/ a).

Opsi lainnya adalah di bawah ini, yang akan melakukan daftar rekursif (/ s) dalam format kosong (/ b).

dir /s /b

Ini memiliki nilai tambah tambahan yang mencantumkan jalur lengkap untuk file. Jelasnya, jika Anda tidak menggunakan / s switch, itu juga hanya akan menghasilkan untuk direktori saat ini.

Anda telah melihat pengalihan keluaran seperti di bawah ini, yang mengalihkan keluaran standar ke file:

dir /s /b>listing.txt

Juga tersedia >> yang menambahkan keluaran ke file:

dir *.exe /b>listing.txt
dir *.config /b>>listing.txt

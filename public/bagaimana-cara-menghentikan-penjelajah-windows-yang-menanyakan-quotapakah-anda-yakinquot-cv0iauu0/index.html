Bagaimana cara menghentikan penjelajah Windows yang menanyakan "apakah Anda yakin?"?

Setiap kali saya memindahkan atau menghapus file di Windows XP, penjelajah tampaknya harus bertanya kepada saya, apakah saya benar-benar ingin melakukannya, setiap kali ada file hanya-baca, atau file yang dapat dieksekusi, atau karena beberapa alasan aneh lainnya. Ini sangat menjengkelkan, terutama dalam hal file yang dapat dieksekusi, karena mereka pasti tidak lebih berharga dari file biasa. Apalagi itu melambat operasi yang seharusnya tidak memakan waktu.

Saya kebanyakan menggunakan alat yang lebih baik seperti FreeCommander, tetapi terkadang penjelajah berguna. Adakah cara untuk menonaktifkan "fitur" ini?

Memperbarui: Ini BUKAN tentang konfirmasi, ketika saya menekan DELETE. Konfirmasi ini baik-baik saja dan saya ingin menyimpannya. Yang ingin saya singkirkan, adalah konfirmasi tambahan untuk menghapus atau memindahkan file yang dianggap penting oleh Windows (tetapi saya tidak melakukannya). Misalnya, saat memindahkan direktori besar, Windows tampaknya melihat semua file di dalamnya, untuk menanyakan apakah saya benar-benar ingin memindahkan foo.exe. Saya mengatakan "Ya untuk SEMUA", lalu menemukan file hanya-baca, saya mengatakan "Ya untuk SEMUA", kemudian menemukan ... dan akhirnya saya menemukan diri saya bersumpah seperti pelaut.

Ini sangat berantakan AutoHotKey skrip yang akan melakukan apa yang Anda inginkan.

Saya bilang berantakan karena ...

  1. Anda menekan tombol hapus Anda. Kemudian mencoba menggunakan Ctrl + c seolah-olah Anda melakukannya di Explorer. Ini juga menyalin string dengan jalur ke file ke clipboard
  2. Itu tidak akan membedakan antara jendela. Kunci hapus Anda hilang jika Anda menggunakan ini.
  3. Itu harus menunggu sebentar sebelum dapat menempelkan path ke command prompt

  4. Ini adalah cara bodoh untuk menekan enter setelah Anda menekan hapus. Serius, itu tidak memperlambat apa pun.

    kirim, {control down} c {control up} Jalankan cmd Tidur, 300 kirim, del / q% clipboard% {return} exit {return}

Sekarang, Anda bisa cukup cantik dengan membungkus semuanya di cek untuk melihat apakah itu folder Explorer. Ini akan menghilangkan dilema mencoba melakukan sesuatu seperti del /q http://superuser.com/questions/pressing-del-then-enter-is-too-slow-for-me jika Anda belum menyalin path file.

Jika Anda membutuhkan info lebih lanjut tentang AutoHotKey, Google adalah teman Anda.

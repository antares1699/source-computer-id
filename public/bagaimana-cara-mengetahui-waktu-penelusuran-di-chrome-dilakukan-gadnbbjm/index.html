menggunakan ChromeHistoryView v1.35 Hak Cipta (c) 2011 - 2018 Nir Sofer

bekerja dengan baik

Deskripsi

ChromeHistoryView adalah utilitas kecil yang membaca file data riwayat browser Web Google Chrome, dan menampilkan daftarnya dari semua halaman Web yang dikunjungi pada hari-hari terakhir. Untuk setiap halaman Web yang dikunjungi, informasi berikut ditampilkan: URL, Judul, Tanggal Kunjungan / Waktu , Jumlah kunjungan, frekuensi pengguna mengetik alamat ini (Hitungan Diketik), Perujuk, dan ID Kunjungan.

Menggunakan ChromeHistoryView

ChromeHistoryView tidak memerlukan proses penginstalan atau file DLL tambahan. Untuk mulai menggunakannya, cukup salin file file yang dapat dieksekusi (ChromeHistoryView.exe) ke folder apa pun yang Anda suka, dan menjalankannya.

NB: - Anda tidak dapat melihat riwayat waktu di google secara default

Ini berlebihan, tapi:

  1. Temukan file / AppData / Lokal / Google / Chrome / Data Pengguna / Default / Riwayat

  2. Salin file ini ke mesin Linux dan jalankan perintah ini:

    sqlite3 History "pilih datetime (visit_time / 1000000-11644473600, 'unixepoch') sebagai datetime, urls.url || '
    'dari kunjungan bergabunglah dengan url di urls.id = visit.url; "> history.html

  3. Buka file history.html yang baru dibuat. Ini akan memiliki daftar seluruh riwayat penjelajahan Anda bersama dengan tanggal dan waktu Anda mengunjungi setiap halaman.

Menjawab karena saya baru saja memiliki pertanyaan ini dan menemukan bahwa jawabannya tampak terlalu rumit.

Saat Anda melihat halaman sejarah salah satu dari Menu > History > History, atau sebagai CTRL> - H , atau sebagai chrome://history/, tampilan menunjukkan waktu (HH:mm) dan url, dikelompokkan berdasarkan hari.
Jika Anda melakukan pencarian, maka hasilnya menunjukkan tanggal (mmm, dd, yyyy) dan url.

Dalam bentuk mana pun, mengarahkan kursor ke atas tanggal atau waktu akan menampilkan stempel waktu yang sebenarnya. Ini disajikan dalam waktu lokal Anda. (GMT melemparkan saya sampai saya menyadari itu berarti "GMT - 4 jam".)

Ex:

Mon Oct 21, 2019 08:08:03 GMT -0400 (Eastern Daylight Time)

PS: Ini dengan Chrome 77.0.3865.120 (Build Resmi) (64-bit)

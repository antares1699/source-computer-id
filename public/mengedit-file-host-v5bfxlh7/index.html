Mengedit file host

Saya ingin mengedit Windows TCP / IP tuan rumah file (di c:\windows\system32\drivers\etc\hosts). Saya bertanya-tanya, apa cara terbaik untuk melakukan ini (di Windows XP)?

Misalkan saya ingin mengalihkan, misalnya, situs a.com ke google.com. Saya dapat menemukan alamat IP dari a.com, tetapi bagaimana cara yang benar untuk mengedit file HOSTS? Saya berpikir untuk menggunakan sintaks ini:

this.is.ip.address   google.com

Namun, ada dua hal:

  • Apakah ini sintaks yang benar? Jika ya, apakah ada cara untuk melakukannya tanpa harus mencari alamat IP dari a.com? Apakah hanya menempatkan a.com ke tempat alamat IP saat ini berfungsi?
  • Apa cara terbaik untuk mengedit file ini? Saya tidak dapat menyimpannya karena sedang digunakan. Biasanya dalam keadaan ini, saya akan masuk ke Safe Mode dan mengeditnya dari sana, tetapi apakah ada cara yang lebih mudah (saya berpikir untuk mematikan beberapa proses sistem ...)?

Anda dapat menemukan alamat IP situs menggunakan ping di command prompt atau menggunakan alat pencarian . Pastikan juga Anda menangani semua kombinasi:

111.111.111.111   a.com
111.111.111.111   www.a.com
111.111.111.111   222.222.222.222

File ini hanya baca, Anda dapat mengeditnya dengan mengklik kanan dan masuk ke properti, lalu menonaktifkan hanya baca untuk sementara hingga Anda selesai mengedit.

Anda seharusnya dapat mengedit file host dengan Notepad, tetapi ini adalah file yang dilindungi secara default di Windows Vista. Anda mungkin tidak dapat menyimpan file karena masalah izin, bukan karena sedang digunakan.

Untuk mengedit file dengan Notepad, buka di bawah akun Administrator, dengan mengklik kanan Notepad di menu Start Anda dan memilih perintah Run as Administrator.

Jika direktori kosong lakukan ini, Jika direktori tidak kosong lakukan itu [duplikat]

Sejauh ini saya punya ini. Apakah ada cara yang lebih efisien?

if ls -1 $HOME/path/to/folder/* >/dev/null 2>&1
then
        echo Directory is not empty.
else
        echo Directory is empty.
fi

Ini adalah latihan standar di semua buku / blog / manual tentang shell: in bash

 [ "$(ls -A /path/to/directory)" ] && echo "Not Empty" || echo "Empty"

Bash tidak memiliki tes standar untuk direktori yang kosong, seperti [-f file], jadi kita harus menulisnya. Tes && lalu 1 || atau 2 adalah cara standar untuk menyingkat pernyataan bersyarat dari formulir

if condition is satistfied
    then do 1
else
    do 2
fi

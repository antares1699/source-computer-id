Bagaimana saya bisa memaksa pemeriksaan disk ketika Windows 7 melaporkan "Pemeriksaan disk telah dibatalkan"?

Saya telah menjadwalkan pemeriksaan disk pada drive C saya. Itu tidak dapat langsung memeriksa disk, karena sedang digunakan, sehingga menjadwalkan pemeriksaan disk untuk boot ulang berikutnya. Namun, ketika saya dengan patuh reboot, pemeriksaan disk tidak terjadi.

Saya menemukan beberapa instruksi yang memberitahu saya untuk menjalankan msconfig dan memaksa reboot dalam Safe Mode. Ketika saya melakukan ini (dan menjadwalkan pemeriksaan disk lain), saya mendapat pesan berikut: Disk checking has been cancelled.

Bagaimana saya bisa memaksa pemeriksaan disk untuk benar-benar, Anda tahu, periksa?

Berdasarkan Dukungan Microsoft , untuk memeriksa apakah jadwal pemeriksaan disk Anda benar-benar diatur untuk melakukan pemindaian pada boot berikutnya (setelah menjalankan perintah CHKDSK /F /R), Anda dapat melihat di registri Windows. Kunci ini (HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\BootExecute) harus memiliki entri:

autocheck autochk * / r \ DosDevice \ C:

Jika hanya autocheck autochk *, maka jadwal dinonaktifkan dan Anda harus menjalankan CHKDSK /F /R, atau secara manual mengubah registri Anda dengan memasukkan argumen /r\DosDevice\C.

Melalui http://www.sevenforums.com/performance-maintenance/71175-disk-check-has-been-canceled-reboot-3.html

  1. Pastikan Anda akan melakukan boot ulang dalam Mode Normal. Mulai ulang.
  2. Tekan F8 berulang kali segera setelah layar boot Anda muncul. (Komputer, yang muncul sebelum Windows.)
  3. Ini akan membawa Anda ke layar seperti DOS. Pilih Repair Recovery Manager. (Anda mungkin diminta untuk masuk sebagai pengguna administrator sehingga Anda dapat mengakses Prompt Perintah.)
  4. Dari Repair Recover Manager, pilih Command Prompt. Ini akan memulai jendela prompt perintah di drive X :.
  5. Masukkan C: untuk beralih ke drive C.
  6. Masukkan chkdsk /f (atau chkdsk /r)
  7. Saat diminta untuk memaksa turun, masukkan Y
  8. Disk sekarang akan diperiksa.
  9. Catat hasilnya dan tutup prompt perintah. (Punyaku tidak bisa menyimpan hasil ke logfile, karena berada di drive C. Bukan smash kereta.)
  10. Mulai ulang.

Saya telah menemukan bahwa teknik di atas masih gagal untuk chkdsk disk, jadi saya menggunakan Windows7 bootable lain untuk menjalankan chkdsk pada disk ini.

Ini bisa berupa CD / DVD Windows7 yang dapat di-boot sebagai disk penyelamat, atau Windows 7 pada mesin fisik yang benar-benar terpisah, atau Windows 7 yang dijalankan sebagai mesin virtual.

Hubungkan sumber Windows 7 sebagai Harddisk penyimpanan baru:

  • untuk CD / DVD. cukup boot mesin dengan CD / DVD dan ikuti petunjuk di atas
  • untuk mesin fisik, hubungkan Hard-drive asli ke mesin baru dan boot mesin "penyelamat" dan ikuti petunjuk di atas
  • untuk mesin virtual, hubungkan Windows 7 di atas sebagai penyimpanan Hardrive dan boot mesin virtual, dan ikuti petunjuk di atas.

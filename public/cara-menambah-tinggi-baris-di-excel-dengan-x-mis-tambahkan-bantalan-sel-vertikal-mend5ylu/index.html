Cara menambah tinggi baris di Excel dengan X. mis. Tambahkan bantalan sel vertikal

Saya memiliki laporan ini dan saya ingin meningkatkan bantalan sel vertikal agar lebih mudah dibaca.

Sekarang, biasanya Anda bisa memilih beberapa baris dan mengubah tinggi baris sampai Anda puas.

Namun, dalam kasus ini, saya memiliki sel dengan teks yang dibungkus dan beberapa baris. Saya tidak dapat mengubah tinggi baris, karena teks yang dibungkus akan terpotong.

Yang ingin saya lakukan adalah meningkat tinggi baris sekelompok sel dengan nilai tertentu. Daripada hanya mengubahnya menjadi nilai tertentu. yaitu.

rowheight = rowheight + X

Tidak

rowheight = X

Sub fixRowHeight()

Dim oRange As Excel.Range
Dim oRow As Excel.Range
Dim dblFactor As Double

Set oRange = Sheets(1).Range("A1:B25")
dblFactor = 5

For Each oRow In oRange.Rows

    oRow.RowHeight = oRow.RowHeight + dblFactor

Next oRow

End Sub

Saya baru saja mengalami masalah ini dan solusi di atas tidak memuaskan dalam beberapa keadaan.

  1. Sisipkan kolom di suatu tempat - mungkin kolom A. Pilih seluruh kolom.
  2. Beri batas atas / bawah dari jenis yang paling tebal - batang lemak padat - terapkan / ok.
  3. Pilih semua kolom lain yang ingin Anda padukan. Pusatkan mereka secara vertikal.
  4. Sembunyikan kolom berbatasan & simpan.

Ini memberi Anda bantalan atas dan bawah sekitar 3 unit / 5px atas & bawah. Dengan arial, itu cukup untuk menjaga bagian atas dan ekor tidak menyentuh tepi sel.

Kode VBA yang diberikan bagus. Tetapi gagal segera setelah spreadsheet Anda dilihat oleh seseorang yang perlu mempersempit beberapa kolom dan menghapus pemformatan manual atau menerapkan pemformatannya sendiri. Jika pengguna tahu apa yang mereka lakukan, mereka akan menekan Ctrl + A dan mengklik dua kali indikator baris untuk melakukan tinggi otomatis semua baris.

Menggunakan pembatas memberikan solusi yang akan selalu berfungsi, kecuali jika diformat secara manual.

Untuk menjelaskan pada jawabannya dari Tiago Cardoso , Saya menambahkan kondisi untuk memeriksa untuk memastikan tinggi baris maksimum tidak terlampaui saat meningkatkan tinggi baris dengan variabel faktor pengubah ukuran.

Sub FixRowHeight()
    Dim oRange As Excel.Range
    Dim oRow As Excel.Range
    Dim dblFactor As Double

    Set oRange = Sheets(1).Range("A1:O367")
    dblFactor = 12

    For Each oRow In oRange.Rows
        If oRow.RowHeight + dblFactor <= 409.5 Then
            oRow.RowHeight = oRow.RowHeight + dblFactor
        Else
            oRow.RowHeight = 409.5
        End If
    Next oRow
End Sub

Berikut makro untuk membantu Anda.

Pertama, ini mengatur perataan vertikal untuk setiap sel menjadi Center.

Selanjutnya, dibutuhkan file tinggi saat ini dari setiap sel dan menambahkan 7.5 untuk itu (ini setara dengan 10 piksel ). Pada akhirnya, ini menambahkan bantalan 5-piksel ke bagian atas dan bawah sel. Sesuaikan saja nilai ini sesuai dengan preferensi Anda.

Sub PadCells()

Set cr = Selection

For Each cr In cr.Rows
    newHeight = cr.RowHeight + 7.5 
    cr.VerticalAlignment = xlCenter
    cr.RowHeight = newHeight
Next cr

End Sub

Menggunakan:

  1. Pilih sel yang ingin Anda sesuaikan tingginya.
  2. Jalankan makro.

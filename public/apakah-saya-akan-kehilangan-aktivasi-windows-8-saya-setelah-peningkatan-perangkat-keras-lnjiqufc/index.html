Apakah saya akan kehilangan aktivasi Windows 8 saya setelah peningkatan perangkat keras?

Windows 8 Pro w / WMC diinstal pada Asus M3A78-EM dengan GPU terintegrasi. Saya ingin meningkatkan GPU dan telah membeli yang kompatibel tetapi belum menginstalnya. Saya juga ingin meningkatkan CPU-nya.

Saya baru-baru ini menemukan beberapa blog di mana dinyatakan bahwa seseorang mungkin kehilangan aktivasi setelah peningkatan perangkat keras seperti GPU atau CPU atau HDD.

Ini benar atau tidak? Jika demikian, dapatkah saya tetap aktif tanpa menelepon Microsoft ?? Kedua versi tersebut dibeli secara legal dari Microsoft.

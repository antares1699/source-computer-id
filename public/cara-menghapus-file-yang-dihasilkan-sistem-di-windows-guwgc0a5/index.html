Cukup SHIFT + DEL untuk menghapusnya secara permanen. Itu aman. Ini hanya lokasi ekstraksi sementara untuk Pembaruan Windows, dan seharusnya telah dihapus secara otomatis, tetapi terkadang tidak.

Ada kemungkinan beberapa file di dalam folder mungkin dikunci (seperti dalam kasus Anda), dan jika demikian, Anda dapat boot ke mode aman untuk membunuhnya atau menggunakan Unlocker .

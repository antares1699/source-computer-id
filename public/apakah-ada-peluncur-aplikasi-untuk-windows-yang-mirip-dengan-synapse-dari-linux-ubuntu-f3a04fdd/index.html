Saya menggunakan fitur Mulai pencarian / peluncuran Windows 7, tetapi saya ingin alternatif untuk itu, sesuatu yang lebih dekat dengan Synapse, dengan jendela yang lebih besar, misalnya.

Launchy harus menjadi hal yang Anda cari.
Dikutip dari Wikipedia:

Launchy adalah peluncur aplikasi open source untuk Windows, Mac OS X, FreeBSD, dan GNU / Linux. Ini mengindeks pintasan di menu mulai, dan file di folder tertentu untuk memungkinkan akses lebih cepat ke program tanpa membuka menu mulai itu sendiri, atau menjelajah ke folder yang relevan. [...] Launchy dapat menambahkan folder dan format file ke katalognya untuk diindeks, sehingga dapat meluncurkan hampir semua hal di komputer, termasuk program, URL (juga dari bookmark) dan dokumen; serta menjalankan pencarian Google, Wikipedia, MSN, dan Yahoo, menampilkan hasilnya di browser default.

Alternatif lain bisa jadi Pelaksana . Dari situs web:

Ini adalah peluncur multiguna dan versi windows yang lebih maju dan dapat disesuaikan. Program ini dimulai karena saya muak menghabiskan terlalu banyak waktu untuk mencari program melalui menu start windows saya yang terus berkembang, dan juga saya melewatkan alat yang dapat memudahkan dan mengoptimalkan alur kerja harian saya. Tentu saja sudah ada program seperti ini yang tersedia, tetapi masing-masing memiliki gangguan atau fitur yang hilang atau terlalu (!) Culun.

Opsi terbaik untuk Windows 7 IMO adalah menginstal www.classicshell.net - ini membuat fungsionalitas Pencarian dari Start Menu lebih cerdas dengan menambahkan hal-hal seperti pencarian substring dan meluncurkan pelacakan frekuensi (sehingga ia belajar untuk memilih otomatis program yang paling sering Anda gunakan ).

Masalah utamanya adalah masih bergantung pada fungsionalitas pencarian dasar yang disediakan oleh Windows jadi:

  • Biasanya lambat bereaksi saat pertama kali Anda menggunakannya setelah Windows dimulai ulang.

  • Itu hanya dapat meluncurkan pengaturan sistem / pintasan Panel Kontrol atau pintasan yang dapat ditemukan di Start Menu (Program, dll.) - ia tidak melihat daftar program yang diinstal (sebelumnya "Tambah / Hapus Program"). Cara saya "mengajarkan" program-program baru adalah dengan secara manual membuat pintasan untuk mereka di folder yang tidak biasa di Program / Aksesori / Pintasan Kustom.

Alasan utama saya lebih memilihnya daripada Launchy and Executor dan Enso Launcher adalah karena itu dapat diakses dengan satu penekanan tombol, biasanya tombol Super, setelah itu saya dapat segera mulai mengetik apa yang saya coba luncurkan, yang persis seperti itu. Saya lakukan dengan Synapse di Linux. Dibandingkan dengan Synapse, saya sangat merindukan kemampuan untuk mengetik ekspresi matematika secara langsung di kotak pencarian dan meminta mereka menghitungnya secara langsung.

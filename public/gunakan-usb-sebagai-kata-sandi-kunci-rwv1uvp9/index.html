Gunakan USB sebagai kata sandi / kunci

Apakah ada cara untuk mencegah komputer melakukan boot ke Windows kecuali ada perangkat USB yang terpasang? Saya menduga ini sulit / tidak mungkin, dalam hal ini apakah ada cara untuk mencegah Windows membuka / masuk kecuali perangkat USB dicolokkan?

Misalnya, jika seorang peretas memiliki akses jarak jauh ke komputer saya, dia tidak akan dapat melewati layar masuk Windows tanpa saya hadir dengan stik USB tanda tangan. Itu juga hanya menambahkan tingkat perlindungan fisik, katakanlah saya telah membagikan kata sandi saya dengan orang lain, mereka masih memerlukan USB saya.

Diekstrak hanya rincian yang diperlukan dari tujuh situs forum

SysKey (SAM Lock Tool) adalah utilitas Windows bawaan yang dapat membantu Anda mengamankan Manajemen Akun Keamanan atau database SAM. SAM Database menyimpan salinan hash dari kata sandi pengguna kami, yaitu dienkripsi dengan kunci startup sistem yang disimpan secara lokal secara default.

Anda dapat menggunakan SysKey untuk menyimpan kunci startup secara lokal di Windows atau di file Flash disk. Memiliki kunci startup disimpan di USB flash drive sebagai gantinya akan memberi Anda keamanan yang lebih baik karena USB flash drive harus disambungkan saat startup untuk dapat masuk dan mendapatkan akses ke Windows.

Anda harus masuk sebagai administrator untuk dapat melakukan langkah-langkah dalam tutorial ini.

Ini hanya akan mengunci komputer Windows saat boot dari startup dingin setelah restart atau matikan. Itu tidak akan mengunci Windows setelah log nonaktifkan, kunci, atau alihkan pengguna.

Setelah Windows dimulai setelah masuk, Anda tidak perlu lagi sambungkan flash drive USB. Itu hanya diperlukan untuk menjadi terhubung saat sistem memulai saat boot.

Anda masih dapat terus menggunakan flash drive USB sebagai file normal. Hanya saja, jangan pernah menghapus file StartKey.Key di dalamnya, atau Anda tidak akan dapat membuka kunci Windows dengannya lagi.

Kunci startup yang ditempatkan pada flash drive USB hanya bagus untuk didapatkan akses untuk masuk ke Windows tertentu tempat ia dibuat.

Catatan: Windows tidak akan memulai sampai Anda menghubungkan kunci startup USB.

Untuk membuat Windows Memerlukan Kunci USB saat Memulai untuk Membuka Kunci:

  1. Anda perlu mengubah huruf drive dari flash drive USB agar memiliki huruf A.

  2. Tekan tombol Windows + R untuk membuka file Jalankan dialog , Tipe syskey , dan klik / ketuk OK.

  3. Jika diminta oleh UAC, klik / ketuk Ya (Windows 7/8) atau Lanjutkan (Vista).

  4. Klik / ketuk Memperbarui .

  5. Pilih (dot) System Generated Password, pilih (dot) the Simpan Kunci Startup di Floppy Disk opsi, dan klik / ketuk OK.

Jika Anda sudah memiliki file StartKey.Key di USB, maka file lama yang sudah ada akan diganti namanya menjadi StartKey.Bak untuk yang baru menjadi diselamatkan untuk itu.

Anda disarankan untuk menyimpan cadangan file StartKey.Key ini di lokasi lain jika USB rusak atau hilang. Dengan cara ini Anda akan dapat menyalinnya ke USB lain dengan huruf drive A menjadi bisa mendapatkan akses ke Windows lagi.

Untuk membuat Windows Tidak Memerlukan Kunci USB saat Memulai:

  1. Lakukan semua langkah yang disebutkan di atas sampai nomor 4.
  2. Pilih Simpan Kunci Startup Secara Lokal opsi di langkah ke-5.

Anda bertanya apakah ada cara untuk mencegah bootup kecuali jika USB Key dicolokkan. Ini mungkin, dan ini membantu melindungi data Anda jika harddisk / komputer Anda dicuri.

Namun, Anda juga menyebutkan bahwa Anda ingin Kunci USB mencegah seseorang dengan akses jarak jauh ke komputer Anda untuk masuk. Bagian kedua dari pertanyaan Anda ini tidak benar-benar masuk akal. Jika seseorang memiliki akses jarak jauh ke sistem Anda, ia menganggap bahwa sistem sudah di-boot. Dalam hal ini, Kunci USB yang diperlukan untuk boot tidak ada relevansinya.

Mungkin yang sebenarnya Anda tanyakan adalah apakah ada cara untuk menonaktifkan Akses Jarak Jauh, atau apakah ada cara untuk hanya aktifkan Akses Jarak Jauh ketika kondisi tertentu terpenuhi (misalnya jika Kunci USB dimasukkan). Salah satu cara untuk melakukannya adalah dengan menulis skrip. Untuk saat ini, dalam jawaban ini, saya hanya akan menunjukkan kepada Anda bagaimana membutuhkan USB Key untuk boot ke Windows.

Saya berasumsi bahwa Anda tidak memiliki TPM (Trusted Platform Module). Kebanyakan orang tidak memiliki ini di PC mereka. Jika kamu melakukan memiliki TPM, Anda mungkin dapat melewati langkah 1 hingga 4. Jika tidak tahu apakah Anda memiliki TPM atau tidak, Anda harus melakukan langkah 1 hingga 4. Meskipun Anda melakukan memiliki TPM, melakukan langkah-langkah tersebut seharusnya tidak membahayakan.

Terlampir di akhir jawaban adalah beberapa gambar yang relevan dengan langkah-langkah tersebut.

  1. Buka Editor Kebijakan Grup. Lakukan ini dengan mencari gpedt.msc di Pencarian Windows.
  2. Arahkan ke Computer Configuration/Administrative Templates/Windows Components/BitLocker Drive Encryption/Operating System Drives.
  3. Klik Ganda Require Additional Authentication at Startup.
  4. Pilih Enable dan pastikan Allow BitLocker without a... dicentang.
  5. Buka pengaturan BitLocker dengan mencari Manage BitLocker.
  6. Pilih Boot Drive Anda (Biasanya C:) dan klik Turn on BitLocker.
  7. Masukkan Kunci USB Anda, dan ikuti langkah-langkah yang diminta.

Setelah selesai, Anda akan diminta untuk memasukkan Kunci USB Anda setiap kali Anda mem-boot Komputer untuk mem-boot ke Windows. Selain itu, jika Hard Drive atau Komputer Anda dicuri, dan pencuri tidak memiliki Kunci USB Anda, data Anda akan dilindungi.

Anda dapat menginstal perangkat Windows ke USB itu sendiri, dan boot dari USB (tetapi pilih perangkat yang cukup cepat untuk tujuan ini, jika tidak maka akan menyedot). Jadi saat USB thumbdrive ada di saku Anda, tidak ada yang perlu di-boot dari PC Anda. Semua partisi lokal dapat dienkripsi dengan TrueCrypt atau perangkat lunak serupa, jadi orang asing lain dengan LiveCD / LiveUSB tidak akan membaca data Anda. Anda dapat memiliki skrip di folder startup instalasi Windows-on-USB Anda untuk memasang otomatis partisi TrueCrypt tanpa memasukkan kata sandi.

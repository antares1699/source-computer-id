Menggunakan "Run as Administrator" tidak melakukan apa-apa

Saya menggunakan PC Windows 7 dan memiliki hak administrator lokal di PC ini. Baru-baru ini, ketika saya mengklik kanan ikon di Windows Explorer dan memilih Jalankan sebagai administrator tidak ada yang terjadi. Tidak ada jendela yang dibuka sama sekali, dan saya tidak melihat proses yang dimulai menggunakan Process Explorer. Saya juga mendapatkan perilaku yang sama saat menggunakan Ctrl + Bergeser + Memasukkan tombol di kotak telusur menu Mulai, seperti yang disarankan di HowToGeek . Apa masalahnya?

Menjalankan program dengan hak administrator membutuhkan informasi aplikasi layanan . Jika layanan itu tidak berjalan, maka Anda tidak dapat lagi menjalankan sesuatu dengan hak istimewa admin, dan lebih buruk lagi, Anda bahkan tidak dapat memulainya. Periksa layanan snap-in (services.msc) dan pastikan bahwa itu tidak dinonaktifkan atau dihentikan. Jika ya, setel ke Manual dan reboot.

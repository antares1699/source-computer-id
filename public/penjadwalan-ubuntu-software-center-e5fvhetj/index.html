'Pusat Perangkat Lunak Ubuntu' tidak secara khusus mengizinkan penjadwalan unduhan / pemasangan perangkat lunak. Namun, penjadwal tugas tujuan umum crontab dapat digunakan untuk mengunduh / menginstal program. Contoh: untuk mendownload / menginstal eclipse pada jam 2:30 pagi:

  1. Pergi ke terminal dan buka crontab untuk mengedit:

    sudo crontab -e

  2. Tambahkan baris berikut:

    * 30 2 * * * apt-get install eclipse > /dev/null
  3. Simpan dan tutup crontab (Ctrl + W untuk menulis)

Crontab akan berjalan pada jam 2:30 pagi dan mendownload / menginstal eclipse. Setiap keluaran terminal dari aktivitas ini akan dikirim ke /dev/null (dibuang).

Tip: situs ini memungkinkan Anda membuat entri crontab secara otomatis:

http://www.htmlbasix.com/crontab.shtml

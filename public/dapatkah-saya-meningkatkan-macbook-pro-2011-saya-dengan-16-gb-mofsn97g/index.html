Dapatkah saya meningkatkan MacBook Pro 2011 saya dengan 16 GB?

Saya memiliki Macbook Pro 15 inci "Awal 2011" yang datang dengan RAM 4 GB dan saya meningkatkan ke 8 GB seminggu setelah membeli.

Saya membeli Modul RAM ini: http://www.amazon.com/gp/product/B002YUF8ZG/ref=oh_details_o01_s00_i00?ie=UTF8&psc=1

Saya ingin meningkatkan ke 16GB, bagaimana saya tahu jika ini didukung?

(Saya menjalankan windows 8 dengan bootcamp, karena saya melakukan banyak pekerjaan di Visual Studio)

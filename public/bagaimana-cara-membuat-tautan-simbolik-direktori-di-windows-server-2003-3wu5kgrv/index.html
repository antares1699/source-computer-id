Bagaimana cara membuat tautan simbolik direktori di Windows Server 2003?

Saya mencoba membuat tautan simbolis direktori di bawah Windows Server 2003 tetapi saya tidak beruntung.

Saya sudah mencoba junction.exe (Sysinternals), ln.exe, dan linkd.exe (Windows Server 2003 Resource Kit).

saya ingin c: \ folder \ subfolder untuk ditautkan \ fileserver \ realSubfolder

Ini berfungsi dengan baik di bawah Windows 7 (menggunakan mklink.exe) tetapi saya tidak bisa membuatnya bekerja di bawah Windows Server 2003.

Adakah yang bisa membantu saya?

Tidak boleh. Symlink direktori diperkenalkan di Windows Vista. Analog terdekat adalah titik reparse direktori (alias junctions), yang dapat dibuat oleh alat Systinternals Junction yang Anda referensikan.

Persimpangan tidak akan mendukung jalur UNC. Ini hanya menyelesaikan untuk jalur lokal.

Persimpangan adalah bagian dari titik reparse, jadi pembatasan yang sama juga berlaku.

Dimulai dengan Vista, Anda dapat membuat symlink, yang akan menyelesaikan jalur jarak jauh. Mereka bahkan akan membiarkan Anda menghubungkan symlink dengan jalur jarak jauh.

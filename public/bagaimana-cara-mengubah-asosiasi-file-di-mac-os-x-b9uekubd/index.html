Bagaimana cara mengubah asosiasi file di Mac OS X?

Secara default, gambar PNG dibuka di Pratinjau. Pada titik tertentu, ini telah berubah di sistem saya sehingga mengklik file PNG diluncurkan Kuas . Meskipun itu aplikasi yang sangat bagus, saya masih ingin defaultnya adalah Preview. Bagaimana cara mengubahnya kembali? (Saya merumuskan judul pertanyaan menjadi sedikit lebih umum - bagaimana melakukan ini untuk semua jenis file?)

Saya mencari cara termudah, paling tersedia, tanpa alat pihak ketiga atau menggunakan baris perintah. Agak mengherankan, System Preferences tampaknya tidak berisi apa pun yang terkait dengan asosiasi file.

Pertanyaan pemula ini mungkin pertanyaan yang mudah ... Saya harap!

  1. Dapatkan Info pada file di Finder (⌘I)
  2. Temukan bagian "Buka dengan:"
  3. Ubah aplikasi ke aplikasi yang Anda inginkan
  4. Klik "Ubah Semua…" untuk menerapkan perubahan pada semua file dengan jenis yang sama.

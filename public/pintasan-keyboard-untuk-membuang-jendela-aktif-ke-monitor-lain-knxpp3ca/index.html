Sejak Windows 7 fungsi ini telah disediakan oleh OS.

Menang + Bergeser + ► : Pindahkan jendela yang difokuskan ke kanan satu layar.

Menang + Bergeser + ◄ : Pindahkan jendela fokus ke kiri layar.

Fungsi-fungsi ini akan saling melengkapi, jadi jika Anda hanya memiliki dua layar Anda tidak akan melihat perbedaan antara kedua fungsi tersebut.

Paksa prompt pada direktori tertentu

Apakah ada cara untuk mengubah prompt pada direktori yang berbeda pada mesin yang sama? Misalnya, saya ingin prompt sederhana "[\ W] \ $" di direktori pengembangan.

Ide ide?

Anda dapat meletakkan sesuatu seperti ini di .bashrc Anda.

prompt-function() {
    case $PWD in
        /a/dev/dir\
        |/another/dev/dir) 
            PS1="[\W] \$"
            ;;
        *)
            # Change this to the default prompt
            PS1="\$ "
            ;;
     esac
}

PROMPT_COMMAND=prompt-function

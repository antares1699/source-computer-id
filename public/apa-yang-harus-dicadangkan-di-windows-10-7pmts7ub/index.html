Apa yang harus dicadangkan di Windows 10

Saya mendapatkan pesan saat boot bahwa hard drive saya akan segera mati, dan harus mengganti hard drive. Saya tidak ingin menggunakan beberapa aplikasi untuk memindahkan data dan program saya, tetapi ingin melakukannya secara manual. Saya mencari daftar umum direktori dan file yang perlu dipindahkan serta cara membuat daftar program yang telah saya instal sehingga saya dapat memutuskan apakah saya ingin menginstalnya kembali di komputer baru.

Saya baru saja mencadangkan Dokumen Saya, Gambar Saya, dan Video Saya. Saya juga mencadangkan file pst Outlook dan bookmark browser. EDIT. Kalau dipikir-pikir, saya mungkin harus mencadangkan C:\Users\Michael di mana Michael adalah satu-satunya pengguna biasa. Saya jarang menggunakan pengguna admin, dan merasa tidak perlu mencadangkannya.

Direktori atau file "tipikal" lainnya yang harus dipindahkan? Bagaimana dengan kunci publik / pribadi, atau apakah ini ditangani di Dokumen Saya?

Bagaimana saya dapat membuat daftar program yang dapat dicetak yang telah saya instal? EDIT. Saya rasa saya menjawab ini menggunakan Get-ItemProperty HKLM:\Software\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall\* | Select-Object DisplayName, DisplayVersion, Publisher, InstallDate | Format-Table –AutoSize

Sistem operasi awalnya adalah Windows 7 Professional (saya masih memiliki DVD) dan diperbarui ke Windows 10 secara gratis. Apakah saya perlu mencadangkan lisensi apa pun?

Ketika saya akan melakukan sesuatu dengan drive yang berisi instalasi Windows, saya peduli tentang:

  • C:\Users\myUser\AppData. Menyimpan semua file konfigurasi untuk program saya, mungkin berguna. Berikut adalah semua hal seperti kunci, bookmark, akun email POP3, konfigurasi browser dan bookmark, dll. Ini adalah folder tersembunyi sehingga Anda harus mengaktifkannya agar muncul di pengaturan Windows Explorer (Pengaturan folder).
  • C:\Users\myUser\Documents
  • C:\Users\myUser\Pictures
  • C:\Users\myUser\Downloads
  • C:\Users\myUser\ periksa di sini untuk folder tersembunyi yang dimulai dengan titik ., seperti .android, .cache, .ssh dan seterusnya. Mungkin berhubungan dengan pengaturan program yang Anda butuhkan nanti.
  • C:\ProgramData. Lebih banyak file konfigurasi program yang bisa berguna.
  • C:\Windows\Fonts. Jika Anda peduli tentang beberapa font tambahan yang Anda instal atau semacamnya, Anda mungkin perlu mendapatkan file ttf Anda.

Untuk mendapatkan record program yang terinstal, Anda mungkin menggunakan script atau bahkan yang perkasa CCleaner !

Akhirnya, setelah Anda menginstal Windows 10 itu akan aktif secara otomatis, jadi tidak ada masalah dengan itu. Saya sarankan mengunduh ISO W10 dari sini dan buat USB yang dapat di-boot untuk menginstal W10 secara langsung, lihat sini Bagaimana cara melakukannya.

Yang biasanya saya lakukan adalah sebagai berikut:

  • Cadangkan folder pengguna standar dari profil Anda ke drive USB eksternal: Desktop, Dokumen, Unduhan, Favorit, Musik, Gambar, dan Video
  • Juga buat cadangan folder yang sama dari profil pengguna Publik
  • Cadangkan file PST Outlook yang dibuat secara manual
  • Cadangkan folder %SystemDrive%\Windows\System32\DriverStore. Ini akan memudahkan Anda instal driver perangkat apa pun (dalam Device Manager) yang mungkin hilang setelah instalasi bersih OS.
  • Memastikan bahwa Pengaturan sinkronisasi Windows 10 sepenuhnya dikonfigurasi untuk login Microsoft itu
  • Pastikan pengaturan sinkronisasi Chrome dan / atau Firefox dikonfigurasi dengan benar

Jika perilaku hard drive Anda membuatnya agak tidak stabil, Anda dapat dengan mudah mengunduh Windows 10 Media Creation Tool dari komputer lain:

  1. Arahkan ke Alat Pembuatan Media Windows 10 halaman web.

  2. Pilih Unduh alat sekarang

  3. Simpan file MediaCreationTool.exe ke desktop Anda dan jalankan.

  4. Akui perintah UAC, jika perlu.

  5. Pada layar yang menunjukkan Applicable notices and license terms, klik Menerima :

  6. Pilih tombol radio ke Create installation media for another PC dan klik Lanjut :

  7. Hapus centang pada kotak ke Use the recommended options for this PC:

  8. Manfaatkan menu tarik-turun untuk memilih bahasa, edisi, dan arsitektur yang sesuai untuk media instalasi Windows 10 yang ingin Anda buat dan klik Lanjut :

  9. Di jendela berikutnya Anda dapat memilih untuk membuat flash drive USB secara otomatis, atau mengunduh file ISO yang kemudian dapat Anda bakar ke DVD untuk menginstal Windows 10:

  10. Setelah Anda selesai membuat flash drive USB atau disk DVD, Anda dapat melakukan booting dari media tersebut untuk memulai proses instalasi Windows 10.

(Sumber: Cara Mengunduh File ISO Windows 10 )

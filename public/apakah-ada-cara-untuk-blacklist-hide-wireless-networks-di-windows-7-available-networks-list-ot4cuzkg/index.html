Apakah ada cara untuk Blacklist / Hide Wireless Networks di Windows 7 Available Networks List?

Ada beberapa jaringan Nirkabel yang dapat dilihat dari rumah saya yang ingin saya daftar hitam / sembunyikan dari daftar jaringan yang tersedia, berdasarkan Nama SSID.

Apakah ini mungkin, dan jika tidak, mengapa?

Untuk lingkungan non-domain, jalankan perintah berikut di prompt perintah yang ditinggikan untuk terlebih dahulu menyembunyikan semua SSID, lalu tambahkan SSID yang Anda inginkan agar dapat digunakan / terlihat:

netsh wlan add filter permission=denyall networktype=infrastructure
netsh wlan add filter permission=allow ssid=yourssidhere networktype=infrastructure

Wilbis — Sumber
  • Ini bekerja dengan sempurna! Lihat info lebih lanjut di technet.microsoft.com/en-us/library/cc755301(v=ws.10).aspx . Tercantum Windows Server 2008 di bagian atas, tetapi bekerja pada Windows 7 Ultimate juga. Anda dapat menggunakan netsh wlan delete filter permission=[denyall|allow|block] networktype=infrastructure untuk menghapus filter add sebelumnya.
    — akTed
  • Selain itu, jika Anda membalik urutan kedua perintah netsh ini, koneksi Anda tidak akan terputus setelah perintah denyall. Terakhir, jika Anda hanya ingin memasukkan SSID tertentu ke daftar hitam (bukan daftar hitam semua lalu mengizinkan SSID lain satu per satu), gunakan netsh wlan add filter permission=block ssid="SSID_TO_BLACKLIST_HERE" networktype=infrastructure.
    — akTed

Hal ini dimungkinkan jika Anda ingin membuat kebijakan grup.

  1. Buka konsol kebijakan grup (Mulai> gpedit.msc> Enter)
  2. Buka: Konfigurasi Komputer> Kebijakan> Pengaturan Windows> Pengaturan Keamanan> Kebijakan Jaringan Nirkabel (IEEE 802.11)
  3. Klik "Tindakan" di menu dan kemudian klik "Buat Kebijakan Jaringan Nirkabel Baru untuk Windows Vista dan Rilis Nanti"
  4. Beri nama dan deskripsi untuk kebijakan tersebut
  5. Centang "Gunakan layanan Windows WLAN AutoConfig untuk klien" (sejauh yang saya tahu, kebijakan ini hanya akan berfungsi dengan layanan internal - jika Anda menggunakan alat pihak ketiga untuk menyambung ke titik akses, kebijakan ini tidak akan berfungsi)
  6. Buka tab "Izin Jaringan"
  7. Klik "Tambah" dan masukkan SSID yang ingin Anda blokir dan pastikan "Izin" adalah "Tolak"
  8. Klik "OK" dan lakukan perubahan lain (seperti menonaktifkan kemampuan untuk terhubung ke jaringan ad-hoc, atau menyembunyikan SSID yang diblokir dari pengguna)
Bagikan

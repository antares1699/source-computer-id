Dapatkah saya "mensimulasikan" PC jaringan virtual

Saya harus menguji program di PC saya yang mengambil jalur dari DB dan kemudian mengerjakan file di jalur ini. Masalahnya adalah bahwa jalur ini merujuk ke \\ STORAGE-AREA \ etc dan di jaringan saya tidak ada mesin dengan nama ini. Jadi pertanyaan saya adalah: dapatkah saya membuat atau mensimulasikan PC atau jaringan berbagi dengan nama tertentu yang misalnya akan mengacu pada C: \ TemporaryFiles?

OS saya adalah Windows 7.

Terima kasih.

9

Anda dapat membuat mesin lokal Anda memalsukan host itu.

Buka file %windir%\system32\drivers\etc\hosts dengan editor teks favorit Anda. Perhatikan bahwa Anda harus menjalankan program menggunakan hak istimewa yang ditinggikan (sebagai administrator).

Di akhir file tambahkan baris berikut:

127.0.0.1 STORAGE-AREA

Setelah ini selesai, buat folder palsu lokal Anda untuk ini dan bagikan sebagai etc, yang kemudian akan memungkinkan Anda untuk mengakses \\STORAGE-AREA\etc menyediakan isinya.

Perhatikan bahwa Anda juga dapat menggunakan alamat IP lain jika Anda ingin menggunakan NAS atau PC lain untuk itu.

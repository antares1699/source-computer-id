Beberapa contoh

Mengubah kata sandi

Ada dua cara untuk mengubah kata sandi di Command Prompt:

  1. Cara diam

Net User "User's name" "New password"

  1. Cara Interaktif

Net User "User's name" *

Berikut tanda bintang (*) petunjuknya pengguna untuk kata sandi memasukkan membuatnya tak terlihat di layar dan tidak dapat direkam di riwayat Command Prompt (cmd.exe).

Catatan

  • Selalu kutip ganda nama dan kata sandi untuk menghindari karakter khusus seperti spasi.
  • Nama pengguna saat ini berisi variabel lingkungan %UserName%.
  • Windows adalah case-insensitive sistem operasi maka kasus huruf tidak menjadi masalah dalam nama variabel; artinya %USERNAME% dan %username%, dan bahkan %uSErNaMe% memberikan hasil yang sama.
  • Hal di atas juga berlaku untuk perintah Windows (namanya), yang berarti net, Net, dan NET semuanya sama.

Menjadikan administrator pengguna

Untuk membuat seorang administrator pengguna Anda harus menambahkan pengguna ke dalam grup Administrator apa yang dilakukan sebagai berikut dalam baris perintah "gaya":

Net LocalGroup Administrators "User's name" /Add

Mengaktifkan pengguna Administrator sistem

Windows memiliki pengguna khusus bernama Administrator yang dinonaktifkan secara default dan yang Anda hubungi saat Anda Lari file yang dapat dieksekusi sebagai Administrator . Pengguna tersebut dapat diaktifkan untuk masuk dengan sebagai berikut (dalam gaya CLI):

Net User Administrator /Active:Yes

Ini mungkin disebut pengguna "terakhir" yang dapat Anda masuki tanpa membuat pengguna terpisah, tetapi itu tidak disarankan. Selain itu, ia mengabaikan UAC dan yang terkait, dan memiliki masalah dengan hal-hal seperti antarmuka metro baru dan tiles-nya, secara default, tanpa konfigurasi ulang.

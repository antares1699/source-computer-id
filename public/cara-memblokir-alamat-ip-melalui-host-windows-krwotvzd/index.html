Cara memblokir Alamat IP melalui Host Windows

Saya ingin memblokir akses ke alamat ip tertentu melalui host windows.

Ini adalah alamat ip 41.190.8.17. Dan ini adalah isi dari file host saya.

# localhost name resolution is handle within DNS itself.

127.0.0.1 41.190.8.17

Jika saya membuka browser dan memasukkan 41.190.8.17, saya langsung diarahkan ke situs.

Tolong bisakah seseorang memberi tahu saya mengapa mengabaikan ini?

Ini tidak akan berfungsi seperti itu, karena file-host hanya digunakan untuk resolusi nama melalui DNS. Karena ini sudah merupakan Alamat IP, Windown (atau aplikasi Anda) tidak perlu melakukan Pencarian DNS.

Saya pikir Anda harus memblokir akses ke IP-Address dalam pencarian dengan firewall windows. Mungkin ini akan membantu, tergantung pada Versi Windows yang Anda gunakan

Anda tidak dapat memblokir alamat ip melalui file host, hanya resolusi domain! Jika Anda ingin meneruskan ips ke localhost Anda harus membuat aturan proxy, tetapi jika Anda hanya ingin memblokir lalu lintas alamat ip, saya akan menggunakan ini.

Simpan ini sebagai block.bat

set ip=%1
netsh advfirewall firewall add rule name="BLOCK IP ADDRESS - %ip%" dir=in action=block remoteip=%ip%
netsh advfirewall firewall add rule name="BLOCK IP ADDRESS - %ip%" dir=out action=block remoteip=%ip%

kemudian dari baris Perintah administratif, jalankan block.bat 192.168.0.1 atau alamat IP apa pun yang ingin Anda blokir Ini tidak mengalihkan ke 127.0.0.1, hanya memblokirnya.

Bagaimana memulihkan file dari direktori 'tersembunyi' dengan Time Machine?

Bagaimana Anda memulihkan file yang (saya percaya) dicadangkan oleh Time Machine saat Time Machine menolak untuk menampilkan direktori?

Secara khusus, saya tidak bisa memulai Lotus Notes. (Ya, saya tahu; Saya tidak begitu menyukainya, tetapi diperlukan.) Saya berpikir bahwa saya harus dapat memulihkan versi kerja terakhir yang diketahui dari Time Machine, tetapi file tersebut ada di:

/Users/jleffler/Library/Application Support/Lotus Notes Data/mail6/jleffler.nsf

dan ketika saya masuk ke Time Machine, itu tidak menampilkan direktori Library sama sekali, yang membuatnya sulit untuk memulihkan file dari direktori itu. Pada tahap ini, saya bahkan tidak yakin bahwa file tersebut di-backup, tetapi saya harus menganggapnya dan saya mengabaikan sesuatu yang jelas.

Lihat cara mengaktifkan tampilan file tersembunyi di Finder, lalu masuk ke Time Machine

Di Terminal

default tulis com.apple.finder AppleShowAllFiles YES

Ini mengatur bendera untuk menampilkan file tersembunyi

killall Finder

Ini akan memaksa Finder untuk memulai ulang

Lalu masuk ke Time Machine, perbaiki / pulihkan apa yang Anda inginkan, lalu

default tulis com.apple.finder AppleShowAllFiles NO

killall Finder

Untuk kembali ke ketidaktahuan yang membahagiakan

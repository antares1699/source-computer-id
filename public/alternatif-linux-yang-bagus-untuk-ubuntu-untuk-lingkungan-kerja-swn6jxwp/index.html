Alternatif Linux yang bagus untuk Ubuntu untuk lingkungan kerja

Saya menjalankan laptop yang layak dengan ram 3GB dan 2GHz Core Duo. Saya menggunakannya terutama untuk bekerja yang membutuhkan beberapa koneksi SSH dan SFTP ke server dan menjalankan VM sebagian besar waktu. Tidak ada yang lebih intensif dari itu.

Saya suka menggunakan Ubuntu 9.10 namun saya memiliki masalah Nautilus dan menu atas / bawah membeku terus-menerus - saya berani mengatakannya, lebih banyak masalah daripada ketika saya menggunakan Windows Vista.

Saya hanya mencari rekomendasi sistem operasi lain yang akan cocok untuk tugas tersebut.

Terima kasih

Ubuntu 9.10 memiliki banyak masalah dibandingkan dengan 9.04 - lihat forum dan daftar Ubuntu. Jika Anda senang dengan Ubuntu, Anda dapat memanfaatkan 9.04 sebagai gantinya (dengan asumsi Anda telah menerapkan semua pembaruan yang tersedia) hingga 10.4 dirilis (yang akan terjadi dalam waktu sekitar satu bulan).

Saya telah menjalankan 10,04 sejak Alpha 3 di netbook saya dan itu baik-baik saja, tidak ada kemacetan atau masalah lainnya. Saya sangat yakin bahwa 10,04 akan jauh lebih dapat diandalkan daripada 9,10.

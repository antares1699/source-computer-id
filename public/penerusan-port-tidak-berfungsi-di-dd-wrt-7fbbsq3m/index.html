Saya baru saja membeli router nirkabel Netgear (3700v4) untuk mengganti yang lain (3700v1) yang gagal. Seorang teman telah membantu saya menjalankan port forwarding yang asli. Salah satu hal penting yang dilakukannya adalah mengekspos Outlook Web Access. Entah bagaimana penerusan port tidak berfungsi pada NetGear R7000 saya. Saya harus memasukkan perintah ini secara manual di bawah Administrasi> Perintah untuk meneruskan port 8080 ...

iptables -t nat -I PREROUTING -p tcp --dport 8080 -d <WAN_IP> -j DNAT --to <LAN_IP>:8080

Saya memiliki masalah yang sama dengan port forward tetapi dapat memperbaikinya dengan menetapkan port forward melalui perintah (CLI), saya menggunakan sintaks di bawah ini untuk membuatnya berfungsi

iptables -t nat -I PREROUTING -p tcp --dport 80 -j DNAT --to 192.168.1.2:80 iptables -I FORWARD -p tcp -d 192.168.1.2 --dport 80 -j ACCEPT

Ubah saja port & IP agar sesuai dengan kebutuhan Anda, reboot juga akan menghapus ini sehingga Anda perlu menyimpan startup untuk menjadikannya permanen.

Menggabungkan dan memformat ulang paragraf di Microsoft Word 2007

Setelah kesalahan salin / tempel di Microsoft Word 2007, saya berakhir dengan teks yang terlihat seperti ini:

This line   breaks up here

continues here, and    so on

here, when    it should all be

in a    single line without

all the random    whitespace.

Saya mengonfirmasi bahwa ada pemisah paragraf dan spasi ekstra di antara setiap baris - mungkin karena baris baru yang di-hardcode di sumber aslinya.

  1. Apakah ada cara (sebaiknya mudah) untuk menggabungkan paragraf di Microsoft Word?

  2. Apakah ada cara untuk memformat ulang paragraf sehingga spasi yang tidak relevan dihapus? Saya dapat mengubah gaya flush, tetapi spasi tetap.

Saya (jelas?) Tidak memiliki pengalaman dengan Word, menjadi lebih dari seorang TeX, tetapi saya telah mencari di Google dan merayapi menu selama beberapa jam dan saya belum menemukan solusi ...

Untuk menghilangkan spasi ekstra, Anda dapat melakukan pencarian / penggantian dan ketik dua spasi ke dalam kotak cari dan satu spasi ke dalam kotak ganti. Kemudian jalankan terus sampai tidak lagi menemukan apa pun. Namun, jika Anda memiliki pengelompokan yang disengaja dari dua spasi atau lebih, ini juga akan mengubahnya menjadi hanya satu spasi. Namun, biasanya dua spasi tidak diperlukan bahkan di antara kalimat, kecuali Anda menggunakan font spasi tunggal (lihat pertanyaan ini ).

Catatan samping: Saat saya menyalin dan menempelkan sesuatu ke Word (mis. Dari halaman web atau dokumen pdf), saya telah menemukan bahwa saya dapat menghindari beberapa keanehan pemformatan dengan memastikan bahwa saya hanya menyalin dan menempel teks biasa tanpa info format dari sumber. Saya biasa melakukan ini dengan terlebih dahulu menempelkannya ke dokumen Notepad kosong, lalu menyalin dari notepad ke Word. Di Word 2010, Anda dapat melewati langkah itu dengan mengklik kanan dan menggunakan opsi "simpan teks saja" di bawah Opsi Tempel. Ini mungkin tidak membantu dalam kasus Anda karena tampaknya ada karakter tambahan yang sebenarnya dalam sumber yang Anda salin.

Edit:

Saya perhatikan bahwa Anda menyebutkan kemungkinan menggunakan makro dan menulisnya. Berikut kode VBA:

Sub FixParagraph()
'
' FixParagraph Macro
'
'
    Dim selectedText As String
    Dim textLength As Integer

    selectedText = Selection.Text

    ' If no text is selected, this prevents this subroutine from typing another
    ' copy of the character following the cursor into the document
    If Len(selectedText) <= 1 Then
        Exit Sub
    End If

    ' Replace all carriage returns and line feeds in the selected text with spaces
    selectedText = Replace(selectedText, vbCr, " ")
    selectedText = Replace(selectedText, vbLf, " ")

    ' Get rid of repeated spaces
    Do
        textLength = Len(selectedText)
        selectedText = Replace(selectedText, "  ", " ")
    Loop While textLength <> Len(selectedText)

    ' Replace the selected text in the document with the modified text
    Selection.TypeText (selectedText)

End Sub

Makro ini menggantikan semua carriage return dan baris baru dalam teks yang dipilih dengan spasi, lalu menghilangkan spasi berulang. Jadi, Anda cukup menyorot teks yang ingin Anda perbaiki dan kemudian menjalankan makro. Saya hanya menguji ini dengan Word 2010 karena saya tidak memiliki Word 2007, tetapi saya yakin ini akan berfungsi pada tahun 2007 juga.

Untuk mendapatkan kode makro ke dalam dokumen Anda di Word 2007, pertama-tama ikuti petunjuk "Tampilkan tab Pengembang" di halaman ini . Kemudian ikuti petunjuk "Menulis makro dari awal" di halaman yang sama (saya menggunakan nama FixParagraph untuk makro saya di atas). Setelah Anda masuk ke editor kode, Anda cukup menyalin badan kode di atas. Ada juga petunjuk tentang cara menjalankan makro di laman web yang sama.

  1. Tekan Ctrl + H untuk membuka Temukan dan ganti
  2. Klik pada Menggantikan tab
  3. Klik pada Menemukan apa lapangan kemudian Khusus
  4. Pilih Tanda Paragraf (Anda akan melihat ^ p di bidang Temukan)
  5. Dalam Ubah dengan bidang Anda tidak perlu mengetik apa pun.

Masalahnya adalah ini akan menggantikan semua paragraf ...

Anda bisa bermain-main dengan karakter khusus itu. Semoga berhasil!

Jawaban utama:

(terinspirasi oleh Nicu Zecheru jawaban)

  1. Pilih teks Anda, yang Anda inginkan untuk menghapus Hentian paragraf
  2. Tekan Ctrl-H
  3. Temukan: ^p (tanda paragraf)
  4. Ganti dengan apa yang sesuai (mis.hanya spasi)
  5. Klik Menggantikan semua . Ini akan memberi tahu Anda berapa banyak penggantian yang dibuat dalam seleksi dan apakah Anda ingin melanjutkan di luar seleksi:
  6. Klik Tidak

Itu saja, sekarang semua tanda paragraf diganti dalam pilihan Anda.

Bonus: Jika Anda tidak tahu cara menampilkan simbol yang tidak dapat dicetak, ikuti langkah-langkah berikut:

Anda perlu membuat yang tidak bisa dicetak simbol terlihat - klik ikon ini di bilah alat:

Sekarang Anda akan melihat tanda paragraf, spasi, dll:

Hapus tanda paragraf (disorot kuning pada gambar) sebagai diperlukan untuk memperbaiki teks. Maka kamu bisa format spasi antar baris sebagai diperlukan melalui paragraf properti.

P.S. tangkapan layar berasal dari Word 2010 tetapi simbol dan fungsionalitas akan menjadi sama di versi Word mana pun.

Cara mengaktifkan penangkapan melalui amixer

Saya ingin mengaktifkan mikrofon di komputer saya melalui baris perintah. Saya menggunakan amixer untuk ini.

amixer set "Front Mic" 66%
amixer set "Digital" 100%
amixer set "Front Mic Boost" 100%
amixer set "Capture" 100%

Ini mengatur pengaturan volume. Masalahnya adalah penangkapan itu dinonaktifkan. Amixer menampilkan keluaran ini:

Kontrol mixer sederhana 'Capture', 0 Kemampuan: cvolume cswitch Tangkap saluran: Kiri Depan - Kanan Depan Batas: Tangkap 0 - 54 Kiri Depan: Tangkap 27 [50%] [-18.00dB] [mati] Kanan Depan: Tangkap 27 [50%] [-18.00dB] [mati]

Saya dapat menangkap aktif melalui kmix setelah itu amixer menunjukkan saluran seperti pada. Tetapi halaman manual tidak menunjukkan cara mengaktifkan saluran melalui amixer.

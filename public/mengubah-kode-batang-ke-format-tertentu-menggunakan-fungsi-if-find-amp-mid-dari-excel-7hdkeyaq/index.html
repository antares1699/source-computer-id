Mengubah kode batang ke format tertentu menggunakan fungsi IF, FIND & MID dari excel

Di Excel saya memiliki sejumlah kolom yang berisi karakter dari berbagai tipe seperti:

WS-S5-S-L1-C31-F-U5-S9-P14 
WS-S5-S-L1-C31-F-U5-S8-P1 
WS-S5-N-L1-C29-V-U16-S6-P6

Saya ingin mengubahnya menjadi 8 karakter menggunakan aturan berikut:

  • pertahankan hanya tiga segmen terakhir
  • hapus U dan tambahkan awalan 0 jika sesuai
  • hapus S dan tambahkan awalan 0 jika sesuai
  • hapus P dan tambahkan awalan 0 jika sesuai

Sebagai contoh:

  • WS-S5-S-L1-C31-F-U5-S9-P14 ubah menjadi 05-09-14
  • WS-S5-S-L1-C31-F-U5-S8-P1 ubah menjadi 05-08-01
  • WS-S5-N-L1-C29-V-U16-S6-P6 ubah menjadi 16-06-06

Saya yakin ada cara untuk menggunakan fungsi IF, FIND & MID untuk mengonversinya di Excel tetapi tidak tahu bagaimana memulainya. Bantuan apa pun akan sangat dihargai.

Memperbarui

Akhirnya, saya ingin mengubahnya menjadi 13 karakter jika memungkinkan misalnya:

  • WS-S5-S-L1-C31-F-U5-S9-P14 dikonversi ke S1-F-05-09-14
  • WS-S5-N-L2-C31-D-U5-S8-P1 dikonversi ke N2-D-05-08-01
  • WS-S5-N-L1-C29-V-U16-S6-P6 dikonversi ke N1-V-16-06-06

Seperti yang ditunjukkan @ygaft, itu mungkin, tetapi akan lama dengan fungsi Excel standar.

Saya menggunakan gratis RegEx Temukan / Ganti tambahan dalam situasi seperti itu, menggunakan ekspresi reguler Anda dapat melakukannya dengan lebih mudah.

Rumusnya:
=RegExReplace(RegExReplace(A1,".*U([0-9]+)-S([0-9]+)-P([0-9]+)","0$1-0$2-0$3"),"0([0-9]{2})","$1")

Bagaimana itu bekerja:

  • fungsi batin:
    • A1: dari isi sel A1
    • ".*U([0-9]+)-S([0-9]+)-P([0-9]+)" mencari pola "... U # -S # -P #" di mana "#" mewakili satu atau lebih angka dan mengingat angka (tanda kurung membuat grup referensi)
    • "0$1-0$2-0$3" menggabungkan angka-angka yang ditemukan di langkah sebelumnya, menambahkan 0 ke semuanya.
  • fungsi luar:
    • RegExReplace(...) - bekerja dengan hasil fungsi batin
    • "0([0-9]{2})" - mencari 0 diikuti oleh dua digit (= kasus di mana 0 tidak diperlukan)
    • "$1" - hanya menyimpan dua digit, menjatuhkan 0 di depan (hanya dalam kasus yang cocok di langkah sebelumnya)

Anda juga dapat melihat lebih banyak penjelasan tentang ekspresi reguler online:

  • Langkah pertama: https://regex101.com/r/4W4dWw/1
  • tahap kedua: https://regex101.com/r/mA8lGs/1

Catatan: Saya tidak berafiliasi dengan add-in itu, cukup gunakan karena membuat hidup saya lebih mudah.

Memperbarui

Anda dapat menggunakan rumus ini untuk kode 13 karakter Anda:
=RegExReplace(RegExReplace(A3,".*-([A-Z])-[A-Z]([0-9]).*-([A-Z])-U([0-9]+)-S([0-9]+)-P([0-9]+)","$1$2-$3-0$4-0$5-0$6"),"0([0-9]{2})","$1")

Cukup jelek,
tetapi Anda bisa mencapainya dengan cara berikut, ini mengasumsikan bahwa data kerja Anda ada di kolom A:

=TEXT(LEFT(RIGHT(A1,LEN(A1)-FIND("U",A1,1)),FIND("-",RIGHT(A1,LEN(A1)-FIND("U",A1,1)),1)-1),"00")&"-"&TEXT(MID(RIGHT(A1,LEN(A1)-FIND("U",A1,1)),FIND("-",RIGHT(A1,LEN(A1)-FIND("U",A1,1)),1)+2,(FIND("P",RIGHT(A1,LEN(A1)-FIND("U",A1,1)),1)-2-FIND("S",RIGHT(A1,LEN(A1)-FIND("U",A1,1)),1))),"00")&"-"&TEXT(RIGHT(RIGHT(A1,LEN(A1)-FIND("U",A1,1)),LEN(RIGHT(A1,LEN(A1)-FIND("U",A1,1)))-FIND("P",RIGHT(A1,LEN(A1)-FIND("U",A1,1)),1)),"00")

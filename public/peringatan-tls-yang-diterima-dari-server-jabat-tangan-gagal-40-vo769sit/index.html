Saya mencoba FTP ke situs web saya. Saya telah memastikan alamat IP saya tidak diblokir dan saya menggunakan porta dan kredentail yang benar, namun saya masih mendapatkan log kesalahan ini:

Saya mengalami masalah ini di ProFTP, dan mendapat pesan kesalahan yang sama di FileZilla. Untuk alasan yang sama seperti yang dinyatakan, FileZilla telah menonaktifkan RC4 pada koneksi TLS. Anda mungkin masih dapat menyambung tanpa pemeriksaan sertifikat (di bawah Manajer Situs - Protokol):

Alternatifnya, dan yang berhasil bagi saya, adalah memberi Server FTP jalur ke sertifikat SSL di server Anda. Ini khusus untuk PureFTP, jadi saya tidak bisa memberi tahu Anda file mana yang harus ditambal - maaf.

Ini mungkin membantu: http://download.pureftpd.org/pub/pure-ftpd/doc/README.TLS

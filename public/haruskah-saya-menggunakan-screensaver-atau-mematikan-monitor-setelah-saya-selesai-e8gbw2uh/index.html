Haruskah saya menggunakan screensaver atau mematikan monitor setelah saya selesai?

Setelah menggunakan screensaver selama lebih dari sepuluh tahun, saya mulai berpikir untuk mengubah cara saya menghemat energi dengan monitor saya. Saya ingin membiasakan diri mematikan monitor setelah selesai menggunakan komputer. Sekarang, saya biarkan saja, dan setelah 15 menit screensaver menyala, dan satu jam setelah itu monitor mati. Saya pikir saya akan menggunakan lebih sedikit daya secara keseluruhan jika saya mematikannya begitu saya selesai. Tentu saja, saya harus membiasakan diri, yang menurut saya bisa saya lakukan. Pertanyaan saya adalah ini:

Apakah ada yang salah dengan menyalakan / mematikan monitor beberapa kali sepanjang hari?

Screensaver tidak menghemat daya. Dalam beberapa kasus, screensaver dapat menyebabkan komputer menggunakan daya LEBIH BANYAK (screensaver 3D yang rumit).

Mematikan monitor modern tidak akan menyebabkan kerusakan yang merugikan bagi mereka.

Jika Anda ingin menghemat daya, yang terbaik adalah mematikannya sepenuhnya saat Anda tidak menggunakannya.

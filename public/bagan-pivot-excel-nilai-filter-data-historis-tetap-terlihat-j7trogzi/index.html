Bagan Pivot Excel - nilai Filter Data historis tetap terlihat?

Saya menggunakan Excel dengan koneksi data SQL untuk menghasilkan Pivot Chart.

Masalah yang saya alami melibatkan filter data di bagian atas setiap kolom, memungkinkan Anda untuk memfilter nilai yang terlihat. Setelah membuat lembar kerja, nilai yang dapat dipilih di filter data sudah benar. Namun, jika item dihapus dari database dan data koneksi data Excel direfresh, nilai yang telah dihapus masih terlihat sebagai filter data. (Jelas memilihnya menghasilkan tabel kosong karena tidak ada data aktual yang tersisa.)

Bagaimana cara memperbaiki perilaku ini, sehingga penyegaran data juga memperbarui daftar kemungkinan nilai filter data?

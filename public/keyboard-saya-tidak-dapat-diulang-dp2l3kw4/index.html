Ini baru saja terjadi pada saya dan itu karena kucing saya berbaring di atas keyboard.

Jika Anda menahan tombol shift kanan selama 8 detik di Windows, Tombol Filter menyala yang membatasi jumlah penekanan tombol sehingga Anda dapat menekan satu tombol hanya sekali dalam periode 1-2 detik. Ini membatasi kemampuan Anda untuk mengetik kata apa pun dengan huruf ganda di dalamnya, membatasi pengguliran Anda dengan tombol panah di situs web, membatasi penggunaan tombol hapus, dll.

Di Windows 7, opsi untuk mematikan ini ada di bawah pengaturan "Kemudahan Pusat Akses" dan kemudian -> "Jadikan keyboard lebih mudah digunakan", lalu hapus centang pada kotak untuk "Aktifkan Tombol Filter" di panel kontrol. Itu tidak terdaftar di bawah pengaturan Keyboard atau Mouse.

Saya harap ini membantu orang lain.

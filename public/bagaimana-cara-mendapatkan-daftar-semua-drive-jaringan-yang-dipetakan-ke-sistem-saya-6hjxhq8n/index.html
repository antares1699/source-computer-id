Bagaimana cara mendapatkan daftar semua drive jaringan yang dipetakan ke sistem saya?

Saya memetakan drive jaringan di bawah akun sistem lokal, maksud saya, saya masuk ke mesin saya menggunakan akun sistem lokal. Sekarang saya keluar dan masuk ke mesin lagi menggunakan beberapa akun pengguna valid lainnya, tetapi saya tidak dapat menemukan drive yang dipetakan itu di penjelajah komputer saya, meskipun saya dapat menggunakan drive yang dipetakan itu di aplikasi saya. Apakah ini normal?

Apakah ada cara untuk melihat semua drive jaringan yang dipetakan di sistem saya?

Saya menggunakan mesin Windows XP.

Ya, itu normal. Koneksi jaringan dan huruf drive yang ditetapkan untuk mereka adalah sesi-lokal. Biasanya mereka menghilang saat logout (saat sesi itu sendiri dihancurkan) dan dibuat ulang oleh Winlogon saat Anda login lagi.

Dengan Local System, itu membingungkan.

Masuk sebagai Local System lagi ( psexec -desi cmd semoga bermanfaat), lalu jalankan net use untuk mendaftar semua koneksi atau net use * /delete untuk memutuskannya.

Saya mengambil drive yang dipetakan dari mesin jarak jauh menggunakan PowerShell sebagai bagian dari skrip untuk memindahkan pengaturan mereka ke mesin baru seseorang: (setel $ OldComputer ke nama atau IP komputer lama) Ini melewati daftar semua drive di sistem jarak jauh, dan jika mereka bukan drive lokal, itu akan mencoba untuk menghapus pemetaan drive itu pada komputer saat ini (jika digunakan oleh sesuatu yang lain) kemudian memetakannya kembali. Komentari baris kedua hingga terakhir (baris di atas untuk setiap, dan tambahkan baris Anda sendiri ke log atau msgbox jika Anda mau.)

$netObj = New-Object -com wscript.Network
$WMI1 = [WMISearcher] "Select Name, Providername from win32_mappedLogicalDisk"
$WMI1.Scope.path="\\" + $oldComputer + "\root\cimv2"
$WMI1.Options.ReturnImmediately
$ColDrives = $WMI1.Get()

foreach ($objDrive in $colDrives) {
      $TestLocal = get-wmiObject Win32_LogicalDisk | ? {$_.DeviceID -eq $objDrive.Name} | % {$_.Providername}
      If ($TestLocal) {
        Net Use /Delete $objDrive.Name >> $LogFile
      }#If
      #Map the drive
      Net use $objDrive.Name $objDrive.ProviderName  >> $LogFile
}#foreach

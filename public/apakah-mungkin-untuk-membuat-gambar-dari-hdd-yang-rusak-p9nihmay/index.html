Apakah mungkin untuk membuat gambar dari HDD yang rusak?

Hard drive saya rusak tetapi saya memulihkannya sedikit demi sedikit. Dibutuhkan waktu lama karena ini adalah HDD portabel USB2.

Tapi saya membutuhkan hard drive portabel saya untuk uni / college. Apakah mungkin untuk menyalin seluruh isi hard drive (setiap bit / byte) ke sebuah image sehingga saya dapat memasangnya dan melakukan pemulihan data dari itu? Jika ya, apa yang harus saya gunakan?

Tergantung.

Jika masalahnya adalah ada masalah perangkat keras dengan membaca byte, maka tidak ada yang benar-benar dapat Anda lakukan (selain membawanya ke suatu tempat).

Jika masalahnya ada pada MFT atau MBR, maka ya, Anda bisa mengkloning drive dan itu akan memungkinkan Anda untuk mengerjakannya nanti.

Jika masalahnya adalah data sebenarnya rusak, tidak ada yang dapat Anda lakukan (umumnya) untuk mendapatkan kembali data tersebut.

Jika Anda menggunakan mesin windows, Anda dapat menggunakan Macrium atau jika Anda menggunakan mesin linux, Anda dapat menggunakan Ping .

Partimage adalah utilitas kloning disk untuk lingkungan Linux / UNIX. Partimage dapat menyimpan partisi dalam banyak format ke image disk.

Cadangkan partisi disk individu. cadangan volume sangat berguna untuk pemulihan jika terjadi kegagalan disk atau kerusakan data.

Parted Magic adalah ISO yang dapat di-boot yang berisi utilitas ini dan lainnya yang dapat membantu Anda memulihkan data Anda.

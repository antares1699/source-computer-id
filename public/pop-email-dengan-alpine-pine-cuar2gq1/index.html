POP EMail dengan Alpine (Pine)

Selama bertahun-tahun saya sering menggunakan Pine, terutama saat kuliah

Bagaimana cara menyiapkan Pine untuk POP ke email saya sehingga lokal di komputer saya? Saya tidak suka menyimpan email saya di server di suatu tempat, saya ingin menyimpannya secara lokal sehingga saya dapat mencadangkannya, dll

Ada yang punya saran?

Dari Konfigurasi Alpine :

Mulailah dengan membuka "Setup Konfigurasi "layar (M S C) dan temukan bagian Preferensi Folder. Dibawah bagian ini, set aktifkan-folder-masuk dan mulai ulang alpine agar perubahan diterapkan. Sekarang buka Pesan Masuk Anda Folder (F Folder Masuk), tapi jangan buka folder INBOX Anda.

Sekarang tiba bagian menyenangkan.

Karena Anda ingin menambahkan folder baru, tipe A. Anda akan disajikan dengan file baris perintah yang menanyakan "Nama server berisi folder tambahan "dan empat opsi di bawah itu. Kami tertarik dengan opsi. Sejak Anda akan menarik email dari server pop3 jarak jauh, Anda mau mengkonfigurasi mail drop, yaitu file opsi terakhir, jadi tekan Ctrl W. Anda akan sekarang ditanyakan nama suratnya drop server (server yang Anda tarik surat dari). Masuk ke server (dan nomor port jika diinginkan), file nama pengguna (yang mungkin termasuk atau tidak termasuk bagian dari alamat server), dan pop3. Setelah selesai, Anda akan mendapatkan sesuatu yang terlihat seperti berikut ini:

pop.server.tld:port/user=username@server.tld/pop3

Jika Anda menggunakan koneksi aman, Anda perlu menambahkan dua hal ke file masuk. Pertama, tambahkan ssl ke bagian akhir. Ketika itu selesai (dan dengan asumsi Anda tidak mau, atau mengalami kesulitan pengaturan, validasi sertifikat), tambahkan novalidate-cert tepat setelah mail drop server. Setelah Anda selesai, Anda harus memiliki sesuatu itu menyerupai berikut ini:

pop.server.tld:port/novalidate-cert/user=username@server.tld/pop3/ssl

Sekarang tekan Enter. Anda sekarang akan diminta folder yang ingin Anda salin suratnya dari; jawaban: Inbox.

Berikutnya: "Nama server yang akan dimuat folder tujuan ". Biarkan kosong, dan lanjutkan.

Sekarang untuk "Folder untuk menyalin email ke". Anda dapat menamai ini apa pun yang Anda inginkan, tapi saya biasanya menggunakan nama pengguna akun email apa pun yang saya atur mail drop up untuk.

Dan saya melakukan hal yang sama seperti di atas untuk "Nama panggilan untuk folder" nama pengguna "".

Dan sekarang Anda memiliki mail drop baru map. Untuk menyiapkan lebih banyak, ulangi sebagai dibutuhkan.

Pengaturan email Alpine juga memiliki petunjuk penyiapan Alpine, termasuk cara menyiapkan Gmail menggunakan POP3 (termasuk gambar):

Ini adalah contoh penggunaan POP3 dari Gmail. Anda harus mengaktifkan POP3 dari akun Gmail Anda di bawah Pengaturan → Penerusan dan POP / IMAP.

Untuk membuat folder masuk baru untuk POP, gunakan Menu Utama → Folder Daftar → Folder-Masuk → Tambah.

Ini akan meminta Anda untuk Nama server berisi folder tambahan:

Name of server to contain added folder: pop.gmail.com:995/novalidate-cert/user=username@gmail.com/pop3/ssl

Ini kemudian akan meminta untuk meminta file 'Folder di' pop.gmail.com… "untuk menambahkan:

Folder on "pop.gmail.com.." to add: INBOX

Isi nama panggilan:

Nickname for folder "INBOX" : INBOX at gmail.com

Ini kemudian akan terhubung ke server gmail dan akan meminta file namapengguna@gmail.com sandi. Masukkan itu, dan INBOX akan terbuka.

Bagikan

  1. Buka file .pinerc untuk mengedit.
  2. Cari opsi "incoming-folders=".

  3. Menambahkan:

    nick_name "#move {server}inbox_folder local_mail_folder"
  4. Simpan versi terbaru dari .pinerc

  5. Buat folder email kosong di lokasi pilihan Anda:

    touch local_mail_folder

Contoh:

Menambahkan:

incoming-folders=GMAIL "#move pop.gmail.com/ssl/user=user@gmail.com/novalidate-cert}INBOX /home/user/mail/gmail"

Dan selanjutnya:

$ touch /home/user/mail/gmail

Jika Anda ingin menentukan ini untuk dua atau lebih akun, pisahkan dengan ",".

Bagikan

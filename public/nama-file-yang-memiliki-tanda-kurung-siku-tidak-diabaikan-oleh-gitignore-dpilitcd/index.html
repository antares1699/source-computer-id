Editor1Saat IniBuka [1] .txt

Karena regex tidak efektif dalam file .gitignore, bagaimana bisa a glob mencakup nama file seperti di atas?

4

6

.gitignore menggunakan ekspresi glob di setiap baris, jadi ketika Anda memasukkan Editor1CurrentlyOpen[1].txt, ia akan mencoba mencocokkan Editor1CurrentlyOpen1.txt.

Tambahkan garis miring terbalik untuk menutupi tanda kurung di entri:

Editor1CurrentlyOpen\[1\].txt

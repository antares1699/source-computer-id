Bisakah VLAN diidentifikasi dengan string daripada angka?

Router wifi saya memungkinkan untuk mengkonfigurasi VLAN yang diberi nama berdasarkan string. Saya mencoba menghubungkannya ke Ubiquiti EdgeRouter saya tetapi tidak dapat membuatnya berfungsi. Lihat di bawah pengaturan apa yang dapat saya ubah:

Tag VLAN dalam paket Ethernet adalah interger 12 bit, memberikan 4096 nilai bernomor. Tidak ada identifikasi string dalam paket.

Karena pengelolaan VLAN hanya melalui angka merupakan beban pada memori manusia dan dengan demikian rawan kesalahan, vendor segera mulai mengizinkan pengaitan nama deskriptif dengan tag numerik untuk meringankan beban itu - tetapi nama-nama ini bersifat lokal baik untuk setiap perangkat atau di kasus terbaik untuk keluarga perangkat yang berbagi database konfigurasi. AFAIK tidak ada cara lintas vendor untuk menyinkronkan tupel nomor-nama.

Agar ini berfungsi dengan bersih, Anda perlu menyiapkan identik pemetaan nama-nomor pada kedua perangkat secara manual, maka semuanya akan berfungsi seperti yang diharapkan.

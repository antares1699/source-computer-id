Skype menunjukkan bahwa saya sedang online ketika saya tidak menjalankan instance Skype apa pun

Baru-baru ini Skype menunjukkan saya sebagai online ke kontak saya bahkan ketika

  • Saya tidak memiliki contoh Skype yang berjalan di desktop saya
  • Saya keluar dari Skype di ponsel saya

Tampaknya, ini adalah fitur, bukan bug . Meskipun banyak orang-orang mengeluh , sepertinya di dunia Skype "program tutup"! = "Saya tidak ada". Perilaku yang disarankan adalah offline secara manual di setiap instance Skype.

Ini tidak dapat diterima, karena banyak kontak saya mengeluh bahwa saya bersikap kasar ketika saya tidak menerima panggilan atau menjawab IM mereka (itu karena saya bahkan tidak menjalankan Skype!).

Apakah ada solusi untuk ini yang tidak melibatkan peralihan ke perangkat lunak yang berbeda sama sekali?

Apakah saya melakukan sesuatu yang salah?

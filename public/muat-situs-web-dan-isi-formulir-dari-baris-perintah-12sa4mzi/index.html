Muat situs web dan isi formulir dari baris perintah

Menggunakan baris perintah Linux (Bash shell) Saya ingin memuat situs web tertentu di browser saya (biasanya Firefox, tetapi yang lain akan baik-baik saja selama berjalan di Linux) dan mengisi formulir yang telah ditentukan dengan beberapa data. Sebenarnya, ini harus dijalankan dari Makefile. Sebagian besar data formulir bersifat statis dan akan disimpan sebagai variabel di Makefile, hanya beberapa bidang yang harus diisi secara manual sebelum mengirimkan formulir secara manual.

Saya tahu cara memuat file situs web yang dimaksud dari baris perintah menggunakan:

firefox <URL>

Tetapi sepertinya tidak ada kemungkinan untuk mengisi formulir secara otomatis dengan variabel yang diberikan pada baris perintah. Apakah ada plugin, dapat dijalankan, atau JavaScript yang memungkinkan saya melakukan ini? Setiap saran dan petunjuk dipersilakan. Saya tidak keberatan mengkodekan beberapa JavaScript.

Saya menyelesaikan ini sekarang menggunakan skrip Greasemonkey yang menyalin parameter URL ke formulir. Makefile memanggil firefox dengan URL termasuk parameternya.

Ini bekerja untuk kebutuhan saya. Beberapa perbaikan akan diperlukan untuk masukan yang lebih kompleks, yaitu beberapa pengkodean URL.

Skrip Pengguna:

// ==UserScript==
// @name           Fill CTAN upload form
// @namespace      http://www.scharrer-online.de/namespace/
// @description    Fills the CTAN upload form with the data provided as URL variables
// @include        http://*.ctan.org/upload.html*
// @require        http://ajax.googleapis.com/ajax/libs/jquery/1.3.2/jquery.min.js
// ==/UserScript==

//var $ = unsafeWindow.jQuery;

// From http://papermashup.com/read-url-get-variables-withjavascript/
// Added: decodeURI
function getUrlVars() {
    var vars = {};
    var parts = window.location.href.replace(/[?&]+([^=&]+)=([^&]*)/gi, function(m,key,value) {
        vars[key] = decodeURI(value);
    });
    return vars;
}

var vars = getUrlVars();

for (name in vars) {
    $("input[name=" + name + "]").val(vars[name]);
    $("select[name=" + name + "]").find("option:selected").removeAttr('selected');
    $("select[name=" + name + "]").find("option[value=" + vars[name] + "]")
            .attr('selected','selected');
}

if ('DoNotAnnounce' in vars) {
    var val = vars['DoNotAnnounce'].toLowerCase();
    if (val == '1' || val == 'yes' || val == 'true') {
        $("input[name=DoNotAnnounce]").attr('checked','checked');
    }
}

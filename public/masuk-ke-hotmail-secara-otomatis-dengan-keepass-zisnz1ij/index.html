Masuk ke Hotmail secara otomatis dengan KeePass?

Apakah mungkin untuk masuk ke Hotmail atau Google Mail secara otomatis menggunakan pengelola kata sandi seperti KeePass? Saya mencoba menggunakan fitur jenis otomatis dari program ini dengan menekan Ctrl + Alt + SEBUAH kombinasi tombol. Untuk beberapa alasan itu tidak berfungsi pada dua layanan web ini, meskipun berfungsi di situs web lain.

  • Setelah sedikit meneliti masalah ini, jelas bagi saya bahwa layanan ini entah bagaimana mencegah penekanan tombol yang disimulasikan yang digunakan KeePass untuk mengetik secara otomatis. Agar ini berfungsi, saya memerlukan plugin untuk browser saya yang bekerja dengan KeePass, seperti KeeForm yang disebutkan di bawah dan KeeFox. Masalah umum lainnya meskipun Anda telah menginstal plug-in ini adalah panjang kata sandi maksimum untuk Hotmail diatur ke 16 karakter. Apa pun yang lebih lama dari itu akan mengakibatkan gagal masuk, meskipun Anda dapat mengetik lebih dari 16 jika Anda melakukannya secara manual di halaman masuk.

KeeForm adalah ekstensi populer untuk Keepass yang dapat meluncurkan situs web, memindai bidang nama pengguna dan kata sandi, dan mengisi informasi itu untuk Anda secara otomatis. Ini mengotomatiskan proses login untuk Anda dan sangat dapat dikonfigurasi.

Saat Anda mengklik dua kali bidang URL di KeePass, biasanya langkah-langkah berikut dilakukan secara otomatis untuk Anda:

  • Mulai browser
  • Buka halaman web login
  • Isi kolom Username dan Password
  • Klik tombol kirim dan login ke situs

KeeForm adalah aplikasi Windows yang didistribusikan di bawah lisensi GPL. Ini telah dikembangkan menggunakan AutoIt dan harus berjalan di semua (dan hanya) sistem operasi Windows.

Unduh KeeForm dari sini dan ikuti ini Cara menginstal dan menggunakan KeeForm .

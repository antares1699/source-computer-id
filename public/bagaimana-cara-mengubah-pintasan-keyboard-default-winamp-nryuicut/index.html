Bagaimana cara mengubah Pintasan Keyboard default Winamp?

Saya ingin memutar musik / menekan jeda ruang (seperti yang terjadi di VLC).

Bagaimana saya bisa mengubah ini?

Edit: Saya mengetahui opsi tombol pintas global, tetapi saya tidak ingin menjadi global, oleh karena itu, saya tidak ingin menjeda / membatalkan jeda setiap kali saya menekan spasi saat menulis pesan ...: D

Tombol pintas Winamp Global dirancang untuk menjadi global, agar dapat terus bekerja bahkan saat Winamp diminimalkan ke latar belakang. Ini memang disengaja dan tidak dapat diubah - yang bisa saya tawarkan hanyalah solusi.

Inilah solusi saya:

  1. Nonaktifkan di Preferensi semua hotkey Global yang tidak Anda perlukan, hanya meninggalkan yang ingin Anda gunakan.

  2. Tetapkan hotkey Global ini ke kombinasi tombol yang tidak biasa yang tidak akan pernah digunakan untuk hal lain.
    Misalnya menugaskan Alt + Bergeser + Kontrol + P. untuk bermain / jeda.

  3. Menggunakan AutoHotkey untuk membuat skrip yang, hanya jika Winamp memiliki fokus, memetakan tombol pintas yang benar-benar ingin Anda gunakan ke pintasan yang ditentukan ini.
    Misalnya peta Ruang ke pintasan putar / jeda Alt + Bergeser + Kontrol + P. .

Bagikan

Saya berhasil membuat bilah spasi bertindak sebagai tombol "jeda" dari WinAmp dengan memodifikasi tabel akselerator di executable dengan Resource Hacker ( http://www.angusj.com/resourcehacker/ ). Misalnya, saya mengedit akselerator dengan ID 40046 (yaitu VK_C, sesuai dengan pintasan saat ini untuk tombol Jeda) di tiga tabel akselerator yang memuatnya (1371, 1372, 1373), dengan menggantinya dengan VK_SPACE (maka Anda harus klik Kompilasi skrip tombol di atas).

Di salah satu tabel akselerator (1372) sudah ada akselerator VK_SPACE. Saya baru saja menghapusnya (karena membiarkannya mencegah bilah spasi bertindak sebagai Jeda / Tidak Jeda jika jendela daftar putar difokuskan). Tidak yakin apa yang dilakukan space tersebut sebelumnya dalam playlist.

Semua pintasan keyboard default lainnya dari Winamp dapat diubah dengan cara yang sama.

Perhatikan bahwa Anda mungkin tidak dapat mengedit Winamp.exe di tempat lain di folder instalasinya (kecuali jika menjalankan Resource Hacker dengan hak istimewa yang lebih tinggi), jadi yang paling mudah adalah membuat salinan di tempat lain dan menyalinnya kembali. Pastikan untuk menyimpan salinan asli yang dapat dieksekusi, jika terjadi kesalahan.

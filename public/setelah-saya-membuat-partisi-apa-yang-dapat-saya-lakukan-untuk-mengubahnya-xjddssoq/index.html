Setelah saya membuat partisi, apa yang dapat saya lakukan untuk mengubahnya?

Saat menginstal ulang Windows XP, saya membuat partisi 3gb dan menginstalnya. Sayangnya, karena sedikit tidak mengerti, saya tidak membuat yang lain dan sekarang drive 250GB saya hanya memiliki kapasitas 3GB yang dapat digunakan.

Apakah tetap ada saya yang dapat mengubah ini, setelah saya menginstal Windows dan menggunakannya sekarang?

Tentu, buka Manajemen Disk (klik kanan 'Komputer saya' -> klik 'Kelola' -> di bawah 'Penyimpanan' -> Manajemen Disk, atau yang lebih sederhana: start -> run -> diskmgmt.msc) di mana Anda dapat membuat yang lain ( atau lebih) partisi di ruang kosong.

Jika Anda ingin membuat satu partisi yang sudah Anda miliki lebih besar, ini tidak dapat dilakukan dengan alat manajemen disk standar Windows XP, tetapi ada perangkat lunak pihak ketiga yang dapat melakukannya.
Server Direktur Disk Acronis adalah salah satu yang saya tahu.

Memperbarui: Ada juga alat baris perintah Diskpart yang dikirimkan dengan XP. Program ini memiliki beberapa batasan.

  • Itu hanya dapat 'memperluas' partisi ke ruang yang tidak terisi ke kiri atau ke kanan (saya pikir ini mencakup situasi Anda).
  • Volume sistem atau boot mungkin diblokir agar tidak diperpanjang.
  • Anda tidak dapat memperluas partisi jika file halaman sistem terletak di partisi.

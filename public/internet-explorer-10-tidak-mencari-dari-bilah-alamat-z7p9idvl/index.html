Internet Explorer 10 Tidak Mencari Dari Bilah Alamat

Saya sedang menyiapkan PC Windows 8 baru milik ayah saya. Di beberapa titik setelah menggunakan utilitas Transfer Mudah untuk mentransfer file-nya dari XP dan menginstal Norton Internet Security - Saya tidak tahu apakah itu terkait, mereka hanya variabel yang saya ketahui, IE10 tidak lagi mencari dari bilah alamat - itu memperlakukan semuanya seperti alamat yang sebenarnya, jadi mengetikkan "anak kucing lucu" akan membuatnya benar-benar mencoba masuk ke http titik tebas garis miring lucu% 20kittens (SU tidak suka saya benar-benar memasukkan tautan ke situs yang tidak ada). Itu bahkan tidak menampilkan daftar penyedia pencarian saat Anda mulai mengetik.

Satu-satunya informasi yang dapat saya temukan di internet tampaknya menunjukkan bahwa ini dikendalikan oleh kebijakan grup dan harus diperbaiki di registri di bawah hkcu / software / Policies / Microsoft / internet explorer, tetapi saya tidak memiliki kunci itu di registri, dan tidak ada kebijakan kelompok yang benar-benar terlibat.

Saya tidak melihat "pencarian dari bilah alamat" sederhana atau pengaturan serupa di pengaturan IE10, tapi mungkin saya hanya melewatkan sesuatu yang sederhana seperti itu.

Saya telah menginstal Norton Internet Security, dan add-on browser yang menyertainya, tetapi saya menonaktifkannya dan pencarian masih tidak berfungsi setelah memulai ulang IE10. Saya juga menginstal ClassicShell, dan menonaktifkan semua add-on IVOSoft yang terdaftar, masih belum berhasil.

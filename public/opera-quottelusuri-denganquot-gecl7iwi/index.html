Di Opera Anda dapat memilih beberapa teks dan klik kanan dan pilih "Cari dengan" untuk mencari teks dengan mesin pencari. Apakah mungkin untuk mengkonfigurasinya untuk membuka hasil pencarian di tab baru?

Ya, Anda dapat mengubah entri di file teks menu. Namun, Anda mungkin ingin mencoba menahan tombol shift. Ini sebenarnya berfungsi untuk semua perintah menu dan akan membukanya di jendela baru, seperti jika Anda menggeser + mengklik tautan di halaman web.

Tautan berikut mungkin berguna untuk Anda:

  • http://www.opera.com/support/usingopera/operaini/
  • http://www.rarst.net/software/opera-context-menu-edit/
  • http://operawiki.info/editinginifiles

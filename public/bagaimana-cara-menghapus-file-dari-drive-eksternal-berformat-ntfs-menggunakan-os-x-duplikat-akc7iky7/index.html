Bagaimana cara menghapus file dari drive eksternal berformat NTFS menggunakan OS X? [duplikat]

Kemungkinan Duplikat:
Sebuah utilitas gratis untuk menulis NTFS di Mac OS X 10.6.5?

Saya memiliki hard drive eksternal 1TB yang menurut saya diformat untuk Windows (NTFS?) Tetapi saya memiliki Mac.

Saat ini saya dapat melihat data di sana tetapi saya tidak dapat menghapusnya.

Adakah yang dapat saya lakukan untuk dapat mengubah format agar dapat menghapus barang tanpa menghapus data apa pun dan memformat ulang hard drive eksternal secara total?

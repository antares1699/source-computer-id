Komputer

Bagaimana cara berbagi satu komputer di antara pengguna yang berbeda?

Saya tahu bahwa setiap sistem operasi yang layak digunakan mendukung banyak pengguna independen; ini memudahkan beberapa orang untuk berbagi satu komputer, dengan dokumen mereka sendiri dan segalanya. Namun, ada satu masalah kecil: pembagian tidak dapat terjadi secara bersamaan.

Remote Desktop atau perangkat lunak serupa lainnya memungkinkan orang untuk terhubung ke komputer dari tempat lain dan masuk sebagai pengguna mana pun, tanpa mengganggu pengguna lain. Namun saya bertanya-tanya, bukankah mungkin bagi banyak orang dengan akun pengguna yang berbeda untuk menggunakan komputer pada saat yang sama, secara lokal, melalui beberapa monitor dan perangkat input? Saya mencoba mencolokkan keyboard, mouse, dan monitor kedua, namun, Windows akan mencerminkan atau memperluas tampilan ke monitor kedua dan perangkat input kedua menjadi sangat mirip ... yah ... Saya tidak dapat memikirkan apa pun cukup tidak berguna untuk membandingkannya.

Bagaimana cara berbagi satu komputer dengan orang yang berbeda? Asumsikan saya memiliki keyboard, mouse, dan monitor untuk semua orang, terhubung ke beberapa kartu video (baik beberapa output pada satu kartu video atau satu monitor / kartu).

Jawabannya tergantung pada apakah Anda menggunakan Windows atau Linux:

  1. Jika itu adalah Windows komputer, nComputing produk akan melakukan apa yang Anda inginkan. Mereka menyediakan perangkat keyboard / mouse / video, yang harganya sekitar $ 75-200 per pengguna tambahan tergantung pada apakah Anda ingin menambahkan pengguna melalui PCI (termurah), USB, atau Ethernet. (Catatan lisensi Windows: Produk ini akan fungsi dengan atau tanpa lisensi Windows tambahan, tetapi Anda harus menyadari bahwa Microsoft mungkin tidak menganggap penginstalan legal kecuali Anda membeli lisensi Windows Server untuk host dan RDS CAL untuk setiap pengguna. Hubungi nComputing untuk detailnya.)

  2. Jika itu adalah Linux komputer, Berguna akan melakukan apa yang Anda inginkan. Mereka hanya menyediakan perangkat lunak, dan Anda menghubungkan keyboard / mouse / display menggunakan perangkat keras komoditas yang kompatibel dengan Linux.

Saya secara pribadi tidak akrab dengan Userful, tetapi saya secara pribadi akrab dengan nComputing. Saya terlibat dalam penerapan uji sebagai berikut:

  • Sebelum: 14 workstation Pentium 4, masing-masing dengan drive IDE dan antara 512MB-1GB DDR, menjalankan Windows XP

  • Setelah: 3 workstation Core 2 Duo, masing-masing dengan drive SATA dan 3GB DDR2, masing-masing workstation dibagi antara 4-6 pengguna melalui perangkat keras nComputing X-series, masih menjalankan Windows XP

Perbedaan kinerja sangat fenomenal, dan semua pengguna akhir sangat senang dengan workstation bersama. Jenis solusi ini tentu saja tidak akan berfungsi dengan baik untuk aplikasi komputasi berkinerja tinggi seperti CAD, GIS, atau pengeditan video, tetapi komputasi bersama masuk akal untuk semua jenis pengguna aplikasi kantor, pusat panggilan, warnet, dll.

Ini pasti berfungsi di linux!

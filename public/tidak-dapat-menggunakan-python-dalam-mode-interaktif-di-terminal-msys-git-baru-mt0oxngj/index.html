Tidak dapat menggunakan python dalam mode interaktif di terminal msys-git baru?

Saya baru-baru ini memutakhirkan laptop saya, yang menjalankan Win7 32-bit, dan laptop baru saya menjalankan instalasi Win7 64-bit.

Saya menginstal git 2.5.1 dari git-scm.com, dan versi python terbaru (3.4.3 dan 2.7.10).

Selama instalasi, saya memilih untuk menggunakan terminal baru (default) yang sebelumnya tidak disertakan dengan instalasi, dan menjalankan terminal setelah instalasi selesai. Namun, ketika saya mengetik python, saya tidak melihat output apa pun (kursor berpindah ke baris berikutnya saat saya menekan enter).

Saya telah mencoba memasukkan perintah python seperti print('hello world'), dan satu-satunya keluaran yang bisa saya dapatkan adalah kesalahan Sintaks jika saya mengetik sesuatu seperti a.4. Tampaknya python sedang berjalan, tetapi saya tidak mendapatkan keluaran. Ini terjadi untuk salah satu versi python yang saya jalankan.

Python tampaknya berjalan normal dengan alternatif git berbasis cmd Windows, tetapi pembungkus konsol normal saya, Console2 tampaknya tidak berfungsi dengan benar, jadi saya tidak dapat menyalin / menempel dengannya dengan sangat mudah.

Ada ide mengapa konsol msys tidak berfungsi, atau bagaimana saya bisa memperbaikinya?

Dari wizard penginstalan:

Jika Anda ingin menggunakan terminal MinTTY yang disertakan dengan MSys2 / Git, Anda harus menjalankan program konsol seperti Python menggunakan winpty.

Pada Git untuk Windows 2.7.1, Winpty disertakan di luar kotak, dan dapat dijalankan seperti ini:

winpty /path/to/python.exe

winpty dapat ditemukan dipasang di Git\usr\bin

Alternatifnya, Anda selalu dapat menggunakan bash aliasing untuk menulis fungsi di .bashrc Anda yang mungkin melakukan apa yang Anda inginkan. Inilah solusi saya untuk mengatasi batasan baru ini:

function maybe_python34() {
    if [ $# -eq 0 ]; then
        /c/Python34/python.exe -i
    else
       /c/Python34/python.exe $@
    fi
}

alias python=maybe_python34

Perhatikan bahwa ada beberapa masalah terkait penggunaan tombol panah untuk mengambil riwayat perintah dalam mode interaktif python.

Git menggunakan Msys, dan sekarang ada yang lebih baik, Msys2 !

Menggunakannya dan modifikasi yang telah dilakukan Git-SCM pada Msys terkait .profile, .bashrc sepertinya cara yang tepat bagi saya.

Sekarang Anda dapat dengan mudah memutakhirkan Msys2 dengan pacman

pacman -Syuu
pacman -S winpty

Git menambahkan alias yang bagus untuk winpty:

case "$TERM" in
xterm*)
    # The following *.exe programs are known to require a Win32 Console
    # for interactive usage, therefore let's launch them through winpty
    # when run inside `mintty`.
    for name in node python ipython php php5 psql
    do
        case "$(type -p "$name".exe 2>/dev/null)" in
        ''|/usr/bin/*) continue;;
        esac
        alias $name="winpty $name.exe"
    done
    ;;
esac

Untuk mendapatkan tampilan cabang Git di Prompt salin file yang orang-orang Git masukkan ke dalam prompt mereka dan sumber di .bashrc Anda (.git-prompt.sh)

if test -f /etc/profile.d/git-sdk.sh
then
    TITLEPREFIX=SDK-${MSYSTEM#MINGW}
else
    TITLEPREFIX=$MSYSTEM
fi

PS1='\[\033]0;$TITLEPREFIX:${PWD//[^[:ascii:]]/?}\007\]' # set window title
PS1="$PS1"'\n'                 # new line
PS1="$PS1"'\[\033[32m\]'       # change to green
PS1="$PS1"'\u@\h '             # user@host<space>
PS1="$PS1"'\[\033[35m\]'       # change to purple
PS1="$PS1"'$MSYSTEM '          # show MSYSTEM
# PS1="$PS1"'\[\033[33m\]'     # change to brownish yellow
PS1="$PS1"'\[\033[34m\]'       # change to pale blue
PS1="$PS1"'\w'                 # current working directory
if test -z "$WINELOADERNOEXEC"
then
    GIT_EXEC_PATH="$(git --exec-path 2>/dev/null)"
    COMPLETION_PATH="${GIT_EXEC_PATH%/libexec/git-core}"
    COMPLETION_PATH="${COMPLETION_PATH%/lib/git-core}"
    COMPLETION_PATH="$COMPLETION_PATH/share/git/completion"
    if test -f "$COMPLETION_PATH/git-prompt.sh"
    then
        . "$COMPLETION_PATH/git-completion.bash"
        . "$COMPLETION_PATH/git-prompt.sh"
        PS1="$PS1"'\[\033[36m\]'  # change color to cyan
        PS1="$PS1"'`__git_ps1`'   # bash function
    fi
fi
PS1="$PS1"'\[\033[0m\]'        # change color
PS1="$PS1"'\n'                 # new line
PS1="$PS1"'$ '                 # prompt: always $
MSYS2_PS1="$PS1"               # for detection by MSYS2 SDK's bash.basrc

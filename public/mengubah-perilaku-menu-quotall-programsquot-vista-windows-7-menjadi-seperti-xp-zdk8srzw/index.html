Secara pribadi saya menyerah dan telah belajar untuk menerima Start Menu baru yang telah coba diterapkan oleh Microsoft sejak Vista. Tetapi jika Anda tidak ingin menginstal aplikasi untuk ini, berikut cara melakukannya (semacam):

Hasil akhirnya akan seperti ini:

Ayo bersenang-senang:

Pertama, pastikan folder My Documents Anda ditampilkan sebagai Menu di Start Menu

  • Klik kanan Menu Mulai dan Pilih Properti
  • Dari Tab Start Menu, klik Sesuaikan ...
  • Telusuri Dokumen item dan memeriksa Ditampilkan sebagai Menu . Menerapkan dan kembali ke Desktop.

Sekarang kita akan memasukkan folder Program dalam daftar itu dan menghapus (jika Anda mau) folder Dokumen.

  • Klik Start Menu dan Klik Kanan itu Dokumen item di daftar sebelah kanan. Pilih Properti .
  • Klik Sertakan Folder ... dan navigasi ke C:\ProgramData\Microsoft\Windows\Start Menu\Programs. Klik Sertakan Map .
  • Jika mau, sekarang Anda dapat menghapus dua folder dokumen dengan memilih mereka satu per satu dan mengklik Menghapus . Jangan khawatir. Anda dapat melihat file Pulihkan Default tombol bawah itu kotak dialog yang akan meletakkan segalanya kembali ke tempatnya jika Anda tidak suka saya t.
  • Terima perubahan Anda.

Satu sentuhan terakhir

  • Klik Kanan Dokumen item di Start Menu lagi dan pilih Ganti nama .
  • Ubah namanya menjadi Programs.

Dan Voilá!

Memulihkannya seperti semula adalah hal yang mudah. Cukup klik itu Pulihkan Default tombol yang kita bahas di atas dan ganti nama item Program kembali ke Dokumen .

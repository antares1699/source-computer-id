Bagaimana cara mengubah pekerjaan pengikatan kunci gnome Ubuntu saya sebagai emacs?

, Halo semuanya

Bagaimana cara mengubah pengeditan pengikatan kunci di Gnome di Ubuntu, dan membuatnya berfungsi seperti Emacs?

Misalnya. Saat saya mengobrol dengan teman saya, dan saya ingin menggunakan Ctrl-a Ctrl-k untuk memindahkan kursor ke kiri baris saat ini, lalu potong baris saat ini, sebagai OSX atau Emacs.
Secara default di Gnome Ubuntu, Ctrl-a adalah untuk "pilih semua" dan ctrl-k untuk apa-apa.

Di manakah pengikatan kunci ini didefinisikan? Dan bagaimana saya bisa mengubah ini?

Saya mencoba gnome-keybinding-properties dan gsettings , tidak menemukan apa pun.

Mungkin saya harus menyusun sesuatu?

Saya menemukan jawabannya pada laporan bug yang menarik

Laporan ini menunjukkan kepada saya dua cara berbeda untuk mengubah skema kunci pada Gnome lama dan yang lebih baru

Saya mencoba ini di Ubuntu 11.10 (dengan Gnome 3), berhasil!

gsettings set org.gnome.desktop.interface gtk-key-theme "Emacs"

Jika gsettings tidak berfungsi, yang berikut mungkin tersedia

gconftool-2 -s /desktop/gnome/interface/gtk_key_theme -t string Emacs

Apakah ada monitor yang berfungsi dengan Windows 8?

Saya hampir siap untuk beralih ke Windows 8, tetapi saya memiliki satu kekhawatiran terakhir: Apakah ada monitor yang berfungsi dengan Windows 8? Saya menggunakan Dynex TV sebagai monitor saya, dan berfungsi dengan baik dengan Windows 7. Apakah akan berfungsi dengan Windows 8? Apa persyaratan monitor untuk Windows 8?

Windows 8 memiliki dua persyaratan resolusi:

  • Anda membutuhkan setidaknya 768 vertikal piksel agar dapat menggunakan aplikasi Metro sama sekali. Ini berarti resolusi 1024 × 768 atau lebih besar. Resolusi yang lebih kecil hanya dapat menjalankan aplikasi desktop lama.

  • Anda membutuhkan setidaknya 1366 horisontal piksel agar dapat menggunakan Metro Snap (menampilkan dua aplikasi Metro di layar secara bersamaan). Ini berarti resolusi 1366 × 768 atau lebih besar 1 .

1 Lebih besar berarti lebih banyak piksel di kedua dimensi. Misalnya, 1280 × 1024 secara teknis memiliki resolusi yang lebih besar dari 1366 × 768 berdasarkan jumlah piksel tipis (1,3 megapiksel vs. 1,0), tetapi tidak cukup lebar untuk memungkinkan Metro Snap.

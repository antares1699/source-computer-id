Bagaimana cara cd ke drive jaringan pada Windows 8?

Saya ingin menjalankan perintah melalui cmd dan folder tempat saya ingin melakukannya adalah drive yang dipetakan jaringan E:.

Dengan menggunakan cmd, saya ingin mengubah ke direktori, tetapi mendapatkan kesalahan:

C:\>e:
The system cannot find the drive specified.

C:\>cd e:
The system cannot find the drive specified.

C:\>cd /D e:
The system cannot find the drive specified.

C:\>

Mencari di SO untuk masalah serupa, saya menemukan posting yang menyebutkan menggunakan pushd:

C:\>pushd /D \\vboxsrv
The network name cannot be found.

Ada tips?

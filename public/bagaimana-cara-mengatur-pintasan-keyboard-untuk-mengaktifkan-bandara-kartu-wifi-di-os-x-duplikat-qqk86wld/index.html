Berdasarkan pertanyaan serupa Bagaimana cara mereset koneksi jaringan (Bandara) dari baris perintah?

Buat dua file applescript:

do shell script "/usr/sbin/networksetup -setairportpower en1 on"
do shell script "/usr/sbin/networksetup -setairportpower en1 off"

Kemudian gunakan fastscripts untuk menetapkan pintasan keyboard kepada mereka.

Aktifkan dan nonaktifkan AirPort menggunakan pintasan keyboard
17 Okt, '08 07:30:03 • Kontribusi oleh: Anonymous

Ini adalah cara termudah yang saya temukan untuk menghidupkan dan mematikan kartu AirPort tanpa menggunakan aplikasi pihak ketiga atau terlalu banyak tombol. Semoga orang lain akan menghargai fungsi ini seperti saya. Catatan: Fungsi ini disesuaikan dengan MacBook, karena tombol F5 dan F6 tidak memiliki fungsi yang ditentukan sebelumnya pada mesin ini. Mesin lain mungkin memerlukan tombol pintas untuk diedit.

1- Buat dua pintasan keyboard. Buka menu Apple »Preferensi Sistem» Keyboard & Mouse »tab Pintasan Keyboard. Klik tanda plus di bagian bawah jendela, dan pilih Semua Aplikasi dari menu pop-up di dialog berikutnya. Dalam kotak Menu Title, ketik Turn AirPort On dengan huruf dan ejaan yang sama persis. Atur Keyboard Shortcut ke F5, lalu klik Add. Klik lagi tanda plus, biarkan menu pop-up disetel ke Semua Aplikasi, dan di kotak Judul Menu, ketik Matikan AirPort dengan tombol pintasan juga disetel ke F5 - hanya satu dari fungsi ini yang akan ditampilkan pada satu waktu.

2 - Di bagian Navigasi Keyboard pada jendela yang sama, ubah 'Pindahkan fokus ke menu status di bilah menu' ke F6 (dan centang kotak untuk mengaktifkan pintasan ini, jika belum diaktifkan).

3 - Pastikan ikon AirPort ada di bilah menu Anda. Kemudian tahan tombol Command dan seret ikon bilah Menu AirPort sejauh mungkin ke kiri. Ini memastikan bahwa ini adalah yang pertama dipilih saat Anda memindahkan fokus ke bilah menu.

4 - Terakhir, mulai ulang sistem Anda. Setelah Mac dinyalakan ulang, tekan F6 untuk memindahkan fokus ke bilah menu, dan F5 untuk mengaktifkan atau menonaktifkan AirPort, tergantung pada statusnya saat ini.

Selain masalah dengan menyeret ke posisi paling kiri (karena "pintasan" Mac lainnya memerlukan posisi yang sama), ini seharusnya berfungsi dengan baik.

#!/bin/sh

device="$(networksetup -listallhardwareports |
grep -E '(AirPort|Wi-Fi)' -A 1 | grep -o "en.")"
[[ "$(networksetup -getairportpower $device)" == *On ]] && val=off || val=on
networksetup -setairportpower $device $val

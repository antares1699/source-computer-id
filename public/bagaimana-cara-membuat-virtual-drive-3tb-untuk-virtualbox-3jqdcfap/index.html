Bagaimana cara membuat virtual drive 3TB untuk virtualbox?

Saya perlu membuat drive virtual besar untuk kotak virtual pada 3TB, namun, ukuran maksimal dalam kotak virtual tampaknya 2TB. Bagaimana cara membuat disk / drive virtual 3TB? Terima kasih.

4
Daniel — Sumber

1 Jawaban

6
6

Anda tidak dapat menggunakan GUI untuk melakukannya, tetapi Anda dapat menggunakan baris perintah seperti:

 vboxmanage createhd --filename 3TB.vdi --size 3000000 --format VDI --variant Standard

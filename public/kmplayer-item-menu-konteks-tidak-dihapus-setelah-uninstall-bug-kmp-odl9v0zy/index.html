KMPlayer: Item menu konteks tidak dihapus setelah uninstall (bug KMP)

Saya telah menghapus KMPlayer tetapi 2 item menu konteks (Mainkan dengan KMP, Antrean di KMP) masih terlihat ketika saya mengklik kanan pada folder mana pun. Bagaimana cara menghapus sisa ini?

OS: Win7 x64

Catatan: Saya telah mencopot KMP pada mesin yang berbeda beberapa kali, dan menurut saya ini adalah bug di KMP.

Saya telah menggunakan RegAlyzer untuk menemukan item yang berisi kunci, nilai, dan data "KMPlayer" di registri. Ditemukan 708 entri! Saya menghapus semuanya dan sekarang sisa menu konteks hilang bersama dengan kotoran KMP lainnya. Saya tidak tahu seberapa besar solusi ini dapat diandalkan, tetapi berhasil. Jika ada, saya dapat menerima solusi alternatif yang lebih baik.

Dan pelajaran yang saya dapatkan: KMP adalah pemain yang bagus kecuali Anda menghapusnya! Itu merusak registri Anda berkat uninstaller (!) Yang luar biasa.

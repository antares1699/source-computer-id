Bagaimana Anda mentransfer akun pengguna ke komputer lain?

Saya sedang bepergian dan saya menggunakan laptop yang bukan milik saya (karena saya meninggalkan laptop saya sendiri di rumah). Saya terlalu terbawa dengan laptop pinjaman dan saya membuat akun pengguna di atasnya.

Saya tidak ingin membuat akun baru ketika saya kembali menggunakan laptop saya sendiri, jadi saya perlu mentransfer formulir akun pinjaman laptop saya ke laptop saya. Apakah ada yang tahu bagaimana caranya?

7

Saya berasumsi Anda bekerja pada Windows-7 dan kemudian jauh lebih mudah untuk mentransfer profil Pengguna. Cukup klik Mulailah tombol dan kemudian ketik di kotak pencarian Windows Easy Tranfer dan klik di atasnya atau Anda dapat menjalankannya sebagai file admin .

Sekarang klik Next

Sekarang pilih opsi ketiga karena Anda harus membawanya bersama Anda.

Sekarang pilih opsi kedua setelah ini Windows akan memindai file dari semua pengguna di PC ini dan kemudian Anda dapat menyalinnya di USB atau HDD eksternal Anda.

Catatan Anda juga dapat menggunakan metode ini untuk mentransfer file dari Windows Vista ke Windows-7 karena fitur ini juga tersedia di sana.

Jika Anda ingin mentransfer Profil pengguna dari XP ke Windows-7 / Vista lalu unduh Transfer Mudah .

Untuk Windows XP Anda dapat mendownload ini panduan .

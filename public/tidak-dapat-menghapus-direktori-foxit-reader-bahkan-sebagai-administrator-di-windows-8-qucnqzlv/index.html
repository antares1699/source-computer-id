Tidak dapat menghapus direktori Foxit Reader, bahkan sebagai Administrator, di Windows 8

Saya mencoba memutakhirkan Foxit Reader dan direktori dalam keadaan tidak terbaca. Saya tidak dapat melihat direktori, atau bahkan menghapusnya, bahkan sebagai Administrator. Bagaimana cara menghapusnya agar saya dapat menginstal ulang?

C:\Program Files (x86)\Foxit Software>takeown /f "Foxit Reader"
ERROR: Access is denied.

C:\Program Files (x86)\Foxit Software>takeown /a /f "Foxit Reader"
ERROR: Access is denied.

C:\Program Files (x86)\Foxit Software>icacls "Foxit Reader" /setowner Administrator
Foxit Reader: Access is denied.
Successfully processed 0 files; Failed processing 1 files

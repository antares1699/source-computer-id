Ubah perangkat yang dikontrol oleh tombol volume tanpa mengubah default

Audio saya dialihkan melalui perangkat kedua (default), yang kemudian diumpankan ke perangkat utama. Mengubah volume perangkat kedua tidak memengaruhi apa pun, meskipun saya setel ke 0.

Windows membutuhkan perangkat default yang dikontrol oleh tombol volume pada keyboard.

Apakah ada cara selain mengubah default, untuk mengontrol perangkat mana yang dikontrol oleh tombol volume?

p.s. Tombol volume secara umum mengacu pada apa yang dapat diimplementasikan secara fisik secara berbeda (sesuatu yang mengirimkan karakter vol atas / bawah)

Juga, saya menggunakan Win8.1, jika itu mengubah sesuatu.

Jawaban BLARG berhasil untuk saya, tetapi saya memutuskan untuk melihat apakah saya bisa membuatnya berfungsi hanya dengan AutoHotKey, dan ternyata ya, Anda bisa.

; Use the Soundcard Analysis script found here to set these parameters
; https://www.autohotkey.com/docs/commands/SoundSet.htm#Soundcard
DeviceNumber := 7
ComponentID := MASTER

Volume_Up::
SoundSet, +2, %ComponentID%, VOLUME, DeviceNumber
return

Volume_Down::
SoundSet, -2, %ComponentID%, VOLUME, DeviceNumber
return

Volume_Mute::
SoundSet, -1, %ComponentID%, MUTE, DeviceNumber
return

Setelah Anda mengatur parameter perangkat dengan benar dan mengonfirmasi bahwa itu berfungsi, Anda dapat mengompilasi skrip menjadi .exe menggunakan pengonversi AHK-to-EXE yang disertakan dengan instalasi AHK dan menautkannya di folder startup Anda untuk memulainya secara otomatis di Gabung. Jika Anda ingin volume naik / turun lebih cepat, Anda dapat melakukannya dengan memodifikasi nilai + 2 / -2 (Saya yakin 2% adalah perubahan default per tekan pada Windows, tetapi rollover mungkin lebih lambat daripada tanpa AHK).

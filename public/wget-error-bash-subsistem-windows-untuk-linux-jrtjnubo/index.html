HSTS adalah pengukuran keamanan opsional di atas HTTPS. Karena koneksi HTTPS itu sendiri berfungsi dan file telah diunduh, ini tampaknya lebih merupakan peringatan daripada kesalahan. Oleh karena itu saya melihat tiga opsi:

  1. Nonaktifkan HSTS pada setiap panggilan menggunakan opsi wget --no-hsts
  2. Cobalah untuk membuat file pada /home/atul/.wget-hsts dengan izin yang sesuai dan coba jika wget menggunakan file ini
  3. Abaikan saja peringatan ini
Bagikan

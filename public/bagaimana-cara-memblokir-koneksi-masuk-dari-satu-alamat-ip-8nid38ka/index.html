Bagaimana cara memblokir koneksi masuk dari satu alamat IP?

Saya telah menemukan bahwa satu alamat IP mencoba RDP ke mesin.

Sekarang saya ingin mendapatkan XP untuk memblokir semua koneksi masuk dari alamat IP ini, dan saya merasa sangat frustasi betapa sulitnya tugas sederhana seperti itu.

Bagaimana cara memblokir koneksi masuk dari satu alamat IP?

Jika koneksi masuk melewati router / modem Anda, itu sudah menjadi ancaman potensial bagi sistem lain di jaringan Anda, atau akan beberapa saat, setelah mendeteksi lubang di infrastruktur Anda.

Kecuali Anda memiliki perangkat keras atau modem yang sangat tua, saya sarankan untuk mencoba memblokir IP terlebih dahulu pada pengaturan firewall di router Anda.

Masukkan frontend web administratif Anda dari router Anda (lihat manual untuk login dan IP standar) dan masukkan IP yang akan diblokir di sana.

Jika ini gagal, instal / konfigurasikan Windows Firewall di XP Anda dan lakukan seperti yang diinginkan halaman ini kata.

Serangan presisi
Jika router Anda meminta alamat ip DAN subnet mask untuk itu:
Subnet masking adalah untuk menentukan rentang IP berdasarkan berapa banyak bit yang diizinkan mulai dari sisi kanan alamat IP. Untuk melayani satu alamat IP, topeng alamat adalah 255.255.255.255 atau / 32, tergantung notasinya. Jika Anda perlu menentukan rentang, pelajari Halaman WIKI .

Pemboman karpet
Blokir semua koneksi masuk - Ini sudah menjadi default di sebagian besar router, saat pemfilteran diaktifkan. Kemudian Anda dapat mengizinkan alamat / rentang IP yang dipilih satu per satu.

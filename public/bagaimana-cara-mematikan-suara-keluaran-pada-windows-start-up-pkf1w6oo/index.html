Rupanya saya salah paham dengan pertanyaan Anda.

Sunting: Saya menemukan cara yang lebih baik:

Auto Mute adalah kontrol volume portabel perangkat lunak yang menyediakan akses ke beberapa fitur menarik. Bisa Misalnya digunakan secara otomatis bisu suara di logoff, suspend dan shutdown sehingga sistem selanjutnya dimulai sunyi.

Pintasan global ditawarkan untuk menghidupkan kembali suara setelah sistem komputer telah dimulai. Itu shortcut default adalah Ctrl F10 yang bisa diubah dalam opsi program jika perlu.

Satu-satunya opsi lain yang disediakan oleh perangkat lunak kontrol volume untuk menjalankan program selama startup dan untuk memainkan a komputer berbunyi bip setiap kali negara suara berubah dari diam menjadi aktif dan sebaliknya.

Auto Mute adalah perangkat lunak portabel program untuk operasi Windows sistem. Program ini kompatibel dengan banyak sistem operasi Microsoft termasuk Windows 7 dan Windows Vista. Bisa jadi diunduh dari situs web pengembang.

Jika tidak, Anda harus melihat program yang sangat lama bernama: SetVol

Ini mungkin masih berfungsi di bawah Windows XP. Deskripsi tersebut berbunyi:

Jika Anda memiliki banyak pengguna yang menggunakan komputer yang sama, aplikasi ini bisa digunakan untuk mengatur level volume secara otomatis setiap kali pengguna masuk di. SetVol memungkinkan Anda membuat kustom pintasan untuk mengatur volume suara yang dihasilkan komputer Anda.

Jika Anda menggunakan + atau -, Anda dapat membuatnya pintasan yang bertambah atau berkurang volume dengan kelipatan 10%. Untuk Misalnya, pintasan yang membuat file volume 10 persen lebih keras akan menggunakan baris perintah Setvol.exe +10. Di Di sisi lain, Anda juga dapat menghilangkan + atau - untuk mengatur volume pada tingkat absolut. Misalnya, baris perintah "Setvol.exe 100" akan mengubah volume hingga maks.

Anda kemudian dapat mengatur skrip startup yang secara otomatis akan menjalankan "Setvol.exe 0" mematikan sistem Anda.

Untuk orang lain: coba mematikan suara start up Windows?

Pergi ke Start -> Settings -> Control Panel kemudian memilih Control Panel "Perangkat Suara dan Audio".

Klik pada tab "Sounds" dan itu akan terlihat seperti ini:

Gulir ke bawah sedikit dan Anda akan menemukannya Mulai Windows dan Windows Logon:

Itu adalah dua hal yang ingin Anda ubah atau nonaktifkan seluruhnya. Untuk mengubahnya, klik menu pop-up itu saat ini bertuliskan "Windows XP Startup" di tangkapan layar saya di atas.

Untuk menonaktifkannya agak tidak intuitif: Anda juga perlu mengklik pop-up menu, tapi kali ini Anda tertarik di opsi atas:

Pilih (Tidak Ada), klik "Terapkan"

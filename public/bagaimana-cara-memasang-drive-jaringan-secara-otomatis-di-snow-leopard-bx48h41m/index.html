Bagaimana cara memasang drive jaringan secara otomatis di Snow Leopard?

Saya ingin secara otomatis me-mount share SMB tertentu setiap kali saya masuk ke akun saya di Snow Leopard. Jauh di masa lalu di Mac Classic, yang harus saya lakukan hanyalah Get Info pada drive yang dimaksud dan memilih Mount at Startup, tetapi Snow Leopard tampaknya tidak memiliki opsi yang setara. Meskipun saya telah menemukan beberapa peretasan yang mengklaim memasang drive saat boot sistem, (1) mereka tidak berfungsi, dan (2) mereka tidak melakukan apa yang saya inginkan: Saya ingin itu dipasang secara otomatis ketika saya masuk dan melepas pemasangan ketika saya keluar, tidak dipasang pada boot sistem.

Apakah ada cara bawaan untuk melakukan ini? Atau haruskah saya menggunakan rute AppleScript? Jika rute terakhir, apakah itu menyiratkan bahwa tidak ada cara untuk mengotomatiskan ini di seluruh situasi jaringan?

Bisakah Anda menambahkan perintah berikut ke item startup Anda?

mount -t smbfs //user@server/sharename share

Secara khusus, saya akan memasukkannya ke dalam Applescript seperti ini:

do shell script "mount -t smbfs //user@server/sharename share"

Kemudian saya akan menyimpan file sebagai aplikasi dan menambahkannya ke item login saya.

Pasang volume yang Anda inginkan. Kemudian, cukup seret volume ke Item Login di dalam System Preferences / Users & Groups / (akun Anda). Pastikan kredensial yang perlu Anda sambungkan disimpan di rantai kunci Anda.

Anda akan melihatnya muncul sebagai Item Login lain, dengan tipe "Folder". Itu dia.

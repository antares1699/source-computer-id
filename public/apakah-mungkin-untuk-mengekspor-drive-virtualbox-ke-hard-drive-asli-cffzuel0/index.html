Apakah mungkin untuk mengekspor drive VirtualBox ke hard drive “asli”?

Saya memiliki VM Windows 7 di Host VirtualBox Mac OS X saya. Drive virtual berkembang secara dinamis dengan batas 20GB.

Apakah mungkin mengekspor hard drive virtual .vdi ke hard drive fisik nyata, untuk dimasukkan ke komputer fisik nyata?

Saya memiliki casing eksternal untuk memasang drive IDE ke USB, dan dapat meneruskannya ke VB, tetapi hanya menyalin semua file di drive virtual tidak akan menyalin hal-hal seperti sektor boot.

Selain itu, jika saya dapat melakukan ini, apakah saya harus mengaktifkan kembali Windows, atau melakukan sesuatu yang istimewa?

Ya, mungkin yang Anda butuhkan hanyalah Acronis true image home 2013 dan sysprerp Buat mesin virtual Anda dengan windows 7 atau vista (xp slighlty difrent) perbarui dengan semua pembaruan dan program windows. buka c: \ windows \ system32 \ sysrep \ sysprerp.exe klik dua kali setelah Anda puas dengan pengaturan Anda, komputer akan menghapus default perangkat keras tetapi tetap mempertahankan windows dan semua pembaruan dan program.

Pastikan "GENERALIZE" dicentang dan "SHUT DOWN" dipilih kemudian instal Acronis true image home di komputer host buka alat dan buat iso boot Rescue Acronis untuk menyimpan ini di desktop atau di tempat yang praktis. Buka kotak virtual dan klik kanan mesin virtual Anda dan masuk ke pengaturan lalu penyimpanan dan klik ikon cd lalu folder "Pilih folder Virtual" lalu pilih Acronis ISO Anda. Klik ok di bagian bawah Nyalakan mesin virtual Anda dan bersiaplah di keyboard. Gunakan tombol panah untuk membuka Acronis true image home dari menu !! (jangan biarkan windows boot atau Anda memulai kembali) Setelah berada di layar pencadangan dan pemulihan, Klik Perangkat di kiri atas layar Anda dan kemudian usb dan cari drive eksternal Anda, klik dan Acronis tiba-tiba akan melihat partisi baru untuk dipindai. kemudian kembali ke perangkat dan klik mouse usb. sehingga Anda dapat menggunakan mouse di Acronis.

Klik drive disk cadangan yang menanyakan apa yang ingin Anda buat cadangannya. Pastikan partisi sistem dan jendela dicentang. Anda harus memiliki sesuatu seperti DISK 2, SYSTEM RESERVED, UNNAMED.

Kemudian pilih telusuri lokasi kemudian temukan drive untuk membuang gambar Anda ke hard drive eksternal berfungsi dengan baik. beri nama file gambar Anda sesuatu yang berguna seperti Windows7homepreium32BIT lalu klik ok dan lanjutkan. Centang kotak di Acronis untuk mematikan. Pergi minum secangkir teh ini bisa memakan waktu 10 menit tergantung pada spesifikasi mesin.

Kotak virtual Anda akan dimatikan dengan kotak virtual Anda mesin host akan mengendalikan hard drive eksternal Anda dan Anda dapat melihat gambar Anda.

Dan ulangi langkah-langkah tentang kecuali kali ini dari menu Acronis pilih pulihkan dan pilih gambar Anda dari hard drive dan pilih hard drive tujuan Anda dan klik lanjutkan salinan prefek windows setiap kali

Kemungkinan besar Anda mengekspor mesin virtual ke mesin fisik, Anda harus mengaktifkannya kembali. Pertama, sistem akan melihat bahwa perangkat kerasnya berbeda (motherboard, harddisk, RAM, prosesor, kartu video). Kedua, Anda akan menemukan bahwa Windows perlu mencari driver untuk perangkat keras tertentu yang diinstal pada mesin (video, NIC, suara, dll). Terakhir, jika Anda mengaktifkan kembali windows pada perangkat keras baru, lisensi Anda untuk OS tidak akan memungkinkan Anda untuk tetap menjalankan virtual dan juga. Ini adalah area abu-abu, tetapi biasanya Anda hanya dapat menjalankan satu contoh OS pada satu waktu (meskipun saya ragu polisi EULA akan menggerebek rumah Anda untuk ini). Ingat saja itu.

Semoga berhasil dalam proyek Anda dan semoga ini membantu beberapa.

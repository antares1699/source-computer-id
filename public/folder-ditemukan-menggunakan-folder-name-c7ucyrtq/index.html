Folder ditemukan menggunakan% folder-name%

Karena kita bisa pergi ke folder temp dengan mengetikkan% temp% di jendela run. Apakah ada cara untuk membuat perintah ini berfungsi untuk folder kita juga?

misal-Jika kita memiliki folder bertuliskan SUPER maka saya ingin folder ini terbuka ketika saya memberikan% SUPER% di jendela run

%temp% mereferensikan variabel lingkungan.

Ini didefinisikan di Control Panel / System / Advanced / Environment Variables untuk XP dan di Windows 7 Anda bisa mencari "lingkungan" di menu mulai dan memilih Edit Environment Variables.

Tambahkan nama variabel yang Anda inginkan di kolom nama, tanpa tanda persentase. Tambahkan folder yang ingin Anda buka di bidang nilai.

Jika ini hanya untuk pengguna, maka terapkan di tingkat pengguna, atau gunakan sistem jika itu untuk semua pengguna.

Tentu.

Untuk mengatur variabel %SUPER%, cukup jalankan perintah ini:

set SUPER=C:\foo\bar\SUPER

Untuk menyetel variabel ini secara otomatis setiap kali Anda membuka terminal, buka

Control Panel -> System -> Advanced System Settings -> Environment Variables...

dan buat variabel pengguna (untuk akun Anda) atau variabel sistem (untuk semua akun) dengan nama SUPER dan nilai C:\foo\bar\SUPER.

Ya kamu bisa.

Sebenarnya Anda perlu membuat variabel lingkungan baru untuk folder Anda agar Anda dapat mengaksesnya dengan cepat menggunakan trik Anda.

Katakanlah untuk C: \ USER \ SUPER:

  1. Klik kanan pada Komputer Saya> Properti> Pengaturan Sistem Lanjut> tab Lanjut> Variabel Lingkungan ...

  2. Tambahkan variabel baru Anda, Anda dapat memilih nama apa pun yang Anda inginkan, nilainya harus jalur lengkap folder Anda. Kasus ini: SUPER dan C: \ USER \ SUPER

  3. Akses dengan mengetik% SUPER%

Ini berfungsi di Windows 7 saya, tidak yakin di versi lain.

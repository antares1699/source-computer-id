Mendefragmentasi sebelum menginstal ulang Windows?

Saya akan segera menginstal ulang Windows XP pada mesin bekas. Data di hard drive saat ini sangat terfragmentasi.

Saya akan memformat drive saat menginstal ulang Windows, jadi apakah ada gunanya mendefrag drive sekarang?

Tidak. Jangan repot-repot. Formatnya akan menghapus semuanya. Defrag benar-benar membuang-buang waktu.

  • juga saat Anda melakukan format cepat, Anda tidak benar-benar memformat seluruh disk, hanya membuat struktur disk baru. Semua data pada disk dibiarkan sendiri dan jika Anda khawatir tentang keamanan, alat yang berusia 12 tahun dengan beberapa alat yang tersedia dapat mengakses data Anda.

  • @ g19fanatic: Saya tidak yakin itu masalahnya. Menurut Microsoft ( support.microsoft.com/kb/302686 ) satu-satunya perbedaan antara format Penuh dan Cepat adalah bahwa "Jika Anda memilih opsi format Cepat, format menghapus file dari partisi, tetapi tidak memindai disk untuk menemukan sektor yang buruk"
    — Siim K
  • @Siim K: Format cepat hanya menghapus MFT, yang "menghapus file dari partisi." Pikirkan tentang ini: Jika benar dihapus semua file, menurut Anda berapa lama waktu yang dibutuhkan untuk memformat cepat HD 1TB, dibandingkan dengan berapa lama yang sebenarnya dibutuhkan? Jika Anda ingin memeriksa kesalahan pada disk, gunakan alat vendor HD untuk melakukannya.
    — afrazier
  • @Siim K: Microsoft tidak akan menjelaskan secara rinci tentang bagaimana file tersebut dihapus. Saat Anda memindahkan file ke 'Recycling Bin' dan kemudian 'delete' secara permanen, itu tidak benar-benar menghapus file tersebut sepenuhnya. Ini hanya menghapus file dari MFT. File tersebut sebenarnya masih dalam format HD ... Itulah yang dilakukan oleh format cepat, menghapus MFT dan membuat pesan baru (disk tidak memiliki file di dalamnya). Datanya masih ada. Format lengkap (dari apa yang saya pahami), selain melakukan pemeriksaan disk, akan menimpa setiap sektor, sehingga benar-benar menghapus data (meskipun masih tidak aman jika Anda paranoid ...)
    — g19fanatic
  • Ya, datanya mungkin masih ada, tetapi tidak akan mengakibatkan fragmentasi. Ini hanya masalah jika Anda akan menjual komputer Anda, dalam hal ini Anda tetap harus menggunakan sesuatu seperti DBAN.
    — Billy ONeal
  • Cara pemeriksaan format penuh untuk bad sector adalah dengan menulis pola standar ke setiap sektor dan kemudian membacanya kembali untuk melihat apakah itu benar. Ini memiliki efek samping dengan benar-benar menghapus konten lama dari sektor tersebut. Itulah mengapa format lengkap lebih aman daripada format cepat.
    — Walter Mitty

Di chkdsk, apa itu spot fixing?

(Terkait dengan, tetapi berbeda dari: Di chkdsk, apa itu pemindaian online? .)

Menjalankan Windows 8.0 Pro, chkdsk memiliki opsi /spotfix yaitu Runs spot fixing on the volume. Apa itu spot fixing? Jika berhubungan dengan memperbaiki drive secara online, saya pikir /scan tanpa /forceofflinefix menanganinya.

EDIT: Ini mungkin terkait dengan tidak memperbaiki kesalahan saat terdeteksi, tetapi menyimpan tempat dan memperbaikinya nanti.

Berikut opsi lengkapnya:

CHKDSK [volume[[path]filename]]] [/F] [/V] [/R] [/X] [/I] [/C] [/L[:size]] [/B]
[/scan] [/spotfix]


  volume              Specifies the drive letter (followed by a colon),
                      mount point, or volume name.
  filename            FAT/FAT32 only: Specifies the files to check for
                      fragmentation.
  /F                  Fixes errors on the disk.
  /V                  On FAT/FAT32: Displays the full path and name of every
                      file on the disk.
                      On NTFS: Displays cleanup messages if any.
  /R                  Locates bad sectors and recovers readable information
                      (implies /F, when /scan not specified).
  /L:size             NTFS only:  Changes the log file size to the specified
                      number of kilobytes.  If size is not specified, displays
                      current size.
  /X                  Forces the volume to dismount first if necessary.
                      All opened handles to the volume would then be invalid
                      (implies /F).
  /I                  NTFS only: Performs a less vigorous check of index
                      entries.
  /C                  NTFS only: Skips checking of cycles within the folder
                      structure.
  /B                  NTFS only: Re-evaluates bad clusters on the volume
                      (implies /R)
  /scan               NTFS only: Runs a online scan on the volume
  /forceofflinefix    NTFS only: (Must be used with "/scan")
                      Bypass all online repair; all defects found
                      are queued for offline repair (i.e. "chkdsk /spotfix").
  /perf               NTFS only: (Must be used with "/scan")
                      Uses more system resources to complete a scan as fast as
                      possible. This may have a negative performance impact on
                      other tasks running on the system.
  /spotfix            NTFS only: Runs spot fixing on the volume
  /sdcleanup          NTFS only: Garbage collect unneeded security descriptor
                      data (implies /F).
  /offlinescanandfix  Runs an offline scan and fix on the volume.

The /I or /C switch reduces the amount of time required to run Chkdsk by
skipping certain checks of the volume.

Windows 8 memiliki tugas pemeliharaan bawaan yang berjalan setiap hari. Windows 8 akan memeriksa kerusakan yang telah diverifikasi ini dan memasukkannya ke dalam sistem untuk diperbaiki nanti. Sekali lagi, ini semua dilakukan saat sistem online. Tahap selanjutnya adalah Perbaiki Spot . Di sinilah chkdsk di Windows 8 sangat berbeda. Spot Fix adalah parameter baru yang memeriksa disk dan memperbaiki masalah apa pun hanya dalam beberapa detik. Waktu yang diperlukan untuk menjalankan chkdsk menggunakan spotfix didasarkan pada jumlah kerusakan, bukan jumlah file seperti pada versi Windows yang lebih lama. Ini berarti semuanya diperbaiki dalam hitungan detik.

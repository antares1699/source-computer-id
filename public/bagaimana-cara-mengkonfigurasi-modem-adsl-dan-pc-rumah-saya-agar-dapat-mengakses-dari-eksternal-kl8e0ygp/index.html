Untuk memperluas jawaban TheTXI

Menggunakan layanan seperti DynDNS Anda membuat akun baru yang akan memberi Anda URL unqiue, misalnya myhome.dyndns.org.

Bergantung pada apakah Anda memiliki Router ADSL atau Modem, prosesnya sedikit berbeda. Saat menggunakan perute ADSL, Anda perlu mengonfigurasi pengaturan IP Dinamis pada perute untuk mengarahkan ke URL DynDNS baru Anda menggunakan nama pengguna dan kata sandi yang Anda berikan. Ini akan memastikan bahwa setiap kali baris me-reset IP baru diberikan ke DynDNS. Proses ini biasanya membutuhkan waktu sekitar 20 menit. Jika Anda menggunakan Modem ADSL, Anda perlu mengunduh perangkat lunak terpisah untuk melakukannya.

Saat menggunakan router ADSL Anda juga perlu mengatur Port Forwarding. Ini akan memungkinkan router mengirim data untuk port tertentu, baik VNC atau RDP ke mesin Anda. Untuk menggunakan ini, saya sarankan untuk memberikan mesin yang ingin Anda sambungkan ke IP Statis secara langsung atau menggunakan IP statis yang ditetapkan DHCP berdasarkan alamat Mac.

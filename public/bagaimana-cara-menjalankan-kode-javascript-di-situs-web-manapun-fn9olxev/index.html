Bagaimana cara menjalankan kode Javascript di situs web manapun?

Bagaimana cara menjalankan kode Javascript di situs web untuk menyelesaikan fungsi atau aturan yang ditetapkan sesuai situs web?

Misalnya, jika saya mencoba mengunduh file dari beberapa filehost yang memerlukan survei ... Sekarang dalam kode sumber, kode tersebut memerlukan respons JSON untuk mengizinkan unduhan file setelah file diunduh (coba tebak!). .. Jika saya menulis kode dalam Javascript, dan menjalankannya di situs itu, mengirimkan kode JSON (true) ke penangan kejadian laman, dapatkah saya mengunduh file tersebut?

Ini hanya sebuah contoh, saya terutama mencoba menjalankan kode Javascript di situs web sesuai kode situs web. Saya pernah mendengar tentang Greasemonkey, dan beberapa ekstensi dan tambahan serupa lainnya, tetapi tidak begitu tahu cara menggunakannya atau untuk apa mereka ...

Tolong bisakah seseorang menjelaskan bagaimana saya bisa melakukan itu?

Anda tidak memerlukan ekstensi untuk itu. Menggunakan konsol .

Untuk membuka tab Konsol, lakukan salah satu dari berikut ini:

  • Gunakan pintasan keyboard Perintah - Pilihan - J (Mac) atau Kontrol - Bergeser - J (Windows / Linux).

  • Pilih Tampilan> Pengembang> Konsol JavaScript .

Sekarang Anda dapat menjalankan Javascript apa pun yang Anda inginkan. Variabel dan fungsi yang ditentukan dalam lingkup global dokumen saat ini juga dapat digunakan.

Misalnya, jika situs saat ini menyertakan jQuery, Anda dapat mengirim JSON ke skrip jarak jauh:

$.post( "http://example.com/test.php", { foo: "bar" } );

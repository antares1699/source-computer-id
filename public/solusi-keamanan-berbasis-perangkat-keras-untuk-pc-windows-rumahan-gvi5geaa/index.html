Telah ada upaya untuk membuat perangkat keras anti-virus tetapi ternyata tidak pernah berhasil.

Dengan googling sedikit, saya menemukan file Pelindung OS Xnet dan Saming NetGuard dan PCGuard tapi deskripsi mereka terlihat sangat meragukan.

Mereka mengklaim melakukan hal-hal yang hampir tidak mungkin, terutama pada saat kartu-kartu ini dibuat (menurut saya pertengahan 90-an):

  • Fitur yang paling kuat adalah file sistem yang dilindungi dapat pulih bahkan ketika pengaturan CMOS rusak.
  • Mudah dipasang dan digunakan, tanpa perintah, tanpa pemrograman, tanpa skrip.
  • Data yang dilindungi tidak menempati ruang Hard Disk yang dapat digunakan , itu berarti tidak ada penyimpanan lain media dan perangkat diperlukan.
  • Memulihkan OS yang rusak, perangkat lunak dan menghilangkan virus, spyware, peretas dengan citra disk yang terlindungi dengan baik. Sebagai pilihan, pengaturan CMOS (yaitu, BIOS) juga dapat dilindungi.
  • Pemulihan data 40GB dapat dilakukan dalam waktu minimal 3 detik, tetapi dapat memerlukan waktu hingga 8 detik
  • Dapat melindungi hingga 2000GB data hard disk.

Jika itu benar-benar mungkin, kita semua akan mengetahuinya dan mereka akan memiliki banyak kartu seperti itu di pasaran.

Saya tidak dapat menemukan ulasan kedua kartu ini oleh situs web terkemuka. Saya pikir kita pasti bisa mengatakan itu minyak ular ...

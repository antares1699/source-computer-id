Anda dapat mengubah teks default dan warna latar belakang di preferensi Firefox (tab Konten, lalu tombol Warna). Jika Anda tidak mencentang kotak di dialog Warna, ini akan memaksa warna-warna ini ke situs web yang menentukan warnanya sendiri.

Untuk situs web tertentu (Gmail, misalnya) Anda dapat memasang ekstensi Greasemonkey yang dapat secara dinamis mengubah hal-hal seperti warna pada laman — asalkan Anda memiliki skrip yang ditulis untuk situs web tersebut.

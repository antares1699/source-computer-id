Bagaimana cara membuat daftar file yang diubah pada atau sebelum tanggal tertentu?

Menggunakan perintah linux, apakah ada cara untuk membuat daftar file yang dibuat satu jam (atau tanggal tertentu) sebelumnya?.

Anda dapat menggunakan perintah find bersama dengan tanda -mmin dan -mtime. Misalnya untuk mendaftar file * .txt di folder Download diubah lebih dari 30 hari yang lalu gunakan ini:

find $HOME/Downloads -name  '*.txt' -mtime +30

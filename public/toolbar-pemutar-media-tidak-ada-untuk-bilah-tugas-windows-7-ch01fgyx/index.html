Toolbar Pemutar Media tidak ada untuk bilah tugas Windows 7

Saya baru saja meningkatkan ke Windows 7 64 bit PC. Pada Windows XP saya bisa mendapatkan Windows Media Player untuk meminimalkan ke bilah tugas dan mendapatkan toolbar WMP yang bagus di Bilah Tugas.

Sekarang ketika saya mengklik kanan taskbar dan memilih "Toolbars", tidak ada toolbar "Windows Media Player" untuk dipilih.

Ada ide bagaimana saya bisa mengaktifkan ini?

Cara Mengaktifkan Bilah Alat Bilah Tugas Pemutar Media Windows di Windows 7

  • Pertama-tama Anda harus mendapatkan salinan file %programfiles%\Windows Media Player\wmpband.dll dari instalasi WMP 11. Anda dapat memperolehnya dari sistem teman Anda atau Anda dapat menggunakan tautan unduhan yang disediakan di akhir posting ini. Ingatlah bahwa Anda harus menyalin dari sistem dengan arsitektur OS yang sama (yaitu 32 bit atau 64 bit).

  • Tempel file ini di direktori %programfiles%\Windows Media Player.

  • Ketik services.msc di Start Menu dan tekan Memasukkan . Pastikan Layanan Berbagi Jaringan Windows Media Player dihentikan.

  • Mulai command prompt sebagai administrator dengan mengetik cmd di Start Menu dan menekan Ctrl + Bergeser + Memasukkan .

  • Ketik regsvr32 “%programfiles%\Windows Media Player\wmpband.dll” dan tekan Memasukkan .

  • Mulai ulang layanan Berbagi Jaringan Windows Media Player (jika Anda telah menghentikannya sebelumnya).

  • Jalankan Windows Media Player dan minimalkan. Klik kanan di tempat kosong di bilah tugas dan pilih Toolbar > Windows Media Player . Abaikan peringatan dan Anda akan mendapatkan toolbar bilah tugas Windows Media Player.

Unduh Windows Media Player wmpband.dll : 32 bit atau 64 bit

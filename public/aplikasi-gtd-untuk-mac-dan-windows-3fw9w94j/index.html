Aplikasi GTD untuk Mac dan Windows

Saya terbiasa dengan aplikasi GTD untuk Mac OS X, seperti Things, Hit List, dll. Namun, saya mencoba mencari aplikasi GTD yang dapat saya gunakan di berbagai OS.

Salah satu fitur utama yang ingin saya miliki adalah sinkronisasi waktu nyata. Jika saya memperbarui di macbook saya, saya ingin melihatnya diperbarui untuk aplikasi di Windows. Apakah ada aplikasi seperti itu?

Sejauh ini, saya dapat menemukan yang tidak memiliki sinkronisasi waktu nyata ini di berbagai sistem operasi.

Alat yang saya gunakan secara teratur adalah Evernote . Ini memiliki antarmuka web, dan klien lintas platform untuk Windows dan MacOSX.

Ini memiliki versi gratis serta berbayar untuk langganan yang lebih komprehensif. Tidak sepenuhnya GTD tetapi memenuhi persyaratan terkait sinkronisasi dan bersifat lintas platform, serta dapat dengan mudah disesuaikan dengan kebutuhan.

Aplikasi web adalah cara terbaik untuk mendapatkan sinkronisasi lintas platform. (Ini tidak benar-benar sinkron, Anda hanya mengakses db yang sama dari OS yang berbeda).

saya menggunakan Toodledo , yang memiliki aplikasi iphone dan juga memungkinkan Anda menambahkan sidebar dan gadget ke gmail, kalender google, dan halaman igoogle. Aplikasi iphone bahkan berfungsi saat Anda tidak memiliki akses internet.

Jika Anda tidak menyukai ide aplikasi web, saya sarankan Anda untuk melihatnya pedagang lilin yang memiliki klien lintas platform yang dapat disinkronkan melalui internet, atau beberapa mod GTD berbasis tiddlywiki seperti GTDTiddlyWiki plus .

Tiddlywiki pada dasarnya adalah aplikasi yang ditulis dalam html dan javascript, jadi ini terdiri dari satu file html yang Anda bawa, yang Anda perlukan untuk menggunakannya hanyalah browser modern.

VirtualBox adalah preferensi saya; Saya dapat menjalankan VM saya dari OS lain. Virtual PC mengunci Anda ke dalam host Windows.

VMWare Workstation mungkin merupakan fitur paling lengkap, tetapi Anda harus membayar. Jika Anda memutuskan untuk beralih dari menggunakan versi Windows ke versi Linux, Anda harus membayar lagi.

VirtualBox memiliki serangkaian fitur yang solid, gratis, dan berjalan pada berbagai OS dan memvirtualisasikan berbagai OS. Daftar fitur mereka sangat kompetitif dengan VMWare.

VirtualPC belum menawarkan banyak yang saya suka. Anda dapat menggunakannya di Windows untuk menjalankan Windows. Hebat, tetapi Windows sebagai OS Host adalah pilihan kinerja yang cukup buruk. Secara pribadi saya pikir itu hanya untuk mencegah Microsoft memberi tahu pelanggan bahwa mereka harus menggunakan produk nama besar lainnya (seperti VMWare, lihat harga saham mereka jika menurut Anda mereka tidak besar).

VMware Workstation luar biasa - daftar fitur sepanjang satu mil, dan terus ditingkatkan - tetapi membutuhkan uang.

VMware Player dan VMware Server keduanya gratis. Keduanya memiliki batasan: VMware Player tidak dapat membuat VM (meskipun tersedia berbagai solusi) dan tidak mendukung snapshot (yang menurut saya sangat berharga untuk pengujian dan eksperimen). VMware Server hanya dapat dikelola melalui antarmuka web yang bagus untuk server tetapi tidak cocok untuk penggunaan desktop, dan tidak memiliki dukungan untuk beberapa snapshot.

VirtualBox adalah produk yang bagus; dikembangkan secara aktif, serangkaian fitur yang bagus (seperti dukungan USB dan beberapa snapshot), dan tanpa biaya kecuali Anda melakukan sesuatu seperti penerapan di seluruh perusahaan atau divisi.

Saya belum pernah menggunakan Virtual PC; VMware Workstation mungkin adalah produk yang lebih baik, dan VirtualBox gratis dan berjalan di semua host dan dengan tamu mana pun, jadi saya tidak membutuhkan Virtual PC.

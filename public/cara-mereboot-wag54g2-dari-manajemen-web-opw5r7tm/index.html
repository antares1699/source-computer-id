Anda dapat melakukan trik backup / restore:

  1. Pergi ke halaman Backup & Restore di bawah Administrasi:

  2. Cadangkan konfigurasi Anda.

  3. Pulihkan dengan cadangan yang baru saja Anda simpan.

Ini membuat router Anda restart tetapi tidak mengubah apa pun dalam konfigurasi!

Buka browser web Anda dan masukkan alamat ini:

http://your.router.ip/setup.cgi?todo=reboot

Misalnya. jika IP router Anda adalah 192.168.1.1, masukkan yang berikut ini:

http://192.168.1.1/setup.cgi?todo=reboot

misalnya jika ip router Anda adalah 192.168.1.1 maka:

http://192.168.1.1/setup.cgi?next_file=reboot_guage.htm

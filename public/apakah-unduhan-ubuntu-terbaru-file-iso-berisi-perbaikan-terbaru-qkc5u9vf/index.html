Apakah Unduhan Ubuntu Terbaru (file .iso) berisi perbaikan terbaru?

Jika saya pergi ke situs web dan unduh file ISO apakah sama persis dengan yang dirilis pada bulan April atau apakah berisi tambalan terbaru (mis. perbaikan keamanan, dll) yang telah dirilis sejak saat itu?

Jika tidak, saya kira saya harus mengunduh seluruh file ISO, menginstal OS, dan kemudian memiliki beberapa ratus MB lagi untuk diunduh yang bukan itu yang saya inginkan.

Ini hanya berisi sampai titik rilis itu.

Setiap pembaruan tambahan untuk perangkat lunak yang terdapat dalam distribusi akan diperbarui setelah Anda menyelesaikan penginstalan.

Apa yang Anda cari adalah distro yang melakukan rilis bergulir. Lihat pertanyaan ini di unix.stackexchange ini dari tahun 2010 tetapi memindainya, masih cukup relevan.

Saya m tidak 100% yakin akan yang satu ini dan tidak punya waktu untuk mengujinya di VM, namun tebakan saya adalah tidak, tidak ada perbaikan pada gambar.

Mengintegrasikan perbaikan ke dalam .iso akan membutuhkan build nightly yang dikompilasi ke dalam file gambar dan itu sepertinya tidak terlalu efisien ketika Anda bisa mendapatkannya melalui satu pembaruan besar. Selain itu, mengingat beberapa perbaikan bergantung pada perangkat keras / lingkungan, memasukkannya ke dalam gambar tidak akan masuk akal (kecuali jika ada gambar .iso yang dibuat untuk setiap jenis perangkat keras yang dapat dijalankan Ubuntu).

Setelah Anda menginstal, cukup sudo apt-get update dan tunggu sebentar :)

Bagaimana cara mengambil screenshot di windows XP di virtualbox?

saya telah menginstal windows xp di macbook pro menggunakan kotak virtual. Saya perlu mengambil cuplikan layar. Bagaimana cara mengambil ini di windows xp di virtualbox?

Cara standar Anda mengambil tangkapan layar di Windows terpukul MENCETAK , tempel data ke, yaitu Paint dan simpan gambarnya. Jika Anda tidak ingin VirtualBox-Window muncul, Anda dapat menggunakan CTRL + F untuk mengaktifkan mode layar penuh dan mengambil snapshot di sana.

Anda harus terlebih dahulu melepaskan keyboard Anda dari jendela VirtualBox dengan menekan Tombol Host (Ctrl Kanan secara default). Kemudian tekan Alt + Print Screen untuk menangkap jendela saja.

Fitur baru ini tersedia di versi 4.0 dan yang lebih baru, jadi ini tidak akan berfungsi pada edisi perangkat lunak yang lebih lama. Yang perlu Anda lakukan adalah menjalankan perintah berikut:

VBoxManage controlvm <vm name> screenshotpng <filename>.png

Itulah keseluruhan keajaiban, di mana nama mesin virtual Anda dan string apa pun yang ingin Anda gunakan untuk mengidentifikasi tangkapan layar Anda nanti.

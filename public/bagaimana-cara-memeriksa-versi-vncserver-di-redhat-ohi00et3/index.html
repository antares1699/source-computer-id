Bagaimana cara memeriksa versi vncserver di Redhat?

Saya ingin memeriksa versi vncserver mana yang saya gunakan di kotak linux saya. Apa perintah untuk itu?

Saat aku lari

which vncserver

saya mendapat

/usr/bin/vncserver

P.S: Saya menjalankan Red Hat Enterprise Linux Server rilis 5.8 (Tikanga)

Untuk mengetahui dari paket (dan versi) file apa, Anda dapat melakukan rpm -qf /usr/bin/vncserver, lihat manual untuk rpm (1). Tetapi perhatikan bahwa RHEL (seperti halnya semua distribusi enterprisey) memilih versi dasar, dan mempertahankan versi itu apa pun yang mungkin muncul selama seri RHEL 5.x dalam kasus khusus ini. Perangkat lunak dasar ditambal dengan backport untuk fitur-fitur baru (jika relevan), perbaikan bug dan yang lainnya. Jadi jangan heran jika Anda melihat versi yang dilaporkan memenangkan kontes popularitas CVE sepanjang masa, yang Anda jalankan adalah tidak yang itu.

Periksa laporan untuk pembaruan RHEL, saya yakin mereka mengirimkannya ke semua pengguna, tersedia di milis (hanya baca), dan di situs web mereka.

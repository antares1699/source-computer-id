Memasang instans Amazon EC2 di Mac OS X

Saya memiliki otentikasi kunci publik yang berfungsi antara Mac OS X dan file Amazon EC2 Misalnya sehingga dari baris perintah saya bisa mengetik yang berikut ini dan berfungsi:

ssh root@[IPAddressOfEC2Instance]

Hal yang aneh adalah saya tidak dapat memasang instans menggunakan "Hubungkan ke Server" di Finder.

Saya sudah mencoba mengetik alamat server berikut ke dalam dialog "Sambungkan ke Server":

ftps://[IPAddressOfEC2Instance]
ftps://root@[IPAddressOfEC2Instance]

Tapi yang saya dapatkan hanyalah

Anda memasukkan nama pengguna atau kata sandi yang tidak valid. Silahkan coba lagi.

Pengguna root pada instans EC2 memiliki kata sandi kosong dan saya bertanya-tanya apakah ada hubungannya dengan itu. Namun, saya tidak dapat mengubah kata sandi untuk pengguna root.

Saya bisa menggunakan SFTP klien untuk terhubung ke mesin, saya tidak bisa memasangnya dengan "Hubungkan ke server". Ia meminta nama pengguna dan kata sandi (untuk pengguna terdaftar) dan root / [kosong] yang tidak diterima. Opsi lainnya adalah "Tamu" yang menampilkan folder kosong di Finder.

SFTP tidak sama dengan FTPS. FTPS adalah FTP menggunakan SSL. SFTP adalah protokol transfer file yang aman (melalui SSH). (Bagaimanapun juga, dukungan FTP / FTPS Mac OS X di Finder cukup menyedihkan.)

Mac OS X tidak dapat memasang share SFTP secara native. Ada cara untuk melakukannya dengan menggunakan MacFUSE. Agak sulit untuk disiapkan dan tidak ada aplikasi GUI non-komersial yang dapat diandalkan untuk mengelolanya. Ada beberapa paket yang harus Anda instal dan banyak peretasan baris perintah agar saham dapat dipasang (jangan biarkan saya mulai memasangnya secara otomatis). Ini benar-benar tidak sebanding dengan usahanya. Saya telah melakukannya beberapa kali sebelumnya, tetapi itu adalah neraka; Saya telah menemukan solusi yang jauh lebih baik: ExpanDrive .

Ini adalah program komersial tetapi bernilai setiap sen. Ini menggunakan MacFUSE dan memungkinkan Anda untuk memasang "drive" SFTP, FTP / FTPS, dan Amazon S3. Anda dapat mengunduh uji coba gratis yang berfungsi selama 30 hari. (Juga, itu tidak akan kedaluwarsa kecuali Anda menutupnya. :))

Agar dapat berfungsi menggunakan pasangan kunci publik, Anda harus menambahkan kunci pribadi SSH ke keychain Mac OS X Anda:

ssh-add -K

Kemudian saat Anda membuat drive baru di ExpanDrive biarkan bidang sandi kosong, tetapi centang kotak "Simpan".

Anda sebaiknya pergi. Hal yang hebat tentang ExpanDrive adalah itu akan secara otomatis menghubungkan kembali drive jika koneksi terputus (wifi turun, Anda tidur / membangunkan mac Anda, Anda reboot).

sdelete alternatif di osx

Di suite sysinternals untuk windows ada yang bagus ini sdelete alat untuk menghapus file individu dengan aman serta menimpa ruang disk yang tidak terisi dengan keacakan atau nol (opsi -c atau -z).

Saya tahu ada fungsi serupa di OSX di manajer disk tetapi saya ingin menambahkan ini sebagai tugas cron untuk mengetahui bahwa semua file saya yang terhapus benar-benar terhapus. Ini juga merupakan cara yang baik untuk mengoptimalkan penggunaan disk mesin virtual.

Apakah ada alat berbasis baris perintah yang setara dengan sdelete di osx?

secureErase [freespace] level device
Securely erase a disk or freespace on a mounted volume.
Level should be one of the following
1 - Single pass randomly erase the disk.

3 - Gutmann algorithm 35 pass secure erase.  Ownership of the affected disk is required.

man srm srm - hapus file atau direktori dengan aman

http://xahlee.org/mswin/file_shredding.html

Di Mac, cukup gunakan menu “Finder▸Secure Empty Trash”.

Baris perintah yang setara adalah "srm". Seperti ini: 「srm -r -s dir_path」. Itu “-R” berarti semua konten di dir, termasuk sub dir. Kata “-s” artinya adil timpa sekali.

shred (bagian dari inti GNU) dan menghapus .

Kutipan dari manual shred; berlaku untuk semua alat:

PERHATIAN: Perhatikan bahwa penghancuran bergantung pada asumsi yang sangat penting: itu sistem file menimpa data pada tempatnya. Ini cara tradisional untuk melakukan sesuatu, tetapi banyak desain sistem file modern tidak memenuhi ini anggapan.

Untuk menghapus ruang kosong, isi sistem file seperti ini:

# repeat 3 times
for x in {1..3}; do
    dd if=/dev/urandom of=/tmp/foo bs=8M
    rm /tmp/foo
done

Jika Anda menginstal MacPorts, Anda dapat mengkompilasi bcwipe.

080938 port info bcwipe
bcwipe @1.7-7 (sysutils, security)
Variants:             universal

Description:          The BCWipe software is intended to give you a confidence
                      that your deleted files cannot be recovered by an intruder.
                      BCWipe repeatedly overwrites special patterns to the files
                      to be destroyed. BCWipe for UNIX offers two wiping schemas:
                      US DoD 5200.28-STD standard and Peter Gutmann's 35 pass
                      scheme.
Homepage:             http://www.jetico.com/

Platforms:            darwin
License:              unknown
Maintainers:          nomaintainer@macports.org

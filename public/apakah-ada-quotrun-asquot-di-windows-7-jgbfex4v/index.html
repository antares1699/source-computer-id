apakah ada "run as" di Windows 7?

Saya memiliki masalah dengan pengguna tanpa admin lokal yang ingin menginstal font di PC mereka, hubungi saya setiap hari. Masalahnya adalah tidak ada run as, seperti di windows XP. Saya tidak ingin keluar dari sesi pengguna dan masuk ke akun admin lokal untuk menginstal font, mereka akan kehilangan produktivitas dan kehilangan waktu saya untuk melakukannya.

Anda dapat mengklik kanan pada aplikasi dan 'Run As Administrator' memberikan kredensial admin atau jika Anda menahan tombol Shift dan klik kanan Anda dapat 'Run As A Different User' dan memasukkan kredensial pengguna dengan hak admin.

Anda juga dapat menyetel item menjadi "Jalankan sebagai Administrator" secara permanen untuk satu akun pengguna atau untuk semua pengguna berdasarkan tautan gambar di bawah.

Klik Kanan -> Properties Pilih tab "Kompatibilitas" dan pilih "Jalankan sebagai Administrator" Klik "Ubah pengaturan untuk semua pengguna" untuk menerapkan pengaturan untuk semua pengguna yang menjalankan item tertentu.

Klik

Dan jika tidak satu pun dari mereka yang berhasil, cobalah yang berikut ini, ini akan sedikit mengguncang: http://tompopov.blogspot.com/2011/05/allowing-non-admin-users-to-install.html

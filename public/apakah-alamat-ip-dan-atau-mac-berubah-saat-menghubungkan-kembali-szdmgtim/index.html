Apakah alamat IP dan / atau MAC berubah saat menghubungkan kembali

Saya memiliki router TP-Link di rumah. Apakah alamat IP dan / atau alamat Mac berubah setiap kali saya menyambungkannya kembali? Artinya ketika saya pergi dan kembali apakah mereka akan berubah? Atau bisakah mereka?

  • Alamat MAC umumnya tidak pernah berubah (meskipun terkadang dapat diubah secara manual, tetapi seharusnya unik dan statis secara global).

  • Alamat IP yang diterima komputer Anda dari router TP-Link mungkin akan tetap, karena ini adalah alamat IP pribadi (misalnya, 192.168.1.42), yang ditetapkan oleh router ke komputer (misalkan Anda menggunakan DHCP untuk mendapatkan Alamat IP, tetapi seperti yang Anda tanyakan, itu mungkin berarti Anda tidak menyetel alamat IP secara manual ...). Karena alamat MAC PC tidak berubah, router umumnya akan tetap memberikan alamat yang sama (tetapi ini tidak terjamin , itu hanya khas perilaku router rumah).

  • Alamat IP "eksternal" yang diterima router Anda dari penyedia layanan Internet kemungkinan akan berubah secara berkala (mis., Setiap hari atau setiap beberapa hari), dan mungkin juga akan berubah setiap kali Anda memulai ulang router.

Alamat MAC adalah nilai tetap. Apakah tidak berubah.
Jika mau, Anda dapat membandingkannya dengan nomor seri untuk kartu jaringan.

IP di sisi lain bisa perubahan. Itu tidak harus berubah, tetapi tergantung pada pengaturan server DHCP Anda, hal berikut ini akan terjadi:

  1. Anda mengonfigurasi IP statis. IP tergantung pada pengaturan di komputer Anda. Server DHCP diabaikan.
  2. Anda menggunakan DHCP. Anda mendapatkan alamat gratis dari kumpulan alamat IP.
    • Jika Anda pergi dan kembali beberapa minggu kemudian alamat IP akan dirilis dan Anda akan mendapatkan yang lain. (Jadi itu akan berubah).
    • Jika Anda kembali sebelum waktu sewa DHCP habis, komputer Anda akan meminta IP dan IP lamanya akan diperbarui. (agar tidak berubah).

Jika mau, Anda dapat menggunakan server DHCP dan memesan alamat IP untuk MAC tertentu. Dengan begitu komputer dengan MAC tersebut akan selalu mendapatkan IP yang sama dari server DHCP.

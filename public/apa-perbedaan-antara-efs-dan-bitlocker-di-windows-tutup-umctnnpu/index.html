Saya menemukan jawabannya dalam hal ini Artikel Microsoft :

Ada beberapa perbedaan antara BitLocker Drive Encryption dan Sistem File Enkripsi. BitLocker dirancang untuk membantu melindungi semua dari file pribadi dan sistem pada drive tempat Windows diinstal jika komputer Anda dicuri, atau jika pengguna yang tidak sah mencoba mengakses komputer. EFS digunakan untuk membantu melindungi file individu di drive mana pun basis per pengguna. Tabel di bawah ini menunjukkan perbedaan utama antara Enkripsi Drive BitLocker dan EFS.

BitLocker vs. Encrypting File System (EFS)

  • BitLocker mengenkripsi semua file pribadi dan sistem di drive tempat Windows diinstal, atau di drive data di komputer yang sama.
  • EFS mengenkripsi file individu di drive mana pun.
  • BitLocker tidak bergantung pada akun pengguna individu yang terkait dengan file. BitLocker aktif atau nonaktif, untuk semua pengguna atau grup.
  • EFS mengenkripsi file berdasarkan akun pengguna yang terkait dengannya. Jika komputer memiliki banyak pengguna atau grup, masing-masing dapat mengenkripsi miliknya sendiri file secara mandiri.
  • BitLocker menggunakan Trusted Platform Module (TPM), microchip khusus di beberapa komputer baru yang mendukung fitur keamanan tingkat lanjut.
  • EFS tidak memerlukan atau menggunakan perangkat keras khusus apa pun.
  • Anda harus menjadi administrator untuk mengaktifkan atau menonaktifkan enkripsi BitLocker setelah diaktifkan.
  • Anda tidak harus menjadi administrator untuk menggunakan EFS.
  • Anda dapat menggunakan BitLocker Drive Encryption dan Encrypting File System bersama-sama untuk mendapatkan perlindungan yang ditawarkan oleh kedua fitur tersebut. Kapan menggunakan EFS, kunci enkripsi disimpan dengan operasi komputer sistem. Meskipun ini dienkripsi, tingkat keamanan itu bisa berpotensi disusupi jika peretas dapat mem-boot atau mengakses file penggerak sistem. Menggunakan BitLocker untuk mengenkripsi drive sistem dapat membantu lindungi kunci-kunci ini dengan mencegah drive sistem dari boot atau sedang diakses jika dipasang ke komputer lain.

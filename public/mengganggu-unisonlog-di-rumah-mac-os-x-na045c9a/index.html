Mengganggu unison.log di rumah Mac OS X.

GUI serentak untuk OS X terus membuat file "unison.log" di direktori home saya.

Itu terbelakang, itu harus menyembunyikan file (".unison.log"), atau menempatkannya di tempat yang tepat, bukan di direktori home saya.

Apakah Anda tahu cara mencegahnya? (tanpa harus meretas sumber dan mengkompilasi ulang)

Saya hanya dapat menemukan daemon yang memeriksa file itu dan menghapusnya, tetapi mungkin ada cara yang lebih baik ...

Lokasi file log dapat disetel di file profil Anda (biasanya ~/.unison/something.prf tetapi karena Anda menggunakan OS X akan menjadi ~/Library/Application Support/Unison/) dengan menambahkan baris logfile = /path/to/log/file. Ini juga dapat diganti pada baris perintah dengan opsi -logfile.

Secara umum, jika sebuah program tidak berfungsi seperti yang Anda inginkan, saya sarankan untuk mencari opsi konfigurasi sebelum memikirkan solusi yang rumit atau memodifikasi kode sumber. Dokumentasinya adalah teman Anda. Dalam kasus ini, mencari "unison.log" dalam dokumentasi akan membawa Anda langsung ke preferensi logfile.

Cara membuka folder iPhoto Library di Mac

Saya ingat bahwa belum lama ini dimungkinkan untuk membuka Finder dan menavigasi ke Pictures, lalu ke folder iPhoto Library dan kemudian masuk ke dalam folder dan membuka subfolder individual dengan jpg di dalamnya.

Tampaknya tidak menjadi masalah lagi - saya melihat iPhoto Library dari jenis "iPhoto Library" (bukan "Folder") dan membukanya secara otomatis membuka iPhoto.

Ini menimbulkan masalah saat mencoba mencari file, misalnya untuk melampirkannya ke email. Program email hanya melihat Perpustakaan iPhoto sebagai paket dan bukan sebagai folder.

Tolong bantu saya memahami, apakah ini pengaturan di Mac saya, atau perubahan di iPhoto? Bagaimana cara menemukan file gambar, selain mencari melalui Finder?

Saya memiliki iPhoto'09 versi 8.1.2.

Terima kasih.

Tidak seperti versi sebelumnya iPhoto'08 iPhoto'09 menggunakan pustaka gaya paket.

Anda dapat mengklik kanan "iPhoto Library" Anda (setidaknya di Finder) dan tekan "Show Package Contents" dan itu akan terbuka seperti folder. Gambar asli ada di folder bernama asli dan gambar yang sudah diedit (di iPhoto) ada di folder 'diubah'. Saya tidak yakin tentang folder 'Data' (alias Data.noindex), itu mungkin hanya cadangan. Untuk memeriksa ulang Anda juga dapat mengklik kanan pada gambar di iPhoto dan memilih 'Show File' dan itu akan membawa Anda ke lokasi sumber.

Anda dapat membuat alias ke folder yang dimodifikasi, data, dan asli untuk mempermudah akses.

EMAILING HAL-HAL: Jika Anda memiliki dialog "buka file" (untuk memilih file yang akan diunggah, dll.) Di sebelah kiri juga terdapat bagian media yang memudahkan untuk mengakses foto iPhoto. Cukup mudah sebenarnya, tidak perlu menavigasi melalui sebuah paket.

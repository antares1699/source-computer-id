Bagaimana cara memformat Mac saya dan menginstal Snow Leopard dari awal?

Saya ingin menghapus instalasi Leopard saya saat ini dan menginstal Snow Leopard di mac saya. Apa cara yang benar untuk mencapai ini?

Sepertinya saya perlu boot ke beberapa utilitas disk untuk memformatnya, tetapi saya tidak begitu yakin bagaimana melakukannya.

Larutan

  • Masukkan DVD Snow Leopard.
  • Hidupkan Kembali komputer Anda.
  • Saat komputer Anda memulai ulang, tahan tombol alt / opsi. Ini kemudian akan menanyakan Anda drive mana yang ingin Anda boot. Pilih drive DVD Anda.
  • Dari sana Anda dapat memilih Disk Utlity dari menu Utilities dan memformat drive, atau memilih opsi "hapus dan instal".

Boot dari Mac OS X Snow Leopard Install DVD, dan dari menu "Utilities", pilih "Disk utility". Dari sana Anda dapat memformat ulang hard disk Anda. Setelah melakukannya, keluar dari Disk Utility dan Anda dapat melakukan penginstalan bersih.

EDIT : Rupanya ada juga opsi "Hapus dan Instal". (Saya belum pernah menggunakan ini sebelumnya). Lihat detail selengkapnya di sini .

Ini artikel dukungan apel resmi tentang semua opsi penginstalan yang tersedia.

Terpikir oleh saya setelah memposting ini bahwa, sementara pertanyaan Anda dengan jelas menyatakan Anda ingin memformat disk Anda, saya ingin menyebutkannya ini akan menghapus semua data di disk Anda, termasuk aplikasi dan data Anda . Begitu buat backup dulu . Atau, Anda bisa melakukan "Arsipkan dan Instal" yang akan memberi Anda OS yang bersih, tetapi biarkan data dan aplikasi Anda tetap rapi.

Bagaimana cara memeriksa apakah direktori berada di bawah kendali versi svn?

Dari skrip bash linux, bagaimana saya dapat memverifikasi jika beberapa direktori berada di bawah kendali versi svn? Saya menggunakan svn versi 1.7.9.

Ketika saya mencoba yang berikut ini

pushd ${WD}; svn info; popd

dan periksa kode kesalahan saya selalu mendapatkan nol, karena popd berhasil, bahkan jika svn info mengembalikan kesalahan. Bagaimana cara melakukan pemeriksaan pada direktori dengan cara berikut?

if [ ... ]; then
    echo "Directory ${WD} not under version control"
fi

Simpan kode pengembalian dari perintah svn dalam variabel dan uji bahwa-:

pushd $PWD
svn info
ERR=$?
popd 
if [ $ERR -ne 0 ];then
    echo "Directory ${WD} not under version control"
fi

Subversion menggunakan direktori tersembunyi bernama .svn (seperti .git untuk Git).

Pertama, direktori tidak di bawah kendali SVN:

riemann$ if [ ! -d ".svn" ]; then echo "Directory not under version control"; fi;
Directory not under version control
riemann$

Selanjutnya, direktori yang berada di bawah kendali SVN ( Crypto ++ menggunakan SVN ):

riemann::cryptopp$ if [ ! -d ".svn" ]; then echo "Directory not under version control"; fi;
riemann::cryptopp$

Dan:

riemann::cryptopp$ ls -al .*

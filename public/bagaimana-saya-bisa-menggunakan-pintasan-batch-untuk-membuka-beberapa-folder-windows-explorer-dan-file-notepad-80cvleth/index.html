Komputer

Bagaimana saya bisa menggunakan pintasan batch untuk membuka beberapa folder Windows Explorer, dan file Notepad?

Saya baru-baru ini membangun komputer dengan SSD di dalamnya (pertama saya) dan sedang dalam proses memindahkan beberapa game Steam saya ke SSD dan secara simbolis menautkannya ke tempat mereka seharusnya berada di utama (mekanis ) harddisk sehingga Steam tidak bugar dan saya mendapatkan kinerja yang maksimal. Untuk tujuan ini, saya mendapati diri saya berulang kali menavigasi ke kedua folder target untuk memindahkan file yang relevan dan membuka file Notepad tertentu untuk memasukkan jalur yang relevan ke dalam template yang telah saya siapkan yang saya tempel ke prompt perintah Windows untuk membuat symlink tersebut. Ini menjengkelkan.

Saya telah membaca sedikit tentang pintasan batch, dan memiliki pemahaman dasar tentang cara menggunakannya untuk membuka program dan file. Namun, saya menemukan banyak saran yang membingungkan tentang cara menggunakan pintasan batch untuk membuka folder tertentu di Windows Explorer, dan tidak ada yang berurusan dengan membuka banyak folder sekaligus. Bantuan apa pun dalam menulis pintasan akan sangat dihargai. Terima kasih atas waktu Anda.

Start.exe adalah temanmu. :)

Memulai jendela terpisah untuk menjalankan program atau perintah tertentu.

Coba sesuatu seperti ini dalam file batch:

@echo off
start explorer c:\
start explorer c:\Windows
start explorer "g:\Path With Spaces"
start notepad.exe g:\filename.txt

DNS berbeda berdasarkan pengguna?

Saya memiliki OpenDNS dengan kontrol akses yang diatur untuk laptop saya. Saya ingin menonaktifkan filter untuk pengguna tertentu, tetapi tampaknya tidak ada cara sederhana untuk melakukan ini.

Pikiran pertama saya adalah mengaktifkan server openDNS dns untuk satu pengguna dan menonaktifkannya untuk yang lain. Ada pemikiran tentang cara terbaik untuk mencapai ini?

OpenDNS nyaman untuk tujuan ini, tetapi tidak ada sistem DNS yang benar-benar dirancang untuk pemfilteran. Orang yang bertekad dan berpengetahuan masih bisa masuk ke "situs buruk" jika mereka benar-benar ingin dengan memasukkan alamat IP. Jika Anda berbicara tentang Windows, Windows secara native tidak mendukung konfigurasi jaringan "per pengguna". Anda mungkin dapat membuat beberapa file *.cmd khusus pengguna yang menggunakan perintah netsh untuk mengkonfigurasi ulang host DNS dengan cepat.

Tapi, Anda mungkin benar-benar perlu menjalankan melalui server DNS jaringan lokal kadang-kadang untuk menyelesaikan host di LAN mereka, seperti di tempat kerja. Menggunakan DNS untuk pemfilteran bukanlah solusi yang saya pilih.

Saya sarankan untuk memecahkan masalah Anda dengan menggunakan proxy HTTP lokal atau program lain yang dirancang untuk pemfilteran web. Jika Anda menjalankan Windows dan menginginkan sesuatu yang "cepat dan mudah" untuk melindungi anak-anak saat online, saya telah menggunakan Perlindungan Web K9 untuk sementara waktu dan sangat senang dengan itu.

Letakkan kata sandi BIOS pada laptop Anda dan nonaktifkan boot CD / USB sehingga mereka tidak dapat mem-boot CD / USB langsung dan melewati semua perlindungan Anda.

Jika Anda hanya ingin memindai mesin windows Anda dari virus, maka Anda dapat menggunakan salah satu dari banyak ISO penyelamat antivirus yang dapat di-boot. Ini dapat digunakan pada USB juga termasuk kemampuan untuk memperbarui (Adaptor Nirkabel yang kompatibel Melalui Ethernet atau Linux)

Kamu bisa gunakan:

  • Sistem Penyelamatan Antivir Avira
  • Penyelamatan BitDefender
  • Disk Penyelamatan Kaspersky
  • Panda SafeCD

Itu hanya sebagian dari banyak.

Jika Anda bersikeras untuk menginstal antivirus pada puppylinux maka Anda perlu memastikan bahwa Anda memiliki image bootable USB yang persisten dan kemudian instal salah satu dari yang berikut ini:

  • Antivirus F-Prot
  • ClamAV
  • AVG

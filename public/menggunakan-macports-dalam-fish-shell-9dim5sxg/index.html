Menggunakan MacPorts dalam FISH shell

Saya telah bermain-main dengan FISH selama beberapa hari dan saya hanya mencoba untuk mengubahnya ke shell utama saya. Itu berfungsi dengan baik tetapi sekarang tidak ada perangkat lunak saya dari MacPorts yang dikenali.

Saya menginstal dan mengikuti saran dari http://ridiculousfish.com/shell/faq.html , menggunakan formulir penginstal paket di sini http://ridiculousfish.com/shell/ .

Skrip python membuat entri ini dari .profil saya ke file config.fish saya

set_default export PATH "/opt/local/bin:/opt/local/sbin:$PATH"

tapi saya mendapatkan kesalahan ini

fish: Perintah tidak dikenal 'set_default'
/Users/ley/.config/fish/config.fish (baris 1): set_default export PATH "/ opt / local / bin: / opt / local / sbin: $ PATH" ^ di . (sumber) panggilan file '/Users/ley/.config/fish/config.fish', meminta masukan standar,

Saya mengotak-atik perintah set sedikit tetapi itu sepertinya tidak banyak membantu.

Ada saran?

Sepertinya Anda menginginkan:

set -xg PATH /opt/local/bin /opt/local/sbin $PATH

Edit: Saya mengerti maksud Anda, sintaks ini membutuhkan waktu untuk membiasakan diri. Ia menginginkan spasi, bukan titik dua, dan tidak boleh dalam tanda kutip, dan semua jalur di PATH Anda saat ini harus benar-benar ada (dalam kasus saya, saya memiliki /usr/local/sbin di PATH saya meskipun sebenarnya tidak ada, dan itu masalah untuk fish).

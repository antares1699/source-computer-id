Lihat terakhir kali saya login di Linux?

Saya ingin melihat login saya sebelumnya ketika saya login menggunakan baris perintah di Ubuntu 10 dan saya bertanya-tanya apakah ada perintah untuk melakukannya?

Perintah harus menunjukkan waktu / tanggal terakhir saya masuk.

4
Brock Woolf — Sumber

2 Jawaban

9
9

Perintah last menunjukkan semua login sebelumnya dan aktif. last $USER membatasi pengguna Anda. last $USER | head -n 3 menunjukkan 3 terbaru.

Daftar login sebelumnya diinisialisasi ulang setiap bulan secara default, dan salinan bulan sebelumnya disimpan (Anda dapat menginterogasinya dengan last -f /var/log/wtmp.1). Ini dikendalikan pada /etc/logrotate.conf.

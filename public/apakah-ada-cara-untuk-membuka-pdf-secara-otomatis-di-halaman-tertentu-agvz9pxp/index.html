Apakah ada cara untuk membuka pdf secara otomatis di halaman tertentu?

Saya membaca buku pdf dan ingin bertanya apakah ada cara untuk membukanya secara otomatis di halaman tertentu (agar tidak harus menavigasi ke halaman terakhir yang saya baca setiap saat).

Terima kasih.

Catatan: Saya menggunakan adobe reader X, tetapi jika ada solusi yang lebih baik di program gratis lainnya, harap katakan.

(Tangkapan layar dari Adobe Reader 9)

Anda dapat mengatur Adobe Reader untuk selalu mengembalikan tampilan terakhir, termasuk halaman terakhir, saat membuka dokumen:

  1. Buka jendela preferensi (Edit => Preferences).

  2. Pergi ke kategori Dokumen di sebelah kiri.

  3. Centang Restore last view settings when reopening documents.

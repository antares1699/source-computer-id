Bagaimana Anda bisa membuat CD dengan media yang tidak dapat disalin? [Tutup]

Ada beberapa tutorial yang tersedia di CD / DVD hari ini. Ketika saya mencoba menyalin konten di hard disk saya, saya tidak dapat menyalinnya. CD bekerja dengan benar jika saya mencoba menggunakan opsi AutoPlay dari CD. Yang perlu saya pahami adalah, bagaimana caranya membuat CD yang bisa berisi media, tapi tidak bisa direplikasi atau diputar dengan player lain kecuali yang tersedia di CD? Apakah ada perangkat lunak seperti itu yang tersedia untuk membuat media semacam itu?

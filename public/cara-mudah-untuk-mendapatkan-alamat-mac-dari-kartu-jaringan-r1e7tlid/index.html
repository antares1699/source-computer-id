Cara mudah untuk mendapatkan alamat mac dari kartu jaringan

Saya menggunakan banyak skrip yang melibatkan alamat Mac dari kartu jaringan saya. Apakah ada file yang bisa saya baca atau perintah yang bisa saya gunakan untuk HANYA mendapatkan alamat Mac? Saya tidak ingin ada tajuk atau apa pun, hanya 1 baris keluaran yang memiliki alamat Mac. Saya ingin menetapkan ini sebagai variabel lingkungan saat login. Saya menggunakan Arch Linux x64

4
Wolfizen — Sumber
  • Perintah apa pun yang biasanya Anda gunakan untuk melihat alamat MAC, masukkan outputnya melalui awk (atau perl, atau ...) dan tarik bit yang Anda inginkan.
    — alexis
  • Ketahuilah bahwa alamat MAC bukanlah nilai statis yang tidak dapat diubah seperti dulu. Hampir di mana-mana itu dapat dikonfigurasi pengguna / diubah.
    — mdpc

11
 VARIABLE="$(cat /sys/class/net/$IFACE/address)"

Sulit untuk melakukannya dengan lebih sederhana.

Sebenarnya:

 read -r MACADDRESS < /sys/class/net/$iFACE/address

bahkan lebih sederhana, dan tidak memulai proses tambahan.

Ini berhasil untuk saya. Mungkin bisa sedikit dioptimalkan:

/sbin/ifconfig eth0 | grep -o 'HWaddr  *[0-9A-F:]*' | sed -s 's/HWaddr *//'

Saya tidak berpikir Anda dapat menyederhanakannya banyak dari ini. Saya melakukan strace di ifconfig, dan tampaknya Anda menarik MAC addr dengan ioctl () pada soket, bukan dari file / proc.

Anda memiliki gambar di Inkscape yang ukurannya lebih besar dari 8,5 "x 11." Anda ingin mencetak dua bagian gambar menjadi dua lembar. Berikut adalah metode yang saya suka gunakan terutama untuk membuat transparansi sablon.

Pro dari metode ini:

  • Tidak memerlukan perangkat lunak tambahan
  • Untuk mencetak sablon positif ke transparansi, Anda dapat membuat bagian-bagian gambar tumpang tindih, memungkinkan penyelarasan kedua bagian lebih mudah dan lebih tepat.

Kontra dari metode ini:

  • Meskipun bukan tidak mungkin, mungkin sulit untuk mencetak dua tepat separuh - artinya tidak ada bagian gambar yang berulang di kedua halaman, serta tidak ada bagian yang hilang. Jadi, jika Anda mencetak di atas kertas putih, Anda mungkin harus mencetak sedikit lebih dari setengah gambar dan kemudian memotong bagian dari satu halaman menggunakan gunting untuk menyejajarkan kedua bagian gambar dengan benar. Jika Anda bersedia meluangkan waktu untuk mengaturnya dengan hati-hati di Inkscape, Anda dapat mencapai dua bagian yang hampir sempurna.

Untuk memulai, buka gambar Anda. Ukurlah, dengan menggunakan inci, ke dimensi yang Anda inginkan.

Buka "File"> Properti Dokumen dan set Ukuran halaman ke US Letter (yaitu 8,5 "x 11"). Jika gambar Anda panjang, Anda dapat memilih Orientation: Landscape. Jika tidak, itu harus disetel ke Potret.

Di dalam kotak dialog ini, Anda juga akan melihat bagian berjudul Perbatasan. Di sini, pastikan "tampilkan batas halaman" dan "batas di atas gambar" dipilih. Jika ya, Anda akan melihat persegi panjang muncul di atas gambar Anda. Saya akan menyebut persegi panjang ini sebagai Halaman .

Pastikan file Anda disimpan sebagai SVG (mungkin jenis file lain akan berfungsi - saya tidak yakin - tetapi saya tahu bahwa PDF tidak akan berfungsi).

Menggunakan Pilih dan Transformasi Alat untuk memindahkan gambar Anda. Karena Anda ingin mencetak dua bagian gambar, atur satu setengah gambar di atas Halaman . Saat Anda pergi ke File> Print, separuh yang berada di dalam Halaman akan dicetak. Separuh gambar lainnya tidak akan.

Terakhir, gunakan Select and Transform Tool lagi untuk memindahkan gambar Anda sehingga paruh kedua gambar berada di dalam Halaman. File> Print, dan voila. Anda harus memiliki dua bagian dari satu gambar besar.

Solusi sederhana dan cepat:

  1. Buat seluruh Inkscape Anda menggambar satu Grup, jika terdiri dari lebih dari satu objek.
  2. Edit ukuran halaman Anda seperti yang disebutkan di atas.
  3. Pindahkan grup sehingga bagian dari grup Anda sesuai dengan batas halaman.
  4. Cetak ke Penulis Dokumen Microsoft XPS ... Saya hanya menyimpan file ke folder yang sama dengan gambar Inkscape saya sebagai 1.
  5. Pindahkan grup sehingga bagian gambar selanjutnya sesuai dengan batas halaman dan cetak ke XPS lagi, simpan sebagai 2.
  6. Ulangi sebanyak yang diperlukan.
  7. Buka dan cetak setiap File XPS

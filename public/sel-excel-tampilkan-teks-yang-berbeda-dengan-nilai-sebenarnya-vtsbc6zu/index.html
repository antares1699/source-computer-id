Sel Excel - tampilkan teks yang berbeda dengan nilai sebenarnya?

Apakah mungkin di Excel untuk memiliki teks tampilan sel yang berbeda dari nilainya?

Saya ingin memiliki sel yang memiliki nilai 1,2,3 ... atau 7 tetapi akan menampilkan hari dalam seminggu sebagai teks - tetapi ketika saya menggunakan nilai sel dalam rumus, itu harus menggunakan angka!

Ada saran?

Anda memang dapat menggunakan pemformatan bersyarat, cukup beresonansi:

Menerapkan format bersyarat ke satu sel sampel (menggunakan sel A1 sebagai contoh)
Buat satu set 7 kondisi format,

Condition formula       Format (Number: Custom)
=A1=1                   "Sunday"
=A1=2                   "Monday"
=A1=3                   "Tuesday"
=A1=4                   "Wednesday"
=A1=5                   "Thursday"
=A1=6                   "Friday"
=A1=7                   "Saturday"

Kemudian salin tempel format khusus (atau gunakan pelukis format) untuk diterapkan ke sel lain sesuai kebutuhan

Jika nilai sel Anda akan tetap statis, maka Anda dapat menerapkan format angka kustom satu per satu untuk setiap sel. Misalnya, pilih sel Anda yang berisi 1 dan terapkan format angka kustom "Senin" (atau "Minggu", bergantung pada preferensi Anda). Begitu pula untuk sel dengan 2, format kustomnya sendiri, dan seterusnya.

Jika nilai sel Anda akan dinamis (dalam kisaran 1..7) maka Anda mungkin dapat menggunakan pemformatan bersyarat tetapi itu akan menjadi canggung.

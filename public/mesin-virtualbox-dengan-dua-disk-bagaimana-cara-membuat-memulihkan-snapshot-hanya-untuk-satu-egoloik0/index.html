Mesin VirtualBox dengan dua disk - Bagaimana cara membuat / memulihkan snapshot hanya untuk satu?

Saya memiliki mesin virtual Linux dengan dua disk, satu dipasang sebagai / dan yang lainnya sebagai / home. Saya ingin mengambil cuplikan dari / tetapi bukan dari / rumah sehingga saya nanti dapat kembali ke sistem yang bersih dan menyimpan data saya.

Apakah ada cara untuk mencapai ini?

Setelah lebih banyak mencari di forum kotak virtual, saya menemukan bahwa ada jenis hard drive virtual yang tidak terpengaruh oleh snapshot: writethrough.

Untuk mengonversi disk, pertama-tama saya harus melepaskan disk pembeda yang terpasang padanya dan kemudian menjalankannya di baris perintah:

VBoxManage.exe modifhd <path to vdi file> --type writethrough

Setelah ini, semuanya tampak berfungsi dengan baik.

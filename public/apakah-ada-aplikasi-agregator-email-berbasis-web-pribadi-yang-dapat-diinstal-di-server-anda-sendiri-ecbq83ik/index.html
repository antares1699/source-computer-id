Apakah ada aplikasi agregator email berbasis web pribadi yang dapat diinstal di server Anda sendiri?

Sepertinya ada aliran masuk situs web agregasi email yang bekerja mirip dengan GMail dengan cara mereka dapat mengumpulkan email yang mirip dengan aplikasi pembaca email Desktop tradisional tetapi semuanya dilakukan di web (Maaf, saya menolak untuk menggunakan cloud kata kunci di sini).

Layanan ini sangat bagus terutama bagi mereka yang tidak terlalu peduli dengan privasi dan yang tidak memiliki keterampilan teknis untuk mengelola server rumah. Bagi sebagian besar dari kita tampaknya harus ada produk di sekitar yang pada dasarnya menawarkan kemampuan tetapi dapat diinstal di server Anda sendiri sehingga server Anda bertindak seperti pembaca email berbasis web dan agar semua data yang dikumpulkan ditarik. server publik secepat mungkin dan hanya disimpan di server pribadi Anda.

Saya telah memperhatikan produk seperti itu tetapi saya belum menemukan sesuatu yang masuk akal. Dugaan saya adalah bahwa produk semacam itu akan sempurna untuk pengguna biasa yang mungkin pernah membeli Windows Home Server atau untuk pengguna yang lebih teknis yang umumnya memiliki server lengkap atau desktop yang dapat ditetapkan di rumah sebagai server untuk menjalankan perangkat lunak tersebut. Saya tertarik dengan solusi apa pun yang mungkin Anda ketahui terlepas dari OS atau teknologinya, tetapi saya lebih suka jika solusinya memiliki tampilan berbasis web yang bagus dan juga kemampuan untuk mengumpulkan email dari beberapa sumber non-pop3.

Server email andal yang menyertakan kemampuan ini adalah David Harris's Mercury / 32 (untuk sistem Win 32-bit). David adalah pelopor dalam email internet dan alatnya sangat canggih. Merkuri sudah ada sejak 1993. Diantara fiturnya adalah:

Mendistribusikan fitur Klien POP3

Dapat dikonfigurasi untuk mengunduh email dari jarak jauh yang tidak terbatas Kotak surat POP3 dan untuk memprosesnya mengirimkan ke sejumlah lokal kotak surat (jadi, Anda dapat memiliki empat kotak surat jarak jauh semua diunduh ke satu pengguna lokal, atau setiap kotak surat jarak jauh sedang diunduh ke pengguna terpisah).

Ini adalah server email lengkap (mendukung POP, SMTP, IMAP, milis, dll.) Sehingga Anda dapat meletakkan front-end email berbasis web atau klien email yang Anda suka. Ada juga komunitas kuat yang dapat Anda hubungi untuk mendapatkan dukungan - David juga sangat responsif dalam menjawab pertanyaan.

Anda dapat mempelajarinya lebih lanjut di (dan mengunduhnya dari) http://www.pmail.com/overviews/ovw_mercwin.htm

Zimbra adalah sumber terbuka dan mendukung akun surat eksternal. Ini berbasis web dan mendukung perangkat seluler.

Juga bukan server, tetapi beberapa klien email mendukung pengambilan email dari banyak akun. Anda memiliki dua opsi dasar. 1) Konfigurasikan setiap akun untuk meninggalkan email di layanan jarak jauh atau 2) tarik semua email ke server lokal. Masing masing punya kelebihan dan kekurangan. Mempertahankan lokasinya saat ini memungkinkan Anda mengubah klien dan meminimalkan dampak kehilangan mailstore. Menarik semuanya memungkinkan Anda menyimpan satu repositori; Namun, semua email Anda ada dalam satu keranjang.

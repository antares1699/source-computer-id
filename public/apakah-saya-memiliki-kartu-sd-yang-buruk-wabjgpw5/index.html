Apakah saya memiliki kartu SD yang buruk?

Saya mencoba menyalin data dari komputer saya ke kartu SD. Setelah beberapa ratus MB, saya terus mendapatkan kesalahan berikut di dmesg:

[34542.836192] end_request: I/O error, dev mmcblk0, sector 855936
[34542.836284] FAT: unable to read inode block for updating (i_pos 13694981)
[34542.836306] MMC: killing requests for dead queue
[34542.836310] end_request: I/O error, dev mmcblk0, sector 9280
[34542.837035] FAT: unable to read inode block for updating (i_pos 148486)
[34542.837062] MMC: killing requests for dead queue
[34542.837066] end_request: I/O error, dev mmcblk0, sector 1
[34542.837074] FAT: bread failed in fat_clusters_flush
[34542.837085] MMC: killing requests for dead queue

Ini semua file yang saya salin dari kartu SD yang lebih kecil. Saya hanya ingin mentransfernya ke kartu baru saya yang lebih besar untuk ponsel saya. Saya mencoba percobaan yang sama dengan file yang berbeda di mesin yang berbeda dan kartu gagal lagi. Membaca data dari kartu lama berjalan lancar. Sistem saya lebih lama dan kartu SD baru (16GB Kelas 4). Mungkinkah komputer saya terlalu tua? Apakah ada tes pasti untuk memverifikasi apakah kartu SD saya buruk?

Karena saya memiliki situasi yang sama dengan kartu SD 1GB sekali. Jika Anda mengalami masalah dalam membaca / menulis kartu SD yang lebih besar (seperti kesalahan penulisan atau Anda tidak dapat memperoleh data Anda kembali), pastikan bahwa pembaca kartu Anda (jika eksternal) mendukung ukuran kartu SD itu dan / atau jika firmware pembaruan tersedia untuk itu untuk mendukung kartu yang lebih besar. Ada beberapa kemungkinan jebakan saat menerapkan dukungan untuk kartu yang lebih besar dan beberapa pembaca kartu lama tidak dapat membacanya. Jika Anda memiliki kartu lain dengan ukuran yang sama yang berfungsi dengan baik, menurut saya itu bukan pembaca Anda.

Alat seperti F3 atau h2testw di Windows bagus untuk menguji kartu, tetapi mereka tidak dapat mendeteksi apakah itu kartu atau pembaca.

Jika ponsel Anda mendukung ukuran itu, Anda dapat memeriksa apakah ponsel dapat mengekspor perangkat mentah (sebagai perangkat penyimpanan USB) saat menghubungkannya ke PC Anda melalui kabel USB. Kemudian jalankan tes melalui ponsel Anda untuk menyingkirkan pembaca kartu yang buruk.

Bagaimana cara membuat posisi baris status Vim di OSX sama dengan Ubuntu?

Di kotak OSX saya, saya ingin melihat detail kursor yang sama yang saya lihat di sudut kanan vim Ubuntu saya.

Di kotak ubuntu saya, baris status ini (mungkin secara teknis bukan "baris status vim", saya tidak tahu) ditampilkan secara default, dan berada di baris yang sama dan di sebelah kanan baris perintah (yaitu apa muncul setelah mengetik ":").

Di kotak OSX saya, saya hanya bisa mendapatkan baris status untuk muncul di atas baris perintah melalui sesuatu seperti:

set statusline=%F%m%r%h%w\ [FORMAT=%{&ff}]\ [TYPE=%Y]\ [POS=%l,%v][%p%%]\ %{strftime(\"%d/%m/%y\ -\ %H:%M\")}
set laststatus=2

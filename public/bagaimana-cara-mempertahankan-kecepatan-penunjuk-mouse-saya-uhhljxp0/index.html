Bagaimana cara mempertahankan kecepatan penunjuk mouse saya?

Setiap kali saya reboot, saya harus pergi ke Control Panel → Properti Mouse → Opsi Penunjuk dan meningkatkan kecepatan penunjuk. Apakah ada cara untuk mempertahankan pengaturan ini?

Saya mengalami masalah yang sama dengan Windows Vista dan Windows 7.

4
Nestor — Sumber

1 Jawaban

7
7

Anda dapat membuang file batch di folder startup Anda:

reg add "HKCU\Control Panel\Mouse" /v MouseSensitivity /t REG_SZ /d 20 /f

Ubah saja sensitivitas sesuai keinginan. 1 (terendah) - 20 (tertinggi). Save as mouse.bat misalnya, lalu letakkan di folder startup di menu start Anda.

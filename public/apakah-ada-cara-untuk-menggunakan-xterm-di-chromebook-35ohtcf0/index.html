Apakah ada cara untuk menggunakan xterm di chromebook?

Saya menggunakan Secure Shell di chrome yang bagus. Sekarang satu-satunya masalah adalah saya tidak dapat menggunakan xterm. Apakah ada cara untuk menginstal X11? Terima kasih!

Untuk Chromebook tradisional, ekstensi shell aman nassh mendukung port forwarding (tunneling) tetapi tidak mendukung penerusan X11. Sebaliknya, jika Anda dapat menginstal perangkat lunak di komputer jarak jauh, Anda dapat mencoba klien VNC atau Google Desktop Jarak Jauh .

Menggunakan Aplikasi Android

Baru saja. beberapa Chromebook dapat menjalankan aplikasi Android. Jika Anda memiliki salah satu manchine ini , Anda dapat mencoba salah satu aplikasi X-windows untuk android.

  • Misalnya, ada Server X Darkside , XServer XSDL dan Klien QVD di Play Store.

Menggunakan Crouton

Beberapa pengembang menggunakan Chromebook mereka dalam "mode pengembang" dan instal crouton Ubuntu . Crouton mendefinisikan lingkungan Linux berbasis chroot yang dapat menjalankan X-windows.

  • Crouton dapat menjalankan X-windows dalam dua mode: menggunakan Xiwi (X11 di jendela) , yang menjalankan X di jendela Chrome, atau menggunakan XORG di layar virtual yang berbeda. SEBUAH video tutorial untuk menginstal Crouton diterbitkan oleh chromeunboxed (lihat X-window mulai pukul 18:00 menit) ..
  • Beberapa waktu lalu, David Crumpton menerbitkan a panduan untuk menggunakan jendela X di Chromebook . Jika Anda mengalami beberapa masalah, ada beberapa petunjuk tentang jalur perpustakaan dan variabel yang dapat Anda periksa.

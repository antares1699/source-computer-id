"DevManView" bekerja jauh lebih baik untuk saya (Windows 10), dan dapat menggunakan nama perangkat, bukan hanya ID-nya. Untuk beberapa alasan, saya tidak dapat ID perangkat yang tepat menggunakan "Devcon" (ini menunjukkannya dalam daftar, tetapi yang lain terpengaruh setiap kali saya mencoba Aktifkan / Nonaktifkan).

Kegunaan utama saya adalah dengan mudah menonaktifkan / mengaktifkan perangkat "layar sentuh" ​​saat menggunakan pena di layar meja saya (sehingga saya bisa meletakkan telapak tangan saya di atas meja saat menggunakan pena tanpa memicu aktivitas penunjuk acak).

Saya tidak memiliki pengetahuan yang diperlukan untuk membuat kumpulan yang akan memeriksa status perangkat dan "membalikkannya", jadi saya membuat 2 pintasan di bilah tugas saya (Satu untuk menonaktifkan, yang lain untuk mengaktifkan) menggunakan ikon yang berbeda (menggunakan devmanview /disable "HID-compliant touch screen" dan devmanview /enable "HID-compliant touch screen").

"DevManView" adalah salah satu dari banyak alat gratis hebat yang dapat Anda temukan di NirSoft.net. Saya telah menggunakan alat mereka selama bertahun-tahun (hampir masing-masing menawarkan opsi GUI & baris perintah).

Solusi Sederhana

Jika Anda tidak punya waktu untuk bermain-main dengan devcon, berikut adalah utilitas yang lebih sederhana:

DevManView adalah cara lain untuk melakukan ini - dan ini jauh lebih mudah diperoleh daripada devcon.exe, tetapi masih gratis.

Saya menggunakan perintah berikut dalam file batch untuk menggunakan menonaktifkan / mengaktifkan perangkat di laptop istri saya saat kami membutuhkannya:

devmanview.exe /disable_enable "SDA Standard Compliant SD Host Controller"

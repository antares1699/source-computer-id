Kirimi saya email jika seseorang menjalankan sudo

Saya telah menambahkan baris di bawah ini ke /etc/sudoers.d/my_sudoers

Defaults    mail_always
Defaults    mailto="email@domain.com"

Saya mendapatkan email setiap kali sudo dijalankan

Bagaimana saya bisa memodifikasi ini jadi saya hanya menerima email ketika orang tertentu menjalankan sudo (Semua orang kecuali 2 pengguna)

Ini sangat mudah; Defaults dapat diterapkan ke pengguna tertentu (dan batasan lainnya), dan alias pengguna dapat dibuat dengan kecocokan yang dinegasikan. Sesuatu seperti ini:

User_Alias NoMail = goodboy, goodgirl
User_Alias MailUsers = ALL, !NoMail

Defaults           mailto="omgsudo@example.com"
Defaults:MailUsers mail_always
Defaults:NoMail    !mail_badpass

Defaults mailto="email@domain.com", mail_no_user

Jika tidak, Anda mungkin perlu memfilter email Anda.

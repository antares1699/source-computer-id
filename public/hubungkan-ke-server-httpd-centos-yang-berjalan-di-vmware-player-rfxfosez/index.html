Hubungkan ke server httpd CentOS yang berjalan di VMware Player?

Saya memiliki mesin LAMP virtual yang disiapkan dengan CentOS 5.5 32-bit yang berjalan di VMWare Player. Itu sedang di-host pada mesin Win 7 64-bit yang menjalankan koneksi jaringan yang dijembatani. Semuanya tampak OK di OS tamu, karena saya dapat menavigasi ke http: // localhost dan dapatkan halaman pengujian CentOS Apache 2. Namun, saya tidak dapat mengakses server web dari browser pada OS host melalui alamat IP dari OS tamu, yang merupakan titik pengaturan ini sejak awal. Saya berhasil melakukan ping dari host ke tamu (tidak ada firewall di OS tamu) juga.

File gambar untuk CentOS diunduh dari sini (hanya menggunakan bagian 1 dari 2):

http://virtual-machine.org/vmware-image-centos-55-i386-download

Jadi apakah ini sesuatu yang aneh di conf Apache atau beberapa pengaturan tersembunyi yang saya lewatkan di VMWare Player?

Ini adalah masalah firewall, jadi jika Anda ingin memperbaikinya dengan cepat, hentikan sementara iptables dengan memberikan perintah berikut:

service iptables stop

Kemudian periksa dengan menjalankan apache di browser host.

Anda juga dapat menambahkan port 80 dan 443 ke iptables:

-A RH-Firewall-1-INPUT -m state --state NEW -p tcp --dport 80 -j ACCEPT
-A RH-Firewall-1-INPUT -m state --state NEW -p tcp --dport 443 -j ACCEPT

Lakukan itu sebelum melakukan dan memulai ulang layanan. Dengan cara ini Anda dapat menjaga firewall tetap berjalan dan mengizinkannya untuk di-ping dari luar.

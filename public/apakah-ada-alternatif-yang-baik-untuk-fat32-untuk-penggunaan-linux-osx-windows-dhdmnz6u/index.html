Apakah ada alternatif yang baik untuk FAT32 untuk penggunaan Linux / OSX / Windows?

Saya baru saja membeli hard drive USB eksternal yang ingin saya gunakan untuk keperluan backup di Linux dan untuk bertukar beberapa data / file dengan teman / kolega baik menggunakan Windows / OSX / Linux.

Jadi saya akan memiliki satu patisi yang diformat dengan sistem file Linux yang khas (katakanlah ext3 untuk kompatibilitas Linux yang baik).

Apa yang akan Anda rekomendasikan untuk partisi kedua yang dimaksudkan untuk bertukar file dengan teman / kolega? FAT32 adalah salah satu solusi tetapi sudah lama dan memiliki beberapa keterbatasan. Saya ingin tahu apakah ada alternatif yang lebih baik yang dapat digunakan di luar kotak dengan tiga OS utama (saya tahu bahwa kami dapat menginstal dukungan ext3 pada Windows misalnya, tetapi bukan itu yang saya sebut di luar kotak).

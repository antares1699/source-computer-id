Bagaimana cara mengatur jadwal mingguan dalam mode org?

Saya ingin mengatur jadwal saya ke mode org, sehingga saya dapat melihat kelas saya dalam agenda. Saya memiliki kelas yang berbeda, diulang setiap minggu, semuanya membutuhkan waktu 90 menit. Saya sedang memikirkan sesuatu seperti ini:

* My schedule
** Differential equations
   <2012-10-04 Thu 11:45 +1w>--<2012-10-04 Thu 13:15 +1w>

Namun, ada dua masalah: Saya hanya bisa melihat file lanjut misalnya dalam agenda, dan saya tidak dapat melihat file akhir waktu. Ada ide?

Saya cukup yakin Anda mencari kelas-diary-org, disebutkan di faq Worg .

Contoh Anda kemudian adalah (dengan asumsi kelas berakhir pada tanggal 20 Desember):

** 11:45-13:15 Differential equations
   <%%(org-diary-class 10 4 2012 12 20 2012 2)>

Anda dapat menambahkan minggu di mana tidak ada kelas setelah nomor terakhir (yaitu 2 untuk hari selasa).

Jika Anda tidak suka waktu di tajuk utama, Anda dapat menggunakan:

%%(org-diary-class 10 04 2012 12 20 2012 2) 11:45-13:15 Differential Equations

Terakhir kali saya memeriksa, diperlukan %% untuk tidak menjorok ke dalam, jadi saya kira masing-masing solusi luar biasa ini memiliki sedikit kekurangan.

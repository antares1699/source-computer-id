Apakah ada perbedaan antara Live CD Ubuntu dan CD Instal Ubuntu

Saya memesan CD Instal Ubuntu dan mendapatkan CD melalui pos (Ubuntu 9.10). Apakah ini Live CD atau Install CD? Apa sebenarnya definisi dari live CD?

Mereka sekarang menjadi satu dan sama.

Live CD memungkinkan Anda untuk mem-boot OS penuh dari CD dan mencobanya sepenuhnya tanpa memengaruhi penginstalan OS HDD yang ada.

Kemudian memungkinkan Anda untuk menginstal sepenuhnya ke HDD jika diperlukan.

Info khusus tentang Live CD Ubuntu tersedia di sini:

https://help.ubuntu.com/community/LiveCD

Demo cepat Ubuntu. Coba Ubuntu tanpa perubahan apa pun pada mesin Anda! Windows atau apapun yang biasa Anda gunakan tidak terpengaruh setelah mencoba ini dan lalu boot ulang (dalam 99,999% kasus).

Hampir semua Ubuntu Cd juga dapat digunakan sebagai LiveCd dan juga sebagai penginstal. Ini adalah opsi default saat boot dari Cd. Hanya beberapa yang non-standar unduhan (seperti "Cd Alternatif") kekurangan fungsi ini. Pengguna Windows mungkin familiar dengan istilah 'boot CD 'atau' CD yang dapat di-boot '. A "LiveCD" adalah lebih dari itu karena memberikan pilihan untuk menjalankan desktop normal lingkungan dengan semua normal program dan beberapa tambahan. Sebuah LiveCD biasanya menemukan koneksi internet Anda dan Firefox harus dapat menjelajahi file internet di sini.

Kebanyakan distro (versi atau "distribusi" Linux) memiliki ini Fungsionalitas LiveCd di file CD penginstal, beberapa memiliki Cd terpisah untuk mengunduh. Jarang sekali menemukan file distro yang tidak memiliki sesi LiveCd di semua. Ubuntu mencoba membuat LiveCD mereka yang paling mudah digunakan.

Keduanya sama. Setidaknya, dalam kasus Ubuntu. Saat Anda mem-boot CD, Anda dapat memulai Ubuntu Linux dalam mode LIVE. Mode ini akan memungkinkan Anda menguji perangkat keras Anda untuk kompatibilitas, misalnya. Hanya satu hal yang perlu diingat: karena semuanya dimuat dari CDROM, semuanya jauh lebih lambat daripada yang diharapkan saat boot dari hard disk. Namun, Anda akan dapat menguji semua perangkat keras Anda tanpa menyentuh disk Anda. Ini memungkinkan Anda melihat Linux dengan lebih baik sebelum mengambil langkah besar.

Saya menggunakan CD seperti itu di masa lalu saat berbelanja laptop baru. Saya membawa CD di toko, meminta perwakilan penjualan untuk mem-boot laptop dengan CD tersebut, sehingga saya dapat melihat kompatibilitasnya. Saya kemudian keluar dari toko dengan laptop yang menjalankan Ubuntu dengan sempurna, tidak repot tentang driver, kompatibilitas, nirkabel, dll. Ini menghemat banyak waktu saya, dan menghindari kesalahan yang berpotensi merugikan saya.

Saya berbicara tentang Ubuntu, tetapi banyak distribusi lain menawarkan CD langsung seperti itu. KNOPPIX adalah salah satunya. Beberapa dari mereka akan menawarkan opsi pada saat boot, untuk memulai LIVE, atau langsung masuk ke proses instalasi. Opsi kedua ini menghemat waktu Anda, karena hanya memuat file yang diperlukan dalam memori sebelum memulai proses penyiapan, sedangkan versi LIVE perlu memuat lingkungan fungsional sebelum mengizinkan Anda memulai proses penyiapan yang sebenarnya.

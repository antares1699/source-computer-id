Bagaimana cara menghapus Windows Mobile Device Center dari Windows 7?

Ketika saya menghubungkan perangkat seluler apa pun, saya tidak ingin Windows Mobile Device Center muncul, seperti yang dilakukan ActiveSync 4.5.

Pengguna panekuk solusi pada forum xda-developer :

Saya bisa memperbaiki ini. Pertama, saya menghentikan dan menonaktifkan kedua WMDC layanan di bawah Manajemen Komputer> Jasa. Kemudian, saya menemukan mobsync.exe di folder Windows / System32, masuk ke "Izin" untuk itu, mengambil kepemilikan, lalu menghapus semua akun kecuali izin "Semua Orang". Untuk Semuanya, saya setel ke "Tolak Semua".

Saya sekarang tidak memiliki WMDC atau hal-hal sinkronisasi Meletup. Itu bagus. Saya masih bisa telusuri telepon dari My Computer> TYTN II. Saya juga masih bisa menggunakan MyMobiler untuk mengontrol telepon Windows, jadi ini tidak mematikan USB koneksi sama sekali.

Semoga ini bisa membantu seseorang.

Bagikan

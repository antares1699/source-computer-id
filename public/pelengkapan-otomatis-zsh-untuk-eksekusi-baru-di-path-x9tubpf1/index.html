Bagaimana cara mendorong pelengkapan otomatis zsh untuk mengambilnya

Gunakan rehash:

  • rehash menghitung ulang daftar hash internal dari isi direktori yang terdaftar di variabel lingkungan jalan untuk memperhitungkan perintah baru yang ditambahkan.

Ulangi terus-menerus

Biasanya, compinit tidak akan secara otomatis menemukan file executable baru di file $PATH. Misalnya, setelah Anda menginstal paket baru, file dalam format /usr/bin tidak akan langsung atau otomatis disertakan dalam penyelesaian. Jadi, untuk memasukkan exectuables baru ini, seseorang akan melakukannya Lari:

$ rehash

'Rehash' ini dapat diatur agar terjadi secara otomatis. Cukup sertakan mengikuti di zshrc Anda:

~/.zshrc
zstyle ':completion:*' rehash true

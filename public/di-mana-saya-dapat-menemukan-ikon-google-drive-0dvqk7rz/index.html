Di mana saya dapat menemukan ikon Google Drive?

Saya membuat tautan khusus ke direktori Google Drive saya di bilah aplikasi Windows 7 Professional. Sekarang saya ingin mengubah ikon itu, sehingga jelas itu adalah folder Google Drive.

Bagaimana cara menemukan ikon Google Drive?

Temukan folder instalasi Google Drive dan googledrivesync.exe yang dapat dieksekusi (misalnya C:\Program Files (x86)\Google\Drive\googledrivesync.exe).

Klik kanan -> Properti -> Ubah Ikon

Karena sebagian besar versi Windows dan program Windows telah atau akan pindah ke 64-bit, ikon ini sekarang dapat ditemukan di sini. Perhatikan, ikon yang ditampilkan pada .exe (mungkin) yang Anda cari, tetapi .exe itu berisi selusin ikon jika Anda memilihnya sebagai sumber, seperti kebanyakan.

C:\Program Files\Google\Drive\googledrivesync.exe

Jika Anda kebetulan masih menggunakan sistem 32-bit (ragu-ragu), seperti yang dikatakan duDE di atas, itu akan menjadi:

C:\Program Files (x86)\Google\Drive\googledrivesync.exe

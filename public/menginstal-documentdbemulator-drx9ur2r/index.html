Menginstal DocumentDb.Emulator

Saya dapat menginstal DocumentDb.Emulator tanpa kesalahan. Namun, ketika saya mencoba meluncurkannya, saya mendapatkan kesalahan berikut:

"Program tidak dapat dimulai karena api-ms-win-core-path-l1-1-0.dll hilang"

Saya menjalankan Windows 7. Apakah emulator memerlukan versi Windows yang lebih baru? Saya tidak dapat melihat persyaratan sistem apa pun di situs web Microsoft.

Dari Persyaratan sistem DocumentDB Emulator :

DocumentDB Emulator memiliki perangkat keras dan perangkat lunak berikut Persyaratan:

  • Persyaratan perangkat lunak
    • Windows Server 2012 R2, Windows Server 2016, atau Windows 10
  • Persyaratan Perangkat Keras Minimum
    • RAM 2 GB
    • 10 GB tersedia ruang hard disk

Jadi mungkin ada beberapa dependensi yang hilang.

Ada halaman umpan balik yang meminta dukungan untuk versi Windows sebelumnya , menyatakan:

Penginstal tidak boleh berjalan pada platform yang tidak didukung. Sekarang ia melanjutkan dan menginstal, lalu emulator gagal dengan error yang tidak jelas tentang api-ms-win-core-path-l1-1-0.dll

Dan sebuah balasan

Terima kasih atas umpan baliknya dan kami akan mempertimbangkannya. Kode backend kami menggunakan beberapa API kripto yang tidak tersedia di Windows 7 sehingga saat ini tidak didukung.

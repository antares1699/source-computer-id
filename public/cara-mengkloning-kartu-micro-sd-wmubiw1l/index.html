Cara mengkloning kartu micro SD

Saya perlu mengkloning kartu micro SD, menggunakan OS Windows atau Linux. Kartu SD harus dikloning sebagai volume keseluruhan, dan bukan sebagai partisi. Ini mungkin berisi beberapa partisi, beberapa di antaranya tidak dapat dikenali oleh Windows / Linux. Pada dasarnya, saya ingin membuat file gambar kartu SD, dan kemudian memasukkan media kartu SD lain dan menyalin file ini ke dalamnya, menimpa semua yang dikandung media ini. Bagaimana saya bisa melakukan ini?

Di Linux, Anda dapat menggunakan dd untuk melakukannya.

dd if=/dev/sdcard1 of=/dev/sdcard2 dengan if adalah sumber asal dan of SD tujuan.

atau jika Anda ingin mengambil gambar terlebih dahulu dan menyalinnya ke SD ke-2 setelah:

dd if=/dev/sdcard1 of=~/sdimage 
dd if=~/sdimage of=/dev/sdcard2

Gunakan saja dd untuk Windows ( http://www.chrysocome.net/dd ).

Jalankan CDM sebagai Administrator:

dd --list

Periksa perangkat mana yang merupakan kartu SD Anda. Dalam kasus saya, itu adalah: \\?\Device\Harddisk1\Partition0. Partition0 mengacu pada seluruh disk, Partition1 ke partisi pertama, dll.

Salin seluruh kartu SD ke file gambar:

dd if=\\?\Device\Harddisk1\Partition0 of=my.img bs=1M --progress

Kemudian masukkan kartu baru dan tulis kembali file gambar ke dalamnya:

dd if=my.img of=\\?\Device\Harddisk1\Partition0 bs=1M --progress

Jika Anda mendapatkan file akses ditolak kesalahan setelah menulis beberapa blok, bersihkan tabel partisi kartu SD sebelum menulis ke sana:

diskpart

DISKPART> list volume
DISKPART> select volume #
DISKPART> clean
DISKPART> exit

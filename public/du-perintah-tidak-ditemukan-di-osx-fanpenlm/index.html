Rupanya, Anda berada dalam mode penyelamatan di Mac. Jika Anda dapat memasang hard disk Anda, Anda dapat menggunakan:

/Volumes/Macintosh\ HD/sbin/du

Selain itu, jika Anda dapat memasang hard disk, Anda dapat menjalankan perintah berikut untuk mengakses setiap perintah:

for path in echo /Volumes/Macintosh\ HD/{sbin,bin,usr/{sbin,bin}}; do PATH=$path:$PATH; done

Bagaimana cara menambahkan item ke Windows 10 Explorer Sidebar

Saya kadang-kadang menggunakan Windows 10 explorer daripada Total Commander yang biasa saya gunakan saat menggunakan windows. Satu hal yang benar-benar mengganggu saya adalah pintasan "OneDrive", karena saya tidak menggunakan OneDrive.

Saya menemukan solusi untuk menghapusnya dengan mengedit registri sini .

Itu bagus, bagaimanapun, saya MELAKUKAN menggunakan OwnCloud dan saya ingin menambahkan pintasan ke jendela Windows Explorer tempat OneDrive dulu berada (sebaiknya dengan logo cloud sendiri).

Saya memiliki pintasan Owncloud di item "Akses Cepat", yaitu baik , tapi saya ingin yang lebih baik.

Catatan: ini juga berfungsi untuk NextCloud

PEMBARUAN: berikut adalah pendekatan yang sedikit berbeda untuk Win10 yang tidak akan konflik dengan Google drive jika Anda menggunakan keduanya - dengan pendekatan lama, jika Anda pergi ke folder ownCloud melalui ikon sidebar, lalu klik kanan file dan coba google drive - share atau opsi lain apa pun dari menu konteks, itu akan berhenti berfungsi sampai Anda memulai contoh baru penjelajah.

Pendekatan ini tidak akan memberi Anda masalah ini.

Untuk memotong ke pengejaran, cukup buat .bat berikut dan pikirkan Jalur Ikon Default dan tentu saja TargetFolderPath (dalam hal ini d: \ ownCloud) (Posting lama dan solusinya ada di bawah blok kode ini untuk .bat Anda

reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79} /ve /t REG_SZ /d "Radius Cloud" /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\DefaultIcon /ve /t REG_EXPAND_SZ /d "C:\Program Files (x86)\ownCloud\owncloud.exe,0" /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79} /v System.IsPinnedToNameSpaceTree /t REG_DWORD /d 0x1 /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79} /v SortOrderIndex /t REG_DWORD /d 0x42 /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\InProcServer32 /ve /t REG_EXPAND_SZ /d %%systemroot%%\system32\shell32.dll /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\Instance /v CLSID /t REG_SZ /d {0E5AAE11-A475-4c5b-AB00-C66DE400274E} /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\Instance\InitPropertyBag /v Attributes /t REG_DWORD /d 0x11 /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\Instance\InitPropertyBag /v TargetFolderPath /t REG_EXPAND_SZ /d D:\ownCloud /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\ShellFolder /v FolderValueFlags /t REG_DWORD /d 0x28 /f
reg add HKCU\Software\Classes\CLSID\{9499128F-5BF8-4F88-989C-B5FE5F058E79}\ShellFolder /v Attributes /t REG_DWORD /d 0xF080004D /f
reg add HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\{9499128F-5BF8-4F88-989C-B5FE5F058E79} /ve /t REG_SZ /d ownCloud /f
reg add HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\HideDesktopIcons\NewStartPanel /v {9499128F-5BF8-4F88-989C-B5FE5F058E79} /t REG_DWORD /d 0x1 /f

Inilah yang saya lakukan. Jika Anda membutuhkan bantuan untuk menghapusnya, biar tahu. Diuji pada Win 10 Pro.
Hasil:

Pada skrip di bawah ini, Anda perlu:

  • Ganti setiap kemunculan CLSID dengan milik Anda ( buat satu online atau gunakan guidgen.exe)
  • Tentukan jalur ke folder Cloud Anda sendiri (dua kejadian): "TargetFolderPath"="x:\\pathTo-ownCloudFolder"

Anda juga dapat mengganti ikon default dengan memodifikasi baris ini (dua kemunculan):
@="C:\\Program Files (x86)\\ownCloud\\owncloud.exe,0"

Jika Anda menggunakan NextCloud, Anda perlu mengubah ke: @="C:\\Program Files (x86)\\NextCloud\\nextcloud.exe,0" untuk menampilkan ikon default.

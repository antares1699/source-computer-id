bash - menyalin banyak file dengan nama yang sama

Saya ingin menyalin file dengan nama yang sama tetapi di folder yang berbeda ke destdir. Masalahnya adalah tentang tidak menimpanya di destdir.

Pada dasarnya ini sudah cukup, meskipun saya tidak tahu cara mengganti nama setiap file. Adapun sekarang mereka semua ditimpa:

$ cp / src / folder / * / file ~ / dest / *

  • Catatan: perintah cp Anda sangat salah. Saya rasa Anda ingin tanda bintang kedua "menarik" nilainya dari yang pertama, tetapi ini tidak pernah terjadi. Jika ~/dest/ tidak kosong, Anda bisa menyalin hal-hal yang tidak Anda inginkan ke dalam direktori yang tidak Anda harapkan.

Inilah salah satu metodenya:

cd /src/folder
for f in */file; do
    cp -v "$f" ~/dest/"${f//\//_}"
done

yang akan menyalin /src/folder/foobar/file ke ~/dest/foobar_file yang seharusnya unik

Anda dapat melihat apakah opsi --backup dari cp tersedia (saya yakin ini tidak ada di MacOS, tapi saya mungkin salah):

 cp --backup=t /src/folder/*/file /dest

akan membuat file.jpg. ~ 1 ~, file.jpg. ~ 2 ~ dan seterusnya.

Cara mengubah satu karakter dalam rumus excel untuk 1500 baris

Saya memiliki lebih dari 1500 baris informasi dengan rumus seperti, = '3'! $ AH $ 6, di mana setiap kolom berisi informasi dari lembar lain di dalam buku kerja. Dalam contoh, sheet berlabel '3' direferensikan. Apakah ada cara untuk hanya mengubah referensi ke sheet lain untuk semua 1500 baris sekaligus? Misalnya ubah rumusnya menjadi = '4'! $ AH $ 6. Saya hanya perlu mengubah referensi ke sheet lain karena referensi sel akan tetap sama. Saya harap ini masuk akal, terima kasih atas bantuannya.

Ada beberapa cara untuk melakukannya

  1. Gunakan VBA » ALT + F11 untuk memasukkan dan ALT + F8 untuk menjalankan makro kecil ini

    Ini memiliki keuntungan bahwa Anda dapat memasukkan bahkan referensi yang tidak valid (misalnya referensi hanya akan ada di PC lain atau di masa mendatang) tanpa perbaharui data dialog untuk setiap sel

    Jika mau, Anda dapat memilih satu kolom dengan columns("A:A") bukan usedrange

    Sub replace()
 Application.DisplayAlerts = False
 For Each cell In UsedRange
    cell.Formula = replace(cell.Formula, "='4", "='3")
 Next
 Application.DisplayAlerts = True
End Sub

  2. Gunakan Excel temukan & ganti fungsi » CTRL + H dan cari dalam rumus.
    Pilih kolom Anda yang ingin Anda ubah sebelum Anda membuka dialog

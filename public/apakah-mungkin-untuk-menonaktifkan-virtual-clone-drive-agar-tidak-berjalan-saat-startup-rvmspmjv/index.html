Apakah mungkin untuk menonaktifkan Virtual Clone Drive agar tidak berjalan saat startup?

Saya telah menginstal Virtual Clone drive di Windows 7, tetapi saya jarang menggunakannya. Saya ingin menonaktifkannya agar tidak berjalan saat startup. Saya sudah memeriksa folder startup dan msconfig, tetapi saya tidak dapat menemukan cara untuk menonaktifkan layanan ini.

  • Itu muncul di MSCONFIG di sistem saya di bawah tab Startup. Itu terletak di tombol Jalankan di simpul WOW64. "WhatInStartup" adalah cara praktis untuk dengan mudah menonaktifkan & mengaktifkan kembali program dan layanan.

Tekan Windows + R (jalankan). Di kotak yang muncul ketik msconfig. Ini akan membawa Anda ke konfigurasi sistem. Buka tab yang mengatakan windows start (atau sesuatu seperti itu, sistem saya tidak dalam bahasa Inggris). Temukan entri yang mengatakan virtual Clone Drive dan hapus centangnya. Ini akan meminta Anda untuk memulai ulang, Anda dapat memilih untuk melakukannya nanti tidak masalah.

Catatan: Saya tidak akan merekomendasikan mengubah registri kecuali Anda benar-benar tahu apa yang Anda lakukan, dan bahkan kemudian, buat titik pemulihan sebelum Anda melakukannya.

Bersulang!

Solusi yang sedikit kurang kotor ...

Jika Anda tidak keberatan memiliki makro di normal.dotm Anda dan memiliki variabel tersembunyi di dokumen Anda, maka Anda mungkin dapat menggunakan makro ini dan mengatur tombol pada akses cepat Anda untuk menjalankannya ... Saya sudah mencobanya dan berfungsi pada dokumen bebas makro sehingga dokumen itu sendiri tidak memerlukan makro untuk menggunakan metode ini, saya akan meletakkan ini di template normal.dotm saya ...

Sub updatePath()
'
' updatePath Macro
'
'
Dim myPath As String
myPath = ActiveDocument.Path
If myPath = "" Then
    'do nothing as the document has no path... needs to be saved first
Else
    If ActiveDocument.Variables.Count = 0 Then
        ActiveDocument.Variables.Add Name:="myPath", Value:=myPath
    Else
        i = 1
        Do While i < (ActiveDocument.Variables.Count + 1)
            If ActiveDocument.Variables.Item(i).Name = "myPath" Then
                ActiveDocument.Variables.Item(i).Value = myPath
            End If
            i = i + 1
        Loop
    End If
End If

End Sub

Dan kemudian tambahkan kode bidang

DOCVARIABLE myPath

yang ketika diperbarui setelah menjalankan makro di atas akan melakukan apa yang Anda inginkan. Memang itu adalah dua atau tiga klik dan bukan hanya pembaruan sederhana, tetapi itu berarti bahwa dokumen yang diekspor mendapatkan jenis bidang yang tepat dan hanya orang dengan makro ini yang dapat mengatakan di mana dokumen tersebut. Sebaiknya disimpan :)

Sayangnya, tidak ada cara untuk melakukan ini dengan bidang FILENAME tanpa makro, bidang FILENAME tidak menawarkan opsi untuk menghilangkan nama file.

Bagaimanapun, berikut adalah makro untuk menyisipkan jalur file, tanpa nama file, bagi mereka yang mungkin membutuhkannya.

Sub InsertPath()
  Dim sPath As String

  sPath = ActiveDocument.Path
  If sPath = "" Then
    MsgBox "You need to save the document before running this macro.", _
           vbOKOnly, "Document not saved"
  Else
    sPath = sPath & Application.PathSeparator
    Selection.TypeText(sPath)
  EndIf
EndSub

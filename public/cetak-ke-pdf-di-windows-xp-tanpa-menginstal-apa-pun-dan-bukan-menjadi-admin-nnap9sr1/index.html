Cetak ke PDF di Windows XP tanpa menginstal apa pun dan bukan menjadi admin?

Di tempat kerja kami menjalankan Windows XP Pro dan saya bukan administrator. Saya secara teratur harus mencetak dokumen untuk dibawa dan saya lebih suka mencetaknya ke PDF dan memuatnya langsung ke ponsel Android saya daripada membawa salinan kertas.

Apakah ada cara untuk mencetak ke PDF tanpa memiliki akses administrator? Selain itu, saya tidak mencari fungsi ekspor ke PDF Open Office, saya ingin ini menjadi printer sehingga program apa pun yang dapat mencetak dapat mengaksesnya. Bisakah saya menjalankan sesuatu seperti ini dari thumb drive?

Edit: Bagaimana dengan mencetak ke file? Bukankah xp secara native mendukung pencetakan postscript? Kemudian bagian kedua itu akan dengan mudah mengubah file .ps menjadi PDF. Ini pada dasarnya hanya akan menambahkan satu langkah ke proses yang saya coba capai di atas. Apakah ini mungkin tanpa hak istimewa admin?

Edit 2: Saya seharusnya menyebutkan, kami tidak memiliki akses ke internet, dan saya tidak dapat menyimpan dokumen saya dengan cara apa pun. Saya hanya bisa mencetak. Itu sebabnya saya ingin mencetak ke file, jadi saya bisa menyimpannya.

Buat pintasan Windows 7 ke halaman pdf tertentu

Apakah ada cara untuk membuat pintasan Windows 7 ke halaman pdf tertentu?

4
julo — Sumber

1 Jawaban

7
7

Jika Anda menginstal Adobe Acrobat Reader, Anda dapat membuat pintasan ke baris perintah yang membuka file PDF ke halaman tertentu:

AcroRd32.exe /A "page=7=OpenActions" myfile.pdf

Akan membuka "myfile.pdf" ke halaman 7.

Anda dapat menemukan lebih banyak sakelar baris perintah sini .

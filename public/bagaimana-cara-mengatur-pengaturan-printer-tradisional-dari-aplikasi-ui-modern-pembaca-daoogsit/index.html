Windows 8 memiliki file model printer baru yang memungkinkan driver printer menambahkan UI bergaya Modern.

Jika pabrikan belum menyediakan UI konfigurasi apa pun untuk perangkat mereka, maka Windows menyediakan beberapa UI standar yang dapat Anda gunakan dengan printer apa pun. Namun, jika produsen printer telah memutuskan untuk berinvestasi dalam memberikan pengalaman yang disesuaikan untuk perangkat mereka, mereka dapat menyediakan aplikasi yang menggantikan UI Windows standar. Kemudian, ketika Anda memutuskan untuk mengubah konfigurasi perangkat, atau ketika konfigurasi perangkat berubah selama pencetakan (misalnya kertas macet), maka Windows akan menampilkan aplikasi yang disesuaikan dari pabrikan kepada Anda.

Anda harus menunggu produsen printer Anda merilis driver Windows 8 yang tepat yang menambahkan opsi tambahan ini untuk aplikasi UI Modern.

Solusinya adalah mengatur opsi default melalui Control Panel => Devices and Printers.

Semua pekerjaan cetak harus menggunakan pengaturan default, yang dapat diatur dari Desktop Control Panel. Dengan pengujian, ini tampaknya berlaku untuk aplikasi UI Modern.

Klik kanan pada printer dan pilih Printing Preferences. Ubah pengaturan sesuai keinginan dan simpan. Coba cetak dari aplikasi Modern UI. Ingat, pengaturan ini akan tetap ada sampai Anda mengubahnya (kembali).

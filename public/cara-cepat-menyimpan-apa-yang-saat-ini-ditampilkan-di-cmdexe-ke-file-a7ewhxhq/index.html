Cara cepat menyimpan apa yang saat ini ditampilkan di cmd.exe ke file

Saya bertanya apakah ada cara / perintah cepat untuk menyimpan keluaran standar saat ini dari cmd.exe atau PowerShell ke file.

Misalnya, saya telah menjalankan banyak perintah di cmd.exe yang menghasilkan seperti ratusan baris output standar. Idealnya, saya mencari satu perintah untuk melakukan "pilih semua" dan simpan ke file secara otomatis.

Catatan: Saya sudah membaca ini . Tetapi saya tidak ingin mengubah perintah asli saya, jadi pengalihan ">" atau ">>" tidak dapat digunakan dalam skenario ini.

Mungkin ada solusi yang lebih efisien, tetapi Anda dapat menyelesaikan pekerjaan dengan sedikit bantuan dari AutoHotkey .

Pertama, menyusun skrip berikut:

ClipboardBackup := Clipboard
Clipboard =

Send, !{Space} ;Menu
Sleep, 100
Send, e ;Edit
Send, s ;Select All

Send, {Enter} ;Copy
ClipWait

FileAppend, %Clipboard%, *

Clipboard := ClipboardBackup

ExitApp

Tempatkan file .exe di file % PATH% direktori. Saya menamai milik saya savecmd.exe.

Sekarang, Anda dapat secara efektif menyimpan konten jendela command prompt dengan menggunakan sintaks berikut:

savecmd > file.txt

catatan: Anda dapat menambahkan lebih banyak ke skrip AutoHotkey untuk sedikit membersihkan output, tetapi berhasil!

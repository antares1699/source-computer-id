Saya memiliki masalah yang sama dengan Windows 7. Solusinya adalah ganti nama saya id_rsa.ppk untuk [nama_komputer] .ppk . Tampaknya, bagi saya setidaknya, Pageant sedang mencari file yang tidak ada.

Jika mengganti nama tidak membantu, Anda dapat menggunakan Monitor Proses untuk memeriksa file apa yang sebenarnya dicari oleh Pageant.

Dengan Monitor Proses, gunakan filter:

  • "Nama Proses - adalah - pageant.exe"
  • "Jalur - berisi - .ppk"

Saya mendapatkan kesalahan ini pada setiap startup Windows 7. Saya menemukan bahwa itu mencoba membuka file .ppk dengan nama yang salah yang telah di-hardcode ke dalam pintasan peluncuran Pageant di folder Startup.

Untuk memeriksa apakah ini masalahnya, buka Start> All Programs> Startup> klik kanan pada Pageant> Properties.

Bidang "Target" menyertakan jalur ke pageant.exe yang dapat dieksekusi, berpotensi diikuti oleh jalur ke .ppk. Verifikasikan bahwa .ppk ini benar-benar ada.

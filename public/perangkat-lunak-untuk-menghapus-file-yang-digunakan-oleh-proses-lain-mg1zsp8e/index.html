Untuk menghapus file / folder yang dikunci oleh sistem. 1.- Temukan folder c: \ users \ xxx 2.- Cobalah untuk mengambil kepemilikan folder dan subfolder: (mungkin akan gagal) 3.- Jika tidak gagal, beri Anda hak penuh atas folder dan subfolder, lalu hapus. 4.- Tetapi karena gagal, kita perlu menemukan proses jendela penyihir mengunci file. 5.- Mulai Microsoft SysInternal \ procexp.exe 6.- Pergi untuk mencari, mencari pegangan 7.- Lakukan pencarian c: \ users \ xxx 8.- Jangan ambil nomor PID yang mengunci file (kolom kedua) 9.- Kembali dan Bunuh proses atau Bunuh pohon proses jika itu pohon. 10.- Kembali ke folder dan ambil pemilik folder dan subfolder c: \ users \ xxxx 11.- Memberi Anda hak penuh atas folder dan subfolder c: \ users \ xxx 12.- Hapus folder c: \ users \ xxx 13.- Masuk kembali pengguna, dia akan memiliki profil baru yang segar.

Cara mengambil kapal pemilik dan memberi Anda hak penuh atas folder dan subfolder 1.- Temukan folder c: \ users \ xxx Anda 2.- Mouse kanan di atasnya | properti | Keamanan | Lanjutan | Pemilik | Sunting | Pilih akun iklan Anda | Ganti Pemilik subkontainer dan objek | Menerapkan 3.- Kembali ke tab Izin | mengubah izin | tambahkan akun iklan Anda | Kontrol penuh | Ganti semua izin objek anak dengan izin yang diwariskan dari objek ini 4.- Sekarang hapus folder c: \ users \ xxx Anda.

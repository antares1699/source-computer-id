Bagaimana cara menghapus program yang isinya sudah dihapus?

Saya tidak sengaja menghapus folder program yang diinstal di folder C:\Program Files (ini untuk produk Autodesk). Jika saya pergi ke alat uninstaller Windows, program tidak terdaftar di sana lagi.

Saya mencoba menjalankan penginstal untuk program itu lagi tetapi menurutnya program tersebut masih ada dan tidak mengizinkan saya menginstalnya.

Bagaimana cara membersihkan semuanya agar saya dapat menginstalnya lagi?

Jika Anda telah menghapus folder dari drive C: maka gunakan CCleaner untuk menghapusnya dari daftar program yang diinstal, dan memperbaiki registri Anda dengan menjalankan CCleaner setelah itu. Itu tidak akan muncul lagi.

Dengan asumsi Anda tidak menginstal perangkat lunak Autodesk lain yang memerlukan konfigurasi ulang ekstensif, Anda dapat mengikuti langkah-langkah yang diuraikan di Dukungan Autodesk , khususnya dari Hapus file dan folder produk Autodesk yang tersisa seterusnya:

Peringatan : Langkah-langkah berikut menginstruksikan Anda untuk menghapus folder di lokasi yang akan mempengaruhi semua produk Autodesk yang diinstal. Ini langkah-langkah mengasumsikan Anda sebelumnya telah menghapus semua produk Autodesk dan periferal terkait dan aman untuk menghapus folder tersebut.

  1. Di Windows Explorer, temukan dan hapus folder penginstalan untuk setiap produk Autodesk yang dibongkar di prosedur sebelumnya. Sebagai contoh:

        C:\Program Files\AutoCAD 2015
    C:\Program Files\Autodesk\AutoCAD 2016

  2. Hapus folder berikut:

        C:\Program Files\Common Files\Autodesk Shared

    Catatan : Lokasi ini mungkin berbeda jika sistem operasi Anda dikonfigurasi untuk menyimpan folder Program Files di drive selain C, atau jika Anda menginstal produk Autodesk ke drive lain dan nama folder.

  3. Hapus folder profil pengguna berikut (gantikan nama login Anda dengan% username%):

    Windows 7

        C:\Users\%username%\AppData\Roaming\Autodesk
    C:\Users\%username%\AppData\Local\Autodesk
    C:\Users\Public\Documents\Autodesk
    C:\ProgramData\Autodesk

    Windows XP

        C:\Documents and Settings\%username%\ Application Data\Autodesk
    C:\Documents and Settings\%username%\Local Settings\Application Data\Autodesk
    C:\Documents and Settings\All Users\Application Data\Autodesk

Windows 7: Tergantung pada pengaturan sistem Anda, Windows Explorer mungkin tidak secara otomatis menampilkan semua subfolder di bawah folder Pengguna. Jika Anda tidak dapat menemukan folder ini, salin dan tempel folder di atas lokasi ke bilah alamat Windows Explorer.

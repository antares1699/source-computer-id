Bagaimana saya bisa mengetik backtick ( ` ) atau tilde ( ~ ) menggunakan keyboard Italia saya di Windows 8?

Sekarang, saya menggunakan ALT + 9 6 . Apakah ada kombinasi tombol lainnya?

Mungkin, inilah solusi terbaiknya:

ITA-Dev Keyboard - Keyboard Italia untuk pengembang

Pasang keyboard ini, lalu lepaskan keyboard apa pun di Panel Kontrol kecuali keyboard baru ini.
Sekarang Anda dapat mengetik karakter ini:

` (AltGr + ')
~ (AltGr + ì)
À (AltGr + shift + a)
È (AltGr + shift + e)
Ì (AltGr + shift + i)
Ò (AltGr + shift + o)
Ù (AltGr + shift + u)
® (AltGr + r) registered
© (AltGr + c) copyright

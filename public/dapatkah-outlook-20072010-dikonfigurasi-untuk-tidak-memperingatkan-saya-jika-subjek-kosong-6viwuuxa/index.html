  1. Buka Outlook
  2. tekan Alt F11 . Ini akan membuka editor Visual Basic.
  3. Perluas Project1 dari panel proyek sisi kiri
  4. Luaskan Objek Microsoft Office Outlook
  5. Sekarang Anda akan melihat "ThisOutlookSession". Klik dua kali itu.

  6. Sekarang salin kode berikut dan tempel kode di bawah ini di jendela baru yang dibuka saat Anda mengklik dua kali "ThisOutlookSession".

    Private Sub Application_ItemSend(ByVal Item As Object, Cancel As Boolean) 
Dim strSubject As String 
strSubject = Item.Subject 
If Len(strSubject) = 0 Then 
Prompt$ = "Mail is without subject. Do you want send it anyway?" 
If MsgBox(Prompt$, vbYesNo + vbQuestion + vbMsgBoxSetForeground, "Subject is empty") = vbNo Then 
Cancel = True 
End If 
End If 
End Sub

  7. Simpan dan Tutup jendela. Itu semuanya.

Sekarang Anda dapat mencoba mengirim email tanpa subjek dari Outlook Anda - Anda akan mendapatkan peringatan yang luar biasa ini.

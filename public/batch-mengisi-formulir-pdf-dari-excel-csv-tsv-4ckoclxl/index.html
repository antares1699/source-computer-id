Mungkin alat open source saya BulkPDF dapat melakukan apa yang Anda inginkan.

Apa itu BulkPDF?

BulkPDF adalah perangkat lunak freeware gratis dan mudah digunakan (Open Source), yang memungkinkan untuk secara otomatis mengisi formulir PDF yang ada dengan nilai yang berbeda. Hanya spreadsheet (Microsoft Excel 2007/2010/2013, LibreOffice atau OpenOffice Calc) dengan nilai yang diinginkan yang diperlukan.

Bagaimana cara kerjanya?

BulkPDF secara otomatis mengenali nilai dalam tabel yang dipilih. Kemudian nama kolom harus ditetapkan secara manual ke kolom formulir. Setelah proses pengisian dimulai, BulkPDF akan menelusuri baris demi baris dan menuliskan nilai sel ke dalam formulir.

Fungsi

  • Mendukung bidang teks, kotak centang, tombol radio, kotak kombo, dan kotak daftar
  • Konfigurasi individu untuk setiap bidang formulir
  • Nama file dapat dikonfigurasi secara bebas dengan nilai otomatis
  • Perlindungan penulisan yang dapat diatur untuk setiap kolom formulir serta untuk seluruh dokumen
  • Simpan dan muat konfigurasi
  • Konfigurasi dapat disesuaikan secara manual (XML)
  • Jalur relatif dalam file konfigurasi dimungkinkan
  • Alat baris perintah tersedia
  • Kompatibel dengan Acroform dan XFA

Jika Anda memiliki pertanyaan, jangan ragu untuk bertanya.

Saya yakin InDesign memiliki fitur untuk menggabungkan spreadsheet dan formulir PDF. Ada juga plugin pihak ketiga berbayar dan skrip VBA untuk melakukan gabungan surat dari dalam Acrobat itu sendiri.

Mengkode solusi tidak terlalu sulit - di masa lalu saya telah menggunakan Python untuk membuat kode solusi kustom yang cepat. Dalam kodenya, Anda harus membaca setiap baris spreadsheet dan membuat file FDF untuk formulir pdf. Setelah Anda memiliki FDF, Anda dapat menggunakan PDFTK untuk menggabungkan formulir PDF asli dan file FDF dan membuat versi formulir PDF yang sudah diisi.

Jika pengkodean bukan urusan Anda, saya membangun situs gratis tempat Anda dapat menggabungkan surat spreadsheet dan formulir pdf di pdfzero.com . Anda dapat mengunggah formulir pdf dan spreadsheet, klik pada bidang pdf yang ingin Anda isi, dan pilih kolom yang ingin Anda gunakan untuk bidang itu. Anda juga dapat menentukan nama file menggunakan data kolom untuk mengatur semuanya. Setelah file diproses, itu akan mengembalikan file zip dengan semua PDF yang digabungkan.

Di perusahaan saya, kami menggunakan secara teratur Penggabungan Surat PDF untuk semua jenis formulir PDF yang harus kami kirim secara massal. Kami merasa sangat mudah digunakan.

Pemakaian:

Anda akan dipandu melalui 4 langkah:

  1. Unggah file PDF
  2. Unggah file Excel
  3. Di Editor, Anda dapat menarik placeholder drag'n'drop ke PDF
  4. Klik untuk memulai pembuatan PDF

Keuntungan:

  • Berfungsi dengan semua jenis file PDF (tidak memerlukan bidang formulir yang dapat diisi)
  • Banyak format spreadsheet yang didukung (Microsoft Excel, Open Office Spreadsheet, CSV, TXT dll.)
  • Templat Formulir PDF baru disiapkan dengan sangat cepat
  • Templat Formulir PDF dapat digunakan kembali dengan satu klik
  • Tidak diperlukan pengetahuan pemrograman

Kekurangan:

  • Paket gratis hanya mencakup pembuatan 100 dokumen

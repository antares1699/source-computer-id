Jalankan file batch sebelum pengguna masuk ke Windows 2003 R2?

Saya memiliki mesin Amazon EC2 (Windows Server 2003 R2) di mana saya ingin menjalankan skrip (file .bat) ketika mesin Windows Server 2003 R2 boot. Ini perlu dijalankan SEBELUM pengguna login. Idealnya Saya ingin memperluas penyelesaian yang sama pada instans Windows Server 2008 R2 saya juga - tetapi Windows Server 2003 R2 sangat penting bagi saya untuk saat ini.

Hanya sebagai FYI, file .bat memperbarui catatan DDNS sehingga mesin EC2 tidak perlu menggunakan IP statis.

Gunakan GPedit.msc

Buka Manajemen Komputer / Windows / Skrip / Startup.

Tempatkan naskah Anda di sana.

Perhatikan bahwa skrip akan berjalan di bawah akun SISTEM LOKAL.

Anda juga dapat menggunakan metode yang sama pada instans 2008R2 Anda.

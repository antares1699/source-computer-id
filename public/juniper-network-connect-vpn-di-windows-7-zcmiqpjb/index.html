Bagi Anda seperti saya yang jawabannya tidak berhasil, ini mungkin:

  • Buka layanan (Start-> Rune "services.msc").
  • Cari "Juniper Network Connect Service"
  • Jika dihentikan, coba mulai.
  • Jika tidak dimulai, centang "Path to executable:" di kotak dialog layanan.
  • Jika jalur salah, buka registri: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\dsNcService
  • Perbaiki jalannya.
  • Mulai dsNetworkConnect.exe (atau luncurkan klien jaringan Juniper)

Saya memposting tentang ini secara rinci di blog saya :

Prosedur instalasi:

Langkah 1: Copot pemasangan versi sebelumnya dari Sambungan Jaringan Juniper. (Jika Anda pernah mencoba menghubungkan kantor Anda komputer maka Anda mungkin telah menginstal perangkat lunak ini pada mesin Anda. Periksa di Tambah atau hapus program)

Langkah 2: Buka Komputer dan klik Atur -> Folder dan cari options -> tab View -> Di bawah Hidden files and folder, klik Show file, folder, dan drive tersembunyi dan klik OK.

Langkah 2: Sekarang Anda harus menghapus beberapa file di komputer Anda. Pergi ke Komputer -> Disk Lokal (C :) -> Pengguna -> "Klik nama pengguna Anda" (Jika Anda tidak yakin dengan nama pengguna Anda, lalu klik mulai. Anda akan menemukan nama pengguna Anda di atas Dokumen)

Langkah 3: Klik pada folder AppData (folder tersembunyi) -> Roaming -> Hapus folder Juniper Networks.

Langkah 4: Instal perangkat lunak Juniper Networks yang Anda miliki diunduh dari tautan di atas. (Klik Ya untuk semua otentikasi pesan)

Langkah 5: Sekarang Jalankan perangkat lunak. Pergi ke Start -> All Programs -> Juniper Jaringan -> Sambungan Jaringan 6.30. -> Sambungan Jaringan

Langkah 6: Sekarang masukkan alamat VPN dan masuk.

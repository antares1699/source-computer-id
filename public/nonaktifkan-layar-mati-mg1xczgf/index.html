Saya memiliki baris ini di file ~/.xinitrc saya:

# turn off Display Power Management Service (DPMS)
xset -dpms
setterm -blank 0 -powerdown 0

# turn off black Screensaver
xset s off

Ini mencegah tampilan mati selama menonton film (Saya juga menggunakan XMonad).

Cara memperbaiki penginstalan .NET framework 4 di Windows 8

mscorlib.dll saya sepertinya rusak. Saya mencoba menginstal ulang dari paket penginstalan .NET 4 mandiri, tetapi .NET 4 terintegrasi ke dalam Windows 8.1. Apakah ada cara untuk memperbaikinya?

Saya telah mencoba System Restore dan sfc. Mereka tidak berhasil.

Saya tahu saya dapat menginstal ulang sistem, tetapi saya menginginkan solusi yang tidak terlalu menyakitkan.

Untuk memperbaiki, gunakan perintah berikut pada prompt perintah ( dengan hak istimewa )

DISM /Online /Cleanup-Image /RestoreHealth

RestoreHealth: Opsi sakelar ini memeriksa kerusakan penyimpanan komponen, mencatat kerusakan ke file log, dan MEMPERBAIKI kerusakan gambar menggunakan Pembaruan Windows. Ini harus mengambil sekitar 10-15 menit hingga sekitar satu jam untuk menyelesaikan tergantung pada tingkat korupsi.

Saya sedang dialihkan ke tempat parkir ketika resolusi dns gagal

Saya baru menyadari bahwa ketika saya memasukkan URL di browser dan menavigasi ke sana, jika situs tersebut tidak ada, saya dialihkan ke situs parkir yang penuh dengan iklan. Dikonfirmasi bahwa situs tidak ada setelah memeriksa informasi WHOIS (domain tersedia, dll.). Pengaturan rumah saya hanyalah router wifi dengan layanan ADSL, dan perangkat saya melalui koneksi wifi itu.

Tes saya sejauh ini :

  • Navigasi ke http://laksdkajsndkajndkasn.net >> dapatkan 302 redirect ke halaman berbahaya
  • Navigasi ke http://laksdkajsndkajndkasn.net (browser lain) >> dapatkan 302 redirect ke halaman berbahaya
  • Navigasi ke http://laksdkajsndkajndkasn.net (seluler terhubung ke jaringan yang sama) >> dialihkan ke halaman berbahaya
  • Navigasi ke http://laksdkajsndkajndkasn.net (seluler terhubung ke jaringan 3G) >> TIDAK dialihkan ke halaman berbahaya
  • Keriting http://laksdkajsndkajndkasn.net >> menyelesaikan alamat IP tetapi mendapatkan respons 404 dengan panjang konten 0. Buka IP tersebut di browser >> mengalihkan ke situs parkir.
  • dnslookup http://laksdkajsndkajndkasn.net >> Saya melihat IP tersebut di bawah "Jawaban non-otoritatif:"

Saya kira ini bisa menjadi sesuatu yang buruk / berbahaya dalam koneksi / pengaturan / isp saya, tetapi saya sangat menghargai setiap petunjuk untuk memecahkan masalah ini.

Itu akan membuat ISP Anda 'membantu' Anda.

http://whatis.techtarget.com/definition/DNS-redirection

Atau seperti yang ditunjukkan di komentar, server DNS Anda telah disusupi atau pengaturan DNS gateway Anda diubah untuk mengarah ke server DNS yang berbahaya.

Lihat http://www.dcwg.org/ untuk informasi tentang salah satu contoh malware pengubah DNS yang menargetkan router SOHO dan cara memeriksa / menghapus masalah.

Terkadang bukan tentang pembajakan DNS.

Beberapa bilah alat / ekstensi mata-mata memeriksa kode kembali dari halaman yang Anda navigasi dan mengarahkan Anda ke beberapa halaman kesalahan kustom atau halaman parkir jika kode yang dikembalikan bukan 200 (OK).

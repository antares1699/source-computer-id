Keychain dienkripsi secara default sedangkan pengelola kata sandi Firefox tidak dienkripsi secara default. Anda dapat menggunakan kata sandi utama dengan pengelola kata sandi Firefox di tools - options - security tab - use master password tetapi pada akhirnya pengelola kata sandi FF hampir 100% tidak aman tanpa kata sandi utama

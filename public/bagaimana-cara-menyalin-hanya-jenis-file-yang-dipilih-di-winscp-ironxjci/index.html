Bagaimana cara menyalin hanya jenis file yang dipilih di WinSCP?

Saya hanya ingin mengunduh file php, html, css, xml dan js dari server FTP. Bagaimana melakukan ini dengan WinSCP (4.2.4)?

  • Secara umum, untuk tujuan ini, dapat bermanfaat wget terlalu. (Perhatikan itu ada versi untuk windows juga). Baca opsi untuk hanya menyertakan pola file tertentu (*.css ...)

Anda harus menggunakan perintah Select Files dari menu Mark (biasanya terikat ke tombol numpad: +) dan ketik pola kecocokan Anda (yaitu *.php;*.js) ke sana. Anda kemudian dapat mengoperasikan file yang dipilih seperti biasa.

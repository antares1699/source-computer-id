Tap mousepad tidak berfungsi pada lxde stabil debian

Saya memiliki Acer Aspire One KAV60 yang menjalankan debian 7.0 stable dengan lxde sebagai pengelola jendela.

Saya tidak bisa mendapatkan mousepad untuk mendaftarkan klik saat saya mengetuknya.

Saya sudah mencoba menginstal gpointing-device-settings, yang berfungsi untuk beberapa pengaturan seperti menggunakan 2 jari untuk menggulir, tetapi tidak melakukan apa pun saat saya mengaktifkan atau menonaktifkan mousepad tap.

Bantuan apa pun akan sangat dihargai.

Berdasarkan tanggapan dari Larry dan user193782, saya menemukan bahwa ini berfungsi untuk LXDE di Debian stable (wheezy):

$ su
$ cd /etc/X11/Xsession.d
$ vi 99-synclient

File 99-synclient harus berisi:

#! /bin/sh
synclient TapButton1=1

Kemudian buat 99-synclient dapat dieksekusi:

$ chmod 555 99-synclient

Silakan coba langkah-langkah ini:

  1. Buka Terminal Root (Menu> Aksesoris> Terminal Root)

  2. Masukkan kata sandi Administratif

  3. Ketik perintah berikut:

    mkdir /etc/X11/xorg.conf.d
leafpad /etc/X11/xorg.conf.d/synaptics.conf

  4. Salin skrip berikut

    Section "InputClass"
Identifier      "Touchpad"
MatchIsTouchpad "yes"  
Driver          "synaptics"                     
Option          "MinSpeed"              "0.5"
Option          "MaxSpeed"              "1.0"
Option          "AccelFactor"           "0.075"
Option          "TapButton1"            "1"
Option          "TapButton2"            "2"    
Option          "TapButton3"            "3"     
Option          "VertTwoFingerScroll"   "1"     
Option          "HorizTwoFingerScroll"  "1"     
Option          "VertEdgeScroll"        "1"
Option          "CoastingSpeed"         "8"
Option          "CornerCoasting"        "1"
Option          "CircularScrolling"     "1"
Option          "CircScrollTrigger"     "7"
Option          "EdgeMotionUseAlways"   "1"
Option          "LBCornerButton"        "8"   
Option          "RBCornerButton"        "9"     
EndSection

  5. Kemudian tempelkan ke file synaptics.conf

  6. Klik File dan Simpan

  7. Logoff dan login lagi .....

Cara menambahkan tata letak keyboard khusus ke Windows 7

Saya memiliki buku catatan baru yang dilengkapi dengan keyboard Swedia, namun karena bahasa ibu saya adalah bahasa Portugis Brasil, saya kehilangan beberapa huruf.

Saya dapat membeli keyboard baru di Brasil, tetapi karena saya hanya kehilangan 2 huruf, saya memutuskan bahwa akan lebih mudah jika saya dapat membuat tata letak keyboard baru yang berisi huruf-huruf yang saya lewatkan.

Untuk melakukan ini, saya telah membuat tata letak khusus menggunakan Pembuat Tata Letak Keyboard Microsoft , namun saya belum dapat menambahkannya ke daftar keyboard Windows. Dengan kata lain, bagaimana cara memasang / mengimpor keyboard ini di komputer saya?

Cukup menyimpan tata letak saja tidak cukup, Anda harus menginstalnya. Untuk melakukannya, ikuti langkah-langkah berikut:

  1. Buka Project → Build DLL and Setup Package
  2. Saat proses selesai, klik Iya untuk membuka folder tempat file setup berada
  3. Jalankan setup.exe file yang Anda buat untuk menginstal tata letak baru
  4. Terakhir, buka Control Panel → Regional Settings untuk memilih tata letak baru

Mentransfer lisensi Windows 8 dari netbook ke laptop

Saya baru-baru ini merawat diri saya dengan laptop baru dan kemudian menyadari bahwa itu datang dengan Windows 8 Core, bukan Pro .

Namun, beberapa waktu yang lalu ketika pertama kali dirilis, saya membeli Lisensi Windows 8 Pro saat ditawarkan (saya pikir £ 25). Saya menginstal ini di netbook Windows 7 saya untuk dimainkan. Yang saya pikirkan adalah jika saya mengembalikan netbook saya ke Windows 7, dapatkah saya menggunakan lisensi itu untuk memutakhirkan laptop baru saya ke Windows 8 Pro?

Mengingat saya telah menginstal 120+ pembaruan untuk membawanya ke Windows 8.1.

Saya hanya ingin tahu apakah saya bisa mendapatkan jawaban yang menentukan sebelum menghapus netbook saya, karena saya telah membaca banyak situs yang kontradiktif sambil mencoba menemukan jawaban.

Jika itu adalah lisensi ritel dan Anda benar-benar mengembalikan mesin asli ke Windows 7, maka ya, Anda bisa.

Jika Anda membeli lisensi sebagai OEM atau Anda gagal mengembalikan mesin asli kembali ke Windows 7, maka tidak, Anda tidak bisa.

Sesederhana itu.

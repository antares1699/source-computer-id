Tidak ada cara untuk membuka jendela properti tertentu secara langsung. Hanya ada jalan pintas ke layar antarmuka "generik", yaitu ncpa.cpl.

Namun Anda dapat menggunakan netsh dari cmd untuk mengatur atau melihat properti secara langsung. Misalnya, jika Anda ingin menyetel alamat IP "Local Area Connection" (nama default untuk antarmuka jaringan utama) ke 192.168.0.5, dengan subnet 255.255.255.0 dan gateway 192.168.0.1, Anda akan menggunakan:

netsh interface ipv4 set address name="Local Area Connection"
source=static address=192.168.0.5 mask=255.255.255.0 gateway=192.168.0.1

Untuk gambaran umum dari semua opsi yang memungkinkan, gunakan:

netsh interface ipv4 set address /?

Untuk melihat konfigurasi saat ini, Anda dapat menggunakan:

netsh interface ipv4 show addresses "Local Area Connection"

Yang kemudian akan menunjukkan:

Configuration for interface "Local Area Connection"

DHCP enabled:                         No
IP Address:                           192.168.0.5
Subnet Prefix:                        192.168.0.0/24 (mask 255.255.255.0)
Default Gateway:                      192.168.0.1
Gateway Metric:                       0
InterfaceMetric:                      10

SET var=%%F
)
ECHO %var%

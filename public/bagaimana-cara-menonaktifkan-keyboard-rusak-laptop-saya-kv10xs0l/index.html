Bagaimana cara menonaktifkan keyboard rusak laptop saya?

Saya memiliki Alienware M17x R2, dan setelah beberapa tahun penggunaan saya mulai mengalami masalah dengan keyboard: Kadang-kadang, dan tanpa peringatan atau penyebab yang jelas (bahkan jika saya tidak menyentuh komputer), tombol S (dan yang terbaru D) akan mulai menembakkan sinyal yang tak henti-hentinya sampai saya menekan tombol yang menyinggung.

Saya telah menggunakan keyboard USB, yang ukurannya pas dan pas dengan keyboard internal tanpa menyentuh tombol apa pun. Namun, meskipun saya mengetik, keyboard yang rusak akan mengaktifkan aliran sinyalnya.

Saya telah menghubungi Dell tentang hal ini, sebenarnya dua kali. Pertama kali, saya secara blak-blakan diberitahu bahwa mereka tidak mengirimkan keyboard pengganti. Kedua kalinya saya diberitahu bahwa mereka melakukannya, tetapi stok mereka habis dan mereka akan menghubungi saya ketika mereka ada lagi. Mereka belum menghubungi saya sejak itu.

Saya tidak bisa menentukan perangkat mana yang akan dinonaktifkan di pengelola perangkat. Berikut tangkapan layar dari apa yang saya miliki:

Tidak ada perangkat keyboard yang memiliki "Disable" dalam menu konteksnya, hanya "Uninstall" ... jadi Anda akan mengerti jika saya ragu untuk mencoba sendiri.

Ada saran tentang cara memperbaikinya?

Pertama-tama saya akan melihat ke BIOS untuk melihat apakah ada opsi di sana untuk menonaktifkan keyboard (atau mungkin input PS / 2). Ini tentu saja dapat membuat laptop Anda tidak berfungsi jika Anda tidak memiliki keyboard USB sepanjang waktu.

Kasus terburuk, Anda selalu dapat mencoba mencopot pemasangan Standard PS/2 Keyboard (keyboard internal Anda) dan melihat apakah Anda dapat mencegah windows mencoba menginstal ulang driver untuk itu.

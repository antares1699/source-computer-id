Saya punya ini di Ubuntu 10.04 menginstal ekstensi PHP Mongo. Melakukan penelitian dan mencoba ini:

# pear search http
The value of config option cache_dir (/tmp/pear/cache) is not a directory and attempts to 
create the directory failed.

Ketika saya membuat direktori dengan tangan:

mkdir -p /tmp/pear/cache
kesalahan "hilang" dan saya dapat menginstal: "pecl install mongo".

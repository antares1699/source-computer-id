Secara otomatis membuat hyperlink-ify sel angka besar yang berisi URL di Excel 2010

Saya memiliki spreadsheet Excel dengan sekitar 900 sel yang masing-masing berisi URL, yang telah saya tempelkan dari program lain.

Jika saya memilih sel dan menekan F2 , lalu tekan Memasukkan , Excel menambahkan hyperlink ke sel, yang Alamatnya adalah konten sel.

Apakah ada mekanisme di Excel 2010 untuk mengulangi langkah ini untuk ratusan sel saya, tanpa harus melakukan F2 Memasukkan melangkah secara manual?

(Saya tahu bahwa saya bisa mengotomatiskannya di AutoHotKey, tapi saya berharap tidak perlu melakukan itu)

Klarifikasi : Saya secara khusus ingin mengakhiri dengan hyperlink yang ditambahkan ke sel saya yang sudah ada, daripada membuat kolom kedua yang berisi hyperlink.

Hal yang saya coba:

  1. Membuat kolom tambahan sementara, dengan =Hyperlink(A1) di A2, sesuai jawaban pengguna301746, dan kemudian menggunakan Tempel Khusus untuk menyalin dan menempelkan pemformatan terlebih dahulu, dan kemudian nilai di atas A1. Ini tampaknya berfungsi, dalam sel di A1 dll ditampilkan dengan warna biru dan bergaris bawah, sehingga tampak seperti hyperlink, tetapi sebenarnya tidak ada tautan - ini hanya menyalin gaya tampilan hyperlink.

Sebagai alternatif dari kunci otomatis, cukup mudah untuk membuat kode di vba. Kode berikut akan melakukan apa yang Anda inginkan untuk semua sel di kolom A mulai dari baris 2.

Sub add_links()
Dim lRow As Long

    With ActiveSheet
        lRow = .Range("A" & .Rows.Count).End(xlUp).Row
        For Each c In .Range("A2:A" & lRow)
            ActiveSheet.Hyperlinks.add anchor:=c, Address:=c.Value
        Next
    End With

End Sub

Oke, saya telah menemukan serangkaian langkah (sedikit jelek) yang setidaknya berakhir dengan teks hyperlink di kolom asli, menggunakan saran penggunaan fungsi HYPERLINK dari user301746, tetapi hanya dengan satu kolom.

  1. Salin semua sel Excel 2010 yang berisi URL
  2. Tempel URL ke Notepad ++
  3. Di Notepad ++, konversikan semua URL ke rumus HYPERLINK Excel - menggunakan mekanisme yang dijelaskan di jawaban ini
    1. Control-H untuk menampilkan dialog ganti
    2. Ubah Mode Pencarian menjadi Ekspresi Reguler
    3. Masukkan (http.*) sebagai file kriteria pencarian (temukan baris apa pun dengan http di dalamnya, dan simpan dalam variabel \1
    4. Masukkan =HYPERLINK\(\"\1\"\) sebagai file ganti kriteria - ganti apa yang ditemukan pada langkah sebelumnya dengan itu sendiri yang terkandung dalam pemanggilan fungsi HYPERLINK
    5. Menggantikan semua - sekarang semua baris tautan Anda tertutup, baris yang tersisa harus berupa judul
  4. Salin semua teks dari Notepad ++
  5. Tempelkan di atas sel asli di Excel 2010. Sekarang semua sel sel Excel akan menjadi hyperlink, tetapi tidak akan diformat agar terlihat seperti hyperlink
  6. Dengan sel yang ditempel masih dipilih, klik tombol gaya Hyperlink, di bagian Gaya pada tab Beranda di Pita, untuk membuatnya terlihat seperti hyperlink.

Saya mengubahnya sedikit untuk menambahkan masukan untuk kolom mana

Sub add_links_Input_Column()
Dim lRow As Long
Dim ColHead As String

ColHead = InputBox("Enter Column Letter", "Identify Column", [c1].Value)
If ColHead = "" Then Exit Sub

    With ActiveSheet
        lRow = .Range(ColHead & .Rows.Count).End(xlUp).Row
        For Each c In .Range(ColHead & "2:" & ColHead & lRow)
            ActiveSheet.Hyperlinks.Add anchor:=c, Address:=c.Value
        Next
    End With

End Sub

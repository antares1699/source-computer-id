Adblock Chrome memperbarui frekuensi - membiarkan beberapa iklan masuk

Saya menggunakan Adblock Plus [edit: awalnya menulis "pro"] untuk chrome, dan kadang-kadang saya telah mencatat bahwa beberapa iklan akan lolos. Setiap kali ini terjadi, memperbarui filter secara manual telah memperbaiki masalah, dan hampir selalu 3 hari atau lebih dari tanggal. Apakah ada cara untuk memaksa pembaruan lebih sering dari setiap 5 hari? Atau adakah solusi lain yang memungkinkan saya untuk tidak pernah melihat iklan?

Tidak ada tempat untuk mengubah ini dalam pengaturan addon. Jika penting, saya menggunakan easylist dan easyprivacy .

[edit 20 Des 2013] Saya beralih dari adblock plus untuk chrome ke adblock biasa, dan hal yang sama terjadi. Setiap beberapa hari, filter akan menjadi sangat usang sehingga iklan akan lolos. Tidak ada tempat bagi saya untuk menyesuaikan setelan filter yang saya ketahui. Adakah yang dapat saya lakukan untuk terus memperbarui filter ini sehingga saya tidak akan melihat iklan?

Frekuensi pembaruan ditentukan oleh langganan filter Anda.

Dari pembaruan filter adblock tidak terlalu otomatis :

Lamanya waktu kami memperbarui daftar filter ditentukan oleh daftar itu sendiri, Jika Anda melihat EasyList, misalnya, ( tautan ), kamu akan lihat di baris 6 yang kedaluwarsa setiap 4 hari. AdBlock membaca ini dan memahami bahwa itu hanya perlu diperbarui setiap 4 hari. Begitulah Server EasyList tidak kelebihan beban.

Daftar EasyList yang ditautkan di atas dimulai dengan baris berikut:

[Adblock Plus 2.0]
! Checksum: EtxsLx21pEe0QkBxTJFxHg
! Versi: 201312211410
! Judul: EasyList
! Terakhir diubah: 21 Des 2013 14:10 UTC
! Kedaluwarsa: 4 hari (frekuensi pembaruan)

Jika Anda berlangganan EasyList ( situs ), artinya filter akan diperbarui hanya setiap hari ke-4. Alasannya hanya untuk menghemat bandwidth di server web mereka.

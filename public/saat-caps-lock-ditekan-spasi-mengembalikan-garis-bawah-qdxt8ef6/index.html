Saat caps lock ditekan, spasi mengembalikan garis bawah

Pada dasarnya, yang saya cari adalah: kapan Kunci huruf kapital pada yang saya inginkan Ruang untuk mengembalikan _ dan saya ingin Bergeser + Ruang untuk mengembalikan .

Detailnya: Ini semacam kelanjutan dari pertanyaan ini Saya bertanya tentang pemetaan ulang file Bergeser + Ruang kunci untuk menjadi garis bawah ( _ ). Saya mendapat jawaban yang bagus tetapi menurut saya itu agak rumit terutama ketika saya memiliki kelingking malas yang tidak suka mengambil sendiri dari Bergeser tepat waktu bagi saya untuk menekan Ruang kunci. Jadi sekarang saya bertanya-tanya apakah ada yang tahu solusi untuk membuatnya jadi Ruang key bisa memetakan ulang untuk menggarisbawahi kapan Kunci huruf kapital dihidupkan. Dan, lebih baik lagi, jika Kunci huruf kapital aktif dan saya menekan Bergeser + Ruang kombinasi itu akan memaksa karakter spasi. Apakah itu masuk akal? Silakan dan minta klarifikasi.

Oh, ngomong-ngomong, solusi untuk pertanyaan yang saya tanyakan sebelumnya mungkin memiliki kekuatan untuk melakukan ini. Ini disebut AutoHotKey . Jika itu berhasil, maka saya sangat senang dengan itu! Terima kasih!

Skrip AutoHotkey:

+Space::
state := GetKeyState("CapsLock", "T")
if state
    SendPlay {Space}
else
    SendPlay _
return

Space::
state := GetKeyState("CapsLock", "T")
if state
    SendPlay _
else
    SendPlay {Space}
return

Apa fungsinya:

  • Jika Kunci huruf kapital aku s tidak aktif:

    • Ruang menyisipkan ruang.

    • Bergeser + Ruang menyisipkan garis bawah.

  • Jika Kunci huruf kapital aktif:

    • Ruang menyisipkan garis bawah.

    • Bergeser + Ruang menyisipkan ruang.

jika kamu mau Ruang berperilaku normal saat Kunci huruf kapital aku s tidak aktif, hilangkan saja blok pertama.

Cara Penggunaan:

  1. Unduh dan instal versi terbaru.

  2. Simpan skrip di atas sebagai underscore.ahk, menggunakan editor teks favorit Anda.

  3. Klik dua kali file tersebut untuk menjalankan skrip.

  4. Jika Anda mau, salin skrip (atau tautan ke sana) di folder Startup.

Bagaimana itu bekerja:

  • +Space:: ... return mengeksekusi ... ketika Bergeser + Ruang ditekan.

    +Space:: ... return mengeksekusi ... ketika Ruang ditekan (tanpa pengubah).

  • state := GetKeyState("CapsLock", "T") memeriksa apakah Kunci huruf kapital aktif.

    • Jika ya, state adalah true dan blok setelah if state dijalankan.

    • Jika tidak, state adalah false dan blok setelah else dijalankan.

  • SendPlay X menyisipkan karakter (urutan) X.

    Anda tidak dapat menggunakan Send di sini, karena ini akan memicu skrip secara rekursif, menghasilkan loop tanpa batas.

  • return mengakhiri skrip pada titik saat ini.

    return kedua adalah opsional, yang pertama tidak. Tanpa yang terakhir, Bergeser + Ruang akan menyebabkan kedua blok untuk dieksekusi.

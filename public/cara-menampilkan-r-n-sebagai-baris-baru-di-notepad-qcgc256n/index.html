Cara menampilkan \ r \ n sebagai baris baru di Notepad ++

Saya menjelajahi file log di notepad ++. File memiliki \ r \ n di mana baris baru harus dimulai, tetapi notepad hanya menampilkan karakter \ r \ n:

[Header]\r\n2011-04-23 13:14:53,867 [1] DEBUG Themis.Program Enter application
2011-04-23 13:14:57,505 [1] DEBUG Themis.Program Exit application
[Footer]\r\n[Header]\r\n2011-04-23 13:23:15,690 [1] DEBUG Themis.Program Enter application

Saya ingin menampilkannya seperti ini:

[Header]
2011-04-23 13:14:53,867 [1] DEBUG Themis.Program Enter application
2011-04-23 13:14:57,505 [1] DEBUG Themis.Program Exit application
[Footer]
[Header]
2011-04-23 13:23:15,690 [1] DEBUG Themis.Program Enter application

Apakah mungkin melakukan ini. Apakah mungkin ada aplikasi lain yang bisa melakukan itu?

  • \ r \ n adalah 0x0D dan 0x0A, atau 0x0D0A. Di Unicode, ini adalah CR (U + 000D) diikuti oleh LF (U + 000A). Bergantung pada bahasa pemrograman Anda, keberadaan garis miring terbalik dan karakter "\ r \ n" mungkin menunjukkan bahwa Anda memiliki beberapa opsi yang disetel dengan tidak benar. Misalnya, di Python, jika string Anda ditandai sebagai "mentah", kode baris baru Anda tidak akan berlaku.
    — Dane

Anda dapat melakukannya dalam 2 langkah

tekan CTRL + H untuk membuka jendela Ganti

1) Pertama, pastikan Mode Pencarian dalam mode Normal. Kemudian ganti \ r \ n dengan beberapa kata acak yang tidak berarti seperti: abcdefg (yang Anda yakin tidak ada dalam teks Anda)

2) Kemudian ubah mode Pencarian ke Diperpanjang atau Ekspresi reguler mode dan ganti kata ini dengan \ n

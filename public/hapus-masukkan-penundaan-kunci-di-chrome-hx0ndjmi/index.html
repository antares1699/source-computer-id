Hapus masukkan penundaan kunci di Chrome

Saya sedang menguji formulir HTML, dan ini mengharuskan saya menyegarkan halaman untuk melihat perubahan yang diperbarui.

Saya ingin menyegarkan halaman tanpa mengirim data POST apa pun, tetapi jika saya menekan F5 atau tombol segarkan ini yang muncul:

Jadi sebagai gantinya, saya harus menyorot bilah URL, lalu tekan Memasukkan .

Tidak apa-apa, tetapi ada penundaan besar-besaran untuk dapat menekan Enter dan menyegarkan halaman.

Cobalah:

  1. Buka halaman mana pun di Chrome
  2. Klik kotak URL
  3. Spam tombol Enter beberapa kali dan pastikan halaman tidak dimuat ulang

Apakah ada cara untuk menghilangkan penundaan ini?

Browser lain tidak memiliki penundaan ini, dan saya juga tidak ingat versi Chrome sebelumnya melakukan ini.

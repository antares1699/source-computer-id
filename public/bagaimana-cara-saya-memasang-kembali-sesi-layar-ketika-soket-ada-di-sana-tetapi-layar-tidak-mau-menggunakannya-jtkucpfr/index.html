Saya telah melihat pertanyaan ini dan pertanyaan ini , tapi tidak satupun dari mereka berlaku untuk saya. Saya memiliki file socket di / var / run / screen dan saya memilikinya - tetapi screen -list tidak mengenalinya dan mengizinkan saya untuk terhubung. Bagaimana cara memulihkan sesi ini?

Anda dapat menyetel variabel lingkungan SCREENDIR ke /var/run/screen. Anda kemudian dapat membuat daftar dan menyambung.

Dengan asumsi sh shell yang kompatibel:

export SCREENDIR="/var/run/screen"

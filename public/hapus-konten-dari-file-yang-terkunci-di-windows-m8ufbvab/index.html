Saya menemukan solusi terbaik untuk saya adalah Powershell> Clear-Content filename.log https://technet.microsoft.com/en-us/library/ee156808.aspx

Cmdlet Clear-Content memungkinkan Anda untuk menghapus konten file tanpa menghapus file itu sendiri. Misalnya, Anda menjalankan perintah ini:

Clear-Content c:\scripts\test.txt

Ketika Anda menjalankan perintah itu, file Test.txt akan tetap berada di folder C: \ Scripts; tidak akan ada data apa pun di dalam file.

Karakter wildcard dapat digunakan dengan Clear-Content. Perintah ini menghapus konten file apa pun di C: \ Scripts yang nama filenya dimulai dengan huruf E:

Clear-Content c:\scripts\e*

Anda tidak terbatas hanya menghapus file teks. Perintah berikut menghapus semua data di lembar bentang Excel:

Clear-Content c:\scripts\test.xls

Dan perintah ini menghapus isi dokumen Word C: \ Scripts \ Test.doc:

Clear-Content c:\scripts\test.doc

Bagaimana cara memberi diri saya hak admin ke server sql, karena lupa kata sandi?

Saya telah kehilangan semua passwords untuk contoh SQL Server, bagaimana cara memberi diri saya akses admin lagi?

Jalankan SSMS sebagai Administrator (yaitu, dengan hak istimewa yang ditingkatkan di server). Ini harus memiliki hak sysadmin, dan Anda kemudian dapat menemukan login Anda dan menetapkan hak yang diperlukan.

Hampir menulis "celana ketat" di sana. Anda mungkin dapat menetapkannya juga, jika Anda menjalankan sebagai admin di server.

Bagikan

8

  1. Buka Pengelola Konfigurasi

  2. Hentikan instance server

  3. Kanan, klik >> properti >> lanjutan >> parameter startup

  4. Tambahkan "-m;" tepat di awal string memastikan tidak ada spasi di manapun dalam string

  5. Mulai ulang server

  6. sqlcmd -S localhost dari prompt perintah

  7. EXEC sp_addsrvrolemember 'domain\username', 'sysadmin'; GO

  8. exit

  9. Pastikan dapat terhubung dari perintah prompt sqlcmd -S localhost -E (jika masuk sebagai pengguna tersebut ke windows)

  10. Hapus "-m;" dari parameter startup

  11. Mulai ulang server database

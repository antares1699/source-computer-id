Bagaimana cara menyalin file di jaringan berbagi menggunakan login yang berbeda dengan PowerShell atau file batch?

Saya memiliki file di lokasi bersama yang ingin saya salin ke hard drive lokal saya. Namun saya tidak akan menjalankan sebagai pengguna yang memiliki akses ke share. Jadi saya ingin dapat mengotentikasi dengan login yang berbeda (yang memang memiliki izin) dan menyalin file.

Adakah yang bisa saya lakukan untuk mencapai ini?

Saya mencoba perintah ini (dalam file ps1):

runas /user:domain\username "copy-file $shareloc $destloc"

Saya mendapatkan kesalahan ini:

RUNAS ERROR: Unable to run - copy-file \\share\some\dir\file.exe C:\dir\file.exe
2: The system cannot find the file specified.

Dalam file batch, Anda dapat terhubung ke share dengan opsi pengguna:

net use x: \\share\some /user:username password

kemudian lakukan perintah salin seperti biasa (baik menggunakan huruf drive, atau menggunakan UNC seperti pada pertanyaan), lalu putuskan sambungan dari berbagi dengan

net use x: /d

(Anda dapat meletakkan * di tempat kata sandi yang akan ditanyakan.)

Penafian: Saya tidak yakin apakah ini berfungsi di domain.

skrip bash shell yang menambahkan output dari perintah

Katakanlah saya memiliki perintah bernama foo yang mencetak nomor ke layar saat dipanggil:

$ foo
3

Katakan juga saya memiliki perintah lain bernama bar yang mencetak nomor lain ke layar saat dipanggil:

$ bar
5

Saya ingin menulis skrip shell yang secara aritmatika akan menambahkan output foo dan bar (3 + 5 = 8). Bagaimana saya melakukannya? (Keluaran dari perintah tidak diketahui sebelumnya. Mereka kebetulan 3 dan 5 terakhir kali dijalankan. Mereka bisa saja sesuatu yang lain.)

Gunakan bash's let untuk mengevaluasi ekspresi aritmetri.

#!/bin/bash
a=`echo 3`
b=`echo 5`

let c=$a+$b
echo $c

Gantilah panggilan ke echo dengan panggilan program Anda.

Alternatif untuk let adalah dengan menggunakan sintaks kurung ganda:

(( c = $(foo) + $(bar) ))

atau

echo $(( $(foo) + $(bar) ))

atau menggunakan variabel, Anda dapat menghilangkan tanda dolar di sisi kanan persamaan:

(( c += $(foo) + num ))

(yang juga menggambarkan tugas tambahan)

Jika Anda menggunakan non-integer, Anda dapat menggunakan bc:

echo "$(foo) + $(bar)" | bc

atau

c=$(echo "$(foo) + $(bar)" | bc)

Salah satu keuntungan menggunakan tanda kurung ganda adalah Anda dapat meletakkan spasi di sekitar operan dan operator agar lebih mudah dibaca:

(( c = ( a + b ) * ( i - j ) ))

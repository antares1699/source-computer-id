Per Pusat bantuan Dropbox , Dropbox tidak menyinkronkan file yang sedang digunakan, jadi file PST Anda mungkin tidak akan disinkronkan secara waktu nyata. Mereka hanya akan menyinkronkan saat pengguna menutup Outlook. Namun, ketika itu melakukan sinkronisasi itu akan hanya melakukan penyalinan diferensial (hanya perubahan pada file yang disalin), jadi tidak perlu mengupload ulang seluruh file setiap saat.

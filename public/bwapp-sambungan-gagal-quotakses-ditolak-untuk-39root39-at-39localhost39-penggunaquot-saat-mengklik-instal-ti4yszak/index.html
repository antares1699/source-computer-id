bWAPP: Sambungan gagal: "Akses ditolak untuk 'root' @ 'localhost' pengguna" saat mengklik instal

Saya telah mengunduh bWAPP terbaru dari sourceforge.net dan mengikuti pedoman INSTALL.txt untuk menginstalnya pada instalasi Kali Linux yang bersih (2017.3).

Saya menggunakan pengaturan koneksi MySQL berikut ini:

// Database connection settings
$db_server = "localhost";
$db_username = "root";
$db_password = "";
$db_name = "bWAPP";

Namun, menekan tombol "Klik di sini untuk menginstal bWAPP." di http: //localhost/bWAPP/install.php menghasilkan pesan kesalahan berikut: "Sambungan gagal: Akses ditolak untuk pengguna 'root' @ 'localhost'".

Saya telah mencari di Google dan tampaknya orang lain memiliki masalah serupa, tetapi tidak ada solusi yang didokumentasikan di mana pun.

Apa yang saya lakukan salah?

Alasannya adalah backend database MariaDB yang diinstal dengan Kali Linux:

MariaDB adalah fork yang dikembangkan komunitas dari database relasional MySQL sistem manajemen yang dimaksudkan untuk tetap bebas di bawah GNU GPL.

Backend database ini tidak mengizinkan root@localhost akses pengguna dari aplikasi.

Saya memperbaiki masalah ini dengan membuat pengguna database baru, bwappu, dengan hak istimewa yang diberikan untuk database bwapp:

GRANT ALL PRIVILEGES on bwapp.* to 'bwappu'@'localhost';

Juga, lihat posting ini: https://stackoverflow.com/questions/28068155/access-denied-for-user-rootlocalhost-using-password-yes-after-new-instal

Sedikit lebih detalisasi untuk jawaban yang benar oleh Nicolas Lykke Iversen:

Pergi ke terminal:

root@kali: mysql -u root -p
enter password 'toor'
MariaDB [(none)]> use mysql;
MariaDB[(mysql)]> create user 'username'@'localhost' identified by 'my_password';
MariaDB[(mysql)]> grant all privileges on bWAPP.* to 'username'@'localhost' identified by 'my_password';

Buka /var/www/html/bWAPP/admin/settings.php dan setel:

$db_username = "username";
$db_password = "my_password";

Lalu pergi ke http: //localhost/bWAPP/install.php dan instal.

PS: nama pengguna & my_password - Anda dapat menggunakan kata apa pun yang Anda suka

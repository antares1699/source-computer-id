Bagaimana cara mengatur posisi jendela awal saya di Cocoa emacs?

Saya memiliki set berikut di file .emacs saya:

 (if (window-system)
   (set-frame-height (selected-frame) 60)
   (set-frame-position (selected-frame) 50 30))

Ini mengatur tinggi bingkai dengan benar tetapi bukan posisi bingkai. Saya sedang berlari:

GNU Emacs 23.2.1 (i386-apple-darwin9.8.0, NS apple-appkit-949.54)

EDIT:

Kode yang ditawarkan di bawah ini tidak khusus untuk Mac. Anda harus mengeditnya (cukup ake salah satu referensi *win*. Inilah yang saya gunakan untuk pekerjaan itu. Sesuaikan sesuai keinginan Anda!

(setq initial-frame-alist
      `((background-color . ,(face-background 'default))
        (foreground-color . ,(face-foreground 'default))
        (horizontal-scroll-bars . nil)
        (vertical-scroll-bars . nil)
        (menu-bar-lines . 0)
        (top . 50)      ;; This is overridden by my-center-frame later.
        (left . 400)        ;; This is overridden by my-center-frame later.
        (cursor-color . "red")
        (mouse-color . "green")))

Anda menggunakan if secara tidak benar, kode yang Anda tempelkan hanya akan menjalankan formulir pertama jika kondisinya benar (lihat sini ). Coba gunakan makro when sebagai gantinya:

(when (window-system)
  (set-frame-height (selected-frame) 60)
  (set-frame-position (selected-frame) 50 30))

Ini akan menjalankan setiap formulir setelah kondisi jika kondisinya benar.

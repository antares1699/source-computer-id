Menentukan spesifikasi adaptor grafis Intel HD

Saya akan membeli laptop baru dan salah satu hal yang akan saya gunakan adalah pemodelan dan rendering 3D; bukan barang animasi yang berat, tetapi lebih banyak model arsitektur Sketchup dan rendering Artlantis.

Yang saya coba cari tahu adalah spesifikasi adaptor grafisnya sehingga saya dapat membandingkannya dengan sistem lain. Sayangnya hanya tertulis Grafis Intel HD 4000 yang tidak terlalu berguna untuk perbandingan.

Adakah ide tentang bagaimana saya dapat menemukan spesifikasi yang lebih detail?

Hanya untuk menguraikan laptop yang saya lihat, saya menganggap pertanyaan ini adalah macbook pro 13 "retina.

Intel HD Graphics 4000 adalah solusi grafis terintegrasi dari prosesor Core i3 / i5 / i7 generasi ke-3. Dengan demikian, ia berbagi RAM inti CPU.

Semua CPU memiliki 16 unit eksekusi, bandwidth memori 25,6 GB / s dan dapat menggunakan hingga 1.720 MB memori sistem. Jam inti GPU maksimum naik hingga 1300 MHz, tetapi bervariasi dari model ke model. ( sumber ).

Untuk daftar CPU (seluler dan desktop) yang menggunakan Intel HD Graphics 4000 dan spesifikasinya, lihat:

  • Prosesor Intel® Core ™ i7 Extreme Generasi ke-3
  • Prosesor Intel® Core ™ i7 Generasi ke-3
  • Prosesor Intel® Core ™ i5 Generasi ke-3
  • Prosesor Intel® Core ™ i3 Generasi ke-3

Untuk membandingkan kinerja berbagai GPU seluler, Anda dapat memeriksa Perbandingan Kartu Grafis Laptop atau Kartu Grafik Seluler - Daftar Tolok Ukur di Notebookcheck.net Tech.

Perlu diingat bahwa tidak mungkin untuk menguji semua GPU menggunakan CPU, RAM yang sama, dll. (Intel HD Graphics 4000 itu sendiri hanya tertanam dalam beberapa CPU seluler), jadi Anda harus mengambil tolok ukur dengan sebutir garam. .

Fakta : Grafis seri Intel "HD4000" terintegrasi ke dalam file Jembatan Ivy generasi CPU. Untuk mengetahui kemampuan grafis prosesor terintegrasi Anda, Anda harus memahami apa sebenarnya prosesor Anda aku s .

Fakta : CPU generasi Ivy Bridge memiliki nomor model yang cocok dengan pola berikut: Intel Core i#-3###<suffix>, di mana masing-masing # merupakan angka. # pertama adalah 3, 5, atau 7. Tiga sisanya dapat berupa angka yang sangat bervariasi. Ini adalah satu atau dua huruf dan menunjukkan fitur khusus dari CPU tertentu, seperti kemampuan untuk melakukan overclock (K); fakta bahwa prosesor cocok untuk penggunaan seluler yaitu laptop (M); atau fakta bahwa prosesornya adalah quad-core, untuk membedakannya dari model dual-core (Q).

Langkah 1

Tentukan tepat Model CPU. Anda harus dapat mengisi keempat dari # kosong di templat nomor model di atas, serta sufiks apa pun (sufiks tidak diperlukan).

Langkah 2

Pergilah sini dan sini dan mencari fitur model spesifik Anda. Perhatikan bahwa sedikit variasi dalam nomor model dapat meningkatkan atau menurunkan versi berbagai fitur GPU.

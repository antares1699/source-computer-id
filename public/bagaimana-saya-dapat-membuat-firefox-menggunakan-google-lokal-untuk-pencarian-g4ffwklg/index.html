Bagaimana saya dapat membuat Firefox menggunakan Google lokal untuk pencarian?

Saya tinggal di Inggris dan sangat menjengkelkan karena hasil Google selalu Amerika. Bagaimana cara mengonfigurasi Firefox untuk menggunakan google.co.uk daripada google.com?

Pergi ke halaman ini dan temukan Google UK dari daftar. Klik tautan itu untuk menginstal google.co.uk dan mengkonfirmasi penambahannya. Mesin baru akan segera muncul di bilah pencarian. Pilih dari bilah pencarian dan selesai.

Anda juga bisa memeriksanya Halaman Bantuan Mozilla untuk solusi alternatif.

Anda juga dapat menyetel mesin telusur default yang digunakan saat Anda mengetik kueri penelusuran di bilah alamat.

Masukkan alamat "about: config", (Jika ini pertama kalinya Anda menggunakan ini, Anda akan diperingatkan bahwa Anda dapat merusak barang-barang di sini)

Masukkan "keyword.url" di kotak "filter". Klik dua kali hasil "keyword.URL", ubah string kueri penyedia pencarian dan tekan OK, misalnya: http://google.co.uk/search?q=

Saya belum menemukan cara untuk mengubah pencarian google halaman muka default firefox, yang masih mengganggu saya (saya baru saja mengganti halaman muka, kecuali sekarang itu adalah satu-satunya cara untuk mengakses tombol sesi pemulihan).

Saya perlu menggunakan pencarian Google lokal Ukraina saya, dan saya tidak dapat menemukan skrip yang sesuai di antara plugin pencarian Google dari Mycroft. Parameter "keyword.url" juga tidak ada di daftar pengaturan Firefox saya (Firefox v49).

Saya telah berhasil menyelesaikan masalah dengan menghapus semua cookie dari google.com. Saya menemukan resep ini di artikel berikut:

Cara memperbaiki pencarian Firefox untuk menggunakan pencarian Google lokal

Penulis menyarankan untuk menghapus hanya cookie bernama PREF. Saya tidak menemukannya di komputer saya, jadi saya menghapus semua cookie - dan itu membantu.

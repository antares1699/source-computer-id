Bagaimana cara berbagi file dari Windows ke Mac OS X?

Saya punya laptop Windows dan laptop Mac lainnya. Saya bertanya-tanya bagaimana saya harus mengatur Mac saya agar dapat membaca file dari laptop Windows saya?

Katakanlah saya memiliki file C:\test.txt dan saya ingin dapat membuka file ini di MacBook Pro saya. Apakah ini mungkin?

Mac OS X dapat terhubung ke jaringan bersama pada mesin Windows. Buat folder bersama di Windows, lalu buka Finder di Mac OS X dan buka menu Buka → Hubungkan ke Server untuk menghubungkan ke share.

Pada dasarnya, Anda hanya dapat mengakses file dan folder yang dibagikan melalui Windows. Ini untuk XP . Di Windows 7 ini sedikit berbeda saya kira (belum menggunakannya). Ini a Tutorial video Microsoft untuk Windows 7 .

Pertama, aktifkan berbagi file sederhana di Windows, dengan masuk ke Alat - Opsi Folder dari folder mana pun dan mengaktifkan kotak centang berikut:

Kemudian klik kanan folder di Windows yang ingin Anda bagikan, klik Berbagi dan Keamanan , dan aktifkan dua kotak centang berikut:

Dari Mac Anda, di Finder, buka Pergi -> Hubungkan ke Server atau tekan Cmd + K dan masukkan nama komputer Windows. Atau Anda mungkin bisa melihatnya di sidebar Finder Anda, lalu Anda cukup mengkliknya dan itu akan mencantumkan semua folder bersama:

Anda kemudian dapat menelusuri file dalam folder ini dan juga menulis ke file tersebut.

Catatan Keamanan: Saya tidak akan membagikan seluruh drive C: / Anda karena ini dapat menyebabkan perilaku yang tidak terduga. Bayangkan seseorang ada di jaringan Anda dan hanya menghapus file penting dari C: / tanpa Anda menyadarinya. Sebaliknya, bagikan hanya satu folder yang dapat diakses Mac Anda dan orang lain.

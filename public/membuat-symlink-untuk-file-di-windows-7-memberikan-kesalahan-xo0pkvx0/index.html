Membuat symlink untuk file di Windows 7 memberikan kesalahan

Saya ingin membuat symlink ke file menggunakan command prompt di Windows 7. File tersebut terletak di partisi E dan saya ingin linknya ada di partisi C. Saya mencoba perintah ini: C:\>mklink /d c:\11.txt e:\Test\11.txt dan dikatakan bahwa tautan simbolis berhasil dibuat tetapi jika saya mengklik tautan tersebut saya mendapatkan pesan kesalahan ini: 'The directory name is invalid'

Bagaimana cara yang benar untuk membuat symlink untuk file dari partisi yang berbeda?

Dengan menggunakan mklink /D Anda menyuruhnya membuat tautan simbolis direktori dan D:\test\11.txt bukanlah direktori. Jadi jika Anda mencoba mengakses direktori C:\11.txt itu akan memberi Anda kesalahan.

Tinggalkan saja /D dan itu akan bekerja.

Mengapa garis miring terbalik ditampilkan sebagai "₩" (tanda Won Korea, "W" + coret) di Notepad ++? [duplikat]

Saya telah menjelajahi internet selama lebih dari 2 jam mencoba menemukan solusi yang berfungsi untuk menyelesaikan masalah ini. Saya mencoba melakukan pelarian luar angkasa dalam file .css di Notepad ++ dan terus menampilkan benda aneh W ini (yang saya temukan adalah simbol mata uang Korea untuk "Won").

Saya telah mencoba semua perbaikan yang jelas dan disarankan yang dapat saya temukan - menginstal ulang Notepad ++, mengubah pengaturan bahasa, dll. Ini hanya melakukan ini di file CSS dan hanya melakukannya di Notepad ++. Hal aneh lainnya adalah bahwa hasil "Temukan" (Cntrl + F) untuk "\" menunjukkan ini sebagai satu-satunya contoh dalam file.

Saya ingin itu ditampilkan

#status.En\ panne{...}

Dua pertanyaan saya adalah:
1) Apakah ada cara untuk menyelesaikannya?
2) Apakah hanya meninggalkannya di sana masih berfungsi sebagai pelarian luar angkasa?

Mampu mereproduksi masalah, tidak yakin bagaimana ini terjadi:

Bisakah saya SSH dari Mac ke Linux, dengan jendela Linux terbuka di Mac?

Ini adalah pertanyaan tentang dukungan X Windows untuk Mac.

Saya kadang-kadang ssh ke komputer Linux, dan memulai perangkat lunak di sana.

Perangkat lunak ini membutuhkan antarmuka grafis (TAMPILAN).

Saya ingin tahu apakah mungkin untuk mengarahkan semua output grafis dari mesin Linux ke komputer Mac lokal saya yang saya gunakan untuk sshing ke mesin Linux.

Anda dapat melakukan X-tunnel melalui ssh di mac; Semua mesin OS X memiliki server X yang diinstal secara default. Anda perlu menggunakan perintah ssh -X [hostname] untuk mengaktifkan X-tunnel, lalu memulai program melalui shell. Biasanya, Anda tidak perlu menentukan tampilan lokal Anda; program harus ditampilkan secara lokal secara otomatis.

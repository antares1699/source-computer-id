Bagaimana cara mengubah ukuran volume NTFS untuk mengisi partisi?

Karena hanya memiliki file ISO Windows, saya menginstal Windows XP ke mesin virtual, dan memindahkannya berhasil ke partisi fisik di komputer saya.

Karena saya melakukan salinan partisi langsung, dari parisi virtual ke fisik, Windows mengira ukurannya masih 10GB, tetapi partisi tempatnya adalah 59GB.

Apakah ada cara atau hack untuk membuatnya mengenali seluruh partisi? Tanpa kehilangan data tentu saja, menghormati ext4 saya atau partisi linux lain yang tidak diketahui Windows XP.

Diskpart cukup menolak perintah extend di atasnya.

EDIT: Saya juga mendengar orang mengalami hal yang sama, saat mengubah ukuran partisi dengan GParted.

Anda tidak dapat mengubah ukuran partisi yang dipasang. Itu sebabnya diskpart menolak extend.

Anda perlu mem-boot live CD atau flashdisk Ubuntu.

Dan jalankan ntfsresize

Berikut sedikit tutorialnya: http://linuxtechres.blogspot.hu/2007/08/how-to-use-ntfsresize-from-command-line.html

Pada dasarnya dikatakan:

Anda perlu melakukan perintah berikut:

  • fdisk -l untuk mengetahui partisi mana yang harus disentuh.
  • ntfsresize --info /dev/<parition> untuk mendapatkan info partisi.
  • ntfsresize --no-action --size <new volume size> <partition> untuk melakukan dry run. (Anda dapat menggunakan unit dalam ukuran misalnya 12000M)
  • ntfsresize --size <new volume size> <partition> untuk melakukan pengubahan ukuran. Ini akan menjadwalkan pemeriksaan chkdsk jadi boot Windows untuk menyelesaikannya.

Anda harus berada di grup disk untuk menjalankan tindakan atau menjalankannya sebagai root.

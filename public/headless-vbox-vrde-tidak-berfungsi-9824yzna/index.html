Headless VBox - VRDE tidak berfungsi

Saya telah mengimpor mesin yang ada pada VirtualBox 4.1.6r74727 yang baru diinstal, berjalan pada kotak Ubuntu Server 10.04. OS tamu adalah Windows Server 2003.

Selanjutnya, saya telah memodifikasi VM sebagai berikut (10.20.30.40 mewakili alamat IPv4 publc asli dari mesin host):

VBoxManage modifyvm "2003" --vrdeaddress 10.20.30.40 --vrde on --vrdeport 3389 --vrdemulticon on --vrdeauthtype external

Setelah itu, saya memulai VM dengan yang berikut:

VBoxManage startvm "2003" --type headless --vrde=on

Ketika saya mengeluarkan VBoxManage showvminfo "2003", saya mendapatkan yang berikut ini:

VRDE:            enabled (Address 10.20.30.40, Ports 3389, MultiConn: on, ReuseSingleConn: off, Authentication type: external)
Video redirection: disabled

Namun, port 3389 tetap ditutup pada 10.20.30.40, dan di localhost dan saya tidak dapat terhubung ke VRDE. Apa alasannya dan bagaimana pendekatan saya untuk memperbaikinya?

Bagaimana cara memulai Chrome dengan hanya beberapa ekstensi yang diaktifkan secara selektif?

Saya mencoba mencari cara untuk membuat pintasan atau file batch, untuk meluncurkan Google Chrome hanya dengan ekstensi "Ad-block" diaktifkan tetapi semua ekstensi lainnya dinonaktifkan.

Saya memiliki pintasan untuk Chrome dengan semua ekstensi diaktifkan (untuk penggunaan biasa) dan pintasan dengan "chrome.exe --disable-extensions" ketika saya perlu menjalankan program yang membutuhkan memori sebanyak yang saya bisa berikan.

Sekarang saya ingin pintasan sehingga saya dapat menjalankan Chrome dengan hanya Ad-Block diaktifkan, karena menjelajah Chrome yang bersih mengganggu dengan semua penambahan yang bermunculan.

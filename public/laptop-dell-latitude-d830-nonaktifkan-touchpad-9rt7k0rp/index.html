Panel sentuh benar-benar menyakitkan karena sepertinya tangan saya terangkat saat saya mengetik. Saya pernah menggunakannya touchpad, saya lebih suka menggunakan pointer jari di keyboard antara huruf G-H-B.

Apakah ada cara untuk menonaktifkan touchpad?

Apakah Anda memiliki ikon "Dell Touchpad" di System Tray, di dekat jam?

Jika ya,

  • Klik kanan padanya dan pilih "Touchpad Properties"
  • Klik "Device Select"
  • Setel "Touchpad" ke "Nonaktifkan"

Jika Anda tidak memiliki ikon tersebut, Anda dapat menginstal aplikasi Dell Touchpad dari situs web Dell (tautan sini ).

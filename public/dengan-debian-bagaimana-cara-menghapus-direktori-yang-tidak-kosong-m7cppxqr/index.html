Dengan Debian, bagaimana cara menghapus direktori yang tidak kosong?

Di Debian, saya ingin menghapus subdirektori dari direktori home saya. Ketika saya mencoba untuk menghapusnya (dengan sudo), saya mendapatkan pesan yang mengatakan bahwa direktori tersebut tidak kosong, tetapi saya tidak dapat melihat file apa pun di dalam direktori dengan ls -l.

Bagaimana cara menghapus direktori ini?

Mencoba

ls -a -l

Untuk menampilkan file tersembunyi, lalu hapus file tersebut.

Atau, untuk memaksa direktori dan semua yang ada di dalamnya untuk dihapus, coba:

rm -r directory

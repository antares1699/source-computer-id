Vim: Cara menghapus buffer vim ketika Anda keluar dari file di vim

Saya membuka 15 file di vim.

vim -p *.php

Saya menutup satu file tanpa menyimpan

:q

Kemudian setelah menutup beberapa saya mendapatkan E173: 15 more files to edit ini.

Tampaknya keluar dari file tidak menghapusnya dari buffer.

Adakah cara agar saya tidak mendapatkan kesalahan ini ketika saya keluar dari file?

Saat ini saya harus menggunakan qall untuk keluar.

Terima kasih sebelumnya.

14

Anda tidak quit file; untuk "menutup" file, Anda delete atau wipe penyangga, sedangkan Anda quit editor itu sendiri. Anda menggunakan perintah berhenti-editor (q) untuk menutup satu buffer, itulah sebabnya Anda mendapatkan pesan kesalahan.

Berikut adalah perintah yang perlu Anda ketahui:

  • :bd menutup satu buffer; yaitu, Vim menghapusnya dari daftar buffer. Beberapa bit informasi, seperti tanda, masih disimpan.
  • :bw menutup satu buffer dan wipes; buffer "benar-benar" dihapus dan semua informasi sementara hilang.
  • :q keluar dari editor; mengasumsikan bahwa hanya satu buffer yang terbuka.
  • :qa (yang merupakan kependekan dari perintah :qall yang telah Anda gunakan) keluar dari editor dan tutup semua buffer. Ini terpisah dari q karena jika tidak, Anda mungkin lupa bahwa Anda membuka file lain dan kehilangan informasi secara tidak sengaja.

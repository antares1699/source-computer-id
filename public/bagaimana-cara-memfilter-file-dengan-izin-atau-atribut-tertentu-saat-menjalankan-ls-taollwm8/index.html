Bagaimana cara memfilter file dengan izin atau atribut tertentu saat menjalankan ls?

Misalnya, jika saya hanya menginginkan file yang memiliki izin g+s saat menjalankan ls, bagaimana saya bisa melakukan ini?

ls tidak menyediakan ini - ini biasanya digunakan hanya untuk mengubah opsi tampilan. Apa yang dapat Anda lakukan malah melibatkan penggunaan find . Sebagai contoh:

find . -perm -g+s

Untuk meniru perilaku ls yang hanya mencantumkan file di direktur saat ini, gunakan opsi -maxdepth 1.

Kombinasi yang mungkin tidak terbatas. Lihat bantuan find yang ditautkan di atas untuk contoh lebih lanjut dan perbedaan sintaks dalam menggunakan - atau / sebelum mode , juga sehubungan dengan / menjadi ekstensi khusus GNU dan + digunakan dalam gaya BSD find.

Akankah Acer Revo baru (dengan Atom 330) cukup cepat untuk menjadi klien / server MythTV?

Sebagai seorang geek, saya sangat menyukai CPU Atom tetapi belum menemukan alasan untuk membelinya :( Meskipun saya berpikir untuk membuat DVR sendiri dengan fungsionalitas NAS dan pusat media. Sayangnya, bahkan Acer Revo saat ini, yang dibangun di atas platform ION tidak cukup cepat untuk streaming video Full HD. Jadi bagaimana menurut Anda, apakah CPU dua inti baru akan membuatnya lebih baik, apakah dapat menampilkan video Full HD, menyimpannya ke disk, dan mentransfer sesuatu melalui jaringan pada saat yang bersamaan? Akankah itu dapat mengubah skala video dari Hulu dan YouTube ke layar penuh?

Anda tidak mungkin dapat menggunakannya sebagai bagian depan dan belakang gabungan (meskipun mungkin tetapi kartu TV akan menjadi masalah, karena Anda hanya dapat menggunakan yang USB), tetapi jika Anda menyiapkan backend terpisah mesin, revo membuat mesin frontend yang sempurna dan senyap, bahkan untuk konten HD.

XBMC (yang dapat digunakan sebagai frontend Mitos) berjalan dengan cemerlang pada ini, seperti halnya frontend Mtyh yang tepat.

Seperti yang telah dikatakan, platform ION berisi chipset NVidia 9400m, yang dapat menangani decoding h.264 tanpa bersusah payah, dan hanya menggunakan seperti 5-10% CPU, bahkan di bawah linux (melalui VDPAU yang disebutkan) - BAHKAN UNTUK FULL HD BITRATE 1080P TINGGI

Saat ini ada utas aktif di Forum XBMC dan saya yakin ada juga di formulir MythTV.

bahkan lifehacker telah melihat ini

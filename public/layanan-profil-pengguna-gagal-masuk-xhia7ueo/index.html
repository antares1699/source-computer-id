Layanan Profil Pengguna gagal masuk

Saya memiliki Laptop Windows 8.1 Pro Dell 15R dan saya mencoba membuat pengguna lokal tetapi saat pertama kali masuk, saya berkata, "Layanan Profil Pengguna gagal masuk." Saya bertanya pada Answer Desk dan saya tidak mampu membayar layanan mereka.

Ada ide untuk membantu saya?

Bug dalam Pembaruan untuk Microsoft Visual Studio 2013 Pembaruan 1 (KB2932965) dapat menyebabkan ini (lihat file laporan bug ). Saya mengatasi masalah tersebut dengan menghapus C:\Users\Default\AppData\Local\Microsoft\VSCommon; Namun, jika Anda ingin bermain aman, Anda dapat mengatur ulang izin pada folder itu untuk mewarisi dari orang tua.

Sudahkah Anda mencoba ini: Anda menerima pesan galat "Layanan Profil Pengguna gagal logon"

  1. Mulai laptop Anda dalam mode aman
  2. Perbaiki profil akun pengguna: Buka editor registri
  3. Temukan HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows NT \ CurrentVersion \ ProfileList
  4. Temukan folder yang dimulai dengan S-1-5 (kunci SID)
  5. Temukan entri ProfileImagePath di panel detail
  6. Jika dua folder bernama S-1-5 ..., tetapi salah satunya diakhiri dengan .bak, tambahkan .ba di akhir nama folder yang tidak diakhiri dengan .bak
  7. Dari nama folder yang diakhiri dengan .bak, hapus .bak
  8. Ubah nama folder dari langkah 6 menjadi .bak, bukan .ba
  9. Klik dua kali folder tanpa .bak di panel rincian, klik dua kali RefCount, ketik 0, lalu klik OK
  10. Ulangi langkah 9 untuk Negara
  11. Nyalakan ulang laptop

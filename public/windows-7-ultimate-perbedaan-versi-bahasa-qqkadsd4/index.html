Windows 7 Ultimate: perbedaan versi bahasa

Apakah ada perbedaan antara versi bahasa yang berbeda dari Windows 7 Ultimate? Jika saya mengerti dengan benar, apa pun versi bahasa yang akan saya beli, saya dapat mengubah bahasa setelah instalasi.

Jadi kesimpulan saya adalah bahwa versi bahasa yang berbeda hanyalah DVD yang dimuat sebelumnya dengan paket bahasa tertentu. Apakah ini benar?

Ya itu benar, meskipun rutinitas instalasi jelas harus sedikit berbeda.

Ada OS tanpa bahasa yang mendasarinya, dan kemudian Paket Bahasa diinstal di atasnya. Ini dapat dilihat dengan jelas oleh diagram di tautan pengembang Microsoft berikut:

http://msdn.microsoft.com/en-us/goglobal/ee461121

Tautan ini juga memiliki deskripsi yang sangat bagus tentang perbedaan pemasangan antara bahasa Inggris dan Cina:

"satu-satunya perbedaan hanya dalam instalasi paket bahasa yang berbeda file untuk Windows 7 versi bahasa Inggris, penginstalan pertama adalah x86_Microsoft-Windows-Foundation-Package_6.1.7600. 16385 paket dasar, kemudian instal x86_Microsoft-Windows-Client-LanguagePack-Package_ 6.1.7600.16385_en-paket bahasa AS

Untuk versi bahasa Mandarin Windows 7, penginstalan pertama adalah x86_Microsoft-Windows-Foundation-Package_6.1.7600. 16385 paket dasar, kemudian instal x86_Microsoft-Windows-Client-LanguagePack-Package_ 6.1.7600.16385_zh-CN paket bahasa "

http://www.wduu.info/archives/6754.html

http://en.wikipedia.org/wiki/Multilingual_User_Interface

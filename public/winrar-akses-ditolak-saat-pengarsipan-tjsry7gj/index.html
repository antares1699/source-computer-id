WinRAR: Akses Ditolak saat pengarsipan

Saya mencoba mencadangkan drive lama dari mesin WinXP di Windows 7 menggunakan WinRAR. Saya terus mendapatkan "Access is Denied" di banyak folder yang saya coba sertakan. Saya tidak benar-benar ingin mengubah izin file pada drive. Apakah ada cara agar WinRAR mengabaikan izin file?

4

7

Saat Anda membuka WinRar pastikan Anda mengklik kanan dan memilih "Run As Administrator". Meskipun Anda adalah pengguna administrator, program yang Anda jalankan tidak akan memiliki hak administrator kecuali Anda melakukannya.

Jika file masih tidak dapat dibaca, artinya file sedang digunakan dan memiliki kunci eksklusif, atau administrator tidak memiliki izin untuk membaca. Satu hal yang saya rekomendasikan adalah penggunaan Komandan Total untuk menyalin file sebelum Anda mencoba memakainya. Saya telah menggunakannya untuk menyalin file yang sulit. Ini akan meningkatkan dirinya ke hak administrator jika perlu.

Jika OS Anda mencegah Anda memiliki akses, dan Anda berada dalam akun administrator, folder tersebut kemungkinan terbatas pada pengguna SISTEM, yang merupakan pengguna yang tidak dapat Anda masuki atau menjalankan aplikasi. Karena itu, WinRAR tidak akan pernah bisa mengesampingkan hal-hal seperti itu.

Yang mungkin ingin Anda pertimbangkan adalah melakukan booting ke lingkungan LiveCD, seperti Ubuntu atau OpenSUSE. Ironisnya, saya menemukan bahwa lingkungan LiveCD memiliki akses yang lebih baik ke partisi NTFS untuk XP, Vista, dan terutama Windows 7. Jika Anda ingin mengesampingkan izin dan mengarsipkan folder besar, inilah cara melakukannya.

Saya menemukan bahwa di Windows 7, menyalin profil dan hal-hal lain dengan "akses terbatas" atau memerlukan akses administratif untuk mengotorisasi salinan, benar-benar payah dalam hal kinerja penyalinan, dan umpan balik kemajuan yang akurat. Ini secara khusus saat melakukannya di Windows 7. Melakukan hal yang sama, pada folder yang sama di lingkungan LiveCD tidak hanya memberi Anda kinerja yang tepat, tetapi juga umpan balik kemajuan yang tepat.

Gunakan Microsoft untuk implementasi luar biasa dari teknologi format Anda sendiri.

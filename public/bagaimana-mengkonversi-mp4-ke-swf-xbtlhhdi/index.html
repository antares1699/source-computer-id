Bagaimana mengkonversi MP4 ke SWF

Dapatkah seseorang memberi saya parameter baris perintah untuk FFmpeg untuk mengonversi video MP4 ke file Adobe Flash SWF?

Pada dasarnya, seperti ini:

ffmpeg -i in.mp4 out.swf

Ini akan secara default mengonversi video ke Sorenson Spark (H.263, atau dikenal sebagai flv1) dan audio ke MP3, menggunakan encoder MP3 apa pun yang dimiliki FFmpeg Anda — mungkin libmp3lame.

Jika Anda perlu mengubah kualitas video, Anda memiliki dua opsi:

  • tambahkan opsi -qscale:v setelah -i in.mp4 dengan nilai dari 1 hingga 31, di mana lebih tinggi berarti kualitas lebih rendah (coba dengan 2 atau 3 dulu). Ini adalah pengaturan VBR.

  • tambahkan opsi -b:v dengan bitrate, mis. 1M atau 500K. Ini adalah pengaturan CBR, tentu saja, dan dapat menghasilkan kualitas visual yang lebih rendah per ukuran file.

Untuk audio:

  • tambahkan opsi -q:a untuk VBR MP3, dengan nilai antara 0 dan 7, 0 adalah kualitas terbaik.

  • tambahkan opsi -b:a untuk CBR MP3, mis. dengan 192K.

Pertimbangkan untuk mengonversi video Anda ke wadah FLV, yang akan menangani lebih banyak jenis codec daripada SWF (yang hanya mendukung H.263, VP6 atau Motion JPEG).

Ini akan memungkinkan Anda untuk menggunakan video H.264 dan audio AAC, yang memberi Anda kualitas yang lebih baik pada ukuran file yang lebih kecil jika dibandingkan dengan varian H.263 / MP3 dalam file SWF. Anda bahkan mungkin memiliki kesempatan untuk menyalin bitstream, yang berarti tidak ada pengurangan kualitas sama sekali.

ffmpeg -i in.mp4 -c copy out.flv

Jika tidak berhasil, enkode ulang:

ffmpeg -i in.mp4 -c:v libx264 -c:a aac -strict experimental -b:a 192K out.flv

Ini akan menggunakan encoder x264. Untuk mengubah kualitas video di sini, Anda memiliki opsi berikut:

  • tambahkan opsi -crf setelah -i in.mp4 dengan nilai antara 0 dan 51, di mana lebih rendah berarti kualitas lebih baik. Defaultnya adalah 23, dan nilai waras dari 18–28.

  • tambahkan opsi -b:v, mirip dengan di atas untuk mode bitrate konstan.

Lihat panduan pengkodean x264 di wiki FFmpeg untuk info lebih lanjut.

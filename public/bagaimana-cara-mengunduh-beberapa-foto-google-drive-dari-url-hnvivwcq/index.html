Bagaimana cara mengunduh beberapa foto Google Drive dari URL?

Saya dikirimi sekitar 500 file gambar Google Drive dan butuh cara untuk mendownloadnya. URL mengirim saya ke Google Drive untuk setiap gambar individu. Apakah ada cara untuk mendownloadnya tanpa harus pergi ke setiap halaman dan menekan download secara manual?

Saya telah mencoba mengonversi tautan ke berikut ini: http://drive.google.com/uc?export=view&id=, tetapi tampaknya tidak dapat meminta pengelola Unduhan untuk mengunduhnya [GetRight].

Anda dapat mengubah tautan gambar Google Drive menjadi tautan unduhan dengan menukar (cukup gunakan alat ganti semua, saya suka TextMechanic sederhana, gratis, online Temukan dan ganti tetapi Notepad menemukan dan mengganti akan melakukan trik).

Saya adalah pengguna Google Drive dan seperti yang Anda katakan, ini hanya masalah mendapatkan tautan unduhan. Contoh di drive saya adalah

https://drive.google.com/file/d/0Bz697J8sF_ekS1ZSV0tGSS1ZbHc/edit?usp=sharing

untuk tautan berbagi, atau

https://docs.google.com/file/d/0Bz697J8sF_ekS1ZSV0tGSS1ZbHc/edit

untuk tautan edit. Tautan unduhannya adalah

https://drive.google.com/uc?export=download&id=0Bz697J8sF_ekS1ZSV0tGSS1ZbHc

seperti yang dijelaskan di artikel ini .

Jadi ini hanya masalah menemukan dan mengganti

https://docs.google.com/file/d/ atau https://drive.google.com/file/d/

untuk https://drive.google.com/uc?export=download&id=

dan menemukan dan mengganti /edit atau /edit?usp=sharing dengan   (tidak ada yaitu menghapusnya)

Setelah Anda selesai melakukannya, Anda dapat mengunduh daftar file dengan berbagai cara, tetapi jika Anda ingin tetap berada di browser, saya telah menulis ekstensi Chrome yang disebut TabSave untuk melakukan hal itu - ada opsi untuk menggunakan daftar manual untuk mengunduh (ini juga memungkinkan mengunduh file di setiap tab jendela).

Anda cukup menempelkannya dari daftar URL yang diedit dan mereka akan mengunduh. Sangat ringan dan Anda tidak perlu khawatir tentang pemasangan yang cerdik dari program freeware ajaib yang hebat ini. Semua kode adalah open source di GitHub (tautan di deskripsi toko web).

Metode 1

Anda dapat mencoba program bernama Pengunduh Gambar URL . Ini adalah perangkat lunak gratis yang dihosting di Source Forge. Berikut ini uraian singkatnya.

Anda dapat mendownload gambar dari daftar URL. Anda hanya membutuhkan file dengan Daftar URL, dan folder tujuan

Ini adalah aplikasi konsol, dikembangkan dengan teknologi .NET.

Saya belum mencobanya sendiri, jadi saya tidak dapat menjamin ini akan berhasil. Tapi saya tidak melihat alasan mengapa tidak. Sepertinya alat yang sangat berguna dan mudah digunakan.

Cara kerjanya adalah Anda membuat file teks biasa yang berisi URL untuk foto yang ingin Anda download, setiap URL di baris baru. Sesuatu seperti ini.

https://plus.google.com/photos/photo_01.jpg
https://plus.google.com/photos/photo_02.jpg
https://plus.google.com/photos/photo_03.jpg
https://plus.google.com/photos/photo_04.jpg
https://plus.google.com/photos/photo_05.jpg
https://plus.google.com/photos/photo_06.jpg
https://plus.google.com/photos/photo_07.jpg
https://plus.google.com/photos/photo_08.jpg
https://plus.google.com/photos/photo_09.jpg
https://plus.google.com/photos/photo_10.jpg
https://plus.google.com/photos/photo_11.jpg
https://plus.google.com/photos/photo_12.jpg

Ini adalah bagian tersulit, karena Anda harus melakukan pekerjaan ini secara manual. Anda kemudian akan menyimpan ini ke C:\Users\You\Desktop\Your_text_document.txt misalnya. Setelah Anda memiliki file ini, Anda kemudian akan menggunakan alat tersebut untuk mencari URL file ini secara online dan mengambilnya.

Anda dapat melihat pada contoh di atas dari tangkapan layar cara mengetikkan perintah Anda. Saya sarankan Anda untuk terlebih dahulu menyiapkan folder (/dest) di mana Anda ingin menyimpan file yang diunduh.

Tautan: http://sourceforge.net/projects/urlimagedownloader/

  1. Unduh dan pasang ResHacker .
  2. Buka ResHacker sebagai administrator dan klik File > Open dan pilih file exe.
  3. Klik folder icon lalu pada salah satu subfolder di sana dan kemudian pada file di dalam subfolder
  4. Klik kanan padanya dan pilih Replace Resource...
  5. Sebuah dialog akan muncul, pilih ikon yang ingin Anda ganti dari daftar di sebelah kanan.
  6. Klik Open file with new icon dan pilih ikon yang Anda inginkan.
  7. Klik Replace.

  8. Klik File > Save.

    • Itu harus as administrator sehingga bisa menulis perubahan ke file di dalam program files.
    • Perhatikan bahwa ResHacker juga mengganti nama file asli menjadi filename_original.
    • Jika karena alasan tertentu Anda tidak dapat menulis ke Program Files, Lakukan saja Save As..., simpan file exe di tempat lain dan ganti nanti.

Edit: Untuk meringkas semua komentar di bawah - terkadang aplikasi menggunakan sumber daya file dll eksternal (dan secara teoritis, exe). Jadi, jika Anda mengubah ikon di exe dan itu tidak berfungsi, periksa file dll aplikasi dan ganti ikon di sana juga.

Ini sangat mudah:

  1. Buat pintasan file di desktop Anda, misalnya myfile.txt.
  2. Kemudian seret myfile.txt ke bilah tugas tempat semua ikon kecil lainnya berada.

Anda mungkin sudah memiliki banyak sehingga Anda dapat menekan panah di sebelah ikon, mencari ikon myfile.txt dan menyeretnya kembali ke toolbar.

Semoga ini membantu.

Gunakan Ninite. Ini adalah program penginstalan otomatis yang memungkinkan Anda memilih grup program dan menginstalnya tanpa pengawasan di komputer Anda. Teamviewer 8 adalah salah satu opsinya, jadi buka situs webnya, klik kotak centang teamviewer 8, gulir ke bawah ke bawah dan tekan get installer. Anda kemudian akan dibawa ke halaman unduh dan itu akan mengunduh file beberapa ratus kb. Itu adalah script install otomatis yang akan mendownload teamviewer 8 dan secara otomatis menginstalnya tanpa melakukan pemeriksaan sistem OS. Saya menguji ini di server Windows Server 2008 R2 saya SENDIRI dan berfungsi mulai 13/1/2013.

Semoga ini membantu.

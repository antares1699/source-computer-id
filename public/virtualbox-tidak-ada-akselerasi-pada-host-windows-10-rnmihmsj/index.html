VirtualBox tidak ada akselerasi pada host Windows 10

Setelah memutakhirkan OS host saya ke Windows 10, saya telah memperhatikan beberapa masalah dengan tamu saya jadi saya melihat sekeliling pengaturan dan memperhatikan bahwa jumlah CPU dan opsi akselerasi berwarna abu-abu .. Saya tidak memperbarui kotak virtual atau melakukan perubahan apa pun dengan tamu saya sejak terakhir kali, jadi hanya yang bisa saya pikirkan adalah bahwa peningkatan tuan rumah menyebabkan ini.

Sekarang saya bertanya-tanya apakah Win 10 memperkenalkan beberapa perubahan mendasar yang entah bagaimana akan mempengaruhi virtualisasi atau hanya masalah saya?

VB versi 5.0.1 Win10 sebagai tuan rumah, Lubuntu sebagai tamu pengujian (tetapi semua tamu memiliki akselerasi dinonaktifkan di dalamnya)

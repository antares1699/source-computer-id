Menjalankan Aplikasi Modern UI / Metro sebagai Administrator di Windows 8

Saya perhatikan bahwa di layar Start Windows 8, saya bisa mengklik kanan program warisan Windows ( Sebuah program yang berjalan di Windows XP, Vista dan 7 ), dan saya dapat menjalankannya sebagai Administrator.

Namun, setiap kali saya mengklik UI Modern Windows 8 atau aplikasi Metro, saya tidak memiliki opsi itu.

Jadi inilah pertanyaanku:-

  1. Mengapa saya tidak dapat menjalankan aplikasi UI Modern sebagai Administrator?
  2. Apakah itu membuat perbedaan sejauh menyangkut keamanan?

Jika Anda mematikan proses explorer.exe dan menjalankannya sebagai administrator, berikut adalah pesan kesalahan yang akan diberikan Windows kepada Anda:

Akses administrator hanya diperlukan di Windows saat aplikasi ingin mengakses file dan folder sistem.

Aplikasi Windows Store dikotak pasir di mana mereka tidak bisa mendapatkan akses ke file sistem atau akses sistem file mentah. Aplikasi Windows Store hanya dapat mengakses perpustakaan dan media yang dapat dilepas dengan sendirinya dan jika ingin mengakses file lain, pengguna harus memilihnya melalui pemilih-buka-file untuk memberikan izin kepada aplikasi untuk melakukannya.

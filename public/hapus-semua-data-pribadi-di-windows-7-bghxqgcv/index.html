Hapus semua data pribadi di Windows 7

Di Windows XP, ada opsi untuk membersihkan semua data pengguna yang disimpan, kata sandi, situs web yang dikunjungi, MRU, pencarian, dan sebagainya dalam satu waktu. Opsi ini tampaknya telah menghilang di Windows 7.

Bagaimana cara membersihkan semua data pribadi pada mesin Windows-7? Apakah saya memerlukan program pihak ketiga? Ataukah botton untuk menyingkirkannya hanya disembunyikan untuk saya lihat?

(Harap perhatikan bahwa ini bukan duplikat karena pertanyaan serupa menanyakan tentang subkumpulan tertentu dari data pribadi saja.)

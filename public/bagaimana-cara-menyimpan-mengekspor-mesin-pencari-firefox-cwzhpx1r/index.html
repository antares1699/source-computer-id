Bagaimana cara menyimpan / mengekspor mesin pencari Firefox

Saya mendapatkan banyak mesin pencari berbeda yang ditambahkan ke bilah alat pencarian saya di Firefox (versi 8). Karena saya akan segera menginstal ulang O / S saya, saya perlu membuat cadangan mesin pencari ini - sebaiknya juga kata kunci yang saya siapkan untuk mereka.

Adakah yang tahu di mana saya dapat menemukan (menyimpan / mengekspor / mencadangkan) pengaturan ini di Mac OS?

Anda dapat menyimpan semua personalisasi Firefox jika Anda menyimpan direktori profil Firefox dan kemudian memulihkan data tersebut setelah penginstalan.

Dokumentasi Firefox menyebutkan bahwa direktori profil di bawah Mac OS adalah salah satunya:

  • ~/Library/Mozilla/Firefox/Profiles/<profile folder>
  • ~/Library/Application Support/Firefox/Profiles/<profile folder>

Informasi berikut mungkin berguna untuk mentransfer hanya mesin pencari dan kata kunci terkait ke profil lain. Saya tidak menemukan cara untuk melakukannya yang akan didukung oleh pengembang, jadi mengikuti ini mungkin berisiko atau mungkin tidak berfungsi dengan versi Firefox yang berbeda. Punyaku adalah iceweasel 17.0.6, ini berfungsi di sana.

Tampak dari pengujian saya bahwa informasi yang terkait dengan Mesin pencari dan kata kunci disimpan di file search.json dan search-metadata.json, dan di folder searchplugins, ketiganya di folder profil Firefox. Cukup salin dari profil sumber, hapus pada profil target melakukan trik untuk saya. Lakukan ini saat Firefox ditutup (dan tentu saja setelah membuat cadangan profil Anda).

Saya tidak dapat menemukan banyak informasi resmi yang relevan, tetapi inilah yang saya temukan jika seseorang ingin menyelidiki lebih lanjut atau mengoreksi dokumen (yang tidak saya lakukan karena saya tidak tahu seberapa umum kesimpulan saya berdasarkan pengujian saya sendiri). Halaman ini dari MozillaZine Knowledge Base (KB) mengatakan bahwa file search.sqlite “tidak lagi digunakan dalam versi Firefox saat ini”. Menurut pengujian saya, ini sebagian tidak benar: menghapus file tampaknya membuat Firefox membangun kembali daftar Mesin Pencari. Halaman ini dari KB juga memiliki informasi yang tidak berlaku untuk versi saya. Ia mengatakan tentang file search.json: "Tidak jelas apa itu karena tidak ada search.sqlite untuk cache", dan tidak menyebutkan search-metadata.json. Halaman ini dari KB, meski memiliki judul yang menjanjikan, tidak banyak memuat informasi yang berguna. Seseorang melakukan beberapa penelitian terkait , tapi mungkin sekarang sudah usang.

Temuan saya didasarkan pada pengujian berikut (di kotak linux). Saya menetapkan kata kunci untuk salah satu Mesin Pencari, menutup Firefox, dan menjalankan "grep -rl mykeyword *" di folder profil Firefox terkait, yang memberi saya hanya pencarian-metadata.json. Demikian pula, mencari nama salah satu Mesin Pencari ("grep -rl CNRTL *") hanya memberikan file search.json. Saya juga memeriksa konten file "search.sqlite" (karena memiliki nama yang tampaknya relevan): jalankan "sqlite3 search.sqlite" lalu ".dump" untuk melihat konten database. Itu hanya memiliki satu tabel dengan hanya satu entri (sesuatu tentang google). Juga file tidak diperbarui ketika saya mengubah mesin pencari, sedangkan dua lainnya melakukannya.

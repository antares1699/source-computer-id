Mode XP tidak ada lagi -

Windows 8 tidak menyertakan mode XP, fungsi dari Windows Virtual PC. Virtual PC sendiri mendapat kapak di Windows 8 yang mendukung serangkaian fitur baru dan platform perangkat lunak pilihan.

Sumber

Jadi, Anda harus menginstal beberapa perangkat lunak mesin virtual lain dan menginstal XP di dalamnya. Anda akan membutuhkan lisensi XP yang valid, jadi jika Anda meningkatkan lisensi XP Anda untuk mendapatkan ke Windows 7 Anda akan kurang beruntung.

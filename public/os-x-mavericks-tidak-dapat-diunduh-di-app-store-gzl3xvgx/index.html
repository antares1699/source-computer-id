OS / X Mavericks tidak dapat diunduh di App Store

Saya mencoba mendownload OS / X Mavericks melalui App Store kemarin. Namun, saya harus meninggalkan pekerjaan sebelum selesai mengunduh dan mematikan mesin saat sedang mengunduh.

Saya berasumsi bahwa saya akan dapat melanjutkan pengunduhan, atau memulai lagi nanti.

Namun, sekarang tidak ada cara untuk mendownloadnya lagi. Ketika saya menemukan OS / X Mavericks di toko, saya mengklik tautan "Unduh". Itu meminta ID Apple saya yang saya masukkan, dan itu sama sekali tidak melakukan apa-apa. Itu tidak mulai mengunduh, dan itu hanya masih menampilkan tombol "Unduh". Mengkliknya lagi tidak akan menghasilkan apa-apa.

Saya pikir ada sesuatu macet di suatu tempat, seperti beberapa bendera yang mengatakan unduhan ini sedang berlangsung atau sudah diunduh. Adakah cara untuk menghapusnya sehingga saya dapat mulai mendownloadnya lagi?

Coba periksa Pembelian Anda di App Store untuk melihat apakah itu ada di sana, jika demikian Anda harus dapat melanjutkannya dan / atau menginstalnya dari sini jika unduhannya selesai. Lihat https://support.apple.com/kb/HT4485 .

Anda juga harus dapat menghapus unduhan dari bagian Pembelian dengan mengklik 'X' yang akan muncul di sebelah 'Tombol Status' dari item yang ingin Anda hapus.

Misalnya pada gambar di bawah ini jika Anda mengarahkan mouse ke kiri tombol 'Downloaded', sebuah 'X' akan muncul.

Jika Tombol Status harus bertuliskan 'Lanjutkan', Anda dapat memilih untuk melanjutkan dengan mengkliknya atau menghapusnya dengan mengarahkan kursor ke kiri tombol hingga 'X' muncul.

Jika gagal, hubungi layanan pelanggan Apple, mereka seharusnya dapat membantu.

EDIT: coba periksa unduhan yang belum selesai melalui bilah menu dan keluar / masuk dan coba unduh lagi.

EDIT: coba periksa dan lanjutkan unduhan di Launchpad Anda, kemajuan harus ditampilkan di / di Launchpad. Anda juga dapat menghapus Aplikasi di Launchpad dengan mengklik dan menahan Aplikasi hingga tanda 'X' muncul di Aplikasi. Kemudian coba unduh ulang Aplikasi dari App Store.

Mungkin bisa dilakukan dalam kegelapan tetapi keluar dari App Store, buka terminal Anda dan jalankan ini:

sudo rm -rf /Applications/Install\ OS\ X\ Mavericks.app

Mungkin rusak dan tidak dapat ditimpa saat mencoba mengunduh lagi. Kemudian buka kembali toko aplikasi Anda dan coba lagi. Juga harap berhati-hati. Perintah RM adalah alat yang ampuh dan berbahaya. Anda pasti harus memiliki backup yang bagus :)

Buka App Store. Telusuri Mavericks. Kotak kecil hanya akan bertuliskan "Gratis" dan tidak ada yang Anda lakukan untuk mendownloadnya. Sebaliknya, klik tautan Mavericks untuk memunculkan seluruh halaman pembuka yang menjelaskan Mavericks. Tombol (lebih besar) sekarang harus bertuliskan "Unduh" dan akan memperingatkan Anda bahwa Anda sudah menginstal Mavericks. Mengklik tombol Unduh akan mulai mengunduh.

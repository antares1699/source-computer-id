Windows XP : Ke C:\Documents and Settings\_YOUR_USER_NAME_\Application Data\Microsoft\Templates

Windows Vista / Windows 7 : Goto C:\Users\_YOUR_USER_NAME_\AppData\Roaming\Microsoft\Templates

Klik kanan pada Normal.dotm dan pilih Open daripada New

Ubah font dalam Normal Style. Setelah Anda selesai, klik Save

Bingo!

Ketemu secara kebetulan meskipun terlalu banyak pekerjaan :-)

Klik di pita pada ikon gaya ke-2 'tanpa spasi', ketik dua atau tiga baris di dokumen, tandai garis dan klik kanan untuk mengubah font dan atau jarak garis dan klik ok.

Klik kanan lagi pada baris teks yang Anda tandai, klik kanan lagi untuk mengubah font / jarak baris dan klik simpan sebagai default, lakukan itu tiga kali!

Setelah tiga kali mengklik 'simpan sebagai default', pengaturan Anda akan disimpan sebagai default!

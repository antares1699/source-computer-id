Pengaturan Windows 7 macet setelah layar “Memulai Windows…”

Saya mengalami masalah saat menginstal Windows 7. Saya perlu menginstal OS dari boot untuk membagi C: menjadi dua partisi yang berbeda, karena ini tidak diperbolehkan saat menginstal dari dalam Vista.

Ketika saya boot dari disk instalasi, saya mendapatkan layar biasa "Windows sedang menyalin file ...", diikuti oleh layar "Memulai Windows ..." - dengan logo jendela animasi atau apa pun. Kemudian sepertinya penginstalan akan dimulai dengan layar biru dan kursor saya dapat bergerak, tetapi di sini semuanya berhenti. Tidak ada lagi yang terjadi, dan penyiapan sepertinya macet. Tidak ada satu tombol pun pada keyboard saya yang berpengaruh, dan yang harus saya lakukan hanyalah membatalkan dan menyalakan ulang.

Saya sudah mencoba menginstal menggunakan dua DVD berbeda (bukan klon), dan hal yang sama selalu terjadi. Apa yang mungkin menyebabkan ini, dan bagaimana cara memperbaikinya?

Beberapa spesifikasi perangkat keras saya adalah sebagai berikut:

Papan Utama: Asus P5E X38
CPU: Intel Core2 Duo E6850 3.0 Ghz
OS: Windows Vista Ultimate 32bit
Papan ketik: Logitech diNovo Edge

.. butuh lebih banyak?

Dan ya, saya masih bisa menggerakkan mouse saya di layar biru, tapi saya belum menunggu lebih dari 10 menit sebelum menyerah (akan mencoba menunggu lebih lama).

Dan tidak, saya belum meminta Microsoft.

Dan saya tidak tahu bagaimana mengatur ulang pengaturan BIOS.

Mencoba perbaikan ini :

  1. Klik Mulai> Semua Program> Aksesori
  2. Klik kanan pada Command Prompt.
  3. Pilih Jalankan sebagai administrator .
  4. Klik Lanjutkan.

  5. Pada tipe Command Prompt:

    sfc /scannow

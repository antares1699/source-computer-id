Luncurkan file / direktori menggunakan Gnome dari baris perintah

Di Windows, saya dapat meluncurkan jendela Explorer dari baris perintah melalui "start (dirname)". Dan ini juga akan meluncurkan aplikasi yang terkait dengan ekstensi file tertentu, jadi saya dapat melakukan "start myfile.txt" dan itu akan membuka file di editor teks default (mis. Notepad). Pada dasarnya, Windows memperlihatkan cara melalui baris perintah untuk melakukan tindakan yang sama seperti klik dua kali di Explorer.

Apakah ada sesuatu yang serupa untuk Gnome? Saya memindai daftar yang dikembalikan oleh "apropos gnome", "apropos start", dan "apropos open", dan tidak ada yang tampak menjanjikan. (Saya menggunakan Debian Lenny, jika itu penting.) Saya tahu saya dapat membuka browser file melalui "nautilus (dirname)", dll, tetapi saya lebih suka menggunakan sesuatu yang menggunakan asosiasi file saya.

Tujuan akhir saya adalah membuat alias dari "start" ke perintah ini karena saya lebih terbiasa dengan antarmuka Windows.

Gunakan perintah gnome-open dan itu akan menggunakan aplikasi yang telah Anda kaitkan dengan jenis file tersebut (MIME). Sebagai contoh:

gnome-open foo.pdf

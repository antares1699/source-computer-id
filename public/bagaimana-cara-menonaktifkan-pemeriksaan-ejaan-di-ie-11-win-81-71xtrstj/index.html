Jadi masalah ini adalah masalah izin. Saat menjalankan IE dalam mode EPM, regkeys sedang diblokir di lokasi HKCU \ Software \ Microsoft \ TabletTip \ 1.7 karena SEMUA PAKET APLIKASI tidak ditambahkan ke izin. Jadi cara untuk memperbaikinya (tolong tanggapi jika ini tidak berhasil), adalah pergi ke

  1. Kunci kemenangan

  2. ketik "regedit" dan klik aplikasi regedit

  3. Navigasi ke HKey_CURRENT_USER \ Software \ Microsoft \ TabletTip \ 1.7

  4. Klik kanan pada direktori 1.7

  5. Pilih "Izin ..."

  6. Klik Add

  7. Klik Advanced

  8. Untuk Lokasi, pilih komputer Anda

  9. Klik Temukan Sekarang

  10. Pilih SEMUA PAKET APLIKASI dan klik OK

Perbaikan akan dihentikan, tetapi sementara itu, ini adalah perbaikan yang benar

Dari Cara Menonaktifkan Fitur Pemeriksa Ejaan Bawaan :

  1. Gerakkan kursor mouse Anda ke pojok kanan atas atau pojok kanan bawah layar untuk mengakses Charms Bar dan klik Pengaturan pilihan.

  2. Sekarang klik Ubah pengaturan PC opsi yang diberikan di bagian bawah.

  3. Ini akan membuka Panel Kontrol Metro atau halaman Pengaturan PC. Klik Umum bagian yang diberikan di sisi kiri pada Windows 8. Jika Anda menggunakan Windows 8.1, klik PC & perangkat -> Mengetik .

  4. Sekarang di sisi kanan, matikan 2 opsi berikut:
    Koreksi otomatis kata yang salah eja
    Sorot kata yang salah eja

  5. Itu dia. Ini akan menonaktifkan fitur pemeriksa ejaan di Internet Explorer serta seluruh sistem operasi.

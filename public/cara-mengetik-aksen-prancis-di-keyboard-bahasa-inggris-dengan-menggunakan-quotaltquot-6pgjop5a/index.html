Cara mengetik aksen Prancis di keyboard bahasa Inggris dengan menggunakan "ALT"

Laptop saya saat ini adalah paviliun HP g6, windows 7 64 bit.

Saya memiliki banyak Pcs sebelumnya, saya menggunakan keyboard bahasa Inggris pada semuanya. Jika saya ingin mengetik é misalnya, saya akan mengetik ALT-130. Pada laptop saya saat ini fungsionalitas ALT tidak ada, saya mencoba bermain-main di panel kontrol tetapi tidak dapat memperbaikinya.

EDIT

  • Saya tidak melihat tombol num lock, itu keyboard saya

  • Saya tidak ingin mengubah keyboard saya menjadi AZERTY, saya ingin menggunakan keyboard standar QWERTY Inggris.

Pertama, pastikan Anda hanya menggunakan Numberpad dengan tombol Num-Lock dihidupkan.

Ingat juga bahwa beberapa program tidak mendukung fungsionalitas kunci ALT, jadi ini mungkin bukan masalah khusus dengan perangkat keras Anda, tetapi mungkin masalah dengan program yang Anda gunakan jika program itu telah berubah. Jika tidak berfungsi dengan [Alt] apakah berfungsi dengan [Alt Gr]?

Jika keyboard Anda memiliki tombol [Function], coba tahan tombol [Function] dengan tombol [Alt] dan ketikkan urutan angka. Misalnya [Fungsi] [Alt] [130] dan lihat apakah berfungsi.

Sebagai metode penyelesaian, meskipun saya tidak menyarankan Anda menggunakan ini sebagai solusi permanen, Anda dapat membuka Peta Karakter: Mulai> Semua Program> Aksesori> Alat Sistem> Peta Karakter dan salin dan tempel karakter secara manual

Namun solusi alternatif lebih lanjut, yang dapat Anda gunakan sebagai solusi permanen, adalah dengan menggunakan Autohotkey dan mengatur hotkey untuk secara otomatis memasukkan karakter tertentu ketika Anda menekan hotkey Anda, misalnya Anda dapat mengikat [Alt] [130] untuk memasukkan [é] - setelah Anda mengaturnya, itu akan berfungsi sebaik Anda [Alt -130] metode yang sebelumnya Anda gunakan.

Alt sisi kiri seharusnya berfungsi tetapi hanya berfungsi dengan tombol numpad. Laptop Anda mungkin tidak memiliki tombol numpad khusus (untuk menghemat ruang) tetapi mereka mungkin memiliki kunci dengan fungsi ke-2. Anda mungkin harus mengaktifkan numlock lalu mencari angka di tombol lain (biasanya di sekitar sisi kanan keyboard Anda). Jika tidak, untuk huruf Prancis saja Anda dapat menahan "alt" dan "Ctrl" sambil mengetik a, e, i, o dan u untuk mendapatkan á, é, í, ó, ú atau Anda cukup menahan alt gr tanpa ctrl untuk hal yang sama.

Solusi lain adalah menambahkan tata letak keyboard Prancis ke jendela Anda. Anda kemudian akan mendapatkan pemilih bahasa di dekat baki sistem Anda (kanan bawah) dan dapat beralih antara tata letak keyboard Prancis dan Inggris untuk saat Anda menulis dalam bahasa Prancis. Anda juga dapat mengatur pintasan untuk beralih di antara 2 pengaturan yang berbeda.

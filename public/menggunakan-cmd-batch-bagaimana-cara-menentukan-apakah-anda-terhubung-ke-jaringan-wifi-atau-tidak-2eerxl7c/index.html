Menggunakan cmd / Batch, bagaimana cara menentukan apakah Anda terhubung ke jaringan WiFi atau tidak?

Menggunakan netsh wlan connect name="your network_name" Anda dapat meminta untuk menyambung ke jaringan WiFi, tetapi tidak dapat memastikan apakah Anda tersambung atau tidak.

Jadi, di CMD / Batch, apa yang akan menjadi baris perintah untuk memeriksa apakah saya terhubung ke jaringan WiFi atau tidak? (Jaringan WiFi mungkin, atau mungkin tidak, memiliki akses jaringan.) [Ini juga harus berfungsi untuk hotspot Seluler juga]

Jika Terhubung, Ini harus menampilkan YES,

Jika TIDAK Terhubung, Seharusnya menampilkan NO,

karena saya ingin menjalankan loop tergantung pada hasil yang saya dapatkan.

Untuk metode alternatif jawaban Milad untuk memeriksa apakah PC Windows Anda saat ini terhubung ke jaringan Wifi, lakukan hal berikut:

  • Buka prompt perintah
  • Masukkan perintah berikut:

WMIC / node: jalur "PutYourPCNameHere" WIN32_NetworkAdapter di mana (NetConnectionID = "Wi-Fi") dapatkan NetConnectionStatus

  • Jika Anda saat ini terhubung ke Wifi, itu akan mengembalikan hasil sebagai berikut:

    NetConnectionStatus 2

  • Jika Anda saat ini tidak terhubung, hasilnya akan seperti ini:

    NetConnectionStatus 7

Anda tidak perlu merujuk PC sebenarnya secara eksplisit. Simpan ini dalam file dengan ekstensi .bat atau .cmd

@Echo off
For /f "usebackq" %%A in (
  `WMIC path WIN32_NetworkAdapter where 'NetConnectionID="Wi-Fi"' get NetConnectionStatus`
) Do If %%A Equ 2 (Echo yes) Else (Echo No)
Pause

Ini harus dimasukkan di jendela cmd (Maaf atas kesalahpahaman)

 For /f %A in ('WMIC path WIN32_NetworkAdapter where (NetConnectionID="Wi-Fi"^) get NetConnectionStatus') Do If %A Equ 2 (Echo yes) Else (Echo No)
Bagikan

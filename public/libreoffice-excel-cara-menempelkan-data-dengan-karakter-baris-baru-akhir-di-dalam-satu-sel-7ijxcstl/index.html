LibreOffice / Excel: cara menempelkan data dengan karakter baris baru / akhir di dalam satu sel

Saya mencoba menempelkan beberapa data ke dalam spreadsheet, tetapi karena data berisi \n karakter di dalamnya, sel di bawah ini ditimpa. Saya ingin menempelkan seluruh bit data ke dalam satu sel.

Misalnya:

some \n
multiline \n
data

Harus masuk ke satu sel, bukan sel dalam 3 baris.

Saya tertarik mengetahui cara melakukan ini untuk LibreOffice dan Excel.

Nah untuk satu sisipan, saya hanya akan menggunakan F2 di sel atau memasukkannya dengan cara lain, sehingga Anda mendapatkan mode edit, lalu menempelkannya.

Cara lain adalah dengan menggunakan input rumus di bagian atas lembar kerja. Pilih sel takdir Anda, lalu pilih masukan rumus, lalu tempel.

Ini lebih rumit, saat Anda memiliki beberapa sel multirow untuk disisipkan.

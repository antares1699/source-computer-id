Apa pintasan keyboard untuk menghapus pemformatan di Office 2008 Mac?

Saya baru saja beralih dari PC ke Macbook Pro baru. Saya senang melihat sebagian besar pintasan keyboard di Word 2008 sama atau hampir sama dengan di Word 2007. Namun, ada satu pintasan keyboard yang sangat saya lewatkan: pintasan pemformatan yang jelas (ctrl + spasi di Windows). Saya tahu fitur tersebut ada karena ada di bilah pemformatan, tetapi saya tidak dapat menemukan pintasan keyboard. Saya mencoba cmd + spasi tetapi itu adalah pintasan keyboard sistem untuk mengganti jenis keyboard. alt + spasi menghapus teks tanpa menghapus pemformatan. ctrl + spasi sepertinya tidak melakukan apa-apa.

Adakah yang tahu apa pintasan keyboard untuk perintah pemformatan yang jelas?

Saya berharap ada satu karena sangat menyakitkan untuk meraih mouse setiap kali saya meletakkan judul dan saya ingin beralih ke gaya paragraf normal ...

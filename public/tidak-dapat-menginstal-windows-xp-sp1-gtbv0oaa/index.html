Tidak Dapat Menginstal Windows XP SP1

Memiliki komputer lama tanpa SP1. Ketika saya mencoba mengunduh SP1, ia memberi tahu saya bahwa ia tidak dapat menemukan server. Bagaimana saya mendapatkan versi offline SP1 untuk diinstal? Saya benar-benar perlu menggunakan SP3 tetapi menurut saya SP1 diperlukan sebelum SP3

Jika tujuan akhir Anda adalah SP3, Anda bisa mulai dari SP2 karena akan memungkinkan Anda untuk beralih dari tanpa SP ke SP2 secara langsung.

Saat ini tidak ada cara untuk menginstal SP1 secara offline melalui publik sumber daya dari Microsoft .

Berikut adalah halaman download untuk image CD SP3:

http://www.microsoft.com/en-us/download/details.aspx?id=25129

Menurut catatan, Anda dapat menginstal SP3 di Windows XP tanpa paket layanan sebelumnya.

EDIT: Informasi tampaknya bertentangan. Halaman ini di MS Technet: http://technet.microsoft.com/en-us/library/cc507836.aspx bertuliskan "Untuk menginstal SP3, komputer Anda harus memenuhi persyaratan perangkat keras dan sistem operasi yang tercantum di bawah ini. Selain itu, komputer Anda setidaknya harus menjalankan Windows XP dengan Service Pack 1 diinstal. Kami menyarankan Anda untuk menginstal Service Pack 2 sebelum menginstal SP3. "

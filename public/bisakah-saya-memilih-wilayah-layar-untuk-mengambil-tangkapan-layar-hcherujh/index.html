Bisakah saya memilih wilayah layar untuk mengambil tangkapan layar?

di windows 10, saya dapat mengambil screenshot seluruh layar dengan menekan Windows key + PrintScreen key.

Dapatkah saya memilih wilayah layar untuk mengambil tangkapan layar?

Terima kasih.

  • Nah, Anda berpotensi dapat melakukan dua hal: Anda dapat memotong tangkapan layar berukuran penuh, atau Anda dapat menggunakan kombo Alt + PrintScrn untuk mengambil tangkapan layar dari jendela yang secara khusus memiliki perhatian.
    — Run5k
  • Alt + PrintScrn tidak melakukan apapun.
    — Tim
  • Bukankah Win 10 memiliki Snipping Tool terkenal yang dimiliki Win 7 .. 8? Ini memungkinkan Anda memilih wilayah.
    — Run CMD
  • Jika Anda memastikan bahwa Anda mengklik jendela yang ingin Anda gunakan untuk tangkapan layar, Alt + PrintScrn harus menangkapnya. Misalnya, jika Anda membuka utilitas Disk Defragmenter, pilih jendela, dan Alt + PrintScrn akan menangkapnya di papan klip.
    — Run5k
  • Jika Anda ingin sedikit lebih fleksibel daripada Snipping Tool, cobalah Greenshot. Open source-nya, dan salah satu utilitas tangkapan layar terbaik di luar sana. Memiliki dukungan untuk mengambil screenshot suatu area dengan zoom dan pembesaran, Anda dapat menggunakan editornya untuk menambahkan teks, garis, dll ke dalamnya dan bahkan memindahkan penunjuk mouse saat berada di dalam. Itu dapat menangkap langsung ke clipboard, file, ke internet, dan banyak hal lainnya.
    — LPChip

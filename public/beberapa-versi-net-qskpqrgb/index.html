Beberapa versi .NET

Di kotak WinXP saya, saya telah menginstal "program" ini:

  • Microsoft .NET Framework 1.1
  • Microsoft .NET Framework 2.0 SP2
  • Microsoft .NET Framework 3.0 SP2
  • Microsoft .NET Framework 3.5 SP1

Apakah saya memerlukan keempat versi tersebut? Dapatkah perangkat lunak yang dikompilasi pada .NET 1.1 berjalan pada runtime 3.5?

Itu tergantung pada Aplikasi .Net. Pertama, menghapus .Net Framework bisa sangat menyakitkan. Setelah terinstal, sebaiknya biarkan di sistem. Jadi pastikan untuk mengetahui persyaratan aplikasi sebelum Anda menginstal .Net Framework.

Pengalaman saya sejauh ini:

  1. Terlihat aplikasi yang membutuhkan 2.0 untuk penginstalan, tetapi menerima jika hanya 3.5 yang diinstal. Saya sebenarnya telah melihat beberapa aplikasi komersial seperti ini. Dan setelah penginstalan, mereka berjalan.

  2. Terlihat aplikasi yang membutuhkan 1.1 dan jika Anda menginstal versi lain, akan merusak aplikasi. Ini adalah aplikasi pembayaran khusus yang dibuat untuk badan pemerintah. Saya tidak terkesan ketika saya harus menghubungi vendor untuk mendapatkan dukungan.

Sebagian besar laptop HP memiliki pengaturan itu di BIOS.

  1. Nyalakan komputer dan tekan berulang kali tombol ESC untuk masuk ke Menu Startup
  2. Tekan F10 untuk pengaturan Bios.
  3. Tekan pada tab Konfigurasi Sistem.
  4. Pilih Konfigurasi Perangkat dari daftar.
  5. Tandai kotak centang di depan NumLock ON saat boot.
  6. Simpan & Keluar

Saya tahu ini adalah trik aneh tetapi berhasil untuk saya (tanpa perlu mengedit registri atau mematikan boot cepat)

  1. Pertama, matikan komputer Anda.
  2. Boot komputer kembali.
  3. Saat Anda mencapai layar login, tekan tombol Num Lock sekali untuk mengaktifkannya.
  4. JANGAN LOGIN ke komputer. Dari layar login, klik tombol daya dan pilih "Shut down" untuk mematikan komputer lagi.
  5. Boot komputer kembali dan Num Lock akan diaktifkan di layar login dan secara permanen menyala, sampai Anda mematikannya sendiri.

Coba ini dan lihat apakah itu berhasil untuk Anda juga.

Buat file bernama Numlock.ps1 dan letakkan yang berikut ini di dalamnya, diikuti dengan baris kosong:

If ([console]::NumberLock -eq $False) {
$wsh = New-Object -ComObject WScript.Shell
$wsh.SendKeys('{NUMLOCK}')}

Buat file bernama Numlock.cmd dan letakkan berikut ini di dalamnya, diikuti dengan baris kosong:

PowerShell.exe -ExecutionPolicy bypass -File "Numlock.ps1"
  • Letakkan file ini di folder yang dapat diakses oleh semua pengguna
  • Buat jalan pintas ke Numlock.cmd
  • Pindahkan pintasan ke folder startup ("C: \ ProgramData \ Microsoft \ Windows \ Start Menu \ Programs \ StartUp")
  • Reboot PC Anda dan numlock harus dihidupkan secara default sekarang

Tombol media macOS High Sierra berhenti bekerja

Sejak memutakhirkan Retina MacBook Pro 15 "Akhir 2013 saya ke macOS High Sierra dari Sierra, saya perhatikan bahwa tombol media (putar, berikutnya, sebelumnya) berfungsi untuk media dalam pemutaran di Safari dan Mail serta iTunes / Quicktime / dll. Tapi sekarang terkadang tombol berhenti merespons atau melakukan apa saja.

Apakah ada cara untuk memperbaiki masalah ini?

Anda mungkin harus menunggu Apple untuk memperbaikinya. Saya perhatikan bahwa ketika saya mengalami ini, jika saya memulai ulang aplikasi terakhir, kunci media mempengaruhi mereka kadang-kadang mulai bekerja lagi.

Misalnya, saya telah menonton video YouTube di Safari dan kemudian mulai mendengarkan musik. Saya tidak bisa mendapatkan kunci media berfungsi untuk iTunes dan mereka juga tidak berfungsi di YouTube. Keluar dari Safari memungkinkan mereka bekerja untuk iTunes lagi. Memulai Safari lagi tidak merusak ini.

Sampai Apple memperbaikinya, Anda dapat menggunakan BeardedSpice . Aplikasi ini mencari aplikasi / halaman web terakhir untuk memiliki fokus dan media yang dapat diputar dan kemudian mengarahkan input kunci media ke aplikasi / halaman web tersebut.

Pada dasarnya, ini menyediakan fungsionalitas yang tampaknya ingin dicapai Apple di High Sierra, tetapi berfungsi lebih konsisten. Sayangnya, respons ini jauh lebih lambat daripada perilaku asli (hingga satu detik setelah menekan tombol media).

Jika yang Anda pedulikan hanyalah iTunes Jawaban Dan Messing mungkin lebih baik untukmu.

Aktifkan Kembali Kontrol Paket di Sublime Text 2

Menggunakan fitur "nonaktifkan paket" dari ekstensi Kontrol Paket di Sublime Text 2 Saya tidak sengaja menonaktifkan Kontrol Paket itu sendiri, dan sekarang saya tidak yakin bagaimana cara mengaktifkannya kembali. Saya menduga ini adalah pengaturan yang disimpan di suatu tempat, tetapi untuk kehidupan saya, saya tidak dapat menemukannya. Ada saran?

4
Millhouse — Sumber

1 Jawaban

6
6

Hapus "Kontrol Paket" dari "ignored_packages" di dalam file Preferensi Anda.

Buka Preferences > Settings-User. Anda akan melihat sesuatu seperti:

"ignored_packages":
[
  "Vintage",
  "Package Control"
],

Jadikan:

 "ignored_packages":
[
  "Vintage"
],

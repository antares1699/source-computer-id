Tata letak kacau di "htop"

Saat ini saya menggunakan openSUSE Tumbleweed (dengan KDE Plasma 5, jika itu penting).

Saya menggunakan htop untuk memantau sumber daya sistem, dan akhir-akhir ini saya mendapatkan tata letak yang berantakan, terutama di area paling atas layar. Informasi tidak mudah dibaca, dan tidak berguna dalam keadaan ini.

Bagaimana saya bisa memecahkan masalah ini?

Itu disebabkan oleh ncurses.

export TERM=linux

Mungkin mengatur ulang konfigurasi ke default akan membantu:

mv ~/.config/htop/htoprc{,.bak}
rm ~/.config/htop/htoprc
touch ~/.config/htop/htoprc

Coba htop sesudahnya. Jika tidak memperbaikinya, Anda dapat memulihkan file konfigurasi lama Anda:

mv -f ~/.config/htop/htoprc.bak ~/.config/htop/htoprc

Tidak dapat menjalankan monitor pada resolusi 1080p asli di Windows 7

Saya memiliki monitor ViewSonic VX2453 24 "yang mendukung 1920x1080p, terhubung menggunakan HDMI. Namun pada resolusi itu, desktop mematikan layar. Dengan menggunakan panel kontrol Nvidia, saya harus mengaturnya ke resolusi kustom 1804x1014 agar dapat ditampilkan dengan benar.

Monitor memiliki driver yang diinstal dengan benar (nama model yang benar muncul di panel kontrol setelah menginstal driver), dan saya menjalankan 64 bit Win7 Ultimate. Saya memiliki kartu GeForce 560 Ti, jika itu membantu. Mengapa ini terjadi?

Biasanya, masalah desktop-off-the-screen adalah salah satu monitor dan bukan komputer.

Tapi satu hal terakhir: periksa untuk memastikan Anda menggunakan kecepatan refresh 60hz. Itu tergantung pada pengaturan Anda tentang bagaimana Anda melakukan ini, baik di windows atau di pusat kendali nVidia Anda.

  1. Atur komputer Anda untuk mengirimkan resolusi 1920x1080 ke monitor.

  2. Buka menu utama monitor (1 di samping). Cari sesuatu yang disebut "Penyesuaian Otomatis". Ini harus dipilih terlebih dahulu. Cobalah, itu mungkin memperbaiki masalah. Jika tidak, mungkin saja disebabkan masalah. Aneh ya ?, tetapi kadang-kadang terjadi dengan kartu video tertentu.

  3. Jadi selanjutnya coba atur ulang semua pengaturan ke nilai default. Di monitor Anda, buka menu (1), dan gulir ke bawah ke Memory Recall di bagian bawah. Opsi di sana akan mengatur ulang kembali ke pengaturan pabrik.

Jika bahwa tidak memperbaikinya, mari sesuaikan semuanya secara manual.

  1. Buka kembali menu utama (1).
  2. Gulir ke bawah ke tangan kecil yang memegang garpu tala (Penyesuaian Gambar Manual).
  3. Gunakan posisi H / V untuk secara manual memindahkan gambar pada monitor di sekitar layar. Gunakan ukuran H untuk mengecilkan gambar di layar jika terlalu besar. Dengan mengutak-atik pengaturan ini, Anda harus dapat memusatkan output dengan sempurna di monitor Anda. Monitor Anda juga tampaknya memiliki tombol fine tune yang mungkin berguna, tetapi saya hanya akan mencobanya setelah mengutak-atik pengaturan posisi H. Ukuran dan posisi H / V.

Cheers, itu seharusnya sudah memperbaikinya.

Ini adalah masalah pemindaian berlebih, jika Anda memiliki monitor ViewSonic yang terhubung dari soket HDMI mini dari kartu grafis, ke port HDMI di monitor, panel kontrol NVIDIA melihatnya sebagai sinyal HDMI. Ini menciptakan overscan.

Cara termudah untuk menghentikan overscan adalah dengan memasukkan entri kustom ke registri komputer. Tautan berikut ini menjelaskannya dengan lebih baik, dan menurut saya info registri yang disediakannya adalah untuk monitor ViewSonic tertentu, jadi bacalah dengan baik sebelum mencoba.

Saya menggunakan monitor ViewSonic dan metode ini berhasil, satu-satunya downside adalah: Anda harus mengulanginya jika Anda memperbarui driver — tetapi hanya perlu beberapa menit.

Perbaiki masalah Overscaling pada monitor VX2435 (dan lainnya) Monitor PC

Jika Anda mencoba metode ini dan tidak berhasil, hapus saja entri registri yang Anda buat.

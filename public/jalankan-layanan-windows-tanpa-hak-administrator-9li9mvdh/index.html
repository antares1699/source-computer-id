Jalankan layanan Windows tanpa hak administrator

Bagaimana cara memulai dan menghentikan layanan Windows 2008 tanpa hak administrator?

Saya ingin memberikan hak pengguna non-admininstrator untuk memulai dan menghentikan layanan Windows.

Ada dua skenario berbeda yang dapat ditafsirkan dari kiriman Anda:

  • Orang yang tidak memiliki akses administrator ingin memulai layanan

Dalam skenario ini, tidak mungkin untuk memulai layanan jika pengguna non-administrator tidak memiliki hak istimewa. Jika Anda tidak memiliki hak istimewa, Anda tidak dapat menghindari izin tanpa semacam retakan yang merusak model keamanan Windows.

  • Orang yang tidak memiliki akses administrator ingin memberikan kemampuan untuk memulai layanan kepada non-administrator

Dalam skenario ini, administrator bisa memberikan kemampuan untuk memulai layanan kepada non-administrator, jadi ini mungkin.

Dari StackOverflow :

Dua arah:

1. Edit properti layanan dan setel pengguna Log On. Hak yang sesuai akan diberikan secara otomatis.

2. Atur secara manual: Buka Alat Administratif -> Kebijakan Keamanan Lokal -> Kebijakan Lokal -> Penetapan Hak Pengguna. Edit item "Masuk sebagai layanan" dan tambahkan pengguna domain Anda di sana.

Serupa dengan file, layanan juga memiliki Access Control List (ACL) yang menentukan siapa yang diizinkan untuk memulai / menghentikan layanan. Masalahnya adalah Microsoft tidak pernah menerapkan GUI atau alat baris perintah untuk memodifikasi ACL ini.

Ada alat freeware pihak ketiga yang disebut SCAcl yang memungkinkan untuk mengubah ACL layanan dan jika Anda memerlukannya, izinkan pengguna atau grup non-administratif tertentu untuk memulai dan / atau menghentikan layanan.

SCAcl http://www.losoft.de/ls_scacl.html

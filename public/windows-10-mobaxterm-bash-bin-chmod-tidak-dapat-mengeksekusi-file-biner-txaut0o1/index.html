Untuk memulai kembali dari penginstalan bersih, coba prosedur ini:

  • Buka MobaXterm.
  • Jalankan perintah open / dari terminal lokal => Windows Explorer akan terbuka di folder slash Anda.
  • Tutup MobaXterm.
  • Ubah nama folder slash Anda di Windows Explorer dari slash menjadi slash-old
  • Buka MobaXterm
  • Verifikasi apakah sekarang berfungsi seperti yang diharapkan.

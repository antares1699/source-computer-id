Bagaimana cara memaksa Internet Explorer untuk menjaga ukuran jendelanya saat dipasang ke dok?

Menggunakan Internet Explorer 10 di Windows 8, ukuran jendela aku s diingat, saat jendela IE berada tidak merapat / membentak ke tepi.

Segera setelah saya menggalangkan IE ke satu sisi layar saya, menutupnya dan membukanya kembali, ukuran jendela disetel ke sesuatu yang lain (mungkin ukuran jendela yang tidak digalangkan?)

Langkah-langkah untuk menunjukkan perilaku

1.) Mulai Internet Explorer. Ini ditampilkan di posisi jendela terakhirnya

2.) Tarik jendela dengan mouse ke satu sisi (atau gunakan + → ). Internet Explorer sedang dipasang ke dok

3.) Tutup Internet Explorer

4.) Mulai Internet Explorer. Itu sedang ditampilkan di bagian terakhirnya dilepas posisi jendela

Alih-alih pintasan Windows bawaan, gunakan sesuatu seperti DisplayFusion untuk membuat pintasan Anda sendiri yang benar-benar mengubah ukuran jendela, yang kemudian akan diingat oleh IE (dan aplikasi lain). Saya pribadi menggunakan Menang + Alt + Papan angka 4 dan Menang + Alt + Papan angka 6 .

Di Windows 7 dan yang lebih lama, Revolusi WinSplit bagus untuk ini, tetapi tidak lagi didukung - dan menurut saya versi terakhir tidak berfungsi pada Windows 8. Ada beberapa alternatif yang mungkin layak untuk dilihat.

Bagian penting yang dirindukan semua orang dengan ini adalah bahwa mereka benar-benar perlu melakukan Tutup File setelah mengubah ukuran ke ukuran yang sesuai dan tidak mengklik x merah atau metode lain untuk menutup Internet Explorer. Jika langkah ini tidak diikuti, ukuran layar / lokasi tidak disimpan atau diperbarui. Di versi Internet Explorer yang lebih baru, Anda perlu menekan tombol Alt pada keyboard Anda bahkan sebelum menu file akan muncul di Internet Explorer dengan pengaturan default.

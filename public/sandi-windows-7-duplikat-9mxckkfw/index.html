Dengan asumsi Anda tidak memiliki enkripsi seluruh hard drive (seperti BitLocker), Anda dapat menggunakan Windows NT / 2000 / XP / Vista / 7 milik Petter Nordahl-Hagen editor kata sandi offline .

  • Ini adalah utilitas untuk menyetel (ulang) sandi pengguna mana pun yang memiliki akun (lokal) yang valid di sistem Windows NT / 2000 / XP / 2003 / Vista Anda, dengan mengubah sandi terenkripsi di file SAM registri.
  • Anda tidak perlu mengetahui kata sandi lama untuk membuat yang baru.
  • Ia bekerja secara offline, yaitu, Anda harus mematikan komputer Anda dan boot dari floppy disk atau CD. Boot-disk menyertakan hal-hal untuk mengakses partisi NTFS dan skrip untuk merekatkan semuanya.
  • Bekerja dengan syskey (tidak perlu mematikannya, tetapi Anda bisa jika Anda kehilangan kunci)
  • Akan mendeteksi dan menawarkan untuk membuka kunci akun pengguna yang dikunci atau dinonaktifkan!

Kon-Boot

Live cd Kon-Boot dalam kondisi kompilasi saat ini memungkinkan untuk masuk ke sistem linux sebagai pengguna 'root' tanpa mengetik kata sandi yang benar atau untuk meningkatkan hak istimewa dari pengguna saat ini ke root. Untuk sistem Windows, ini memungkinkan untuk memasukkan profil yang dilindungi kata sandi tanpa sepengetahuan kata sandi.

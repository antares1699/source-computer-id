Berapa banyak koneksi yang diperbolehkan ke folder atau printer bersama Windows 7 Home Premium?

Saya memiliki klien yang menjalankan bisnis kecil dengan 4 sistem desktop, dua di antaranya tidak mahal [

Sistem XP Pro saat ini digunakan sebagai "server" file untuk lembar waktu dan data QuickBooks. Ini juga berbagi printer ink jet HP. Klien ingin menonaktifkan sistem ini karena (1) jelek [memang] dan (2) menggunakan terlalu banyak daya [memang].

Jika kami berbagi folder di salah satu sistem Windows 7 Home Premium, apakah akan ada masalah saat menghubungkannya dengan hingga 3 komputer lain?

Bagaimana dengan printer sharing?

Saya samar-samar ingat melihat bahwa Windows 7 kurang dapat digunakan untuk tujuan "server" dan memiliki batasan yang ketat pada jumlah klien. Tapi sepertinya saya tidak bisa menemukan angka-angka itu.

Di jaringan saya sendiri (lebih dari 12 sistem), kami tidak memiliki masalah berbagi dari Windows 7 Ultimate ke beberapa sistem lain jika diperlukan.

Saya malu karena saya tidak dapat menemukan jawaban untuk pertanyaan ini dalam beberapa hari pencarian.

Saya dapat melakukan peningkatan kapan saja dari salah satu sistem ini ke Pro jika itu akan meningkatkan kemampuan untuk berbagi darinya. Saya tidak dapat meyakinkan klien untuk menempatkan "server nyata" ke dalam jaringan mereka.

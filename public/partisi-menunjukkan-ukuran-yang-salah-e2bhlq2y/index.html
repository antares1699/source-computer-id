Partisi Menunjukkan Ukuran Yang Salah

Saya mengalami masalah ukuran kecil di drive C saya (di mana Windows XP diinstal). Jadi saya memutuskan untuk memperbesar ukuran drive ini dengan menggunakan perangkat lunak "EASEUS Partition Master 4.0 Home Edition".

Saya tinggal di drive E saya yang lain yang memiliki total ruang 42GB untuk memberi ruang ke drive C menggunakan perangkat lunak di atas. Saya mengalokasikan sekitar 10GB lebih banyak ruang ke drive C dari E. Tetapi selama proses tersebut, perangkat lunak memberikan kesalahan berikut.

"Pembaruan informasi sistem gagal"

Jadi prosesnya tidak berhasil. Sekarang masalahnya adalah drive E saya hanya menampilkan total ruang 11GB padahal sebenarnya 42GB. Drive beserta semua file dan foldernya dapat diakses tetapi hanya berukuran salah. Ini menunjukkan 11GB yang salah bahkan dari opsi Manajemen Disk. Kemana perginya ukuran saya yang lain?

Bagaimana cara mengatasi masalah ini?
Bagaimana cara memulihkan sisa ukuran drive E saya?

Edit: OS: format WinXP, NTFS

Saya mencoba ini menggunakan Thunderbird 3, Outlook Express 6 dan Outlook 2003, tapi saya harap ini dapat membantu siapa pun yang membaca QA ini.

Di Thunderbird:

  • CTRL + A untuk memilih semua pesan
  • klik kanan & Simpan Sebagai
  • simpan semua pesan di dalam folder (disimpan sebagai file .eml)

Buka Outlook Express

  • seret semua file eml yang Anda simpan sebelumnya ke Outlook Express

Buka Outlook

  • klik Impor di menu tarik-turun
  • pilih Impor dari program atau file lain
  • pilih Outlook Express

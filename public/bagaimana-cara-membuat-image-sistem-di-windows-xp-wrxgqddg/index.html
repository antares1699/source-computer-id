Sistem dasar untuk Anda adalah:

  1. Buat satu gambar setelah Anda mengaturnya seperti yang Anda inginkan dan sebelum virus apapun dapat masuk ke komputer. Cadangkan gambar ke USB / CD-DVD / drive lain lalu beri label folder dengan tanggal dan biarkan hingga pemulihan diperlukan.
  2. Atur jadwal pencadangan diferensial (setiap hari jika diinginkan) dan buat cadangan setidaknya ke partisi yang berbeda. Anda harus mencadangkan ke USB, dll. Seminggu sekali, untuk berjaga-jaga. Anda dapat menggunakan cadangan Windows / NT untuk bagian ini.
  3. Bersihkan cadangan dengan CD anti-virus sebelum pemulihan karena mereka akan terinfeksi. Gambar pemulihan yang akan saya gunakan adalah Macrium Reflect ™ Free Edition. Ini juga membuat CD penyelamatan yang dapat di-boot untuk membantu. Itu datang dengan a file bantuan / instruksi yang baik .

Daftar sistem cadangan bagus lainnya:

Norton / Acronis True image / dll

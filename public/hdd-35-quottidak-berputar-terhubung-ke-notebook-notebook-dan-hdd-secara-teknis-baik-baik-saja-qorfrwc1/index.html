HDD 3,5 "tidak berputar terhubung ke Notebook. Notebook dan HDD secara teknis baik-baik saja

Saya menghubungkan HDD 1TB 3,5 "dengan kabel breakout ke notebook saya. Saat saya menyalakan notebook, HDD tidak berputar juga tidak dikenali. Saya mengaktifkan / menonaktifkan opsi Bios AHCI dan LBA ke Dos / lainnya, dengan tidak berhasil. Mungkinkah konektor daya sata pada notebook hanya memberikan daya yang cukup untuk HDD 2,5 "?

Secara teknis semuanya bekerja dengan baik. Biasanya saya menggunakan notebook dengan SSD. HDD 2,5 "yang terhubung ke kabel breakout berfungsi. HDD 3,5" yang terhubung ke PC desktop saya berfungsi. Saya mendapatkan beberapa HDD lama yang siap dijual melalui ebay. Memiliki PC ke-2 untuk diuji dan dihapus akan sangat berguna bagi saya.

Sebagian besar HDD 2,5 "hanya memerlukan rel daya 5V dari kabel daya SATA untuk berfungsi penuh. Jika Anda melihat HDD portabel apa pun yang hanya didukung oleh USB, Anda akan melihat tidak ada kabel daya tambahan dan hanya disk 2,5"; karena USB hanya menyediakan 5V. Ya, beberapa disk dilengkapi dengan kabel USB "Y" untuk daya tambahan tetapi ini untuk arus tambahan, bukan tegangan.

HDD 3,5 "memerlukan rel daya 12V dan 5V. Sekali lagi, HDD desktop 3,5" biasanya dilengkapi (atau memerlukan) sumber daya eksternal karena sambungan USB itu sendiri tidak dapat menyediakan daya yang cukup.

Saya akan menebak bahwa notebook SATA Anda tidak memiliki daya 12V karena tidak pernah membutuhkan daya HDD 3,5 ".

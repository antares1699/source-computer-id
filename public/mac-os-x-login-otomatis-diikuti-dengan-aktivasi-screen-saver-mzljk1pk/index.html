Tambahkan skrip login. Itu bisa dilakukan melalui Shell:

open -a ScreenSaverEngine

atau melalui Applescript:

tell application "ScreenSaverEngine" to launch

Applescript mungkin adalah cara termudah jika Anda tidak terbiasa dengan skrip Shell. Buka Script Editor (Spotlight it), tempelkan script di atas, dan klik File->Save As. Pilih Application sebagai format file. Simpan di suatu tempat, ~/Applications, ~/Library/Scripts, dll. Buka Accounts.prefPane, klik tab item masuk untuk akun Anda, tekan tombol tambah, lalu cari di mana Anda menyimpan skrip Anda.

Jika Anda benar-benar khawatir tentang keamanan, Anda harus menonaktifkan autologin.

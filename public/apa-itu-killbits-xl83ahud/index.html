Apa itu killbits?

Windows baru saja menyarankan saya untuk menginstal "Pembaruan Keamanan Kumulatif untuk ActiveX Killbits." Apa itu Killbits? Bagaimana saya / komputer saya dirugikan jika saya tidak menginstal pembaruan Killbit?

Dari Wikipedia :

Tujuan utama dari killbit adalah untuk tutup lubang keamanan. Jika vendor menemukan bahwa ada keamanan lubang di versi tertentu dari file Kontrol ActiveX, mereka dapat memintanya Microsoft mengeluarkan "killbit" untuk itu. Pembaruan Killbit biasanya diterapkan ke sistem operasi Microsoft Windows melalui Pembaruan Windows.

Killbit adalah sebuah flag di Windows Registri yang menandai CLSID sebagai tidak aman. CLSID bertindak sebagai serial nomor untuk perangkat lunak yang dimaksud - GUID unik yang harus ada untuk masing-masing bagian dari perangkat lunak yang berperilaku sebagai file Kontrol ActiveX. Jika ActiveX container menemukan bahwa CLSID dari a entri killbit cocok dengan CLSID dari perangkat lunak, kemudian perangkat lunak diblokir agar tidak berjalan di penampung ActiveX. Jika vendor ingin merilis file versi yang diperbarui kemudian mereka merilisnya dengan CLSID yang berbeda.

Jadi pada dasarnya, daftar KillBits adalah daftar komponen ActiveX yang tidak akan pernah diizinkan untuk berjalan di sistem Anda karena mereka memiliki lubang keamanan. Daftar ini membutuhkan pembaruan rutin.

Anda juga dapat memeriksa di sini untuk beberapa informasi yang lebih terperinci tentang KillBits daripada yang ditawarkan Wikipedia: http://blogs.technet.com/b/srd/archive/2008/02/06/the-kill_2d00_bit-faq_3a00_-part-1-of-3.aspx

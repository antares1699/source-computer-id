Cara mengaktifkan kembali tampilan laptop di Windows XP saat buta

Ayah saya memiliki laptop Lenovo S10. Saat berlibur, dia menghubungkannya ke monitor eksternal dan menonaktifkan tampilan laptop dengan Dialog Properti Tampilan Windows. Sebelum pergi, dia menonaktifkan layar eksternal dengan mengira layar laptop akan aktif secara otomatis. Tidak.

Sekarang ketika kita mem-boot laptop, tidak ada tampilan yang berfungsi. Tombol Fn (misalnya Fn + F2 dan Fn + F3) tidak berguna. Saya bisa mendapatkan tampilan laptop untuk "on" dan "off" tetapi hanya menampilkan kegelapan. ("Aktif" adalah kegelapan yang menyala.) Saya juga dapat mendengar monitor eksternal diklik saat saya menggunakan tombol Fn untuk mengalihkan tampilan, tetapi sekali lagi, layar kosong.

Layar BIOS baik-baik saja. Layar pemuatan Windows XP baik-baik saja. Tetapi ketika sistem operasi dimuat, ia tidak tahu apa yang harus dilakukan. Mode aman hanya menampilkan teks boot. Itu tidak benar-benar menampilkan OS Windows.

Saya telah menyambungkan kembali monitor eksternal dan boot ke mode aman, tetapi hasilnya sama.

Bagaimana Anda mereset pengaturan tampilan Windows XP sementara tidak dapat mengakses Windows XP?

Ini tembakan yang panjang tapi ...

  1. Tekan Tombol Windows (buka menu mulai)
  2. Tekan 'C' (panel kontrol terbuka)
  3. Tekan 'D' (harus menyorot "display" tetapi ini dapat menyorot sesuatu yang lain jika karena alasan tertentu panel kontrol memiliki kumpulan ikon yang berbeda dari default)
  4. Tekan 'Enter' (Biarkan properti tampilan panel kontrol memuat)
  5. Tekan 'Shift-tab' (harus menyorot tab)
  6. Tekan panah kiri 4 kali (Anda akan dibawa ke "pengaturan")
  7. Tab dua kali (harus memilih kotak kombo pemilihan monitor)
  8. Tekan panah bawah (beralih ke properti monitor sekunder)
  9. Tekan 'Alt-U' (memberi tahu properti untuk "digunakan sebagai utama")
  10. Tekan 'Alt-A' untuk menerapkan pengaturan

Jika Anda memiliki komputer lain yang menjalankan Windows XP, Anda selalu dapat mencoba langkah-langkah ini juga untuk "merasakannya" sebelum melakukannya secara buta. Hal lain untuk dicoba jika ini tidak berhasil adalah lewati langkah # 8 dan coba jadikan monitor 1 sebagai yang utama. Saya membuat asumsi bahwa monitor eksternal terlihat sebagai monitor 1 tetapi mungkin monitor laptop tersebut adalah nomor 1.

Dalam kasus saya, driver tampilan telah dihapus sepenuhnya. Saya mengunduhnya ke thumb drive dari sini: http://consumersupport.lenovo.com/ot/en/driversdownloads/drivers_list.aspx?categoryid=43

Saya kemudian menggunakan Microsoft Narrator untuk menavigasi Windows:

  1. Boot Windows
  2. Ctrl-Esc untuk membuka menu mulai
  3. Ketik R untuk lari
  4. Ketik "narator" dan tekan Enter

Menggunakan tombol tab, tombol panah dan spasi untuk bergerak dan memilih opsi, dan Narator untuk memberi tahu saya apa yang terjadi di balik layar kosong saya, saya memasukkan thumb drive, membuka Windows Explorer, meluncurkan penginstal driver, melalui InstallShield dan reboot.

Kemudian saya menyalakan Narator lagi, menggunakan Ctrl-Esc, R, "control panel", enter untuk masuk ke panel kontrol, pergi ke System> Tab Hardware> Device Manager> Display Adapters> adaptor yang baru dipasang, dan memilih untuk menyalakan adaptor. Saya keluar dari dialog adaptor, keluar, pengelola perangkat, dan keluar dari jendela sistem. Masih di panel kontrol, saya membuka kontrol Tampilan, pergi ke tab pengaturan dan memilih adaptor yang tepat.

Berhasil!

Barang terakhir yang diketahui berhasil untuk saya.

Tepat sebelum Windows mulai memuat, tekan F8. Pilih Barang terakhir yang diketahui dari menu. Kadang-kadang dibutuhkan beberapa kali percobaan untuk memunculkan menu. Anda punya waktu yang tepat untuk menampilkannya.

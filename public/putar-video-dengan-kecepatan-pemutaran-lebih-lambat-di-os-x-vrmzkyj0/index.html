Putar video dengan kecepatan pemutaran lebih lambat di OS X

Saya memiliki film yang saya rekam dengan 240 frame per detik. Saya ingin memutarnya dengan 24 frame per detik di Mac saya. Bagaimana saya bisa melakukan ini?

Bagaimana cara memperlambat pemutaran hingga kecepatan 10% untuk file MP4?

Dalam opsi QuickTime Player, mengklik tombol maju akan meningkatkan kecepatan sebesar 0,1. Mengklik tombol mundur memutar video secara terbalik. Anda dapat memperlambat video dengan menjalankan ini di AppleScript Editor:

tell application "QuickTime Player" to set rate of document 1 to 0.1

Atau melalui terminal:

osascript -e 'tell application "QuickTime Player" to set rate of document 1 to 0.1'

Kecepatan pemutaran VLC dapat diubah sedikit demi sedikit dari bilah menu.

Di menu, terendah yang dapat Anda atur adalah 0,25. Jika Anda menekan ⌘- tarif dapat disetel lebih rendah lagi, tetapi hanya untuk nilai tetap 0,5, 0,33, 0,25, 0,12, 0,06, 0,03, dan 0,02.

Di Movist, Anda dapat menekan ⇧⌘ ← atau menyesuaikan penggeser di HUD kontrol:

Movist dapat mengurangi kecepatan bermain hingga setengahnya, tidak lebih.

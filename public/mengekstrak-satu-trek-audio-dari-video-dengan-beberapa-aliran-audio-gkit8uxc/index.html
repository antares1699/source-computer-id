Mengekstrak satu trek audio dari video dengan beberapa aliran audio

Saya memiliki file MKV dengan 4 aliran audio MPEG AAC. Saya ingin mengekstrak salah satu aliran tersebut ke dalam file audio terpisah, tetapi fitur "Konversi" di VLC tampaknya tidak memiliki opsi untuk memilih aliran audio, dan jika saya melanjutkan, file yang dihasilkan tidak dapat diputar.

Saya sudah mencoba mencari solusinya di Google, tetapi saya tidak menemukan hasil apa pun tentang beberapa aliran audio dalam satu file video.

Anda dapat mengubah trek audio di menu Preferensi VLC.

VLC Preference 
   Audio 
      Show All 
          Input Codec 
              Track Setting 
                   Audio Track -1

ubah -1 menjadi 0 untuk trek audio 1, 1 untuk trek audio 2, simpan dan lakukan konversi seperti biasa.

Jangan lupa untuk menyetel -1 lagi sebelum Anda keluar!

Saya yakin Anda dapat menggunakan Handbrake untuk ini.

Unduh untuk platform Anda di https://handbrake.fr/ .

Setelah dipasang, klik tombol "Sumber". Pilih file MKV. Ini akan dimuat, dan dalam menu di bawah "pengaturan keluaran" Anda dapat memilih audio. Sebuah daftar akan muncul dengan trek audio apa yang ingin Anda ekstrak.

Berikut ini bekerja pada Windows 7. Ini mungkin bekerja pada orang lain.

Opsi Folder memiliki tiga tab. Anda dapat memilih tab mana yang dipilih saat dibuka:

Tab Umum - C:\Windows\System32\rundll32.exe shell32.dll,Options_RunDLL 0

Lihat Tab - C:\Windows\System32\rundll32.exe shell32.dll,Options_RunDLL 7

Tab Pencarian - C:\Windows\System32\rundll32.exe shell32.dll,Options_RunDLL 2

Di mana Manajer Kredensial di Win10?

Saya menggunakan Windows versi lokal (Polandia). Itu berarti mengetik 'Credential Manager' dalam pencarian tidak menghasilkan sesuatu yang relevan. Saya tidak dapat menemukan apa pun yang cocok dengan nama dalam bahasa saya di Pengaturan Windows. Semua artikel mengatakan 'ketik pencarian' atau 'temukan di pengaturan' tetapi tanpa mengetahui terjemahan apa (... tentu saja tidak langsung karena tidak ada yang cocok) yang digunakan Microsoft, saya tidak dapat menebak di mana itu. Bisakah Anda memberikan mis. tangkapan layar, panduan terperinci tentang elemen UI mana yang harus diklik (tidak bergantung pada nama bahasa Inggris) atau perintah - atau hanya terjemahan Polandia yang kebetulan diterapkan Microsoft - untuk meluncurkannya?

(Saya salah mengetik kata sandi saya di TortoiseGit dan mengingatnya seperti itu dan sekarang gagal masuk ke repositori karena telah menyimpan kredensial yang tidak valid dan saya tidak dapat menghapusnya.)

SF. — Sumber
  • Lakukan Win + R pada keyboard (buka kotak Run), lalu ketik: certlm.msc, tekan ok. Ini akan memunculkan sertifikat untuk mesin lokal, jika Anda hanya ingin melihat sertifikat untuk pengguna saat ini, ketik: certmgr.msc
    — Moab
  • Saya berhasil menghapus sertifikat buruk dengan cara berbeda: di command prompt: cmdkey /list untuk menampilkan sertifikat dan pengenalnya, lalu cmdkey /delete:git:http://gitlab.... untuk menghapusnya.
    — SF.

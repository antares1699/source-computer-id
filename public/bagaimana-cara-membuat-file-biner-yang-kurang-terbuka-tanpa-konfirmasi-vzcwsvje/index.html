Bagaimana cara membuat file biner yang kurang terbuka tanpa konfirmasi?

Saya memiliki beberapa file log yang berisi beberapa byte yang bukan karakter yang dapat ditampilkan (karena tanda tangan biner pada permintaan).

Setiap kali saya membukanya dengan lebih sedikit saya ditanya:

$ less /var/log/file.log       
"/var/log/file.log" may be a binary file.  See it anyway?

Bagaimana cara mengurangi lewati langkah ini dengan opsi atau konfigurasi baris perintah?

4

7
7

Gunakan opsi -f:

-f or --force
              Forces non-regular files to be opened.  (A non-regular file is a directory or
              a device special file.)  Also suppresses the warning message  when  a  binary
              file  is  opened.   By  default,  less will refuse to open non-regular files.
              Note that some operating systems will not allow directories to be read,  even
              if -f is set.

Anda dapat menambahkan opsi ini ke variabel lingkungan LESS di .bashrc untuk menjalankan less secara default dengan -f.

export LESS='-f'

Tambahan tamu: Header kernel tidak ditemukan untuk kernel target

Saya mengalami kesulitan menginstal Penambahan Tamu di Oracle Linux 7. Saya memiliki VirtualBox versi 6.1.4. Saya memiliki tambahan tamu versi 6.0.4

Mesin memiliki versi kernel 4.14.35-1902.7.3.1. Saya berhasil menginstalnya dalam versi kernel 4.1.12-124.24.3 di atasnya. Untuk melakukannya, saya mengikuti instruksi dari halaman ini: https://www.thegeekdiary.com/centos-rhel-7-change-default-kernel-boot-with-old-kernel/

Langkah-langkah instalasi:

yum -y install kernel-uek-firmware-4.1.12-124.24.3.el6uek.noarch.rpm
yum -y install kernel-uek-4.1.12-124.24.3.el6uek.x86_64.rpm
yum -y install kernel-uek-devel-4.1.12-124.24.3.el6uek.x86_64.rpm

awk -F\' '$1=="menuentry " {print $2}' /etc/grub2.cfg
grub2-set-default 0 #depends on the above result
grub2-mkconfig -o /boot/grub2/grub.cfg
And reboot.

Ketika saya menjalankan uname -r saya mendapatkan ini: 4.1.12-124.24.3.el6uek.x86_64

Pada titik ini, semuanya baik-baik saja sampai saya harus menginstal Penambahan Tamu.

Ketika saya menjalankan ./VBoxLinuxAdditions.run kesalahan ini dikembalikan:

Saya memutar ulang /sbin/rcvboxadd setup tetapi itu adalah pesan log yang sama. File log baru saja menampilkan "modprobe vboxguest gagal". Saya juga mengekspor variabel lingkungan KERN_DIR sebagai /usr/src/kernels/4.1.12-124.24.3.el6uek.x86_64

Apa yang saya lakukan salah? Apakah ada yang tahu cara menginstalnya dengan benar? Jika ada yang punya ide, saya akan sangat menghargai umpan balik.

Pada VM CentOS 6 saya, paket pengembangan kernel tidak diinstal, jadi saya harus melakukan:

yum install kernel-devel

Ini, bagaimanapun, menciptakan ketidaksesuaian antara versi pengembangan kernel (2.6.32-754.31.1.el6) dan versi kernel sebenarnya (2.6.32-754.el6). Anda dapat memeriksa semua versi ini menggunakan yum list kernel*. Jadi, saya kemudian memperbarui versi kernel saya dan mem-boot ulang:

yum update kernel kernel-headers
reboot now

Di sistem saya, file Penambahan Tamu autorun.sh masih belum selesai setelah ini, karena paket gcc hilang:

yum install gcc

Setelah menginstal gcc, penginstalan Penambahan Tamu berhasil, dan berlaku setelah boot ulang lagi.

Mari kita lihat pesannya:

VirtualBox Guest Additions: Kernel headers not found for target kernel
5.3 (in may case).

itu kernel target adalah versi kernel yang digunakan OS. Anda bisa melihatnya dengan:

uname -r

5.3

Itu header kernel dibuat oleh paket kernel-devel: Anda dapat melihat versi dari kernel-devel dengan:

dnf list kernel-level*

kernel-devel.x86_64                             5.10

Saya memiliki header Kernel untuk kernel versi 5.10, tetapi kotak virtual mencoba menggunakan header untuk kernel versi 5.3.

Anda harus mem-boot OS menggunakan kernel versi 5.10, bukan 5.3

untuk melihat apakah Anda memiliki kernel versión 5.10:

dnf list kernel-core*

kernel-core.x86_64                             5.3                           @updates 
kernel-core.x86_64                             5.9                          @updates 
kernel-core.x86_64                             5.10

Itu ada.

Sekarang Anda hanya perlu memperbarui grub2 agar dapat melihat versi kernel baru:

grub2-mkconfig –o /boot/grub2/grub.cfg

reboot

Sekarang periksa lagi versi kernel mana yang digunakan OS:

  uname –r
 

 5.10

ini memecahkan masalah saya.

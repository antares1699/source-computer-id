Menemukan file hanya dengan mengetahui ekstensi?

Saya agak tidak teratur dalam kebiasaan menyimpan file saya, jadi saya akhirnya menggunakan fungsi Pencarian di Windows Explorer lebih sering daripada yang seharusnya. Saya bertanya-tanya, mungkinkah Windows menemukan semua file dengan ekstensi tertentu (seperti, katakanlah misalnya semua file .rar) jika ya, bagaimana caranya?

4

5
5
  • Buka Windows Explorer.
  • Buka C:\
  • Ketik *.rar ke dalam kotak pencarian dan tekan Memasukkan .

Kemudian tunggu proses pencarian untuk menemukan semua file rar di C-drive Anda.

EDIT:

* adalah wildcard, jadi Anda juga dapat melakukan hal-hal tingkat lanjut dengannya. Misalkan Anda mengetahui file Anda bernama <something>final.<extension>, Anda dapat mencari *final.* untuk menemukan semua file, yang namanya (tidak termasuk ekstensi) diakhiri dengan final atau Anda dapat mengetik kind: atau size: ke dalam bidang pencarian untuk mencari berdasarkan jenis file atau ukuran file.

Bagikan

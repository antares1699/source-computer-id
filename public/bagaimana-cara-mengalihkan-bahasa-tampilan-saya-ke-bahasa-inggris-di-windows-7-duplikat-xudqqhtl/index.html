Bagaimana cara mengalihkan bahasa tampilan saya ke bahasa Inggris di Windows 7? [duplikat]

Saya tinggal di Israel dan saya membeli Windows 7 Professional dalam bahasa Ibrani. Saya ingin mengganti bahasa ke bahasa Inggris.

Di Panel Kontrol -> Wilayah dan Bahasa, saya tidak memiliki opsi untuk mengalihkan bahasa tampilan ke bahasa Inggris karena hanya bahasa Ibrani yang diinstal. Di Pembaruan Windows, tidak ada pembaruan bahasa. Saya sudah menunggu tiga hari, dan masih belum ada update bahasa.

Ada Ide?

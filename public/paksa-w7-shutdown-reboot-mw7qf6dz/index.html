Paksa W7 Shutdown / Reboot

Ketika saya pergi untuk me-reboot laptop saya (yang sering saya akses dari jarak jauh), sering kali laptop akan hang dan sebuah proses akan sepenuhnya mencegahnya untuk mematikan atau me-reboot. Apakah ada peretasan registri atau apa pun yang dapat saya lakukan pada sistem Windows 7 saya yang akan memaksa program tersebut untuk menutup?

Anda dapat membuat skrip perintah seperti ini ..

@echo off
shutdown -f -s -t 0

Dan simpan script ini di desktop dengan nama shutdown.cmd. Dan ini akan segera mematikan PC Anda.

Dan jika Anda menggunakannya dari jarak jauh maka Anda dapat mencoba perintahnya

Shutdown -f -s -m \\ComputerName

Atau Anda juga dapat mengatur waktu untuk mematikan PC jarak jauh Anda ...

Shutdown -f -s -m -t 0 \\ComputerName

Ganti -s dengan -r untuk menghidupkan ulang PC.

Gantikan 0 dengan batas waktu Anda. Untuk informasi lebih lanjut lihat ini artikel . Anda juga lihat buat jalan pintas untuk mematikan .

Pengaturan registri untuk tugas akhir otomatis saat shutdown: Ketik regedit dalam perintah run dan arahkan ke jalur berikut

HKEY_CURRENT_USER > Control Panel > Desktop > create string key called AutoEndTasks > mengatur nilai ke 1

PERINGATAN: ini akan selalu mematikan Windows, meskipun aplikasi terbuka dengan perubahan yang belum disimpan.

Apa yang berhasil bagi saya untuk menghindari pop-up dengan tombol Force shut down atau Cancel:

  1. Setel kunci registri:
    • HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows\System tambahkan kunci: AllowBlockingAppsAtShutdown dengan nilai dword: 1.
    • HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control tambahkan kunci: WaitToKillServiceTimeout dengan nilai string: 1.
    • HKEY_USERS\.DEFAULT\Control Panel\Desktop tambahkan kunci: AutoEndTasks dengan nilai string: 1.
  2. Mulai ulang.

Akan tetapi, mungkin berbahaya jika tidak memberikan cukup waktu agar layanan atau aplikasi ditutup dengan baik.

Sini berikut ini juga diusulkan: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management, nama kunci: ClearPageFileAtShutdown, nilai dword: 1. Windows 7 saya tidak perlu dimatikan tanpa sembulan dan lebih jauh lagi, pengaturan kunci tersebut dapat mengakibatkan pematian yang sangat lama.

Jika tidak berhasil, coba juga: HKEY_USERS\.DEFAULT\Control Panel\Desktop dengan nama kunci WaitToKillAppTimeout dan nilai string: 1.

Bagaimana cara menghapus file konfigurasi untuk program yang tidak diinstal?

Saya pada dasarnya mencari apt-get purge untuk program yang sudah dihapus instalasinya.

(Saya menjalankan pemerasan Debian dan menggunakan aptitude untuk manajemen paket.)

Urutan berikut akan mencantumkan paket yang Anda deinstalled,

dpkg --get-selections | grep deinstall

Anda dapat mengubahnya ke purge dengan pengganti sed.
Itukah yang kamu cari?

Anda dapat menjalankan daftar purge itu melalui dpkg --set-selections dan
jalankan dpkg --purge --pending pada paket yang ditandai ...

Atau, sesuatu seperti,

for pkg in $(dpkg --get-selections | grep deinstall | awk '{print $1}'); 
do 
  dpkg -L $pkg; 
done

akan mendaftar file-file yang tergantung dari paket deinstall ini.

Secara tepat Anda hanya memfilter paket yang di-deinstall tetapi masih memiliki file konfigurasi yang tersisa (yang berada dalam status 'c').

  1. tekan L

  2. filter untuk ~c

  3. bersihkan semua paket yang terdaftar dengan menekan _

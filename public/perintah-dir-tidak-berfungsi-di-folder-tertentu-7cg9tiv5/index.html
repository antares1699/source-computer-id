Anda berada di "dummy" Special-folder My Documents Windows 7 (yang kosong).

Coba lakukan dir ..\Documents dan Anda akan melihat dokumen Anda.
(atau cd ..\Documents diikuti oleh dir).

Windows 7 mengubah My Documents menjadi Documents. Dalam explorer masih bernama My Documents tetapi itu hanya 'nama tampilan'. Itu tergantung bahasa. Dalam versi Belanda itu adalah Mijn Documenten tetapi (pada harddisk) masih menunjuk ke Documents.

Untuk program lama yang masih mengharapkan My Documents direktori ini dibuat dan di bawah Windows dialihkan ke Documents. Namun Anda berada di prompt perintah yang tidak tidak lakukan pengalihan ini.

Saya tidak tahu bagaimana Anda masuk ke direktori ini di prompt tetapi Anda seharusnya melakukan cd Documents alih-alih cd My Documents.

Kamu bisa membaca sini direktori lain apa yang diubah. Lihat juga sini untuk informasi tentang "Folder khusus".

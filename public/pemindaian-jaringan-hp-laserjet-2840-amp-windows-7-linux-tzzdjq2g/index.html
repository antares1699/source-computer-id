Pemindaian Jaringan: HP Laserjet 2840 & Windows 7 (+ Linux?)

Saya memiliki printer multifungsi HP Laserjet 2840. Ini adalah perangkat yang luar biasa, tetapi pada Windows 7 tidak dapat memindai jaringan. HP tampaknya tidak mendukungnya dalam hal apa pun yang lebih besar dari Vista, dan mereka tidak datang dengan solusi.

Saya ingin tahu apakah ada orang lain yang berhasil mengatasi masalah ini.

Satu kemungkinan bagi saya adalah saya memiliki server Gentoo yang selalu aktif yang dikonfigurasi untuk melihat printer sebagai faks dan printer menggunakan hplip. Saya telah membagikan ini melalui samba dan saya dapat mencetak dengan baik melalui samba. Apakah ada cara serupa untuk melakukan pemindaian jaringan? Adakah yang tahu dari mana saya mulai mencari ini?

Sepertinya banyak orang mengalami masalah yang sama dengan Anda.

Sepertinya ada tidak ada solusi resmi di luar sana. Ini beberapa solusi itu mungkin berhasil.

  1. Menggunakan campuran driver Vista dan Windows 7 ( tautan dengan tutorial langkah demi langkah)

    Setelah beberapa jam kami menemukan solusi untuk menginstal driver dan membiarkan Toolbox bekerja sepenuhnya sejauh ini (pemindaian push / pull, fax) menggunakan campuran driver Vista dan Windows 7.

  2. Turunkan versi ke Win7 32-bit ( tautan , lihat balasan 2009/11/9)

    Saya dapat menginstal perangkat lunak versi terbaru ke windows 7 32bit. Pemindai dan semua fungsi bekerja dengan sempurna. Untuk 64bit, Anda kurang beruntung. Saya ragu hp bisa bekerja di atasnya. Saya akhirnya menurunkan versi ke 32bit sehingga saya bisa mendapatkan semua driver untuk bekerja dengan laptop saya.

Saya berhasil membuat ini berhasil dengan melakukan hal berikut:

  1. Klik kanan HP Solution Center di menu mulai dan jalankan sebagai administrator
  2. Klik Pengaturan di bagian bawah layar
  3. Di bawah Scan Settings - Klik pada "Scan to ... Setup"
  4. Saat kotak dialog muncul, pilih opsi yang Anda inginkan dan klik "Perbarui Perangkat"

Setelah selesai, komputer Anda sekarang akan terdaftar dalam daftar Pindai Ke di printer.

Semoga membantu.

XP adalah versi Windows terakhir yang pemindaian jaringannya dilaporkan berfungsi dengan andal untuk HP Laserjet 2840.

Untuk versi Windows yang lebih baru dari XP, banyak utas yang ditulis tentang pemindaian jaringan yang tidak berfungsi, tetapi hanya sedikit jika ada solusi yang ditemukan yang benar-benar berfungsi.

Alih-alih berkelahi dengan pemindaian jaringan, seharusnya menjadi masalah sederhana untuk menghubungkan printer ke Windows 7 melalui USB, kemudian membagikannya sebagai printer jaringan melalui Windows.

Solusi ini menghindari perkelahian dengan pengandar pencetak lama yang tidak disesuaikan dengan Windows 7.

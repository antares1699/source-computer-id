Hapus Otomatis Download (dan / atau Sembunyikan Otomatis Bilah Download) di Chrome?

Adakah cara agar Chrome secara otomatis menghapus file dari daftar unduhan setelah selesai diunduh?

Jika tidak, apakah setidaknya ada cara untuk menyembunyikan bilah unduhan secara otomatis setelah beberapa detik? Sangat menjengkelkan saat menghabiskan tampilan layar, dan jika saya tidak dapat menghapus unduhan saya dari daftar utama, setidaknya saya ingin menyembunyikan bilah unduhan agar saya dapat melanjutkan pekerjaan saya.

Saya tahu ini adalah pertanyaan lama, tetapi untuk semua orang yang, seperti saya, tiba di sini dalam pencarian untuk menyingkirkan bilah unduhan, ada cara resmi tidak resmi untuk melakukannya:

  1. Ketik URL chrome://flags/ di mahakotak.
  2. Gulir ke bawah ke "UI Unduhan Baru" dan aktifkan lalu mulai ulang Chrome.

Mengatur bendera ini sepenuhnya menghilangkan bilah unduhan yang sebenarnya bukan pertanyaannya tetapi setidaknya itu memberi Anda pilihan apakah akan mempertahankan bilah atau tidak.

Pembaruan: Mulai hari ini, 10 September 2014, tampaknya bendera ini tidak lagi ada di versi Chrome saat ini.

Diuji pada OSX 10.9.4 w / Chrome Versi 37.0.2062.120, dan Windows 7 w / Service Pack 1 dan Chrome 37.0.2062.120 m

Untuk apa nilainya, Anda dapat menutupnya dengan keyboard dengan melakukan:

  • Windows: CTRL - J , CTRL - W

  • Mac: CMD - BERGESER - J , CMD - W

Perintah pertama memunculkan unduhan Anda di tab baru, yang kedua menutup unduhan dengan cepat, mengirim Anda kembali ke tab sebelumnya. Setelah itu, bilah 'Unduhan' akan ditutup.

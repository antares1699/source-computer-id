Jalankan program 16 bit pada mesin 64 bit dengan sistem operasi 32 bit?

Saya memiliki laptop 64 bit (yaitu sudah diinstal sebelumnya dengan Win 7 Versi 64-bit). Anda tahu, saya bisa saja membeli sistem 32-bit tetapi saya ingin meningkatkan RAM menjadi 8 GB dalam satu tahun atau lebih. Saya sudah tahu bahwa saya tidak dapat menjalankan program 16 bit di lingkungan 64 bit, namun pertanyaan saya adalah ini:

Apakah saya dapat menjalankan program 16 bit pada mesin yang sama ini jika saya menginstal Win 7 versi 32-bit?

Ya, batasan kode 16-bit adalah ketika prosesor berjalan dalam "mode panjang". OS 32-bit tidak akan mengalihkan CPU ke mode lama.

Anda juga dapat menggunakan emulasi (seperti DosBox) untuk menjalankan program 16-bit.

Ya, Windows 7 32-bit dapat menjalankan program 16-bit. 64-bit Windows 7 tidak bisa.

"Windows versi 64-bit tidak mendukung komponen 16-bit, proses 16-bit, atau aplikasi 16-bit" http://support.microsoft.com/kb/896458

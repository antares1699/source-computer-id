Zotero adalah plugin firefox yang mengelola kutipan dan referensi seperti catatan akhir, kecuali bahwa itu juga dapat menyimpan snapshot halaman web dan file pdf yang sebenarnya.

Ini juga memiliki fungsi ekspor bibliografi yang dapat menghasilkan file bibtex, antarmuka dengan catatan akhir, dan kantor terbuka.

Halaman depan situs web mereka memiliki video demo singkat yang menunjukkan bagaimana situs web itu sadar dan dapat menarik informasi bibliografi langsung dari amazon, googlebooks, jstor, dan situs web lainnya. Alat penelitian yang cukup bagus.

Saya menggunakannya dengan bibtex / lateks sendiri.

Periksa eKitaab untuk manajemen eBook:

eKitaab adalah Manajer Katalog eBook perangkat lunak untuk mengelola koleksi besar dari ebooks. eKitaab memungkinkan Anda membaca buku di pembaca asli mereka di desktop, atau di perangkat seluler ini bukan pembaca ebook.

eKitaab menampilkan daftar semua buku Anda memiliki, dengan judul mereka, penulis, deskripsi dan foto buku selimut. Ini juga memungkinkan Anda dengan mudah mengkategorikan buku dengan membuat Daftar Bacaan, dan menambahkan Tag. eKitaab mengunduh semua informasi dari Internet, berdasarkan ISBN dari buku, atau dengan mencari oleh penulis atau judul.

eKitaab dirancang untuk menjadi sangat luar biasa aman untuk eBook ... eBook diri mereka tidak pernah berubah. eKitaab menyimpan informasi dasar tentang buku dengan mengubah nama file menjadi sertakan judul, pengarang dan ISBN. Informasi lainnya diunduh dari Internet dan tidak disimpan secara lokal (tetapi cache diterapkan, untuk mengurangi unduhan berulang).

eKitaab adalah perangkat lunak gratis dan sumber terbuka, berjalan di Linux, Windows, dan Mac.

Pilihan lainnya adalah Alexandria :

Alexandria adalah aplikasi GNOME untuk membantu Anda mengelola koleksi buku Anda.

Alexandria juga gratis dan open source.

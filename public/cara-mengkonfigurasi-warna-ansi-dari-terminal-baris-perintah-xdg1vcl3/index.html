Cara mengkonfigurasi warna ANSI dari terminal baris perintah

Saya secara khusus menggunakan bash di Mac OS X, tetapi menurut saya jawabannya adalah Unix-generik.

Saya sudah menggunakan warna ANSI, tetapi saya ingin menetapkan ulang warna yang berbeda ke folder, file tersembunyi, dll.

Memperbarui: Ini tidak berguna seperti yang saya harapkan, karena Anda tidak dapat menetapkan warna tertentu ke file tersembunyi, dan juga direktori tidak dapat selalu diwarnai sama, karena warna bit izin lengket lebih diutamakan daripada direktori (hijau zaitun di tangkapan layar)

Ubah profil ~ /. Anda dengan berikut ini. Anda juga bisa meletakkan ini di ~ / .bashrc atau ~ / .bash_profile Anda tergantung pada apa yang sudah Anda miliki dan gunakan. CATATAN: ini menunjukkan pengaturan warna default. Ubah e pertama menjadi B jika Anda ingin melihat direktori merah.

    LSCOLORS=exfxcxdxbxegedabagacad

    export LSCOLORS

    export CLICOLOR=1

Simpan ~ / .profile Anda dan kemudian lakukan:

    . .profile

Dari baris perintah.

Berikut adalah pengaturan warna dan urutan atribut yang berbeda dari halaman manual ls:

LSCOLORS        The value of this variable describes what color to use for which attribute when colors are enabled with CLICOLOR.  This string is a
                 concatenation of pairs of the format fb, where f is the foreground color and b is the background color.

                 The color designators are as follows:

                       a     black
                       b     red
                       c     green
                       d     brown
                       e     blue
                       f     magenta
                       g     cyan
                       h     light grey
                       A     bold black, usually shows up as dark grey
                       B     bold red
                       C     bold green
                       D     bold brown, usually shows up as yellow
                       E     bold blue
                       F     bold magenta
                       G     bold cyan
                       H     bold light grey; looks like bright white
                       x     default foreground or background

                 Note that the above are standard ANSI colors.  The actual display may differ depending on the color capabilities of the terminal in
                 use.

                 The order of the attributes are as follows:

                       1.   directory
                       2.   symbolic link
                       3.   socket
                       4.   pipe
                       5.   executable
                       6.   block special
                       7.   character special
                       8.   executable with setuid bit set
                       9.   executable with setgid bit set
                       10.  directory writable to others, with sticky bit
                       11.  directory writable to others, without sticky bit

                 The default is "exfxcxdxbxegedabagacad", i.e. blue foreground and default background for regular directories, black foreground and red
                 background for setuid executables, etc.

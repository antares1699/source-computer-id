Bagaimana cara melewati interaksi pengguna dalam skrip?

Saya mencoba untuk mengotomatiskan penginstalan program melalui skrip shell. Ada beberapa langkah yang memerlukan interaksi pengguna dari baris perintah (konfirmasi, ya / tidak, dll.). Apa cara terbaik untuk melewatinya? Saya tahu bahwa satu kemungkinan adalah menggunakan skrip yang diharapkan. Hanya ingin tahu apakah ada cara yang lebih baik / lebih bersih untuk melakukan ini.

( echo yes ; echo 1024; echo yes ) | install.sh

Dari utas lain:

Jika perintah Anda tidak peduli seberapa cepat Anda memberikan masukan, dan Anda tidak benar-benar perlu berinteraksi dengannya, Anda dapat menggunakan heredoc.

Contoh:

#!/bin/bash
command_you_want_to_execute <<EOD
interaction_1
intereaction_2
EOD

Jika Anda membutuhkan percabangan berdasarkan output dari program, atau jika program Anda sama sekali sensitif terhadap waktu perintah Anda, maka Harapkan adalah yang Anda inginkan.

Bagaimana cara menyimpan semua tab di jendela FireFox? [duplikat]

Kadang-kadang ketika saya meneliti tentang masalah perangkat lunak (yaitu bug yang kemungkinan besar saya buat), saya banyak mencari di Google sehingga banyak tab yang terbuka di FireFox. Saya tidak menggunakan IE :)

Tetapi FireFox hanya mengizinkan saya untuk menyimpan tab terbuka untuk file sesi terakhir . Apakah ada cara yang bisa saya lakukan perintah FireFox untuk dipatuhi => untuk jendela utama FireFox yang berisi semua tab terbuka ini, simpan sesi sebagai nama "meneliti bug # 232 - file tidak disimpan" sehingga saya dapat kembali lagi nanti dan mengklik tombol agar semua tab itu terbuka lagi.

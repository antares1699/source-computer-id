Simbol euro pada keyboard menolak untuk bekerja

Saya diminta untuk melihat masalah di mana simbol Euro (€) akan menolak untuk dimasukkan saat menekan AltGr + E (kombinasi tombol default untuk simbol pada keyboard Jerman).

Baik di Word dan Outlook, menekan kombinasi tombol tidak berpengaruh. Klien meyakinkan saya bahwa masalahnya adalah global (mempengaruhi semua aplikasi). Setelah pemeriksaan pertama di area notifikasi, saya tidak dapat menemukan aplikasi apa pun yang saya curigai menyebabkan ini.

Itu E kunci berfungsi dengan baik jika tidak. Itu AltGr kunci tampaknya berfungsi sebagaimana mestinya juga. Mendesak AltGr + M akan mencetak µ misalnya. Jadi kunci yang macet tidak mungkin terjadi.

Tata letak keyboard aktif disetel ke bahasa Jerman, jadi itu juga seharusnya tidak menjadi masalah.

Klien memberi tahu saya bahwa itu berfungsi dengan baik di masa lalu dan bahwa dia sudah mengganti keyboard ke tidak berpengaruh (dan saya percaya dia).

Saya hanya dapat menggunakan sistem secara singkat dan tidak dapat memecahkan masalah lebih lanjut (untuk hari ini).

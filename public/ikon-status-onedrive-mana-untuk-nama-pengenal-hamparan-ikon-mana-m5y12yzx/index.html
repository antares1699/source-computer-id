Ikon / status OneDrive mana untuk nama Pengenal Hamparan Ikon mana?

Mirip dengan pertanyaan ini , Saya ingin menonaktifkan beberapa lapisan ikon shell OneDrive secara selektif. Registri (HKLM \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ Explorer \ ShellIconOverlayIdentifiers), namun, hanya mencantumkan: OneDrive1, OneDrive2, OneDrive3, OneDrive4, OneDrive5. Apa artinya masing-masing?

Setelah beberapa kali mengaduk-aduk, berikut hasilnya:

  1. Tidak menyinkronkan (x merah)
  2. Disinkronkan - Dibagikan
  3. Sinkronisasi - Berbagi
  4. Disinkronkan
  5. Menyinkronkan

Bagi yang penasaran, registri memiliki CLSID dari masing-masing kunci di atas. Anda kemudian dapat memuat objek COM itu dan menanyakan di mana letak ikon (file plus nomor gambar). Kemudian Anda dapat melihat di file untuk menentukan ikon (menggunakan misalnya, IconsExtract Nirsoft). Berikut beberapa contoh kode dari proyek VS2013:

    #include "stdafx.h"
    #include <Shlobj.h>
    #include <iostream> 
    #include <tchar.h>

    int _tmain(int argc, _TCHAR* argv[]){
      LPCOLESTR str_clsid = L"{FB314EE0-A251-47B7-93E1-CDD82E34AF8B}";
      HRESULT hr;
      CLSID clsid;
      IShellIconOverlayIdentifier* pISIOI;
      DWORD dwFlags;
      int Index;
      wchar_t* wszIconFile = new wchar_t[128];

      hr = CLSIDFromString(str_clsid, (LPCLSID)&clsid);
      CoInitialize(NULL);
      hr = CoCreateInstance(clsid, NULL, CLSCTX_INPROC_SERVER, IID_IShellIconOverlayIdentifier, (void**)&pISIOI);
      hr = pISIOI->GetOverlayInfo(wszIconFile, 128, &Index, &dwFlags);
      std::wcout << L"Icon Path:\n    " << wszIconFile << L" " << Index << std::endl;
      pISIOI->Release();
      CoUninitialize();
      return 0;
    }

Berikut adalah pembaruan (deskripsi diambil dari CLSID di Editor Registri):

  • OneDrive1 - Kelas ErrorOverlayHandler
  • OneDrive2 - Kelas SharedOverlayHandler
  • OneDrive3 - Kelas UpToDateCloudOverlayHandler
  • OneDrive4 - Kelas UpToDatePinnedOverlayHandler
  • OneDrive5 - Kelas SyncingOverlayHandler
  • OneDrive6 - ReadOnlyOverlayHandler Class
  • OneDrive7 - Kelas UpToDateUnpinnedOverlayHandler

Selain itu, ada 3 CLSID yang tampaknya sudah lama:

  • {1BF42E4C-4AF4-4CFD-A1A0-CF2960B8F63E} - UpToDateOverlayHandler2 Kelas
  • {7AFDFDDB-F914-11E4-8377-6C3BE50D980C} - Kelas ErrorOverlayHandler2
  • {82CA8DE3-01AD-4CEA-9D75-BE4C51810A9E} - SyncingOverlayHandler2 Kelas

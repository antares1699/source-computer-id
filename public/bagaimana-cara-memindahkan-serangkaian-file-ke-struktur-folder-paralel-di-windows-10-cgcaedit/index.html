Bagaimana cara memindahkan serangkaian file ke struktur folder paralel di windows 10

Saya memiliki folder yang memiliki folder gambar dan video yang ingin saya pisahkan menjadi folder terpisah berdasarkan jenis file. Saya ingin ini menjadi skrip sehingga saya dapat menjalankannya kembali ketika lebih banyak file yang diimpor

Struktur file saat ini

Saat ini gambar dan video berada dalam folder yang digabungkan berdasarkan tanggal

Images/
    2018-08/
          image1.png
          Video1.mp4
    2018-09/
          image1.png
          video1.mp4

Struktur Target

Setelah menjalankan skrip, saya ingin struktur folder dan nama file dipertahankan dengan tujuan agar gambar tetap berada dalam struktur gambar dan video dipindahkan ke struktur terpisah.

Images/
    2018-08/
          image1.png
    2018-09/
          image1.png
Videos/
    2018-08/
          Video1.mp4
    2018-09/
          video1.mp4

Apa yang sudah saya coba sejauh ini

Saya telah melihat ke jendela pindah perintah tetapi tampaknya hanya berlaku untuk satu folder.

Juga saya melihat kemudian solusi untuk Bagaimana cara menyalin file dan mempertahankan strukturnya? . Ini hampir menyelesaikan masalah saya kecuali saya tidak memiliki daftar file yang telah ditentukan sebelumnya hanya ekstensi yang ditentukan sebelumnya dan folder awal.

Beberapa catatan

  • Folder yang lebih tinggi mis. Gambar dan Video akan ditentukan sebelumnya dan bersifat statis
  • Sub-folder di bawah gambar perlu direplikasi secara otomatis
  • Nama file mungkin duplikat di antara perpindahan

Lihat robocopy yang terintegrasi dengan Windows 7 dan yang lebih baru.

Itu dapat memindahkan file dan folder dengan topeng tertentu.

Dalam kasus Anda, itu harus bekerja menggunakan sesuatu seperti ini: robocopy \images \videos *.mp4 /copyall /s /mov

Saya akan merekomendasikan pengujian ini pada beberapa folder dan file sebelum menjalankannya pada struktur folder besar yang dapat benar-benar campur aduk, sampai Anda mendapatkan perintah yang tepat.

https://docs.microsoft.com/en-us/windows-server/administration/windows-commands/robocopy

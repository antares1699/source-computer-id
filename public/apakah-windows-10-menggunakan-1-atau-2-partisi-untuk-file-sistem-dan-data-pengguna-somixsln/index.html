Secara default, tidak. Itu tidak. Semua data pengguna Anda disimpan di bawah profil pengguna Anda di C:\Users

Tapi memang begitu sepele untuk memindahkan My Documents dan folder lainnya ke drive lain, dan sebagian besar program memungkinkan Anda dengan mudah untuk mengubah lokasi pemasangan.

Memindahkan direktori Users atau direktori profil di bawahnya memang rumit.

Tetapi ini sama dengan Linux, Linux dapat menggunakan satu partisi dan berakhir dengan program dan data pengguna pada drive yang sama juga, semuanya tergantung pada bagaimana Anda mengatur mesin untuk memulai dan seberapa dapat dikonfigurasi penginstal.

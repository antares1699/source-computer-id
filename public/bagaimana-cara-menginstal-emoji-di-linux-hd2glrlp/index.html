Bagaimana cara menginstal Emoji di linux?

Saya menggunakan Fedora 17. Saya ingin melihat ikon Emoji di komputer saya. Saat ini saya melihat ubin persegi di sebagian besar simbol di halaman Wikipedia ini . Tidak yakin apakah ini ada hubungannya dengan Peta Karakter dan cara mendapatkan font yang diperlukan.

Instal gdouros-symbola-fonts paket : pkcon install gdouros-symbola-fonts. Kemudian restart browser Anda dan halaman Wikipedia itu akan terlihat jauh lebih menarik.

Untuk siapa pun yang tidak menggunakan Fedora, kunjungi Font Unicode untuk Skrip Kuno halaman untuk mengunduh ini dan jenis huruf lainnya langsung dari penciptanya, George Douros.

Bilah Tugas Menghilang di Windows 8.1

Tampaknya bilah tugas menghilang secara permanen setiap kali saya menggunakan aplikasi Metro tertentu di Windows 8.1. Artinya, ketika saya keluar dari aplikasi, bilah tugas hilang dari layar tempat aplikasi Metro muncul (namun, selalu tetap di monitor saya yang lain). Satu-satunya cara saya dapat menyelesaikan masalah ini tampaknya dengan memulai ulang sepenuhnya. Tak perlu dikatakan, ini benar-benar menyakitkan.

Apakah ada cara untuk mencegah hal ini (selain tidak menggunakan aplikasi Windows Store)? Jika tidak, apakah ada cara yang lebih baik untuk memperbaiki masalah ini setelah bilah tugas menghilang (lebih baik daripada memulai ulang)?

Mulai ulang penjelajah alih-alih memulai ulang komputer dan lihat apakah itu berfungsi.

  1. Ctrl + Bergeser + ESC untuk membuka Task Manager.
  2. Klik kanan Windows Explorer, restart.

Ini akan menutup jendela penjelajah aktif Anda.

Saya menemukan solusi ini untuk Taskbar dan ikon yang hilang:

  1. Buka Pengelola Tugas
  2. Klik kanan pada Explorer, Properties
  3. Buka tab Keamanan, klik Lanjutan
  4. Ubah kepemilikan menjadi Semua Orang. Centang semua kotak izin.

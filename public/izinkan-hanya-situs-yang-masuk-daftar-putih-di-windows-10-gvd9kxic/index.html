Izinkan hanya situs yang masuk daftar putih di Windows 10

Di Windows 7 dan 8 Microsoft Family Safety menyertakan fitur di mana Anda dapat memblokir akses ke semua situs web kecuali yang secara eksplisit tercantum dalam daftar putih di situs web Family Safety. Dengan Windows 10, fitur ini telah dihapus, meskipun Microsoft mengatakan bahwa mereka berencana untuk mengembalikan fitur itu di masa mendatang (lihat https://account.microsoft.com/family/faq/ ).

Sementara itu, adakah cara untuk mencapai fungsi serupa dengan program atau layanan lain? Layanan ini harus berfungsi terlepas dari browser mana yang digunakan.

Jika tidak masalah untuk memblokir akses semua orang yang menggunakan komputer itu, Anda dapat mengedit file HOSTS ke null-rute situs web yang tidak disetujui, mengarahkannya ke alamat IP yang tidak ada (atau, jika Anda ingin mengganggu, alihkan semuanya ke sesuatu seperti nyan.cat ).

Sebagai admin, tekan Win + R dan ketik

%systemroot%\system32\drivers\etc\

di kotak dialog, dan tekan ENTER. Pada daftar file di Explorer, klik kanan pada "hosts", klik "Open With" dari menu konteks, dan kemudian buka dengan Notepad. Setelah Anda masuk, Anda dapat mengeditnya seperti ini:

#allowed sites
173.194.33.128 google.com
198.252.206.16 superuser.com
#blah blah, more sites here

#blocked sites
<paste the contents of http://pastebin.com/63hfPx8M here>

Ini null-merutekan semua TLD (.com, .net, dll.) Dan hanya mengizinkan melalui situs yang masuk daftar putih.

Untuk memasukkan lebih banyak situs ke daftar putih, tambahkan baris untuk setiap situs web ke file HOSTS dengan memasukkan alamat IP situs web (lihat di bawah), lalu spasi, lalu nama situs web. Jangan letakkan "http: //" di nama, atau apa pun setelah .com / .net / TLD apa pun yang dimiliki situs web tersebut.

Untuk mencari IP sebuah website, pertama lakukan Win + R, lalu "cmd", lalu Enter. Saat kotak hitam muncul, ketik ping <website> dan tekan Enter. (Sekali lagi, jangan masukkan "http: //" atau yang lainnya.) Beberapa hal akan muncul, tetapi yang penting adalah IP, yang ada dalam tanda kurung:

C:\Users\Me>ping google.com

Pinging google.com [173.194.33.128] with 32 bytes of data:

Jadi IP Google adalah 173.194.33.128.

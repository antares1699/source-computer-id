Aplikasi Windows 7 disisipkan di tengah bilah tugas, bukannya ditambahkan ke bagian akhir

Ketika saya membuka instance dari aplikasi yang sama di Windows 7, instance yang baru dibuka dimasukkan ke bilah tugas segera setelah instance yang ada, bukan di akhir daftar seperti Windows XP. Bisakah ini diubah?

Sunting: Harap dipahami bahwa pertanyaan ini bukan tentang menggabungkan. Saya telah menggabungkan cacat untuk waktu yang lama. Pertanyaan ini adalah tentang di mana di bilah tugas aplikasi baru ditempatkan, bersebelahan dengan aplikasi lain dari jenis yang sama vs di akhir daftar (seperti XP).

Jawaban Asli

Klik kanan Taskbar> Properties> tombol Taskbar dropdown> Never Combine (atau gabungkan saat penuh)> OK (klik kotak centang ikon kecil dan taskbar akan menyusut)

atau jika Anda benar-benar ingin seperti XP

Pilih tema klasik Windows. Mulai> Panel Kontrol> Tampilan dan Personalisasi> Tema> Windows Klasik.

Namun, saya sarankan Anda mencoba cara Windows 7 yang baru, setelah satu atau dua minggu itu menjadi kebiasaan dan jauh lebih berguna daripada cara klasik, bagaimanapun juga IMO.

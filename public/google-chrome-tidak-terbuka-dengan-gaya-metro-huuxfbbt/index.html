Google Chrome tidak terbuka dengan gaya metro

Saya telah membaca bahwa build dev dari Google Chrome dapat bekerja dalam gaya metro di win 8 jika itu adalah build dev, merupakan browser default, dan disematkan untuk memulai. Saya melakukan semua itu, dan itu dimulai dalam mode desktop, bahkan ketika saya memulainya dari menu mulai. Saya menggunakan versi 24.0.1312.2 dev-m atau Google Chrome dan Windows 8 Pro. Apa yang dapat saya lakukan agar Chrome terbuka sebagai Aplikasi Metro?

Jika Chrome disetel ke default dan masih tidak diluncurkan dalam mode Metro, meskipun membukanya dari Layar Mulai, Anda harus menginstruksikan Chrome untuk membuka dalam Mode Windows 8.

Untuk melakukan ini, Klik pada menu Chrome dan pilih "Luncurkan Ulang dalam Mode Windows 8"

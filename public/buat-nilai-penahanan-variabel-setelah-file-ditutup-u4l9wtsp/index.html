Buat nilai penahanan variabel setelah file ditutup

Sama seperti judulnya, saya mencari cara untuk membuat variabel menahan nilainya bahkan setelah file Excel 2010 yang berisi kode VBA ditutup, dan mengambilnya dengan nilai terakhirnya, saat berikutnya saya membuka file tersebut.

Edit: Saya menyadari fakta bahwa saya dapat menyimpan data "secara eksternal" di luar kode VBA, baik dalam sel atau file eksternal atau dengan cara lain. Saya mencari cara yang HANYA melibatkan kode VBA dari file Excel tersebut. Jawaban yang menunjukkan bahwa ini tidak mungkin, juga merupakan jawaban yang dapat diterima, tentu saja.

Anda bisa melakukan sesuatu yang mirip dengan:

ThisWorkbook.CustomDocumentProperties.Add _
Name:="Version Number", _
LinkToContent:=False, _
Type:=msoPropertyTypeNumber, _
Value:=7

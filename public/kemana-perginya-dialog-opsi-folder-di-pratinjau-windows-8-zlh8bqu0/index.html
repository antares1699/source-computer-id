Kemana perginya dialog opsi folder di pratinjau Windows 8?

Apakah ada yang menemukan cara untuk memanggil dialog opsi folder di pratinjau Windows 8. Saya ingin memulai program saya dengan sekali klik dan saya selalu ingin melihat ekstensi file.

Edit:

Untuk file dan ekstensi tersembunyi ini tautan membantu. Tapi saya juga ingin menghilangkan klik dua kali.

Ini mungkin. Buka jendela penjelajah apa pun lalu cukup klik File pada Pita, lalu klik Change folder and search options.

Dari sini, Anda akan mendapatkan dialog normal.

Untuk membuka satu klik pilih Single-click to open an item (point to select) dari bagian Click items as follows pada tab General.

Untuk menampilkan ekstensi file, pilih Hide extensions for known file types dari daftar besar di bawah Advanced settings pada tab View.

Di toko Windows 8 mereka menawarkan file Aplikasi TuneIn . Bagaimana cara menghubungkan aplikasi ini ke akun TuneIn saya sehingga saya tidak perlu menambahkan stasiun radio favorit saya secara manual?

Untuk masuk ke akun yang sudah ada (atau buat yang baru):

  1. tekan menang + c atau gunakan hotspot di sisi kanan layar Anda untuk menampilkan pesona batang.
  2. Klik pengaturan.
  3. Klik Sign In / Sign Up

Ingat: Pengaturan di bilah pesona bersifat kontekstual (seperti penelusuran). Ini tempat pertama untuk mencari jenis pertanyaan ini.

Saya tertarik bagaimana Anda dapat mengatur enkripsi disk untuk Ubuntu, dengan sesedikit mungkin kerumitan.

Saya baru saja melakukan ini minggu lalu. Ini sangat mudah jika Anda tidak melakukan dual boot atau apapun. Saya pergi dengan semua pengaturan default, dan hibernate / suspend saya berfungsi dengan baik dengan LUKS saya

Jika Anda tidak keberatan menginstal ulang OS, maka itu sangat mudah dari CD instalasi alternatif. Anda harus mengunduh CD instalasi alternatif untuk arsitektur apa pun (i386 atau amd64, misalnya 32 bit atau 64 bit) yang ingin Anda instal.

MULAI SEBAGAI

Jika Anda belum melakukannya, pastikan Anda melakukannya karena Anda akan menghapus hard drive Anda sepenuhnya . Tentunya Anda ingin mencadangkan folder rumah Anda dan semua data penting lainnya! Dan jika Anda menginginkan daftar praktis dari program yang telah Anda instal, Anda dapat menjalankan perintah

dpkg --get-selections

Jika Anda ingin mengeluarkannya ke file di folder rumah Anda, Anda dapat menjalankannya

dpkg --get-selections > ~/filename

Dimana nama file adalah nama file teks Anda. Saya biasanya menyebutnya "paket terinstal 20090901" atau semacamnya. Kemudian pastikan Anda memasukkannya ke dalam cadangan Anda. Saat Anda memulihkan, akan mudah untuk menginstal ulang semua paket yang diinstal sebelumnya, terutama jika Anda tidak mengubah versi. Gunakan perintah ini:

 dpkg --set-selections < ~/filename

AKHIR ASIDE

Bagaimanapun, berikut adalah metode mudah untuk mendapatkan enkripsi disk penuh dengan LUKS menggunakan CD instalasi alternatif:

  1. Unduh ISO penginstalan alternatif untuk arsitektur Anda.
  2. Bakar CD alternatif dan boot dari itu.
  3. Pilih item menu pertama, "instal Ubuntu" dan Anda akan masuk ke penginstal berbasis teks. Ini tidak terlalu berbeda dengan penginstal grafis - Anda akan diminta untuk memberikan data zona waktu, pengaturan bahasa / keyboard, dll.
  4. Ketika sampai pada bagian di mana Anda perlu mengatur partisi Anda, Anda dapat memilih opsi "gunakan seluruh disk dengan LVM terenkripsi" dan kemudian gunakan frasa sandi.
  5. Lanjutkan menginstal, mengatur nama pengguna, nama host, dll.
  6. Setelah Anda menginstal sistem dan memulai ulang, layar boot akan meminta frasa sandi untuk mendekripsi.
  7. Masukkan frasa sandi yang benar dan sistem Anda akan boot secara normal dengan biaya kinerja yang sangat rendah.

Saya sengaja tidak jelas dengan jawaban ini karena saya pikir sangat penting untuk memahami dengan tepat apa yang Anda tuju, mengapa, dan bagaimana Anda akan mencapai tujuan Anda jika Anda akan melakukan enkripsi disk penuh. Jika Anda memerlukan sesuatu yang berbeda (keyfile alih-alih frasa sandi, dual boot, partisi yang disesuaikan, dll.), Kita juga dapat membahasnya.

Jika Anda tidak memerlukan enkripsi seluruh disk yang lengkap dan bahkan enkripsi per pengguna akan terlalu banyak, Cryptkeeper memungkinkan Anda mengenkripsi barang pada basis per direktori. Sangat mudah untuk mengatur setidaknya di Ubuntu 10.04. Cukup instal paket Cryptkeeper (pastikan Anda telah mengaktifkan repositori Universe).

Kemudian Anda baru saja meluncurkan Aplikasi -> Alat Sistem -> Cryptkeeper dan klik ikon kunci yang baru saja muncul di bilah tugas Anda. Di sana Anda dapat membuat folder terenkripsi baru hanya dengan mengklik dengan mouse Anda. Pemasangan folder terenkripsi Anda terjadi hanya dengan mengklik lagi ikon kunci, memilih folder yang Anda inginkan dan memasukkan kata sandi Anda.

Setelah Anda menyelesaikan pekerjaan Anda, berselancar porno atau apa pun yang ingin Anda enkripsi, Anda dapat dengan mudah melepas folder terenkripsi Anda.

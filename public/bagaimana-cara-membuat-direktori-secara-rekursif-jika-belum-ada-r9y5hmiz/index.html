Bagaimana cara membuat direktori secara rekursif jika BELUM ada?

Ini sepertinya utilitas umum dan saya bertanya-tanya apakah ada sesuatu yang sudah ada. Saya mencoba menempatkan logfile timestamped di direktori setelah skrip berjalan, tetapi perlu memastikan subdirektori (dan direktori apa pun di sepanjang rantai) ada terlebih dahulu.

Misalnya, jika jalur relatif lengkap adalah ./2014/Backups/Logfiles, saya ingin memeriksa apakah 2014 tidak ada, buatlah, jika Backups tidak ada, buatlah, jika Logfiles tidak ada, buatlah ....

Sejauh ini saya punya sesuatu seperti ini (tapi itu hanya satu langkah): LOGFILE_DIRECTORY = "./ 2014 / Backup / Logfiles"

if [ ! -d "$LOGFILE_DIRECTORY" ]; then
  mkdir "$LOGFILE_DIRECTORY" >> $LOGFILE 2>&1
fi

EDIT, Menambahkan solusi

if [ ! -d "$LOGFILE_DIRECTORY" ]; then
  mkdir -p "$LOGFILE_DIRECTORY" >> $LOGFILE 2>&1
fi

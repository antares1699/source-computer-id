Bagaimana cara menambahkan pemetaan ex-mode di Vim?

Saya baru saja membuat fungsi Vim untuk mengganti linebreak Windows dengan yang Unix. Sekarang saya ingin bisa mengetik :linebreak dan menjalankannya. Inilah usaha pertama saya:

function UseUnixLineBreaks()
  :%s/^M/\r/g
endfunction

cmap linebreak :call UseUnixLineBreaks()<CR>

(Ngomong-ngomong, ^M itu adalah karakter kontrol - ketikkan dengan Ctrl+v Ctrl+m)

Ketika saya mencoba menjalankan ini dengan mengetik :linebreak, itu berjalan, tetapi berperilaku aneh: setiap huruf dari perintah muncul secara berurutan, menghapus yang sebelumnya. Misalnya, saya melihat :l, lalu :i, lalu :n, dll. Saat saya mengetik huruf terakhir, itu langsung berjalan.

Saya ingin seluruh perintah muncul, sehingga saya dapat melihat bahwa saya telah mengetiknya dengan benar, lalu tekan enter untuk menjalankannya.

Bagaimana saya bisa melakukan ini?

Menemukannya

Ternyata yang saya inginkan bukanlah pemetaan sama sekali, melainkan sebuah perintah. Jadi, baris terakhir harus diganti dengan:

command Linebreak call UseUnixLineBreaks()

Perhatikan huruf besar "L" - perintah yang ditentukan pengguna harus dimulai dengan huruf kapital. Lihat :help command.

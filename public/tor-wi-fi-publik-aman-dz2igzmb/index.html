Sayangnya, Tor tidak sepenuhnya aman. Tor mengenkripsi data, ya, tetapi Tor hanya mengenkripsi antara Anda dan relai keluar. Kecuali ada enkripsi di luar (di atas?) Tor, mis. SSL, relai keluar dapat mengintip data Anda yang telah didekripsi. Layanan proxy / VPN semacam ini hanya dapat dipercaya seperti orang-orang yang menjalankan server, dan dalam kasus Tor secara harfiah siapa pun dapat mengatur relai keluar anonim.

Jika Tor adalah satu-satunya pilihan Anda, pertanyaannya adalah: apakah Anda mempercayai operator WiFi publik (dan pengguna lain, jika tidak dienkripsi, pengguna lain dapat melihat data Anda) lebih banyak atau orang asing acak di suatu tempat di dunia?

Satu-satunya cara untuk benar-benar aman adalah jika ada beberapa bentuk enkripsi ujung ke ujung, mis. SSL, antara Anda dan tujuan akhir data Anda. Kecuali itu, menjalankan server Anda sendiri untuk terhubung (melalui VPN terenkripsi) tidak masalah, tetapi seseorang juga dapat mencegat lalu lintas antara server Anda dan tujuan akhirnya. Tetap saja, jauh lebih aman daripada Tor atau WiFi publik.

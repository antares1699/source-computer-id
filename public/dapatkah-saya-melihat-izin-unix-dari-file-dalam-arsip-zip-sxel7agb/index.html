File IIRC Zip yang dibuat pada Windows default ke 0777 untuk direktori, dan 0666 untuk file ketika diekstrak pada sistem UNIXy menggunakan alat Info-ZIP.

File zip yang dibuat pada sistem UNIXy menggunakan alat Info-ZIP akan mempertahankan izin UNIX yang sebenarnya.

Perintah zipinfo Info-ZIP akan menampilkan izin ini.

Anda dapat menemukan biner Windows alat Info-ZIP di situs web mereka ( http://www.info-zip.org/ ) atau sebagai bagian dari Cygwin ( http://www.cygwin.com/ ).

(Saya mantan pengelola Info-ZIP untuk platform BeOS dan QNX. Dahulu kala. :-))

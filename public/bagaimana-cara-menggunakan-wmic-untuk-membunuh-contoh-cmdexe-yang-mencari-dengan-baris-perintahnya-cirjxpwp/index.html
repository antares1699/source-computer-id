Bagaimana cara menggunakan wmic untuk membunuh contoh cmd.exe yang mencari dengan baris perintahnya?

Saya telah melihat bahwa dimungkinkan untuk menggunakan wmic untuk menemukan id dari proses mencari command line yang digunakan untuk memulainya.

Tapi saya punya 2 masalah.

  • Saya perlu menggunakan ini dalam file batch jadi saya tidak bisa membaca output dari wmic dan memanggil taskkill Saya harus menggunakan semacam grep
  • Saya tidak bisa menemukan cara mencari kecocokan parsial. Saya lebih suka mencari bagian dari baris perintah.

Apa yang dapat saya temukan adalah contoh ini:

wmic process where "CommandLine=start.bat", pid

Tapi yang ingin saya lakukan lebih mirip dengan:

wmic process where "CommandLine like %start.bat%", pid | taskkill

Inilah yang saya maksud dengan baris perintah: "cmd.exe" / c C: \ Mysql \ start.bat "

Apakah ada cara untuk melakukannya membunuh proses mencari baris perintah yang memulainya?

Cara menggunakan wmic untuk mematikan contoh cmd.exe yang mencari dengan perintah itu garis?

Tampaknya Anda dapat melakukan ini secara native dari batch Windows dengan format di bawah ini. Cukup masukkan nama file batch Anda (yang merupakan string yang ditemukan dengan opsi WMIC COMMANDLINE) di bagian %start.bat% seperti yang saya lakukan di bawah — saya menggunakan start.bat dalam contoh saya di bawah ini seperti yang Anda gunakan dalam contoh Anda.

CONTOH PROSES FILE BATCH BATCH KHUSUS WMIC

WMIC PROCESS WHERE "COMMANDLINE LIKE '%start.bat%'" CALL TERMINATE

Bacaan Lebih Lanjut dan URL Sumber Daya: http://www.dedoimedo.com/computers/windows-wmic.html

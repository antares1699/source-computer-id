Tidak dapat menginstal Java 7 update 9 32-bit di Win 7 64

Karena ada beberapa versi Java di laptop saya, saya memutuskan untuk menghapus semuanya dan hanya menginstal ulang versi saat ini. Saya menghapus semua versi 64-bit dan 32-bit menggunakan Revo Uninstaller.

Setelah mengunduh versi saat ini, ketika mencoba menginstalnya, saya mendapatkan pesan bahwa versi 64-bit dan 32-bit tidak dapat diinstal karena sudah diinstal. Baik Program dan Fitur Windows 7 maupun Revo menunjukkan tidak ada Java yang diinstal di al. Saya kemudian mencoba Microsoft Fixit untuk masalah penginstalan dan pembongkaran (KB 2438651). Saya kemudian dapat menginstal versi 64-bit, tetapi terus mengalami masalah dengan versi 32-bit. Saya menjalankan CCleaner, Glary Utilities dan Auslogics Registry Cleaner, dan melihat kunci Java yang akan di-uninstall. Setelah membersihkan registri, saya mencoba menginstal Java versi 32-bit sekali lagi, tetapi tidak berhasil.

Ada saran tentang cara memperbaiki masalah ini?

Terima kasih atas panduan yang dapat Anda berikan.

Ini adalah masalah yang sangat umum ketika Java dihapus dengan tidak benar. Saat Java hilang, penginstal Java masih menganggapnya masih terpasang. Ini menentukan apakah benar atau tidak dengan memeriksa registri sistem Anda untuk kunci tertentu.

Cara mengatasinya sederhana. Hapus kunci ini dan Java tidak akan menginstal masalah.

Buka menu start Anda di Windows dan ketik regedit jalankan program yang muncul.

Perluas jalur folder HKEY_LOCAL_MACHINE\SOFTWARE\CLASSES\INSTALLER\PRODUCTS Kemudian periksa semua folder. Mereka semua harus terlihat seperti hash dari beberapa jenis. Klik masing-masing dan hapus seluruh folder ketika Anda menemukan satu yang ada hubungannya dengan Java.

Bagian yang sulit sudah selesai, sekarang untuk mengurus dua yang terakhir ...

Hapus juga yang berikut ini.

HKEY_LOCAL_MACHINE\SOFTWARE\JavaSoft\Java Runtime Environment

HKEY_LOCAL_MACHINE\SOFTWARE\wow6432node\JavaSoft\Java Runtime Environment

Siap. Jalankan penginstal Java sekali lagi dan seharusnya berfungsi dengan baik.

Bagaimana cara saya "mengglobalisasi" binari di Linux?

Misalnya, biner apa pun yang ditempatkan di /usr/local/bin akan berjalan di lokasi mana pun. Jika saya menempatkan binari di dalam /myname/local/bin bagaimana saya bisa mencapai hal yang sama?

Untuk mengaktifkan jalur kustom Anda secara default, Anda dapat menambahkan:

export PATH="/myname/local/bin:$PATH"

di bagian bawah ~/.bashrc Anda (hanya untuk pengguna) atau /etc/profile (untuk semua pengguna sistem) atau file baru seperti /etc/profile.d/mypath.sh

Jalur pencarian untuk binari disimpan di file variabel lingkungan , PATH

Anda dapat memeriksa nilai saat ini:

[Steven @ scstop: ~]% echo $ PATH

/ opt / local / bin: / usr / bin: / bin: / usr / sbin: / sbin: / usr / local / bin

Perhatikan bagaimana ini merupakan daftar direktori yang dipisahkan oleh titik dua. Ketika Anda mengetik "mybinary" pada prompt, shell Anda akan melalui direktori ini (secara berurutan) dan menjalankan direktori pertama yang ditemukannya.

Untuk menambahkan / myname / local / bin ke PATH, lakukan ini (dalam shell seperti bash)

ekspor PATH = $ PATH: / myname / local / bin

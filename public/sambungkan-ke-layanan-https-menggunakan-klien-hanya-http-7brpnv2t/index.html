stunnel aku s apa yang kamu cari:

sudo stunnel -c -r google.com:443 -d 127.0.0.1:8888

Ini menyiapkan sesi SSL ke pihak jarak jauh (Google dalam hal ini), dan membuat pendengar di localhost port 8888. Anda dapat menggunakan 80 jika Anda belum memiliki pendengar.

Kemudian Anda mengakses localhost: 8888 dan Anda akan mendapatkan situs remote.

Jika Anda menggunakan Windows, opsi baris perintah tidak didukung, jadi buat file stunnel.conf dengan parameter di dalam:

[remote]
accept = 8888
connect = google.com:443

Kemudian sebut saja dengan

stunnel -c stunnel.conf

Berikut skrip node.js yang melakukan apa yang saya inginkan:

var http = require('http');
var https = require('https');

http.createServer(function (req, resp) {
    var h = req.headers;
    h.host = "www.example.com";
    var req2 = https.request({ host: h.host, port: 443, path: req.url, method: req.method, headers: h }, function (resp2) {
        resp.writeHead(resp2.statusCode, resp2.headers);
        resp2.on('data', function (d) { resp.write(d); });
        resp2.on('end', function () { resp.end(); });
    });
    req.on('data', function (d) { req2.write(d); });
    req.on('end', function () { req2.end(); });
}).listen(9999, "127.0.0.1");
console.log('Server running at http://127.0.0.1:9999/');

Host dan port lokal keduanya di-hardcode, tetapi akan cukup mudah untuk menjadikannya parameter baris perintah.

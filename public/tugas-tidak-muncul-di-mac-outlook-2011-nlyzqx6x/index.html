Tugas tidak muncul di Mac Outlook 2011

Tempat kerja saya saat ini menggunakan Mac dan tempat kerja lama saya menggunakan Windows. Di tempat kerja lama saya, saya banyak menggunakan fungsionalitas Tugas Outlook untuk mengelola beban kerja saya.

Saya memahami bahwa fungsionalitas Tugas di Outlook 2011 untuk Mac sangat terbatas, jadi saya sangat senang menemukannya "cara-cara" yang berguna ini untuk memanfaatkan Tasks secara maksimal .

Masalah saya adalah tugas saya tidak muncul di folder Tugas, atau di mana pun dalam hal ini. Meskipun saya mencari judul tugas yang baru-baru ini saya temukan, saya masih tidak dapat menemukannya.

Setelah beberapa Googling saya temukan utas forum ini yang menunjukkan bahwa mungkin ada masalah dengan database Outlook , yang mengarah ke Microsoft KB.

Jadi saya menjalani semua langkah yang disarankan untuk membangun kembali / menambahkan identitas baru menggunakan "Microsoft Database Utility" - teorinya adalah bahwa jika saya membuat identitas baru, saya dapat menguji pembuatan tugas menggunakan identitas "batu tulis kosong".

Saat saya mengubah identitas default ke identitas yang baru saya buat menggunakan Microsoft Database Utility (harus memulai ulang komputer) Pembuatan tugas masih tidak berfungsi.

Setiap ide dihargai, saya sangat merindukan fungsionalitas tugas di Outlook 2010 untuk Windows.

Saya menggunakan El Capitan dengan Outlook 2016. Tugas saya tidak akan muncul, meskipun saya dapat menambahkan tugas dan menampilkannya di ponsel saya dan di OWA. Saya mendapatkan tugas saya untuk muncul dengan menambahkan dan menghapus direktori profil Outlook saya ke Spotlight Privacy.

Saya menggunakan artikel KB Microsoft (ditautkan di atas) dan menemukan langkah-langkah ini berhasil untuk saya:

  1. Tutup Outlook.
  2. Buka System Preferences> Spotlight> Privacy
  3. Seret ~ / Library / Group Containers / UBF8T346G9.Office / Outlook / Outlook 15 Profil / ke daftar Privasi Spotlight (atau klik "+" dan arahkan ke folder). Jika Anda tidak dapat menemukan folder Perpustakaan Anda: MacWorld: Cara melihat folder ~ / Library
  4. Klik "-" untuk menghapus folder tersebut.
  5. Keluar dari Preferensi Sistem.
  6. Buka Outlook 2016.

Saya kemudian melihat tugas saya.

Bagaimana cara menghapus kata sandi kunci SSH agar tidak diingat oleh Mac OS?

Mac OS X (Snow Leopard).

Saya tidak sengaja memilih opsi "ingat kata sandi di rantai kunci saya" saat memberikan frasa sandi kunci pribadi ssh saat menyambung ke server jauh.

Sekarang, ketika saya ssh ke server (bahkan setelah reboot) tidak pernah meminta kata sandi. Bagaimana cara membuatnya meminta saya lagi?

-M

Buka Akses Keychain (di / Applications / Utilities /), cari kunci SSH (coba ketik SSH di kotak pencarian), klik kanan dan pilih "Hapus".

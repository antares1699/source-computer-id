Bagaimana cara membuat drive pemulihan untuk Windows 8.1?

Saya baru saja menginstal Windows 8.1 di komputer saya. Saya mencoba membuat drive pemulihan.

Saya klik mulai, ketik "pemulihan", tekan "Buat drive pemulihan". Saya klik berikutnya sekali, dan inilah yang saya lihat:

Kami tidak dapat membuat drive pemulihan di PC ini

Beberapa file yang diperlukan hilang. Untuk memecahkan masalah saat PC Anda tidak dapat memulai, gunakan disk atau media instalasi Windows Anda.

Ini tampaknya menjadi masalah umum dengan Windows 8.1, tetapi sejauh ini saya belum menemukan solusinya.

Saya berasumsi Anda mencoba membuat drive pemulihan USB.

Menurut situs microsoft sini :

Peringatan

Membuat drive pemulihan akan menghapus semua yang sudah tersimpan di file Flash disk. Gunakan flash drive USB kosong atau pastikan untuk mentransfer data penting apa pun dari flash drive USB Anda ke perangkat penyimpanan lain sebelum menggunakannya untuk membuat drive pemulihan USB.

Jika Anda memiliki kunci yang valid, mungkin itu salah satu masalah berikut:

  • Anda tidak memiliki DVD Windows 8.1 asli yang dimasukkan ke dalam drive DVD (Anda harus memiliki DVD Instalasi Windows asli untuk membuat disk pemulihan).
  • Anda tidak memiliki drive USB yang dimasukkan, atau drive USB Anda tidak diformat, tidak diformat dengan benar, atau rusak.
  • Anda tidak memiliki partisi pemulihan yang digunakan untuk menginstal pemulihan (untuk menginstal di partisi pemulihan, Anda harus membuat partisi terlebih dahulu, karena proses ini tidak membuat partisi baru).
  • Versi Windows Anda rusak.
  • Versi Windows Anda tidak asli dan telah rusak sebelum dipasang.

Juga, menurut situs Microsoft sini , pastikan Anda mengikuti petunjuk berikut:

Langkah 1 : Hubungkan drive USB ke komputer dan cadangkan semua data sebelumnya melanjutkan ke langkah berikutnya karena drive akan dihapus selama proses.

Langkah 2 : Beralih ke mode desktop. Tekan tombol Windows + R untuk luncurkan dialog Jalankan. Pada dialog ketik control.exe lalu tekan enter kunci untuk meluncurkan Panel Kontrol.

LANGKAH 3 : Klik ubah Lihat oleh ke Ikon Kecil untuk melihat semua alat dan opsi. Klik Pemulihan untuk meluncurkan Jendela pemulihan.

LANGKAH 4 : Selanjutnya, klik Buat drive pemulihan pilihan. Klik Ya untuk perintah Kontrol Akun Pengguna untuk meluncurkan Buat wizard drive pemulihan. Jika Anda diminta memasukkan kata sandi admin, tolong lakukan itu.

Langkah 5 : Klik tombol Berikutnya, wizard akan memindai file komputer untuk semua drive USB yang terhubung.

LANGKAH 6 : Berikut ini layar, pilih drive USB yang ingin Anda gunakan sebagai drive pemulihan lalu klik tombol Next, dan terakhir klik tombol Create. Windows akan mentransfer file yang diperlukan ke drive dan drive pemulihan harus siap.

Saya memiliki masalah yang sama, dalam kasus saya masalahnya adalah file winre.wim hilang dari sistem saya. Ini berisi file pemulihan Windows.

Memasukkan DVD Windows 8.1 tidak membantu.

Untuk mendapatkan salinan winre.wim Windows 8.1, saya punya dua opsi (mungkin ada yang lain).

Satu - Pada sistem Windows 8.1 yang berbeda, buka prompt perintah yang ditinggikan dan ketik:

reagentc /disable

Ini menyalin file winre.wim dari partisi pemulihan tersembunyi ke C: \ windows \ system32 \ recovery, salin dari sana. Setelah selesai, aktifkan kembali lingkungan Pemulihan Windows:

reagentc /enable

Dua - Jika Anda memiliki Windows 8.1 ISO atau DVD, pasang install.vim dan salin winre.wim dari sana. Buat direktori kosong seperti C: \ mount. Kemudian buka kembali prompt perintah yang ditinggikan dan ketik:

dism /Mount-wim /wimfile:D:\sources\install.wim /index:1 /mountdir:C:\mount /readonly

kemudian buka C: \ mount \ windows \ system32 \ recovery dan salin Winre.wim dari sana. Lepaskan:

dism /Unmount-Wim /Mountdir:C:\mount /discard

Sekarang Anda memiliki salinan Winre.wim, salin ke C: \ windows \ system32 \ recovery pada PC yang hilang.

Akhirnya lagi di prompt perintah yang ditinggikan:

reagentc /enable

Ini akan memindahkan Winre.wim dan mengaktifkan lingkungan pemulihan, Anda sekarang harus dapat membuat drive USB pemulihan.

Tombol tengah mouse berfungsi seperti Alt-Tab

Mouse nirkabel saya (Microsoft Wireless Mobile Mouse 4000 pada Windows 7) mulai bertindak seperti ini: Saat Anda menekan tombol tengah di browser (mengharapkan tautan terbuka di tab baru), ia akan bertindak seperti jika Anda menekan Alt + Tab , membuka jendela dengan ikon aplikasi selama beberapa detik dan kemudian masuk ke aplikasi yang terbuka secara acak.

Tombol roda di Control Panel Mouse opsi disetel ke "Balik" (default). Mengembalikan default dan memulai ulang komputer tidak membantu.

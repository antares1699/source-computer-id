Tidak dapat me-rebind Ctrl-B ke Ctrl-A di tmux

Saya telah menyetel opsi untuk melepas ikatan Ctrl + B dan mengikat tmux Ctrl + SEBUAH tapi Ctrl + SEBUAH tidak bekerja, Ctrl + B masih. Saya mendapatkan konfigurasi tmux saya di tempat yang benar, seolah-olah saya mengubah beberapa opsi lain, saya dapat melihat hasilnya.

unbind C-b
set-option -g prefix C-a
bind-key C-a send-prefix

Apa yang salah? Cara membuatnya bekerja?

Konfigurasi ini berfungsi untuk saya. Anda mungkin belum me-restart server tmux Anda.

Dari man 1 tmux :

Pada tmux, sesi ditampilkan di layar oleh klien dan semua sesi dikelola oleh satu server. Server dan setiap klien adalah proses terpisah yang berkomunikasi melalui soket di /tmp.

Salah satu metode adalah dengan menghentikan semua yang ada di dalam tmux, jadi tmux ls menghasilkan "tidak ada server yang berjalan…". Pada saat ini, perintah tmux berikutnya akan memulainya lagi, memuat file konfigurasi baru Anda.

Baca juga ini: Tmux tidak mencari sumber .tmux.conf saya . Menurut salah satu jawaban, ini seharusnya berfungsi tanpa perlu memulai ulang server:

Ini juga dapat dilakukan dari dalam tmux, dengan menekan Ctrl + B (kunci awalan) dan kemudian : untuk memunculkan prompt perintah, dan mengetik:

source-file ~/.tmux.conf

Untuk memperluas jawaban Kamil Maciorowski, untuk memicu file konfigurasi baru, Anda dapat menghentikan sesi Anda dengan:

  • perintah terminal tmux kill-session -a dengan -a untuk semua
  • bergabunglah dengan sesi Anda yang ada dengan tmux a dan menggunakan awalan Anda ( CTRL + SEBUAH atau B ) untuk mengetik perintah :kill-session.

Buka Notepad dan salin / tempel kode berikut:

REGEDIT4
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\IniFileMapping\Autorun.inf]
@="@SYS:DoesNotExist"

Simpan file sebagai DisableAutorun.reg. (Anda harus yakin untuk mengubah "Simpan File sebagai Jenis" menjadi "Semua File" sebelum menyimpan, atau Windows akan mencoba menyimpannya sebagai .txt meskipun Anda mengetik dalam .reg).

Temukan file yang baru saja Anda simpan dan klik dua kali file tersebut untuk menjalankannya. Anda akan menerima prompt yang menanyakan apakah Anda ingin menambahkan data ke registri. Klik ya untuk mengizinkan modifikasi.

Metode di atas membatalkan permintaan apa pun untuk autorun.inf dan berfungsi di XP Home atau Pro, serta Windows Vista / 7.

Catatan: Jika Anda memilih untuk menonaktifkan autorun menggunakan metode lain (misalnya metode yang disarankan John T.), Anda harus menginstal terlebih dahulu tambalan autorun dari Microsoft yang diduga menyelesaikan masalah yang menyebabkan autorun berjalan bahkan setelah itu dinonaktifkan .

Untuk menonaktifkan autorun di semua perangkat:

Install patch ini dari Microsoft .

  1. Klik Mulai, klik Jalankan, ketik regedit di kotak Buka, kemudian klik OK.
  2. Temukan dan kemudian klik entri berikut di registri:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer\NoDriveTypeAutorun

  1. Klik kanan NoDriveTypeAutoRun, lalu klik Ubah.
  2. Di kotak Data nilai, ketik 0xFF untuk menonaktifkan semua jenis drive. Atau, untuk menonaktifkan drive tertentu secara selektif, gunakan nilai yang berbeda seperti yang dijelaskan di bagian "Cara menonaktifkan fitur Autorun tertentu secara selektif".
  3. Klik OK, dan kemudian keluar dari Editor Registri.
  4. Nyalakan ulang komputernya.

Bagaimana cara menghapus (basis) dari Powershell?

Setelah menginstal Anaconda untuk python. (base) muncul sebelum direktori di Powershell. Bagaimana cara menghapusnya?

Notat (base) ada untuk memberi tahu Anda lingkungan virtual tempat Anda berada. Ini sangat berguna saat Anda menggunakan lingkungan yang berbeda pada sistem yang sama.

Jika Anda tidak melakukan apa pun dengan python, Anda dapat menjalankan conda deactivate hingga Anda ingin menggunakan lingkungan python. Anda dapat menghentikannya secara otomatis dengan conda config --set auto_activate_base false

Jelas sekali fungsi prompt Anda telah dimodifikasi, mungkin oleh salah satu profil.

Lihat definisi dengan:

(Get-Command prompt).Definition

Dan lihat apakah Anda dapat mengevaluasi asalnya dengan melihat file profil yang memungkinkan:

$PROFILE|Format-List -Force

Anda juga bisa membiarkan PowerShell melakukan pekerjaannya:

## Q:\Test\2019\07\21\SU_1462281.ps1

$PROFILE.PSObject.Properties | 
  Where-Object Membertype -eq Noteproperty | ForEach-Object{
    if (Test-Path $_.Value){
      Select-String -Path $_.Value -Pattern 'function\s*prompt'
    }
  }
#

WINE dan virus Windows

Mungkin pertanyaan konyol, tapi saya bertanya-tanya ... WINE memiliki izin untuk mengedit folder (profil) tertentu dan menyediakan perangkat lunak berbasis Windows untuk dijalankan di Linux. Mungkinkah itu juga menyediakan lingkungan bagi virus Windows untuk dijalankan? Seperti menjalankan konektor botnet berbasis Windows, pengirim spam, menginfeksi program atau folder (WINE) lain, dll.

MEMPERBARUI:
Lebih lanjut, apa saja bahaya yang mungkin terjadi pada sistem Linux Anda yang memungkinkan hal-hal seperti itu berjalan?

  • Wine hadir dengan DLL-nya sendiri, yang biasanya tidak memiliki lubang keamanan yang sama dengan yang disediakan oleh Microsoft. Itu berarti Virus yang menargetkan kerentanan dalam sistem DLL kemungkinan besar tidak berhasil.

Itu mungkin.

Saya baru-baru ini menguji beberapa skenario di Ubuntu pada mesin virtual yang menjalankan trojan dan sejenisnya. Karena cara WINE dijalankan, mudah untuk melihat dan menghentikan proses, tetapi WINE dapat menjalankan malware - bahkan sebagai layanan.

Namun, biasanya tidak terlihat oleh pengguna akhir karena kebanyakan target Internet Explorer atau kunci registri yang tidak melakukan apa-apa.

Bahkan dengan hal di atas dikatakan, saya tidak pernah melihat cukup jauh ke file untuk benar-benar menemukan apa yang mereka lakukan - saya yakin itu menunggu perintah atau serupa.

Kabar buruknya adalah, tanpa pengalaman Linux yang baik, sulit untuk dihapus. Saya mencoba menghapus dan menginstal ulang WINE, tetapi konfigurasi dan berbagai file yang disimpan masih tersisa - Anda harus melakukan penghapusan manual lengkap dari konfigurasi dan drive WINE (Terletak di '~ / .wine'), dan berbagai skrip konfigurasi lainnya.

Saya tahu itu terinfeksi karena setiap kali saya me-restart mesin setelah dengan sengaja menginfeksi, services.exe (saya pikir itu disebut) dan berbagai proses WINE secara otomatis dimulai dengan mesin dan membutuhkan waktu CPU yang sangat besar di mana karena tidak pernah melakukan hal seperti itu di masa lalu .

Itu dimungkinkan, asalkan virus kompatibel dengan WINE :)

Hal besar tentang virus adalah ketergantungan dan cara penyebarannya. Di WINE, tidak semua fitur Windows akan tersedia, dan Anda mungkin akan mendapatkan kesalahan ketergantungan (dll) saat mencoba menjalankan beberapa.

Saya ingat artikel lama yang menguji beberapa virus paling populer di WINE, dan hasil akhirnya mengecewakan karena sistem tidak pernah terpengaruh secara parah.

Edit: Menjalankan virus Windows dengan Wine - bacaan yang bagus.

Bagaimana cara keluar dari aplikasi layar penuh yang macet di Windows 10?

Pada Windows 7, menekan Ctrl + Alt + Delete dan memilih Pengelola tugas hampir selalu menyembunyikan aplikasi layar penuh apa pun ke bilah tugas. Jika tidak, setidaknya itu menempatkan bilah tugas di atas aplikasi yang macet.

Saya macet saat memuat di windows 10, dan tidak ada trik dari Win7 yang berhasil. Saya bisa masuk ke menu Ctrl + Alt + Del. Saya dapat memilih Manajer tugas, tetapi program yang dibekukan masih dalam layar penuh.

Saya akhirnya memecahkan masalah dengan keluar, tetapi itu juga menutup jendela lain yang saya tidak ingin tutup.

Apakah ada alternatif dari apa yang biasa saya lakukan di Windows 7? Mungkin beberapa pengotak-atik registri akan membuat mode layar penuh lebih stabil?

Cara yang biasa untuk masuk dan keluar dari mode layar penuh adalah dengan menggunakan F11 kunci.

Jika ini tidak berhasil untuk Anda, coba pukul Alt + Ruang untuk membuka menu aplikasi dan klik (atau gunakan keyboard) untuk memilih Mengembalikan atau Memperkecil .

Cara lain adalah dengan memukul Ctrl + Bergeser + ESC untuk membuka Task Manager. Kalau masih tertutup oleh always-on-top program layar penuh, tekan Alt + HAI untuk membuka menu Opsi dan kemudian Memasukkan memilih Selalu di atas untuk Task Manager.

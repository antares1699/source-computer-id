Synology DSM 6.0 SSH - perintah ether-wake tidak lagi berfungsi

Sejak saya meningkatkan ke DSM 6.0, saya tidak lagi memiliki kemampuan untuk 'membangunkan perangkat lan di jaringan saya. Saya biasa melakukan SSH dari komputer jarak jauh dan menjalankan perintah berikut:

ether-wake -i eth0 AA:BB:CC:DD:EE:FF

Ini akan mengirim paket WOL ke desktop saya. Namun ketika saya mencoba dan menjalankan perintah ini setelah pembaruan, saya mendapatkan kesalahan berikut:

~ash: ether-wake: command not found

Apakah ada yang punya solusi untuk ini?

4

7
7

Mereka mengganti perintah dengan:

synonet --wake [MAC address] eth0

atau jika Anda memiliki ikatan:

synonet --wake [MAC address] bond0

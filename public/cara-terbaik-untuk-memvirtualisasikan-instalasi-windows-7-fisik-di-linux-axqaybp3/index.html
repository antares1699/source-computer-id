Cara terbaik untuk memvirtualisasikan instalasi Windows 7 fisik di Linux?

Sejak saya melihat seorang teman menjalankan Windows 7 di dalam Mac OS X dengan VMware Fusion, saya telah mencoba melakukan hal yang sama di laptop saya, untuk menghindari reboot setiap kali saya harus menggunakan aplikasi Windows.

Namun, tampaknya itu bukanlah tugas yang mudah. Saya ingin tahu apa opsi perangkat lunak terbaik untuk memvirtualisasikan partisi windows 7 yang ada di ubuntu (10.10). Saya tahu bahwa itu mungkin dengan VirtualBox, VMware Workstation dan QEMU, tetapi semua panduan dan tutorial yang saya lihat tampaknya sudah ketinggalan zaman (seperti 3 tahun).

Pertama saya tidak akan merekomendasikan menjalankan instalasi Windows yang sama baik secara native maupun virtual. Perbedaan perangkat keras antara VM dan mesin sebenarnya cenderung membuat Windows menjadi sedikit rumit.

Pertahankan instalasi Windows Anda saat ini untuk penggunaan asli, dan buat instalasi baru dalam produk virtualisasi untuk dijalankan dalam Ubuntu.

Saya pribadi menggunakan VirtualBox dan akan merekomendasikannya setiap saat. Saya dulu adalah penggemar berat VMware, tetapi VirtualBox baru saja mendapatkan keunggulan karena jauh lebih terbuka daripada VMWare. Bahkan produk "gratis" VMWare tampak dibatasi dan kikuk jika dibandingkan. Sejak VMWare Server (yang biasa saya gunakan sepanjang waktu) pergi ke front end berbasis web, saya tidak memiliki masalah akhir dengannya, dan itu hanya menolak untuk bekerja pada beberapa sistem operasi (Anda harus menerapkan tambalan untuk membuatnya bekerja dengan Ubuntu).

QEmu Saya bahkan tidak menempatkan di kelas yang sama dengan VirtualBox dan VMWare. Meskipun memiliki kegunaannya (saya menggunakannya untuk menguji-boot distro perangkat lunak saya) kinerja dan kegunaannya jauh di bawah VirtualBox.

Jadi, VirtualBox.

Pastikan Anda menggunakan versi 4, bukan versi 3 yang lebih lama yang ada di utama Ubuntu. Anda harus pergi ke Situs VirtualBox dan tambahkan repositori ke file sumber apt Anda atau unduh secara manual. Saya lebih suka menambahkan repositori karena ia melakukan pembaruan untuk Anda.

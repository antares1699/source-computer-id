Cara menyetel opsi peluncuran untuk program di OS X [duplikat]

Kemungkinan Duplikat:
Bagaimana cara menyampaikan argumen baris perintah ke item Dock?

Pada PC Windows, jika saya perlu memberikan opsi peluncuran apa pun ke aplikasi (misalnya, browser Chrome), saya dapat melakukannya dengan membuat pintasan ke Chrome.exe dan menambahkan opsi ke akhir properti Target pintasan.

Bagaimana saya bisa melakukan hal yang sama di Mac? Secara khusus saya perlu memberikan beberapa opsi peluncuran ke browser Firefox / Chrome saya. Saya menjalankan Mac OS X 10.7.3.

Dalam sistem Unix (OS X adalah salah satunya), opsi untuk program diberikan pada baris perintah, biasanya diawali dengan "-". Jadi, untuk menjalankan firefox dengan, misalnya sesi pribadi, buka terminal dan masukkan:

 $ firefox -private

Jika Anda ingin menjadikan ini default, setiap kali Anda menjalankan firefox, Anda dapat membuat alias BASH. Edit file .profile di direktori home pengguna Anda dan tambahkan baris ini:

alias firefox="firefox -private"

Setelah Anda menyimpan file itu, setiap kali Anda menjalankan firefox, Anda akan menjalankannya dengan opsi yang diinginkan.

Terakhir, untuk mendapatkan daftar opsi baris perintah yang tersedia untuk firefox, jalankan firefox -h dari terminal. Berikut adalah keluarannya di Linux:

Dari baris perintah, Anda dapat menggunakan berikut ini:

open -a ProgramName --args your program arguments

Karena aplikasi OS X adalah paket sebagai bundel aplikasi dan binari mereka biasanya tidak pada $PATH, cara "Unix" yang biasa untuk meluncurkan aplikasi biasanya terlalu rumit kecuali Anda bersedia alias atau alias semuanya. Bandingkan (jika Anda menginstal Firefox ke folder aplikasi default):

open -a Firefox
/Applications/Firefox.app/Contents/MacOS/firefox

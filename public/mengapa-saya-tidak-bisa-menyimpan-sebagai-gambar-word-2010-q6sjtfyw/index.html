Simpan gambar atau grafik SmartArt Anda sebagai file .gif, .png, atau .jpg

Anda dapat menyimpan gambar atau grafik SmartArt dalam format file grafik seperti Graphics Interchange Format (.gif), JPEG File Interchange Format (.jpg), atau Portable Network Graphics Format (.png).

  1. Klik gambar atau grafik SmartArt yang ingin Anda simpan dalam format file grafik.
  2. Pada tab Beranda, di grup Clipboard, klik Salin.
  3. Pada tab Beranda, dalam grup Clipboard, klik panah di bawah Tempel, lalu klik Tempel Spesial.
  4. Dalam kotak dialog Tempel Spesial, dalam daftar Sebagai, klik Gambar (GIF), Gambar (PNG), atau Gambar (JPEG).
  5. Klik kanan grafik, lalu klik Simpan sebagai Gambar.
  6. Ketikkan nama untuk file grafik Anda, telusuri ke lokasi tempat Anda ingin menyimpan file, lalu klik Simpan.

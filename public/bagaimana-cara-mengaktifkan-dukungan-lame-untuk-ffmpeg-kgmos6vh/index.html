Bagaimana cara mengaktifkan dukungan LAME untuk ffmpeg?

Ini bukan duplikat. Saya Debian 6 Squeeze dan menginstal ffmpeg menggunakan apt-get. Saya tidak mengkompilasinya dari sumber. Kemudian saya menginstal timpang dari http://www.rarewares.org/ . Tetapi ketika saya mencoba menggunakan ffmpeg -i some.flv -acodec lame -ab 128k my.mp3 saya melihat kesalahan Unknown encoder 'lame'. Jika saya mencoba menggunakan libmp3lame daripada lumpuh kesalahannya mengatakan Unknown encoder 'libmp3lame'. Tetapi lumpuh terpasang. Ada petunjuk?

Mungkin ffmpeg Anda tidak diinstal dengan dukungan LAME. Saya hanya mengatakan Anda mengunduhnya dari sumber dan mengkompilasinya dengan --enable-libmp3lame, atau tepatnya:

$ ./configure --enable-gpl --enable-liba52 --enable-libgsm --enable-libxvid \
--enable-libamr_nb --enable-libamr_wb --enable-libmp3lame --enable-libogg \
--enable-libvorbis --enable-libfaac --enable-libfaad --enable-shared

Ini kemudian harus menggunakan instalasi lame Anda. Jika tidak bisa: Dapatkan LAME dari sini .

Tidak perlu menginstal dari sumber ... Paket ffmpeg yang berasal dari deb-multimedia.org (5:0.7.13-dmo2 ATM) memiliki dukungan libmp3lame.

Saya melakukan hal berikut di Squeeze (sebagai root):

apt-get install deb-multimedia-keyring

Tambahkan ke /etc/apt/sources.list:

deb http://www.deb-multimedia.org squeeze main non-free

kemudian

apt-get update

maka jika Anda memeriksanya dengan sudo apt-cache policy ffmpeg, itu akan menunjukkan bahwa ffmpeg akan datang dari deb-multimedia.org, bukan dari repo default. Begitu:

apt-get install ffmpeg

dan itu harus mencakup dukungan LAME. (Saya juga menginstal lame ... Saya tidak yakin apakah itu diperlukan.)

Sejak Juli 2011, paket lame tersedia di main Debian, lihat halaman ini untuk detailnya: http://packages.qa.debian.org/l/lame.html .

Harap dicatat bahwa repositori paket deb-multimedia yang dirujuk dalam jawaban lain di sini tidak didukung, dan tidak direkomendasikan oleh Debian. Faktanya, itu sama sekali tidak berafiliasi dengan Debian. Informasi lebih lanjut dapat ditemukan di halaman wiki ini: http://wiki.debian.org/MultimediaCodecs

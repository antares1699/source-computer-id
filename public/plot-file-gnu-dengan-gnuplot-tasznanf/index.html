Plot file .gnu dengan gnuplot

Saya memiliki file dengan ekstensi .gnu. Ini berisi perintah gnuplot. Saya ingin menampilkan grafik, menggunakan gnuplot. Jadi, perintah mana yang tepat untuk ini.

Anda dapat menjalankan gnuplot secara langsung dengan

gnuplot

Lingkungan terbuka di konsol dan Anda dapat menambahkan perintah Anda setelah gnuplot>

load 'file.gnu'

Jika Anda tidak berada di direktori yang sama, Anda harus pergi ke sana sebelumnya atau menentukan jalur untuk file tersebut.

Anda bisa melihatnya ini tutorial dan sini adalah beberapa contoh.

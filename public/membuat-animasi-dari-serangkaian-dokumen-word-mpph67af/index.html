Membuat animasi dari serangkaian dokumen Word

Saya ingin membuat film "selang waktu" dari proses pengeditan multi-bulan dari dokumen Word yang besar . Saya telah membuat skrip yang berjalan setiap 15 menit, menyimpan salinan file jika telah diedit, meninggalkan saya dengan folder yang berisi banyak salinan dokumen, dinamai dalam urutan kronologis, menangkap evolusi dokumen. (Dokumen ditulis selama beberapa bulan, jadi perekaman layar langsung, dll. Bukanlah suatu pilihan).

Untuk membuat film secara manual, saya akan membuka setiap file, centang "View> Multiple Pages", sesuaikan faktor zoom untuk menampilkan semua halaman dokumen saat ini, dan buat screenshot. Bilas dan ulangi. (Film harus selalu menampilkan semua halaman dokumen, tidak masalah jika halaman dikurangi menjadi "thumbnail")

Bagaimana saya bisa mengotomatiskan proses ini? Saya akan melihat dua strategi; yang pertama menggunakan Word dan beberapa solusi skrip seperti VBA untuk mengotomatiskan proses manual yang dijelaskan di atas. Atau menggunakan alat baris perintah untuk membuat ikhtisar dokumen kata sebagai grafik, dan menggunakan skrip batch sederhana untuk merender semua dokumen. Saya tidak dapat menemukan solusi yang berhasil menggunakan salah satu dari dua strategi ini. (Saya sedikit lebih suka opsi pertama karena saya ingin antarmuka pengguna Word terlihat di film)

Tampaknya dari kiriman Anda memiliki ratusan, jika tidak ribuan, dari versi satu dokumen Word, dan Anda ingin membuat video evolusinya.

Berikut ini secara umum bagaimana saya akan mengatasi masalah seperti itu:

  • Unduh dan instal produk gratis berikut:

    • ShareX - produk tangkapan layar
    • AutoHotkey - manajer skrip makro
    • VirtualDub - editor video

  • Preset ShareX untuk menangkap wilayah layar tempat Word akan muncul, dengan tombol pintas untuk menangkap wilayah ini, dan mengaturnya untuk menyimpan tangkapan layar di beberapa folder kosong

  • Buat daftar nama semua file dokumen dalam urutan tanggal naik:

    dir /B /N D *.doc > script.ahk

  • Gunakan editor teks apa pun yang dapat memijat skrip ini ke dalam AutoHotkey script sehingga masing-masing nama dokumen di atas menjadi beberapa baris sebagai berikut:

    • mengeksekusi "document-name"
    • sleep jumlah detik yang diperlukan untuk memuat dokumen
    • kirim hotkey ShareX ke desktop agar screenshot diambil
    • tidur sebentar
    • tutup Word, menemukannya melalui judulnya, dengan mengirimkannya Alt + F4
    • tidur sebentar

  • Jalankan skripnya

  • Gabungkan kumpulan gambar menjadi video Timelapse menggunakan VirtualDub

Sebagai alternatif untuk membuat skrip yang dapat menjangkau ribuan baris, seseorang dapat menggunakan perintah AutoHotKey Loop, Files dan sort (Lihat manual ).

Skrip AutoHotkey mungkin memerlukan beberapa menit untuk dijalankan, jika tidak berjam-jam jika Anda memiliki ribuan dokumen. Uji terlebih dahulu pada sebagian kecil dokumen.

Jika Anda tidak tahu AutoHotkey, mulailah dengan tutorial yang ditemukan di situsnya dan google untuk tindakan tertentu. Ada banyak artikel dan contoh skrip untuk ditemukan, dan juga anggota forum AutoHotkey sangat membantu.

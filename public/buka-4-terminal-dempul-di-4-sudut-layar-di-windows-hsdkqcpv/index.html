Buka 4 terminal dempul di 4 sudut layar di Windows

Saya memiliki jalan pintas yang membuka terminal PuTTy. Ketika saya mengklik pintasan 4 kali (atau sekali), dapatkah jendela PuTTy sejajar di 4 sudut layar dan mengisi 4 area yang sama? Bagaimana?

Windows dan PuTTy tidak memiliki ini sebagai opsi default. Mungkin ada perangkat lunak pengatur layar otomatis di luar sana, tetapi saya tidak mengetahuinya (Anda dapat bertanya di http://softwarerecs.stackexchange.com ).

Yang akan saya lakukan adalah membuat file AutoHotKey naskah. Di bawah ini adalah versi yang sangat umum dari apa yang dapat Anda lakukan untuk meluncurkan 4 jendela PuTTy dan mengaturnya secara merata di sekitar layar dengan satu pintasan:

SetTitleMatchMode, 2
SetTitleMatchMode Fast

Run, % "C:\Putty\Install\Folder\putty.exe"
WinWait putty
;Upper Left
WinMove, putty, , 0, 0, 960, 540

Run, % "C:\Putty\Install\Folder\putty.exe"
WinWait putty
;Upper Right
WinMove, putty, , 960, 0, 960, 540

Run, % "C:\Putty\Install\Folder\putty.exe"
WinWait putty
;Lower Left
WinMove, putty, , 0, 540, 960, 540

Run, % "C:\Putty\Install\Folder\putty.exe"
WinWait putty
;Lower Right
WinMove, putty, , 960, 540, 960, 540

Ini mengasumsikan bahwa Anda memiliki layar HD (1920x1080). Anda dapat memasukkan informasi situs dan login ke setiap jendela, sehingga mereka secara otomatis akan memulai koneksi yang Anda inginkan untuk masing-masing jendela juga. Anda hanya perlu mencari perintah baris perintah PuTTy yang Anda perlukan untuk situs yang Anda gunakan dan memasukkannya sebagai bagian dari perintah Run untuk setiap jendela.

Jawaban ini menggunakan Snap Assist dari Windows 10 dan AutoHotKey .

Snap Assist dapat menempatkan dan mengubah ukuran jendela pada setengah layar atau seperempat layar dengan hotkeys Corner Snap berikut:

  • Menang + Kiri , Menang + Naik : Layar seperempat bagian kiri atas
  • Menang + Kiri , Menang + Turun : Layar seperempat bagian kiri-bawah
  • Menang + Baik , Menang + Naik : Seperempat layar kanan atas
  • Menang + Baik , Menang + Turun : Seperempat layar kanan-bawah

Skrip AutoHotKey berikut akan menempatkan notepad di file layar seperempat kiri atas dan akan mengubah ukurannya menjadi ukuran seperempat layar:

SetTitleMatchMode, 2
SetTitleMatchMode Fast

Run, % "C:\Windows\System32\notepad.exe"
WinWait Notepad
Send {LWin down}{Left}{LWin up}{LWin down}{Up}{LWin up}

Anda harus mengubah skrip sebagai berikut:

  • 3 baris terakhir harus direplikasi untuk setiap pemanggilan dempul
  • Dalam Jalankan perintah , ganti path ke notepad.exe dengan path ke putty.exe. Parameter dempul ditambahkan segera setelah program dengan kosong pemisah. Jika parameter berisi karakter khusus, apit dengan tanda kutip ganda.
  • Dalam Perintah WinWait , parameter harus berisi beberapa kata pengenal atau sebagian kata pada judul jendela yang unik untuk permintaan dempul itu
  • Tombol di Kirim perintah harus dimodifikasi untuk menentukan seperempat layar yang tepat. Kode kunci AutoHotKey lain untuk digunakan adalah: {Down} dan {Right}.

Agar ini berfungsi, Windows 10 Snap Assist harus diaktifkan, yang merupakan statusnya secara default. Jika perlu diaktifkan, luncurkan aplikasi Pengaturan dari Start Menu, klik System, klik Multitasking di sebelah kiri, dan atur opsi Snap. Dua opsi berikut harus disetel ke Hidup:

Bagaimana saya bisa masuk ke BIOS di Sony Vaio VPCEH25EN saya

Saya memiliki Sony Vaio, model VPCEH25EN dengan saya. Ini referensinya (Sony Vaio VPCEH25EN) . Itu datang dengan jendela 7 yang sudah diinstal sebelumnya. Saya ingin menginstal Fedora bersamaan dengan itu. Untuk itu saya telah mencoba masuk ke BIOS mesin saya.

Saya menekan lama tombol Assist di laptop saya. Namun itu tidak berhasil. Ketika saya menekannya, itu terlihat

Windows sedang memuat berkas

di layar hitam dan kemudian ditampilkan

Windows dimulai

Kemudian masuk ke mode Penyelamatan Perawatan Vaio. Di sana saya hanya punya tiga pilihan

  • Mulai Wisaya Pemulihan
  • Alat
  • Keluar

Tidak ada opsi BIOS yang ditampilkan

Saya pikir, Windows tidak mengizinkan saya untuk masuk ke mode BIOS mesin saya, karena mulai memuat file windows setiap kali saya mengklik tombol Assist.

Bagaimana saya bisa masuk ke Sistem BIOS saya?

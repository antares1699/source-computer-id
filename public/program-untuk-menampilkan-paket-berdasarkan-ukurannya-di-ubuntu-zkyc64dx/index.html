Program untuk menampilkan paket berdasarkan ukurannya di Ubuntu

Saya berhasil mengisi partisi Ubuntu saya dengan menginstal sejumlah besar paket. Ketika saya menginstalnya, saya berpikir tentang masa depan, jadi ada sejumlah paket yang tidak akan saya gunakan selama beberapa bulan.

Sekarang untuk pertanyaan saya: Apakah ada program yang akan membantu saya menemukan kandidat untuk dihapus? Saya ingin melihat daftar paket yang diurutkan berdasarkan jumlah ruang disk yang mereka konsumsi. Program baris perintah juga bagus, tapi GUI yang bagus merupakan nilai tambah.

Dua jawaban:

Dari baris perintah:

Instal paket wajig, dan kemudian jalankan

wajig large

Dan Anda akan mendapatkan daftar paket terbesar yang diinstal.

Halaman man online untuk itu sangat tidak berguna; dokumentasi utama terletak di http://www.togaware.com/wajig/

Menggunakan GUI

Synaptic memiliki fungsionalitas bawaan ini. Cukup luncurkan Synaptic, pilih "semua" di kolom sebelah kiri dan klik tajuk kolom "ukuran" untuk mengurutkan paket berdasarkan ukuran yang diinstal.

Kedua metode tersebut akan memberi Anda nilai yang persis sama. Saya lebih suka menggunakan baris perintah karena saya biasanya menginstal menggunakan aptitude, jadi saya dapat menelepon aptitude purge sesudahnya dan cukup yakin setiap jejak telah dihapus.

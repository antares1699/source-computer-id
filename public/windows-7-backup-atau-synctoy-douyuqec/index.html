Windows 7 Backup atau SyncToy

Sebelum menginstal Windows 7, saya telah menggunakan SyncToy untuk menyinkronkan semua file penting saya ke hard drive eksternal cadangan. Windows 7 memiliki utilitas cadangan bawaan dan saya bertanya-tanya apakah mungkin saya harus menggunakannya.

Apakah Windows 7 Backup menawarkan keunggulan dibandingkan menggunakan SyncToy?

Saya melakukan hal yang persis sama. Saya juga mencoba Windows 7 Backup. Perbedaannya adalah SyncToy (dan SyncBack kerabat dekatnya) menyimpan file dalam struktur yang persis sama dengan yang Anda lihat di komputer Anda. Anda dapat dengan mudah memahami dan menyalin file yang dicadangkan.

Cadangan Windows 7 memiliki antarmuka yang bagus, dan saya pikir akan memungkinkan Anda melakukan pemulihan sistem penuh dari lokasi cadangan. Namun tampaknya menyimpan file yang dicadangkan dalam folder dan direktori yang berantakan.

Saya akan meringkas sebagai berikut:

  • Synctoy / Syncback untuk backup file
  • Cadangan Windows 7 untuk pemulihan sistem

Win 7 backup lebih merupakan solusi backup daripada solusi sinkronisasi. Ini memungkinkan Anda untuk memilih file dan pengaturan dan kemudian mencadangkannya ke satu file yang dapat dipulihkan nanti. Itu juga dapat menyimpan cadangan tambahan, dan saya yakin Anda dapat memiliki banyak titik waktu dalam satu cadangan. Jika Anda mencari solusi cadangan, maka itu bagus. SyncToy lebih merupakan alat sinkronisasi, yang membuat folder menjadi folder yang cocok b.

Jika Anda ingin mengakses file Anda dari kedua lokasi, SyncToy. Jika Anda ingin pemulihan sistem dengan cepat, maka saya akan menggunakan Backup.

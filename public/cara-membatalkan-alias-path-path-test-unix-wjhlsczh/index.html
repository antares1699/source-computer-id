Cara membatalkan alias, PATH = $ PATH: ./ Test - (Unix)

1.) Di terminal saat kita membuat alias untuk baris perintah apa pun, masa pakai alias itu selama terminal terbuka.

alias new_command=old_command

contoh: alias dog=cat

Tapi bisakah kita secara eksplisit batalkan perintah alias? yaitu membatalkan alias yang ditugaskan ke perintah itu tanpa menutup terminal.

2.) Juga saat kita melakukan sesuatu seperti PATH=$PATH:.:/Test

Kami menambahkan file ke direktori itu di mana semua perintah disimpan. Bagaimana kami membatalkan ini? Salah satu cara melakukannya adalah dengan menutup terminal, bagaimana dengan cara lain?

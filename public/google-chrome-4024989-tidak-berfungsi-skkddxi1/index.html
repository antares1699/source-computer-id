Apa yang mungkin menjadi pilihan, adalah mencadangkan file Bookmarks dan kemudian menginstal ulang Chrome dari awal jika salah satu file di profil Anda rusak.

  1. Cadangan file Bookmark dari %localappdata%\Google\Chrome\User Data\Default
  2. Copot pemasangan Chrome
  3. Hapus folder %localappdata%\Google\Chrome secara manual
  4. Pasang ulang Chrome
  5. Salin file Bookmark kembali ke %localappdata%\Google\Chrome\User Data\Default

EDIT - sebagai tanggapan atas komentar tentang penginstalan BitDefender :

Coba ini di BitDefender (mode ahli):

  1. Klik Antivirus | Perisai | Pengaturan lanjutan
  2. Tambahkan Chrome ke Pengecualian, C:\Documents and Settings\Tom\Local Settings\Application Data\Google\Chrome\Application\chrome.exe

Ada perbedaan antara bookmark dan halaman yang paling sering dikunjungi, yang muncul di halaman Tab Baru. Yang pertama dapat Anda lihat pada bilah alat tepat di bawah bilah alamat (seperti pada gambar Anda), yang lainnya dihasilkan dari riwayat penelusuran Anda.

Penanda disimpan di %localappdata%\Google\Chrome\User Data\Default\Bookmarks. Halaman yang paling sering dikunjungi tidak disimpan, tetapi dibuat dari %localappdata%\Google\Chrome\User Data\Default\History* file

Rekomendasi saya adalah Anda harus menyimpan seluruh profil Anda, yaitu folder %localappdata%\Google\Chrome\User Data\Default dan menyalinnya ke lokasi yang aman. Karena sepertinya Google Chrome macet di komputer Anda tidak bergantung pada profil Anda, Anda dapat mencoba memulihkan bookmark / halaman yang paling sering dikunjungi di komputer teman. Sementara itu, Anda juga dapat mencoba memperbaiki masalah dengan Google Chrome:

  1. Copot pemasangan Google Chrome
  2. Hapus seluruhnya folder ini: %localappdata%\Google\Chrome
  3. Telusuri registri Anda untuk menemukan string "Chrome" dan hapus semua kunci dan nilai terkait
  4. Mulai ulang
  5. Perbarui database antivirus Anda dan pindai komputer Anda dari virus
  6. Unduh versi terbaru Google Chrome dari http://www.google.com/chrome . Jangan gunakan kembali pemasang lama - pemasang mungkin terinfeksi oleh virus.
  7. Instal dan coba jalankan

Jika ini masih tidak membantu, maka masalahnya sangat rumit dan membutuhkan perhatian dari seorang profesional TI (dengan remote control atau serupa). Jika Anda memiliki teman seperti itu - biarkan mereka mencoba membantu. Mempekerjakan profesional biasanya cukup mahal, saya kemudian akan merekomendasikan menggunakan browser lain atau menginstal ulang Windows.

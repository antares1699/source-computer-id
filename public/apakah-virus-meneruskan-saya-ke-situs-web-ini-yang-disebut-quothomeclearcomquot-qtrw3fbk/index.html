Apakah virus meneruskan saya ke situs web ini yang disebut "home.clear.com"?

Setiap kali saya mencoba mengakses situs web, saya diteruskan ke http://home.clear.com.

Di Firefox, peringatan akan muncul yang mengatakan:

home.clear.com menggunakan sertifikat keamanan yang tidak valid. Sertifikat tidak dipercaya karena tidak ada rantai penerbit yang diberikan. (kode kesalahan: sec_error_unknown_issuer)

Di Chrome dan IE, tidak ada peringatan yang muncul dan saya dibawa ke home.clear.com. Saya telah menjalankan Norton Anti-Virus tidak berhasil serta program spyware lainnya (Hitman, saya kira).

Saya baru-baru ini memasang adaptor jaringan Netgear eksternal karena saya mengalami masalah koneksi di kediaman baru saya dan adaptor baru tampaknya menyelesaikan masalah hingga masalah ini muncul.

Beberapa minggu yang lalu saya mengklik scam jenis pembajakan klik Facebook tetapi tidak mengalami masalah apa pun sampai sekarang.

Saya melihat pertanyaan ini tetapi tidak dapat menyelesaikan masalah: Peramban meneruskan ke situs web acak secara otomatis

Ini bukan virus / spyware. Jika Anda memiliki notebook yang menggunakan adaptor Intel WiMax (Beberapa model Toshiba, khususnya), Anda akan diarahkan ke home.clear.com untuk mendaftar layanan Clear WiMax jika Anda mencoba menggunakan bagian WiMax perangkat. Jika Anda terhubung ke titik akses nirkabel, mungkin terhubung ke modem WiMax yang menunjukkan perilaku yang sama (Layanan tidak aktif).

Mengenai kesalahan sertifikat keamanan, itu bisa jadi banyak hal. Periksa jam dan kalender Anda untuk mengonfirmasi bahwa Anda telah menyetel tanggal / waktu / zona waktu yang benar. Jika Anda tidak terhubung ke koneksi internet yang berfungsi, Anda mungkin menerima kesalahan sertifikat karena browser Anda tidak dapat memverifikasi status pencabutan sertifikat.

Apple Music melompati lagu

Beberapa hari yang lalu saya melihat iTunes melewatkan lagu. Saya tidak begitu mengerti apa yang terjadi jadi saya mulai menyelidiki:

  • Jaringan yang berbeda tidak membuat perbedaan
    • tidak ada masalah dengan router / jaringan saya
  • Perangkat yang berbeda tidak membuat perbedaan (OS X, iOS, tvOS)
    • tidak ada masalah dengan instalasi / sistem operasi / perangkat lokal saya
  • Terjadi pada lagu dari "Untuk Anda", "Baru", dan perpustakaan saya.
    • tidak ada masalah dengan Perpustakaan Musik iCloud atau perpustakaan lokal saya
  • Langganan berbayar dan aktif (disetujui oleh dukungan iTunes Store)
    • tidak ada masalah dengan langganan itu sendiri
  • Tidak masalah dengan lagu berbayar dari iTunes Store
    • ... tapi saya tidak bisa begitu saja membeli seluruh perpustakaan saya dari iTunes
  • Tidak masalah dengan lagu yang saya simpan secara offline
    • ... tetapi saya tidak bisa begitu saja mengunduh setiap lagu yang ingin saya dengarkan
  • Tidak masalah dengan radio (mis. Beats 1, radio tematik seperti "On The Dancefloor", radio khusus seperti "[interpreter] Radio" dan "[lagu] Radio")
    • ... tapi saya tidak bisa begitu saja memulai radio untuk setiap lagu yang ingin saya dengarkan

Saya membuka kasus dengan Dukungan iTunes Store dan masih menunggu tanggapan (mereka memberi tahu saya bahwa saya mungkin harus menunggu hingga satu minggu). Jadi, jika mereka dapat mengatasinya, saya akan dengan senang hati membagikan solusinya dengan orang lain yang mungkin mengalami masalah yang sama.

Apakah ada yang pernah mengalami masalah yang sama dan dapat memperbaikinya (menggunakan Dukungan Apple atau tidak)?

Adakah yang tahu cara memicu beberapa rutinitas untuk mungkin menghapus beberapa jenis cache di akun Anda, yang mungkin menyelesaikan masalah?

Saya pernah mengalami masalah serupa di masa lalu karena saya mengubah perpustakaan saya di disk, dan cache masih mencari lagu lama.

HARAP DICATAT: Saya tidak pernah menggunakan iTunes Cloud: Hanya menggunakan file lokal di disk. Jika Anda mengalami masalah dengan file cloud, sayangnya saya tidak memiliki pengalaman dengan ini.

Itu entah bagaimana akan berakhir dalam loop kadang-kadang, yang mungkin merupakan campuran kode buruk + saya tidak memperbarui perpustakaan.

Solusi bagi saya adalah:

Hapus folder yang dipantau.
Hapus perpustakaan (tetapi bukan file di disk).
Tambahkan kembali folder yang dipantau dan biarkan iTunes mengimpor ulang file.

Menurut pengalaman saya, iTunes mengalami kesulitan dalam mengelola file yang diubah.

Saya tidak benar-benar menggunakan iTunes lagi karena alasan seperti itu, tapi saya harap ini membantu.

Bagikan

Menggunakan ikon dari Windows DLL di autorun.inf

Saya ingin menetapkan ikon untuk drive yang dapat dilepas sebagai ikon dari DLL Windows bawaan menggunakan autorun.inf. Sejak Halaman Wikipedia di Autorun.inf menunjukkan bahwa ikon dapat diambil dari DLL dan digunakan dengan indeks, menurut saya yang berikut ini akan berfungsi (# 54 dari imageres.dll adalah ikon tempat sampah daur ulang lengkap):

[autorun]
label=USB Drive
icon=%WINDIR%\System32\imageres.dll,54

Tapi ternyata tidak. Ikon tersebut muncul sebagai ikon "ikon tidak ditemukan" default saat saya melepas dan menyambungkan perangkat lagi. Bahkan jika saya menggunakan jalur absolut untuk sistem saya C:\Windows\System32\imageres.dll,54, ​​itu tidak berhasil. Bisakah ini dilakukan sama sekali, dan jika ya, bagaimana?

Bisakah saya mengupgrade CPU laptop saya

Saya memiliki Lenovo Ideapad Y510 dengan prosesor dual core 1,6 Ghz, ram 1GB, HDD 160GB. Saya ingin memutakhirkan cpu saya saat ini dengan yang terbaru seperti i3, i5 atau i7 untuk mendapatkan lebih banyak kecepatan tetapi tidak tahu apakah itu memungkinkan. Saya tidak tahu bagaimana mendapatkan semua detail teknis dari pc jadi saya telah melampirkan tangkapan layar dari layar boot, semoga Anda akan menemukan semua info yang diperlukan tentang pc saya untuk menjawab pertanyaan ini.

Saya seorang pengembang java jadi kebutuhan dasar saya adalah menjalankan aplikasi eclipse dan java dan server aplikasi dengan db seperti MySql dengan sangat lancar di mesin saya. Sebelumnya saya menggunakan Win7 tetapi mesin berjalan lambat jadi saya beralih ke Ubuntu 12.04 tetapi tidak banyak perbaikan jadi saya berpikir untuk memutakhirkan cpu dan memori. Saya memerlukan saran Anda bahwa cpu mana yang dapat saya tingkatkan dan haruskah saya juga memutakhirkan ram.

Tidak, Anda tidak dapat memutakhirkan seri Dual Core ke seri I3 / I5 / I7 yang baru, karena arsitekturnya lebih baru dan minimal memerlukan soket yang berbeda.

  1. Anda dapat mengganti CPU di sebagian besar laptop, karena tidak disolder pada motherboard. Jelas Anda harus tetap berpegang pada soket yang tepat. Dalam kasusnya, Anda dapat mengupgrade dual core 1.6 ke T9900 3.06 GHz misalnya.

Berikut adalah postingan forum tempat pengguna lenovo melaporkan keberuntungan mereka dengan mengupgrade CPU laptop ini: http://forums.lenovo.com/t5/IdeaPad-Y-U-V-and-Z-series/y510-CPU-Options/td-p/14857

Silakan beli model CPU yang mereka laporkan BEKERJA untuk laptop Anda.

  1. Anda tidak memerlukan sistem pendingin tambahan, produsen laptop biasanya memasang sistem pendingin yang tepat di semua model untuk memangkas biaya dengan jelas. Biayanya lebih mahal untuk mencari atau membuat sistem pendingin yang berbeda daripada hanya menggunakan 1 pada semua model dan hanya mengubah spesifikasi CPU / RAM / HDD.

  2. Jika ingin segera mengupgrade hasil ram terlebih dahulu, seperti yang disarankan oleh member lain. 1GB ram sangat kecil hari ini, dan tongkat ram tidak memerlukan biaya. Meskipun saya harus mengatakan bahwa saya menjalankan server ubuntu dengan java / mysql / apache pada prosesor 2gb dan intel ATOM dan tidak memiliki masalah dengan kecepatan. Anda dapat mencoba distribusi mint sebagai gantinya? itu memiliki footprint yang jauh lebih ringan.

  3. Mysql di SSD sangat cepat.

Sangat jarang bisa mengupgrade cpu di laptop.

  1. Motherboard laptop tidak benar-benar dibuat untuk diperbaiki saat ini. Jadi mengganti cpu adalah keras (jika memungkinkan karena perubahan soket)
  2. CPU baru yang lebih kuat mungkin membutuhkan lebih banyak pendinginan daripada yang dapat disediakan laptop Anda

Saran standar untuk mempercepat laptop adalah memaksimalkan RAM dan membeli SSD

Biasanya tidak, laptop tidak menggunakan soket ZIF untuk motherboard. memutakhirkan cpu di laptop hanya sedikit lebih sulit daripada di desktop, karena fakta bahwa lebih sulit untuk dibuka daripada desktop, dan cpu lebih sulit didapat. Masalah pertama Anda adalah Socket CPU. CPU Seri "I" mengikuti soket 1155 dengan FSB 1066 atau 1333 tetapi saya rasa perangkat Anda tidak dapat dijalankan dengan masalah penting ini. Pada gambar Anda, Anda dapat melihat Ram 1 slot dengan 533 Bus. perangkat Anda dapat berjalan dengan spesifikasi DDR2 dan Anda tidak dapat menggunakan DDR3 untuk Laptop Anda.

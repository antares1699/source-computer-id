Jalankan Startup Disk Creator sebagai Root

Saya mengalami masalah dengan izin saat mencoba membuat disk mulai menggunakan Startup Disk Creator. Saya dapat menyelesaikan tugas ini dengan menggunakan UNetBootin tetapi disk yang dibuatnya gagal untuk boot. Apakah ada cara untuk memulai Startup Disk Creator dari terminal sebagai root?

Saya sudah mencoba mencarinya menggunakan dengan memasukkan 'start' atau 'startup' dan memasuki tab tab tetapi tidak berhasil. Namun saya dapat menemukannya dari penelusuran dari dasbor ke rumah. Apa yang kulewatkan di sini?

Perintah untuk pembuat disk mulai adalah usb-creator-gtk. Jalankan sebagai root dengan gksudo usb-creator-gtk.

Di masa mendatang, Anda mungkin menemukan perintah apropos berguna. Ini memungkinkan Anda untuk menemukan perintah yang sesuai.

Dalam hal ini, apropos startup memberi saya:

default.pa (5)       - PulseAudio Sound Server Startup Script
genisoimagerc (5)    - startup configuration file for genisoimage
start-pulseaudio-kde (1) - PulseAudio Sound Server KDE Startup Script
start-pulseaudio-x11 (1) - PulseAudio Sound Server X11 Startup Script
startup (7)          - event signalling system startup
usb-creator-gtk (8)  - Ubuntu startup disk creation tool for Gtk+

Saya tidak memiliki akses ke mesin Ubuntu saat ini dan "Startup Disk Creator" sepertinya merupakan aplikasi Ubuntu. Bagaimanapun, apa yang dapat Anda lakukan adalah menjalankan ps untuk membuat daftar proses yang berjalan sebelum dan sesudah menjalankan program:

  1. Buat daftar proses yang sedang berjalan milik pengguna Anda dan simpan ke file

    ps xo user,command | cut -d ' ' -f 2- > file1

  2. Buka "Startup Disk Creator" dari menu dan kemudian jalankan kembali perintah di atas, simpan hasilnya ke file lain

    ps xo user,command | cut -d ' ' -f 2- > file2

  3. Sekarang bandingkan kedua file tersebut, seharusnya hanya ada satu perbedaan, "Startup Disk Creator"

    diff file1 file2

  4. Itu saja, Anda sekarang tahu perintah sebenarnya yang digunakan untuk meluncurkan program dan dapat meluncurkannya sebagai root.

Anda mungkin juga tertarik dengan berbagai program lain yang dapat membantu Anda membangun USB yang dapat di-boot. Inilah beberapa favorit saya:

  • Multiboot USB langsung
  • unetbootin
  • YUMI

Sumber informasi yang bagus untuk hal semacam ini adalah flashdisk linux situs web.

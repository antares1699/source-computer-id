Masalah RAM dengan Dell XPS 1640

Saya memiliki laptop ini dari tahun 2009

  • N07X1604
  • STUDIO XPS 16: INTEL CORE 2 DUO T9900 (3
  • MEMORI: 4096MB (2X2048) 1067MHZ DDR3 DU

Seperti yang Anda lihat, ini hadir dengan 2 modul 2gb 1067MHZ DDR3. Sekarang saya sudah mencari di dokumentasi di sini http://support.euro.dell.com/support/edocs/systems/sxl16/en/index.htm

http://support.euro.dell.com/support/edocs/systems/sxl16/en/sg/sg_en.pdf di halaman 63

sehingga tampaknya mendukung ram 8gb. tetapi satu 4gb tampaknya tidak didukung?

Saat saya memasukkan dua modul 4gb ddr3 1066mhz, tombol nonaktifkan suara menyala dan tombol naikkan volume menyala dan tombol turunkan volume berkedip. sistem tidak bisa boot. Dengan memori lama, itu boot.

Jadi, apakah ram 8gb mungkin? atau tidak? banyak situs web (yang menjual ram) mengatakan bahwa xps 1640 dapat memiliki ram 8gb.

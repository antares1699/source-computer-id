Mengapa ada kesalahan dengan rumus Excel sederhana ini menggunakan MOD dan ROW ()

Saya mencoba untuk mengatur gaya baris bergantian pada pilihan. Saya klik pada baris pertama dan seret ke bawah ke baris 145 (dimana data berakhir).

Lalu saya masuk ke Format Bersyarat, memilih aturan baru, dan mencoba memasukkan rumus ini:

=MOD(ROW(),2)=0

Ketika saya mengklik OK, saya mendapatkan kesalahan yang tidak terlalu membantu ini di Excel 2016:

There's a problem with this formula. 
Not trying to type a formula? When the first character is 
an equal (=) or minus (-) sign, Excel thinks it's a formula.
<snip>

Ada apa dengan rumusnya?

Hapus saja bagian =0 dan itu akan berfungsi.

Saat Anda membuat aturan, pilih opsi Use a formula to determine which cells to format, lalu masukkan rumus berikut ini:

=MOD(ROW(),2)

Jika Anda ingin memformat kumpulan baris lainnya, gunakan yang berikut ini:

=MOD(ROW(),2)-1

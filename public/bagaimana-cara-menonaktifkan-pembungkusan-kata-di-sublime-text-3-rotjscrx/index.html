Bagaimana cara menonaktifkan pembungkusan kata untuk file yang diberikan?

Saya sudah mencoba menyetel: "word_wrap": false di setelan Spesifik Sintaksis Pengguna, tetapi opsi ini tidak berpengaruh.

Menurut beberapa sumber online, seharusnya ada tanda koma , setelah salah.

"word_wrap": "false",

Anda juga dapat mengatur lebar bungkus menjadi angka besar:

"wrap_width": 9999,

Sumber SUBLIME TEXT 3: bagaimana cara menonaktifkan bungkus kata dalam output build? - Dukungan Teknis - Forum Sublime

Dan ada juga Word Wrap: Toggle dari palet perintah:

Item Word Wrap: Toggle dari Command Palette menggunakan toggle_setting perintah. Ini menetapkan preferensi langsung di tampilan, yang memang menimpa default, pengguna, atau sintaks tertentu preferensi.

Anda dapat kembali ke perilaku pengaturan khusus pengguna / sintaks dengan menutup dan membuka kembali semua file di mana Word Wrap: Toggle item dari Command Palette digunakan, sebagai tampilan pengaturan spesifik akan hilang saat file ditutup. Atau, Anda dapat mencapai hal yang sama tanpa menutup file apa pun dengan membuka konsol ST Python (Lihat menu -> Tampilkan Konsol) dan ketik view.settings().erase('word_wrap') Memasukkan . Ini akan jelas pengaturan tampilan khusus untuk tab aktif.

Anda juga bisa membuatnya untuk menghapus pengaturan dari semua tampilan terbuka di jendela saat ini sekaligus, menggunakan [view.settings().erase('word_wrap') for view in window.views()] Memasukkan .

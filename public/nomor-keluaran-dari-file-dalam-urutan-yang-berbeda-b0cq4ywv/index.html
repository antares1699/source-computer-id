Coba lakukan ini di perl

 perl -lane '$c=0; for (@F){ print "$F[$c]\t$F[$c+=1]" if $F[$c+1]}' file.txt

Atau terurai:

perl -lane '
    $c=0;
    for (@F) {
        print "$F[$c]\t$F[$c+=1]"
            if $F[$c+1];
    }
' file.txt

PENJELASAN

  • lane saklar berarti: l = baris baru; a = autosplit dalam larik @F; n = seperti while (<>) operator berlian ajaib; e = saklar dasar untuk menjalankan perintah
  • $c=0 menetapkan 0 ke penghitung
  • for (@F) { untuk setiap elemen dari baris saat ini
  • $c=0: mencetak elemen array dengan indeks $ c + tab + $ c + 1
  • if $F[$c+1];: terapkan baris terakhir hanya jika $ F [$ c + 1] bukan null

Atau menggunakan pesta (algoritma yang sama), mungkin lebih dapat dibaca oleh manusia untuk pemula:

while read a; do
    arr=( $a )
    for ((i=0; i< ${#arr[@]}; i++)); do
        [[ ${arr[i+1]} ]] && echo "${arr[i]} ${arr[i+1]}"
    done
done < file.txt

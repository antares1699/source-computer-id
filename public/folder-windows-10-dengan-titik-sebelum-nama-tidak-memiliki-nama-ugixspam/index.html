Folder Windows 10 dengan titik sebelum nama tidak memiliki nama

Saya memutakhirkan dari Windows 7 ke Windows 10, dan banyak folder saya yang memiliki titik sebelum namanya sekarang tidak memiliki nama, tetapi ada dan berfungsi, hanya nama kosong. Bagaimana Anda memperbaikinya? Sebagian besar folder ada di folder C: \ Users \ username saya. Saya memiliki 26 folder dengan titik sebelum nama di folder induk C: \ Users \ username saya, tetapi hanya 3 nama acara. Jika Anda membuka properti untuk setiap folder, Anda sebenarnya dapat menemukan namanya, tetapi tidak ditampilkan di File Explorer. Terima kasih sebelumnya atas balasan apa pun.

Sunting: Ketika saya pergi ke Properties, itu akan mencantumkan jenis file sebagai (misalnya, .android) "Folder (.android)" bukan hanya "Folder", seperti folder lainnya. Jadi menurut saya jenis folder itulah yang menjadi masalah.

Saya menemukan bahwa solusi paling jelas dan sederhana sebenarnya berfungsi di windows 10:

Buka regedit.exe dan buka:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced

tambahkan DWORD baru bernama ShowFolderExt dan setel nilainya menjadi 1.

Windows File Explorer memiliki sejarah panjang dalam berbohong kepada Anda.
Itu tidak menunjukkan kepada Anda apa yang sedang terjadi. Ini hanya menunjukkan kepada Anda apa yang menurutnya harus Anda ketahui. Yang bagus untuk pengguna rata-rata yang hanya memiliki gagasan samar apa file atau folder sebenarnya, tapi ini membodohi cukup menjengkelkan untuk powerusers dan sys-admin yang mencoba memecahkan masalah.

Dan tampaknya versi Windows 10 lagi-lagi sedikit lebih buruk dari versi sebelumnya.
Folder masih ada: Explorer tidak menampilkannya, karena Explorer berpendapat bahwa folder tidak boleh memiliki ekstensi.
(Ini adalah gagasan yang kembali ke MS-DOS: Dalam folder sistem file FAT lama secara teknis TIDAK dapat memiliki ekstensi sama sekali. Sistem file modern jauh lebih fleksibel, tetapi Windows masih memiliki pemikiran yang tersisa sejak masa itu. Jika Anda tidak percaya saya: Buat folder yang diakhiri dengan satu titik. Lalu coba hapus.)

Satu-satunya hal yang dapat Anda lakukan adalah menggunakan utilitas pengelola file alternatif (tidak dibuat oleh Microsoft) yang menunjukkan kenyataan kepada Anda.
Dan alternatif tersebut biasanya jauh lebih mampu daripada Explorer.

Secara pribadi: Saya belum pernah menggunakan Explorer untuk sesuatu yang serius sejak Windows 3.1. Saya selalu menggunakan pengelola file alternatif.

Di Windows 10, ekstensi file tidak ditampilkan secara default di file explorer.

Ini harus dilakukan: Klik kanan pada ikon Windows> Panel Kontrol> Tampilan dan Personalisasi> Opsi File Explorer> Lihat Tab> Hapus centang "Sembunyikan ekstensi untuk jenis file yang dikenal"

Mengirim email dari alamat nama domain

hai, saya baru saja membeli nama domain dan ingin mengirim email dari blah@mydomain.com. Saya dapat menggunakan alias dari akun hotmail / gmail saya, tetapi penerima terkadang melihatnya sebagai "my@hotmail.com atas nama blah@mydomain.com". Adakah cara untuk hanya menampilkan "blah@mydomain.com"? Ada server SMTP gratis yang dapat saya atur / gunakan? Terima kasih

Karena spam, sebagian besar (jika tidak semua) ISP dan penyedia email tidak mengizinkan Anda untuk memalsukan email menggunakan server SMTP mereka.

Anda harus mendaftar untuk paket email yang dihosting yang akan memberi Anda server SMTP untuk digunakan dengan domain Anda.

Registrar domain Anda mungkin menawarkan paket seperti itu.

Jika tidak ada banyak penyedia hosting yang menawarkan paket hanya email.

Satu opsi tidak gratis adalah mendaftar ke Google Workspace (sebelumnya Google G Suite, sebelumnya Google Apps) yang memberi Anda akun Gmail lengkap (juga Google Dokumen, Google Kalender, dan lainnya) tetapi menggunakan domain Anda, bukan @ gmail.com

apa yang dapat saya lakukan jika aplikasi tidak mengenali pengguliran horizontal?

Saya memiliki mouse dengan roda gulir horizontal (MX Master oleh Logitech), tetapi pengguliran horizontal hanya dikenali oleh beberapa aplikasi (misalnya Excel dan Chrome).

Aplikasi lain (seperti SQL Server Management Studio, alias SSMS) tampaknya tidak mengenali gulungan horizontal. Dalam contoh ini SSMS memiliki panel "hasil" untuk hasil kueri, yang memiliki bilah gulir horizontal, tetapi roda gulir horizontal tidak memindahkannya.

Apakah ini sesuatu yang harus didukung oleh programmer aplikasi? Atau mungkinkah ada beberapa peretasan yang memungkinkan saya membangun dukungan sendiri? Saya menjalankan Windows 7, kebetulan.

Saya juga menggunakan MX Master dan mengalami masalah ini.

Perbaiki

Buka software Logitech Options Anda dan dari panel "Mouse", klik lingkaran putih di atas Thumb Wheel Scroll. Ini akan memunculkan opsi perilaku. Entah mengapa, mengubah perilaku menjadi "Zoom In / Out" tampaknya memperbaiki masalah pengguliran horizontal ini (setidaknya, saya dapat mengonfirmasi ini untuk SSMS.)

Coba ini untuk aplikasi lain - jika pengguliran horizontal Anda terputus, di tempat lain, coba gunakan pengaturan Khusus Aplikasi untuk hanya mengubah perilaku dalam aplikasi yang Anda inginkan.

Selamat Menggulir!

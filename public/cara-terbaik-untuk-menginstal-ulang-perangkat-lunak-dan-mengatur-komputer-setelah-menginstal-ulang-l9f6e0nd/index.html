Cara terbaik untuk menginstal ulang perangkat lunak dan mengatur komputer setelah menginstal ulang

Saya selalu bertanya-tanya apakah ada cara yang baik untuk memberikan pengalaman penginstalan ulang yang cepat.

Setiap kali saya menyiapkan komputer baru atau menginstal ulang komputer yang sudah ada, saya menggunakan satu hari atau lebih untuk mendapatkan semua perangkat lunak dan pengaturan pengaturan, jadi lingkungannya seperti yang biasa saya lakukan.

Itu termasuk menginstal alat Visual Studio, Office, DropBox, SysInternals dan sebagainya.

Apa solusi di luar sana untuk mengotomatiskan proses atau hanya membuatnya sedikit lebih mudah? Kadang-kadang sulit untuk menemukan semua utilitas lagi dan membaca banyak posting blog lama tentang prosedur instalasi yang aneh lagi.

Saat ini saya menjalankan Windows 7 dan OS X.

Untuk menyalin instalasi OS X, file dan semuanya, Anda cukup melakukan backup dengan Time Machine, reboot dari DVD instalasi menggunakan opsi "Restore System from Backup", dan kemudian tunggu. Proses ini sepenuhnya otomatis.

Jika Anda ingin menginstal ulang hanya Sistem, tanpa menyentuh data Pengguna, gunakan Time Machine lagi: pulihkan semuanya dari tertua salinan disk Home yang Anda miliki, yang memiliki semua utilitas yang Anda inginkan. Setelah selesai, gunakan Time Machine untuk memulihkan file terbaru Pencadangan folder pengguna.

(Saya tidak begitu yakin mengapa Anda hanya ingin menginstal ulang Sistem. Seperti kebanyakan sistem UNIX, OS X tidak benar-benar menurun dari waktu ke waktu. Saya rasa ini mungkin berguna jika Anda benar-benar tidak beruntung dan terkena serangan MalWare.)

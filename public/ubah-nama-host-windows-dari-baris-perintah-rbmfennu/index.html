Ubah nama host windows dari baris perintah

Pertanyaan ini terkait dengan https://stackoverflow.com/questions/54989/change-windows-hostname-from-command-line , tapi saya dalam situasi yang berbeda. Alat netdom tidak akan berfungsi di komputer yang tidak tergabung dengan domain. Saya sudah mencoba mengatur nama host dari registri, tetapi saya bertanya-tanya apakah ada cara lain dari baris perintah.

9

Langsung dari tautan yang Anda berikan:

wmic computersystem where name="OLDNAME" call rename name="NEWNAME"

Kemudian reboot dengan

shutdown /r /t 0

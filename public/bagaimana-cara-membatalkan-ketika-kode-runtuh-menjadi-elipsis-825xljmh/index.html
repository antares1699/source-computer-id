Bagaimana cara membatalkan ketika kode runtuh menjadi elipsis?

Saya tidak sengaja menekan kombinasi tombol yang menciutkan kode saya menjadi simbol elipsis kuning.

Adakah yang bisa memberi tahu saya caranya membuka saya t? Tidak ada tombol atau simbol di sisi kiri yang bisa saya klik. Mengklik, mengklik dua kali, mengklik kanan pada elipsis tampaknya juga tidak membantu.

Runtuhnya blok kode seperti itu disebut sebagai "Code Folding".

Pintasan untuk melipat:

  • OSX: Perintah + Pilihan + ]
  • Windows: Ctrl + Bergeser + ]

Berlangsung:

  • OSX: Perintah + Pilihan + [
  • Windows: Ctrl + Bergeser + [

Jika Anda tidak memiliki teks yang dipilih, itu akan melipat / membuka blok kode tempat kursor berada.

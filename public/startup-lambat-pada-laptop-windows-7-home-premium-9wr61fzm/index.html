Hal pertama. Karena Laptop Anda adalah sistem Inti ganda, pastikan Anda menggunakan kedua inti tersebut selama boot sistem. Untuk melakukan ini:
1. Ketik "msconfig" (tanpa tanda kutip) di Vista atau Windows 7 mulai kotak pencarian menu dan tekan enter.
2. Klik Continue jika Anda mendapatkan perintah UAC. Secara default, Anda tidak akan mendapatkan perintah UAC di Windows 7.
3. Arahkan ke tab "Boot" dan klik tombol "Opsi lanjutan".
4. Centang kotak bernama "Jumlah prosesor" dan juga pilih jumlah prosesor dari kotak drop-down.

Anda juga bisa mencoba ini: Mengaktifkan fitur pengambilan Boot-Only.
Modifikasi Registry Peringatan di depan

Klik Mulai.
Buka kotak teks Cari program dan file, ketik regedit.
Klik Regedit.exe
Jika jendela UAC muncul, klik Yes.
Di registri, temukan subkunci ini: HLKM \ SYSTEM \ CurrentControlSet \ Control \ SessionManager \ MemoryManagement \ PrefetchParameters.
Di panel kanan, klik dua kali EnablePrefetcher.
Ubah nilainya menjadi 2 dan klik OK.
Di panel kanan, klik dua kali EnableSuperfetch.
Ubah nilainya menjadi 2 dan klik OK.
Keluar dari registri dan mulai ulang komputer Anda agar perubahan diterapkan.
Saat Anda menghidupkan komputer lagi, prefetcher akan mencadangkan lebih banyak sumber daya untuk proses boot daripada booting dan peluncuran aplikasi.

Ketiga, pastikan Anda tidak memuat banyak program lain saat boot. Anda dapat memeriksa saat Anda berada di MSConfig.exe program lain apa yang sedang dimuat saat boot. Seperti yang ditunjukkan pada gambar di sini, saya telah menonaktifkan sebagian besar milik saya.

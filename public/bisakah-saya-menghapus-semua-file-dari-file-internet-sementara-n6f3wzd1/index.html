Bisakah saya menghapus semua file dari File Internet Sementara?

Saya telah membuka folder berikut di Windows 7 Professional:

C:\Users\Igor\AppData\Local\Microsoft\Windows

Folder ini berisi Temporary Internet Files, yaitu 2GB.

Saya tidak menggunakan Internet Explorer; bagaimana cara menghapus semua file dari direktori ini dan apakah aman untuk dilakukan?

Anda bisa menggunakan CCleaner ( http://www.piriform.com/CCLEANER ). Jika Anda hanya ingin menghapus file IE, hapus centang semua kotak di tab Windows dan Aplikasi, di samping file sementara IE, kemudian analisis dan bersihkan.

Ya, Anda dapat secara manual menghapus semua yang ada di folder itu, saya telah melakukannya sebagai pemeliharaan rutin selama bertahun-tahun, beberapa file atau folder mungkin terkunci jika ada program yang terbuka dan menggunakannya, lewati saja selama proses penghapusan.

Saya menyimpan pintasan ke folder itu di desktop saya atau disematkan ke jumplist saya.

Ini lebih cepat dan lebih lengkap daripada menggunakan perangkat lunak untuk melakukannya.

Beberapa folder dan file di sana akan direplikasi oleh Windows atau program lain segera setelah penghapusan atau reboot.

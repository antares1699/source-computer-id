Konversi MBR ke GPT - Tanpa OS apa pun

Saya baru saja mendapat laptop baru tanpa OS (ada FreeDOS, tapi sekarang tidak lagi). Ketika saya pergi ke Penginstal Windows dan mencoba membuat partisi baru dari ruang yang tidak dialokasikan, saya tidak mendapatkan pesan Kesalahan apa pun tetapi penginstal tidak dapat membuat partisi. Di bagian bawah jendela penginstal, ada peringatan yang mengatakan,

Windows tidak dapat menginstal pada partisi MBR *. Pada sistem EFI, Windows hanya dapat menginstal pada partisi GPT *

Bagaimana cara mengubah partisi MBR ke GPT * tanpa OS apa pun?

* Saya tidak yakin apakah itu partisi atau bukan

Ini mudah, ketika Anda membaca panduan instalasi, Anda dapat setiap saat menekan Shift + F10 - prompt perintah muncul.

Kemudian tulis Diskpart, lalu List Disk. Anda melihat di sana daftar disk Anda, lalu pilih salah satu yang ingin Anda ubah.

Select disk 0 (jika Anda ingin menginstal pada disk pertama, misalnya). Bersihkan dengan Clean, dan ubah menjadi GPT Convert GPT. Selamat menikmati

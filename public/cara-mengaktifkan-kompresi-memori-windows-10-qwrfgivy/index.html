Cara mengaktifkan kompresi memori Windows 10

Baru-baru ini saya menjalankan program yang benar-benar ingin menghabiskan memori sistem saya. Sering kali saya bahkan tidak punya satu gigabyte lagi. Hal ini menyebabkan sistem saya macet dan akhirnya sering macet. Saya melihat di task manager ada variabel yang memberi tahu saya memori terkompresi saya. Saya belum mengaktifkannya, tetapi dari suaranya, itu bisa menyelesaikan masalah kurangnya memori saya sampai saya membeli lebih banyak. Bagaimana cara mengaktifkan kompresi memori Windows 10?

Catatan: Ya, saya telah melihat ke tempat lain sebelum memposting, yang mengejutkan saya tidak menemukan banyak, hanya hal-hal yang saya temukan untuk menonaktifkannya karena alasan yang aneh.

Bagaimana cara mengaktifkan kompresi memori Windows 10?

Kompresi memori diaktifkan secara default. Namun, itu muncul karena alasan apa pun, saat ini Anda telah menonaktifkannya. Dalam prompt perintah Administrator PowerShell ketik perintah berikut untuk menonaktifkan kompresi memori

Disable-MMAgent -MemoryCompression

Dalam prompt perintah Administrator PowerShell ketik perintah berikut untuk mengaktifkan kompresi memori

Enable-MMAgent -MemoryCompression

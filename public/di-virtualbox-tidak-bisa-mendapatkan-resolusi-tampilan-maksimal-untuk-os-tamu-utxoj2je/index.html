Di VirtualBox tidak bisa mendapatkan resolusi tampilan maksimal untuk OS tamu

Saya menggunakan VirtualBox dengan Windows 7 sebagai OS tamu. Namun pengaturan resolusi maksimal yang bisa saya dapatkan di panel kontrol hanya 1264x602. Bagaimana cara mendapatkan resolusi full HD yang saya miliki di host Windows 7?

tim — Sumber
  • Tidak mengetahui pengaturan Anda (memori host, kemampuan grafis, dll.), Saya tidak dapat mengatakan dengan pasti, tetapi saya yakin Anda mungkin belum mengalokasikan cukup memori ke kartu video VM. Anda dapat menambahkan hingga 128mb, meskipun menurut saya 32mb sudah cukup. Jika berhasil, beri tahu saya, dan saya akan menambahkannya sebagai jawaban.

    — Kerri Shotts

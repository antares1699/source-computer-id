Apa kerugian dari mengaktifkan virtualisasi?

Saya sedang mengembangkan aplikasi untuk Android. Emulator merekomendasikan untuk mengaktifkan virtualisasi.

Tapi saya tidak yakin apakah itu bagus untuk komputer saya dalam jangka panjang? Juga, pc saya merekomendasikan saya untuk menonaktifkan virtualisasi.

Praktis, ini hanya mengaktifkan bagian 'khusus' dari CPU Anda khususnya dirancang untuk mempercepat virtualisasi. Pada dasarnya ini memungkinkan VM untuk mengakses instruksi spesifik dan berbicara ke memori sedikit lebih langsung dan menghasilkan kinerja yang meningkat secara signifikan.

Anda tidak membutuhkannya jika Anda tidak menjalankan VM (jadi banyak pengguna tidak membutuhkannya). Ada beberapa teoretis eksploitasi seperti pil biru tapi tidak ada di alam liar.

Banyak sistem datang dengan ekstensi virtualisasi yang diaktifkan secara default dan tidak ada masalah dengan mereka.

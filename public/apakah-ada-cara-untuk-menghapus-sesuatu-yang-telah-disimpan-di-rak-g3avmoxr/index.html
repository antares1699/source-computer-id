Apakah ada cara untuk menghapus sesuatu yang telah disimpan di rak?

TortoiseSVN memiliki opsi untuk menyimpan / menyembunyikan perubahan. Saya tidak dapat menemukan cara untuk menghapus sesuatu yang disimpan di rak. Solusi yang saya gunakan tidak terbatas pada copy pekerjaan bersih dan kemudian mengembalikannya.

Ini mungkin bukan jawaban, tapi saya pikir Anda harus menunggu TortoiseSVN untuk mencapai SVN versi 1.11, dan hingga saat itu untuk terus menggunakan solusi Anda.

Baru di versi 1.11 adalah perintah x-shelf-drop (rak-lepas) untuk menghapus rak, seperti yang tercantum dalam artikel Apache Rak di Svn-trunk .

Arus Versi TortoiseSVN dideskripsikan sebagai:

Versi 1.10.1 saat ini ditautkan ke pustaka Subversion 1.10.2.

TortoiseSVN's Unshelve tidak menghapus (menjatuhkan) rak untuk saya, terlepas dari apakah konflik hasil unshelving atau tidak. (Saya menggunakan TortoiseSVN 1.11.1, Build 28492)

Solusi: hapus / jatuhkan rak menggunakan SVN secara langsung, bukan melalui TortoiseSVN. Di konsol:

$ svn help x-shelf-drop
x-shelf-drop: Delete a shelf.
usage: x-shelf-drop SHELF [PATH ...]

  Delete the shelves named SHELF from the working copies containing PATH
  (default PATH is '.')

…

$ svn x-shelf-drop "Name of shelf"
deleted 'Name of shelf'

Dokumen: https://cwiki.apache.org/confluence/display/SVN/Shelving+in+Svn-trunk

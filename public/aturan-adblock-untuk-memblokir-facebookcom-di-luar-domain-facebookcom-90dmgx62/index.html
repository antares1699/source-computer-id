Aturan AdBlock untuk memblokir facebook.com di luar domain facebook.com

dari: http://news.ycombinator.com/item?id=3035418

Jadi saya menulis beberapa aturan filter di file filter saya, kemudian saya mengunggahnya ke domain saya, terlihat seperti ini:

[Adblock Plus 1.1]
||facebook.com^$domain=~facebook.com|~facebook.net|~fbcdn.com|~fbcdn.net
||facebook.net^$domain=~facebook.com|~facebook.net|~fbcdn.com|~fbcdn.net
||fbcdn.com^$domain=~facebook.com|~facebook.net|~fbcdn.com|~fbcdn.net
||fbcdn.net^$domain=~facebook.com|~facebook.net|~fbcdn.com|~fbcdn.net

dan saya menambahkannya ke Adblock Plus sebagai filter. Saya dapat melihat aturan ini di opsi filter, dll.

Tetapi tidak berhasil !: [maaf telah memberikan url situs contoh]:

http://9gag.com/gag/308206

Jadi di bawah gambar idiot ada bagian "komentar facebook" - dan tidak diblokir dengan aturan ini.

Bagaimana cara mengkonfigurasi CTRL + H sebagai backspace di bawah Google Chrome?

Saya tidak dapat membuat CTRL + H melakukan apa pun selain "Buka riwayat" di bawah Google Chrome. Saya mencoba plugin Chrome yang memungkinkan untuk mendefinisikan kembali pintasan keyboard tetapi saya hanya dapat menambahkan "tindakan" atau apa pun artinya ke pintasan keyboard dan tampaknya itu dilakukan dengan menggunakan beberapa peretasan JavaScript raksasa.

Inilah pluginnya:

https://chrome.google.com/webstore/detail/mgjjeipcdnnjhgodgjpfkffcejoljijf

Bagaimanapun, saya tidak dapat membuat CTRL + H berfungsi sebagai tombol spasi mundur di bawah Chrome ( yaitu hapus kunci di kiri kursor). Saya khususnya ingin itu bekerja di bilah alamat / bilah penelusuran (yang merupakan "bilah" yang sama di bawah Chrome) dan juga dalam bentuk edit teks.

Jadi bagaimana cara membuat CTRL + H berfungsi sebagai tombol spasi mundur di bawah Google Chrome? (Debian GNU / Linux Wheezy di sini).

Perhatikan bahwa jika jawabannya adalah menggunakan plugin untuk Chrome yang memungkinkan untuk mengubah pintasan, maka saya memerlukan metode yang tepat, langkah demi langkah, tentang cara melakukannya karena saya sudah mencobanya dan tidak dapat membuatnya berfungsi.

Saya tahu ini adalah pertanyaan lama, tetapi saya menemukan ini ketika mencoba melakukan hal yang sama dengan Anda, jadi saya akan memposting ini.

Saya menggunakan Chromium dan saya menggunakan Ubuntu 12.04 (sekarang saya menggunakan Gentoo), tetapi itu tidak masalah.

Pelaksana:

$ gsettings set org.gnome.desktop.interface gtk-key-theme "Emacs"

akan mengubah pengikatan kunci ke pengikatan Emacs. Jika Anda baik-baik saja dengan itu, maka ini saja. Namun, jika Anda ingin menggunakan Ctrl-a sebagai "pilih semua", ada sedikit pekerjaan yang harus dilakukan.

Lihat /usr/share/themes/Default/gtk-2.0-key/gtkrc dan /usr/share/themes/Emacs/gtk-2.0-key/gtkrc. Sintaksnya seharusnya tidak terlalu sulit untuk dipahami. Salin semua pengikatan kunci yang ingin Anda gunakan ke /usr/share/themes/Default/gtk-2.0-key/gtkrc. Jangan lupa untuk menambahkan

class "GtkEntry" binding "foo"
class "GtkTextView" binding "foo"
class "GtkTextView" binding "bar"

bagian, yang dapat ditemukan di akhir file.

GNOME

Pengaturan harus tercermin dengan beralih ke tema Emacs dan beralih kembali ke tema Default.

$ gsettings set org.gnome.desktop.interface gtk-key-theme "Emacs"
$ gsettings set org.gnome.desktop.interface gtk-key-theme "Default"

Non-GNOME

Buat dan tambahkan berikut ini ke ~/.gtkrc-2.0

gtk-key-theme-name = "Default"

Mulai ulang Chromium dan pengaturan akan terlihat.

Hal terbaik tentang ini adalah meskipun Anda menyetel Ctrl-w untuk menghapus satu kata, Anda masih dapat menutup tab saat Anda tidak sedang memasukkan.

P.S. Saya juga menggunakan keyboard hacking bahagia dan saya senang menemukan seseorang yang menggunakannya juga :)

Ubuntu 18.04 (GTK3)

Edit: 11/01/2018

Setelah hampir 5 tahun, saya memperbarui posting ini untuk Ubuntu 18.04. Saat ini saya menggunakan Firefox, tetapi berikut ini harus independen dari browser apa yang Anda gunakan.

Di GTK3, metode ~/.gtkrc-2.0 tidak lagi berfungsi. Sebagai gantinya, edit ~/.config/gtk-3.0/settings.ini dan tambahkan baris berikut di akhir:

gtk-key-theme-name=Emacs

Jika Anda ingin mengubah binding, binding Emacs dapat ditemukan di /usr/share/themes/Emacs/gtk-3.0/gtk-keys.css, yang dapat disalin ke ~/.config/gtk-3.0/gtk.css dan dimodifikasi.

Beberapa referensi:

  • https://mail.gnome.org/archives/gtk-list/2011-March/msg00108.html
  • https://developer.gnome.org/gtk3/3.2/GtkSettings.html#GtkSettings--gtk-key-theme-name

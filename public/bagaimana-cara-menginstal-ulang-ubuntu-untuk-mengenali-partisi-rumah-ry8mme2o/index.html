Bagaimana cara menginstal ulang Ubuntu untuk mengenali / partisi rumah?

Saya telah menjalankan Ubuntu 9.04 sejak dirilis dan pada petualangan pemasangan terakhir saya, saya mengambil risiko dan membuat partisi terpisah untuk saya. /rumah direktori. Untuk alasan yang tidak penting di sini saya perlu menginstal ulang Ubuntu. Jadi saya pergi melalui gerakan dan memilih partisi root untuk dihapus dan digunakan sebagai titik instalasi dan tidak berpikir saya perlu melakukan hal lain saya melanjutkan instalasi.

Setelah bangun dan menjalankan saya menginstal dan menjalankan gparted dan melihat bahwa partisi saya yang telah saya atur sebelumnya /rumah tidak dipasang, dan tidak memiliki titik pemasangan, jadi saya rasa saya baru saja menyiapkan root agar memilikinya /rumah termasuk.

Dapatkah saya memperbaikinya tanpa menginstal ulang?

Saya rasa Anda tahu nama partisi Anda dari gparted. (mari gunakan / dev / sda3 untuk contoh ini)

Yang perlu Anda lakukan hanyalah mengedit fstab Anda dan menambahkan dan memasukkan partisi ini agar dapat dipasang sebagai / home: (buat cadangan fstab terlebih dahulu)

$ sudo cp /etc/fstab /etc/fstab.bak
$ sudo gedit /etc fstab

Tambahkan garis:

/dev/sda3 /home ext3 defaults 0 2

Sebuah reboot harus membawa drive ini ke mount / home seperti yang Anda cari. Jika UID pengguna Anda pada penginstalan asli dan penginstalan baru berbeda, Anda mungkin mengalami masalah ... tapi itu untuk pertanyaan lain.

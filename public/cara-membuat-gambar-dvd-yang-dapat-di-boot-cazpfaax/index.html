Ini sangat sederhana:

  1. Unduh, instal, dan jalankan PowerISO
  2. Klik Tools lalu Make CD/DVD/BD Image File...
  3. Centang baik .iso atau format yang Anda inginkan untuk membuat gambar
  4. Pilih folder tujuan
  5. Beri nama file dan klik Save

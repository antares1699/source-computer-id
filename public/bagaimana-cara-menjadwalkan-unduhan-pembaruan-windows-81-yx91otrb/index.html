Lalu lintas internet saya dibatasi oleh ISP dan gratis hanya selama periode tertentu sepanjang hari (2: 00-07: 00). … Bagaimana saya bisa menjadwalkan "pembaruan windows" untuk "windows 8.1 pro" untuk memulai / berhenti pada waktu yang ditentukan?

Secara default, Windows akan memeriksa pembaruan pada jam 3 pagi yang sesuai situasi Anda , jadi Anda harus membiarkannya disetel untuk memeriksa pembaruan secara otomatis daripada melakukannya secara manual. Saya t Sebaiknya membangunkan sistem pada jam 3 pagi untuk melakukan pemeriksaan , tetapi jika tidak melakukannya bahkan ketika disetel ke pembaruan otomatis, maka itu akan melakukannya saat Anda boot lagi, yang mungkin sudah lewat jam 7 pagi. Untuk membiarkan Windows membangunkan sistem Anda di malam hari untuk memeriksa pembaruan:

  1. Pergi ke Pusat aksi
  2. Cari Perawatan Otomatis bagian
  3. Beri tanda centang di kotak centang Allow scheduled maintenance to wake up my computer at the scheduled time
  4. Secara opsional, atur waktu yang tepat (ke jam) kapan Windows harus bangun dan memeriksanya

pada saat itu (02.00 - 07.00), PC sedang mengunduh video & file lainnya. Jadi bagaimana bisa membiarkannya menjalankan pemeliharaan jadwal pada saat itu karena ini dapat mempengaruhi pengunduhan dan proses lainnya. Saya ingin menjalankan hanya "pembaruan windows" bukan pekerjaan pemeliharaan lain pada saat itu.

Anda ingin mengunduh pembaruan Windows hanya selama periode bandwidth bebas antara 2-7 pagi, tetapi Anda tidak ingin unduhan pembaruan Windows mengganggu pengunduhan P2P selama periode yang sama? Saya tidak yakin apa pertanyaan Anda lagi, tetapi jika Anda bertanya tentang mencegah sistem melakukan boot ulang setelah pembaruan, Anda dapat melakukannya. Bagaimana tergantung pada edisi Windows yang Anda gunakan.

  • Windows Professional dan yang lebih baru:

    1. Jalankan gpedit.msc
    2. Navigasi ke Konfigurasi Komputer → Template administratif → Komponen Windows → Pembaruan Windows
    3. Setel No auto-restart with logged on users for scheduled automatic updates installations hingga Enalbed

  • Windows Home dan bawah:

    1. Jalankan regedit
    2. Arahkan ke HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows\WindowsUpdate\AU
    3. Pilih Edit → Baru → DWORD
    4. Masukkan NoAutoRebootWithLoggedOnUsers untuk nama nilai
    5. Klik dua kali dan masukkan 1 untuk nilainya

Salah satu metode adalah tugas terjadwal.

Update.bat

wuauclt.exe /detectnow

Contoh ini berjalan pada jam 2:00 setiap hari.

SchTasks /Create /SC DAILY /TN "Late Night Update" /TR “C:\Update.bat” /ST 02:00

Anda mungkin memerlukan skrip saudara untuk membatalkan berbagai hal pada pukul 6:45 pagi agar aman.

Killer.bat

taskkill.exe /f /im wuauclt.exe

dan tugas yang dijadwalkan.

SchTasks /Create /SC DAILY /TN "Bat Killer" /TR “C:\Killer.bat” /ST 06:45

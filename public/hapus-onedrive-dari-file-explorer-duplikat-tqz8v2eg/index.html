Hapus OneDrive dari File Explorer [duplikat]

Saya ingin menyingkirkan opsi OneDrive dari panel navigasi File Explorer. Artikel era pratinjau yang saya temukan merekomendasikan tweak registri, yang agak tidak elegan, dan saya berharap ada cara yang lebih baik untuk menghapusnya.

gpedit.msc -> Kebijakan Komputer Lokal \ Konfigurasi Komputer \ Template Administratif \ Komponen Windows \ OneDrive

Aktifkan "Cegah penggunaan OneDrive untuk penyimpanan file". Ini akan menonaktifkan dan menghapus semuanya setelah restart

Jika Anda ingin menghapus OneDrive secara permanen , lakukan hal berikut:

Buat file teks dengan membuka Notepad

Pergilah link berikut dan salin / tempel kode ke dalam file teks:

@echo off
cls

set x86="%SYSTEMROOT%\System32\OneDriveSetup.exe"
set x64="%SYSTEMROOT%\SysWOW64\OneDriveSetup.exe"

echo Closing OneDrive process.
echo.
taskkill /f /im OneDrive.exe > NUL 2>&1
ping 127.0.0.1 -n 5 > NUL 2>&1

echo Uninstalling OneDrive.
echo.
if exist %x64% (
%x64% /uninstall
) else (
%x86% /uninstall
)
ping 127.0.0.1 -n 5 > NUL 2>&1

echo Removing OneDrive leftovers.
echo.
rd "%USERPROFILE%\OneDrive" /Q /S > NUL 2>&1
rd "C:\OneDriveTemp" /Q /S > NUL 2>&1
rd "%LOCALAPPDATA%\Microsoft\OneDrive" /Q /S > NUL 2>&1
rd "%PROGRAMDATA%\Microsoft OneDrive" /Q /S > NUL 2>&1 

echo Removeing OneDrive from the Explorer Side Panel.
echo.
REG DELETE "HKEY_CLASSES_ROOT\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6}" /f > NUL 2>&1
REG DELETE "HKEY_CLASSES_ROOT\Wow6432Node\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6}" /f > NUL 2>&1

pause

Kemudian simpan sebagai RemoveOneDrive.bat dan keluar

Jalankan skrip dengan hak admin dan One Drive akan hilang

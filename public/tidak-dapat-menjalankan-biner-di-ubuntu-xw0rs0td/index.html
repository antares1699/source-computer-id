Tidak Dapat Menjalankan Biner di Ubuntu

Saya tidak percaya saya menanyakan ini ... jelas saya dalam kondisi langka hari ini.

Mencoba menyiapkan mesin Ubuntu baru dan baru saja mengunduh 'p4' (Perforce klien baris perintah). Ini adalah unduhan file tunggal - biner yang terhubung secara statis, jadi saya baru saja melakukan:

wget http://www.perforce.com/downloads/perforce/r09.2/bin.linux26x86/p4

... langsung ke / usr / bin. Cukup sederhana. Kecuali:

root@aj-ubuntu:/usr/bin# ll p4 
-rwxr-xr-x 1 root root 748808 2010-02-11 16:54 p4
root@aj-ubuntu:/usr/bin# ./p4 
-su: ./p4: No such file or directory
root@aj-ubuntu:/usr/bin# /usr/bin/p4 
-su: /usr/bin/p4: No such file or directory

Apa yang sebenarnya terjadi di sini ...?!

Terima kasih sebelumnya atas ejekan Anda :)

-aj

executable statis tidak bergantung pada pustaka .... itu membutuhkan kernel yang kompatibel, yaitu yang memiliki panggilan sistem yang sama seperti yang dibangun di atasnya.

p4 adalah klien perforce, dan klien Anda dibuat berdasarkan kernel v2.2.5 10 tahun lalu ...

itu akan bekerja dengan (lama tidak didukung) kemudian kernel 2.2,

ini dapat bekerja dengan (bulan sejak tidak didukung) kernel 2.4,

itu tidak akan bekerja dengan kernel 2.6 dan 3.x (masih didukung)

Mereka membuat versi untuk 2.6 klien http://www.perforce.com/downloads/complete_list tersedia untuk kebutuhan Anda.

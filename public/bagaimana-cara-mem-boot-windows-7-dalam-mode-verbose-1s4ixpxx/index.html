Bagaimana cara mem-boot Windows 7 dalam mode verbose?

Baru-baru ini penginstalan Win 7 saya membutuhkan waktu yang sangat lama untuk boot dan agar saya dapat mendeteksi masalah, saya perlu mem-boot Windows dalam mode verbose.
Sekarang saya membutuhkan windows untuk membuang semua yang dilakukannya dalam proses boot di layar monitor. Saya bertanya-tanya bagaimana ini bisa dilakukan?

Di atas How-To Geek ada postingan tentang cara melakukan ini.

Tekan Start, ketik msconfig, untuk masuk ke jendela System Configuration di bawah ini

Pilih tab Boot dan periksa OS boot information. Memeriksa log mungkin juga berguna untuk membaca setelah booting.

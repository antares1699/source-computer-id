Melihat CLEO dan FEBE .

FEBE ( Ekstensi Cadangan Lingkungan Firefox ) memungkinkan Anda mencadangkan ekstensi Firefox dengan cepat dan mudah. Faktanya, ini lebih dari sekadar mencadangkan - Ini benar-benar akan membangun kembali ekstensi Anda secara individual menjadi file .xpi yang dapat diinstal. Sekarang Anda dapat dengan mudah menyinkronkan browser kantor dan rumah Anda.

CLEO ( Penyelenggara Ekstensi Perpustakaan Kompak ) adalah ekstensi Firefox yang bekerja dengan FEBE untuk mengemas sejumlah ekstensi / tema ke dalam satu file .xpi yang dapat dipasang.

juga, OPIE ( Impor / Ekspor Preferensi Dipesan ) adalah ekstensi Firefox yang memungkinkan Anda untuk mengimpor dan mengekspor preferensi ekstensi yang Anda pasang. Ini berguna saat memasang ekstensi di profil baru, atau menyinkronkan beberapa pemasangan Firefox.

Saya juga membuat profil saya dicadangkan untuk pemulihan dan mulai menggunakannya di komputer baru.

Saat ini (2019) semua ekstensi / addons yang diinstal disimpan dalam format instalasi (.xpi) di folder profil. (... Profil \ xxxxxxx.default \ ekstensi)

Jadi, Anda dapat membuat zip semua xpi menjadi satu.

Omong-omong, metode ini umum baik untuk FF Quantum dan ekstensi web barunya (di atas ver. 57) dan Firefox yang menggunakan ekstensi XUL (di bawah ver. 57)

Perhatikan bahwa beberapa file dapat memiliki nama yang dikaburkan seperti {b9db16a4-6edc-47ec-a1f4-b86292ed211d} .xpi

Menggunakan about: support Anda dapat menemukan nama yang terkait dengan addon ini

AbbrevMode bekerja di Gnus.

Dari tautan itu (dengan beberapa pengeditan):

Mendefinisikan Singkatan

Ketik kata yang ingin Anda gunakan sebagai ekspansi, lalu ketik 'C-x a g' dan singkatan untuk itu. Contoh:

Ini adalah teks paragraf pembuka. ESC h Ctrl - u 0 Ctrl - x Sebuah g Buka Kembali

Saya: Itu nol setelah C-u. Bersama-sama mereka membuat definisi singkatan C-x a g berlaku untuk kawasan yang dipilih, yang merupakan paragraf, dalam hal ini, yang dipilih oleh M-h (Esc-h atau Alt-h). Definisi singkatan dapat dibuat untuk diterapkan hanya ke mode utama saat ini dan bukan secara global dengan menggunakan C-x a l (itu adalah "el" bukan "g").

Menyimpan Singkatan

Gunakan 'M-x tulis-singkatan-file' dan hanya tekan RET ketika dimintai nama file. Itu defaultnya adalah “~ / .abbrev_defs”. Dalam masa depan, Emacs harus menyimpan singkatannya secara otomatis saat Anda keluar dari Emacs.

Juga, jika Anda menginginkan singkatan disimpan setiap kali Anda menyimpan file (C-x s) atau keluar dari emacs, tambahkan berikut ini ke .emacs Anda:

(setq simpan-singkatan t)

Memulihkan Singkatan

Gunakan 'M-x baca-singkatan-file' untuk membaca singkatan, atau lebih baik lagi, tambahkan mengikuti ke file ~ / .emacs Anda:

(diam-baca-singkatan-file)

Ini akan membaca definisinya dari file default, “~ / .Abbrev_defs”.

Menggunakan Singkatan

‘Mode singkatan M-x’ mengubah mode Singkatan. Anda juga dapat memasukkan yang berikut ini ke dalam file ~ / .emacs jika Anda menginginkannya selalu di:

(singkatan-mode 1) Sebenarnya, Anda mungkin harus melakukan:

(setq default-singkatan-mode t) Saya tidak begitu yakin mengapa.

saya lagi: Setelah mode singkatan aktif, mengetik singkatan yang diikuti dengan spasi akan membuatnya segera diperluas. Namun, jika Anda tidak ingin hal itu terjadi, Anda dapat membiarkannya mati dan memperluasnya secara manual seperti ini:

Buka Ctrl - x Sebuah e

menjadi:

Ini adalah teks paragraf pembuka.

Gnus menggunakan Message-Mode untuk membuat pesan, jadi mungkin yang satu memiliki beberapa fungsi khusus untuk itu.

Saya akan menggunakan pengait khusus di message-setup-hook untuk itu (yang menanyakan apa yang ingin saya tanyakan dan membuat template). Perhatikan bahwa hook ini dipanggil untuk berita email dan usenet, baik pesan baru maupun balasan. Tetapi seharusnya mudah untuk membedakan kasus ini dari konten buffer.

Sederhananya, Anda dapat melakukan sesuatu seperti ini:

(setq my-F3-keymap (make-sparse-keymap))
(global-set-key [(f3)] my-F3-keymap)

(define-key my-F3-keymap [(n)]
  (function
   (lambda ()
     "insert now is..."
     (interactive)
     (insert "Now is the time for all good men to come to the aid of their country."))))

(define-key my-F3-keymap [(N)]
  (function
   (lambda ()
     "insert now is... with newlines"
     (interactive)
     (insert "Now is the time\nfor all good men\nto come to the aid\nof their country."))))

(define-key my-F3-keymap [(f)]
  (function
   (lambda ()
     "insert external file contents"
     (interactive)
     (insert-file "c:/temp/foo.html"))))

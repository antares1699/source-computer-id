Aplikasi Linux untuk menyesuaikan beberapa file log (seperti OS X Console.app)

Saya selalu mengikuti file log. Pada waktu tertentu, saya membuka setengah lusin jendela terminal yang hanya menjalankan tail -f.

Di OS X, aplikasi Konsol sangat bagus. Semua file log biasa ada di sana (sistem, mail, apache, dll) dan Anda dapat menambahkan log kustom Anda sendiri. Semuanya di satu tempat, di antarmuka yang dapat ditelusuri. Jika Anda reboot, jalankan saja Konsol dan Anda berada tepat di tempat Anda tinggalkan.

Apakah ada yang mirip dengan Console untuk Linux?

tail dapat mengikuti banyak file. Anda hanya perlu memasukkan beberapa nama file di baris perintahnya. Dan keluarannya diberi label sehingga Anda dapat dengan mudah membedakan keluaran dari file yang berbeda.

Sebagai contoh:

% tail -F /tmp/a /tmp/b

==> /tmp/a <==
Wed 31 Aug 2011 22:33:22 MDT

==> /tmp/b <==
Wed 31 Aug 2011 22:33:24 MDT
Wed 31 Aug 2011 22:33:34 MDT

==> /tmp/a <==
Wed 31 Aug 2011 22:33:38 MDT

==> /tmp/b <==
Wed 31 Aug 2011 22:33:40 MDT

==> /tmp/a <==
Wed 31 Aug 2011 22:33:43 MDT

Mengikuti , adalah tailer file log berbasis java open source. Ini memiliki antarmuka tab (satu tab per file), dan mengingat file yang Anda buka di antara sesi.

  • Pantau ("ikuti") file teks yang informasinya ditambahkan secara asinkron (mis. File log)
  • Lompat dengan cepat ke atas atau bawah file yang diikuti
  • Hapus konten area teks yang menampilkan konten file yang diikuti (tunggal atau sekaligus)
  • Hapus konten file yang diikuti (satu per satu atau sekaligus)
  • Konfigurasikan strategi buffering & latensi utas yang mengikuti file
  • Buka file menggunakan Drag-and-Drop
  • Pertahankan data sesi di seluruh permintaan Ikuti (yaitu, Ikuti mengingat file mana yang dibuka pengguna sehingga dapat membuka kembali file tersebut saat dijalankan lagi)
  • Jeda file log dan putar ulang saat dijeda
  • Sorot & cari teks dalam file log
  • Mulai ulang log saat mengikuti

Alternatif lain yang sangat bagus adalah glogg , dan ini yang paling dekat dengan Konsol OSX yang saya temukan sejauh ini.

Ini tersedia di Pusat Perangkat Lunak untuk Ubuntu 11.04+ atau Anda dapat menginstalnya melalui baris perintah untuk rilis yang lebih lama.

glogg adalah aplikasi GUI multi-platform untuk dijelajahi dan dicari file log yang panjang atau kompleks. Ini dirancang dengan programmer dan sistem administrator. glogg dapat dilihat sebagai grafis, interaktif kombinasi grep dan kurang.

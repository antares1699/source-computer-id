Tidak dapat mengubah ruang atas / bawah untuk header / footer di LibreOffice, dan menonaktifkan Header / Footer setelah mengekspor ke Office 2007

Saya telah menulis Tesis saya di LibreOffice. Semuanya baik-baik saja, tetapi Universitas saya harus memberikan Tesis dalam format yang sangat spesifik. Di LibreOffice saya tidak dapat mereplikasi ruang "Dari Atas / Dari Bawah" untuk header / footer. Akibatnya, templatenya salah dan saya harus memperbaikinya.

Jadi, saya menyimpan dokumen sebagai MS Word 2003 (dari LibreOffice) dan membuka * .doc dari Office 2007 Word saya. Di MS office saya tidak dapat mengubah batas halaman dan mengedit / mengubah Header / Footer. Secara khusus, opsi Edit Header / Edit Footer dinonaktifkan di Ribbon.

Bagaimana saya bisa mengatasinya?

Jika saya mengerti dengan benar, Anda perlu mengatur margin antara batas halaman dan bagian atas Header ke nilai tertentu. Anda dapat melakukannya dengan mudah di LibreOffice Writer, jadi Anda tidak perlu memanipulasi dokumen di MS Word.

Ada nilai spasi berbeda yang dapat Anda atur:

  • Di Halaman tab, Anda dapat mengatur Margin atas / bawah dari seluruh halaman . Jika Anda mengaktifkan Header / Footer, ini adalah margin antara "batas kertas" fisik dan tepi atas header) / tepi bawah footer. Header / Footer ditempatkan dalam margin yang ditentukan pada tab halaman:

  • Di Bagian atas dan bagian bawah tab, Anda dapat mengatur tinggi header / footer dan spasi antara header / footer dan konten utama . Secara tidak langsung, nilai-nilai ini menentukan margin konten halaman utama dengan batas halaman "fisik":

Jika ada margin tertentu yang ditentukan, pastikan untuk tidak memilih Dynamic Space / Auto fit height pada tab Header / Footer.

Jadi, "gambaran besar" terlihat seperti ini (kuning muda: konten utama; biru: header; hijau: footer):

1a, b, c dan d diatur pada Halaman tab; 2a/2b dan 3a/3b di Bagian atas dan bagian bawah tab masing-masing. Margin atas dari area "konten utama" adalah jumlah dari 1c + 2a + 2b; margin bawahnya, sebaliknya 1d + 3a + 3b.

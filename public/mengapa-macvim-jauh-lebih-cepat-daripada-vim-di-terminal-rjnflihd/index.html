Mengapa MacVim jauh lebih cepat daripada Vim di Terminal?

Saya telah menggunakan Vim dan MacVim sebagai alat pengembangan utama selama dua minggu terakhir dan telah memperhatikan bahwa kecepatan gulir kursor jauh lebih cepat dan lancar di MacVim daripada di Vim di Terminal. Ini perbedaan seperti siang dan malam. Kedua aplikasi menggunakan file .vimrc yang sama. Adakah yang tahu apa yang dapat menyebabkan perbedaan kinerja?

Vim dijalankan di shell Anda yang dijalankan di terminal Anda: Vim mewarisi banyak lapisan latensi, sebagian besar terkait dengan menggambar ulang jendela dan penanganan input.

MacVim tidak dijalankan di terminal dan mesin renderingnya jauh lebih cepat. Ini tidak terlalu bagus, meskipun, ketika Anda terbiasa dengan GVim di Linux.

Hubungkan ke FTP klien dengan alamat IP klien dinamis

Saya ingin terhubung ke FTP klien saya. Dia harus memasukkan IP saya ke daftar putih setiap saat karena IP saya dinamis. Bagaimana cara saya terhubung ke servernya?

4

4
4

Saya akan merekomendasikan Anda untuk menyewa virtual private server (VPS) dengan IP statis dan menggunakannya sebagai file mesin lompat untuk mengakses server FTP klien Anda dari alamat tetap.

Sekarang ada banyak terminologi tertentu yang mungkin digunakan setiap vendor, tetapi jika diperbolehkan menggunakan Amazon Web Services, maka Anda mungkin mencari EC2 instance nano dikonfigurasi dengan file IP elastis .

Dia mungkin menggunakan Port Knocking. Dengan cara ini, tidak peduli IP apa yang Anda miliki, Anda akan "mengetuk" ke urutan port dan daemon Knock akan mendeteksi urutan tersebut dan oleh karena itu menambahkan aturan sementara untuk memungkinkan Anda terhubung.

Dalam lingkungan berbasis Linux Anda dapat menggunakan knockd dalam hal ini, meskipun saya yakin bahwa untuk lingkungan OS lain ada daemon yang setara.

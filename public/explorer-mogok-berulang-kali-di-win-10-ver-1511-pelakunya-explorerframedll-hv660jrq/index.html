Baru-baru ini (dalam 3-4 hari atau lebih) penjelajah saya mengalami crash secara teratur tetapi tidak konsisten (yaitu upaya terus-menerus untuk melihat folder tertentu pada akhirnya akan berfungsi.) Saya memeriksa log Windows Event dan menemukan pesan kesalahan umum ini:

Mesin menjalankan Windows 10 x64, sering diperbarui, tidak ada perangkat lunak bajakan dan telah berfungsi normal hingga masalah yang dijelaskan di atas mulai muncul.

Apa yang harus saya lakukan sekarang?

Sunting :
(1) Kemungkinan pemicu :

  • pembaruan driver GPU (Radeon Software Version 15.11)
  • beberapa upaya tidak sabar untuk mengakses folder Homegroup bersama: ketika saya melihat Explorer kesulitan, saya terus melakukan spamming, dan akhirnya memulai ulang komputer

(2) Saya pikir saya menemukan perbaikan tetapi ternyata tidak! Saya menjalankan sfc /scannow di Command Prompt (dengan hak istimewa Administrator). Segalanya kembali normal untuk sementara waktu tetapi baru saja Explorer macet lagi. Log Peristiwa Baru:

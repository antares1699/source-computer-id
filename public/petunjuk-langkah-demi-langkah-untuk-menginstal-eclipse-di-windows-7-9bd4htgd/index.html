petunjuk langkah demi langkah untuk menginstal eclipse di Windows 7?

Hampir tidak dapat dipercaya bahwa saya tidak dapat menemukan ini (terutama di mana pun di situs gerhana), tetapi bagaimana cara menginstal Eclipse? Saya menggunakan Windows 7.

Saya telah mengunduh eclipse yang merupakan file zip besar. Setelah di-unzip, tidak ada file install, hanya exe untuk menjalankan eclipse (itu kejutan pertama). Menjalankan itu, ia mengeluh bahwa tidak ada JRE (saya pikir itu akan datang bersamanya). Jadi saya pergi dan mengunduh JDK. Itu masih tidak dapat menemukan apa yang dibutuhkan jadi saya harus secara manual memodifikasi variabel lingkungan Path untuk menunjuk ke jre / bin.

Sekarang tidak dapat menemukan jvm.dll.

Saya yakin saya bisa meretas dan membuat ini berfungsi, tetapi semua yang saya lakukan tampaknya salah, dan saya ingin melakukannya dengan cara yang bersih & benar. Saya bersumpah saya telah menginstal Eclipse sebelumnya dan jelas tidak perlu memodifikasi variabel lingkungan apa pun.

Apakah ada instruksi di suatu tempat yang baru saja saya lewatkan?

Periksa langkah 7 di url di bawah ini tentang menggunakan eclipse di lingkungan Winodws http://www.wikihow.com/Download,-Install,-and-Run-JDK-and-Eclipse

Setelah Anda mengunduh arsip Eclipse, Anda harus melakukannya dekompresi file zip, yang akan membuat Eclipse terbuka map. Anda mungkin ingin mengekstrak arsip ke root C: \ drive, sehingga membuat folder “C: \ eclipse”, atau tinggal memindahkan hasil ekstraksi eclipse folder ke root drive C: \ jika Anda sudah mengekstraknya. Karena Eclipse tidak memiliki penginstal apa pun, akan ada file di dalamnya folder Eclipse bernama eclipse.exe. Anda dapat mengklik dua kali pada file tersebut untuk menjalankan Eclipse.

Saya juga belum menemukan installer gerhana atau file setup untuk windows. Jadi kelihatannya ini adalah cara paling umum untuk mulai menggunakan eclipse di windows.

FTP dari Windows Explorer dalam mode aktif?

Saya ingin menggunakan Windows Explorer untuk menyambung ke situs FTP dalam perpindahan aktif. Di mana dan apa yang harus saya konfigurasikan untuk mengaktifkan ini?

4
mafu — Sumber

1 Jawaban

9
9

Ini dikonfigurasi dari Control Panel - khususnya Opsi Internet. Buka tab Advanced dan gulir ke bawah hingga Anda menemukan kotak centang berlabel "Gunakan FTP Pasif (untuk kompatibilitas modem firewall dan DSL)". Hapus centangnya.

Ya, Windows Explorer mengikuti pengaturan Internet Explorer.

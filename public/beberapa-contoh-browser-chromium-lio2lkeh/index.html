Beberapa contoh browser chromium

Saat saya memulai chromium dengan proxy, contoh lainnya akan menggunakan proxy juga.

Apakah ada cara untuk menjalankan beberapa contoh secara bersamaan? Hanya saja chromium tidak berfungsi seperti firefox, sehingga Anda dapat memutuskan situs mana yang harus melalui proxy, dan mana yang tidak boleh melakukannya. (Saya menjalankan xfce4, dan menggunakan opsi baris perintah untuk mengaktifkan proxy)

Kurangnya kontrol proxy yang disematkan telah menjadi salah satu keluhan Chrome dan Chromium, itulah sebabnya saya masih mengandalkan Firefox yang bagus untuk hal semacam ini. Para pengembang hanya memutuskan untuk menyerahkannya pada sistem operasi untuk menangani proksi, yang menimbulkan masalah ketika Anda ingin menjalankan proksi untuk BitTorrent, yang pengembangnya yang picik juga sampai pada kesimpulan yang sama tidak bersemangat. Ini adalah salah satu area di mana Firefox masih jelas lebih unggul.

Namun, platform Chromium mengekspos API Proxy Chrome untuk ekstensi, yang memungkinkan pengembang ekstensi membuat ekstensi Chrome keren yang memecahkan masalah ini. Dari yang saya lihat di Toko Web Chrome, Proxy SwitchySharp menonjol karena deskripsinya mengatakan itu menggunakan API alih-alih mengubah setelan sistem.

Artinya, dengan memasang ekstensi ini, Anda dapat menentukan situs mana yang melewati proxy dan mana yang tidak, dan pada saat yang sama hanya membatasi proxy untuk memengaruhi satu instance browser Chrome / Chromium tersebut.

Selain itu, untuk pengembang yang ingin mempelajari lebih lanjut tentang API, lihat Ekstensi Google Chrome - chrome.proxy

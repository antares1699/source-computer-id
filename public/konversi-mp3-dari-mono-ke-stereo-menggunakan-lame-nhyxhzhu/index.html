Konversi mp3 dari mono ke stereo menggunakan Lame

Aplikasi mengharuskan file mp3 yang diterimanya dikodekan dengan 2 saluran (stereo), tetapi file asli hanya memiliki 1 saluran (mono).
Kami menggunakan Lame untuk konversi, tetapi saya gagal membuatnya mengkloning saluran mono dan membuat stereo palsu.

lame.exe -m j mono.mp3 stereo.mp3

-m d atau -m s tampaknya juga tidak berfungsi. Apakah mungkin melakukan ini dengan orang lumpuh?

Jawaban atas pertanyaan saya sendiri adalah: Tidak - tidak mungkin dengan orang lumpuh

Solusi penyelesaian saya adalah menggunakan alat baris perintah sox dan melakukan duplikasi saluran sebelum konversi mp3:

sox mono.wav -c 2 stereo.wav

Kemudian setelah itu gunakan lame:

lame -m j stereo.wav stereo.mp3

Anda dapat melakukannya dengan ffmpeg. Berikut adalah perintah yang saya gunakan untuk file mono mp3 dengan sampling rate 11025 Hz dan bitrate 40 kbps:

ffmpeg -i mono.mp3 -ac 2 -ab 96000 -ar 22050 stereo.mp3

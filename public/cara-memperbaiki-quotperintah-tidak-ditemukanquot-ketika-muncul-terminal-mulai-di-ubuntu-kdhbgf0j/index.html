Cara memperbaiki "perintah tidak ditemukan" ketika muncul terminal mulai di ubuntu

Ketika saya menjalankan terminal di OS dasar saya (Ubuntu 14.04 dist), itu muncul sebagai berikut:

Support: command not found
anargu@anargu-notebook-PC:~$

baris pertama ( Dukung: ... ) muncul setelah mencoba menginstal nodejs tetapi saya mengalami masalah untuk menginstalnya dan saya mencoba menginstal seperti 5 kali dengan cara yang berbeda dan akhirnya saya dapat menginstalnya. Tetapi " Dukungan: perintah tidak ditemukan "muncul setiap kali saya menjalankan terminal.

Saya baru di ubuntu dan saya tidak tahu apa yang dimaksud dengan Dukungan tetapi saya dapat mengetik beberapa perintah seperti ls, cd, sudo apt-get dan itu bekerja dengan baik. Jadi, tolong seseorang dapat membantu saya?

Kesalahan bash "xxxxxx: command not found" muncul saat Anda mencoba menjalankan perintah / program yang tidak diinstal atau tidak di jalur, dll. Sesuai komentar Hastur, mencari "Dukungan" di .bashrc adalah ide bagus, dan periksa juga .login & .profile jika ada (menurut saya mereka tidak harus berjalan hanya untuk membuka terminal.

Bisa di file ~/.bashrc. Secara umum untuk menemukan kesalahan misteri saya akan mengujinya seperti ini:

Coba buka terminal dan ketik

source ~/.bashrc

Jika kesalahan muncul, maka pasti ada sesuatu di sana yang menjadi masalahnya. Lihat melalui file untuk baris yang menyebabkan kesalahan - bisa baris demi baris yang ditempelkan ke terminal jika tidak berlebihan, atau tambahkan baris echo "step 1" ... echo "step 2" dll untuk mempersempitnya. Kemudian hapus / ubah garis yang melanggar.

Terima kasih Xen2050, saya sedang memeriksa file bashrc saya (dengan echo "step1" ...) dan saya menemukan bagian ini:

###-tns-completion-start-###
if [ -f /home/anargu/.tnsrc ]; then 
    source /home/anargu/.tnsrc 
fi
###-tns-completion-end-###

Saya bisa mengidentifikasinya karena saya memasang skrip asli ( https://www.nativescript.org/ ) dan bagian kode ini sesuai dengannya. Ketika saya mengomentari (menaruh beberapa "#") baris itu,:

Support: command not found baru saja menghilang.

Saya pikir saya memecahkan masalah ini hanya dengan mengomentari bagian kode itu karena saya tidak dapat menemukan masalah apa pun saat ini. Terima kasih banyak!

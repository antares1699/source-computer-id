AutoHotKey: memetakan ulang Alt + Shift + Tab ke (L) Win + Shift + Tab

Saya mencoba menggunakan AutoHotKey untuk mendapatkan Alt + Tab dan Bergeser + Alt + Tab di Windows agar memiliki perilaku yang sama seperti Cmd + Tab dan Bergeser + Cmd + Tab di macOS.

catatan: Cmd di mac keyboard mengirim Menang kunci.

Apa yang saya dapatkan untuk bekerja adalah:

LWin & Tab:: AltTab

Apa yang belum bisa saya lakukan adalah mendapatkan LWin + Bergeser + Tab mengirim Bergeser + Alt + Tab . Saya sudah mencoba:

  1. #Shift & Tab:: ShiftAltTab (kesalahan: harus menentukan L atau R untuk # dalam kasus ini)
  2. LWin & Shift & Tab :: ShiftAltTab (Tidak dapat menggunakan 3 kombinasi khusus)

  3. Lwin & Tab::
       if GetKeyState("Shift")
           {
           Send {ShiftAltTab}
           }
       else
           {
           Send {AltTab}
           }

Saya juga mencoba dengan garis kembali setelah setiap baris Send. Tidak ada yang terjadi saat menekan

  • LCmd + Tab (== LWin + Tab ) atau
  • LCmd + Bergeser + Tab (== LWin + Bergeser + Tab ).

Adakah cara cepat dan sederhana untuk melakukan ini? Saya ingin menggunakan AHK karena ini memodifikasi kombinasi untuk meniru banyak perilaku keyboard Mac lainnya.

    If GetKeyState("Shift","P")
        Send {Alt Down}{Shift Down}{Tab}

        Send {Shift Up}{Alt Up}
        AltTabMenu := false 
    return

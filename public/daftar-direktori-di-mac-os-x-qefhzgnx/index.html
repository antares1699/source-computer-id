daftar direktori di Mac OS X

Saya membuang banyak file (musik dan lainnya) ke Macbook baru saya yang mengkilap, dan karena saya lebih nyaman dengan linux daripada Mac (pada titik ini) saya cenderung menggunakan terminal.

Saya melakukan ls -al pada file yang saya transfer, dan beberapa memiliki "@" di akhir string izin, dan beberapa tidak.

Sesuatu seperti:

drwxrwxr--@ 93 user staff etc.
drwxrwxr-- 107 user staff etc.

Yang tanpa "@" dapat dilihat di Finder dan diakses oleh program lain-- file dan direktori "@" tidak terlihat. Adakah yang bisa menjelaskan apa artinya "@", dan bagaimana chmod (atau apa pun) sehingga saya dapat menggunakan file-file ini? Saya berasumsi bahwa ini adalah semacam bendera sistem tetapi saya tidak tahu bagaimana cara membatalkannya. Chmod 777 tidak berpengaruh dan saya sudah memiliki filenya.

Terima kasih

Link ini menjelaskan bagaimana simbol '@' digunakan untuk menandakan bahwa file tersebut memiliki 'atribut tambahan' yang dapat dilihat dengan mengetik:

xattr -l <nama file>

Atribut tambahan digunakan untuk menyimpan informasi tambahan tentang file. Misalnya, saat file diunduh dari internet, file tersebut mungkin berisi atribut tambahan yang memicu pesan 'peringatan file ini telah diunduh ..' saat dijalankan.

Seperti yang dikatakan JT.WK, '@' menunjukkan atribut tambahan yang dilampirkan ke file. Coba gunakan ls -lO@ pada file - -O menunjukkan bendera file, dan -@ menunjukkan nama (tetapi bukan konten) dari atribut yang diperluas. Dugaan saya adalah Anda akan melihat com.apple.FinderInfo xattrs, dan bendera hidden (catatan: Saya mengasumsikan Mac OS X 10.5 atau 10.6 di sini - versi yang lebih lama tidak memetakan bendera invisible Finder menjadi sebuah bendera file). Dengan asumsi saya benar tentang itu, Anda dapat menghapus bendera dengan:

chflags nohidden /path/to/file

(Jika Anda menggunakan Mac OS X v10.4, tanda Finder hampir tidak dapat diakses dari baris perintah - beri tahu saya dan saya akan melihat apa yang dapat saya hasilkan.)

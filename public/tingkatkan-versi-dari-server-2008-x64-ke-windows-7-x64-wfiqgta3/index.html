Tingkatkan versi dari Server 2008 x64 ke Windows 7 x64

Apakah mungkin untuk "meningkatkan" dari Windows Server 2008 x64 ke Windows 7 x64? Jika ya, apa saja yang didapatnya? Ada tautan ke petunjuk tentang cara melakukan ini?

Saya tahu bahwa Vista dan Server 2008 berbagi banyak fitur dan jadi saya rasa ini mungkin. Saya telah menggunakan Server 2008 sebagai mesin desktop saya (kesalahan) dan sekarang ingin pindah ke Windows 7 x64 tetapi tidak ingin melalui siklus instalasi mesin penuh.

Sejauh yang saya tahu, berpindah dari OS tingkat server ke OS tingkat konsumen tidak pernah mungkin dilakukan dalam keluarga OS Windows. Tidak ada "Windows 7 Server", tetapi ada Windows Server 2008 R2, yang tampaknya merupakan jalur pemutakhiran logis dari Windows Server 2008. Jika Anda hanya menggunakan sistem Server 2008 sebagai stasiun kerja, dan Anda ingin pindah ke Windows 7, maka saya akan mengatakan bersiaplah untuk penginstalan baru.

Jika Anda membutuhkan opini kedua, MS juga memiliki file Penasihat Peningkatan Windows 7 tersedia.

Anda tidak dapat memutakhirkan dari OS Server ke OS Desktop. Microsoft secara eksplisit memeriksa jalur peningkatan dan tidak akan meminta untuk memutakhirkan Windows Server ketika mencoba menginstal Windows 7. Seperti yang telah disebutkan, jalur logis adalah untuk meningkatkan ke Windows Server 2008 R2. Ini akan dirilis pada minggu yang sama dengan Windows 7.

Bisakah saya membuat tombol mod xmonad menjadi ctrl-alt?

Alt dan ctrl digunakan secara individual untuk banyak hal, dan keyboard saya saat ini tidak memiliki tombol Windows. Dapatkah saya membuat kunci mod xmonad menjadi ctrl-alt dan bukan hanya satu tombol? (Misalnya, ctrl-alt-1 harus ditempatkan di desktop pertama, tetapi ctrl-1 dan alt-1 harus diteruskan ke aplikasi saat ini.)

4

Ya kamu bisa. Ini sebenarnya agak sederhana. Saya akan memberikan contoh dan kemudian menjelaskannya. Contoh ini datang langsung dari versi lama file konfigurasi saya yang Anda dapat menemukannya secara online di sini . Sekarang untuk beberapa kode:

, ((altMask .|. controlMask, xK_Left  ), prevWS)
, ((altMask .|. controlMask, xK_Right ), nextWS)

Dan hanya itu yang ada di sana. Ketika saya menekan Alt + Ctrl + Left itu mengeksekusi prevWS ( dari CycleWS ) dan ketika saya menekan Alt + Ctrl + Right itu mengeksekusi nextWS. Dan hanya itu saja. Anda akan menemukannya di bagian myKeys pada konfigurasi XMonad saya dan jika Anda membutuhkan bantuan lebih lanjut, tanyakan saja pada saya di komentar.

Saya harus mencatat bahwa di file konfigurasi:

altMask = mod1Mask

Alt harus menjadi salah satu mod-? kunci.

Oh, dan anggap saja Anda baru mengenal XMonad, jika Anda tidak ingin terjun ke dalam membuat file konfigurasi yang lebih besar seperti milik saya dan ingin menyimpannya untuk saat ini dan cukup tambahkan atau modifikasi beberapa kunci saja. Saya merekomendasikan menggunakan fungsi EZConfig seperti yang ditunjukkan dalam contoh fungsi utama di sini .

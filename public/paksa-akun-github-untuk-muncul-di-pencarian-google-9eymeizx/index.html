Anda tidak dapat memaksa Google untuk mengindeks halaman jika tidak mau. Anda dapat menyarankan satu ke Google dengan menggunakan tautan ini:

https://www.google.com/webmasters/tools/submit-url

Jika Anda adalah pemilik / administrator situs web Github, Anda akan memiliki lebih banyak opsi yang tersedia untuk Anda dengan menggunakan Alat Webmaster Google

Jawaban Garrulinae secara teknis benar, tetapi mengabaikan informasi penting. Verifikasi kepemilikan dengan Google Search Console (GSC) akan meningkatkan peringkat pencarian akun Github, repositori, atau situs web Anda, karena didokumentasikan di google.com , jadi jelas metode itu lebih disukai. Tapi saat ini tidak mungkin.

Banyak situs mengizinkan verifikasi kepemilikan melalui pengaturan atau prosedur yang dilakukan dengan Google, seperti Squarespace, Medium, dll., Di mana Anda secara tidak langsung mengirimkan info verifikasi Google untuk halaman yang Anda kontrol (properti, untuk menggunakan terminologi GSC Google). Setelah pengiriman, situs-situs ini menangani kode verifikasi dan file .html untuk Anda. Sayangnya itu tidak mungkin dengan LinkedIn dan Github, karena mereka dimiliki oleh Microsoft, yang tidak memiliki pengaturan seperti itu dengan Google.

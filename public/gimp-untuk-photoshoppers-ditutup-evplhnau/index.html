Tautan : Tweaks Berguna Untuk Membuat GIMP Ganti Photoshop mencakup beberapa fitur yang dimiliki Photoshop secara default, tetapi harus diaktifkan secara manual pada GIMP.

Tutorial : Gimp-tutorials.net

Tutorial : Gimpology.com

Tutorial : Cara membuat GIMP memiliki Tata Letak Photoshop

Tip : Klik Edit -> Pintasan Keyboard, dan baca daftarnya. Ini adalah ringkasan yang bagus untuk semua fungsi GIMP, pintasan keyboard apa yang memanggilnya, dan di mana lokasinya di menu.

Tip : Jangan berharap salah satu pintasan keyboard sama! CTRL + D sekarang Duplicate, bukan Deselect. Transformasi Seleksi sekarang SHIFT + T, bukan CTRL + T, M sekarang Pindah, bukan V ...

Tip : Lapisan default tidak memiliki "Alfa Chanel", jadi menghapus konten (melalui penghapus, atau DEL) akan menghapusnya ke warna latar belakang, bukan transparan.

Tip : Interpolasi transformasi default di GIMP adalah "Cubic", sedangkan di Photoshop adalah "None". Anda mungkin memperhatikan saat menskalakan warna solid dan bertanya-tanya mengapa warnanya memudar ...

Tip : Space Bar PINDAH sekarang, bukan PAN. Jadi Anda tidak lagi memindahkan pandangan Anda di sekitar gambar, Anda sekarang memindahkan gambar di bawah pandangan Anda ... (Pada titik ini saya sedang mempertimbangkan VM'ing windows ...)

Tip : Saat menggambar garis dengan alat pensil, jika Anda ingin garis lurus (horz / vert / diag) di Photoshop, Anda harus menahan SHIFT sebelum mulai menggambar. Di GIMP, klik sekali di mana Anda ingin memulai, lalu tahan shift + ctrl, lalu klik di mana Anda ingin mengakhiri.

Tabel kunci diberi nama dalam deskripsi perintah list-keys dan dijelaskan sebelumnya di halaman manual:

  • vi-edit, emacs-edit

    Digunakan saat mengedit baris pada prompt perintah.
    Misalnya. melalui command-prompt, confirm-before, atau petunjuk "Goto Line", "Search Up", atau "Search Down" dalam mode salin

  • vi-choice, emacs-choice

    Digunakan saat memilih dari daftar.
    Misalnya. via choose-client, choose-session, choose-window, choose-tree, choose-list, choose-buffer, atau find-window

  • vi-copy, emacs-copy

    Digunakan saat melihat scrollback atau keluaran lainnya.
    Misalnya. copy-mode, run-shell output, atau kesalahan dari file konfigurasi startup atau source-file.

Kunci yang terikat dalam tabel ini aktif ketika berbagai "mode" aktif. Mereka tidak memerlukan kunci Prefix, karena "mode" itu sendiri menggantikan interaksi normal apa pun dengan panel aktif.

Selain itu, binding dalam tabel ini dibatasi untuk perintah tujuan khusus tertentu (misalnya terkait dengan pergerakan dan pengeditan), dan setiap kunci hanya dapat diikat ke satu perintah (yaitu \; tidak dapat digunakan untuk mengikat kunci ke urutan beberapa perintah).

"Perintah mode" ini tidak dijelaskan di halaman manual, tetapi Anda dapat menemukannya dengan memeriksa daftar untuk tabel (misalnya tmux list-keys -t emacs-copy, dan lain-lain) atau dari kode sumber (mode-key.c); nama mereka umumnya deskriptif sendiri (misalnya cursor-up, page-up, cancel, dan lain-lain).

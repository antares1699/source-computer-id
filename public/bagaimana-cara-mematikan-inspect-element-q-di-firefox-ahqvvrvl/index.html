Bagaimana cara mematikan Inspect Element (Q) di Firefox?

Saya mendapatkan Firefox 12 baru dan saya menekan "Inspect Element (Q)". Saya terbiasa menekan "Inspect" dan mendapatkan FireBug, sekarang saya mendapatkan alat inspeksi Firefox. Kelihatannya bagus, tapi bagaimana cara mematikannya - bukan menonaktifkannya, tapi berhenti memeriksa halaman saat ini?

Saya harus menutup tab, saat mengerjakan aplikasi web besar, masuk kembali dan atur ulang semua yang saya uji untuk kembali ke tampilan normal. Buang-buang waktu.

FF 14 mengaktifkan kembali inspektur (hanya untuk bersenang-senang saya kira) bahkan jika Anda telah menonaktifkannya sebelumnya.

Untuk menonaktifkan, ubah berikut ini di about: config

devtools.inspector.enabled                 false
devtools.styleinspector.enabled            false
extensions.firebug.hideDefaultInspector    true

Tetapi dengan ekstensi ini Anda dapat menonaktifkan pintasan: Keybinder: https://addons.mozilla.org/en-us/firefox/addon/keybinder/
(Catatan di atas adalah untuk mengganti Pintasan yang Dapat Disesuaikan oleh Tim Taubert, yang tidak lagi dikembangkan / dihapus)

Config -> Pemetaan pintasan -> ubah> (temukan pintasan, nonaktifkan).

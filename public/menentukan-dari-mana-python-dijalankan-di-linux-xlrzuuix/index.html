Hal yang biasanya Anda cari bukanlah di mana Python itu sendiri berada, tetapi di mana perpustakaannya berada. Untuk itu saya akan merekomendasikan ini:

(dari dalam shell Python)

import sys
import pprint
pprint.pprint(sys.path)

Di luar itu Anda masih dapat melakukan which python secara alami.

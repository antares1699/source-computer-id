Perbedaan Antara Pemindai Keamanan Microsoft dan Alat Penghapusan Perangkat Lunak

Apa perbedaan antara MSS dan MSRT?

4
kenneth — Sumber

1 Jawaban

6
6

Mereka melayani tujuan yang agak berbeda. Pemindai Keamanan Microsoft akan meninjau komputer Anda sesuai permintaan untuk menemukan virus atau malware apa pun. MSRT secara khusus ditargetkan pada sejumlah kecil malware (khususnya botnet umum), dan tidak dirancang untuk dijalankan secara manual. Sebaliknya, MSRT dirancang terutama untuk rilis Pembaruan Windows bulanan, setelah itu secara otomatis akan memindai komputer Anda di latar belakang (Jika Anda mengaktifkan Pembaruan Windows, MSRT berjalan sebulan sekali tanpa Anda sadari). Jadi, MSRT adalah upaya untuk menghentikan botnet tanpa pengguna harus mengambil tindakan apa pun, sedangkan MSS adalah alat sederhana untuk pengguna terkait yang akan memeriksa semua yang diketahui Microsoft.

Anda dapat membuka MSRT dengan mengetik di MRT ke dalam kotak perintah Run, ini akan membuka GUI untuk MSRT , klik tautan "Lihat daftar ...", di sini Anda dapat melihat daftar rentang target yang sempit dari alat tersebut. Daftar ini kadang-kadang diperbarui melalui Pembaruan Windows, ketika daftar diperbarui, secara diam-diam akan menjalankan alat di latar belakang untuk mencari ancaman baru. Log untuk MRT disimpan di C: \ Windows \ Debug \ mrt.log

Bagaimana cara menggunakan perintah "temukan" untuk membuat daftar file dengan panjang nama file tertentu?

Di folder saya, saya mendapat file seperti

/data/filename.log /data/filename.log.1 /data/filename.log.2 /data/filenamefilenamefilename.log /data/filenamefilenamefilename.log.2

Saya ingin menggunakan perintah "find" untuk membuat daftar file yang panjangnya lebih dari 15 karakter.

Saya telah mencoba yang berikut, tetapi tidak ada yang berhasil:

find ./ -type f -iregex "/^.*{15,1000}$/" -print
find ./ -type f -iregex "/^.*{15}$/" -print
find ./ -type f -iregex "^.*{15}$" -print
find ./ -type f -iregex ".*{15}" -print
find ./ -type f -iregex ".{15}" -print
find ./ -type f -iregex ".{15,1000}" -print

Tidak yakin dengan cara yang benar?

Terima kasih!

Saya akan menyarankan versi Paul jika Anda ingin mencocokkan hanya pada nama file. Jika Anda ingin menggunakan -regex untuk mencocokkan di jalur lengkap, Anda dapat melakukannya, mis.

find /data -type f -regextype posix-egrep -regex ".{15}"

atau untuk 15 karakter atau lebih

find /data -type f -regextype posix-egrep -regex ".{15}.*"

Periksa halaman manual untuk mesin regexp berbeda yang tersedia untuk digunakan dengan -regextype.

Anda juga dapat menggunakan opsi -regex untuk mencari hanya nama file dengan 15 karakter:

find /data -type f -regextype posix-egrep -regex ".*/[^/]{15}"

atau 15 karakter atau lebih:

find /data -type f -regextype posix-egrep -regex ".*[^/]{15}"

Parameter name menerima globbing sederhana, jadi berikut ini akan berfungsi:

 find . -type f -name '????????????????*'

Jadi itu adalah 16 tanda tanya diikuti dengan tanda bintang. Tanda tanya cocok dengan satu karakter, tetapi harus cocok, jadi 16 karakter berturut-turut memastikan bahwa ada 16 karakter dalam nama file. Tanda bintang di akhir mengizinkan karakter tambahan apa pun, memenuhi persyaratan "lebih dari 15 karakter" dengan mencocokkan "16 karakter atau lebih".

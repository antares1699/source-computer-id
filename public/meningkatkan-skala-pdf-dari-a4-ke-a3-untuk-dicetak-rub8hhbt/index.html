Meningkatkan skala PDF dari A4 ke A3 untuk dicetak?

Saya memiliki dokumen PDF yang diformat dalam A4. Ini terdiri dari lusinan halaman diagram dan sangat kecil untuk dibaca dengan nyaman saat dicetak. Apakah ada cara untuk menskalakan dokumen hingga A3 untuk dicetak?

Saya akan berasumsi ini dapat dilakukan dari preferensi printer atau preferensi pengaturan halaman di akrobat tetapi saya tidak dapat melihat apa pun!

Menggunakan Ghostscript.dll :

gswin32c.exe -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sPAPERSIZE=a3 -dFIXEDMEDIA -dPDFFitPage -sOutputFile=output.pdf input.pdf

Tidak perlu menskalakan PDF mengajukan jika Anda hanya membutuhkan hasil cetakan kertas diskalakan hingga A3. Acrobat reader (versi terbaru 9.3.1) pasti memiliki opsi untuk melakukan itu. Itu dapat menskalakan halaman PDF apa pun ke atas atau ke bawah ke ukuran media apa pun yang tersedia di printer Anda.

Pengaturan dapat ditemukan di bagian kiri bawah halaman depan pada dialog pencetakan Acrobat / AcroReader. Anda harus menemukan area 'pengaturan halaman' dengan daftar drop-down 'menyesuaikan ukuran halaman' dengan beberapa tanda centang tambahan untuk mengaktifkan / menonaktifkan (kutipan string diterjemahkan dari bahasa Jerman kembali ke bahasa Inggris). Ini bahkan dapat digunakan untuk membuat cetakan n-up atau pengenaan pamflet dari ukuran halaman A4 biasa, letter atau PDF lainnya.

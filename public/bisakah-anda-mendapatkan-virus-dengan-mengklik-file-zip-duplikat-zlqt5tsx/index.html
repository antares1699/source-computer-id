Seperti yang dikatakan Hack-R dalam jawabannya, file Zip dapat berisi virus.

Anda tidak bisa tertular virus hanya dengan tunggal mengklik file Zip. Sunting: Secara teknis dimungkinkan, jika Anda memiliki satu klik untuk menjalankan pintasan yang diaktifkan. Lihat paragraf di bawah.

Mengklik ganda adalah cerita yang berbeda. Jika itu adalah file Zip asli dan berisi file yang terinfeksi, Anda tidak akan terkena virus kecuali Anda mencobanya menjalankan file yang terinfeksi. Misalnya Anda memiliki file Zip yang berisi virus, yang disebut virus.exe. Mengklik dua kali file Zip tidak akan menginfeksi Anda, tetapi mengklik dua kali virus.exe (dan karena itu, menjalankan virus.exe) dapat menyebabkan Anda terkena virus.

Ada satu skenario lain yang bisa menjadi faktor penyebab. Tidak jarang orang yang tidak bermoral memalsukan file Zip. Mari kita ambil virus.exe lagi sebagai contoh. Ubah nama file virus.exe menjadi virus.zip.exe dan Anda memiliki pengaturan default di Windows Explorer untuk menyembunyikan ekstensi dari jenis file yang dikenal, akan muncul file tersebut bernama virus.zip. Pengguna yang tidak curiga akan mengira file tersebut adalah file Zip, meskipun ikonnya salah. Mengklik dua kali itu akan menjalankan file yang terinfeksi virus.

Anda bisa mendapatkan 0wned dengan melihat gambar:

"Microsoft Windows GDI + rentan terhadap buffer overflow, yang disebabkan oleh pemeriksaan batas yang tidak tepat. Dengan membujuk korban untuk membuka file gambar TIFF yang dibuat secara khusus, penyerang jarak jauh dapat memanfaatkan kerentanan ini untuk meluap buffer dan mengeksekusi kode arbitrer pada sistem dengan hak istimewa dari korban. "

referensi: http://www.iss.net/security_center/reference/2114192.html

Kode, dijalankan dengan kredensial yang sama dengan yang Anda miliki sebagai pengguna. Misalnya dengan menggunakan browser atau alat bawaan lainnya untuk mengambil data dari internet.

Hal yang sama juga berlaku untuk tipe file lainnya. Ketika Anda memulai sebuah program (bahkan program unzip yang tampak tidak bersalah) dan jika program itu rentan terhadap masukan tertentu .. maka Anda mungkin mendapatkan apa pun yang diinginkan penyerang.

Yang mengatakan: peluangnya cukup rendah, meskipun ada satu.

Karena itu, pesan kesalahan yang Anda tunjukkan si tidak gejala virus atau malware lainnya. Itu adalah pesan kesalahan standar yang Anda terima ketika arsip yang dikompresi rusak (yaitu file rusak, seperti jika itu ada di flash drive yang tidak dikeluarkan dengan benar) atau jika tidak lengkap (seperti jika unduhan Internet tidak sepenuhnya lengkap).

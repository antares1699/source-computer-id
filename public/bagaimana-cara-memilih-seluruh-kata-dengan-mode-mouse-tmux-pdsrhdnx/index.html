Bagaimana cara memilih seluruh kata dengan mode mouse Tmux?

Mode tetikus tmux default tampaknya memilih kata per karakter - ini agak terlalu terperinci bagi saya; jika saya mengklik [atau mengklik dua kali] di tengah kata, saya ingin ia mulai memilih seluruh kata.

Apakah ada cara untuk melakukannya? Terimakasih banyak!

Tahan Shift dan klik dua kali teks tersebut. Ini mengirimkan input mouse langsung ke terminal (xterm, terminal, dll) dan melewati mode mouse tmux.

Catatan: Saya menguji ini di xterm dan gnome-terminal. Untuk terminal lain, Anda mungkin perlu menahan tombol pengubah yang berbeda.

Seperti xtmux bisa melakukan itu, tetapi itu tidak akan berlaku di ssh. Setara iTerm dari Shift adalah Alt. Saya belum menemukan solusi tmux murni, tetapi Anda dapat mempertimbangkan salin / tempel mode vim: set-window-option -g mode-keys vi bind-key -t vi-copy 'v' begin-selection bind-key -t vi-copy 'y' copy-selection bind-key -t vi-copy 'Escape' copy-selection Dengan itu dan set -g mode-mouse on, Anda dapat mengklik dan menahan di awal pilihan Anda, tekan w dan perintah vim lainnya sampai Anda memilih yang Anda inginkan, dan kemudian lepaskan.

Karena khawatir tentang ruang disk Windows membutuhkan waktu lama, saya akan memberikan bagian saya.

  • Jelajahi folder utama setiap pengguna dan folder Dokumen Saya, lalu hapus file yang tidak diperlukan
  • Jalankan cleanmgr. Pada tab Advanced-nya, hapus juga pos pemeriksaan Pemulihan sistem (jika Anda mengaktifkannya sama sekali)
  • Hapus file hibernasi (hiberfil.sys) dan / atau pindahkan file swap (pagefile.sys)
  • Hapus semuanya di C:\Windows\SoftwareDistribution\Download, yang berisi cache unduhan Pembaruan Windows
  • Hapus semuanya di C:\Windows\Temp
  • Hapus semuanya di C:\Documents and Settings\<user>\Local Settings\Temp
  • Hapus folder C:\Windows\$Nt*Uninstall* yang tersembunyi, yang memungkinkan Anda untuk menghapus pembaruan yang Anda instal dari waktu ke waktu (bukan berarti Anda membutuhkannya, bukan?). Berhati-hatilah bukan untuk menghapus folder $hf_mig$
  • Cari partisi untuk *.log file dan hapus
  • Menggunakan CCleaner mungkin juga membantu (atau mungkin tidak)

1.) Matikan pemulihan sistem, lalu hidupkan kembali dan kemudian buat titik pemulihan sistem manual, ini akan membersihkan ruang disk yang manggung atau lebih, lakukan ini tepat sebelum Anda mencadangkan.

2.) Kemudian buka prompt perintah dan ketik

CD\

tekan tombol enter, lalu

hapus * .tmp / s

tekan tombol enter (berhati-hatilah saat mengetik perintah erase dengan benar, spasi antara e dan *, antara p dan / juga)

ini akan menghapus semua file temporer Anda yang aman untuk dihapus dan dapat menumpuk seiring waktu.

3.) mengosongkan tempat sampah

4.) Hapus semua akun "pengguna" Anda tidak membutuhkannya lagi. Berhati-hatilah dengan apa yang Anda lakukan di sini, jika hanya ada satu akun pengguna. Jangan hapus.

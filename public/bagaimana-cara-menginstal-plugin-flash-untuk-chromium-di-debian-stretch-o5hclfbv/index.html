Bagaimana cara menginstal plugin flash untuk Chromium di Debian Stretch?

Mulai dari Debian Stretch, paket pepperflashplugin-nonfree hilang. Namun, setelah penginstalan manual, mengunjungi chrome://version menunjukkan bahwa flash dinonaktifkan. Bagaimana cara menginstal plugin flash untuk Chromium di Debian Stretch?

Arsip multimedia Debian memiliki paket flashplayer-chromium, yang berisi file-file yang diperlukan (terutama libpepflashplayer.so). Tambahkan saja

deb http://www.deb-multimedia.org stretch main non-free

ke /etc/apt/sources.list, lalu

apt-get update
apt-get install deb-multimedia-keyring
apt-get install flashplayer-chromium

Saat menambahkan deb-multimedia-keyring, terima paket yang belum diverifikasi, atau buka http://www.deb-multimedia.org/ untuk petunjuk rinci tentang menambahkan kunci secara manual.

Harap dicatat, bahwa arsip ini berisi versi terbaru dari beberapa paket terkait multimedia, jadi apt-get upgrade akan meningkatkan beberapa paket peregangan standar.

Harap hapus komentar setiap baris di file sources.list, tetapi:

deb http://http.debian.net/debian/ jessie kontributor utama tidak gratis

Kemudian, jalankan pembaruan bakat && pencarian bakat flashplugin-nonfree.

Itu harus menemukan paket-paket berikut:

https://packages.debian.org/jessie/flashplugin-nonfree

Dalam skenario terburuk, Anda dapat mengunduh paket dari tautan di atas dan menginstalnya melalui dpkg -i.

Cara yang tepat seperti 23 oktober 2018:

Pergi ke https://get.adobe.com/flashplayer/ dan salin tempel versi flash, mis 31.0.0.122

Unduh plugin dengan contoh di ~/.mozilla/plugins/libpepflashplayer.so

Buat skrip pembungkus baru untuk menjalankan chromium, dengan contoh ~/bin/chromium:

#!/bin/sh

export CHROMIUM_FLAGS="--ppapi-flash-path=~/.mozilla/plugins/libpepflashplayer.so --ppapi-flash-version=31.0.0.122" # <= check mathing fetched version
exec /usr/bin/chromium "$@"

Kemudian

chmod +x ~/bin/chromium

Tambahkan ~/bin pada Anda variabel PATH pada ~/.bashrc untuk menjalankannya tanpa harus meletakkan PATH penuh. Jika Anda menggunakan peluncur dengan alt F2 , cari cara mengedit PATH untuk manajer desktop Anda.

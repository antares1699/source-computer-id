Membuat gambar dari seluruh sistem

Saya memiliki beberapa komputer yang menjalankan Windows XP dan semuanya memiliki perangkat lunak yang sama yang diinstal pada perangkat keras yang sama. Saya ingin membuat satu gambar, sehingga jika komputer pengguna mati, saya dapat memulihkannya dengan gambar tersebut. Itu akan menghemat waktu saya dengan tidak menginstal Windows lagi dan semua perangkat lunak.

Selain itu, setiap pengguna memiliki favorit mereka sendiri, desktop dll. Saya ingin mendukungnya juga, sehingga setelah pemulihan gambar, saya dapat memulihkan favorit mereka, desktop, dll.

Pertama yang Anda inginkan adalah membuat gambar seluruh harddisk Anda. Berikut beberapa pilihan perangkat lunak yang sangat populer untuk melakukan ini:

  • Acronis True Image
  • Paragon Backup & Recovery 2010 Advanced Gratis
  • Macrium Mencerminkan Gratis
  • Clonezilla

Selanjutnya Anda ingin membuat salinan dari setiap profil pengguna. Windows XP memiliki utilitas untuk itu. Menggunakan utilitas ini dijelaskan di situs ini:
http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/sysdm_userprofile_copy.mspx?mfr=true

Kamu bisa gunakan USMT untuk membuat cadangan profil. Ini akan berfungsi untuk domain dan pengguna lokal. Anda perlu membaca dokumentasinya, tetapi ini cukup mudah. Ini akan menyimpan semua pengaturan juga.

Kemudian untuk membuat gambar Anda dapat membuat satu gambar dan menggunakan sysprep untuk membuat gambar OS secara umum. Ini akan menghapus apa pun yang tidak diperlukan untuk OS, tetapi tetap menyimpan perangkat lunak yang Anda instal di sana. Anda kemudian dapat selalu memuat gambar sysprep ini dan akan memakan waktu sekitar 15-20 menit untuk menginstal OS di komputer. Ini lebih lambat daripada membuat salinan biner, tetapi memberi Anda keuntungan dalam membuat instalasi umum untuk dipasang di komputer manapun.

Anda dapat membuat salinan persis yang akan lebih cepat dipulihkan, tetapi Anda perlu menyimpan beberapa versi untuk setiap pengguna. Ada banyak program di luar sana seperti acronis dan norton ghost. Saya pribadi menggunakan acronis.dll dan itu bekerja dengan baik!

Ketika Anda memutuskan apa yang ingin Anda lakukan atau apa yang Anda butuhkan, Anda selalu dapat bertanya tentang bagaimana menggunakan utilitas itu jika Anda tidak yakin. USMT adalah utilitas baris perintah, tetapi sangat mudah digunakan.

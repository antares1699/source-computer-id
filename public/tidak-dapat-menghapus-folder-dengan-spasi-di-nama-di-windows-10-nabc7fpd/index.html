Tidak dapat menghapus folder dengan spasi di nama di windows 10

Saya tidak tahu bagaimana ini terjadi tetapi saya memiliki folder di drive D: saya yang tidak dapat saya hapus. Pesan kesalahannya kira-kira seperti ini (diterjemahkan dari bahasa Jerman):

Elemen tidak ditemukan
Elemen tidak lagi ada di D: \ Downloads \
Periksa lokasi Elemen dan ulangi prosesnya

Nama foldernya adalah Yuki (ada spasi di belakang i) Saya tidak tahu bagaimana ruang itu sampai di sana tetapi sepertinya ada di sana.
Folder kosong (tidak ada elemen tersembunyi)

Saya tidak dapat menghapus folder, namun saya dapat membuat folder di dalam folder jika saya right-click > new Saya hanya memiliki satu opsi yang tersedia yaitu folder (diberi label dengan ikon-admin (saya rasa disebut demikian, perisai biru / kuning) Jika, misalnya, saya membuat folder test di Yuki, saya dapat menghapus folder test dan saya dapat membuat semua yang biasanya saya dapat folder, teks-doc dan sebagainya (di folder test)

Saya sudah mencoba untuk:

  • Hapus folder biasanya right click > delete (memicu kesalahan di atas)
  • Hapus folder dengan Cmd (Sistem tidak dapat menemukan file yang ditentukan)
  • Hapus folder dengan "Semuanya" (tidak ada yang terjadi)
  • Hapus folder dengan Cmd di safe-boot (Sistem tidak dapat menemukan file yang ditentukan)

Sekarang saya kehabisan ide dan berharap siapa pun di sini dapat membantu saya (saya minta maaf jika bahasa Inggris saya tidak dapat dipahami, saya akan mencoba menjelaskan semua yang muncul.

Edit1:
Saya lupa menyebutkan: Ya, saya mencoba mengganti nama folder terlebih dahulu, saya mencoba untuk menghilangkan ruang dan mendapatkan pesan kesalahan ini

nama file sumber dan tujuan sama

Jika saya mencoba mengganti namanya seperti Yukirename atau bahkan Yuki rename saya mendapatkan pesan kesalahan yang sama ketika mencoba menghapus file.

Tidak perlu boot ke Linux; Anda dapat menggunakan sintaks \\?\ khusus untuk menghapus direktori ini (jika kosong) dari prompt perintah:

rd "\\?\D:\Downloads\Yuki "

Atau, jika Anda menjalankan dir /x dari direktori di atas yang bermasalah, Anda juga akan melihat nama pendeknya yang dapat Anda gunakan untuk menghilangkannya. Jika nama pendeknya ternyata YUKI~1, ini juga akan menghapusnya:

rd YUKI~1

Jika Anda tidak ingin segera menghapusnya (misalnya jika tidak kosong), Anda dapat mengganti namanya agar lebih mudah dikelola:

ren YUKI~1 Yuki

Bagaimana cara saya menandatangani driver mode kernel 64-bit secara digital?

Saya memiliki driver mode kernel dan saya harus menginstalnya pada 64 bit win 7. Perlu ditandatangani secara digital. Saya menandatanganinya secara digital menggunakan dseo13b.exe. Tetapi ketika saya memuat driver, saya mendapatkan kesalahan dalam log peristiwa sistem yang mengatakan

Pengemudi gagal untuk memulai karena kesalahan berikut: Windows tidak bisa verifikasi tanda tangan digital untuk file ini. Perangkat keras terbaru atau perubahan perangkat lunak mungkin telah menginstal file yang ditandatangani dengan tidak benar atau rusak, atau yang mungkin merupakan perangkat lunak berbahaya dari yang tidak dikenal sumber.

Saya tidak ingin menggunakan mode penandatanganan tes. Bagaimana cara mengatasi ini? Apakah saya perlu mendapatkan sertifikat dari Microsoft?

Saya telah mengembangkan driver dan sekarang membuatnya bekerja pada mesin 64 bit.

Perusahaan saya mungkin membeli sertifikat dari verisign tetapi Apa yang harus saya lakukan setelah saya memperoleh sertifikat. Bagaimana cara menghubungkan file driver dengan sertifikat yang saya dapatkan? Dan juga bagaimana cara menghubungkan sertifikat silang yang diunduh dari internet dengan sertifikat yang saya dapatkan dari verisign? Saya membaca doc KMSC_WalkThru (Cara Melepaskan-Menandatangani Modul Kernel) tetapi hal-hal ini tidak jelas darinya. Bisakah Anda membantu.

Ya, Anda perlu membeli sertifikat dari Otoritas Sertifikat Tepercaya. Jika ada yang bisa membuat sertifikat, akan ada banyak sertifikat yang mengklaim sebagai "Microsoft Corporation" dan itu akan menjadi surga virus.

Yang bisa saya tawarkan selain itu adalah file batch berikut yang saya jalankan dari VS2010 di pasca-build. Ini menggunakan sertifikat dari penyimpanan sertifikat komputer, bukan file. Alasan mengapa begitu rumit adalah saya menggunakannya dalam banyak situasi berbeda untuk banyak proyek berbeda.

Sign.bat

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Signs the project output.

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Usage
:: 
:: Post-build event command line:
:: Call "$(ProjectDir)..\..\Sign.bat" "$(ConfigurationName)" "$(TargetPath)"
:: 
:: Run the post-build event:
:: When the build updates the project output

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Input Parameters
:: 
:: %~1   $(ConfigurationName)    The file's configuration.  This function will
::                               use a different certificate for "Debug"
::                               configurations.
:: %~2   $(TargetPath)           The full path of the first file to sign.
:: %~3+  FileName                The names of the remaining files to sign.
::                               These files must reside in the same directory
::                               as %2.

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Validate the parameters.
If "%~1"=="Debug" Exit /B 0
If "%~1"=="" Goto Error
If "%~2"=="" Goto Error
Goto Valid

:Error
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Report that the syntax is incorrect.
Echo One or more parameters are missing.
Echo.
Echo %~nx0 configuration filename1 [filename2 ...]
Echo.
Echo configuration      The project configuration.  Usually "Debug" or "Release".
Echo filename1          The full path of the first file to sign.
Echo filename2          The names of addition files to sign.  These files must
Echo                    reside in the same folder as "filename1".
Echo.
Exit /B 1

:Valid
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Change to the assembly's folder.
%~d2
CD %~dp2

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Prepare the list of files to sign.
Set FileList=
:CreateFileList
Set FileList=%FileList% %~snx2
Shift /2
If Not "%~2"=="" Goto CreateFileList

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Sign the assembly files.
Set Store=my
Set Certificate=type the name of your certificate here
Set TimeStampUrl=http://timestamp.verisign.com/scripts/timestamp.dll
C:\WinDDK\7600.16385.1\bin\x86\SignTool.exe Sign /s "%Store%" /n "%Certificate%" /t "%TimeStampUrl%" %FileList%
If %ErrorLevel%==1 Exit /B 1

::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
:: Verify the digital signature is valid.
C:\WinDDK\7600.16385.1\bin\x86\SignTool.exe Verify /pa %FileList%

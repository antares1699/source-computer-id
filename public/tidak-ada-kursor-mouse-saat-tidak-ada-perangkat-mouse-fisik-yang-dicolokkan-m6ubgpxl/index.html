Tidak ada kursor mouse saat tidak ada perangkat mouse fisik yang dicolokkan

Saya memiliki PC mini yang terhubung ke TV saya dengan Windows 10. Saya "mengontrolnya" dengan VNC. TightVNC diinstal sebagai layanan di PC. Klien saya adalah Hippo LITE di iOS. Ini berfungsi dengan baik ketika ada mouse yang dicolokkan ke PC (meskipun saya tidak menggunakan mouse, tetapi klien VNC saya). Ketika tidak ada perangkat mouse yang terpasang di pc, saya tidak dapat melihat kursor. Kursor menghilang segera setelah mouse dicabut.

Bagaimana cara mempertahankan kursor saat tidak ada mouse yang dicolokkan?

Anda dapat mencoba mengaktifkan fitur tombol mouse (jika diaktifkan, kursor mouse selalu ditampilkan):

  1. Buka Panel Kontrol
  2. Buka Pusat Kemudahan Akses
  3. Klik Jadikan mouse lebih mudah digunakan
  4. Klik pada kotak Turn on Mouse Keys

Jadi saya melakukan beberapa pengujian, orang telah memposting solusi yang benar di sini tetapi urutan operasi membuat perbedaan besar.

Pertama, beberapa poin yang harus dilakukan:

Ada dua cara untuk mengaktifkan / menonaktifkan Tombol Mouse:

  1. Di panel Pengaturan ui:

  2. Setelah Tombol Mouse diaktifkan seperti yang ditunjukkan di atas, Anda dapat mengaktifkan / menonaktifkannya dengan penekanan tombol yang berbeda tergantung pada bagaimana Tombol Mouse diatur:

Untuk membuat segalanya semakin membingungkan, antarmuka Panel Kontrol gaya lama memiliki beberapa opsi yang sedikit berbeda. Saya mengakses panel ini dengan mengklik dua kali pada ikon baki Tombol Mouse:

Saat mengaktifkan / menonaktifkan Tombol Mouse gunakan jendela Pengaturan yang lebih baru, bukan jendela Panel Kontrol yang lebih lama. Memeriksa kotak centang nonaktifkan / aktifkan di UI lama sepertinya tidak berfungsi.

Jadi dalam langkah-langkah berikut ketika saya berbicara tentang menonaktifkan Tombol Mouse, maksud saya metode # 1 di ui Pengaturan , bukan # 2. Anda dapat mengetahui apakah Tombol Mouse dinonaktifkan dengan benar ketika tidak ada lagi ikon baki untuk itu.

Hal utama berikutnya yang didapat adalah tindakan menghapus mouse secara fisik adalah penyebab windows menyembunyikan kursor. Bahkan jika Anda mendapatkan konfigurasi Tombol Mouse dengan benar, segera setelah Anda mencabut mouse secara fisik, kursor mungkin akan hilang.

Satu-satunya cara untuk membuatnya muncul kembali menggunakan Tombol Mouse adalah dengan Menonaktifkan -> Mengaktifkan kembali seluruh sistem Tombol Mouse, yaitu metode # 1 di atas. Mengalihkannya dengan Num-Lock saja tidak akan berhasil.

Triknya adalah ini harus dilakukan setelah Anda mencabut mouse Anda.

Begitu:

  1. Sepenuhnya Nonaktifkan Tombol Mouse di Pengaturan.
  2. Cabut perangkat penunjuk apa pun
  3. Aktifkan Tombol Mouse di Pengaturan.

Anda dapat menggunakan tombol tab dan panah untuk menavigasi ke ui yang benar tanpa mouse. Setelah Anda menyorot penggunaan tombol Nyala / Mati Mouse spasi untuk mengaktifkan / menonaktifkan.

Saya telah menguji semua jawaban yang diberikan, namun, solusi sebenarnya (diuji pada pengaturan yang bersih) adalah:

  1. WIN + R -> ms-settings:tabletmode -> Pilih "Bawa saya ke mode desktop / desktop" & "Jangan tanya saya dan jangan beralih"
  2. WIN + R -> control access.cpl -> "Jadikan mouse lebih mudah digunakan" -> Nyalakan "Kotak Tombol Mouse"
  3. Tekan "SHIFT KIRI + ALT KIRI + BLOQ NUM" beberapa kali sampai popup jelek muncul meminta Anda untuk (lagi) Mengaktifkan "Tombol Mouse", klik Ya.

Coba lepaskan mouse Anda, sekarang seharusnya tetap terlihat :)

Bagaimana Memperbaiki Masalah Mouse yang Hilang?

Untuk memperbaiki masalah hilangnya penunjuk tetikus, petunjuk langkah demi langkah yang diberikan di bawah ini harus diikuti:

1. Login dengan akun administrator.

2. Klik ubin Desktop dari layar Mulai untuk membuka jendela desktop.

3. Setelah di layar desktop, arahkan mouse ke sudut kanan bawah.

4. Dari opsi yang ditampilkan, klik Pengaturan.

5. Dari panel Pengaturan, klik Panel Kontrol.

6. Pada jendela Control Panel, klik kategori Hardware and Sound.

7. Pada jendela Hardware and Sound, klik Mouse di bawah kategori Devices and Printers dari panel kanan.

8. Pada kotak Mouse Properties yang ditampilkan, pergi ke tab Pointer Options.

9. Pada tab yang dipilih, hapus centang Sembunyikan penunjuk saat mengetik kotak centang di bawah bagian Visibilitas.

10. Setelah selesai, klik OK untuk menyimpan perubahan yang dimodifikasi.

11. Tutup jendela Perangkat Keras dan Suara setelah selesai.

Diambil dari: http://www.tomshardware.com/faq/id-1656322/windows-mouse-cursor-disappearing.html

Kemas ulang file 7z ke file zip di linux

Saya memiliki direktori dengan banyak file .7z, saya perlu mengemas ulang masing-masing sebagai arsip zip, bagaimana saya dapat melakukan ini secara otomatis di linux?

Gunakan skrip berikut dan jalankan dari direktori tempat file .7z Anda berada:

#!/bin/bash

TMPDIR=tempdir_$$

for x in `ls *.7z`; do
    mkdir $TMPDIR
    cd $TMPDIR
    cp ../$x .
    p7zip -d $x
    zip -r ../${x%.7z}.zip *
    cd ..
    rm -rf $TMPDIR    
done

Ini akan meninggalkan .7z file Anda di tempatnya dan membuat .zip file dengan nama yang sama.

Skrip menyalin file .7z ke direktori sementara sebelum mengekstraknya karena biasanya dihapus setelah mendekompresi file.

Saya telah membuat skrip sesederhana mungkin sehingga Anda dapat dengan mudah mengetahui cara kerjanya.

Skrip hanya akan bekerja dengan .7z file yang tidak memiliki spasi atau karakter khusus lainnya dalam namanya.

Anda juga dapat menggunakan alat arepack yang merupakan bagian dari rangkaian alat atool. Biasanya ini adalah yum install atool atau apt install atool. Anda mungkin juga perlu menginstal paket p7zip-full dan p7zip-rar jika belum diinstal. Setelah ini tersedia, Anda dapat melakukan ini:

$ arepack -e -F zip *.7z

Ini akan mengonversi semua file .7z sebagai file .zip. Anda masih perlu menghapus file * .7z, tetapi itu dapat dilakukan seperti ini:

$ rm -f *.7z

  -F, --format=EXT         override archive format (see below)

  -V, --verbosity=LEVEL    specify verbosity (0, 1 or 2)
  -p, --page               send output through pager
  -0, --null               filenames from standard in are null-byte separated
  -E, --explain            explain what is being done by atool
  -S, --simulate           simulation mode - no filesystem changes are made
  -o, --option=KEY=VALUE   override a configuration option

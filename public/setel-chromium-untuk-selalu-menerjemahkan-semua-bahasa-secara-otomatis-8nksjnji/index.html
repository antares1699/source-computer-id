Setel Chromium untuk selalu menerjemahkan semua bahasa secara otomatis

Adakah cara untuk menyetel Chromium agar selalu otomatis menerjemahkan situs web dalam semua bahasa, jadi saya tidak perlu melakukan "Selalu terjemahkan situs web dalam LANGUAGE" setiap kali saya menemukan situs web dalam bahasa baru? Saya menggunakan Mac OS X.

Ini adalah pertanyaan lama, tetapi terus muncul ketika saya sedang mencari cara melakukan ini, dan akhirnya saya menemukan caranya.

Saya mengubah file preferensi untuk menambahkan setiap bahasa ke daftar putih terjemahan Chrome, yang merupakan preferensi yang memilih perubahan "selalu terjemahkan". Berikut langkah-langkah untuk melakukannya:

  1. Tutup Chrome, jika sedang berjalan. Saya tidak yakin itu mentolerir monyet dengan file preferensi saat sedang berjalan.
  2. Temukan translate_language_list.cc file di sumber Chromium. Anda bisa mengambilnya dari Web.
  3. Salin konten array kDefaultSupportedLanguages ke editor teks favorit Anda dengan fungsi cari-ganti yang layak.
  4. Gunakan editor untuk mengganti koma dan semuanya setelahnya di setiap baris dengan : "en", (dengan asumsi Anda ingin selalu menerjemahkan ke dalam bahasa Inggris). Anda dapat menggunakan file preferensi Anda sebagai contoh (lihat langkah berikutnya) jika ada kebingungan pada formatnya.
  5. Temukan file preferensi Chrome Anda (milik saya di ~ / .config / google-chrome / Default / Preferences, di Linux) dan ganti konten array translate_whitelists (ada di dekat bagian bawah, di milik saya) dengan yang baru saja kita buat, simpan , dan (kembali) mulai Chrome.

Saya juga mengubah array translate_accepted_count di file preferensi agar memiliki angka positif untuk setiap bahasa, tetapi saya rasa itu tidak perlu.

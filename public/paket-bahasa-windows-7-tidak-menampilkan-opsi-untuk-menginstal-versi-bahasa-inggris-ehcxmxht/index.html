Catatan : Itu Paket MUI bahasa Inggris untuk Windows 7 dapat diperoleh dari Pembaruan Windows secara langsung.

Mengganti paket MUI yang sudah diinstal

Ini orang Jerman Wilayah dan bahasa (Region und Sprache). Di sini Anda dapat beralih di antara bahasa tampilan yang diinstal:

Jika Anda ingin menghapus paket MUI

Harap dicatat, menghapus paket bahasa seharusnya tidak diperlukan untuk bisa beralih ke bahasa Inggris. Namun, jika Anda ingin menghapus paket, begini caranya:

  1. Buka Wilayah dan bahasa panel kontrol dan klik Instal / hapus instalan bahasa ...

  2. Klik Copot pemasangan bahasa tampilan

  3. Copot pemasangan paket bahasa Jerman (jika sudah terpasang):

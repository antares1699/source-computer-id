Folder startup Windows 10 hilang

Mengapa saya tidak melihat folder startup di menu mulai Windows 10 saya?

Saya menggulir ke bawah menurut urutan abjad - R, S dan tidak ada Startup di sana. Jika saya membuka %AppData%\Microsoft\Windows\Start Menu\Programs\Startup saya dapat melihat foldernya ada di sana dan saya memiliki dua item di dalamnya. Bagaimana cara memperbaiki menu mulai agar juga ada sebagaimana mestinya?

Pergi ke beberapa struktur folder dalam di bawah% appdata% mencari folder ini sebenarnya bukan cara yang mudah digunakan ...

  1. Buka Explorer ke keduanya Memulai folder:

    • C: \ Pengguna \ pengguna \ AppData \ Roaming \ Microsoft \ Windows \ Start Menu \ Programs \ Startup
    • C: \ ProgramData \ Microsoft \ Windows \ Start Menu \ Programs \ Startup

    Klik kanan pada folder, seret ke Desktop dan buat pintasan, lalu tambahkan pintasan ke Mulailah menu "Semua Aplikasi" .

  2. Mungkin lebih nyaman menggunakan pengganti pihak ketiga untuk yang baru, kurang berguna, Mulailah menu, seperti Shell Klasik Menu Mulai , atau cabang baru dari aplikasi itu, Buka Shell ( Shell Klasik pengembangan telah berhenti, meskipun bekerja dengan baik pada WinVer 1809, Build 17763.504; tetapi penggunaan di masa mendatang mungkin membutuhkan Buka Shell ). Lanjutkan seperti di atas, dan letakkan pintasan di tempat yang nyaman. Ini memiliki keuntungan karena menyediakan GUI yang disesuaikan untuk anda spesifikasi yang tidak berubah pada setiap pembaruan MS Windows. Contoh pengaturan yang saya suka ada di bawah ini.

Microsoft telah membuat perubahan yang lebih konyol untuk membuat Windows lebih "ramah pengguna". Menurut pendapat mereka, pengguna lebih kesulitan melihat folder startup di menu start mereka, daripada mendapatkan keuntungan.

Ya saya tahu. Ini lembu jantan. Itu tidak benar, tapi rupanya itulah visi mereka.

Sepengetahuan saya tidak ada cara untuk mengembalikan folder ini ke menu awal. Anda selalu dapat membuat pintasan atau menyematkan folder ke menu mulai Anda (untuk menyematkan, pertama naik satu folder, lalu klik kanan dan pilih pin ke menu mulai).

Perhatikan, cara termudah untuk menuju ke folder itu, adalah dengan mengetik shell:startup di jendela File Explorer apa pun.

Folder startup khusus pengguna dapat diakses dengan shell:startup

Versi semua pengguna dapat diakses dengan shell:common startup

Ini dapat dimasukkan di bilah alamat Explorer atau kotak Jalankan.

(Ini harus bekerja pada semua versi Windows sejak setidaknya XP)

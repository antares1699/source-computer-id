Seperti yang dibahas sini , Windows Defender sepertinya menjadi penyebab yang membuat File Explorer menjadi sangat lambat digunakan.

Jadi saya ingin mematikannya sepenuhnya - menggunakan Real-time Protection off hanyalah mematikan sementara; Windows 10 akan mengaktifkannya kembali nanti.

Ambil kepemilikan dari kunci HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender. Sekarang setel nilai DisableAntiSpyware ke 1.

Ini menghentikan Windows Defender.

p.s. Tautan unduhan untuk alatnya.

Di Windows 10 Build yang lebih baru (1703, 1709), ini tidak berfungsi lagi. Windows mencegah pembuatan kunci di bawah HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender.

Di sini, buka HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender, buat DWORD DisableAntiSpyware dan setel ke 1:

Sekarang reboot dan Defender tidak boleh mulai lagi.

Perhatian: DisableAntiSpyware diabaikan untuk Windows 10 (1903 dan versi yang lebih baru) sejak Agustus 2020:

DisableAntiSpyware dimaksudkan untuk digunakan oleh OEM dan Profesional TI untuk nonaktifkan Antivirus Pertahanan Microsoft dan gunakan antivirus lain produk selama penerapan. Ini adalah pengaturan lama yang tidak lagi diperlukan karena antivirus Microsoft Defender otomatis menyala sendiri mati ketika mendeteksi program antivirus lain. Pengaturan ini tidak ditujukan untuk perangkat konsumen, dan kami telah memutuskan untuk menghapusnya kunci registri. Perubahan ini disertakan dengan Microsoft Defender Platform antimalware versi 4.18.2007.8 dan KB 4052623 yang lebih tinggi. Edisi Enterprise E3 dan E5 akan dirilis di masa mendatang. Catatan bahwa pengaturan ini dilindungi oleh perlindungan gangguan. Perlindungan kerusakan tersedia di semua edisi Home dan Pro Windows 10 versi 1903 dan lebih tinggi dan diaktifkan secara default. Dampak dari Penghapusan DisableAntiSpyware terbatas pada versi Windows 10 sebelumnya 1903 menggunakan Microsoft Defender Antivirus.

Saya tidak yakin persis apa yang telah Anda teliti, tetapi berikut adalah beberapa solusi potensial cepat untuk Anda.

MATI SEMENTARA:

  1. Buka Windows Defender dengan mengklik tombol Start Gambar tombol Start. Di kotak pencarian, ketik Pembela, lalu dalam daftar hasil, klik Pembela Windows.
  2. Klik Alat, lalu klik Opsi.
  3. Klik Administrator, pilih atau kosongkan kotak centang Gunakan program ini, lalu klik Simpan. Izin administrator diperlukan Jika Anda dimintai kata sandi atau konfirmasi administrator, ketik kata sandi atau berikan konfirmasi.

TAUTAN: http://windows.microsoft.com/en-au/windows/turn-windows-defender-on-off#turn-windows-defender-on-off=windows-7

NONAKTIF PERMANEN:

Terima kasih banyak kepada TOPGUNDCP untuk solusi yang memungkinkan di sini.

PS. Saya belum mengujinya (karena saya tidak ingin menonaktifkan Windows Defender saya sendiri), tetapi ini sepertinya titik awal yang baik.

  1. Buka Admin Command Prompt dan ketik: gpedit.msc
  2. Manuver ke: Konfigurasi Komputer-> Template Administratif-> Komponen Windows-> Windows Defender
  3. Klik dua kali pada "Turn Off Windows Defender" dan pilih "Enabled" lalu klik "Apply"

PERINGATAN: Setelah menonaktifkan "Windows Defender", Anda mungkin tidak dapat mengaktifkannya kembali. Saya sarankan sebelum mencoba ini, buatlah gambar cadangan agar Anda dapat mengembalikannya seperti semula.

Seseorang dapat menonaktifkan perlindungan Real-time Windows Defender melalui aplikasi Pengaturan, tetapi ini hanya sementara, karena Windows akan mengaktifkannya kembali secara otomatis setelahnya reboot berikutnya.

Untuk mematikan perlindungan Real-time secara permanen, saya mencantumkan dua metode di bawah ini. Reboot diperlukan setelah masing-masing.

Metode 1: Menggunakan Editor Kebijakan Grup Lokal

Metode ini membutuhkan Windows 10 versi Pro atau lebih tinggi.

  • Buka Editor Kebijakan Grup Lokal (gpedit.msc)
  • Navigasi ke:
    Konfigurasi Komputer / Template Administratif / Komponen Windows / Antivirus Pembela Windows / Perlindungan Real-time
  • Klik dua kali Matikan perlindungan real-time
  • Pilih Diaktifkan
  • Klik baik .

Metode 2: Pengeditan registri

Metode ini akan berfungsi untuk semua versi Windows.

Buat dan jalankan file .reg berikut:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender\Real-Time Protection]
"DisableRealtimeMonitoring"=dword:00000001

Mengubah pengaturan ini mengharuskan perlindungan Tamper dimatikan.

Untuk informasi lebih lanjut tentang fitur ini lihat artikel Microsoft
Lindungi pengaturan keamanan dengan perlindungan kerusakan .

Untuk metode menonaktifkan fitur ini, lihat artikel
Cara Mengaktifkan atau Menonaktifkan Perlindungan Tamper untuk Windows Defender Antivirus di Windows 10 .

Namun, mematikan perlindungan Tamper tetapi tidak melalui aplikasi Pengaturan memerlukan kepemilikan kunci registri HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender\Features, yang mungkin menyebabkan masalah di masa mendatang dengan Pembaruan Windows.

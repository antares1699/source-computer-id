Mengisi warna secara otomatis sesuai nilai dalam sel di spreadsheet

Pertanyaan : Apakah mungkin untuk mengisi setiap sel dalam spreadsheet dengan warna yang nilai grayscale-nya sama dengan nilai sel?

Penjelasan:

Pertimbangkan saya memiliki spreadsheet sebagai berikut:

Semua nilai berada di antara 0-255 (yang sesuai dengan 256 warna abu-abu). Sekarang saya ingin mengisi setiap sel dengan warna yang nilainya adalah nilai dalam sel. Jadi keluaran yang diharapkan adalah sebagai berikut:

Selain itu, perhatikan bahwa warna font di setiap sel hitam atau putih sesuai dengan warna latar belakang.

Apakah ada metode sederhana di Excel untuk melakukan ini dalam beberapa langkah atau sekaligus?

Buat Makro yang melakukan iterasi melalui sel Anda, membaca nilai dalam sel dan menetapkannya ke warna seperti ini:

Range("A1:A6").Interior.Color = RGB(200,160,35)

TheUser1024 benar, dan memiliki suara saya.

Berikut ini solusi siap pakai:

Public Sub Demo()
  Dim cel As Range
  Dim myRange As Range

  Set myRange = Range("A1:E10")

  For Each cel In myRange
    cel.Interior.Color = RGB(cel.Value, cel.Value, cel.Value)
    If cel.Value > 127 Then
      cel.Font.Color = RGB(0, 0, 0)
    Else
      cel.Font.Color = RGB(255, 255, 255)
    End If
  Next cel
End Sub

Edit:

Kode ini tidak akan rusak jika ada string atau nilai negatif ...

Public Sub Demo()
  Dim cel As Range
  Dim greytone As Long
  Dim myRange As Range

  Set myRange = Range("A1:E10")

  For Each cel In myRange
    greytone = CLng(Abs(Val(cel.Value)))
    cel.Interior.Color = RGB(greytone, greytone, greytone)
    If greytone > 127 Then
      cel.Font.Color = RGB(0, 0, 0)
    Else
      cel.Font.Color = RGB(255, 255, 255)
    End If
  Next cel
End Sub

Tambahkan ekstensi, jpg, ke banyak file

Saya memiliki folder dengan banyak folder dengan banyak file dan mungkin lebih banyak folder dengan lebih banyak file, di mana beberapa file kehilangan ekstensinya. Saya yakin semuanya adalah jpg, tetapi saya bisa saja salah. Adakah ide bagaimana menambahkan kembali ekstensi ke semua file ini tanpa melakukannya satu per satu?

Saya bisa melakukannya di Windows 7 atau Ubuntu 8.10.

Untuk Linux (atau MSWindows w / CygWin)

Jika Anda hanya ingin menambahkan sufiks ke file yang sebenarnya JPEG, coba ini:

$ find . -type f  ! -name '*.jpg'  -print | while read f
> do case "$(file "$f")" in
>    *JPEG*) mv -iv "$f" "$f.jpg" ;;
>    esac
> done

Yang akan:

  1. Jalur cetak yang berupa file tanpa a * .jpg akhiran ( Temukan ),
  2. Periksa konten file-file itu ( file $ f ),
  3. Untuk file JPEG, ganti namanya dengan _ .jpg akhiran.

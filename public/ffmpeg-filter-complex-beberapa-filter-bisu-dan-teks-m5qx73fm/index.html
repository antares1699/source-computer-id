ffmpeg filter_complex beberapa filter bisu dan teks

Saya memiliki 2 perintah ffmpeg bekerja dengan baik sendiri

ffmpeg -i test_video.mp4 -vcodec libx264 -filter_complex "drawtext=fontfile=test.ttf:text=awesome:fontcolor=white:fontsize=20:x=0:y=10+th"  test_video_text.mp4 -y

dan

ffmpeg -i test_video.mp4 -vcodec libx264 -filter_complex "volume=0"  test_video_text.mp4 -y

saat menggabungkan perintah ini sebagai

ffmpeg -i test_video.mp4 -vcodec libx264 -filter_complex "drawtext=fontfile=Lato-Light.ttf:text=awesome:fontcolor=white:fontsize=20:x=0:y=10+th,volume=0"  test_video_text.mp4 -y

Jika Anda memiliki filter kompleks dengan lebih dari satu input dan output, Anda perlu secara eksplisit menentukan aliran input dan output mana yang digunakan oleh filter mana. Selain itu, untuk memisahkan rantai filter, Anda harus menggunakan titik koma (;), bukan koma sederhana (,). Terakhir, Anda harus memetakan keluaran dari rantai filter tersebut ke file keluaran menggunakan opsi -map.

ffmpeg -y -i test_video.mp4 -vcodec libx264 -filter_complex \
  "[0:v]drawtext=fontfile=Lato-Light.ttf:text=awesome:fontcolor=white:fontsize=20:\
   x=0:y=10+th[outv];\
   [0:a]volume=0[outa]"\
   -map "[outv]" -map "[outa]"\
   test_video_text.mp4

Saya menyisipkan garis miring terbalik agar lebih mudah dibaca. Lihat filtergraph dan filter_complex dokumentasi untuk lebih jelasnya.

Selain itu, karena -y adalah opsi global, Anda harus memindahkannya ke awal perintah.

Apakah hanya saat Anda memulai? hal satu kali? Anda dapat mencoba menggunakan Process Explorer dari SysInternals (sekarang dimiliki oleh Microsoft) untuk mencoba melihat apakah program baru diluncurkan tepat sebelum dimulai. Tidak seperti Task Manager normal di Windows, Process Explorer menunjukkan program mana yang membuka dan menutup dengan menyorotnya.

Jika itu terjadi secara konsisten, coba bunuh program yang tidak benar-benar Anda kenali dengan pengelola tugas atau penjelajah proses sampai program tersebut hilang. Jika ini hanya terjadi selama startup, jalankan msconfig dari dialog Run Anda (tekan WINKEY + R), buka tab startup, dan mulai hapus item startup dan mulai ulang hingga masalahnya hilang.

Apakah ini terlihat seperti jendela perintah hitam? Jika demikian, mungkin skrip startup yang sedang berjalan.

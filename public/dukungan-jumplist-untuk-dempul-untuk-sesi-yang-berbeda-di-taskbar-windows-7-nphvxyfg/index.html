Dukungan Jumplist untuk dempul untuk sesi yang berbeda di taskbar windows 7

Saya bertanya-tanya apakah ada yang tahu bagaimana melakukan dukungan jumplist untuk sesi dempul yang berbeda di taskbar windows 7?

Penelitian saya, yang diposting di komentar jawaban Diago, membawa saya ke entri blog ini: Putty dengan dukungan jumplist Windows 7

Ini oleh orang yang sama yang mengirimkan entri daftar keinginan dan tambalan .

Ada sebuah versi pra-kompilasi dari Putty dengan dukungan jumplist di bagian bawah pos. Bekerja dengan baik untuk saya di Windows 7 Ultimate 64-bit.

Dukungan untuk jumplist Windows 7 telah ditambahkan ke versi resmi 0,61 (dirilis pada 2011-07-12, lebih dari 4 tahun setelah 0,60!).

Dukungan untuk fitur antarmuka pengguna baru Windows 7. Manajemen jendela Aero baru sekarang harus bermain dengan baik dengan penanganan pengubahan ukuran jendela PuTTY yang rumit, dan daftar lompat Windows 7 sekarang didukung sehingga Anda dapat meluncurkan sesi yang disimpan langsung dari bilah tugas.

Catatan Rilis bisa dibaca sini dan binari dapat diunduh sini .

Dukungan jump-list adalah fitur aplikasi dan oleh karena itu hanya dapat diterapkan oleh pengembang. Aplikasi harus memiliki kemampuan untuk melacak file yang baru-baru ini digunakan, dan ada kumpulan API pengembang khusus yang dapat digunakan untuk mendukung fitur tersebut.

Saya tidak dapat mengingat apakah Putty masih aktif dikembangkan, tetapi Anda kemungkinan besar perlu membuat permintaan fitur untuk ini. Seperti yang ditunjukkan oleh Snark ada permintaan untuk ini sini sudah, dan kode sebenarnya tersedia sini jika Anda bersedia untuk menyusun sendiri.

Kesalahan berikut terus berulang bahkan setelah saya mencoba yang disarankan:

rm .git/index
git reset

Tapi saya masih mendapatkan kesalahan ini:

fatal: index file corrupt
fatal: 'git status --porcelain' failed in submodule '[directory path omitted]'

Apa yang salah? Apa yang harus saya lakukan?

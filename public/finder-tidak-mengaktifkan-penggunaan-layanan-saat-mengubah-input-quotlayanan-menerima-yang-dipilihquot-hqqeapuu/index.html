Finder tidak mengaktifkan penggunaan Layanan saat mengubah input "Layanan menerima yang dipilih"

Saya memiliki Layanan pada automator yang berlaku untuk File dan Folder di Finder. Awalnya ini disimpan sebagai input "Teks".

Masalahnya, Finder tidak akan menampilkan Layanan ini ke File dan Folder. Jika saya pergi ke pengaturan, itu masih ditempatkan di bawah kategori "teks".

OS X terkadang mengabaikan perubahan pada layanan setelah disimpan di Automator, kemungkinan karena cache. Jadi jika layanan dimulai sebagai file teks layanan, kemungkinan perubahan Anda tidak diambil. Untuk menghindari masalah ini, pilih File »Duplikat di Automator dan simpan layanan dengan nama baru, dan coba lagi. Kemudian hapus layanan asli dari ~/Library/Services.

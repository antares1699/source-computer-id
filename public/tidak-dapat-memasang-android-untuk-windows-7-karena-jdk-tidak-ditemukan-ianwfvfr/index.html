Tidak dapat memasang Android untuk Windows 7 karena JDK tidak ditemukan

Saya telah menginstal JDK SE7u4 di windows 7. Saya mencoba menginstal Android di Windows 7 menggunakan installer_r18-windows.exe tetapi tidak berhasil.

Jendela pengaturan menunjukkan pesan yang mengatakan bahwa Java SE JDK tidak ditemukan dan

EROR: Gagal menemukan versi Java untuk C: \ Windows \ System32 \ java.exe
Sistem tidak dapat menemukan file yang ditentukan.

Jika Anda yakin Anda telah menginstal JDK dan tidak dikenali dengan benar, cukup setel variabel lingkungan JAVA_HOME untuk diarahkan ke sana

Saya mengklik kanan Komputer> Properti> Pengaturan Sistem Lanjutan> Variabel lingkungan . Saya membuat variabel sistem BARU yang disebut JAVA_HOME dan menetapkan nilainya sebagai C:\Program Files\Java\jdk1.7.0_04\bin atau C:\Windows\System32\java.exe.

Saya menjalankan penginstal lagi - pesan kesalahan yang sama masih muncul.

Anda mungkin memiliki masalah yang berbeda, tetapi berikut adalah solusi yang saya temukan untuk pesan kesalahan yang sama:

  1. Buka C:\Windows\system32
  2. Ubah nama java.exe menjadi yang lain, misalnya java2.exe
  3. Jalankan penyiapan Android SDK lagi

Terimakasih untuk: http://www.androidbeats.in/android-sdk-failed-to-find-java-exe/4251/

Ini membuat saya gila selama beberapa hari. Saya tidak dapat menyelesaikan masalah melalui penginstal studio Android, tetapi menemukan solusi:

  • Pergi ke http://developer.android.com/sdk/index.html
  • Daripada mendownload "paket Android Studio" (file exe), scroll ke bawah dan pilih "SDK tools package" (file zip).
  • Ini berfungsi dengan sempurna dan memungkinkan Anda untuk memilih semua paket yang ingin Anda unduh, termasuk paket yang diinstal oleh penginstal.

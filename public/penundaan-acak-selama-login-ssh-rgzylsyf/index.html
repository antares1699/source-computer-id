Ini adalah masalah umum di Ubuntu. Anda harus menonaktifkan pencarian DNS terbalik di server Ubuntu Anda dengan menyetel UseDNS ke no di /etc/ssh/sshd_config. Ada banyak artikel tentang topik ini, ini hanyalah salah satunya.

Di Linux dengan GNU grep :

grep -F --text -o --byte-offset mystring binaryfile

Contoh:

$ grep -F --text -o --byte-offset option /bin/tar
226542:option
237529:option
237612:option
...

Penjelasan parameter:

-F, --fixed-strings
      Interpret PATTERN as a list of fixed strings, separated by newlines,
      any of which is to be matched.  (-F is specified by POSIX.)

-a, --text
      Process  a binary file as if it were text; this is equivalent to the
      --binary-files=text option.

-o, --only-matching
      Print  only the matched (non-empty) parts of a matching line, with
      each such part on a separate output line.

-b, --byte-offset
      Print the 0-based byte offset within the input file before each line
      of output. If -o (--only-matching) is specified, print the offset of
      the matching part itself

Untuk menghitung kejadian, tambahkan | wc -l ke baris perintah.

Di Linux, Anda dapat menggunakan strings -a [filename] | grep [string] | wc -l

Di Windows, Anda dapat menggunakan strings * | findstr /i [string], dengan utilitas string dari Sysinternals.

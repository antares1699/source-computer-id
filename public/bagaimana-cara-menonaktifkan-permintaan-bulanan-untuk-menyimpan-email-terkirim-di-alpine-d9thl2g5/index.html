Di bawah Setup> Config, lihat Aturan Pemangkasan hampir sampai ke bawah.

Dari bantuan dalam alpine:

OPSI: Aturan Pemangkasan

Secara default, Alpine akan menanyakan pada awal setiap bulan apakah atau tidak Anda ingin mengubah nama folder email terkirim Anda menjadi nama seperti dikirim-surat-bulan-tahun. (Lihat fitur Pangkas Menggunakan YYYY-MM untuk mengubah format folder ke sent-mail-yyyy-mm.) Ini juga akan menanyakan apakah Anda ingin menghapus folder surat terkirim lama. Jika Anda sudah mendefinisikan Baca Pesan Folder atau Pruned Folder Alpine juga akan menanyakan tentang pemangkasan folder tersebut.

Dengan opsi ini Anda dapat memberikan jawaban otomatis untuk pertanyaan-pertanyaan ini. Itu nilai defaultnya adalah menanyakan apa yang ingin Anda lakukan.

Enam kemungkinan nilai untuk opsi ini adalah:

bertanya tentang mengganti nama, bertanya tentang menghapus
Ini adalah defaultnya. Alpine akan menanyakan apakah Anda ingin mengganti nama file folder dan apakah Anda ingin menghapus setiap folder lama.

tanyakan tentang ganti nama, jangan hapus
Alpine akan menanyakan apakah Anda ingin mengganti nama folder, tetapi tidak akan bertanya tentang atau menghapus folder lama.

selalu ganti nama, tanyakan tentang menghapus
Ini berarti Anda ingin selalu menjawab ya dan memiliki Alpine secara otomatis ganti nama folder jika memungkinkan. Anda juga akan ditanya tentang penghapusan folder lama.

selalu ganti nama, jangan hapus
Ini berarti Anda ingin selalu menjawab ya dan memiliki Alpine secara otomatis ganti nama folder jika memungkinkan. Tidak akan ada penghapusan yang lama folder.

jangan ganti nama, tanyakan tentang menghapus
Artinya Anda ingin selalu menjawab tidak. Alpine tidak akan mengganti nama file map. Anda akan ditanya tentang menghapus folder lama.

jangan ganti nama, jangan hapus
Artinya Anda ingin selalu menjawab tidak. Alpine tidak akan mengganti nama file map. Folder lama juga tidak akan dihapus.

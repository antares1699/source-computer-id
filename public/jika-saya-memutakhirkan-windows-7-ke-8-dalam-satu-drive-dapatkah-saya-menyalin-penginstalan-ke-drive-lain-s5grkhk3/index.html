Jika saya memutakhirkan windows 7 ke 8 dalam satu drive, dapatkah saya menyalin penginstalan ke drive lain?

Saya ingin membeli drive SSD untuk menginstal windows 8, tapi saya sangat penasaran. Jadi jika saya memutakhirkan Windows 7 ke 8, dapatkah saya membeli drive SSD di masa mendatang dan menyalin instalasi saya ke dalamnya?

Karena cara kerja partisi NTFS, Anda perlu memastikan bahwa partisi di drive lama cukup kecil untuk direplikasi ke SSD.

Kemudian Anda dapat menggunakan sesuatu seperti Clonezilla atau beberapa utilitas kloning drive lain untuk memindahkan data ke drive baru.

Selama tidak ada hal lain pada perangkat keras mesin yang berubah, Anda seharusnya tidak mengalami masalah.

Ini rumit, dan sedikit berisiko jika Anda tidak mencadangkan data Anda terlebih dahulu, tetapi mungkin saja.

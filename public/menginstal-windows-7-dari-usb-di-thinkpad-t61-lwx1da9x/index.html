Menginstal Windows 7 dari USB di Thinkpad T61

Saya mencoba menginstal Windows 7 Professional dari USB 3.0 flashdrive, di Thinkpad T61. Masalahnya adalah, Thinkpads BIOS tidak akan mendeteksi flash drive sebagai media yang dapat di-boot, dan tidak memungkinkan untuk melakukan boot dari itu.

Apa yang saya lakukan:

Dukungan USB BIOS yang Diaktifkan di BIOS (diaktifkan secara default)

Di menu startup, tambahkan HDD USB ke urutan boot (ada tanda '-' di depannya)

Membuat media instalasi Windows 7 dengan UNetbootin, WinUSB (alat linux) dd dan Grub4DOS.

Seperti yang Anda ketahui, saat ini, saya hanya memiliki akses ke mesin Linux untuk membuat flashdisk.

Apa yang terjadi:

T61 BIOS menampilkan '-USB HDD' dalam menu urutan booting. Tanda '-' menunjukkan bahwa flash drive yang terpasang saat ini tidak dapat di-boot. Flashdrive yang sama (dengan gambar Windows yang sama di atasnya) melakukan booting tanpa masalah pada Dell D430 dan Lenovo Y550. Juga, Ubuntu 12.04 menginstal USB yang dibuat dengan Unetbootin ditampilkan sebagai bootable (tanda '+' di menu urutan boot BIOS) dan boot dari menu boot F12.

informasi tambahan

thinkwiki.org mengatakan bahwa beberapa BIOS Thinkpad tidak menggunakan MBR pada flashdisk. Ini menyarankan penggunaan pemuat boot Extended-IPL, tetapi tautan yang disediakan rusak dan tampaknya tidak ada mirror.

Ini mungkin hanya solusi, tetapi masalahnya cukup spesifik, bahwa solusi solusi itu cukup baik:

Seperti yang disarankan MrJackV, saya mencoba mem-boot PLOP Linux dan menggunakan manajer bootnya untuk mem-boot flashdisk Windows saya. Ternyata dengan flashdrive PLOP dicolokkan, dan Flashdrive Windows 7, BIOS dapat melihat kedua drive di menu pemilihan boot dan memungkinkan saya untuk mem-boot penginstal Windows 7. Melakukannya lagi, tetapi dengan flashdisk Windows 7 saja, tidak ada flashdisk, sama seperti sebelumnya.

EDIT: Jika seseorang menghadapi masalah berikut

Metode ini menyebabkan masalah yang tidak terduga - pengaturan Windows 7 tidak memungkinkan saya untuk menginstal OS pada drive. File log mengungkapkan, bahwa drive yang saya coba instal bukanlah drive boot utama. Menggunakan alat yang disediakan dengan utilitas pengaturan, saya membuat partisi utama yang dapat di-boot pada drive internal, menyalin file pengaturan dan mem-boot darinya, dengan drive internal menjadi satu-satunya yang dicolokkan ke sistem.

Saya mengalami masalah serupa pada Thinkpad X200 saat drive USB disiapkan dengan alat MS. Laptop tidak bisa boot, hanya menampilkan kursor yang berkedip. Masalahnya adalah bahwa alat MS memformat drive USB sebagai NTFS, yang tidak dikenali oleh semua BIOS. Ketika saya menyiapkan drive USB secara manual, memformatnya sebagai FAT32 dan kemudian membongkar file Windows 7 ISO di sana (instruksi ditemukan di internet), akhirnya boot dan dengan cepat menginstal Windows 7 di HDD dengan lancar. Drive USB disiapkan menggunakan laptop Windows 7 yang berbeda, karena X200 memiliki HDD yang dihapus, hanya dengan format NTFS - dengan Parted Magic.

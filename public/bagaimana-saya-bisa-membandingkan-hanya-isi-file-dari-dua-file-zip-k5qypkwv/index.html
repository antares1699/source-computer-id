Bagaimana saya bisa membandingkan hanya isi file dari dua file zip

Saya membuat dua file zip ini dengan jarak sekitar 24 jam.

$ ls -l /tmp/*.zip
-rw-rw-r-- 1 scott scott 1401289 Mar 21 20:11 /tmp/inc1.zip
-rw-rw-r-- 1 scott scott 1401289 Mar 22 19:29 /tmp/inc2.zip

Anda dapat melihat ukurannya sama, karena direktori yang sedang di-zip tidak berubah.

Namun, file-file ini telah dilihat dalam 24 jam terakhir, jadi waktu aksesnya telah berubah. Untuk alasan ini, saya tidak dapat membandingkan ritsleting menggunakan diff atau cmp.

Apa cara yang andal untuk membandingkan isinya? Misalnya, saya sedang mempertimbangkan untuk membandingkan keluaran

ls -l <file> | awk '{print $5}'

pada dua file. Apakah ada solusi yang lebih baik?

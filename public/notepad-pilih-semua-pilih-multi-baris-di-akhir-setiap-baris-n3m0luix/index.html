Notepad ++ pilih semua, pilih multi-baris di akhir setiap baris

Dalam sublimetext saya sering menggunakan kombinasi tombol CTRL + A, CTRL + SHIFT + L, END untuk sering menyorot semua baris dan melanjutkan mengedit setiap baris dengan teks umum. (Lihat gambar)

Apa kombinasi tombol untuk ini di notepad ++ atau tidak bisakah Anda melakukan ini sama sekali di notepad ++?

Notepad ++ tidak memiliki satu opsi / pintasan untuk dengan mudah mendapatkan banyak tanda sisipan di akhir beberapa baris.

Trik yang saya gunakan itu:

  • CTRL + klik di setiap baris (tidak masalah di bagian mana baris yang Anda klik) Anda ingin menambahkan satu tanda sisipan di akhir.
  • Saat Anda menempatkan file kursor pada setiap baris yang ingin Anda edit, tekan AKHIR kunci untuk pindahkan semua tanda sisipan ke akhir baris tempat mereka ditempatkan.

Anda dapat melakukan ini dengan mode kolom. Fungsionalitas ini diaktifkan secara default. Untuk beralih ke mode kolom, ada dua cara untuk melakukannya: Alt + Shift + Tombol panah atau: Alt + Mouse menyeret Untuk menggunakan keyboard, Anda perlu menggunakan tombol Alt + Shift + Panah. Dengan menggunakan keyboard, Anda memiliki kontrol lebih terhadap kata atau karakter yang ingin Anda gunakan.

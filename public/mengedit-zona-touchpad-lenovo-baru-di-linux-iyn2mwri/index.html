Mengedit zona touchpad Lenovo baru di Linux

Baru saja mendapatkan Lenovo T440p - suka laptopnya, perangkat kerasnya hebat - dan saya sangat bersemangat untuk menggunakan Mint Linux di atasnya. Instal berfungsi dengan baik, tidak ada masalah show-stopper, tetapi ada satu gangguan terus-menerus dengan itu yang mendorong saya kembali ke Win7: Saya tidak dapat mengklik kanan atau klik tengah dengan Trackpoint, dan zona klik kanan bawah untuk touchpad menempati hampir seperempat permukaannya.

Karena Lenovos baru memiliki touchpad Synaptics yang menggantikan tombol lama untuk Trackpoint (gangguan saya dengan mengganti tombol fisik dengan tombol virtual adalah masalah yang sepenuhnya terpisah), perlu diberitahukan dalam perangkat lunak untuk mendeteksi klik di klik kanan dan tengah Area -klik sebenarnya adalah klik kanan dan tengah, dan tidak membaca klik di tempat tersebut sebagaimana seharusnya.

Saya akan sangat senang menghapus klik kanan bawah dan memiliki area klik kanan / tengah yang ramping di bagian atas untuk Trackpoint, tetapi saya tidak tahu cara memperbaikinya. Saya tidak cukup terbiasa dengan Linux untuk membuka isi OS sendiri (ini adalah pemuatan Linux murni pertama saya yang saya gunakan sehari-hari), dan saya hanya mencoba untuk mendapatkan ini sehingga saya dapat menerimanya. . Apakah ada solusi berbasis GUI untuk ini, seperti tab Thinkpad di kontrol Mouse Win7, atau adakah versi yang sudah diedit yang bisa saya salin / tempel?

Terima kasih untuk bantuannya!

Sunting: Setelah mengotak-atik lebih banyak dengan pengaturan Win7 Touchpad, sesuatu yang mirip dengan apa yang dibangun di Win7 juga akan baik-baik saja: gerakan satu jari pada touchpad tidak lagi menggerakkan kursor (Trackpoint hanya memindahkannya), klik kanan / tengah adalah dipulihkan ke Trackpoint, dan gerakan dua jari masih berfungsi di bidang sentuh untuk menggulir. Sekarang, saya harus menyebutkan bahwa saya tidak mempertimbangkan opsi "nonaktifkan Touchpad" di panel kontrol Linux sebelumnya karena saya berasumsi itu akan menonaktifkan touchpad sepenuhnya dan saya tidak akan dapat mengklik apa pun. Jika saya salah dalam hal itu, beri tahu saya dan saya akan dengan senang hati kembali ke Mint dan mengaturnya.

Saya mengambil risiko dan melakukan mengutak-atik file .conf yang saya coba hindari, dan menemukan apa yang perlu saya lakukan.

Di /usr/share/X11/xorg.conf.d/ adalah 50-synaptics.conf, yang merupakan file conf yang mengontrol touchpad pada T440p (dan lainnya). Dan di sana ada bagian ini:

Section "InputClass"
    Identifier "Default clickpad buttons"
    MatchDriver "synaptics"
    Option "SoftButtonAreas" "50% 0 82% 0 0 0 0 0"

Itu adalah pengaturan default untuk area di bidang sentuh yang sesuai dengan tombol klik kanan di bidang sentuh. Setelah banyak, banyak mengutak-atik untuk mendapatkan area yang didefinisikan ulang dengan benar, inilah yang harus disalin-tempel untuk mengubah area klik kanan menjadi strip tipis di area kanan atas touchpad:

Section "InputClass"
    Identifier "Default clickpad buttons"
    MatchDriver "synaptics"
    Option "SoftButtonAreas" "50% 0 0 45% 0 0 0 0"

Sejauh yang saya mampu lakukan saat ini, tetapi itu menyelesaikan sebagian besar dari apa yang saya inginkan. Saya hanya memposting ulang di sini kalau-kalau ada orang lain yang perlu melihat cara memetakan ulang zona di touchpad Synaptics.

Bagaimana cara melewati kata sandi pengaktifan Lenovo Thinkpad?

Kakak saya memiliki laptop Lenovo Thinkpad.

Dia memutuskan untuk memformat laptopnya, tetapi setelah format itu dia tidak dapat menyalakan laptopnya karena kata sandi penyalaan. Kata sandi diperlukan saat start-up.

Pertanyaan saya adalah: apa itu power-on password dan bagaimana kita memintasnya? Kami mencoba melepas baterai tetapi tidak berhasil.

Pada ThinkPad yang relatif baru, tidak ada cara lain untuk menggunakan supervisor password, selain mencoba menyingkat daya ke beberapa chip, yang di tangan yang tidak terlatih adalah cara yang baik untuk menggoreng motherboard sepenuhnya. Tidak seperti kebanyakan pengaturan level BIOS, termasuk kata sandi pada beberapa mesin lain, melepas baterai CMOS TIDAK akan mengatur ulang kata sandi supervisor di ThinkPad.

Obat Lenovo untuk ini adalah penggantian motherboard.

Power on password dapat diatur ulang dengan melepas baterai CMOS selama sepuluh menit. Manual untuk thinkpad Edge 11 & E10: http://download.lenovo.com/ibmdl/pub/pc/pccbbs/mobiles_pdf/75y6925_01.pdf

Menyetel ulang power-on password didokumentasikan di Hardware Maintenance Manual:

  1. Matikan komputer.
  2. Lepaskan unit baterai.
  3. Lepaskan baterai cadangan.
  4. Nyalakan komputer dan tunggu hingga POST berakhir. Setelah POST berakhir, prompt kata sandi tidak muncul. POP telah dihapus.
  5. Pasang kembali baterai cadangan dan unit baterai.

Ubuntu, mencari file dan juga mencari di semua sub-direktori?

Jika saya berada di jalur direktori tertentu, saya ingin mencari semua file di folder / sub-folder mana saja.

Bagaimana saya bisa melakukan ini?

Di terminal:

Untuk membuat daftar semua file di direktori saat ini dan semua sub-direktori:

$ find .

Untuk mencari file berdasarkan nama di direktori saat ini dan semua sub-direktori:

$ find . -name <file_name>

Atau untuk mencari di semua file di direktori saat ini dan semua sub-direktori untuk string tertentu:

$ grep -R "<string>" *

Jika Anda ingin mencari GUI dengan subdirektori, gunakan gnome-search-tool. Anda dapat menemukannya di Pusat Perangkat Lunak Ubuntu (12.04 - penguin yang tepat) atau Anda dapat menginstalnya langsung di terminal:

sudo apt-get install gnome-search-tool

Sarang HKCU diadakan di NTUSER.DAT (atau nama file serupa) di bawah direktori profil:

C:\Users\<NAME>\Ntuser.dat

Anda dapat mengaktifkan melihat file sistem atau saat membuka file di regedit pilih HKEY_LOCAL_MACHINE, Buka File »Load Hive , Browse ke C:\Users\<NAME>, Ketik Ntuser.dat ke dalam kotak nama file dan pilih Buka .

Anda berada di direktori yang benar, kumpulan registri adalah file tanpa ekstensi file, dengan pengecualian HKCU, yang merupakan file NTUSER, dat

http://ezinearticles.com/?Windows-XP-Registry-Files-Revealed---Structure-And-Location&id=763552

Di registri, ada sarang yang sesuai dengan beberapa file. Hives adalah kumpulan kunci, subkunci, dan nilai yang berisi konfigurasi untuk sistem operasi dan program. Di registri, ada 5 grup: HKEY_CLASSES_ROOT (HKCR), HKEY_CURRENT_USER (HKCU), HKEY_LOCAL_MACHINE (HKLM), HKEY_USERS (HKU), dan HKEY_CURRENT_CONFIG (HKCC).

Untuk sarang yang sesuai, Anda akan melihat beberapa jenis ekstensi: tanpa ekstensi, .alt, .log, dan .sav.

Tanpa Ekstensi: Kumpulan registri lengkap

.alt: Salinan alternatif dari kumpulan registri

.log: Log perubahan yang terjadi dalam kumpulan registri

.sav: Ini adalah cadangan otomatis yang dibuat dari saat pengaturan atau penginstalan program terjadi.

Berikut file-file yang sesuai:

HKCC: System, System.alt, System.log, System.sav

HKCU: Ntuser.dat, Ntuser.dat.log

HKLMSAM: Sam, Sam.log, Sam.sav

HKLMSecurity: Keamanan, Keamanan.log, Keamanan.sav

HKLMSoftware: Software, Software.log, Software.sav

HKLMSystem: Sistem, System.alt, System.log, System.sav

HKU.DEFAULT: Default, Default.log, Default.sav

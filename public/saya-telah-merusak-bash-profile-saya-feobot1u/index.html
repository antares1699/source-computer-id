Saya telah merusak .bash_profile saya

Saat mencoba menginstal sesuatu di mesin Mac OSX saya, saya rasa saya telah merusak file ~ / .bash_profile saya. Saya mengeditnya, dan tiba-tiba, sekarang, ketika saya membuka terminal, saya tidak dapat menjalankan apa pun, atau bahkan mencantumkan file:

$ ls
-bash: ls: command not found

Jalankan perintah ini untuk memperbaiki profil bash Anda, dengan mengganti nama pengguna Anda di:

/usr/bin/sudo /usr/bin/nano /Users/YOURUSERNAME/.bash_profile

Saat mengedit jalur Anda, jangan lupa untuk menyertakan : $ PATH ke akhir string jalur baru Anda. Seharusnya dibaca seperti ini:

export PATH = "/my/new/path:$PATH"

Ini kemungkinan penyebab masalah Anda.

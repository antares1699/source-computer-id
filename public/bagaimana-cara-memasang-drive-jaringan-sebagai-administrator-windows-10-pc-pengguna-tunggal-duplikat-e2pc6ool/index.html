Bagaimana cara memasang drive jaringan sebagai administrator Windows 10 PC pengguna tunggal [duplikat]

Di PC saya ketika saya harus memasang drive jaringan, itu tidak memberi saya opsi apa pun tentang apakah akan terhubung sebagai pengguna biasa atau administrator. Ini menyebabkan masalah di kemudian hari ketika saya mencoba dan mengakses drive melalui IDE saya (Integrated Developement Environment) karena IDE berjalan dengan izin administrator yang tinggi dan karena drive jaringan tidak dipasang sebagai administrator, aplikasi tidak memiliki izin untuk mengakses drive netork .

Masalahnya didokumentasikan di sini

https://support.microsoft.com/en-gb/help/3035277/mapped-drives-are-not-available-from-an-elevated-prompt-when-uac-is-co

Solusi 2 mengatakan untuk hanya memasangnya dari file bat sebagai administrator, tetapi tidak memberi saya instruksi tentang cara melakukan ini. Dan saya bingung apakah tidak ada cara untuk melakukan ini melalui UI Windows saat saya memasang drive?

  • Drive selalu dipetakan dalam konteks pengguna. Menjalankan sesuatu sebagai admin berarti menjalankannya sebagai pengguna yang berbeda, dan pengguna lain tersebut akan membutuhkan peta drive-nya sendiri. Ini bukan masalah atau kesalahan atau bukti dari apa pun yang rusak.
    — music2myear
  • @ music2myear saya mengerti ini bukan bug, tetapi pengguna saya memiliki izin admin dan saya tidak memiliki pengguna admin seaprate jadi bagaimana saya bisa memetakan drive sebagai pengguna terpisah itu.
    — Paul Taylor

Buat file batch yang berisi

net use x: \\server\share

Kemudian klik kanan dan jalankan sebagai administrator. Jika Anda ingin tetap ada setelah reboot, tambahkan perintah dengan /persistent:yes.

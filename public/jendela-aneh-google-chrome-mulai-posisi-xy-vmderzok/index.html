Jendela aneh Google Chrome mulai posisi xy

Saya telah mendapatkan masalah jendela Google Chrome yang aneh di mana posisi startup jendela sekarang hampir keluar dari layar. Saya tidak dapat menemukan file pengaturan apa pun di C:\Documents and Settings\UserName\Local Settings\Application Data\Google\Chrome\Application\3.0.195.38 untuk mengontrol posisi X dan Y dari jendela dan ini menjadi mengganggu sekarang. Setiap bantuan dihargai.

  • WinXP Pro
  • Chrome versi 3.0.195.38
  • Dell Inspirion 10 "netbook
  • Tidak ada layar eksternal atau ekstra yang pernah digunakan dengannya

Saya baru saja mengalami masalah serupa di mana jendela chrome akan keluar dari layar setiap kali saya memulainya kecuali saya memaksimalkannya. Mengedit posisi jendela di "C: \ Users \ UserName \ AppData \ Local \ Google \ Chrome \ User Data \ Local State", seperti yang disarankan taspeotis, tidak berfungsi - posisi yang salah selalu masuk ke dalamnya. Bahkan setelah menghapus seluruh file. (Dan ya: Saya menghentikan peramban setiap kali sebelum mengotak-atik file Status Lokal ini ...)

Inilah yang saya sebut jauh dari layar:

     "bottom": -31901,
     "left": -32000,
     "right": -31773,
     "top": -32000,

Apa yang memecahkannya bagi saya adalah memaksimalkannya (sehingga saya bisa melihat jendela) dan kemudian menyeret semua tab yang terbuka ke jendela baru.

Mungkin ini juga dapat membantu orang lain pada suatu saat :-)

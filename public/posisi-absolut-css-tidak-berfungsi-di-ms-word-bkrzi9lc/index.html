Posisi absolut CSS tidak berfungsi di MS Word

Ini adalah contohnya:

<html>
<head>
    <title>word test</title>
</head>
<body>
    <div style='position: absolute; width: 30px; height: 50px; top: 100px; left: 20px; border-color: black; border-width: 1px; border-style: solid;'>
        <p>Hello!</p>
    </div>
</body>
</html>

Simpan sebagai "word.doc" dan buka di MS WORD.

Posisi absolut tidak berfungsi!

Div ditampilkan di bagian atas dokumen dan dengan lebar 100%. Mengapa? Saya tidak bisa menggunakan tabel html.

Versi di ms word: 2003

Oke, saya punya masalah yang sama. Apa yang saya lakukan adalah memformat dokumen html saya dalam kata dan menyimpannya. Kemudian saya menggarisbawahi markup menjadi apa yang sebenarnya saya butuhkan. Anda perlu memasukkan namespace vml ke dokumen html Anda (xmlns: v = "urn: schemas-microsoft-com: vml")

<html xmlns:v="urn:schemas-microsoft-com:vml">

Maka Anda membutuhkan kode berikut, pilih id sesuka Anda, Anda juga dapat memilih posisi gambar kurang lebih gratis. Gambar ada di depan teks.

<!--[if gte vml 1]><v:shape id="Bild_x0020_1" type="#_x0000_t75"
style='position:absolute;left:0;text-align:left;margin-left:338.55pt;
margin-top:26pt;width:105pt;height:138.75pt;z-index:251658240;'>
<v:imagedata src="http://c7.valuewalk.com/wp-content/uploads/2011/12/billg4_web1.jpg"/>
</v:shape><![endif]-->

Meskipun saya tidak yakin apakah ini berfungsi untuk Word 2003, Versi saya adalah 2010.

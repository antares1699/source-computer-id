Bagaimana cara mendapatkan hasil dari penyelesaian otomatis bash sebagai string?

Yang saya cari bukanlah cara menambahkan pelengkapan otomatis di bash untuk program lain, tetapi cara untuk mendapatkan hasil dari apa yang akan terjadi di bash ke string, pada dasarnya:

autocomplete "/etc/apa"

akan kembali

/etc/apache2/

jadi saya bisa menggunakannya di skrip saya.

Bantuan apa pun sangat kami hargai, karena yang bisa saya temukan hanyalah cara menggunakan fungsi bash "lengkap", yang melakukan sesuatu yang berbeda;)

Bash memiliki perintah built-in compgen yang dapat banyak membantu Anda. Misalnya, jika Anda ingin mengetahui direktori mana yang akan dilengkapi secara otomatis saat Anda mengetik cd /etc/s<TAB>:

$ compgen -d /etc/s
/etc/security
/etc/sgml
/etc/skel
/etc/ssh
/etc/ssl
/etc/subversion
/etc/sudoers.d
/etc/sysctl.d
$

Kemudian, perintah berikut akan memberi tahu Anda direktori mana yang akan Anda masuki sebelum cd /etc/se<TAB>:

$ compgen -d /etc/se
/etc/security
$

compgen dapat menggunakan sebagian besar opsi dari perintah complete. Lihat manual bash untuk lebih jelasnya.

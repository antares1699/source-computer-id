Bagaimana cara membagikan Google Kalender saya dengan istri saya sehingga dia dapat melihat semua detail acara?

Di Google Kalender, saat saya berbagi kalender utama dengan istri saya dengan mengetikkan alamat emailnya, satu-satunya pilihan adalah "Lihat hanya senggang / sibuk (sembunyikan detail)".

Namun, saya bisa berbagi lain kalender yang telah saya buat (bukan kalender utama saya) dan memberikan hak kepada siapa pun untuk melihatnya atau bahkan mengedit kalender.

Saya mempertimbangkan untuk membuat kalender sekunder dan menggunakannya sebagai kalender utama saya — biarkan kalender utama saya yang sebenarnya kosong. Namun, ketika Anda menambahkan janji, itu default ke kalender utama. Saya yakin saya akan terus menambahkan janji temu ke kalender yang salah.

Jika istri Anda tidak memiliki Kalender Google sendiri, ada pilihan lain. Dengan asumsi Anda menyinkronkan milik Anda ke Outlook, Anda juga dapat menyinkronkan dengan istri Anda. Di komputernya, tutup pandangan, masuk ke akun Google Anda, ikuti langkah-langkah untuk mengatur sinkronisasi kalender dari komputer istri Anda dan hasilnya adalah kedua Pandangan akan disinkronkan melalui Kalender Google. Ada masalah waktu - Saya belum mengalami masalah apa pun. Saya tidak memiliki dua komputer yang terkoordinasi dalam hal interval waktu antara sinkronisasi.

Menekan ALT , dan ketik 1 3 7 pada papan tombol pada keyboard Anda. Saat Anda melepaskan ALT , itu akan muncul.

ALT + 1 3 7

ë

Memeriksa http://www.asciitable.com , atau tabel ASCII pilihan Anda untuk kode ASCII yang dapat Anda ketik.

Anda punya beberapa pilihan.

Menggunakan alt + number, mengambilnya dari peta karakter, dll. Bagi saya, saya cukup beralih ke keyboard AS-Internasional.

Di Vista:

  • panel kendali
  • pilihan daerah dan bahasa
  • keyboard dan bahasa
  • ganti keyboard
  • Menambahkan
  • Inggris (Amerika Serikat)

Setelah Anda beralih ke sana, Anda dapat memasukkannya dengan mengetik " dan e .

Anda dapat beralih dengan memilih melalui bilah bahasa atau dengan mengikatnya ke kombinasi tombol tertentu.

Ini mungkin berlebihan jika Anda hanya menginginkan huruf itu, tetapi ini juga memungkinkan Anda mengetik karakter beraksen lain dengan mudah.

Secara umum, Anda dapat menemukan kombinasi tombol pada mesin Windows dari applet Peta Karakter. Ini terletak secara default di Start | Aksesoris | Alat Sistem. Klik salah satu karakter dan Anda mendapatkan kombinasi tombol di sudut kanan bawah alat. Anda juga dapat menyalin / menempel dari sana untuk karakter yang jarang digunakan.

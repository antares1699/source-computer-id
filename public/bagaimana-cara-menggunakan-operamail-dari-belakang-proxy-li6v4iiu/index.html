Bagaimana cara menggunakan operamail dari belakang proxy?

2 kasus di sini: - di rumah, saya dapat menggunakan operamail untuk mengambil email dari ... katakanlah server IMAP gmail - di tempat kerja, saya menggunakan proxy, dan ketika saya mencoba menggunakan operamail untuk mengambil email, gagal dengan pesan umum 'Sambungan dengan server IMAP tiba-tiba terputus.'

Jadi, dapatkah operamail bekerja dari belakang proxy?

Opera Mail biasanya menggunakan proxy yang sama seperti yang ditentukan dalam preferensi, untuk seluruh program.

Faktanya, di beberapa perusahaan, hanya beberapa port yang tersedia melalui proxy. Jika Anda tidak dapat menggunakan port imap (Anda dapat mencoba dengan program klien email lain, seperti Thunderbird), maka itu juga tidak akan berfungsi di Opera Mail.

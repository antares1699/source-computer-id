Prosesor seluler di desktop?

Prosesor yang ditujukan untuk laptop biasanya membutuhkan lebih sedikit pendinginan (sehingga tidak terlalu berisik) dan jauh lebih hemat energi, sekaligus memberikan daya komputasi sesuai permintaan.

Saya berpikir untuk membangun server rumah, tetapi saya ingin meminimalkan kebisingan dan haus akan daya. Prosesor atom terlalu lemah untuk tujuan saya (saya ingin menjalankan beberapa hal yang menghitung sesekali, dan saya tidak keberatan jika server itu menjadi lebih berisik saat itu). Apakah mungkin menggunakan prosesor seluler dalam mobo desktop biasa?

Saya akan merekomendasikan untuk tidak melakukannya. Jika efisiensi energi adalah yang Anda cari, Anda dapat memperhatikan fitur CPU tertentu. Chipset Intel yang saya lihat menyesuaikan penarikan daya berdasarkan beban saat ini. Ini menjadi agak umum di CPU server (meskipun saya belum benar-benar mencarinya di perangkat keras konsumen).

Apa yang biasanya Anda dapatkan di CPU laptop adalah sesuatu yang harganya lebih mahal per GHz karena ada lebih banyak rekayasa di belakangnya agar tetap kecil, senyap, dan sejuk. Saya tidak ingin membayar ekstra untuk ukuran kecil kecuali saya akan mendapatkan keuntungan darinya.

Dalam hal kebisingan, Anda dapat membayar sedikit lebih mahal untuk sistem pendingin CPU yang lebih tenang dan kipas sasis. Setiap toko penggemar komputer lokal (biasanya bukan jaringan besar) harus memiliki staf yang dapat mengarahkan Anda ke arah yang benar.

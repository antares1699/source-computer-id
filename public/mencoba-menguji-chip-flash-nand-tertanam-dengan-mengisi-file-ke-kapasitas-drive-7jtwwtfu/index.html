Mencoba menguji chip flash nand tertanam dengan mengisi file ke kapasitas drive

Saya mencoba menguji chip flash nand tertanam menggunakan perintah dd (tulis data acak ke file):

dd if=/dev/urandom of=/nand/storage/testnand/test.raw bs=100000 count=50 2> /dev/null

Namun, unit yang saya uji menjalankan busybox v1, dan telah ditambal. Perintah dd tidak dikenali. Apakah ada perintah alternatif yang akan membantu saya mencapai hasil yang sama?

3
3
head -c $((100000*50)) /dev/urandom > test.raw

akan menghemat 100000 * 50 = 5000000 byte, menggabungkan Bash Ekspresi aritmatika dan parameter head -c number_of_bytes.

Untuk debugging: memperluas ide allquixotic, perintah ini mencetak "$ i done" pada 1000, 2000, dll.

{ for i in {1..100000}; do echo $i; if (( ($i % 1000) == 0 )); then echo "$i done" 1>&2; fi; done } > test.raw

Anda dapat mencoba

  • cat /dev/urandom > /nand/storage/testnand/test.raw
  • for i in $(seq 1 10000000); do echo $i >> /nand/storage/testnand/test.raw; done

Saya tidak dapat memikirkan banyak hal lain yang akan berhasil jika Anda bahkan tidak memiliki dd atau cat ...

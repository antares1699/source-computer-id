Saya melacak apa masalahnya di sini - tulis cache.

My Computer > Manage > Device Manager

Disk Drives > [My USB device name] > Properties

Jika kemudian Anda memilih tab "Kebijakan", Anda mendapatkan dua opsi: "Optimalkan untuk penghapusan cepat", dan "Optimalkan kinerja".

Jika saya memilih "Optimalkan untuk kinerja", saya mendapatkan layar biru saat melepas perangkat.

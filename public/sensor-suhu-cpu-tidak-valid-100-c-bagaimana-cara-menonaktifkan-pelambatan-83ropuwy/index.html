Sensor suhu CPU tidak valid (100 ° C?) - Bagaimana cara menonaktifkan pelambatan?

Saya memiliki prosesor Quad core Q9550, dan menampilkan suhu 86 - 100 ° C. Saya cukup yakin itu tidak benar, karena hanya beberapa menit yang lalu saya telah menerapkan kembali parse termal dan setelah itu BIOS menunjukkan 83 ° C. Dan beberapa alat jendela lainnya telah menunjukkan hingga 100+ C.

CPU tidak di-overclock, pengaturan BIOS adalah default, dan pendingin juga default. Windows berjalan dengan baik pada awalnya, tetapi setelah 4-5 jam menjadi sangat lambat - saya kira pembatasan diaktifkan? Jika ya, bagaimana cara menonaktifkannya? Saya kira, ini adalah kesalahan motherboard (Asus P5Q-E), jadi saya harus menggantinya sebagai perbaikan pamungkas?

Memperbarui: Saya telah mengubah pendingin ke yang lain dan sekarang berfungsi seperti pesona. Yang sebelumnya telah dipasang dengan buruk selama 2,5 tahun (saya salah satu orang bodoh).

Anda harus memeriksa kembali apakah pasta termal Anda telah diterapkan dengan benar. Perhatikan bahwa apa pun yang lebih besar dari 0,1mm antara CPU dan heat sink bermasalah. Anda harus membuat lapisan pasta termal setipis mungkin.

Nyalakan dan langsung menuju ke BIOS. Ini akan mulai sekitar 25 ° C (suhu kamar) dan naik dari sana. Jika suhu Anda lebih dari 60 ° C dalam waktu kurang dari 60 detik untuk mencapai sensor BIOS, maka ada sesuatu yang salah dengan heat sink Anda.

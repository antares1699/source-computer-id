Menggunakan WinKey keyboard Razer sebagai tombol Command untuk Mac

Saya baru saja membeli MacBook dan saya mencoba menggunakannya dengan keyboard Razer Lycosa saya. OS X tampaknya tidak dapat memetakan - atau bahkan mendeteksi - tombol Windows sama sekali. Saya ingin menggunakannya sebagai tombol Command. Apakah ini mungkin?

Terkadang "mode permainan" diaktifkan di Lycosa, yang terkunci Tombol Windows ( kunci perintah di Mac).

Petunjuk tentang cara menonaktifkan mode permainan tersedia di Situs Razer

Anda dapat mengaktifkan mode permainan untuk Razer Lycosa dengan menekan dan memegang Logo Razer di Panel Sentuh dan kemudian menekan Windows kunci. Ini akan menonaktifkan kunci Windows. Untuk mengaktifkan kembali Windows Anda hanya perlu menekan dan menahan Logo Razer di Panel Sentuh lalu tekan kembali tombol Windows.

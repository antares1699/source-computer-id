Tidak seperti Virtual PC 2007, mengapa "Undo" disk pada Virtual PC Windows 7 tidak pernah digabungkan ke hard drive virtual utama (.vhd)?

Menggunakan disk "undo" pada Virtual PC 2007, saat VPC dimatikan, program akan bertanya, "Apakah Anda ingin menggabungkan perubahan ke dalam hard drive virtual utama, atau membuang perubahan, atau menyimpan perubahan ke dalam file terpisah ".

Tetapi pada Virtual PC Windows 7, saya mencoba menggunakan gambar yang sama seperti di atas, dari situs web

http://www.microsoft.com/downloads/details.aspx?FamilyId=21EABB90-958F-4B64-B5F1-73D0A413C8EF&displaylang=en

tetapi kali ini, ketika saya mematikan PC virtual, tidak ada pertanyaan yang diajukan. File undo (.vud) terus bertambah setiap kali saya boot dan mematikan, dan .vhd asli tetap utuh. Apakah ada sesuatu yang bisa dilakukan untuk menggabungkan, atau membuang perubahan itu? File .vud sekarang berukuran sekitar 1.2GB.

Pembaruan: sepertinya belum ada yang tahu jawabannya?

Berikut beberapa info dari file bantuan VPC di Win 7:

Jika mesin virtual sedang berjalan atau hibernasi, matikan. Untuk instruksi, lihat Matikan virtual mesin secara manual.

Buka folder Mesin Virtual.

Klik kanan nama virtual mesin, lalu klik Pengaturan.

Di panel kiri Windows Virtual PC Pengaturan, klik Batalkan Disk.

Di panel kanan, klik Terapkan mengubah atau Buang perubahan, lalu klik OK.

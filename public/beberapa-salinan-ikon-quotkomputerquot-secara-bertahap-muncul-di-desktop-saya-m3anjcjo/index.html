Komputer

Beberapa salinan ikon "Komputer" secara bertahap muncul di desktop saya

Saya baru saja pindah ke Windows7 x64, dan saya mengalami masalah aneh ini di mana seiring berjalannya waktu semakin banyak salinan ikon "Komputer" muncul di desktop saya.

Berikut adalah contoh, jumlah ikon saya hingga 4.

Jika saya mencoba menghapus salah satunya, semuanya akan terhapus. Tapi, setiap kali saya me-restart komputer, itu kembali ke hanya satu ikon.

Saya telah mencari di Google dan di situs web Microsoft, dan saya tidak dapat menemukan bantuan apa pun.

Ada ide mengapa ini terjadi atau bagaimana memperbaikinya (tanpa menginstal ulang Windows).

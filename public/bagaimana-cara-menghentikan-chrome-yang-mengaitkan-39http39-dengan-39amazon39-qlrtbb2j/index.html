Bagaimana cara menghentikan Chrome yang mengaitkan 'http' dengan 'Amazon'

Saat ini setiap kali saya mencoba mencari di Google Chrome menggunakan omnibar, jika kueri saya dimulai dengan 'http', saya akan tergerak untuk mencari di Amazon.

Contoh: Misalkan saya tidak dapat mengingat apa itu kode status HTTP tertentu. Saya mulai mengetik "http 200" di omnibar. Namun, segera setelah saya menekan spasi setelah mengetik 'http', omnibar salah 'mengenali' ini sebagai Amazon, dan mengirimkan sisa kueri '200' ke sana.

Bagaimana cara memutuskan hubungan antara kata 'http' dan layanan pencarian 'Amazon' di Google Chrome?

(Tidak ada kombinasi tombol yang akan 'membatalkan' pencarian Amazon - bahkan mengutuk ke Google Search akan dibatalkan jika saya kemudian melanjutkan mengetik. Satu-satunya solusi adalah pergi ke beranda Google, yang mengalahkan inti dari omnibar).

Sumber: https://support.google.com/chrome/bin/answer.py?hl=id&answer=95653

  1. Klik ikon kunci inggris pada bilah alat browser.
  2. Pilih Options (Preferences di Mac dan Linux; Settings di Chromebook).
  3. Klik tab Basics.
  4. Klik Manage search engines di bagian "Pencarian".
    • Hapus mesin pencari : Pilih mesin telusur dari daftar dan klik x yang muncul di akhir baris.
    • Tambahkan mesin pencari : Gulir ke bagian bawah dialog dan isi bidang berikut:
      • Tambahkan mesin telusur baru : Masukkan nama panggilan untuk mesin pencari.
      • Kata kunci : Masukkan pintasan teks yang ingin Anda gunakan untuk mesin telusur. Gunakan kata kunci untuk melakukan pencarian kata kunci .
      • URL : Masukkan alamat web untuk mesin pencari.
    • Edit mesin pencari : Pilih mesin telusur dari daftar dan klik bidang yang ingin Anda ubah.
    • Jadikan mesin telusur sebagai default : Pilih mesin telusur yang ingin Anda gunakan sebagai mesin telusur default, lalu klik tombol Make default yang muncul di baris.

Yang menarik adalah kata kunci . Coba edit pencarian Amazon, dan lihat di sana. Jika Anda tidak memerlukan pencarian Amazon, menghapusnya juga akan berhasil.

Mengapa Ctrl + Tab berhenti bekerja saat membuka halaman yang disematkan flash?

Ini hewan peliharaan besar yang saya miliki. Setiap kali saya menonton video di youtube.com atau megavideo.com, saya tidak bisa hanya Ctrl + Tab untuk mengubah tab. Saya harus mengklik di beberapa bagian halaman tanpa video untuk membuatnya berfungsi, tetapi bahkan itu tidak selalu terjadi.

Pemutar flash mengambil alih pengikatan tombol Tab saat memiliki fokus. Jika flash player di dalam Firefox memiliki fokus, saat Anda menekan Ctrl + Tab, Firefox hanya melihat tombol Ctrl karena flash mengabaikan fakta bahwa Ctrl sedang ditekan dan menghentikan penekanan tombol Tab.

Sayangnya, saya belum menemukan solusi untuk masalah ini.

Triknya adalah mengalihkan fokus keyboard dari Flash dan kembali ke browser:

  1. Klik pada latar belakang halaman (di luar area Flash)
  2. Sekarang Ctrl+Tab bekerja!

Alasan Dan benar: saat aplikasi Flash memiliki fokus (misalnya karena Anda mengklik tombol Play) kemudian penekanan tombol Anda pergi ke aplikasi Flash (berguna jika itu adalah permainan) dan bukan ke browser.

Dengan mengklik di luar aplikasi Flash, Anda mengembalikan fokus ke browser, sehingga browser akan menafsirkan penekanan tombol Anda lagi. Ini tidak terbatas pada Ctrl+Tab tetapi untuk setiap penekanan tombol normal (yang berarti bahwa tombol Win atau Ctrl+Alt+Del masih ditangani oleh Windows seperti yang Anda harapkan, tetapi itu karena tombol-tombol ini tidak ditujukan pada browser pada awalnya).

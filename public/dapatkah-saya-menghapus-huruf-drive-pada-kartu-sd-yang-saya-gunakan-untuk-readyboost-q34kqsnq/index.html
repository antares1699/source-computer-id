Dapatkah saya menghapus huruf drive pada kartu SD yang saya gunakan untuk Readyboost?

Saya menggunakan Kartu SD 4GB di laptop Windows 7 saya dengan Rreadyboost dihidupkan untuk kartu tersebut. Di komputer saya, kartu SD menunjukkan huruf drive-nya sendiri. Karena saya hanya menggunakan kartu SD untuk Readyboost dan tidak ada yang lain, saya lebih suka kartu itu tidak ditampilkan di Komputer Saya. Dapatkah saya menghapus huruf drive di Manajemen Komputer (sehingga Kartu SD tidak muncul di Komputer Saya), atau apakah tidak ada huruf drive yang menonaktifkan fitur Readyboost?

Alat kecil yang berguna bernama HideDrive melakukan triknya. Diuji dan bekerja di komputer saya sendiri dengan Readyboost pada stik USB. Anda masih dapat mengakses drive, meskipun itu tersembunyi dari Explorer yang persis seperti yang ingin saya lakukan, hanya tidak pernah berhasil. Kata peringatan, alat restart komputer Anda ketika Anda menekan terapkan.

Jika Anda suka berpetualang, Anda juga dapat mengedit registri secara langsung. Pada dasarnya, navigasikan ke Komputer \ HKEY__CURRENT__USER \ Software \ Microsoft \ Windows \ CurrentVersion \ Policies \ Explorer dan edit NoDrives ke nilai desimal menggunakan bagan di bawah ini. Cukup tambahkan angka untuk huruf drive yang ingin Anda sembunyikan. Jadi untuk menyembunyikan drive I :, setel ke 256.

Setidaknya di Windows 8.1, ReadyBoost tidak memerlukan huruf kandar. Anda dapat menghapus surat di Manajemen Disk dan itu akan tetap berfungsi, bahkan setelah reboot (kernel menahan kunci pada file ReadyBoost.sfcache, entri biasa muncul di log peristiwa dan lampu berkedip saat membuka program, seperti yang diharapkan)

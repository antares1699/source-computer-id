Bagaimana cara menambahkan folder yang berisi titik koma ke Windows PATH?

Saya perhatikan bahwa titik koma ; adalah karakter yang valid untuk file Windows (NTFS) dan nama direktori.

Namun, titik koma digunakan sebagai pemisah elemen untuk variabel lingkungan PATH.

Katakanlah saya memiliki folder bernama C:\ "main() {printf('%d', 42);}" (heh, bagus, juga berisi tanda persen) - bagaimana saya bisa menambahkan folder ini ke sistem PATH?

  • Seperti yang telah dikatakan, Anda benar-benar harus menghindari penggunaan karakter seperti itu di jalur - Mereka akan membuat banyak program tidak curiga yang Anda tidak perlu menulis ulang kode sumber untuk menanganinya. Apa yang saya sarankan, jika Anda harus memiliki jalur semacam ini untuk program tertentu, menulis file batch yang menetapkan jalur yang diperlukan oleh program ini, mengeksekusi program tersebut, lalu mengembalikan jalur ke normal setelah selesai. Kemudian Anda dapat meletakkan file batch itu di jalur di suatu tempat untuk dipanggil saat Anda perlu menjalankan program itu, tanpa mengganggu jalur untuk semua program lain.
    — C. M.

Menggunakan tanda kutip ganda cocok untuk saya:

SET PATH=%PATH%;"C:\main() {printf('%d', 42);}"

Ubah Gambar / Logo ubin Windows Store di Layar Mulai Windows 8

Saya melihat posting ini: Bagaimana cara mengubah gambar ubin dari aplikasi UI Modern? tetapi tidak berisi metode apa pun untuk mengubah gambar ubin Windows Store.

Adakah yang bisa memberi tahu saya bagaimana saya bisa melakukan itu?

4
Elmo — Sumber

1 Jawaban

11
11

  1. Arahkan ke C:\Windows\WinStore dan buat cadangan folder images.

  2. Ambil kepemilikan dari folder images.

  3. Buka folder images.

  4. Untuk menyesuaikan ubin normal, edit file logo.png dan logo.scale-x.png dan untuk menyesuaikan ubin lebar, edit file tilewide.png dan tilewide.scale-x.png.

  5. Buka Layar Mulai dan buat ukuran ubin Toko lebih kecil atau lebih besar dan Anda akan dapat melihat perubahannya.

CATATAN: Trik unik ini ditemukan oleh saya dan tidak diposting di mana pun.

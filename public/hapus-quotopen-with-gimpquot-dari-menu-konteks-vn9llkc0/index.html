Hapus "Open with GIMP" dari menu konteks?

Saya baru-baru ini mencopot pemasangan GIMP di laptop Vista saya, tetapi masih ada opsi Open with GIMP pada menu konteks untuk sebagian besar jenis file gambar, seperti JPEG dan PNG. Mengklik opsi ini menghasilkan dialog kesalahan.

Apakah ada cara untuk menghapusnya?

Anda dapat menggunakan Crap Cleaner (CCleaner) untuk mengatasinya atau mengedit registri untuk HKLM_Classes dan mencari .JPG, .PNG, .BMP, ada kunci di sana untuk Open, saya telah memberikan jpegfile sebagai contoh, lakukan hal yang sama untuk .PNG dan sebagainya ... HKEY_CLASSES_ROOT \ jpegfile \ shell \ Buka dengan GIMP (Cadangkan registri Anda sebelum melakukan ini ..)

Semoga ini membantu, Salam Hormat, Tom.

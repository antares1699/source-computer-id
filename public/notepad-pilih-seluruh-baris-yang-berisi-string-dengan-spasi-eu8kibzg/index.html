Notepad ++ - Pilih seluruh baris yang berisi string dengan spasi

Saya ingin memilih seluruh baris yang mengandung kata State Province.

Dari menu Search> Find, menggunakan (.*State.*) akan memilih seluruh baris yang mengandung kata State. Tetapi menggunakan (.*State\sProvince.*) tidak.

Apa perintah regex yang benar?

4
deathlock — Sumber
  • Beberapa poin: (1) secara umum lebih baik untuk memastikan seluruh baris cocok, terlepas dari ragam atau pengaturan RE, dengan menggunakan ^ dan $ (di dalam atau di luar tanda kurung, meskipun ini tampaknya tidak perlu); (2) Anda dapat menggunakan spasi literal sebagai pengganti \s; (3) jika Anda memilih State Province sebelum Find maka string akan muncul sebagai string pencarian default, yang dapat Anda tambahkan dengan pre- dan post-ambles; sekarang (4) Anda dapat mengganti spasi dengan \s jika Anda mau, dan Anda mungkin melihat karakter lain dalam string dokumen, termasuk duplikasi, tab, atau karakter spasi lainnya (misalnya tidak terputus).
    — AFH

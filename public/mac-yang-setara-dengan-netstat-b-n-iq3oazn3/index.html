OS X memiliki perintah netstat, tetapi tidak menampilkan informasi tentang program yang terkait dengan koneksi jaringan. Jika Anda ingin melihatnya, Anda perlu menggunakan lsof sebagai gantinya. Perhatikan bahwa itu harus dijalankan sebagai root (yaitu dengan sudo) untuk melihat program pengguna lain:

sudo lsof -i

lsof juga memiliki banyak pilihan untuk mengontrol apa yang ditampilkan:

sudo lsof -i tcp -nP   # show TCP unly (no UDP), and don't translate IP addrs and ports numbers to names
sudo lsof -i 6tcp -stcp:listen   # show only IPv6 TCP ports in the listen state
sudo lsof -i @10.11.12.13   # show only connections to/from 10.11.12.13

... lihat halaman manual untuk lebih lanjut.

Seperti yang telah ditunjukkan, netstat juga tersedia di Mac OS X sebagai alat baris perintah.

Jika Anda lebih suka alat GUI yang terintegrasi dengan OS, Anda dapat menggunakan utilitas jaringan yang terletak di sini: /Applications/Utilities/Network\ Utility.app

Selain netstat, ini juga memungkinkan akses mudah ke utilitas jaringan lainnya. Gambaran yang baik tentang utilitas jaringan dapat ditemukan di artikel "Cara menggunakan Utilitas Jaringan" .

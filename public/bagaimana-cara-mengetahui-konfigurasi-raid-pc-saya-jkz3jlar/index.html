Bagaimana cara mengetahui konfigurasi RAID PC saya

EDIT

Hanya kata ulang singkat dari pertanyaan asli saya ...

Menggunakan Windows XP bagaimana cara menentukan disk / drive / partisi mana yang berada di bawah kendali sistem RAID1 perangkat keras saya?

Pertanyaan asli saya:

PC saya sekarang berumur beberapa tahun. Ketika saya membuatnya, saya memiliki beberapa drive yang disiapkan sebagai RAID 1 - Saya juga memiliki beberapa drive lama yang memiliki foto / musik saya, dll. Di atasnya yang saya masukkan. Saya juga punya beberapa partisi. Akibatnya saya memiliki drive C, E, F, G, dan H yang terlihat. Saya tahu saya memiliki sistem saya di RAID 1, tetapi saya tidak bisa mengingat bagaimana yang lain diatur dan mana (jika ada) dari drive saya yang lain (baik itu drive sebenarnya atau hanya partisi) yang juga merupakan RAID.

Pada dasarnya, salah satu drive saya sekarang penuh dengan foto dan musik saya jadi saya perlu mengatur ulang tetapi saya ingin memastikan bahwa saya memindahkan data berharga ini ke drive RAID. Bagaimana saya bisa mempelajari tentang pengaturan drive disk saya?

saya sedang berlari

  • Windows XP (jangan tertawa! Tidak rusak jadi saya tidak memperbaikinya!)
  • AliSATA, perangkat keras RAID

Maafkan saya jika saya menggunakan terminologi yang salah - saya adalah pengembang web bukan orang IT - bagi saya komputer saya adalah kotak yang penuh dengan kabel dan benda berputar! :-) Namun, saya cukup cerdas dan mampu jika saya diberi instruksi yang baik ... Anda mungkin perlu sedikit membodohi saya!

Anda dapat dengan mudah menemukan yang berikut ini:

1. Login ke komputer Anda dengan akun yang memiliki hak Administrator.

2. Klik kanan pada Komputer dan pilih Kelola

3. Pilih Device Manager, lalu cari dan luaskan kategori Storage controllers

Anda bisa menemukan pengendali serangan

Apakah Ninite merupakan solusi trusuted untuk manajemen paket awal pada instalasi baru / bersih Windows 7 64Bit?

Saya ingin membuka kembali pertanyaan dari tautan di bawah ini, di mana beberapa paket disarankan selain Ninite.com seperti allmyapps.com. Manajer paket untuk Windows

Yang ingin saya ketahui adalah apakah mereka semua dapat dipercaya untuk menginstal di Windows 7 (64bit) sehingga pengelola:

  • Instal perangkat lunak versi terbaru.
  • Mendukung penginstalan 64 bit jika memungkinkan.
  • Menghapus iklan / bilah alat / hal serupa.

Dua poin terakhir dari pertanyaan sebelumnya bagus tetapi bukan prioritas dalam persiapan pemasangan yang bersih

  • Menyediakan cara untuk terus memperbarui program setelah penginstalan.
  • Jika saya dapat menambahkan penginstal khusus ke perangkat lunak, itu adalah nilai tambah yang besar.

Saya lebih peduli di sini tentang menggunakan aplikasi yang sah yang dapat saya percayai untuk membangun dasar gambar bersih dari sistem operasi saya dengan semua aplikasi pilihan diinstal tanpa keributan dan spam / bloatware.

Sehubungan dengan Ninite:

memasang versi terbaru dari perangkat lunak

Iya. Ninite akan menginstal rilis perangkat lunak terbaru (stabil).

mendukung penginstalan 64 bit di mana bisa jadi

Pastinya. Dari halaman Ninite - Ninite memeriksa bahasa PC Anda dan dukungan 64-bit untuk menginstal versi terbaru dan terbaik dari setiap program.

menghapus iklan / bilah alat / hal serupa

Iya. Ninite menginstal aplikasi dengan pengaturan default dan berkata Tidak ke bilah alat dan sampah lainnya jika memungkinkan. Saya telah mengujinya dengan Java dan tidak ada toolbar yang diinstal. Mengutip situs Ninite - Ninite menginstal perangkat lunak dengan cepat dengan pengaturan default dan mengatakan "tidak" ke toolbar browser dan sampah lainnya.

menyediakan cara untuk menyimpan program diperbarui setelah instalasi

Sayangnya tidak ada. Perangkat lunak diinstal "sebagaimana adanya". Jika perangkat lunak (seperti Firefox) dilengkapi dengan pembaru, Anda dapat menggunakannya, tetapi Ninite sebenarnya bukan pengelola paket dalam arti yang sebenarnya, lebih merupakan alat penyebaran perangkat lunak yang lebih mudah. Anda mungkin tertarik dengan Perbarui Pemberitahu untuk tujuan ini.

jika saya dapat menambahkan penginstal khusus ke file perangkat lunak, itu nilai tambah yang besar.

Tak satu pun dari solusi di utas lain yang dapat melakukan ini sejauh yang saya tahu. Jika Anda ingin menginstal perangkat lunak lain yang tidak terdaftar di Ninite secara otomatis pada penginstalan bersih Windows, sebagian besar paket penginstal menyertakan sakelar baris perintah dapat Anda gunakan untuk menginstal perangkat lunak secara diam-diam.

Untuk informasi hebat tentang menyebarkan perangkat lunak pada penginstalan bersih, AppDeploy.com adalah Bagus sumber.

Apa perbedaan antara Run registry key dan Startup Folder (di Start Menu)?

Apa perbedaan sebenarnya di antara keduanya?

Keduanya memiliki tujuan yang sama: menjalankan program pada startup windows.

Perbedaan yang jelas adalah:

  • Pengguna dapat dengan mudah mengelola Folder Startup
  • Jalankan registri tampaknya menjalankan program lebih awal saat komputer dimulai.

Saya tidak berpikir ini cukup untuk membenarkan keduanya. Apa yang saya lewatkan?

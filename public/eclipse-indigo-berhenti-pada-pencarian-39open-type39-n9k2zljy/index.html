Eclipse Indigo berhenti pada pencarian 'Open Type'

Saat saya mencoba mencari kelas Java dengan Ctrl-shift-T (sembulan Jenis Terbuka), Eclipse berhenti berfungsi saat saya mengetik 1 karakter. Biasanya diperlukan waktu sekitar 8 detik untuk 'mencairkan', tetapi terkadang tidak bisa kembali sama sekali .. Saat macet, saya melihat bahwa proses gerhana membutuhkan sekitar 1Gig mem dan CPU sekitar 100%!

Saya sudah mencoba membuat ruang kerja baru, menyesuaikan eclipse.ini (ukuran perm, nilai memori berbeda), dimulai dengan -clean dan terakhir instal ulang seluruh IDE. Tidak ada yang membantu ..

Saya menggunakan plugin berikut: JRebel dan m2e. Saya putus asa mencari solusi karena masalah ini menyebabkan saya kehilangan banyak waktu.

Sistem: Ubuntu 12.04 LTS 64 bit, 4GB mem, Intel core i7 860 @ 2.8 Ghz.

Semoga ada yang tahu solusinya. Terima kasih atas waktu Anda.

Bagaimana cara mengimpor template email HTML ke Thunderbird?

Ini mungkin mudah, tapi saya tidak bisa memahaminya. Saya telah mengunduh template email HTML dari http://www.campaignmonitor.com/templates/ yang tampaknya hanya berupa file + gambar HTML.

Bagaimana cara menggunakan ini dengan Thunderbird untuk mengirimkannya sebagai email HTML?

Lihat petunjuk di bawah ini untuk memasukkan template HTML Anda:

Saat Anda membuat pesan HTML, Anda dapat menyisipkan tag HTML atau mengedit yang ada Tag HTML di dalam pesan:

  1. Secara opsional, untuk mengedit tag HTML yang ada, pilih bagian dari pesan, atau pilih seluruh pesan (untuk Misalnya dengan menekan Ctrl + A atau dengan memilih Edit -> Pilih Semua).
  2. Pilih: Sisipkan -> HTML ...
  3. Edit kode HTML sesuai keinginan.
  4. Tekan Sisipkan.

Lihat di bawah untuk ekstensi yang menyediakan cara lain untuk mengedit tag HTML.

Catatan: Beberapa orang yang menerima Anda pesan mungkin tidak melihat efek dari perubahan apa pun yang Anda lakukan pada HTML tag. Misalnya, pengguna Thunderbird mungkin memilih untuk menampilkan pesan sebagai teks biasa atau HTML sederhana, dan klien webmail mungkin mengubah HTML Anda dengan berbagai cara.

TOLONG DICATAT:

Saat Anda melakukan ini, pastikan bahwa Anda memiliki semua gambar yang dihosting di suatu tempat. (server web, server hosting gambar, dll.) Tentu gambar mungkin terlihat saat berada di komputer Anda, tetapi saat gambar tersebut ada di orang lain, itu tidak akan terjadi. Mereka akan melihat placeholder gambar yang rusak.

Referensi: http://kb.mozillazine.org/Edit_HTML_source

Oke, jadi akhirnya saya menemukan "retasan". Peretasan ini membutuhkan pembuatan ekstensi untuk menangani permintaan tersebut. Saya sudah mencoba beberapa upaya untuk "membajak" Chrome dengan cara yang lebih mudah tetapi tampaknya Chrome cukup aman untuk menghentikan saya melakukan itu dan ini yang paling dekat yang bisa saya dapatkan.

Pertama, buat direktori kosong di suatu tempat yang dapat diakses di hard drive.

Buat file manifest.json dengan konten berikut:

{
    "name": "Open Chrome URLs",
    "version": "1.0",
    "manifest_version": 2
}

Buat file open.html dengan konten berikut:

<html>
<head>
<title>Open Chrome URLs</title>
<script type="text/javascript" src="open.js"></script>
</head>
<body>
</body>
</html>

Buat file open.js dengan konten berikut:

window.addEventListener("load", function(){
    var key = "secretKey"; // replace "secretKey" with your own secret key
    if(window.location.search == "?key=" + key && window.location.hash.length > 1){
        chrome.tabs.update({
            'url': "chrome://" + window.location.hash.substr(1) + "/"
        });
    }else{
        document.body.appendChild(document.createTextNode("Invalid"));
    }
});

Ganti kunci rahasia dengan milik Anda sendiri jika diinginkan.

Kemudian, buka halaman Extensions (chrome://extensions/).

Centang kotak "Mode pengembang" dan klik "Muat ekstensi yang belum dibuka paketnya" dan pilih direktori yang baru saja Anda buat.

Anda sekarang akan melihat ekstensi baru muncul.

Salin id ekstensi.

Terakhir, mulai Chrome dengan URL berikut sebagai parameter.

chrome-extension://nihlceAnywayPutTheExtensionIdHere/open.html?key=secretKey#settings

Ganti bagian pertama dengan id ekstensi dengan yang baru saja Anda salin.

Juga ganti secretKey dengan yang Anda setel di atas.

Anda juga dapat menggunakan sebagian besar URL Chrome selain pengaturan.

Catatan: Anda memerlukan pintasan ke Chrome, bukan tautan Internet.

Semoga berhasil!

Pengikut AutoHotkey script dapat digunakan untuk berbagai browser berbasis Chrome.

; This script opens the "About" page in a Chrome-based Web browser
; -------------------------------------
; To use a different Chrome variant, change the following variables
EnvGet, pf, ProgramFiles(x86) ; For 64-bit browsers: ProgramW6432
dir = %pf%\Google\Chrome\Application ; Browser's drive and directory
protocol = chrome ; Start of the URL, and also the name of the Windows process
tabName = New Tab - Google Chrome ; How the browser names new tabs
; -------------------------------------
proc = %protocol%.exe ; Full name of the Windows executable file
app = %dir%\%proc% ; Full path to the browser
winTitle = ahk_exe %proc% ; Identify the window by the executable file's name
Menu, Tray, Icon, %app% ; Use the browser's icon for this script
Process, exist, %proc% ; See whether the browser is already running
If NOT ErrorLevel ; Browser is not currently running, so run it
 Run, "%app%"
WinWait, %winTitle%,, 200
If ErrorLevel
{
 MsgBox, 48, Error, Browser window was not found.`n`nRef: %A_ScriptFullPath%
 Exit, 1
}
WinActivate, %winTitle%
Send ^t ; Open a new tab in the browser
WinWait, %tabName%,, 20
If ErrorLevel ; The new browser tab was not found
 Exit, 1
Sleep, 100
WinActivate, %winTitle%
Send %protocol%://settings/help{ENTER} ; Open the browser's "About" page
Exit

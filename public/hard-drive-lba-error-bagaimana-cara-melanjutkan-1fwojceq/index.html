Hard drive LBA Error: Bagaimana cara melanjutkan?

Saya memiliki hard drive (produk samsung). Saya tidak bisa boot ke Windows, jadi saya menguji hard drive menggunakan Utilitas Samsung yang tersedia di: http://support-us.samsung.com/cyber/popup/iframe/pop_troubleshooting_fr.jsp?idx=51176&modelname=HD250HJ 1

Saat menjalankan pemindaian, saya melihat bahwa disk memiliki kesalahan LBA.

Apa sebenarnya kesalahan LBA itu? cara apa pun untuk menyelamatkan data, atau Hard drive?

Utilitas pemeriksaan hard drive menyarankan agar saya memformat Hard drive. Apakah itu berarti saya dapat menggunakan disk dengan aman setelah format level rendah & tidak akan ada masalah?

Jika Anda memberikan informasi lebih lanjut tentang model drive yang Anda gunakan serta tentang kesalahan yang dilaporkan utilitas Samsung, Anda mungkin mendapatkan jawaban yang lebih baik. Tanpa mengetahui detail situasi Anda, yang dapat kami lakukan hanyalah berspekulasi. Nah, itu, dan mungkin melambaikan tangan kita dengan liar di udara untuk memberi penekanan.

LBA adalah singkatan dari Pengalamatan Blok Logis . Dalam hal ini saya berasumsi bahwa ini hanyalah cara lain, yang mungkin secara teknis lebih akurat untuk merujuk pada " sektor buruk ".

Kecuali hard drive Anda sangat lama, sistem file OS Anda mungkin bahkan tidak tahu apa-apa telah terjadi. Untuk sementara waktu sekarang firmware dalam hard drive telah secara transparan menangani kesalahan penulisan sektor dengan memetakan ulang sektor tersebut ke lokasi lain pada drive. Jika Anda dapat menyalin semua data pada drive ini dan tidak ada kesalahan yang dilaporkan, tampaknya sangat mungkin bahwa sektor yang menyinggung bahkan belum digunakan oleh sistem file Anda. Sektor buruk mungkin hanya ditemukan karena alat diagnostik Samsung yang Anda gunakan melakukan pemindaian lengkap semua sektor pada drive.

Hal pertama yang harus dilakukan dalam situasi seperti ini adalah menyimpan data Anda, yang telah Anda lakukan.

Selanjutnya saya akan merekomendasikan agar Anda melihat file PINTAR. data untuk mengemudi. Mungkin diagnostik Samsung dapat melakukan ini untuk Anda. Jika tidak, tersedia utilitas freeware yang akan melakukan ini untuk Anda. Secara khusus, lihat apa nilai dari S.M.A.R.T. daftar atribut di bawah ini.

  1. ID 005 (0x05) Jumlah Sektor yang dialokasikan kembali
  2. ID 188 (0xBC) Batas Waktu Perintah
  3. ID 196 (0xC4) Hitungan Acara Realokasi
  4. ID 197 (0xC5) Hitungan Sektor Tertunda Saat Ini
  5. ID 198 (0xC6) Hitungan Sektor Tidak Dapat Dikoreksi

catatan: Drive Anda mungkin tidak menerapkan semua atribut ini. Anda bisa merujuk ke meja ini untuk deskripsi atribut.

Apa "utilitas pemeriksaan hard drive" yang Anda gunakan? Apakah itu memberikan informasi lain selain rekomendasi untuk memformat drive?

Untuk apa nilainya, sebenarnya tidak ada yang namanya a "format tingkat rendah" dari hard drive. Istilah itu tidak benar-benar memiliki arti untuk hard drive setidaknya selama satu atau dua dekade sekarang. Yang dapat Anda lakukan hanyalah memformat tingkat sistem file (partisi di) drive.

Alasan di balik saran ini mungkin mencoba untuk "memaksa" drive untuk memetakan kembali sektor yang sejauh ini hanya gagal saat dibaca. (Lihat deskripsi di tabel untuk ID 197 (0xC5) Hitungan Sektor Tertunda Saat Ini .)

Jika itu benar maka "format" hanya akan berguna jika setiap sektor ditulis bukan hanya untuk dibaca. Sebagai alternatif, Anda dapat menggunakan alat penghapus disk yang menulis nol untuk setiap sektor di drive. Itu mungkin yang ingin Anda coba lakukan jika drive itu Hitungan Sektor Tertunda tetap bukan nol.

Seberapa "aman" untuk terus menggunakan drive ini? Siapa tahu??

Ini pasti selalu praktik yang baik untuk memastikan Anda memiliki cadangan data Anda. Selain itu, saya menyarankan untuk terus memantau S.M.A.R.T. atribut untuk melihat apakah drive terus menurun dari waktu ke waktu. Sektor buruk atau tiga kurang lebih diharapkan hari ini dan bukan alasan untuk berhenti menggunakan drive.

Namun, saya akan melihat pola kegagalan berulang dan biasanya meningkat dari waktu ke waktu sebagai tanda kematian drive yang akan datang. Jika Anda melihat ini dan drive masih dalam garansi, minta Samsung untuk menggantinya.

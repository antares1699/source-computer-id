Ya, itu adalah distribusi Linux pertama yang saya coba. Saya masih memiliki CD-ROM di suatu tempat.

Saya harus mengkompilasi ulang kernel untuk menggunakan port printer. Itu ditempati oleh beberapa perangkat TPC / IP melalui port printer (!).

Pembaruan: Saya menemukannya. Teks di CD-ROM (untuk referensi):

Plug-and-Play Linux. Sistem operasi multimedia siap internet dengan sumber. Musim panas 1994. File file CDROM universal ISO 9660 dengan ekstensi Rockridge. Yggdrasil Computing Incorporated / Freesoft. 4880 Steven Creek Blvd. Suite 205. San Jose, California. 95129-1034. Amerika Serikat. bebas pulsa: (800) 261-6630. (408) 261-6630. faks: (408) 261-6631. Untuk menginstal, masukkan floppy di drive A, masukkan CDROM dan reset komputer. Membutuhkan 386/486/586 CPU, 8 MB RAM, floppy drive, Sony CDU-31A, Sony 535, Mitsumi, Matsushita / Panasonic / LaserMate atau SCSI CDROM dengan Adaptec 154x / 174x, Bustek 542B, Future Domain 8xx / 16xx, other other Pengontrol berbasis TMC-950, pengontrol Ultrastor 14F / 24F / 34F, atau Western Digital 7000faast SCSI.

Unggahan mungkin menyusul.

Peringatan: Setidaknya beberapa versi Yggdrasil tidak bekerja dengan ATAPI / IDE CD-ROM standar, Anda memerlukan salah satu dari (ISA) kartu scsi (atau meniru mereka jika Anda menggunakan beberapa jenis mesin virtual) untuk dapat melakukan lebih dari itu. muat saja kernel dan perhatikan panik karena kernel tidak dapat memasang sistem file root.

Anda mungkin dapat meletakkan image cd pada hard disk (atau partisi) (sekunder) dan mengarahkan kernel boot penginstalan ke image tersebut.

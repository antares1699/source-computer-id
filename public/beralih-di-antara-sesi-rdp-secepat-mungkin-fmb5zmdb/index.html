Klien desktop jarak jauh dilengkapi dengan pintasan keyboard alternatif untuk menyediakan pintasan keyboard umum di dalam sesi RDP Anda sehingga Anda dapat menyimpan pintasan asli untuk tindakan di desktop Anda. Dengan cara ini, Anda masih bisa menggunakan Alt + Tab untuk beralih dengan cepat di antara sesi RDP.

  1. Saat sesi RDP Anda dimaksimalkan, tekan tombol pulihkan pada bilah koneksi biru (atau Ctrl + Alt + Istirahat ), sehingga mode dimaksimalkan.

  2. Ubah ukuran jendela RDP ke ukuran maksimum sehingga memenuhi hampir seluruh layar. Tetapi beberapa area desktop jarak jauh Anda akan berada di luar layar sehingga jendela klien menjadi dapat digulir. Jika Anda ingin menghindari kebutuhan untuk menggulir, Anda memiliki dua pilihan, tergantung pada preferensi penggunaan Anda:

    • Menjaga skala konten yang sama : menurunkan resolusi layar sesi RDP. (Saya biasanya lebih suka yang ini.)
    • Menjaga resolusi layar yang sama : memperkecil layar jarak jauh. Klik kanan bilah judul jendela sesi RDP ketika berukuran sebagai "dipulihkan" dan di menu, periksa item menu
      ☑ Ukuran Cerdas .

  3. tekan

    • Alt + Tab untuk beralih di antara sesi RDP. Ini yang kamu minta.
    • Alt + PgUp atau Alt + PgDn untuk beralih antar tugas dalam sesi RDP.
    • Alt + Rumah untuk membuka menu Start di sesi RDP.
    • Ctrl + Alt + Akhir untuk menekan Ctrl + Alt + Del dalam sesi RDP

Mengingat seberapa besar perhatian yang diberikan pengembang Klien Desktop Jarak Jauh untuk membuat pintasan keyboard alternatif, saya akan mengatakan ini adalah cara yang dirancang untuk menggunakan beberapa sesi RDP dengan mudah.

Tip: jika Anda menemukan diri Anda sendiri lebih lama bekerja di dalam satu jendela sesi jarak jauh, tekan Ctrl + Alt + Istirahat untuk membuatnya menjadi layar penuh dan menggunakan semua pintasan keyboard standar. Tekan lagi untuk mengembalikannya ke ukuran standar yang dipulihkan dan mengembalikan semua pintasan keyboard standar kembali ke Windows Anda. Semua bisa dilakukan tanpa mengambil mouse dan mengklik ikon.

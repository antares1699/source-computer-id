Untuk apa opsi "Aksesibilitas Windows"?

Apakah mereka berguna? bagaimana memanfaatkannya?

Pertama, sini adalah tautan ke situs web Aksesibilitas Microsoft di Rumah. Ini menjelaskan bagaimana mereka dapat diaktifkan untuk digunakan pada mesin Anda.

Tujuan utama dari fungsi-fungsi ini adalah untuk meningkatkan kegunaan Windows bagi pengguna dengan berbagai disabilitas, misalnya berbagai tingkat gangguan penglihatan (termasuk kebutaan), gangguan pendengaran, cacat fisik, dan gangguan fungsi motorik.

Mereka berguna jika Anda memiliki beberapa jenis kekurangan yang dirancang untuk dibantu oleh alat, tetapi saya tidak bisa lebih tepat tentang kebutuhan pribadi Anda tanpa informasi lebih (mungkin sangat pribadi). Saya akan merekomendasikan (sekali lagi) membaca tautan yang disediakan untuk informasi lebih lanjut. Perhatikan juga bahwa pada Windows 7, Ease of Access Center memiliki opsi "buat rekomendasi" yang akan mengajukan pertanyaan kepada Anda dan kemudian memberikan rekomendasi tentang opsi apa yang paling berguna bagi Anda.

Fungsinya mencakup (daftar ini mungkin tidak lengkap):

  • Kaca pembesar layar / penskalaan DPI untuk meningkatkan keterbacaan layar, bagi pengguna dengan gangguan penglihatan.

  • Penyesuaian kontras / warna, untuk pengguna dengan gangguan penglihatan.

  • Kemampuan untuk mengonfigurasi komputer agar dioperasikan tanpa layar, untuk pengguna tunanetra
    (menggunakan narasi layar dan input khusus / kontrol ucapan).

  • Konversi peringatan suara ke pemberitahuan visual, untuk pengguna tuna rungu / gangguan pendengaran.

  • Kemampuan untuk mengkonfigurasi mouse dan keyboard agar "lebih mudah" bagi pengguna, bagi pengguna dengan gangguan kontrol motorik halus.
    (sesuaikan kecepatan mouse, kecepatan klik dua kali, kecepatan pengulangan keyboard, dll.)

  • Kemampuan untuk mengonfigurasi sistem agar dapat dioperasikan tanpa mouse dan / atau keyboard, untuk pengguna dengan gangguan kontrol motorik dan / atau cacat fisik.
    (pada keyboard layar dan kontrol ucapan, misalnya)

  • Beberapa opsi untuk mengurangi kerumitan penggunaan layar.
    (tombol sticky / toggle / filter dan menonaktifkan animasi, dll)

Catatan Samping

Beberapa dari fungsi ini berguna di luar hanya para pengguna yang dianggap memiliki disabilitas, jadi jangan mengabaikan mereka begitu saja! Dua contoh yang saya tawarkan:

  • Istri saya menemukan penskalaan DPI sangat berguna ketika dia memecahkan kacamatanya dan sedang menunggu pasangan pengganti. Dia terlihat agak lama, dan menaikkan skala DPI, untuk membuat semua yang ada di layar tampak lebih besar, membuatnya lebih mudah untuk menggunakan komputernya tanpa mengalami sakit kepala yang parah.

  • Ketika putri saya mulai belajar menggunakan komputer (usia 3), dia mengalami kesulitan dalam mengontrol mouse dengan akurat dan dengan cepat menjadi frustrasi. Mengurangi kecepatan gerakan, meningkatkan waktu klik dua kali dan menyalakan jejak mouse membuatnya lebih mudah untuk digunakan, dan seiring waktu saat dia menjadi kompeten dengan mouse, saya perlahan-lahan memindahkan opsi ini ke pengaturan "normal".

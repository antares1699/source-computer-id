Apakah ada plugin Google Bookmarks untuk Chrome?

Saat ini saya menggunakan Google Bookmarks melalui plug-in toolbar Google untuk Firefox. Saya merasa ini sangat berguna karena memungkinkan saya memiliki penanda roaming daripada harus memelihara penanda pada masing-masing mesin dan juga memulihkannya saat membangun ulang mesin.

Apakah saat ini ada cara untuk menggunakan Google Bookmarks di Chrome? Ini adalah satu hal yang saya tunggu yang mencegah saya beralih dari Firefox ke Chrome.

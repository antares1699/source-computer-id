Jumlah minimum hard drive yang dibutuhkan untuk RAID 10?

Saya ingin membangun server file rumah yang menjalankan Windows Server 2012, dengan hard drive WD Red yang dikonfigurasi di RAID 10.

Berapa jumlah paling sedikit dari hard drive berikut yang perlu saya beli agar dapat dikonfigurasi di RAID 10? Saya telah meneliti dan beberapa situs mengatakan 4. Apakah ini akurat?

Apakah saya memerlukan hard drive tambahan untuk menginstal OS (Windows Server 2012), atau apakah saya akan menginstalnya di hard drive yang telah saya konfigurasikan di RAID 10? (Itu mungkin pertanyaan yang bodoh, tapi saya hanya ingin memastikan.)

RAID 10 membutuhkan setidaknya 4 drive . Dua set dua, karena Anda melakukan pencerminan (RAID 1) dan striping (RAID 0). Jika Anda melakukan RAID perangkat keras, maka Anda dapat menginstal OS pada array.

Untuk menjawab pertanyaan Anda tentang menginstal OS pada drive non-RAID yang terpisah, sisi negatifnya adalah Anda tidak akan mendapatkan keuntungan dari RAID sejauh menyangkut OS Anda. Jika disk gagal, Anda akan mengalami waktu henti dan harus menginstal ulang OS dari awal atau dari cadangan. Karena ini adalah server rumah, saya ragu down-time menjadi masalah. Di sisi lain, menurut pengalaman saya, Anda akan mendapatkan penginstalan yang jauh lebih lancar dan tidak rumit jika Anda meletakkan OS pada drive non-RAID yang terpisah.

RAID 10, atau 1 + 0, adalah garis cermin. Oleh karena itu, Anda memerlukan satu set 2 drive untuk satu mirror (RAID 1) dan setidaknya satu set lagi (dari 2) untuk mencapai garis Anda (RAID 0). Jadi, 4 drive akan menjadi kebutuhan minimum Anda.

Jika Anda menggunakan array RAID 10 hanya untuk penyimpanan, terlepas dari server Anda, maka ya, Anda mungkin menginginkan drive lain untuk OS. Namun perlu diingat, ini tidak akan ada di array RAID.

Jika Anda ingin semuanya ada di array RAID, maka tidak perlu drive tambahan. OS juga akan diinstal pada array.

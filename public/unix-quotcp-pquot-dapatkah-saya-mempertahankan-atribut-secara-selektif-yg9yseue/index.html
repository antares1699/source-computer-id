Unix "cp -p". Dapatkah saya mempertahankan atribut secara selektif?

Saya ingin menyalin waktu modifikasi dan akses, tetapi bukan ID pengguna. Jika saya menggunakan

cp -p source target

Itu akan menyalin segalanya.

Saya mencoba menyalin file ke pengguna lain tetapi tetap mempertahankan tanggal aslinya.

Dari manual cp tahun GNU coreutils:

-p sama dengan --preserve=mode,ownership,timestamps

Jadi, Anda sedang mencari

cp --preserve=mode,timestamps source target

Tetapi jika Anda menggunakan beberapa sistem operasi non-GNU, Anda mungkin tidak dapat menggunakan opsi panjang ini dengan cp. Dalam hal ini, Anda dapat mencoba rsync, di mana Anda dapat menentukan secara rinci atribut mana yang harus dipertahankan (cari pria halaman untuk "melestarikan"):

    -H, --hard-links            preserve hard links
    -p, --perms                 preserve permissions
    -E, --executability         preserve executability
    -A, --acls                  preserve ACLs (implies -p)
    -X, --xattrs                preserve extended attributes
    -o, --owner                 preserve owner (super-user only)
    -g, --group                 preserve group
        --devices               preserve device files (super-user only)
        --specials              preserve special files
    -t, --times                 preserve modification times

Jadi, untuk menyerupai perintah cp di atas, gunakan perintah seperti

rsync -pEt source target

Untuk menguji perintah sebelumnya, Anda dapat memulai "dry-run" dengan -n. Tambahkan juga parameter verbose -v untuk melihat apa yang terjadi:

rsync -nv -pEt source target

Namun, saya tidak yakin, apakah file mengakses waktu akan disalin juga.

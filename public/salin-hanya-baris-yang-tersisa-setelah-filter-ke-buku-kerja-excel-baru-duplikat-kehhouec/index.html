Salin hanya baris yang tersisa setelah filter ke Buku Kerja Excel baru [duplikat]

Kemungkinan Duplikat:
Pemformatan bersyarat Excel 2010: menyorot duplikat secara selektif

Saya memiliki file Excel dengan koneksi data eksternal yang disiapkan. Ini menarik data langsung dari database, dan memberi kita sekitar 450 baris. Baris tajuk memungkinkan kita untuk memfilter data di lembar, dan kami menggunakan ini sebagai alat tujuan umum ... Saya akan menggunakan filter untuk mempersempit apa yang saya lihat berdasarkan kriteria yang berubah tergantung pada keadaan.

Seringkali, setelah memfilter data, saya hanya ingin mengirim rekaman yang difilter ke orang lain. Saya ingin menyalin / menempel hanya baris yang tersisa ke Buku Kerja baru untuk dikirim melalui email. Sayangnya, ini tidak berhasil. Saat saya menempelkan data, data masih menempel semua data. Baris yang difilter masih ada di buku kerja ... mereka hanya disembunyikan. Saya ingin mereka benar-benar hilang dari file baru.

Bagaimana saya bisa melakukan ini?

Setelah memfilter data (yaitu, menyembunyikan data yang ingin Anda kecualikan),

  1. Pilih data yang terlihat yang ingin Anda ekspor.
  2. Tipe Alt + ; . Ini adalah jalan pintas untuk perintah Select Visible Cells, yang dapat Anda tambahkan ke Quick Access Toolbar jika Anda mau (lihat di grup "Commands Not in the Ribbon").
  3. Salin, dan tempelkan ke buku kerja baru. Anda akan mendapatkan hanya data yang terlihat.

Intellisense hanya berfungsi jika Anda tersambung ke database SQL2008 atau lebih tinggi. Jika Anda menggunakan studio manajemen untuk terhubung ke server sql2000 atau sql2005, tidak akan ada intellisense.

Juga, ada opsi untuk menyalakan / mematikannya. Di SQL Server Management Studio, klik Tools -> Options. Luaskan "Editor Teks" dan klik "Semua Bahasa". Buat suer "Anggota daftar otomatis dan Informasi parameter" keduanya dicentang.

Standby laptop di Linux / Ubuntu

Salah satu alasan saya saat ini tidak menggunakan Ubuntu adalah karena standby tidak berfungsi dengan baik. Itu melakukan salah satu dari yang berikut:

  • tidak pergi ke standby
  • crash saat standby
  • macet saat melanjutkan dari standby

Ini terjadi di Ubuntu 7.04, 8.04 dan 8.10. Saya belum mencoba 9.04. Saya menggunakan Dell Inspiron 17xx.

Pertanyaannya adalah:

Pada distribusi Linux apa yang sering Anda gunakan untuk aplikasi desktop, apakah Anda tidak bermasalah dengan standby?

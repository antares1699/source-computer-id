Memberikan bahwa folder2 misalnya memiliki file tambahan bernama test4, tetapi mereka berbagi semua file lain yang sama, Anda selalu dapat menggunakan diff.

diff folder1 folder2

Dalam kasus kami, itu akan menghasilkan:

Only in folder2: test4

Dengan asumsi Anda ingin menemukan file / direktori yang hilang di folder2 dari folder1, Anda melakukan:

ls folder2 > /tmp/f2.txt
ls folder1 | grep -v -f /tmp/f2.txt

Itu akan mencantumkan semua item yang ada di folder1 tetapi tidak di folder2, Anda dapat melakukan sebaliknya untuk menemukan kedua perbedaan tersebut.

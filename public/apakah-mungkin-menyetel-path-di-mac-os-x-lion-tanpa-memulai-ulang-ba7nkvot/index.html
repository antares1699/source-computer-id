Apakah mungkin menyetel PATH di Mac OS X Lion tanpa memulai ulang?

Apakah ada cara untuk menyetel PATH pada /etc/paths tanpa perlu mem-boot ulang Mac Anda? Saya tahu saya bisa melakukannya dengan menggunakan alias tapi itu tidak permanen.

Misalnya, saya ingin mengatur PATH untuk MySQL.

Saya berasumsi Anda ingin variabel PATH disetel di lingkungan terminal Anda. Jika Anda menggunakan bash (default di Mac OS X), Anda dapat menambahkan

export PATH=${PATH}:/your/path/to/mysql

ke file .bashrc Anda di direktori home

NET START <SERVICE> - bagaimana / di mana mendapatkan nama layanan?

Saya perlu membuat skrip umum untuk memulai ulang layanan:

net stop <service>
net start <service>

Masalahnya adalah saya tidak tahu nama layanannya. Misalnya untuk "printer spooler" adalah nama "spooler".

Bagaimana cara menemukan nama untuk layanan apa pun?

Anda dapat melihat daftar Layanan Windows Anda dengan mengikuti langkah-langkah berikut:

  1. Klik kanan ikon "Komputer" (biasanya di Desktop Windows atau di Menu Mulai)
  2. Pilih "Kelola" (ini membutuhkan akses Administratif)
  3. Klik dua kali pada "Layanan dan Aplikasi" (untuk mengembangkannya)
  4. Klik "Layanan"

Daftar di sisi kanan di bawah kolom "Layanan" menunjukkan semua layanan Anda, diurutkan menurut abjad secara default.

Catatan: Selain menggunakan perintah "Net", untuk kontrol manual Anda juga dapat menggunakan GUI untuk Memulai / Menghentikan banyak Layanan Windows yang tersedia.

Jalankan services.msc untuk mendapatkan daftar semua layanan.

Klik properti pada layanan yang Anda inginkan, di sana Anda dapat melihat "nama layanan" dan "nama tampilan".

Nama layanan adalah apa yang perlu Anda gunakan dengan perintah net.

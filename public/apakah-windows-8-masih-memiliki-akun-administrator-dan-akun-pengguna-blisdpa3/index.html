Saya menggunakan opsi Instalasi Mudah VMWare, yang membuat Anda memasukkan nama pengguna dan kata sandi untuk apa yang dijelaskannya sebagai akun administrator:

Ada akun Administrasi di Windows8 tetapi disembunyikan secara default, Untuk Aktifkan untuk ditampilkan di daftar pengguna yang terbuka Prompt Perintah dan jalankan perintah di bawah ini:

net user administrator /active:yes

Untuk menonaktifkannya (jangan tampilkan di daftar pengguna)

net user administrator /active:no

Dan jika Anda ingin tahu bahwa Nama Pengguna Anda Akun Lokal atau Akun MS juga Administrator atau Reguler pengguna pergi ke Panel Kontrol> Akun Pengguna dan Keamanan Keluarga> Akun Pengguna .

Itu ditunjukkan di bawah foto kiri.

Anda dapat mengakses Panel Kontrol dengan mengklik kanan pada layar Mulai, memilih "Semua Aplikasi", dan menggulir ke kategori "Sistem Windows".

Sejauh yang saya tahu, windows 8 memperkenalkan konsep baru yang disebut "akun Microsoft" dan "Akun lokal".

  • akun Microsoft

Akun Microsoft adalah akun Windows Live ID dan kata sandi Anda. Administrator dapat masuk dengan mereka dan dia juga dapat menambahkan akun Microsoft lain ke sistem, jadi lain kali siapa pun dapat memilih akun MS mereka sendiri untuk masuk.

  • Akun lokal

Sama seperti sistem windows tradisional, ini termasuk akun administrator dan akun tamu (pengguna). Secara default Anda login sebagai administrator, tetapi Anda dapat mengaktifkan akun tamu di panel kontrol. Kemarin, saya melakukan itu dan menyadari akun tamu sangat terbatas dan hampir setiap program yang Anda mulai, Anda harus memasukkan kata sandi akun administrator. Dan saya belum bisa menemukan cara untuk mengonfigurasi hak istimewa akun tamu.

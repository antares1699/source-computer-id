Cara menemukan dan mengganti jalur tautan di beberapa file Excel sekaligus

Kami memiliki folder besar yang berisi dokumen kantor di server kami. IT ingin mengubah nama folder dari Mehper_repository menjadi Planning_reports. Karena sebagian besar file Excel di subfolder saling terkait, saya perlu mencari dan mengganti setiap konten sel yang berisi \\FileSrv\Mehper_repository\ menjadi \\FileSrv\Planning_reports\. Masalahnya adalah, Excel's Find / Ganti utilitas hanya dapat mencari di dalam Lembar atau Buku Kerja. Bagaimana cara mencari di dalam folder / subfolder hanya dengan mencari di Rumus dan mengganti ekspresi itu?

Saya akan menyarankan produk komersial PowerGREP :

PowerGREP adalah grep Windows yang kuat alat. Pencarian cepat hingga besar jumlah file pada PC Anda atau jaringan, termasuk teks dan biner file, arsip terkompresi, MS Word dokumen, Spreadsheet Excel , PDF file, file OpenOffice, dll. Temukan file informasi yang Anda inginkan dengan kuat pola teks (ekspresi reguler) menentukan bentuk yang Anda inginkan, bukan teks literal. Cari dan ganti dengan satu atau banyak reguler ekspresi secara komprehensif memelihara situs web, kode sumber, laporan, dll. Ekstrak statistik dan pengetahuan dari file log dan besar kumpulan data.

Pendekatan makro:

Saya tidak dapat menjawab ini dengan tepat, tetapi bagaimana dengan menggunakan program makro sehingga satu-satunya hal yang harus Anda lakukan adalah membuka file satu per satu dan menjalankan makro? Itu mungkin layak jika koleksinya tidak terlalu besar, jika tidak, Anda mungkin harus datang dengan pemrograman solusi khusus ...

Memprogram solusi otomatis ubahsuaian:

Berdasarkan jawaban A. Scagnelli, Anda dapat menggunakan bahasa .NET favorit Anda dan melakukannya (contoh C #):

  • Secara rekursif menghitung subdirektori .
  • Hitung setiap file di semua direktori yang ditemukan, filter pada ekstensi excel.
  • Otomatiskan Microsoft Excel dengan cara ini: http://support.microsoft.com/kb/302084
  • Dan ganti teks dengan cara ini: Bagaimana Saya Bisa Mengganti Teks di Spreadsheet Excel?

Kode semu:

openExcel

// Fills the list recursively with aboslute paths of the subfolders of path.
enumerateFolder(list, path)

// Enumerate paths. 
foreach(path in list)
{
   // Enumerate files
   foreach(file in path)
   {
      openFileInExcel
      replaceFileContents
      closeFileInExcel
   }
}

closeExcel

Penjelasan:

Cukup teruskan jalur ke direktori induk dan referensi ke daftar saat Anda memanggilnya secara rekursif. Ini memungkinkan Anda untuk mendapatkan daftar folder file mana yang dapat Anda hitung dalam loop ganda.

Biarkan Excel terbuka selama seluruh algoritme, cukup 'buka, ganti, dan tutup' di loop dalam.

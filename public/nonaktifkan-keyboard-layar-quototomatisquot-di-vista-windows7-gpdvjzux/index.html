Nonaktifkan keyboard layar "otomatis" di Vista / Windows7

Saya menggunakan PC HP multi-sentuh, dan saya ingin menonaktifkan keyboard di layar yang muncul saat kolom input teks mendapatkan fokus.

Ini adalah keyboard yang sama seperti saat Anda menyentuh sisi kiri layar.

Apakah ada cara untuk menonaktifkan keyboard ini?

4
Adam Harte — Sumber

Jawaban Anda bukan untuk apa yang diminta. Ingin mengetahui cara menutup "Panel Input PC Tablet". Ikuti instruksi di bawah ini:

Untuk keluar dari Panel Input Saat Anda mengklik tombol Tutup, Panel Input bergeser kembali ke tepi layar. Jika Anda ingin mematikan Panel Input sepenuhnya sehingga tidak lagi berjalan di desktop Anda, lakukan hal berikut:

1. Buka Panel Input.

2. Klik Alat, lalu klik Keluar.

Untuk menonaktifkannya, ikuti langkah-langkah berikut:

  1. Klik Mulai, lalu klik Panel Kontrol.

  2. Klik Uninstall Program.

  3. Di panel kiri, klik Aktifkan atau Nonaktifkan Fitur Windows.

  4. Kosongkan kotak centang Komponen Opsional PC Tablet.

  5. Klik OK.

  6. Saat diminta, klik Restart Now dan biarkan komputer Anda restart.

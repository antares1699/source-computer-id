Yang ingin Anda lakukan adalah menyiapkan server SIP di LAN Anda. Ada beberapa, maka Anda akan dapat menggunakan klien SIP seperti Ekiga, atau apa pun nama jendela itu, untuk mendapatkan fungsionalitas seperti Skype (VOIP). Ada banyak server seperti itu, seperti asterisk atau freeswitch.

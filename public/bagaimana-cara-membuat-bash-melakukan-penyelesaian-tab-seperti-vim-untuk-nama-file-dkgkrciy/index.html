Bagaimana cara membuat bash melakukan penyelesaian tab seperti vim untuk nama file?

Saya ingin bash memutar nama file yang cocok daripada mencantumkannya, seperti perilaku penyelesaian tab default di vim. Apakah ada perintah mudah untuk melakukan ini? Ini pada dasarnya adalah kebalikan dari pertanyaan ini .

10

Peta Tab sampai menu-complete.

Di ~/.inputrc Anda:

"\C-i": menu-complete

Baca ulang file tersebut agar efektif di shell saat ini dengan menekan Ctrl - x Ctrl - r .

Anda dapat menggunakan perintah berikut pada prompt perintah untuk mengaktifkan fitur untuk sesi saat ini jika Anda ingin mengujinya tanpa mengedit file ~/.inputrc, misalnya.

bind '"\C-i" menu-complete'

  • Apakah ada cara untuk membuat SHIFT + TAB melewati daftar secara terbalik?
    — ubiquibacon
  • @ubiquibacon: Secara umum, di terminal tidak ada perbedaan antara Tab dan Shift + Tab (atau karakter kontrol bergeser vs. non-bergeser lainnya). Namun, Anda dapat menggunakan tombol lain untuk melakukan langkah mundur. bind '"\C-q" "\e-\C-i"' akan menyebabkan Ctrl + Q melakukan ini (atau tambahkan "\C-q" "\e-\C-i" ke ~/.inputrc. Anda mungkin perlu menambahkan stty -ixon ke file ~/.bashrc untuk mengaktifkan Ctrl + Q (dan Ctrl + S). \e- (Escape minus) adalah yang membalikkan arah.
    — Dennis Williamson

Yum Instal perangkat lunak-properti-umum di EC2

Saya mencoba menginstal software-properties-common pada instans mikro EC2 dan mendapatkan error berikut:

$ sudo yum install software-properties-common
Loaded plugins: priorities, update-motd, upgrade-helper
 amzn-main/latest                                                                   | 2.1 kB     00:00     
amzn-updates/latest                                                                | 2.3 kB     00:00     
No package software-properties-common available.

Hasil yang sama mencoba menginstal python-software-properties (yang merupakan paket yang sama untuk versi OS yang berbeda).

Jadi saya menebak bahwa repositori paket amzn-updates dan amzn-main tidak berisi salah satu dari paket ini dan saya mungkin perlu tambahkan repositori ?

Akan melaporkan kembali setelah melihat-lihat sedikit lagi. Sementara itu, masukan lebih dari diterima.

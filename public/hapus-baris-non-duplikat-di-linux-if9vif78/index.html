Hapus baris non-duplikat di Linux

bagaimana cara menghapus baris non-duplikat dari file teks menggunakan program linux linke sed, awk atau lainnya?

Contoh:

abc
bbc
abc
bbc
ccc
bbc

Hasil:

abc
bbc
abc
bbc
bbc

Daftar kedua telah menghapus ccc karena tidak memiliki baris duplikat.

Apakah mungkin juga untuk menghapus baris, yang bukan duplikat DAN baris yang hanya memiliki 2 duplikat, dan meninggalkan yang memiliki lebih dari 2 baris duplikat?

Solusi yang diposting oleh orang lain tidak berfungsi pada Debian Jessie saya: mereka menyimpan file tunggal salinan dari setiap baris duplikat, sementara itu pemahaman saya tentang OP itu semua salinan dari garis duplikat harus disimpan. Kalau sudah paham kan OP, maka ...

  1. Perintah berikut

    awk '!seen[$0]++' file

    menghapus semua baris duplikat.

  2. Perintah berikut

    awk 'seen[$0]++' file

    mengeluarkan semua duplikat, tetapi bukan salinan aslinya: yaitu , jika garis muncul n kali, itu mengeluarkan baris n-1 waktu.

  3. Lalu perintahnya

    awk 'seen[$0]++' file > temp && awk '!seen[$0]++' file >> temp

    memecahkan masalah Anda. Garis tidak dalam urutan aslinya.

  4. Jika Anda menginginkan garis yang memiliki dua atau lebih duplikat, Anda sekarang dapat mengulang yang di atas:

    awk 'seen[$0]++' file | awk 'seen[$0]++' > temp

    simpan n-2 salinan dari baris yang memiliki n> 1 duplikat. Sekarang

    awk '!seen[$0]++' temp > temp1

    menghapus semua baris duplikat dari file temp, dan sekarang Anda dapat memperoleh apa yang Anda inginkan ( yaitu hanya garis dengan n> 1 duplikat) sebagai berikut:

    cat temp1 >> temp; cat temp1 >> temp

  5. Jika Anda perlu melakukan ini untuk garis yang muncul N atau lebih banyak kali, perintah berikut

      awk 'seen[$0]++ && seen[$0] > N' file

    lebih sederhana daripada merangkai N kali perintah awk 'seen[$0]++' file.

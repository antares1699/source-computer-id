OpenWRT mengabaikan tugas terjadwal?

Router OpenWRT 10.03.1 saya (Linksys WRT160NL) mengabaikan crontab saya (diatur melalui Sistem-> Tugas terjadwal). Saya ingin me-reboot router setiap malam, karena router cenderung terkelupas setelah beberapa hari. Jadi saya memasukkan ini dalam tugas saya:

0 5 * * * reboot

Saya juga mencoba

0 5 * * * root /sbin/reboot

tapi ini juga diabaikan. Apakah ada semacam pengaturan yang harus saya lakukan agar cron benar-benar berjalan? Ketika saya masuk ke router melalui ssh, dan menjalankan ps, saya tidak melihat proses cron.

Secara default OpenWrt tidak mengaktifkan layanan cron. Untuk memulainya dan mengaktifkan startup otomatis selama reboot berikutnya, Anda perlu menjalankan perintah berikut:

/etc/init.d/cron start
/etc/init.d/cron enable

Perintah pertama memulai layanan cron satu kali, tetapi tidak mengubah konfigurasi startup, sehingga tidak akan dimulai secara otomatis setelah reboot. Perintah kedua mengubah konfigurasi startup (membuat symlink pada /etc/rc.d) sehingga layanan cron akan dimulai selama boot, tetapi tidak segera memulainya.

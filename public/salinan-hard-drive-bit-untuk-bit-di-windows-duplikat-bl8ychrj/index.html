Clonezilla

Bagaimana dengan CloneZilla ?

Clonezilla, berdasarkan DRBL, Partclone dan udpcast, memungkinkan Anda melakukan backup dan pemulihan bare metal. Dua jenis Clonezilla tersedia, Clonezilla live dan Clonezilla SE (edisi server). Clonezilla live cocok untuk pencadangan dan pemulihan mesin tunggal. Meskipun Clonezilla SE untuk penyebaran besar-besaran, Clonezilla SE dapat mengkloning banyak komputer (40 plus!) Secara bersamaan. Clonezilla hanya menyimpan dan memulihkan blok yang digunakan di harddisk. Ini meningkatkan efisiensi klon. Di Ruang Kelas C NCHC, Clonezilla SE digunakan untuk mengkloning 41 komputer secara bersamaan. Hanya butuh sekitar 10 menit untuk mengkloning citra sistem 5,6 GBytes ke 41 komputer melalui multicasting!

Norton Ghost

Apakah Anda benar-benar menginginkan salinan bit-demi-bit? Apakah Anda benar-benar ingin menyimpan bit yang ada di ruang kosong di disk Anda? Apakah Anda benar-benar peduli dengan apa yang ada di ruang kosong?

Saya pikir Anda hanya ingin mendapatkan semua data Anda dari drive lama ke drive baru (termasuk hal-hal sepele, seperti boot sector dan tabel partisi), tetapi Anda tidak perlu bit-demi-bit.

Preferensi pribadi saya untuk memindahkan seluruh partisi atau drive adalah Norton Ghost .

EaseUS Todo Backup Gratis

Saya suka EaseUS® Todo Backup Free 3.0 untuk gambar disk dan kloning.

EaseUS Todo Backup Free mengintegrasikan klon disk dan pencadangan & pemulihan data yang kuat. Ini menyediakan serangkaian fitur pencadangan & pemulihan terkaya

  • Salinan dan klon disk / partisi: Salin disk / partisi atau volume dinamis dalam file dengan mode file untuk menghemat waktu dan ruang.

  • Pemulihan universal sekali klik: Kembalikan sistem ke perangkat keras yang berbeda untuk penggantian perangkat keras

  • Pencadangan sistem sekali klik: Cukup satu klik untuk mencadangkan & memulihkan sistem dan aplikasi Anda ke gambar di Windows atau dengan CD yang dapat di-boot.

  • Penyimpanan cadangan: Sistem cadangan atau data ke hard drive lokal, hard drive eksternal, drive USB, CD / DVD atau server FTP jarak jauh.

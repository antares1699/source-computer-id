Jika saya membeli 64-bit, apakah saya masih dapat menggunakan info dari komputer 32-bit pada komputer 64-bit, dan kemudian mengambilnya kembali dan memulihkannya ke 32-bit?

Konten file itu sendiri tidak ditentukan oleh arsitektur sistem operasi. Misalnya, dokumen teks kompatibel secara universal dengan semua komputer (terlepas dari cara menangani akhir baris). Tidak masalah jika Anda menggunakan 32-bit atau 64-bit. Logika yang sama ini dapat diperluas ke sebagian besar data file (jadi dokumen MS Word / Excel, model 3D, dll ...).

Saat kita berurusan dengan dapat dieksekusi program, semuanya tergantung. Secara umum, asumsikan bahwa program biner hanya akan mengeksekusi arsitektur asalnya. Ketahuilah bahwa Anda tidak dapat menggunakan program 64-bit pada sistem 32-bit.

Apa yang Anda maksud dengan "info"? Apakah Anda sedang menulis program? Desain web? Dokumen?

Desain web, pada umumnya, seperti dokumen, tidak akan memiliki masalah apa pun dalam mentransfer pekerjaan pada komputer dengan panjang instruksi yang berbeda, karena mereka adalah fungsi dari program yang menafsirkannya.

Pemrograman yang dapat dieksekusi umumnya akan berfungsi jika Anda menggunakan kerangka kerja aplikasi (seperti java) dan sebagian besar sistem operasi dapat menjalankan aplikasi 32-bit pada perangkat keras 64-bit tanpa banyak masalah.

Jadi jika "info" yang Anda transfer antara komputer "yang memiliki bit berbeda" terdiri dari salah satu dari yang di atas, Anda tidak akan mengalami masalah dan sebaiknya lanjutkan dan nikmati lebih banyak bit.

Masalah hanya akan terjadi jika Anda menulis kode program sendiri dan menggunakan instruksi yang bergantung pada arsitektur 64bit yang ingin Anda jalankan pada sistem 32bit. Seperti yang saya katakan di atas, komputer 64bit cukup mampu menjalankan perangkat lunak 32bit. Saat Anda memiliki program yang menggunakan instruksi 64bit secara eksplisit dan Anda mencoba menjalankannya di komputer 32bit, Anda akan mengalami masalah.

Saya menjalankan lingkungan campuran di tempat kerja saya. Beberapa orang menggunakan Windows XP 32bit, beberapa Windows Vista 32bit, beberapa Windows 7 32bit dan beberapa Windows 7 64bit. Tidak ada masalah untuk berbagi pekerjaan di antara karyawan mana pun. 99% program yang bekerja pada platform 32bit akan bekerja pada platform 64bit kecuali mereka memerlukan driver khusus yang bergantung pada platform.

Tidak ada perbedaan penggunaan untuk semua jenis dokumen yang akan dibagikan antara 2 arsitektur tersebut.

Jadi jika Anda akan membeli komputer, gunakan sistem 64bit. Akan ada lebih banyak masalah bagi Anda untuk menjalankan sistem 32bit jika Anda ingin menjalankan aplikasi 64bit (yah itu tidak mungkin) kemudian sebaliknya.

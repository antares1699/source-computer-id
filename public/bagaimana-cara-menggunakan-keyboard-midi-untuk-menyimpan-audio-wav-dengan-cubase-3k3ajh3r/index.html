Bagaimana cara menggunakan keyboard MIDI untuk menyimpan audio wav dengan Cubase?

Saya memiliki pengaturan berikut: Yamaha DGX-505, PC Windows 10, driver USB MIDI Yamaha, Manajer Studio Yamaha, dan Cubase LE AI Elements 8. Saya telah mencapai titik di mana saya dapat menambahkan trek MIDI di Cubase, mainkan catatan di keyboard dan melihatnya muncul di Cubase.

Masalahnya adalah ketika saya mencoba mengekspor file sebagai wav atau mp3, yang saya dengar hanyalah keheningan. Apa yang kulewatkan di sini?

Setelah merekam, ketika saya menekan tombol play, saya hanya mendengar output pada keyboard yang sama dengan yang saya gunakan untuk memainkan not. Saya tidak bisa mendengarnya di PC itu sendiri. Saya telah mencoba mengubah output trek MIDI menjadi Microsoft GS Wavetable Synth internal, dan telah mengaktifkan kedua output di Studio Manager

tapi saya masih tidak mendengar apa-apa di PC saya.

MIDI pada dasarnya merekam penekanan nada dan tweak parameter.

Apa yang Anda rekam saat ini ditransmisikan kembali ke keyboard midi, itulah mengapa ini berfungsi.

Yang ingin Anda lakukan adalah mengunduh Instrumen VST (plugin yang merupakan penyintesis) dan menetapkan trek midi Anda sebagai Instrumen VST. Kemudian Anda ingin mematikan suara keyboard. Sekarang, jika semua dilakukan dengan benar, saat Anda bermain di keyboard, speaker komputer Anda harus memainkan suara yang dihasilkan Instrumen VST.

Setelah merekam, memutar lagu, Instrumen VST masih akan membuat suara, dan ekspor gelombang / mp3 tidak akan diam.

Jika Anda benar-benar ingin menggunakan suara keyboard Anda, Anda harus menghubungkan kabel antara saluran keluar keyboard dan saluran masuk komputer, dan memastikan bahwa Cubase juga memantau saluran masuk tersebut.

Pada akhirnya, ketika Anda memainkan musik di keyboard, Anda akan diminta untuk mendengarkannya sepanjang waktu melalui speaker komputer. Jika itu tidak terjadi, ekspor wave / mp3 akan dibungkam.

Layar desktop jarak jauh: Font dan ikon di jendela desktop jarak jauh berukuran sangat kecil

Saya terhubung ke windows server 2012 melalui remote desktop.

Komputer jarak jauh adalah server Terminal di mana minimal 240 pengguna terhubung.

Font dan ikon di jendela desktop jarak jauh sangat kecil, setiap pengguna lain yang terhubung ke server yang sama tidak memiliki masalah dengan ukuran font atau ukuran jendela, semuanya tampak normal.

Tetapi ketika saya terhubung ke remote, ukuran jendela dan font sangat kecil. saya tidak dapat menggunakan aplikasi apa pun di remote karena ukuran font. Mohon saran ..

** Saya mencoba dengan mengubah pengaturan tampilan dan mengatur ukuran layar menjadi lebih besar pada Koneksi RDP tetapi tidak berhasil.

** Pertama saya telah terhubung melalui komputer dengan DPI normal dan menutup sesi saya (Keluar). Saya mencoba keesokan harinya dengan masuk RDP dari Laptop masih masalah yang sama: /.

Ini mungkin disebabkan oleh fakta bahwa protokol RDP melakukan pengalihan DPI. Jika Anda masuk (awalnya) dari mesin dengan tampilan normal (mis .: 96dpi), maka Anda terhubung dari laptop yang memiliki DPI tinggi tanpa keluar (mis .: laptop 1080p 13 "apa pun yang menampilkan 120dpi), semuanya akan terlihat kecil karena sesi akan menggunakan setelan DPI yang telah disetel di awal.

Cara mengatasinya adalah masuk dari komputer baru, simpan dan tutup semua program, buka cmd (Win + R -> cmd) dan ketik logoff. Tunggu 1-2 menit, lalu sambungkan kembali. Kali ini, semua aplikasi dan font harus dirender dengan resolusi yang tepat.

Selain opsi yang tercantum di atas (OpenOffice bagus, BTW):

  • iWorks akan membuka file-file ini, saya yakin. (bukan solusi gratis, tetapi Mac Anda mungkin memiliki iWorks.)
  • Open XML File Format Converter untuk Mac akan memungkinkan Anda untuk membuka di versi sebelumnya dari Office untuk Mac (bukan solusi gratis juga, tetapi Anda mungkin memiliki salinan dari versi yang lebih lama)
  • Kacang (gratis tetapi beberapa batasan) www.bean-osx.com/Word97.html Dukungan Word 2007 (.docx) di Bean:

Bean menggunakan layanan konversi file Apple untuk membuka dan menyimpan file format Word 2007. Harap dicatat bahwa karena keterbatasan konverter format file, gambar hilang saat membuka file format Word 2007 asli di Bean dan saat menyimpan dokumen teks kaya sebagai file format Word 2007 di Bean. Yang juga hilang adalah penyorotan, tabel, daftar teks, dan pemformatan paragraf lanjutan. Maaf, beginilah cara Apple menerapkannya. Pengguna akan diperingatkan tentang hal ini saat menimpa file format Word 2007 yang tidak dibuat oleh Bean, untuk mencegah hilangnya informasi. Bean tidak boleh dianggap sebagai pengganti Word, atau pengolah kata lainnya yang dirancang untuk dokumen kompleks yang melibatkan, misalnya, kutipan.

Jika Anda memiliki fusi VMWare, Parallels, atau beberapa teknologi virtualisasi lain dan gambar windows, Anda dapat menginstal Windows Word 2007 Viewer gratis.

Anda juga dapat menggunakan layanan ini:

www.docx2doc.com

atau docx-converter.com

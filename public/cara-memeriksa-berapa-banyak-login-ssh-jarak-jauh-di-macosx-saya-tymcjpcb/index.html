Cara memeriksa berapa banyak login ssh jarak jauh di macosx saya

Saya mengizinkan login ssh jarak jauh ke Mac saya. Namun saya hanya dapat melihat login lokal saya saat menjalankan perintah WHO . Saya yakin sesi jarak jauh telah masuk ke Mac saya melalui ssh.

localhost% who
kane     console  Jul 10 22:21 
kane     ttys000  Jul 10 22:21 
kane     ttys001  Jul 10 22:21 
kane     ttys002  Jul 10 22:21 
kane     ttys003  Jul 10 22:21

Di host linux, ia memasukkan Mac saya melalui perintah di bawah ini,

user     12098 11413  0 21:01 ?        00:00:00 /usr/bin/ssh -L 27002:127.0.0.1:27002 -R 27002:127.0.0.1:27003 -q -N -o stricthostkeychecking=no -R 20008:localhost:6500 -R 20002:localhost:22 username@myhost.com

Tapi di host linux, WHO perintah dapat mencantumkan login ssh. Bagaimana saya bisa mendapatkan login ssh jarak jauh yang terdaftar di Mac saya?

user@ubuntu-server:~$ who
user     tty1         2012-07-05 18:26
user     pts/1        2012-07-23 22:10 (123.120.xxx.xxx)

Saya tidak memiliki Mac untuk diperiksa, tetapi di Unix umumnya ada beberapa perintah lagi untuk menunjukkan siapa yang masuk:

  • siapa (Anda sudah menyebutkan)
  • w (dengan informasi rinci)
  • pengguna (Anda juga mencoba; hanya menunjukkan nama pengguna yang saat ini masuk)
  • jari (dengan beberapa informasi pribadi)

Jadi mungkin salah satu yang belum dicoba (yaitu w, jari) melakukan tugasnya dengan benar untuk Anda? Jika tidak, beberapa kemungkinan lainnya terutama untuk OSX tercantum dalam artikel osx - Bagaimana mengetahui kapan dan pengguna mana yang masuk ke sistem .

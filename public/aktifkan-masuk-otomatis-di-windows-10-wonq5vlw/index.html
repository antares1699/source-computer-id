Aktifkan masuk otomatis di Windows 10

Sulit untuk mengetikkan kata sandi saya setiap kali saya memulai VM Windows 10 saya, terutama karena Anda tidak dapat menempelkan kata sandi ke dalam kotak. Saya ingin mengatur login otomatis.

Mengikuti instruksinya sini Saya mencoba untuk menonaktifkan opsi "Pengguna harus memasukkan nama pengguna dan sandi" pada netplwiz. Tapi pilihannya tidak ada ! Ini harus berada di atas daftar "pengguna untuk komputer ini". Apakah ada pengaturan yang perlu saya ubah untuk memungkinkan saya mengubah pengaturan?

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\PasswordLess\Device]
"DevicePasswordLessBuildVersion"=dword:00000000

0 = aktif
2 = mati

Kamu bisa gunakan Autologon.exe (Alat Sysinternals) atau menggunakan kunci registri:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon]
"AutoAdminLogon"="1"
"DefaultDomainName"="DOMAINNAME"
"DefaultUserName"="USERNAME"
"DefaultPassword"="PASSWORD"

Diekstrak dari: http://www.sysadmit.com/2016/01/windows-configurar-autologin.html

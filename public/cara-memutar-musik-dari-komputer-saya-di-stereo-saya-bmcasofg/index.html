Cara memutar musik dari komputer saya di stereo saya

Apa sajakah cara yang baik untuk memutar musik di komputer saya melalui stereo / receiver di ruangan lain?

Saya telah menggunakan Linksys Music Bridge, tetapi tidak dapat diandalkan (hanya terhubung melalui nirkabel ketika rasanya seperti itu), dan tidak mendukung Windows 7 yang baru saja saya instal.

Saya memiliki router nirkabel Linksys.

Terima kasih!

Salah satu caranya adalah dengan mendapatkan file Penerima Kotak Pemeras dan instal Server Kotak Pemeras di PC Anda. Anda kemudian dapat melakukan streaming secara nirkabel di sekitar rumah. Penerima biasanya dibundel dengan pengontrol sebagai Squeezebox Duet , tetapi jika Anda memiliki iPod Touch atau iPhone, Anda bisa mendapatkan file iPeng aplikasi untuk mengontrolnya.

Ini mungkin terlihat berlebihan untuk koneksi ke satu stereo. Keuntungan dari pendekatan Kotak Penjepit adalah Anda dapat menambahkan beberapa penerima (klien lain, lihat di bawah), dan mengalirkan musik ke setiap penerima satu per satu, atau menyinkronkannya untuk mendapatkan audio multi-ruangan.

Ada beberapa lainnya Produk kotak pemeras Anda mungkin ingin mempertimbangkan juga.

Saya memiliki Squeezebox Server yang berjalan sebagai layanan di Ubuntu Acer Revo saya dan menyukainya, lihat ini pertanyaan di situs SE saya untuk lebih jelasnya.

Saya pribadi menggunakan Creative Xmod Wireless dalam pengaturan saya saat ini karena saya memiliki persyaratan aneh untuk memerlukannya untuk diputar secara bersamaan di dua lokasi.

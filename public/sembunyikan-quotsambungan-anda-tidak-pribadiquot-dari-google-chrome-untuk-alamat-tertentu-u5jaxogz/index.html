Sembunyikan "Sambungan Anda Tidak Pribadi" dari Google Chrome untuk Alamat tertentu?

Kami memiliki situs web internal (di belakang VPN kami) yang menggunakan https / ssl. Setiap kali saya mengunjunginya, saya mendapatkan pesan kesalahan ini:

Koneksi Anda tidak bersifat pribadi

Penyerang mungkin mencoba mencuri informasi Anda dari address.removed.com (misalnya, sandi, pesan, atau kartu kredit).

NET::ERR_CERT_AUTHORITY_INVALID

Server ini tidak dapat membuktikannya address.removed.com ; sertifikat keamanannya tidak dipercaya oleh sistem operasi komputer Anda. Ini mungkin disebabkan oleh kesalahan konfigurasi atau ada penyerang yang memotong sambungan Anda.

Apa yang dapat saya lakukan untuk memperbaikinya? Bisakah saya mengalamatkan alamat ke daftar putih? Dapatkah saya memasang pin pada sertifikat ssl atau sesuatu agar dapat diterima?

Saya menggunakan Chrome di Windows 7.

Berdasarkan komentar Anda, tampaknya Anda tidak tahu cara mengimpor sertifikat ke penyimpanan sertifikat Windows dan juga tidak memahami cara mengekspor sertifikat itu sendiri dari browser.

Panduan berikut akan memungkinkan sertifikat apa pun yang diekspor dan diimpor dengan cara ini berfungsi dengan Internet Explorer, Microsoft Edge, dan Google Chrome. Ini juga akan berfungsi untuk semua browser yang menggunakan penyimpanan sertifikat pengguna di Windows. Firefox menggunakan penyimpanan sertifikatnya sendiri, karena Anda tidak menggunakan Firefox, panduan ini tidak akan berlaku untuk itu sehingga proses tidak akan didokumentasikan.

Bagaimana cara saya memperoleh sertifikat?

  1. Klik pada kunci aman seperti yang ditunjukkan di bawah ini

  2. Klik Koneksi seperti yang ditunjukkan di bawah ini

  3. Klik Informasi Sertifikat seperti yang ditunjukkan di bawah ini

  4. Klik Detail seperti yang ditunjukkan di bawah ini

  5. Klik Salin ke File seperti yang ditunjukkan di bawah ini

  6. Klik Lanjut seperti yang ditunjukkan di bawah ini

  7. Klik Lanjut seperti yang ditunjukkan di bawah ini

  8. Buat file dan tekan Lanjut seperti yang ditunjukkan di bawah ini

    1. Pilih Selesai seperti yang ditunjukkan di bawah ini

  1. Selesai

Bagaimana cara menambahkannya ke penyimpanan sertifikat di Windows 7?

  1. Jalankan perintah berikut

  1. Lakukan operasi Seret dan Jatuhkan pada file yang dibuat.

  1. Seret dan Jatuhkan file yang dibuat ke dalam Otoritas Sertifikasi Root Tepercaya gudang

  1. Selesai

Apakah ada klien baris perintah untuk pemutaran 8tracks.com? [Tutup]

Saya harap saya tidak melakukan spamming dengan pertanyaan semacam ini di sini, tetapi: apakah ada cara untuk bermain 8tracks.com daftar putar dari baris perintah Linux menggunakan, misalnya, mplayer atau mpg123?

Ada 8tracks.com API , tetapi sejauh ini saya belum menemukan klien baris perintah berdasarkan itu. Dan sebagai non-coder, saya juga tidak tahu bagaimana menulisnya sendiri.

Saya lebih suka solusi Perl / Python, tapi itu tidak penting. Terima kasih banyak atas petunjuknya!

EDIT: Ini adalah proyek di Haskell, tetapi saya belum mencobanya sendiri: https://github.com/vikraman/8tracks

Jika ada yang membutuhkan otorisasi pengguna yang cepat dan kotor, ikuti API resmi atau jalankan saja perintah curl --request POST -d "login=USER&password=PWD" https://8tracks.com/sessions.xml.

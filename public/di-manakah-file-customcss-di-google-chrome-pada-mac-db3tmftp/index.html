Di manakah file custom.css di Google Chrome pada Mac?

Saya menggunakan MacOS 10.8.3 dan Chrome Versi 26.0.1410.65. Dan saya ingin mencoba mengubah gaya devtools.

Ketika meletakkan kode ini di Terminal:

 ~/Library/Application Support/Google/Chrome/Default/User StyleSheets/Custom.css

Dapatkan kesalahan ini:

-bash: /Users/Baris/Library/Application: No such file or directory

Saya memeriksa jalur direktori dan benar-benar saya belum memiliki direktori Chrome pada Application Support/Google

Di mana custom.css disimpan di Mac saya?

Saat Anda menggunakan shell seperti Bash dan memasukkan jalur yang berisi spasi, spasi putih tersebut akan dianggap sebagai pemisah argumen. Sebagai contoh,

~/Library/Application Support/Google/Chrome/Default/User StyleSheets/Custom.css

akan dilihat oleh shell sebagai tiga argumen:

~/Library/Application 
Support/Google/Chrome/Default/User 
StyleSheets/Custom.css

Logikanya, ini tidak ada. Saat Anda menggunakan open untuk membuka file CSS, ini akan memberi tahu Anda:

File / Users / werner / Library / Application, / Users / werner / Support / Google / Chrome / Default / User, dan /Users/werner/StyleSheets/Custom.css tidak ada.

Jadi, Anda perlu mengutip variabel Anda , atau keluar dari ruang:

open -e "~/Library/Application Support/Google/Chrome/Default/User StyleSheets/Custom.css"
open -e ~/Library/Application\ Support/Google/Chrome/Default/User\ StyleSheets/Custom.css

Lihat juga Kutipan dan melarikan diri . Untuk detailnya yang berdarah, lihat posting ini: Nama file di Shell

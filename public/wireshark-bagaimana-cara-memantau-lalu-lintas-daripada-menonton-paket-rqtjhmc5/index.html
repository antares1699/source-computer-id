WireShark: Bagaimana cara memantau lalu lintas, daripada menonton paket?

WireShark pandai menunjukkan kepada saya setiap paket yang dikirim atau diterima melalui antarmuka.

Tapi saya benar-benar mencoba menemukan cara untuk memonitor lalu lintas di mesin. Dengan kata lain, daripada melihat:

00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532
00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532
00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532
00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532
00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532

Saya ingin melihat:

00-03-FF-54-D8-DF => 00-03-FF-54-D8-DE  TCP  10.0.0.11:32532 => 10.0.0.12:80  5
00-03-FF-54-D8-DE => 00-03-FF-54-D8-DF  TCP  10.0.0.12:80 => 10.0.0.11:32532  5

Dan jika memungkinkan, logika yang lebih maju:

Listen MAC        Listen Addr     Source MAC        Source Addr     Proto  In  Out
================= ============    ================= =============== =====  ==  ==
00-03-FF-54-D8-DE 10.0.0.12:80 <= 00-03-FF-54-D8-DF 10.0.0.11:32532 TCP    5   5
00-03-FF-54-D8-DE 10.0.0.12:80 <= 00-03-FF-54-D8-D3 10.0.0.42:53213 TCP    5   5

Dengan kata lain, agregasi dan pengelompokan.

Bagaimana saya bisa menggunakan WireShark untuk memantau lalu lintas, daripada menangkap paket?

Wireshark adalah program yang hebat, tetapi ada alternatif freeware dan open source lain yang mungkin lebih mudah digunakan. Favorit saya saat ini adalah Monitor Jaringan Microsoft dan SmartSniff .

Menggunakan SmartSniff dalam mode ringkasan harus memberikan informasi yang Anda inginkan. Opsi> Opsi Lanjutan> Mode ringkasan: aktifkan; Opsi> Opsi Lanjutan> Ambil informasi proses: aktifkan; Options> Resolve IP Addresses: aktifkan.

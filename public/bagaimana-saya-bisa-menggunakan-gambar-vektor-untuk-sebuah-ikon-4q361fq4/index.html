Bagaimana saya bisa menggunakan gambar vektor untuk sebuah ikon?

Saat saya menggunakan Ctrl + roda mouse, saya dapat mengubah skala ikon di desktop saya. Beberapa ikon, naik ke ukuran sewenang-wenang seperti Skype dan Recycle Bin.

Namun beberapa tidak. Mereka tetap pada 256x265.

Saya yakin beberapa program menggunakan ikon vektor.

Format vektor apa yang diterima oleh windows 7 / Vista untuk ikon?

Edit: Ini jelas bukan 256x256.

Windows tidak dapat menggunakan file vektor untuk ikon secara native, hanya saja dari Vista dan seterusnya ukuran ikon maksimum ditingkatkan menjadi 256x256 dan bahkan sekarang tidak semua pengembang pihak ketiga telah menyusul.

Program menyimpan beberapa versi ikon mereka dalam file sumber daya yang berada di exe atau dll. Jika versi resolusi tinggi tidak tersedia maka Windows mungkin tidak berusaha untuk menskalakannya ke ukuran penuh. Itulah mengapa Putty memiliki ikon kecil dan Skype tidak.

Lihat sini untuk detail tentang menemukan ikon versi besar.

Namun Anda dapat menggunakan program vektorisasi untuk membuat versi vektor dari ikon yang ada sehingga Anda dapat memperbesar dan menyimpannya pada resolusi yang lebih tinggi.

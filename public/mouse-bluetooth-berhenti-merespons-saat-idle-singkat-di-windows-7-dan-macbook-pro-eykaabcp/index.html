Mouse Bluetooth berhenti merespons saat idle singkat di Windows 7 dan Macbook Pro

Saya memiliki Microsoft Notebook Mouse 5000 dan saya telah memasangkannya dengan Macbook Pro saya (Akhir 2007) yang menjalankan Windows 7 x64. Masalahnya adalah, setiap kali mouse diam selama lebih dari beberapa detik, setiap kali saya memindahkannya lagi, dibutuhkan 1-2 detik untuk bangun, yang sangat mengganggu.

LANGKAH PERTAMA

  1. Pergi ke Control Panel
  2. Buka Pengaturan Lainnya
  3. Cari opsi BLUETOOTH dan bluetooth akan muncul di PERANGKAT DAN PRINTER
  4. Pilih UBAH PENGATURAN BLUETOOTH
  5. Pilih TAB PERANGKAT KERAS
  6. Pilih GENERIC BLUETOOTH ADAPTOR
  7. Klik Tombol SIFAT
  8. Klik tombol UBAH PENGATURAN
  9. Pilih Tab MANAJEMEN DAYA
  10. Hapus centang pada kotak yang bertuliskan "Izinkan komputer mematikan perangkat untuk menghemat daya" dan OK jalan keluar. (2)

Oleh Darren in http://social.msdn.microsoft.com/Forums/en-US/5c2a89ab-051d-44f3-ab01-1628a60141ed/bluetooth-mouse-issues?forum=samsungpcgeneral

TAHAP KEDUA

Cegah kartu PCI express memasuki mode hemat daya. Lihat gambar terlampir .... oops gambar hanya untuk geek!

Buka opsi daya / pengaturan lanjutan / PCI express dan atur pengaturan ke OFF.

Mengambil petunjuk dari ini , Saya menyadari masalahnya bukan pada mouse, melainkan pada sistem. Jadi saya telah mengubah dari driver Apple Bluetooth Adapter (diinstal oleh BootCamp - versi saya adalah 3.2) ke driver Generic Bluetooth Adapter dan sekarang penundaan bangun menganggur minimal, hampir tidak terlihat.

Untuk melakukan ini:

  1. Buka Device Manager (Mulai | cari "Device Manager")
  2. Luaskan "Radio Bluetooth"
  3. Klik dua kali "Apple Bluetooth Adapter"
  4. Pilih tab Driver
  5. Klik Perbarui Driver
  6. Pilih "Jelajahi komputer saya untuk perangkat lunak driver"
  7. Pilih "Biarkan saya memilih dari daftar driver perangkat di komputer saya"
  8. Pilih "Generic Bluetooth Adapter"
  9. Klik OK dan tunggu saat menginstal. Mungkin perlu sedikit waktu, tetapi pada akhirnya, semua perangkat Bluetooth Anda harus tersedia.

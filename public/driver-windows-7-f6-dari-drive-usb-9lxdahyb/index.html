Driver Windows 7 F6 dari Drive USB

Saya melakukan penginstalan baru Windows 7 dan saya memiliki RAID perangkat lunak terpasang. Saya "perlu" menginstal driver melalui proses F6 yang terkenal itu. Secara teknis, itu bisa dilakukan setelah OS di-boot, tapi singkatnya, ini tidak akan bekerja dengan baik untuk saya.

Apakah mungkin untuk menginstal driver F6 selama instalasi menggunakan USB, atau haruskah itu selalu dilakukan dengan menggunakan disket? Apa alternatif saya?

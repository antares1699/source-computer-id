Katakanlah Anda memiliki aplikasi bernama TestParam yang mengambil sejumlah parameter dan menggemakannya kembali di baris baru:

>TestParam thisisatest
thisisatest

Ketika ada spasi, itu menganggapnya sebagai pemisah antara parameter:

>TestParam this is a test
this
is
a
test

Ini sama dengan mengetik

>TestParam thisisatest
thisisatest
dan pada dasarnya Anda melewatkan empat string berbeda. Jika Anda ingin meneruskan satu parameter yang memiliki spasi:

>TestParam "This is a test"
This is a test

Ketika Anda melihat kesalahan, ia mengatakan tidak dapat menemukan parameter untuk "DISTRIBUSI \ engineer \ drft-tm \ 60kV" yang merupakan parameter kedua yang diteruskan oleh perintah Anda karena Anda memiliki spasi. Jika jalur Anda memiliki spasi, tidak apa-apa, tetapi Anda perlu memberi tahu PowerShell bahwa itu semua adalah satu parameter dan semuanya hanya satu string:

Set-Location "H:\09- DISTRIBUTION\engineer\drft-tmp\310-DOUGLAS\60kV BLDG Design DWGs"

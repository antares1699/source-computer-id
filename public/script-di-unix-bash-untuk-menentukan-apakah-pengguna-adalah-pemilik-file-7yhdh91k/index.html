Script di UNIX (Bash) untuk menentukan apakah pengguna adalah pemilik file

Sesuatu yang membutuhkan sejumlah argumen, di mana setiap argumen akan menjadi jalur ke file atau direktori. Jika pengguna memiliki jalur tertentu, maka pengguna harus memeriksanya untuk melihat apakah jalur tersebut mewakili file normal DAN apakah file tersebut dapat dieksekusi. Jika ya, maka skrip Anda harus mengeksekusi / menjalankan file tersebut.

Terima kasih atas bantuannya. Saya telah mencoba berbagai hal terlalu lama dan saya frustrasi.

magic() {
    for p in "$@"; do
        [ -O "$p" -a -x "$p" ] && /bin/sh "$p"
    done
}

baca 'uji manusia' untuk melihat apa yang dilakukan pemeriksaan tersebut.

Tes -O memeriksa apakah pengguna saat ini adalah pemilik file atau folder.

if [ -O "$FILENAME" ]; then echo 'Owned!'; else echo 'Nope!'; fi

Lihat daftar operator uji file (dan bagaimana melakukan tes secara umum .)

Menggunakan stat di * nix untuk tujuan ini sulit dilakukan secara portabel karena perbedaan antar platform.

Untuk mendapatkan user id saat ini, Anda melakukan:

id -u

untuk mendapatkan pemilik file, Anda melakukan:

stat -c "%u" file.name

untuk menguji apakah satu nilai sama dengan yang lain yang Anda lakukan:

if [ "$first" -eq "$second" ]
then
    ....
fi

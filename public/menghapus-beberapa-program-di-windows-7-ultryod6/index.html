Menghapus beberapa program di Windows 7?

Pertanyaan ini memiliki maksud yang berbeda dengan " Mengapa Anda tidak dapat menghapus beberapa program sekaligus di Windows? "

Jawaban dalam pertanyaan itu terfokus dengan ketat tentang mengapa windows tidak mengizinkan " serentak "pembatalan instalasi program.

Saya tertarik dengan cara mencopot beberapa program TANPA mengklik tanpa henti "Anda yakin?" atau dialog yang mengganggu saya tentang ketergantungan untuk setiap item yang ingin saya hapus. Saya tidak peduli jika program dihapus satu per satu, atau secara bersamaan selama saya tidak harus duduk di sana, memilih setiap program dan menjawab dialog tentangnya. Dalam situasi di mana Anda perlu menghapus 20+ item, ini dapat menambah banyak waktu.

Pasti ada cara sederhana untuk melakukan ini, bukan?

Cara saya melakukannya sekarang adalah dengan membuka Panel kontrol, ketik istilah pencarian di kotak teks "Program dan Fitur Pencarian", lalu hapus satu per satu setiap item yang perlu saya hapus. Ini bagus untuk 1 atau 2 item, tetapi jika ada banyak lalu apa pilihannya? Saya berharap saya bisa membuat beberapa pilihan dan menghapusnya secara non-interaktif (setelah mungkin bertanya apakah saya sangat yakin).

Meskipun pertanyaan ini mirip dengan " Menghapus program secara diam-diam melalui CMD ", jawaban yang diterima sini menyediakan metode untuk mencari dan menghapus program dengan mudah. Ini secara substansial berbeda dari membuat skrip untuk menghapus satu program "secara diam-diam" melalui skrip. Salah satu jawaban lain juga memberikan alternatif GUI untuk menghapus banyak program - sekali lagi, sangat berbeda dari pertanyaan lainnya.

Anda bisa menggunakan PowerShell dan WMI untuk menemukan program berdasarkan pola pencarian, dan kemudian mengeluarkan penginstalan.

Berikut adalah contoh skrip yang berhasil saya gunakan berkali-kali:

$apps = Get-WmiObject -Query "SELECT * FROM Win32_Product WHERE Name like '%Partial Product Name%'"

foreach ($app in $apps) {
    "Name = " + $app.name
    $app.Uninstall()
}

Jadi, misalnya, mengubah filter ke '%Microsoft%' akan mencoba menghapus semua program yang terdaftar di Tambah / Hapus program yang memiliki kata "Microsoft" di namanya.

Anda juga bisa memperluas kueri WMI ( WQL ) dengan OR perintah untuk mencari lebih dari satu pola pada waktu yang bersamaan.

Info lebih lanjut:

Linux: dapatkah saya membuat direktori yang tidak ada saat membuat file baru?

Dapat saya lakukan:

mkdir messages

lalu:

touch messages/hello.txt

Apakah ada perintah yang akan melakukan keduanya - buat direktori jika tidak ada, lalu file kosong? Sesuatu seperti:

touch -p messages/hello.txt

Tidak dengan apa pun di coreutils, tapi inilah fungsi bash untuk itu:

mktouch() {
  mkdir -p "$(dirname "$1")"
  touch "$1"
}

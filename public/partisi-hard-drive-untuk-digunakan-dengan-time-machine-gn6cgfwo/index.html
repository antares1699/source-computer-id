Partisi Hard Drive untuk digunakan dengan Time Machine

Apakah mungkin untuk mempartisi hard drive menjadi dua dan menggunakan satu partisi untuk Time Machine dan yang lainnya untuk kegunaan lain? Saya menjalankan Mac OS X 10.6.5 jika ada perbedaan.

EDIT : Untuk klarifikasi untuk referensi di masa mendatang: Saya ingin melakukan ini dengan hard drive eksternal.

Selain itu, TM akan menghabiskan semua ruang kosong di partisi itu, yang bisa jadi sedikit merepotkan.

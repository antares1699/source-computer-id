Solusi saya adalah pergi ke / dev / snd; sudo chmod a + rw *; pulseaudio -k

Saya perhatikan bahwa baik pengaturan suara gnome atau alsamixer tidak mengambil perangkat sebagai pengguna biasa, tetapi alsamixer mengambil perangkat sebagai root. Semuanya baik-baik saja sekarang.

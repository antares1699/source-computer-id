Dapatkah saya menghapus file kosong yang tertinggal di desktop saya dengan penghapusan virus?

Komputer saya terinfeksi virus. Saya memindainya dengan AVG dan membersihkannya. Itu meninggalkan saya dengan file kosong yang tidak dapat saya hapus. Ketika saya klik kanan padanya, satu-satunya pilihan yang diberikan adalah OpenHomePage, properties, Start without Add on dan Create Short Cut.

File tersebut tidak memiliki nama dan opsi menu konteks delete. Saya tidak dapat memindahkannya ke folder lain. Itu tidak muncul dengan sendirinya pada cmd. Saya tidak tahu bagaimana menghapusnya. Inilah tampilannya di desktop saya:

[EDIT]

Saya mencoba Unlocker, tetapi file kosong tidak memiliki opsi "hapus" atau "buka kunci". Saya juga mencoba mem-boot ke mode aman, tetapi itu juga tidak memberi saya opsi "hapus".

Sekali lagi, masalahnya adalah file tersebut tidak menawarkan saya opsi untuk menghapusnya. Saya dapat menyeretnya ke Recycle Bin, tetapi tidak ada yang terjadi saat saya melakukannya, bahkan tidak ada peringatan. Saya tahu ini terkait dengan semacam virus / trojan. Saya mencoba menjalankan berbagai program antivirus termasuk AVG, tetapi filenya masih tetap di desktop. Jika saya pergi ke C:\Users\*someUser*\Desktop, file tersebut tidak ada. Ini hanya di desktop seperti di desktop layar komputer.

Tidak semua item di Desktop adalah file yang sebenarnya - beberapa di antaranya, seperti "Recycle Bin" atau "Computer" standar, ditambahkan oleh ekstensi shell.

Karena Anda sudah mengetahui GUID, mulailah dengan Editor Registri di HKLM\Software\Microsoft\Windows\CurrentVersion\Explorer (kemungkinan besar Desktop\NameSpace, tetapi mungkin subkunci lain). Regedit memungkinkan Anda mengekspor kunci, sehingga Anda dapat bereksperimen dengan setiap hasil: ekspor ke file, hapus, jika tidak membantu - impor kembali.

Perhatikan bahwa opsi menu konteks yang Anda daftarkan sangat mirip dengan ikon desktop "Internet Explorer" bawaan, jadi bisa jadi itu menjadi rusak dan harus dibuat ulang. Tapi, saya tetap akan mengatakan "hapus".

Jawaban jayjzc sepertinya sudah tidak berfungsi lagi, seperti yang dikemukakan oleh Keltari. Ini adalah solusi yang disediakan oleh Microsoft jika aplikasi Anda (Chrome) tidak terdaftar:

  1. Pertama pergi ke Aplikasi Default di Pengaturan Windows.
  2. Gulir ke bawah untuk menemukan Setel default menurut aplikasi sebagai opsi terakhir dan klik di atasnya.
  3. Sekarang Anda harus disajikan dengan halaman aplikasi, gulir ke bawah sampai Anda melihat Google Chrome . (Jika Anda memiliki beberapa Chromes, cukup klik yang pertama Anda lihat dan Anda akan baik-baik saja).
  4. Klik Google Chrome , lalu Mengelola .
  5. Anda harus berada di halaman dengan Microsoft Edge sebagai default untuk kebanyakan dari mereka. (Ada beberapa aplikasi lain yang dapat Anda atur sebagai default, tetapi untuk saat ini jangan khawatir tentang mereka).
  6. Terakhir, klik setiap aplikasi Microsoft Edge, dan pilih Google Chrome. Ini harus menjadikan Chrome sebagai browser default Anda.

Lapisan Photoshop ada di sana tetapi saya tidak dapat melihat apa pun

Saya telah menghabiskan sepanjang malam mengerjakan tata letak ini untuk klien dan saya 100% yakin saya menyimpan pekerjaan saya - setiap 5-10 detik pekerjaan yang saya simpan.

Begitulah. Saya membuka Photoshop hari ini untuk menyelesaikan beberapa pekerjaan, saya perhatikan bahwa meskipun layer saya ada di sana dan diberi nama, mereka tidak muncul di dokumen / kanvas.

Misalnya, jika saya memiliki layer teks dengan teks merah, saya memilih layer dan warna di atas di panel properti berubah menjadi warna yang saya berikan pada semua header saya - tetapi konten sebenarnya (teks dalam kasus ini) tidak ada di mana pun. halaman.

Terus memecahkan masalah, saya menekan Ctrl 0 jadi saya bisa melihat seluruh dokumen. Kemudian saya mengarahkan mouse ke salah satu lapisan lapisan yang bertingkah aneh, saya mengarahkan mouse ke atas gambar mini lapisan, saya tekan Ctrl dan klik sehingga dapat otomatis memilih konten pada lapisan itu dan sekali lagi, tidak ada.

Ada ide tentang apa yang mungkin terjadi?

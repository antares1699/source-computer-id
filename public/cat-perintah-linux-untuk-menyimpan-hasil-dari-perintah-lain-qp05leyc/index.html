JIKA Anda hanya ingin mencari file tertentu. Seperti jawaban Ignacio:

find -somecriteria > somefile.txt

Jika Anda ingin mencari file tertentu, dan output konten file-file ini.

find -somecriteria -exec cat {} + > somefile.txt

Tampaknya kebutuhan Anda adalah menemukan file dengan FILENAME daripada beberapa kriteria lain.

Ada utilitas yang jauh lebih baik untuk ini daripada 'temukan' yang tersedia, yang disebut 'temukan'. Ia menggunakan database lokasi untuk menemukan file berdasarkan nama, jadi ini jauh lebih cepat daripada mencari melalui sistem file.

Pertama, jalankan 'updatedb' sebagai root untuk memastikan database lokasi Anda mutakhir - dan mungkin tambahkan pekerjaan CRON untuk menjalankannya setiap malam jika belum ada.

Kemudian, gunakan 'cari nama file' atau 'cari -r regexp' untuk mencari file.

Saya menyebutkan opsi kedua karena 'cari abc.log' akan menemukan misalnya abc.log.1 dan abc.log.1.gz dan apa pun yang cocok. Jadi, 'cari -r abc.log \ $' dalam hal ini HANYA akan mencari file bernama abc.log

Akhirnya, seperti yang telah disebutkan, untuk memasukkan keluaran yang dihasilkan dari perintah ke dalam sebuah file, gunakan> seperti di:

locate -r \^abc.log\$ > /home/me/all.abc.log.files

Ya, Anda dapat menggunakan pengalihan:

$ find . > some_file_name.txt
$ cat some_file_name.txt

Simbol > di akhir perintah mengarahkan ulang output apa pun yang biasanya masuk ke konsol untuk menulis file, namanya setelah >.

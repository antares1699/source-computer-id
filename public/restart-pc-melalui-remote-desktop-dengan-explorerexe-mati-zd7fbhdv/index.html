Restart PC melalui remote desktop dengan Explorer.exe mati

Saya mengalami situasi yang sangat tidak biasa di mana saya melakukan remote desktop ke PC Windows 7 dari laptop saya. Proses Explorer.exe telah dimatikan sehingga menu Start dll tidak tersedia. Oleh karena itu, sangat sedikit yang dapat saya lakukan dengan PC saya. Jadi saya perlu me-restart PC saya. Bagaimana cara me-restart PC saya setelah proses Explorer.exe dimatikan?

Anda bisa masuk ke Task Manager dalam sesi desktop jarak jauh dengan menekan CTRL + BERGESER + ESC .

  1. Klik Mengajukan
  2. Klik Tugas Baru (Jalankan ...)
  3. Tipe explorer.exe dan klik baik

Itu akan langsung me-restart shell Explorer tanpa perlu me-restart PC Anda.

Jika Anda dapat mengirim ctrl-alt-del ke komputer jarak jauh, maka Anda harus dapat meluncurkan pengelola tugas. Dari sana Anda dapat mengklik File-> Tugas Baru. Luncurkan prompt perintah (cmd.exe).

Dari sana, Anda dapat memulai ulang komputer dengan penundaan 1 detik dengan perintah berikut:

shutdown / r / t 1

Ada situasi di mana latar belakang desktop terlihat dan penunjuk mouse berfungsi, tetapi Explorer mati dan ctrl-alt-del dan shift-ctrl-esc tidak melakukan apa pun, meskipun keyboard aktif dan dapat digunakan untuk keluar dari screen saver. Remote Desktop berfungsi tetapi tidak membantu atau menyediakan fungsionalitas lebih dari pada komputer host.

Saya percaya ini terjadi ketika Pembaruan Windows / Pembaruan Microsoft telah memulai reboot, tetapi ada sesuatu yang mencegah penutupan terakhir. Jika tombol daya dikonfigurasi untuk tidur atau hibernasi, tampaknya tidak ada cara untuk memulihkan dari ini; setelah bangun atau kembali dari hibernasi, Anda berakhir tepat di tempat Anda sebelumnya. Saya biasanya akhirnya hanya mencabut steker sehingga pada boot saya berikutnya, saya akan diminta apakah akan memulai Windows secara normal.

Kadang-kadang saya menemukan bahwa saya bahkan tidak dapat menggunakan Ctrl + Alt + Menghapus atau Ctrl + Alt + Akhir , jadi saya telah menemukan solusi lain:

  1. Buka Desktop Jarak Jauh
  2. Sebelum Anda menekan hubungkan, klik opsi tarik-turun
  3. Buka tab Program
  4. Centang kotak yang bertuliskan "Mulai program Mengikuti di Sambungan"
  5. Masuk ' penjelajah '(tanpa tanda kutip) ke dalam Jalur Program dan Nama File Kotak
  6. Hubungkan Biasanya

Saya memiliki masalah serupa dengan salah satu instance VM kerja kami dan sangat sulit untuk mendapatkan I.T. untuk mem-boot ulangnya sepanjang waktu, dan sepertinya saya tidak dapat membuka Pengelola Tugas saat instance dalam keadaan ini.

Dalam kasus saya Ctrl + Shift + Esc tidak berfungsi, jadi saya melakukan restart jarak jauh seperti yang dijelaskan di panduan ini . Dari PC lokal Anda, jalankan perintah berikut di cmd.exe:

shutdown /r /m \\remotepcname

Ini berfungsi untuk saya karena saya menggunakan akun domain yang sama di kedua mesin. Bergantung pada bagaimana domain dan kebijakan grup Anda disiapkan, ini mungkin berhasil atau tidak untuk Anda.

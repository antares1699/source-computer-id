Nonaktifkan Java Updater di registri

Pembaru Java meminta untuk mengunduh pembaruan baru. Bagaimana cara menonaktifkan fitur ini di banyak komputer di domain? Ini kotak centang ini:

Tetapi tampaknya tidak mungkin untuk menghapus centang di registri. Saya mencoba:

32-bit:

[HKEY_CURRENT_USER\SOFTWARE\JavaSoft\Java Update\Policy]
"EnableJavaUpdate"=dword:00000000

64-bit:

[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\JavaSoft\Java Update\Policy]
"EnableJavaUpdate"=dword:00000000
"EnableAutoUpdateCheck"=dword:00000000
"NotifyDownload"=dword:00000001
"NotifyInstall"=dword:00000001

Masalahnya, DWORD EnableJavaUpdate menyembunyikan tab Update "di JavaControl Panel, tapi saya tidak ingin menyembunyikan tab ini, batalkan saja kotak centang ini .

Saya mencoba untuk menghapus centang pada kotak centang dan membandingkan konten jalur registri secara visual.

Saya juga mencoba ProcMonitor Sysinternals, tetapi tidak dapat menemukan tempat di registri tempat informasi ini disimpan.

EDIT :

1.7.0_67 Java pada x64 bermasalah, bahwa EnableJavaUpdate di registri disetel ke 1 dan tidak berfungsi (tab Pembaruan tidak ditampilkan). Lihat bukti .

1.7.0_51 Java pada x64 bermasalah, seluruh jalur registri hilang di komputer. Lihat bukti .

1.7.0._51 Java pada x86 memiliki path pada registry dan juga tab Update pada dialog Configure Java. (Saya tidak memiliki tangkapan layar).

Saya harus berhenti meminta untuk mengunduh versi baru dan SETELAH ITU menyelesaikan pembaruan semua versi lama Java di semua komputer.

Dalam pengalaman saya, menonaktifkan Pembaruan Java menggunakan Java Control Panel tidak berfungsi (ini bukan masalah UAC yang dibicarakan Oracle) dan begitu pula nilai registri terkait di bawah kunci "Kebijakan". Itu tidak masuk akal, saya tahu.

Apa yang berhasil hanyalah menghapus nilai SunJavaUpdateSched dari tombol "Run" Windows. Tentu saja, jika Anda memperbarui atau menginstal Java secara manual, kunci ini akan dibuat ulang. Karena Anda menyebutkan memiliki "banyak komputer", berikut adalah contoh cara melakukannya menggunakan baris perintah di Windows 7 64-bit:

reg delete "HKLM\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run" /v "SunJavaUpdateSched" /f

Sumber Cara Menonaktifkan Pemberitahuan Pembaruan Java di Komputer Jaringan Windows 7

Ubah Kunci Registri Java

Buka RegEdit di PC Windows 7. Sebelum membuat perubahan registri, disarankan untuk membuat cadangan terlebih dahulu.

Untuk menonaktifkan Java Update Notifications, telusuri yang berikut ini kunci registri

Pengguna 32bit:

HKEY_LOCAL_MACHINE \ SOFTWARE \ JavaSoft \ Java Update \ Policy

  • Ubah EnableAutoUpdateCheck menjadi 0
  • Ubah EnableJavaUpdate menjadi 0

Pengguna 64bit:

HKLM \ SOFTWARE \ Wow6432Node \ JavaSoft \ Java Update \ Policy

  • Ubah EnableJavaUpdate menjadi 0
  • Ubah NotifyDownload menjadi 0
  • Tambahkan DWORD baru bernama "EnableAutoUpdateCheck" dan setel ke 0

Sisa artikel ini menjelaskan cara menggunakan Kebijakan Grup Windows untuk menonaktifkan pemberitahuan Pembaruan Java. Anda memerlukan komputer Windows 7 dengan Java terbaru yang diinstal dan Windows Server 2008 yang menjalankan Kebijakan Grup Anda.

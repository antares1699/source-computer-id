Dari Situs web Avira :

Saat ini, produk Avira belum siap untuk Windows 8 dan Windows Server 2012 (Dibangun pada Windows 8).

Avira bekerja sama dengan Microsoft untuk mencapai kompatibilitas produk secepat mungkin. Oleh karena itu, dapat dikatakan dengan pasti bahwa produk Avira akan kompatibel dengan Windows 8 pada kuartal pertama tahun 2013 ini.

Informasi yang sama ini juga akan ditunjukkan kepada Anda saat Anda mengklik tombol "Periksa solusi online" setelah gagal diinstal pada Windows 8 [ sumber ].

Menghapus nama masuk dari daftar IsiOtomatis?

Karena teman saya menggunakan komputer saya untuk masuk ke akun Gmail-nya, saya tidak lagi masuk secara otomatis seperti sebelumnya, jadi saya harus mengetik nama masuk dan kata sandi saya setiap saat.

Saya tidak menemukan cara menghapus nama logonnya dari daftar, dan kembali login secara otomatis.

Menentukan Sistem Operasi dan Browser pasti akan membantu, tetapi saya tetap akan mencobanya.

Saat Anda berada di bidang login, tekan tombol panah bawah ↓ . Ini harus membuka daftar drop-down dengan login. Masih dengan tombol panah bawah, pilih login yang diinginkan (milik teman Anda) dan tekan DEL kunci. Ini akan menghapus login teman Anda dari daftar. Jika milik Anda adalah satu-satunya login yang tersisa, itu harus dipilih secara otomatis di waktu berikutnya.

Ini adalah fungsi pelengkapan otomatis. Bergantung pada browser yang Anda gunakan, Anda dapat mematikannya dan menghapus riwayat pelengkapan otomatis.

Internet Explorer 6:

  1. Pergi ke Tools -> Internet Options.
  2. Pilih tab "Konten" dan klik tombol "LengkapiOtomatis".
  3. Kemudian batalkan pilihan "Formulir"
  4. Kemudian klik pada tombol "Clear Forms" dan itu akan menghapus sejarah.

Firefox 2 dan Netscape 9:

  1. Pergi ke Tools -> Options.
  2. Pilih tab "Privasi" dan batalkan pilihan "Ingat apa yang saya masukkan di formulir dan bilah pencarian"
  3. Kemudian di bagian "Data Pribadi", klik tombol "Bersihkan Sekarang ..." dan pilih "Formulir tersimpan dan riwayat pencarian"
  4. Klik pada tombol "Bersihkan Data Pribadi Sekarang".

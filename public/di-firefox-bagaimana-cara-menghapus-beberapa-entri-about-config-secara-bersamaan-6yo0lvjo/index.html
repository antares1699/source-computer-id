Di Firefox, bagaimana cara menghapus beberapa entri about: config secara bersamaan?

Saya menggunakan versi pengembangan Firefox, dan ada banyak entri ubahsuaian di bagian about: config yang ditambahkan oleh ekstensi yang tidak lagi terpasang. Ini daftar yang sangat panjang, dan tidak ada cara yang jelas untuk memilih semuanya dan menghapusnya secara bersamaan. Ini hanya memungkinkan untuk memilih dan menghapus entri satu per satu.

Apakah ada cara untuk menghapus beberapa entri sekaligus di about: config?

Metode sederhana adalah melewati dialog about: config. Buka file prefs.js, yang ada di profil Firefox Anda, di editor teks saat firefox tidak berjalan. Hapus baris yang tidak Anda pedulikan lalu simpan. Buat cadangan juga jika Anda ingin memastikan Anda tidak membuang sesuatu.

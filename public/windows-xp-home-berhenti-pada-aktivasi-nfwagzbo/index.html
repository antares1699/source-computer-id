Windows XP Home berhenti pada Aktivasi

Saya membantu teman memperbaiki instalasi XP Home yang rusak. Kami melakukan penginstalan perbaikan, dan jika saya masuk ke mode aman, saya dapat melihat bahwa semuanya terlihat baik sekarang. Sayangnya, mode aman adalah satu-satunya cara saya dapat masuk (bahkan tidak dapat menggunakan mode aman dengan jaringan). Jika saya mencoba masuk dengan cara lain, ini memberi tahu saya bahwa komputer tidak diaktifkan dan masa tenggang telah kedaluwarsa. Saya dapat memilih untuk membatalkan (pada saat itu saya kembali ke prompt login) atau melanjutkan. Jika saya melanjutkan, saya mendapatkan layar dengan latar belakang biru muda dan kursor mouse yang dapat saya pindahkan, tetapi saya tidak pernah melihat wizard aktivasi.

Saya sudah mencoba ini:
http://social.microsoft.com/Forums/en-US/genuinewindowsxp/thread/fb5c9851-dddc-4261-836a-955313398ac7

Ada ide lain?

Secara pribadi jika itu milik saya, saya akan menggunakan mode aman untuk membuat cadangan data dan kemudian melakukan instalasi bersih.

Bergantian coba gunakan msconfig untuk mematikan semua yang tidak penting dalam layanan, startup dll dan coba mulai dalam mode normal lagi.

Juga gunakan CD Bootable AntiVirus untuk memeriksa seluruh drive apakah ada yang jahat, kalau-kalau itu masalahnya. Lihat http://www.techmixer.com/free-bootable-antivirus-rescue-cds-download-list/ untuk daftar. Jika Anda belum memilikinya.

Pertama saya setuju dengan melakukan penginstalan bersih, karena jelas ada beberapa masalah mendasar dengan Windows yang tidak diperbaiki oleh penginstalan perbaikan.

Cobalah untuk memunculkan jendela Aktivasi Windows dalam Safe Mode:

Mulai | Lari % systemroot% \ system32 \ oobe \ msoobe.exe / a

Anda kemudian dapat menggunakan opsi Aktivasi melalui Telepon.

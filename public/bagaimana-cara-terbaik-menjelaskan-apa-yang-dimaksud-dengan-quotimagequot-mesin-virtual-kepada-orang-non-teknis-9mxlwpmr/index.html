Saya akan menjelaskan di sepanjang baris:

Alih-alih menggunakan hard drive fisik, Kami menggunakan file yang digunakan mesin virtual. Sistem operasi di mesin virtual mengira bahwa file gambar adalah hard drive. Dan itu akan menggunakannya seperti itu. Jadi semua file dan folder disimpan di dalam file itu.

Jadi pada dasarnya sama dengan hard drive kecuali "dalam file", bukan perangkat keras fisik.

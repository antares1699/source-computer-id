Excel: bagaimana cara menonaktifkan mode pilihan perluasan (tombol F8) yang diaktifkan oleh tombol "-"?

Menggunakan Excel 2007, setiap kali saya menekan tombol "-", mode pilihan perluasan diaktifkan. Perhatikan bahwa di program lain tombol "-" berfungsi seperti yang diharapkan. Ini sangat membuat frustrasi karena saya tidak dapat memasukkan nilai negatif.

Saya dapat menonaktifkan mode tambahan dengan menekan [F8], tetapi lain kali saya menekan "-", mode itu akan kembali!

Bagaimana saya bisa mengatasi masalah yang mengganggu ini?

Saya akan mencoba beberapa hal untuk melihat apakah Anda dapat menemukan akar masalahnya.

Sudahkah Anda mencoba keyboard baru / berbeda? Mungkin mengirimkan F8 bukannya tanda hubung (siapa tahu). Anda juga dapat mencoba keyboard di layar untuk melihat apakah menggunakan - menyebabkan masalah yang sama.

  • Pada Windows, buka Start -> Run dan ketik osk untuk mengaktifkan keyboard di layar
  • Di OS X, ikuti petunjuk ini untuk mengaktifkan papan ketik pada layar

Anda mungkin juga memeriksa untuk melihat apakah ada pintasan keyboard kustom yang ditetapkan di Excel. Di Excel 2003, Anda dapat memeriksa pintasan menggunakan langkah-langkah ini di sini .

Terakhir, Anda dapat mencoba "Deteksi dan Perbaiki" di Excel / Office, yang mungkin dapat memperbaiki masalah tersebut.

  • Di Excel 2003 atau XP, pilih opsi "Deteksi dan Perbaiki" dari menu bantuan
  • Di Excel 2007, pilih Orb, "Opsi Excel" di bagian bawah menu, lalu tab "Sumber Daya", lalu tombol "Diagnosis".

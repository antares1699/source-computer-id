Menggunakan perangkat lunak peta pikiran FreeMind, bagaimana cara spasi node?

Saya menggunakan FreeMind dari http://freemind.sourceforge.net/

Saya mencoba memberi node saya ruang bernapas. Dalam HTML, ini akan sama dengan padding node, atau margin, atau line-height. Adakah cara untuk menerapkan pemformatan seperti itu di FreeMind?

Lihat gambar ini untuk menemukan mungkin 2-3 piksel ruang antara gelembung simpul saudara. Saya ingin meningkatkan jumlah jarak ini, atau sebagai alternatif, bantalan di dalam batas simpul.

Dari http://freemind.sourceforge.net/docs/features/0_8_0/Features%200.8.0.html

Mendekati sisi dalam (yaitu sisi yang lebih dekat ke simpul akar) dari a node Anda akan melihat bahwa kursor mouse berubah dan elipsis kecil muncul. Di area ini Anda cukup menarik dan memindahkan node yang menekan tombol mouse (kiri). Selain itu, jika Anda menekan tombol kontrol secara bersamaan, semua node pada level yang sama mendapatkan jarak yang lebih besar atau lebih kecil. Dengan kata lain, saudara kandung dari node yang dipindahkan mendapatkan lebih banyak atau lebih sedikit ruang di antaranya.

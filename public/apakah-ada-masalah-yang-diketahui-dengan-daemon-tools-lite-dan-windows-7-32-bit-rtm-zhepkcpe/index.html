Apakah ada masalah yang diketahui dengan Daemon Tools Lite dan Windows 7 32-bit RTM?

Halaman unduhan untuk Daemon Tools Lite Commercial (yang gratis) versi 4.30.4 mengatakan bahwa Windows 7 Beta didukung (SPTD 1.58). Namun Pro Standar dan Maju versi mengatakan Windows 7 adalah " sepenuhnya "didukung dan menggunakan SPTD 1.60.

Jadi, haruskah saya mengabaikan peringatan dari Windows 7 "Program Compatibility Assistant" dan langsung saja menginstal versi Lite? Atau haruskah saya menunggu sampai ini diperbaiki (atau membeli versi Pro)?

Adakah yang mengalami masalah dengan versi Lite yang berjalan pada Windows 7 32-bit RTM?

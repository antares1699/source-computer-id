Bagaimana saya dapat menjalankan NetMeeting yang telah dihentikan pada Windows 7?

Microsoft menghentikan NetMeeting beberapa waktu yang lalu dan tidak lagi disertakan di Vista atau Windows 7. Saya telah membaca bahwa ada perbaikan cepat untuk membuatnya berfungsi di Vista, tetapi apakah ada hal seperti itu untuk Windows 7?

Saya tahu ada alternatif yang tersedia, tetapi saya bertanya-tanya apakah ada yang berhasil membuatnya bekerja pada Windows 7? Saya hanya tertarik untuk menjalankan komponen klien NetMeeting, untuk menyambung ke rapat yang diselenggarakan di mesin Windows XP.

Netmeeting berfungsi di Windows 7. Begini caranya, dengan asumsi Anda terhubung ke jaringan pribadi.

  • Instal mesin virtual XP dan jalankan
  • Di bilah judul, Tools->Settings->Networking
    • Ubah adaptor dari NAT ke adaptor jaringan perangkat keras yang Anda pasang
  • Siapkan VPN di VM XP, jalankan, dan sambungkan ke jaringan pribadi Anda
  • Aktifkan firewall windows di VM XP (opsional tetapi disarankan)
  • Start->run->conf
    • Konfigurasikan netmeeting

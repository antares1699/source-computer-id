Saya sudah mendengar tentang serangan menggunakan file PDF di Windows dengan Adobe Acrobat dan Foxit Reader.

Apakah Linux rentan terhadap serangan ini saat menggunakan penampil PDF default di KDE atau Gnome atau bahkan xpdf? Apa pemindai PDF yang baik untuk menentukan apakah file PDF itu jahat?

Linux umumnya menggunakan poppler atau xpdf untuk mengurai file PDF, yang tidak memiliki kerentanan yang sama. Jelas, Acrobat Reader untuk Linux kemungkinan memiliki setidaknya beberapa kerentanan yang sama seperti rekan-rekan Windows dan OS X-nya.

Apakah saya perlu menyertakan urutan escape ANSI di \ [dan \] dalam prompt bash?

Saya ingin menyesuaikan prompt bash saya menggunakan kode pelarian ANSI dan telah memperhatikan bahwa di beberapa howtos kode-kode tersebut disertakan dalam \[ dan \] sementara di tempat lain tidak.

Misalnya, ini bekerja sama dengan baik untuk saya:

PS1='\e[0;32m foo '
PS1='\[\e[0;32m\] foo '

Aku ingin tahu apa bedanya?

Dapatkah router berkabel disiapkan sebagai router utama, bertindak sebagai DHCP untuk semua perangkat yang terhubung secara nirkabel ke router sekunder?

Saya mencoba menyiapkan jaringan rumah saya dengan penyiapan berikut: Router utama - router berkabel dengan 8 port untuk menghubungkan semua outlet di rumah Router sekunder di port ruang komputer - Nirkabel, akan memberikan sinyal nirkabel ke rumah

Jika saya mematikan DHCP di router sekunder, apakah router utama dapat menetapkan alamat IP ke perangkat yang terhubung ke sistem secara nirkabel? atau apakah saya harus membiarkan DHCP 'On' di router nirkabel?

Ya, dengan DHCP diaktifkan dari router pertama, router / AP nirkabel akan memungkinkan DHCP untuk melewatinya. Dengan pengaturan seperti ini, router kedua bertindak seperti AP lebih dari sekadar router. Pengaturan yang diusulkan sebagai berikut:

  • 192.168.0.1- router utama / dhcp

  • 192.168.0.2- AP / router nirkabel

  • 192.168.0.3.20- disediakan untuk hal-hal lain yang dapat Anda tambahkan di masa depan

  • 192.168.0.50-249- RENTANG DHCP Slot -Aktif untuk peralatan yang terhubung.

Perhatikan ruang mati dari 192.168.0.21-.49 yang jika Anda memerlukan lebih banyak ruang untuk peralatan ip statis seperti webcam, server, lebih banyak AP ... di masa mendatang. Atau jika Anda menginginkan lebih banyak alamat DHCP. Jaringan Anda akan mendapat ruang bernapas jika diperlukan.
